description

「真剣にお金を稼ぐことを考えるエンジニアの集い（仮称）」から、正式名称が「rake:money」に決まりました。

そのrake:moneyの第2回を開催します。
初回は手探り状態だったので飲食をしながらの懇親会形式でしたが、第2回からは会場を借りて勉強会をまず行い、その後に懇親会を行う形にします。

スケジュールとしては、


  • 20:00〜22:00 勉強会

  • 22:00〜23:00? 二次会（懇親会）

のように実施したいと思います。
なお勉強会だけの参加の場合は会費は400円程度、二次会の会費は3,000〜4,000円の予定です。

参加希望で二次会も参加を希望される方は、必ずコメント欄上部の二次会の参加にチェックを入れてください。

また今回から参加される方は、連絡のためにGoogleグループのrake:moneyに参加をお願いします。

■Googleグループ rake:money

さて今回のお題は「フリーランスがこの先生きのこるには」です。
ゲストにきのこる先生をお招きし（嘘）、現役フリーランスもそうでない方もいっしょに今後のフリーランスのサバイバル戦略について考えましょう。

先の見えないこの不況の中で、フリーランスのIT系エンジニアも案件の減少や単価の低下に悩まされていますが、エンジニアの横のつながりを大事にし、知恵を出し合って成果を共有すれば乗り越えられるはず。

つきましては参加者の皆様方に、恒例の課題を出します。
それぞれの立場別に質問を出しますので、その回答を用意してきてください。


    a. 現役フリーランスの方
    b. 過去にフリーランスの経験がある方
    c. 一般会社員の方
    d. フリーから法人成りした方

の４パターン別に質問があります。
自分に当てはまるパターン向けの質問の回答をお願いします。また自分のパターン以外の質問に対して回答してもかまいません。
この４パターンに当てはまらない方の場合は、任意に答えられそうな質問を選んで回答してください。


    ≪現役フリーランス、過去にフリーランスの経験がある方共通≫
    ab1. フリーランス何年目？
    ab2. 現在の使用言語、その他技術は？
    ab3. フリーランスになった経緯は？
    ab4. どのエージェントに登録している（いた）？　仲介手数料率、営業のフォロー具合は？
    ab5. エージェントとの付き合い方、単価交渉のテクニック等あれば
    ab6. 直受けの経験はある？　そこに至った経緯は？

    ≪現役フリーランスの方向け≫
    a1. どの言語・どんなスキルの単価が高く、また将来性があると思うか
    a2. 独自の営業をしているなら、その方法や工夫していること
    a3. 屋号はつけている？ また事業用のHPはある？
    a4. 将来の展望、キャリアプラン等あれば
    a4. 法人化を考えている？　Yesならその理由およびどんな会社にするつもりかを

    ≪過去にフリーランスの経験がある方向け≫
    b1. 現在の職種は？
    b1. フリーランスだった期間は？
    b2. フリーランスをやめた理由は？

    ≪一般会社員の方向け≫
    c1. フリーランスになることを考えたことがある？ Yesならその理由も
    c2. フリーランスに対するイメージ
    c3. フリーランスの人に聞いてみたいこと

    ≪フリーから法人成りした方向け≫
    d1. フリーランスだった期間は？
    d2. 法人成りした理由
    d3. 現在の会社のビジネスモデルを教えてください

また第1回でやった、推薦図書の持ち寄りを今回も行います。
今回のテーマに沿っていてもいなくても構いませんので、お金にまつわるお勧めの本を各自1冊ずつ持参してください。
なお、前回のどなたかが持ってきた本と重複しても大丈夫です。
