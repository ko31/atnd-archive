description

いま、「ソーシャルメディア」が注目を浴びています。ソーシャルメディアとは、ユーザーが情報を発信し、形成していくメディアのこと。ブログやＳＮＳ、ツイッター、さらにはYoutube、Ustream、facebookなどで発信した情報を、お互いに情報を共有・発信していくことで形成される、全く新しい形のメディアです。様々な情報がネットを飛び交い、これを如何にビジネスに活かしていくのかが大切にされる時代です。

ほとんどの方がまだ気づいてらっしゃいませんが、これらソーシャルメディアが旧来型のWEB のマーケティング手法を根底から変えつつあります。まさに新たな「ネット激変期」が到来したといっても過言ではありません。

本セミナーでは、これまでネット販売等で成功してきた講師がこれから勝ち残っていくために必要な知識や知恵を皆
さんに伝授して参ります。是非この機会をお役立てください！
