description

こんな機会が無ければ行かない人間ドックにフリーランスのみんなと一緒に行こうという毎年恒例にしたい企画
今年の会場はNTT東日本関東病院になります。

参加締め切りは7/6です。

メリット

  • 診察の合間にフリーランス同士会話することで暇つぶしが出来る。
  • 行こうと思って行けなかった人間ドックに参加するきっかけになる。

注意点

  • 人間ドックは健康保険の対象にならないことと、このイベントではしっかりした施設で受けるため高額です。検索すればある程度安いところもありますので個人で受けることも検討してみてください。
  • 必ずしも、毎年受ける必要はありません。ただし、何年も検査を受けていない人、35歳以上の人は推奨です。
  • 表示されている時期、費用は、ATNDのシステム上のものです。実際には予約の上で決定されます。
  • 多人数かつ数カ月先の予約をとることになるため、時期を選ぶことは出来ませんが、２週間前までなら調節やキャンセルが可能です。
  • ATNDのシステムで連絡を取りますので通知が来る設定にしておくか、たまに見てください。必要な返事がある程度ない場合イベントから外さざるをえない場合があります。

参加資格

  • 以前参加したことがあるか、参加者の誰かの面識（紹介）があること
  • 会社などで人間ドックを受けることが出来ないこと

時期

  • 未定ですが、予約の関係上7月～9月になります。お盆は避けます。
  • 人数が多い場合2グループに分けるかもしれません。
  • 全員の予定を合わせることは出来ないと思うので、予め了承ください。ただし、予定が合わなくても、近い日程で人間ドックを受けておくことをオススメします。

費用

男性 67,500円、女性 76,500円
オプションによっては増えます。
一般的な人間ドック費用は、男性55,000円ほどなので、今回は少し割高だと考えてください。

予約処理

  • こちらで一括で予約を取ります。そのため、住所などを聞くことになります。これが嫌な場合は個人で同日に合わせて予約をしてください。
  • 検査結果はこちらに通知はされません。
  • オプション項目は、あらかじめ主催者に個別に依頼しておく必要があります。場合によっては一括で予約できないため、個人で手続きをして貰う場合があります。

時間帯

大体朝から昼過ぎに終わります。
一緒に昼食を食べて懇親会とする流れになります。
