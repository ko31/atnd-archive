description

我々日本PRは「日本を世界に発信することで、日本の観光産業を活性化させる」ことをコンセプトに活動しております。
今回のSocial Nippon Cafeでは、
日本文化である「茶道」について、講師の資格もある方に講話頂きます。

言葉は聞いたことがあっても、普段の生活の中ではなかなか触れる機会が無く、
あまり「茶道」について知ることが出来ないと思います。
ハードルが高いイメージもありますが、「おもてなしの心」は日本文化として根付いているものであり、
日本人なら一度はお話を聞いてみませんか？

知れば知るほど奥が深いものではありますが、体験談という形でお話頂きます。
せっかくのチャンスですので、
是非とも日本人の方はもちろん、外国人の方も、「茶道」に少しでも興味がある方は是非とも気軽にご参加ください！

また、我々は「日本」を通じた親睦を深めたいという想いもあり、
参加者同士でお話頂ける時間もご用意しております。
茶道好き、日本好き、etcな方たちが集まりますので、同じ志向を持った人と繋がることも出来る場となっております。

ご参加いただける方は、右上の赤いボタンから申し込みをお願いいたします。
　

～プログラム～

１．日本ＰＲ 紹介
　簡単に我々 ＮＰＯ法人日本ＰＲの団体概要および活動内容等についてご説明させて頂きます。
２．「茶道」について 講演
　「茶道」について講話頂きます。
３．ＰＲタイム （各自任意）
　皆さんに何かを発信したい方、お一人5分以内のお時間の中で、是非自分の想いをお話ください。
４．交流会
　　参加者同士の交流をすることで新たなつながりを見つけてください。
　　ちょっとした軽食もご用意しております。
