description

訪日外国人旅行者の行動データの分析サービスを提供されているアクセンチュアさま、台湾・中華圏に特化した各種プロモーションサービスを提供されているVponさま、日本最大級の訪日旅行者向けwebサイトJapan Infoの運営を行っているGroodの3社合同で、急拡大を続けている訪日旅行・海外進出市場の最新情報をお伝え致します。

※ご好評につき、上限人数を拡大致しました。一部立ち見となってしまう可能性がございますがご了承ください。

目的

大きな市場ではあれど言語の問題等で取り組みハードルが高い訪日旅行・海外進出市場における知見を共有することで、業界全体の活性化促進を目指します。

対象

訪日旅行、海外進出、越境ECなどの市場での展開に興味をお持ちのメーカーさま、旅行代理店さま、小売店さま、観光地さま、広告代理店さま等。

各企業運営サービス

  • アクセンチュアさま
    • インバウンド・サテライト
      • Wi2さまが2014年12月から提供を開始した、訪日外国人向けの無償Wi-Fiサービス「TRAVEL JAPAN Wi-Fi」アプリを国内にて利用している訪日外国人の情報を活用したサービスです。Wi2さまのWi-Fiアクセスポイントから生成される、匿名化された訪日外国人の位置情報データと、アクセンチュアさまのデータ分析基盤を組み合わせることで、訪日外国人の動態や使用言語などの情報に基づく効果的なプロモーション施策を、誰でも手軽に実現することが可能になります。
  • Vponさま
    • 各種プロモーションご支援
      • 優秀な研究開発チームやビッグデータ処理分析技術、そして広告主に対するビジネス開拓能力により、マクドナルドさま、コカコーラさま、アメリカン・エキスプレスさま、シティバンクさまなど1000社を超える有名ブランドにサービスを提供されています。ユニークユーザは4.5億を超え、広告業務は東京、上海、広州、香港、台北など750以上の都市をカバーしています。現在、上海/東京/台北/香港にオフィスを展開されているアジア地区でもっとも成長しているビッグデータ広告会社です。多くの賞を獲得しており、2015年には「フォーブス・チャイナ トップ100」において、中国で潜在力のある未上場企業第3位に選出されています。
  • Grood
    • Japan Info
      • 訪日旅行者向けを対象とした日本最大級のwebサイトです。特に繁体字圏（台湾・香港）のアクセスは他大手訪日旅行者向けサイトの約3倍となっており、Facebook約35万ページいいね！、Twitterフォロワー約6万を有し、月間約2,000万リーチ、約25万アクションを獲得しています。また、通常のバナー・記事広告以外にも記事作成代行やリサーチ等を通じて包括的に訪日旅行需要の取り組みの支援を行っております。
