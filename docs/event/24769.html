title 『新しい価値とは何か？』
catch 2012年、「つながり」のその先を語ろう
description

「ソーシャル◯◯」という言葉が一大ブームになり、つながる事で何か新しいものが生まれるのではないかという期待感から、様々なソーシャルネットワーク系サービスが生まれてきました。
　2011年、確かに「ソーシャル◯◯」は人と人が新しくつながるカタチを世に浸透させる事に成功したかの様に見えます。しかし、浸透したが故に、つながる事で何か新しいものが生まれる！　という漠然とした希望に対して懐疑的になり始めた人が増えたのもまた事実なのではないでしょうか。
　そんな中、2012年は、もはや「つながり」という生易しいものではなく、そこから「新しい価値を生み出せるか？」という、次のステップが求められているように思います。言い換えれば、それが出来なければ見限られてしまう局面にまで来ていると思うのです。

…ならば、「つながり」を超えた先に見据えなければならない「新しい価値」とは何なのか？
ビジネスに常に付き纏う「今求められている価値」という問題を、様々な年代のパネリストをお呼びして提案・発信することはCoworking Officeを運営する者の責務だと考え企画致しました。
