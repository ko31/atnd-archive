description

朝の時間を有効的に使いたい、一番楽しめ、ポピュラーな朝活で、朝活初心者の方おススメの場になります。

【読書をする】行為は知識をインプットすることですが、本の内容を人前で話す（アウトプットする）ことも行うことで、より一層、理解を深めたり、内容が定着するキッカケになったりするでしょう。

また、インプットとアウトプットのバランスを取ることで話す力・聞く力を伸ばし、相手との共通点を探したり見つけることで会話力ＵＰにつながります。

人前で紹介する内容を準備することで、自然と分析力・まとめる力が身につきます。

ビジネスの実践に繋げることを目標として読書をしている人の中には、本の内容をなかなか実践に繋げることができないという悩みを抱えている方も多いのではないでしょうか？

読書会での感想を共有し、議論することは読書と実践を繋げる第一歩の役割を果たすでしょう。

知識や情報を広めたい、価値観・視野を広げるキッカケづくりをしたい、会話を楽しみたい、早起きのキッカケを作りたい方などなど、お気軽にご参加ください。
