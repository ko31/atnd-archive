【初級デザイナー向け】実戦形式でデザイン力を身につける！デザインハッカソン＠駒沢大学前

■なにをするのか

Webデザイナーとして働き始めたばかりの人や、スクールに通ってるけど実務経験が無い人にオススメのイベントです。
当日発表されるお題をテーマに、1日かけてチームでデザイン制作をしてもらいます。
最後は作ったデザインを10分間でプレゼンテーションしてもらいます。
終わった後は軽食やお酒つきの懇親会も行いますので、よろしければご参加ください。

■こんな人にオススメ

経験半年以上のデザイナーの方から応募可能
※独学のみの方はポートフォリオをご提出ください。

■持ち物

ノートPC

■場所

〒154-0012 東京都世田谷区駒沢5-26-7 駒沢パークサイドテラス ノース3F
※池尻大橋の方ではないのでご注意ください。

出席者を確認し、スキル的にちょうどいい塩梅に3チームに分けておきます。
当日は3チームそれぞれにディレクターを配置します。

■運営会社について

株式会社MAISON MARC：
2015年創業。「仲間の幸せを共に実現し、MAISON MARCに関わる全てを幸せに」を企業理念として他業種企業と協業し、WEBサービスの企画設計・開発・マーケティングを行っております。

株式会社ALCHEMY：
WEB系領域の制作に強みを持つスタートアップ企業。実務経験よりも人柄や向上心を重視した採用を行い、業界未経験の若手も多数在籍。
