WANT SUNDAYとは

今春より 株式会社めい が始める
"REDIY（リディ）"の初イベントです。
コンセプトは「ひとりのほしいを、みんなでつくろう」です。
こんな未来をつくりたいという一人の願いが、集まった人に伝わる、拡がる。そんな企画になったらいいなという想いを込めてWANT SUNDAYをはじめます。
キーワードは『京都"も"よくなるアイデア』です。京都で活動されていなくても、京都もよくなるアイデアになるかも、と思ったらぜひアイデアを話す人としてぜひご参加ください！

もちろん一般参加もどしどし募集しております！^^

※WANT SUNDAYはDetroit Soupという企画を参考にしています。詳しくはコチラの ブログ を御覧ください。

REDIY（リディ）とは


自分らしい暮らし方と働き方を共に考え、学び、作ってゆく場所です。
参加者と一緒になってビルをセルフリノベーションし、
さまざまな機会や経験、場所や物をシェアすることで
自分の人生をカスタマイズするための挑戦を応援します。

Resource：資源
Recycle：循環 　
Research：調査

Design：デザインする
Decide：決断する
Discover：発見する

Let’s DIY!!
