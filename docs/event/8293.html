今月の勉強会は、Ustreamをテーマに開催します。

mixiやTwitterの方では書いていたと思うのですが、11月に開催される
三篠公民館の公民館まつりに参加します。そこで、展示と、インターネット
配信を行うので、その準備の為に、今回の勉強会ではUstreamをテーマに開催します。
CamTwistを使ったり、QuartzComposerでCamTwistのプラグインを
作るなどして、インターネット配信の準備を行う予定です。

公民館まつりでお手伝い頂ける方もそうでない方もご参加下さい。

開催日時:2010年 10月30日(14:00 から 17:00)
開催場所:三篠公民館　研修室１
参加費:無料

勉強会の参加登録は、ATNDで受付中です。

広島マックユーザーグループ|10月30日(土)の勉強会 : ATND
http://atnd.org/events/8293

それからもう一つ、公民館まつりの展示で、Macの写真をスライドショーにして
見せようと思っているので、皆さんが使っているMacの写真を募集中です。

Twitterを使われている人は、ハッシュタグ #3sasaFes2010 を付けて、
Twitpic等にアップロードしたアドレスをTwitterで投稿して頂くだけでOKです。

Twitterをされてない人は、何処かウェブ上にアップロードして、
メールか何かでアドレスをお知らせ下さい。宜しくお願いします。

ちなみに公民館まつりは、11月13日(土曜日)、14日(日曜日)に開催されます。
当日お手伝い頂ける方は出来るだけ、10月30日(土)の勉強会に参加して頂けると
助かります。
