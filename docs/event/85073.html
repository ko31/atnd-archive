概要

発表者の開発中のアプリを参加者でぼろくそに言う会
ぼろくそに言われるのは発表者だけです
言う側だけの参加、募集中！

目的

発表者は開発中アプリの改善点を得る
参加者は発表者がぼろくそに言われる様を見て、自分のアプリを改善するヒントを見つける

参加条件

  1. アプリがリリースされるまで、アプリの内容を秘密にできる人
    （開発中アプリを配布するため）
  2. twitter等で事前に連絡の取れる人

日時

2017年2月11日（土）15:00〜18:00（予定）
2017年2月18日（土）15:00〜18:00（予定）
2017年2月12日（日）15:00〜18:00
*開催日時再変更しました、すみません。

アクセス

15:00までに「グランフロント北館タワーC １階 ナレッジプラザ（吹き抜けの広場）」に集合ください。
まとまって入ります。
遅れて来られる方は個別対応します。
管理者のTwitterアカウント（@y_sakaki）宛にメンションをしてください。

参加費

無料

発表の流れ

  1. 持ち時間は30分
  2. 開発中のアプリを参加者に配布
    （開催３日前から事前配布あり）
    　TestFlight / apkファイルなど
    　そのため、管理者がTestFlightでの配布用のメールアドレスを個別に伺います
  3. 発表者はアプリ概要や特に見て欲しいポイントがあれば伝える
  4. 参加者はアプリを触り、指摘事項を共有スプレッドシートに書く（匿名可）
  5. 参加者はアプリに対して感想・指摘・アドバイスを言う

ルール（策定中）

  1. 遠慮はしない
  2. お世辞は言わない
  3. 理詰めで指摘するように心がける
  4. 発表者がギブアップしたら即終了
  5. 発表者は反論しない
  6. 暴力禁止
  7. 発表が終わったら仲良くする
