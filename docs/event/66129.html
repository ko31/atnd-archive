description

今、キャリアカウンセラーの方はもちろん、現場リーダーやマネジャーの方も、キャリアの視点を持ってメンバやクライエントに接することが求められています。

本講座では、以下に示す5つの力とビジネスに関する知識を身に付け、多様化・複雑化した環境下において、クライアントやメンバが持っている思いや強みを引き出し、その人ならではの生き方・働き方を共に創り上げていくことができる人材になることを目指します！

【身に着ける5つの力】
① 共感力：相手の気持ちや思いを相手の立場で理解し一緒に喜び悲しめる力
② 診断力：一定のキャリア開発の手法により相手を診断する力(本講座ではキャリアアンカーを主とする)
③ 質問力：適切な質問により相手も気づいてない潜在的な思いを引き出す力
④ 助言力：相手の改善点を相手が素直に受け入れられるよう助言する力
⑤ 動機力：相手の潜在的な思いを引き出し行動に踏み出させることの出来る力
