tebaben01

tebabenとは、Webサイト制作全般に関わる勉強会です。

tebaben開催のきっかけ

tebanomiで知り合った方々で「何か勉強会をやりたいねー」というゆるーいノリがきっかけです。
手羽飲み（tebanomi）だから、手羽勉（tebaben）で良くね？というノリで命名されました。

勉強会の内容

  • 各自が使ってるこれ便利！なツールを発表
  • 事前にアンケート（紹介したいツール、知りたいツール）
  • アドオン（Firefox、Chrome）も可！

タイムスケジュール

  1. 自己紹介（30分）
  2. ツール共有（60分）
  3. 質問・共有など（60分）
  4. 次回勉強会の議題決定（30分）

tebanomi vol4 同時開催

もちろんtebanomiも同時開催です。
tebanomiとは、コーダー飲みやCSS Niteで知り合ったWebサイト制作に何らかの形で関わる方達の交流会の場です。
堅苦しいものではなく、軽い感じの飲み会です。
第一回が手羽先だったので、安易にtebanomiと名付けました。
