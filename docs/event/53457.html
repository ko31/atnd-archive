ACHIEVUSとは「ACHIEVE（達成）」と「US(私達）」の造語です。

「参加者全員で１つの目標を達成する」という意味で、単純な勝ち負けを競うゲームではありません。

お互いに協力しながらゲームを通じて一つの目標を達成することを目的とした新感覚のゲームです。
そして遊びながら、自分の人生における夢や目標を達成するための「リーダーシップ」と「思いやり」が自然と身についてしまうのです。

その秘密はゲームで使われる「17の成功原則（言葉）」と「ルール」にあります。

17枚のカードを揃える過程で、夢を叶えてきた成功者達に共通している心構えを学べるようになってます。
そしてゲームをしながら、チームで一つの目標に向かうことで自然と仲良くなり、参加者みんなで楽しく交流できます！

　開発者はゲームの達人、ジャーミーレイザー。 ３年前にナポレオンヒルの成功哲学に出逢い、もっと早く知りたかった、という思いから家庭用の教育ゲームを作りました。 「世代を超えて分かち合い喜べるゲーム」 「勝ち負けが無いカードゲーム」 「心(Heart)と頭(Mind)を一致させてゆくワーク」 「６０分で社会や人生の縮図が体験できる遊び」 「遊びながら夢が叶う魔法のようなエクササイズ」 「成功哲学が自然と身につく教育ゲーム」etc
