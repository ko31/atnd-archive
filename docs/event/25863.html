a-blog cms 1.5 がリリースされました。2年ぶりに仙台での勉強会を行います。新しい機能の紹介や実際のカスタマイズをハンズオンで実践頂きます。今一番要望が多い、Facebook対応や、スマートフォン対応について紹介します。

■日時

2012年3月13日(火) 13:00-18:00

■場所

TRUNK | CREATIVE OFFICE SHARING

■費用

無料

■対象

a-blog cms のカスタマイズを勉強したい方

■内容

まだ、変更になるかもしれませんが、このような感じを予定しております。

1.5の新機能紹介

  • iPhone対応
  • 新しいテーマの話
  • モジュールの変わったところ
  • インポート機能

ブログのテーマのハンズオンカスタマイズ

  • Facebook 対応
  • Twitter API対応
  • ユニットグループの調整
  • レスポンシブ Web デザインの問題点を改善
  • Syntax Highlighter の実装

■講師紹介
　
h4. 山本 一道 ( Kazumichi Yamamoto / @kazumich )

  • 有限会社アップルップル 代表取締役
  • WCAN ( Web Creators Association Nagoya）主宰
  • kazumich.log

2000年秋に最初の「a-nikki」という日記 CGI を公開しました（この時に考えたインターフェイスが a-blog cms まで引き継がれています）。当時はプロバイダに勤めていましたが、2002年に独立し Web 制作を個人事業主として開始、2004年秋にはアップルップルを法人化して「a-blog」を開発しリリースしています。また、名古屋の Web 業界を盛り上げるための活動として、WCAN という Web 制作者のための勉強会の主宰を2000年より行っております。
