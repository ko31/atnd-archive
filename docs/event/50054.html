春色タマネギ収穫体験ツアー

title 春色タマネギ収穫体験ツアー　
catch 5/16~5/18＠淡路島
description

こんにちは、島の情報室のカナヤマ＆オオタです。
島の外に住んでいる、生まれも育ちも淡路島の２人が、
島内外のおもしろい人たちと出会える場をつくりたくて、
このツアーを企画しました。

社会人になると、男女問わず新しい出会いってなかなかありませんよね。
さらに出会っても、学生時代みたいに長い時間一緒にいられないから、
なかなか仲良くなれなかったりします。
だったら、長い時間一つの目標に向かって作業すれば、
仲良くなれるんじゃない？
ちょうど今は淡路島の名産のタマネギの季節だし、
みんなで一緒に収穫したり、お料理したら楽しそう！
そんな気持ちで始めた企画です。

それに、せっかく淡路島まで来てもらうんだから、
田舎暮らしにも興味を持ってもらいたい！
そこで、田舎で仕事づくりのアドバイザーをしている伊藤洋志さんや、
淡路島移住一年生のご夫婦に、お話を伺います。

伊藤さんには田舎で元手をかけずに小さな商売を始めるコツを、
移住一年生のご夫婦には、
夫婦をうまくやるコツなんてのを教えてもらいます。
田舎暮らしに興味あるけど、周りに話が通じなくて〜、
みたいな方には、ぴったりかも。

ただ、いちおう“出会い”とは言ってますが、
カップル成立は二の次なんです。
畑で婚活☆とか、淡路島の農家さんの結婚相手募集〜♪ではないのです。
それよりも、みんなで作業したら楽しいし、
生活に必要な技も身につくし、田舎暮らしするなら
知っておきたいことも、先人に学べたりする。

友達できたらいいな、
ついでにパートナーが見つかればラッキー！
みたいな気軽な気分でご参加願えたら〜。

個人的には、作業後に温泉入って飲むビールは、かなりおいしい！
そういう、みんなで楽しくわいわいできる場を作れたらうれしいです。

田舎暮らしのリアルを体験するのがテーマです。

１日目は農作業の中でも辛いといわれるタマネギ収穫にトライ！
南あわじのカナヤマ家のタマネギ畑でお試しにやってみましょう。

２日目はタマネギ料理体験。
田舎じゃ旬の季節には、毎日同じ野菜が食卓にのぼります。
豊かな食生活のため、料理のレパートリーを増やしましょう！
淡路島の新タマネギを使って、いろいろ試しにお料理します。
伊藤さんや移住一年生のご夫婦を囲んで、
おしゃべりしながらごはんを食べます。

■日程：5月16（金）夜〜18（日）昼
　※雨天決行、荒天の場合は未定

■集合場所：
陸の港西淡バス停、西淡志知バス停
※車で直接金山家に来ていただくことも可能です（要連絡）。
※時間や場所の関係でバスが出ている時間帯に来られない場合は個別対応も可。

1979年生まれ、香川県出身。
モンゴル武者修行ツアーや、京都の一棟貸しの宿「古今燕」を運営する
ナリワイ実践家。
文筆活動も行い、著書に『ナリワイをつくる』（東京書籍、2012）、
『フルサトをつくる』（『ニートの歩き方』著者phaさんと共著、東京書籍、2014）。
ほか、予定に『ナリワイと服（仮）』（サウダージ・ブックス）。
※ナリワイをつくるについては このページ を参考に

　　　お話を聞く会

　　午後：淡路島で伊藤洋志さんの新刊トークショーを行います。
