a-blog cms 勉強会 大阪：概要＆内容

さて今回で4回目になりますa-blog cms ユーザー勉強会大阪、今回も少人数ですから参加者のわからない事や実際に自分で導入中に出てきた疑問や質問、つまづいたりはまったりした事などをできるだけ解決できるような場にしたいと思います。
もちろん使った事無い人や初心者の人、興味がある人、今後仕事で使ってみようと思っている人、MTやWPとどんなところが違うのか知りたい人なども是非ご参加ください。

a-blog cmsもリリースから３年経ち、ver.1.5.2がリリースされたばかりですので、新しくなった部分などを確認しながら、実際にどんなことができるのか各自で触りながらの体験していく感じです。

特に今回は、最初から入っている企業サイト制作のひな形にできる「site」テーマの中をじっくり見ながら、どこを触ればどうカスタマイズできるのかを探ってみましょう。
このテーマはよく出来ているので、予算がない案件でもコレをベースにすこしカスタマイズすれば充分ペイできる内容で作れますから、小さな低予算案件でも受注できる体制がとれるようになるかもしれませんよ。

今回は、全く初心者の方（＋前回までの復習をしたい方）と、もうある程度わかってるからその先をやりたい人と、開始時間をずらそうと思います。

■時間

15：00〜19：00まで。

※初心者＆初心者の人と復習したい人
15：00から開始します。
19：00まで後から参加する人たちと一緒に引き続き勉強してもらって大丈夫です。疲れたから16時00分で終わりたい人は退席してもらってもOKです。

※もう初心者コースはわかってる人
16：00から開始します。

■PCの持ち込みについて

各自PCをご用意ください。
できれはあらかじめローカル環境にa-blog cmsをインストールして参加頂くとスムーズに開始できますのでチャレンジしてみてください。

ローカル環境にインストール ampps編 Windows/MacOSX共通
amppsはioncubeローダーが標準設定済なので、下記のMAMP・XAMPPより簡単にインストールできます。
「 AMPPSでローカル環境を作ってみる 」
「 a-blog cmsローカル環境インストール 」
「 ローカル環境にインストール ampps編 Windows/MacOSX共通 」
こちらの記事を参考にすればできるはず！

もしも「やってみたけど出来ないよ〜。」という人もそれで参加するのを諦めないで会場でインストールするところから一緒にやってみましょう。一人でつまづく事もみんなでやればきっとできると思います。

ローカルマシンにではなく、実際にサーバーにインストールして使いたいという人は、a-blog cmsが動くサーバーを事前に契約済ませておいてください。（今ご利用のサーバーが対応していればそれで大丈夫ですし、「 サーバーカウボーイ 」さんであればa-blog cms簡単インストールも対応していますし、お試し期間もあるので利用するとかもいいと思います。）
当日はインターネットに繋がる環境を用意する予定ですが、何かトラブルで繋がらない事があるかもしれませんので、サーバーで動かしたい人はイーモバなど各自で繋がるように準備ください。
