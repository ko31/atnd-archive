アプリ海外進出セミナー

　写真共有の「Snapeee」が世界で大ヒットのマインドパレット社をゲストに迎えて、
海外子会社とともにアプリの海外進出をサポートするアドウェイズがセミナーを行います。

　現地ユーザーのネット環境、習慣、属性などを踏まえて、実際に海外でヒットしているアプリの実例と
その秘訣をご紹介いたします。
「中国ってジェイルブレイクが多いって聞いた、マネタイズできるの？」「Snapeeがアジアで大ヒットしてる
理由は？」などの疑問にお答えできる内容です。

【内容】
■株式会社マインドパレット　取締役副社長 （CTO）神尾 隆昌氏
Snapeeeはリリース半年でアジアを中心に100万DLを達成。
国内でもGREEやKDDIと提携し着実に拡大、3月30日にAndroid版もリリースと勢いにのる。
そうしたプロモーションやマーケティング戦略、開発に当たっての経緯をアプリ開発の第一人者である
CTOの神尾氏に披露して頂きます。

■株式会社アドウェイズ　海外事業部（愛徳威広告(上海)有限公司）　堀井良威
昨年から海外展開を急加速させ、中国・米国・香港・インドネシア・シンガポール・フィリピンなど
海外子会社も各地に設立。日本を含めたデベロッパーのアプリグローバル展開支援事業を行う。
今回のセミナーでは中国をメインに中国スマートフォン市場・中国スマートフォンユーザーの実態、
デベロッパーを取り巻くプロモーション手法の違い等を講演いたします。
