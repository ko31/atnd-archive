description

日本の伝統的な着物や呉服関連商品を日本と海外へ販売するネットショッピングサイトをスタートしました。　商品を掲載して販売したい呉服店の方、すでにショッピングサイトを持っているが販売チャネルを増やしたいお店の方に最適です。　セミナーでは、デモサイトで概要をご説明いたします。　

■和装美どっとこむの概要

「和装美どっとこむ」 は無料で掲載でき、販売時に販売手数料（18％）を頂く、成功報酬型の販売代行サイト事業です。　商品掲載数などの制限はありません。　画像や商品情報を送って頂ければ弊社にて掲載、オンラインショップにてインターネット販売いたします。　サイトは日本語、英語の商品PRが可能で、海外へも販売できます。　お客様が「和装美どっとこむ」のサイトのショッピングバスケットで商品を購入する手続きをして頂き、その後、出店者様へご連絡して、出店者とお客様との直接販売・納品請求して頂きます。

■掲載呉服店募集セミナー日程

日時　：　4月12日　午後2から2時45分

場所　：　JR横浜駅北口下車　かながわ県民センター　4階

参加費用　：　無料
