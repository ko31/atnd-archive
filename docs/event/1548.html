来る9/24(木)にFlash OOPの第34回勉強会を行います。

今回はゲストにソーシャルゲームを世界的に展開しているRockYou!の創業メンバーであるJia Shen氏、石塚 亮氏、そしてdangoの三並 慶佐氏、石橋 広在氏をお招きし、海外から見たFlashのソーシャルゲームの現状を紹介していただきます。

また、Adobe Edge Newsでも紹介された、AIR／FlashによるFeliCa対応のアプリ開発について、ソニー株式会社のFeliCa企画開発部門の方にお話いただく予定です。

勉強会終了後は、交流会も予定しています。

皆さん是非ぜひご参加ください。
