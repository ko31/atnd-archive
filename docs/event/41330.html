皆様、大変長らくお待たせいたしました。データセンターな人の夕べ実行委員会です。
前回の夕べから約半年経ってしまいました。
間もなく夏ですが、いまもデータセンターは皆さまのデータを安全にお預かりしつつ、
元気に電気を消費していますね。

さて、今回の夕べは更に日本の中枢に近づきます（笑）
何と首相官邸すぐ隣、三菱総合研究所にお邪魔することになりました。

今回は、まず場所を提供いただく三菱総合研究所より、データセンター事業の紹介をいただきます。
日本のCIAとも言われている情報収集能力、何でも知ってるJDCC事務局高橋さんも登場予定。お楽しみに。

次はネットワークネタです。SDN(Software Defined Network)はやってますよね。でも本当に使えるの？
そんな素朴な疑問に関して、ネットワークインテグレータが本音トークバトルを披露。

最後は前回の夕べで盛り上がった都市伝説の続編です。前回時間が足りなくてお話しきれ
なかったネタをデータセンターおやじたちは更に話したいそうです。

さぁ、というわけで今年の夏も楽しく乗り切りましょう。
