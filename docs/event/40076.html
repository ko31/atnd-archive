特設サイト

自宅ラック式オンサイト合宿2.0 特設サイトに飛びます

目的

1. 各自テーマを決めて好きな研究をする　
2. そのためのインフラはチェックインが完了次第、みんなで作る

→検討の結果、開催の目的を変えました。中小企業向けのサービスに限定するとつまらないという意見があったためです。(6月30日)

何をするの？

1. 民宿のフロントから外壁伝いに4階(または3階)に有線LANを引き込み、成端する
2. ルータを設置し部屋間で別々のセグメントとなるようなネットワークを作成
3. DNS、DHCPサーバを構築
4. その後温泉を楽しみ、食事後、部屋に戻って各自好きなサービスを構築

費用および移動手段

1. 3部屋の素泊まりルームチャージとします。
2. チェックインは交渉の結果12時からとなりました。(通常は15時)
3. 部屋代36,000円を人数割りします。(8名なら4,500円/人、10名なら3,600円/人、12名なら3,000円/人)
4. 移動費は各自負担です。車を出せる人はご協力をお願いいたします。
5. 越後中里駅より徒歩0分です。但し本数極端に少ないので注意。

ルール1：プライバシーに配慮します

1. 女性の参加者はご希望でしたら別部屋となるよう配慮いたします。
2. 子連れ参加もご相談ください。実績はあります。

ルール2：要領よく構築して温泉に入りましょう

前回の反省を踏まえ、民宿側よりチェックイン時間をサービスしていただきました。

1. 構築時間を長めに取れるので、チェックイン後は早めに基本構成を作ってしまいましょう。
2. 当日は1時間ごとに進捗確認打ち合わせをしましょう。
3. 構成図を紙で用意するよう検討します。
