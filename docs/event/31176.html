description

今回は二社合同での開催です。 SHIFTBRAIN 社の方をお招きし、
ワークショップ形式で WebGL, Three.js 関連の技術を学んでいく予定です。

実際に手を動かしながらになりますので、各自エディタと WebGL の動作するブラウザ (Google Chrome 推薦) をご準備ください。

題目

  • Three.js イントロダクション (30 分程度)
    • Three.js でできることや特徴を手短に。
  • Three.js の DOMRenderer の紹介 (30 分程度)
    • Three.js の中でも変わり種な DOMRenderer を紹介します。
  • Blender と Three.js で作る 3D コンテンツ (60 分程度)
    • インタラクティブな 3D コンテンツの制作ノウハウを紹介します。
  • GLSL でシェーダプログラミング (90 分程度)
    • シェーダーの紹介とともに実際にシェーダプログラミングを学んでいきます。

対象者:

  • WebGL または 3D プログラミングに興味のある方 〜 ある程度 3D プログラミングの経験がある方

関連かもしれないリンク:

- GLSL Sandbox: http://glsl.heroku.com/
- Blender: http://www.blender.org/
