概要

第5回目のPure Data パッチングサークルを開催します。
パッチングサークルはPure Dataの気軽な勉強会です。
2013年の7月より奇数月に開催しています。
前回に引き続き、新宿Hatchup様のイベントスペースを使用させて頂けることになりました。
Pure Dataにご興味がある初心者の質問から実際に使っているユーザーのパッチの披露まで、自由に発表したり情報交換をしていきましょう。

場所

Hatchup
東京都新宿区新宿5-18-20 新宿オミビル4階﻿

参加費

無料

内容

  • Pdのパッチや作品、使用例の発表（詳細未定）
  • 参加者の飛び入り発表タイム
  • 出てきた質問や疑問をみんなで考えるタイム

発表予定

*平田陽一　libpdについて紹介
*中西宣人　自作楽器、Electronic Nights Greeceのレポート
*松村誠一郎　expr~で曲作りVol.2、GEMの可能性を探る
*中村優生　arduino+センサ+Pdによる自作楽器

主催

松村誠一郎(Pure Data Japan)
横堀雅之(テトラスタイル)

備考

当日は、特にコンピュータやPdの用意はありません。ご自身でPd-extendedをインストールしたパソコンをご持参ください。
（重要）Pd-extended(Mac)のインストール
Mac OSXにPd-extendedをインストールする場合は、32bit版の最新版の方にしてください。OSが64bitでも問題なく動作します。
64bit版の方はβ版で時々動作がおかしくなるのを確認しています。
