title Tokyo Midtown DESIGN TOUCH 2008
catch 東京ミッドタウンは、今秋デザインイベント「Tokyo MidtownDESIGN TOUCH 2008」を開催します。
description

本イベントは、“JAPAＮ VALUE（日本の価値・感性・才能）”を創造・結集し、“世界に発信し続ける街”を目指す一環として“デザインを五感で楽しむ”というコンセプトで昨年よりスタートいたしました。
今年はコアイベントとして、日本人デザイナーおよび日本にゆかりのあるデザイナーの作品などを対象として、「DESIGN TOUCH Auction」を開催します。
日頃、体験する機会が少ないオークションにおいて、壇上で次々と紹介される数々の優れたデザインをお楽しみいただけるイベントです。このオークションを通して、世界的にも評価の高い日本のデザインを五感で楽しみ、生活を楽しく、豊かにすることを目指しています。
また、オークション初心者の方に、レクチャーつきの「Challenge Auction」もご用意しています。

現在、出品の募集を行っております。
