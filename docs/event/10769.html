12月の東京勉強会の告知です。
今年最後の勉強会はSencha TouchとExt JSから少し離れて、最近話題のTitanium MobileやNode.jsにフォーカスをあててみたいと思います。
とりあえず募集人員は25名で始めますが、希望者が多い場合は50〜60名まで入れる部屋を確保していますので（ビル・フロアは同じ）、ぜひご参加ください。
[12/06] 50名に増員しました
[12/08] 既に定員オーバーしていますが、これまでのキャンセル率を見るとまだ大丈夫だと思われます。参加を希望される方は念の為に補欠登録をお願いします。

アジェンダ は下記の通り：


  • Titanium Mobile Introduction（Appcelerator 増井さん a.k.a @masuidrive）

  • Node.js Rocks!（Node.js 日本ユーザグループ@masahiroh ）
  • Sencha Touch + Node.js + WebSocket（仮題）（小堤）

注意事項：


  • キャンセルされる方はできるだけ早く、そして必ずキャンセル処理を行ってください（補欠登録の方が参加可能なのかどうか分かりません）。補欠登録となった方は、繰り上げの通知は行きませんので、時々チェックしてください。

  • 勉強会後の懇親会（実費）の会場を予約するために懇親会参加のアンケートを設けています。必ずご回答ください。

過去の勉強会の資料についてはExt Japanのブログよりご覧いただけます。

参加費は無料です（但し、懇親会は有料となります）

勉強会のハッシュタグは#extjapanでお願いします
