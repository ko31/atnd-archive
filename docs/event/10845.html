松戸アートラインプロジェクトで行われている企画＜図書館＞にて、
アーティストによる作品紹介（アーティストトーク）を行います。
アーティストと参加者が、作品の面白さや魅力について、フランクに話せる場です。
日時　2010年12月11日　13：00〜
場所　＜図書館＞（松戸市根本3-9アクシス根本1F）
参加費　500円
アーティスト
・村上裕
・内田聖良
・代本板（中島佑太×石幡愛）

アーティストプロフィール
・村上裕｜MURAKAMI Hiroshi
１９８３年うまれ。考えごとが好き。
何かになる前の時間。何でもない。何でもない。ことは可能だろうか。
いわゆる発話のまえのくちごもったかんじあるいはずっとどもった感じ。

・内田聖良｜UCHIDA Seira
美術作家、ニート・フリーター　http://siruchd.web.fc2.com
1985年埼玉県うまれ。武蔵野美術大学油絵学科卒。既製の価値観に囚われない思考をつくる表現方法を探る。
１１月の「東京の条件」プロジェクトでは千代田区秋葉原で、秋葉原によく通う人たちとアートの間で制作。

・代本板｜DAIHONBAN
アーティスト・中島佑太と研究者・石幡愛の子ども向けワークショップユニット。2010年結成。
中島佑太…1985年群馬県出身。2008年東京芸術大学卒業。出来事をデザインするアーティスト。
石幡愛…1984年福島県出身。東京大学大学院博士課程在籍。まちを遊びと学びの場にする研究者。

※＜図書館＞とは
松戸アートラインプロジェクト2010に参加している「代本板（中島佑太×石幡愛）」というワークショップユニットが、松戸の方々のいらなくなった本と、その本にまつわるエピソードを集めて作る、おしゃべりできる＜図書館＞です。
http://daihonban.exblog.jp/
