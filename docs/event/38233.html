2013年新入生歓迎LT祭り！


今年もieで新入生歓迎LT祭り開催します！

内容としてはこれまでと同様、
これから情報工学科に入学する新入生に対するアドバイスや、
学校生活のこと、はまっているオススメの趣味や、
他で開催する会の紹介など、話す内容は基本自由です。

発表者の発表時間は

5分程度

を予定しています！

新入生も在学生も外部の方も、たくさんの参加を願っています！
よろしくお願いします！

発表するためのがPCない人など、困っている方がいれば声かけてください〜〜〜！！



日程と場所


日程 : 4月29日(月) 14時〜17時


日程 : 4月20日(土) 14時〜17時
日程変更、本当にすいません。

場所 : 琉球大学工学部1号棟322教室
