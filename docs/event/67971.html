セガ×あいちゃれ

ゲームにまつわるサービスと進化（アイデアの出し方講座付き）

あいちゃれ 2015のキックオフイベントとして、セガゲームス セガネットワークス カンパニーの渡辺 京介様、谷 英高様、相澤 純様をお迎えし、【ゲームにまつわるサービスと進化】をテーマとしてご講演いただきます。また懇親会では、セガのスマホゲーム大会も開催します！

日時・場所

2015年7月16日(木) 18:00-20:30（クリエーションコア1F CC101） 【交通アクセス】 【キャンパスマップ】

あいちゃれ とは？

ジャンル不問、開発言語不問、実行環境不問のソフトウェア創作コンテストICT Challenge+R（正式名称： あいちゃれ）は、 高校版（小学生・中学生・高校生） と 大学版（大学生・大学院生） の二本立てで同時開催します。
あいちゃれは、立命館だけに限らず「 全国の小学生・中学生・高校生・大学生・大学院生 」も応募可能です！

http://www.ict-challenger.jp/
