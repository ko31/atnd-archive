description

自由大学「ナリワイをつくる」講座の参加者を中心とした同窓会です。
お互いのもっているものを持ち寄って、ナリワイしつつ賑わいましょう！

「ナリワイをつくる」参加者だけでなく、ご友人の方も参加可能です。

タイムテーブル

(16時〜)18時〜21時

　　（お手伝いは16時から。手巻き寿司は17時から）

もちろん途中参加可能・途中退出可能です。

時間
16:00- 準備 準備できる方集合ー！ ごはんを浸水
買い出し行くよ。鍋など借りに行くよ。 ヒモを会場に張る。
17:00- 料理作り 手巻き寿司のお米と海苔セット！鍋作りー。 下準備完了
18:00- チラシ作り チラシ作りするよー、ヒモに吊るすよー 無理に作らなくてもいいですよ
18:30- 食事 手巻き寿司まきまき。チラシを見ながらワイガヤ〜！
20:30 片付け みんなでお片づけ
21:00 お片付け完了！あとは流れで！

参加費・お持ち物

・参加費：1000〜1500円くらい
・手巻き寿司の具材になりそうなおかず
・ナリワイを発表したい方・仲間を集めたい方は、そのためのポスター
　（A1など大きめのサイズで事前に印刷をお願いします）

参加費の利用目的：以下のものに使います
・基本的なお飲物{ビールと焼酎とウーロン茶}
・手巻き寿司に使うお米とノリ
・トマト鍋（たぶん）
・おつまみ
・交流用に使う、チラシ作りのための紙・付箋など（予定）
・会場費 500円/人

内容

今回のお食事は手巻き寿司です。
皆さんで巻き巻きしておいしい手巻き寿司を食べましょう！
また伊藤さんにご提案頂いたトマト鍋も作る予定です。

今回は立食パーティな感じで、ナリワイ最後の講義で行う「チラシ発表」的なものをできたらいいなと思っています。その場で描いて頂けるように紙を置きますので、事前準備などは必要ないです。また中身についてもそこまで深く考えず大丈夫です。やりたい方だけで結構ですので、気負わずにどうぞ。もちろんやりたい方はどうぞ力を込めてください！その際は、A1などの大きいサイズでポスターの作成をしたものをお持ちください。（手書きでも印刷でも、見る人の印象に残ればなんでもよいです。作品など実物でもよいですよ！）
（ナリワイの初期に受講したメンバーは「チラシ発表」はやっていませんが、ご指導願いましょう！）

それではみなさま、参加登録よろしくお願いします！
