講演内容

櫻庭 祐一

何が変わった JavaFX 2.0

2011 年 10 月 3 日にリリースされた JavaFX 2.0。JavaFX Script を捨て、Java の GUI ライブラリに大変容をとげました。
本セッションでは、JavaFX 2.0 での変更点について、デモを交えて紹介していきます。

発表資料

関谷 和愛

GroovyFX – Groovy な JavaFX 2.0

JavaFX 2.0 のポイントの一つは、さまざまな JVM 言語からの利用が可能になったことです。JavaFX をより簡単・便利に使うための Groovy なラッパー、GroovyFX の概要や基本的な使い方などについて、デモを交えて紹介します。

発表資料

深井 康行

JavaFX 2.0 + Scala → ScalaFX

JavaFX が「Java の GUI ライブラリ」になったことにより、JavaVM 上で動作する他の言語からも使えるようになりました。
Scala の柔軟な言語仕様を生かした「ScalaFX」について、簡単なサンプルコードを交えて概要をお届けします。
