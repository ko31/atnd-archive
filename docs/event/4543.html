Tumblr ゥNishinippori meetup　どうやってやろうかな～とか思ってたんですが、変に仕切るのもお金かかるのも嫌なので・・・

6月19日（土）の昼12時～20日（日）始発くらいまで西日暮里フィレンツェずっと借りる！

って事だけ決めました。昼間のチャージはなしで、夜だけすこしチャージとると思います。よろしくお願いします。来たい時間に来てもらって、ダラダラいてもらえればと思います。

一応飽きないように、土曜昼はボドゲ会状態で、大量のボードゲームやカードゲームを置いておくので遊んでくだしあ。遊ばなくてもいいけど。

あと、レンタするかどうにかしてプロジェクターも置いておくので、お互いの dsbd 晒したりして男を誇示したりするのに使ってください。

そしてフィレンツェはインターネット無料ですので、ノーパソとかもってくるといいです。電源も当日は開放しますよー
