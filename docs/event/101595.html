第１章：Ruby によるプログラミングスキルの獲得

・そもそもRubyってなあに？
Ruby は ウェブサービス開発で最も多く使われている言語です。
少ない記述量でプログラムを組めるなどの特徴がありますが、
開発者が日本人なので日本語の参考資料が豊富にあるという点もオススメの理由です。

Ruby はインターネットの発展により Ruby on Rail というウェブサービスを提供する仕組みに進化しました。
開発効率が極めて高く、初心者の修得も容易なため、多くのIT企業が Ruby on Rail を利用してウェブサービスを提供しています。

なので！
Ruby をマスターする事でエンジニアとして活躍できるチャンスがグッと拡がるのです。
当スクールでは、最短最速で IT 業界での就業を可能にするために、Ruby 学習に特化した学習プランを提供しています。

・プログラミング体験レッスン
子供や初心者でもわかりやすい言語で指導、効率良く実践に役立つ学習ができる事が当スクール最大の魅力です。
苦手意識を持たれている方でも、正し学習方法で取り組めば、プログラミングスキルの修得は簡単であることを体験レッスンで実感頂けます。

・就労支援のご案内
エンジニア経験が無くても、獲得したプログラミングスキルを活かし就労できるシステムを準備しています。
就労までのサポートに加え、就労後もコミュニティサービス（通称ギルドサービス）を利用することで様々なサポートを得ることができます。初めての実務も安心です。

・キャリア相談
プログラミンスキルの修得は、IT業界での就労に向けた確実な一歩ですが、そこはまだ入り口にすぎません。
すでに年収1200万円を達成した先輩エンジニアたちが、今後のキャリアプランの相談に応じます。
業務上のどんな悩み事も解決します。（（講師には業務外の相談に強いエンジニアが在席しています）
