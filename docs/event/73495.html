アプリ界の皆様2016年が始まりますよ！

皆様いかがお過ごしでしょうか？
最初は「アプリ和尚」って呼んでね！って言っていたのにいつの間にか自分でも「アプリ」って付けるのを忘れてすっかり「和尚」としか呼ばれなくなった「くまんぼう」です。

「年末年始何にもイベントやらないの？」っていう無言のプレッシャーを（勝手に）感じたので新年会をやることにしました。

アプリ開発者の皆様、レビューサイトの皆様、アドネットワークの皆様、アプリ好きの皆様
とにかくアプリ関係の方でしたら何方でもご参加頂けます。
※ただし、あくまでも「楽しむ会」ですので営業目的で参加されるのはご遠慮ください。

詳細

  • 会場： プロント 池袋サンシャイン通り店 貸し切り
  • 日時：2016年1月22日 19時〜22時（18時40分より開場、受け付け開始　19時乾杯！！）
  • 会費：4500円（3時間飲み放題付きプレミアムパーティコース）

18時40分から受け付けを開始致します。
受付の際にお名前の確認と会費を集めさせて頂きますので、おつりの無いように4500円（ 二次会 も出られる方は8000円）ご用意頂けますすようにご協力お願い致します。m(_ _)m

ご注意

2016年1月8日に締め切ります。最大120名ほどの会になりそうですが、会場人数確定のため参加希望の方はお早めにお申し込み下さい。
キャンセルについては2016年1月15日まで承ります。
◆◆◆◆◆◆◆◆
1月15日以降のキャンセルや当日のドタキャンに関しては実費を請求させて頂く可能性がありますのでご注意下さい！m(_ _)m
