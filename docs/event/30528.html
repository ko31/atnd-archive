イベント概要

新バージョンのリリースが待たれるjQuery Mobileについて、
1.2から新しく追加される予定の機能や気になる今後の動向など、最新情報をまとめて解説します。
ライブコーディングやデモを交えつつ様々なTipsをご紹介。
jQuery Mobileの利用経験がある方に、より実務的なスキルを身に付けていただける内容を予定しています。

また、勉強会終了後は懇親会も開催しますので、講師の方や参加者との情報交換・交流を深めて下さい！
