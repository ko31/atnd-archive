description

　Information Technology（情報技術）の発展により、ビジネス環境が大きく変化しています。メディアを提供するビジネスでは、付加価値のある新たな取り組みが不可欠となっています。ビジネス戦略を理解し、複合的なメディア活用により、課題を解決できる人材が求められています。

　クロスメディアエキスパートとは、情報資源を複数の適切なメディアに展開することで、情報発信の相乗効果をもたらすことができる人材です。

　クロスメディアによる課題解決の過程では、ステークホルダーとなる様々な立場のビジネスパーソンと、コミュニケーションを図るシーンが数多く存在します。その要求に応えるには、コミュニケーションや経営、情報技術に関する幅広い知識を体系的に学習することが不可欠になります。

　本ガイダンスでは、著しい情報技術の進化や新しい社会需要への対応を踏まえ、クロスメディアエキスパート認証試験の主旨、エキスパートに求められる能力、必要な知識や経験、学習方法などを中心に解説を行ないます。
