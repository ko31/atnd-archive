セッション概要

Windows Phone 7.5 (Mango)ではSilverlightとXNAの両方を使ったアプリケーション開発が可能になっています。また、カメラや加速度計、コンパスなどのセンサーもアプリケーションから利用できます。本講演では、Windows Phone ARアプリケーション サンプル Mango Teapot を基に、カメラ・センサーおよび3Dグラフィックスを活用したAR的な表現方法について解説します。

セッション概要

Windows 8 Developer Preview版の新しいユーザーインターフェースとして登場した「Metro」。デスクトップ画面が用意されていることも確認できるが、Metroに対応したアプリケーション開発が近い将来主流になることは間違いない。Windows Phone 7のMetroは既にユーザーが利用しているものであり、また、それ以外のデバイスに目を向けてもアプリケーションにおけるデザイン的要素の重要性は高く、アプリケーション開発におけるユーザーインターフェースやデザインの重要性はますます高まってきていることは、疑いようがない。

今回は、アプリケーション開発のなかでユーザーインターフェースやデザインを
どうやって考えていくるのか、「Metro」を題材に検討していこう。

Windows Phone JSONクライアントの開発環境 構築手順

概要

Windows Phoneアプリケーションには通信は欠かせません。とはいえ、アプリケーション作成の際に、まずサービスを作らないと開発を始められないというのでは腰も重くなります。
しかし、WCFでJSONサービスを簡単に作成でき、そのクライアント作成もさほど負担が無ければ、アプリケーションロジックに専念できますね？
今回は、JSONにフォーカスして環境構築の流れをお話しします。
