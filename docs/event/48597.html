[2014/3/26追記]
さくらのイベントでおなじみの「さくらのクラウド」2万円分クーポンチケット＆桜葉愛グッズを、来場された方全員にプレゼントいたします！

昨年に続き、今年もさくらの夕べが福岡に再上陸！
福岡と言えば「もつ鍋、ラーメン、さくらの夕べ」と言われるようにがんばります！

今回は、動画配信サービス「Gemediar」やオンラインビデオチャット「1meeting」を提供する、
グルー株式会社様にご登壇いただきます。

さくらのナレッジでも寄稿していただき（詳しくはコチラ）、今、地元で注目のスタートアップ、グルー様が
さくらの夕べにご登場！今回は「1meeting」のサーバインフラについて深くお話をお聞かせいただきます。

また、弊社代表の田中から、仮想サーバ作成ツール「Docker」（←便利！）をテーマに
ウェブサービスの運用方法をお話いたします。

懇親会では、スクリーンを開放していますので、ネタのある方は自由にお使いください。

ご友人・ご同僚をお誘いの上、ぜひ遊びにいらしてください！
