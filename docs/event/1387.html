テーマはズバリ

**********************************「Web制作の近未来像」～HTML5の登場からコンテンツまで～**********************************

菊池　崇（きくち さとし）さん

講演タイトル
「HTML5 Loading」

講演概要

最近、注目を浴びてきたHTML5、しかし自身でマークアップをしたことがある人は少ないのかもしれない。

このセッションは、最初につまづきがちなFlow ContentなどのContentモデルについて、section要素、article要素、header要素、hgroup要素、footer要素、aside要素、などのHTML5の新しい要素にフォーカス。

JavascriptをつかったHTML5未対応のブラウザへの実装方方法。最新のCSS３やマイクロフォーマットとの組み合わせを紹介。

**********************************

長谷川 恭久（はせがわ やすひさ）さん

講演タイトル
「コンテンツ戦略を意識したデザインアプローチ」

講演概要

コンテンツは Web サイト制作においてマーケティング、ビジュアル、機能など 様々な分野に影響を及ぼす重要な要素です。

当たり前の要素として存在 しているコンテンツですが、わたしたちは Web の特性を活かしてコンテンツを 企画・開発しているでしょうか。

コンテンツがサイトデザインと密接な関係を 保てているでしょうか。

今セッションでは Web デザインとコンテンツ戦略の関連性とアプローチに ついて紹介。また、あるサイトを題材にコンテンツを意識してリニューアルを するなら何が出来るかを模索していきます。

**********************************
