概要

「久留米プログラミングフリークス　スマートフォン分科会」とは、スマートフォンやタブレットPC上で動作するアプリケーションの開発や、それらの利用についての情報交換を目的とする、「久留米プログラミングフリークス」を母体としたコミュニティです。

日頃からアプリケーションを開発されている方から、プログラミングなんてした事もないけれどもやってみたい方、こんなアプリケーションがあったら便利なのに……というようにアイデアをお持ちの方、その他スマートフォンに興味をお持ちの方を募集しています。

iPhone, Android, Windows Phone, iPad など、機種は問いません。

今回は、実際に Android 開発環境を用意し、 Android 開発とはどんなものかを体験頂けるような内容になる予定です。
本当に Android 開発をやったことがない方向けの内容になっています。
手順に沿って全員で進めていきますので、プログラミングをされた事がない方でもご参加いただけます。
キーボードの入力がそれなりに行えるならばOKです。

必要なもの

今回は、可能であれば各自自分のノートパソコンに予め開発環境を用意して、お持ち頂きたいと考えています。

多少手順は多いですが、難易度はそこまで高くありませんので、是非挑戦されてみて下さい。
以下に参考にできるサイトのリンクを置いておきます。

しつこいようですが、開発環境は可能な限りご用意いただけると助かります。

Windows の方 → Xperia（エクスペリア）口コミ体験談 様
Mac OS Xの方 → kitakatsuLab 様 (4ページに渡っておりますので、ご注意下さい)

参加費 : 500円。
