セミナー概要

プレゼン資料作成やWEB制作、メディア運営等、様々な用途に欠かせない画像素材、皆さんはどのように活用していますか？

情報を的確に伝えるうえでビジュアルコンテンツが大きな影響力を持つ今、インターネットなどで容易に画像を取得・購入できるようになったこともあり、業界・職種を問わず、様々な用途で画像を利用されているかと思います。

しかし一方で、実は画像に関するトラブルが増えてきていることをご存知でしょうか？
正しい知識がないために、ある日突然トラブルに遭遇してしまった…！ということも有り得るのです。

本セミナーでは、安心して画像をご利用いただけるよう、画像に関する権利の知識や、権利に違反することなく正しく画像をご利用いただくための注意点などをご説明いたします。
普段画像利用に関して疑問に思っていることや、「こんな使い方は大丈夫？」といったご質問にもお答えいたします。

現在画像素材をご利用になっている方、また、これから画像素材の利用を検討されている方、ぜひご参加ください！

こんな方へおすすめ

  • 業務で画像素材を使用される方
  • 画像素材の購入・管理をされている方
  • これから画像素材を使いたいと考えている方
    ※他社様のストックフォト素材をご利用の方でも、お気軽にご参加ください。

プログラム

  • 画像トラブルの事例紹介
  • 押さえておきたい画像にまつわる３つの権利
  • ストックフォトのライセンスの違いって？
  • 正しく画像素材を使うために
  • 質疑応答
