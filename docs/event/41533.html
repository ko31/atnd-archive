description

KORGのアナログ・シンセサイザー「volca」を持ち寄って、back 2 backするイベントです。会場で流れる音はvolcaのみ！演奏するは参加者のみなさん！机にミキサーだけを置いてお待ちしております！
korg volca
http://www.korg.co.jp/Product/Dance/volca/
＊リクエストにお応えして持ってない人でも参加OKです！しかしback 2 backには参加してもらいます！「いきなり出来ない!!!」と思ってもご安心ください、僕がキッチリ教えます！

会費
1,000 – 2,000円の予定、酒類の持ち込み可能です。酒類500円、ソフトドリンク200円程度で販売しますので手ぶらでもOKです！

会場
場所は渋谷の空にある隠れ家。土足厳禁＆全面禁煙のクリーンな空間です。お手数をかけますが、喫煙者の方は会場外で喫煙スペースでお願いします。

ルール
1. 入場時にカードに名前を書いてもらいます。カードを引いてback 2 backの順番を決めます。
2. プレイしていて、もう限界だな／もう十分だなと思ったら、カードを引いて次の人を決めます。
3. SyncのClockは同様のテンポのものを２つ用意します。テンポを変えたい時は演奏中に上手く変えてください。

お願い
＊複数台使用可能です。みなさんがvolcaだけ持ってくれば大丈夫なように、多チャンネルのミキサーを準備します。４台以上使う場合は連絡ください。
＊電源アダプタは使用せずに電池駆動するようにしておいてください。
＊当日は演奏の動画撮影をします。僕が個人的にYoutubeなどにアップすることをご了承ください。顔を積極的に映すことはしません。顔出し絶対にNGの方は事前に申し付けください。
