description

分散ログ収集基盤 ”Apache Flume” の運用やカスタマイズの基礎について、勉強会を開催します。

Flumeに興味はあるけど、詳しく調べる時間がなかなか取れないかたや、試してみたいけど、インストールが面倒そう・・・。そういった方々向けに基本的なセットアップ方法や、プラグイン拡張の基本的な実装方法など、Flumeに初めて触れる際に必要となる内容を中心にご紹介します。ぜひ、ご参加ください！

なぜ分散ログ収集が重要なのか？、Flumeの動作アーキテクチャや、内部の主要なモジュール構成・クラス構成、など。Apache ログ集計の設定例など。

２．Flumeプラグインカスタマイズ入門

Flumeプラグインを自作するための開発環境や、実際のプラグイン例を交えて紹介します。
