会場および集合場所

  • 個人住宅(一軒家)が会場となります。相鉄線上星川駅 改札前に集合。
  • 集まり次第、テクテク歩いて会場に向かいます。
    （※重量物持参者はバス推薦。横浜西口９番のりば「釜台住宅第三／上星川駅」行。「釜台住宅第三」で下車。）

注意点

  • 基本的な内容は自宅ラック勉強会#3に準じます。
  • 開催者はネットワーク機器を多く持っていません。ハイスペックなものなど縁がないです。可能であれば各自BGPをはじめとした色んなプロトコルを喋れる機器を持ち寄っていただけると素敵なネットワークを構築できるかもしれません。（RTX1000とかRT57iとかRTA55iあたりと6本足のSEIL/x86を持っています。）
  • 開催者は自称"永年アマチュア"なので相応の技術力しかありませんが、代わりに好奇心とやる気があります！
  • 開催地には冷蔵庫に加えターキーが丸ごと焼ける大きさのコンベックオーブンもありますが台所付近は立入禁止です。そしてもちろん（22Uながら）ラックがあります！
  • お菓子は開催者にあまりセンスがないので期待しないでください。たぶん母親に泣きついて解決すると思います。
  • 自宅にラックが無くてもご参加いただけます。
  • なお、お手持ちのラックの持ち込みはフルラックでなければ可能かと思われますが、（駐車場設置を前提に）事前にご相談ください。
  • リモート拠点なので本部とUstなどで中継できると良いなぁと考えています。しかし開催者宅には必要機材が不足していますので、できればどなたかに持ち込んでいただけると大変助かります（詳細は後日）。
  • 参加者各位は、できれば5mぐらいのLANケーブルを必要本数ご持参ください。接続はノンインテリのGbEなHUBで計15ポートの用意ができます。
