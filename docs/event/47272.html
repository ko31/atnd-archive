「フロントとバックエンドの連携的なとこ知りたいんです」

ゼロからはじめるサーバサイド勉強会 #2 やります！

  • HTML / CSSを理解している
  • JSをコピペで動かせる（JSのコードを見たことある・なんとなく読める）
  • フロントエンドエンジニアになりたい

今回は、サーバサイド #2として前回お話できなかったWebAPIに関する講義をしていただきます。

▼zerostudyについて

「ゼロからはじめる～～勉強会」として、webに関する色々を、改めて基礎からみんなで勉強しよう！という企画です。
気付き・発見・復習、のなにかきっかけになれるような勉強会を目指しています。

講義は、既にwebに関わっている方、興味がある方を対象に、初心者/中級者向けの内容です。学生さんも大歓迎。
（上級者な方、お話をすることで勉強したい方は、ぜひお声かけください！）

「基礎を蓄えてもっと良いクリエイティブができるようになりたい主催者」が、みなさんとがんばっていきたいと思います(｀・ω・´)
