Adobe GAMING 国内初セミナー
スマートフォン向けAIRアプリ開発の裏側をお話します。

Adobeゲーム開発に興味ある方集合！

2012年12月Adobe Game Developer Toolsが発表となりました。
これを受け、アドビ システムズ 株式会社からのAdobe Game Developer Tools概要の説明と、スマートフォン向けAIRアプリ制作セミナーを行います。

2012年の4月にリリースされたスマートフォン向けAIRソーシャルゲームアプリケーションの製作の裏側と開発ノウ・ハウをご説明致します。

リッチ系ゲームにご興味あるかたは是非ご参加ください。

予定プログラム

  1. 開場挨拶
  2. アドビ AIRの概要説明
  3. AIRソーシャルゲームアプリケーション制作事例
  4. Adobe Game Developer Toolsについて　http://www.adobe.com/go/gaming_jp
  5. 各ソーシャルゲームプラットフォームのAIR対応状況
  6. オートクチュールの説明
  7. 開発技術者募集説明
  8. 質疑応答
  9. 懇談会（お酒とお菓子）

参加対象者

  • AIRを用いて開発を行なっているエンジニアの方
  • AIR開発に興味が有る方
  • これからiOS/Androidアプリ開発を始めようと思っているエンジニアの方
