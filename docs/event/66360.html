第１回　医療療法科学学会シンポジウム

世の中様々な医療や療法があります。
各分野でご活躍の方々に、ご自身の考え方や取組みをご紹介して頂きます。
医学・美容健康・物理学・経営学・農学における各様々な分野から多角的な視点で捉えたものです。
あのＣＤやアイボを開発した天外伺朗さんや、多数のメディアで登場する美容家の岸紅子さんらにご講演をいただきます。
最後は、各講演者全員によるパネルディスカッションも行いますよ。

　＜講演内容と講演者＞　司会　人見ルミ

　■未来医療創生プロジェクト「未来医療はリセット医療」 〜観点と関係性の革命〜
　　　新板橋クリニック院長清水公一
　■ホリスティックビューティの取組みについて
　　　日本ホリスティックビューティ協会理事長岸紅子
　■物理学から観る医療　声による特性の可視化とリトリート
　　　音色香の空間創造代表理事柊木匠
　■ホロトピック・センター：病院に代わる新しい概念
　　　ホロトロピックネットワーク代表天外伺朗
　■動物介在療法から考える福祉と農業の関わり
　　　東京農大バイオセラピー学科川嶋舟
