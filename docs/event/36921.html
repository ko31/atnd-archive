テクノロジー×野菜栽培の話題が好評だったのでもう一回やります

皆さん、野菜食べていますか？
あなたが食べている野菜は当然どこかで栽培された野菜です。
そしてその背後には肥料から収穫まで、様々なテクノロジーが隠れているようです。

テクノロジー×野菜栽培をテーマに12月に勉強会を開催しましたが、好評につき再び開催します。
自宅で楽しめる家庭菜園から大規模農業まで、テクノロジーで野菜栽培をエンハンスドさせる仕組みについて、自宅のバジルを自慢しつつ実際の野菜工場の事例を挙げつつ、のんびり意見交換していきましょう。

場所は西麻布のNOMAD NEW’S BASE！

・「野菜栽培とテクノロジー」というキーワードにグッと来る人
・すでに野菜栽培とフィジカルコンピューティングを組み合わせている人
・面白そうな物を見物したい人
まで、午後の柔らかな日差しを浴びながらゆるやかに語りあいましょう。
