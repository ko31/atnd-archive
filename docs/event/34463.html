description

日本Androidの会　秋葉原支部　ロボット部　第9回　勉強会を開催します

概要

  • 日時 2012/12/9 13:00-17:30
  • 場所：Robotma.com（ろぼとまどっとこむ）
  • 費用：ろぼとまの使用料500円＋板金工作希望の方はアルミ板代500円／枚（１枚で足ります）
  • 必要なもの:別途下記にまとめがあります。
  • 工具：　会場にある工具を使用できますが、使い慣れた道具があればお持ち込もOKです。

プログラム

  • オープニング
  • 自己紹介
  • メインセッション

（以下の仮プログラムで調整中）

１）ウッチーの板金の基本講座

２）店長のろぼとまにおいてある板金工具の使い方
タップ、折り曲げ機、切断機、３Ｄプリンタ、アルミ切削機etc．．．

３）ＴＡＫＡの、「家でちょっと板金をするときにあると便利な工具：私はこんな感じでやってます」

  • なにか作るタイム

作るものを固定すると汎用性が無いのと、4軸ロボなどですと時間が足りないかもしれないので、今回は、「自分で用意したラジコンサーボのブラケット（関節パーツ）を作ってみる」ということで考えています。ブラケットは汎用性のあるデザインを当日紹介しますので、自分のサーボ用にサーボの採寸～板金まで、を体験します。

作業内容（２～３時間）
１）サーボ採寸（ノギス２，３本はこちらで用意します）
２）構造検討
３）製図（紙に書くorPCでプリント）
４）アルミ板に図を貼る
５）切断
６）穴あけ
７）曲げ加工
８）タップ切り
９）ヤスリがけ

必要なもの

  • 最低必要

紙、えんぴつ（細い線が書けるもの）、三角定規、のり（普通の工作用）

  • できれば持ってくる

PC＋なにかしらのCADソフト、サーボ、ノギス（できればデジタル式）、電源タップ

サーボは持っていなければ、当日ろぼとまで購入できます。安いもので結構ですが、その後自分でなにかロボを作ろうと思っている人は、それなりのサーボを用意してください。
CADをお持ちでない方は、「鍋CAD」がいいかもしれません。
（フリー版でOK）
http://www.vector.co.jp/soft/dl/winnt/business/se477968.html

鍋CADはロボワンビルダーなどでよく使われているようですが、詳しい人のアテが無いので初心者の方は少し練習してきてください。

懇親会

近くの居酒屋で調整中！
