PHPの 『 超 』 初心者向け勉強会です。

PHP＋MySQL勉強会は、基礎編と応用編があります。

こちらは基礎編になります。

MySQLをはじめて勉強する方に向けた内容になります。

難易度は 『 全く難しくない 』 です。

対象者は 『 プログラム未経験者 』 です。

PHP初心者勉強会と中級者勉強会は定期的に開催しています。

定期的に開催しているPHP初心者勉強会は、PHPは未経験でも他のプログラムの経験がある人を対象としています。（C言語、JAVA、Perlなど）

そのため、初めてプログラムを学習する方は、内容を理解することができませんでした。

PHP 『 超 』 初心者勉強会は、プログラムが初めての方を対象にしています。

勉強会の進め方は、プロジェクターに映した画面を見ながら、一緒に操作を行って進めて行きます。

全体のスケジュール

  1. インターネットの仕組みについて
  2. Webプログラミングの概要
  3. XAMPPについて
  4. MySQLというデータベースの使い方について勉強します
  5. MySQLの基礎を勉強します
  6. XAMPPというソフトについてるphpMyAdminというソフトを使います
  7. データベースとテーブルとレコードの関係について
  8. レコードの操作（INSERT,SELECT,UPDATE,DELETE）
  9. 複雑なSQL文

上記の内容を４回に分けて行います。

ご案内

  1. 時間は午後7時から午後9時までの2時間です
  2. 参加費が2000円かかります
  3. ノートパソコンをお持ち下さい。（なくても参加できます）
    ※貸し出しできるノートパソコンが１台だけあります。
  4. インターネットに接続する環境はありません。
  5. 飲食物の持ち込みはできません（レストラン併設のため）

講義にはXAMPPというソフトウェアを使います。
これはパソコン上（Windows,Mac）で、Webサーバーを実行するプログラムです。
（基本的にインターネットで使用されるWebサーバーと同一のプログラムです）

Windows版
http://www.apachefriends.org/jp/xampp-windows.html

Mac OS X版
http://www.apachefriends.org/jp/xampp-macosx.html

XAMPPのインストール方法は、以下のURLをご覧下さい。
http://www.db.is.kyushu-u.ac.jp/rinkou/mysql/xamppinstall.html

インストールができなかった方は、インストールを手伝いますので、少し早めに会場に来て下さい。
