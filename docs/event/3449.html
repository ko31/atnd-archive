■日本Androidの会 中国支部 第８回勉強会のご案内

　国内でもソニーより、Android携帯となるXperiaが発表され、
　技術者の方々からの注目度も高まってきております。
　新しくAndroidの開発をしてみたい！
　だけど、躓いてしまった・・・そんな人に向け、
　今回はハンズオン形式orコードラボ形式で
　Androidアプリ開発の基礎部分を身につける勉強会を開催します。

　今回の題材としましては、昨年末より「Windows 100%」上で執筆を行っております、
　「Androidプログラミング講座」のサンプルアプリを題材とし、進めていきたいと思います。

　雑誌を購入されて躓かれた方も、始めてAndroidに触れられる方にも分かりやすく
　説明させて頂きたいと思いますので、皆様のご参加を心よりお待ちしております。

　　日本Androidの会　幹事　中国支部長　重村浩二
