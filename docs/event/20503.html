準備

Titanium Studio

Titanium Mobileによる開発を行うために必要な総合開発環境(IDE)です。
以下の手順でインストールしてください。

  1. http://www.appcelerator.com/ にアクセス
  2. 「Download Titanium」からTitanium Studioをダウンロード（要アカウント登録）
  3. 上記でダウンロードしたファイルを実行してインストール

※Titanium Studioを起動するにはインターネット接続が必要になります。

Android SDK

Titanium Mobileはアプリをビルドする際にAndroid SDKを利用しますので、以下の手順で導入してください。

  1. http://developer.android.com/sdk/index.html にアクセス
  2. ご利用のプラットフォームに合わせてAndroid SDKをダウンロード
    （Windowsをご利用の場合はexeファイルのインストーラを推奨します）
  3. 上記でダウンロードしたファイルを実行してインストール
    • Macの場合は任意のディレクトリに配置してください
  4. SDK Managerから各バージョンのSDKをダウンロードしておいてください
