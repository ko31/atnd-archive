description ITのチカラをキミに、こどもに。―――“Idea Makers Project”

何かを創りだすとき、ITが欠かせない時代になりました。世界で注目されるFaceboo、Amazon、LINE・・・数えきれないサービスが、ITのチカラでできています。海外ではイギリスやオーストラリアをはじめ、ITサービスをつくりだす「プログラミング」技術を義務教育化する流れがでています。日本でも2019年、小学校で必修化が計画されています。
そんななか、tdiはITのプロとして、プログラミングを小中学生に教えるプロジェクトをはじめます。
そして、いっしょにプログラミングを進めてくれる学生を募集します。プログラミング初心者でも大歓迎。
これからの時代に必要なITを、こどもに教えることを通じて、ステップアップしてみませんか?

３つのイイこと

“Learn！”　こどもにITを教えられるようになるため、こども向けの授業を基にレクチャーを実施。ITと授業の進め方を学びます。自身のITスキルアップのチャンスです。
“Work!”　学んだことを実際にこども向けの授業で実践します。働いたらその分報酬もあります。
“Enjoy!” 私たちの想像をはるかに超えるこどもの成長を見れる楽しみがあります。そしてその成長が自分の成長にも繋がります。
