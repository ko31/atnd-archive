イベント概要

jus共催 第43回大暴れシェル芸勉強会 へ、大阪から YouTube と Twitter を使ってリモート参加します。 UNIX シェルを使って、ワンライナーでテキスト処理などの問題を解いていく、ハンズオン形式の勉強会です。
難易度は入門レベル～難しいものまであり、初心者・玄人の区別なく？参加していただけます。

午前の部

今回もないよ。残念！

午後の部 : シェル芸勉強会

シェル芸勉強会は、小さい問題をワンライナーで解いて、コマンドの基本的な使い方からマニアックなオプションまで知って使いこなすようにシェル力をつける勉強会です。

難しそう……というイメージがあるかもしれませんが、問題が解けなくても、解答をなぞるだけでも勉強になると思います。

勉強会で使うデータは、GitHub で公開される予定なので、Git でデータを取得できるようにしておいてください。

会場について

今回は、大阪工業大学 枚方キャンパス の教室をお借りして開催します。

いつもの会場とは場所が異なるため、ご注意ください。

アクセス方法 を確認の上、余裕をもってお越しンス。

12:45分ごろを目処に、 1号館1階のエントランス にお集まりください。集まりましたら、スタッフが引率し、会場に案内します。

遅れて来られる方はスタッフが迎えに行きますので、到着後に @3socha までご連絡ください。 会場をお借りしている大阪工業大学さんのご迷惑になりますので、必ず 1号館1階のエントランスでお待ちください。

持ち物

  • UNIX環境のあるノートPC（Mac・Windowsでも可）

※Windowsで参加している人もたくさんいますのでご安心を。
(Cygwin、MSYS2、Git bash、Bash on Windows、Chromebookなど…)

※会場に WiFi 環境がないため、レンタルのモバイル WiFi ルーターを手配する予定ですが、通信が不安な人は自身で用意していただいた方が良さそうです。

※会場のスクリーンの入力端子がVGAしかないため、LT大会でPCの画面を映したいときは変換コネクタをご持参ください。
