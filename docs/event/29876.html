【増席10名】Webプログラミングスタートアップセミナー【座学 + ハンズオン】

■セミナー内容

プログラミング初歩からWebエンジニア必須のスキルについて学び、今業界で求められている人材についても解説いたします。ハンズオンではHTML5を通しプログラミングの仕組みと楽しさを体感していただきます！！

  • プログラミングとは？
  • プログラミング言語の種類
  • 現在使われているWebアプリケーションの解説
  • 今後身につけておくべきスキル
  • 実際にプログラミングしましょう（ハンズオン）

※ご自身のノートPCをご利用の方はChromeかSafariをインストールしてお持ちください。
※会場にも用意はありますが、数に限りがありますのでご了承ください。

こちらのセミナーも是非ご参加ください！
Webデザイナー・HTMLコーダーのためのPHPスタートアップセミナー～PHPでできること～

■講師紹介

松本 雄貴 氏

フリーのシステムエンジニア・プログラマー。専門はWebアプリケーションだが、現在東京デザインテクノロジーセンター専門学校でAndroid、JAVA、システム工程管理の講師を務める。JavaScript 、PHP、SQL、Java、Perlなどのプログラミング系から、ネットワーク系（CCNA）、サーバー系（Linux）にも幅広く対応可能なマルチ講師。
株式会社マルチメディアスクール・ウェーヴ特別講師
