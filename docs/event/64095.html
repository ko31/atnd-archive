WEBマーケティングは本当に進化しているのでしょうか？新しいツールなどのテクノロジーは発達し、WEBユーザーもマーケット規模が拡大している中で、本当に事業者はWEB事業の機会をモノにできているのでしょうか？結局事業は根底にあるマーケティング戦略や実行するチカラが組織に備わっていなければ成果を得られないのではないでしょうか？

実行力のあるWEB組織とは？そんな基本に戻ったテーマを掘り下げ、参加者の方々とともに具体的な事例を暴露しながら本当のWebマーケティング組織について考えるトークセッションを開催します。

Dr.シーラボ、オイシックスでCMOとして実践し、多数のコンサルを手がける西井氏を招き、WEBサービスベンダーのi3DESIGN芝氏とUNCOVER TRUTH小畑氏の３者コラボセッションで解き明かします！

「予算なし！リソースなし！でもうまくいく！オイシックスに学ぶ　実行力のあるWEB組織」
この表題にはそんな思いが込められています。実際にWEBマーケティングの現場で日々奮闘されている方々、WEB組織の責任者の方々、皆さんに聞いてもらいたい真実を突き詰めるセッションにしたいと思っています。皆さま、ふるってご応募ください！
