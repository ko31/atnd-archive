description

趣　旨

東日本大震災発生から半年余りが経ちました。この間、同志社大学からも多くの学生・教職員が自主的・自発的に関わってきた被災者支援・ボランティア活動も、一定の振り返りや評価、今後を見据えた展望が必要な時期を迎えています。一方、被災地内外でボランティアとして活動し、キャンパスに戻ったみなさんの中には、活動を通じた見聞や体験が大きく重い記憶として刻まれており、中には人知れず重荷や衝撃に悩む人たちもいると聞きます。

そのような課題を受けて、さまざまなかたちで被災者や被災地に関わった学内のボランティア経験者、それらの活動に関心のある人たちが共に集い、お互いの体験や感じたことを語り合い、聴き合うことで、参加者にボランティア活動を通じて得たさまざまなものを自分なりに受けとめ、今後の糧とするきっかけにしてもらうとともに、被災者や被災地と関わるということ、つながるとはどのようなことなのか、今後についても考える機会としたいと思います。

概　要

日　時：2011年10月15日(土) 13:30〜15:40 (開場13:00)

会　場：同志社大学今出川キャンパス クラーク館２階 クラーク・チャペル

＝申込不要・参加費不要＝

対　象：被災者支援やボランティアに参加／関心のある、同志社の学生・教職員のみなさん

本企画は、同じ学園で学びや研究を共にする仲間同士の寛いだ関係性、雰囲気の中での語り合い、分かち合いを目的としています。そのため、他大学および一般のみなさんのご参加は、原則として受け入れておりません。ご理解くださいますよう、お願い申し上げます。

後　援：同志社大学キリスト教文化センター

協　力：同志社大学学生支援センター　同志社大学カウンセリングセンター

主　催：東日本大震災ボランティア体験を分かち合う集い 実行委員会
