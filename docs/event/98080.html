BLT - Buddhist Lightning Talk

日本初の仏教系LTを開催します！

BLTは、おひとりにつき5～10分ほど、仏教について語るイベントです。見学だけのご参加でも構いません。

仏教に関することならテーマは自由です。例えば「おススメ瞑想法・坐禅法」「私の推しの宗派」「私と仏教との出会い」などなど。

瞑想を始めたばかりの方、仏教に興味のある方、いろんな業種の方のご参加をお待ちしております。

発表方法も自由です。フリーWi-Fiとプロジェクターとホワイトボードはありますが、喋りだけでも構いません。

当日の飛び入り参加も歓迎します！

注意点

開場は13:00で、開始は13:30です。

飲食可です。暑いので飲み物は持参していただけると幸いです。おやつなども持ち込んで頂いて構いません。

発表方法について

発表される方は、コメントに発表されるタイトルと概要を書いて頂けると嬉しいです。当日変更されても構いませんし、気が変わって発表しなくなっても構いません。

当日の飛び込み発表もアリです！

プロジェクターを使われる方は、スライドを予めウェブ上に上げて頂くか、メールでお送りください。アドレスは osanaga@gmail.com です。
