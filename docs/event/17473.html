技術ひろば.net 2011年7月 勉強会 – 中止

都合により、技術ひろば.net 2011年7月の勉強会は中止させていただくこととなりました。
参加を検討していただいた方にはお詫び申し上げます。

以下の要領で「技術ひろば.net 2011年7月 勉強会」を開催します。
「技術を楽しむ、技術でつながる」の理念の元、人と技術を大切にする活動を行っています。
どなたでもお気軽にご参加ください。

開催概要

■日時 : 2011年7月17日(日) 13:00～17:30
■会場 : 首都圏コンピュータ技術者株式会社 2F 大会議室
　http://mcea.jp/tokyo#maptitle
■会費 : 無料（勉強会後の懇親会は別途有料）

内容

7月の技術ひろば.net は LightSwitch 特集です。
7月26日 (時差のために日本では 27日？) に正式リリースが決まった LightSwitch について考える機会にしてみませんか。

簡単に業務アプリケーションが作れると言われる LightSwitch ですが、馴染みのない方はそれでも不安に思うかもしれません。基本的な操作から説明しますので、未経験の方や「LightSwitch って何？」 という方も安心してご参加ください。
LightSwitchと SharePoint Server との組み合わせ、@SQLAzureJP さんによる LightSwitch × SQL Azure で、一歩進んだ活用についても紹介します。

いつものように お茶会（フリートーク） もやります。お茶でも飲みながら、おしゃべりや情報交換をしましょう。

※LightSwitch については 7月30日(土) に別の勉強会(http://sqlazure.jp/r/ls01/) も予定されています。ハンズオンも予定されており、技術ひろばとは別の切り口で取り上げていただけます。そちらへの参加もご検討ください。
