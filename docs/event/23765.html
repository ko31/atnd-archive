Android開発者が参加者をサポートします。

参加費：
　1,000円（お菓子と飲み物を用意します！）

前提スキル：
　・プログラミングの経験のある方。（JAVAでなくてもOK,独学可）

参加者が持参する物：
　・名刺（無くても可）
　・ノートPC（できればEclipseとAndroidSDKをインストールしてきてください）
　・もっている方はアンドロイド端末
　※１、希望者にはノートPC貸出（数台）
　※２、Eclipse、AndroidSDKのインストールもサポートします。

開催条件：
　・最低3人以上参加が在る場合開催します。
　　参加者が3人未満の場合は中止します。

予定：
　1.Eclipseの基本操作方法説明
　2.プロジェクトの作成
　3.レイアウトエディタを用いた画面作成
　4.ボタンアクションの作成
　5.画面遷移
　6.じゃんけんアプリの作成

ソースコードを見ながらの技術紹介
　・SQLiteを用いたDBアクセス
　・タッチセンサーを用いたマルチタッチの実装
　・アプリケーション公開時に行う、難読化の注意点
