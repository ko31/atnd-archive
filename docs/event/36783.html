catch 横浜へなちょこプログラミング勉強会
description

共通のお題でみんなでプログラムを書きましょうというイベントです。
お題は主催者（鍋谷）が用意します。
言語は自由です。今のところ ruby, java, perl, Haskell が多いようですが、Prolog, Gauche, Erlang なども大歓迎です。

難易度や分量は、前回、前々回と同じぐらい簡単にするよう心がけます。

22時近くに終わりますが、その後、希望者で飲みに行きましょう。

開催情報

参加費 1000円です。タネマキ( http://tane-maki.net/ ) の利用料金です。
必要なもの プログラムを書くための道具。無線LANのついたノートパソコンがお勧めです。
あと、筆記用具とノートがあると役に立つかもしれません。
飲食 会場でコーヒーなどが飲めます。
電源 会場の電源事情が充実しているので、電源タップも不要です。
ネットワーク環境 会場の無線LANが使えます。

タイムテーブル

19:00 – 19:15 受付・準備
19:15 – 19:30 自己紹介
19:30 – 19:40 問題の説明
19:40 – 20:40 みんなで問題を解く時間。解けた方は休憩。
20:50 – 21:40 各自のプログラムの解説。ひとり5分程度。

→ その後飲みに行きましょう。

補足

主催者がこの手のイベントの開催に不慣れなので、いろいろ大目に見てくださると助かります。
特に初参加の方は過去問を見ておくと（できれば解いておくと）良いと思います。過去問は下記の表からたどれます。
会場は前々回とは異なり、タネマキ です。ご注意下さい。
