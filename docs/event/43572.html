実習機材紹介

RaspberryPi … Linuxの動作する教育用格安ボード。西暦2000年のパソコンと同じぐらいの能力があります。裸のボードにディスプレイ、キーボード、LANなどがつながるようになっています。
RaspberryHabu …RaspberryPiに接続するアドオンボードです。RaspberryPiはいろんな機器が接続できるようになってますが、扱いが難しく壊れやすい問題があります。このアドオンボードをつなげることで、簡単に、間違っても壊れないように実験ができるようになります。

工具類は会場側で用意しますが持ち込みもOKです。
RaspberryPiについての相談などもご遠慮なくどうぞ！

実際の作成手順

「RaspberryHabuとは」 バージョン20130710,20130827組立解説

ご用意いただくもの

  • 自分のPCをお持ちください。Windows/Mac/LinuxいずれでもOKです。
  • Raspberry Pi。もしお持ちでない方は当日購入できます。購入希望の方は申込時のアンケートフォームに記入ください。
  • SDカード。Raspberry Pi用のLinuxをあらかじめインストールしてください。もしお持ちでない方はインストール済みのSDカードを当日購入できます。購入希望の方は申込時のアンケートフォームに記入ください。
  • <補足> アンケートフォームが正常に動作していないようです。
    Raspberry Pi および SDカード希望の方は会場でお申し出ください。
    Raspberry Pi (Type B) は￥3600, SDカード(4GByte,イメージ書き込み済み)は￥600です。
    また、ご希望の方はRaspberryHabu First→Basicアップグレードキット（追加￥1000）も当日購入できます。
