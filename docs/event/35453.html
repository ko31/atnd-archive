概要

Unityを使ったゲーム製作の初心者(作ったことが無い人も含む)が、簡単なゲームを作れるようにステップアップする為の勉強会です。

  • Unityでゲームを作ったことが無いなら今すぐ作りましょう
  • ゲーム制作の知識が無いなら今すぐ勉強しましょう
  • global game jamに出たいならUnityのゲームの作り方を勉強してとりあえず参加しましょう

ギークハウス武蔵小杉を global game jam の会場の1つにしようと思っています。Unityの良さをみんなに伝えて、簡単なゲームでもいいので作って参加できると理想です。とりあえずやってみませんか？

詳細

  • 【日時】1/14 13:00〜18:00(懇親会 18:00〜)
  • 【場所】 ギークハウス武蔵小杉 (道順は こちら を参照)
  • 【開場】12:30頃開場
  • 【参加費】500円
  • 【対象】Unity初心者やギークハウスに興味のあるUnity開発者
  • 【その他】Unity4.0を予め ダウンロード してインストールしておいてください

内容

0. イントロダクション(13:00〜13:30)

この勉強会についての説明、この家の設備の説明、参加者の簡単な自己紹介など。

1. Unityのチュートリアル(仮) ( @mi_kami ) 13:30〜15:00くらい

基本的な操作方法とゲームの作り方を学びます。
こちらの資料 を使ってチュートリアルを進めていきます。
Unityマスターブック あたりを参考図書として持っておくと便利かもしれません。

2. もくもくゲーム製作(15:00〜17:00)

自由にゲームを作りましょう。
特にゲームが思いつかなければ幾つかテーマをこちらで用意しておきます。
分からないことがあれば教え合いながらやっていく感じで。

3. 結果発表

実際に作ったゲームや今日の作業についてみんなで発表しあって、分からないことを質問したり、意見交換したりしましょう。

4. 懇親会とLT(18:00〜) ※希望者のみ

ギークハウス武蔵小杉でお酒+鍋を用意しておきます。
お酒飲んだりご飯食べて親睦を深めつつLTしましょう。
(LTは人集まらなそうなのでやらないかも…)

※発表者募集中

講師枠(1〜2人)

1人30分〜1時間(応相談)で、Unityのゲーム制作の講師(発表)をやってくれる人を募集します。

  • XXを作る為のチュートリアル
  • Unityでゲームを作る際のTips
  • 他、テーマは何でもOKです(一応初心者向けで)

LT枠(1〜5人)

1人5分〜10分くらいの持ち時間で、Unityに限らずゲーム制作に関連する内容なら何でもOKです。
例えば…

  • Unityの技術的な発表
  • こんなゲームを作ったよ or 作りたい
  • 最近面白かったゲームについて
  • レベルデザイン
  • Unity以外のゲームエンジンについて(Unrealとか, cocos-2dとか)

その他

  • 開始時間より少し早めの時間に来てもらえれば環境構築のサポート等やります(ギークハウスの案内や雑談等も)
  • 後日facebookグループを作ります
