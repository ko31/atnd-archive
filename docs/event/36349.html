CreateJSはFlashクリエイターのためのJavaScriptライブラリ

Flashのオーサリング経験を生かして、HTML5/Canvas上のアニメーションを作り出せます。

  • PC/iOS/Android対応 → OK!
  • Flash CS6でオーサリングして書き出し → OK!
  • AdobeやMicrosoftがスポンサーについていて（ちょっとは）心強い → OK!
  • ASライクな記述でJSが書ける → OK!
  • FlashDevelopでの開発 → OK!
  • 商用案件での実績 → OK!
  • これまでの経験を生かせる → OK!

非常に優れたライブラリではありますが、日本語での情報が少なく手を出しづらい面がありました。この勉強会の活動を通して、ノウハウの発掘と共有を行います。

CreateJSに興味ある方は是非ご参加ください！

会場の変更、参加数枠の拡大はいたしません

1/31、会場変更、枠拡大ができるかを検討するため、CreateJS.jpサイトとtwitter上で会場提供の呼びかけを行いました。場所の提供のお申し出をいくつも受けました。しかしそれぞれにオペレーションや会場を貸していただく状況を想定して改めて考えると、私たちに運営経験が不足しており想像の及ばない部分が折り重なっていることに気がつきました。
継続的な運営のためには、少なくとも第一回目は想像のつく範囲の準備をしっかり行い、大きな会にしないことがむしろベストであるとの結論に至りました。

ご期待いただいた方、会場の提供を申し出ていただいた方、申し訳ありませんでした。

今回は会場の変更、枠の増加はいたしません。何卒ご容赦ください。
