11/15(水)開催　第27回KROWLセミナー
『現役デザイナーが語る！デザイナーの仕事とホンネ』

OPEN 19:00 / START 19:30 / CLOSE 21:00

KROWLセミナーについて

KROWLセミナーではWeb業界現場のナマの声や、スキルアップに役立つ情報を毎月お届けしています。
Web制作・運営、インターネット広告、SNS、ソーシャルゲーム、オンラインショッピング、ポータルサイトの運営など・・・
Web業界への転職をお考えの方、キャリアチェンジやスキルアップをお考えの方に向けた内容となっています。
特に実務未経験の方、初心者の方にピッタリの内容ですので、お気軽にご参加下さい！

イベント概要

メイントーク

プロのデザイナー3名をゲストに迎え、各々が携わってきた案件の紹介や社内での役割、大切にしていることなどをトークセッション形式でお伝えいたします。Web、グラフィック、プロダクトとあらゆる領域に共通するデザインの思考法や戦略など、デザインを仕事にしていく上での大切なことが学べます。特にデザイン初心者の方にぴったりの内容ですので、これからデザイナーとしてステップアップしていきたいとお考えの方や、デザイナーへの転職をお考えの方にオススメです！

【登壇者プロフィール】
「デザイン業界に就職・転職するための学校。」の講師含むゲストの3名にご登壇頂きます！

■グラフィックデザイナー　藤木 まな
（ http://mana-works.sakura.ne.jp/ddw/ ）
チームとしてイノベーションを起こす仕事をするため、企業デザイナーの正社員として働きながら、「ちょっと困った」「新しいことがしたい」という個人の方のためのデザインを請け負っています。

■グラフィックデザイナー　たじま ちはる
（ https://chi-design-ooo.com ）
戦略を立ててデザインで効果を上げるところまで関わるお仕事をしています。大手商業施設の広告ツールや店頭サイン、ブランドのポスター、広告サイトなどを制作してきました。

■WEBデザイナー　吉川 雅彦
（ http://yoshikawaweb.com/ ）
プログラマーからデザイナーへ転身。受託web制作会社、フリーランス、webサービス会社などを経て現在は、HTML、CSSのコーディングを中心に、マーケティング、ディレクション、デザイン、プログラミングなどを行っています。

■デザイン業界に就職・転職するための学校。
（ http://designgakkou.jp/ ）
デザイン事務所が作ったデザイン学校です。
現役のデザイナーが、制作現場の声を伝え、活きたノウハウを持った即戦力になるデザイナーを育成します。
学校に通いながらデザイナーを目指したい方、
会社に勤めながらデザイン業界への転職の準備をしたい方など、
デザイナーになりたい人の夢をかなえる学校です。

地図

主催： 株式会社 アイ・ファクトリー

アイ・ファクトリーは博報堂アイ・スタジオグループとして、2011年に設立したデジタル制作×人材サービスの会社です。大手・有名企業のコーポレートサイト、キャンペーンサイト、バナーなどのデジタル制作と、自社の教育システムを活かしたIT/デジタル人材の育成を行っています。また、業界未経験から現場で活躍できるクリエイターをこれまでに多く輩出した実績をもつ会社です。

KROWL とは

これからIT/デジタル業界でチャレンジしていきたい方を対象に、セミナーや面談、キャリアコンサルティング、インターンシップ等の就業支援を行っています。
IT/デジタル業界を知っている企業がサポートしているからこそ、就職活動もスムーズに。是非KROWLの就業支援を体験してください。
