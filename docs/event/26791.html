吉祥寺にあるオシャレなカフェで、お互いの好きな本を語り合って交換をしてみませんか？？

今回は文学小説に限定した読書会を開催します。

日本人作家を主に、ストーリー物を中心とした本をテーマにみなさんに紹介していただき、
持参していただいた本をみんなでシェア（交換）させていただきます。

お気軽にご参加くださいませ。

～～～シェア読書会とは？～～～
自分の持っている本の中から、他の人にも読んでもらいたい本を持ってきて紹介＆シェア！

いろんな背景を持っている人が集まるから、
手に取ったことのないジャンルの本に触れるチャンスになったり、
読みたかったあの本とまた出会えたり、
知り合ったことのないような人と繋がったり。。。

そんな、本と人が出会って繋がるイベントです。

持っていく本が思いつかない方は、手ぶら、本０冊でもOK!
他の参加者が持ってきた本のシェアを受け、次回以降で、お持ちいただければかまいません。

【Ｑ＆Ａ】
Ｑ．本はどんなジャンルを持ってくればよいですか？
Ａ．今回は現在文学をテーマにしております。ストーリー物の小説、文庫などをご持参ください。
例）村上春樹、村上龍、伊坂幸太郎、メジャー・マイナー問いません。

Ｑ．読書会って物凄い敷居の高いイベントじゃありませんか？
Ａ．本が好きならノープログレム！

Ｑ．知らない人同士で本の紹介とか緊張しちゃいます。。。
Ａ．初めての時はそうかもしれませんが、馴れると快感になります、笑

Ｑ．シェアした本が誰の手に渡ったかわからず、手元に戻ってきません。
管理はされているのでしょうか？？
Ａ．シェアする本は手元から、「手放す・与える・循環する」を基に運営しております。
返却・回収等は各自、自己責任でお願いいたします。
シェアしたくない場合は、ご持参いただいた本の“紹介”だけでも問題ありません。

Ｑ．イベント運営費（￥300）とはなんですか？
Ａ．現在、5名のメンバーで「シェア読書会Shablio（シェブリオ）」としてイベントを運営しております。メインの会場をベースに、派生系として開催しており、今後のイベントの継続性・繁栄も含めて徴収させていただきます。参加者の方も継続して付き合えたり、自発的に展開できるような読書会を目指しております。
