OSGeo財団日本支部（OSGeo.JP）とは

OSGeo.JPとは　OSGeo財団（The Open Source Geospatial Foundation）は、高品質のオープンソース地理空間ソフトウェアの支援と構築のために設立されました。この財団の目的は、コミュニティーが先導して、オープンソースプロジェクトの利用と開発を促進することです。OSGeo財団の日本における公式の支部が「OSGeo財団日本支部（OSGeo.JP）」です。
