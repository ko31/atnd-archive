description

最近のマーケティングのキーワードに「動画」「OnetoOne」「CRM」が挙げられ、
多くの企業が様々な取り組みを行っています。

この度弊社では、これらのキーワードを包括する全く新しいマーケティングソリューション
「PRISM」（パーソナライズド動画）の取り扱いを開始致しました。

当セミナーで、その概要や仕組み、活用方法をご説明するとともに、
最近の企業による動画活用の動向についてもお伝えします。

スピーカーには「PRISM」を開発された弊社パートナー企業である、
株式会社クリエ・ジャパンから責任者を迎えて直接お話しいただきます。

画期的なソリューションですので、
マーケティング担当者様・事業企画担当者様必見のセミナーです。

皆様のご参加を心よりお待ちしております。
