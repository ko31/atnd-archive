description

いままでなかった、父母が学ぶ中学受験の算数教室が荻窪に開講。中学受験で最も合否に直結する科目であると言われる算数。しかし、中学受験を経験されたご家庭ならお分かりの事と思いますが、中学受験の算数はとにかく難しい。お子様に「これどうやって解くの？」などと質問され、小学生が解く問題なんてたいした事はないだろうと思っていたら、全くわからない。試行錯誤したあげく、まだ習ってもいない数学（三角関数など）をイチから教える事になってしまったなどという話はよく聞きます（笑）中学受験の算数は子供たちにとっても、それに応えるお父様・お母様にとってもやっかいな代物なのです。すうがくぶんかの講座で中学受験の算数をしっかり学んでお子様に尊敬されてみませんか？

学ぶことで得られるもの
１）中学受験算数についての知識（解法／レベル／問題傾向）などがわかるようになる。
２）中学受験についての情報（学校情報／受験生の健康管理）を得られる。
３）同じ境遇（中学受験を控えるお子様をお持ちの保護者様）の受講者同士の情報交換の場としても活用できる。
