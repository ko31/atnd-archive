GREE Tech Talk #09

GREE Tech Talkとは

進化し続けるインターネット技術を
特定のジャンルにとらわれることなく追求していくために
グリーが年４回開催する技術勉強会です。

最前線で活躍する国内外の技術者をお迎えし
毎回異なるテーマについて
みなさまと一緒に考え、議論する場を提供しています。

http://techtalk.labs.gree.jp/

テーマ

GREE Tech Talk 9回目のテーマは「MySQL」です。
今回は、MySQLについてみなさまと一緒に学びを深めたいと思います。
