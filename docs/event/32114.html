イベント概要

ビッグデータ活用の重要性が叫ばれるようになって久しいですが、事業の現場からは色々な苦労話も聞こえてきます。
今回は、今更聞けないHadoop技術の基本の話から陥りがちな失敗例や苦労話、サービス運用の現場での具体的な活用事例などを通して、技術サイドとサービスサイドの両面から学べる勉強会を開催します。

Hadoopエバンジェリストの猿田さんと、HITソーシャルゲームを提供しているgloopsでデータマイニングを担当する井澤さん・滝さんから、Hadoop活用のリアルをお話いただきます。

また、勉強会終了後は懇親会も開催しますので、講師の方や参加者との情報交換・交流を深めて下さい！

【主催】
Web CAT Studio （株式会社リクルートエージェント）

【会場】
リクルートGINZA8ビル（東京都中央区銀座8-4-17）

講演者

猿田 浩輔 氏

2009年NTTデータに入社し、一貫してOSSの検証/整備、案件への適用を担当。特にHadoopとその周辺技術の検証/整備や案件適用に従事し、経済産業省からの委託業務の実証実験にも参画。2011年1月に「Hadoop徹底入門」を執筆。

井澤 正志 氏

2001年東北大学理学部化学系卒業。文部科学省入省。国立大学（人文社会科学系）の研究推進、原子力安全行政、独立行政法人（研究開発型）の研究企画などに携わる。2006年4月退職。同年、携帯サイトコンテンツ配信事業会社に転職。経営企画を担当。2011年2月、gloopsに転職。企画部マーケティンググループを経て現職。

滝 隆行 氏

新卒では3DCADを用いた設計業務に従事。
その後、25歳でIT業界へ転身し、SESでBtoB開発、モバイルWeb開発などを経験。某企業のアンケートサービスからHadoop・HBaseに関わることとなる。gloops入社後は、エンジニアとしてマジゲートのメインエンジニアを担当し、5月からデータマイニングに配属となり、Hadoop・pigを使用した解析業務を担当している。
