スピーカー: ゆたかさん (@tmyt) Microsoft MVP for Windows Platform Development

Azure MLを使って何かやる (サトヤサイド / 50分)

近年、様々な場所で「機械学習」という言葉を良く耳にします。
私たちの身の回りには様々なデータであふれており、これらのデータを効率的に分析・学習し活用することで、状態検出や予測など大きな価値を生み出すことができます。
そこで今回は、この機械学習を非常に手軽に・分かりやすく体験でき、実サービスにも柔軟に展開することのできるAzure MLを用いて、身近なデータの分析を行ってみます。
Microsoftの技術と開発者のアイデアが紡ぎだす未来を、一緒に体験してみましょう。
何を使うか、成功するかは当日のお楽しみ。
