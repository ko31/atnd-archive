description

【講座のポイント】
　ポリマーアロイ（高分子多成分多相系）における相構造形成の基礎を理解するために、ポリマーブレンドの相溶性・相分離からブロック共重合体のミクロ相分離まで解説し、ポリマーアロイの調製についても触れる。また、結晶性成分を含む高分子多成分系の高次構造形成に関して最近得られてきた知見も述べる。

【プログラム】

１．ポリマーアロイ（高分子多成分多相系）とは？

２．ポリマーブレンドの相溶性
　　2-1．ポリマーブレンドの相溶性の特徴
　　2-2．UCSTとLCST
　　2-3．Flory-Huggins理論と自由体積理論
　　2-4．ランダム共重合体ブレンドの相溶性

３．ポリマーブレンドの相分離過程と相構造
　　3-1．核生成・成長とスピノーダル分解
　　3-2．相分離過程と相構造
　　3-3．外的要因下での相分離

４．ブロック共重合体のミクロ相分離
　　4-1．ドメイン形態とドメインサイズ
　　4-2．ブロック共重合体を含むブレンド
　　　　　・マクロ相分離とミクロ相分離

５．ポリマーアロイの調製
　　5-1．耐衝撃性
　　5-2．アロイ化技術
　　　　・相構造制御、相容化剤、リアクティブプロセッシング

６．結晶性高分子多相系の結晶化と相構造形成
　　6-1．高分子の結晶化の基礎
　　6-2．ランダム共重合体（LLDPE等）の融解挙動
　　6-3．結晶性高分子ブレンドの結晶化とラメラ構造
　　6-4．結晶性ブロック共重合体の相構造形成
　　　　・結晶化とミクロ相分離
　　　　・ミクロドメイン内での結晶化の特徴
　　6-5．結晶性−結晶性ブロック共重合体の高次構造形成
