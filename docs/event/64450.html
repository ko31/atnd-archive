description

スマートフォンアプリのUI,UXデザインノウハウを短時間で効率的に学び、
よりスキルを高めたい方に向けたワークショップを開催します。

ワークショップではプロトタイピングにチャレンジいただきますが、
DeNAが社内のアプリ開発に導入した『Prott』を使用した新しい形式のワークショップです。

講師は、DeNAで様々なプロダクトの制作実績を持つクリエイターが務めます。
ワークショップの後は、懇親会の開催も予定しています。
どちらも参加費は無料ですので、ぜひお気軽にご参加ください。

▼学べること
・ターゲットの設定、ペルソナ設定、ユーザーシナリオの考え方
・スマホならではの、プロトタイピング作成ノウハウ
・作成したプロトタイプをデザイン評価する手法
・時間が限られた中での、効率的なブレスト手法
・ビジネス上のゴール設定ノウハウ

▼タイムスケジュール
・19:30～19:45
座学を織り交ぜつつワークショップの全体的な説明
・19:45～20:15
グループに分かれての課題取組・プロトタイピング
・20:15～21:30
発表と講評会、リフレクション、Q&A、アンケート記入
・21:30～22:00 懇親会
