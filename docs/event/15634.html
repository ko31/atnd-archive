【プログラム】
◆講師の言葉
　モータは人類の利便性の追求からその使用量は増加し続け、その中でも永久磁石式モータが制御性や効率の良さから顕著に増加している。自動車もハイブリッド車、電気自動車と永久磁石式モータの採用が増加している。永久磁石は省エネ、エコの観点から今後重要な材料である。しかし　レアアース希土類磁石は中国の輸出規制等でその供給は左右される傾向にありマスコミに連日取り上げられている。このような状況下では「脱レアアース技術」が日本のモータ技術として必要性が高まってきている。このためにはモータのメーカー、ユーザー、品質管理、保守等に関与するものはそれらの正しい知識と最適設計法、使用法を正しく理解することが必要不可欠である。本セミナーを受講することにより、永久磁石の知識、使い方と各種永久磁石モータの原理、特徴、理論及び簡易設計法が1日コースで習得できる。モータの発生トルクはその原理に基づきモータの簡易等価回路から、電卓やエクセル計算ベースで(磁界解析ソフト無で)簡単に計算する方法を解説する。これは設計法にもなる。次にレアアース磁石を用いないモータ技術を解説する。安価で調達容易なフェライト磁石でネオジム磁石を代行する技術、ギャップ対向面積増加技術、超集中巻きモータの例、高磁性材料(アモルファス)の例を紹介する。最後に、最近永久磁石を用いないモータとして脱レアアース対策の一候補として見直されている　SR(スイッチトレラクタンス)モータに関して、その原理、簡易トルク計算法、騒音発生ラジアル力の簡易計算、騒音原因と対策、高トルク化及びセンサーレス駆動(インダクタンスの変化利用)につき解説する。
　講師は日本電産サーボ(株)の技師長を永らく勤め、名誉顧問で平成22年年1月に退職し、現在は足利工業大学及び群馬大学にて電気機器設計、製図や電気工学、製品開発戦略等の教鞭をとっている。日本電産サーボ(株)にては3相ステッピングモータの発明と開発を始として約45年に及ぶ各種のモータの開発と多数の特許取得の経験を有する。本セミナーではメーカーでの設計開発と大学での教鞭の経験を活かして実用的で分かりやすいセミナーを実施する。

◆修得知識
・永久磁石の知識、各種永久磁石モータの原理、特徴、簡易設計法、最適な磁石の用い方
・安価磁石で高級磁石を代用する考え方及びSRモータの技術

◆プログラム
Ⅰ. トルク発生を理解するための磁気回路と永久磁石
　　1. 磁気回路の基礎
　　2. 磁化曲線
　　3. 永久磁石(種類、使用法、着磁法)

Ⅱ. 永久磁石式モータのトルク理論とトルク計算例の実際　
　　1. DCモータ(IBL則法)
　　2. ブラシレスDCモータ(IBL則法)
　　3. ステッピングモータ(仮想磁路法)

Ⅲ. モータの高トルク化技術
　　1. 磁路パーミアンス増加技術
　　2. 巻線法、巻線極数、相数と高トルク化技術

Ⅳ. レアアース磁石を用いないモータ技術
　　1. ネオジム磁石をフェライト磁石で代行する技術例
　　2. ギャップ対向面積を増加した2つの例
　　3. 超集中巻きモータの例
　　4. モルファス等の高磁性材料による例
　　5. SRモータによる例(低騒音化、高トルク化、センサレス化)
