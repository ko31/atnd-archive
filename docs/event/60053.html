「ホントに数時間で3Dコンテンツを作れるようになった！」

jQueryさえわかればたった2時間で3Dコンテンツを作成できるjThreeの勉強会を
アドウェイズで開講！ jThree開発者の松田光秀さんに1時間の講習をしていただき、
その後、1時間のハンズオンで実際に3Dコンテンツを作ってみようという勉強会です！

参加条件はただ１つ、「jQueryが好きである」ということです。
jQueryが好きで最低限、
$( “div” ).animate( { width: “500px” }, 1000 );
の意味さえわかれば技量に関係なくjThreeでWebGLを動かせるようになります。
