description

■3.11以降、「何かしたい」と思ったまま、止まってはいませんか？

3月11日の地震直後、恐らくほとんどの人が考えたと思います。

「自分にできる支援は何だろう」

しかし、自分が被災地に行くと足手まといになるかもしれない、
何も知識がないし、行くあてもない、日々他のことで忙しい…。など
当初の想いを実行に移せていない方は少なくありません。

震災から5ヶ月が経とうという今、復興支援の形は多様化してきています。

ボランティアをされている方々を支援するボランティアであったり、
いまいる地元で現地のニーズを把握し連携をとりながら行うボランティアであったり…。
自分のライフスタイルや性格などに合った復興支援の形がきっとあるはずです。

でも、まだ多くの人が「何か自分にもできたら」と思ったまま、
そこで、立ち止まってしまっています。

そんな人に、自分に合った復興支援と出会うチャンスや、被災地を元気にする力に目覚めらるきっかけ、一歩踏み出すきっけけを作って頂きたくて、今回のイベントは企画されました。

■第一線で活躍するビジネスプロデューサー２人から「動くための方法」を学ぶ。

では、どうすれば、何か自分なりに「何かアクションする」ことができるのでしょうか？

それを学ぶために、今回、震災以前から第一線で様々なプロジェクトに関わり、そして、震災以降も、 復興プロジェクトにいち早く「動いた」２人のプロデューサーの方を講師 にお招きすることにしました。（※講師プロフィールは下記プロフィール欄を参照下さい）

前半はトークライブ形式、後半はグループワーク（対話）形式
前半、お２人の震災復興プロジェクトの話を聞いて、普段疑問に思っていることをクリアにするチャンスにするとともに、ただ話を聞いて終わりでなく、後半には実際に集まった参加者でグループを作り、自分たちでできる復興支援を具体的に考える時間も設けています。次のアクションにつなげるのがこのイベントの目的です。講師のお二人から、実際にプロジェクトを進める上でのアドバイスも直接頂け、自分たちでできる復興支援を具体的に考えられるのが、このイベントの特徴です。

午後の予定も空いていて、引き続き交流・ディスカッションを続けたいという人には、そのまま会場を用意しています。よろしかったら、ぜひ昼食を持参して参加下さい。
※13:30以降の参加は任意です。

２人のプロデューサーの対談が聞けるまたとないチャンス！
そして、復興支援に対する新たな気付きや、ヒント、出会いを得るチャンスです。

夜まで参加すると、同会場にて開催予定の神宮花火大会交流イベントに参加できます。（参加費別途）
