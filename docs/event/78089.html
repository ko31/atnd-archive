facebook、twitter、LINE、pinterest、snapchat…
世界中には様々なSNSが存在する中、
世界トップクラスの人気を誇っているのが、インスタグラムです。

ユーザー数4億人
日本ユーザー数810万人
女性のユーザー数７割

世界中の団体や企業も、インスタグラムに参入。
独自の世界観を、写真を通して世界中に伝えています。

「写真で世界とつながることができる。」
これがインスタグラムの最大の魅力だと思います。
そんな人気のインスタグラム。

始めたはいいものの、
facebookやtwitterとは少し違う、、。

 

・フォロワーの増やし方がわからない、
・なぜ、いいね！がつなかいのだろう、
・どの加工を選べばいいのだろう、
・写真が上手く撮れない。。。

そんなふうに感じたことはありませんか？

 

今回の講座では、
あなたの投稿を見たインスタグラムユーザーが、
思わずいいね！やフォローしたくなってしまう
写真撮影や加工のポイント、とっておきのフォロワーの増やし方
などを、お伝えいたします。

インスタグラムの中では、ユーザーの方に好まれる加工や
写真のジャンルが幾つかあります。

 

「どんな写真を投稿すれば、伝えたいことがフォロワーさんや
多くの方に伝わるだろう？」

 

投稿するときに僕は、そんなことを考えながら、加工をしたりしています。
今でも全ての投稿が成功しているわけではありませんが、
写真センスの全くなかった私が、フォロワーの方を意識しながら、
撮影と加工を繰り返しているうちに、

気がつけば、
自分でも驚くような反応をいただけるようになりました。

 

まだまだ試行錯誤しながらではありますが、
僕がインスタグラムを運営するにあたり、意識しているポイントを
8つに絞り、今回ご紹介させていただきます。

参加者全員に特別テキストもご用意いたしますので、
ぜひご参加いただき、あなたのアカウントを
より多くの人に見ていただけるアカウントに一緒に育てていきましょう！
