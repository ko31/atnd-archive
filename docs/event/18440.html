Tech Party 2011 とは

全国のコミュニティが合同で 8/20(土), 8/27(土) に勉強会を行い、 USTREAM 中継を通じて連携するイベントです。
Microsoft Tech・Ed Japan の開催延期から、各地のコミュニティが同日にイベントを開催しようといった動きが起こり、 せっかくなのでご一緒にといったところが発端です。

Hokuriku.NETでは３つの部屋を用意して１つは通常の勉強会のように北陸でセッションを行い、残りの２部屋で他のコミュニティのライブ中継したりコミュニケーションを取ろうと考えています。

ぜひ気軽に足を運んでください。

講演概要

■Lyncを使って、ここまでできる！（リアルタイム コミュニケーション開発）

  • マイクロソフト　松崎剛さん
    最近にわかに人気の Lync ですが、本セッションでは、この Lync を使ってどのようなカスタム機能の構築が可能か説明します。結論から言うと、実は、かなりなことができます。このセッションを通して、その大きな可能性を感じて頂ければと思います。
    なお、開発関連の内容ですが、Lync の特徴や、Lync Online、Lync Mobile の話など、基本的な内容についても最初のほうでおさらいします。

■Windows 7 活用術 ～おさらい編～

  • @ramu_mystery
    Windows 7が発売されて、この10月で早くも2年。しかし、従来のWindowsとは操作方法が変わってしまって「何だか使いづらい」と感じている方が多いようです。この2年間でいただいた質問を中心に、どうすればWindows 7をスムーズに活用できるのか！？一度おさらいをしてみましょう。既に持っている人はもちろん、まだ持っていない人もどうぞ！

■ASP.NET ＋ HTML5

  • @Fujiwo
    Visual Studio を使って、今話題の HTML5 な Web アプリケーション作成に挑戦します。

■Windows Phoneのアプリケーションライフサイクルについて

  • @kabakiyo
    Windows Phoneのアプリを作るうえで欠かせないアプリケーションライフサイクルについてお話したいと思います。アプリを起動してから終了するまでの流れを解説できればと思っています。

その他

  • 当日PCをお使いになる方は電源タップや延長ケーブルを持ってきていただけると助かります。
  • 無線LANを用意しますがライブ中継を行う予定ですのでご使用を控えていただくこともあります。
