中途採用のポジション増加に伴い、弊社として初めて中途採用向けセミナー＆人材交流会を開催することが決まりました。

弊社代表ももちろん、事業開発部門や営業部門のトップも直接ご参加いただいた方々へお話しをさせていただきます。 業界や事業、一緒に働きたい人物像などをお話しさせていただき、質疑応答のお時間や、軽食を用意した懇談会（自由参加）のお時間も予定しております。

下記に一つでも該当する点がありましたら、是非お気軽にご参加ください！

  • 世界に通じる事業作り・サービス作りを経験してみたい
  • 最先端テクノロジーを用いたソリューションを自身の手で提供してみたい
  • 急成長ベンチャーで思いっ切り働いてみたい
  • アドテク業界に興味がある（未経験歓迎です！）
  • 何となく会社や事業がおもしろそうで気になる

【予定コンテンツ】

■アドテクのこれからと、ジーニーのこれから

[代表取締役 工藤智昭]

早稲田大学大学院 理工学研究科 情報ネットワーク専攻 修了。 在学中にインターネット広告関連のベンチャーを起業。 株式会社リクルート（現 株式会社リクルートホールディングス）に入社し、アドネットワーク事業推進を担当。 リクルートの広告を起点として、日本最大のエリアアドネットワークを構築。リクルートでは、全社イノベーション賞、New RING準グランプリ、Goodデザイン賞などを受賞。 2010年4月に当社を設立。

■ジーニーのプロダクト作りと戦略

[取締役 事業開発部長 廣瀬寛]

京都大学 経済学部 卒業。 株式会社リクルートメディアコミュニケーションズ（現 株式会社リクルートコミュニケーションズ）に入社し、地域アドネットワーク事業のアライアンスを担当。 リクルートアドネットワーク、リクルートオーディエンスネットワークの商品開発に携わった後、当社取締役に就任。

■最先端の現場で求められる人材

[執行役員 営業部長 第二メディア部長 吉村卓也]

南山大学 経営学部 卒業。 ベンチャー企業の支店立ち上げに従事した後、SNS事業で起業。 株式会社リクルートメディアコミュニケーションズ（現 株式会社リクルートコミュニケーションズ）に入社。リクルートアドネットワーク、リクルートオーディエンスネットワークの営業/クリエイティブディレクター、大手広告主の広告最適化を担当。 リクルートでは、トップガン選考対象、社内論文コンテスト受賞、FPA受賞、51期特別賞受賞。

[第一メディア部長 水野敦紹]

静岡大学 人文学部 卒業。 株式会社ブシロードに入社し、取引先のコンサルティングから イベンドの企画運営までを幅広く担当。 その後、インターネット広告代理店を経て当社へ入社。 設立2期目より現場を牽引し、2015年4月より部長へ就任。
