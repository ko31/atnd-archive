description

これまでの講義主体の勉強会から、今回は趣向を凝らして「人間観察ワークショップ」を行おうと思います。

ITに関係なくとも、世の中のあらゆるサービスや行動を観察することで、いいものを作るヒントに、もしくは、新たな側面からの発見ができるかもしれません。
例えば、テーマパークからは人を楽しませるためのヒントが、お化け屋敷からは人を恐怖に陥れるヒントが得られると思います。

そこで注目したのが「学園祭」です。学園祭は、人を楽しませるため喜ばせるための工夫が詰まったイベントなので、まさしく行動観察の宝庫です。また、11月は学園祭シーズンです。

というわけで、11/17〜18に開催される北九州市立大学の学園祭にて行動観察ワークショップを行います。

皆で旅行がてら、行動に注目しつつもわいわいと楽しく学園祭に行くのはどうでしょうか？

内容

  • 行動観察ワークショップ
    **「楽しめてない人のその原因を探し、どうすれば楽しめるかを考える」
    • 1）楽しそうにしてない人を見つける
    • 2）その人を観察する
    • 3）皆で、楽しくするにはどうしたらいいか考える（大学内の教室に移動）

という大まかな流れで行います。なお、グループ毎にするか個人にするかは参加人数次第にしたいと思います。
