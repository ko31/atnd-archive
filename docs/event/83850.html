オウンドメディアとセキュリティ入門

2016年12月7日（水）に、ECサイトやWebメディアなどを運営してる方を対象にしたセミナーを開催いたします！

顧客に自社のECを見つけてもらいやすくするためにはどうすればいいの？
競合他社と差別化するためには？
サイトの安全性を高めるためには？
セキュリティ対策って具体的に何をすればいいの？

などの悩みをお持ちのECサイト運営者やWebメディア運営者様に最適なセミナーとなっております。

メディア運営やコンテンツマーケティングを得意とする京橋ファクトリーと、
情報テクノロジーを活用した先進的ビジネスソリューションとサービスを提供する
サイバーセキュリティクラウドが一緒になって、ECサイトやWebメディアなどを運営してる方を対象に
これからのECサイトやWebメディアにおける ”攻め” と “守り” の戦略についてご紹介いたします。

1部 〜モノからコトを訴求して集客しよう〜

・オウンドメディア導入のメリット
・ECでのオウンドメディア成功事例について
・オウンドメディアを成功させるために必要なコンテンツとは
・オウンドメディア運用に必要な体制や予算、人材とは

登壇者：株式会社京橋ファクトリー　　前田 裕司

大手電機メーカーの航空関連部門での海外営業や新製品開発などを経て、2015年京橋ファクトリー入社、執行役員COOに就任。
兵庫県立大学経営学部にて特別講師。次世代マーケティングプラットフォーム研究会運営メンバー。日本マーケティング学会会員。

2部 ～情報漏えいする前に！最低限のセキュリティ対策～

・急増するWebサイトのセキュリティ被害状況。
・セキュリティ被害がECサイトへ与える影響とは！？
・Webセキュリティ対策の３つのチェックポイントとは！？
・明日からすぐに出来る具体的なWebセキュリティ対策！

登壇者：株式会社サイバーセキュリティクラウド　 山本 裕貴

営業部サブマネージャー兼プロダクトスペシャリスト。企業規模を問わずWebサイトのセキュリティ対策に関する課題解決を得意とし、サポートまでを一貫して携わる。フジテレビへの出演をはじめ、サイバーセキュリティに関する勉強会など、講演多数。
