Pythonおじさんの@nakayoshixです。
7/30(土)にPython札幌が約一年ぶりで復活します。(Python札幌シーズン3)
その前の準備段階として、まずはプレ勉強会を開催することにしました。

追加の注意事項

  • 定員の数を8名から12名に増やしました。
    • 椅子とテーブルは12名が物理的な限界となります。
    • プロジェクタはテーブルの上に置くことから、食事をする余裕は無さそうです。
    • そのため、勉強会としての二時間は飲み物だけで食事はなしとします。
  • (その代わりに)勉強会後の20:30〜を懇親会(お食事会)の時間としました。
    • 粋ラボ定食(850円, 野菜系向き？)
    • 豚丼(950円, 肉系の人向き, 予約が必要？)
    • その他にも夜の部のメニューがあります。

プログラム(予定)

  • 簡単な自己紹介 (全員)
  • Python札幌の今までの活動の歴史を簡単に振り返る (@nakayoshix)
  • PyGameについての概要紹介 (@giginet)
  • Emacs上のPython開発について(××補完計画) (@nakayoshix)
  • 今後の活動の方針や役割分担等について話し合う (全員)
  • 7/30(土)では誰がどんな発表をするかをある程度決定する (全員)

なお、もし時間が余れば以下もやるかもしれません。(多分無理だと思いますが…)

  • EmacsおじさんのPython IDE体験記 (@nakayoshix)
  • 6/4,5に開催されたTDDBC札幌2.0のPythonチームA, Bでやったことの振り返り (@ayako119, @giginet)

参加費:

  • 一人500円 ＋ 飲み物代1杯分。
    • 食事をする人は食事代のみになります。
    • 夜の部の定食は850円となります。
    • 豚丼を確実に食べたい人は予約してください。
    • オーガニック創作料理の店なので、本当は玄米菜食(長岡式酵素玄米)がお薦めです。 (@nakayoshix)

その他:

  • 会場には Wifi があります。
  • 会場にはスクリーンとプロジェクタもあります。
  • Ustream 中継は未定ですが、もし希望が多ければ考慮します。
    • 豚丼以外のメニューを知りたい方は、ぐるなびのページを見てみてください。
