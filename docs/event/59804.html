講演概要

Quartz Composerってなに？(担当 keiso)

過去に何回か広島マックユーザーグループでも紹介しているQuartz Composerですが、今回は「Quartz Composerってなに？」と題して、Quartz Composerが何もので、どのような事が出来るのかと言った事を中心に紹介します。

一般的にプログラミングというとコードを書いて実行する形が多いですが、Quartz Composerはグラフィカルなユーザーインターフェイスで、マウスを動かして必要なパーツを繋いでいくだけでリアルタイムに動作する映像を作る事が出来ます。

※Quartz Composerは開発者向けのサイトで無料で配布していますので、OS X 10.4以降を使用されている方は、どなたでもインストールして使う事が出来ます。

Processing講座(担当 tettou771 さん)

Processingとは、映像表現に特化したプログラミング言語（開発環境）です。
言語の中では習得にかかる時間が短く応用も効くので、はじめてプログラミングに触れる人にもおすすめです。

今回は、実際に何ができてどんなことに向いているのかということをYouTubeの動画や実際に動くサンプルを交えながら解説します。
質問があればそれに答えながら、雑談的に話ができたらと思います。
「自分でも何か作ってみたい！」という人が何人かいれば、統合開発環境の使い方やプログラミングの方法についても少し触れようと思います。
