イベント概要

web業界の中でも、最近特に人材ニーズが高まっているwebデザイナー。
ここ１年半で、求められるポイントは大きく変化してきています。
そんな、企業から求められる「リアルなwebデザイナー像」を、
採用責任者の話や、具体的な選考事例などから考えてみる勉強会です。

講演の他に、ご希望の方はキャリアアドバイザーとの簡易相談会にもご参加いただけます。
講演・相談会ともに費用は無料ですので、お気軽にご参加ください。

【主催】
Web CAT Studio （株式会社リクルートエージェント）

【会場】
グラントウキョウサウスタワー24F 2424会議室（東京都千代田区丸の内1-9-2）

　「参加希望」の旨をご連絡ください。
