catch ウェブポで年賀状送るよ！
description

有名人でも著名人でもないけれど、私 tmd45 からの年賀状が欲しいひと募集します。

  • オンライン年賀状サービス『ウェブポ』（ https://webpo.jp/ ）を利用して送りますので、tmd45 に住所を知られること無く、年賀状をお受け取りいただけます。詳しくは『ウェブポ』でご確認ください。
    • ウェブポ – よくあるご質問（FAQ）｜受け取り手続きについて
  • Twitter のフォロー・フォロワーさん、Facebook のお友だちに応募してもらえると嬉しいです。年賀状にひとことコメント付けます（笑）
  • 全く知らない人から応募していただいてもいいですが、私から年賀状もらって嬉しいですか･･･？お年賀コメントも多分テキトーになってしまうので、その辺は空気を読んでくださるとｳﾚｼｲ（・ω・｀ ）
  • 年賀状の送付元は「ウェブポ事務局」になります。

追記 2011/12/14

  • 「家族に見られたら困るようなメッセージは付けないで欲しい」とかご要望ありましたら、個別にご連絡ください。（コメントに書いていただいても構いません）

追記 2011/12/14 の 2
