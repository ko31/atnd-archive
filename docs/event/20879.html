description

今、Androidはスマートフォン市場でシェアを伸ばし、もっとも注目を浴びています。その最大の魅力が「アプリ」。

大きな会社も個人でも、自由にアプリを作り、マーケットに出品・配布できます。
一方では、どうやって開発すればいいのかがわかりにくく、途中で挫折するかたもしばしば見受けられます。

そのような方々のために「さいしょのいっぽ」を踏み出すためのお手伝いをしたいと考え、今回の講座を企画しました。

講座内容

１．インストール
２．開発ツールの使い方
３．プロジェクトの作り方・構造・カスタマイズ
４．基本的なJavaファイルの提供、構造の理解とオリジナルアプリの制作

てんこ盛りではありますが（汗）、難しいJAVA言語の基本勉強ではなく、理屈の理解は足らなくても「計算機」や「じゃんけんゲーム」程度のものは作れるような、実践的なものにします。

この講座が、「つぎのにほめ」につながることを期待しています。

当日までにご用意いただきたいもの

・PC（Windows / Mac）
・開発ツール(さいしょのいっぽのはんぽまえをご覧いただき、開講までにインストールを完了しておいてください。)
・筆記用具
・参加費（4,000円：税込）
・無線LAN含め、ネット環境は用意しておりません。

その他
