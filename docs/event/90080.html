description

働き方改革は、長時間労働の改善が注目されておりますが、本質的なテーマは生産性の向上、創造性ある仕事への転換にあります。テレワークをはじめとする様々な取り組みが政府主導で展開され、東京オリンピックを迎える2020年に向けて、今まさに企業力強化の絶好期といえます。
この度、中堅・中小企業様向けの情報化を推進するべく、情報インフラ／情報コンテンツ／情報活用の３分野を融合した「経営情報ソリューション」のセミナーを開催いたします。 利活用効果をイメージいただける展示紹介や、政府の助成金紹介などのコーナーを設けておりますので、是非ご参加いただけますよう宜しくお願いいたします。

＜第1セッション＞13：30～13：45
協立情報通信の取り組みのご紹介
～情報活用の進化と情報創造コミュニティー
企業経営には情報化/情報活用の進化が欠かせません。協立情報通信は、情報創造コミュニティという「場」を設け、「①インフラ・②コンテンツ・③活用能力」の三分野を統合したソリューションサービスを提供し、お客様の情報進化をサポートする体制の充実に努めております。お客様の情報活用による価値づくりに貢献するべく、当社の考え方ならびに取り組みについてお話いたします。

（講師：協立情報通信株式会社　代表取締役社長　長谷川 浩）

＜第2セッション＞　13：45～14：45
働き方を変えるインフラ＆コミュニケーション
働き方改革が叫ばれる中、テレワークなど新しい仕事スタイルが広がりつつあります。テレワークにより勤務の効率化だけではなく、個人の生産性向上と組織運営の合理化、さらには社会の活性化が期待されています。これらを支えるインフラソリューションサービスをご紹介します。

「どこから」働く？　安心・安全なテレワーク環境を実現するデスクトップ環境
安心、安全にICTを活用してテレワークなどを実現するためには、仮想デスクトップ環境やシンクライアント環境の活用が１つの方法となります。本セッションでは、セキュリティと運用管理の合理化を実現するNECのデスクトップソリューションサービスをご紹介します。

（講師：日本電気株式会社）

いつでもどこでも内線電話！あなたのスマホが内線に
いつでもどこでも内線電話を実現するのが、NTTドコモ「オフィスリンク」サービスです。通信交換機（PBX）と連携させることで、保留や転送などが外出先、自宅などから利用できます。今回は、オフィスリンクによって通話コミュニケーションの合理化を実現した事例をご紹介いたします。

（講師：株式会社 NTTドコモ）

所有から利用の時代！「センターレンタルサービス」のご紹介
従来は大きな投資を要したシステムが、クラウド等の月額方式で利用できる時代となりました。当社では、オフィスリンク連携の通信交換機（PBX）を、設置型でありながら、工事費・運用保守サービス込の月額レンタルにて提供しております。投資リスクも無く、機能アップ・利用効果を継続的に実現できるサービスをご紹介いたします。

（講師：協立情報通信株式会社）

休憩（15分間）
　　　
＜第3セッション＞　14：50～15：50
生産性を高めるソフトウェア活用と運用モデルをご紹介
働き方改革の重要なポイントは、社員力をアップすることです。具体的には、ソフトコンテンツ等の利用において、情報活用力を向上させることにあります。ここでは先進の業務アプリ、グループウェアの活用モデルをご紹介いたします。

OBC基幹システムの概要とそれをベースにした働き方改革への道
“働き方改革に取り組んだ” という実績と、目に見える “確かな効果” は、働き方改革の鍵です。OBCでは、全ての企業が自社で業務時間是正の取り組みができる、全69モデルの「OBC流 働き方改革」をご提案しています。

（講師：株式会社オービックビジネスコンサルタント）

マイクロソフト Office 365 でチーム生産性を向上させる情報活用をご紹介
場所を問わず、どこでも仕事ができる環境を創ることにより、社員の自律性とチームの生産性を向上させることができます。Office 365はその環境を提供する情報プラットフォームです。日々進化するOffice 365の最新機能のご紹介と共に、チーム活用の運用事例、生産性効果事例等をご紹介します。

（講師：日本マイクロソフト株式会社）

経営情報ソリューションサービスのご紹介
働き方改革の掛け声のもと、クラウドやモバイルを基盤とした情報運用モデルの革新が図られようとしています。OBC奉行、Office 365をベースに、お客様の情報活用の進化を支えるソリューションサービスを提供し、お客様の情報革新のご支援サポートに取り組む当社のサービスをご紹介します。

（講師：協立情報通信株式会社）
