description

女流演出家である山田真実がt t u (ティー・ティー・ウー)を旗揚げしました。
南池袋・雑司が谷にあるカフェ「ロケットカフェ」を会場にして、
場所からイメージした演出に挑戦致します。

テキストには、フランスの小説家兼映画監督でもあるＭ・デュラスの
「太平洋のおとこ」を使用しています。

第一回公演のテーマは、いれものから発想した中身をつくることを試みます。
観客が観劇前と観劇後に見る景色や空気を異なるものとして味わってもらえればと思います。
作品と空間がどのように寄り添えるかを模索と、
キャスト・スタッフを女性に絞り、女性目線からの焦点を探していきます。

小説では、「わたし」が記憶していることや、
「わたし」による記述や撮影によって記録していることことを想像しながら
読み進めていくのが読者の醍醐味ですが、今回は記録／記憶された対象＝「あなた」を
実在させて、小説にはできない解釈の開拓を目指します。

本作品に上演にあたって、カフェ公演ならではの飲み物やデザートなども提供予定です。

＜あらすじ＞
映画監督の「わたし」が、海辺で、恋人の「あなた」を撮影している。
時が過ぎて、恋人は姿を消し、ひとりきりになった「わたし」は「あなた」についての
記録を書き始める。
が、しかしやがて再び「あなた」不在の映画を撮ることを決意する。
そのとき、いなくなったはずの「あなた」が「わたし」の目前にたち現れてきて・・・
