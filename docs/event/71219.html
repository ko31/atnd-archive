description

『Gadget Fest. (ガジェットフェスト)』は、エンジニアの技術力,それぞれが持つビジョンを発信/表現する場です。多種多様な技術・製作物やアイディアが存在することを他業種の方々に触れてもらえる場でもあります。また、このフェスティバルが、エンジニア同士や他業種の方々との今後の活動の繋がりを得る機会となればよりうれしいです。このフェスティバルが行えれることにより、それぞれの価値を高めていくと同時に地元の情報サービス産業の発展を目指しています。
作品はスマホアプリ、メディアアート、IoT、電子工作、ウェブサービスなど多岐に渡ります。 新しいもの好きや機械好きな方、ぜひ会場までお越しください。

=====
第一会場 Noblesse Oblige @柏 10/31~11/1
=====
JR柏駅から徒歩７分の場所にある千葉のコワーキングスペースです。
テーマパークのようなオシャレな空間の中で個人エンジニアの作品をご紹介します。

作品展示は11時から17時まで行っています。 製作者による作品発表会は13時からです。
=====
第二会場 江戸川大学@流山 11/3(祝)
=====
江戸川大学の学園祭の一角をお借りして、 学生さんと一緒に作品の展示を行います。

=====
第三会場 KOIL（柏の葉オープンイノベーションラボ） 11/14~15
=====
TX線柏の葉キャンパス駅からすぐ近くの場所にある 大型コワーキングスペースKOILの場所をお借りして、作品発表と展示を行います。

会場によって出展内容が異なりますので、ぜひ会場までお越しください。
