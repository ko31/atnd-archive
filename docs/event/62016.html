誰かと話す機会は毎日のようにあります。
そして、日々の会話の中で相手に好印象を持ったり、悪い印象を持ったり、イメージは様々です。
たくさんの人に好意・好印象をもたれている人は、ある共通した「話し方」がいくつもあります。
コーチング・コミュニケーション学から「人に好意を持たれる『話し方』」を習得し、
職場や日常生活がより明るく楽しくなる人間関係を築きませんか♪

～セミナーの内容～

・面識の無い人にでも好かれることが出来る
・自分を印象付けるための「３つの法則」
・相手の欲望を満たす言葉を伝える
・友好な人間関係を築く質問の仕方
・相手のほめて欲しい所を見つける技術

～このセミナーはこんな方におすすめです～

・人に好かれる自分になりたい
・職場や友人との人間関係を良くしたい
・会話力を身につけたい
・たくさんの人脈を作りたい
・コミュニケーション力を身につけたい
・部下のやる気を引き出したい
・営業成績を良くしたい
etc…
