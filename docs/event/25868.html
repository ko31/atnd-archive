catch すぐに使える「すごい会議」のエッセンスと実例を大公開
description

こんな悩みはありませんか？

  • 社長や優秀な幹部の分身が、あと5人ほしい。
  • 経営陣と現場の目標意識にズレがあると感じる。
  • このままのワンマン社長で今後も伸び続けるだろうか。
  • 堂々巡りの戦略会議が多く、燃費が悪いと思う。
  • 気になっている計画はあるが、なかなか進まない。

上記のひとつでも当てはまりましたら「すごい会議」で解決しましょう！
すごい会議は、規模・業界を問わず高い成果を上げています。

例えば、

家電メーカー（売上400 億）
国内の市場全体が10%以上落ち込む中、1年間で売上15％増。

不動産仲介管理会社（売上5 億）
1年間売上5 億→10億、利益1500万円赤字から5000万黒字へ。

通販会社（売上1500 億）
2か月間で6億のコスト削減に成功。

これらの成果のドライバーとなるのは、すごい会議によって生まれる「志」。世の中にある多くの研修は、それぞれに素晴らしいものですが、「志」そのものを生み出すプログラムはそう多くありません。今日はそれを体感しつつ、ノウハウをお持ち帰りいただくセミナーです。

講義内容

  • 会議の生産性が100 倍になるシンプルな方法
  • 組織に高い志を生み出すには？
  • パフォーマンスが劇的に上がる「問題の扱い方」
  • 実際に「すごい会議」で生まれた画期的な解決策の実例紹介

こんな成果が得られます

ポイント１
経営メンバー全員がワクワクでき、それを信じてがんばれるような高い目標を立てることができます。
ポイント2
目標達成のために、誰がいつもでに何を実行し成果を出すかが明確になります。
ポイント3
セクショリズムに陥ることのない、最適な役割分担ができます。
ポイント4
解決策のわからない問題に最善のアクションが得られる仕組みを得られます。

上記のすべてを「今実際に直面している経営課題」をテーマに行い、実際に行動を起こすための具体的な計画と役割分担、全員のコミットメントを得ますので、その場で経営に貢献するインパクトが得られます。
