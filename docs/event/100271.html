【ランキングバトルについて】
　
CYCLOPS様とのコラボ企画として、
「oreRevo x CYCLOPS ランキングバトル」が始動！！
　
oreRevoが主催するオンライントーナメントの戦績と、
CYCLOPS様の地下施設で開催されるトーナメントの戦績を
2か月間記録し、上位入賞者のプレイヤーには賞品が贈呈されます！

https://www.oreevo.net/orerevoxcyclops

【その他・注意事項】

★試合中にラグ・不具合が発生した場合についての対処
・ラグを感じた場合は、タイムが「99」から「85」になるまでに操作を手放してください。
その後、配信チャット欄にてラグが発生した旨を報告ください。再試合を致します。
(規定時間を過ぎて動いたり、ジャンプしたりコパ連打していた場合はノーカウントになります)
・コントローラーの不具合が発生した場合は、発生したラウンドを負け試合とします。
・その他の急ぎの連絡や不具合等が発生した場合は直ちに配信チャットにて申告して下さい。
・参加者数が定員の半分以下の場合は中止とさせていただく可能性が御座います。
・トーナメント中のキャラクター変更は不可と致します。
・遅刻は進行が遅れる原因となり、他の参加者にも影響が出ます。
　なるべく配信を開いて自分の順番に遅れないようにご協力お願いします。
