概要

C言語は落とし穴が多く、相談できる人がいないとつまづきやすいですよね。 というわけでC言語をみんなで学ぼうという趣旨の初心者向けC言語勉強会を開催します。

資料を元に会を進めますので資料をダウンロードしてください。 この会では資料にある練習問題を全て解くということに目標を設定します（基本的なC言語の文法を問うものを作ったので、全て解けたら初心者は卒業です）。 解く速度も人によって違うと思うので、解いていく中で聞きたい事などを質問するという、半自習のようなスタイルで進めていきます。

初めて参加するという方も歓迎です。C言語ができるようになりたいという方の参加をお待ちしています。

当日準備

ノートPCを持参してください。 開発環境を初めて構築する場合、OSに応じて、以下のソフトをインストールしておいてください。 困ったことがあれば当日に聞いてもかまいません。

Windows : Visual Studio Community Edition

  • （場合によってはインストール最終盤でプログレスバーが数時間進まなくなることがあるようですが、その場合実はインストールが完了しているのでそのまま閉じても問題ありません）
  • インストールしたらこれに加えて、「新しいプロジェクト...」をクリックして出てきた画面の左ペインで「VisualC++」を選択し、2つあるツールの上のほうを追加インストールしてください。 これで開発環境の構築は完了です。それ以降はこちらの3.以降を参考にしてください。

MacOSX : Xcode

  • OSのバージョンによって使用できるXcodeのバージョンが異なります。こちらのまとめを参考に、Xcodeのバージョン一覧ページからあなたのOSに対応したXcodeをインストールしてください。

使用する資料

C言語の簡易問題集のような形式をとったhtmlファイルが入っています。detailsタグを使用しているため、Google Chrome、またはOperaを使用して閲覧してください。また、問題集以外にもC言語の学習に役立つリンク集と、Visual Studioのショートカット集も入っています。
