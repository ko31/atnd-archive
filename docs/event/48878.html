catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

※１セッション目の重村さんのセッションは4月に延期となりました。
　楽しみにされていた方申し訳ございません。
　代打として、藤本がコミュニケーションの簡単なワークショップやります！

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各40分弱くらいで2,3個する予定です。

■現在予定している内容：
　

●セッション1(40分)
　『私の「書く」ということ』日本Androidの会 中国支部長 重村さん

　＜主な内容＞
　これまでの執筆経験を元に、記事の作り方やきっかけについてお話しします。

　＜重村さんの紹介＞
　日本Androidの会　運営委員　中国支部長。
　山口県、広島県を中心に活動するエンジニアです。
　今月、『Android SDKポケットリファレンス』という本を上梓しました。
　書店に並んでますので、宜しければお手に取って見てください。

　

●セッション2(30分)
　『Sassの魅力ついて』広島工業大学の武田さん

　＜主な内容＞
　CSSのメタ言語であるSassついてのセッションです。
　Sassがどんなものなのか、基本機能やフレームワークについて話します。
　シンプルなCSSに対して複雑なことが可能になるSassの魅力について
　知っていただきたいです。

　＜武田さんの紹介＞
　広島工業大学3年の武田寛です。
　情報学を専攻しており、webディレクターを目指して日々勉強しています。
　現在は就職活動中でWeb業界に貢献していこうと思っております。

　Facebook : https://www.facebook.com/takecha.mf
　Twitter : https://twitter.com/Hiroshi_squall

　

●セッション3(40分)
　『EC-CUBEで仕事をするということ』システムフレンドの西村さん

　＜主な内容＞
　EC-CUBEもバージョンアップを重ね2.13になり、
　この度「オープンソースECサイト構築ソフトEC-CUBE公式ガイドブック」も
　2.13に対応したバージョンが発売されました。
　これまで何度かEC-CUBEのセッションをさせていただきましたが
　久しぶりにEC-CUBEの今をお伝えしようと思います。
　弊社システムフレンドの取組み、新機能で市場がどう変わったのか、
　新機能で苦労した点などを紹介します。
　あまり活動していないEC-CUBE公式エバンジェリストが
　久しぶりに仕事をします！！

　C-CUBE公式ガイドブック―オープンソースECサイト構築ソフト 最新Ver2.13対応

　＜西村さんの紹介＞
　職業はプログラマーです。

　ブログ：眠るシーラカンスと水底のプログラマー
　Twitter：coelacanth
　Facebook：西村 誠

　CodeZineで速習Windowsストアアプリを連載中
　書籍「基礎から学ぶWindowsストアアプリ開発」基礎から学ぶ Windowsストアアプリ開発
　「プロのためのWindowsストアアプリプログラミングガイド」ソフトバンククリエイティブ
