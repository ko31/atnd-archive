description

オリジナルブランドの化粧品、スキンケア用品を作りませんか。

折角アロマテラピーの資格や、ハーブの検定に受かったのに生かせる場がないとか、ショップやネットでイベントや開業などを予定していて、オリジナルな化粧品を販売したり、プレゼント配布してみたい人。
オリジナルブランドで化粧品や、メイクアップ用品、スキンケア用品や香水など作ってみませんか。

販売が目的でなくても大丈夫です。

100から1000本ぐらいの小ロットでも製造できますので、配布用のノベリティやグッズとしても利用できます。

知識ゼロ、経験ゼロでも問題ありません

  • 下記のようなことを今回はお話します。
    1. 化粧品成分の読み方の基礎知識
    2. アロマやハーブ、植物エキスの機能の基礎知識
    3. ＯＥＭでの製造に関する基礎知識

詳細は、 こちら を参考にしてください。

化粧品はハンドメイドで作ることはできますが、売ることや配布することはできません

薬機法により、販売や配布を目的とする場合、専門の許可された工場で製造する必要があります。※販売に関しては製造元のライセンスにもよりますが、通常のＯＥＭ委任先での製造の場合、販売に資格は必要ありません。
