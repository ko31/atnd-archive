description

初学者向けのExcelVBAの勉強会を那須塩原で開催します。

「ExcelVBAを仕事で活用したい」と思っている方、「プログラミングを始めたいけれどもどのように学習すればいいか分からない」と悩んでいる方、「キャリアアップのためにExcelVBAを学びたい」という方、みんなでExcelVBAを学んでいきましょう。

概要

プログラミングが初めての方でもわかりやすいように、市販の入門書を用いて実際にプログラミングを書きながらExcelVBAを学習していきます。

必要なスキルについて

  • 以下の基本的なスキルがあれば参加可能です。
    1. Windowsの基本的な操作ができること
    2. Excelの基本的な操作ができること
      (Excelの関数、VBAの知識が無くても大丈夫です。）

必要な持ち物について

  • 以下の条件を備えたノートパソコンをご用意ください。
    1. OSがWindowsであること
    2. Excelがインストールされていること

※ 市販の入門書はこちらで用意いたします。
※ 上記の条件を備えたノートパソコンをご用意できない方はこちらでご用意することも可能です。
