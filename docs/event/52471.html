Webディレクター向け講座『これから重要視される"コンテンツ"の話をしよう』セミナー

「と、コラボ」とは？ このセミナーは一方的に話を聞くセミナーではなく、自分から参加し、学び、発言していくセミナー。

企業と企業がコラボし、クリエイターとクリエイターがコラボし、スキルを高めていく場、「と、コラボ」です。

マーケティングだけでなく、SEO でもよく耳にする「コンテンツ」という言葉。良いコンテンツを作りましょうと言いますが、何をもって『良い』と判断するのか難しいことがあります。誰が作るのか、何が必要なのかハッキリしないままでは作ることはできませんし、作ったら終わりというわけではありません。Web サイトやアプリ開発の行程で最後のほうまで放置しておくと大変なことになるのが"コンテンツ"です。
今回はこの"コンテンツ"を扱う仕事にフォーカスし、分析・提案・そして設計のためのヒントを紹介します。
本講座は下記のような疑問を抱いている方に最適な内容です。

- なぜコンテンツの仕事が注目されているの？
- 具体的にどのような作業が必要なの？
- ユーザー視点でデザインすれば良いんじゃないの？

Web ディレクター、Webデザイナー、プロデューサーという職域を限定した話ではないので、コンテンツって何？コンテンツの仕事について知りたい！と考えている方やまだ詳しくは分からないけど、コンテンツの仕事に興味があるという方におすすめです。

クリーク･アンド･リバー社へすでにご登録がお済の方はセミナー参加費無料！また、まだご登録がお済でない方もご登録をしていただくと無料で受けられるセミナーです。今後のキャリアアップやステップアップにお悩みの方にはキャリアカウンセリングも行っております。
