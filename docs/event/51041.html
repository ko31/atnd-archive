ハプティクス技術セミナー

～ 力覚デバイスPhantomを使用したプログラミング ～

2014年5月30日(金) 14:00-16:30
株式会社スリーディー（新横浜）

はじめての力覚プログラミング。PC上のCGモデルをさわってみたい！

Phantomデバイスによる力覚プログラミングを始められる方を対象にしたセミナーです。
これから力覚プログラミングにチャレンジされたい方のご参加をお待ちしております。

こんな方におすすめのセミナーです！

  • ハプティックデバイスを使用したプログラミングをこれから始めたい
  • 開発ツール OpenHaptics の基本的なプログラミング方法を理解したい
  • 動的な干渉シミュレーションを設計支援ツールとして活用したい
  • 医用システムに関する研究をしたい
  • 研究のネタ探し (学生様のご参加も歓迎！)

研究キーワード

  • ハプティクス(触力覚インタラクション)
  • ロボティクス
  • バイオメカニクス

プログラム

時間 プログラム内容
14:00-14:15 ご挨拶・概要説明
14:15-15:30 Phantomデバイスによる力覚プログラミング
Phantomデバイスの開発ツールキットである OpenHaptics を利用した、簡単かつ高度なプログラミングテクニックについて、これから使用される方向けに資料(机上)をもとに解説します。
15:45-16:15 ハプティック技術を用いた応用開発事例
当社のハプティック技術を用いた開発事例 (医療:手術シミュレーション,製造:干渉シミュレーション など) を紹介します。
16:15～ 個別デモ（自由参加）
3次元力覚入出力デバイス Geomagic Phantom ＜ジオマジック ファントム＞
ユーザーがPC上の仮想オブジェクトに触れて操作できる高精度の力覚インタラクションを実現するデバイスです。
リアルタイム干渉計算ソフトウェア SmartCollision ＜スマートコリジョン＞
高精度な衝突判定と、衝突判定後の適切な処理(干渉の解消・力覚呈示)を実現することを可能にした、リアルタイム干渉計算ソフトウェアです。

※1つのセッションは、質疑応答時間を含みます。
※開始・終了時刻やプログラム内容を変更する場合がございます。
