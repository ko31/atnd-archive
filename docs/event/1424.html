5月と8月の2回、特にテーマを決めない形での WordPress 勉強会を開催し、大勢の方に来ていただき大変活況でした。そのときの反応をふまえて、今後は定期的にセミナー+懇親会の形式で小イベントを開催したいと思います。

このイベントの概要、趣旨

WordPress に関する草の根的なイベントです。
セミナー(約2時間) + 懇親会(約2時間) の2部構成でおこないます。
セミナーはあらかじめ決まったテーマで講師が話をします。
懇親会は飲み食いしながらの自由なトークタイムです。日頃困っていることなど、話題はなんでも OK です。
参加費は2,000円です。うち1,000円は飲食代(近所のスーパーで買い出し)にあてられます。
今後は隔週くらいのペースで毎回違ったテーマで続ける予定です。

第1回 (9/12) について

テーマ: WordPress のテーマの仕組みについて。テンプレートデザインで役に立つ話。

日時: 9月12日(土) 17:00 – 21:00
場所: AIP cafe (わかりにくい場所にあるので初めての方は地図をよく見てお越しください)
