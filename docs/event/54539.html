鷺沼Book倶楽部始動！　集まれ、本好き！

鷺沼Book倶楽部は、鷺沼近辺に在住の本好きのための活動です。毎月１回、鷺沼駅周辺のレストランや貸し会議室で集まります。ルールは簡単！　毎回のテーマに沿ったあなたのお気に入りの本を１冊持ってくるだけ。

鷺沼Book倶楽部は、本の紹介ゲームであるビブリオバトルの公式ルールにのっとって活動しています。

2014年8月27日（水）
受付：午後6時45分　開始：午後7時　終了：午後9時半ごろ
