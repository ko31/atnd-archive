description

キラリと光る地域には、地域を引っ張る頼もしきリーダーが存在する。

「ヒムカレッジ」は、地域の課題と真っ向から向き合い、
先駆的な取り組みを実践しているリーダーたちから、地域にある課題に対する
アプローチの仕方を学ぶ月１回のカレッジ（定期講座）です。

地域の「今」を知り、これからの宮崎、そして地域づくりについて
ちょっぴりマジメに、だけどフランクに語り合いませんか？

講座の概要
