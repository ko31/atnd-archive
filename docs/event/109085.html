description

本セミナーは、難解な数式やプログラミング言語を一切使わず、図と例え話だけで、ビジネス現場のデジタル化（DX）に必要なローカル5Gの基礎から応用まで、すべて解説します。

「えっ、そんなことできるの？」と一見、滑稽に感じるかもしれません。

たとえば、難しい専門書を読み込んだあと、頭の中でいろいろ咀嚼しながら整理し、ああ、そういう意味だったのか……ガッテン！」と思わず納得し、感動する瞬間があると思います。

つまり、時間をかけて5Gの難しい無線通信理論やモバイルシステム技術を学ばなくても、明確なイメージを掴むことによって5Gの本質とAIとの親和性を理解することができれば、ローカル5G×AIによるビジネスのデジタル化をスタートアップさせることができます。

本セミナーでは、5Gを含むモバイル通信からIoT、AIまで広い知見を持つ講師歴30年のGVA理事がわかりやすく解説します。
