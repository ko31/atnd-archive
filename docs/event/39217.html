jsCafeの紹介

jsCafeは、以下の特徴を持ったjavascript勉強会です。

  1. 基本的な所から勉強していく
  2. 誰でも気軽に参加／発表出来る
  3. 最後に皆でアウトプット出す

今後

1)勉強会

  • デザイナー：jQueryMobile等jQueryプラグイン、CSS Framework
  • 開発者：node.js、jasmine等test系tool

2)アウトプット

  • 最後に皆でWebアプリ開発
