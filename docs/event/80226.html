POP作りの経験がない方でも、デザインが苦手な方でも、iPadを使ってお店のPOP作成ができるようになります！無料アプリ「POPKIT」を使ったPOP作成講座です。
iPad、POPKITを使ったことがない方でも、講師が丁寧にサポートしますのでご安心ください！

■セミナー内容
POPの基本的な考え方を習得し、POPKITを使ったPOP作りができるようになります。
第1部:POPの役割と作成のための6つの基本ステップ、POPKITの使い方
第2部:POP作成実技（作成したPOPはお持ち帰りいただけます）

■こんな方にオススメ
POP作成初心者、iPad初心者、店舗のPOP担当の方、企業の広告・販促ご担当者、その他POPにご興味のある方

■受講資格

■受 講 料

ご自身のiPad（iPad2以降のモデル※カメラがついていればiPad2以降です）を必ずご持参ください。
