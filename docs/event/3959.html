4/28開催　USTREAMER養成講座 初級編

description

世界にどこでもドアを開けていく中継サイト
”Ustream“がブレイク間近です。
2000年に世界一周する洋上からの
インターネット放送を実現。
ソーシャルカレー、ソーシャル引越し、ソーシャル徹夜など
先駆的な取り組みをしてきた
ヒマナイヌの川井氏がスタートした
日本初のUSTREAMER養成講座。

USTREAMERとは最小限の機材を使ってひとりで
中継できるスキルを持ち
かつソーシャルストリームに流れるコンテキストを
現場にフィードバックして
中継を立体的に進行していく
デジタルモデレーターの能力を持った人材です。

初級編では基本的なUstreamの機能を理解し、
webカメラとノートパソコンでの
基本的な中継機能を習得します。
中継にビュアーを呼び込む重要な機能である
ソーシャルストリームの扱い方、
ハッシュタグの設定の仕方などにも触れます。
講座内のミニワークショップでは来場者と
共に画面の中にアクセス可能な
このメディアならではのユニークなコンテンツの
可能性を探ります。

本講座を受講することでその日からUstreamを
使って新しいライブコンテンツの
時代に参加することができます。お気軽にご参加下さい。

日時：2010年4月28日（水曜）19:00 開場 19:30〜21:30 (2時間)
場所：Co-Net by battanation（中央区日本橋小網町）
受講料：3000円
　　　　講座終了後に懇親会があります（懇親会参加費2000円）
　　　　懇親会フード担当／Rockaku
定員：30名
講師：川井拓也（ヒマナイヌ）
　　　岩沢卓(Co-Net by battanation)

講師プロフィール

ヒマナイヌ：川井拓也
TV-CM、映画、CG、テレビ番組、ウェブ、デジタルサイネージ、ケータイから
雑誌、書籍、ソーシャルメディアまであらゆるメディアのディレクション経験があり
主な書籍に「SNSビジネスガイド」「Twitterマーケティング」などがある。
Twitterでは気づきの犬印で9000人のフォロワーにライフログを発信している。

Co-Net by battanation:岩沢卓
VJ・映像制作の経験を活かし、youtube/ustreamなどを活用した
情報発信を行っている。
オフィス街日本橋にAltanative Office"Co-Net"を開設し
イベント・トークショーなども企画している。
