InfoTalk#24: クラウドプラットフォームの活用事例

InfoTalkは，ICT関連の熱い技術，面白い活用等を取り上げ，いろいろと議論したりする場（勉強会と懇親会）です。今回は，クラウドプラットフォームの活用に関する興味深い話題をいろいろと提供していただきます。産業技術大学院大学の関係者以外の御参加も歓迎いたします。是非，御参加ください。

御願い等

  • 今回も御陰様で多数の参加申し込みいただき，ありがとうございます。
    勉強会の資料とか，懇親会の準備の関係がございますので，急用等で欠席されるときは，ご面倒でもATNDから取り消し手続きを御願いします。
  • 今回は当初の定員以上の申し込みをいただきましたが，会場には若干残席がありますので，残席は懇親会の参加者を優先したいと思います。
    このATNDで「補欠」として登録し，懇親会のほうにも登録してください。
    システムの制約から，当ATNDの登録は補欠のままですが，勉強会にご参加いただけるように処理します。
  • 受付開始は18時頃からです。今回は参加者が多いので，当日は混雑すると思います。
    ちょっと早めに到着いただけると助かります。いろいろ準備してますが，すみませんが，いろいろご協力ください。
    受付では参加登録時に使った「姓名の頭文字相当のアルファベット2文字」が必要です。あと，名刺を持参ください。

講演概要

クラウドを活用したアプリケーション開発事例

  • 講師
    • 遠藤 博樹 氏＝株式会社ワークスアプリケーションズ （産業技術大学院大学 2007年度 修了生）
  • 概要
    • クラウドというキーワードは頻繁に耳にするが、これまでと何が違うのか、メリットは何か等がわかりにくい。今回は、パブリッククラウドであるAWS（Amazon Web Services）を開発環境として利用し，実際にアプリケーション開発を行った事例を取り上げ、クラウドを利用する上でのメリット，デメリットを紹介する。

クラウドが実現する情報システム

  • 講師
    • 馬場 俊彰 氏＝株式会社ハートビーツ CTO（産業技術大学院大学 2年生）
  • 概要
    • AWS（Amazon Web Services）あるいは国内クラウドで利用し，各種システムを構築・運用してきた経験から，クラウドの活用事例，活用ポイント，今後の展開をエンジニア目線で紹介する。また，AWSが拓いたエンジニアの未来展望にも言及したい。
