Titanium mobile

iPhoneアプリに興味を持ち、いざ開発しようとプログラム勉強を始めたとき、「Objective-C」の難解さでつまずいた…そんなことはありませんか？

「Titanium mobile」（Appcelerator社製）は、Web出身の人には馴染みのある Javascript を使用して、簡単にiPhoneを作成することができ、しかも同時にandroidアプリも作ることができるとてもお得なソフト。

全くはじめての人でも、1日で、実際にアプリを作ろうという、（ムチャな?!）講座です。

みなさんのご参加をお待ちしています。

講座内容

（操作の仕方〜基本的な構造〜パーツ説明〜サンプルアプリ）では実際にソースを手打ちしてアプリを制作します。
