description

9月11日、渋谷のカフェスペースを借りて1日カフェを開店します。

カフェスタッフが自慢の料理やデザートを振舞います。
他にも簡単なアロマテラピーをしたり、こだわりのフリーペーパーの展示、手作りアクセサリーを展示したり、それぞれのスタッフが思い思いの方法でおもてなしいたします。お酒やコーヒーを飲みながら日曜の昼下がりをのんびりお過ごしください。

ただ、このカフェは参加者の皆さんがカフェ作りに携われる様にもしたいと思います。

例えば、カフェでかけてみたい音楽があればCDをお持ちいただければかけられます。音楽がきっかけで話題がもりあがって友達の輪が広がればいいなと。
また、ブックカフェっぽく、お気に入りの本、みんなに読んでもらいたい本があれば、自作のポップを書いて置いておくこともできます。読書カードなんかも挟んでおけば、みんなの感想なんかも聞くことなんかもできます。
ですので、是非お気軽にCDや本・雑誌などをお持ちください。

他にも何かカフェでやりたいことがあれば、時間と場所の許す限りではございますが、実施できればなと考えておりますので、是非ご相談ください。

カフェ作りに興味のある方
とにかく何か楽しそうなことに興味がある方
趣味のあう友達を作りたい方

それぞれの楽しみ方で参加していただければと思いますので、お気軽にご参加ください。
