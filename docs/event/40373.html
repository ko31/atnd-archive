イベント内容

  • Windows 8/RT および、8.1 のストア アプリを作成するハッカソンです。
    世界最速の"8.1"ハッカソンになりそうです。
  • 初心者向けには、アプリの作成方法からストアに公開する方法までをワークショップ形式で解説します。
  • 経験者や開発者登録をお持ちの方は、参加者どうしで情報交換しながら、自由にアプリケーションを作成していただきます。
  • Windows 8.1 Preview と Visual Studio 2013 Preview がダウンロードできるようになったので、本気で、8.1のプログラムを作っていただいて構いません。キリッ。

修正履歴

  • 2013-06-05 イベント公開

ハッカソンとは

  • ハッカソン（Hackathon）とは、ハック（Hack）とマラソン（Marathon）を組み合わせた造語ですが、このイベントでは、ハックとは「プログラミングする」という程度の意味で使っています。つまり、みんなで集まり、アプリの開発・発表をする会のことです。

参加条件

特にございません。初めての方、初心者の方、大歓迎です！
デベロッパーの方、デザイナーの方、興味のある方ならどなたでもご参加頂けます。

用意していただくもの

  • 特にありません。（自分のPCを持参した方が開発しやすいという方は、持参してください。）
  • PCを持参してもしなくても、会場に用意してあります。（Core-i7/8G/SSD。ただしタッチ式ではありません）
  • Win8.1 preview や VS2013 preview が公開されたので、頑張って、会場PCにインストールしました。

お題

  • 「梅雨を吹き飛ばせ」あるいは「梅雨を楽しもう」のどちらかを選択。
