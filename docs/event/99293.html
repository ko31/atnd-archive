■内容
１００円ロボット部とは？
100円グッズと低価格なマイコン等でロボットを作っていく部。
（マイコン使わなくてもよい）
今回は「自由研究する」をテーマにアイデア出し、作成。
制作のアイデアが浮かばない方は赤べこの紙製ロボット版「ベースベコ」
を作る形になります。
Tello等の小型のドローンも飛ばせます。

■流れ
１）テーマに関してのアイデア出し。
２）各アイデア発表（過去作の自慢でもいい）
３）新規の場合は解決方法を募集。
４）材料１００円ショップ買い出し＆ラフ作成
５）評価

■参加資格
１）ロボットを作りたい人
２）他人を尊重する人
３）高価な機材を自慢しない人（笑
４）検索結果で話をしない人　ひとまず、やる人
５）あんまり神経質じゃない方

大人のみでも参加可能！！

■申込方法
こちらの申込に保護者の名前にて、申請して下さい。

■著作権について
発生する権利に関しては、各自のものですが、100円ロボット部で管理、調整するものではありません。
発生したアイデアや仕組みに関して、100円ロボット部を広める為に掲載・公開することに了承下さい。

■注意事項
制作時に発生したケガや破損事故に関しては自己責任になります。
運営上の注意事項に従って頂けない場合は退場して頂く場合があります。
写真撮影等の取材がありますのでご了承ください。
制作で出たゴミは持ち帰りを基本とします。
