　ビジネスチャンスのキーワードが「英語」だと感じていませんか？

◆今は英語が話せないけれど、話せるようになりたい

◆自身の事業を海外へ発信したい

◆facebook等で海外の人と英語で交流したい

◆海外のビジネスパートナーを見つけたい

□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■

2020年オリンピック開催までに年間2000万人の外国人観光客を受け入れることを目標に、政府は幾つかの入国制限の緩和を発表しています。
英語が話せる、外国人の友達がいるなど、グローバルな活動のチャンスが あればいいのですが、何も無い場合はどうすればいいのでしょうか？
『ワールドビジネスガッツ』は参加者の皆様とのデスカッションを通してグローバル化のアイデアやヒントを共有しあう会となっております。

◆―――――――６月～８月のテーマは―――――――◆

【クールジャパンで広がるビジネスチャンス】
（１つのテーマを基にデスカッション　3ヶ月おきに完結します）

○日本から海外に発信できるサービス・商品づくりとは？
○日本を訪れる外国人の方々へ提供出来るサービス・商品とは？
・・・そのビジネスの組み立て方とは？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

今回は
「グローバル・コミュニケーションスキル」を開催します。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

英語圏ではミーティングの切り出し方から異なります。
話題の作り方など、海外の方とのコミュニケーションスキルを身につけましょう。
実際に外国人講師と一緒にデスカッションを行います。
もちろん毎テーマ１回のみのご参加もOKです。

※英会話が得意でない方でも、前回参加されていない方でも参加しやすいかたちに工夫しています。
