第17回Android勉強会in札幌

title 第17回Android勉強会in札幌
catch 今回はゲーム祭り
description

今回は、ゲーム開発特集を考えています。
Androidにおけるゲーム開発初心者から中級者を対象とした内容になっていますので、いつもは業務系ソフトを作っているんだけど実はゲームに興味が有るような方、または学生さんの参加をお待ちしております。
内容は、主にアクションゲームを意識したものになると思います。(SNSゲームなどはまた別の方法論になろうかと思います)

また後半には北海道におけるAndroid周辺事情や、(Androidに限定せず)アプリマーケティング上の生の声を、パネルディスカッションを通して討論してみようというお試し企画もあります。
どちらかと言うと経営層や営業向けの話となるかと思いますが、なかなか無い機会だと思いますので参加検討下さい。
パネリストとして参加されたい方はmaimuzo@gmail.comまで連絡下さい。
Hangout経由で中継予定なので、ネット越しのパネリスト参加もOKです。
またHangout Airで公開予定です。(ただ会場からの実績ないのでコケたらごめんなさい)
