想定外をなくす 実践 安全分析(HAZOP)セミナ

title 想定外をなくす 実践 安全分析(HAZOP)セミナ
catch 津波、放射線を題材に
description

震災では想定外をなくすことがいかに大切かを技術者の方々は認識されていることと思います。想定外をなくし、潜在的な危険を洗い出す国際規格になっている手法HAZOP(hazard and operability study)があります。HAZOPは表のように対称性のある誘導語を用いて，設計者から利用者までの幅広い方の参加によって想定外をなくす手法として利用できます。IECの国際規格であるFTA,FMEAの後で国際規格になったHAZOPは、分析できる手法として航空宇宙産業、自動車産業などで幅広く用いられるようになってきました。名古屋市工業研究所では、2008年よりHAZOPの技術者教育を始めています。今回は、津波、放射線を題材に、演習形式で、HAZOPの演習形式、グループ議論によるセミナを実施することになりました。システムの設計者，防災担当者，原子力担当者の方を初め，幅広い方の参加をお待ちしています。

日時：平成22年5月10日（火曜）13：00―17：30
