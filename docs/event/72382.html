description

　環境NGOの環境教育フォーラム21は12月のeco検定合格に向けて、東京、大阪、名古屋、川崎で模擬問題対策の「直前ポイントアップ講座」を開催します。検定試験にでやすい問題を試験に出やすいテーマ別に解きながら、弱点を克服し、得点力を10点以上アップ、合格ライン突破を目指します。試験にでやすい環境問題の時事ニュースも問題形式で習得できます。自宅で一人ではなかなか勉強が手につかない、実力を試したいという方におすすめの講座で、これまで参加された皆様からは「合格に大変効果があった」と好評を得ています。

※開催時間は各会場とも原則午前10時から午後4時半まで　12月9日は午後1時、11日は12時半～
