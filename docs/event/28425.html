description

　ものづくりカフェ（仮）の第一回目の公開日が決まりましたので
　報告させていただきます。

　５月４日（金）１２：００～１８：００
　急な日程ですみません（＾＾；）

○場所
　ものづくりカフェ（仮）はでんでんタウンの吉野家の隣の三越ビル２階にあります。
　ビルに入って右手の階段を上ったあと、左手に進んだ一番手前の部屋です。（ドアに表記あり）

○公開設備について
　まだ設備と利用規約が不十分でなので、すぐ使用して頂ける設備はほとんど無いのですが
　まずはスペースの公開を通じてものづくりカフェ（仮）を紹介させて頂ければと思っています。

　ものづくりカフェ（仮）は、まだまだ未完成で物の少ない空間です。
　ただ、それは裏返せばこれからどんどん設備を充実させるという
　夢や希望が詰まってるとも言えます（笑）
　よろしければその辺りのメンバーの思いも聞いて頂ければ幸いです。

○おまけ
　ものづくりカフェ（仮）では、
　備品としてコーヒーメーカーのバリスタを導入しています。
　希望者の方には無料でコーヒーを提供させて頂きます。

それではお時間のある方は気軽にお越し頂ければと思っておりますのでよろしくお願いします。
