・プログラマーさんをはじめ、普段お絵かきやベジェとは縁は無いけど興味はある…
…でも描けないor描き方がわからないor描く勇気が無い、といった方向けの
ゆるいお絵かき勉強会です。

・まずは簡単な図形でウォーミングアップをし、
最終的にはちょっとしたアイコンを作る、といったステップアップタイプの流れを予定しています。

・飽くまでベジェ曲線を使ってお絵かきができるようになることが今回の目的です。
特定のツールを使いこなす為のイベントではありませんのであしからず。

・当日の映像配信は予定しておりません。

【必要なもの】
・Adobe Illustrator、Adobe Photoshop、Microsoft Expression Design、Microsoft Expression Blend、Inkscapeのいずれかがインストールされたマシン
（※体験版でも構いません。　※これら以外にもマウス操作でベジェによる描画ができるソフトをお持ちの方はそれでもOKです。）
・500円
（※お店の都合上ワンドリンクオーダー制となっています。何卒ご理解の程お願い致します。）
