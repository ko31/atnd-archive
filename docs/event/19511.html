食べるもよし飲むもよし、土手を転がるもよし！秋の晴空の下で、芋煮しませんか？

毎年欠かさず開催している WACS 芋煮も今年で3回目！
食べるもよし飲むもよし、土手を転がるもよし！秋の晴空の下で、芋煮しませんか？
恒例の仙台・山形芋煮食べ比べ、なぜかいつも海の幸が豊富なバーベキュー、そして、準備から参加で女子力アップ！？
最後に、これだけ言わせてください…
「芋はにて食え！」
クリエイティブに係る全ての方が、（ほぼ）手ぶらで参加可能！ご家族、友人との参加も大歓迎！
周りにいて、ウズウズしてる仲間をお誘い合わせのうえ、奮ってご参加ください！

開催概要

参加登録がうまくできない場合は、メールにてご連絡ください。折り返し登録申し込みのフォーマットをお送りいたします。
organizer@wacs.jp

開催日時：
2011/10/10（月・祝日）

9:00 準備
11:00 芋煮
（食べたり飲んだり、転がったり）
16:00 解散

  • 準備からお手伝いいただける方は 9:00 過ぎに会場にお越しください。ご褒美として、早めに乾杯できます！
  • 事務局スタッフは 8:00 から設営してるはずです。
  • 途中参加、帰宅ももちろん可能です。
  • 雨天中止となります。中止の場合は事前にこのページ、オフィシャルサイト、Facebook、Twitter でお知らせします。
場所：
七北田公園の河原

  • 公園奥の河原が会場です。わかるようにサインボードを掲げておきます。
  • 会場には有料駐車場があります。ただし、当日は混雑が予想されますので公共交通機関のご利用をお勧めします。
  • 会場まで歩いて10分、仙台市営地下鉄泉中央駅が便利です。
参加費：
2,000円（中学生以下無料）

  • 事務局で食べ物、飲み物は準備しますが、もちろん差し入れ大歓迎です！
  • 会場に到着しましたら事務局スタッフにお支払いください。領収書をお渡しいたします。
  • 途中参加、帰宅の場合も金額は変わりません。
料理（予定）：
  • 仙台・山形芋煮
  • 海の幸たくさんバーベキュー
  • 秘伝の味噌を使った焼きおにぎり
  • 秋の味覚 and more …
飲み物（予定）：
  • ビール（主に新ジャンル）
  • カクテル
  • チューハイ
  • ノンアルコールビール
  • ソフトドリンク
  • レッドブル and more …
その他：
  • 寒さ、雨対策は各自お願いします。
  • 飲酒運転は厳禁です。飲酒される場合は、必ず公共交通機関などをご利用ください。
  • ゴミは事務局で準備するゴミ袋へきちんと分別して捨ててください。
  • 他の参加者や利用客に迷惑をかけないよう、節度も持ってはしゃぎましょう。
  • 商品の販売、勧誘などを目的とした参加はお止めください。
