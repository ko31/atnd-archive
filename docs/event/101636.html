『離職者率Down』『利益、業務効率Up』のための働き方改革セミナー

主催： 株式会社賢嶺

内容

『離職者率Down』『利益、業務効率Up』のための働き方改革セミナー 人材のスキルや能力に依存しない、組織、チームで結果を出す手法を学ぶセミナーになります。

株式会社賢嶺では、 11月26日（月）NLC新大阪8号館にて、 従業員のやる気を高め、業務効率を向上させ、利益を高める 『目に見えて結果が出る働き方改革セミナー』 入門編を開催します。

働き方改革を改めて考え直し、時短、有給消化だけではない、 組織のためにもなり、働く方のためにもなる方策を学びます。

参加者同士で意見交換をし、今後の人脈作りも目指します。

【セミナー内容】

  1. そもそも働き方改革の目的を考え直す
  2. マネジメント力の向上が重要
  3. 他社のムダを学び、対策を考える
  4. 対策の具体例（コラボ・アウトソーシング・断捨離）
  5. ルールとマナーを使いこなす
  6. 対人関係性構築
    1. 管理職は偉いわけではない
    2. 確証バイアス
    3. まずは、会議を変える
    4. ルールを変える
    5. 働き方改革チームを構成し、権限と責任を与える

　　17:00 名刺交換・意見交換会

　　17:30 セミナー終了予定

【講師】

株式会社賢嶺　代表取締役

カイロスビジネスデザインワークス　代表

山岡　哲也（やまおか　てつや）

　1969年4月生まれ。43歳。 2児の父。

　三重県津市出身。京都大学工学部卒。同大学院工学研究課修士過程修了。 卒業後、大手ゼネコン清水建設に入社するが、2年弱で退社。 その後、「自ら自分の仕事を創る」と志し、 衆議院議員選挙の選対本部への参加から、 ラーメン店の店舗開発・運営、大手訪問販売の営業を経て、 モチベーション維持の大切さやコミュニケーション能力の重要性を知る。 その後、大手教育会社の協力会社にて、人材派遣業界を経験し、 家庭教師派遣業の立ち上げと、中国への進出を責任者として重責を全うし、独立。 現在、マネジメントや働き方改革、業務改善に関するセミナー講師やその支援を行っている。
