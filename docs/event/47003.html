■GlobalGameJamとは

Global Game Jam（以下、GGJ）は国際的なゲーム開発者団体であるIGDAが主催する、世界的なゲーム開発イベントです、世界各国の会場に集まったゲーム開発者たち（Jammer）がチームを結成し、48時間で提示されたテーマにそってゲームを企画開発し、世界中のゲーム制作者と競い合います。
2009年から開催され、今年は5回目となります。前回の2012年大会では世界47カ国、242の会場に10,684人が参加し、2,209のゲームが開発されました。今年は2014年1月24日～26日にかけて開催されます。
