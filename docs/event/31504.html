■イベント概要

2010年から続いているSocialTopRunnersを開催します！過去発表企業は、gumi / KLab / Drecom / Kayac / スクウェア・エニックス / gloops / pokelab / colopl / CyberX / Sumzap / nijibox / Crooz / akatsuki / その他トップSAP企業。業界のトレンド、開発、企画、マネタイズ、そして戦略のリアルを体感するイベントです！イベント後の懇親会では参加する開発者同士の情報交換も重要ですよね！このイベントからソーシャルゲーム業界が始まったと言えなくもない、要チェックイベントですよ〜！

■登壇者

（１）Pokelab社

　Androidゲーム、iOSゲームで日本のトップを疾走中！ソーシャルゲーム初期に参入し、若いチームで作り上げた実績とノウハウはスマホ時代のビッグウェーブに乗って更なるフェーズに展開中です！開発面、企画面、マーケティング面、戦略面で一歩先行くポケラボの今を、代表取締役社長 前田様にお話いただきます！業界随一のナイスガイです！

（２）スクウェア・エニックス社

　日本のゲーム史上、燦然と輝くブランドタイトルを輩出してきたトップ企業の「スマホ戦略」を語ります。ソーシャルゲーム業界で大ヒットタイトルを産み出した名プロデューサーの次の戦略をダイレクトで、生で、伺いましょう〜！HatchUp的に大喜び、業界的に大注目の発表です！
