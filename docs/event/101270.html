『ATEAM TECH MeetUp』を開催！テーマは「見せます!Webサービス開発を支えるミドルウェア・ツール」！

スマホ向けゲーム開発や日常生活に密着したWebサービスなどを運営するエイチーム。名古屋で定期開催しているWebエンジニア向けカジュアル勉強会（トークセッション）＆交流会『ATEAM TECH MeetUp』を大阪(梅田)にて開催します。

エイチームのWebサービス開発で使用しているミドルウェア・ツールについて、開発事例を交えてお話します。また『ATEAM TECH MeetUp』は、エンジニアのアウトプットの機会を増やし、交流の活性を目指しておりますので、エイチームエンジニアと参加者のみなさま、また参加いただいたエンジニア同士の横のつながりを深めるために、参加者のLTセッションや懇親会もご用意しております。みなさんのミドルウェア・ツールについても、積極的に発信していただきたいと思います。

「ATEAM TECH」とは

ゲームやインターネット業界で働く技術者向けに勉強会や交流、アウトプットいただける場を設け、新しい気づきや成長につながるような機会を提供することで、技術力の向上や業界のさらなる発展を目指します。業界の技術トレンドに関する共有や交流の場を通じてより深い知識を得て、さらには参加者同士でのつながりもできればと考えております。

エイチームライフスタイル 金融メディア事業部 エンジニアチーム

得意技術分野は、バックエンド（PHP）

セッション概要

規模の大きいデータベースにおいてデータベースのスキーマ変更をするには、従来はサービス停止を伴うメンテナンスが必要でした。オンラインスキーママイグレーションツールの導入によって、サービスを停止する事なくマイグレーションする方法についてお話します。

セッション登壇②：サービスを支えるジョブ管理システム

発表者：kytiken

エイチームライフスタイル 金融メディア事業部 エンジニアチーム

得意技術分野は、バックエンド（Rails）

セッション概要

エイチームライフスタイルでは複数のサービスを運営しており、その裏ではジョブ管理システムrundeckが活用しています。rundeckの導入に至った背景とrundeckがどのように活用されているかを紹介します。

エイチームLT登壇：サービスで活用したRailsパフォーマンスチューニングツールの話

発表者：sakupa

エイチームライフスタイル 金融メディア事業部 エンジニアチーム

得意技術分野は、バックエンド（PHP）

LT概要

エイチームライフスタイルが運営しているWebサービスで実際に活用したパフォーマンスチューニングツールについてお話します。実際に直面した問題での活用事例とツールの特性をご紹介いたします。
