イベント概要

今年は新年会、CloudStack Days Tokyo などのイベントを実施してまいりましたが久しぶりに普通のユーザー会を実施したいと思います。 今回のトピックとしては実際に利用しているユーザー事例や、6月にLPI Japan様よりリリースされた Apache CloudStack技術者認定試験(略称: "ACCEL") の紹介を予定しています。

現場での CloudStack のユースケースを知りたいIT基盤管理者の方、オープンソース クラウドに初めて触れる初心者の方など是非ご参加下さい!

CloudStackおよびユーザー会について

CloudStackは、Apache Software FoundationのOSSプロジェクトとして開発されており、IaaS型クラウドを実現するソフトウェアとして、多くのクラウド事業者や 北海道大学アカデミッククラウド をはじめとした学術機関などで実績があるソフトウェアです。詳細：Apache CloudStack Project

日本CloudStackユーザー会(JCSUG)は、日本語による CloudStack に関する情報発信、情報共有を行い、CloudStack の普及および人材育成に貢献することを目的として活動しているコミュニティです。詳細はユーザー会Webサイトをご覧ください。
