ゲーム開発もくもく会

ゲーム制作もくもく会 は、12/23と12/26の二日の9:00-21:00に、みんなで集まってもくもくとゲームを書く会です。
開発環境・対象プラットフォームともに自由です（開発に使う計算機は各自準備してください）。

両日19:00から1時間半くらい、その日にやったことをみんなでLTみたいに発表する機会を設けたいと考えてます。
もちろん参加は自由で、相変わらずゲーム書きながら聴くくらいのラフなスタイルでいきましょう。

会場の端っこにUst用のエリアを設けて、たまに休憩がてら参加者の人へのインタビューみたいな感じのことをしようと考えています。もちろんUst映るのに抵抗がある方はインタビューを断っていただいても大丈夫です。基本的にはUst用エリア以外は映らないように調整します。

対象

  • 普段からゲーム書きたいなーと思ってるけどなかなかその機会を作れないでいる人
  • 具体的な方法は知らないけどゲーム書いてみたいなーと思ってる人

日程

  • 2010年12月23日(Thu) 9:00-21:00
  • 2010年12月26日(Sun) 9:00-21:00

やること

  • 9:00-19:00: 黙々とゲームを書き黙々とお菓子を食べ黙々と話す
  • 19:00-20:30: 黙々とLT
  • 20:30-21:00: 会場から脱出

補足

  • 途中の入退室や、どちらか片方の日だけの参加も全然OKです
  • 電源タップやお菓子・飲み物などは基本的にスタッフで揃えますが、その他あったら便利そうなものや差し入れなんかを持ち込んでいただくとテンション上がる感じでいいと思います
  • その他 雑多な内容を以前ブログで書いた のでこちらも参照のこと
