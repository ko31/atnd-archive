description

OPTPiX SpriteStudio 中級者デザイナーさん向け勉強会を、下記の日程で開催いたします。

※ SpriteStudio で一度アニメーションを作った事がある方を前提としています。
※ SpriteStudio を触ったことのない初心者の方は、2～3ヶ月に1度開催している初心者デザイナー向け勉強会をお勧めいたします。

日時 2015年9月29日（火）
タイムスケジュール 18:45　開場
19:00　勉強会 開始（途中休憩あり）
20:35　Q＆A（質問タイム）
20:45　終了（アンケートの記入）
21:15　完全撤収
場所 ウェブテクノロジ 会議室
東京都豊島区西池袋5-14-8 東海池袋ビル2F

池袋駅（JR・地下鉄各線）西口より徒歩12分 (出口C2から7分)
要町駅（有楽町線・副都心線）より徒歩4分 (出口6から3分)
設備 電源あり
無線LANあり
参加費 無料
担当 大野正樹（SpriteStudio スペシャリスト）

内容

  • OPTPiX SpriteStudio 中級者のデザイナーさん向けに、以下の内容を考えています。
    ・インスタンス機能の活用
    ・補間アニメとコマアニメの複合テクニック
    ・立体を意識したアニメーションの作成
    ・細かいテクニックいろいろ
    など…
  • プログラマーでなければ分からない用語などはなるべくデザイナーさんに分かりやすい言葉で説明しようと思います。
  • Q&Aでは、SpriteStudio 本体および PSDtoSS、TexturePacker プラグイン等のアニメーション製作に使用するツール全般のご質問をお受けしています。
  • プレイヤー、SDK等の再生プログラムに関するご質問は こちらの「SDK・プレイヤー・その他のツールに関するお問い合わせ」からお寄せください。
  • 1時間程度の基本的なセッションと事前のアンケートでいただいた内容をもとに30分程度。合計1時間30分から最大2時間までを予定しています。

その他

  • PCは必須ではありません。可能であればご持参ください。
  • 当日の進行をスムーズにするためにも、アンケートをお願いしています。勉強会で聞きたいことを気軽にリクエストしてください。
  • 席に限りがあるため、募集人数は8名です。
  • 途中参加OKです。お気軽にご参加ください。
  • 少人数なので質疑応答がしやすいと思います。お気軽にどうぞ。
  • ATND経由での申込が難しい場合は、弊社問い合わせフォームの「ライセンス、購入、その他に関するお問い合わせ」よりご連絡ください。
