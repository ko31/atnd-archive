Reblog Advent Calend*e*rとは

かつて、それはただの typo でした。しかし一世紀の時を経てそれは、名も無き者達が叫ぶ、「残り99%のためのアドベントカレンダーを！」というセリフは銀河を飛び交う電磁波として、昇華されることとなりました。そうです、それはこのインターネッツから永遠に認知されることのなかった者達の叫びです。

Advent Calendarとは

かつて地球には、クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーがありました。21世紀にはこれが転じて、ギークコミュニティで、クリスマスと呼ばれる祝日までの25日間に1つずつ箱に閉じ込められた魂を公開する儀式が行われたそうです。これを Advent Calendar と呼びました。

一世紀前の様子はこのようなものでした。

ルール

あなたが reblogger なら簡単です。

自分が「これが reblog だ!」という画像なり、記事なり、引用なり、犬なり、猫なり、をひとつ見つけてきて、reblog するなり tumblelogging するなり煮るなり焼くなりなんでもあり。
ルールは単純。
このイベントのコメント欄に何か URL を貼るだけ!
執筆順は、このイベントに参加した順です。最初の人が12月1日、2人目が12月2日…と続いていって、最後の人が12月25日です。
