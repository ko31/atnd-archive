7日間で5人でモニターとして検証を行いました。
（ルール）①毎日、作業報告をすること
　　　　　②行動すること
　　　　　③成約が1件もなければ全額返金。

結果は5人のうち4人は7日間以内に成約を取り、1人はルール①、②を守らず成約取れませんでした。
なので7日間で80%が結果が出せたインターネットビジネスです。

今主流の副収入を得るためのビジネス。アフィリエイトはブログを書くなど労力がいり、せどりは仕入れに資金が必要だったりと準備にお金も時間もかかる。

しかも上記のビジネスの講座料はかなり高額で何十万といったものが多い。
売れるまでフォローしますといいながらお金だけ受け取っておしまい。
そんな業者ばっかりです。
なので今回みなさんに知ってもらいたいということで3000円で開催します。

今回紹介するインターネットビジネスは
・知識、経験、肩書き不問。主婦や学生誰でもできる。
・営業で売り込むということはしない。
・このインターネットビジネスのモデルの市場に競合がいない。
・1日に何時間も作業をするというものでなく仕組みだけ作って収入を得る。
・パソコン、スマートフォンがあればできる。

講座の中でも話しますが実践した人は全員結果が出ました。
そしてインターネットビジネスの知識・経験がない方でも結果を出すことができるのでその売れる仕組みを教授します。

※物販やアフィリエイトではありません。
株、FX、不動産など投資とは関係ありません。
競馬、ロトなどのギャンブルとは関係ありません。
