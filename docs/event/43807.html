description

MojoliciousをPerl5.8で動作させるmojo-legacyや、I/OのMojolicious記事などでおなじみの Mojoliciousハッカー @jamadam さんが東京に滞在しているタイミングで、以前と同様の飲み会をしようと思います。幹事はYAPC::Asia Tokyo 2012/2013などでのmod_perlトークが一部で有名な私 @xtetsuji です。

場所は五反田の原価BARを予定していますが、予約が入らないかもしれませんし、定員は適当です。その時は Twitter などで場所変更等について適宜連絡いたします。

19時開始予定ですが、20時くらいまでに集まれればいいなくらいのアバウトな感じです。

予約を取るか取らないかわからないですが、原価BARの収容能力だけ人は集まれます。参加人数は暫定的に定めています。

原価BARのシステムは入場料1500円を来店時に払うシステムです。最初はビックリするかもしれませんが、後の飲み食いは文字通り原価で食べられるので、総額としては3000円程度で満足できるんじゃないかなと思います。

急遽開催なのですが、Mojoliciousやmod_perlに興味のあるPerlハッカーの方、また単に飲みたいPerlエンジニアの方、お待ちしています。
