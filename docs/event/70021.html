description

９月１２日（土）の映画講座でメイク講座を開催します。映画、およびショービジネス全般についてのメイクの役割を解説します。 最近はインディーズ作品において、自分でメイクしなければならない場合はあると思いますし、オーディションなどでも自身でもメイクは必須といえます。今回は、「メイクで損をしている女優さんが多い」という話を聞いたことからの発案となりました。皆様お忙しいとは思いますが、この機会にプロが教えるセルフメイク術を体験してみませんか？ 女優さんはもちろん、男優さんやスタッフの方など、さまざまな方にお越しいただければと思います。

※自身のメイク道具をお持ちの方は御持参ください。

主催・映像企画アマルコルド 申し込み お問い合わせ horii-hikaru@nifty.com
