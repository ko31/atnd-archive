Scale Model Builders Fukuoka(SMBF)は、AFV、航空機、艦船、自動車などのスケールモデル製作を趣味とする方々同士の情報交換やコミュニケーションを目的としている模型サークルです。

ポータルサイト「Scale Model Builders!!」
SMBFサークルご案内サイト

福岡市および近郊にお住いのスケールモデラーの皆様と実際にお目にかかり、模型についての話をする機会として、模型茶会（模型をネタにお茶でも飲みながらダべるオフ会）を実施しております。
実際に作成された作品を持ち寄ったり、技法や製作中の悩みについての相談、情報交換や、様々なお話ができる機会になればと思っております。
これまでひとりで黙々と製作を行っていたという方がほかのモデラーの方と出会う機会になれば幸いです。

イベント名のとおり、喫茶店でゆっくりと模型についての話に花を咲かせようという趣旨のオフ会となります。お酒が絡むオフ会ではありませんので、お子様連れ、未成年の方のご参加も歓迎いたします。
まずは顔合わせがてら気軽にご参加いただければと思っております。

ここで出会えた皆様と、今後様々な活動をご一緒できればうれしく思います。

主催：Scale Model Builders Fukuoka 暫定代表　Highmount
