description

WEBサイト作成に関わる人が集まる勉強・研究・交流の地域コミュニティです。　
WEBサイト作成に関わる プログラマー、SE、HTMLコーダー、WEBデザイナー さんなどが集まります。

東京や関西で　webサービスに関わる人が集まっての研究、懇談会が行われ、
WEBサイト作成のスキル、意欲向上の大きな手助けとなっています。

群馬県でも地元の技術者、サービス関係者を集めWEBサイトの作成スキル、意欲向上の手助けになればと思い
開催を決意しました。

※ 2F 202会議室 で行います。
※13:30 開場、14:00開始です。
※参加費: 初回参加は「無料」です。
※2回目参加は有料(￥200～￥500 くらい?)の予定です。(会場の使用料金などなど。。)
※今回が第一回の開催です。
※食事会の料金は「割り勘」です。

開催内容

挨拶

  • 簡単な研究会の説明をします。(14:00～14:15)

発表 (14:20 ～ 15:30)

  • 「WEBアプリ(CGI) のセキュリティーホールについて」kanayannet
  • 「どきっ！三行で作るランダムダンジョン！？～WEBもあるよ！～」parrot_studio
  • 「WordPressとPluginの拡張に関して」terakuma
  • 「mod_perl、APC、Rack」の速度比較 kanayannet(時間が余ったら)

※ 発表 :1人当たり5〜30分を目安に考えています。
やりたい方募集中です。プロジェクターを用意してあります、PC(要VGA出力)を持参していただければご利用出来ると思います。

※ すでにHPおよびWEBアプリ作成経験者が多く集まる予定なのでやや専門的な話になると思います。
※ LAN はないので、必要な方は emobile など持参願います。

懇談会 (15:30 ～ 17:00)

  • 適当わいわい free talk です。
  • 全員食事会(お茶だけでもOK)希望ならやや早めに行きます。

食事会 (17:00 ～)

  • その場の希望者で行く予定です。（場所未定）
