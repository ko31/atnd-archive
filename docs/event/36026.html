【習得できる知識】
1)スプレイの理論と応用
2)オンロール方式スロットノズル（ダイ）コーティング
3)上記特殊用途適用分野

【講座の趣旨】
　一般的スプレイ塗装方式では難易度の高いコーティング材や被塗物など、ニッチの分野での成功例と開発のポイントについて紹介する予定です。
　本講演のテーマでもあり世界的に注目を集めているハイパワーＬＥＤへのＭＴＳ蛍光体薄膜コーティングプロセス以外にＰＥＦＣなどの燃料電池電極形成、ＤＳＣなどの太陽電池電極形成、飲料缶などの容器への液体、粉体コーティング、エレクトロニクス分野の特殊コーティング、超臨界性流体を応用したコーティング方法などに関しても紹介する予定です。

【プログラム】
１．液体、溶融体、粉体コーティング
　1-1.液体の各種霧化方式の紹介
　1-2.エアレススプレイの理論
　1-3.２流体スプレイの理論
　1-4.特殊コーティング（液体、溶融体、粉体）
　1-5.特殊スプレイ方式の応用分野

２．ＬＥＤ蛍光体薄膜コーティング
　2-1.既存方式の課題
　2-2.ＬＥＤ側壁のカバー及び薄膜の必要性
　2-3.ＭＴＳプロセス
