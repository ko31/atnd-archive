description

マッピングパーティとは、OpenStreetMap(OSM)というみんなで作る地図に地理情報を登録する方法を経験者とともに体験するイベントです。金沢市近辺にお住まいでOSMに興味をお持ちの方がいらっしゃいましたら、ぜひご参加ください。経験者・未経験者を問わず歓迎致します。お気軽にご参加ください。

街歩きをしながら、気になったPoint Of Interest: POI（お店や観光名所などの興味深い対象物）をチェックし、喫茶店などに入りパソコンを使って実際のWebで動く地図にデータとして登録して行きます。お好みでどのようなPOIをチェックしても構いません。

コースは駅前から南下して大野庄用水沿いを歩くつもりですが、参加者のご意見などにより臨機応変に対応したいと思います。

1.集合日時、場所

2014/9/21(日)13:00集合
金沢駅前交番付近（東口）
雨天決行（歩ける程度なら決行。判断が難しい場合は前日及び当日朝、この場でお知らせします）
主催者はオレンジ色の工事作業員のようなOSMロゴ入りのベストを着用しています。

8.マッピング（マップの描き方）の例

例えばコンビニをOpenStreetMapで表現する場合には以下のように描きます。
1)まずコンビニの位置にノード（点）を打ちます。
2)そのノードに以下のようなタグ（属性を表わすキーと値の組み合わせ）を書き込みます。
shop=convenience （コンビニ）
name=FamilyMart （名称）
