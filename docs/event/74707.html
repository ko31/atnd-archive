title 子育てに自信をなくしたお母さん向け　一瞬でイライラを消す方法

description

今までの私から生まれ変わる。ワクワク子育ての秘訣が分かります。

●育児は育自、自分育てです。
子供ともっと楽に関われるシンプルなスキルや、
お母さん自身の心のあり方などを、お伝えしたいと思います。

●こんな人におすすめ
初めての子育てに行き詰まって
思ったように子育てができなくて悩んでいるおかあさん、
周りの期待に応えてきた頑張り屋さんのお母さん
子供との向き合い方がわからないお母さん、お待ちしています。

子供との向き合い方（特に２歳から1２歳位まで）
お任せください。
子供はとても素直なのでお母さんが変わると
子供にも驚くほど早く変化が起こりますよ。

生まれたばかりの子供を抱いた時、
それだけで幸せな気持ちになったのに
いつの間にか子育てが苦しくなっている。

子育てがつらい、
子供にイライラして可愛いと思えない
母親を放棄して逃げ出したい
我慢できずに叩いてしまう
私は母親失格なんじゃないか･･･

こんな事を思いながらの子育ては苦しいですよね。
過ぎてしまったら戻ってこない子供との今を
毎日しんどい思いをして過ごすのはもったいない！

同じ子育てなら子供の将来にワクワクしながら
楽しい子育てしませんか？

私はその場しのぎではなく
お母さんが子供と一緒に居られる毎日が
楽しくて幸せな時間になるようにお手伝いします。

内容
・子供との向き合い方。
・自分の心の声を聞く（怒りの感情、自分の中の決め事をほどいて
楽になる）
・子供に伝わる話し方、（叱り方・褒め方）

●期待効果
子育てに前向きになれて
帰る頃には心が軽くなって子供に会うのが楽しみになりますよ。

●主催者からのメッセージ
子供も、お母さんも、
毎日がワクワクで幸せに過ごせますように…。
あなたはダメなお母さんなんかじゃありませんよ。

子育てに行き詰まってこのセミナーにたどり着いたお母さんは
とても良いお母さんだと思います。
苦しいのはそれだけ子供を愛している証拠。

子育てにはコツがあります。
ちょっとの事で子供との関係も
自分との向き合い方もグッと楽になります。

セミナー後は
少し心が軽くなっているはず。
楽しみにしていらしてください。
