【講演主旨】
本講演では、日本のものづくりの風土文化、職場力の低下の原因についてはじめに言及し、トヨタ勤務時に得られた経験を元に管理者にとして部下をやる気にさせる仕掛けとコツについて解説いたします。また、ヤル気を起こす上で最もトヨタで留意した心のマネジメントの重要性を解説すると共にメンタルヘルス対策についても、できる限り、参加者の皆様が業務上、少しでも役に立つ内容を解説するようにいたします。

【キーワード】
1.ものづくり伝承
2.マネジメント
3.現場
4.トヨタ
5.やる気
6.リーダー
7.メンタルヘルス

【プログラム】

1.失いつつある日本のものづくりの風土文化
　1.1　欧米の「経営手法、考え、評価方法」を取り入れてきた日本の企業
　1.2　失われつつある愚直で勤勉な日本の風土
　1.3　時代とともに変化している日本の社会構造と価値観

2.おろそかになりつつある伝承行動
　2.1　企業における、従能層の弱体化による不祥事
　2.2　日本企業のノウハウの伝承率の推移
　2.3　発生している問題は職場力の低下が原因

3.管理者にとって心のマネジメントの重要性とメンタルヘルス対策とは?
　3.1　企業が生き残って行くために大切な人づくり
　3.2　人財育成とは人間力ある社員を育てる
　3.3　企業が見逃している　意外に重要なこととは?
　　3.3.1　会社が金をかけなくても、大きな業績を伸ばす手段
　3.4　社員のやる気で変わる仕事の成果
　3.5　「人のやる気」が、成果や業績に大きく貢献
　3.6　会社を辞めたいと思っていた新入社員を目覚めさせた先輩の行動
　3.7　リーダーの想いは必ず伝わる～リーダーが部下をやる気にさせた事例～

4.管理者に必要な2つのマネジメント
　4.1　職場運営に大切なマネジメント
　4.2　「マニアル通りの仕事をする」ある社員(店員)の事例
　4.3　人は二つの環境がかみ合って高い人間力になる

5.部下の心をやる気にさせる仕掛けとメカニズム
　5.1　社員がやる気を無くす心のメカニズムを理解していますか?
　5.2　社員がやる気になる時の心のメカニズムとは?
　5.3　業務遂行におけるリーダーの大切な行動とは?
　5.4　部下の心をつかむリーダーの行動とは?　課長の成功例
　5.5　仕事の厳しさと、人の優しさの大切さ
　5.6　管理者としての真の仕事とは?
　5.7　家族の工場見学　～社員のやる気育成の事例～
　5.8　部下の能力を引き出すことは上司の勤め
　5.9　不器用な新人社員に与えられた重要な仕事　～社員のやる気事例～
　5.10　日のあたらない所に目を配ると観えてくるものがある～上司が部下をやる気にさせた事例～
　5.11　プロジェクト遂行におけるリーダーの存在要素
　5.12　生産を支えている同僚の姿を知った時　～全員参加で会社が動いている～
　5.13　管理者の部下対応はメリハリのある行動
　5.14　皆から「棒にも箸にもかからぬ」と言われた者が、ある日を境に立ち直った事件
　5.15　心のマネジメントで変わる社員の姿

6.管理者としての必要な「PDCA」行動

7.部下にやる気を出させる理解、行動
