★講演内容

Amazon Web Services(AWS)を始めようと思っているけど、キッカケが無くて手を付けていない方！

新UIとVPCアカウントに対応しました。

◆EC2インスタンスの操作
◆セキュリティグループの操作
◆Amazon EBS, スナップショットの操作
