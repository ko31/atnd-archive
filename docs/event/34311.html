Eucalyptus Advent Calendar 2012 JP

なにこれ？

日本の CloudStacker が Advent Calendar をやるようなので、バチバチッと対抗心を一方的に燃やして Eucalyptus Users Group Japan としても Advent Calendar をやってみようと思います。

ξ*ﾟ⊿ﾟ)ξ＜ちょ、誰よ「日本CloudStackユーザ会のパクリ」とか言ってんの
ξ*〃〃)ξ＜べ、別に向こうが賑わってて羨しいとか思って立てたわけじゃないんだからねっ

Advent Calendar については こちら を参考にしてください。

皆さんのエントリ

皆さんのエントリは こちら にまとめています。

仕様

このページ右上にある「このイベントに参加登録する」ボタンをクリックして頂き、参加コメント欄にて「何月何日に書きます」的な表明をしてください。書いたらこのページの下部にあるコメント欄に「何月何日のエントリです URL～」みたいなことを書いてアピールしてください。(もしくは こちらにも皆さんのエントリを転記して頂けると幸いです )
ちなみに記事などの長さに制限はありません。思いの丈を綴ってください。もちろん、内容が短くても薄くても OK です。参加することに意義があります。(むしろ参加してよぉ… (ノT-T)ノ )

不明な点は日本 Eucalyptus ユーザ会のメーリングリストか、Twitter で @habuka036 に訊いてください。

媒体

インターネット上で閲覧できる場所なら何処でも OK です。slideshare でも OK ですし、何なら Twitter でツブやいて Togetter でまとめるとかでも OK です。ML に投稿してくれても OK です。(アカウントを作れないと閲覧できない場所の場合、もしかすると JEUG のサイトに転載させてください的なお願いをするかも…)

お題

  • こんな環境や構成や規模で使っているぜ
  • こんな改造をした Eucalyptus を使っているぜ
  • AWS と連携してみたぜ
  • こんなツールと組み合わせて使ってみたぜ
  • こんなコードをコミットしたぜ
  • コンポーネントを擬人化して薄い本を作ってみたいぜ
  • 「おい、羽深！」「こんなログ吐いて動かねーぞ」とか
  • その他、色々

まぁ、ぶっちゃけ何でも(「僕とEucalyptusとTシャツ」とかでも)いいので Eucalyptus に関するネタなら何でも OK です。
