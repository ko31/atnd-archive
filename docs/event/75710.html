DOMOデータ活用入門セミナー ～ビジネスを刺激するデータ活用術～

本セミナーは、顧客データ、売上データ、Webサイトのアクセス解析データ、ソーシャルメディアなど、様々なデータが社内外に散らばっており十分に 活用できているとは言えない状況を踏まえ、「経営や業務状況の俯瞰（現状把握）」、「より良い意思決定を即座に実施」、「レポーティング業務の自動化」と いった課題を解決するためのヒントを得ていただくことを目的としております。

Domoは、従来のBI（ビジネス・インテリジェンス）ツー ルと何が違うのか？ Domoが提供するクラウド型のビジネス管理プラットフォームにより、経営や業務へどの様な効果をもたらすことが可能となるのか？などについて、デモンストレーションを交えてご紹介いたします。

  • ビジネスリーダーが抱えるデータ活用に対する不満
  • 従来のBIが持つ課題
  • ビジネス管理プラットフォーム「Domo」および、事例の紹介

参加対象

経営層、経営企画、営業、マーケティング、業務、財務の関連部門で、社内外のデータを経営や業務に役立てたいと考えている方
※競合企業にお勤めの方、個人の方の申し込みはお断りさせていただきます。また、申し込み多数の場合、対象の方を優先させていただきますので、ご了承ください。
