農園ボランティア

◆活動内容

基本的に簡単な農作業です。
春は種植え、秋は収穫などですが、
植える野菜の種類により作業内容は異なります。

雨天の場合は、
小雨の場合は雨具を着用し農作業をして頂く予定ですが、
室内（ビニルハウス又は倉庫）などでの作業となる場合があります。
