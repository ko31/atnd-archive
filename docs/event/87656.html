筑波大(非公認)音ゲーサークルTsukuba Evolved Another(TEA)が設立されて今年で5年目となります。 つきましては、毎年恒例となりましたシェーキーズでのピザ会兼総会を開催いたします。 みんなでピザを食べて地力を上げましょう!

ピザ会のあとは筑波大学春日エリア内の図情図書館の会議室で総会を実施します。 内容は以下のとおりです。

  • TEAの活動の説明
  • これからの活動について
  • 次期会長の選定

…などを行う予定です。

ピザ会だけの参加や、総会だけの参加も歓迎です。 筑波大音ゲーマーとの交流の絶好のチャンスですので、皆様ふるってご参加ください。

念のため定員を設定していますが、人数が極端に多くならない限りは全員参加できる見込みです。 祝日のランチ時は、シェーキーズは予約できないので、開店時間に集合する形となります。

今回の総会では、会長の引き継ぎを行うとともに、今後のTEAの活動方針も話し合う予定です。今後のTEAのあり方について検討したい方は是非お越し下さい。意見がある方は@TEA_tkbにリプライするかSlackに書き込んでいただければ検討します。
