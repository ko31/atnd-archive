7/25火曜中野に暗号通貨勉強会1.2部分けて開催します。
★1.2部両方とも企画内容は同じです。
企画は4つに分けてます。

1.
8/1のbitcoin問題対策やどの様にその期間過ごせばいいか？！
また、それに関連してbitcoinの保管の方法やその方法の種類なども一緒に話してもらいます。

2.
暗号通貨の最新情報、マル秘ネタ(あるかないかは秘密です)、質疑応答の時間も設けさせていただいてます。

3.
暗号通貨のトレードなどやる時間がない方はどうやって増やしていくのか？
マイニングやico とは何か？
昆明でのマイニング最新情報など

4.
勉強会の最後に暗号通貨グループの無料版への参加の仕方の説明もするのでまだ参加してない人は是非参加してください。

7月の勉強会は何回かありましたが毎回違う雰囲気の勉強会になりどの勉強会よりもすごい勉強になるので1回来た方でも再度お越しいただけたら幸いです！

懇親会

懇親会ならではの話とかも聞けたりするので最後まで楽しめ勉強しお腹いっぱいで帰っていただける懇親会になってますので参加してください。
