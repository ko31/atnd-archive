description

VMware社のクラスタファイルシステムであるVMFSの父であり、現在PernixData社のCTOとして活躍するSatyam Vaghani氏の来日に合わせ緊急開催！
PernixData FVPを利用することで仮想化できずに残ってしまったデータベースサーバを仮想化し、運用コストやライセンスコストを抑えながら物理で動作していた頃よりも更に早いパフォーマンスを得ることができます。
また、PernixData FVPのWriteの高速化機能により、高いWriteパフォーマンスが必要とされるVDI環境でもサーバに搭載された高速なFlashメモリーにパフォーマンスをオフロードし、ストレージのコストを抑えることが可能です。
これらの劇的な変化はESXiカーネルモジュールとしてインストールされるPernixData FVPによって、まさに一瞬で実現されます。

本セミナーに参加へのご参加で
・ 既存のデータセンタが抱えるストレージのパフォーマンス問題の解決方法
・ PernixData FVPの導入・運用がどれほどに簡単か？どれほどの高速性が得られるか？
・ PernixData FVPの導入メリット・推奨構成、今後予定のアーキテクチャ変革とその効果
・ 低遅延ネットワークを利用した際のPernixData のストレージ高速化の効果
を理解いただくことが可能です。

また、当社SEのおすすめ構成やPCIe Flashを提供するVIRIDENT社、および高速インターコネクトを提供するMellanox社のディストリビューターであるアルティマ社との共同検証についてもご紹介。さらに、今後予定されている拡張機能についてもCTO本人から解説予定です。
次世代データセンタストレージアーキテクチャを考える上で、必見の内容で、まさにデータセンター内で起こる革新を1日で体感できるセミナーとなっております。
