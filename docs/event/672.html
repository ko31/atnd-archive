catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：200円(会場費をみんなで割り勘)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　●セッション１(25分)
　『自分で作る Google』高橋さん


　＜内容＞
　私たちはよく Google などの検索エンジンを利用していろいろなものを検索していますが、
　中の動きについては知らないことが多いように思います。
　簡単にですが、 PHP などのスクリプト言語でミニ Google を作り、その動きを理解してみましょう。

　＜自己紹介＞
　株式会社システムフレンド プログラマ
　いきものがかりとか聞きながら毎日プログラム組んでる人です。
　情報視覚化や自然言語処理等を遊びながらやりつつ、カートシステム改造の仕事をしてます。

　●セッション２
　『他の人のデザインに自分勝手に講釈をたれる』かわかみさん

　＜内容＞
　かわかみがその辺で拾ってきたチラシ
　そのへんでキャプチャってきたWEBサイトを元に
　「何でこーいう作りになったのか」を
　かなり自分勝手に解釈たれます。

　時間があまったら

　・コンパクトデジカメでこの辺くらいまでは撮れる
　・デザイン本はここを見て買え!
　・神戸おかいものガイド
　・阪神の交流戦ユニフォームについて
　・カープの孝市じゃない方の尾形
　　営業にえこひいきされてる疑惑について

　のどれかでも…

　＜かわかみさんの紹介＞
　有限会社リクソン所属
　http://www.wrixon.net/

　磯辺焼きのにおいだけでご飯３杯食べられるのに
　TV中継が始まると思わず我慢してしまうほど
　阪神とカープ好きなWebデザイナー

　●セッション３
　(１)『地方でも頑張ろう！クリエイター＆プロダクション』まるやまさん

　＜内容＞
　弊社ココノヱは最近地方で頑張ってるプロダクションの一つとの
　評価をいただけるようになりました。
　地方には地方なりの大変さがありますが、そんな中で過去を振り返り
　他社との差別化を図る為にやって良かった事・悪かった事などを挙げて
　みたいと思います。

　(２)『運用益減を埋める提案を考えよう』

　＜内容＞
　数年前から提案材料の一つとしてCMSの実装を進めて来たプロダクションは
　多いと思いますが、結果更新等での固定収入が薄くなってきてはいませんか。
　まあうちがそうなだけなんですが！何か考えないとしんどいです！

　＜まるやまさんの紹介＞
　丸山 祐司
　株式会社ココノヱ ディレクター

　株式会社ココノヱ
　http://9ye.jp
