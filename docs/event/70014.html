PSoC組込みボード勉強会

イベント ユーザー
event_id 70014
title PSoC組込みボード勉強会
catch
description

PSoC初心者向け実習で、PSoCボードを使って体験してみるワークショップです。
外付け部品なしでアナログ回路・デジタル回路を組むことが可能です。
PSoCの手軽さと開発のしやすさを体験いただけます。

日時

2015年10月3日（土）10:00～17:00
※途中、昼食タイムと休憩をとります。

場所

ファブラボ北加賀屋（コーポ北加賀屋1F）
大阪市営地下鉄四ツ橋線「北加賀屋駅」4番出口から徒歩10分

料金

お一人500円(ワンコイン)
※料金は当日会場でお支払いください。

必要機材

1.　ノートPC (Windows Vista,7,8.1)
2.　PSoC 5LP Prototyping Kitボード
　　※秋月電子参考リンク PSoC 5LP Prototyping Kit
3.　WindowsPCに Cypress PSoC Creator3.3(最新版)を導入
　　※CYPRESSサイト PSoC Creator 3.3 for Windows
　　※レジストレーション(無料)を済ませておいてください。
4.　ブレッドボード（ジャンパ線、その他お手持ちの電子部品をお持ちください）
5.　USB延長ケーブル(必須)
6.　USBマイクロケーブル(必須)
7.　延長コンセントタップ
　　※貸出機あり（PSoC3 Development Kit or PSoC 5LP Prototyping Kit）
　　※貸出機をご希望の方はコメント欄へお願います。

ワークショップの概要

1　PSoCって何？（他のマイコンとの違い、特徴など）
2　PSoC Creatorの使い方
3　まずはLチカをしよう
4　PWMを学ぶ
5　割込み処理
6　シリアル・デバッガ
7　アナログ回路のキモ
8　質疑応答
※適宜、休憩をはさみます。
※実施内容は都合により変更になる場合があります、ご了承ください。

講師

野口 哲也
