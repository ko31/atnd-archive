DDIポケット HYBRID W-ZERO3 WS027SH 2年縛り卒業式（HYBRID W-ZERO3 WS027SH 2years Contract Graduate Celemony）

DDIポケットが発売した、＃（シャープ）製の不具合端末、HYBRID W-ZERO3の発売日から、早２年が経とうとしています。
発売日は2010年1月28日。

発売日に新規契約をした方は、2012年2月が更新月になり、端末の2年縛りが解けていつでも機種変更の時期を迎えます。
また、翌年2013年2月には新DDIポケット定額プランGの縛りも解け、いつでも違約金なくスッキリ解約出来るようになります。

サードパーティー製充電器を探すのが大変だった厄介な子。
でもこいつに付属する充電器は、+Dと-Dが短絡してあり、
IS03もIS04も、T-01Cも何でも充電できる賢い子だったりしましたね。

遂にWindows MarketplaceでのWM6.x系アプリの新規登録を打ち切るとMSが大々的に発表し、ますます立場がなくなっていったこの子。

２年縛りが解ける2012年1月28日に、皆さんで一緒に卒業式をしましょう！

会場について

DDIポケット本社のあった、東京都港区虎ノ門付近もしくは
ハゲ銀行本社のある汐留付近の居酒屋で行います。

ドレスコードについて

卒業式ですが、無理して袴や振袖、学ランやセーラー服で来る必要はありません。
ただし節電のため店内の空調が弱い可能生がありますので、
必ずハイパークールビズもしくはハイパーウォームビズでお越し下さい。
