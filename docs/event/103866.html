=====
あなたは出版に関して、こんな間違いをしていませんか？
=====

・出版するのには膨大な時間と労力がかかる
・本さえ売れれば何とかなると思っている
・無名の私がベストセラー本なんて作れない
・編集者に響く企画書を一生懸命考えている

これらは、出版の際に、ほとんどの人が
知らずに犯してしまう間違いの一部です。

でも、これらは「最もよくある間違い」
に比べれば、取るに足らないことです。

その間違いとは

【「紙」で出版しようとしている】

ということです。ここに大きな問題があります。
=====
莫大なコストと膨大な時間がかかる
=====

紙の書籍の場合、大半が自費出版になります。

比較的誰でも出版できますが、売れずに終わる書籍が大半となります。

自費出版の場合、出版までに1年以上かかり
コストも1冊300万円以上掛ることも珍しくありません。

一方、電子書籍の場合、自費出版となるケースが多いのは
紙の書籍と同じです。

ですが、最短2日で出版できますし
コストも数千円〜数万円で、できでしまいます。
=====
印税収入が10倍以上違う
=====

紙の書籍の印税は5～10%程度、電子書籍は「70%程度」です。

たとえば、電子書籍を1,000円で販売した場合、印税収入は700円。

一方、紙の書籍を1,000円で販売した場合
印税を5%とした場合、印税収入は50円となります。

つまり1000冊売れたとしたら、

電子書籍・・・70万円
紙の書籍・・・5万円

これだけの印税の差がでます。
(しかも、紙で1000冊売るのは、とてつもなく労力とコストがかかります。)

一方で、電子書籍の場合、出版した方の中には 
全くの無名でも、月90万円を超える印税収入
を手にしている人もいます。

=====
紙の書籍の市場は減少、無名で出版は無理・・・
=====

書籍の販売額は、年々低下しています。
書籍が売れづらくなっているからです。

だから、出版社は売れそうな本しか扱わなくなるので
無名の人はまず出版は無理です。

一方で、電子書籍の市場は拡大し続けています。

それに伴い、誰でも気軽に出版できる
ハードルの低さから、出版全体の質が大きく低下しています。

逆を言えば、ライバルが弱いので、今参入すれば余裕で勝てます。

つまり書籍の質に着目し「クオリティが高い」
といわれる書籍を出版することが最低条件になってくるということです。

=====
では、どうすればいいのか？
=====

を「2日間という短期間で」作り上げることをコンセプトに、、、
