イベントの概要

記念すべき第1回 Chiyoda.rb 勉強茶会を開催致します。
皆様お誘い合わせの上、奮ってご参加ください。

対象者： Rubyプログラマー

場所

  • 株式会社万葉様オフィス: http://everyleaf.com/
  • 住所: 〒101-0054 東京都千代田区神田錦町3丁目16-11エルヴァージュ神田錦町8F

タイムテーブル

  • 19:00 開場・受付開始
    • 紅茶、お菓子が振舞われますので、会場設営にご協力ください。
  • 19:30 第1セッション（担当：瀧内）
    • 自己紹介を兼ねて、最近の Ruby 動向について話題を持ち寄り、掘り下げていきます。
    • 特に無い場合は Rubyflow から気になる話題をピックアップします。
  • 20:15 第2セッション（担当：河本）
    • Rails3周辺の話題について調査レポート。
    • asset pipeline などなど。
  • 21:00 閉会
    • 後片付けにご協力ください

備考

  • 自慢のお菓子・紅茶などをご持参いただけると盛り上がります。
  • 発表者・関係者枠を別途用意しております。参加者は全部で20人となる予定です。
