「リフレクティング・アプローチによる新しいチームカウンセリング」

北欧生まれの社会構成主義／ナラティブの代表的チームカウンセリング。リフレクティングアプローチによるチームカウンセリングは、まるで水面に反射された光が壁に投影されるように、自己の物語や問題の本質が見えてきます。誰もプレッシャーに感じることはなく、とても穏やかにして、あたたかく全員が気付きを共有できるカウンセリング。差異がもたらす不思議なグループダイナミズムを体験してください。

セミナーを受講すると・・・

  1. 相談者役にとって複数の専門家／キャリアカウンセラーがサポートしてくれる安心感が持てる
  2. 短時間でマルチな視点でナラティブストーリーが展開できる
  3. 差異（違い／ずれ）が生み出す問題の本質からの驚くような相乗効果にこころが動く。
  4. 全員のが疑似体験できて、気づきを共有できる。あたたかい気持ちで終えることができる。
  5. スーパービジョン的体験ができる。
  6. クライアントの困難と思われた問題が解決・解消する。

以上のような効果を感じることができるようになります。
