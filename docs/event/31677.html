Network Forensics Puzzle Contest に挑戦

第8回、第9回に引き続きオンライン上で開催されているパケット解析コンテスト Network Forensics Puzzle Contest のPuzzle に挑戦します。
今回は第3回の出題「Puzzle #3: Ann’s AppleTV」の解析を試します。参加される方は、可能な範囲で出されている問題の解析を行ってください。なおこの Puzzle は過去の出題ですので、既に正解が公開されています。勉強会では手軽に利用できるパケット解析ツール (Wireshark など) を使って、どのような手法で解析できるかを検討していきます。

解析結果の発表

解析手法とその結果の発表を希望される方は、にメールでご連絡ください。
またそれ以外にも当日抽選で発表者を指名させていただく場合があります。

出題内容

出題を日本語訳しています。参考にしてください。

Ann と Mr. X は新たな作戦本部を作っています。貴方と貴方の調査チームは犯人引き渡しのための書類手続きが完了するまで、彼女の行動を秘密裏に監視しています。最近になって、Ann は新品の AppleTV を手に入れ、それの固定 IP アドレスを 192.168.10.10 に設定しました。最近の彼女の行動を示すパケット キャプチャがここにあります。

貴方はフォレンジック調査員です。あなたの任務は Ann が何を検索したかを見つけ出し、彼女の興味についてのプロファイルを作成し、含まれている証拠を明らかにすることです。

  1. Ann の AppleTV の MAC アドレスは何か
  2. Ann の Apple TV が HTTP リクエストで使用する User-Agent string は何か
  3. Ann が AppleTV で最初に検索した単語は何か (インクリメンタル サーチのすべてについて)
  4. Ann が最初にクリックした映画のタイトルは何か
  5. 映画のトレーラー (予告編) の URL は何か (“preview-url” で示されています)
  6. Ann が二番目にクリックした映画のタイトルは何か
  7. その映画を購入する価格はいくらか (“price-display” で示されています)
  8. Ann が最後に検索した単語は何か

証拠のパケット (evidence03.pcap) は以下からダウンロードできます
http://forensicscontest.com/contest03/evidence03.pcap
MD5 (evidence03.pcap) = f8a01fbe84ef960d7cbd793e0c52a6c9

  • 会場にはインターネット アクセス環境がありません。必要な方は自前で 3G/LTE/WiMAX などの環境をご用意ください。
