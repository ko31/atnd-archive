WordPressユーザーの交流会 – 2012年12月23日（日）WordBench埼玉勉強会

WordPress 3.5 がリリースされるというタイミングで、WordBench埼玉を行います。
時期としてもクリスマスが近いですので、3.5リリースパーティとクリスマスパーティも兼ねています。

こちらは、WordPress3.5リリースパーティ＆クリスマスパーティの出欠席の確認となります。

WordBench埼玉12月のWordPress勉強会については、
WordPressの勉強会 – WordBench埼玉12月＆WordPress3.5リリースパーティ
より参加表明ください。

18:00 – 20:00
WordPress3.5リリースパーティ＆クリスマスパーティ

会場：
埼玉県さいたま市大宮のコワーキングスペース「7F」

参加費：
WordPress3.5リリースパーティ＆クリスマスパーティ：3000円（学生は2000円）（ケータリングやアルコール・ソフトドリンクの実費です）

WordBench埼玉の運営メンバー

2012年12月23日（日）WordBench埼玉は、以下のメンバーとWordBench埼玉スピーカーにより運営されています。
