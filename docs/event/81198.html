朝活・夜活をMIXERで主催したい方に向けたセミナーを開催します！

セミナーや勉強会をやりたい人、やろうかなと思っている人
ランチタイムを使って、交流会やセミナー、イベントをやりたい人
朝の時間でカフェイベントやその他朝活をしたい人
上映会・ママ会・料理教室・撮影会・展示会などを開きたい人
木の床と机の、空間を活かしたイベントを考えている人
日本橋や神田周辺の良さをPRするようなイベントを考えている人

MIXERでイベントやセミナー、勉強会を行っていただくために、どんなことができるのかなど疑問質問にお答えします！

・どんなレイアウトで使えるの？
・どんな備品があるの？
・飲食は可能なの？
・どんな告知をしてくれるの？

などなど、イベントを開催する際の疑問を解消し、MIXERもそうした方々へサポートをいたします。

セミナーや勉強会の講師やイベント企画者へのサポートとして、本セミナーを受けてくださった方には、スペースの価格をセミナー主催者応援価格として、応相談させていただきます。
