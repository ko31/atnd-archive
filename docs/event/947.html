　多数の方の申し込みありがとうございます。補欠申し込みの多さに少々びびっています^^;。会場の広さに限界がありますので、今回は45名を定員とさせて下さい。申し訳ありませんが、補欠の方はキャンセルをお待ち下さい。

　もし、「補欠なんだけどどうしても入りたいよーー」と言う方は、発表に立候補するという裏道があります。発表を希望される方は、ATNDのコメント欄か、個別に江川までご連絡くださいませ。

　発表を希望したはずなのに名前がない。。。あるいは、発表を希望したつもりがないのに名前が載っている・・・と言う方は、お手数ですがご指摘ください。m(__)m

しばらくお休みをしていたAndroid SDK WGセッションを開催します。

　Android SDK WG とは、日本Androidの会（ http://www.android-group.jp/）のワーキンググループのうちのひとつで、Android SDKの理解を深めながら、開発者同士仲良くなろうぜ！を目的としたフランクな集いです。定期的に週末の午後を使って「セッション」と言う名の｛勉強会｜交流会｝を開催しています。
　Android SDKについての会話をしようという共通のマインドがあれば、サンデープログラマー、上級者、iPhoneプログラマー、どなたでも参加可能です。ぜひお気軽にご参加ください！
