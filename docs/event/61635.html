description

高度化、専門家、複雑化、国際化など様々な変化にさらされるWeb業界で働く方を対象に、自らキャリアをデザインしてWeb業界で生き抜くための「スキル」と「パフォーマンス」を向上するシリーズセミナー。
第一部では、某メディアサイトの副編集長をお招きし、コンテンツ作成の現場ノウハウをお話しいただきます。
（諸事情によりメディア名は非公開とさせていただきます。詳細は個別にお問合せください。）
第二部では、ソニー（株）、日本モトローラ（株）などで半導体設計エンジニアと従事したのち、業務プロセス改革やプロジェクトメンバーの意識改革を行うコンサルタント、コーチとして世界で活動する弊社取締役の倉田隆弘が、組織や個人の目標達成を高速化する思考法について語ります。

第1部：スキルアップ！

ネットメディアの編集者になって早3年。
その前は広告プランナーとしてWebプロモーション業に携わってきました。
広告という視点でターゲットの認知・興味を高める施策に取り組み、
そして現在は「ネット記事」や「コンテンツマーケティング」に向き合う日々。
その”編集”と“広告”の視点で私がつかんだ、
「読まれるコンテンツ制作」に関するノウハウを事例満載でお話させていただきます。
お楽しみに！

1.自社サイトのトラフィック増に、ネット記事が使えるワケ

（「うちの商品を紹介して！」という依頼を頂くメディア側の本音を、

3.読まれる＆トラフィックが集まるネット記事の作り方
（ネットメディアでヒットする！　検索エンジンで上位になる！）

プロフィール

ソニー（株）、日本モトローラ（株）にて半導体の設計開発に従事する。ソニーでは米国のシリコンバレー（サンノゼ）にて中心メンバーとしてLSI設計開発組織の立ち上げを行う。また、シックスシグマ・マスターブラックベルトとして、品質改善、新規設計開発、研究開発のプロジェクトをコンサルティング支援、その数は300プロジェクトを超えた。さらに、ソニーの研究開発業務に適したシックスシグマ手法であるDFSS(Design For Six Sigma)も開発。その後、米国のシックスシグマコンサルティング会社(Pivotal resources) にスカウトされ、IT技術を中心としたビジネスを展開している企業のシックスシグマ導入において研修、指導を行う。ソニー退職後も、画像処理技術をコアとしたITベンチャー会社にて画像処理LSIの開発をプロジェクトマネージャーとして統括したり、LSIの設計開発を行っているソニー関連会社の品質改善、プロジェクトマネジメント、企業内コンサルタント育成、ワークショップ・プログラムの開発を行う。
2012年2月、（一財）生涯学習開発財団認定コーチの資格を取得し、ソニー（株）のマネージャーに対するマネージメントコーチングを開始する。現在は、SWOT分析、SWOTクロス分析を中心とした事業計画の為の戦略立案ワークショップの海外展開（英語）も行っている。
