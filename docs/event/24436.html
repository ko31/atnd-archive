description

個人で19インチラックを使いこなしてみよう。という勉強会。

おっとあたりで19インチラックをついカッとなって買ってしまったが、
無秩序な配線やサーバー設置による「ケーブルの樹海」にならないよう、
ラック導入直後からプランを立てるための勉強会です。

やること

  • LANケーブルの敷設、成端
  • LANパッチパネルの成端
  • 19インチラックへの、ネットワーク機器取り付け作業
  • ケーブル包縛および丸札とりつけ作業

注意点

  • 会場には机がありません。その代わりシャーシ型SW-HUBを使います
  • 冷蔵庫はありません。その代わり19インチラックがあります
  • 暖房がありません。各種機器の廃熱で間に合います。
