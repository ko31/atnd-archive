description

《 年収250万円のOLから、不動産資産6億円の投資家に！あなたはお金から好かれていますか？？ 》

ある日、突然の事情によって、向き合わざるを得なくなった「お金」の問題。
2人の子どもを抱えての危機に、それまで自分が「お金」に関していかに無知だったのかを実感。
子どものため、そして自分の経済的自由のために勉強した株式投資、不動産投資によって、資産ゼロから6億円を築きあげた星野陽子氏。

「誰しもがきちんと勉強し行動すればお金はついてくるのだ」
という信念を持つ星野陽子氏が、本セミナーにて、
「お金を生むヒント」「お金が集まる習慣」「お金に増えてもらうという投資家的思考」 を伝授！
何か行動しなければ、お金は自然に増えるものではありません！

逆境からいかにして「お金からの自由」を得るに至ったのか？？ 「お金に好かれる法則」、星野陽子氏があなたに伝授します！

《デフレ脱却！今こそ求められる投資戦略とは！？》

アメリカNYダウは史上最高値を更新!
量的金融緩和第三弾（QE3）縮小で、新興国から先進国へ戻る投資資金。
一方、ヨーロッパでは、「ゼロ金利」からついに「マイナス金利」の時代到来！？
日本企業もアベノミクスによって稼ぐ力を取り戻し始めたかに見える現在。
しかし、消費税10％への引き上げや、国民年金の株式の運用比率の引き上げ検討など、足元ではこれまでの歴史上類を見なかった政策が、次々と行われています。
商品市場を分析して50年。あまたの時代の転換点を目撃してきた古川昭治氏が、今何を考え投資するべきか、熱く語ります！！
半世紀もマーケットと向かい合い続けている古川昭治氏の知見は、これからのあなたの資産運用を考えるうえで、きっとヒントになります。
