【無礼講歓迎】開催直前！ IT/Web業界の注目企業の中の人とホンネで語れる転職イベント

~IT/Web業界注目の企業が相手でも、聞きたいことはガンガン聞いちゃおう~

◎イベント概要

IT/Web業界の大手企業から今注目を集めるベンチャー企業まで、採用担当者やあの社員とホンネで触れあうお祭り「第2回PooleExpo！」を開催します。

Q1. カヤックさん、サイコロ給を見越して基本給が低かったりしますか……？

Q2. 楽天さん、さすがに社員全員は英語を話せませんよね？

Q3. チームラボさん、猪子さん語録に「ヤバいものをつくりたい」とありますが、「猪子さん、ヤバいなー」と思ったときはどんな時ですか？

などなど、きわどい質問をどんどんぶつけてみてくださいね。

（その他のぶっちゃけ質問例はLIGブログでも紹介中！）

会場には、射的やヨーヨー釣りなど、旅行券や会社見学券、企業からのプレゼントなどが当たるスペースも。夏の終わりに「縁日」の屋台をめぐるように、いつもとは少し違った雰囲気のなかで、いつもとは少し違ったカタチの縁と出会っていただけたら嬉しいです。

もちろん、今回もLIGメンバーがコンシェルジュとしてサポートします。 どこの企業を見ればいいのかわからない 自分から話しかけるのはちょっと緊張する そんなときは、いつでもわたしたちにお気軽に声をかけてください！ あなたにぴったりの企業までご案内いたします。

◎PooleExpo！とは？

Pooleでは、デザイナー / エンジニア / ディレクター / ライター・編集者だけではなく、セールス / マーケティング / コーポレートスタッフなど、Webの仕事に従事する人たち全を「Webパーソン」と呼びます。

『Poole Expo！』は、そんなWebパーソンの方々を対象とし、Web業界で知名度のある大手企業から今注目を集めているベンチャー企業まで、Pooleのインタビュー記事や求人に出ている採用担当者や社員と直接話せるイベントです。

さらに…、

あの会社で働くには、どうしたらいいんだろう

あの会社は、どんな人が働いているんだろう

求人やインタビューに出ていたあの人と話をしてみたい！

ベンチャーに興味はあるけど、会社の雰囲気がわからない

出展企業に興味があるけど、会社訪問する時間が取れない

というみなさま、お気軽にお越しください！

◎IT / Webパーソンのための「縁日」がテーマです

「縁日」という言葉を聞くだけでわくわくしませんか?

わくわくする理由は、「あの人に会えるかもしれない」という期待感と いつもと少し変わった場所に新鮮さがあるからかもしれません。

今回のPoole Expoでは「縁日」をテーマとし、 「会える期待感」と「いつもと違う転職イベント」を演出することで みなさまの「いい出会い」の場を提供します。

射的やヨーヨー釣りなど、旅行券や会社見学権等、豪華商品の当たるみなさんの参加していただけるブースもご用意してお待ちしております！
