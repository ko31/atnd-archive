description

　エッセイスト、画家、イラストレーター、アーティストとして
　国境を超え、幅広く活躍されている山﨑氏。

　２０歳よりアムウェイビジネスで躍進し、ほとんどの世界
　タイトルを最年少で記録達成。

　２２歳で起業し、現在は３つの会社を運営。

　そんな山﨑氏のやる気の源はどこにあるのでしょうか？
　どうやって、想像だにできない膨大なタスクをこなしているのでしょうか？

　「すごい事はアッサリ起きる。

　そんな事を受け入れた瞬間から、世界は変わる。

　目にみえているのはあなたの意識したことなんだ。

　聞こえているのもあなたの耳が選択した言葉だけ。

　ゆっくり息を吐き出して、目の前の現実を信頼しよう。」
　

　一言一言がまるで作品であるかのように、
　素敵な言葉を紡ぎ出す山﨑氏。
　
　
　そんな山﨑氏のお話を直接聞ける
　めったにない大チャンスです！
　
　皆様お誘い合せの上、お越し下さい！
