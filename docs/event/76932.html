5/1(日)、町田で唯一の銘柄黒毛和牛、完全一頭買いの和牛専門焼肉店『闘牛門』さんとコラボして、和牛5時間食べ放題＆飲み放題BBQパーティーを開催します！

  

 とにかく旨い肉と美味しい酒で、楽しくBBQをしたい！

そんな思いに共感してもらえたら、ぜひ参加してください(∇)ﾉ

 

 【和牛専門店『闘牛門』って？・・】

www.tougyuumon.com

食肉卸直営店だからこそできる！品質の高い和牛をリーズナブルに提供している、今、大人気の和牛専門の焼肉屋さんです。

とにかく『闘牛門』さんの品質へのこだわりは徹底しいて『偽装牛肉、産地偽装、等級の虚偽、勝手に作った名前だけのブランド牛等で牛肉業界全体、生産者の方々の信頼を、信用を失っておりますが、『闘牛門』は、一切の虚偽・不正は致しません。』と堂々と胸を張って経営している信用力の高いお店なのです。

  今回『闘牛門』さんの群を抜いた肉質そのままに、開放的な“天空のビアガーデン”で黒毛和牛食べ放題BBQパーティーを大開催！

 

食べ放題企画だからと言って、質は落としません！

 

他の肉とは一味違う肉質の純国産の黒毛和牛をご用意、個体識別もする安心の肉をぜひ楽しんでください！

 

 【天空のビアガーデンって？・・】

 

町田駅から徒歩数分、結婚式会場などでも名高い、ホテル・ラポール千寿閣

 町田に宿泊なら｜ホテルラポール千寿閣

その屋上に期間限定で開設されるビアガーデンです。

 

全200席、大型プロジェクターも設置！広々と開放感のある天空のビアガーデンを、当日は私達の貸し切りにしていますので、少しだけ特別なプライベートゾーンとして、様々なシチュエーションを気軽に楽しんで貰えると思います。

 ペット同伴もOK!

 

 

 【FUNNY NETWORK コンセプト】

楽しく＆信頼で広げる人との繋がり！紹介・口コミから輪が広がっている集まりで、安心して気軽に楽しく参加できます。もちろん友達やファミリーとご一緒での参加大歓迎です！

 前回、津久井湖畔で開催したBBQイベントも、初対面の方が大半でしたが、いつの間にか一体感も生まれるくらい楽しく盛りあがりました！

 

新聞記事にも取り上げられました！

新しい出会いを楽しむのもよし、友達同士＆ファミリーで盛り上がるもよし、とにかく食を追求するもよし、のんびり青空を眺めながら美味しいお酒と素敵な音楽に浸るもよし♪ HAPPYな時間を過ごしましょう！

 ※参加制限等はありませんが、和やかな会にするために、男女比率半々を目指しているため、予約先着順とさせていただきます。

 

 【和牛食べ放題の他にも..】

☆生ビールはもちろん、各種カクテル・ソフトドリンク飲み放題！

☆メニューは、黒毛和牛カルビ/黒毛和牛ロース/牛タン、さらには、国産野菜/豚カルビ/豚トロ/ソーセージ/鶏モモ/イカ・ホタテ/ 塩キャベツ/枝豆/ライス/カレーが食べ放題！アイス(別料金）もご用意しています。

☆私たちBBQマスターが、ワニ肉といった珍しいお肉や、秘伝レシピBBQ料理なども準備しています。

☆ストラップを配布していますので、一日中出入り自由です。ボーリング場も併設されているので、輪が広がったらみんなでボーリング大会もおススメ！
