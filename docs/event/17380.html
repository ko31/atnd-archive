講演概要

■ Titanium Mobileってなに？

  • @masuidrive
    • JavaScriptでiPhone/Androidアプリを開発できるTitanium Mobileの解説。Titanium Mobileってナニモノなのか、どのように動いているのか、CoronaやngCoreとなにが違うのか、何がが出来て、何が出来ないのかなど実際にサンプルやアプリを通じながらお話しします。

■ TitaniumでAndroidアプリをリリースまでやってみた

  • @kurain
    • Titanium Mobileを利用してAndroidアプリを開発するときにハマりやすい罠を、実際の開発で起きた不具合からご紹介します。またiPhoneとAndroidで同時に開発、リリースまでした簡単なアプリのコードを解説し、アプリの着想から、開発、広告の導入、そしてリリースまでにどのようなハードルがあったかを披露します。

■ 私が Appcelerator, Inc. に入ったワケ

  • @masuidrive
    • いちユーザだった@masuidriveが今なぜAppceleratorで働いているのか？英語のしゃべれない私がAppceleratorに就職するまでの話をさせていただきます。これであなたもAppceleratorの社員になれる!?

kanazawa.jsとは

kanazawa.js は主に金沢市・石川県・北陸地方近辺の JavaScript デザイナー・プログラマによる非営利団体です。各個人のプログラミングスキルの向上を目的に、勉強会ならびにカンファレンスの開催、情報交換などを行うコミュニティを目指しています。

FAQ

JavaScript（プログラミング）全然分かりませんが参加してもいいですか？
学ぶ意欲さえあればどなたでも参加できますが、事前にTitanium Studioをインストールなど触ってみてからセミナーに参加されると良いかもです。

諸事情により参加できなくなりました。
出来る限り早く、ATNDにてキャンセル手続きお願いします。また @kanazawajs まで連絡くれると嬉しいです。

WiFi環境はありますか？
ありません。WiMaxはつながります(｀･ω･´)ゞ

パソコンを持参した方がいいですか？
パソコンがあればコードを動かすことができるのであったほうが良いかと思いますが、電源は十分にはありません。周りの方とシェアしてお使いください。

飲食可ですか？
基本的に不可ですが、蓋のついた飲み物（ペットボトル等）であれば問題ありません。

Twitter ハッシュタグとはなんですか？
発言の最後に半角スペース＋「 #kanaj」を入れてつぶやいていただくと「#kanaj」に自動的にリンクが張られて、kanazawa.js関連の発言を参照することができます。ぜひ中継や覚え書きなどにご利用ください。
