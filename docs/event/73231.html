description

ご覧いただきありがとうございます。

主催者の東と申します。
友人・知人からは「あずまっくす」と呼ばれてますので、
そちらの方でも覚えていただければと思います。

あなたはこんなことを思っていませんか？

・給料が少ない
・もっと収入を増やしたい
・趣味にお金がかかる
・お小遣いだけでは好きなことができない
・高くて美味しいものをもっと食べたい
・とにかくもっと稼ぎたい
・何か副業を始めてみたい
・いつか起業したい
・ビジネスの勧誘を受けて迷っている

お金に限ったことではなくても、

・日常がつまらない
・仕事がつまらない
・何かやりたい
・現状を打破したい
・自分を変えたい

など、感じたことはないでしょうか？

もし、1つも当てはまらない場合は、
このまま読み進めても無益ですのでお止めください。

さて、現状に満足していなかったり、
向上心があったりする人の多くは、
ビジネスに取り組むことを考えます。

ただ、ビジネスに取り組んでいる人のほとんどが、

ビジネスの基本が全然わかっていなかったり、
収入が生まれないまま長く続けたり、
ビジネスをコロコロ変えたり、
勧誘型のビジネス（MLM・ネットワークビジネス）をやって失敗したり、
挙句の果てに、借金を作ったりすることもあります。

また、

・何から始めればいいのかわからない
・ビジネスの知識を身につけたいけど少し怖い

なんて考えている方もいるでしょう。

そして1人で不安を抱え、悩み、苦しみ、
諦めてしまうのです。

私は、どうすればその悩みや不安を取り除くことができるのか、
自分が役に立てることがないか考え、

そういった想いから、自らの知識や経験を
伝えていくことを決意し、多くの人と出会うことにしました。

今回のセミナーでは、

「学校や会社では教えてくれないお金の話」をはじめ、
「ビジネスをはじめからていねいに」お伝えし、
「来月１０万円稼ぐ」ための思考や行動をお教えしていきます。

お金を自分で稼いだことが無い全くの初心者の人や、
ビジネスをやってきて成果が出ていない人には、
必ず学び行動に移していただきたい内容にしています。

すべてのビジネスの基本となるお話をしますので、
何となくビジネスをやっている場合には
答え合わせの意味でもお聞きいただきたいと思います。

「ビジネスのセミナー」と聞くと、
取っつきにくさがあったり、
難しそうに思われるかもしれませんが、

堅苦しい感じではなく、お茶会のような楽しい雰囲気で
ざっくばらんにお話し、気軽に参加していただきたい
と思っています。

・はじめに～なぜ稼ぐ必要があるのか～
・学ぶときに意識する５つの段階とは
・収入を増やすための２つの方法とは

・ビジネスで結果を出すために必要な３つの要素とは
・子どもが野球選手になりたいと言ったらどうするのがいいか

・ビジネスは算数。売上を算出する式とは

・多くのビジネスマンが理解していない４つの働き方とは
