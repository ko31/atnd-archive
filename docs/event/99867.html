概要

こんな人にオススメです！

  • RaspberryPi を使ってみたい
  • RaspberryPi を持っているけど、眠らしている
  • 周囲との親睦を深めたい方

ご参加される方は、各々の作業や学習をもくもくと進めていただく形式となります。

全くの初心者の方へは、主催から、raspbianインストール、起動までのサポートが可能です。

参考資料としてラズパイマガジン2018年8月号を用意しております。 https://trendy.nikkeibp.co.jp/pcls/raspi/#magazine

費用

  • 会場費として、500円（飲み物・お菓子込み）となります。

  • フリードリンク、お菓子
  • モニターの貸し出し可能 ※HDMIケーブルには限りがありますので、各自ご準備くださいませ。
