description

身内に言い出しづらい相続。そうだ、税理士に聞いちゃおう！

相続ってなんだ？という超初心者（私）がわかるように税理士の小澄氏が勉強会を開いてくれることになりました。
いつ相続になるかわからないので気軽に知れる場を作りました。
ゆるい感じで教えてもらう予定なので気軽にご参加ください。

アジェンダ

・“争族”にならないための笑顔相続のポイント
・“相続財産をもらう側” が知っておかねばならないこと
・相続と贈与について

対象

・これから相続を迎える方。
・相続のことが気になっているが誰に相談したら良いか分からない方。
・「家族間で相続の話をするのは気が引ける」とお悩みの方。
