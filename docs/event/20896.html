catch 奇才・ジョー・デイヴィスが語る「世界の捉え方」
description

10月22日(土)のARTS FIELD TOKYOレクチャーでは、来日中の奇才・ジョー・デイヴィスが自身の活動について語り、国内では一般上映されていない彼の活動を撮影したドキュメンタリー映画『HEAVEN+EARTH+JOE DAVIS』も特別同時上映します。

ジョー・デイヴィスは、アーティストであり、哲学者であり、科学者でもあります。彼は私たちがイメージするどんな"アーティスト像"にもあてはまらない人物。同じく、彼のプロジェクトはどれも私たちの想像を超えたものばかり。好奇心・探究心の塊であるデイヴィスの姿勢・行動が周囲の人をひきつけ、彼のとてつもないプロジェクトを実現可能にしています。

ありふれた決まりごとに、知らず知らずのうちに慣れている私たちにとってデイヴィスのプロジェクトは一見"異常"にも見えます。そんな彼のプロジェクトのテーマは、「すべてのことはしばしば、いちばん意外な方法でつながっている」ということ。世の中に存在するあらゆる決まりごとと衝突しながら、自分の信念を貫き続け、彼独自の"世界の捉え方"を探求しているのです。

では、そんな彼がなぜプロジェクトを実現できたのでしょう？彼のなにが人をひきつけるのでしょう？なぜNASAやMIT、ハーバード大学が彼のプロジェクトに興味を持つのでしょう？

デイヴィス本人が語るこの貴重な機会に、是非足をお運び下さい。
