来る 7/19 の Yuki Kajiura LIVE 後にオフ会を開催致します。
ライブには全国から有志が集結されることと思いますので
全国の梶浦由記ファンと交流を深め、かじうらーの輪を広げませんか？

1人での参加も大歓迎です！
私も初のオフ会は1人での参加でしたし遠慮せずご参加頂ければと思います。
昨年の Yuki Kajiura LIVE でも何度かオフ会を開催して
ライブやイベント毎に会える仲間が沢山増えました。
まだライブ後に予定のない方は是非ご参加ください！
7/18～19の公演は2日連続ですが、翌日も休みですので
是非みんなで集まって vol.#12 のツアーを語り合いましょう！
　
　

【日時】
　　7/19（日）19:30～21:30（予定）
　　集合場所は会場入り口付近。
　　詳細な場所については当日か前日につぶやきます。

　　時間はライブの終了時間によって前後する可能性がありますが
　　19:00に終わると仮定して、集合～参加者の確認などを行うと
　　オフ会の開始はライブ終了から30分後を想定しています。

　　ライブの終了時間が想定と異なった場合でも
　　集合場所へお集まり頂けますようお願い致します。
　　アンケート記入や物販などの都合もあると思いますので
　　集合場所にすぐ集まれそうにない方は早めに
　　幹事までご連絡くださいませ。
　
　

【場所・定員】
　　隠れ家個室ダイニング　季作
　　http://www.hotpepper.jp/strJ000861609/

　　会場から徒歩5分ちょいだと思います。
　

【会費】

【その他】
　　会場では名札を用意する予定です。

　　毎度、ライブ前には直前オフと称して近隣の喫茶店などで盛り上げています。
　　こちらは当日の流れで決める会なので詳細などはお伝えできませんが
　　物販後で時間を余らせるようでしたらご一緒できますので
　　希望される方は別途ご連絡ください。

　　当日はオフ会参加者と一緒に行動していることだと思いますので
