html5j Webプラットフォーム部 始動します。

html5j Webプラットフォーム部とは。

Webが生まれて25周年が経ちました。今では、WebはPCの世界を抜け出し、様々なプラットフォームで動き始めています。
スマートフォン、車載端末、デジタルサイネージなど、そのプラットフォームは多岐にわたり、今後もどんどん増えていくと予想されます。が、しかし。もちろん、全てのプラットフォーム上で同じコードが同じ様に動くとは限りません。
html5j Webプラットフォーム部では、その差異やノウハウを共有して、様々なプラットフォームで動かすための知識を培おうではないか！という活動をしていきます。

勉強会第1回目は、html5でネイティブアプリが実装できる3つのプラットフォームを学びます！

Windows8アプリケーション、FirefoxOSApp、ChromeAppsについて、それぞれ概要をお聞きした後にパネルディスカッションをしていただきます。

開発者向け、Webプラットフォーム初心者向けの内容となっています。
(※)JavaScriptの経験があるとより楽しめます

セッション概要

セッション1. HTML5 で開発する Windows 8 ネイティブアプリ

日本マイクロソフト株式会社　物江 修さん

Windows 8 ではネイティブアプリを HTML + JavaScript でも開発することができます。
「Windowsストア アプリ」と呼ばれるこのアプリの開発は、Web のエンジニアがいままで培ったスキルセットをそのまま生かしてアプリを作成することができるようになっていま す。
このセッションではWindowsストア アプリの具 体的な開発方法について紹介します。

セッション2. Basis of Firefox Apps

Mozilla Japan　浅井智也さん

Firefox OSのアプリについてざっくりお話しします。

セッション3. Chrome Appsの作り方

Google Developer Expert (Chrome)　吉川 徹さん

Web技術で開発できるChrome Appsの概要と簡単な作り方をお見せしたいと思います。余裕があればデベロッパープレビュー版であるモバイル対応のお話しも。

パネルディスカッション概要

セッションをしてくださった御三方に、パネルディスカッションをしていただきます。
対応OSや開発ツールからドキュメントやストア手数料、デプロイ方法など様々な点について赤裸々に語っていただきます！
時間があれば質疑応答も。。
