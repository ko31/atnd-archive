description

配色はセンスではなく理論です！
楽しみながら身に付けましょう。

ゲームや塗り絵、モザイクや折り紙など、
5つの楽しいワークショップで
効果的な配色テクニックを学びます。

色彩の持つ力をもっと効果的に使いこなしてみたい、と思っているあなたに。
配色理論は実践し、自分の目でその効果を確認するのが一番の近道です！

これまで開催してきた様々な色彩セミナーの中から、
楽しみながら配色テクニックを学ぶことのできる、
大人気のワークショップだけを集めて、たっぷり体験していただきます。

「色彩と配色 応用ワークショップ」は色彩検定協会の公式認定を受けた講師による、
文部科学省後援の「色彩検定」に基づいた講座です。

講師のデザインの現場経験に基づいたオリジナルテキストを用いており、
実践的で分かりやすい内容が特徴です。

2013年から開催している「色彩と配色 基礎セミナー」の応用編ですが、
初めてのご参加でもじゅうぶん楽しめるようになっています。

たった1日で、素敵な配色を作る自信を付けましょう！

※必要なものは、「はさみ」・「のり」・となります。
※「新配色カード199a」をお持ちの方はご持参ください

対象：

  • 中学生〜大人まで（※大人向けの講義あり）
  • 配色のセンスに自信がない方
  • 配色理論を楽しみながら学びたい方
  • デザインのスキルを上げたい方
  • DTPやWebのデザイナー、ディレクターの方
  • ファッションやインテリアなど日常生活で色彩を活用したい方
  • 色彩検定3級にチャレンジしてみたい方
