description

赤ちゃんと出かける時、少し大きいお兄ちゃんお姉ちゃんが赤ちゃんを踏みそうで怖く感じる事ありませんか？
ここは、ハイハイはまだできないゴロンゴロン赤ちゃんが、歩く走る子供のいない安全な場所でママとのんびり過ごせる、赤ちゃん広場です。
また、子育てサロンなどで、「お友達をつくらなきゃ！」「おしゃべりしなくちゃ！」と気負って疲れてしまう事ってありませんか？
この「タマんち」では、ママとベビーが二人だけでも十分くつろげるよう、女子に人気のマンガや本を準備しています。
無理に声をかけなくても、読書している間に、ふと会話が生まれるような場所です。
マンガ以外に、初対面ママが楽しみながら親睦を深められる、テーブルゲーム大会や、今しか取れないベビーコスプレ写真撮影会なども行います。
「出かけたいけど、赤ちゃんと一緒だと行くところがないわ～」と感じてるママさん！ぜひぜひ、遊びにきてください～ヽ(´▽｀)/
