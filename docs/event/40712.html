■概要
CloudStackとは、IaaS型クラウドを実現するソフトウェアとして、多くの企業や学術機関などで実績があるソフトウェアです。ユーザー会は、日本語でのCloudStackに関する情報発信・情報共有により、CloudStackの普及および人材育成に貢献することを目的としています。
今回は、Apache CloudStack 4.1に実装されている多数の新機能について解説していく事をメインに構成しました。CloudStackの基本的な解説に関しては、 CloudStack徹底入門 や、 CloudStack Overview を参照ください。
また、LT（ライトニングトーク）枠も募集しています。LT枠で話したい方はコメント欄ににお願いします。
