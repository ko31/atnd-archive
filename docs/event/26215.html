2日間限り、松戸駅前に劇場が出現します。
リーディング劇、落語(っぽいエンタメ)、コントの3本立て。

・コント
　出演：劇団彗星inc.
　文学座付属研究所の同期であった
　マサノソウと林流太により、2011年5月に旗揚げ。
　年に2回のペースで新宿を拠点に
　マサノソウのオリジナル作品を上演している。

・落語
　出演：イリュージョン亭メロンソーダ　
　いつもは普通のサラリーマン。
　エンタメをお茶の間にお届けする為、東弄西走しております。
　落語家ではありません。落語っぽいエンタメ。
　フワッとした感じでお楽しみください。

・リーディング劇
　原作：『松戸滅亡小説』吉野元春
　構成/演出：深谷水
　出演：佐藤南菜/村林悠美/脇田祐爾/赤石和弘/ホシマリ/吉野元春/渋谷朱美 他

　「 私(達)の(生まれた)町(が失くなった話) 」

　2011年7月のある午後、
　千葉県北西の、居住人数48万の町、松戸市がなくなった。
　爆発は松戸市の全域に及んで、
その境界線内を塗りつぶすように、きっちりと焼け野原にした。
　近隣の地域には傷一つ無く、ただ灰が降った。
　松戸は原野以前の状態に返され、私は帰ることができなくなった。
