内容

今回のLOB Tech Nightは弊社が Google Cloud Platformを利用し、
広告配信システムを開発した裏側を皆さまにお届けしたいと思います！

（当日のテーマを簡単にご紹介♩）

1.Kubernetesで実現できるスケーラビリティ、安全なデプロイ、コンテナ開発のメリット

2.広告配信で重要となってくるログのデータパイプラインなどの事例

3.TerraformやPacker + Ansibleを用いた構成の定義

4.プロビジョニングの自動化

この他にも本番運用までの道のりや、比較的新しい技術を使って良かったこと、苦労したことなどを余すことなくお話します！

スピーカーは弊社 Site Reliability Engineerチームのメンバーです。
