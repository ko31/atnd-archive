description

保育業界（保育園・幼稚園・幼児教室など）を志望する学生のための
就活イベント「ココキャリセミナー」を好評につきまたまた開催することになりました！

さらに、今回の「ココキャリセミナー」は、
前回までに比べ、内容が大幅にパワーアップ！

第一部：就活セミナー
第二部：ミニ合同説明会

という二部構成になっております。

第一部では、
現役の保育士さんや幼稚園教諭さんとざっくばらんに話ができる座談会の時間がありますので、
「保育現場」の話や「仕事への疑問」はモチロン！
「面接のコツ」なんかもドンドン聞いてみましょう。

第二部のミニ合同説明会では、
保育施設・保育企業の採用担当の方と直接話すことが出来ます。
つまり、内定に直結する、チャンスになっています。

第1回・第二回に参加した学生さんでも楽しめる内容になっていますので、
保育業界を志望している方は是非ご参加ください。
