description

3月30日（日）に大崎ゲートシティーにあるカジュアルなワインダイニングで海老原ファームより取り寄せている有機野菜を使って皆でベジスイーツを作る会を開催します！

会場内には色とりどりのサラダやベジフルドリンク、簡単な軽食もお出しします！

オーブンにスイーツを入れている間、ワインを飲みながら様々な方と交流をしましょう。

■海老原ファームのお野菜でスイーツ作り

■ベジフルドリンクも楽しめる！

■軽食ビュッフェとワインで皆と交流を深める！

初めてお会いする方同士でも交流を深める事が出来るコンテンツが充実！！

色とりどりで楽しい雰囲気の中、スイーツ作りを楽しみましょう。
