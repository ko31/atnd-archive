■NEC Cyber Security Tech Session

複数のアンチマルウェア製品で、最新のマルウェアなど100検体の検知率を検証してみよう
～あなたのセキュリティ製品評価用プログラム（疑似マルウェア）で、業界で評判のAIに挑んでみませんか～

NECは、サイバーセキュリティの若手技術者、或いはサイバーセキュリティに関心がある学生の方々と共に学び合い、次世代のサイバーセキュリティを共に担うために連携し、共創する場「NEC Cyber Security Tech Session」を立ち上げたいと考えています。 今回、このプロトタイプとしてのイベントを開催致します。

従来型のマルウェア対策製品では、近年増加する未知のマルウェアに対して 対抗することが難しくなっています。

そんな中、AI技術を用いて未知のマルウェアの検知を可能とするマルウェア対策製品が登場し、その実力が気になる方もいらっしゃるのではないでしょうか。

そこで、NEC Cyber Security Tech Session では、「あなたのセキュリティ製品評価用プログラム（自作疑似マルウェア）」や「気になる新種のマルウェア」さらに「誤検知しそうな正常ファイル」などをお持込いただき、AIを活用したマルウェア対策製品でどのような結果が得られるかを見てみよう！という勉強会を開催します。

業界で評判のAIに挑んでみませんか？ もちろん、マルウェア等をお持ちでない方やこれから勉強したい方のご来場も歓迎いたします。
