【演出家のための音響講座】

ー演劇は総合芸術であるー
演劇には、舞台に立つ役者だけではなく、
音響、照明、道具など、様々な効果をよりよく生かすことで良い作品が出来上がります。
昨年は、「演出家のための照明講座」として、照明が持つ力をどのように生かしていくのか、ということを知ってもらう講座を開催しました。

今年は、音響講座を行います。
音をどのように演出に生かしていくか、学びませんか。

＊このような人に＊
音響初心者の人
演出をすることになったけれども、音響について勉強したい人
役者をしているけれども、音響がどのように芝居に影響するか知りたい人
照明をしているけれども、音響はどのようにプランを立てているか知りたい人
音響をしているけれども、改めて音響について考えたい人
舞台監督をしているけれども、音響との打ち合わせで困っている人
制作をしているけれども、音響のことを知らない人
普段は芝居を観る専門だけれども、音響の仕組みを知りたい人
芝居をしていないけれども、音楽が好きな人
などなど

＊今回の講座は、実際に音響機器を操作する方法を学ぶものではなく、音響効果をどのように生かすか、ということを学ぶ内容となっています。
