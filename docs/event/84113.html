description

デジタルから離れて、少しロマンチックな時間を体験しよう。
昨年二度に渡って開催し、ご好評いただいたデジタル・デトックスが、
リニューアルして帰ってきました！

電子レンジやテレビなどの便利な生活家電に加え、スマホやPC、電子書籍など
インターネット機器の急速な普及が進んで久しく経ちました。
私たちの生活を便利に快適にしてくれるデジタル機器ですが、
こんなにも便利になってしまったからこそ、
IT疲れや電磁波の影響に不安を感じている方、
デジタル依存を自覚している方も少なくないのではないでしょうか。

現代では、私たちの体はデジタル進化のスピードに対応できず蓄電するばかりで、
ほんの少し前まで自然と触れ合うことで出来ていた放電がうまく出来ていないといいます。
デジタルを生活から切り離すことは出来なくても、上手に付き合っていくにはどうしたらいいのでしょう。

そこで亀時間では、デジタル機器が今よりもっと少なかった時代のロマンチックな時間を
今なお謳歌する鎌倉おじさん、“フーテンのうーさん”と一緒に
デジタルとバランスよく付き合う事を考え、鎌倉の魅力やアナログ生活の魅力も再発見して貰えたらいいな。と考えました。

このイベントがみなさんのデジタルツールとの関係を見直すきっかけとなり、
蓄電されていたデジタルむくみがストンと落ちるキッカケになれば幸いです。

協力：トランジションタウン鎌倉

【デジデト案内人】フーテンのうーさん

鎌倉で生まれ、鎌倉で育ち約半世紀。
持続可能なまちづくりを目指したトランジションタウン鎌倉というコミュニティの中心的存在。
「T=楽しく、T=つくる」をテーマに自然や人と繋がった心豊かな暮らしを実践中。
観光スポットではないところにこそ、本来の鎌倉の魅力がある事を伝える“鎌倉あるものさがし”主催。
