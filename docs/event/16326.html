6月30日のセミナーはこちら

2011年になって、本格的にAndroidOS搭載の携帯端末がリリースされ、いよいよスマートフォンが主流となる時代が近づいてきました。こうなるとさらに重視されるのがUIよデザインです。機能面もさることながら、UIやデザインがまずいとユーザーが逃げてしまうため、致命的なことになります。
ただ、このUIやデザインについては、残念ながらエンジニアが弱い領域です。そこでWebデザイナーの登場です。UIやデザインが本職のWebデザイナーが、開発の知識をあまり持たない状態でもちょっとした機能を搭載したアプリが作れたら素晴らしい！　そんな発想からこのセミナーは生まれました。今回は、Webデザイナー向けに”Titanium Mobile”について、「風呂グラマー」としても著名な増井雄一郎氏にたっぷりと語っていただきます。

※Titanium Mobileとは・・・公式サイトはこちら
Titanium Mobileは、あなたのWebスキルを生かして、JavaScriptで iPhone/ Android などスマートフォン向けのネイティブアプリを構築できるフレームワークです。UIやネットワークなどの基本機能に加え、マルチメディアやソーシャル機能など300を超えるAPIを使い、インタラクティなアプリや、ソーシャル、メディアリッチなど様々アプリを構築できます。

推薦図書　「Titanium Mobileで開発するiPhone/Androidアプリ」

＜講演概要と講師プロフィール＞

■第1部　Webデザイナーが、ちょっとした機能を搭載した Android / iPhoneアプリを作る方法

【講演概要】
JavaやObjective-Cが分からなくても大丈夫！　JavaScriptだけで、Android / iPhoneアプリが作れるツールキット、Titanium Mobileを使ってスマホアプリを作りませんか？
このセミナーでは、Titanium Mobileの概要から、実際のスマートフォンアプリ開発手法を詳しく解説します！　特にHTMLコーディングができたり、Flashの使えるWebデザイナーの方、必見です！
ちなみにiPhoneのMogSnapもTitanium Mobileで作られています。

【講師プロフィール】　増井雄一郎氏 Appcelerator Inc. Platform Evangelist　@masuidrive
2009 年までAjax、Ruby on Railsなどの技術を使った Web アプリケーションの構築や雑誌・書籍への執筆をフリーランスとして日本で活動。PukiWikiなどのソーシャルアプリケーションの企画・開発を得意とする。2008 年より渡米し iPhone、iPad関連のアプリケーション開発に従事。主な著書:「Ajax 実装のための基礎テクニック」(技術評論社)、「PukiWiki 入門まとめサイトをつくろう!」(翔泳社)
