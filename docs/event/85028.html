コンテンツ

1. CSP による防御とリアルワールドレポート収集（Jxck さん）

CSP は脆弱性を作り込んだ際に、攻撃の成立を水際で防ぐ強力な機能としてブラウザの実装も進んできました。 この強力すぎる機能を使いこなす上で、レポートの収集は欠かせません。

こうした CSP の代表的ディレクティブや、実際に収集されるレポートなどについて解説します。

2. 脆弱性発見者が注目する近年のWeb技術（西村 宗晃）

近年のWebブラウザに搭載されている技術を、その脆弱性とともに紹介します。

Server Push、HTTP/2、FlyWebの話をします。 体験型のデモもご用意する予定ですので、是非、お手元の端末にFirefox Nightlyをインストールしてご来場ください。
