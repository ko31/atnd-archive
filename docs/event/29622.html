Web やデザイン系の人が比較的多い（気がする）下北沢オープンソースカフェに、openSUSE (Linux distribution) ユーザー、つまり OS 系の人が集まって発表と Ustream 配信を行う（ちょっとカオスな）イベントです。

参加登録せずに来る人が多いので、参加人数は当てにしないで下さい。

チャンネルがいつもと異なりますのでご注意ください:
http://www.ustream.tv/channel/osscafe

お題

  • Linux サーバーのセキュリティを考えよう（仮）
    • あなたの VPS は大丈夫？


  • Web アプリケーションのセキュリティ対策（仮）

  • LinuxCon Japan 2012 に行ってきた

  • 発表者募集中です。(飛び入り LT も可)

アナウンス

  • IT勉強会スタンプラリー対象です
  • 開始時間前もカフェは営業しているので、早めに集まると良いでしょう。(午後一は別のイベントで混雑している可能性あり)
  • 会場での飛び入りが多くなりそうなため、参加人数は目安です。Facebook からも登録できます。
