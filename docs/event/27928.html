title 「ゼロからwordpressのテーマを作ってみよう！」セミナー
catch
description

Wordpressで、サイトを制作していたら、クライアントから「ココを直して！」「こここうしてくれたら嬉しいんだけど」と言われ、「え？それは。。。」と思った事はないだろうか。そんなのだったら、もう、テンプレートを使う意味ないじゃん。フルスクラッチで作った方が早いじゃんと思ったことはないでしょうか。けど、そんな時、あなたには、「Wordpressのテーマをフルスクラッチで作る知識はあるのでしょうか」テーマのタグを触った途端に、レイアウトがグシャっと崩れたり、あぁぁぁぁ〜と叫ぶことはなかっただろうか。今回、そんなcssやhtmlの知識が不安な方のために「Wordpressのテーマをフルスクラッチで作ってみよう！」セミナーを開催します。

＜こんな方にお薦め＞
→Wordpressのテーマをフルスクラッチで作ったことがない方
→Wordpressのテーマをフルスクラッチで一度でも作ってみたいと思っている方
→Wordpressのテーマをカスタマイズしていると、グシャとレイアウトを潰した事がある方
