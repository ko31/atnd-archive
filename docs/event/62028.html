description

女性の就業率は高まり、政府も「2030」と女性の社会進出を支援しているって言うけど、
正直そんなことはどうでもよくて、

「私はどうするんだっけ？」

入社して数年、それなりに仕事も覚えて、やりがいを感じることもある。

でも、私この会社で一生働くのかな？
結婚はなるべく早めにしたい。
出来れば子どもも欲しい。
それでも働き続けるのかな…？

デキる先輩ワーママたちは朝4時起きでメール処理や弁当作りらしい。
そんなの無理、そこまで頑張れない。
でも誰でも出来る仕事は嫌だ、何か専門性のある仕事をしたいって思う私はワガママ？

誰が決めるでもなく、自分自身で覚悟を持って、
「私」のワークとライフを考えるセミナーです。

一方的に講義を聞くのではなく、
簡単なワークをしながら考えてアウトプットして頂く形式です。
