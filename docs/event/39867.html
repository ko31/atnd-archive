< htmlday > 公式サイト

概要

初心者の方に対する HTML5 やその開発ツール （いまのところ Chrome Develper Tool を予定) とバージョン管理ソフト Git の勉強会・ハンズオンを行います．

主に Windows ユーザを対象に考えていますが Mac, または Linux でも何でもウェルカムです．

必要なもの

  • 公開しても良いメールアドレス

※GitHub と Bitbucket のアカウントを取得する際に用います．

※よく分からない人は Gmail のアドレスを取得してくればいいでしょう．

あると望ましいもの

  • ノートパソコン
  • ノートパソコンをインターネットに接続出来る機器 (モバイルルータ等)

会場に WiFi 環境が整っておりますのでインターネット機器は不要になりました！

※どうしても持参出来ない・持っていないという場合は持っている方とペアを組むことを考えています

  • 電源タップ

※持って来られる方は持ってきて下さると助かります．
