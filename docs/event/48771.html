description

◆趣旨：
　高性能・高品位のフィルムを作るためには、各工程において発現する現象を理解し、ポリマー、プロセス両面の適正条件を設定しなければならない。特に、光学用途では要求品質が厳しいため、フィルムの品位に関わる対策が重要となる。また、製膜・延伸技術はマクロな特性付与だけでなく、表面特性にも関係するため、表面の分析・評価技術の知識も大切となる。本講座では事例を紹介しながらそれぞれの対策につき解説する。
