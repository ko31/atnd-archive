description

習得できる知識
充填構造の評価方法、調整方法、高機能化例（二次電池、医薬錠剤、超硬セラミクス、被覆肥料、粉末化粧品など）

趣旨
粉末原料の個別粒子の物性のみならず集合体としての粉体特性の定量的評価に基づき、最適な充填操作により、素材/成型体の高機能化が系統的に行えることを示す

プログラム
１．充填構造の表現方法
　1-1.空隙率分布
　1-2.空孔径分布
　1-3.混合度

２．充填構造の調整方法
　2-1.供給
　2-2.圧密
　2-3.混合
　2-4.被覆

３．高機能化特性例
　3-1.強度
　3-2.液透過/浸透性
　3-3.溶解性
　3-4.徐放性
　3-5.光透過性/遮蔽性
