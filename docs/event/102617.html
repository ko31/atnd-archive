description

昼活に参加すれば、あなたの理想のライフスタイルが描けるようになり
将来の方向性がわかります。

現在の生活に不満は無く、仕事もは問題なくこなせている。
しかし、このままでは、何かいけないのではないか。。。

何か必要なのではないかと思っている人は結構います。

不確実な未来への不安でしょうか。

あなたのような人が昼活に来ることはよくあります。

それは良い兆しだと思います。

なぜなら、

「真剣に人生を考え始めた」という事ですから。

過去の延長線上にあなたの望む未来があるのなら昼活に来なくて結構です。

しかし、
そうでなければ環境を変えて新しい人に出会う必要があります。

人は人でしか気づけません。
あなたの行きたい方向をブラッシュアップしましょう。

是非、参加してみて下さい。

【この会の目的】
　同じ悩みや不満を抱えている人たちがどうやったら人生に変化をつけ前向きなライフスタイルに
　出来るか意見交換をし、価値観を変え、新しい一歩を踏み出せるような場所です。
　次にどんな知識が必要か分かり、行動を起こしてみたいなと
　思わせる場所として提供しています。（そうありたいと思っています）

【こんな人におすすめ】
　①夢ややりたい事があるが誰にも話せない
　②生産的な時間の使い方をしたい
　③前向きで応援してくれるコミュニティに参加したい
　④変化を起こすためにどんな知識、考え方が必要か知りたい
　⑤同じ価値観を共有できる仲間がほしい

【期待出来る効果】
　①進むべき道がわかる。
　②次にどんな知識や考え方が必要かわかる。
　③あなたを引き上げてくれる環境が必要だという事がわかる。
　④日々、積み上げる良い習慣を身に着ける事が大事だという事がわかる。
　⑤そのためにはあなた自身がちょっとづつ変わらなくてはいけない事がわかる。
　
【会の流れ】
　①自己紹介（今日来た目的）
　②今読んでいる本の紹介
　③フリートーク

【注意事項1】
　セールスや勧誘、コーチング、ネットワークビジネスをされるいる方、
　会の健全性と被害防止のためお断りしております。
　また、目的の無い参加も中止する場合がございます。

【注意事項2】
　参加申請は22時までにお願いします。
　申請後、参加確認をメールにてお送りさせて頂きます。
　その返信をもって参加確定となります。
　返信無く参加されても席が無い事がございます。
