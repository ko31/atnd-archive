M→Fes #とは

M→Production主催で行う映像発表イベントです。
おそらく2016年最後の映像上映イベントだと思うので、今年のまとめとなるような楽しいイベントにしましょう。
新作映像はもちろん、今年公開した映像作品であれば応募していただいてOKです！
自分が作った作品を公開して、"実はこんなところをこだわった""ここを見てほしい"という瞬間を共有しましょう！！！

ある人は言いました、「 早く最高になりたい 」

みんなで最高になりましょう！！！！！!

参加者の映像作品出展は必須ではございません。
会場で視聴したいという一般参加も大歓迎です！
一般参加の方はそのまま参加登録を、映像出展参加者の方は下記【作品応募】をよく読んでから参加登録をお願いします。

出展者数・映像作品の時間が未定のため開始・終了時刻が変更になる場合があります。変更があった場合は twitter 、もしくは ホームページ から連絡いたします。
出展登録受付は終了しましたが、一般参加登録は引き続き受け付けております。

タイムスケジュール

超重要な情報です。
12:30　開場
13:00　上映開始（第一部）
14:30　第二部開始
15:45　第三部開始
17:00　終了予定

会場への行き方、入り方は twitter にて案内しますのでそちらもご確認ください。
会場前に受付を設けておりますので、そちらで受付を行ってから入場となります。
忘年会参加者は受付時に参加費を頂きますので準備をお願いします。

【作品応募】 出展登録受付は終了しました

今後、参加登録フォームより出展登録をしていただきましても、一般参加者として扱いますのでご了承ください。
M→Fesでは出展していただける映像作品を募集しています。
今回から大学外からの、 外部参加 も受け付けております。
下記説明、注意事項をよく読んで、参加登録時のアンケートフォームに従って記入して下さい。

作品について

  • 1作品の長さは 1~5分程度 でお願いします。
  • ジャンルは自由ですが、著作権や公序良俗的なものは守ってください。
  • 同名での複数出展もOKです。
  • 推奨動画フォーマットはmp4形式(H264 / AAC)です。
  • 推奨解像度は1280x720(720p)又は1920x1080(1080p)です。
  • 推奨アスペクト比は16:9です。
  • 推奨フレームレートは30fpsです。その他一般的なフレームレート(23.976,29.97,24,25,59.94,60)でも対応します。

スケジュールについて

  • 11月20日(日)23:59 : 出展応募締め切り (出展者名とメールアドレスをお聞かせください)
  • 11月23日(水)23:59 : サイト発表用出展者情報・サムネイル提出締め切り
  • 11月23日(水)23:59 : 制作中作品の一次提出　(作品の予定時間を合わせてお聞かせください)
  • 12月11日(日)23:59 : 作品の最終提出　(上記規格に合わせて上映できる状態でお送りいただきます)

作品の一次提出は字コンテ・絵コンテ・音楽のみでも構いません。

提出について

  • サイト発表用出展者情報・サムネイル提出は、出展応募確認後、出展者様宛にメールを送りますので内容に従い返信にて提出をお願いします。
  • 一次・最終提出は ギガファイル便 や firestorage にて、同メールアドレス宛に連絡をお願いします。

メールにてアナウンスを行うため、常時メールボックスの確認をお願いします。

注意事項

  • 応募作品は新作はもちろん、今年公開した映像作品であればそちらの応募もOKです。 去年以前に公開した作品の応募は申し訳ありませんがお控えください。ですが、 去年以前に公開済みの作品をリメイク・再編集・一部使用するということは可能です。
  • 楽曲・素材・作品内容など、権利関係を含めた全責任は各出展者が負うものとし、上記内容について何らかの問題が発生したとしてもM→Productionは一切の責任を負いかねます。
  • 運営側で上映が不可能と思われるような性的・過激な表現が映像内に含まれている場合、出展者と相談の上、修正もしくは上映をお断りさせていただく場合があります。
  • その他、不明点・疑問点等ありましたら下記連絡先よりお気軽にご連絡ください。
