description

ネットショップの開設・運営のためのオープンソースシステムとして、日本国内では今やデファクトスタンダードとなりつつあるEC-CUBEのインストールから初期のカスタマイズまで、導入を検討している入門者を対象とした、宣伝を目的としない客観的なテクニカルセミナーです。

1.EC-CUBEのインストール

当てずっぽうなやり方では確実にまごつく、システムのインストール作業（特にレンタルサーバ環境）を、無駄なくスムーズに進めるためのポイントを解説します。

2.EC-CUBEのセキュリティ

ECシステムのセキュリティは設置者の責任です。EC-CUBEのようなオープンソースシステムではしばしば弱点となる、セキュリティの強化方法について、最低限度押さえておきたいポイントについて具体的に解説します。

3.EC-CUBEのSEOとコンプライアンス

競合サイトが多数乱立するネットショップのSEOは年々困難なものとなっています。その一方で消費者保護などコンプライアンス（法令順守）対策も欠かせません。それぞれの重要なポイントについて具体的に解説します。

4.EC-CUBEの初期カスタマイズ

モバイル対応やSNSリンクなど、いまどきネットショップでは必須のシステムカスタマイズについて、具体的に解説します。

会場：ITビジネスプラザ武蔵 5階 研修室2

秀和システムの「EC-CUBE公式ガイドブック基本編」の内容に沿って解説しますが、受講者が持参しなくても内容は理解できます。
