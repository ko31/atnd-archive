description

独立して自分の店を持ちたい！
飲食店で働く方にはそんな思いをお持ちの方も多いと思います。

しかし自分の店を開業・経営する上では
個人事業・法人の選択、事業計画、種々の届出、資金繰り、経理、そして確定申告など避けて通ることのできないハードルがいくつも待ち受けています。

これらのことを何となくで済ませた結果、あとになって問題に直面するというケースも少なくありません。
飲食店が3年以内につぶれる確率は90％にものぼると言われる中、しっかりとした知識をもって開業したいものです。

このセミナーでは飲食店の開業にあたって多くの方が頭を抱えるこれらの事項について、
開業準備からの流れを順を追ってわかりやすくご説明いたします。

また、簡単な個別のご相談にも無料でお答えします。
開業しようか悩まれている方、開業しようと考えているがどういった準備が必要なのかいまいちわからないという方など、
どうぞお気軽にご参加ください。
