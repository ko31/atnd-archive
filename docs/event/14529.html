前月２９日に開催した俺のハンバーグ主催のチャリティーイベント

その第２弾として、4/11（月）に花見イベントを開催します！！

内容としては、弊社運営のアイリッシュパブ「TAP BORROW??」店内に
どどーんっと豪華な桜を飾っちゃいますので、自粛ムードを吹き飛ばして
みんなで思いっきり花見を楽しんじゃいましょう！

なお、今回のチャリティーイベントは、うちのお店の常連さんである
@pink22258 さんの「これから新学期を迎える被災地の子どもを応援したい！」
という思いに賛同して開催決定しました。

ですので、参加条件としては
●会費として3,500円（飲み放題＋コース料理（ハンバーグ含む））と
●ひとり１品以上、新品の文房具を持参してください！

なお、集めさせていただいた文房具類は『スマイルバッグ（笑顔袋）』として
パッケージして被災地の子供に届ける下記プロジェクトにご提供します。

http://gift.negau.org/
