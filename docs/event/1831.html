■目　的：Movable Type全体に貢献し、明日のMT界を背負っていく、エキスパートの育成

■参加対象：

Movable Type 4.2x以降でサイト構築をおこなったことのある方を対象とする、中上級向けの勉強会です。
最低条件をクリアしてない方は、何も得ずに終わる可能性が高いです。
特に、教えてもらえると考えている方は参加をご遠慮ください
勉強会まで1ヶ月以上ありますので最低レベルをクリアされてから、ご参加ください。
ただし、いちばん必要なのはやる気です。条件をクリアされた方は私たちは歓迎します。

◇第１部：セミナー（45分予定）

    シックス・アパート株式会社　執行役員（製品企画担当）　金子 順さん

◇第２部：ワークショップ（120分予定）

Movable Type Motionを使った実践ワークショップです。MT界の実力者がグループリーダーを務める、当勉強会の人気コーナーです。前回の参加者からは、ワークショップの時間を長くという意見が多かったので、120分としました。

下記のクラスで学習をおこなっていただきます。

  1. Movable Type 5.0テーマの作り方（中級・技術者向け）
    テーブルリーダー　金子さん
    http://www.goodpic.com/
  2. プラグイン作成（中級・技術者向け）
    テーブルリーダー　藤本さん
    http://www.h-fj.com/blog/
    サブリーダー　奥脇さん
    http://www.tinybeans.net/blog/
  3. 管理画面のカスタマイズ（中級・技術者向け）
    テーブルリーダー　野田さん(アルファサード有限会社)
    http://junnama.alfasado.net/online/
  4. MT5の構造を知り使い道を考える（企画・ディレクター向け）
    テーブルリーダー　蒲生
    http://www.dakiny.com/

※内容については諸処の都合により、変更もあります。
