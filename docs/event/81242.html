豪華な洋館でハロウィンパーティ

会場の広さは約200㎡!
参加者同士が仲良くなれる謎解きゲームも開催！

高田馬場駅徒歩7分。
シャンデリアの素敵なヨーロッパのお屋敷のような会場で、ハロウィンパーティを開催します！会場の広さは約200㎡。様々なコンセプトのお部屋があります。
プロヴァンス調の部屋、鳥かご部屋、暖炉、欧州家具。
男女別に更衣室がありますので、会場で着替えることも可能です。
素敵な仮装の方には特別なプレゼント！

ハロウィンパーティが楽しい5つのポイント

  1. 参加者が仲良くなれる謎解きミステリーゲームを開催
  2. 特大ハロウィンケーキをご用意！
  3. 怪人やパフォーマーを招待予定
  4. DJが盛り上がる曲をプレイ
  5. 去年好評だった200個のキャンドルを、今年も飾り付け
