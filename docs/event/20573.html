description

ボードゲームをはじめて間もない初心者の方や何回かやってみて
面白そうだけどボードゲーム持ってないしメンバーもいない・・・でもやりた～い

そんな方にもっと経験して面白さを知っていただくために
『初心者のためのアナログゲーム交流会』を開催いたします。

できるだけ、わかりやすく説明いたします。
やったことないけど興味ある～って方でも大丈夫です。

経験者さんには、インストを手伝ってもらいます。
説明の勉強がてら参加いただけると助かります。

【プレイ予定のゲーム】（変更あり）
カタンの開拓者たち（ボードゲームといえばコレ！という超定番的なゲーム）
世界の七不思議（世界でゲーム賞を数々受賞している話題作）
6ニムト（単純明快、現在もっとも安価で購入できるカードゲーム）
インカの黄金（行くか戻るかの選択のみで遊べるバッティングゲーム）
マンマミーヤ（釜に具材とレシピを投げ込んでピザを作る記憶ゲーム）
ごきぶりポーカー（嫌な害虫を相手に押し付けるブラフ系ゲーム）
など

グループやご家族でのご参加もOKですよ。
参加希望の方は、コメントにて参加表明してください。
ゲーム経験や自己紹介を入れていただけると助かります。

お待ちしてま～す。(*･ω･)ノ
