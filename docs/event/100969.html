description

・あなたの個性とコミュニケーション力があがったら、
もっと人と深く仲良く付き合えるようになると思いませんか？
・一人一人の個性を伸ばしたり、
気付きあったりし合う人間関係や
コミュニティがあったらあなたは何をしたいですか？
・個性を発揮すればするほど、
協調性がなくなるかもって
不安になったことはありませんか？

私自身、いつもコミュニケーションで悩んでいました。
うまく人と話せない、緊張する、そんな自分を隠すために使っていたのが、ファッションでした。
外見を変えることで「できる人を装う」「友達が多そうにみせる」ことを追求してきました。
実際、それは成功していました。
だけど、根本的な自信はないまま。
おしゃれをし続けないと私は孤独なのではないかといつもどっかに不安を抱えていました。
しかし、そんな私があるきっかけで今までにない変化を手にいれることができました。
緊張せずに人と付き合えるようになり、いつも心が安心で、交流できるようになりました。
ファッション以外の話題も楽しめるようになりました。
１番変わらなければいけない変化をさしおいて、
外見の変化や、知識をつけることでの変化、
経験を積むことでの変化をしようとしてしまうのが今までの私でした。

しかし、本当の変化を身に付けたとき、
すべてがワクワク、ゲーム感覚でいきるようになり、
自分というアバターを使いこなし、
アイデンティティの変化からファッションも
今まで以上にもっともっと楽しめるようになりました。

根本の変化から、個性とコミュニケーションがアップするファッションセミナーを通して、
多くの人に今までにない変化のきっかけのヒントを伝えることができたらいいなぁと思っています。

こんな人向け

・チャレンジしてみたいけど、
大きな変化ってなんか怖いし、
小さな変化を積み重ねたい人
・自分のことを受け入れてほしい人
・ファッションに諦めがある人
・新しい自分に出会いたい人

ファッションコンサルタントをお呼びして楽しいセミナー形式でお届けします♪
