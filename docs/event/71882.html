■驚きの事実


厚生労働省は10月27日、「厚生労働白書」を公表しました。
それによると、今から１０年後の2025年以降は
男性は3人に一人が、
女性は5人に一人が、一生結婚できないと発表しています。

５０歳の時までに一度も結婚したことがない人は、その後結婚しないと仮定して、その比率を生涯未婚率としているのです。

これまでの統計で、５０歳までに一度も結婚したことがなければ、その後も結婚できないと想定されてしまっているのです。

今40代で独身のあなたは焦りましょう。
今30代で独身のあなたも急ぎましょう。

さらに、現状は厳しいものです。
結婚しても3組に一組は離婚、3組に一組はセックスレス。
幸せな結婚をしている人は極々僅かです。

■結婚力アップ


でも、安心して下さい。
今回は、潜在意識の力を上手に使い、あなたの運命を探り、その運命を望む方向に変え、幸せな結婚が出来るようにする方法を実習します。

運命は一筋の決まったものでなく、樹木の枝葉のようにたくさんの選択肢があります。
幸せになる選択肢を選びつつ楽しい人生を生きていく方法があるのです。

この方法で、私もこれまで出逢ってから一度も喧嘩したことがない幸せな結婚生活を続け、
多くの方々もこの方法で、幸せな結婚生活を手に入れています。

■実は、


多くの方に催眠や瞑想をお伝えしていく中で、大きな気付きを得ました。
それは、「不幸な結婚生活を続けていると、自分達が不幸なだけでなく、子孫も不幸にする。」です。

そこで、私は、意識の世界を学ぶ前に結婚して不幸になっていた結婚をやめ、再婚しました。
今は、とても幸せです。

意識の力を上手に使うと、真に望む理想のパートナーと出会い、幸せな結婚生活ができます。
結婚相手によって、人生がこんなにも変わるのかと痛感しました。

そこで、幸せな結婚生活をする方法をお伝えする決心をし、多くの方に幸せになって頂いています。

■セミナーでは、瞑想で、幸せな結婚生活を得る方法を見出します。


瞑想と言うと多くの方は、心を落ち着け「無」になる方法と思っています。

実際には、可能性は無限大です。

そこで、瞑想で、下記の様に、自分の運命を探ります。

運命の中の多くの選択肢の中から、幸せになる選択肢を見出し、
さらに、幸せになる選択肢を選び続ける方法をお伝えします。

■瞑想で、人生の全体像を観察する


あなたは、生まれてから死ぬまでの、自分の人生をしっかり見つめたことがありますか？

通常の意識状態では、既成概念が邪魔をして、真正面から見ることができないばかりか、雑念がわいてきて、すぐに気が散ってしまいます。

瞑想の意識状態になると、集中力、洞察力、ひらめき力が高まります。
そこで、人生の様々な場面をありありとイメージし、真正面から観察できます。

■瞑想で、あなたの人生の「体験」を観察する


これまでの人生で出会った重要な人々、
これからの人生で出会う重要な人々、
それら、一人一人と出会う意味が分かります。

■瞑想で、あなたの人生の「気づきや学び」を観察する


人生での様々な体験を感じ取り直すと、そこから様々な気づきや、学びを見いだせます。
人生は苦しみながら、学ぶだけでなく、楽しみながら学ぶこともできると分かります

■瞑想で、あなたの「人生の目的と、その実行方法」を見いだす。


人生での体験や、気づきや学びを元に探っていくと、人生の目的が分かります。
さらに、内なる英知とつながり、人生の目的の具体的な実行方法が分かります。
幸せな結婚です。
