「Office 365 って聞いたことがあるけど、何が出来るの？」

本セミナーは Microsoft Lync を利用したオンライン セミナーです。
以下のいずれかの環境が必要ですので、事前にご確認・準備の上ご参加ください。
＊安定した接続の為に、以下の “A” もしくは “B” をお勧めします。
また、可能な限り有線 LAN でのインターネット接続をお勧めいたします。

A. Lync Basic [ 無償 ] がインストールされている環境
B. フルバージョンの Lync クライアントがインストールされている環境
C. ブラウザーで参加可能な Lync Web App [ 無償 ] が動作するブラウザー/ OS環境
