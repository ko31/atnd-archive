description

※定員15名となっています。参加できなくなった方はきちんとキャンセルして、キャンセル待ちの方に枠を譲ってあげてください。

ブロガー同士の交流を目的としたイベントです。ブロガーみんなで渋谷のコワーキングスペース「Lightningspot」に集まって、まったりブログを書いたり、交流したりしましょう。他の人がブログを書いている様子を見学して、参考にしちゃうのもありです。

場所代として1人500円を現地にて徴収します。電源・無線LANあります。

今回はコワーキングスペース「Lightningspot」から無料のハンバーガー差し入れがあります。

ハンバーガーを食べつつ、ブログを書いたりしましょう！
