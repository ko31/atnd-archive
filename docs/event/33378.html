RubyMotion再び

RBCでは、6月、7月、9月とRubyを使ったスマホアプリの開発をテーマに、イベントを頻繁に開催してきました。今月のイベントは、この流れに沿って（？）、7月のイベントで取り上げたRubyMotionを再び扱います。

7月で扱ったときと比べて、RubyMotionで開発したiOSアプリもいくつかリリースされ、日本語の情報も色々と出てくるようになりました。

RubyMotionについて、いま語らずして、いつ語る？といった状況になったかと思います。

RubyMotionのおさらいから、開発環境面、実際に作ったiOSアプリのコード紹介など盛りだくさんの内容を用意しています。

RubyMotionを使っている方はもちろんのこと、ちょっと気になる方もふるっての参加をお待ちしています。
