来場いただくにあたって

ソーシャルニュースの未来勉強会＠mixiですが、会場のビルの1Fに19:30までお越しください。もし遅れてこられる場合には、twitter @hiroumi、または090-6530-2001までお電話ください。
飲料、食べ物、お菓子の持ち込みなどもOKです。

よろしくお願い致します。

定員を15人から20人に増やしました。

イベント概要

株式会社ミクシィ広報の徳田さんと、ユーザー参加型ニュースサイトnewsing.jpを運営している株式会社マイネット・ジャパンの藤川さんにmixiチェックや、ニュースのソーシャル化に対する対応について話をしてもらった後、参加者全員でソーシャルニュースの今後、可能性について座談会形式でディスカッションをしてきます。

参加の条件はひとつだけ、ブログを書いていること。

今後のソーシャルニュースについて、皆さんと一緒に考え、ぜひブログやソーシャルサービスを通してイベント後も議論を提示したり、紹介していって頂ければと思います。

お気軽にご参加ください。
