天空要塞からの生還

description

「天空要塞からの生還」は、高さ7mのアスレチックを使ったアドベンチャー型脱出ゲームです。プレイヤーは、暗号やパズルが仕掛けられたアスレチック施設に閉じ込められます。暗号やパズルをすべて解き明かし、脱出の鍵を手に入れることがゲームの目的です。前作の「天空要塞からの脱出」とは異なり、暗号やパズルを解くための情報は、上空にそびえるアスレチックだけでなく、様々なところに仕掛けられています。また、2人の世界チャンピオンという豪華なゲストも登場し、会場をより一層盛り上げます。ボリュームたっぷりの“地獄の試練”があなたたちを待ち受けています。

＜ストーリー＞
核戦争によって文明が滅んだ時代…。そこは血と暴力によって、強者が弱者を支配する世界だった。
北部で勢力を拡大するスカイボーン軍団。あなたたちはその軍団の一味に囚われてしまった。連れてこられたのはスカイボーン軍団の砦。そこはかつて“天空要塞”と呼ばれた軍事施設だった。
「今からお前たちにいくつか試練を与える。この試練を突破できたものは、わがスカイボーン軍団の誇りある兵士として迎え入れよう。突破できぬものは死あるのみ！」
どうやらここから生還するためにはひとまずその試練とやらをなんとか突破しないといけないらしい……。
あなたは果たして生還することができるだろうか？
