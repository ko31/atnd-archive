イベント概要

17時からのスタートですが、飲み放題タイムは18：30からになっています。18：30までは飲みたい人が各自でお支払頂くシステムです。

17：00〜18：30　プレ飲みタイム（各自でお支払）

  • 18時半まではビール一杯いくらで希望される方にというスタイルになります。そのビールも八蛮のご厚意によりなんと1杯300円！
  • つまり駆けつけ10杯飲んでも3,000円！（笑）

18：30〜21：00　飲み放題＆食べタイム（会費2,000円）

  • 18時半からは八蛮の定番、飲み放題＆「俺が作ったメシを喰えっ！」システム。「好きな物だけ食べるなんて自分の可能性を小さくするだけだぜ」という大将の思想があるかどーかはともかく、結構色んな食った事がないようなウマいもんが出てきます。
  • 会費はお一人様2千円ポッキリで後は言い出しっぺが持ちます（笑）。人数は結構入りますので、当日ご予定が無い方、あっても昼間アキバに行って夜はヒマという方、旅から夕方には東京に戻るという方など、一杯やりに是非どうぞ（笑）。
  • イベント内容（予定）
    1. 各自持ち寄ったアニソン音源を大音量で流しまくる！
    2. アニソンマイスター（仮称）によるディープなアニソントーク
    3. プレゼントじゃんけん大会（やるかもしれない？）

基本的には飲みながらワイワイと音楽を聞くイベントです。

開催場所

地ビール居酒屋「八蛮」

東京都中央区銀座2-14-9

  • 日比谷線・浅草線 東銀座駅からのアクセス
    • 東銀座駅3番出口(歌舞伎座の前）を出て歌舞伎座越えて一つ目の角を左に曲がって下さい。そのまままっすぐ歩いて二つ目の交差点を越えた左手に八蛮があります。※歌舞伎座改修工事に伴い、現在3番出口は封鎖中ですので、5番出口をご利用ください。
  • 有楽町線 新富町からのアクセス
    • 新富町駅1番出口（中央区役所の前）を出ていただき銀座ブロッサムの方に真っ直ぐ三吉橋を越えていただいてセブンイレブンを越えた角を左に曲がって50m歩くと右手に八蛮があります。

銀座の地ビール　自家製八蛮ビール

八蛮で作っているヴァイツェンは南ドイツで作られている小麦のビール。バナナのようなフルーティな香りと酵母が浮遊していることによる白濁が特徴です。

苦味が少なくフルーティなので、ビールが苦手な方でもきっと大丈夫。ぜひ、一度飲んでみてください。もしかしたらこれからビールが病みつきになっちゃうかもしれませんよ。

お店の中でビール製造

八蛮の大将はお酒好きで、ビールが好き。 とうとう醸造免許を取ってしまいました。
額に汗して、本気で作ってます。だから、本気で美味い地ビールです。

なんと店内にビール工場を用意！
だから、普段は決して飲むことの出来ない新鮮なビールを味わうことができます。
1回の仕込みに、かかる時間はなんと12時間！前日に仕込み水を沸かすところから、準備を始めて翌日11時に仕込みスタート。
仕込み終了は夜11時になります。

大手企業のビールと同じ醸造工程は全て手作業。
小さな小さな銀座の醸造所で、飲み放題でお得にビール楽しんでいただくための一工夫です。
