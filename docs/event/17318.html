description

復興支援「福島・会津蕎麦と地酒を楽しむ会（そば会）」を7月20日に開催致します。前回６月２２日の蕎麦会は５１名様の御出席を頂きました。売上金の一部を日本赤十字社を通じて寄付させて頂きました。御出席いただいた皆様、本当にありがとうございました。

会津蕎麦と地酒を楽しむ会の内容です。

御料理：各市場直送の新鮮な御刺身と旬の食材を使った御料理五品。
お蕎麦：福島県会津産の石臼挽き蕎麦粉を手打で御蕎麦に致します。
　　　　食べ放題ですので、心ゆくまで御堪能下さい。
お 酒：南会津町（旧南郷村）の地酒、「花泉」を御用意しております。
花泉は本数限定ですがビール焼酎等は飲み放題です。
時間　受付18時　　会費：５０００円
開始18時30分　　終了20時00分
