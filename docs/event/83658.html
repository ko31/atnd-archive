Raspberry Pi や NanoPi (Piボード)で無線LANのアクセスポイントを作りましょう。

Piボードは電子工作に使うことが多いですが、LinuxボードとしてにWi-Fiドングルを刺して、hostapd でアクセスポイントを作ることができます。
無線LANアクセスポイントは￥2000ぐらいで買えてしまいますが、自分でログを取ったりいろいろ組み合わせたりすることのできるように作るには hostapd でやってみましょう。

今回はPiボードを持ち寄り、インストール・Linuxのおさらい・hostapdをやってみます。
誰でもWelcomeです！！

参加無料です。Piボード持ってない人は、会場でNanoPi-NEOセットの購入もできます。

やること

・Piボードを使い、
・配線してSDカードから起動
・sshでログイン
・Linuxのオペレーションざっとおさらい
・無線LAN設定してみる
・hostapd設定してみる
・ipforwardの設定してみる
・DHCPDの設定してみる
・natの設定してみる
・質疑応答
