いつも笑顔でこんちくわ。@conchikuwaです。

お久しぶりにですが、市川ブログ合宿を開催したいと思います。

市川ブログ合宿もこれで3回目となります。

川崎のブログ合宿にも参加させていただいておりますが、やっぱりブログ好きが集まって、ブログ書いたり、呑んだり、喋ったりするのは楽しいものです。

場所は今まで通り千葉県市川市の私の事務所で開催します！！

みんなで集まってブログ談義するのは楽しすぎる！！

ブログ合宿を開催する理由はこの一言につきます。

なんでわざわざ集まってブログ書くの、呑むの、喋るのって思う方もいるかもですが、いやマジ楽しいっす。参加した事ない人は参加してみて下さい。楽しいから。多分。。。

特に出し物はありません。

他のブログ合宿ではLTやったりしておりますが、市川ブログ合宿では特に何も考えていません。基本的にゆる〜くかる〜いノリの集まりです。

ブログ書きたかったら書く、呑みたかったら呑む、喋りたかったら喋るって感じ。

ただブログ好きが集まって、ブログを書いたり、ブログをツマミに呑んだり喋ったりするだけで、いろいろと科学変化、相乗効果があって楽しいと思います。

市川ブログ合宿

なんで千葉県市川市かというと私の事務所があるから。

ただそれだけです。

駅前とかで安く場所を貸してくれるところもあったけど、市川市在住じゃないとダメとか、アルコール持込み不可だったりしたので、やっぱりうちの事務所にしました(^^♪

千葉県といってもちょっとガンバレば東京都に歩いて行けるとこなんで、そんなに不便じゃないと思います。

ちょっと駅から歩くけど。。。（徒歩15分くらいです。）

事務所の外観は↓な感じ。（リンク先では建物にリカーハウスNAGASHIMAという看板が写っていますが、現在、看板には文字がないです。）

市川ブログ合宿の内容

まぁ特にイベントも何も無く、ただ集まってブログを書くもよし、ブログについて意見交換するもよし、気楽に楽しみましょう。

  • 日時：2012年12月9日（日）11：00〜17：00
  • 場所：花RiRo事務所
    〒272-0025　千葉県市川市大和田3-18-3　1F
    電車 → 総武線本八幡駅、都営新宿本八幡駅、京成八幡駅 徒歩15分程度
    車 → 京葉線 市川インターから10分程度
  • 人数：最大12名（最低遂行人数一人。でもそれは寂しい。。。）
  • 参加費：タダ（飲食物は各自負担でお願いします。）
  • 持ち物：ブログを書く道具
  • 対象者：ブログを既に書いている人
  • やる事：ブログを書く。（できればブログ合宿について1記事、以外にもう1記事。まぁこのあたりも適当に。。。）、呑む、喋る等々

その他注意事項

大した事ではないですが。。。

  • 電源はありますのでご心配なく。
  • 無線LANはfonルータがありますが、何人繋げるかよくわからないので、モバイルルータをお持ちの方はご持参下さい。
  • 車でお越しの場合、駐車場はありませんが駅から事務所までの間にコインパーキングがあります。値段は1日500円〜1000円くらい。
  • 参加費は無料ですが、飲み物や食事は出ません。各自用意下さい。
  • お湯は沸かせます。電子レンジあります。火は使えません。
  • 事務所脇にドリンクの自販機あります。徒歩5分くらいの場所にデイリーヤマザキあります。
  • 特に予定はしていませんが、希望者がいれば合宿後、呑みにでも。

Facebookにイベントページを作ります

合宿参加者のみの秘密のFacebookのイベントページを用意します。開催まで1ヶ月弱ありますので、それまでそちらで交流していただければ幸いです。

お願い

ATND経由で連絡が取れるよう、参加登録時にメールアドレスも登録をお願いします。またTwitter経由でFacebookのイベントページの案内をさせていただきますので、Twitterにて私@conchikuwaをフォロー願います。またフォローの際にブログ合宿に参加するよとツイート頂ければ幸いです！

とにかく集まってブログを楽しみましょう！

合宿というと気合い入れて、一生懸命ってイメージがありますが、市川ブログ合宿はそんな事はなく、ゆるく楽しくをモットーに開催できれば幸いです！！
