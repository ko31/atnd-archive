description

関東では @kinoG_ts さんがいい感じのペースで「セッション練習会」を開催していますが、関西でももっとこういうのやりたい！！
ということで、始めてみることにしました。

ジャズやってみたい人、初級者から上級者まで何でもあり！楽器も問わず。歌の人は、やりたい曲の楽譜を4,5枚コピーを持ってきてね。

スタジオにはグランドピアノと、ドラムセットがあります。アンプはありますがギターやベースは持ち込みが必要です。

なお、基本的に出自が演奏系(+ジャズ系)ニコ生コミュから来ていたりしますが、ニコ生主さんに限らずどなたでもokです！

ジャズスタンダードを軸にしていますが、たとえば数人での参加で、みんなで合わせられるとか、リードシートが5枚ほどある！とかならジャズ以外の曲でもOK!　

趣旨はみんなでセッションを楽しもう、なのでそのへんよろしくです！

お金：スタジオ代を割り勘
二次会：終わった後ノリで考える
