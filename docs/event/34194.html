注意点

今回の開催場所は、株式会社Fusic（福岡本社）さんです。
ということで、Fukuoka.php二回目の出張開催で、かつ、「PHP Matsuri 2012」のふりかえりイベントとなっており、しかも、初の土曜日開催。

※今回はFusicさんのご厚意により、会議室のスペースをご提供頂けることになりました。

趣旨
サブタイトルに「～ Ato no Matsuri ～」を冠し、やりますよ5回目！
PHP Matsuri 2012でこんなところが楽しかったよ。
とか、こんなこと話してきたよ。
とか、こんなことで悔しい思いをしたよ。
とか、こんな素敵な出会いがあったよ。
とか、深夜の闇が凄かったよ。
とか、とか、とか。
参加した人も、参加できなかった人も、参加したかった人も、来年参加したい人も、どうぞどうぞ奮ってお集まりください。
この場であの時の「雰囲気」を共有しましょう。

内容（※以下のスケジュールは確定ではありません）

13:00～18:00 自由時間(開発タイム 出入り自由です)
18:00～20:00 発表時間
20:00～22:00 懇親会

自由時間はphpmatsuriの雰囲気さながらに、各自の開発時間、翻訳時間、作業などにあててください。ファミコンやビールの持ち込み大歓迎です。この時間中は何時に来てもかまいませんし、途中で抜けても大丈夫です。
同時開催で、天下一ビール会を開催して最強のビールを決定する予定です。あなたの知ってる最強のビールをご持参ください！（冷蔵庫はないです・・・）

発表時間は、自由時間で作ったものの発表や、phpmatsuriで言い足りなかったこと、phpmatsuriに参加した感想など、自由に発表する時間です。

懇親会は会場、もしくは会場周辺のお店で行います。

セッションやりたい方、絶賛募集中です。
@fukuokaphp 、 @akase244 、 @cakephper 宛て（どのアカウントでもOKです）にツイートして頂くか、もしくは、 Facebookグループ に書き込んで頂ければと思います。

参加費用

勉強会＆懇親会：食べながら＆飲みながらやるので、3000円くらいを集める予定です。
