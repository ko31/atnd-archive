description

エイプリルフールネタを作るハッカソンです。

エイプリルフール直前の3月30日〜4月1日早朝までを使って、渾身のネタを作りたいと思います。とりあえずぼくんちがシェアハウスなので、そこのリビングでやろうと思います。途中参加歓迎。完成したら発表して（あるいはせんと）帰宅でおｋ。ぼくは出社直前まで作り続けます（ｷﾘｯ

テーマは「エイプリルフール」です。

作るものはWEBサイトでもアプリでもイベントでも動画でも、馬鹿げてさえいればなんでも構いません。
ひとりで作ってもチームで作っても構いませんし、何作品作っても構いません。
別になんも作らずに、来るだけ来てずっとアニメ見ててもだれも怒りません。

参加費

無料（ご飯は出ません。食材をお持ち頂ければ台所をご利用頂けます）

タイムテーブル

こちらはあくまでも目安です。
最初から最後まで居る必要はありません。

食事のタイミングで作品発表の希望者を募りますが、
秘密のままローンチして1日の正午にネタばらしとかもありです。
