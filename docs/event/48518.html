■講師のご紹介

酒巻　瑞穂（Mizuho Sakamaki）

物流系での開発、主にデータベースからバックエンド技術の構築/運用に従事する。
現在は.Net系技術を用いてのフロントエンド開発に携わっており、Microsoft系フロントエンド技術に精通している。
日本で開催されたTest the Web Forward Tokyoにてテストケース数の部門にて表彰されておりコミュニティー活動においては、エンタープライズ向けWeb技術者コミュニティ 「html5jえんぷら部」 のスタッフとして、業務システムエンジニア向けのセミナーや勉強会に貢献している他FireFoxOS CodeReadingコミュニティーメンバーとしても活動している。
