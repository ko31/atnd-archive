description

新緑の姫路城を歩いて地図を作りませんか〔雨天決行〕

姫路城周辺を歩いてOpenStreetMapの地図を作りましょう。

  • 参加費: 無料（姫路城下町図書館への光熱費・飲み物代のカンパ歓迎します）
    どなたでも参加できます
  • 会場: 姫路城下町図書館
    • http://www.openstreetmap.org/?mlat=34.8329&mlon=134.6912#map=16/34.8329/134.6912
    • 道順: http://umap.openstreetmap.fr/ja/map/openstreetmap-in_3492#17/34.83044/134.69135

必要なもの

  • 筆記用具。クリップボードがあると便利。
  • 外でマッピングをする人は動きやすい服装。
  • OpenStreetMapのアカウント(OpenStreetMapの編集にはアカウントが必要になります。登録されていない人は事前に作成しておいてください。
    • OpenStreetMapユーザー登録: https://www.openstreetmap.org/user/new )
  • データ登録・編集用のノートPC(OpenStreetMap編集ソフトのJOSMがインストールしてあると、よりベターです。)かスマートフォン・タブレット
    • PC: JOSM: http://josm.openstreetmap.de/
    • iOS: Pushpin: https://itunes.apple.com/jp/app/pushpin-osm/id565850639?mt=8
    • Android: Geopaparazzi: https://play.google.com/store/apps/details?id=eu.hydrologis.geopaparazzi&hl=ja
  • 日焼け対策は忘れずに。
  • 当日は事故等に遭わないよう充分に注意するとともに、万が一の事態に備えて各自で必要に応じて保険等の手配を行ってください（全体で保険加入は行いませんので注意してください）。

  • ノートパソコンでの作業をされる方はテーブルタップをご用意ください。
  • 光熱費、飲み物（備え付けのコーヒー）のカンパ（数百円程度）をお願いします。
