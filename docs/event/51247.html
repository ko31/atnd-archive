title eco検定合格対策「直前ポイントアップ講座」（東京・渋谷）
catch 想定問題対策で得点力10点アップ
description

　ＮＴＴドコモなど携帯3社のエコ検定対策公式サイトの運用や各種環境セミナーなどで環境教育に取り組んでいる環境教育フォーラム21は第16回eco検定の合格に向けて、検定直前での想定問題対策で得点力を10点アップし、合格ライン突破を目指す「直前ポイントアップ講座」を東京、大阪、名古屋、横浜で開催します。
　直前ポイントアップ講座ではできるだけ、数多く問題を解くことで、正確な知識を身につけていただくとともに、eco検定の出題スタイルに慣れていただきます。重要な環境問題の時事ニュースも問題形式でチェックします。これまで参加された皆様からは大変、好評を得ています。
また、「土日は仕事」という方のために平日も開催します。

※開催時間は原則、午前 10時から午後 4時半まで ）
