■コンセプト

　社内での新規事業の立ち上げ、フリーランス、社会起業、ベンチャー起業など、「仕事を自ら作り出していくこと＝創職」をキーワードに、21世紀における日本や世界がどのように変化していくのか、また、新しい社会における働き方、ライフスタイルについての議論を行い、新たな社会システムを構築するための方法を模索する。

■トーク内容（キーワード）

・多様な働き方、新たなライフスタイルの出現
　- 新卒一括採用の弊害（本当に就職氷河期なの？）
　- 会社組織への適合不全（埋没ニート問題）
　- 組織社会からネットワーク社会へ（組織のマネジメントからネットワークのオーガナイズへ）
　- 「キチキチ系」から「ワクワク系」へ

・ポスト消費社会とは
　- 消費社会＝市場で貨幣で交換されるモノやサービスを提供するための仕事
　- 創造社会＝承認、貢献、つながりによる交換

・創造社会に出現した新たな「部族」
　- iPhoneなどのITツールを駆使し、SNSなどで表現活動を行う「都市ノマド」の出現
　- twitterなどの新しいメディアを使いこなす「露出社会」の到来

■パネリスト

・ 玉置沙由里さん @sayuritamaki 女。MGの日記。

　（ライフスタイルクリエーター。大阪府生まれ。京都大学総合人間学部卒業後、野村総合研究所で経営コンサルタントとして複数のプロジェクトを経験。東京・田端でシェアハウス「LSCH」（Life Style Creator’s House）を運営する。）

・ 田中冬一郎さん @waon2007 輪音プロジェクト

　（輪音プロジェクト代表、大阪市のアートサポートセンター中之島4117企画委員、 FP。会社員生活（銀行員）の傍ら、関西各地のアートNPO団体にスタッフとして 参加した後、2007年から“若者に表現の場所と機会を提供する”NPO団体 「輪音プロジェクト」の代表としての活動をスタートさせている。 また個人活動として、FPとしての知識を活かし、金融セミナーの講師として 定期的に金融リテラシー啓発イベント“オカネのお話”を各所で主宰、開催。）

・ 増井街穂さん @maho_net_cafe maho-net cafe

　（大阪・南船場でカフェ「maho-net cafe（マホネットカフェ）」を運営するマホネットの代表取締役。デザイン専門学校2年在学中の2010年5月に起業、現在は休学しカフェ経営に専念する。）

・ 小笠原太一さん @factosaka Facto
