第3回は、

  • Chapter3 Types

の続きからスタート予定です。

必要なもの

  • Scalaの処理系(2.7以降。2.7.5が好ましい)がインストールされたノートPC
  • Scala Language Specification(PDF)またはそれ(の一部)を印刷したもの。当日読む範囲について、事前にざっとでも良いので目を通しておいていただけるとありがたいです。
  • 構文規則(BNF等)の読み方は、知っているという前提で。
