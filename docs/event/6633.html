エキスパートPythonプログラミング の読書会を開催します。

本の文章そのものの読み合わせはせず、簡単な解説と質疑応答が中心のワイワイ参加型です。このため、事前にざっと読んでおくことをお勧めします。またPythonが動作する環境（ノートPC）があると良いですが必須ではありません。

初回は1章から始めたいと思います。参加者の状況次第で2章に入るかもしれません。

今のところ月イチ、月初火曜開催を予定しています。
