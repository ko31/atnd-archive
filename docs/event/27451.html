大人気アプリ"maskingtape"でiPhoneアプリ開発を学ぼう


女性を中心に大人気の「maskingtape（マスキングテープ）」
http://itunes.apple.com/jp/app/maskingtape/id412845850?mt=8

このアプリの製作者を講師に迎え、iPhoneアプリのプログラミングを学ぶワークショップです。本講座は5月よりサービス提供開始予定の「iPhoneアプリ開発スクール RINGO（リンゴ）」の特別モニタリングイベント（通常価格60,000円予定・全3回）です。

本サービスでは実際にリリースされているiPhoneアプリのソースを教材とし、Objective-Cによる実践的なプログラミングを学びます。iPhoneアプリ開発やってみたい、どういうふうに勉強したらいいのかよく解らない、そのような方にぴったりの内容です。一緒に学んでみませんか？

丁寧なレクチャーを提供させていただくため、少人数制になっております。どうぞお早めにお申し込みください。

受講条件

  1. iPhoneアプリ開発初心者（なんらかのプログラミング言語経験がある方におすすめします）
  2. 下記の日程にすべて参加できること（グループワークのため、時間厳守でお願いいたします）
    ・4/9(月) 13:00 〜 16:00
    ・4/16(月) 13:00 〜 16:00
    ・4/26(木) 13:00 〜 16:00
  3. 作業用PCとして、下記のスペックのMacをご準備いただけること
    ・Mac OS X 10.7(Lion)搭載　Intelベースのもの ・無線ネットワークに接続できること 
  4. 開発ツールXcode4.2以降をインストール済であること
    ・本講座はプログラミングを中心にお伝えしますので、デベロッパ登録の有無は問いません 
  5. モニターとしてアンケートにお答えいただけること
    ・アンケートにご記入いただいた内容は、プロモーションに使用させていただくことがあります

講師

高木 和夫
中学のころ「パピコン」という名前のPCを買ってもらったことからプログラミングの世界に。友人達とNakasone Softというグループを作って自作ソフトを出そうとするも断念。その挫折感からかその後長らく眠っていたが、大学院時代になってNetRezonatorというアップレットで文化庁メディア芸術祭優秀賞などを受賞。その後もぽつりぽつりと自作を発表し続けています。
iPhoneでのアプリ開発歴は2年弱。自分のアイデアを形にし育てていく楽しさを伝えられたら、と思っています。よろしくお願いします。

柴木 みずほ
iPhoneアプリ開発は初めて11ヶ月。
昨夏「1TapTimer＋」をリリース、作ったアプリが日々世界のあちこちでダウンロードされることにワクワクの毎日です。5月に起業予定。ソフトウェア会社で社員教育を長く担当した経験を生かして、プログラミングを始めたい方へiPhoneアプリの開発支援サービスを提供します。
