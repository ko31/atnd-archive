description

知る人ぞ知る長岡の珍スポット、『居酒屋　夕暮れ』でバーベキューやるよー。

道具は夕暮れ備え付けのものを使用。

肉・その他の食べ物は持ち込みで。

その日の流れ的には１８時に集まれる人だけで駅に集合して、イトーヨーカ堂で買物してから店に行こうと思います。

野菜は俺が持っていこうかなーって思っているので、買うのは肉中心になると思うけど。

一応お店の中庭でバーベキューという感じなので、雨が降ったら中止になるかも。

そん時はまあ適当に飲みにいこうぜ。

一応調整さんで日程調整したけど、やっぱみんなの予定合わせるの難しいねー。

仕方ないから独断で３１日土曜日に決定させてもらいました。

今回来れない人も、次回はよろしくお願いしまーす！
