description

約40棟のシェアハウスを運営するRバンクが手掛けた新たな物件、34戸男女mix専用の大型のシェアハウスが京急線「花月園前」駅より徒歩4分。京急東北線「鶴見」駅より徒歩14分の場所に10月よりオープンします。コンセプトは大人のクラブ活動。その出来立てのお洒落なシェアハウスの中で、学生の頃の様にクラブ活動をしてみませんか？シェアハウスに興味がある方も、クラブ活動に興味がある方も、ぜひご参加ください♪

■スケジュール

11:00　スタート（10:30開場）
11:25～12:30　アウトドアクラブ～見晴しのいいウッドデッキでBBQ～
　食材の準備や炭の火おこしなどの準備から協力して、みんなでBBQをやりましょう！
12:30～13:00　トランプクラブ
　みんなで食後にババ抜き大会！シェアハウスの生活はこんな感じ？
13:00～13:30　長縄飛びクラブ
　「yokohama 花月園倶楽部」は屋上も自慢。バドミントンや長縄飛びが出来る位広いんです！
14:00～　内覧ツアー
　一足先にシェアハウスをご見学頂けます。
14:30～　個別相談会
15:30　終了
先着順でお申込みの受付を致します。

■日時 　10月25日（日）11:00～15:30
■場所　神奈川県鶴見区岸谷
・京急線「花月園前」駅　徒歩4分
・JR「鶴見」駅　徒歩14分
・JR「鶴見」駅西口よりバス乗車、「柳町」下車　徒歩1分

■料金　500円（食費として）　※お飲み物は個別でご用意ください。
