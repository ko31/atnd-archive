テーマ「Windows Phone 7 アプリ開発」

近々auから発売となっているスマートフォン「Windows Phone 7」について、アプリ開発の勉強会を開催します。C#を少しでもかじったことのある人も楽しめるセッションにする予定です（反対にレベルの高いセッションはしません・できません）。Windows Phone 7の実機をさわってもらいながら楽しい時間を共有できればと思います。少しでも興味があれば、ぜひご参加ください。

セッション内容

  • パノラマビューアプリを作ってみよう by TWorks
    Windows Phone 7の特徴的として、独特なUIや生産性の高い開発ツールがあります。セッションではUIの中でも特徴的な「パノラマビュー」について、解説＆実装をやってみようと思います。
  • Silverlight for Windows Phone Toolkit August 2011 触ってみた by 蜜葉
    いわゆるWPToolKitがMangoに対応しました。
    今回のアップデートでMango OSに使われているいくつかのコントロールをアプリ内で容易に利用することができます。
    また、サンプルファイルとしてxap形式でも配布されています。
    　
    セッションでは、まず実機にデプロイしたサンプルxapを眺めたあと、
    またーりとそれぞれのコントロールを使い方を見ていきましょう。
  • Windows Phone 7とAndroidの比較 by 遥佐保

セッション担当

TWorks

楽しいアプリ制作の会 代表です。仕事でスマホ系アプリの設計・開発をやっています。個人的には、Windows Phone 7の開発は、他のスマートフォンのものより簡単で楽しいです。ここらへんをうまく伝えたいと思います。

みつばたん

Windows Phone Arch 代表です。SilverlightやPHP,JSから、KINECT,Wii Remote Hack、電子工作、メディアアートまで色々やっています。WindowsPhone 7では、最近まで月刊 I/Oで"WindowsPhone 7 Programming 入門"を連載していました。現在は@ITで連載をしています。

遥佐保

Silverlight Square 代表です。

セッションのレベル感

  • Windows Phone 7 アプリ開発に興味のある人
  • Windows Phone 7 の実機をさわってみたい人
  • C#をかじったことのある人
  • iOS/Androidのアプリを作ったことのある人（開発方法の比較ができると思います）
