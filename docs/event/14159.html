■開催趣旨：

　Flashの技術はスマートフォンなどの展開も含めて日々進化、楽しいものになっていっています。
　FlashBuilder(元FlexBuilder)というツールはプログラマー向けの開発ツールですが、
　デザイナーさんとの連携や開発手法など、私も知らないことがたくさんです。
　本勉強会はFlashBuilderを中心に、将来的には広くFlashについての技術を
　磨いていくための勉強会をめざして立ち上げました。
　みなさん、Flashについて一緒に学びましょう。

　●セッション１　

『Android と AIR と CMS で加速する Flex』まよシステムズ赤木さん

　＜内容＞
　２月のWTMにて a-blog cms とFlex連携など話させて頂きました。
　今回はその延長でAndroid/AIRでもやってみた事などを含めて、
　前回話せなかった内容や追加した事などを発表してみます。

　＜赤木さんの紹介＞
　まよシステムズ としてWebシステム開発を中心に活動しています。
　「こういうときはこうだよね」という型通りの開発だけでなく、
　「こういうやりかたもあるよね」という提案ができるように。
　中国地方唯一らしい a-blog cms パートナーです。
　CMSだけでなく、オリジナルのシステム開発や他システムとの連携、
　社内システムにクライアントアプリまでなんでもやってます。
　Augment8メンバーです。
