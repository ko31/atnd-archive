a-blog cms 勉強会 大阪：概要

2011年4月5日にa-blog v1.4がリリースされました。その後2ヶ月後の2011年6月にリリース２周年を迎えでv1.4.2がリリース。 TwitterやFacebookとの連動機能 も搭載されました。
a-blog cmsは、公開2年周年を過ぎて日本全国で150社ほどのWeb制作会社、450サイトで利用されています。公開されている事例は a-blog cms 事例紹介 でご覧になれます。

今回は少人数ですから参加者のわからない事や実際に自分で導入中に出てきた疑問や質問、つまづいたりはまったりした事などをできるだけ解決できるような場にしたいと思います。
もちろん使った事無い人や初心者の人、興味がある人、今後仕事で使ってみようと思っている人、MTやWPとどんなところが違うのか知りたい人なども是非ご参加ください。

今後定期的なものにしていきたいと考えています。

■内容

6月11日に開催した「 a-blog cms 1.4 紹介セミナー in OSAKA 」をしっかり復習。前回参加できなかった人でも大丈夫です。むしろしっかり復習できるので大歓迎です。
持ち寄った疑問や質問など実際に触りながら解決していきましょう。

■時間

会場は13：00〜17：30まで借りています。
準備や撤収などありますので実質勉強会の時間は
13：15〜17：20
くらいになると思います。

■PCの持ち込みについて

各自PCをご用意ください。
できれはあらかじめローカル環境にa-blog cmsをインストールして参加頂くとスムーズに開始できますのでチャレンジしてみてください。

Macの場合はMAMPを使ってインストールします。
「 MAMPを使ってa-blog cmsをローカル環境で動作させる 」
こちらの記事を参考にすればできるはず！

Windowsの場合はXAMPP for Windowsを使ってインストールします。
「 XAMPP for Windowsを使ってa-blog cmsをローカル環境で動作させる 」
こちらの記事に沿ってチャレンジしてみてください。Macよりもちょっとハードルが高いかもしれません。

もしも「やってみたけど出来ないよ〜。」という人もそれで参加するのを諦めないで会場でインストールするところから一緒にやってみましょう。一人でつまづく事もみんなでやればきっとできると思います。

ローカルマシンにではなく、実際にサーバーにインストールして使いたいという人は、a-blog cmsが動くサーバーを事前に契約済ませておいてください。当日はインターネットに繋がる環境を用意する予定ですが、何かトラブルで繋がらない事があるかもしれませんので、サーバーで動かしたい人はイーモバなど各自で繋がるように準備ください。

■参加費

参加費自体は無料です。
会場（会議室とプロジェクターなど備品）のレンタル代金を人数で割った額を申し訳ないですがお願いします。
日曜なので13時〜17時30分までで約15,000円なので参加者10人集まれば1,500円くらいになります。

■主催者

おかだよういち ( Yoichi Okada / @okada41 )

兵庫県の西の端っこでWebサイト制作や撮影の仕事をフリーランスで活動して10数年。セミナーやイベントなど関西エリアだけでなく東京や名古屋にも適度に出没します。a-blog cmsの前身a-blogの頃からのビジネスパートナー。世間のMTだWPだの話題には目もくれず、a-blog cmsの設計思想や使いやすさに惚れ込みひたすら皆におすすめするエバンジェリスト。
