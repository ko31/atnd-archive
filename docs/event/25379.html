「社会人のためのWebデザインスクール」体験レッスン　2/20（月）　14:00〜17:00 と 19:00〜22:00

3月から、飯田橋駅そばでWebデザインスクールを開きます。
このWebデザインスクールはのテーマは 「就職に役立つ実践的Webデザインスクール」 です。
無意味な検定の合格や、Photoshop や Dreamweaver などのソフトの使い方を一通り習って終わりのスクールではなく「Web業界に就職し、活躍する」ことを明確な目的としています。

もちろん、Photoshop や Dreamweaver などのソフトの使い方を全く知らない方でも理解できるよう1から丁寧に説明します。
しかし、このスクールは「ソフトの使い方を覚える」ことをゴールとはしません。

ソフトの使い方を覚えて、教科書と同じ物を制作しただけで卒業した場合、その後就職してWebデザイナーとして仕事をするまでの間には大きなハードルがあります。
「ソフトの使い方を覚える」とか「教科書と同じ物を作る」ことはスタートラインに立ったに過ぎません。

Web制作者として就職するためには、学歴も資格や検定も関係ありませんが、ポートフォリオ（自分の作品集）が必要です。
自分の作品がないと、面接担当者の方はあなたがどの程度のものを作れるのか判断できません。

ポートフォリオに載せる作品は、スクールで言われた通りに作ったものや教科書通りに作ったものではなく「自分で作った」と胸を張って言えるものが必要です。
私たちはそのお手伝いをします。

体験レッスンについて

本スクールは3月1日から5月末までの毎週月・火・木の14:00〜17:00と19:00〜22:00開講予定ですが、それに先立ち2/20（月）　14:00〜17:00 と 19:00〜22:00に体験レッスンを開きます。
体験レッスンの参加費用は ¥2000（会場にてお支払い）です。

これからWebデザイナーになりたい方、現在デザイン以外のお仕事をされていてこれからデザインもしてみたいという方など、ぜひお気軽にお越しください。

体験レッスンでは、Photoshopの基本的な使い方やボタン画像の作り方などをお教えする予定です。

必要なもの

  • ノートパソコン（Windows XP、Vista、7のいずれか、または Mac OS 10.4 以降）
  • Adobe Phtoshop CS2 以降（※注）
  • 教科書として HTML/XHTML&スタイルシートレッスンブック を使用するため、できれば持っておいたほうが良いかと思います

※Adobe Phtoshop CS2 以降をお持ちでない方は、レッスン開始までに 無償体験版 をインストールしておいて下さい。体験版の期限がすでに切れている方はお申し込み・お問い合わせの際にその旨をお書き下さい。Dreamweaver、Fireworks、Illustrator はなくても構いません。
