ニコファーレで頂上決戦。 CODE VS 5.0決勝トーナメント！

相手の隙を衝くアルゴリズム活用力と、的確なコーディング技術を競い合うプログラミング・コンテスト。

今回のゲームは「鬼ごっこゲーム」。
追いかけてくる犬から逃げつつ、集めたアイテムで相手の邪魔をするゲームです。

ゲームを攻略する「最強のAI」を作り、プログラマ日本一に輝くのは誰だ？？
