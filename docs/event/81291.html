セミナー概要

＜テーマ＞副業ライターで年収100万円向上セミナー

＜日時＞2016年10月8日17時～19時

＜会場＞大阪梅田（参加者のみメールにてご連絡）

＜人数＞5名限定

＜価格＞3000円（9月中のお申し込みで1000円割引）

＜内容＞
「月数万円で良いから副収入が欲しい・・・」と思われているサラリーマンの方や主婦、学生を 対象にWebライターとして確実に副収入を得る方法を「0」からみっちり教えます。 5名限定で「勉強会」のようなカジュアルな雰囲気で進めていきます。 先着順ですのでお早めにお申し込みください。
＜参加者特典＞(セミナー限定プレゼント）
・「ランサーズで年収100万円向上させるノウハウ」をまとめた10個のコンテンツをプレゼント!! ・セミナー内容をまとめたPDFをプレゼント!! ・納税日本一「斎藤一人」の204個（約70時間分）のマインドセット音声・マインドマップをプレゼント!! ・副業に関する無料相談（対面・スカイプ）を1回無制限で受ける権利をプレゼント!!

このセミナーでは・・・

・会社の給料だけでは少ない・・・
・副業を始めたいが何からすれば良いか分からない・・・
・ネットビジネスをしてきたがどれも続かずに終わってしまった・・・
・会社に縛られないライフスタイルを送りたい・・・

と考えている方に、「Webライター」という副業方法で0からインターネットを使って稼ぐ方法を2時間みっちり教えていきます。セミナー後は希望者のみ近くのカフェで無料相談を行います。

＜カリキュラム＞
・「Webライター」で稼ぐとは？
・副業挫折経験から学んだ初心者が継続できる方法は？
・なぜ副業することが大切か？
・「Webライター」で稼ぐメリット・デメリット
・初心者が仕事を取るために大切な3つのこと
・月3万円～5万円の戦略
・月8万円～10万円の戦略
・アンケート実施

主催者

名前：松山利基（24歳）
出身：大阪府

大卒で世間で言う「大企業」に入社。しかし、会社員としての働き方に疑問と不安
を感じ「Webライター」として活動を開始。

2ヶ月目に月収9万円を達成。その後、会社員の傍ら月収8～10万円を継続的に稼ぐ。
そして、この経験を広め自分のように悩む人を救っていこうと決断。

現在は、会社を辞め、「情報1つで人生の選択肢は増える」を理念に、副業コンサルティング、Webマーケティング勉強会、セミナー運営、ゲストハウス運営、若手起業家を集めた読書会などを実施。
