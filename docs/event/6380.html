title 【アド「ベンチャー」な男たちが、はっきりさせる！】
catch ～あいまいな疑問たちを、はっきりさせることは冒険だ～
description

人生は冒険なのか？

人生の冒険とはなんだ？

それらをはっきりさせましょう。

人生のビジョンに始まり、

人生に必要ない些細なことまで、

「はっきり」と定義づけできる大人はかっこいい。

そんな思いを基に、

ココロの中のあいまいな疑問たちと触れ合いながら、

人生における大切な分岐点を、

バッサリはっきりさせていきましょう。

人生の分岐点を「えいっ」と振り切る瞬間は、まさに冒険です。

人がどう言おうが、ココロがすっきりしたなら、それは冒険です。

そんな時間を、はっきり団と共に。

え？

「はっきり団」とは？

「はっきり団」とは、

たとえば、

○「結婚するの？しないの？はっきりして！」（人の）

とか、

○「おはようございます。」は厳密に何時から何時まで？

とか、

○『コップいっぱいの水』と言った時のいっぱいのラインって？

という、まだまだ解明されていない、

あいまいな疑問を、はっきりすっきりさせる集団です。

そしてそして、

今回のカッコいい大人サミットは、

「はっきり団」の顧問であり、

前回の「かっこいい大人とは？サミット」にて、

熱く日本の未来を提言した男、

●東忠ヘッドウォータース代表取締役　疋田正人

と、

「カッコいい大人とは？サミット」主催者、

今夏、画期的なメンタルケアソフトを、

ニンテンドーＤＳから発売する新進気鋭のカウンセラー、

そして、「はっきり団」代表の、

●詩人　兼　カウンセラー　佐藤由明

の二人が、はっきりさせたい疑問、難問に立ち向かい、

最終的には、はっきり団に判断してもらうという趣旨の２時間。

だって、

二人だけで答えを出すのなら、

二人でガード下で飲めばいいじゃん！

ということで、

下記のような疑問、難問について語り合いたいお客様と、

会場の垣根を越えて、

「この夜だけは」と思える夜を過ごします！

『例題』

借金は冒険？

借金の定義とは？

リスクは冒険？

リスクの定義とは？

等々に答えていき、

最終的には、、、

愛とは何か？

まで、はっきりさせてしまう予定です。

あなたの持参する疑問にもお答えします。

カウンセラーだって、

カウンセリングが必要なんだ！

教えてくれよ！

ということで、あのカウンセラーから、あなたへの難問も用意します。

あっ、

今回は、

ＵＳＴＲＥＡＭ配信はしません！

公にできません！

はっきり＆すっきりのために、

今回もお酒を用意します。

ドシドシ飲んで、人生を噛みくだいてください！
