#エルベで腹を満たす会

イベント ユーザー
event_id 85111
title #エルベで腹を満たす会
catch
description

西武つくば店の閉店を惜しみつつ， 随一のドイツ料理屋であるエルベで有志を募り，豚肉の塊（アイスバイン）などで腹を満たす会を開催します． アイスバインは予約が必須なのですが，成人男性であれば1人でLサイズも食べることができそうなので，事前に店に問合せ限界まで作って頂くことを考えています．

会費は1人あたり最低5,000円程度を想定しています．また，陶器ジョッキで飲むエルベのドイツビールは格別ですが，酒は飲みたい人が好きなだけ飲み，その分払うことを考えています．場合によっては2時間飲み放題(2,000円)にします．

参加人数等が読めないため，日程も含め多くがまだ未確定ですが，とりあえず仮の登録フォームとしてATNDを立ち上げました．*参加希望者は1月中（1月31日 23:59）までに登録をし，イベントページのコメント欄に次の5点を投稿してください．*登録時点で全てを埋めなくても構いません．ただし，ATNDではコメントの編集が出来ないため，最新のコメントを正しいものとして扱います．
なお，開催日から5日以内のキャンセルは参加費に応じたキャンセル料金を請求することがあります．
