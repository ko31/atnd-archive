HTML5の主要な新機能の一つ、Canvasを使って ゲームアプリ お絵かきアプリを作ります

当初、ゲームの作成を予定していましたが、『超』初心者にゲームはちょっと無理そうです。

そこで、基本的なCanvasの使い方と、簡単なお絵かきソフトに変更しました。

HTML5のCanvas機能を使うと、HTMLにドロー機能を加えることができます。

学習内容

  1. この勉強会はJavaScriptの基本的な書き方を理解している必要があります
  2. HTML5のCanvasについて勉強します。
  3. Canvasの使い方
  4. Canvasに図形を描く
  5. Canvasに線を描く
  6. ゲームを作りながら楽しくCanvasの仕組みについて勉強します！
  7. HTML5のゲームアプリの作り方（考え方）
