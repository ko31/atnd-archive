各メディアの報道などを受け、将来が不安に思っている方が多くいらしゃると思います。
今まで”常識”と思われていたことが非常識に変わりつつある・・。
皆様も肌で感じていると思います。

常識は時代により変わります。

今後待ち受ける嵐の時代をどう乗り越えるか。

明るい未来にすべく、
時代が変われど必要な『お金の在り方』を学ぶ勉強会を開催します！

勉強会では、『キャッシュフローゲーム１０１』を教材に致します！

★キャッシュフローゲーム会とは？
金持ち父さん貧乏父さんシリーズの著者ロバートキヨサキ氏が開発したお金に関する知識を高めるためのボードゲーム型教材です。
ゲームでは給料をもらいながら、株や不動産を売買し、更にキャッシュを増やし、より大きな資産を買うというサイクルを繰り返すことで不労所得を増やし、経済的自由を目指します。不労所得が支出を上回ることが最終目的です。

★お金の正しい知識を学べます！
ゲームでは、皆さん自分の好きな職業につきますが、同じ職業でも株や不動産を売るタイミングが違います。
自分の所持金や市場の状況が次々と変化するからです。
変化に対応できる力は現実の世界では必須のため、リスクなく学ぶことができます。

★財務諸表（損益計算書（P/L）、貸借対照表（B/S））の書き方が学べます！
財務諸表を読む機会は普段あまりないと思います。ゲームでは、財務諸表を書きながらゲームを進めますので勉強になります！

★同じ価値観、意識の高い仲間に出会えます！
お金の知識を学びたい！と思って行動されている方がほとんどですので、非常に刺激になります！また元経営者の方や大手企業を独立した事業家の方々など人生の成功者と言われる方も参加しているため、成功者の思考に触れることができます！

◼ゲーム会の流れ◼
