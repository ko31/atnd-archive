catch 人や薬任せにせずに自分の体は自分で治そう！
description

始めに水分補給のため甜茶を飲みます。

準備としてはヨガマットを敷き、お香（アロマ）を焚きます。

そして、正座で５分間、目を瞑り心と体を鎮めます。

自力整体は人に触られることなく、自分でする整体なので納得がいきます。

痛みを自分自身で調節することによって、体のゆがみを治します。

肩こり、腰痛、手足の疲れを取り、終わるとすっきりとした爽快感があります。

骨盤の歪みを発見しバランスの取れた体になるように自分自身で調節し

続けることで下腹部や下半身などのダイエット効果も期待できます。

１時間１０００円（ヨガマットのレンタル料、アロマ、お茶代として）

豆禅ランチ付き　２０００円（食前、食後どちらでもＯＫです）

学割　上記の値段から５００円引き

自力整体をすることによって、自分の体のバランスのズレが自分の身体感覚を通して理解できるようになり、

それをまた、自力整体によってほぐし、治していきます。

人任せではなく自分でやるので納得がいきます。

レンタルできるヨガマットは4人分あります。

慣れてきたら座布団一枚でもＯＫです。

まず、この初級クラスでは、自力整体をやっていく上での流れ、

ポーズの形、種類などを体験しながら覚えていきます。
