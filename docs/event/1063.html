「おっとりした人が成功するための7つのステップ」

description

おっとりした売り込み下手な人は
ビジネスでも人生でも成功できないのか？？

もしあなたが、

・商売下手
・売り込みが苦手
・儲からないことをしがち
・断れない
・押しが弱い
・「いい人」と言われる
・「やさしい人」と言われる
・無口なほう
・人見知りする
etc…

といったタイプの方でしたら、今回の話は必聴です。

物怖じせず、人見知りせず、ガンガン自分や会社や商品・サービス
を売り込み、成果を上げている人たちを横目に、

「いいなあ。あんなふうになりたいけど、こんな性格の自分には
無理だ」

と、ため息をついたりしていませんか？

でも大丈夫です。

自分は自分。

自分に合った方法でやっていきましょう！

その方法を指南してくれるのが、『コーヒーとサンドイッチの法則-
「利益を獲得する」ための６つの戦略』の著者、竹内正浩氏です！

★竹内正浩（たけうち まさひろ) プロフィール★
1981年福岡県生まれ。経営コンサルタント。
産学連携ＮＰＯイノプレックス顧問。
九州大学在学中より株の研究と投資を始め、20～30％でトップレベルといわれる投資収益率において、100％をはるかに超える。
その後、感銘を受けたダン・ケネディ(アメリカの著名経営コンサルタント、トップマーケッター)の日本でのライセンス権を,単身でアメリカまで直接交渉に行き獲得。
(株)レブドラを設立し、経営戦略、マーケティングなどの戦略策定のコンサルティングを手がけた後、07年に同社を売却。
08年に著書『コーヒーとサンドイッチの法則-「利益を獲得する」
ための６つの戦略』(東洋経済新報社)を出版。

おっとりした人でもブレイクできる！

世の中には、２タイプの人間がいます。

・ガンガン系　
・おっとり系

です。

ガンガン系は、相手の気持ちを意識しすぎず、自分の思うがままに
売り込みできるタイプの人間です。

反対に、おっとり系は、相手の気持ちを意識しすぎて、売り込みが
できずにいるタイプの人間です。

もし、あなたが、相手のことを「こうしてあげよう」と思ったり、
人から「あなたはやさしい人・いい人ですね」と言われたり、他人
への奉仕が自分にとっての幸せである場合は、おそらくおっとり系
だと言えるでしょう。

はっきり言って、おっとり系の人は、成功しづらいです。

なぜなら、おっとりしていると、知らない人へのテレアポや訪問販売などが苦手、つまり、売り込みが難しいため、必然的に売上が立ちづらいからです。

しかし、そんなおっとり系のあなたでも、そのままの自然体でうまくいく方法論があります。

それが「おっとりした人が成功するための７つのステップ」です。

これまでに2000冊以上のビジネス書を読破し、1000社を超える企業分析を行ない、数百人の経営者・起業家との面談・接近戦を行なってきた竹内正浩氏が発見した、「おっとりした人が成功するための7つのステップ」は、おっとり系の経営者・起業家・ビジネスパーソンにとってまさに世紀の大発見！

仕事のやり方、人との付き合い方、相性、適性などは人それぞれ。

おっとり系には、おっとり系に合ったものがあるわけです。

無理にガンガン系・イケイケ系・オラオラ系…の人たちによる武勇伝的成功法則を実践して、かえってストレスにさらされたり自信をなくしたりする必要はありません。

ドーピングして無理やりガンガン系に変貌する必要もありません。

おっとり系の自分のキャラクターに合ったブレイクの仕方を学びましょう！
