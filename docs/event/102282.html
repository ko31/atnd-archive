description

この会はソウル町オタの、町オタによる、町オタのための会です。

お申し込みの前に以下の事項をよくお読みください。

すでに何度もソウルの町歩きしている方達とひたすらマニアックにおしゃべりを楽しみたいと思っています。この会は、オタク的知識の持ち寄りパーティーです。そのため、主催者がなにかをレクチャーしたり、資料を配布したりはしません。参加なさる方は、写真、地図、経験、知識などを必ずお持ちください。

注意事項

  • この会は『町オタになりたい人』の養成講座ではありません。すでに沼に落ちてしまっている人のためのものです。オタク度が薄い方の場合、酔う可能性があります。お申し込みの際には十分にご注意ください。
  • 次の沼にお住いの方（次のジャンルのオタクの方）、歓迎します。
    1. 日帝強占期を中心とした歴史沼
    2. 現代史を中心とした歴史沼
    3. 近世を中心とした歴史沼
    4. 漢城、京城、ソウル地図沼
    5. 漢城、京城、ソウル地形沼
    6. 道路＆鉄道沼
    7. 写真沼
    8. タルトンネ沼
    9. 市場、路地沼
※ となり沼の住人も歓迎します。

＊このイベントに関してのお問い合わせは主催者にお願いします。会場にお問い合わせにならないでください。
