description

SoftBankの大野さんが幹事の、iPhoneアプリ等関係者おつかれさま会です。
100人埋まってしまいましたが、席を増やせないか調整中です。追加枠希望の方は取りあえず補欠登録をしておいていただければと思います。。

iPhoneに関わる開発者等で集まって、渋谷で１件貸し切っての忘年会を行います。
情報交換、アプリ宣伝、お仕事さがしから、純粋に食べて騒いでの人まで、色々な交流ができればと。
とりあえず大野さん代理で、Twitterアカウントiphone_dev_jp（管理人：@fladdict）がATNDを立てました。

  • 日時：2010年12月20日19:00～22:00, 19:00開場、20:00開演。
  • 場所：渋谷, スペースシャワー・ティーヴィー・ザ・ダイナー
  • 予算：5000円（飲み放題＆食べ放題）

iPhoneアプリやアクセサリ等に関わられるみなさま、お誘い合わせの上、お気軽にご参加くださいませ。
よろしければ、登録ついでに告知を拡散していただければと。

よろしくお願いいたします。
