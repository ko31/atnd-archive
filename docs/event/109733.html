日本サブスクリプションビジネス大賞2019 表彰式

サブスクサービスの日本No.1を決定

基調講演に日本マイクロソフトさんを迎え、日本のサブスクリプションサービスのNo.1を決定します。本大賞はサブスクリプションのビジネスモデルを実際に運営されているサービスを応募頂く公募型の大賞で、サブスクリプション型のサービスを振興するとともに新たなサブスクリプションサービス創出のきっかけづくりのため、創設されました。
