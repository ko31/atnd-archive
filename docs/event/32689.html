description

概要

検閲に対抗するための、合法的なハッキングについて研究する勉強会です。

大好評につき、第3回目開催です。（ 前回 ）

  • 検閲を妨害する方法
  • 検閲を回避する方法
  • 検閲の手の及ばない別なネットワークを構築する方法

などについて学びます。

  • コンピュータそんなに詳しくなくても根気でカヴァーする気があるなら大丈夫です
    • 作業量は少なくはないので、慣れるまでは根気は要ると思います
    • 学位はなくても大丈夫です
  • Unix系OS未経験の方は、まずUbuntuを入れてWebサーバを動作させてみるとか、そのへんからはじめてUnixに慣れましょう
  • 講師、受講生の別はありません。お互い知ってることを教えあい、情報交換をしましょう
  • インターネットの検閲に反対する立場の人なら誰でも参加できます
    • 例えば、ACTAもTPPも原発も反対ではないという立場の人でも問題ありません
    • 検閲反対以外の主義主張は問いません
      • ただし当日その話されても他のメンバー困ると思うんで自分のblogにでも書いてください

持ち物

  • hack用ラップトップ
    • ない人は18時から買いにいきましょう
    • 予算は3000円〜10000円程度
  • 作業用ラップトップ
    • なくてもなんとかなる
    • あったほうが便利
  • USBメモリ
    • 1GB以上のものが必要です

タイムテーブル

  • デモに参加しない方は19:00〜20:00にJAM Akihabaraまで来てもらえばOK
  • デモ不参加でラップトップ購入する必要がある人は @ubuntu_baramaki まで連絡ください（当日のデモの動きがよくわかんないので）
時間 タイトル
18:00 – 19:00 まだhack用ラップトップを用意していない人は中古屋をまわってラップトップ選び
19:00 – 22:00 JAMでhack開始
22:00 - 解散

場所について

  • このイベントのために借りきっているというわけではないです
    • 最悪の場合全員座れない可能性もあります
    • まあでも日曜夜結構空いてるし大丈夫でしょう
    • 大丈夫じゃなかったら二手にわかれるとかも考えてます
  • 電源あります
  • 無料のWiFiあります
  • ワッフルがおいしいです
  • コーヒー紅茶500円〜

予習

  • 自分で作業できる人はhack用マシンにUbuntuなどのUnixライクOSを入れてきてください
