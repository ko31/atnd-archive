■イベント概要

2010年から続いているSocialTopRunnersを開催します！過去発表企業は gumi / KLab / グリー / バンダイナムコスタジオ / Drecom / Kayac / スクウェア・エニックス / gloops / pokelab / colopl / CyberX / Sumzap / nijibox / Crooz / akatsuki / その他トップSAP企業。業界のトレンド、開発、企画、マネタイズ、そして戦略のリアルを体感するイベントです！イベント後の懇親会では参加する開発者同士の情報交換も重要ですよね！このイベントからソーシャルゲーム業界が始まったと言えなくもない、要チェックイベントですよ〜！
