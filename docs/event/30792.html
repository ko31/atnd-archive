description

老後生活費の柱である『公的年金』。
自助努力としての『生命保険』。

どちらも必要性を感じて加入しているものの、実は多くの方が内容を理解していないのではないでしょうか。

セミナーでは公的年金の基礎知識と、将来を見据えた賢い保険の入り方をわかりやすくお伝えします。

この機会に年金や保険に対するモヤモヤをすっきり解消しませんか？

講師は銀行・保険会社・保険代理店で働いてきたファイナンシャルプランナー。
長年金融業界で働く中で『金融知識がある／ないで大きな違いが出る』ことを体感してきました。

公的年金・生命保険も重要な金融知識です。
基本を学んでおくときっと今後の人生のお役に立ちますよ！

人数に限りがありますので、お早めにお申し込みください。
