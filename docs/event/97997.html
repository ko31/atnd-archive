こんにちは！主催者のKJです！

皆さん資産運用に興味はありますでしょうか？

このご時世もはや新卒で務めた会社に定年まで務める「終身雇用制度」は崩壊しつつあります。技術の進歩で今まで人間が行っていた業務もAIに奪われていくでしょう。

そんな時代に収入源が勤めている会社の給与のみって恐ろしいことだと思いませんか？まだ定年間近の50代は逃げ切れるかもしれません。

しかし、僕のような20代の未来は絶望的です。今務めている会社もいつまでいれるか分かりません。将来もらえる年金はほんのわずかかもしれないです。

会社を定年まで努めあげても、「老後はわずかな年金しか収入源がなくてセカンドライフを楽しむなんて状況じゃない。」なんて笑えないですよね。

私たちにはそんな不安な将来が待っているのに、ただ黙々と会社で働くなんてことをしている場合ではないのです。今すぐにでも他の収入源を得るために動いていかなければならないのです。

「じゃあ何をすればいいのか。」と思ったあなた。ぜひ「資産運用カフェ交流会」に参加しませんか？ きっと得るものがあるはずです。

「資産運用カフェ交流会」の参加するメリットは下記の通りです。もし下記を読んで共感してくださる方や、興味がある方はぜひ一緒に語りあいましょう。有意義な時間にしましょう！ 僕自身も勉強したいです！

※気軽に皆さまと話をしたいため参加人数は2人までとしております。興味がある方はお早めに参加申し込みお願いします。状況に応じて参加枠は増えるかもしれませんが基本は2人を想定しております。

【資産運用カフェ交流会参加のメリット】 ・資産運用・投資についてカジュアルに語りあえる。

私も資産運用セミナーなどにたまに参加するのですが、なかなか参加者同士でカジュアルに語り合う場ってあまり無いんですよね。

講師から一方通行のインプット型ですよね。ぜひ自分の考えや知識をアウトプットして、自分の知識の整理を行いましょう。

また、自分の普段のコミュニティとは別の人同士で語りあう事はとても刺激的です！

・投資を始めるきっかけができる

僕は仮想通貨、株、不動産投資を行っています。スペシャリストでもないですが、当然投資に当たってはある程度勉強をしてきましたし、少ない投資額ながらも100万程度の利益は出しております。

「投資には興味があるけどまだ手を出せていない」方や初心者の方には参考になる知識の共有はできます！

また私からのアウトプットだけでなく、参加者の意見なども吸収することで僕自身も新たな気づきを得たいと考えています。　

・参加費無料 参加費は無料です。（ご自身の飲食代は自己負担でお願いします。）

「ちょっとした隙間時間があって何をしようかなー」と考えているあなた。無料なので気軽に参加できますし、都内のアクセスが良い場所で開催しています。

資産運用・投資に興味のある方 （自分自身の勉強も兼ねているので、既に投資を実践している方も大歓迎です！）

【こんな方にお勧め】

・会社に依存しなくても生きていける「経済的自由」を求めている方

・資産運用・投資に興味がある方

・カジュアルに資産運用・投資について語り合いたい方

・勧誘が嫌いな方（完全に個人で開催するので、変な勧誘は一切無いです。）

【参加をご遠慮していただく方】
