セミナーの概要

各社クラウドサービスが多く発表される中、高速かつ安定したマシン間帯域で注目されている
Google Cloud Platform が待望の新価格を発表しました。
クラウドプラットフォームにて IaaS（Infrastructure as a Service ）の選択肢を考える上で、
Amazon Web Services との対抗が明確化になり市場では比較が求められています。

今回は、Google技術集団のトップゲートと各業界エージェンシーのパイオニアである
クリーク・アンド・リバー社が Google 共催によるイベントを開催します。

Googleエンジニアが最新の状況を披露するとともに、トップゲートの Google API エキスパートが
開発現場から有益な開発プラットフォームを説明します。
求められる技術者スキルが変化する中、是非ともこの貴重な機会をご活用ください！

なお、Google・トップゲートが全面バックアップするワールド・カフェを企画しています。
日頃会う機会が少ない優秀なエンジニアとの出会い・興奮するワークショップで
価値のある体験を保証します。
そしてイベント当日はクリーク・アンド・リバー社＆トップゲートによる
「ある」発表も予定していますのでどうかお楽しみに！

今後開催予定のないイベントで、お席に限りがありますのでお早めにお申込みください。
