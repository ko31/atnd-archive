description

セッションの目的

2020東京オリンピックでは、世界中の人々が東京そして日本各地を訪れますが、最大のクール・ジャパンコンテンツは日本人の精神・文化・行動にあると考えています。身近なコンテンツに気づき、これを広める草の根活動にチャレンジするチームを生み出すことが、本セッションの目的です。

対象者

東京オリンピックのタイミングで何かやりたいと思っている方。
日常に何らかの変化を持ちたいとお考えの方。新しい働き方へのヒントを探している方。
国籍・年齢・性別・職種問わず、いろんなバックグラウンドの方たちにご参加いただきたいです。

Program
受付
18:40 〜 18:55 受付
参加者にお集まりいただく前の時間、少々お待ちください。

このあいだに、
・参加費(500円)のお支払いただく
・名札シールに名前を書いていただく
をおねがいします。

オープニング・トーク
18:55 〜 19:05 サークル（Circle）
これまでの活動を振り返りつつ、今回のセッションの趣旨を高野からお話します。
今回は、アイデアに一歩近づくためにアクションしてみるチームを生み出すことにありますが、その意義や進め方について説明します。

19:10 〜 19:25 ストーリーテリング（Storytelling）
前回のセッションから生まれた3つのFBページを題材として、その担当者から
・作成の思い
・FBページ作成ストーリー
・作ってみてどう思ったか
を語っていただきます。

多様な知識を持ち寄る
19:25 〜 19:40 ワールドカフェ（World Cafe）
オープニングトークを聞いての感想や、自分がやってみたいことのアイデアを、周りの人3-4名で自由に話します。
メンバを変えて、2セットやります。

新しい問題の構図を描く
19:40 〜 19:50 マグネットテーブル（Magnet Table）
自分が手がけたいと思うアイデアをA4シートに書きます。
または、すでに活動テーマがある人は、「相談したい課題」をA4シートに書きます。

A4シートを見せ合って、３‐４人のチームを作ります。

アイデアを広げる
19:50 〜 20:10 ブレインストーミング（Brainstorming）
(1) 具体化したいアイデアで集まったチーム

チーム内で具体化したいアイデアを出し合います。

その中から、自分たちが取り組むアイデアを一つに決めます。
その上で、自分たちのチームが、「何を行動していて」それによって「何が生まれているか」を、次の3段階で考えてみます。

・3年後のイメージ / 1年後のイメージ / 3ヶ月後のイメージ

(2) 相談したい課題で集まったチーム

課題を深く理解できるように、提案者から想いを引き出すように質疑をしてください。

その上で、自分たちのチームの活動テーマを見直し、「何を生みだすために」「何を行動していて」「いま何に困っている」かを整理して下さい。(1)と同様に、時間軸を意識するとよいでしょう。

・3年後のイメージ / 1年後のイメージ / 3ヶ月後のイメージ

プロトタイピングする
20:10 〜 20:30 クイックプロトタイピング（quick prototyping）
チームで頭の体操をします。

「はじめて日本にやってきた◯◯人の△△さんを、日本に来てよかった！」と思わせるストーリーを作ります。

20:30 〜 20:50 クイックプロトタイピング（quick prototyping）
(1) 具体化したいアイデアで集まったチーム

ブレストで出した「3か月後のイメージ」を実現するまでのストーリーを作ります。
そのストーリーの中で、チームメンバの一人ひとりが何をするのかも決めてしまいましょう。

(2) 相談したい課題で集まったチーム

課題を解決するストーリーを作ります。
そのストーリーの中で、チームメンバの一人ひとりが何をするのかも決めてしまいましょう。

アクションを生み出す
20:50 〜 21:20 サークル（Circle）
各チームのストーリーを発表してもらいます。
また、その中で、ひとりひとりの役割と感想を話してもらいます。
