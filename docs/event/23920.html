description

2011年は、盛んにBIGデータ時代の到来と叫ばれ、
2012年は、それをどう活用するかが問われる年になりました。

特に企業のマーケティング担当者にとってBIG DATA時代に
どのような施策を打てばよいかが大きなテーマとなってきました。
本勉強会では、その中で担当者がぶつかる課題を
最新事例を取り入れながらわかりやすく解説していきます。
