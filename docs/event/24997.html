title 就活に役立つFacebook活用セミナー
catch ～自分のコミュニケーションスタイルを確立する～
description

Facebookの登場により、就活のスタイルが双方向へと変わりつつあります。
今や就活生の半数はFacebookを使用していると言われています。

一方で、

　・Facebookをどう使って良いのかわからず始められない
　・Facebookアカウントを作ったけど、どう使っていいのかわからない
　・Facebookは使ってはみたけど、どうすれば就活にアピールになるかわからない
　
という声もよく聴きます。

このセミナーでは、人事の視点をもとにFacebookと就活をつなげるノウハウを紹介します。
エントリーシート、筆記試験を飛び越え人事と直接やり取りする術を獲得する
まさに就活２ステップジャンプのためのセミナーです。

また、セミナー後には、参加した学生同士、また社会人と交流できる懇親会を用意しました。

今まで、文字数の限られたエントリーシートや数分の面接では自分をアピールできないと
苦しんでいた皆さんも、このセミナーを通じて自由な自己表現を身に付けてください。

もちろん、
これからインターンシップ、就活を控えている２年生や、
学生生活で利用したいと考えている１年生の参加も、歓迎です。

コンテンツ

  • １）そもそもFacebookとは？
    　
  • ２）書籍でも解説していないFacebookの最新機能紹介
    　　　・企業にダイレクトに自分をアピールするには？
    　　　・プロフィール登録であなたを最大限にアピールしよう！
  • ３）人事からのFacebookアドバイス
    　　　・人事にとってどんな情報が魅力的なの？
    　　　・ぶっちゃけFacebookの情報で不採用につながることってあるの？
  • ４）Facebookを利用する事で参加できる就活イベント
    　　　・人脈は就活の財産！就活仲間を作ろう！
  • ５）実際にやってみよう。
    　　　・パソコン／スマートフォンを使った演習
    　
  • ６）学生、社会人の交流会
    　　　・軽食、お酒を楽しみながら社会人の本音を聞こう！

　※内容は、変更になることがあります。
