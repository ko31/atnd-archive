description

このたびpaperboy&co.（以下ペパボ）は、東京、福岡に続き、京都に開発拠点となるスタジオを開設するため、一緒にサービスを作る開発者の方々を募集いたします。

これに先立ち、スタジオ立ち上げ担当者であるさちこが実際に京都へ移り住み、11月より準備室を開設して「おこしやす！プロジェクト」を開始しました。

　▼paperboy&co. / 京都「おこしやす！プロジェクト」どすえ
　　http://www.paperboy.co.jp/kyoto/

そこで、プロジェクト開始にあたり、わたしたちペパボがどんな会社なのか、現在どのようなサービス開発を行なっているのか、京都スタジオの採用活動について、ペパボのスタッフが直接皆さまに向けてお話するトークイベントを開催いたします。

■開催日時
　2011年12月10日（土）15:00～17:00（14:45開場）

　～懇親会について～
　イベント終了後、会場にて交流会を開催する予定です。
　お時間のご都合のつく方は、ぜひご参加ください。

■プログラム
　１．「京都スタジオ開設にあたって」
　２．「ペパボサービス開発の現状とこれから」（宮下剛輔）
　３．「ザ・インタビューズ」（吉田健吾）

　○スピーカー
　　吉田健吾　@kengochi（株式会社paperboy&co.副社長）

　　宮下剛輔　@gosukenator（同・技術責任者）

　○ＪＲ京都駅から
