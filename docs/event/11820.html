Arduinoで遊ぶ会

description

Arduinoをネタに雑談やら情報交換やら作品のアイデア出しをしつつ、工作して、最後に発表しましょう。初心者も経験者も参加OKです。Arduinoの開発環境は基本各自でご用意ください。事前のやりとりはtwitterハッシュタグ#arduinoplayでやりましょう。

＜概要＞
日時　2011年02月6日(日) 13:00-18:00（12:30開場）
会場　ガジェットカフェ (高円寺駅 徒歩5分)
http://gadgetcafe.jp/access
対象　どなたでも
持物　Arduino開発環境はご持参ください。
主催　ガジェットカフェ事務局
会費　1000円

※高円寺から秋葉原は片道30分。「抜け買い」OK！
※ガジェットカフェでは「Arduinoをはじめようキット」などの初心者向けのパーツを５点販売しています。⇒　メニューはこちら
※主な貸出可能なもの：無線LAN、電源、ノートPC（２台）、ハンダごて（数本）、ニッパー、ラジオペンチ、ドライバーセット
※ドリンクフリーですよ。

懇親会：
作った後、18:30-20:30に懇親会（2000円）を予定しています。

USTREAMインターネット生中継:
ガジェットカフェ公式チャンネルからご視聴下さい。
※来場者は顔が映ると困る方はスタッフまでお申し出ください。
