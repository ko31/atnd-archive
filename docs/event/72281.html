さて2015年も気が付けば１か月半程度となっておりました。
2015年を振り返りたいけど、忘年会シーズンを少し外してやっていきたい、そんな素直な
気持ちで少し早い「ドキッ！少し早いゲームだらけの忘年会！」を開催したいと思います。

・概要
日程：11月28日（土曜）
場所：西日暮里の喫茶店フィレンツェ（確定しました）
時間：17:00～22:00
予算：2000円（１杯＋食事）２杯目以降は各自清算になります。

　　　人数を確定したいと思います。
内容：

ゲーマーとしては今年どれだけ積みゲーを消化でき、今なにが積まれるのかも記憶にない、
これは老後のために積んでいるって事にして、一喜一憂してる間に気が付けば
来年になっていた、そんな感じで過ごされる方も多いと思います。

それはさておき、プレイをしながら、見ながら酒を飲み語らうそんな会があってもいいと思います。

そんな感じで、ゲーム機を持ち込んで忘年会でもやってみようかと考えております。
もちろんPC持ち込みで自作ゲームを遊んでもらってもOK！
去年は大貧民とかアナログゲームとかやってたような気もしますが、遊びながら2015年を振り返る、むしろ振り切る会です。 熱くなれ。
