こんにちわ！

ホームページの作り方を少し勉強したけど、イマイチ何を勉強したらいいかわからない・・・。
そんな、まだ駆け出しの初心者同士が、
モチベーションを高めるため、一緒にもくもく勉強する会を開催さるせていただきます！

また、WEBマーケティングについても皆で話し合えたらよいのではないかと思っています！

このイベントの目的

この会に参加することにより、HTML,CSSの基礎や何を勉強したらいいのかわかるようになります。
みんなで教えあって、効率よく問題を解決していければと思っています。

参考にする教科書は

「これからの標準を身につけるHTML＋CSSデザインレシピ」や「Progate」をベースにわからないところを教え合えればと思います。

こんな方が対象です。

  • プログラミングに興味があるけど、よくわからない
  • HTML,CSSを学ぼうと思っている
  • 興味があるくらい・・・

などなど、 理由はさまざまですが、 まずは。みんな集まって、一緒に勉強しましょう！ そこから、見えてくることが多いはず！
