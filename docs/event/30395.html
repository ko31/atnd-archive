FuelPHP (PHP5.3 前提で記述されたオープンソースフレームワーク) の勉強会です。

パーフェクトPHP(小川雄大，柄沢聡太郎，橋口誠著)のサンプルアプリケーションの mini blog を FuelPHP で各自が実装し、最後に発表する形式を計画しています。

元のコードは下記からダウンロードできます。
http://gihyo.jp/book/2010/978-4-7741-4437-5

github から入手したい方はこちらです。
https://github.com/kenjis/perfect-php-mini-blog

selenium テストコード(主に正常系のみ)はこちらです。
https://github.com/ounziw/selenium-for-miniblog

対象者

FuelPHP に興味がある人。
パーフェクトPHPを読んだことがあるor上記(http://gihyo.jp/book/2010/978-4-7741-4437-5
)のソースコードを読んで理解できる人
※勉強会では PHP の基礎からの説明は致しませんのでご了承ください。

用意するもの

パソコンを持参することを推奨します。(持参できない方はコメント等でお知らせください)
会場に無線LANあり。
