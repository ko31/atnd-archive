Open Build Service (OBS) 道場

イベント ユーザー
event_id 31911
title Open Build Service (OBS) 道場
catch 名古屋会場
description

openSUSE の開発で使われているパッケージ開発ソリューションの OBS の使い方について学ぶ会です。OBS は openSUSE だけではなく、Ubuntu / Debian, Fedora などのパッケージも管理でき、MeeGo などの外部のプロジェクトでも採用実績があります。

予定する内容

  • OpenBuildService ってナニ？
  • OBS の基本操作 & コマンドラインクライアント
  • RPM パッケージの作成
  • できたら OBS サーバーも立ててみよう(^^)

対象者

  • なんとなく興味のある人
  • 他のディストリビューションの開発者で OBS を知りたい人
  • パッケージを各ディストリビューション向けに配布したい人
  • openSUSE の開発に参加したい人

その他

  • Linux （ディストリビューション不問）のインストールされたノートPC をご持参ください。勿論仮想PC でも OK！
  • ノートPC には予め、OpenBuildService クライアントを こちら からインストールしておくと当日楽ちんです(^^)
  • Web UI からも OBS が利用できます。クライアントコマンドは不要という方は非力なネットブックでも OK！
  • 依存パッケージをダウンロードするのにネットワークも使用するため、モバイルルーター等もあると便利です。
    • 一応共有ルーターもこちらで準備します
    • OSC クライアントをインストールした自宅サーバに SSH で…なんてできれば、帯域負荷も軽減できますね。
  • 会場費（1200円）を割り勘とさせていただきます。ご協力くださいm(_ _)m 但し学生は無料です。

場所について

  • 笠寺のガイシホールの横を通過しながら駅の反対側まで辿り着くと、見えてきます。
  • 辿り着く自信がない！という方は、12:45 頃に笠寺駅改札前に集まってください。
    • Geeko（SUSE のマスコット）のぬいぐるみを持った案内人がそこにいる予定です(^^)


夜の部

おそらくどこかでやることになると思います。金山まで行けば何かしら店があるでしょう！
