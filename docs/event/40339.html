■イベント概要

「私立・プログラミングキャンプ」 は、IPAが主催する 「セキュリティ・キャンプ」 をリスペクトして企画された”プログラミング好き”な若手が集まる ハッカソンイベント です。

「セキュリティ＆プログラミングキャンプ」が「セキュリティ・キャンプ」に変わった 昨年から開催 しており、今年で第２回となります。
キャンプ独特の、"濃い人達が集まって変なテンションで技術を語る場"を、僕達の手でDIYしましょう！


シプキャン東京大会では、世界中でハッカーの体を蝕んできた悪しき文化「徹夜」を一掃したそうですが、関西大会では 素晴らしい文化として保護 することにしました。がんばりましょう。
万が一寝たい人は布団とかはないので、寝袋を持ち込むなどしてください。
1日目はハッカソン、2日めの午前に発表会をする予定です

諸事情により１日開催になりました。健康は大切です。

今年のシプキャンは初のツアー方式を採用しています。

東京大会：2013年8月18日 http://atnd.org/events/39809
関西大会：2013年8月24日
