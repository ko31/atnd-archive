title セラピストーク
catch 〜セラピスト同士のトークイベント〜
description

数多くの様々なリラクゼーションサロンやお店、
そこで働く様々なスタイルの現役セラピストやスタッフ、元セラピスト、
または「これからセラピストになりたい」と思っている人さらには国家資格保有者の枠を越えて「人を癒す仕事に従事している者」という共通点にフォーカスをあてたトークイベント、セラピストーク。

同業他社で働くセラピスト同士の交流や情報の交換はもちろん、異業種で活躍する仲間との出会いが開く新しい可能性との出会い、
普段の仕事のなかでつい見失いがちな「初めたての頃の気持ち」の再認識など、参加することで得ることはたくさんあるはず。
さらに今回のスピーカーは経験豊富で個性豊かな一線のセラピストたち。

日常を通じて出会ういろいろな話は、きっと新しい発見や刺激を与えてくれること間違いなし！
ぜひ、この機会にセラピストークに参加して色々な仲間と出会ってみませんか？
