description

これまでは、
スマートフォン向けリアルタイム通信エンジンといえば、
サーバにコードを書かなくても簡単にマルチプレイが実装できるPhoton、
サーバにコードを書く前提のモノビットエンジン、というポジションでした。

しかし、2016年4月に、モノビットエンジンにも、サーバにコードを書かなくても
簡単にマルチプレイが実装できる新エンジン「Monobit Unity Networking」が登場し、
以前までの各エンジンのポジショニングが大きく変化しました。

要するに、「サーバにコードを書かなくても簡単にマルチプレイを実装したい」
場合の選択肢が、今まではPhotonしかありませんでしたが、
これからは、モノビットエンジンも選択肢に入れることが出来るようになりました。

今回の勉強会では、
「サーバにコードを書かなくても簡単にマルチプレイを実装したい」
という要件が求められているケースで、皆様が適切な通信エンジンを選定するために
必要となる情報を、まとめてご提供できればと思います。

■勉強会アジェンダ
１，機能比較
２，性能比較

４，料金の違い
５，クラウドサービスの違い
