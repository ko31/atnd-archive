第１回イーブイカップ

のえる主催の第１回イーブイカップを開催します…！

ロータスに集まって一番イーブイを愛している人を決める闘いです！

豪華な優勝賞品＋参加者全員に参加賞も用意していますので、ぜひぜひ参加して下さいね！

◆会場の様子
GAME BAR LOTUS

イベント概要

本イベントは「第１回イーブイカップ」として開催致しますが、ポケモンカードの交流会としての側面も大事に開催したいと思っています。

トーナメント参加は先着１６名、以降の枠は交流会参加枠とさせて頂きますので、イーブイカップ参戦希望の方は早めのエントリーをお願いします。

イベント参加の確認事項

●試合形式
・スタンダード60枚
・サイド6枚
・試合時間：30分
・スイスドロー三回戦
・ゲームに勝利した場合、勝利点３点＋デッキの中の【イーブイ】【イーブイＧＸ】【イーブイ＆カビゴンＧＸ TAG TEAM】の数が得点となります。（イーブイがデッキに入っているほど、勝利時の得点が多くなります。）
・上記以外は公式ルールに基づく

●持ち物
・大会に参加するためのデッキ
・ダメカン/コイン/GX等各種マーカー（未所持の方には貸し出しもあります。）
・大会参加費

優勝賞品

優勝者には、優勝トロフィーとしてでっかいイーブイ＋イーブイコインをプレゼントします！

＋参加者全員にサプライズがあります。

最後にじゃんけん大会でブイズグッズのプレゼントも予定しています..！
