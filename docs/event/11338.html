TitaniumMobileについて

  • 発音 カタカナ表記は「タイタニウムモバイル」
  • 基本的にJavaScriptのみで記述し、Objective-CやJavaで書くことなく
    iOS と Android 向けのネイティブアプリケーションを開発する事ができます
  • 参考リンク: Appcelerator Titanium MobileでiPhoneアプリ開発を始めよう!!

この勉強会について

  • TitaniumMobileに興味のある方なら初心者から経験者までどなたも参加頂けるような会にしたいと思っています
  • プログラマでない方には少し難しい内容もあるかもしれませんが、iOS向けアプリやAndroidアプリが手軽に作成できる事は解って頂けると思います
  • Twitterのハッシュタグは #TiMobHama にしました（変？、異論は認めますのでもっと良いのあったら提案して下さい）

      既に参加登録済みの方は当時受付時に「学生」と伝えて下さい。

PCの持ち込みについて （MacをPCって言うと怒られますか？）

  • 講習会を聞きながら実際にTitaniumMobileを触ってみたい人は事前にインストールだけ済ませておく事をオススメします
    当日ネット環境があまり良くない可能性もありダウンロードに時間がかかるのはもったいないので
  • 動作に必要なのは
    • iPhone向け開発をしたい人 : XCodeとTitaniumDeveloper
      • iPhone向けの開発を行ないたい場合は当然Mac(OSX)が必要になります
      • 特にXCodeは3.5GBくらいあるので当日ダウンロードは難しいと思います
    • Android向けの開発をしたい人 : AndroidのSDKとTitaniumDeveloper
  • 開発環境構築に参考になりそうなサイトをいくつか載せておきます
    • Appcelerator Titanium MobileでiPhoneアプリ開発を始めよう!!
    • Titanium と Aptana でモバイル開発
  • インストールについて解らない人がいたら事前に質問投げてくれてもかまいません
