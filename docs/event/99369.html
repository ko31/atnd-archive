突然ですが！
応募者に「入社したい！」と思ってもらえるような採用活動が出来ていますか？
採用活動って、実は洗脳そのものなんです。

「せっかく応募が来たのに、採用に結びつかない」「面接が苦手」「人が採用できない」とお悩み中の皆様！
入社意欲をアップさせる話術を伝授しますので、是非マスターして頂き、応募者の心を鷲掴みにしちゃいましょう！

《セミナー詳細》

◇第1部:採用と洗脳のノウハウセミナー◇
面接における成功ポイントを中心に、内定まで繋げる必勝法をお伝えします。
心理学をつかって、応募者を洗脳?!しちゃいましょう!

講師:(株)賢嶺　代表取締役　山岡 哲也 氏

◇第2部:人材定着セミナー◇
仕事を探している人に対して、どうアプローチすれば応募に繋がるのでしょうか。
採用戦略研究所のこれまでの実例を交えながら、応募者を集める手法をご紹介します！

講師:(株)採用戦略研究所　山本 光太郎 氏

◇終了後:個別相談会◇
終了後には、希望者限定で個別相談会も実施いたします!
疑問点やご質問など何でもお気軽にご相談下さい!
