＜講演内容＞

日本国内においてソーシャル・カジュアルアプリの市場も飽和し、中小ならびに個人のアプリが口コミでランクインする事は困難な市場になりました。

国内市場の競争が激しくなった今こそ、海外進出のチャンスとも言えます。

そこでアドフリくんは、国産アプリを海外市場に進出させるべく、アプリの海外展開セミナーを開催いたします。

また、最新アプリマネタイズ戦略はもちろん、進化したアドフリくんで、新たに提供できるようになった動画広告の導入技術や広告運用についても参加者限定で公開いたします。

奮ってご参加ください！

＜講演者＞

登壇１：株式会社Zucks　海外アプリ広告担当　村上 悠 氏
『海外市場の最前線！国産カジュアルゲームの海外展開について』

「ザグマガ！」で海外アプリ情報をお伝えする連載「海外Appジャーナル」でお馴染みの村上氏による登壇です。
海外で実際に展開しているディベロッパー紹介からその実績まで公開！海外展開時にするべき対策などを皆さまに伝授いたします。

登壇２：株式会社ジェーオーイー　代表取締役　石川 丈 氏
『最新作「ようとん場MIX」も大好調！海外でもヒットを続けられるその訳は・・・』

600万ダウンロードを突破した豚育成ゲーム「ようとん場」の最新作「ようとん場MIX」も大好調！
台湾、香港でもヒットを続けられる、そのノウハウについてお話し頂きます。

登壇３：株式会社ADFULLY　メディア窓口担当　池田 大地
『アドフリくんが教えるアプリマネタイズ戦略(動画広告)について』

株式会社ADFULLYはアプリに特化したSSP「アドフリくん」を提供してます。
SSPだから分かる、動画広告の収益性や効果的な広告フォーマットを公開したいと思います。
ご参加される皆さまに、更なる収益UPを伝授いたします！

＜主催：株式会社ADFULLY概要＞

当社は、SSP「アドフリくん」というアプリの広告収益最大化をサポートするサービスを提供しております。
私達の一番の特徴は、元々メディアやコンテンツ事業を運営していたメンバーが
独自の「メディア目線」で作り上げた、他の広告事業社とはちょっと違うSSPサービスを提供している事。
「メディア１００％の提案」をモットーにサービス提供している当社は、
既に多くのアプリ開発者や企業にご賛同頂いております。
