description

複数の人間の利害によって選択する行動が決まるような複雑な状況のことをゲーム的状況といいます。このようなゲーム的状況を分析・研究する数学的な理論がゲーム理論です。このようなゲーム的状況は、対人関係・ビジネス・恋愛・交渉事など日常のいたるところに存在しています。ですからその重要性は計り知れません。しかしながら、残念なことにこのゲーム理論は難しいことでも評判。多くの人がその重要性を認識しながら、学ぶ事をあきらめてしまっている現状があります。そこで弊社すうがくぶんかは、平易にゲーム理論の成果を学んでいただけるように、独自のカリキュラムを開発しました。場所もアクセスしやすい新宿での開講。この期にゲーム理論をイチから学んでみませんか？

学ぶことで得られるもの
１）複雑な利害が絡む場合の判断・行動に対する指針を持つ事ができる。
２）ゲーム理論を根拠とした経済・経営の分析を理解できるようになる。
３）日常的なできごと（対人関係・恋愛など）もゲーム理論で分析できるようになる。
