発表プログラム

【プログラム1】 PHP on Windows – 窓の上の PHP -

発表概要
PHP と言えばやっぱり Linux & Apache が定番！ですよね。
では、Windows ではどうでしょうか？
Windows 標準の IIS (Internet Information Services) でも動く？やっぱり Apache を入れたほうが良い？そもそも Windows でまともに PHP が動くの？？パフォーマンスは？？？
…などといった疑問にお答えできればと思います。
またサーバサイドだけではなく、Windows での開発環境についての最新情報 (？) もお伝えします！
発表者
きよくら ならみ (Okayama IT Engineers Community)
プロフィール
　メーカー系 SIer を経て現在は地元企業の社内 SE・プログラマとして勤務。
　Web アプリケーションプログラミングとの関わりは前世紀から C や Perl による CGI プログラミングまでさかのぼり、現在で IIS + ASP.NET でのアプリケーション開発を主な業務としています。
　Microsoft の .NET Framework 関連のテクノロジを利用する事が多く、ASP.NET をはじめ Silvelright や Widnows Phone 7 について、岡山を始め西日本のコミュニティで講師としての登壇も多数こなしています。

【プログラム2】 PHPer のための php 最適化

発表概要
今回は
　○　php をいつも書いてるけど include と require の違いがわからない＞＜
　○　echo と print ってどっちがいいの？
などなど知ってそうでなかなか調べないことを説明していきます。
発表者
曽根 壮大 (株式会社スクイズ研究所)
プロフィール
オープンラボ備後，日本 PostgreSQL ユーザ会所属。
日々、WEB 系エンジニアとしてインフラからプログラミングまで設計・実装を幅広く担当してます。

【プログラム3】 セキュリティの基礎知識 Vol.2

発表概要
前回はアプリケーション全般に適用できるセキュリティ対策の基本中の基本を紹介しました。今回は意外に知られていない IT セキュリティの概念について解説します。キーワードは ISO13335、ISO27000 です。
発表者
大垣 靖男 (PHP 技術者認定機構 顧問)
プロフィール
日本 PostgreSQL ユーザ会理事、PHP 技術者認定機構顧問を務め、PHP 開発者ならほとんどの人が手にしている「PHP ポケットリファレンス」や、「Web アプリセキュリティ対策入門」など多数執筆。
