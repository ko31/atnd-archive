勉強会カンファレンス2009

イベント ユーザー
event_id 718
title 勉強会カンファレンス2009
catch 勉強会について勉強会主催者が議論するカンファレンスです。
description

空前の勉強会ブームの中発足した、勉強会を主催する人たちのコミュニティ「勉強会勉強会」のカンファレンスを開催します。
午前中は、カンファレンスに参加される皆さんの知識を合わせることを目的に目覚まし勉強会を開催いたします。(参加自由)
本番の午後からは、社内勉強会についてや社会的意義などの話しをし、ビアバッシュにて懇親会について熱い議論をいたします。
勉強会を主催したことのある方から、これから主催して見たいと思っている方、迷っている方など、勉強会を開くことにご興味のある方のご参加をお待ちしております。

「新型インフルエンザについて」と「勉強会勉強会のカンファレンス 二次会」に関してはページ下部をご覧ください。
