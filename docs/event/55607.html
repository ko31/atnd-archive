description

Frohでは、気楽な雰囲気で学びつつ、新しい出会いを提供する場所を目指しており、毎月第４金曜日に「起業家、起業を目指す人、起業に興味のある人」のためのイベントを定期開催しています。

今回は、「ｅコマース大乱戦」を中心に、起業に役立つコンテンツを提供していきます。

もちろん、起業家でなくても、意識の高い学生・社会人の方も参加できます。途中からでも参加できるコンテンツになっているので、皆様の参加をお待ちしています。

【開催日程】
８月２９日（金）１９：３０～２１：３０
※原則毎月第４金曜日に開催予定。

【場所】
Froh Coworkingに併設の四ツ谷シェアカフェ
新宿区本塩町７　新井ビル１階

【参加費】
社会人2000円、学生1000円

【こんな人におすすめ】
・近い将来に新規ビジネスを起こして本気で勝負したい方
・サラリーマンを辞めて本気で独立起業したい方
・実際に事業を始めてはいるが、事業を本気で浮上させたい方
・就職を考えず、本気で起業で生きていきたいと考えている学生
・コワーキングに興味がある人、新たな繋がりが欲しい人

◇コンテンツ
【①Newビジネス研究】
内容：研究会メンバーで選択したカテゴリの新ビジネスの最新事情を追っていき、共有していきます。

今回は、「ｅコマース大乱戦」を読み解きます。 参加者の希望により、カテゴリは随時増やしていきます。

【②ビジネスアイデア創出】
内容：毎回、様々なお題を出題し、ビジネスアイデア創出の鍛錬をしていきます。

今回のお題は、当日発表します。 過去のお題の例：「スマートフォンのイアホンジャックを使った新しいビジネスを考えてください。」

【③ビジネスプランブラッシュアップ】
内容：参加者、ゲスト、お題のビジネスプランについて、メンバーを交えて良いものにしていきます。

今回は、講師が実際に支援しているベンチャーのビジネスをブラッシュアップする予定です。

【講師の紹介】
世良信一郎 Shinichiro Sera
株式会社イーフープ代表取締役

　大手特許事務所・知財コンサルティング事業部での経験を経て、2009年に独立。株式会社イーフープ代表取締役に就任し、起業家育成事業に従事する。同社にて四ツ谷にシェアオフィスとシェアカフェを運営し、東京の起業者をサポートするとともに、神奈川県藤沢市に在する慶應藤沢イノベーションビレッジのインキュベーション業務の委託を受け、約２０社のベンチャー支援に携わっている。 　また、平成２５年からは、徳島県のクリエイティブネットワークコーディネーターとして、県内のクリエイティブ産業の集積、新事業創出の仕掛け作りをサポートしている。さらに、平成２５年からSeed Technology Capital Partners合同会社においても社員として、経産省の新事業創出支援事業、神奈川県のかながわ成長産業イノベーション事業において新事業創出の支援を行っている。1974年東京生まれ。上智大学卒業。

四ツ谷駅からすぐなので、ふらっとあそびにきてくださいね。
お友だちと一緒でも、お一人でも、大歓迎です！
