description

【習得できる知識】
1．高精度アナログ信号処理技術の概要と基礎回路設計技術。

2．高精度アナログ信号処理回路に用いられるリニアーICデバイス（オペアンプ、計測アンプ、A-Dコンバーター、D-Aコンバーター等）に関して、
　　2-1．製品概要、製品動向、技術資料活用方法
　　2-2．機能別製品の特長、特性・仕様の定義、応用・実装技術
【講座のポイント】
　本セミナーは、工業計測機器、化学分析機器、医療機器、電力制御機器等に代表される産業用アプリケーションでのアナログ信号処理技術の基本と、実アプリケーション回路に用いられる高精度リニアーICについて詳しく解説します。
　高精度リニアーICは高精度オペアンプ、計測アンプ、絶縁アンプ、A-Dコンバーター、D-Aコンバーター等多くの種類がありますが、本セミナーではこれらのデバイスの特長、特性・仕様、応用技術、実装技術等について解説します。

【プログラム】
１．高精度アナログ信号処理回路の概要
　　1-1　信号処理回路概要
　　1-2　精度と誤差要因
２．高精度リニアーICの詳細
　　2-1　高精度リニアーICの動向と現状
　　2-2　共通基本特性、特性・仕様
　　2-3　高精度オペアンプとその応用回路
　　2-4　計測アンプとその応用回路
　　2-5　プログラマブル・ゲインアンプとその応用回路
　　2-6　絶縁アンプとその応用回路
　　2-7　特定用途用アンプとその応用回路（電流センスアンプ、ログアンプ、熱電対アンプ）
　　2-8　サンプル/ホールド・アンプとその応用回路
　　2-9　A-Dコンバーターとその応用回路
　　2-10　D-Aコンバーターとその応用回路
３．実装、応用技術
　　3-1　実装上の注意項目
　　3-2　応用技術としての技術資料の活用方法
