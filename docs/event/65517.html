Hello,Future!

イベント ユーザー
event_id 65517
title Hello,Future!
catch
description

あなたの人生の軸となるものはなんですか？

この質問には、正解はないと思います。

5/23（土）、福岡市のスタートアップカフェにて、「Hello, Future!」を開催致します。
「Hello, Future!」とは、「生き方に正解なんてない」という価値観をシェアするという趣旨のもと、
様々な生き方について知り、自分の生き方について考え直す機会を与えるイベントです。

Speaker

≪開会の挨拶≫
- 高島宗一郎（Soichiro Takashima）
1997年KBC九州朝日放送に入社。福岡の朝ワイド番組のキャスターなどを務める。
2010年12月に福岡市長就任。経済政策、特に創業支援に注力し，2014年3月には国家戦略特区を獲得。規制改革等による新しい価値を生み出す環境づくりに精力的に取り組む。

　・東京オリンピック･パラリンピック競技大会組織委員会顧問
　・スタートアップ都市推進協議会会長
　・世界経済フォーラムGLOBAL SHAPERS 福岡HUB代表

≪講演≫
- 江崎ひとみ（Hitomi Esaki）
佐賀県の日本酒酒蔵の蔵人。2015 Miss SAKE SAGA。1991年佐賀県武雄市生まれ。ほぼ福岡県育ち。佐賀大学進学と共に佐賀へ戻り、在学中に佐賀県のPR活動をする佐賀県エヴァンジェリストとして活動。大学卒業後、新卒で酒蔵へ就職し、酒造りをしている。

- 畠山千春（Chiharu Hatakeyama）
1986年生まれ。3.11をきっかけに「自分の暮らしを作る」をテーマとした活動を開始。2011年から動物の解体を学び、2013年狩猟免許取得、解体ワークショップや狩り／皮なめしなどを行う。食べもの・仕事・エネルギーの自給を目指す「いとしまシェアハウス」運営。2014年『わたし、解体はじめました―狩猟女子の暮らしづくり』（木楽舎）を出版。
ブログ：ちはるの森（http://chiharuh.jp）

- 木村一郎（Ichiro Kimura）
1987年生まれ。大学院修士課程を卒業後、都内のIT企業にエンジニアとして就職。退職後、福岡市内に住みながら都内のITスタートアップの経営に参画。2015年にanect株式会社を創業し、アジアをターゲットとしたインターネットメディアを作るために日々奮闘している。

Global Shapers Community Fukuoka

グローバルシェイパーズコミュニティーとは
世界経済フォーラムによって任命される33歳以下の若者によるコミュニティ。

優れた潜在能力、また実績を持ち、社会に貢献する強い意思を持つ33歳以下のメンバーによって構成され、メンバーはシェイパーと呼ばれます。

現在、世界に350以上ものハブがあり、4,400人以上がシェイパーとして活動しており
地域における社会課題を解決するプロジェクトなどを行うことを目的としています。
