事の発端は、木津潤平建築設計事務所　木津潤平氏と株式会社ＭＪファクトリー　馬嶋とで軽くお茶を飲みながら話をしているときに

木津氏から「朝活」って知ってる？やってみない？というところからはじまりました。

藤沢市鵠沼海岸という土地柄、地元に根ざして仕事をしている人も多いですが

東京へ通勤している人もやはり多いです。

また、いろいろな業種の人たちが暮らしていると思います。

朝活について話していたところ

普段すれ違っているであろう人たちを朝活をハブとすることで

何か面白いことが起きるといいね！

というところで盛り上がりました。

現在のところでは、隔週の火曜日AM8:00〜9:00にて行っています。

場所は「木津潤平建築設計事務所」を朝活の場として開放をして頂いています。

まるまる出席をしてもよいですし、

通勤途中にちょっと顔を出し見るという程度でも構いません。

ふるってご参加をお待ちしています。

ここでの朝活としてのスタイルは、

明確にテーマなどを決めずにその場で井戸端会議的に話しをするスタイルで行っています。

  • 話の中でアイデア・ヒントを見つける
  • 耳寄りの情報をシェアする
  • 最近読んだおすすめの本を紹介する
  • 企画について相談する

などなど、特に決まった内容があるわけではありません。

緩めの空気感で長く続けていければいい、そんなノリで行っています。
