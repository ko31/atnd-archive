description

駅前大通二丁目の名豊ビルと開発ビルを再開発し、商業ビルの中に、豊橋市図書館の分館として「まちなか図書館」(仮称)を整備する計画が着々と進行しています。

私たち「豊橋まちなか図書館倶楽部」は、豊橋を元気にするために、若者がいっそう集まる街にするために、新しいまちなか図書館に大きな期待をよせています。市民が新図書館づくりで果たすべき役割は何か? 岡本真さんが掲げる「支え合う図書館」をキーワードに、みんなで一緒に考えてみませんか?

講師プロフィール

岡本真（おかもと・まこと）
1973年生まれ。1997年国際基督教大学（ICU）卒業。編集者等を経て、1999年、ヤフー株式会社に入社。Yahoo!知恵袋を企画・設計を担当。

2009年、アカデミック・リソース・ガイド株式会社を設立。「学問を生かす社会へ」をビジョンに掲げ、新図書館等の文化機関の整備に関わりつつ、ウェブ業界を中心とした産官学連携に従事。

著書に『未来の図書館、はじめませんか？』（青弓社、2014年）、『ウェブでの＜伝わる＞文章の書き方』（講談社現代新書、2012年）、共編著に『ブックビジネス2.0』（実業之日本社、2010年）ほか。
