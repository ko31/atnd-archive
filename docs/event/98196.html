manifes2018
マニアックな技術の祭典

■manifes 2018とは

全国から技術に興味のあるエンジニア学生が集う祭典 「manifes」 昨年の大好評を受けて、今年も開催！ 全国の友達を増やしたい、技術の話や相談ができる人がほしい。 そんな学生さんが多数集まります！ 持ち物は技術への興味、それだけ！ 学年・地域問わず誰でも参加OK！ 当日は社会人ITエンジニアの方にもご参加いただき、 年齢や地域、立場を超えて技術への好奇心だけで人と人とが繋がる場を作ります。 また、今回のイベントでは当日集まって終わりではなく、 エントリーいただいた方だけのグループ作成や、定期的な情報発信も行う予定です。 学生同士の気軽な情報交換や自分の趣味を語れる場が欲しいという方も、 この機会をお見逃しなく！

参加はこちらから→ https://careerselect.jp/event_offers/event_detail/?id=420

■参加方法

参加方法として、「登壇」「観覧」の2つがあります。 【登壇】 自分の好きなテーマで5分間のLTをしていただきます。 ・マニアックな言語 ・マニアックな制作物 ・技術のマニアックな使い方 ・その他 上記いずれかの分野でご応募いただきます。 ご応募いただいた方の中から、選考を勝ち抜いた方に当日LTをしていただきます。 また、当日はご登壇いただいた方の中から、参加者による投票で「最優秀賞」が選ばれます。 投票で最優秀賞に選ばれた方には豪華景品が…！？ 昨年のプレゼン内容例： 地獄から来た言語「Malbolge」について Unityでプレゼン資料を作る Go言語を使っていろいろなアセンブリでMutexを実装してみる 【観覧】 当日、LTを聞いていただく学生さんを募集します。 学年・経験など一切問いませんので、ぜひお気軽にご応募ください。

■その他

エントリーの際、登壇と観覧どちらを希望するかをアンケートにてご回答いただきます。 【登壇を希望される方へ】 キャリアセレクト運営局により選考が行われますのでご了承ください。 選考に関しては、エントリーいただいた後にメールで別途ご案内いたします。 こんな方におすすめ ・技術が大好きな方 ・ものづくりが大好きな方 ・学生エンジニア同士のつながりが欲しい方 ・学生エンジニアのコミュニティを求めている方 参加メリット ・エンジニア学生同士のつながりができる ・学生だけではなく社会人エンジニアの方ともつながりができる ・技術の話を思う存分できる ・技術の話を思う存分聞ける
