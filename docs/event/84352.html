就職活動前の大学生を対象とした就活勉強会を開催します。
来年から就職活動を始める学年はもちろんのこと、大学1年生、2年生も歓迎いたします。

勉強会は各4回ですが全て無料です。また、各回単発での参加も可能ですので、お気軽にご参加ください。
第1回は12月23日（金）19時から東京大学駒場キャンパス５号館５１８教室で行います。
大手企業からのベンチャー転職者、官僚から財団法人への転職者、ベンチャーからの起業家など、様々なキャリアパスを持つ講師がサポートいたします。

以下の事項に該当する方は是非ご参加ください。
・就活を始めたいが、何からやっていいのか分からない
・ベンチャーや大企業などの違いが知りたい
・将来のキャリアパスが想像できない
・インターン企業をどのように選べば良いか分からない
