description

映画『うまれる』上映会を通じて、
今、妊活を頑張っている人たち、
過去に頑張っていた人たち、
これから結婚や妊活を頑張ろうとしている人たち、
子どもを望む全ての人たちにエールを送りたいと
妊活支援のイベントを企画しました。
　　　　
上映会の後には、『うまれる』に出演している東陽子さんと
不妊体験者の支援団体NPO法人Fineの松本亜樹子代表を
ゲストに迎えてトークセッションを行います。
　　
不妊で悩んでいる人たちにとって、
『うまれる』上映会は参加しにくいという声も
あるかと思いますが、主催者の行木も、
ゲストの東陽子さんも、Fineの松本代表も
不妊治療経験者です。
ぜひ、この機会に一緒に『うまれる』を通じて
いろいろと感じたり、考えたり、
前に進むキッカケにして頂けたらと思っています。
　　　
　
「妊活支援上映会」ではありますが、どなたでもご参加頂けます。
上映会、トークセッションともに
「命を繋ぐこと」「生きるということ」について、
皆さんと一緒に考える機会になればと思いますので
ぜひご参加ください。
