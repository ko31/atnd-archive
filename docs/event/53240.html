description

鳥取のデザイン＆アートを知っていただくイベント「アートットリ　TOTTORI Arts & Design Works」を東京・代官山で開催します。

7月19日（土）～21日（月・祝）の期間中、鳥取の手仕事やデザイン、ユニークなアート活動などの展示やトークイベントがあります。

今回、12万人以上のフォロワーのフェイスブックページ「いいね！鳥取（Tottori fanclub）https://www.facebook.com/tottorifan」の写真展も同時開催します。

日時：7月19日（土）～21日（祝）11時～17時
会場：代官山ホワイトルーム（渋谷区恵比寿西１丁目３１ー１２）
