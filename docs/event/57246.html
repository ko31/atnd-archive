概要

大好評だった 第1回 Japan Xamarin User Group Conference を終えて、関西でもやって欲しいとの声がありましたので第2回は関東、関西の両方で開催することにいたしました！

関西は こちら です。

東京はだいぶ盛りだくさんになりました。

Evolve 2014 の体験レポート、Xamarin 本 の著者 増田さんの Xamarin.Forms の話、Prism と Xamarin の話、iOS アプリ開発秘話など。

LT でも Unity、UnifiedAPI や Xamarin.Mac、先日の Evolve で提携が発表された Worklight になどなどついてお話いただけることになりました。

今最も HOT なモバイルクロスプラットフォーム開発ツール Xamarin の最新情報を GET しましょう！
