概要

  • チームkatagaitaiの実施する，CTFに関する勉強会です．過去問を解きつつ技術解説をします．
  • 学生〜社会人若手の方を対象にわいわいやりたいと思ってます．
  • Web, Binary, その他[未定]に興味のある， 初心者抜けたくらいの人 や 強くなりたい人 を対象にしています．
    • 初心者向けではありません が，聞くだけ聞きたい〜という方であれば初心者でも参加はWelcomeです．
    • 初級者の定義が曖昧ですが， CTF for ビギナーズじゃ満足できない と思ったらこちらに．
    • 当日の雰囲気については 去年のまとめ が参考になると思います．
  • 未定箇所については決まり次第ATNDを更新しますので，こまめにチェックしてください！！！

準備しておいて欲しいもの

  • Web
    • Chrome系のブラウザ(Chrome, Chrome Canary, Chromium)
    • git
    • XSSでデータを受ける環境(もしあれば)
    • wgetとかgrepとかが動く環境(Reversing用の環境でwgetが動けばOK)
    • BurpSuite(Free版でいいです)
  • Reversing
    • Linux 環境（Ubuntu 14.04 x64を推奨）
    • 環境構築
      • # apt-get install docker.io lib32stdc++6 lib32z1 g++ make binutils
      • # docker pull klee/klee:1.0.0
      • # docker run -ti —name=klee_container —ulimit=‘stack=-1:-1’ klee/klee:1.0.0
        • klee@9263227d82ff:~$ みたいなプロンプトになれば成功。exitすると戻れます。
      • # git clone https://github.com/Z3Prover/z3.git
      • # cd z3
      • # python scripts/mk_make.py —python
      • # cd build
      • # make
      • # make install
    • x86 ELFが解析可能なディスアセンブラ
      • IDA Freeware 想定で解説します
