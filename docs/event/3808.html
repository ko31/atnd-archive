description

さくら咲くこの季節、クラスの親睦会も兼ねてお花見をしちゃいましょう！


・場所取りはAM7時から幹事が先に場所取りする予定です。

・食べ物、飲み物は準備する事にしました。
その為、参加費制にする事にしました。

・飲み物（一人：缶２本程度）＆お菓子で
参加費お一人様　１,０００円徴収致します。（仮）

また、途中で足りなくなったり追加してもらいたいもの
があれば、買い出しに行きます。

・飲み物は重いので「お酒のカクヤス」で注文して
現地配達を頼みます。

・他に遊ぶものがあれば持参してもらえるとさらに楽しめるので、
いろいろと持ち寄ってもらえたら助かります。

・ニンテンドーDSとか持ってる人いたら、通信対戦とかできるのでぜひ持って来ましょう！
（最大８人プレイは熱い！）


・解散時間は14時となってますが、あくまで目安なので
解散というよりは、14時頃以降は各自自由行動で…という感じです。
（結構緩めの雰囲気だと思うので、全体的に自由っぽいとは思いますが…w）

・当日、よみうりホール（有楽町）でDIGITAL FRONTIER GRAND PRIX 2010の発表があり、
そちらに参加したいけど、お花見も…という方は、13時頃に先に退散でも構いません。

代々木公園から近い駅で、「明治神宮駅」から「日比谷駅」までが15分程度で到着出来ます。
（日比谷駅からよみうりホールまで徒歩3〜5分程度）
それか、「原宿駅」から「有楽町駅」までの移動でも30分弱くらいで着きます。


・場所取りや買い出しに自分も協力したい！という
気のきく申し出をしたい方、随時募集してます。
秋間にご一報下さい！（mail: masayo.akimaアットジーメールドットコム まで）

・他に、何かやりたい事とか楽しめる案があればなんでも相談してくださいー！
せっかくの縁で一緒のクラスとなった繋がりを大事にしていきましょう。
教室ではない分いっぱいはっちゃけちゃいまSHOW！

・一人一人が主役です。（大事なポイントです。）
参加される方はお互い協力し合って楽しい会にしましょう！！
