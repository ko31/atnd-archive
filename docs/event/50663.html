description

“新茶の季節です。さて、さわやかな初ものを味わいながら、
編み物しませんか？
かぎ針編みの基本、円を編む方法で自分のオリジナルのコースターを編みます。
出来上がったら、そこに湯のみを置いて一息ついちゃいましょ。”

毎月その季節にあったお茶を自ら全国から厳選し、丁寧に急須でお淹れしている、
日本茶アーティスト 茂木雅世が店主をつとめる小さな小さなお茶屋。
今回はぬいぐるみ作家 内山彩乃さんのあみもの教室も同時開催。
お昼休みにお弁当持参でお越しいただくのはもちろん、打ち合わせやちょっとほっとしたい時、気軽にいつでもご利用下さい。

マイボトルやコップをお持ちいただければ、その場で急須で淹れたお茶を出来立てのままテイクアウトすることも出来ます。

ペットボトルのお茶では味わえない絶妙なうまみやあまみ、香りにゆっくりじっくり癒されにきてください。

今までにはなかった新しい煎茶の空間と時間を是非味わいに来てください。

料金：お茶会300円 1杯／あみもの教室500円
