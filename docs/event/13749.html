延期のお知らせ

三陸沖を震源とする非常に強い地震により、東北地方を中心に広い範囲で大きな被害が発生しております。
被災された方々には心からお見舞いを申し上げます。

３月14日（月）に予定していました、Android Nightについては延期させて頂きます。

すでに登録して頂いた方々にはご迷惑をおかけしますが
ご理解いただきますようお願いいたします。

Welcome Androider

ゆるーく、らくーな感じでお集まりください。
皆でアプリ作ったり、センサー組み込んだり、ハンダ付けしたり、新しい端末自慢したり、皆で改良したり、行く末を案じたり、ワイワイする場になればいいなと考えています。
自作アプリのプレゼンも大歓迎です♪

～Androidな人なら開発者、ユーザー、ビジネスマン、起業家だれでも参加OKです～

参加費５００円です。飲み物やおつまみ代になります。
途中参加、途中退室も可です。お気軽にどうぞ。

●Source code reading

今まではNightの中で行っていましたが、より良いものにするためAndroid Nightとは別のイベントとして開催予定です。
