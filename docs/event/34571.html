iPhoneアプリ開発初心者勉強会

iPhoneアプリ開発も初心者（中級者）同士
情報共有する場があったらいいなと思い開催します。

【対象】

  • Xcodeをインストールしてみたけど何したらいいか分からない人。
  • 絵やアイディアには自信があるのでプログラムを組んでくれるパートナーを 捜したい人。
  • スマートフォンアプリ制作に関する情報を共有したい人。
  • 自分のアプリを宣伝したい人。
  • 初心者に色々教えたい人。
  • 色々教えてもらいたい人。

【内容】

  • 各自もくもくと好きに自分のアプリを作って
    相談し合う（情報共有し合う）場としたいです。
  • 私でよければアプリの作り方をご説明します。
    私の実力は↓のアプリをやっと作れるくらいですので
    難しい事は聞かないで下さい。。。
    iPhoneアプリ：九九衰弱
