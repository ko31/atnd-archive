概要

北海道の Drupal ユーザーコミュニティ「Drupal さっぽろ」が主催する meetup (勉強会) です。地域の Drupal ユーザーの情報交換と交流を目的に、月に一度、札幌と小樽で開催しています。今回は札幌での開催です。

開催予定は、年間計画をホームページに掲載し、
http://drupalsapporo.net/
個別の予定を ATND に公開後、Facebook、Twitter を通じて告知します。
https://www.facebook.com/drupalsapporo
https://twitter.com/drupalsapporo

北海道の Drupal コミュニティとして情報交換と交流の輪を広げ、さらなる活用につなげていきましょう。

参加料：500円（会場費として）

内容予定

(1) 前回の復習と宿題

  • D7 Theming のおさらい
  • D8 Theming の基礎
  • D8 モジュールの作り方
  • サンプル

下記チュートリアルを一緒にトレースしましょう。
目標：カスタムブロックを作るところまで
https://www.drupal.org/docs/8/creating-custom-modules

(2) フィールドとエンティティ

Drupal 8 の Entity API の資料を一緒に読み進めましょう。
目標：Drupal Console で生成したエンティティのソースを理解する
https://www.drupal.org/developing/api/entity

  • Drupal Console について
  • エンティティの作成
  • 生成されたソースの確認

(3) 温泉合宿に向けて

  • 現状と今後の予定
  • その他
