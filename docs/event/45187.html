★参加者の方の準備

  • Eclipse 4.3.1 を新規インストールしておいてください。
    　http://eclipse.org/downloads/
  • 上記Eclipse環境に、Google Plugin for Eclipseをインストールしておいてください。
    　https://developers.google.com/eclipse/docs/getting_started
    　https://dl.google.com/eclipse/plugin/4.3

【重要】
古いバージョンのEclipseやGoogle pluginでは、
ハンズオン内容が実行できない可能性がありますので、
既にEclipseを導入済みでも、必ず新規に環境をひとつ用意して下さい。
※Eclipseは、1PCに複数導入可能です。
