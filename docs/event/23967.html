description

みかん・オレンジ・グレープフルーツ・はっさく・ぽんかん・夏ミカン・レモンなどなど、さまざまな柑橘類を食べ比べます。
「晩白柚（ばんぺいゆ）」や「日向夏」などのレアものや、1個ン百円もする超高級みかんも試食します。
また、前回好評だった、味覚のうち酸味だけを消すという「ミラクルフルーツ」にも再チャレンジです。

さらに“口直し”には、広島県呉市の三宅水産から「がんす」を取り寄せて、がんすアラカルトをご用意します。
焼酎と炭酸水も用意しますので、お好みの果実をその場で絞って生果汁サワーをお召し上がりください。

なお、オプションとして、初春の運試し・中山競馬「東雲賞」にチャレンジ！ という企画もございます（当日15:00発走）。
