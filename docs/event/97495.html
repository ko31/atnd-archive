東京で活動する社会人、大学生が参加するテニスサークルです。金融機関に勤めている同僚とテニススクールに通うようになり、友達が増えてサークルとして月3回程度活動しています。初心者~初中級を対象にしてますので実業団でプレーしている選手が1~2名コーチとしてテニススクールばりのコーチをしてくれます。

2018年6月30日テニスコートが確保できたので、参加者を募集します。
テニスコーチを呼んでレッスンをしてもらう中の参加者の1人として参加の募集です。入会金や年間費等はもちろん不要。

■都内都営・区営・都内マンションのコートを利用
（場所:千代田区・港区・文京区・新宿区・江東区・墨田区など）
■1名コーチをお願いして、球出しやレッスンをしてもらう形式
　（参加時に都度払い、入会金、年間費等はありません）
■50名程度所属して1回の参加はコート面数次第ですが9～17名（1、2名コーチ）]
■テニスを１から始めたい方、社会人からテニスをはじめた初級の方が中心(対象）です
■食事会は自由参加
■大学生～社会人まで幅広く参加
■大学生は6大学出身者が多いです（途中からサークルに入れないから？社会人サークルに入る）
　社会人は金融機関出身者が多いですが、友達の紹介でIT、コンサル、通信、メディア、製薬系の方等

★日時：2018年6月30日（土）テニス16：00～18：00　※15:45　大横川親水公園テニスコート（錦糸町駅より徒歩）

★参加費：初回：2500円（こちらから支払をお願いします）
　　　　　2回目からは現地払い2,000円

★食事会　18:30以降（自由参加 各自精算）
