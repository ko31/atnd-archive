description

各業界の第一線で「地方創生」に携わる経営者が一堂に会し、様々なテーマで議論を交わすセミナーを開催。古民家再生、クラウドソーシング、地産地消の電力、地方特産品ブランディングなど、いろいろなアプローチで地域活性に携わる中で見えてきた手ごたえや成功事例、抱える課題や今後の未来などを語ります。

■下記に当てはまるような方は是非ご参加下さい！
・地方創生関連のお仕事に携わっていて、いろいろな知見を学ばれたい方
・自身のふるさとへの貢献を考えていて、どのような貢献の仕方があるか知りたい方
・地方創生や地域活性というキーワードで自身の経験・能力を発揮したいと思われている方

登壇者は、皆さん一般企業で様々な経験をし、事業を立ち上げた方々です。ビジネス感覚を持ち、如何に事業を成功させつつ、地方活性に貢献するか。その貢献が事業成長にも直結し、名ばかりでは終わらない「実のある地方創生」「継続性ある地方活性」に携わっています。質疑応答の時間も設けているので、皆さんの日ごろお持ちの疑問などをこの機会に投げかけてみてください。

【登壇者プロフィール】
株式会社クラウドワークス 代表取締役社長 CEO 吉田浩一郎 氏

1974年兵庫県神戸市生まれ。東京学芸大学卒業後、パイオニアなどを経て、ドリコム執行役員としてマザーズ上場を経験した後に独立。2011年11月に株式会社クラウドワークスを創業。翌年3月にサービスを開始したクラウドソーシングサービス『クラウドワークス』を展開。2014年12月東証マザーズ上場。2016年6月現在、登録会員は95万人、利用企業は上場企業をはじめとして13万社にのぼる。2015年には、経済産業省 第1回「日本ベンチャー大賞」審査委員会特別賞を受賞、グッドデザイン・未来づくりデザイン賞受賞。2016年には、一般社団法人新経済連盟理事に就任。著書に『クラウドワーキングで稼ぐ! ―時間と場所にとらわれない新しい働き方（日経新聞出版社）』『クラウドソーシングでビジネスはこう変わる（ダイヤモンド社）』などがある。

一般社団法人ノオト 代表理事 藤原岳史 氏

株式会社NOTE 代表取締役　株式会社NOTEリノベーション＆デザイン 代表取締役

東京・海外でのIT企業の経験などののち、故郷である篠山に拠点に、兵庫県内を中心に古民家再生に取り組む一般社団法人ノオトの立上げにかかわる。

今秋には、国家戦略特区の認定事業として、旅館業法の特例で、複数の古民家のフロントを一元管理する「篠山城下町ホテルNIPPONIA（ニッポニア）」を開業させる。

この取り組みは、日本政策投資銀行などが設立した「観光活性化マザーファンド」と株式会社但馬銀行が協調した投融資で資金調達し、補助金に頼らない地域創生をめざす。

みんな電力株式会社 代表取締役 大石英司 氏

1969年、東大阪市生まれ。広告制作会社、PCソフト開発会社を経て凸版印刷へ入社。

電子出版・有料デジタルコンテンツ流通の先駆けとなる「ビットウェイ」の起案・事業化、「原宿スタイルコレクション」「IKKO美脚革命」などのプロデュースを行う。

2011年、｢みんな電力株式会社｣を設立。小型ソーラー充電器「てのひら発電　空野めぐみ」の開発・販売、太陽光やバイオマス発電の設置・コンサルなどの他、「顔の見える発電所」をテーマに電力小売り事業を行う。

ENGAWA株式会社 代表取締役社長 牛山隆信 氏

1981年4月10日　長野県生まれ。武蔵野美術大学デザイン情報学科卒。カルチュラル・エンジニアリング専攻。ITコンサルティング会社を経て、2009年、株式会社サニー サイドアップに入社。デジタル領域における広報宣伝や事業コンサルティングを担当する新設部署にて、大手企業の広報業務を多数経験。特に 海外への日本の魅力発信とインバウンド対応をテーマとした広報案件に携わる。2013年、新規事業の開発を担当するムーブメントパー ティー戦略本部の創設、31才の若さで本部長に就任（最年少）。2014年「OMOTENASHI NIPPON」プロジェクトを立ち上げ、「おもてなしセレクション」の企画・戦略立案から、パートナー企業とのアライアンス構築等、幅広 い業務に従事し、日本全国地域の文化・産業の発展に取り組む。2015年12月サニーサイドアップがジャパンブランドの海外への魅力発 信・ビジネス支援を行う子会社、ENGAWA株式会社を設立し、代表取締役社長に就任。OMOTENASHI事業やインバウンド景気に沸 く外国人向け英字ライフスタイルマガジン「Tokyo Weekender」事業の拡大を推進している。

【ファシリテータープロフィール】
株式会社プラスディー 代表取締役 Founder 本田晋一郎 氏

1979年9月26日愛知県生まれ。慶應義塾大学理工学部統計学専攻を卒業し、株式会社WOWOWに入社。音楽、映画、スポーツチャンネル等の宣伝/プロモーションに携わったのち、ITベンチャー企業へプロデューサーとして入社。洞爺湖サミットのデジタルプロモーションなどを手掛けた後、2008年株式会社プラスディーを設立。映画の企画制作、デジタルプロモーションを中心に事業活動を行う傍ら、地方創生に関わる地域ブランディング/プロモーションを地元・豊田市と連携して取り組んでいる。愛知県豊田市第8次総合計画審議委員として2040年を見据えた地域づくりにも携わる。
