    [内容 1部(午前)初級サーバ管理者向けハンズオン]
  • PHP + Webサービス by 飯島さん

    • 概要: WebサービスでPHPを使えるようにする 。また拡張モジュールのインストールとPHP-PAMを使って認証を統合する。ただし、PHP言語の解説は致しません。なお、実習用のマシンは、必ず参加者が準備お願いします。もしマシンを準備できない場合は、ハンズオンが行えません。
    機材とソフトウェア: MacBook / Mac OS X Server v10.6 on VMware Fusion 3.0での実習を想定しております。


    [内容 2部(午後)サーバ管理者向けプレゼンテーション]

  • WebサービスのSSL by 新居さん

    • 概要: 「SSLは1つのポートに1つの証明書しか使えない」という事実を軸にして、SSLや証明書の話を交えて、マルチドメインのWebサーバ運用に関して解説をします。

  • モバイルアクセスサーバ by 大久保さん

    • 概要: モバイルアクセスサーバについて、iPhoneでのデモを交え

  • ネットブート パート1 by 田畑さん

    • 概要:「NetBoot part1」NetBootについて解説します。Snow Leopard ServerではNetBoot/NetInstall/NetRestoreが利用可能ですがまずはNetBootから。
