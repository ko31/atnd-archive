注目！！ （残席わずか！）

どうすれば融資を有利に進められるかを知りたくありませんか？

現役の日本政策金融公庫の担当者様を招いて直接お伝えします。

こんな方にオススメ
・融資を検討している経営者の方
・融資を一度断られた方
・銀行マンや日本政策金融公庫から直接話しを聞きたい方
・創業を考えられている方

◯内容（銀行から借りられることができる人とできない人の違い）
・融資を受けるには、何から始めたら良いか？
・融資担当者はどこを見るのか？
・融資を有利に進めるポイント
・融資に有利な決算書とは？
・制度紹介
・質疑応答

◯開催日時（2回開催します。どちらかだけの参加も可能です。）
日時：2014年7月18日（金）

講習：10：00~11：30　懇親会　11：30~12：30（融資担当者様にも出席いただきます）

◯場所
愛知県名古屋市中村区名駅3-5-7　「楽楽」2階

◯受講料
1500円（懇親会費・昼食代含む）昼食不要の方は1000円
