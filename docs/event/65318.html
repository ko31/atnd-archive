　桜もすっかり花吹雪となり、街には、初々しい新入生や新社会人の姿が行き来する４月となりましたが、皆さんは、いかがお過ごしですか？年度初めのバタバタの中で、すっかり朝食会の開催案内が遅くなってしまったこと、お詫び申し上げます。そして、今回よりイベント管理ツールを新しくしてご案内いたします。

　今月４月号の特集は、「『頭がいい人』の条件が変わった。」でした。人材教育・育成に関わる私からすると、何を今さら感があるのですが、皆様はどうとらえていらっしゃるのでしょうか。“知識偏重の教育から、“考え方”重視への方針転換は、腰の重い文科省も昨年度ようやく、大学入試を2020年変えることを決めたことが記憶に新しいことかと思います。本文では直接的な表現ではかかれていませんが、もう一つの大きな変化は、“IQ”より“EQ”での評価、すなわち人間力が問われる時代になってきたということでしょう。

　もう一つ、今月号で私の目を引いた特集が、「廃墟」巡礼。神戸在住の女性カメラマンの方が昨年、「女子的産業遺産探検」という写真集を出版されたり、探検ツアーも開催されたりで、少しブームになりつつもあります、廃墟。早速、世界にも目を向けてネタを集めてきたカメラマンの、そのスピードに脱帽しました。

会場はいつも通り、阪急の高架下、西口と東口の間にあるタリーズコーヒーです。三宮駅の浜側ではなく、山側に出ていただく必要があります。タリーズコーヒーは近くに数店舗ありますが、お間違いのないよう、ご注意ください。

朝食会では、真剣な話から、軽いノリの話まで、いろいろな話題でおしゃべりしています。 初めての方も、どうぞお気軽にご参加下さい！

　Byたけ（@lohasway）

クーリエ・ジャポン DE 朝食会＠神戸　とは？ クーリエ・ジャポン朝食会とは、世界1500のメディアから厳選した記事を紹介している講談社の『クーリエ・ジャポン』という雑誌について、朝ごはんを食べながら、感想や意見を語り合う読書会のこと。読者が主催となって、東京、名古屋、大阪、福岡など各地で行われています。

神戸の朝食会では、会の中でも記事を紹介していきますので、クーリエの読者はもちろん、まだ読んだことがないという方にもご参加いただけます。

  • グローバルな視点を求めている人
  • いろんな国のこと、世界から見た日本に興味のある人
  • 自己成長のヒントを探している人
  • 面白い人とつながりたい人
など、多くのご参加をお待ちしています。学生さんも大歓迎です！クーリエという共通の話題を通じて、新たなつながりをつくり出し、お互いの見識を深められる場としていきたいと思います。
