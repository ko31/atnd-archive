title ガジェットカフェミーティング 2011 SUMMER
catch モノづくりの未来はここにある
description

近年、オープンソースハードウェアの登場によって、テクノロジーを活用したモノづくりを誰もが始められる時代になりました。近い将来、テクノロジーを身近な何かと絡めた無数の境界領域から、新しいモノづくりが次々を誕生していくだろう。そう確信している。なぜなら、その未来は既にここにあり、見て、聞いて、触れることができるのだから。

ガジェットカフェミーティング 2011 SUMMERは、モノづくりの未来を切り開くクリエイターとプラットフォーム運営者によるプレゼンテーションを中心とした交流イベントです。各々の最近の作品や活動の紹介、そこに込められたビジョンについてお話いただき、参加者と共有していきます。出演者は順次決定中です。
