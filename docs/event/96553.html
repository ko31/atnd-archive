【テーマ】

「今さら聞けないクラウド入門　～Microsoft Azureのススメ～」

クラウドに興味があるけど、どんなものなのか今さら聞けない。。 　　　　　　　　　　　　　　　　そんな方々向けにクラウドの初歩から実践的に学べるワークショップを実施します。

今となってはなかなか聞けないクラウドとはどんなものなのか、 という基礎部分を中心に解説をするセミナーとなります。

最初にクラウドとはどんなものかを説明した後、最近ニーズが増えつつあるMicrosoft Azureについて、デモを交えながら分かりやすく解説します。

オンプレのWindowsサーバをクラウドに移行することをご検討の方には最適です。

【ワークショップ概要】

第一部:クラウドの基礎とMicrosoft Azureのご紹介（45分）

講演者：川村

講演内容：クラウドの基礎の説明をした上で、Microsoft Azureとはどんなものか、何が優れているかを紹介します。

第二部:Microsoft Azureを使ってみる（45分）

講演者：吉崎

講演内容： Microsoft Azure上に仮想マシンを構築したり、サービスを利用したり、Microsoft Azureの基本的な利用方法をデモで解説いたします。

第三部:懇親会(30分)
