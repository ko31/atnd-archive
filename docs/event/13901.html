title HAPPY EARTHDAY OSAKA 2011
catch ハッピーアースデイ大阪２０１１　～　環境のことを考え、幸せを感じる　～
description

アースデイとは地球のことを考えて行動する日として、世界的な環境ムーブメントとして各地で展開されています。今回は「幸せを感じること」をテーマに、大阪４大公園の一つ久宝寺緑地にて日本初、学生主体のアースデイを開催します！　来場者の方々をはじめ、この運動に関わる全てのひとが幸せを共感できることを目指します。
　またさる３月１１日に発生いたしました東北地方太平洋沖地震について、ハッピーアースデイ大阪でも当日、義援金箱の設置、及び、有志の出店者様から『当日売上の何パーセントかを義援金』に当てる他、さまざまな支援等を検討しております。一人でも多くの方にご協力のほどをお願い申し上げたい次第でございます。

ぜひご参加お待ちしております。

《開催概要》
開催日：2011年03月26日11:00～17:00・27日10:00～16:00
開催場所：久宝寺緑地（大阪府八尾市） 修景広場周辺

主催：ハッピーアースデイ大阪実行委員会
共催：都市公園久宝寺緑地指定管理共同体
後援：大阪府・大阪市・東大阪市・八尾市・八尾市教育委員会・大阪府社会福祉協議会・八尾市社会福祉協議会（順不同）

企画内容：展示・出店販売・ＬＩＶＥ・ワークショップ・マラソン等のスポーツイベント・トークショー・アート展・キャンドルナイトを開催予定等

※出店販売：自然素材、オーガニック、フェアトレードなどをテーマにおよそ１１０ブースの出店予定。

※Live出演者予定：Lovearth Camp・日食なつこ・theSing2YOU・MAYA BELLY EARTH COMPANY・drumno
※アート出演者予定：HARMONISMによるライブペイント他

料金：入場無料
