モバイル最新動向勉強会

Androidを中心とした最新のモバイルOSの動向についての勉強会です。
現在調整中ですが、Tizen, Firefox OS, Ubuntu Phone, 等についてもお話できたらと考えています。
Chromecast と NexusQ をWi-Fiで連携させたり、 LEAPモーションのおさわり会を予定。
そして、会場からのモバイルOSについての質問をお答えするコーナーを企画いたしました。
