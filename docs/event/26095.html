description

Ｑ 「え！？株式投資とかＦＸってキケンじゃないの？」
Ａ「株式投資やＦＸで損をしてしまうのは単にリスク管理できてないからである。本物のリスク管理ができれば、利益が出てしまうのが投資である！」
日々投資を研究し続け、Iクワドラントのテクニックを学べる初心者株塾開講！
投資家の視点から見た、本物のリスク管理の方法を学ぶことができます。
当株塾はまだ株式投資をしたことない人、株式投資をしてみたが思うように結果が残せていない人、以前買った株を
今後どうしようか迷っている人向けです。
投資はキケンではありません。何がキケンなのかよく分からないで投資することがキケンなのです。
投資で儲けが出せないのはリスク管理ができていないだけである！と言い切る塾長が教鞭をとります！
投資でラットレースを抜け出したい方は必見です！
