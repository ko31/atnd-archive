■ホームページ作成（広告・宣伝・販路開拓等の取り組み）に使える
返す必要のないお金が
５０万円までもらえます
と言われたら、どうしますか！

起業後３年で、すでに５０名以上の経営者の方の集客のお手伝いをしてきた
五味正巳　が　今回初めて、補助金の専門家と組んで
１１月４日に受付を開始したばかりの
『小規模事業者持続化補助金』　の説明と、
その補助金を使って、どんな取り組みができるのかをお話しする
『セミナー』を開催いたします。

セミナー内容

今回　開催するセミナーは、
•ホームページ作成・広告・宣伝をしたいけど、何をどうやったらいいかわからないし
そもそもそんなお金、出せないことはないけど
失敗したときのことを考えると怖い
•順調なので、もっと広告・宣伝して事業を伸ばしたい

そんなあなたにピッタリの内容になっています。

今回の補助金は、もらいっぱなしで返す必要のないお金です。

■補助金が使える施策

使える施策としては

１、 ホームページ作成

その他に

２、 ランディングページ作成
３、 インターネット広告（ＰＰＣ広告・フェイスブック広告等）
４、 チラシ作成・チラシポスティング
５、 コンサルタントへのコンサル料等があります。

例えば　今のホームページからはあまり集客ができていないとします。
そんな場合に、今のホームページを作り直す絶好のチャンスです。

また　まだホームページを持っていないあなたなら
ホームページを作る、最高のタイミングです。

補助金でホームページが作れて
さらに余った金額を
ホームページ以外の広告・宣伝に使うことができます。

かなりお得だと思いませんか？

ホームページを例に出しましたが
使える施策はまだまだあります。

今回のセミナーでは
どんな施策をどうできるのか
そして　補助金についての詳しい説明をします。

■セミナーでわかること

セミナーでわかることの一部を紹介すると・・・
•補助金の仕組みがわかります
•補助金のもらい方がわかります
•補助金の使い道がわかります
•補助金を使ったホームページ作成方法がわかります
•補助金を使った集客方法がわかります
•補助金を使った集客方法を、どのように使って集客していくかがわかります

★もし　あなたがホームページを作り直したり、
新たに作ったりしたいと思っているのなら,
さらに広告・宣伝にお金をかけたいと思っているのなら
この補助金をもらわない　という選択肢はあり得ません。

■今回の補助金のポイント

1.使った金額　全額もらえるわけではありません。
使った金額の３分の２までの金額の補助です。

2.すべての施策が終わった後でもらえるお金です。

■こんな人にお勧め

ホームページは持っているけど、そこからの集客が全然できていないので
作り直したい　と思っている。
または、まだホームページを持っていないので作りたいと思っている。
でも、ホームページから集客できるかどうかわからないのに、
大きなお金を掛けるのはどうしたものか　と迷っている。
そんな方が補助金を獲得する大きなチャンスです。

■期待できる効果

補助金を使って作ったホームページからの集客が期待できます。その他にも
ランディングページ作成・リスティング広告・チラシ作成・チラシポスティングなどに使って集客効果を高めることができるようになります。

■主催者からのメッセージ

今回のセミナーは　集客の専門家　五味正巳　と
補助金の専門家　行政書士の　大橋真行　の
両　専門家が行うセミナーです。

補助金をもらうためのセミナー
申請書の書き方のセミナー
申請を通すための書き方セミナー

などは　各地の商工会議所で無料で行っていますが
今回のセミナーは　補助金を獲得した後の
ホームページ作成・広告・宣伝の方法や、やり方まで
突っ込んでお話しする
他にはないセミナーになっています。

この機会を逸すると
来年度には実施されるかどうかわからない補助金です。
