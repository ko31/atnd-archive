医療保険の基礎セミナー

イベント ユーザー
event_id 44242
title 医療保険の基礎セミナー
catch 知っておきたい賢い医療保険の入り方
description

テレビCMや通信販売でよく見かける医療保険。
「種類がたくさんありすぎて、どれを選べばよいかわからない」という声をよく聞きます。

あなたは医療保険に入っていますか?
保険金額や内容を理解していますか?
保障はいつまで続きますか?

セミナーでは医療保険選びのポイントを基本からわかり易く説明します。
無駄なく、漏れなく、賢く保険に入りましょう!

【セミナー内容】
・そもそも医療保険ってなに?
・ほんとうに必要なの?
・保障はどれくらい必要?
・公的健康保険との関係は?
・合理的で無駄なく加入する方法は?
・病気があると入れないってほんと?
・先進医療ってなに?
・パンフレットを見る時のポイントは?
など。
セミナーでは皆さんが抱いている医療保険の疑問やお悩みを解決します。

同じような保障内容でも、契約の仕方で保険料に大きな違いが出てきます。
総額で数百万円の違いになることも珍しくありません。

講師は銀行・保険会社・保険代理店で働いてきたファイナンシャルプランナー。
長年金融業界で働く中で『人生には金融知識が必要。あるか無いかで大きな差が出る』ことを体感してきました。

医療保険も重要な金融知識の一つ。
社会保障が削減されつつある中、老後を見据えて若いうちから備えておきたいですね。

セミナーは参加者の理解が深まるよう少人数制。皆さんと雑談をしながらすすめます。
多くの参加者から『大変役に立った』と好評をいただいております。

セミナーでは金融商品の販売勧誘は一切いたしません。お気軽にご参加ください。

【無料個別相談】
セミナー終了後、希望者のみ個別相談を実施しています。(後日予約制)
・今入っている保険の内容がよくわからない。
・女性向けの医療保険はどのような内容なの?
・ガンや重い病気はいくら治療費がかかるの?
などのお悩みにお答えします。

席に限りがありますので、お早めにお申し込みください。
