description

OXID eShopはドイツで開発されているオープンソースeCommerceシステムで、豊富な機能とパフォーマンス、簡単に使える操作など、たくさんの特徴があります。　ドイツでは多くのOXIDで構築されたＥＣサイトがあり、オンライン決済をはじめ、拡張機能も豊富にあります。　日本でのサンプルサイトとして　和装美どっとこむ(http://wasoubi.com) 福祉ＥＣショップ(http://shop.yuuyake.or.jp)　

概要：　機能や使い方の概要をハンズオンで学びます。 ECサイト構築ソフトを検討している方に最適です。
参加者：　どうのような機能があるのか、まずは、導入前に機能を確認したい方、概要を知りたい方、事業のマネッジメント層に最適。

参加費用　　19,600円（別途消費税）

日時　　4月4日（水）　10：00－16：30

場所　　JR川崎駅東口下車　徒歩15分　サンピアンかわさき

内容　　オープンソースソフトウェアeCommerce – OXID eShopの概要、操作、商品登録、管理ほか
