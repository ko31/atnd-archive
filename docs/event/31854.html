イベント概要

2012/8に開催された「おばかアプリ選手権」でWeb CAT Studio賞を受賞した
株式会社鳥人間さんプレゼンツ！

プログラマなのに製品を作ってしまった、ハードウェアハッカー(Maker)鳥人間の久川さん、
大手医療機器メーカーのエンジニアから、一人家電メーカーに独立したBsizeの八木さん、
同じく大手家電メーカーの商品企画から独立し、家電メーカーを立ち上げたCerevoの岩佐さん、
３名のお話から、「おばかアイデアを形にする方法」や「モノづくりの新しい形」について考えてみるイベントです。

「ソフトウェアエンジニア」「ハードウェアエンジニア」「デザイナー」など、
様々な属性の方が集まって、自由に情報交換をしていただける場です。

また、勉強会終了後は懇親会も開催しますので、講師の方や参加者との情報交換・交流を深めて下さい！

【主催】
Web CAT Studio （株式会社リクルートエージェント）

【会場】
グラントウキョウサウスタワー24F セミナールーム24A（東京都千代田区丸の内1-9-2）

講演者

株式会社鳥人間 久川 真吾氏　

草履から人力飛行機や飛行石まで、Webもスマホもハードも自作が趣味かつ仕事のハッカー。
株式会社鳥人間として、電子工作コンテスト、Mashup Award、おばかアプリ選手権2012など、受賞多数。

ビーサイズ株式会社 八木 啓太氏　

デザインエンジニア。1983年生まれ。大阪大学大学院修了。電子工学を専攻。
富士フイルム株式会社にて、医療機器の機械設計に従事。
デザインを独学し、2011年、Bsize設立。

株式会社Cerevo 岩佐 琢磨氏

立命館大学理工学部卒業後、松下電器産業（現・パナソニック）に入社。デジタルカメラ、テレビ、DVDレコーダー等のネット連携家電の商品企画を担当。
2007年、家電のスタートアップ企業として、株式会社Cerevo（セレボ）を設立、代表取締役CEOに就任。
インターネット連動のデジタルカメラ「CEREVO CAM」、Ustream配信機器の「Live Shell」が、
日本・北米を中心に注目を集めている。

・デザイナーの方
・おばかアイデアをお持ちの方
・クラウドファンディングに興味がある方
