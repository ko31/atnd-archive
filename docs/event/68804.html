「士業は資格を取れば稼げる！」そんな時代は終わりました。

今回は、「行政書士の開業講座」で人気の講師をお呼びして、士業限定のセミナーを開催します！
若くして行政書士事務所を開業し、苦労した経験や知識を存分にお聞き出来ます！

弁護士であろうが、税理士であろうが、行政書士であろうが、
稼げない人は稼げない。人助けできない人は出来ない。
その理由は、すべて『自分自身』にあります。
難しい試験を合格し、厳しい実務をこなし、いざ開業したが、
どうやって集客したらいいのかわからない・・・。
高いお金を払ってホームページを作ってみたものの・・・。
口ベタで営業トークが苦手・・・
事務所運営の仕方ってどうするの・・・？
など、様々な問題が出てきます。
そんな時にどうしますか？
実は多くの方が、間違った方法を教えてるセミナーや開業講座、実績のない先輩や知人。
に教わっています。
大事なのは、『士業の前に経営者』という事です。
そこに気がつかずに、集客のノウハウだけ、実務の知識だけ？こんなものをマスターして何になるんでしょうか？
もっと本質的な事を、今回のセミナ－で学んで頂ければと思います。
皆さんが知っての通り、しっかりと集客・経営すれば、利益率が高く、確実に儲かる商売が士業です！
そして、他業種と比べ、ライバルが少ないのも事実です。
それは、多くの士業さんが、肝心のお客様を呼び込む…いわゆる集客方法知らない。
又は間違った方法を実践している方が多い業界なのです。
ですので、今回のセミナーを聞いた方は、
実践的な集客方法を学んでもらい、周りとの差別化が出来、
より自分らしい事務所運営が出来るはずです！
今回は、士業で開業予定、又は開業後1年未満の方を対象に、
『開業して必ず悩む集客を「無理せずゼロから生み出す方法」』
についてお話します。

※最後に交流の場を設けておりますので、名刺のご用意をお願い致します。

2009年行政書士試験合格。2010年26才という、当時の業界では、かなりの若手として、独立開業。集客、実務、すべてが未経験のまま、手探りで事務所運営していく中で、本当の意味で悩んでいる人を救うためには、自分の力だけでは無理だと感じ、『無理せず開業講座』を設立し、セミナー活動を通じて、100名以上の全国の行政書士などとつながる。
全国の士業とつながったことを機に、お客様の相談を聞く専門家として、心のケアーに専念したいと、福井県のお寺に安居（お坊さんの修行）
2015年行政書士として復活し、窓口行政書士として、相談を聞くことに専念した、全国唯一のお坊さん行政書士として、現在、会員制（集客・起業）コンサル相談室『無理せず起業＆人生塾』の塾長として、活動中。

阿部行政書士から。

このセミナーは、
士業の世界を知りたい、士業の世界が見えず悩んでいる
士業として開業してやっていく自信が無い
士業として開業したが、何をしたらいいか分からない
士業になったが、不安で不安で仕方ない
ビラや名刺を一生懸命配っているが、なぜか成果が出ない
自分の強みを出せといわれたが、その強みがわからない
開業本を読んだが、結局どうすればいいかわからなかった
行政書士として、今はうまく行っていないが、一生頑張っていきたい！
士業の実務は色んな人が教えてくれるけど、それ以外のことも聞きたい
きれいごとや、建前じゃなく、本当の士業の世界を知りたい。
そんなあなたに、お勧めのセミナーです。
わたしは、行政書士として開業したのが、26歳のとき。
よくある、コネや、金なしなんてレベルではなく、社会人経験もほぼなく、何にもない状態で、ぽんと、行政書士の世界に飛び込みました。
そして、今でこそたくさん増えた、行政書士の開業講座ですが、わたし達の頃は、小さなものしかなく、しかも、実務を教えるだけというものばかり。
いや、正直、今でもこの流れは変わっていないんですね。
実務が出来れば、行政書士として成功できる？
集客ができれば、行政書士として、集客出来る？
そんな幻想に、わたしたちは、何度も振り回され、つぶされ、悔しい思いをしながら、5年間、必死に事務所を守り、生き抜いてきました。
多くの資格の学校や、開業講座は、お客さんほしさに、甘いことをいいます。
『行政書士になったら、1千万円稼げますよ』
違うんです。
『行政書士になったら、人助けできますよ』
まったく違います。
『行政書士になったら･･･』
そもそも、ここが間違っていることに早く気がついてください。
わたしは、この、『行政書士』という囚われの中にいたとき、すごくすごく苦しい思いをして生きてきました。
もうやめてしまおうと、何度も、看板をはずしたり、付け直したり、友人になきついて、仕事をもらったりと、本当にたくさん嫌な経験をしました。
そんなときに、尊敬する士業の先生に、「自分に出来ないことは、仲間に助けてもらえばいいんだよ。だって、行政書士の前に、経営者でしょ？」といわれたときに、初めて自分の過ちに気がつきました。
そう、行政書士は、自分の一部であって、行政書士という資格に、使われる自分であってはいけないんだということに。
わたしは、この囚われを抜け出してから、士業の仲間が増え、お仕事が増え、今まで感謝されなかった、お客さんから、笑顔で、「ありがとう」といわれるようになりました。
そして、その囚われを、捨てたことにより、他の士業の先生から、依頼が来るようになり、今では、「阿部さんは、お客さんを完全に任せられる、数少ない先生だ」といわれるまでに、なりました。
そう、行政書士として、生きていくためには、行政書士ではなく、経営者としての自覚を持つことからすべてがスタートするんですね。
その自覚に気がつき、育て、そして、自分たちなりの活用方法を考える。
そういうことが出来る士業になって欲しいと思います

今回のこのセミナーでは、集客だけではなく、経営の基礎まで学べるセミナーになっております。

ですが、「今すぐに売上を２倍にしたい！ラクしてにお客様を増やしたい！」という非現実的な事をお考えの方は、

今回のこのセミナーは対象外となります。

そうではなく、

「毎月数件からでも、安定的に集客したい！」

「無理せずに確実にお客様を増やしたい！」

「しっかりとした考え方やノウハウを身に付けたい！」
