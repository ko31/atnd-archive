catch BaserCMSテーマ制作のハンズオンセミナーです。
description

10月10日（日）、BaserCMS の第２回勉強会を開催いたします。
第１回と同様、BaserCMSの導入から、テーマ制作までができるようにハンズオン形式での勉強会を予定しています。BaserCMSのコア開発者から実際の制作に役立つTIPSなどもお伝えします。

【スピーカー】
ryuring：BaserCMSコア開発者

【勉強会内容】
・BaserCMSのインストール
・BaserCMSのテーマ制作
・質疑応答など

■ 日時: 2010年10月10日（日）16:30 – 19:30
■ 参加費: 無料
■ 定員: 20名
■ 会場: デジタルハリウッド福岡校セミナールーム
■ 住所: 福岡市中央区天神1-7-11 イムズ11F
■ 参加資格: BaserCMS に興味がある方なら誰でも参加OKです。
■ 用意するもの:
次のものはなくてもいいですがあった方がより理解を深めれると思います
・ノートパソコン
・PHPが実行できる環境（XAMPP、レンタルサーバーなど）
※ 当会場にインターネット接続の用意あり

■ 懇親会: 場所未定

■ BaserCMSとは
「コーポレートサイトにちょうどいいCMS」をコンセプトに、
オープンソースとして公開されている無料のCMSです。
シンプルかつ、カスタマイズ性が高いという特徴があり、
ITリテラシーが高くない方でも簡単に扱えるように
シンプルでわかりやすいCMSを目指して開発が続けられています。
≫ BaserCMS公式サイトはこちら
