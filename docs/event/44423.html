いきさつ

去年の夏は「Bar Windows Phone in Nagoya」、今年の夏は「Bar Windows Azure」。
では、今年の秋は？
もちろん「Bar Windows 8」ですよね！
10月18日に予定されている Windows 8.1 の一般公開を祝い（？）ながら、一緒にお酒を飲むイベントです。
（お酒を飲めないかたは、もちろん、ソフトドリンクでもＯＫ！）

そもそもこの「Bar なんとか in Nagoya」シリーズは、東京・渋谷の「Bar Android」が、2012/4/23に「Bar Windows Phone」として開催されたのをヒントにしています。

そして、渋谷の「Bar Android」は、毎週月曜限定のバーなので、名古屋でも同様に毎週月曜限定でオープンしている「名古屋Geek Bar」の会場をお借りします。

どんなイベントですか。

勉強会とかではありません。
みんなでワイワイ／ガヤガヤとWindows 8 や 8.1 について自由に語り合う集いです。

今回は、新製品というよりバージョンアップのリリースなので、あまり派手にせず（笑）、落ち着いた雰囲気で進めて行こうと思っていますが、さてどうなるでしょうか．．．

なお、GeekBarというお店の営業の一環として行われるので、好きな時間にやって来て、好きな時間に帰っていただいてかまいません。（ただしチャージ ￥1,500 のお支払いが必要です）
