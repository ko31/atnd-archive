このたび浜松市は、首都圏で活動するベンチャー企業関係者や起業志望者に向けて、市への拠点誘致等を目的とした浜松の魅力発信のための「第4回浜松市ベンチャーフォーラム START UP CROSSING」を東京の「BPM」において開催することを決定しました。浜松市は、2016年4月に開設した「はままつ首都圏ビジネス情報センター」が行う①ベンチャー企業誘致、②工場・研究所誘致、③産業人財獲得支援プロジェクトのうち「ベンチャー企業誘致」を主眼とするものであり、今回が第４回目となる取組です。

浜松市は首都圏と関西圏のほぼ中間に位置し、四季折々の豊かな自然と温暖な気候に恵まれており、｢2018年度版20政令指定都市『幸福度』ランキング｣では映えある第１位を獲得しています！また、稀代の起業家や数々の世界的企業を輩出している街として近年では多くのベンチャー企業が集まり、行政も様々なベンチャー支援策に取り組んでおります。そこで今回は、「持続的な静岡・浜松エリアへの交流人口の拡大」の実現をはかり、企業が悩む”人材確保”や”働き方改革”における浜松市の取組等をご紹介するとともに、「革新的な人材の確保と育成戦略」を切り口に、浜松市の鈴木康友市長を始め、大手企業人事統括責任者や近年注目を集めるベンチャー起業家による講演と交流会を行います。

多くの起業家を生み出し、現在も多くの起業家が拠点を置いて活躍するイノベーション都市・浜松市のさらなる魅力を知っていただくためのイベントです。皆さまのご参加を心よりお待ちしております。

【主催：浜松市】
本フォーラムの目的：
首都圏ベンチャー企業等を対象に、浜松市のベンチャー誘致への取組と魅力を発信し、進出を喚起・促進する。
