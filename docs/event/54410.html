費用

必要なもの

  • コアタイムの前後にフリーハンズオンタイムがあります。つくりたいものがあったらお持ちください。

プログラム

12:00-13:00 パート1 ハッカソンタイム1

  • 有志が早目に集まって何か作るタイム

13:00- パート2　コアセミナー

13:30-14:15　メインセッション 「Raspberry Pi B+ × 62783　で楽しいワークショップ」（内容は変更の場合があります）

何でもできそうな Raspberry Pi。でも3.3V I/Oで電流も貧弱な Raspberry Pi。 これにトランジスタアレイTD62783APG を組み合わせると15V大電流駆動ができるようになり、リレーもダイオードもモーターも簡単につなげられて最強かも？

簡単に楽しいワークショップをやってみましょう。

  • 内容　
    - 赤外線リモコン
    - リレー制御
    - モーター制御
    - 音の制御

Raspberry Pi　その他機材は貸出しますが、自分のRaspberry Pi がある場合はお持ちください。
Type A/TypeB/TypeB+いずれもOKです。

14:30-15:00　LTセッション

  • 募集中！！！

15:00- パート3 ハッカソンタイム2

自分の工作タイム
しながらミーティングなど

16:30 片付け＆お掃除＆撤収＆補習
