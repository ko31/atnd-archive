この勉強会について

サーバーサイドJavascript実装のnode.jsに興味のある人が集まって､
あれこれ話をする勉強会です｡

勉強会の情報は Google+のページ にまとめていきます｡

Logoデザイン by @defstarrick

勉強会の内容

今回はwebsocket+αモジュールの Socket.IO を使ったハンズオンを行います｡
リアルタイムWeb技術の本命として注目の集まるwebsocketをnode.jsで体験しましょう｡

今回はUstreamで勉強会を中継する予定です｡
チャンネルは ong_ust です｡

Socket.IOについて

Socket.IOはwebsocketをより簡単に使えるようにしてくれるnodeモジュールです｡

参加者の想定レベル

  • node.jsを自力でインストールできる方
  • Socket.IOに触ったことがない方から熟練者まで

参加に必要なもの

  • 無線LAN接続可能なノートPC
  • node0.6.xが利用できる環境
    • nodeのインストールは nodebrew を使うと楽ちんです
  • npmコマンドが使えること
    • nodeをインストールすると一緒に入ります
  • gitコマンドが使えること
    • ubuntuだと sudo apt-get install git です
