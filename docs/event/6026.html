そもそもオペレッタってなんやねん？

  • オペレッタ（オペラッタ）とは、セリフを喋る部分が全部歌になっているという演劇のスタイル「オペラ」の、大衆向けバージョンだそうです。
  • オペラだと堅い話をやるのに対して、オペレッタは喜劇だそうです。
  • 今回は原語（ドイツ語）じゃなくて日本語でやるそうなので、話を知らなくても大丈夫らしいです。
  • オペラだから「ブラボー」って言っていいらしいですよ今回は！

  • 18:30開場、19:00開演

  • 終わった後、駄目出し飲み会でもしましょうか！
  • 大貫アイ氏はWebブラウザのOperaユーザですが、そのOperaとは全く関係ありませんので、あしからずご了承下さい。行ってもOpera関係者は多分いません。
