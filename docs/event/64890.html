description

本愛好家たちが一日本屋になりきる「古本市」です。小説、随筆、写真集、画集、絵本、装丁が美しい本・・・もちろん絵本も！ 出店者がそれぞれ持ち寄る「美しい本コーナー」「絶対売れない（と思う）本コーナー」を設けるなど企画本コーナーも気になります。毎年好評をいただいている工作コーナーでは、海に棲んでいる生き物をみんなで描いて、長い絵巻もの（海版、鳥獣戯画！）を作ります。そのほかにも、楽しい企画盛りだくさんです。青空アート市初日と同じ開催日。アート市で楽しんだ後は、本の山がつくる独特の時間と、家主のつくるお菓子やコーヒーでほっこりなごんでいってください。
開催場所は目と鼻の先が一色海岸という一軒屋「ミユキハウス」です。
―世界で一番海に近い古本市へ遊びにいらっしゃいませんか？
