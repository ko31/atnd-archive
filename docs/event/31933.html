CloudStackユーザ会(CloudStack勉強会) – 第9回

オープンソースのIaaS基盤ソフトウェア【CloudStack】は、
リッチなGUIと強力なネットワーク機能を備え、多数のクラウドサービス事業者で
採用されているエンタープライズ向けの基盤ソフトウェアと言えます。
今年度初めには Apache 2.0ライセンスとなりApache Software Foundation(ASF)に
寄贈され、Incubatorプロジェクトとして活発に開発がされています。
今後クラウド事業者基盤や企業内プライベートクラウドとして更に広く利用される事が期待されます。

第9回のCloudStackユーザ会では、CloudStack全体枠、CloudStack活用方法枠、CloudStack連携枠と
3つの枠を設け、Apacheプロジェクトでの取り組みからクラウドサービス事業者の裏側、
将来的なエコシステムの展望までを幅広く発表頂く予定です。

是非この機会にCloudStackについて広く学びましょう！
