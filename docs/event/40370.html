description

　

幸せの祭典、『東』へ再臨！

ついに5度目の聖誕祭！ 時は来た！

秋葉原の公園で、パッションフルーツで乾杯。

“聖誕祭”の歴史は2009年7月、たった5人のささやかなセレモニーからスタートしました。

それから回を重ねるごとに参加者が増え、前回2012年には京都「VOX Hall」で参加者78名で大々的に開催。

5度目となる今回は、場所を再び「東」に移して開催いたします。

埼玉県鴻巣市内に、“聖誕祭”にふさわしい場所を一旦確保することができましたが、諸般の事情により変更。

今回は東の最速上映アフターオフの会場を使い、聖誕祭を開催いたします！

　

さらに今回は前日に関西で聖誕祭3～魂の青い鼓動～

が連動開催されます！こちらも合わせてご検討ください

開場（12:30～13:00）

大泉学園駅北口から歩いて3分の場所にありますので、直接会場へお越し下さい。
カラオケ受付には「100人部屋の参加者です」と伝えれば大丈夫です。

第1部　トークタイム『フレッシュプリキュア！の話をしよう！』（13:05～13:50）

フレッシュプリキュア！を見始めたばかりの方も、何回も見ている方も、この機会に話してみませんか？

このシーンが好き！このシーンにちょっと思うところが…

話すテーマはなんでもかまいません。

［MC］TOM

第2部 『プリソンカラオケタイム』（14:10～15:00）

いろんなオフでおなじみのプリカラタイムを聖誕祭に初めて導入！
聖誕祭を作り上げるのはみなさんです！

［MC］なし

第3部 『キュアパッション聖誕祭5』（15:00～18:00）

どこにいても、どんな時でもただ真面目に、精一杯がんばる少女「せつな」の5度目の聖誕をせつなとラブの軌跡を振り返りつつ祝います。
そして聖誕後にはパッションフルーツを手に乾杯します（数量限定となります）。

［協力］わいえぬ
