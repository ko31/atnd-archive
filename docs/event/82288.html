OS自作をやっている皆さんで集まりましょう！

このイベントの対象は次のような方々です。

  • 自作OSに興味があり、OS関係の本を読んでいる
    • 「30日でできる！OS自作入門」を含め、OSの本ならなんでも可
    • 今日から読み始めてもOKです。
  • 既に自分のOSを作り始めている
  • 自作OS入門者にいろいろ教えたい上級者

※Linux、BSD等や組み込み系OSを触るのでもOKです！

当日はOS関係の本を読んだり、自分のOSを作ったり、分からない部分を他の人に質問したり、OSに関する事なら何をやってもOKです！

強制ではありませんが、最後に各自の当日の活動内容をLTで報告する時間を設けます。奮ってご参加ください。

特別ゲスト

  • 浜松 OS 自作友の会 から Langur さん。 浜松の自作 OS コミュニティはどんな感じで活動されているのか、気になりますね(^^♪
  • 『12ステップで作る組込みOS自作入門』著者 坂井さん。 皆さんの中にも、本を持っている方は多いのでは？
  • 『30日でできる！OS自作入門』著者 川合さん。 当もくもく会で最も読まれる本の著者です。いろいろ質問しちゃいましょう！

注意

7F から 27F につながるエレベータの入り口がロックされています。送付されたバーコードを使って入館証を取得し、ゲートで読み込ませてください。 遅れる方は Twitter @uchan_nos まで事前にご連絡くださると幸いです。
