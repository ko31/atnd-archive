catch 簡単にしかも短時間に界面活性剤を分析する手法を解説！未知の界面活性剤の混合物を簡単、正確に分離分析する方法を解説！
description

＜講座のポイント＞
　界面活性剤を含有する各種の製品について、迅速に簡便にしかも安価かつ正確に界面活性剤をはじめとした配合成分を分析する手法を解説いたします。講演では台所用洗剤、精密電子部品洗浄剤、業務用リネン洗浄剤、金属洗浄剤、繊維用油剤、印刷用インキ、抗菌スプレー、皮膚用化粧品など界面活性剤を含有する製品の具体的な分析手法を紹介することにより、実際的な分析法を理解していただきます。特に複数の界面活性剤の混合物からの分離、定量手法や分離された界面活性剤の構造解析手法についても解説します。効率的に正確に界面活性剤を分析する手法であり、すぐにでも役立つノウハウを多数解説したく思います。

＜受講対象者レベル＞
　初心者から上級者まで対象。界面活性剤に関わる分析を行っておられる全ての研究者、技術者に対し効率的でかつ実用的な分析手法を紹介します。
