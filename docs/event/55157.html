description

＜概要＞
自宅で出来る輸入・国内転売のやり方・利益の出し方をゼロからあなたに教えます。
あなたが本気で取り組みたいなら必見！成績上位で豪華特典あります。

＜内容＞
突然ですが、
「１００円ショップ節約生活はもうイヤ」「少しでも生活費が多く欲しい」
「まだ子供が小さくて外に働きに行くことができない」
「もっと働きたいのに保育園の時間が決まっていて収入が決まってしまう」
とお悩みではないですか？

実は私も数年前まで仕事をいくらしても、お金が貯まらず、
これ以上稼ぐには時間がないという状況でした。

今回は、そんな私が「在宅で利益１００万円／月」を達成した
輸入・国内転売のやり方を【集中講座】であなたに教えます。

これが出来れば、
　● もちろん、収入が増えます
　● 在宅の空き時間で稼げるようになり、家事や子供との時間を多く取れるようになります
　● １００円ショップ節約から卒業できます
　● 流行に敏感になるので、自分自身のオシャレに役立ちます

では実際に輸入・国内転売とはどのような事を行うのか、
簡単に流れをご紹介しようと思います。
（講座内で詳しくご説明するので、ここでは簡単に）

【輸入・国内転売ビジネスの流れ】
販売方法もわかってしまえば難しい方法ではありません。
とても簡単です。
＜安く仕入れる　→　利益をのせて売る＞
たったこれだけのことです。

基本的にビジネスはこの原理で成り立っています。
飲食店の原価は20～30％というのはとても有名なお話です。
しかし、みなさん飲食店へは行きますよね？
自分で作れば1/3ぐらいで食べられる物をわざわざ食べに行きます。

何故でしょう？？？
（答えは「集中講座」でお話しします）

このようなビジネスの基本も学べるので、アルバイト・パートしかしたことがない方なども
しっかりと知識がつき、わかってくればこんなに単純なんだ！！と思います。

輸入や国内転売で一番利益に直結する重要な【利益のでる商品の探し方】を徹底的に説明します。
これにより、どうすればリスクが少なく利益を出し続けていけるのかをご理解いただけます。
集中講座では、実際に商品を探す時間を設けています。
あなたも【利益のでる商品】を見つかられるので、ご安心ください。

【講師はどんな人】
あなたは今、こう思ってはいませんか？
「よくある怪しい話しっぽい」「うさんくさい」
「物を売るのって大変じゃないの？」
「輸入？？？英語がわからないからできなさそう。。。」
「まずは何をしたらいいの？資金は必要？？」

最初は誰もがわからないことばかりで不安ですが、
私も最初はあなたと同じ状況でした。

居酒屋でアルバイトをしながら、コツコツ貯金をして、
切り詰めて生活をしていた時代も正直ありました。
そんな私にも出来て、さらにそんな私があなたと顔を会わせて
きちんとサポートするのでご安心ください。

私は知識ゼロから色々な失敗を経験しながら3年間で、
月商７００万、月の利益１００万を超えるまでになりました。
やった事は、＜安く仕入れる　→　利益をのせて売る＞だけです。

【本当に私にも出来るの？】
そうはいっても、あなたもやった事がない中でチャレンジするのは不安ですよね。
今まで私が教えてきた皆さんもそうでした。
ここでは過去の生徒さん達の声をご紹介したいと思います。

■ Aさん ３２歳 専業主婦 子供３歳
全くやったことのない私が参加したらみなさんに迷惑かけるんじゃないかと思ってました。
でも丁寧に教えてくれたので、私でも身につけられました。
私は子育て中で昼間しか時間がないので、働く事は諦めて節約に徹していましたが、
物流を始めて半年で、昼間３～４時間で月２５万位の利益がやっと出るようになりました！
本当にありがとう御座いました。
～～～～～～～～～～～～～～～～～～～～～～～

■ Bさん ２８歳 内職 主婦 子育て中
旦那の収入が少なく、１個１円の内職のアルバイトをしていました。
内職では月に３万円しか稼げなく、将来の不安で押しつぶされそうでした。
＜講師名＞さんに丁寧に何度も教えてくれたので、
今では安定的に売れる商品をいくつか見つけられ、
月に１０万円以上の利益がでるようになりました。
思い切って参加して本当に良かったです。
～～～～～～～～～～～～～～～～～～～～～～～

【豪華特典】
＜安く仕入れる　→　利益をのせて売る＞ので、商品を買うお金は必要になってきます。

「小額でもリスクを負うのは怖い」「そんなお金ないよ！」「やっぱりか！」
という方の為に、豪華特典を用意しました。

私たちは業務拡大の為、いつも「やる気」があり「真剣に取り組める」方を探しています。
ですので、講座の中で特にそちらに該当すれば、
私たちがアルバイト費をお支払いし、作業を手伝ってもらいながら、
あなたの成長をサポートしていこうと思います！

かけられる時間ではありません。時間は平日昼間の４時間程度でいいのです。
重要なのは、「やる気」と「真剣さ」です。

これはリスクがまったくありません。
ノウハウを学んだ後で、独立するもよし、私たちと続けるもよし、
のあなたにとって最高の提案なはずです。

【保証】
さらに、そのような目的でやっておりますので、
受講されて「全然役に立たない」と思われた方は、
おっしゃって頂ければ、全額返金いたします。

これは私たちの自信です。

【最後に】
ここまで読んで頂いたあなたは、かなり特殊なニーズがあるようです。
普通なら、ここまで読みません（笑）あなただから読めたのです。
（そうなる様に書いています）

そんなあなたは私たちとマッチする可能性が非常に高いです。
あなたには私たちも是非教えたいですし、生活を変えてほしいと思います。

輸入・国内転売は、＜安く仕入れる　→　利益をのせて売る＞だけです。
これは【利益のでる商品】が探す事ができれば、誰でも可能なのです。
センスは特に必要ありません。あなたのやる気と努力のみです。

私も最初は何も知りませんでしたし、何も出来ませんでした。
そんな私がサポートするので、あなたもご安心ください。

少ない自由な時間を全てパートに使って、
4～5万円のお給料をもらうしか方法がない、と思ってなら人生観が変わると思います。

このビジネスで人生が変わっている方がとても多くいます。
あなたも私たちと共に、あなたの描くステキな人生に変えて行きましょう。
