第01回 Fusion-io ってどんだけスゴイの?

今回は、 ストレージ友の会 は今後どのように進めていくのか、どのような目的なのか、どうしてこんな事を始めたのか、そのような事柄について、みなさまにご説明させていただくことを主としております。
そして、これからともに活動していく皆様と、懇親の場をもたせていただくことで、ご挨拶などさせていただければと考えております。

また、みなさまの大好きな Fusion-io について、実際のフュージョンアイオー株式会社の方をお招きして、製品についてご説明いただく予定となっております。

是非、この機会にご参集くださいますようお願い申し上げます。

なお、こちらは「 一般枠 」となっております。
当日はニフティさんへ名刺を提示することになります。必ず、名刺に Twitter ID を付記してお持ちください。
