乃木坂Perl Users とは

"IT勉強会"と呼ばれる類のイベントです。主に以下のような方をターゲットにしています。

1. Perlについて興味がある方
2. Perlを使っている方
3. Perlコミュニティに貢献したい/している方
4. Perlに関連して一言物申したい方

参加される方、blogやtwitter等にレポートを書いてくださると、運営チーム一同喜びます。

なお、今回は株式会社Eq様に会場を提供していただきました。この場を借りてお礼申し上げます。
