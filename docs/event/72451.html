description

「何をするか」「どこで働くか」よりも、
「誰と働くか」が大事。そんなあなたへ。

「お会いした社員が魅力的だったので決めました」。
実はそんな声が中途入社理由で一番多いのが当社の特徴です。
リクルートキャリアでは、MUSTよりも、CANよりも、 働く人のWILLを何よりも大切にしています。

だから、社内では「こうしたい」という意志を持った人が溢れている。
それを応援するために、手を差し伸べて、 気付きを与えてくれる人も溢れている。
共に成長できる仲間がほしいなら、 これ以上の場所はないと自負しています。

そんな社風や社員の魅力を知っていただけるような場として「MEET’S」をご用意いたしました。
場所は、本社最上階にある社員食堂「空箱」。 夜はお酒も飲めるレストランになります。

参加者は、異業界からの中途入社者3名と、配属部署のリーダー2名を予定。
選考とは関係ありませんので、転職理由や、それぞれが成し遂げたいWILLなど、
ぜひ自由に質問してみて下さい。

１つでも当てはまる方は是非ご参加ください！

・「リクルートキャリアの文化やひとにちょっと興味がある…！」という方
・営業力を身に着けたい！と思っていらっしゃる方
・いろんな人と語ることで、キャリアに対するもやもやとした不安を棚卸したい方
・最近目の前の仕事に手いっぱいだけれども、本当は青臭い会話がしたい方
