スキー行こうよ、スキー。
参加者は現在CEOのOZAKI、及びAdvertising Managerのマツダです。
オザキ・ホールディングスの社員の友達の友達くらいなら誰でも参加OKなんで、一緒に雪山をエンジョイしましょう。

まだスキー旅行のパンフレットが無いので詳しくは未定。参加者の希望をできるだけ取り入れます。
以下予定。

日時：２月中。参加者の希望で調整するけど、一応候補⇓。値段的に平日がいいな。
①2月10日の深夜～13日早朝。②2月17日の深夜～20日早朝。③2月24日の深夜～27日早朝。
スケジュール：１泊４日（１日目の夜関西出発、４日目の早朝関西着、すべるのは２日間）
場所：信州のどこか。白馬希望。
参加締切：参加の意思は早めに出して欲しいけど、最終決断は年内でもOK。
参加方法：オザキorマツダに伝えること。facebook推奨。
参加したいな～と思う人はできるだけ早めに、「参加したい」「無理な日程」「その他コメント」を書いてください。

例：
参加します！でもまっちゃんと２人だけならちょっと…
13日、25日は無理。木曜は研究会だけどサボればOK。
去年志賀高原行ったから、今年は白馬あたりがいいな。
