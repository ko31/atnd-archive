「概要」

対象：


    • プログラミング初心者、未経験者
    • Pythonを使ってリコメンドシステムをつくってみたい人。
    • 人工知能について興味のある人。
    • たまたま土曜日に時間の空いている学生さん。


＊レコメンドシステムとは？



    なにかを購入したお客さんにさらに他の商品を買ってもらうために、「〜〜という製品があなたにオススメですよ」といった形でレコメンド(お勧め)していくシステムです。有名な例を挙げると、アマゾンでお買い物をするとの「おすすめ商品」がアマゾンのサイトのバーナーに出てきたりします。

この勉強会のゴールその1：

クロス集計の理解をふかめる。(未経験者にもわかるように説明します。)

この勉強会のゴールその2：


下記のようなPythonのごくごく基本的な構文をつかえるようになる。
If
For
Input
List

Print
内包表記
dict型　items()

勉強会の内容：

勉強会は以下の４ステップを予定しています。


    • ①Pythonの言語に関する基礎知識のインプット。

    • ②練習として、基本的な構文をアウトプット。

    • ③応用として、以下のような演習をする。

      • 　→【演習1】評価システムのAPIを実際に眺めてみて、どのようなシステムになるのか、考えてみる。
      　→【演習2】参加者で協力して、ひとつの評価システムを作ってみる。
    • ④オープンソースの紹介
