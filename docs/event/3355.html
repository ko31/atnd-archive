Twitterで知り合った合唱人が「せっかくだし集まって歌ってみない？」という趣旨の元、何となく集まって何となく歌ってみようという会です。

・まだ知り合っていない人、Twitterをやっていない人も歓迎です

・途中参加、途中離脱OK

・参加人数によっては開催されない場合があります

・何の曲を歌うかは未定です (「木下牧子 アカペラ・コーラス・セレクション」を予定？)

・高いレベルを目指していません。楽しく歌いたいだけです
　レベルの高くない演奏に満足できない方はご遠慮下さい (案外レベルの高い合唱になる可能性もありますが)

・取りまとめ人は下手の横好きです。多くを期待しないで下さい

・ご意見、ご質問はコメント欄、または取りまとめ人の「ひこ@紫のバラの人似 (https://twitter.com/hikos )」まで

・参加方法がどうしても分からない方は、取りまとめ人まで@を飛ばすか、ここにコメントをして下さい。可能な限り対応します。
