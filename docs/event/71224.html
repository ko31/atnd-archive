description

空の産業革命ともいわれるドローンの出現により、一般ホビー層への普及も著しく、法整備の話題や世界各地のニュースが連日報道されています。一方、商用向けの活用サービスも広がり始め、国内外のメーカーが次々に新しい機種を開発するなど、ものづくりに携わる中小企業にもチャンスが見込まれます。

本イベントでは、ドローンを取り巻く最新情報、国内外の動き、最新技術、業界の動きなど、関西から情報発信の場を設けることで、安全なドローン活用の普及と、更なる発展を後押しすることを狙いとしています。
第一回目となる「関西ドローン会議2015」では、業界を取り巻く最新情報のほか、自然災害大国日本の災害対策をテーマに、火山噴火や降雨による土砂災害時のドローンの活用について掘り下げた内容を予定しています。
