あえて20代を会社で過ごさない理由

イベント ユーザー
event_id 28364
title あえて20代を会社で過ごさない理由
catch 20代の皆さん、会社以外の時間を大事にできていますか？
description

私達は今、企業に属していません。

それは、会社でサラリーを得て、
親元を離れ、生活基盤を確立しただけで、自立したと誤った錯覚を持ちたくないからです。

もし
20代という貴重な時間帯を自分のためではなく、
自分以外の何かのために一生懸命費やしているとしたら、
後に残るものは何でしょう 。

私達は、こう考えます。
20代の時間帯こそ、苦労していい。
あえて困難を選択し、確かな自己を確立すべきだ。

だから、私達は学ぶことを選択しました。
成人した男女だからこそ、学ばなければいけないことがあります。
何のために？自分のためです。

・挨拶をしていますか？
・時間を守れていますか？
・人の話を心から聞いていますか？

当たり前のことが、当たり前に出来ない。

それぐらい、あいまいな部分があるのではないですか。
それを認めて、一つ一つ、「出来る大人」になることが大切だと、私達は考えています！

では、どうしたらいいでしょうか？

私達には、 自分を悪くする要因 がいくつかあります。
例えば、偏見、先入観、自負心、嫉妬心、猜疑心、ねたみ、そねみ。。。
誰でも少なからず持っていますよね。

あなたは、 どのようにこれらを克服しますか？

私達は、これらを克服することに挑戦しています。
20代の皆さんとともに、意見を交換し、
その解決策について考えたいと思います！

自分の納得する生き方をしましょう！
あなたは、20代の貴重な時間をどのように使いますか？

本当に好きなことをしたい人集まれ！

【プログラム】
・20代のうちに挑戦すべき5つの壁
・「会社を辞めても良い」と決断できる5つの要因
・会社では学ばない5つの要素
なお、当日は、プログラム中に
本当にしたいことをするために挑戦している
5人の若者が登場します！
