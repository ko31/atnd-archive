description

ソーシャルをキーワードに、多くのSNSがすでに利用されています。今や企業や地方自治体も、利用しているところはいっぱい！

今回は、自由度の高い拡張性を獲得したOpenPNE 3について、メイン開発会社である株式会社手嶋屋の代表取締役社長手嶋様に来阪して頂き、講演の運びとなりました。

また、導入利用実績でSNS界日本一のOpenPNE 2では、2009年のITメディア賞を受賞した語学学習相互添削SNSの運営会社ランゲート株式会社様や、団塊の世代の大阪人のポータルサイトSNSの運営組織NPO法人おおさかシニアネット様といった、運営者様たちの講演も用意しております。

さらにディスカッションタイムでは、デジタルハリウッドのインターネットマーケティングの教授、塩見先生にも参加して頂き、講演者と会場での質疑応答をおこないます。
