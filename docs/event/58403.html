概要

webにおける新しいデザインを考える取り組みの一環として、UI作成ツール「Framin」でのアプリ作成を体験いただくイベントを開催いたします。

イベント内容

昨今のweb技術の進化は，webとデザインに新たな関係をもたらす可能性があります．
この可能性に注目し，9/20開催の「 デザイナー視点で考えるWeb 時代のアプリケーション (Mozilla Japan様主催)」で新しいWeb ビジュアルデザインについて考えてきました．
その取組みとして，プログラムに慣れていない方でもWeb ビジュアルデザインを表現することができるアプリケーション「Framin」を開発しています．(「Framin」概要[p.6]は こちら )

今回は「Framin」を使ってアプリ作成を体験いただき，アプリケーションアイデアやビジュアルの表現を一緒に考えていきます．イベント冒頭では，INFOBARをはじめとしたauデザインモデルを手がけた砂原 哲が作ったアプリをご紹介し、楽しむコツをお伝えします．

難しいプログラミングの知識は必要ありません．みんなで楽しみましょう！

お持ちいただくとよいもの

「Framin」はお好きな画像を使って，アプリをデザインするツールです．

  • ノートパソコン等のデザイン道具、お気に入りの画像やイラストを是非お持ちください．また、microUSBケーブルをお持ちの方はご持参ください．
  • その場で、アイデアを試作するための紙や色鉛筆、クレヨンなどアナログな道具も各種ご用意しています（写真として取り込みます）．
