FxOSコードリーディングミートアップ#9を開催します！

祝Mozilla Japan十周年

開催日の8月18日はMozilla Japan十周年です！

何をするの？

基本的には Firefox OS のソースコードを読みます。各自読みたいところをあーだこーだ言いながら読んでいます。
ずっと誰かとしゃべっている人もいれば、黙々と読んでいる人もいます。
Firefox OS をもっと知りたい/Firefox OS について話したい人達がゆるく集まれるのが、ミートアップの持ち味です。

オープンソースのソースコードを読んだ事が無い人や、初めての人も大歓迎。
分からないことも話してみれば、誰かが教えてくれるかもしれません。

何が要るの？

Firefox OS への興味とノート PC だけで OK です。
ちょっとした調べ事なら FxOS コードリーディングで運営している OpenGrok が使えます。

OpenGrok

※ GMO インターネット様よりコミュニティ支援として ConoHa VPS の提供を受けています。

FxOSコードリーディングって何？

Firefox OS のソースコードを読みたい人のコミュニティです。
iOS, Android に続く第３のモバイル OS の一つ Firefox OS。そんな Firefox OS をもっと深く知りたい。
そんな思いの人達がソースコードを読んで意見交換しあうコミュニティです。

読んでみたいけどどうしていいか分からない。
読んでみたけど分からないことを聞いてみたい。
読んでみたから誰かに話したい。
・・・そんな人達をつなぐ場です。この機に一緒に読んでみませんか？
