description

　TRPG黎明期から様々に活動されてきた伏見健二先生の関わったTRPGシステムを遊ぼうというイベントです。
　タイトルの「ブルブルコン」は初期の「ブルーフォレスト物語」から一昨年の「ブルーシンガーＲＰＧ」までを遊ぼう！　という意味です。

●日時：2018年５月19日（土）10時開場～19時閉会
●メイン会場：埼玉建産連研修センター ２階の201会議室（部屋が変更になりました）
　JR埼京線、武蔵野線の武蔵浦和駅約10分
　自動車で来場希望の方はご相談ください。駐車場のスペースには上限がございます。

　５月17日23時現在、本ページ以外からＰＬ18名予約いただいております。
　
●会費：冒険者コース（一般）1000円
　　　　魔法使いの弟子コース（未成年）500円
　　　　学生コース（学生）100円
　　　　勇者コース（応援参加）3000円
　　　　語り手コース（ＧＭ、スタッフ）500円、ゲスト無料
　オプションとして、
　　・お昼券（おいしい欧風カレー弁当）500円
　　・どの卓でもＯＫ割：希望する卓の人気が高いときに別の卓に移ることを了承しておくことで200円引き（冒険者コース、魔法使いの弟子コースと併用できます）

　「勇者コース（応援参加）」には、卓分け優先権＋「ウーパールーパー」または「ラビットホール・ドロップス」のうさぴょんサイコロ1500円分）がついてきます。
　短時間卓の前半・後半の片方だけに参加でも参加費はかわりません。片方だけの参加の場合はあらかじめ連絡いただけると助かります。
　「学生コース」では、当日受付時に、学生だという証をお見せください。

◆参加区分（一般、未成年、応援）について、コメント欄かTwitterやメールで当日日までにご連絡ください。

◆お昼券をお求めの場合、コメント欄か、Twitterかメールにて早めにご連絡ください。日によっては間に合わないことがあります。

　定員は、ほかの申込方法からの申込状況で増減します。

　タイムスケジュールなどはイベント関連ページ http://www.tamasuna.jp/blue/ でご確認ください。
