「やりがいのある仕事をしたい」
「勤務時間や勤務形態にとらわれずに働きたい」
「自分のやりたいことで稼ぎたい！！」

そんなふうに思っていらっしゃる方はぜひ社会起業大学の無料セミナーにご参加ください。

社会起業大学は、一人ひとりの想いや強みを最大限に引き出し、
それを社会の役に立てていくことで、持続可能なビジネスを作り上げる方法を学ぶことを通じて、
起業家・社会起業家を育成するビジネススクールです。

無料セミナーでは、豊富な事例やディスカッション・グループワークをふんだんに盛り込み、
自分の中にある「想い」、本当の「強み」を磨きながら、真の社会課題を深堀し、
最も自分の価値を発揮できる稼げるポイント見出す方法をお伝えします。

当日は、

・やりたいことを実現している起業家・社会起業家の人たちはどのようにビジネスを作り上げていったかを知る
・自分の強みや想いと社会課題解決がつながる、自分なりのビジネスを考えてみる
・起業家・社会起業家の卵のビジネスプランプレゼンテーションを聞いて、良い点、課題点をフィードバックする

といったセッションをご用意しております。
