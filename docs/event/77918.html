セミナー概要

お隣の国といえども、中国の通信事情は日本のそれとは大きく異なっており、世界一の人口を擁して購買力がぐんぐん上がっている魅力的なマーケットを求めて中国への進出、中国での事業の拡大を進める日系企業のIT担当者を苦しめています。 本セミナーでは、中国の通信事情だけではなく最新のビジネス環境をご紹介するとともに、最近爆発的な需要増を呈しているエンタープライズ向けクラウド市場の動きについて解説します。さらに、IIJが中国で開発した新サービス「IIJ GIO CHINAサービス」の開発秘話をご紹介します。

タイムテーブル

18:30～19:00 30分 受付開始

飯田橋グラン・ブルームのオフィス入口より入館後、2階IIJ総合受付へお越しください。スタッフが待機しております。スタッフが不在の場合は、080-8040-9448までご連絡ください。
19:00～19:30 30分 企業をとりまく中国の通信事情

講師：IIJ　グローバル事業本部 グローバル企画部 サービスソリューション課　植木 敏文

「グレートファイアウォール」や「通信の南北問題」など、中国には想像すらできない通信事情やお国事情があります。ライセンスや政府の方針転換など変化が大きいこともビジネスにとってはリスクになります。本講演では、中国特有の通信事情を中心に日本企業をとりまくビジネス環境も解説します。
19:30～19:55 25分 爆雲が始まる！！中国クラウド市場を徹底解説

講師：IIJ　グローバル事業本部 グローバル企画部 企画課　袁 航

中国でクラウドが盛り上がっています。中国産最大手"アリ雲"が君臨する中、AWS、SoftLayer、Azureが相次ぎ進出し、現代の"戦国時代"とも言えます。本講演では、中国クラウド市場の全貌を解明した上で、エンタープライズ企業が求めるクラウドサービスについて解説します。
19:55～20:00 5分 Break
20:00～20:30 30分 中国仮想化プラットフォームサービス開発秘話

講師：IIJ　サービス基盤本部 システム技術部 グローバルサービス基盤課　久保田 悠生

IIJが中国のクラウド市場に切り込むために開発した新サービスについて、構築エンジニア自らが紹介します。新たなビジネスモデルを追及し、わずか7ヶ月 で誕生したサービスの開発途中では様々な難題が発生。現地事情も交えてご説明 します。
20:30～21:00 30分 質疑応答

講師紹介

    株式会社インターネットイニシアティブ

    • IT業界でシステム運用、ソフトウエア開発分野に携わる。海外ではドイツで3年間現地日系企業のIT環境構築を経験。帰国後はDC等のインフラ構築に携わり、IIJでは主にGIOのグローバル展開でサービス運営、企画、プロモーションを担当。
  • 袁 航
    株式会社インターネットイニシアティブ
    グローバル事業本部　グローバル企画部企画課

    • 中国上海生まれ、日本在住11年目。明治大学大学院国際マーケテイング専攻を修了し、大手日系SI会社勤務を経て2012年にIIJ入社。現在、海外クラウド事業のマーケテイング企画を担当しています。
  • 久保田 悠生
    株式会社インターネットイニシアティブ
    サービス基盤本部 システム技術部 グローバルサービス基盤課

    • IIJに新卒入社後、約4年間IIJ GIOサービスインフラのストレージに関わる設計/構築から運用までを担当。その後、海外GIOサービスのストレージインフラ構築、業務設計/構築、運用などを担当。最近は業務効率化といった運用改善に注力。
