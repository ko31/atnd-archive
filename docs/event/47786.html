おしながき

去年に話題となったImmutable Infrastructureやその周辺ツール、具体的な適用例についていろいろなトークをしましょう。キーワードとしては、

  • Immutable Infrastructure
  • Blue-Green Deployment
  • Docker
  • Serf
  • Apache Mesos
  • Dokku
  • Flynn

などを想定してます。これらのツール/仕組みはまだまだ開発中でこなれていないものが多いですが、知見を持ちよって次世代のDevOpsを考えていきましょう! 開発が盛んなDockerの便利な使い方なんかもいろいろ掘り下げてみたいですね

なにかご要望、質問などありましたら、 @stanaka までどうぞ!
