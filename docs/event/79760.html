事例でわかる！プロジェクト管理ツール選択のポイント

～「Seavus Project Viewer」、「Microsoft Project」、「Redmine」の比較～(参加特典あり)

「Redmine」、「Microsoft Project」、「Project Server」の事例を通じ、プロジェクト管理ツールの活用方法をご紹介するセミナーを開催します。

いろいろなプロジェクト管理ツールを調べても具体的にどう違うのか、自社に合ったツールは何かを判断するのは至難の業です。長年多くの企業のプロジェクト管理に携わってきた実績と経験をもとに、プロジェクト管理ツールとして多くの企業に選ばれているこれらのツールを分かりやすく比較します。

高い品質を誇る「Seavus Project Viewer」で予算 VS 生産性の悩みを解決

さらに今回は、高い品質の「Microsoft Project」のビューアーを開発・販売している スウェーデンのSeavus 社からシニアコンサルタントのボヤン・タセヴスキ(Bojan Tasevski)氏を招き、まだあまり日本で知られていない「Seavus Project Viewer(以下、SPV)」を紹介いたします。
「SPV」が目指すのは、これまで予算の制限で「Microsoft Project」を購入できなかったメンバーへの活用です。予算と生産性の狭間で悩んでいた企業への新たな解決策として「Microsoft Project」と「SPV」の相乗効果を提案します。デモも行いますので実際の画面をその目でお確かめください。また、「Primavera P6」 のビューアーである「Seavus Primavera Reader」もご紹介します。

経験豊富な講師陣

講師陣には「Project Onlineではじめる最強最速のプロジェクトマネジメント入門」(日経BP社)共著者の、佐山 宇宏、大石 守、吉村 隆男が顔を揃え、経験豊かなエキスパートに質問がしやすい30人限定の少人数制のセミナーとなっています。

こんな方におススメします。

プロジェクト管理ツールの導入を検討中の方、現在のプロジェクト管理ツールを更に活用したいと思っている方、「Microsoft Project」をご利用中で「SPV」をご存知なかった方におすすめの内容です。もちろん「SPV」ご利用中の方にも新たな機能のご紹介があるのでおすすめです。
