講演概要

JavaOne 2012 サンフランシスコ・最新アップデート

  • 出演
    • 日本Oracle株式会社　シニアJavaエバンジェリスト　寺田佳央氏( @yoshioterada )
  • 概要
    • 2012 年 4 月 、日本で７年ぶりとなる JavaOne が開催され Java が今後も非常に重要であることが広く認識されました。
      本セッションでは米国のJavaOne 2012 で発表された Java の最新トレンドについて分かりやすく紹介します。
      本セッションを受講いただくことで今後の Java の方向性についての理解を深めていただくことが可能です。

プロジェクト・ラムダ・ライジング

  • 出演
    • 天領倉敷Scala/岡山Javaユーザ会　角田裕樹氏( @razon )
  • 概要
    • Java8 SEの目玉機能といえばやはり、Project Lambdaです。
      本セッションではラムダ式とは何か、そしてProject Lambdaとはどのようなものなのか解説します。
      Java8のリリースに先駆けて、実際ベンリなProject Lambdaをマスターしましょう。
      Wasshoi!
