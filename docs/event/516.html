description


今話題のパケット通信。ＡＰＲＳに関する勉強会を開催します。
正確な位置情報というのは昨今の事件・事故で役立っています。

県内各所へ主要設備を設けネットワークを確立する事で、
電波上の情報流通を適切かつ有効に使う事が出来ます。
ＪＡＲＬ広島県支部では災害時活動への有用性を検証し、
ＡＰＲＳシステムを活用した防災・救助活動を推進して参ります。

広島をモデルケースとしてアマチュア無線が社会に貢献できる
活動をする為に必要な知識と設備を共に勉強しましょう。
