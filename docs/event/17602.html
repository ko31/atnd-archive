CloudStackユーザ会 + RightScaleユーザ会共催 第1回 大阪勉強会

Eucalyptus, OpenNebula, OpenStackなどオープンソースのクラウドインフラ基盤も
多数あります。その中でも Cloud.com社が開発・公開している【CloudStack】は
他のオープンソースクラウドにはない、リッチなGUIとFW, LB, HA等の機能を備え
多数のHyperVisorをサポートするなど、ひときわ異彩をはなっています。
また国内でも大手データセンター事業者であるIDCフロンティアで CloudStack が採用されるなど、国内での事例も登場し始めています。

併せて、Amazon Web Services(Amazon EC2/S3)やRackSpace Hostingなどの
クラウドサービス用の管理サービスとして最も広く利用され有名なのが【RightScale】です。
併しながら、国内では日本語での機能説明の資料少なく、RightScaleのSeverTemplateを
始めとする機能やメリットについてほとんど知られていません。

RightScacleにはクラウド環境を管理する協力な機能の他、CloudStack環境を管理することも
可能で、Amazonクラウド(AWS)とCloudStack環境を併用したハイブリッドクラウド環境の
構築手法はソーシャルゲーム最大手 Zynga ZCloudで採用されるなど
クラウド環境統合の最先端の手法で、注目を集めています。

今回は、Cloud.com社、RightScale社、雲屋株式会社、クリエーションライン社の協力により、CloudStackユーザ会、RightScaleユーザ会を大阪で同時開催する運びになりました。

是非この機会にCloudStack、RightScaleについて学んでみましょう！
