久々にプレゼン大会を行います

29回目のSWWDCは、プレゼン大会を実施します。

今回のプレゼンのテーマは自由です（ただ、できればiPhoneのことも絡めてください）。

SWWDCはiPhone開発者の交流を兼ねた勉強会なのですが、このところ「仕事でiPhoneアプリ作ってない」という方も多いので、iPhone関係なくてもokにしました。

というわけで、いろんなネタが出てくると思います。ぜひご期待ください。

もちろん、聴くだけの参加もokです～。ゆるい感じでやっていますのでお気軽にご参加ください。

登壇予定

  • kyasuさん 「iBeaconで博物館の音声ガイドの実例報告」
  • magickworxさん「ニュース系アプリの UI の設計と実装」
  • asmzさん 「初参加です。自己紹介がてら何か」
  • tototti 「MachineLearningで宝くじhack(仮)」

タイムテーブル

  • 13:50- 開場
  • 14:00- 開始
  • Section1 (45min)
  • 休憩 (15分)
  • Section2 (45min)
  • 休憩 (15分)
  • 雑談・交流タイム (45min)
  • まとめ・後片付けなど (15分)

発表希望の方へ

  • 発表したい方は、その旨をATNDのコメント、twitterの@totottiへのmentionなどでお知らせください。
    • ↑の際に、タイトルが決まっていれば、それも書いてください。なければざっくりとしたテーマでokです。
  • 持ち時間は15分以内(質疑応答込み)となっております。
    • もっと時間がほしい人は、「30分 or 45分ください」といっていただければ、調整します〜。
  • プレゼンのテーマは自由ですが、できればiPhoneのことも絡めてください...
    • 例：最近アプリをリリースしたので紹介します
    • 例：このフレームワーク使ったら便利なので紹介します
    • 例：ちょっとバグ取れないので手伝って下さい(プレゼン時間をデバッグに使う、高等テクニック)
