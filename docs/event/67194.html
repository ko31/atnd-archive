Percona MeetUP Tokyo 国内初のPerconaイベント開催！

MySQLのフォークとして高い信頼性と性能でユーザ数を伸ばしているOSSのDB「Percona」。ユーザ数が拡大し、事例が増えているにもかかわらず、あまりネット上に情報がないのが実状です。

今回国内で初めての、Perconaイベントを開催します！MySQLとの違い、Percona社の開発者によるTech Talk、利用事例まで盛りだくさんの内容です。
「Perconaを知らなかった！」という入門者から、「ウェブの情報は知り尽くしている」というヘビーユーザまで、オープンソースのデータベースにご興味がある方、気軽にお越しください！
