description

昨今、多くの企業が取り入れているクラウドサービス。普及率が年々伸びている今もなお、その技術や活用シーンは日々進化しています。IIJが提供するクラウドサービス「IIJ GIO」も、多くのエンジニアがテクノロジーや安定運用に尽力し、開発・運用しています。
本セミナーでは、開発や運用について等、普段お話しすることのないIIJ GIOの裏側を、サービスに関わるエンジニアからお話しします。ぜひ、ご参加ください。

プログラム

18:30～19:00 30分 受付開始

飯田橋グラン・ブルームの2階IIJ総合受付へお越しください。受付時にはATNDユーザ名の確認とお名刺を1枚頂戴いたしますので予めご準備ください。
19:00～19:55 55分 煩雑化する大量の監視アラートを整理

講師：IIJ サービス基盤本部 サービス運用システム部 M&Oサービス開発課 福原 亮

監視システムからの大量のアラートに頭を悩ませていませんか？
IIJ GIO統合運用管理サービスの、SaaS型の監視、チケット管理、ジョブ管理サービスをご紹介。 そして、これらサービスを支えるバックエンド基盤「アラートフィルター」を徹底解説します！
19:55～20:05 10分 休憩
20:05～21:00 55分 クラウドサービスの作り方

講師：IIJ クラウド本部 クラウドサービス1部　田口 景介

多くのサービスがIaaSのように複雑であり、サービスシステムの構造が詳細に語られることはめったにありません。
OpenStackなどには頼らず、数十のマイクロサービスから構成されたIaaSの基盤がどんな構造をしているのか興味はありませんか？どんなメンバーが、どんな体制で、どんな開発プロセスで成し遂げたのか、できる範囲でお話しします。また、そんな開発メンバーが自ら運用にも携わる、開発と運用の在り方にも触れたいと思います。

講師紹介

■福原 亮

サービス基盤本部 サービス運用システム部 M&Oサービス開発課 課長。システム運用設計、サポートセンター運営を経て、現在は運用管理サービスの企画・開発を担当。運用負荷削減に向けた、機能開発を推進しています。

■田口 景介

クラウド本部 クラウドサービス1部 部長。オンラインストレージサービスやIaaSなど、クラウドサービスの企画、開発、運用を担当。開発者自身が直接サービス提供、運用に携わり、ユーザーと接することがサービスを成長させる最大の原動力と考えています。
