description

現在の日本だけで資産を作る・守るということが容易だと考えられる方はこの先は不要ですので読まないでください。

日本経済が揺れ動く中、今後は日本だけでなく世界に目を向けることがとても大切な状況になりました。
特に金融資産については、世界の金融の中でも日本はかなり遅れをとっていることが顕著になってきています。

このままであなたの資産は問題ないでしょうか？

お金の流れについては情報が多すぎて特にわかりにくい現状です。そして国内のリスクを考えると海外の情報が少ないことが容易にわかります。これから海外に目をむける方は、今まで得られなかった情報も確実に目に入ってきます。

このセミナーでは不動産を中心とした資産形成についてお話します。その中でも確実に不動産が熱くなる地域に注目することがこれからのあなたの資産形成・資産構築・資産運用に大きく役立ちます。

このセミナーでは、本当に熱くなる地域

「インドネシア・バリ島の最新の不動産情報」をお届けします。


”まだ国内だけでもいいのではないのか？”
”なぜ海外不動産なのか？”
と考えていませんか？

”日本国内だけ”　”日本を含め世界も”
あなたはどちらを選びますか？

☑資産運用には他にどんな種類があるのかわからない…
☑そもそも資産形成について勉強中…

もし、この中にひとつでも当てはまる項目があるとしたら・・・
あなたがこれまで知る事ができた情報だけでは「まだ何か足りない」と感じていたら・・・

あなたにとってピッタリなセミナーとなることは間違いありません。

なぜならこのセミナーでは本気で日本を良くしよう、そして世界でも日本は全然立ち回れる！
という気持ちでビジネスを展開している方がインドネシア・バリ島の最新の不動産情報をお届けできるからです。

○浦山智宏
株式会社メサージュ　取締役
1974年6月生まれ、埼玉県出身、京都在住

バリの大富豪・丸尾孝俊（兄貴）氏が活躍するインドネシア・バリ島を拠点とした『PASTIインドネシア』の総合販社としてビジネスを展開。兄貴の片腕である宮本会長とともに１年を通してバリ島と日本と行き来している。
バリ島視察ツアーの企画、日本国内でのセミナーなど活動範囲は、香港も含めてアジア数か国で展開。
　国内では複数の会社経営および他社へのコンサルティング業務もこなすなどビジネスは多岐に携わる。
