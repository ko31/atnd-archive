Apps for Officeの利点は

1. Excel,Word,PowerPoint等の高度な関数や入力しやすいインターフェイスをそのままUIにしてデータはSharePoint, SQL, Salesforce, EXCEL等に保管できます。
2. Office365があればOffice Web Appsで動作します。
3. Office Web Apps でも ローカルoffice でも動作します。
4. Office Web Apps 経由で、Ipad, Androidでも動作します。
5. １つのコードで全部の環境で動作します。
