こんな人にオススメです！

  • AWSとフリードメインでゼロから公開サービスを作ってみたい
  • 普段はフロントエンドだが、SQLも勉強したい
  • EC2のセキュリティグループも勉強したい
    この勉強会では、ハンズオン形式で実際に開発していきます。
    お土産として、ご自身のAWSアカウントでEC2, RDSインスタンスとソースコードを持ち帰れます。
    ※ EC2, RDSは無料枠があります。

対象者

  • ゼロからのサービス公開方法を学びたい方
  • SQLの基礎を学びたい方

このハンズオンでできるようになること

  • フリードメインを取得
  • AWSでサーバー構築
  • EC2のセキュリティグループ設定
  • MySQLでデータベース構築
  • PHPの簡単なバックエンド開発
  • Nginx の設定
  • SQLの基礎

参加要件

  • 事前準備として、AWSのアカウント登録を完了している人
  • Webサービスの予備知識がある人

事前準備

1. AWS アカウント作成
2. テキストエディタ

  • サーバーにsshで接続しながらテキスト編集できるソフト: vim emacs など
  • 上記の使い方がわからない方：「Eclipse」を事前にインストールください。
  • Eclipse: Windows 32bit版
  • Eclipse: Windows 64bit版
  • Eclipse: Mac 64bit版

3. aws の pem ファイルでsshできる環境

  • windowsをお使いで、わからない方は以下を事前にインストールください。（Cygwinなどを既にインストールしている場合は不要）
  • git for windows
