あなたが本当にやりたいことで稼げるようになるためには、何が必要でしょうか？

無料セミナーでは、豊富な事例やディスカッション・グループワークをふんだんに盛り込み、
自分の中にある「想い」、本当の「強み」を磨きながら、真の社会課題を深堀し、
最も自分の価値を発揮できる稼げるポイント見出す方法をお伝えします。

当日は、

・やりたいことを実現している起業家・社会起業家の人たちは
どのようにビジネスを作り上げていったかを知る
・自分の強みや想いと社会課題解決がつながる、自分なりのビジネスを考えてみる
・起業家・社会起業家の卵のビジネスプランプレゼンテーションを聞いて、
良い点、課題点をフィードバックする

といったセッションをご用意しております。
