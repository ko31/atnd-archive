Advent Calendarとは

かつて地球には、クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーがありました。21世紀にはこれが転じて、ギークコミュニティで、クリスマスと呼ばれる祝日までの25日間に1つずつ箱に閉じ込められた魂を公開する儀式が行われたそうです。これを Advent Calendar と呼びました。

かつての様子はこのようなものでした。

  • Reblog Advent Calend*e*r 2010
  • Reblog Advent Calend*e*r 3011

かつて、それはただの typo でした。

登録方法

  1. 参加表明する人は「参加登録」してください
  2. 参加表明する人は、ユーザープロファイルかコメントかどこかに tumblr id を晒してください
  3. その tumblr id の FAN Mail に reblogadventcalender7889.tumblr.com の招待状を送付します
  4. 参加した人には reblogadventcalender7889.tumblr.com の管理者権限を付加します
    1. 管理者権限を持った人は、他の人を招待しても良し
    2. reblogadventcalender7889.tumblr.com のデザインをグチャグチャにしても良し
    3. イヤなヤツのアカウントをBANしても良し
  5. コメント欄かどこかに自分が担当する日付を宣言してください
    1. ただし 12/1から12/23までとします。早いもの勝ち
    2. 12/24,12/25は下記のルールを参照
  6. それ以外には twitter で参加表明して誰かに拾ってらうか、だれか管理者権限のある人を探して招待してもらってください

ルール

  1. 12/1から順番に、一人丸一日を担当
    1. たくさんリブログしても良し
    2. 濃い一発をかましても良し
    3. なんでも良し
  2. 当番がいなくなった時点または当番がいない日、そして12/24 0:00からはフリー
    1. つまりバトルロワイヤル
    2. 目指せ 7889post/day
  3. 12/25が終ったら Members | Tumblr にて “Leave this blog” します
  4. 最後の一人が “Leave this blog” した瞬間に reblogadventcalender7889.tumblr.com は消滅します
