電子書店で販売するためのEPUB制作実践

電書ラボでは、既存電子書店で販売できるEPUBを制作するための、
実践的なセミナーを開催します。
現在電子書店では、それぞれ異なるビュアーを提供しているので、
それら多くのビュアーでの表現を考慮する必要があるからです。

セミナーでは、
①実践的な制作方法をツールごとに紹介、
②電書ラボで開発した仕様・チェッカーなどの紹介、
③EPUB3.1などの電子書籍をめぐる状況の報告、
を4回にわたって紹介します。

出版社・制作者の方々の参加を歓迎します。

第一回講演概要

EPUBの作り方01　でんでんコンバーター

  • 講師＝高瀬拓史（電書ちゃんねる）

電書ラボ制作仕様見出しテンプレート

  • 講師＝田嶋淳（電書ラボ・三陽社）

第三回までは制作実践（1時間程度）と
電書ラボのツール紹介（30分程度）、質疑（30分程度）。
第四回は電子書籍をめぐるさまざまなトピック（各30分程度）。
