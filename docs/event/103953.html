名古屋【3月 / 受講料無料】AutoCAD・BIM（Revit）講座

～ Modis Engineer Academy ～

未経験からエンジニアへ
アデコグループのエンジニア養成講座


お申し込みはこちらからお願いします


当講座のポイント

■ 未経験OK！
 知識・経験がゼロの方でもエンジニアとして就業へ。

■ 完全無料！
 ITスクールの受講料は通常数十万円以上かかりますが、Modisの養成講座は無料です。受講時に必要なパソコンもこちらで貸出します。

■ 働きながら学べる！
 集合研修は毎週土曜日。翌週まで復習の時間があるため講座内容をより深く理解できます。

■ 就業をサポート！
 研修だけでなく、キャリア相談や仕事紹介・就業後フォローまでサポートします。


Modis Engineer Academyは未経験の方でも熱い想いがあれば、 年齢問わずチャレンジできるプログラムです。
Modisは「キャリア開発があたりまえの世の中をつくる」というビジョンのもと、「未来に繋がる仕事」を創出し、皆さんの「キャリア開発したい」という想いに寄り添って積極的に支援していきます。

カリキュラム

建築知識座学＋e-Learning

AutoCAD座学＋実機演習

BIM（Revit）座学+実機演習
