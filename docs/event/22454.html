description

「アジャイルサムライ ― 達人開発者への道」 の読書会を行います。
順番に輪読していき、セクションごとに本の内容についてディスカッションする形式ですすめていきます。
読書会とディスカッションを通じて、モヤモヤ感が生まれることもあると思いますが。現場でアジャイルの考え方をどのように活かすことができるか、考えていきましょう！

内容・範囲

輪読形式での読み合わせを行います。
読み合わせ後、セクションごとに参加者で疑問や意見を話し合っていく形式を想定しています。

第10回は、以下の範囲を予定しています。

  • 「第6章 ユーザーストーリーを集める」（6.4 ストーリー収集ワークショップを開催しよう 〜）
