◆こんな人におすすめ
　・自社サイト（本店）運営しているEC担当者
　・これから自社サイト（本店）を立ち上げようと思っている方

◆ここがポイント
モールの依存からなかなか脱却できない方必見

◆こんな課題をお持ちの皆様へ
　・ショッピングモール店舗はやっているけど、自社サイトは、展開方法がよくわからない。
　・ショッピングモールは売れているが、自社サイトは売上が上がらない。
　・以前に自社サイトを展開したが「集客・売上」に苦労し撤退してしまった。

◆講座で学べること
　①脱モール依存。脱コンサル依存。
　②価格競争から抜け出す。
　③即戦力のEC担当者になれる。

◆講座開催の背景
ECビジネス展開では、ショッピングモールの活用は欠かせません。
しかし、いつまでもショッピングモールだけに依存した展開を続けていては、
事業の飛躍的な成長は見込めないのではないのでしょうか。
そこで成長の鍵になるのが自社サイト店舗になります。
とはいえ、ショッピングモールで培ったノウハウだけで自社サイトを始めても、
なかなか売上が伸びないという壁にぶつかることが多いのではないでしょうか。
こちらの講座では、自社サイトで成功するノウハウをお話します。

◆カリキュラム
　①ショッピングモールと自社サイトの戦略の違い
　②注力すべき業務、そうでない業務（例：顧客接点（自社）＜リスティング（外注））
　③他社との差別化、ブランディングの方法
