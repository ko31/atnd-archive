description

MySQL単独では沖縄で初となる勉強会を開催します。

今回は日本オラクルでMySQLの導入支援を担当している梶山が、「MySQLとはなにか」を改めてご説明し、最新の導入事例や今後の開発の方向性についてご紹介いたします。MySQLはオープンソースのデータベースとして世界で最も普及しています。日本でもWebサイトや携帯ゲームなどのデータベースとして幅広く活用されているほか、公共機関や民間企業のシステムでも利用が拡大しています。

MySQLを現在ご利用中の方や今後ご利用を検討されている方、またMySQLがどのようなものかをこれから知りたい方は、ぜひご参加ください。

日本MySQLユーザ会について

日本MySQLユーザ会（略称: MyNA = MySQL Nippon Association）は2000年3月に発足し、日本でのMySQLの普及とMySQLユーザ間のコミュニケーションを図ることを目的としています。現在の主な活動はMLでの意見交換ですが、各地で様々な勉強会や飲み会を開催しています。
