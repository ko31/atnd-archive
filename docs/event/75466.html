【夢のような旅を実現する】

  • ドリームトリップスへのお誘い
    • 何よりも楽しい仲間と旅行を楽しみたい方
    • 家族で格安に旅行をしたい
    • 毎年のように旅行を楽しみたい旅好きの方

    • 積立型旅行倶楽部が全てのご要望を叶えます

あなたの周りに旅に出たいと考えている方はいませんか？

世界中、あなたの望むところへとご案内致します。

  1. 2005年アメリカで開業した旅行倶楽部
  2. 日本は2016年度開業予定
  3. 大阪梅田オフィス開業に伴う説明会
  • 世界中の一流ホテル（4つ星5つ星のホテル）に2つ星半以下の価格で泊まれるというコンセプトをもとにアメリカではじまったワールドベンチャーズは2005年創業されました。
  • 創業より11年目となる現在では、世界29ヶ国に展開されており、会員制では世界最大の旅行会社です。 【会員数300万人以上 日本国内10万人以上 （2015年度概算)】

特徴

世界中のホテルや航空会社と提携し、年間約1万1000回以上のツアー（DREAM TRIPS）やホテル、エアー、クルーズなどが格安（世界最安保障）で利用できるサービス（rovia）を提供しています。
