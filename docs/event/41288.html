※事前準備　追加のお願い※
ハンズオンセミナーをおこなうにあたり、
下記プロジクトをAsset Storeからダウンロードして、
インポートしておいてください。

「Mecanim GDC2013 Sample Project」（無料）

100MBありますので、必ず事前にダウンロードしておくようにお願いします。
（Asset Storeの使い方はこちら→ http://gihyo.jp/dev/serial/01/unity-asset-tech/0001?page=2 を参照してください）

前回ご好評をいただきましたハンズオンのUnityセミナー第2弾！！
現在約50万人以上の開発者が使用している「Unity」。

Unity4からの新アニメーションシステム「Mecanim」を使って、
世界中から最高に興奮するランニングゲームと称されたあの大人気ゲームを
真似して作ってみましょう。

・Unityを立ち上げたことはあるけど、良くわからなかった。
・「Unity」を学びたいと思ってた！
・昔少しだけ使ったことあったけど、もう一度勉強したいなぁ。

こんな思いを持ったメンバーと一緒にハンズオンでゲーム作りを勉強が出来ちゃいます。
しかも今回はグループワーク方式で進めさせていただきますので、
みんなで相談をしながら自分達だけのゲームを作ってみましょう。

＜スケジュール＞********************
「Mecanim」を使ってランニングゲームを作ろう！(120分)
　・Unityエディタの操作説明
　・Asset Storeの紹介
　・モデル・アニメーションのインポート
　・アニメーション状態遷移の説明
　・MecanimのAnimator画面操作
　・スクリプトでMecanimを操作
************************************

また今回はGMOインターネット様の協賛により参加費が無料となっております
勉強会の後には懇親会も予定しておりますので、
たくさんの方との情報交換をおこなってください！
