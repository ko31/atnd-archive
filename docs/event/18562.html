TDD Boot Camp 札幌 2.1

札幌では通算５回目のTDDBCが9.24に開催される予定です。
今回のテーマは「開発（デベロップメント）」。
これまでのTDDBCは、どちらかといえば「プログラミング」に重点を置いていたと思いますが、今回のTDDBCは趣向を少しだけ変えて、「開発（デベロップメント）」をテーマとして行います。

どこが違うの？

実際にアプリケーションを開発している中で、どうTDDを実践していくかがテーマになります。したがって、「お題」の実装難易度としてはやや易しくなりますが、「お題」の曖昧さや理不尽な仕様変更は増えるかもしれません。アジャイル的な開発で、TDDを実践しながら、簡単なウェブアプリケーションを開発します。

サポート言語

今回のTDDBCでは、サポート言語/フレームワークは限定されます。
これはテーマが開発であるため、ある程度は不慣れな言語/フレームワークでやるのも１つの課題であると考えるためです。勿論、各チームにTA（サポートスタッフ）を付けるため、全員が「解らない」という状況にはなりません。

サポート予定の言語/フレームワークは次のものを想定しています。これはサポートスタッフの集まり具合によりますので、今後変更される可能性は高いですが、ご了承下さい。

  • Ruby on Rails / Ruby
  • Slim3 / Google App Engine for Java
  • Django / Python

この他、ウェブアプリケーションフレームワークであれば何でも候補となりますが、最終的にはTA(サポートスタッフ)次第です。尚、候補となるフレームワークの要件は以下の通りです。

  • ユニットテスト環境が十分に整備されている
  • 簡単なウェブアプリケーションであれば覚える事はそれほど多くない
  • どマイナーな言語ではない

尚、運営としては、PHPや.net系のサポートができる人がとても欲しいと考えています。

サポートスタッフの募集

というわけで、まずはサポートスタッフを募集します。サポート可能な言語/フレームワークを明記の上、参加登録をしてください。尚、募集が多い場合は先着ではなく、サポート言語/フレームワークなどを考慮した中で選択となりますので、ご了承下さい。
