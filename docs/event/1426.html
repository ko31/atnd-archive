ご挨拶

OSC2009 Nagoyaにご来場のみなさん，どうもありがとうございました．そして，セミナー講師のみなさん，および会場スタッフのみなさん，本当にお疲れさま．

今回，このOSC2009 Nagoyaのふりかえり（反省会）と，フリー・オープンソースソフトウェアについて考える世界的なイベント，Software Freedom Dayを名古屋で祝う会を，同時開催したいと思います．

あっと，最後になりましたが，東海地域の勉強会・コミュニティのみなさん，もう一度大同団結してみませんか？せっかく集まる機会ですので，コミュニティ合同の勉強会を企画してみました．ただし，正式なプログラムはつくりません。かわりに、時間の余裕がある限り、1人10分～20分のライトニングトークス(LT)を続けてみましょう．


では，みなさまのご参加を，心よりお待ちしております．

概要

  • OSC2009 Nagoyaふりかえり
    • 参加できなかった人も一言どうぞ
  • Software Freedom Day を名古屋で祝う会
    • Software Freedom Day — 参加グループ一覧
    • （みなさんの自己紹介も兼ねて）"わたしのお気に入りのFLOSS"と題して，会場でインタビューをおこないたいと思います．
  • NGB (Nagoya Godo Benkyokai)
    • 東海地域のコミュニティならばどこでも参加大歓迎
    • LT形式の発表でお願いします
    • 発表は必須ではありません！

LT大募集

10分〜20分のLT（ライトニングトークス）を大募集．OSCでしゃべり足らなかったこと，来年のOSCでやりたいこと，近い将来のイベント告知，コミュニティ結成のアナウンス、etc. まったくOSCに関係ない発表も歓迎です:) エントリーの表明やイベントに対するご意見はコメント欄にどうぞ．


追記(9/8)：ちょっと長めの発表（20分超）もOKです．事前にコメント欄にご連絡ください．
