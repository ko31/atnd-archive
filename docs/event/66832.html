「オウンドメディア」、「コンテンツマーケティング」、「インバウンドマーケティング」・・・

最近、これらの用語を聞くことはとても多いのではないでしょうか？

従来型のマス広告やディスプレイ広告の効果が薄れてきている現在では、メディアを通してユーザーとの接点を生み出すマーケティングの手法が注目を浴びています。

多くの企業がこぞって自社や自社サービスの広報のためにメディアを運用をしており、現在はまさにメディア最盛期の時代とも言えます。

しかし、メディア運用の成功事例などは調べると出てくるものの、実際にどうやって運用しているのかはよくわからない人も多いのはないでしょうか？

当イベントでは、
プロライター・編集者として数々のメディアの立ち上げに関わってきたモリジュンヤ氏のセミナー、
実際にメディアを運用して企業・サービスの広報に努めている編集者3名でのトークセッション、
参加者全員で意見交換する交流会を通して、
企業やサービスの広報にメディアを活用することの重要性や、如何にしてメディアを運用していくか、ということについて考えていきます。

メディアの運用に興味がある学生さん、ライター・編集者に興味が有る学生さん、将来起業したい学生さんは是非ご参加ください！
