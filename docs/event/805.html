description

Zope/Ploneの開発者で集まって開発方法やディスカッションなどをやろうと思います。

スプリントのゆるい版みたいな感じで開催します。基本、毎月開催の方向で。参加費は無料です。

会場は無線LAN(今度こそ)と有線LANがあります。

Zope/Ploneの開発に興味の有る方は、是非ともこの機会に一緒に勉強しましょう。

当日のタイムテーブルはだいたいこんな感じのつもりです。

  • 11:00-11:30 自己紹介とか
  • 11:30-12:00 やること決めたり準備とか
  • 12:00-13:00 昼食
  • 13:00-17:00 もくもくと開発
  • 17:00-18:00 成果発表(できたとこまで)
  • 18:00-18:30 後片付け
  • 18:30- 打ち上げ(いける人で)
