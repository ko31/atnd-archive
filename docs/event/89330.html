description

みなさん、映画『ちょっと今から仕事やめてくる』はご覧になりましたか？

終身雇用が当たり前だった時代が過ぎ、現代では働き方・仕事に対する意識も多様なものになってきています。

職場がブラック過ぎる。上司とそりが合わない。
「もし、今仕事を辞めたら。」
そんな妄想をしてみるものの、なかなか行動に移せるものではありません。

そこで、同じ悩みを抱えた人たちで集まって交流を深め、次のアクションを考えてみませんか？

今回のイベントは『転職・独立』をキーワードに集まった10名程度のメンバーで行うちょっとした交流会です。
転職活動や起業は、人生の大きな転機。相談相手が多ければ多いほど有利になると言われています。

また、今回ナビゲーターとして、キャリアコンサルタントの方をお招きします。
ご自身も起業の経験がある方で、転職、独立を中心に様々なアドバイスを頂けるかと思います。キャリアコンサルタントの方に気軽に話を聞ける良い機会になれば、と思います。

ただし、基本的には交流会として実施予定ですので、レクチャー形式ではなく、みなさんが楽しく会話が出来る場になれば、と考えています。

何となく仕事をやめたい、と考えているけど、具体的に行動に移せない人。
転職サイトには登録したけど、何となく眺めているだけの人。
いくつか面接は受けたけど、イマイチ、ピンと来ない。
転職したいけど、就職相談会に行くほどじゃない。
そろそろ独立かな？でもきっかけが掴めない。
起業したいけど、なかなか決意が固まらない。
サラリーマンはもういいや！

そんな人たちで集まって、話し合ってみませんか？
上司や、同僚、友だちには話しづらい離職の話。
初対面だからこそ話せる話もあると思います。

諦めた夢への挑戦を語るもよし。
ブラックエピソードの愚痴でもよし。

このままだと、きっとこのまま。
次の一歩を踏み出すためのきっかけの場としてご活用ください。
