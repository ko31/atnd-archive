100万人のキャンドルナイトに関連して、
冬至の暮らしを楽しむ、暦をかんじる暮らし、ネイチャーライフなどをキーワードに。
ていねいな生き方・暮らし方のご提案、きっかけになれたら・・・なプログラムです。

１２月２３日（木・祝）１６時～２２時の間で。
日南町菅沢　ときわすれ清水屋　（古民家のお蕎麦屋さんです）
「ときわすれ清水屋さんと囲炉裏の夜」
からっぽの夜、からっぽの時間　を、開催します。

冬至に呼びかけられている、１００万人のキャンドルナイトに賛同して、
キャンドルナイトではなく、囲炉裏の火をかこんでの囲炉裏ナイトをします。
囲炉裏の火を囲んで、時間を気にせずに、ゆっくりと夜と火を楽しむ、
からっぽの時間を一緒にすごしてみませんか？

このプログラムは、開催時間の間であれば、
いつ来てもＯＫ．いつ帰ってもＯＫなプログラムです。
（事前申込み制ですので、予め、何時くらいにきて何時くらいに変えるかを伺います。）
なので、お客様の来やすい時間にお越しいただけますし、都合に合わせてお帰りいただけます。

基本料金は、１０００円です（軽食込み）
いろりでぼーっと過ごしたり、雪があれば、雪で遊んだり、こちらで用意してある
旧暦や暦、和の暮らしの本を開いてみるのもよし。自由にお過ごし下さい。
かぼちゃのほうとうの軽食がつきます。

オプショナルのからっぽプログラムがあります。こちらは、別途参加費が必要となります。
強制ではなく、自由参加です。興味のある方は、この時間に合わせてお越し下さい。
当日参加でも良いですが、人数の把握のため、事前申込みのときにあわせて、お伝えください。

オプショナルのからっぽプログラム
◎かぼちゃのほうとう作り　１６時～１７時　　無料
◎頭をからっぽにして、体の声に耳をすます、
　自力整体を体験してみよう　　　　　　　　　　１８時～（所要時間６０分）
◎心をからっぽにして、森の声に耳をすます、
　ナイトハイクにでかけてみよう　　　　　　１８時～　と　２０時～　（所要時間６０分）
※自力整体の体験と、ナイトハイクは別途参加費が１０００円です。

お申込み・お問合せは、セレンのＨＰから。
