概要

  • 休日、家にいてもダラダラしちゃう方、周りが作業している環境だとはかどります！お互いに質問し合って情報共有もできます！
  • プログラミングしても良いし、絵を描いても良いし、ブログ書いても良いし、書籍を読んでも良いです。
  • 最初に、1人1分ぐらい自己紹介と今日何をするのかを軽く喋ってもらいます。
  • 最後に、今日、何をしたかを1人1分ぐらい軽く喋ってもらいます。
  • 終わった後は懇親会をしますので良かったら参加していって下さい。家主が何か料理します。お酒、ソフトドリンクもあります。
  • 途中参加途中退出OKです。
  • 家主はゲームプログラミング全般やUnity関連なら質問にお答えできます。

会費:

  • 無料
  • 懇親会 1000円

場所

  • ギークハウス京都東福寺住所: 京都市伏見区深草車阪町20-33
  • 会場は閑静な住宅街の一軒家です。
  • wifiとお茶とジュースがあります。

注意事項

  • 最寄り駅は東福寺ではなく、京阪鳥羽街道駅(徒歩13分)になります。東福寺駅からもちょっと遠いですが歩けます(徒歩18分)。
    • 駅から、坂道を登るのでがんばってください。
