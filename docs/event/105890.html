「音楽を聴く会」やっていきます。

ここらへんにフライヤー

日程

令和元年 5月25日 (土) 13時ぐらいから

場所

A講義棟 A-114

内容

音楽好きが、好きな音楽を紹介する音楽鑑賞会です。
10〜30分程度で蘊蓄とか話したり話さなかったりしながら、あなたが好きな音楽を紹介してください。

やり方としては、曲名と、(知っていれば)ジャンルを言って曲をだらっと流す感じです。 語りたくて仕方がない人はジャンルに関するスライドでもつくってLTっぽく話してもいいですよ。

参加する方は
・ほしい時間枠 [10分, 20分, 30分, 45分] ※無指定の場合は30分
・テーマやらジャンルやら ・都合のよい時間帯(いつでも良いなら無記入でOKです)
を参加コメントに記載してください。
例：30分 J-hiphop いつでもOK → 主催が適当に割り当てた30分の時間枠に従って紹介

注意事項とか

DJイベントではないので、どんな音楽でも構いません。
「俺しか聴いてないやろw」みたいな音楽は大歓迎です。
演奏会とかではないので音源は持参してください。 例：PCからスライド&スマホから音楽；スライドに音楽埋め込み；Youtube；Soundcloudなど 音楽と一緒にMVも見せたいとかもOKです。 発表者はATND参加登録推奨です。テクノ部(音楽技術部)以外の音楽好きも歓迎します。
聴き専参加OKです！その場合ATNDで参加表明は不要です。

飛び入り参加も可能としますが、音源は持ってきてください。
Youtubeは非公式公開(公式チャンネル以外での公開)は基本的に非推奨、公式チャンネルは問題なしとします。
