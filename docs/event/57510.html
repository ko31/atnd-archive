description

常に変化し続けるWEB業界。
数多くある技術の中から、何を選択し、何を学び、どう進むべきか悩んでいるクリエイターは少なくないように思います。
本セミナーでは数多くのインタラクティブなサイトを制作してきたunouplusの戸田芳裕さんを講師にお迎えし、
長年WEBを中心としたお仕事をされてきた中でWEBにとどまらずアプリ制作を手がけるに至った経緯や、
その仕組みや面白さ、制作物へのこだわりなどをご紹介いただき、
今後クリエイターとしてWEB業界を生きていくためのヒントをお話しいただきます。

今まさにWEB業界で働きながら技術を学んでいる方はもちろん最近のWEB制作の動向を知りたい
デザイナー、マークアップエンジニアの方にも役に立つ内容となっております

セミナー終了後には講師も交えたクリエイター懇親会を開催します。
参加費は無料ですので、是非お名刺持参の上ご参加くださいませ。

参加方法はコチラから！
