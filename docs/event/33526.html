description

仙台iOS開発者向け勉強会 第9弾を開催します！

内容

しばらく間が開いてしまいましたが、来たる11月17日（土）に、iOS開発者向けの勉強会を開催します！　

仙台でiOS開発者が集まって話をする機会も貴重かと思いますし、情報交換の時間もご用意しております。是非ご参加ください〜。

対象

  • iOSアプリ開発をやっている方
  • プログラム経験があり、iOSアプリ開発もやってみたくて、本を1冊くらい読んだことがある方
  • 初心者のかたも歓迎です。（ただ、若干内容がわかりにくいかもしれません…）

タイムテーブル

Session 1 「Core Animation for iOS」 by @ushiostarfish さん
サンプルソースコードはこちら https://github.com/Ushio/SWWDC_CoreAnimation 		60 min
- 休憩 10 min
Session 2 「UITableViewで無限CoverFlowを作る」 by @h_mori さん 60 min
- 休憩 10 min
Session 3 質疑応答、井戸端会議、懇親会の相談、会場の後片付け
12/1の 2012仙台クロスコミュニティ大忘年会でSWWDCの紹介LTをしてくれる人を決める。 		70 min

会場

  • 仙台市シルバーセンター 第三研修室 (宮城県仙台市青葉区花京院１丁目３番２号)
  • 会場使用料(2400円)を参加者で割り勘にします
  • ネットワーク環境はありませんので、各自ご用意ください
  • 電源延長タップをご持参いただけると助かります

懇親会

たぶん、終了後に希望者を募り、どこかの居酒屋で懇親会をすると思います（つまりノープランです）。すべては、その場で決めます！
