http://www.google.com/url?sa=D&q=https://spreadsheets.google.com/spreadsheet/viewform%3Fformkey%3DdHZFNW1Fc0gyUTFFbVg4WVJPZ0xvVUE6MQ

勉強会が活発に行われているAndroidですが、ジャンルが限定的だったり、逆に広すぎたり、と難しい物があります。
また、ハンズオン系のイベントは少なく、参加者がもっと詳細に聞きたいと思っても聞く場所が無かったり、
ライブラリ・フレームワークを開発している人がもっと詳細に説明したい！使用者を増やしたいと思っても、効果的に広めていく場所が無い問題があります。

そこで、プチGDD(Google Developers Day)っぽいもの+もくもく会を計画しています。

Python Hack-a-thonを例にして、以下の混合形式での実施を考えています。
・ハンズオン
・Hack-a-thon、もくもく会
・発表
　・Lightning Talk
　・成果発表

メタっぽい内容から、細かい内容まで抑えて、かつ自由度の高い勉強会・もくもく会にしようと思ってます。
途中参加も自由、何をやるのも自由、ちょっと来て作業して帰る。
そんな気軽に参加できるイベントにできたらいいな、と思ってます。

そこで、ハンズオンの内容や、運営に関してのアンケートを実施いたします。
ハンズオンの講師をやってくださる方や、LTの内容も広く募集しておりますので、ご協力お願いいたします。

アンケートの回答が手間でも、興味のある方は参加だけでもよろしくお願い致します。
