description

【講座のポイント】
　iPhoneを皮切りにスマートフォンは世界において急速な普及を見せています。先進国だけでなく新興国にも急速に広まると考えられるスマートフォンは、既存の携帯電話・エレクトロニクス業界だけでなく、あらゆる社会の変革を促します。スマートフォンにより今後どのような社会変化、ビジネス変化が起こるのか？そして、サービスプロバイダー、端末メーカー、部品メーカーが今後注目しなくてはいけないことは何か？ということについて解説します。

【受講対象者レベル】
　サービスプロバイダー、メーカー、部品メーカー等、既存の携帯電話関連の業界に従事している方を中心としながらも、幅広い方が対象

【プログラム】
１．スマートフォンと携帯電話業界の現状
　　1-1　世界における現在の普及動向、販売動向

　　1-2　Apple VS Android

　　1-3 その他のプレーヤー

　　1-4 日系メーカー・キャリア等の対応状況と戦略

２．スマートフォンが引き起こす変化
　　2-1　スマートフォンの普及予測、技術トレンド予測

　　2-2　社会生活に与える影響・アプリケーション（先進国、新興国、日本国内）

　　2-3　ビジネスに与える影響・アプリケーション（広告、コンテンツ、その他）

３．日本のプレーヤーに対するインプリケーション
　　3-1　サービスプロバイダーにとって

　　3-2　携帯電話キャリアにとって

　　3-3　端末メーカーにとって

　　3-4 部品メーカーにとって
