●ポケモン選出方法・ルール
★重要★個体変更ありルールです。
-使用出来るポケモンは最大6匹です
-当日の準備時間中、こちらから配布するトレーナーカードに、使用予定のポケモンを6匹ずつ選び、それぞれのポケモンに①～⑥の番号を付けて、種族名を記載していただきます
-対戦毎にトレーナーカードを相手に見せ、見せたカードに記載されたポケモンの中から更に3匹を選び、順番を決めて対戦を行います
-試合前のQRコード読み取り手順は以下の通りです
①アプリを一度落とし、再起動をする
②トレーナーカードを見せ合う前に、QRコードにて対戦申し込みを行う
③じゃんけんを行い「勝った方」がQRコードを差し出し、「負けた方」がカメラで読み込む
④パーティの書かれたトレーナーカードをお互いに見せ合う
⑤使用するポケモン3体を90秒以内に選出し、対戦を始める
⑥2戦目以降は、お互いが「再対戦」を押して対戦を始める
-同じ種族のポケモンを入れることは出来ません。ただし、アローラ・フォルム違いは別種族として扱います
-★重要★トレーナーカードに記載後、同じ種族・名称のポケモンを、 途中で別の個体に入れ替えるのはOKです。
-★重要★大会進行中のわざマシンを使用しての変更・サードアタック（＝2つ目のスペシャルアタック）の解放、強化はNG とします
- 対戦相手のわざ情報を、故意・意図的に他人へ伝達・共有したことが発覚した際には、その時点で失格 とさせて頂きます。また故意でなくても注意・警告の対象となり、注意・警告が重なった場合は失格とさせて頂きます。
- トレーナーカードは、大会終了後に回収 をさせて頂きます。

- 大会上位者の選出・わざ構成は、結果発表を兼ねて開示する可能性があります ので、予めご了承ください

●対戦中のバグ・通信不良時の対応について
-電波良好であることを事前に確認済みですが、バグや通信不良が起きて対戦の正常な続行が不可能になった場合、両者合意の元再対戦を行なってください
-再対戦の際は、次の規定に従って再対戦を開始してください
★規定1： 自身の陣営でバグ・フリーズが発生し、再対戦の依頼をした場合
→依頼を受けた側が下記①②のどちらかを選択できる
★規定2： 相手の陣営でバグ・フリーズが発生し、再対戦を提案をし、相手側が合意した場合
→提案をした側が下記①②のどちらかを選択できる
　 ①初手対面を同じままで再対戦を行う
　 ②初手対面をお互いに変更して再対戦を行う
★規定3： いずれの場合も初手以外の2匹については、選出変更可能とする
★規定4： 2度目のバグ・フリーズにより試合が中断された場合、次の再対戦は大会本部席にて行う
