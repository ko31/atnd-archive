基調講演【A-1】13：00～14：20
①クラウド時代に向けた情報創造コミュニティーの取り組みのご紹介
企業経営には情報化/情報活用の進化が欠かせません。協立情報通信は、情報創造コミュニティという「場」を設け、「①インフラ・②コンテンツ・③活用能力」の三分野を統合したソリューションサービスを提供し、お客様の情報進化をサポートする体制の充実に努めております。お客様の情報活用による価値づくりに貢献するべく、当社の考え方ならびに取り組みについてお話いたします。
（講師）協立情報通信株式会社　代表取締役社長　長谷川 浩

②次世代型のビジネス変革を支える取り組みについて
～デジタルトランスフォーメーションで共に作る未来～
“デジタルトランスフォーメーション(DX)“がビジネスにどのような変革をもたらし、新たな収益や価値を生み出すかと共に、世界中の人がよりよく生きることのできる未来の社会づくりについてご紹介します。
（講師）日本電気株式会社

会計制度【A-2】14：45～15：45
早めの対応が効果的！
「消費税10％増税・軽減税率制度の正しい理解と企業実務への影響」
消費税１０％への再増税そして軽減税率の導入は2019年10月に延期されましたが、残すところあと2年。この2年間で企業は、軽減税率制度にどのように対応する必要があるのか？
本セミナーでは、2度に渡って延期された消費税１０％増税と軽減税率制度について解説します。今一度しっかりと消費税改正の内容を抑えていただくことで、今後の実務への影響や対応すべきポイントをつかんで頂きます。
（講師）アクタス税理士法人
-——————-
業務効率化【A-3】16：15～17：30
①複雑・難解！「平成30年改正配偶者控除」
～制度の理解から実務テクニックまですべてを丸ごとご紹介～
平成30年1月より、配偶者控除・配偶者特別控除の税制改正が施行されます。企業には、世帯主・配偶者ダブルでの年収要件や今年から来年の2年間をかけて準備すべきことが発生し、複雑な制度を正しく理解して対策を講じることが急務となります。本セミナーでは、改正内容や給与・年末調整業務への影響と、企業がいつまでにどのような実務を行う必要があるのかを、給与システムを中心とした運用方法を交えて詳しくご紹介します。
（講師）株式会社オービックビジネスコンサルタント

②総務ご担当者様必見！
労務手続きの不安と紙の手作業を徹底的になくす新登場「労務管理サービス」の全貌を初公開！
入社・退職や氏名・住所変更に伴う社会保険手続きでは、紙の書類を回収・作成・届出するだけでなく、ミスや漏れ抜けに対する不安からチェックや確認を重ねるため、非常に時間がかかっているのが実態です。本セミナーでは、「労務管理サービス」を使った労務手続きの効率化を徹底解説します。
（講師）株式会社オービックビジネスコンサルタント

【B-1】11：00～12：00
クラウド時代を向かえ、様々な働き方に対応する企業ネットワークのご提案
企業の生産性向上のための働き方改革は、多様な業務モデルの推進とも言い換えられると思います。これを支えるITネットワークも日々進化しております。働き方改革を支えるセキュア・堅牢なネットワークの最新情報をお伝えします。

【B-2】13：30～14：30
今日から取り組める！MSクラウドを活用したチーム生産性を向上させる情報活用をご紹介
場所を問わず、どこでも仕事ができる環境を創ることにより、社員の自律性とチームの生産性を向上させることができます。Office365 はその環境を提供するクラウド型情報プラットフォームです。日々進化する Office365 の最新機能のご紹介と共に、チーム活用の運用事例、生産性効果事例等をご紹介します。
（講師）日本マイクロソフト株式会社

【B-3】14：45～15：45
スマホ・タブレット活用のためのドコモクラウドサービス
