講演概要

レスポンシブWebデザインは流行だと思われる傾向があるようですが、一時的なものでなく今後より注目されていく技術になることをお伝えできればと思います。

また、レスポンシブWebデザインと表裏一体でもあるモバイルファーストというコンセプトがあります。

これは、講演者がWeb担当者フォーラムで初めてモバイルファーストについての記事を記載し、日本でも注目されたのでご存知の方も多いと思いますが、これらについても触れてご紹介させていただく予定です。

ウェブだけの活字だけでば理解できなかった事をこのイベントを通してリアルに理解することでしょう。

海外情報をいち早く入手している菊池氏からの正確な情報をお伝えできればと思います。

１時間ではイントロダクションしかお伝えしきれないかと思いますが、制作面もビジネス面でもコンセプトは重要になります。その理解の足がかりになればと考えています。

  1. W3Cが考えるレスポンシブWebデザインとその流れ
  2. メリットとデメリットを考える
  3. 海外、日本のレスポンシブWebデザインのご紹介
  4. モバイルファーストとは？
  5. 間違って理解しないためのモバイルファーストとレスポンシブWebデザインの関係
