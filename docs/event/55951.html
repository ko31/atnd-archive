description

残り２名の募集となります。

■読書をする目的は何ですか？

皆さんは、何のために読書をしますか？
本を読むこと自体が目的ではなく、本から得た知識をビジネスに活用するためではないでしょうか？
この読書会では、読書会が終わった時に「あなたが自分のやりたい次の行動が明確になっている」ことが一番大事なことだと思っています。

■読書はしたいけど時間がないという悩みはありませんか？

しかし、日々の仕事が忙しく、なかなか本を読めない。
気付くと、買ったものの読まないで放置している本が山積みになっているなんてことはないですか？
買った本は、少なくとも買った時点では、あなたにとって何かしら興味があったり、読んでみたいと思った本ですよね。
にもかかわらず、読まずに本棚に飾っておくはもったいないと思いませんか？

■特別なスキルは必要なく、とにかく気軽に実践できる読書法です。

私は以前、フォト―リーディングの講座を受講し、習得しています。
この技術により短時間で本の知識を手に入れ、実行に移すことができています。
フォトリーディング以外にも世の中にはたくさんの速読術が溢れています。
しかし、実用的なスキルにするには、どれも習得には講座を受講し、その後、相応の練習を積む必要があり、一朝一夕には身に付かないのです。
その点、この読書会は、特別なスキルは一切必要ありません。
お忙しい皆さんでも気軽に実践できる読書法なのです。

■どうして次の行動を明確にできるのか？

この読書会の手法は「しつもん読書会」と言い、実は読書会の中で、ほとんど本を読みません。
その代わり、私のファシリテーションのもと、質問に答え、回答を参加者でシェアしてもらいます。
これで、自分自身で考えたことに加えて、他の参加者5名の回答から新しい気付きやアイデアを手に入れられるのです。
そして、質問に回答していくだけで、次の行動が自然と明確化できる質問の流れになっています。

■行動して、PDCAサイクルを回すことが重要です。

しかし、読書だけで簡単にビジネスに活かすことはできないというご意見もあると思います。
偉そうなことを言う立場にはありませんが、私も一人ビジネスをしている人間として強く思うことがあります。
それは、とにかく行動し、失敗したら課題を明確化して、また次の行動を起こす、つまりPDCAサイクルを回し続けることの重要さです。
つまり、行動を起こさない限り成功することはあり得ないのです。

■ビジネス仲間ができます。

質問に回答し、回答をシェアしていると不思議なことに以前から知り合いだったかのように打ち解けられます。
それは、考えていること、思っていることをシェアすることで、その人の人となりが伝わるからだと思います。
経験上、ビジネスにおいて利害関係なしに相談し合えるような仲間を作るのは容易ではないです。
是非、ビジネス仲間をたくさん作ってください。

今回の対象ジャンルは「マーケティング」になりますので、以下のような方におススメです。
・マーケティングに興味があり本を買ったけど、結局、読めていない
・経営のためにマーケティングの勉強が必要と思っている
・利害関係なしにマーケティングを学ぶ仲間が欲しい
・集客に悩んでいる
・読書の時間がなかなか取れず、短時間で本のエッセンスを吸収したい
・読書してインプットできても、アウトプットできていない

それでは、お会いできるのを楽しみにしています。
