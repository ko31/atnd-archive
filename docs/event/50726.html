今回は、教育パワーランチをスタートさせたきっかけなどをお話させて頂いた後、教育パワーランチのコンセプトのひとつである『みんなで学び新しいきっかけに出会う』機会として、病の進行により身体が動かなくなる中でも、前向きに大好きな料理をする方法を考え、現在は電子レンジを使った料理レシピを数々ＣＯＯＫＰＡＤにて紹介している『かっぺキッチン』さんとコラボ、みんなでこの教育パワーランチ用に作られたレシピ、しかも前回の旭川教育パワーランチにて、旭川ベジ部コラボ企画で生まれた『きっかけ』からスタートした新レシピに参加者全員でチャレンジしたいと思います。

東神楽町での初の教育パワーランチと言うことで、
北海道教育ユニットEduの能正さんの方で
まずは教育パワーランチって何？と言う部分に少しお話しをした後、
かっぺキッチンさんの方で、
今回は特別に！いつもCOOKPADに出していない新しい料理を
みんなでちょっとですが調理して一緒に食べると言う流れで
実際に、やってみて感じる、学ぶ機会としてみたいと思います。

参加費８００円（食事代、材料費のみ）

参加者については、このパワーランチについて何？って思った方や、
関心・思いのある方であればOKです！お待ちしてまーす！

金曜日昼間の開催となりますが、是非お立ち寄りください☆
