Windows Phone Arch Tokyo#4 (MADOSMA当日祭)

Windows Phone ArchはWindows Phoneのコミュニティです。
http://wp-arch.jp/

先日de:code2015の会場で鮮烈なデビューを果たした4年ぶりのWindows Phone「MADOSMA」の非公式発売日記念イベントを開催します。

JR品川駅から会場までの行き方

詳しくは→こちら←で確認してください。

ざっくり
１．改札出て、港南口（東口）方面へ歩いてください。
２．そのままデッキを右（南）に進み、途中「CAFE de CRIE」という喫茶店を右に曲がるとMicrosoftの入口があります。
３．入口を入った所に担当者が居ますので、入館証を受け取ってゲートをくぐり、エレベーターで31階へお上がりください。

20時など遅れてきて既に担当者が居ない場合は以下Twitterでメンションを飛ばしてください。

  • 集合時間以降にご来場の場合は、2F受付から@hatsune_までご連絡ください。スタッフが迎えに行きます。
  • 20時以降は入り口が閉まりますので、2F入り口前から@hatsune_までご連絡ください。スタッフが迎えに行きます。

入館情報ご提供依頼

会場をお貸し頂く日本マイクロソフト様に提供する入館情報として以下の情報をお送りください。

  • 名前（ハンドルネームNG）
  • 所属
  • メールアドレス
    なお、お送りいただいた情報は入館情報としてのみ利用し、他への利用は行いません。また入館情報として提出後はお送りいただいた情報は削除し、Windows Phone Archで保持は致しません。
    お手数をおかけしますが、よろしくお願いします。

送信件名：MADOSMA
送信先メールアドレス：hatsune[at]wankuma.com
[at]を@に書き換えてください。

テーマ

みんなでMADOSMAを持ち寄って、一番素敵なMADOSMAを決定しましょう。
また、日本で一番高いMADOSMAタワーを発売日につくっちゃいましょう。

こんな理由にピンときたら、ぜひ参加ください。

  • 当日店頭でMADOSMAを入手できたので
  • 急ぎ帰宅して宅配を受け取ったので
  • 当日入手できなかったけどMADOSMA実機を見たいので
  • 購入予定はないけど興味があるので

当日は、軽食やお飲み物をご用意させて頂きます。
