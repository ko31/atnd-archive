title 小江戸マッピングパーティ
catch OpenStreetMapに触れてみよう！
description

OpenStreetMap上に地図を描く方法を経験者とともに体験するイベントです。小江戸（川越）の町を楽しみながら歩きます。
経験者・未経験者を問わず歓迎致します。お気軽にご参加ください。

1.日時、集合場所
10/29(土)13時00分～17時00分(13:00集合)
西武新宿線　本川越駅ホーム北端１F改札を出たところ に集合

雨天決行（歩ける程度なら）

2.費用
・参加費無料
・交通費、食費等は各自で負担ください。

3.募集人員
・制限はありませんが5名ほどを目安に考えています。

4.持参するもの
・最低限筆記具があればOKですが、以下のようなものがあるとさらに楽しめます。
・デジカメ
・スマートフォン
・GPSロガー
・モバイルなPC

5.服装など
・徒歩で歩く予定ですので軽装でお越しください。

6.目標
＜共通＞
様々なアイディアで商店街活性化に取り組んでいる 「川越一番街商店街」 周辺の OpenStreetMap を充実させる
＜初心者＞
・OSM上にノード（お店などの点）を打てる
・タグ付けの概要を理解する
＜経験者＞
・現地でなければ分からない情報の収集と登録
・自分で設定したテーマの実践
・初心者ヘルプ

7.タイムスケジュール（当日の状況により多少変わる可能性があります）
13:00～ 集合、資料配布
13:05～13:30 行動予定の説明、現地まで歩きながらマッピングの説明
13:30～15:00 マップ情報収集（２班ほどに分かれる）
15:00～17:00 マップ編集作業（喫茶店等を予定）
17:30～19:30 川越の駅周辺で懇親会（自由参加、実費割り勘）

8.当日までの事前準備
＜必須＞
・事前にOpenStreetMapアカウントを作成しておいてください。
手順はこちらのビデオを参照
＜任意（可能な方のみで構いません）＞
・現地付近のBingによるリモートマッピング（建物など）
・以下の関連情報をご一読ください。

9.関連情報
◆OpenStreetMap操作ガイド（Potlatch2編） (約6MB)
◆OpenStreetMap
◆ドキュメント（日本語トップ）
◆日本でのタグ付けの基本（主に道路）
◆タグ一覧（カテゴリ別）
◆タグ一覧（50音順）

10.マッピング（マップの描き方）の例
例えばコンビニをOpenStreetMapで表現する場合には以下のように描きます。
1)まずコンビニの位置にノード（点）を打ちます。
2)そのノードに以下のようなタグ（属性を表わすキーと値の組み合わせ）を書き込みます。
shop=convenience （コンビニ）
name=FamilyMart （名称）
