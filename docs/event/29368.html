description

マイコン学習の定番となったArduino(アルドゥイーノ)の使い方を１から学習できる教室です。
参加条件としてArduinoの開発環境が動作するノートPC（Windows 又はMac で USBポートを備えるもの）を持ち込める方に限らせていただきます。

費用：各コース参加費　１５００円
教材費：レンタル教材を使用する場合は　１回５００円　教材をお持ち帰りになる場合は教材をお買い上げいただきます。

Arduino-UNO互換マイコンボード　\1,900
USBケーブル　\250
ブレッドボード \250
接続ケーブル \200
LED＆抵抗等 \100

参加年齢の制限はありませんが、小学生の方は保護者の送り迎えが必須となります。

＊初めての方は初回はこのコースを受けていただきます。
以降,6/6,6/9,6/17,6/18,6/20,7/2,7/11,7/14,7/21と同じ内容で10回開催予定です。

その後は、順次サーボを動かしたり、センサーやＬＣＤディスプレイを使ったがジェットを作成するコースを開催いたします。
