description

チームのコミュニケーションについて考える勉強会。
例えば、

  • デザイナー・非デザイナー間で議論するとき、実際のイメージ画像を軸に議論したいけど、うまくコミュニケーション出来るツールがなくて結局テキストで議論している。
  • エンジニア・非エンジニア間で、必要以上にエンジニアに気を使ってしまったり、苦労を理解してくれない非エンジニアに不満を抱えていたり。
  • リモートのエンジニアにお願いしているんだけど、もっとスムーズにコミュニケーションをとる手法やツールとかない？
  • エンジニア同士のコードレビューでケンカになってしまった…

Chatwork, HipChat, Corabbit, Screenhero, Skype, Redmine, PivotalTracker, Github などなど。
自社での取り組みや体験を共有して、よりよいチームビルディングを考えたい、という目的の勉強会です。

StartupTechnologyとは

  • スタートアップの経営者
  • スタートアップの中の人
  • スタートアップとお仕事をしているフリーランスの人
  • スタートアップに興味がある人

などスタートアップに関わっている、関わりたい方のためのコミュニティです。

スタートアップに必要な技術・知識・スキルはステージによって様々です。
急激な成長を求められるため、うまくまわらない、知見が足りないなど様々な問題を抱えているところが少なくありません。
そこで知見を共有したり、相談出来る場所を作ろうというのが会の目的です。
