【講座趣旨】
画像処理/認識を進化計算法を用いて最適化・自動生成する方式である進化的画像処理並びにそれを認識に応用した進化的画像認識技術の原理と方法について,最新の研究事例の紹介を交えて解説する。

【プログラム】

1　画像処理/認識技術の現状と課題
　1.1　画像処理技術の問題点
　1.2　画像認識技術の問題点
　1.3　最適化・自動化の必要性について

2　進化計算アルゴリズム
　2.1　GA(Genetic Algorithm;遺伝的アルゴリズム)
　2.2　GP(Genetic Programming:遺伝的プログラミング)
　2.3　ES(Evolution Strategy;進化戦略)
　2.4　PSO (Particle Swarm Optimization)
　2.5　DE(Differential Evolution)
　2.6　GRAPE(Graph-structured Program Evolution)
　2.7　その他の進化計算アルゴリズム

3　画像処理 + 進化計算 = 進化的画像処理
　3.1　数値パラメータ最適化
　3.2　直列フィルタ列最適化
　3.3　木構造型画像処理最適化
　3.4　ネットワーク型画像処理最適化
　3.5　進化型回路網による画像処理回路の自動構築
　
4　画像認識 + 進化計算 = 進化的画像認識
　4.1　画像特徴量の最適化と自動選択
　4.2　画像特徴量の自動生成
　4.3　前処理の最適化
　4.4　画像認識プロセス全体の自動構築
　4.5　人に分り易い認識器の自動構築
