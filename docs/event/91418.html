『ファミキャリ！転職フェア』とは…

株式会社クリーク･アンド･リバー社が企画・運営する人気ゲーム会社５社による合同転職説明会です。

現在、ゲーム業界は深刻なクリエイター不足という問題を抱えています。そこで、当社は優秀なクリエイターの発掘とゲーム業界の発展を目的に、業界のトップ企業5社による合同転職説明会を開催することになりました。
参加するゲーム会社は、「株式会社カプコン」「株式会社スクウェア・エニックス」「株式会社コナミデジタルエンタテインメント」「株式会社ディー・エヌ・エー」「株式会社エイチーム」の全5社。福岡・大阪・東京の3ヵ所で開催します。

当日は各社の採用担当者が会社説明、開発体制、募集職種などについてお話します。ネットだけでは得られない内容ばかりの1時間です。質疑応答のお時間も設けておりますので、企業への理解を深めた上でご応募・選考に進んでいただけます。
人気ゲーム会社が一同に会しての転職説明会はまたとない機会ですので、ぜひこの機会にご参加ください。

★転職をご希望の方はもちろんのこと、今すぐ転職を考えていない方のご参加も大歓迎です。
★各ゲーム会社への理解を深め、今後のキャリアアップにお役立て下さい。
