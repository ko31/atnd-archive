「TDDってなんだっぺ?」な人達のためのゆるい勉強会

ケント・ベック『テスト駆動開発入門』の内容をゆるく勉強します。

難易度の高い勉強会にはなりようがないので、お気軽にご参加下さい。

  • 参加者数によっては変更される場合があります。上記は上限値です。

ワークショップに必要なもの

ワークショップでは、実際にプログラミングを行います。そのため、JavaおよびJUnitの実行が可能な環境を整えたPCが必要です。
ただし、複数人体制での作業となるため、参加者全員が持参する必要はありません。

  • PCの持参の可否をコメントにて明記下さい。
  • サポートの対象外となってしまいますが、Java以外の言語の使用を希望される方は、希望言語および対応するテストツールについても併記して頂けると助かります。
  • 基本的には、最近リリースされたEclipseでHello Worldが実行できる状態になっていれば問題ありません。

注意事項

  • この勉強会はTDD Boot Campではありません。この勉強会の目的はTDD本に載っている内容の学習です。TDDBCとは趣旨が異なります。
  • 駐車場は有料です。1時間100円となっています。ただし、受付の機械に駐車券を通せば、1時間分は無料になります。
  • 館内は禁煙となっております。喫煙の際は所定の喫煙所をご利用下さい。
  • 会場内での飲食は問題ありませんが、ごみはお持ち帰り下さい。
  • 電源の容量が1500Wまでとなっているため、参加者全員への電源供給は難しいかも知れません。
