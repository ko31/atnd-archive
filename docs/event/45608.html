description

この度、GDG Shikoku では、Androidアプリ開発を推進すべく、アプリ開発中級者向けの「おいしいアプリ開発[中級編]」を高松で開催します。

主催：ぷち勉強会、GDG Shikoku、日本Androidの会 香川支部
日時：2013年11月30日(土) 13:30頃～16:00頃
場所：コワーキングスペースgain-Y（ガイニー）（http://gain-y.com/）
必須スキル：Android で Hello World 作成経験
準備物：パソコン（Android 開発環境をインストール済）、Android端末（エミュレータ不可）
参加費：500円

おしながき

レシピ１：画面の作成
レシピ２：ENTER でボタンイベント
レシピ３：メッセージのカプセル化
レシピ４：Peer の作成
レシピ５：INTERNETパーミッション
レシピ６：Intent による値渡し
レシピ７：Intent による値の受け取り
レシピ８：Handler
レシピ９：IPアドレスの取得
レシピ10：ServerSocket

中級編ということで、 Android 開発環境は各自で用意してきていると仮定しています。 Hello World は作成できたが、どんなものが作れるのか分からない方を対象としています。
中級編は初の試みですので、今回がシナリオテストを兼ねていますことをご了承ください。
本番は来年以降に各地で順次開催予定です。

パソコンを持って、是非参加してください。

いつもと場所が違いますのでご注意ください
