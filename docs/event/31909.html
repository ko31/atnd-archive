description

オライリーから出版された『オープンソースで学ぶ社会ネットワーク分析――ソーシャルWebの「つながり」を見つけ出す』をテキストに、みんなで社会ネットワーク分析について勉強する会です。

日々分析を行なっている方、「TwitterやFacebookの分析やってみたいけどよくわからない」という初心者の方、エンジニアの方、エンジニアじゃない方、社会人の方、学生の方など誰でも大歓迎です。気軽にご参加ください！

進め方

今回は懇親会開催のため、テキストおよびLTのみの内容となります。

テキストの輪読（40分）
1章ずつ担当者を決めて発表していただきます。
↓
LT（合計30分）
「こんな分析やってみました」「こんなツール使ってます」のようなSNAに関するネタの発表。いやむしろSNA関係なくてもいいですｗ
↓
演習（40分）
実際に手を動かして理解を深めましょう。
