gumistudy@福岡 は昨年11月19日から2度目の開催になります。

今回は KLab株式会社 様との共催 & gumi福岡新オフィスのお披露目会とさせていただくことになりました!
現在の開発現場ではバージョン管理システムの利用は不可欠になっており、
gumi では Gitを、KLab では Bazaarを標準的に使用しています。
各社が実際に現場でどのように使用しているか、実運用に沿って紹介し、
それぞれのバージョン管理システムの特徴が共有できる勉強会にしたいと思います。
今回はそれだけではなく、開発言語や本番のサーバ構成など、技術的な観点から各社の紹介も行います。
Git や Bazaar などの技術にご興味の有る方だけでなく、各社にご興味のあるエンジニアの方々はどうぞご参加下さい。
最後に懇親会も行いますので、そちらにもどうぞご参加下さい。
