ゲームを作りたいけど何からすればいいか判らない、
興味はあるけどプログラムとか難しそう…、その夢Unityが叶えます！

最近話題のゲーム開発環境Unityについて基本知識や採用事例など、ゼロからはじめる入門セミナーを開催します。
講師にユニティ・テクノロジーズ・ジャパン 高橋啓治郎さん、山村達彦さん、ゲストに開発室Pixel 天谷大輔さんをお迎えして、Unityやゲーム開発のことについて色々お聞きします！

「セミナーとか初めてなんだけど行っていいの？」って方、大丈夫です。普段着で来ていただいて大丈夫ですし、お一人様もたぶん大丈夫。ゲームを作りたいひと、作っているひとなら大丈夫。

セミナー終了後、同会場で交流会をおこないます。 こちら からお申し込みください。
