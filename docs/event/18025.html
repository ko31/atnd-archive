講義資料

  • PDF on Google Docs

教科書

谷崎 『リー代数と量子群』 を使います。

ただし、持ってなくても参加できるように工夫はしたいと思います（できなかったらスミマセン）。

進め方

  • 教科書をベースに、講師役（@maophilia）が解説をしていきます。
    • 第1回の今回は、第1章＋2章以降の導入をやります。
  • 基本的に教科書に沿って進めますが、講師の 気分 配慮で飛ばしたり順番を変えたり話題を追加したりします。
  • 質問・ツッコミ歓迎。 質問がないとどんどんサクサク進むので、気軽に割り込みをかけて下さい。

事前準備など

  • やる気のある人は 1章 を予習しておいてください。
    • 事前にどのへんが分かりにくいかをtwitterで #start_repr タグつきでつぶやいといてください。当日の説明の参考にします。
  • 線形代数を理解してることが大前提です。でも、分からなければ突っ込んでもらえれば説明はします。
    • 計算自体は閃きのいるtrickyな計算とかはなく、ひたすら機械的に計算していくだけで済むので、そこまでhardではないでしょう。

その他

  • 会場代として 1100円 程度申し受けます。
  • 会場は 「CLTT読書会」名義 で借りています。当日会場を探して困らないで下さい。
  • 主催者（@bonotake）はズブズブの素人です。ただの段取り役ですので誤解なきよう
