description

スペース・デジタルポケットでは、ビスケット開発者の原田博士が、ビスケットの秘密を教えてくれたり、一緒に開発にチャレンジしてくれるスペシャル講座を実施します。この連休に、新しい学びや表現を体験してみませんか。

コンピュータが最もコンピュータらしいこと、それは「プログラミング言語を作れる」ということです。
このすごさを何かに例えてみるなら、「プログラムで何でも作れる＝忍者が術をつかって何にでも変身できる」ということだとすると「プログラミング言語を作れる＝新しい術を発明できる」ということでしょうか。

この講座は子どもたちに大人気のビスケットの中身について解説します。
仕組みの解説と理解のために文字の言語（Javascript）を使いますが、初めての方でもわかるように解説しますので、Javascriptの入門講座でもあります。

講座の内容

　１）ビスケットの中の人になって考えてみよう（CSアンプラグドによるビスケットの動作の理解）
　２）ビスケットの中身は3つに分かれている
　３）ビスケットのプログラム（メガネ）と同じ動きをさせてみよう（Javascript入門）
　４）ビスケットのコンパイラとインタプリタの話

講座の抱負

「よく聞かれる質問に、「ビスケットをやったあと何を勉強すればよいですか」というのがあります。
ビスケットは必ずしもコンピュータの専門家を目指していない、すべての人を対象に作りましたが、この質問は少し専門的な方向に進むにはどうしたらよいですか？という意味でしょう。
よくあるプログラミング入門では、ゲームや実用的なアプリを題材にしたものが多いですが、それではコンピュータの深い面白い部分になかなか迫れないと思います。学会や大学の講座でもあまりこういう話はしませんが、知りたい人にとってはプロでも面白い内容だと思います。そういうのを見せることなんじゃないかな、というのが、その質問の答えです。
（講座では、一番難しい部分で三角関数が出てきますが、習っていなくても心配ありません。）
