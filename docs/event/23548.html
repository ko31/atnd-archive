マインドマップってなんとなく知ってるけど、実はやったことが無い。
自分なりにやったことはあるけど、これでいいの？
実は、マインドマッパーで普段から描いてます。
という方（興味のある方なら誰でもOK）を対象にマインドマップの勉強会を行いたいと思います。

都合のあう方、是非、参加お待ちしております！

日時：2012年1月9日(祝）PM1:30～4:45（会場：PM1:15）
場所：大阪市中央公会堂　http://osaka-chuokokaido.jp/index.php
費用：500円（会場代として）
定員：15人
持ち物：4色以上のカラーペン（用紙は用意します）

内容：概略の説明
　　　マインドマップでメンタルケア（練習）
　　　マインドマップで行動計画（実践）
　　　
　　　描いたものをグループでシェアしあいながら
　　　ゲーム感覚で楽しんでいただけるような内容を考えています

準備：マインドマップとは何かを自分なりに予習しておいてください
　　　できたら簡単なものでいいので描いてみておいてください

目的：マインドマップが自分なりに描けるようになる

主催：中田珠美（勝間塾コミュ名：tamami）
※1年前に玉川一郎さんのマインドマップの講座を受けていて、
　普段、ブックレビューや勉強する時などに使用しています。
　（インストラクター資格はありませんので、学んだことをシェアする会になります）
