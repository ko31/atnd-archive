7月5日（土）　ユニオンベースBフィールドでサバゲ会

description

7月5日(土)にユニオンベースBフィールドで開催いたします。
毎度の運営進行は、 朝木 と けろよん お手伝いに 狐古

基本的にはいつも通りの縛りのぬるい感じのゲーム内容ですので初心者の方も歓迎です。
とは言っても、安全の為のルールはキチンと守って頂きます。
レギュレーションはごくごく普通で、弾数制限等はございません。詳細は以下の通り。
・セミオート・フルオート問わず全ての銃でパワーは0.98Ｊ以内
・使用BB弾はバイオのみ（プラスチック、セミバイオは禁止。）
・減速アダプタの使用は禁止
・基本的にブラインドファイア禁止
・レーザーサイトはすべて使用禁止
・ゾンビ行為禁止
その他サバゲアイテムの手投げグレネード、グレネード、クレイモアも使用可能です。

■必要な物、参加費

・参加費　カレー付き　3500円
　※カレー不要の人はコメ欄か狐古に申請して下さい。500円引きになります。
・エアガン
・BB弾（バイオのみ）
・ゴーグル
・顔面を保護するマスク、四肢保護のニー、エルボーパッドや手袋があるとベター。
・結構、頭ぶつけると痛い所もあるので、帽子やヘルメットもあった方がベター。

フィールドレンタル品あります。
電動ガン（マガジン、バッテリー込み） ￥2,500
ウエア ￥1,000
ゴーグル ￥500

■参加申し込み期間

とりあえず6月30日いっぱいまでの受付とさせて頂きます。
ご友人の方で参加希望の方いらっしゃればお気軽にご相談下さい。

■服装について

服装は自由です。動きやすく汚れても良い服装でお願いします。
ただし、赤や黄色が目立つ服や靴、グローブ、帽子はご注意ください。
チーム分けの色が赤と黄色ですので、誤認される可能性もありますので、
現場でそれらを外して頂く場合がございます。
またBB弾は当たると痣が出来る程度には痛いので、厚手の布地のお洋服や
フェイスガード、露出度控えめの物をお勧めします。
基本的には自己責任なので、自由にどうぞ。

■ゲーム時間と第一ゲームまでの流れ

時間は09：00~17：00の予定です。
08:00にはフィールドゲートが開き、駐車場に入れるようになります。
終了は17：00、完全撤収。

■交通手段について

基本的には現地集合現地解散。レンタカーや車の乗合等、各自自由にやっていただければと。
電車の場合は、京成電鉄の「京成酒々井」駅が最寄りで、そこからタクシーになるかと思いますので、地図を持参して行くとスムーズかと思います。

■ゲーム内容

基本的にはフラッグ戦メインです。
ゲームルールによっては、セミオート縛り等もあります。
ハイペースでゲームを進行いたします。スムーズな進行にご協力ください。
「補給が間に合わない」「疲れた動けない」等は遠慮せずに休憩してください。30分もすれば次のゲームに入れます。

■お昼ご飯、お弁当について

基本としてはフィールドでカレーライスが出ます。（参加費に含まれてます。）
ですので、昼食のカレーが要らない方はコメント欄等で申告して下さい。
コンビニ弁当持参もOK。

■ちょっとしたお願い　…けっこう大事

お仕事の都合等やその他諸事情で参加を伏せたい人も多いと思いますので、開催中や終了後は、
・個人単位で『今日サバゲして来た』と言うのは自由で。
・誰が参加していたかに関しては『触れない』方向でお願いします。
・TwitterやMixi等で『○○さんお疲れ様でした～』も出来るだけ空気読んで下さると助かる人も多いかと思います。
（自分が参加していた事を表明する事に問題ない人同士のやり取りについては特に問題ありません）
・ですので、基本的に本ページコメ欄使って頂くのを推奨します。

■雨天の場合■

前日、午後0：00の天気予報にて判断いたします。

その他にご質問等ありましたら、コメント欄か狐古までどーぞ。
