注意点

40名以上の登録が発生した場合

40名を超えた時点でお店側に連絡しますので、会場の拡張が可能な場合は定員を50〜60名程度まで引き上げます。

ただし、ギリギリに急増しても対応しきれない可能性が高いです(特に3日前を過ぎた後は食材の都合もあります)。お早めのご登録をよろしくお願い致します。

登録・キャンセルについて

お店側の都合上、ある程度人数を確定したいので、３日前の14時までにAtndへの登録をお願い致します(3日前までの段階での人数分で食材を発注するそうです。)

3日前の時点で定員を登録者数と同じにし、以降はお店側と調整しながら定員を広げる形となります。

尚、当日申し込みは若干名が限界となります。早めの登録をお願い致します。

また、キャンセルも同様、３日前(8月17日)の14時までにお願い致します。
3日前の14時までは、キャンセル料は発生致しません。
3日前の14時を過ぎると、最悪キャンセル料(全額)が発生する可能性があります。

当日の午後以降のキャンセルにつきましては、下記の例外を除いてキャンセル料(全額)が必ず発生いたします。ドタキャンについても下記例外を除き、後日請求となります。お気を付け下さい。


  • 当日申し込み等でキャンセルの分を穴埋めできる場合

飲み物1杯目について

基本的にビール(発泡酒)でお願いしていますが、ビールが苦手な方、未成年の方は事前に教えて下さい。カクテル・ソフトドリンクを人数分お願いします(1杯目のカクテルはカシスオレンジ・ソフトドリンクはオレンジジュース固定になります。また、もしかしたらビール以外は1杯目ソフトドリンクのみになる場合もございます。その時はご了承ください。尚、2杯目からはご自由に注文可能です。)

未成年の方の飲酒について

法律で決められておりますが、未成年の飲酒はご遠慮ください(お酒は20歳になってから)
1杯目につきましても、ソフトドリンクを希望者分注文いたします。

お酒について

一気飲み等はおやめください。
他人にお酒を強要することも禁止です。
お酒のマナーを守りましょう。
