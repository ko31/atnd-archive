日本Androidの会秋葉原支部ロボット部　第29回勉強会を開催します。

+ 開催場所はちっちゃいものハウスです。ちっちゃいものくらぶの秘密基地です。リノベーションが終わり２回目のお披露目となります。

ケミカルでイノベーションを起こす話

RaspberryPiの本を執筆した話

まだまだ人気の衰えないRaspberry Pi。今回ヘッドレスに特化したハンドブック的に使える書籍を作りました。電子工作入門といいながら玄人にも役に立つ内容を目指しています。今回は書籍執筆に伴い開発した、小学生がRaspberry Piを使うためのソリューションを紹介。

Linux を組み込みに使った時のext4ファイルシステムのハック

頭の痛い組み込みLinuxのファイルシステム。roとか、RAM上に展開する手法が使えればいいのですが、実ストレージをrwでマウントしないといけない場合のHACK。

LPC810でリモコンロボットっぽいもので、ワークショップをやったよ

Raspberry Pi のフレームバッファで動画再生

GPIO から omxplayer を使うデモ。

海外のWiFi/Bluetoothモジュール動向

海外では、中華bluetoothチップとWiFiチップを使った、安いモジュールが流通しています。技適を通っていないので日本では使えませんが、海外ではどのような動向なのかを紹介します。

  • bluetoothシリアルモジュール
    • Arduinoオートリセット
    • I/Oポート制御
  • WiFiシリアルモジュール
    • ATコマンドで接続、IP通信
    • SDKで、WiFiチッププログラミング
