description

数学の社会における応用は数あれど、暗号にも数学が大きく関わっているということをご存知ですか？実は、皆さんがクレジットカードなどで安全に買い物が出来るのも、実は数学のおかげなのです。この講座では暗号がどのように発展してきたのかを、暗号の歴史を追いながら皆さんと共に学んでみたいと思います。暗号の歴史を学ぶことは、数学的にも当然面白いのですが、登場してくる人物たちの人生も波乱万丈で、ドラマティック。伝記を読むような感覚でも楽しめます。

学ぶことで得られるもの

１）社会に応用されている暗号の基礎知識について学べます。
２）暗号の発展の歴史について深く知ることが出来ます。
３）簡単な暗号なら、すぐに解読できるようになります。
