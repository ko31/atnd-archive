以下、残骸

RubyとGoで書かれたマルチ言語PaaSであるCloud Foundryについての勉強会です。

Cloud Foundryに関する発表ならなんでもアリな勉強会です。

Cloud Foundryの基盤そのものの話だけでなく、

  • Cloud Foundryを自分の○○環境で構築してみた！
  • Cloud Foundry上でこんなアプリケーションをデプロイしてみた！ などという、使ってみた系の体験談も大歓迎。
