description

【習得できる知識】
中国における労働環境の現状・課題を認識してもらい、人事・総務面から見た対応策の具体例を知っていただく。同時に、中国人のものの考え方を理解していただき、中国人を活かすマネジメント手法を習得して頂く。

【講座の趣旨、ポイント】
日本人と中国人の違いを、在日華僑の視点で具体的に説明します。中国人が普段明かさない日本人観も説明し、より深く中国人を理解して頂きます。また、さまざまに変化している中国の労働環境を、１．中国の所得倍増や法律による労働者保護などの外部環境面、２．工場内の労働力の変化など内部環境の視点で解き明かし、それにより発生する課題の対応策を人事面・総務面から事例として紹介します。合わせて、中国人をどの様に活かすかのマネジメントプロセスも具体的に説明します。実務に直結した内容で、すぐにでも役に立つ実践的な講座となればと思っております。

【プログラム】
１．中国における労働環境の現状と課題
　　1-1.所得倍増計画や西部・内部開発による沿海部の労働市場の変化
　　1-2.法律による労働者保護の動き
　　1-3.人口調査などから見る労働市場の変化
　　1-4.中国沿海部における人事上の共通の課題

２．ビジネス成功の大前提　～相互理解をするために～
　　2-1.日本人と中国人の違いを理解する
　　　（１）日中間における組織と個人のとらえ方の違い
　　　（２）効果的な企業運営を阻害している要因
　　2-2.中国と中国人を理解する
　　　（１）「万里の長城」に見る間違った中国観
　　　（２）日本人と中国人の感覚の違い―距離・時間・雑談・リスク・バランスなどにおける感覚の違い

３．中国で成功するための具体的方策
　　3-1.人材育成管理についての体系的整理
　　3-2.人材育成のシステムづくり
　　　（１）人材育成の総合計画づくり
　　　（２）キャリア開発
　　3-3.課題に対する人事面での対応策
　　3-4.課題に対する総務面での対応策
　　　　中国工場経営における食堂管理の重要性
　　3-5.中国人をどのように活かしマネジメントするか
　　　（１）自分自身のミッションを明確に発信すること
　　　（２）中国人の日本人観を知ること　―　徐福東渡について　―
　　　（３）中国人を活かすマネジメントプロセスを形成すること
