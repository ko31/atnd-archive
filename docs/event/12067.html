【プログラム】

1　画質評価技術の問題点と解決の方向性
　1.1　画質評価法の種類と特徴
　1.2　画質評価の問題点と困難さ
　　1.2.1　主観/客観評価法と問題点
　　1.2.2　画質と生体反応
　　1.2.3　デジタル画像の画質劣化の特徴)
　1.3　画像機器出力画像の変調構造解析(画像変調の観点から)
　　1.3.1　パルス振幅変調(PAM)画像としてのソフトコピー画像解析
　　1.3.2　パルス変復調(PWM)画像としてのハードコピー画像解析
　　1.3.3　視覚系における画像復調機構

2　視覚系の画像復調特性に基づく画質評価法とそのモデル
　2.1　視覚系の適応的画像復調特性
　　2.1.1　眼球での適応的画像観測特性;焦点調節特性
　　2.1.2　視覚系の時空間適応的視知覚応答特性
　2.2　画質評価に必要な視覚特性
　2.3　視覚系の画像復調機構のモデル化
　　2.3.1　数理モデル
　　2.3.2　視知覚応答特性の理論的再現
　　2.3.3　モデル正当性の検証

3　実際問題へ適用した場合の各種評価法の性能比較
　3.1　疑似中間調画像に対する鮮鋭さ評価への応用
　　3.1.1　鮮鋭さ評価尺度に必要な条件
　　3.1.2　画像伝送帯域幅依存特性の再現
　　3.1.3　視距離依存特性の再現
　　3.1.4　エッジ強調画像に対する鮮鋭さ評価
　　3.1.5　各種従来評価法の性能比較とその性能限界
　　3.1.6　画像復調機構を組み込んだ鮮鋭さ評価法と従来法との性能比較
　3.2　画像ノイズ(粒状性)評価への応用
　　3.2.1　ハードコピー画像ノイズに対する視知覚応答の再現
　　3.2.2　粒状性評価尺度
　　3.2.3　電子写真方式/インクジェット方式プリンタのノイズ評価
　　3.2.4　ウィナースペクトル法(従来法)の性能とその限界
　　3.2.5　画像復調機構を組み込んだノイズ評価法と従来法との性能比較
