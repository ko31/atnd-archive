講演概要

・Internet Explorer 9 + HTML5 (マイクロソフト　ジニアス平井さん)

このセッションではInternet Explorer 9 の新機能について紹介します。
また、IE9 の対応でますます加速する HTML5 / CSS3 についても解説します。

・Webの限界を超えるリアルタイムWeb（.NETラボ 高尾哲朗さん）

IE 9の登場でいよいよ本格化してきそうなHTML 5ですが、その目玉は「video、audioタグ」、「CSS3対応」、「WebSockets」です（文書構造化なども新機能ですが、目玉機能としては少々弱いようです）。前者の2つは、Web記事も多くIE9の紹介でも多くの情報が提供されていますが、WebSocketsはIE 9で今後サポートするかどうかは未発表です。
WebSocketsはHTML5仕様策定初期に標準化範囲に入っており、双方向通信を実現する技術ですが、Silverlight + WCFではかなり以前（SDK3）から双方向通信を実装する技術が提供されていますので、実装手順をご紹介します。

・「忙しい人のための　かそうかにゅうもん」（山本美穂さん）

仮想化とかクラウドとかよく聞くキーワードなのですが、実際どういう仕組みでどんなメリットがあるのかということを、忙しい人のために駆け足でおはなしします。

・はじめてのVSTO（鈴木孝明さん）

Visual Studio Tools for Officeを使ったMicrosoft Officeソリューション (アプリケーションレベルのアドイン) 開発について、デモを中心に簡単に説明します。
どんなことができるか、どうやって書くかなどの雰囲気を味わっていただければと思います。
