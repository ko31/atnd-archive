Windows Phone 8 SDK の開発に関するブログを投稿する Advent Calendar。
とりあえず言語問わず募集してみました。

参加方法とルール

このイベントに参加します。
コメントで何日に参加するか指定してください。
すでに参加されている方は当初の予定通りにうめていますが、変更を希望される方はコメントもしくはTwitterでお知らせください。
また複数回の参加OKです。（というより参加人数が足りていないので大歓迎です）
その場合もコメントにてお知らせください。
その日の0時0分から23時59分までの間にブログを投稿して、コメントかTwitterでつぶやいてください。
ハッシュタグは #wpdev_jp をお借りします。

管理人が表をうめます。
