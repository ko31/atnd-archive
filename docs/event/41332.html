ゲームエンジン Unity を使ってシンセサイザーを作ってしまおう！という試みです。

7/10 15:00 更新 – 参加者枠を 24 人にまで拡大しました

内容

Unity には wav/aiff ファイルを使って音を鳴らす仕組みがありますが、今回はそれらを一切使いません。JavaScript を使って、簡単な計算だけですべての音を合成（シンセサイズ）します。

このワークショップでは次のようなことを解説します。

  • 波形ってどうやって計算するの？
  • ドレミのド、は何 Hz ？
  • 波形が変わると音も変わる。
  • エフェクトで音を豪華に変身！

簡単な JavaScript をいくつか組むことになりますが、プログラムの経験が無い人に向けた解説となっていますので、その辺りの経験は無くても全然オッケーです！

ワークショップの最後には、PC のキーボードを使ってドレミを奏でられる簡単なシンセサイザーが出来上がる予定です。
