「ソーシャルイノベーション」をテーマに2012年度のカンヌ国際映画祭でもワールドプレミアセールスされたドキュメンタリー映画「LIGHT UP NIPPON」の上映と慶應SDMの研究科長の前野隆司先生とLIGHT UP NIPPON代表の高田佳岳さんのトークセッションを行います！

学術界からハーバード大学の訪問教授も務めた慶應SDM研究科長の前野隆司先生、実務界からIPOしていくような上場企業にも匹敵するインパクトを世の中に与えたLIGHT UP NIPPON代表の高田佳岳さんのソーシャルイノベーショントークは必見です！！

趣旨
慶應SDMが大きな柱の１つとして掲げるイノベーションについて、3.11の震災後に東北の被災沿岸地域を中心に世の中にソーシャルイノベーションを生み出したライトアップニッポンを題材にして、学び、議論をすることによって、ソーシャルイノベーションについて考える機会を提供します。
2012年のカンヌ国際映画祭でもワールドプレミアセールスされたドキュメンタリー映画ライトアップニッポンを上映し、その活動や、そこで生み出されたこと、世の中へのインパクトを理解します。その後、ライトアップニッポンの高田代表と慶應SDM前野教授のトークセッションを通じて理解を深めます。最後に、参加者でソーシャルイノベーションについて議論します。

【略歴】 前野隆司さん

1984年東京工業大学工学部機械工学科卒業、1986年東京工業大学理工学研究科機械工学専攻修士課程修了、同年キヤノン株式会社入社、1993年博士（工学）学位取得（東京工業大学）、1995年慶應義塾大学理工学部専任講師、同助教授、同教授を経て2008年よりSDM研究科教授。2011年4月よりSDM研究科委員長。この間、1990年-1992年カリフォルニア大学バークレー校Visiting Industrial Fellow、2001年ハーバード大学Visiting Professor（訪問教授）。

高田佳岳さん

2001年東京水産大学卒業、ダイビングインストラクターを経て、2002年に東京大学大学院海洋研究所に入学。修士課程でロシア北極圏に駐在し、海洋生物と北極圏の海洋汚染について研究。2005年同大学院終了後は博報堂に勤務。2013年に博報堂を退職後、株式会社ハレを起業する。2011年、被災地10数カ所で、鎮魂と復興の祈りを込めた花火を一斉に打ち上げるイベントLIGHT UP NIPPONを主宰、一般社団法人LIGHT UP NIPPON代表理事。
