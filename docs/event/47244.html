Startup Live!を2014年2月1日(土)開催します。
　

■開催背景

2010年以降ソーシャルメディア、スマートフォンなどの各領域の拡大や日本のベンチャーキャピタルの投資規模拡大により急速に立ち上がるベンチャー企業が増加して参りました。以前に比べ、大企業から起業する人材層や、大企業から数名から数十名のベンチャーの経営幹部となるケースも増加しております。
それらの背景を踏まえ、スタートアップに特化し、成長ベンチャーの社長、CTOと直接会えるStartup Live!を開催します。当日は成長著しいベンチャー企業の社長、CTOが5社登壇します。

■当日概要

・イベント名： 成長ベンチャーの社長、CTOと直接会えるStartup Live!
・日時：2014年2月1日13～17時(12時半開場)
・会場：リクルートキャリアアカデミーホール（グラントウキョウサウスタワー）
東京都千代田区丸の内1-9-2
・人数：50名規模 エンジニア職(アプリ・インフラ問わず)
・主催：リクルートキャリア
