語ろう！Web制作者のキャリアパス

説明

昨今、Web業界はものすごいスピードで変化しています。
HTML,CSS,JSで多様な表現ができるようになったり、バイラルメディアが次々に誕生したり、IoTが盛り上がってきたり……。

そんな中で、Web製作者（Webデザイナー / コーダー / フロントエンドエンジニア)の皆さんは将来どうなっていたいですか？
スペシャリストとして技術を極めるのか、多彩な能力を発揮するジェネラリストになるのか、はたまたジョブチェンジするのか、フリーランスになるのか、起業するのか…etc。

そんな疑問を、かたや社員として、かたやフリーランスとしてWeb制作の最前線でご活躍中のクリエイターにお聞きしちゃいます。

HTML5, CSS3,JavaScriptなどのフロントエンド技術はもちろん、制作の環境や雇用の形態、将来の目標までをパネルディスカッション形式でざっくばらんに語り合います。

また、イベントの後半では参加者の皆さんおよび、開発者と大会王者の二人を交えてHTML5カルタの実演も行います！

これからこの分野でキャリアを形成していきたい人も、すでにキャリアを形成してきた人もどちらの方も楽しめる内容を目指します。

登壇者

堀口 誠人

(株式会社LIG フロントエンドエンジニア)

2012年に株式会社LIGに入社。
マークアップエンジニア、アシスタントデザイナーを経て今に至る。
「まだ俺本気出してないから」が口癖だったが、2014年 第一回HTML5カルタ大会 優勝 / 2014年 第一回ZenHack 優勝など、2014年辺りからようやく本気を出し始める。
2015年からはオンライン授業サービス「スクー」にて講師を務める。

酒井 優

(フリーランス Web制作者 / 講師 / WEBCRE8.jp主催者/ HTML5カルタ開発者)

1980年8月生まれ フリーランス。
Webサービス運営会社でWebデザイナーからディレクションまでの仕事を経験したのち、現在は受託制作と講師業の二つの形でWeb制作業界に関わっている。
Web制作者向けブログ「WEBCRE8.jp」を運営するほか、Web制作者向けのHTML5学習ゲーム「HTML5KARUTA」の制作・販売を行う。
HTML5の技術者コミュニティー「html5j」や日本ディレクション協会などのコミュニティーに所属しつつ、自身でもセミナーイベントやWebを使った企画などを幅広く行っている。

hireLinkとは

hireLink（ハイアリンク）はクリエイターとエンジニアのための無料勉強会 / 交流会です
その名前には"hiring"=雇用と"higher link"=志の高い者同士のつながりという意味が込められています
成長企業のエンジニアやクリエイターを招き、パネルディスカッション形式で最新トレンド技術を学習
勉強会後の交流会では企業代表と直接お話しいただき、就業やキャリアアップにつなげていただけます

就職への最短コース

hireLinkはただの勉強会ではありません。
勉強会アジェンダは企業が今まさに求める人材ニーズを反映しています。
企業担当者とのマッチング次第では、その場であなたの就業が決定することも。
20 - 30人規模の勉強会だから人混みの中で独りぼっちになることはなく、必ず参加の機会があります。

絶品コーヒーをどうぞ

会場には出張焙煎人永井健司氏によるカフェが出店。
芳しいアロマにおもてなしの心を添えて一杯一杯丁寧にドリップします。
美味しいコーヒー、カフェラテと軽食がきっと緊張感を解きほぐしてくれるはずです。
