catch 弱者の交渉方法
description

交渉に困るときって、どんな時でしょう？
ズバリ、「自分の立場が弱いとき」なんですよね～。
自分の立場が弱く、しかも先方に「悪い印象を与えたくない」場合には、強気に出たり、条件をふっかけたりできないですよね。

例えば、会社の上司に条件をふっかけたりできるでしょうか？
それが「図々しい奴」などの悪印象であれば、今後にも影響してきます。

巷にある交渉本は、対等の関係か、強気に出られる1度切りの関係での話がほとんどなんですよね。
「まず最初はふっかけろ」みたいな。
そこで今回は、実生活で一番困る「自分の立場が弱いとき」の交渉方法についてお教え致します！
これを聞けば、弱い立場でも「満足する」交渉ができるようになります！
是非のご参加お待ちしております。
