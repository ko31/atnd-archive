description

Noritsuna氏はOESF(Open Embedded Software Foundation)の常任理事をされているかたで、現在台湾に在住です。以前は日本Androidの会の事務局長をされていました。今回は、どういうわけか広島に用事があるらしく、愛媛にも立ち寄って頂くことになりました。

せっか愛媛に来られる貴重な機会ですので、Noritsuna氏を囲んで勉強会のようなものを企画してみました。特に内容は決めておりませんが、OESF、組込系Andoridの標準化の話、Linaroの話をして頂ければと思っております。台湾といえば今ちょうどアジア最大のコンピュータショーCOMPUTEXの開催中ですよね。そのあたりの土産話も伺えればと思っております。

取りあえず、以上のようなゆるい内容ですが、コメント等にご要望等入れて頂ければなるべく皆さんのご希望に添えるようにしたいと思います。
Noritsuna氏もきっと見てくれていると思います。

懇親会ですが、当会場から徒歩15分くらいのところに弊社道後事務所がありますので、そちらで行いたいと思います。ただ駐車場がないので、車でお越しの方は大学に置いて頂くか、各自有料駐車場を探して頂くことになります。
食べ物・飲み物は各自持ち込みですのでよろしく御願い致します。
或いは近くのスーパーに買出しに行った場合、いつも割り勘で千円前後です。

懇親会場ではプロジェクタを用意しますのでミニプレゼンとかも可能です。
ただ、ネット環境を使う場合は各自で設備をご用意下さい。

懇親会の開始時間ですが愛媛大学での勉強会後、道後温泉で一風呂浴びて懇親会に突入ということになるかもしれませんが、一度事務所にお立ち寄り頂ければタオル等はこちらでご用意いたします。

日曜日ではありますが、楽しくためになる勉強会にしたいと思いますのでみなさまのご参加をお待ちしております。
