Seleniumについて

Webアプリケーションの自動テストを実施するにあたって、より厳密なEnd-to-Endテストを実施するためにはブラウザの操作を自動化できることが不可欠です。
Seleniumはブラウザの操作を簡単に自動化できる、世界中で利用されているツールです。

内容

まずはSeleniumとは何ぞや、というところを簡単にご紹介します。
その後、ハンズオン形式でSeleniumを使ってブラウザの操作を自動化する簡単なプログラムを作成します。

サイボウズについて

サイボウズでは、大規模Webアプリケーション開発の一環として、自動テストを積極的に取り入れています。
エンジニアブログ「Cybozu Inside Out」では、自動テストなど様々な技術情報を発信しています。
http://developer.cybozu.co.jp/tech/
