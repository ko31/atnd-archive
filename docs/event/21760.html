講師

有川　智浩 Link
大学卒業と同時に国内生命保険会社に入社。その後、複数の外資系生命保険会社において、人事部門や法務コンプライアンス部門の部長職を歴任。
2005年から昨年まで、執行役員として人事総務部門と法務コンプライアンス部門を統括。「給与制度や人事評価制度などの各種人事制度の新設や改廃」「人材教育体系の構築や社内の階層別研修の実施」「各種法令対応」「監督官庁対応」などに携わる。
昨年までの約10年以上にわたり、新卒採用の責任者として10,000人を超える学生の採用選考に関与。キャリアコンサルタント。
