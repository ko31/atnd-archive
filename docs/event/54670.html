description

習得できる知識
腐食機構の基本的な理解，電気化学の基礎，淡水の水質化学
防食法の概要，水処理による防食

趣旨
水溶液腐食のメカニズムを基礎から平易に解説する。水の存在により腐食はなぜ促進されるのか，水質は腐食にどのように影響するのかなどを考えるとともに水処理などの実際の防食技術について事例を通して学ぶ．

プログラム
１．金属はなぜ腐食するのか
　　1-1　腐食とのかかわり
　　1-2　腐食の基本的考え方
　　1-3　腐食の電気化学

２．主な金属材料と腐食形態
　　2-1　鉄鋼材料
　　2-2　非鉄材料
　　2-3　主な腐食形態

３．水質および腐食性環境の影響
　　3-1　水の循環と淡水の水質
　　3-2　水質と腐食
　　3-3　大気環境における腐食
　　3-4　土壌・コンクリートにおける腐食
　　3-5　微生物腐食

４．腐食を防ぐためには
　　4-1　防食法の分類
　　4-2　環境の制御による防食
　　4-3　表面処理
　　4-4　電気防食
　　4-5　防食設計・施工と防食管理
