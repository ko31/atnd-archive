開催概要

このページは、Global Game Jam 2017 ギークハウス京都東福寺の参加申し込みページです。

本会場は48時間フルオープンとなっています。
参加費は無料です。

タイムスケジュール

時間 予定
20日(金) 18:00〜19:00 開会式・テーマ発表
　　　　19:00〜20:00 全体自己紹介・チーム分け
　　　　20:00〜21:00 チームブレスト
　　　　21:00〜 一旦終了。残る人は開発開始
21日(土) 9:00〜 開場
　　　　16:00〜17:00 取材が来る(1時間ぐらい)
　　　　19:00〜20:00 α版発表会

22日(日) 9:00〜 開場
　　　　15:00〜16:00 開発終了・作品のアップロード
　　　　16:00〜17:00 発表会
　　　　17:00〜19:00 試遊会・懇親会
　　　　19:00〜 終了

Global Game Jam 2017 とは

週末に世界各地の会場に集まって、ゲームをつくろう。
Innovation. Collaboration. Experimentation.
Global Game Jam（以下、GGJ）とは、ギネス記録をもつ世界最大のGame Jamです。

全世界に会場が設けられており、同じ日に同じテーマでゲームを開発するハッカソンイベントです。

参加条件

みんなでわいわいとゲームを製作したい18歳以上の方(18歳未満の方の参加はgeektfkj+gamejam2017@gmail.comまで一度ご連絡下さい。)

注意事項

  • 無線LANをご利用いただけます。
  • 会場内及び建物内は全面的に喫煙・火気・高熱を発するモノ厳禁です。
  • 飲食は所定の場所でお願いします。
  • シャワーもあるし、タオルもあるので、ふとんもあるのでサクッと宿泊できます。
  • ふとんには数に限りがありますので、ご入用の場合は事前にコメント欄などでご連絡下さい。
  • ノートPC等の貸出は行っておりません。
  • コタツもあります。
  • 開発に必要な機材は各自でお持込ください。
  • 開発に使用するツール・アセットは予めインストールをお願い致します。
  • お渡しする名札入れに手書きの名前カードを入れて見えるところに付け、会場内では外さないようにお願いします。
