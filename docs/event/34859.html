★講演内容

スマートデバイスの急速な普及とともに
iOS、Android、Windows 8と様々なモバイルOSが登場し一気に多様化が進んでいます。

この流れのなかで、アプリ開発者が同時に複数のプラットフォームに対応するアプリを開発する手法としてハイブリッドアプリが注目を集めています。

本セミナーでは、アドビシステムズが開発するハイブリッドアプリ開発ツール【PhoneGap】を用いたクロスプラットフォーム対応のハイブリッドアプリ開発について解説を行います。

※内容は変更になる可能性がございます

★講演者プロフィール

■田中正裕 氏
アシアル株式会社　代表取締役社長

1981年 香川県出身。
小学生を米ダラスとロサンゼルスで過ごし、インターネットを知る。
中学・高校時代にはフリーソフトウェアやシェアウェアなどの作成を活発に行う。

2000年に東京大学工学部への入学と共に、
株式会社ホライズン・デジタル・エンタープライズ、
株式会社キュリオシティのアルバイトを経て、
2002年7 月にアシアル株式会社を設立。当時、大学3年生、21歳。

その後、東京大学大学院を中退し、現職に専念。
「超極める！PHP」や「日経ソフトウェア特集」、ITPro「PHPウォッチ」など多数の執筆 / 講演をこなす。
