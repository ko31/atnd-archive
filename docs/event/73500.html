今回のデザイン塾はクリエイターのために人気コンテンツ「夢王国と眠れる100人の王子様」（以下夢100）の開発を行うプロデューサー、ディレクター、クリエイターが直接ゲーム仕様、世界観構築、ファンとの向き合い方など、ゲーム制作・運営について語ります。 ゲーム開発者、これから開発者を志す学生の方におすすめの内容となっています。また、情報交換の場として懇親会・名刺交換の時間も用意しています。

■夢王国と眠れる100人の王子様とは

「夢王国と眠れる100人の王子様」は、「夢王国」を舞台に女の子なら誰もが憧れる「王子様との恋」を描く、新感覚の女性向けスマホパズルRPGです。夢王国に現れた『ユメクイ』によって夢を喰べられ、眠りに落ちてしまった王子様たちと出会い、彼らを目覚めさせることで、甘いストーリーを楽しむことができます総勢100名以上の豪華声優陣が出演し、あらゆるシーンで王子様の声を聞きながらゲームを進めることができます。

http://www.yume-100.com/

■内容

女性向けゲーム制作に特化した組織「GG Studio」においての企画の仕事

『夢100』のプロデューサー、ディレクター、クリエイターが制作事例を交えながら、女性ゲームの世界観、シナリオ、キャラクターの作り方やゲームの作りこみ方などについて語ります。 『夢100』の立ち上げから現在にいたるまでの開発秘話について、企画書などを交えながらご紹介いたします。また、講義後は、皆様からの質問をもとにしたパネルディスカッションも行います！

■「GG Studio（Girls Game Studio）」とは

「GG Studio」は、サイバーエージェントグループの女性向けゲーム開発をする専門組織です。 「女性が本気で楽しめるゲームを創る」をコンセプトとし、 女性ゲームプレイヤーならではのスマートフォンの利用方法や興味関心にあわせたサービス開発を行っています。 また、ゲーム開発への活用を目的として、アンケート調査やマーケティング調査などを実施するとともに、女性向けゲームを創りたい優秀な人材発掘・応援を目的としたコンテストを開催いたします。 今後さらに拡大が予想される女性向けゲーム市場において、女性向けゲームを代表するヒットタイトル創出を目指してまいります。 https://ggstudio.amebaownd.com/
