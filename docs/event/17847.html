description

この度10月16日(日)に神戸市主催の街歩きデザイン都市イベントの一つとしてKIMONO×KONOMIが参画することとなりました！

着物姿で街を歩き、乙仲通り周辺の神戸のモダンな街並みの中にある歴史を感じる建築デザインとギャラリーのアートに触れます。

最後は途中で買い込んだスイーツをみんなで食べながら、当日の感想などの話をしながら交流を深め、人と人が繋がることによって生まれる市民同士のデザインが神戸にはあるというイメージを発信します。

当日巡る神戸スイーツのお店の詳細

・パティスリー モンプリュ
製造スタッフ４人、誰が欠けても、モンプリュのお菓子はできません。
ひとりひとりが、純粋においしいお菓子を作りたいと
全力を傾けて頑張っています。
お菓子作りには人柄が表れます。
純粋で素直でありたいと思っています。と語る林シェフの作るスイーツはとても繊細でほっとする甘美な魅力のあるものばかりです。

・フリュティエ コムシノワ
果物をたっぷり使って、ボリュームもたっぷり・・
ひとつひとつのケーキがとっても楽しそうで、いきいきとした表情をもっています。

・パティスリー グレゴリーコレ
「パティスリー グレゴリー・コレ」は、1998年4月23日に神戸市の元町3丁目にオープンしました。フランス王朝時代から育まれたフランスの菓子文化に現代の新しい豊かな感性を開花させた、日本的アレンジの全くないまさに芸術作品ともいえるデザートの数々を提供し続けています。
