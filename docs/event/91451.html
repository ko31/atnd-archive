description

来読会は自分が読んで面白いと思った本を紹介し合う読書会です。

本が好きな人、読書の楽しさを誰かと共有したい人、他の人のおすすめ本を聞きたい人、興味はあるけど何を読んでいいかわからない人、どなたでも歓迎です。

主催2名を含めて定員7名の小さな会になりますので、リラックスしてお話できると思います。

・日時
　2017年10月15日（日）10:30～12:00
　5分前にはお越しください。
　奥の方のテーブルになると思います。目印を置いておきます。
　途中参加、途中退室は可　事前にご連絡ください。

・場所
　カフェ・ベローチェ　大宮店
　JR大宮駅西口より徒歩5分

・持ち物
　紹介したい本1冊
　時間があれば複数冊紹介して頂くことも可能です。

・費用
　100円（運営費）
　※カフェのため、ワンドリンクの注文をお願いします。

会の基本コンセプトは「読書を通じて未来を考える」ですので、もし紹介したい本がたくさんある時は、選ぶ参考にしてください。コンセプトはあくまで参考ですので、他の人にもすすめたいものならジャンル問わず何でも構いません。特に身構えなくて大丈夫です。のんびりとした雰囲気で進行していきたいと思います。

会の進行は以下の通りです。
①自己紹介②持ってきた本の紹介2～3分③本についてお話
②～③を人数分繰り返します。

注意事項
ネットワークビジネスやマルチレベルマーケティングへの勧誘等が目的の方の参加はご遠慮ください。わかった場合は、今後の参加を禁止します。
「読書会」や「勉強会」を呼び水に、ネットワークビジネスへ勧誘することが多いようです。来読会はそういったもののない、安全で楽しい会を作りたいと思って立ち上げましたので、ビジネス絡みでの参加はやめてください。
