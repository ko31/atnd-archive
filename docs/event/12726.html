体系的にOpenFOAMを勉強する格好の機会ですので、
合わせてご参加いただければと思います。

２，　今回は鹿島技研の挾間さんをお呼びして、咳の飛沫シミュレーションを題材にソースコード解読の基礎についてのお話をいただこうと思っております。既存のコードをどのようにカスタマイズしたのかを中心にお話をいただき、それを通じて本勉強会でも目下の課題となっているソースコードの読み方について勉強したいと思っております。

３，　初参加の方には可能な範囲で自己紹介をしていただいております。
①経歴
②OpenFOAM使用歴
③OpenFOAMで自分が今取り組んでいる課題
④OpenFOAMでこれから取り組みたい課題
をPowerPoint等でご用意ください。
