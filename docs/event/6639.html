title さんだECO DE NIGHT2010
catch えびす町万燈籠
description

初めまして、私どもは兵庫県三田市にある西宮神社三田分社（えべっさん）の者です。
この度、私ども三田分社が町おこしをしようと、さしすせその会を立ち上げました。
キャンドルナイトのような灯りの祭典を３年前よりえびす万燈籠として行なっておりましたが、当初地元の神社のイベントとして行なっておりましたところ、多数の方が見学に来られ、「このような灯りの綺麗なことをされるのであれば、もっと多くの人にこの灯りを広めてほしい」と言う希望が多数寄せられました。
そこで、それならば当分社で何かできないかと考え、三田の町おこしをしていこうと計画を立てました。
他府県からもこの灯りの祭典を見てもらいたいという願いを込めたイベント、「灯りを消してキャンドルの灯りに酔いしれてみませんか」をテーマに「さんだＥＣＯ ＤＥ ＮＩＧＨＴ ２０１０」のイベントを開催致します。
朝は朝市、昼はフリーマーケットや工作教室、電気自動車の展示等、夜はイベントメイン会場では二胡やジャズ、大学生によるよさこい等、沢山の催し物を行ないます。また、サブ会場の神社境内では巫女による古式にのっとった巫女の舞なども行ない、会場をメインに周辺地区へ大規模に拡大した、ローソクの灯りを灯した様々なテーマの通りを楽しんで頂けるようにしておりますので、周囲を散策し、どうぞゆったりとした時間をお過ごし下さい。なお、会場周辺には駐車場がございませんので、各交通機関をご利用いただくか、駅周辺の駐車場をご利用下さい。
さんだ ＥＣＯ ＤＥ ＮＩＧＨＴ ２０１０は、兵庫県三田市三田町の小寺公園にて­朝9時から朝市、10時からは縁日や工作教室、夕方4時頃からはメインの灯りの祭典のオープニングを行ないます。
夕方6時30分より点灯式をはじめます。尚、点灯式典後ご来場の皆様方­にもローソクを持って頂き、沿道沿いに置いてある様々なキャンドルにあなたの手で灯りを着­けて頂く事ができます。
多数のご参加をお待ちしております。
なお、雨天の場合は８月２８日（土）に順延致します。
