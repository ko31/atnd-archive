～データ×クリエイティブ×運用ノウハウで効果を最大化する動画広告の最新トレンドについて～

　

概要

若年層を中心にマス広告だけではリーチできないユーザーが急増し、大手広告主を中心に広告費のデジタルシフト化が止まらない。その中においても動画広告市場の成長率は目を見張るものがある。
動画広告を成功に導くために必要なクリエイティブ×運用についてのノウハウを動画広告市場におけるキーマンをお招きし、最新事例をご紹介いたします。

　

Program

【1】動画広告においてもPDCAを実現させる運用型クリエイティブ制作について

  • 動画広告もPDCAを回すことが成功の第一歩です。第一部では運用型クリエイティブを実現するGMO NIKKO独自のソリューションについてご紹介いたします。
    またこれまでクリエイターのセンスに頼っていたクリエイティブ制作においてデータを元にしてクリエイティブを制作することでユーザーが求めているクリエイティブを制作することができるソリューションについてもご紹介いたします。

　
【２】Secret Contents

  • スペシャルゲスト登壇予定！
    　

【３】動画広告最新事例紹介

  • 動画広告の最新事例についてご紹介いたします。
    年間1億円以上を運用している弊社運用コンサルタントと年間5,000本以上の動画クリエイティブをディレクションしている弊社クリエイティブディレクターが登壇いたします。また今回スペシャルゲストとして大企業を中心にEC、メディアなど多種多様な企業様がご利用されているKaizen Platform日本事業部長筑波様もお招きし本セミナー限定の特別インセンティブもご用意しております。
