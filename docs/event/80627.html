古き良き言葉に豆腐と水は旅させるなとあります。
ならば酒道にも通じるものがあると思います。
物流技術が進んでいる昨今その場に出向かなくても、各地の地酒が手に入る時代であります。
そんな時代だからこそ敢えて、その地に出向き土地の風土を感じその地に想いを寄せて酒を呑む
これこそが、左党最大最大の贅沢ではないか?
と考えこのバスツアーを企画いたしました(´▽｀)
