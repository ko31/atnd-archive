LT is 何？

ライトニングトーク（英: Lightning Talk）とはカンファレンスやフォーラムなどで行われる短いプレゼンテーションのこと。様々な形式があるが、持ち時間が5分という制約が広く共有されている。(Wikipediaより)

先期落とした単位に…黙祷…！！
学内LT 今期4回目の開催です！


今回も持ち時間を「基本5分」として開催したいと思います。
もちろん聞き専参加も大歓迎！
Ustream配信もする予定ですが、学内の人はぜひ直接聞きに来てください！

是非登壇して話したい！って人は @tdusielt_info か、dendai.talklive.info_atmark_gmail.comまで題名を添えてどうぞ！

今回のテーマは「自由」
ポテチの新しい食べ方、開発の進捗、サービスの宣伝、何でも結構です！
