Mashup Awards 9特別企画

合宿型Mashup Camp Supported by Startup Weekend Japan
〜この夏最大のHackathon〜
※実際には泊まりません

毎年夏の終わりから秋にかけて開催される日本最大級の開発コンテスト、「 MashupAwards 」。

この夏Mashup Awardsは新しい挑戦を行います。
Mashup Awards9（MA9）が開催される直前に、
MA9の本選とも連動をするMA史上最大規模のハッカソンイベント
を開催することとなりました。
※MashupAwards9に応募するための作品を3日間で作るという、MA9のプレイベントになります。
※運営協力として「週末起業体験」Startup Weekend Japanにご参加頂く事になりました！ Startup Weekend Japan

7/19-21の３日間かけて、チーム結成から、アイデアソン、ハッカソンと、盛りだくさんの内容を準備しております。
本プレイベントにて「最優秀賞」を勝ち取ったチームには豪華な副賞をご用意しております。

副賞（予定)
開発支援金として賞金10万円
MA9の一次予選の免除の権利

毎年のMashupAwardsに興味があるけれど、

・仲間が見つからない
・かたまった時間が取れない
・切磋琢磨するライバルが欲しい
・参加企業の方とリアルに交流する場が欲しい

等の理由で参加出来なかった方、見送られた方、、
ぜひ、この機会にご参加下さい！！(`・ω・´)

今回は、なんと、、、 特別企画 と言う事で、

全て 無料 で参加頂けます！
開催期間中の食べ物、飲み物お出しいたします！！

ぜひMashupAwardsの前哨戦としてご参加下さい！(`・ω・´)
個人での参加、チームでの参加どちらでもOKです！

よくある質問

  • 何をするイベントなの？
    MA9事前企画として、3日間で応募作品を作るイベントになります。
  • LANや電源は使えますか？
    会場ではゲスト用無線LAN・電源が利用可能です。
  • 参加者の条件は？
    特にございません。MashupAwardsに参加頂いた方も、頂いてない方もみなさまご参加頂けます。
    未成年の方は夜間の参加は制限させて頂く場合がございます。
  • どのような職種の方が来ますか？
    エンジニア・開発者・デザイナーの方が多いと想定しております。
  • 開発できなくても平気なの？
    開発者の方とチームを組んで頂きますので大丈夫です。
  • 友達と参加してもいいですか？
    ご友人との参加も大歓迎です。同じチームになりたい場合、当日お申し出ください。
    ※ご友人の方もATNDで参加登録をお願いします。
  • 持ち物は？
    開発、作品制作に必要な機材、準備をお願いいたします。
