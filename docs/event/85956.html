毎月第３水曜日は、「TECH BAR」と称して、キャッシュオン形式のワンコインBARを運営します。

◆ TECH BAR とは？ ◆ =====

ＩＴ業界に携わる方々限定で、気軽に情報交換や、お仕事の話をできる場所を提供しています。

19時オープンとなりますが、通常のBAR同様、好きなタイミングで出入りができるキャッシュオン形式のBARスタイルとなっております。

ビール、ワイン、梅酒、ジン、ウォッカ、カシス、ソフトドリンク…etc

各種ドリンクを揃えておりますが、一部ＩＴ業界用語のオリジナルメニューをご用意します。

案件のご相談などございましたら、その場で御見積や相談もできる会ともなっております。エンジニアもたくさんいらっしゃいますので、技術についての相談も可能です。

◆ こんな方にオススメ！ ◆ =====

◯勉強会だけでなく、お酒片手に交流したい方
◯IT業界の最新情報を収集したい方
◯とりあえず安いし、飲みに来たい方

◆ 本イベントを楽しめるように皆様へお願い ◆ ====
本イベントはIT業界に親和性が高い業種、および隣接する業種の方々向けの会となります。新しい出会いが生まれると思います。社会人としての心構えでお願い致します。

①お名刺１枚と参加費を丁度ご用意の上、受付にお並びください。

お名刺は多めにお持ちください。30〜40枚程度あればよろしいかと思います。
