「業界動向、マイクロソフトとHTML5の関わり 」

＜内容＞
第一部：「マイクロソフトを取り巻く業界動向」
スマートフォン、タブットが世に登場してから日常生活が一変した人も少なくない、そんな状況が生まれています。マイクロソフトと言えば、Windows・Office・マウス そういう人が非常に多いかもしれません。中の人から見たマイクロソフトと、これまでの歩みと、現在の取り組みについてお話しさせていただきます。

第二部：「マイクロソフトとHTML5の関わり」
Internet Explorer 6 が登場してから10年の歳月が流れました。10年前、皆さんは、どのようなコンピュータを使用されていましたでしょうか？この十年を振り返りながら研究成果と、どのような想いでHTML5のような、今後の Web 標準に臨んでいるかをお話しできればと思います。Internet Explorer 9、10 そして 次期 Windows と今まさに技術の大きな転換点を迎えています。
