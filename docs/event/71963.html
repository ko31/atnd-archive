description

マーケティング発想を実学として習得します。実例をベースに、マーケティングビジネスの構造(領域と機能)と、いま、起こっている環境変化、その影響を理解すること。そして、その中核機能となる"共感"を洞察するプラニングスキルを、実践演習を通じて身につけること。さらには、新しい価値の再発見と可能性を描く捉え方のイノベーションにチャレンジします。
