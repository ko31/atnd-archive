1. MongoDB 2.4の注目の新機能ハンズオン

楽天 窪田さん @crumbjp

MongoDB 2.4で、注目の3つの新機能をハンズオンします。
・Working Set Analyzer
・$setOnInsert
・capped array

楽天 下地さん @nntsugu
