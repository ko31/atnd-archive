概要:

まっさらな状態から 開発環境の構築 〜 割り勘アプリの作成 までを1日で行います。Java(dalvikですが)の基本的な知識も"アプリを作りながら"合わせて習得。当日の講座ではあまり扱いませんが、単体テストやエラー処理を考えるのにとても適したアプリを作ります。参加費は無料です。

当日Androidテスト部第5回MTG が同じ場所で開催されますので、よろしければそちらもどうぞ。
http://atnd.org/events/13299

お願い:

Androidアプリを1本でも作ったことのある方には本当に得るものが一切ありませんので、どうか初心者さんに席を譲ってあげてください。

講師/初心者補助:

@snsk
@miyatay
@tmk_bata (feat. #jagy) *前半補助/後半生徒
@sassy_watson

対象とする受講者:

何らかのプログラムを1行以上書いたことのある人。

例:print(“hello perl”); や document.write(“hello javascript”);とか #include int main(){printf(“hello c¥n”);return 0;} とか。仮に、public class Hello { public static void main(String[] args){System.out.println(“hello java”);}} だったりすると尚可。動かした経験があると更に可。

1本作ったことあれば最高てゆかそういう人は初心者補助願います。もしそれがAndroidアプリの方は得る物無いので席譲ってあげてください。

進め方:

初心者度合いによって朝から段階的に進めます。メニューを見て好きな時間帯から参加OK。

時間 内容
9:00-13:00(4.0h) AndroidSDK(開発環境)の構築とDDMSでFizzBuzz
13:00-16:00(3.0h) 「割り勘計算機アプリ」を作ってみよう(Activity, Intent, 静的/動的Layout, 基本UI 部品の制御)
※ここから来る人はAndroid開発環境構築とHelloWorldまで終わっていること
時間があれば 万が一時間余ったらドラアプリで音を鳴らして遊ぼう！

※環境構築に自信のある方は9時じゃなくてもOKです。お昼の時間は各自適当に。
前半は13:00の段階で参加者全員が環境構築できていることを 全員で 目指します

持ち物:

CPU: 1.2Ghz RAM: 1GB 以上 OS: WindowsXPまたはMacOSX10.5 以上 のノートパソコン。

※ネット接続環境があればなお可。無くても大丈夫です。
※もし出来る方はJDK1.6をインストールしてきてください。

備考:

ネット接続環境が無い方のために、開発環境の構築に必要なものは基本的にUSBで配布します
