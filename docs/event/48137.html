「フィギュアスケートに女子シングルがあるなら、結婚にも女子シングルがあってもいいんじゃないかって（笑）」

地味で冴えないOL、BAVAAAL(独身)は焦点の合ってない目でそう語った。
ウェディングドレス着てみたいし、ブーケトスだってしてみたい。
でも私もいい年、夢見る少女じゃいられない。
なら一人で結婚式するしかないじゃない…！

「大丈夫！わたし、ばばーるさんのために司会するから！」

ポールダンサーmecavはアクロバティックな技の数々を封印、マイクを握り締め、読めない漢字にルビを振る。

「ばばーる氏が一人で結婚式ですって…？」

世界的バーレスクダンサーでばばーるの盟友・Cherry Typhoonは、本拠地カナダから緊急帰国。

「独身だったら、ばばーるにプロポーズしたいくらいだよ（笑）」「もう！誠ちゃんたら！」

媒酌人を務めるは宝塚のエロフェッショナル・ビート・サイエンティスト、佐伯誠之助と妻・幸代。

「ばばーるちゃん、ええ感じやで！」

新婦を生暖かく見つめる竹ち代毬也は奇妙な動きでウォーミングアップ開始。

「まさかエア結婚式とはね。」

エアギター日本一のタイトルを持つチョコバット鵜飼はチューニングを始める。

「花嫁NTRモノのエロ漫画はねえか！スケベな新妻モノはねえか！」

ANNAリスクのメンバーはエロ本を拾いに鴨川河川敷を散歩する。

「…俺を選べよ、ばばーる。」

友達だと思ってた男肉 du Soleilが急接近!
触れる指先、見つめあう目と目。
らめぇ、頭が沸騰しそうだよぉ！

さらにはとんでもない大物シークレットゲストがやってくる…！！！

BAVAAALがお送りする「セクシー冠婚葬祭シリーズ」第二弾！
去年の「セクシー生前葬」に引き続き、笑いあり、涙ありのドタバタ結婚コメディ！
あの鐘を鳴らすのはあなた！

CAN YOU CELEBRATE?

出演

新婦 : BAVAAAL

元々はファイヤーダンサーだったが、近年は「歌って踊れる炎のOL」を名乗りカオスな芸風で己をさらけ出す。
その人生観に共感するオフィスレディーが増えてきている模様。

新郎 : 該当者なし

媒酌人 : 佐伯誠之助・幸代ご夫妻

言葉では言い表せない、神々しいパフォーマンスでオーディエンスを沸かせる誠之助氏と、
「こんな可愛い人が佐伯さんの奥さんなの…？」新婦自身が驚いたロリ妻幸代夫人。
お二人は媒酌人について何をするのか具体的には分かっていない。

司会 : mecav

新婦のマブダチ、ポールダンサーmecav。
今回は類まれなる身体能力と表現力を捨て、あえて司会に徹するはずだが…。
漢字が苦手。
けして若さと人気に嫉妬した新婦がポールではなく司会をさせたわけではない。

？？？ : 竹ち代毬也

新婦が悩んだとき、いつも手を差し伸べ道を説いてくれたピスタチオのおっさん。
竹ち代さんがいなかったらとっくにパフォーマンスとかやめてた。
美しい肉体と独特のセンスによるその舞はまじヤバイ。
尊敬するパフォーマーの一人。

？？？ : Cherry Typhoon

現在カナダを拠点とし世界を股に掛け活躍するバーレスクダンサー。
俳優・脚本家・イベントオーガナイザーなど多彩な分野で才能を発揮する。
まじすごい人。
新婦のパフォーマンススタイルを肯定してくれたパイセン的存在であり、飲み友達でもある。

友人代表その1 : チョコバット鵜飼 fromしゃれこうべ

エアギター2012日本チャンピオンかつ世界ランク6位、向日市在住。
昨年の「セクシー生前葬」で喪主ばばーるの本名を晒したので、しばらくの間シカト
するも基本的にはいい奴なので呼びました。エアじゃないギターも上手い。
ラー油作りを生業とし、文字通り『油を売って』生活している。

友人代表その2 : ANNAリスク

新婦が参加している茶番ユニット。
エロ漫画朗読と、演目の酷さに定評がある。
昨年の「セクシー生前葬」ではANNAリスク茶番のため、喪主ばばーるが自腹でダッチワイフを購入。

攻略キャラ : 男肉 du Soleil

どんなに会社で怒られても、彼らのパフォーマンスで笑顔になれました。
舞台についての確かな知識とサブカル入り混じる熱い演技でメキメキ頭角を現しているダンスカンパニー。
新婦はいつしか彼らの同人誌（18禁・BL）を描くほどまでに魅了される。
飛び散る男肉が新婦の女を呼び覚ます。
