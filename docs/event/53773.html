“心”と“体”の健康にフォーカスした新しいイベント

健康生活フェアでは、従来のような“体”の健康だけでなく“心”の健康にも焦点を当て、最新の医療や健康サービスとその利用者をつなぐイベントです。

当日は帯津良一先生や南雲吉則先生を始めとして、各分野の著名なドクターをお招きし、講演を行います。また、ブースではさまざまな企業が出展し、最新の医療器具、エイジングケアや美容関係のアイテムを体験・購入できます。

さらに、スペシャルイベントとして、代替医療の世界的権威であるディーパック・チョプラ博士の来日記念講演も開催されます。
