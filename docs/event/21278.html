description

同人関係の方々で集まって温泉で日頃の疲れを癒し
スケブ交換からお絵かきについての話までなんでもありの
グダグダのんびりとした時間をすごせたらいいな～なんて考えてます。

まだ企画の立ち上げも途中の段階です。
日時などの詳しいことは、皆さんの都合に合わせて決めていきたいと考えています。
（本ページで29日としているのは仮です）
なのでご意見があったらバシバシ伝えてください！

一応考えているのは「湯元　小金湯」と言う温泉です。
南北線真駒内駅より無料送迎バスが出ています。
駅で集合して行くのがいいかなと思ってます。

こちら
http://koganeyu.jp/

日帰り入浴700円
タオル持ってくのめんどくさい！って方は+200円でタオル浴衣が追加されます。

ロビーで話すだけじゃなく、じっくり話がしたい！という方が多いようならば
貸座敷を借りることも検討しています。
ですが、どれくらいの人数か分からないので、まだまだ保留の段階です

お食事についても皆さんのご意見を優先したいなあと考えてますのでお待ちしています。
お食事処もありますが、温泉メインだと軽食で十分なのかな？

と、ここまですっごーく適当に書きましたが
あくまでも僕が大まかに考えているだけで
「もっといい温泉があるよ！」
「もっと良いプランがあるよ！」
なんてご意見があればぜひ教えていただきたいと思ってます！

こういう募集かけたら需要あるのかな？どれくらいの人が集まるんだろ？
という思い付きでとりあえずたてた試験的ものなので
先述の通り、今の時点で日時を決めてはおりません。

参加希望を出しても、
後から決まったプランによって都合が悪くなった場合は
途中で不参加希望となっても全然問題ありません笑

「予定にもよるけど行ってみたいかも～」と言う方がいましたら
参加希望の方お願いします！
