スマホアプリ開発者向けのマネタイズSDK、Unity Adsの小規模な勉強会です。

Unity Adsを実装しているアプリケーションを見ながら、「どのような見せ方をすれば収益を上げやすいのか」、「ユーザーに好まれる広告の見せ方はどういったものか」を学んでいきます。

アプリ開発者の方を優先してご案内させていただきますため、
場合によっては、参加をお断りする場合もございます。
ご了承くださいませ。

今回は、Unite 2015 Tokyoでの講演をベースに、より深く質疑応答ができるような内容を想定しています。
