【概要】
色彩やかなブルーライトが印象的なデザイナーズハウス！
唖然としてしまうほど独創的な外観『JASMAC AOYAMA』
世界的に有名なイタリアの偉才建築家アルド・ロッシ氏が手がけ、
建築業界で著名な『プリツカー賞』に輝いた実績をもった一軒家邸宅♪
ヨーロピアンな雰囲気の極上バンケット！
有名ドラマの撮影でも使用されＴＶ番組やラジオ・雑誌など様々なメディアから注目され続けています。
そしてイタリアンテイストの暖色系で統一された暖かな雰囲気の中
交流が盛りあがること間違いなし♪
出逢い急増のこの季節を逃すのはもったいない！
恋活婚活したい方、異業種交流好きな方、
ただただ素敵なPartyを想像し楽しみにお越しくださいませ。
【行き方】
地下鉄表参道A4出口を出て国道246号青山通りを右にまっすぐ進みます。しばらく進むと右手にコンビニポプラが見えてきます。ポプラが見えたらそこを右に曲がりひたすらまっすぐ進みます。青南いきいきプラザの十字路をすぎてさらにまっすぐ進むと右手に会場がございます。
