複数回（８−１０回）にわたり弊社が実戦で使われてきたアプリ開発技術を伝授します。
今回がその第一回目になります。

■内容
・開発環境の準備
・Coffeescriptの予備知識
・Titanium-Cli（Titanium Studioさようなら）
・シミュレータ・実機テスト
・Tishadowによる爆速テスト
・iOS/Androidアプリ開発の注意点
・実戦紹介一：グルメ探しアプリ（GMAP、WEBAPI）
※すべての講義内容は弊社実際のプロジェクトで経験したものに基づく

技術キーワード：Titanium Coffeescript Javascript Jade Tss Tishadow Slimframework Objective-C AWS GCE

参加者は基礎なプログラミング能力を有することが望ましい。
