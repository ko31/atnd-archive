「ドミニオン」「世界の七不思議」「禁断の島」などを体験しよう！

土曜日の朝～夕方まで奇跡的に時間が取れましたので、昼ゲーム会を開催したいと思います。初めての方、ボードゲームに興味がある方、どなたでもお気軽にご参加下さい。

とりあえず3-4人募集ですが、もし、もう少し集まるようでしたらもう1卓立てれます（必要ならさらにもう1卓立てれる場所はあるけど、多分駐車場が無い）。

参加費などはもちろん無料です。

注意事項

  • 昼食は車で数分の所にあるローソンなどで事前に買ってきて頂くか、買出しに出て下さい。（私は事前に用意します）
  • 弁当・ペットボトルなどのごみはお持ち帰りください。
  • 駐車できるスペースにご注意下さい。詳細はこちらのイベントの地図を。

プレイするゲームは、参加した方のご希望にあわせて、軽いのかた重いのまでなんでも(^^)

大人気の「ドミニオン」や、2011年のドイツ年間ゲーム大賞にノミネートされている「世界の七不思議」や「禁断の島」なども体験できます。
