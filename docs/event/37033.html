こんにちは！　はーしぇんと申します。

2013年の一発目…というには少々遅くなってしまいましたが、
昨年に引き続き、本年もRD-Soundsカラオケオフを開催していきたいと思います！
オフ会コンセプトは変わらず、凋叶棕や少女病を中心とした
RD-Sounds楽曲について歌ったり語ったりするオフ会ですー

では興味のある方は以下概要を上から下まで熟読いただき、奮ってご参加ください！

※本オフ会は個人によるファンイベントです。
　RD-Sounds様に問い合わせ等なさらぬようお願いします。
※ATNDにはTwitter・mixi・googleアカウントなど、お手持ちのIDでログインできます。
　ログイン後、参加者メッセージ送信のためのメールアカウント設定をお願いします。

【参加登録期限】

2013年3月1日　23：59

【日時】

2013年3月3日（日）
カラオケ … 12：00～18：00　（6時間）
二次会 … 18：30～　（2～3時間程度を予定）

【場所】

パセラリゾーツ AKIBAマルチエンターテインメント
http://www.pasela.co.jp/shop/akiba_multi/pasela_akiba_multi/index.html

最小4人～最大8人

【料金】

カラオケ … 2400円

【内容】

◯集合について

集合場所は秋葉原駅付近となる予定です。
詳細な場所、時間、および幹事の目印を、募集締め切りの後、
参加者の方にご連絡致します。

◯カラオケについて

最初に歌う人から、時計回りで順番に歌ってもらうようにします。
始まってから3周する間は、凋叶棕や少女病等の「RD-Sounds曲縛り」でお願いします。
他の人との曲の被りは気にせずに、歌いたい曲を入れてください！
4周目以降は、基本的にどんな曲を歌ってもOKです。

◯二次会について

秋葉原周辺の居酒屋を予定しています。
人数がある程度決まり次第、予約を行ってご連絡を致します。
予算3000～4000円程度の所を探そうと思います。

【諸注意】

・他人に迷惑をかけないよう、常識と良識のある行動をお願いします。
・特に、ナンパや宗教勧誘等の行為は、発見次第ご退出いただきます。
・前日以降のキャンセルは、キャンセル費が発生する可能性があります。
・会費は集合時に集金致しますので、ご準備をお願いします。
・飲み物、食べ物の注文をされる際には、幹事に一声お掛け下さい。
・カラオケオフ中は禁酒です。
・室内禁煙です。煙草はロビーや廊下等の喫煙スペースでお願いします。

ご参加いただける方は、コメント欄に、

①成人済or未成年
②2次会までフル参加orカラオケのみorカラオケ遅刻/早退（時間も併せて）
③何か一言！

をご記入のうえ、参加登録してください。

以上、ご不明な点があればお気軽にご質問下さい！
