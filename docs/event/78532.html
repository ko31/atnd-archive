ゴスペル空間は、国内外を問わず厳選したこだわりのアーティストによってお送りする「短期集中型ゴスペルイベント」です。
ゴスペル経験者はもちろんのこと、初めてゴスペルを歌うという人もゴスペルを堪能できる内容で、
一緒に歌ったり、演奏を聴いたりすることに加え、曲やゴスペルを深く知ることが特徴です。

https://www.youtube.com/watch?v=zl1JUYp02Gc

■日程：２０１６年９月１７、１８、１９日

※1 Workshop
・いずれか1日、または2日だけの参加も可能です。
・各日内容は異なります。

※2 Optional Class レナード講義「The History Of Gospel Music」
奴隷時代から現代までの「ゴスペル音楽の歴史」について、各時代の曲も紹介しながら
お届けいたします。
・本Workshop参加者のみ受講いただけます。
・事前にOptional Ticketの購入が必要です。

※3 Sunday Service
・日曜日は、アメリカからのゲスト講師とともにSunday Service=キリスト教礼拝をおこないます。
・参加は自由です。
・参加費も無料です。※任意の席上献金があります。

※4 Optional Class ロバート講義「A Walk In The Faith」
ゴスペル音楽界の第一線で活動する中で彼が経験してきた「信仰の歩みとは」について、
カート・カー氏とのエピソードを交えながら話していただきます。
・本Workshop参加者のみ受講いただけます。
・事前にOptional Ticketの購入が必要です。

※5 Harvest Time
・神さまからの恵みに感謝して、3日間練習してきた曲をコンサート形式で歌い ます。
・参加者は、客席で歌います。
・ワークショップ3日間参加した人のみ参加できます。
・見学不可

■ゲスト講師(Director)

レナード・バークス/Leonard Burks
アメリカで最も歴史あるゴスペルのコンベンション「NCGCC」の重鎮。彼を超える指揮や指導をする人はいないほど。

ロバート・ヘンダーソン二世/Robert Henderson II
Kurt Carr and Kurt Carr Singersのキーボーディストとして過去2回来日し演奏を行う。
