description

エンジニアとして活躍し続けるためには スキルアップが欠かせない と聞くけれど、 その方法がわからない… 。どのようにスキルアップをすれば 市場価値もアップ するのかを知りたい…。
などのお悩みを持つエンジニアを対象に、 「エンジニアとして生きのこるためのスキルをキャッチアップする技術を学べるセミナー」（無料） を12月8日（木）に開催いたします。
東京で大好評だったこのセミナー。大阪での開催は初！ですので、この機会をお見逃しなく！
また今回は、講師・塩谷さん著者「Web制作者のためのGitHubの教科書」の技術書もプレゼント！皆さまのご参加をお待ちしております。

日時 12月8日（木）19:00開場　19:30～21:00
費用 無料
定員 50名
※応募多数の場合は、抽選となりますのでご了承いただければと思います
