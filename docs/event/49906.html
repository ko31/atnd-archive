『もう一段ステージをあげたい女性のための自己変革の技術』

catch 去るか、残るか、起業するか～自分らしいキャリアを作るために～
description

女性リーダーを育成したい企業の意向に反して、優秀な女性ほど企業を去って行く傾向にあるようです。

本当に今の場所では、自分を活かすことは出来ないのでしょうか？転職すれば問題は解決するのでしょうか？

本セミナーでは、自分自身のコミュニケーションスタイルを振り返り、ビジョン・ミッション・価値観を明確にし、
本当に得たいキャリアを築くために、何を変えるのか、どう変わるのか、取るべきアクションは何かを考え、
プログラムを設計します。

どこの企業・場においても通用し必要とされるために、組織でもう一段ステージをあげるために、
必要なコミュニケーション、リーダーシップ、コンピテンシーを磨きます。

自分のミッションは何か、人生において成し遂げたいことは、今の会社ではどうしたら実現出来るのか、

そのためにどう自分を変えればいいのか、深く考える一日となるでしょう。

希望者にはフォローアップコーチングを行います。

■プログラム■

１．自分自身を知る

(1)強いリーダーのコミュニケーションの特徴

(2)自分自身の課題を明確にする

(3)女性の特性を考える

２．ビジョン・ミッション・価値観

(1)目指したい絵を描く

(2)なんのために働くのか

(3)判断基準となる価値観

　何が大切で、妥協出来ないのか

３．キャリアプランを設計する

(1)これまでの仕事の棚卸をする

(2)自分のコンピテンシーを認識する

(3)ビジョン達成に必要なスキルと知識

(4)魅力的な履歴書とは

４．女性のコミュニケーション

(1) 女性リーダー20の悪癖

(2) 表現力を身に着ける

(3) 存在感を示す

(4) ネットワークを作る
５．明日へのアクション

明治大学文学部卒業後、第二精工舎（現セイコーインスツル）入社。経理部に勤務。

その後、キャセイパシフィック航空にCAとして入社。香港をベースに勤務。

同社在職中、トレーニングセンターで英国企業人材開発機関『IPD』会員資格を取得しサービス指導・育成に携わる。

帰国後、ヨコハマグランドインターコンチネンタルホテル、エルメスジャポン、シャネル株式会社において
トレーニングマネージャー。　2007年イタリア系ラグジュアリーブランド人事部長。

国内外のブランドの理念浸透、富裕層をターゲットとするサービス人材の育成、人事労務施策策定に携わる。

2007年認定プロフェッショナルビジネスコーチ取得。世界的コーチ、マーシャル・ゴールドスミス博士の指導を受ける。 　　　　　　　　　　　　　　　　　　　　　

2012年4月ビジネスコーチとして独立。

企業の管理職コーチング、人材開発、女性リーダー育成、「おもてなし」の風土醸成などを主に活動。

2013年11月合同会社オモテナシズムを設立し、「おもてなし」を企業戦略・人事戦略として社会に浸透させる活動を開始。
