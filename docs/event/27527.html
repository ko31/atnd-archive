コラボカフェalike　毎週恒例の交流会です！

今回は「起業家ランチミーティングIn根津」です！

起業家や起業準備中の人、または起業に興味がある人限定のランチ会

みんなで起業についての疑問など解決したり、
ビジネスコラボが出来たりと
楽しく有意義な交流会となっています♪

また会場はこだわりのコーヒーが美味しい「みのりカフェ」
下町にありながらイベントを沢山開催していて、いろんな人々が集う新しいカフェです。

美味しいランチ（３種類から選べるランチにサラダ・スープ・コーヒーもしくはハーブティつき）を食べながら皆でお話しませんか？

イベントの申し込み～当日終了までの流れ

※ホームページなどからもお申し込みを受付しておりますので、こちらで残席がある場合でも満席の時があります。その場合は担当者より早急にご連絡いたしますのでご了承願います。

2.当日は１０分前より入場可能

3.ランチメニュー三種類とドリンクを選んで頂きます。

４．他の人たちとご自由に会食・会談をして頂けます。

※wifiつながります。

★期待出来る効果★
人脈作り・新しいビジネスが生まれる・起業について知れる・起業家同士の友人が出来る・ビジネスパートナー探し・起業について相談が出来る・疑問が解決できる

★対象者★
起業家・個人事業主・経営者・起業準備中の方・起業してみたい方
（保険業界の方は恐れ入りますがその法人の代表者のみとさせて頂きます。）
