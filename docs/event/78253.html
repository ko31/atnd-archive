30名に増枠しました！

＠ITの動画連載企画「セキュリティのアレ」始まって以来、初の公開収録を開催いたします。
もちろん参加費用は無料です。

●公開収録コンテンツ
　1本目：6月のセキュリティ時事ネタ（16時スタート予定）
　2本目：増刊号（17時30分スタート予定）

●注意事項
　・基本的に皆さんの顔が出ることはありません。
　・ヤジは禁止（但し、もっとココが聞きたい！などのご意見はOK!）
　・撮影開始後の途中入場は禁止（受付に人もいません）
　・入場時間は「15:45～16:00」「17:15～17:30」の2回です。
　　（撮影状況により、入場時刻が遅れる場合があります）
　・セミナールームの扉が開いていない場合は、受付付近でお待ちください。
　（セミナールームは受付向かって右の部屋です）
　・予定変更によりこれなくなった場合は、キャンセル処理をしてください。

【重要】
　アイティメディア株式会社、7月に移転を行います。
　本撮影は、移転後となりますので、ご来場の際は十分ご注意ください。
　最寄り駅は「麹町」です。（青山一丁目ではありません）

公開収録なんかに興味を持っている人がいるかどうか全くわかりませんが…
とりあえず、やってみようと言うことで企画しました。
一人も来なくても収録はします！！

とはいえ…一人も来ないと寂しいので、みなさまのご参加お待ちしております。

【ご来場のみなさまへ】
・会場は、受付向かって右側（と言うか、ほぼ後ろ）のお部屋となります。
・部屋の扉が閉まっているときは撮影中です。決して扉を開けずに、受付付近待合場所でお待ちください。
・扉が開いている間は、出入り自由です。但し、撮影開始後は基本出入り禁止となりますので、お手洗いなどは済ませて会場へお入りください。
・出演者から発言を求められた場合、映り込みたくない方は、その場で発言してください。
・基本私語・ヤジは禁止です。
・撮影の内容に応じた合いの手、笑い声などはOK！もっとココが聞きたいなどのご質問もOKです。
・専用の受付が設けられたり、誘導看板が出たりしません。とにかく入場時間にアイティメディアの受付付近でお待ちください。
