イベント名:
とことん見せます Unity ゲーム開発 on マイクロソフト

概要

Windows, Windows Phone, Xbox One – 3つのプラットフォームを実行環境とした Unity ゲームの開発方法をご紹介します。Visual Studio Tools for Unity （旧 UnityVS）を使用した開発方法に加え既存の Unity ゲームの移植方法などもデモを交えてご紹介します。
また、株式会社ゼンリン様との共同ビジネス『Japanese Otaku City』でも話題の 株式会社ポケット・クエリーズ の佐々木様にもご登壇いただきます。なんと、このセミナーのために特別に梅田近辺のモデルも公開いただけるとのことですので、ぜひご期待ください！セミナー終了後には簡単な懇親会も行いますので、ネットワーキングの場としてもご活用いただければと思っています。このセミナーは大阪限定で行わせていただきますので、ぜひこの機会をお見逃しなく！

講演概要

Universal Windows Apps 開発概要

  • セッション内容
    • Windows, Windows Phone, Xbox One – 3つのプラットフォームはゲームの実行環境として活用できます。このセッションではUnity を活用する前に知っていただきたい、Universal Windows Apps の開発方法をご紹介します。XAMLとDirectXの相互運用、Visual Studio のグラフィック診断機能などにも触れます。

クエリちゃんと大阪ちいさんぽ 〜 既存UnityタイトルをWin移植してわかったこと 〜

  • セッション内容
    • 3D都市モデルアセット『Japanese Otaku City』（株式会社ゼンリン様との協同ビジネス）についてご紹介します。本セミナーの為に、大阪向け特別バージョンとして開発中の梅田近辺のモデルも特別公開。関連して、3D都市モデルアセットや既存UnityゲームのWinプラットフォーム移植時に遭遇した各種課題と解決策も同時にご紹介します。

Visual Studio Tools for Unity ( UnityVS ) による コーディングとデバッグ、UnityによるUniversal Windows Apps 開発

  • セッション内容
    • Visual Studio Tools for Unity （旧 UnityVS）を使用した開発方法に加え Windows 8.1 や Windows Phone 8.1 および Xbox One 向けの Unity ゲームの移植方法についてデモを交えてご紹介します。
