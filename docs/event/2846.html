PHP 初心者 勉強会 第28回目です。

PHP 初心者 勉強会 は、第20回目から、"初心者" 勉強会と"中級者" 勉強会に分れました。

この勉強会の対象者は、"PHPは初めてだけど、他のプログラムの経験がある方"です。
プログラムそのものが未経験の方は、『超』初心者勉強会にご参加下さい。

途中から初参加される方へ

PHP初心者勉強会は、定期的に行っているシリーズ企画です。
前回の勉強会で解説した内容は再度説明しません。
勉強会の進め方なども説明しませんので、初参加の方は早めに来てレクチャーを受けて下さい。

その他

参加費用は無料ですが、貸会議室の代金が500円かかります。
（初参加の場合だけ1000円になります。2回目以降は500円になります）

勉強会は19時から開始ですが、会議室は18時より使えます。
18時からボランティアの中級者がいますので、開発環境の質問などが可能です。
また、課題についての不明点など、わかる範囲でお答えします。


※　注意　※
ドタキャン（当日キャンセル）・バックレ（未キャンセルの不参加）が非常に多く
席が余っているにも関わらず補欠者が参加できない状況が続いています。
また、会議室の費用を10人で割って計算しているため、不参加者がいると赤字になってしまいます。

補欠待ちの方がいた場合には、連続でドタキャン・バックレを行った参加者を管理者権限でドロップすることがあります。
あらかじめご了承下さい。
