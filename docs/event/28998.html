来る５月２７日、いよいよ開催される

『Mac、iPhone、iPadで簡単：動画と音楽の作り方すべて見せます』

各方面より「iPhone３分映画祭」はいつ再開？の言葉を受け、再開ではなく、啓蒙からもう一度、という、映画祭の前段階、制作の事例やノウハウのプレゼンテーションという本気イベント。それだけに「もっと聞きたい」「ピンポイントで知りたいところがある」そういう方も多いと思います。そこで懇親会という形で有名ビデオブロガーとの交流の場所を設けることといたしました。

  • 参加できるかたはもちろんどなたでも！でも、
    1. 動画制作をやってみたいけどどこからはじめていいかなあ
    2. 動画制作やってます！でも行き詰まり気味
    3. あのゴージャスな面々と杯を交わしたい！
    4. イベント行きたいんだけど夕方からしか空いてないんだよね

なんて人たちにぴったりです。

  • お酒の席となりますので未成年の方は参加できません。
  • 酔っぱらいすぎて乱れぬように。でも3時間、たっぷり飲んでしゃべりましょう。
