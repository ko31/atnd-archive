description

石川県の伝統料理の一つ、とり野菜。野外でとり野菜を煮込んでつまみつつ談笑などし、台風一過を楽しもうという会です。ご家族お誘い合わせ歓迎。

場所：石川県金沢市卯辰山常磐不動尊横の広場（これの真ん中の広場です。航空写真で見てね。）

行き方：経路


  • 兼六園方面から：兼六園下交差点を橋場町方面に進み、ガソリンスタンド（Eneos）の交差点を右折、天神橋を渡って卯辰山を登り、「卯辰」などを過ぎて200mほど行くと、左カーブ手前に右斜めに降りる小道（鳥居があるのが目印）があります。その突き当たりに広場があります。

  • 森本方面から：橋場町を過ぎて兼六園下手前、ガソリンスタンド（Eneos）の交差点を右折、天神橋を渡って卯辰山を登り、「卯辰」などを過ぎて200mほど行くと、左カーブ手前に右斜めに降りる小道（鳥居があるのが目印）があります。その突き当たりに広場があります。

駐車場：4台分くらいしかありません。少し下ったところの菖蒲園に駐車する、少し上がったところの望湖台に駐車するなどして、10分ほど散歩を楽しんで頂くかも知れません。

参加費：無料、飲食物持ち込み歓迎します。（鶏肉、野菜、みそ、ラーメン、鍋などは主催者が用意します。）
