description

『2.5次元医学部』とは、日本が世界誇るアニメ漫画ゲーム文化を担う制作者と、次世代医療を担う若手研修医たちによる、次世代医療エンタテイメントを企画プロデュースしていくプロジェクトです。

今回はプロジェクト立ち上げにともない、現役研修医とアニメ・漫画・ゲーム関係者によって今後どのような企画を行なっていくのかをトークする公開企画会議を行います！

また当プロジェクトからはクロスメディア展開（ポッドキャスト配信、書籍、漫画、アニメ、ゲーム、XRなど）も想定しています。

■出演　2.5次元医学部（現役研修医とアニメ漫画ゲーム制作者による企画集団）
■開場19時30分　開演20時00分
■料金1000円（1ドリンク別500円）
