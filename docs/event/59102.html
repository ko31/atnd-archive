テーマ

ITよろず相談会

この勉強宴会を始めた時はビールやソフトドリンクで「乾杯！」してから始める事が特徴でした。その後変遷があり最近は普通の勉強会のようになってしまっています。今回と次回はちょっと開始当初を思い出すという意味で多少今までと違ったスタイルで開催します。

今回のテーマはよろず相談会。相談のある方は、事前にメールをいただければ先着順で優先的に相談を受け付けます。それに対してみんなでよってたかって解決策を考えようという趣旨です。最初1時間くらいは私が仕切りますが、その後は普通の飲み会になっていくはずです（←成り行き任せ）

テーマの性質上ブログ等で公開して欲しくなければ申し出てください。もちろん飲み会だけに参加して頂いても結構です。

集まった相談は随時ここにアップします。
