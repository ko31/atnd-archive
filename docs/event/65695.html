【パワーマーケティング体験勉強会】へ
今なら無料ご招待！！


起業後もビジネスを継続できますか？
起業1年以で６０％、３年では８０％がビジネスを継続できず
消えていく現実を知っていますか？

起業後１年間で６割のビジネスが消えていきます。
なぜしょうか？

売上を作ることができないからです。

起業は誰にでもできます。
大切なのは起業後、本当に売れるかです。

私（松本）と会えば、
本当に舌足らずで、
話ベタでまともに会話もできない
売れない営業マンだったのがよくわかると思います。

でもその反面、
どうやって、世界一の売上を作ったか、
どうやって、ボロボロな会社を数年で年商10億円近くまで成長させたのか、
不思議に思うに違いありません。

売るのは営業力だけじゃありません。
「売る力」はもっと別なものです！！

世の中には多くのノウハウや方法、ハウツーがありますね。

でも、実際の起業の現場はどうなっていますか？

勝てるのは極わずか、ほんの人握りです。
それどころか生き残ることすら厳しくないですか？

いいサービなのに売れない・・・
大切な商品なのに買ってもらえない・・・
そうして泣く泣く消えていく起業家を
たくさん見ています。

この勉強会では
起業の現場でどうやって売上を作っていけばいいのかを
● 数々の質問
● 頭の体操
● ケーススタディ
● ワーク
などを通して、
リアルに、そして真剣に考えます。

営業や売り方に自信がない方・・・
売り方がよくわからない方・・・

そんな起業を目指す方でも
そんな起業家でも
あなたの大切な商品やサービスを売り
ビジネスを継続させるためには
「売る力」が必要です！！

この勉強会では
「起業家の売り方」を
リアルに、そして真剣に考えます。

後悔先立たず！
起業に待ったナシ！

ぜひ、この勉強会ご参加ください。

少人数で行う勉強会です。
でも、いっぱい持って帰れます！！

[こんな人にオススメ]

  • 今日から6ヶ月以内に起業する
  • 年商0～1000万円のビジネスオーナー
  • 営業力に自信がない
  • 売り方や売上の作り方がよくわからない
  • 具体的な売上のイメージがない、その実感がない
  • 起業後の売上設計が不安
  • ノウハウやハウツーではなく自ら戦略的になりたい
  • モチベーションや気づき、学びだけではなく、現実の現場で成果を作りたい
  • 将来やビジネスの未来が見えず不安
  • 起業前も起業後もビジネスの相談相手が欲しい
  • 心にブレーキがかかる
  • パワーマーケティングに興味がある

[期待効果]

  • 売り方や売上の作り方がわかります
  • 営業力なしでも売る方法がわかります
  • 起業後の売上について現実的に取り組めます
  • 営業力は売上を作りますが、売上が必ずしも営業力ではない事がわかります。
  • 起業後も夢を持ち続けビジネスとも向き合えます。

[講師プロフィール]
パワーメッセージ ジャパン株式会社代表取締役。年商1億円未満経営者と起業志望者の専門コンサルタント。元学生寿司職人。

中学卒業後15歳で単身アメリカへ、高校、大学と米国で過ごす。大学で経営学を学び初めてマーケティングに触れる。

帰国後は外資系企業を中心に営業とマーケティングの仕事に従事。営業売上世界一、瀕死状態の会社の営業の立て直し年商1億円未満の会社を10億円近くまで伸ばす、等など、数々の現場実績を作る。

帰国後、社会人デビューしたての頃、内気で口下手、対人恐怖から、まともに会話もできない、有り得ない、売れない営業マンになり、精神的にもどん底。その後、売り込めない性格を逆手に、営業力ではなく、やり方の工夫で独自の道を開き、売上アップを実現。「営業力だけが売上ではない」「究極のセールスは営業がいらないこと」「成功できるのはトップ20％かボトム20％」「成功者のノウハウでは成功できない」などの持論で、営業が苦手な経営者のための売上倍増の方法をアドバイスする。

全米NLP協会公式認定及び日本NLP協会公式認定プラクティショナー修了資格。
