内容

今回は、参加者の皆さんへ宿題を出します。
当日、考えてきたことを持ち寄って付箋紙に書いてもらいます。
それらを元に、参加者の皆さんで「良い体験・良くなかった体験」について話し合い
なぜ良いと思ったのか、良くないと思ったのかを話し合います。

宿題

「良かったと思う体験、良くなかったと思う体験をそれぞれ3個以上書き留めてくる」
「行動観察の対象としたいものをいくつかピックアップする」
「UX Fukuokaで取り上げたいネタを考える」
※いずれか1つでもいいし、全部考えてきてもいいです。
