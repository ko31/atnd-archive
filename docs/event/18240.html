2011年ハローステップ就職応援プラザ in いくの

catch 大阪市主催の「無料」就職応援イベント
description

大阪市では、大阪市内の居住者で就職について悩んでいる人などを対象に、就職応援イベントを開催します。

当日は下記のとおり、様々な相談ブースのほか、就職支援セミナーも開催します。各種相談、講座は全て無料です。
「自分の思うように活動が進まない」「一人で考えることに疲れた」などの悩みを持っている人はお気軽にご参加・ご相談ください！

【各種相談ブース等】※当日受付
　　●求人票の閲覧、紹介、職業相談（ハローワーク大阪東）
　　●求人検索（インターネット活用）
　　●求人情報の展示、求職登録、キャリアカウンセリング（大阪市しごと情報ひろば）
　　●職業興味チェック（大阪市地域就労支援センター）
　　●60歳以上の求人情報の提供、職業相談（高齢者無料職業紹介所）
　　●60歳以上の就業相談、各種技能講習情報提供（シルバー人材センター）
　　●職業相談、各種情報提供（大阪市母子家庭等就業・自立支援センター）
　　●就労支援相談（大阪市地域就労支援センター）
　　●就業相談、各種情報提供（大阪市障害者就業・生活支援センター）
　　●職業訓練相談（大阪府ラーニングアドバイス事業）
　　●福祉のお仕事相談（大阪府福祉人材センター）
　　●就労相談（大阪ＮＰＯセンター）

【就職支援セミナー】※事前申込み可
　　●「印象の良い履歴書・職務経歴書などの書き方」（11時00分から12時00分）
　　●「職業訓練セミナー」（13時00分から14時00分）
　　●「面接での話し方、聞き方」（14時00分から15時00分）
　　●「職業訓練セミナー」（15時00分から16時00分）

お問い合わせは、大阪市地域就労支援センター（月～土10：00～17：00 フリーダイヤル0120-939-783）まで。
