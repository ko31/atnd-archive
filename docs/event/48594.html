参加費

スペース利用料1000円になります。

乱立するIDE環境

  • Visual Studio
  • IntelliJ IDEA (PhpStorm, RubyMine 等も含む)
  • Vim (IDEっぽいカスタマイズがされていればIDEとみなします)
  • NetBeans
  • C++Builder
  • Xcode
  • Eclipse

等々…各IDE環境それぞれに一長一短ありますが、とりあえず「僕/私の使ってるIDEのココがスゴイ」をアピールし合う会です。聞くだけでもOK。

言語縛り無し、OS縛り無し。とにかくいろんなIDEの良いところ悪いところを共有しましょう。
また、IDEにとどまらずデバッグ手法やその他ツール一般ノウハウについても共有できればと思っています。

流れ

前半 (3時間程度)

1人5～30分程度の発表を考えています。
「この機能はある？」「どのバージョンがオススメ？」などなど質問もガンガンしていきましょう。

■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■
発表者募集中。誰でも発表OKです。内容被ってもOKです。
アピールしたいIDEまたはデバッグ手法がある方は、
アピールしたいIDEまたはデバッグ手法を添えてコメントにお書きください。
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

後半 (3時間程度)

実際に興味あるIDEをもくもくといじる時間を設けようと思います。
もくもくといじりながら、詳しい人にいろいろ聞きましょう。

突発LTも歓迎します。
