概要

都内近郊WEB/IT/広告業界の方でサバイバルゲームやってる方ていますか？
5/7に千葉の木更津（品川から1時間弱）にあるVISIONというフィールドで交流サバゲ企画したら参加する人いませんか？
結構広いフィールドなので上限人数はありません。

8時～9時の間に現地に集合ですが、途中参加/途中まで参加も大歓迎です。

複数名で参加される方もできれば各人ATNDに登録してください。難しい場合は参加登録時のコメントに人数と各レンタルの数を記載してください。

加藤あきのり @slowbirds
東京都のVitalifyというWEB開発/制作会社でディレクタ/PMをやっています。
サバゲ歴は半年くらい。仕事ではWEBサイトのシステムやソーシャルアプリ、iPhoneアプリ、Androidアプリとか。なんでもやります。オフショアででの開発が多いです。
愛用銃はマルイのM933（MAGPUL MOEカスタム済）
直接のお問い合わせが必要であれば slowbirds[あっと]gmail.com までご連絡ください。

ほぼ毎月社内で人を集めてサバゲーしてます。メンバーには女性もおりますよ
Facebookにて写真とかアップしてます。

参加資格

  • WEB、IT、広告業界従事者。またはその同伴者
  • 上記業界に興味のある方
  • 同業界と付き合いのある業界の方
  • 18歳以上の方

もちろん初心者の方大歓迎。（最初に希望者向けに簡単な講習の時間を取ります）
道具や服はレンタルできます。（別途フィールドで支払っていただきます。3000円～4000円くらい）

かかるお金に関してはなにを借りるか、どうやっていくかによって変わります。

開催場所

VISION
というアウトドアフィールドで行います。
車：品川からアクアライン使って1時間くらい
電車：電車は難しいと思うので参加者が集まった段階で車（レンタカーの検討含む）の割り振りを行います。

初めての方

最初30分くらいは希望者の方に初心者向けの装備の使い方、ゲームのルールや進め方を講習しますので初めての方も安心してご参加ください。
（その間、経験者の方は別途ゲーム可能です。）

ゲームルール

  • 相手の弾に当たったら
    大きな声で「ヒット！」と叫んで退場する。ヒット後はしゃべってはいけない。

基本的にはこれだけです。あとは各ゲームによってルールが違います。

実際に行うゲームは当日お知らせしますが弊社恒例の物であると

  • フラッグ戦（15分）
    お互いの陣地のフラグを奪取するか敵軍の殲滅/復活なし
  • 立て篭もり戦（15分）
    建物に数人が立て篭もり攻めこむ。敵軍殲滅か建物内のフラグ奪取/復活なし/攻めこみ側弾数制限あり/制限時間で殲滅されなければ立て篭もり側の勝利
  • ヒーロー戦（15分）
    立て篭もり戦の逆。大人数で立て篭もり、1～2で殲滅する/復活なし/立て篭もり側弾数制限あり/制限時間で殲滅されなければ立て篭もり側の勝利
  • 耐久戦（40～60分）
    フラグなし、弾数制限なし、復活あり(自軍陣地でカウント)で40～60分間戦い続ける。制限時間後のHIT数の多いチームが勝ち

開催条件

最低実施人数：10名
天候：小雨決行
レンタカーに関して：希望者が少ない場合は弊社の社員の車とかへの相乗りもあり！

注意点

  • 事故防止のため18歳未満の方は参加できません。（保護者の方と一緒であれば来場可能です）
  • 参加確定後の詳細の連絡はATND、Facebook、Twitterなどで行う予定です
  • コスプレ、ネタ大歓迎！（肌の露出がある場合は痛くても泣かない。という条件がつきます）
  • 汚い言葉遣いで罵る。大声で騒ぐ。など興奮しすぎて大人の心を捨ててはいけません。サバゲは大人のスポーツです。
  • 服装は自由ですが怪我防止のため最低限長袖長ズボンをおすすめします。
    迷彩服のレンタルも可能です。
  • セーフティエリアでは必ずロックすること。誤って発射することのないようにしてください（事故、怪我の責任はとれません）
  • ゲーム中の事故、怪我、トラブルは当事者同士で解決してください。主催者側では責任はとれません。
