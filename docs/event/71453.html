◆　(株)ＤＮＰハイパーテック (http://www.hypertech.co.jp/)：
　1994年に(株)ハイパーテック設立。2015年7月に国内ICカード市場でトップシェアを持つ大日本印刷(株)の完全子会社になった。アプリの安全性診断サービスや情報セキュリティ関連のソフトウェアやサービスを提供し、高い評価を得ており、受賞歴も多数。
アプリの改ざんを防止するクラッキング対策ソフトウェア「CrackProof（クラックプルーフ）シリーズ」や、セキュリティの低い環境においても耐タンパ性を保持し、重要な暗号化データの保護や改ざん防止を実現するホワイトボックス暗号ライブラリ「diaCrypto」などは代表的商品。
　CrackProofはソフトウェアの実行プログラム自体にプロテクトをかけることで、クラッカーなどによるソフトウェアの解析を防ぎ、さまざまな違法行為を防止することが可能。diaCryptoは、WhiteBox 暗号として設計・実装することで、一般的に必要とされる暗号強度(AES同等かそれ以上)を確保しつつ、クラッキング耐性を高めている。
　当日は、難解なセキュリティ技術について、クラッキング手法をデモしながら、その商品特性やセキュリティの状況を分かりやすく解説する。
＜プレゼンター＞　　　　 代表取締役社長　　　小川　 秀明　氏

◆　(株)NHKメディアテクノロジー　 (http://www.nhk-mt.co.jp/) :
　NHK-MTは、2008年に放送技術系のNTS（NHKテクニカルサービス）とIT系のNCS（NHKコンピューターサービス）の2社の統合で誕生。現在、クラウドサービスを基盤とした映像制作支援ツール「Coll@Board」(コラボード）の開発と試験運用を進めている。これは、マイクロソフトのパブリッククラウドサービス「Microsoft Azure」を採用し、日本マイクロソフトの技術支援を受けて、放送業界や映像制作にかかわる分野で活用できる実用的なツールにすることを目指している。主な機能として、コンテンツのアーカイブのほか、編集、コンテンツ管理、そして協働作業を可能にするワークスペースがある。
これを導入することで、制作現場からの素材収集、メタデータ作成、ラフ編集、プレビュー、アーカイビングなどが可能になり、迅速で場所を選ばない、時間的・物理的な制約からも解き放たれて、映像制作や協働作業が可能になる。NHKが2015年10月から放映する新番組『新映像の世紀』で、このシステムが先行使用され、膨大な映像データから編集スタッフが映像素材を効率よく選び出せるようにした。
当日は、Coll@Boardについてデモを交えて説明する予定。
＜プレゼンター＞ 放送技術本部ビジネス開発部
　　　　　　　　　　　　　　 チーフ・エンジニア　　栗岩 範和 氏

◆　(株)ゲオネットワークス (http://geo-networks.co.jp/):
　愛知県名古屋市中区に本社を置く、レンタルビデオショップ・古着ショップなどを全国に展開するゲオグループのネット系中核会社。2014年10月1日に稼働。2015年9月1日からは、幅広いエンターテインメント作品のレビューをユーザー間でシェアできるレビューSNSアプリ「クチコ」の運用を開始した。
　このアプリは、商品レビューを通してユーザー同士が繋がることができる。映画だけではなく、音楽、マンガ、ゲーム、CDなどの幅広いエンタメ作品において、ユーザー同士で作品の評価やレビューコメントなどの情報交換ができる。
自分の好きな作品を集めたコレクションボードを公開することもでき、ユーザー同士のコミュニケーションが活発化される。ゲオ会員であれば、レンタル履歴も表示・閲覧が可能。また、作品へのQ&A機能や店頭在庫とも連動しているため、会員証機能アプリ「ゲオ公式アプリ」とも連携し、オムニチャネルリテーリングの強化へと繋げていく。既存のゲオ会員の利用に留まらず、たくさんの映画ファン、エンタメファンにも利用促進を図り、累計1000万レビューを目指している。
　当日は、オムニチャンネル戦略における役割も含めて、クチコの説明をする予定。
　＜プレゼンター＞ 　ネットワーク運営部　 部長　　谷　行介　氏

◆　(株)ゲオネットワークス (http://geo-networks.co.jp/):
　映像配信コンテンツの企画・制作の実績のあるエイベックス・デジタル(株)と提携し、2016年2月から映像配信サービス「ゲオチャンネル」を開始する予定。店舗レンタルや宅配レンタルと組み合わせたオプションもある“ハイブリッドVODサービス”。　
　ゲオチャンネルは、映画や海外ドラマ、キッズ、アニメ、人気アーティストのミュージックビデオやライブ映像、成人向けなど13ジャンル、8万以上のタイトルを用意。ゲオ店舗には平均的に5万タイトル以上、宅配レンタルには24万タイトル以上を用意し、利用者はオプションサービスを組み合わせることで、「すぐ見るVOD」「身近なレンタル店」「品ぞろえの宅配レンタル」を利用シーンに応じて使い分けることができる。メールアドレスを登録したゲオIDを使い、ワンストップでこれらが利用可能になる。VODサービスは、スマートフォンやタブレット、パソコン、テレビのマルチデバイスで展開。1アカウントにつき最大5台までデバイス登録が可能。
　ゲオは映像ソフトのレンタルサービスを中心に全国に約1200店舗を抱え、年間のユニーク利用者数は1600万人を超える。ゲオチャンネルの開始時には無料ゾーンを設け、ゲオ会員全員を新サービスの無料会員とする計画もある。
　＜プレゼンター＞ 　VOD事業部・コミック事業部　 部長　　設楽 亮　氏
