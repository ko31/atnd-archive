19:30 〜　【 超実用的！Photonの使い方講座 】

「オンラインゲームをイチから作ってちゃんと実装したい！」 「好きなキャラクターを同期させたい！」

そんな皆さまのお望みを実現すべく、Unity上でPhotonを使いこなすコツをレクチャーします！
オブジェクトの同期をきちんと行う方法や、イベントの通知方法など、オンラインゲームに必要な要素を伝授します。

　
　

20:30 ～　【「ポポポ認証」を支えるネットワーク技術 】

Oculus Game Jamで開発された「ポポポ認証」のネットワーク技術に関する舞台裏を、
開発に携わったPhoton運営事務局の山本が紹介します。みんなでピョマりましょう！体験も可能です。

　

21:00 〜　歓談

ポポポ認証体験できます！
