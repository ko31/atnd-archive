Laravel4 の電子書籍である Laravel Testing Decoded 日本語版 の輪読会です。
当イベントに参加するには Laravel Testing Decoded 日本語版 を購入するか、購入予定の方でないと話題についていけないと思います。（原書のみを購入された方もOKですが、イベントで扱う本は日本語版です）

基本的には予習をしていなくても大丈夫なように進めます。
数章ずつ区切って、章の担当者を決めて読み進めます。

想定している参加者層 (以下の基本的な知識をお持ちの方)

  • PHP 5.4
  • Laravel 4
  • Composer

日本語の電子書籍

  • Laravel Testing Decoded (JP) 待望の、Laravel4ユニットテスト解説本
