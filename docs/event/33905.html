C# Advent Calender 2012

12月の1日から25日まで、参加者が日替わりでブログを書く毎年恒例のイベントです。C#に関するネタなら何でもOKです。特に制限は設けないので、コンソールアプリでも、WPFでも、ASP.NETでもなんでもござれです。

参加方法

このイベントに参加します（先着25名まで）
申込時希望日に12月○日の午前0時から午後12時までの24時間の間にブログを書きます。告知方法は、ブログを書いたらここのコメント欄でお知らせ下さい。（もっと良い方法を考えるかもしれません）

ゆるいイベントなので気楽に楽しんでください。

希望日を必ずコメント申し出てください(先着順)
