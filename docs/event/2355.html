description

冬の訪れを感じさせる季節となりましたが、皆さんにおかれましてはますますご清祥のこととお慶び申し上げます。いよいよ2009年もあと1ヶ月を残すのみとなりました。会員の皆さんのご活躍によりまして、こうして2009年も無事に終えようとしております。

さて、今年も忘年会を開催します。万障お繰り合わせの上、ぜひ、お集りください。昨年までは、事務局に開催準備等をお願いして来ましたが、今年は、私たち「コミュニケーション対策本部」が幹事を務めさせていただきます。
会員の呼びかけにより会員全員が顔を揃えての忘年会を催しです。この機会に今年１年の慰労をかねてウェブ業界の更なる発展のため、また当協会の会員相互の親睦を深める忘年会にご参加下さい。
（協会員全体の記念撮影を予定しております。）概要は下記の通りです。

■ 日時：12月15日（火）　午後7時～午後9時

■ 場所：スタンダード会議室 ※東京駅より徒歩6分、地下鉄銀座線：京橋駅より徒歩1分

■ 会費：3,800円 ／当日お支払いください／領収書発行します

■ ご参照：昨年の忘年会のアルバム
