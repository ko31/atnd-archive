catch ジオキャッシング＋ゴミ拾い＋ちばレポ！
description

6/9に開催したジオキャッシング勉強会（CITOに向けたプレイベント）の第2回を開催します。今回はCITOの主目的であるゴミ拾いも実際に行います。また、同時開催中の「ちばレポ」にも応募するという欲張りな企画です。ジオキャッシングは大人も子供も楽しめます。経験者・未経験者・家族連れを問わず歓迎致します。お気軽にご参加ください。

1.日時、集合場所
9/14(土)13時15分～17時00分(13:15集合)
JR京葉線稲毛海岸駅改札を出たところに集合
（主催者は目印に黄色のGPSロガーを首から下げています）

雨天決行（歩ける程度なら）

2.費用
・参加費無料
・交通費、食費等は各自で負担ください。道中の買い食いは自由です。

3.募集人員
・15名ほど。

4.持参するもの
・スマートフォン（お持ちでない方はご相談ください）
・GPSロガー（任意）

5.服装など
・歩きやすい服装でお越しください。
・ゴミ拾い等で手が汚れますので軍手をご持参ください。
・銭湯に行かれる方（希望者のみ）はタオルをご持参ください。

6.タイムスケジュール（当日の状況により多少変わる可能性があります）
13:15～ 集合(JR京葉線稲毛海岸駅)
13:15～13:30 行動予定説明(人数によっては2チームに分かれます)
13:30～17:00 キャッシュ探し＋ゴミ拾い＋ちばレポ
17:00～ いったん解散（JR総武本線稲毛駅付近）
17:00～20:00頃まで 銭湯「松の湯」入浴後、JR総武本線稲毛駅周辺で懇親会（希望者のみ、実費割り勘）、現地解散

7.当日までの事前準備
ジオキャッシングのアカウント作成
