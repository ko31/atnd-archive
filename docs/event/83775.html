内容

今回は、
エイチームに興味をもってくださった皆さまに、
エイチームのオフィス見学ツアー付きの会社説明会＆座談会をご用意しました。

その名も・・・
＜水9プロジェクト＞

水9プロジェクトとは、
「エイチームをとりあえず気軽に見に来てほしい。」
そのような想いから発足したプロジェクトです。

エイチーム技術者との座談会を、水曜日夜９時から定期的に開催いたします。

エイチームの「実際」を見て、知って、感じて、その後の検討材料にしていただければ幸いです。

とってもラフな会ですので、どなたもお気軽にご参加いただけますよ。
