今回は、４，【自己催眠ダイエット、なりたい自分に】　です。

世の中に様々なダイエット方法がありますが、なかなか上手くいきませんね。

自己催眠をダイエットに使うと、すんなり痩せて、リバウンドもありません。

自己催眠で理想の体型になるとっておきの方法をお伝えします。


女性も男性も、痩せすぎず、太り過ぎず、健康的な身体に魅力を感じます。

女性は、胸板が厚く、がっしりした体系の男性に魅力を感じます。
それは本能的に、自分と子供を守ってくれる強い人と感じるからです。

男性は、胸とヒップが大きく、ウエストがくびれている女性に魅力を感じます。
それは本能的に、自分の子供をしっかり産んで育てられると感じるからです。

男女ともに、身体的に魅力を感じるには、本能的な理由があるのです。

自己催眠を活用し、潜在意識に働きかけると、理想の身体を手に入れることが容易になります。
理想の身体になると、自信も出てきて魅力も増します。

さらに、自己催眠でなりたい性格になることも可能です。

身も心も、ますます魅力的になってしまいましょう。

映画で主役となるような俳優さん達、ロックスターや歌手、アイドルグループは、皆とても魅力的な体型をしています。

誰もがそんな魅力的な体型でいたいのですが、現実は厳しいものです。

六頭身・胴長・短足・肥満体。
このなかで、肥満体は改善することができます。

そこで、様々なダイエット方法が氾濫しています。

でも、多くの方は、努力空しく、なかなか効果の出ないダイエット方法に、ダイエットを諦めます。

少なからず、やせることができた方も、早々にリバウンドしてしまい、ダイエットを諦めます。

町中で見る様々な人々を観察して、「やせたら魅力的になるのになあ」って方はたくさんいらっしゃいます。

諦めなければ、魅力的な体型になれ、その体型を維持できるのです。


「諦める」事をやめる、それは、意識の力でコントロールできます。
意識の力でコントロールする方法、それが、自己催眠法なのです。

ソニーで研究してきた自己催眠法は、とてもパワフルです。


自己催眠状態で、やせて魅力的な自分をイメージし続けると、「思考が現実化」します。
通常の意識状態で為し得なかった事も、自己催眠状態なら容易に実行可能になります。

「自己催眠ダイエット」は、意識の力を使って、上手にダイエットする方法です。
そして、リバウンドもありません。

これまで、何度ダイエットに失敗してきた方でも大丈夫です。
意志の弱い方も、意志が強くなれます。

このセミナーで活用するオリジナル自己暗示文のパワーは協力です。

多くの方が、魅力的な体型を実現してきた自己催眠ダイエット。
あなたも、魅力的な体型を維持し、幸せな結婚をしましょう。

そして、オリジナル自己暗示文の作り方は、様々な分野に応用でき、なりたい自分になれます。
