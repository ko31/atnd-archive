ヒカ☆ラボ一覧は こちら！！

★講演内容

今やフロントエンドの界隈では最も注目を集めるReact.js。

ReactNativeの登場により、
そのフィールドはネイティブアプリにまで広がっています。

Webメディア「SELECK」など、複数の事業を展開するリレーションズ株式会社でも、
React/ReduxからReactNativeまで、Reactの周辺技術を活用して開発をしています。

今回は、実際の事業での事例を交えながら、Reactを使った開発について紹介します！

登壇企業：リレーションズ株式会社

★講演者プロフィール

■諸見里 圭太氏　（リレーションズ株式会社/リードエンジニア）

2014年にリレーションズに入社。
PHP、ネイティブアプリなどの開発を経て、
現在はWebメディア「SELECK」の開発全般(React.js/Node.js/AWS)と、
取材・記事執筆・編集を手掛ける。

■奥宮 誠一氏　（リレーションズ株式会社/エンジニア）

2000年よりプラト株式会社にてコンシューマ向けソフトウェア、
ウェブサービスの開発に携わる。
2015年 リレーションズ株式会社に入社。
修理マッチングサービス「スマートメンテ」のシステム開発に従事。

■中田 一成氏　（フリーランス）

2014年からフリーランス。
グロースのための施策・開発や、フロント周りの開発がメイン。

React Native meetup主催。
