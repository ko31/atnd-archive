Magento workshopとは

世界中で利用されているeコマースプラットフォーム・Magentoの管理者向けの勉強会です。
Magentoの情報は英語圏では充実したものがあり、教育コースも提供されています。
この勉強会ではMagentoを使ってサイトを運営したい管理者の方や、デザイナーの方向けにMagentoの使い方をワークショップ形式でレクチャーする会です。

対象者について

  • サイトオーナーの方
  • デザイナーの方

※プログラマの方には退屈かもしれません。

注意

この勉強会はワークショップです。
実際にMagentoを触りながらレクチャーを受けることで、効率よくMagentoの使い方を理解することに重点を置いています。
ですので、ノートPCを持参して下さい。
できればレンタルサーバやノートPCにMagentoをインストールされることをお勧めします。

ノートPCを持参されない場合、非常に退屈な時間を過ごすことになります。
その場合、主催者側では対処のしようがありません。
