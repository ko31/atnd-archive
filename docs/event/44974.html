毎年恒例、忘年会LT

今年は 株式会社クレイ / KRAY Inc. 様の場所を借りての忘年会なので、だらしない大人は出入り禁止 ちゃんと調べてから来てね！

■参加対象者

  • 普段のpgcafeは出れないけど、忘年会だけは行くぜ！
  • 今年こんなモノ or サービス作ったよ！
  • 飲み食いしながらLT、マジ楽しい
  • 酒が入らないと人前で話すとか絶対無理
  • 例の彼が正月モチ食えるかどうか知りたい

■費用

  • ポットラック（料理持ち寄り）形式で、持ってきた人は飲み物代 1,000円のみ。
  • 何もかもめんどくさいって人は食べ物（ピザ）＋飲食代で 2,000円 の予定です。

※ 毎年揚げ物ばっかりで、机の上を茶色で埋めないといけないと思ってる人がいるようです。ニオイだけで胃もたれするのでやめてください。

■場所

株式会社クレイ / KRAY Inc.
吉祥寺から歩いて３分

■設備

  • プロジェクタ
  • 無線LAN
  • 電源は参加人数が多いと足らない可能性アリ
