〇概要
在宅勤務（リモートワーク）を希望されている障がいがある方向けに、在宅勤務で求められるスキルや現在の求人状況をわかりやすくお伝えします。

セミナー参加について不安なことがありましたら、担当講師は障がいがある方向けのセミナー実績が豊富ですので、お気軽にご相談ください。

〇セミナー内容

〇セミナー実績
大手損保会社や就労移行支援事業所、就労継続支援B型事業所などで、障がいがある方向けに研修会を開催し、障がい区分にかかわらずセミナー満足度97%の高評価をいただいています。
