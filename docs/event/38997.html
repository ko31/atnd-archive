catch 自宅ラック勉強会では、Asteriskが無慈悲な仕様変更を架電する顧客に!!!
description

音声系で遊んでみよう。ってことで、音声アプリケーションサーバー"Asterisk"を軸に色々遊んでみます。

やること

  1. IP電話機をAsteriskに繋げてみよう
  2. 黒電話を呼び鈴として使ってみよう
  3. ナカヨの電話機でパーク保留してみよう
  4. CiscoのIP電話機をハックしてみよう

会場および集合場所

個人住宅(アパート)が会場となります。
京急梅屋敷駅 下りホーム東口前にて集合。
集まり次第、会場まで移動します。
※それっぽい服きた人が目印です

遅れてくる方は個別にフォローします。

注意点

  • ナカヨ電話機の数に限りがあるので、今回の勉強会も少数です。
  • 6名くらいまで入りますが、4名以降の人は立ち見になりますのでご了承ください。
  • 48Vや200Vな電源がありませんので、冷蔵庫より大きい主装置持ってこられても困ります。
  • INS64回線の代わりにVoIP-GWがありますので、小型レガシーPBXなら、持込OKです。
  • PoE-SWを複数用意していますので、IP電話機持ち込みOKです。電源アダプタなしでもOKです。

懇親会など

懇親会もあります。参加したい人はドウゾ。
