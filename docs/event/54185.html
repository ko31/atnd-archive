description

ほぼ内輪での開催ですので、参加メンバーと全く面識のない場合はお断りすることがあるかもしれません。

梅雨も明けてビールの美味い季節になりました。
みんなでガヤガヤしながら楽しくお酒を呑みましょう！！
今回は、桜ノ宮です。梅田ではありませんがシャトルバスで梅田から移動可能です。
普段ガーデンウェディングするような場所で、スモークしたお肉を焼き焼きしてトルティーヤで巻いて食べるNYスタイルのバーベキューを堪能したいと思います。

いろいろご案内

  • 食べ放題ではありません。
  • 飲み放題です。ビールの他、ワインなどいろいろな飲み物があるそうです。飲めない方にはソフトドリンクやノンアルコールのカクテルもあるとのことです。
  • 人数減やキャンセルは、8月7日中にお知らせください。それ以降の申し出の場合は店舗の予約ルールの都合参加料がかかります。
  • 人数はひとまず8名ですが、増やすことは可能かも…
  • 店舗はOAPタワーにあります。梅田からシャトルバスが10分もしくは15分おきにでております。
    シャトルバスの案内はこちら
  • お知り合いの方など、遠慮なくお誘いくださって構いません。
  • 不明な点などございましたら、お気軽にお問い合わせください。

締切：8月6日 おしどりが寝るまで！
