株式会社スパイア（http://www.spireinc.jp/）が主催する、
スパイアスマートフォンラウンジ（月1回開催予定）の第４回目となります。

【スパイア スマートフォンラウンジとは】
スマートフォンに纏わる内容で毎月テーマを決め、
それに精通された方々をゲストスピーカーとしてご参加いただき、
業界全体のナレッジ共有、参加者同士の横の繋がりの醸成を
目的とした、株式会社スパイア主催のセミナーとなります。

＊尚、今回より、参加者様をスマートフォンサイト保有者様／アプリ
ディベロッパー様／又は、今後スマートフォンサイトやアプリを出される
予定の方に限定しております。サイト／アプリ運営者間の横連携強化を
会合の目的とする為です。ご理解下さい。
＊＊そのため、参加希望の方は、参加コメント欄に「既にリリースしている
アプリ名を書く」又は、「”今後アプリ出す予定あり”と書く」のどちかを
お願い致します。

【第4回詳細】
■説明■
「黎明期は過ぎた！アプリを事業化すること」では、
”とりあえずスマートフォンアプリを出して経験を積もう”というフェーズ
は終わり、各社戦略修正し、より事業化の向けてフォーカスを決めて
動き出している中、各社の事業化への取組についてお話し頂きます。

当社もスマートフォンアプリを積極的にリリースしたり、スマートフォン
広告を営業させて頂く等する中で、数多くのアプリディベロッパーの皆さん
と知り合い、語り合う機会が御座います。その中で感じるのは、各社が
スマートフォンアプリで収益を上げる、事業を上げる事を意識して
ここ数ヶ月の間で軒並み「戦略修正」又は「戦略を定める」という行為を
行っている事（特に、昨年から活動を積極化しているトップランナー企業
ほどその傾向にあると思います）。

今回は、そのような黎明期は過ぎ、次のフェーズに突入しようとしている
中で、各社がどのような展望を持っているかをお話頂きます。
