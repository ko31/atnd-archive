こんにちは、第19回デザイン勉強会開催のご案内です。
今回は先日大田区産業プラザPiOで行われたWordPressの公式イベント、WordCamp TOKYO2013のデザインセッションで語られた「現場の Web デザイナーが語る、WordPressをさらに使いこなすヒント」の延長座談会になります。

当日話せなかったオフレコ話。伝えきれなかったポイント、実際に使っているプラグインや技など、その場で質疑応答も含めながら進お話ししていただこうと思います。

■スピーカー

・井村 圭介／ファンタラクティブ
Webデザイナー/Webエンジニア。フリーランスとしてWebデザイン、WordPressサイトの構築の受託案件や講師・講演などを中心に活動中。
｢本格ビジネスサイトを作りながら学ぶ WordPressの教科書｣掲載のサンプルサイトデザインの執筆を担当。
WordCamp Tokyo 2013ではWeb制作リーダー。

・後藤賢司／よつばデザイン
企画、Webデザインを中心に主に東京と大分で活動中。
子ども服と親子ライフサイト「COCOmag」を運営。
「ゼロから始めるWordPress勉強会」をSmartLight中畑さん、Co-Edo田中さんと共同主催。
Facebookグループ「webクリエイターのための情報交換所」やってます。

■内容

・自己紹介
・事前用意された一部スライド
・質疑応答による自由テーマでのお話し

■対象

・WordPressに興味のあるデザイナー
・WordPressでサイト制作しているWeb制作者、デザイナー
・初級者はもちろん、中級者も歓迎です。

デザイナーの卵さん、ディレクターさん、商品開発関連の方でもどなたでも結構です。
お時間ある方は是非ご参加下さい。
