description

Jellyとは、コワーキング体験会。コワーキングってどんなの？コワーキングスペースってどんなとこ？と、思っている方、Jellyがチャンスです。

今回のJellyはスペシャルセッション、新潟のコワーキングスペースプロデューサの西村和久さんのお話があります！

会場は、なんと！新しくオープンするコワーキングスペースです。
仙台駅から徒歩５分と超便利な場所にできる「クラウドガーデン」が会場です！

Jelly終了後はそのままクラウドガーデンにて懇親会です。（懇親会の申し込みは別です）

その他詳細はブログに書いてありますので、よくお読みのうえ、お申込みください。

※会場の都合上、定員を15名としました。出入り自由なので多少オーバーしても大丈夫とは思いますが、定員譲り合い立ち見などでご協力をお願いいたします。
