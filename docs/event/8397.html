現在開催中のウェブアプリケーション開発コンテスト Mashup Awards 6 に連動して、今年より実践形式の開発イベント MA6 Mashup Camp (マッシュアップ・キャンプ) を開催します。

Mashup Camp ってなに？

Mashup Camp は、参加者それぞれノートPC を持ち込んで、 みんなでわいわい開発しちゃおうっていうイベントです。キャンプという名前ですが基本的に日帰りです。

MA5 では Hackathon という名前で1度だけ開催しました。MA6 では、たくさんの人に気軽に参加してほしいという思いから Mashup Camp という名前にし、定期的に開催してきました。

  • 9/9(木) 第1弾 Mashup Caravan – Maps Talks の報告
  • 9/11(土) 第2弾 Mashup Camp – Maps Day の報告
  • 9/15(水) 第3弾 Mashup Caravan – Girls Talks の報告
  • 9/18(土) 第4弾 Mashup Camp – Girls Day の報告
  • 9/22(水) 第5弾 Mashup Caravan – Social Platform Talks の報告
  • 9/25(土) 第6弾 Mashup Camp – Social Platform Day の報告
  • 10/2(土) 第7弾 Mashup Caravan – Beach Talks in Okinawa の報告
  • 10/6(水) 第8弾 Mashup Caravan Delux の報告
  • 10/9(土) 第9弾 Mashup Camp – Chrome Extensions / Web Apps Day の報告

【開催概要】

今回の Mashup Camp は Google AppEngine コミュニティである appengine ja night (ajn) との共催イベントです。

appengine ja night は appengine ja hack-a-thon (ahack) という名前でハッカソンを定期的に開催していて、今回が7回目にあたります。そして、今回の ahack は、コラボレーションと Google AppEngine を共通テーマに MA6 参加者が集まって、実際に AppEngine で動くウェブアプリケーションを作って、MA6 に応募することを目的としたハッカソンになります。

  • 名称：　Mashup Camp – Collabo Day with appengine ja hack-a-thon #7
  • 日時：　2010年10月13日(水) 20:30 開始 ～ 22:00 終了（事前ミーティング）
    　　　　2010年10月16日(土) 10:00 開始 ～ 18:00 終了
  • 場所：　リクルート メディアテクノロジーラボ
  • 住所：　東京都中央区銀座7-2-6 リクルートアネックス1ビル B1F
  • 地図：　http://www.doko.jp/search/shop/sc1141072/

  • 参加申込フォーム：　http://atnd.org/events/8397

ヌーラボ 染田様、サイボウズ 丹野様、北崎様、TIS/SonicGarden 松村様をアドバイザーとしてお迎えします。

Google AppEngine、Google Apps、Google Apps Marketplace に詳しい @shin1ogawa さん他 ajn メンバーにチューターとして参加いただきます。また、事前ミーティングの中で希望者が集まれば @shin1ogawa さんによる1-2時間程度の AppEngine と Slim3 のプチコードラボも開催できますので、AppEngine 初心者でも安心してご参加いただけます。

Mashup Camp をきっかけとして MA6 応募対象 API と AppEngine を活用したウェブアプリケーションを作って MA6 に応募しましょう。

【マッシュアップ推奨 API】

Mashup Camp で作るウェブアプリケーションは、次の API とのマッシュアップを 推奨 します。 これらの API を使わず、他の API を使ってもかまいません。

  • グーグル
    • Google App Engine
    • Google Earth API
    • Google Maps API
    • Google Apps API
  • TIS/SonicGarden
    • youRoom
  • ヌーラボ
    • Backlog
    • Cacoo
  • サイボウズ
    • サイボウズLive

【連携イベント】

Mashup Caravan – Collabo Talks

Mashup Camp 当日に作る作品のネタ出しやチーム決め、顔合わせを兼ねた事前ミーティングを 10/13(水) に開催します。

10/13(水) は、2部構成となっており、

  • 第1部 18:30 – 20:30 で API提供企業より講演を行っていただきます

  2. 10/13(水) 20:30 Mashup Camp 事前ミーティングで構想を企画にする
  3. 10/16(土) 10:00 Mashup Camp 当日で企画を実際に開発する

という流れになりますので、ぜひ Mashup Caravan から合わせてご参加ください。
