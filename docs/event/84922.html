個人で出来る格安スマホ・SIMカード代理店説明会　
携帯料金節約しませんか♪

主催：一般社団法人日本自由化事業協会
http://www.jlba.club/blank-1

スマホ・タブレットの格安SIMカード代理店説明会です。
携帯料金を節約したい方にもおすすめです。

通常代理店になるには法人である事や、膨大な費用と非常にハードルの高い世界です。
今回の説明会内容は日本で唯一、個人で扱えるお話しです。

もしあなたが今から、今後確実に普及する格安スマホ・SIM取扱店事業に参入出来たら、
大きなチャンスと可能性があると思いませんか？

【格安スマホ・SIM代理店募集要項】

《特定負担金》 (税別)
◎初回 16,960円〜
(3コースから選択)
◎月々 4,960円～
(代理店システム使用料+SIMカード使用料)

《特長》
●専門知識・経験・営業能力・話術不要（説明会有り）
●販売ノルマ・ペナルティ無し
●店舗・在庫・仕入・配送・集金不要
●通信費月額使用料の一部が手数料収入 (毎月発生)

参加ご希望の方はメールにてご連絡下さい。
