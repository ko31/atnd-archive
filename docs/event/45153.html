description

サポートが終了するWindowsXPのPCを、WindowsXPに似たセキュリティサポートのある無償のOSを体験するセミナーを開催します。

2014年4月9日にWindowsXPは、Office2003とともにMicrosoftによるサポートが終了します。
サポートが終了されると、いままであった更新（アップデート）がなくなりセキュリティ上の危険が高まります。
Microsoftの「 WindowsXPとOffice2003サポート終了の重要なお知らせ 」では、そのことが詳しく紹介されています。
とはいえ、つい最近までWindowsXPを搭載していたPCは販売されていましたし、まだまだ元気なPCを廃棄するのはもったいない。
廃棄するにも費用がかかるということで、なんとか現役で使いたいと思う方も多いのではないでしょうか。

この参加体験セミナーでは、WindowsXPのPCを持ち寄り、WindowsXPに似たセキュリティサポートのある無償のOSを体験していただきます。
体験後、PCにOSを載せ替えたい方は、年賀状を作成するまでを手を動かしてやりたいと思います。
ぜひ川口周辺の方々にお集まりいただきたく、ご案内いたします。
