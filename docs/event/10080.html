description

Solaris に関するテーマをみんなで勉強していきましょう。という集まりです。
テーマについてスピーカーが話をしますが、参加メンバーも情報を出し合ってワイワイと進めていきます。
今回は、ネットワークインターフェースの冗長化を可能とする IPMP (IP network multipathing)と Linkaggregation、さらに、ルータの冗長化プロトコルである VRRP (Virtual Router Redundancy Protocol)について勉強します。
ええ、Solaris にも VRRP はあるんです。


    • 13:30 – 14:00 自己紹介等
    • 14:00 – 15:30 IPMP と Link Aggregation 機能によるネットワークインターフェースの冗長化
    • 15:30 – 16:30 VRRP 機能によるルータの冗長化

詳細等については、東京 OpenSolaris 勉強会 2010.11 ページで確認をお願い致します。
http://wikis.sun.com/pages/viewpage.action?pageId=216502061
