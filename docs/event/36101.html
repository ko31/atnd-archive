global game jam in ギークハウス武蔵小杉は無事終了しました。 こちら でレポート記事が読めます。以下2つ、この会場で作られたゲームです。

チーム1: ワンコとおさんぽ

チーム2: 1MinLife

global game jamとは

国際ゲーム開発者協会(IDGA)が主催する、48時間世界同時ゲーム開発ハッカソン！
数人でチームを組み、時間内にテーマに沿ったゲームを作って提出します。デジタルゲームに限らず、カードゲームやボードゲームを作って提出するクリエイターもいます。ゲームならば形式は問わないみたいです。

概要

ギークハウス武蔵小杉をglobal game jamの会場として 登録しました ！
正式に神奈川県川崎会場となりました。
収容人数が少なめですが、一緒にGGJを楽しいゲームを作る仲間を募集します。

詳細

  • 【日時】1/25 17:00(開会式)〜1/27 18:00(全行程終了)
  • 【場所】 ギークハウス武蔵小杉 (道順は こちら を参照)
  • 【開場】16:00頃開場
  • 【参加費】1000円
  • 【参加条件】GGJに参加する人、ゲームを作りたい人、プログラマの場合はUnityが多少使えること、近隣住民に迷惑をかけない人(住宅地なので)
  • 【設備】ネット、電源、軽食、トイレ…他、一般的な民家が備えている物は大抵あります
  • 【周辺】コンビニ徒歩5分、モスバーガー徒歩10分

ギークハウス武蔵小杉とは

東横線武蔵小杉駅(神奈川県川崎市)が最寄り駅のシェアハウスです。プログラマーを始めとする10人で住んでいます。定期的にイベントを開催していて、ついこの間Unity勉強会をやりました。

ギークハウス武蔵小杉からは個人でUnityのゲームを作っている@mi_kamiと@aya_eiyaがGGJに参加します。プログラム、サウンド、グラフィック、シナリオ…何か出来る人で集まって、作れるものを作ってみましょう！
