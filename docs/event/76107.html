きたる3月26日に旧佐世保市戸尾小学校の建物を流用した「させぼ市民活動交流プラザ」において「もくもく会」を開催します。

もくもく会というのは、みんなが興味のあるものを持ち寄って、もくもくとその作業に没頭したり、集まってきた人と雑談するという、ゆるい会です。

知ってることや知らないことをたずさえて、佐世保近辺の人に参加してもらえたらなんか面白いかなと思っています。

現在参加予定者は6名ほどです。参加希望の方は一報入れていただけると嬉しいです。が、当日突然参加されるのでも別にかまいません。

駐車スペースは車40台分くらいのスペースがありますので、問題ないかとは思いますが、当日同施設内で別のイベントが重なった場合、駐車できなくなる恐れもあります。
なるべく乗り合わせで来られるとよいと思います。

飲み物は施設内に自販機がおいてあるので心配いりません。

とりあえずコンピューターやプログラミングやカメラや企画などに明るい人たちが集まる予定です。

もちろん、それ以外にも何かしらに情熱を燃やすかたであれば、どなたでも大歓迎です。

面白そうだと感じたら、かるい気持ちで参加してみてくださいね〜。
