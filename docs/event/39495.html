Webデザイン･プログラミングスクールWAVE


現在WAVEではスクール新規入校生を募集しています。そこで今回は体験レッスン・カウンセリングの参加者を募集致します。Web業界への転職相談、学習プランのカウンセリング、最大2週間の体験入校など。申し込みは下記フォーム、電話にて受け付けております。

勉強会という学習スタイル


企業で求められるスキルは多くHTML5・CSS3・JavaScript、UI/UXや各種フレームワークなど。もちろんmswave新宿本校で学ぶのはそれだけではありません。自ら考え問題解決する力、チーム仕事をするコミュニケーション力。新宿校では勉強会を中心とした学習スタイルで学びます。ソフトの使い方だけではありません。納期のある課題、チームでの制作。受け身ではなく現場を意識した学習スタイルで、モチベーション高く学ぶことができます。


目標に合わせたコースをセレクト＆カスタマイズ


一人ひとりに与えられた時間は限られています。目標達成に向け、必要なスキルを正しく選び過不足なく学習する必要があります。業界の動向を反映した、目的に合わせた学習プランを作ります。未経験からWebデザイナーを目指す「Webデザイナー就転職コース」はPhotoshopとIllustratorを使用したデザインからWeb標準に沿ったコーディングまでを学びます。同じコースでも目標に合わせて内容はカスタマイズされます。すべてのコースがあなたに合わせたオリジナルです。


Web IT業界に特化した人材育成


グループ企業のコンテンツワンはWeb ITに特化した企業です。
社内にアプリケーション開発部門も持ち、人材部門はWeb IT業界多数の企業と直接のやり取りをしています。現場からの情報、求められるものをダイレクトにお伝えすることができます。未経験からクリエイターへ、さらにその先までのキャリア形成のサポートの体制があります。


制限はほとんどありません


Web関連技術を身につけるには長い時間がかかります。没頭する必要があります。毎日来て学習してほしい、朝から晩まで学習してほしい。そんな環境を用意しています。最短最速で世界に通用するWebデザイナー、クリエイターになる。
mswave新宿本校では自由に使って学ぶ環境を豊富に用意しています。
