■ イベント概要

この度よちよちAndroidでは、夜子ままさんを講師にお迎えして、Android勉強会を開催することになりました。
Android開発初心者向けに、講習を行います。Androidアプリ開発を自分でしてみたいけどJavaがよくわからない人向けに、
Javaの勉強を兼ねてAndroidアプリを作ってみるという勉強会です。
奮ってご参加下さい。

この勉強会はよちよちAndroidサンデープログラミングの会（よちよちAndroid）が主催しております。
よちよちAndroidとはこちらのサイトをご覧下さい。

この勉強会は、よちよちAndroidの会員以外の方も参加大歓迎です。

勉強会の後で懇親会を予定しておりますので、是非ご参加下さい。懇親会は勉強会に参加されない方も歓迎です。ATNDはこちら

■ 持ち物

Android開発環境の整ったPCと筆記用具（アンケート記入用）と参加費を持参して下さい。

■ 参加資格

初心者向けの内容となりますので、プログラムをあまり作ったことがない方が対象となります。

  • VBAやJavaScriptの経験はあるけど、Javaはまだないとか、Javaをはじめたけど、写経(参考書のプログラムを一字一句同じように書き写して学習すること)しかしてなくてよく理解していない方など、初心者の方。
  • 【重要】Android開発環境が出来ている方。(OSの種類は問いません)
    Java JDK
    Android SDK
    Eclipse
    のセットアップが済んでいて、Android開発入門本のプログラムなどを少し書き写すなどして、プロジェクトをエミュレータで確実に動かせる方
    当日は開発環境構築済みとして勉強会を始めますので、まだ出来ていない方は当日までに必ずセットアップをお願いいたします。
  • ATNDにメールアドレス設定をされている方
    まだの方はユーザー設定→メールアドレス設定より説明に従って設定をお願いいたします。
