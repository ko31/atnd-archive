更新：2014/08/08 23:50

天気予報によると雨確定なので、本イベントは中止となりました。

代わりに、魚巡りをしましょう。

集合時間： 12:00
集合場所： 品川駅 改札前の時計台のあたり

概要：品川でお昼ご飯を食べて、品川アクアスタジアム、晩ご飯、日本橋でナイトアクアリウム、というコースを予定。

品川で食べたいもの/行きたいお店があったら、ぜひ教えて。

==== 以下中止 ======

そうだ！夏だ！お祭りだ！浴衣だ！

浴衣を着て、夏の東京を楽しもう！

そんなわけで、参加者全員浴衣を着用（MUST!!)して、ぶらぶらと写真撮影でもしながら、楽しみましょう！

集合場所

JR上野駅『公園口』
東京メトロ銀座線 『浅草駅』 雷門方面改札口前
（都営浅草線で来る人がいたら連絡をください）

持ち物

  • 浴衣
  • カメラ（ご自分の体力にあった装備で）
  • 日差しや熱中症対策（日傘や扇子、日焼け止めなど）
  • 元気な身体

内容

上野の不忍池で夏まつりが行われているので、屋台をぶらぶらと食べ歩きましょう！
（出店が少なかったりと、あまり楽しむ場所がなければ、さっさと浅草へ移動しちゃいましょう）

移動距離が大きくなりそうなので、上野は中止で、直接浅草に集合で！お昼も浅草でご飯にしましょう！

その後、上野駅から銀座線で浅草へ。
浅草で和スイーツでも食しつつ浅草寺のほうへお散歩。
浴衣を着ながら、日本の和を堪能！

浅草を散歩したあと、みんなが元気だったら、
日本橋で行われているアクアナイトか、品川のアクアスタジアムでお魚を見に行こう！

アクアナイト（浴衣割は無いが、浴衣着用で日本橋近辺で割引特典が多々あり）
http://h-i-d.co.jp/art/nihonbashi/　

品川アクアスタジアム（浴衣着用で入場半額）
http://www.princehotels.co.jp/shinagawa/aquastadium/

注意

普段着慣れない浴衣なので、みなさん無理をしないように。

当日、浴衣を着用してこなかった人は、全裸で歩いてもらうことになります！！！

[検討中]
雨天の場合は、浴衣をやめて私服で、アクアナイトでも見に行きましょう。
小雨の場合は、上野をあきらめて、浅草か日本橋オンリーで行きましょう。
