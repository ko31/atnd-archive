要するに忘年会です。

今年一年もいろいろございました。
昨年のD-dayよりの流れに着いていけない。何か違う……けど好きなんだよ！もしくは好きだったんだよ！そんな普段吐けない鬱憤を発散しませんか？
我々は…その姿を心から…応援するものです…！

参加につきまして

なんとかマスターに対してネガティブな感想を持たれてる方を鼓舞したいというイベントでありますので、やっぱり俺、今のうんたらマスター好きだよ！と心の底から言えちゃう方、え？アニメいいじゃん。何が不満なの？という方は参加をお控えくださいませ。
