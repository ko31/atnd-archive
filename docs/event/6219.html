catch 〜汚ねえ花火だ〜
description

千葉県の富津海水浴場にて、海水浴・BBQ・スイカ割り・花火などをして、一緒に夏の思い出をつくりましょう！

※花火については、帰宅時間と日の落ちる時間の関係で中止になりました(m;__)m

▼タイムテーブル

・10:00〜11:00　現地に集合

・12:00〜　BBQ開始（自己紹介有り）

・15:00(?) ドラゴンボール割り

・17:30　片付け開始

・18:00　解散

※空き時間は自由に行動してください。

▼持ち物

・水着

・バスタオル（必要なら着替え）

・紫外線防止グッズ

・水遊び用ガジェット（浮き輪とか水鉄砲とかあると海は楽しいです♪）

アクセス方法は各自にお任せしています。

車で向える方（同乗して行ける方）は問題ないと思いますが、

その他の方はバスで行くのが、電車で行くより乗り継ぎが少ないのでお勧めです。

１.横浜駅 東口バスターミナル 18番のりば → 木更津駅　[京浜急行・アクアラインバス] ¥1,500

http://www.keikyu-bus.co.jp/aqua/03.html

１.品川駅 → 木更津駅　[京浜急行・アクアラインバス] ¥1,300

http://www.keikyu-bus.co.jp/aqua/01.html

２.木更津駅 西口 → 富津公園　[富津線バス] ¥300(?

http://www.city.futtsu.chiba.jp/kurashi/bus/futsu.htm

といったルートで現地に辿りつけます。

到着したら、車の先発組が場所を確保していると思うので、合流してください！

※バスでのアクセスは、渋滞の可能性を考え、7〜8時台のバスに乗ると、余裕を持って集合時間に到着できるのではないかと予想しています。
