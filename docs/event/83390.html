description

本年3月14日にMac版のプレビュー1が公開されて以来、大きな反響を呼んでいる『Adobe XD』のWindows版が年内に公開予定です。これはただMac版をWindows版としてポーティングするのではなく、タッチUIなどWindows 10の能力を引き出す『Adobe XD』になるとも発表されています。

また、Windows端末といえば、その高いパフォーマンスで究極の一台と言われている『Surface Book』は、複数のクリエイティブツールを起動して制作を進めるクリエイティブワークにも最適な選択肢です。

この２つの組み合わせが、いかに最高の制作環境を生み出すのか、そのヒントをご体現いただける本セミナーにどうぞご参加ください！

主催：ウチダスペクトラム株式会社、協賛：日本マイクロソフト株式会社、アドビシステムズ株式会社
