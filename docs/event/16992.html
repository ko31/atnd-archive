Windows CE/Windows Mobile/Windows Phone 6.xからWindows Phone 7へのアプリケーション移植　ステップ・バイ・ステップ

組込OSのデファクトスタンダードとして、今でも飲食店の注文システムで現役として使われているWinCE。

そしてPDAの先駆けとなり、W-ZERO3で普及のピークを迎えたWindows Mobile。

それまではバージョンアップしても必ずアプリの下位互換が保たれていたのに、
WP7からは互換が見捨てられてしまう。

せっかく始めたMarketplaceもついにWP6.xまでのものは終止符を打つことが発表。
新規でアプリが登録出来なくなる日ももうすぐ。

たくさんのアプリを後生に残すには移植が必要です。

そこでこの勉強会で、WP7への移植のポイントや、
代替APIがあるかを探しながら、改めて移植の重要店を振り返ることを目的に企画しました。
