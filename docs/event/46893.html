15:15-15:30Session01「WEB制作者の今後のキャリアについての道標を考える」
2014年になりWEB制作の手法はますます変化し、だれでもできるようになりつつあります。結果、制作のみの仕事はどんどんなくなることでしょう。そんな中我々はどうしたらいいのでしょうか。興味を持てるものを探して、それに進んでみる。なるべく仲間やファンを増やしながら、そんなテーマについて一緒に考えてみたいとおもいます。		WP-D Fes #01 Megane Fes in GMO主催
大串 肇さん
15:30-15:45Session02「2013年デザインで使ったサービスツールまとめ」
2013年度デザインをする上でとても便利だなと思ったツールやサービス、アプリの総括紹介をします。今年もよろしくお願いします。		デザイナー／イラストレーター仲野 宏美さん
15:45-16:00休憩（15分間）　
16:00-16:15Session03「業種問わずおすすめしたい！恋したWebサービス7選」
あけました！新年一発目は、昨年心奪われたWebサービスを業種職種問わず幅広くご紹介します。Web業務の人でもそうでもない人も恋に落ちちゃってください♡		株式会社バイタリフィ
千葉 礼美さん
16:15-16:30Session04「プレビュー至上主義 : 2014年のWEBデザイン環境はこれ!」
プログラマが「テストファースト」を唱えるなら、デザイナが主張すべきは何? それが「プレビュー至上主義」です。書くより先に見る、書きながら見る、しかもほぼ最終成果物に近い形で。
現実には、バックエンドと関わりながらでは難題でした。でも、2014年現在、問題はほぼ解決しつつあります。粘土細工のように手を動かしたモノがそのままカタチになる、そんな理想のデザイン環境を作ります。		リブライズ〜すべての本棚を図書館に〜 マスター
河村 奨さん
16:30-16:45Session05 「法人化決意までの過程」
いろんな人がいろんな経験やポリシーをもって会社を設立、経営していると思います。
10年も前の話になりますが、その１例として、私の場合の法人化決意までの過程について発表します。
法人化や独立を検討している人へ何かの参考になれば幸いです。		株式会社ハートビーツ
藤崎 正範さん
16:45-17:00休憩（15分間）　
17:00-17:15Session06 「珍味ベクトル外積3種盛り」
ベクトルの中でも「外積」は、考え方のわかりにくい計算です。けれど、いきなり概念を捉えようとするのでなく、何に使えるかを知れば、計算式そのものは簡単な四則演算です。今回は2次元のベクトルに絞って、3つのインタラクティブなサンプルを例に、外積がどう使われているのかをご紹介します。		野中 文雄さん
17:15-17:30Session07「今すぐに使える! Internet Explorer 11 F12 開発者ツール」
Internet Explorer には IE8 の時代から Web ページをデバッグするためのツール「F12 開発者ツール」が用意されています。
F12 開発者ツールを使用すればオンライン上の Web ページもデスクトップからシームレスにデバッグすることができます。このセッションでは F12 開発者ツールの具体的な使い方について紹介します。		日本マイクロソフト株式会社
