どんなイベントですか。

勉強会とかではありません。
みんなでワイワイ／ガヤガヤとWindows Azureについて自由に語り合う集いです。

ただし、やや濃い内容になってしまうかもしれません。
しかし、前回ほど濃くしない予定です。（笑）
エンジニアでない方にもWindows Azureの素敵な世界を知っていただくために、さまざまなジャンルの話題を提供する予定です。
マイクロソフト品川本社とリモートでつないで、遠隔地座談会も開催する予定です！

また、GeekBarというお店の営業の一環として行われるので、好きな時間にやって来て、好きな時間に帰っていただいてかまいません。（ただしチャージ ￥1,500 のお支払いが必要です）

いきさつ

昨年（2013年6月）、いろんな方のリクエストを受けて「Bar Windows Azure」を開催しました。おかげさまで大盛況でした。
そして、この2月26日、Windows Azureの日本リージョンが開設されたのを祝おうと、第2回を開催します。
