キャンセルについて（5/21追記）

万が一キャンセルする場合は、5/23（水）までに必ずキャンセルしてください。それ以降のキャンセルは、キャンセル料頂く可能性があります。
ただ、キャンセル料徴収するのも面倒臭いので、ドタキャンはくれぐれもご遠慮ください。。。


申し込みについて（5/21追記）


そろそろ人数を確定させたいので、5/22以降の申し込みはご遠慮ください。

タイムスケジュール（5/7追記）

何時に来てもいいですが、10時に来れる人はなるべく10時に来て手伝ってください！


  1. 10:00〜 集合/仔豚撮影会/丸焼き開始

  2. 12:00〜 バーベキュー開始

  3. 14:30〜 こんがり焼けた仔豚を食べる

  4. 16:30〜 後片付け開始

  5. 17:0017:30〜 一旦解散・有志で２次会
