今年もやってまいりました、"Blender BOF in OSC Kyoto"。
オープンソースカンファレンス2013 Kansai@KyotoへBlenderの日本のユーザー会であるBLUG.jpがブースを出展(土曜のみ)するので、そのついでにBOF的な何か?をやってしまおう…というか、お店番してくれる方を 若干名 募集、ということ。
http://www.ospn.jp/osc2013-kyoto/

特に深い意味はありませんが、少しの時間でもブースに立ってみたいという方、お気軽にどうぞ。

オープンソースカンファレンス2013 Kansai@Kyoto は 8月2日(金)、8月3日(土)に開催されます。BLUG.jpは8月3日(土)のみブース出展します。

やること

  • お店番をする( 一瞬でももちろんok)
  • 飴ちゃんをくばる
  • blenderについて説明してみる

利点

  • blender知ってそうな人、興味ありそうな人と話が出来る(出展側として、会場に来た人と)
  • (ATNDで参加表明することで)盛り上がっている感を演出

注意点?

*「少しのぞくだけー」の方も登録しちゃってくださいな。参加はいつの時間でも構いませんので。

  • bofとは言ってもブース前で内輪だけで喋っていることは無いようにね。

ということでよろしくですー。
