description

職場や友人間で必ず避けられない「人間関係」。
普段はコミュニケーションがうまく取れていても、どうしても「こんなタイプの人は苦手」、「どうもしっくりこない」といった場面に遭遇することがあります。
このセミナーでは、様々なパターンでのコミュニケーションの取り方をご紹介します。
どんな場面でも人間関係のトラブルから切り抜けられる方法を学び、いつでも快適に過ごせるコミュニケーションを学びませんか♪

～セミナーの内容～

・人間関係は変えられる
・すぐ揚げ足を取る人との会話
・反論ばかりしてくる人との会話
・ネガティブな発言を繰り返す人との会話
・過剰なおせっかいの人との会話
・権威にこだわる人との会話

～このセミナーはこんな方におすすめです～

・職場や友人との人間関係を円滑に行ないたい
・コミュニケーション力を身に付けたい
・上司、部下の間でのトラブルを解消したい
・いつでも楽しい会話を繰り広げたい
・人と接する機会が多い
・いつでも人間関係がうまくいく環境にありたい

etc..
