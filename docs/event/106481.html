description

「胎内記憶」という言葉をご存じですか？

誰にも話していないし、知っているはずもない、
生まれる前のことを子どもが語りだした、
という話があります。

産婦人科医の池川明先生ほか、
胎内記憶の専門家が出演しているドキュメンタリー映画
「かみさまとのやくそく～あなたは親を選んで生まれてきた～」
によると、

子どもがおなかの中にいる時も
お父さん、お母さんの声は聞こえていて、

お父さんが声かけをたくさんしていると、
生まれた後もにっこり笑ってくれるようなのです。

お父さんが声かけをしていないと、
顔を見た時に笑わなかったり、泣いてしまうようなのです。

「おなかの中のお子さんに、絵本を読んであげて欲しい！」
「お子さんとのコミュニケーションを良くするきっかけにして欲しい！」
「奥さんとのコミュニケーションも、もっと良くなって欲しい！」
そんな思いで「お父さんのための朗読会」を開催いたします。

夫婦で参加されるもよし、
ご主人おひとりで参加して、奥様をびっくりさせるもよし。

ピンときた貴方！ぜひ遊びにいらしてください！
いつの間にか忘れていた「内なる子供」を思い出すかも知れませんよ。

★こんな方におすすめ
・まもなく父親になるけれど、何から準備したらいいのかわからない方

・妻が妊娠してから、コミュニケーションがイマイチうまくいってないなぁ
　と感じている方

・生まれてくるわが子に、ニコッと笑ってもらいたい方、ことさらに泣かれて
　欲しくない方

・絵本の読み聞かせに興味のある方

・日々仕事に追われ、何とか家庭とのバランスを取りたいと思っている方

埼玉県蓮田市生まれ、八王子市在住。
高校時代に声楽、大学時代に演劇を学ぶ。

次男が通う小学校での「絵本の読み聞かせ」をきっかけに、
2016年11月より「朗読＆ゲームの会」を継続開催。
2019年4月、般若心経朗読会を主催するとともに、
Double Spin プロデュースの朗読劇”cry!!”に出演。

この他にも、八王子市内の保育園で保育士向けの朗読を行うなど、
朗読活動を精力的に実施。

朗読を通じて人と人とが繋がり、
笑顔が生まれて新たな繋がりができていく。
そんな世の中を創るため、絵本の読み聞かせや朗読活動を展開中。
