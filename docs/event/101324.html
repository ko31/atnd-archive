description

2018年の10月2日、Windows Server 2019 がGAしました。 「クラウドシフト」を優先させるべく、しばらくメッセージを控えてきたWindows Serverですが、Windows Server 2008/R2 EOS の流れも含め、あらためて活動を本格化させ、11月5~7日の Microsoft Tech Summit 2018 でも、セッションやブースで情報発信を行う予定にしています。

そして、Windows Serverに関係するコアな皆様とともに、その流れをさらに加速させるべく、Microsoft Tech Summit 2018最終日の夜に、パーティを開催することになりました！

協業・競合関係の枠を超えて、これまでのWindows Serverビジネスへの労をねぎらいながら、皆様で楽しい夜にしたいと考えております。関係者の皆さまは、是非ともご参加ください。
