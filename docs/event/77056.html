テーマ

モブキャストでは新作ゲームを短期間で開発すべく独自の開発スタイル「MSGD(Mobcast Style Game Development)」を推進しています。
今回はその「MSGD」において新規ゲーム開発への興味をお持ちの方を対象に
具体的にどのような取り組みを実施しているのか、お話をさせて頂きます。
合わせて実際にMSGDによるプロト開発に携わりたいフリーランスの方を求めており、モブキャスト独自の「プロ契約」という働き方についてもご説明いたします。
軽食とお酒を準備しておりますので、現在フリーランスとしてお仕事をされている方やフリーランスを検討されている方も、お気軽にお越しください！

求めている業務ポジション

プロト開発を行うフリーランスのエンジニアを募集中。
1～2か月間の短期集中、常駐、週5勤務ができる方を希望！
Unityを使って個人でゲームアプリを何本も開発している経験をお持ちの方、
企画から開発まで一貫してゲーム開発を行なった経験をお持ちの方をお待ちしております。
