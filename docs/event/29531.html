description

「ITパスポート」は、国家資格ですが、それを持っていると就職ができる！、何かになれる！というものではありません。
医師や看護師、弁護士の資格とは違います。

でも、パソコンやスマートフォンを仕事・生活に活かし、自分が直接持ったり利用したりしていなくても、乗り物の運行、銀行のATM、炊飯器・洗濯機などの家電、数え上げたらきりがないほど、ITの恩恵を受けています。
また逆に、情報漏えいや、何気なく発信した写真や情報が、とんでもない人のところまで届いたりと、少なからず被害にもあっています。

今や、IT無くしては、仕事・生活が成り立たない！といっても過言では無いほど、好むと好まざるとに拘わらず、私たちの仕事・生活の基盤になってきています。

私たちはどれほど、この「ICT（IT）」について知っているでしょうか？
「ICT（IT）」を知らなくても生活に支障はないかもしれません。
でも社会で働くことを考えると、「知らない」では済まされない問題が起こってきます。
自分のことだけでは済まされない状態も考えられます。
セキュリティを考えるには、少なくとも初歩的な構造だけでも知っておいたほうが良いでしょう。
せっかく便利になるなら「IT」を使いましょう。
「IT」を活用するには、しくみを知っておいたほうが、より活かすことができます。

今まで私は、システム開発やプログラム製造など「作る」側に居たので、
もっともっと「ICT（IT）」を利用・活用してもらいたいと強く思うのです。
「使う」側にたって、便利に使うことを実践しています。

国家資格である「ITパスポート」は、「利用者」「開発者」の両者共通の情報技術基礎知識の証明となります 。
社会人でも学生でもITパスポートの資格を有することは将来や自身の資質を高める上でも有益となります。
ストラテジ・マネジメント・テクノロジの全般に幅広い知識を有している証となります。

主な対象者：
　　新入社員、一般社員、IT知識を習得したい方、職業人としての知識を身に付けたい技術者
前提条件：
　　前提条件はございません
主なカリキュラム：
　　1.企業と法務・経営戦略
　　2.プロジェクトマネジメント
　　3.基礎理論
　　4.コンピュータシステム

「ITパスポート」は、企業活動・ビジネスに欠かせない経営全般・IT管理・IT技術について、幅広い基礎知識を持っていることを国が証明します。

ＯＪＴで業務に必要な専門知識を習得できますが、お客様への直接の提案には、お客様である経営者の方が抱えている課題に関してピンポイントで会話ができるスキルが必須になります。つまり、若い社員が経営者の目線で会話できるスキルを一日も早く身につけるのが非常に重要になっています。

ところが、これは非常に幅が広く、研修やＯＪＴで特定のスキルを身につけるのとはわけが違い、効果的で体系立った学習法が今のところ確立されていませんでした。 そこで、先ず手始めに「ITパスポート」の取得をお考えになっては如何でしょうか。

多忙な社会人の学習方法として、現業と平行で学べるように夕方から週２回のコースを用意いたしました。
