description 株式会社ティーケーピー(本社：東京都新宿区、代表取締役社長：河野貴輝、以下TKP)は、12月22日(木)に「TKPガーデンシティ品川」ボールルームで広瀬香美のコンサートを開催します。 本イベントは「広瀬香美　『香美別邸2016』　ピアノ弾き語りライブ　」と題し、広瀬香美の歌声とピアノだけで盛り上げる「実力勝負」の弾き語りツアーで、好評をいただいた2015年に引き続き、広瀬香美の情熱的でパワフルな歌声と可憐なピアノ演奏の凄みを、「TKPガーデンシティ品川」からお届けします。 ピアノ弾き語りの「香美別邸」を開催するのは都内ではこちらの公演のみとなります。2015年はビュッフェ付きのディナーショーを二部制で開催しましたが、2016年はライブに重点を置き、一部制で2時間に渡る広瀬香美のステージをご堪能いただけます。

「TKPガーデンシティ品川」は品川駅から徒歩1分と交通アクセスが良く、ご家族連れでも楽しめるよう4歳以上の入場が可能です。ご来場いただいたみなさまへ、クリスマスシーズンのひと時を素晴らしい音楽と快適な空間を共に愉しんでいただける時間をお届けいたします。
