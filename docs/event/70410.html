あなたのお家にいらないもの、ありませんか？

白門祭フリーマーケットとは？
　「多摩地区最大級のフリーマーケット」です！　昨年はなんと60組を超える出店者さんと、5000人近くの来場者さんで賑わいました！
今年は「交流を楽しんでもらう」ことをコンセプトに開催します。

服、アクセサリー、靴等、普段から使えるものが格安価格で販売されています！

今年のテーマは「ハロウィン」。豪華で美しい装飾が、中大のセントラルプラザをハロウィンの世界観に染め上げます。
また、交流が楽しめるイベントも各種用意しています！　ここでしかできないイベントもあるかもしれませんよ…？

さらにさらに！　このフリーマーケットは出店料が無料！　受付から出店まで、当団体のスタッフが丁寧に対応いたします。
是非出品してください‼
