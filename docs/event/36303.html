description

●ＳＰＩの点数の取り方を学びます。
　・ＳＰＩ性格検査から判断する「優秀な人材結果」を学びます。
　・「優秀な人材結果」と自分のギャップを埋める方法を学びます。
　・適正検査⇔エントリーシート⇔面接の一貫性を理解します。
　・一貫性が確保できているか確認します。

★それができると、どんなメリットがあるの？
　・優秀な人材像としてアピールできます。
　・面接人物像に違和感がなくなります。

★適性検査対策ができたら・・次は何をすればいいの？
　・採用担当者に与えた「優秀な人材」イメージに合う
　志望動機
　やりきったこと
　長所・短所
　自己ＰＲ等を記入し、
　「優秀な人材」であると証明を強化します。

★証明の強化ができたらどうなるの？
　・採用担当者は、会ってもっと確認したいと思い、面接に進ませます。

◎ＳＰＩ攻略法

◎ＳＰＩ性格検査簡易版実施

◎ギャップを埋める行動
◎適性検査対策

受講のメリット

◎どんな仕事が自分の価値観に合うのか理解できます。
◎適性検査対策について理解でき、計画が立てられます。
◎性格検査の重要性が理解できます。
◎採用担当者にイメージさせる「優秀な人材像」が理解できます。
◎なりたい自分の性格に変わる第一歩を踏み出します。
◎エントリーシートに繋げる「強み」が理解できます。
