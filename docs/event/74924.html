2017卒 採用を成功させるための最新メソッド
～採用勝ち組企業が行っている差別化とは～

□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■□■

昨今の求人倍率の向上により、年々採用は厳しさを増すことが安易に予想出来ます。
人財を採用する為には「攻め」ることが重要な要素として捉えられがちですが、
勝ち組企業が意識している「守り」こそ競合と差別化を図る最重要要素です。

エントリー数、面接辞退率、内定辞退率等、
採用における現在のＫＰＩの各数値は本当に適正なのか、
疑うことが勝ち組企業になる第一歩です。

【以下に?がつく事業主様、お悩みのある事業主様、是非、ご参加下さい！】

□求人に力を入れている。
□競合と比較された際に自社を選んでもらいたい。
□優れた人財を採用したいと思っている。

セミナースケジュール -———————————————-

受付：14:45～

一部：15:00～16:15
【2016卒市場のまとめと2017卒採用のポイント】
採用を成功させるには、過去のデータより市場を分析し、
採用のポイントを押さえることが不可欠となります。
自社だけではまとめきれない過去データ、惜しみなくお伝えします。

二部：16:25～17:40
【採用勝ち組企業が実践している守りとは！？】
採用勝ち組企業が行っている「守り」の差別化とは！？
守るべき場所、タイミング、手法、余すことなくお伝えします。

終了：18:00
