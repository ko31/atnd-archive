イベント概要

今年もFuelPHP Advent Calendarをやりたいと思います。

FuelPHPは、今、日本で最も注目されているオープンソース(MITライセンス)のPHPフレームワークです。FuelPHPに関する情報は、FuelPHP まとめ Wikiを参照願います。

このイベントは、技術系アドベントカレンダーの一環で、アドベントの期間中、参加者が順に1日1つずつ、FuelPHPに関する記事をブログに書いていくというものです。12月1日から12月25日まで先着25名です。

25名に満たない場合は、希望により複数回書いてもかまいません。内容は FuelPHPに関することなら自由です。

FuelPHPユーザでなくてもFuelPHPに興味があれば参加していただいてまったく構いません。

電子出版

ブログの記事をまとめて無料の書籍として電子出版します。出版したくない方は参加しないようにお願いします。

参加方法

1. このATNDに参加登録します。

2. 下記Googleスプレッドシートの担当したい日付に記入してください。
https://docs.google.com/spreadsheet/ccc?key=0AvXVonuYHbBsdHBqbjBfbzR4Z2RBa0l2c3RRTl91RHc

3. その日になったら自分のブログなどで記事をアップしてください(予め書いておいて担当日に公開するのが楽だと思います)

4. 書いた記事のURLを、Googleスプレッドシートの「記事URL」に貼り付けてください。書いたことをハッシュタグ(#FuelPHPAdvent2012)付きでツイートしてもらえると助かります。

※ 担当日は早い者順なのでお早めに。
