Cheese Rush!!

description

前回、大人気だったチーズだらけの交流パーティー「Cheese Rush」を大崎ゲートシティーにあるワインダイニングで開催!!!

3月27日（木）に大崎ゲートシティーにあるカジュアルなワインダイニングで厳選したチーズやワイン、日本酒を楽しみながら交流を深めるパーティーを開催します。

会場内には専門家による厳選したチーズはもちろん、チーズのスイーツやチーズに合う日本酒やワインがズラリと並ぶ予定です。

■ラクレットを食べつくしちゃおうっ!!
あの「アルプスの少女ハイジ」のワンシーンで出てくる「あのチーズ」をみんなで楽しみましょう＾＾

■モッツァレラチーズを皆で手作りしよう
当日はモッツァレラチーズを手作りできるブースもご用意しています！できたてのモッツァレラチーズは絶品ですよ♪

■店内には専門家が選んだ様々なチーズがズラリ
会場内のビュッフェ台にはチーズの伝道師が選ぶ様々なチーズがズラリと並ぶ予定です！！
会場で焼いた自家製フランスパンと一緒に楽しんでくださいね。

■チーズに合わせた日本酒やワインをご用意!!
当日はこの日の為にご用意した日本酒やワインが楽しめます！

■チーズを使ったスイーツもご用意します！

■他にもビュッフェ料理やドリンクの飲み放題も、もちろんご用意しています！！

初めてお会いする方同士でも交流を深める事が出来るコンテンツが充実！！

Cheeseだらけの楽しい雰囲気の中で専門家が厳選したチーズやワイン、日本酒と一緒に様々な業種業界の方と交流を楽しみましょう。
