description

あなたは

・終身雇用と年功序列の終了が意味すること
・今の年金制度が将来必ず破綻すること
・人工知能の発達がもたらす常識の変化

これらの問いにはっきりとした答えを出すことができますか？

退職金と年金が保証された豊かな時代は終わり、いま私たちはとても不安定な時代を生きています。
人口ピラミッドが逆転したため、年金制度は確実に崩壊します。NISAやDC年金の裏のメッセージは「もう国や会社で退職金の面倒を見切れないので、自分の身は自分で守ってね」ということです。

しかし年金や退職金が保証されていないこんな世の中でもまだ「会社にいることが安全・安心・安定」という人が多いのが現状です。
もちろん「いま」は仕事があって給料があって、ある程度安定しているでしょう。

しかし10年後はどうでしょうか？
人工知能が発達し、いまある仕事の70%以上がコンピュータに取って代わるそうです。
30年後はどうでしょう？
退職金と年金がない状態で老後を楽しむ余裕があるでしょうか？

さて、ではもう一つ質問です。
漠然とした不安を持っている方が多い中で、実際になにか行動を起こしている人が少ないのはなぜでしょうか？

それは「いつ起こるかわからないから」です。
いつ年金・退職金がなくなるかはわからない。いつ自分の仕事が人工知能にとられるかわからない。いつ自分の生活が危うくなるかわからない。

地震と同じですね。
いつ起こるかわからないから事前に対策する人が少ないのです。

しかし確実に「その時」は来ます。
そして私たちは「その時」に向けて確実に時間を過ごしているのです。

それでもまだ「いま楽しければいい」「いまは安定してるからいい」と言えますか？
「その時」に向けていま絶対に知っておかなければならないお金の教養を分かりやすくお伝えします。
