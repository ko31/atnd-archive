仙台で Perl の話が思いっきり出来る場所

やりましょう、今回も居酒屋で Perl 談義

仙台、宮城はもちろん、東北など近県の Perl ユーザのご参加お待ちしております！

最近の Perl 動向を探るもよし、自分の作った Perl コードを披露するもよし、Perl な昔話をするもよし、Perl の話をしないのもよし、初めて参加される方でも安心、優しいお兄さんたちが、しっかりエスコートさせていただきます！

Perl の上級、初級は問いません。少しでも興味ある方、是非ご参加ください！

PHP ですけど、Ruby なんだけど Perl てどんなん？って方も大歓迎！

10月には YAPC::Asia 2011 Tokyo も開催されます。仙台でも Perl 盛り上げていきましょう！

PM とは…

Perl Mongers（パールモンガース）の略。地域ごとの Perl ユーザグループの集合体です。発祥はニューヨーク。現在は、世界各地に Perl Mongers が分布し、それぞれの地域でのコミュニティーを形成しています

Sendai.pm（準備室）は、そんな Perl Mongers を仙台にも作るべく発足しました
