概要

「お客さんにとって価値のある成果を毎週届けるには、どうすればいいんだろう？」
ここは、アジャイルサムライの輪読を中心に、その疑問について考えを深めていく場です。

事前準備

毎回、2章ずつ進める予定です。
「アジャイルサムライ ――達人開発者への道」の以下の章を事前に読み、疑問点や気になった点、重要だと感じた点などをメモしておいてください。

第3章「みんなをバスに乗せる」
第4章「全体像を捉える」

なお、今回はインセプションデッキを実際に作成するワークショップを通して、アジャイルサムライに書かれている内容を実感する場にしたいと思います。これまでの回に参加していない方でもついていける内容ですので、お気軽にどうぞ。

なお、書籍は、 amazon や オーム社 などから入手可能です。
オーム社のものはPDF版もありますのでお勧めです。
