Beginner's Disco 〜Club Event for Club Beginner〜

title Beginner's Disco 〜Club Event for Club Beginner〜
catch
description

クラブ行ってみたいけど行った事ない。。
そんなあなたの為にやりますよ、クラブ初心者のためのクラブイベント！

全員がビギナーなので安心！クラブの雰囲気だけを味わえる最高の入り口となっております。

■ここが違う！■
①日曜昼間に開催！夜のアフターは各自でお願いします。
②豆乳や青汁など使用した健康カクテルをご用意！
③踊れなくても大丈夫！インストラクターによるエアロビもコンテンツとして用意してます。
④EDMからJ-POPまでジャンルを横断した楽曲をご用意。 皆が知っている曲ばかりを選んでいます！
⑤音量控えめの交流&レクチャータイムをご用意。クラブでありがちな音がうるさくて会話できない。という事がないように交流タイムも設けております。

従来のクラブの概念から離れた、
全く新しいクラブ初心者のためのイベントです！
※勿論、普段から通っている方もおいでくださいませ！

■対象者■
・クラブに興味あるけど1人じゃ行けない
・クラブミュージックには興味ある
・ちょっと身体を動かしたい
・おしゃれなクラブに行ってみたい
・みんなで騒ぐのが好き
・クラブが大好きでよく通っている
・日曜日の昼間を有効に活用したい
・全然踊れないけど、そういうノリは好き
・でもチャラいノリは嫌い
・なんかおもしろそう
という方は是非是非お越しくださいー！

■詳細■
日時：6月5日 14:00〜16:30 (OPEN 14:00)
場所：EDITION(港区赤坂8-10-22 NEW SHINSAKA bldgB1)
会場は雰囲気を楽しめるように1流の場所を用意しております。
https://www.facebook.com/edition.aoyama/
定員：80名
料金：学生￥2,000／社会人￥3,000（いずれも前売り)
　　　※ワンドリンク付き
　　　※18歳以上入場可能（未成年はアルコール不可）

■注意■
・ネットワークビジネス関連の勧誘、不適切な行為や言動（ナンパやセクハラ行為）の見られる方はご退出頂く場合が御座います。

<Q&A>
Q1、変な男がいそうで怖いんですけど（女子目線）
安心してください。チャラチャラナンパ野郎を見つけた場合は、主催者が責任を持って追い出します。何かあればすぐに主催者を呼んでください、飛んでいきます。

Q2、人見知りなので1人で行くのが不安です
一人の人でも楽しめるよう主催側が参加者の皆さんを終始お繋げ致します。
クラブでよくありがちな喋る人がいないという状態にならない事に努めます。

Q3、全然踊れないですが
安心してください。踊れなくても大丈夫なんです。知っている音楽で気分がノッてきたら手を振ればいいんです。

主催：Beginner’s Disco実行委員
