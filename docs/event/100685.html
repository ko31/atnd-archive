■開催概要

おまたせしました。10月の遊部＠ヒロロ　10月14日（日）に開催です。

今月も遊びたいゲームを持ち込んでの「遊部」です。いつもどおり、無線LAN（ヒロロのWiFiとは別）と電源タップを用意します。

コレを持ち込むぜ！と事前に連絡があったゲーム機等は以下の通り

  1. スプラトゥーン2
  2. 大乱闘スマッシュブラザーズ for WiiU
  3. バーチャファイター5 Final Showdown
  4. ファミリーコンピュータミニ
  5. スーパーファミコンミニ
  6. PSVitaTV、PlayStation4
  7. ボードゲーム

その他、持ち込みたい！持ち込むぜ！と考えてる方は、事前に連絡をお願いします。
せっかく持ってきいただいても電源や場所の関係で設置できない場合もあります。

■持ち込みについて

ニンテンドースイッチは以下の点をご確認ください

  1. みまもり設定
    →　解除もしくは解除番号が解らないと会場の無線LANに接続できません
  2. ニンテンドーアカウントへの登録
    →　登録されていないとフレンド交換等ができません
  3. ニンテンドーオンラインサービスへの申込み（有料）
    →　申し込んでないとネットを介しての対戦ができません

持ち込む前におウチで上記内容を確認ください。遊部の会場やスタッフでは対応できない場合があります

　自分のゲーム機やコントローラー、ケーブルなどは、自分のものであるとわかる印などをお願いします

　スマートフォンやタブレットのゲームアプリなど、常時通信しながらプレイされるものに関しては、会場の無線LAN接続をご遠慮いただく場合がございます

　他人のゲーム機やコントローラーを使う場合、破損等がないよう注意してプレイください

■その他注意事項

・大人も子どもも楽しく遊ぶ場です。中学生未満のお子さんが参加する場合は、申込みも含めできるだけ保護者の方も一緒に来場し、一緒に遊んでコミュニケーションを深めましょう。
・年齢性別に関係なくいろんな人と交流してください。参加される大人の方は、プレイして自分の強さを見せるだけでなく、「ゲームのおかげで勉強も楽しくなった、結婚できた、今の仕事でも役立ってる」など、参加してる子ども達にゲームで得た経験や楽しさを伝えてもらえると嬉しいです。
・スプラトゥーン2やスマッシュブラザーズなど、多人数で遊ぶゲームでは、途中で「あっちのゲーム遊ぼう」と中抜けしないように。負けてても悔しくてもゲーム開始から終了まではプレイしましょう。
・運営スタッフからの指示にご協力をお願いします。
・会場での個人的な荷物は運営側で管理・把握しきれませんので、ご自分で管理ください。
・会場内で撮影しSNS等にアップされる場合、他の参加者が特定しにくいような加工等をお願いします。

飲食したゴミなどは、可能な限りご自分でお持ち帰りください。

その他ご不明な点や質問があれば、お気軽にこちらのページか主催者までご相談ください。
