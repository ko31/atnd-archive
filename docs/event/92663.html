■セミナー概要
BENLY、KIZUNA、Lisutoが集結！ 腹に落ちる生の越境EC/インバウンド情報をお届け！越境EC最前線の担当者が登壇！ 越境ECインバウンド 結果を出している3社が集結！

こんなお悩みありませんか？海外展開に興味があるけどリスクが怖い。海外で売れるイメージが湧かない。いくらかかるのか、いくら儲かるのか想像つかない。社内リソースが足りなくて着手できない。本セミナーに参加すれば そのお悩み解決できます！ このセミナーにご参加いただければ、海外トレンドや海外販路開拓手法を 現役越境EC最前線で働く担当者から具体的に 話を聞けます。

なかなか機会のないチャンス。 ぜひ、この機会に世界のボーダーラインを越えていきましょう！
