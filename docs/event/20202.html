description

Unityは、今年のCEDECプログラミング・開発環境部門において最優秀賞に選ばれ、今大注目のゲームエンジンです。そこで、これからUnityを勉強してみたい、何かを作ってみたい、という人のために勉強会を開催します。Unityに興味のある方なら、どなたでも参加可能です。

イベント内容


  • 前半: 1時間でFPSを作る
    Unityの基本機能の解説の後、参加者全員でFPSゲームを制作します

  • 後半: Unityスクリプト入門編
    ゲームで必要な動作を実際にスクリプトを使って、参加者全員でゲームを制作します

参加対象
Unityを使ったゲームなどのコンテンツ制作を勉強したい人(社会人の方でも参加可能です)
