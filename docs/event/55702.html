description

アベノミクスの成長戦略の一つ『女性活用、女性管理職を2020年迄に30％』という宣言があり、『ウィメノミクス』というキーワードで、東京で９月12日（金）から9月14日（日）に開催される「女性が輝く社会に向けた国際シンポジウム（WAW! Tokyo 2014）」に連動し、9月は全国で女性が輝く社会を目指す「シャイン・ウィークス（Shine Weeks）」と位置づけ女性の活躍促進のムーブメントを起こす期間となっています！
女性をはじめとして、高齢者の方々、障害者の方々、海外から日本に来られたいろんな方々がダイバーシティという概念を正しく理解し、未来ある世界を一緒に切り拓けたらと思っています。
