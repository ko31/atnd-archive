catch 顧客の要望に応える提案活動のために
description

　インターネットの普及やメディアの多様化により、消費者を取り巻く環境が大きく変化しました。容易にアクセスできる情報源の登場により、取得可能な情報量は大幅に増えています。例えば消費者の購買プロセスについては、インターネット上の情報により、商品を比較検討したうえで購入を行うことが一般的になりました。

　また、購入後に、その商品についての感想をブログやソーシャルネットワークサービス（SNS）などにより共有をすることも珍しくありません。

　消費者に対し、豊富な情報を効果的に提供するため、インターネット活用のほか、ポスターやチラシ、新聞、テレビ・ラジオなどのマス媒体と組み合わせるマーケティング戦略の一環として、「クロスメディア」が注目されています。

概要

　本講座では、印刷業界をはじめとするメディア提供事業者にとって、顧客の要望に応え、広範な提案をするために不可欠な情報として、クロスメディア動向概要や事例について紹介をします。

講師

田上 歳哲（たのうえ さいてつ）

　第1期クロスメディアエキスパート認証試験に合格。インハウスデザイナーとして活動後、商業美術印刷会社に移籍。グラフィックソフトの販売部門にて、広報兼マーケティング・コミュニケーションを担当。マニュアル制作部門では海外向けプロモーション用コンテンツ作成、電子マニュアルの多言語化、多媒体への技術資料の同時発信、顧客製品のコンテンツの試作など、数多くのメディア情報業務に携わる。現在、クロスメディア関連の講師活動や記事執筆を展開中。
