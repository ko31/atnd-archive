グラントウキョウサウスタワー2Fの受付にお越しいただき、

イベント概要

web業界の中でも、最近特に人材ニーズが高まっているwebデザイナー。
企業から正社員として求められるwebデザイナーのスキルや働き方について、
企業からの声や具体的な選考事例などから考えてみる勉強会です。

講演の他に、ご希望の方はキャリアアドバイザーとの簡易相談会にもご参加いただけます。
講演・相談会ともに費用は無料ですので、お気軽にご参加ください。

Web CAT Studio （株式会社リクルートエージェント）
