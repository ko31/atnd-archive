description

重力／Nｏtｅでは、来る9月に「オープン・ディスカッション」を開催します！
作品のつくり手と観客が上演にむけて話し合いを積み重ねることで、「演劇が生まれる場所」を立ちあげていく試みです。

通常のワークインプログレスと違って、今回は舞台作品が生まれる前に必要な創作コンセプトを探す現場を体験してもらいます。
ヘンリック・イプセンの代表作『人形の家』から浮かび上がるテーマや問題意識についてディスカッションをしていき、約2週間かけて集まったメンバー全員でこれからつくる舞台作品の素材となる言葉やイメージを見出していきます。

またディスカッションが終わったら、参加したメンバーには事務所の壁になにか書き残してもらいます。こうして記録・蓄積されたアイデアや話し合いから生み出された言葉やイメージ、そして参加者の言葉ひとつひとつが、12月と来年2月に予定している『人形の家』の上演に結びつきます。

いつも何気なく観客席で観ている演劇。それが舞台作品になる前の試行錯誤を覗いてみるとともに、上演に向けたコンセプトがまとまっていくプロセスに参加してみませんか？
