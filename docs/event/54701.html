description

はじめまして、勉強会主催者の中元学といいます。
現在は大阪に住んでいまして、大阪、神戸で勉強会を開く講師をしていますが、もともとは僕はサラリーマンでした。大学も四国の大学、愛媛大学出身です。

学生時代から将来独立して何かをしたいなぁとは思っていましたが、そこから先の一歩もどのように出していいのか、わかりませんでした。
新卒で一旦就職したものの、1年で退職し、若さと行動力に任せて、何もわからないまま、25歳の時、ワーキングホリデーでシドニーに行き、語学と海外の感覚を身につけてきました。
帰国後、専門商社に入り、海外営業のサラリーマンへ。
お金の勉強、社会の勉強と思い、株を始めましたが、感情まかせなトレードで結果が出せず、コツコツ増やしては、ドカンと負けてしまうトレード方法でした。
いわゆる普通の「投資は趣味で株をやってます」という人でした。

そこでなんとかしないといけないと思い、いろいろと友人と話したり、コツコツドカンしない方法を考え始め、感情に左右されない方法にたどり着きました。

やがて貿易関係から自分で自営業を始めることとなりました。
そして、投資では、リーマンショックの時にはトレードでも周りには無い成績を収めることができ、そこからすべて利益確定し、不動産投資、そして海外投資へと進みました。

当時を振り返って思いますと、投資や運用などの観念がなかったのですが、大きな動機付けになったのは、キャッシュフローゲームだったと思います。
社会人を4つに分けた考え方や、お金と労働、そして自分の時間との関係など基本的なことの勉強になりました。
そして、いろいろなビジネスをしている中で、日本を取り巻く環境や、自分自身の時間とお金の使い方、そして将来設計の大切さを痛感しはじめました。
現在では、10年前に独立開業したビジネス、そして不動産投資、海外投資をしながら、さらに海外の状況皆様にお伝えして、将来設計のサポートをしています。

今回のセミナーは、難しい話はありません。
自分の転機のきっかけになったキャッシュフローゲームをしながら、普通のサラリーマンだった僕が、独立するきっかけになったことや、そこから投資、運用をし始めたこと、失敗や成功、またその中身などをお伝えするとともに、富裕層が活用している海外の面白いスキームをご紹介させていただこうと思っています。
またこの四国でのセミナーが良ければ定期的にキャッシュフロー勉強会を立ち上げていきたいと思っています。
是非会場でお会いしましょう。

中元　学

1973年生まれ　41歳　大阪市在住
報徳学園高校　愛媛大学卒業
有限会社ディー・エヌ・ティー　代表取締役
アジアビジネスリンク　代表

1997年社会人になった後で、ワーキングホリデーで単身渡豪。
フィッシャーマンズワーフで働きながら、語学を習得。
帰国後、専門商社に就職し、アメリカ担当として、海外営業、輸出業務に携わる。
2002年に衣料品販売として独立開業し、2006年頃から不動産投資、2010年から海外投資を開始。現在、アジアビジネスリンクとして、アジアと日本を結び付けるビジネスを展開していて、そのアドバイザーとして従事している。
現在、大阪、神戸を中心に海外事情を通した資産形成の勉強会を行っている。
