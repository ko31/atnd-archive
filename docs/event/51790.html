セッションの目的

日本企業を中心に、なぜイノベーションが生まれにくいのか。
イノベーションを阻害する要因や、新しい価値観を共有して活動をするため第一歩となるきっかけを提供します。

対象者

どなたでもご参加いただけます。

詳細説明

「会社の中でイノベーションが生まれにくい」。

昨今、日本企業の閉塞感とともに聞かれる言葉ですが、 なぜイノベーションが生まれにくいのでしょうか。
イノベーションを阻害する要因は、どこにあるのでしょうか。

社内メンバーを巻き込み、新しい価値観を共有して活動をするために必要なポイントについて、 ダイアログ（対話）を通じて考えてみませんか。

これからの企業が求められるものがCSRやCSVと表現されるように、今後は、一社では解決できない 事業課題がますます増える社会環境において、イノベーションを起こすためには社内概念を新しく変えていく必要もあります。
知恵を発揮し、互いに持ち寄り、新たな一歩を踏み出すためのセッションをお届けします。
