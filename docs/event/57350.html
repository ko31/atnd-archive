開催情報

開催日時: 11/1 14:00-17:00
開催場所: 喫茶室ルノアール　神田南口駅前店マイ・スペース 2号室
参加資格:Blenderのソースコードに興味のある方(実際にソースコードをコンパイルしたことがない人でもokです。)
参加費: 会議室使用料を人数で割ったもの(400円)+一杯分の注文 (合わせてだいたい1000円いかないくらい?)

今回はBlenderのソースコードにはじめて触る、ということになりますので、基本的な部分から、以下のような内容を参加者全員で話し合っていく形式にしたいと思います。

会の流れ

担当 内容
@lab1092 * 「自分が見たいファイルはどこですか？」- 見たいファイルの探し方
* 「Blender2.5からBlender2.7の変更点で見るBlenderのソースコード」
(コードリーディングを始める前段階くらいの話題を中心に)
@chyama まずドキュメントのある段階のコードでビルド・実行できる環境を用意しておいて、そこからドキュメントとコミットログを眺めながら最近のBlenderのコードの状態を確認できるような流れにしたいと考えています
(ビルドの対象となるバージョンについては確認中ですが、可能であれば当日までにBlenderのビルドに必要な環境の構築及び実際のビルドに挑戦しておいていただけたら)
@tsurumaru * ビルド環境を整える際やデバッグ時に気になった点
* コードリーディング:Python API 、Blenderゲームエンジン周りを予定
他 お茶しながら(?)ソースコードネタ(参加される人で何かありましたらその話題でも)

(事前に交渉していた @tsurumaru さんが参加できることになりましたので、「特別枠」として参加いただきます。)

必要な事項

当日までの準備

  • Blender のビルドにチャレンジしてみてください。Windowsのビルド手順は @chyama さんのBlog記事 第1回 Blenderソースコードリーディング 準備：Blenderをビルドする を参照してください。
  • gitリポジトリのチェックアウトは時間がかかるため、済ませておくと捗る、そうです。

当日

  • 一応XGAサイズを映せるプロジェクター持って行こうかと思いますが部屋の大きさがそんなに大きくないので、期待しない方がいいと思います。
  • 各自、Blenderのgitリポジトリのチェックアウトを済ませたPCがあると良いです。チェックアウトする対象のリビジョンは最新のもので構いません。
    • よくわからない場合は、 http://www.blender.org/download/ にいって、Source Code タブから、ソースコード(blender-2.72b.tar.gz)をダウンロードの上、展開しておいてください（実際にコンパイルしない場合はこちらでok）
  • 多分電源は確保できるはず。延長ケーブルは持っていきます。
  • なお、当日会場にて飲み物を一杯、個々に注文していただく必要があります。
