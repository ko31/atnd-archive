あなたの得意分野を掛け合わせ、ものづくりにかける想いを伝える！

創業69年目を迎える株式会社住まいず。

鹿児島県の木材を使った「こだわりの注文住宅」を販売しています。

家づくりという舞台で、あなたの得意分野を生かして「ワクワクする家づくり、暮らし方」を届けてみませんか？

「私だったらこんなことしたい！」そんなアイデア持ち込みも大歓迎です。

先輩インターン生とのご飯会とは？

インターンシップのこと、会社のこと、仕事のこと、そして大学生活のこと。
手づくりのご飯を一緒に食べながら、一足先にインターンを始めている先輩とワイワイ語り合う場です。

インターンシップフェアでは聞けなかった、あんなことやこんなことを
コーディネーターと一緒にズバッと質問してみましょう！

もちろん、あなた自身のことも存分に語ってくださいね。

ゲスト企業：株式会社住まいず

”本物”を追求したものづくり

現社長の有村健弘氏で１１代目を数える株式会社住まいず。

霧島を拠点として、注文住宅を中心に建築に関わる企画・設計・施工・監理メンテナンスを行なっています。約３５０年の山仕事を経て、９代目有村吉一氏のときに山林業の看板を上げ、やがて製材業も開始。

そして１０代目の有村吉孝氏が建築の世界へと歩み出したそうです。

「自分の山の木で建てる」「泊まれるモデルハウス」などの独自の取り組みも行っており、広く事業をやりたいわけではなく”本物”を追求していたら自然と今の事業の形になっていきました。

このように、”本物”を追求した、想いのこもったものづくりをされている会社です。
