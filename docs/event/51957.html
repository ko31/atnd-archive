オンラインゲームの開発を一気に加速させる Photon
今回のワークショップは、ユニティちゃんゲームジャム直前の特別講習として、Unity と Photon を使って実施します。 「リアルタイム」×「マルチプレイ」 を短時間で実装するためのデモンストレーションをご覧に入れますので、是非PCを持ち込んでその場でオンラインゲームを作っちゃってください！

　
　　　　　　　 【特別編】ワークショップ内容

< 1 > 複数のキャラクターを同一ルーム内でリアルタイムに同期させる
< 2 > キャラクター同士が弾を撃ち、衝突判定を同期させる

　
　
◆本ワークショップは、ゲームジャム開催前の特別編として実施されます。
◆実装に必要なUnityの操作やスクリプトの解説を中心に行います。
◆ゲームジャムに参加する予定の無い方でも、本ワークショップは参加可能です。
