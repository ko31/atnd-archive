別府の愛を食の力で伝えたい！「welovebeppuミニバル」

2016.8.27(土)に開催の、 「全国自治体職員のつどい第1回全国大会 in大分」 に合わせて開催されるミニ路地裏バル「welovebeppuミニバル」。今回は、県外の人にぜひ薦めたい別府の路地裏の名店16店舗にご協力いただき、チケット1枚で「ワンドリンク＋おつまみ一皿」を楽しんでいただけます。

チケット一般販売は、限定50セットのみ行います！

welovebeppuミニバルは、「全国自治体職員のつどい第1回全国大会 in大分」の参加者のために企画されましたが、それ参加しない方向けにも限定でチケットを販売いたします。先着順ですので、ぜひ、このページからご予約ください。

チケットについて

チケットは、イベント参加のお店で「1ドリンク＋かんたんなおつまみ」のセットと引き換え可能のチケットが3枚綴となっています。
当日発売価格は３枚綴 2,400円です。
ただし、このページにてご予約の方は、前売料金2,100円にてご購入可能です。

参加店について

この飲み歩きイベントの参加店は別府駅から徒歩12分程度までにある17店舗。
参加店一覧と参加時間、メニュー一覧 を参考にしてください。

チケットの引き換えについて

ご予約いただいたチケットの引き換えは、以下の場所で可能です

  • 別府駅改札前　2016.8.27(土) 16:30頃～20:00頃まで
  • 温泉道名人会事務所前(やよい商店街 天狗のお社横)　2016.8.27(土) 20:00頃～22:00頃まで

注意事項その他

  • チケットは、welovebeppuバル参加店舗にてご使用いただけます。
  • 入店の際は、バルで来たことをお店にお伝え下さい。お店によっては、イベント専用席のみで対応し、空席があっても予約などでご案内できないことがあります。
  • お店では店のスタッフの指示にしたがってバルをお楽しみ下さい。
  • お店により、営業時間であってもバル参加時間ではないこともあります。
  • お店の混み具合によってはお待ちいただくことも。並ぶのもバルの楽しみ。情報交換などしながら、のんびりとお楽しみください。また、あとに待っている方がいたら、早めに切り上げて次の方に席を譲って下さい。
  • チケットのご利用は原則1 店舗1 日1 回までです。
  • １回の利用でチケットが3枚必要な店舗もございます。
  • 回数券は複数のお客様で使用可能です。
  • 当日は当日券（冊子）、限定で１枚券も販売予定です。
  • 購入済み回数券・1 枚券は換金・払い戻しには応じかねます。
