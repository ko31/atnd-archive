2014/7/12にお台場で行われるオープンハードカンファレンス 2014 Tokyo/Summer。
遅ればせながらキックオフを行おうと思います。

日時

7/1 19:30-21:30

場所

喫茶室ルノアール　飯田橋西口店マイ・スペース 会議室2

費用

会議室利用として、ワンドリンクオーダー制のところで調整しています。ワンドリンク分のみご負担ください。（ソフトドリンク系です）

内容

・展示・講演者へのガイダンス
・ボランティアスタッフの割り振り
・懇親会のプランニング
・ファミリー体験コーナーのプランニング

スタッフ希望の方へ

ぜひともエントリーお願いします。
今回のスタッフはボランティア形式となりますが、
せめて懇親会だけは無料参加とできるようにしようと思います。

(懇親会はピザ焼き石窯を持ち込んで、無限ピザパーティーを行おうとしています。)
