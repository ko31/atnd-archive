あなたのアメブロ、作った目的は何ですか？

会社のホームページ代わりで使っていらっしゃる方も多いですよね。

「ブログを毎日書いているが反響が無い」と悩まれていませんか？

パソコン初心者の方にも分かりやすく、活用法をお伝えします。

あなたのアメブロが優秀な営業マシーンに変わります！

◆日時 7月6日(水) 18:30～20:30

◆開催場所 SOHO BOX 北浜

◆受講費 3,000円（当日お支払い下さい） 先着20名

◆対象 個人・事業者でホームページを持っていない方。ブログを書いているが反響が少ない方。
パソコン初心者 など ご興味があればどんな方でもOKです。

◆講師 デザイン会社CUBES 代表 小林 英樹氏
http://www.cubes.cc/

1988年から10年間、商業印刷物の制作を得意とする
大阪市内のデザイン会社勤務。
より大きな仕事、クライアントに提案の出来る仕事がしたい気持ちで、
1998年からは、大阪市内の印刷会社に転職し、営業とWeb部門の責任者として7年間活躍。
その後、2006年にデザイン会社CUBESを設立。
大手企業のキャンペーンサイトから個人事業者のホームページ制作まで幅広く活躍中。
従来の印刷会社・デザイン会社・ホームページ会社とは違う視点で、活動の場を広げている。

◆内容
アメブロとホームページの違い。
アメブロを使うメリット・デメリット
アメブロからの新しい展開
ビジネスブログの書き方
質疑応答 ・・・etc
