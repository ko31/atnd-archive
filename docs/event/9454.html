description

日曜の朝早めに出て、渋滞になるべくかからないようにして草津まで言って日帰り風呂に入ろうかというたくらみ。

そもそものぐの代車が７人乗りであることから端を発した。

あさは７時くらいには絶対出発していたい。理想６時。←かとおもったけど、渋滞の中あずみんを聞くのも楽しいよねと思い始めた。←いやいやそれでも8時には出発でしょ。
ぐ車の定員は７。ミスター車が出るならそれも追加。

帰りは最寄り駅orドア前までぐが送り届けるなんと親切なタクシー仕様

いきさき　渋川伊香保ICから吾妻国経由草津温泉
　　　　　　191.9km 約4時間11分（渋滞予測込み）

行きの休憩　上里SA（みすた合流＆みすた車に何人か乗ってね）

帰り　松井田妙義IC経由各自宅もしくはアクセスのよい駅

集合場所　秋葉原駅の東側、ヨドバシカメラよりさらに裏手の国道4号沿い
　　　　　ココイチ秋葉原国道4号店まえ。
　　　　　　　秋葉原国道４号店　　　　　　　

住所 東京都千代田区神田松永町16番地内尾松永ﾋﾞﾙ1階

　　　　　　　わからなかったらぐの携帯に０９０－２３１９－９８２９
