えっ！？
中高年のダイエットは食事制限じゃないってホント？！

およそ40代以上の中高年のみなさま用の、パーソナルトレーニングジム心身健康倶楽部。
その代表の枝光聖人が、この度、蔓延する糖質制限ダイエットに警鐘を鳴らし、楽に続けられる中高年のダイエット方法をお伝えします。40代以上のダイエットは、方法を知ってから堅実に。

更に！中高年のお尻トレーニングの方法もお伝えします！また、先日テレビでご紹介した、自宅で出来る簡単なエクササイズもご紹介しますよ！

脂肪の錘をそっと脱ぎ捨て、「あの頃」と同じように。
あなたの体はまだまだ軽くなり、そして力強く動き始めます。

このチャンス。
逃さないようにしてください。

参加費無料、心身健康倶楽部の各店舗で開催します。
