description

●受講対象
外観品質保証を実務的に管理・統率する管理者・責任者（品質管理・製造・技術など）

●習得できる知識
　科学的外観検査の基本である“フォーゾーンメソッド”，そして目視検査，自動検査の効果的な適用方法とポイント，適用における注意点について解説する。さらに外観品質問題への取りくみ，改善についても，実践的な視点で解説する。

●講座の趣旨
　多くの企業や工場で，外観品質保証が共通的課題になっている。外観品質保証は，難度が高い課題である。主観で左右される要素が多く，曖昧なまま，議論や取り組みが行われることになりがちである。外観品質を保証するためには，外観品質というものの特質を理解し，その上で，筋道をたてた品質保証活動が必要である。
　そこで本セミナーにおいて，外観検査を中心に，実践的外観品質保証のための方法とノウハウを解説する。
