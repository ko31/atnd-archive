イベント概要

関西のiOS Developerが集結！？iOS開発者向けのイベントを開催します。
iphoneアプリの開発で著名なディベロッパーが集まり、UI/UX、画像加工処理技術、CoronaSDKの件など、

また、勉強会終了後は懇親会も開催しますので、講師の方や参加者との情報交換・交流を深めて下さい！

資料の共有にスライド共有サービス「Slidrs（スライダーズ）」を使用します。

Slidrs iPhone/iPad アプリのダウンロードはこちら => Slidrs – AppStore
ブラウザで見る場合はこちら => http://slidrs.net/sessions/210/

発表者と手元の資料を自動的に同期して閲覧することができます。
ネット中継をご覧になるかたも手元で資料を閲覧できます。
会場に来られるかたも、座席によってはプロジェクターが見にくい場合がありますので、あらかじめ設定しておくとよいでしょう。

ブラウザで閲覧することもできるようになりました。アプリと同様にページの追尾機能も有効です。
http://slidrs.net/sessions/210/

参加者のかたの設定手順

  1. AppStoreから「Slidrs」のアプリをダウンロードして下さい
    • Slidrsのダウンロード => Slidrs – AppStore
  2. 起動してセッション番号「210」を入力してセッションに参加する
  3. 右上の「手動閲覧中」を押して「自動閲覧」にする
  • 後は放っておくと自動的に資料切替やページめくりが追随されます
  • 画面に触れると自動追尾が切れますので、再度発表者に自動追随する場合は「手動閲覧中」を押して「自動閲覧」にしてください

講演者・テーマ

@Takeshi_applilさん　アプリル株式会社　金谷武士氏
*「シニア向けアプリのUIを考える」
シニア向けアプリのUIについてお話します。

@dorapro_manaさん 株式会社ドラプロ　代表取締役　福西真奈氏
*「GameSaladで楽々ゲーム作り」
楽々ゲームが作れるGameSaladを使って、10分間ライブコーディングでゲームを１つ作ります。作るゲームは「バナナハンター」(Android／iPhone／iPadでリリースしています）

@tinpayさん 株式会社スタジオノモス 福井章平氏
*「広告の最適化」
広告や、広告の実装についてお話します。

@calm__designさん　calmdesign 小野哲生氏
*「CoronaSDK Ambassadorになったわけ」
CoronaSDKのご紹介をします。

@studioshinさん　スタジオシン 北村真二氏
*「iOS UIフルスクラッチ」
iOSのカスタムUIパーツを作る方法についてお話致します。

@m_hashさん　タンデムシステムズ株式会社　代表取締役　橋本雄氏
*「初めての Titanium」
Appcelerator Cloud Services を使ってみた！

@setoh2000さん　瀬藤誠氏
*「SmartEverにおけるアプリ高速化の工夫と課題」
軽さを最優先に開発したEvernoteクライアント – SmartEverの製作時に工夫した点や課題について発表します。

@3156さん　夏木雅規氏
*「UnityによるiPhoneアプリ開発」
Unityを使ったiPhoneアプリ開発について、ツールの使い方やNative拡張についてお話したいとおもいます。

@kitano_owさん　OneWorld株式会社　代表取締役 北野剛史氏
*「iOS画像加工のススメ」
CoreImage、OpenGLフレームワークやvImageなど、画像加工によく利用させる技術の検証を行います。

@oishiさん　株式会社フィードテイラー　代表取締役　大石裕一氏
*「エンタープライズiOSの最新動向」
iOSデバイスはもはやオモチャではなくなり、立派なビジネスツールとなりました。iPadの発表以後、数百台/数千台規模の導入が相次ぎ、昨今では企業内使用のみを前提としたエンタープライズiOSアプリ開発需要が急増しています。iDEP, OTA, MDM, InHouse, VPP, Configurator, Profile…等など、エンタープライズiOSで欠かせないキーワードを幾つか紹介すると共に、iOS6のApple告知文から見るエンタープライズiOSの将来予想をお話致します。

@ku_sukeさん サイバーエージェント　川畑雄補氏
*「WEBとネイティブのハイブリッドアプリの勘所」
HTML5が進化する中、WEBとネイティブの使い分けで工夫した点をご紹介します。
