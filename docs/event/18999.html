【RightScaleユーザ会 第4回】

夏真っ盛り皆様いかがお過ごしでしょうか。
暑さに負けずクールにクラウドサーバを運用したい場合には、RightScaleがお勧めです。
RightScaleは、Amazon EC2をはじめとするAWS(Amazon Web Services)等の
クラウドサービス用の管理ソフトウェアとして最も広く利用されているサービスで、
ディザスタリカバリーやハイブリッドクラウドを構築するための機能も備えています。

機能も豊富なことから、利用方法については前提となる知識とテクニックがあります。
今回は敢えてマニアックな話題はせず Amazon EC2・RightScaleの登録方法から
RightScaleの機能説明、運用テクニックなど、RightScaleを触る上で必要となる知識について、
RightScaleユーザ会のメンバーが(親切丁寧に?) 説明します。

是非皆様ご参加ください！
