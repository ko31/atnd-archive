description

　このコースではIT系未経験者の方に向けてSQLについての基本的な知識、操作方法を4時間（2時間/日×2日間）で学習します。SQLは基本構文が単純なこともあり、初めての方でもそれほどハードルは高くはありません。しかし、実際のシステム開発の場では簡単なSQLだけでは対応できないこともあり、正しいSQLの知識を身につけることが非常に重要になってます。このコースでは基礎をしっかりと理解することで、継続的にSQLの学習を続けていけるようなスキルをつけることを到達点としています。

■4/27（月）10:00～12:00（2時間）
＜SQLとは何か、テーブルの作成、SQL文を使った基本的な操作方法＞
　こちらでは、SQLとは何か、ツールの使用方法、SQL文を使ったテーブルの作成から基本的な操作について学びます。SQLの知識がない、IT系未経験の方におすすめです。

＜学習内容＞
１．データベースとは
　初心者の中には、どこかで聞いたことはあるけど、具体的なイメージまではわからない、という人が多いと思います。考え方としては、データベースは「複数の人が共有利用するデータを貯めてある入れ物」です。初心者の人でも理解しやすいように図書館の本の貸し出しに例えながら解説させて頂きます。

２．SQLとは
　データの操作や定義を行うためのデータベース言語（問い合わせ言語）です。いきなり言語構文の説明をしてもチンプンカンプンになってしまうので、①データベース定義、②データベース操作、③トランザクション管理に分けて詳しく説明させて頂きます。

３．MySQLの起動
　最も基礎的なMySQLを起動する方法と、停止する方法を解説します。またMySQLの操作にはコマンドプロンプトを使いますので、便利なようにショートカットを作成することにします。

■4/28（火）10:00～12:00（2時間）
＜課題から学ぶSQLの基本操作＞
　SQL文を使った操作（テーブル作成～データ挿入、参照、更新、削除）、結合（内部結合・外部結合） を学びます。弊社が用意したSQL課題を元に、基礎をしっかりと固めて頂きます。

＜学習内容＞
４．SQL基本操作
　・テーブル作成(create)
　・データの挿入(insert)
　・データの変更(update)
　・データの削除(delete)
　・データの参照(select)
　・データの並び替え(order by)
　・データの条件指定(where)
　・SELECT句のさまざまな指定

５．SQLのデータ型について
　・整数型(int)、日付型(date)、固定長文字列(char)、可変長文字列(varchar)など

６．テーブルの結合
　・テーブルの結合の考え方
　・外部結合(left join, right join)、内部結合(inner join)とは
　・結合文における検索条件の書き方の解説
