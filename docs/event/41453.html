第17回「ネットワークパケットを読む会（仮）」を 7月22日(月) に開催します

今回は Windows のファイル共有に利用されている SMB (Server Message Block) / CIFS (Common Internet File System) の動作を、実際のネットワーク パケットを解析して勉強してみたいと思います。
企業内や家庭のネットワークで Windows ファイル共有が広く利用されていますが、これに使われている SMB / CIFS プロトコルは元々 Microsoft の独自規格ということもあってあまり理解されていないようです。しかし実際にはファイル共有に伴うトラブルは非常に多く、その場合ネットワーク パケットから SMB / CIFS の動作を確認することは重要な情報確認になります。
そこで今回は (主催者の勉強もかねて) この SMB / CIFS プロトコルの基本をパケット解析を通じて勉強していきたいと思います。
参加希望の方は、このページから参加申込をお願いします。
