description

Google App Engine のハッカソンです。python/java/jruby/scala/clojureなどなど、さまざまな方が参加してコードを書きながら気が向いたときにでも情報共有する、という気軽で楽しいhack-a-thonです。


＜お願い＞
エントリ時のコメントに、当日使用予定のプログラミング言語・予定しているhackを記述してもらえると助かります。参加する人同士、興味があるところだと思うので。あと、面白い自己アピールなんかも書いてもらえるときっと人気者です！


＜やること＞
自由にコードや記事などを書きましょう。
#ahack4では、上記ほぼすべての言語でappengineと戯れるかたが参加しているので、使ったことないけど気になる言語に興味のある方もなにかしら聞くチャンスかも知れません。
また、html5/android/iPhone/iPadの開発に興味のある方も、バックエンドをappengineで実装してみたりするのにいろいろ聞いたりできてちょうどいいかもしれません。
googleのdatastore（BigTableをバックエンドとする永続化機構）やGoogle Strage/BigQuery（詳細は未公開ですが、大量データの高速な分散処理を得意とすると思われるサービス）に詳しい方も参加するはずなので、クラウド的技術に興味のある方も参加すると面白いと思います。
本質的に自由な感じなので、個別にチームを作ったりしたいかたは、コメントをつけておくと良いかと思います。
終了一時間くらい前に、話すのも聞くのも自由なLTを予定しています。

会場を管理されているthe SOHOさんのご厚意により、カフェでワンドリンク注文＋昼食が参加費となります。なので、なるべく午前中からの参加をお願いします。結果としてお金が落ちればいいので、LTしてくれるひとにドリンクおごりとかでもいいです。
