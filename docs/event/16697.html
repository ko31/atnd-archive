「せまいドレスコードでムダに集まりたい！」がとりあえずのgommの遠足のテーマです。
今回は、ボーダーでおしゃれタウンをムダに闊歩しましょう！

[ざっくりしおり]

  1. 遠足のテーマ：ボーダー＆メガネでムダに集まりたい。
  2. 遠足の決まりごと：ボーダー柄のお洋服とメガネを着用してくること。
    　また、帽子の着用を推奨します。どのような帽子でもOK！
    　ボーダーのお洋服も、どんなボーダーでもOKです。
  3. 遠足のおこづかい：自由。　※第2部参加の方は、4000円かかります。
  4. ＜時程＞
    ―第1部―　
    15時30分くらい　徐々に集まる。
    　　　　　場所：ローソン神宮前四丁目店　付近。
    　　　　　（ラフォーレ原宿のある交差点の近く、表参道沿いにあるローソンです。
    　　　　　　ごむの2人が小旗を持って立っている予定です）
    　■お集まりいただいた順に、お1人ずつボーダーの写真を撮らせていただきます■

　16時くらい　原宿から渋谷に向かって、ぞろぞろ歩きます。

　16時30分くらい　ハチ公前あたりで記念撮影をして、解散。

　―第2部―
　16時30分くらい　井の頭線に乗って、高井戸に向かいます。

　17時くらい　「しましまカフェ」http://shimashima-cafe.com/　にて、ごはん＆スイカ。
　　　　　　　食事+スイカに、ドリンク飲み放題で、￥4000です。

　19時30分くらい　解散。

※第1部、第2部どちらかのみのご参加も大歓迎！
