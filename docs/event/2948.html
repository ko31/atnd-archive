こちらは『PHP 初心者 勉強会』 から分割した、PHP 中級者 勉強会です。

PHP 中級者 勉強会では、少し難易度の高い勉強を行います。

対象者は『PHPのプログラムがある程度できる方』です。

ご注意！！

中級者勉強会は、初級者よりも高度な要求をします。初級者のノリでは絶対についてこれません！

PHPフレームワーク勉強会では、チーム制で各フレームワークの勉強を行っております。

多数の方から応募を頂いておりますが、現在の所、フレームワーク未経験者の方の応募は遠慮いただいてます。

自分のレベルが微妙と思われる方は、phpbeginners@gmail.comまでお問い合わせ下さい。

趣旨

PHP フレームワーク勉強会では、代表的なPHPフレームワークを使って、実際にWebアプリの作成を行い、それぞれのフレームワークの使い方や特徴を学びます。

理想は、すべてのフレームワークの使い方に熟知することです。

対象フレームワークは以下となります。

  • CakePHP
  • symfony
  • Zend Framework
  • CodeIgniter
