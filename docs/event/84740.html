概要

「WEBサーバ（Apache）って何してるの？」
「どうやって動いてるの？」

「WEBサーバ」という言葉、何となくわかった気になって使っていませんか？ 中途半端な理解だとちょっとしたトラブルも原因がわからず学習のつまづきになることが多くあります。

今回は改めてWEBサーバを仕組みから解説し、実際に手を動かして中身まで詳しく見ることで徹底的に理解します。 比較対象としてDBサーバやメールサーバにも少し触れ、より理解を深めます。 土台となるネットワークの知識も交えて解説するので、この機会にあまり触れることのないスキマの知識を身につけましょう。

主な対象者

  • WEBプログラミング初心者／入門者
    （特にXAMPP/MAMPを使っているがそれが何をしているのかよくわかっていない方）
  • ネットワーク／サーバ運用担当の初級者
  • Linuxサーバ初級者
  • 前回の「知ってるようで知らないWEBの仕組み体験ワークショップ」参加者はより理解が深まります

注意

今回はあくまでサーバの仕組み、使い方の解説です。
サーバの構築方法、設定方法は解説しません。

事前準備等

  • ノートPC持ち込み推奨（スマホのみでも一部体験できます※）
  • こちらが用意するWifiまたはご自分のテザリングによるインターネット接続可。
  • コマンドプロンプト／ターミナルの利用経験があればなお良いです
  • 電源は限りがあるため、可能な方はバッテリー駆動でお願いします。
  • 持ち込みPC等無しで講義を聞くだけでも大丈夫です

スマホで体験する方は以下のアプリをインストールしておいてください。

Android版「Telnet / SSH Simple Client」

iPhone版「iTerminal - SSH Telnet Client」

講義内容

  • WEBサーバによるWEBサイト表示の仕組み
  • TCP/IP基礎（IPアドレス／ポート番号）の解説
  • 手作りWEBリクエストによるサーバの中身大分解
  • メールサーバやDBサーバも手作りリクエスト？
  • まとめ解説
