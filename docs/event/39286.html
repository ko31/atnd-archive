概要

Node.jsのインストールから始めたい デザイナー･コーダー･プログラマなど 超初心者向けの内容です。

そのため、Node.jsのインストールから始めます。

今、Node.jsはV8エンジンを使ってサーバサイドの開発をするだけのものでは無くなっています。
例えば、LESSやGrunt.jsなどは、npm (Node.jsのパッケージ管理ツール)を利用することが前提になっています。

この勉強会への参加に際して、 インストーラ を使ったNode.jsのインストールを行なって来ていただいても構いませんが、Node.jsのバージョンを切り替える管理ツールの利用をおすすめします。

  • Windows: Nodist
  • Mac|Linux: Nodebrew

当日、上記ツールのインストールから始めます。

少しでもNode.jsを触ったことのある方にとっては少し物足りないセッションになるかもしれません。

それでも参加したいという方はぜひサポート役として参加ください。
※ npm install, npm startなどのコマンドの意味がわかる方は、なるべくサポートとして参加してください。
