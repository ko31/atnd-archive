□ 1st Session [ソーシャルメディアを利用したプロモーション]
───────────────────────────────────
◆ 渡邊大介 (わたなべ だいすけ) ALPHAROOK,INC.

プロモーションやイベント企画、MovableType を利用したWeb 制作を中心に活動中。

ソーシャルメディアに触れることで見えてきた今後の方向性など…

□ 2nd Session [HTML5のお話]
───────────────────────────────────
◆ 福田優弥 (ふくだ ゆうや) 株式会社インフォマーク

主にMovable Typeを使ったサイト構築・運営を行っています。まだまだ勉強中なので一緒に理解を深めていけたらと思います。

───────────────────────────────────
◆ 高見祐介 (たかみ ゆうすけ) 有限会社ソララ

でグラフィックとWeb のデザインやプランニングを中心に制作されている高見佑介氏は、メディアにとらわれない企画・構成を目指す。趣味は写真と自転車。
