ビブリオバトル（知的書評合戦）

ビブリオバトルってご存知でしょうか？ここに詳細に解説されています。
http://www.bibliobattle.jp/

簡単に言うと、1人5分間で自分の選んだ本がどれだけ面白くてためになるかを発表し合って、最後に全員投票で「チャンプ本」を決めるという集まりです。　　

今回は初めてですので、出来る限り公式ルールどおりにやります。選ぶ本についての縛りはかけません。ただ、廃刊などで買えない本は避けて下さい。ただし、ここは関西IT勉強宴会。上流工程勉強会です。そこに集まる方々の票を本気で勝ち取るつもりで発表してください。

発表者

最大10名とします。10名×10分＋投票20分=2時間とします。初期エントリーは5名でMLで追加募集中です。10名で募集を打ち切ります。
　※MLへの入り方は次のアドレスに空メールを送って下さい。
“KwansaiIt+subscribe@googlegroups.com”

現在の発表予定者は次の６名です。
　渡辺幸三さん、杉本啓さん、久保敦啓さん、下山吉洋さん、 野村昌由さん、山本幸伸さん、佐野初夫

ルール

1. 発表参加者が読んで面白いと思った本を持って集まる
2. 順番に一人５分間で本を紹介する
3. それぞれの発表の後に参加者全員でその発表に関するディスカッションを２～３分行う
4. 全ての発表が終了した後に「どの本が一番読みたくなったか？」を基準とした投票を参加者全員一票で行い,最多票を集めたものを『チャンプ本』とする
