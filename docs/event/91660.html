こんにちは！私は静岡大学のサークル、視聴覚教育研究会（AVEC）のものです。

子ども大会とは、私たちが主催する、未就学児・小学生を対象としたイベントです。毎年、テーマにそって私たちが作ったたくさんの遊びで思いっきり楽しめます。

今年のテーマは「海」。海にすむたくさんの生き物たちと一緒に光り輝く海の中を探検しませんか？ダンボール迷路や風船の部屋など、どの学年でも楽しめる遊びをたくさん用意しています。是非お越しください。

※駐車場がありませんので、公共交通機関をご利用ください。
