MANABIYAとは

“MANABIYA” は、teratailがおくるITエンジニアの問題解決カンファレンスです。
【Web】､【AI】､【生存戦略】など､10個のテーマごとにトークセッションを実施｡
さらに､メインとして各テーマの登壇者によるパネルディスカッションを用意｡
全国のエンジニアから集めた"疑問"を議題にディスカッションを行い､"疑問"に対する「解」を導き出します｡

Day2（3月24日）は別途申し込みが必要です。
Day2への申し込みは コチラ
