catch アラフォーカップルのぶらぶらユーラシア放浪の軌跡
description

人生の半分が過ぎ、モラトリアムから２０年過ぎた働き盛りのアラフォー世代の２人の男女が２０１０年６月６日にユーラシアを放浪すべくウラジオストックに船で旅立った。
時代の流れの早い昨今、ロシア、ヨーロッパ、イスラム・中東圏、旧ユーゴスラビア圏、インド、アジアを見てきた。感じてきた。
その体験を、そして旅の醍醐味を写真をまじえて語ります。
軽食と飲み物をご用意いたしますので、ほろ酔いで鋭いツッコミや「いい質問ですね」とつい言ってしまう質問もアリのイベントです。
「私これからどうしよう」「私これからどうしたいかわからない」「長い旅に出てみたい」「ちょっとした刺激を受けてみたい」など、そんな人におすすめです。

東北地方太平洋地震の発生により、今回のイベントは急きょチャリティーイベントとさせていただき、参加料の一部を義援金として納付いたします。
