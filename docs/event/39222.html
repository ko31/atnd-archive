皆様、
ご機嫌いかがでしょうか。

5月25日(土)、池袋勤労会館にて『blender勉強会』を開催いたします。
blenderは無料で使用できる3DCGソフトです。
フリーソフトでは考えられない高度な機能があり、手軽にCGを作成できます。
また、今流行りのゲームエンジンソフト unity とも相性がよく、一度覚えてしまえばとても親しみやすいツールです。
今回は多くの方からのご要望があり、開催致します。

ご質問等あれば知っている人が答える、またはその場でググる、勉強会です。
セミナーではありません。
『一人で学ぶよりみんなで学ぼう』をキャッチフレーズに楽しく学んで行きましょう。
プロ、アマ問いません。
まだ使ったことがない方、一度使ってみたけど挫折した方など・・・

解り易くご案内いたします。
すでにスキルをお持ちの方、
アウトプットすることでさらに理解が深まるかと思われます。
ご参加ください。
5月25日(土)　１８時より２１時半まで、池袋勤労福祉会館、第５会議室にて、
お待ちしております。
参加費：1000円
予めblenderをインストールしてきていただければ幸いです。

ご参加頂く際に、初心者、中級者、プロ等、スキルを記載頂ければと思います。
