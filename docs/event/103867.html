発表

BLE5の長距離モードを試す

秘密結社オープンフォース　河野

BLE5 Soc nRF52840 が入ったトレーニングボードアオノドン2019の開発も終わりに近づき、いろんな実験をはじめています。今回はBLE5長距離モードの実力をチェック。

スクロールサイネージ（Raspberry Pi Zero W、カラーLED 64×64dot）

日本Androidの会秋葉原支部ロボット部 部長　清水

AliExpressのカラーLEDパネル 64×64dot　を動かしてみました。(前回はAdafruit の32x32dot）
