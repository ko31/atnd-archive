昨年5月の The JUI 2008 に参加いただいた jQuery UI ライブラリ開発リーダー Paul Bakaus さんが再来日中。
と思ったら、もうすぐドイツに帰国されてしまうとのことで、
急遽、［The JUI 2009 Returns］ を開催することになりました！
jQuery に限らない、広く JavaScript のユーザインターフェースに関する技術系カンファレンスです。
今回は、準備時間もないので、ゆるーく。（告知開始：7月7日～）
