夏を過ぎてしまいましたが、今年もCGビール会を開催します！

今回はイギリスとベルギーのビールを初心者向けに教えてもらいます。
前回同様に店内の生樽と瓶ビールも飲み放題なので、ドイツビールが飲みたいという方もご安心ください。

料理もたっぷりでますのでコンディションばっちりでお越しください。

初参加大歓迎です。僕もビールはあまり詳しくないので一緒に楽しみながら勉強しましょう～
よろしくお願いします！

※今回は開始時間を30分早くしました。時間にご注意ください。

[内容]
・前半は少量のビールを実際に飲み比べながらビールのあれこれを教えていただきます。

・後半はパーティタイムです。料理を楽しみながら飲み放題で盛り上がりましょう！

・前回好評でした利きビール大会もやります。景品でます！

注意事項
・キャンセルがある場合は2日前までにお願いします。当日キャンセルはキャンセル待ちや代理の人がいない限り全額会費を頂きます。
・勉強会のときはビールの量は少なくしますが、それでもそこそこの量にはなりますので、3時間で中ジョッキ2杯～3杯が飲めない方はご遠慮ください。
・遅刻された場合でも、特に料理の取り置き等はいたしませんのでご注意ください
