【お知らせ】
・当団体による熊本でのSTAND UP参加企画は全て無事終了しました。
　この場を通じて、ご協力いただいた皆さまに御礼申し上げます。
　ただ、参加企画に参加できなかった方でも、まだSTAND UPには参加できます。
　開催期間終了まで、ぜひ熊本から多くの方のご参加をお待ちしています。

参加企画詳細

あくまでも以下の内容は予定です。
今後変更になる可能性もありますので、このページを随時ご確認ください。

１，国際交流会館学生ボランティア向けワークショップ → 9月27日に中止が決定しました。

  • 日時：2011年10月1日（土）14:00〜16:00
  • 場所：熊本市国際交流会館 5階和室
  • 内容：「格差」をテーマに既存教材を活用したワークショップ
    • なお、上記企画は、国際交流会館の学生ボランティアを対象に行われるため、
      一般の方の参加は要相談となります。参加希望の方は、下記のコメント欄か当団体事務局までご連絡ください。

２，アジアンマーケット2011 → 10月8日に無事終了しました。

  • 日時：2011年10月8日（土）14:00頃（STAND UP実施予定時間帯）
  • 場所：熊本市国際交流会館前
  • 内容：10月の「アジアンホリデー」の一環として行われる
    　　　東アジアの食や雑貨などを提供するイベント（10月9日まで行われます。 詳細PDF ）
  • 注意：実施当日が雨あるいは台風による強風の場合は、翌10月9日（日）に変更となります。

３，青年海外協力隊募集説明会 → 10月15日に無事終了しました。

  • 日時：2011年10月15日（土）14:00〜16:00
  • 場所：熊本市国際交流会館 4階第3会議室
  • 内容：青年海外協力隊の活動紹介や体験談発表、秋募集に関する説明（ 説明会詳細 ）
    • 説明会終了後（16:00）にSTAND UPを実施する予定です。

主催・共催・協力
