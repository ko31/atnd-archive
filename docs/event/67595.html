合唱オフとは

学生時代、誰もが必ず経験した『合唱』を、今一度、みんなで集まってやりたい！そんなシンプルな企画です。

  • 課題曲（決定版！！）

  • オフ実施要綱

    • ７月末までを第一次参加者募集期間とし、その後、参加者毎の課題曲・パート割り振りを行います。
    • 伴奏・指揮ができる方は、参加表明の際にコメントにて教えてもらえると助かります。
    • 基本的に、練習は各自・各パートにお任せすることを想定していますが、一人でも練習できるよう、譜面と各パートの音源の提供を予定しています。
    • 本番を迎える前に全体での顔合わせも兼ねて、練習オフを１０月頃に開催するつもりです。極力全員参加でお願いします。
    • Twitterでのハッシュタグは #合唱オフ としたいと思います。Twitterアカウントをお持ちの方は是非ご利用ください。
    • 本サイト（ATND）にメールアドレスを登録いただくことで、当方からのメッセージがメールで受け取れるようになります。ただ、サイト運営会社からメルマガが届く可能性があります（規約に書いてありました）ので、苦手な方は、定期的にメッセージの確認をお願いいたします。
    • 円滑なコミュニケーションがとれるよう、コミュニケーションツールについては継続検討していきます。
