今年は誰かと一緒に行きたい！

今年の LDD’11/Fall in KUSHIRO は、9/17の開催です。
プログラムはコチラです。
去年の募集結果は僕1人のみでしたが、今年も茜道東ツーリストの募集をかけます。

北見発、釧路経由、北見行き

車内で技術的な話をしながら、
LDD’11で技術的なお話を聞いて、
懇親会でもっと多くの人と話をして、
次の日に釧路の美味しいものを食べて、
また帰りの車内でお話を。

そんなツアーです。
