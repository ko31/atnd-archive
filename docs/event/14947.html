78米2011　キックオフイベント　土づくり&BBQ

title 78米2011　キックオフイベント　土づくり&BBQ
catch 主催：78会　協力：農業再生法人hototo
description

いよいよ、はじまります「78米2011」！！今年は最高のロケーションで皆様とご一緒させて頂きます。キックオフイベントを5月4日、5日に開催いたします。（GWでゴメンなさい）
また、当日の現地ではちょうど桜が満開であると予想されています。今年、花見を逃した方も、ぜひこの機会に桜の下でBBQを楽しみましょう。

【お申込の方は、info@78kai.netに連絡先メールアドレスをご連絡ください。】

■タイムテーブル
5月4日（水）
10：50：集合（現地）
11：00：オリエンテーション　 （参加の皆様のご紹介とプロジェクトのご説明）
12：00：今年現地でお世話を頂く、hototo　の農業・自然体験プログラム
　　　　　現地でお昼ごはんの準備、自家製ピザ窯で手作りピザ体験。
　　　　　お子様には、で魚のつかみ取り体験など、後にBBQで大人は、
　　　　　畑での農作業　－－－－　夏に美味しい野菜をご用意。　　

夕　方：ご希望の方は、近隣の温泉へ　（ほったらかしの湯）、
　　　　バーベキュー：新鮮な野菜と甲州の恵み、ワインなども楽しんで頂ける特別な企画

4日のみ参加の方は夕食後、解散

夜　　：ご希望者には近隣温泉宿・、人数によってはロッジ等で、宿泊の手配をいたしたします。5000円～／1人（大人）
※別途、各自で宿泊先を探される方は、それでも問題はございません。　

5月5日（木）
10：00：78会の田圃（1.5反＝1500㎡）にて、土づくりの作業
12：00：ご参加頂く人数によってですが、お弁当または焼きそば（BBQ型）など
13：00：田圃の前の神社では、このエリアの年に一度のお祭りが開催されています。
　　　　現地の皆様との交流頂けます。　
15：00：田圃の作業が終了後、後片付け、ご希望者はまた温泉に。一休みの後、解散。

■ご準備いただくもの
・汚れても大丈夫な、動きやすい服装　　（両日参加の方は2日分）
・朝夕は冷え込む可能性がございますので、温かい服装やブランケットなど
・昼間は天気が良ければ気温が上がります。汗をふくタオルや日光対応のものを。
・農作業の出来る靴　（水は、まだはっていない状態なので長靴で無くても大丈夫です）
・雨が降る可能性もございますので、その準備を
・日焼け止めや虫よけなど、気になられる方はご準備ください
・宿泊の方は、宿泊のご準備を
・農業と自然を楽しむ気持ちで、御参加ください
