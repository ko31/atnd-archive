★講演内容

・Windows 10, Windows 10 Mobile の最新情報
・Windows Phone、又は既存アプリをWindows 10 へ移植する方法
・Xamarin や Unity などのミドルウェア ランタイムを使用して構築したアプリをWindows 10 へ移植する方法
・Web サイトや Web アプリを Windows 10 ユーザーも利用できるように移植する方法

※内容は変更になる可能性がございます。

★講演者プロフィール

■津隈　和樹（つぐま　かずき）氏

　　　　

2011年に日本マイクロソフト株式会社に入社。
マイクロソフトの最上級のサポート契約である
プレミア契約をしていただいたお客様を対象に、
Internet Explorerや.Net アプリなどのトラブルシューティング、
技術セミナーなどを通して開発者支援を行う。
現在では、Windows Store の開発者を対象に技術支援を行う
Windows AppConsult の日本およびAPACの技術リードとして活動。
