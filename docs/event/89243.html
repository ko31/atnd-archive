若手人気デザイナーとのコラボデザインのスマートフォンケースやアートキャンバスのプリントサービスの発表と共に、コラボデザインの展示、販売をするアートイベントを開催いたします。またライブペイントやワークショップなど人気作家のアートと直に触れることができる企画も盛りだくさんです。事前登録で限定ステッカープレゼント！

17：00～　抽選会
18：00　　終了

イラストレーター・少女画家
長尾高徳

1982年生まれ　青森県出身　東京在住

日本のかわいいの原点である1960～70年代。

その動物の布団柄や少女画をベースにしたイラストやグッズを制作、販売。
ロフト、パルコなどのクリエイターイベントに参加し、また、雑誌「ね～ね～」、文具フリーペーパーにて4コマ漫画を連載。
近年ではサンリオ「マイメロディー」や「カピバラさん」とのコラボレーション作品や生保労連（全国生命保険労働組合連合会）の公式キャラクターに選ばれたりと活躍の幅を広げている。

参加クリエイターのご紹介

いわさきゆうし
1982年埼玉に生まれる。

中央大学在学中にデザインに興味を持ち卒業後、 アミューズメントメディア総合学院・キャラクターデザイン学科に入学。 ITコンテンツ会社にてデザイナー、企画運営を経験後、2011年に独立。 現在はフリーのイラストレーターとして活動中。 自身も飼っているハリネズミを中心とした動物をモチーフに、 メルヘンな世界観をつけペンを使い表現している。 ペン画以外にも雑貨、アニメーションなど様々な手法を用いて作品を発信中。 ロフトや東急ハンズ、パルコ、伊勢丹、ラフォーレ、デザインフェスタなどクリエイターイベントに積極的に参加している。

●2013年『ハリネズミICカードステッカー』がminne award 2013 人気作品賞を受賞。
●2014年『ハリネズミのハンガー』がハンドメイド大賞にノミネート。 ●2015年 個展『ハリネズミといろいろ展』FEWMANY 新宿マルイアネックスで開催。
●2015年 イラストを全面的に担当した飼育書『はじめてのハリネズミとの暮らし方』日東書院本社より。
● 2015年『台湾漫画博覧会』サンリオの人気キャラクター・マイメロディ40周年を記念したコラボレーション作品を発表。
●2015年 個展『ハリネズミのてくてくお散歩展』谷中の雑貨店 HOW HOUSEで開催。
●2015年 個展『ハリネズミとAからZのできごと展』FEWMANY 新宿マルイアネックスで開催。
●2016年 個展『ハリネズミとつよそうなおともだち展』谷中の雑貨店 HOW HOUSEで開催。
●2017年 個展『MY HEDGEHOG』東京青山タンバリンギャラリーで開催。
