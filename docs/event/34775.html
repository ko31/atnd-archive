イベント概要

ハッカソンとは「ハック」と「マラソン」を合わせた造語で、作ることが好きな人が集まって、その場でアプリを作ってしまおう！というイベントです。

今回のテーマは「時計」です。テーマが簡単な分、出来上がりの差を競う面白さがあります。それをより効果的になるようにデザイナと一緒に作る「プログラマとデザイナのためのハッカソン」を目指しています。さらにプログラマは「TDD（ペアプロ）を実践する」というテーマもあります。

プログラマは2日に分けたイベントで、最初の日に「Visual Studioハッカソンのための事前勉強会」として、TDDBCC#のセッションを行います。今回始めてTDDを知る参加者でも分かるように、内容は初心者向けです。C#でTDDを始めたい人、実際のC#でのNUnitの設定、UIのテスト方法。ペアプロの実体験です。

なお、2日連続のイベントではありますが、勉強会だけの参加でも問題ありません。

  • プログラマ　20名　(プログラマとデザイナで時計を作るVisual Studioハッカソン × TDDBC 大阪 for C#)
  • デザイナ　10名　(デザイナとプログラマで時計を作るVisual Studioハッカソン)

2013年はじめのTDD Boot Camp in 大阪

今回のTDDBCはハッカソンで物を作りをながらペアプロを実践するというスタンダードなTDDBCとは違います。王道のTDDBCはTDDBCC#開催の1週間前に 2013年はじめのTDD Boot Camp in 大阪 と 2013年はじめのTDD Boot Camp in 大阪 外伝 があり、こちらの参加を合わせておススメします。TDDを学ぶには絶好の機会です。
