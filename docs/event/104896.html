◆人気交流会が守谷初開催！地元の方はもちろん、茨城方面やTX沿線の事業主さん、近隣エリアを応援したい方も大歓迎です！

【日時】2019年4月23日（火）
18：00～20：40（受付開始17：45）

【会場】アパマンショップ守谷店2F会議室(守谷駅徒歩1分）
〒302-0115
茨城県守谷市中央4-12-19

【参加費】3000円　（ビール・お茶・お菓子）
※入会金や年会費、キャンセル料は一切ございません。当日受付で参加費のみお支払いください。

【対象者】
・経営者の方、社会人の方、起業予定の方
・守谷市および近隣エリアで人脈を広げたい方
・ビジネスの情報交換をしたい方
・コミュニケーションをスキルアップしたい方
・起業予定、または経営に興味のある学生や主婦の方
※ご自分のビジネスだけでなく、地域貢献や活性化を考えた人脈づくりをしようと思っている方を歓迎いたします。
※守谷市以外の方でも、このエリアでの人脈作りにご興味ある方はご参加可能です。

【席数】30席（守谷優先枠15席・一般枠15席）
※茨城方面・TX沿線の方も、守谷枠でご参加いただけます。
※同業種が10％を超えたときは、制限をかけることがございます。

【参加ルール】
●ネットワークビジネス、展示販売（毛皮・高額スーツ・宝石等）先物取引販売、能力開発教材販売、政治・宗教関連の方は参加をお断りいたします。
※副業の方を含む。
●強引な勧誘、迷惑行為は、いかなる業種・職種であっても禁止いたします。発覚した場合は退出いただき、以後の参加をご遠慮いただきます。
また、会の終了後に個別にアポイントを取られる際も、このルールを厳守してください。
●異業種交流会の質を保つため、ご参加される業種・職種に偏りが出た場合は、人数を制限をすることがございます。
●男女交際を目的とした方のご参加はお断りしております。誤解を招くアプローチもご遠慮ください。
●会場内で寄付を求めたり、署名集めをするなどの行為はご遠慮ください。
●会場によっては物品やチケットの販売など金銭のやり取りを禁止しているところがあります。
直接販売は主催が許可した場合のみとさせていただきます。
定している方は、事前にお知らせください。
