description

【楽読】は読書スピードが何十倍にもなるような速読ではありません。

楽読の強みは読書スピードが理解度、記憶度を落とさずに、2～7倍で速く読めるようになることです。
今まで数百人の検証結果、どんなに読書が苦手な方も速く読めるようになっています。

しかもそれを【楽しく】習得していただけます！！

●速読の歴史
　速読がどのようにして伝わってきたか。

●速読の理屈
　そもそも速読とは何なのか？

●どうしたら速読ができるのか。
　『ちゃんと記憶に残るの？』
　『本当に速く読めるの？』
　『右脳をつかって読むってどういうこと？』
　

●速読ができたら、どうなるのか。
　速読ができる人は【脳の使い方】がちょっと違います。
　速読はほんの入口です。
　趣味や仕事、その他の何事にも活かすことが可能です！！

速読術や記憶術のご説明をさせていただき、
その後実際に速読を体験していただきます。

楽読体験

●読書速度計測
　おそらくされたことある方は少ないと思います。。。

●眼筋トレーニング
　速読をできるようにするために、眼球を動かします。

●見るトレーニング
　『読む』ではなく『見る』トレーニングをいたします。
　（理由はセミナーでご説明いたします！！）

●トレーニング後の読書速度計測
　実際どのくらい速くなったかをチェックします。

トレーニングはとても簡単で
『え？こんなトレーニングで速く読めるようになるの？』
と疑問を抱かれる方もいらっしゃいます。

しかし・・・なぜか読書速度があがってしまうのです！

みなさんも実際に体験なさってみてください！＾＾

※最近では【脳トレ】としても需要が高いため、脳の話、そしていくつか効果的な脳トレもご紹介したします！

楽読、そして脳トレ・・・。

すごく不思議で、おもしろい体験をしていただきます！
