title フリーランスのための営業術
catch 〜ツボとコツがあれば、誰でもゼロからスタート出来る！〜
description

独立したいけれど、どうやって仕事を受ければいいのかわからない・・
独立したけれど、どうやって仕事を増やしていけばいいかわからない・・

そんな方々のために、実際にゼロからスタートして売上げを伸ばし続けている講師の体験を元に「フリーランス/クリエイターの営業術」セミナーを開催致します。
また実際にセミナーを受けて学んでも、なかなか実行できないあなたのために、「実行できる自分になる」方法も伝授致します。
【主な内容】
1.講師の体験談（会社退社〜独立して3ヶ月）
2.フリーランス/クリエイターのための営業術
・これまでの固定概念、自分の常識に対しての執着を手放す
・魅力あるメール
・惹きつける人間になる
・営業するときの心の持ち方
・最初の仕事を獲得のその後のコツ
3.実行できる自分になる
4.出席者同士での名刺交換、アンケート等

■講師
奏クリエイト代表　山中直美

イラストレーターとして5年、DTPやマルチビジョン、コンテンツ制作などジャンルを問わず8年経験。その後、09年1月あるきっかけがありゼロからのスタートを決意し独立。
人脈を広げていかなければ仕事がゼロになるという苦難の体験を経て、コミュニケーション・営業スキルの大切さを実感する。
新規獲得を得るため試行錯誤を繰り返しついに成功パターン5つのテクニックを編み出す！今ではクライアントからの氏名が後を経たず、多忙な日々を送る。特別な営業力や人脈がなくても、『ツボとコツがあれば、誰でもゼロからスタート出来る！』という実体験を広めるべくセミナー活動も精力的に行う。現在では、マスター・メンタルスクールに出会い、人間力を上げるには自分自身を知ることが必要だと実感しインストラクターとして活動
