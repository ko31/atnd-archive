学生団体iUSキックオフイベント（新メンバー募集説明会）を４月３０日に行います！

iUS は 『ユーザー同士が情報を共有することで、コミュニケーションを通してデバイスやメディアへの理解を深めるとともに、ユーザー同士の「絆」を生み出す』 という想いのもと、スマートフォンを使ったイベントの企画・実施や、学生のソーシャルメディア有効活用の促進などを行っている学生団体です。

２０１０年の８月８日に発足し、この短期間でサイバー鬼ごっこやiPhoneユーザーの会など多くのイベントや、ソーシャルメディアについての情報の発信などを行ってきました。
そしてこれからも団体として更なる発展をしていきたいと思っており、そのためにも多くの仲間が欲しいと思っています。

今回のキックオフイベントは iUS とはどんな団体なのか、どんなメンバーがいるのか、どんな雰囲気なのかなど、私たち iUS という団体を知ってもらおうという目的のイベントです。

学年は関係ありません！現在スマートフォンを持っていない、TwitterやFacebookなどのソーシャルメディアを使っていない、関係ありません！スマートフォンに興味がある、ソーシャルメディアに興味がある、学生団体に興味がある学生、 ぜひ来てください！
