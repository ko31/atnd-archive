■イベント概要

2010年から続いているSocialTopRunnersを開催します！過去発表企業は、gumi / KLab / グリー / バンダイナムコスタジオ / Drecom / Kayac / スクウェア・エニックス / gloops / pokelab / colopl / CyberX / Sumzap / nijibox / Crooz / akatsuki / その他トップSAP企業。業界のトレンド、開発、企画、マネタイズ、そして戦略のリアルを体感するイベントです！イベント後の懇親会では参加する開発者同士の情報交換も重要ですよね！このイベントからソーシャルゲーム業界が始まったと言えなくもない、要チェックイベントですよ〜！グリー、バンダイナムコスタジオは初参加です！

■登壇企業

今回は【ゲームの未来2013】 を、3つの視点で占います！この業界の未来に興味がある方は、必見必聴です！

（１）グリー：国内のリーディングカンパニーとして、グローバルマーケットを牽引すべく様々な取組をしかけているグリーによる発表です。今回は、ゲームx広告の未来を”グローバル目線で”語ります！（広告系）

（２）バンダイナムコスタジオ：ソシャゲランキングの上位に鎮座するゲーム業界の重鎮が、”ゲームの未来を創る！”今回は、バンダイナムコスタジオの執行役員 山田さまがソシャゲ／スマゲ開発の今後の方向性をお話します！ゲームの形はこれからどうなっていくのか？（開発系）

（３）Pokelab：今最も熱い企業といえば、Pokelabさん以外にありません。2012年最も躍進した会社は次にどんなタイトルをしかけるのか。”ポケラボのゲームはこれからどうなる？”シニアプランナー後藤さまによる大注目のお話です！（企画系）
