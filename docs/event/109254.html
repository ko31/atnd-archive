第3回　GJS ROBOT GEIO ゲーミングロボットで遊んでみようの会

イベント概要

GJS ROBOT GEIO を使って、 ロボットプログラミング、ロボット好きのIT交流会を開催致します！

ARなどの技術に興味がある、FPSゲームが好き、ITエンジニアたちと交流をしたい、 そんな方々はぜひ参加をお待ちしております！

GJS ROBOT GEIOとは

GJS GEIOは、スマホやタブレットと連携できるカメラ付きロボット。

同ロボットをコントロール可能な専用アプリケーションを使える。First Person View（一人称ビジョン）システムと、人工知能（AI）によるビジュアル認識機能を採用した。さまざまなゲームモードを収録している。
