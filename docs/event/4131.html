　『みつか万博音楽祭～宴＠万博記念公園～』





★平成２２年５月５日（ミツカの日）



　私事ではございますが、この万博公園でゲリラ的結婚式をあげてから早や４年・・・子供は３人・・・平社員から取締役へ


　あの時の誓いを忘れないよーに、毎年同じ場所でパーリーを開かせて頂いております



　とはいっても人間飽きのくる性分は仕方がなく、今年は少ーし趣向を凝らし、渚音楽祭や同日開催の茨木音楽祭に対抗すべく“音楽祭”を開こうと思います





　演者はあなた！





★なにか楽器（音のなる物）を持ってきてください



　た・と・え・ば・・・ジャンベ、ディジュ、ギター、ウクレレ、三線、サックス、ハーモニカ、オカリナ、尺八、縦笛、口笛、お腹、お尻、ホーミー、ボイパ、フライパン、スティールパン、フランスパン、ＣＤラジカセ、ターンテーブル（大きいスピーカー等は怒られちゃうよ）などなど



　フリーに集まってフリーにセッション♪



★飲食物は自給自足、物々交換、楽市楽座スタイる



　おいしいお酒とおいしいお肴、みんなが持ち寄ればまるで食の博覧会やぁ！



　ＢＢＱは離れたスペースへ連れて行かれちゃうので御注意を



★時間はopen９時半～１７時over



　今年こそは朝イチから皆さんをお迎えします！



★場所は「わくわく池の冒険ひろばのある芝生」



　万博公園マップ

　　⇒http://park.expo70.or.jp/pdf/guidemap.pdf



★雨天は決行コケッコッコー







　その他お問い合わせは私(090-3941-1051)までヨロチクビ







　楽しむのもあなた！



　自然の中で自由にまったり音を楽しみませう♪
