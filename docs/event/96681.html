人の悩み３重苦（採用できない、続かない、育たない）にお悩みの多店舗運営のオーナー様へ

●講師紹介

自律自走する組織をプロデュースする
人材の採用から育成に繋がる仕組み作りの専門家　

５店舗以上を経営している飲食企業など組織運営が難しいと言われる
企業のマネジメントサポートをしており、
特に人財の採用から育成までの仕組み化を中心とするサポートに従事。

２０店舗を運営する企業の店長やマネージャーを経験すると同時に、
同社で人事評価制度の作成・導入も行なう。

クライアントからは「ここまでやってくれるのか」とサポート体制が評価され
契約当初からコンサルの１年後の継続を依頼される事も多い。

コンサルタントとしては１年目から大手ビールメーカーに認められ、
タッグを組みながら飲食店多店舗運営企業のコンサルをしている。

自身のノウハウを活かし業界トップクラスの営業レベルを誇る
プルデンシャル生命のセールスマンのコンサルや定期勉強会を通じて
売上の拡大に繋げてた。

また、ノウハウの一部をまとめた著書
『組織が変われば人は勝手に育つ　人財マネジメント』は
Amazonランキングの組織改革や人事・労務部門など
４部門で１位を獲得しました。

●セミナー内容

当セミナーは下記のお悩みを抱えている方にオススメの内容です。

●スタッフの入れ替わりが激しく、人が育たないと嘆いている方
●採用費ばかりが高くなり、利益が減って不安が高まっている方
●昔みたいにアルバイトが社員になる流れを作りたい方
●自分がいない所で、クレームをもらう事が増えて悩んでいる方
●スタッフをもっと評価してあげたいが、適正な評価方法が分からず困っている方
●スタッフが不満ばかり言い、悩みを抱えている方
●もっと安心してお店を任したいけど、任す人がいない方

飲食業界の９割以上の店舗で人材不足という悩みを抱えています。

確かに学生のアルバイトにおける飲食店のニーズが
以前と比べて減少傾向にあるのは確かです。

給料が安い割に大変、人間関係が面倒だという理由で敬遠されているようです。

しかし、このようなご時世でも採用費を全くかけずに、
アルバイト・正社員の採用を毎年行えている企業もあります。

何が違うのでしょうか？
やっぱり時給の高さでしょうか？

いえ、決してそんな事はありません。
そうした採用費をかけていない店舗でも時給は他の店舗とさほど変わりは無いのです。

ですが、多くの企業は間違って認識してしまっているんですね。

・人が採用できないのは給料が少ないからだ。
・店長の能力が足りないから、人材が辞めていく
・昔と比べて良い人材が採用できなくなった

こうした愚痴をよくお聞きします。

その状況を打開しようと、この３つを改善する為に
OFF JT（店舗外研修）での店長研修、給与UPなどをされたりします。

それでも改善しないところがほとんどなんですよね。

目の前に現れた問題に直接対処しても
内情は良くならない事って結構あるんですよね。

私は問題を根本から対処する事で
組織を自立自走化させていきます。

その方法に人事評価制度を活用しています。

ただ人事評価制度を導入するだけでは
組織は自立自走化しません。

そもそもの人事評価制度の認識が違うんです。

人事評価制度は
人を評価する制度と書きますが、
決して評価や給与のために使うものではないんですよね。

下記が当セミナー内容の一部です。

●　なぜ人が辞めるのか？
人が辞めるのはタイミングごとに理由が異なります。それを理解する事で時期毎に何を気をつければ良いかが明確になり、辞めない運営が出来るようになります。

●　9割の企業が間違える人材定着法とは？
色々なアプローチを実践しても成果が上がらないのは、人が定着する前提が間違っているかも知れません。間違いに気付き、成果の上がらない無駄な労力をかける日々から抜け出しましょう。

●　社員が働きたい会社の３つの条件とは？
あなたの会社で今何が足りなくて、どんな対策をすればスタッフから働きたいと言われる会社になれるのかが分かるようになります。

●　人が育つ為に必要な３つの条件
人が育つ為には、研修や制度の前に整えておくべき３つの条件があります。これを知っておくとスタッフの教育カリキュラムに無駄がなくなり、効率的にスタッフの成長が見込めます。

●　サービス力が上がり続けるマニュアルの作り方とは？
マニュアルにはサービス力を上げるものと下げるものの２種類が存在します。サービス力を上げる1つのポイントを抑える事でスタッフが自分で考え行動するようになり、店舗のサービスレベルが上げ続ける事が可能になります。

●　人材育成の為のリーダー６つの心得とは？
リーダーシップのあるマネージャーや店長を育てる為には、スキル・ノウハウの前に共有すべき心得があります。これを共有することでスタッフから目指される店長が育っていきます。

●　求める人材が育つ人事評価制度の３つの条件とは？
人事評価制度を導入しても人が育たない企業は数えきれないほどあります。最大限活用する為には抑えるべき３つの条件があるのです。ここを抑えて運用すれば育成スピードが倍増します！

●　利益を圧迫しない人件費算出方法とは？
人事評価制度を導入したら人件費率が高騰した企業もあるぐらいです。人事評価制度を導入して利益を圧迫していれば元も子もありません。この算出方法を理解すれば利益体質を作れ、会社もスタッフもWin-Winの関係が作れます。
