新しいスポーツをメインにした「クリエイター大運動会」を
東京ピストル と ソウ・エクスペリエンス が合同で開催します！

運動不足の人、一緒に遊びませんか？
記念すべき第１回大会の今回は、

キンボール行います！

従来のスポーツの常識を超えたボールゲーム、キンボール。
1チーム4名、3チームでプレーするユニークなスポーツで、
直径122cm、約1kgという超巨大なボールを使って
「ヒット」「レシーブ」を繰り返して得点を競います。

対格差や年齢差、男女差など関係なく、
みんなが楽しめるスポーツ、キンボール。
初対面の人同士もあっと言う間に打ち解けられます。
一緒に日頃の運動不足、解消しましょう！！！
