大人気のゲストハウスとGaitomoのコラボ国際交流パーティーです。
なんと今回は山Pや錦戸亮や赤西仁をはじめとする芸能人(海外セレブではジョ ニーデップ、キャメロン・ディアス含む2000名以上)が、プライベートで毎週来店している六本木の最高級クラブ club NEW LEX を全館貸切で国際交流パーティーを開催致します☆
六本木の好立地な会場ですので、大変多くの外国人が参加されます♪芸能人も通う、六本木の最高級クラブで最高に素敵な時間を一緒に過ごしませんか？・゜
会場はクラブですが、音楽は控えめにお話を主に楽しんで頂くスタイルのパーティーです。踊ったりするパーティーではございません。

外国人は皆日本語が話せますので、英語に自信が無くても安心してご参加頂けます。ヨーロッパからアジア圏まで幅広い国の外国人たちが参加されます。 たくさんの新しい出会いをこの機会に楽しんでください。

Gaitomoは、日本語を話せる外国人ばかりを集めたので、英語に自信がない方でも安心してご参加頂けます。
外国人との出会いに、パートナー探しに、外国語の勉強に、異文化交流に、新しい出会いを求めている人に、パーティー好きな方にお勧めのパーティーです。 お一人様参加の方が多数いらっしゃるパーティーですので、安心してご参加頂けます。
