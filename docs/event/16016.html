description

独立も、開業も、クリエイターの皆さんなら一度は考えたことがあるはず。
しかし、専門家や経験者でないほとんどの方にとって、それは困難に見えがちです。なぜなら、独立・開業には、手順と手続きについて正しい知識が必要だから・・・。

そこで、来たる5月27日（金）、書籍『クリエイターの渡世術　20組が語るやりたいの叶え方』でおなじみの独立開業応援相談室が、クリエイターを対象にした開業支援講座を開催します。

独立・開業に際し、「どんな手続が必要なの？」「何から手をつけていいかわからない」といった疑問を持つクリエイターの皆さま。ふるってご参加ください！

【プログラム】（予定）
１．独立するということ（参加者との対話＋心構え解説）
２．退職手続～開業までの流れ（必要手続＋スケジュール＋心構え）
３．個人か？　法人か？（形態判断方法＋諸手続＋費用）
４．開業準備のポイント（10のポイント＋事業計画書作成の重要性）
５．質疑応答
