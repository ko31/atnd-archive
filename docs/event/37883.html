「概要」

対象：Pythonをはじめて使う勢いの人。「プログラミング始めたいんだけどな」みたいな人。

内容：プログラミングの導入として、Pythonを使い、簡単なゲーム（数え上げのゲームなど）を製作します。使う構文は以下のとおり。プログラミング初心者の方向けです。

ゴール：下記のようなPythonのごくごく基本的な構文を使えるようになる。さらにこれらを使ってゲームを作る。
If
For
Input
List
While
Def
Print

ゲームのルール：プレーヤーは二人。それぞれ代わりばんこに数を1から数えていき、先に30を数えたら負け。ただし、一回のターンに言える数は3まで。
e.g.プレーヤーAが1,2,3　→　プレーヤーBが4,5　→　プレーヤーAが6　→　プレーヤーBが7,8　→・・・→プレーヤーAが27,28,29　→　プレーヤーBが30
　→プレーヤーBが負け。
