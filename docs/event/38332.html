description

猫と一緒にビジネスのアイデア企画を学ぼう。

猫のいるコワーキングスペース、ネコワーキングでビジネス企画についてワークショップを行います。

ビジネスモデルのフレームワークを学びつつ、
普段考えていることのアウトプットをする！
参加者でシェアし合うことで、理解を深めていきます。

会社で新規事業に携わっている方
起業を計画中、または興味のある方
身の回りにある何かでビジネスを考えてみたい方
ただビジネス企画について興味のある方
そんな方々にオススメです！

参加費：無料　※場所代として５００円だけかかります。

人数：最大２５名限定　最少開催人数８名
※前日１９時の時点で開催の有無を判断いたします。

内容：アイディアを出しあうワークショップです。
以下の流れで進めていきます。

■Learn
・まずはビジネスモデルのフレームワーク「ビジネスモデル・キャンバス」について学びます。

■Make
・実際にビジネスモデル・キャンバスを作ってみましょう

■Share
・作ったビジネスモデル・キャンバスを発表し、共有してみましょう。
・フィードバックを行いながら、より理解を深めていきましょう

※事前にビジネスモデルジェネレーションについて、以下の教材で学んで頂けると、より当日の理解が深まります。
