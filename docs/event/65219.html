概要

このイベントはFirefox OS and Raspberry Pi, WoT ハンズオン内の展示会専用の募集ATNDです。

2014年末、ついに日本国内でもFirefox OS端末が発売され、つい最近Raspberry Pi 2 も発売されました。 そんな注目を集める二つのデバイスを、テーマとしたブース展示を開催いたします。

Firefox OSコミュニティーとJapanese Raspberry Pi Users Groupでの合同コミュニティーイベント「Firefox OS and Raspberry Pi, WoT ハンズオン」イベント内にて、コミュニティーや企業様によるブース展示を行います。

こちらの募集枠では、コミュニティーによる制作物の展示やおさわり、企業様による最新機器のトライ＆タッチなどを目的としたブースの展示を行わせて頂きます。

無料ですので、出展内容に興味がありそうなものがあれば、気軽におたちよりください。

ブース 概要
Firefox OSコミュニティー FxOS&Pi の WoT デモ、WoTで月面探査車
Japanese Raspberry Pi Users Group 放射線観測モニタリングポスト
複数のＨＤＭＩ映像を同期再生するRaspberryPiクラスター
RaspberryPi用の各種実験キット
展示キットの販売
