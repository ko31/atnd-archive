イベントの概要

『UNIX 4.3BSDの設計と実装』を読みながら、OSの理解を深める会です

P.22(2.2 カーネルの構成）～P.43（練習問題）を読む予定です。

以下のような進め方を考えています。

1. センテンスを音読する。
2. 内容について議論する。
3. セクションが終わるまで1～2を繰り返す。
4. ソースコードの該当箇所を探す。（本には明示されていないため）
5. ソースコードについて議論する。
6. 本を読み終わるまで1～5を繰り返す。

これはあくまで案です。実際にやってみながら方法を固めていこうと思います。

ご参加をお待ちしています。
