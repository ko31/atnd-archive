第１回社会変革共創ラボ

イベント ユーザー

title 第１回社会変革共創ラボ
catch
description

NPO法人 sopa.jpは、社会を変える事業創出のためのフューチャーセンター「社会変革共創ラボ」を開設します。「社会変革共創ラボ」は、“事業を通じた社会問題解決“を目的とし、起業家・NPO・大企業・政治家・行政関係者など様々な立場の関係者が幅広く集まる場をつくりながら、事業アイデアの創出、社会起業家の養成、社会的事業への投資などを手がけていきます。

　この度開催する第１回社会変革共創ラボでは、「子育て×グローバル」をテーマに、日本で初めて民間学童保育の事業化に成功したキッズベースキャンプの元創業者であるネクストマーケット鈴木規文氏、バングラディッシュで保育事業を支援している秋田智司、sopa.jp 理事長小田るいと共に、国内における子どものグローバル教育、東南アジアの教育格差解消などについて、参加者全員で対話を通じ事業アイデアを考えていきます。

また、後半では、建築家の遠野未来氏から「子育て×建築」をテーマに
どんな想いで活動されているのか？
どんな建物をつくってらっしゃるのか？
を話していただきます。
また、子ども達と古民家を一緒に作っていくようなワークショップや秘密基地を
作りたいという想いをお持ちの方なので、参加者の方と一緒にアイディア出しなども
考えてくださっています。
