無料体験講座のご案内

スマートフォンアプリ開発エンジニア育成プログラム

このイベントは、12月1日より開始するスマートフォン向けのアプリ構築を実践的な形式で学ぶセミナー＆ワークショップの無料体験講座です。

3時間の体験講座の中で、スマートフォンサービスおよびアプリ業界の今、エンジニアの仕事のスタンスを知ることができます。また後半は、スマートフォン向けサービスのフロントエンドエンジニアの仕事を、HTMLコーディング・CSSコーディング・jQuery Mobileの利用を通して 体験していきます。

知る

  • スマートフォンサービス・アプリ業界概況
  • スマートフォンサービス・アプリ業界におけるエンジニアの仕事 フロントエンドエンジニアの仕事
  • カリキュラム説明とプログラムのゴール

体験する

  • スマートフォン向けHTMLコーディング・CSSコーディング
  • jQuery Mobileを利用したスマートフォン向けアプリの構築体験

本講座のご案内

スマートフォンアプリ開発エンジニア育成プログラム（全10回）

12月1日（土）より、スマートフォン向けのアプリ構築を実践的な形式で学ぶセミナー＆ワークショップを福岡でスタートします。

今回は、ユーザーインターフェース開発で注目されているJavaScriptを中心に講義形式とワークショップで、１回3時間／全10回で行います。
モバイルUI向けのフレームワークである、jQuery Mobile（jQueryベースでモバイルWeb画面を実装できるフレームワーク）やTitanium Mobile (ネイティブUIをJavaScriptで実装できるフレームワーク)を利用しながら、実際にアプリケーションをリリースするために必要な技術やティップスを習得できる貴重な機会となります。

講座は、全10回（予定）　12月1日（土）から開始予定、隔週の開催を予定しています。 今回のワークショップは、全10回の有償セミナー＆ワークショップとなります。
