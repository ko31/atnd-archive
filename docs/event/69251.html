2015年8月22日 (土) 13:00からKITOKURAS 図書室にてWordBench香川を開催いたします。

開催概要

  • WordPressに限らずその他CMS・HTML5・CSS3等最近のWeb周りに関しての雑談なり意見交換したいと思います。

  • ライトニングトーク 発表者募集中！ 発表したい方がいらっしゃいましたらご連絡ください。

  • 場所は山一木材さんの道挟んで向かいにあるKITOKURAS cafe敷地内にある図書室です。ゆるーい感じに進行していきたいと思いますので敷地内の森へお散歩もご自由に！

WordBenchについて

WordBench.org は、日本各地におけるローカルな WordPress ユーザーコミュニティを応援するためのスペースです。

参照元：https://japan.wordcamp.org/2009/04/19/wordbench/

対象者

  • WordPressに関心のある方（香川県民以外でもOK）
  • 個人、フリーランス、デザイナー、プログラマー、企業のWeb担当者等
  • WordBenchに加入していない方でも大丈夫です
