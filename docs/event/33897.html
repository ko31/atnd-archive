ライトニングトークの時間は5分です。

最後に WCAN mini Markup vol.11を行ったのは2009年10月22日です。
約3年ぶりにWCAN mini Markupを開催します。
WCAN mini Markup vol.12はHTML5、CSS3、JavaScriptのライトニングトーク大会を開催します！
ゲストに益子 貴寛さんがお越しいただきます。

ちなみに、同じ日 13:30から、 サイバーガーデンBiz コピーライティング 実践トレーニング＆ワーク 名古屋 というセミナーも予定されています。こちらもオススメです！

ライトニングトークのテーマ

HTML5、CSS3、JavaScriptなどに関連した内容でライトニングトークしてみんなで情報交換しましょう！

  • HTML5でコーディングしていて迷ったこと
  • HTML5でこんなサイト制作したよ
  • HTML5 API（Web Storage、File API、canvasなど）をつかってみた
  • CSS3を使ってこんな表現をしてみました
  • LESSやSass（SCSS）を導入してみた
  • Twitter Bootstrapを使ってサイト制作してます
  • JavaScriptやjQueryのTipsをご紹介します
  • スマートフォンサイトを制作しました（jQuery Mobileなど）

など

ライトニングトークしてくださる方 を募集しています！
聞きにくる方 もぜひご参加ください！

ゲスト 益子 貴寛さん

北海道から サイバーガーデン の益子さんが名古屋に来られるという事で、この日に名古屋のマークアップエンジニアさんが集まる勉強会ができたらという事で企画してみる事にしました。
