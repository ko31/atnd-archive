概要

C++WG小委員会エキスパートメンバーの有志により下記のテーマに関した発表をする予定です。

  • 第1回 勉強会によるフィードバックのレポート
  • アロケータのrealloc(resize)機能
  • future関係のAPI改善
  • ExecutorとScheduler
  • string view
  • サイズ付きのデアロケーション
  • 連想コンテナの異種混合比較ルックアップ

他のテーマの候補

  • out of thin airの規定
  • Filesystem
  • optional
  • 数値リテラルの桁区切り文字
    • http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2013/n3661.html
    • http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2013/n3781.pdf
