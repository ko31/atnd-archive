description

OpenStreetMap8歳の誕生日を記念して地図を描く方法を経験者とともに体験するイベントです。巣鴨の町を楽しみながら歩きます。経験者・未経験者を問わず歓迎致します。お気軽にご参加ください。

江戸と昭和の香りが漂うヒューマンスケールの町並みは歩いていて気持ちの良いところです。高齢の方がよく散策されるだけあって、バリアフリーへの配慮が比較的行き届いています。商店街沿いのPOIやバリアフリー関連タグに着目しながらマッピングします。

1.日時、集合場所
8/18(土)13時00分～17時00分(13:00集合)
JR巣鴨駅北口改札を出たところ に集合
（目印に黄色のGPSロガーを首に下げています）

雨天決行（歩ける程度なら）

2.費用
・参加費無料
・交通費、食費等は各自で負担ください。

3.募集人員
・制限はありませんが5名ほどを目安に考えています。

4.持参するもの
・最低限筆記具があればOKですが、以下のようなものがあるとさらに楽しめます。
・デジカメ
・スマートフォン
・GPSロガー
・モバイルなPC

5.服装など
・徒歩で歩く予定ですので軽装でお越しください。

6.目標
＜共通＞
巣鴨地蔵通り商店街のマッピングコンプリート！
＜初心者＞
・OSM上にノード（お店などの点）を打てる
・タグ付けの概要を理解する
＜経験者＞
・自分で設定したテーマの実践
・初心者ヘルプ
・（余裕があれば）バリアフリー関連情報の収集と登録

7.タイムスケジュール（当日の状況により多少変わる可能性があります）
13:00～ 集合、資料配布
13:05～13:30 行動予定の説明、現地まで歩きながらマッピングの説明
13:30～15:00 マップ情報収集
15:00～17:00 マップ編集作業（喫茶店等で）
17:30～19:30 巣鴨駅周辺で懇親会（自由参加、実費割り勘）

10.マッピング（マップの描き方）の例
例えばコンビニをOpenStreetMapで表現する場合には以下のように描きます。
1)まずコンビニの位置にノード（点）を打ちます。
2)そのノードに以下のようなタグ（属性を表わすキーと値の組み合わせ）を書き込みます。
shop=convenience （コンビニ）
name=FamilyMart （名称）
