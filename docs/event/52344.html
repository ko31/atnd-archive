「みんつむ」とは

「みんつむ」は、みんなでつむぐWeb制作に関わるすべての人々のための新しい勉強会です。

今回は、タスクランナー「Grunt」の初心者向け勉強会です。導入方法からタスク自動化の考え方までをみんなで学びましょう。

Gruntとは

Gruntはタスクランナーと呼ばれる、Web制作で必要な作業を自動化するツールです。Gruntを使うと、例えば次のようなことができます。

  • ファイルの結合
  • SassやLessといったメタ言語のコンパイル
  • HTML、JS、CSSなどの圧縮
  • CSS Spriteの生成
  • LiveReload（ファイルを保存すると自動でブラウザ表示を更新）

などなど…
