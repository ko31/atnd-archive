catch GitHubの基本からオープンソースプロジェクトへのコントリビュートまで
description

バージョン管理ツールとして、Subversionに換わりGitが広く使われるようになってきました。
それに伴い、Gitをベースとしたプロジェクト管理ツールであるGitHubを利用したOSSプロジェクト
も増加しています。GitHubではプルリクエストモデルが採用されていますが、従来の開発のモデル
とは大きく異なり、どうやってコントリビュートするのか、ちょっと分かり難いところがあります。

本ハンズオンでは、Git/GitHubを利用したOSSプロジェクトのフォークとプルリクエストを送るまで、
GitHubの基礎を勉強するとともに、OpenStackを例としたプロジェクトへのコントリビュート方法を
ご紹介します。

■内容

  • GitHubによるOSSプロジェクト管理モデルの紹介(お話)
  • トラッキングシステムへのバグ・フィーチャーの投入(演習)
  • プロジェクトのフォークとブランチの作成(演習)
  • リベース・プルとプルリクエストの送付(演習)
  • プルリクエストへのコメントが着いたときの対応方法(演習)
  • その他(時間が余れば)
  • Swift周りのコード解説
  • オープンソースカンファレンス2012Spring
  • Swift実証実験について
  • Novaコードについて

※ おぷ☆すた主催なので、OpenStackに関する議論を時間が余ればさせて
頂きますが、興味がない人はスルーで結構です(汗
