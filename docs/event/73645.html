概要

北海道の Drupal ユーザーコミュニティ「Drupal さっぽろ」が主催する meetup (勉強会) です。地域の Drupal ユーザーの情報交換と交流を目的に、月に一度、札幌と小樽で開催しています。今年最初は小樽での開催です。

開催予定は ATND でご案内、Google グループ、Facebook、Twitter を通じて告知します。
https://groups.google.com/forum/?hl=ja#!forum/drupal-sapporo
https://www.facebook.com/drupalsapporo
https://twitter.com/drupalsapporo

セッションは持ち寄りで発表テーマは常時募集中。北海道の Drupal コミュニティとして情報交換と交流の輪を広げ、さらなる活用につなげていきましょう。

参加無料

内容予定

持ち寄りセッション

常時募集中です。発表テーマをお持ちの方はお知らせください。随時このページに追加・更新していきます。また、当日の飛び入り参加も大歓迎です。

Vagrant/VirtualBox でローカル Drupal 環境を作る

前回に引き続き、Vagrant と VirtualBox を利用してローカルの Drupal サイト環境を構築する手順を整理します。今回は構成管理に Ansible を利用します。作った環境で Drupal を動かしながら情報交換＆ディスカッションしましょう。PC を持参される方は Vagrant と VirtualBox をインストールしておくと、一緒に手を動かしながら進められると思います。

なお、会場施設に Wi-Fi 環境がないため、当日は個人のモバイル Wi-Fi ルータを共有する予定です。不安定な場合もあるため、テザリング等の接続環境を用意できる方は各自で準備いただけると助かります。

懇親会

終了後は近隣の飲食店で懇親会を予定しています。（要実費）
