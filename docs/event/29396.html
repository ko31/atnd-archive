イベントの概要

技術系イベントで行われることが多いLTを、マンガ好きがマンガについて語る目的でやりませんか？

ライトニングトークとは？

５分間程度の短い時間で行われる発表を連続して続けていく形式のプレゼンテーション。「Lightning Talk(s)」の頭文字を取って「LT」と略される。

  • 話す方のメリット
    1. ５分なら通常の講演ほど準備は大変ではないし、気軽
    2. 失敗しても５分我慢すればいいだけ
  • 聞く方のメリット
    1. ５分なので気楽に聞ける
    2. 多くの人の話を聞ける
    3. つまらなくても５分我慢すればいいだけ

イベント趣旨について

  • 発表の場の提供（好きなマンガを紹介したい人へ）
  • マンガ好き同士の交流

考えたキッカケは、友人とお気にいりマンガを持ち寄って、
カラオケBoxに入り、ひたすらマンガを読むというのがけっこう楽しかったんだ！

その延長でこんなイベントやったら楽しそうだなーと。

たくさん人数がいれば、それだけ色々なマンガが集まるし、
ここが面白い！と意気投合 したり、 新しい発見があったりすると思う！

発表したい人募集！

というわけで、ライトニングトークする人を募集します！

  • ルール
    1. 持ち時間は５分
    2. 発表形式は自由（資料なし・パワポ・模造紙 なんでもおｋ）
    3. 必ず自己紹介を入れてね
    4. 発表するマンガを事前に教えてね（他の方と被るのを防ぐため）
  • 会場の用意
    1. PCと大型モニターあります
    2. 持参したマンガをモニタに表示できるように準備します
    3. マイクあります

希望者は、参加登録時のアンケートを「発表する」にして、 発表するマンガのタイトルを添えて、
@xtomoki へDMまたはmixiのメッセージでご連絡ください。
もし他の方と作品が被っていた場合は調整させていただく可能性あり。
