セッション概要

セッション1

  • タイトル
    • Firefox OS用サンプラーアプリを作ってみた
  • 講演者
    • 一般社団法人 Mozilla Japan　清水 智公　: @chikoski
  • 概要
    • HTML5 で作られたアプリといえば、まだまだ文字や位置のようなデータを扱うものが主流であるように思います。一方、他のプラットフォームでは音声や動画を記録、加工、保存するアプリは普通に流通しています。Firefox OSもメディアデータの処理を行うための機能と能力を備えています。その中で代表的な3つのAPI、WebAudio API, Device Storage API, GetUserMedia の3つを取り上げます。これらを利用して作成したサンプラーアプリの例に、これらの利用方法と連携についてご紹介するととともに、Firefox OSでの音声データの処理について解説します。

セッション2

  • タイトル
    • Firefox OSだってARしてみたい！
  • 講演者
    • 株式会社グローバルサイバーグループ　藪下 正美　: @aoi_nagatsuki
  • 概要
    • AR好きデスカー！　FxOSはカメラが使えないからARできないのは過去のこと。最新のFxOSでのカメラの使い方とWebGLやgeolocationを組み合わせたARのやり方を伝授しちゃいますよ！

セッション3

  • タイトル
    • Firefox OSで何が変わる？、アプリケーションの開発ライフサイクル
  • 講演者
    • Firefox OS コミュニティ　酒巻 瑞穂　: @SakamakiM
  • 概要
    • 昨今までのモバイルやWebシステム開発は、Native中心の開発からハイブリッドを経て、HTMLやCSS、JavaScript中心の開発に変わりつつあります。
      Firefox OSの登場により、コーディング作業のみならずそれを取り囲む状況も大きく変化しそうです。
      コーディング準備やその補助機能、生産コスト（楽をできるか）がどのように変化するのか・・・皆さん気にならないですか？
