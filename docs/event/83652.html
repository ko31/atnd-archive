MapAm❤re Workshop Series - OpenStreetMap Intensive Course

Workshop

As part of the MapAmore Initiative's Community Support Programme, an Intensive OpenStreetMap Workshop series will be offered to volunteers from the Polytechnic University of the Philippines San Juan Campus community.

Exclusive to PUP SJC students and invited guests.

Course Outline

Session Topic Notes
- Introduction to OpenStreetMap
- Editing with iD Basic, Advanced
- Tagging
- Using Umap
- Using Tasking Manager
- Field surveys using FieldPapers
- Photo-mapping using Mapillary
- Photo-mapping using OsmAnd Android only
- Editing with JOSM Basic, Intermediate
- Map-athon
- Participatory Mapping with Local Communities with invited participants from local barangays
- Extracting OSM data
- Other geospatial technologies Satellite Navigation Systems, GIS, etc.
