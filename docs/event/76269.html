イベント概要

GAM（Game Application Information Exchange Meeting）：ゲーム・アプリ情報交換会

ゲーム・アプリに関わる方々向けの情報交換会となります。

2016年3月に行われたトークイベント＆情報交換会においては、ゲーム業界に携わる多数のクリエイターの方々に ご参加頂きました。本当に有難うございました！

引き続き、ゲーム・アプリ業界関係者同士の横の繋がりを深めてもらいたい思いで、
第4回目を企画させて頂きました。
是非、情報共有や人脈作りの場にして頂けると嬉しいです。

参加条件

ゲームやアプリに携わっている方

  • 企画、開発、運営ご担当者様
  • ゲームエフェクトに関わるご担当者様
  • ゲームサウンドに関わるご担当者様
  • その他、ゲーム業界に携わっている方
