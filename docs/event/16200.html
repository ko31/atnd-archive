description

なぜ、人は努力するのか？
それは、得たい結果があるから！です。

大切な事は、その努力や頑張りが、
自分の人生を豊かにすることに結び付いているかどうか？ということです。

一生懸命努力をして、努力が目的になってしまい
その結果が
「こんなはずじゃなかった・・。」
というのは、残念なことです。
人間関係も一緒です。

せっかくなら、
今の努力そのものが自分にとって価値がある。
そして、その努力が自分や関わる人の幸せに結びついている。
こう、確信を持ちたいものです。

●「自分を知る」コミュニケーションには、いくつかのポイントがあります。

・方法論を知る
・どんな人生を送るかは、自分を知ることと人間関係にある

本セミナーでは
「自分を知る」だけでなく、ワークや理論を通して、実際自分にとって大切なものを見つけていきます。

自分にとっての大切なものが分かることで
自分の努力の方向が、欲しい結果に結びついているか分かります。
そして、自信をもった考え方ができるようになります。

また自分のブレも少なくなるでしょう。

◆お伝えする内容は
「自分を知る」コミュニケーションＮＬＰ体験セミナー

第１チャプター
「人間関係を豊かにする」
・深い、信頼関係を作るコミュニケーション法を体験
・感受性をもって相手のこころの動きを知る
・親密な交流をはかる

第２チャプター
「自分を知る」
・人生を決める自分の基本姿勢
・自分を突き動かす行動の動機
・最高の人生をつくるための無意識を見える化する

第３チャプター　
「ＮＬＰとは」
・（お楽しみに）
