catch テーブルゲーム初心者の方も歓迎！週末金曜、渋谷でテーブルゲーム・勉強会を開催します。
description

週末金曜日、勉強会という名目で「ドイツ年間ゲーム大賞」を一緒にプレイしませんか？
ボードゲーム・テーブルゲーム初心者の方や私と面識のない方でも大歓迎です。

渋谷駅から少し離れていますが、みんなでワイワイやりましょう。
既に5名参加予定ですので、お一人からでもお気軽にご参加ください。

■開催日
2012年3月23日（金）
■開催時間
19:00 ～ 22:30

■会費
500円（ソフトドリンク・アルコール等をご用意いたします。）
※フード等はありませので、予め1Fファミマ等をご利用ください。

場所

個人開催ですが会社の会議室をご提供頂きました。

ガーラバズ（会議室）: http://www.galabuzz.jp/map.html
住所: 東京都渋谷区渋谷3-12-22　渋谷プレステージ2F（1Fファミマ・正面横入口）
※建物内2F正面向かって一番右が会場の会議室になります。

    色と形を合わせてタイルを配置して高得点を狙うゲーム。

    かれらをケルトと呼ぶ。
