HBaseとCassandraの討論会　第2回

どうも日本Cassandraユーザー会の@kimteaこと荒浪一城です。

HBaseとCassandraは、よく比較対象になり議論されるのですが、
これをネタに討論会をしているみると、何か白熱した議論が展開され
そうで、おもしろそうな気がしそう・・・と思い企画しました。

前に出て一人の方が発表する勉強会方式ではなく、みんなで飲み食い
しながら全員平等の立場で語り合うというのが、この会の趣旨です。

この度、Gemini Mobile Technologiesさんが、日本初のRiptano社に
よるCassandra Hands-onトレーニング（日本語通訳付き）を開催を
記念して、ゲストとしてRiptano社のNate McCallさんを囲んで、
開催致します。

英語がわからない方も、大歓迎。参加お待ちしています。
