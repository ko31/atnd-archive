～セミナー内容～
SPA開発において、サーバサイドで悩みたくない。
サーバをあまり意識しないで開発したいと感じている方は多いのではないでしょうか。

ReflexWorksは、XMLやJSONなどの構造化データの読み書きができるドキュメント指向のKVSです。
ACIDトランザクションをサポートしており、
ソフトスキーマ（アプリケーションで定義するスキーマ）を採用、項目の追加が容易です。

また、AngularJSの双方向データバインディングと相性がよく、
開発者はModelとViewに集中して開発できます。

ReflexWorksではPDFやEXCELのデータバインディングもできます。
さらに、Yeoman、Grunt、Bowerなどのクライアントビルド環境が使え、
CircleCIやTravis CIによるデプロイの自動化も可能です。

もうフルスタックエンジニアはいらない、HTMLとJSの知識さえあれば
Webサービスを開発できる。そのような世界を私たちは目指しています。
このセッションでは、AngularJSとReflexWorksによるSPA開発について、
実際の事例とDEMOを交えて説明したします。
