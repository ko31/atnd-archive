description

ゲームを作りたい意志はある、しかしモチベーションがわかない、続かないなど何かと腰の重い方々、集まることで一歩を踏み出してみませんか。

  • 一日使って各自でもくもくとゲームを作ります
  • 各自ノートパソコンを持参して下さい。またエディタ、ライブラリ、フレームワークなど用意し、ゲーム作りが初められる状態にしておいて下さい。
  • WiFiはないですが、テザリングはできます
  • 電源タップは持っていきます
  • 写真は和室になっていますが、今回の会場は洋室です
  • 開始時間・終了時間は９時から２２時となっていますが、これは予約している会場の時間で、参加の時間は自由です。２２時まで取ってありますが、もう少し早く終わりにします。最後に軽く発表ができたらいいなと思います﻿
  • 参加費は、施設利用料1600円を人数で割りたいと思います

立川市 幸学習館　第２教室：
