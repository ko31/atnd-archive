■ONE DEGREEと「Webとデザイン月イチ勉強会」って？
デザイン・制作業でつながるコミュニティ – ONE DEGREE（ワンディグリー）-では、月１回のペースで「Webとデザイン月イチ勉強会」を行っています。

WordPress / Illustrator / Photoshopなど、主に初心者がつまづきがちなところをカバーし、最初の壁を乗り越えるサポートができたら嬉しく思います。

■自由時間
デザインやウェブ制作に関するスキルやノウハウの相談・シェアのため、可能なかぎり自由時間を設けるようにしています。

・今回の企画内容について参加者同士で話す！
・デザイン、ウェブ制作について今困っていることがある！
・皆と話して仲良くなりたい！

などなど、ツールの使い方の相談から、制作の仕事に関する悩みを相談したり。
何度も参加されている方であれば、前回の話題について気づいたことを共有したり。

その日の企画に縛られることなく、自由に話ができる時間です。
主催も、WordPress / Illustrator / Photoshopを日頃から扱っていますので、この時間も是非活用していただければ幸いです。
