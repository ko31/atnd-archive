catch ～善光寺の人気宿坊、吉祥院でのヨガと、農業体験のデトックスな休日～
description

都会の喧噪や忙しい日常から離れて、自分とゆっくり向き合う時間を過ごしませんか？
美しい自然と静寂のお寺で心とカラダをリラックスさせてあげましょう。
そうすることで、様々な「気づき」がきっとあるはずです。

ヨガは合計3回のレッスンになります。
それぞれの時間に合わせた心地よいヨガを皆さんで楽しみましょう。

善光寺の宿坊では、ヘルシーな精進料理を頂きます。
ヨガで五感が研ぎ澄まされると、素材本来の味を感じられるはず。
吉祥院さんの精進料理は美味しいと、女性に好評です。

そして、2日目には「お朝寺」を体験できます。
これは、宿坊に宿泊しないと参加できないものです。
ぜひ、この機会に、体験してみてくださいね。

最後のヨガを終えたあとは、農業体験の時間もご用意しています。
自然と思いっきり触れ合い、美味しい秋の野菜を収穫します。
取れたて野菜を調理し、みんなで頂きましょう。

2日間のリトリートはヨガだけを目的としたものではなく
参加者同士の交流の場でもあります。

2010年も残りわずか・・・。
この1年を振り返り、自分自身と向き合ってみませんか？

今回は初開催につき、特別料金でご参加頂けます！
