2012年1月7日（土）に、京都大学で「第9回ノンジャンルのためのGIS勉強会」を開催します。
大学院生、学部生、若手研究者、GISを仕事にする、あるいはGISに関心のある若手の社会人が中心となり、
GISについてのびのびと語り合う場です。
今回で9回目の開催となり、ひとまず中間決算の場となる予定です。
関心を持たれた方はぜひご参加ください。

タイムテーブル
15:00　開会・あいさつ・近況報告
15:20　話題提供とブレーンストーミング
　話題提供者（順不同・敬称略）
　1）鈴木　翼（新潟大学大学院博士後期課程）　「WebGISを使おう！」
　　Webマッピングのごくごく基本的な説明と、インフルエンザ発生調査のデータとArcGIS Explorer Onlineを使った簡単なデモンストレーション（予定）
　2）渡邉　泰崇（立命館大学）「本当にGISでは実現不可能なのか？－研究方法に対するアプローチの在り方を考える－」
　　自身の研究活動を事例に、GISの利用可能性を改めて検討するとともに、研究方法に対するアプローチの在り方を問う
18:00頃　閉会
