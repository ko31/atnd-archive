description

Web業界の職種の方、またはその業種に興味のある方に特化した読書会を開催します。

Web業界は別名「総合格闘技」と呼ばれるような職業でもあります。

PM、ディレクター、デザイナー、プログラマー、SE、マーケティング、営業職など、、、
多岐に渡るスペシャリストが集まってひとつのサービスが実現する、もしくはたった1人で実現する業界でもあります。
現場では関われない部署の人の思いや、他人の仕事術などをWebに関わる本を通して集まれる会を開催します。

シェア読書会の派生版として、web業界のネタに特化したジャンルでの読書会になります。

～～シェア読書会とは？～～
お互いのオススメ本を持ち寄って紹介し、気に入った本を交換しあう会となります。
普段、自分では手に取らないような書籍と出会ったり、改めて読んだ本のアウトプットをすることで、
自分がどう思っているのか。。。
新しい発見や繋がりを提供できる場所を目指しております。

詳細は下記参照
http://www.facebook.com/Shablio

まずは第1回目の開催となりますので、多々至らぬ点もあるかと思いますが和やかに楽しめる会にしたいと思います。
