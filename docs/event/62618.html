痛みを誤魔化さずに生きるために知っておくべきセルフコンディショニングの知識と実践

description

先日、高校女子バスケ部を指導する機会をいただいたのですが、その時「あーやっぱり」と思ったのは、ほとんどの生徒が、痛みを我慢したり誤魔化したりして、騙しだまし練習を続けている現状でした。
　
そして揃って口にするのが「医者に行っても特に原因が分からなくて、痛み止めと湿布で様子を見てくださいって言われて・・・。の時は痛くなくなったから・・・」という内容の言葉でした。
　
特に慢性的な痛みの多くは、痛み止めや炎症を抑える薬を飲んでいれば、時間経過と共に良くなります。しかし身体を酷使し続ければ、また同じ個所を傷めることになります。それって怪我や痛みが「治った」と言うのでしょうか？
　
何度も同じ個所を傷めるほど、また傷めている期間が長いほど、元の状態に戻すのにも時間と困難を要します。また触らないと気付かない痛みも多々ありますので、爆発寸前の筋肉や関節を抱えている人も大勢いらっしゃるのが現状です。
　
　
またマラソンの季節のせいなのか、自分のブログを観覧しにきてくださる検索ワードで「アキレス腱の違和感」「足底筋膜炎」「膝の痛み」「坐骨神経痛」という言葉がザーッと上位に来ていたこともあり、今回は下半身の痛みや違和感を軽減させるための、セルフコンディショニングの講習会を開催することにしました。
　
足底筋膜炎、捻挫、アキレス腱の腱鞘炎、シンスプリント、膝の痛み、腸脛靭帯炎、坐骨神経痛、腰痛など、腰から下の筋肉や関節でよく聞かれる怪我、それによる痛みを軽減し、再発を予防するためのセルフコンディショニングの講習会です。
　
　
「医者に行ったけど分からないと言われたので我慢している」「朝起きると痛い」「天候の悪い日は痛い」「走っていると後半痛くなる」などなど、様々な症状や痛みのレベルがあり、その全てを無くせる、とは言えません。しかし、現状の痛みの７割８割を軽減することは出来ます。多くの人がその方法を知らないだけです。私はその方法をお教えするだけです。治すのはみなさんの努力と自然治癒力なのです。その「身体を自分で整える」という作業をしなければ、根本的な解決にはなりません！、と強く言われる日は、そう遠くないでしょう。
　
講習会後、足が軽くなった、地面を踏む感覚が全然違う、めっちゃ歩きやすい、と満足していただけるよう、最善を尽くさせていただきます。
