仙台iOS開発者向け勉強会 第7弾を開催します！

内容

来たる7月21日（土）に、iOS開発者向けの勉強会を開催します！　

進行は、いつも若干ゆるふわです。セッション内容はグダグダなことが多いですが、ライブコーディングを見つつ話をしたりと、なかなかギークな内容になることが多かったり。わりとガチ系の勉強会です。

仙台でiOS開発者が集まって話をする機会も貴重かと思いますし、情報交換の時間もご用意しております。是非ご参加ください〜。

(お願いなど)
  • 会場にはネット回線がありません。各自、モバイルwifi等をご持参ください。
  • 電源が壁際にしかないので、長めの延長タップを持参していただけると助かります
  • 13時頃には会場を開けるようにしますー。お待ちしております。
  • 会場使用料が6400円なので、参加人数で割り勘させてください。スイマセン・・・

対象

  • iOSアプリ開発をやっている方
  • プログラム経験があり、iOSアプリ開発もやってみたくて、本を1冊くらい読んだことがある方
  • 初心者のかたも歓迎です。（ただ、若干内容がわかりにくいかもしれません…）

タイムテーブル

7/11追記：セッションが決まりました！
Session 0 自己紹介・情報交換（雑談 / 世間話 / 井戸端会議） 60 min
Session 1 InterfaceBuilderを使ったグラフィカルなアプリの作り方 by @ta9_maさん

「アプリは全部IBでUIパーツを配置している」というta9_maさんから、IBを使ったアプリの作り方についてのセッションしていただきます！「もしかしたら変態的な使い方をしているかもしれない」とのことですw 		60 min
Session 2 GameKitを使ったBluetoothプログラミングについて by @taiko19xxさん
Bluetooth通信を使用したアプリをリリースされているtaiko19xxさんから、iOSのBluetoothプログラミングについてセッションしていただきます！（内容は主催者からのリクエストに応えていただきました。俺得でスイマセン＞＜） 		60 min

場所

  • 会場は、エルソーラ仙台の特別会議室です。仙台三越定禅寺通り館の5階になりますー。
  • 一応椅子は20脚あるのですが、机が大きい円卓一個だけなので、結構ギリギリです・・・

懇親会

今回は勉強会の懇親会は開催しませんー。同日 17:00から、仙台のiOSユーザーのオフ会iOSendaiミートアップVol.2が開催されるので、飲みたい方はこちらに合流しましょう〜。
