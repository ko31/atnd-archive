5月に仙台、会津若松、東京など複数の都市で行われ好評だった、復興支援アプリケーション開発の為のオフラインミーティング、Hack For Japanのアイディアソン/ハッカソンが再度開催されます。

7/23（土） アイディアソン
7/30（土） ハッカソン

今までも複数の有意義なプロジェクトがHack For Japanから生まれています。皆様是非ご参加ください。

このミーティングは、現地の生の情報を入手し、他の開発者の方と情報交換をしつつ、また、各自が実際に開発作業を行うことができるイベントです。

Hack For Japanは、震災からの復興を継続的に支援するための、IT開発を支えるコミュニティです。

“東日本大震災に対し、自分たちの開発スキルを役立てたい”
―IT開発者の皆さんのこうした声をもとに、Hack For Japanは生まれました。
→詳しくは：http://www.hack4.jp/home

凄腕の「ハッカー」でなくても、僕ら開発者の誰もが、ほんの少しずつでも、必ず何かの貢献ができると信じています。
会場には、アイデア出しや開発をサポートするチューターもいますので、初めての方もお気軽にご参加ください。
プロジェクトを進行中の方も、情報交換・協力者探しの場としてもご利用いただけます。
当日は、１人で自分のプロジェクトの開発作業に没頭することもできますし、あるいは、既に活動中のチームに加わって共同開発に参加することもできます。
また、プログラマに限らず、企画やデザイナが得意な方も、歓迎しています。どちらか1日のみといった参加も可能です。

他会場の情報についてはこちらをご確認ください。
http://www.hack4.jp/RelatedInfo/Events/FutureEvents/110723mtg

参加は無料です。（ノートPCは持ち込んでください）
皆さまの積極的なご参加をお待ちしています。
