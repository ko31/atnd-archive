Web Browser Users Festival

ウェブのつかいてと ウェブブラウザのつくりてと ウェブコンテンツのつくりてをつなぐ あたらしいカタチのユーザーオフイベント

開発者だけじゃない。
ユーザも参加して、ステキなWebをつくっていきませんか？
ウェブを使う人みんなで新しいWebについて楽しく話すことが目的のイベントです。

初回である今回のテーマは「ユーザビリティ」。

Web開発者のみなさんは、ユーザビリティについてユーザから生の声を聴くチャンスです。
ユーザのみなさんは、より使いやすいWebについて開発者へ伝えるチャンスです。
みなさんでより良いWebについて話し合いましょう！

現在、Opera Software ASA より Web エヴァンジェリストであるダニエルさんのセッションをはじめ、幾つかのライトニングトークを予定しています。
自分も発表してみたい！という方は、開発者ユーザ問わず申し込みにあわせてその旨ご連絡ください。
(ユーザの方は「こんな Web の使い方をしています」のような話題でも全然構いません) LT の募集は一旦打ち切りました><

それでは、みなさんの参加をお待ちしています。

参加申込みについて
