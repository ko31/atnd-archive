title 《お菓子食べ放題のキャンデービュッフェ開催》Supreme
catch White Day Party
description

ゲストDJにはモデル、ラジオパーソナリティーとして人気のシャウラが出演！！お菓子食べ放題のキャンディービュッフェも開催！！

　『音と空間の宝石箱』
宝石箱に飛び込んだような感覚・・・・・
全てのゲストがそんな不思議な感覚を覚える特別な時間。
非日常的な最上級のエンタメーテイメント異空間

「Supreme」

スィーツのビュッフェと言えば、ケーキなどですが、キャンディービュッフェはキャンデー、チョコ、マシュマロ、グミなどのお菓子で行い、海外ではウェディングを中心としたパーティで大人気のビュッフェです。こちらのご自由に、会場でご賞味頂けるだけでなく、気に入ったお菓子は最後に小袋に詰めてお持ち帰り頂けます。会場で知り合った素敵な方や、ご家族、ご友人などへのプレゼントにも最適です。

海外お菓子の量り売りで有名な「CANDY・A・GO・GO」様とのご協力で当イベントで開催が決定いたしました。
会場となるthe BARの雰囲気と相まって、見た目にも味でも楽しんで頂ける事間違いなしです。

　ゲストDJにはモデルとして活躍し、前回ゲストDJとしてご参加頂いた黒過ぎるDJ「伊津野亮」氏のラジオ番組の相方などを務める傍ら、都内の大箱などでもDJプレイを重ねている

「Shaula Vogue 」

が参加いたします。
巷のモデルDJと言われるモデルとは一線を画す美貌とDJプレイをお楽しみください。

ショーケースにもスーパーモデルクラスのスタイルを誇る女性４人組ユニットの

「Juicy Circus」

のショーをご用意。ワンボーカル、スリーダンサーのショーは見応えありです。
　
　もちろん、DJ陣もメインDJのDJ WARA-Zを始め、the BARフロアではEDMやメインストリームを中心とした上がる音楽を、スカイガーデンフロアではR＆BやHIPHOPなどを中心としたゆったり聴ける音楽をセレクトいたします。

また、レジデントDJのRYO KAWACHIは今回がバースデーバッシュ！！メインフロアのコアタイムに会場を限界まで盛り上げてくれること間違いなしです。

　名物となっているヘアメイク、ボディジュエリーなどの美容系コンテンツも内容をパワーアップして基本メニューは無料でお試し頂けるようになりました。是非、お楽しみください。
