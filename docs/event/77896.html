正しい知識を伝えることで
生徒に喜ばれる講師になりませんか？

〜身体のパートでは〜

それぞれの臓器の部位や役割はもちろん
食事や７大栄養素との関係をお伝えします。

身体に現れる症状に対して
栄養を摂ることで改善が見込めることもあり
普段の生活の中での食生活を考えることで
予防医学として活用することも可能です。

〜心のパートでは〜

日本人が先進国の中でも足りていないと
言われている心の問題”自己肯定感”
まずは自分自身の心を知ることで人間力を高めます。

さらに
今注目されている話の聴き方 / 質問力を学び
講師として活動する際の人の心のケアの方法が解ると同時に
交渉力を身につけることが出来ます。

【先輩講師の声】
〜〜〜〜〜〜〜〜〜

良い意味で期待を裏切られました。
テキストに書いてある内容よりも
講師の先生の口から出る言葉に気づかされることがとても多かったです。

損得ではなく、受けることで人としての幅が広がりました。

〜〜〜〜〜〜〜〜〜

〜今回の説明会＆体験会では〜

身体部門トレーナー：松田千恵子先生より
治癒力マイスター認定講座の全体像・流れを説明して頂くとともに
身体の新常識やTVでも話題の腸内フローラ
脳や遺伝子、免疫力と自律神経、栄養など
総合して学べるスクールとして、様々な知識をお伝えします。
（※参加者の方に合わせて話すため、講座の内容は一部変更する場合があります。）

あなたも心と身体の知識を学ぶだけでなく
多くの方へと伝える講師になりませんか？
