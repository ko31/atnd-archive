description

2018年5月25日よりEUの新しい個人データ保護法（一般データ保護規則：GDPR）が施行されます。GDPR違反の場合、最大全世界売上の4%以下、または、2,000万ユーロというこれまでにない多額の制裁金が課せられることになります。
本アカデミーでは、GDPRの概要をご説明するとともに、GDPR対応で重要となる欧州における個人データの適切な処理と欧州域外へのデータ移転に関するIIJ GIOを用いた有効な解決策をご提示します。

～GDPRがもたらす日本企業への脅威に関する正しい認識～

GDPRへの対応は欧州でビジネスを展開する全ての日本企業にとって必須のものとなりますが、その認知度はまだ十分とは言えません。本講演では、GDPRへの対応が何故必要なのか？という基礎的な情報をご提供します。

～クラウドサービス「IIJ GIO」を用いた有効策のご提示～

GDPRへはIT対策として何を行う必要があるのでしょうか？その具体的なステップを解説するとともに、IIJ GIOが世界に先駆けてご提示するGDPRに対する有効なソリューションの詳細について解説します。
20:25～20:45 20分 質疑応答
