講座概要

WordPressと並ぶCMSの大御所「Movable Type」

CMSとはWebサイト作成のスキルがなくてもWebサイト作成、管理が出来るシステムのことです。 MovableType.netは、標準テーマのカスタマイズだけでスピーディーにレスポンシブデザインのウェブサイトが構築できる、インストール不要・サーバー管理不要のウェブサービス型CMSです。 Webサイトの制作予算に応じ、標準テーマをベースにして構築コストを抑えたり、カスタムフィールドを活用した高度なカスタマイズや、HTMLレベルで自由にデザインすることも可能です。

こんな方におすすめ！

・CMSに興味がある方

・Movable Type 未経験の方

・Webサイトの制作コストを抑えたい方

・低コストで安全なWebサイトの運用環境を提供したい方

・小規模・中規模のWebサイト制作を効率化したい方
