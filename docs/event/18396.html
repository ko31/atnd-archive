秋葉原勉強会: 楽しく宿題を片付けよう！

みなさん、勝間塾の宿題できてますか？

なかなか一人で続けるのはツライものがありませんか？

そこで次の宿題をやっつける(！）ための勉強会を企画しました。

（私自身、「ボイス」がよくわからないので
「第8の習慣」を読んでから参加する予定です。）

  • 宿題【7月】
    1. 課題図書2冊
    2. 自分キャリアの歴史をまとめる
    3. ボイスを探す
    4. 5年後、10年後のキャリアを設定し、そこにいたるパスを書く

参加する方は、必ず１つ以上終わらせてから参加しましょう。
やってみてよくわからなかったことや、疑問に思ったことを話し合いましょう。

開催要項

  1. 日時:2011/8/16(火)19：00-21：00
  2. 集合：現地集合（勉強カフェ　秋葉原ラーニングスタジオ）
  3. 予算：約￥700～1,400(人数による）
  4. 定員：5～10名

秋葉原ラーニングスタジオを見学させていただきましたが
落ち着いた雰囲気で、カフェのように他の人の眼を気にせず
ゆっくり語りあえる点がよいと思います。
ちょっと費用はかかりますが、カフェでプライベートな話を
するのはちょっと・・・という方、ご一緒しましょう。
