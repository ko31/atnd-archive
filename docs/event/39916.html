■内容

Google アナリティクスを使っていると出てくる疑問を逆引きで説明します。
よくある使い方系のセミナーとは違い、現場で使っている人にしかわからない細かいネタも盛り込んでいきます。

当日はおススメのアドバンスセグメント（約70個）とカスタムレポートの共有URLもお渡ししますので、翌日からのお仕事に活かしてみてください。

説明する内容(一部抜粋)
・先週のデータを素早く見るには？
・スマホだけのアクセス見るには？
・時間ごとにアクセスを見るには？
・URLとタイトルを同時に表示するには？
・アドバンスセグメントを並び替えるには？

■講師紹介

森野誠之 ( @uneidou )

主に中小企業を中心としてGoogleアナリティクスを利用したサイトの分析、改善提案やリスティング広告を用いた集客改善など、サイト運営の手伝いを行なっている。最新情報を抑えながら地 方かつ中小企業向けのノウハウをわかりやすく説明できる数少ない人物。 愛知大学非常勤講師。

毎日発行されるWebマーケティング情報のメルマガがひそかに人気です。
