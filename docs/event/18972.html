第5回日本Androidの会千葉支部勉強会

日本Androidの会千葉支部では第5回日本Androidの会千葉支部勉強会を開催いたします。

学生さんは長い夏休みの最中だと思いますが、夏休みに少しAndroidをいじってみませんか？もちろん、仕事がある社会人もWelcomeです！

概要

今回は、簡単なAndroidゲームアプリ開発のハンズオンを行います！夏休みなのでゲームを作りましょう！

今回は、あらゆる世代で楽しめる（はず）の「スーパー○○オ(つなぎのおじさんが土管に入たりキノコ食べたりカメ踏んだりするやつです)」の超簡易版です！

対象

今回のゲームアプリ開発は、ハンズオン形式で行います。ゲームが動作することを目標に開発を行います。いずれかのプログラム言語の簡単な経験でもあるとよいですが、経験などがない場合も開発の雰囲気は伝わるように進めたいと思います。

開催日程

開催日：2011年8月27日(土) 12:30～16:45(受付開始は12:15)
場　所：千葉大学　工学部２号棟　１０３室

(西千葉駅を降りてすぐにあります千葉大学です。来場はできるだけ公共交通機関をご利用ください。公共交通機関以外でご来場の場合、周辺の駐車場をご利用ください。)

注意事項
当日は12:30〜開始でございますが、今回お借りする教室では軽食を含め食事は行えません。あらかじめ食事を済ませた上でご参加いただきますようお願いいたします。

同大学院・知能情報コースの教育・研究についてご紹介させていただきます。
なお、発表者は情報画像と言いながら音声情報処理（音声認識、話者認識）と福祉工学（手話認識）が専門なので、やや話が偏るかもしれませんがご容赦下さい。

ハンズオンの詳細

超簡易スーパーマ○オを作ってみよう！

サンプルアプリを題材にして、ゲームアプリの作り方、考え方を見ていきます。
これを聞けばオリジナルのゲームを作れるようになる！（かも！？）

ハンズオンの構成

  1. Canvasを使って画面に絵を描いてみよう！
  2. 画面タッチで絵を動かしてみよう！
  3. SurfaceViewとTimerを使って動きのある画面を作ってみよう！
  4. 自キャラと敵キャラの動きを組み込もう！
  5. 当たり判定と点数表示を組み込んで完成！
  6. マ○オの戦いがいま、始まる・・・！
