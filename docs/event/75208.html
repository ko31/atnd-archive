～仏教を説かれたお釈迦様（ブッダ）の智恵から、　人生で大事なテーマを学べます～

日常で、こんなことを思われたことはありませんか？　

「私らしく、迷いなく生きていきたい。
　でも、そもそも私って何だろう？」
　
「毎日、会社と家の往復で同じことの繰り返し。
　いったい、何のために生きているのかな？」　

「職場の人間関係に、もう疲れた…。
　どうすればいい人間関係をつくっていけるのかな？」

職場でも家庭でも、
このような悩みを人知れず抱えている人は少なくありません。

それらの疑問に対し、
仏教（ブッダの智恵）ではどのように教えられているのでしょうか。

※内容は、講座ごとに変わります

■この講座に参加して、得られること。

お釈迦様の教え、変わらない真実を知ることができます。

何を言われても、自信を持って生きることができます。

人生を語りあえる仲間が得られます。

◆参加者の声◆

～自分のことなのに、わかっていなかった～

今まで自分のことは自分が一番よく分かってると 思っていましたが・・
私のことなのに、よく分かっていなかったことが分かりました。
初めての参加でしたが、仏教の言葉を私にも分かる言葉で話されるので、とてもよく分かりました。
（20代・女性）

～生まれてきたのは何のためかがわかった～

日頃から何のために生きているのかを考えることが多いのですが、
生まれてきたのはこの為、人生の目的を達成することはできる、本当の幸せがあるのだ、ということを知れたことは大きいです。
ありがとうございます。
（30代・男性）

～原因と結果の関係を聞けて、気持ちが楽に～

「頑張っているのにどうして、報われないんだろう」と思っていましたが、
講座で、原因と結果の関係を詳しく聞けて、結果を気にせず努力していけば、時期がくれば返ってくると知れて、気持ちがとても楽になりました。
（30代・女性）

～思いやりの精神を知り、心も軽やか～

会社の上司と人間関係がうまくいかず、転職考えていたのですが、
仏教の、思いやりの精神を知ってから 心が大分軽くなり、また頑張れそうです。
思い切って参加してみて良かったです。有難うございました。
（20代・女性）

仏教を一から楽しく学べる分かりやすい講座です。
質問の時間もあります！
