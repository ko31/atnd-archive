description

2010年11月に開催された東京デザイナーズウィーク期間中に（株）NTTデータエンジニアリングシステムズブースにご来場されたお客さまに、下記イベントを実施することになりました。積層造形に関心のある方や、もう一度詳しく説明を聞いてみたい方、必見のワークショップです。実機のデモもございますし、ご自身のデータでの造形も行う予定です。またとない機会ですので、是非ご参加ください。
なお、ワークショップの関係で、定員は5名とさせていただきます。

参加者特典　ご自身の3Dデータで造形が可能です。3Dモデリングソフト（ライノセラス等）で作成したデータをお持ちいただくと、後日造形させていただきます。

  • STLもしくはIGESデータをUSBメモリーなので当日ご持参ください。
  • 造形の大きさは、ご希望の大きさにならない場合もございます。事前にご了解をお願いします。
  • お持ちいただいた3Dデータは、他の参加者がご覧になることもございます。また、造形物はウェブページ等で公開させていただくこともあります。事前にご了解をお願いします。

プログラム内容
受付開始13：00　ワークショップ開始時間13：30

  • 積層造形装置（FORMIGA）の操作とデータ作成のデモと体験
    • 装置構成について
    • 操作体験（装置準備～取出し作業～造形開始）
  • データ作成作業の流れとMagics(造形用の編集ソフトウエア)の紹介
    • Materialise(マテリアライズ)とMagics（マジックス）について
  • STL編集から造形開始までの体験
  • Magicsとそのモジュールの体験
    • Pシリーズでの活用例（Magics+Smartspaceデモ）
      （プログラムは事前の予告なく、変更になることもございます。ご了解をお願いいたします。）
