    Qt5 のセンサー API を使ってみたい

    Qt5 からは各種センサーを扱う為の API が用意されています。今回はタッチパネル操作処理をこのセンサー API で実装する仕組みについて紹介します。
    スピーカー： You＆Iさん （@you_and_i） プログラミング生放送 名古屋支部 支部長

    Android-Binding Before/After

    Android アプリ開発で、View 層と Model 層を分離する「Android-Binding」についてお話します。Android-Binding は Binding 機構を提供してくれるライブラリです。Android-Bindnig の機構を使えば、View 操作コードとロジックコードが分離でき、スパゲッティーコードを減らすことができます。
    スピーカー： 大場 知悟さん （@tworks） 楽しいアプリ制作の会 代表

    JSON の Windows クライアント作成時のポイント

    Windows 8/RT ストアアプリや Windows Phone、WPF（Windows Form も含む）などで、JSON データの受け側プログラムを記述するときの違いや注意点についてまとめました。特に日付型データで悩んだ方（悩みたくない方）必見です！

    スピーカー： Lundi さん （@Lundih） 勉強会 MiCoCi 共同代表

    Windows 8.1 ストア アプリ FIRST LOOK for 8.0 developers! [SS]

    Windows ストア アプリの開発をやってる人に、Windows 8.1 のヤバい話をしちゃいます

    スピーカー： 山本 康彦 (やまもと やすひこ)さん （@biac）　BluewaterSoft
