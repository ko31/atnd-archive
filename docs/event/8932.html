広島 Emacs勉強会

広島でも Emacs勉強会をやろう！ということで軽いノリでやることになりました。
発表者も募集しています。
こんなことやろうぜ！的なことがあれば #hiroshima_emacsとか @eielh までお願いします。

内容(予定)

  • 会場入り前に昼食(お好み焼き)時間は
  • 会場準備
  • ポジションペーパを使った自己紹介
    ポジションペーパとは: http://eto.com/d/PositionPaper.html
    • ポジションペーパにかいてくるとよさそうなこと例
      • 自己紹介の要約的なこと
      • twitterなどなどオンラインの活動拠点とか
      • 私にとってemacsとは？emacsのすきなところは？
      • 作成したelispとか
      • あなたのテキストエディタ(or IDE)歴
      • おもな作業環境とか最近やってることとか
      • 参加しようと思ったわけ
      • まったく関係ないけど宣伝したいこととか
      • etc….. (ほかに書いてきて欲しいことがあれば教えて！)
  • auto-installとかEmacsの拡張について(仮) – @NeXTSTEP2OSX
  • emacsでの開発環境とかいろいろ(おもにlRuby) (仮)- @eielh
  • 突発でLTどうぞ
  • emacsについて自由に情報交換
  • 片付け
  • 会場退出後にどこかで雑談(つけ麺？)

自作のelispとかおすすめのelispとか紹介などなんでもいいので発表してくれる方を募集。

勉強会前にお好み焼きを、勉強会後に広島つけ麺でも食べにいこうかと思います。


お好み焼き参加者は 広島そごう入口からくり時計の前に 11時集合でお願いします。
来ようかと考える方は @eiellhまで連絡をくださると助かります。
\# あと近場のおすすめの店おしえてください


勉強会後の雑談は人数すくなそうなので特に予約とかしてないです。
