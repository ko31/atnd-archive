・プロフィール
80年代、PC-8001mkIIでプログラムを作りはじめ、Z80Aニーモニックを習得。その後C++を身につける。
90年代、LC2を購入しMacに入門。一方でインターネット技術やJavaなどを学習。
2000年、関西テレビソフトウェア社に入社しウェブコンテンツ制作に従事。その傍らで2007年からデジタルハリウッドにて講師を勤める。
趣味でiPhoneアプリ開発を始め、2011年に退社しiPhoneアプリ開発とスクール運営を手掛けるアプリル株式会社を設立（http://www.applil.com/）。
