description

多種多様な演出で人々を魅了する遊技機コンテンツ。先日発表されたパチスロ機における「規制強化」は業界に大きな影響を与えようとしています。様々な情報が飛び交う中、「今回の規制強化の内容」「今後求められる演出とは」「遊技機業界の動向」など液晶開発のクリエイター向けたセミナーを開催致します。

講師には、業界紙「遊技日本」よりパチスロ規制強化の第一報を掲載した（株）遊技産業研究所の鈴木政博氏・長谷川豊氏をお招きして、ご説明いただきます。

遊技機液晶開発やグラフィック業務に身をおいていながらも、開発の繁忙によって業界全体の動きや問題を知る機会が少ないのではないでしょうか？

「業界関係者」は勿論の事、「遊技機業界問わずデザイン業務に従事している方」「各種コンテンツ企画・プランナーの方」「遊技開発に携わりたい方」は必見です。

遊技機業界の専門家から規制の流れや今後の業界の展望を無料で聞ける事はなかなかありません。この機会に是非ご参加下さい。
