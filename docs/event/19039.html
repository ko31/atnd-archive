catch ～地図の未来を描こう～
description

地図といえばどんなものを思い浮かべますか。

最初に出会ったのは、絵本の中に出てくる「宝の地図」でしょうか？それとも、小学校での地図帳でしょうか。
そんな地図といえば紙だったのが変わり始めたのはいつごろからでしょう？

車にはカーナビが搭載され、パソコンからは世界地図が見れるようになり、更には地球も眺められるようになりました。
そして、携帯電話にもGPSが搭載され、手のひらにナビゲーションシステムが収まるようになりました。
今では、写真に位置情報（緯度経度等）が書きこまれ、いつどこで撮影されたのかまでわかるだけでなく、例えは「セカイカメラ」のようなスマートフォンと連携した様々なアイデアが生まれています。

そして、仕事のやり方も変わってきました。
あらゆる情報が地図の上に載せられるようになると、それまで全く無関係であった情報が、地理情報を通じて面白いようにつながっていきます。

その地理情報を使ったGIS(Geographic Information System)という仕組みを活用し、仕事の改善に取り組んできた直方市の自主研究グループが、その活動実績が評価され、世界最大のシェアを誇るGIS開発業者のESRI社より「SAG賞」を受賞しました。
http://www.nishinippon.co.jp/nnp/item/247657

今回は自主研究グループの代表としてアメリカ、サンディエゴでの授賞式に参加した直方市職員のである松田欣也さんをゲストにお迎えします。また、地図業界大手である国際航業(株) の植原雅子さんをゲストにお迎えし、地図の歴史についてお話頂きます。

今回のOff Cafeでは、お二人のゲストのお話をお聞きしながら、地図がどのように生まれ、そして私たちの生活をどのように変えてきたのかという地図の歴史について学び、これからの地図の活用方法や、地図で生活がどのように変わっていくのかを参加者のみなさんと一緒に考えてみたいと思います。
