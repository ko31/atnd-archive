1/24（土）カラオケ＆炭火焼肉で新年会

description

タイトル通り新年会したいな～って思い企画しました。
参加はウル４関係の人が多くなるかと思いますが、オフ会みたいなノリなので皆さんお気軽にご参加下さい。
カラオケだけ参加、焼肉だけ参加でもOKですよ～

日時：2015年1月24日（土）
場所：兵庫県神戸市の神戸駅周辺
簡単な流れ：昼間～夕方までカラオケして、から美味しい炭火焼肉食べに行きます。

【詳細】
～カラオケの部～
時間：13:00～18：30頃
場所：ファンキーポリス神戸店http://www.funkypolis.com/shop-kobe.htmlに現地集合して下さい。
JR神戸駅の北へ徒歩１分です。
料金：昼のフリータイム６５０円でソフトドリンク飲み放題。
ちなみに、利用出来る時間が10：00～翌朝5：00までと書いてますが間違いではないですｗｗｗ
【備考】
カラオケはアニソンでも、ボカロでも、演歌でもなんでも好きなの歌ってもらってOKです。尚、社長は主にV系ロック歌います。
カラオケ歌わなくて聴いてるだけ、雰囲気を楽しむだけでもOKなんでお気軽にどうぞ。

～炭火焼肉の部～
時間：19：00～20：00頃
場所：たじま屋http://tabelog.com/hyogo/A2801/A280109/28004023/
住所は兵庫県神戸市兵庫区西出町2-3-4
料金：4,000円くらいになると思われ。
食べ放題、飲み放題ではないので食後の料金を人数割りします。
【備考】
神戸駅から南へ国道沿いに徒歩約12分くらいです。
立地が若干悪いのですが、上質で美味しいお肉をそんなに高くない料金で提供してて常に予約が一杯になってるお店で、本来なら１ヶ月以上前から予約しないとまず土日は空いてないのですが、社長のコネとかをゴニョゴニョしたら予約取れましたｗ
ホンマに俺もめったに行けないお店なので食事だけでも参加する価値はありますよ。
写真とかはURLをご参照下さい。写真の量が多すぎるのでここには載せれませんｗ

～終わってから～
特に何も決めてませんｗ
三宮に行ってゲーセンに行くなり、２次会に突入するなりはその場のノリで。
尚、翌日は社長はカギさんを神戸案内＆観光するので一緒に回りたい方はどうぞ(´ω｀)

こんな感じなのと、告知が遅くなったので日があんまりありませんが皆さんご参加宜しくお願いしますね～。
