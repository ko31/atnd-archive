★★ 女 性 の フ ィ ロ ソ フ ィ ー ★★

しかし現代社会は、女性に多様な役割を要求してきます。
愚痴も言えず、ストレスの蓄積に発散が追いつきません。

男性より女性が疲れる時代になり、多くの女性が悩み迷っています。
悩みは堂々巡り、考えないと自分らしさの道は、見えてきません。

これからの世界で女性の輝きを失わない為、内面の魅力・心の免疫を会得して欲しいと願い、女性限定で企画します。

難しい言葉ばかりの哲学でなく、子供時代の様なシンプルな疑問から、考える力を伸ばすレッスンです。
