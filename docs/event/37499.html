第2回 Twilio API 勉強会。

2013年の春！もうすぐです。
アメリカのクラウド電話API Twilioが日本でサービスされる予定です。
そのTwilioの機能を使って面白いサービスを作る開発者向けの勉強会です。

■■ 電話APIとは？ ■■

どんなサービス（システム）からでも特別な機器なしに簡単に、しかも格安で電話につなぐことができるAPIです。

今年注目のクラウド電話API Twilioのテクニカル勉強会です。
短いコードを書くだけでウェブや様々なサービスから電話を掛ける事ができるクラウド電話API Twilio！
その電話APIの可能性や楽しみを皆さんにご紹介します！！

今回はPHPを利用して架電を分岐させる処理を皆さんと作ってみたいと思います。
例えば）
プッシュ音「1」を押すとメッセージを流す。
プッシュ音「2」を押すと音声ファイルを再生する。
プッシュ音「3」を押すと他の電話に電話をかける。

ビールを飲みながら盛り沢山のTwilioの機能を使って楽しく電話APIを試してみませんか？
新しい技術に興味深いエンジニアの皆さんの参加をお待ちしています。
