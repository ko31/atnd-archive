『Office 365 実践セミナー！ ～移行時/導入後のお悩みポイントをMicrosoft MVPが解決します～』

※Microsoft MVPとは
無償で Microsoft製品の情報・知識を一般ユーザーに提供し、尽力している個人を Microsoft 社が表彰しています。
そのため Microsoft MVP はその分野のプロとして、一目置かれる存在です。

この度、サーバーエンジニアとしてご活躍されている方、現在は運用・保守業務に従事しているが更なるキャリアアップを目指される方々を対象として、無料セミナー『Office 365 実践セミナー！ ～移行時/導入後のお悩みポイントをMicrosoft MVPが解決します～』、および会社説明を2月21日（水）に開催いたします。

今回のセミナーでは、Office 365移行と導入後の保守サポートを実際に行っているMicrosoft MVPが、今後クラウド導入支援を行っていくエンジニアの方々を対象に、クラウドサービス移行時の事前確認ポイントや導入後の運用でハマりやすいポイントについて、最新の実例をもとにご紹介致します。参加者の方でOffice 365導入で実際に悩んでいるポイントがあれば、事前に内容をお知らせください。

＜対象者＞
・サーバー構築、もしくは運用・保守業務を1年以上経験されている方
・インフラエンジニアとしてキャリアアップしたい方
・プログデンスへの応募を今後検討したい方
　※本セミナーは選考会ではありません。
　※サーバーエンジニアとして活躍中の 20代半ばから 30代半ばの方々を中心に好評いただいております。

■詳細・申込ページ https://www.progdence.co.jp/seminar/o365_seminar_2018-02.html
