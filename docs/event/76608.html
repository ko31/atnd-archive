catch
description

日本で、そして、海外で。世界中の人に「日本語」を教えてみませんか。
「日本語を日本語だけで教える方法」があるから、
語学力を気にせず、世界中の人たちと交流できます。

「日本語教師」、「日本語ボランティア」という言葉を聞いたことがありますか？
これらは、皆さんが毎日話している「日本語」を「外国人に教える」お仕事・ボランティアです。

しかし、「日本語を教える」と言っても、どんな人たちに？どこで？どうやって教えるの？と
分からないことだらけではないでしょうか。

そこで、この講座では、外国人に日本語を教えることに興味を持たれた皆さまに現役日本語教師が
「日本語教育の基礎知識」と「外国人から見た日本語」について、わかりやすくお話します。

この春、何かを始めたい・・・と思っている方。
日本人であることを活かした国際交流を一緒に始めましょう！

語学力、職業、年齢は一切問いません。「興味がある」が参加資格です！
