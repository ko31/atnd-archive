9月28日から10月2日まで米国で開催された"MySQL Central @ OpenWorld"で発表された最新情報をご紹介！

・現在LT発表者を絶賛大募集中です！　発表希望の方はTwitterで@yyamasaki1宛に連絡頂くか、コメント欄にて話す内容を書いて下さい！

紹介予定の内容

  • MySQL 5.7 DMR 5 ( ダウンロード )
    オプティマイザの新コストモデル、コネクション生成負荷を削減、マルチスレッドスレーブ、など
  • MySQL Cluster 7.4 DMR ( ダウンロード )
    共有ディスクを使わずにアクティブ-アクティブのクラスタ構成を実現、パフォーマンスを更に向上
  • MySQL Fabric 1.5 ( ダウンロード )
    データシャーディングと高可用性を実現できるフレームワーク、OpenStackとの連携、シャーディングキーに日付型/文字列型も追加でサポート
  • MySQL Workbench 6.2 ( ダウンロード )
    MySQL Fabricにも対応、GIS Viewer、Visual Explainを更に強化、Accessからのマイグレーション支援、など
  • MySQL Enterprise Encryption ( 概要 )
    機密データを保護するために公開鍵を使ってデータを暗号化
  • Oracle Enterprise Manager for MySQL ( 概要 )
    Oracle Enterprise ManagerからMySQLも監視
  • Early Access Features(Labリリース) ( ダウンロード(Labリリース) )
    クエリー・リライト・プラグイン、New データ・ディクショナリ、透過的なページ圧縮、グループ・レプリケーション、MySQL HTTP Plugin、など
