話題の就活論者２人が対談!?

10万部のベストセラー『就活のバカヤロー』著者・石渡氏と
ジョブウェブの代表取締役・『就活廃止論』著者・佐藤氏に
よる夢の対談が埼玉大学にて実現!!!

就活を始める今、一番考えなければならないこと。

それは「働く会社」ではなく、「働き方」

単なるハウツーでは終わらない、就活に長年向き合っている2人だからこそ語れる、本当の就活のあるべき姿とは!?

他では聞くことの出来ない、就活制度の裏側から、今求められている人材像まで余すところなく語って頂きます。

また、これは単なる就活セミナーではありません。

自分の労働観・人生観を考え直すためのセミナーです。

就活を終えた４年生も、就活なんてまだまだ先と思っている１、２年生もこの機会に自分の価値観と向き合ってみませんか？
