title 1日完結！女性創業講座
catch これだけは知っておきたい創業のポイント
description

「特技や経験を活かして　創業したい」「いつか　自分の夢をかなえたい」
「雇用されない生き方をしたい」「好きなことで収入を得たい」
そんな女性のために、創業の基礎知識をわかりやすく、具体的にご紹介します。

豊富な創業事例から、女性が創業する際の成功ポイント・注意点がしっかりわかります。さらに、金融機関のナマのお話も聞けます。融資の予定が無い方も、必見です。この機会に夢を現実に近づけましょう。

☆特徴その１☆
創業の基礎知識が　ピンポイントでわかる！

☆特徴その２☆
金融機関の担当者のリアルな話が聞ける！

☆カリキュラム☆
9：30～10：30　女性が創業するきっかけ
・女性の創業事例　～　成功事例／失敗事例　
・女性ならではのビジネスモデルとは
・創業者に必要な資質～主体性、独創性、柔軟性、実務能力

10：30～12：00　「アイデア」を「ビジネス」にするには？
・アイデアを組み立てるステップ
・ターゲット層の明確化が肝要
・「売れるしくみ」はこう作る
・競合に勝つには？～スモールビジネスの戦い方

13：00～14：00　個人事業と法人事業のちがい
・個人事業ではじめるか／法人にするか
・法人の種類～株式会社、合同会社（ＬＬＣ）、ＮＰＯ法人、一般社団法人、等
・おもな開業手続き

14：00～15：30　ビジネスプランは必ず作りましょう
・ビジネスプランとは？
・収益性をシミュレーションしよう
・良いビジネスプランと悪いビジネスプラン

15：30～16：30　金融機関は創業者をどう評価するか
・創業時の融資制度
・金融機関が見た創業者の評価ポイント
・女性創業者への融資の事例
・質疑応答
