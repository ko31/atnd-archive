◆Bonheur（ボヌール）

web shop URL → https://www.creema.jp/creator/27420

「Bonheur（ボヌール）」はフランス語で「幸せ」や「幸福」を意味します。 皆さまにアクセサリーを身に付けて、少しでも気分が上がったり、幸せな気持ちになっていただけたらなと思いこの名前を付けました。

Bonheurでは樹脂ノンホールピアスを中心に制作しています。 （ピアスに変更も可能です。※一部変更できない作品がございます。）

私自身が金属アレルギーでアクセサリーを気軽に 付ける事ができず樹脂ノンホールピアスに出会いました。 でも、なかなか気に入ったデザインがなく 自分で作り始めたのがきっかけです。

シンプルなデザインで、色々なシチュエーションに似合う アクセサリーを目指し、日々制作しております。

何気ない日常に、ささやかな彩りを添える事ができたら嬉しいです。

※こちらの写真はイメージとなります。

◆　good feel good sign

ハーバリウムとは、特殊なオイルにドライフラワーやプリザーブドフラワーを 瓶に入れた「植物標本」です。 お部屋のインテリアとしてお楽しみいただけます。

キレイな植物のハーバリウムを見て、 少しでもシアワセな気持ちになっていただけたらと思い制作しています。

※こちらの写真はイメージとなります。
