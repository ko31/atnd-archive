GitHubやGitの使い方をみんなで勉強します。筑波大生向け。

主に全く使ったことのない方や、活用方法がわからない方を対象とします。 プロの方も大歓迎ですが、講師や教材開発をお願いするかもしれません。

今のところできそうだなと思っているテーマとして、

  • GitHubとGitについて（初回はこれ）
  • Markdownについて
  • GitHub Pagesで自分のWebサイトを作ってみる
  • http://open-klis.github.io/ を共同編集してみる
  • CIについて
  • Git Challengeをやってみる
  • OPACを作って破壊する
  • Issueの使い方
  • GitHub flowとか
  • GitHubで論文/文章を管理する
  • Gitコマンドについて
  • GUIツールについて

辺りを考えています。「これを知りたいんだ！」というのがある場合は教えていただければ採用するかもしれません。

open-klisってなに

去年の今頃に登場した「何か」です。当時もGitHubの使い方をみんなで勉強しようとして作った組織ですが、現在に至るまで１度も活動したことがありません。KLCのGItHub講座というのが実施されるらしいということで様子見していた記憶がありますが、KLCでは開催しないらしいのでリベンジしてみようと思った次第です。
