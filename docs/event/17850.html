梅雨も明けたし暑いし、BBQやジンギスカンの季節到来！

でも準備は面倒だし、ドライバーは酒飲めない…(´・ω・｀)

そこでビール園！

駅から送迎バスがあるので運転手にも優しいし、
ビール園だから出来立てビールを飲めないとお嘆きの方も大満足！

手もみラム・塩漬ラム・味噌漬ラムの３種ラムジンギスカンが美味いらしく、
お酒が飲めない貴女・貴方も食事だけできっと満足！(∩´∀｀)∩
