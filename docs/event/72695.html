RETURN OF TUMBLELOG

エピソード6「タンブルログの帰還」

まさにtumble、崩壊的である。

( via yuiseki )

我らが愛する tumblr がこの世界に華々しいデビューを果たした2007年の春、tumblr には reblog という機能はありませんでした。我々は reblog を愛してやみませんが reblog はこの tumblr の魅力の一部でしかありません。今回の RAC ではこの tumblr の原点である tumblelog の精神を取り戻したいと考えています。

  • tumblelog についての調査報告 – 2013

Advent Calendarとは

かつて地球には、クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーがありました。21世紀にはこれが転じて、ギークコミュニティで、クリスマスと呼ばれる祝日までの25日間に1つずつ箱に閉じ込められた魂を公開する儀式が行われたそうです。これを Advent Calendar と呼びました。

かつての様子はこのようなものでした。

  • Reblog Advent Calend*e*r 2010
  • Reblog Advent Calend*e*r 3011
  • Reblog Advent Calend*e*r 7889
  • Reblog Advent Calend*e*r mach33
  • Reblog Advent Calend*e*r No.5

d*e*r とは

かつて、それはただの typo でした。

RAC ( reblog advent calend*e*r ) とは

RAC/history を、ご覧ください。

リブログ枠無し(25日25組)、リブログ禁止、一人一アカウント、アカウントBAN

  1. 今回、リブログ枠はありません。参加できるのは25日間で25組までです。タッグマッチによる参加は許可します
  2. 今回、リブログによる投稿は禁止とします。投稿される post は全てフレッシュでなければなりません。dashboard の外側に存在するまだ見ぬ世界を切り刻んでください
  3. 今回、一人が使用できるアカウント数は一つとします。リブログを禁止しますので1日で投稿できる post 数は原理上、最小で1、最大で150となります(75post×2day)
  4. 今回、参加者が居ない日については、投稿を禁止とします
  5. 上記のルールが破られた場合には、会場自体は消去されます。自主的アカウントBANとなります。その際に、再度会場が設定されるかどうかは状況により判断します

参加資格

  • tumblr account を持っていること

参加方法

  1. このページの[このイベントに参加登録する]から登録してください
    • ユーザープロファイルか参加コメントで自分の tumblr の url がわかるようにしてください
  2. コメント欄に自分が担当したい日付を宣言してください
    • 日程は早いもの勝ちです
  3. atnd のメッセージ機能を使用して 会場 にログインする鍵をお渡ししますので、入場してください

競技規則

コンテンツを詰めた麻袋を武器に河川敷で殴り合ってるイメージ（もちろん無言で）

( via mtakeshi )

年に一度、コンテンツを詰めた麻袋を武器に河川敷に集まるイベントです（もちろん無言で）。 会場 はこちらで用意しました。

  1. 会場は RAC 専用の tumblr アカウント( tumblr group ) を使用します
  2. 12/1から順番に、一人一日を担当します
  3. 12/25 24:00 の鐘が鳴ったらカーテンコールとなります。12/26 24:00 まで、参加者各位は思い思いにリブログするなりなんなり好きにしてください
  4. 12/26 24:00 の鐘が鳴り終ったら 会場 は消滅します
