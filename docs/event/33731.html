ルール

１）参加したい方は参加したいという思いを強く込め「参加！！！」と一言入力していただいて【参加】ボタンを押して参加してください。

２）担当したい日にちをコメントに書いてください。１日じゃなくても連続３日とかでもいいです。誰も名乗りでない日は、ボクがなんか書きます。

３）書く内容は自由ですが「小ネタ」限定とします。スゴい記事書いたら空気読めない人扱いの方向です。

４）書いた記事の URL をコメントで報告してください。担当日の宣言と投稿報告でコメント欄が祭りの賑わいになるかもだけど気にしないでください。

５）万が一人数がオーバーしたりして、日数と合わない場合は同日複数名の方が担当するのもアリにします。

６）記事の投稿ができなくなった場合は Twitter とかで DM 貰えれば対応します。多分誰か書いてくれるはずです。誰もいなかったら、ボクがなんか書きます。

“early” Advent Calendar とは？

そんなものありません。勝手にさせてもらってます。
Advent Calendar の１ヶ月前の１１月１日〜３０日の小ネタ限定で誰かがブログを投稿します。誰もいなかったら、ボクがなんか書きます。

Advent Calendar とは？

技術系Advent Calendarでは基本的に，12月1日から25日までの25日間，プログラミング言語縛りなどの最低限のルールで，毎日別の人（同じ人が複数回担当することもある）が自分のblogにて記事を書くという形式になっている（特設サイトを用意して，記事をまとめる手法もある）。なお，国内ではATNDの参加順がそのまま執筆する日の順番となっていることが多い。
