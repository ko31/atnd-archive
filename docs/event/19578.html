description

勝手にシリーズ化計画。
4月、6月と開催して、8月は多忙＆暑さに負けてお休みしましたが…
3回目やります！
幹事の都合でまた土曜日の夜です。しかも3連休の真ん中とかでごめんなさい。

六本木に被災地のお酒を持ち寄って、美味しいお好み焼きつつきながら飲みましょう♪
持ち込みのお酒は、基本4合瓶でお願いします。
それより小さい瓶の場合、2〜3本同じ銘柄を持ってきていただけるとみんなで飲めて嬉しいな。
一升瓶がわーっと集まるとさすがに飲み切れなくて困ったことになりますので、なるべくお控えくださいm(_ _)m

最初の乾杯はお店の飲み物（ビールでもラムネでもウーロン茶でもなんでも可）でやります。はてなの売上貢献にご協力お願いします。

お酒はあんまり/全然飲めないけど…という方は、ソフトドリンクや料理の注文で快く持ち込みOKしてくれた「はてな」の売上に貢献を！

持ち込みのお酒は、全員強制ではありません。手に入る方よろしくお願いします！
料金は、はてなメニュー注文分（食べ物＆飲み物）を若干重み付けして割り勘。
おそらく3000円〜高くても5000円程度との予想です。
持ち込み酒は計算しきれないと思うので、すみませんが振るまい酒ということにさせてくださいm(_ _)m

なお、途中参加/途中退場無問題！
参加費も配慮しますので、遅刻しちゃう〜とか早く帰らなきゃ〜とか気にせず遠慮せずどんどんいらしてくださいね。

そして今回は、「はてな」に集まってわいわいやるきっかけにもなった
マインドマップ仲間の「ワインの会」
絡みで、おめでたいことが重なったので、そのお祝いも兼ねて！
はてな久々の方にも来ていただけると嬉しいな♪
