description

　今年も「マリンワールド海の中道」での総合学ゲー展を目指して、「海と海洋生物」をテーマにしたゲームやアートなどの制作を始めます！
　今回の八耐はいつものような「制作会」ではなく、総合学ゲー展へ向けたスタートアップイベントとなります。総合学ゲー展での展示作品制作などに興味のある方は、是非、ご参加下さい！
　当日はマリンワールド海の中道から高田館長と海をテーマにしたゲーム「アクアノートの休日」、「イージーダイバー」の飯田和敏さんを、ご招待しております！
　１４時からの講演に関しては学ゲー展参加者以外の方々の聴講も可能となっておりますので、興味のある方々は、是非、ご来場下さい。
