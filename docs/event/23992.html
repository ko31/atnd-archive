講演概要

JRubyで作るAPIサーバー

  • 出演
    • 岡山Ruby/Ruby on Rails勉強会　山本和久氏( @kazuhisa1976 )
  • 概要
    • エンタープライズ向け業務でも広く使われるようになってきたRubyですが、Excel出力や帳票出力は
      まだまだJavaで使用できる部品が多いのが現状です。
      今回はJRuby自体の説明と、Sinatra + GlassFishを使用したExcelファイル編集サーバーを作った例を説明します。
      Javaユーザーの皆様に「JRubyで少し柔らかくなったJava」の世界を楽しんでいただけたらと思います。

JavaFXで開く新世代GUI

  • 出演
    • 櫻庭祐一氏( @skrb )
  • 概要
    • JavaにおけるGUIライブラリは10年以上ほとんど進化がありません
      でした。しかし、JavaFXによって新たなステージに踏み出します。
      Java SE 8でGUIライブラリの標準となるJavaFXは、アニメーション
      やエフェクトなどを取り入れることで表現力を高め、さらにXMLによる
      UIの構築やCSSによるデザインの指定を可能にしています。
      本セッションでは新しいUIライブラリであるJavaFXについて、デモを
      交えながら紹介します。

Java7再入門講座

  • 出演
    • 岡山Javaユーザ会　吉田貴文( @zephiransas )
  • 概要
    • 待望のJava7がリリースされて約半年が経過しましたが、まだまだJava7って何？ってひとは多いはず。
      そんな人に送る初心者向けJava7入門講座。
      Java7まったく使ってない人でも、理解できるよう、デモを交えながらJava7を再度紹介します。
      これを聞けばJava7を使ってみたくなること、間違いなし？！
