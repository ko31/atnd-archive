内容：
エンタープライズJava関連のSIやプリセールスを経験し、2007年より日本アイ･ビー･エムにてWebSphereのプリセールスを担当している須江信洋氏をお招きし、「Javaアプリケーションサーバー最新動向」「ハイブリッド・モバイル・アプリケーション」についてお話しして頂きます。プライベートでは、SIの現場でGroovy関連の技術を活用する方法について日々試行錯誤している。JGGUG（Japan Grails／Groovy User Group）サポートスタッフ。JJUG（Japan Java User Group）幹事。としても活躍中。

(1)Javaアプリケーションサーバー最新動向
- 最新仕様Java EE6でのアップデート、特にアプリケーション開発の効率化に貢献するポイントについて
- Java EE6対応アプリケーションサーバー、特にTomcat/TomEEとWebSphereについて
- OSSと商用製品の差異や使い分けのポイントについて

(2)ハイブリッド・モバイル・アプリケーション
- Web技術(HTML5/JavaScript/CSS)によるクロスプラットフォーム対応モバイル・アプリケーションの開発
- Webアプリケーションをネィティブ化できるOSS: PhoneGapの紹介
- PhoneGapをコアとして採用した派生ソリューション: Worklightの紹介
