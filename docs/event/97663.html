ComGAME #1 / in 街コス痛コレ2018初夏

小矢部市 街づくりNPO 石動まっちゃプロジェクトが主催のイベント「街かどコスプレイヤー＆痛車コレクション（ 過去イベントの様子 ）」その会場となるエリアの建物を1つ借りてそこでComGAMEを実施致します！ComGAMEに参加する方はこのページで参加表明をお願いします。

ComGAME について

会場内には、フリーのテーブルがいくつが御座います。ザックリ説明しますと 「好きなゲームを持ち寄って自由に遊ぼう～！」 というイベントです。集まった人たちで持ち寄ったゲームを用いて交流や大会などをして楽しめるスペースになります！

  • ゲーム機を持ち込みたい方へ
    ゲーム機を持ち込みたい、もしくはテーブルを使ってゲームをしたい（スマホゲームやボードゲームなど）方は、こちらのページで参加表明した後、 ComGAME テーブル予約フォーム より別途ご連絡下さい。
    ※先着10名様（長テーブル各1つ）となっております。コード類、モニターなども各自でご用意下さい。
    ※既に地元ゲームコミュニティを優先して割り当てております。
  • シャドウバース大会 参加者の方へ
    シャドウバースのオリジナルグッズなどがゲット出来る大会を会場内にて実施します。
    参加者の方は、 こちらのページ でルールなどをご確認の上、このページと合わせて参加表明をお願いします。（お手数ですが、両方で参加表明下さい。）
    ※先着48名の大会となっております。

タイムテーブル

10:00 OPEN（会場、設営）
11:00 START
13:00 シャドウバース大会
18:00 CLOSE

※11:00~18:00の時間中は、自由に持ち寄ったゲームを楽しんでもらって構いません。

参加費

500円
※リストバンドをお渡ししますので、参加者の方は着用をお願いします。
※コスプレイベントに参加されているコスプレイヤーさんは参加費無料です。

参加コミュニティ

・ToyamaGaming（シャドウバース大会の実施）
・富山一武道会（ドラゴンボールファイターズ対戦会）
・富山スト5対戦会
・石川対戦会（格闘ゲーム全般）
※コミュニティ関係なく、個人でテーブルを借りて自由に遊んでもらっても大丈夫な雰囲気です。

場所

住所：小矢部市石動町4丁目25
（石動駅前より徒歩3分）

注意事項

・コスプレイベントと同時開催です。コスプレイヤーの方を無断で撮影することはNGです！
・会場内の飲食はオッケーです。ゴミはゴミ箱へお願いします。
・貴重品、持ち込んだ物は各自で管理して下さい。
・スタッフからの指示があった場合、従って下さい。
