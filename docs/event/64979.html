Instagramマーケティングの最前線がわかる勉強会

企業のブランディングのためだけでなく、ECサイト活性化ツール、キャンペーン効果増大ツールとして。また販路を開拓したい個人の販促ツールとして。
Instagramマーケティングの最前線を知る人々に講演をして頂きます。

  • 本イベントはInstagramの世界観を毀損するような商用利用を奨励するものではありません。
  • 既存のユーザーとの利用意図を尊重し、交流を深める中で、最適な効果を出すための方法について理解を深める意図で行われます。

本イベントの対象
  • 企業のマーケティング担当の方
  • オンラインで商品を販売している個人の方
  • その他Instagramに興味がある方はどなたでも

イベントのテーマ
  1. Instagramアカウント運用の基礎
  2. Instagramキャンペーンの成功・失敗事例
  3. Instagram x EC
