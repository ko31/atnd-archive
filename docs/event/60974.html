勉強会講師：森 昭暢氏

勉強会内容
AndroidStudioを使い、電卓アプリを作るというものです。
AndroidStudioとは、ソフトウェアを作るためのソフトウェア（統合開発環境=IDE）の一つです。
2014年12月にGoogleより正式リリースされた、google純正のAndroid開発環境です。
機能もEclipseとほとんど変わらないため、また今後主流になりうるため、
今回はこちらで勉強会を進めていきます。

勉強会当日持ち物

・AndroidStudioをインストールしたノートパソコン（OS問わず）
・電源

受講のために必要な開発環境

計1時間10分程です
◆1.ノートパソコンにAndroidStudioをインストール（30分）
http://developer.android.com/sdk/index.html

※MACの方で「JAVA　NOT　FOUND」と出る方は
http://support.apple.com/kb/DL1572
からJAVAをインストールし再起動してみてください。

◆2.エミュレータを動かすため、ADVのダウンロード（30分）
AndroidStudioを起動し、SDKマネージャを起動します。
ツールバー下の緑のAndroidアイコンの隣
「Android 4.4.2 API 17」にチェックをし、右下のボタンからインストールしてください。

参考リンク：http://shared-blog.kddi-web.com/activity/229

◆3.日本語化（10分）
https://github.com/yuuna/IDEA_resources_jp

右下のDownloadZipからダウンロードし、その中のresources_jp.jarをAndroidStudio/lib
フォルダにコピーしてください。その後再起動します。
参考リンク：http://androidstudio.hatenablog.com/entry/2014/07/20/170308)
