description

今まで必死で働いてきて、会社を育ててきた。
売上が伸び、従業員が増えた。顧客も増えていった。
忙しくなっていくことがむしろ楽しかった。

大変なこともいろいろあった。やめていく仲間もいた。
しかし事業は大きくなり、一定の軌道に乗った。
従業員を雇って、税金も払っている。
我々のサービスがなければ困る顧客だっている。

しかし、ふとよぎる想い。

今、従業員は幸せか？お客様は喜んでいるか？これからの社会に必要とされる会社か？
自分自身は、心から楽しんでいるか？

もう一度問い直してみるときが来たのかもしれない。　
このビジネスは何のためか？

「産業人の使命は貧困の克服にある。」

「私利を追わず公益を図る。」

偉大なる先輩たちは、志のもとに事業を行ってきた。
もちろん、自分自身がかかわる事業が社会の役に立ってないとは思わない。むしろ、必要とされ今なお存続しているこの事実は、社会に価値を提供している証拠だと思う。

しかし、あらためて『社会起業』という概念のもとに事業に取り組んでみるのもいいかもしれない。

この度10月よりスタートする社会起業大学アドバンスクラスの担任講師は、日本最大級のソーシャルエンタープライズであり、ガイアの夜明けにも登場した株式会社ボーダレス・ジャパンの代表取締役副社長の鈴木雅剛氏。

シェアハウス、革製品、子供服など多岐にわたる事業の成長とソーシャルインパクトの最大化を支えてきた鈴木氏が日々の戦いの中から培った実践知と、世界を変えるソーシャルビジネスを生み出す経営者を支援したいという想い、そして、社会起業大学の6年の実績と多くの期待が集まり、このたび社会起業家を最短距離で軌道に乗せる「アドバンスクラス」が生まれました。

貴方の経験とアイディアの融合で、世界を変える事業を生み出していきます。

授業は、一人ひとりの事業プランに対するフィードバックやディスカッションを徹底的に行い、実践×検証×結果を短期間に繰り返します。
鈴木氏は、時に厳しく本質を見抜いて指摘し、方向性を指南し、行き詰った時には相談できる経営アドバイザーとなり、一人ひとりの課題や目標を把握し、的確に分析し、進めていく密度の濃い授業スタイルです。

貴方の事業が、貴方の経験が、未だ解決出来ていない社会課題で苦しんでいる人を救う事が出来ます。
そして、ソーシャルインパクトを起こすことが出来ます。
