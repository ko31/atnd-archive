description

Nii Fine Arts では長年、知的財産・法務に従事されておられる弁護士 野中光夫さんをお迎えし、著作権についての勉強会を行っております。第2回となる今回は「フェアユース－作家と利用者のせめぎ合いー」と題し、今までの判例を用いて、権利者である作家と利用者とがどう折り合いをつけているかという点を皆で考えられればと思っています。
著作権法のおさらいもありますので第1回を受講しておられない方も無理無くご参加いただけます。ご興味のある作家・アート関係者の方のご参加、お待ちしております。

野中光夫さん略歴：
　平成　元年　３月　　名古屋大学　卒業
　平成　元年　４月～　ローム株式会社にて，知財・法務関連業務に従事
　平成１９年　３月　　大阪大学大学院高等司法研究科　修了
　平成１９年　９月　　司法試験合格
　平成２０年１２月　　弁護士法人名城法律事務所へ入所　（弁護士登録）
　平成２５年　１月　　同事務所　パートナー
　　　　現在に至る

参加費は￥1,500です。
終了後、懇親会も予定しております。(予算：￥3,000程度)
