　キャンドルナイト @ CANDY STORE ROCK vol.5


　キャンドルのやさしい灯りのもと
　《ココロ》に沁みるアコースティックミュージックと
　《カラダ》にやさしいアロママッサージ&美味しいイタリアンでおもてなし。
　肩肘張らない『ユル』くて『ココチヨイ』キャンドルナイトをご用意します。

　※この夏は4年に一度のワールドカップイヤーということで

　　既に熱戦が繰り広げられておりますが

　　19日はオランダ戦とキャンドルナイトと

　　お忙しい一夜になるため

　　20:30からは暗闇でサッカー観戦っつう

　　ダブルミーニングイベントに

　　グレードアップします。

　　このため、イベントスタートが19:00となりますので

　　ご来場いただく際はご注意ください。

　5回目となる今回のゲストミュージシャンは
　玄人ミュージシャンを唸らせ
　子供を踊らせる
　“東京ローカルホンク”
から
木下弦二

　大人のアコースティックミュージックを奏でる
　りべおか

　アロマセラピスト 宮崎かよ


　イベント全般を通して

　オーガナイザー成田ケンヂによる

　厳選したアコースティックミュージックのDJプレイ。


　キャンドルナイトを愉しんでいただける
　日常の疲れを癒すプログラムをパッケージ。
