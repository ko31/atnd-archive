煎茶部のきっかけ（vol.1時）

東京Ruby会議10 で、 @ginkouno さんと SenchaTouch 使ってる人いませんかね…みたいなお話がきっかけです。

僕は、2年程前、SenchaTouch 1.0 リリース前後あたりに使っていました。
その後、Titanium の登場により、あっ(ﾟдﾟ)！という間に Titanium ユーザが増えたように思います。当時の SenchaTouch ユーザは、数年後にはまた Sencha の流れがくるんじゃないかな〜なんて言っていたものでした。

半年程前、HTML5 が注目度が増す中、Facebook のニュース（“FacebookのザッカーバーグCEO、「HTML5に賭けたのは失敗」　Androidアプリも間もなくネイティブに – ITmedia ニュース”）が話題となったりしましたっけ。

そして、1ヶ月くらい前ですかね、Fastbook (“Fastbook: SenchaによるHTML5が”使える”技術であることの証明”) がリリースされ、SenchaTouch に興味を持った人もいるんじゃないかと思います。

最近ネイティブアプリを作ってみて思うのですが、やっぱり Web アプリ開発者にとって SenchaTouch はけっこうアリな選択肢なんじゃないかな〜と思いつつ、最近の SenchaTouch はどうなってるのかなあと思った次第です。

それと、最近SenchaTouchの本とか出版されてますし、ブーム来てるんじゃないかなあ〜と思う次第です;)
