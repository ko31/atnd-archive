description

多大な時間とコストをかけてつくる会計数値を、決算書作成や税金計算の為だけに活用していませんか？毎月の財務報告が、利益と費用の報告だけに終わっていませんか？

会計は、利益を計算するだけでなく、経営状況を正しく映し出す最も有効な仕組みです。会計数値を正しく活用すれば、将来を予測することも可能です。

今回のセミナーでは、会計数値を経営意思決定に活用するための方法を、具体的な事例を交えてわかりやすくご説明いたします。

お客様の声

  • 損益分析の具体例を示していただいた点が役立った。
  • 固定費と変動費に分けて考える点については、今まで大まかな把握しか持てなかったが、理解度が高まった。
  • 経営分析指標の使い方がよくわかった。
  • 経営分析が必要な優先順位等の知識を得ることができた。
  • 4つの分析の体系的理解ができた。

講師

株式会社OAGコンサルティング 執行役員　深谷 晃広

講師プロフィール

平成13年に太田・細川会計事務所へ入所（現ＯＡＧ税理士法人）

平成21年に㈱ＯＡＧコンサルティングのシニアコンサルタントに就任

企業の「事業計画策定支援、財務分析、会計システム構築支援、金融機関対応支援」といった業務などを中心に行なう。

特に、経営者や財務責任者からは、「会計数値から経営実態を把握するアプローチ方法」に支持を得る。

「中期5ケ年計画立案教室：戦略の日」など
