さわってみよう Firefox OS

ドキュメントを読んでがんばれば、普通の人でもAndroid端末に入れられるようになってきたFirefox OSですが、フロント界隈の人にとってはまだまだ難しいところ。

「さわってみよう Firefox OS」ではFirefox OSの実機を数台準備しますので、それぞれ好きなようにあそんでみます。

普段Webアプリを作っている方、これから作ってみたい方、そしてそれらをFirefox OSの実機にインストールして見てみたい方はぜひ来てください！

概要

Firefox OSの概論から始まって、既存アプリのカスタマイズや、Webアプリの作成など、レベルの合う人どうしでその場でグループを作り、やりたいことをできるように調整します。

＃「ハッカソン」という名前をつけていましたが、ガツガツやるというよりは「ゆる〜くさわってみる」という趣旨のほうを重視しているので、名前を変えました。
「情報共有」という意味で最後に発表はしますが、「何かをつくる」ことが目的ではありません。

対象

デザイナー
UIデザイナー/UIデベロッパー
HTML/CSSがわかる方
JavaScriptがわかる方
（どれかひとつでももちろんOK）

まだまだ開発段階のFirefox OSなので、デザイン的に検討の余地はあるそうです（浅井さん談）。
ホーム画面や設定画面、コール画面、連絡帳など、デフォルトで入っているアプリをカスタマイズできるのは、今はFirefox OSしかありません。デザイン的に見直し、実際に改善できるアプローチを考えてみませんか？
