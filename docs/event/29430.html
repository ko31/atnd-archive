イベント概要

　ITの潮流を大きく変え得るイベントが、米国で4月から5月にかけて相次いで開催されました。「Software Defined Network（SDN）」や「OpenFlow」に関する専門イベント「Open Networking Summit 2012」（ONS 2012）と、サーバーハードウエアのオープンソース化推進に関するカンファレンス「Open Compute Summit」です。

　ONS2012は、今ネットワーク業界で最も関心を集めているSDN/OpenFlowの震源地となっている場です。この分野では、ストレージ、サーバーの仮想化が進む中、最後のネックとなっているネットワークの仮想化を実現すべく、日夜様々なプレーヤーがしのぎを削っています。ONS2012では、米グーグルや米ヤフー、NTTコミュニケーションズ、NEC、シスコ、VMwareなど関連するプレーヤーが一堂に介し、大いに盛り上がりました（ 関連記事 ）。

　Open Compute Summitは、米フェイスブックを中心にサーバーハードウエアのオープンソース化を進める場です。IT業界ではパッケージからオープンソースソフトウエア（OSS）の活用へ舵を切る動きが見えてきています。Open Compute Summitは、そんな動きがハードウエアにまで及びつつあることを示しました。ここでは、フェイスブックが設計したストレージや、幅が21インチという新仕様のサーバーラックなどが新たに公開されたほか、米ヒューレット・パッカード（HP）や米デルなどの大手メーカーがOCP仕様の製品を開発したことなどが明らかにされました（ 関連記事 ）

　これらの動きについて、日経コンピュータと日経コミュニケーションは相次いで現地へ記者を派遣。本誌の最新号を中心に多くの記事を執筆しました。

今回の勉強会では、そんな現地で取材したから分かるナマの情報を一挙公開します。

・グーグルが明らかにした「100％OpenFlow」ネットワークの正体とは？
・SDN/OpenFlowはネットワーク業界を一変するのか？
・台湾ODMが“ベンダー中抜き”を加速？
・ハードウエアにまで至ったオープンソース化の流れはどこまで広がる？
などなど

　SDN/OpenFlowに関心をお持ちの方、オープンソース、クラウドの新たな流れに興味のある方など、広範囲のIT関係者にとって必見の内容です。
