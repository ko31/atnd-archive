イベント内容

マイクロソフト MVP 澤田賢也さんをお招きして、Windows Server 2012 Essentials の勉強会を行ないます。
今回のテーマは、「Anywhere Access機能」です。

Anywhere Access 機能 (リモート Web アクセス、仮想プライベート ネットワーク、 DirectAccess などの総称) を有効にすると、インターネットに接続していればどこからでも、時間やデバイスを問わずにサーバー リソースにアクセスできるようになります。

スマートフォンやタブレットを日常的に使う人々にとっては、魅力的な機能ですね！
サーバーが遠隔地にある場合にも便利な機能ですよね！

なお、Anywhere Access の R2 Preview での新機能も、合わせて解説していただけそうです！

一緒に勉強しましょう。
