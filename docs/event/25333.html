毎週月曜日に池袋で行っている池袋朝活会・・・通称イケカツです。
第2回目もテーマを設けます。

お題は「学ぶ事について」です。

自分はこういうものを学んでいる。
学んだ事によってこういう事がよかった。
これを学ぼうと思う。
こういう方法で学ぶと良い。

等々、「学ぶ事について」いろいろざっくばらんに語り合いたいと思います。

定員は10名までとさせていただきます。

【参加方法】
