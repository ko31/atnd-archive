FiS Project 成果発表会

FiS Projectは千葉工業大学の学生・OBを中心とした技術系サークルです．
当サークルではiPhone・AndroidのスマートフォンアプリやWebサービスの開発を行うことを主な活動としています．

今回はFiS Projectにおける開発の成果や，その他ソフトウェア開発に関する様々なことについて，発表形式で報告します．

参加対象

Webやソフトウェア開発に興味のある学生の皆さんは是非ご参加ください．
発表したいという方は当方までご連絡ください．
千葉工大の学生でなくても歓迎です．

2012年7月22日 日曜日 9:00-12:00
