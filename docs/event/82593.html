こんにちは。ジャイロ経営塾事務局です。
11月8日(火)に「リーダーの育成」をテーマに、少人数マネジメントセミナー「第5回ジャイロ経営塾」を開催します！

講師は、ナイキ・LVMHグループのゲランの日本法人で社長を務めた秋元氏、同じく外資系企業・地方のメーカー等様々な企業の経営トップを務めた田所氏の両氏です。現在お二人は、経営コンサルタントとしても国内外で活躍し、ベンチャー企業をはじめ数々の企業の業績向上・発展に貢献しています。

昨今、経営環境が急激に変化する中、企業の経営・事業運営においてリーダーシップがますます重要になってきています。

ところが、日本の企業にはリーダーシップを発揮できる人材は少なく、リーダーの育成は大きな課題となっています。
どのようにリーダーを選定し・教育していくべきなのか、講師のもとにも、多くの企業の経営層・人事担当者から切実な相談が寄せられています。

このような背景のもと、第5回のジャイロ経営塾では「リーダーの育成」をテーマに検討します。

数多くの企業で優秀なリーダーを育ててきた講師陣とともに、リーダーの育成が難しい理由を紐解き、次世代リーダーをどのように育成するか検討してゆきます。
また、少人数で話せる場だからこそ、普段相談しづらい、あなたの会社が抱える課題やこれからの具体的な取り組みについても議論することも可能です。

これまで講師が企業で実践してきた具体的な実例や、コンサル活動で指導している内容の紹介も交えながら、あなたの組織におけるリーダー育成の筋道を見出しましょう！

＜ジャイロ経営塾とは＞

ジャイロ経営塾とは、ナイキジャパン・LVMHグループ企業をはじめ様々な企業の経営トップを歴任されたお二人である、秋元征紘氏・田所邦雄氏を講師としてお招きし議論を進める、勉強会形式のマネジメントセミナーです。
今回のテーマは、すべての経営者にとって重要な課題である「リーダーの育成」です。
参加者の皆さまと講師のディスカッションに重点を置き、マネジメント上の課題について全員で議論し、結論を見出してゆきます。
どうぞ、奮ってご参加ください！

＜講師の紹介＞

◎秋元征紘 （元 ナイキジャパン代表取締役社長、ワイ・エイ・パートナーズ株式会社代表取締役、株式会社FiNC社外取締役）
日本精工（NSK）のオーストラリア、アメリカ、カナダでの海外事業の経験の後、ケンタッキーフライドチキン、ペプシコーラ、ナイキ、LVMHグループのゲランなど米国・欧州の代表的外資4社の日本法人トップマネージメントを務める。現在、"新たな日本的経営の提案"として「ジャイロ経営」を提唱。ジャイロ調査の基づく戦略的経営コンサルティング、「ＷＡＣＣの原則」に基づく独自のグローバル人材養成研修や講演を始め、研究会・執筆活動など、広範かつ精力的に展開中。

◎田所邦雄（元 御木本製薬代表取締役、TCCコーポレーション株式会社代表取締役、株式会社FiNCアドバイザリーメンバー）
外資系企業、地方のメーカー等の経営トップを務めた後、経営コンサルタントを開業し国内外で活躍、ベンチャー企業を始め数々の企業の業績向上・発展に貢献している。また、企業の中堅社員等、社会人を対象としたウェールズ大学日本校MBAで教授を務める等、実践的な教育で、高い評価を得ている。

＜過去のジャイロ経営塾＞

第1回では、社員のやる気を高め企業（担当組織）の業績を向上させるリーダーの重要な役割を確認しました。
第2回では、リーダーに求められる意識・行動を具体的な事例も交え検討しました。
第3回では、リーダーがどのように組織文化を創っていくべきか、具体的な施策も含め検討しました。
第4回では、ダイバーシティ組織に求められるリーダーシップについて検討しました。
