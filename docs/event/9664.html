どんな勉強会？？

  • 内容
    • プログラミング言語Scalaについて基礎からやっています。いつでも初心者大歓迎！

    • Ust流します。オンラインでも参加OK！
    • 場所は渋谷。セルリアンタワーでやります！
  • 対象者
    • Scalaに興味があるJavaやPHPなどのWebプログラマのみなさん
      • もちろん社会人でも学生でもニートでも構いません。
      • 環境を用意してない（Scalaのインストール未だとか）の人でもOK！ 一から説明します。

    • Scalaを業務に使えるようになる！
    • ＋α（みんなで決めましょう）
  • 備考

    • ノートPC（強く推奨）
    • 電源（バッテリーに自身のある人はいいです）
    • 電源タップ（主催側である程度用意しますが、心配性な方はお持ちください）
    • イーモバイル等（無線LANもご用意しておりますが、もしご自身の接続環境をお持ちであればそちらをご利用ください。）
    • Scala参考資料（コップ本または、その他の書籍等）

前回の宿題！

「Scalaで何を作りたいのか。Scalaに何を期待するのか。」
宿題おさらいの時間で、これについて参加者全員で考え、話し合いたいと思います。
参加者の皆さんは、事前にこの内容について少し考えてきていただけると、良い議論ができるかと思います。

もくもく勉強タイムとは

少人数のテーブルに分かれて、コードを書いたり、資料を調べたりして勉強します。
ひとりで勉強するのは大変ですが、もくもく勉強タイムには以下のようなメリットがあります。

  • 一緒に考える仲間がいます！
  • 詳しい人（講師、またはScala経験者）が近くにいるので、なんでも質問できます！
  • 初心者の人に教えることで、より理解を深めることができます！
