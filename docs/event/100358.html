【10/5開催（無料・五反田）】

欧米豪向けのインバウンドを成功させるSNS運用とは？


近年成長が著しい、欧米豪の英語インバウンド市場。その市場動向と訪日外国人の認知獲得・集客に欠かせないSNS活用をお伝えします！

昨今、インバウンド市場は大きく成長しています。実際に訪日外国人に接する機会が増えたと体感する人は多いのではないでしょうか。その可能性に期待を感じ、事業展開を検討している企業や経営者もいらっしゃることでしょう。
特にインバウンド市場は中国を始めとするアジア諸国を対象とする企業が多いですが、最近では英語圏の欧米豪が注目を集めています。実際に北米、欧州、豪州の訪日外国人は、2018年のいずれの月も前年比増を記録しています（日本政府観光局「統計データ」より）。また、旅行支出額はオーストラリア、イギリス、アメリカ、シンガポールと英語圏の国が上位に入っています（日本経済新聞「インバウンド統計リポート」より）。

そこで、今回のセミナーでは、「訪日コム」運営の株式会社movからインバウンド研究室 室長・田熊氏をお招きし、インバウンド市場の消費動向や、欧米豪英語圏に向けた施策の成功事例・失敗事例を語っていただきます。また弊社・アディッシュからは、訪日外国人の来日以前の旅行情報源として、個人ブログに次いで利用されているSNS（国土交通省 観光庁「訪日外国人の消費動向　平成29年10-12月期 報告書」）の活用について、多言語SNS運用代行サービスの提供で蓄積したノウハウをお伝えします。
