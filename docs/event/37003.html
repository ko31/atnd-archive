■内容

以前にくらべるとだいぶマシになったとはいえ、生涯にわたって仕事を続けたい女性にとって、まだまだ「楽勝！」とは言えない世の中です。
男性との体力差を感じた瞬間や、結婚や出産などのライフイベントを乗り越えて仕事を続ける方法を考えたときに、「フリーランス」という形態を意識したことがある方も多いのではないでしょうか。特にWebの仕事は比較的場所を選ばずに従事できることもあり、環境が大きく変わりがちな女性にとって好都合なことも多いといえます。

本セミナーでは、Web業界にてフリーランサーとして10年近いキャリアを持つ講師が、自身の経験をもとにさまざまな角度からお話します。

基本的には女性に向けた内容ですが、もちろん男性も大歓迎です。
今は制作会社で働いていて将来は独立したい方、すでに独立したけど悩みを抱えている方、フリーランス志望の彼女と結婚を考えている彼氏さんなど、「Web」「フリーランス」というキーワードでピピッときた方は是非ご参加ください。

■講師紹介

千貫 りこ さん

  • KICKS Web

フリーランスのWebクリエイター。

某企業における自社サイトのWebマスターとして4年間、その後転職したWeb制作会社にてHTMLコーディング専任者として4年間勤務した後、2005年に独立。

小～中規模サイトの企画・制作や大規模サイトの(X)HTMLコーディングといった制作業務の他、書籍・雑誌記事の執筆、プライベートセミナー開講なども行う。各制作会社との取引はもちろんのこと、出版、呉服、店舗、作家、NPO法人などクライアントの業種は多岐に渡る。
