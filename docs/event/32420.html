wit preasents, Web directors meeting vol.1

「wit」が主催する、ウェブディレクターのための勉強会！
第1回のテーマは、「ウェブディレクターの職域と職能を考える 〜己を知り、友を知る〜」。

ウェブディレクターの仕事と言えば、「何でも屋」「雑用全般」などとも言われますが、
実際には、個々によってその職域と職能は大きく異なります。
そこのところをきちんと見直し、自覚的となることで、
プロフェッショナルとして、業務の質を高めることを目指します。

具体的には、下記のような分類をベースに、参加者各位それぞれが自身のあり方を考え、
理想的には、これらの情報をインフォグラフィックス化するところまで発展させたいと考えています。

☆職域(1) 業務領域について
プロデュース／プランニング／クリエイティブ／システム／オペレーション／マーケティング

☆職域(2) 取り扱い領域について
例）コーポレート／ブランド／ショップ／EC／ポータル／LP／ウェブアプリ／スマホアプリ　など

☆職能(1) スキルセット
例）デザイン／コーディング／システム開発／アクセス解析／企画プレゼン／仕様設計　など

☆職能(2) 研究テーマ
例）スマホ時代のUI設計について／O2Oビジネスについて　など

▼本勉強会の目的
1. 己の職域と職能に自覚的となることで、責任範囲を明確にし、業務の質を向上させる
2. 学ぶべき職域と職能を明確にすることで、スキルアップを促進する
3. 互いの職域と職能を知ることで、ディレクター同士の適切な業務連携を促進する

▼開催概要
主催者： ウェブディレクターユニット「wit」

開催時間： 18:30〜20:30

▼当日の流れ
1. まずは趣旨を説明「それってどういうことなのかを考える」（30分）
2. そしてみんなで「本当にそれでいいのかを考える」（30分）
3. まとめに「結局それでいいのかを考える」（30分）
4. 最後に「今後どうして行くのかを考える」（15分）
5. おあとの懇親会「なにも考えずに飲み食い談笑する」（帰るまで）

フリーランスなウェブディレクター3名により設立されたワークユニットです。

定期的に勉強会や交流会などを開催していきます。
