目的

RightScale（RS）を更に便利に使うためには、自分たちが日常使いたい、又は使っているserverの設定の内容をServer Template化してRSにもって置くのは非常に重要です。更に、国内では常識的なserver構築手法のServer Templateが誰でも使えるようにpublic公開されていることは、新しくクラウドに取り組む人の良い道しるべになるでしょう。RS Server Template hackathonでは、自分のPCとアカウントでServer Templateを書くことで、RSのServer Templateの理解を深め、日本向けの安全で安定したServer Templateの増加を目指します。

Japan RightScale User Group
