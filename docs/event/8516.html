description

3WIN（スリーウィン）では2010年10月末までの期間限定で「ホームページ担当者向け無料個別相談会」を神戸・元町にて開催いたします。

「無料個別相談会」では、ホームページの運営やリニューアルによくある悩みやご質問をマンツーマン形式でプロの担当者にご相談していただけます。また、様々な業種の方にご参加頂きたいため、期間中は平日、休日問わず「無料個別相談会」を開催しております。

今回の「 無料個別相談会」を企画した理由は、今まで実施したセミナーにおいて、参加者から具体的なケースにおいてのご質問を多く頂いた点、アンケートを行った結果、個別のご相談を希望している方が多くニーズが高い事によります。

開催概要

【場所】神戸市中央区栄町通５丁目２−２PVCビル

【日程】2010年10月31日（日）まで毎日開催

【申込方法】3WIN公式ホームページよりお申し込みください。

【申込URL】 http://www.3winproject.com/

申し込み方法

お問い合わせページ（こちら）より、必要事項をご記入の上、ご用件欄に「ご希望の日」「ご希望の時間」をご記入のうえ、送信ください。

「ご希望の日」「ご希望の時間」につきましては、ご希望にそうことが難しい場合がございますので、複数記載いただきますよう、お願い申し上げます。

送信後、通常、１営業日程度で担当者よりご連絡いたします。
