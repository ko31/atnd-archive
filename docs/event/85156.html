★講演内容

★★2017今年の冬も開催します、フリーランス感謝祭！★★

フリーランスの皆様に感謝の気持ちをお伝えしたく、今回も企画させていただきました！

　★当日は交流会あり！
　★参加者全員に限定プレゼントあり！
　★総額数万円のフリー支援サービス割引あり！

参加費無料ですので、ぜひ気軽に遊びに来てください！！

第1部：ライトニングトーク

今回はキャリア支援・マネープラン・士業紹介に関するサービスを展開されている企業3社を招いて

『フリーランス必見のサービス活用術』をテーマに、ライトニングトークをしていただきます。

『スキルチェンジをしたいが、実績を積む機会が無い』

『将来に必要なお金について知りたいが、何から手をつけたら良いか分からない』
『会計や法律に関するトラブルについて相談したいが、どうしたら良いか分からない』..etc

当日は相談ブースも設けておりますので、 交流会の際に登壇者へ直接質問や相談することも可能です。

フリーランスとしての今後の活動についてお悩みの方、是非イベントに来て・見て・聞いてみてください！

第2部：交流会

フリーランスの皆様同士の交流の場を設けたいと思い、今回も交流会を行います！

ささやかですが、お食事とお飲み物もご用意いたしますので、
お酒やソフトドリンクを片手に情報交換していただき、交流を深めていただけたらと思います！
当日は参加者全員にプレゼントをご用意しておりますのでお楽しみに！

※ 内容は変更になる可能性があります。


★講演者プロフィール

■株式会社Dive into Code 代表取締役　野呂 浩良 氏

MBAエンジニア講師。異業種・異職種への転職を4度経験。
あらゆる時間を計測し、未経験の職務でゼロから短期間に成果をあげる独自の生産性向上手法を確立。
ワークスアプリケーションズの特待生制度「問題解決能力発掘プログラム」の突破経験と
独立起業過程でエンジニア人材の不足を痛感した原体験から、
実務で稼げるエンジニアを輩出するスクール「DIVE INTO CODE」を創業。

◆ サービス概要

■株式会社リロ・フィナンシャル・ソリューションズ 企画営業グループ　阿部　みなみ 氏

弊社は福利厚生アウトソーシング事業のリーディングカンパニーである、リログループのグループ会社です。
独立系ファイナンシャルプランナーや税理士等と提携をしており、
フリーランスの皆様へ知って得するマネー情報を発信するセミナーの企画や、
フリーランスならではのお金や法人成りに関してのお困り事を相談できる窓口サービスをご提供しております。

◆ サービス概要
◆ レバテックフリーランス掲載記事

■株式会社ココペリインキュベート SHARES事業部　執行役員　兼子真人 氏

会社員として働く傍ら起業し、野菜の卸売りやTV番組作成、Tシャツ販売サイトなど様々な事業を行う。
2015年に2度目の起業を行い、今も様々なプロジェクトに参画するパラレルワーカー。
自身が法律や税務に関する悩みを多く抱えていたこともあり、士業のスポット相談サービスである『SHARES』に参画。
当日は実体験や毎日寄せられる相談からフリーランス方々に多い悩みやリスクについてお話しします。
