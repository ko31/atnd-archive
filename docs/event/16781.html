title LiveMomo『酔ったついでに』
catch 百道浜の居酒屋から酒番組UST
description

お酒が好きな皆さん、中洲とはひと味違う金曜日の夜を一緒に楽しく過ごしませんか？福岡のお酒メーカーや販売店、飲食関係の方を数名お呼びして、いろいろなお話を伺います。普段聞けない、あんな話やこんなネタ、業界トークも飛び出すかも。
会場は、天神から20分ちょっと。17日（金）当日は、ドームでホークスの試合もないので、行き帰りが大混雑することもないはずです。わざわざ百道浜まで来る価値は十分ありますよ！
飲めない方も、雰囲気だけでもぜひどうぞ。当日、店長からのスペシャルや、気分次第のプレゼント企画？にも期待してください。突然の番組出演も歓迎です。

出演者： 欧グルマンさま：チーズショップを福岡市内に3店舗。手軽に楽しめるチーズや通向けのこだわりのチーズ、珍しいチーズをお取り扱い。
http://www.au-gourmand.com/
スピリッツさま：「n.e.o（ネオ）プロフェッショナルプレミアムジンジャーエール」や燻製辛子明太子、オリジナル商品などの販売。
http://hakatasugaya.com/
綾杉酒造場さま：創業寛政5年と福岡で200年の伝統を誇る蔵元。天神には「酒のぎゃらりい・綾杉」。山笠の酒も。
http://www.ayasugi.co.jp/
会　場： 「飲中八仙歌（いんちゅうはっせんか）」
地下鉄西新駅・福岡タワーから徒歩12分　西鉄バス「医師会館前」バス停下車徒歩1分
定　員： 10数名程度？（出演者・関係者を除く）
予　約： 必要ありません　当日途中参加OK！ただ、ここで『行くよ！』宣言していただけると嬉しいです。
参加費： 無料（ただし、飲食代は各自でご負担をお願いいたします）
その他： 店内の電源は、配信スタッフ以外利用できません。ご了承ください。
また、テーマがテーマなので、酔った勢いで2次会へ流れていくことは十分予想されます。
ぜひ、最新情報はTwitter（ @livemomo ）のタイムラインにもご注目を！
