電線のないまちづくりセミナー

イベント ユーザー
event_id 50647
title 電線のないまちづくりセミナー
catch 「郊外における電線電柱類の景観への影響と景観向上手法」
description

景観研究のスペシャリストが、電柱電線の景観に与える、
影響をわかりやすく解説。
さらに、それらを撤去して、いかに景観を向上させるかの
手法を大公開！
NPO法人電線のない街づくり支援ネットワークと寒地土木研究所
のコラボ企画に、要注目！

■講師紹介
松田　泰明(まつだ　やすあき)氏
独立行政法人土木研究所寒地土木研究所
地域景観ユニット・総括主任研究員
＜プロフィール＞
国交省北海道開発局にて主に道路インフラ調　
査・計画・設計・施工監理や道路施策を担当。
2004年国交省北海道開発局旭川開発建設部道路調査官、
2006年寒地土木研究所の地域景観ユニット設置に伴い現職。
現在は、「道路景観」をはじめ、「道の駅」や「ドライブ観光」など、
主に良好な景観形成による社会資本の質の向上や観光利活用に関する研究のほか、
研修等の講師や道路事業における景観対策、地域での景観まちづくりなど
について技術支援を行っている。
＜研究テーマ＞
景観分野：「景観の評価手法」、「道路や歩行空間の景観向上手法」、
「道路施設の色彩設計」、「電線電柱類の景観対策」、
「インフラ整備に関わる景観の社会的効果」、「積雪寒冷地の道路緑化」など。
観光利活用：「道の駅の機能向上」、「国際的ロードツーリズムの振興」、
「わかりやすい道路案内誘導手法」など・・。
