こんにちは！　はーしぇんと申します。

C82お疲れ様&「騙」頒布記念ということで、
凋叶棕メインのカラオケオフを開催いたしますので、
興味のある方は以下概要を熟読のうえ、奮ってご参加ください！

※ATNDにはTwitter・mixi・googleアカウントなど、お手持ちのIDでログインできます。
　ログイン後、参加者メッセージ送信のためのメールアカウント設定をお願いします。

【日時】

2012年8月25日（土曜日）
カラオケ … 13：00～18：00
二次会 … 18：30～　（2～3時間程度を予定）

【場所】

カラオケパセラ　秋葉原電気街店
http://www.pasela.co.jp/shop/akiba_denki/index.html

【開催人数】

最小4人～最大8人　（最大人数は予定）

【料金】

カラオケ … 2000円
二次会 … 3～4000円程度を予定

【内容】

◯集合について

集合場所は秋葉原駅周辺となる予定ですが、
詳細な場所、時間、および幹事の目印は参加者の方にのみご連絡致します。

◯カラオケについて

最初に歌う人から、時計回りで順番に歌ってもらうようにします。
始まってから3周する間は、凋叶棕や少女病等の「RD-Sounds曲縛り」でお願いします。
他の人との曲の被りは気にせずに、歌いたい曲を入れてください！
4周目以降は、基本的にどんな曲を歌ってもOKです。

◯二次会について

会場周辺の居酒屋を予定しています。
人数がある程度決まり次第、予約を行ってご連絡を致します。
予算3000～4000円程度の所を探そうと思います。

【諸注意】

・他人に迷惑をかけないよう、常識と良識のある行動をお願いします。
・特に、ナンパや宗教勧誘等の行為は、発見次第ご退出いただきます。
・前日以降のキャンセルは、キャンセル費が発生する可能性があります。
・会費は集合時に集金致しますので、ご準備をお願いします。
・飲み物、食べ物の注文をされる際には、その都度精算をお願いします。
・カラオケオフ中は禁酒です。
・室内禁煙です。煙草はロビーや廊下等の喫煙スペースでお願いします。
・未成年の方は、二次会への参加はできませんので予めご了承ください。

ご参加いただける方は、コメント欄に何か一言、および

・遅刻/早退の時間（する予定の方のみ）
・二次会参加の可否

をご記入のうえ、参加登録してください。

以上、ご不明な点があればお気軽にご質問下さい！
