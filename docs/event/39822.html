■異業種交流会プラスパートナーズとは？

プラスパートナーズの交流会は、ビジネスに熱い情熱をお持ちの方の為に、様々な業種・職種の方とコミュニケーションを深めていただき、有意義な人脈形成や情報交換を行いまして今後のビジネスに役立てていただく事を目的に立ち上げられた交流会です。
異業種交流会プラスパートナーズ

新宿、渋谷、秋葉原などの都内と大阪で開催しております。

■交流会概要

第２１回は、初企画のクリエーター交流会を開催いたします。2団体共同での開催となります。
私、山本もweb、印刷、映像等のデザイナーとして制作業務に関わっておりましたが、会社勤務であっても独立しても、技術面の情報交換や協業のパートナーとしての人脈が本当に有り難く感じておりました。
また、全く異分野のクリエーターからも良い刺激を頂くことも多々ありました。
是非この機会に、様々なクリエーターと交流されてはいかがでしょうか。
（対象参加者：webデザイナー、DTPデザイナー、映像クリエーター、フォトグラファー、イラストレーター、アパレルデザイナー、スタイリスト、ヘアメイク、ネイリスト、ダンサー、俳優、作家など、他にも少しでもクリエィティブの要素をお持ちのお仕事をされていらっしゃる方でしたら大歓迎です。クリエーターへの仕事発注をご検討されている企業の方も歓迎です！）

今回は共同開催での交流会となり、参加人数も40～50名程となりそうです。
現在、イラストレーター、webデザイナー、グラフィックデザイナー、フォトグラファー、映像・CG制作、マーケティング会社、出版社、印刷会社、SEO・アプリ開発、俳優など多種多様な方が参加予定です！  是非、有意義な人脈を築いていただければと思います。

■参加ルール

プラスパートナーズでは皆様に安全に交流会にご参加いただけるように以下のルールを定めております

  • 参加者様同士安全にご交流いただくという趣旨に反する以下の業種の方の参加を固くお断りしております。
    万一業種を偽って参加した場合には、発覚次第即刻退場していただきます。その際、参加費は返却いたしません。
    1. ネットワークビジネス（その他類似ケースも含みます）
    2. マルチ商法やその類
    3. 先物取引
    4. 能力開発ビジネス
    5. 宗教・政治の勧誘
    6. 生保・損保業の方（主催者の紹介での参加1～2名は除く）
      ※副業として営業されている方も含みます。
  • 強引な勧誘、同意を得ずに説明開始、しつこい営業電話やアポ取り、その他相手に迷惑のかかる行為全般を禁止します。
    社会人としてのマナーにご配慮願います。
    交流会の最中に強引な勧誘や迷惑行為を行った方は即刻退場していただきます（返金なし）。
  • 名刺は、お名前、ご住所、電話番号等ご連絡先がわかるものをご用意ください。
    （手書きの名刺はご遠慮ください）名刺をお持ち頂けない方は、ご参加できません。
