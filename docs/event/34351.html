内容

今年10月末にアメリカ ジョージア州で初めて開催された WordPress コミュニティサミット へ参加した時の様子を、三好( @takayukister )さんに30分くらいお話してもらいます。

残りの時間で、他にWordPressに関して話したい方がいらっしゃればその時間を設けますのでお知らせください。

その他の時間は、飲んだり食べたりしながらWordPressに関するご質問等に三好さんや他の詳しい方や私がそれに答えるようなざっくばらんというかワイワイとした感じでやりたいと思います。

参加費と参加対象

参加費はドリンク飲み放題、軽食付き2500円です。参加対象はWordPressに興味を持っている方であればどなたでも。

11月28日、20時（午後8時）〜22時（午後10時）
