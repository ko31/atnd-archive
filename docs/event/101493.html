description

ECビジネスを立ち上げた経営者が経験談を交えながら、どのように起業、創業したのかをお話しします。起業やビジネス立ち上げの際に「何をしていいかわからない」「どうやったらいいの」という漠然とした不安や疑問を持っている方に、コツをつかんでもらい、「自分でもできそう」「まずはやってみよう」というモチベーションを持っていただき、一歩を踏み出していただくためのセミナーです。

海外とのジョイントベンチャーでECビジネスを立ち上げた株式会社ディージーワン、そして、ゲーム事業をはじめとしメディア、広告関連ビジネスを抱えるグリー株式会社グループのそれぞれの経営者が体験談も踏まえながら、起業や創業に必要な心構え、取り組むポイント、コツをお話します。他では聞けない経験談も飛び出すかもしれません！起業・創業、社内起業を目指している方も必見のセミナーです。
