LOCAL PHP部 勉強会について

「LOCAL PHP部 勉強会」は、隔月で札幌で行われている、PHPを主とした勉強会です。
「LOCAL PHP部」という名称ですが、勉強会に参加するのに特に条件はなく、どなたでも参加することが可能です。
LOCAL PHP部についての詳細は、以下をご覧下さい。

  • LOCAL PHP部公式サイト： http://www.local.or.jp/clubs/php
  • LOCAL PHP部 Googleグループ： https://groups.google.com/group/local-php?hl=ja
