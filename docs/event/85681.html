今年のCampまで残り３週間を切りました。
おかげさまで、申込みベースでは既に２０名以上のお申し込みを頂いています。
残り数名の参加者の募集をしております。
実在する企業の情報をもとに、ケーススタディをしながらビジネススキルを鍛える機会は
滅多に無いと思いますので、奮ってご参加ください。
-——————————-
■FRI Campとは
-——————————-
FRI Campは、合宿形式でグループワークを行い、社会人レベルの実力を身につける場。
実践的なビジネススキルの向上を目指すモチベーションの高い学生を対象に、過去12年の歴史の中で26回に渡り開催。
参加者からゴールドマンサックス、マッキンゼー、リクルート、Googleなどの一流企業に多数の内定者を送り出すと共に、世界でも評価された社会起業家やベンチャー起業家も輩出してきました。
Google、外資コンサル、銀行、メーカー等で高い実績を上げる社会人がスタッフを務めます。

■FRI Campの特徴
-——————————-
・実企業を元にしたケースに取り組む：
　昨年は、エイチ・アイ・エス、すかいらーくの二題を出題

・大学や年次も違うチームで、課題をまとめるリーダーシップを訓練：
　昨年は、北は北海道、南は九州地方から、大学1年生～大学院生まで幅広い層が参加

・現役社会人が3日間付きっきりで、ワークをアドバイス：
　ビジネスプランコンテストに初参加でも心配不要。豊富なセミナーと、いつでもメンターに相談できる環境を用意

・Campを通じて見えた課題を元に、各個人にフィードバックシートを送付 ：
　就職活動だけでなく、人生にも役立つ、客観的かつ高レベルな分析を実施

・Camp終了後も、社会人スタッフと参加者限定の定期的な交流会を実施 ：
　一緒に課題に立ち向かったからこそ、終わってからも長く付き合える、かけがえのない仲間との出会い
-——————————-
■日時 / 会場
-——————————-
2017年2月24日(金)～2月26日(日)（2泊3日の合宿形式）
代々木 国立オリンピック記念青少年総合センター

■応募概要
-——————————-
【対象】大学生、大学院生（分野･学年は問ず、短大生･専門学校生含む）、および若手社会人（入社1年目程度）
【定員】30名（予定）※先着順となります
【参加費】20,000円（食費・宿泊費込）

■内容
-——————————-
チームビルディング(リーダー選択) / プロジェクトマネジメントセミナー / ビジネスプランニング グループワーク / 論理思考、プレゼンセミナー /社会人とのナイトセッション(就職・起業相談など可) / 最終報告プレゼンテーション / 社会人から and 参加者同士のフィードバック　

社会人とのナイトセッションでは、模擬面接・キャリアアドバイスも実施します(希望者のみ)。
進路に悩んでいる方、面接を早めに経験したい方、厳しい面接を控えている方、はぜひ活用してください。

※Camp終了後に、3日間のグループワークに対するフィードバックシートを各参加者にお送りします
※Camp終了後に、社会人と参加者による懇親の場をご用意しています
