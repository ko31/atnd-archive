概要

現在、書籍「基礎から学ぶ facebookアプリ開発」をもとに勉強会を行っております。
http://www.amazon.co.jp/dp/4863540949/

2012年3月21日（水）に第3回目 が行われますが、その前に第2回目までの復習を行います。
前回までの参加者でわからなかったことがあった方や、未参加者だけど興味を持っていただいた方向けのイベントです。
コーヒーを飲みながら、ゆるりとお話しましょう。

途中参加を検討される方へ

現在輪読中の本は、PHPでの開発経験がある方にとっては、比較的やさしめの内容です。
準備にさほど時間は要さないと思いますので、途中からの参加でも全く問題ないと思います。

Facebook API は、もちろんのこと、YouTube, Ustream, Google, Amazon 等の API 群、
CakePHP や、 enchant.js を使ったシューティングゲームアプリ等、
色々なものに触れることができる内容となっておりますので、興味がある方は是非ご参加ください。

対象者

  • facebookアプリの開発に興味のある方
  • 勉強会の内容を知りたい等、とりあえず話だけでも聞いてみたい方
  • 前回までの内容でわからなかったこと、質問したいことがある方
  • 技術の話をできる仲間・環境を渇望している方

ウェブプログラマ向けの内容となっておりますが、お話だけですので、どのような立場の方でもOKです！

持ってくるもの・準備してくるもの

手ぶらでお越しいただいて問題ありません。
