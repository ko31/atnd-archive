description

【習得できる知識】
レンズの光学設計で必要な知識　　：　近軸理論、3次収差、光線収差、最適化方法、MTF評価

【受講対象】
現在レンズ開発を行っている方もしくはこれから始められる方。
また、大学などで光学に関係した研究、勉強されている方でレンズの設計に興味のある方

【講座のポイント】
　収差やＭＴＦの基礎的な話からはじめ、レンズの光学設計の手順に沿って、非球面を含むズームレンズを例に３次収差補正、光線収差最適化、ＭＴＦ評価、偏芯感度と一連の設計作業をご紹介します。
　受講者がレンズの光学設計での必要な収差補正やMTF評価判断が習得できることを最終目的とします。
