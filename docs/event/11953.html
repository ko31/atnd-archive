description

bookMentoring（ブックメンタリング）とは、
「まちの読書家による読書ガイド」 というコンセプトで
展開する、まったく新しいタイプの読書会です。
 
もっと本を読んでみたいけど、何を読んでいいかわからない。
いつもと違う本に手を出してみても、いまいちおもしろさが感じられない。
そんな悩みに、 まちの読書家(＝メンター) がお答えします！

専門家の解説ではなく、自分と同じ一般の社会人であるメンターが
その本のどこにおもしろみを感じているのか、聞いてみませんか。

当日は、買ったけど挫折して「積読(つんどく)」になった本を
持ち寄り、読書家にアドバイスをもらう機会も用意しています。

もしあなたが今の読書になんとなく物足りなさを感じているなら、
読書家に学ぶ90分はきっと刺激的な時間になるはず。
どうぞお気軽にご参加ください。
