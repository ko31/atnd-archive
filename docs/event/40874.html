2013年9月23日（月･祝）デザートストーム川越にてサバイバルゲームを行います。

どうも、主催こしがやです(‘ω’)

川越のデザートストームを借り切ってゲームしたいと思います。
広いフィールドなので皆さんで遊べたらなと思います。

基本的に身内と初心者メインのぬる～いゲーム会です。
一応、主催の面識のある方メインの身内向けのゲームとなっておりますゆえ、ご友人を誘われてご参加なされるようでしたら、集まりの趣旨を理解してご参加いただけますようお願いいたします。

■参加募集締め切り：２０１３年９月１０日

■場所：埼玉県川越市　デザートストーム川越

■定員：30名　

■ATNDに登録されない（できない）ゲスト参加の方をお連れになられる場合は、
お手数ですがコメント欄に人数とその方のお名前（HNで可）をご記入頂くようお願いします。

■必要な物、

参加費：4,000円（参加者10名以上の場合。参加者が10人以下なら使用料の40,000円を頭割り。20名以上なら団体割引で一人3,500円）
※デザートストーム川越会員は会員証提示で500円OFF

エアガン：電動・ガス・エアーコッキング式のエアソフトガン
※持っていない場合フィールドでレンタルあります。
（レンタル料金は こちら ）
昼食：お弁当の注文が可能。一食300円

※お弁当とレンタル装備ご希望の方は募集締め切りまでに
　コメント欄に表明してください。

使用BB弾：６ｍｍ完全バイオＢＢ弾。
　
装備：サバイバルゲーム用ゴーグル（シューティンググラス不可）、顔面を保護するマスク等推奨。

　他に、怪我を防止するのにグローブ手袋、ニーパッド、エルボーパッドなどが役に立つので、
　ご用意されると宜しいかと思います。

　フィールドは起伏に富んだ場所や砂利道等があり非常に滑りやすい場所も存在します。
　滑りにくい厚底の靴を推奨。

　任意ですがお名前（HN）が判るような名札をご用意装着いただければ幸いです。

■レギュレーション

パワー制限は銃刀法に定める規制値以下で、

エアーガンは、0.20g BB弾　初速98m未満
※0.25g弾　初速88m未満
※減速アダプターの使用禁止
※レーザーサイト使用禁止

フリーズコールタッチ、ナイフアタック不可（痛くなさそうな末端を撃って下さい）。
　基本的にブラインドファイア（遮蔽物より銃のみ出しての発砲)禁止。
　外部ソースの使用禁止。

ゲーム全般的な縛り、弾数制限などは行わない方向ですが、セミオンリー戦やハンドガン戦などゲーム内容によっての制限はあります。
初心者の方でも楽しめるような内容をで進めていきたいと思います。

■諸注意

　買い出し等でフィールド外に出られる際は装備、BDU等着用のまま外出されないようお願
　い致します。

■ちょっとしたお願い
諸事情で参加を伏せたい人も居ると思いますので、終了後ですが
個人単位で「今日サバゲして来た」と言うのは自由ですが、
誰が参加していたかに関しては『触れない』方向でお願いします。
TwitterやMixi等で「○○さんお疲れ様でした」も出来るだけ空気読んで頂けると助かる人も多いと思います。
（自分が参加していた事を表明する事に問題ない人同士のやり取りについては特に問題ありません）
交流等は本ページコメント欄をお使いください

フィールドまでのアクセスは こちらのほうを参照

※なお、アクセスの写真中にある黄色いゲートの箇所が非常に狭いので、
ハンヴィー等の車幅の大きい車は注意ですｗ

■雨天中止に関して
天気予報等により雨天が予想される場合、
前日の23:00までにこちらのページにて中止かどうかの発表をいたします。
