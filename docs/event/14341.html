WEEKEND WORKSHOP vol.15は「竹場」を開催します。「竹場」とは大橋重臣氏が展開するワークショップで参加者は竹のパーツをつかって自由に構造物を作ることができます。このワークショップは2010年３月に行われたBEPPU PROJECT2010 で発表されて以降、定期的に開催されています。
参加費は無料となっておりますので、公園に遊びに出かけるようにお気軽にお立ち寄りください。

※WEEKEND WORKSHOPとは
毎月第一・第三日曜日にplatform01にて開催されるクリエーター (アーティスト、デザイナーなど)によるワークショップです。主に清島アパートに居住、または制作活動の拠点を持っているクリエーターや、BEPPU PROJECTが招待したアーティストなどにより企画されます。

企画・運営：NPO法人 BEPPU PROJECT
