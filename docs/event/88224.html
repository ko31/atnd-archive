自社会員向けサービス、情報配信、店舗集客のヒントに

スマートCRMプラットフォーム『betrend』は、顧客管理データベースを中心として、“行動履歴管理”“分析・抽出”“情報配信”といったカテゴリの機能が豊富に揃っているCRMツールです。スマートフォンを活用し、企業からお客様へ、継続的かつ効果的なコミュニケーションを実現します。スマートフォンアプリ・LINE@・ハガキDM・SMS・カスタムIVR・メールなど、多様なコミュニケーション手段に対応しています。

本セミナーは、SNSと顧客管理の使い分けとは、『betrend』で何ができるのか、スマートCRMとは何かなど、『betrend』をご理解いただき、いくつか企業アプリの販促事例もご紹介させていただくことで、スマートフォンを活用した効果的な販促施策をイメージしていただける内容となっています。

セミナー概要

  • デジタルマーケティングにおける、SNSと自社会員の位置付け
  • 最新の『betrend』でできること
  • スマートCRMとは？
  • 企業アプリ事例のご紹介　

こんな方におすすめ

  • スマートフォンを活用した会員サービスを実施したい
  • お客様の行動履歴（来店状況・購買・ポイントなど）を販促に活かしたい…など
