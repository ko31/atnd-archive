■内容：
Apache CloudStackはソースコードだけではなく、ドキュメントもコミュニティによるオープンな体制で作られています。今回は皆で集まってApache CloudStackのドキュメント翻訳を行います。
https://www.transifex.com/projects/p/ACS_DOCS/

はじめて翻訳に参加する人のために、Apache CloudStackのドキュメント翻訳環境の使い方や、翻訳されたドキュメントのビルド方法について簡単な解説を行った後、実際に皆で黙々と翻訳を行います。

■注意事項
今回は座学の勉強会ではなくハンズオンで翻訳を行います。以下の点にご注意ください。
・対象：
　CloudStackの基礎知識があり、少しでも環境を触ったことがある方
・持ち物：
　・翻訳作業用にノートPCを必ずご持参ください。
　（電源コンセント、WiFi環境は会場にあります）
　・翻訳されたドキュメントをビルドするには、Linux環境が必要です。
　　あらかじめ各自でご用意いただいてお越しください。
　　VirtualBoxやVMware Player上の仮想マシンでも構いません。
　（翻訳用の環境構築は CentOS 6.X を用いて、当日に解説します）
