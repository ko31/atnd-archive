description

経営者の健康、従業員の健康、そして従業員の家族の健康。
どうしたら健康行動を実践できるのかに重きを置いた教育セミナーです。
健康に係る「正しいこと」を生活に係る「いいこと」に変えるアプローチに
ついて、全国に多くのファンをもつ二人、メディカルドクターの石黒伸氏と
マーケティング専門家の西根英一氏が解説します。

①「朝と昼、何を食べたら集中力を発揮できるか？」50分
何のために健康になるのですか？健康になることに目的や意味を持たせないと
健康行動は継続できない。

講師：医療法人アクア　アクアメディカルクリニック院長　石黒伸　氏（医師）

＊HARU JUiCE　監修医
＊専門は高齢者・認知症・神経難病患者の在宅医療と栄養（オーソモレキュラー）療法。
＊近著に「告白します。僕は多くの認知症患者を殺しました。-間違いだらけの
日本の認知症医療-」

②「社員の検診受診率を上げるための方法」50分
「価値を伝える」から「価値が伝わる」へ。健康行動にマーケティングを導入する。

講師：株式会社ヘルスケア・ビジネスナレッジ　代表取締役社長　西根英一　氏

＊事業構想大学院大学事業構想研究所客員教授、千葉商科大学特命教授。
＊専門はヘルスケアマーケティング、市場調査法。
＊近著に「生活者のニーズから発想する健康・美容ビジネス-マーケティングの基本-」
