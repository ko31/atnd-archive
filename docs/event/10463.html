尼崎市内の青少年ボランティアが企画運営するキャンドルナイトイベントです。
2200個のキャンドルを使ったキャンドルアートは尼崎市内で行われるものとしては過去最大級。
世界に一つだけのオリジナルキャンドルが作れる体験コーナー、ゴスペルやクラリネットの生演奏、
世界中のホットドリンクが飲める大学生企画によるカフェも出店しています。ぜひ、ご参加ください。


■イベントテーマ『人と人のつながり』
高度情報社会と呼ばれる現代、携帯やインターネットなどの登場で情報のやりとりは簡単かつ便利に
なりましたが「人と人が直接会って話をするコミュニケーションの大事さ」を考えてほしい…
その思いから、電気と比べると暗くて使い難い「キャンドル」の明かりを「人と人のコミュニケーション」に例え
ゆれる炎の温かみから、心と心の触れ合いに通じるものを感じてもらおうと企画しました。


【尼崎市　青少年センターとは】
青少年の健全な育成と福祉の増進を図るため、各種の事業を積極的に推進する施設です。
尼崎市内在住・在勤の青少年とその団体・グループ及び青少年団体の指導者が利用できる
会議室、研修室、音楽室、体育館があります。
