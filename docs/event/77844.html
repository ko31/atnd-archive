大阪初開催！AWS（Amazon Web Services）が気になる方必見です！

本セミナーでは、

・AWSの構築をおこなったことがなく、問題なく稼動するか心配…
・AWSの移行や運用方法について相談したい…
・DBの運用を止めずに移行したい
・簡単にオンプレ環境の仮想マシンを移行したい

などなど、AWSについて興味がある方や運用・移行方法を検討中に方にAWSの魅力を余すことなくお届けいたします！

第1部では、関西地域で圧倒的な導入実績とノウハウを持つ、AWSアドバンスドコンサルティングパートナーの株式会社ターン・アンド・フロンティア様をお招きし、AWS移行や最適な運用方法などを事例を交えてご講演いただきます。

第2部では、仮想環境からAWS（クラウド）へのかんたんなVM移行.仮想環境間(VMware/Hyper-V)でのリアルタイムなレプリケーションを実現する”Zerto”についてZerto社カントリーマネジャー松澤氏をお招きしご講演いただきます。

第3部では、株式会社クライムから"DBMoto"を使った簡単に出来るオンプレDBからAmazon RDSへ移行・連携の手法をご紹介します。
