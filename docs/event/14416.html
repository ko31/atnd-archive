齋藤正和　プロフィール

1976年生まれ。
情報科学芸術大学院大学卒業。
映像における撮影者のまなざしを意識し，カメラを被写体を撮影する機械ではなく，撮影者のまなざしを記録する機械と捉え，撮影者のまなざしが映りこんだヴィデオ作品 を制作。
また、プログラムによる自動編集などデジタル技術によって映像を扱う可能性を模索しながら映像作品を制作を行い、 “Fluxus 2010（2010）”, “next : New Generation of Media Artists（2004）” など国内外の映画祭, 展覧会にて発表。
最近は、日常の”スケッチ”をベースとした数分の短い映像群からなる”休日映画”をweb上で展開。 近年は複数台の大型ディスプレイを用いたビデオインスタレーション「いつでも どこへでも 巡業」 シリーズを発表。
