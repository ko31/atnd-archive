★講演内容

2月28日(木)に名古屋駅付近にて、
日本マイクロソフト社とヒカラボがコラボして
女性のキャリアを応援するための
『Women Think Next-Nagoya 2019 #Powered by ヒカ☆ラボ』という
イベントを開催します。

こちらは国連が3月8日に定めている「国際女性デー」に紐づけたイベントとなります。

本イベントでは多様なキャリアを歩んでいる
日本マイクロソフト社員の方々にご登壇いただき、
女性のキャリア形成に関してお話をいただきます。

応募登録は こちらのイベントページからお願い致します。
https://sway.office.com/NLGRABDAIAzKKI60


※ 内容は変更になる可能性があります。
