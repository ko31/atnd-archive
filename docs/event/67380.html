◆　(株)ジークス (http://www.zyyx.jp/)：
　台湾に本拠を置き米サンフランシスコにオフィスを構えるbOMDIC Inc.と提携し、2015年6月からデジタルヘルス用ウエラブルシステム「スタミナメーター：GoMore」を日本で発売した。GoMoreは、乳酸値の予測機能付き心拍計とiPhoneアプリを組み合わせた、アスリートのためのスタミナメーター。特許を取得した独自アルゴリズムにより、心電図を読み取ることで、アスリートのスタミナを数値化することに成功。血中の乳酸値を予測して、疲労度を燃料メーターのように、パーセンテージでリアルタイム表示する。計測データはスマホのアプリと同期して、現在のスタミナの状態や過去のトレーニングの状況を把握でき、トレーニングの最適化を図ることが可能。
　当日は、福井県鯖江市に本拠を置くアイディッシュ(株)が商品化し、ジークスがそのソフトウエア開発を手がけたインカートクックシステム(ICS)についても説明する予定。これは、専用の食器およびトレイと専用IHフードカートを組み合わせたレシピソフトと加熱プログラムをパッケージした食事提供システム。炊きたてのご飯とアツアツ料理を提供する。トレイに食材を盛り付けた主食・主菜・汁物の食器をのせ、トレイをIHフードカートに差し込めば、生食材から自動的に3点同時に自動加熱調理ができる。多人数同時に提供できるシステムとともに、一人用についても開発中。
　＜プレゼンター＞ 　　　　　 　代表取締役社長　　　　　 渡辺　浩　氏

◆ NTT東日本(株) (http://www.ntt-east.co.jp/):
　日本最大手の電気通信事業者。正式名称は東日本電信電話(株)。エンタープライズ分野などにおけるセキュアなネットワーク経由のパブリッククラ ウドアクセスに対する利用ニーズに応えるため、Amazon Web Services(AWS)の日本法人であるアマゾン データサービス ジャパン(株)が提供する「AWS専用線アクセス体験ラボ」に、｢フレッツ・キャスト｣をダイレクトに接続した検証環境の提供を開始。この体験ラボでは、 オンプレミス環境と AWS 東京リージョンを 1 Gbps の専用線で接続したシステム構築の検証に必要な環境を提供。この環境により、ユーザー企業は接続検証の準備に費用や時間、手間などを負担することなく、ダ イレクト接続時のシステム動作やネットワーク機器設定などを、｢フレッツ光｣を利用している拠点から検証することができる。
　フレッツ・キャストとは、｢フレッツ 光ネクスト｣｢フレッツ光ライト｣を契約している顧客へインターネットを経由しないでデータを届けるサービス。SNI( application Server Network Interface)を利用し、IPv6プロトコルを使ったユニキャスト/マルチキャスト通信が可能。マルチキャスト通信では、送信元から送られた1つの データをフレッツ網の中にあるネットワーク機器が複製し、全ての宛先に送信するため、多くの宛先に対してデータを同時配信できる。VODや映像・音声の同報配信に最適。
　＜プレゼンター＞　　　ビジネス開発本部　第二部門
　　　　　　　　　　　　　　　映像サービス担当部長　　　高橋　立典　氏
　
◆　(株)ゲオネットワークス (http://geo-networks.co.jp/)):
　愛知県名古屋市中区に本社を置く、レンタルビデオショップ・古着ショップなどを全国に展開するゲオグループのネット系中核会社。2014年10月1日に稼働した。2015年3月からスマートフォン向けの恋活・婚活アプリ『kano kare（カノカレ）』をリリース。全国の約1300店舗のゲオのショップを拠点として告知することで、都市部だけでなく、地方の男女にも積極的に浸透を図っている。業界初の携帯番号による知り合いブロック機能や、24時間365日の厳重な監視体制など、安心・安全に利用できる機能や体制を備え、会員登録から相手探し(検索やマッチング)までは無料で利用できる。メッセージの交換(要年齢確認)や便利機能の利用については、男性のみ有料(期間定額制)となっている。24時間365日の厳重な監視体制を採り、GEO IDの利用やゲオアプリの最大限の活用により、会員数を急増させている。
　日本の未婚率は年々増加し続けており、婚活ビジネス市場がクローズアップされている。この市場動向とともに、『kano kare』の最新状況について説明する予定。併せて、映像・エンタテインメント分野において、ゲオグループが推進するオムニチャンネルリテイリング戦略についても解説する予定。　　
　＜プレゼンター＞ 　　　ネットワーク営業部　　部長　　　西山 真経　氏

◆　ソニー企業(株) (http://www.sonykigyo.jp/ )：
　ソニービル運営、イベントプロモーションのほか、施設プロデュースを実施するプロパティマネジメントを手掛ける。また、外国人観光案内所やデジタルコンテンツ開発などの新規事業開発にも取り組む。その一環として、内閣府のふるさと名品開発促進のモデル事業にも推挙された“舞台めぐりサービス”を提供している。これは、アニメ作品の舞台となった場所を訪ねる「聖地巡礼」に着目したプロジェクト。コンテンツとその舞台となっている地域を結びつけ、世界最大のコンテンツツーリズム・プラットフォーム構築を目指す。聖地巡礼を楽しくできるスマートフォンアプリ『舞台めぐり』も提供。このアプリでは、作品の舞台となった場所までのルートの確認、ARキャラクターとの写真撮影のほか、特定スポットでは作品声優によるオリジナルボイスも聴くことができ、巡礼の思い出をアルバムに残すこともできる。
　アニメ作品との公式タイアップも増えており、14タイトルがリリースされ、今後4タイトルが追加予定。リアルイベントとの連動も、埼玉県聖地横断ラリー、大洗あんこう祭をはじめ盛んになっている。当日は、舞台めぐりプロジェクトの全貌と今後の方向性についてプレゼンテーションする予定。
＜プレゼンター＞　事業開発室　コンテンツツーリズム課　「舞台めぐり」チーム
