description

趣旨：

オープンソースソフトウェアのCMS(コンテンツ管理システム)DrupalやWordpressを使っている、もしくはこれから検討しているＮＰＯ法人や非営利活動をしている団体さん向けに、無料のDrupal、Wordpressによるホームページ作成勉強会を毎月2回、開催しております。

ホームページやDrupal、Wordpressの使い方などでお困りの方、サポートいたします。

内容：

無料です、Drupal、Wordpressの使い方やホームページの作成、構築運営でお困りのことをサポートいたします。

介護福祉・地域医療などの事業者、NPOやボランティア団体向けに、ブログ感覚で更新ができるホームページをホスティングしており、その操作の勉強会としても利用していただいております。

また、パソコンや周辺器機のハードウェアトラブルも対応致します。

参加希望の方は、このATNDサイトから申し込み、当日、会場へお越し下さい。

同時開催：

パソコン相談・勉強会も開始します。　無料です、どなたでも、パソコンやインターネット、ホームページ、facebook、Twitterなどのソーシャルネットワークおよびハードウェア故障などでお困りのことをサポートいたします。
