ここ数ヶ月、マイクロサービス、サーバーレスといった技術が大変な盛り上がりを見せています。サーバレスアーキテクチャを一言で表現すると、「サーバを自前で用意せず、マネージドサービスを活用してシステムを構築する」ということです。マネージドサービスとは、AWS API Gateway・AWS Lambdaを代表とする、サーバーレスで、マイクロサービスを構築するための主要コンポーネントのことです。これによって、モバイルやウェブ、IoT アプリケーション用のスケーラブルなマイクロサービスをより簡単に構築することができます。しかし、現状、事例といっても、「AWS Lambdaを使ってみた」程度のものがほとんどで、マイクロサービスを組み合わせたトータルなシステムをサーバーレスで構築した事例までは、ほとんど公開されていません。

株式会社スタイルズでは、サーバーレス技術によって、 Trackrr.ioという、いま注目のsoracom Airを中心としたIoT機器を活用した誰でも使えるIoT/GPSプラットフォームを開発して、アルファバージョンとして公開しました。今回は、サーバーレス技術に興味のある技術者の皆さんのために、構築事例として Trackrr.ioをご紹介したいと思います。

Trackrr.ioサービスサイト

https://trackrr.io

セミナー内容

セミナーの中心となるのは、サービスとしての Trackrr.io の宣伝ではなく、どのようにして Trackrr.io を構築したか、その個々の実装等の解説になります。
