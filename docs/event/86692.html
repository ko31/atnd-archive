description

たった60日で1億2,900万円もの売上アップと同等の効果が出た嘘のような手法です。

すぐに効果が出る取り組みをしたい経営者は絶対に参加すべき内容です。
ホテル・旅館の経営者、コンサルタントや士業は必見のセミナーです。
あるホテルの総務担当者が社内保管書類をコピーしただけで、たった60日で1億2,900万円もの売上アップと同等の効果が出た嘘のような手法です。
このセミナーでは単に知識やノウハウをお伝えするだけでなく、増益源発掘につながる手法を手に入れていただきます。

【限定特典】 売上upよりカンタン！　
『増益原資発掘のツボチェックシート』
　無料プレゼント付き

売上を2倍！3倍！と爆発的にアップさせるよりも、はるかにカンタンに、はるかに早く、その上ほとんどお金をかけずに取り組める手法で眠っている増益原資を掘り起こします。

あなたは次のようなことで悩んでいませんか・・・
☑「固定資産税、光熱費、社会保険料、消費税など固定費が重く、それらを払うために働いているような気がしている」
☑「インバウンドへの対応をしたいが人件費や広告費をしっかり捻出できない」
☑「稼働率が上がらず、ブライダル、宴会、会議、飲食部門なども思うように伸ばせていない」
☑「ネット予約サイトからの予約が増え過ぎて利益率が下がる一方」
☑「古い経営体質から抜けられずもどかしい。」
☑「安心して子供に後を継がせられない」
この中に、一つでも該当するものがあれば、このセミナーがあなたのお役に立ちます。

あなたに質問です。

「なぜ、売上を上げるだけではお金が残らないのですか？」

あなたは、この質問に、すぐに答えられますか？

もしも、この質問に明確に答えられないのであれば、
それが、資金が増えない原因かもしれません。

日本の会社は、お金が増えない仕組みの中にいます。
さらに、日本の社長は3代相続するとお金が残らないとも言われています。

そのせいか、
『売上が増えれば増えるほど苦労も増える』
という経営者は数多く存在します。

売上も上がり、安定的に繁栄していくためには、
勝ちパターンとも言うべき鉄則があります。

この鉄則によって、

　・すぐに取り組めて、大幅な売上アップに匹敵する増益ができる
　・金融機関との関係が良好になり資金調達がスムーズになる
　・時代に則した設備投資が行えるようになり稼働率が安定する
　・福利厚生制度が充実しスタッフのやる気がアップする
　・身内に安心して引き継がすことができる

などの効果が期待できます。

本セミナーは、
資金をかけることなく、短期間で、売上の大幅アップにも匹敵する効果が出た、『増益原資発掘術』をお伝えする日本で唯一のセミナーです。

こんな方に効果があります。

・ホテルや旅館の経営者
・ホテルや旅館をクライアントに持つ士業やコンサルタント
・ホテルや旅館を顧客に持つ設備業者や工事業者
・多額の固定費がかかっている事業の経営者
・クライアントの固定費が高額な士業やコンサルタント

こんな業種に向いています。

・ホテル・旅館業
・売上高５０億くらいまでの建設業、製造業、運送業、飲食業、卸売業、小売業
・学校、病院
・分譲マンション管理組合、ビルオーナー
