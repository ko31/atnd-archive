京急ガッタンゴットンしてまぐろ食いに行くぞおう

※みさきまぐろきっぷとは？
京急線三崎口までの往復＋バス代＋まぐろの食事＋レジャー施設利用料がセットになって、
品川を起点とした場合、3,060円とお得な旅券パックです。

■集合
品川駅高輪口 10:00頃集合

■切符の買い方
普通に券売機で買えるみたい。詳しくはこちら。

■行程
京急ｶﾞｯﾀﾝｺﾞｯﾄﾝして終点の三崎口まで行きます。
※途中、金沢八景⇔新逗子、堀ノ内⇔浦賀の枝線乗車予定。

昼食は三浦海岸～三崎口のどこかでまぐろめしを食う予定。

レジャーは風呂かな？集まったメンバーで相談しましょう。
バスでぐるっと回ってもよし。

■予算
みさきまぐろきっぷ3,060円 + 枝線区間640円

■車中
カップ酒３本まではセーフ。

■二次会
