description

Rapha “Unofficial” Ride、第2回はCafeを起点に初夏の藤野を走ります。基本的にアップダウン基調ですが、距離を短めに設定し、After Rideを充実させます。もちろん参加条件はRaphaを愛するIndividualなRiderであること。

雨天の場合はCafe Restaurant Shu集合のFun Meetingにします。
その際は電車、お車でお越し下さい。

◎会場までの交通案内
最寄り駅：JR中央線 藤野駅
駅から：1.5km 徒歩15-20分
会場の装備：自転車スタンド、空気入れ（TOPEAKフロアポンプ）、交換チューブ（ブチル）
