h1. この企画は参加者少数のため中止とさせていただきます。
ご協力ありがとうございました。




mixiなどでもグループ・クラスなどを越えて再び繋がってきているかと思います。
そこで、一度大きめの同窓会を開きたいと思います。
が、正式に同窓会をしようとするとはがき送付、集計など作業が多くなりますので
基本的にはここでの表明のみをもとに準備します。
(どうしてもここにアクセス出来ない人が居る場合は、出来る人が代筆などしてあげてください)

開催の連絡も、口コミ(mixiなどのサイトを含む)で行います。
出来れば、全員に連絡が行くようにしたいので　近い人はもちろん、その周囲まで広がっていくよう
口コミをよろしくお願いいたします。

基本的に、個別の連絡は行わない予定です(特殊な事情や緊急時などを除く)。
特に、開催直前などにはマメにチェックしてください。

その他、ご意見・ご希望などありましたらどしどしご連絡ください。

参加表明時のアンケートは○をつけるでもよし　思う事を書くでもよし
ご自由にどうぞ(結果が反映される場合もされない場合もあるかと思いますがその辺はご容赦のほどを)
