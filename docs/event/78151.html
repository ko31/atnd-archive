ビデオ通話で恋活・婚活パーティ「マイ恋」をリリースしました。

「マイ恋（マイコイ）」とは、株式会社マッチバンク（代表取締役：松本明伸 / 所在地：大阪府大阪市）が提供する「ビデオ通話でパーティに参加するマッチングアプリ」です。

今までの恋活、婚活マッチングサイトではなかったビデオ通話機能を用いたマッチング、コミュニケーションを実現。
毎日決まった時間になるとアプリを起動するだけでクラウド上の婚活パーティに無料で参加できます。

ぜひアプリを立ち上げて、ご参加ください。
