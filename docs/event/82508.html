Raspberry Piを使っていろいろやってみよう

えれくら！TypeRは、Raspberry Piを主題とした勉強会です。

「Windows10 IoT Core for Raspberry Pi」、各種Linux系ディストリビューション、そのほかのOSも話題に含め、RaspberryPiについての、広い範囲での情報交換を行います。
RaspberryPi にご興味のある方、どんどんご参加くださいませ。

ご用意いただくもの

  • Raspberry Pi各モデル
    RaspberryPiには現在、モデルが数種類ありますが、持っているモデルをお持ちください。
  • Micro SDカード
    SDカードは、OSをインストールするのに使います。8GB以上の容量があるClass 10以上のものをご用意ください。
  • ノートPC
    できれば、ノートPCをお持ちくださいませ。試してみた結果をそのまま持ち帰ることができるので便利です。

  1. インストールで困ってる人お助けタイム [10:30-12:00]
    開発環境や実機へのOSインストールで困っている方は、ぜひお助けタイムへご参加くださいませ。

    • Windows10 IoT Core for RaspberryPiに関するインストールについて
      RaspberryPiは、 RaspberryPi2またはRaspberryPi3 が必要です。
      OSインストール用MicroSDとして、 8GB以上の容量があるClass 10以上 のものが必要です。
      開発用のPCは、 Windows10 がインストールされているもの が必要です。HDDやSSDには30GB以上の空き領域が必要です。
      また、PCには VisualStudio 2015 Community Update 3 と、各種SDKをインストール しておく必要があります。
      開発環境がかなり重いので、快適に開発作業をおこなうために、 PCはなるべく高性能なもの をご用意ください。
  2. 今回の話題について説明・自己紹介など [13:00-13:20くらい]
