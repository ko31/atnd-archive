■講演概要

【Web業界でのキャリア形成に悩まれている方・新しい技術への志向が強い方必見！！】
市場にWebが一般化してきて久しく、近年ではWeb系事業会社へ就職・転職も数多くみられるようになってきました。今回のイベントではゲーム、BtoC、BtoBを展開する急成長企業にて活躍する4名の技術責任者・CTOにキャリア形成の極意を語ってもらいます。最新技術トレンドを掴み、大規模マネジメントも経験し、急成長Web企業でエンジニア集団を率いている各技術責任者・CTOが過去どのようなキャリアを形成し、未来にどんなキャリアを描いているのかをお聞きすることができるかと思います。当日現在パネルディスカッション形式で、技術責任者・CTOが考える最新技術トレンドに関するお話もいたします。フルスタックエンジニアが叫ばれるようになって久しい今だからこそ、改めてAWS/GCPやCIツールによる自動化等・高負荷対策などに対する各技術責任者の考えや意見を聞けます！パネルディスカッション中盤では自身のスキルや考えを社会に対してアウトプットすることの重要性やその方法についてもお話しますので、技術アピールに悩まれている方もぜひふるってご参加ください！

また今回のイベントでは懇親会時にキャリア相談の機会も設けておりますので、お気軽にお声掛けください！

■講演者プロフィール

株式会社Speee　エンジニアマネジメント責任者 兼 エンジニア採用責任者　是澤大志氏

1979年生まれ。2000年からIT業界に飛び込み、20代の頃は主にリードエンジニアとして、30代からは組織やプロジェクトのマネジメントを中心に活動。 6回の転職と10社での多種多様な事業開発を経験。起業や個人事業主として活動していた時期もあり技術顧問や技術アドバイザーなどの経験も積む。 シーエー・モバイルにて研究開発部署の立ち上げやALBERTで技術責任者、フリマ系ベンチャーでCTOを経験し、現在はSpeeeでWebマーケティング事業部の開発責任者を務めたのち新卒・中途エンジニア採用を中心にエンジニアの組織系の責任者として携わりながら、スタートアップの技術支援を行なっている。

<img src =“http://netjinzaibank.s3.amazonaws.com/img/event-cto1708/prof02_265-325.jpg”, width =“265”, height = “325”, >

株式会社Kaizen Platform　CTO　渡部拓也氏

2004年⼀橋⼤学商学部卒業。同年NTT コミュニケーションズ株式会社に⼊社、その後、株式会社グラファイトやニフティ株式会社などでエンジニアとして数々のプロジェクトに参画し、クライアント/サーバ型システムの構築やインターネットサービスの開発に携わる。2010 年にグリー株式会社 NativeGame事業本部で開発と事業の責任者を務める。2014 年スマートニュース株式会社で広告プロダクトマネージャを務め、2016 年 10に Kaizenplatform.Inc.（現・株式会社 Kaizen Platform）参画、CTO に就任。

<img src =“http://netjinzaibank.s3.amazonaws.com/img/event-cto1708/prof03_265-325.jpg”, width =“265”, height = “325”, >

株式会社よむネコ 技術責任者　高野政徳氏

学生時代より個人でゲーム開発活動を行う。1998年にソニー・コンピュータエンタテインメント（現ソニー・インタラクティブエンタテインメント）に入社。新ハード向けのローンチタイトルや、世界初の AR ゲームなどの開発に携わる。その後、ガンホー・オンライン・エンターテイメント株式会社に転職し、MMORPG の運営やスマホ事業起ち上げ時の技術責任者を務める。現在は株式会社よむネコの技術責任者として、VR 向け RPG の開発に取り組んでいる。

<img src =“http://netjinzaibank.s3.amazonaws.com/img/event-cto1708/prof04_265-325.jpg”, width =“265”, height = “325”, >

株式会社gumi　CTO　幾田 雅仁氏

1997年システム会社に入社。大手パソコン通信サービスやポータルサイトの開発に携わる。その後、大手ポータルサイト運営会社を経て、2007年金融系企業に入社。クレジット決済のプロセッサでシステム開発課を率い、決済システムを刷新する。2012年ゲーム業界のサーバ負荷に魅せられてgumiに入社。各種ゲーム共通マイクロサービスやライブラリ開発、社内外の技術勉強会の講師などを経て、2016年CTOに就任
