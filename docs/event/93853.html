真のソーシャルメディア活用【戦略・戦術・運用編】～2018年、絵に描いた餅で終わらせない！ ～

description

デジタルマーケティングにおいて、ソーシャルメディアはユーザー接点として欠かせないツールであり、現在ではユーザーの行動の多様化に合わせて複数のソーシャルメディアアカウントを運用している企業も少なくありません。

しかしそういった中で、「なかなかマーケティング成果を上げられない」「ソーシャルメディア施策の効果を感じられない」といった声も多く耳にします。

そこで本セミナーでは、ソーシャルメディア活用の戦略・戦術・運用編の3部構成とし、その運用を通じてマーケティング成果を出すためのヒントをわかりやすくご紹介いたします。

＜＊ご参加特典＞
本セミナーにご参加いただいた方には、季節や記念日、イベントをテーマにしたFacebook投稿のコンテンツ作成にご活用いただける『ソーシャルメディア運用ご担当者のための年間投稿カレンダー』をセミナー後日にお送りいたします。

▼概要
●日時：2018年1月19日(金) 15:00〜17:15（14:30受付開始）
●定員：70名・無料（主催企業と同業の方はご遠慮ください）
●対象：企業のマーケティング、広告宣伝、経営企画等のご担当者さま
●主催：株式会社メンバーズ
●会場：虎ノ門琴平タワー3F大会議室（東京都港区虎ノ門1-2-8）　アクセス

▼アジェンダ
●第1部　15:00～15:30
【戦略編】デジタルマーケティング全体から見るソーシャルメディア戦略
株式会社メンバーズ　ユニットリーダー／プランナー　堤下 大輔
複数のソーシャルメディアにオウンドメディア、企業が運用するメディアはますます増え続けています。そういった企業を取り巻く環境の中、あらためてデジタルマーケティング全体におけるソーシャルメディア戦略策定のポイントとその重要性について解説します。

●第2部　15:30～16:00
【戦術編】プラットフォームとユーザーの変化～ソーシャルメディア戦術のケーススタディ～
株式会社メンバーズ　ユニットプロデューサー　飯嶋 佳子
Instagramの台頭をはじめ、ユーザーが利用するソーシャルメディアプラットフォームとその使い方は常に変化、多様化しています。こうした状況で、企業はどのようなコミュニケーションをしていくべきなのか、​FacebookやTwitterなど、プラットフォーム別のベストプラクティスを元に、ソーシャルメディア活用戦術をご紹介します。

●休憩　16:00～16:10

●第3部　16:10～16:50
【運用編】マーケティング成果を上げるためのソーシャルメディアPDCA運用
株式会社メンバーズ　ユニットリーダー／ソーシャルメディア・コンサルタント　福島 享之
ソーシャルメディアの戦略・戦術に沿ってマーケティング成果を上げていくには日々の運用が重要になります。複数のソーシャルメディア、オウンドメディアと連携した運用などを踏まえながら、「どのような運用・効果検証をしていけば良いのか」「なぜソーシャルメディアの運用が重要なのか」をLexus Internationalさまのコンテンツメディア「VISIONARY」運用事例他を用いてご紹介いたします。

●質疑応答・アンケート記入・名刺交換　16:50～17:15

▼講師紹介
堤下 大輔
株式会社メンバーズ　ユニットリーダー／プランナー
前職ではシステムエンジニアとして投資信託の資産運用システムの開発・運用に従事し、大規模システム移行支援を行った。 2012年にメンバーズに入社し、ソーシャルメディアを中心として多数アカウントのプロモーション戦略の立案・運用を手がける。FIFAワールドカップのプロジェクトにおいては、グローバルでのSNS運用を統括し、リアルタイムマーケティングを実施した。現在は、グローバル企業のデジタルマーケティング戦略の立案や、コンテンツマーケティング施策の企画から実行までを幅広く担当。

飯嶋 佳子
株式会社メンバーズ　ユニットプロデューサー
出版社を中心に、雑誌およびWebメディアの企画、編集、ディレクション業務を経験し、メンバーズに入社。ソーシャルメディアを中心としたコミュニケーション、コンテンツ、デジタルマーケティングのあれこれで日々奮闘中。

福島 享之
株式会社メンバーズ　ユニットリーダー／ソーシャルメディア・コンサルタント
旅行代理店にてWebサイトのコンテンツ企画、制作ディレクション、デジタル広告出稿管理、事業開発、SNS運用担当などを経験し、2014年にメンバーズに入社。入社後はFacebookを中心とした企業のデジタルマーケティング支援、各企業が掲げる戦略に基づいた、コミュニケーションを整理し、ソーシャルメディアを活用した戦略策定、コンテンツ企画、広告のプランニングなどを実施。大手クルマブランドのグロースハックプロジェクトを担当。主にコンサルティング活動を専門領域として担当。
