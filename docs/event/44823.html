RubyとGoで書かれたマルチ言語PaaSであるCloudFoundryについての勉強会です。

今回はCloud Foundryの原点に立ち戻ろう！ということで、Cloud Foundryを触り始めた人向けの発表をいくつか募集したいなと考えています。

また、

  • Cloud Foundryを自分の○○環境で構築してみた！
  • Cloud Foundry上でこんなアプリケーションをデプロイしてみた！
    という体験談も大歓迎。

もちろん、これまでのCF輪読会のメインであった、CloudFoundryを深掘りするような発表もOKです。
