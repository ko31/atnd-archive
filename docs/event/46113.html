description

電子回路の知識がなくても簡単に電子工作できるLittleBitsとKORGがコラボレートしたモジュラー・アナログ・シンセサイザー Synth Kitの発売記念として、LittleBits + KORG Synth Kit workshopを無料開催します。

商品の開発秘話から魅力的な使い方に加え、研究者・アーティスト、電子工作方面からSynth Kitに興味をお持ちの方々をお招きしSynth Kitについて語って頂きます。

食事メニュー（別料金）も用意されているので飲みながら食べながらSynth Kitを楽しみましょう！

http://jp.littlebits.com/synth/

東京都港区西麻布１-7-11 霞ハイツ B1F

JulieWatai
24歳で単身イタリアに渡り、イタリアの美術書籍出版社DRAGO社からフォトグラファーとして「SAMURAI GIRL」（2006年）を出版。NYのMOMAに置かれるなど世界的に販売されており、累計120万部を売り上げる。
帰国後も、フォトグラファーとしての活動をしながら、「天野あい」名義でのタレント活動や、mishmash*juliewataiとしての音楽活動など積極的に露出をしている。
2013年9月に開催された「Tokyo Crazy Kawaii Paris」でオフィシャルフォトグラファーを務め、さらにパリにあるファッションビル「CITADIUM」のXマスキャンペーンの店内デコレーションにJulieの作品が多数展示されている。
http://www.kahunarock.com/juliewatai/

渡邊恵太

2013年４月より 明治大学 総合数理学部 先端メディアサイエンス学科 専任講師。
主に知覚や身体性を活かしたメディアデザインやWebと日常を融合するインタラクション手法の研究に従事。
http://persistent.org/

sawako
名古屋出身、東京とニューヨークに６年ずつ在住後、ノマディックに活躍中の音響作家。デジタル技術を用いながらオーガニックで瑞々しい世界感を紡ぐ。これまでに、12kなどから通算４枚のソロアルバムをリリース。Sonarフェスティバル（バルセロナ）やWarmup at P.S.1/MoMA（ニューヨーク）など世界各地でパフォーマンスを勢力的に展開。Nicolas Collinsの「Handmade Electronic Music」等で作品が紹介されている。慶応義塾大学環境情報学部卒業、New York大学ITPで修士号。
http://www.troncolon.com
