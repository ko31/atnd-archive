description

　様々な分野の習い事を1日でお子さまが体験できます。
　お子さまにはそれぞれの「好き」があります。
　それを探すためには、お子さまが幅広い体験をすることが大切です。
　お子さまが夢中になれるものが何かわからない子育てパパママは必見のイベントです。

▼ こんな子育てパパママにオススメです！
・ 子どもが何に興味があるのかわからない
・ そろそろ子どもを習い事に通わせたいけど、探す時間をなかなかとれない
・ 子どもと一緒に楽しく遊ぶ時間が欲しい

▼ こんなことが学べます！
・ お子さまが1日で様々な分野の習い事を無料で体験できます
・ お子さまの習い事に対する興味・関心や傾向をつかめます
・ 親子で体験できる習い事もあり、一緒にお楽しみいただけます

▼ 習い事紹介

　習い事体験中のお子さまの写真をプレゼント
