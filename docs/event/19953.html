２０１１年４月の宣伝会議アプリマーケティングカンファレンスで好評だったため、続編として、
再度、iPhoneアプリ開発会社KAPPE代表の佐藤氏が登壇します。

前回のARアプリからさらに、位置情報に特化したiPhoneアプリの技術・企画の事例紹介を行います。

・ワインリンクARアプリ（位置情報連動店舗マップ）
・キタ―コンパス（位置情報ゲーム）
・エクシオライン（位置情報コミュニケーション）
など

▼出演者プロフィール：
[ 佐藤輝幸 ]
1978年生まれ、株式会社かっぺ代表取締役社長。
iPhoneアプリ「一分間英単語シリーズ」「ワインリンク」「EXEO LINE」などの企画・設計・ディレクションを担当。NPO法人78会理事・千葉大学非常勤講師もつとめる。
