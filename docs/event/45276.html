  • イベント概要
hackathon の後に、わりとイベント少なすぎるから皆気合い入れてあつまるからコンスタントにイベントやろう
見たいな話をしてて、 Shibuya.pl という Plack のイベントをやる事にした。

という事で、近年の Perl 製 Web アプリ制作に欠かせない Plack/PSGI 周りをネタを集めた勉強会を開催します！
わりと皆さん気楽な感じでお越しいただければと思います。
