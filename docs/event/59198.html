第35回 FreeBSD勉強会

FreeBSD 10.1-RELEASE新機能紹介

講師
後藤大地（BSD Consulting, Inc.／ONGS Inc.）

FreeBSD 10系の最新版となる「FreeBSD 10.1-RELEASE」が公開されました。FreeBSD 10系は新機能の取り込みや安定性の高さなどから、最初のリリースであるFreeBSD 10.0-RELEASEから実用性の高いバージョンとされています。10.1-RELEASEには11-CURRENTからバックポートされた新機能の導入やドライバのアップデート、各種ソフトウェアのアップデートなどがとりこまれています。

FreeBSD 10系で導入された新機能をおさらいするとともに、10.0-RELEASEから10.1-RELEASEへアップデートする方法のおさらい、新機能の紹介などを行います。これからFreeBSDを使ってみたい方、いま10系を使っている方などにオススメの内容です。
