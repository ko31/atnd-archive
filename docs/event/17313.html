Javaの難問・奇問・珍問が並ぶ、JavaOneの名物セッション 「Java Puzzlers」

明日から使えるJava SE 7

櫻庭 祐一 – Java in the Box

7月28日にリリースされる Java SE 7
Project LambdaやProject JigsawはJava SE 8に移行してしまいましたが、
Project CoinやFork/Join Frameworkなど、さまざまな機能が導入されます。
そこで、本セッションでは、Java SE 7を明日からでも使えるように、
Java SE 7の主要な機能をデモを交えて紹介します。

Advanced Tech Night（略：ATN）について

この勉強会は、Web／エンタープライズ系を中心とした、
ソフトウェア開発の先端技術を学ぶことを目的としています。

主なターゲットは、

  • 先端技術の情報を知りたい
  • 先端技術を、仕事に活かしてみたい
  • 自分が興味を持っている技術について語りたい

という想いを持つ、ソフトウェアエンジニアです。
（もちろん、「これから学びたい」という初心者の方も大歓迎です。）


この数年で、クラウド、スマートフォン、分散並行処理、HTML5等、
技術革新の速度も加速度的に増してきており、
ソフトウェアエンジニアがお互いに新しい知識を共有しながら、
交流する場が必要だと感じ、本勉強会を立ち上げました。


私たちも、まだまだ手探りの状況ですが、
今後、最新の技術を取り上げていく予定です。

一緒に勉強していきたい、という方は、ぜひご参加ください。
