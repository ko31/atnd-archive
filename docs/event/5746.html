概要

http://computer-forensics.sans.org/challenges/ で開催されている SANS Digital Forensics and Incident Response Challenge のパケットを読む会をやってみます。
時間があれば事前に上記の SANS のページから evidence file を入手して、解析しておいてください。解析ツールは自由です、というか商用のアナライザー使える人はぜひ当日も持ってきてください。

当日の持ち物



  • ノートパソコン
  • 解析ツール (ネットワーク アナライザ、フリーの物なら Wireshark とか Microsoft Network Monitor とか)
  • 参加費 (会場代割り勘 + 飲物代、1000円程度)

お知らせ

  • 一応マイスペースを予約しましたが、人数が集まらなかったら予約はキャンセルして一般席でゴソゴソします。
  • 参加表明された方は必ず当日にこの ATND ページを確認してください。
  • ご質問などは http://www.hebikuzure.com/ を参照して hebikuzure にメールしていただくか、Twitter で hebikuzure 宛お願いします。(hebikuzure をフォローしていただくと最新情報が得られるかもしれません)
