description

関ゲ部（←作りかけ）で自作ゲーム(制作中含む)ぷち展示会＆交流会的なものをします。

もともと、ゲーム制作者が集まれるようにと作った場所。
この場所で、みんなの自作ゲーム（制作中可）を
遊んだり、アドバイスをしたり・もらったり、その場で作ったりして過ごしましょう。

その後は、交流会。ゲームの話で盛り上がりましょう。

この日は、Biz Libraryを1日関ゲ部の貸切にします。
ゲーム大好きな人たちが集まり、ゲームを楽しむ1日にしましょう。
制作していないけど、ゲームで遊びたいだけという人も大歓迎です。

日時：11月24日（月・祝）
　　　ぷち展示会：13時～18時
　　　交流会（飲み物と軽食を用意しています）：18時～21時くらい
　　　＊途中入退出自由

場所：Biz Library

参加対象：ゲーム制作者、ゲーム制作をしたいと考えている人、ゲームで遊びたい人、関ゲ部に興味のある人など
ゲームが好きな人なら誰でも歓迎です。

参加費：ぷち展示会＋交流会（飲み物と軽食を用意しています）　3000円
　　　　ぷち展示会のみ　1500円　　　　
　　　　交流会（飲み物と軽食を用意しています）のみ　2000円
　　　　Biz Library会員　ぷち展示会無料、交流会参加費1000円

＊参加表明の際に、
・両方参加
・ぷち展示会のみ参加
・交流会（飲み物と軽食を用意しています）のみ参加
を明記ください。

なお、交流会（飲み物と軽食を用意しています）は準備の都合上、前日以降のキャンセルはキャンセル料を全額いただきます。
