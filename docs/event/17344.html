description

どうせ生まれてきたからには、デッカい仕事をしてみたい。だけど、資金もなけりゃ、チャンスもない。とか思ってたりしませんか？

数名の人間が集まってできた会社がありました。彼らはRuby on Railsで開発をしていく生産性の高さだとか、快適さだとかを身にしみて感じていたので、Rails開発をもっと気軽、かつ便利にはじめられる環境を提供するサービスをはじめようと考えました。でも、彼らにはすごい資金があった訳ではないので、Amazonのクラウドサービスを利用して自分たちのクラウドサービスを立ち上げました。

小さなオフィスではじめたビジネスですが、次第に世界中のRubyコミュニティから受け入れられ、今では世界を代表するクラウドビジネスの成功者になりました。

今週の金曜日、この会社。「 Engine Yard (http://www.engineyard.com)」が福岡にやってきます。

彼らが歩んできたストーリーは、もしかしたらみなさんの「これからのストーリー」なのかもしれません。

来福するのは、EngineYardのCEO John Dillonさんです。

これを機会に、小さな組織でも世界で成功する秘訣を学んでしまいましょう。

入場無料で、もちろん通訳もつきます。ぜひ、この機会を、おみのがしないように。
