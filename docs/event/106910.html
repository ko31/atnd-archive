    チームと組織開発のためのオンボーディング道具箱

    松本 潤二（まつもとじゅんじ）さん



    《プロフィール》
    松本屋
    プロジェクト・ファシリテーター、アジャイルコーチ
    CPCC（Certified Professional Co-Active Coach）
    チームや組織に入りコーチングやファシリテーションを活用して変容を促す実戦経験に長ける。15年程の開発経験ののち、2004年頃からアジャイルプロセス協議会と日本コーチ協会東京チャプターに身を置き日本への普及活動を行ってきた。現場主義に重きを置き、机上の理論ではなく実際に実践・考察し講演やワークショップで伝えている。活動はIT業界にとどまらず、国際交流事業や福祉分野などでのファシリテーション実践と研修などを行う。
    info@matsumoto-ya.com


    《詳細》
    時代や学校教育の変化に伴い、新しい人材を迎え入れるにも昭和・平成の時代から続く社員教育や組織の受け入れ態勢では上手く行かなくなってきています。貴重な人材にいかにしてチームや組織に馴染んでもらうのか、招き入れる側の仕組みとマインドセット作りなど課題が山積しています

    個人をチームや組織の色や形に染めその枠の中に当てはめるのではなく、個人の特性を見極め可能性を探りつつ、その個人が自ずと中に足を踏み入れていくような環境を作ることが必要です。また各々の変化に常に意識を向け、継続的に環境を整え続けていく必要があります。

    今回の講演では、コーチングやファシリテーションの知識と実践経験の中から、ヒントとなるツールを紹介しつつ、皆さんの実践の場で活用できるように体験学習を交えて行います。

    《想定する参加者》
  • 新しい人材を迎え入れるチームや組織
  • これからの人材教育や研修のヒントが欲しい人
  • チームを運営しているリーダ
  • 思い通りに相手が動いてくれないと感じている方
  • 自分自身が新しい環境に身を置く方
