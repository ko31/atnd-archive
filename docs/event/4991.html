□ 1st Session [写真がもっと大好きになる授業]
───────────────────────────────────
◆ 下川 高広 (しもかわ　たかひろ)
冨士スタジオ

自己紹介 :
最近、Wii Sports Resortにはまってます。
そうそう、カートもやってました。人間の技量とセッティングだけで勝負が決まるのはおもしろい。
あと、バスケもしてます。（・・・ミニバスのコーチですが・・・）
しまなみも自転車で渡ったことがあるんですが、ものすごく達成感があってよかったよ。
今度、いっしょにやろうや！！

経　歴 :
1989年3月東京工芸大学短期大学部写真技術科卒業
1989年4月株式会社資生堂入社　宣伝部写真制作室に勤務
1991年4月冨士スタジオへ専従
1991年6月広島県写真家連合会府中大会にて自由作品の部で銀賞を受賞
1991～1997年資生堂宣伝部写真制作室現役＆OB展出展
1993年6月広島県写真家連合会三次大会にて自由作品の部入賞
1998年3月ふくやまアートプロジェクト　野外彫刻を撮ろうにおいて優秀賞
2001年7月中国ブロック写真家連合会第36回島根大会にて自由作品の部で入選
2004年8月広島県写真家連合会福山大会にて自由作品の部で銅賞を受賞
2006年8月広島県写真家連合会尾道大会にて自由作品の部で金賞を受賞
2007年3月第24回技能グランプリに広島県代表として出場し、銀賞受賞（備後地方では初入賞）
2007年8月広島県写真家連合会三次大会にて自由作品の部で銅賞を受賞
2008年8月広島県写真家連合会府中大会にて自由作品の部で入選
2010年4月APA（社団法人日本広告写真家協会）・JPS（社団法人日本写真家協会）同時入会　広島県で同じ入会は2人目

セッション概要 :写真撮影するのに必要な構え方、シャッターの押さえ方をはじめ、基本的な操作を始め、必ずうまく見える構図などを組み入れながらワークショップ形式で行います。

2. 三脚 (持ってる人だけ)
