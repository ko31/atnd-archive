第八回　アプリ開発ハンズオン

学生・初心者　大歓迎!!!

アプリ開発ハンズオンの第八回目です。
iOSアプリ作成に興味のある方、奮ってご参加下さい。
途中参加も可能です。

注意：Android開発の申込が無いため、今回は中止とさせて頂きます。iOS開発に関しては引き続き募集しておりますので、気軽に参加ください。

参加資格

以下の条件を満たしていればＯＫです。

  • 開発に使用するＰＣを準備できること
    iOSアプリ開発を体験したい方はMacを持参してください
    環境構築は済ませておいてください(Android・iOS)
    ※実機テストの端末、ネットワーク環境などは準備します

ハンズオン内容

<< android >>
　　BroadcastReceiverを使ってみよう！
　 画面のON、OFFや、自分で飛ばしたメッセージを受け取っちゃおう！
<< iOS >>
　 【テーマ：通信】
■対象者・開発内容は参加者のスキルに合わせて変更するので初心者大歓迎。
　　 ■解説・同期通信・非同期通信とは。・WebAPIとは。・XMLのパースとは。
　■開発内容・Webサイトのソースコードを取得する。・iOSからWebAPIを使ってみる。(GoogleSuggestAPI, WikipediaAPI)
　　 ■目標・NSURLConnectionクラスの理解・XMLのパース処理の理解・WebAPIの使い方を理解
