概要

「久留米プログラミングフリークス　スマートフォン分科会」とは、スマートフォンやタブレットPC上で動作するアプリケーションの開発や、それらの利用についての情報交換を目的とする、「久留米プログラミングフリークス」を母体としたコミュニティです。

日頃からアプリケーションを開発されている方から、プログラミングなんてした事もないけれどもやってみたい方、こんなアプリケーションがあったら便利なのに……というようにアイデアをお持ちの方、その他スマートフォンに興味をお持ちの方を募集しています。

今回は、実際に Android 開発環境と Titanium Studio を用いて、 Android アプリケーションをハンズオンで開発します。
今回のテーマは「ラーメンタイマー」です。
午前中いっぱいと午後に基礎と応用を全員で作成します。
その後チームを作り、チームごとにラーメンタイマーを拡張し、最後に発表を行っていただきます。

なお、参加費は無料です。

必要なもの

Android 開発環境と Titanium Studio をコンピューターにインストールし、ご持参ください。

自力で環境をご用意頂くのが難しい場合は、 5/9 にココチカフェ様 ( 久留米市日吉町2-3飛永ビル1F )にて KuPF 運営ミーティングにおいて環境構築を行いますので、ご参加下さい。
