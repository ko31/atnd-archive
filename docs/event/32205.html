■異業種交流会プラスパートナーズとは？

プラスパートナーズの交流会は、ビジネスに熱い情熱をお持ちの方の為に、様々な業種・職種の方とコミュニケーションを深めていただき、有意義な人脈形成や情報交換を行いまして今後のビジネスに役立てていただく事を目的に立ち上げられた交流会です。
異業種交流会プラスパートナーズ

新宿、池袋、渋谷、秋葉原など主に都内で開催しております。

■交流会概要

毎回ご好評いただいておりますIT交流会を開催いたします。
弊社IT交流会は、システム開発会社、webデザイナー、企業のweb担当者、ITを今後取り入れて業務効率化を計りたいと思ってらっしゃる企業の方など、幅広く参加される交流会です。
今回はイベント団体e-venz(イベンツ)さんとの共同開催となり、人数も大幅増になる予定です。

毎回多数の案件発注やビジネス協業が発生し、ご参加の方から感謝のご報告を受けております！
今回は新宿にて開催いたします。
現在、SEOコンサルティング会社、エンジニア派遣会社、システム開発会社、webデザイナーなど多種多様な方から参加申し込みを頂いております。
バリバリのIT業界の方、少しだけITに関わっていらっしゃる方、ITにご興味がある方、大歓迎いたします。

■参加ルール

プラスパートナーズでは皆様に安全に交流会にご参加いただけるように以下のルールを定めております

  • 参加者様同士安全にご交流いただくという趣旨に反する以下の業種の方の参加を固くお断りしております。
    万一業種を偽って参加した場合には、発覚次第即刻退場していただきます。その際、参加費は返却いたしません。
    1. ネットワークビジネス（その他類似ケースも含みます）
    2. マルチ商法やその類
    3. 先物取引
    4. 能力開発ビジネス
    5. 宗教・政治の勧誘
    6. 生保・損保業の方（主催者の紹介での参加1～2名は除く）
      ※副業として営業されている方も含みます。
  • 強引な勧誘、同意を得ずに説明開始、しつこい営業電話やアポ取り、その他相手に迷惑のかかる行為全般を禁止します。
    社会人としてのマナーにご配慮願います。
    交流会の最中に強引な勧誘や迷惑行為を行った方は即刻退場していただきます（返金なし）。
  • 名刺は、お名前、ご住所、電話番号等ご連絡先がわかるものをご用意ください。
    （手書きの名刺はご遠慮ください）名刺をお持ち頂けない方は、ご参加できません。
    ルール詳細
