description

大阪でぷよぷよオフ会を開催します。ぷよぷよが好きな人なら誰でも参加OK。対戦が好きな人、ぷよキャラを愛している人、絵を描くのが好きな人、コスプレが好きな人、いろんな人が参加予定です。

参加予定者のTwitterリスト
https://twitter.com/inosendo/osakapuyooff130323

実施内容は『ぷよぷよ』シリーズ（主に『ぷよぷよ！！(20th)』『VCAぷよぷよ通』）での対戦が中心ですが、おしゃべりしたり、他のゲームで遊んだり、絵を描いたりなど、基本的に自由です。2部屋取っており、途中から希望者でカラオケも行う予定。途中参加、途中退室可能です。

オフ会中に『ぷよぷよ！！(20th)』を使用したぷよぷよ大会を開催します。
詳細ルールはこちら→ 2013年3月23日 大阪ぷよぷよオフ会開催

参加費：1000円
※会場の「オフ会ランチフリー」コースの料金です（ランチ付き。ドリンク飲み放題＆アイスクリーム食べ放題）。

会場：カラオケZERO4 西中島店
※最寄り駅は「JR新大阪駅」「阪急南方駅」「大阪地下鉄 御堂筋線 西中島南方駅」です。

持参品：持っていれば3DS、DS等の本体と『ぷよぷよ』シリーズのソフト、充電ケーブルなど （なくても大丈夫です）
※会場にモニターが3つあるので、Wii 3台を接続して使用します。Wiiのコントローラーをお持ちの方は可能ならご持参ください。

当日のスケジュール（予定）：
　11:00～12:00　自己紹介、フリータイム
　12:00～13:00　ランチ
　13:00～16:00　ぷよぷよ大会
　16:00～21:00　フリータイム

ぷよぷよ大会終了後のフリータイムでは、別室でカラオケ（JOY SOUND）も行う予定です。カラオケ用の部屋は、靴を脱いで上がる座敷形式になっています。

また、フリータイム中にテトリスオンライン世界一のプレイヤーであるhebo_MAI氏によるテトリスエキシビションも実施する予定です。

会場にはコスプレをされる方のための更衣室が用意されています。参考

遠方から来られる方で宿泊場所が必要な場合、会場のカラオケ店には午前8時までの「夜フリータイム」があります（料金表）。また、同じビル内にネットカフェもあります（Bbc@fe ビィビィカフェ新大阪店）。

当イベントに関するお問い合わせはTwitterでイノセンス @inosendo までお願いします。

主催：ぷよぷよポータル！！
