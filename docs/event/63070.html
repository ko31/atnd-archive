【CAMPHOR- リクルート】1 day Hackathon

イベント ユーザー
event_id 63070
title 【CAMPHOR- リクルート】1 day Hackathon
catch Web サービスを１日で作りきってみませんか?
description

作りきる！

今回のイベントでは、「アイデア→コーディング(開発)→リリース (公開)する」という一連の流れを体験できます！

懇親会を含め、参加費は無料です。

何をつくる？

当日に作るものはWebに関係するものですが、細かいアイデアは参加者の自由です。
普段考えているアイデアを一緒に形にしましょう。

※ 開発時間が限られるため、アイデアはイベント当日考えるよりも事前に考えて頂けるとスムーズにご参加頂けます。

例

  • HTMLとCSSで自己紹介ページを作る。
  • JavaScriptでテトリスを作る。
  • Ruby on Railsでブログを作る。
  • Pythonで検索エンジンを作る。

対象者

何かを作って公開したい学生。
最低でもHTMLを少し書いたことがある人を想定しています。
HTMLに自信の無い方は以下の動画を事前に観て頂けると安心して参加頂けます。
HTML入門 (全16回) – プログラミングならドットインストール

持ち物

ノートパソコン
Windows でも Mac OS X でも Linux でも構いません！

メンター

当日はメンターとして CAMPHOR-メンバー が会場に居ます。解らない事があった際にはお気軽にご質問下さい。

当日のスケジュール

  • 10:00-10:30 オープニング
  • 10:30-17:00 開発&リリース (昼食は各自で)
  • 17:00-18:00 成果発表
  • 18:00-20:00 懇親会

懇親会のフードスポンサーは株式会社リクルートホールディングス様です！

景品

イベント終了時に優秀な製作物を決め、制作者には株式会社リクルートホールディングス様より以下の景品が贈呈されます。

・レッドブル1ケース(24本)
・人事とのお食事券(人事にご飯をご馳走してもらえます！)

その他

本イベントは、京都IT系学生コミュニティ CAMPHOR- 主催、株式会社リクルートホールディングス後援により開催します。
CAMPHOR- HOUSE 内では、電源と Wi-Fi をご利用になれます。
会場の場所が少々分かりにくいため、
お困りの場合は @CamphorKyoto にリプライを飛ばして頂けると対応致します。
