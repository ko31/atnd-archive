◆どんなイベント？

今年はどんな年でしたか？
楽しかったり悲しかったりぼっちなうだったりすると思いますが、
1年を振り返ってみて、ああだったら良かったのにとか、こうすれば良かったなとか、
色々と反省点、改善点が出てくると思います。
どうすればより良い来年を迎えられるでしょうか？
考えてみましょう。
そして今回はそれをテーマにしてハッカソンしましょう。
一年の締めくくりのハッカソンです。

※ハッカソンとは

　「Hack」と「Marathon」を合わせた造語のことです。
　あるテーマのもとに、丸一日マラソンをするように必死になって開発をして、
　最終的に成果を出すことを目標にしたものです。
　メンバには色々役割があり、初心者でも何かしらやることはあります。
　気軽にとはいきませんが、初心者の方でも気負わずに、ぜひご参加ください。

※アイデアソンとは

　ハッカソンで開発するアイデアを出すために行うものです。
　今回はアイデアソンとハッカソンの二本立てになっているので、
　ご都合に合わせて、どちらか片方の参加でも構いません。
　もちろん、両方参加することも出来ますし、両方参加をオススメします。

テーマ

『来年に向けて、こんなのあったら良いなと思えるアプリ、サービス』を開発しましょう！！

  • 今年の反省を来年に生かせるアプリ、サービス
  • 今年の失敗を来年の成功に変えるアプリ、サービス
  • 今年のぼっちを来年こそはぼっちじゃなくなるアプリ、サービス

などなど。

◆日程

2011/12/23　アイデアソン

  1. 集合
    プレゼント(※後述)は持ってこなくて大丈夫です。
  2. オリエンテーション
    今回のハッカソンの説明をします。
  3. 1次アイデア(テーマ)発表　個人発表(提案)
    チームわけをするネタにします。あらかじめアイデアを考えてきてください。
    具体的なものでなくて構いません。具体的な内容は2次アイデア発表までに決めます。
  4. チームわけ
    「1次アイデア(テーマ)発表」で出たアイデアに賛同できるところに集まります。
    人数が多いところは半分に分かれてください。目安は1チーム5人前後です。
    なお、同じテーマでも違うものが最終的に出来ることを想定しています。
  5. アイデアソン
    チーム単位でアイデアをブラッシュアップしていってください。
    最終的には実際にどんなアプリ、サービスを作るかを決めてください。
    プレゼン資料作成もお忘れなく。
    なお、アイデアソンで出たアイデアは ここ にまとめてあります。ハッカソンから参加の方は一読ください。
  6. 2次アイデア発表　チーム発表
    アイデアソンの結果をプレゼンして頂きます。
    アイデアとプレゼンの評価を、それぞれ参加者全員にして頂きます。
  7. 得点発表　中間報告
    各チームの現時点での得点を算出し、発表します。

2011/12/25　ハッカソン

  1. 集合
    プレゼント(※後述)を忘れず持ってきてください。
    プレゼント引換券を渡します。
  2. ハッカソン
    アプリ、サービスを開発していってください。
    プレゼン資料作成もお忘れなく。
  3. 成果発表
    アイデアソンでの得点の高い順から、ハッカソンの結果をプレゼンして頂きます。
    アプリ・サービスとプレゼンの評価を、それぞれ参加者全員にして頂きます。
  4. 得点発表
    各チームの最終得点を算出し、順位をつけて最下位から順に発表をします。
  5. 表彰
    賞品贈呈。
  6. プレゼント交換イベント
    後述。

◆プレゼント交換イベント

プレゼント交換をしましょう！！
ルールは
①1000円までのプレゼントを１つ買ってくる。
　※料理に自信があっても、食べ物はやめておきましょう。
②寄せ集めて、ハッカソンの順位で、1位から順に好きなものをとっていく。
　※プレゼント引換券が必要です。25日の受付時にプレゼントと交換でお渡しします。
　　受け取ったら無くさないようにしてください。

諸事項

  1. 日本Androidの会、GTUGのイベントなので、絶対ではありませんが、開発するものにAndroidやGoogle技術を一部でも使用頂きますようお願い致します。
  2. 開発用にノートPCが必要となります。また、紙のノートなどがあるとまとめるときに便利です。
  3. お昼は各チーム毎に自由に取って頂いて構いません。
    ハッカソンでは時間が惜しくなり、持ち込む方もよくいらっしゃいます。
  4. 会場は飲食可能です。
  5. ゴミは各自お持ち帰りください。ご協力お願い致します。
  6. ネット環境はありません。必要な方は各自ご用意ください。
    各チーム無線で1回線はご用意できると思いますが、速度の保障までは出来ません。
  7. コンセントは各テーブルにご用意できる予定です。
    (会議室の前、中央、後ろの三箇所にコンセントがあり、仮に6チームの場合)
    ただし、口数に限りがあるので、タップや延長コードを各自ご用意ください。
  8. ハッカソンから参加される方は、当日スムースに開発に着手して頂くため、事前にアイデアソンの内容の確認をお願い致します。
    アイデアソンで出たアイデアは ここ にまとめてあります。
    なお、「こんなアイデアがあるから、ハッカソンでは別チームとして（もしくは1人で）開発したい！」という方も歓迎です。(12/18追記)
    アイデアソン参加者の方はアピールしたい点の記載と、想定される技術などの記載をお願い致します。(12/24追記)
  9. 参加費についてのお願いを「参加費」の項目に追記致しました。ご協力をお願い致します。(12/18追記)
