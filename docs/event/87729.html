XSHELLが開発しているisaax。Raspberry Pi や Edison , NanoPi などのLinuxが動くSBM(Single Board Computer)をカンタンに使うことのできるシステムです。

今回は Raspberry Pi を使って、インストール・プログラム登録・電子工作をお手軽にやってみましょう。

issaxを使うと、

  • sdカードにインストールが初期設定込みでラクチン
  • PCでプログラム作ってGitHubにpushするとRaspberryPiの内容がスグ更新
  • 電源とネットワークさえRaspberryPiにつなげばいいのでカフェでファッショナブルにだってできる

使い方を説明しながら、電子工作までやってみます。

XSHELLとは

シングルボードコンピューターを使って、プロトタイピングから、プロダクションモデルまでをシームレスにつなぐ、IoT向けのPaaSを開発しています。
WEBアプリケーション技術、ハードウェア技術、3Dモデリング技術を組み合わせて、IoTの事業開発支援も行っています。
オフィスはレーザー加工機、3Dプリンタ、小型CN旋盤、オシロスコープなどがあって、ミニFablabみたいになってるよ！
