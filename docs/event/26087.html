Windows Phone勉強会＠四日市とは？

2011年9月から日本でも販売が開始されたWindows Phoneについて、利用ユーザ・アプリ開発者のそれぞれの視点から、みんなで一緒に勉強して行く会です。東海地方でのWindows Phoneの勉強会が少ないことから、このような場を立ち上げました。

第2回は「WP7アプリ開発の始め方」と題して、Windows Phoneアプリケーション開発の始めの一歩を踏み出せるよう、アプリ開発の基礎知識を3つのセッションに分けて紹介します。第3回以降では、これらのセッションを踏まえて、より発展的なテーマを扱ったり、ワークショップによる演習へと繋げていく予定です。

Windows Phoneアプリケーションを開発したいと考えている方、また現在開発している方に是非参加していただければと思います。対象者は初級～中級レベルを想定していますが、上級者の方々も是非ご参加ください。

セッション概要

※セッション内容は予定であり、諸事情により変更する場合があります。

セッション1 「WP7アプリ設計・開発の流れについて（仮）」

Windows Phone 7のUIガイドライン、審査要件、メトロス​タイル、UX、またWP7端末の特性（センサーやハード​ウェア仕様など）などを踏まえて、WP7アプリをどう設計するか見ていきます。

また、アプリ開発の環境構築からAppHubへの登録までの流れも合わせて紹介します。

セッション2 「Windows Phoneの開発ツール」

WP7アプリはどのように作られるのか、SilverlightとXNAの2種類の開発スタイルについて紹介するとともに、開発ツールを使った実際のコード作成の流れ（プロジェクト作成→コード作成→エミュレータでのデバッグ、実機デバッグ）を紹介します。開発ツールはVisual Studio 2010とExpression Blendを紹介する予定です。

また、WP7アプリ開発で利用できるライブラリにも触れる予定です。

セッション3 「Silverlightで開発するWindows Phoneアプリ」

このセッションでは、SilverlightによるWP7アプリ開発の必須知識を紹介します。対象となる技術要素は以下を予定しています。
・XAML（テンプレート、スタイル、リソースも含む）
・UIコントロール
・画面遷移
・データバインド
・ライフサイクル

これらを用いることで、単純なアプリケーションを作成できるようになります。
