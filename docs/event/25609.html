AgileShibuya meetup #06

アジャイルソフトウェア開発者のためのコミュニティ「Agile渋谷」の定期meetupです。

  • アジャイル実践者
  • これからアジャイル開発者を目指す方
  • アジャイル開発に悩みをお持ちの方
  • 非エンジニア職の方も歓迎（実際に人事の方なども参加されています）
  • 渋谷界隈ではない人でもOK（新宿、品川近辺のかたも参加されています）

アジャイルに興味がある方は、お気軽にご参加ください。

「Agile渋谷」というコミュニティとは？

アジャイルソフトウェア開発のプラクティスを実践的に学びあい、共有できる活動をする人々です。
フェイスブックグループを利用して、主にやりとりをしています。ぜひ、ご参加ください。

  • Agile渋谷 – Facebook グループ
