概要

「PHP の次世代フレームワーク Symfony2」に実際に触れて学ぶ、 ワークショップスタイル の勉強会です。
今回は、第1回よりもすこし進んだ内容になります。

Symfony2 は、現段階（プレビューリリース 4）ではフルスタックのフレームワークというにはまだまだ足りない機能などがあり、開発者からも「実運用するのは crazy」と言われている段階ですが、それでも DI コンテナやアノテーション等、魅力的な機能をすでに十分な完成度で試すことができます。

こういった機能を、ドキュメントなどが未整備で、エラーなどが出ても構わない（むしろそれがいい）くらいの勢いで触りたい！、そんな crazy/addicted ユーザー向けのハッカソン的な勉強会です。

主催・協賛

  • 日本 Symfony ユーザー会 主催
  • ジンガジャパン株式会社 協賛

参加条件

  • ワークショップで作業できる PC をお持ちいただく必要があります。
  • Symfony2 に関して多少の基礎知識があることを前提とします。
    • 第1回の勉強会の基礎編程度（ レポート をご参照ください）
  • Symfony2 や関連ライブラリが動作する環境は、事前に準備しておいてください。
    • サンドボックス の check.php 等でご確認ください。
    • サンドボックスは、勉強会開催日が近くなりましたら、勉強会用のパッケージを配布します。
    • PHP は 5.3.2 以上が必須です。
  • 参加登録時に、アンケートへ回答お願いします
    • アンケートの内容を元に、あらかじめグループ分けを行います

内容とスケジュール

今回は、 ワークショップと LT になります。

  • 13:30 開場、準備

ワークショップの内容は、以下のようなものになる予定です。
作業は個人個人で行いますが、グループ単位でお互いにヘルプしながら進めていただきます。

※MySQL向けの開発を行います

【1】 CRUDを作る
　1. Entity準備
　2. Entityを追加するコマンド作成
　3. indexアクション、ページ
　4. フォームの作成
　5. 新規追加アクション、ページ
