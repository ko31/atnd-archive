実施概要

数々のソーシャルゲーム、モバイルゲームアプリを生み出している株式会社gloops主催による、ITエンジニア向け合同勉強会です。

勉強会では、gloopsエンジニアスタッフに加え、日本Node.jsユーザーグループ代表の古川 陽介氏、および株式会社ピクセルグリッドの小山田 晃浩氏を講師にお迎えし、Node.js、WebGLなどのWeb技術の最新動向をご紹介するとともに、モバイル向けWebアプリケーション、ネイティブアプリケーション開発への活用の可能性を探ります。

セミナー終了後は同会場にて、弊社エンジニアも参加させていただき、ご参加の皆様の交流・情報交換を目的とした懇親会も予定しておりますので、是非ご活用ください。

登壇者

  • 古川 陽介 氏 （株式会社ディー・エヌ・エー/日本Node.jsユーザーグループ代表）
    国内大手電機メーカーを経て、DeNAに中途入社。大学院で情報科学を専攻し、検索技術を研究。前職では文書管理システムの開発などに携わる。一方、社外活動でNode.jsの技術者コミュニティに参画し、現在は日本のユーザーグループの代表を務める。そこで培ったNode.jsの技術を活かして、いままでにないユーザー体験を創ってみたいとDeNAへ。現在、リアルタイムWebを実現する独自のテクノロジーを、ゲーム開発に展開していくプロジェクトのリードエンジニアを担当。
  • 小山田 晃浩 氏 （株式会社ピクセルグリッド）
    株式会社ピクセルグリッドに所属するフロントエンドエンジニア。HTML を中心にフロントエンド技術を駆使した Web コンテンツの実装を行なっており、特に CSS, SVG, canvas, WebGL の扱いを得意とする。2011 – 2015年 に Microsoft MVP for IE を受賞。
  • 佐々木 善行 （株式会社gloops）
  • 伊藤 学 （株式会社gloops）
