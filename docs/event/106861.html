いよいよ来週開催！

2019年6月末にてついに日本国内最終上映と映画『タイタニック』。
というわけで、約2年半ぶりに神戸で映画『タイタニック』を語ります。

公式イベントサイト・詳細はこちらになります。
→　http://eede.me/kobe/

日本ラスト上映
→　 2019年6月29日〜30日　バルシネマしんこうえん（公式サイト）
http://www.palcinema.net


シナリオや映画ライター経験者として、シナリオ的に映画『タイタニック』を分析し、最も重要なシーン30秒を語ります（そもそもシナリオ本編に書いてあるのです）。


また、タイタニックに関する最新情報や豆知識をはじめ、
あわせてご覧いただきたい関連映画タイトルなどもご紹介します。



映画『タイタニック』国内最終上映とあわせてお楽しみいただければ幸いです。



【開催内容（予定 ）】

・当初の企画提案が落ちて、ジェームズ・キャメロン監督が考えた内容変更
・映画『タイタニック』の主人公は誰？
・シナリオの基礎知識、海外映画と邦画のシナリオの違い
・映画『タイタニック』のシナリオをみてみよう
・映画『タイタニック』の最も重要なシーンはたった30秒だ！
・映画『タイタニック』のもうひとつのエンディング
・タイタニックについての豆知識や都市伝説（沈没したのは別の客船だった説）などをご紹介。
・3隻あるタイタニックの姉妹船はすべてが悲劇の沈没？
　その姉妹船を描く女スパイ映画が意外と面白い
・タイタニックの沈没映像は世界初のニュース映画？
・タイタニック動員ギネス記録は大阪の映画館が持っている？
