description

転職したい、恋愛結婚に出会いがない、モテない
など悩みのある方に

あなたは、自分のことをどれだけ理解していますか？
行動、結果を見ていても、今だけの時間に追われるサイクルで終わってしまいます

自分が何に向いているか？
何を一生の仕事にしたいか？
どんな人と相性がいいのか？

など自分を知っていないと、ゼロスタートで続けるサイクルに変わっていきません

結果を出したいのに出せない
行動したいのに、できていない
やる気が出ない
仕事が好きじゃない
失恋する

どれも感情が根本の原因なのです

だから、低い感情のまま、行動してもいい結果を出せることはないのです
低い感情では、続けることもできないし、行動を始めるスタートもしない
続けられないから、結果もでるわけがないのです

あなたが続けたい行動は何か？
自分の傾向パターンに気付いていくから、低い感情を高い感情に変えていくから、行動や結果も変わっていけます
まずは、あなたがどんな行動ができていないか？
どんな結果を出したくて、そのためにどんな行動をしているか？
できない理由は、感情は何か？
根本を探っていきます

タイムスケジュール
仕事、お金、恋愛結婚などのストレスになっている行動、結果を書き出していきます
出したい結果と感情、行動のギャップを分析していきます

2時間参加でのメリット
できなかった行動、でなかった結果が変わります
すぐに実践できる対策、改善策が見つかります
やる気、自信、計画性が着実に上がります
今までのゼロスタートサイクル、続かないサイクルが変わります
仕事、お金、恋愛結婚の出会いや結果が変わっていきます
