おまけ

高専付近の飲食店の一覧を載せておきますので、昼食なににしよう、と迷ったら参考にしてください。
・喫茶店 やま：正門を出てすぐ
・中華料理屋 金山：正門を出てすぐ。ちなみに懇親会も中華です。
・喫茶店 YU-YU：南門を出て左、すぐの交差点を右折。ご飯ものは少ないかもです。
・モレラ岐阜：正門から出て北へ徒歩10～15分程度。フードコートやレストラン街有。

正門から出てモレラ岐阜方面(北)へ15分程度行くと157号線沿いにコンビニ(ファミリーマート)やゲンキーもあります。当日晴れていれば、高専敷地内の野外であればご飯が食べられます。
