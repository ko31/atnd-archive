【OOPJogとは？】
世間のジョギングブームの波に便乗して、2009年の春頃に始まったジョギングイベントの一つです。
有志を募って続けにくいジョギングをしようというところまではよくある話なのですが、参加者の母体が主にIT技術者のため、OOP(オブジェクト指向)をはじめとした、技術的世間話をしながらジョグろうというノリになっております。
通常のOOPJogは主に首都圏在住者を対象としており、ジョギングの場所も通例、皇居の周りとなっていますが、今回はRubyKaigi2010開催に合わせ、つくばの街で開催することとなりました。

about “OOPJog”
Since 2009 spring, OOPJog start in Tokyo. This event is JOGGING around the Japanese inperial palece.
in this event, Not only jogging but TALKING ABOUT PROGRAMMING. Because main atnder of this event is almost Programmer.
Now, in my own way, I open this event in RubyKaigi2010 in Tsukuba.

【走るの？】
参加者が普段、日中座りっぱなしのプログラマー中心であるため、決して体力があるとは言えません。そのため、「走る」というよりは小走り、いわゆる「ジョギング」のスピードよりもやや遅めのぬるま湯設定となっております。
また、参加者の希望によってはいっそ「歩く」こともあり、これをOOPWalkといいます。

run?
No. Speed is so slow. Because I am lazy and laziness is virtue.

【格好は？】
ジョギングというには憚れますので、全身サイボーグのごとき高価なジョギング装備をする必要はありません。が、あくまでちょっと走りますし、涼しい朝方といっても夏なので、汗も大瀑布です。よって着替えられる「短パン」「Tシャツ」、また足を痛めないように「履き慣れたスニーカー」（もちろんランニングシューズがあれば最高ですが）の３種の神器を推奨しております。

prepare clothes?
You should have T shirt, short Pants, sneakers. It is not necessarily to prepare Hard-core-jogging-suits.

【RubyKaigiに影響はない？】
開始は早朝の7:30開始を予定しています。走行時間はどんなに多く見積もっても45分以内ですので、3日目の最初のプログラムにはホテルに帰ってシャワーを浴びて荷物を取ってチェックアウトする程度には間に合うでしょう。

be late for first program of RubyKaigi?
Don’t worry. In longest case, this event takes 45 minutes.

【3日目なんだけど？】
主催者も懇親会でハッスルした後、どこかのドンチャン騒ぎに合流し、泥酔しながらRejectKaigiの資料を作って、限界まで自分を試すつもりですので、大丈夫です！

be tired?
only 5km. If you don’t get excersise recently, you will conquer 5km maybe.

【どれくらい走るの？】
コース設定を「キョリ測」というサービスを利用し、作成してみました。→コースを見る(SEE THE COURSE OF JOGGING)
会場の「つくば国際会議場」から、市街地から公園の中を走り抜ける、爽快感溢れる魅力的なコース設定を行いました。
だいたい5kmくらいを予定しており、慣れた人なら25分くらい、まったくの久しぶり、ゆっくりのペースですと多分45分程度の行程となります。
イベント主催者の経験則からいきますと、普段身体がなまりまくっている、という方で無茶しないでゆっくり走れる限界値が5kmくらいであり、走ってる最中こそしんどいものも、程よい疲労感と、満足感も得られる距離設定だと思っています。
また、折り返し地点となる公園の手前で、一周しないで元来た道を引き返すという避難ルートも用意されており、その場合は3km未満となりますので、急激な運動による「俺はもうダメだ、みんなと一緒にこの先には行けない…」という状況もフォローしております。

course
5km around the hall of RubyKaigi2010. => SEE THE COURSE OF JOGGING
START 7:30a.m 8/29 Sunday morning.
From in front of hall (at side of the street).
I wear T shirt printed O’rilley. You find me.

【なにを話すの？】
OOPJog中に主に話されるテーマを列挙しますと、「OOPとは何か」「より良きプログラマとして生きていくためには」「Rubyの内部実装」「(会社|業界|社会|世界)が如何に自分に優しくないか」「彼女ができないことについて」などです。

What is we are taking about?
We talk about Passionate. Passion defeat dirty code around us and make us happy.
