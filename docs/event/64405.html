第22回 モトスポーツランドしどきをレンタルバイクで走ろうの会（しどきの会）

主催：TEAMまえのめり

  • 開催日：2015年4月25日（土）※雨天中止
  • 会場：モトスポーツランドしどき

モトスポーツランドしどきを貸し切りにして、初心者も気兼ねなく本コースをのんびり楽しんじゃおうという企画です。

  • バイクに乗ったことがないが興味がある
  • バイクは乗るけれどオフロードの経験はない
  • オフロードの経験はあるけれど自前のバイクはない

といった方々に気軽に参加して頂ける会です。

モトスポーツランドしどきについて

モトスポーツランドしどきは「初心者及び成熟した方々のフリーライドを楽しむコース」として運営されています。
そのため「初心者優先」が徹底されているコースです。
コースは起伏に富み、まるで自分で操縦するジェットコースターのようです。

そして最大の特徴は着の身着のままフルレンタルでモトクロスを体験できることにあります。
レンタルバイクの種類も豊富で、レベルに合わせて車両を選ぶことができます。
装備のサイズも充実しているので、大人から子供まで全装備レンタル可能です。

費用

仮で15000円を設定していますが、基本的にはしどきのサイトに掲載されている費用がベースとなります。
レンタルバイク、レンタル装備、走行料の合算が当日の費用となります。
費用は下記考慮の上で調整の後、追ってご連絡致します。

- 昼食などの手間賃を考慮してキリの良い費用にさせて頂くかもしれません。
- 当日はMFJインストラクターを招聘する予定です。インストラクターによるレクチャーを受講したい方は別途費用をご相談させて頂くかもしれませんので、こちらもご了承下さい。

【2015.4.20 追記】
参加費は、バイクレンタル料＋装備レンタル料＋走行料の合算の100の位を切り上げた金額とさせて頂きます。

例１）TT-R125LWレンタル、装備フルレンタルの場合
　　　　6,480円＋4,200円＋3,200円＝13,880円　→　参加費 14,000円

例２）バイク持ち込み走行の場合
　　　　3,500円　→　参加費 4,000円

参考情報（レンタルマシンの個人インプレ）

  • PW80
    自動遠心クラッチの2stバイク。
    子供用だけど大人も乗れるかと。

  • CRF70
    自動遠心クラッチなのでバイクに乗ったことがない人はこれかと。
    小さいので身長160cmくらいの人までですね。あとGPコースには入れません。

  • KLX110
    横置きエンジンのミニモト。ミニコースで走ると楽しい！
    走れるようになるとバンクを使って身体を地面と水平にして走ることができます。

  • CRF125F
    セル付きの最高に乗りやすいファンバイクです。すごく素直な操縦性のマシンです。
    XR100から続く乗りやすさで、上級者もこのマシンで練習しているくらい素性のいいマシンです。

  • TT-R125
    初心者に最適です。僕はこのバイクが好き過ぎて自分で手に入れたことがあるほどです。
    軽くて足もよく、ギャップ程度のジャンプなら難なくこなします。
    CRF125Fと並んで初心者に超絶オススメの一台です。

  • YZ85L
    ものすごくピーキーで面白いですが、パワーバンドを維持できないとまともに走りません。
    僕の体格(179cm 67kg)だと非常に不安定です。
    それ故バランス感覚が培われると思い、自分でKX85-Ⅱを買ってしまいました。

  • CRF150RII
    YZ85のサイズと同じですが、4st 150ccでYZ85とは比べ物にならない乗りやすさ。
    パワーバンドも広くトルクもそこそこあってめちゃくちゃ面白いです。
    オフ経験者の女性や小柄な男性にベストマッチです。

  • YZ125
    オフ経験者、少なくとも1度は草レースに出たことがある人推奨です。
    YZ85よりパワーバンドは広いですがそれでも狭いです。
    軽くてバイクが前に出るので体が置いていかれやすく、アクセル全開固定で吹っ飛ぶことも。
    車格がフルサイズで安定しているので、足が届く人には安心感があります。

  • YZ250F
    パワー、トルク共に強烈。オンロードの感覚で言うと600のSSです。
    オフ経験者、少なくとも1度は草レースに出たことがある人推奨です。
    フルサイズなので安定していますが、サスが固めなので経験者じゃないと乗りにくいかも。
    ハンドリングは安定しています。

  • 250SX-F
    KTMの4st 250ccモトクロッサー。セルつき！素敵！
    YZ250Fと比べるとエンジンがより元気でステアリングがかなりクイックです。
    フロントが振られる傾向があるように感じます。

  • WR250R
    市販車最強トレール車。YZ250Fと比べると重いけど同じくらい加速します。
    しかもセルつき！フルサイズ初心者はコレがオススメです。

  • セロー250
    ゲロ最強トレール車。ギヤ比が物凄く低速に振ってあるのでエンストしにくいです。
    WR250Rくらい重いけど、乗ったら大好きになる乗りやすさです。
