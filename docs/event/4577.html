description

この勉強会では、TracLightningを題材としてプロジェクトにおける開発や課題の管理を効率的に行う方法について勉強していきます。
今のところ以下のような内容を考えています。


  1. Subversionによる開発コード・ドキュメントの管理効率化

  2. TracのWikiによる情報共有・ノウハウの蓄積

  3. Tracのチケットシステムによる課題管理の効率化

  4. Subversion と Trac の連携による情報の整理

  5. Trac プラグインによるプロジェクトに特化した開発環境の整備


今回の勉強会は「Trac プラグインによるプロジェクトに特化した開発環境の整備」です。
今回がこの勉強会の一応の最終回ということになります。


初めて参加する方は、TracLightningをあらかじめインストールしてきてください。


現在参加費は一人 500円 を予定しております。（多少増減する可能性があります）


持ち物ですが、実際に動かしながら体感して勉強した方がわかりやすいと思うので、なるべくPCをお持ちください。
（同時にコンセントの数が不足することが予想されるため、できれば電源タップもお持ちいただけると助かります）


また、当日の勉強会の開催場所ではネットワーク環境はありませんので必要な方はご自身でネットワークにつなぐためのイーモバ端末等をお持込ください。

また、第一回から第五回までの勉強会の資料もURLのサイトで公開していますので、復習したい方や今までどんなことをやったのかを知りたい方はどうぞ！
