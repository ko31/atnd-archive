description

「3Dプリンターで自作のキャラクターグッズを作って販売したい」と考える人向けの勉強会です。
特に漫画・アニメ作品の2次創作に3Dプリンターを使ってみたい人は必見です。

本勉強会では3DCGに興味のある人が、最短で自作のグッズを即売会で販売できる方法について考え、必要な知識を共有します。

但し実習方式ではなく3Dプリンター出力のデモのみ行ないます。
3DCGツールの使い方を知らない人でも歓迎します。

でも3Dプリンターで一味違うグッズを作りたい人は必見です！
