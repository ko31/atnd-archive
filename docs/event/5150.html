description

お料理イベント新企画！野菜ソムリエの講師の方に美味しい野菜の調理法・食べ方を習いながら恋活をしましょう。ペア・グループで共同作業をして頂いた後は美味しいお料理を食べながらの交流会。席替えも交えながらたくさんの異性の方とお話できるチャンスをお作り致します。普段何気なく食べている野菜も実は奥が深いかもしれまんね!～講師～宮崎木綿子（みやざきゆうこ）
野菜ソムリエ【日本野菜ソムリエ協会公認】家族の健康を考え勉強をスタート。野菜や果物の奥の深さに感激し野菜ソムリエの資格を取得。現在は「野菜や果物をおいしく食べてもらう方法を広く知ってもらい、野菜果物を食べる事で健康増進につながる事を知ってもらいたい」との事で官公庁関係の依頼で、小学校で食育授業を行う等、レシピ提案などをしながら幅広く食育活動を行っている。
公認業務：　ベジフルティーチャー
（特定非営利活動法人　青果物健康推進協会-ベジフルセブン）
