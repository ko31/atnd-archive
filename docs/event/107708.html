JR千葉駅の駅ビルに出来たビアガーデンでハースストーンの炉端の集いを兼ねた呑み会を開催します。

日時:2019年8月10日（土）
14:30〜17:30

14:15　JR千葉駅改札集合

予算:1人5000円　3時間制　食べ・飲み放題となります

会場:ペリエ千葉５F
JR千葉駅の駅ビルです

砂漠のウルドゥムのよりも暑い夏、潤すには酒は欠かせない！熱さの止まぬ「燃えよ！炉端の集い！」にあやかって、悪党らしく酒池肉林の宴といきましょう！

定員:7人（主催含めて8人）

当炉端の集いはキャンペーンの支援を受けており、景品の提供・集合写真の撮影を行います。撮影された写真はBlizzard Entertainment社とその関連会社の広報活動に使用される可能性があります。本イベントへの入場を以って、上記内容にご同意頂いたものといたします。
またヒーロースキン「ネムシー」の取得も可能となっております。

注意:酒類の提供・入場者全員同じコースでの入店となります。未成年の方の参加はご遠慮願います。
