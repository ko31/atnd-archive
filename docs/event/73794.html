機械学習を始めよう！

Azure Machine Learningを使いこなせるようになろう！

目的・目標

本セミナーでは、機械学習実行基盤サービスである「Azure Machine Learning (以下、AzureML)」の活用方法を、演習を通して学びます。合わせて機械学習の基本についても学習します。

受講後、参加者は以下ができるようになります。

  • 機械学習の基本的な知識の習得
  • AzureML実行環境の作成
  • AzureMLを用いた、統計分類/回帰分析/テキスト分析/リコメンドエンジン作成など、様々な分析

こんな方におすすめ

  • Azure Machine Learning に興味はあるが、使ったことがない人
  • Azure Machine Learning を使ったことはあるが、リコメンドエンジン作成などの深い使い方はしたことがない人
  • 機械学習に興味があるが、実際の実現方法がわからない人
