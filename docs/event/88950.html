description

それぞれが自分の勉強する為に必要なものを持ってきて、自分の勉強をもくもくとする会です。

一人で家で勉強しても寂しいしはかどらないという方はぜひご参加ください。

雑談しながら進めてもらって大丈夫です。

※初回なので、いろいろ不備があるかと思いますが、怒らないでください。

設備

電源はありますが、数に限りがありますので譲り合って使ってください。

2時間程度ですので、できれば十分な充電をしてきてください。

Wi-fi環境はありますが、電波が弱く繋がらない場合もあるので、必須な方はご自身でモバイルルーターなどご用意ください。

自習内容

Web＆プログラミングと銘打っていますが、PHP、Java、Javascript、HTML、CSS、ホームページ制作、MySQL、サーバー環境構築、データベース、C言語、Unity、github、WordPress、ブログ更新、アフィリエイトサイト、ECサイト、サイト運営業務、何をしても構いません。

全く関係ない資格試験勉強でも、絵の勉強でも、なんでも可です。

但し、周囲を汚す可能性があることや、強い臭いなどで迷惑をかけることはご遠慮ください。

持ち物

自分が勉強するのに必要なもの

電源・Wi-fiはありますが、最悪使えないことも想定して勉強できるようご準備ください

日時

2017年6月20日（火） 19：00～

場所

フレンドリンクメンバーズオフィス「セミナールーム」

東京都新宿区四谷1-9 天龍ビル 6階

定員

10名程度（席に余裕はあるので増員は可能）

費用

無料

スケジュール

19：00～　開場（早く来ても入れません）

21：00　解散

飲食

食事不可

ドリンクはフタ付きの飲み物のみ（会場に自販機あります）

その他

定員に余裕がある限り、当日申込み＆参加も歓迎です。急に時間が空いた場合などにどうぞ。

主催者は、主にPHPを使ってWebサイト製作をしています。自分の勉強があるので長い時間は取れませんが、簡単な質問程度なら答えられます。他にも参加者同士でわからないところを聞き合えると素敵だと考えています。
