description

　庄内経営者塾は、東北公益文科大学が文部科学省から採択を受けた「地（知）の拠点整備事業」のひとつの柱として実施する起業家育成講座です。本学学生と地域の若者を受講生として迎え、地元企業の経営者が定期的に登壇。指導教員と起業家ネットワークの支援を受けて、受講生がビジネスアイデアをプラン化し、スタートアップまでつながるよう、支援をしていきます。
　今回は、地元経営者によるセミナーの第一弾。当塾塾長であり、本学理事長でもある平田牧場会長新田嘉一先生よりご講話をいただきます。

　学生は誰でも受講可能ですが、事前申し込みが必要です。５月２日（金）１７時までご連絡ください。

　セミナーの開始時間は１３時３０分からですが、１３時２０分には着席していただきます。間に合うように来場ください。

お問い合わせは、鎌田（本学准教授）までお願いします。
