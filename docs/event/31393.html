description

私たち、sopa.jp（ソパ　ドット　ジェイピー）は、
「想いと仕組みで社会変革を加速する」をミッションとして、
現在、「小１の壁」問題解決のために、
想いを持って学童保育を運営する起業家に対し、
開業・運営支援する仕組みを提供しています。

この度、学童保育事業の経営に興味・関心のある方を対象とした
「民間学童経営セミナー財務・人材編」を開催いたします。

12時30分から16時までの、セミナーの中で、
民間学童経営の全てのノウハウを惜しみなく公開いたします。
（8月25日から参加された方は、
次回、第3期民間学童経営セミナーの前編にご参加頂く事が可能です。）

■ 学童保育を開業したけれど集客に苦しんでいる方
■ 新店舗を開発する上で、どの様な地域・業態にすべきか迷っている方
■ もっと効率の良い運営の方法があるのではと悩んでいる方

にとって、申し込み頂く価値のあるものになっています。
是非、詳細を一読頂ければと思います。

ーーーーーーーーーーーーーーーーーーーーーーーーーー
セミナーでは日本随一の学童保育のノウハウが学べます
ーーーーーーーーーーーーーーーーーーーーーーーーーー
セミナーに参加して本当に学童保育のノウハウを学べるの？
もしかするとそう思われる方もいらっしゃるかもしれません。

私たちのノウハウは、
学童保育の民営化・チェーン化に初めて成功した起業家や
現役学童保育経営者のアドバイスを受けながら、
学童保育の業態や地域の特性も考慮したオリジナルのもので、
日本随一のノウハウだと自負しています。

ーーーーーーーーーーーーーーーーーーーーーーーーーーー
「小１の壁」の早期解決のためノウハウを出し切ります
ーーーーーーーーーーーーーーーーーーーーーーーーーーー

また、なぜ、そこまでしてノウハウを惜しみなく教えてられるの？
もしかするとそう思われる方もいらっしゃるかもしれません。

私たちの最大目標は、「小１の壁」問題をできる限り早く解決する事です。

そのためには、今保有しているノウハウを使って自らが
事業者として運営するよりも、
想いを持つ起業家にそのノウハウを使って
学童保育を広めてもらう方が理にかなっていると私たちは考えています。

そこで、私たちは、直接学童保育を
事業運営しないスタンスをとっています。
自分たちで事業を行っていないからこそ、
学童保育の他の事業者と競争ではなく共創の関係を築く事ができ、
ノウハウの出し惜しみをする必要がなくなります。

ーーーーーーーーーーーーーーーーーーーーーーーーーー
セミナー料を上回る価値をあなたに提供します
ーーーーーーーーーーーーーーーーーーーーーーーーーー

今回の学童保育経営セミナーでは、

・収益化の鍵を握る集客のコツ。
・送迎を使った学童保育のバリューアップの仕組み
・学童保育のアセットを活かした新たな収益源の獲得
・新店舗の業態ごとの必要投資額、回収額、期待収益の見込み

など既に経営している方にも納得戴けれる内容になっています。

今回のセミナーの値段は、個人５万円、法人１０万円です。
（個人向けには特別割引制度あり。別途詳細に記載）

あなたは、この金額を高いと思いますか？安いと思いますか？

例えば、集客のコツをつかみ生徒が１人でも増えたらどうでしょうか？
送迎によって生徒が増えたり、単価が数千円あがったらどうでしょう？

それだけでセミナー代を上回る価値を得ることができます。

実際、これまで受講してきた第0期、第１期生の方には、
本当にこの価格でここまでノウハウを公開していいのですか？
と大変喜んでいただいています。

更に、今回は特典として、経営相談を無料でおつけいたします。
（これは 本来１回５万〜１０万円を頂いているものです）

セミナーに興味を持った方は、
是非お申し込みください。
