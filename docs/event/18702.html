7ヶ月ぶりにCSS Holicが帰ってきました！（ほんとすんません）

今回から第3土曜日に定期的に開催する方向になりました。
また内容についても「明日すぐに役に立つ勉強会」をベースに、
「持ち寄り質問コーナー」を最後の1時間に必ず設ける予定です。

今まで通り、普段思っている悩みや、Holicだからこそ聞ける今更な質問などなど、なんでもいいのでぜひ持ち寄ってください。

例）デザイナーからこういう難題を出されたんですが、みなさんはどう実装しますか？

[2011.09.12 追記]
今回のお題が決まりました。題して
「激動のHTML5時代 〜これからのMEの在り方を探る〜」
です。（大いに盛ってますｗ）

ここ半年でME界隈での状況がかなり一変したなーと感じています。
そこで、今後習得すべきスキルや、この先どういうことをしていったらいいか、
なんていう疑問や不安など、そうしたわだかまりを解消しよう、というのが目的です。

なので、そうした質問や「みんなどうしてる？」的な質問を事前に用意してもらえるとうれしいです。
今回は誰かにメインで話してもらうのではなく、みんなで色々話す、という流れになります。
