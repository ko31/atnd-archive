1年ぶりみんな元気？ 豚○オフ2013の告知です！

初参加の方でも楽しく参加して食事していただけるイベントです。
丸焼きはインパクトありますが、いつも皆さん楽しんでいます。（子供も）
初参加の方が増え、豚◯の◯が広まればいいな～

前回好評だった鹿肉料理もあり！（丸焼きではありません）

参加者80人以上集まったら豚ちゃん2匹だよ！



場所はまたまた
淀川河川公園 西中島地区　バーベキュー指定区域で！

バーベキュー指定区域


下記チェックしておいてね！　

  • 飲み物・アルコール類は各自で持ち込みください。
  • 基本的にゴミは各自でお持ち帰りください。
  • 下に座れるように大きめのブルーシートはこちらで用意しますが、各自持って来ていただけると助かります。
  • 調理や作業用のテーブルなどは用意しますが基本何もないグランドなので、イスやテーブルが必要な方は各自でご用意下さい。
  • 紙コップや紙皿なども用意しますが、出来ればマイカップ・マイお皿・マイお箸を持ってきていただけるとエコでええですね。
  • トイレは仮設トイレがあります。
  • 水道は水飲み場程度な感じです。
  • 当日朝から場所取りとかしてくれるナイスな方がいたら嬉しいなーｗ
