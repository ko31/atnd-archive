「やればいいと分かっているのに動けてない」

「もっと伸び伸び仕事をしたいのに思い通りにできない」

そんな状況を変えたいあなたへ。

 

心とカラダと思考のリキミをはずし、

自分のキャパシティーを広げる

『リラックスモチベーション』体験会です。

 

■「時間がない！」と焦ってばかり。。。

■やることが多くてパニック！何から手を付けていいか。。。

■うまくいかせたいと考えてばかりで行動できていない

■いろんな人にいろんな事を言われて揺らいでしまう

■アドバイスを受けた直後は「やるぞ！」と思うのに結局実行できずじまい

 

1つでも当てはまった方、オススメです！！

 

 

「やらなきゃ！」「やろう！」と思ってもできない。

 

目標達成するには、夢を叶えるには

「行動が大事」というけれど

実際思い通りに行かないって多くないですか？

 


私は、これまで10年以上

２,５００人以上の就職支援をしてきました。

 

その中で、まさに「やらなきゃ！でも、できない」を「できた」に変えてきました。

 

なかには、

1年以上失業していたのに年収アップ転職を叶えた人や

ずっと躊躇していた有料カウンセラーとしてデビュー、起業に踏み出した方もいます。

 

 

2500人以上の方を観察していて

 

うまくいっていない時は

「やらなきゃ！」「がんばろう！」と思うほどにリキミが出て

自分の良い部分が全く出せていないことに気付きました。

 

そのリキミは、

体だけでなく心や思考にも大きく影響をしているのです。

 

キャリアカウンセリングに加えて

自分に試して効果のあったセラピーテクニックを取り入れると

メンタルも安定し、

柔軟性が増して行動が加速すると気付きました。

 

絡み合った「心とカラダと思考」の

３つのリキミを上手にはずせると

今以上に、本来持っている力を発揮、活用できるように変わるのです。

 

 

真面目にがんばることを知っているあなたに

リキミをはずすという新しいアプローチで

まだ使いこなせていない力を発揮できるようになっていただきたい。

 

そう思って、脳科学や発達学、セラピーテクニックと

10年のキャリアカウンセリング経験からオリジナルの

リラックスモチベーションにまとめました。

これまで無料だったお客様も快く有料に移行してくださり順調に進んでいます。

自分にとって要不要の判断が早くつき悩まなくなりました。
