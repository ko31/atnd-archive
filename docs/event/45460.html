【AID勉強会_Unity祭】

「Unity4.3」 ＆ 「AID Unityプラグイン」 リリース記念。ユニティ・テクノロジーズ・ジャパン様、ポケットクエリーズ様、日本マイクロソフト様に講演頂きます！

～Unity4.3詳細・開発事例・マネタイズに関して～

-

【参考】
・ Unity、2D 開発ツール群を含む 4.3 アップデートをリリース
・ アプリ向けの全面型アドネットワーク『AID』、Unity プラグインリリース

概要

AID定期勉強会は、無料アプリにおける3大要素
「開発」「DL拡大」「マネタイズ」について、情報提供を行っていく場です。

例えば、以下のような点について各専門家の方にお話頂きます。
『もっと効率的にアプリ開発出来ないの？』
『お金をかけずにどうやってダウンロードを増やすの？』
『収益を上げる為に気を付ける点は？』

定期開催致しますが、登壇者の方や発表内容は随時変更致しますので、是非ご都合が宜しい際にご参加下さいませ。

【１部】Unity4.3で出来ること！

「Unity4.3で何ができるようになったの？」「何が凄いの？」という点を実際にUnityの中の方、ユニティ・テクノロジーズ・ジャパンの鎌田様にご説明頂きます！

Unity Technologies Japan
コミュニティエンジニア　鎌田 泰行　様

Web開発会社で勤務をしながら個人的にUnityに興味を持ち、コミュニティ活動を積極的に行う。その後、Unity Technologies Japanに転職。コミュニティエンジニアとしてWebサイトの管理やコミュニティイベントの運営などを行っている。日本Androidの会秋葉原支部Unity部の庶務兼部長。

-

【２部】Unityアプリ開発事例！

ゲームロジック作成/ユーザー分析の為のUnityとサーバー通信の実装方法など、実際の開発アプリを事例にご紹介頂きます！

株式会社ポケット・クエリーズ
代表取締役　佐々木 宣彦　様

-

【３部】Unity on Windows 8.1

マイクロソフトの最新プラットフォーム Windows 8.1は、タブレットに最適な「Windowsストアアプリ」を実行できます。このセッションでは、Unity 4.3を利用してWindowsストアアプリを開発する流れ、Windowsランタイムを呼び出すプラグインの開発方法などをご紹介します。

日本マイクロソフト株式会社
デベロッパー＆プラットフォーム統括本部　エバンジェリスト　大西　彰　様

C言語のプログラマから出発して、SE、コンサルタント、サポート、プリセールス、アーキテクトなどの経験を重ね、いくつかの挫折を乗り越えながら、IT業界に身を置いて25年目。2005年9月にマイクロソフトに入社。マイクロソフトでは メディアを使ったマーケティング、Silverlightテクニカルマネージャとして著名な動画配信サービスのSilverlight 対応プロジェクトを支援、Silverlight大全を執筆、Windows Phoneのエバンジェリストを経て、現在 Windows 8のエバンジェリストを担当。米国マイクロソフト本社のUnity担当と連携しながら、Unity on Windows 8.1を推進中。趣味は当てのない散歩。座右の銘「踏まれてもなお立ち上がる道の草」
