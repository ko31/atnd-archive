パーティー概要
さぁ、明日からいよいよ12月に突入ですね！
クリスマスを一緒に過ごすお相手はお決まりですか？？

昨年12月にラウンジ激戦区の六本木に堂々とOPENした『 ESPRIT LOUNGE 』がついに 1周年！
なんと本日は土屋アンナさんがゲストで登場いたします！
東京を代表する最新のトレンドを取り入れ今宵も盛り上がり必須でGO!
（※二部制）

そして、気になる今宵のコンセプトはお客様の満足度をさらに向上させるため
二部制にチャレンジ！
序盤（第一部）はなんと
「 Pre Christmas☆カジュアルセレブ異業種交流PARTY 」
◇男女共に本気でオシャレをした方のみとさせて頂きます。
（※自己判断でもちろんOK！）
ということは、東京のオシャレ男子とオシャレ女子が一同に集うわけです！

普段は東京の第一線で働くビジネスマンVSオシャレやトレンドには敏感なステキ女子。
当然素敵なお相手が見つかりますよね！？

そして、後半戦（第二部）はなんと1,000名を超えるお客様が来店予定!!!
（※e-venz紹介であれば一部からそのまま居残れます！）

トレンドの映像・照明・音楽設備に魅了されながら、
たくさんの人！人！人！で集まって理想のお相手探しをどうぞ！！
＜以下会場（ESPRIT LOUNGE）紹介＞
※六本木駅より徒歩1分の好立地。
エントランスに一歩踏み出せば、
2フロア120坪の空間に飛び込むワイドLEDパネル！

直径4mの吹抜けには様々な角度から、約1,000本の光を使用したこだわりの演出。
そして、国内では入手することが不可能のDJブースは最新のALLMIXをセレクトし、
コミュニケーションを抜群に後押しするBGMがさりげなく会話の隙間に入り込みます♪♪
