「成約率９８％の秘訣」作家・営業コンサルタントの【和田裕美先生】をゲスト講師にむかえ、「バカの壁」で知られる東京大学名誉教授 【養老孟司先生】を招き、

【社会に愛される企業づくり / 顧客に愛される営業術】をテーマに基調講演、ライブトークが開催されます。
昨年は２３０名のご参加いただき、今年は千代田区「全電通労働会館」にて開催致します。

2017年11月に新刊「遺言」を完全書き下ろしを出版した養老先生に【やりがい、生きがい、自己実現】の基調講演。
成約率９８％の秘訣の著者、作家・営業コンサルタントの和田裕美先生より【顧客に愛される営業術】について講演していただきます。

さらに事前質問や会場アンケートを通して参加型のライブトークを開催し、より実践的な愛される営業術を学びます。

閉会後の懇親会では養老孟司先生と和田裕美先生にもご参加いただきます。
