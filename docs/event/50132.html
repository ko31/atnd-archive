description

受講対象・レベル
生産管理（生産計画の策定）やサプライチェーン管理の業務に関わる方など、初中級者向け。
習得できる知識
生産管理業務の基礎。
自動車メーカーの生産管理、受発注管理の概要。
趣旨
NPW（Nissan Production Way）と呼ばれる日産自動車の生産システムの特徴について、主に生産管理およびサプライチェーン管理の側面から解説します。トヨタやホンダ等、他の自動車メーカーとの比較考察も行う予定です。それにより、各メーカーの生産システムの違いが浮き彫りになると同時に、自動車の受注生産が抱える課題も見えてきます。
プログラム
１．日産生産システムと受注生産

　　　2-1-2 月間生産計画
　　　2-1-3 週間生産計画
　　　2-1-4 生産日程計画
　　2-2　部品調達およびサプライヤーとの連携
　　　2-2-1 発注内示
　　　2-2-2 ロット・アクチュアル順序納入
　　　2-2-3 シンクロ納入
３．NPWにおける受注生産の実態と課題
　　3-1　受注生産の内実
　　3-2 トヨタ、ホンダ、三菱自動車の事例
　　3-3　各社の生産システム比較
　　3-4 自動車の受注生産が抱える課題
