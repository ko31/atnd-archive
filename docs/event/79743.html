申し込み方法

ATNDだけでなく、
フォーム にも登録してください。

締め切り 9/5(月)

概要

会津大学に他大学の方々を呼び、オンサイトコンテストを開催します。
この合宿を機会に他大学の人との交流を深めましょう！
コンテストはAOJを使いオープンで行いますので、会場に来られない人でもオンラインで参加できます！
http://judge.u-aizu.ac.jp/onlinejudge/contest.jsp

場所

会津大学の研究棟1階 演習室2(std2)で行います。キャンパスマップは こちら から確認できます。演習室2がある研究棟はバス停付近の大学正門からまっすぐ奥に進み、体育館から右に曲がれば着きます。初日はバス停に誘導スタッフを用意する予定です。祝日のため学食、売店は利用できません。(参加費500円はお菓子代です)
アクセス方法

環境

演習室2のマシンのOSはSunOS 5.11で、キーボードはUS配列です。その他無線LANも使えるので持参のノートPCも使えます。持参のキーボードは使えるかどうかは保証し兼ねます。

日程

3日間で計3回のコンテストを行います。
この日程は多少変更される場合があります。
オンサイトでは3人でチームを組んでもらいます。

  • 1日目 : 9/17(土) 立命館大学セット
