description

【こんな方にお勧め】
・知的財産の理解不足により、いつか大きなトラブルが生じないか不安に感じる、あるいは、トラブルに巻き込まれたことがある。
・知的財産に関するセミナーは、難解なものが多く、どうしても敬遠してしまう。
・せっかくの特許情報を「営業活動」に活用したいが、やり方がわからない。

【ポイント】
・トラブルに巻き込まれないために、特に押さえておきたい「重要ポイント」を強調して説明致します。
・初学者の方にも、イメージが湧きやすいように「ケーススタディ」を中心とした解説を行います。
・特許情報をどのように営業活動に活かすか、そのポイントについて具体例を交えつつ説明致します。

【セミナー受講の効果】
・知的財産に関わるトラブルに巻き込まれなくなる！
・ケース毎に注意すべきポイントを理解することができる！
・営業の話題作りとして「特許情報」を利用することができるようになる！

【プログラム】
１．はじめに
　1-1.特許情報が話題作りに大活躍
　1-2.特許に関する知識不足がトラブルのきっかけに
　1-3.知的財産権侵害で販売が中止に
　1-4.種々の知的財産権
　1-5.契約と不正競争防止法

２．権利侵害、本当に大丈夫？
　2-1.お客様の要望は断れない？
　2-2.他社の技術・デザインを真似するリスク
　2-3.雰囲気を似せるだけでもだめ？
　2-4.商品名やロゴマークに注意！
　2-5.お客様からの情報を使って商品開発？

３．権利侵害はこうして起こる
　3-1.その特許、権利は存在しますか？
　3-2.特許公報の情報、権利があるのはどれ？

４．その権利、本当に存在する？
　4-1.営業で機密情報が流出、そのリスクは？
　4-2.その仕様書、お客様に渡して大丈夫？
　4-3.パンフレットから情報が流出
　4-4.データで渡すときには特に要注意！

５．営業資料作成時の意外な盲点
　5-1.記事の利用に要注意
　5-2.営業資料と著作権
