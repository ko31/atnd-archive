過去の開催履歴とこれからの開催予定はこちら

PHPのプログラミングの書き方を勉強します

前回、中途半端な感じだった関数とクラスの使い方をより詳しく解説します。

対象者は『プログラム未経験者』です。

PHP 『 超 』 初心者勉強会は、プログラムが初めての方を対象にしています。

勉強会の進め方は、プロジェクターに映した画面を見ながら、一緒に操作を行って進めて行きます。

内容

  1. 前回の基礎編の簡単な復習
  2. PHPの関数の書き方
  3. PHPの関数の使い方
  4. クラスの書き方
  5. クラスの使い方
  6. 外部ファイルの読み込み
