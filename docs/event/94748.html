Windows 10向けに最適化されたアプリケーション開発を実践していますか？</23>

Windows 10がリリースされて一年以上が経過し、「Windows 10対応」のソフトウェアも多くリリースされています。しかし、Windows 10対応の実情を聞いてみると、最新OSでの動作環境を行ったのみであったり、ユーザーインターフェイスが旧来のままで対応を謳っていたりと、その実情はまちまちです。

今回、新規にWindows 10対応のアプリケーションを開発する方、従来のアプリケーション資産をWindows 10に適合させていきたいといった方を対象にWindows 10対応のソフトウェア開発手法を解説するセミナーを開催。最新のユーザーインターフェイスに適合した画面設計の方法やアプリケーションメンテナンスの方法、さらには、Visual Basicなどの旧資産を最新プラットフォームに適合できるようにコンバートする方法などを紹介します。
