description

企業での情報発信ツールとしてホームページは事業の基盤となっております。多くの企業や事業者がホームページを持っておりますが、地図情報などを本当に使いこなしているか、役に立っているのか、収益にフィードバックできているのか？、さまざま課題があります。　

そこで、Googleマップと連携し、不動産物件情報サイトを構築運営することで、最大限のメリット引き出せる、オープンソースCMSの仕組みを、ワークショップで学びます。　

CMSは、はじめての方、不動産ショップ経営層の方、Goolgeマップと連携した事業を検討している方などに最適です。
