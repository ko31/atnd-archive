★CyberAgentグループ、株式会社シロク主催★

日本にインターネットが普及し始めて、もう間もなくで20年が経とうとしています。

昨今はスマートフォンの台頭に伴い、
アプリやその周辺産業が活況を呈しておりますが、
いつの時代でも、ユーザーから人気な「WEBメディア」は存在しています。

メディアも特性がいくつかあり、「好かれるメディア」があれば
その一方で「嫌われるメディア」もあります。

本セミナーでは、実際にWEBメディアを運営されている、
最前線の方にご登壇頂き、各社の「メディアに対する考え方」や、
「自社メディアの運営方法やその特徴」について、具体的なケースをもとに
ご紹介頂きます。

※本セミナーは、Webサービス事業者の方等を中心としたセミナーになります。
※広告代理店様のご入場は固くお断りしております。
※名刺の無い方のご入場もお断りしております。名刺を2枚、ご持参下さいませ。
※イベント情報につきましては、随時更新させて頂きます。

◆こんな方にオススメ！

・WEBメディアやECサイトを運営しているけど、課題への対処方法が分からない
・最近自社オウンドメディアの運営担当になったけど、そもそもの考え方や運営方法がよく分からない
・自社メディア内で活用出来る、マーケティングの最新事例を知りたい
・実際の運営者から、ぶっちゃけ話をたくさん聞きたい
・メディア企業さんとの親睦、交流を深めたい

◆登壇企業

・株式会社産経デジタル
・C Channel株式会社
・株式会社サイゾー
・GMOアドパートナーズ株式会社（トークセッションのみ）

◆主催

株式会社シロク（株式会社サイバーエージェント子会社）

弊社シロクは、グロースハックという考え方を日本に定着させ、
アプリやWEB市場のサービス発展に貢献したいと考えております。
