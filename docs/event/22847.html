第20回XOOPS Cubeサタデーラボ + TOKYOPenリリースパーティー開催！

12月24日（土曜）16時00分より築地テラスにて、第20回XOOPS Cubeサタデーラボ + TOKYOPenリリースパーティーを開催いたします。

12月24日はXOOPS Cube LegacyをベースにRYUSにて開発されているTOKYOPen1.0のリリース日です！
そこで、築地テラスのワンフロアを貸し切り、TOKYOPenのリリースパーティーを開催します！
今回のサタデーラボは、普段の勉強会＋懇親会というスタイルでわなく、完全にパーティー形式になります。
パーティーでわいわい盛り上がりながら、合間にTOKYOPenのプレゼンも予定しています。
是非ご参加ください！

サタデーラボはじめての方へ

「XOOPS Cubeに関わる色々な人たちとの交流の場を作りたい」という気持ちから始まりました。

内容はXOOPSに関する勉強会＋懇親会（自由参加）です。勉強会ではXOOPSに関する発表を聞いたり、質疑応答の形の「発表形式」、持ち込んだパソコンで自分のサイトで困っていることやXOOPSにかんする色々な質問を行う「ワークショップ形式」などとしています。

かっちりとしたセミナー、勉強会ではなくXOOPSに関わるあらゆる方が楽しめるものにしてきました。初心者さんもベテランさんもどなたでもお気軽にご参加ください！
