description これまでのウェブマーケティングでは、ニーズが顕在化しているユーザーを刈り取る「獲得重視」型の手法が主流でした。

一方、近年注目されているコンテンツマーケティングは、潜在層のユーザーに対する訴求にも焦点をあてて将来の売上につなげる「顧客育成」が得意な手法です。
コンテンツマーケティングでは情報を探し求めているユーザーに対して、いかに良質でいかに適切なコンテンツを届けるかが重要なポイントとなっております。
そのままでは「顧客育成」型の手法であるがゆえに、売上に繋がるまでの時間がかかる場合もありますが、既存の施策と相乗効果を生みつつ運用することが可能です。

一方、コンテンツマーケティングにすでに取り組んでいる方のよくある失敗例として、ノウハウ不足のためにアクセスが伸びなかったり、コンテンツ戦略が粗いためにアクセスが伸びても売り上げにつながらなかったりといった声も聞かれます。

今回のセミナーでは、コンテンツマーケティングの基本的な考え方をお話しした上で、CV獲得に向けてどのようなアクションをとっていけばよいのかまでをご紹介いたします。
