Webデザイナーなら１度はお世話になったことのあるWebデザインの
ギャラリーサイト「デザインリンクデータベース」の管理人とスキーマのコラボテー
マです！

今回のテーマはズバリ「自分イチオシのWebデザインを語る！」です。

自分の好きなデザインを持ち寄り、それについてみんなで話し合う！

スタイルはいつもどおりラフな感じですので、気張らずにご参加下さい！

一押しのデザインはないけど、是非勉強したいという方ももちろんOKです。
きっと新しい発見があること間違いない！

※自分の好きなデザインはプロジェクターで見ることができます。
開催日が近くなりましたら、スキーマより事前に好きなデザインの情報をお送りいた
だけるようご連絡致します。

自分の好きなデザインは特にないけど、勉強したいだけという方の参加も問題ござい
ません！
