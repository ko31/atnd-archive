大変申し訳ありませんが、諸事情により延期となりました。3月もしくは4月以降に再度日程を調整いたします。

OpenStreetMap( http://www.openstreetmap.org/ ) という、
Wikipediaのようにみんなで自由に使える地図データを作ろう！というプロジェクトがあります。

Web地図といえば真っ先に思い浮かぶものはGoogleマップですが、Googleマップには色々と制限があります。
また、通常の地図は数ヶ月周期で地図が更新されますが、
OpenStreetMapでは『ビルが建った』『仮設住宅ができた』などの変化を
リアルタイムに地図に更新することができます。

Googleマップをはじめとした他のWeb地図とOpenStreetMapの違い、ライセンス、編集方法、APIなどについてご紹介します。

OpenStreetMapに興味のある方であれば、誰でもご参加出来ます。
