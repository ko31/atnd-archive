IDA Demoを利用して、以下のことを試みる勉強会です。

・bootイメージ抜き出し
・kernelイメージとRAMDISKの分離・展開
・kernelイメージの解析
・initプログラムの解析
・libc.soなどdynamic link libraryの解析
・カーネルの解析（詳細編）
・bootloaderの解析
等々

以上の作業を通して、IDAの使い方、機能に慣れ親しんでいきましょう。
