概要（ローレイヤー勉強会とは）

ASM/Kernel/CPU辺りに関する勉強会です

C# が出て来て Web 2.0 が氾濫してもすべてのコンピューター技術を CPU レベルで理解しないと気がすまないアナタ。NIC や PCI のファームウェアで動作するバックドアを書いてしまったり、Intel CPU の脆弱性を発見してドキドキしているアナタ。MS VC++ や gcc のヘンなオプションを見付けて、ついつい深追いしてしまうアナタ。
コンピュータ ネットワークやセキュリティを中心に、いろいろな技術をとことんまで深追いしようというのが、ローレイヤー勉強会です。

ローレイヤーの技術を勉強中、研究中な方々との情報交換の場になればと思います
