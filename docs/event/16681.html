Varnishに関する話をしよう

6/16のVarnish3.0の正式リリースに合わせてVarnish勉強会を開催します。
日本語の情報に乏しく、国内での実践例があまり見つからないVarnishですが実はいくらかのウェブサービスでは活用されています。

記念すべき第1回目は、サービスへの導入例や導入失敗例などの発表を中心に国内のユーザの情報交換を行ってみたいと思います。

また、世界各地で行われるVarnish3.0リリースパーティのひとつとして勉強会後にビアバッシュを予定しています。
http://v3party.varnish-cache.org/
