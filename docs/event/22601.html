description

2012年１月８日に我が家で鍋パします！

場所は荻川駅の近くです。詳しい場所はDMにて教えます。

参加資格は私がフォローしている方なら誰でも！

前回の失敗を生かし、参加者全員が楽しめる会にしたいと思います。

開始は１８時で、一人一品何か持ち込んで来てもらいます。

会費は買出しに行って、その金額を参加人数で割ます！
ほとんどお酒の値段になると思うので　1000～1500円くらいだと思います。

車で来る方は、駐車場に限りがあるため、各自乗り合わせの上参加してもらえると家主は大変助かります。

最悪、駐車場借りますので、車で来る方は事前に申し出て下さい。

電車で来る方は荻川駅まで来たら、私が迎えに行きますので駅についたら連絡を下さい。事前に連絡先を教えておきます。

０次会について

鍋の開始時間は１８時ですが、鍋の仕込み、準備、買出しを手伝ってくれるメンバーも募集です。

１３時から作戦会議をするので、我が家に集まって下さい。

この鍋パは皆の協力なくしては出来ません。食器や鍋その他テーブルなど、足りないものがあると思うので、持ってる人に貸してもらうということもあります。その場合はどうか協力をお願いします。

あと、キッチン、トイレ、バスルームと自由に使ってもらって結構です！

前回はキッチン争奪戦が勃発しました。料理をする方は早めにやるか、他の人の手伝いなんかするとスムーズにいけるかと思います。

火の扱いだけは十分に注意してください。家は燃えても建て直せますが、人が燃えたら元には戻りません

Q＆A

Q.一人一品とあるんですが、なんでもいいんですか？

なんでもいいです。作って来てもらってもいいですし、気になるあのお店から買ってきてもいいです！

Q.忙しくて一品とか用意出来ません。手ぶらでもいいんでしょうか？

事情があって用意が出来なくてもＯＫです。強制ではないので無理して用意しなくて大丈夫です。ただ、それぞれが持ち寄ったら楽しいですよってことです。

Q.みたま邸で作るっていいの？

どうぞどうぞ、キッチンは自由に使って戴いて結構です。

Q.１８時とか開始早くて間に合わねぇよ！もっと遅くしろ！

お店ではないので、遅刻しても大丈夫です。

Q.泊まって行きたいんだけど、迷惑じゃない？

前回の鍋の時はほとんどが泊まって行きました。ただ布団がそんなにないので寝袋とかある人は用意しておくと安心して寝れます。基本雑魚寝です。

Q.１月とか寒すぎだろJK

床暖房があります

Q.途中で買出しとか行きたいんだけど大丈夫？

コンビニが歩いて３０秒ほどの所にあります。自由に行ってもらって構いません。ただ、買出しに行く際叫んだり近所の人に迷惑かけないでください

Q.０次会の時間早すぎじゃね？

前回の失敗のうちの一つで、３時間前から集まったのですが時間がほんとにもう本当に止まればいいのにな状態だったので、今回はちょっと早めにします。

Q.あたし新潟の人じゃないんだけど参加していいの？

どうぞどうぞ、フォローしてる方なら県外でも外人でも宇宙人でも。

Q.1月8日とか日曜なんだけど？

９日が成人の日で祝日なので比較的連休が取りやすい日かなっと！

Q.餃子食べたいよ。餃子

食べましょう。餃子＆ビール！
