第10回MMD伊香保温泉オフ ミクミクバスの搭乗希望を受け付けます！

ぷらんぶらんにて、第10回MMD伊香保温泉オフの開催予定が告知されました。どうしても足が確保できない方をお連れすべく、高田馬場よりミクミクバスを運行します。

出発場所について

日産レンタカー高田馬場駅前店より出発いたします。
→地図はこちらからもご覧いただけます

運行コースは、往路復路とも以下ルートの予定です。

  • 高田馬場駅←→関越自動車道練馬インターチェンジ←→上里サービスエリア←→現地

費用について

  • 以下の金額（目安）がかかります。これを同乗者全員での割り勘となります。
    1. レンタカー48時間（キャラバン）：14,547円
    2. 高速代金：8,560円
    3. ガソリン代：8,100円(@7km/l 換算)
    4. 任意保険：2,592円
      【合計[おおよそ]】33,799円

集合時間と出発時間

2016-10-29 10:45集合、11:00出発となります。

その他ご注意いただきたいこと

  • 参加者多数の場合はバスのサイズをマイクロバスに拡張、24人定員として、費用が
    変わる場合があります。予めご了承ください。
  • 基本的には往復のご搭乗となります。片道だけの場合には申込時にその旨コメントにご記入ください。
  • 出発時は最寄り駅までのお迎えはありません。出発場所にお集まりください。
  • 帰り道には大幅な帰路コースを外れない範囲で、最寄り駅での下車が可能です。申込時にその旨コメントにご記入ください。
  • 自動車運転免許をお持ちの方は、可能であればご持参ください（運転の交代を強制するものではありません）。なお、バスがマイクロバスに拡張された場合は、この限りではありません。
