時間セッションスピーカー
14:15-15:00受付　
15:00-15:15会場説明・注意事項等　
15:15-15:30Session01「絶望の国の起業とセーフティネット」
日本の起業活動は世界的に見てとても低調です。日本の起業環境を海外と比較しつつ、起業が低調な原因や、起業の抜け道を（出来るだけ）アカデミズムの観点から考察します。 絶望的な日本の現状を打開するための起業論です！		加々美 文康さん
15:30-15:45Session02「最近話題になったセキュリティ」
2013年もいろいろなセキュリティ事件/事故がありました。その中からクリエーターのみなさまに特に知っていただきたいものをピックアップして対策などをご紹介します。		株式会社インターネットイニシアティブ（IIJ）
ももい やすなりさん
15:45-16:00休憩（15分間）　
16:00-16:15Session03「ものづくりにかける時間 – 工数の見積もりと計画づくり」
半年間、"営業"の肩書を持って見積もりマシーン業やってました！
すべてのものづくりニストへ向けて、
・作業に見合った見積もり方
・見積もり精度を高めるための計画
・正確な見積もりを出すことの重要性
についてお話します。ウルトラハッピーな計画をつくりましょうー！		株式会社バイタリフィ
千葉 礼美さん
16:15-16:30Session04「たった500円でいいの！？ワンコインマーケット・ココナラ活用術！」
誕生日に何をプレゼントすればいいの…。パンフレットに挿絵イラスト入れたいんだけど自分じゃ描けないしプロにお願いする程のものでもない…。起業したいけどアイディアに自信がなくて誰かに相談したい…。そんなとき、たった500円で相談出来る人がいる、それがココナラです。いますぐ使える！ためになる！活用事例をご紹介します。		日本シーエイダブリュウ株式会社
原田 敦さん
16:30-16:45Session05 「高セキュリティ低コストでのサイト制作」
近年、サイトオーナー企業でのセキュリティへの意識が高まる一方で、制作のための予算はそこまで確保されていないのが現状です。
弊社のクラウドサービスSPIRALを活用して、高セキュリティのWebサイトを低コストで構築した例をWordPressとの連携事例と合わせてご紹介します。		株式会社パイプドビッツ
山本 和輝さん
16:45-17:00休憩（15分間）　
17:00-17:15Session06 「HTML5 時代に必要なスキル」
HTML5の時代が来ると囁かれて３〜４年立ちます。実際にHTML5の情報が満たされていくなか、制作者に求められる能力も変わろうとしています。これから新しく到来する時代に必要なスキル、そして目標を達成するための１つの考え方をお話させていただきます。		Microsoft MVP (Bing Maps Development)
デジタルハリウッド講師
山崎 大助さん
17:15-17:30Session07「RIAからSPAへ、HTML５が導く技術革新」
いま、なぜSingle-page Applicationなのでしょうか？歴史的背景やHTML5の技術進化を踏まえたSPAの概要とそのために必要となる技術についてお話します。今後、業務アプリケーションはSingle-page Applicationでの構築が当たり前になるでしょう。		 佐川 夫美雄さん
