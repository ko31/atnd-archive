講演内容

好評だった「1人でソーシャルゲームを作る肝」。

一本のゲーム企画ができあがるまでに必要なフェーズをすべてお伝えします！

田中一広（たなか かずひろ）
株式会社Wuah（ワー） 代表取締役社長

企画・設計・デザイン・ライティング・・・
コンテンツに必要な技術を全て持ち合わせた、マルチに活躍中の注目人物が登壇！！！

NTTドコモ公式サイトをはじめとして、モバイル公式サイトやモバイル広告サイトのプロデュース＆ディレクションを多数経験。
Webシステム、iPhoneアプリのシステム設計・プログラミングも実績を積み、システム設計を見越した企画が可能である。
さらにFLASHやIllustratorも使いこなし、FLASHコンテンツではShockWave iRiver特別賞アンド審査員賞を受賞。
モバイルWebゲームのシナリオライターも行い、モバイルサイトのライティング経験も多数。
