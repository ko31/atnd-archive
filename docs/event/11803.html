description

新橋烏森口から、ニュー新橋ビルを右に見て、まっすぐ歩きます。
1〜2分歩くと、小さなスクランブル交差点、左手前はKFC、その先にはパチンコ屋、そこを左に曲がります。
曲がって1ブロック行くと、右前に競馬情報のあるJRAのビルがあるので、それを過ぎたところの小さな路地を右へ。
さらに15mくらいに角に立ち飲みワインバーがあるので、そこを左へ。
するとすぐに、平人坊の看板が足下に置いてあります。
