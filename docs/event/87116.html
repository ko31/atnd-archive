はんだづけ！はんだづけしよう！！

購入したキットや自分で設計した回路の、はんだ付けとか組み立てとかをやる会です！

自分ちでやるとダラダラしちゃってダメだなーって人とか、作業中にちょっと相談相手がいたらうれしい人とか、いつもの場所じゃなくて違う雰囲気のとこではんだ付けしたい人とか、ぜひどうぞ :-)

もちろん、はんだ付け初心者の方もOKです！
「今キット組み立てしてるけど、他の人がどうやってはんだ付けしてるのか参考にしたい！」なんていう方もご参加くださいませ！！

途中から途中まで参加もOK！
キャンセル待ちの方がいた場合は、できるだけ参加枠調整します ので、とりあえずキャンセル待ちで登録してみてください。

ご用意いただくもの

  • 組み立てたいモノ
    今作業中のキットや、自作回路、自作メカ、その他はんだ付けや組み立て作業が必要なものならなんでもOK！！
    ※大がかりなもの(搬入に困るサイズのものや材料がすごく多いなど)は、作業スペースに入れない可能性があるので、ご遠慮ください
  • 工具
    自分の作業に必要な道具をお持ちくださいませ。
    半田ごて・ニッパー・ピンセット、カッター、はさみ、等があれば大丈夫。
    一応、貸し出し用の工具も用意しておりますが、共用になります。

タイムテーブル

会場は10時30分から開いてます。早く到着した方は、5階の休憩スペースでお待ちください。

タイムテーブルは以下の通りですが、大体で決めてますので、前後したりします。
午前だけ、午後だけ、途中から参加、途中まで参加なども大丈夫 ですので、ご都合のいいタイミングでご参加ください。
