◆はじめに◆
こんにちは、ハースストーン楽しんでいますか？
ハースストーン友達を増やして、もっと楽しんでみませんか？

◆コンセプト◆
このイベントは「ハースストーンが好きな友達を作ろう！」というイベントです
オフラインのイベントは様々ありますが、
「参加してみたら既に皆仲間がいる状態だったらどうしよう」
「１人でオフライン参加はちょっと勇気がいる」
「あんまり強くないからイベントに行っても楽しめないかもしれない…」
そんな風に思って参加するのを躊躇していませんか？

このイベントはそんな貴方にとってピッタリなイベントです！
なぜなら、そういう方を対象にしたイベントだから
是非このイベントで友達を見つけ、ハースストーンをもっと楽しみましょう
過去にも開催し、参加された皆さんに好評頂いています
是非参加をご検討ください

◆参加ターゲット◆
・ハースストーン友達を増やしたい方
・オフラインのイベントに参加経験がほとんどない方
