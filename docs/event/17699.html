MySQL 『 超 』 初心者勉強会

PHPでWebプログラムを行う場合、現在は一括りで考えられるMySQLについて勉強します。

本格的なWebプログラムを行う上で、MySQLについての知識は必須といえます。

この勉強会では、MySQLの概要と基本的な使い方について勉強します。

学習内容

  1. MySQLの基礎の基礎を勉強します。
  2. MySQLはサーバー上でどのように運用されているのか。
  3. MySQLを使う理由（メリット）。
  4. MySQLの構造。
  5. MySQLの実際の使い方。

注意事項

  1. 勉強会はプロジェクターに映した講師の画面を見ながら、自分のノートパソコンにプログラムをします。
  2. ノートパソコンをお持ち下さい。
  3. 午後７時３０分から午後９時３０分の約２時間です。
  4. 費用が3000円かかります。
  5. 領収書の発行ができますのでお問い合わせ下さい。
