description

ただ、お酒飲んでネットでは言えないけど、言いたかったことを言って「え～ありえなーい」とか言われながら笑い合う女子高生みたいな遊びをお酒の勢いで楽しむオフ会です。

参加資格
以下のどっちか1つを満たすこと
「はてなブログ／ダイアリー／ブックマークのいずれかを使っている人」
「はてなはやってないけど、はてな村から監視されてるよ…という人」
「はてなはやってないけど、三沢文也含めはてなの人のブログはよく読む人」

であれば大歓迎です。それ以外の人も歓迎したいですが、参加者の半数ははてなブロガーなので内輪ネタを出されたらわからなくなる可能性を承知の上でお越しください。

【続報】
参加者に未成年がいてお酒を飲まないこと、年配者が多めなこと、ブコメなど各種反響で「ちゃんとしたものが食べたい」という声からちゃんとしたお店を抑えました。ご協力、お力添えをいただいたshowgoさんに感謝いたします。
