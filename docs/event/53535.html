description

中津の居酒屋なにわで、はもちり鍋を食べましょう。
決してきれいなお店ではありませんが、味は保証します。

はもちりは予約していますが、当日のメニューで食べたい一品ものがあればそれもオーダーしていただいて大丈夫です。
みんなで分け合いましょう。

鱧が一人前でも相当な量なので、お店の方のアドバイスに従って４人前をお願いしています。
当日はその分と飲み物代、追加で頼んだものがあればそれも足して、割り勘にする予定です。
費用はおよそ５、０００円の前後くらいかと思います。
（お酒を大量に飲んだ場合はこの限りではありません。）

食いしん坊の皆さんのご参加をお待ちしています！

※Googleマップで住所を検索すると、ちょっと違う住所が示されるらしいですので、お間違いないようにご注意ください。
梅田から歩くか、地下鉄御堂筋線中津駅１番出口より徒歩10分くらいです。
不安内の方がおられましたら、駅で待ち合わせてご一緒しますのでお知らせください。
