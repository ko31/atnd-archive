9/25のハッカソンにも参加表明をお願いします。

iPhoneアプリに興味を持ち、いざ開発しようとプログラム勉強を始めたとき、「Objective-C」の難解さでつまずいた…そんなことはありませんか？

「Titanium mobile」（Appcelerator社製）は、Web出身の人には馴染みのある Javascript を使用して、簡単にiPhoneを作成することができ、しかも同時にandroidアプリも作ることができるとてもお得なソフト。

全くはじめての人でも、２日間（講座＋ハッカソン）で、実際にアプリを作ってもらおうという、（ムチャな?!）講座です。

HTMLを多少知っていればそれでOK。必要なものは、Mac( OSX 10.6以上、iOS SDK 3.2/4.1)。シミュレータでの動作確認になります。

みなさんのご参加をお待ちしています。

講座参加料は講座とハッカソン（２日分）で6,000円。

講座（操作の仕方〜基本的な構造〜パーツ説明〜サンプルアプリ）では実際にソースを手打ちしてアプリを制作します。

ハッカソン(アミダくじでグループ決め〜アプリ案作り〜アプリ制作・画像制作〜発表会)は毎月月末に行います。次回は9/25。

初心者向けの講座です。経験者のかたはご遠慮下さい。

なるべく多くの方に参加いただきたいので、終了間近でのキャンセルはご遠慮下さい。

Macをお持ちでなければ、Macをお持ちのお友達とご一緒にご参加いただいても結構です。

事前にご準備いただきたいものがありますのでご確認ください。
