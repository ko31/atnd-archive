アキナジスタ・スマートフォンマーケティングセミナー vol.3

title アキナジスタ・スマートフォンマーケティングセミナー vol.3
catch スマートフォン・アドネットワークの新潮流とスマートフォンにおける電子書籍市場の動向
description

　4月より2 回開催して大変好評をいただきました「スマートフォンマーケティングセミナー」の3 回目をモバイル、インターネット業界関係者様向けに開催することとなりましたのでご案内申し上げます。
　今回のセミナーは、現在急速に成長しているスマートフォン市場における「アドネットワークの新潮流」及び、市場成長の中で変化が求められる「電子書籍市場の動向」について開催させていただきます。
　本セミナーは、今後の貴社の事業戦略にお役に立てるのみならず、スマートフォン広告市場と電子書籍市場の発展に資するものと考えております。
　席数に限りがございますので、奮ってお早めにお申し込み下さい。

日時: 2011 年7 月29 日（金）14:00～16:00（開場13:30）
会場: 大手町サンケイプラザ（千代田区大手町1-7-2）201 号室
定員：最大50 名（原則1 社1 名様まででお願い致します。）
参加費：無料
お申込み：7月22日（金）午後5時までに下記メールアドレスにご連絡ください。
