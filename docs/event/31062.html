――――――――――――――――――――
■卒業時にスカウトも！？
『Application boot camp（アプリケーションブートキャンプ）』とは？
――――――――――――――――――――

『Application boot camp』は、
HTML5、CSS3、JavaScript、iOS、Androidなどの最新技術を習得できる、
無料の講座（全10回・30時間）です。

対象は「自分の新しい未来を切り拓きたい」という志を持つ、
ウェブアプリケーション開発実務経験を持ったクリエイター。
卒業時、成績が優秀だった方には、協賛企業からスカウトがあることも。
リアルとビジネスSNS『Biz-IQ』を使った新しい「クリエイタースクール」で、
クリエイターの育成、業界の活性化を行って参ります。

――――――――――――――――――――
