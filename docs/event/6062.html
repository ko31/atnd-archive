description

ビジネスシーンや交流会、そしてセミナーなど…
個人と個人を繋ぐツールとして誰もが持っている「名刺」。

渡したり、受け取ったりすることで会社対会社の関係、
そして個人と個人の関係を繋ぐ大切な一枚でもあります。

しかし、そんな名刺の大切さを解っていらっしゃる方が
実際どの程度いらっしゃるでしょうか。

例えば、初対面の方と名刺を交換したその後、
どのように管理をされていますか？

せっかく交換しても、名刺フォルダーに入れっぱなし、
ましてや机の片隅に投げっぱなしでは、まるで宝の持ち腐れです。

宝の山とはすなわち『人脈』です。

その人脈は、もしかすると人生を変える『金脈』となり得るかもしれません。

たった一枚の名刺で、過去と未来が繋がっていきます。

そして、このチャンスを活かすかどうかは、
必要な時に必要な情報を取り出せる
「情報管理能力」にかかっているといっても過言ではありません。

・これまで80社を超える顧問先企業をサポートし、
参加企業5,000社を超える交流会を開催してきた早川が
いかにして『人脈』を『金脈』に変えてきたか？

・毎月1,000枚以上の名刺を配る早川が、
どのようにして個人の名前と顔、そして会社名や出身地まで
覚えているのか？

・そして、どのように新規ビジネス、新規案件創出につなげているのか？

このセミナーでは、早川の人脈構築方法から名刺管理の仕方まで、
ビジネスシーンで必ず役に立つノウハウをお伝え致します。

ぜひご参加いただき「人生を変える」大切な一枚を
経営資産にする方法をマスターしてください！
