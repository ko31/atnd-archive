M→Fes #とは

『最高の、その先へ。』

我々は2XXX年よりやってきた、次世代の映像制作を牽引する候補者養成機関、通称「M→Fes」です。

2019年、例年の会場であった中野キャンパスホールが確保ならず、M→Fes 2019は事実上の未開催となります。

そこで私たちは、Adobe After Effects CC 2XXX に搭載された「タイムコード(Timecode)」を応用し、タイムマシンを開発。2019年のM→Fesを開催させるべく、私たちは2019年へと向かいました。

本日、新たなステージにてM→Fes 2019の開催が確約されましたことをご報告させていただきます。

映像を愛する皆様、これが次世代のM→Fesです。

初めて映像制作・映像出展を行う皆様、成長したあなた達を、未来でお待ちしております。

2XXX-12-14-03:15

【作品応募】

M→Fesでは出展していただける映像作品を募集しています。
大学内外問わず受け付けています。
下記説明、注意事項をよく読んで、参加登録時のアンケートフォームに従って記入して下さい。

作品について

  • 1作品の長さは 1~5分程度 でお願いします。
  • ジャンルは自由ですが、著作権や公序良俗的なものは守ってください。
  • 1出展者につき1作品とさせていただきます。同名義での複数出展はできません。
  • 動画フォーマットはmp4形式(H264 / AAC)でお願いします。
  • フレームレートは30fpsでお願いします。
  • アスペクト比は16:9でお願いします。
  • 解像度は 1280×720 (720p) もしくは 1920×1080(1080p)でお願いします。
  • 【2019年追記】会場のLEDディスプレイの大きさは4736x768です。1920x1080でご提出頂いた場合は、こちらで1280x720 (720p) にリサイズを行わせていただきます。
  • 【2019年追記】 TUNNEL TOKYO 様の会場に合わせて超大画面映像の製作が可能です！詳しくはメールにてお知らせいたします。

スケジュールについて

  • 10月23日(水)23:59 : 出展応募締め切り (出展者名とメールアドレスをお聞かせください)
  • 10月26日(土)23:59 : サイト/パンフ公表用出展者情報提出締め切り
  • 10月26日(土)23:59 → 10月27日(日)23:59 : 制作中作品の一次提出（作品の予定時間を合わせてお聞かせください）
  • 11月2日(土) → 11月5日(火) ：出展者情報公開
  • 11月16日(土)23:59 : 制作中作品の二次提出
  • 11月30日(土)23:59 : 作品の最終提出　(上記規格に合わせて上映できる状態でお送りいただきます)

提出について

  • 各提出方法は、出展応募確認後、出展者様宛にメールを送りますので内容に従い返信にて提出をお願いします。
  • 今年度はメール提出以外に映像提出フォームを仮運用しています。 お使いいただければ幸いです。
  • 上記フォームが何らかの理由で使用できない場合は、一次・二次・最終提出は ギガファイル便 や firestorage にて、同メールアドレス宛に連絡をお願いします。

メールにてアナウンスを行うため、常時メールボックスの確認をお願いします。

【必読】注意事項

  • 応募作品は新作はもちろん、今年公開した映像作品であればそちらの応募もOKです。 去年以前に公開した作品の応募は申し訳ありませんがお控えください。ですが、 去年以前に公開済みの作品をリメイク・再編集・一部使用するということは可能です。
  • 楽曲・素材・作品内容など、権利関係を含めた全責任は各出展者が負うものとし、上記内容について何らかの問題が発生したとしてもM→Fesは一切の責任を負いかねます。
  • 運営側で上映が不可能と思われるような性的・過激な表現が映像内に含まれている場合、出展者と相談の上、修正もしくは上映をお断りさせていただく場合があります。
  • 1グループにつき複数名の出展者・登壇者がいらっしゃる場合は、代表者の方のみ出展者登録をお願いいたします。
  • その他、不明点・疑問点等ありましたら下記連絡先よりお気軽にご連絡ください。
  • 会場の都合上、今年度の音響設備強化につきましては、現時点では未定となっております。ご了承ください。

作品応募が多数の場合について

作品応募が多数となった場合は、時間の関係上、どうしても全作品を上映できない可能性があります。 その際、 10月26日(土) 10月27日(日)の一次提出後に選考を行う場合があります。選考で優先させていただく条件は以下の通りです。

  • 運営からの連絡が随時取れる方
  • オフラインの映像イベントに出展した経験が少ない方
  • 新作の映像を出展される方

選考を行う場合、 10月26日(土) 10月27日(日)までに登録していただいたメールアドレス宛にご連絡いたします。
出展応募された作品を全作品上映したいという思いは変わらず、時間の許す限り調整を重ねる所存でございます。大変恐縮ではございますがご了承ください。
