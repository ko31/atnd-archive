iOS/Macに関する勉強会です

ニコニコ生放送によるインターネット配信について

当日の配信はこちら => 4/14 iPhonedev-jp 勉強発表会 – ニコニコ生放送

ニコニコミュニティ => iPhonedev-jp ニコ生出張所

ニコニコ生放送による中継（4時間程度を予定）および大阪会場への中継などを行う予定です。
一応録画の形も取りますが1週間ほどで観れなくなるタイプでの録画になると思います。
ですので「映るのはちょっと」という方は少し気をつけてください（スクリーンの方を映すので参加者の方にカメラは行きませんが、声はのるかも知れませんので気をつけて、という意味です。）

放送を觀るのにコミュニティに参加する必要はありませんが、メンバーが増えると放送が安定するのでよかったら参加してください。

懇親会について

資料・スライドの共有について

SYNCNELの会議機能を利用して資料を共有します。発表者と手元の資料を自動的に同期して閲覧することができます。
ネット中継をご覧になるかた、大阪会場のかたも手元で資料を閲覧できます。
会場に来られるかたも、座席によってはプロジェクターが見にくい場合がありますので、あらかじめ設定しておくとよいでしょう。

SYNCNELのダウンロードはこちら => SYNCNEL – AppStore

参加者のかたの設定手順

  1. AppStoreから「SYNCNEL」のアプリをダウンロードして下さい
    • SYNCNELのダウンロード => SYNCNEL – AppStore
  2. 起動して新規アカウントとして以下で登録→ログインして下さい
    こちらから自動設定できます => 自動設定する
    • 表示名 : [任意]
    • ユーザ名 : audience@iphonedevjp.syncnel.biz
    • パスワード : audience
  3. ディスクドライブの一覧が出てきたら画面一番下の「会議」タブをタップして下さい
  4. 「iPhoneDev jp勉強会」をタップして下さい
  5. 後は放っておくと自動的に資料切替やページめくりが追随されます
  6. 画面に触れると自動追尾が切れますので、再度発表者に自動追随する場合はリンクボタンをタップして下さい

SYNCNEL「会議機能」について

  • PDF資料を配信する事ができる
  • 発表者がPDFのページめくりをすると追随してページが開く
  • 発表者がPDFを切り替えると追随して自動的にPDFをダウンロードして開く

お願い

ネットの接続数が不足する可能性があるので持参できるかたはモバイルWiFiルーターなどをお持ちください。
ただし、あまりにアクセスポイントの数が多いと接続できませんので当日は運営スタッフの指示に従ってご利用ください。

電源タップを持参できるかたはできるだけ持ってきてください。
よろしくお願いします。

キャパシティの関係上、多くのひとは机が無く、イスのみになると思われます。あまり大きなPCなどは広げにくいですのでiPadやMacBook Airなどをお持ちになることをオススメします。
