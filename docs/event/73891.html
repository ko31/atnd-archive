freeeを使ったカンタン確定申告 in 成城ホール 2/28(金)

平成27年分の所得税の確定申告期限は 3月15日(火)です。

・freeeで記帳をやってるけれど、この後どうすればいいの？
・そもそも freeeの入力が正しいのか不安だなぁ～
・確定申告に向けてやるべきことが多すぎて、大変だ！

そんな、個人事業主の皆様に向けて確定申告セミナーを開催いたします！

セミナー終了後、質問時間もありますので、この機会に確定申告の疑問を解消してください。

日時

2016年２月28日(日) 13：30～15：30

対象者

freee 導入済の個人事業主・又は freeeを使って確定申告を行う予定の個人事業主で、以下の不安を感じている方

  • 確定申告について不安のある方
  • 確定申告の具体的な手順を知りたい方
  • freee をあまり使いこなせていないと感じている方

効率的、かつ、低コストで確定申告を乗り切るには、クラウド会計の活用は不可欠です。

今回のセミナーでは、クラウド会計freeeを使った確定申告についてわかりやすくご説明いたします。
クラウド会計freeeの概要も説明しますので、初心者の方も安心して参加してください。
