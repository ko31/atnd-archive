Medical Innovators’ Conference

イベント ユーザー
event_id 31125
title Medical Innovators’ Conference
catch 〜未来を切り開く医療学生のネットワーク〜
description

【イベント概要】
医療学生および医療分野で活動している若手イノベーターによる講演。現在のご自身の活動、活動に至った経緯、これからの展望など、参加学生を刺激する講演をお願いする。

【イベント趣旨】
社会の中での医療を考え、未来の医療を見据えて行動する学生の集い。本イベントを起点に学生の医療に対する積極的な活動を促進します。
1, 医療を変える意気込みのある学生間のネットワーキング
2, ハブとなる学生イノベーターの活動促進
3, 医療学生向けイベントアプリのプロモーション

【スピーカーの紹介】

・吉永和貴

所属：慶応義塾大学医学部5年　株式会社DYMインターン
高度化、多様化する医療ニーズへ対応するためのサービスをITの側面から構築すべくプログラミングを勉強中。今までに国試サイトや災害医療アプリ、Evernote拡張機能などの開発を担当しました。将来、みんなで現場から医療を改革しましょう！

・藤田優美子

所属：東京大学薬学部薬学科5年　 ビジネスコンテストTRIGGER2011年度運営
ー「誰か」じゃなくて「自分」が変えるー
日本医療の課題に自ら向き合い、医療学生、薬学生として今何ができるか、将来すべきことは何かを考え活動中。現在は患者と医療者間のコミュニケーションを「視覚」で繋ごうとデザインを勉強しています。小さな事でも自分で気づいて行動する、その姿勢が大事だと思います。

・二見あかね

所属: 聖母大学看護学部国際看護学専攻4年　医療法人社団KNI　経営企画室
どうしてこんなに医療格差があるんだろう…そう思って以来、『カンボジアに病院を作ること』が私の夢になりました。これまで途上国でのボランティア活動や国連インターンを通じて国際保健に向き合ってきました。そして今、少しずつその『夢』に近づいています。みんなの力で医療を変えていきましょう。

・中村孝彦

所属: 大阪大学医学部医学科5年　医療法人社団KNI　経営企画室
経済産業省様から御支援を受け、理事長の北原茂実先生の下、日本の医療の産業化・国際化に向けて真正面から取り組んでおります。現在、「デジタルホスピタルプロジェクト」「ワンコインドックプロジェクト」を担当しており、週６勤務で日々奮闘しております。今の自分で皆様にお話することはあるのだろうか、と大変悩みましたが、田沢さんにはお世話になっていることもあり、今の自分の思いを正直にお話しようと思ってこのお話をお受け致しました。恐れ入りますが、宜しくお願い申し上げます。

・末政 憲司

所属：経済産業省商務情報政策局情報政策課（東京大学工学部卒・国家Ⅰ種経済職）大学時代は医療社会システムを研究。診療システムの可視化に挑み、主に脳血管障害を対象としたリハビリをモデル化。そのモデルをもとにデータを抽出し、様々な比較分析から「医療の質」を追求する。理系出身の傍ら、法律・経済を勉強し、医療政策の新しい切り口を模索。「医療で日本を変える」をモットーに日々奮闘する社会人1年目。「医療」なのになぜ厚労省ではなく経産省なのか。そう思った人ほどぜひ聞いてみて下さい！

・江島 健一

所属：長崎大学医学部５年　AMSA Japan
医師が働く現場がよくなれば、その先にいる患者さんがより良い医療を受けることができるのではないか。そのような考えを持ち、医療ｘITｘ経営など様々な面から医療を見つめ、仲間を見つけ、行動する。地方にいるからこそ見える世界もあると思い、今回このような素晴らしいお話を引き受けさせて頂きます。
