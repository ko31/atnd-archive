【こんな方に向けてのセミナーです】
Webデザインに興味があるが、何から始めたら言いのか分からない。
Webデザイナーとしての仕事にチャレンジするためのスキルを習得したい。
Webデザインとプログラミングについて基礎からじっくりと学びたい。

【セミナー内容】
今までに200名程生徒に指導経験のあるリーダークラストップエンジニアによる、Webサイト制作未経験からでも現場即戦力になれるスキルを身つけるセミナーです。Webサイト制作の基本となるHTML,CSS,PHを基礎からじっくりと学びます。それだけではなく、プログラミングの基礎や、データベースにも触れていますので、Web制作に必要なスキルとを総合的に学ぶことができ、また、エンジニアとしての最低限の知識も手に入ります。
自分がプログラミングしたWebサイトが完成していく工程を見る楽しみを体験することが出来ます。
「夢を語る会」では、Webデザイナーを目指す仲間との交流を通して、自身の方向性や、新たな一面を発見することが出来ます。
”ITのもたらす未来の世界”や”市場とITの関係”を学ぶ日もあり、ITについて知るには充実した内容となっております。
