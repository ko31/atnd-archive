★講演内容

このイベントは2015年6月にヒトメディアにて実施されたイベント
「大集合!!ソシャゲ業界にいたあの人は今」の続編です。
イベントの趣旨は、スポットがあまり当てられなかった「キャリア」にスポットを当て、
ソーシャルゲーム業界で得た経験は、他業界でも通用する素晴らしいものである事を
世の中に伝える事にありました。

今回はその第2弾として内容を少し深堀し、内容は「スキル」にフォーカスし、
より規模を大きくするために、ヒカラボとコラボして実施させて頂きます。

急速にサービスが普及し、爆発的にユーザーが増え続けていた中で、
技術面、ツール面、管理面、UI・UX面などにおいて、
どのような工夫でそれを乗り切ってきたのか、また、どの様にチーム作りをしてきたのか、
そしてその時の経験が、現在においてどう活きているのかを、ざっくばらんにお話をしていきます。

イベントを通して、ソーシャルゲーム業界への正しい理解と、
一人でも多くのソーシャルゲーム業界経験者が自分のスキルに
自信を持ってもらえる様になれればと思っています。

もちろん、今回も懇親会の場をご用意していますので、
個別で聞きたい事はどんどん聞いてしまって下さい！！

皆様からのご参加をお待ちしております!!

※内容は変更になる可能性がございます。

　現場には大手のWebサービス運営事業者からの転職者やソーシャルゲームの開発経験者なども多く、
　前職でのスキルを存分に発揮して活躍をしています。
