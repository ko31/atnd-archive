こんにちは！
生産者の声を発信するウェブメディア「日本食べるタイムス」です。

皆さんは農家ににどんなイメージを持っていますか？
ずばり農家は、百の仕事をこなす”百姓”なのです。
知恵や技術はもちろん農家の魅力はたくさん。

そこで、今回は「現代農業」編集長石川啓道氏を迎え、日本の農業の現状、これからの農業界についての講演会を行います。

石川氏の講演の他にも、山形県小国町の生産者さんとのトークセッション、小国町の生産物を食べながらの交流会など盛りだくさんなイベントとなっております。

また、8月に山形県小国町で開催する農業インターン（農業実習体験、大学生・社会人歓迎）の説明も行います。

「おいしいものが食べたい」「農家さんと話したい」というような小さな興味から農業を学びたい人。農業の現場を知りたい人。
何もしていない空いた時間を何か興味のあることにあてたい人。

日本の農業について共に学び、農業界の明るい未来をつくっていきましょう！
