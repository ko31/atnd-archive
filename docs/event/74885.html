Photon勉強会、再始動！

進化したWebhooksと今話題のIPv6対応について、語ります！

『FINAL FANTASY GRAND MASTERS』『リトルノア』『ファミスタドリームマッチ』など、大型スマホゲームで多数採用されている、Unityとの相性も抜群のネットワークエンジンである「Photon」。

2016年第1回目の勉強会では、アプリ業界で今話題となっているIPv6に関する話題を中心にお話をしていきます。Photonの対応状況はもちろんのこと、NAT64+DNS64についても簡単にお話していく予定です。

また、アップデートして利用可能サービスが広がったWebhooksに焦点を当てて、機能のご紹介をしていきます。
そしてついにリリースのPhoton Server V4についてもお話する予定ですので、お楽しみに！

セッション終了後、参加者同士やPhotonのエンジニア、GMOアプリクラウドのスタッフと直接お話できるネットワーキングタイムも設けておりますので、ぜひこの時間もご活用ください。

なお今回を第1部として、この後もUniteでのセッション(第2部)や5月勉強会(第3部)を予定しており、全3部作としてお届けする予定です。そちらへのご参加もぜひお待ちしております！
