内容

勉強会リピート第1回

  • 音の大きさとアンプの出力の関係について
    1. 音の大きさの単位（dB SPL vs phon）
    2. デシベルの計算
    3. スピーカー＆ヘッドホンの「能率」とは
    4. 演習
  • 真空管の開発略歴
    1. 熱電子放出の発見と二極管（ダイオード）
    2. 三極管の発明と問題点・特許の回避と四極管～五極管
    3. そして伝説のビーム管へ

交流会

交流会では、参加者同士でお互いに親睦を深めていただけたらと思います。また、真空管に限らず自作オーディオ作品の持込み展示、交換・頒布もどしどしやってください。

勉強会第2回

  • 真空管の動作
    1. 三極管のVP-IP特性、VG-IP特性
    2. 三極管の「三定数（gm･rp･μ）」
    3. 五極管のVP-IP特性と、三極管との動作の違い
  • 真空管の命名規則
    1. 米国式命名規則
    2. 欧州式命名規則
    3. 日本式命名規則
