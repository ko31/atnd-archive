動画配信サービス市場を牽引する、ベンチャー2社 SHOWROOM と schoo によるMeetupです。

両社は「動画配信」サービスとしてのイメージが強いですが、それは開発のごく一部にすぎません。むしろ、市場を切り開くために、会社の数かぎりないチャレンジを支えてきた開発体制、マインドこそが両社の特徴なのです。

そんな開発チームが、サービス急成長の背景で起こった、開発における失敗談や問題に対して、両社が実践してきた打ち手や解決してきたことなどを赤裸々にお話します。
