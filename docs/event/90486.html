description

外で遊ぶと暑いので、もくもく会やります。

もくもく会とは、
1つの場所に集まって、各自がもくもく(黙々)と勉強したり、本を読んだりするだけの会です。家では集中出来ない！気分を変えて作業をしたい！という人におすすめです。　

もくもく会はもともとIT系の人たちの文化ですが、その枠を広げ、より多様な人たちで集まって作業をしようと思っています。

本・漫画を読む人、資料を作る人、レポートを書く人、資格の勉強をする人、イラストを描く人、ブログを書く人、編み物をする人、たまった『あとで読む』を消化する人、なんでもありです。もちろん、IT系の方も大歓迎です。オススメは漫画と編み物です。

かなりゆるい会にする予定ですので、成果発表なしでやります。
自己紹介も任意です。ただ集まって、作業するだけ。
無料のコワーキングスペースとして使っちゃってください。

ただし、ちょっぴり雑談多めにしたいです。
少しでも、交流が生まれれば、と。
