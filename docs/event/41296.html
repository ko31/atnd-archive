ぺちぱな。

　ぺちぱな。（PHPer-na）は長崎のPHPを使っている人の意です、一応。別に長崎に限りませんがPHPを使っている人、興味がある人、使っていない人、興味がない人、とりあえず一緒にお話をしてお話を聞いて学んでいきませんか。

　と言う事を主旨にやっておりますが、今回は初心に帰りましてPHP超入門をやります。「え、なに、いまさらなの？」ていうのは重々承知しておりますが、それはそれ、初めての方はそのままに、そうでない方はお遊び気分でやってみましょう。

　今回は二部構成です。最初にPHP入門をやりまして、その後に恒例のLT的発表会をやろうと思います。いつもよりも余計に時間を取るので今まで発表していなかった方もこれを機会に是非一度どうぞ。もちろんこれまでやってくださっている方の再度の発表もお待ちしております。連絡してくださいね、プログラムに組み込みますので。

　PHPerの方も、そうでない方も、是非お気軽に遊びに来てください。待ってます！ こわくないよー

会場

　長崎市立図書館研修室1、2。

　会場の場所が少しわかりにくいです。 長崎市立図書館 Nagasaki City Library 図書館について に図がありますが、正面入り口から入場後、1階図書室へ入らずそのまま右へ折れ多目的ホール脇の階段を登ってください。二階の左奥に会場の研修室があります。

備考

　ノートPCなどがあればより楽しめます。その際、MAMPとかXAMPPとか、或は自前PHP環境を用意してきていただけるとありがたいです。
　当日会場に早めに来ていただければインストールをお手伝いする事もできます、午前10時より待機しておりますので！

　ノートPCなどをお持ちの方は是非持参ください。無線LANアダプタなどがあるとインターネット接続が可能です。
　twitterなどで随時発言していただけると嬉しいです。

　インターネット接続回線としてWiMAXを用意していますが、ご自分でどうにかできる方にはそうしていただけると助かります。一回線しか用意できないので、あまりに多くなると接続できなかったり極端に遅くなる可能性があります。

　電源タップをお持ちの方も持参していただけると嬉しいです。床埋め込み式のコンセントが複数接地されていますが、アダプタなどは挿し難いです。こちらでも何本か準備しておりますが、何とぞご協力ください。

　終了後、この為のというわけではないですが、懇親会を開きたいと思っています。参加は自由ですし、逆に懇親会だけ参加でもOKです。
　ただし、酒乱の気のある方はご遠慮ください :-)
