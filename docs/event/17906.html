はじめまして、学生企画ユニットTEAM Improveの舘山と申します。

私たちはこれまで、主にクラフトビア（地ビール）の振興​活動など、

企画・イベントを通じて「日本の閉塞感を打ち​破る」ことにチャレンジしてきました。

今回は、シリーズ企画「クラフトビアを楽しむ会Vol.​3」のイベントの告知です。
＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿​＿＿＿＿＿＿
3・11、日本は未曽有の大震災に襲われ、たくさんの人​が亡くなり、
日本経済は大打撃を受けました。
いま、震災から約4ヶ月が経過し、徐々に復興は進みつつ​あります。

私たちは、震災の残した爪あとと、これから長い年月つき​あって
いかなくてはならないでしょう。
つき合い方はボランティア活動、チャリティー、募金イベ​ントなど
さまざまあると思います。

私たちTEAM Imprveはイベントを通じた付き合い方をご提案しま​す。

今回のイベントでは、若者を中心に被災地で作られたビー​ルを片手に、
被災地の農産物で作られたおいしい料理を食べながら、こ​の国の将来や
これから私たちにできることについて考え、話し合おうと​思います。

このイベントは
「実際に被災地の商品（ビール、農産物）を購入し、消費​することで、
被災地の経済に貢献しよう」というコンセプトで行われま​す。

「震災で地元が被災して、何かやらなきゃと思うけど、行​動できてない」
「何かしたいと思ってるけど、ボランティアはちょっと…​」
と思っている若者のみなさま。

ビールを通じて、ちょっと貢献してみませんか？
