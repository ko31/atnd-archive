【プログラム】
◆講師の言葉
　IEEE802.11n無線LANで実用化されたMIMO通信技術は、WiMAXを経由して、LTEで始めて携帯電話に本格的に導入される。本セミナでは、MIMO通信技術を設計・開発するうえで必要となる基礎理論と設計技術を与える。
　本講座では、LTEの概要を紹介するとともに、送信および受信信号処理、適応リソース制御、伝搬路シミュレータ、アンテナ設計、マルチユーザMIMO、基地局連携MIMOを詳解する。

◆修得知識
・MIMOに関する基礎理論、通信方式、信号処理、制御方式、システム設計、伝搬シミュレータ、アンテナ設計、マルチユーザ通信、連携通信、標準化動向
