a-blog cms Training Camp 2014 Autumn

イベント ユーザー
event_id 56129
title a-blog cms Training Camp 2014 Autumn
catch 今回は知多半島の先っぽで合宿らしい合宿です。今回は9回目になります。
description

合宿のいいところは、他の人が作ったサイトの普段見る事の無い管理側をどう作っているのかを知る事ができる数少ない機会です。是非、クライアント様の紹介許可を得て参加下さい。

初心者の方についてはアップルップルからの初心者向けのセッションを用意しますし、仕事を持って参加頂ければ悩んでいる部分についてを解決する事ができると思います。

10分くらい発表して頂ける方も募集しています。

日時

2014年11月21日（金）13:00 – 22:00
2014年11月22日（土）9:00 – 12:00

費用

  • 20,000円（1泊2食付） ※17日（月）以降のキャンセルにはキャンセル料がかかります。ご了承ください。

プレゼント

  • 参加者全員：独自ドメイン拡張+1（30,000円相当）
  • 発表者全員：非商用利用スタンダードライセンス（個人ブログとかに使ってください）

※ どちらも年内を申請期限とします。

会場

  • 師崎荘（愛知県知多郡南知多町大字師崎字高岩57）

電車、飛行機をご利用の方

中部国際空港に 12:00 集合を予定しています。（会場までの送迎バスあり）
名古屋駅から中部国際空港への移動は「名鉄空港特急ミュースカイ」が便利です。
中部国際空港アクセス　時刻表

自動車をご利用の方

お車の方は師崎荘へ現地集合でお願いします。

受付

  • 今回の費用は2日間通しでの費用になります。
  • 部屋は大部屋になる可能性があるそうです。
  • 受付用に名刺を1枚ご用意下さい。（皆さんと名刺交換する事になりますので多めに持ってきて下さい）
