ご連絡

趣旨

前回同様Amazon Web Serviceで提供されるサービスについての利用方法や課題の共有がメインになります。
今回も堀内さんにお越しいただける事になりました。

第０回の勉強会から４ヶ月ほどしか経過していませんが、OpsWorksをはじめとする新サービスの発表や、相次ぐ値下げ発表などで知らない間にメジャーになってるサービスがあるかもしれません。

ぜひともこの機会に情報交換しましょう。

参加対象

AWSを使った事は無いが興味がある方、AWSを使ったことがあるがもっと情報が欲しい方、AWSの使い方を広めたい方、AWSを使ってる仲間が欲しい方、等々、ご気軽にご参加ください。
