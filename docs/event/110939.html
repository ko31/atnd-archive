都会と農村を掛け合わせたら何ができるの？

都会に住む者と地方に住む者が集まり、それぞれの地域資源を元に、スキルを掛け合わせながら実現できる「可能性」を考えていくワークショップです。

今回、掛け合わせる地域は、都心から車で5時間。コンビニやスーパー、そして信号機もない。豊かな自然と文化にあふれた「長野県天龍村」です。

ワークショップを通して都市と農村を繋げることでできる可能性と自分たちのやりたいを掘り起こしながら、実現する道を一緒に探り提案していきます。

人とつながり、地域とつながる。
やりたかったこと見つけて、ともにはじめませんか？

＼たとえば、こんな提案ができるかも／

・特産品を使った商品づくり
・飲食店とコラボした新メニューづくり
・日本一の星空を探すツアーづくり

＼来て欲しい人／

・農村交流したい人
・新しいことをはじめたい人
・自分の経験とスキルを活かしたい人

＼天龍村について／

東京から車で5時間。天龍村は長野県の南端に位置し、村の南側は愛知県と静岡県に接しています。面積の9割以上が山林で、村の真ん中を南北に流れる天竜川と、その支流が造るV字渓谷の中に集落が点在する人口1,300人弱、高齢化率が60％と全国でNo.2の「純山村」です。
