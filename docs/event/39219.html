2013年5月4日　妙見山遊覧飛行マッピングパーティ

title 2013年5月4日　妙見山遊覧飛行マッピングパーティ
catch 初夏の妙見山で遊覧飛行を楽しみつつ、マッピング
description

今年もGWの妙見山にてヘリでの遊覧飛行（注：曇より悪天候の場合、中止）が行われます！！

それに合わせて妙見山マッピングパーティを開催したいと思います。

妙見山のマッピングデータを取りつつ、
初夏の妙見山をハイキングします。
お昼を山頂で取り、
その後、頂上付近の臨時ヘリポートから妙見山～知明湖周辺～妙見山の5分間の遊覧飛行を行います。
遊覧飛行中もマッピングデータ採集するもよし
マッピング用航空写真を自分で撮ってみるのもよし
景色を眺めるのもよし

遊覧飛行後は下山します。
採集したマッピング用データは後日、各人でOSMに書き込みましょう。

会費（遊覧飛行料金）

小人：\3,000-
