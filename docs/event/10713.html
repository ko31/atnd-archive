description

23時位から集まって閉店の5時まで、まったり飲み食いしながらアナログゲーム（ボードゲームやカードゲーム）で遊ぶ会です。
24時～5時は、この会の参加者でお店を貸し切ります。

会場の清貧は、自分で料理が出来るお店なのでゲームの休憩にお料理してても大丈夫！
時間内は出入り自由なので、ちょっとだけ参加もできます。
居た時間だけかかるタイムチャージ（30分200円。最大1600円）と飲食代は自己精算になります。
参加者みんな用に料理を作ってくれた人の材料費は、食べた人で値札シールを分担します。

主催者で、「パンデミック」「Dixit」「あやつり人形」「カタン」「ゴキブリポーカー」「キャット＆チョコレート」「お邪魔者」「勝利への道」あたりは持ち込む予定です。
「ワードバスケット」「UNO」「ジェンガ」等はお店にあります。
それ以外のゲームを持ち込んでくれる方大募集！
ゲームのかぶりが気になる方は、コメントで持ち込み予定を書いて調整しましょう！

他の持ち込み予定：「はやぶさ君の冒険」「ドミニオン」「ジャングルスピード」「bluff!!」「スリードラゴンアンティ」＆コメントに出てきたもの

初心者も、一度清貧に行ってみたかったという方も、みんなの夜食をひたすら作りたい方も大歓迎。
みんなでわいわい遊びましょう～！

今後も週末や連休中日の深夜を狙って定期的に開催しようと思ってますのでよろしくお願いします。

※仮に定員50名にしています。頑張れば70人位は入れるらしいです。
※24時以降はゲーム会参加者のみの貸切になります。
当日は早めの時間帯が満員なんだそうで、23時丁度に来ると席が空くの待ちが発生するかもしれません。
待つのが嫌な方は、23時半～24時以降にいらっしゃるとベストかも。
※当日飛び入り参加もOKですが、↑なこともあって、アバウトに人数把握したいので、参加できそうでしたら参加ボタンを押して下さい。
