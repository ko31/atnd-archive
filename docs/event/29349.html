WWDC2012の前夜祭をやります！

皆さん、WWDC2012に行きますか？

今年のWWDCチケットはスムーズに2時間半で売り切れましたが、ゲットした方はラッキーですね。

チケットを入手したら、航空券の予約は出来ましたか？どのホテルを予約しましたか？WiFiルーターを用意しましたか？サンフランシスコでの観光予定はありますか？

一番重要なのはやはりWWDCですね。例年のWWDCだと、新しい友だちを作るのがみんな一番楽しいことですので、前夜祭で様々な友達ができたらいいですね。一緒に行きたい場所を決めたり、一緒に興味があるセッションを約束したり、一緒に関心の話題を交流したり、さらに、LTをやりたければもちろんできますよ！

さて、出発前に、Voyage Groupの社内バーAJITOにて飲みながら、食べながら、いろいろな情報交換、友達作りなどをしましょう。

費　用：2000円/人

食べ物：ピザ、寿司

飲み物：ビール、チュウハイなど飲み放題

設　備：プロジェクト、VGA Adapter for Mac、HDMI Adapter for iPad2

Let’s make new friends!

@TonnyXuがVoyage GroupのAJITOで皆さんのご来場をお待ちしております。
