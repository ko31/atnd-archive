★講演内容

仮想通貨の取引所・販売所の運営を行うbitFlyerと、
クラウド名刺管理サービスを提供するSansanが
各々のサービス・プロダクトを支える技術の裏側や、開発フローなどをご紹介。
特殊事業を展開している企業ならではの独自の工夫について、
各社の現場の若手エンジニアがLTやパネルディスカッションを通して語りつくします。

＜bitFlyer　柴田・江田＞

インターネット上で流通する仮想通貨の取引所・販売所の
運営を手がける株式会社bitFlyerのエンジニアが登壇します。

大規模なトランザクションが発生する、
24時間365日体制の取引所・販売所システムの開発舞台についてお話し致します。

＜Sansan　高橋・加畑＞

7,000社が導入する法人向け名刺管理サービス「Sansan」の開発
および先日発表されたばかりの新サービスの開発に従事するエンジニアが登壇致します。

【何を学べるか】
「大規模なサーバーとトランザクションをAzureを使って支えるシステムの裏側」
「C#でのアプリ開発時に起きやすいトラブルとその解決方法」


※ 内容は変更になる可能性があります。


★講演者プロフィール

■bitFlyer　柴田 航平 氏

東京大学 理学部数学科卒。
中学生の頃にプログラミングをはじめる。
2015年にbitFlyerへ入社、LightningというbitFlyer 取引所サービスをはじめ、
ほぼ全ての当社サービスの開発に携わってきた、スーパーエンジニア。
bitFlyerではフルスタックエンジニアとして
フロントエンドからバックエンドまで全て担当。

■bitFlyer　江田 周平 氏

学生時代は大阪大学にて多目的最適化問題に従事。
卒業と同時に、2012年日本マイクロソフトに入社。
テクニカルサポートエンジニアとして、
延べ数千件のお客様の技術的課題の解決に貢献。
得意言語は C#。
在職中に共著で執筆した Azure の技術本については、
数か月間 Amazon の書籍のクラウドカテゴリ部門で1位を獲得。
2018年1月より、bitFlyerに参画。
趣味はマジック。

■Sansan　高橋 洸 氏

2013年4月から日鉄日立システムエンジニアリング株式会社で
電子帳票パッケージの開発・導入に従事。
2016年3月からは Sansan株式会社に入社し、
名刺管理サービスSansanの開発に従事。
2017年10月からは同社で新サービスの企画・開発に携わっている。
担当領域はサーバーサイドからフロントエンド。
よく用いる言語・技術は C# / .NET Framework / vue.js / TypeScript 。

■Sansan　加畑 博也 氏

2013年4月、株式会社富士通研究所に入社。
オフィス利用ソフトウェアの研究開発に取り組む。
2014年11月、Sansan株式会社に入社。
以降、法人向け名刺管理サービスSansanの開発を担当している。
PostgreSQLがちょっと好き。
