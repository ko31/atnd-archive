プログラミング自習会開催します。

自己紹介

こんにちは。
現在プログラミングを勉強中の @96kuroguro です。

普段はWeb制作をしています。

現在の技能 HTML　CSS　（PHP）
学習中の技能 Javascript　PHP　スマホアプリ開発

ここ最近ツイッターやスマホの登場の為か、
Webサービスやアプリ開発など、
世の中のプログラミングの活動が活発になってきたように感じます。

その流れに影響されて、自分でも何か作ろうと思いました。

現在、空いている時間に一人でプログラミングを勉強していますが、
一人では緊張感もなく怠けてしまうことも多いので、
一緒に開発をして張り合える方が居ればいいなと今回の会の開催を考えました。

こんな状況に共感していただける方は、
ぜひお越しいただけると幸いです。

あくまで目的はいつもと違った環境で
集中して学べる場を作りたいだけなので、
参加に関しては基本無料です。

ご自由に参加して頂けると嬉しいです。

※ちなみに主催者も割と人見知りなので、
「リア充怖い」という方も安心して（？）ご参加ください。

対象者

  • ノートパソコンをお持ちいただける方
  • プログラミング（またはWeb制作等）を勉強中の方
  • 自分で学習する意思のある方

※上記に当てはまる方はどなたでも参加可能です。

主催の @96kuroguro は、プログラミング初心者です。
講師がいないので自習がメインとなります。
自分でやりたいこと、やり方を考えながら周囲のメンバーと相談もしながら学習予定です。

ご自身で書籍を持ち込み頂いたり、
プログラミング学習サイト 等を活用して自習して行きましょう。

こんな方にもオススメ

  • プログラムをする仲間を作ってモチベーションを維持し続けたい方
    初心者同士、切磋琢磨しながら成長していければと思います。
  • JavascriptやPHPを覚えたい方、スマホアプリを作りたい方。
    主催者が学習予定なので、少なくとも一人は学習仲間になれます（笑）
    もちろん他の内容での参加も歓迎ですのでぜひご参加ください
  • 自分のサイトやブログを作りたい方
    プログラミングの敷居が高いとお考えのHTML初心者の方も歓迎します
    HTMLやCSSに関しては多少アドバイスはできるかもしれません

イベントの内容

上にも書きましたが自習がメインとなります。

特にこちらから何かテーマを出すこともありません。
途中休憩を挟みつつ、黙々と自習する予定です。
休憩の際に雑談・相談が出来るといいですね。

開始時に同じ空間で勉強するもの同士、
簡単な自己紹介をしていただきます（名前とその日学習する内容をヒトコト発表）。

終了時には、その日学習した内容をヒトコト発表していただきます。（任意）
