Rails for Beginners Vol.1

ビギナー向けRails勉強会の第1回です。

ハンズオンセミナーでは無く、各自がやりたい事をやりつつ、分からないところなどを相談しつつ、サポート出来るところはサポートしつつみたいな感じでゆるくやりたいと思っています。

初心者の方でも歓迎です。(Rails初心者、プログラミング初心者どちらでも大丈夫です。)
Railsやりたいけど何からやればいいか分からない、プログラミング(Ruby,Rails)に興味あるけど難しそうなど、何から始めたらいいか分からない人の始まりの一歩になればいいなと思います。

Rails for Beginnersについて

http://rails4beginners.github.com/home/

主な対象：

  • エンジニアとRailsで一緒に開発したい「デザイナー」
  • 学校でRubyを触ってRailsでもっとWebアプリなどを開発したい「学生」
  • もっとRails力を高めたい経験の浅い駆け出し「エンジニア」
  • エンジニアと一緒に仕事をしたい、エンジニアがやっている事を理解したい「ディレクター／プロデューサー」
  • 継続して学び続ける意志、姿勢を持っている人

サポート可能対象：

  • Mac、Git、Ruby、Rails、GitHub、Heroku
