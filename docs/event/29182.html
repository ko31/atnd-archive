おまたせしましたー、第2回NFC勉強会です。（2012/6/5 update）

今回は、ハンズオン ( ＋LT ) 。

ハンズオンのテーマは、なんと 「NFCタグを作ってみよう！」 です。
しかも、オリジナルデザインのラベル印刷とかじゃなくてアンテナから自作です！

ほんとに出来るか分かりませんが、たぶん光るタグまでは出来ると思います。
（これだけでも交信範囲の確認などに使えて便利です）

LTやりたい方は、 @ohsaka 宛てに御連絡ください！

■キャンセル待ちの皆様へ

会場と準備の都合で参加者を少なめにさせて頂いていましたが、
あらためてシミュレーションしてみると、実際にLEDタグ／NFCタグの
製造自体は1時間かからず終わりそうかな、、、と。
なので、補欠者の方で「どうしても参加したい！」という方は、
お席のお約束は出来ませんが、よろしければ会場にお越しください。
当日あつまった人数とかノリを見て、入れ替え制とか立ち見とかで、
お楽しみ頂けるように柔軟に対応したいと思います。

■参加費／当日の持ち物

参加費は無料ですが、下記の物品持参にご協力頂けると幸いです。

・筒状の何か、をご持参ください。

　　→アンテナコイルを巻き付けるのに便利ですので、何かしらお持ちください。
　　→たとえば、トイレットペーパーの芯とか理想的です。

・必須ではありませんが、ハンダごてを持ってる方は御持参頂けると嬉しいです。

・LEDや銅線は、こちらで用意しますが、持参のものを使いたい場合の持ち込みなど勿論歓迎です！

■NFCとは

Near Field Communicationの略称で、近距離の無線通信規格のことです。
NFCに対応したAndroid端末でSUICA、PASMO、TASPOなどのカードの情報を読み取ることができます。
12/2に発売されたGalaxy Nexusや11/24に発売されたGalaxy S2 LTEに搭載されていて、iPhone
5にも搭載されると噂されている注目の技術です。
