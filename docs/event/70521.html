家づくりトークショー「2000万円台で建てた！建築家との家づくりのポイント」

9/27（日）田園都市線・たまプラーザ「3丁目カフェ」にて

内容

今週9/27（日）に、田園都市線沿線を中心に展開する建築家集団「田園都市建築家の会」主催のトークショーが、たまプラーザ「3丁目カフェ」で開催されます。
3名の建築家が、自身の手がけた身近な価格帯（2000万円台）の事例をもとに家づくりのポイントをわかりやすくお話します。
●お席に限りがありますのでご予約をお願いいたします

（事務局へのお問合せ）
　TEL　045-912-3456
　メール　info@denen-arch.com

●日時：9/27（日）
　・ご相談会：11時～17時
　・トークショー：14時～15時
●場所：「3丁目カフェ」（たまプラーザ北口）

お気軽にお越しください！
