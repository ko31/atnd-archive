ArduinoちょこっとセミナーはArduinoやRaspberry Pi、IoT、電子工作など、はじめてみたいけどよくわからない人向けの「初心者歓迎」のセミナーです。今回はArduinoです。


＜こんな人が対象＞

  • 電子工作もプログラミングもやったことがない
  • なんとなく知ってるけどやったことがない
  • 以前にC言語を学習したことがあるがつまづいた
  • Arduinoの入門書読んだけどつまづいた
  • 昔に電子工作に入門したがつまづいた
  • ちょっとどんなものか体験してみたい

こんな方々を対象に初心者を歓迎したセミナーを開催します。

＜参加費用＞

2,500円
(会場の利用料と部品代込み)


＜コンセプト＞
プログラミングや電子工作は１人で学習していると「うまく動作しない」「エラーが出て進めない」などの問題が発生してどうにもならず学習自体をやめてしまうことが多くあります。

初学者の場合、ちょっとした質問をできる環境があれば解決できることも多くあるので、セミナーをしながら都度質問ができる場を作りたいと考えて開催しました。

もし作りたいものがあれば、ざっくりとしたアドバイスもできると思いますし、そのためのアクションを小さなステップに分解するためのアドバイスもできると思います（あくまで参考程度に）。

＜どんなことやるの？＞
30~60分くらい自作のWeb学習動画を見てもらいながら、
わからないところを都度質問してもらう形で進めます。
超初心者向けのため、内容としてはLEDチカチカ＋αくらいを想定してます。

一方通行のセミナーというよりは、対話したりしながら個々に合わせて進められます。
質問することで「うまく動作しない」「エラーが出て進めない」などの問題は解決できると思います。

60~90分くらいはそのように進めて、残りは自分で色々いじったり、
学習したり、質問したり、他の勉強や作業したり、もちろん途中退場もありです。

＜電子工作もプログラミングもやったことないけどOK？＞

はい、全然OKです。


＜持ってきていただくもの＞

  • ノートパソコン（無線LANに接続できるもの）
  • パソコンの電源ケーブル
  • Arduino Uno
  • USBケーブル（プリンターケーブルで使うアレです）

※Arduino Unoは、もしなければ貸出しますので「Arduino貸出希望」とコメントしてください。


＜事前にやっていただくこと＞

Arduino IDE最新版のダウンロードとインストール

Arduino IDE（最新版はv1.7.8）
http://www.arduino.org/software

Arduino IDEのインストール方法（okuzawatsさんのブログ）
http://okuzawats.com/install-arduino-ide-20150202
