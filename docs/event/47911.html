第2部　「エバール®」フィルムを用いた壁紙の特性と汚れ防止技術、耐久性評価、今後の展開

【13:15-14:30】

講師:(株)クラレ　エバール事業部　フィルム販売部　ご担当者　様

【キーワード】
1. 機能性壁紙
2. 汚れ防止
3. 表面強化
4. 耐薬品性
5. 抗菌性
6. 非吸着性
7. エバール

【講演概要】
機能性壁紙の要求性能には、耐薬品性、耐汚染性、耐傷性(表面硬度)、抗菌性などがある。一般的な壁紙の素材である軟質塩ビの上に「エバール®」フィルムHF-Mをラミネートすることにより、「エバール®」の化学的特性に由来するマルチ機能を備えた機能性壁紙を得ることが可能である。「エバール®」フィルムHF-Mには特殊な艶消し加工と塩ビに熱融着可能な接着剤をコートしているため、既存の軟質塩ビへの接着も容易であり、さらにはエンボス加工での意匠性も兼ね備えているため、既存の壁紙製造プロセスでの加工が可能である。
「エバール®」の化学構造に由来する特性、壁紙用「エバール®」フィルムHF-Mの優れたマルチ機能と臭気物質に対する非吸着性、および今後の用途展開の可能性について紹介する。
