WordCamp とは？

全国のウェブに関わる方を対象にした、WordPress勉強会、カンタンに行ってしまうと
「WordPress のお祭り」です。Web 制作力の向上と、他県の優れた Web 制作者との交流、
また WordPress ってなに？という初心者の方への気軽な講習を目的としています。
参考： WordCampの内容

WordPress( ワードプレス ) とは？

簡単なブログから、複雑なウェブサイトまでを構築できるコンテンツ管理システムで、日本でも多くの
ウェブサイトに利用されています。オープンソース（誰でもカスタマイズ出来るように元となるソースコードを
公開しているシステム）のため、日本でも沢山の方が WordPress に関しての情報交換を行なっており、
実際に会場を設けての定期的な勉強会も盛んに行われています。
（WordCamp Nagoya　175名参加・WordCamp Yokohama　800名参加・WordCamp Fukuoka　400人参加 ・ WordCamp Kobe　600人参加 ・ WordCamp Tokyo　800人参加 などなど）
参考： WordPress 日本語
