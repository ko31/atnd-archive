【MUGENUP流　3DCGアートディレクションセミナー】

3DCGのより良い制作とは！？　無料セミナーを2月18日（土）開催！

ソーシャルゲームを中心に、3DCGや2Dイラスト、映像など、年間1万5千件以上のアートクリエイティブを作り出してきた株式会社MUGENUPが、株式会社アカツキ様のご協力のもと、無料セミナーを開催いたします。

3DCGを用いてゲームを開発する様々な企業やCGデザイナーの皆様を対象に、クライアントである企業側と受託を受ける外部パートナーとが一緒にいいモノを作っていくためのノウハウをご紹介いたします。

主なポイント（予定）

  1. 3DCG制作において、みんなが頭を悩ませるのはどんな時？
  2. 開発企画、量産、運営…相談のタイミングによって変わる注意すべきポイント。
  3. コミュニケーションコストを下げ、トラブルを避けるコツ。
  4. 外部パートナーと、スムーズに仕事を進めるディレクションとは？
  5. お互いにハッピーに一緒にいいモノを作るために。また一緒にお仕事をするために。 　…他

開催概要

  • タイトル： MUGENUP流　3DCGアートディレクションセミナー　～外部パートナーと一緒にイイものを作るには？～
  • 日付： 2017年2月18日（土）
  • 時間： 受付13:30、開始14:00、懇親会15:20、終了16:20（予定）
  • 場所： 株式会社アカツキ内 会議室
  • 費用： 無料
  • 備考： 受付および懇親会用に、お名刺（2枚）のご用意をお願いいたします。
