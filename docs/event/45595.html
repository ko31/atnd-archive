【追記】 CloudStack Day Japan 2014 事前登録開始しました。

CloudStack Advent Calendarの季節になりました!

Advent Calendar とは

クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーです。
転じて、技術系コミュニティで、クリスマスまでの25日間にTipsを1つずつ、Blog等で公開するイベントを
Advent Calendar と呼ぶそうです。

CloudStack Advent Calendarでは、 大晦日 までやります。

参考までに昨年の CloudStack Advent Calendar jp 2012

仕様:

  • 参加される方は、 管理票 を確認し、 いつ書くか を 管理表 と ATND にコメントすること。
  • イベントに参加した順で 12/1 から 1 日ずつお持ちの blog, Web 日記, slideshare等に記事を上げる。
    ※入門編でも、上級編でも、イベントレポートでも、トラブル情報共有でも、ビジネス関連でも、CloudStack関連のことであれば なんでもOK !
  • 12/1 が1 人目、12/2 は 2 人目、… 12/n は n 人目 … 12/31 は 31 人目、のように。
  • 記事を書いたら コメント欄 にタイトルとURLを貼ること。
