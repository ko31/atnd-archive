講演概要

No1. Windows 7 タスクバーを使い尽くす(田中達彦さん)

Windows 7のタスクバーは従来のWindowsのタスクバーより大幅に進化しています。
Windows 7タスクバーの活用方法や知っておくべきことなどを解説します。
タスクバーの活用例として、Internet Explorer 9 も紹介します。

No2. LightSwitch で 最速 RIA 開発 + α ?(池原大然さん)

このセッションでは現在ベータ版の Visual Studio LightSwitch というツールの概要を概説させて頂きます。
並びに拡張機能であるエクステンションの作成に挑戦します。できるといいなあ～

No3. Windows Phone 7開発入門(西村誠さん)

Windows Phone 7の開発基礎と躓いたところを紹介

No5. ビジネス・シュミレーションゲーム「THE・商社」

ビジネス・シュミレーションゲーム「THE・商社」を通じてあらためてみんなでコミュニケーションやリーダーシップについて考えてみましょう。

1. コミュニケーション、営業の大切さ
2. 企業戦略を理解し動くことの大切さ
3. PDCAサイクルを回す、自律型人材になることの大切さ
4. チームワーク、リーダーシップの大切さ

ゲームは4～5人1組になって、チームで対抗戦を行います。
難しいルールなどはありませんので、純粋にゲームとして楽しんでください！

講演者概要

No1. マイクロソフト：田中達彦さん

  • 田中達彦のブログ

1997年マイクロソフトに入社。現在はWindowsテクノロジーを広める役割を担う。
以前は教育機関や学生を対象にしたマーケティングや、
Visual Studioなどの開発ツールの製品マーケティングを担当。

The Student Dayでのセッションのほか、多い年では年間50回の講演もこなす。
過去には、TechEdやThe Microsoft Conference（MSC）の基調講演への登壇や、
.NET Developers Conferenceでのゼネラルセッションを行う。

マイクロソフト入社以前は、ボーランド株式会社で開発ツールや表計算ソフトの日本語版の開発を担当するなど、開発畑を渡り歩く。日本オリジナルの製品であるTurbo C++ 5.0の開発者でもある。

2004年度、2005年度、2006年度の電気通信大学非常勤講師、
2007年度、2008年度の同志社大学嘱託講師も兼務。
「ゼロからはじめるプログラミング」などの著書のほか、雑誌への執筆も多数。
