description

ゼロから新しいビジネスモデルの
構築をすることはとても大変な作業です。

名経営者であっても次に進む道を決めるのは
最も難しい判断のひとつ。

せっかくならはじめから
自分の強みを活かしたビジネスモデルでスタートさせたい。

そんな風に思いませんか？

世の中のすべてのビジネスモデルは
７種類２２分類の型で説明ができる！

本講座の開発者である
一般社団法人シェアブレインビジネススクール代表理事
である中山匡氏が１０年という歳月をかけ
小資本で成功している事業を調査した結果
集まったのが１,０６１の成功モデル。

そして、それらを分析し、
試行錯誤するなかで発見されたのが
小資本で起業可能な７種類２２分類のビジネスモデルです。

ビジネスモデル基礎セミナーは
この７種類２２分類の手法に基づき
あなたの強みをパズルのように核融合させ
あなたオリジナルのビジネスのアイデアを
構築していくプログラムです。

【次のような方におすすめ】

こちらのセミナーでは
特に次のような方の受講をお待ちしています。

〇事業を拡大したいけれどどうすれば良いのかわからない方

〇やりたい起業アイデアはあっても競合が多い分野のため起業に踏み出せずにいた方

〇現在の事業にもう１つ売上の柱となる新たなビジネスを構築したいと思う方

〇新規事業の企画に悩んでいる方

〇ビジネスモデルの発想法に興味がある方

〇コンサルタントや士業の方で、
　顧客にビジネスモデルの作成、発想法を教えたい方

【セミナー内容】

1.ビジネスとは何か？

2.ビジネスモデルとは何か？

3.今の自分の事業のビジネスモデルを描いて見る？

4.今の事業に足りない部分を発見する。

5.ビジネスモデル構築の手順

【セミナーで学べること】

７種類２２分類の型を使うと
なぜ競合ゼロのビジネスモデルが構築できるのかを
理解していただけます。

また、ご自身のビジネスに新しい「型」を追加すると
全く違った「競合ゼロ」のビジネスモデルに
生まれ変わらせることができることを
体感していただけます。

ぜひ自分に最適なビジネスモデルを
見つけましょう！
