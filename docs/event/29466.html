最新技術、さわれてますか？

HTML5/CSS3/Node.js/WebRTC/WebIntents/Twitter Bootstrap2/WebGL/Three.js/Indexed Database API/Backbone.js/SPDY/WebSocket/face.js/Tower.js/Device APIs/etc,etc…

昨今のWeb技術の進化はめまぐるしいばかり。実際に手を動かして触るということができていない方も多いのではないでしょうか？

この「Web先端技術味見部」では、そうした 真新しい技術を実際に触ってみる ことを目的とした勉強会です。

どんな勉強会を目指してる？

始めたばかりの勉強会ですので、具体的な進め方は試行錯誤の上決まっていくと思いますが、取り急ぎ以下のようなイメージをしています。

  • とにかく最先端・・・「世界一キャッチアップが速い」勉強会を目指しています。なので、事前設定されたテーマは「仮」のもの。ワクワクするような新技術が登場したら、当日いきなり差し替えになることもありえます。
  • 少人数・・・10数人程度で、身軽に最先端の技術を追いかけたい。
  • 広く発信・・・この勉強会は、ITmediaさんとコラボしてたり、Ustreamで発信したり、広く情報共有することを目指します。
  • みんなで学ぶ・・・なにぶん最先端なので、「教えてくれる人」がほとんどいない。なので、プロジェクタを活用しつつ、ゼミ形式でみんなで学びましょう。
  • 楽しく、アットホームに・・・ゆるい雰囲気で、楽しく、でも思い切り最先端技術をねぶりましょう！終わったあとは有志で飲み会にGo！

今回の仮テーマは「Vert.x」「JSX」です。


Vert.xは、Java仮想マシン上で動作する非同期サーバプログラミング環境です。Node.jsと比較されることも多いプラットフォームですが、「JVM上で動作する」「JavaScriptだけではなく、JRubyやGroovy、Javaでもプログラミング可能」というユニークな特徴があります。

当日ですが、JSXに変更です！
JSXはDeNAが開発した新しい言語です。コンパイルするとJavaScriptに変換されるタイプの言語ですが、「静的型付け」「コンパイル後のJavaScriptコードが手で書くよりも高速」という特徴があるそうです。

このテーマは「仮」のものです！ これより最新で、ワクワクするような技術が出てきたら、そちらに差し替えることもあり得ますのでご了承ください。
