起業したいと思っている人たちが、気軽に集まれるサロンです。

具体的でなくてもOK！何でもありのサロンです。

「働く」と一言で言っても、雇用されて働くだけではなく、

「起業して働く」という選択肢もあります。

一緒に、三方よし（売り手よし、買い手よし、社会よし）

のビジネスを考えませんか。

　

このサロンでは、起業動機の掘り下げと

人事・労務分野の基礎知識講座を

ざっくばらんな座談会形式で行います。

お金の話も大切ですが、

「人」はその事業（夢）を叶える「資源」だからです。

そして夢の語りは、就業規則を作成する際にも役立ちます。

規則の目的を労使で共有できるからです。

途中、笑いヨガで笑い、

タフさとゆとりと楽観性を

取り戻す方法もご紹介します。

笑いヨガとは、

世界的には、Laughter Yoga（ラフターヨガ）と読み、

日本語では、「笑いヨガ」と訳します。

インドの内科医マダン・カタリア博士が考案した、

子どものあそび心をベースとしたユニークな笑う体操です。
