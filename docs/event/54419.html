イベント概要

オープンハードカンファレンスの打ち上げイベントとして、無限ピザパーティーを行います。部外者も広くWelcomeです。楽しくピザを食べましょう！

オープンハードカンファレンスは、お台場で2014/7/12に行われました。
http://oshwc.project2108.com/event/oshwc2014tokyosummer/

オープンハードカンファレンス懇親会では、オープンハードウェアとして開発した珪藻土石窯を使った無限ピザパーティが行われました。

今回は、より改良された珪藻土石窯を使い、ライブピザパーティーを行います。

珪藻土石窯とは

断熱性能、耐熱性能に優れた素材である珪藻土ブロックを用いることで、本来部屋を占領する大きさのピザ石窯をコンパクトに。また優れた熱効率の特性のために100V電源環境でも500℃まで加熱できる性能を達成しました。
珪藻土は自然素材であり、お酒の濾過などに使われるように人体に無害な特性を持ちます。また加工も簡単であり、廃棄時にも崩せば土に戻る、小容量熱源を高効率で利用できる、など環境負荷が著しく低い素材として更なる利用シーンが期待できます。
