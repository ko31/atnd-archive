チベットの精神的リーダー、ダライ・ラマ法王がティーチングのたびに引用し、口にする、
有名な仏教の古典テキストを学べる、イブニングワークショップです。

「入菩薩行論（にゅうぼさつぎょうろん）」は
今から1300年ほど前にシャーンティ・デーヴァによって記されたものです。
このテキストには、まず人を助けるために、
【仏陀の境地に到達するという願望を私たちの中でいかに育むか】について実践的なアドバイスがあります。

●菩薩行とは何か
●菩提心をはぐくむ方法、
●菩薩としての具体的な実践方法
などについて言及されている、美しい詩のような仏教の古典です。

現代に生きる私たちが、目覚めた心や意識を育んでいく必要性について
明確に書かれた指南書となっています。

講師のDr.バリーはインドのダラムサラに20年以上住み、ダライ・ラマ法王より戒を授けられた僧侶であり、西洋医師というユニークな経歴をもつ人物です。その温かい人柄と真摯さ、ユーモアあふれるティーチングに定評があります。
