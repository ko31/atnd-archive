★講演内容

IoTって結構趣味の領域でやってる方は多いけど、
それを実際にビジネスでやるってぶっちゃけどうなの？

そんな疑問に答えるべく、ビジネスとして真剣にIoTに取り組む各社による
パネルディスカッションを開催します。

サービスやプロダクトに関わる話から、ここでしか聞けないIoTの辛さや裏話など、
ざっくばらんにお話しいただきます。

登壇者に、IoTの領域のスタートアップで実際にビジネスとして取り組まれている
dotstudio株式会社　代表取締役CEO　菅原のびすけさん、
株式会社no new folk studio　代表取締役 菊川裕也さん、
株式会社スカイディスク CTO 大谷祐司さん、
モデレーターには、ハードウェアスタートアップを支援する
モノづくり施設「DMM.make AKIBA」の上村遥子さんを迎え
熱いトークセッションを繰り広げる予定です。

トークセッションの後の懇親会では、
登壇者と一緒にIoTやスタートアップに関する話題で盛り上がりましょう！


※ 内容は変更になる可能性があります。


★講演者プロフィール

■dotstudio株式会社
代表取締役CEO　菅原 のびすけ 氏

1989年生まれ。岩手県立大学在籍時にITベンチャー企業の役員を務める。
同大学院を卒業後、株式会社LIGにWebエンジニアとして入社し、Web制作に携わる。
2016年7月よりdotstudio株式会社を立ち上げ、
Developer Relations と IoT領域を中心に活動している。

日本最大規模のIoTコミュニティであるIoTLTや、
JavaScript RoboticsコミュニティNodeBotsの主催、
IoTバックエンドサービスであるMilkcocoaのエバンジェリストとしても活動中。

■株式会社no new folk studio
代表取締役 菊川 裕也 氏

一橋大学商学部経営学科を卒業後、
首都大学東京大学院にて芸術工学を専攻。

2014年より新しい音楽演奏用インタフェースの研究として開発を始めた
スマートフットウェア「Orphe」の製品化をきっかけに
株式会社no new folk studioを設立。
「日常を表現にする」をキーワードにIoT製品を提案していく
ハードウェアスタートアップ企業として注目を集める。

受賞歴に日本メディア芸術祭審査員特別賞、
日本グッドデザイン賞、Music Hack Day Barcelona – Sonar賞他。

■株式会社スカイディスク
CTO 大谷 祐司 氏

サイバーエージェントでインターネット広告のシステム部門を立ち上げ、
子会社の技術責任者に就任。
その後インテリジェンスで技術責任者としてエンジニア組織を立ち上げる。
2017年より福岡に移住。CTOとしてSKYDISCに参画する。
技術が大好きなエンジニア。

モデレータ：
■DMM.make AKIBA
コミュニティマネージャー 上村　遥子 氏

2009年より、web広告業界でのメディア立ち上げ及び運営にも携わりながら、
広告企画営業として、金融からゲーム業界まで幅広く経験。
長らくIT業界にいたのち、2016年より、業界としてIoTの世界へ踏み入る。
現職DMM.make AKIBAでは、コミュニティマネージャーとして、
スタートアップ支援、コミュニティ活性に奔走中。

英語より仏語を得意とするため、仏語圏案件の担当をし、
仏スタートアップとの交流、情報交換も行う。
