こんちくわ。
ブログ大好き@conchikuwaです。

ブログ書くことがもう日課となっております。ブログを書いて情報をアウトプットするとそれ以上にインプットがあって最高でございます。

でもブログを書くって結構孤独で寂しかったりします。

また他のブロガーさんのブログに対する考えやどのようにブログを更新しているのか、チョー気になってまいります。

なのでその思いを解決すべく千葉県市川市の私の事務所で市川ブログ合宿〜みんなで集まってブログを書こう！〜を開催いたします！

キッカケは一宮ブログ合宿

ブログ合宿をやりたいなと思ったキッカケは尊敬する有名ブロガー兄弟「五藤ブラザース（@goryugoさん、@nori510さん）」の「一宮ブログ合宿」の記事を読んで。

ブログ合宿のレポもとにかくタノシソー！！

一宮ブログ合宿にはメッサ参加したかったけど、参加できなかった。やっぱり一宮、愛知県はちと遠い。

次またあったら、参加したいけど、参加できるか分からない。んじゃぁ関東でもやればいいんじゃねって思って、前にブログにやりたいなって書いてみたら、ブロガーさんにやりましょうと賛同をいただいた！！

嬉しくて涙チョチョ切れジョータイっす(^^♪

なので、有限実行！6/2（土）に千葉県市川市の私の事務所でブログ合宿開催します！！

市川ブログ合宿の趣旨は一宮ブログ合宿と同じで、いつもは一人で孤独感に苛まれながら、時にはイライラ、時にはニヤニヤしながら行なっているブログ更新という作業をみんなで楽しくやってみようよと。

ブロガーさんが集まっていっしょにブログ書いたら何かと面白い事があるんでは。。。

また自分の当たり前が、人にとっては当たり前でなかったり、逆に人にとっての当たり前が、ナニソレスゲーって事だったり、集まって書くことで一人では気付かない気付きがあるんでは。。。

まぁとにかく集まって楽しくブログ書きましょう！！

市川ブログ合宿

なんで千葉県市川市かというと私の事務所があるから。

ただそれだけです。

もちろん都内でも良かったけど、場所を借りるとお金かかるし、タダで場所を貸してくれる知り合いがいるほど甲斐性がなかったり。。。

千葉県といってもちょっとガンバレば東京都に歩いて行けるとこなんで、そんなに不便じゃないと思います。

ちょっと駅から歩くけど。。。（徒歩15分くらいです。）

事務所の外観は↓な感じ。（リンク先では建物にリカーハウスNAGASHIMAという看板が写っていますが、現在、看板には文字がないです。）

花屋－宅配花屋さん花ＲiＲoの店舗について

市川ブログ合宿の内容

まぁ特にイベントも何も無く、ただ集まってブログを書くもよし、ブログについて意見交換するもよし、気楽に楽しみましょう。

  • 日時：2012年6月2日（土）11：00〜17：00
  • 場所：花RiRo事務所
    〒272-0025　千葉県市川市大和田3-18-3　1F
    電車 → 総武線本八幡駅、都営新宿本八幡駅、京成八幡駅 徒歩15分程度
    車 → 京葉線 市川インターから10分程度
  • 人数：最大12名（最低遂行人数一人。でもそれは寂しい。。。）
  • 参加費：タダ
  • 持ち物：ブログを書く道具
  • 対象者：ブログを既に書いている人
  • やる事：ブログを書く。（できればブログ合宿について1記事、以外にもう1記事。まぁこのあたりも適当に。。。）

その他注意事項

大した事ではないですが。。。

  • 電源はありますのでご心配なく。
  • 無線LANはfonルータがありますが、何人繋げるかよくわからないので、モバイルルータをお持ちの方はご持参下さい。
  • 車でお越しの場合、駐車場はありませんが駅から事務所までの間にコインパーキングがあります。値段は1日500円〜1000円くらい。
  • 参加費は無料ですが、飲み物や食事は出ません。各自用意下さい。
  • お湯は沸かせます。電子レンジあります。火は使えません。
  • 事務所脇にドリンクの自販機あります。徒歩5分くらいの場所にデイリーヤマザキあります。
  • 特に予定はしていませんが、希望者がいれば合宿後、呑みにでも。

とにかく集まってブログを楽しみましょう！

合宿というと気合い入れて、一生懸命ってイメージがありますが、市川ブログ合宿はそんな事はなく、ゆるく楽しくをモットーに開催できれば幸いです！！
