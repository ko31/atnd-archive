奥多摩～青梅をのんびりサイクリング、奥多摩トレックリング（通称奥トレ）。#34は8月24日（日）に東京の地酒・澤乃井の酒蔵見学に行きましょう！

日本酒ファンのみなさん、お待たせしました～。笑。
毎回奥トレの打ち上げで大人気の澤乃井、僕も大好きですが、東京にいるとなかなか酒蔵見学をしたことは少ないみたいですね。奥トレのコースには、東京の地酒・澤乃井の酒蔵もあるので、今回はそんな酒蔵を訪ねてみたいと思います。ここは渓流沿いに日本酒のビアガーデンが併設されていて、夏季限定の甘酒かき氷も出ます！見学のあとは渓流を吹き抜ける風とせせらぎの中でかき氷食べてのんびりしましょか。
初めての方も、何度も来てくれている方も、久しぶりに仕事や近況とかいろいろ話しながら、ゆっくりサイクリングできればと思うのでぜひぜひ遊びに来てください☆

奥多摩から青梅に降りて、自転車返したあとは河辺の温泉・梅の湯で汗を流して、ウワサの澤乃井ガッツリ飲んで帰りましょう！
ぜひぜひみなさんの参加をお待ちしてまーす☆
