ＦＣＦ04 リブンカコミュニケーション
９月３日（土）１１：００開場
ミューザ川崎研修室にて開催！

はじめに

　いま、教育や研究における専門性は細分化し、全体の見通しをもつことが難しくなっています。そんななか起きた原発事故は、異なる専門をもつ人たちが相互理解できないという問題を僕たちに突きつけました。どうしたら理系文系にかかわらず、専門の違う人たちがコミュニケーションを取ることができるのか。ぜひＦＣＦで、あなたの学ぶ専門分野を他の人たちに分かりやすく説明してください。

FCF04について

　ＦＣＦはプレゼンテーション大会です。学生間コンテンツネットワークでは、今まで3回にわたり、情報系の学生と社会人がプレゼン発表を通じて交流できる場を提供してきました。今回、第４回を行うにあたって、内容を大きく刷新し、イベント名をＦＣＦに変更。学生プレゼンコンテストを中心にした新たな交流の場として、皆様に提供いたします。ＦＣＦは、プレゼン発表と懇親会で構成されています。プレゼンによって、互いに刺激を与えあってもらい、懇親会にて、発展へのアイデアを交換していただければと思います。皆様、ふるってご参加ください！

参加方法

　どなたでも参加できます。当日、直接会場にお越し下さい。途中での入退室もＯＫです。発表を希望される方は予め登録が必要ですので、参加登録をお願いします。
また、当日、会場に一般参加者としてお越しになる場合は、予めATNDから事前登録をして頂きますと、会場での手続きがスムーズになります。ご協力ください。

なお、万が一人数が会場の上限に達した場合は、こちらのWebページにてアナウンスし、入場制限を設ける可能性もございます。ご了承下さい。

募集要項

　ＦＣＦでは、学生プレゼンコンテスト、一般発表の参加者を募集しています。
希望者は参加登録をお願いします。

学生プレゼンコンテスト：提案部門（テーマ：理文科コミュニケーション）

　あなたの学ぶ専門分野を他の人たちに分かりやすく説明してください。または、異なる専門知識をもった人たちが、相互に交流するための解決法を示してください。持ち時間は７分です。

学生プレゼンコンテスト：やってること部門（テーマ：無制限）

　あなたが現在、活動している・作っているものを、
ＦＣＦという場で、存分に広めてください。別の視点から見た新たな展望が生まれたり、新たな仲間が見つかったりするかもしれません。持ち時間は７分です。

一般発表（テーマ：無制限）

　学生に向けて、あなたの仕事や、研究について、発表してみませんか？　持ち時間は１５分です。

主催・協賛

　このイベントは、ＮＰＯ法人学生間コンテンツネットワークが主催し、次の団体の協賛・協力を受けて実施されます。

神奈川工科大学 東京都市大学
