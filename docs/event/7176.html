description

TracLightningツールについて勉強します。

第２回目のテーマは、予定を変更して「ワークフローのカスタマイズ（基本編２）」です。
第１回目の続きとTracLightning開発者Ｏかもと氏によるTanabata.trac再演「TracLightningの開発に見るテストコードを1行も書かずにTDDする方法」を行います。
第１回目にご参加くださった方は、前回配布した演習用資料と、ご意見をいただける方はアンケート用紙を忘れずにご持参ください。
注）演習用資料について、既に配布済みの方には今回用意いたしませんので、予めご承知おきください。

TracLightningにすでに用意されているプラグインを活用して、ワークフローの変更方法を紹介します。
１回目に引き続き２回目のゴールは、自分が思い描くワークフローに変更できるようになることです。

初めて参加する方、もしくはバージョンが2.4.X以前の方は、『TracLightning 2.5.0』以降のバージョンをあらかじめインストール（アップグレード）してきてください。よろしくお願いいたします。

http://sourceforge.jp/projects/traclight/releases/

参加費は一人 500円 です。

持ち物ですが、実際に動かしながら体感していく演習中心の勉強会なので、なるべくPCをお持ちください。

（同時にコンセントの数が不足することが予想されるため、できれば電源タップもお持ちいただけると助かります）

また、当日の勉強会の開催場所ではネットワーク環境はありませんので必要な方はご自身でネットワークにつなぐためのイーモバ端末等をお持込ください。

（勉強会の内容自体はネットワークにつながなくても大丈夫な内容を予定しております。）
