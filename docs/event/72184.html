PS2辺り～今世代機までのコンシューマ、PCで描画周りを中心にシステム全般を実装する仕事が多いプログラマ。

ゲームエンジンの中でも、実機で動作するRuntime Framework部分の基本的な設計と処理の流れについて解説し、各種ゲームエンジンの動作を理解する際の手掛かりになる事を目標としています。
概念的な話なので、明日すぐ業務に使えると言ったことは絶対にありません。
