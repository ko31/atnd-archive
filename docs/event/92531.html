■概要

このイベントをきっかけにあなたもIT企業で働くエンジニアを目指してみませんか。
【初心者歓迎・無料】知識ゼロから学ぶプログラミング：Webアプリ開発体験会 Ruby超入門編は、プログラミングをこれから学びたい！Webサービスを作りたい！という超初心者向けの無料プログラミング勉強会です。

この講座は 【初心者歓迎】知識ゼロから学ぶプログラミング 2週間でWebアプリ作成講座(12/2~12/14)の体験講座になります。

日本のITエンジニアの約30%に当たる13万人が利用するプログラミング学習サービスのpaizaラーニングが主催する、個別指導型でWebアプリケーションの作り方が学べる実践的な講座です。

■Rubyとは？

Rubyは素早くWebサービスを構築するのに向いたプログラミング言語で、近年はクックパッドや、価格.com、マネーフォワード、Airbnbなど、多くのWebサービスの開発に使われています。 転職、あるいは新卒の求人でRubyの開発スキルを求める企業も多くあります。

■対象者

  1. プログラミング初心者の方、Webサービスを作りたいと考えている方
  2. ITエンジニアになってみたい（手に職を付けたい）と考えている方
  3. Ruby未経験で、学びたいと考えている方
  4. 18歳以上で日本語がネイティブレベルの方

■体験会の内容

ITエンジニア75 万人受験のスキル評価システムに基づいたカリキュラム
paizaが転職サービスで展開する、延べ65万人以上の現役エンジニアが受験したプログラミングスキル評価システムのデータと、paiza掲載の1,900件以上の求人票や内定・採用に至った方のスキルデータを元にカリキュラムを作成。開発現場で役に立つスキルを身につけることができます。

Ruby入門編

  1. プログラミング初歩
  2. 条件分岐、比較演算子
  3. ループ処理

個別指導形式で、動画の学習教材を元にステップごとに実際にプログラミングを行い、分からなければ講師に質問し、その後演習課題を解き、理解を深めながら進めていくスタイルの講座です。
一斉指導型ではないので、自分のスピードに合わせた学習が可能です！

■本講座のゴール

多くのWebサービス開発の現場使われているRuby on Rails を使う上でベースとなるRubyの基礎の習得をめざします。
プログラミングに興味があるけど何から始めて良いか分からない方、自分の考えたWebサービスを形にしたいと思っている方、ITエンジニアでRubyを素早く覚えたい方などに最適な講座です！
この講座は 【初心者歓迎】知識ゼロから学ぶプログラミング 2週間でWebアプリ作成講座(12/2~12/14)の体験講座になります。
