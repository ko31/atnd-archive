日本android東北支部のサブグループAndroideaグループで出たアイディアの実現第一弾！


■趣旨


高校時代に友達で図書館やファミレスで勉強したノリで，とにかく仲間と時間をshareして各自のコーディングをするものです。

■内容

開催時間内に，各自が適当なタイミングでフェードインして始まります。
式次第はありません，講師もおりません。
ときどきおしゃべりアリで，お勉強する！という内容です。（おしゃべりは，twitterやチャットもあるかもしれません）

■その他

  1. ネット接続のゲストアカウントを申請しますので，当日ネット接続が必要な方はお知らせください。
  2. Core2ノートパソコンの貸し出しできます。
  3. DevPhone1,2実機をお使い頂けます
  4. コーヒー，紅茶のみのドリンクバーを設けます
  • Androideaグループは，日本Androidの会東北支部のサブグループです。思いついたことや，コードの書き方で困っていること，一緒にやりたいこと等々。考えがまとまってからだけでなく，思いつきでも投げられる。そんな気軽なノリで，真面目にコミュニケーションするグループです。
