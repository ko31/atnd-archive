Hack For Japan ハッカソン 7/30 仙台会場

“東日本大震災に対し、自分たちの開発スキルを役立てたい”
―IT開発者の皆さんのこうした声をもとに、Hack For Japanは生まれました。

このHack For Japanのアイディアソン/ハッカソンが再度仙台にて行われます。
7/23 アイディアソン ( 懇親会 )
7/30 ハッカソン ( 懇親会 )

今までも複数の有意義なプロジェクトがHack For Japanから生まれています。皆様是非ご参加ください。

イベント概要

今回は学生のIT技術勉強会STDIO.Sと共催です。学生さんの若いパワーに触れるチャンス！

ハッカソンについて

前回5/22の開催では復興時計が商品化の道を歩み、絆レポートというITの枠を超えたプロジェクトも生まれました。

技術レベルよりもやる気を重視してご参加ください。
やりたいことに技術が追いつかないときはチューターが支援してくれます！

仙台会場チューターは以下の通りです。TwitterAPIに強いチューター陣ですよ
