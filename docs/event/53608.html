description

「高速・ハイスペック・安定・低価格」をすべて満たそうとするとクラウドサーバには多くの課題があります。本セミナーでは、ベアメタルサーバが可能にするこれからのインフラストラクチャ・レイヤにおけるクラウドサービスについて、「ベアメタル型アプリプラットフォーム」の特長をわかりやすくご説明します。
また、開発チームの技術スタッフによるデモでは初公開となる最新バージョンのコンパネをご覧いただきます。

※ ベアメタル型アプリプラットフォームについての詳細は、ウェブサイトをご覧ください。

■ 講演内容・タイムスケジュール

15:30 -
開場・受付開始

16:00 – 16:40
「クラウドの利便性×物理の優位性」が生む、現場のための新しいインフラサービス
株式会社リンク 営業部　部長　阿部 了一

インフラ担当者が本来求めているものはサービスの安定稼働をいかにコストを抑えて実現するか、ということではないでしょうか。必要とする要件を満たすことができるインフラサービスが何かを考えることは、そのインフラが仮想か物理かで悩むこととは別の問題です。 ベアメタル（OSの入っていない物理サーバ）がクラウドと相性がいい理由、既存のホスティングサービスとの違いについて、ベアメタル型アプリプラットフォーム開発の裏側をご紹介しながら、これまでのサーバ運用の概念を変えるその特長をわかりやすく解説します。

16:40～16:50
休憩

16:50～17:30
デモを視覚体験しながら図解で解説「コンパネの先には何があるのか」
株式会社リンク　営業部　営業技術　山本 誠一郎

ベアメタル型アプリプラットフォームのコントロールパネルを実際にみながら、コンパネ上の操作によって何ができるのかをわかりやすく解説します。リソースプールと呼ばれるベアメタルサーバ群からどのようにサーバを立ち上げるか、言葉だけではイメージしにくいところを紹介。

17:30 ～ 18:00

18:00
閉会
