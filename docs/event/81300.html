通常、チラシは0.5秒で捨てられます。

そのぐらい早く判断されてしまうのです。

しかし、６つのポイントをおさえれば、

効果のある「売れる」チラシがカンタンに作れます。

チラシはほんの１例です。

食べログ・FB・ツイッター・ＳＥＯ、ＰＰＣ。

手法には、それぞれ「長所」「短所」があります。

使っているユーザーにも特性があります。

『どんな人が、どの媒体を使っているのか』

『費用対効果が１番良い手法を選ぶには？』

ここを考えて動かなければ、効果の良い“当り”集客方法を選ぶことはできません。

残念ながら、開業後にゆっくり集客手法を調べている暇はありません。

ランチ・ディナーの営業、仕込み、経営管理など

開業後はやらなければいけないことがたくさん。

だからこそ、開業前に基本的な知識だけはつけておくべきです。

今回は、『開業前に知っておきたい！集客のイロハ』と題し、

看板・チラシ・インターネットなど集客手法のキホンをお教えします。

・まずは「注意」をひくこと！そこから入店させるめには？

・やっぱりインターネットの知識も必須。０から学ぶネット広告

・フェイスブックを始める最適なタイミングとは？

・私は「これ」で成功しました！先輩オーナー、実践の集客方法を紹介
