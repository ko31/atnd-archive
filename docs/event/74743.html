内容

交流会（Music with 規格外　打ち上げ）

「交流会」はお集まりいただいた方々の親睦と情報交換の場です。また、SHLPHEEDさん開発の「C’MOY＋ダイヤモンドバッファ ヘッドホンアンプ」の基板も頒布されます。
またご自身で開発されたキットなども持ち込み大歓迎！　頒布もどしどし行ってください。

真空管アンプ設計勉強会（ポタ研スペシャル）

「真空管アンプ設計勉強会」は、＠ぽっけが月1回開催しているものをポタ研向けに編成した特別内容となっています。
※定例の勉強会では、約半年（6回）程度で最低限の一球アンプの設計ができることを想定しています。また２年程度で大型のアンプが組み立てられる（シャーシ加工などの知識も込み）ことを目指します。ご興味ありましたら当日「＠ぽっけ」までお声かけください。

  • 真空管の動作（第2回開催内容）
    1. 三極管のVP-IP特性、VG-IP特性
    2. 三極管の「三定数（gm･rp･μ）」
    3. 五極管のVP-IP特性と、三極管との動作の違い
  • 基本の３回路（第3回開催内容：予定）
    1. カソード接地回路
    2. プレート接地回路
    3. ゲート接地回路
    4. 第4の回路！？
  • 真空管の特性カーブ(VP-IP・VG-IP)測定法（特別内容）
    1. 測定法概略
    2. 測定に必要な器具

※特別内容は通常の勉強会では行いません
※勉強会は板書中心の講義形式となっています。当日は筆記用具をご持参ください。（忘れた方の為に実費にてご用意も致しますが数に限りがあります）
