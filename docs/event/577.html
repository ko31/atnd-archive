title OOPJog#2
catch 大切なのは継続です
description

今回は「OOPの歴史」をメインテーマに、各自得意とするオブジェクト指向言語についてリプレゼントしながら、OOPの歴史の輪郭を浮き彫りにしていく感じで走っていきましょう。
イベントの概要については後述の「Jogについて」をご一読ください。

[ 公式タグ ]：Jog (Jog系のイベントが隆盛してきたらoopjogにするかも？？)

[ 参加資格 ]：オブジェクト指向言語の使用経験がある方、もしくは使用経験が無くても興味の有る方

[ 走行コース ]：こちらをご覧下さい。（@takiuchi 再利用させて頂きます。）

[ 集合場所 ]：東京メトロ日比谷駅A11番出口から出て、ナチュラルローソン、ロッカーがある付近、又は出口を出てすぐの交番前あたり(開始時間が近くなった場合移動しています)
遅れた場合はある程度待ちますが、走り出している場合は皇居を反時計周りに走り、走者と合流することになっています。

[ 準備運動 ]：A11出口を出て日比谷公園に入ってすぐ、ゴミ箱が近いベンチの辺りで最低20分前くらいから準備運動を行うつもりです。

[ 持ち物 ]：基本的にジャージなどの服装、ランニングシューズ等は各自ご用意ください。着替え場所として駅構内のトイレを使用していますが、着替えたまま現地にいらっしゃる人も多いです。
また、走行後、体を冷やすといけないので汗を拭くタオル等を持参することをオススメします。
持ち物は駅構内ロッカーに預けることになりますので、財布／携帯等の貴重品を持参したい場合は、携帯可能なポーチやショルダーバッグのご利用を検討してください。


[ 名刺交換について ]：集合後、軽く自己紹介を行う時間を設けたいと思います。その場でpokenや名刺などを用いた名刺交換を行ってください。

[ 設備利用について ]：シャワー等の設備利用は今の所考慮できていないです。
今回も21時集合ですが、終了21:30だとシャワーを借りれる場所があるとのこと（@kwappa）ですので、希望の声によって決定しようかと思っています。

[ ポリシーについて ]：健康を目的としたイベントでもあるので、走行後の公式な居酒屋などでの懇親会は予定しておりません。

[ Jogについて ]：takiuchiさん発祥の企画です。MerbJogからフォークして定期化したイベントという位置づけです。
詳しくはtakiuchiさんの勉強Jogのすすめ – s21g::Hello, world!や、私のOOPJog#1完走！！などのエントリをご覧下さい。
