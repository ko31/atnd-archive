いのちの教室

「1人でも多くの人に心肺蘇生方法を伝えたい」「1人でも多くの人に安全な環境の下、スポーツを楽しんでほしい」「万が一事故が起こっても誰もが冷静に対処できる環境を作りたい」そんな想いで活動している ATHLETE SAVE JAPAN より特別講師を招き 「いのちの教室」 を開催します。
今回は 岡田麻央さん にお越しいただきます。

・・・・・・・・・・・・・・・・・

◇AEDの使い方を知り『いのち』を守る

日本では年間で約7万5000人が心原性心肺機能停止（突然死）で死亡しています。その数は、一日に約200人、7.5分に1人が心臓突然死で亡くなっています（※）。

突然に心臓や呼吸が止まったり、これに近い状態になった時に、胸骨圧迫(心臓マッサージ)や人工呼吸を行うこと、AED(自動体外式除細動器)を用いた電気ショックを行うことで突然死からの「いのち」を救うことができるのです。
心肺蘇生法は子どもから大人まで、知っていれば万が一の事故の時、大切な「いのち」を救うことができるのです。

(※)一般財団法人日本AED財団より引用

◇アスリートから学ぶ「夢」「挑戦」「生きること」

岡田麻央さんのトークセッション では、バスケットボールを始めたキッカケから現役時代のこと、スポーツや幅広い経験から感じたことなどをお話しいただきます。

「夢」や「挑戦」について語っていただき、「生きること」について一緒に考える場になればと思います。
