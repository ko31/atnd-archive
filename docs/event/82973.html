<span style=”text-decoration: underline”>「資産価値」にこだわったマイホームを買えば住宅ローンがそのまま貯金に

前人未到の人口・世帯数減少時代。その時あなたの生活は…

あなたは今、人口も世帯数も減少する時代に突入していることを実感しているでしょうか。私たちは歴史上、誰も体験したことのない、日本が初めて経験する人口減少の真っただ中に身を置いています。

これからマイホームを買っても、30年後には空き家率が40％になるという試算もあるほど、家余りの時代がくることが予想されています。その事態に陥ったら、あなたにどのような影響があるでしょうか。

郊外住宅の悲鳴。本当にマイホームを買っていいのか

「動くに動けない…」、とある郊外住宅からの悲鳴です。

1980年代、郊外に夢の新築マイホームを求めるブームが巻き起こりました。あれから30年超が経過、住宅ローンを払い終えこれから始まるバラ色の老後が一転、悲痛な叫びに変わっているのです。

いったい何が起こっているのでしょうか。これから本当にマイホームを買ってもよいのでしょうか。

決して不安を煽りたいわけではありません。事実として買ってはいけない住宅があること、そして一方で資産価値に優れた住宅もあることをそのままお伝えしなければならないのです。

「資産価値」にこだわる新しい住宅購入法で自由な住生活を

だからこそ私たちは、その解決策としてマイホームの新しい買い方を提唱します。それはずばり「資産価値にこだわる」ことです。
マイホームの買い方を理解すれば、住宅ローンがそのまま貯金となり、ローンにも家にも縛られなくなるのです。

  • 今、水面下で起こっている人口大移動問題とは。郊外住宅から悲鳴が起こる理由
  • 空き家が増える時代になると、あなたの暮らしがどのように変わるのか
  • 資産価値にこだわるとは具体的になにを意味し、どのように実践するのか
  • 長い目でみて安心なマイホームを手に入れる「これからの新しい住まいの買い方」とは
  • 物件よりももっと大切なこととは。多くの人が陥る致命的な誤りとは。買ってはいけない住宅とは
  • なぜ今この話をするのか。大転換期を迎えている国の住宅政策とは
  • なぜ不動産屋は押し売りをするのか。不動産屋に騙されないため知っておくべき業界の仕組みとは
  • 資産価値にこだわるマイホームを手に入れたあなたが手にする未来とは

すべて丁寧かつ具体的にお答えし、住宅を買う「方法」を教えます。なぜ「今」この話をするか、その意味もお分かりいただけるでしょう。絶対に知っておくべき7の秘訣、どうか持ち帰ってください。

<span style=”text-decoration: underline”>漠然とマイホーム探しに踏み出してはいけない

「マイホームを買う時に何に気を付けたらいいの？」「どうやって不動産屋を選んだらいい？」「住宅ローンを組むのが怖い」「まだ買うのは先だけど、漠然と気になっていることがある」「すでに物件を決めかかっているけど気になることがある」…

いまさら聞けないこと、こっそり聞きたいこと、どうぞすべて吐き出してください。すべての疑問にお答えします。まずはあなたのマイホームの疑問を解消し、資産価値を生む住宅を手に入れる方法をあなたのものにしてください！

<span style=”text-decoration: underline”>物件の売り込み1秒たりともなし。もっと大切なことをお伝えします

講演で物件を紹介したりしつこく売りつけることは絶対にしません。せっかくのお時間、物件情報に1秒たりとも割きたくありません。マイホームの買い方そのものを全力でお話させていただきます。

物件を売り込むことはもはや私たちの仕事ではないのです。どういうことか、その答えも含めてぜひ講演を聞いてください。

「不動産屋は怪しい」といわれ、このような当たり前のことを強調しなければならないことは悲しい限りです。私たちからなにかを売りつけることはありません。どうかお客様はご安心なさってご予約・ご来場ください。

<span style=”text-decoration: underline”>ベストプロに選ばれた気鋭のバイヤーズエージェント「加藤豊プロ」が講演

これまで1,000件以上の不動産取引を見てきたミトミ代表の加藤。悪しき慣習・常識を打ち破り、不動産会社は物件（モノ）売りではなく、サービスを提供するエージェントでありたいと考え、精力的に活動しています。

マイベストプロにも不動産売買の専門家として認定されました。取材や執筆も精力的に行っています。

もともと出版業界に身を置いていたことなどから不動産業界の「怪しい」と思える部分こそ、明確に情報発信しています。
