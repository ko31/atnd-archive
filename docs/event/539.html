Debug Hacks Conference FAQ


(1) Debug Hacks Conferenceってなんですか？
オライリージャパンから出版される
DEBUG HACKS――デバッグを極めるテクニック＆ツール
の出版を記念して、著者らによる講演と展示即売会です。


(2) Debug Hacks Conferenceの参加資格はありますか
誰でも参加できます。無料です。


(3) Binary Hacksを持っているんですが参加できますか
誰でも参加できます。無料です。


(4)　表紙のブタみたいなものは何ですか？
蚊遣り豚といって、蚊取り線香を中にいれるものです。


(5)　人生をデバッグしたいのですが
Debug Hacksは主にソフトウェアのデバッグについて
書かれた書籍なので、残念ながらお力にはなれません。
しかしひょっとしたら人生の達人にカンファレンスで
出会うかもしれませんね。
