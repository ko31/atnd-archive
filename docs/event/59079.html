catch PMOになるための実践的なスキルが身に付くeラーニング特典付き！（セミナー参加者全員）
description

システムの開発を行う上で、PJTを完遂するためにPMやPLの負担増が課題視されており、
その解決方法の１つとしてPMOの役割が注目されており、ニーズも高くなっています。
その結果、従来のIT業界のキャリアパスでは注目されていなかった、
PMOというキャリアの選択肢も出てきています。

□　そもそもPMOとは？


企業やスキルに応じてその役割は多少異なりますが、
主には、PJTの品質向上のためのプロジェクトマネジメント手法の標準化や、
各PJT間の調整、また、PJTのモニタリングやPMのサポートを担う方を指します。

□　こんな方にオススメ


■PMやPLとして就業中でマネジメント経験を活かして新しいキャリアやプライベートを重視した働き方を考えている方
■開発現場でSE・PGとして就業されている方で、次のキャリアとしてPM・PL以外の道も知りたいという方

【Prepトレーニング（会議ファシリテーション）】
会議を開催する上で必要となる基礎的な準備の知識、スキルを「TP4」という独自のフレームワークに沿って学びます。準備において最も重要な「Purpose（目的）」の設定方法と会議アジェンダとシナリオ作りの「Process（手順）」にフォーカスし、事例を交えて説明していますので、実践的な学習内容となっております。また、各章末には確認問題も用意していますので、学習の理解度を測るに最適です。

【プロジェクトマネジメントが失敗する理由と リスクマネジメント】
プロジェクトマネジメントがなぜ難しいのか、どのような原因で失敗するのかを学び、そこからリスクマネジメントの重要性やポイントを学習します。プロセスやツール・技法等の実務的な知識エリアの学習ではなく、プロジェクトマネジメントとリスクマネジメントの概念的な講義になります。

■eラーニングを受講するメリット
・プロジェクトマネジメントについて、体系的に学ぶことができる
・実例を元にした受講内容となっているため、実務に活かせる
・未経験からでもPMOへチャレンジできる可能性が広がる
