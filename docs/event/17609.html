description

業務でECサイトを構築している方が集まり、構築していて困ったこと、
ショッピングカートに望むことなどを話す会です。

今回は「SOYShop」の開発元の日本情報化農業研究所より、SOYShopの
開発に主に関わっている「齋藤毅」さんをお招きしての開催となります。

SOYShopの今ある機能を知ると共に、開発元と直接意見を交わす絶好の
機会です。是非ご参加ください。

今回は内容を実のあるものにするために、募集人数が6名と少なくなって
おりますので、参加申し込みはお早めにお願いいたします。

【開催日】

2011年7月18日(月曜日)　13：30～17：00(予定)

【対象】

  • 制作会社にお勤めの方、またはフリーランスの方
  • 業務でECサイトの構築を行われている方
  • (現時点で利用していなくても)　SOYShopに興味のある方

【SOYShopとは】

SOYShopは国産のショッピングカートで、デザインとサイト構造の自由度の
高さや柔軟な拡張性が売りのシステムです
http://www.soyshop.net/

【募集人数】

6名

【参加費】

無料

13:45～14:30 SOYShopの紹介　(齋藤毅)
14:30～14:45 休憩
14:45～17:00 意見交流会
