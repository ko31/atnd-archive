定員を増やしました。また、開始時間が10分後ろにずれていることに御注意ください。

Javaの名著である「Effective Java」の訳者であり、「Jave SE 8実践プログラミング」の翻訳や「プログラマー“まだまだ”現役続行」等の著書でも有名な柴田芳樹氏が講演（ 第７回とくしまＯＳＳ勉強会 ）のために、来徳されます。
また、柴田氏は「 徳島には、ジャストシステムに勤務していた1997年5月から1998年4月までの1年間住んでいました 」とのことで、徳島にもゆかりのある方です。

講演後に、柴田氏を囲んで交流を深めるために懇親会を企画しました。

第７回とくしまＯＳＳ勉強会に参加できなかった方もご参加お待ちしております。

また、送別会や徳島マラソン(3/22)の壮行会と重なるため、お申し込みはお早めにお願いします。
