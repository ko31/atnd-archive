概要

Wolfram Research社による数式処理システムかつプログラミング言語であるMathematicaについてみんなでわいわいやる勉強会です。
Mathematicaをお持ちでない方でもぜひご参加ください。
LTをしたいという方も募集しています。Mathematica / Wolfram Alphaの話題だけでなく、技術系であればOKですので、興味がある方はぜひ@aomoriringoまでご一報ください。

不明な点はTwitterアカウント@aomoriringoまでお願いします。
