第1期Social & Smartphone app school(SS app school)をスタートいたします。
※改めまして日程確定致しました！

第1期は「mixi Touchアプリ開発」講座です。（全4回）
講師に株式会社フォリフ エンジニアの中村様をお迎えして実施します。
今回は記念すべき第１期ですので参加費は無料です！
ぜひお気軽にご参加ください！
参加者全員でmixi Touchアプリを作りましょう！！

※参加希望者はこちら→http://eventforce.jp/event/10422から申し込みしてください※

イベント・セミナー内容
■内容
　－全4回の参加でiPhone向けブラウザゲーム(mixi Touchアプリ)を実際に作れるようになるのが目標です
　－Android向けの移植作業についても触れる予定です！
　－講師に株式会社フォリフ　エンジニアの中村様をお迎えし、ケーススタディとして、実際のアプリを題材にして学んでいきます
　－講義間には課題がありますので短期間でしっかり力をつける事がでます
　－第1期目の開催ですので初めての方ばかりの参加です。お気軽に参加下さい

■日時
　第1回：2月9日(水)「スマートフォン向けJavaScriptライブラリとmixi for Touch」
スマートフォン特有のJSライブラリの紹介、簡単なサンプルとmixi for Touchについて説明します。
　第2回：2月16日(水)「スマートフォンブラウザゲームを作ってみよう(1)」
第1回で学んだライブラリを使ってスマートフォンブラウザでゲームを作って頂きます。
　第3回：2月23日(水)「スマートフォンブラウザゲームを作ってみよう(2)」
第2回に引き続きスマートフォンブラウザでゲームを作って頂きます。
　第4回：3月9日(水)「発表会」
今回の講座で開発したゲームを皆さんに発表して頂きます。
　※各回：19：30～21：30(自習時間として22:30まで解放)

（http://www.tsrental.jp/location/shibuya/map.html）

（１）サーバサイド：PHP Ruby Perl Python Java C++の経験 もしくは
　　 クライアントサイド：Javascript HTML5 Flashの経験 を

（２）3年以上(30代)、2年以上(20代後半)、1年以上(20代前半・学生)　　もっている方

■定員：20名
