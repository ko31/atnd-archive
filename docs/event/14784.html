description

参加申し込みは終了いたしました。

これは、父の会のお父さんたちが、母の日にお母さんと子供たちにサービスすると言う企画です。

当日、小戸公園でのBBQではお母さん達はお客様。

ちなみに、父の会の会員限定イベントとさせてもらいます。
また、幼稚園の公式行事ではありません。

なるべく手ぶらで参加できるように

企画していますが、次の点は各自でご準備ください。

  • 敷物、椅子やテーブルなど。
  • 飲み物(アルコールに関しては飲酒運転は厳禁です)
  • ご飯などの炭水化物(肉と野菜はこちらで準備します)
  • どうしても焼いて食べたいモノ
  • 使い慣れたお皿やお箸(ある程度の紙皿やコップ、割り箸は用意しますが数に限りがあります)
  • 使い慣れたトイレットペーパ(現地のトイレには紙が備え付けられていません)

なるべく安価に済むように

企画していますが、以下の金額になります。

  • 中学生未満は無料(高校生以上は有料)
  • 1,500円/人
  • 駐車場代は各自でお支払い願います。最大で500円打ち止め

したがって、ご両親と幼稚園児一人の場合には、3,000円+駐車場代となります。
当日会場に持参してください。

お子様は、基本的に各家族で

とても魅力的な遊具も多くあります。
ただし、海辺でもありますので、各自お子様からは目を離さないように楽しく過ごせるように注意しましょう。
BBQ会場と遊具の場所は少し離れていて目が届きません。
お子様を遊具で遊ばせるのはある程度食事が終わってからということにしましょう。
子供らが一緒に散らばってしまうとBBQどころじゃなくなっちゃいますので。

申し込みは

人数が重要になります。
このページから申し込まれる場合には、高校生以上◯名、小学生以上◯名、園児◯名。と書いてください。
複雑ですみません。入場申請する人数は小学生以上をカウントするので複雑な書き方になっています。

キャンセルについて

通常のキャンセルは、前日までに連絡をお願いします。
ただし、お子様の発熱などの場合は例外として8日朝8時までに電話連絡してください。
天候調査は7日1500に実施します。結果は直ちにメールで連絡します。
中止の場合には順延代替行事はありません。

開催時間帯を変更しました。
一般参加者は、11時から15時となります。
