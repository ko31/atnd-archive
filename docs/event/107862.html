oreRevo対戦会50回記念！トーナメントやフリー対戦で試合しまくれ！

毎週水曜日でお馴染みの「oreRevo x CYCLOPS」もお陰様で開催50回目を迎えました。
普段、平日では参加出来ない方達もこの機会に集まって一緒に対戦しましょう！
もちろん初心者の方から上級者の方までどなたでも大歓迎です。
19時にはイベント終了予定なので、その後はメシ食べにいくも良し、アケに行くも良し、直帰するもよし
