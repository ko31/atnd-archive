4/25、4/26秋田開催！占導師による「より良い未来を創っていくための氣質＆運期セミナー」

description

占いからさらに一歩進んだ占導師による、秋田で初のセミナー・鑑定を開催いたします。
占導師は、占って当てて終わりではなく、その人に合ったより良い未来に導く助言までを行います。

この2日間のセミナーでは、生まれ持った自身の性質を知り、その活かし方、他の方との円滑なコミュニケーション法を知ることができます。また、自分の運期の流れを知り、その時での最適な行動の仕方を学ぶことができます。

自分を知り、時の流れを知り、最適なタイミングで行動にうつす。
そんな知恵を知り、氣づくことで、自分の力でより良い未来は創っていけるのです。

講師を務めるのは、四国陰陽師末裔の占導師幸輝氏。この道30年近く、鑑定件数は4万件以上と実績十分。陰陽学を取り入れた－から＋への転換方法やバイオリズムを知ることで、楽（ラク）に生きられるようになったと、参加者からは大好評。セミナーや鑑定などで全国で人気の鑑定士ですが、日本で足を踏み入れていない地は、残すところ新潟と秋田のみ。この度新潟でも開催が決定し、今回最終地の秋田での開催となります。

「未来は変えられる。未来は自分で創れるもの」

より良い未来を自らの手で創っていける人を全国に増やしていきたい。自分の力で幸せになれる人を増やしていきたい。その想いを胸に全国で活動されている、愛で満ち溢れた幸輝氏のセミナーです。

特に秋田は鬱や自殺者が多い県と言われており、他の方からはネガティブ、保守的というマイナスのイメージを強く持たれているようです。秋田出身の私としては、秋田の人は心根が優しい人たちばかりだと思っています。
但し、自分の内に溜め込んでしまったり、コミュニケーションがうまくいかなかったり・・・。
私は過去を振り返った時、小さい頃の家族との想い出は、楽しかった想い出より、両親がケンカし合っていたイメージが強く残っています。自分自身やパートナーだけではない。子供、両親、周りの大切な人たちにも知らず知らずのうちに影響を与えてしまっているものなのです。

そんなことを想い出しつつ、実際に私も東京で受講したこのセミナーは、ぜひとも秋田の方にこそ知っていただきたい内容かと思い、今回協力開催することに至りました。

幸輝氏は、2011年の東日本大震災の時もちょうど仙台でイベントを開催中だったとのこと。そこで被災され、その時より仙台を第2の故郷とし、東北に強い想いを抱き、今でも仙台で多く活動されています。
その時の経験から【命】についてさらなる大きな学びを得、【命】の大切さについて様々な形で発信を続けられている先生です。

資質によって、人生は約50％近く決まってしまうと言われています。
自分の資質を知り、無意識に身についてきたその癖を良い方面に使いこなし、自分（自然）のリズムに従って生きることで、生き方がより楽（ラク）になるでしょう。
事実は一つ。でも、その受け止め方、捉え方によって、その先の未来は変わってきます。
その先の人生をマイナスにするにも、プラスにするにも自分次第なのです。

参加すると、良い「氣づき」が得られることは間違いありません。
秋田の皆様、ぜひ一度ご参加ください！
