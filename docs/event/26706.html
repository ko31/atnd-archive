description

「アジャイルサムライ ― 達人開発者への道」 の読書会を行います。
順番に輪読していき、セクションごとに本の内容についてディスカッションする形式ですすめていきます。
実践的なアジャイルプラクティスについての章に入ってきました。現場でできる事取り入れられる事をみんなで考えていきましょう:D

開始時間が30分遅くなります（終了時間は同じです）
道場主の都合で申し訳ありませんがよろしくお願いします。

内容・範囲

輪読形式での読み合わせを行います。
読み合わせ後、セクションごとに参加者で疑問や意見を話し合っていく形式を想定しています。

第17回は、以下の範囲を予定しています。

  • 「第11章 現場の状況を目に見えるようにする」冒頭 ～

スケジュール

20:15 – 20:20 休憩
20:20 – 21:00 読書会（続き）
21:00 – 21:10 撤収作業

※時間は目安であり、休憩時間が前後することがあります
