3/13開催 はじめてのAlibaba Cloud ～Alibaba Cloud入門編～ Alibaba Cloudセミナーのご案内

セミナー概要

Alibaba Cloudとは、ご存知中国の巨大ECサイト淘宝網を支える、アリババ社が開発したクラウド基盤サービスです。2016年12月には日本リージョンを開設しました。 SBクラウドはそのAlibaba Cloudの日本リージョンの運営およびサポートなどを行っています。

3月13日(火)のAlibaba Cloudセミナーは、はじめてのAlibaba Cloudと題し、クラウドサービスを初めて利用される方や、クラウド初心者の方に向けた入門セミナーとなります。パブリッククラウドのメリットやAlibaba Cloudの特長、プロダクト・活用事例の紹介を行います。 パブリッククラウドやAlibaba Cloudにご興味をお持ちの方、ぜひこの機会に当セミナーにご参加ください。

セミナーアジェンダ

オープニング：16:00～16:15

「オンプレミスからクラウド移行のメリットや課題」
　SBクラウド マーケティング企画部 大石 裕一

クラウドへの移行が加速する昨今。オンプレミスとクラウド、それぞれの特徴を比較するとともにAlibaba Cloudの概要とメリット、ビジネスへの活用方法などをソフトバンクとアリババの合弁会社であるSBクラウドがご紹介します。

第1部：16:15～17:00

「はじめてのAlibaba Cloud ～Alibaba Cloud入門編～」
　SBクラウド株式会社　プロダクト技術部　ソリューションアーキテクト　森 真也

クラウドサービスを初めて利用される方や、クラウド初心者の方に向けて、クラウドとは？というところから、Alibaba Cloudの特長、コアプロダクトの紹介から、最近新たに加わったビッグデータに関わるプロダクトまで、実際の構成例や活用事例、デモと合わせてご紹介します。
この機会にAlibaba Cloudの特長やソリューションを理解する機会にしていただければと思います。

質疑応答：17:00～17:20

※質疑応答の後、個別のご相談も承っております。
