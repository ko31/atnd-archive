★講演内容

Python 標準ライブラリを利用して、
CSVデータを加工する小さなプログラムを作ります。
プログラムの実装を通して、ソフトウェアの作り方について最初の一歩を学びます。

・基本的な文法のおさらい
・プログラムの仕様
・csvモジュールの利用
・jsonモジュールの利用
・urllibモジュールの利用
・zipモジュールの利用
・成果の確認
・実際の開発現場での利用例

※ 内容は変更になる可能性があります。

★講演者プロフィール

■児島　孝典 氏

繰り返し作業を自動化するのが好きなソフトウェアエンジニア。
Pythonを使い出したのも、日々行うビルド、デプロイ作業を自動化するため。
最近はGoに軸足を移しつつあるが、
いまでも小さいプログラムをサッと書くときはPythonを愛用している。
