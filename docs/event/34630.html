WordPressの勉強会 – 2012年12月23日（日）WordBench埼玉勉強会

（2012年12月22日追加）
定員いっぱいになりましたが、満席でもどうにかしますので、補欠の方もお気軽にお越しください！

WordPress 3.5 がリリースされるというタイミングで、WordBench埼玉を行います。
時期としてもクリスマスが近いですので、3.5リリースパーティとクリスマスパーティも兼ねています。
なお、WordBench埼玉12月のWordPress交流会については、
WordPressユーザーの交流会 – WordBench埼玉12月＆WordPress3.5リリースパーティ
より参加表明ください。

WordPressに関するセッションは午後からですが、午前中から集まってWordPressについて話し合うアンカンファレンスも行います。日頃の疑問点など、ぜひ詳しい人に聞いてみましょう！

9:00 – 12:00
有志によるアンカンファレンス
（WordPressについて聞きたいこと・話したいことを自由に話し合える場です。）

13:00 – 13:30
WordBench埼玉から参加する人の受付

13:30 – 14:15
「WordPress 3.5 の新機能」
おでっせいさん（ @odyssey ）
WordBench埼玉モデレーターで、WordPress 日本語版作成チームのおでっせいさんに、2012年12月にアップグレードのWordPressのバージョン3.5版について、その新機能の解説などしていただきます。
http://8bitodyssey.com/

14:30 – 15:15
「学生のWordPress利用状況」
上田和真さん（ @ueda1023 ）
フリーランスのウェブデザイナーで、学生を対象としたWordPressワークショップを開催している上田さんに、学生さんのWordPress導入事例についてお話いただきます。
http://kazuma.techcy.jp/

15:30 – 16:15
「レビューブログの書き方講座」
鈴木珠世さん
数多くのブログを見てきている、アフィリエイト仲介業者大手のリンクシェア・ジャパン株式会社の鈴木さんに、どのようにレビュー記事を書くべきか、アフィリエイト事例も踏まえてお話いただきます。
http://www.linkshare.ne.jp/

16:30 – 17:15
「BuddyPressを使って、超大規模ブログポータルを作ろう！」
ITかあさん（ @chihiro_kaasan ）
WEB系プログラマーITかあさんに、WordPressで複数サイトが作れるマルチサイト機能と、オープン型SNSが作れるBuddyPressプラグインと、それらに関連するデータベースの話をしていただきます。
http://www.kaasan.info/

17:30 – 18:00
１人５分のライトニングトーク

・「HubSpot」について
藤祐太郎さん（ @3Studio ）

・コワーキングスペース公式サイトをWordPressで作る
星野邦敏さん（ @khoshino ）

・WordPressで提供する Web API
Yuko Toriyamaさん（ データベースに接続できません ）

・5分で出来るChrome Extensionsの話
小島勝茂さん（ DESIGN SATELLITES ）

・明日からすぐに役に立つWordPressを使ったWEBサイト制作のコツ
大串肇さん（ megane9988 ）

18:00 – 20:00
WordPress3.5リリースパーティ＆クリスマスパーティ

会場：
埼玉県さいたま市大宮のコワーキングスペース「7F」

参加費：
アンカンファレンス＆WordBench埼玉：500円（午前中から来ても午後から来ても500円です。フリードリンクや無線LAN・電源など有り。）
WordPress3.5リリースパーティ＆クリスマスパーティ：3000円（学生は2000円）（ケータリングやアルコール・ソフトドリンクの実費です）
（WordBench埼玉12月のWordPress交流会については、
WordPressユーザーの交流会 – WordBench埼玉12月＆WordPress3.5リリースパーティ
より参加表明ください。）
