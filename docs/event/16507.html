description

みなさんはこの世界にいったいどれぐらいの人々が暮らしているかお解りだろうか。
2010年10月現在の国連推計によると69億人もの人間がこの世界に暮らしており、
2011年中には70億を越えると考えられている。
そんな人口増加の一途を辿る社会の一員として私はふと立ち止まって考えた…

　　　　果　た　し　て　何　人　と　ふ　れ　あ　っ　た　の　だ　ろ　う

我々は常日頃からお茶の間を介して人と人とが円滑なコミュニケーションを行う…そんな暮らしを提供する合意形成支援を夢見てきた。
そして…私は志を同じくしたColleagueを集めこの団体を結成した。
我々は新しい３つのコミュニケーションの形を提案し、
21世紀をパラダイムシフトする事を考えている。
以下に、その概要を示す。
賛同なされる方は是非我々のイベントにお越ししていただき、そのideaとactionを直接肌で感じてほしい。

Social Communication=ひとつひとつの地域に密着した深いキズナ
（社会的コミュニケーション）
Universal Communication=日本だけじゃなく欧米や東南アジア諸国を巻き込むglobalなキズナ
（普遍的コミュニケ-ション）
Innovative Communication=既存のinteractionに捉われない新しい形のキズナ
（革新的コミュニケーション）
