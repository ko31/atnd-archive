ほぼ毎日皆さんが行う「会話」。
会話の中で沈黙が続いたり、相手の考えていることが分からなくなってコミュニケーションがうまくとれなかったり・・・
そんな時がありませんか？
相手が今どんなことを考えているかが分かり、そしてその事を相手に話してもらえるような会話が、お互いの関係をより良いものとします。
この講座で「相手の話を自由に引き出す」話法を身につけ、円滑な人間関係を築きませんか♪

～セミナーの内容～

・沈黙の時間は嫌な時間ではない
・相手に満足してもらう話法
・全てが明るみになる質問の仕方
・気持ちをふさいでいる相手から本音を聴く

～このセミナーはこんな方におすすめです～

・コミュニケーション能力を高めたい
・人の話を聞く機会が多い
・相手の本当の気持ちを知りたい
・快適な人間関係を築きたい
・営業成績を上げたい
・スムーズな会話を行いたい
・人が集まるような自分になりたい
etc…
