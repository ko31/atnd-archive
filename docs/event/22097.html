概要

アクセスが増えると I/O 処理がボトルネックになりがちなソーシャルアプリ運用。アプリの設計によってはランダムアクセスが爆発的に増えるので、I/O 性能を向上させた環境で運用する必要があります。

本セミナーでは、ソーシャルアプリ運用時の I/O 性能向上に効果的な ioDrive と KVS の運用を中心に、at+link が提供している「 at+link アプリプラットフォーム 」のご紹介を行います。

ソーシャルアプリを専用サーバで運用するメリットに加え、at+link アプリプラットフォームの特長である Key-Value Store 「okuyama」を使ったサービスや事例のご紹介に加え、ioDrive 開発元であるフュージョンアイオー株式会社の大浦氏を招き、ioDrive のメリットについてお話を伺います。
