ver2012_02

「ボードゲームサークル響」は、
沖縄で海外製アナログボードゲームで遊ぼうという会です。
主催（RX）が遊びたいがために勝手に開催しています。

ここでのボードゲームは、海外製アナログゲームを主に指します。
日本では昔から人生ゲームやモノポリーが有名ですが、
ここ10数年で毎年大量に発売されている現代のボードゲームは、
もっと駆け引きで頭使ったり、新しい楽しさがあります。

参考記事：http://d.hatena.ne.jp/nakarx/20111017/1318853806
遊んでる写真：http://goo.gl/cSnAW

毎月、オープンに参加者を募集するゲーム会を開いています。
RX個人分でも100種近くゲームを揃えておりますので、
飽きない程度に皆で遊び倒してます。 持ち込みがあると喜びます。

海外製アナログボードゲームとか、未知もいいとこですが
常連も交えて、最初にルール説明するようにしてますので
初心者でも大丈夫な感はあるかなと。
戦略と運が乱れ飛ぶ妙味のある楽しさへ、興味ある方はどぞ。

参加者は学生さんが多めですが、大人も結構来ます。
特に制限は無いので、沖縄で対戦する面子が
なかなか集まらないといった人にもオススメ。
一回顔だしてみて、それから合うかどうか決めて下さいな。

ブログにも色々書く予定ですので、
雰囲気を調べるのに使ってください。
http://hibiki-gamers.info/

skype, mixi, facebook, googlegroupなど、
各種ネットツールにて告知してます。詳しくは上記URLで。

ぜひ、ご友人などお誘い合わせの上、一緒に遊びましょ。
