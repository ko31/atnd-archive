description

先日、スマートフォンでのカード決済サービスで先行していたSquareが、日本上陸を果たしました。迎え撃つ楽天やPaypal, Coineyなども、手数料を下げたりリーダーを無料にするなど、競争が激化しこれから本格的に普及が進んでいきそうです。

そこで、そのトレンドを知りたい＆本格的に使ってみたいエンジニアが集まる勉強会＋フリーマーケットを開催したいと思います！

正直、手探りでの開催なので、疑問点や要望などはどんどん問い合わせお願いします。

スマホカード決済勉強会

スマホカード決済の概略と現状、サービスの普及で開かれる可能性などについて参加者と勉強できればと思います。

発表者を募集します！自薦他薦を問わず連絡下さい！
（各決済サービスのAPIで遊んでみるネタや、少額決済トレンドの話などをして頂ける方、大歓迎です）

福岡Techフリマ

スマホカード決済のトレンドを感じ取り、早速リーダーを入手したエンジニアも多いとおもわれる今日この頃、自分のカードで100円を決済して、100円銀行振り込まれた〜ヽ(｀▽´)/ 、という程度で満足したりしていませんか？

Squareなどでアカウントを作っている方は、是非出店してみませんか？勉強会に参加するTech系の人がお客様となりますので、その人達に買ってもらえそうな品がベターです。

なお、Squareなどは1週間程度でリーダーの入手からアカウントの作成まで可能ですので、今からでも間に合うかも（保証はいたしかねますｗ）。

出店内容の例:

  • 積読＆塩漬け状態の技術書
  • イベントでもらったDuk◯とかS◯aserとかのTシャツ
  • LEDブリンクしただけでホコリをかぶっているAr◯uino
  • 3Dプリンタの出力時に発生したしたボツ作品
  • 技術コンサル15分間 などなど

※キワモノ系の出展は各自の良識で判断頂くようお願い致します。

出店者以外はお客様として買い物をお楽しみ下さい！（もちろん要クレジットカード）
実際に購入される様子を見るなど、野次馬見物していただいても構いません。

なお、あまりにも出店者が少ない場合は中止になる可能性があります…。
