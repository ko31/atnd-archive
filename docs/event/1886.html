「愛媛Android勉強会 part.2」のお知らせ

８月２２日の「日本Androidの会四国支部勉強会」参加の皆様からの
ご要望にお答えし、第二弾として android のビジネスモデルを考え
る場を企画いたしました。日本Androidの会事務局長の来松に合わせ
る形となり急遽実施です。

な飲み会（懇親会）です。
