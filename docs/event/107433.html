2019年7月16日に、株式会社リコーと株式会社和久環組にて、「不動産・建築業者向け収益アップ」セミナーを開催します。

本セミナーでは、株式会社リコーより360度カメラTHETAで撮影したパノラマコンテンツのプロモーション活用法や導入事例について、株式会社和久環組｜リノベ不動産よりリノベーション事業収益化のポイントについてご説明させていただきます。

不動産仲介業者様、建築設計施工会社様の収益アップに役立つセミナーとなっておりますので、ぜひご参加ください。
