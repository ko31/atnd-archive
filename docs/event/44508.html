イベント概要

「スマフォト女子会（仮）」は、スマホのカメラを使ってよい写真を撮影してみよう！という群馬の女子会です。

初開催の今回は、みんなでランチをしながら良い写真を撮るコツを研究します。

美味しく、楽しくいろんな写真を撮ってみましょう！

ランチの後は、撮った写真をみんなでお披露目し合って、和気あいあいと意見交換をしてみます。参加者同士でいろいろと交流もできるといいですね〜♪

対象者

  • 女性（学生さん、会社員、主婦などどんな方でも大歓迎です！）
  • スマートフォンを所有している（iPhone、Android）

上の２つは必須条件ですが、それ以外は・・・

  • 写真が好き♪
  • スマホが好き♪
  • ランチ好き♪
  • みんなと交流したい♪
  • 育児中の気分転換に♪
  • 同じようなことが好きな友達が欲しい♪
  • もっとスマホの使い方に慣れたい♪

などなど、この会に興味がある方であればどなたでもOKです！

お子さま連れの方でもOKです！（お店によっては事前に連絡しますのでお知らせください。）

当日持参する物

  • スマートフォン

開催場所

  • 福カフェ（高崎市連雀町71大手前ビル１階）
  • コワーキングスペースSOMETHIN’ ELSE ※ランチ後こちらに移動します。

参加費用

  • 会自体の参加費は無料です。
  • ただしランチは各自注文となります。

タイムテーブル

  • 11:30 〜 13:30：高崎市あら町近辺のお店に現地集合してランチ＆撮影会。
    　　　　　　　　食事を注文したら自己紹介をしましょう。
  • 13:30 〜 13:45：コワーキングスペースへ移動。（徒歩で数分）
  • 13:45 〜 14:30：みんなで写真をお披露目します。
    　　　　　　　　写真を撮るコツ、加工アプリについてなど、
    　　　　　　　　質問したりアドバイスしたりいろいろ情報交換しましょう！
  • 14:30：解散

その他

質問や連絡事項などございましたら、このページのメッセージや SOMETHIN’ ELSE サイトのお問い合わせフォームからお気軽にお知らせください。
