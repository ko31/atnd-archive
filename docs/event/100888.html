description

株式会社クリーク･アンド･リバー社は、ゲーム業界・アニメ業界のデザイナーを対象にKLab株式会社の「ポートフォリオ相談会」を開催します。
ご自身のスキルや魅力をアピールするのになくてはならないポートフォリオですが、実際に作り方について悩まれるデザイナーの方も多いのではないでしょうか。
また、皆さんがどんなものを作って、転職活動をしているのか知る機会もなかなかありません。
そこで今回はKLab株式会社にてヒットコンテンツに携わっているデザイナーをお呼びし、面談形式で一人ひとりに合ったポートフォリオのアドバイスをします！
相談会のほか、ポートフォリオセミナーや実際に使われているポートフォリオの展示なども予定しています。
ポートフォリオの制作でお悩みの方に間違いなく役立つイベントですので、ご興味のある方は是非お申し込みください。
皆様のご参加をお待ちしております！

※応募者多数の場合は抽選となります。

■KLab株式会社■

弊社の特徴は、有名IPのゲーム開発を多数手がけていること。有名アニメや家庭用ゲームなどをモチーフとしたモバイルオンラインゲームを、国内外のたくさんのユーザーに提供しています。 社員一人ひとりの意志を尊重しているため、自分から「こういうことがやりたい！」とアピールし続ければ、チャンスを広げられる環境です。一つ上のステージを目指したい方は、ぜひ弊社をご検討ください！
