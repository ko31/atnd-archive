description

かつて日本にはたくさんのスナックが存在し、そこでは夜な夜な様々な人が人生を語った･･･
「ママ、今日俺会社でこんなことがあってさ･･･」
「俺、これから先どう生きていけば良いんだろう･･･」
「あんた！しっかりしなさい！！」
こんな会話が数多く繰り広げられてきたはずだ。
会社帰りの気軽な憩いの場、それがスナック。
今ではすっかりその存在が薄れてしまったが、慌ただしい毎日を送る現代人にこそ スナックのように気軽に愚痴ったり、相談したり、はたまた隣の席の人と語ったり･･･ そんな場所が必要なはずだ！
時にはママに叱って貰いたい！そんな夜もある･･･

そこでキャリフィットでは「スナックキャリ魂（コン）」を開店。
※あくまでスナック風のイベントですので本当のスナックではありません（笑）

ママやボーイはキャリアコンサルタント、カウンセラーの資格を持つスペシャリスト！
スナックのようにお酒を片手に持ちながら相談するも良し、イベントに参加している他の人と交流するも良し！
特に学生は様々なバックグラウンドを持った社会人の話を聞くチャンス！

社会人でキャリアや人生について話したい方、他の方と交流を深めたい方、学生さんで社会人の話を聞いてみたい方などふるってご参加下さい！
※宗教やネットワークビジネス、ナンパなどの行為は一切禁止致します。そのような行為を見かけた場合はすぐに退出して頂きます。
また、大人数の方々がいらっしゃるため、長いコンサルティングが出来かねますのでご了承下さい。グループワークになる可能性もあります。
