オンラインゲームの開発を一気に加速させる Photon Cloud
今回のワークショップは、Unity と Photon Cloud を使い、「シューティングゲーム」を 「リアルタイム」×「マルチプレイ」 で構築する 【 応用編 】 です。

　
　　　　　　　　　　　　　　　　　 【応用編】ワークショップ内容

複数のキャラクターが同一ルーム内でリアルタイムにシューティングを行うゲームを構築。
オブジェクトを同期し、さらに動的な「シューティング」の要素を付与します。
この作業を手早く実現するために、Photonのネットワークエンジンがどのように活躍する
のか体感してください。Unityでの操作に加え、コーディング部分もレクチャーします。
　
　
※ 受講の準備
本ワークショップは【導入編】を受講なさったことがある方を対象としていますが、
Photon Cloudに触れたことがある方でしたら【導入編】は未受講でも【応用編】から
参加していただいて大丈夫です。
