イベント概要

Ubuntu 15.04のリリースを記念したオフラインミーティングイベントです（主催：グリー株式会社）。

テーマは「手軽に使えるコンピュータ」。Ubuntuの「誰もが無料で、かつ自由に使える」ことを生かした、最近話題のIoTやRaspberry Pi2等の廉価なハードウェアをターゲットにした、「廉価なハードウェアで使える自由なOS」として、いろいろな可能性について紹介する予定です。

前方のセミナー、後方のハッカソンという雑然とした空気とともに、Ubuntuの魅力を語って頂ければ幸いです。

参加費・参加資格等

参加費は無料です。ささやかではありますが、無料のパーティーランチと飲み物を用意してお待ちしております（皆様のご要望によりからあげを増量しております）。

Ubuntuを使ったことがある、あるいは興味がある場合、タダ飯を目当てに来て頂いても問題ありません。

詳細は、 https://wiki.ubuntulinux.jp/Events/OfflineTokyo201506 を参照してください。
