【リピーター続出！】第19回仙台クッキングコン

catch
description

月一で日曜日に開催する、男女交流のお料理イベントです。
男女グループ(４～５人）に分かれて手作り料理を食べながらお友達を作って頂きます。

料理（４～５品）はどれも簡単ですが美味しくてボリュームも満足！
また「フードコーディネーター」資格を保有する料理上級者がお手伝いする為、
料理初心者の方でも安心して参加できます。

一般的な街コンとの大きな違いは

＝＝＝＝＝＝＝＝＝＝＝＝＝＝
小規模（最大３０名）
共同作業をする
イベントが日中
ノンアルコール
＝＝＝＝＝＝＝＝＝＝＝＝＝＝

という４つのことですが、
それによってこんないい所がクッキングコンにはあります！

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

①【空気が穏やか】
皆様「朝からの活動」「料理をする」「ノンアルコール」でも参加OK！という方達ですので、
穏やかな空気感の中で交流を楽しめます♪

②【スタッフの目が行き届く】
「席だけ言われて、あとは完全放置だった・・・」そんな街コンの経験はありませんか？？
クッキングコンは全６テーブルの調理実習室１部屋の中で、スタッフ２～３名が運営にあたりますので、
皆様が困った時にはすぐにサポートに行けます♪

③【人柄がよくわかる】
お酒の場で言葉だけの交流では、本当の部分は見えづらいものです。
「料理」という段取りやコミュニケーションが必要な作業を一緒に行うことで、
どんな人なのかがよくわかります♪

④【１人１人とゆっくり話せる】
とはいえお話の時間もやっぱり必要！調理後の自由トーク時間（お茶・お菓子提供）は
最低１時間～１時間半はありますので（しかも小規模）、
「簡単な自己紹介で終わっちゃった。。。」なんてことはありません♪

⑤【連絡先は必ず交換できる】
自由トーク時間の最後に必ず連絡先交換の時間を設けます。
「合いそうな人がいたんだけど、恥ずかしくて連絡先交換できなかった・・・」なんてことはありません♪

⑥【おいしいものを、お腹いっぱい食べられる】
「街コンで簡単なつまみとお菓子しか出てこなくて、ずっとお腹がすいていた・・・」
なんて経験はありませんか？？
クッキングコンのメニューは担当フードコーディネーターが旬の食材、栄養、量など
あらゆる面から考え、しかも簡単でおいしいレシピにしている為、お食事に対しての心配はご無用です。
是非、おかわりして食べてください笑

⑦【解散後、各自で次のイベントに行きやすい】
こちらで２次会などのセッティングはしておりませんが、解散時間はおよそ１４時～１４時半。
「夜まで予定が無い」という方も多いですよね？
そんな方同士でカラオケ（昼だから安い！）やボウリング、カフェなどに行かれるケースは珍しくありません。その場限りで終わらないのも、「朝活」だからこそ、ですね！

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

こんな理由で、クッキングコンはリピーターも続出しています（感謝感謝です泣）！！

というわけで素敵な出会い（恋人に限らず、お友達でも）を探しに、
ぜひ「仙台クッキングコン」をご利用ください！皆様のご参加お待ちしています♪
