MODXを新たに始める人へ

今回の麦乃大学（MODXCMS勉強会）は、ハンズオン形式の入門講座です。
WebMatrixを使い、MODXを手軽にはじめてみましょう。
MODXのインストールから、簡単な企業サイト公開までを、お手持ちのパソコンで体験していただけます。
また、勉強会のあとには懇親会を予定しております。

概要

  • WindowsPCに、WebMatrixをインストール
  • WebMatrix上に、MODXをインストール
  • 実際にMODXでウェブサイトを作ってみる
  • メールフォームを作ってみる
  • チャンク・スニペットを使ってみる
  • ExpressWebでサイトの公開
  • 質疑応答

パソコン持参のお願い

今回のトレーニングには、WindowsPCが必要です。
以下の要件を満たすノートパソコンの持参をお願いいたします。

  • Windows 7、もしくはWindows Vista(ServicePack2以降)が快適に稼働するPC（仮想マシン・BootCamp可）
  • 無線LANにてインターネットに接続できること。
  • WebMatrixをインストールしても問題ないこと。
  • ※WebMatrixをインストールできない場合、デモサイトを使用してのハンズオンとなります。
    この場合ウェブブラウザが使用できれば、OS等問いません。

当日は無線LANおよび電源をご用意いたしますが、
個別の接続の設定についてはサポート致しかねる場合があります。

当日は２名〜４名でチームを組んで頂きますので、
チーム内でパソコンを共用して頂いても構いません。

当日のご案内

当日は、直接会場（品川マイクロソフト本社　２Fロビー）へお越しください。
ウェブマトリックスの看板（緑色の人）でセキュリティカードをお渡しいたします。
会場では電源、無線LANを用意しております。
SSID「muginodaigaku」（暗号化なし）に接続し、インターネットに接続していることを確認してください。

ご参加のほど、お願いいたします。
