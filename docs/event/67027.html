description

7月19日（日）に千葉県四街道市のフィールド「いくさ」にてゲーム会を開催いたします。
7月の暑い時期ですが、暑い陽射しや雨を防いでくれる森林フィールドで思いっきりアウトドアゲームを楽しみましょう！

□開催場所

サバイバルゲームフィールド 　戦（いくさ）
http://ikusaba.jimdo.com/
千葉県四街道市中台268

□日時

2015年7月19日(日)　09：00時～17：00時
※AM09:00～ 受付開始　ゲーム開始は10：00時頃より

□費用

フィールド使用料：一人2500円
※別料金でお弁当を注文できます。
　こちらは10:00時までの受付となりますので、朝の参加受付時にお知らせ下さい。

□レギュレーション

・フル・セミオート問わず、パワーの上限は0.98J(0.2ｇBB弾で初速約98.95ｍ/ｓ)
・弾数制限無し。
・使用可能なBB弾はバイオ弾のみ、プラスチック・セミバイオ弾は使用出来ません。
・減速アダプター・レーザーサイトの使用禁止。
・フィールド内ゴーグル着用，シューティンググラス可。
・ナイフアタック禁止。
・ゾンビ行為、ダメ・ゼッタイ！

□服装

ドレスコードはありませんが、森林フィールドなので動きやすく汚れてもいい服をご用意下さい。
怪我防止のため帽子や露出の低い服装に、グローブやエルボー・ニーパッド等の使用をお勧めします。

□基本ルール

10分～15分ほどのフラッグ戦やハンバーガーヒル戦、ノルマンディー戦を予定。
ゲーム間の休憩は約10分です、スムーズな進行にご協力願います。

□交通

http://ikusaba.jimdo.com/%E4%BA%A4%E9%80%9A%E3%82%A2%E3%82%AF%E3%82%BB%E3%82%B9/
上記URL先の交通案内を御覧ください。
現地集合・現地解散、送迎バスはありませんのでレンタカー・自家用車等乗り合わせの上お越しください。

□レンタルをご希望の方へ

フィールドにて銃や服のレンタルが可能です。
ご希望の方は下記URL先の一覧を参考に、7月15日までにコメント欄にて御一報下さい。
販売のものに関しては当日申請でも大丈夫です。

□その他の注意

・フィールド内での火気使用厳禁
・セイフティーエリアでの発砲・空撃ちの禁止（重要！）
・近くのコンビニがちょっと遠いので、飲み物や食料等購入をご検討の方は高速に乗る前かSAでの調達をお勧めします。

□雨天の場合

当日の朝、06:00時にフィールド側と相談の上ご連絡します。
天候次第では前日判断の可能性もあります。

□申し込み締め切り

7月15日までにATNDへの参加表明をお願いします。
グループで参加の皆様は、お手数ですが代表者の方でメンバーの人数等まとめた上でご連絡頂ますようお願い致します。
またこの際、レンタル品の有無も合わせてご連絡頂ますようお願いします。
