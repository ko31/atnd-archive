＜講演内容＞

日々進化を続けるアプリ業界で生き抜くための、
最新マネタイズ戦略やトレンドをアプリディベロッパー様を中心に豊富な事例を交えて講演して頂きます。

今回は弊社と共同でアプリ内課金の導入サービス 「itemstore」 の提供を開始したカイト株式会社との共同開催です！

豪華講演者の登壇を予定しておりますので、お聴き逃しの無いよう奮ってご参加ください！！

19:10～
登壇① 株式会社ADFULLY 小室

「アドフリくん、動画リワードの最新状況（傾向から実績データまで）」

SSPだからこそわかる、動画リワードの数字データと傾向をセミナー限定で公開したいと思います。

19:25～
登壇② 株式会社ブシロード 広瀬 和彦 様

「版権作品のゲーム化と広告収益モデル」

日本のみならず海外にも高い知名度を誇るTVアニメ「クレヨンしんちゃん」を
題材にしたゲームアプリ「クレヨンしんちゃん 嵐を呼ぶ炎のカスカベランナー!!」を
２０１４年４月にリリース。

著名な版権作品を本格的な広告収益モデルでゲーム化した事例として、
そのノウハウや事後分析を紹介して頂きます！

19:50～
登壇③ 日本エンタープライズ株式会社 上吉原 一貴 様

「アプリ収益の構成と実像とコツ」

コンテンツサービス事業、ソリューション事業などを手がける
日本エンタープライズ株式会社様。

女性のための健康サポートアプリ「女性のリズム手帳」
スマホでカンタン！出品無料のフリマアプリ「Dealing」
「おでかけ」をもっと便利に！もっと楽しく！する交通情報サイト「ATIS交通情報」
スマホ、タブレット、PC向けの総合電子書籍サービス「BOOKSMART」
など、など実績多数！

幅広い実績をもとに語られる広告マネタイズ、アプリ内課金の成功・失敗事例。
アプリ内課金のポイントノウハウまで余すことなく語っていただきます！

20:05～
登壇④ カイト株式会社 田村 秀一 様

「CTOの視点の先に見えた広告マネタイズとアプリ内課金」

およそ2万アプリに導入実績のある、広告・課金・プロモーション等の
クラウドサービス（mBaas）を提供されてきたカイト株式会社様。
2016年5月から「最短10分であなたのアプリにアプリ内課金が導入できる
驚愕のサービス！」”itemstore”の提供を開始されました。

アプリマーケティングを全方向から支援するカイト様のCTOが語る
これからのアプリマネタイズとは！
課金ユーザーへの管理・アプローチ手法とは！
itemstoreの真価とは！

（"itemstore"はアドフリくんと連携することで、月額利用料が無料になります）

株式会社ADFULLY 概要

当社は、SSP「アドフリくん」というアプリの広告収益最大化をサポートするサービスを提供しております。
私達の一番の特徴は、元々メディアやコンテンツ事業を運営していたメンバーが独自の「メディア目線」で作り上げた、他の広告事業社とはちょっと違うSSPサービスを提供している事。
「メディア１００％の提案」をモットーにサービス提供している当社は、既に多くのアプリ開発者や企業にご賛同頂いております。

カイト株式会社 概要

カイト株式会社は、IT、モバイル、特にスマートフォンという市場において、“アプリ開発者をハッピーにする”という基軸からブレずに様々な事業を展開しています。
プラットフォーム事業としては、クラウドサービス『itemstore（アイテムストア）』があります。
アプリ開発者向けに、実装ハードルの高かったアプリ内アイテム課金を容易に導入でき、広告以外のマネタイズをサポートしてくれるサービスです。
メディア事業としては、『peep（ピープ）』があります。
サブカルチャークリエイターとそのファンの繋がりをコミュニケートする”写真と声のメッセージングアプリ”です。
