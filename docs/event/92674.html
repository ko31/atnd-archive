講演概要

MySQLの公式GUIツール MySQL Workbench

  • MySQL Workbenchはオラクルが開発しているMySQLの公式GUIツールです。MySQL WorkbenchにはMySQLの運用管理に役立つ機能のほか、SQL開発に役立つ機能、DB設計に役立つ機能、データ移行に役立つ機能も兼ね備えています。このセッションでは、MySQL Workbenchの機能や利点をデモを通してご紹介させて頂きます。

MySQL Enterprise Monitor による運用監視

  • MySQL Enterprise MonitorはMySQLの運用監視ツールでMySQLサーバーの安定稼働やチューニングに役立ちます。このセッションでは、MySQL Enterprise Monitorの機能や利点をデモを通してご紹介させて頂きます。
