= イ ベ ン ト の 特 徴 =

【少人数制なので企業の人事担当者や現役エンジニアと密に話せる！】
大人数で行われる一般的なイベントとは違い、少人数のグループで
採用のキーマンや現役エンジニアと接することができます。企業の話を
じっくり聞いて、積極的に情報を得てください。

・まず初めに、参加学生の皆さんに対し参加企業が1社ずつプレゼンを行います。
そのプレゼン内容をもとに、「1ターム30分×5ターム」のタイムスケジュール
に従って各企業ブースを自由に回っていただきます。

・話を聞けなかった企業やもっと詳しく話を聞きたい企業については、イベント
終了後のフリータイムをご活用ください。

・参加者の皆さんに、アカリクオリジナル就職ガイドをプレゼントいたします。

席数に限りがございますので、お早めにご予約下さい。

ご不明な点はなんなりとアカリクイベント運営事務局までお問い合わせ下さい。
