船上のピアニストと各種弦楽器の生演奏を背景に…
2013年最初にして最大のあわてんぼう達のクリスマス物語が幕を開けます。

X’mas特別企画！！　異業種交流パーティーで規模2,000名！

多くの皆様からご期待に応えました！！　一人参加も大勢参加もすぐに仲間ができちゃう圧倒的に賑やかなクリスマスを創造してみせます。

今宵限りの突き抜けまくった限定企画！

あわてんぼうのサンタクロースが東京湾船上に大集合！

2,000人が出逢い、語り、騒ぎ、踊り、酔いしれる事間違いなし！
とってもハッピーな戦場のピアニスト＆メリークリスマスで乾杯～♪

☆★SANTA with GIRLS★☆　2,000人大集合！あわてんぼうのサンタクロース「パルプンテ」物語2013の幕開け！

オープンエアのテラス席では爽やかな風が吹き抜け最高！！　夜景がさらに巨大客船を包み込む！

3階層に分かれた船内はこれまでにも多くの感動を呼ぶシーンを生んできました！！

巨大客船を丸ごとパルプンテ！！
そこはまさに楽園ムードの漂う雰囲気！

サンタ！　トナカイ！　プロの生演奏！
覆面レスラー？　ドラクエ？　多種多様なコスプレあり！！

これからのクリスマスStyleは思いっきり楽しんだもん勝ち！！
指くわえてる場合じゃなく慌てて下さい！

スタッフ全員で最高に素敵なクリスマスイベントをお約束！！

素晴らしい夜景を眺めながらの非日常空間、魔法のような幻想的な時間はまさにパルプンテ！
