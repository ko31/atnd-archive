昨今の「働き方改革」や「少子高齢化」による人材確保が、企業の至上課題となっており、 新しい労働力としてRPA「Robotics Process Automation」が注目されています。
RPAは、仮想知的労働者（Digital Labor）とも言われ、 パソコン上で行われるさまざまな事務処理をソフトウェアロボットに代行させることで、 事務作業者のさまざまな業務を自動化させるための仕組みです。
RPA開発プロジェクト経験豊富な日本初開催となったRPAロボットコンテストで優勝したチームの講師が、 RPAツールのインストールから簡単なプログラム作成までを解説します。 ぜひ、プログラム開発を学びたい方はご参加ください！！
