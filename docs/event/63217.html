■【第2回】講師紹介
株式会社ラクスル　代表取締役　松本恭攝氏

＜講師紹介＞

1984年生まれ。慶應義塾大学卒。A.T.Kearneyに入社し、コスト削減プロジェクトに従事
する中で、6兆円の市場規模がある印刷業界に効率化が行われていないことに気づき、
印刷業界の仕組みを変えるべく2009年9月にラクスル株式会社を設立。印刷会社の非
稼働時間を活用した印刷のEコマース事業を展開。現在は中小企業のオフラインでの集
客活動を支援する「集客支援プラットフォーム」事業の展開を加速させている。未上場段
階で累計58億円の資金調達に成功。

～起業に必要な『スキル･アイデア･マインド』、
　すべてお話しする贅沢な90分間！～

社会課題の解決や既存産業に革命を起こす先輩企業家が講師となり、
起業のきっかけ、起業前に身に付けておきたいこと、過去の失敗談など起業のリアルを、
『スキル・アイデア・マインド』といった3つのポイントを中心に、具体的な事例を交えながらお話しいただきます。
当日は、先輩起業家たちに直接質問したり、あなたが考えているアイデアや
悩みへのアドバイスを貰える場もご用意しています。
もちろん、各社に入社して学ぶというチャンスも。
将来起業を選択肢の一つとして考えている方はもちろん、
スタートアップや新規事業に興味がある方、
次世代のビジネスリーダーとして、社会に変革や新たな価値を生み出したいという
強い意志や意欲を持った社会人の方はぜひご参加ください。

■下記のような社会人の方はふるってご応募ください。

✔高い資質や実績を持ち、自身の力を試したいと考えている方
✔次世代のビジネスリーダーとして、社会に変革や新たな価値を生み出したい方
✔起業やスタートアップへの転職、新規事業に強い興味がある社会人の方
✔なんとなく今の環境では成長感が感じられない方
✔優秀な人や起業家との接点を持ちたい方
✔ビジネスを通して社会を変えることに興味をお持ちの方

本講座では、
・これまでのキャリアと起業のきっかけ
・起業前に学んでおくべきこと
・社会を変えるビジネスモデルの創り方
・事業を共に創る仲間の見つけ方　など
企業における「リアル」についてお伝えします。
