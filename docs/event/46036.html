CreateJS今日この頃

CreateJSがマイナーアップデートされました 。前のリリースでは、イベントモデルの改訂を始め、とくにEaselJSライブラリが大きく変わりました。そこで、おもにEaselJSについて、最新の0.7.1にもとづいて扱いがどう異なってくるのかを説明します。

なお、演目の細かな内容は適宜修正しますし、当日変えることもあります。USTREAMは予定しておりませんが、レジュメとサンプルは FumioNonaka.com に後日公開します。

講演概要

お品書き

  • CreateJS今日この頃
    1. 新たなイベントモデル
      • マウスイベント
      • イベントのバブリング
      • EventDispatcherクラスの変更
    2. Matrix2Dクラスの変換行列でxy座標を回す
      • Pointオブジェクトのxy座標を変換するMatrix2D.transformPoint()メソッド
      • Matrix2D.transformPoint()メソッドを3次元空間の回転に応用する
    3. Ticker.tickイベントのタイミングにrequestAnimationFrameを用いる
      • Ticker.timingModeプロパティ
      • アニメーションをrequestAnimationFrameで行う
    4. EaselJS 0.7.1で修正されたおもなバグ
      • Ticker.on()メソッドが関数を返さない
      • ColorMatrixクラスのバグ3種盛り
