イベント概要

Ubuntu 16.04 LTSのリリースを記念したオフラインミーティングイベントです。

テーマは「LTSですよ！」。今回リリースされたUbuntu 16.04 LTSは2年ぶり6回目のLTS（長期サポート版）です。長期サポート版ということで、12.04や14.04からのアップグレードを考えている利用者も多いでしょうし、今回からUbuntuデビューを考えているご新規さんもいらっしゃることでしょう。イベントには、すでに16.04を使っているユーザーも来るはずですので、その雰囲気について訊ねてみてはいかがでしょうか。

前方のセミナー、後方のハッカソンという雑然とした空気とともに、Ubuntuの魅力を語って頂ければ幸いです。

参加費・参加資格等

参加費は無料です。ささやかではありますが、無料の食べ物とからあげと飲み物を用意してお待ちしております。

Ubuntuを使ったことがある、あるいは興味がある場合など、どんな方でも来て頂いて問題ありません。

詳細は、https://wiki.ubuntulinux.jp/Events/OfflineTokyo201606 を参照してください。
