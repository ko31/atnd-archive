この度、大学院生の基礎能力を高く評価し積極的に採用したいと
考えているＩＴ企業が集まる、2012年修了者対象の
インターンシップイベントを開催いたします。

「授業で少し学んだだけ」の人も「勉強を始めたばかり」の方も
「ＩＴ企業で自分を鍛えたい」でもＩＴ企業に関心があれば参加大歓迎です。
業界や企業について理解を深めることはもちろんのこと、少人数・予約制の
イベントなので、企業の人事担当者と密に話のできるまたとない機会です。
　　　　

==　開催日時 ・ 開催場所 ・ 参加対象者 ・ 参加方法 ==

□タイトル：「IT業界サマーインターンシップイベント」

■参加費：無料

□対象：授業・研究などでプログラミング経験がある方で ＩＴ企業への就職を希望する2012年度新卒の学生

（学部生、修士課程学生、博士課程学生）

※プログラミング能力やＩＴ企業への関心等についてご確認
させていただく場合がございます。

□参加企業数：6～7社（予定）

≪イベント概要≫
企業ごとのブースを4～5人1組のチームで順に回っていただきます。
1ターム40分の中では、事業内容やインターン情報の説明を聞くことができます。

≪特徴≫
少人数制なので企業の人事担当者と密に話せる！
大人数で行われる一般的なイベントとは違い、少人数のグループで
採用のキーマンと接することができます。企業の話をじっくり聞いて、
積極的に情報を得てください。

優良IT企業が数多く参加する当イベントを利用して
効率良くかつ質の高い就職活動スタートダッシュをを実現してください！

ご不明な点はなんなりとアカリクイベント運営事務局までお問い合わせ下さい。
