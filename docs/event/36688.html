a, 学生ヨル会議って？

私たち学生ヨル会議とは、学生が夜に寄り集まって話し合う発信型のプラットフォームです。
2012年の1月1日にNHKで放送された「ニッポンのジレンマ」を観た大学生たちがソーシャルメディア上で集まり、京都でこの学生ヨル会議が生まれました。
地域のコミュニティスペース等をお借りして、様々なテーマについて議論し、その様子をUstreamで全国に放送。地域間、世代間を超えた情報共有し、何か新たな気づきを生み出すこと目指しています。

b, 環境chプロデューサーよりメッセージ

いつも口にしている食べ物がどのような過程をたどり、みなさんの食卓に運ばれているか考えたことはありますか？地球規模でモノが循環し、あらゆる資源と密接に関わって人間の「食べる」動作に辿り着きます。そんな「食」を切り口に「地球環境」を考えたら、人間と自然はもっと近くなれるのではないかと考えました。それに食べることってみんな好きじゃないですか。おいしいものを食べて幸せになって、楽しくエコできたらいいですよね？そんな楽しい時間をみなさんと共有できることを楽しみにしています。

プロデューサーインタビュー記事はこちら→『わたし、エコちゃんって呼ばれてるんです！』

c, 学生ヨル会議よりメッセージ

ヨルには２つの意味があると考えています。
１つは何か新しいことが始まりそうなワクワクする時間帯としての「夜」。もう一つはフラッと気軽に集まれるという意味での「寄る」です。家が近いし…暇だし…といった方までお気軽にご参加ください。
