イベント概要

北東アジアOSS推進フォーラム年次大会とは

日本・中国・韓国それぞれの政府機関、OSS推進団体が一堂に会し、年次フォーラム大会を開催地持ち回りで毎年開催しています。今年は日本大会として 11月14日～16日、横浜にての同大会を開催することとなりました。本イベントへの参加は、ディナーパーティーや本大会イベントなどの参加権として有償チケットが必要となります。

　詳しくはこちらをご覧ください http://ossforum.azurewebsites.net/

3日目に開催するイベントはオープンセミナーとして無料参加 OK

午前中は中国・韓国から来日した推進団体所属企業や大学の先生に、テクニカルなトレーニング講師を務めてもらいます。 午後には基調講演として「グローバルでイノベーションを実践する」企業団体のお二方、また、日本マイクロソフト様やNECソリューションイノベータ様の OSS 関連の講演も予定しています。

  • 11月16日開催セミナーは誰でも無料で参加可能です。
  • 午前・午後それぞれだけでも参加可能です
  • 当日名刺 2枚をご持参の上、ご来場ください

【注意】 お申し込みは ATNDではなく、下記からお願い致します

◆ 基調講演
個別の食と健康管理 ― デジタル社会が可能にする個人医療・健康・栄養摂取データの最大限の活用法

西出 香 様（Kaori Nishide）

TNO（オランダ応用科学研究機構）　Healthy Living 日本代表
http://www.tno-pharma.com/

＜講演概要＞
ＴＮＯは応用研究開発で国際的に高く評価されている総合の研究機関(NPO)。デジタル社会では生活者個人が食と健康の管理ができるいう可能性、3Dフードプリンターの開発と共に実現されるであろう課題(ホメオスタシス・栄養摂取・運動・ライフスタイル改善の意欲)をご紹介します。

＜自己紹介＞
京都大学農学部、ワーヘニンゲン大学環境科学修士、ブラッドフォード大 学経営修士修了。オランダで畜産･水産の大手企業本社で分析、戦略、M&Aに携わった後、2013年より現職。農林水産省、文部科学省にて検討委員歴任。法政大学Global MBA兼任講師。

シリコンバレーを越えるアジアのイノベーション

スタートアップコンソーシアム沖縄代表

アジアをネットワークで結びシリコンバレーを越えるイノベーションを起こせる時代に来ていることについてお話します。

福岡県 北九州市出身。元ガンホー会長孫泰蔵率いるmistletoe顧問。 スタートアップカフェコザ代表として起業による地方創生の成功例として多く のメディアによる掲載、政府省庁・行政機関より多くの視察を受ける。総務省 ICT地域活性大賞奨励賞受賞。スマートグリット、IoTなどに関して行政関連講 演多数、大学などで講義も行う。りゅうぎんスタートアッププログラムを設計・ 運営アドバイスに当たる。西日本最大のシェアオフィスfabbit設計に参画。


◆ スポンサーセッション
OSS & PaaSで今すぐ始めるデータ解析

藤田 稜 様（Rio Fujita）

日本マイクロソフト株式会社
GBB OSS Data TSP

＜講演概要＞
Microsoft AzureにはOSSベースのデータ解析ソリューションが数多くあり、それらを組み合わせることで様々な用途や容量のデータ解析が迅速に始められます。本セッションではMicrosoft自身の事例を含むデータ解析の実際と、デモを通じてデータ解析の糸口を掴んでいただきます。
