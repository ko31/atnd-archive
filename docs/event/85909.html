≪満足度83.3％*≫のワークショップを開催します！
*（５段階評価の最上位の割合）

　＜Come Up Osakaって何？＞
・Come Up Osakaは、「悩み」を「経験」で解決するワークショップです。

仕事で悩んでいる。家族のことで悩んでいる。人間関係のことで悩んでいる。
悩みはひとそれぞれにあると思います。

Come Up Osakaにはメンター（指導者）はいません。
同じ「経験」をした人が「経験談」をシェアすることで「悩み」を解決します。

＜こんなことが体験出来ます＞
・参加者の方の解決方法をヒントにして「選択肢」が増えます。

私の経験ですが、悩んでいるときに人に相談しても、
「～だから問題が解決しないんだよ」「～だからこういうことをしてはどうかな？」
と相談しても結果、アドバイス、指導や指摘のようなことになり、いまいち心に刺さりませんでした。

「それ、わかる。俺も同じようなことで悩んだ。その時、俺はこうして解決したよ」
「なるほど。私の友達の経験だけど、こういう風に考え方を変えると上手くいったみたい」
とあくまで経験者が経験談を互いにシェアします。

自分の価値観を変える必要はありません。

誰かの解決方法（経験談）を聞いて、「それ、いいな。そうしたい」と思えれば取り入れてみる。

どのように解決するかは「悩んでいる人が選択できる」と思っています。

Come Up Osakaでは、ワークショップ会場から自宅に帰るときには足取りが軽くなるよう、悩んでいる人が力強くなれるような運営をしていきたいと思っています。

＜Come Up Osakaで得られるもの＞
・自分にはなかった、考えられなかった解決方法、価値観に気づく９０分です。

例えば、仕事の人間関係で悩んでいる人は、職場の同僚に悩みを打ち明けるのは難しいと思います。
そういった方は、Come Up Osakaで異なる解決方法、価値観が得られます。
多様な考え方を１人で考えるのではなく、参加者の経験から気づくことができます。

1.会場について
　会場：グランフロント北館　７Ｆナレッジサロン

　大阪駅からグランフロント北館へ向かう場合、大阪駅２Ｆの連絡デッキと繋がっている南館を抜けて北館へ進みます。
　南館を抜けて北館へと進むと北館の２Ｆに到着します。
　１Ｆに降りるにはエレベーターか北館に入って２５ｍほど進むと進行方向右側にあるエスカレーターで降りることができます。
　北館１Ｆ中央あたりに総合インフォメーションがあります。
　ナレッジサロンへはベンツショールームの隣にあるＣＡＦＥ　Ｌａｂとアクタスという家具屋さんの間にある「北３」のエレベーターで７Ｆにあがります。
　「北３」のエレベーター７Ｆの出口直ぐにナレッジサロンの受付があります。
