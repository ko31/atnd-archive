現在は、Web制作者をはじめとしたWebに関わる皆様にWindows Azure の利用促進を行っています。

Windows Azure ハンズオン「Windows Azure Web サイト上にWebサイトを構築する」
クラウドを活用したWebサービスやWebサイトの構築・運用が進む中、マイクロソフトのクラウドサービス「Windows Azure」の注目が高まっています。その機能の1つ「Windows Azure Web サイト」は、レンタルサーバーと同じような費用感でクラウドを利用できるため、そこにWebサイトを構築・運用するサイトが増えてきています。
本セッションでは、Windows Azure Web 概要をハンズオン形式にてご紹介します。また、事例を交えながら構築、運用のポイントも合わせて説明します。
Windows Azureに興味があるけど、まだ試したことがない方、是非この機会にWindows Azureを試してみませんか？
