（90分セミナー）『伸び悩む日常英会話＆TOEICを10週間で抜群に伸ばす具体的方法＆トレーニング』

このセミナーでは、英語を英語のまま捉えるためにはどうすればいいのか、そのエッセンスと基本トレーニングをお伝えします。語学は短期集中で結果を出せるもの。私たちの練り上げてきたトレーニングをどうぞ直接見定めてください。

（セミナー他日程） https://englishbay2future.doorkeeper.jp/events/upcoming

※真剣に取り組む腹積もりのないまま、とりあえずで登録される方は、基本的にお断りしています。

イングリッシュベイ青山とは？

10週間で苦手な英語を克服し、半年後には転職、留学、昇進を成し遂げるための英語トレーニングジムです。英語が現場レベルに達していない方、現状の語学力が足りない、英語コンプレックスから抜け出せないながらも何とかしたい方など、本気で克服したい20代、30代の方々に対して私たちも本気で臨みます。

単刀直入に言わせてください。

私たちイングリッシュベイ青山の使命は、語学の壁を本気で乗り越えたい方々が10週間で英語を英語のまま「話す」「聞ける」「読める」やり方を身に付け、英語力を実践レベルまで引き上げること。これ以外にありません。

始まりは2012年、練り上げた独自の第二言語修得メソッド（英語脳ピラミッド）をビジネスマン向けに提供することからスタートしました。私たちの定義する『英語脳』とは、英語を英語のまま瞬時に理解する思考技術です。暗記を軸とする日本人の一般的な学習方法では、この『英語脳』の状態に到達することは相当な困難を伴います。そもそも短期間での英会話力向上や、TOEIC高得点奪取の困難さは、皆さんもご存知のとおりでしょう。

私たちはずっと、欧州で取り入れられている第二言語修得メソッドを、どうしたら日本人（従来の学習方法を重ねてきた日本人）に伝わりやすく、しかも、最短の方法と成り得るのかチャレンジを重ねてきました。単なる試験の解法テクニックではなく、日常生活で実践的に使える英語力について重ね重ね、問い続けてきました。

英語の修得に、生まれ育った環境・学歴・性格は関係ありません。

私たちトレーナーも、実は、語学の壁を過去乗り超えてきました。どのトレーナーも10代、20代の頃に英語の壁にぶち当たり、そこから必死で勉強して語学力を身につけています。当時、「どんなに必死でやっても語学力が満足にいかず、もしかしたら本当に自分は英語の素養がなくて、このまま話せなくてどうにもできない人生になってしまうかもしれない。」そんな大きな不安を抱いていました。それでもただ諦めず、粘り強く努力を重ねていたら、ふと英語のコツに気付けて、その後も、あともう少し、もう少しを重ねてきました。このイングリッシュベイ青山のトレーニングを過去に受講し、短期で結果を残したトレーナー、さらには最近まで外資や海外でキャリアを積み重ねてきたトレーナーもいます。

彼らは口をそろえて、「このトレーニングをもっと早く知れたら学生時代、語学で苦労しなかったのに。」と言います。私たちの強みは、できないところから、コツを身に着け、一瞬で「できる」に変わった瞬間を、実体験として経験していることです。

主なご対象の方

  • 明確な目標もしくは目的のある主に社会人の方（例えば、留学、昇進、海外転勤、転職など）

  • 留学経験にあまり恵まれず英語（もしくは英会話）が苦手な方
  • 語学力向上に欠かせない「スケジュール管理」＆「プライベートバランス」のコツを心底身につけたい決意ある方
  • 過去TOEICスコアで５００以上もしくは、英検２級以上の方。（なおゼロから始めることをご希望の方は別途、メニューを用意していますのでご安心ください。）
  • ご年齢が２３歳～４０歳の方（学生時代に一定の積み上げのある方については、この年齢枠の限りではありません）

  • グループワークでの協調性をお持ちの方

  • ご紹介者がいらっしゃらない方は、事前アンケートにご参加の目的および動機について明確にご記載をお願いします
