description

サッカーW杯、夏季五輪に次ぐ世界3番目のスポーツイベント、2019年ラグビーW杯が日本で開催されます。本講座では、日本ラグビーフットボール協会の日本代表の広報として、常にグローバルな視野で、現場のサポートと国内外へのPR活動に尽力されてきた渡邊まゆ子氏に、ラグビーW杯2015年のイングランド大会のレポートや日本及び海外チームの食のサポートの様子について、お話ししていただきます。また、弊社代表の橋本より、ラグビーWカップ日本代表の栄養サポートの経験、海外栄養士からのヒアリング、さらに外国人選手も多いパナソニック ワイルドナイツでの普段のサポートを通して、国内外のラグビーの食のサポートの現状とニーズについて解説いたします。

■時間：2016年1月23日(土) 10:00〜12:00
