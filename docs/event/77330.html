description

山形県の鶴岡、米沢、真室川町を取り上げ、
伝承・在来食物を舌で感じながら、
味覚に関する映画や生産者様達の声等、
食べて、見て、聞いて、知る、地方と都会を繋ぐ会。

食べ物付き情報誌『山形食べる通信』で取り上げられた「きくらげ」をはじめ、
山形の食材をたっぷり使用したお料理でおもてなしをします。
当日は味覚レッスンの映画上映会と地元生産者さんはじめ
「在来作物で味覚のレッスン」の渡辺監督、
「山形食べる通信」の松本編集長からのメッセージmovieも紹介予定です。

古きものを生かしながら伝え続け生活をしている地方の雰囲気を、
名産の食材と生産者の思いを感じながら、
都会に残る古き良き古民家asagoroで堪能できる会になります。

都会と地方の繋がりや、知るといったきっかけになれる時間を過ごしたい方、
是非ご参加ください。

～こんな方におすすめです～

・自分は山形県民だ
・出身は山形じゃないけど、山形がすき
・山形の美味しい食べものを食べたい
・在来作物とはなにか知りたい
・食育について前から興味があったがなかなか機会がなかった
・とにかく日本酒が好き！利き酒をしたい！という方etc…
