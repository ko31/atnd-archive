完全予約制

司会・講師　西垣裕里（社会保険労務士、認定笑いヨガ講師、精神保健福祉士）

起業したいと思っている人たちが、

気軽に集まれるサロンです。

具体的でなくてもOK！何でもありのサロンです。

「働く」と一言で言っても、雇用されて働くだけではなく、

「起業して働く」という選択肢もあります。

起業と言っても、

自分の身の丈に合ったスモールビジネスです。

だからこそ、「夢」を語ることが大切です。

一緒に、三方よし（売り手よし、買い手よし、社会よし）

のビジネスを考えませんか。

私は、社会保険労務士として特に、

女性・若者・障碍でお困りの方への

「働きやすさ」と「暮らしやすさ」のために、

多方面からサポートしたいと考えています。

参加した皆さまが、「自分にもできるかも！」

と思ってもらえる場を目指します。

このサロンでは、起業動機の掘り下げと

人事・労務分野の基礎知識講座を

（お金の話も大切ですが、

「人」はその事業（夢）を叶える「資源」だからです。）

ざっくばらんな座談会形式で行います。

そして、起業するにあたり不安なことや様々な葛藤

の分かち合いもできたらと思っています。

夢の語りは、

就業規則を作成する際にも役立ちます。

規則の目的を労使で共有できるからです。

ぜひ、お気軽にご参加下さい。

起業が、すぐに思うようにならな

いこともあるでしょう。

そんな時は、笑いヨガで笑い、

タフさとゆとりと楽観性を

取り戻す方法もご紹介します。

笑いヨガとは、

世界的には、Laughter Yogaといいます。

Laughter Yogaは、「ラフターヨガ」と読み、

日本語では、「笑いヨガ」と訳します。

インドの内科医マダン・カタリア博士が考案した、

子どものあそび心をベースとしたユニークな笑う体操です。
