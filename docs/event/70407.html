今回も東京

東京です。

前回、予約無しで行ったけれど超人気店で入れる気配がなかった中野のマグロマートに予約を入れました。今度こそマグロ祭りです。

開催場所

中野のマグロマートです。中野駅から少し北へ歩きます。バスで最寄りまで行きたい方は Twitter @xtetsuji にお気軽におたずねください。

現状の定員が4名なのは、予約を4名で入れたからです。補欠で入れてもらえれば増員交渉します（時期によってはできないかもしれません）。

質問とか

質問は Twitter @xtetsuji などでどうぞ。

費用

参加費は無料になっていますが、とりあえず飲み食いした分の割り勘です。あんまり高いところには行かないと思います。

遅くとも9月中に人数とコースを確定します。参加した人にアンケートをするかもしれません。
