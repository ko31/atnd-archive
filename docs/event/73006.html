description

スポーツの世界的大会の日本開催に向け、各地で事前キャンプ招致合戦が始まり、インフラ等の整備はもちろん、選手のコンディション調整に影響を及ぼす食事にも、きめ細かい準備が必要となります。
来日する海外のアスリートや外国人に喜ばれる食事やサービスについて、これまでの実績や独自の調査結果などを整理してお伝えするとともにホスピタリティのプロフェッショナルである、塩島賢次氏より訪日外国人へのおもてなしについてお話しをしていただきます。
全国の自治体、宿泊施設、旅行会社、飲食店、栄養士必見の講座です！

■時間：2016年2月27日(土) 13:00〜15:00
