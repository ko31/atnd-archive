□講座内容

「牛乳はカルシウムが多いので骨粗しょう症を防ぐ」「バターより植物性マーガリンの方がカラダに良い」
「葉もの野菜は緑色が濃いほうが新鮮でおいしい」「砂糖や甘いものがカラダや脳の疲れを取る」などなど。
みなさんは、これらのことを聞いたことはないですか？
子どもの頃、おかあさんに言われませんでしたか？
テレビコマーシャルやテレビ番組で聞いたことないですか？

日本の栄養学は1980年代くらいから格段に進歩してきましたが、
この30年の間にそれまで常識とされていたことが次々と覆されてきています。
今まで常識で正しいと思ってきたことが、実は間違っていた！なんてことがあちこちにあります。
この講座では、その中でもっとも実践的で大切な、新しい食育のテーマを７つに絞って具体的にお伝えします。

ご自身の健康はモチロンのこと、家族の健康のために、
知っておくと役立つ正しい情報をしっかりと持ち帰ってください。

【７つの非常識、健康と食育の基礎講座】
１．Yes、No　クイズ！
