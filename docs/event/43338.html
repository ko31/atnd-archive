【Vol.4】データマイニングは魔法の杖か？

「人気Webサービスの裏にデータマイニングあり」――ビッグデータのビジネス活用が盛んに叫ばれるようになると同時に、その巨大な山から「宝」を掘り出すデータマイニングに今、大きな注目が集まっています。ビッグデータを解析・分析することで、Webサービスをもっと魅力的なものにしていける。そんな期待感が世の中を席捲しつつも、一方で「データマイニングは万能ではない」という声も同時に聞かれます。

では、データ解析・分析は、ビジネスの現場においてどこまでの威力を持つものなのか。そして、その限界はどこにあるのか。Webサービス企業三社におけるビジネスへの活用事例を紐解き、その実態に迫ります。

プレゼンター

●ヤフー株式会社／角田 直行

2005年にヤフー株式会社へ入社し、これまでに地図や路線情報、検索などのサービスからYST（Yahoo! Search Technology）や検索プラットフォームなどバックエンドの開発に関わる。現在は全社横断的なデータ解析を担う部門にて全サービスのログを中心とするデータの収集、処理を行うインフラ基盤を中心にレコメンデーションや要素技術などのソリューション全般について技術統括を行っている。

●株式会社ドリコム／ところてん

某通信会社の研究所で情報セキュリティ、ビッグデータの研究に従事した後、ドリコムに入社。現在はデータ分析グループにおいて、データサイエンティスト(笑)として、ソーシャルゲームの統計分析、KPI設計、バランス調整、ログの可視化、収益予測、イベント企画等を実施。最近の趣味はTwitterのタイムラインを「ガルロワ」のキーワードで監視すること。

●株式会社Gunosy／関 喜史

株式会社Gunosy共同創業者。東京大学大学院在籍中の2011年10月にGunosyを立ち上げ。担当は推薦システムの開発を中心とした研究開発領域で、『情報を世界中の人々に最適に届ける』を目的に、機械学習とデータマイニングをベースにしたパーソナルマガジンを50万人以上のユーザーに届けている。現在も工学研究科技術経営戦略学専攻博士課程に籍をおく。2011年度未踏クリエイター。
