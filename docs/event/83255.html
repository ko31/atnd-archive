Raspberry Piによく似たデバイスNanoPi。
Raspberry Piと同じく、Linuxで動作して電子工作などをつなげることができます。
今回はNanoPiで最も安いNanoPi Neoを使って社内WiKiを使ってみます。
誰でもWelcomeです！！

参加費はNanoPiセットです。NanoPiは持ち帰ります。（自分のNanoPiを持ってきても構いません。）

やること

・NanoPiを使い、
・配線してSDカードから起動
・sshでログイン
・Linuxのオペレーションざっとおさらい
・ちょっとだけ電子工作
・LED点けたり消したり
・Webサーバインストール
・PHPインストール
・pukiwikiインストール
・動作確認と環境設定
