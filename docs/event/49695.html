description

これからの時代を生き残るために、どうやったらただの技術者からビジネスを創造できるクリエイターになるれるのか？

主催者からのメッセージ

「このままWebデザイナーとしてやり続けていいのだろうか・・・」
「新しい技術がどんどん出てくるけどなかなか追いつけない・・・」
「日々技術の勉強はしているけどなかなか収入に反映しない・・・」
「制作業という激務な仕事をこのまま続けていけるか心配・・・」

と感じたりすることはありませんか？

これは僕自身がWeb制作会社で働きはじめて、
2年目ぐらいで感じたことでした。

ぶっちゃけ、デザインやプログラミングは本当にすごい人は上には上がいて、
「勝てないな」と思うようになりました。

それから僕は制作（受託）だけに頼らなくても生きられるスキルを身に付けて、
去年5年間勤めた会社を辞めて独立しました。

「脱Web制作」を意識していたということです。

同じようにモヤモヤとした不安や限界を感じているあなたに、
次世代型のWeb制作スキルを活かした働き方について話たいと思います。

Web制作スキルを持っていることは、
実は今の時代どんな人よりもアドバンテージがあります。

テーマは『Webデザイナーのための「脱Web制作」プロジェクト』です。
Web制作者のためのネクストステージという感じですね。

今回この企画は初めて試みなので少数で開催します。
定員は5名にしていますが、参加者は1人でもいいと思っています。

というのも僕と同じような感覚の人が1人でも見つかればいいと思っているからです。

こんな方におススメです

  • Web制作の仕事をしている方(会社・フリーどちらでも)
  • 以前Web系のお仕事をしていた方
  • 簡単なWebサイトなら作れる人
  • 技術だけでないWebデザイナーになりたい方
  • 「激務で薄給の制作を続けるのは限界がある」と感じている方
  • ランディングページ（LPO）に興味がある方
  • Webマーケティングに興味がある方
  • スキルアップに興味がある方
  • 自分でビジネスを作ってみたい方
  • 自分で企画する力を身に付けたい方

どれか1つでも当てはまっていれば是非参加を！

なぜこれを開催するのか

長年疑問を持ち続けてWeb系の仕事をしている人にずっと提案したかったからです。

内容

Web制作スキルを活かして次のステージへ飛躍するための話
いったんゼロにしてパラダイムのシフトチェンジを目指します。

主催者

楠本真也
1984年生まれ。大阪芸術大学卒業後、不動産系の上場企業に就職するも1年で退社。
「やっぱりクリエイティブな仕事をしたい」と思い、Web業界へ転身。
Web制作会社で5年間で300サイト以上の法人クライアントを相手にWebデザイナー、フロントエンジニアとして働く。
給料の少なさから制作の仕事をしながら個人事業で2年間で1400万円売上。2013年会社を退職し独立。
