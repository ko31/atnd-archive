アニメプロデューサーが業界のリアルを語る！３セッション

デジタルハリウッド大学 国際アニメ研究所では、

アニメ産業の現状を多角的にキャッチアップできる

フォーラムを開催します。

日本のアニメ産業、クラウドファンディングでの製作資金調達、中国のコンテンツビジネス

に焦点をあて、アニメ・ビジネスの課題や新しい取り組みを解説する2時間です。
セッション後は、スピーカーを交えた名刺交換のお時間も設けております。
アニメ産業に携わる方、アニメファンの方はもちろん、あまりアニメに馴染みはなくビジネスとしてご興味があるという方にもおすすめの機会です。お気軽にご来場ください。


 


０．本日のイントロダクション


高橋光輝 氏（たかはし・みつてる）
デジタルハリウッド大学 国際アニメ研究所


映像製作会社にてCM、PV、TVなどの製作従事しつつ、教員を兼務。その後、デジタルクリエイター育成の教育機関で統括業務も行う。同時に大学及び研究所にてメディア・コンテンツ分野の研究員として活動。特にメディアミックスや映画やアニメなどコンテンツ分野で著書も発行。著書として『アニメ学』NTT出版、『コンテンツ産業論』ボーンデジタル等がある。


１．日本のアニメ産業の変遷と現状


岩田圭介 氏（いわた・けいすけ）
株式会社エー・ティー・エックス　代表取締役社長


1979年に東京12チャンネル（現テレビ東京）入社、営業～編成～報道を経て1993年から編成局で数多くのアニメ番組をプロデュース。その後コンテンツ事業局でアニメ事業を担当、現在に至る。 『新世紀エヴァンゲリオン』『ポケットモンスター』『ＮＡＲＵＴＯ』『遊戯王デュエルモンスターズ』『ケロロ軍曹』『ヒカルの碁』『テニスの王子様』など多くのヒット作品を手がけ、現在まで関わったテレビアニメは数百本にのぼる。


２．クラウドファンディングを活用した
       アニメ製作資金の調達方法と海外展開


真木太郎 氏（まき・たろう）
株式会社ジェンコ代表取締役


1955年岐阜県生まれ。1977年に早稲田大学法学部を卒業後、東北新社入社。1990年パイオニアLDC入社。1997年に株式会社ジェンコを設立。主なプロデュース作品として「機動警察パトレイバー」「天地無用！」「あずまんが大王」「千年女優」「東京ゴッドファーザーズ」など。


３．中国におけるコンテンツビジネスの現場

高橋光輝 氏（たかはし・みつてる）
デジタルハリウッド大学 国際アニメ研究所

張宇 氏（ちょう・う）
日本三聯集団株式会社　株式会社サンレン
