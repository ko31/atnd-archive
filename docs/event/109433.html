The Document Scanner is a image scanner application that make your phone to scan documents using your phone camera as well as pick image from gallery to scan.

Using Document Scanner you can scan any type of document like photos, receipts, reports, bills etc. The scan file can be converts in different scan modes that is Sharpen, Greyscale and Black & White.

Document Scanner allow you to rotate image after scan the document.

Document Scanner allows you to make PDF file form your scanned images. It also has rearrange images option to re-order images as your needs.

You can share your scanned document by this app on other plateforms like-

- Email
- Facebook
- Whatsapp
- Google Drive etc

The Document Scanner app have feature to capture image from any angle of document and it scans and correct the document image edges. So it is best for any situation of scanning.

The Document Scanner app provides optimised file so that it can easily transfer on any platform using internet.

The Document Scanner app have rich image viewer with zoom feature. So you can view any document at any size.
