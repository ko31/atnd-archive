セッション概要

まもなくリリースされる新OS Windows 8に備えて、改めてMetroスタイルアプリについて一緒におさらいしましょう。

Windows 8に導入される「Metroスタイル」は、タッチ操作を意識した新しいUIです。このため、Metroスタイルアプリは、ガイドラインに従って作成する必要があります。Metroガイドラインに準拠したアプリケーションが作れるよう、デモを交えながらご紹介したいと思います。

また、合わせて、Windows 8の面白い使い方をご紹介します。これまでExpression Encoderなどを使わないとメディア配信が出来ませんでしたが、Windows 8はこれら機能の一部を標準で備えています。Windows 7と連携もできるので、2台目のPCをWindows 8にする方におすすめの内容です。

エバンジェリスト養成講座
～北陸マイクロソフトコミュニティ限定版～

西脇 資哲


プロフィール

IT業界の著名カリスマエバンジェリストの一人(Wikipediaによる)。

エバンジェリストとはわかりやすく製品やサービス、技術を紹介する職種。現在はマイクロソフトにて多くの製品・サービスを伝え広めるエバンジェリスト。1990年代から企業システム、データベース、Java、インターネットのビジネスに関与し、1996年から約13年間オラクル社にてエバンジェリストとして従事。その後、2009年にマイクロソフト社にてエバンジェリスト活動を継続。コミュニケーションやデモンストレーションといった分野での講演や執筆活動も行い、IT 企業だけでなく、製造業、金融業、官公庁でのプレゼンテーション講座を幅広く手がける。

著書に「エバンジェリスト養成講座～究極のプレゼンハック100～」など。


セッション概要

毎回満席となる有償講座として有名な「エバンジェリスト養成講座」を北陸コミュニティの方々に特別限定版としてお届けをいたします。エバンジェリストだけではなく、あらゆる職業において、「伝える」ということは大変重要な役割となっています。伝わるプレゼンテーション、魅せるデモンストレーションを実現するためのテクニックなどを北陸コミュニティの方々限定でお届けをします。
実際にOffice365などを使用してのデモンストレーションもご覧頂きながらお届けします。


ASP .NET MVC4でWindows AzureのSpring Release機能を余すところなく使ってみる！

勇 大地


プロフィール

第二次思春期と第三次成長期に突入している Microsoft MVP for Windows Azure でございます(/ω＼)ｲﾔﾝ


セッション概要

2012年6月、 Windows Azure に多くの新機能が公開されました。Node.js, Java, PHP 等のさまざまな OSS との連携も大幅に強化、WEB SITES による簡易デプロイ機能の提供、VIRTUAL MACHINES による IaaS 環境の提供、Linux のサポート、Git デプロイのサポート等々、インパクトの大きな機能が多く提供されました。これら新機能に ASP.NET MVC4 を組み合わせ、様々な状況（オンプレ連携、クラウド移行）・デバイス（旧ブラウザ、モダンブラウザ、タッチデバイス等々）に対応可能で柔軟なアプリケーションを構築することが可能なことを紹介したいと思います。
より強力になった Windows Azure を利用した開発をとくとご覧あれ。


Featuring Project Silk ＆ Liike:
楽しい “モダン” Web 開発のちょっとディープなお話

井上 章


プロフィール

ASP.NET をはじめとする Web 開発テクノロジー好きのエバンジェリスト。ハードウェア設計に始まり、Z80, DSP のアセンブラプログラミング、C++/MFC/ATL による Windows プログラミング、そして ASP.NET をベースとした Web 開発やソフトウェア開発に関する書籍執筆などのエンジニア人生を経た後、2008 年にマイクロソフトに入社。高校時代に付けられたあだ名 “チャック” を使って、いろいろなところで活動中。今回は、久しぶりの北陸出張でルンルン気分な今日この頃でございます(/ω＼)ｲﾔﾝ


セッション概要

マイクロソフト patterns ＆ practices チームから、モダン Web アプリケーション開発ガイダンス兄弟 (通称: Project Silk / Liike) がリリースされています。このガイダンスで提供されるドキュメントとサンプルコードを元に、今や必須とも言える HTML5/CSS3/jQuery などのクライアントサイド実装を中心とした “モダン” Web アプリケーションの実装例をご紹介し、Web 開発の今後を考察します。
