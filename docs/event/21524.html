description

先日、ひらっちゃんが東京出張してきた際、高円寺「ひもの屋」で突如浮上したこのイベント。ひらっちゃんが住む愛知県豊田市と東京の中間点…っていうには些か東側なんですけど、沼津港は美味しい魚を食べさせてくれる店が軒を並べています。
さあっ！あなたも沼津で食べまくり、ついでにツーリングを楽しむイベントに参加しませんか？

なお、東西いろんな処からいらっしゃいますが、東名沼津ＩＣ付近は集合可能な広い場所はなさそうとの情報を得ました。
とりあえず沼津千本浜公園の駐車場で集合、そこから沼津港に向かいたいと思います。

沼津港では、鮨庵さいとう みなと旬彩街店でガッツリ食べまくりたいと思います。
（丸天は予約とれませんでした。ゴメンナサイ）

鮨庵さいとう みなと旬彩街店(食べログ）
