大会概要

ポケモンGOトレーナーバトルのスーパーリーグの大会を開催します。ポケモンGOをさらに楽しむ事でポケモンGOマナー向上やトレーナーの交流を深め、地域社会の活性化に努めます。

参加方法

画面下の参加ボタンを押して参加下さい。
もしくは主催者twitterへダイレクトメール。
又はpgomoka@gmail.comへメールお願いします。当日参加もお待ちしております。

試合形式と当日の流れについて

受付

・参加者は大会当日13:30までに会場にご集合ください。
・入場受付時に受付、くじ引き、参加費用500円支払をして頂きます。

予選リーグ

・総当たり戦で予選を行います。
・3本勝負2本先取。

※予選終了時に同順位となった場合は、対象者で勝負し、勝者が決勝トーナメントへ進めます。複数人の場合は総当り勝者がトーナメントへ進めます。

決勝トーナメント

・予選上位16名によるトーナメント戦
・3本勝負2本先取。
・準決勝ののち3位決定戦を行い、決勝戦を行います。

表彰式・総評

優勝者にトロフィー、上位ベスト8入賞者に記念品を贈呈します。

基本共通ルール

・大会で使用できるポケモンは計6体とし、同じポケモンを複数使用する事は不可。
・6体のポケモンを記入した用紙をお互いが見せ合います。
・使用するポケモン3体を2分以内に選出して決定します。お互いに時間を測定お願いします。
・選択した6体のポケモンの中で、大会中の技マシン使用、別技構成の同じポケモンを使い分けは選出時間内であれば可能。
・じゃんけんにて勝った方がQRコードを出し、負けた方がカメラで読み込みます。
・6体の登録パーティに入っていないポケモンを使用した場合は、いかなる理由であっても負けとなります。
・エラー等のトラブルがない限りは再戦ボタンを選択して連戦して頂きます。
・アプリおよび端末がフリーズした場合、フリーズ・不具合を互いに確認後、再戦して頂きます。この時のパーティ変更は不可とし、お互い必ず同じ先手にて再戦して頂きます。

大会当日に必要なもの
