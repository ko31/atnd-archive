「サラリーマン不動産投資家」の先駆け、田中実氏！
サンワード貿易セミナー初登場！！

田中氏は、「予測されるあらゆる情報を収集分析し、１つの仮説戦略を立てて、実行する！」ことが大事だといいます。
歴史上の人物である織田信長や豊臣秀吉、かの有名な松下幸之助も仮説戦略を実践していたといいます。

田中氏は、バブルが起こることを予測し、
バブルが崩壊することを予測してきました。
結果、290万円から6億円の資産形成をするに至りました。

これらすべて、予測される情報を分析して、仮説戦略実行の結果だと言います・・！
田中氏が30年に1度という仮説と、現在実践している戦略とは・・！？

東京は今、2020年のオリンピック開催に向け再開発の真っ只中・・！
交通網の利便性がどんどん向上しています！
ハイパーインフレもくるであろうと予測する田中氏が今注目している不動産とは・・！？

さらに田中氏は、去年から貴金属の投資で1500万円の実績を上げたといいます。
「谷深ければ、山高し」という
田中氏が今最も注目する投資先とは・・？
