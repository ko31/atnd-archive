BIT VALLEY -INSIDE- Vol.12

渋谷界隈にオフィスを構える企業による合同勉強会のコミュニティ、BIT VALLEY -INSIDE-。
参加はオープンなので渋谷以外の企業の方も参加いただけます。

第12回は倉貫 義人さんの著書（発売は8/31を予定）である、『ザッソウ　結果を出すチームの習慣　ホウレンソウに代わる「雑談+相談」』のゲラ原稿のABD読書会として開催します。

開催日：2019年8月15日（木）
会場： 渋谷ヒカリエ 17F（レバレジーズ株式会社 本社）
東京都渋谷区渋谷2-21-1
参加費：無料

スケジュール（予定）
※当日内容を変更する場合もあります。ご了承ください。

19:00 受付開始

19:30 〜21:30
『ザッソウ　結果を出すチームの習慣　ホウレンソウに代わる「雑談+相談」』のゲラ原稿のABD読書会


《ABD開催に寄せて》
仕事の基本とされてきた「ホウレンソウ」に代わり、チームを活性化させる「ザッソウ（雑談＋相談）」の重要性を説く『ザッソウ　結果を出すチームの習慣　ホウレンソウに代わる「雑談＋相談」』。本書のアクティブ・ブック・ダイアローグ®読書会をとおして、全国各地でチームビルディングについての対話、共創が生まれることを祈っています。
