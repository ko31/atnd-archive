サーバ３分クッキング講座

自分のパソコンからサーバに対して初期設定からWEBサーバ、DBサーバ等を３分以内に構築することを目指します。

クックパッドのインフラエンジニアの方が作られた「 itamae 」というツールを使ってサーバを構築していきます。

目次

1. プログラム言語Rubyの環境設定
2. Rubyを取り巻くツールrbenv, bundler等の設定
3. サーバ構成管理ツール「itamae」のインストールと使い方
4. サーバの初期設定について
sshの設定、ユーザの追加、ユーザへの管理者権限の付与、ファイヤーウォールの設定、不要なサービスの停止、アプリケーションのアップデート
5. WEBサーバを構築してみる
nginxでバーチャルホストの構築
6. DBサーバを用意してみる
MySQLの初期設定

参加費

無料

対象

Rubyやってみたい（実際は今回あまり触りません）
Linuxサーバに興味がある
itamaeを触ってみたい
サーバの構築に疲れ気味

用意するもの

ノートパソコン(Windows/Macどちらでも可能OK)
