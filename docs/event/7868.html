ねこんかつ子猫ＰＨＯＴＯ企画です！
４SEEN人気企画です☆猫好きの方、写真が好きな方、癒されたい方大歓迎！恵比寿のおしゃれな猫カフェで男女ペア（またはグループ）で猫photoを撮ります。カメラが苦手な方も大丈夫！プロのカメラマンがコツを伝授します♪～講師紹介～石原さくらさん
「幼少の頃から私の傍らにはいつも猫がいました。絵を描くことが好きで猫や犬を写真に撮っては絵に描いていました。私たち人間ほど長生きはできない彼らなのでお別れをする悲しみがありました。寂しくなったときに自分で撮っておいた猫の写真を見ると、その泣き声や感触などの思い出あふれて温かい気持ちになりました。写真の力の素晴らしさと撮ることの楽しさを最初に教えてくれた猫に敬意を込めて、猫たちの写真を撮っています。＜略歴＞
1995年 東京工芸大学写真学科入学
1996年 パフォーマンス集団大豆鼓ファームで舞台写真を撮りまくる。
1999年 東京工芸大学写真学科卒業
2001年 カメラマンとしてのシゴトを覚える
2002年～ 女性誌を中心に活動
2007年 まごくんという野良猫を拾ってから猫熱が再燃
2008年 愛玩動物飼養管理士２級を取得
2009年 2月　キャットショーの楽しみを知り、ますます猫に魅了されて被写体は猫中心となる。
2009年 9月　個展「tobi-neko」を仙川niwa-coyaにて行う
2010年 2月　愛玩動物飼養管理士1級取得
当面の目標はハイクオリティな猫のブリーディング。撮影が終わった後は、みんなで作品を見ながらをおしゃべりを 楽 し みましょう。猫カフェは隠れたデートスポット。かわいい猫と一緒だと、相手との距離も自然に縮まるみたいです☆
