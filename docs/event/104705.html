ポケカ新歓会

新年度という事で、ロータスに集まってポケモンカード新歓会を開催します！

ポケカを始めたばかりの方も、始めたばかりのプレイヤーを一緒に歓迎してくれる方も大歓迎！(　ﾟдﾟ)

豪華景品が当たるビンゴ大会も開催します！ぜひぜひ参加して下さいね！

新年度もヤルカ！ポケカ！

◆会場の様子
GAME BAR LOTUS

イベント概要

当日はフリーバトルや、デッキ作りのポケカ相談。ジムバトルやチャンピオンズリーグへ一緒に取り組む仲間つ作りをサポートします！

「今からポケモンカードを始めたい！」「ポケモンカードを遊ぶ仲間が欲しい！」という方もぜひぜひ参加して下さいね。初心者の方は、テキストやルールでわからない事や困ったことがあれば遠慮なく周りスタッフに声をかけて下さい。ゼンリョクでサポートします！

経験者の方も初心者の気持ちに帰って、和気あいあいと楽しく遊びましょう！
