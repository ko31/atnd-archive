《SUMMARY》
◎世界で通用する「ビジネスエンジニア」の育て方とは

日本初、日本発のITグローバル企業を目指して株式会社ヘッドウォータースを創業。
いち早くアジアにも進出。
その創業者が進出した先のインドのとてつもなく優秀なエンジニアの姿を体感するなかで、
これからの日本のエンジニア像、求められる競争力について問題提起。
技術バカのSEをビジネス全般に精通した「ビジネス・エンジニア」として鍛え上げる方法論を紹介。
そして、会社の中でSEが命がけで仕事に取り組める組織のあり方や、多くのSEが実は苦手としている
コミュニケーション能力やマネージメント能力をどう磨くか、などについて語る。

《PROFILE》
篠田 庸介

1989大学を中退し、草創期のベンチャー企業に参画
1997年株式会社スマートビジョン設立
2005年株式会社ヘッドウォータース設立、現職に就任
2008年株式会社東忠ヘッドウォータース設立
2009年ベトナム・ハノイにLifetime Technologies Co.,Ltd.を合弁会社として設立
2011年 カンボジア・プノンペンにてJC IT Co.,Ltd.を関連会社として設立
2014年ＵＡＥ・ドバイに子会社設立
