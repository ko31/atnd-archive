新機能についてのアンケート

こんにちは。ATND運営事務局です。
ATND運営事務局では、今後皆様に「より便利に」、「より楽しく」ATNDをご利用いただくために、以下の新機能の開発を検討しています。

①資料共有機能

イベント作成者の方がイベントの関連資料(パワーポイント・エクセル・ワード etc.)をアップロードできる機能です。
当該イベントの参加者は自由に資料のダウンロード・閲覧ができます。

②会員登録せずに参加できる機能

ATNDに会員登録することなくイベントに参加できる機能です。
参加者を代理で登録したり、ATNDを利用したことの無い友人・知人を気軽に招待できるようになります。
また、会社の懇親会やプライベートな飲み会の出席管理などにも気軽にご利用いただけます。
会員の参加登録との併用が可能です。

上記2機能に関する皆様のご意見をお伺いしたく、アンケート専用イベントを作成いたしました。
また、その他新機能のご要望も受け付けております。
アンケートにご回答いただける方は、本イベント参加時に表示されるアンケートフォームにご入力下さい。
みなさまの忌憚のないご意見をお待ちしております。

回答期限：2015/06/22 18:00
