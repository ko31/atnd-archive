IT系エンジニアがお金を稼ぐことについて真剣に考えるための集い、rake:moneyの第3回を開催します。
当日のスケジュールは以下の通り。


  • 19:55 集合＠オラクル青山センター2Fロビー

  • 20:00〜22:00 勉強会＠オラクル青山センター13Fセミナールーム

  • 22:00〜 二次会（懇親会）

二次会の参加を希望される方は、必ず登録フォームの「二次会に参加する」にチェックを入れてください。

また今回から参加される方は、連絡のためにGoogleグループのrake:moneyに参加をお願いします。

■Googleグループ rake:money



ドタキャン禁止、遅刻も禁止、恥ずかしいセリフ禁止！

会場はオラクル様のご厚意により、本社のセミナールームを貸していただけることになりました。
入館のためには一人ひとり入館証が必要で、あらかじめ担当者の方に人数分の入館証を発行していただきます。その受け渡しのために2Fロビーにいったん全員集合する必要があり、もし遅刻者がいればそのたびに13Fセミナールームから2Fロビーまで迎えにいく必要が生じます。

ですのでドタキャンおよび遅刻はいろんな人たちに迷惑をかけることになります。それをご承知の上、ご参加くださるようお願いします。


さて今回のお題は「エンジニアが月100万円稼ぐ方法」です。
このご時勢、コンスタントに月100万円を稼ぐのはなかなか難しいとは思いますが、過去の体験談を含めてどうすれば効率の良い稼ぎ方ができるのかを話し合いましょう。

今回も参加者の皆様方に、課題を出します。
必ず参加者全員、各質問につき最低１個ずつの回答を当日までに用意してきてください。


    1. 瞬間風速でも、月100万円以上稼いだことがありますか？
    2. （＞1がYESの方）どうやって月100万円稼ぎましたか？
    3. 自分の周りに月100万円以上稼いでいる人がいりますか？
    4. （＞3がYESの方）その人はどうやって月100万円稼いでいますか？
    5. エンジニアが月100万円以上稼ぐための戦略として、何かアイディアをどうぞ

また恒例の推薦図書の持ち寄り（第１回、第２回）を今回も行います。
お題に沿っていてもいなくても構いませんので、お金にまつわるお勧めの本を各自1冊ずつ持参してください。
なお、これまで誰かが持ってきた本と重複してもかまいません。

なお開催にあたり、Ust番長、書記を募集します。
Ustreamが配信できる環境を持っていて番長をやってもいいという方、ネットにつながる環境を持っていてリアルタイムでWiki議事録を編集する書記をやってもいいという方は、アンケートフォームにチェックを入れてください。
