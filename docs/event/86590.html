description

自家採種した種や買った種…家の畑に蒔くと余ってしまうけれど、みんなでシェアし合えば、集まった人の数だけ種も集まって、多様性に富んだみんなの財産に！

種とともに、想いや育て方・おいしい料理法などをシェアできたり、その後も繋がり合いながら、種の育ちや実りについてシェアし合えるのが、コミュニティで種交換することの楽しさ、喜びです！

農家さんや、自然農を愛する仲間による『種や畑のレクチャー』、土に触れながらの種まき、種ごはんのシェアも盛りこんだ、贅沢ないちにち。
ご家族、こどもたちの参加も大歓迎！みんなどろんこになって楽しんでくれるはず！

みんなで“野良仲間の輪”を楽しみましょう！

◆日時：4月1日（土）10：00〜15：30

◆会場：榊（さかき）家のはたけ
真岡市根本1483
（連絡先：080-2211-1329）

✳︎瓦屋根の平屋の家です。石の門に表札があります

◆内容：
1）種の交換会&種のレクチャー
自家採種した種や、購入した固定種の種や苗を持ち寄り、みんなで想いとともに交換しましょう！
農家さんによる、種とりなどのレクチャーも行います。

2）畑のレクチャー&種まき
自然農を愛する仲間による畑のレクチャーと種まきの時間。
土の様子や生えている草の様子から知る畑の元気度、種まきのコツなどを聞きながら、みんなで土に触れ、種を蒔いてみましょう！

3）種ごはんのシェアランチ
種がテーマの持ち寄りランチ。豆や芋、雑穀や米も種ですね！種を味わいながらみんなで語り合いましょう！

◆タイムスケジュール：
10：00 スタート&自己紹介
10：30〜 畑のレクチャー&種まき
12：30〜 種ごはんのシェアランチ
13：30〜 種の交換会&種のレクチャー
15：30 終了

◆持ちもの：
種交換用の種や苗（自家採種または購入した固定種のもの）、野良着（汚れてもいい服）、農具（ショベルやスコップなど。あればでOK）、軍手、長靴、着替え、種ごはん、マイ食器（箸・皿・カップ）

✳︎交換する種がない人は、ドネーションやギフト、地域通貨じねんでも交換できます！じねんを使う場合はじねん手帳を忘れずに。
