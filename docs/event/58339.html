2部 15:00- 司会： 田村祐 【Lightning Talk】
1. 川口勇作 「Rprofile作成のススメ」：普段使うCRANの指定や毎回使うパッケージの読み込みなどをR起動と同時に自動設定できる「Rprofile」について紹介します。（5min） 投影資料
2. 川口勇作 「非線形の相関関係を検出する指標の算出」：従来の相関係数では検出できなかった非線形の関係を検出できる指標MICとMICを算出するためのminervaパッケージの紹介をします。（10min） 投影資料
3. 福田純也 「Rを用いた一般化可能性理論」：古典的テスト理論である相関係数や信頼性係数からは評価が困難であった，主観的評価に影響を与える複雑に絡み合った要因を評価する方法である一般化可能性理論を紹介します。（5min）
4. 田村祐 「一般化線形混合モデル入門の入門」：外国語教育研究のデータを例に，lme4パッケージのlmer関数の基本的な使い方を説明します。（30min） 投影資料
5. 阿部大輔 「Rでテキストマイニング」：tmパッケージを使ってテキストマイニングをする方法を紹介します。（20min）
6. M, Yoshino 「非線形回帰（レーベンバーグ・マルカート法）と信頼区間（ブートストラップ法）」：技術者が直面する曲がったデータを片対数，両対数やロジットなどで関数フィティングし、その信頼区間、予測区間を求めます。使用するライブラリは非線形回帰のパッケージ「minpack.lm」です。デフォルトのnlsは、ガウス・ニュートン法が使用され、データが暴れているとエラーになるので、レーベンバーグ・マルカート法を使って回帰します。また、信頼区間は非線形回帰の定石であるブートストラップ再標本化を使って算出します。（30min） 投影資料
7. 阪上辰也 「モザイクプロットいかがですか」：分割表を可視化する一手段としてのモザイクプロットについてお話します。（10min）

＊第一部の始まり(13:00-)と第二部の始まり(15:00-)に合わせて開場いたします。第一部・第二部のみ参加も大歓迎です。
