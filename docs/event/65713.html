四年目に突入！　第十回『#関西ロドクラ集まってみよう』開催です♪
日本の夏、ロドクラの夏。台風さんのご参加はお断りの方向で。

開催地は、恒例となりました京都府立『丹波自然運動公園』です。
京都縦貫道丹波 IC を降りて少し走ったところです。
12時に、国道9号線東側の入り口から入ってずっと奥にある芝生広場に集合してください。
みなさんでまったりロードスターを愛でたりロドトークしましょう(´▽｀*)
もちろん、恒例の自己紹介大会やじゃんけん大会も予定しています。

お弁当を用意する予定ですが、参加者多数の場合、足りなくなるかもしれません。ご了承ください。
喫茶室の軽食等もございますので、是非ご利用ください。

現役ロードスター乗りじゃなくても、ロードスターに興味のある方なら大歓迎！
ロードスター以外にも、PORSCHE 911 や Boxter、スイスポ、Cappuccino、Copen などなど、ご参加くださってます♪
もちろん、助手席参加も大歓迎♪
『初心者だし……』て方も、是非是非！
クルマ以外のお話も、いろいろしましょう(´ω｀)

みなさまのご参加、お待ちしております！
