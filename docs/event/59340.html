講演概要

SpringBootを用いたドメイン駆動設計

  • 出演：風間 淳一郎
    • 株式会社マネーパートナーズソリューションズ システムマネジメント部部長。
      Web系のシステム開発を経験した後、2004年より金融系の開発に従事。
      2007年より現職。SpringFrameworkを2004年から利用。
  • 概要
    • 現場で実際に運用されているドメインモデリングをSpringBoot基盤に載せた形で紹介します。
      なるべく具体的なコードを用いて説明したいと思っています（残高/入出金辺りの具体的なコード例をSpringBootピュア(+ Lombok)な実装で作ってみようかと思っています）

オムニチャネルソリューション~hybrisとSpringフレームワーク

  • 出演：ムニンドラ・エッデラ
    • ハイブリスジャパン株式会社 シニア コンサルタント。
      B2Bウェアハウスシステムのシステムインテグレーションのプロジェクトにて来日、その後日本に滞在。
      Fioranoデータインテグレーションソフトウェアでの国内エンジニアとして従事し、2012年にハイブリスに参画。
      Spring Framework を含めて様々な Java フレームワークを経験し、現在はハイブリス導入の際の技術コンサルタントとしてプロジェクトを支援。
  • 出演：マーシャル・ジェン
    • ハイブリスジャパン株式会社 シニア コンサルタント。
      法律判例Webアプリケーションシステムプロジェクトにて2006年に来日。日本での業務を継続し、楽天株式会社に入社。
      楽天にてバックエンドエンジニアおよびWebサービスチームにてSpringをベースにしたI/F構築を経験した後、2014年にハイブリスジャパンに入社。現在はムニンドラとともにお客様の導入プロジェクトの支援に従事。
  • 概要
    • グローバルの多くの著名企業にて採用されているオムニチャネルソリューション hybris。
      この hybris のソリューションの概要と、そのソフトウェアの中で採用されている Spring フレームワークのテクノロジーとそのメリットについて紹介

試験に出るSpring

  • 出演：土岐孝平さん
    • 大学で情報工学科を専攻。業務では多くのシステム開発プロジェクトに従事。
      現在では主にJavaEE（特にSpring Frameworkを得意とする）を使用したシステムのアーキテクトやコンサルタントとして活動しており、研修講師・記事執筆等もおこなっている。
      また、これまでの経験から、保守／運用を楽にするツール開発を独自におこなっている。
  • 概要
    • Springを如何に学ぶか。詳しくは当日のお楽しみ♪
