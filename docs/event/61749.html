概要

2014年12月、KDDI は「つくる自由」をテーマにFirefox OS スマートフォン Fx0 を発表・発売しました。
Fx0 はHTML5等のWeb技術をベースとしたFirefox OSを採用し、アプリケーションやサービスをつくる楽しさを体験できるスマートフォンとして提供しています。

一方で Web 技術は、様々なモノが Web を介してつながる WoT（Web of Things）の世界でも利用され始めています。これまでスタンドアローンで動作してきた組み込みデバイス（モノ）が Web につながることで、インターネットやソーシャルの世界と実世界がリンクし、ヒトの生活に新しい刺激を生む可能性を秘めています。

昨年、KDDIは英国 ARM 社が推進するデバイス開発プラットフォーム mbed と出会い、以来 mbed の Web につながるモノを簡単につくることができる点に注目してきました。
今回のイベントでは、発売されたばかりの Fx0 と、デバイスづくりの簡単さを特徴とする mbed を組み合わせて、Web につながるモノづくりと、モノとヒトをつなげる Web アプリケーションづくりを行います。
