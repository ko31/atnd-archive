description

概要

なんとなく、自転車走行会を開催します。
楽しく安全に自転車走行を楽しみたい方は誰でも参加していただけます。

ペースは、バラけないようなペースを維持します。
途中、なんちゃってスプリントレースなど行い遊びながら、走行できればと考えています。
涼しいですが、水分補給にスポーツドリンクなどを持参してください。

難波自転車道抜けたら、休憩します。
その他、必要があれば休憩いれます。

一般道も走ります。一般道は交通規則を守り
一列で走行しましょう。

詳細情報

  • 集合場所：シルベストサイクル梅田店　前　目印は緑色のtrekです
  • 走行距離：約28km
  • 走行時間：約1時間30分
  • 服装：自由（ぴちぴちな服装じゃなくてもOK）
  • コース：難波自転車道-淀川-十三
  • コース図：画像参照

※梅田店住所：　大阪府大阪市北区梅田2丁目5－2 大阪新サンケイビル 1階

用意するもの/あったらいいもの

  • 水分500ml程度（必須）
  • パンク修理キット

参加資格

  • 自転車を所有している方（ママチャリでもOK）
  • 自転車で30km程度、走れる方
  • 15km程度で休憩いれます。30kmはあくまでも目安です
