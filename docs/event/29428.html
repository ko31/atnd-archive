概要

待ちに待った第3回です。今回は、前回の応用編です。

  • もっとWidgetを使ってみよう！
  • Widgetを使って、アプリを作ろう！

と言うことで、ちょっとハッカソン的な感じにします。

進め方

お菓子持ってきていいよ！！
ペアプロを導入しています。知らない人とペアになって、一緒に頑張ってみてください。

ハッカソン・・・？

Hack + Marathon = Hacathon!!
と言うことで、時間内にアプリの

  1. アイデアを考えて
  2. 実装をする

を行います。が、まだまだアイデア分からないとか、不安と言う人も多いと思うので、ひな形となるサンプルアプリケーションを作りました。
当日は、GitHubからのサンプルアプリケーションのダウンロードと、インポートを最初に行います。

持ち物

開発環境が構築されたマシン。
お菓子。ゴミの処理は責任をもって。

ネットワーク環境

学生は、学内無線LANを利用してください。一般の方は、自分で用意をして頂くか、学内の生徒とペアを組んでください。

電気通信大学学生でない方へ

上記の通り、ネットワークの環境を提供できません。ですので当日の資料やサンプルコードを事前にダウンロードしておいてください。
いずれも、勉強会前日に最新版をアップロードします。アップロード先のリンクは、リンクの項目を参照してください。

もくもく会

当日10時から、もくもく会と称して復習会を行います。
公式ブログで公開している「Android道第2回　補完資料」を利用して復習を行います。
今までの勉強会に参加してない人、前回以前の内容に不安がある人、そして開発環境に不安がある人は是非参加して私や周りの人に質問などをしてください。
