コッド論文を読もう

最初にＲＤＢ理論を打ち立てたコッド博士。でも実際に論文を読んだ人は少ないのではないでしょうか。今ＲＤＢの原点を振り返ってみましょう。

英語が読める方は事前に読んでおいてください。
「A Relational Model of Data for Large Shared Data Banks」 （ＰＤＦが開きます）
http://citeseerx.ist.psu.edu/viewdoc/download?doi=10.1.1.110.1845&rep=rep1&type=pdf

英語が苦手な方は、次の対訳を読んでください。
http://www.onas.asia/home/kwansaiit/doc/1970対訳.pdf

綺麗にプリントしたい方は、全国どこのセブンイレブンでも結構ですので、コピー機に行き「ネットプリント」で「29674020」と入力して下さい。パスワードは設定していません。★2011.12.15まで印刷可能です

　白黒A4で45ページが印刷出来ます。１枚20円ですので900円です。

ＲＤＢがどういう問題意識からどういう理論で生まれたか。そういう想いを語り合いましょう。後半の忘年会だけ参加でも結構です。
