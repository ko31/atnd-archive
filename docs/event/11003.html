概要

いいから飲め。
あと川と Linux (それ以外の Free OS でも可) について語れ。
Physical Computing でもいいぞ。
そして来年の抱負を語ろう！

riverlinux とは？

カーネル読書会 #106 の二次会にて、

「カヤックとリナックスって語呂似てない？」
「じゃあリバーカヤックをもじってriverlinuxとかどうよ？」

という酔っぱらいの狂った思考回路からできた怪しげな集団です。
Google Groups で活動してますが、なんかいろいろ置き場所とかまとめたいんで、来年あたりはどっかで VPS 借りてあーだこーだやりたいですなあ。

とりあえずメンバー優先ですが、river か linux か両方に興味がある方は歓迎です。
気軽に飲んで楽しみましょう。

場所と時間
