Couchbase のソリューションアーキテクトである Perry Krug 氏の緊急来日が決定しました！ Perry さんを囲んで久しぶりに couchbase-jp meetup を開催したいと思います。

Couchbase Server 2.0 のリリースも間近となってきました！最新動向や開発の裏話、気になる質問を Perry さんにぶつけましょう :D

また、今回の meetup では Node.js や HTML5 を駆使したソーシャルゲーム開発を得意とされている Wizcorp さんにご協力いただき、Node.js と Couchbase Server の連携についてのセッションも予定しています。

会場が変更になりました！ The venue has been changed!

変更後の会場は イグニス・イメージワークス株式会社 様のオフィスになります。
東京都渋谷区恵比寿1-18-18　恵比寿東急ビル8F

会場の入り口はセキュリティの都合上 20:00 に閉鎖されますので、それまでに会場入りするようにお願いします。やむを得ず遅れてしまう場合は Twitter @ijokarumawak またはお知り合いの参加者の方へご連絡ください。

なお会場ではWiFiアクセスポイントの提供はございません、予めご了承ください。

The revised venue is Ignis Imageworks Corp..
Ebisu 1-18-18 Shibuya Tokyo, Ebisu Tokyu Bldg 8F

The doors are open until 8pm for security reasons.
Please come into the room by then. When you arrive after 8pm, please mention about that to @ijokarumawak on Twitter or some participants you know.

No WiFi access point will be provided at the venue.
