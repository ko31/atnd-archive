世界№1のソーシャルゲームプラットフォーマーを目指す株式会社ディー・エヌ・エー、
ソーシャルアプリ事業を主軸とし史上最短での東証一部への市場替えを達成したKLab株式会社。

競争が激化するIT業界の中でも急成長する両社において、
エンジニアの採用や研修部門の責任者であり「Mobage を支える技術 〜ソーシャルゲームの舞台裏〜」（DeNA　著/技術評論社）の全体監修に携わった武部氏。そして、取締役であり技術部門責任者である天羽氏というエンジニア採用の中核を担う2名をが登壇。

何百人ものキャリア形成に携わってきた両者がエンジニアの「キャリアのススメ」について語ります。

今後のエンジニアに求められるスキル・マインドとは？
夢を実現するために今、するべきこととは？
エンジニア、もしくはビジネスパーソンとしての価値を高める方法とは？

当日は自身のターニングポイントとなった場面なども振り返りながら、「キャリアのススメ」の要素を紐解いていきます。

今後も技術力を磨いていきたいとお考えのエキスパート志向の方も、ゼネラリスト志向の方も今後のキャリア形成を考えるヒントとしていただけると幸いです。
奮ってご参加ください。
