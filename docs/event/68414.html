description

オンラインゲーム業界でひそかに注目を集めつつある、ゲームの移管・協業・買取によるコンテンツの二次流通マーケット。"セカンダリ市場"とも呼ばれるこの分野に早くから注目し、さまざまなパブリッシャーのゲームの運営に携わってきたマイネットが、ゲームプランナー・ディレクター、クリエイターを対象とした勉強会を実施します。

当日は、外部からも登壇者をお招きし、スマフォゲーム運営のコツについて講演を行います。

また、会の終わりにはマイネットや社外のゲームプランナー・ディレクターとの自由交流会を予定しています。お酒・ソフトドリンク、軽食などもご用意しておりますので、ぜひご参加ください！

ATND経由でのご応募以外からも、多数のご応募をいただいています。

登壇者紹介

松本 啓志

株式会社マイネット
『神姫覚醒!!メルティメイデン』ディレクター



マイネットの人気タイトル「ファルキューレの紋章」にプランナーとして携わり、2014年9月からはマイネット初の買収タイトルである「神姫覚醒メルティメイデン」のディレクターとして活躍。
三度の飯より「スマブラ」が大好きで、自身もスマブラのような誰もがワイワイと楽しく笑いながら楽しめるゲームを、いつか生み出してみたいと思っている。
ディレクター歴としては短期間ながら、マイネット社若手期待のディレクターとしてすでにイベント等での登壇歴も有り。独自の視点で切り込む「萌えアセット」は社内外で好評を得ている。

松田 崇志

株式会社トイディア
代表取締役CEO 兼『ドラゴンファング』プロデューサー



子供の頃から「ぴゅう太」に触れて人生を歪めるが自覚なし。高校受験に大失敗し、超ヤンキー高校に入りそれはそれで馴染むが、その後絵が描きたくなり親に土下座をして多摩美術大学に受験させてもらう。
大学卒業後はSEGA AM2研にデザイナーとして入社。（代表作：シェンムー、フェラーリF355チャレンジ）
さらに、マイクロソフトXboxスタジオ設立と同時に引きぬかれ、様々な作品を生み出す。（代表作：ファントムダスト、ロストオデッセイ）
iPhone4に触れたことで起業の時と判断し、トイディアを設立。現在同社にて『ドラゴンファング』を運営し、140万DLを突破。同タイトルの海外展開についても積極的に挑戦をしている。
