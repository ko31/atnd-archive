description

「疑問力　×　行動力　＝　売上向上」

起業したものの、ビジネスに行き詰まりを感じておられる、経営者であるあなたへ。

熱い想いを持ち、練りに練ったビジネスモデルを持って、世の中に変革を起こすべく起業した。
自社サイトも作り、Facebookの投稿も欠かさない。ブログの更新も怠らず、会社案内も製品・サービス案内も丁寧に作り込んだ。
「良いものであれば売れる！」そう考えていた。

･･･しかし現実は･･･
売上が上がらない。想定をはるかに下回る売上額。
製品・サービスは間違いなく良いものなのに･･･。
何故？？？

焦って、マーケティングや売上アップの方法を教えるセミナーにも参加した。
各々の講師の話は納得できるし、事実素晴らしいノウハウだ。
早速実施してみる。
･･･しかし結果はどれも芳しくない。
このままでは自分の熱い想いが世に伝わらないまま終わってしまう。

一体どうしたら前に進めるのか？売上が上がるのか？

売上がなかなか上がらないスタートアップ企業の経営者や老舗企業でも伸び悩んでいる経営者の方、とても重要な「ひとつのこと」を見落としている可能性があります。

私も創業半年くらいから売上が低迷し、そこから抜け出すためにかなり焦っていました。
しかし、ご縁をいただいて出逢ったビジネスの師から教えられたことを素直に実践し、状況は一変しました。

結果は 「売上3.2倍」 。現在も案件の発生が継続的に見えつつあります。

その師から教わったことの中で最も重要なことの一つが、「疑問力」と「行動力」なのです。
「そんなことで売上が上がるのか？」「当たり前じゃないか」というお声もあるでしょう。
しかし、その二つのチカラについて真剣に考えたことはおありでしょうか？

今回の限界徹底突破セミナーでは、「疑問力×行動力=売上向上」の公式を成立させるために必要なことをお伝えします。
そしてその場で、「疑問」に対して徹底的に考えていただきます。
重要なのは、「セミナー終わって帰ってから」ではなく、「今すぐその場で」やることです。
私のビジネスの師が創業時に実践し、爆発的に売上増を実現した内容を、皆様にお伝えします。

セミナー概要

１．イントロダクション～「スイッチ」が入った瞬間（ウソのようなホントの話）
２．陥ってはいけない落とし穴
３．疑問力と行動力の圧倒的な重要性
４．徹底ワーク

※今回限りの特典！！　師が勧めてくださった珠玉の「これは絶対読むべき一冊」を、ご参加の皆様にプレゼント致します。
