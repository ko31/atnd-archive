★講演内容

ゲームアプリの開発で大成功を収めている講師が再登壇！！

スマートフォンが普及しきった現在、
通勤・通学の電車やバスの中で周りを見回せば、
誰もがスマートフォンでゲームを遊んでいます。

「自分もスマートフォン向けのゲームアプリを作れれば、
　収益が得られるのでは…？」
……と考えるものの、
何から手を出せばいいかわからない、
難しそうで一歩が踏み出せない…。

…という方のために、まずはAndroidから！
カンタンなAndroid向けゲームアプリの作り方を講演します。

参加者全員に完成版のプログラムファイルを配布しますので、
「どんなゲームアプリを作るのか？」という企画部分と、
「どうゲームアプリを作るのか？」というプログラム部分が学べます。

講演内ではソースコーディングは行わないため、
プログラミングの知識は問いませんが、
配布されるプログラムファイルをビルドし、
アプリを完成させるためには、
Java 並びに JavaScriptの基礎知識が必要となります。

※内容は変更になる可能性がございます。
