■開催概要

テーマは「リアルタイム通信」です。
ゲームサービスを運営しているエンジニア様向けの勉強会です。
本イベントはゲームアプリ専用クラウドサービス「GMOアプリクラウド」が総合プロデュースしています。

第一部：株式会社クラウドクリエイティブスタジオ
　
　Unity によるスマートフォン向けアプリ「DEAD CITY」への Photon 適用事例

　3月末にリリースされたタイトル「DEAD CITY」では、Photonを利用したリアルタイム通信を行い、
　マルチプレイを実現しています。
　「DEAD CITY」でPhotonを採用した経緯と適用事例、特に一番の問題点であるモバイルにおける
　リアルタイム同期をどのように対策したかをご紹介します。

第二部：Photon by GMO
　
　GMOアプリクラウドではじめるPhoton Server v4

　GMOアプリクラウドでは、Photon Serverをすぐに使うことが出来るテンプレートをご用意しています。
　今回はその活用方法とPhoton Serverのカスタマイズについてご紹介いたします。

第三部：懇親会

　お食事とお飲み物（アルコールもあり）をご用意して懇親会も開催いたします。
　ぜひ皆様の交流の場としてご活用ください。
