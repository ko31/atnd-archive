★第1期プロアスリートネイルトレーナー養成講座ご好評につきまして、第2期プロアスリートネイルトレーナー養成講座を開催決定＆募集を開始！！★

開催日：　７月２日（土）～８月２８日（日）　　
　　　　　毎週日曜開催、全８回集中講座（一部土曜日開催）

開催場所：東京都渋谷区渋谷エクラート渋谷ビル　５F

受講時間：10:00-13:00、 14:00-17:00 / 13:00-14:00 ランチ休憩

※開催日・受講内容が変更になる可能性がございます。詳細の講座日程、講座内容はお申込み後にご連絡させていただきます。

アスリートネイルトレーナーとは、アスリートネイル協会が定義する新しいネイリストのジャンルです。アスリート専門に爪のメンテナンスを施すことができます！

養成講座は、
＊スポーツトレーナーや柔整鍼灸師などのスキルアップを考えている方
＊アスリートの第二の職業として活動したい方
＊スポーツチームのマネージャー、コーチ等の指導者でアスリートをサポートされている方
＊ネイリストの方のスキルアップとして幅広く活動したい方
＊治療院やネイルサロンのメニューに導入したい方
＊本気でアスリートの為に活動・貢献したい方アスリートのを支えたい方

などなど、どなたでも受講可能です♪

受講詳細は、WEBサイトよりご覧いただけます。
