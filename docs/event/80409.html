8/24(水)開催　第12回KROWLセミナー
- IT/デジタル業界の"いりぐち"がココにある -

OPEN 18:30 / START 19:00 / CLOSE 20:30

KROWLセミナーについて

Web制作・運営、インターネット広告、SNS、ソーシャルゲーム、オンラインショッピング、ポータルサイトの運営など・・・IT/デジタル業界現場のナマの声や、スキルアップに役立つ情報を毎月お届けしています。IT/デジタル業界でまだキャリアがなくスキルアップしたい方など、どなたでも歓迎ですのでふるってご応募ください。また、キャリアコンサルティング、インターンシップ情報を含めたKROWLの最新情報もお知らせいたします。
※友人とご一緒に参加もOK！

第12回 イベント概要

第1部 ： 講演
今回のセミナーでは、デジタルクリエイティブ企業でテクノロジー×プロデュースの融合に取り組み、部門責任者として人材マネージメントにも携わるゲストが講演します。
現在Webづくりでとても重要な”膨大な生活者のデータを仕事に活かす力”を顧客生涯価値を最大限にするためにデータを適切に解釈し、活用する方法を語って頂く予定です。
これからの現場で役立つトピックが満載ですので是非お楽しみに！

第2部 ： 質疑応答
ここをもっと詳しく聞いてみたい、現場でのココが知りたい、そんな疑問・質問にお答えします！ 現場で活躍するディレクター、デザイナー、エンジニアに気軽に自由に質問できる時間を予定しています。 また、ポートフォリオのご相談も受け付けます。（※作品を持ってきた方限定）

地図

主催： 株式会社 アイ・ファクトリー

博報堂アイ・スタジオグループの一員として、2011年に設立したWeb 制作会社。大手企業ならびに有名企業のキャンペーンサイトなどを中心にWeb サイトを制作・運用し、2014 年から人材サービスを開始。設立時より育成に力を入れ、Web 業界未経験者から現場で活躍できるクリエイターをこれまでに多く輩出した実績をもつ制作会社。

KROWL とは

これからIT/デジタル業界でチャレンジしていきたい方を対象に、セミナーや面談、キャリアコンサルティング、インターンシップ等の就業支援を行っています。
IT/デジタル業界を知っている企業がサポートしているからこそ、就職活動もスムーズに。是非KROWLの就業支援を体験してください。
