昨年も大好評だった利き酒師ライター市田真紀さんによる日本酒セミナー「備前焼で体感！雄町米で醸す地酒の魅力」を開催します。

セミナーは、来年1月26日（金）に開催する「備前焼で愉しむ雄町米の地酒BAR」の前、15:30～16:15（受付15:00～）に行います。
　雄町米のウンチクから備前焼の魅力までじっくりと体感していただけます！
