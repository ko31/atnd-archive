ドミニオン愛好家によるトーナメント大会です。
普段は仲間内でひっそりとやってるのですが、結構そういう人多そうだなー、色んな方を集めて対戦してみたいなーと思い企画してみました。

■概要
・3～4人戦のスイスドロー形式でドミニオンのトーナメント大会を行う。
・大会優勝者には、８月発売予定のドミニオン収穫祭を贈呈する。
・使用するドミニオンセットは、参加者持ち込みのセットを使用する。
　主催側では独牙の１セット（＋α）しか用意ができないので、持込歓迎。
　※持ち込み可能な方は参加表明時にその旨記載をお願いします。
・途中参加、途中退室は可能。（点数計算は参加した分だけで行う）

■参加費
５００円

■トーナメントの進行ルール
・３～４人の人数で試合を行う。
・試合は５回行い、試合ごとのデッキはこちらが指定したものを使用する。（デッキの詳細は下記参照）
・１試合は最大５０分で、時間切れとなった場合はその時点での点数を計算する。
・各試合の点数は、順位と勝利点を用いて以下のように算出する。
点数＝順位点（１位：６点、２位：３点、３位：１点、４位：０点）＋勝利点/１０（小数点以下切捨て）

・試合終了ごとに点数を集計し、合計点数が近い人同士で、次の試合を行う。（スタッフが決定する）

■ドミニオンの対戦ルール
・基本的には、ドミニオン日本語版のマニュアル（主催持参）に沿って行う。
・順位については、勝利点が同じなら後手のプレイヤーが上位とする。　
・ミスプレイにより、ゲーム進行に誤りがあった場合の処理は、各テーブルごとに話し合って決める。
（ミスプレイ前に戻すのか、そのターンをパスして貰うかなど）
・ルールの解釈、ミスプレイの処理で揉めた場合は、スタッフ裁定となる。
