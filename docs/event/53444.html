description

実用的なプログラミング問題を多数用意しているインターネットサイト「CodeResume」(コードレジュメ)を使用し、自分のレベルにあった問題を選んでプログラミングしていく勉強会です。わからないことは先生から聞いたり、持参した本やインターネットで調べたりして問題を解いて、プログラミング、データ構造、アルゴリズムを身につけることができます。

今回は通常の勉強会とは異なり、対象をC++のLevel.3以上を勉強している方を対象にします。Level.3とは、STL、データ構造、アルゴリズムなどを勉強している方で、今までに書いたことのあるソースコードの行数が約1万行を超えている方です。勉強会の後半は、データ構造とアルゴリズムの例題を解説します。

この部屋を出たときに、どれくらい戦闘力が上がっているかな！？

対象：
すでにプログラマだが、もっとスキルアップしたいという社会人の方
今はプログラマではないが、プログラミングを勉強したいという社会人の方
プログラマになりたいという学生の方
(純粋にプログラミングを学びたいという方も、参加していただいて構いません。)
