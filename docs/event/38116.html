description

ふとしたことから小松未歩さんゆかりの地、神戸を訪れることになりました。

ツイッターでは今も色褪せることない小松未歩さん好きな人たち。日がなお会いできたらいいのになって思っても、なかなかそういう縁を探すのも、ましてや自分で機会を作るというのも大変で。

だから、小松さんの聖地 神戸 に託つけて、きっかけだけでもと思ってイベント作成してみました。

小松未歩さん好き同士、ちょっと会えたらお茶でもしましょうヽ(´▽｀)/

ツイッターでお会いするのと同じように、オフでもそんな緩い感覚で。

自分には、小松未歩さん大好きな方々に楽しんでもらえるような企画はとてもできないので、それぞれで自由にゆかり巡りやカラオケなどなど楽しんでもらって。そしてもしついでがあれば、どこか途中でお会いできるといいですね。

何よりこのイベントを見たのをきっかけに、それぞれが思い思いに小松さんに浸る時間が生まれてくれたら本望です。

目的としては、

  • ついでがなければなかなか会えない、小松未歩さん好きな方々との交流

目安としては、

  • 小松未歩さんが好き
  • 自分のフォロワーさん（またはそのお友達）

そんな方々を中心にイメージしています。

ツイッターで繋がる小松未歩さん好き同士、ご縁があればお会いしましょう。
今も普段、小松さんの曲を聴いてるよ〜というくらいで全然大丈夫です。

特にここで参加表明しなくても、興味が湧いたらツイッターででも直接ご連絡くださいませ。

ちなみに今回の神戸の旅、自分達（２名）のあしどりは次のような予定です。

03/24

横浜を出発（午前中）
→ 途中、どこか寄り道
→ 日没前頃に、大阪のどこかに到着By新幹線（こだまorひかり）
→ 大阪周辺でのんびりと
→ 大阪のどこかに宿泊

03/25

大阪のどこかを出発
→ 新神戸に午前中に到着 By 新幹線
→ 午前中は、のんびり気ままに異人館を散策
→ お昼は、南京中華街で食事（四川菜館or東天閣）
→ 食後はその足で、大丸神戸店と水上署前付近を散策
→ 午後4時を限度に神戸を出発

全体を通してゆるい予定になっているので、
こんな辺りでなら会えるかも〜みたいなことがあったらぜひご一報くださいね。
