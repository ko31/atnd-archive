TDD Boot Camp in Tokyo

テスト駆動開発(Test Driven Development)について、座学だけでなく、実習形式で手を動かして体得することを目的とするイベントです。

「実際にやってみる」これが大事!!

コンセプト

以下のような人たちがTDDに関するスキルを“学びあえる”場をイベントととして提供します。

  • テストコード書く意味がいまいち理解できていない
  • TDDに興味はあるが、なかなか実践できない
  • 自動テストを書けるスキルを身につけたい
  • テスティングフレームワークを十分に活用できていない感がある
  • 実際にTDDでソフトウェアを開発したい・見てみたい・体験したい
  • 他のエンジニアはどうやってテスト書いているのか見てみたい・気になる
  • 実際の現場で使えるTDDに関するヒントを見つけたい人
