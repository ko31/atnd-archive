description

トーストマスターズクラブとは、参加型のスピーチ学習会(例会)を通して、話し方やプレゼン、話を聴く力やマネジメント(リーダーシップ)能力を楽しみながら向上させることを目的とする、米国の教育NPOです。世界で12,500以上の支部(クラブ)があり、110カ国以上、26万人以上が参加。著名経営コンサルタントのトム・ピーターズ氏が著書『ブランド人になれ!』にて紹介しているほか、米国を中心に政治家や起業家の参加実績も数多くあります。18歳以上なら、経験や能力を問わずどなたでもご参加できます。

横浜オーシャントーストマスターズクラブは、水曜夜、英語と日本語の両方で開催。英語でのスピーチ練習により、グローバル時代に必須のコミュニケーション力を上達できます。さらに、母国語(日本語)のスピーチ練習により、論理的な話の構成の仕方をはじめ、言語を問わず人前で魅力的に話すための基礎体力を効率的に磨くことができます。

月2～3回水曜夜、横浜・関内付近の海沿いの公民館を中心に開催。例会後、任意参加の懇親会もあります。ご見学は無料、入会金4,000円、会費月1,000円で、近隣のビジネスパーソンや大学生を主な対象としています。

詳しくは、ホームページをご覧ください。
