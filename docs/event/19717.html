iOS開発者向け勉強会＠仙台 第二弾を開催します！

※(09/14 update!) 好評につき、定員を10→15に増やしてみました。

内容

先月開催したiPhone dev thk(東北・仙台) 勉強会！ 2011/08に引き続き、今月も開発者向けの勉強会を開催します！

勉強会と言いつつも、雑談用時間が1時間もあったり、本セッションもライブコーディングを見つつ話をしたりと、ユルめの会になっております。お気軽にご参加ください。

タイムテーブル

現時点で、タイムテーブルはおおむね以下のようになっております。(09/14 update!) @h_mori さんのセッションが決まりました！


  • 自己紹介・情報交換・雑談・世間話（1h）

  • Session1：Cocos2Dについて ライブコーディング by @tototti （1h-1.5h）

  • Session2：TapkuLibraryについて by @h_mori さん（0.5h – 1h）

  • 懇親会の相談 / 来月の勉強会について（10分）

参加される方へ

・MacBookなどをお持ちの方は、ご持参下さい。（コーディングを行う時間があるかもしれません）
・Pocket-Wifiにてネット環境を用意しますが、接続数に限りがあります。emobile等をお持ちの方は、ご持参下さい。

懇親会

勉強会終了後に、懇親会を実施予定です。場所などの詳細については、当日会場で決定致します。

後援

  • みやぎモバイルビジネス研究会
  • 東北デベロッパーズコミュニティ

勉強会の開催のご後援をいただいております。ありがとうございます。
