追記４（４／８）
今年のお花見は２４名の参加者という過去最大の盛り上がりでございました。
大勢の参加にもかかわらず怪我や行方不明者もなく桜も綺麗で笑いの絶えない大変たのしい一時でした。
また、大量のおみやげ日本酒は全て飲み干され、一度は参加者の養分になりましたこと嬉しく思います。
皆様おつかれさまでした＆ありがとうございます。ｍ（＿＿）ｍ

会計のご報告
おみやげや買い出しの費用は１０円単位繰り上げて合計し、幹事の裁量でほぼ公平に会費を頂きました（２０００円弱の余剰金が発生）。
余剰金は二次会、三次会に繰り越しました。ご了承くださいませ。

補足
二次会の会計は３０００円でしたので、ちんみるとメイドスキーは１０００円バックします。
最終的に−４００円という奇跡的にピッタリな収支でしたｗ

4月7日　13時より代々木公園でお花見やります！　アニソン好きなら誰でもカモン！

場所確保、南門の近く http://m.google.co.jp/u/m/dm73HM

追記３（4/5）
昼ごはんは事前にすませるか会場にお弁当をお持ちください。
当日は寒いので空きっ腹だと危険です！

寒いので「保温ポット」か「大きめの保温水筒」を持ってる人はお湯を入れて持ってきてくれると助かりまっす。

追記２（4/2）
会場は代々木公園に決定！　詳しい場所は後日連絡！

追記（3/26）

お花見といえばツマミ、おみやげ持ち込み大歓迎！
