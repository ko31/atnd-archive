概要

tokyo Linux Study は、Linux 運用に関する技術的な情報を交換する勉強会です。

第３回は HandlerSocket と Spider についての講演と座談会を行います。その後、軽食と缶ビールで懇親会も。

題して 「MySQL は HandlerSocket と Spider がビールの会」 です！

今回は講演される方がビールがあまりお好きでないとのことでしたので「烏龍茶の会」にしようかと思っていたのですが、参加される方にはやはりビールを振る舞いたい、という気持ちから、やっぱりビールの会にしました。

参加される方、懇親会で講演者にビールを勧めないでくださいねっ！

気軽に参加できる勉強会として実施しますので、興味のある方、都合の合う方はぜひご参加ください。

お待ちしてます！
