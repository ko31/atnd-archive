肉×技術＝熱い夜。

今後のキャリアについて悩んでる方、まさに転職を考えてる方にオススメ！
高級焼肉を食べながら、一流企業のCTOクラスとキャリアについて語り合う！

キャリアミートテックについて

アドベンチャーズがエンジニアに贈るスペシャルイベント、その名も「キャリアミートテック会」が、2015年7月2日（木）にIT企業の聖地・渋谷で開催されます。

今後のキャリアについて悩んでる方、まさに転職を考えてる方にオススメ！ 今イケてる技術会社のCTOや技術責任者の方と、技術やキャリアについて、美味しい焼肉を食べながら、美味しいお酒を飲みながら、ざっくばらんに語り合いましょう(^^)/

キャリアミートテックの特徴について

  • お肉は最高ランクの黒毛和牛（A5）をご用意（しかも無料!!）
  • 今ホットな会社が参加
  • 各企業の技術のトップが参加
  • とにかく超ゆる〜く、技術者同士の交流ができる！

すぐにでなくても、転職を将来的に考えている方でも大歓迎

「もし転職した場合、自分がどんな仕事をするのか知りたい」

「どんな開発環境や体制なのか知りたい」

「働いている人たちや社内の雰囲気が知りたい」など

とにかくこのイベントの趣旨に共感した方は是非ご参加下さい！

※応募多数の場合は抽選とさせて頂きます。
