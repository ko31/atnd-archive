デザイナさん、コーダーさん、プログラマさん、企画さんも、そろそろ「HTML5」覚えないと…

次世代HTML5は2014年までに勧告される予定！まもなくです。そろそろ覚えないと遅れをとってしまいます！

ワークショップ内容

このワークショップは、HTML5を学びます。
制作経験のないデザイナやコーディング担当の方々、初心者でもHTML5でのサイト制作に興味をお持ちでしたら、ご理解いただけるような入門講座です。（ただし、座学ではありません。ガリガリソースを書いていきます。）

・HTML5　SEOを意識した要素の書き方
・CSS3　デザインの見栄えを良くするスタイルシート
・JavaScript　WEBサイトを動的なWEBアプリへ進化
・canvas　ゲームに最適なキャンバス

用意するもの

・ノートPC：
　　・safari または chrome をインストール
　　・UTF-8の文字コードを扱えるテキストエディタをインストール
・参加費：5,000円(リピータの方は領収書などご持参くだされば1,000円割引します。)
