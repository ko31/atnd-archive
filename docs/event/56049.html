アベノミクスのまやかしに騙されないで、 著名な投資家や富裕層のスキームを真似しませんか？

海外の著名な投資家は長期投資が基本です。 短期ではなく、長期的な視点に立てば、既に成功している著名な投資家の真似をしたらいいとは思いませんか？ 資産を確実に増やしたり、形成するには、著名な投資家や富裕層のスキームを真似するのが1番なのです。 大企業や著名な投資家は未来の利益やリターンを考え、新興国に投資をスタートしている。 先進国と新興国の最大の違いは、これから消費や国を担う若者の人口が多いことやその若者の多くが、日本やアメリカのように発展したい！

アベノミクスのデフレからインフレの意味を知っていますか？

自分の勉強会でも最近は、インフレの罠を教えていますが、多くの人はインフレの意味や長期的な怖さや罠を全く理解していない事に驚きます。

DCF（ディスカウントキャッシュフロー）は現在と未来のお金の価値は違う（基本的に将来のお金の価値は下がる）と いう思想で未来の資産価値を現在の資産価値に割り戻して計算する手法

学資保険で一番儲かりそうなイメージがあるのはソニー生命。今の利率だと18年振り込んで114.1%の返金らしいので、18年間で14.1%儲かる

日銀と政府は物価上昇率を2%に設定しているので、仮に18年間2%のインフレが 続くと今の100万円は18年後の142.8万円と同じ 言い換えれば、今の100万円を142.8万円にできなければ損失

なので、仮に一括で100万円振り込んで、ソニー生命で18年後に114.1万円に したところで、実質的には損失を出しているという実に面白い

別の見方をすれば、インフレ2%下で18年後に114.1万円になる100万円の投資は 現在価値で割り戻してあげれば、80万円になる

すなわち、100万円を一括投資すれば自動的に20万円の損失。このからくりをほとんどの人は知らない

DCFの概念がわかっている人は終身保険などにも入らない 。終身保険は加入時に将来受け取る保証金額を決めるので、DCFがわかっている人は誰も入らない 。

入るのは保険会社の罠にかかった無知の人だけ 。 政府と日銀は全力を持ってインフレを起こそうとしている、世界中の 中央銀行も金融緩和を継続する 。お金がばら撒かれ続けるので、普通に考えればインフレになる 。

これを知っても0.02％の利子しかつかない銀行預金で、お金を減らしたい人は減らしてください！
