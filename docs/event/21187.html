description

ある中学校では、たった5色の絵の具だけで絵を描くことを教えているそうです。
子ども：「先生、水道の蛇口を書きたいのですが、銀色がありません。」
先生：「蛇口をよく見て色を分解してみなさい。蛇口はいろいろな色で構成されている。」
12色の絵の具だと、土は黄土色、森は緑と固定概念かに縛られ、対象が持つ複雑な表情に気が
付かない。これは先生の言葉。
たった5色で出したい色を作る努力を重ねることで、創作に熱中し自分の表現力に気が付くそうです。
この記事を見て、「ああっ」と腑に落ちるものがありました。全ては相似形です。
この話を新しいまちづくりの考え方に置き換えてみましょう。
大都会のカタログには何でも12色あって、わが町のカタログには5 色しかない気がするわけです。
混ぜ方を知らなければ、高い金を出して６色目、７色目を東京から買い取って来るしかない。
メディアを通じて、「東京では春の新色13色目が登場しました！」と言われると、どうしてもそれが欲しく
なってきます。
わが町にはいつも東京より選択肢が少ない気がします。
これが世の中の縮図のような気がしてきました。
でも、「混ぜ方」を学べば、実は数万色を自在に操れます。出来合いの色を100種類持っていて混ぜ
方を知らないよりも、５色しか持っていないけど、混ぜ方を知っているほうがいいじゃないか！という訳です。
これが、地域の自立した生き方ではないでしょうか。
山口覚さんのブログより　http://blog.livedoor.jp/yamaguchi1000gen/

今回のOff Cafeでは、ＮＰＯ法人地域交流センター津屋崎ブランチ（http://1000gen.com/）代表の山口覚さんをゲストにお招きします。
山口さんは、地域活性化には都会へ流出した若者が戻ってくることこそが一番大切な事だと確信。13年間の東京生活から福岡に戻り、今では津屋崎を活動拠点に「新しいまちづくり」に取り組んでいらっしゃいます。
さて、そんな山口さんに聞いてみたいことはたくさん！津屋崎ブランチの取り組みだけではなく、まちづくりに対する想いにも、直に触れてみたいと思います。
