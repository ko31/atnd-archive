イベント概要

皆さんハースストーン楽しんでいますか？？マンモス年はあと一つ拡張を残していますが2017年は間もなく終わりますね！( ´∀｀ )

さて、一年の最後といえば忘年会！お酒を飲んでハースストーンを語り合いながら一年を振り返りましょう！！！！

ハースストーン忘年会　in 東京　ここに開催します！

詳細

新拡張のリリース予想日の翌日開催ですので新弾についてもプレイしながら語り合いましょう！

参加条件

Hearthstoneを楽しんでいる人！プレイサーバーは問いません！(主催はNAとAsiaに対応できますがEUは、、、笑)

開催日時および参加費

2017年12/9(土)19:00～22:00

参加費：3500円 ここでいう参加費がそのまま飲食代金になります

場所

居酒屋本舗 虎の子　新宿・歌舞伎町

東京都新宿区歌舞伎町１－１７－１３　新宿ピックペックビル４Ｆ

アクセス：JR新宿駅から徒歩5分、または西武新宿駅から徒歩3分

集合場所および時間：　18時50分にドン・キホーテ新宿歌舞伎町店前に集合です！土曜日なので混雑が予想されますが分かりやすい目印掲げる予定です！

タイムテーブル

タイムテーブル的なものはありません！！従ってトーナメントをはじめとした企画等も考えていません！笑

はじめに簡単な自己紹介の時間だけ設けますが、後は各自で勝手に料理とお酒に舌鼓を打ちながらハースストーン片手に盛り上がっちゃいましょう！！！

炉端の集いに関して

本イベントはBlizzard社の炉端の集いに申請中です。おそらく承認され、看板付き酒場であるのでイベント中にてウォーロックの新スキン：ネムシ―は取得可能です。しかしバグなどにより取得できない可能性があります。あくまでネムシ―はついでと考えておいてください。忘年会を楽しみましょう！！

諸注意

  • 会場は居酒屋であるためもちろんハースストーンをプレイできる機器はありません。各自端末をご用意ください。

  • wi-fiもありません。ご了承ください。

  • 会場に充電場所はないので、予備バッテリーを持ってくることを推奨します

  • おそらく無いと思いますが盗難・紛失に関しましては主催は責任を一切負いません。貴重品の管理は自己責任でお願いします。

  • 遅刻/当日の欠席される場合は分かり次第すぐに主催(@HS_roguelover)に連絡をお願いします

  • 宗教・マルチ等の勧誘行為は一切禁止します。

  • 居酒屋での開催であるため未成年の参加はお断りさせていただきます。申し訳ありません。
