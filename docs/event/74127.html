[講演＆ワークショップ]　病の声をきく　病から「感じる」ものがたり

病や痛み、悩みにはそれぞれのプロセスやメッセージ、そこにある感情などの「ものがたり」がある。
そのものがたりを感じとりながら、一つ一つを癒していく。
「自分自身が主人公」
病からの声を受け取り、望む未来へと進む為のサポートを・・・。

【講師】ホリステックセラピーみちあかり代表　住田まり子
　　　　　　　　　　　ヒーリングセラピスト　もりふみか
