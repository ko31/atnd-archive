【令和元年記念　全員参加型パーティ交流会でビジネスの和を広げよう！】

令和になって、今年もやります！
年に一度のお祭り騒ぎ。
踊るあほうに歌うあほう、同じあほなら、いっしょに騒いで笑って泣いて飲んで食べて！
いっしょに盛り上げて、もう何もこわくない！

盛り上げたパワーで、ビジネスも波に乗せちゃいましょう！

パーティーの内容

１）自己紹介という名のアピールタイム
Studio NU-VU (スタジオヌーブ)という場所に、今日あなたが来てるよ、というアピールタイムから始まります。
ビジネスにつなげるためのパーティーです。
自分の仕事は誰のどんな悩み解決になるか、具体的にアピールしましょう。
気になった人には、話しかけて名刺交換して、仲良くなって帰ってくださいね。

パフォーマンスで会場の注目を集めて、人気者になりましょう。

プレゼン、歌、踊り、そのほかのパフォーマンスで、始めからテンションMAXです。
ピアノとドラムがあります。自由に使ってください。

※パフォーマンスなんてできません、とあきらめずにご連絡ください。いっしょに考えますよ。
※ピアノの演奏をして欲しい、という方は事前にご連絡ください。

２）ディスコダンスタイム
ブレイクダンスで全米1位、世界第2位のKAZUさんとダンスを踊りましょう。
運動神経が悪い人は、実はいません。
言葉の呪縛を解き放ち、体の仕組みがわかったら、あなたも今日からディスコクイーン。

３）似顔絵プレゼント
OLGATIO GALLERY　代表　宙（SORA）さんにあなたの似顔絵を描いてもらえます。
その場でプレゼント。
宙（SOAR）さんは、テレビ番組に出演したり、
藤田ニコルさんや関西コレクションのモデルさん、堀ちえみさんといった方の似顔絵も描いてます。
OLGATIO GALLERYのサイトはこちら
http://olgatio.ninja-web.net/
