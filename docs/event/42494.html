アキナイトとは？

アキナイトは京都出町柳商店街の一角で開催されているビジネス勉強会です。
ビジネスのきほんのきほんを学ぶこと。そして、実際の企業のビジネスモデルを読み解くことに挑戦します。このワークショップは初心者の人でも楽しく学べるようになっていますので、お気軽にご参加ください:)

Ⅰ. アイスブレイク(18:00-18:30)

簡単な自己紹介ワークショップをします。
初めての参加者でもなかよくなれます。
イベントが終わってからもつながれるような仲間を見つけましょう！

Ⅱ. 今朝の朝ごはんから考えるマーケティングのきほん(18:30-20:00)

ビジネスのきほんはマーケティングを学ぶことから始まります。
今朝の朝ごはんはなにを食べましたか？パン？それともご飯？？
「なぜそれを買ったのか」「なぜその店で買ったのか」を考えることからマーケティングは始まります。

ポストイットを使いながらアイデアを膨らましながら考えていきます。参加者同士で意見を出しあいながら進めていくので、難しいと感じずに楽しくビジネスを勉強できます。

Ⅲ. "スカパー"の新しいビジネスモデルを見抜け！(20:00-21:00)

さて、マーケティングのきほんを学んだあとは、いよいよ実際の企業のビジネスモデルを読み解いてみます。
今週のビジネスモデル読み解きは”テレビ業界”です。テレビはいまだに多くの人が利用していますが、近年、若者のTV離れをよく聞きますね。各企業がどんな生き残りをかけてどんなビジネスが展開しているか、見抜きましょう！

また、使用するビジネスモデルの読み解きのツールはピクト図解です。
様々なビジネスの形を誰もが分かるレベルにシンプルに図式化することに優れています。基本の解説と演習から行うので、初めての方もご安心してご参加ください。

参考: ピクト図解とは？

①ピクト図解の基本的な考え方レクチャー
②新聞記事を使ったビジネスモデル読み解き
③答え合わせと解説
