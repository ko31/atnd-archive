＜このツアーは香港で開催されます＞

ルシャスデリシャスでは2005年から、より香港の食材や食べ物、食にまつわるさまざまなことをお伝えしたいと実際に市場や街を歩き、レストランでの食事を体験しながら楽しく香港の食文化や食習慣、食材、調味料等を学ぶフードカルチャーツアーを開催しています。ありきたりな観光ツアーや一般ガイドのツアーとは違い櫻井景子（香港在住18年、香港で料理教室を主宰し15年の食の専門家）がガイドする「食」に特化したスペシャルなフード＆カルチャーツアーです。
​
　香港タウンガスクッキングセンターでのお料理レッスンと同時参加でより香港の食材や調味料、料理に理解が深まります！
　　　　　　　　　　　　　　　　　　　​　　　　　　　　　　　　　
ありきたりの観光ツアーではなく、まるで香港に住んでいるかのように香港の市場を歩いてみませんか？
​
＜旅行者の方も＞
香港の食を深く知りたい方、香港ならではの食材や調味料の紹介や説明、購入先の情報、購入アドバイス、日本へ帰ってからの代用品の紹介など、幅広く知ることが出来ます。

＜香港在住者の方も＞
香港在住の方には普段のお買い物に役立ち、いつも通っている道がまるで違う場所のように感じるかもしれません。表面だけの街歩きでは体験出来ない内容で、きっと身体全体で香港を感じていただけると思います！
​
　また大人数でいただく美味しいランチも魅力です。新しい料理との出会いもお楽しみください。
​
ルシャスのフードツアーに参加すると・・・
香港が初めての方も香港通になれる！
香港が2回目以降の方はまるで常連のように歩ける！
香港在住者、ヘビーリピーターの方は見慣れた街で新しい発見

＜こんな内容です＞

上環乾物1日ツアー

■見どころ

香港島の歴史ある街・上環を歩くツアーです。
1.　香港最大の乾物街にあるおすすめの乾物屋へご案内し、乾物の紹介、買い方、使い方　　を説明いたします。乾物食材や薬膳に興味の有る方にぴったり！
2.　香港らしいお土産が買い揃う乾物店、調理器具屋・乾麺屋・中国茶荘・ガラス食器店・チャイニーズのお菓子屋、お薦めピータン屋など巡ります。
3.　ランチに香港の有名店で化学調味料無添加の飲茶コース料理をいただきます。
4.　日本語通訳つきで漢方医の診察が受けられます。
5.　上環マップがつきますので、その他お連れ出来なかったおすすめの店なども合わせてご紹介します。ツアー解散後や他の日にも自分の足で歩くことが出来ます。
