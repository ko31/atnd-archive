「第1回文学フリマ京都」に懇親会開催のお知らせです。

出店者、一般来場者、事務局スタッフも一緒になっての親睦会です。
これからの新たな展開へのきっかけづくりの一助になればと思っています。

懇親会々場があるのは「三条木屋町」。路上アーティストがそこここで音楽を奏で、森鴎外の「高瀬舟」に描かれた、有為転変の人生を象徴するような高瀬川が流れています。その傍には現代建築の中でも最も前衛的といわれる安藤忠雄氏の設計になる「TIME’S Ⅰ&Ⅱ」も佇んでいます。京都ならではの夜の観光も同時に楽しんで頂けるものと思います。

「文学フリマ東京」に倣い「行商タイム」も準備しています。
お食事を共にしつつ、会場ではゆっくり語り合えなかったことも、この場で！

この貴重な機会、ぜひ参加をご検討ください。
