〜都内初？！中高生が企画・運営するプレゼンのイベント！〜

「日本人はプレゼン下手」
そう言われてしまう原因は、教育にある。
そんな現状を打破するために立ち上がったのが、私たちSpeak to Start。

4/5に Speak to Start の教育プログラムを初めて受けた、中高生9名のプレゼン大会が行われます！

《Speak to Startの教育プログラムとは？》
海外の教育を受けたことがある帰国子女の女子高生が、Show and Tell などの海外の授業を元に、現在の中高生にピッタリなものを考えました。
これがSpeak to Start独自の教育プログラムです。
今回の大会もそのプログラムの一部です。

《今大会の見どころ》
新たなスタイルの教育を受けた中高生による、奇跡のプレゼン。
そして大人もたっぷり学べるプレゼンのプロの講演もあります！
さらに、3人の帰国子女の女子高生が、日本の教育について語るセミナーも設けています。

Speak to Start 決勝戦、4/5(日)。
日本のプレゼン教育の新たな幕開け、お見逃しなく。
