日本MySQLユーザ会(MyNA)のイベントです。MySQL について語りながらユーザどうしで親交を深め、盛り上がりましょう。
今回は 4/12-15 に米国サンタクララで MySQL Conference が開催された直後ということもあり、もしかしたら Conference に参加された方々からのおみやげ話なども聞けるかもしれません。

■参加予定の皆さんへお願い：
・開催内容に変更がある場合はこのページで案内します。お手数ですが本ページを適宜ご確認ください
・費用のかかるイベントのため一旦4/22(木)15:00 の人数で準備の規模を検討したいと考えています。参加表明お悩みの方は上記日時までに表明いただけると幸いです。

■開催概要
・日時：　2010/04/24(土) 15:00～。19時頃中締め。（開場 14:30）
・場所：　グリー株式会社様　5F セミナールーム
　　　　　　東京都港区六本木4-1-4 黒崎ビル (1Fがフォルクスワーゲン)
　　　　　　正面入口ではなく裏口のエレベータから 5F に上がってください
・参加費： 3,000円
・15時～： セミナー形式（呑みながら気楽にどうぞ；講師も呑んでます:-)）
・16時頃～： 懇親会（呑み喰いしながら楽しく交流を深めましょう）
・19時頃：　中締め
-————————————————-
　
■セミナー内容
（１）「Drizzleの "いま"」（Drizzle Project： 前坂 徹さん(株式会社ミクシィ)）
　MySQLの軽量化フォークである Drizzle(ドリズル)。そのコミュニティで精力的に活動をされている前坂さんに、Drizzleの特徴や現状についてお話いただきます。
特に Drizzle のストレージエンジン回りのお話に力が入ると予測されます。
4/12～15 に米国で開催された MySQL Conference で講演された内容をぎゅっと凝縮して日本語でお話しいただく予定です。
　
　
（２）「とみたまさひろに訊く『MySQL』」(日本MySQLユーザ会代表：とみたまさひろさん)
　MySQLとのつきあい方、活用の仕方は人によって様々です。
MySQLユーザ会設立前からのコミュニティメンバーであり、MySQLの最初の日本語化も行った、とみたさんに『MySQL』とのお付き合いの日々や最近の関心事項などを伺います。
最近では滅多に聞けなくなったユーザ会設立時やMySQL黎明期の話も聞けるかも？
-—————————————-
　

■補足
　最初からビールや軽いおつまみなどを用意します。前半は呑みながら「ちょっと詳しい人」の話をみんなでわいわい聞くという趣向で、後半はお食事を加えて参加者どうし懇親していただくという趣向となっています。
　肩に力の入らないゆるやかな会にしたいと思っていますので、MySQLの話題にたくさん触れたい方のご参加をお待ちしています。
