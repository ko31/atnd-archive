『インフラ寺子屋』ライターのAkira Tsumuraです。

ブログ発のイベント第1回として、インフラエンジニアがまったり雑談することのできる交流会「お寺のお茶会」を開催します。

『インフラ寺子屋』とは、新米インフラエンジニアや、キャリアアップを目指すエンジニアを助けるためのブログ、及び勉強会です。

お茶会では、ブログライターのAkira Tsumuraとnullpopopoが聞き手となり、参加された方のお話をお伺いしたり、アドバイスをする場です。また、参加者同士で交流して頂いても構いません。

また、ブログを書くにあたり、FacebookやTwitterで、拾える限りのご意見・ご感想は拾っていっていますが、なかなか『声』が集まらないのが現状です。
今後のブログや勉強会へのご意見や、ご感想、インフラ雑談など、お話頂けると幸いです。

そんな感じで、ふらっと遊びに来て頂けると幸いです。
