“そもそもココ飲みってナニ？”

「今日、ココで飲みます」

元来、集まって飲むのが好きな僕ら…だけど事業はSES。
それぞれのお客様先で日々の仕事をしています。

自分が選んで入社した会社と違う文化やルールの中で
毎日、仕事をするわけです。

だから不安やストレス、時に不条理を受けることもあります。

だから悩みや困りごと、話せる相手も機会も大事なのです。

それで集まる機会や動機を生むためにも
「今日、ココで飲みますっ」って
社内チャットシステムに、飲む場所と時間を投稿すれば
その飲み会の飲み代は会社負担にするという福利厚生を創った。

それが「ココ飲み」誕生の想いと背景です。
(代表工藤が最初に創立した今は別の会社で生まれた福利厚生)


グローバルセンスでも“ココ飲み”は続けたい

だけど…
『スタートアップの我々は経費はかけられない』

そもそも…
『”居酒屋でタダ酒飲もうぜ“が目的じゃなくて、仲間と会うきっかけ作りがココ飲み。給与前だろうが出費がきつい時期だろうが、気楽に呼びかけ合うための経費だったのだからさ…』

だったら…
『経費は減らして機会は減らさないようにしよう』

ということで…
『自社オフィスで』
『料理は手作りで、飲み物はディスカウントで』

そして…
今のホームパーティ形式のココ飲みは誕生しました。

安室奈美恵さんやポケモンで日本が賑わい、
