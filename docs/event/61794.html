“ウルフ村田”緊急来福！資産運用戦略を語る

description

【ウルフ村田が福岡にやってくる！】
自らを“株芸人”と揶揄し、いまやテレビ雑誌等に引っ張りダコの“ウルフ村田”こと村田美夏氏が福岡にやってきます。
2013年には1年で2億円もの利益を生み出したという村田氏。
自らの“株芸人”ぶりを、ありのままのトレードをご本人の口から語ってもらいます。

お相手は、1年間で70万円から3億円を作った男として、30代にして伝説になった男、午堂登紀雄氏。いまや資産は5億円を超えると言います。
東京をはじめ、大阪、名古屋、仙台、札幌と、サンワード貿易のセミナーでは全国各地を回ってきました。
不動産投資を軸に様々な投資を語る午堂氏。
大人気のセミナー講師が福岡上陸です！

【株式で2億円 VS　不動産で資産5億円】
東京大学を主席卒業された村田氏。
銀行員から始まる数々の異色経歴を経て、現在ではエンジェル投資家としての一面もお持ちです。株式をはじめ多くの投資ツールを駆使する村田氏がご自身の投資戦略を語ります。

かたや、20代までは終電まで仕事に追われる「普通のサラリーマン」だった午堂氏。
普通のサラリーマンがいったいどうやって億単位の資産を作っていったのか？
午堂氏独自の視点とは？　
村田氏と午堂氏の熱いトークセッションにご期待ください。

【大投資家の驚きの接点とは？】
一見、なんの接点もない二人の講師。
実は「コモディティ投資」という共通項があります。

講師陣のコモディティ投資トレード大公開！？ 「コモディティ投資」「FX」「海外・国内不動産」などさまざまな投資で資産形成に成功した午堂氏。 彼が「攻めの投資」と位置づけるコモディティ投資の内容とは！ 一方、ずいぶん古くから「コモディティ投資」も手掛けていたという村田氏。普段から「人生＝ボラティリティ」と語る村田氏の赤裸々なトレード実態が暴かれる？ 今回は実際に村田氏、午堂氏のコモディティ投資を担当するサンワード貿易人気ディーラー山田高志が登壇！

“お2人の共通項”山田高志が、これまでのリアルな失敗や成功例を踏まえて、経験者のここでしか聞けない話を繰り広げます。

大きく動く市場でこれから使える実践投資の参考にしてください。

じかに成功者たちのエッセンスに触れてください！！

さらに2人が今年注目する投資先を参加者だけに限定公開！！

福岡での開催は機会が限られます。

この機会をお見逃しなく！！
