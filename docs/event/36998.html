description

デベロッパの未来をチラ見するカンファレンス！！

ステークホルダーの多様化に伴う要件の変化、短納期化、予算縮小などシステム開発を取り巻く環境は大きく変化しています。
オンプレミスからクラウドにシステム基盤を移行するSIer、OSSや公開APIを積極的に取り入れシステム開発するソフトハウスなども増えてきました。
こうしたアーキテクチャの変遷期はエンジニアにとってチャンスであり、リスクでもあるはずです。

いかにして今までの経験を活かしつつ、市場価値の高い技術を習得するか？

この不変のテーマに対して、システム開発の先駆者から現場の変化、エンジニアの役割の変化を改めて学び、明日からのシステム開発を変える第一歩とするイベントを開催します！
