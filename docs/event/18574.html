description

　「100万回遊べるカードゲーム」として世界で人気沸騰中のカードゲーム「ドミニオン」の遊び方を学び、実際に体験しましょう。中学生以上から大人まで、幅広くどなたでもご体験頂けます。参加は無料です。お気軽にお申し込みください。

【ドミニオンとは】
　2008年にアメリカで発売され、瞬く間に世界中で大ヒットし、各国の賞を総なめにしたカードゲームです。2人～4人で同時に遊ぶことができ、カードの組み合わせに応じて「戦略」を組み立て、一番多くの「勝利点カード」を獲得したプレイヤーが勝利します。しかし、勝つ為には「運」も必要です。家族でも、友人同士でもお勧めです。

【世界の面白ゲーム体験講習会について】
　この「世界の面白ゲーム体験講習会」では、世界で人気のボードゲーム・カードゲームを毎回紹介し、参加者同士で遊ぶことでより深く、そのゲームについて学んで頂きます。日本ではまだまだ一般的でない、主に大人向けの海外のボードゲーム・カードゲームの普及を目的としています。第一回のゲームは大人気の「ドミニオン」です。

■参加費：無料
■対象：
　子供の部：中学生～高校生
　（カードを乱暴に扱わない約束ができるなら、ゲーム好きの小学生５、６年生も対象とします）
　大人の部：世界の面白ゲームに興味のある大学生および社会人
　　　例）友人と休日に遊ぶゲームを探している人
　　　例）家族でできるゲームを探している人
　　　例）コンピューターゲームに飽きてしまった人
■駐車場 4台
　【注意】上記のGoogle Mapsの地図は場所が間違っています。地図上でもっと右の方に、「向陽会館」があります。「URL」のリンクを開くと表示される地図の中央がそれです。
■その他
・ドミニオンは２セット（最大８人まで）用意しております。
・８人を超える参加者がいた場合、交代までの間、他のボードゲーム・カードゲームを紹介しますので、それでお遊びください。
・昼食やお飲み物は各自ご用意ください（すぐ近くにサークルKがあります）。
