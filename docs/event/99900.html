■横浜インベスターズについて

「横浜インベスターズ」は、横浜で開催している不動産投資の勉強会です。

  • 不動産投資をこれから始めようとしている方
  • 既に不動産投資をしている方
  • 専業大家さん
  • サラリーマン大家さん
    • など、不動産投資に興味を持っている方なら誰でも参加可能です。

■開催形式と注意事項

＜タイムスケジュール＞

1.開始時間までに集合

横浜駅近辺の会議室で開催することが多いです。
参加の際は直接会場までお越しください。

2.自己紹介

参加者全員が集まった時点で、最初に軽く自己紹介をしていきます。
お名前・職業・趣味・ご自身の不動産投資のスタイル・不動産投資を始めた経緯など、1～3分くらいで気軽にお話し下さい。

3.最近の「インプット（INPUT）」の紹介

・最近読んだ不動産投資関連の書籍
・不動産ニュースや市況
・勉強中の資格
など、ここ1か月程度の「インプット」について、お一人ずつご自由にお話しください。
あなたが得た気づきや今後の投資に取り入れたい点など、共有して頂けるとありがたいです。

4.最近の「アウトプット（OUTPUT）」の紹介

・最近内見した物件
・最近取得した物件
・入居が決まった物件
・売却した物件
・最近実施したリフォーム
など、ここ1か月程度の「アウトプット」＝「成果」について、お一人ずつご自由にお話しください。
あなたが成し遂げたことを全員で共有して、喜びを倍増させましょう。

5.お悩み相談＆フリートーク

最後に、不動産投資で悩んでいることについてみんなで共有しましょう。
悩みが解決するかどうかは保証できかねます（ｗ）が、話すことで気分が軽くなったり、解決のヒントを得たりすることはあるかもしれません。

6.会費支払いと解散

最後に会費を集めて解散となります。
その場で仲良くなった方とは連絡先交換などもご自由にどうぞ！

＜開催人数＞

3～10名

＜参加費＞

500円 ※会場費の分担をお願いします。

＜注意事項＞

  1. 会議室内および施設全館禁煙です。
  2. 会議室内での飲食は可能です。
  3. 施設内にはゴミ箱がありません。ゴミはお持ち帰りください。
  4. ネットワークビジネスや宗教の勧誘、また他の参加者への迷惑行為はご遠慮ください。読書会とは関係のないビジネスへ勧誘したりするなどの目的の参加は認めておりません。参加された方の中でそのような勧誘行為を受けた場合はすぐに主催者までご連絡下さい。
  5. 体調不良などにより当日キャンセルされる方は、開始日の前日までにご連絡ください。

最後に

不動産投資や大家業は、ひとりで自由気ままに行える仕事ですが、その分孤独も伴います。
似たような志を持った者同士、鼓舞しあったり、勇気づけあえるような場を築くことができたら幸いです。
