過去の開催履歴とこれからの開催予定はこちら


PHPからメールを送信するプログラムの書き方を勉強します

当勉強会は、"PHP初心者" と “PHP『超』初心者” を分けています。

こちらは初心者向けの勉強会になります。

こちらのページを参考にどちらから始めるか判断して下さい。

PHP『 超 』初心者勉強会は、プログラムが初めての方を対象にしています。

PHP初心者勉強会は、ある程度プログラムができるようになった方を対象としています。

勉強会の進め方は、プロジェクターに映した画面を見ながら、一緒に操作を行って進めて行きます。

内容

  1. インターネットのメールの仕組み
  2. php.iniの編集
  3. メール送信プログラムの書き方

※ 今回の勉強会は『超』初心者には難しい内容です。

XAMPPをインストールしてください

講義にはXAMPPというソフトウェアを使います。

これはパソコン上（Windows,Mac）で、Webサーバーを実行するプログラムです。

（基本的にインターネットで使用されるWebサーバーと同一のプログラムです）


Windows版
http://www.apachefriends.org/jp/xampp-windows.html

Mac OS X版
http://www.apachefriends.org/jp/xampp-macosx.html

XAMPPのインストール方法は、以下のURLをご覧下さい。
http://www.db.is.kyushu-u.ac.jp/rinkou/mysql/xamppinstall.html

インストール後、http://localhost/にアクセスして、インストールが成功したか確認して下さい。

インストールに失敗した方は、phpbeginners@gmail.comにご相談下さい。
