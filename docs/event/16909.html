先日ドイツから取り寄せた頼まれ品のボードゲームの受け渡しついでに、ボードゲーム会を開催します。

どなたでも、ご自由に。初心者の方も歓迎です。(^^)

。。(o_ _)o 3ﾚﾝﾊﾟｲ…ﾓｳﾀﾞﾒﾎﾟ ＼(・∀・ ) ﾖｼﾖｼ

ちょっと頭を使わないといけないけど、勝つには運も必要・・・そんな海外のいろんなボードゲームを楽しく遊びましょう。

ドイツ年間ゲーム大賞を受賞した「ドミニオン」や「ディクシット」、また、今年のゲーム大賞ノミネート作品の「禁断の島」や「ストラスブール」など取り揃えています。

難しいゲームは苦手、という方には、選んだレシピどおりに鍋の中へスパイスをうまく入れるゲーム「ア・ラ・カルト」や、テトリスみたいなパズルをみんなで一斉に解くゲーム「ウボンゴ」などの軽いゲームもありますよ。(^^)

短い時間ですが、集まったメンバーで気軽に楽しみましょう。
（いちおう、25時ぐらいには一旦解散予定ですが、希望者がいれば延長も可能です）
