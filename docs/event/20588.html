description

新島は、当時、未曾有の国難であった黒船来航に揺れる日本にあって、藩からの支援もなく、無一文で 国禁を侵し脱国、 何の寄る辺もないまま渡米し、人々の良心だけを頼りに数々の奇跡を起こし、ラットランド教会にて5000ドルの献金を受け帰国、智徳教育普及のため、日本で初めての民間による大学設立に人生を投じました。

日本はいま、戦後最大の国難に直面し、国家そのものが激しく傷ついたまま、復興への舵取りを見定められず、国政は混迷を極めています。
想定外の事態に遭遇し、国家全体が暗澹とした空気に包まれる中、唯一、光明を見いだせたのは、忘れ去られてしまったかに見えた日本人が元来持っている道徳美を、この悲惨な震災によって目にする事が出来たことです。日本のモラルの高さ、日本的道徳美が世界で絶賛されたことは、絶望的な状況にあって民族としての誇りと自負を喚起させ、国民一人一人が力の限りを尽くして、復旧・復興に向かって結束を固めようとする原動力となりました。

良心の教育によって日本を救おうとした新島の生き様は、復興に向けて生きる人たちに、勇気と多くの教訓を与えてくれると信じ、現在映画化の企画を進めてます。映画化にあたっては広く同志社に関わる方々のご意見も伺いたいと思っています。

また新島襄生誕170周年にあたる2013年は、NHK大河ドラマで新島の妻・八重が取り上げられることもあり、新島精神、同志社スピリッツを世界に発信するまたとないチャンスだと思っています。

同志社大学ホームカミングデーが皆様との幅広い意見交換、情報交流の場にできればと思っていますので、どうかよろしくお願い致します！
