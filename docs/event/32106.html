description

今回は、他の有名アプリディベロッパーにお話頂くのではなく、アプリディベロッパー以外のプレイヤーから、話をしてもらうセミナーとなります。
広告を出している側や、それを運用する代理店、広告配信の事業者という立場からみる、いつもとは一味違ったマネタイズセミナーです。

席が限られておりますので、今回のみご参加はアプリデベロッパー様に限らさせて頂きます。
（広告代理店、広告配信サービス運営会社の皆様申し訳ございません。）

各スピーカー様に約20分ほどお話頂いた後5分ほどと、セミナーの最後にまとめての質疑応答の時間を設けております。
セミナー終了後に名刺交換会の時間を設けていますので、交流の場としてご活用ください。

【ゲストスピーカー＆プログラム】

18:40頃 開場

19:00 開演のご挨拶

アドイノベーション株式会社 代表取締役 石森博光

19:05～19:25『あなたのアプリのクリック単価が安い理由』（20分）

アドイノベーション株式会社 CMO 細川竜馬

広告代理店でアドネットワークの運用担当、広告主側での出稿担当の経験に基づき、クリック単価の決まる仕組みや、その考え方をお話します。広告を出す側の考え方を知ることで、収益の上げやすい広告を考えることができます。

19:30～19:50 『nendから見るアプリ広告の現状とこれから』（20分）

株式会社ファンコミュニケーションズ 二宮様

日本最大級純国産のスマホ向けアドネットワーク「nend」の事業責任者である二宮様に、アドネットワーク運営側からみたアプリ広告についての状況をお話頂きます。

19:55～20:15 『リワード広告によるアプリのマネタイズ』（20分）

タップジョイ・ジャパン株式会社 Publisher事業開発部長 村上様

グローバル規模のリワード広告アドネットワークを提供するTapjoy様より、リワード広告を活用したアプリのマネタイズ方法についてお話を頂きます。アプリへのリワード広告実装例、成功事例などをご紹介いたします。

20:20～20:40 『1impsの価値をどう高めるか』（20分）

モーションビート株式会社 メディアプラットフォーム事業部 事業部長　植様

インプレッション毎の収益性を上げる為にアドネットワークやRTB広告を使いどのようにメディア収益の最大化を図るかの方法についてお話頂きます。

20:45～20:55 質疑応答

21:00～21:30 名刺交換
