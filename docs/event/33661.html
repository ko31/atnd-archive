2012年11月5日（月）から 2013年2月20日（水）までの毎週月曜・水曜の19:00〜22:00 に、渋谷駅新南口より徒歩30秒の LightningSpot にて「社会人のためのWebデザインスクール」を開きます。

内容は以下の通りです。

  • Webデザイン入門
  • Photoshop によるWebデザイン制作
  • Illustrator によるベジェ曲線、シェイプの作り方
  • Dreamweaver による XHTML + CSS コーディング

このWebデザインスクールの目的は、「Web制作の第一線で活躍するためのスキルを身につける」ことにあります。

単に Photoshop や Dreamweaver の操作方法を覚えるだけではなく

  • 自分でオリジナルのデザインを考えるためのデザイン概論
  • 美しいデザインをより効率的に作るための方法
  • 効率よく XHTML や CSS を記述するための考え方と具体的な手法

などについてお話します。こういったことは、他のスクールではあまり教えることのない、このスクールの強みだと思っています。

このWebデザインスクールの体験レッスンを開催します。
