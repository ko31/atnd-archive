Zope/Ploneの開発者で集まって開発方法やディスカッションなどをやろうと思います。

スプリントのもうちょっとゆるい版（？）ですが、第０回として次回以降の方向性もあわせて決めていきたいと思います。

参加費は無料です。

会場は無線LANと有線LANがあります。

Zope/Ploneの開発に興味の有る方は、是非ともこの機会に一緒に勉強しましょう。

06-02追記人数が一杯になったんで、ちょっと席を増やしました。参加申し込みはお早めに

当日のタイムテーブルはだいたいこんな感じのつもりです。


  • 11:00-11:30 自己紹介とか
  • 11:30-12:00 やること決めたり準備とか
  • 12:00-13:00 昼食
  • 13:00-17:00 もくもくと開発
  • 17:00-18:00 成果発表(できたとこまで)
  • 18:00- 打ち上げ(いける人で)

会場内は飲食可ですので、好きなおやつを持ってきてください。
徒歩1分くらいのところにサンクスがあります。
