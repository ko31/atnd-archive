概要説明

20代～30代、フリーエンジニアが集まって交流する会です。 合同会社TEAMGUILDが主催のイベントですので、TEAMGUILDに興味がある方はもちろん、フリーランスとしての新しい働き方を模索したい方、フリーランス同士の繋がりを求めている方など、フリーエンジニアならどなたでも参加いただけるイベントです。

気軽に参加して楽しめる会にしたいと思っています。

しかも、はじめて参加の方は、無料でインドカレーなどが食べ放題！

参加する方は、ぜひツイート／シェアなどお願いします★
