「ゴスペルを歌ってみたい」
「でも定期的には通えないし」
「いきなり経験者の方々と歌うのにはハードルが高い」
そのような皆様からの声にお応えして、ゴスペルの基礎を３回にぎゅっとまとめた講座です。
これからゴスペルを始める方、またはゴスペルを歌っているけどあらためて初心に帰って始めてみようかなという方におすすめです。

5月31日（火）13：00-14：30「STEP1 はじめに」

ゴスペルの意味と簡単な歴史、曲の紹介、実際のレッスンの進め方、また歌詞によく出る英語の意味などをみていきます。
1. ゴスペルって一体何？
2. どんな曲があるの？
3. どんな風にレッスンを進めていくの？
4. 良く出てくる英語の発音と意味は？
5. 指揮にあわせて歌うことにチャレンジ！

6月7日（火）13：00-14：30「STEP2　具体的なレッスン」

ゴスペル曲を用いながら実際のレッスンの中で行う身体のストレッチ、ボイストレーニング、リズムトレーニングなどを行います。
1. 前回までの簡単なReview
2. 歌に関係があるストレッチとは？
3. どんな歌でも必要不可欠！ボイストレーニング
4. 苦手克服！リズムトレーニング
5. ゴスペル曲を歌ってみよう！

6月21日（火）13：00-14：30「STEP3 -曲をじっくり-」

ゴスペル曲の歌詞に込められた意味や、作者の背景などを見つつ、曲とじっくり向き合います。
1. 前回までの簡単なReview
2. 前回のゴスペル曲の続きを歌ってみよう！
3. 歌詞に込められた意味は？
