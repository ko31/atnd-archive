title 効果的な毛髪混入防止対策【大阪開催】
catch 食品工場・医薬品工場・包装材料工場などの衛生管理
description

衛生管理の技術が進化していく中で、いぜん毛髪混入の問題はなくなっていません。しかも、毛髪対策の状況は工場ごとに異なります。そのため、いくら毛髪混入防止対策を実施しても問題が解決しない工場もあります。
　本セミナーでは、頑張っているのに毛髪対策が上手くいかないといった状態に陥りにくくするために、どのように毛髪混入防止対策を考え、進めていくべきなのかを体感していただきます。

【プログラム】
１．はじめに
　　毛髪混入が減らないわけ、毛髪対策の考え方
２．【座学１】毛髪と毛髪対策の基礎
　　2-1.毛髪とは何か
　　2-2.毛髪の特徴
　　2-3.毛髪対策のセオリー
３．【座学２】毛髪診断調査
　　3-1.毛髪診断調査の目的
　　3-2.毛髪診断調査の手法
４．【演習１】毛髪診断調査の体感
　　4-1.毛髪診断調査結果の評価と分析
　　4-2.対策の立案
５．クロージング
まとめ、質疑応答
