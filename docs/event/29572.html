備考

Unityで迷路ゲームを作る予定です。プログラムに関する知識はなくても作れます。プログラム組めるとちょっぴり高度なことできるかも。

また、３Dゲームを制作する都合上あまりに古いPCだと動作しない可能性があります。予めUnityをインストールし、サンプルゲームが動作可能かを確認していただけると幸いです。

また、終了後に懇親会を予定しております。会場は「誠」という居酒屋さんで費用は一人あたり3000円程度です。
