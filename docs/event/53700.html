description

主催（罪）です。

さて、前回は夏コミで新作が出ると勝手に思っていた主催が、ゲムマで予想外の新エキスパンションが出て、突発的に遊びたがった結果のオフ会でした。
ので、今回は当初計画していたオフ会のスパンに戻りました。
前回は小規模ながら大変盛況でしたので、今回も当然六都市同盟を基軸にして遊びたいと思っております。

ハートオブクラウンに興味があるけどプレイ環境がない方や、デッキ成長型カードゲームのベテランの方も、
様々なプレイヤーとの交流を通じて、良い時間を過ごせればと思い企画しました。
関西でハートオブクラウンをプレイしている皆様の参加を是非お待ちしています！
（最近は、持ち込んで頂いた他のゲームなどで盛り上がることも多いですが、今回は新作に絞ってプレイするはずです）
