Code HAIKU 2012

Webサービスの開発に興味のある、学生や若手エンジニアを対象とした勉強会です。
第一線で活躍するプログラマから、現場で使われている技術やWeb開発のノウハウを学べます。
俳句の町である松山にちなみ、各講演者には味わいのある短いコードをひとつ、披露してもらう決まりになっています。
コードの俳人たちが詠む、至極の句をぜひ堪能してください！

※年齢制限はありません。勉強会を覗いてみたい方、引率の方、自称若手エンジニアの方、どなたでもお気軽にご参加ください。

講演概要

「Code HAIKUとは」門屋 亮

開催にあたってのあいさつ。もちろんコードはよみます！

「若いプログラマが問題を解くとこうなる」村崎 大輔

Webサービスを開発するにあたっては、技術面においても問題を知ることはとても大事なことです。
若さゆえの過ちをいくつか紹介します。

「よんで よんで よまれて よんで」山村 英五郎

自社の古いプロジェクトや、他社からの引継ぎのプロジェクト、オープンソース etc…
他の方が書いたプログラムを読む機会は、とても多いと思います。
そんなプログラムをどのようにして読み解いているのか？というのを、実体験をあつめてお話させていただきます。

「ブラウザやああブラウザやブラウザや」佐藤 鉄平

クロスブラウザ対応とかFeature detectとかそのあたりの話をしようかなと思います。

「Web標準とHTML5セキュリティ」竹迫 良範

ウェブ標準とは一体何だったのか？
HTML5ができるまでの苦悩とこれからの未来について
豊富な実例とデモを交えてお話しします。

懇親会について

勉強会終了後、サイボウズ松山オフィス内で軽食を取りながらの懇親会を行います。参加は自由です。
