大好き祭りってもう5年目なの!？

なんやかんやで5年目を迎えましたアキバ文化をみんなで楽しむお祭り「アキバ大好き！祭り」
今年も中央通りのベルサール秋葉原にて2月に開催します。
当日会場にて手伝っていただけるボランティアさんを募集しています。
お友達同士での参加もOK。秋葉原が大好きまたはイベントが大好きな人はぜひご参加ください。

アキバ大好き！祭りについては、 こちら を参考にしてください。

募集概要

開催日程 2/20(土)　11：00～22：00
　21(日)　11：00～18：00
時間 20日、21日いずれも 10：00－18：00のうち3H～
◇ 両日10時から朝礼があります
◇ 1日通しでお手伝いくださる方歓迎します
集合場所 ベルサール秋葉原１階
募集人数 50名くらい
参加資格 1）18歳以上の男女
2）秋葉原が好きな方、これから好きになる方
3）イベントの成功に向け協調性を保てる方
  • 業務内容
    1. 会場内のサポート・誘導、列整理
    2. 受付(迷子、忘れ物対応含む)
    3. 会場内清掃、パトロール
    4. 写真撮影
    5. 外国人来訪客の応対(通訳ボランティア)

備考 ボランティアスタッフの方には吊り下げ式のスタッフカードをお渡し致します。

  • ビル風が強いので寒さ対策必須です。
  • コスプレは禁止していませんが会場が本当に寒いので薄着の衣装は推奨しません。
    　(アキバに関連ある作品だと一部の常連ボランティアが喜びます)
  • ボランティアスタッフの着ぐるみでの参加は原則禁止します(作業がしにくいため)
  • また更衣室も現状ご用意の予定はありませんので何卒ご了承願います。
  • 通しで参加されるボランティアさんには軽食がつきます。

お問い合わせ

ボランティアへの参加希望の方はまずこちらのサイトにて参加表明をしてください。
今回初めて大好き祭りのボランティアの参加をされる方、また前回までに参加した方で連絡先が変わった方は
メールで件名に｢アキバ大好き！祭りスタッフ参加希望｣と明記し、
以下の項目をお書きの上、volunteer@akiba-df.comまでお送りください。

①お名前
②ニックネーム（もちろん本名のみでもOK）
③twitterアカウント
④電話番号（当日こちらから連絡がとれるものにしてください）
⑤メールアドレス（こちらから連絡がとれるものにしてください）
⑥参加できる日と時間
⑦自己ＰＲ(希望するボランティアがある場合)

必ずPCのアドレスを受信許可しておいてね♪
