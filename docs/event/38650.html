【　今回のテーマ・概要　】つくって遊ぼう

今回は特に明確なテーマは設けず、webの技術で作ってきたものを持ち寄って発表する、ということになりました。

「去年こんなのを作ってました」や「このJavaScriptライブラリを使ってみました」「俺の開発環境はこうです」など、なんでも結構ですので、持ち時間10分～30分を目安に発表して頂ければと思います。
他の参加者も、それに応えるように「俺ならこうする」「こんなツールを知っている」等の質問やツッコミを入れたりしながら相互に学びあったりするようにできれば理想です。

発表ネタは大小問わず、

  • JS (create.js, トゥイーン系JS, node.js など)
  • Flash / AIR (Stage3Dなど)
  • CoffeeScript, TypeScript等触ってみました的なアレコレ
  • アプリ
  • ゲーム
  • 拡張現実AR
  • キネクト/ LeapMotion

等なんでも歓迎です！

というわけで、参加される方は『当日発表するネタ（可能な場合）』や意気込みなどを添えて参加ボタンを押してくださいませ。

【　大阪てら子とは　】

てら子は、みんなで参加して学ぶweb系勉強会のワークショップです。

発足当初は、Flashをメインテーマとしていましたが、時流の変化と共に現在では広くweb全般をテーマになりました。ただ、他の勉強会さんよりかはFlash指数は高めかと思います。
