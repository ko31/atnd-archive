Hi Everyone!

【英会話超～初心者専門】
オーストラリア生活を満喫するための英会話講師Lisaです♪

10年以上のオーストラリア生活(ワーホリ、大学留学、仕事、ホストマザー、国際結婚etc)
経験者だから知っている★通常の英会話教室の1/20の時間で初心者が英語を話せるようになる、
今のあなたにあった秘密の学習法を診断いたします。

私自身、TOEIC200点のスタート。その後、留学したものの、英語が聞き取れず言いたいことも言えない日々で挫折を味わい
ましたがあることをやるとで最短で使える英語をマスターできました。

こんな方のための個別診断です。

〇オーストラリアに住んでいても相手が何をいっているかわからなく会話にならない
〇英語で話しかけられるとつい目をそらしてしまう
〇カフェでコーヒー1杯オーダーさえ緊張
〇日本人以外の友達が作りたいけど今の英語力では無理( ﾉД`)ｼｸｼｸ…
〇仕事での業務英語はなんとかなるが、それ以外の雑談ができるようになりたい

このような悩みから解消され
オーストラリア生活に不安なく自信をもって話せる英会話力を
通常の20分の一の時間で身に着けるためのあなただけの学習法を
個別で診断いたします。

あなたは遠回りな英会話学習、考えをしていませんか？
以下をチェックしてみてくださいね♪

●単語張をまずは徹底的に暗記すればいい

●オーストラリアにいるんだから生活していれば、そのうち身につく

●オーストラリア英語に慣れるために、スラングを勉強している

●英会話CD教材を常に聞き流していれば、いつか話せるようになる

実は、これすべてあなたの命の時間を無駄にしている学習法です。

今、最もあなたに重要なことは、学習の優先順位です。

英会話学習の無駄をなくし、話せるようになるための優先順位を明確にしましょう。
すると英会話学習の期間は驚くほど短縮できます。

英会話初心さんのほとんどの方が上記のような考えをお持ちになっていて
オーストラリアにいても話せるようにならないと後悔されています。

(内容は時間の都合などにより一部変更になる場合があります)

本来であればこの学習法は当教室の生徒さんにしか公開していません
