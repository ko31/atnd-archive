description

ほとんどの人は、運動を始めるときに『方法』に目を向けてしまいます。
でも、これは大きな間違いです！

運動するときにまず初めにやらなければいけないことは、
自分のカラダの状態を把握すること。
『自分』に目を向けることです！

自分の状態をわからずに選んだ方法は、本当にあなたに必要な内容でしょうか？
時間とお金とエネルギーの無駄遣いをしている可能性があります。

あなたの、

・骨格タイプ
・弱点

を明確にすることが大切です。

骨格のタイプによって、“なれる”カラダと“なれない”カラダが変わってきます。
骨格を明確にすることにより、『あなたにしかなれない』カラダがわかります！

そして、多くの人は世間のイメージに流されて運動をしています。筋トレが流行れば筋トレ、ヒップアップが流行れば尻トレ。それ、本当にあなたに必要でしょうか？

トレーニングはあなたの弱点を改善するものです。
あなたの弱点を明確にし、あなたを美しく魅せるのを邪魔しているところに対してエクササイズを行うのが、最も効率的で、効果的です。

体験レッスンでは、

・あなたの骨格タイプは？
・あなたの姿勢は？
・あなたの足のラインは？

を明確にしていきます。

人間はバランスが整っているものを、美しいと感じる生き物です。

あなたのカラダのバランスを乱している弱点を明確にしていきます。

それをもとに、あなたの希望と弱点を合わせて、
あなたに必要なエクササイズを体験していただきます。

筋トレマシンでは感じることができない、
今まで眠っていた筋肉が目覚める感覚を感じていただき、
理想のスタイルを体験していただきます。

※体験レッスンは90分です。
