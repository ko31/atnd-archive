catch
description

驚異的な市場成長率を誇るソーシャルゲーム業界は、今後グローバル展開で更なる成長を遂げることが期待されています。
一方、実際に海外進出を果たしたデベロッパーの成功談・苦労談の情報はまだ国内にそう多くはありません。

当セミナーでは、日本オンラインゲーム協会よりゲームポットやジークレストほか、既に海外展開で著しい実績を上げる企業によるパネルディスカッションで生の情報をお伝えするほか、シード・プランニングによる市場調査レポート、日本マイクロソフトによる最新技術動向や海外進出事例を交えたグローバルプラットフォームの情報をご紹介致します。

また懇親会では登壇者を交え関係者とコミュニケーションを図って頂くことも可能です。
貴社のグローバル展開戦略立案のご参考に、是非ご参加下さい。
