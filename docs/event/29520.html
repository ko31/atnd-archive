description

大阪では、“街と人のコミュニケーション”を軸に、街に集う人びとや地元民間企業が一体となり、夏至の夜のパーソナルなキャンドルナイトを呼びかけるイベントとして、夏至に先駆けて、６月１３日（水）西梅田ナイトを開催。

６月１３日（水）西梅田ナイトでは、会場内の各施設と周辺施設の協力による２０時から２時間のライト・オフを実施。

地元の専門学生・大学生や関西を中心に活躍するクリエイターによるキャンドルを使ったインスタレーション「キャンドル回廊」。
クリエイターと、地元の小学生のおともだちが参加してくれている、日本古来のあかり「行燈」にヒントを得た「ペーパーバッグ行燈展」。
来場者の方が参加して作り上げる「メッセージキャンドル」。

西梅田公園では、ＣａｎｄｌｅＪＵＮＥがステージデコレーションをしたキャンドルの灯の中での、アコースッティックライブ「Ｔｗｉｎｋｌｅ　Ｌｏｖｅ　Ｌｉｖｅ」を実施。
出演アーティストは、高野寛、ｙｅｙｅ、Ｊｉｍama、リクオの４組。

キャンドルナイトに参加しているクリエイター達の作品が手に入る「キャンドルナイトマーケット」も開催。

茶屋町ナイトに引き続き、西梅田ナイトでも、「ゆかたｄｅキャンドルナイト」を提案しています。

ゆかたで来場いただくと、ブリーゼブリーゼで「おめかしハンカチ」がもらえたり、ヒルトンプラザでクーポンがもらえたり、ハービスＰＬＡＺＡ／ＰＬＡＺＡＥＮＴでは各店舗から素敵なサービスが受けれたり、いろいろなお得な企画もあります。

街なかに一夜限りのキャンドルの森が現れるひとときを、ぜひ体験してください。

詳しくは公式ＨＰをご覧ください。
