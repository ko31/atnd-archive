デ部第7回めのミーティングです。

１３時開始ですが、入館手続きが必要なので開催場所の1Fに12:45集合でお願いします。

今回はLighting Talk(LT)以上Tech Talk(TT)未満のNeta Talk(NT)形式でやってみます。
プレゼン時間は最大１０分、質疑応答が５分。６人ぐらいの発表者を想定しています。
プレゼンが終ると興味のあるNetaに集まり、モクモクするなりワイワイするなり自由にどうぞ。
発表希望者は下のコメントに内容を（できるだけ短く）書いて下さい。
Neta例：
・独自UI紹介
・Androidで流行ってるアプリ紹介
・Android関連API紹介
・Android関連技術紹介
・作ったアプリ紹介
・etc …

技術ネタじゃなくてもOKです。
ぶっちゃけモバイルに関係しそうならなんでもw


ミーティング後は懇親会という名の忘年会を開催予定。当日、参加の可否を聞きます。

デ部は、ガチでAndroidアプリを作り上げることを目的としたグループです。
デ部の活動に参加される方は、デ部のGoogle Groupsに参加した上で、まだの方は自己紹介の記入をお願いします。（以下のリンクを参照）

自己紹介は、Groupsに参加したら同じアカウントで編集できるようになるはずですが、なぜかたまに無理なこともあるようなので、もしシートを編集できないようでしたらご連絡下さい。

・デ部サイト
・Android DEvelopers’ cluB (abbr. デ部) @Google Groups
・デ部　自己紹介
