浴衣で納涼船withポーケン

イベント ユーザー
event_id 985
title 浴衣で納涼船withポーケン
catch Poken Meetup
description

「「「夏だ！海だ！Pokenだ！！！」」」

と言うことで、Twitterで唐突に開催の決定した、浴衣で屋形船withポーケン企画。
今回、残念ながら屋形船ではありませんが、東京湾納涼船にて、
ローコストで船上パーティが出来ると言うことで開催が決定しました。
お一人様￥4,000-で飲み放題＋軽いおつまみ程度が付きます。
（一人2500円 ＋ 個室を確保する条件（オードブル２皿）の30000円を20人で頭割り（一人1500円）の計4000円です。）
みなさん、奮って、浴衣姿でご参加ください！
もちろん浴衣じゃなくてもOKですよ♪

※上限が20人となっていますが、当日券がある可能性が高いことと、
　もし早く20人埋まってしまった場合、追加予約しようと考えているので、
　定員を過ぎてしまっていても、参加したい！と言う方は遠慮せずにお問い合わせください。


※当日、
Pokenの直接販売も行います！
持っていない方でもそのまま気軽にご参加ください！

※当日、決断ポトフのダダ漏れ取材が行われることになりました！
なので、個室の一部を、肖像権フリーコーナー（顔出しOKの人向け）にしようかと考えております。
ご理解とご協力のほど、お願いします。


◆日時：
　7月20日（月）（海の日祝日）
　18時45分集合
　19時15分出航なので、余裕を持って30分前に現地、もしくは45分前に最寄り駅集合としたいと思います。

◆場所：
　竹芝客船ターミナル
　├ゆりかもめ「竹芝」駅すぐ
　├JR山手線・京浜東北線「浜松町」駅 徒歩8分
　├都営浅草線「大門」駅 徒歩11分
　└都営大江戸線「大門」駅 徒歩11分

◆予算：
　一人 4千円
