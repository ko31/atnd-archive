概要

ALT is

  • Aizu.LT (先代の方々がやっていた) と Alternative からとった名前。LTイベント。

やること

  • LT (Lightning Talk) : 交代しながら、5~10分間(+α)で発表をしていきます。
    • 技術系の話に限定します
    • 聞くだけの参加ももちろんOKです。
    • 交流目的でも良いんじゃないかと思います。

やらないこと

  • ストリーミング配信、もしくは録画の配信
  • 主催側でのスライド回収

募集について

  • 話していただきたい内容について: 技術系の話
    • 技術的な話であれば、その他細かいことは問わずです。(程度は守ってください)
    • これまでの内容については、過去のイベント履歴を参照してください。(まとまっていない箇所については... 申し訳ありません :bow: )
  • 対象について: 会津関連の方 (どのような関連・関係かは問いません)
  • 登録期限について: 開始時間まで
    • ただし、会場・登録人数によっては制限が入るかもしれません
  • 発表しない方も参加にしていただいて大丈夫です。
  • もし発表が確定している方がいれば、コメントいただけると嬉しいです!!!

当日のこと

  • 時間 : 13:00 - 17:30 (仮)
  • 場所 : 会津大学内 (部屋未定)
