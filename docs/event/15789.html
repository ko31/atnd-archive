本ATNDシステムにて受付

交通手段：

自動車：
　カーナビの設定は「恩方中学校」または「聖パウロ学園高校」で設定下さい。
　住所は八王子市下恩方町２９３８番地、聖パウロ学園高校手前です。
　大久保信号機から約５００ｍ先の右手に恩方中学校がありますので、校門前を左折して下さい。
　通学路なのでスピードの出し過ぎには充分ご注意下さい
　※町会の依頼により大久保信号機からは進入しないで下さい
　　恩方中学校正門前を左折して下さい

電車：
　①東京駅=中央線中央特快(高尾行)⇒高尾駅
　　07:20発 ⇒ 08:18着(\890)-58分
　②千葉駅⇒御茶ノ水駅⇒高尾駅
　　06:32発 → 08:18着(\1,380)-1時間46分
　③横浜駅⇒東神奈川(横浜線(八王子行))⇒八王子中央線((高尾行))⇒高尾
　 07:12発 → 08:26着(\890)-1時間14分

『高尾駅』北口→『陣馬高原下行き』バスに乗車『大久保』バス停下車
　発車時刻：7時00分 / 7時50分 / 8時50分 (乗車時間20～30分)
　大久保信号機を左折して「聖パウロ学園」の看板を目印にいらして下さい。バス停から約１０分です。
　車の方は駐車場が御座いますがなるべく乗り合いでお願いします
