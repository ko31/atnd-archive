【講座の趣旨】
　技術者こそ、今マーケッティングを学ぶことが重要です！　組織のビジョンをベースとし、マーケティング技法を駆使した戦略策定の方法を修得します。安易なＳＷＯＴ分析では無く、本質的な活用方法「自社・顧客・市場・競争分析」から戦略を策定する体系的なプロセス」を演習中心で修得します。

３つの何故？の答えを考えて下さい！　
　Step1　:　何故〇〇を買う必要があるのか？
　Step2　:　何故、今？　〇〇を買う必要があるのか？
　Step3　:　何故、貴方（貴方の会社）から？　〇〇を買う必要があるのか？

【習得できる知識】
　１．マーケティングの基礎ノウハウが活用できる
　２．自社の商品（サービス・ソリューション）の分析ができる
　３．自社のサービス・お客様・敵の「見える化」ができる
　４．お客様の価値に訴求するマーケティングが理解できる
　５．業界の環境と競争分析ができる
　６．戦略と実施計画が策定できる

【プログラム】
１．自社のミッション・ビジョンの確認
２．商品（ソリューション）分析
３．トレンド分析・ライフサイクル
４．お客様価値（４Ｐから４Ｖ）
５．３Ｃ（４Ｃ）分析　自社何？敵は誰？
６．顧客分析
７．業界を取り巻く環境分析
８．競争分析
９．弱み問題点分析
１０．ＳＷＯＴ分析
１１．戦略策定
１２．実施計画とリスク分析
１３．お客様満足度とブランド、リピーターの獲得極意
