Google App Engine勉強会 in 盛岡

定員を 50→60に増やしました!!

岩手県盛岡市で、GoogleAppEngineの勉強会を開催します。
当日は、Google API Expertの方に来盛していただき、

  • Python
  • Java

に分かれてハンズオン形式で勉強会を実施します。

参加費はありません。
Googleのエンジニアと、GoogleAppEngineを使いこなしているAPIExpertの方々に岩手で会える滅多にない機会です！懇親会も企画していますので、聞きたいことは気軽に質問しちゃってください。

主催

Hack For Iwate

共催

盛岡情報ビジネス専門学校

日時

5/19 15:00～18:00
5/20 10:00～15:00

場所

盛岡情報ビジネス専門学校 http://www.morijyobi.ac.jp/

会場

会場は、盛岡情報ビジネス専門学校さんのご協力をいただきました。

  • 電源と無線LANが利用可能です。ノートPCと延長コードをご持参ください。

スケジュール

1日目(5/19)

午後からGAEを紹介する簡単なセッションを行います。
翌日用の環境構築のお手伝いをしたあとに、懇親会を開催します。

15:00～17:00 GAE紹介セッション
17:00～18:00 環境構築

2日目(5/20)

PythonとJavaに分かれて、ガイドに従い簡単なプログラムを作成するコードラボを行います。

10:00～12:00 午前の部
13:00～15:00 午後の部

講師陣

  • 松尾 貴史さん (@tmatsuo)
  • 小川 信一さん (@shin1ogawa)
  • Ian Lewisさん (@IanMLewis)
  • 上田 哲広さん (@najeira)

対象

参加者の前提として 「コンピューターで何らかのプログラムを書いたことがある人」 を対象とします。プログラミング言語等は問いません。

懇親会について

懇親会は、会場か盛岡駅近辺で考えており、 会費は3,000円 です。
懇親会会場は決まり次第、Twitter等で追って連絡します。
※Twitterの @Hack4Iwate アカウントのフォローをお願いします。
