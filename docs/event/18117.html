シネパ!　エピソード３
「実写版　忍たま乱太郎」（監督：三池崇史）を観て、飲もう！

シネマなパーティ「シネパ!」の始まりです！
話題の映画を映画館で一緒に観た後に、飲みながら映画トークしよう！
（※映画を観た方は、飲み会だけの参加もちろんＯＫ！）

「実写版　忍たま乱太郎」の話に始まり、初めて観た映画、初めてデートで観た映画、号泣した映画、好きな役者、監督、今度観たい映画などなど初対面同士で映画トークを楽しみましょう！

日時：８月３日（水）夜の最終回を観覧予定（現在、時間未発表）
劇場：新宿バルト９　
http://wald9.com/index.html
映画：「実写版　忍たま乱太郎」（監督：三池崇史）
http://wwws.warnerbros.co.jp/nintama/
