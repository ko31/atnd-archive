■概要

第二回勉強会のメインテーマは 「AWS 初心者向け勉強会」です。
AWSを知らない、触ったことがないという方向けのハンズオンを行う予定です。
AWS エバンジェリスト玉川憲さんからのAWS overview & update、その他恒例のLTも行います。
懇親会 ももちろん行います。

■ハンズオンのための準備

AWSアカウントを取得しておいてください。
こちらに資料があります。
３０分くらい？かかります。
