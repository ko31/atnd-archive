マニフレックス大阪ショールームでは、質のよい眠りに関心のある女性のためのアカカツ（セミナー）“ボンジョルノ！マニフレックス”を開催しています。朝の時間を有効活用していただけるよう、朝7時30分にスタートする体験型セミナーで、参加費500円は全額寄付されます。おいしい軽食と飲み物をご用意して、皆様のお越しをお待ちしています。

マニフレックス朝活 6月・7月プログラム

2017年7月21日（金）朝7:30～8:30

代謝UP!トレーニング

早起きで気になる部分をボディメイク　～朝トレで毎日を健康に元気よくスタート～
マットレスの上で行うトレーニングなら起きた瞬間からスタートできます。筋肉を動かすことで身体が温まり、しっかり目が覚めるので朝からテキパキ動くことができ代謝UPにもつながります。
短い時間でもコツコツ続けることで素敵ボディを手に入れることができますよ！
簡単で手軽に出来るトレーニングですのでお気軽にご参加ください。

講師：亀割 七奈代（かめわり ななよ）
プライベートジムStyledgeにて「将来、いつまでも若く・健康に生活をする」をモットーに管理栄養士兼ボディデザイナーとしてお客様への食事指導、トレーニング指導を担当。

参加費：500円（焼きたてクロワッサンと紅茶付き）
