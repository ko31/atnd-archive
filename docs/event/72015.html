ちょっと気が早いですが、12月に入って忙しくなってしまう前に、
今のうちに、お正月飾りを作っておきませんか？

「筆文字」と伝統的な「裏打ち」の両方を、一度に体験できる欲張り講座です。

次の方におすすめです。
・書道初心者（筆の使い方からご説明いたします）
・すっかり書道とはご無沙汰で、もう一度チャンスが欲しい。
・お教室には通う時間はないけれど、筆を使うコツを知りたい。
・自分の作品を簡単に手軽に飾ることができる、裏打ちを経験したい。

リクエストに応じて、その場で手本を用意いたします。

幼少時代から書道塾に通い続け、大学院でも書道を専攻、現役中学高校書道教員、書道芸術院審査会員。受賞経験多数。
指導した生徒からも受賞者多数輩出。

これまでに得た技術・知識を惜しみなくお伝えいたします。

墨の香りに包まれて、リラックスして心を休め、
あなただけの世界に一つの壁飾りを作りませんか？
