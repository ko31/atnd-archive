プラスチックの強度特性と破面解析

イベント ユーザー
event_id 41909
title プラスチックの強度特性と破面解析
catch プラスチック材料の基本特性、破壊のメカニズム、破面観察と解析技術、破断面の事例を易しく解説！
description

【講座の趣旨】
　使用中のプラスチック製品が損傷した場合、材料の種類とその強度、設計形状、使用環境など、色々なことが考えられる。本セミナーはこれらの問題を簡易的に解決するため、材料の基本的な特性、破壊のメカニズム、破面観察と解析技術、破断面の事例など易しく解説し、即実務が可能な技術が習得できる内容のものである。なお、これらの内容について、金属材料の場合と比較しながら解説する。

【プログラム】
１．プラスチック材料の基本的な強度・破壊特性
　　１）プラスチックの強度と破壊
　　２）負荷応力と種類と応力形態
　　３）環境による劣化
　　４）き裂の発生と進展挙動
２．使用時に役立つ力学的特性
　　１）引張り特性の見方
　　２）曲げ特性、衝撃特性について
　　３）疲労強度特性について
３．クラック現象の解明
　　１）クレーズとクラックの違い
　　２）ストレスクラックの特徴
　　３）ソルベントクラックの特徴
４．使用環境下における損傷のメカニズム
　　１）熱分解による損傷
　　２）加水分解による損傷
　　３）紫外線劣化による損傷
　　４）放射線劣化による損傷
５．破断面解析調査の仕方と手順
　　１）破断面解析の手順
　　２）起点の見つけ方
　　３）破面解析のポイント
６．破断面から損傷原因を探ろう
　　１）マクロ的破断面の見方
　　２）ミクロ的破断面の見方
　　３）引張り破断面について
　　４）曲げおよび疲労破面について
７．プラスチック製品のトラブル対策
