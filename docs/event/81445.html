description

最近、将来の自分のキャリアを考えることが多くなった。
今の仕事は嫌いじゃないけれど、このままずっと働くイメージが持てない。
昔から挑戦してみたかったことをやり残している。
もっとこうしたら良くなるのに…と思いながらも気づかないふりをしてきた。
そういったあなたの中にくすぶっている思いを形にしてみたい！と思ったことはありませんか？

気持ちはあっても経営なんてよくわからないし、お金のことや営業のことも詳しくない。
周りに話したら笑われるかもしれないし、好きなこと、やりたい事を仕事にするのは我儘だという人もいる。
「稼ぐことはそんなに甘くはない、君にはまだ無理だ」という人もいるでしょうし、
どうすればいいのか分からないから、自信をもって「大丈夫！出来る！」と言い切れないという気持ちもある。

それでも、自分の心の中にある「やりたいこと」が消えないのであればその想いは本物ではないのでしょうか。
自分の想いを信じることで新しいビジネスの種が生まれ、価値が生まれるはずです。
その想いを形にしたビジネスで誰かの役に立つことが実感出来れば、苦労も乗り越えられます。
もちろん、実際に事業を始めたら、うまくいくことばかりでないのも事実です。
しかし、あなたの想いを形にしたビジネスが、多くの人の共感を呼ぶことが出来れば、それはビジネスの成功に繋がります。
あなただからできたことがビジネスとして、多くの人の役に立つことが出来るかもしれないのです。

社会起業大学は、アイデアを本気で形にし、実行する環境があります。
一人一人と向き合い事業プランや課題を聞き、時に寄り添い、時に厳しくアドバイスをし、あなたがやるべきビジネスを見出すまで、とことん伴走をします。

授業では、老若男女、バックグラウンドも価値観も違う仲間が集います。
心からあなたの成功を信じてくれる仲間から、様々な角度のフィードバックがあることで、多くの共感を集めるビジネスが生まれます。共感が集まるビジネスは、資金調達や顧客獲得、スタッフ採用、パートナーとの出会いに繋がり、軌道に乗ります。

自分の中にある気持ちを形にする方法があるのなら知りたい。
そう思った方は、無料体験授業で自分らしさ×起業を知るところから始めてみませんか？
