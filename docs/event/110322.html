■イベント概要

秋葉原駅からもほど近い東京浅草、蔵前の空室オフィススペースが11月30日VR体験スペースとして開放！更にVRChat内でも同時にワールドが開かれここに「リアル・フレンドプレスワールド」が当日限定で誕生します！
会場に用意したOculus Questやフルボディトラッキング仕様のVIVEをかぶって、リアルフレンドプラスワールドに旅しよう！
当日は百戦錬磨のVRChatユーザーが参加者の皆さまのVR体験をサポートします！Oculus Questではマルチプレイでのゲームへ。PCVRでは高クオリティのVRワールドへダイブしよう！BeatSaberなど他のVRゲームも体験可能です！

■イベントの趣旨

アストネスVR体験会の基本的趣旨は、VR技術やサービスの普及を願う当イベントの主催・共催・協賛者が会場とVR機材、誰もに開かれたVRコンテンツのみを用意して、参加者の皆さまが主体性を持ってVRを体験して頂く【自主・参加型ミートアップ】です。

■こんな人に来てほしい

・VRChatに興味あるけど一緒に遊ぶ友人もいないし中々始めるチャンスがなかった
・VRの魅力や遊ぶときの疑問についてVR空間を通じてVRChatユーザーに直接聞いてみたい！
・デスクトップ勢だけどVR環境でVRCを体験したい！
・広い部屋でフルボディトラッキングを体験してみたい！

■注意事項

・イベントで用意する機材はレンタルサービスの在庫品です。商品なので大切にお取り扱い下さい。
・VRChatはVR世界の向こうに人がいるゲームです。お互いに配慮しあい、気持ちの良いワールドの雰囲気作りにご協力下さい。
・VR世界内においてのルールは全てVRChatの規約に帰属します。プレイ中にスタッフの指示があった場合は従って下さい。
・スタッフからのお願いやご注意に応じずゲーム内及び会場でのルールに違反される方はご参加をお断りする場合がございます。

■用意する機材

当日は以下の機材をご用意する予定です。
・VIVE：2台＋VIVEトラッカー：6台　2台のVRシステムでフルボディトラッキングを体験できます。
・VIVE Cosmos：1台
・Oculus Quest：3～4台
・ゲーミングノートPC：1台（PCスペックはこちらから確認して下さい。）
PCVR（VIVE / VIVE Cosmos）を動かすPCが１台しかないので、ゲーミングノートPCを持って来て頂ける方がいると助かります。PCが用意できれば、当日動かせるPCVRが増えます。PC持参できる方は、アストネスTwitterまで事前にお知らせ頂けると助かります）
