description

10/27のTitanium Meetup Tokyoでは、Appcelerator社 CEOをJeff Haynieが来日し、Titanium MobileやAppceleratorの展望などについて、直接お話します！

質疑応答の時間も取りますので、直接質問できる絶好の機会です。

Titaniumで聞きたい事、言いたいことある方は是非この機会に直接聞いてみませんか？

すでにTitanium Mobileをお使いの方だけでなく、まだ悩んでいる、これからと考えている方のご参加もお待ちしています。

同時に、日本側からTitaniumで作られたアプリ/ライブラリの紹介LTを行いたいと思います。
こちらでご発表頂ける方は、ATNDのコメントか、@appcelerator_jaまでメッセージをお願いします。
LTに登壇いただいた方には、粗品の赤いTシャツを用意しております。

なお、講演については英語となりますが、比較的理解しやすいと思いますので、
英語が苦手！と言う方も、ぜひご参加ください。

質疑応答などでは、コミュニティーの方々によるサポートが付きますので、こちらも
英語の苦手な方もお待ちしています。

なお、Usteamで中継も予定しています！来られないかたは、こちらでぜひ！
http://www.ustream.tv/channel/titanium-meetup-tokyo
