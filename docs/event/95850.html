第４回ダンパWEBを開催します！

今回は、「DANCERUSH STARDOM」を使用したパフォーマンス大会となります。

出場をご希望の方は、アンケート欄に下記を全てご記入願います。
記載漏れがある場合、参加受付されません。

  • (1) プレイヤー名
  • (2) 動画のアドレス(YouTubeやファイル共有サイトなどのアドレス)
  • (3) アピールメッセージ

優勝者には優勝賞品を進呈します。

ルール

  • DANCERUSH STARDOM の「プレミアムモード」で撮影した動画を投稿下さい。それ以外は失格となります。
  • 動画は、DANCERUSH STARDOM 稼働日(2018年3月23日)以降のもののみ有効とします。稼働日以前の動画は失格となります。
    *　動画は、一般公開／限定公開　いずれかの設定として下さい。「非公開」の場合、失格となります。
    *　曲は「プレミアムモード」で選曲可能な曲のみとします。
  • 　プレイモードは１人プレイ、２人プレイいずれかとします。３人プレイ以上　或いは　２台の筐体動画を合成した動画は、失格となります。
    *　参加人数、曲のレベルは不問とします。
  • 　最後までプレイし切って下さい。途中でFailしている動画は失格となります。
    *　動画編集は認めません。編集した場合失格となります。
    *　動画投稿期間中、動画の入れ替えを認めます。
  • 下記動画は失格対象です。
    • 公序良俗に反する内容
    • 極端に露出度の高い衣装
    • 周囲に迷惑をかけている映像
    • その他運営側が不適切と判断した内容
  • 今回「審査準備期間」を設けました。投稿動画の設定に不備があった場合、是正する事で失格を回避できる制度です。ただし、動画の撮り直しは認めません。撮り直しが必要な内容（例えば途中でFailしているなど）は、失格となります。

スケジュール

  • 開催概要説明 2018/02/12（ストリーミング配信）
  • 審査員募集 2018/30/01～2017/04/29
  • 動画投稿 2018/04/01～2018/04/29
  • 審査準備期間 2018/04/30～2018/05/04
  • 審査期間 2018/05/05～2018/05/13
  • 結果発表 2018/05/19（予定）
  • 公開中の動画削除 2018/05/31

注意事項

  • 審査期間中、投稿頂いた動画をこちらのチャンネルで一般公開します。
  • 投稿頂いた動画は、2018/05/31 に削除します。
