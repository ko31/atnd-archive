遅延評価とプログラムの論証について

Haskell入門者のための勉強会 スタートHaskell の 第6回 （最終回）のATNDです。

スタートHaskellは、入門者から初心者の方を対象に、実際にコーディングをしながらHaskellを学ぶ勉強会です。月1回ペースで全6回を予定しています。

今回は プログラミングHaskell の 12章 から 13章 までの範囲の演習問題をみんなで解いていきます。

12章：遅延評価
13章：プログラムの論証

また 新刊「簡約!? λカ娘(二期)」 の頒布があります。
