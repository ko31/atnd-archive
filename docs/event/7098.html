大阪市東成区内で行われる「東成区　地蔵盆まつり」
毎年、町内の子どもたちの健全な成長と地域の親睦のために行われています。
今回は「Ａ-yan!!」が参加させていただくことになりました！
「アート戦士エーヤンダー５」公演を行います。

子どもから、大人、お年寄りまで一緒に楽しめる地域のお祭りなので、
東成区以外の方も、ぜひともお気軽に遊びに来てください。
よろしくおねがいします。
