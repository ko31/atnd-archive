title 色彩工学の基礎と応用
catch 色彩を客観的に定量化するには？色彩工学の基礎から色の数値化及び測定方法の考え方まで詳説！
description

【受講対象】
・品質マネージメント及び色彩管理の初心者及び実務者

【習得知識】
・色彩を客観的に定量化する方法及び留意事項

【講座のポイント】
　色を数値化して表示することによって、色の客観的な測定が可能になる。色の数値化及び測定方法の考え方並びに標準化の動向について紹介する。また、測色において注意しなければいけない留意点及び色の数値化の応用事例として、条件等色、演色性評価方法について紹介する。

【プログラム】
１．色の数値化
　1-1．CIE表色系
　　　　1-1-1．XYZ表色系
　　　　1-1-2．CIELAB表色系及びCIELUV表色系
　　　　1-1-3．CIE色差式
　　1-2．CIE表色系における物体色と光源色の扱い方
　　1-3．CIE表色系の応用
　　　　1-3-1．条件等色とその評価方法
　　　　1-3-2．演色性とその評価方法
　　　　1-3-3．演色性評価の問題点と研究動向

２．色の測定方法
　2-1．視感測色方法
　2-2．物理測色方法（反射及び透過物体色の測定方法）
　2-2．反射及び透過物体色の測定における留意点
