■Techcyが企画する初学者向けのプログラミング講座です。

Techcy（@techcy_jp）主催の学生＋社会人向けワークショップです。

Techcyは主に小学生＋中学生にプログラミングを学ぶワークショップを開催しています。

次回イベントはこちら

さらにTechcyは大学生+社会人を対象としたプログラミングの学び場を用意しています。

※プログラミングを学ぶ「きっかけ」がほしい
※プログラミングを学びたい、でも学ぶ環境がない
※オンライン学習サービスを利用しても続かない

と感じている方には絶好の機会かと思うので、ぜひ一度お越し頂ければと思います！

■テーマ

Wordpress（参考URL）とはお洒落でオリジナリティのあるメディアを簡単に作成・運営できるツールです。最近では企業やNPOなどでもWordpressを導入してコーポレートサイトを運営する組織も増えて来ましたが、個人でメディアを運用するのにもかなり利便性の高いツールとなっています。

当イベントを通じて参加者のみなさんが個人のメディアを十分に運用できるレベルまで達成することを目標にコンテンツを設計しています。

■対象者

  • Wordperssを利用してブログやメディアの運用を検討している方
  • 学生でも社会人でもOK！
  • その他、周りに使っている人がたくさんいるけどもいまいち使い方がわからない！という方など初学者の参加をお待ちしております！

■参加費

  • 学生 1,000円 (当日会場でお支払いいただきます)
  • 社会人の方 2,000円 (当日会場でお支払いいただきます)
