description

エンジニアが各々持ち寄りのテーマで勉強会ができるような
ネットワークを作りたいと思っています。
勉強会の場所は当社で無料で貸し出しいたします。

まずは、興味のある方同士で、交流会の場を持ってはどうかと思い、
懇親の場を企画しました。

情報交換も兼ね、勉強会に興味のある方の参加を募ります。

当日の流れ
・自己紹介タイム
・勉強会企画アイデア発表

※なんと、飲み物／フード／アルコールが無料！
※主催者からの連絡メールを受信するために、メール登録をお願いします

■対象者

・勉強会がしたいけど、場所がなくて困っているエンジニアの方
・勉強会に参加してみたいけど、敷居が高くて踏み切れない方
・勉強会を主催していて、新しいメンバーを募りたい方

※上記に当てはまる場合でも、営業や採用目的の方は参加できません。
※本交流会は、当社の転職斡旋のための登録目的ではありません。
　あくまでも、エンジニアに強い・クリエイターに強い紹介会社として
　認知いただくための施策の一環として、開催させていただきます。
