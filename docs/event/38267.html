書籍 「はじめてのiPhone/iPadアプリ開発」の出版を記念して、初心者向けiPhone開発の無料セミナーを行います。

セミナー内容

iPhoneアプリ開発に必要な情報、アプリ開発に必要な技術スキルなどのお話の後に、実際にXcode(Mac)を使って簡単なアプリを作ってみるハンズオンを行います。Macをお持ちの方はぜひMacをお持ち下さい。

講師は 「はじめてのiPhone/iPadアプリ開発」著者の 吉田裕美 ( EY-Office ) です。

会場では、書籍 「はじめてのiPhone/iPadアプリ開発」 の販売も行います。

参加資格

  • 書籍 「はじめてのiPhone/iPadアプリ開発」 を購入された方
  • 書籍 「はじめてのiPhone/iPadアプリ開発」 を購入を考えている方
  • iPhoneアプリを開発してみたいと思っている方

ハンズオン資料

http://redmine.ey-office.net/projects/ios/wiki/

Macをお持ちになる方へ

  • Macには予めXcodeをインストールして来て下さい。XcodeはApp Store アプリ(Mac App Store)から無料でインストールできます。
  • 電源コンセントは十分には用意できませんので、バッテリーを充電してきて下さい。
  • 無線LANは利用できます。

会場の注意

  • 会場の産報佐久間ビルの入り口は写真のように通りから奥まっていますので注意して下さい。

  • 会場(8F クラスメソッド株式会社)は 18:30 からオープンしています。
  • 会場のビルは 19:00 で正面入り口が閉まるので、19:00以降は右側の通用門から入場下さい。
