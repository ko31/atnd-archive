立花証券ストックハウスではマーケットアナリスト 平野憲一氏（日経CNBCコメンテーター・K Asset代表）にお越しいただき、「新春相場の展望と有望株　～米中貿易戦争への基本観～」と題した株式セミナーを実施いたします。当セミナーは東京八重洲にて開催いたしますので、東京近郊にお住まいのお客様はこの機会に是非ご参加ください。
