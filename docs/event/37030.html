第2部　ポリオレフィン・金属等の接着性改良技術
【12:15-13:30】

講師:福島大学　

【キーワード】
1.ポリオレフィン
2.高分子表面改質
3.接着

【講演主旨】
ポリエチレンやポリプロピレンに代表されるポリオレフィンの材料は安価、軽量、高強度であるが、ぬれ性、接着性、塗装性に欠ける。簡単な改質法では、その効果の持続性がなく、プライマー付きのアクリル系接着剤は高価で、耐水性が乏しい。これらの高分子に加えて、PET、ポリカーボネート、シリコン樹脂、フッ素樹脂などの材料にも適用可能で、その改質された性質が長時間退化しないような表面改質法を開発した。その方法は、材料劣化がなく安全性があり、多くの材料を親水化でき、接着できない材料を接着可能とする(デンプンやPVAから高価な接着剤まで使用可能)・さらに、これまでできなかった水性インクを用いたインクジェット印刷が可能となる。水に濡れ、接着可能なシリコン樹脂は医療用器具に最適である。一般的な改質法の解説と新法の成果を、実物を提示しながら解説する。その用途は限りなく広い。聴講の結果、意外な用途が生まれる可能性を期待したい。

【プログラム】

1.難加工性高分子とは
　1.1　ポリオレフィン(ポリプロピレン、ポリエチレン、ポリメチルペンテンなど)
　1.2　超高分子量ポリエチレン
　1.3　PET
　1.4　ポリカーボネート
　1.5　シリコン樹脂(ゴム)
　1.6　フッ素樹脂
　1.7　カーボンファイバーFRP;車の軽量化

2.高分子材料の表面改質法
　2.1　物理的方法;放電処理、火炎処理、サンドブラストなど
　2.2　化学的方法:化学反応、グラフト化
　2.3　提案の方法
　2.4　改質効果の評価

3.ぬれについて

4.接着のメカニズム:化学結合、アンカー効果、プライマー

5.新改質法とその成果
　5.1　ポリプロピレンの吸水性
　5.2　超高分子量ポリエチレンの接着
　5.3　PETの水性インク印刷
　5.4　シリコンゴムの水性塗料の塗装
　5.5　シリコンゴムを用いた医療器具
　5.6　フッ素樹脂の接着、塗装などの見本

6.新たな可能性:新素材の開発、二次加工など

第4部　異種材料接着(樹脂/金属)における界面解析とその事例 ぬれ性と接着性
【15:15-16:30】

講師:FIA　代表　福山 紅陽　氏

【キーワード】
1.ぬれ性,接触角
2.表面張力
3.界面張力
4.表面自由エネルギー
5.界面自由エネルギー

【講演主旨】
超親水・超撥水・超撥油など,表面のぬれ性を制御する技術が目覚しく発展しています。一方,ぬれ性は,接着性,離型性,防汚性など,工業的に重要な各種特性にも密接に関連していることが知られています。本セミナーではまず,ぬれ性を直感的かつ簡便に評価できる指標でありながら,固体最表面の特性を鋭敏に反映する接触角の概念について説明します。次に,ぬれ性を決める因子である表面張力,表面自由エネルギーの概念と由来について説明します。さらに応用として,表面自由エネルギー(表面張力)の成分分けの概念に着目した表面自由エネルギー解析について解説します。

【プログラム】

1.ぬれと接触角
　1.1　接触角から何がわかるか?
　1.2　接触角測定の表面感度～膜厚と表面被覆率

2.ぬれと表面張力
　2.1　表面張力,表面自由エネルギーとその由来
　2.2　Youngの式～接触角と表面張力との関係
　2.3　ぬれ性を制御するにはどうすればよいか?

3.表面自由エネルギーと接着性,離型性
　3.1　Dupreの式～界面分離でのエネルギー保存
　3.2　接着性がよいとはどういうことか?
　3.3　離型性がよいとはどういうことか?
　3.4　Young-Dupreの式～接着性と接触角の関連づけ

4.表面自由エネルギー解析
　4.1　表面自由エネルギーの成分分けとは?
　4.2　表面自由エネルギー解析から何がわかるか?
　4.3　接着性・離型性と表面自由エネルギー成分との関係

5.まとめ
