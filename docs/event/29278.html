description

　先月「ウェブ屋さんだってプロダクトが作れる！ スーパーコンシューマーによるプロダクト開発の裏話。」にて好評を博した中谷氏のセミナー開催の第２弾となります。
　今回は視点を変えて、音声メディアの活用について、お話させていただきます。

セミナー概要

ウェブ屋が考えた「旅道場」というサイトがオモシロイ。
このサイトでは、旅行のガイドブックを販売しているのだが、
音声ガイドブックなのである。

世の中のガイドブックはインフォーメーションが多いが、
このガイドブックは体験を伝えるガイドブックなのです。

ーテレビで、ラジオが変わった。
ーネットで、ラジオが変わった。
ーそして、ポッドキャストの可能性

なぜ、ウェブ屋がこんな企画を考えたのか、生まれた経緯や可能性、
そして今後について解説していただきます。
旅行好きの人は何かヒントが得られるかもしれません。

旅道場の話は前半の1時間にして、後半は僕が体験して感動したワークショップをさせて頂きたいです。集まったメンバーでテーマを変えるかもしれませんが、「目の前の仲間とプロジェクトを始めるワークショップ」をしたいと考えています。

旅道場｜旅をもっと楽しくする音声ガイドブック

旅道場が生まれた経緯（今回のタイ旅行の目的－タイ釣り旅行記）
