１からわかる仏教講座は予備知識がなくても基礎から学べます。

21世紀は心の世紀と言われますが、毎日がすばらしいという人もあれば、何をやってもなんだかむなしいという人もあります。

その差は一体どこにあるのでしょうか？

仕事・収入・交友関係・家族・才能によらず、幸せそうな人もあれば、人にはわからないような悩みを抱えている人もあります。

そんな一人一人の生きる意味を明らかにされ、幸せとは何かをわかりやすく教えられているのが仏教です。

仏教はとにかく難しそう、日常に取り入れるのは大変そう、と思われがちですがそうではありません。

今まで解決できなかったような悩みもすっきりするヒントが仏教には教えられています。

◆講座内容◆

「人は何のために生まれ、生きていくのか」を
お釈迦様が説かれた仏教から学びます。

お経の中には生きる意味を知らされたよろこびを
「人身受け難し、今已に受く」と説かれています。

生きる意味がはっきりすれば勉強も仕事も、
健康管理もこのため、と日常のそれぞれが意味を持ち
心から充実した毎日が送れます。

◆過去参加された方の感想◆

日頃から何のために生きているのかを考えることが多いのですが、生まれてきたのはこの為、人生の目的を達成することはできる、本当の幸せがあるのだ、ということを知れたことは大きいです。ありがとうございます。（30代・男性）

多くの人々はどう生きるか、ということで苦労しているように思いますが、「なぜ生きるか」ということを考え始めれば、それまでの生き方と違った方向を生きられるのだと思います。（女性）

会社の上司と人間関係がうまくいかず、 転職も考えていたのですが、布施の精神を知ってから心が大分軽くなり、また頑張れそうです。思い切って参加してみて良かったです。 有難うございました。（20代・女性）

◆講師◆

飯田隆史講師

「文化や世代を越えて仏教のすばらしさを伝えたい」と１５年以上にわたり一からわかりやすい仏教の勉強会を開催。関東、東海、関西、北陸地方の国内だけに留まらず、マレーシアや韓国にも招待され、年間の講座回数は１００回以上に上ります。
