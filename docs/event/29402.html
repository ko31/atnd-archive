概要

ゲームの理論をビジネスに応用する、今旬のキーワード「ゲーミフィケーション」。
本セミナーでは、その基礎知識から実践まで、導入事例をまじえながら、思わず訪問したくなるECサイト運用のヒントが学べます。ECだけではなくビジネスに幅広く活用できるゲーミフィケーションについてこの機会に学んでみませんか。

●ECサイトにテコ入れしたいけどいい方法はない？
そうお考えの方に、ゲーミフィケーションをオススメします。ECサイトの売り上げアップ、
ブランディングにも役立ちます！

●そもそも、ゲーミフィケーションってなに？
ゲームではないアプリケーションやウェブサイトにゲームの理論を応用。
ユーザに楽しんでもらうことでファン作りにもつながるため、双方にとって大きなメリットを生む手法です。

●ECサイトにゲーミフィケーションって難しいんでしょ？
本セミナーでは、株式会社ゆめみ / 株式会社神戸デジタル・ラボ / at+link（エーティーリンク）の３社が、
活用事例などを通しながら「ECサイトで実践するためのうまい方法」を余すことなくお伝えします。

ーこんな方にオススメー
■ECサイト運営担当者の方
■起業家・経営者の方
■ゲーミフィケーションについて知りたい方
