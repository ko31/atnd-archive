Sharing for the Action ! vol.1

「被災地を支援したい！」

このイベントはそう考えている人が集まり、「自分にできることはなんだろう？」についてみんなで考える交流会です！


途中参加＆途中退場OKです！！気軽にふらっと立ち寄る感覚でご参加ください☆


学生でも出来る事。社会人でも出来る事。
さらに大きなアクションにしていくため、協力できることを全員で考え、見つけていきましょう。すでに動いている団体の活動紹介に加え、現地で活動をしてきた方によるレポート報告も行います。

詳細：http://somode.info/social/share/



○日時　2011年4月21日(木)　16:00 – 19:00


○プログラム

1.パネラー自己紹介＆今回の趣旨説明

2.現地で活動をしてきたボランティアによるレポート報告

3.被災地支援プロジェクトのプレゼン
　400L for Hope　Shotaro Uehara -テントセン,日本ブラインドサッカー協会 ,トライバルメディアハウス
　Raise A Hope Yusuke Yamanaka -somode
　チームフッコー　上村ゆうや -TABLE FOR TWO,ちきん

4.交流会

プレゼンター紹介


Shotaro Uehara｜twitter,facebook｜400L for Hope

慶應義塾大学４年　NPO法人ETIC.を通じて、日本ブラインドサッカー協会でインターン中。
今回の震災に際し、学生の仲間とともに被災地支援活動「400L for Hope」を立ち上げ、学生1,000人の献血と、100万円の義援金を募っている。

Yusuke Yamanaka｜twitter,facebook｜Raise A Hope

企業でwebマーケティングを専門に行う傍ら、ソーシャルミュージックレーベルsomodeを運営。
社会貢献と音楽を掛け合わせた、活動を行っている。 今回は阪神大震災の経験から、復興時に大きな貢献をしたメッセージと音を届けたい、という 想いから「Raise A Hope」を立ち上げる。

上村ゆうや｜twitter｜チームフッコー

「１人の１００歩より、１００人の１歩ずつ。」東京外国語大学大学院　修士１年。
NPO法人 TABLE FOR TWOでインターン中。今回の地震に対して『チームフッコー』をスタートさせ、ジャストギビングでは97,600円の寄付金を集めた。
