description

8月4日に開催されたAndroidカフェのスピンオフとして、
毎週木曜日にAndroidミニカフェを開催いたします。

対象者は、Androidカフェと同じくAndroidを開発をしているプログラマー、
エンジニアの方ほか、デザイナーの方、学生さん、はたまた
最近スマートフォンを利用を始めた方、検討中の方でも、
「Androidに興味、関心」があれば参加OKです。

ミニカフェでは、先日カフェへ御来場いただいた方からのアンケートの意見も踏まえながら、今後のカフェの運営についてや、勉強会の企画、
コンテストを見据えた共同アプリ製作、その他皆さんで、アイデアを出して、一緒にAndroidで実現したいことを、話しあう気軽な場にしたいと考えています。

◆開催概要
・日時：８月１１日（木）１９：００～２１：００

・会場：Androidオープンラボ 　八王子市天神町16－１　エイビット天神町ビル
