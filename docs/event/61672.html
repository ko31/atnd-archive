リクルートグループ唯一の「人材採用総合サービス」を展開するリクルートジョブズが、UI・UXデザイナーの即戦力人材を募集！
リクルートジョブズが、クリーク･アンド･リバー社運営による採用説明会＆選考会を開催致します。説明会をお聞きになって選考会への参加を希望される方は、書類選考免除で全ての方とお会いさせていただきます。

当日は、現場で活躍するリクルートジョブズの社員が、実際の業務内容や社内の雰囲気を赤裸々にお話するトークセッションも予定しております。
説明会に参加することで業務内容だけでなく、ご自身が実際に働くイメージを持って頂けるかと思います。興味はあっても応募を迷われている場合も、まずは話を聞いてみるというスタンスでも結構です。

リクルートジョブズの事業に少しでも共感された方々に、なるべく多くお会いしたいと思っております。どうぞ、この機会にお気軽にご参加ください。

最新テクノロジーとカスタマーインサイト、
クライアント接点を掛け合わせ、日本の社会課題解決にチャレンジする醍醐味
2012年10月のリクルートグループの分社化に伴い発足した、リクルートジョブズ。
目指したい社会として「シゴトとヒトが良い関係で、一人ひとりの未来につながっていく社会」を掲げ、リクルートグループの中でアルバイト、パート、派遣から正社員まで、幅広い雇用領域を担当し、多種多様な雇用機会を創出しています。
同社は今後、積極的にテクノロジーを活用し、日本の労働市場に対峙していてまいります。

今回、改めてリクルートジョブズの特徴や、クリエイター視点から見た働く魅力についてご紹介します。

リクルートジョブズの特徴1
グループで唯一、多種多様な雇用形態における人材採用に関する総合サービスを展開
リクルートジョブズが運営するサービスは「タウンワーク」「フロム・エー ナビ」「はたらいく」「とらばーゆ」など、アルバイト、パート、派遣から正社員まで、『グループで唯一、多種多様な雇用形態における人材採用に関する総合サービス』を展開しています。

対峙しているマーケット環境は、少子高齢化に伴う労働力人口が減少し、2020年には労働力人口の半数以上が45歳以上になる、という予測が出ています。同時に産業構造の変化やテクノロジーの進化など私たちを取り巻く環境は大きく変化していきます。
その変化していく労働市場と対峙し、多種多様な働き方の実現に貢献していきたいと考えています。

リクルートジョブズの特徴2
毎月のように新規事業・サービスの企画・立ち上げを実施！
一人ひとりが企画～開発リリースまでワンストップでチャレンジできる
リクルートグループでは毎月、グループ横断で新規事業の企画提案を行う制度があり、全社員が自由にアイデアを持ち寄り、応募することができます。さらにリクルートジョブズ内でも事業開発室で新規事業の立ち上げや他部署でも新たなサービスの開発など、積極的に取り組んでいます。

今後、少子高齢化による人口の減少によって深刻な人手不足が社会問題化しつつあります。そうした重要な社会課題を解決することに、リクルートジョブズは本気で取り組んでいきます。
そのために今のリクルートジョブズには社員一人ひとりがクライアント、カスタマーの課題を解決するサービス開発を企画～実現までワンストップで行う環境があります。

リクルートジョブズの特徴3
2014年4月にIT戦略室が発足！
「リーン」「アジャイル」「スクラム」等イノベーションを生み出す開発環境
リクルートグループはIT戦略を明確に打ち出し、リクルートジョブズでも2014年4月、新たにIT戦略室を開設。現在、少数精鋭ながらも様々な新しい事業の開発を行っています。

メンバーの半数が中途入社。中途入社の方も今まで培ってきた経験を存分に活かしていただける環境や体制が整っています。
前職は大手SIからWebサービス企業まで多種多彩で、全労働力人口約6500万人のカスタマーが対象となるサービスを提供し、社会に広く貢献できるリクルートジョブズに魅力を感じて、続々と仲間入りしています。その中で現在取り組んでいるのは「リーンスタートアップ」「大規模開発のアジャイル＆スクラム導入」といった、先進的な開発手法が目白押し。

社会的な意義だけでなく、一人のIT技術者としても非常に魅力的でかつ刺激的な開発環境の中でチャレンジできるのが、リクルートジョブズで活躍する最大の魅力なのです。
