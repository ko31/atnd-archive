イベント概要

ネット系ベンチャーを起業する・働くには、エンジニア、デザイナーであることがその条件に挙げられるほど、最近はエンジニア・デザイナーを評価する流れがあります。

ですが、成功しているベンチャー企業は、全てエンジニア・デザイナーが経営陣として辣腕を振るっているのでしょうか。

答えはもちろんNOです。

今回は、エンジニア、デザイナー出身ではありませんが、事業を着実に成長させているベンチャー社長をお招きし、コーディング、デザインスキルがない経営者の会社運営について、エンジニア・デザイナーをどのように採用しチームを作るか、及びそれを踏まえて経営者として意識して取り組んでいることについて、赤裸々にお話いただきます。

2010年に株式会社 FrogAppsを設立し、代表に就任。外食とネットの幸せな関係を目指し、2011年10月にソーシャルアプリ「miil」をリリース。
