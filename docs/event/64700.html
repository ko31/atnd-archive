Outline

This meetup is for everybody wanting to learn a little about Rust - new, promising system programming language. Everybody form beginners to total pro is welcome as we are still far from experts and are willing to learn from more knowledgeable, as well as share everything we've learned so far about Rust. Hopefully we will be able to hack some fun stuff together!

Requirements

Rust-y computer. On how to install Rust please follow instructions on the Rust official site.

Entrance fee

None

趣旨

Rustという最新の期待されたシステムプログラミング言語勉強会です。初心者から上級者まで問わず募集をしていますので、興味をもっている方が来てくださることを心からお待ちしております。私達自身Rust上級者というわけではないので、参加者の皆様と一緒にRustについて勉強をしていきたいと考えています！一緒にRustで面白いものを作りましょう！

持ち物

Rustがインストールされたノートパソコン。インストール手順は Rustウェッブサイト を参照してください。

参加費

無料
