IIJ Technical NIGHTは、2003年から毎年11月に3日間にわたって開催している「IIJ Technical WEEK」をカジュアルでコンパクトにしたイベントです。

2回目となる今回のテーマは「AIとセキュリティって実際のところどうなの？」です。
「AIとセキュリティ」の組み合わせについては、昨今、色々なところで色々な事が語られていますが、一方で言葉だけが一人歩きしている気配もあります。
そこで本イベントでは、IIJが考える「AIとセキュリティ」についてご紹介すると共に、IIJの取り組みが単なるかけ声だけでなく地に足のついたものであることをお伝えできればと思っています。

講演終了後には軽食を囲んでの話者との交流会もあります。みなさま、どうぞお気軽にご参加ください！

※毎年恒例の「IIJ Technical WEEK」は今年も11月に開催を予定しています。

18:30～ - 受付開始

※受付時間は18:30～19:30までとなります。それ以降にお越しの方は2階受付にて、携帯電話080-1108-6349までご連絡ください。係りの者が参ります。
19:00～20:00 60分 Session1：
セキュリティインテリジェンスに対するIIJの取り組み
講演者：穴吹 健一（IIJ セキュリティ本部 セキュリティビジネス推進部 セキュリティオペレーションセンター）

IIJがこれまで取り組んできたデータ分析案件の事例や機械学習の検証内容を紹介するとともにそれらの技術をどのようにセキュリティに応用していくかについてお話しします。

Session2：
IIJの大規模セキュリティ分析基盤のアーキテクチャ
講演者：宮川 健（IIJ セキュリティ本部 セキュリティビジネス推進部 セキュリティオペレーションセンター）

IIJではIIJの各種サービスのログを分析することを通じて、独自のセキュリティインテリジェンスを獲得し、そのインテリジェンスを各種サービスの運用チームや利用者に提供することを目指しています。
この独自インテリジェンスを獲得するための分析基盤のアーキテクチャのご紹介をしつつ、なぜそのようなアーキテクチャを選択したのか、設計/実装/運用上の勘所などについて解説します（ビッグデータ系技術の概説を含む）。
20:00～21:00 60分 Beer Bust

ビールとピザをつまみながら、講演者と参加者の皆さんで気軽に意見交換をしましょう。会費は無料で途中退席もできますので、お時間の許す限り、ぜひお気軽にご参加ください。（ソフトドリンクもご用意しています）
