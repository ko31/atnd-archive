12/25まで、毎日1人が1つPHPのTipsを自分のblogに書いていくイベントです。
去年実施した2010年のまとめは以下。

「PHP Advent Calendar jp 2010 まとめ」
http://www.1×1.jp/blog/2010/12/php_advent_calendar_jp_2010_summary.html

■

書くTipsは、10分で考えて、5分で書ける内容で ok です。

普段 PHP を使っていて、便利な機能、好きなテクニック、興味のあるネタなど、
さらっと書いて下さい。（がっつり書いても良いです:D）

参加方法

  1. 当番表を Google Docs に作っているので、記事をアップしようと思う日の行にTwitter ID を書いて下さい。今回、電子出版を行いたいと思うので掲載可否についても合わせてお願いします。
    https://docs.google.com/spreadsheet/ccc?key=0Aq2CXbU2cA_sdHpTd0RGNkVaT1p4ZXllTjZURnp6T2c
  2. 表明した日に記事をアップする。（記事は前日までに書いておくと当日慌てなくて良いですよ！）
  3. アップした後は下記コメント欄に記事URLを投稿して下さい。

※ATNDの参加表明無くても当番表に記載があれば ok です。
※当番日は先に書いたもの勝ちなのでお早めに:D
