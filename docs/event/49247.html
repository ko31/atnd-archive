title トヨタ＆ビジネススクールで学んだ「１枚フレームワーク」仕事術
catch あらゆる業界・職種の方が身につけておくべき「１枚」仕事術の意味・意義をお伝えします！
description

●受講対象者
・報告書や自分の考えを、常に紙１枚程度の分量にまとめられるようになりたい方
・自分の考えをロジカルにまとめられる思考の型を身につけたい方
・相手に伝わる、上司を説得できる提案・報告・プレゼン力を高めたい方

●習得知識
・一流企業に勤める「仕事ができる人」に共通する３つの力とは？
・３つの力を自力で高めていける「１枚フレームワーク」とは？
・「１枚フレームワーク」の使い方・書き方とは？

●趣旨等
□トヨタはなぜ「紙１枚」なのか？MBAスクールはなぜペーパーレスなのか？
２つの業界トップ企業での勤務経験を交え、あらゆる業界・職種の方が身につけておくべき「１枚」仕事術の意味・意義をお伝えします。

□なぜ今、「１枚フレームワーク」仕事術なのか？
ネット・スマホ・タブレット普及などの文脈から、今ビジネス環境で何が起きているかを明らかにします。

□「１枚フレーム」の書き方・使い方
明日からさっそく職場で実践できるよう、体験重視のワークを多数行います。整理する力・考える力・伝える力をその場で鍛えます。

●演習内容
「１枚フレームワーク」の書き方を学ぶワークを多数行います
※受講者持参物：ノートおよび色ペン（緑・青・赤の最低３色）

●プログラム
１． はじめに：トヨタで学んだ「わかるかどうかvsできるかどうか」
　（１）セミナー・研修・講演等を受講するうえで最も大切なこと、それは「○○」
　（２）トヨタで働きながら観察・発見した、できるトヨタマンの共通点とは？
　（３）トヨタ用語「なぜを５回繰り返す」を、ホワイトカラーが
　　　　　盲目的に参考にしてはいけない理由

２．なぜトヨタは「１枚」なのか？
　（１）『ＰＲＥＳＩＤＥＮＴ』でも特集が組まれた！
　　　　　トヨタの「カイゼン」を生み出す「紙１枚」文化
　（２）入社２日目、４年目、８年目、トヨタの研修では何を学ぶ？
　（３）２兆円の利益を叩きだす企業で働く社員が、
　　　　　１枚を書きながら鍛えている３つの力とは？

３．なぜMBAスクールはペーパーレスなのか？
　（１）転職１週間後の衝撃：「浅田さん、紙つくってる暇があったら仕事してください」
　（２）前置きだけで終了する１０秒会議：
　　　　　「あの～来月のｗｅｂサイトのコンテンツなんですが・・」
　（３）社員の４人に一人はＭＢＡを持っている職場
　　　　　ｖｓ全社員のうち４人しかＭＢＡを持っていない職場

４．なぜ「伝わる組織コミュニケーション」をしたいなら「１枚」なのか？
　（１）ワーク１：東京ディズニーランドの道案内
　（２）２０１３年ビジネス書ランキングが「コミュニケーション」系の本ばかりの理由
　（３）トヨタで学んだコミュニケーションの本質的な知恵：「伝えたかったら、○○○な！」

５．なぜ今、「１枚」なのか？
　（１）スマホ・タブレット端末が普及することで失われた「ある力」とは？
　（２）ネットの普及・Ｇｏｏｇｌｅ検索が一般化したことで失われた「ある力」とは？
　（３）池上彰さんのニュース解説が年越し特番で組まれてしまう異様な国：日本

６．仕事の型を身につける「１枚」フレームワーク：「エクセル１」＆「ロジック３」
　（１）「エクセル１」＆「ロジック３」と「トヨタの１枚」の共通点
　（２）「１枚」フレームワークの書き方・使い方・書くコツ
　（３）ワーク２：「エクセル１」＆「ロジック３」で
　　　　　仕事内容紹介・仕事の問題解決、優先順位付け、等

７．まとめ／質疑応答／名刺交換
