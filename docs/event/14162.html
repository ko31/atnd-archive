第2回　北九州デザインシンポジウム

イベント ユーザー
event_id 14162
title 第2回　北九州デザインシンポジウム
catch デザインと売れるしくみづくり～マーケティング発想で新領域を開拓する～
description

北九州市では、市内中小企業がデザインの重要性を再認識し、独自性を活かしたデザインの導入を行い、自社製品の高付加価値化や競争力強化を図ることを目指した取り組みを行なっています。

　その一環として、北九州市と西日本工業大学が共同で「第2回北九州デザインシンポジウム」を開催し、講演とパネルディスカッションを通して、デザインとマーケティングの重要性について考えていきたいと思います。

日時
　平成23年3月30日（水曜日）15時～17時30分
場所
西日本工業大学　小倉キャンパス303　講義室（北九州市小倉北区室町　　
　　1-2-11）（リバーウォーク北九州・大学棟内）
内容
第1部　講演
「デザイン活用の新たな方向を探る」

九州大学大学院芸術工学研究院　教授　森田昌嗣氏
