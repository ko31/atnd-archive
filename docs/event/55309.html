第4回のテーマ「ポートフォリオ」

「普段から作らなければと思っているけど中々忙しくて手が出せない…」

ポートフォリオってこんなイメージですね。

だったら勉強会でみんなでアイデアを出しながら作ってみませんか？

今回のポートフォリオ制作は

  • WEB用のポートフォリオサイト
  • 対面用の営業資料としてのポートフォリオ

の2種類のどちらかを任意で決めていただき、ワークショップで制作します。

5人1テーブルで参加者同士、ワイワイと「こんな見せ方はどう？」「アイデアはどう？」と話しながら楽しく制作できたらと思います。

　

勉強会の概要と構成

勉強会では、有志による発表を聞く時間と、全員で参加するワークショップ（ハンズオン）が必ず含まれる構成にしています。

WEBデザイナーの能力は大きく分けてデザイン、コードの2種に分かれるため、２部構成で双方に発表とワークショップを含めるようにしています。

※今回のポートフォリオ制作は時間がかかる分野ですので、1度のワークショップで行います。

知識を仕入れること、実践形式のワークショップを両方盛り込むことで、幅広い参加者に満足してもらえる形を目指します。

発足の背景

  • 関西にデザイナーの勉強会が少ない
  • 勉強会の目的は情報共有かレベルアップである
  • レベルアップは大きく分けて「新技術の習得」「既存技術の効率化」である
  • たいていの勉強会は「新技術の習得」や「個人的発表」に偏っているので、このバランスがとれるものがほしい
  • バランスをとってワークショップを入れれば、定性的な分野であるデザインなら初心者も上級者も学べる
  • 定期開催で関西のWEBデザイナーの「基地」を目指す
