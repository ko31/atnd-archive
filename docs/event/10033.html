title more trees night vol.4
catch -TOUCH WOOD SH-08C Debut special night-
description 美味しいお食事とお酒を頂きながら、
ゲストと一緒に森林について考えるmore trees night。
4回目となる今回は、
先日発表となった、NTTドコモの2010-11冬春モデル。
その中でも特に注目を浴びている「TOUCH WOOD SH-08C」の開発秘話などを、
NTTドコモの担当者さんから伺います。
さらに当日は、SH-08Cのモックアップも展示いたします。


　◇ 追加ゲスト決定！！ ◇
　シャープ株式会社、オリンパス株式会社のご担当者さんも参戦決定です。
　「TOUCH WOOD SH-08C」の開発にたずさわった4社が勢揃いいたします！

日時：11月29日（月）19:30 OPEN / 20:00 START
会場： Restaurant-I
（東京都渋谷区神宮前1-4-20 パークコート神宮前1F）

ゲスト：岡野 令（株式会社NTTドコモ）
　　　　木戸 貴之（シャープ株式会社）
　　　　鈴木達哉（元オリンパス株式会社理事）
ファシリテーター：水谷 伸吉（more trees事務局長）
会費：6,000円。
　more treesに関わる多彩なゲストによるトークショーの1部と、
　ゲストを交えてお料理やお酒を楽しみながらの歓談タイムの2部構成によるイベントです。
　お食事は、フィンガーフードとフリードリンクになります。
　※会費の一部は、more treesの森林保全活動に役立てられます。
定員：30名限定（定員を超えた場合は抽選となります。あらかじめご了承ください。）

◇ 岡野 令さんプロフィール：
2000年慶應義塾大学経済学部卒業、株式会社NTTドコモ入社。
主に、佐藤可士和氏やStudio CONRAN、EMILIO PUCCI・kate spade・marimekko・ANTEPRIMA・BURTON など、
外部デザイナー/ブランドとのコラボレーション商品の企画、プロモーションまでを含めた全体のプロデュースを行う。
これまで手掛けた商品に、N702iD・MUSIC PORTER X・70Xiμシリーズ・SH-06A NERV・N-07A・2010年夏/冬のコラボレーションモデル等。
また、夏野剛氏のもと、キッズケータイの商品企画にも参加。プロモーションの分野では、
キッズケータイの守る・学ぶのコンセプトのもと、子どもたちが一流のアーティストやミュージシャンと触れ、
創造性や感性を育むためのプロジェクト「クリエイティブキッズ」の企画を行い、
現在も「みんなの特別授業」として継続中。
受賞歴として、グッドデザイン賞、モバイルプロジェクト・アワード等。
2009年にはヒノキの間伐材を利用した携帯電話試作機「TOUCH WOOD」を、
森林保全を呼びかけるmore treesや、木材圧縮成形加工技術を持つオリンパス、シャープと協力して開発。
CEATEC JAPANへの出展により、新しいエコへのアプローチとして国内外を問わず高い反響を呼び、2011年春モデルとして商品化を実現。


◇ 木戸 貴之さんプロフィール：
1984年明石工業高等専門学校電気工学科卒業、シャープ株式会社入社。
主に、1989年からAV機器の商品企画業務に携わり、1997年からはPHSの商品企画を担当、
以後、NTTドコモ向けの携帯電話、FOMA、スマートフォンを手掛ける。
これまで携わった商品に、PHSのパルディオ641s、カメラ付き携帯電話の初期のmova SH251i、
FOMA SH901iSなどFOMA SH906i以降は、「TOUCH WOOD」など様々な試作機も含め、NTT
ドコモ向けすべてのプロジェクトに関わる。

◇ 鈴木 達哉さんプロフィール：
日本大学生産工学部機械工学科卒業、1970年オリンパス光学株式会社入社、開発部門に勤務、
1986年世界初の生活防水カメラ「AF-1」、1988年には限定2万台が予約段階で全て完売となった
「O（オー）・プロダクツ」、1991年には単独機種で500万台、シリーズ全体で2500万台以上という
売り上げを記録したμ（ミュー）シリーズの開発責任者として数々のヒット商品を世の中に送り出す。
開発したものがことごとくヒットしたとはいえ、数年すれば陳腐化してしまう商品に疑問を持ち
「愛着を持って長く利用できる商品を開発したい」という思いが「木」という材料にたどり着き
2003年から本格的に木材の研究を始める。
2006年2年に1度ドイツケルンで開催される世界最大のカメラショー「フォトキナ」で
圧縮木材技術で製作した「木の外装デジタルカメラ」を参考出展する。
2008年「木材の三次元圧縮加工技術の研究と精密機器部材への応用」というテーマで
日本塑性加工学会賞「技術開発賞」を受賞。
2009年、NTTドコモ、more trees、シャープと協業でCEATEC JAPAN 2009「TOUCH WOOD携帯」を参考出展。
2009年12月定年退職。
