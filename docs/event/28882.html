イベント概要

注目のスタートアップ4社のTOPが、エンジニア・クリエイター向けにサービス開発やチームビルディング、
マネタイズについて、リアルな本音を語るパネルディスカッションを実施します。

【パネル参加者】
エンジニア・クリエイターのためのクラウドソーシングサービス『crowdworks』
自分が行きたいと思う旅を投稿し、それに共感した人達で旅に出るサービス『trippiece』
時間割を学生同士で共有できる、大学生のための革新的な時間割アプリ『すごい時間割』
あなたの代わりに記憶し、知りたいことを教えてくれる、情報、知識の整理・共有サービス『vingow』

いつかは事業を起こしたいとお考えの方、スタートアップのサービス開発に興味をお持ちの方、
エンジニア・クリエイターとして面白いサービス作りを学びたいとお考えの方は、ぜひご参加ください。

勉強会終了後は、同会場にて懇親会も開催。
パネル参加者や聴講者との情報交換・交流を深めていただければと思います。
