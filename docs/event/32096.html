第2回関西ソーシャルゲーム勉強会

この勉強会の趣旨

関西方面でソーシャルゲームを作っているみなさんに交流の場を作りたい！ というのが開催の理由です。
関西でソーシャルゲーム開発に関係している方々が、ぜひこの機会に情報交換 ができたらいいなと思います！
今後もこうした機会を設けていきたいと考えていますので、よろしくお願いします!!

※諸注意

  • 会場はサイバーエージェントさんの会議室をお借りしますので、会場とトイレ以外の場所には立ち入らないで下さい。
  • LANケーブルやコンセント等は、スタッフと講演者のみが使用します。一般参加者の方のご利用はご遠慮ください。
  • Wi-Fiは会場にて提供いたします。電波の輻輳（ふくそう）を防ぐため個人用Wi-Fiルータのご利用はお控え下さい。
  • ビルの安全の都合上、１時間を経過した遅刻の場合会場内には入れませんので、よろしく御願いいたします。

参加資格について

特に必要ありません。下記に該当される方歓迎します。

  • なんらかの形でソーシャルゲーム開発に関係している方！
  • これからソーシャルゲーム開発やってみたいという方！
  • LTや講演をやってみたい！
