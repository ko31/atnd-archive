第1講　電気自動車機械部品の最新研究動向～モータを中心に～
【13:00～14:30】

【講演趣旨】
電気自動車の主要部品である電気モータは、古くから多くの研究がなされてきた。原理的には
19世紀に発明されており、それ以降大きな変革はないが改良が続けられてきた。特に、
小型化の進歩は著しい。また、近年の自動車はインテリジェント化が進み、
多くのアクチュエータが使われている。
アクチュエータには、モータ型とソレノイド型に大きく分かれる。
本セミナーでは、電気自動車のパワーユニットとして使われるモータについて、インホイルモータを
含めた最新動向について述べる。さらにアクチュエータとして使われる小型モータとソレノイドに
ついても言及し、自動車の基本性能に応用されている事例を示す。
1.モータの基礎
2.モータの種類
3.自動車へのモータ搭載例
4.自動車のバイワイヤとアクチュエータ
5.モータ型アクチュエータ
6.ソレノイド型アクチュエータ
7.自動車のインテリジェント化への応用事例

第2講　自動車用電動真空ポンプの開発と電気自動車への搭載事例
【14:45～15:45】

【講演趣旨】
次世代車両として注目されている電気自動車であるが、広く普及させるには充電等のインフラ整備の
課題だけでなく、既存資産を活用出来る事も必要である。弊社では、従来多用されている負圧を
利用したブレーキシステムが、電気自動車では利用が出来なくなる課題に着目し、電動真空ポンプを
開発した。また、電気自動車以外にもハイブリッド自動車、ディーゼルエンジン車等への需要も
高まっている。本セミナーでは、電気自動車のブレーキシステムと電動真空ポンプの関係、弊社の
電動真空ポンプ開発への取り組みと電動真空ポンプの採用事例を紹介する。
1.車両のブレーキ構造の概略
2.電動真空ポンプの必要性
3.ポンプの寿命を延ばす為のポイント
4.商品性を確保する為のポイント
5.電動真空ポンプを使用したエネルギー節約の考察
6.搭載事例

第3講　EV/HEVに向けたカーエアコン・コンプレッサーの開発
【15:50～16:50】

【講演趣旨】
小型デバイスから電気自動車に至るまで多様な磁性材料が使われており、近年、その中でも
インダクタやリアクトルには金属粉末からなる圧粉磁心が多く用いられている。これら磁心には
大電流通電による磁気飽和を抑制できる高い飽和磁化Bsと、磁気的にも低損失な特性を付与させることで
高い効率特性を実現することが強く求められる。
そこで理論的には結晶磁気異方性をもたない磁性材料である非晶質金属磁性合金、その中でも
一際高いガラス形成能を有し優れた非晶質安定性を示す金属ガラスに着目し、FePBNb系の新しい鉄基の
磁性金属ガラス合金粉末であるセンティクスを開発、PC向け高効率インダクタ用磁心材料として実用化した。
1.はじめに
・金属磁性材料の位置づけ
2.パワー系軟磁性材料の位置づけと開発指針
3.金属の歴史と磁性材料
4.金属ガラスと磁気的性質
5.鉄基FePBNb系金属ガラス合金粉末 “ SENNTIX ”
・基本合金設計　
・金属ガラス化　
・金属ガラス合金粉末化
