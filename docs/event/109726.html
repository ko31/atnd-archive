優秀なITエンジニアを確保したくても人手不足に悩む企業が多い近年、ベトナムでのITエンジニアを活用した新規サービスの開発手掛ける専門家達が、自らの経験を元にした『イノベーションを生み出すグローバル開発チームの作り方』についてお話をします。

上流工程専門のシステム開発やオフショアマネジメントを手掛けるGrrow社、ハノイにて20名の規模で開発を手掛けているスクーティー社、ホーチミンにて100名の規模で開発を行っているVitalify Asia社、3社それぞれの視点や経験からグローバル開発チーム構築にあたっての成功･失敗のポイントについて語ります。

◆こんな方にお勧め ・新規事業（ITサービス）を開発したい方 ・既存のITサービスを改善したい方 ・グローバル開発チームを構築したい方

◆開催日時： 2019年11月6日（水）16:45受付開始、講演17:00～19:00

◆開催場所： 東京都台東区上野1-17-6 広小路ビル7F コガソフトウェア株式会社内の会議室 https://goo.gl/maps/188CyJoLQ9oWXjY67 JR御徒町駅　徒歩5分 地下鉄上野広小路駅/上野御徒町駅　徒歩3分

◆講演内容 （１）17:00～17:30 新規事業の立上げにおけるPoCの在り方／考え方、システム開発の観点について 講師：株式会社Grrow　代表取締役社長兼CEO　光部氏

（２）17:30～18:00 新規事業をベトナムの開発チームと立ち上げる際の注意点、成功と失敗のポイント 講師：株式会社スクーティー　代表取締役社長　掛谷氏

（３）18:00～18:30 新規事業の立ち上げを可能とする開発チームの構築、100人規模の組織で取り組んでいること 講師：Vitalify Asia Co., Ltd.　PMO Manager　加藤氏

（４）18:30～19:00 3講師によるディスカッションと質疑応答

◆参加費用：無料　※名刺をご持参ください。

◆募集人数：最大30名様

◆ご注意 講演内容は、都合により変更となる場合があります。 また、ネットワークビジネスの勧誘など本セミナーの趣旨に添わない方の参加はお断りする場合があります。

◆本件の問い合わせ先 株式会社バイタリフィ　営業統括部　稲垣、石黒、酒井 info@vitalify.jp 　https://vitalify.jp/vietnam_offshore/contact/　　 TEL：03-5428-6346
