Pepperと遊ぼう!!お台場のコワーキングスペースMONOによるPepper体験セミナー

Pepperくんと遊ぼう

お台場のコワーキングスペースMOMOにPepperがやってきました。 Pepperと一緒に遊びませんか！Pepperに興味がある方、開発してみたい方、冬休みで子どもと一緒にロボットと遊びたい方大歓迎です。 本イベントではPepperが出来ること、Pepperの動かし方、開発の仕方、実際どのように活用しているかをご紹介するセミナーです。Pepperに興味があるかたは是非遊びに来てください。

こんな方にオススメ

  • Pepperを見た事がないけど興味がある人
  • Pepperの購入を検討している人
  • Pepperのアプリ開発に興味がある人
  • 新しい事に興味があること
