何をやるの？

フォトショップを使えるようになりたい人が、公式サイトのチュートリアルを必死こいてこなしていく作業会です。 https://helpx.adobe.com/jp/photoshop/tutorials.html

もし他にやりたいことがある場合、別の課題をやっていただいても構いません。

講義とかはありません。主催者も超初心者です。 先生はいませんが、参加者の中にフォトショ歴の長い人がいます。聞いたらたぶん助けてくれます。

やりたいチュートリアルは自分で決めて構いません。 ちなみに、私の個人的目標は「Webバナーの作成」です。

スケジュール

13:30 開始　簡単な自己紹介 公式チュートリアルから自分のやりたい作業を選択

13:45 必死こいてチュートリアルを作業する。分からなくなったら周りに聞いてみる。

15:10 作業終了 みんなで知識をシェア、フィードバック

15:30 完全終了

対象者

  • フォトショップを身につけたい方
  • 一人で作業してると心が折れそうになる方
  • たまに他の人と使い方を相談してみたい方
  • ゆるーくフォトショ仲間を探している方

場所

コワーキングスペース「タネマキ」 横浜駅から徒歩8分。相鉄線平沼橋駅からは徒歩3分。 https://tane-maki.net/access/

〒220-0073 神奈川県横浜市西区岡野1-3-10 サニーコート横浜 1F

費用

  • 参加費は無料ですが、会場費だけ現地でお支払いください。
  • フォトショップ作業会だけ参加の方は2時間500円でご利用いただけます。
  • それより前に来て一人で作業したい、残って作業したい、という方は1000円で9:00-18:00の間ご利用いただけます。 （朝から作業される場合、会場のタネマキは12:00-13:00は一旦お昼休憩でクローズしますのでご注意ください。せっかくなのでみんなでランチ行きましょう。）
  • イベント時間帯の区分を含むタネマキ月額メンバーの方は、無料で参加できます。

※参加者用に会場の席を確保しておりますので、欠席する場合はキャンセル処理を、30分以上遅刻する場合は必ず連絡ください。 　（連絡なく長時間遅刻された場合は席が確保できないことがあります。）

ゆるく楽しくやる作業会です！参加をお待ちしています！
