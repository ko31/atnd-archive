カーネル/VM探検隊を関西から見る会とは

watch kernel_vm

第六回カーネル/VM探検隊に参加したい、発表したいけど東京は遠すぎる！ということで、関西から相互にustを繋いで参加する実験的な企画。開催地が遠くて諦めていた皆さんふるってご参加ください。

会場

cd Kyoto

@takeokaさんのご厚意により株式会社アックスさんの会議室を使わせてもらえることになりました(会場への地図)。開場は当日12:30からになります。

E2BIG

50インチくらいまで伸ばしても何とかなりそうなプロジェクタを用意しました。でっかい画面でustを見れます。ust画質だけど。

発表について

chmod g+w kernel_vm

関西からの発表枠が30分程なら用意できるとのことだったので、3人程までならust中継で関西からLTができそうです。発表ネタがある人はコメントまたは@fadis_までリプライでお願いします。発表時間は準備等を含めて30分に収まるように、1人あたり5分から7分で考えています。会場の電波状況によっては十分な画質で中継が出来ないかもしれません。手持ちの無線モデムやテザリング可能なスマートフォンがある場合は持参することをオススメします。また、念のため発表資料はでっかい字で書きましょう。

ハッシュタグについて

boost::hash

ハッシュタグは#kernelvmを使いましょう。カーネル/VM関西で使用した#kernelvm_kansaiではないので注意してください。
