Rfm Code Reading #15

2012年2月4日（土）午前10時よりRfm（FileMaker API for Ruby）のソースコード読書会「Rfm Code Reading #15」を開催します。

Rfm Code Readingの概要

FM-Tokyo（FileMaker Pro 東京ユーザーズミーティング） の分科会活動の一環として、Rfm Code Readingでは、Rfmのソースコードを読み解くことにより、FileMaker ServerのカスタムWeb公開やRuby、およびRuby on Railsを使った開発のスキルアップを目指します。今回はGinjo-Rfm 2.0（https://github.com/ginjo/rfm）の新機能や変更点について確認していく予定です。

RubyおよびFileMaker ServerのカスタムWeb公開に興味のある方のご参加をお待ちしています。

開催日時

2012年2月4日（土）午前10時から正午まで
