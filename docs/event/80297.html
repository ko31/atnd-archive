◆セミナー内容
せどり転売をやっているあなたの利益を３倍にできる、
商品製作ＯＥＭインターネット販売セミナー
①転売ビジネスで伸び悩んでいた僕が、商品製作で大成功した理由
②安定的な収益を確保できる資産型収入のモデルを築くためには

⑦商品製作で、信頼できる優秀なパートナーと組む必要がある理由
　　
　
【セミナー詳細】
　
中国でオリジナル商品を作って、インターネット、特にAmazonアマゾンで販売する方法は、
　
■場所に制限されない
■時間に制限されない
■自分のペースで仕事ができる
■事業規模は自由自在
■集客が必要ない
■ライバル不在のところで販売できる
■作業時間が短時間
といった数多くのメリットがあります。
　
セミナーでは、その中国でＯＥＭで商品を作り、Amazonで販売する物販ビジネスの方法をお伝えします。

【講師プロフィール】
　　
大上 達生（おおがみ たつお）
　
Web輸入ビジネスプロデューサー、インポートエース塾塾長、オークファン講師
　
１９７４年京都生まれ　東京西国分寺在住
　
会社員として働くかたわら、2006年からインターネットを使ったビジネスに参入。
　
アフィリエイトや情報商材等に取り組むがことごとく挫折。合わせて約200万円を失う。
　
そんな中、短時間で効率的に稼げるアマゾンを使った輸入ビジネスに出会い、起業独立、多くの仲間の助けもあり、月商500万円越えをはたす。
　
2014年、自ら成果の出たインターネットを使ったビジネスで利益を上げる方法を体系化し、インターネットビジネスの中で最も短時間で稼ぎやすいアマゾンを活用した輸入を学ぶインポートエース塾を開始。
　
受講中に結果を出す塾生が続々現れ、大きな成果をあげる。
　
2015年からは自ら商品の製作と販売を開始し、トップ商品製作プロデューサーである小笠原満と共に、インポートエース塾にて、商品製作OEM講座を開始している。
