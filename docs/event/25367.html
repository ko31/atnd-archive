Ruby Programmer School

あなたも、スクールで学んで、一流う企業や地元ＩＴ企業に
就職しませんか？（学歴不問です）

≪４月スタート ≫

　　福岡市の一流企業や、ＩＴ企業と多数提携、
　　東京本社の上場机上などＩＴ企業も受け入れ態勢。

≪ スキルゼロでも、ＯＫ ≫

　　スキルゼロの方でも、当スクールで企業で即業務ができるカリキュラムを学習します。
　　初年度より、受け入れ企業と多数提携しています。文系・理系は問いません。
　　企業が求める人材は、発想力が豊かな人、アイデア豊富な人、こつこつ努力する人、
　　まじめな方、ゲームが好きな方、何かアピールポイントを持っていれば、ＯＫです。
　　バラエティに富んだ人材を求めています。

≪ ゼロから学び、就職先企業直結のスクール ≫

Ruby Java Ruby,Java PHP,Python
４ヶ月コース ４ヶ月コース 夜間１年コース ８月から

　一流企業で業務できれば、キャリアアップも可能です。
