description

創業・起業をお考えの方や間もない方は必見です!!!起業に関わる司法・行政のさまざまな手続きや準備すべき事柄は、非常に多くあり、ついつい見逃しがちなことや気付かないことが沢山あります。それらを放っておくと、事業の存続にも関わります。「知らなかった」では済まされないことを、スペシャリストから学んでみませんか？！
株式会社・個人事業主・ＮＰＯ団体・社団法人・・・・設立の形態問わず、どなたでもご参加ください。

【講師】
司法書士・行政書士
Ｌ.Ｃ.Ｓ合同法務事務所 代表：　河喜　多洋

平成14年3月北九州大学経済学部卒
平成15年1月行政書士試験合格
平成19年10月司法書士試験合格

北九州市門司区に事務所を開設し、法律に関わるサービスの提供を通じて、将来にわたっての「安心と喜び」をお届けすることに心掛けています。さらに、地域の方々に、親しみやすく、法律問題も早期発見をすることで早期解決に努め、ワンコインサービスを提供し、「ちょっと知りたい」に応える身近な司法・行政書士として活躍されています。
現在では、一般社団法人ソシオファンド北九州のパートナーとして、NPOをはじめとするソーシャルビジネスの中間支援団体に所属して、「北九州市をより良くする」ための活動に、そのスキルを提供しています。
