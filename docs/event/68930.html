真夏の北鎌倉大作戦！　　 　　

それは、この夏あなたが「一皮むける」ための１日…　　

思い出してみてください。

この夏休みや学生時代に、「やりたい！」と思っていた事について。

あるいは、ひとまず就職はするけど将来本当はこんなことがしたい！と思っていた事について。　　

「やってみたい！」ってせっかく思ったのに、その後にはすぐ、

　 「でも、〜だからできない」

　 「自分なんて、〜だからどうせダメ」

そんな風に思って、結局行動できていなくて。

だからますます次の行動が遠のいてしまう…　　

そんなあなたに。　　

今夏こそ一歩前進するための、とっておきの企画をご用意しました。

暑い夏に嬉しい避暑地、北鎌倉から湘南までの約20kmという長い距離を歩きながら、

ゆっくりと語り合い、最後には夏の風物詩、BBQで楽しみましょう！

この1日を過ごしたのち、きっとあなたは自分のやりたいことを、

本気でできると思えるようになって、

すぐにでも始めたくて仕方なくなるはずですよ！

私たちがあなたとゆっくり語り合う環境に、

「長距離を歩く傍ら」を選んだのには、実は訳があります。　　

それは、向かい合って話すのではなく、一緒の方を向いて並びたいから。

そしてそれ以上に、一歩を踏み出しつづけながら話すことで、

小さな一歩を踏み出すことはどんなに簡単で、大切なのかを一緒に感じたいから。　　

また、20kmもの距離を歩いた経験のある人は、どれ程いるでしょうか。

もし、やってみよう！と思い立っても、

１人だと疲れたとき、どこかで心が折れてしまうかもしれません。

でも、１人ではできそうにないことも、みんなでやれば、なんだかできそうな気がしてきませんか？　　

20kmも歩いたあと、歩いてきた海沿いの道を振り返り見渡しながら、

新たな仲間と共に夕日を眺める気持ちを想像してみてください。

夕日に輝く往路からは、ちいさな一歩の積み重ねで、こんなに大きなことが出来たんだって、きっと感じられるはずです。

そして、やりたいことに関して、アクションを起こさずにはいられなくなっていることでしょう！　　

やりたいことの「はじめの一歩」を、始めるための一歩、また一歩。

私たちと一緒に、踏み出しに来てみませんか？　　

真夏の北鎌倉大作戦、一緒に成功させましょう！

運営一同、あなたのご参加をお待ちしています。
