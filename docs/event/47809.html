title 1日で簡単マスター！ 色彩と配色 基礎セミナー
catch DTP／Web／デザイン／色彩検定対応
description

配色はセンスではなく理論です！
仕事や趣味、資格取得まで、
あなたが必ずスキルアップできる
色彩理論に基づく効果的な配色テクニックを学びます。

私たちの身の回りはいつでも色彩にあふれています。
ファッション、インテリアをはじめ、家電製品や自動車などのプロダクト、Webや広告などのデザインにおいて、
かっこいいなとか、冴えているなとか、居心地がいいなと感じる色彩もあれば、
これダサい、なんだかぱっとしない、どうも落ち着かないと感じる色彩もあります。
たとえば、同じ形をした服やクルマであっても、色彩と配色をかえるだけで、人に与える印象も変わってしまいます。

このような色彩の持つ力をもっと効果的に使いこなしてみたい、と思ったことはありませんか？

2回目の開催になる「色彩と配色 基礎セミナー」は
色彩検定協会の公式認定を受けた講師による、
文部科学省後援の「色彩検定」に基づいた講座です。
色彩検定3級に相当する知識を、たった1日で、効率よく学ぶことができます。

また、講師のデザインの現場経験に基づいたオリジナルテキストを用い、
楽しく学べる配色実習もたくさん盛り込んでいますので、
とても実践的で分かりやすい内容となっています。

色彩理論を学び、さらに配色テクニックを身に着けることで、
配色をより効果的に、より適切な印象に導くことが出来るようになるのです。
仕事や趣味、資格取得まで、あなたが必ずスキルアップできる
色彩理論に基づく効果的な配色テクニックを、1日で学びます。

※必要なものは、「はさみ」・「のり」となります

  • 最短最速で色の知識を身につけたい方
