英語で話してみよう！練習会

description

英語を勉強しているけど、なかなか話す機会がない…

そんなお悩みをお持ちの方、是非集まって英語で話してみませんか。

英語に自信がなくても、事前に準備しておけば問題ない内容です。

（主催者はTOEIC465点！^^;;　まだまだ初心者です）

【プログラム】

1. 役割紹介　司会／タイマー※／集計係※※　

2. 自己紹介（1人1分×参加者）

3. 最近あった出来事（What’s up report）（1人1分×参加者）

4. 単語クイズ（1人2問×参加者）

5 準備スピーチ（1人2分×参加者）
〜テーマ：あなたのオススメの英語勉強法〜

6.最優秀スピーチ賞発表（集計係）

7.感想（日本語）（2分）

約1時間を予定しています。

※タイマー…
スピーチの時間を計測し、時間を知らせてくれる係を募集します。

1の役割紹介で、役割を説明していただきます。

※※集計係…
5の準備スピーチにおいて、ベストスピーカー賞を皆さんに投票してもらい、

集計してくれる係を募集します。

1の役割紹介、6の発表をお願いします。

（5/3追記）—————————————————————

上記の2、３、５については即席で話せる方は準備は必要ありません。

が、4の単語クイズは事前準備が必要ですので、よろしくお願いします。

【流れ】

ある単語を選んで、その説明英文を3つほど用意します。

例えば、「book」という言葉を選んだとします。

１回目の説明

" A number of pieces of paper, usually

with words printed on them."

これで誰かが回答できれば終わり。誰も分からなければ次に
進みます。

2回目

“Contained information,stories,or poetry,for example.”

3回目

・・・

こんな感じで進めて行きます。お１人２単語ご用意ください。

用意する例文は、英英辞典の引用でOKです。もちろんご自身で考えても大丈夫です。

ご不明な点があればご遠慮なくお尋ねください。
よろしくお願いします。
