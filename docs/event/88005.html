クラウド時代の今、クラウドサービスを利用するという概念はもはや当たり前になっています。  
クラウド型グループウェアを導入している国内企業の内、約4割がMicrosoftが提供している「Office 365」を利用しています。  
  
すでにOffice 365を導入した方、これからOffice 365を導入する方、　　  
オフィスでよく利用するWord、Powerpoint、ExcelだけではないOffice 365の様々な機能説明を含め  
「O365の基礎をきちんと身につけたい」  
「O365の利活用方法を社内に浸透・促進させたい」  
「O365の新機能をうまく利活用したい」  
「O365の一番良い使い方と事例が知りたい」と思っている方に向けて  
テラスク365では、「Office 365を知ってみよう！活用してみよう！」というテーマでセミナーを開催しております。 
 
6月のセミナーでは、前半に「Office 365の基礎概要」について、  
後半では「Office 365を使った働きかた事例」についてデモを交えてお話しいたします。 
 
メイン講師として Microsoft MVP for Office 365であり、テラスク365統括部長の‘もくだいさん’ が  
さまざまな不満や不安、疑問をお持ちの方へ向けて優しく、楽しく、しっかりとレクチャーいたします。 
また、前半でOffice 365 の概要にも触れるため、「Office 365って何だろう？」「Office 365で何ができるんだろう？」  
そんな疑問をお持ちの方にもきちんと学んでいただける内容となっております。
