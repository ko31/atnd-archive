「 助成金・補助金 」という国の起業・創業支援制度をご存知ですか？

起業に興味がある方、起業して間もない方向けに、
顧客・市場を理解し顧客価値を創造するためのマーケティング、
およびビジネスプランを事業計画書として具体化することを主な内容とする
「独立・開業　創業スタートアップセミナー」を開催します。
起業して事業基盤を確立するために必須のセミナー内容です。

最近の助成金・補助金の代表的なものに、
女性や若者などの地域での起業・創業や、
後継者による家業を活かした新分野への挑戦を資金面で支援する
『創業促進補助金』があります。対象者から事業計画を募集し、
その実施に要する費用の3分の2を助成（最大200万円）するもので、
これまでに6,299件が採択されています。

そのほかにも、起業後の経営に役立てることのできる
下記のような助成金・補助金が多数あります。

■創業促進補助金：最大200万円（補助率3分の2）
新たな需要を創造する新商品・サービスを提供する創業に対して、
店舗借入費や設備費等の創業に要する費用の一部を補助

■中小企業・小規模事業者ものづくり・商業・サービス革新事業：最大1,000万円（補助率3分の2）
中小企業・小規模事業者が試作品・新商品の開発や生産プロセスの改善、
新しいサービスや販売方法の導入などの事業革新に取り組む費用に対する補助

■小規模事業者持続化補助金：最大50万円（補助率3分の2）
小規模事業者が、商工会・商工会議所と一体となって作成した経営計画に基づく、
販路開拓等に取り組む費用を補助

採択されるためには、審査の着眼点を押さえた申請書を作成する必要があります。
本セミナーでは、「補助金」についてご紹介し、採択される確率を
高めるための申請のポイントをお伝えします。
また、申請の準備として、事業化企画書を作成していただきます。

多くの皆様のご参加をお待ちしております。
