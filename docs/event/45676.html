■ Twilio API 勉強会？

クラウド電話 API Twilioが好き、もしくは興味をもっている方デベロッパーを対象にした勉強会です。
デベロッパー同士でいろんな情報交換などをしながら楽しめます。
■ 参加対象

デベロッパー（言語、分野は問いません。）

■ 勉強会のゴール！

Azure上でTwilioアプリケーションを動かしましょう！

Twilioは驚くぐらい簡単にアプリケーションと電話が繋がります。
その楽しさを皆さんと共有できればと思います。

PHPやRubyなどのプログラミングでTwilioを組む事によって遊び方は広がります！
Twilioで提供している様々なライブラリーを使ってTwilioをより楽しんでみたいと思います。
