「DJイベントって、お腹すきません？」

DJイベントなのにガッツリとカレーを食べられる、ホームパーティのようなイベント【Especia!】が開催されます！
ちなみにカレーを作るのは知る人ぞ知るレゲェDJのMEDEですｗｗｗ
靴を脱いで上がる、ゆるゆるなお店での開催なので”お友達の家に遊びに行く”ような感覚で是非ご参加ください。
気分はホームパーティなので、なんと【食べ物の持込は可！】飲み物は残念ながら不可ですが、カレーの他に食べたいものがあったらみんなで持ち寄りましょう！

※会場の広さやカレーを作る関係で、予約制をとらせていただきます。
定員は20名程度となっていますので、来たい方はお早めにご登録ください！

※アンケートについて
このイベントに申し込む際に「カレーが必要ない場合のみ×を」というアンケートがあると思いますが、
こちらの不手際で「×」しか選べず、選択無しで登録することができなくなっているようです。
大変申し訳ございませんがみなさん「×」としていただき、カレーが不要な方はコメント欄や直接Twitter等で
スタッフにご連絡ください。
Twitter:@open_starmix

《会場：こころ、踊る、カフェ　への行き方》
東京都杉並区高円寺南3-57　藤巻ビル3F

JR高円寺駅より徒歩3分
高円寺南口を右に線路沿いに歩きます。
パル商店街を左手に入り駅を背に２分程歩くと
右手に「サンマルクカフェ」 があります。
そのすぐ次の路地を右に入ると５軒ほど先に「ちゃこーる本舗」
という居酒屋が左手に見えますので、その建物の右手の階段を
上がった３階が「こころ、躍る、カフェ」です☆

靴を脱いで上がるお店なので、足元が気になる方は靴下を持参した方がいいかも、です！
