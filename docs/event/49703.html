catch 世界を知ろう！
description CHAPPE TOWNでは「世界を知ろう」「人と出会って世界が広がる」をテーマに毎月一つの国を取り上げ、

国際交流イベントを開催しています。

その国出身のゲストスピーカーの方にその国の文化・ライフスタイル・ワークスタイルなど役立つ情報を、

海外や英語に興味のある方・海外への留学や就職・移住を考えている日本人の方々に向けてお話ししていただいております。

「文化」「住む」「学ぶ・働く」の3つの側面から、その国の「人々」「価値観」「生き方」を探っていきます。

まずはその国の人々を知ることから始めよう！

その国の文化･慣習や、「もしその国で住むなら？」　「その国ではどんなことができるのか？」

皆さんと一緒に知っていきたいと思います。

イベントのプレゼンテーションは英語で行われますが、英語のレベルは問いません。

プレゼンテーションを聞いた後は、軽食を楽しみながらのフリートークのお時間もございます。

ゲストスピーカーの方に直接質問をしたり、参加者の方同士でその国についてお話したり・・・

イベントを通して知らない世界について知るだけでなく、実際に人とのつながりができるのもCHAPPE TOWNイベントの特徴です。

今回の国はモンゴル！
Let’s get to know about Mongolia!

モンゴル国、通称モンゴルは、東南アジア北部に位置する国家です。
東と南を中国・ないモンゴル自治区と、西を中国・新疆ウイグル自治区と、北をロシア連邦とそれぞれ接する内陸国です。

モンゴルについて、モンゴル人ゲストスピーカーから直接お話を聞きましょう！
