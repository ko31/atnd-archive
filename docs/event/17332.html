description

　「もやしもん」3巻にも登場した、世界一臭いと言われるスウェーデンの缶詰「シュールストレミング」。便利になったこの世の中、ネットで購入できちゃうわけで。だったら買っちゃう！？食べちゃう！？悶えちゃう！？大試食会開催します！

イベント概要

開催日時：2011年7月16日(土)12:00-17:00(雨天決行！)

開催場所：蓮花寺大杉公園 新潟県長岡市蓮花寺1993-4(旧三島町)

参加費：大人 500円(受付時に徴収)、お子様は無料

その他：ジャガイモ、タマネギ、パン、トマト、牛乳、炭酸水などの副菜はある程度こちらで用意しておきますが、他にも持ち込んでもらえると助かります！またアルコール類はこちらでは用意しませんので、飲みたいものを各自で自由にご用意ください。(でも飲酒運転はダメ！絶対！)

　当日開封するシュールはすでに手に入れてあります！現在冷蔵庫にて厳重保管中！

　ご家族、ご友人を誘いやすいようにチラシを作成しました！レッツ配布！是非ご活用下さい！
