【 ヒカ☆ラボ 】
データベースエンジニア必見！
『SI Object Browser』でサクっとパフォーマンスチューニング

★講演内容

データベースのパフォーマンスをチューニングするには高度な知識が必要…
いえいえ、そんなことはありません！

「コツ」と「ツール」があれば、誰だってできるんです。

データベース開発支援ツール「SI Object Browser 」開発者自らが、
製品を使用したOracleデータベースのパフォーマンスの悪化原因を
調べる方法からチューニングする方法についてお教えします。

データベースエンジニアエンジニア必見です！

乞うご期待！

※内容は変更になる可能性がございます

■戸坂龍太郎（とさか　りゅうたろう）
　株式会社システムインテグレータ　製品企画室長

■後迫潤（うしろざこ　じゅん）
　株式会社システムインテグレータ　「SI Object Browser」開発スペシャリスト

ECパッケージ「SI Web Shopping」など、自社パッケージのカスタマイズ案件に従事。
その豊富な開発経験に基づいたDBパフォーマンスチューニングのコツをお教えします。
また、「SI Object Browser」の開発者ならではの裏技やテクニックもご紹介します！
