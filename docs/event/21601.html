description

昨今、スマートフォンなどの普及に伴い、ソーシャルメディア市場は急速に拡大しています。
これまでは「Web」「ゲーム」と各分野に分かれていた垣根がなくなり、各々の分野で大きな融合が始まっています。
そのため、ソーシャルメディア業界ではWebクリエイター、ゲームクリエイターの方々の経験者採用を積極的に行っています。

私たちクリーク・アンド・リバー社大阪支社で開催するのは、そんな世の中の動きに敏感で、関西で活躍の場を求めているWeb・ゲーム分野のクリエイターの方々を対象にし、有益な情報と現場での生の声をお届けする転職個別相談会です。
特にソーシャル系の求人は新しいマーケットの特性で秘匿性が高く、また採用要件の定義づけが難しく非公開の求人が多数あります。

各々の分野で日々多数の企業と接しているエージェントがしっかりとお話をさせていただくことで、ご希望条件や今後の方向性を把握し、よりよいお仕事をご案内させていただきます。

この機会にぜひ転職個別相談会へお越しください。

現在お仕事をされているクリエイターの方々の中から、こういった分野に興味をお持ちで、しかし転職のきっかけがなく悩んでいる方々に対して、有益な転職のきっかけをご提供させていただくことが今回の目的となります。

もちろん今すぐ転職をお考えではない方にも、業界の動向や必要とされるスキルなど今後を見越したアドバイスをいたします。

・ゲーム系クリエイター（デザイナー、プランナー、ディレクター、プログラマーなど）
・モバイル系クリエイター（デザイナー、プランナー、ディレクター、プログラマーなど）
