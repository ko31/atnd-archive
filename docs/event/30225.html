description

　環境教育フォーラム21（東京・文京、代表半田裕之）は6月30日、新宿文化センターで「ビジネスに役立つ環境法講座・一日でわかる環境法対策」を開催します。環境法を企業で運用している実務責任者を講師に、実際のビジネスで環境法にどのように取り組み、また、ビジネスにどのように生かしていくかを学習します。企業のCSR、法務担当者、中堅企業の経営者を主な対象に、複雑化し改正が相次いでいる環境関連の法律を体系的にわかりやすく説明します。

　以下のような方にお勧めのセミナーです。
　・企業法務、CSRを担当しているが、自社が対象となる法律、規制について確認しておきたい
　・廃棄物処理など環境問題に関連する事業を行なっているので自社が関連する法律、規制につい　 て知っておきたい。
　・環境から新しい事業を展開するうえでのヒントが欲しい

　本講座は企業で環境問題の実務を担当する経験豊富な当フォーラムの主席講師が担当します。環境法の概要と各企業が対象となる法律と義務などをわかりやすく解説します。仕事に直結する知識が１日で身につきます。
　セミナーの概要は下記のとおりです。

◆環境法の基礎知識
　環境法とはなにか。どのような法令があり、どのように読むのか。環境条例の状況など。

◆環境法・条例のポイント
　地球温暖化、改正大気汚染防止法・水質汚濁防止法、公害規制、土壌汚染、化学物質、廃棄物の排出事業者への規制、委託処理契約書とマニフェスト、2011年４月改正廃棄物処理法のポイント

◆環境法への対応方法
　あなたの会社がどのような環境法の対象になるのかマトリックスに記入しながらその場で学習していただきます。

◆質疑応答
