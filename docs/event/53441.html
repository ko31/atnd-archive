日韓大好きが集まる広場とは

日本が好きな韓国人と韓国が好きな日本人同士の交流を行っている、個人が開いている日韓親睦会です。
オンライン上で知り合いになって、直接会って友達になる。「韓国と日本のどこへ行っても会える仲間がいる」を目指しております。

毎月第３の土曜日にソウルの「大学路（地下鉄４号線惠化駅）」で定期交流会があります。

よくある質問

年齢代

  • 20～50代

どんな人が参加するんですか？

  • 主にこのような方がよく参加しています。
    1. 日本語の勉強をしている韓国人、日本留学を準備している韓国人
    2. 日本留学に行ってきて日本語が忘れたくない韓国人
    3. 韓国に長期出張に来ている韓国人友達が作りたい日本人
    4. 韓国に留学をしている韓国人友達が作りたい日本人
    5. 幅広い年齢代の人と交流して見聞を広めたい人
    6. 韓国旅行の際に現地の人に案内してもらいたい日本人
    7. 韓国旅行の際にもっと楽しく遊びたい日本人

参加費

  • １次会：15,000ウォン、2次会：10,000万ウォン

時間

  1. 集合時間：地下鉄4号線、惠化駅4番出口に出て右のベンチのあるところで7時集合、7時15分に店に移動
  2. 1次会：7時30分から9時半
    団体写真撮影の後2次会の店へ移動
  3. 2次会：～11時
    その後は自由です。
