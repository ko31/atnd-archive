概要

飯塚から世界へ！をキーワードにソフトウェア技術者たちが集結し、テクノロジーについてディープに語り合う場、e-ZUKA Tech Night。

さて、各大学の夏休みもそろそろ終わり。e-ZUKA Tech Nightも再始動です。

Vol.4は「 e-ZUKAスマートフォンアプリコンテスト 」の開幕を記念して、「スマートフォンアプリの今」をテーマに開催します。
アプリの作り方ももちろん大事ですが、何を作るかはもっと大事。そこがはっきりしないと応募さえままなりません。ということで、スマートフォンアプリをいろいろな視点で見つめてみたいと思います。

LTを募集中です！テーマに関係ないもので構いません。
