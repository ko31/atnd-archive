TokyuRuby会議について

  • TokyuRuby会議は東京で4回目の開催となる Regional RubyKaigi です。
  • Ruby に興味を持つエンジニアが集う Tokyu.rb 主催の LT 大会です。
  • 飲み食いしつつ、みんなで LT をして盛り上がろうというイベントです。

参加費

  • 参加費は無料
  • 飲み物・食べ物は各自持ち込みです（忘れると悲しいことになります）
  • 主催メンバーへのカンパ(酒・食い物)をお待ちしております（皆様の善意によって成り立っています！）

会場設備について

  • 当日は電源タップの持参をお願いします。
  • 無線LANなどのネットワーク設備はありません。各自ご用意をお願いします。

食べ物等について

  • ゴミは各自で持ち帰っていただくようお願い致します
  • なので、持ち込んだ食事は食べきりましょう！
