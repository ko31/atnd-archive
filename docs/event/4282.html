個人的な話ですが、Google WaveのAPIとかまぁだいたい分かったんですけど、ツールとしてはぶっちゃけ全然活用してません。でもTL見てるとWaveを便利に使ってる風なtwitが流れてきてたりして、実際これ使いこなすとどんなふうに便利なんだろうとか思ったり。

私が知りたいと思ってると言うことは、同じようなことを思ってる人ほかにもいるだろう。逆に、語りたいと思ってる人もきっといるはず。いや、いて欲しい。

ということで、Google Waveを便利に使ってるみなさん、全然使いこなしていない私たちになにか語ってくれませんか？

特に熱い想いとかなくていいので（あってもいいけど）、普通に使って普通に便利だったこととかを普通に話してもらえると満足です。あまり使いこなしていない人はたぶんそういうのが聞きたいはず。

  • 一人頭10-20分位（喋れる人が少なそうなので伸びました）
  • 内容は

    • こんな時こんなふうに使ってますとか
    • よく活用してる拡張とか
    • 仕事で使っている場合はどうやって導入したかとか
    • Google Waveを使って彼女ができましたとか
    実際に使ってみてどうこうという話なら何でも構いません

みなさんの参加表明お待ちしてます。LTできそうな人は申し込みの時にぜひチェックボックスにチェックしてください。

お話していただいた方々

  • pino0505 : 以下のようなことで良かったらLTできます。 今の開発案件でWaveを活用しています。大阪と東京で開発しているのですが、WaveでQ&Aのやり取りや仕様の共有を行っています。
  • Seasons : @Seasonsです。 最近は、執筆活動やらiPhone開発のやりとり（趣味）やら 技術系の情報共有やらWaveを使ってます。 何か皆様のお役に立てれば幸いです。LTします。
  • foota : 他の方の活用法に興味があります。／ http://handasse.blogspot.com/2010/03/google-wave.html 以前書いたブログ記事ですが、こんな感じで良ければ何か話せると思います
  • faith_and_brave : 社内の打ち合わせや、雑誌作成にあたっての著者陣とのやりとり、複数人との公開しにくいやりとり、雑談等に使ってます。このへんの事例や、今まで経験した中でWaveが不便だと思ってる点、どう折り合いをつけているかをお話できればと思います
