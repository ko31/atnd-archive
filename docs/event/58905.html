12月15日（月）に、米英のオフ会を新宿のカラオケにて行います。
前日のハンチプの興奮のままに翌日も米英を語りませんか？
参加を希望する方は以下をお読みいただき、回答をツイッターのDMにて芳雅までお願いいたします。

回答期限：11月15日（土）

①カラオケのみ参加

②カラオケ前のお茶だけ参加

③カラオケ後の晩御飯だけ参加

④カラオケの前にお茶をするのであればお茶から参加

⑤カラオケの後に晩御飯があれば晩御飯まで参加

⑥とりあえず全部参加

参加人数によって当日のスケジュールを考えます。
現段階で確実なのはカラオケのみです。
お茶、晩御飯は、申し訳ございませんが参加人数によっては予約等の手配をしませんので、気が合った方、お時間の合う方同士でご相談ください。
一定の人数が集まるようであればそちらも手配いたします。

カラオケのみをこちらで手配する場合は、時間は11時～18時(19時)くらいを検討しています。
（先にお茶をする場合はカラオケの開始時間が11時半や12時からになる可能性もあります）
途中からの参加、途中での中抜け、OKです。
ただし料金は一律となってしまいますのでご注意ください。
年末年始が近く、通常の平日料金ではないということですので、金額は分かり次第皆様にお知らせいたします。

カラオケではお絵かきしながら米英ソングをオススメし合ったりしたいと思っています。
お絵かきは強制ではありませんので、絵描きさんじゃない方も遠慮なくご参加ください。
主催がゆるいので、米英っていいねアハハって感じのゆるいオフ会です。

なにか気になる点、ご質問がありましたらツイッターやDMからどうぞ。
