ハッカソン

event_id 46288
title ハッカソン
catch はじめてのハッカソン
description

デザイナー・コーダー（フロントエンドエンジニア）・プログラマー、みんなで集まって何か作りませんか。

作るものは自由！

そして、参加費は無料です。

どんな集まりか。

「誰かと一緒にものを作りたいけれど、ハッカソンに参加するとなると今の自分ではスキルが足りないかも。」
やる気はあるのにそう思って躊躇っているクリエイターさん、多いと思います。

※「はじめての」と付いていますが、Web初心者さんを対象にしているわけではありません。
ハッカソンに対する敷居を少しでも下げるために「はじめての」と付けています。

・デザインは出来るけどWebデザインは慣れていない方
・UI/UXは分かるけど、デザイン自体は不慣れな方
・元々はコーダーでデザインをやり始めた方
・やる気はあるけれど、スキルが低くて参加しづらい方
・作業が遅くて相手に申し訳ない！！でもやる気はあるのよ？な方
・デザイナーとして申し込みたいけど、自信ないからなあ。。。

そんな方々に参加して頂きたいです(_)

誰かと一緒にものを作る。

自分の出来ること、相手の出来ること、それぞれが全力を出し合って作って貰います。

3人程度のグループを作り、もくもくして貰います。

お互い知りたい情報を共有してみたりも大歓迎ですし、デザインとコーディング両方やりたい人、コーディングからプログラミングまでやりたい人、分担は人それぞれでOK★

参加理由は人それぞれ！

・スキルを上げたい
・いつもと違う環境＆発想の人と作ってみたい
・ただ、面白そう。

みんなで楽しくをモットーに。

競い合うようなハッカソンではありません。
1日で作品を作ることで四苦八苦して貰います。
作るものはグループ内でご自由に決めて下さい。

ex.)InstagrumのWeb APIを使って好きな画像一覧サイトを作ったり・・・
思わずWeb APIを使った「何か」を作りたくなる★Web API紹介

参加コメントに以下のいずれかを書いていただけるとありがたいです。

デザイナーとして参加します。
コーダーとして参加します。
フロントエンドエンジニアとして参加します。
プログラマーとして参加します。
