description 【第一部:ワークショップ】

　本イベントでは、トップベンチャーの一人である孫泰蔵氏をお招きし、ワークショップと講演会を通じ、参加したすべての方が「起業」について考える機会を提供します。

　第一部ではワークショップを開催し、6つのテーマからビジネスアイデアを提案していきます。１つのテーマあたり10人ほどでグループを形成し、ITx農業やITx芸術・音楽など、それぞれのテーマと様々なコンテンツを掛け合わせ、これまでにないビジネスアイデアの創出を目指します。議論が生き詰まった際には、孫泰蔵氏をはじめ、県内の起業家やインキュベーター、キャピタリストの方々によるサポートを受けることができます。また、翌日（11月2日）、そのアイデアをプレゼンテーションしていただきます。優秀グループには、各種景品をご用意していますので、奮ってご参加ください。

　第二部では、ワークショップ参加者によるビジネスアイデアのプレゼンテーションと、それに対するコメント、および孫泰蔵氏による講演が予定されています。孫泰蔵氏による講演は、事業内容の紹介、起業支援のご活動、起業家としての働き方について、90分ほどお話いただく予定です。

　　　

第二部　講演会＆プレゼンテーション

　　　　日時：平成26年11月2日（日）13:00－16:00（受付時間 12:30-）

　　　　場所：大分県立図書館　視聴覚ホール

　　　　定員：250名
