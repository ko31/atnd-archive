結果を出せる組織に！体を動かすチームビルディングセミナー

株式会社EGGS代表取締役　木村太郎

チームビルディングとは？

チームビルディングは、組織をただのグループから結果を出せるチームにするために、体を動かしながら行う画期的なプログラムです。

初めて会った人たちが、新しくチームを作るときに使うことはもちろんのこと、既存のチームの問題点を浮き彫りにしたり、コミュニケーションを円滑にしたりと、目的の共有を強化し固定しがちなチームの役割を壊し、新しい発想を生むきっかけを作ることができるのも、このチームビルディングの魅力です。

こんな人におすすめ！

  • 新しいチームを結成し、これから組織で成果を出したい
  • メンバーのモチベーションが低く、組織に閉塞感を感じている
  • 上司部下の壁を取っ払って、組織を活性化したい

チームビルディングを素質論でフィードバック！

有限会社FGP 代表取締役　國貞幸司

素質論とは？

人は生まれながらにして、様々な能力や特性が備わっています。

素質論はそういった特性を生年月日で統計化した理論です。

前半におこなったチームビルディングの結果をもとに、素質論の視点を加えたフィードバックをしていきます。

体を動かした後の頭の整理に最適です！

後半　異業種交流会　名刺交換だけで終わらない人脈作り！

20:00〜21:00　異業種交流会

大好評！ クロッシングシートシステムを用いた新感覚の異業種交流会です。ビジネスパートナーや新しい発想を得たいなら今がチャンスです！

クロッシングシートシステムとは

交流会で会って方と人間関係が続いていますか？

多くの場合、交流会で会った人とは名刺交換だけで終わっていることが多いと思います。

お会いした人とビジネスや趣味で接点を見つけようと思っても長くお付き合いできる人にならない。そのようなケースを多く耳にします。

当交流会は、接点がなくても【人と人が純粋なつながり】【想いと想い】【心と心】のつながりを築けるようにクロッシングシートシステムにより次のようなキーワードでつながれることが特徴の一つです。

  • お互いに刺激し会える人
  • 短所を補ってくれる人
  • 意思疎通がしやすい人

    など

交流の目的にピッタリの相手が見つかります！

クロッシングシートには、目的に応じた自分にピッタリの方の番号が記載されています。交流会にて参加者お一人に１枚ずつクロッシングシートをお配りします。

クロッシングシートに書かれた番号の人を探して交流してみてください。

長くお付き合いできる人脈が築けること間違いなしです！！

是非この機会にお試しください！
