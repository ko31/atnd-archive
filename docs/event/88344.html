毎日、大量のご要望・クレームなどのお問合せが届いている…
ユーザーの声をサービスに取り入れたいが、取り組み方がわからない…
アプリ・サービスの品質を向上させたい…、継続率を高めたい…

【ユーザー満足度の高いアプリ】を提供するためには様々な悩みがつきものです。
運用コスト・工数削減が求められ、人材を確保することも難しい昨今、
残業時間を増やして対応している、というご担当者様も多いのではないでしょうか。

本セミナーでは、他社事例を交えながら、ユーザー満足度の高いスマホアプリ運営の秘訣を
以下のポイントを中心にご紹介します。

　●カスタマーサポートの軸からユーザーの声を取り入れる方法
　●ユーザーの声を分析・可視化する方法

日頃の業務が忙しいが、最新情報は常に吸収したい！！という方におすすめです。
お昼休みにPCやスマホからご覧いただけますので、ぜひこの機会にご参加ください！！
こんな方におすすめです

　　　■スマホアプリの開発運営に、ユーザーの声を積極的に取り入れたい
　　　■カスタマーサポートを軸に、サービス改善を行いたい
　　　■カスタマーサポートの品質を向上させたい・改善したい
　　　■他社の運用事例を聞いてみたい

セミナー内容

12:30～12:40
アプリの品質管理を通したユーザ満足度の向上
●概要
スマートフォンアプリの品質管理の観点から、ユーザー満足度向上の為のツール活用についてお話致します。「なぜアプリの品質問題が起きるのか」といった根本の話から、「どうすれば品質を改善できるのか、どうすれば顧客の不満を解消できるのか」といった具体的な方法の話まで、データや事例を交えてご紹介致します。

●講師
FROSK株式会社
事業推進部　マーケティングマネージャー
吉井　文学

●企業HP
HP：http://www.frosk.co.jp/

12:40～12:50
ユーザー視点のカスタマーサポート運用について
●概要
当社が提供するカスタマーサポート代行サービスの運用実績を通して、ユーザー満足度を高める運用手法・サービス改善手法などのカスタマーサポート対応テクニックをご紹介します。 成長している企業が導入するカスタマーサポート代行サービス、運用を効率化する為に導入しているシステムについて具体的な活用事例を交えてご説明します。

●企業HP

12:50～13:00
ユーザー満足度が高まるメール対応のポイント
●概要
メールのお問合せ対応を通してユーザーの満足度を高めるポイント、ユーザーの声をサービスに活かすための運用・フロー作りなどを事例を交えながらご紹介します。
