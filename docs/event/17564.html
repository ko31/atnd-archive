最近の「メールの誤送信対策」や、「在宅勤務対応」「スマートフォン対応」
など、どれから対策を考えていけばよいか？メールシステムの拡張にお困り
になっておられませんか？

CTCSPはダウジャパンが提供するメールソフトウェア『Terrace Mail Suite』
を活用し、メールシステムの構築を始め、既存のメールサーバからのデータ
移行、データ保全、周辺システムに至るまで、検証済みの構成をワンストッ
プに短納期かつ低価格なソリューションを提供することをこのたび、発表し
　　ました。

本セミナーでは、『Terrace Mail Suite』を核に、ワンストップメール
ソリューションのメリットを説明させて頂きます。また、スマートフォンか
らのアクセスのデモも来場者に公開しておりますので、その場で操作性を感
じ取って頂くことも可能です。
