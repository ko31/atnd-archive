description

世界にどこでもドアを開けていく中継サイト”Ustream“がブレイク間近です。

日本語化もされて、ますます広がるUSTREAM、これはまさに世界を変えるネット生中継です。

そこへ、5/19日本で初めてUSTREAMに関する本が出版されます。

USTREAMERとは最小限の機材を使ってひとりで中継できるスキルを持ち かつソーシャルストリームに流れるコンテキストを現場にフィードバックして 中継を立体的に進行していくデジタルモデレーターの能力を持った人材です。

　今回の初級編では基本的なUstreamの機能を理解し、webカメラとノートパソコンでの 基本的な中継機能を習得します。中継にビュアーを呼び込む重要な機能である ソーシャルストリームの扱い方、ハッシュタグの設定の仕方などにも触れます。 講座内のミニワークショップでは来場者と共に画面の中にアクセス可能なこの メディアならではのユニークなコンテンツの可能性を探ります。

　本講座を受講することでその日からUSTREAMを使って新しいライブコンテンツの時代に参加することができます。お気軽にご参加下さい。

　なお本講座はUSTREAM中継されますので受講中の様子が中継されることをご了承の上お申込み下さい。
