title 浦安マッピングパーティ
catch OpenStreetMapに触れてみよう
description

OpenStreetMap上に地図を描く方法を経験者とともに体験するイベントです。
経験者・未経験者を問わず歓迎致します。お気軽にご参加ください。

1.日時、集合場所
6/19(日)10時～17時(9:50集合)
京葉線新浦安駅　東口駅前広場

雨天決行（歩ける程度なら）

2.費用
・参加費無料
・交通費、食費等は各自で負担ください。

3.募集人員
・特に制限はしませんが10名ほどを目安に考えています。

4.持参するもの
・当日、資料等を配布しますので最低限筆記具があれば手ぶらでも構いませんが、
　以下のようなものがあると便利です。
・デジカメ
・スマートフォン
・GPSロガー
・モバイルなPC

5.目標
・OSM上にウェイ（道路などの線）を引ける
・OSM上にノード（お店などの点）を打てる
・浦安地区の被災/復興状況の体感とマップでの表現を考える
　（今回は大きな通り沿いを主な対象とする予定です）

6.タイムスケジュール（当日の状況により多少変わる可能性があります）
10:00～11:00 行動予定の説明、周辺を歩いてみる
＜人数に応じて、最大各5名程度のチームに分かれる＞
11:00～14:00 チーム別にマッピング
14:00～17:00 成果発表と編集作業（場所や手順は調整中）
17:30～19:30 懇親会（自由参加、実費割り勘）

7.当日までの事前準備
＜必須＞
・事前にOpenStreetMapアカウントを作成しておいてください。
手順はこちらのビデオを参照
＜任意（可能な方のみで構いません）＞
・浦安周辺のBingによるリモートマッピング
・AndroidやiPhone向けのOSMマッピング用アプリ（GPXファイルロギングアプリまたはPOIエディタ）のインストール

8.関連情報
◆OpenStreetMap操作ガイド（Potlatch2編） (約6MB)
◆OpenStreetMap
◆ドキュメント（日本語トップ）
◆日本でのタグ付けの基本（主に道路）
◆タグ一覧（カテゴリ別）
◆タグ一覧（50音順）
◆iPhone関連
◆Android関連

9.マッピング（マップの描き方）の例
写真のコンビニをOpenStreetMapで表現する場合には以下のように描きます。
1)まずコンビニの位置にノード（点）を打ちます。
2)そのノードに以下のようなタグ（属性を表わすキーと値の組み合わせ）を書き込みます。
shop=convenience （コンビニ）
name=Sunkus （名称）
disused=yes (休業中)

実際のマッピング例
