CSS Nite in SAPPORO, Vol.8は、「札幌IT飲み会」とのコラボ企画。
新年会兼ライトニングトーク特集です。

ライトニングトーク（短めのプレゼン）を聞きながら、みんなで楽しく新年会。

参加者で希望される方は、ぜひライトニングトークに登壇ください。聞くだけの参加も、もちろん大歓迎です。

参加費

2,800円（飲食代実費／当日会場払い）
※ 料理代・お酒代込みです

TAPIO sweets garden（タピオ スイーツガーデン）
