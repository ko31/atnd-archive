ぬRubyとは

  • 和室でぬるく Ruby をさわります。
  • Rubyをさわりたい人は誰でも参加できます。

ながれ

  • 9時すぎからわらわらと集まりはじめます。
  • 10時くらいに「私は誰です」「今日はぬるくこれします」を言います。
    • すぎてからきた人はきた時に言います。
  • ぬるくRubyをさわります。
  • 16時までにかたづけをします。
  • かたづけ終わったら撤退します。

部屋について

  • 定員が10名の部屋なので、人がいっぱいだと寝そべれないかもしれません。
  • 電源の口の存在は未確認……（もし電池きれたらペアプロかお話しましょう）
  • 会場にwi-fiはありませんがわたしのwi-fiが使えます。(先着３名様まで)
  • 時間内入退室自由です。ふらっときてふらっと帰れます。

その他

    • 会場に貯金箱置くので入れてください！
