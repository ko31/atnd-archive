内容

7/9に開催される TDD Boot Camp in Tokyo #tddbc でのustreamを見ながらTDDを勉強しようという内容です．
ですので， TDDBC東京のATND の内容に準拠します．

ustreamの調子が悪かったりした場合は適宜調整していく予定です．

基本はまったりしつつも東京からのTDDBCの空気を感じようという集りと思ってください．

ちょっと15分早く始めますが，自己紹介を先しようと考えてます．
また終了後は後片付けを手伝っていただけたらありがたいです．

当日の使用言語について

TDDBC東京のATND に準拠するため，お題等は東京に依存しておりますので使用言語は東京のATNDを参考にされてください．

現在のところ

  • PHP:PHPUnit
  • Ruby:RSpec
  • Java:JUnit

上記が対応言語になっています．

RubyのRSpec以外はなんとかこちらでもサポートできると思います．
Rubyの場合はTest::Unitでサポートさせていただきます．
あとPythonは対応できます．

また，TDD未経験の方は通常使用されている言語で参加されることをオススメします．

TDD経験者で上記以外の言語での参加希望者は是非参加していただけるとありがたいです．
