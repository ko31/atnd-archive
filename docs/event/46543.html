■勉強会の概要

WordPressに代表される、オープンソースCMSを使用してみたけれど、オリジナルサイトの制作に煩わしさを感じている方。

また、SNSサイトのようにコミュニティ機能（＝認証機能）が充実した ホームページを作成してみたいという方の参加を歓迎いたします。

尚、Drupalの初期ユーザや、ビジネスの現場でサイト制作に関わっていらっしゃる方も是非ご参加ください。

■1月のテーマ　

　　良く使う拡張モジュールの設定、使い方 その３

勉強会用サイトにインストールされている拡張モジュールを使って、
設定方法や使い方を学ぶ

拡張モジュールとして、以下のモジュールについて、

11月(その１)で説明したモジュール：　ctools、Panels、Rules、Metatag、Variable、Views、

12月(その２)で説明するモジュール：　XML sitemap、Colorbox、Insert、Libraries、Module Filter、Pathauto、site map、

･ 1月その３で説明するモジュール：

Webform、CAPTCHA、CKeditor、IMCE、ｊQuery Update、Date、Calendar、Google Analytics、ほか

･ 時間に余裕があれば、Drupal 8 の概要説明とコアとなった Views モジュール機能の概要

･ DrupalやCMSに関する質疑応答、ほか
