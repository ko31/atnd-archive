こんにちは、Open Network Lab、Managing Partnerの前田です。

第４期への応募締め切りが１０月３１日です（第４期への申し込みはこちら）！

そこで、第４期「Seed Accelerator」応募説明会を１０月１７日(月曜日)に開催します。

多くの方が参加しやすい様に、午後6時30分開場、7時開始予定です。
イベント後半は懇親会を予定しており、会場のスタッフに声をかけてもらえれば質問にお答えしたいと思いますのでご参加お待ちしています。
そして不明な点や心配な点をこの時にクリアにして頂いて、ぜひ募集をしてみて下さい。
なお当日飛び込みの参加も歓迎ですが、ATNDで参加表明してもらえてると嬉しいです。
