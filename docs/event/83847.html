【世界一大切な相手にサプライズしたい人あつまれー!!】

- あなたにとって、いま一番感動させた相手って誰ですか？ -

このプロジェクトは大切な相手にすごく感動してもらうイベントです。
恋人？奥さん？旦那さん？お母さん？お父さん？親友？

最近の日本と言えば、感動する場面や機会が減ってきて、
引き換えに寂しいニュースが多かったりします。

その裏にはちゃんと愛を持って人を
『好きだなぁ。愛おしいなぁ。』と想える人もたくさんいます。
このプロジェクトはそんな人達を集めて祝う面白くて幸せなイベントです。

##方法：動画を覚えて贈って、"想い出"を相手に届けて涙を引き出す。
（想いを伝えるのはお金じゃなく、気持ちだというのを体感するイベント）

##特典：相手に一味違った感動を与えられる

##創る物：想い出の動画
（他に作りたいものがある場合は別途ご相談下さい。当日は受け付けられませんが、別日ご相談に乗ります。）
