■Apple User Groupとは？

Apple User Group（アップル ユーザグループ）は、アップル製品（Mac、iPhone、iPad、iPod等）を愛する会です。
『AoMacクラブ』は横浜北部エリアのメンバーが中心のApple User Groupです。

■月例会について

毎月第3土曜日に月例会（交流会・勉強会）を開催しています。
アップル製品の最新情報や、話題のSNS（facebook、twitter等）、広くIT関連の情報に興味ある方、
興味はあるけど「何にも解らないから教えて〜！」な方、
お住まいが横浜でない方でも、
老若男女、Ladies & Gentlemen, Boys & Girls !!
誰でも参加できます。

【第1部】
公式ブログ（上記URL）で取り上げた記事を元に、直近一ヶ月の最新情報をお復習いします。情報に詳しいメンバーが解説を加えてくれます。

【第2部】
参加者が持ち寄った最新情報、最新デバイスなどを題材にフリートークをしたり、またビギナー参加者がいらした場合は「質問コーナー」など、参加者に合わせた内容にしています。

■参加方法

横浜Apple User Group『AoMacクラブ』の交流会を上記の要領で開催します。
参加希望の方は右カラム“赤地の【このイベントに参加登録する】”をクリックし、
指示に従って参加登録してください。

■初めて参加される方へ

初参加の方は・・・、
上記の【URL】をクリックして公式ブログも併せてご覧ください。
※※登録なしで、交流会当日に見学、飛び入り参加もOK！大歓迎です。
