八王子校舎の衣食住について

八王子校舎は宿泊施設がありませんので、基本的には、制作のためにオールナイトで活動ができる制作スタジオとなります（インドアなキャンプだと思ってください）

就寝道具として、布団セットを業者よりレンタル（有料：１人 1800円程度）いたしますので、それを利用します。
就寝時、女子は鍵がつく部屋を使用して貰う予定です。

スタジオ内には小規模ですがキッチンがあります。夕食は、皆で考えています。

朝と昼（一部 夕）に関しては、コンビニや多摩センター駅前の施設などを利用する予定です。昼間に一度くらい流しそうめんができればと考えています。

食費については、制作をする場合、贅沢しなければ 晩ゴハン一食あたり大体500～600円位徴収予定です。

校内には、水圧の高いシャワーが完備されていて使えます。
（時間帯は共同で使用している団体と時間割）

PCルーム横の階段出口に喫煙スペースがあります。スタジオ内において、その場所以外では喫煙はしないでください。

参加費は、上記の就寝道具や運営費と合わせて、2000円の学生負担となります。（交通費・食費は別途かかります。）
