概要

2月17日に東京で開催された、 【初心者向け】Sencha いろはの「い」 を大阪で再現します。

HTML5 Webアプリケーションフレームワークの Sencha Ext JS / Sencha Touch、そしてその開発ツール（Sencha Cmd/Sencha Architect）などのご紹介を始め、Sencha Touch を利用した簡単なアプリケーションの構築をハンズオン形式で、0から完成まで一緒にやっていきます！

「Webアプリを作ってみたい」
「使ってみたいけど、何すれば良いの？」

といった初心者の方でも気軽にご参加ください！

講師ご紹介

中村 久司 ( @martini3oz )

大学の卒業論文を終えた時、何かを学習したいとプログラム言語を学び始める。趣味ではBASIC09、Pascal、C言語を学び、天理大学情報センターに在籍中は、主にDelphiやAccess VBAで業務システム構築に従事。Webでの業務アプリ開発を模索している際、Sencha Ext JSと出会い、小堤 一弘 氏に師事しSenchaフレームワークについて基礎から徹底的に学ぶ。Sencha製品を愛するあまり、安定した大学職員生活を捨て転職を決意し、現在、株式会社ゼノフィに在籍。
ゼノフィでは数多くのSencha学習テキストを執筆、米Sencha社公認のオフィシャルトレーナー。 Sencha Touch 2実践開発ガイド 著者。
