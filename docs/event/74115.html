description

ロボットカーの制御プログラムを工夫して、ライントレース走行に挑戦しよう！

　2016年２月上旬より開催される「かわしり春ものがたり」の一環として、「ロボットカー・プログラミング体験講座」を行います。

　ロボットカーを「運転」させるプログラムの作成を体験することで、コンピューターを使った「計測と制御」の基本を学ぶ講座です。

　世界で多くのMakerたちが愛用しているマイコン「Arduino」を頭脳に、フォトセンサーでコースを読み取りながら走行する「ライントレース走行」に挑戦します。

　ゲームじゃ味わえない「リアルに動く」感動をぜひ体験してください。

　くまもと工芸会館　熊本市南区川尻1-3-58
