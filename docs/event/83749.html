description

　リクルートグループのIT／ネットマーケティングソリューションを牽引するリクルートテクノロジーズが、ITエンジニアのスキルアップ・キャリア形成のための勉強会を連続開催！　全4回にわたり、そのエンジニア環境やリアルな技術開発・運用を徹底的に解説します。

　第1回のテーマは、ネットインフラ領域。これまで多数の大規模インフラ構築や仮想化案件を手掛け、Azureエバンジェリストとしても活躍してきた高岡将（たかおか・すすむ）が登壇。多数の巨大サービスを展開するリクルートのインフラ事情と、インフラエンジニアならではのキャリア形成術をお届けします。


◇講演内容◇
「リクルートのネットサービスインフラを支えるエンジニア環境」とは
テーマ：大規模インターネットシステムのインフラ開発、運用環境（クラウド、オンプレミス）

＜概要＞
リクルートが展開するネットサービスのインフラは、一つ一つが比較的大規模、エンタープライズ向けとそうでないものが混在しています。今回は、それらの集合体の考え方や、組織体の考え方、インフラ、エンジニアメンバーの環境などを、オンプレミス、クラウドといった切り口から、実際の現場メンバーと共に紹介いたします。


◇講演者プロフィール◇
名前：高岡 将（たかおか すすむ）
所属：ITソリューション統括部　インフラソリューション２部

＜経歴＞
大手金融企業、ネットベンチャーにてコンシューマー/エンタープライズ両分野で一貫して大規模インフラの構築や仮想化案件を経験。直近では外資系メーカー、ベンダーにてプリセールス、エバンジェリストなど活躍し現職へ。大規模システム、オンプレミス、クラウド、ユーザー、ベンダー、メーカー全てを網羅する。
