【 Dual Boot #1】

Androidで楽器を作る


“Dual Boot”とは、会津エリアを中心に開催されるAndroid/iPhone/GAE向けのアプリケーション開発セミナーです。

#1の今回は、Androidで楽器を作成してみます。


日時: 10:00-17:00
場所: 会津大学UBIC
要件: Javaができる人
テキスト: https://sites.google.com/a/gclue.jp/android-docs-2009/
参加費: 無料
主催: 日本Androidの会 会津支部
協賛:会津大学UBIC
Android実機: セミナー用に20台用意します。
セミナー用マシン: 各自持参してください。


【 事前に準備してもらいたい事】



開発環境を構築してきてください。EclipseでAndroidエミュレータが起動する状態のマシンを持参してください。
