第８回　アニメーション講習会となります。

今回は初の鹿児島開催となります！！

株式会社トリガー様より取締役兼アニメーションプロデューサーの舛本氏をお招きし講習会を行わせていただきます！

一月からは新作アニメ
「リトルウィッチアカデミア TVアニメシリーズ」も放映がされています！！

ここでしか聞けない、制作秘話などレアな話も必聴です！！

【第一部】（６０分）

【アニメーション講義】

第一部では主に２Ｄアニメーションの制作工程を詳しく説明し、そこに携わるプロスタッフの業務内容と責任、必要とされる技術を紹介します。

【第二部】（６０分）

【トーク】

第二部ではアニメーション制作の今と未来をテーマに、会場に来ていただいた方々の興味あるテーマについてトークします。

【第三部】（４０分）

【質疑応答】
参加者の質疑に返答いたします。

テーマ一覧

  • オリジナルアニメーションと原作付きアニメーションについて
  • TVシリーズ、webアニメ、ショートアニメ、劇場作品について
  • アニメの中間素材について
  • アニメでのビジネス
  • 海外の物販とイベントの印象
  • デジタル作画について
  • アニメ制作会社のこれから
  • アニメを仕事にすることのおもしろさと大変さ
  • 求められる人材
  • トリガー福岡スタジオの目指すもの
  • etc…
