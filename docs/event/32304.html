Android日本橋でんでんタウンもくもく会第7回
「イチからできるUSB Host API」 ハングアウト

14時ごろから、日本Androidの会神戸支部の鈴木@ksksueさんによる「USB Host API」をイチからはじめるセミナーを行います。
このセッションは大阪日本橋ROBOBA会議室にて開催される「Android日本橋でんでんタウンもくもく会第7回」にて行われ、ハングアウト中継での参加となります。

「USB Host API」とは？：AndroidとマイコンのUSB接続手法で、ADK,MicroBridgeに続く第3の接続手法。Android 3.1以上が必要だが、マイコン側はUARTがあれば接続可能。多くのマイコンがUARTを実装しているためAndroid側さえ準備できればほとんどすべてのマイコンと接続できる可能性を持つ。
