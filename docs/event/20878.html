「HTML5」と「スマートフォンアプリ」にピンときたら！

このワークショップは、HTML5を学びながら最終的にスマートフォンサイトの制作を目指します。
スマートフォンサイトの制作の経験のないデザイナやコーディング担当の方々、さらにHTML5でのサイト制作に興味をお持ちの初心者のかたにもご理解いただけるような入門講座（ワークショップ）です。

ワークショップ内容

  1. HTML5　スマートフォンサイト用のHTML5
  2. CSS3　デザインの見栄えを良くする。
  3. JavaScript　WEBサイトからWEBアプリへ
  4. Canvas　グラフィカルなアプリ

そのほか、「twitter」「位置情報」「フレームワーク」「iPhone / Android アプリ化」についても触れます。

用意するもの

  1. PCは Mac または Windows
  2. ブラウザは safari または chrome
  3. 文字コードを変更できるテキストエディタ（秀丸、TeraPad、サクラエディタ、miなど）
  4. 参加費4,000円
  5. ネット環境はご用意しておりません

その他
