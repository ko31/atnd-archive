プロの力で、あなたの人生一番のベストショットを！

イベント ユーザー

title プロの力で、あなたの人生一番のベストショットを！
catch 　～ナチュラルかつきちんと感のある、印象的なプロフィール写真撮影会
description

【こんな方のための、特別イベントです！】

★写真うつりのわるい方、撮られるのが苦手な方

★カメラの前では、緊張して笑顔になれない方

★自分のお気に入りのプロフィール写真を持っていない方

★実は、３年以上前の写真を、いまだにプロフィールに使っている方

★プロのヘアメイク付きで、モデルのような写真をとってもらいたい方

【一般的な写真撮影会とは、まったく違います！】

これまでの写真館で撮る、証明写真だと…

「服やメイクに自信がない」

「どんなポーズや表情で写っていいかわからない」

「緊張してベストショットにならない」

そんな不安もありました。

でも、この撮影会なら

「プロのメイクやファッションコーディネートが、あなたのビジュアルを全面サポート！」

「どうやって写ればいいのか？　角度や表情、身振り手振りまで、ポージングをマンツーマンで指導」

「プロフィールなどに自由に使えるさまざまな角度からの写真を、データでお渡し。名刺、ブログ、パンフレットまで広く活用できます！」

仕事にも！　プライベートにも！　プロフィール写真にも！

一生使える、ナチュラルかつきちんと感のある写真を手に入れることができます。

【つまり、まとめると、このイベントは…】

★プロのヘアメイクさんにスタイリングしてもらって

★プロのスタイリストさんに服と着こなしを選んでもらって

★プロの演出家にポージングを指導してもらって

★プロのカメラマンに写真をいっぱい撮ってもらおう！！！

という、夢のようなゴージャス企画なんです。

…ヘアメイクまでならまだしも、ポージング指導までを含む

４名のプロがあなたを全力でサポートする写真撮影会なんて、他にありますでしょうか？

いえ、聞いたことがありません！
