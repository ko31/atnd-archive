description

あなたは日常で、こんなことを思われたことはありませんか？

●私らしく、迷いなく生きていきたい。

　でも、そもそも私って何だろう？

●毎日、会社と家の往復で同じことの繰り返し。

　いったい、何のために生きているのかな？

●イライラして、仕事もプライベートもうまくいかない。　怒りっぽい自分がイヤになった…

●努力はしているのになかなか報われない…　もうがんばるのをやめてしまおうかな。

●職場の人間関係に、もう疲れた…。

　どうすればいい人間関係をつくっていけるのかな？

職場でも家庭でも、このような悩みを人知れず抱えている人は少なくありません。

――――――――――――――――――――――――――――――――――

・もしあなたが、仏教を学び、人生のむなしさの根本原因を知り、

　むなしさを感じなくなるようになったとしたら…

・もしあなたが、お釈迦様の知恵から、良好な人間関係を築く方法を知り、

　人間関係のトラブルに悩まされることがなくなったら…

・もしあなたが、仏教で説かれる“本当の私”を知り、迷いのない人生を送ることができたとしたら…

素晴らしいとは思われませんか？

それらの悩み・疑問の“根本原因”と“解決方法”を教えていかれたのがお釈迦様であり、

2600年前に説かれた仏教です。

当講座では、年間講演数100回以上の実力派仏教講師により、一からわかりやすく仏教を学ぶことができ、

人生の悩み・疑問への答えを得ることができます。

過去の感想 因果の道理(運命の原因と結果の法則)は人生の全てに

通づるもので、これからも心に留めて生活したい
と思います。もっとお話聞きたいです。(50代・女性)

周囲や環境を変えても、何も変わらない。
全ては自分の心、自分次第だということは、
分かっているようで、分かっていませんでした。
「運命の原因と結果の法則」をハッキリ聞かせてもらい、自分を変える、そして他人を大切にすることを
もっと頑張りたいと思いました。　(20代・女性）
