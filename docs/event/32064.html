catch 機能安全の基本を理解してISO 26262を効果的に実践しよう！
description

【講座のポイント】

　自動車分野の機能安全規格ISO 26262が発行して９ヶ月になろうとしています。本規格を効果的に実践するためには、機能安全とは何かの基本を理解するのが早道です。基本規格IEC 61508を基に機能安全の基本から説いて、ISO 26262対応への課題と認証対策について説明します。

【受講対象者・レベル】

　自動車サプライヤ関連の上級管理職、設計開発リーダー、設計開発部員、電子制御技術者、ソフトウェア開発者、その他のISO 26262機能安全をマスターしたい方など

【プログラム】

１．機能安全の基礎

　　1-1　基本規格IEC 61508開発の背景、必要性，目的
　　1-2　安全確保における機能安全の位置付け
　　1-3　機能安全の基本的考え方とフレームワーク

２．ISO 26262の特徴

　　2-1　ISO 26262開発の経緯
　　2-2　プラントと自動車との比較
　　2-3　ISO 26262の用語法
　　2-4　SILとASIL
　　2-6　SFF(安全側故障割合)とフォールトメットリクス
　　2-7　決定論的能力とASIL分解

３．ISO 26262は何を求めているか

　　3-1　安全マネジメント
　　3-2　機能安全アセスメント
　　3-3　機能安全監査
　　3-4　リスクアセスメント
　　3-5　妥当性確認
　　3-6　適合確認
　　3-7　システム開発の要求事項
　　3-8　ハードウェア開発の要求事項
　　3-9　ソフトウェア開発の要求事項
　　3-10　支援プロセスについて

４．機能安全規格の今後の展開について
