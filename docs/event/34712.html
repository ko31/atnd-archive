■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション1(30分)
　『Webに活かす映像制作術』　INDeVisual 浅野さん

　＜主な内容＞
　映像はノンリニア編集からいよいよファイルベース編集が当たり前の時代に。
　安価ながら、高機能／高画質の機材も手に入りやすくなり、
　「映像制作」がより身近なものとなってきました。
　このセッションでは、例えばイラストやテキストと同じような感覚で
　手軽に、Web上へ「映像」という情報を放流することの可能性をお伝えします。
　今回はその下準備として、ビデオカメラや編集ソフトなどが疎遠なところにあるWeb制作者の方を対象とし、
　Web制作でも割と身近なツールを用いて１本の映像を仕上げていくことを趣旨とします。

　＜浅野さんの紹介＞
　映像作家。
　「iNDeVisual」という任意団体で、
　映像制作や良い（けど大きな資本がつかない）映像作品を
　よりフリーに発表できる環境作りなどに取り組んでおります。
　（代表以外に人材がないため、バカでも肩書きは「エクゼクティブ」）

　

●セッション2(40分)
　『あなたのつくったWEBサイト、生きてますか？』大林さん

　＜主な内容＞
　一般的にコンテンツは制作過程よりも公開されて閲覧状態のほうが
　遥かに長い時間を過ごします。

　運用していく上で「いつでもアクセスできる」という状態を保つのは
　案外簡単ではない、というのは運用に携わった事のある方ならば
　実感されたことがあるのではないでしょうか。

　そこで「監視」というキーワードからWEBの世界を見たとき、
　いったいどんなことに注意してサーバを見て行かなければならないのか
　インフラエンジニアの視点からご紹介します。

　＜大林さんの簡単な紹介＞
　2004年くらいからサーバを触りはじめ、
　2007年からWEBの世界に入りました。

　WEB でユーザーの目に触れない部分はだいたい守備範囲なので
　インフラエンジニア、プログラマー、稀にディレクターなど
　できることは頑張ってやってます。

　専門は LAMP とネットワークなWEB全般の裏方まわり。

　統合監視で使われる ZABBIX もちょっとだけご紹介。

　

●セッション3(40分)
　『自由過ぎるCMS、concrete5を上手く使うためのお作法』
　コンクリートファイブ ジャパン株式会社 佐々木多生

　＜主な内容＞
　非常に優秀な汎用CMSのconcrete5ですが、何でも色んなやり方で出来る為に、
　よく知らない人にとっては何をどうしていいのか解らなくなりがちです。
　今回はそんなconcrete5を上手く使うためのお作法のお話しをします。

　＜佐々木さんの紹介＞
　コンクリートファイブ ジャパン株式会社 CTO
　http://concrete5.co.jp/
　concrete5日本語チーム コミッター
　concrete5.org Community Leader

　株式会社クロスキューブ 代表取締役
　EC-CUBEコミッター
　EC-CUBE シルバーインテグレートパートナー

　concrete5 Japan 日本語公式サイト
　http://concrete5-japan.org/
