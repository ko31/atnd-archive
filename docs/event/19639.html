Arduino 勉強会

2011年9月17日に会津大学にてArduinoの勉強会を開催いたします。内容はArduinoの入門者向けで、Arduinoの概要から入り、開発環境の基本的な使い方を覚えて、実際にブレッドボード上に簡単な回路を作り、LEDを点灯させたりセンサーの値を読み取ったり、ということを行う予定です。

講師はへちまネットの塩野 ( @kshiono ) が行います。講師自身ソフトウェア系の人間で電子回路には全く詳しくありませんが、Arduinoはそんな人でも簡単に入っていけるように工夫されている魅力的なプラットフォームです。電子回路のことがよく分からない方でもArduinoに興味がある方は、この機会に是非一緒に勉強していきましょう！

また、電子回路に詳しい方で勉強会をサポートして頂ける方の参加も歓迎いたします。

場所と日時

場所： 会津大学
日時： 2011年9月17日(土) 13:00～18:00

内容

  • Arduinoとは？
  • 開発環境の使い方
  • LEDを点灯させてみる
  • スイッチのオン・オフを読み取ってみる
  • センサーの値を読み取ってみる
    • 光センサー
    • 温度センサー
    • 湿度センサー
    • 人感センサー
  • 覚えたことを組み合わせて何か作ってみる

勉強会への参加について

  • 各自ノートPCを持参して下さい。(Arduinoやその他の電子部品はこちらで用意します。）

その他あった方が良いもの

  • C言語の基本的な知識（Arduinoで動かすプログラムはC言語で書きます）
  • 高校物理程度の基本的な電気の知識
