三度目の内輪向けサバイバルゲーム大会です。
基本的にゲーム、コミック、立体、及び編集等の内輪業界人とその周辺の方向けの身内ゲームとなっておりますが
趣旨をご理解頂けるご友人の方がいらっしゃればお誘い合わせの上ご参加下さると幸いです。

■参加申し込みですが、基本的に9/22日正午までの受付とさせて頂きます。
ご友人の方で参加希望の方いらっしゃれば本ページをお教え願えますようお願いします。
■レンタルに関してはこちらを参照の上、9/22までに当ページのコメント欄にご記入お願いします。

■こちらのサイトに登録されない（できない）ゲスト参加の方をお連れになられる場合
お手数ですがコメント欄に人数とその方のお名前（HNで可）をご記入頂くようお願いします

尚、連絡用共有Twitterアカウントとして『@TGT_notice』を利用しております。
当該アカウントにフォローリクエストを頂けますと告知等行えますので宜しくお願い致します。

■ちょっとしたお願い
締め切り等やその他諸事情で参加を伏せたい人も多いようなので、終了後ですが
・個人単位で『今日BB-JUNGLEでサバゲして来た』と言うのは自由。
・誰が参加していたかに関しては『触れない』方向でお願いします。
・TwitterやMixi等で『○○さんお疲れ様でした～』も出来るだけ空気読んで下さると助かります。
（自分が参加していた事を表明する事に問題ない人同士のやり取りについては特に問題ありません）
・ですので、基本的に本ページコメ欄使って頂くのを推奨します

・参加費2,500円
・昼食（フィールドに事前注文でお弁当もあるそうです　700円）
