description

「女子力」を身に付けたいけれど…　自分なりに努力をしているけど…
着飾ったり、女らしくしたり、それは、心から望んでしていることですか？
外側から鎧のように貼り付けただけの、「よそ行きの自分」を演技していませんか？

「ヴィーナスヨガ」は、女性のための、女性の心と身体に合った、女性が本来持つ能力・美しさを引き出すヨガです。

ヨガをベースに、さまざまな行法を合わせることで、女性の「本能」の部分を呼び覚ましていきます。

現代人が忘れがちな、「本能」を取り戻すことで、心身やホルモンのバランスが調和し、
女性でいることが楽しくなり、フェロモンや女性ホルモンがカラダの内側から巡り始めます。
自然なシステムを取り戻した女性のカラダは、女らしさや色気、美しさが内側からあふれ出てくるのです。

カラダが固くても、全く問題ありません。難しいヨガのテクニックも必要ありません。
女性限定ですので、リラックスした空間でワークを受講できます。

ファシリテーターのYukiは、ヨガ歴11年。
アメリカのヨガインストラクター資格を取得したのち、ヴィーナスヨガに出会い、発祥の地オーストラリアへ修行の旅に出る。

シドニーにて ヴィーナスヨガ ティーチャートレーニングを受講し、資格を取得。
その時に一緒に資格を取得した、日本、ドイツ、フランス、スロバキア、オーストラリア人の女性たちと共に、ヴィーナスヨガを広める活動を行っています。

当日の流れ：
１、自己紹介(１０分程度)
２、このヨガの特徴を説明
３、ヨガワークと行法(１時間程度)
４、質疑応答　懇談　

対象としている方：
「女子力アップ」という言葉に、焦りを感じつつも違和感を覚える方
「かわいい」を卒業して、「かっこいい」女性を目指したい方
女性のカラダの悩み（生理のトラブルや婦人科疾患）を抱えている方、女性のカラダのシステムを好きになれない、めんどくさいと思っている方
エクササイズ寄りのヨガに飽きてきた方
