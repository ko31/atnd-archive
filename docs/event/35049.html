description

OpenStackはオープンソースのクラウドインフラ構築ソフトウェアとして内外で注目されていますが、海外でのビジネス採用状況に比べると日本国内ではまだまだ実験・検証的な利用に留まっているのが現状です。

このような状態を改善し、日本国内でのOpenStackのビジネス利用の促進を図るため、情報交換の場として「OpenStackビジネス交流会」を開催いたします。

交流会は、OpenStackのビジネス活用に関するプレゼンテーションと、参加者間での交流を目的とした懇談会の2部構成で開催いたします。

OpenStackのビジネス活用にご興味のある方は、是非ともご参加いただきますよう、お願いいたします。

■開催概要
日時：2013年1月17日(木) 18時〜20時30分（受付開始17時45分）
会場：GMOインターネット株式会社内、【11F シナジーカフェ GMO Yours】
東京都渋谷区桜丘町26番1号 セルリアンタワー 11階
費用：無料（Sponsored by GMOインターネット）
定員：50名
幹事：藤原 優一（GMOインターネット）・宮原 徹（日本仮想化技術）

セルリアンタワーオフィス入口（道路よりエスカレーター上がり左側）で入館証（ゲストカード）をお渡しします。

■プレゼンテーションの内容
プレゼンテーション1：
『日本発のOpenStackソリューションの可能性』
宮原 徹（日本仮想化技術 代表取締役社長兼CEO）
日本仮想化技術では「BareMetal OpenStack」を開発し、OpenStackの次期バージョンである「Grizzly」へのマージを行っています。その他、OpenStackとZabbixとの連携など、OpenStackの新機能開発への取り組みを御紹介します。

プレゼンテーション2：
『MidoNetによって実現する新しいIaaSクラウドの形』
加藤 隆哉（ミドクラ　代表取締役）
今注目を浴びているクラウドインフラ構築プラットフォームのOpenStackとネットワーク仮想化の統合ソリューションであるMidoNetの組合せで実現できる新しい仮想データセンタの形と、それよるビジネス、オペレーション等に対するインパクトについてお話させて頂きます。

プレゼンテーション3：
『GMOインターネットにおけるOpenStackの活用と今後の発展』
塩原宏明（GMOインターネット）
商用サービスとしてすでに展開している「お名前.com VPS（KVM）」では、実際にOpenStackを活用することによってどのようなビジネスメリットを享受しているのか。また、OpenStackの進化を取り込むことによって、今後の顧客ニーズにいかに応えていく予定かをお話させていただきます。
