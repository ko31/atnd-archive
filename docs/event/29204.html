description

今月は、 「イケテルgem勉強会」 と題して、Ruby の gem をみんなで作ってみましょう!!

rails を始め、rubygems.org 等で配布されている gem を使った事がある方は沢山いると思いますが、gem を作った事がある方は意外と少ないと思います。

今回は、みんなで gem の仕組みを勉強して、作ってみたいと思います。

ぜひともお気軽にご参加ください。

日時

5月26日(土) 13:00〜18:00
(受付は12:30〜)

場所

住所: 福岡市博多区博多駅東1丁目17番1号　福岡県Ruby・コンテンツ産業振興センター 5階

タイムテーブル

時間 内容
13:00 – 勉強会の概要とRuby開発サーバの紹介
13:30 – Rubygemsの概要
14:00 – ハンズオン
15:30 – チームでGemを作ってみる
17:00 – チームの発表
17:30 – 公開スタッフミーティング

事前にやっておくといいこと

  • Rubygems Guide の Make your own gems を見ておくと良いです(英語)
    • http://guides.rubygems.org/make-your-own-gem/

必要なもの

  • 無線LAN付きのノートPC
    • Rubyがインストールされていること
      1. Ruby 1.9.3のインストール
        ※インストール後の確認例
        $ ruby -v
        ruby 1.9.3p194 (2012-04-20) [i386-minfq32]
      2. Rubygemsのインストール
        ※インストール後の確認例
        $ gem -v
        1.8.23
    • Windowsの方は、http://rubyinstaller.org/ でインストールすれば1も2もインストールできます
    • その他OS、MacOSやLinuxなどの方はお任せします。
    • 分からないところがありましたら、当日会場にてスタッフにお問い合わせいただくか、コメント欄にてご質問ください
