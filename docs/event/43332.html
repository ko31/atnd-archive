回答者の作られたノート:

銭青少年今日のテレビ 3 生息地よく知られているアーティスト、穏やかな「小さなタイガー チーム」のメンバーのいっぱい 20 年以上からの文字の彼がされているどのように何を変更していない、静かで控えめ、貧しいレトリック、強迫観念とトップ本当と厳格な自己管理がレベルの誠実な彼限り、急増した Shang ステージ、スクリーン、すぐに鮮やかな、スーパーブランドコピー舞台芸術と、人々 の健康には彼は間違いなく私達の見事ないわ、彼は年"小さな shuaihu"のタイトルの「ハンサム」word に今彼の人口があり、ない再びのように延々 と、関与してこの名声を任せることはできないもう一度"ハンサムなカバー」より自分自身をエキサイティングな。

貿易ルートの建設は質問に知らせて好きか嫌い、悪役の人

ささいなシックな ！: の Chi 鵬を招いて満足している詳細については、2011 年末にはささいを新しい映画を持っているポリ町のポスト プロダクションの Lu Zhanyuan という名前の男の役割を果たしてそこを聞いた他の演劇 3 ダウン再生する、このアプローチは、ほとんどの魅力は何ですか？

Banny 陳： Lu Zhanyuan 文字と私は物事をよりこの役割のよう焦点を当てて、専用とよりスムーズに再生私はまたどのような圧力のリリース ユニットは私の友人、私は思うプッシュ 3 ショー価値がある、仕事は楽しいものだったので。

ささいなシックな ！: 自然な役割を演じたいですか？

Banny 陳： こともありません。 自分自身を好きなロールの比較おなじみ。 プレイが好きで私は今だけ商業道日質問の下で部門ドラマの固定内部大きな悪役を演じた、またこの悪い実際にはかなりの、彼はかわいいは自己意識です左絶対右を拒否した私はまたかなりのようなこの役割によると、人々 の強い。 誰も彼との愛と憎しみはと考えています。

ささいなシックな ！: もつれたドラマが今までにこのような役割を再生するか？

Banny 陳： ただ始めたも言んだことができます、ときにはかなりのもつれのスクリプトを参照してくださいのかなり不本意ながら。 侯のエピソードを発見するこの役割は、ルイヴィトンコピー挑戦が私を参照してください yiqian が決して再生されなかったのタイプを決定次。 再度背部参照してくださいしていたスクリプトも思っていた戻っていたが、その後もが演奏する。 方法解釈の自分自身と私は部門のスタイリングの私もこの役割とブローカーを参照していますが、事故がだと思います。
