Firefox や Thunderbird の拡張機能、Jetpack などのアドオン開発、あるいはアプリケーションプラットフォームとしての Mozilla 関連の技術についてみんなで勉強する小規模ワークショップです。

その他、時間割や懇親会、Lightning Talk の応募などについて詳細は modest サイトのイベント情報ページをご覧ください:
https://dev.mozilla.jp/events/workshop03


会場には無線 LAN 環境を用意しますが、ネットワークキャパシティには限りがありますので、お持ちの方は各自 E-mobile などを持参いただけると助かります。


懇親会参加希望者は参加登録のコメントにその旨記入お願いします (勉強会の参加費は無料ですが懇親会費は 2000円 いただきます)。
参加登録時に書き忘れた人はコメント欄にでもご記入ください。


追記: 参加者が増えそうでしたので広めの会場を確保して定員を 40 から 60 に増やしました。


追記: ギリギリになりますが 70 名まで受け付けます。参加者多数の場合最後に来た人は講師の目の前の席になりますがご容赦ください。
懇親会も参加希望者が多数となった場合、本勉強会の登録先着順で懇親会参加者 43 名までとさせていただきます。予めご了承ください。


追記: 現在の懇親会申し込み状況からすると懇親会は希望者全員参加可能になると思います。(・・).
