description

異業種交流会ではさまざまな業種の方々と名刺交換できるので、
人脈が5人、10人、100人と急激に広がっていきます。

そして、あなたのビジネスにつながり可能性もあります。

私は過去、他人が開催している異業種交流会にただ参加していました。

するとたった3回、交流会に3回出席しただけで
年間60万円のコンサルティング契約を3件も締結できたのです。
（私は普段、ビジネスコンサルティングをしています）

それだけでなくセミナーの講演依頼もいただけて、
ものすごくビジネスにつながっています。

また、人脈が広がればいざあなたの困ったときに
たくさんの方があなたを助けてくれますし、
飲み仲間も見つかるかもしれません。

人脈はとても大切な「資産」です。

ぜひあなたの資産を築いてください。
