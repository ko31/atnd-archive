ゲーミフィケーションとは一体何か？

既に死語化が囁かれている「ゲーミフィケーション」と言う「コトバ」。

しかし、ビジネスやサービスの設計において、我々の世代以降にとっては
これほど浸透力のある概念も、他にはなかなか見当たりません。
それ故に、死語化した後もその概念や考え方はとても有効です。

コトバが死んでしまう前に、ゲーミフィケーションを学び、体験して見ませんか？

座学２時間、ワークショップ１時間ぐらいで予定しています。

対象者

ゲーミフィケーションとは、
何らかのビジネスやサービスを構築、分析、改善するための手法の一つです。
従って、上記の業務に従事している人が対象となります。

ゲームの話は沢山出ますが、ゲーム企画 だけ の話ではありません。
むしろ、ゲームの企画手法を、ビジネスに転用する話がメインです。
このあたりはあしからず。

PowerPoint か KeyNote の入っているPCを持参してくれるとありがたいです。
