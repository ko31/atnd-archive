15:00-15:10 会場説明 　
15:10-16:10 01. 「MODO First Impression -最強のポリゴンモデラとの出会い-」
MODO初心者が語るMODOの魅力とは…　最強のポリゴンモデラと名高いMODOへの想いを、機能を全然把握していないこの私が勢いに任せて熱くぶつけます。突っ込みどころ満載、ご指導ご鞭撻絶賛大募集のゆるいセッションとなりますが、勉強会の敷居を下げるためと思ってご容赦ください。もちろん初心者ながら伝えたいことは多々ありますので、他のモデラをご愛用の方もぜひご覧ください！		 デザインサテライツ
新井 克哉さん
16:10-16:20 休憩 　
16:20-17:20 02. 「Unity によるライブコンサート演出の試み」
6月17,18日に東京・青山にて開催された松浦雅也氏のライブコンサートにおいて、Unity をベースとしたVJシステムが用いられます。このシステムでは、ゲームエンジンというUnityの特性を活用し、複雑な3D CGシーンとライブパフォーマンスのリアルタイムな同期を実現しています。このセッションでは、本システムにおいて用いられた技術や手法の紹介を通して、リアルタイムな映像表現にUnityを用いるというアプローチの可能性についてお話しします。		 Unity Technologies Japan 合同会社
高橋 啓治郎さん
17:20-17:30 休憩 　
17:30-18:00 03. 「Live2Dのモデリング〜モーション付け」
live2Dで始めるモデリング、live2D Cubism2.0の変更点、2.5D関連のソフトをざっくりと紹介します。		 naotaroさん
18:00-18:30 04. 「なぜなにブレンダ」
最近やたらとネット上で叫ばれてるBlender。一体なぜBlenderを使うのか。
UIの難解さと、魔の右クリックを超えた先にあるクリエイティビティとは。
Blenderを選択する理由と連携が可能なソフトウェアについても可能であれば解説いたします。		 ハヤシヒカルさん
