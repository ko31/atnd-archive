CSS Nite in OSAKA, Vol.22につきまして、多数の参加表明ありがとうございます。

当日の会場ですが、部屋の大きさには、定員に対して若干の余裕があります。
早くお申し込みいただいた50名の方には机のある席を用意する予定ですが、
補欠の方であっても、椅子のみ、あるいは立ち見でも構わないということでしたら、
当日、お越しいただいて構いません。

8/28（土）大阪産業創造館で行われるCSS Nite in OSAKAのプレイベントです。

今回はiPhoneアプリ「健康計算機」などをリリースしている中学生プログラマ、Tehu（てふ）さんが「増減カウンター」をアプリとしてつくります。

当日は実機テストや公開ではなく、エミュレータを使ってiPhoneを擬似的にPC上で操作出来るようにします。

プログラミングの濃い内容ではなく、Xcodeという開発環境をつかって、 なるだけコードを書かずにiPhoneアプリのUI制作を実現できるか？というテーマのセッションを行う予定です。
（Objective-Cの専門講座ではありませんが、若干のコーディングは入る予定です）

一般的に難しいというイメージを持たれがちのiPhone／iPadアプリ制作やObjective-C言語ですがGUI制作ツールも存在し、意外と知られていないフローも存在します。
簡単なインターフェイスを制作する内容の一部を取り上げます。

対象：他プログラミング言語でも基本的な構文は読んだ事がある方やプログラミング初心者を対象とします。
（開発を専門にされている方ほど高度な事は行いません。）

公演時間：
約1時間を予定しています。
（今回は出演者が未成年ということで17時開演とし、18時半ほどで終演とします）
