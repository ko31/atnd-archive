SocialGameJam概要

GameJamとは？

GameJamは参加したプログラマー, グラフィックデザイナー達が複数人でチームを組み、
与えられたお題に沿ってチームごとに開発するイベントです。

SocialGameJamとは？

ソーシャルゲームにフォーカスを当てたGameJamとなります。
WEBとアプリ両方でソーシャルゲームを作成します！
いままでに無いソーシャルゲームを作成しましょう。
ソーシャルグラフを利用したいままでに無いソーシャルゲームを作成しましょう。
最終的にWEB場で成果物を披露していただきます。

対象者

以下に当てはまる方のみご参加ください。
1, コンシューマー・WEB・同人問わずゲームを作ったことがあるかた。
2, ２日間でゲームを完成させる熱意があるかた。
3, ゲーム制作に意欲があるかた。
4, 面白いソーシャルゲームを作りたいかた。
