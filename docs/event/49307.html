■イベント概要
新言語「Hack」はFacebookが2014年3月24日オープンソースでリリースしたプログラミング言語です。HHVM（HipHop Virtual Machine for PHP）向けの開発言語で，静的型付き言語の性質を取り入れPHPでの素早い開発を実現し，PHPとの互換性があるのが特徴です。

2014年4月1日，世界初となるHack解説書が電子書籍として技術評論社より発売されました。

今回はその出版記念として，著者の米林正明氏（株式会社Abby代表取締役社長）をスピーカーに迎え，「Hack」の新機能，魅力，今後の可能性についてご紹介いただきます。

後半では，米林正明氏と馮富久（株式会社技術評論社）の両名によるトークセッションを実施いたします。
