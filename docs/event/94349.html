description

会社のゴールでなく、あなたの人生のゴールを考える。
このカフェ会の目的は、人生を変えたい前向きな人達が集まり、少人数で意見交換する事により、深い話しができ、新しい価値観に触れ、それまでの考え方・行動を変えて人生を変える一歩を踏み出す（行動する）きっかけの場にしたいと思っています！

＜カフェ会はこんな人にオススメ＞

① 夢を持って仕事をしたい
② 世の中の役に立ちたい
③ 同じ価値観を共有できる仲間がほしい
④ 将来の進むべき道を模索している
⑤ 新しい自分の可能性を見つけたい
⑥ 他人の意見や価値観にふれ、行動のきっかけを掴みたい

カフェ会の流れ

・自己紹介
・フリートーク
・お勧め本があれば紹介

このセミナーで学べること
私は入社13年目の33歳　某大手の会社員です。私がこのカフェ会を主催している目的は

私自身が皆さんと出会い、新しい考え方や意見に触れ、自分の中の古い価値観を打破し、
どこででも生きていける自分に変革するため（自分のため）にやっているというのが目的です＾＾

皆さん！ 自分のために、何か具体的な行動を起こしていますか？
具体的な行動とは　例えば

・ 本を読む
・ カフェ会に参加する
・ 勉強会やセミナーに参加する

など　自分の空き時間を、自分の将来のために投資する事です！

同じ会社、同じ愚痴を抱える同僚と話しをしても現在の価値観から抜け出すことはできません！

他のフィールド、あるいは、自分の行きたいフィールドにいる方と触れ合うことで
自分の価値観を変えることが何より大事なのです。
是非一度ご参加いただき、ご自身の夢、目標、考え方、悩みを共有しませんか？

将来目標が明確な方でも、他人の夢、目標、考え方、悩みを聞くことで更にそれらが具体化し、ブラッシュアップされるかもしれません。

このカフェ会の出会いが将来のあなたの重要なパートナーとの出会いの場になるかもしれません。
是非みなさんのお話を聞かせてください。

仕事のこと、趣味のこと、将来の目標、なんでも良いです。
何かしたい、何か始めないといけないと思っているだけでは何も変わりません。

まず行動してみましょう！
最初の一歩を踏み出すだけで世界は大きく変わります。

◎期待できる効果

① 最初の一歩を踏み出す行動があなたにとって人生を変える出会いがあり、
　 大きな転換が起きる可能性がある！

② 他のフィールド、あるいは、自分の行きたいフィールドにいる人と触れ合うことで自分の価値　　　観を変えることができる。

③ やりたい事を見つけられるヒントが得られる。

注意事項
セールスや勧誘、コーチング、ネットワークビジネスをされるいる方、
会の健全性と被害防止のためお断りしております。

◯主催者自己紹介

私は、３３歳の某大手企業につとめているサラリーマンです。
私がこのカフェ会をを主催している目的は、
私自身２０代で人生の壁にぶち当たったときにこんな人、こんな場所が
あったらもっと早く人生に対して真剣に捉えていたかもしれないと思ったのが
キッカケです。

２０代の頃は私自身、仕事や恋愛、趣味と多くの事を経験させて頂ましたが、
２０代の終盤に差し掛かったときに、働いても働いても増えない貯金、
給料、そして時間も無い状態で今後、今の会社で働いても幸せになれる
だろうかと思っていた頃がありました。
同期、友達に話しても、なんとかなるよ！とか言われ、なんとかなんないから
相談してんだよ！って心の中で思い、上司を見ていてもあんな覇気の無い、目標すらも
無いような上司にはなりたくないなと思うのが常でした。

あなたも同じ状況ですか？

そうなんですよ。人生を前向きに捉えた時に前向きな人がなぜか周りにはいない！！
とういう状況ですね。

そんな２０代の思いや考えに微力ながら相談にのれればなと思い、このカフェ会を設けてます。

今を変えるためには習慣を変えなくてはいけません。
環境を変えて人生に前向きな人とつながる事を心がけなくてはいけないのです。

会社の飲み会は楽しいですか？
それはあなたの人生のとって優先度が高いですか？

同僚や上司の愚痴は気持ちいいですか？
それはあなたの思考をネガティブなほうへ引っ張っていませんか？

あなたは周りを変えようとしてませんか？
変わるべきはあなた自身なのです。

私自身、この会を通して人生に前向きなひと、異業種、経営者、時には
国の政策アドバイザー等、普通では考えられない人とお会い出来るように
なりました。毎日刺激を受ける日々です。目標もしっかり出来ました。

このカフェ会がちょっとでもあなたの学び、行動への呼びかけになればと思います。
