【その１】

コムシス情報システム株式会社 様によるセッション「課題管理システム【Spur】」のご紹介です。

今までExcelで行っていた課題管理を、OPEN INTRA-MARTプロダクトを利用して
Webシステム化をした事例を紹介していただきます。

  • 全体の課題、PJ内の課題等、絞り込み表示がしにくい。
  • 過去履歴が活用しにくい。
  • 現在時点の担当は誰なのか分かりにくい。

こういった課題を、OPEN INTRA-MARTプロダクトを利用して
どのように解決したかを紹介していただく予定です。
intra-mart Web Platformでの運用も考慮し構築されたシステムとのことで、
この点も見逃せません。必見ですっ！

【その２】

こんにちは【Hello, World!】（ Node.js 編）〜初心者の 初心者による 初心者のための ハンズオン〜

  • アジェンダ
    • Node.js で Hello, World! (標準出力 編)
    • Node.js で Hello, World! (http.Server 編)
    • Node.js で Hello, World! (http.Client 編)
    • Node.js で Hello, World! (ファイルシステム 編)
    • Node.js で Hello, World! (WebSocket 編)
    • まとめ（Hello, World! 達の成果を結集）
  • 環境構築方法
    • ●STEP 1
      * VirtualBoxをダウンロード＆インストールしてください。
      *       http://www.virtualbox.org/wiki/Downloads より ダウンロードできます。
    • ●STEP 2
      * ハンズオン用のVirtualBox仮想マシンイメージをダウンロード＆解凍して下さい。
      *       http://oss.intra-mart.org/study_meeting/2010-11-09/Ubuntu-8.04-NodeJsHandsOn.vdi.zip
      * （ファイルサイズ、デカイです。1.58 GBです。すいません。。。）
    • ●STEP 3
      * 仮想マシンの登録と起動してください。以下サイトが参考になるかもしれません。
      *       http://www.ubuntulinux.jp/products/JA-Localized/virtualbox
    • ●STEP 4
      * ハンズオンで使用する、ソースコードをダウンロードしてください。
      *       http://oss.intra-mart.org/study_meeting/2010-11-09/2010-11-09_NodeJsHandsOn_src.zip
  • 参考情報
    • http://oss.intra-mart.org/pipermail/open-im-dev/2010-November/000168.html
