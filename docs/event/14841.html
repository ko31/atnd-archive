title FBF　ブランディング＆マーケティング
catch 『ブランディングとソーシャルネットワークの関係』
description

マスメディアによるトップダウン的なマーケティングから、ユーザーによる
ボトムアップのマーケティングを可能にしたソーシャル・ネットワーキング・
メディア。

人が繋がっていくこと、情報を共有すること・・・
Facebookを中心としたソーシャル・ネットワークの概念は
今後のマーケティングに重要な意味を成します。

しかしソーシャル・ネットワークの本領は、外的要因に繋がるマーケティング
よりも人のマインドの内側に訴えかけるブランディングで、
その力を発揮します。

Facebookのブランディング活用を考える前に、本来のブランディングについて
しっかりと理解する必要があります。

この度、ブランディングやマーケティングにおいて数々の実績をお持ちの
宇佐美清氏をお迎えし、ブランディングの考え方や、
ソーシャル・ネットワークとの考え方についてお話いただきます。

大変貴重な機会です！
広告・マーケティングに携わっている方や、ソーシャルネットワークをビジネス活用したい方はぜひご参加ください。

■日時：平成２３年４月２２日（金）　
　　　　１８時会場　　
　　　　１８時３０分スタート～２１時終了

■場所：福岡県福岡市中央区天神2丁目14-8 福岡天神センタービル8F

■参加費：３，０００円

■講師：宇佐美清氏（USAMIブランディング株式会社代表）
