オンラインゲームの開発を一気に加速させる Photon Cloud
今回のワークショップは、1月24日（金）〜26日（日）に開催される 『GlobalGameJam2014』 の【緊急対策編】と位置づけ、GameJamですぐ使えるPhoton Cloud の使い方を実践します。

　
【 GGJ緊急対策編 ラインナップ 】

① ロビー作成
② ルーム作成
③ ルーム入室
④ フレンド検索

Photon Cloudの無料アカウントを使って、２時間のワークショップで①〜④をすべて実施します。これをマスターすれば、あとはゲームコンテンツを考案するだけ。

GameJamの短い時間でも、オンラインゲームが作れちゃうんです！
