ギークハウス沖縄で家賃そんしましょう。

※家賃そんとは
家賃を稼がなくてはいけないギークハウス沖縄ですが、
「自分の金を使わずにサービスとか使ってそれで家賃が賄えればいいね」という夢物語を思い描いてまして、

それを頑張って実現できるよう、毎週土曜日に、家賃そんという名のもくもく会を実施しようと思います。

せっかくなので、ギー沖の静かな場所でもくもく開発をしたい、静かな時間を過ごしたいという方はご参加ください。
