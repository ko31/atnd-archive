description

◆昨今、ソーシャルビジネスや社会的企業という言葉を耳にする機会が増えてまいりました。しかしその言葉の意味を判らない方が多いのではないでしょうか。行政をはじめ、社会的には重要な位置づけとされてきています。そのような社会的背景を踏まえて、主催、日本政策金融公庫　共催、北九州市のソーシャルビジネスセミナーを開催することとなりました。

◆社会起業家育成のプロフェッショナルが、ソーシャルビジネスに興味・関心のある方、NPO法人、任意団体、ボランティアグループ、起業を目指している方を対象としたソーシャルビジネスセミナーを開催いたします。
地域で社会的課題を解決し、安定的かつ継続的な雇用を創出するいわゆるソーシャルビジネスを支援するために、数多くの社会起業家を育てた専門家の講演及びソーシャルビジネス支援機関の施策・事業説明を行います。まさに、「ソーシャルビジネス成功の秘訣を惜しみなく伝授！」するセミナーです。参加費無料のセミナーとなっておりますので、この機会にぜひご参加下さい！

■内容
第1部　主催機関等施策・事業説明…◎九州経済産業局 ◎日本政策金融公庫
第2部　ソーシャルビジネス成功の秘訣
講師：社会起業大学・九州校　専任講師　前川　卓三氏
第3部　参加者同士の交流会（希望者のみ）

■講師　前川　卓三
大学卒業後大手紡績会社にて12年勤務後、海外取引のベンチャー起業を立ち上げる。退職後は、グランドワーク三島「内閣府地域雇用創造事業」10億円プロジェクトに参加。豊富なビジネス経験を活かして、200人以上のビジネスプランの作成をサポート。その中から多くのビジネスコンペ受賞者を輩出した。特定非営利活動法人グランドワーク三島管理部長を経て現職。
