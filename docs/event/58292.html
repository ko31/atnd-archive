FxOSコードリーディングって？

Firefox OS (FxOS) のソースコードを読んで何かしら言い合ったり発信しあったりするグループです。

大体毎月 Mozilla オフィスでミートアップという名のオフ会をしています。
それ以外は集まり毎に調整さんなどで予定聞いてイベント立てています。
※当コミュニティではGMOインターネット様よりコミュニティ支援としてConoHa VPSの提供を受けています。
ConoHaについてはこちら→ http://www.conoha.jp/community
