catch 「検討しておきます」を「今すぐお願いします」に変える商談ストーリーを作る
description

商談の最後で「今すぐお願いします」と言われることを増やしたい方へ

商談の最後に「検討しておきます」と言われることがありますよね。
言うまでもなく、そのほとんどは断り文句で、本当に検討する方は少ないですね。
ではなぜこのように断られてしまうのでしょう。
その原因のひとつは、見込客にプロダクト・サービスの魅力がよく伝わっていないことがあります。
伝え方を変えるだけで、「今すぐお願いします」に変わる可能性は十分あります。
ではどのように変えればよいのでしょうか？

方法は大きく２つあり、ひとつは、見込客があなたのプロダクト・サービスを使うと実現できる姿を、ストーリーで伝えること。
もう一つは、あなたの会社がなぜそのプロダクト・サービスを販売しているか？その背景経緯、理念、ビジョン・ミッションのストーリーを伝えることです。

なぜなら、消費者はプロダクト・サービスのクオリティ以外に、あなたの会社に共感できるか？ということを重視して購入の意思決定を行なっているからです。

しかし、広告コピーやwebサイトの原稿を書く時、ストーリーを考えて制作する方は少ないのではないでしょうか？
そもそもそれはわかるけど、一体ストーリーってどうやって作ればいいの？という方がほとんどだと思います。

このワークショップでは、ストーリーを構築するための思考フレームを用意し、ファシリテーターがリードしながら参加者同士がディスカッションしながら見込客に伝えるストーリーをつくり上げます。

異業種の参加者からのアイディアが、社内スタッフだけで行うときには考えもしなかった気づきを与えてくれます。

もう「検討しておきます」と言われたくないという方はぜひご参加ください。
