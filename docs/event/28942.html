平塚で読書会ならぬ，読書飲み会を開きたいと思います．

一人一冊，最近読んだオススメの本を持ち寄って，

それを紹介しながら，ピザとビールを飲む会です．


あまり周りに本について熱く語れる仲間がいなかったため，

個人的に人を集めて，開催したいと思います．


今回が初回なので，

基本的にみんな初対面です(笑)

なので湘南地区で読書仲間が欲しい方はどしどし参加してみてください．


どれくらいの人が参加してくれるのかわからないですが，

基本的に4人以上集まったら開催したいと思います．


開催場所は，平塚駅から徒歩5分程の所にある

イタリアンレストランを予定しています．


もし，何か質問や提案があれば，

下記のTwitterのアカウントに気軽に連絡してください．

https://twitter.com/#!/Atsushi_twi


是非参加をお待ちしております♪
