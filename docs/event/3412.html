description

クックパッドとシリウステクノロジーズの合同勉強会の申し込みフォームです。

申し込みは締切ました。

※公式ハッシュタグ⇒　#ciripad

日本最大の料理サイトを運営するクックパッドと、今注目のロケーションベース広告配信サービス、「AdLocal」を運営するシリウステクノロジーズ。
その実践的な開発の裏側を、両社のエンジニアが発表いたします。

当日は、「実践的な開発の裏側」というテーマで、参加者の方の発表も受け付ける予定です。ぜひご参加ください。
発表時間は５分です。発表をご希望の方は、申込時のアンケートフォームより、プレゼンタイトルと内容を書いて申し込みをしてください。参加者抽選の際には、プレゼン発表される方を優先させていただきます。

参加者の皆さまと有意義な情報交換をさせて頂ければ幸いです。
