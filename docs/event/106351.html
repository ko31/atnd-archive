初心者による初心者のためのMySQLクエリチューニング

概要

＼初心者だからこそ初心者目線での勉強会を開催します／

クエリチューニングってなんぞや？という人が、
スロークエリを見つけて原因を分析し、
適切なインデックスを貼れるようになることを目指します！

勉強会の後は懇親会！
ざっくばらんにお酒とピザを食べながらお話しましょう！

  1. クエリチューニングとは
  2. スロークエリを見つける

  4. インデックスを張る
