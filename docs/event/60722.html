description

ドル円７年ぶり安値、2015年も円売りは続くのか。。。

アメリカ量的緩和終了の中、日銀は金融緩和を発表し、
2014年終わり間近に120円台の新たなステージに到達！

直近の弊社セミナーにて講師陣からは、 今後160円を目指すのではないか？との声も飛び出す中、

2015年の為替はいったいどんなトレンドに。。。。。。。

もうこの流れに乗り遅れたと思われた人もいるはず。。
しかし、2015年はドル円だけではない？
新興国通貨、資源国通貨など次なる狙いは？？

低金利時代だからこそ投資をしてみたいと思っている方必見！！ 今トレード中だけどなかなかうまくいかない。何が原因なのかわからない。。という方必見！！ 外貨預金とFXの違いは？双方のメリット・デメリットとは？？わからないことだらけという方必見！！

ここだけでしか聞けない？話にも注目！！

ファンダメンタル重視で、市場が何に注目しているのかを分析し、その材料が生み出す流れに沿ったトレード戦略をベースにしている山岡氏が初登場！！

インターバンクディーラーとして、１０年以上のキャリアを持つ
