ユーザー会の目的

サービスデスク、サポートデスク、カスタマーサポートなどの業務に特化したオープンソースのOTRS（Opensource Ticket Request System）を知るため／研究するための勉強会です。
海外ではAsteriskを始めとしたIPテレフォニーと組み合わせてコールセンター向けのソリューションとしても広く使われており回を重ねるごとに事例や使い方など様々な角度から勉強していきたいと思います。
ユーザー会ではOTRS通じて、IP電話サービスや帳票ツールなど様々なOSSと連携し、ITサービス（ヘルプデスク、ITサービスマネージメント）の運用を支援する活動を行って参ります。

OTRSはITSMを運用するための機能もありITシステムの運用管理も行えます。

（２）OTRS3.3.8のインストール
