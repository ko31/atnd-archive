『ATEAM TECH MeetUp_Vol.09』のテーマは「エイチームフィナジーのバックエンド開発」！

スマホ向けゲーム開発や日常生活に密着したWebサービスなどを運営するエイチーム。Webエンジニア向けカジュアル勉強会（トークセッション）＆交流会『ATEAM TECH MeetUp』を大阪（梅田）にて開催します。

エイチームのエンジニアリングについて、開発事例を交えてお話します。また、今回の『ATEAM TECH MeetUp』では、参加者のライトニングトークセッション（以下：LTセッション）の時間を設けております。参加エンジニアのアウトプットの機会を増やすことにより、エンジニア同士の交流を促進し、会社や組織などの垣根を越えた横のつながりを深めることを目的としています。みなさんのご経験、開発事例などについても、積極的に発信していただきたいと思います。

「ATEAM TECH」とは

「ATEAM TECH」はゲームやインターネット業界で働く技術者向けに業界の技術トレンドに関する勉強会や交流会です。知識のアウトプット・インプットの場を設け、新しい気づきや成長につなげる機会を提供することで、技術力の向上や業界のさらなる発展を目指します。自社サービスならではの考え方や業界の技術トレンドに関する共有、交流の場を通じてより深い知識を得て、さらには参加者同士でのつながりもできればと考えております。

セッション詳細

セッション①『段階的なシステムリプレースを実現するデータ同期』

  • 所属：株式会社エイチームフィナジー　テクニカルソリューション部
  • 担当サービス：ナビナビキャッシング
  • 得意/興味がある技術分野：PHP

セッション概要

PHPで作られたWebサービスを、Rubyを使った新システムへとリプレースするにあたり、URLごとに段階的に新システムへと切り替えていくことになりました。リプレース中に新システムと旧システムとを同時に稼働させるために行った、新旧システム間のデータ同期についてお話します。

セッション②『Rails APIの新規開発を担保するテストについて』

  • 所属：株式会社エイチームフィナジー　テクニカルソリューション部
  • 担当サービス：ナビナビキャッシング
  • 得意/興味がある技術分野：Ruby / Ruby on Rails

セッション概要

新卒社員として配属された部署で、新規事業のフロントエンドとバックエンドの両方の役割を任されました。担当したプロジェクトではいくつかのドキュメントツールやテストライブラリを使用しています。バックエンドのAPIはRailsで開発しており、その際のテストとして使用しているrswagについてお話します。

ライトニングトーク①『Next.jsはどのようにSSR(サーバーサイドレンダリング)を実現させているか』

  • 所属：株式会社エイチームフィナジー　テクニカルソリューション部
  • 担当サービス：ナビナビキャッシング
  • 得意分野：PHP

LT概要

JavaScriptのフレームワークであるNext.jsにはデフォルトでSSR機能が備わっています。SSRはどのように実装されているのか、そもそもSSRは今必要なのか、についてお話します。
