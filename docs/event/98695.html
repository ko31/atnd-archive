今回は戸越八幡神社社務所様にて外交評論家の加瀬英明先生をゲストに迎え、近年産業革命・IT革命と目をみはる産業の発展の一方、
「環境破壊、戦争、道徳なき経済発展による弊害と深刻な社会問題」を「日本の精神性、神道」が世の中を調和する解決策に

繋がるのではないか、との想で 「－世界平和を願う日本の心・神道とは？－」の講演会を開催させて頂く運びとなりました。

同時に、伊勢神宮様に「映画界初の奉納」（平成30年年5月）がされた「神楽鈴の鳴るとき」の試写会を同時開催します。

映画通して、深く、楽しく、神道・神社を知って頂けたら幸いです。

当日は本殿にて昇殿参拝をしてから、映画鑑賞、加瀬英明先生の講演会というスケジュールになります。なお今回特別に戸越八幡神社様から

お心使い頂き、昇殿参拝御祈祷を無料で儀礼頂ける事になりました。「神楽鈴の鳴るとき」にも上映される「舞」を、巫女不二子も奉舞させて頂きます。

大切な方と・ご家族で是非、素敵な思い出をお持ち帰りください。（お子様も同伴可）

「清め祓い・学び・直会」を通して皆様の益々のIYASAKA（弥栄）なご縁になります様、御祈念申し上げます。
