description

北海道パケットキャプチャ勉強会「Hokkaido.cap」の第3回勉強会です

  • 本勉強会は1ヶ月に1回の開催を目標にし、平日2時間程度のナイトセミナー方式で実施しています。
  • 参加費は 【無料】 です。
  • 参加者全員でわいわいやる形式です。
  • 実際に操作しながら勉強しますので、各自Wiresharkインストール済みPCの持ち込みをお願いします。
    PCの持込が出来ない方向けにPCの貸し出しは行いたいと思いますが、その場合Windows PortableApps版を使用しますので、USBメモリだけは持参願います。
  • 後援 : 一般社団法人LOCAL インフラ部

第3回のメニューは以下のとおりです

  1. ネットワークトラブル ケーススタディ（基礎編）
  2. パケット解析 CTF
  3. ライトニングトーク
    　- Windowsでのパケットモニター作成 : 札幌ワークス SINさん
  • 会場は18:30より入室可能です。
  • 内容は初心者向けです。これまでパケット解析をしたことのない人でも大歓迎です。
  • 実践パケット解析 をベースに実施しますので、本書を持っているとより理解が深まると思います。
    今回は本書第7章の内容をベースに、Wiresharkを使ってネットワークトラブルを切り分けする手法について勉強します。（本を持っていない方向けに簡単なスライドは準備します）
  • 後半は、CTF（Capture The Flag）形式で、パケット解析を題材にしたクイズにチャレンジします。
  • LANアナライザとして、Wiresharkを使用します。
    初参加、またはWiresharkを使った事がない方は Hokkaido.cap#1の発表資料「Wiresharkの使い方（基礎編）」 を参照しておくとよいと思います。
  • わからないところがあれば、みんなで教え合いながら楽しくやりましょう。

LT発表者を募集中です！

  • 5～10分程度で、パケット解析の経験談、ネットワークセキュリティに関する話題、イベントの宣伝など、内容は何でも結構です。

LT発表をご希望の方、および不明点・問い合わせは Twitter:@yamaki_panda または Twitter:@smokeymonkey までご連絡ください。
