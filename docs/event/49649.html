description

WordBenchは WordPress ユーザーのための地域ソーシャルコミュニティです。
京都を中心としたコミュニティとして2014年度 第04回を開催します。

WordBench京都はWordPressの好きな人が集まってみんなで作るコミュニティです。
登壇者もゲストを呼ぶのではなくコミュニティの中から話したい人が話します。
なので登壇者も初心者が多いのがこのコミュニティの特徴！
・初心者だけどWordPressを導入して苦戦したことを話します！
・WordPressをバリバリ使ってますがこのプラグイン使って苦戦したこと話します！
・書籍などにはよく書かれているテーマだけどデザイナーにはわかりにくいので間単に説明して欲しい！
など毎回登壇者によって勉強会のレベルや内容は異なります。
ガッツリしたセミナー形式から情報交換など様々ですがアットホームなコミュニティですので気軽にご参加ください。

WordBench京都

開催日時

2014/04/27(日) 13:50～17:00
　※13:20から会場に入れます。

対象者

WordPressに興味がある方なら誰でもOK！
これからWordPressを始めようと思っている全くの初心者も大歓迎です。
京都住民でなくてもOKです。
WordBenchに加入していない方でも大丈夫です。
2014年度 第04回ですが毎回独立した勉強会です。初参加OKです。

コンテンツ

今回は、WordPressをより快適に使うための情報をみんなで共有しましょう！
いくつか話は準備します。

(1)WordPressを使ってて困った！どうする？
WordPressフォーラムのご紹介など。

(2)良いプラグインがあるんだけど日本語がない。
お客さんに納品するときに英語だとね。
じゃ日本語化しちゃえばいい！
※その日本語化ファイルをプラグイン開発者にも渡すこともできるよ！

(3)このプラグイン便利なんだけどずっとバージョンアップしてくれるかな。
じゃプラグイン評価して誉めちゃいましょうｗ
評価するだけで開発者は気分いいらしいですよ。
こういう機能欲しい場合、要望方法わかれば拡張してくれるかも。

(4)もうすぐ新しいバージョンがでるけど、今のサイト動くかな。
テスト用バージョンがあるよ。そこで動作検証を！
気づいたら報告して修正お願いすると本番リリース時点ではちゃんと動くよ。

などなど
