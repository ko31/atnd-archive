創業6年で7つのインターネット事業を立ち上げ、組織を560名まで急拡大させてきたビズリーチの創業者である取締役2名がビズリーチ創業時のストーリーを、本音を交えて赤裸々に明かしながら、「事業立ち上げ成功方程式」についてお話します。

当社は、代表取締役の南、取締役の永田と取締役の竹内の3名が経営チームを組み、それぞれの強みを活かしながら、急成長を牽引してきました。

【次々に事業を生み出すビズリーチ】

ビズリーチは、日本とシンガポールで複数のWebサービスを展開するインターネット企業です。

業界で不可能と言われた求職者課金モデルを日本で初めて展開し、ハイクラス転職市場に革命を起こした「ビズリーチ」から始まり、人材・教育分野で新しいインターネットサービスを次々に生み出し続け、従業員数は毎年倍増しています。

この中で、驚くべきなのは、ビズリーチの黎明期である創業1年後の2010年に、ビズリーチの一事業としてセレクト・アウトレット型ECサイト「LUXA」を立ち上げ分社化し、ルクサは合計13.3億円の資金調達を実施し、会員数100万人を超えるまで成長したことです。そして、従業員数約150名のルクサは、本年5月にKDDIの連結子会社となりました。

これらの”再現性のある”事業創りを牽引してきたのは、永田と竹内の二人です。

【大手企業でインターネット事業を創ってきた二人】

インターネット黎明期からソニーグループ、AOLジャパン、ヤフー、match.comと、日系大手企業・外資系企業でインターネット事業のプロデュースやマーケティング職を歴任してきた永田と、リクルートの基盤フレームワーク開発などに従事してきた竹内。日系大手企業・外資系企業での経験を活かしてベンチャーを立ち上げた二人ならではの経験談もお話させていただきます。

【外資系コンサルファーム出身のモデレーターが鋭く質問】

そして、モデレーターには、外資系コンサルティングファーム、ベンチャーの外資系インターネット企業を経て、ビズリーチに参画した30代前半の津村がみなさんを代表して、二人に鋭く質問していきます。

これまで、事業創りのプロである二人が一緒にみなさんの前で事業創りの秘訣について話すことはありませんでしたが、初めてお話させていただくことになりました。一回限りの開催となりますので、ぜひこの機会をお見逃しなく！

ビズリーチについて

「インターネットの力で、世の中の選択肢と可能性を広げていく」をミッションに、 ・管理職とグローバル人材に特化した会員制転職サイト「ビズリーチ」 ・20代のためのレコメンド型転職サイト「careertrek」 ・セレクト・アウトレット型ECサイト「LUXA」（現在は分社化） ・ビズリーチのアジア版「RegionUP」 ・日本最大級の求人検索エンジン「スタンバイ」 ・キャリア女性のための転職サイト「ビズリーチ・ウーマン」 ・友達とクイズで競える学習アプリ「zuknow」 を運営。従業員数は564名（2015年8月現在）。 サービス詳細

講演概要

永田 信 / 取締役

  • 1995年、南イリノイ大学プロダクトデザイン学科を卒業後、インターネット黎明期からWebデザイナーとしてキャリアをスタートさせる。その後、株式会社ソニー・ミュージックコミュニケーションズに入社し、エンタテインメント領域とエレクトロニクス領域のWebサイトやECサービスのディレクション業務に従事。2000年にはAOLジャパン株式会社へ転職し、Webサイトのプロデュースやマーケティング、海外チームとのプロジェクトを手掛ける。その後、ヤフー株式会社にてソフトバンクとの共同事業であるYahoo!BB事業の立ち上げに従事し、2004年には世界最大の婚活サイトmatch.comの日本支社の立ち上げからCOOとして日本最大の婚活サイトにまで成長させる。2008年、株式会社ビズリーチの創業期にマーケティング担当として参画し、取締役に就任。2010年にはルクサ株式会社の創業にも従事。

竹内 真 / 取締役

  • 2001年、電気通信大学情報工学科を卒業後、富士ソフトABC株式会社（現・富士ソフト株式会社）に入社し、エンタープライズサービスを中心にさまざまなソフトウェアを開発。2007年に富士ソフトを退職後は、フリーエンジニアとして株式会社リクルートの基盤フレームワーク開発などに従事。2008年には株式会社ビズリーチの創業に参画し、CTOとしてサービス開発を手掛ける。また同年、Web開発・制作会社である株式会社レイハウオリを設立し、代表取締役に就任。2010年、ルクサ株式会社創業に伴い、同社CTOとして約1年半の間、立ち上げに従事。現在は株式会社レイハウオリ代表取締役と株式会社ビズリーチの取締役 スタンバイ事業本部長を兼任する。社外では、mobyletメインコミッターなどのOSS活動や、各種講演会やセミナー講師としても活躍。

モデレーター 津村 信太郎 / スタンバイ事業本部　ビジネス開発部 部長

  • アメリカで14年間を過ごし、慶應義塾ニューヨーク学院（高等部）を卒業後、帰国して慶應義塾大学に入学。2006年、卒業後は外資系コンサルティングファーム Arthur D. Little Japan、ベンチャーの外資系インターネット企業 グルーポン・ジャパンを経て、ビズリーチに2013年入社。RegionUP事業の仕組みづくりをした後、日本最大級の求人検索エンジン「スタンバイ」事業本部でビジネス開発部 部長を務める。
