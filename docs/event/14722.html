Effective Java 第2版 を集まって読む かなりゆるい 会です。
しっかりした読書会を期待されていたら、先に謝っておきます…。

対象

  • Javaをしっかり勉強したい。
  • 一人で読むのは不安。
  • 他の人の解釈を聞きたい。
  • EffectiveJavaを復習したい。

Javaの基礎的な文法（ifとかforとか）で詰まったりするのは厳しいかもしれませんが、
突っ込んだ所（Genericsのゴニョゴニョとか）は随時誰かが補足してくれる事を期待してます。

つまり「EffectiveJavaを読もう」と思った時点で、参加条件はほぼ満たしています。

持ち物、準備

  • Effective Java 第2版
  • ノートPC
    • あったほうが便利、無くても大丈夫だと思います。
  • 心構え
    • グダグダに耐える必要があるかもしれません
    • 脱線を許容する必要があるかもしれません
  • 予習
    • 1章。
    • 以降はその場で決める感じで。

内容

Java7の足音がやっと聞こえてきた事ですし、現時点のJavaの足場をしっかり固めましょう。

今回はEffectiveJavaを題材にした雑談くらいのつもりでご参加ください。
今後の進め方とかも含めてゆるゆるとした感じで考えてます。
EffectiveJavaはお持ちください。読んでおくと話が弾むかも知れません。

ノートPCは動かしたいときに使うので、有ったほうがいい（無くても大丈夫な）感じです。
