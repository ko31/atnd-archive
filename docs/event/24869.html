開催概要

インプレスジャパンから発売された 『Android Security 安全なアプリケーションを作成するために』 の書籍出版と増刷決定と、さらには 電子書籍化 を記念し、書籍のテーマであるアンドロイドアプリのセキュリティ対策に関する講演を行います。また、講演会場で本のサイン会と簡単な懇親会も開催します。

参加費：1,000円（希望者には領収書を用意します）
主催： 株式会社インプレスジャパン コンピューターテクノロジー編集部
会場協賛： 株式会社コンテンツワン

  • 上記書籍の会場販売は行いません、サインを希望される場合は事前にご用意ください
  • 参加費（懇親会費含む）は受付時に回収させていただきます。おつりが出ないようご協力お願いします。

タオソフトウェア株式会社

サーバサイドアプリケーション開発会社としてスタート。Google社のAndroidに発表当初より着目し、研究開発を開始。現在はAndroid専業（受託開発）として、Android端末上で動作するアプリケーションやAndroid端末と連携して動作するサーバ側アプリケーションを数多く手がけている。日経BP社主催 Android Application Award 2010 Spring にて「tWakeUpCallMaker」が大賞を受賞。2010年5月に、プログラミング不要でAndroidアプリケーションを作成できるサービス「DOROKURI」をサービス開始。記事執筆、講演など多数。ブログ（開発者向け情報を発信） http://www.taosoftware.co.jp/blog/
