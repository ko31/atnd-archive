セミナーへのお申し込み


※またご好評につき、定員を50名に増員致しました。




2010年1月23日に名古屋にて行われた浅野 智先生による同テーマのセミナー（トライデントコンピュータ専門学校様主催）を受け、WEB制作におけるコンセプト検討以前の分析の重要性を再認識いたしました。 

そこで名古屋を中心としてWEB構築に関わる方々を対象に、HCD（人間中心設計）への理解をさらに深め、より良いWEBのあり方を共に学ぶ場として今回のセミナーを設けさせて頂きました。
 

今回のプログラムはより実践的な側面を求め、HCDへの理解から実際のWEBサイト企画のプロトタイピングまでの一連の流れを5回でじっくりと学ぶ事ができるものとなっております。
