会場について

当日は日曜のため、ビルの入り口が閉まっていると思います。
到着された方は @irof もしくは @datsuns までmentionくださいませ。

TDDBC大阪1.0および2.0参加者向けの続編イベントです。
集まって続きやったり、発展を考えたりしませんか？

やること

TDDBC岡山に向けて、TDDに関するディスカッションをしてみようかなと。
無視してひたすらTDD素振りしてもいいです。

参加条件

以下のいずれかにあてはまる方。

  • TDDBC参加経験のある方
    • TDDBC大阪1.0参加者
    • TDDBC大阪2.0参加者
    • その他の地域のTDDBC参加者
  • TDDBCに参加したことの無い方
    • TDDを実践をされている方
    • TDDBC参加者の直接の知人で、その方と一緒に参加される方
