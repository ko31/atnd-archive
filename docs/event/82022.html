～解説概要～

DevOpsという言葉は聞いたことがあるけれど、いまいちよくわからないという方が多いはず。

DevOpsの本質は「短い時間で価値を創出すること」です。では、どうすれば短時間で価値を創出できるのか？

開発面ではAgileが有効なアプローチ方法ですが、インフラ面では「自動化」が最も効くのです。自動化するには、自動化に耐えられるインフラストラクチャーである必要があり、”クラウド”と呼ばれるプラットフォームこそが、自動化を実現するのに最適な環境です。

本セッションでは、DevOpsにおけるインフラ自動化の位置づけについてお話しした後、具体的なプラクティスとして Infrastructure as Code のテクノロジーを使用し、サーバー展開や構成を自動化するテクニックについてご紹介します。

クラウドプラットフォームには、Microsoft Azureを使用いたします。恐縮です。＃安納
