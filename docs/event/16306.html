description

急なお知らせになってしまい誠に申し訳ありません．
GTMF 2011 大阪では「懇親会の設定なし」だそうですが，せっかくみなさんが集まる機会ですので，飲み会をしようと思います．
IGDA 関西からの呼びかけで，GTMF 有志の方との共催になります．
セミナーの内容を議論したり，講師に直接質問したり，仕事のくだをまいたりする良い機会ですので，
皆様お誘いあわせの上ご参加ください．

参加費はお釣りのないようにしていただけると御の字です．
