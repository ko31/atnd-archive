持ち寄りセッション

常時募集中です。当日の発表テーマをお持ちの方はお知らせください。随時このページに掲載します。

勉強会：テーマ「Drupal Console」

今回は、ANNAI（株）の青山さんを講師に、Drupal 8 時代の開発に必須のツールと目される「Drupal Console」を勉強します。

実習にはインターネット接続が前提となります。当日の環境は有線 LAN であれば安定したネットワークが使えますが、下記を参考にできるだけ事前にセットアップを済ませておいてください。

USB-有線 LAN アダプタをお持ちの方は持ち込み推奨です。

  • MACの方
    • php 5.6 or 7.0 (php 5.5 系では動作しません)
    • sqlite3 (標準で入っているかも)
    • composer
    • Drupal Console (1.0.0 はインストーラーではなく composer でのインストールが推奨です)
      • 注）1.0.0 リリース前でバタバタしていて ＠stable がだいぶ古い (0.11.3くらい) ので、＠stable → 1.0.0-beta5 に読み替えてください。
  • Windows の方
    • 下記資料をご覧ください：
      「Drupal ConsoleをWindows上にインストール」
      • 注1）まだ Drupal Console の日本語ドキュメントにマージされていません。
      • 注2）こちらも MAC 同様、＠stable → 1.0.0-beta5 に読み替えてください。
      • 注3)「Composerを使ってDrupal Consoleをグローバルにインストールする」の手順を以下で読み替えてください。
        $ composer global require drupal/console:1.0.0-beta5
        $ cd ~/AppData/Roaming/Composer/vendor/drupal/console/
        $ curl https://patch-diff.githubusercontent.com/raw/hechoendrupal/DrupalConsole/pull/2507.patch | patch -p1
        $ curl https://patch-diff.githubusercontent.com/raw/hechoendrupal/DrupalConsole/pull/2567.patch | patch -p1
