a-blog cms Training Camp 2014 Spring

イベント ユーザー
event_id 48416
title a-blog cms Training Camp 2014 Spring
catch 名古屋駅から徒歩5分のベースキャンプ名古屋を会場に開催です
description

前回は、知多半島の南端にある師崎荘で開催しておりましたが、今回は開発元のアップルップルが運営しているコワーキングスペースを利用しての合宿イベントとする事にしました。宿泊は近所にあるビジネスホテルに泊まって頂き2日間しっかり合宿しましょう。

9日、10日どちらかのみの参加も可能です。

初心者の方についてはアップルップルからの初心者向けのセッションを用意しますし、仕事を持って参加頂ければ悩んでいる部分についてを解決する事ができると思います。

合宿のいいところは、他の人が作ったサイトの普段見る事の無い管理側をどう作っているのかを知る事ができる数少ない機会です。是非、クライアント様の紹介許可を得て参加下さい。

遠方から愛知入りされるの参加者（飛行機・新幹線）中から抽選で1名様に a-blog cms のベースライセンスをプレゼントさせて頂きます。

日時

2014年5月09日（金）13:00 – 22:00
2014年5月10日（土）10:00 – 15:00

費用

  • 本編：2,000円（2日間）＋ 懇親会：3,000円

会場

  • ベースキャンプ名古屋

受付（追記しました）

  • 今回の費用は2日間通しでの費用になりますので、1日だけでも2日間でも同額になります。
  • 受付時に本編＋懇親会の参加費を集めます。
  • 受付用に名刺を1枚ご用意下さい。（皆さんと名刺交換する事になりますので多めに持ってきて下さい）

宿泊について

どこも5分以内で 7,000-10,000円くらいの近所のホテルを紹介しておきます。

  • Station Inn Nagoya
  • 東横イン 名古屋駅桜通口新館
  • 名鉄イン名古屋桜通
  • ホテルリソル名古屋
  • ロイヤルパークホテル ザ 名古屋
  • キャッスルプラザ
