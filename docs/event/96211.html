セッション（順不同）

セッション1　ゼロからはじめるゲームプログラミング

takeuchi yuusuke さん

プログラミングはある程度勉強したけどゲームの作り方が良くわからない人向けに、ゲームプログラミングの基礎を説明します。 具体的には、敵を避けるだけの簡単なサンプルゲームを通して、ゲームループ、コマンド入力、当たり判定などゲームに最低限必要なことを一通り学んでいきます。

セッション2　Windows 10 の診断データを覗いてみよう

マイクロソフト ディベロップメント Masaru Iritani さん（@msdmairitan）

Windows 10 初回利用時に必ず聞かれる診断データ送信の設定。何となくそのまま同意している人も、とりあえず拒否している人も、実際に Windows 10 がどのようなデータを送っているか、それらがどのように使われているかを一緒に見ていきましょう。

セッション3　「かなしきだるやなぎちゃん」

Ｙｏｕ＆Ｉ さん（@youandi）

詳細： 松山市に住む無職の“だるやなぎ”。 Twitterタイムラインを観察しながら優雅に過ごしているが、愛車のバイクを購入し、生活は一変。 ふてくされて、家出をし、愛媛各地を東へ、西へ。愛と哀しみの大冒険が始まる――！ だるやなぎが海老を獲りに行く話です。UIテストの自働化について話します。
