イベント概要

2013年に設立され,毎年600%以上の成長率を続ける株式会社Branding Engineer(以下BE)。
その成長率を実現するためには代表取締役 河端/高原の2者の最恐の営業力が大きく貢献しています。

2人でスタートした会社は現在従業員50名。Yahoo!/ワークスアプリケーション/リクルート/電通/indeed/アクセンチュアなどIT業界やHR業界だけでなく、幅広い業界から仲間が集まっています。

BEでは、2016年には1億円以上の資金調達を行い3年後の上場をめどに現在在籍数の2倍以上の人材を募集しています。
新規事業立案者/企業営業/新卒事業/など非常に多くのポストをご用意しています。
今回は、その圧倒的成長率を誇るBEの代表2名に登壇していただき、現在の営業力をどのように培ったのかやお自身の半生について話していただきます！

イベント当日は参加者全員の乾杯でイベントを始めさせていただきますのでお楽しみに！

こんな方にオススメ！

・起業したい方
・新規事業を立ち上げたい方
・社会にイノベーションを起こしたい方
・企業内起業したい方
・常識をぶっ壊したい方
・新しい行動を起こすきっかけが欲しい方
・BEで上場を経験したい方

▼スピーカー

河端保志 | Branding Engineer 代表取締役CEO　

1989年生まれ。2014年電気通信大学大学院卒業。同院にて人工衛星の姿勢制御の研究を行う傍ら、個人で営業活動やコンサルティング業務を経験。
リクルートやDeNAでのインターンを経験したのち、2013年大学院在学中に（株）Branding Engineerを創業。エンジニアのための職場選びサービスBranding Engineerを展開中。

高原克弥 | Branding Engineer 代表取締役COO

学生時代にスタートアップ3社で開発や人事を経験後、大学在学中の2013年、エンジニアの価値向上を目指す、受託開発、人材会社を設立。
以後日本におけるエンジニアの価値向上のため、様々なサービスの展開と人材育成を行う。
