就活にどんな準備をしたらいいのか不安な就活生のみなさん、

自己PRの仕方ってどうすればいいんだろう？
自分に合っているのはどんな企業だろう？
面接官に見られるポイントは？
などなど、就活における悩みや疑問、不安があるのではないでしょうか？

この意見交換会では、かつて就活生だった社会人にそうした疑問や考えを聞けるチャンスです！
悩みを相談したり、意見交換などをしたり、働くことに関する酸いも甘いも様々なことが知れる貴重な機会です。
普段会社で働いている方々が就活生のみなさんの疑問にホンネでお答えします。

就活に少しでも不安や疑問を感じているのならぜひ参加してみてください。
元気な社会人の交流から、一歩を踏み出すパワーをもらいましょう！

　

また、かつて就活生だった社会人のみなさん、

年齢や職業問いませんので、やってみて効果のあったことや、今ならわかる就活に本当に必要な大事なことなど、貴重なノウハウをぜひ教えていただきたいです！

また、書類だけではわからない就活生個人の良さや人柄などを知ることができ、今の就活事情を知ったり、いい社員に育っていきそうな就活生を見極めるヒントを得たり、インターンに誘ったり、勤めている会社の本当の良さを知ってもらう機会でもあります。

互いに良き機会としてこの場を活用いただけたら幸いです。
