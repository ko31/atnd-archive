Wakame-vdc / OpenVNet Advent Calendar 2014企画

Wakame Projectに関する記事を毎日誰かが担当して書き、発表をする、と言うものです。12/1に始まり、クリスマスが来たら終わります。

Wakame Projectは、これまでがそうだった故に、他の基盤と異なってかなり敷居も高いはず。なので、一番良かった記事に、プレゼントを用意します。(ページ最下段をご参照ください)

また、ご参加者全員を新年会にご招待いたします。

応募・参加方法

本ページ、もしくはAdventerにて募集をいたします。12月何日を担当するかを表明してください。記事の執筆が完了しましたら、MLにポストしてくださいますようお願いいたします。記事は12/25日中必着です。

日付はできるだけ被らないようにお願いしますが、25以上の記事が集まっても構いませんので、その際は適当に誰かと被せてくださいませ。

やってみて、まだ書けそう！と思ったらぜひ期間中も追加でご参加ください。一人何記事でもOKです。って言うかむしろ、そうして書いてもらえるなんて超嬉しいです！

発表媒体

ネットで認証無く、モダンブラウザ(Chrome等)にてだれでも閲覧できる形式であれば何でもOKです。(ブログ、Facebook、Twitter…）
