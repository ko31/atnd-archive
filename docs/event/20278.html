description

株式会社東海クロスメディア研究所（名古屋市中村区名駅３－２４－８、代表取締役三輪知生）は、実践事例に学ぶマーケティング講座「逆境をバネに！中小企業の新事業開発秘話」と題したセミナーを１０月８日（土）に名古屋市内で開催する。成功事例を元にこれからのモノづくり企業のあり方について考える内容となっている。

［解　説］
わが国はいま熾烈な国際競争の荒波に晒され、大企業は海外に工場を移転、統廃合にも直面し、中小企業はその傘下で大企業に頼ってはいられなくなってきている。

当セミナーでは、リーマンショック後に新事業展開を果たし、自社で新商品を開発した２社に直接話を聴き、中小企業が新事業開発を成功に導く秘けつについて学ぶ内容となっている。

【概　要】実践事例に学ぶマーケティング講座

【対　象】このような方にビッタリ！
　 □ 過去の成功体験に縛られて、なかなか新しい分野に踏み出せないでいる方
　 □ このまま事業を進めていったとしても、明るい未来像が描けないという方
　 □ 新しい事業を始めようとしても、先立つ投資資金を拠出できないという方
