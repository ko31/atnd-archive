夏からGitを始めよう！

  • Visual Studio Code というエディタを使って
  • コマンドを使わずに
  • Git を基礎から学びます

Gitとは？

  • ソースコードを管理するシステム
  • コードの更新履歴を取れるので途中で動かなくなっても以前の状態に戻せる
  • 特に複数人で開発するときには必須

GitHubとは？

  • Gitを使ってソースコードを管理するWebサービス
  • Web上に自分のソースコードを公開できる
  • 進捗管理やWebページを公開できる仕組みもある
  • 開発者にとって欠かせないサービスの一つ

対象者

明治大学の学生のみ参加可能
総コン外でも参加できます

  • Git、GitHubを全然分からない人
  • CUI、Shell、コマンドに慣れてない人
  • 自分のソースコードを公開したい人
  • ポートフォリオ(自分のサイト)を作りたい人

GitHubを使ってポートフォリオを作れる Jekyll と、スライドが作れる reveal.js を学びます。
