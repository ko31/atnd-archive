catch SUPER ILLUSTRATION SCHOOL
description

SUPER ILLUSTRATION SCHOOLでは、出版不況の最中に100万部以上
売れた、『恋空』、『赤い糸』など20タイトル以上の携帯小説を書籍
化し、合計500万部以上を販売した有名編集者で、まさに携帯小説ブー
ムの生みの親とも言える伊東寿朗さんを迎えてクリスマスの記念講義
を開催します。講座タイトルは「進化する市場が求める出版とイラス
トレーション」です。

また授業後には、佐藤校長によるワイン基礎講座＆ワインパーティを
開催します。
ワイン基礎講座では、葡萄それぞれの特徴や、国別の味の違いを解説
しながらそれぞれのワインを参加者で試飲して、ワインの特徴と安く
てもおいしいワインの見分け方などを伝授します。
おいしいイタリアンデリもご用意いたしますのでお気軽にお申し込み
ください。
