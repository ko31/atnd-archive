Raspberry Pi 用アドオン基板RaspberryHabuの配布

当日はんだづけして試してみる人は、I/Oボード基板（テスト版）＋コネクタ＋トランジスタアレイ＋LEDのみの
セットを無償提供します。
後半ハッカソンタイムに制作実習し、フィードバックをお願いします。Raspberry Pi 本体その他は貸出できます
が、必要数の把握のため申込時のアンケートにチェックをしてください。
RaspberryHabuについてはこちら http://togetter.com/li/509049 (配布テスト版はこのバージョンとは少し異なります)
