description 
来年の3月に次期 安定版である コードネーム "squeeze" がリリースフリーズに入る予定です。
これに合わせて世界各地で　Debian Bug Squashing Party[0] が行われる予定です。

今回、日本でも東京エリアDebian勉強会と合わせて Bug Squashing Party を行います。
以下の日程で行いますので、Bug Squashing に協力していただける方の参加をお待ちしております。
RC バグを潰しまくるのはもちろん、パッケージメンテナンス、ドキュメント/po翻訳などのBug Squashingを
やりたい方も参加できます。
Debian / オープンソースの開発者が集まりますので、GPGサインをやりたい方もどうぞ。
飲食ができますので、みなさんおやつ等を持って参加してください。


IRC
 * IRC サーバー : osdn.debian.or.jp
 * IRC チャンネル : #debianjp
 * 文字コード : UTF-8

Bug Squashing 内容
 * RC バグ潰し
   - RC バグリスト / http://bugs.debian.org/release-critical/
   - RC バグリスト詳細版 / http://bts.turmzimmer.net/details.php?bydist=both&sortby=packages&ignhinted=on&ignclaimed=on&ignnew=on&ignpending=on&ignbritney=on&ignotherfixed=on&new=7&refresh=1800 
 * IPv6 対応 (予定)
   - IPv6 バグ一覧 http://bugs.debian.org/cgi-bin/pkgreport.cgi?dist=unstable;tag=ipv6 
 * piuparts bug
   - piuparts バグ一覧 http://bugs.debian.org/cgi-bin/pkgreport.cgi?tag=piuparts;users=debian-qa@lists.debian.org&archive=both 
 * メンテナンスしているパッケージの squeeze に向けての修正
 * po / ドキュメントの翻訳 


コーディネイター
 * 岩松 信洋 iwamatsu at debian.org
 * 荒木 靖宏 ar at debian.org 


参加者が用意するもの
 * unstable が動作しているマシンおよび開発環境
    - pbuilder/cowbuilderの整備
    - 修正するソフトウェアのソースコード等 
 * ネットワークの動作確認 有線/無線(WEP,WPA1/2)
    - 会場は無線メインになる予定です
 * デバッグ用の資料、機材
 * ネットワークケーブル、ハブ
 * 電源タップ、延長ケーブル
 * お菓子や飲み物
 * GPGキーサインしたい人はフィンガープリントを印刷したものと、公的な証明書（パスポート、運転免許証） 

* 参考資料
   - Bug Squashing とは(英語) http://people.debian.org/~vorlon/rc-bugsquashing.html
   - バグの潰し方(英語) http://wiki.debian.org/BSP/BeginnersHOWTO


その他
 * e-mobile を所有している方は回線共有のご協力をお願いするかもしれません

    * 出前対応可能
