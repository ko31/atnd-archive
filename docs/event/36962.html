Web Components概要
WebComponentsとは、これまでのWebに欠けていた「UIコンポーネント」の考え方を導入するもので、Web開発を根本から変化させる可能性があるテクノロジーです。本講演では、WebComponentsの基礎概念や仕様の詳細についてお伝えします。
