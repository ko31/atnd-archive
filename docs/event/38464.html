description

株式投資をこれから始めようと思う方。
これまで何度もトライしたが、うまくいかない方。
株式投資は所詮ギャンブルでしょ、と思っている方。
株式投資は危なすぎて・・・と思っている方。
子供にはギャンブルでない株式投資を教えたいと考えている方。

　株式投資成功のカギは「売らない株式投資」です。この機会にメイクイット式株式投資をマスターしませんか。

　メイクイット式株式投資の基本スタンスは売らない株式投資、株式投資本来の目的である、株主としてともに成長する企業への投資です。買売買売を繰り返す「トレード」でなく、売ることは考えない「投資」にこだわると、株式投資ってなんて素晴らしいのだろう！株式投資ってこんな簡単だったんだ！そんな世界が目の前に開かれます。

　セミナー終了後には、投資に最適銘柄一覧をお土産に持って帰っていただきます。

主な項目
1．「投資」と「トレード」の区別から始まる株式投資。
2．こんなに素晴らしい株式「投資」。
3．株式投資は魔法の貯金箱。
4．こんなに役立つ、投資指標の利用。
5．自分なりの銘柄選択基準を確立しよう。
6．売り時の判断。

　この講座はチャートやテクニカル分析は使いません。チャート分析を希望される方は、他のトレーダー養成講座をご利用下さい。
　アベノミクスで世の中が大きく変わる入り口の今、これからの豊かな生活のためにも、株式投資の利用術を身につけて下さい。
