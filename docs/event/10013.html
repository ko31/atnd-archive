RightScaleユーザ会(RightScale勉強会) – 第3回

Amazon Web Services(Amazon EC2/S3)やRackSpace Hostingなどの
クラウドサービス用の管理ソフトウェアとして最も有名と言えるのが RightScale(http://www.rightscale.com/) です。
併しながら、国内では日本語での機能説明の資料がなく、知名度に比較してRightScaleのSeverTemplateや
利用方法や利用のメリットについてほとんど知られていません。

今回は、RightScale社と雲屋株式会社の協力により、RightScaleの副社長のJosh Fraser氏と
カスタマーディレクターElise Arthurs氏をお招きし、RightScaleの概要から利用メリット、
今後の展開について幅広くお話いただくことになりました。

RightScaleのTシャツや帽子もプレゼントしてもらえると思います！
是非この機会にクラウドについて深く学んでみましょう！
