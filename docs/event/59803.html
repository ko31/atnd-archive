description

創業融資に強い起業支援の専門家が、公庫や銀行では絶対に教えてくれない公的融資獲得のための成功方法を伝授します。

【講演内容】
・本当に無担保・無保証人で融資が実行されるの？
・その他大勢の同業者から抜け出す方法
・実績がないからこそ借りられる
・公的融資の急所を知る！
・超低金利って、どれくらい？
・多くの方がつまずく、自己資金って何？
・金融機関のホンネ、教えます
・融資の成功確率を高める事業計画書作成の3つのポイント
・資金使途によって、融資の確率が変わる？！
・面談対策！モノはいいよう

【受講特典】
① レポート「資金の不安が消える５つの方法」をプレゼント！
② 起業家支援機構シンボルキャラクター「ひよこさま」のノベルティグッズをプレゼント！
③ 個別コンサルティング（通常３万円）が１万円！
あなたの業種・資金でいくら融資が受けられるか、どのようなビジネスモデルにすることで自社が安定した売上をあげることができるようになるか、アドバイスします。

＜こんな方にオススメ＞
・起業したが売上が伸びず、資金繰りに不安を抱える方
・事業規模拡大のため、資金が必要な方
・公的融資がどんなものか興味がある方
・開業資金が少ないのが理由で、起業が「夢」のまま終わっている方
・好きなことを仕事にして、ワクワクした人生を送りたい方

＜受講メリット＞
・資金繰りの不安を解消する方法を知ることができる
・1,500万円の融資がおりた「事業計画書」を見ることができる
・事業が加速するために必要なポイントがわかる
・自分のビジネス戦略を見直すきっかけを作る

＜講師＞
柴原由佳（（社）起業家支援機構　代表理事）
静岡県生まれ。
大学卒業後、製薬メーカーに勤務。その後、病院・ＩＴ企業など転職を繰り返す。
派遣という働き方に不満と限界を感じ、2011年「自分で起業しちゃう？」という気軽な気持ちから「家事代行・ベビーシッターサービスゆうか」を創業。
当初は、効率の悪い集客・お客様からのクレーム・社員教育の難しさなど、数々の失敗を繰り返しながら、どの業態にも当てはまる成功哲学を修得。
数ヶ月で軌道に乗せ、その２年後に当該事業を売却した実績もつ。
自ら起業し成功した経験をもとに、同じことで悩んだり迷ったりしている起業家の励みと支えになりたいと考え、「起業してよかった！」と言える実業家の育成に貢献するため「（社）起業家支援機構」を設立。
現在は、起業家の誕生をお手伝いすることにより、市場競争や雇用の促進に繋がり、国や地域の更なる成長発展に寄与したいという思いから、公的融資の支援やセミナー、コンサルタント活動を積極的に行っている。
尊敬する人は祖母。女性起業家である祖母の影響は大きく、「お客様に助けられて生きている、という感謝の気持ちを忘れずに。」という祖母の言葉を大切にしている。
