★講演内容

また、エンジニア組織についてや、技術を軸とした組織としての風土づくり・採用についてなど
組織論などについても話す予定です。

※内容は変更になる可能性がございます。

★講演者プロフィール

■和田　修一氏（株式会社nanapi　取締役　CTO）

中央大学経済学部を卒業後、2005年楽天株式会社に新卒で入社。
サーバやストレージなどのインフラ周りの設計・運用などを担当。
2009年に株式会社nanapi(旧株式会社ロケットスタート)の取締役CTOに就任し、
主にnanapiの開発に携わる。
現在はインフラを中心とした技術分野から、取締役として経営・マネージメントなどを担当。

CTOの職務経歴書１単位が足りなくて卒業出来なかったおかげでCTOになっている｜nanapi CTO 和田修一氏
