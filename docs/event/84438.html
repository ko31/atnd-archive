■FRI Campとは
-——————————-

FRI Campは、合宿形式でグループワークを行い、社会人レベルの実力を身につける場。
運営スタッフは全員現役社会人。参加者には、国や業界に関係なく、どこでも通用する本物の力を磨いて頂きます。

■FRI Campの特徴
-——————————-

・大学や年次も違う中で、ゼロからイチを生み出すためのグループワークを体験
・現役社会人が3日間付きっきりで、参加者の皆さんをサポート
・就職活動やキャリアなど、課題や悩みに対して現役社会人がアドバイス
・Campを通じて見えた課題を元に、各個人にフィードバックシートを送付
・Camp終了後も現役社会人や過去Camp参加者との定期的な交流を実施

■日時 / 会場
-——————————-

2017年2月24日(金)～2月26日(日)（2泊3日の合宿形式）

■応募概要
-——————————-

【対象】大学生、大学院生（分野･学年は問ず、短大生･専門学校生含む）、および若手社会人（入社1年目程度）
【定員】30名（予定）※先着順となります
【参加費】20,000円（食費・宿泊費込）

■内容
-——————————-

チームビルディング(リーダー選択) / プロジェクトマネジメントセミナー / ビジネスプランニング グループワーク / ロジカルシンキング、プレゼンテーションセミナー /社会人とのナイトセッション(模擬面接あり) / 最終報告プレゼンテーション / 社会人から and 参加者同士の 個人フィードバック
