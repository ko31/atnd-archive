概要

名古屋でiPhone開発者の勉強会をやりましょう。

まあ、気楽にやりたいので、気楽にご参加下さい。
近隣の方も、遠くからの方も歓迎です。

ありがたいことに無料で会場を貸していただけるので、参加費は無料です。

発表者・概要

発表予定
・hirobe
・grandeforestaさん：iCloud Storage APIについて
・itok_twitさん：AssetsLibraryについて（iOS5対応版）
・Goodia Inc.さん：ヒットアプリ(無料アプリ)の裏側
・岸川さん：TestFlightの活用について（余裕があればPhotoAppLinkの話も）

LT予定
・b1tl1feさん
・pypupypaさん
・aTKumaTさん
・sngmrさん

募集中！
発表・LTする人は参加者枠とは別カウントで優先参加可能とします。

30分程度の正規発表と、5分程度のLTを募集します。
発表すると、参加者の方に名前覚えてもらえますよ。

発表したい方は、コメントに書くかk.hirobe@gmail.comまたはtwitterの@hirobeまでご連絡下さい。

ネタとしてはiPhoneのアプリ開発に関係するものならなんでもOKです。

iPhone Developer Program未登録の方も参加可能としますので、発表される方は（参加される方も）、NDAに引っかかるβ版の話はしないようにしてください。
＃無事iOS5もリリースされましたね！

参加希望の方へ

発表者が足りない場合は、参加者にLTか自己紹介をお願い(強制)すると思いますので、ご了承願います。

なお、内容は特に入門者向けというものではありません。
開発未経験の方は、話についていけるように、少なくともXCodeからシミュレータ起動するあたりは自習しておくことをおすすめします。

なお、iPhone Developer Programへ登録していない方も参加可能です。
