これが結論！２時間で分かる起業セミナー

title これが結論！２時間で分かる起業セミナー
catch 起業の専門家が、市販の起業本には載っていない、 門外不出かつ、痒い所に手が届く「孫の手」のような講座を開催します。
description

【セミナー概要】
「個人事業主と法人と、何が違うのかよく分からない」
「起業してもうまくいくか心配」
「節税って何をすればいいの？」
「個人事業主で起業したが、売上が上がらない」

上記の項目で1つでも気になった方は必見！
起業や会社設立を考えている方を対象に、自分だけでは解決できない疑問をその場で解消します。

「知識」のみではなく、実際に使える実務をお伝えする「実践型」のセミナーです。

＜第1部＞
開業手続の専門家が、他のセミナーでは聞けない、個人事業主の場合と会社にした場合、結局どちらが得なのかの違いを「シミュレーションを見せながら」解説します。
＜第２部＞
会社設立で失敗しないために大事なことと、設立書類の肝である「定款」を作る際の急所を、こっそり教えます。

【セミナー詳細】
・会社設立か個人か、自分に合う事業形態を選ぶ基準を知る！
・個人と法人、何が違うの？
・会社設立をすると得になる境界線をシミュレーションを見せながら伝授！
・損をしない節税対策 のしかた
・資本金はいくらがいい？
・融資が成功しやすい起業の方法
・起業・会社設立の全体の流れを掴む
・経営者が知っておくべき法人税と消費税
・ネットにも本にも載っていない会社設立の急所とコツ！
・定款を作成する際の注意点とは？

【受講特典】
① 特別レポート「資金の不安が消える５つの方法」をプレゼント！
② 起業家支援機構のキャラクター「ひよこさま」ノベルティグッズをプレゼント！
③ 創業融資に関する個別コンサルティング（通常３万円） 無料！
あなたの意向を充分に理解した上で、創業融資の専門家として、あなたの希望と利益を最大限に実現するための方法を模索し、提案します。
④ 会社設立･許認可等に関する個別相談（通常２万円） 無料！
会社設立、許認可取得、規約･契約書･法定書面等の作成、その他法的手続や法律問題について、あなたの不安を解消します。

【こんな方におすすめ】
・起業を考えている方
・会社設立を検討している方
・創業間もない方
・経営者の税務・節税に関心のある方、
・結局、個人と法人、どちらが自分にとって得なのかを知りたい方
・事業を早く軌道に乗せて、早く自由を獲得したい方
・会社設立の手続を自分でやってみたい方
※ 同業者･士業の方のご参加は、ご遠慮ください。

【受講メリット】
・ 個人事業主で始めるか会社を作るかの結論が出る。
・ 個人で始めた場合の、法人化のタイミングが分かる。
・ 自分で調べても分からない情報や、自分の中には無かった発想が得られる。
・ いつでも相談できる仲間が見つかる。
・ 会社設立の手続が自分でできるようになる。

【講師プロフィール】
柴原隆宏（しばはら法務事務所 代表）

松井秀喜選手と同じ昭和49年生まれ。
日立製作所に入社するも、すぐに辞めてしまい、寮からも追い出される。その後約８年間、建設会社で現場作業や現場監督に従事する。
その傍ら、独学で法律を勉強し、法務担当者へ転身。
約５年間、化粧品会社の法務責任者として、訴訟業務、知的財産業務、契約書の作成・審査、広告法務、個人情報保護、登記申請、法律相談等の法務実務経験を積む。
現在は、起業家を支援する行政書士。
＜信条＞
・「知る」 ことは、「知らないことが増える」 ことである。
・唯一の復讐の方法は、彼らより楽しく生きることだ。
