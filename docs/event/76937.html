description

introduction

webエンジニアが初めてIoT製作に取り組んだ際の経験談を元にIoT製作初心者向けのセミナーを開催致します。何から始めたら良いのか、IoTの勉強方法、躓いた際の解決方法などをお話しいたします。 また、IoT、ハードウェアを専門としている北神博士を交えてのパネルディスカッションも行ないます。ハードウェア未経験の方でも、中級以上の方でも楽しめる内容となる予定です。

【こんな方にオススメ】

  • WebエンジニアでこれからIoT始めてみようと思っている方
  • ハードウェアに取り組んだが技術的に躓いてしまっている方
  • 趣味でこれからIoTを始めようとしている方

【共催】

テモナ株式会社
テモナは、リピートECに特化したテクノロジーとサービスで、生活者のQOLとネットショップのLTVを向上させます。

　　　
株式会社バニーホップ
Bunnyhop Inc.は、情報技術とデジタルデバイスの融合により、全く新しいユーザー体験を生み出すチームです。

【メンバー】

北神雄太

テモナ株式会社 IoT技術顧問。
日々IoTについて研究している。博士であり発明家。
RaspberryPiやEdisonなどを次々にハードウェアと繋げていく。

山﨑康平

日出春来

webエンジニアとしてたまごリピートの運営に携わっている。
