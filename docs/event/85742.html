産業技術大学院大学 大学院説明会

　産業技術大学院大学への入学を検討されている方に、より詳しく教育内容や育成する人材像などをご説明します。
また、教職員が皆様の疑問に直接お答えします。

産業技術大学院大学 創造技術専攻

産業技術大学院大学 創造技術専攻は、品川シーサイドにキャンパスを持つ、感性と機能の統合デザイナーとしてイノベーションをもたらす｢ものづくりアーキテクト｣を育成する、公立大学の大学院です。

当日の内容

  1. 大学の紹介
  2. 入学試験について
  3. 希望専攻ごとに分かれて、各専攻の説明
  4. キャンパスツアー
  5. 質疑応答・個別相談
