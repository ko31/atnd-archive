準備

Windowsの場合は、VirtualBoxにUNIX環境を入れればオケ。Cygwinでも・・大丈夫かな？
GNU版のsed、awk、grepなどを使うことが予想されます。Linux系ならまず大丈夫。
また集計などで便利なOpen usp Tukubaiコマンドもオススメ。

  • GNU系のコマンド
    MacはBSD系なので、homebrewで下記をインストール。
    $ brew install coreutils
    $ brew install gnu-sed
    $ brew install gnu-awk
    $ brew tap homebrew/dupes
    $ brew install homebrew/dupes/grep
    高速化したGNU grepをインストールする
    FreeBSDの場合はportsからインストールお願いします。
  • Open usp Tukubai
    入手先は下記のリンクを参照。わからない場合はサポートさせていただきます。
    FreeBSDの場合はportsからインストール可。
