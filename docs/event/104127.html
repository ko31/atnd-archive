手作り・手芸キットの専門通販のさくらほりきりの「展示・講習・販売」イベントです。

自社独自のノウハウで開発したキットは、針や糸を使わないのでお子様からご高齢の方まで手芸をお好きな方なら誰でも“簡単に” “楽しく”作りいただけます。

また和紙や布はセットされておりますが、商品に型紙が入っているのでご自宅にあるお好きな柄を使って、オンリーワンのものをお作りいただくことができます。

他にも、会場限定のワークショップ開催やご来場いただいたお客様にプレゼントもご用意♪

この機会にぜひお誘いあわせのうえ、日々の忙しさを忘れ『手芸で楽しむ時間体験』ができるさくらほりきりハンドメイドマーケットへのご参加を、心よりお待ちいたしております。
