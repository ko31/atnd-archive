description

　エレクトロニクスの分野におけるエンジニアの人的・技術的な
交流を目的とした集まりです。以下の分野のエンジニアの方々を
対象としています：

【対象とするエンジニア】
　プリント基板の設計エンジニア
　マイコンのハードウェア設計エンジニア
　FPGAの設計エンジニア
　ASICの設計エンジニア
　フルカスタムLSIの設計エンジニア
　電子部品の設計エンジニア
　アナログ回路の設計エンジニア
　高周波回路の設計エンジニア
　パワーエレクトロニクス回路の設計エンジニア
　メカトロニクスの設計エンジニア
　各種ロボットの設計エンジニア
　等々

【活動内容】
　月に２回程度、ウィークデイの午後に集まりLTや雑談などの
人的・技術的な交流を行います。

【参加費】無料、ただし会場にて注文した飲み物代は有料となります。

【注】三鷹ハードウェアエンジニアカフェ（仮称）は、三鷹プログラマーズ
カフェおよび（株）まちづくり三鷹のご協力のもと、三鷹プログラマーズ
カフェの分科会として設立されました。なお、９月３０日（木）の第１回は
三鷹プログラマーズカフェとの合同開催となります。
