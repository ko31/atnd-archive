description こんな人にオススメです。

□ 生理痛が重い
□ 皮膚がかぶれる
□ 経血の量が多い
□ 布ナプキンの使い方を知りたい
□ 冷え性に悩んでいる
□ 自分を大事に出来ない
□ もっとハッピーになりたい

＊＊＊＊＊＊＊＊＊＊＊

紙ナプキンから布ナプキンに変えた事で、

①皮膚のかぶれがなくなった
②生理痛が軽くなった
③経血量が減った　　などの
嬉しい変化を感じる方がたくさんいらっしゃいます。

でも、
「使い方がよくわからない・・」「モレが心配・・」など、
なかなか使い始めるきっかけがないと、
使いづらいものがありますね。

そこで、布ナプを使うメリットや、
布ナプの使い方をお伝えし、
手縫いで布ナプまで作れちゃうという、
欲張りなイベントを企画しました。

布ナプ作りは、肌にも環境にも優しい、
オーガニックコットンの布を使います。

お裁縫が得意でない方も、簡単に作れますから、
気軽に参加してくださいね。

『気になるけど踏み出せない』という方、
『最近使い始めたけど、もっと知りたい』という方、
ハッピーな女子会に、是非、ご参加ください。
