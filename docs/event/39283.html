Illustratorを使ってみたいけど、どうしたらよいかよく分からない、という方を対象としております。

IllustratorをインストールしたノートPCをご持参ください。

以下の流れですすめたいと考えております。

　１）新規データ作成
　２）環境設定など

　４）下絵
　５）パス操作 / パスファインダ
　６）配色
　７）フィニッシュ
