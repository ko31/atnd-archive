【Weekend company】スタートアップ

「沖縄のエンジニア、デザイナー、ディレクターなどが集って、週末だけの事業を興しちゃおう」の会です。

言語・スキル制約はなくお互いがもっている知識を、共有し合い、更なるスキル向上を図りましょい

ふわっとしてますが対象のかたは以下

  1. 色んな言語を学びたい
  2. インフラの知識を身につけたい
  3. 設計の知識をつけたい
  4. デザイン UI/UXの知識を身につけたい
  5. 企画、プロデュースに挑戦したい

などなどを想定しています。

第一回のコンテンツ

  1. Weekend companyの主旨説明
  2. 参加者メンバーの自己紹介、スキル、この会でやりたい事
  3. 企画がある方はプレゼン発表
  4. チーム分け企画考案
    ※2チームに分かれて、企画、システム構成を考えていきましょう
    ※事前に企画まとめの方法をアドバイスベースで説明させて頂きます
  5. 企画案の発表会
  6. 懇親会　振り返り

是非追加したい内容がございましたら、枠を設けますのでご連絡下さい♪
