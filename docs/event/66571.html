description

株式会社FOVEは5月19日より、米クラウドファンディングサービス
「Kickstarter」にてVRHMD「FOVE」の出資受付を始めた。わずか3日で目標金額の25万ドルを達成し、まだまだ出資者が増えそうな勢いだ。

最大の特徴は、視線追跡を備えており、目で操作したりバーチャル世界のキャラとコミュニケーションが取れるというところ。VRHMDのハードは海外組が多い中、日本発のベンチャーとして世界に挑むのもユニークだ。

VRHMDはモノを触ってみたほうが魅力が一発でわかりやすいということで、ぜひ日本のVRコミュニティーにこのFOVEの面白さを知ってもらうために、参加者全員がFOVEをかぶれる体験会を開催します。当日はFOVEの小島由香CEO、ロックラン・ウィルソンCTOをお迎えして開発秘話をお聞きし、参加した方々からの質問を受け付ける時間も用意。日本ではまだあまりデモを実施してない期待のVRHMDを、ぜひその目でご確認ください。
