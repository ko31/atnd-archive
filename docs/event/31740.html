JavascriptにてAndroidやiPhone両方のアプリが開発出来る、
「 Titanium Mobile」 を使ってスマートフォンアプリ開発スキルを学ぶ講座です。
これからスマフォアプリ開発を始めてみたい方におすすめの講座です♪

※ 当講座は全８回で構成されており、今回はその2回目です。
全8回の概要については当スクールのページを御確認ください。

現役のTitaniumエンジニアによる監修で、実践に則したカリキュラムをご提供♪

基本スキルの理解はもちろん、陥りやすい問題点と対処法も含んだカリキュラムとなっています。
豊富な経験談もおりまぜ、実践に必要なスキルを網羅した内容でご提供致します。
TCAD Course Labs – Japaneseの翻訳にも携わる講師による
Titanium Certified Application Developer(TCAD)  と 同様のスキルを習得を目標においたカリキュラムです。

講師 ： 八木 都志郎 様

2010年よりTitanium MobileでiOS/Androidアプリ開発に従事、エンターテイメント系 アプリを中心にこれまで15本以上のTitanium Mobile製スマートフォン向けアプリを リリースしています。

第2回 GPS編 講義詳細

最初に：

  • プロジェクト
  • 新規作成 ビルド
  • WindowとTab
  • WindowにMapViewを表示
  • 現在位置の取得（GPSがオンになっているか必ずチェック）
  • 現在位置にピン（Annotation）を表示
  • 質疑応答、実習課題 （指定された場所の地図を表示しよう）

次に：

  • Geocoding（Google Maps APIでデモ）
  • ReverseGeocoding（Google Maps APIでデモ）
  • 移動時にGPS情報を送信し続ける
  • マップにルートを表示 向き（電子コンパス）を取得
  • 質疑応答、実習課題（簡易な道案内アプリを実装）
