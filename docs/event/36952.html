【講演主旨】
本セミナーでは、まず知的財産とこれを取り巻く状況を踏まえた上で、知的財産経営と知的財産の価値評価との関係を学びます。その上で、知的財産権価値評価を行うための具体的な手法と代表的モデルについて、基礎的理論の解説を経た上でケーススタディを通じて実践的に理解を深めます。そし
て、知的財産による資金調達、知的財産の信託等、最近注目を集めている新たな活用法についても解説し、新事業・新製品開発における有用な知的財産の活用方法、知財戦略の立て方を修得していただきます。
