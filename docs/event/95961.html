愛媛で起業と仲間づくりの一歩目を踏み出す54時間！

ご好評につき、Startup Weekend Ehime vol.6 開催します！

Startup Weekend（スタートアップ・ウィークエンド）は、世界150ヶ国・1200以上の都市、日本25都市以上で開催されいる起業体験イベントです。金曜日の夜から日曜日までの54時間で仲間づくりをしながらアイデアのプロトタイプを作りあげていきます。机上の空論で終わらず、週末の3日間で集中的に起業のきっかけや成果を生む実践的な取り組みとなります。

参加費も7食ついて、6,000円！ぜひ、ご参加ください！

特にこんな方におすすめのイベントです！

  1. 起業を目指している又は起業している学生、社会人の方
  2. 新しい ビジネスがアイデアから形になるまでを学びたい方
  3. デザイナーやエンジニアでアイデアを形にするのが好きな方
  4. 起業仲間や先輩起業家とつながりたい方
  5. 刺激的な体験をしたい方
