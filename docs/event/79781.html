2016年、2回目の勉強会は特別編として、CG Worldの連載やEIZOのセミナーでも有名なパーチの長尾さんをお招きして「カラーマネジメント」についてお話してもらいます。

事前にある程度カラーマネジメントについての知識をつけてからお話を聞いた方が理解が深まります。

長尾さんがCG Worldで連載していたカラーマネジメントの話が現在もウェブで見られます。セミナー参加前に必ず目を通しておいてください。
<a href =“http://cgworld.jp/regular/cms001.html”>http://cgworld.jp/regular/cms001.html

勉強会は13:30からですが、会場は13:00から借りているので早めにお越しいただいても大丈夫です。17時までに撤収が必要ですのでご協力ください。

いつもは無料の勉強会として会場費のみを参加者に支払ってもらっていますが、今回は有料セミナーとして税込3,000円（セミナー参加費＋会場費）が必要ですので、ご注意ください。

セミナー終了後は希望者を募って懇親会を開催する予定です。

過去の活動内容については、C3Dのウェブサイトをご参照ください。
<a href =“http://c3dpoly.com/”>http://c3dpoly.com/

【勉強会の内容 】
午後（13:30〜17:00）
*長尾さんにカラーマネジメントについてお話してもらいます。

【勉強会参加費】
