description

　
・やりたい事がわからない
・仲間が欲しい
・何か始めたいけど、何をしたらいいかわからない
・自信を付けたい

そんな疑問をその場で解決します！

「これからの時代は「人としての力」が必要になる」
　
「これからは「お金」ではなく「人」を稼ぐ時代だ」

ホリエモンやキンコンの西野さんが言っている通り、人の時代がやってきています。
　
AIやロボットが発達していき、誰でもできる作業は駆逐され、働き方革命が推進される中で「あなただから」と言われる人が世間から必要とされています。
　
　
YouTuberやクラウドファンディングの台頭はまさにその象徴です。
　
これからは自ら何かを発信し、行動していかなければ、生きづらい世の中になりつつあります。
　
　
「やりがい」や「生きがい」、「自分らしさ」とは何か？
　
本当はあなたはどんな生き方がしたいですか？
　
共に考えていきましょう(^^)

足立大和（あだちやまと）
　
ガッコウ＋株式会社　研修事業部　営業部長
ＡＳＫアカデミー・ジャパン株式会社公認アソシエート・トレーナー
　
目標達成研修トレーナー及び講師として年間約180日をこなし、延べ3000人以上に研修を提供。
　
研修トレーナーとして年間50回以上研修を主催。
　
2015年ベンチャー企業での完全歩合制の営業に所属し、初年度から個人で年商1500万円の売上を作る。
　
その実績を買われヘッドハンティングを受ける。
　
さらに入社して半年後、研修事業部営業部長に抜擢される。
　
営業部長に就任してから1年半で自ら登壇する研修で売り上げ3591万2790円を記録。
　
「人々の目標を明確にし、目標達成に全力で拘ることが幸せだと思える社会創造への貢献」と「大人が堂々と夢を語る世の中の実現」が理念。
