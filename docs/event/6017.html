Ｗ杯１回戦を渋谷スクランブルでみんなで観戦しようツアーです。
ツアーとありますが、実質的には「みんなで盛り上がろう」企画。

「代表ユニフォーム持ってないから盛り上がれない」
「仕事帰りだからスーツじゃないといけない」
「でもめっちゃ盛り上がりたい！！！」
「一人で渋谷行くのは不安だよ…」

こんな方々で集まってみんなで飲みながら観戦しよう！
っていう企画です。（ちなみに企画者は上４つ全てに当てはまる）

今なら酒缶１本無料でご提供します！
参加費は無料（あ、カンパいただけると嬉しいです☆）

２２：００　渋谷モアイ像前（渋谷駅南口）
