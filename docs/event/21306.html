内容

アプリ開発、出版、講演と精力的に活動する「Androidレイアウトの魔女」あんざいゆき さんを講師にお迎えして、Android 3.0で導入されたFragmentの基礎講座/ハンズオンを開催します。

受講対象はAndroidで「Hello World」以上のアプリケーションの作成経験のある方。かつ、各自ノートＰＣにAndroidの開発環境をインストールの上、会場まで持参していただけるかた。

Androidの開発環境はAndroid4.0のSDK（エミュレータ、Eclipseとpluginを含む)をインストールの上参加ください。OSはMacOS,　Ubuntu,　Windowsのいずれでもかまいません。

会場には電源、無線LAN環境があります。

講師紹介

あんざい ゆき
uPhyca Inc. （http://www.uphyca.com/）代表取締役社長。
Android 女子部 副部長。GDD Phone を手に入れたことをきっかけにAndroid アプリの開発をスタート。自分の作ったアプリが実機で動くことに感動し、以来Androidアプリの開発にのめり込む。
図書館予約アプリ「Libraroid」や新幹線エクスプレス予約用ブラウザ「BooXpress」、Android2.3 で追加されたNFC機能を使ってSuicaの履歴を確認できる「Suica Reader」などのアプリを公開している。
・ブログ : Y.A.M の 雑記帳 http://y-anz-m.blogspot.com/
・書籍 : Android Layout Cookbook アプリの価値を高める開発テクニック http://amzn.to/fwpyVn
