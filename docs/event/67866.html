上流工程にチャレンジ（要件定義とユースケース）
～現役エンジニアによる講義～

こんなエンジニア向けの勉強会です。

  1. そろそろ上流工程にチャレンジしたい！！
  2. エンジニアを生涯の仕事としたい！！
  3. スキルアップして独立・転職を検討したい！！
  4. 要件定義・ユースケースの概要を掴みたい！！
  5. オブジェクト指向について理解を深めたい！！

【勉強会でできるようになること】

  1. ユースケース（ユースケース図・ユースケース記述）を書けるようになる
  2. 要件定義、外部設計など上流工程の流れを理解できる
  3. オブジェクト指向の理解が深まる

講演概要

要件定義・ユースケースについて

  • 概要: ユースケースを中心に要件定義におけるオブジェクト指向の考え方について解説します。

オブジェクト指向とは？

  • 概要:「オブジェクトとは？」から始まり、オブジェクト指向の利点はや体的な設計・実装について解説します。

ユースケース演習

  • 概要: 午前中の内容を踏まえて実際にユースケースを使って要件定義書を作成します。
