＜政府CIO補佐官募集コラボイベント＞
非常勤のCIO系業務として顕在化している業務の代表例として、「政府CIO補佐官」という職務があります。政府CIO補佐官の年次募集の時期に合わせて、現役政府CIO補佐官から「働き方」の紹介を行うイベントを開催いたします。セミナーを通して、兼業でのCIO系業務の働き方について理解を深めていただくとともに、政府CIO補佐官への応募について一歩踏み出すきっかけにしていただければ幸いです。

■セミナー情報

1.日時 2019年12月10日（火）　17:30-19:00

2.場所
TKP虎ノ門駅前カンファレンスセンター　カンファレンスルーム4B
東京都港区虎ノ門1-4-3 NT虎ノ門ビル 4F　(銀座線虎ノ門駅 1番出口徒歩1分)

3.参加費 無料

4.定員 30名

5.プログラム内容（予定）

(1)ご挨拶（イベント趣旨説明）/ CIOシェアリング協議会理事　坂本俊輔
(2)講演「デジタル・ガバメント戦略実現に向けた政府CIO補佐官の期待役割について（仮題）」 / 内閣官房IT総合戦略室 奥田直彦　内閣参事官
(3)パネルディスカッション「兼業で政府CIO補佐官を務めるという働き方」 / モデレーター：理事 坂本俊輔　パネリスト：政府CIO補佐官3名（後述）
(4)ご案内「2020年度政府CIO補佐官募集案内 」/ 内閣官房IT総合戦略室 味園琢磨　参事官補佐
