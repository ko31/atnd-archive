テーマ

「5年後の私は、今と同じ働き方を続けられているのだろうか…」

「長時間労働」

「土日休みに切り替えたい」

「家庭を持っても働き続けたい」

「在宅勤務って本当にできるの？」

働き方についての悩みは尽きないものです。

そんな働き方について、エンジニアという職種から考える機会を提供させて頂くイベントです。

プログラム

タイトル 内容
第1部　ITエンジニアになった先輩の体験談を聞いてみよう アパレル業界やバスガイドなど、全くの異業種からエンジニアに転身した方から、なぜエンジニアという選択肢を選んだのか、実体験を交えてお話し頂きたいと思います。
第2部　ITエンジニア体験授業を受けてみよう ITエンジニアって実際はどんな仕事なの？本当に今からなれるの？そんなことをお考えの人が多いのではないでしょうか。ここでは、エンジニアの初歩を学べる授業をご用意しております。
第3部　先輩に直接相談してみよう 販売スタッフから、営業から、事務職から…さまざまな経験を経てエンジニアになった方々から直接話が聞ける場を設けております。

　※プログラムは予告なく変更になる場合がございます。

　※途中参加・途中退出可能です。

プレゼンター

山崎 大助（ジーズアカデミー主席講師）

Microsoft MVP、デジタルハリウッド大学大学院 准教授でもあり、プログラミング、エンジニア教育のパイオニアとして数多くのエンジニアを輩出。Webアプリの開発を始め、雑誌・書籍・Web メディアでの執筆、Web 関連セミナーを主宰するなど、幅広く活動している。主な著書に『jQuery レッスンブック jQuery2.X/1.X 対応』『レスポンシブWeb デザイン「超」実践デザイン集中講義』など

林 由子（はやし　よしこ）

株式会社ヌーラボ　Backlog開発エンジニア。地元福岡の高校を卒業後、5年間バスガイドとして働く。仕事を続けるうちに、結婚などのライフイベントを経ても働き続けられる職場を求め、エンジニアとしての転身を決意。現在は、結婚を機に福岡県にある離島で暮らしながら、週1・出勤／週4・リモートワークのスタイルで、ヌーラボのプロダクト『Backlog』の開発に従事する。
