description

朝イチで行うhack-a-thonです。
特に参加条件はありません。参加者で目標を共有することもしない予定です。


＜発端＞
主催者が無職になる予定なので、怠惰な生活をしないようにがんばるのが目的です。


＜お願い＞
・エントリ時のコメントに、当日予定しているhackを記述してもらえると助かります。参加する人同士、興味があるところだと思うので。あと、面白い自己アピールなんかも書いてもらえるときっと人気者です！
・会場は、特に会議室等を押さえるものではないので、他のお客さんの迷惑にならないようにしましょう。


＜やること＞
自由に朝の時間を使いましょう。
自分一人では継続するのが厳しくても、朝早くに集まる習慣を付けるきっかけが作れると思います。きっと生活が改善されたり、脳の働きが良くなったりすると思います。


＜場所について＞
地下1Fです。


＜参加費＞
飲食代などは自費です。


＜開始時間＞
開店時間が7:30なのでその時間に合わせるのがいいと思いますが、別に仕事でもないので遅刻してもかまいません。


＜終了時間＞
各自で決めてください。


＜その他＞
・PC／電源／無線LANなどの環境は各自で用意してください。
・座席などは別々になってもかまわないと思うので、適宜空気を読みましょう。
・ハッシュタグ： #mhack1
・その他、当日のご連絡はtwitterにて@marblejenkaまでつぶやきください
