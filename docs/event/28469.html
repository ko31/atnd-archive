内容

5月の技術ひろばは、キャリアデザインセンターさんの会議室をお借りして開催します。
マイクロソフトの技術は実はそれほど詳しくないという方にもわかりやすく紹介します。マイクロソフトの技術を使っている方は技術の汎用性や他の環境との接続性を再確認する機会としてご参加ください。

キャリアアップを考えている方も、他の参加者の熱意に触れて自分の意識を高めたい方もご参加ください。
もちろん 「MS の技術に慣れているが、いつも通り勉強会に参加する」という方も大歓迎です。
異なる立場の参加者がお互いに刺激し合いながら交流しましょう。

  • 「OData って何？」 （瀬尾）
    OData (= Open Data Protocol) は、データの問い合わせと更新のための汎用的なプロトコルです。OData に対応したサーバーとクライアントとは、相互に相手を限定せずに通信することができます。
    実際に OData の Producer （サーバー側）と Consumer （クライアント側）を作りながら、プロトコルの利用方法を見てみます。
  • 「Windows Azure に見るクラウド開発環境とは」 （MS 関田さん）
    Windows Azure の登場から2年以上が経過して、事例が増えると同時に様々な領域での利用が進んでいます。その一つの利用が柔軟で相互接続性の高い開発環境にあります。このセッションでは、現在の開発環境を整理するとともに、.NETのみならず、その他の言語も含めて最新の開発環境についてご紹介します。
  • 「Windows Phone 超入門」 （初音さん）
    （詳細調整中）Windows Phone、マイクロソフトの技術に精通していない技術者にもわかる Windows Phone の入門セッションの予定です。MS 技術に慣れている方にも役に立つ情報をあわせて紹介していただきます。
