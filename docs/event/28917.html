セミナー概要

　企業ウェブサイトを中心に世界的に利用されているソフトウェアに WordPress がありますが、工夫次第ではさらに多様なサイト構築が可能となります。今回の勉強会は、マルチサイト（複数サイト）とそれと連携したオープン型SNS（ソーシャルネットワーキングサイト）を構築できるBuddyPress について、株式会社コミュニティコムの星野邦敏さんを招いてお話を伺います。

　WordPress は バージョン3 以降、シングルサイト単体の構築に使う場合だけでなく、マルチサイト（複数サイト）としてもサイト構築が比較的簡単にできるようになりました。
　まず前半は WordPress を利用したマルチサイト構築について、マルチサイト機能の実装方法について、運用や注意点を交えて説明させていただきます。

　また WordPressマルチサイト機能に加えて、それと連携したプラグインに BuddyPress プラグインがあります。BuddyPressプラグインを利用することで、ユーザグループ、コミュニティやフォーラムなど本格的なオープン型 SNS（ソーシャルネットワーキングサイト）が構築が可能になります。
　後半はマルチサイト構築に加え、BuddyPress プラグインの利用方法を、運用や注意点など含めて説明させていただきます。但し、時間の都合上、今回はテーマカスタマイズ等は含まない基本的な実装メインの内容となります。何卒ご注意お願いいたします。
