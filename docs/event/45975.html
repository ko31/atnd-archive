Japan.R とは

日本各地で統計処理ソフトウェアである R に関する勉強会が行われています。

  • Tokyo.R
  • Tokyo.R 女子部
  • Tsukuba.R
  • Wako.R
  • Nagoya.R
  • Osaka.R
  • Shiga.R
  • Hirosima.R
  • 沖縄R同好会(Okinawa.R)

これらの勉強会の主催者や参加者が年に一度集まって開催されるイベントが Japan.R です。

( Japan.R 2013 ATND はこちら http://atnd.org/events/44682 )

懇親会

当日の勉強会終了後、18:00 – 20:00で同会場にて懇親会(ピザパーティー)を予定しています。

参加を希望される方は受付時に参加費(2000円)をお支払い下さい。

なお学生は無料でご参加頂けます。

申込期限

当日でも参加可能ですが、事前の人数把握のため参加希望者は登録をお願いします。
