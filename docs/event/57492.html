概要

第7回目のPure Data パッチングサークルを開催します。

パッチングサークルは2013年の7月より奇数月に開催しているPure Dataの気軽な勉強会です。
と言いつつ、今回から諸事情により偶数月開催となりました。
今回も新宿Hatchup様のイベントスペースを使用させて頂けることになりました。（Hatchup様は10月いっぱいで引っ越しをされるとのことなので、こちらの会場を使用するのは今回が最後になる予定です）
前回はハッカソン形式でしたが、今回は再び勉強会、発表会形式になります。
Pure Dataにご興味がある初心者の質問から実際に使っているユーザーのパッチの披露まで、自由に発表したり情報交換をしていきましょう。

  • 出てきた質問や疑問をみんなで考えるタイム

主催

松村誠一郎(Pure Data Japan)

備考

当日は、特に主催者側でコンピュータやPdの用意はありません。ご自身でPd-extendedをインストールしたパソコンをご持参ください。
（重要）Pd-extended(Mac)のインストールについて
Mac OSXにPd-extendedをインストールする場合は、32bit版の最新版の方にしてください。OSが64bitでも問題なく動作します。
64bit版の方はβ版で時々動作がおかしくなるのを確認しています。
