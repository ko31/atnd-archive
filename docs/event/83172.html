テーマ

Processing
※Processingが主となっていれば、その周辺分野も可。

技術系のアドベントカレンダーは、ブログ記事の投稿が多いですが、スケッチ投稿も大歓迎です。

ルール

1. コンテンツとしてProcessingに関する「作品の投稿」や「ブログ記事」や「プロジェクト」の紹介 など を行う。(他にも面白い企画があれば是非参加ください)
※作品の場合、原則として技術解説 又は ソースコードを公開する。
※不明な点があればATNDメッセージや、ATNDコメント、 TwitterDM 、 Twitter@ に気軽にご相談ください。
※コラボレーション企画等も面白いですよね。 (例)

2. コンテンツの公開は原則として、当日の0:00より公開できるように前日までに作成、管理人に連絡する。

3. コンテンツがイベントの内容にふさわしくないと判断した場合、主催者として本コンテンツの掲載をお断りさせて頂く場合がございます。(次のようなものがふさわしくないコンテンツの例です)

  • 特定の方(団体)へ迷惑をかけるコンテンツ
  • イベントの趣旨と異なったコンテンツ
  • イベント(やその内容)に対する否定的なコンテンツ
    ※イベントの趣旨に沿った内容であれば、問題ないはずです、イベントを円満に運営するために設けさせていただきました。
