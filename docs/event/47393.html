description

町家で商いを考える。

あなたのスキなことは、きっと多くの人を巻き込むことができます。
持続可能な"商い"にして、多くの人と楽しみながら働きましょう。

アキナイトは"スキ"を"商い"に変えようと試みるイベントです。
あるコミュニティスペースに1ヶ月に一度、夜集まって、スモールビジネスをつくります。
実践と振り返りを1ヶ月間周期で繰り返し、参加者同士で学びを深めます。

今回はゲストがとっても豪華！
そして場所はなんと町家！

「集まった仲間と商いをつくること」をコンセプトに置きながら場所も形も変えていくのがアキナイトのスタイルです。

①ゲスト起業家代表者プレゼン『起業のレシピ』

藤井ジャック孝騎（ふじいじゃっくこうき）【経営コンサルタント】
24歳。滋賀県大津市生まれ、福岡育ち。立命館アジア太平洋大学、アジア太平洋マネジメント学部卒業。大学時代にはインキュベーション施設での起業家の方々の支援並びに広告の営業、その他ベンチャーキャピタル会社2社でのインターンシップ。その後、2012年に選考倍率1750倍の難関である大手老舗人材教育コンサルティング会社に入社し、パーソナルコンサルタントとして、企業経営者、各種士業の方々を中心にサポート。現在、株式会社ビジネスバンク西日本支社立ち上げの依頼を受け、2014年1月入社。起業家・起業志望者向け事業コンサルティングを行っている。（※今回の起業家代表者プレゼンで講演をしていただきます）

※約30分間の講演となっています。今回のイチオシです！

②ゲスト起業家とフリートーク『なぜ起業したのか、なぜ起業するのか』

①扇沢友樹（おおぎさわともき）【シェアハウス経営】
25歳。学生時代にマドリが好き過ぎて不動産の道を決意。内定を辞退し大学卒業と同時に株式会社めいを創業。シェアハウスめいちゃんちや京都版トキワ荘事業では住人が『好きな事を仕事にする』のを応援する大家さん役。不動産にコンテンツを入れて幸せでクリエイティブなコミュニティ作りを目指す！
②中尾 怜史（なかお さとし）【広告コンサルタント】
Dクリエイト株式会社代表取締役建設業界で、システム開発、営業、現場監督を7年間おこない、自身が開発・販売に係わった新技術が中小企業単体では始めての国土交通省推奨技術として認められました。その後、広告業に転身。現在は店舗に向けてリピート率向上に特化した広告システムの提案と営業代行業務を行い新規事業の立ち上げの支援を行っています。
③山川勝弘（やまかわかつひろ）【学習塾経営】
24歳。学習教室を自営で2013年11月にスタート。小商いに興味を持ってアキナイトを立ち上げ、自分で商いを実践中。そろそろ飯が食えるようになるかも。。学生時代には「College cafe」「学生ヨル会議」などスキなもので人がつながれるような場所づくりに励んできた。学生から20代前半で商いを実践し、アイデアを共有できるような場所をどんどん作っていきたい。（※今回のアキナイトの代表であり主催者）
④重光あさみ（しげみつあさみ）【映像ディレクター・イラストレーター】
1989 年生まれ。京都在住。京都精華大学卒業後、 クリエイティブユニット「Focha!」を結成。カラフルでポップな映像やデザインが特徴。 2013年の祇園天幕映画祭で制作した「よーじや」のCMは観客賞を受賞。テクノポップアーティスト「spoon+」の PV 監督、tofubeats『水星』PV の製作や、神戸のアイドルグループ「KOBerrieS」PV 監督などを行う。2012年から始めた京都の鴨川で毎月22日に巨大ショートケーキを囲んでピクニックを行うGranewtonPicnic! というイベントを主催。テレビや雑誌でも活動を取り上げられる。 HP
⑤川人ゆかり（かわひとゆかり）【キャリアコーディネーター/旅人】
大学卒業後、人材紹介会社に就職し一貫して約3年間企業の採用支援に携わる。退職後はソーシャルビジネスのシーズを探る無期限の旅に出かけ一年数ヶ月に渡り世界を一周。帰国後は前職の経験を活かし「日本の若者にもっと多様な選択肢を」をミッションに、独立。現在はローカルキャリアカフェという地域活性中間支援団体の代表として移住に関するイベントの企画・プロデュースなどを手がける傍ら、グローバルキャリア教育プログラムの企画開発や司会業なども行っている。
