■基礎から学ぶWEBサイト企画塾
http://powerstory.jp/0509plan

WEBサイト（ホームページ）企画塾は、
参加した塾生の皆様が、幸せな自己実現の＜スキル＞をひとつ持ち帰っていただくことが目標です。
●自分自身の持つ＜強み＞を理解し、コンテンツ化するスキルを得る
●複雑なWEBサイト構造を、簡単に理解し、自分で構成が考えられるスキル得る
●在宅ワークで、少しでも利益が得られるWEBサイトを企画できるスキルを得る
その他、塾生の現在のスキルに合わせて、内容を進行いたします。

個人事業主、週末起業家、子育て、介護などで忙しい女性、在宅勤務を考える方を対象に、
プロのホームページの作り方や考え方を、誰でもわかりやすい内容で学んで頂きます。

特に企業、事業、個人、商品のブランディングやブランドコンテンツを作りたいと思っている方には、
最適な内容になっています。

※少人数で一緒に簡単なワークを楽しみながら、学びますのでリラックスしてご参加ください。
※パソコンは使用しません。

 

●第１部　午後14時～16時30分　（12名限定）
これから公開したい、自分のホームページ編

講義とディスカッションを通じて、
パーソナルブランディングを強調した、
ホームページ（WEBサイト）の企画書を制作します。

【対　象】
・WEBサイトやブログを公開したい、見直したい人
・在宅ワークで、WEBを利用したい方
・WEBを活用して、アフィリエイトや
ショップ運営などを考えている方
・WEBで自己実現を考えている方　など

【内　容】
・WEBサイトの今を知る（事例を見ながら、訴求力のあるWEBサイトとは何かを知る）
・自分のブログが変わる、日記風コンテンツでブランディングするコツ
・アクセスを稼ぐために、公開する内容の「質」にこだわるには・・・
・自分を知る、ブランディングする近道とは？
・続かない人が続けられるサイト運用法とは？
・WEBサイトで在宅ワークを考える　など

【受講のお申込は、この行をクリックしてください】

 

●第２部　午後17時30分～20時　（12名限定）
自分の強みで活性化、PRコンテンツ企画編

インターネットを利用してビジネスをするために、
アクセスユーザーの興味と行動を喚起させる
PRコンテンツの企画書を作成します。
ショップサイトの事例を交えながら
運用のコツも学びます。

【対　象】
自分でブログやネットショップを運営している方
ネットショップの運用を担当している方
ネットショップや、アフィリエイトなどに興味がある方
自分のホームページをリニューアルしたい方
ビジネスでホームページを活用したい方

【内　容】
・WEBサイト分析
～できるWEBサイトは何処が違う？～
・コンセプトワーク、コンテンツメイキング法～新しいコンテンツを企画しよう～
・どこで公開するのか？何を公開するのか
・アクセスを集めたいキーワードを選ぶ
・自己実現の近道：WEB実現マップを作る
・続けられる運用ノウハウ　など

【受講のお申込は、この行をクリックしてください】

 

■講師／パワーストーリー企画事務所　代表　押岡秀多
これまで企画したWEBサイトの業種と内容

・健康食品メーカー　通販サイト　コンテンツ企画
・化粧品メーカー　通販サイト企画
・鉄道会社　沿線観光情報サイト企画

その他、新規顧客誘致のためのコンテンツ企画や
ファンサイトも多数企画
