第1部　パワーデバイス用封止材における耐熱性・放熱性向上
【10:15-11:30】

【講演主旨】
　現代社会に「電気」は欠かせません。パワーデバイスは、電気=電力の制御を行う半導体部品の総称であり、これらの性能は電力エネルギーの消費量に大きく影響します。近年、地球の温暖化を防止するためエネルギー消費量の削減=省エネルギー化が叫ばれており、パワーデバイスの高性能化に注目が集まっています。また、パワーデバイスは動作により発熱を伴うので、これらで使用する材料には耐熱性や放熱性が必要となります。本セミナーでは、民生用パワーデバイスにおける高耐熱・高放熱性材料の開発経緯、現状の問題、および、今後の対策などについて分かりやすく、かつ詳細に解説します。

　【プログラム】
1.はじめに
1-1　弊社成長戦略における再生可能エネルギーへの取組強化について

2.バワーデバイスの封止技術
　2.1 封止法:樹脂、気密
　2.2 封止材:エポキシ、シリコーン

3.パワーデバイス用封止材の評価
　3.1 一般特性:業界規定項目
　3.2 信頼性:耐湿性、高温特性、他

4.パワーデバイス用封止材の課題
　4.1 高耐熱化:多官能、強靭骨格
　4.2 高純度化:低ハロゲン化
　4.3 高放熱化:熱伝導率と対応状況

5.パワーデバイス用封止材の高耐熱化

6.パワーデバイス用封止材の高純度化

7.パワーデバイス用封止材の高放熱化
　7.1 フィラー
　7.2 充填技術
　7.3 表面改質

8.パワーデバイス用新規放熱性封止材
　8.1 既存封止材の問題
　8.2 新規放熱材の開発

第2部　大電流化・耐熱性向上に向けたパワーモジュールの熱設計と求められる材料
【12:15-13:30】

【講演概要】
　パワーモジュールの市場および用途の拡大と共に、高耐圧化、大電流化、高熱伝導化、高耐熱化の要求が高まっている。そこでパワーモジュールのパッケージ構造や材料は、上記の要求項目を満足させるために非常に重要な要素技術である。
　本講義では、パワーモジュールの構造やパッケージ材料に必要な性能について、特に、熱設計に欠かせない要素技術、実応用での電力損失・温度シミュレーション方法、また、パワーモジュールの温度上昇と信頼性との関連性について解説する。

【プログラム】
1.パワーモジュールの市場要求
　1-1　応用マップと要求事項
　1-2　パワーチップの技術動向

2.パワーモジュールのパッケージ基本構造
　2-1　絶縁と熱抵抗
　2-2　組立技術と信頼性設計

3.パワーモジュールの電力損失と放熱
　3-1　電力損失と温度上昇
　3-2　熱の流れ(放熱)
　3-3　計算例(シミュレーション)

4.パワーモジュールの放熱技術
　4-1各部材料の熱特性
　4-2高放熱構造
　4-3構造・材料と放熱特性

5.パワーモジュールの温度と信頼性
　5-1動作温度
　5-2パワーサイクル寿命
　5-3温度サイクル寿命
