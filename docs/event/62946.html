消費税、相続税など、税金に関する話題が世間を騒がす昨今。書店では「節税」をテーマにした書籍が多く並び、少しでも納税による負担を軽減しようと情報収集を行なう人が増えています。

特にオーナー社長にとっては、利益を最大化するという点において「法人税」だけでなく、自身の「所得税」に対してどのような対策を行なうべきか、あらためて問われる時代が訪れています。

そんな中、法人税と所得税を双方削減する方法としての「保険の活用」が、あらためて注目されていることをご存知でしょうか。

償却資産の購入や従業員の賞与授与など、様々な節税法が存在する中、短期間で導入でき、絶大な節税効果を持つ保険は、オーナー社長の中で今や定番となっていますが、間違った活用法により、思ったよりも効果が実感できないというケースもあるのです。

そこで、今回のセミナーでは、年間2万件のファイナンシャルコンサルティングの実績を誇るブロードマインド社を招き、「移転スキーム」を活用して会社の利益を低税率で個人へ移転することで、法人税と個人の所得税を同時に減らし、5年後のオーナーの手取り収入を劇的に増やす、究極の「保険活用術」について詳細にご説明致します。
戦略的に保険を活用することで、これまで以上の節税効果が期待できます。

そして本セミナーの第2部では、ブロードマインド株式会社の舘林芳夫氏を招き、2015年12月からストレスチェックを従業員に受診させることが義務化されるという「改正労働安全衛生法」への新しい保険活用法についてもご解説致します。
同時に、政府労災でカバーできない領域の保障方法や複数かけている火災保険料を下げる方法など、知らないと損する損害保険の利用方法も案内致します。

日々、情報収集に奔走される忙しいオーナー社長にとっては、法人生保と法人損保という二つの方向から「有効的な保険の活用法」について学べる、貴重なこの機会を、是非ご活用下さい。

皆様のご来場を、心よりお待ちしております。

～講師紹介～


ブロードマインド株式会社　櫻井祐二氏

大手都市銀行にて、中小法人向け融資業務に従事。不動産バブルの狂騒と崩壊それぞれを経験。不良債権処理に明け暮れる中で、「企業オーナーに本当に役立つ仕事とは何か？！」を自問した結果、保険営業の世界へ飛び込むことに。銀行サイドからの視点による資金繰り対策や債務軽減対策は、法人オーナーから高く評価されている。業界最高の名誉となるMDRT成績資格終身会員。
税理士とタイアップした、法人税・所得税・相続税の軽減対策も人気が高く日本各地でセミナーを開催している。




ブロードマインド株式会社　舘林芳夫氏

2009年　ブロードマインド株式会社入社。損害保険のスペシャリストとして、わずか数名の損害保険事業部を、3年で特級代理店（ハイグレードエージェント）へ導く。現在は、上場企業やオーナー企業のリスクマネジメントに全国を奔走。 高級フレンチレストランのソムリエ経験から富裕層向けのリスクマネジメントにも定評がある。
