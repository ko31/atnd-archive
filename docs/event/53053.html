2005年にスタートさせたこの関西スポーツネットワーク交流会、通称「カンスポ」。次回がいよいよ31回目になります。

今回は原点に立ち返り、
「関西のスポーツ界をいかに盛り上げるか」
をテーマに掲げたいと思います。…

ゲストは今年、Baseball First Leagueを開幕させた仕掛け人、高下沢氏。
今までにない新しい取り組みをここ関西でスタートさせました。
「野球とは何か」
「スポーツとは何か」
をBaseball First Leagueの取り組み、主旨などを通じてお話しをいただけたらと思います。

そして後半は恒例の交流タイム。
大阪の夜景を見ながら関西のスポーツ界をいかにビジネスとして立ち上げ盛り上げていくか、そのためのネットワーク構築に存分にご活用くださいませ。

【特別講師プロフィール】
高下　沢（こうげ　たく）
1984年生まれ。広島県呉市出身。Baseball First League共同代表、兵庫ブルーサンダーズ球団代表、三田市観光協会理事。
小学校3年生より地元チームでソフトボールを始める。大学に進学後、日米国際親善野球大会で首位打者と打点王を獲得。日本初のプロ野球独立リーグ・香川オリーブガイナーズに同年、大学を休学して入団。成績不振による自由契約後、復学して就職。サラリーマン生活をスタートさせるも、野球への夢が捨てきれず再び独立リーグの世界へ。関西独立リーグ加盟４球団を渡り歩くことになる。
2010年神戸9クルーズの解散により、兵庫県三田市をホームタウンとした兵庫ブルーサンダーズを立ち上げる。「NPO法人が運営する野球チーム」、「県庁所在地以外にホームタウンを持つ球団」という試みは、日本初の取り組みとして注目されている。2011年より、芦屋大学との教育提携がスタート。大学生がプロとアマの垣根を越えた新しい取り組みもスタートさせる。地域に愛される球団・選手の育成を目指して、地域貢献活動を行うことで地域の方々に若さと元気を発信する球団作りを目指している。
