description

多くの会社で作っていらっしゃる理念やロゴマークですが、御社では何のために作って、どう使っていますか？
本来、会社のロゴマークやコーポレートカラーは自社の姿勢や理念を象徴し、広くコミットするためのものです。

会社のビジョンやミッションを作成している会社が多いですが、自社の社員やスタッフにはその想いを無意識レベルに落とし込んで行動できて工夫をしていますか？
ロゴマークや看板に自社の姿勢や考えをリンクさせることで、社員やスタッフの行動が変わり、お客様にも自社の姿勢を伝えることができます。
新しくロゴや看板を作り直すのではなく、今ある物にしっかりと意味を落とし込み、社員の育成やお客様との関係作りに使うためのセミナーです。

前回のセミナーでは、ご参加の方から｢考えてみたこともなかったけど、たくさんの気づきがありました。｣と大変ご好評をいただきました。

◆こんな方はぜひ！
　・ロゴマークを使って自社の思いや姿勢をお客様に広く知ってもらいたい方。
　・自社の理念を理解し、それに基づいて自ら行動ができる社員を育てたいとお考え方
　・自社の軸、ロゴマークについて再度考えたい方

◆前回ご参加いただいた方のご感想

　　・自社の理念は言葉が抽象的で分かりにくいと気付いた(４０代男性・経営者）
　　・ロゴマークに意味付けをするべきだと感じました。　(５０代女性・経営者）
