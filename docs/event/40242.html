「野球交流会」の第8回です。久しぶりにやります。

最近野球してますか？
僕はしていません。野球はいつでもやりたいのですが、WBCを見てからさらにやりたくなりました。
ということで、みんなで雑談しながらゆるゆる野球をしましょう。
堅苦しいルールはほどほどなくして初心者でも楽しめるゲームにしたいと思っています。

それと今回新しく固定チーム制・選手表彰を導入します。
チームが固定されることによって打率や打点を計算し、互いに切磋琢磨することができます。
もしこのページにハンドルネームで参加申請をしてもらえるならば、
次回以降のこういったイベントページで表彰結果を発表しようと考えてます。

あと今回飲み物の持参は不要です。

その他何か意見があればコメント、アンケートによろしくお願いします。
いずれも未記入可です。
