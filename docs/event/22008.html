【SaCSS ＋ とは】

実際にコードを書きながら使い方や作り方などを学び、仕事でも使えるようになる！制作出来るようになる！というのを目的とした勉強会です。

通常のSaCSSとは別物としてのSaCSS ＋（SaCSSプラス）です。

実際にコードを書いたりしながら、使い方・作り方を学び、分からなければその場で質問し、しっかりとした作り方を自身で実践しながら学ぶ勉強会です。

そのため、時間も通常の倍の時間をかけて勉強をする内容です。

今回は+1に引き続き、jQueryを使う会を予定。jQueryを使って、目標の動作を達成するためのコードを黙々と書く勉強会、 個々人の実践メインの内容です！

【通常のSaCSSとの違い】

通常：
情報交換や最新技術を学ぶために、ノウハウや最新情報の共有を目的としたセミナー

SaCSS ＋：
時間をかけて、たっぷりじっくり、実際にコードを書きながら学ぶ勉強会！

【SaCSS + 2 の対象者】

  1. jQueryを使ったコードを書いていて困ったことがある方
  2. jQueryのプラグイン利用くらいしか経験がないが、将来的にカスタマイズの知識も必要な方
  3. jQueryの知識をより得たい方
  4. 普段勉強の時間や機会がないが、集中して勉強をしたいと思っている方
