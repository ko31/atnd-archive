ハースストーン日本語化二周年ということで、Blizzard社に協賛・スポンサード頂き東京・大阪で炉端の集いを同時開催します！
炉端の集いに参加してハースストーン日本語化二周年を共に祝いましょう！

本イベントでは各種企画の他、ハースストーンをモチーフとしたケーキや参加賞、賞品等をご用意する予定です。
奮ってご参加下さい！

※本ページは東京の応募ページです。同日開催される大阪の応募ページはこちらです。

イベント内容

・ハースストーン東西戦
同時開催の大阪会場とリンクして、東西の代表者が激突します！豪華賞品あり！
関東の代表を決めるトーナメントのエントリーは、当日の入場時に先着で受付となります（先着48名）

・3人チームによるBO3(対戦は同時進行/チームはランダム選出です)
・スタンダードフォーマット
・Asiaサーバ
・シングルエリミネーション
・チーム内のヒーロー被り有り
・マッチ間のデッキ変更有り
・マッチ間のヒーロー変更有り
・他の対戦者の観戦禁止
・対戦中の回線切断について
　→対戦に復帰できた場合：そのまま継続
　　対戦に復帰できなかった場合：当該マッチを切断した側の敗北として扱う
・ゲーム外補助ツールの使用禁止

・ゲスト相談コーナー
ゲストの蒼汁さん、Yanakenさん、wingさんによる相談コーナーです。
相談内容は自由！東日本を代表するトッププレイヤーと交流するチャンスです！

・炉端の喧嘩
炉端の集い限定のゲームモード「炉端の喧嘩」を使用しての対戦会です。
10月のルールはこちら。
