現役大学生を対象に、就職活動準備やスキルアップ、そして就職に向けての人脈づくりを目的とした、現役の企業経営者・大手企業のMBAホルダーによるビジネススキル・マーケティング講座です。マーケティングの基礎知識から学びますので、初心者でも参加可能です。

この募集は、５月１３日（日）に開催される無料体験模擬講座の募集です。

体験講座（1回目）13:00-15:00
体験講座（2回目）16:00-18:00

1回目と2回目は同じ内容です。
1回目と2回目は別々のATND掲示板で募集しておりますので、ご注意ください。（こちらは2回目の募集掲示板です。※1回目の募集は次のATND掲示板をで行っております。 http://atnd.org/events/28586 ）

体験講座の内容

「就活に役立つマーケティング入門」講座
（ケーススタディなどを通じて、マーケティングの基礎を学んでいきます。）

基礎的な内容ですので、マーケティングの基礎知識がない方でも参加頂けます。

１）　マーケティングって何？　～マーケティングの定義と範囲～
２）　事例を通じてマーケティング基礎知識を学ぼう　～ケーススタディ
　　　２－１）　セブン・イレブンは「何屋さん」？
　　　２－２）　焼肉「牛角」は何故成功したのか？
　　　２－３）　サントリー「伊右衛門」に見るブランド戦略
　　　　　　　　　　　※ケース内容は変更される可能性があります。
３）　マーケティング知識を就活に活かす
　　　３－１）　自分自身を「マーケティング」する方法
４）　マーケティングの実践的学習方法
５）　質疑応答
