【新規リリースのご案内】- ELITES(エリーツ) - Ruby on Rails編


プログラミングスクールELITES(エリーツ)に新たな「ELITES Webエンジニアコース（Ruby on Rails編）」をリリースいたしました。同コースを受講することで、未経験者でも最低限必要な基礎的な知識から、卒業後をしっかりと見据えたカリキュラムをこなすことが可能です。



「ELITES(エリーツ)」および「Webエンジニアコース」について


「ELITES（エリーツ）」とは、本気でプログラミングを学びたい人向けの「実務経験カリキュラム有り」の中期型プログラミングスクールです。当スクール「Webエンジニアコース」においては、インターネット上のWebサービスを作れるエンジニアになるための必要な知識を全て学べ、 実践経験を積むことができるカリキュラムを提供しております。又、ELITESは実務での通用力を最重視しており、職業エンジニアとして独り立ちし、世の中にサービスを生み出すことの出来る「生産者」を生み出すことに注力しております。

学習内容：【Ruby on Rails編】

HTML / CSS / Bootstrap / JavaScript / jQuery / git / Ruby / Rails / MySQL / Linux(CentOS) / Apache / Slack



ELITES CAMP一般募集！


６か月間の中期プログラミングスクールのカリキュラムとなる、4日間のブートキャンプ「ELITES CAMP」ですが、今回特別に一般募集を開始致します。 「ELITES CAMP」のみの参加ＯＫ！その後「ELITES本科」への受講を検討頂くことも可能です。 ELITES CAMP参加後、ELITESへの受講となった方へは、特別割引も！！ 本気でプログラミングを学びたい方のための特別応援企画です。
