第１６回勉強会を開催します。

◆「Androidとマイコン連動の紹介とC#によるマイコン開発（基本編）」
　　講師：有限会社A.PLUS 代表取締役　富永 勉 氏
◯Android組み込み開発の現状紹介
◯C#でマイコンを開発しよう！（紹介編）
・.NET Micro Frameworkを用いたマイコン開発の紹介
・開発環境の紹介と簡易ハンズオン
・各種センサやLEDの制御などのデモ　など
◯質疑応答、フリートーク

※簡易ハンズオンについて
・ハードウェア（netduinoもしくはnetduino plus）は５台程度をシェアする予定です。
　数が少ないので、netduinoもしくはnetduino plusをお持ちの方はご持参ください。
・開発環境構築やサンプルコードの動作確認をしたいかたは、下記の３つのソフトウェアをインストールしたPCを各自でご持参ください。
　1. Microsoft Visual C# Express 2010
　2. Microsoft .NET Micro Framework v4.1 SDK
　3. Netduino SDK v4.1
　　http://www.netduino.com/downloads/netduinosdk_32bit.exe (for 32-bit Windows)
　　http://www.netduino.com/downloads/netduinosdk_64bit.exe (for 64-bit Windows)
　注）詳細は、Netduino開発元が提供するPDFをご参照ください。
　　　http://www.netduino.com/downloads/gettingstarted.pdf

ではでは、お気軽にご参加ください。
