『開発者支援制度Information Meeting』

Cloud Core VPSでは『Cloud Core VPS開発者支援制度』といって
日本の様々な開発コミュニティを応援しています。

支援制度としては、開発サーバーの無償提供と、勉強会用の
会場提供をしておりますが
今回、開発者支援制度Information Meetingと称して
開発サーバーとして提供するCloud Core VPSのデモと
提供する勉強会用の会場と見学、また開発コミュニティの交流会を開催いたします。

◆開発者支援制度について
http://cloudcore.jp/vps/develop/

◆詳細内容（予定）
・CloudCore VPSのご紹介
・CloudCore 開発者支援制度について
・参加者しているコミュニティ紹介　
・懇親会（軽食あり）＆ライトニングトーク

◆日程：2011/12/7（水）19：00-21：00
◆参加費：無料
◆定員：20名
◆主催：KDDIウェブコミュニケーションズ
