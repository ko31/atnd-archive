パーティー概要
X’masまでちょうどひと月。
都会の街並も徐々にクリスマスムード一色に…。
心躍る季節に一足早いクリスマスイブ前月夜際をお楽しみください。
この時期にいい人に出会えたら最高ですね(^^)
またディズニーペアチケットが当たるゲーム大会も用意しているのでお楽しみに!!

☆会場は飲食総合情報Webサイトぐるなびで、常に上位にランクインする人気店『1st.5Garden』をお借りいたします。
店内は2フロアに分かれており、メインフロアは白をモチーフに、5mある天井に煌びやかなシャンデリアとガラス張りの贅沢な非日常空間を演出。
また最上階から銀座の夜景を眺める事ができるガーデンテラスの2フロア行き来しながら、クリスマスイブ前月夜祭パーティーをお楽しみ下さい。

◇出演(予定)クリエイティスト
占いなど
行き方：東京メトロ銀座線銀座駅からの案内
B4出口を出てすぐのT字路を右に曲がります。曲がってすぐ左手にGAPが見えます。そこの道をひたすらまっすぐ進み続けると右手に銀楽ビルディングが見えてきます。そこの十字路を右に曲がってください。しばらく進むと左手にギンザファーストファイブビルが見えてきます。エレベーターで11階に上がり、そこが受付となります。
(銀座一丁目駅C-6番出口2分、銀座駅A-13徒歩3分、JR有楽町駅京橋口2分)
