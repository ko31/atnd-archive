description

大手出身の営業達がベンチャー企業のリアルを余すこと無く語る合同採用イベントです！！

　
「ベンチャーってそもそもどうなの？」
「ベンチャーの営業スタイルってどんな感じ？」
「超忙しいイメージなんだけど実際どうなの？」

ベンチャー企業に興味はありつつも、こういった疑問や不安があって、「なかなか応募に踏みきれない･･･」、「結局名の知れた大企業や有名企業の方がいいのかな~」なんて悩んだり迷われてたりしていませんか？

今回、各メディアでもよく取り上げられている注目ベンチャー3社が結集し、実際に各社で働く大手企業出身の営業達が本音で語り合い、ベンチャー企業の営業職について「是非、本当のところを知ってもらおう！」という趣旨の採用イベントを開催することになりました！！

皆様からの質問に対してNGなしで何でも回答！
ホームページや求人広告からだけではわからない現場の「生」の声をお届けします！

■こんな方にオススメ

　- ベンチャーに興味はあるが、まずは実際に働いている人の話を聞いてみたい
　- ベンチャー企業で働くメリット・デメリットを聞きたい
　- ベンチャーと大手企業、ぶっちゃけどっちがいいのか聞いてみたい
　- 色んなべンチャーの話を聞いてみたいが、一社ずつ話を聞きに行く時間がない
　- とにかく、この3社に興味がある
