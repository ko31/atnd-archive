description

問題解決を学べば、自分の可能性を最大限に高める事ができ、 自分にできる最大限の夢を達成できるようになります！

・夢はあるけど、どうやって達成すれば良いかわからない。
・あの人は、ものごとの本質を突くのが上手いなぁ。
・いつの間にか、自分では絶対に解決できない事に挑戦していた。​
​
こういうことを思ったことはありませんか？​

問題解決とは言い換えると「ものの見方・考え方」です。
問題解決を身に付ければ、

・仕事や日常生活で直面する問題に対処できます。
・自分は何をするべきか、より具体的に、鮮明に認識する事ができます。
・今よりも深く思考を掘り下げる事ができ、ものの本質を見極める事ができます。
​
そして、自分の可能性を最大限に高める事ができ、
自分にできる最大限の夢を達成できるようになるのです。
​
昔の私は「考えるよりもやってみる方が早い！」というポリシーを持ち、まずは挑戦が信条でした。
そんな私が、問題解決について学んだのは転職先の先輩からです。私は体中に衝撃が走り、
​
何故今まで誰も教えてくれなかったんだ！？問題解決を知っていれば、
「やってみるよりも考えた方が早い」ではないか！
​
と思いました。私は、単純にものの考え方がわかっていなかったので、
やってみないと何も見えなかっただけなのです。
私はこのスキルを広く伝えたいと思い、この勉強会を立ち上げました。
​
問題解決を身に付ければ「何が問題なのか」「原因はどこにあるのか​」「どうやったら達成できるのか」
正しく考える事ができます。「自分で考え、行動する」「論理的に周囲・上司を説得する」
事が強く求められる現代社会では、必須のスキルです。
​​
難しそうだと思うかもしれませんが、「それだけ！？」と思う程、単純で簡単なスキルであり、
人生に大きな好影響をもたらすので、是非ともご参加ください。
