北陸出身のみなさま・北陸にゆかりのあるみなさま

7月12日（土）新宿にて【北陸人限定】の交流会開催！

今回で第5回目を迎える国道8号線会とは、
富山・石川・福井そして新潟を加えた4県にゆかりのある人のための交流会です。
北陸自慢の日本酒を用意しております。

4月に開催された第4回の8号線会では、
20〜30代を中心に20名以上の方が集まりました。

北陸出身者はもちろんのこと、
北陸での通学・通勤経験があるなど北陸にゆかりのある方、北陸が好きな方など、
たくさんの北陸人のみなさまにご参加いただき、
今回も一緒に北陸の日本酒の良さを再発見しながら、新しい繋がりを作りたいと思います。

初参加やお一人での参加の方にも楽しんでいただけるよう努めますので、
是非お気軽にご参加ください。

北陸の日本酒は富山・石川・福井・新潟の銘酒を各2種類ずつ用意しますので、お楽しみに。
