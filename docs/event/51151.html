ＢＢ-ＪＵＮＧＬＥで森林戦！

title ＢＢ-ＪＵＮＧＬＥで森林戦！
catch 生い茂る木々、点在するバリケード、ブッシュと地形を味方につけろ！
description

来月6月14日（土）に千葉のフィールド「BB・JUNGLE」にて祐樹主催のゲーム会を開催いたします。
森林フィールドを舞台に初心者や女性の方でも楽しめる会ですので、ご興味のある方は是非お誘い合わせのうえご参加下さい。

※5日現在、別所での募集で20名程の参加申し込みが来ていますが
フィールドの面積にかなりの余裕があるので、引き続きこちらでの参加者募集を行いたいと思います。

□開催場所

サバイバルゲームフィールド BB-JUNGLE　ジャングルフィールド
http://www.sky.sannet.ne.jp/pintail/bb-jungle/bbjungle.htm
千葉県千葉市緑区高田町1586

□日時

2014年6月14日(土)　09：00時～17：00時
※AM09:00～ 受付開始　ゲーム開始は10：00時より

□費用

フィールド使用料：一人2500円
※別途700円でお弁当を注文できます。
　こちらは10:00時までの受付となりますので、朝の参加受付時にお知らせ下さい。

□レギュレーション

・フル・セミオート問わず、パワーの上限は0.98J(0.2ｇBB弾で初速約98.95ｍ/ｓ)
・弾数制限無し。
・使用可能なBB弾はバイオ弾のみ、プラスチック・セミバイオ弾は使用出来ません。
・減速アダプター・レーザーサイトの使用禁止。
・フィールド内ゴーグル着用，シューティンググラス可。
・ナイフアタック禁止。
・ゾンビ行為、ダメ・ゼッタイ！

□服装

ドレスコードはありませんが、森林フィールドなので動きやすく汚れてもいい服をご用意下さい。
怪我防止のため帽子や露出の低い服装に、グローブやエルボー・ニーパッド等の使用をお勧めします。

□基本ルール

10分～15分ほどのフラッグ戦とカウンター戦をメインで行いますが、カウンターを用いたロングゲーム（1時間ほど）も予定しています。
ゲーム間の休憩は約10分です、スムーズな進行にご協力願います。

□交通

東金有料道路、高田インターを出て１分ほどです。
現地集合・現地解散で送迎バスはありませんので、レンタカー・自家用車等乗り合わせの上お越しください。

□レンタルをご希望の方へ

装備品等レンタルが可能ですが、事前予約制のため＜＜フィールド確認中＞＞日までにメールにてお知らせ下さい。
販売のものに関しては当日でも大丈夫です。
料金は下記の通りです。

□その他の注意

・フィールド内での火気使用厳禁
・フィールド内障害物の移動禁止
・セイフティ（座席等）からフィールドへの発砲（シューティングレンジをご利用下さい）
・近くのコンビニがちょっと遠いので、飲み物や食料等購入をご検討の方は高速に乗る前かSAでの調達をお勧めします。

□雨天の場合

当日の朝、06:00時にフィールド側と相談の上ご連絡します。
天候次第では前日判断の可能性もあります。

□申し込み締め切り

開催1週間前、6月8日（日）の23:59時までにATNDへの参加表明をお願いします。
グループで参加の皆様は、お手数ですが代表者の方でメンバーの人数等まとめた上でご連絡頂ますようお願い致します。
またこの際、レンタル品の有無も合わせてご連絡頂ますようお願いします。
