ゆるーく、らくーな感じでお集まりください。
皆でアプリ作ったり、センサー組み込んだり、ハンダ付けしたり、新しい端末自慢したり、皆で改良したり、行く末を案じたり、Androidについて雑談したり、軽くお酒を飲みながらワイワイする場になればいいなと考えています。

Androidに興味ある人なら開発者、ユーザー、ビジネスマン、起業家だれでも参加OKです。

参加費500円です。飲み物やおつまみ代になります。
途中参加、途中退室も可です。お気軽にどうぞ。

※今回から、Androidのソースを読む “Source code reading”も会場の一部を使って同時に実施します。ReadingとNightと、お好きなほうに参加ください。

※前回vol.8の雰囲気はこんな感じでした。
