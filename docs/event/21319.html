経堂の癒し系もんちゃん@mont_happyといやしい系かおる@kaorun67の「メガネ食堂」、
第3回は1/2のカオル宅でちっさくで開催します。

国際薬膳師でもあるもんちゃんの「薬膳」の知恵と
かおるがちょっとはまってる「発酵」の知恵をプラスして
旬を生かした身近な食材を使いつつ
ココロもカラダもほっとするごはんをご用意します♪

+

暑い夏が過ぎてから急に冷え込んだりまたまた暑さが戻ったり
このところの気候の変化はけっこうカラダに堪えます。
とはいえ実りの秋！おいしいモノがたっくさん。
旬の食材をふんだんに使ったメニューを予定しています。
インフルエンザや風邪が流行り出すこの季節に解毒作用のある食材、
また乾燥しやすい秋は呼吸器系疾患が多いので、肺によい食材も。

薬膳と発酵のチカラでもっと元気に！

　+++
