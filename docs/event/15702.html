イベント概要

今回のつ部ミーティングは、 Ogaki Developer Day 2011 Spring と合同で開催します。

つ部としては、KDDI様よりAndroid 3.0（Honeycomb）搭載タブレット「Xoom」をお借りしての、Xoomタッチ＆トライを行います。（現在、10台ほどお持ちいただけるとのことです！）

これを機会にAndroid 3.0に挑戦してみませんか？
