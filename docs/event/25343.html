description

個人で19インチラックを使いこなしてみよう。という勉強会。

  • ついカッとなって中古のDHCPアプライアンス製品を買った。折角だから誰かに構築してもらおうか。
    そんな自宅ラッカーを応援するための小規模イベントです。
  • 冗長構成のDHCPサーバを実際に設定するための、初級者向けハンズオン勉強会となります。
  • 前回の「自宅ラック　配線勉強会」より、次回開催のネタ話がいくつか出ましたので、
    先陣切ってイベントとして開催することにしました。

やること

  • NetWyvern DHCP 2台構成の構築、既存ルータへのコンフィグ追加
  • パケットキャプチャ(Wireshark)
  • DHCPサーバを攻撃し、サービス不能状態にする(Metasploit,Letdownなどを予定)
  • 冗長化の効果確認

会場および集合場所

  • 個人住宅(マンション)が会場となります。JR馬橋駅 改札前に集合。
  • 集まり次第、タクシーで会場まで移動します。(割り勘でお願いします)

※白いニット帽をかぶった人が目印です

注意点

  • 開催者には技術力がありません。その代わり周りの皆様が助けてくれるような気がします。
  • 暖房が若干弱いかもしれません。各種機器の廃熱も期待できません。
  • 轟音どころか音が物足りないと思いますが、暖かく見守ってください。
  • 上級者の参加は可ですが、作業は経験の浅い方にお譲りいただき、盛り上げ役に徹していただけますと幸いです。
  • おやつはレアチーズプリンとなる予定です。(変更になる可能性有)
  • 自宅にラックが無くてもご参加いただけます。(勧誘されるとは思いますが)これから勉強してみたい方、是非いかがでしょうか。
