講演概要

デベロッパー＆プラットフォーム統括本部
第一クライアントプラットフォーム推進部 部長

大学卒業後、外資系メインフレームメーカーに SE として入社し、製造業（自動車）のお客様を担当。その後マイクロソフトに入社し、技術サポート、コンサルティングなど技術系業務を経験し、いったんマイクロソフトを退社。その後、縁があり、マイクロソフトに復職し、ＭＶＰリードを担当。2005年からエバンジェリストを統括する管理職となり、現在はクライアントプラットフォーム、主にWindows 8, Windows Phoneを担当するエバンジェリストを統括している。
