名古屋版「HTML5とか勉強会」を9月15日（土）に開催！

HTML5をはじめとするWeb技術が盛り上がると同時に、Web技術者同⼠の交流や情報交換も活発に行われるようになりました。
東京で「HTML5とか勉強会」を開催する html5j.org は「つながる、学べる、盛り上がる」をテーマにWeb技術者の知識向上と交流の場を作ってきましたが、この度、全国各地でもWeb技術者同士が交流や情報交換ができる勉強会をスタートします。

第1回は名古屋で開催。
Web技術はもちろん、現場動向やワークスタイルもテーマにした4セッション＋パネルディスカッションで盛り上がります！

★ セッション内容

【セッション1】自社サービス×ソーシャル×モバイルの最前線を支える技術と挑戦のやりがい
昨今急激に普及が進んでいるスマートフォンにおけるWeb画面で、先進的なUIUXを実現するエンジニアリングの重要性とHTML5+CSS3+JSの役割についてご紹介します。
その題材として、DeNAのモバイルに特化したブラウザベースのソーシャルゲームを取り上げ、その最先端の取り組みや考え方をご紹介します。
発表者：DeNA　満尾有希さん

【セッション2】“まぜたら危険”なJS/CSS！一歩引いた視点のコーディングの考え方
開発者にも企画・クリエイティブなど幅広いフェーズで活躍が求められる今、開発者として最新技術トレンドを追うことも大切ですが、企画やビジネスの全体を捉えてチームに提言できることも大切です。
セッション2では、現場の状況を振り返りながら「コードを書けるだけの開発者」から「一歩引いた視点で企画もみれる開発者」になる方法を考えます。
発表者：一條美和子さん
