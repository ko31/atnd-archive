catch 業務モデリング(業務プロセスの可視化)を同時に行う組織診断
description

・使いモノにならない“組織診断結果”はもう要らない。
・結果に具体的な解決策を求めることはもちろん、結果だけでも十分、対価に値するモノが欲しい！
・組織の生産性を高めたい、学習する組織へと変革していきたい

企業の変革を「業務プロセス」と「組織」の軸でご支援するカレンコンサルティングが作り上げた新しい組織診断プログラムが『Organization Viewer』です。

【タテだけではなく、ヨコも直接診断する日本初のプログラムです】
　⇒タテ方向(経営・管理職・現場)＝経営・会社/上司・部下に関する事項
　⇒ヨコ方向(部門間)＝業務プロセス上の問題、部門間の連携・関係性(セクショナリズム等)に関する事項

【圧倒的なアウトプット！】
「組織診断結果」に加えて、部門の「業務フロー」と「業務定義」も出来上がります。

セミナーは、ピーター・センゲ(MIT)の"学習する組織"等のアカデミックな見地と、業務の可視化（モデリング）から得られるコミュニケーション効果をわかりやすくお伝えいたします。

■こんな方にオススメします！
・「個」と「組織」 両方のパフォーマンスを高めたい
・現場より、経営・組織・業務等の課題を拾い上げたい
・組織と業務プロセスの整合性を調べたい
・組織編制と同時に現場の効率化、生産性を高めたい
・部門間の連携をもっとはかりたい 等

■当日のスケジュール(予定)
