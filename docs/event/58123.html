CMSインサイドは「OSSコンソーシアム CMSビジネス部会」が運営する、CMSの機能を徹底比較する無料セミナーです。WEB制作者／担当者の方がCMSの選定をする際の参考資料としていただく事を目的としています。CMS導入をお考えの企業、大学、システム開発会社様など、どなたでも参加していただけます。各回で取り扱うCMSは一種類で、各CMSを代表する方に2時間半の講習をお願いしています。

　※新しいお知らせはFacebookページをご覧ください。

CMSインサイド 番外編は11/07 (金)16:00～18:30の開催

今回は番外編として、商用CMSを取り上げます。

いつでもインストールして使ってみることができるOSSと違い、使い込む機会が少ない商用CMS、しかも大規模サイトになればなるほど求められる「静的」出力！
オープンソースCMSでビジネスをしている方には必見のセミナーです。

オープンソースCMSも商用CMSも構築経験をお持ちの言問株式会社の藤田拓様に、実務を踏まえた貴重な内容をお話いただきます。

【講師よりのコメント】

WebCMS案件のRFPを見ると「静的なHTMLファイルの生成」といったような項目を目にすることがあります。
日本ではCMSに求められる要件として「静的」というキーワードはとても身近なものです。
そんな静的CMSとしての代表格はMovable TypeとNORENでしょう。

また、海外では動的CMSが多く、静的CMSはあまりなかったのですが、近頃Webエンジニアに人気なのがStatic Website Generatorという静的CMSです。
これらの静的CMSにフォーカスを当て、それぞれのメリット・デメリット、かつ、今後の展望をお話します。
