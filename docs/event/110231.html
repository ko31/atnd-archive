【第1回ポケモン剣盾オフ会詳細】
■日時
2019/12/8（土）13:00～18:00
■場所
仙台駅周辺
■参加者定員
14名+運営2名
※先着順となります。予めご了承ください。
※「プレイしないで見学だけ…」もOKですが、通常入店になります。
※当日キャンセルや遅刻がある場合は、見学者から立候補を募る場合があります。
■参加費
1,500円
（一部アルコール、ソフトドリンク、月変わりフード食べ飲み放題付き）
■参加条件
・ポケットモンスター　ソード・シールドのいずれかをお持ちの方
・みんなで楽しくプレイできる方(一緒にプレイする方の批判は禁止)
■イベント内容
《前半戦》ポケモンバトルトーナメント
・みんなでLet’sポケモンバトル！トーナメント形式でチャンピオンを決めましょう！
・ストーリーとポケモンの育成が全く進んでいない！という方でもポケモンをレンタルして対戦ができる機能がございますのでご安心ください！レンタルポケモンについては当日こちらで指定致します！

～ルール～
・バトル形式：シングルバトル・バトルタワールール
・使用できるポケモン：ポケットモンスター　ソード・シールドで手に入るポケモン
※ガラル図鑑No.398、No.399、No.400のポケモンは禁止とします。
・対戦に使用するポケモン：レベル1～100のポケモン3～6匹
※レベル51以上のポケモンは、対戦時に自動的にレベル50になります
※レベル49以下のポケモンは、そのままのレベルで対戦に使用されますのでご注意ください
・自分の手持ちに同じポケモン、同じ道具の使用不可

《後半戦》レイドバトル周回
・みんなで協力してレイドボスを倒し、強力なポケモンや夢特性を持つポケモンをゲットしましょう！

■その他
・質問等があれば主催者「@siro_spla2」までご連絡下さい。
・会場について詳細を知りたい方はTwitterのDMを用いて連絡致します。
→ 主催者「@siro_spla2」をフォローしDM受入許可をお願い致します。
