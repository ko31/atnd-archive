インターン参加を迷っているあなたへ。経験者から色々聞いてみよう！

「今年の夏、インターンシップに参加しようか迷っている。」
「インターンに興味はあるけど、不安も大きい。」
「自分にはどの企業が合っているのか、分からない。」
そんなあなたに、オススメのイベントです！

今まで経験したことのない、1ヶ月以上のインターンシップ。
まずはインターン経験者とじっくり話して、不安を解消してみませんか？

インターンOBOGとの交流会とは？

インターンシップのこと、会社のこと、仕事のこと、そして大学生活のこと。
手づくりのご飯を一緒に食べながら、一足先にインターンを始めている先輩とワイワイ語り合う場です。

インターンシップフェアでは聞けなかった、あんなことやこんなことを
コーディネーターと一緒にズバッと質問してみましょう！

もちろん、あなた自身のことも存分に語ってくださいね。
