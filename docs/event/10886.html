今年の冬至もやりますよ！

「西伊豆キャンドルナイト」

賀茂村が西伊豆町と合併するに当たり、
何か恩返しできないかと、私たち
西伊豆町在住ガラス作家の会が
考え出したイベントです。

細々と、口コミで（笑）続けてもう7回目になります。

各作家が、オリジナルキャンドルホルダーを制作し、
披露するという、全国でも珍しいイベントです。

翌日は天皇誕生日で、お休みですよ！

お泊りで、ぜひぜひ西伊豆へお越しください。

例年通り、黄金崎クリスタルパークさんの
駐車場横の芝生広場で予定しています。

あの！紫苑さんのパンやお菓子もちろん！もやってきます。

16時から20時ごろまでやりますので、
皆さん西伊豆の夕陽を堪能してから
ぜひお立ち寄りくださいね。

クリスタルパークさん企画展示室内では
「西伊豆ガラス作家展」が、12月18日から開催されます。

キャンドルホルダーだけでなく、各作家の
今の作品を、この機会に合わせてご覧ください。

「西伊豆キャンドルナイト」

12月22日（水）　翌日は祝日ですよ〜
16時から20時まで
黄金崎クリスタルパークさん屋外にて
