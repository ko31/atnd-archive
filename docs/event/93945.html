SNSで集客のため動画を配信したい！
スタジオで撮影をしたい！
無料で映画のようにカラー調整をしてみたい！
撮りためたこどもの動画を音楽とテロップを入れて作品にしたい！

ダビンチ・レゾルブ編集ソフトが使える基礎コースです。
https://www.blackmagicdesign.com/jp/products/davinciresolve/
この機会にご参加ください。

第一部:スタジオ撮影
午前会場：9:30-12:00
すたじお金魚　2階スタジオ (​池袋）
東京都豊島区東池袋3-13-11
池袋駅から徒歩8分
※開場9:15、時間差で1​階スタジオも使います。

スタジオ撮影ではドールをモデルに撮影をします。
手持ちの一眼レフカメラやスマホで手ブレが少なくなる、
滑らかな映像を撮る方法もアドバイスします。

第二部:ダビンチ・レゾルブ編集
午後会場：13:00〜16:30
サニーハイム池袋 405会議室
東京都豊島区東池袋1-47-2 4階
※すたじお金魚から徒歩3分、開場12:10

ダビンチ・レゾルブ編集セミナーでは、編集機能を中心に
「読込み設定、サクサク編集、カラー調整、書き出し」の
一連の編集作業を習得できるコースとなっています。

通常 15,000円のところ2018年スタートアップ特別価格10,000円（税込み)

定員：８名

■■■■■　受講へのご準備■■■■■■
以下が参加条件です。
申し込み前の事前確認のためソフトをインストールしてください。

（１）ノートパソコン
メモリ16GB以上実装のパソコン。
※練習素材があるためPCに20GBの空き容量をご用意ください。
Mac： OS 10.10以降を推奨。 MacBook Pro 15inch Retina モデル等
Windows： Windows 8.1 64bit以降

（２）カメラ
一眼レフカメラを推奨しますが、コンパクトデジカメやスマホでも参加できます。
パソコンに写真・動画ファイルを移動できることを確認ください。

（３）インストール後の動画確認
無料の画像編集ソフトDaVinci Resolve14をインストールし、初期画面の表示を確認ください。
