Force.com開発者も盛り上がってきてるのになぜコミュニティが無いんだ！？ということで、みんなで集まってMeetupしてみましょうというのが発端です!!

Salesforce.comの提供するCRMの基盤部分を切り出した形で2007年に発表されたForce.comですが、いわゆる十八番であるコーディングレスでの画面自動生成やワークフロー、数式項目などは有名ですが、意外にもApexやVisualforceを利用すればゴリゴリにカスタマイズでき、コードアーティスト達の活躍の場は多いのです。

「まだForce.comをガチで使ったことはないけど興味はある…」という方の参加も大歓迎です。

今回が記念すべき初回ということで、初代リーダーの @yonet77 さんのご挨拶＆壮大なろくろ回し（？）から、 @mitsuhiro さんによるSummer ’12イケてる新機能の紹介と、発起メンバーのLT大会を予定しています。

※もちろん我こそはと発表してくださる方を絶賛募集中ですので、何かネタのある方はコメント欄にて「発表タイトル」を記入してくださると嬉しいです。

また、ここ最近でFacebookグループを作成しました。こちらもよろしくお願いします！
