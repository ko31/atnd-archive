◇「グロービッシュ」とは？
グロービッシュ（Globish）とはグローバルに使える共通語を目指してフランス人のジャン・ポール・ネリエールが提唱した、国際英語の一種です。
単純な英文法と、英語の中でも特に使用頻度の高い英単語1500語のみを使ってコミュニケーションを取るのが特徴で、日本人をはじめとする英語を母国語としない非ネイティブの人々のために国際共通語として最適化された英語といえます。
http://globish-j.com/index-j.html

◇「侍グロービッシュ！」とは？
「侍」に日本の小さな会社（Startup）・起業家・社会人・学生が坂本竜馬のように大志をもって世界を渡っていくという意味を込め、日本の侍（日本人）むけに英語とうい刀（武器）を与え、日本人としてブロークンでも自分の伝えたいことを刀一本で伝えていこう、というメッセージが「侍グロービッシュ！」というブランドに込められた思いです。

「え～と英語苦手なんですけど・・・」な方から
「TOEIC 高得点だしかなり出来るけど～」な方まで
「あっこうすれば英語は簡単かも」という気分にさせてしまいます。
