description

「あの人の研究所」では，技術を無駄遣いすることによって情報科学研究とコンテンツの分野の双方に新しい刺激を与えるような研究をまとめて，｢あの人の研究論文集」を発行しています．
（参考 http://www3.kitanet.ne.jp/~narumin/anohitonoronbun_2011w.htm ）

そのオフラインミーティングとして，3/13に勉強会＋デモを開催します．
・こんなもの作ったんだけどどうでしょうか
・こういう技術動向があるけど，こういう分野に応用できないでしょうか
というアイデアをフランクに発表(一人五分程度)しあう勉強会と，作ったものを見せ合うデモです．

発表者と参加者を募集しています．
発表したい方はコメントで書き込んでいただけると助かります．
参加者が10人集まらなかったら中止します！

→当日0時時点で参加者が10人集まらなかったので，勉強会はなしでデモだけになりました．かむたろー氏，樋口啓太氏，櫻井翔氏，前田智祐氏によるデモは行われますので，ふるってご参加ください．

■時間割
11:00～12:00 あの論勉強会
13:00～14:00 あの論デモ

■デモ
・かむたろー，おっぱいみるくの重畳表示によるユニバーサル搾乳プレイ
・樋口啓太，1分に1000枚の画像を見せる走馬燈システム
・櫻井翔，日記マンガ
・前田智祐， ECB君
・とたっけ，未来日記
