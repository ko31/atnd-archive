catch 最優秀者には、夢をカタチにする資金（最大2000万円）サポートが決定！
description

NPO法人みんなの夢をかなえる会が主催し、昨年2010年に始まった「みんなの夢アワード」。たくさん の夢で社会を明るくするため、今年も開催します。最終選考委員は、ワタミグループ会長・わたなべ美 樹。社会を元気にするパワフルな夢にアワードを贈り、夢の実現をわたなべ美樹が“応援”します。
この度、その“応援”の一環として、最優秀者には夢をカタチにする資金(最大2000万円)をサポートす ることが、決定しました。(提供:ワタミグループ)。受賞者により提出された事業計画/面接などにより、 主催者と出資者が協議のうえ、金額を最終決定します。
エントリーの受付は11月15日(火)まで。この機会にあなたの夢をカタチにしてみませんか？

■みんなの夢アワード2011全体概要・流れ
　
当会が企画運営を行う“５年先の夢を名乗って、未来の名刺を作ろう”というコンセプトのホームページ『未来の名刺』【http://www.miraimeishi.net/ 】において、エントリーされた夢の中から、一次選考にて選ばれた夢の持ち主に二次選考として、「未来の名刺交換会」にてプレゼンテーションを行っていただきます。さらに、会期の最後には中野サンプラザにて、わたなべ美樹を選考委員として最終選考会を開催し、みんなが夢をもちたくなるすてきな夢に「みんなの夢アワード」を贈ります！

１．エントリー　2011年10月1日（土）〜11月15日（火）
専用フォームからエントリーを受け付けます。【http://www.miraimeishi.net/award/ 】

２．一次選考　2011年11月16日（木）～11月24日（木）　発表 2011年11月25日（金）
投稿された内容を元に、事務局により一次選考通過者（8～10名程度を予定）を決定、サイトにて発表します。【 http://www.miraimeishi.net/award/ 】

３．ニ次選考 in 未来の名刺交換会 2011年12月5日（月）
「未来の名刺交換会」の中で、一次選考通過者によるプレゼンを実施、二次選考委員により3～5名が最終選考へと進みます。（一次選考通過者は、二次選考前に、より詳細な夢のプランなどについてレポートをご提出いただきます）

４．最終選考＠中野サンプラザ 2012年1月30日（月）
最終選考では、二次選考通過者によるプレゼンを実施、最終選考委員によりアワード受賞者を決定します。当日の様子は、Ustreamでの中継も予定しています。
