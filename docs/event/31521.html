12:00 – 18:00 質問したり作業したり。 途中参加、途中退出も、ご自由に。

概要

Jenkinsを実践で使うための勘所を押さえていくことを目的とした勉強会を開催しております。
詳しくは、以下を参照してください。
SwimmyStudy #10 Jenkins実践入門 vol.4

こちらは途中参加を検討されている方、今までの参加の中でわからなかったことがある方向けのフォローアップイベントです。とりあえず、話だけでも聞いてみたいという方も、お気軽にご参加ください。

内容

Vol.3までにやったこと、Vol.4までにやるべきことの確認＆作業
Jenkins実践入門 vol.3 リポート

その他、
LT資料を作るもよし
Jenkins + PHP環境をセットアップするもよし
Git, GitHub, git-flow, CakePHP 等の環境をセットアップするもよし
作業をしながら質問する機会としても使ってください

参考になるサイト

Jenkins + PHP環境のセットアップしたい → PHPでTDD＆CIワークショップ、Jenkins + PHP の各種プラグインパート資料
Jenkins + CakePHP環境をセットアップしたい → CakePHP2.0+Jenkinsで継続的インテグレーションを行う方法
Gitについて知りたい → Git Pro
GitHubについて詳しく知りたい → WEB+DB PRESS Vol.69
git-flowについて知りたい → ゼロからわかるgit-flow

対象者

  • Jenkinsや継続的インテグレーションを知らない方・興味のある方
  • Jenkinsは知っているが、実プロジェクトでどのように使えばいいかわからない方
  • 技術の話をできる仲間・環境を渇望している方

基本的には特定の講師や発表者がいるような勉強会ではありません。皆が参加者であり発表者です。

持ってくるもの・準備してくるもの

  • 開発が行えるPC
  • 書籍「Jenkins実践入門」 （必須ではありません）
