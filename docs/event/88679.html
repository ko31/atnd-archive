description

篆刻は中国を起源としてその歴史は古く、今回は外部講師の方をお招きして、篆刻の歴史を学び、実際に彫ることも行います。彫った作品は記念に持ち帰りいただけます。少なめの定員となっておりますのでお早めにお申し込みください！

【篆刻教室体験内容】
■目的
篆刻文化を知ろう
篆刻の製作体験をしよう

■内容
篆刻の歴史（中国史）、現代中国の篆刻と日本の篆刻・交流、鑑賞
篆刻制作の説明　道具、手順など
篆刻制作　文字選び、印のデッサン、印面に文字を書く、刻す、捺す、作品化

■最終形態
印を押した作品と彫った石とは箱に入れて持ち帰っていただけます篆刻 (3)

■道具（講師の先生にご用意いただきます）
新聞紙・下敷き・小筆・硯・半紙
印刀・印箱・印材（石）・印泥（朱肉の様なもの）
印床（刻す際に石を抑え持ちやすく又、怪我防止をするための道具）
印矩（印をまっすぐに捺すための道具）

■講師紹介
益山浩子先生
書道歴20年、書道師範
大学・大学院(書道専攻)
カルチャースクール・高校書道講師
読売書法展入選
