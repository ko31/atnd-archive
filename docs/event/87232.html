description 今までした中で一番大きいサイズはどれくらいですか？ 元彼で、20cmくらいです。ちゃんとは計っていないけれど、両手で息子を握っても先っぽが出ちゃうくらいです。 ぶっちゃけ、今もたまに連絡取っちゃうくらいで、別れてからもどうしても思い出しちゃって、多分、この先も一生、あの馬並みの肉棒の感触が私は今も忘れられません。 反対に一番小さかったのは？どれくらいの大きさ？ 二つ前の彼氏です。多分、大きくなっても10cmもないくらいでした。 正直、顔はかなりのイケメンだったけれど別れたのは、シアリス 通販結局は下半身の相性で、最終的にはアソコが本当に歴代の彼氏と比較しても小さかったことが一番の嫌だった本当の理由なのかも。 理想のペニスの長さや太さは？ あまりに長すぎると奥に当たって痛いので、15cmくらいあれば嬉しいです。 太さはうーん…これくらい！（そういって自分の手首を握って見せてくれました。） 会社の同僚や女の子同士で飲み会をしている時もこの話で盛り上がるけど、年齢と共にイケメンよりも大きそうなエロそうな人を選ぶ割合が多くなっているかも。 ほとんどの女の子が舐める時に、ちんちんの形とか硬さとか太さとかも絶対に確認して歴代の彼氏と比較してると思います。 友達の中には、大きさのランキングを作っている女の子も本当にいましたよ！ やっぱり男の人のペニスのサイズは気になりますか？ 関係ないっていう女の子は、正直、私は嘘つきだと思います（笑）。 小さくて可愛いなんてことも絶対にないです！恋人とはSEXで満足できなければ私は長くは付き合えません。 男性がペニス増大サプリメントを飲むことについてはどう思いますか？ 女の子もお化粧したりスタイルを維持するために体操したり最新の流行のトレーニングをしたりしているのだから、男の子にもチンチンが根元から大きく太くなれるように本当はもう少し頑張って欲しいかも。

今までした中で一番大きいサイズはどれくらいですか？ どれくらいなんだろ？大きすぎて痛くて入らないくらい…（笑）口でするのも苦しくて嫌だった…余裕で22、23cmくらいの長さは軽くありそう。 反対に一番小さかったのは？どれくらいの大きさ？ 初めて高校生にした相手で6cm位だと思います。 初めては根元まで挿入すると痛いって聞いてたけど、そんなことなくて、しかも、全く気持ちよくなくてびっくりしました。 二人目の彼氏の平常時から男性器が勃起したのを見た時、本当の男を知りました。 理想のペニスの長さや太さは？ 下半身が大きすぎるのは困るけど小さすぎる方が絶対嫌です。 やっぱり挿入された時にゴリゴリ当たらないと中で私はイケないかも。 やっぱり男の人の威哥王ペニスのサイズは気になりますか？ 関係ないっていう女の子は、正直嘘つきだと思います（笑）SEXで満足できなければ私は長くは付き合えません。 昔、女の子のグループの間で出回っていた、男友達の中で息子を比較して「誰が一番アソコが大きそう？」のランキングが出回ったんだけれど、これって良く考えると実際に本当に周りでモテてる男の順位とほとんど一緒だったし（笑） 男性がペニス増大サプリメントを飲むことについてはどう思いますか？ 体に悪くないんですか？別に副作用とかないなら、すぐにでもペニスが太くなるサプリをガンガン飲んで欲しいかも。 彼氏とか結婚する人が小さいのは…申し訳ないけど、一回でも大きい肉棒を味わった事ある女の子は浮気しちゃうんじゃない？
