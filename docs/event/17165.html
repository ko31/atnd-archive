Korea*Japan Developer's Summit 2011

description

4月のスタートアップカンファレンスに引き続き、
デベロッパー向けイベントを韓国にて開催します！！

本サミットでは、韓国と日本のiPhone/Androidなどのスマートフォンアプリを開発するデベロッパーが集います。双方のアプリマーケットやプロモーションの特徴についてプレゼンテーションを行い、さらに韓日の相互展開についてパネルディスカッションを行います。

日時：2011年6月28日(火)　19:00-21:00
場所：韓国ソウル Success Shop
費用：無料　※懇親会は実費
人数：80人程度

※日本語-韓国語の相互通訳が入ります。
※韓国語でのイベント概要についてはこちらを御覧ください。
