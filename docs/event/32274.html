title 【無料】マネーセミナー
catch 公的年金基礎知識＆将来を見据えた保険加入
description

老後生活費の柱である『公的年金』。
自助努力としての『生命保険』。

どちらも必要性を感じて加入しているものの、実は多くの方が内容を理解していないのではないでしょうか。

セミナーでは公的年金の基礎知識と、将来を見据えた賢い保険の入り方をわかりやすくお伝えします。

この機会に年金や保険に対するモヤモヤをすっきり解消しませんか？
基本を押さえておくときっと今後の人生のお役に立ちますよ！

知識を身につけて豊かなセカンドライフを実現しましょう。

セミナーでは公的年金の基礎知識や、長期視点で見た生命保険の入り方をわかり易くお伝えします。

・公的年金の仕組み
・将来いくら年金がもらえるのか
・必要な自助努力について
・生命保険の保障期間考え方
・ムダの無い生命保険の入り方
・加入方法による保険料の違い
・貯蓄型保険と掛捨型保険の特徴
など

講師は銀行・保険会社・保険代理店で働いてきたファイナンシャルプランナー。
長年金融業界で働く中で『金融知識がある／ないで大きな違いが出る』ことを体感してきました。

公的年金・生命保険も重要な金融知識です。
このセミナーに参加すると、基本を身につけることができますよ。

参加者の理解を深めていただくため、セミナーは毎回少人数制。アットホームな雰囲気で行っています。
金融商品の勧誘・販売は一切行いません。
お友達を誘ってお気軽にご参加ください。

この機会に年金・保険の悩みを解決しましょう！

セミナー後、更に詳しい相談をご希望の方に、無料個別相談を実施しています。（希望者のみ。後日予約制）
・年金定期便の見方がよくわからない
・いま入っている保険の内容を詳しく知りたい
・ライフプランシュミレーションをしてほしい
などのお悩みにお答えします。
