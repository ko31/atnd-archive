【注意】筑波大学生限定企画です。

企画概要

6週間という限られた期間でゲームの企画・制作・発表を行う.

【詳細】

・２～４人程度のチームで１つのゲームを制作する
・使用ツールは自由（RPGツクールを使おうが０から作ろうが自由にできる）
・制作期間は６週間とし、締め切りの延長は一切認めない
・チーム内での仕事分担はチーム内で話し合って決める
・１チームはできるだけ別々のサークルの人で構成する（他団体交流）
・チームが１年生のみに偏ることは避ける
・６週間後に最終発表としてプレゼン＋デモを行う
・優秀作品は同意の下,夏コミで配布する可能性がある

企画の目的

６週間という短い期間だが,実際にプロジェクトを企画から完成まで経験することができる.これにより,プロジェクトを円滑に回すにはどうしたら良いか（タスク管理能力）やどうすれば効率良く制作ができるか（技術力）の向上が期待できる.
またゲーム作りをしてみたかったけど「プログラムはできるけど絵が描けない」「絵は描けるけどプログラムが書けない」という理由により今までゲーム制作ができなかった人達が互いに手を組むことでこの問題を解消できると考えられる。

備考

・何か質問があれば下にあるコメント欄によろしくお願いします。
・もちろんのことですが、参加は筑波大学生限定でお願いいたします。
・プログラミングも絵も描けないけど、ゲーム作りに参加してみたいという方でももちろん歓迎します。

チーム決めについて

場所：18:30に３C棟ラウンジに集合してください。（確定しました@5/14）
・一年生同士で組ませることはしないようにします。
（ゲーム製作において詰むことを避けるため。一年生には上級生をともにチームに組み込んで上級生が一年生を支える感じで行きたいと思います。）
