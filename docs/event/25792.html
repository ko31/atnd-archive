GPS/GigaPan講習会 by MAPconcierge

概要

毎月開催しているハンディGPSやパノラマ機材などフィールド機材の講習会です。（ショップでご購入いただいた方は参加費無料です）。懇親会ではNPOオープンコンシェルジュのメンバーと合流してミニFOSS4Gのようにオープンソースジオツール関連のライトニングトーク大会を予定しております。


  • 日時： 2012年3月31日(土) 14:00〜21:00

  • 場所： 東京・西麻布 NOMAD NEW’S CAFE, Lat,Lon: 35.660988,139.72425

  • 会員参加費： 無料（オンラインショプGPSconcierge）

  • 一般参加費： 2,000円（当日現金支払い）、懇親会のみは無料

  • 懇親会（兼NPOオープンコンシェルジュ勉強会）： 2,000円＋１品持ち寄り。（手ぶらの場合は3000円）

  • 後援：一般社団法人オープンストリートマップ・ファンデーション・ジャパン／OSGeo財団日本支部

詳細スケジュール

GPS講習会： 14:00〜17:00
最新機種 GARMIN eTrex 実機を使った操作方法と、
取得したデータ管理、オープンストリートマップへのデータ入力など一連の作業を説明。
新型UAVのデモンストレーションも行う予定です。

GigaPan講習会： 17:00〜19:00
パノラマ撮影 GigaPanの実機を使いながら、実際にパノラマ撮影・撮影後のつなぎあわせ処理、GigaPan.comへのアップロード等一連の指導をいたします。

懇親会： 19:00〜21:00
NPOオープンコンシェルジュメンバーと合流して、昨今のオープンソースジオツール事情やライトニングトークをしながら、楽しく皆さんと情報交換しましょう。
