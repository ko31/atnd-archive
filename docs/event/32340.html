Minami.rb＋Okinawa.rb＋Shibuya.rb共催の
RailsによるHackathonを開催します！

Rails初心者の方もベテランの方も、一緒にRailsで遊びませんか？

フットワークの軽いRailsだからこそ、
１週間もあればそれなりのサイトができてしまいますよね。
初心者の方も、経験者と何か一つ作り上げることで、
次につながるものがあると思いますので、遠慮しないで参加してください。
デザイナさんの参加も大歓迎です。

そして、今回は何と言っても、最後の集合場所は沖縄！
Hackathonのあとは、おいしい沖縄料理とビールに舌鼓をうって、
Ruby/Rails談義に花を咲かせましょう。

ご質問等は、お近くのコミュニティー（Minami.rb / Okinawa.rb / Shibuya.rb）へお問い合わせください。

それでは、みなさまのご参加をお待ちしています。
