description

小中学生と親を対象にプログラミング（特に情報オリンピックを中心に）の授業を行います。

政府はコンピューターのプログラミング学習を小中学校で2020年から段階的に必修にする方針を固めた。
プログラミング能力は様々な仕事に役立つ能力だけでなく、子供にとって頭脳開発や論理的思考力を身につけるのに
役立ちます。

本コースは小中学校の生徒を対象に、キーボードのさばきや簡単な算数問題をはじめ
「情報オリンピック」コンテストや「パソコン甲子園」にチャレンジできるようにプログラミング知識を伝授します。

【一回目の内容】
・プログラミングとはなにか。
・キーボード、エディタ、コンパイラーの紹介。
・JOI（日本情報オリンピック）の紹介。
・はじめてのプログラム：コーディング→コンパイル→実行。
・C・C++標準入出力。
・ユークリッドの互除法。
・エラトラステネスの篩。
