第10回目の WordBech 長崎を開催します!

WordBench は WordPress ユーザーのための地域コミュニティです。WordBench 長崎では月1回程度で勉強会を行っています。WordPress を勉強したい方、ユーザ同士で交流したい方、情報交換したい方、お気軽にご参加ください。

何かやりたいという方、◯◯について話したいという方も大歓迎です。WordPressを中心にWebに関することやもくもく会などいろいろやっています。

  • WordBench 長崎
  • WordBench 長崎 Facebookグループ

今回の内容

今回はデザイナーとして活動されているPRISM!の中村さんを講師にお招きして、ロゴ制作をベースとしたデザインの勉強会を行います。

ロゴ制作をベースに色・書体・造形・制作に必要な要素をカテゴリー別に説明していただいたり、制作事例を交えながらお話いただけるそうです！

今回はデザイン会なので、デザイナーさんはもちろん、デザインについて知りたいプログラマーさんにもオススメです！

もし時間が余ったときはLTタイム、雑談タイムみたいな感じにしたいと思います。WordPress やデザインなどについて聞きたいこと、相談したいこと、話したいことなどお持ちよりください。
