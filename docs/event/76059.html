ビジネス交流会活用の奥義を即実践！

ヒトリダチ大学のセミナーです。

第7回目の今回は、
副学長の小林が登壇。

「アポ1回で仕事を惹き寄せる！印象を残すアポイント術」
というテーマで行います。

みなさんはアポイントをどうされていますか？

先方の事務所でのアポイント
喫茶店でのアポイント
ランチを食べながらのアポイント
おしゃれなカフェでアポイント

いろいろな場所でアポイントを取って
面談をされているかと思います。

しかし！

そのアポイントではあなた自身は
印象に残すアポイントができていますか？

実は、アポイントには
相手を惹きつけるツボとコツが存在します。

このツボとコツを知っているか
知らないかでは、
大きく成果に差が生まれてきます。

1度のアポイントが
その場限りで終わってしまっている方には
うってつけの内容となっております。

全てのアポイントは
全ての仕事につながっています。

ぜひ相手を惹きつけるアポイントの本質を
学びにいらしてください。

ご参加お待ちしております。

セミナー後は、ヒトリダチ大学の説明や今後の展望も
お伝えしていく予定です。

こちらも合わせてお楽しみに！
