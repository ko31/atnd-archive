description

テーマは「キャリアデザイン」。人生の約７割は仕事に関わって生きていきますが、そこにあなたらしいデザインを持っていますか？

このランチは、東海地区の社会人や学生で、仕事や人生で成長、成功したいと願っている人の集まりです。世界のベストセラーであり、マネジメントなどの経営書、自己啓発書などのルーツであるバイブル（聖書）からヒントをつかんで、自分らしく成長•成功し、一緒に喜び合える仲間作りを目的にしています。

予定しているトピックは、以下の通りです。

12月10日（土）
「自分らしく、最高のキャリアを築こう」
社会福祉法人の事務局長とメーカーで翻訳業務に携わっている女性の２人がキャリアと就活のポイントについてシェアしてくれます。

1月21日（土）
「自分らしく、最高のキャリアを築こう」
上場企業での人事、企画室に所属している男性と大手学習塾でエリアマネジメントを担当している女性からキャリアと就活のポイントをシェアしてくれます。

2月21日（土）
「自分らしく、最高のキャリアを築こう」
クリエイティブを生かして仕事をしている広告代理店社長とファッションメーカーで企画営業している男性が成功するキャリア形成についてシェアしてくれます。

この会に参加するメリットはたくさんあります！

１、牧師や経営者からビジネスとバイブルから実用的なシェアが聴ける！
２、異業種の人たちとネットワークが出来る！
３、同世代の社会人の友達が作れる！
４、就活生は現役社会人とコンタクトを取れる！就活のアドバイスを聞ける！
５、経営者や起業家とコンタクトを取れる！
６、おいしいカクテルが飲み放題！

会場はHUB名駅店です。４０名程度は入れますが、１２月から２月は就活生応援キャンペーンとして大学生先着３０名を限定に参加費を割引します。

学生は１２００円を１０００円、社会人は１７００円！
(軽食、ノンアルコールカクテル飲み放題）
