現在開催中のウェブアプリケーション開発コンテスト Mashup Awards 6 に連動して、
今年より実践形式の開発イベント MA6 Mashup Camp (マッシュアップ・キャンプ) を開催します。

お知らせ

  • Chrome Extensions に加えて Web Apps も対象テーマになりました。
  • 事前ミーティングは終了しました。当日のみの参加者は引き続き募集しています。

Mashup Camp ってなに？

Mashup Camp は、参加者それぞれノートPC を持ち込んで、 みんなでわいわい開発しちゃおうっていうイベントです。
キャンプという名前ですが基本的に日帰りです。
MA5 では Hackathon という名前で1度だけ開催しました。

MA6では、たくさんの人に気軽に参加してほしいという思いから Mashup Camp という名前にし、定期的に開催してきました。

  • 9/9(木) 第1弾 Mashup Caravan – Maps Talks の報告
  • 9/11(土) 第2弾 Mashup Camp – Maps Day の報告
  • 9/15(水) 第3弾 Mashup Caravan – Girls Talks の報告
  • 9/18(土) 第4弾 Mashup Camp – Girls Day の報告
  • 9/22(水) 第5弾 Mashup Caravan – SocialPlatform Talks の報告
  • 9/25(土) 第6弾 Mashup Camp – SocialPlatform Day の報告

開催概要

Google Chrome をテーマに集まって Google Chrome の理解を深めながら、実際に Chrome Extensions や Web Apps を作ってみるイベントです。

本イベントをきっかけとして MA6 応募対象 API を活用した Chrome Extensions や Web Apps の作品を作って MA6 に応募しましょう。

対象者

  • Google Chrome に興味がある方
  • JavaScript でのプログラミング経験がある方
  • Tutorial: Getting Started がなんとなく理解できれば OK です
