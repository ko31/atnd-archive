スマートフォンメディア様同士の議論と交流のためのイベント
『マネタイズナイト』16回目の開催のお知らせです！

今回のテーマはこちらです。

「スマホ広告におけるマネタイズの考え方と運用事例」


スマホマネタイズは、今でもまだ成功事例は多くでてきておりません。
そんな中でも広告を中心としたマネタイズを行っているメディアも増えてきております。
現在の市場の中で、売上を最大化するためのアイデアやソリューションとして、
一体どういったものがあるのでしょうか。

また、スマホ広告における効果的な運用についても各メディアが試行錯誤するところです。
アドネットワークと純広告、あるいはPC/モバイルといったその他デバイスがある中で、
どのような運用体制で広告を管理をしているのか。

具体的なアイデアや運用事例を基に皆様で議論をしていきたいと思っております。

今回もゲストスピーカーの方よりお話を頂きつつも、ご参加の皆様で
あらゆる可能性を追求し、答えにつなげていきたいと思っております。

本イベントはいわゆるセミナーではなく、参加者の皆様にも積極的に議論に参加して頂く
座談会的スタイルで実施しています。会場はカジュアルな雰囲気となっております。
また途中、歓談の時間も充分にご用意しております。

メディアとメディアを繋ぐ交流会スタイルとして場のご提供をさせていただいております。、
運営者様同士の情報交換の場としても活用ください。

※参加資格
　メディア運営者（サイト/アプリ）に限り参加対象とさせていただきます。
