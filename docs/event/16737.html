description

3Dポリゴンを表示させるライブラリのOpenGLを使って情報交換しながら開発しましょう。未経験者・初心者でも歓迎します。３Dモデリングができる人も歓迎します。

・場所
一度、松山市総合コミュニティーセンターに集合してから主催者の家に行きます。
松山市総合コミュニティーセンターの中のカフェレストラン北斗の隣に休憩スペースがあるので、そこに集合して下さい。
★追記★
eyemovicの社長が会社の会議室を使ってもいいと言って下さったので、人数が多い場合は会議室に行きます。

・参加費
無料（主催者の家の場合）
※ 参加者が多く、他の会場を借りる場合は、会場費用を割り勘で負担にします。

・必要なもの
ノートPC（有線LAN、無線LANは家にあります）

・あったほうがよいもの
開発本

・追記
プラットフォームはWindowsでもAndroidでも対応できます。Macについては勉強中です。あと、WebGLも勉強中です。
３Dモデリングをこれからやってみたいという方は、フリーツールでMetasequoiaやblenderや六角大王などがあります。Metasequoiaなら主催者が使い方を教えれます。

・連絡先
twitterで@minmin0530に連絡するか
minmin0530@gmail.comに連絡下さい。

OpenGLはiPhoneやAndroidでも使われており、PCやPlayStation系のコンシューマゲーム機でも使えます。3D描画の基本ですので、一緒に勉強しましょう。
