昨年認知症と診断された母親と日々奮闘しながら介護と仕事の両立をしています。
自分自身が疲れ切ってしまったら、介護は本当に暗く、つらいものとなってしまいます。
そんな中で出会ったのが「アロマハンドセラピー」でした。

このハンドセラピーをとても気持ちよく受けてくれます。
日頃の精神的なストレスをこのセラピーを通して心の交流が図れることが実感できました。
できるだけ多くの人にこの体験をしていただきたく思っています。
ぜひ日々の介護で精神的に疲れている方にお越しいただきたいと思っております。
