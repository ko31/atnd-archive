「結婚力」とは、結婚するだけでなく、幸せな結婚生活を持続する力です。


結婚力を上げれば、
未婚の方は、理想のパートナーを惹き付け、結婚し、
既婚の方は、夫婦仲を理想な状態に改善し、幸せな結婚生活が送れます。

■幸せな結婚の為に重要なものはたくさんありますが、最も重要なものは、「心」です。



相手を思いやる心、自分に価値を見出す心、人生の荒波を乗り越えるための心、
暖かい心、等、心を磨くことによって、人生が輝き、結婚力が上がります。

ところで、あなたは、心を磨く良い方法をご存じですか？

それが、この結婚力アップシリーズです。
ソニー(株)で研究してきた潜在意識活用方法の20年の集大成です。

今回は、自己催眠状態で、実際に内なる叡智に繋がり、心を磨きます。

■潜在意識の機能の中に「内なる叡智」というのがあります。



自己催眠状態になって、エゴからの声である雑念を抑えていくと、頭がスッキリしてきます。

すると、結婚等の悩み解決に役立つ、良いアイデアがどんどん浮かんできます。
その情報源を「内なる叡智」と呼びます。


■参加者特典



セミナー時に実習した誘導音声やセミナー動画を1ヶ月間無料で視聴出来るメンバーサイトに案内します。
キャンセルはもったいないですよ。

これまで、約1000名の方に参加して頂いているラディアンスの無料セミナーです。
参加した殆ど全ての方が、催眠状態や瞑想状態を体験していると言う、凄い実績です。

今回は、あなたに自己催眠状態になって、内なる叡智に繋がった状態を体験して頂きます。
全くの初心者でも、ベテランの方でも、着実に体験できます。

■本当の催眠状態とは



多くの人は、催眠状態とは、催眠ショーのように、意識がどこかに飛んでいてしまって、人からコントロールされやすい状態だと勘違いしています。

実際の催眠状態は、意識が非常にはっきりしていて、自分の意志でしっかりと行動できます。


■自己催眠で内なる叡智に繋がった状態のメリット



・集中力、閃き力、洞察力が良くなる
・とっても素晴らしいメンターを得られる


■自己催眠で、集中力、閃き力、洞察力が良くなるとは



通常の意識状態は、コップに泥水を入れてかき回して濁っている状態です。
向こうがよく見えません。

自己催眠状態は、コップの泥水の泥が沈殿し、透き通っている状態です。
向こうがよく見えます。

そこで、自分の気持ちや、相手の気持ちがよく分かるようになり、
どんなことを提案したら、相手が喜んでくれるかがわかります。

良いアイデアがたくさん浮かんでくるようになるので、
仕事やプライベートで起こる様々な出来事にも上手に対応できるようになります。


■自己催眠で、とっても素晴らしいメンターを得られるとは



通常の意識状態で、考えているときは、
あれにしようかなぁ、これにしようかなぁ、と迷ったり
それは今までの経験からして上手くいくはずがない、と否定的になったり
自分の業界では、常識的には、この方法をとるのだ、と頑固になったりします。

自分自身の既成概念という屁理屈を、頭が色々言ってきています。


自己催眠で、内なる叡智に繋がった状態とは、
頭が言う屁理屈を上手に静めた状態です。
これまで自分が学んできた知識や、体験してきた経験を上手に引き出すことが出来ます。

幸せな結婚をするための、とっても素晴らしいメンター（助言者）になってくれます。
