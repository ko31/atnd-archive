iPhoneで写真を撮る人の集まりiPhoneographers!のHOKKAIDOでもイベントを行おうと思います！
基本的にはフォトウォークをしたいと思うんですが第1回目ということでみんなで集まってお茶でもしながら写真の話しなどなど話せたらと思います！

もちろん写真好きなら誰でもOK！普段は一眼で撮ってるけどiPhoneで撮るのも興味ある！や興味ない人もOKです（笑）

北海道は広いのでなかなか参加できない人もいるかもしれませんが場所は札幌で行おうと思います！
もちろん他県の方も参加OKです！

参加費はお茶代と交通費のみです。

開催日は3月9日土曜日13時〜です！

天気予報で晴れマークがでていたので予定を変更して13時〜フォトウォークをしてそのあと軽くお茶でもしながら写真について話しでもしようと思います！

集合場所は札幌駅西口南側のオブジェです！

当日まで参加募集しているのでぜひぜひ参加お待ちしています！
