　2-1. 有機-無機複合化によるバリア性付与
　2-2. 金属蒸着によるバリア性付与

3. 水蒸気バリア性評価
　3-1. JIS法
　3-2. 高感度水分透過率測定法

4. デバイスを用いた実装評価(有機ELへの応用)
　4-1. デバイスの構成例
　4-2. 特性評価の具体例
　4-3. 封止材の実装評価における課題

第3部　有機EL封止材料における水分透過率の測定・評価
【14:45-16:00】

【講演主旨】
有機EL等のフレキシブル電子デバイスは、表示装置、照明、太陽電池などさまざまな用途で社会に広がりつつある。
水蒸気・酸素バリア性は、これらのデバイス寿命を決定的に支配するので、ハイバリア膜形成方法とバリア性を正確に測定する評価装置が求められている。
本講座では、ハイバリア膜の水蒸気・酸素透過率測定装置の測定原理と測定限界,ハイバリア膜の形成方法について述べる。

【プログラム】
1.水蒸気透過率測定装置のニーズと現状の技術状況

2. 10-6g/m2/day とはどのくらいの量か?

3.水蒸気透過量の主な測定法
　・差圧法
　・等圧法(カルシウム法、キャリアガス法)

4. 従来装置の問題点
(何故、従来装置で10-6g/m2/dayの水蒸気透過率を測定できなかったのか)

5. 新型ハイバリア性能測定装置の紹介

6. 測定例(他の測定法との比較)

7. バリア膜形成方法
　・バリア性能を決定する要因
　・バリア膜形成例
