微小液滴の形成パターンと均一化・高速化

title 微小液滴の形成パターンと均一化・高速化
catch ～対流、沈降、乾燥パターンをそれぞれ考察～
description

【講座の趣旨】
　微小液滴を乾燥するとブロードリングやスポークラインなどのマクロなパターンやフラクタルな多彩なミクロパターンが必ず形成される。本稿ではまず、濡れ性や乾燥過程に直接関連する界面科学の基礎知識を分かり易く詳細に解説する。ついで、乾燥過程は対流、沈降、乾燥の３過程が重複、相関しながら進行することをビデオ観察などを通して説明する。そして、対流、沈降、乾燥各過程で形成されるパターンの最新結果を述べる。最後にこれらの散逸パターンの制御について均一化の効率性と高速性の面から考察する。

　界面科学の基礎にも力点を置きつつ、ビデオ観察や多くのカラー写真を通して乾燥過程の実態を生き生きと受講者に伝えたい。

【プログラム】
１．微小液滴の濡れ性・乾燥過程と界面科学の基礎の重要性
　　1-1 界面力の重要性
　　1-2 幌馬車隊の行動と界面科学
　　1-3 微小液滴の濡れ性＜水と油＞
　　1-4 コロイド粒子はマイナスイオンに帯電する
　　1-5 電気二重層
　　1-6 微小液滴乾燥過程の直視ビデオ観察
　　1-7 微小液滴乾燥過程概観＜対流、沈降、固化乾燥過程＞
　　1-8 微小液滴は必ず乾燥パターンを発現する
　　1-9 微小液滴、液滴、大量液の乾燥過程の類似点と相違点
　　1-10 乾燥過程で大きさ、形、しなやかさなどの溶質情報が選択・増幅・伝達される！
２．微小液滴の対流パターン
　　2-1 身の周りの対流パターン
　　2-2 ポリメチルメタクリレート（PMMA）コロイド分散液液滴の対流パターン

　　2-4 コロイダルシリカ水およびエタノール分散液液滴の対流パターン
　　2-5 対流パターン形成の最新スキーム
３．微小液滴の沈降パターン

　　3-2 コロイド分散液液滴の沈降パターン

　　3-4 沈降パターン形成の最新スキーム
４．微小液滴の乾燥パターン

　　4-2 コロイド水分散液液滴の乾燥パターン
　　4-3 ゲル球水分散液液滴の乾燥パターン
　　4-4 階層的会合コロイド粒子水分散液液滴の乾燥パターン
　　4-5 コロイド有機媒体分散液液滴の乾燥パターン
　　4-6 コロイド分散液液滴乾燥過程の無重力実験
　　4-7 低分子、高分子イオン溶液液滴の乾燥パターン
　　4-8 水溶性高分子溶液液滴の乾燥パターン

　　4-13 乾燥パターン形成の最新スキーム

　　5-1 均一化をめざす
　　5-2 高速化をめざす
