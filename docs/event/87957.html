「 新入社員教育や指導を担当する 先輩社員 ＆ 上司の情報交換 交流会 」は
はじめての方、不慣れな方も参加しやすい空気作りを徹底し
特定テーマでの開催のため
参加者のみなさんには、必ず有益な気付き、発見をして頂けると思います。

 

当日は、明確なスケジュール進行と、イベント要素を持たせ
単なる名刺交換、フリーの情報交換に終始することなく
自由でフランクな雰囲気・空気感を保ちつつ、メリハリある進行を心がけ
参加頂いた方の満足を追求しています。

 

新入社員の扱いに悩んでいたり、どうすれば、うまく育ってくれるのか？
学生から社会人に意識を切り替え、仕事に集中してくれるのか？
どこまでガマンして見守ればいいのか？
新入社員の教育係りになったものの、何から伝えればいいのか？
会社や上司は、何もアドバイスしてくれないし……。

 

このように新入社員の教育に当たる先輩・上司は
それぞれの立場で、悩みを抱え、思考錯誤しています。

 

そして、社内で相談してみても、ビジネス書を読んでみても
しっくりくる解決策を得られそうにない。
そんな風に、解決できない悩みを抱えている人は、大勢います。

 

今回は、そんな悩める先輩・上司・新入社員教育係りの方を対象に
「 新入社員教育や指導を担当する 先輩社員 ＆ 上司の情報交換 交流会 」
を開催します。

あなたと同じように、４月に新入社員を迎え
同じような悩み、課題、不安を抱えた方々に参加して頂きます。

上から目線のアドバイスや、一般論ではなく、参加者の実際の体験・経験から

◆現場で体験した最新情報の交換
◆こういう場合は、こう切り抜けたよ、といった成功事例の交換
◆こんなケースでホントに悩んでるんだけど、誰か解決策ないですか？といった呼びかけと解決
◆異業種の方から同じテーマの話を聴き、ヒントと気付きを得る
※開催主への個別相談　当日の相談コンサルタントはコチラ

等が、可能です。

 

100冊の本を読むより、同じような境遇にある人と実際に会って話す方が
得られる気付き・発見、刺激は大きいはずです。
人は、刺激によってのみ、短期間で問題解決や成長ができます。

今まで社外の交流会や、コミュニティに参加した経験の無い方でも
開催主スタッフが、丁寧にアシストします。
安心してご参加ください。
