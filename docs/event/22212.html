概要

Boot Campと称した技術系のイベントが全国各地で開催されていますが、このVi Boot Campでも実際に手を動かすことで、学習コストが異常に高いと言われるViエディタに馴れる事が目的です。
また開催地が群馬という事で、いくつかの趣向も用意しています。
本当のViと群馬を堪能しましょう！

趣旨

移動するには h,j,k,l …
文字を入力するには a や i …
といった内容から始まる説明は、Vi の解説やチュートリアルでよく見かけるものですが、必ずしも適切でありません。
実際に Vi に使い慣れてみると分かりますが、それらのコマンドの使用頻度は低い部類に入ります。そういった事実を知らずに a や i に慣れてしまうと、後々のコマンドの習得に弊害をもたらすことさえあります。

Vi コマンドには無駄なものがほとんどありません。コマンドの数も決して多くはありません。
一方でコマンドを覚えただけでは Vi を思いのままに使うことができません。

Vi への誤解や疑問を解決し、この一日で全ての Vi コマンドを体験して、Vi 習得のロードマップを描きましょう。

対象

普段Viをお使いの方から、これからViを始めたい方まで、Viスキルを少しでも高めたいと考えている方。
修業が好きな方。
