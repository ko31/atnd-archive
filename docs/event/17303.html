とりま立ててみた。

8/21-22あたりで挑戦しようかと思ってます。
8/20-21(土曜に泊まって日曜戻り）で行きます。
昨年は日帰り登頂でしたが、今年は山頂付近で一泊してご来光みようかなと。
予定が合うと良いんですがー。

ルート参考

プリンスルート(2008年8月に、登山を趣味とする皇太子徳仁親王が以下略)とか魅力的
皇太子さまはこう行った↓

  1. 富士宮口五合目
  2. 新六合目で分かれて宝永山
  3. 宝永火口から御殿場口登山道7合目
  4. 御殿場口7.9合目の山小屋（赤岩八合館）に1泊
  5. 登頂とお鉢巡り
  6. 御殿場口の大砂走り経由で

山頂の山小屋に泊まるとすると、18時までにはチェックインしないと厳しいだろうからちょい迷っているところ。

当日の予定

富士宮ルート山頂の山小屋に宿泊します。
出発は富士宮口ですが、混み具合に応じて宝永火口から御殿場ルートに避難するかも？

東京　 6:56発 こだま633号(自由席)[新大阪行]1時間12分
品川　 7:04着
新富士 8:08着
新富士 8:30発 登山バス
富士宮口 11:00頃 到着予定
山頂山小屋 16時〜18時 チェックイン

下山は楽なので、剣が峰→お鉢巡り後、御殿場ルートの下山道を降ります。
宝永火口経由で富士宮口にJoin予定。

装備参考

要るモノ

  1. ザック＆レインカバー（中身が濡れないように）
  2. 靴＆靴下（足首までがっちり固められるモノ）
  3. 雨具（防寒具も兼ねる）
  4. 手袋（素手や軍手だと雨降ったら死ぬ）
  5. ヘッドライト
  6. ショートスパッツ（下山時、砂が入らないように）
  7. 水

準・要るモノ

  1. 日焼け対策（帽子・アイウェアなど）
  2. カメラ用の防水処理
  3. 杖（随所で金剛杖が売られてますが）
  4. 食料（ウイダーインゼリー的なものや、飴など。普通は固形物を喰う気にならない）
  5. ジップロック（何かと便利です）
