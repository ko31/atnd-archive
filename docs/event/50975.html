catch SNS等で利用するプロフィール写真をよりキレイにしてみませんか。（講師：吉田 聖未さん）
description

■日時

2014年6月17日（火) 19:30-21:00（受付19:00-）

■費用

3,000円（当日払い）

■場所

ベースキャンプ名古屋

■持ち物

  • PhotoshopがインストールされているPCが必要です。（バージョンは最新でなくても大丈夫です。）
    Photoshopをお持ちでない場合、事前に体験版をインストールしてください。
    Photoshop CC 体験版 | Adobe Photoshop 無料ダウンロード （体験版は30日間です。ご注意下さい。）
    先着2名までiMacの貸し出しが可能です。希望される方は申込完了後に本ページ一番下のコメント欄に、貸し出し用PC希望の旨をコメントしてください。先着順となります。
  • 名刺1枚（受付でいただきます。）
  • 筆記用具（アンケートをご記入いただきます。）

■内容

ソーシャルアイコンや証明写真などに使用する画像を製作しながら、簡単な肌などの修正やトーンの調整の仕方を覚えましょう。

◯人肌の直し方
◯トーンのあやつり方
◯顔の整形/ポーズの修正

その他、Photoshopに関してわからないことなどありましたら何でもご質問ください！
