「今流行のAWS(アマゾンウェブサービス)にサイトをアップしてみよう」セミナー[5時間マスター / 座学 + ハンズオン] #2

■勉強会内容

第二回の勉強会になります。

5時間でAWSの基礎からサーバ構築まで実践を通して学習します。

AWSの知識として押さえておくべき事を全て通して行うため、業務など
で今後活用を検討されている方はすぐに役立ちます。

セミナーで使用する資料は全てお持ち帰り頂けますので、ご自宅で復習
が十分に行えます。

AWSのおすすめサイトのご紹介などもさせて頂きます。

今回のセミナーではフリーソフトのCloudBerry/WinSCPを使ってサーバに
ファイルのアップを行います。

•サーバにアップするサイトは自由
アップできるご自分のサイトがあればそちらをお持ち下さい。
その際にご自分のドメインを使用する場合はそちらもお持ち下さい。
※特にアップするサイトがない場合にはこちらで仮サイトを用意しております

〜 セミナーの学習タイムテーブル 〜

時間 内容
13:00-16:00 •AWSの基礎
　EC2/RDS/S3とは
　CloudBerry/WinSCPのソフトの使い方
　EC2を使って仮想サーバ構築の基礎
　EC2にキーペア利用WinSCPでのサイトアップ手順
　RDSでデータベース作成の基礎
　S3にCloudBerryで動画アップ手順
　ドメインRoute53の設定の基礎
16:00-18:00 •AWSの実践
　仮想サーバ上にサイトを実際に設置する

※時間帯は進捗により前後する可能性もございますので、ご了承ください

備考：
ドットインストールのAWSをみてからご参加頂けると
よりスムーズに進行できます。
