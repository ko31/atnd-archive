description

フィルム製造・加工工程には、そのプロセスおよび設備に起因する特徴的な帯電気現象が関与しています。そして、その特殊性と複雑さにより、基本的な理解が進まず、対策に苦慮されている事例が多いようです。今回は、フィルムプロセスに対象を絞って、静電気の基礎から現場で使える具体的な知識まで解説いたします。
