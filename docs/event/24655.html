アジャイルサムライ読書会 in エイチーム道場【第3回】

株式会社エイチーム にて開催される「アジャイルサムライ」の読書会です。
エイチーム社内外を問わず、参加者同士が広く本書への理解を深め、それぞれの現場へとその価値を持ち帰る事を狙いとします。

■対象者

問いません。
ただし、参加者は何らかの形で
「複数人が関わるサービス・プロジェクト」
の開発もしくは運営に関わっている（もしくは今後関わろうとしている）状況が望ましいです。

※開催第3回となりますが、今回を機に新しく参加される方も積極的に募集しています。

■参加者へ事前のお願い

  • あらかじめ各自にて書籍を購入しておく事。（使用する書籍は アジャイルサムライ ）
  • 当日は可能な限り本書の持参をお願いします
  • 各回ごとに定められた範囲を対象に、あらかじめ下読みをお願いします。
    • なお、その際に 自身が現在関わっているプロジェクトと照らし合わせた上での気付き・疑問・感想 を心に留めておきましょう。

■読書会の進行について

  1. 4～5人ごとにテーブルに分かれ、それぞれ以下の内容についてディスカッションを行います。
    • 今回の対象範囲を読んで得た気付き・疑問点について
    • どのように自己の業務に活かせるか
    • 他者の意見・感想を聞いた上で更に得られた気付き
  2. 前回に引き続き、 ワールド・カフェ の形式を用います。
    • 各ディスカッション（ラウンドと呼ぶ）ごとに、テーブルを囲むメンバーを変更します。
  3. 各ラウンド毎に、それぞれのテーブルで得られた気付き・ノウハウについて全体で共有します。
  4. 時間があれば、ふりかえり(KPT)を実施します。

■今回の対象範囲は？

第III部 アジャイルな方向づけ（前半）

  • 第6章：ユーザーストーリーを集める
  • 第7章：見積もり：当てずっぽうの奥義

■会場について

○開催場所

株式会社エイチーム　ミーティング広場
(住所:名古屋市西区牛島町6番1号　名古屋ルーセントタワー32F)

○アクセス詳細

  • 高層階エレベータ(西側)より、32Fの受付までお越しください。
  • 32Fエレベーターホールに張り紙を貼り誘導する予定です。
  • 20時以降はルーセントタワー（ビジネス棟）への入館が出来ません。20時以前の入館をよろしくお願い致します。

  • 無線LAN利用可。
  • 電源は席によっては利用可。（対話中心のため、電子機器はほぼ利用しない想定）
