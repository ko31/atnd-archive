持ち物:
Androidのソースコードリーディングについては、ハンズオンぽい感じで進めます。ただ聞くだけでも結構ですが、より楽しむには
・Android SDK + Eclipseなどの開発環境が整ったノートPC
・下記サイトから予めAndroidのソース一式をダウンロードしておく
http://source.android.com/source/index.html
または
https://github.com/oesf
の2点が必要となります。
※今回は概念+環境セットアップ中心です。今後の勉強会で、このリーディング環境をベースにコードリーディングを続ける予定なので、続けて参加される予定の方は、なるべく今回環境を準備しちゃって下さい。
※参考　Macな方なら黙ってrepoをインスコ。Winな方なら以下を参考に。
http://blog.cnu.jp/2010/10/15/android-sourcecode-reading/
