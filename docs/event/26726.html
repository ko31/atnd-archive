※原則学生の方向け

Webサービス運営、Webベンチャー起業に興味がある学生の方向けに、WebベンチャーファイナンスとM&A売却についての重要性を学ぶための講演会を開催します。
外部講師として、「起業のファイナンス」の著者であり、人気メルマガisologueを発行されている公認会計士の磯崎哲也氏（元・ミクシイ監査役）、
および東京大学経済学部・McKinsey&Co.出身で過去数件のWebベンチャーの起業・M&A売却を成功したシリアル・アントレプレナーで株式会社SpotlightのCEO柴田陽氏のお二人をお招きし、理論面・実務面の両方からベンチャー起業におけるファイナンス計画とM&A売却の重要性を教えて頂きます。

講演会は３部形式で、松尾研、磯崎哲也氏、柴田陽氏がそれぞれ主に以下の３テーマについて講演会を行います。（講演タイトルは変更されます）
1. Webベンチャー研究におけるファイナンスの重要性　（松尾研）
2. ベンチャー企業に求められるファイナンス政策とM&A売却　（磯崎氏）
3. Webベンチャーの事業計画とM&A売却の実際　（柴田氏）

これからWebベンチャー起業を考える学部学生、修士学生の方に対して、普段の講義では学ぶ機会のないベンチャーファイナンスとM&A売却について最も分かりやすく学ぶ機会となっておりますので、お誘いあわせの上奮ってご参加ください。
