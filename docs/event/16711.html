“Ruby Summer of Code: NArray on OpenCL” – Kazuyuki Tanimura

In the Ruby Summer of Code 2010, the speaker has developed the NArray on OpenCL that utilizes hardware resources such as GPU. The goal of the project is to translate the NArray from C to OpenCL. This project
allows Ruby programmers to facilitate implementing and executing scientific (CPU-intensive) applications faster. The NArray’s characteristics matches OpenCL (GPU) applications well, and OpenCL is hardware agnostic so that the programs using OpenCL should run on various CPUs/GPUs. This means the translation of NArray to OpenCL does not detract the feature of the Ruby’s portability. In the presentation, the speaker will introduce the overview of Ruby Summer of Code and discuss technical difficulties of the project.

“HTML5で出来るようになる事” – 高木敦也

現在、HTML5は大きな注目を浴びており、仕様と実装が日々活発に行われています。今回の発表では、HTML5で出来るようになる事を紹介すると共に、私が個人的に開発している、Flashを使用せずにウェブブラウザ同士でファイルの送受信を可能にするアプリケーションについてお話ししようと思います。
