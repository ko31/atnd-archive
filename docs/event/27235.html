概　要

Webから汎用までソフトウェア開発を行う為に必要な統合開発環境(IDE)の導入・活用法や
開発環境自体の構築方法まで、構築～プログラミングの現役システマーによる初級者向け
無料セミナー。

対　象　者

システム開発に興味のある方が対象なのでお気軽にご参加下さい。

参　加　費

無　料

内　容

「 WebMatrixでらくらくWebサイト構築」
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　株式会社テイルウィンドシステム　藤森 大地
　　　Windowsプラットフォームで動くローカルのWeb開発環境「WebMatrix」のインストールから
　　　環境構築までと、簡単なサイト構築を実機デモを行いながら説明します。

「仮想データベースサーバ構築」
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　株式会社テイルウィンドシステム　岡田 裕二
　　　VMWarePlayerを使用しての仮想DBサーバの構築方法を実際に構築を行った環境で
　　　実機デモを交えて説明を行います。環境：CentOS 6.0、MySQL5.5

「 開発現場で使えるExcel実践活用 」
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　株式会社テイルウィンドシステム　波江野 篤
　　　Excelを使用してTeratermマクロを組み合わせてリモートサーバの簡易表示機能、ACCESSデータ
　　　ベースから設計書の自動生成、DB設計書からスキーマの自動生成等の説明を行います。

「 HTML5 ～FLASHからの転換～ 」
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　株式会社テイルウィンドシステム　山崎 功貴
　　　FLASHに替わる企画として注目を浴びている「HTML5」。JavaScriptとCSS3を含めた
　　　意味での「HTML5」を、今どのように扱っていけばよいのかに焦点を当てて説明します。

「 PHP+Smartyによる動的Webサイト構築入門 」
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　株式会社テイルウィンドシステム　佐藤 昌和
　　　Smartyは機能と表示を分離する為のテンプレートエンジンとして非常に優秀で、多くのPHPの
　　　システムで使われています。Webサイト構築の事例を交え、PHPとSmartyの基本とメリットを
　　　説明します。
