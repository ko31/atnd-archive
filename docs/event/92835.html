description

近年、マスコミによる様々な事故や不祥事の責任者追及の場面を目にすることはほぼ日常茶飯事となってきました。また、インターネットやSNSの普及により民間レベルでも、犯人探しや集中的な非難が容易となりました。このような日々行われるスケープゴーティングに違和感を覚える人、また自分がその対象になってしまうかもしれないと漠然とした不安を感じる人は少なくないのではないでしょうか。
　何故犯人探しが行われるのか、この問題を冷静に観察するにはどのような視点が必要か。これらの疑問を考える上で必要となる知見を参加者の方々に提供したいと思います！その中でも今回は、善人がなぜ戦争やテロなどの悪を行うのか、そしてそれがなぜ繰り返されるのか、そのメカニズムの一端について紹介します。
また、戦争などの異常事態のみならず日常生活でも、罪のない人が攻撃の対象となるスケープゴーティング現象が生じることがあります。これについても紹介します。
現代において注目すべきトピックであるスケープゴーティングを一緒に学びましょう！

そして、今回講演していただくのは社会心理学の第一線で活躍されておられる釘原直樹先生です。
釘原先生は、大阪大学人間科学研究科社会心理学研究室の教授で、主に、「グループ・ダイナミックス 集団と群集の心理学」、「スケープゴーティング 誰が，なぜ「やり玉」に挙げられるのか」、「人はなぜ集団になると怠けるのか」など執筆しておられる、まさに集団心理・人間心理のエキスパートでいらっしゃいます。
普段ニュースなどで得た知識だけを知ってるのではなく、釘原先生だからこそ語ることのできる、素人とは違う研究者の視点をぜひご堪能ください！
これであなたも正しい知識で対策もばっちりです！
