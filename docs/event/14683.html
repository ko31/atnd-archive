【お待たせしました！「Re:」 vol.2開催します！】

今回も出演者及びスタッフはオールP.T.A.女子です！
4/20という事で、当初発売予定していたPerfumeの新曲をもじってサブタイトルを決めましたが、発売延期になったので当日新曲をかける事は出来ません。
その代わり「チーム・アミューズ」による震災復興に向けたスペシャルソング「Let’s try again」をかけられたらなと思っています。
当日アミューズ募金も行う予定です。
(会場の渋谷Aurraのすぐ近くに某本社、某カフェもあります)

オールジャンルのイベントなんで誰がどんな曲かけるか楽しみにしてて下さい！
メンバーもスタッフもオール女子ですが、お客様まで女子限定ではありません。
平日ですが、仕事後など予定空いてたら是非来て下さい！

例えばアニソン
例えばJ-POP
例えばテクノポップ

【出演者紹介】
★icchan (DF/ananda)
Perfume系イベントDFでDJデビュー。
毎月第一水曜日には渋谷club asiaで行われるエレクトロ系イベントanandaにDJ kiliと一緒にレギュラーで出演している。
Ustreamではicchan23という番組で時々DJを披露。
Live8でremixにも挑戦中。
4つ打ち好き。のっち好き。sfp好き。
最近では「実はドジっ子説」と言われ、なんとか払拭したいと試みている。

★kili (DF/ananda)
2009年冬、六本木で開催されたedgeでDJデビュー。のはずだが本人にとっては葬りたい過去でもある。
現在はasiaで毎月開催されているanandaでのレギュラーDJに加え、錦糸町某所で密やかに開かれている練習会に参加して腕を磨いている。
よくお腹をこわす。

★Mikofu(じゃぱんぐ♪/E3!!!/DF)
2005年『♪♪♪』主催と共にＤＪデビュー。
同年「じゃぱんぐ♪」にGEST DJから正式にレジデントDJへ。
2006年80年代洋楽を中心のイベント「80’s HITS AGAIN」開催。
2007年後半からジワジワきたPerfume熱。
2008年にスイッチが入り、 Perfume系DJイベント
「E3 (ELECTROnic Edge Effect)」 に参加。
得意ジャンルはJ-POP 80～00年代＆アイドル歌謡曲AND洋・邦Rock 洋物80’s
「ｸﾗﾌﾞDJって４つ打ちやＲ＆ＢだけぢゃないんだZE」な魂でスピン。

★バブル（おねパル/AMP/変）
J-POP、アニソン原曲を中心にスピン。
幼少期からBLに目覚め、週１レギュラーでMANDARAKEに通う
自身でも二次創作を発表したり、酒芸を披露したり
勢いの止まらないアラサーDJ

★ス-パ-ハウス∞超家(SEVENTH HEAVEN)
奇跡のおちょぼ口Perfumeフリコピユニット。
メンバー全員のおでこが広く、メンバー全員の口がおちょぼである。
練習と称して集まってはTVで宝塚を見ている。
お菓子の消費スピードがすごい。
