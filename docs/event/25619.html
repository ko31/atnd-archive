セッション1：

すぐできるスマートフォン対策・WordPress編

旧型携帯電話、高機能携帯電話、スマートフォン、タブレットなどPC以外の多様な端末に対応できるWEBサイトの構築について、具体的にテーマをカスタマイズする方法を紹介します。なお当日、会場でサンプルテーマをダウンロードできます。

講師：美多幸夫

（元国際デザインカレッジ・金沢情報ビジネス専門学校専任講師）

セッション2：

ブログみたいにWEBアプリが作れてしまう、新感覚のCMS講座

Canvasの装備でグラフィックの表現力が豊かになったHTML5では、これまで敬遠されがちだったJavaScriptのプログラミングが、ずっと親しみやすくなっています。テキストエディタ＋ブラウザのようなこれまでのスタイルから脱した、オールインワンのCMS（コンテンツマネージメントシステム）を使って、まるでブログを書くようなライトな気分でWEBアプリケーションを開発することが可能となりました。WEBの世界からhtmlファイルを消してしまったのがMovableTypeやWordPressなどのCMSですが、さらにcssファイルとjsファイルも消してしまう、オリジナルCMSの全容を紹介します。

講師：美多幸夫

（元国際デザインカレッジ・金沢情報ビジネス専門学校専任講師）

セッション3：

PowerPoint不要、HTML5流プレゼンテーション術

HTML5の新機能を生かせば、臨機応変、リアルタイムに強いプレゼンテーションが可能となります。

ライトニングトーク：

WEB系のテーマで10分程度お話いただけるスピーカーを3名程度募集しています。希望者はコメントにお書き込み願います。

会場：ITビジネスプラザ武蔵 5F 研修室1
