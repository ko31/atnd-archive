MySQL Casual Talks を福岡でやります！！

MySQL Casual Talks in Fukuokaでは、 カジュアル にMySQLについて語り合う会です。

カジュアルな会なので不安に思うことは何もありません。

福岡には、まさかりを担いでいる人や、椅子を投げる人はいないので安心してご参加ください。

  • Main Talk(20分)、LT(5〜7分)を応募中です！内容はMySQLに関することであればなんでもOKです。
  • 応募方法は下記の通りです。応募する際は、twitter_idとトークのタイトル(仮のものでかまいません)を記載してください。
    1. atndのコメント欄に応募する旨を投稿する。
    2. FBグループ MySQL Casual Talks in Fukuoka に参加申請して、応募する旨を投稿する。
    3. twitterで Spring_MT 宛にメッセージを送る

学生、新卒の方も奮ってトークに応募してください！もちろん参加だけでもOKです。
