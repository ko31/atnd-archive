概要

今年6月にSencha Ext JS 5がりリリースされ約6ヶ月が経過しましたが、
Senchaを既にご存知の方も、そうでない方もいらっしゃると思います。
少しずつ札幌にもSenchaは浸透してきている中で、今回はExt JS 5に重点を置いて、
新機能や開発事例などの紹介をさせて頂きます。
システムの導入をご検討されている企業様やWebシステム開発を行う為にフレームワークの選定をされている方、
Senchaに触れてみたい方どなたでもお気軽にご参加下さい。
