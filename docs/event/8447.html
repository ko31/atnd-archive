riverlinux Kick Off Meeting

イベント ユーザー
event_id 8447
title riverlinux Kick Off Meeting
catch Linux愛好家のための川遊びサークル、始動！
description

riverlinux は Linux 愛好家のための（主にカヌー・カヤックをベースとした）川遊びのサークルです。
とはいえ、飲み会の席でなんとなく盛り上がり、その帰りの電車の中で google group を作ったばかりなんですが(笑)。
サークルの概要は こちら をご覧くださいまし。

んで、せっかく作ったんでキックオフミーティングをやりたいと思います。

場所

多摩川　丸子橋たもと　川崎側　多摩川台公園
（最寄り駅：東急東横線　新丸子駅 or 東急東横線・目黒線・多摩川線　多摩川駅）

スケジュール

11時 東急東横線　新丸子駅　改札集合 東急多摩川駅　改札集合
11時半 準備、セーフティトークなど
〜16時 カヤックを漕ぎ、合間合間に昼ごはんを食べ、Linux について語り、お茶をするなど
〜16時半 撤収
〜？ 反省会 @ そこら辺の居酒屋など

参加費

Free (as in “free beer,” not as in “free speech”
ただし反省会は実費精算

必要なもの

  • Linux や Free OS を愛する気持ち (笑)
  • カヤックを愛する気持ち、or カヤックにチャレンジしたいという好奇心
  • 昼ごはん（各自）　その場で調理するものは NG
    • 途中でスーパー寄るので買出ししてもいいです。
  • 飲み物。安全上、過度の飲酒をされた方はお断りしますが、少しビール飲むぐらいなら OK です。
    • 一応運動なのでペットボトルのドリンクぐらい持ってきた方がよろしいですよ。
  • 濡れてもいい着替え。できれば上半身はユニクロなどの即乾長袖Tシャツ、下半身は水着にジャージあたり。ジーパンは万が一泳ぐと体に張り付いて泳ぎにくいので絶対やめてください。
    • ワタクシのでよければ、ユニクロのフリースとジャージの下、それぞれ持って行きます(Lサイズ)。
    • なお着て乾かすにはちと時期が時期なので、着てくる服とは別にご用意ください。

主催者側が用意するもの

  • 着替えテント
  • ボート (ポリエチレン艇ｘ４、折りたたみ艇ｘ１とありますが、漕げる人間が常に水上で待機しなければならないので -1 です）
  • その他ギア (パドル、ライフジャケット、ヘルメット、靴)
  • パドリング用ジャケット (水しぶきから水濡れを守るためのもの)
  • お茶セット。

(注)

  • 未経験者歓迎です。
  • ゴミは全員分回収して持って帰りますのでよしなに。
  • 水温はそれほど冷たくないですが、水にあたってから風に当たると冷えるので、風を通さないウィンドブレーカーなどを持ってくることをおすすめします。
  • まずはオフに参加してから group の参加を決めたい、という方がいてもよいですが、可能ならば先に group にはいっていただいたほうがうれしいかな、と。

なお riverlinux では年内にあと１回カヤックイベントを考えております。
そちらは一泊二日のキャンプツアーになるため、やや敷居があがりますが、難易度はさほど変わりません。
本イベントの参加如何に関わらず、ぜひ前向きにご検討ください。

つことで、よろしくっ。
