6月11日(土)に、全日本学生ゲーム開発者連合（全ゲ連）第18回交流会を実施します。 前回までとは会場が異なっています。参加予定の方は必ずご確認下さい。

講演者募集は 5/24(金) に締め切りました。

交流会について

全ゲ連は、有志が講演を行う交流会と、その後の食事会である懇親会の二部構成になっています。

このページのイベントでは、交流会に一般参加予定の方の参加登録をお願いします。 交流会自体は登録無しでも当日の出入り自由ですが、 これは大まかな参加者の把握と公開を目的としたものです。

懇親会の参加登録については次項の懇親会用ページで行って下さい。

また、交流会で講演をしてくださる方は後述のフォームにて講演者登録を行って下さい。

懇親会について

懇親会に参加される方は、以下のイベントの参加登録をお願いします。 交流会に登録された方も別に登録する必要があります。

https://atnd.org/events/76987?k=43c0bff6cd5f257868fddaf44824e389

講演内容とタイムテーブル

13:30-13:40 基調講演 / 水鳥 (@mizuooon)

通常講演

13:45-14:20 学生ゲーム開発のすゝめ / Mimaki SiON (@SiON_PN)

学生時代に同人ゲーム開発を行っていた経験を元に、学生がゲーム開発を円滑に進めるために知っておくべきことについて講演します。

14:30-14:55 「塗り」を表現する技術 / Es (しゅみぷろ) / from 東京電機大学 Amusement Makers

「塗り」を表現する技術 ゲーム中、物に色を塗るために必要なグラフィックス関連の知識についてのお話をしようと思います。

15:10-15:45 一般的なチートの手法と対策について / 黒河優介 (@wotakuro)

ゲームのオンライン化により、チート自体が問題となるケースがちょくちょく発生します。本公演では、一般的なゲームのチート手法（考え方）と、その簡易的な対策手法の話をしたいと思います。

※悪用されたくないので、チートの具体的なやり方等については一切触れず、チートの考え方だけをご紹介します。

15:55- ライトニングトーク講演

PlantUMLでモデリング / とりすーぷ (@toRisouP)

クラス図や状態遷移図、シーケンス図といった図を爆速で描けるようになるツール、「PlantUML」の紹介をします

東方2次創作ゲームの試遊コーナーを作ってみた＠例大祭 / ekus-U（えくす） (@ekusU)

昨年の東方Projectオンリーイベント「博麗神社秋季例大祭」で東方2次創作ゲームの合同試遊コーナー『博麗電脳試遊会』を主催・運営しました。参加サークル数15、参加作品20本超のゲーム試遊イベントを運営する上で面白かったことや苦労話とかが出来たら良いかなーと思います。
参考：昨年の試遊イベントの特設ページ

2ちゃんねるにおけるゲーム製作と展望 / ぷり◆XYX0N.O646 (@xyx0no646)

２ちゃんねるは皆さんご存知の巨大匿名掲示板です。あまり目立たないのですが、匿名だからでこそできる創作活動が日々行われています。 ゲーム製作を題材に、創作の場としての2ちゃんねるを紹介します。

ゲームエンジンAltseed / ナムアニクラウド(@NumAniCloud) / from 筑波大学 AmusementCreators

Amusement Creators発のゲームエンジン「Altseed」の特徴と近況を紹介します。シーン/レイヤー/オブジェクトの構造やpsdからのレイアウト読み込み機能など、ゲーム制作に役立つ機能を使うことができます。

スタッフ参加者の募集

全ゲ連では学生のスタッフ参加者を募集しています。 スタッフ参加者は、当日に受付やマイク渡しなどをスタッフとして行っていただく代わりに懇親会を半額で参加できます。

スタッフ参加者は学生限定で、人数については2,3人で締め切ります。
