RaspberryPi使っていろいろやってみよう

みなさま、RaspberryPiをお持ちでしょうか？
お持ちの方、どんなふうにお使いでしょうか？
お持ちでない方、どんなことができるか知りたくないですか？

えれくら！TypeRでは、ふつうのえれくら！とは別に、RaspberryPiを主題とした勉強会をしていきます。
RaspberryPiをお持ちの方には「今こんなことやってて楽しい！」という話をしていただけたり、お持ちでない方には「こんなことができるんだなー」ということを知っていただけたり、そんなイベントにしたいなーと考えています。

第1回目ということで、とりあえず、以下のような方々を対象に、と考えています：

  1. RasPi買ったけど買っただけ or 欲しいけどまだ買ってない
  2. RasPi買って使ってみたけど Linux をちょびっと起動してみたくらい

あっもちろんガチで使い込んでる方もご参加くださいませ！お待ちしてます！

ご用意いただくもの

  • RaspberryPi
  • ノートPC
  • SDカード

RaspberryPiにはモデルが2種類ありますが、どちらでもかまいません。お持ちのものを持参くださいませ。

RaspberryPi・ノートPCをお持ちでない方でも、会場からのご配慮により、少数ですが貸し出し用のRaspberryPiを貸していただけるそうですので、ご参加いただけます。

SDカードは、OSをインストールするのに使います。4GB以上の容量があるClass 4以上のものをご用意ください。変換アダプタがあれば、MicroSDでも大丈夫です。

内容

第1回目なので、かるーくジャブを打つ感じで :-)

  1. RaspberryPiについて
    RaspberryPiとは何か、とか、各モデルの違いとかあたりを軽く紹介します。
  2. RaspberryPiで利用するOSについて
    RaspberryPiで動作するOSには何種類かありますが、それぞれについて簡単な紹介を行ったり、インストール方法の解説をしたりします。
  3. GPIOを利用するプログラミングについて
    RaspberryPiのGPIOピンを使ったプログラミングを、簡単な例を使って紹介します。
  4. (発表希望者ありの場合)発表・LTなど
    何か持ちネタを発表したいという方やちょびっとだけど喋りたい方は、枠を作りますので、ぜひお願いします。
