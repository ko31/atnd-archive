description

タンに毒が入っていると定評のある牛さんが宮崎国から日本国の首都へ入国してくるそうなので、手厚い歓迎をしてあげましょう。

会場について

牛の五六八 を予約しました。
いろいろおいしいですが、特にサーロインがとてもおいしいです（要脂耐性）。

予約について

19:00 から 9 名の「ワタナベ」で予約しました。コースとかは指定してないので全力で肉を食って下さい。
座敷の10人席を用意する予定らしく、それを超えても頑張ればもう少し入るそうなので、ちょっとぐらいなら増えても大丈夫です。

待ち合わせについて

別に直接お店に行っても構いませんが、自分は 18:50 ぐらいまでJR浜松町駅の南口改札前あたりで待ってるので、何となく清らかなめるぽんっぽい人を見つけたら声を掛けてください。肥は掛けないで下さい。あと野菜も食べさせないで下さい。
北口と南口（とモノレール乗り換え口）があるので間違えないように気をつけましょう。
