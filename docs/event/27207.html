夢の語り合いは最高のコミュニケーション

Social Web Designersの610です。

突然ですが、最近、自分の夢を話したりしていますか？

幼稚園や小学校に通っていたときは「将来の夢はなんですか？」と先生からよく聞かれたものですが、中学生→高校生→大学生と大きくなるに連れて、その時間がだんだん少なくなっているように感じました。

それから5年、10年、20年、30年・・・「あなたの将来の夢」は何でしょうか。

３月のはじめに行われた第１回夢語りの参加者からの意見やその空間を通して次のようなことを感じました。

自分の夢を人に語るためには自分の中でしっかり整理されている必要があります。またそのアウトプットを通じて自分の考えを整理・確認することもできるので、いい機会になることは間違いありません。
また、他人の夢を聞いて刺激を受けるという点も大きなプラスになると思います。その過程や目標を目指すプロセスからなにか得られることがあるかもしれませんし、得られたノウハウを実際に実践してみるというものいいですね。

私達としては、この経験を通じて、同じ空間を過ごしたもの同士で継続的な関わり合いできればと考えています。可能であればソーシャルメディアを使って継続的にやりとりをし、その中で助け合えたりできればと幸いです。

夢語りは、知らない人同士が夢を語る空間です。第二回の今回は公開で２名を募集します。
