ご案内

Perl Mongerの集まり(PM)を国際展示場（ビッグサイト）前（っていうか周辺）で開催します。といっても、この時期この辺りの一所に集まるのは極めて困難かつ迷惑ですので、各自その辺をぶらぶら散策しつつ、他の参加者に出くわしたらLTを始めるなり、テクニカルトークを行うなり好きにやってください。

他の参加者とどうやってmeetするの！？

その疑問はもっともなことです。各自の創意工夫で解決することが求められます。それだけではどうにもならない場合は、勇気・友情・根性で補ってください。ちなみにKokusaitenjijomae.pm主催は国際展示場内 東N37b 辺りにいるんじゃないかなぁ。

同日に行われている他イベントについて

ところで、この日はKokusaitenjijomae.pm会場周辺で大規模なイベントが行われているようです。そのイベントに参加される方で初めての方は こちらのサイト を参考にされるのが良いかと。
