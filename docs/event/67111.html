description

「なんのために生まれて なにをして生きるのか
こたえられないなんて そんなのはいやだ！」
「何のために生きるか」を考えてみませんか？
特別、生き方について考えなくても、
生きていくことはできます。

でも、自分の生き方について考えることは、
主体的に自分の人生をつくっていくということ。

自分が何のために生まれて何をして生きるのかが
しっかりわかっている人は、
とても素敵ですよね。

生きることについて真剣に考える場所をつくりたい、
こんな思いに突き動かされて生まれたのが
「生き方デザイン塾」です。

生き方デザイン塾とは、
「生き方」をテーマにみんなが集える場所。
いろんな切り口やアプローチによって
自分の思い込みの枠をはずし、
自分の生き方を磨き、作りなおす作業を通して
変容のきっかけとなる場です。

生き方デザイン塾では、
自分の生き方を自分でつくっている4人が、
3日間に渡り
それぞれの得意分野で講座を開催します。

この講座に参加することで、
ゆっくり自分自身を見つめ、
生きることについて、
心の奥底にある本当の気持ちに気づくことができます。

ただ、講座とはいっても、
教わることを目的に来たら何も得られません。

ここは、自分にとって必要なものを
掴みとるための場所でもあります。
そのために、
ナビゲーターもビジョンに基づいて全力投球します。

私たちの姿勢を見せること、
それによって何かを感じ取ってもらうことも、
生き方デザイン塾の目的だからです。

「生き方についてもっとちゃんと考えたい」
「他の人はどんなふうに考えてるのかな？」
「腹を割って、生き方について話してみたい」
そんな熱い気持ちを持つあなたをお待ちしています。

あなた自身の生き方について、
考える時間を持ってみませんか？
