description

日々日常、心に感じるストレス。

人の悩みのほとんどは対人関係の中に生まれます。

そしてその原因はすべて自分の中にあるということにお気づきでしょうか。

 

「いやいや、わたしは悪くない、あの人のせいでわたしはこんなに苦しんでいる」

そう思っている間は、その状況から脱却することはできません。

だからといって「全部わたしが悪いんだ…」と自分を責めていても事態は変わりません。

 

日々感じる家族や友人・職場の人との関わりの中で生まれるストレスは、自分で生んだもの、なくすのも自分。

すべて自己責任で解決できる。

そう納得できた時、ストレスが一気に片付き、楽に人づきあいができるようになるのです。

 

ストレスを感じる心には人それぞれ傾向があります。

ご自分の心の傾向を知って、感情の生まれる原因を探って、自分の中で納得して。

という流れを経験することで、悩みが悩みでなくなり、課題が課題でなくなり、生きるのがぐんと楽になります。

 

☆なんとなく生きづらい。 

☆日々ストレスに追い立てられるように過ごしている。 

☆家族や会社の人との付き合いがつらいけれど、逃げ出すわけにはいかない。

☆もっと自分らしく生きていきたい。 

☆わたしの根本って何？

 

と感じていらっしゃる方へ。

自分で感情をコーディネートし、整理し、ストレスをコントロールできるようになるワークをご一緒に進めていきませんか？

 

 

 

具体的なワーク内容は、ノートに自分の心の動きを書き留めるという作業です。

ただひたすらに書き出す、それだけでも十分なのですが、うまく「自分」にたどり着ける方法をご一緒させていただくのがこのワークショップの目的です。

自分一人でワークを続けるにはつらく苦しい段階もありますので、後々のフォローアップの機会もご用意しております。

 

そうしてしばらく続けると、本当に自分が求めているものに出会えます。

自分の在り方という「道」に迷いにくくなります。

これからの人生を今までより楽に過ごしたいとお考えの方にはお勧めのメソッドです。
