TOC-CCPMインプリメンターコースのご紹介

「CCPM導入に向けて、しっかりと基礎から研修を受けたい」という数多くの声にお応えして、TOCに関する日本随一の豊富な実績とノウハウを持つゴール・システム・コンサルティング株式会社と、国内産初のCCPM対応ソフトウェア『Being Management-CCPM』を開発した株式会社ビーイングの共催による『TOC-CCPMインプリメンターコース』の開催が決定しました。
本コースは「TOCの理解」「CCPMの理解」を理論的に学ぶ基本編－1日と、「CCPM対応ソフトウェアの操作方法」「プロジェクトファシリテーション」「自企業での展開手順書作成」など、企業での実践方法を実習形式で学ぶ実践編－3日で構成され、CCPM導入のリーダーになりえる人材を育成します。

  • 基本編のみを受講された方は、以降開催の実践編参加資格が得られます。
  • 実践編のみの受講はできません（基本編－1日の受講が必要です）。
  • 実践編修了者には、ゴールシステムコンサルティング認定「TOC-CCPMスペシャリスト」 の資格が付与されます
    (コース付属のTOC通信教育講座「CCPM入門編」修了時の発行)。
研修費用、スケジュールなど詳しい情報とお申し込みは当社HPをご覧ください
