description

開催主旨：
バレーボール競技におけるアナリストの方々、またはアナリストを目指す方々が、スポーツにおける情報戦略活動の実態やバレーボールの競技特性の理解、バレーボールの科学的・専門的知識を習得することによって、アナリストの資質向上を目指すとともに、バレーボール選手のサポート体制の整備・確立・充実を目的として本セミナーを開催する。

主催：
公益財団法人日本バレーボール協会、全日本大学バレーボール連盟

主管：
日本バレーボール協会科学研究委員会調査部情報戦略班、及び
全日本大学バレーボール連盟科学研究委員会

開催日：
平成23年12月10日（土）～11日（日）

開催場所：
1日目：東京工科大学（蒲田キャンパス）　3号館30202教室
〒144-8535　東京都大田区西蒲田5-23-22　TEL.03-6424-2111（代）
2日目:とどろきアリーナ内
〒211-0052　川崎市中原区等々力1-3　 TEL.044-798-5000

集合・受付：
１日目：１２：３０～　2日目：９：３０～

講師（予定）：
JVAアナリスト育成インストラクター
全日本大学バレーボール連盟科学研究委員会委員

食事・宿泊：
お手数ですが、各自手配をお願いいたします。

対象レベル：
本セミナーは入門・初級レベルのアナリストを主な対象といたします。
※入門・初級レベルを対象とした講座であり、各基礎講座の内容は、これまで行われた育成事業（セミナー、ワークショップ、アカデミー等）と同内容も含みます。

参加資格：
参加資格は特に定めないが、以下のいずれかの項目に該当する者が望ましい。応募者多数の場合には書類選考し決定する。
1. チームでスタッフとして活動している学生
2. 中学・高校生はもちろん、大学生やクラブでプレーされている方
3. 情報戦略活動に興味・関心があるバレーボールの指導者
などで、アナリストを1から学びたいと意欲のある方、大歓迎！！

参加人数：
40名を上限とする。なお出席者が　20名に満たない場合には開催しない場合がある。

持ち物：
ノートパソコン※、筆記用具、ノートなど。
※ノートパソコンは、OSがWindows XP、Windows Vista、Windows 7であれば対応可能。

参加費：
一般社会人の場合：15,000円／学生の場合：7,000円（学生証持参）
※いずれも資料代を含みます。
但し、10日の夕食会は大学生以上のみの参加。会費として別途、一般社会人：5,000円　学生：3,000円。これらの参加費用は、講習受付時に徴収させていただきます。
