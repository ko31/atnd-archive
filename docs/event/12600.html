広告、Web、ゲームなど業界内外でのネットワーキングも兼ねて、サバゲーやりたいと思います！！

サバゲー好きな人、やったことないけど興味のある人、４月にサバゲーフィールドが予約できましたので、
参加者募集中です！一人だけでの参加でもＯＫです。電動ガンを持っていない人も当日レンタルできます。

ぜひご参加くださーい！

■場所：　千葉 バトルシティー・ユニオン
　現地集合、現地解散となります。
■集合時間：　9:30 現地集合
■アクセス：　
　・電車の場合　ＪＲ総武本線・成田線「佐倉」駅よりタクシーで１０〜１５分
　・車の場合　佐倉ＩＣより3.5km
　詳細ＭＡＰはコチラ
■参加費：　4,000円（昼食代、備品代入り）
　参加人数によって金額が変動する場合があります。

※ＢＢ弾は各自ご用意ください。
※チーム分けは集まり具合によって後日決めたいと思います。


＜サバゲー初めての方＞
※レンタル銃、ゴーグル、迷彩服が必要な方は、個別にフィールドで申し込んで借りていただくようお願い致します。
　常時３０〜４０丁は常備しているとのこと。
　レンタル料金はコチラ

　・手袋（軍手など）
　・スニーカーなどの動きやすい靴
