description

欧米ではすでにスタンダードになりつつあるプラスチックカード型のギフト・プリペイドカードが日本でも注目され始めています。多彩な使い道で導入する企業や店舗が増えています。贈答としてのギフトカード、チャージして繰り返し使えるプリペイドカードとして。顧客をリピーターとして囲い込むポイントカードにもなります。
また、従来の紙の商品券では出来なかった利用状況の収集や分析、商品やサービスの改善につなげるマーケティングにも活用できる。これからの小売、サービス業に欠かせないツールとなるでしょう。
その、1〜2ヶ月で導入可能というギフト・プリペイドカードの「バリューカード・ASP」の説明会を開催致します。

【こんな方はぜひご参加ください】
◆新しい顧客コミュニケーションツールを探している方
◆全く新しい、ブランド訴求ツールを探している方
◆すぐにでもプリペイドカードシステムをド入したい方。
