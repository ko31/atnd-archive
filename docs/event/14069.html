title アナログラム
catch ～Instagram写真展～ feat. SPCT
description

Instagramのいいところは、
何気ない日常の一こまを切り取るところ。
そしてその写真から人とのつながりが生まれるところ。

その何気ない日常の一こまをたくさん集めて見ると
撮影者の人となりもなんとなく見えてくる。

テーマは特にありません。
だれかの毎日、ちょっとだけ覗いてみませんか。

今回の展示はグループ企画展です。

展示の方法は、Instagramの世界をアナログで再現すること。
写真を額に入れて、はなれて鑑賞するスタイルは取りません。

プリントした写真の裏面に、ジオタグ、タイトル、説明を付けて吊るして展示します。
お客様は気軽に手に写真を取って、自由にめくって観ることが出来ます。

そして、気に入った写真にはスタンプ、付箋紙でコメントを付けてもらいます。
会期中、一度取りまとめて、出展者にお見せして
必要であればコメントを返してコミニュケーションを図ってください。
会期中に会場でコメントを返すことも出来ます。

最終日にはクロージングパーティーを開催します。
どのようなコメントが集まったか、お互いに交流して楽しみましょう。

出展者の募集は締め切りました。

会期は4月21日（木）〜5月3日（火・祝）
（4月27日はお休み、この間にコメント管理をお願いします）
