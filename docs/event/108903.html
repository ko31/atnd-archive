【セミナー内容】
消費増税、年金支給70歳、働き方改革、人生100年時代と様々なことが、目まぐるしく移り変わりゆく、この日本。

ある生命保険会社のアンケートによると、老若男女問わず80％以上の方が、自分自身の将来に不安を感じていると回答しています。
その一方で将来の不安を解消するために、今から万全な対策をしていると回答した方は、わずか4％に過ぎないようです。

健康・仕事・家族・お金…将来の不安を挙げればキリはありません。
サラリーマン・OL・公務員などなど、給与所得者の方が将来に向けて、やらなくてはいけない事は色々あります。
とは言え、何から始めればいいかわからない…そんな方が多いのでは無いでしょうか。

人生100年時代と言われる現代を生き抜く術は、案外身近なところにあったりするものです。
あなたは、様々な情報が入ってくる現代社会で、正誤を仕分けする力を兼ね備えていますか？
間違った情報や噂を鵜呑みにしていませんか？
本当に正しい情報と誤った情報が交錯してしまい、大きなチャンスを逃していませんか？

正しい情報を入手するには、常にアンテナを張り巡らせ、いち早く気付くことが必要です。
それらの情報は「いつ」「どこで」「誰が」「何のために」「どのように」発信をしているのか？
情報は「量」では無く「質」が大変重要です。

本セミナーでは、あなたとあなたの家族の将来を真剣に考えて頂くために「正しい情報」をしっかりと提供させて頂きます。

「何とかなるだろう」の楽観的なマインドは、もう終わりにしましょう!!

「何から始めればいいのか！」を徹底的に教えます!!

【こんなことが学べます】
本セミナーは不動産会社の営業マンが登壇しないスタイルです。
現役のFP代表がマンション経営の真の魅力を語ります。

◆このような方は是非ご参加ください
・不動産投資には興味があるけど、しつこく営業されそうで一歩が踏み出せない
・不動産投資そのものに対し、あまり良いイメージが無い
・なかなか貯金ができない
・税金や保険料が高いと感じている
・将来に対して不安を感じている
・お金に困らない人生を歩みたい

◆プログラム内容（一部）
・日本社会の現状と問題
・節税と資産形成の関係
・給与所得者、公務員がまず一番先にやらなくてはならない事
・なぜ今、都心のマンション経営が人気なのか
・マンション経営を取り組む際に気をつけるべきこと
