description

　「100万回遊べるカードゲーム」として世界で人気沸騰中のカードゲーム「ドミニオン」の遊び方を学び、実際に体験しましょう。中学生以上から大人まで、幅広くどなたでもご体験頂けます。参加は無料です。お気軽にお申し込みください。ドミニオン以外のゲームも多数用意しています。

【ドミニオンとは】
　2008年にアメリカで発売され、瞬く間に世界中で大ヒットし、各国の賞を総なめにしたカードゲームです。2人～4人で同時に遊ぶことができ、カードの組み合わせに応じて「戦略」を組み立て、一番多くの「勝利点カード」を獲得したプレイヤーが勝利します。しかし、勝つ為には「運」も必要です。家族でも、友人同士でもお勧めです。先日ドイツで開催された「第一回ドミニオン世界選手権」では日本人が優勝し、話題をさらいました。

【世界の面白ゲーム体験会について】
　この「世界の面白ゲーム体験会」では、コンピューターゲームではない、世界で人気のボードゲーム・カードゲームを毎回紹介し、参加者同士で遊ぶことでより深く、そのゲームについて学んで頂きます。日本ではまだまだ一般的でない、主に大人向けの海外のゲームの普及を目的としています。第２回のゲームは、第１回に引き続き、大人気の「ドミニオン」です。
　なお、すでに様々なゲームを楽しんでいる方がご参加された場合には、ドミニオン以外にもゲームを用意しておりますので、お楽しみください。ゲームの持ち込みも歓迎します。

　大人の部：世界の面白ゲームに興味のある大学生および社会人

　　　例）家族でできるゲームを探している人
