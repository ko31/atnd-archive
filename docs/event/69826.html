イベントの概要：

モバイルアプリのバックエンド共通機能をAPIで提供するBaaS「アピアリーズ」を利用したiOSアプリ開発をハンズオン体験いただくワークショップです。受講者にはアピアリーズの商用アカウントが半年間無料となるキャンペーンコード及びカスタマイズして商用利用も可能なサンプルアプリソースコードを差し上げますので、その日から本格的なモバイル開発を始められます。

共催：

株式会社アピアリーズ

対象受講者：

iOSアプリ開発の経験がある方（初級者も歓迎です）
BaaSを利用したアプリ開発に興味のある方

ワークショップ概要：

ワークショップ形式でアピアリーズを用いたアプリ開発の実践講義を行います。
アピアリーズを利用することで、サーバサイド機能を使ったアプリケーションを簡単に作成できるようになります。
本講義で配布するサンプルアプリのプロジェクトを利用することで、最終的には、自分自身のオリジナルアプリが作成できます。

※聴講のみの希望者も大歓迎です。iOS開発には自信がないな…という開発者の方、あるいはディレクター、デザイナー、PMの方などもお気軽にご参加ください。また、ワークショップ終了後の22:00から簡単ではありますが交流会の場も設けております。もちろん参加費は無料です。

ワークショッププログラム内容：

19:00～19:10　アピアリーズの概要紹介
19:10～22:00　Developers Workshop
22:00～23:00　懇親会（自由参加・自由解散）

1. アピアリーズの概要紹介

BaaS「アピアリーズ」を利用するメリット
アピアリーズの利用事例

2. Developers Workshop

第1部：アピアリーズ概要について

　－開発環境の確認、無線LANセットアップ
　－BaaSとは？
　－アピアリーズを知る。
　－アピアリーズのアカウント作成
　－JSON Data API を使ってみる
　－データを検索する

第2部：写真投稿＆メモアプリ「FeedSample」について

　－プロジェクトファイルのダウンロード
　－完成版アプリの確認とアプリの機能解説
　－アピアリーズ iOS SDK のダウンロードと準備
　－StoryBoardとViewControllerの実装解説
　－アピアリーズ JSON SDK を利用したデータ参照の実装
　－アピアリーズ ファイルSDK を利用したデータ参照の実装
　－写真投稿画面の実装解説
　－アピアリーズ JSON SDK を利用したデータ登録の実装
　－アピアリーズ ファイルSDK を利用したデータ登録の実装

第3部：「FeedSample」の機能拡張、まとめ

　－プッシュ通知の機能解説
　－プッシュ通知の実装解説
　－サンプルアプリの完成、動作確認
　－本日のまとめ
　－より実践的なO2Oアプリの紹介、解説
　－アピアリーズドキュメントサイトについて
　－質疑応答

講師、および開発者同士の交流の場として活用ください。

植田 洋次
