title D&AD Talk Event
catch Creative Directors in Conversation vol.6
description

世界で最も厳しいと定評のある審査を経て選考される国際的なデザイン・広告賞、D&AD賞。最も多くの国からエントリーを募った2014年度より、輝かしい賞を受賞したクリエーターおよびに審査員による体験談、審査の現場についてお話しいただきます。海外のデザイン・広告賞への関心・参加が増える今、出演いただくクリエーターたちのリアルな言葉とともにD&AD賞を分析します。海外のデザイン／広告賞を目指す方にとって、必ずヒントが見つかるはずです。2014年11月に開催されれたvol.5に続き、新たなゲスト出演者を迎え開催します。

出演：
古平正義（FLAME／アートディレクター／グラフィックデザイナー）
2013年度グラフィックデザイン部門審査員。2007年グラフィックデザイン部門＆タイポグラフィー部門イエローペンシル、2014年度クラフトフォーデザイン部門インブック等、多数受賞

八木義博（電通CDC／クリエーティブ・ディレクター／アートディレクター）
2010年度アートディレクション部門、2014年度クラフトフォーアドバタイジング部門審査員。2009年度パッケージデザイン部門イエローペンシル、2014年クラフトフォーデザイン部門イエローペンシル等、多数受賞

色部義昭（日本デザインセンター色部デザイン研究室 / グラフィックデザイナー）
2012年度ブックデザイン部門ノミネーション、2014年度グラフィックデザイン部門ノミネーション受賞
