catch スクー代表がビジョンをプレゼン、更に社員やエンジニアが登壇、募集中のお仕事をご紹介！
description

何かを学ぼう、と考えたとき、まず思いつく選択肢は、 書籍・勉強会・学校など……、 ずっと昔から続いているものばかり。 これだけインターネットという技術や、様々なデバイス、 高速通信回線がある今だからこそ、もっと楽しく、もっと高い学習効率で、 もっと便利に学べる学びの選択肢をつくることができるのではないか。 インターネット×学習でイノベーションを起こすため、 「世の中から卒業をなくす」というビジョンを叶えるため、 僕たちには、まだまだ仲間が必要です。

そこで、創業以来、初めての中途採用説明会を開催します！ 説明会では、社長のプレゼンはもちろん、どんなお仕事があるのかを、実際に複数の社員とも会える会場でご紹介します。

当日のプログラムは、
①代表の森健志郎による、スクー社の説明とビジョン紹介
②現在募集中の各職種について、担当の社員が紹介
③職種ごとに分かれて、質疑応答＆面談会
となっております。

▷ こんな方を募集しています！◁

【サービス開発部門】

◎開発マネジャー
WEBサービスにおけるサーバサイド・バックエンドのPHP及びSQL言語でのアプリケーション開発・運用経験のある方、スクラム等のアジャイル開発手法やウォータフォールモデルを用いたチーム管理経験のある方

◎デザイナー
IT企業／制作会社での制作実務経験が２年以上あり、UI設計の実務経験が１案件以上ある方

◎WEBエンジニア
WEBサービスにおける開発経験が２年以上ある方

◎iOSエンジニア
Objective-CによるiOSアプリケーションの開発・リリース経験がある方

◎Androidエンジニア
JavaによるAndroidアプリケーションの開発・リリース経験がある方

興味のあるお仕事は、どちらの時間帯で開催されるのかを確認の上、エントリーしてくださいね。
