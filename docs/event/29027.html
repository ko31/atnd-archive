description

『みんなで集まって、楽しい事をしよう。』
ただ、そんな単純な思いつきだけで開催された、そんなイベントです。

幼いときってみんなで集まって、みんなで企画(考えて)して
遊んだ記憶ないですか？

それが、楽しい事にそのまま直結する。
そんな幼いときの感覚をもう一度味わいませんか？

一生自分らしく生きたいと思う。
人の好き嫌いは千差万別。
否定も肯定も美しくはない。

人生ってどんなゲームよりも楽しめる。

「トリが引くぐらいのトリ肌」な企画をみんなで
考えて実行しちゃいましょう。
どんな簡単な企画でも、楽しいと思ったらイイじゃん。

例えば、Ａ.『みんなでＢＢＱしよ～』とか簡単な企画を、
B.『じゃあ、ちょっと普通のＢＢＱじゃない感じにしよう。』
C.『まず場所は?』
A.『う～ん・・・？普段は、緑地公園とかだから・・』
B.『もっとみんなが気軽に集まれる場所にしよう!!!』
C.『じゃあ、難波でしよう!!!』
B.『それ!!!いいね～!!』
A.『わかりました。では、ビルの屋上借りれるとこ交渉してきます。』
B.C.『ほんとに!!?？』
こんな感じで、簡単な企画でもみんなが楽しめる企画をみんなで
実行して、楽しみましょう!!
ちなみにこのＢＢＱ案は、前回参加してくれた方の案を企画進行中です。

大人になると、そんな事を考える機会もなくなってませんか？
そんな話をしながら、楽しくお食事しながらお話できたらいいですね。
そんな名目の交流会です。

前回は４/30.5/6にイベントを開催し、合計４７人の方に参加
頂きました。
今回、何人集まるかはわからないですけど、ちょっとでも
楽しい事したいなって人が来てくれたら嬉しいです。
