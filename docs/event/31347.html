IT Campとは？

長崎を盛り上げよう！という趣旨のもと活動する「ラブナガ！」の毎年恒例のイベント「ITキャンプ」を今年も開催することに決定しました！
所謂ハッカソン形式の勉強会で、PCを持ち寄ってみんなでわいわい開発しようぜ！というイベントです。
一人で引きこもって開発してても楽しくないでしょでしょ！

開発って言われたって…

Q.「自分1人で完成させられるか不安…」

A.キャンプは複数人のチームに分かれて行います。PG、インフラエンジニア、マネージャー、デザイナー、プレゼンター…プロジェクトメンバーのように役割分担をして開発を進めていきます。
コード書けないし…と開発を諦めているそこの貴方、役割、あるんですよ。
チーム開発をすることで開発友達ができるのも大きな魅力です。

Q.「作りたいもののアイデアなんて思い浮かばないぜぇ？ワイルドだろぉ〜？」

A.どうしてもアイデアが思い浮かばない、そんなワイルドな貴方にもアイデアを思い浮かばせる仕掛けを作っております！

Q.「ふえぇ＞＜チーム全体で考えてもわかんないところがでてきたよう＞＜」

A.行き詰まったチームも大丈夫。ｼｭﾋﾞｯと手を挙げていただければスタッフが駆けつけます。
コーディングの仕方から長崎のカステラ事情まで、ユーモアあふれるスタッフが貴方のお手伝いをさせて頂きます。
