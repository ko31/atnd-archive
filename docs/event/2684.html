PHP 初心者 勉強会 第27回目です。


PHP 初心者 勉強会 は、第20回目から、初心者 勉強会と中級者 勉強会に分れました。

この勉強会の対象者は、『PHPは初めてだけど、他のプログラムの経験がある』方です。

PHP初心者勉強会では、PHPプログラムの勉強過程を6段階に区切って、
簡単なプログラムから少しずつ難しいプログラムに挑戦していきます。

次回の難易度は6段階の3番目。保存したログの編集です。

途中から初参加される方へ

PHP初心者勉強会は、定期的に行っているシリーズ企画です。
前回の勉強会で解説した内容は再度説明しません。
勉強会の進め方なども説明しませんので、早めに来てレクチャーを受けて下さい。

その他

参加費用は無料ですが、貸会議室の代金が500円かかります。
（初参加の場合だけ1000円になります。2回目以降は500円になります）

勉強会は19時から開始ですが、会議室は18時より使えます。
18時からボランティアの中級者がいますので、開発環境の質問などが可能です。
また、課題についての不明点など、わかる範囲でお答えします。
