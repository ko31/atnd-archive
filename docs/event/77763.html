description

「クラウドサービスを導入し、利益を生むデータ活用の実現」をテーマに、新規事業立上げ、事業再編を支援するデータ分析のプロが講演。 さらに、Salesforce上で売上・予実・請求・集金をノンストップで実現するクラウドサービス効果的な活用方法について、導入事例を交えてご紹介します。

◇主催企業：ブレインハーツ株式会社、株式会社CloudPayment
◇特別後援：「社長はデータをこう活かせ！」著者　株式会社エムキューブ・プラスハート　取締役　平井明夫

こんな方におすすめです

・データを基に経営判断をしたい方
・利益を生むデータの活用をしたい方
・売上アップや業務効率化に興味がある方
・業務改善のためのクラウドサービス導入を検討している方

セミナースケジュール

16:00～ 開演

16:00～16:50　第1部
社長はデータをこう活かせ！～利益を伸ばすためのデータ活用方法とデータ収集のポイントを解説</>

経営者向けデータ活用指南書『社長はデータをこう活かせ！』(日本実業出版社)の著者が、 セグメント別損益・予算管理・経営計画等を用いて、赤字の原因、高利益の顧客や商品等を見極め、会社の利益を伸ばす方法を解説します。

株式会社エムキューブ・プラスハート　取締役　平井明夫

17:00~17:30　第2部
営業部課長様、必見！売上速報が貴社の営業活動をスピードアップ！

『ウランバ』は、Salesforceで使える売上管理クラウドサービスです。日々の見積、請求業務から売上状態を自動集計し、 年間計画との比較をリアルタイムに社内共有し、その時点での売上目標値がタイムリーに判断できます。 また、「経理のミカタ」との連携で見積、請求から回収に至る一連の作業を効率化します。

ブレインハーツ株式会社　営業部長　都田 喜彦

17:30~18:00　第3部
請求業務のオートメーション「経理のミカタ」による効果的なデータ活用を実現する業務の仕組み作り</>

Salesforceと連携した請求クラウド『経理のミカタ』による売上・売掛金管理業務の効率化をはじめ、経営データを活用した収益改善の具体的な方法について、 導入事例やデモンストレーションを交えてお話します。

株式会社Cloud Payment　営業推進本部　本部長　青木誠

18:00 希望者個別相談会
※業務の課題や導入に関する相談に講師が個別対応します。
