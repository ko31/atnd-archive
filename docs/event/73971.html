title ★２０代のための不労所得勉強会★
catch
description

不労所得とは

簡単に言えば働かずに得られる毎月の収入です。

それを得るコツは、若いうちからコツコツ始めることです。

投資はコツコツ始める事で将来的に大きなリターンを得る事ができます。

この文書を読んでいる方は、

将来的には投資の知識が必要になると成功意識の高い方だと思います。

ですが、

投資に興味があるがどうしたらいいかわからない。
失敗してしまう可能性が高いから投資はやめておこう。

と思っているのではないでしょうか。

そのような方々に投資を身近に感じてもらいたくこのセミナーを開催します。

私も、投資を独学で始めたのですが、たくさんの失敗を重ね
そこから、コツコツと利益を出してきました。

ですが、投資を教えてくれる方が近くにいれば、余計な失敗をする必要はなかったことやアドバイスをくれる方が知り合いにいれば、より早く投資感覚を身につけることができたのではと思います。

これからの時代、投資感覚は現代人の必須スキルになってくると思います。

それは、投資を通して知識をつけることで、
今まで見えなかったものが見えるようになるからです。

それに、たくさんの書籍にも万人に投資感覚が必要な時代がやってくると書いてあります。

会社や国に守られ、文句を言わずに頑張っていれば報われる時代は終わりました。

だれもが、自分の財産は自分で守り、
自分の人生を自分で築く必要のある。

そんな時代です。

その時代をわくわくして過ごすのか、
びくびくして過ごすのか、

それは、あなた次第だと思います。

会社や国に文句を言ってるだけでは
自分の人生は守れません。

なので、若い時から、投資や不労所得について学ぶことは
今の時代とても大切だと思います。

これを学ぶか学ばないかで将来的に大きな差が開くのは確かです。

でも、いきなり投資を学ぶというと
とてもハードルが高く、難しく感じるかもしれません。

大丈夫です。安心して下さい。

いきなり株やFX、不動産について話し始めたりはしません。

まずはじめに、

私がバイブルにしている金持ち父さんの著者が開発したボードゲーム
「キャッシュフロー」をしていただきます。

これはモノポリーをより現実的にしたもので、遊び感覚で投資を学ぶ事ができます。

ゲームが初めての方も、投資未経験のかたも楽しく、和気あいあいとゲームをして頂きたいと思います。

そして、

なぜ２０代をメインに不労所得勉強会をするのか

私のポリシーとなる部分なので記載しておきます。

それは

今の若者に未来を変える力があると信じているからです。

教育制度はゆとり世代かも知れません。ですが、私たちは今の激動の時代を子供の頃から体感していること、そして様々な苦労を乗り越えて来た世代です。

そして

今の産業時代の負の遺産である社会システムや固定観点に疑問をもち

新しい価値観をニュートラルに受け入れられる世代です。

私は、今の時代を搾取される世の中だと思っております。

この傾向はますます強くなっていくでしょう。

国はなりふり構わず

取れる所から取っていくでしょう。

そうでもしなければ、国を維持する事ができないからです。

でも、それは私たちがお金に無頓着で
自分たちで世の中を変えることはできない。
自分には関係ない。
今生きるので精一杯だ。

そのような言い訳、無関心を続けたがために
結局、今の現状に流れついたなのではないでしょうか？

私は、投資を通して世の中を知ることや

リアルに感じることが

とても大切な事だと思っています。

その先に

未来を変えるための

正しい知識、洞察力が養われます。

そして、お金を得て

まず自分たちが豊かになり人生を楽しむことが必要だと思っています。

そうすることで、周りを助ける余裕。世の中を変える活力が生まれます。

私の知り合いにも、今の時代の在り方に疑問や矛盾を感じ
独自で動いてる方々も多いです。

ですが、私はみんなで力を合わせ

大きな夢を叶えられるような仲間つくりを
勉強会を通してつくっていきたいと思っています。

今の時代はとても恵まれています。自分の人生が豊かになると信じる人には
その可能性とチャンスが必ず訪れます。

このセミナーがあなたの人生を取り戻す一つのきっかけになれば幸いです。

あなたのご参加を心よりお待ちしています。

「未来は僕らの手の中に」
