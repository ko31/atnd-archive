description

【講座のポイント】
　情報技術を始めとする技術革新の進展によってディジタル製品は高度化し、複雑化する現状では、提供されるモノの機能と使用する生活者の身体の機能との間のギャップが拡大し、そのモノ本来の機能や価値が十分に発揮されなかったり、使用する側から見て使いにくいものが提供されたりする懸念も、さらに増大しています。
　これらの状況の中で、本講演で、使いやすいマンマシンインタフェースを提供するためのスマート化対応の画面仕様書について、短期間に、より信頼性が高く、効果的で、満足度の高い仕様書を作成できるようになります。誰にでも使い勝手のよい画面仕様書作成手順・ノウハウおよび色彩に関する効果的画面設計・アイコン作成について分かりやすく徹底説明いたします。更に、このスマート化対応タッチディスプレイ作成手順に基づき、アンドロイドによる位置情報サービスGUI の事例演習を行います。

【習得知識】
コンピュータの初心者、中級者が本セミナーの受講により、使い勝手の優れた、画面仕様書について、短期間に、より信頼性が高く、効果的で、満足度の高い仕様書を作成できるようになります。

【プログラム】
１．画面仕様の問題点と解決ノウハウ
　1-1.画面仕様の問題点
　1-2.画面情報の可視化による解決ノウハウ

２．画面仕様の設計・評価法
　2-1.ユーザインタフェース設計項目
　2-2.画面インタフェース設計方法

３．色彩に関する画面設計
　3-1.色相、明度、彩度など
　3-2.色彩に関する効果的画面設計法

４．アイコン設計法
　4-1.アイコンデザイン手法
　4-2.アイコン作成の手順

５．スマート化対応タッチディスプレイ画面仕様書作成法
　5-1.タッチディスプレイのメリットとデメリット
　5-2.タッチディスプレイの座標検出の仕組み
　　　a. 抵抗膜方式の座標検出の方法
　　　b. 投影型静電容量方式の座標検出の方法
　　　c. タッチスディスプレイ「iOS]の特徴

６．アンドロイドによる位置情報サービス事例
