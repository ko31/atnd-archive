◯インモビジャパン株式会社 ビジネスディベロップメント　統括部長　篠原好孝様

グローバルでモバイルアドネットワークを展開するInMobiの
日本支社でビジネスディベロップメント部門を担当。
パブリッシャー・デベロッパーの広告収益の最大化のため、
スマートフォン広告黎明期から、効果的な広告マネタイズの導入を提案、支援している。

◯MILLENNIAL MEDIA 株式会社　グローバルマネタイズソリューション事業部　プロデューサー 樋口隆文様

アドネットワーク専業ではアメリカ最大のMillennialmediaでローバルマネタイズソリューション部門所属。主に日本から世界でのマネタイズを担当。
リッチメディアの普及と、Millennialmediaのアメリカと日本での知名度のギャップを埋めるべく日々奮闘中。

◯グーグル株式会社 オンラインパートナーシップグループ モバイルビジネス　スペシャリスト 坂本 達夫様
モバイルビジネス ストラテジスト

2011年9月にグーグル株式会社に入社。日本における AdMob のビジネスデベロップメントを統括。セミナーの講師も多数経験。時々コスプレをして講演することでも有名。

◯ユナイテッド株式会社　取締役 兼 執行役員メディアカンパニー長 手嶋浩己様

・スマートフォンメディアを中心とするメディア事業を統括

・初の本格的グローバルサービスとして投入したのが、スマートフォンきせかえコミュニティ「CocoPPa」
