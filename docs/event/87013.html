★講演内容

【IoTスタートアップの聖地「DMM.make AKIBA」主催！IoTイベント第二弾の切り口は「開発言語」！！】

DMM.make AKIBAが、WebエンジニアさんのIoTに対する素朴な疑問や不安に答える
トークセッションの第二弾。
ご好評をいただいた第一弾に続き、
今回もWebエンジニアさんにとって敷居が高い（気がする）IoTについて、
最前線で活躍するWebエンジニア/ソフトウェアエンジニアの皆さんを交えて
トークセッションを行います。

今回のテーマは「開発言語」。
IoTを検索すると専用の開発ボードや複雑なセンサーの話が出てきます。
ですがその一方で、JavaScriptやPythonなど、
Webエンジニアの皆さんが扱う「いつもの言語」で開発する手法も増えています。
また「Web Bluetooth API」のような、ハードウェアとの親和性が高いブラウザAPIも
注目されています。

いつもの言語、いつもの技術でIoTに触れることができるなら、
Webエンジニアの皆さんにとってのIoTのハードルもどんどん下がるはず。
Webの視点からこれらの開発手法を紹介しながら、
皆さんからの「IoTを始めたいんですが何の言語を覚えれば良いんですか？」
という質問に対して「いつもの言語でOK」と答え続けるトークセッションを行います。

このイベントがWebエンジニアの皆さんが
「はじめの一歩」を踏み出すきっかけになりますように！

■DMM.make AKIBA エヴァンジェリスト
　岡島 康憲 氏 （モデレータ）

現在はエヴァンジェリストとして、外部への情報発信並びに事業企画の業務を担当。

■dotstudio株式会社 代表取締役CEO
　菅原 のびすけ 氏


1989年生まれ。岩手県立大学在籍時にITベンチャー企業の役員を務める。
同大学院を卒業後、株式会社LIGにWebエンジニアとして入社し、Web制作に携わる。 
2016年7月よりdotstudio株式会社を立ち上げ、
Developer Relations と IoT領域を中心に活動している。
日本最大規模のIoTコミュニティであるIoTLTや、
JavaScript RoboticsコミュニティNodeBotsの主催、
IoTバックエンドサービスであるMilkcocoaのエバンジェリストとしても活動中。

■株式会社Cerevo

2010年 芝浦工業大学大学院 修了。
2013年まで、自動車メーカー子会社にて車載ECUの電気回路設計を担当。
その後、株式会社Cerevoに入社。
回路設計を中心に、量産ライン、製品認証、デモ用ソフトウェア等、
