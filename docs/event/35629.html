description

スターケバブのケバブをまとめ買いするよ！
参加するときは当日食べたい個数と持ち帰りたい材料の個数をコメントに残してね！

どんな料理かイメージ出来ないって人は、ここから買う予定だから写真とか参考にするといいかも。あるいはググるべし。欧州の方では割とメジャーなファーストフードで、ハンバーガーと同じくらいのノリで構えてれば大体合ってると思う。

費用

その場で食べる場合 1個あたり300円
材料(肉(100g)+パン+ソース)を持ち帰る場合 1個あたり 200円 300円

参加登録について

ある程度必要な量が見積もれると助かるから、このくらい食べるよ！って分かってる人はアンケート欄のチェックを入れておいてね！
（アンケート欄は回答しなくても良いけど、回答しなかった勢は当日足りなくなっても文句言わないように）

持ち帰りの注意事項

冷凍食品だ！保冷バッグとか用意しないと持ち帰ってる間にデロデロになってしまうかもしれない。あとソースを小分けするボトルも持参でよろしく。

冷凍肉は2食分づつパックされてるけど凍ったままだと切り分けられないので材料または肉だけ持ち帰る場合の個数は偶数でお願いします。
あと大量にまとめたパックの方がお得なので、俺は4食とか10食とかまとめて凍ってるやつでいいぜ、とかだと少し買えるかも。
冷凍肉とパンは電子レンジで温めるかフライパンで焼くといい感じになる。
