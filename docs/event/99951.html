マーケティングの実態と具体的な活用方法と効果的な集客方法を知る2本立てセミナーです。

前半は マーケティングの役割と効果・活用例を解説、後半は 前半を踏まえて 具体的なマーケテイングツール「Googleマイビジネス」を使って何ができるかを 簡単に話します。

　

参加方法

Facebookイベントページで「参加予定」にしてください

イベント概要

１９５０年代半ばに、アメリカから日本に導入されたマーケティング。それは生産者主導→販売者主導→消費者主導 へと変化しながら今に至ります。現在 消費者主導のマーケティングは 物売りの便利な考えから、人が望むものを提供し自分自身が 世の中に役だっていることを実感しつつ 対価を得るための 考え方に変化しているといっても過言ではありません。そこで 今回 以下の内容をお届けします。このセミナーに参加すると、あなたの事業や仕事ですぐに活用できる”技”を持ち帰ることができます！・・・多分。

講師概要

永野 浩（兄）

  • 1992年 松山大学卒業後に上京、コンピュータ企業でシステムエンジニアを5年
  • 1997年 日本オラクル株式会社で3年間 マーケティングを担当、子会社設立メンバー担当
  • 2000年 ミラクル・リナックス株式会社設立し 約6年間を 経営・バックオフィス・マーケティングを兼務
  • 2006年 日本マイクロソフト株式会社にて 8年間を 主にWindows製品のマーケティング業務に携わる
  • 2014年 調理師免許取得後、飲食業界に転進、東京と徳島にて複数の飲食店で調理業務に携わる
  • 2017年 8月に松山帰郷、現在 飲食店開業準備中

永野 護（弟）

  • 2011年　携帯アフィリエイトを開始
  • 2013年　SNSに特化したアフィリエイトに注力
  • 2014年　塾生約300人のSNSアフィリエイト塾の講師を務める
  • 2016年　1年で 2000人を超えるアフィリエイト塾でのメイン講師を担当。渋谷・御茶ノ水等で300名を超える勉強会に登壇
  • 2018年　4月 松山市議会議員選挙にてWeb選挙参謀を担当。5000票以上を獲得し4位当選に大きく貢献
  • 現　在　Googleなどの検索エンジンからの流入等で売上、アフィリエイターの中でもSEOとSNSの両方で売上を持つ非常に珍しいポジションを築き、松山市内の企業や団体にSNSをメインとした集客のアドバイスや指導も行っている
