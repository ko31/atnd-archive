趣旨

紅葉 (ちと早いかも) の中で温泉に浸かりながら、泊まり込みでとことん DNS を学びましょう。インターネットシミュレータ (VITOCHA) 上で DNS サーバを動かしながら動作の理解を進めます。特にいま一部で話題の DNS キャッシュポイズニングについて重点的に学びましょう。参加者からのライトニングトーク (時間延長可) も歓迎します。

会場

  • 中京大学人工知能高等研究所
    • 名鉄 浄水駅よりスクールバスあり ( Cダイヤ )、愛知環状鉄道 貝津駅からは徒歩8分
  • 猿投温泉金泉閣

会費

  • 宿泊参加: 15,000円
    (夕朝食+宿泊(4人/1室)+酒 / 飲まない人と学生は割引で13,000円)
  • 中京大のみ参加: 0円
    (日曜は通学バスがないため最寄り駅まで徒歩もしくはタクシー料金が必要)
  • 猿投温泉でのセッションは原則宿泊参加でお願いします (二日目別行動は可)
  • 2日目昼食は学食休みのため出前などで別途精算します
