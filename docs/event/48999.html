JVM Operation Casual Talks

普段Webアプリケーションを書く言語がJavaの人もそうでない人も、多種多様なミドルウェアがJVM上で動作する昨今、JVMそのものの運用(モニタリング・チューニング)をどうしているか、どうすればいいのか、ということに頭を悩ませている人は多いのではないでしょうか。

で、今回はそういった話題について、普段どうしているか、どうすべきなのか、どうしたいのかなどをカジュアルに語りあいたいと思います。お気楽な感じです。

Ustream中継はありません。
