description

「アジャイルサムライ ― 達人開発者への道」 の読書会を行います。
順番に輪読していき、セクションごとに本の内容についてディスカッションする形式ですすめていきます。
本も半分が過ぎ、プロジェクト運営など実践的な項目についての章が増えてきました。現場でアジャイルの考え方をどのように活かすことができるか、考えていきましょう:D

内容・範囲

輪読形式での読み合わせを行います。
読み合わせ後、セクションごとに参加者で疑問や意見を話し合っていく形式を想定しています。

第15回は、以下の範囲を予定しています。

  • 「第9章 イテレーションの運営：実現させる 9.5 ステップ2：開発：作業する」～

スケジュール

時間
19:00 – 19:10 自己紹介
19:10 – 20:00 読書会（第9章 9.5～）
20:00 – 20:10 休憩
20:30 – 21:00 読書会（続き）
21:00 – 21:10 撤収作業

※時間は目安であり、休憩時間が前後することがあります
