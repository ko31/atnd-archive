イベント概要

3/18 – 3/19 に US で開催のイベント dotNetConf 2015 を受けて、その日本版となる dotNetConf 2015 Japan を 4/11 (土) に品川で開催します。

.NET Core 5 と ASP.NET 5 の技術セッションに加えて、C# 6 と Roslyn に関したディープなセッションとパネルディスカッションなどを通して、.NET 2015 の最新情報がキャッチアップできる 1 Day オフラインイベントです。

また、同日にMS品川オフィスのセミナールーム A, B で開催となる Japan Xamarin User Group Conference (JXUG) との共同開催イベントにもなります。

こちらの dotNetConf Japan と JXUG Conference のどちらに参加登録しても、すべてのセッションに参加可能となります。
