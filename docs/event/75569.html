ＮＰＯ法人日本交渉協会認定トレーナーの心理交渉トレーニング 【質問術の回】

ほとんどの日本人は交渉が苦手！

私たちの暮らす日本では、「協調性」や「調和」を重んじており、争いごとを避ける傾向にあります。交渉そのものを避けてきたことで、世界的にみても「交渉下手」と言われています。

現在の日本では、価値観がどんどん多様化し、人と人が分かり合うためには「交渉力」が必要不可欠になってきました。

営業先や取引先、上司や部下とのコミュニケーションで、

「自分の意見を通したは良いが、相手をうまくコントロールできない。」
「相手の希望ばかり聞いてしまい、自分だけ損をした。」

という経験はないでしょうか。

ほとんどの日本人は交渉が苦手ですから、このような心理衝突は当たり前なのです。

交渉≠シビア

交渉のコツさえ覚えてしまえば、決して難しい話しではありません。
交渉時に必要な準備、話しももって行き方、、など交渉のポイントを抑えることで、 あなたも交渉上手になれます。

リアル交渉（脳）トレーニングでは、実際にロールプレイングをし、 交渉アナリスト認定トレーナーがレビューを行うので、交渉のコツを身につけやすいです。

効果的なトレーニングを続けて、あなたも交渉上手になりませんか！
