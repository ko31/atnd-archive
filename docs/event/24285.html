description

２０１２年も、ボランティに行きたいと思います！

行こうと思ってたけどまだ行けてない人、 是非一緒にいきましょう。

『支援したいけど何ができるかわからない』
『行っても邪魔になりそう』

と支援したくても今一歩踏み出せない方は多くいると思います

その思い無駄にしたくない・・・

とはいっても一人で参加するには東京から宮城だとしたら
高速代で片道１万円近くかかりガソリン代で
片道１万５千円程度はかかり往復すると数万円かかってしまう・・・

東京発のボランティアは一泊だし仕事が・・・・

しかし日帰りでもみんなで車で行けば割り勘で安く済みます

支援活動をしたくてもできなかった方、是非ご参加ください

ガソリンスタンドやコンビニなどは営業再開しています
食事や飲料もちゃんと売ってますのでご心配なく

支援活動の内容は、ボランティアセンターに依頼があった住宅の瓦礫の除去や
お家の解体、畑作業など様々なものがあります
いずれも初めての人でも出来る作業です！
現地状況では人手がいくらあっても足りません

現地には日本人、外国人、老若男女問わず全国から気持ちだけで集った
ボランティアが数多くいます

宮城県の放射線量は東京よりも低いくらいです。

念のためガイガーカウンターも持って行きます。

ボランティア詳細

スケジュール

●１月２８日（土）
２４：００集合
２４：３０分出発
高田馬場駅ＢＩＧＢＯＸ前のロータリー真ん中の広場集合です。

●１月２９日（日）
０８：３０津波復興支援センター仙台到着　作業開始
１５：００頃作業終了
１６：００仙頃台出発
１７：００頃温泉予定
２３：００頃高田馬場到着(道路状況により前後します)

※日帰りです

持ち物

道具、軍手、マスクはこちらで用意します。着替えやタオルは持参してください。
帰りに温泉に寄ってきますので、バスタオルがあってもいいかもしれません。
(レンタルで借りることもできます)
服装は汚れてもよい服、靴（長靴が好ましい）で来てください。
ボランティア保険（天災型）は現地で入れます。

参加費

６０００円（ガソリン代、高速代、軍手、マスクなど備品）
現在は車１台の予定です。８名まで参加できます。
もし足りない分はみなさんからいただいた募金から、
もしくはこちらで負担しますので追加で費用がかかることはありません。
また余りが出た場合は募金もしくは物資として支援させていただきます。

定員

８名（車出せる方いれば定員が増えます）
男女は問いません。
車出せる方募集中です。その場合、定員が増えます。
現地のボランティアセンターに団体登録してニーズに合わせてもらうので
現地でバラバラにならず行った人数でまとまって作業します。
