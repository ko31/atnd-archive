1.健康食品販売

【講演主旨】
法令遵守を意識した健康食品事業の進め方をレクチャーします。主な対象は①健康食品の広告表現について分からない方、②健康食品と薬事法の関係について分からない方、③健康食品の広告表現の考え方の根拠を明確に知りたい方となります。それ以外でも本セミナーに関心のある方に対して実務を行っていくうえで参考となる情報をできる限り、平易に解説いたします。

【プログラム】

1.健康食品の定義
　1.1　医薬品
　1.2 医薬部外品
　1.3　栄養機能食品
　1.4　特定保健用食品との比較

2.法令等が危惧する健康食品の弊害
　2.1　薬事法
　2.2　厚生労働省通知より理解する

3.法令等に合致する健康食品の判定方法
　3.1　明らか食品
　3.2　本質
　3.3　表示される効能効果
　3.4　形状
　3.5　用法用量

4.医薬品等の広告の該当性基準

5.実際の広告表現事例で理解を深める

6.健康食品事業に関係する主な法律
　6.1　薬事法
　6.2　食品衛生法
　6.3　JAS法
　6.4　健康増進法
　6.5　景品表示法
　6.6　特定商取引法
