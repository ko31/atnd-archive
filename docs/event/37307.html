世の中は街コン流行り！そこで「プログラムコンパイベント」を開催します。3月16日（土）10時、御茶ノ水に集合！

御茶ノ水駅から徒歩３分、とても交通の便がよく、近くに大学もあるアカデミックな環境の中、無線LANも使える綺麗な教室でHTML5のプログラム勉強会を開催します。

HTML5でプログラム？

HTML5はWEBアプリケーションのプラットフォームです。今回はcanvasタグとJavaScriptのプログラムを使って動くコンテンツを作りましょう。サンプルはこちら 「春の夜の櫻」

参加条件はプログラマ？

いいえ、むしろ初心者向けです。IT男子や、Web女子、ネット関連なら、企画、運営、カスタマー。異業種でも「HTML5」ピンときたシングルな方でしたらご参加いただけます！初心者向けですのでネットサーフィンができて、メールを打ち英文入力に抵抗がなければ特にHTMLやプログラムの知識は不要です。
社会人常識をお持ちで、面倒なトラブルを起こさず、ドタキャンしない方であることは言うまでもありません。
