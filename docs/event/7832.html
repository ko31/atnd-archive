description

近頃、拡張現実(AR)が徐々に企業のプロモーションキャンペーンや展示会などのイベント取り入れられるようになってきました。

おそらく皆さんも一度は聞いたことがあると思います。

そこで、PR会社が考えるARの利用形態について考える機会を設けました。

「ARって何？」というところから事例を紹介し、皆さんの企画に役立てることを目的にしています。
もちろん、PR会社以外の参加者も大歓迎です！←ココ大事です！！

大事なことなので2回書いておきます。

PR会社以外の参加者も大歓迎です！

講師として、ある時は流れのUstreamerとしてイベントの中継と実況をこなし、ある時はARの講師までやってしまうハチヨンイズム副編集長江口晋太朗氏(@eshintaro)をお招きします！

＊登壇調整を行っていたAR三兄弟さんですが、諸事情によりスケジュール調整が困難となりましたので、またの機会に調整をしたいと思います。期待されていた方、申し訳ございません。(9/13時点)

■プログラム
・ ごあいさつ、ゲスト紹介
・ ゲストによる講義(デモを予定)
・ 質疑応答
・ 参加者同士のアイスブレーキングを兼ねた簡単なワークショップ（ARを取り入れた簡単な企画ブレスト⇒発表)

皆さまの参加をお待ちしてます！
