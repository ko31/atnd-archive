description

てづくり電気自動車教室ではEVガイドラインを網羅するための警報装置の設置方法や異常警告灯の配線など、今後のEV事業を考える上での必要な改造方法やナンバー取得に向けた申請方法を学んでいただきます。また自動車関連企業の方・家庭用蓄電池として改造ＥＶに興味のある方などの参加が多くおられます。

EVhondaでは、全国に改造EVを普及する上で改造ＥＶ製作の指導と的確なアドバイスのできる人材育成としてEVアドバイザー研修会も同時に行なっています。

「てづくり電気自動車教室」でコンバートEV（改造ＥＶ）製作するのは生徒だけではなくEVアドバイザーも一緒に学ぶ、EV事業に実際に取り組んでいるEVアドバイザーからEVビジネスへの取り組み方などを聞くことも可能。
