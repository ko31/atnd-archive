クラフト・ハンドメイドビジネス 創業セミナー

伝統工芸、工芸製品、ハンドメイドの製品などをビジネスに繋げようと考えている方のためのセミナーです。

開催日時

【第1回】　10月17日（月）　18:30 ～ 20:00

【第2回】　11月7日（月）　18:30 ～ 20:00

料金

無料

場所

西陣産業創造會舘3階（カンファレンスルーム）

講演概要

第1回

  • 講師
    • 唐紙師・かみ添代表 嘉戸 浩(かどこう)氏
  • 概要
    • 伝統工芸に新たな価値を見いだすものづくりからの起業体験者のお話を伺い、新たな気付きを得て、ビジネスの創出へと繋げていただくためのセミナーです。

第2回

  • 講師
    • 唐紙師・かみ添代表 嘉戸 浩(かどこう)氏
    • 染織家 染司よしおか 吉岡 更紗(よしおか さらさ)氏
  • 概要
    • かみ添の嘉戸浩さんと染司よしおかの吉岡更紗さんは、同じ年代で「物を作ることを生業としている」若手です。今までのこと、これからの事を対談形式でお話しいただきます。

お申込み
