O’Reillyより刊行されている「集合知（collective intelligence）」をテーマにした集合知プログラミングの勉強会です。

ブームに乗り遅れたけど、なんとなく読みたい方、Pythonコードに興味がある方などを対象にしています。
また、本書にはPythonコードも満載ですので、現在の文法規則に則ったコードレビューなどもできればと考えています。
