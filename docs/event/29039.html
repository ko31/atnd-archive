浜松もやりますよ！ Titanium Mobile 勉強会

Appcelerator Inc. の増井さんをお招きして、Appcelerator Cloud Service のお話と、それを使ってライブコーディングをしていただきます。話題の ACS について質問できるチャンスです！

Titanium Mobile の初心者さんから上級者の方まで、この機会にぜひお越しください。

※ Titanium Mobile とは iOS/Android のアプリを Objective-C や JAVA を書かずに、JavaScript で開発できてしまう素敵なソフトウェアです
