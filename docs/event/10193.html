description

【概要】
会議室やイベント会場の様子を読み取り
twitterにつぶやく。
そんなリアルbotづくりを
やってみます。

【内容】
実環境にセンサーを組み込んで、
人の動きに対して反応しtwitterに勝手につぶやくような
マシンを作って行く予定です。

部屋に出入りする人数をカウントし
その増減時間変化などで盛況っぷりをツイートする
カウンターをつくる構想がひとつあります。

応用すれば、
・まもなく席がうまりそうなう。
・○○のトイレ満室なう。
のようなこともできるかもしれません。

アイデア出し、ラフ設計、パーツ集めから、プロトタイプ作成まで
夕方までをめどに進める予定です。

CO-WORKSHOPとは？
講師が内容を考え枠組みを用意し、その上で参加者がワークするスタイルのワークショップは準備が大変だったりします。一方、フワフワとしたコンセプトのワークショップというのは、もしかしたら参加者になにも提供できずに終わってしまう可能性から大っぴらに開催しにくかったりします。

「けど、みんなでアイデア出し合ったら面白くなるかもしれないのに、やらないのはもったいないよね。フワフワとした思いつきをしっかりとしたワークショッププログラムにするのはちょっとハードルが高い。だったら最初っから、みんなでワークショップの内容を作ってくんだって宣言しちゃって、声かけてみたらいいんじゃない？」

CO-WORKSHOPはそんな経緯で生まれた、"みんなでつくる"ワークショップです。お題に対して、それをオモシロくするアイデアを参加者が出しながら内容を決めていきます。なにができあがるかはやってみてのお楽しみです。ワークショップのプログラム自体がインプロビゼーション、さあやってみましょう。
