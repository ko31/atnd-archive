Lagopusユーザ会

本BoFでは、気軽に触れる Open Source SoftwareのOpenFlow Switch 実装である"Lagopus"を中心に、 Software-Defined Networking / OpenFlow周辺技術について語り合いましょう。

Lagopusに興味あるかたはもちろん、 Lagopusに限らずOpenFlowに関連した話題に興味あるかたの参加お待ちしております。
