森タワー入館の都合上、名簿の作成が必須となります！ATNDの登録と合わせてこちらのアンケートのご記入をお願いいたします！
締め切り：7/5 17:00（すみません、7/7 8:30で締め切りました！）

大変お待たせいたしました。直前となってしまい大変申し訳ありませんが、名簿作成のアンケートを開始いたしました。ご協力お願いいたします！

名簿作成のアンケートとあわせて興味のある章と、「自分なりにまとめてみた」方の募集を開始いたしました。
時間は5分〜15分程度を想定しております。
（1章丸ごとでも、各パートでもかまいません）

今回はこちらの本の読書会を行います。
内容・形式などは現在練っているところです。が、各章毎に担当をもうけてコメントを交えながら読んでいければと考えております。
（ご意見等あればコメントを募集しております）

進め方

進め方としては各種テーブルにわかれて各章を

  1. （あれば）自分なりにまとめてみた（全体発表）
  2. （軽く）テーブルにて自己紹介
  3. 声に出して読む（各種テーブル挙手制）
  4. 気になるところを挙げてみる
  5. 席替えタイム
    と進めてみたいと思います。

全て読み切れるとは思いませんが、発表者の有無と、アンケートの要望順で決めていきたいと考えております。

入館の仕方

受付は六本木ヒルズ森タワー(ヒルズだとみんなに思われてる一番高いビル)の正面受付の近く（スターバックスなどがあるところ）です。
13:00より受け付け開始いたします（終了14:00）

受付後、14Fまでご案内いたします。

遅刻された方はtwitterにて@s_woolへ連絡、
または080-3359-1117までお電話ください

最後に

大規模な読書会は初めてでやり方を模索しながらで、ずいぶんざっくりとした進め方となってしまいましたが、楽しく読んでいければと思います。
よろしくお願いいたします！
