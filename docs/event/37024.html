第2回Web APIで学ぶPHP・JavaScript勉強会

イベント ユーザー
event_id 37024
title 第2回Web APIで学ぶPHP・JavaScript勉強会
catch 気候の良い季節になりお出かけする機会も増えます。
description

人気のWeb APIを利用して見た目で楽しく、みるみる成果がわかるPHPやJavaScriptの文法を勉強する会を企画しました。
これまである程度Web API＋PHP・JavaScriptで何かを作る勉強会を繰り返してきましたが、少し文法などに重点を置いて、表現力を拡げたり、ユーザーにとってメリットのあるアプリ作りを目指していく勉強会として2013年新年の始まりとともにスタートしました

※IT勉強会スタンプラリーのスタンプに対応!

■今回のテーマ
・利用するWeb API：YOLP(Yahoo!の地図)、YOLPカセットroロケタッチスポット、その他少し増えるかも知れません
・今回のメイン言語：JavaScript,PHP

気候の良い季節になりお出かけする機会も増えます。
そんなときにはどこかに出かけるという場所のことを考えます。その場所や周辺地図のことを取り上げていきたいと思います。

■参加者対象者(参加に必要なスキル)
・HTML,CSS,JavaScriptをコピペベースで構わないのでテキストエディタで編集して表示確認ができること
・HTML,CSSを編集して表示内容を変えたり色やレイアウトを変えるなど簡単な編集作業ができること
・レンタルサーバーを契約しアップロードしてページを公開できること。勉強会ではロリポップ（コロリポプラン月額105円）で動作を確認したものを用意しています。
・ローカル開発環境をご用意いただけると尚良いのですが、難しければレンタルサーバーにアップしての動作確認も今回は可とします。

■参加費用
無料。ただし、会場のみでの開催でUstream中継などはありません。
会場での開催なのでまわりの方に聞いたり、画面を見せ合いながら勉強して行くことができます。
会場を無料でお借りしている関係上、会場でお世話になるマルチメディアスクールWAVE様へのfacebook「いいね」＆チェックインなどをお願いしております。

・名刺2枚（1枚は名札に使用します）
