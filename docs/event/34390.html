Delphi Advent Calendar 2012

技術系エンジニアの間で毎年開催されている Adevent Calendar ですが、Delphi では今まで開催されていません！
それを今年こそ開催したいと思います。
Delphi ユーザーも年末のお祭りに参加しましょう！

Advent Calendar とは

技術系のアドベント・カレンダーとは、12/1 ～ 12/25 日まで、１日１人が各々のブログやウェブページに、技術やテクニックを紹介するイベントです。
もちろん、ここでは Delphi の Advent Calendar ですので、VCL や FireMonkey の使い方や、IDE の小技とか、Delphi に関わる話なら何でも構いません！

基本的には１日１人が担当しますが、人数が満たない場合は、１人が複数の日にちを担当しても構いません。

参加方法は？

  1. ATND にログインする
  2. このイベントに参加する
  3. 下記の情報をコメントするか #delphiadvent2012 を付けてツイートしてください
    • 名前（ツイッターのユーザー名などでＯＫ）
    • 名前のリンク先（ブログやツイッターなど）
    • 担当したい日付
      • 参加したい日にちが被った場合、先着とします。
    • タイトル
      • とりあえず未定でも構いません
  4. 参加表明した日になったら、ブログを更新します
    • 更新した旨をコメントするか #delphiadvent2012 を付けてツイートしてください

参加したい日にちのコメント、および、ブログが更新された場合、随時タイムテーブルを修正します。

Delphi のユーザーであっても、そうでなくても Delphi に興味があれば、どなたでも自由に参加していただけます。
