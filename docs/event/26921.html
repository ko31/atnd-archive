チュートリアルパート (10:30-12:00)

enchant.js 初心者向けのチュートリアルです。基礎から始めて、enchant.js を用いた用いたゲーム開発のオーバービューを提供します。

  • ノートパソコンをお持ちください
  • ブラウザは Google Chrome 推奨です
  • コンセントは人数分用意できません。十分充電した上でお越しください
  • プログラミングが初めての方や不安な方は、 公式サイトのリファレンス や、 ドットインストールの3分動画レッスン 、 書籍「ゼロからはじめるenchant.js入門」 での予習をおすすめします。

セッションパート (13:00-16:00)

  • セッション×3
    • 伏見遼平 (enchant.js 開発チームリーダー)「enchant.js でゲームを作るコツ」
    • 高橋諒 (gl.enchant.js 開発者) 「gl.enchant.js で始める WebGL 3Dプログラミング」
    • 清水亮 (UEI代表取締役社長兼CEO) 「9leap ＆ enchant.js」ロードマップ
  • LT
    • 「animation.enchant.js のつかいかた」by 開発チーム
    • 「それ、CoffeeScriptで簡単にかけるよ(仮)」 by hidesukeさん (@hidesuke)
    • 「gl.enchant.jsモデルデータをラクラク作成」 by 楢崎大志さん (@daishi_hmr)
    • (未定)
    • 登壇者募集中です！
  • 9 Minutes Coding Battle
    • enchant.js を使って、9分間でゲームを作るコーディングバトル
    • 会場の投票で勝敗を決します
    • 挑戦者募集中!

ハッカソンパート (16:00-18:00)

  • enchant.js 関連のイベントでは恒例の、当日その場でゲームを作るハッカソンです。
  • 今回は初めての試み「90 Minutes Coding Battle」としてコーディングバトルを行い、優勝者には豪華賞品を贈呈します!!
  • 時間内にゲームが完成した方は、壇上で発表していただきます。講評もあります。
  • バトルといっても、講師や発表者に質問しながら開発できます。初めての方も気軽にご参加ください。また、周りに困っているユーザがいたら助けあいましょう。

ライトニングトーク登壇者募集中!

制限時間5分です。あなたの自慢のハックや、開発したプラグイン、プログラミングの手法などを紹介してください。5〜6枠募集中です。応募はコメントに書くか、twitterにて @enchantjs_ja に連絡ください。

備考

  • チュートリアルからハッカソンまでフル参加の方は、お持ちのノートパソコンの電池が切れてしまうかもしれません。極力、座席の近くにタップがあるよう努力しますが、電源タップを人数分用意することはできませんので、各自でタップをお持ちいただければ確実です。
  • また、周りに電源がなく困っている人がいたら各自協力して乗り切りましょう!
