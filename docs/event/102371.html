AEviewerとFont Managerの開発者、Alex White (アレックス・ホワイト)に会いましょう。

アレックスの開発する AEviewer を利用することで日々のルーティーンワーク整理したり、日常の作業を改善することができます。 Font Manager. を利用することで、アニメーションフォントの制作が可能になります。また、AEviewerのテンプレート活用方法やAfter Effectsのスクリプトやエクステンションを活用したワークフローの改善についてのアイデア紹介やディスカッションを予定しています。

講演概要

  • インストール方法
  • aepの準備と活用方法
  • ビデオ/オーディオ/グラフィックファイルのプレビュー
  • フォントのプレビューとカスタマイズ
  • スクリプトの実行
  • お気に入りやコレクションファイルの整理方法
  • Font Managerとの統合方法
  • 独自スクリプトの作成と活用に関するワークフロー
  • 独自スクリプトのテンプレート化
    • AEviewerのテンプレートのパッケージ化
    • パッケージ化したテンプレートのインストール方法
