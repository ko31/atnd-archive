概要

北海道の Drupal ユーザーコミュニティ「Drupal さっぽろ」が主催する meetup (勉強会) です。地域の Drupal ユーザーの情報交換と交流を目的に、月に一度、札幌と小樽で開催しています。今回は札幌での開催です。

開催予定は ATND でご案内、Google グループ、Facebook、Twitter を通じて告知します。
https://groups.google.com/forum/?hl=ja#!forum/drupal-sapporo
https://www.facebook.com/drupalsapporo
https://twitter.com/drupalsapporo

北海道の Drupal コミュニティとして情報交換と交流の輪を広げ、さらなる活用につなげていきましょう。

参加料：500円（会場費として）

内容予定

持ち寄りセッション

常時募集中です。発表テーマをお持ちの方はお知らせください。随時このページに追加・更新していきます。今回のテーマは JavaScript なので、各種 JS ライブラリのモジュール使用事例などは特に大歓迎です。

勉強会：テーマ「Drupal と JavaScript」

今回は Drupal で JavaScript を使うための基礎知識を整理したいと思います。サイトに .js ファイルを組み込む方法、Drupal の JavaScript API の概要などを確認した後、JS ライブラリの使用例として isotope で Views の出力をグリッド表示するサンプルを作ってみましょう。

懇親会

終了後は近隣の飲食店で懇親会を予定しています。（要実費）
こちらも是非お気軽にご参加ください。
