納涼もんご祭りでMongoDBチューニンガソンを開催します！

MongoDB JPが主催するMongoDBのイベント「納涼もんご祭り」でプチ・チューニンガソンを開催します。
AWS上のMongoDBで負荷ツールによる性能評価でハイスコアを競います。

納涼もんご祭り公式ページ



チューニンガソンとは

チューニンガソンとは、MongoDBのベンチマークを競うイベントです。運営が作成したベンチマークサーバーが、あなたのMongoDBインスタンスに定期的にベンチマークを実行しスコアを計測します。
MongoDBインスタンスはAWSインスタンスとして参加者に配布されます。スコアは会場のスクリーンに映し出されて定期的に更新されるので、自分の順位がリアルタイムでわかります。

MongoDBのボトルネックを探し出し、チューニングを施してハイスコアを目指しましょう！

内容（情報は随時更新していきます）

・予め用意されたデータベースへのクエリースピードを競う
・具体的な評価手法
　　query test server を運営側で準備、そこからクエリをDB直接投げクエリスピードを集計
・環境
　　aws medium ianstance 1人1台
