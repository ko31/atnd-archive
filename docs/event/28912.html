【受付は終了いたしました】
Webデザイナーやアプリ開発者を対象に、Windows PhoneとMetro UIをテーマとした
無料セミナー「Windows 8を見越したWindows Phoneアプリ開発」を5月24日（木）に開催します。
※サイト構築のためのトータルデザイン誌『Web Designing』の連載「Jump into Metro!」と
連動したセミナーです。

「Windows 8」でメインのアプリケーション環境となる「Metroスタイル」を先取りできる機会です。
モバイルアプリ開発を行われている方、Metroデザインに興味がある方、お気軽にお越しください。

※詳細は サイト をご参照下さい。

>>セミナー内容———————————————————————————————

*Metroスタイルのインターフェイスデザインの解説

*Metroスタイルに則ったネイティブなWindows Phoneアプリケーションの開発
　UI 構築やインタラクションデザインを設定することができる
　「Microsoft Expression Blend」を、オーサリングツールとして使う
　開発手法を解説します。

*開発者向けの専用サイト『App Hub』の登録からアプリケーションの申請手順
