description

わずか1時間でご自身で撮影した写真からオリジナルメッセージカードを作り、
はじめてのデザインを現役クリエイターから学んでいただけます。
お子様も一緒に体験できますのでぜひお連れください。

IllustratorやPhotoshopを使ってオリジナルメッセージカードの作成ににチャレンジ。
グラフィックデザインのプロがわかりやすく操作を説明、
自分で撮った写真やお気に入りの素材を使って、目を引くデザインのポイントを
伝授させて頂きます。

メッセージカードの作成に興味のある方、「illustrator」や「Photoshop」等、
実際にプロのデザイナーが使うソフトを体験してみたい方等のご参加をお待ちしています。
新しくOPENしたSTUDIO京都の雰囲気を是非楽しんで下さい！
