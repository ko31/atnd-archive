description

このセミナーでは失敗や成功を経験したからこそわかる「大切なこと」を皆様にお伝えしていきます。

参加者様限定！！大好評の「お楽しみ抽選会」がセミナーの最後にあります。
※商品券が当たりますのでお楽しみに♪

◆参加者特典
①商品券が当たる抽選会がございます
②参加者全員に軽食をご準備させて頂きます
③また、本セミナーでは講義内容を更に深めていただくために、個別相談も承っておりますので、ぜひご活用ください

◆セミナー内容
このセミナーはこんな人にオススメです！
・将来のために何か始めたいけど何をしたらいいのかわからない。
・不動産投資には興味があるけど一歩踏み出せない。
・周りに成功している人や相談できる人がいない。
・融資が受けられるのかわからない。
・家族の理解が得られない。
この中で一つでも当てはまる人は要チェック！！

不動産投資を始めるには様々な壁があります。
私はその壁を越え不動産にまつわる様々なトラブルを経験しながら…
投資歴5年で資産2億円まで増やす事が出来ました。

◆当日のプログラム
投資歴5年で資産2億円になった女性投資家。
フリーアナウンサーとして仕事をする傍ら5年間で様々な投資を行う。
①総資産２億円までの道のり
②実際に経験した失敗＆トラブル事例
③失敗から学んだ法則

セミナー後は、お楽しみ抽選会、講師を囲んだ懇親会もございますのでお楽しみに♪

◆講師プロフィール
chisato
女性投資家／元フリーアナウンサー

フリーアナウンサーとして仕事をする傍ら５年前から投資を行う。
不動産ではアパートを２棟１５室所有し、太陽光は３基所有。
またブランド品の販売も行う。
2019年１月より株式会社T＆Cの代表取締役社長に就任。
「不動産賃貸経営博士」のコラムニストとしても活躍中。
