description

全国で防災教育の必要性が叫ばれる一方、学校現場では時間や経験の不足など普及に向けての課題が山積しています。
そんな中、いつ起きるかわからない災害に備えるために「今すぐできる防災教育」への注目が高まっています。
EDUPEDIAに掲載の防災教育実践を通じたワークショップ等を通じて「今すぐできる防災教育」を体験し、これからの防災教育について考えてみませんか。
