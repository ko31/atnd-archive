description

公的資格・民間資格を問わず、人事評価や採用などの業務知識を体系化して付与し、専門家としてスキルを証明する資格をこれまで見つけることができましたか？
人材測定コンサルタント®は、わが国初ともいえる人事評価や採用業務の担当者様向けのスペシャリスト資格で、特定非営利活動法人 人事コンサルタント協会が認定する名称独占資格です。資格取得後は、社内外で通用する専門家としてスキルアップが期待できます。人事担当者様だけでなく、人材ビジネスのコーディネータ様やこれから就職をする学生様にもおすすめの資格です。
今回のセミナーを通じ、ぜひ資格の魅力に触れてみてください。

＜プログラム＞　※内容は予告なく、多少変更する場合がございます。
１．人材測定コンサルタント®資格とは
　（１）今、企業が人事担当者の専門資格を求めている
　（２）資格取得のために何を学ぶのか？資格取得のメリットとは？
２．人事評価制度運用のむつかしさ
　（１）今、企業の現場で起こっていること
　（２）企業戦略と人事制度
３．激変する採用環境への対応
　（１）人口動態と採用
　（２）企業の戦略ステージと人材像の変化
４．人事担当者に必要なスキルセット
　（１）行動科学、組織心理学
　（２）人事マネジメントの全体像

※セミナー終了後、個別相談会を開催します（予約不要）

＜対象＞　法人・団体の人事担当者様、人材ビジネスのコーディネータ様、学生様

＜参加費＞　無料

＜期待できる効果＞　採用や人事評価に必要な知識・技能を深化させることで、組織内だけでなく、組織外でも通用する“頼られるプロ”として、一層のプレゼンス向上が期待できます。

＜特典＞　認定講座受講料20％OFF券進呈

【大阪・名古屋開催あり】2月24日（金）大阪会場、2月20日（月）名古屋会場での開催も予定しております。

＜講師＞
特定非営利活動法人　人事コンサルタント協会理事長　各務　晶久

【経歴】
　同志社大学卒、関西学院大学大学院修了。大学卒業後、川崎重工業株式会社を経て日本総合研究所の主任研究員に転身。人事制度設計、組織風土改革、研修制度設計、事業計画策定等のコンサルティングに従事。その後独立して株式会社グローディアの代表取締役に就任。2015年に特定非営利活動法人　人事コンサルタント協会理事長に就任。
