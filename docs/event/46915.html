【基調講演】
世界で注目すべきデジタルイノベイター50人に選出された水口 哲也氏
が語るマーケティングを変革するゲーミフィケーション
その他複数企業が参加『デジタル・エンターテインメント・スタジオ』

お申込みURL
<a href = “https://www.goodfind.jp/2015/seminar/1288” target = "_blank">お申し込みはコチラ！

2015年卒業予定の大学生／大学院生 対象

ゲームデザインとは、人間の本能と欲求の再デザインです。設計するクリエイターは、人種や国境を越えたウォンツと向き合いながら、その達成への道筋を考え、新しい体験に変えます。あらゆる成功したサービスに共通している考え方、すなわち、人間を夢中にさせる仕組みとは何か。その手法を他分野へ展開することで、社会にどのような影響を与えられるか。本イベントでは、デジタルエンターテイメント企業が集結し、エンターテイメント、ゲームの力を使って世界を変える面白さを語ります。

登壇者プロフィール
水口 哲也 氏

メディアクリエイター/慶應義塾大学大学院メディアデザイン研究科・特任教授など講師多数。1990年セガ・エンタープライゼス入社、情感デザイン研究室室長、セガの開発スタジオであるUnited Game Artists代表、2003年キューエンタテインメント株式会社共同代表を経て、2012年水口クリエイティブオフィス(Mizuguchi Creative Office) を設立。2002年欧州アルスエレクトロニカにおいて、インタラクティブアート部門Honorary Mention、経済産業省デジタルコンテンツグランプリ・エンターテインメント部門サウンドデザイン賞、文化庁メディア芸術祭特別賞などを受賞。2006年には「Digital 50」（世界で注目すべきデジタル系イノベイター50人）の1人に選出される。

参加企業

  1. コロプラ
    市場拡大を続けているiOSやAndroidのネイティブアプリ開発力と、位置情報アプリ（携帯端末のGPS機能を使用したアプリ）のパイオニアとして培った独自のビジネスモデルに強みを持つ。
  2. ドリコム
    ソーシャルゲーム、ソーシャルラーニング、ソーシャル広告という3つの分野でネイティブマーケットNo.1を目指す知的ヘンタイが集うインターネット時代の"ものづくり企業”。
  3. マイネット
    “-beyond language-言語を超えるコミュニケーションを実現”するというメッセージをもとに、高度なマーケティング・開発ノウハウと、ジャパニーズ・カルチャーを融合させたプロダクトで、2012年にはAndroidマーケットでNo.1の売上を獲得。
