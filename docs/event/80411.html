description

世界３大広告賞である「カンヌライオンズ」では、今年もクリエイティブが光る数々の施策が注目を浴びました。その中でも特に注目度が高かったのが「動画コンテンツ」です。

現在、オンライン・オフラインを問わず様々な広告配信・PR施策において、クリエイティブの重要性が高まり、その表現として動画コンテンツが世界的な潮流となっています。
その動画コンテンツの今後について、カンヌライオンズ 2016 にも現地で参加し、日本でも数々の動画PRを手がけるビルコム株式会社の代表である太田滋様をゲストに迎え、

今、社会的に拡散・話題化され得る動画コンテンツの特徴とは何かについて、動画コンテンツを流通させるマーケットプレイス「VISM」の事例なども取り上げながらお話させて頂きます。

　　　　　  

--ゲスト講師——
