今回、「成長請負人」Growth Contractor講座の開講にあたり、講座の説明会を兼ねたプレ講座を開催いたします。
ご都合に合わせてご参加いただけますよう、各回とも全10回に渡る「成長請負人」Growth Contractor講座のエッセンスや講義風景を体感できる時間となっております。

本講座では、「社会を変える」事業に取り組むビジョナリー・リーダーの事業成長を加速させるために、自身の専門性を活かし、「成長請負人」として、個別プロジェクトでの成果を通じて成長にコミットできる人材の養成を目指します。

高齢化に伴う低成長時代の到来と少子化による人口減少社会に突入し、日本には全体的に閉塞感が漂うなか、リーマンショック、東日本大震災後、こうした閉塞感の打破に挑戦する、新しい価値観を持った起業家・活動家が続々と誕生しています。
しかしながらその一方、こうしたビジョナリー・リーダーの多くが、スタートアップ段階に存在する様々な制約により、期待した成長を実現できず、小規模にとどまっているケースが少なくありません。

起業家・活動家は社会の宝であり、子供と同じ。
起業家・活動家が育たない社会に未来はない。

だからこそ、現実を変革するために、ビジョナリー・リーダーを支える事業運営面の「エンジェル（起業家のビジョンに共感し、将来の成長に投資する人々）」ともいうべき人材「成長請負人」を、もっと社会に増やしていくことを本講座のねらいとします。

「社会を変える」事業に取り組むビジョナリー・リーダーの事業成長を加速させるため、
自身の専門性を活かし、成長にコミットする「成長請負人」のマインドとスキルを学びたいという方、ご自身の成長のために「やりたいこと」「できること」を整理してみたいという方、この機会にぜひ「成長請負人」のエッセンスを学んでみませんか？
