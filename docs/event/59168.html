catch ISUCON裏のあんなこと、こんなこと
description

ISUCON出題者がビールを飲みながらしゃべる！

多くの参加者を迎えるようになったISUCONの裏には、問題、レギュレーション、ベンチマーク、環境構築そして当日の運用・採点に至るまで、さまざまなところで出題者の悩み・判断そして楽しみがあります。

このイベントではそんなあれやこれやについて、過去4回のISUCON出題者たちが記憶をたどりつつまとめてしゃべったり、お互いに話しあったりします。ISUCON参加者に向けた話はおそらくあまりありませんが、出題側の考えていることはもしかしたらちょっと興味深いかもしれません。

出題者たちはビールを飲みながらしゃべります。参加者の皆さんにも適当な飲みものの持参をお薦めします。出題者たちへの差し入れも歓迎します :-)

※ ただし未成年の方の飲酒はお断りいたします
※ 飲酒してのイベントなどけしからん、という方の参加についてもお断りさせていただきます

19:30の開始に遅刻した方は会場に入れません。遅刻せずお越しください。
