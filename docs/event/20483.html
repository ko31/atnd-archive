てらさわが久々にＤＪパーティやります！

さいきんＤＪする機会があっても１時間未満だったことが多いので、
ぼくのＤＪプレイを聴かれたことがある人も少ないのではないかと思います。
今回は少なくとも４時間はぼくが回し続けますｗ
人前でのロングセットが久々なので、リハビリテーションと題しました。

かけるジャンルはハウスを中心に、シティポップス、ブレイクビーツ、ドラムンベース
なんかをかけていきたいと思っていますが、その場の乗り次第です。
ぼくの一番得意とするオールジャンルスタイルでいきます。
このイベントのために、ひさびさにレコード買い集めてたりするんだからネッ！

場所はこちらになります。
http://www10.ocn.ne.jp/~barcyde/barcyde-map.html

１５人くらいで満杯になる（ＤＪブースもある）バーです。
ウィスキーとラム酒の種類が豊富です。
ごはんものは無いので、食べてからいらしてください。
大丈夫、長時間やっているので（たぶん朝まで）、いついらしてもＯＫです。

料金は、ドリンク代だけでＯＫ！１杯だけなら１０００円以下！
でも、いっぱい呑んでもらえるように、いい音楽かけますよ。

当日は全プレイustで見ていただけるようにしようと思っています。
アドレスは当日発表します。
またひょっとしたらゲストＤＪがあるかも？！ないかも？！といった感じです。

連休のど真ん中ですが、お近くの方でお暇でしたら是非いらしてください！
音楽でおもてなしします。

ご不明点等あればコメントいただければと思います！
よろしくお願いします！
