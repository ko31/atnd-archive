内容

4月の技術ひろばは 「開発者のためのUX (再)入門」 がメインテーマです。
UX も浸透してきたように見えて、 「なかなかピンと来ない」とか、「画面設計のこと？」、「WPF や Silverlight を使うこと？」という方が実はまだまだ多いようです。
開発者の立場で UX を理解するための第一歩（または復習）として、気軽に遊びに来てください。もちろんデザイナーの方も歓迎です。

今回はセカンドファクトリーさんの会議室をお借りしての開催です。セッション以外でも、ラウンジで社員の方と話ができるなど UX を体感するチャンスです。
連休初日の開催ですが、ビジネスのヒントとして、ネットワーク作りに、多数のご参加をお待ちしています。

  • 「Windows Live SDKを使ってみよう」 （エモやん さん）
    豊富なサービスを備えるWindows Liveオンラインサービス。
    今回は、OAuth 2.0を用いたサービスへの認証と、SkyDrive API を使用したオンラインストレージのファイル操作について紹介します。
    WPFと組み合わせて、実際にフォトビューワを作りながらポイントを解説します。
  • 「LightSwitch から UX を見てみる」 （瀬尾）
    開発が簡単で、画面についても開発者が知恵を絞る余地が少なそうな LightSwitch。
    画面デザインが定型的なのは本当にデメリットなのか、そして拡張機能で見た目や操作性を変える意味を考えてみます。さらに LightSwitch V2 の OData 対応により画面デザインやデバイスの自由度が上がったことをデモを通して紹介します。
  • 「ユーザーエクスペリエンスの捉え方」 （2FC 小川さん）
    ユーザーエクスペリエンスの概念の捉え方、よく間違われやすいユーザビリティとUXの違い、ソフトウェア開発における今後のUX／UIの展望などを語ります。
