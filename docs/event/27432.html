description

スキルを使った新しい社会貢献のスタイル“プロボノ”を進めるサービスグラントには、営業、マーケティング、広報、システム開発、編集、コピーライト、デザイン、プロジェクトマネジメントなどなど、会社以外にも身に付けたスキルや経験を社会のために活かしてボランティアを始めてみようというアクティブなビジネスパーソン、クリエイターが集まっています。

ボランティアといえど、それぞれ異なる強みを持ったメンバーが４～６名からなるチームを編成し、社会課題の解決に取り組むNPOなどの非営利団体の広報や資金調達、事業計画をお手伝いしています。

サービスグラントでは実施するプロジェクト件数を今後も増やして、より多くの人に今までには知らなかった”社会課題のコト””他業界、他業種で活躍するチームメンバーのコト”など新しい出会いや視野の広がりといった「サービスグラントする」楽しみを経験していただき、これまで以上に多くのNPOの活動をサポートいきたいと考えています。

　これまでの活動の中で約75のプロジェクトが立ち上がり、無事に完了を迎えていますが、
■実際にいったいどんな人が、どんな雰囲気でやっているのか？　
■それって、本当に参加して楽しいものなのかどうか？？　
など、気になるトコロを見きわめていただくためにも、ぜひお気軽に説明会にお立ち寄りいただければ幸いです。
説明会の所要時間は1時間程度を予定しております。

　さまざまなバックグラウンドをお持ちのみなさまが集う説明会、そこはサービスグラントならではの異業種・異職種の社会人ネットワーク！という一つの醍醐味を味わう入り口です。

　お仕事帰りに、お休みの日の朝に、などご都合のよきところでぜひお誘いあわせの上、お気軽にお立ち寄りください。

●内容
サービスグラントとは？／これまでの実績／参加者の声、NPOの声／サービスグラントの流れ〜立ち上げから完了まで／各ポジションの役割・分担の内容　etc.
