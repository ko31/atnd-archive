IT活用カフェ 第3回: iPadについて語ろう

Let’s Talk iPad。と言うことで、iPadをテーマに、

  • iPadを使用してみたい
  • iPadを使っているけど、もっと使いこなしたい
  • 他の人がiPadをどのように使っているのか知りたい
  • iPadアプリを見せびらかしたい
  • iPad？？iPhoneの大きいやつでしょ？

と言った方々が集まり、情報交換が出来るような場をご用意したいと思います。

10.12にiOS5がリリースされ、10.15にiPhone4Sが発売されましたので、これらのキーワードも話題になるのではないかと思っています。
