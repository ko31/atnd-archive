同年代を繋ぐコミュニティ「85en / はちごえん！」の5月イベントです。

4月に桜丘カフェを貸し切って開催したファーストイベントは90名弱もの同年代が集まり、色々な新しい繋がりが生まれました。

5月は前回とは違い、もっと強い繋がりを創っていきたいということで、人数を少なめの20名程度に設定し互いのことをよく知ることができるようなイベントを開催しようと思います。

　

トークセッションイベント「85jam / はちごじゃむ！」

“85jam/はちごじゃむ！”とは、プレゼンターが自身に関することを5分でトークしていくイベントです。web界隈だとライトニングトークと呼ばれるものに近いのではないでしょうか。ちなみに「85jam」の由来は"jam"に「自由な掛け合いでその場でしか出せない音を創る」という意味があることから、85世代同士のそれということで名付けられました。

プレゼンターは、5分という決められた時間の中で自身に関する仕事や趣味について1つのテーマを設定し、スライドとトークで自分のことを楽しく伝えてもらいたいと思います。
オーディエンスもトークを聞いて、みんなと話して、飲んで、そんな感じでそれぞれが交流を深められる、セッションできるイベントにしていきましょう。

85enは同年代が集まることで新たなことが生まれる、そういった目的でイベントを開催しています。ぜひそんなイベントにしていきたいと思います。
