description

iPhoneアプリに興味を持ち、いざ開発しようとプログラム勉強を始めたとき、「Objective-C」の難解さでつまずいた…そんなことはありませんか？

「Titanium mobile」（Appcelerator社製）は、Web出身の人には馴染みのある Javascript を使用して、簡単にiPhoneを作成することができ、しかも同時にandroidアプリも作ることができるとてもお得なソフト。

全くはじめての人でも、２日間（講座＋ハッカソン）で、実際にアプリを作ってもらおうという、講座です。

HTMLを多少知っていればそれでOK。必要なものは、Mac( OSX 10.6以上、iOS SDK 3.2/4.1)。シミュレータで動作確認を行いますので、今すぐにiPhoneは必要ありません。

みなさんのご参加をお待ちしています。

講座（操作の仕方〜基本的な構造〜パーツ説明〜サンプルアプリ）では実際にソースを手打ちしてアプリを制作します。

講座参加料は6,000円です。講座に参加された方は、毎月末に行うハッカソン（アミダくじでグループ決め〜アプリ案作り〜アプリ制作・画像制作〜発表会）に無料で参加することができます。（次回は7/31）　

初心者向けの講座です。経験者のかたはご遠慮下さい。

なるべく多くの方に参加いただきたいので、終了間近でのキャンセルはご遠慮下さい。

Macをお持ちでなければ、Macをお持ちのお友達とご一緒にご参加ください。なお、TitaniumにはWindows版もありますのでWindowsでの参加も可能ですが、講座ではMacを元にした説明となりますのでご了承ください。

事前にご準備いただきたいものがありますのでご確認ください。

スマートフォンは必須ではありません。検証用に欲しい方はこちらで購入できます。
