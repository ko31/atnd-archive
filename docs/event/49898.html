イベント ユーザー

title Web制作ブートキャンプ
catch 【株式会社メンバーズ　Presents】 ～大規模サイト制作★実践力養成トレーニング～
description

専門学校や職業訓練校を修了されて、あるいは印刷物のデザインのお仕事から、Web制作者・Webデザイナーとして就職・転職を目指す方々にとって、最大のハードルは実務経験の不足。

クリエイターの皆様の就職・転職をお手伝いするクリエイター・エージェンシーの株式会社クリーク・アンド・リバー社では、特に求人の多い大規模サイト関連の実務と直結した実践力養成トレーニングを開催します。

新卒の方、印刷物のデザイン等、他業界からのキャリアチェンジをお考えの方、Web業界内での営業から制作職へ等の職種チェンジを希望される方などに、ぜひご受講いただきたい講座です。
この講座を通じて、あなたの「スキルの即戦力度」と「面接での受け答え」が変わります！

★こんな方にオススメ！　企業に求められる「あなた」を目指す！★
　●Web業界に就職活動中の新卒未経験の方
☆目指す姿☆仕事内容を具体的にイメージできている／入社後も意欲的に取り組んでくれる
　●印刷物のデザイン等別業界からの転職を目指す方
☆目指す姿☆印刷物のデザインとWebデザインの違いをしっかり理解している

★具体的な内容★
大規模サイト制作・運用に必要とされる「スキル」と「マインド」を以下のように習得できます。
●多人数での分業に絶対に必要となる「スケジュール・段取り」の力
●大規模サイト構築のスタートラインである「レギュレーションの理解」力
●入社直後から即戦力を目指す「HTML・CSS・DreamWeaver・画像制作等のピンポイント反復練習」
●チーム意識やプラスαのサービス精神と提案力など、「大規模サイト制作に必要なマインド」等
