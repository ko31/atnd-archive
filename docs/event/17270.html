セッション概要

Java SE 7 で切り開く新しい Java の世界について

寺田 佳央 ( @yoshioterada )

2006年Java SE 6が登場５年が経過し、今夏ついに待望の
Javaの新バージョン Java SE 7 がリリースされます。
本セッションではJava SE 7で提供される新機能や
言語仕様の変更点等を分かり易く紹介します。

祝Preview卒業！Google App Engine for Java

小川 信一 ( @shin1ogawa )

Google App Engine for Javaは今年後半にPreview版を卒業し、正式リリースされます。
新しい機能どんどん追加され進化し続けるGoogle App Engineプラットフォームを
新機能も含めて45分でサクサクと紹介します。

NetBeansでJava SE7を勉強しよう

きしだ なおき ( @kis )

もうすぐJava SE7がリリースされます。Java SE7では、APIだけではなく文法も拡張されました。
しかし、新しい文法を覚えるのはおっくうだという人も多いのではないでしょうか。
そういうときは、IDEに新しい文法を教えてもらえばいいのです。
今回は、NetBeans7がどのようにJava SE7文法を「教えて」くれるか紹介します。

JavaFX 2.0 – リッチクライアントのためのUI基盤

櫻庭 祐一 ( @skrb )

JavaFX 2.0はJavaのUIライブラリとして生まれ変わりました。
JavaFX Scriptを捨て去り、JavaのAPIとなったJavaFX 2.0は
JavaFX 1.xの使いやすさをそのままにリッチクライアントを
構築できるようになります。

JavaFX Scriptに特有のバインドなどの機能も、Javaで扱えるようになり、
豊富なGUIコンポーネントやアニメーション、エフェクトなど
さまざまな機能が提供されています。
その中には、WebKitをベースにしたブラウザコンポーネントも含まれています。

本公演ではJavaFX 2.0がどのようなものなのか、デモも交えてご紹介します。

AWS ではじめる Programmable Cloud

染田 貴志 ( @tksmd )

今年 3 月に東京にデータセンターをオープンしたことで、
注目度も日増しに高まっている Amazon Web Services (以下、AWS)。
次々と新サービスがリリースされている AWS の概要を押さえたうえで、
その AWS の各サービスを操作する AWS SDK を紹介します。

もう Java EE 6 でいいんじゃない

寺田 佳央 ( @yoshioterada )

Java EE 6 は 2009年12月に正式リリースされ１年以上が経過しました。
J2EE 1.4 までは Java EE というと重量で実装が困難、開発効率が
悪いなどの悪いイメージがありましたが、Java EE はバージョン 5 より、
かんたん開発にシフトし、Java EE 6 では、その集大成ともいえる
実際に扱い易い軽量フレームワークへと生まれ変わりました。
既存の軽量フレームワークの長所を取り入れかんたんになった標準の
Java EEを利用する事で、開発効率が向上するだけではなく、運用・
管理面でも大幅に負担を軽減します。
本セッションでは、Java EE 6 の概要とその参照実装である
GlassFish について分かりやすく紹介します。
