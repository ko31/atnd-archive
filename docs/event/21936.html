ひと通りのスケジュールが終了しました。
みなさんお疲れ様でした。
これ以降は特にスケジュールを設けないので、今後 Advent の記事を書かれた方はコメントにでも書き込んでおいて下さい。
ついでに @manga_osyo にもリプをもらえると助かります！

追記

25日まで埋まりましたが、26日以降に参加されても全然問題ありません。
引き続き参加者を募集しています。

ルール

C++11 に関する記事を、参加順に1日1つ書いてもらい、コメント欄にURLを貼っていきます。

1番目の人が1日目、2番目の人が2日目…って感じです。

内容は、追加される機能以外にも、標準ライブラリについてや C++11 だからこそのテクニックなどでも構いません。

参加順に書いていくので、ネタは早い者勝ちです。

各コンパイラの対応状況

  • gcc 4.7
  • clang 3.0
  • Visual Studio 2010
