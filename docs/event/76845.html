　新年度、始まりました！例年にないペースで桜が咲くなあと思っていたら、嵐がきたり、そして、大きな地震まで。阪神淡路大震災経験者としては、心が切なくなると同時に、あの時の恐怖感が蘇ってきて、20年たっても薄れない記憶に戸惑ったりもしております。

　さて、クーリエジャポンは今月から雑誌もデジタルに移行。スマホから気軽に読めるようになった一方で、“付箋がはれない！”、“書き込みやマーカーが入れられない！”等々、やれば別の方法でできるのかもしれませんが、今さらながら紙媒体との付き合い方の違いに戸惑っている今日この頃ですが、皆さんはいかがですか？

　特集記事は「Features」という形で毎週アップされるようになったようで、まだ全て読み切れてませんが、読んだ中で印象的だったのは、
「難民危機で壊れゆく世界」
https://courrier.jp/feature/47364/

　大量の難民が押し寄せる現実は、なかなか想像し難いところもありますが、生々しい現場の事例にその大変さが少しは理解できるものとなっており、この春から移住促進のお仕事も責任範疇になった私にはありがたかったです。

　会場はいつも通り、阪急の高架下、西口と東口の間にあるタリーズコーヒーです。三宮駅の浜側ではなく、山側に出ていただく必要があります。タリーズコーヒーは近くに数店舗ありますが、お間違いのないよう、ご注意ください。

朝食会では、真剣な話から、軽いノリの話まで、いろいろな話題でおしゃべりしています。 初めての方も、どうぞお気軽にご参加下さい！

　Byたけ（@lohasway）

クーリエ・ジャポン DE 朝食会＠神戸　とは？ クーリエ・ジャポン朝食会とは、世界1500のメディアから厳選した記事を紹介している講談社の『クーリエ・ジャポン』という雑誌について、朝ごはんを食べながら、感想や意見を語り合う読書会のこと。読者が主催となって、東京、名古屋、大阪、福岡など各地で行われています。

神戸の朝食会では、会の中でも記事を紹介していきますので、クーリエの読者はもちろん、まだ読んだことがないという方にもご参加いただけます。

  • グローバルな視点を求めている人
  • いろんな国のこと、世界から見た日本に興味のある人
  • 自己成長のヒントを探している人
  • 面白い人とつながりたい人
など、多くのご参加をお待ちしています。学生さんも大歓迎です！クーリエという共通の話題を通じて、新たなつながりをつくり出し、お互いの見識を深められる場としていきたいと思います。
