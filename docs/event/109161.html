開催概要

はじめまして、Sumo Logicです。

100%クラウドでログ・メトリクス分析/Cloud SIEMを提供しているSumo Logicが、日本で初開催となる「Sumo Logic Japan Meet-UP!」を開催します！

Sumo Logic Japanは、2019年10月に日本オフィスを立ち上げ、様々なお客様への導入が進んでおりますが、 もっとエンジニア同士の繋がりを深めて行きたい！と思い、今回日本で始めてMeet-UP!を開催する事になりました。

まだ日本で活動を始めてから1年弱、Sumo Logicを知らない方も多いと思いますが、世界では2,000社が採用、日本でもゲーム関連や仮想通貨関連の企業様をはじめ、その他幅広い業種の企業様に採用されています。

ログ分析やメトリクス分析、セキュリティ（SIEM）に興味が有る方ならば、Sumo Logicを知らない方でも 楽しめる内容になっております。是非交流を深めましょうー！

https://www.sumologic.jp/
