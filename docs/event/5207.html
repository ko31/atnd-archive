title 東方の彼方から
catch -ゆらめく光と音の彩り-
description

今年の後楽荘のキャンドルナイトは「東方の彼方から」と称し
インド百弦琴、
サニトゥールの演奏とガラスの作品をろうそく灯りで愉しむ企画です。

宮下節雄さんによるサニトゥールの透明感のある音色は、
繊細に澄みわたり、空間を清らかに満たしくれます。
古典インドの精妙な調べをお楽しみ下さい。

ライブ後、黄昏時の新緑の庭を眺めながら、
後楽荘特製、鮎を中心とした料理を深まりゆく闇とろうそく灯りでご堪能下さい。

午後八時に電気を消して、ろうそく灯りのもと
茶室、燈くらなどで展示した青木邦恵さんのガラス作品をご覧下さい。

東方の彼方から届いた調べと光の雫を、
後楽荘の空間で紡いでいきます。
