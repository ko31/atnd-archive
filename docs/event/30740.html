■講演者プロフィール

新清士（しんきよし）氏 　ジャーナリスト（ゲーム・IT）
連載に、日本経済新聞電子版「ゲーム読解」、ビジネスファミ通「デジタルと人が夢見る力」、週刊アゴラ「ゲーム産業の興亡」。米「Game Developers Magazine」（2009年11月号）でゲーム産業の発展に貢献した人物として「The Game Developer 50」に選出されている。1970年生まれ。慶應義塾大学商学部、及び、環境情報学部卒。他に、立命館大学映像学部非常勤講師。国際ゲーム開発社協会日本（IGDA日本）副代表。
