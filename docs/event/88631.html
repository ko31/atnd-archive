セミナー内容

♡step１

  • 『内面から輝き、一番に選ばれる女性になる方法』
    〜素敵な女性に魅せる心構え、言動を身につけます〜
  • 講師：山田マキ
    プロフィール
    客室乗務員としての経験を生かし、接遇やコミュニケーションをテーマにした研修や講演を行っている。昨年、「あなたの魅力を爆発させる方法」（文響社）を出版後は、コーチング、セミナーを通じ、クライアントの魅力を爆発させることに尽力している。

    ♡step２

    『ワンランク上の男性を引き寄せる、魅力的な立ち居振る舞いレッスン』

    第一印象で記憶に残せるエレガントな動き（立ち方・座り方・歩き方）
    合コン・婚活パーティ・お見合いで使える、また会いたいと思わせるイイ女のしぐさ

    • 講師：田村亜弥
      プロフィール
      都内スタジオを中心に、姿勢改善、ウォーキング、立ち居振る舞い、ダイエットを指導。女性の美と健康のためのサポートをしている。
      15年間のクラシックバレエ経験をベースとした、しなやかな動きとエレガントな立ち居振る舞いに定評がある。
    ♡step３

    『１年以内に結婚できる私になる』

    婚活座談会。
    自分に合った婚活方法の選び方。

      少人数制の丁寧なサポートに定評があり、大手相談所から移って来られる方も多い。
