description

SEOに取り組む企業は星の数ほどあれど、SEOの専任担当を置かれている企業はかなり少数派なのではないでしょうか。Web担当者が業務の一環としてSEOも見ている、というケースがほとんどだとかんじています。実際、弊社のクライアントもそういった体制が多いです。

兼任でSEOを見ているために、SEOに関する施策立案や分析に十分な時間が取れず、何をしたらいいのかわからず漠然と業務を進めてしまっていることはないでしょうか？

今回はそんな悩めるWeb担当者様向けに「Web集客・SEOマンツーマンお悩み相談会」を実施いたします。

「効果がすぐ出るSEO事典（翔泳社）」の著者である岡崎が2hがっつり壁打ち相手になります。

単にアドバイスを伝えるだけでなく、2時間がっつりお話しながら、一緒に資料を作ったり、ワイヤを引いたり、すぐに実務につながる何かを生み出すことがこの相談会のゴールです。

こんな方におすすめ

  • SEO対策を業務の一部とされているWeb担当者様
  • 書籍やネットでSEOの情報は収集しているが、それが正しいのかよくわからない
  • 実施すべきか悩んでいる施策はあるが、身の回りに相談できる相手がいない
  • 担当サイトで次に何をするべきなのか思いつかない
  • SEOの効果測定をどのように行えばいいのかわからない 　

「お悩み相談会」で得られるもの

  • プロのSEOコンサルタントによるSEOに関する正しい情報
  • 検討中の施策の優先順位付け、ブラッシュアップ、効率的な進め方
  • 貴社サイトの分析結果に基づくSEO改善提案。担当サイトで次に行うべき具体的な施策
  • 貴社サイトにおける適切なSEO効果測定方法の立案
  • 競合サイトのSEO分析。それに対する戦い方
  • SEOコンテンツの具体的な制作方法。テーマ企画、体制づくりのサポート
  • デザイナー、エンジニアへの修正指示書（一緒に作ります）
  • 上司への提案・説得資料作りのサポート
