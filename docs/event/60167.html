description

ドル円7年ぶり安値、日銀緩和で売り続くのか…
アメリカ量的緩和終了発表の中、日銀は追加の金融緩和を発表し、2014年終わり間近に120円台の新たなステージにまで到達！
そんな中、新興国通貨、資源国通貨はいったいどんな動きをするのか？？

もうこの流れの中では乗り遅れたと思われる方もいるはず。。
しかし、その中でも狙える通貨はあるはず、
低金利の時代だからこそ投資をしてみたいと思っている方
必見！！

今トレード中だけどなかなかうまくいかない。何が原因なのかわからない。。という方　必見！！ 外貨預金とFXの違いは？双方のメリット・デメリットとは？？　わからないことだらけという方　必見！！

ここだけでしか聞けない？話にも注目！！

FXトレードは、マーケットが注目するテーマに対して、マーケットの大勢がいかに考え、どのようなポジションに偏っているかを推理することが重要と語る水上氏！！
海外銀行ディーラーの中には、来日時に水上氏を訪問し、相場の見通しを聞きにくる人もいるほど。そこまで信頼を得る水上氏と他のディーラーとの違いとは？？

【手っ取り早く継続的に儲ける方法は無い！

　　　　　　　　　唯一の方法は基本を自分のものにしていくこと】
相場は｢通貨を買ってる人、売ってる人」で動いています。マーケット参加者の動きを先回りして読むことが、今後の相場を読むために必須です。
その投資家たちの動きを読むために３０年相場で培った知識、経験と共に、独自の相場情報を伝授します。

サンワード貿易が贈る2015年チャンスをつかむための緊急セミナー！

水上紀之氏プロフィール

バーニャマーケットフォーカスト代表。
東京外国為替市場で「三和の水上」の名を轟かす。
長年の経験と知識に基づく精度の高い相場予測には定評がある。
公式ブログはFXブロガーランキングのベスト５を常にキープ！（13年12月現在）

１９７８年、上智大学経済学部卒業後、三和銀行(現、三菱東京UFJ銀行）に入行。 １９８３年よりロンドンや東京、ニューヨークで為替ディーラーとして活躍。銀行ディーラー時代、当時の歴代年間収益最高記録を更新！ １９９６年よりＲＢＳ銀行にて、外国為替部長を経て、為替営業部長を歴任。 現在は、ＦＸストラテジストとして日経ラジオや金融情報誌等で活躍中！
