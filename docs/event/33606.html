description

転職活動の第一歩は、「職務経歴書」の完成度を上げること！

こんな経験ありませんか？
・無料人材紹介会社のウエブエントリー登録するが反応がない。
・「履歴書」「職務経歴書」を郵送するが反応がない。
・面接上で突っ込んだ質問が多い。
・想定外の質問が多い。

それは、職務経歴書に書くべきことが書かれていないからです。
・企業が求めている人材をイメージし、それにマッチした能力をアピール。
・「何ができるのか」に答えていない。
・成果が明確でない。
・成果のものさしがない。⇒普通なのか？低いのか？高いのか？不明
・読みやすいレイアウトでない。
・10秒で全体が読める内容でない。
・面接で質問に誘導する書き方になっていない。
・読み手に興味を持たせる文章になっていない。

１０種類の職経歴書サンプルからベストな形を作成します。
・＜要約＞はあなたに興味を持たせるキャッチコピー⇒見出し
・＜職務経歴＞はあなたの「できること」をアピールするカタログ。
・企業に合わせた＜成果・実績＞を記入する。
・＜成果・実績＞をアピールするものさしを用意する。
・＜セールスポイント＞は、最後の詰め。
　あなたに会ってみたい・・・・・と思わせる内容。
　興味を持たせ、「聞いてみたい」と思わせる内容。
　「聞いてみたい」⇒質問を誘導⇒自分の土俵に持ち込む。
