M→Fes #とは

僕らは今年も数多くの"最高"を味わってきた。
前回より明らかに巨大だったエンターテインメントショー、戦士たちの熱き3日間、そして秋の某上映イベント…。

「私たちは常に『早く最高になりたい』と思っている」
「だが、前回以来、家に帰りwebにアップロードされた動画を見ても、いくら高いヘッドフォンで聴いていても、『 何かが足りない 』と感じてしまう……。」

ある人は言いました、「 もう一度最高になりたい 」

みんなで最高になりましょう！！！！！！！！！！！！！！

M→Fes2017は M→Production 主催で行う映像上映イベントです。
今年のまとめとなるような楽しいイベントにしましょう。
新作映像作品 はもちろん、 今年公開した映像作品 も応募OKです！
大きなスクリーン、大きな会場で自分の映像を上映してみませんか？
自分が作った作品を公開して、 "実はこんなところをこだわった""ここを見てほしい" という瞬間を共有しましょう！！！

M→Fesは 誰でも参加無料！
参加者の映像作品出展は必須ではございません。
会場で視聴したいという一般参加も大歓迎です！
一般参加の方はそのまま参加登録を、映像出展参加者の方は下記【作品応募】をよく読んでから参加登録をお願いします。
※当日のタイムスケジュールは後日公開します
※前回のM→Fes2016は こちら

また、今年は 新企画「お前らに見て欲しい映像！」 コーナーを予定しています！
こちらのコーナーは、今年のまとめをする企画としまして、みなさんが大好きな作品をみんなと一緒に見て最高になろうという企画です！
すべてのM→Fes参加者に、 みんなにどうしても見て欲しい映像作品のタイトル（URL)と、どうしても見て欲しい理由（熱い想い） を参加登録時のアンケートにて記入していただきます。 （こちらは任意でお願いします）
多く寄せられた映像作品もしくはとても熱い想いが込められていた映像作品いくつかを当日会場で上映します！
もちろん、上映する作品は制作者に許可を取った上で流す予定のため、最悪の場合何も流せないこともありますのでご了承下さい。
（11/27追記）→こちらの企画は懇親会で上映致します

【作品応募】 応募は締め切らせていただきました

M→Fesでは出展していただける映像作品を募集しています。
大学内外問わず受け付けています。
下記説明、注意事項をよく読んで、参加登録時のアンケートフォームに従って記入して下さい。

作品について

  • 1作品の長さは 1~5分程度 でお願いします。
  • ジャンルは自由ですが、著作権や公序良俗的なものは守ってください。
  • 同名義での複数出展もOKです。
  • 推奨動画フォーマットはmp4形式(H264 / AAC)です。
  • 推奨解像度は1280x720(720p)又は1920x1080(1080p)です。
  • 推奨アスペクト比は16:9です。
  • 推奨フレームレートは30fpsです。その他一般的なフレームレート(23.976,29.97,24,25,59.94,60)でも対応します。

スケジュールについて

  • 11月11日(土)23:59 : 出展応募締め切り (出展者名とメールアドレスをお聞かせください) 出展者多数の為予定より早めに締め切らせていただきました
  • 11月25日(土)23:59 : サイト/パンフ公表用出展者情報提出締め切り
  • 11月25日(土)23:59 : 制作中作品の一次提出　(作品の予定時間を合わせてお聞かせください)
  • 12月9日(土)23:59 : 作品の最終提出　(上記規格に合わせて上映できる状態でお送りいただきます)

作品の一次提出は字コンテ・絵コンテ・音楽のみでも構いません。

提出について

  • 各提出方法は、出展応募確認後、出展者様宛にメールを送りますので内容に従い返信にて提出をお願いします。
  • 一次・最終提出は ギガファイル便 や firestorage にて、同メールアドレス宛に連絡をお願いします。

メールにてアナウンスを行うため、常時メールボックスの確認をお願いします。

【必読】注意事項

  • 応募作品は新作はもちろん、今年公開した映像作品であればそちらの応募もOKです。 去年以前に公開した作品の応募は申し訳ありませんがお控えください。ですが、 去年以前に公開済みの作品をリメイク・再編集・一部使用するということは可能です。
  • 楽曲・素材・作品内容など、権利関係を含めた全責任は各出展者が負うものとし、上記内容について何らかの問題が発生したとしてもM→Productionは一切の責任を負いかねます。
  • 運営側で上映が不可能と思われるような性的・過激な表現が映像内に含まれている場合、出展者と相談の上、修正もしくは上映をお断りさせていただく場合があります。
  • その他、不明点・疑問点等ありましたら下記連絡先よりお気軽にご連絡ください。
