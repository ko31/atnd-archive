イベント概要

Amazon EC2やAmazon S3に代表されるアマゾンクラウド = Amazon Web Services(AWS)は、ユーザの要望に応じて機能がどんどん追加され、まさにかゆいところに手が届くクラウドサービスになってきています。今回は、名古屋で初回の勉強会ということで、記念すべき0回目と題して勉強会をさせて頂きたいと思います。

AWSに興味があって使ってみたい方、AWSは触ってみたけれどももう少し深く知りたい方、AWSをすでに使いこなしていて他の方にその使い方を広めたい方、AWSユーザーグループの名古屋のコアメンバーになられたい方、気軽にどしどしご参加ください。

立ち見をすれば40名ほど入れるスペースがあるのですが、まずは20名で募集をさせて頂きます。

開催プログラム

  • 19:00 ～ 19:05 会場案内・アジェンダ
  • 19:05 ～ 19:35『アマゾンクラウド概要紹介』- エバンジェリスト玉川憲 (Amazon Data Services Japan株式会社)
    AWS入門者の方に向けて、アマゾンクラウドの概要を紹介いたします。初期費用不要、従量課金、スケールアウト・インが自在、開発スピードを高めることが可能で、システムレベルの作業から解放されるアマゾンクラウドのメリットをお話します。参加者のレベルにあわせて省略する可能性もあります。
  • 19:35 ～ 20:15『クラウドのデモンストレーション』
    既存Webアプリをその場でクラウド化しよう！このデモンストレーションでは、典型的なLAMPのWebアプリケーションを例にとって、スケールアウト可能なクラウド・アーキテクチャに変更していく実践例をデモンストレーションを交えご紹介します。フロントエンドにロードバランサ(Amazon ELB)を備え、その後ろに仮想サーバー(Amazon EC2)により並列化されたアプリケーションサーバー、後段にはデータベース・クラスター(Amazon RDS)を備えたクラウドアーキテクチャの魅力をぜひご体験ください。
  • 20:15 ～ 20:50 質疑応答 & ライトニングトーク

ライトニングトーク発表者も募集中でございます。希望者はお名前(ハンドル名可)と題名を記載の上、コメント欄にてご連絡ください。
