description

あなたの未来は応援してくれる『環境』と積み重なる良い『習慣』で決まります

就職して3年、5年、10年で自分の身の在り方を見直す時期は大体そのタイミングでは
ないでしょうか。このタイミングで心境の変化が起こります。

今、あなたはこう思っているのではないでしょうか？

・日々満員電車でストレスを感じながら、この状況をずっと続けなければ
　ならないと思ったらゾッとする。
・会社内や、友達との会話での不平不満、愚痴、ニュースのゴシップ等にうんざりしている。
・意味の無い、憂さ晴らし的な飲み会に飽き飽きして来た。
・朝、活動している人がどんな人なのか興味がある。
　

私も同じ思いを抱えた時期がありましたが、何か変化を起こす。行動するには勇気がいります。

誰もが自信が無い状態から始めます。その小さな一歩を出来る人だけが、
変化を体感し経験値を上げれるようになります。

あなたの行動範囲を広げるタイミングが来ています。

あなたの未来は、
あなたを応援してくれる『環境』と毎日積み重なる良い『習慣』で変わります。
ちょっとだけ前に進んで見たい方は是非参加してみて下さい。

【この会の目的】
　同じ悩みや不満を抱えている人たちがどうやったら人生に変化をつけ前向きなライフスタイルに
　出来るか意見交換をし、価値観を変え、新しい一歩を踏み出せるような場所です。
　良い出会いによって、あなたらしい人生にするには夢や目標が必要だということがわかります。
　次にどんな知識が必要か分かり、行動を起こしてみたいなと思わせる場所として提供しています。
（そうありたいと思っています）

【こんな人におすすめ】
　・夢ややりたい事があるが誰にも話せない
　・生産的な時間の使い方をしたい
　・前向きで応援してくれるコミュニティに参加したい
　・変化を起こすためにどんな知識、考え方が必要か知りたい
　・同じ価値観を共有できる仲間がほしい
　
【期待出来る効果】
　①どこに向かえばいいのか進むべき道がわかる。
　②次にどんな知識や考え方が必要かわかる。
　③あなたを引き上げてくれる環境が必要だという事がわかる。
　④日々、積み上げる良い習慣を身に着ける事が大事だという事がわかる。
　⑤そのためにはあなた自身がちょっとづつ変わらなくてはいけない事がわかる。

【会の流れ】
　①自己紹介（今日来た目的）
　②今読んでいる本の紹介
　③フリートーク

【注意事項1】
　セールスや勧誘、コーチング、ネットワークビジネスをされるいる方、
　会の健全性と被害防止のためお断りしております。
　また、目的の無い参加も中止する場合がございます。

【注意事項2】
　参加申請は22時までにお願いします。申請後、参加確認をGメールにてお送りさせて頂きます。
　その返信をもって参加確定となります。返信無く参加されても席が無い事がございます。

【自己紹介】
　私はIT業界で働いている34歳、独身のサラリーマンです。
　地元は九州で工業高校を卒業後、東京の自動車会社で就職、１０年勤め転職をしました。

　なぜこの会を主催しているのかというと、私自身あなたと同じように
　今の現状に不満があり将来の悩みがあった時期がありました。

　自分が人生を前向きに捉えた時、周りにはだれも人生を良くしようという人はいなく、
　日々、心の葛藤があったのを覚えています。　
　そんな同じ思いや話したいけど話す人がいないそういった思いや考えに微力ながら
　力になれればなと思いこの会を設けてます。

　朝活をやろうと思ったきっかけもやはり朝活で出会った人でした。
　最初の参加は不安がありました。勧誘を受けて嫌な気持ちになるのではないかと。

　しかし、出会いで人生の選択肢が変わりました。

　本当に良い出会いでした。その方は今も良くさせてもらっているのですが、
　夢を語り、仲間が成長するためにはどうアドバイスをすればいいか、そして自らが実践し
　背中で見せるという姿勢。こういう人とつるみたい！こうなりたい！と思ったんです。
　それが朝活をやろうと思ったきっかけでした。

　この会を通じて、新しい人と出会い、異業種の方と情報交換の場として価値観の共有を
　目指し常に自身をアップデートできる環境に身を置き充実した人生にして行きます。
　来て頂いた方には次に何をすべきか明確にイメージでき、一日をより良いスタートが
　切れるようにしたいと思ってます。

　有難い事に起業家や、個人投資家、経営者、時には国の政策アドバイザー等、
　普通では考えられない人とお会い出来るようになり最新の情報を得られるようになりました。
　この会を続けたお陰です。

　夢も仲間と一緒にやりたい事もできました。日々ワクワクしながら生きてます。

　この時間がちょっとでもあなたの学び、行動への呼びかけになればと思います。

【主催者メッセージ】
　当会は約２年続いているのですが、その間に色々な人と出会う事が出来ました。
　夢や目標が見つかった人や社会にこんな貢献をしたい！という人、また、私も朝活やりたい
　という人達です。

　そんな仲間が出来るなんて夢にも思っていませんでしたが現実出来ました。
　あの時に選択肢を行動する事に選んだ事で未来が変わったのです。

　世界は一瞬で変わる事はありません。（宝くじが当たって億万長者にならない限り）
　私は日々1°角度を変えた事によって15°になり30°になり60°になったのです。

　未来を変えるというのは日々の良い習慣を積み重ねるしかないと思います。
　その事に早く気づき、あなたの初めの一歩になればなと思います。

　最後まで読んで頂きありがとうございました。
　では会場でお会い出来る事を楽しみにしてます！
