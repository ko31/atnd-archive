☆ワインカルチャー交流会Ｖｏｌ. ４～銀座アートギャラリーカフェ「カフェメニューを愉しむフランス語」

●カフェでお馴染みのメニューをフランス語で理解できるようになる！
●五感で味わう講座なので、フランス語力ゼロでも楽しめる！
●銀座のアートギャラリーカフェにて展示されたパリのアーチストの選りすぐりの作品を堪能！

●BGMにはフレンチポップをセレクト。フレンチムード一色で会話も楽しく弾み、新しい「つながり」が生まれる！
●フランス帰りの女性シェフによる会のテーマに合ったオリジナル料理を提供！
●女性には嬉しいスイーツ付！

毎回大盛況のワインカルチャー交流会。４回目は、４月１８日（土）に銀座のギャラリーカフェ「ART FOR THOUGHT」にて行います。「ART FOR THOUGHT」では常時、現代美術家の絵が展示され、アートを思う存分堪能できる素敵な空間です。

銀座にはギャラリーが数多くありますが、このお店はコミュニケーションカフェとしての特徴があります。今回はパリ在住の女性アーチストAnna Mullerさんの作品が展示されます。

今回のミニセミナーでは、つづきひろこさんによる「カフェメニューを愉しむフランス語」という講座を行います。

フランス語と聞くと敷居が高く感じられそうですが、この講座では文法抜きに目と耳と声で味わっていただきます。今回は、フランスのカフェでお馴染みの料理を、実際に声に出して発音しながら、意味も理解していきます。また、食卓の場面でよく使われる簡単な表現なども取り入れ、フランス語をより身近に感じていただきます。

フランス旅行の際に、カフェメニューが理解できると嬉しいですよね。セミナー後の交流会では、フランス帰りの女性シェフによる、今回のテーマに合ったオリジナル料理とフランスワインを提供します。また、女性には嬉しいデザートメニューも提供します。

交流会でのBGMにはフランス語の音の世界を広げるフレンチポップをセレクト。フレンチムードに彩られた空間でのワイン交流会。参加された方同士の会話も大いに弾み、新しい「つながり」が生まれることでしょう。是非、ご参加ください。

※つづきひろこさんのプロフィール
大学卒業後、3年間フランスへ語学留学。帰国後、フランス語翻訳会社に勤務。
翻訳のコーディネート、語学学校の運営管理、広報業務に従事。
渡仏経験から得たフランスでの生活者視点を活かして、食を通じてフランス語を学べるフランス語入門教室を秋から開講予定。
