【概要】
ソーシャルビジネスグランプリは、ソーシャルビジネスに精通した有識者による講演や、実績を評価された政治起業家の表彰や活動報告、そして社会起業家の学生たちによるプレゼンテーションを中心に、参加者による投票等を行う参加型イベントです。

【内容】
医療・介護制度の綻び、うつ・ひきこもり、地域過疎、環境問題、やりがい・生きがいの喪失・・・。多くの課題に溢れる現代において、これまでのやり方の延長線上には解決策は見いだせなくなってきています。 第11回を迎える今回のソーシャルビジネスグランプリには、そんな社会課題に屈することなく、むしろ課題解決に向けて目を輝かせ立ち向かう人々が集います。 既成概念を、組織の壁を、自らの限界をもぶち壊し、より良き社会の実現にむけて突き進む。困難を受け入れ先行きがみえない不安な状況を楽しみながら、新たな世界の創世に挑む社会起業家の原石たち。
そんな創造的破壊者たちが、練り上げた志を世界に向けて力強く語ります！

【魅力】
ソーシャルビジネスグランプリは社会起業大学から選別された実力者が考えた事業プランを参加している皆さんに評価をしてもらい、投票をしていただきます。一番、票数が高かった方がグランプリとなります。
　
　ソーシャルビジネスグランプリに参加すればソーシャルビジネスのプレゼンを聞けるだけでなく、現在活ている社会起業家達の話までもが聞けちゃいます。もしかしたらあなたが選んだビジネスが将来行われているなんて事もあるかもしれません。『将来起業を考えている』『ソーシャルビジネスに興味がある』『プレゼンの勉強をしたい』なんて人は参加を強くお勧めします。
是非気になった方はソーシャルビジネスグランプリで検索！！【その他】のURLからもホームページをご覧になれます。
