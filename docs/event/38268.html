description

当日は１９：００開場、１９：１５スタートですが、１８：３０から開場には入れます！

iOSやAndroid等、スマートフォン用ゲームの業界は、相変わらず活況です。

JavaScriptやFlashベースのフレームワークも多種多様にリリースされ、今後ますます発展の可能性がある業界ですが、その中で結果を残すのは容易ではありません。

私自身ゲーム業界からは一線引いた身ではありますが、まだまだお役に立てることもあるようなので、おこがましくも企画させて頂きました！！

お時間ある方は是非、ご来場下さい！！

費用について

勉強会自体は無料ですが、会場である basecamp NAGOYA 様のご利用料金、500円が必要となります。

当日の内容（予定）

  1. ゲーム企画で必要なこと
  2. プロトタイプとして必要なもの
  3. 作るべきドキュメント、作らなくても良いドキュメント
  4. 効果測定は何を計るべきか？
  5. チュートリアルの考え方
  6. マネタイズに関して

参加希望の方へ

まだゲーム作ってない人も大歓迎です。
学生さんももちろん大歓迎！
ただし、プログラム系の開発話は出て来ません。

開催するお前は誰やねん？という方へ

数億円 / 月のソーシャルゲームの企画をしたり、
ASではありますがゲームの開発経験があったり、
ソシャゲーのコンサルト経験があったり、
Apple に AppStore 用のバナー作ってもらったアプリを企画したり、
ゲーミフィケーションの研究と実践をやってたりする人です。
本当にさわりの部分ではありますが、まだまだお役立て出来る部分はあると思います。
