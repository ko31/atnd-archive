概要

ついに国内キャリア初のFirefox OS端末Fx0が発売されました。
Fx0やFirefox OS Flame、Open Web Boardで動くFirefox OSアプリを作ってみましょう！
会場に来れば席と電源とインターネットはあります。とりあえず書きましょう！！！

必要なもの

  • Firefoxの入ったPC
  • Firefox OS端末 (あれば)
    • Firefox OS Flame, Fx0, Open Web Boardを人数分用意していただいています。

Open Web Boardについて

Open Web Boardについては こちら をごらんください。

ハッカソンでOpen Web Boardを使用された方はKDDI株式会社様の提供でそのままOpen Web Boardをお持ち帰りいただけます。お持ち帰りを希望される方は2,30分程度の説明を受けていただきます。

すでにOpen Web Boardをお持ちの方はお持ち帰りいただくことができません。ご了承ください。
