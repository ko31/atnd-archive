description

来年の干支「寅」にちなんだ作品を募集します。参展示期間は11/29〜12/26、最後の二日間（12/25、12/26）は神楽坂の実験ギャラリー「フラスコ」にて展示となり「音」と「Art」のコラボレーション展示となります。初日から12/20までの受付期間を設けますので、徐々に作品が増える感じになるでしょう。また作品と別にポストカードを各人3種まで受付可能です。参加費は￥3000、先着20名様まで限定させて頂きます。

作品の規定
　・来年の干支「寅」にちなんだもの。
　・67cm×高さ37cm×奥行き10cmに収まるものであれば複数可。
　・平面、立体作品を問いません。
　・販売可能である事（価格は各人自由に設定下さい）

注意事項
　・売上の20％マージンを頂きます。
　・12/23日営業終了後〜12/25日は神楽坂への移動期間となります。
　・25日お昼までに作品の移動を各人お願いします。発送も可能ですが、
　　梱包・送料は各人負担下さい。
　・期間中に売約となる場合がありますが、最終日まで作品はキープ致します。
　　（ポストカードは逐一販売しますので補充が必要な場合は連絡致します）
　・26日夜売上精算＆撤収となります。

タイムスケジュール
　11/29　AM11時より受付＆展示開始（寅の市・千石空房編）
　12/23　千石空房にての展示終了
　12/25　フラスコに作品移動＆再展示
　12/25　13時（予定）「寅の市・フラスコ編-1-」開始（20時頃終了）　
　12/26　13時（予定）「寅の市・フラスコ編-2-」開始
　12/26　16時よりライブ演奏を交えたギャラリーとのコラボ（20時頃終了）
