catch 『見晴るかす雪原を歩いたあとは温泉でほっこり』
description

奥入瀬渓流では、春は雪の回廊で有名な八甲田山と、情緒満点の秘湯「酸ケ湯温泉」にご案内するツアーを開催いたします。
高いところでは約９ｍにも及ぶ八甲田名物の「雪の回廊」。
ツアーでは雪の回廊の上をガイドと共に散策いたします。
八甲田の山並みを眺めながら、スノーシューで雪原を散策、
爽快なほど気持ち良い景色は、都会の謙遜を忘れさせてくれる、極上の光景が広がります。
散策後は、湯治場としても300年以上の歴史のある酸ケ湯温泉でご入浴。
八甲田山中にたたずむ鄙びた風情の温泉は、総ヒバ造りの混浴の大浴場「ヒバ千人風呂」が有名です。
ヒバの香りと硫黄の香りに包まれた情緒ある光景は、心身ともに癒されます。
雪解けのこの時期にしか見ることができない、
贅沢な光景を是非ともご堪能ください。
