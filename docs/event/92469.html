イベント概要

本イベントは、全くの未経験からITエンジニアとしてキャリアチェンジを目指す人に向けたセミナーです。

「IT業界にはどんな職種があるの？」「文系でもエンジニアになれるの？」「やっぱりブラック企業は多いの？」 「エンジニアになるためには何から始めればいいの？」「フリーランスエンジニアって実際どうなの？」

などなど、未経験からキャリアチェンジを目指す場合はわからないことがたくさん。

そんな人たちに向け、未経験からエンジニアになった現役バリバリのエンジニア２名が、

実体験エピソードやぶっちゃけどれだけ稼げるようになったのかを、ざっくばらんに告白します。

講座に参加するメリット

  1. 目指すべきエンジニア像を明確にすることによって、これから何をしていけばいいかわかる！
  2. エンジニアになることのメリットデメリットがわかる！
  3. エンジニアになるための学習方法がわかる！
  4. 自由な働き方をするにあたってのポイントがわかる！

おすすめの人

  • 手に職をつけ、自分のスキルで食べていきたいと考えている人
  • IT業界の仕事に興味のある方 ※経験不問
  • エンジニアになりたいけど何から始めればいいかわからない人
  • 自由な働き方をしたい人
  • とにかく稼ぎたい人
