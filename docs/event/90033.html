概要

このイベントは『九州アプリチャレンジ・キャラバン 2017』に参加している学生向けの勉強会です。

アプリ開発や、ウェブサービス開発をどうやっていけばいいかわからない方は是非ご参加ください。これらの開発をスタートするのに必要な知識を指導します。

この勉強会は

メンターが何かを発表するわけではなく

パソコンを使いながら

それぞれの個別の問題について１対１で指導していきます。

この勉強会は？

『九州アプリチャレンジ・キャラバン 2017』で今後企画されている『オムニバスイベント』の中の一つです。この勉強会では技術的なテーマを特に設けずに、「とにかく何をしていいのかわからない！」とお困りの方の相談に乗ったり、とりあえず開発をスタートできるような状態までお手伝いすることを目的としています。

今後も『コアイベント』『オムニバスイベント』は、たくさん企画されていますが、このイベントではメンターの都合もあり、少人数で実施させていただきます。

予めご了承いただけると幸いです。

どんなことが身につく？

  • アプリ、ウェブサービスを作るための基礎知識
  • JavaScript（jQuery や各種フレームワーク）
  • Java（Struts2, Spring Framework, Spring Boot）
  • Ruby（Ruby on Rails）
  • Android 開発
  • Unity 開発（主にゲーム開発）
  • チーム開発技法

これらの技術領域については、サポートできる体制があります。

何か漠然と相談したいという方でも歓迎しております。
