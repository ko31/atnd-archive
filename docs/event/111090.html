★ダンエボリレー動画？

みんなで課題曲を踊って、それを私がつないで編集して、ごあいさつ欄に並んでる様な動画をみんなで作ってみようかという話です。

  • 投稿人数がどんなに少なかろうと作りますｗ参加者少なそうだしやめよっかな……では無く、自分が踊りたいかどうかのみに従っていただければと！！
  • 逆に参加人数オーバーした場合、補欠登録が可能ですが、補欠の方は私の編集機器スペック次第で参加いただけない場合がございます。ご了承ください。（現在29人まではいけることが証明されてますｗ）

★動画投稿要領

＜はじめに注意事項等＞

ルールを守って明るく楽しく元気良く！カッコよく！！
（振り付けの詰め具合は問いません。今回課題曲むずいし。）

  • 公序良俗に反する物はやめましょう
  • 振付の指定は特にないです（ダンスラ勢でもいいよ？）が、 投稿者自身が１曲踊りきった動画 の投稿をお願いいたします。
  • 一人で複数の動画を投稿するのはナシです。
  • 音はこっちでハメますので、音声差し替え等していただかなくても大丈夫です。
  • 筐体前じゃなくてもOKです。 だって筐体が（嗚咽で続きが言えない）
  • 再生順は平等にEXCELのRAND関数で決めます。（再生箇所指定不可です）
  • その他再生順番の定義は以下の通りです。まず、RAND値順に昇順で並べて……
RAND値 名称 定義
最小 サムネ 投稿動画の一部を動画のサムネにします
最大 右下 投稿動画がフルで右下に小さく表示されます（代わりにリレー内不参加）
備考 アンケートで権利を辞退した方や、サムネや右下経験者が対象になった場合、
サムネの権利は「最小値＋ｎ」、
右下は「最大値－ｎ」
（ともにｎ＝1,2,3･･･）と繰り越します。
  • ただ、動画の編集は私が仕事家事育児の合間にやりますので、
    少し時間がかかったり拙かったりでもご容赦ぐだざい……ぐるじお。【懇願】

＜課題曲＞

Gimme a Big Beat

理由

  • 参加者向け希望曲蓄積値一位（ラブキャン終了時点）
    （補足）希望曲アンケートの得票数はその曲のリレーが開催されるまで蓄積していきます。
    これはつまり、 毎回あなたが参加して投票し続ければいつかその曲のリレーが開催される 仕組みです。

＜参加方法＞

  1. 筐体前でも家でも深海でも宇宙でもいいので課題曲を踊った動画を撮る
    ★おすすめスマホ撮影道具（ステマ）★
    　キャンドゥの『スマホ三脚スタンド伸縮タイプ』＋適当な広角レンズ
  2. 動画をYoutubeやGoogledriveなどのファイル共有サイトに上げる
  3. ATNDにアカウント登録、ログインする（登録済みの人はログインのみ）
  4. ATNDの本イベント画面にて参加登録後、表示されるアンケートに記入してください。

＜アンケート内容＞

項目 補足 解説
お名前 必須 動画内に表示します。もちろん、本名である必要はありません。
プレイ場所 必須 動画内に表示します。
動画内コメント 動画内に表示します。省略の場合は参加コメントを引用します。
動画アップURL 必須 ＜参加方法＞2.でアップした先を正確に記載ください。そっから私が動画ダウンロードします。差し替えは期限まで何度でもOKなので、とりあえずエントリーしたいときはご自身のYOUTUBEチャンネルのアドレスなりなんなり入れてもらっても構いません。
サムネ権利譲渡 サムネは他の人お願い、って人はチェック入れてください。
右下権利譲渡 右下出ずっぱりよりもリレー内で動画使って欲しい、ってときはチェックお願いします。
次々回以降リレー動画やって欲しい曲 項目の通りです。次回曲決定の基本的な考え方は以下の通りですがイメージ的には こちらのツイート が参考になるかと。
・リレーの度に参加者が投票
・曲ごとに投票値は蓄積、課題曲になるとリセットされ次回投票から再蓄積（以下、蓄積値）
・蓄積値が上位のものから次回曲を決定

※動画内に表示する各項目は、公序良俗に反する場合や無しの場合は適当に決めますｗ
※ATNDの登録とかよくわかんない助けてって人は、最後の　★その他　を見てください。
