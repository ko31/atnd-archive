★講演者プロフィール

■大谷 和紀氏
2011年VOYAGE GROUP入社。
アドテク業界は株式会社adingoのSSP「Fluct」の開発から、
現在は株式会社ZucksにてZucks Adnetworkの開発と運用を、少数精鋭チームで運営中。
よく使う言語はScala、Scheme。好きなクラスはOption。
気になる手続きはmap$。得意な芸風はシステム移行。芸歴10周年。
http://twitter.com/katzchang

■粟飯島 勝明氏
2013年 VOYAGE GROUP 入社。
エンジニア歴17年。
アドテク歴は(自社メディア向け配信サーバ運用も含めれば)10年ほど。
現在はおもにオンプレ側配信システムの構成および運用管理、
AWS側にもちょっかいを出しつつある。
好きな技術は cobbler/puppet/vagrant といった構成管理ツール、
スクリプト言語は python が好き、気になるのはやっぱり erlang。
得意な技術は、ネットワーク機器から各種サーバのベース構築までのソロ作業。
http://twitter.com/katz_arc
