京都で開催！みんなでスマートフォンアプリを作ろう！

※こちらはエンジニア参加者募集枠です。デザイナー参加者募集枠はコチラ

開催内容

JavaScript で iOS/Android アプリが作れる Titanium mobile を使って１日でスマートフォンアプリを作ろう！という企画をやってみようと思います。

参加者をいくつかのチーム分けをして、その日のうちに設計〜実装〜完成させてしまうイベントです！
ですが、いわゆる凄腕な人たちが集まってやるハッカソンってな感じではなく、パーティーといことで、飲み食べアリで楽しくワイガヤな感じでやりたいと思いま〜す！

エンジニア／デザイナーお互い協力してアプリを作るということは本業でやっていないとなかなか体験できませんので、Titanium Craft Party で楽しく体験してみませんか？？

時間は１日しかないので、凝ったアプリは作れませんが、いっしょにアプリを作れる経験はとっても貴重ですよ！！

参加資格

  • エンジニア：Titanium mobile でアプリを開発し事がある方（またはしたい方）
  • デザイナー：スマートフォンアプリの UI デザインをした事がある方（またはしたい方）

デザイナーさんは Titanium mobile の知識は不要です。

エンジニアさんに朗報！世界初 Titanium 本の著者 @donayama 氏が参加される（かも）です！
心強い味方がいるので、コレで安心ですね（笑


参加費

参加費は 1000 〜 2000 円を予定しております。
合計を参加人数で割る感じになると思います。

持参するもの

  • エンジニア：Titanium Studio がインストールされている PC
  • デザイナー：UI デザインに必要なアプリ（Photoshop、Illustrator etc）が入った PC
