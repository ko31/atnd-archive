title 第5回 Sugamo.fuji
catch 富士を観る会
description

昨年までで富士は完食。今年からは富士山を観る撮る会へ。
初回の今回は、三ツ峠あたり。

Aプラン: 三ツ峠
Bプラン: 黒岳

三つ峠は有名コース。
歩きやすく、もともと有名で、某アニメの影響で混みそう。
撮影ポイントがいろいろあって楽しそう。

黒岳は三つ峠の隣の縦走路。
河口湖を真南に望む角度上、富士山の見映えが非常に良い。
稜線歩きを楽しめそう。

Bプラン: 黒岳縦走ルート が開放されました。

スケジュール

新宿に6時集合。
富士急行線は SUICAは使えない 。現金を用意するか事前に窓口等で購入しておくこと。

06:00 集合。
06:22 新宿。中央線中央特快(高尾行)
07:05 高尾。中央本線（東日本）(大月行)
07:48 大月。富士急行線(河口湖行)。
08:20 三つ峠。
08:43 河口湖。
09:00 富士急山梨バス。天下茶屋行き

以下、コースタイム

  1. 天下茶屋→御坂峠まで25分
  2. 御坂山まで40分
  3. 黒岳まで1時間35分
  4. 破風山まで40分
  5. 新道峠まで20分（エスケープポイント）
  6. 大石峠まで1時間25分
  7. 大石峠から「大石峠入り口BS」まで1時間10分
  8. 大石峠入り口から「あけぼの荘前」まで1時間ほど

新道峠から1時間35分で「あけぼの荘前」
あけぼの荘前から16:04に河口湖行きバス
大石峠入口発だと、17:43, 18:43

新道峠まで3時間40分。ここから下ると、行動時間5時間15分
大石峠まで5時間5分。ここから下ると、行動時間6時間15分（＋1時間）

新道峠までのペースを鑑み、帰りはアドリブで。

  • 河口湖駅からの高速バス（新宿直通。17時くらいまで空きありそう）
  • 河口湖駅16:00出発のホリデー快速 富士山2号（三つ峠16:21）
  • 上記より遅れれば、普通に電車。
  • 河口湖周遊レトロバス時刻表
  • 富士急山梨バス時刻表
