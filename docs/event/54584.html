javascript で STG！

javascript で動作するシューティングゲームをみんなで作ろう！　という企画と連動した、javascript でゲームを作成するノウハウや便利なツールなどに関する勉強会＋懇親会です。

特設ページ：JSSTG 夏の javascript シューティングゲーム祭り

参加資格

連動企画へのシューティングゲーム作品投稿者の方はもちろん、ゲームの作成に興味のある方や純粋に javascript の可能性に興味のある方など、どなたでも参加OKです。
楽しい勉強会にしたいので、お気軽にご参加ください。
