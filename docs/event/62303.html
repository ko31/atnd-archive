【デジハリ×SmartWork】トークセッション＆交流会　”フリーランサーの処世術”

就職・転職・異動・起業など…春は何かと新しい「生き方」が始まる季節。
色々な選択肢がある中で「フリーランス」という生き方に憧れをもつ人は多いと思います。

今回はフリーランスとして世界をまたにかけて仕事をしている方から、フリーランスになりたての方まで、様々な経緯とフィールドで活動している３名を迎えて、『フリーランスぶっちゃけトーク』を展開していきたいと思います。

フリーランスをこれから志す方はもちろん、既にフリーランスとして仕事をしている方の参加も大歓迎！

トークセッション後は交流会も行います！
様々な方と繋がって頂き、あなたの「生き方」に繋がる話、出会いなど、何か一つでも持ち帰っていただけたらと思います！

登壇者プロフィール

　

*　和泉 直蒼氏
「伝統文化メディアプロデューサー / 現代アートイベントプロデューサー」として、芸術祭のプロデュース・WEB・印刷物・映像、カメラマン・プロジェクションマッピング・書・音楽の制作を生業とし、国内だけでなく、海外でも幅広く活動。３月からは鎌倉FMのパーソナリティも月一で勤める予定。

　*　湯山 由理氏
「WEBディレクター / CA塾講師活動 / 読者モデル」として活動。外資系航空会社CA・CA体験教室教官として6年勤務し、結婚を機に退社。元々興味のあったWEBのお仕事を勉強するために、昨年2月にデジハリ渋谷校に入学、同年11月からフリーランサーとして仕事を開始。今後はコーディング・WordPress・PHPなどのスキルを伸ばすだけでなく、『企画からの物作り』を心がけている。

　*　小菅 達矢氏
「Keinous」の屋号で活動。ヒトとヒト、ヒトとコトを繋げる通称『リンクマン』。
HTML5・CSS3・JavaScriptのフロントエンド方面を駆使したWEBやゲームの制作を得意とする。また最近では、自身の知識や経験を元にした内容を基に、講師・ライター・セミナー登壇・音楽制作など幅広い活動を行い、『関わる誰もが、挑戦するエネルギーであふれるコミュニティ』作りの形成を目指している。
