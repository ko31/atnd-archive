■イベント内容
4人で8チームを組み、以下の内容で対戦していただきます。
今回は今までやったことのないルールで進めたいと思いまして
どうしようかと悩んだ時に知り合いから
「ガチヤグラとかどうですか？」と言われまして。
( ﾟДﾟ)ﾊｯ!ガチヤグラ中心のオフって私やってない！と思い！
①【ガチヤグラ！】
ガチヤグラにて交流会を進めていければと思います。
ただ、通常のガチヤグラでは面白くない。みんなと集まって楽しみたい。。
私は悩みました。。。どうしよう。。。
そんな時またまた知り合いが「ヒメヤグラって知ってます?」

( ;ﾟДﾟ)!?!?＜ヒメヤグラ！？！？！？

そうだ・・・。ヒメヤグラをやろう。。。。。！！！！

ヒメヤグラとは！
簡単に説明いたしますと各チームごとにヤグラに乗れるのは【一人】だけ
その一人はチームで話し合って決めていただきます。
そして使用するイカ＆タコちゃんをヤグラ乗ることが出来る人は【ガール】に
その他メンバーは【ボーイ】に設定していただきます。
そうすることで【ガール】のみがヤグラに乗ることが出来るという簡単な仕組みになっています。
ボーイが誤って乗ってしまった場合はペナルティとしてリスポーンに帰っていただきます。
神様を降臨させるヤグラは神聖な領域。。。
ガール以外乗ることは許されないのです。。。
ヒメヤグラは二先で各組合せで最大三試合まで行います！

②チャージャーを避けて橋を渡しきれ～！
こちらはヒメヤグラが終わった後の残り時間で、もしやれそうならやってみたい！という
ちょっとしたミニゲーム形式の企画です！
残り時間によってはお流れになる可能性もあるのでご注意ください！
簡単に説明いたします！
ホテルニューオートロの左側にカナアミがあると思うのですが
そこをステージ中央のブロックの上から狙ってくる4Kスコー×4の射線をヒト速等を付けずに
かいくぐり渡り切っていただこうという企画です！
渡り手（仮に渡る側を【渡り手】と呼ばせていただきます）と
狙い手（4JKスコープ）をヒメヤグラで決めたチームのリーダーがじゃんけんで決めていただき
交互にプレイして頂きます！

渡り切った皆様には【回避のプロ】の称号を授与させていただきます！
全員を撃ち落とした皆様には【射撃のプロ】称号を授与させていただきます！

とまぁ、茶番はさておき。
