服部野茶会

イベント ユーザー
event_id 40281
title 服部野茶会
catch 服部緑地でお茶会しますねん。
description

豊中市の服部緑地公園でお茶会をします。
所謂「野点」というやつですが、お抹茶を点てる茶道とは違い普通に紅茶、中国茶、日本茶を淹れます。

当日の淹れ手は私おいなりと日本茶インストラクターのいっしーさんが淹れます。
正午くらいから木陰でダラダラとやってますので是非お立ち寄り下さい。
野外ということで細かい作法は無視したゆる～いお茶会を目指してますのでお気軽に参加して下さい。

お茶には詳しく無いけど勉強したい！という方も遠慮なくご参加下さい。
日本茶はいっしーさんが、紅茶はおいなりが優し～くお教えします。
自分で淹れてみたいという要望もあれば淹れて頂きます。
自宅で淹れてるお茶がうまく淹れられないから美味しく淹れてみてよ！という人はその茶葉を持参していただければ研究して指南します。
お茶なんて興味ねーよバーカという人は甘い物を食べに来て下さい。
お茶は無料で飲み放題です。

当日の茶葉は専門店でも飲めないような高級茶をはじめオーソドックスなタイプからマニアックなタイプなものまで用意していきますのでこの期を逃すと損しまよ奥さん！
※注意：香料で着香したフレーバードティーは淹れませんのであしからず。

参加費用は頂きませんが、代わりにお茶請けを持参して下さい。
自分で作った自慢のお菓子でもどこかで買ってきても良いですし、現地で作るねんという猛者はそれでも良いです。

開催場所は公園内の「谷あいの原っぱ」あたりの林の中に入った木陰を予定してますが当日まで細かい場所はわかりません。
大勢の人がBBQをしてる場所なので谷あいの原っぱにあるレストハウスに着いたら連絡していただけると迎えに行きます。

自由参加ですが来られる方は一応参加表明をしておいて下さい。
来られる時間も帰る時間も自由です。
ランチ後に野外でお茶しませう。

集え！はっちょりりょくちぃ～！
