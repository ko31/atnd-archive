Androidユーザの交流会的な飲み会です。
秋葉原にある、知り合いのカフェをこのイベント用に開けてもらえることになりました。

秋葉原Androidアプリ開発勉強会の懇親会も兼ねていますが
・Androidユーザ
・携帯のキャリアやメーカーの方
・Androidを使ったビジネスに興味のある方
など、アプリ開発はしていなくても、Androidに興味があって、
Androidを酒の肴に、いろんな人と交流したい人は是非ご参加ください。
大型ディスプレイがあるので、そちらでアプリの紹介などをしていただける方も募集しています。
（twitterで@mouri45までご連絡ください）

月１くらいでやろうと思っているのですが、料金的なところがちょっと悩み中です。
今回は試しに¥3,000円で
料理3〜4品
ビール２杯orソフトドリンク／その他アルコール3杯
でお店のほうにお願いしてみました。
あとは、個別で頼んでいただく感じで考えています。
（もし、ちょっと覗いてみたいだけの方は単品でもたぶんOKです）
