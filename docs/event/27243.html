title 観桜会2012
catch かくし芸必須で。
description

　
よし50％ォ！

というわけで毎度おなじみ微妙な降水確率となってまいりました。

ちょうど満開になるっぽいのにね。

みんなてるてる坊主しこたまつくって軒先に吊るしといてください。

僕と黒ウサギはこれ以上余計なハリキリ無用ということで。

　
　
そんじゃすちゃらかちゃんちゃんと最終確認です。

【集合場所について】
→当日早朝より場所取りをします。その首尾如何によってどこにお弁当広げるかが変わりますので、どこに陣取ったかをまた各自に連絡します。ヘンなとこでもゆるしてね。
　
　
【各自持参品について】
→飲物は各自で用意してね。ちょっと歩くと酒屋さん・自販機はあります。言わずもがな飲酒しての運転はダメ。絶対。
→食器類はお弁当に箸はついてくるでしょうし、多少の紙皿割り箸紙コップはこちらで用意します。でもマイ皿マイ箸マイカップあるとエコだしかっけーっス。
→敷物は基本薄手のブルーシート一枚ぺろんレベルですので、より快適なお花見をお求めならば座布団的なクッション的なチェア的な奴を持参ください。
→防寒対策・雨対策は各自抜かりなきようお願いします。
→あとは必要かな？と思うものを思いのまま持ってくればいいと思うよ。

　
【雨天の場合】
→こりゃぁダメかな？ってレベルの雨の場合でもイベント自体は行いますので何はともあれ弥彦まできてね。中止はありません。
→弥彦公園に集合してぐるりと散策し、そののちに弊社まで移動して頂いて食事会といたしましょう。電車ご利用の方は駅まで送迎いたしますよん。

　
【その他】
→何かご質問等はコメでもメールでも電話でも。
→遅刻早退はある程度ご自由に。でも欠席の際は連絡下さい。
→んー。

　
　
　

それでは皆の衆。

諸事万端整えて。

満開の桜の下で会いましょう。
