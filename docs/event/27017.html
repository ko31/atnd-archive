「WEBマーケの仕事とはなんぞや？」

短期の登録目標、中長期のPR戦略、自社HPの最適化、LPO、TD作成、媒体選定やADNW運用での代理店さんとのやりとり、ソーシャルメディア、社内ヘの広報、広告の入稿作業や運用、レポーティングの嵐、アトリビューションに第三者配信ってなんだよ。　etc・・・

なんでもかんでも任される割に、あまり業務が理解されず、社内で相談出来る人も少ない、、、せっかく裁量も大きく、面白い仕事をしているのに、ひとりで悩みがちな毎日・・・。そんなWEBマーケター（すみません、かなり自分の気持ちに寄ってますが 笑）が集って、「悩み」、「つけるべきスキル」、「未来」、「周囲とのコミュニケーション」、「こんなやり方でうまくいった！」、「みんなどんな勉強してるの？」
などなど、「もっと楽しくWEBマーケをする」、果てには、「日本のWEBマーケを楽しくするぜぇ！」・・・

大げさになってしまいましたが、まぁ、とにかく「WEBマーケを話したい！！」というWEBマーケターが集ったら、知識共有とかもできるし、面白いよね？という思いつきの会です。

タイトルにあるとおり、特に「登録獲得」をミッションにしている方中心に
集まれれば良いかと思っておりますが、そうでなくても全然OKです。
（極論、画像の本、読んだ人ならいいです。。）

とにかく、話しませんか？
