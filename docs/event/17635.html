title 広島マッピングパーティ
catch OpenStreetMapに触れてみよう！
description

OpenStreetMap上に地図を描く方法を経験者とともに体験するイベントです。
経験者・未経験者を問わず歓迎致します。お気軽にご参加ください。

1.日時、集合場所
7/17(日)10時30分～17時(10:30集合)
広電(路面電車)　県病院前

雨天決行（歩ける程度なら）

2.費用
・参加費無料
・交通費、食費等は各自で負担ください。

3.募集人員
・制限はありませんが5名ほどを目安に考えています。

4.持参するもの
・最低限筆記具があればOKですが、以下のようなものがあるとさらに楽しめます。
・デジカメ
・スマートフォン
・GPSロガー
・モバイルなPC

5.服装など
・徒歩で歩く予定ですので暑さ対策を考慮した軽装でお越しください。

6.目標
＜共通＞
OSC2011広島 の開催予定地である 「県立広島大学」周辺のマップ を充実させる
＜初心者＞
・OSM上にノード（お店などの点）を打てる
・OSM上にウェイ（道路などの線）を引ける
＜経験者＞
・現地でなければ分からない情報の収集と登録(道路の詳細属性etc.)
・初心者ヘルプ

7.タイムスケジュール（当日の状況により多少変わる可能性があります）
10:30～11:00 行動予定の説明、周辺を歩いてみる
11:00～14:00 マップ情報収集
14:00～17:00 マップ編集作業（喫茶店等を予定）
17:30～19:30 懇親会（自由参加、実費割り勘）

8.当日までの事前準備
＜必須＞
・事前にOpenStreetMapアカウントを作成しておいてください。
手順はこちらのビデオを参照
＜任意（可能な方のみで構いません）＞
・県立広島大学周辺のBingによるリモートマッピング（建物など）
・AndroidやiPhone向けのOSMマッピング用アプリ（GPXファイルロギングアプリまたはPOIエディタ）のインストール
・以下の関連情報をご一読ください。

9.関連情報
◆OpenStreetMap操作ガイド（Potlatch2編） (約6MB)
◆OpenStreetMap
◆ドキュメント（日本語トップ）
◆日本でのタグ付けの基本（主に道路）
◆タグ一覧（カテゴリ別）
◆タグ一覧（50音順）
◆iPhone関連
◆Android関連

10.マッピング（マップの描き方）の例
例えばコンビニをOpenStreetMapで表現する場合には以下のように描きます。
1)まずコンビニの位置にノード（点）を打ちます。
2)そのノードに以下のようなタグ（属性を表わすキーと値の組み合わせ）を書き込みます。
shop=convenience （コンビニ）
name=FamilyMart （名称）
