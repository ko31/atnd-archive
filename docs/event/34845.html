ビジネスモデル・ジェネレーションは、
ビジネスモデルを9つの要素に分解することで、より深いレベルで
顧客、販売チャネル、パートナー、収入の流れ、原価構造などを理解する手法です。

この度、全国各地でビジネスモデル・ジェネレーションの
ワークショップ講師をされている
浅野智 氏をお招きしまして
福岡の企業を実例にワークショップを行います。

経営に関わる方、ビジネスの現状分析をしたい方、
クライアントを深く知り提案していきたい方の
ご参加をお待ちしております。
