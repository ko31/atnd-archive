当日の成果

Awodey本（下記参照）の Exercise 1.9 (a), (b) を解きましたとさ．

  • Coqチーム : github
  • Agdaチーム ： github

概要

各自、圏論の問題を持ち込んで、Coqを使って証明する one-day proofathon です。

当日共有するライブラリは A. Megacz, Category Theory in Coq です。

  • こいつを写経するもよし、
  • 自分で圏論の定義書いてこいつで答え合わせするもよし、
  • これを使って適当な問題を証明していくもよし、
  • まったく無視して独自路線に走ってもよし

なお、S. Awodey, Category Theory （Awodey本）があると幸せかもしれません。

Coq初心者も圏論初心者も歓迎。周りの人に聞きまくってください。

なお、過去会参加者は、そのときの続きをやってもらっても構いません。

タイムテーブル

10:00-10:30 始めの時間（準備、自己紹介や意気込みなど）
10:30-18:00 ひたすら証明を書く時間
18:00-19:00 成果を披露する時間

注意事項

  • 各自、自分で証明したいお題、やりたいことを自分で持ち込んでください。こちらからは特に用意しません。
  • Coqが実行出来る環境を各自用意してください。また、上記のライブラリも、各自ダウンロードしておいてください。
  • 当日の部屋は 「CLTT読書会」名義 で借りています。お間違えなきよう。
  • ネットワーク環境は、準備できるか不明なのでこれも各自で。
  • お昼はどうするか、始めの時間で考えます。
