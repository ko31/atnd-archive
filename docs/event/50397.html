グロービス名古屋校の皆様、お待たせしました。GECカフェ名古屋（イベント名ピッチ大会）Vol.2を開催します！

前回に続き、GEC名古屋とコワーキングスペースMYCAFEの共催として、「ピッチができる場づくり」を提供していきます。「飲みながら、柔らかい雰囲気でピッチをする／聞く」という場です。

当初はゆるくスタートしつつも、今後の展開としてはVCを巻き込んでのピッチ大会に育て、「名古屋発の起業がドンドン生まれる」生態系を創っていきたいと考えています。

GECメンバー、GMBA生だけではなく、単科生、GMS生もピッチ可能です！更に、グロービス生（単科生，GMS生含む）の同伴であれば、グロービス外の方でも参加OKです。同様に、MYCAFE会員の方もピッチ予定です。

前回の2月開催時、当初のピッチ予定者は数人だったものの、「俺にも喋らせろ！！！」という方が次々と表れ、ピッチ開始から会場終了の時間までずっと誰かが前に立って喋っている、という状況に…。大盛況でした！

＜5分ピッチ者募集します＞
時間が許す限り5分ピッチを行っていきます。
ピッチ内容はビジネスに関する事であれば自由です！「こういう事業アイデアを持っているけど、意見が欲しい」「このビジネスを手伝ってくれる人を募集したい／紹介して欲しい」というものから、「今こういう事をやっているのだー！」というビジネスの紹介，「今度プレゼンしなきゃいけないんだけど、練習させて！」というものまで、自由に柔らかく使って下さい！

ピッチ希望者は事前に日高まで、FBのメッセージにて連絡下さい。尚、当日の飛び入りピッチも大歓迎ですが、時間切れとなる可能性もあるので、「絶対にピッチしたい！」という方は必ず事前に連絡下さいね～！
