イベント概要

2014年創業して以来、スペース数は1万件を突破しました。

当初はRails Onlyで構成されていたサービスも機能追加や組織の拡大に従って、
Microservicesの導入が必要になってきました。

スペースマーケットではフロントエンド/バックエンド両方にJavaScript(React.js/Apollo/Typescript)を採用しています。 このMicroservices化により得られた知見を皆さんに共有できればと思います！

以下の内容を発表する予定です。

  • スペースマーケットの開発組織について
  • スペースマーケットで使われている技術について

※ 発表内容は変更される可能性があります。ご了承ください。

登壇者・登壇内容紹介

小林 春彦（リードエンジニア）

ヤフー株式会社でYahoo!Days、Yahoo掲示板などソーシャルサービスの開発/運用エンジニアとしてサービス開発に従事。 その後、ヤフー全体で利用されるミドルウェアの立ち上げ・開発を経てRealNetworksに入社。 リードソフトウェアデザインエンジニアとして動画共有サービスRealPlayのローンチに従事。 2014年8月よりスペースマーケットに参画。

万谷 勇輝（フロントエンドエンジニア /リーダー）

新卒でSIerに入社しQAエンジニアや新規事業の企画・開発に関わった後、2017年2月にスペースマーケットにiOSアプリエンジニアとして入社。 iOSアプリ開発はもちろん、ReactNativeを使用したハイブリッドアプリ開発を経て、Webフロントエンドの開発など、アプリ〜Webまでフロントエンド全般の開発に関わる。

山本 陵介（バックエンドエンジニア / エンジニア）

Webクライアント、Web APIの企画開発を担当。 元男子シンクロのパフォーマー・インストラクター。その後Webディレクターとしてサイトリニューアルや新機能開発のUI・UX設計、制作進行の経験を経て2017年1月よりWebエンジニアとして株式会社スペースマーケットに入社。 業務では直近だと新規事業や検索サービスのマイクロサービス化を担当しているので、そのあたりの技術的なことについて話します。
