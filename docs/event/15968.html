放浪息子ロケ地めぐり＋お茶会

放浪息子イベントに合わせたtwitter有志のオフ会です（放浪息子イベントおよびその主催者とは一切関係ありません）。飛び入り、途中参加歓迎です。各自の飲食にかかる費用や交通費以外に、参加費などは必要ありません。

参加希望者のかたはこのATNDで参加登録して(twitterのアカウントでログインできます)、twitterで @hourou_off をフォローしてください。参加が確実でなくてもかまいません。ATNDでの登録方法(パソコンで確認しました)は以下のとおりです。

  • 「ログインしてください」の赤いボタンを押す
  • 「おすすめログイン方法」から「Twitterでログイン」を押す
  • Twitterのidとパスワードの入力画面が出てきたら入力。出てこない場合もあります。
  • 「ログインしました」という表示が出るので「このイベントに参加登録する」の赤いボタンを押す
  • 参加コメントを書いて（書かなくてもかまいません）ずっと下にある「このイベントに参加登録する」の赤いボタンを押す

お知らせのほか、開催中の途中経過などは随時 @hourou_off からツイートします。関連ツイートにはハッシュタグ #hourou_off をご利用ください。

当日は放浪息子の完全設定資料集など、目立つように目印を持ち歩きますので、お気軽に声をかけてください。

ロケ地めぐりは、ニ鳥くんの自宅や中学校、1話の歩道橋のモデルになったあたりを歩きます。参考ページ：http://integratyper.blog11.fc2.com/blog-entry-569.html

おおまかにつぎのようなスケジュールで考えていますが、人数や交通状況などをみながら調整します。

11:00 小田急線成城学園前駅中央改札口を出たところに集合
11:20 砧小学校・大蔵住宅周辺のロケ地を散策
12:50 成城学園前駅付近で昼食
13:30 台場にむけて移動
14:30 台場アクアシティのフードコートでお茶会
15:30 解散
