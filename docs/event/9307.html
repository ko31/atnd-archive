今年最後の登山

先日、霧ヶ峰・赤城山と登ってきましたが、紅葉がそろそろ見頃です。
多分、今年最後の登山になると思います。
今回登る山は神奈川県の仏果山（747m）という山です。

装備、持ち物について

  • 赤城山の時点ですでに肌寒い気候でしたので、
    雨具と防寒は忘れずに お願いします。
  • 食料は前日までに用意して下さい。
    都内を早朝出発の予定ですので、途中で購入する時間はほとんどないと思います。
