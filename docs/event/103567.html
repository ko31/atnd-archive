🏀就活の気分転換に体を動かしたい！

そんな学生さんを大募集！

IT事業と語学事業を展開する全研本社株式会社が、学生さんとの交流会を行います。
フットサル交流会がご好評につき、バスケットボール交流会も開催決定！一緒にプレイを楽しみませんか♪

就活中の学生さんなら、どなたでも参加ＯＫ！
社員と一緒にバスケをしながら、仕事のことや働き方などフランクに話せる交流会です。

女性の参加も大歓迎！初心者でも気兼ねなくプレイできます(^^)
お気軽にご応募ください！
