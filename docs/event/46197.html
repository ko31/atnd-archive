増えていく WordPress 案件。実務に直結する技術をあなたに

WordPressテーマ制作に関する講義やブログ記事は溢れています。しかし、実務においてはイレギュラーな仕様や実装のコストパフォーマンスを考慮できるスキルが求められています。

本セミナーでは、jQueryプラグインのWordPressへの組み込み、フックを使ったWordPressプラグインのカスタマイズなど普段初心者向けの講座ではあまりお話しない内容をコードを触りながらお話します。特にカスタム投稿タイプやAjaxとの連携が必要な下記のjQueryプラグインを中心に扱う予定です。
- bxSlider（スライドショー）
- Magnific Popup（ポップアップ）
- Infinite Scroll（無限スクロール）

また、本セミナーでは全時間の半分を割いてハンズオンを行います。
座学で勉強した気にならず、実際に手を動かして持ち帰っていただけます。
今回紹介するメジャープラグイン以外にも、本セミナーで得た知識を利用し、様々なプラグインに応用していくことができるでしょう。
今後の実務に直結することをお約束いたします。

Web Developer Selection（WDS）とは

Web Developer Selection（WDS）は、「数多くある技術から、Web Developerが最適な知識や技術をセレクション。セッション+ワークショップで提供する、とことん実践主義にこだわった技術力向上を目指す場」です。
「レスポンシブWebデザインの実践ワークフロー」でDesign Spiceの小川さんや、「FireworksユーザーのためのPhotoshop講座」をStocker.jpのなつきさんに、また、「CSSレイアウトの理解と実践」ではKOJIKA17のこじかさんなど、多数の講師の方にお話頂いております。

講師プロフィール

井村圭介 氏
Webデザイナー。WordPressに関して国内法人の中でもトップクラスの技術を誇るプライム・ストラテジー株式会社での勤務を経て2013年に独立。
デザインだけでなく、マークアップやWordPressを中心としたCMS構築を扱っている。
WordCamp Tokyo 2013では公式サイト制作を担当した他、デザイナーセッションに登壇。
Stocker.jp/SpaceやマルチメディアスクールWAVEでWordPress講座の講師をつとめる。
