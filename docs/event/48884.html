そもそもこれはどういう会なの

「Haskell」 というプログラミング言語を勉強してみよう、という会です。
Haskell は 強い静的型付けの関数型プログラミング言語、という位置付けらしいです。
何を言っているのか、という感じなのですが、つまりプログラミング言語にもいろいろ種類や方言などがあるわけなのです。
例えば C言語 は弱い静的型付けの手続き型プログラミング言語だったり、 Java はオブジェクト指向言語だったりします。

これらは全てプログラミング言語であり、書き方が全く異なります。
しかし、違う言語を学ぶことによって新しい考え方を得たり、今まで理解できなかったことが理解できたり、自分の得意な言語を見付けることができたりするなどの発見があります。

そこで、最近流行りの強い静的型付けの関数型言語である Haskell に手を出して、どういったものなのか見てみよう、という目的があります。

なので、対象としてはプログラミングやプログラミング言語などに興味がある人などが当てはまります。
