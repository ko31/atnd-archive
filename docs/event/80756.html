イベント概要

エンジニア目線の求人サイト「Forkwell Jobs」主催イベント第20弾！

今回は「バイトル」、「はたらこネット」、「ナースではたらこ」などの求人メディアを運営している、ディップ株式会社とのコラボイベントを開催します。

既存サービス「バイトル」「はたらこネット」は、バックアップ、災害対策を目的にAWSの導入、iOS版「バイトル」にはビーコン対応するなど、様々な技術が導入されています。 今まで公開されてこなかったサービスの裏側をエンジニアが語ります！

これらの技術の導入や、新サービスの開発などに関わっているエンジニアの方々の中には元々SIer出身の方が多く、どのような前職での経験を活かして現在の仕事につなげているかについて探ります。

また、 『入社後に身につけるべきスキルとは？』
か身についたスキルはどんなものがあったのか？

今後、事業会社や大規模サービス運営する会社で勤務をしたいと考える方にとって知っておいていただきたい内容です！

▼こんな人におすすめ

  • 現在SI企業に所属しているが、事業会社への転職を考えている方
  • サービス開発に集中できる環境を希望している方
  • 自分の今までの経験（SIでの）が活かせるか不安な方
  • 新規事業の開発など市場のニーズを感じながらサービス開発をしたい方
  • ソースコード管理はGitでやりたい方

コンテンツ内容

第一部：ディップ社の開発環境について

  • ディップ社の中でのエンジニアの働き方
  • エンジニアのサービスへの関わり方
  • 開発環境の変化、歴史
  • 開発チームの雰囲気、働き方
  • エンジニア以外（デザイナー、インフラ、ディレクター、営業）との関わりについて　など

第二部：実際に働いているエンジニアによるパネルディスカッション

  • ディップ社を選んだ理由は？
  • 今までの経験の中で何を御自身の得意分野として活かせているか？　　
  • 入社してみて感じた良かった事、前職での経験が活かせる事は？　　
  • この会社に入社したからこそ得られたと思う経験について　　　など

第三部：懇親会

ディップ社のエンジニアとお酒や軽食をお召し上がりいただきながら気になることを個別に聞ける会です！

登壇者プロフィール

ディップ株式会社　　

栗生氏
国産GISエンジン開発会社に新卒入社し、GIS関連のパッケージ製品開発や受託プロジェクトに携わりC言語、Java等を使用した開発だけでなく、技術営業や顧客サポート業務の経験を積む。 その後小規模SIに転職し、博物館の情報展示システム、顧客先の社内業務支援システムや採用管理システム、Webサイトなど、さまざまなシステムの導入から開発、運用、サポートまで幅広くかかわる。 2012年4月にディップ株式会社にジョインし、現在はディップのスマホアプリ全般の開発を行う部署で開発ディレクション業務を担当。 より多くの方々に使いやすく、喜んでもらえるアプリを届けるため、日々奮闘中。

上田氏
新卒で某メーカーのIT関連子会社に入社し、グループ内の基幹システムの開発・保守を担当。中国オフショア開発のブリッジエンジニアとしてかかわる。その後、フィーチャーフォン向けWebサイトの開発を行っている会社へ転職しキャリア公式サイト、企業サイトの開発・保守・運用を担当し、某メーカーのIT関連子会社に転職。（1社目の競合）基幹システムのリプレイス案件、ITとは全然関係ない案件、新規事業企画など渡り歩く。 2015年6月にディップ株式会社にジョインし、Webサービスの開発・保守・運用を担当。企画から携われるエンジニアとして成長するべく奮闘中。 技術は雑食。なんでもやります！インフラにも興味あります。

白川氏
20代は営業・小規模SIer・広告制作・個人システム開発営業等を経験し、30代から本格的にWebシステム開発を開始。 業務上の経験言語はDelphi、JSP、PHP、Perl。主にフィーチャーフォン向けWebサイトの構築から保守運用、開発リーダーを努める。 2011年4月にディップ株式会社に参画、プロジェクトマネジャーを経て、現在は組織マネジメント、チームビルディング、エンジニア採用を主な業務としている。 個人活動としては、技術を錆びさせないために、一日一行以上はソースコードに触れることを目標。ココ数年はnode.jsを好んでいたが、最近の流行りはgolang。好きなエディタはAtomとIntelliJ。
