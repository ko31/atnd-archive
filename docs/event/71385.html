OSC 2015 Tokyo/FallでCLSxTokyo vol.1を開催

OSC2015 Tokyo/Fallにて、第1回OSSコミュニティー・リーダーシップ・サミット “CLSxTokyo vol.1”を正式開催します。

公式ハッシュタグ： #CLSx #CLSxTokyoOSC 公式ハッシュタグ： #osc15tk

開催概要

2015年10月25日(日) 14:00-14:45形式：【アンカンファレンス】

概要：オープンソースコミュニティ運営を考える。リーダーたちが運営の問題をもちより議論する、コミュニティー・リーダーシップ・サミット会場: 明星大学 日野キャンパス 28 号館

  1. 開催教室：303(仮)(28号館 3F)
  2. 企画詳細：http://www.ossj.jp
  3. 参加申込：事前登録不要(どなたでもご参加いただけます)
  4. 担当：OSS Japan CLSxTokyo 実行委員会
  5. 講師：ファシリテーター：太田 昌文　コーディネーター：今駒 哲子 CLS
  6. 本家サイトhttp://www.communityleadershipsummit.com/ OSPN:https://www.ospn.jp/osc2015-spring/modules/article/article.php?articleid=3

『CLSxTokyo vol.1 』～オープンソースコミュニティの意義を考える～
