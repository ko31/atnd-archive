東京リージョンの開設で、ますます期待が膨らむ「Amazonクラウド」について、エバンジェリストの玉川氏に解説いただきます。
特に、今回の東北太平洋沖大震災で直面した課題や影響なども、事例を交えてご紹介いただきます。

また、社団法人 クラウド推進機構（CUPA）代表理事 荒井氏、理事 竹下 氏にも登壇いただき、クラウドの最新情報について解説いただきます。

【主なセミナー内容】＊変更になっています

●はじめてのAWS (20分 玉川氏)

　初めての方向けに、AWSをクイックに分かりやすく説明します。
　いま非常に注目を集めているAmazonのクラウドを20分で理解して
　しまいましょう。

●JAWS-UG & AWSの震災復興の取り組み(15分 竹下氏)

　JAWS-UGは震災復興に取り組んできましたが、その活動を紹介します。

●クラウドを用いたディザスタリカバリとAWS最新動向 (25分 玉川氏)

　大震災の影響もあり、AWSのお客様からクラウドを用いたDRについて
　多くの問い合わせを頂いております。
　ディザスタリカバリの話に絡めて、AWS最新動向の話をしたいと思います。

●WordPress on AWS -WPを5分で立ち上げよう-　(10分 玉川氏)

　クラウドのデプロイを非常に簡単にするCloudFormationというAWSの
　最新機能を用いて、5分間でWordPressを立ち上げてしまいましょう。
　CloudFormationの説明とともに、WPをその場で5分で立ち上げてしまい
　ます。

●FMS on AWS (10分 荒井氏)

　Adobe FMSをAWSの上で走らせてみましょう！
　従量課金制であのFMSが使えてしまいます。

●福岡有志によるクラウド活用ＬＴ（40分 ライトニングトーク）
