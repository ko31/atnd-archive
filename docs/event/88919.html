ついぴの会について

ピアノが好きな方であればどなたでもご参加可能です。ピアノ連弾や練習中の曲、暗譜でなくてもOKの「ゆる〜い弾き合い会」です。ピアノが弾けないけど聴いてみたい、という方も、ピアノ以外の楽器の方も、ぜひ気軽にご参加ください！

初めての方は「ついぴとは」もご覧ください。

開催概要

日時：
　2017年7月15日（土） 13:00〜16:00

会場：
　金沢湯涌創作の森　交流研修棟 1F

参加費：
　演奏者 500円
　聴くだけ 無料

定員：30名

  • 11時までに開場します。個別リハーサルはこの時間帯に行います
  • 付近に飲食店が少ないので、外で昼食を。飲食持参の場合は受付棟にてお摂りください
  • 会場への送迎を希望される方はお伝えください

参加方法

  1. 本ページの「イベントに参加登録する」ボタンからエントリーしてください。 聴くだけの方も必ずエントリーしてください。 Twitter, Facebook, Google+ のいずれかのアカウントが必要です。
  2. 7月9日（日）12:00までにアンケート回答をコメントボードに記入する。（アンケートは下記参照）
  3. 当日、元気についぴに参加する。

アンケート（7月9日12:00までにお願いします）

  1. お名前（Twitter, Facebook, 本名いずれも可）
  2. 演奏される方は曲名、作曲者、演奏時間（10分以内）
    複数曲可。 聴くだけの方は「聴くだけ」とお書きください
  3. 自己紹介または演奏する曲についてのコメント
  4. ひとこと質問：スイカやかき氷など、夏にしか食べないものがあると思いますが、その中で絶対に外せないものを教えてください！
  5. その他
    • 演奏時間が10分を超える場合は幹事にご相談ください
