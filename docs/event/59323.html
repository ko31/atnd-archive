概要

今年で3回目の開催となるSADP

10月~11月に渡る2ヶ月弱の期間で、大学(院)生がアプリを開発しました。
本年度は6チームが真剣に練った各々の制作物を紹介します。

当日の内容構成

  • 午前の部
    • 各チームがアプリ開発を通して学んだ技術的Tipsをご紹介します
  • 午後の部
    • 6チームによるアプリコンテストを開催します

参加費無料, 途中入退場自由です。
