description

※主に知り合い対象の企画です。主催に面識のない方は参加をご遠慮ください※
（なお企画進行は調整中のため、ところどころかわります）

【概要】

「漫画の描き方は人それぞれあるけれど、一体何がどう違うんだろう？」
「何考えてコマ割っているの？」
「他人と比べると自分はどんな漫画を描いてるんだろう？」
そんなことをぼんやり知るために、同じプロットからみんなで漫画を描いて見比べてみよう！きっと面白い！という企画です

【主な流れ】

Maity（http://meity.jp）というサイトを利用して、プロットやおしらせ、描いた漫画などをDMで共有していきます。
＠絵描きさんの場合
1.このイベントに参加
2.プロットのメモが作成されるとURLがDMで届きます
3.さらっと漫画を描きます
4.そのメモのコメント欄に画像のURL（ついぴく等）を乗せて共有します
(5.他の人のも見てみてコメントとか質問とかしてもたのしいかも)

＠文字書きさんの場合
1.このイベントに参加
2.Maityにログイン
3.メッセージを作成しプロット（やそれにコメントなど）を書き込みます
4.「閲覧できるメンバーの追加」＞「ATNDから選ぶ」のID欄にこのイベントのIDを入力します
5.送信するとメンバー全員にプロットのURLがDMで送信されます
6.あとは漫画が出るのを待つだけ

【決まりごと】

＞プロットについて
・創作に限ります
・漫画にしたときに1~2P程度になる文量がベストです
・プロットという体ではありますが、形式はSSのようであったり台本のようであったりお好きにどうぞ。なんとなく絵書きさんが書きやすいかなあと思う感じにしてください
・登場人物の細かい設定などがある場合は補足としてかいても良いです
・イメージと違う！とかおこるのはなしです
・絵描きだけどプロットも書いてみたいとか全然ありです

＞全体について
・まったりとしたお題企画なのでゆったりやってください
・比較企画ではありますが、批評企画ではないです。特に書き手側が望まない限り、かしこまった批評はなしの方向で。個人でやってください。
