★講演内容

「Unity」というキーワードを目にする機会があっても、
自分とはどこか遠い世界の出来事のように感じてはいませんか？
ひとりでゲーム開発をやるなんてとても無理・・・
大手企業が続々と参入する現状では、そう考えるのも無理はありません。
実際問題として、素人が片手間に作ったゲームが世間に注目され、大ヒットする可能性はかなり低いでしょう。
しかし、Unityの用途は、なにもゲーム開発だけに限定されているわけではないのです。
教育や医療、建築、物販など、さまざまな分野にゲームエンジンの利用が広がっています。
仮想空間に構築された本物そっくりの箱庭ワールドは、お堅いビジネスの世界にも極めて適合性が高いのです。
開発者のコミュニティが成熟し、技術ノウハウが蓄積した今だからこそ、Unityをはじめてみませんか？
