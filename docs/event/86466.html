description

1月の横濱クリエイティブ新年会に続きクリエイターとクリエイターが「出会い」
また「学び」の場となる場所を作りたい。またビジネスに繋がり発展の一助となる
事を目的に企画しました。今回は勉強会！
勉強会テーマは、
～「やっちまったな！」ＰＰＡＰから学ぶ商標の大切さ～
クリエイター（デザイナー、イラストレーター、カメラマン、ライター、プログラマ、
映像制作、広告業界等）の方々、実はあなたのお仕事にもとっても関わりのある知的財産を守る商標！今回はその商標にスポットを当ててみたいと思います。講師には特許事務所湘南ラボ　湯口文丸弁理士の先生に講義を行ってもらいます。
また、勉強会後には参加者や講師の先生を交えて更新会も行います。
名刺交換・自己紹介の時間を設けています。
（多めに名刺を持参ください。前回も名刺切れの方が続出しました。）
