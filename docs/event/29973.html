闇鍋プログラミング勉強会の資料とその他

エンジニア（主にプログラマ）が主体となって発表する学生、社会人関係なしのプログラミング全般の勉強会です！


●対象者

  1. 仕事/趣味関係なくプログラミングをしている
  2. プログラミングが好き
  3. プログラミング言語が大好き
  4. 普段使っている言語以外にも興味がある

●勉強会のテーマ

この勉強会では、特にテーマを設けません。
しいて言えば「プログラミング全般」です。
プログラミングに関係するものであればなんでも構いません。
主催者は「ごった煮」「闇鍋」を目指しています。
