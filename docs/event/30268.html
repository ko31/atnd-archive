title 仕事術コンテスト
catch ～あの人はどう仕事をしているの？　今すぐ実践、等身大の仕事術！～
description

毎日仕事をこなされている方なら、一度は思ったことがあるはずです。

”隣のあの人は、いったいどのようにして、日々の仕事をこなしているのだろう？”

日々山のようになっているタスクを前に、少しでも効率を上げようと、悪戦苦闘の毎日。
ビジネス書を参考にしようとしても、まさに玉石混交、
使えるものもあれば、実践には遠く及ばないものまでさまざま。

それでも山のようなタスクを効率よくバリバリこなしている、隣のあの人。
ビジネス書には載らないけれども、まさにそこには、さまざまな創意工夫があるはずです。

そんな創意工夫を共有し、アドバイスをもらい、競い合い、高めあっちゃったりしちゃいましょう。

本イベント参加者の方、全員に、持ち時間８分で、
ご自分が実践していらっしゃる仕事術をプレゼンしていただきます。

「こんな創意工夫しているぜ！」から、「ちょっとした小ワザ」まで。
もちろん、それらを踏まえた「お悩み」もOK。

すべての方の発表が終わったあと、
みなさんには、どの方の仕事術が一番すばらしかったか、投票していただきます。

見事１位に輝いた方は！　なんと！
その後の懇親会に無料ご招待！！

ぜひふるってご応募ください♪

発起人について

たかはしともこ@deep_sea_blue

今回、言いだしっぺをしておりますたかはしともこです。
仕事でもそうですが、プライベートも、意外と「やらなくちゃいけないこと」や「やっておきたいこと」って多いですよね。
うまいことみんな格好良くこなしたいわけです。
なのでビジネス書を読んで、創意工夫をするんですが、残っていくテクニックもあれば、
理想論のみで面倒で終わってしまうものもあります。
そんな中、ふと隣を見れば、みんなかっこいい事やっていて、なんだか凄そう。

プライベートの「やらなくちゃいけないこと」が重なり、手帳にきちんとメモできず、うまく管理できなくなったとき、
みんなどうしているのだろう、とツイッター上でつぶやきました。
そんなつぶやきを拾っていただき、このようなコンテストを開催するに至りました。

ご尽力くださった、しんすくさん、みずのりさん、なかさやさん、KENさん、あすかさん、感謝しています。

また、自分の創意工夫が誰かの役に立つこと、もしくは、みなさんの創意工夫、楽しみにしています。
ご参加をお待ちしております。
