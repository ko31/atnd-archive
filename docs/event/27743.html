【SaCSS ＋ とは】

実際にコードを書きながら使い方や作り方などを学び、仕事でも使えるようになる！制作出来るようになる！というのを目的とした勉強会です。

通常のSaCSSとは別物としてのSaCSS ＋（SaCSSプラス）です。

実際にコードを書いたりしながら、使い方・作り方を学び、分からなければその場で質問し、しっかりとした作り方を自身で実践しながら学ぶ勉強会です。

そのため、時間も通常の倍の時間をかけて勉強をする内容です。

今回は、前回のSaCSS vol.32でもとりあげた「CSSアニメーション」について、それを取り入れたスマートフォンサイトの作成について、実際にコードをかきながら勉強していきます。

JavaScriptでのアニメーションの差異なども一緒に確認しながら実際にコードを書いていきます。

【通常のSaCSSとの違い】

通常：
情報交換や最新技術を学ぶために、ノウハウや最新情報の共有を目的としたセミナー

SaCSS ＋：
時間をかけて、たっぷりじっくり、実際にコードを書きながら学ぶ勉強会！

【SaCSS + 6th の対象者】

  1. CSSアニメーションについてを学びたい方
  2. JavaScriptだけでなくCSSでのアニメーションも取り入れたい方
  3. 普段勉強の時間や機会がないが、集中して勉強をしたいと思っている方
  4. 個人で勉強したことの確認として参加したい方
