Microsoft Student Partners 主催 Windows ストア アプリ 開発の会 in 東京

description

10月26日発売予定の Windows 8 上で動作する Windows ストア アプリ （Metro スタイル アプリ）の開発をいち早く体験するチャンスです！

私たち Microsoft Student Partnersでは、是非あなたに Windows ストア アプリ を開発してもらいたいと考えています。まだWindows Storeに登録されているアプリが少ない 今 が、あなたのアプリをたくさんの人に使ってもらうチャンス！ 私たちと一緒に Windows ストア アプリ を作りましょう！

当日はMSPフェローの、 三村 、 稲山 、 菊池 があなたをサポートします。

タイムテーブル（予定）

時間 内容
12:30 – 13:00 受付
13:00 – 13:05 挨拶と概要説明
13:05 – 14:00 Windows ストア アプリ について
14:00 – 15:00 Windows ストア アプリ プログラミングことはじめ（C#）
15:00 – 17:00 ハンズオン開発
17:00 – 18:00 作品発表会

場所

Microsoft 品川本社
住所: 東京都港区港南 2-16-3 品川グランドセントラルタワー

部屋に直接は入れませんので、当日はエントランスホールにお集まりください。

ご持参いただくもの

  • ノートPC、もしくはタブレットPC
    • 必ず何かしらの形で Windows 8 と Visual Studio 2012 をインストールして来てください。
    • VMware Player や Virtual Box などの仮想マシンにWindows 8をインストールするのが一番簡単です。
    • タブレットPCを持参される場合は、キーボードやマウスも持参されるとハンズオンの際に便利です。
  • C#の知識
    • ある程度文法がわかっていれば大丈夫です。
  • 作ってみたいアプリのアイデア
