中部クリエイティブ業界フェスタカンファレンス2016

変化する市場と、成功のためのUnityのディベロッパー支援

Unity Technologies Japa合同会社 アカウントマネージャー 森嶋大樹氏

GAIRA（中部ゲーム産学協議会）は、愛知県、名古屋市の後援（予定）を受けて下記の日程でクリエイティブ業界フェスタを開催します。今年のクリエイティブ業界フェスタは、中部圏のデジタルアニメーション、ゲームなどデジタルコンテンツ制作を行う学生そして一般のクリエイターに向けて技術カンファレンスを実施します。

タイムテーブル　セミナールーム2(36名)

時間 発表者 内容
15:30 - 16:20 Unity Technologies Japa合同会社　森嶋大樹氏 講演

講演者名 森嶋大樹

講演者略歴

UNITED株式会社にてSSP AdStirのセールスを担当。およそ1000万DLのアプリマネタイズコンサルを経て。FULLER株式会社にてアプリデータの仕事に従事。Unityではアカウントマネージャーとしてディベロッパー支援を全国で行う。

講演内容

これからゲームを作ろうとする皆様に向けて。中部でUnityの勉強会を開きたい方向け。ゲーム市場等が大きく変わっていく状況をご説明しつつ、そのなかでこれからゲームに携わる方に何をUNITYとして、どうサポートをしていくのかをお話させて頂きます。是非、ご参加頂きお話を交わさせて下さい。
