日本語版 Zmanda Cloud Backup 4.0 が新登場

日本初クラウドバックアップのリーダー Zmanda 主催の 無償 セミナーです。
当日は、クラウドのエキスパートの玉川 憲 氏(Amazon Data Services Japan)をお迎えしています。

今回は、特に、次の様な方向けの内容となっております。

  • シンプルでお手頃価格、けれどもパワフルなバックアップソリューションをお探しの方
  • バックアップソリューションを評価中の方
  • ディザスタリカバリやクラウドストレージについてお考えの方

■ Zmanda Cloud Backup とは？

ZCBは、WindowsマシンのデータをAmazon S3に低価格でバックアップできるソリューションです。月々4.95ドルと転送コスト(0.15ドル/GB)の料金で、お手軽にディザスタリカバリが実現可能で、Windowsのファイルのみならず、SQLサーバやExchangeなどのバックアップを取得することができます。 また、ZCBは、プロプライエタリなフォーマットを使用していないので、オープンなフォーマットでのバックアップを可能にします。毎月の料金は、Amazonを通じてクレジットカードでお支払いいただけます。
