description CloudStackユーザ会(CloudStack勉強会) – 第7回

オープンソースのIaaS基盤ソフトウェアは多数ありますが、
その中でも【CloudStack】はリッチなGUIとFW, LB, HA等の機能を備え、
エンタープライズ向けの機能をサポートするなど、異彩をはなっています。
4/3にはApache 2.0ライセンスとなりApache Software Foundation(ASF)に
寄贈されたため、独自ディストリビューション開発やOpenStackとの
パッケージ化など今後更に活用の選択肢が増えることが期待されます。

第7回はゲストとして、Citrix社 Cloud Platform Group VP
(Product Marketing)のPeder Ulander 氏をお招きし、
Apache 2.0ライセンスとASFプロジェクト化によるインパクトや
今後のコントリビューションについてお話をしてもらう予定です。

是非この機会にCloudStackについて深く学びましょう！

★★当日は入館申請のため、名刺またはお名前・連絡先の記載が
必要となります。手続きの効率化のため名刺を是非ご持参ください。★★
