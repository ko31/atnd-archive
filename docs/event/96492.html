概要

北海道の Drupal ユーザーコミュニティ「Drupal さっぽろ」が主催する Meetup (勉強会)です。地域の Drupal ユーザーの情報交換と交流を目的に、月に一度、札幌と小樽で開催しています。4月の勉強会は CMSMix Sapporo の札幌開催だったので、今回は小樽での開催です。

北海道の Drupal コミュニティとして情報交換と交流の輪を広げ、さらなる活用につなげていきましょう。

参加料：200円（会場費として）

内容予定

Drupal もくもく会（13:30～17:00）

今回は定期開催の「もくもく会」（自習形式の勉強会）です。いつもの実習用公開サーバーと centos7dsm を用意します。実習サーバーには最新版をインストールしますので、実際に見て、触って、体験することができます。

入門用のチュートリアルも準備中です。初めての方も大歓迎。仕事で Web に関わっておられる方から、ホームページ構築に興味をお持ちの一般の方まで、お気軽にご参加ください。時間中いつでも出入り自由です。

持ち寄りセッション

後半はテーマ発表の時間を設けます。内容は問いません。ふるってご登壇ください！

お願い

今回の会場は施設の Wi-Fi がありません。スタッフのモバイル ルータを開放しますが、インターネット接続は基本的に各自で用意ください。
