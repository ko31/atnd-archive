みなさん　こんにちは。SOTRYの斉藤です。今年もはや2月。あっという間に過ぎていきます。売上アップの相談等を頂く機会がありますが、毎日の延長では、伸びしろは決まってくるようです。そんな時、少し方法を変える　目線を変えるといったものが有効なこともあります。

さて今回は、「googleで推奨されているサービスを利用した中小企業や初心者でも活用できるサービスの概要と設定方法について」勉強会を行いたいと思います。

サービス自体は、設定完了後基本的には、無料で行うことができます。

もちろんPC　タブレット　スマートフォン等に対応します。

この機会に、なかなか効果が出ないご担当者　より良い方法模索のご担当者の方　ぜひご参加ください。
