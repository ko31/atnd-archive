セミナー内容 / 朝活の流れ

■ セミナー内容

◎ 「クラウドファンディングを成功させるための秘訣」
プチセミナー

「クラウドファンディングで資金を集めたい!」
と思う方、
クラウドファンディングをした事がある人の
体験談を聞いた事はありますか？

今回は
クラウドファンディングで10万～100万集めたい
と考える方を対象に

● クラウドファンディング成功のためやる事
● 経験者が苦労したこと
　+
● (そもそも) クラウドファンディングとは?
について、

現在、クラウドファンディングを使って
店舗の改装費用を集めている、
「原宿・ガレージパーク」のオーナー、
野口 亮さん
　+
( おまけの1名 ) に
話を聞きます。

ご期待くださいね!!

.

野口 亮 (のぐち りょう)

原宿・ガレージパークのオーナー。

のがきっかけ。

よーし見てろよ!
毎日入りびたりたくなる、お洒落カフェに変身
させてやる!!
と、

● お洒落な内装
● 美味しい料理
● 楽しいイベント　の3点セットを整えようと
奮闘中。
