description

気持ちのいい昼下がり、ママたち、パパたちの休日を応援します。
お子さんをプレイスペースであやしながら、ハンドマッサージ、女性のための占いで癒しのひとときを楽しみ、レアもの絵本を眺めながら、家族の写真をプロフォトグラファーに撮ってもらう。
また、サイズアウトした子供服などみんなの「あげたいもの」と「ほしいもの」を交換して、リブリスの木をデコレーション。
最後にどんな姿に変身するか乞うご期待！？
こんな盛りだくさんのイベントでゆっくり一日をお過ごしください。
