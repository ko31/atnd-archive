第3回アトムハウスハッカソン

　みんな大好きアトムハウスハッカソンを開催します。

やること

　各自、コメントでやる予定のことを書いておきましょう！

会場情報

　ぼくんちの情報です。

  • 光回線の無線LAN（2.4GHzと5GHz）完備
  • 電源及び延長タップ完備
  • 安定化電源および、ブレッドボード有り
  • スピーカー（BOSE Companion3）でDJプレイ可能
  • ディスプレイ三台（27・21.5・17インチ）使用可
  • Kinect1台有
  • 夕ごはん提供有

今回のクックソン

　今回はあそなすさんのご好意により、生ハムの原木が入荷します。夜はひたすら生ハムを使った料理を提供する予定です。飽きるほど生ハムが食べられるのはアトムハウスハッカソンだけ！
