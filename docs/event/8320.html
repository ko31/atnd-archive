description

身の回りの事柄をフリートークスタイルでお話する【マーケティング庵】
では、「CD不振の音楽業界。どう再生するの!?」と題して、10月の研究
会を開催します。

参加資格者は、マーケティング関係者や広告関係者、学生、マーケティン
グに興味のある方など、特に制限していませんが、むしろ、直接マーケテ
ィングに関わらない方々の自由なご参加をお待ちしています。
特に、今回は音楽が三度の飯よりも好きな方々にお越し願えれば幸いです。

今回のテーマは、アメリカではすっかり影を潜めてしまった大型CDShop
に顕著なように、音楽産業の構造変化と、今後どのようにその産業が復興
するのか、向谷実x中西圭三プロジェクト（下記参照）のように、Twitter
やUSTなどを巧みに利用した、ミュージシャンの産地直送音楽（向谷氏の
命名）の事例や、業界関係者の意見なども踏まえながら探ってみる機会に
なればと思います。

TwitterとUstreamで曲が誕生、向谷 実氏が試みた「音楽産地直送」とは

＊マーケティング庵は、身の回りに起きる様々な事象を早期に捉えて、マ
ーケティング視点で理解を深める研究会です。
コンビニスイーツやオーガニック食品の試食などから激安酒場の研究に出
かけて、実体験してみるなど、参加者で体験しながら理解を深める研究会
でもあります。
目的は愉快に時間とテーマを共有することです。

主催は、マーケティングリサーチのプロフェッショナル 山野井 守
Twitter:@lefty_carbo

開催に関しては、某大学でマーケティングとデザインの関係を教えている
佐藤豊彦がサポートしております。
参考までに、佐藤が企画及びパーソナリティを担当するAppleCLIPです。
