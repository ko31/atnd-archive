description

オフラインリアルタイムどう書くの 本番・参考問題 で出した過去問の解説をする会です。

  • 過去問を解こうとしたけど解けなかった
  • 解けたもののなんか違う気がする
  • 他の人の解答を見たけど意味がわからない
  • どういうふうに考えたらああなるのか知りたい

という方は是非。
オフラインリアルタイムどう書くに参加したことのない方も歓迎です。
オフラインでリアルタイムにどう書くを行いたい方は http://atnd.org/events/40389 へどうぞ。このパーティーではどう書くはしません。
解説するのはいまのところ鍋谷ひとりの予定ですが、加勢してくださる方の参加も歓迎します。

主たるターゲットは 11回の参考問題「"等差数列":http://nabetani.sakura.ne.jp/hena/ord11arithseq/ 」を考えていますが、別の問題でもアドリブで対応します。

参加費は 100円前後になると思います。会場費 700円を、人数で割ります。

どの問題について聞きたいのか、何を聞きたいのかを事前にコメント欄に書いて下さると、対応がちょっとマシになると思います。

タイムスケジュールも何もなくなにもかもアドリブ対応ですので、ある程度グダグダになると思いますが、仕様です。

過去問のリストは以下のとおり：

  • ポーカー
  • Tick-Tack-Toe
  • 二値画像の回転
  • Bit Tetris
  • ボールカウント:野球
  • Y字路巡り
  • フカシギの通行止め
  • テトロミノ認識
  • Rails on Tiles
  • 大貧民
  • L-intersection
  • 続柄
  • XY-Sort
  • のんびり座りたい
  • エントロピー符号
  • ビットボンバーマン
  • 数を作る
  • バス代
  • ポーカーの残り＋
  • ハニカム歩き
  • 等差数列
  • ビットあみだくじ

終わったら吉田飯店で飲みましょう。
