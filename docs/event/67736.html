description

家づくりを通して、鹿児島の人と地域と文化をつなぎます

コンセプトは【AS YOU LIKE＋１】。
家族の想いや暮し方、趣味や楽しみを一番に。
そこへ家づくりのプロとして、「想い」をプラスワンさせてほしい。

「もっと自由な家づくりがあってもいいのではないか？」
そんな代表の冨ケ原さんの想いから生まれた粹家創房は創業12年を迎えました。

今回は地元に根ざした工務店、粹家創房の新たな挑戦のお手伝いをしてくださる方を募集します。

さぁ、暮らしをDesignするプロジェクトに参加してみませんか？

社員さんとの座談会とは？

インターンシップのこと、会社のこと、仕事のこと、はたまた人生のこと。
様々なテーマについて、受入企業の社員の方と参加者がざっくばらんに語り合う場です。

インターンシップフェアでは聞けなかった、あんなことやこんなことを
コーディネーターと一緒にズバッと質問してみましょう！

もちろん、あなた自身のことも社員の方に存分に語ってくださいね。

時代が変われば住まい方も変わっていく。
好きな街、住み慣れた環境、幼い頃からの思い出がたくさん詰まった家。

必要なものをひとつずつ選んでしつらえていくこと、

暮らしそのものを丁寧に創り上げていくことなのです。
まずは、「新しい暮らし」をイメージしてみることからスタートしましょう。
