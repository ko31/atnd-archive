来る3月11日（日）に「第1回スロー・キャンドル・イベント『3.11を忘れない』」を催します。場所は横浜・青葉台 AU LAPIN AGILE。地震の起った午後2時46分に黙祷からスタート。手作りキャンドルの店『Arbre Candle』の協力を得て、静かなキャンドル・イベントにしたいと考えています。参加費はキャンドル・ナイトと同様に1000円（飲食代は含みません）。小学生までのお子様は無料です。

いつものようにスロー・イベントは「空っぽの時間」を過ごします。また、『絆』をテーマとしたスローミュージックなどのパフォーマンスをしてくださる方々もいらっしゃいます。

もちろん、収益金の一部は被災した方々に直接届く方策で寄付させていただきます。

3月11日（日）
14:00 open / 14:30 slow candle event start / 14: 46 黙祷
→おそらくスローに夜まで続きます。。
