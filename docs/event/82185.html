参加登録時に，アンケート（交流会参加/不参加）にご協力ください。交流会のますの寿司を用意するために，早期に参加者数が把握できると助かります。参加検討中の方も，主催者に状況をお知らせいただけると助かります。

　オープンCAE（OpenFOAM，Salome-mecaなど）に関心を持つ技術者，研究者，学生などが集まり，気軽に情報交換・相談する場所を作ることを目的に，勉強会を開催します。
　すでに使用経験のある方はもちろん，興味はあるけれど使い方が分らないという方も大歓迎です。お気軽にご参加ください。

＜講習会について＞
タイトル： paraview講習 -pythonスクリプトを用いた可視化処理の自動化-（講師：中山勝之）

内容（予定）：
　Pythonスクリプトを利用した、可視化処理の自動化についての講習を行います。マウス・キー操作のレコーディング機能を用いた操作の自動化手法について、マクロ機能についての解説、演習を行います。

　講習にはparaview5.0.1を使用します。（OpenFOAM 4.0,4.xに対応するparaviewのバージョンです。windows版、Mac版は利用可能ですが、Linux版を推奨します）

講習会に関するご注意：
　今回の講習会では，ParaviewのPythonスクリプト機能を利用します。
　OpenFOAMをソースパックからインストール（ソースコードをダウンロードし，自らコンパイル）した場合，Pythonの機能を使えないことがあります。下記の書類を読み，ご利用環境に問題がないかをご確認ください。
ParaviewのPython連携機能の確認方法・対策ＰＤＦファイル

　万が一，Pythonスクリプト，トレース機能が使えない場合には，同書類に記載の対策をとるか，勉強会で作成したVirtualbox用仮想マシン環境をご利用ください。

　今回も，勉強会終了後に，有志による交流会を開催します。交流会では，富山の名産 ますの寿司 食べ比べを開催します。
