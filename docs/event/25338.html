【講座趣旨】
本セミナーでは、近年のアプローチである画像局所特徴量と統計的学習手法の組み合わせによる物体認識についてわかりやすく解説します。SIFT(Scale-Invariant Feature Transform)やHOG(Histograms of Oriented Gradients)を中心に画像局所特徴量について概説した後、このような特徴量と密接な関係にある統計的学習手法のAdaBoostについて物体検出を例に実践的に学びます。最後に、最新の研究動向として局所特徴量の関連性に着目した物体検出法と今後の展望について紹介します。

【プログラム】

1　スケール・回転変化に不変な局所特徴量(SIFT:Scale-Invariant Feature Transform)
　1.1　SIFTのアルゴリズム
　1.2　SIFTによる特定物体認識
　1.3　高精度化(PCA-SIFT, GLOH)
　1.4　高速化(SURF)
　1.5　学習による高精度・高速化(Randomized Trees)
　
2　局所特徴量と物体検出
　2.1　Haar-like特徴と顔検出
　2.2　HOG特徴と人検出(Histograms of Oriented Gradients)
　2.3　その他の画像局所特徴量
　
3　統計的学習手法による識別器(Boosting)
　3.1　AdaBoostのアルゴリズム
　3.2　Real AdaBoostのアルゴリズム
　3.3　AdaBoostとReal Adaboostの比較

4　局所特徴量の関連性に着目した物体認識
　4.1　Adaboostによる物体検出システムの構築
　4.2　局所特徴量の関連性に着目した物体検出(Joint特徴)
　4.3　今後の展望
