title 大学・専門学校のためのウェブマーケティングセミナー
catch 出願数増加のためのウェブ・スマートフォン戦略
description

ウェブサイト、スマートフォンサイトで情報を発信しているが効果的な活用ができているかわからない、オープンキャンパス・体験入 学の集客にウェブを活用したい・・・このような悩みをお持ちではありませんか？本セミナーでは、大学・専門学校向けに、資料請求、オープンキャンパス集客、出願までウェブ・SNSを活用し、どのように集客するか、活用法を実際の調査データを元に解説します。

【セミナー概要】
■進路検討に関するインターネットの接触・検索動向
■ウェブ、スマートフォンを使った集客方法（保護者層へのアプローチと高校生へのアプローチ）
■出願率を上げるためのウェブマーケティング施策（認知促進から出願へ）
■ SNSを活用したマーケティング戦略

【日時・場所】

参 加 費：無料
