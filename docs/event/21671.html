description

世界中600都市以上で開催されているエコやサステイナビリティをテーマにした憩いの場「グリーンドリンクス」。
日本でも、震災以後それぞれの「地域」への思いから、
多くの地域でさまざまな個性をもったグリーンドリンクスが始まりました。

つくばでも、都市「東京」と秋葉原から電車で45分で行ける最先端の田舎「つくば」をつなぐことをコンセプトに、
昨年イベントを開催してきた「Tokyo Tsukuba project(TxTprolect)」が中心となり、
「グリーンドリンクスつくば」を開催していくことになりました。

今回は、＃0と題し個々のオーガナイザーの帰属するコミュニティがツアーを企画しました。
それぞれのツアーでつくばに触れていただたうえで、夜みなさんで終結し、
東京近郊の田舎を再考するような、トークセッションとワークショップを行います。

サイクリングツアー【レンタサイクル×トレジャーハントツーリング】

つくば駅でレンタサイクルを借りて、渡されたマップに載っているチェックポイントを巡ってトレジャーハントツーリング（宝探し）をします。自転車の街・つくばの魅力を五感で感じながらポイントを巡って、街と遊ぶ楽しみを感じてみてください。
