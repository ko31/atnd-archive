description

香川県の素晴らしい魅力を１泊２日かけて案内する観光旅行です。
皆さんは香川県といえば『うどん』をまず想像されますよね。
しかし香川県には多くの特産品や観光地があり、他県では味わえない魅力に溢れています！

今回は主催者のぶ～けが皆様を香川県の隅から隅までご案内いたします♪
主な観光内容は下記の通りです。

（1）香川県の味、本場のうどんを心行くまで。
（2）ジャンボフェリーで行く、小豆島のオリーブ旅。
（3）ことでん(高松琴平電気鉄道)に乗って金比羅山。
（4）高校なのに馬に乗れる?!都会では味わえない乗馬体験。
（5）？？？（来てからのお楽しみ♪）

基本は参加される方が観光したい場所を指定し、それに沿ってプランを考えます。
ですから皆様が作る観光旅行といっても過言ではないのです。
さぁ、あなたも香川県で楽しい思い出を作りましょう！

※参加される方は参加登録の際、コメント欄に行きたい場所（上記以外）を一つ明記ください。
　可能な限り観光プラン内に添えるよう努力いたします！
