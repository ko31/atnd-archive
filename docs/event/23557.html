☆講習の対象者
　.NET上で、OpenGLを扱いたいけど、始めるまでのとっかかりが大変である方を想定した講習です。
　1．C／C＋＋のプログラムの知識がある方。
　2．グラフィックの経験ないかたでも、興味ある方。
　3．ノートパソコンを持ち込める方。(ただし、オプションで有料レンタルもあります)
　言語はC＋＋.NETを用います。

☆講習の目標到達
　1．お持ちのPCを持ち込んでいただき、そのPCのOpenGLの開発環境を整えることと
　2．それぞれのライブラリーのプログラミングの経験を身につけること

☆講習の内容　
　説明＋実習で、実習重視(3:7)です。

　作成するソフトのサンプルの動画見られます。
