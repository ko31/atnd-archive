どこでもいっしょ暦2017 の無料配布イベントのご案内

イベントについて

こちらは株式会社ビサイドが無料配布いたします「どこでもいっしょ 暦2017」のエントリーページです。

エントリー手順

  1. こちらのページで「イベントに参加登録する」を押してエントリーしてください
  2. アンケートで、受け取り方法を入力します。
  3. 無事にエントリー完了すると登録完了メールが届きます
  4. 登録完了メールの 受付番号 は手続きで使用しますのでメールは大切に保管してください

受け取り方法について（2種類）

１）ビサイド本社 受付で受け取り

  • イベント期間中、本社一階受付が開いている 平日朝10時 〜 16時くらいまでの間 に来社ください。
  • 会社受付に 受付番号 、 ATND でのユーザ名 の記入用紙がありますので、それぞれを記入します。
  • ATND でのユーザ名 が記入された封筒（カレンダーが入っています）がありますので、探してお持ち帰りください。
  • 受付の電話でスタッフ等を呼び出していただく必要はございません。

２）専用申込封筒での受け取り

  1. 下記URLよりPDFを入手し専用申込用紙を印刷します。
    第一回の封筒とは異なりますので、必ずこちらをお使いください。
  2. イベント期間中に返信用紙に必要事項、および ATNDより届いた登録完了メールの 受付番号 と ATND でのユーザ名 を裏に記載し、切手を貼って投函します。
  3. エントリー時のユーザ名が見つからない場合は受付できませんのでご注意ください。その場合、申し込み封筒を返送させていだきます。
  4. 準備に時間がかかることがありますので、申し込みを行われてから2週間ほどはお待ちいただけますと幸いです。

▶︎ 専用申込封筒のダウンロード

注意事項

  • できるだけたくさんの方にお配りしたいので、今回より １名さま１部 でのお申し込みに限定させていただいております。
  • 来社時の交通費や、申込封筒の印刷費用、返信切手代に関してはご負担いただけますようお願いいたします。
  • 配布期間は約一ヶ月間を予定しておりますが、なくなり次第終了させていただきます。
  • キャンセルが発生するとサイト上での番号とメールで届く受取番号がずれてしまいますが、メールの方の受付番号で構いません。
