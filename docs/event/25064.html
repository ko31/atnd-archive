この会は IT 勉強会のように、みんなで料理を教え合う勉強会です。対象者としては、普段料理をする機会が無いけど、上手くなりたいという初心者が中心となる予定です。また素人に教えてあげようという普段から自炊をしている人も大歓迎です。

料理を通して交流しませんか？

当日の予定

  1. 初台周辺で買い物。もしかすると時間が余るかも
  2. 角筈地域センターに到着、調理器具などを出してもらう
  3. デザートとピザを作り始める
  4. ピザ発酵中にキッシュを焼く
  5. 作りながら片付け
  6. 作ったものを食べる
  7. 21時30分には完全撤収
