description

受講対象・レベル
コアシェル型微粒子にとどまらず、マイクロカプセルについて興味のある方、これから研究を試みようとする方、実際に研究に携わっておられる方などを対象といたします。
趣旨
コアシェル型微粒子の基礎的事項として、調製法、シェル材種とコア材種の組合せによる構造、機能などについて平易に説明する。そして、最近の文献における実際の応用分野とコアシェル型微粒子の調製例について紹介する。
