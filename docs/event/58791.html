映画「タイタニック」の最も重要のシーン（34秒）はここだ！

OSAKA映画教室「北加賀屋シネマスタート」は次回「週間マガリ」にて開催します。
https://www.facebook.com/shukan.magari/info

週間マガリとは、昨年大阪市北区にそびえたった城で、日替わりで個性的な店長が運営している「雑誌みたいな」カフェアンドバー。

映画・ゲーム・出版業界に興味のある方であれば、どなたでも新発見のあるイベントです。

【北加賀屋シネマスタート ver.1.2 開催内容】

キホン的には、12:00〜16:00の間は、映画バー「銀幕酒場」として営業いたします（約140分）。そのなか、以下のコアタイムは講演形式にてお送りします。

※あくまでもバーでの開催ですので、
途中参加・途中退出自由。

■意外と知らない映画ライターの日常（12:30〜13:10）

例えば、映画ライターは年間に何本見るのか、LiLiCoさんは年間何本見ているのか、いつ見るのか、雑誌の原稿ギャラはいくらなのか…といった出版業界ネタなどのお話をします。

■映画「タイタニック」の最も重要のシーン（34秒）はここだ！（14:00〜15:00）

シナリオ的分析から、映画「タイタニック」の最も重要なシーンを解説いたします。

これが分かっただけで、明日から映画の見方がちょっと変わります。
下松シネマトーク、神戸元町シネマスタートも含めて、今回で6回目の講演。
かなりバージョンアップしています。

■懇親会

参加希望者で、2次会と称して飲みます。
※懇親会からの参加希望の方は、15:30〜16:00に「週間マガリ」においでください。
