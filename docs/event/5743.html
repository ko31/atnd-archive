title Terma Time de キャンドルナイト2010
catch 暖かいキャンドルの光の下,あなたの心の灯明を見つめてみませんか？
description

太古から人々は火を囲み語らい合ったといいます。現代においては人工の光により火の光にふれ合う機会が無いのではないでしょうか？
古代から変らぬ知恵を伝え続けているチベットのお話をそんなキャンドルの明かりの下、語らい合ってみませんか？

Terma Timeは毎週金曜日の夜。まるでバーのような雰囲気の美しいギャラリーで、深い心の話をナビゲートいたします。より善い自分に気づくために、ちょっと立ちどまってみませんか？

講師は、チベット仏教でも最も古い歴史と深みを持つニンマ派の寺院にて、足掛け10年以上、多くの高僧から学び、修業を実践してきたキャリアを持つ牧野宗永氏。日々の生活の中で、自分がどうあるべきか、の話を分かりやすくセッションとして楽しんでいただけます。

講師：牧野宗永（まきのそうえい）
時間：約1時間（定員最大4名のグループセッションのため、お一人あたり約15分
場所：アートギャラリー・ノルブリンカ（地図）
参加費用：2,000円
