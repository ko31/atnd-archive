Facebook Nightとは？

Facebook Nightとは、Facebookアプリの開発サポートを行うfb.developers’+が主催する勉強会です。
毎回ゲストをお呼びして、Facebookアプリの開発事例やマーケティング情報について講演していただきます。

第17回目となる今回は、Facebook Japanの佐々木氏/NewsPicksを運営している株式会社ユーザベースの杉浦氏などにご登壇頂き、今年4月にバージョンアップされたGraph API2.0について、またサービスへのFacebook活用法についてお話頂きます。

『Graph API2.0における変更点とその対応について』
2014年4月30日にGraph APIが2.0にバージョンアップされました。Graph API 2.0では、取得する権限によっては申請/承認が必要になるなど様々な変更が行われています。今回その変更点と対応について、各ディベロッパー様に影響が大きいと考えられる部分の内容を重点的にご説明頂きます。

『アプリサービスの成長に欠かせないFacebookプロダクトの紹介』
Facebookには、みなさまのアプリやWebサービスの成長にご活用が可能なプロダクトが複数提供されています。その中には最近提供が開始されたものなど、まだ日本ではあまり一般的でないものもあります。今回そうした各プロダクトについて改めてご紹介頂きます。

『NewsPicksにおけるFacebookを使用した集客とグロースハック手法の公開』
経済情報に特化したソーシャルニュースメディア｢NewsPicks｣では、数多くのユーザー様をFacebookを通して獲得しています。その集客手法やユーザー様の獲得における導線の工夫などをご紹介して頂きます。
またFacebookのインサイトを使用した改善活動など、NewsPicks独自のKPIの追い方やそのグロースハック手法についても合わせて公開頂きます。		 株式会社ユーザベース
