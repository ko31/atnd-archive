description

『「はたらく」を考える広場』は

「働く事で、もっと人生を輝かせたい！」 という想いを持つビジネスパーソンに向けて、
「ビジネスの最前線」で活躍しながら、
自分のはたらくスタイルを貫き「社会へ貢献」してきた
４名のリーダー達からのメッセージを発信します。

この場所で受け取ったメッセージを「絵の具」にして、 今までの「はたらく」スタイルに、明日のための「新しい色」を加えてみませんか？

これからの社会を支え、リードしていく「あなた」にとって、 人生の中で最も多くの時間を費やす「はたらく」ことに「新たなデザイン」を加える、 そんな「未来のためのアトリエ」のような場所を一緒に体感しましょう。

こんな想いを持つ方にお勧めです！！
・働きながら社会貢献する方法を知りたい。
・もっとワクワクしながら働きたい。
・毎日の仕事に、もっと価値観を見出したい。
・会社でもプライベートでも、もっと新たなことにチャレンジしてみたい。
