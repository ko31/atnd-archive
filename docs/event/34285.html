詳細

JavaOne 2012 サンフランシスコ最新アップデート(寺田 佳央)

2012 年 4 月 、日本で７年ぶりとなる JavaOne が開催され Java が今後も非常に重要であることが広く認識されました。本セッションでは米国のJavaOne 2012 で発表された Java の最新トレンドについて分かりやすく紹介します。
本セッションを受講いただくことで今後の Java の方向性についての理解を深めていただくことが可能です。

JavaOne に行って授業で教える内容を変えました (小出 洋)

JavaOneで収集した情報は九工大での教育内容にも反映させています．
例えば Swing はもはや要らない子なので，「オブジェクト指向
プログラミング」でも JavaFX2 を教えることにしました．
この授業は，きしだなおき著の「創るJava」を教科書にしています．
このなかの Swing を JavaFX2 にどう置き換えていくか，置き換え
たらどうなるかについて議論します．

Java EE 7世代に生き残るシステムを作ろう(岩崎 浩文)

JavaOne 2012にてEE 7の詳細が共有されましたが、
片や現状はまだEE 6しかない状況の我々がいま、
何をできるのか。7で変更予定の仕様を考えながら
今できる対策を検討します。
