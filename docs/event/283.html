catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：200円(実費)

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　●セッション１
　『CSS Niteビギナーズ(大阪)に行ってのまとめ＆感想』akiさん

　CSS Nite ビギナーズ（大阪版）
　http://cssnite.jp/beginners/osaka/


　＜内容＞
　こんにちは。
　aki☆です。

　web標準という言葉の前に、
　xhtml＋cssマークアップが基本となっています。

　とはいえ、
　xhtml＋cssマークアップであれば、いい、
　というものでも、ないとおもいます。

　日頃、目の前の仕事のなかで、
　いろいろな要素とのかねあいをはかりながら、
　「最低限」と「最大公約数」を補うよう
　意識しながらマークアップをしているつもりですが、

　目の前の案件と、近しい周囲とのカンファレンスだけでは見えてこない、
　「本当に、これでいいのかな」

　の部分を確認する意味もこめて、
　先日行われた

　CSS Nite cssビギナーズ(大阪版)へ、
　参加してきました。

　名前のとおり、初級内容ではありますが、
　再確認するという意味では、
　有意義だったと思います。

　今回はこの内容を
　かなえさんのピンチヒッターとして、
　感想も含めてお話することになりました。

　8つのセッション、5時間半の内容なので、
　駆け足になるかとは思いますが
　よろしくお願いします。

　＜akiさんの紹介＞
　CSS Nite in HIROSHIMA の実行委員。
　システム会社内で自社CMSを導入したサイト構築を主に
　Web制作全般担当。
　どちらかというとマークアップエンジニア。
　いろいろな意味でwebデザインという分野を模索、勉強中。
　今年も目標「しっとりした大人の女性」
　（※）一生クリアできる気が全くしないけど夢は大きく。

　●セッション2
　『デザイン講座 クロスメディアでの事業展開』川原さん

　＜内容＞
　商業デザインはグラフィックデザインに始まり
　今、若いデザイナー達はWEBデザインに移行しつつあります。
　不景気なこの時勢ですが、
　クロスメディアのコーディネーターや
　マーケッティング、コンサルタントは
　引っ張りだこです。

　WEB、新聞広告、電波媒体（ラジオ、テレビ）、DM
　うまくマーケットに合わせて効果的に使う
　コーディネータ、及びディレクターが必要とされています。
　デザインとクロスメディアについて学んではいかがでしょうか！

　＜川原さんの紹介＞
　現在、
　日本グラフィックデザイナー協会（ 通称 JAGDA ）
　広島地区代表幹事
　現在、株式会社中本本店にて
　グラフィックデザイン
　WEBコーディネーター
　クロスメディアでのソリューション事業展開中


　●セッション３
　『僕がChumbyでやりたいこと』松崎さん

　＜松崎さんの紹介＞
　某家電メーカでソフト開発をしている。
　ガジェット、ネット家電大好きな27歳男性。

chumby(チャンビー) 日本公式サイト

　＜内容＞
　1. Chumbyの紹介
2. widgetの紹介
3. widgetの開発
4. 僕がChumbyでやりたいこと
5. まとめ
