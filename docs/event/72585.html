当社は、SSP「アドフリくん」というアプリの広告収益最大化をサポートするサービスを提供しております。
私達の一番の特徴は、元々メディアやコンテンツ事業を運営していたメンバーが

「メディア１００％の提案」をモットーにサービス提供している当社は、
既に多くのアプリ開発者や企業にご賛同頂いております。
