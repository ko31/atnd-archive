Cygames 採用セミナー　
　　　　〜UI＆2Dアニメーション編 〜

概要

以前デザイナー向け勉強会として満員御礼で開催しました「アニデザNight」・「UI Night」。
この度Cygamesへの就業にご興味がある方を対象に、＜UIデザイナー＞＜アニメーションデザイナー＞2職種合同のセミナーとして再開催が決定いたしました。
当日はUIデザインチームとアニメーションデザインチームのマネージャーが登壇。CygamesのUIデザイナー・アニメーションデザイナーのこだわりを公開いたします。
またセミナー後は実際の開発現場をご案内する「オフィスツアー」・現場スタッフとの「個別面談」も実施致します。

タイムスケジュール

・19:15　　受付開始
　　　　　　会場15階受付ソファにてお待ちください。

・19:30　　開始

　　　 1.　会社説明
　　　 2.　CygamesのUIとアニメーションについて
　　　 3.　オフィス見学
　　　 4.　個別面談
　　　　　　

・22:00　　終了予定
