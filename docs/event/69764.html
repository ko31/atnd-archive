ヒカ☆ラボ一覧は こちら！！

★講演内容

近年、SIerからWebエンジニアにキャリアチェンジを希望する方が増えています。
「キャリアチェンジが出来た人とそうでない人はなにが違うのか？」
「SIerからWeb業界に行くためにどんなことしたの？」
「SIerとWeb業界って違うっていうけどどう違うの？」
「未経験から来る不安ってどうやって取り除いたんだろう？」
「実際にキャリアチェンジしてどうなの？」
「どういう求人媒体を使うべきなの？」
といったお悩みを抱えている方向けに、
講演・パネルディスカッション・懇親会を通して、
SIer出身の2人の経験を本音でお伝えさせて頂きます！

※内容は変更になる可能性がございます。

★講演者プロフィール

【具志堅 雅 氏】

日立系SIerに新卒で入社しC#で法人向けネットバンキングシステムの開発から
エンジニアキャリアをスタート。
その後はてなブックマークをはじめとするwebサービスに魅了されて一念発起し、
Webエンジニアにキャリアチェンジ。
現在は株式会社 じげんの社内最大規模事業部に在籍しながら
グロースハック中心の施策/新規機能の実装を
RubyOnRailsやSwiftで実施。
業務時間外では自身で立ち上げた「morei」にて講師や
実体験をベースとした講演や勉強会などを述べ10回以上実施。
その他にもアフィリエイトソン最優秀賞受賞などハッカソン系イベントで多数の受賞歴あり。

Qiita

【粟島 正俊 氏】

ERPパッケージソフトを展開するワークスアプリケーションズに新卒で入社し、
人事システムの開発に携わる。
その後ビジネスマン向けサービスを展開するジェネストリームにてCTOを務める。

ある日、交通事故をきっかけに「人はいつ死ぬかわからない」と危機感を覚え起業。
現在は株式会社テックファクトリーの代表兼エンジニアとして活動。
音楽コミュニティアプリや複数のWebメディアを運営している。
エンジニアとしては、Webのバックエンドからフロントエンド、iOSアプリ開発、
インフラまで幅広くこなす。
