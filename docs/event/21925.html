去年は Vim がなかったみたいなので（Emacsはあったのに！）立ててみました。
Vim への愛を語ってコメント欄にURLを貼る簡単なお仕事です。

Advent Calendar に関しては、 去年のまとめ記事 などを参照して下さい。

2周目について

Vimmer の Advent Calendar は今年の11月30日までらしいのでこのまま続けていきたいと思います。
これ以降は特に書く順番を決める予定はないので、記事を書いたらその時の順番でコメントしてもらうだけで構いません。
もちろん、既に書いた方が2周目をしてもいいですし、まだ書かれていない方が参加されても問題ありません。
気が向いたときにでも気軽に参加してもらえればと思います。

追記

25日まで埋まりましたが、26日以降に参加されても全然問題ありません。
引き続き参加者を募集しています。

ルール

イベントに参加した順に 1日1回 Vim の 愛を語って、 記事を書いてもらい、コメント欄にURLを貼っていきます。
1番目の人が1日目、2番目の人が2日目…って感じです。

内容は Vim本体 でも Vimスクリプト でも Vimプラグイン でも Vim に関係する事であれば何でも構いません。
それぞれの情報を共有して、自分の Vim をパワーアップさせましょう！
