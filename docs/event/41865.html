イベント概要

8月21日午後開催予定だった「マイナビ クリエイターズ Power Session ～スマートフォンアプリ開発「最前線」攻略」ですが、テーマごとに切り分けて、より参加しやすい形にして、2回に分けて開催することになりました。
8月21日19:00～21:00は「Vol.1 スマホアプリUI攻略」と題して、フラットデザインをはじめとしたスマートフォンアプリのデザインのトレンドや、プロトタイプの作成について、ソシオメディアの上野学氏とツクロアの秋葉ちひろ氏にお話しいただきます。
9月13日19:00～21:00は「Vol.2 cocos2d-x/Unity」と題して、現在スマートフォンアプリの統合開発環境の定番であるUnityとcocos2d-xを取り上げ、日本cocos2d-xユーザー会代表の清水 友晶氏とKLabの松田 龍弥氏にご登壇いただきます。

それぞれ個別にご参加いただけるほか、両方にご参加を希望される方向けのお得な「通し券」もご用意いたしました。

アプリ開発実務に携わる人、これからスマートフォンアプリの開発に関わりたい人、どちらの方にも役立つ内容となっておりますので、ぜひ参加をご検討ください。
