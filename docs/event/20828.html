講師

鷹野 雅弘（たかの・まさひろ）

Webサイトの構築やコンサルティングを中心に、WebやDTPに関しての講演やトレーニングのほか、スクールなどのカリキュラム開発も手がける。テクニカルライターとして10冊以上の著書を持つほか、書籍の企画や編集なども行っている。2005年から CSS Nite を主宰。
主な著書に『 効率的なサイト制作のためのDreamweaverの教科書 （CS5.5対応版）』（共著、毎日コミュニケーションズ、『 できるクリエーターDreamweaver独習ナビ 』（インプレス）、『 Illustrator CS4 完全制覇パーフェクト 』（共著、翔泳社）、企画編集を行った書籍に『 現場のプロから学ぶXHTML+CSS 』（毎日コミュニケーションズ）など。

  • 株式会社スイッチ
  • twitter： @swwwitch
2011年9月23日に、『効率的なサイト制作のためのDreamweaverの教科書（CS5.5対応版）』という書籍を出版されました。これを記念して9月に東京で開催された『 CSS Nite in Ginza, Vol.59 』の内容を、名古屋のみなさんに、さらにグレードアップしてお届けします。

佐藤 歩（さとう・あゆむ）

有限会社アップルップルでWebサイト制作やシステム開発の他、「a-blog cms」の開発に携わるWebエンジニア。PHPが母国語で、最近はJavaScriptや(少しだけ)Rubyも可愛がるプログラマをやっています。
フロントの実装が好きですが、実際の業務はサーバーサイド・クライアントサイド問わず広く浅くやっていて、Web技術全般の話に関心を向けています。

  • 有限会社アップルップル 　Webエンジニア
  • twitter： @ahomu
  • ブログ： http://havelog.ayumusato.com
  • GitHub： https://github.com/ahomu
9月にWCAN 2011 Autumnで大好評を博した『はじめてのjQueryMobile』の内容をさらに、CSS Nite in NAGOYA ♭001向けにボリュームアップしたセッションをお届けします。
