アナログ・モジュラーシンセ“つながる”ワークショップ

2015年、楽器界で一番ホットなのが「アナログ・モジュラーシンセ」。

当日は、モジュラーシンセの基本的な使い方が学べると同時に、色々なモノとつなげたり、
触れたりして遊びながらそのサウンドの魅力を体感できます。

手荷物で持参できる程度のものであれば楽器（種類問わず）の持ち込み大歓迎！
一緒に“つなげて”新しいサウンドを探求しましょう！

みんなでワイワイガヤガヤと楽しめる、アットホームなイベントです。

シンセ好きなら初心者でもマニアでも大歓迎！
みんなでアナログ・モジュラーシンセの面白さを堪能しましょう！

当日はシンセ監督・大須賀淳氏ともお話できます。
『ナニワのシンセ界』の裏話なんかも聞けるかも？

●ナビゲーター
大須賀 淳（おおすが じゅん）
1975年生、福島県出身。シンセ監督。株式会社スタジオねこやなぎ代表取締役。企業ビデオ等様々な映像・音楽コンテンツを制作すると同時に、書籍や雑誌での執筆、世界トップクラスのオンライン学習サイト「lynda.com」等での講師、製品デモなども数多く務める。2014年、日本初のシンセサイザードキュメント映画「ナニワのシンセ界」を監督。近著は「作りながらおぼえるDTM入門」（秀和システム）ほか。 2015年4月上旬　新著「21世紀のアナログシンセサイザー入門」刊行予定
