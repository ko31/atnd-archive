WordPressもくもく勉強会@群馬について

WordPressを使っている人や興味を持っている人が集まって、それぞれ自分のペースで好きな作業をする、いわゆる「もくもく会」です。
勉強会といっても特に先生も生徒もいるわけではありませんので、何か困ったことがあれば周りの人に聞いたりしてみてください。
イベントに参加するのはちょっと気後れする、WordPressは好きだけど詳しいわけじゃない、そんな方でも気軽に参加できるゆるっとした場にできればと思います。

やること

WordPressにかかわることなら、各自何をやってもOKです。

  • 新しいWebサイトやブログを作る
  • 運営中のWebサイトやブログのカスタマイズ
  • プラグインを作る
  • ブログを書く
  • WordPress本の読書

などなど、自分でやりたいことを考えてきてください。

必要な物

ノートPC、書籍など各自作業に必要な物をご持参ください。

電源、無線LANは開催場所のものが利用できます。

コワーキングスペース内にあるWordPress本も利用できます。
