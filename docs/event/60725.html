協力 : 株式会社金沢大地

マルツ金沢西インター店で開催しているものづくりイベント「Maker Starter」
今回第8回目は、「ものづくりの力で、農業を始めやすく、続けやすいものにしよう。」をテーマとし、
農家さんと対話しながらものづくりを行う「農家ソン」を開催します。

1日目は、農業の現場に行き、農家さんと対話し、
チームごとにものづくりを行うためのアイデア出しを行います。
優秀なアイデアのチームには景品も出ます。

2日目(ハッカソン)は こちら

参加登録については、本ページか、
kanazawa@marutsu.co.jpに連絡をお願いいたします。

皆様の参加をお待ちしております！
