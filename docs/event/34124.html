イベントについて

今年10月末にアメリカ ジョージア州で初めて開催された WordPress コミュニティサミット に参加した倉石政典、三好隆之両名からの正式報告会を、記念すべき、第1回 WordBench 札幌 勉強会内で行います。WordPress の歴史とコミュニティの概要、そしてサミットで話し合われたこれからの WordPress について触れる予定ですので、現在 WordPress とそのコミュニティに関わっている方、また、これから関わってみたいと考えている方は是非ご参加ください。3部目には思わぬ発言が飛び出しそうなプラグイン制作者3名による座談会がありますのでお楽しみに。

内容

  1. WordPress の歴史、ライセンス、コミュニティについて（田中広将）
  2. WordPress コミュニティサミット2012 報告会 (倉石政典、三好隆之)
  3. プラグイン制作者座談会 (倉石政典、田中広将、三好隆之)

スピーカー紹介


倉石政典 (a.k.a. Kuraishi or tenpura)
タイニービット有限会社 代表。日本語版パッケージにデフォルトで含まれる WP Multibyte Patch プラグインの作者であり、マルチバイト関連などでコア開発にも貢献している。


田中広将 (a.k.a. hiromasa)
WordPress 関連の書籍を複数執筆、日本人向けプラグインの配布を行う JSeries サイトの整備など日本の WordPress 界の草分け的存在。


三好隆之 (a.k.a. takayukister)
合同会社ロックロブスター 代表。世界で5番目に多くダウンロードされているプラグイン Contact Form 7 の開発者であり、世界のユーザーに対してフォーラムなどできめ細かなサポートを行なっている。
