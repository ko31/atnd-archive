description

【第一部　午堂登紀雄氏が海外不動産の投資戦略を丁寧に解説】！
午堂氏は、サラリーマンとして勤務する傍ら、
不動産投資を中心にCX、FX等を運用し、
現在では5億円以上の収益を形成された方です。

今回のセミナーでは、今後狙い目の不動産情報から、
CX、FXの注目銘柄、エントリーのタイミングまで、
午堂氏自身の投資戦略を過去の事例や実体験も絡めながら、
初心者の方にも分かりやすく丁寧に解説していただきます！
これを機に一から投資手法を身につけたい方にも必見です！

【第二部　ミスターストップ高！天海源一郎氏登場！】
株式投資の世界で"ミスターストップ高"
としておなじみの天海源一郎（てんかいげんいちろう）氏。
広島での講演が決定いたしました！

今回のセミナーでは、株式の見通し、「有望銘柄選び」の話から、
CX、FXまで、天海氏が「評論家」ではなく、
実際に売買をしている『個人投資家』の目線で、
「天海節」を炸裂させます！

・足元の相場を個人投資家的視点でみる
・この出来事に注目せよ！
・私はこのような投資行動を取っている
・今の「銘柄」の選び方
・エントリーのタイミングを判断するには？
・・・など。

数々の「銘柄バトル」にも勝ち抜いてきた天海氏のセミナーは、
きっとあなたにとって頼りになるはず！
当日は、瞬く間に時間が過ぎていくことでしょう。

【第三部　天海氏と午堂氏を交えてのトークセッション】
天海氏・午堂氏による講演後、セミナー開催毎に、
その時々の旬な情報と金融業界を取り巻く環境に合わせ、
毎回テーマを変えて展開する、弊社コンシェルジュスタッフによるミニセミナー。

今回は天海氏と午堂氏を交えて、実際の取引例を用いた
トークセッション形式で色々な状況でどう動くべきなのか等、
分かりやすくご紹介します。
是非ご参考にしてみてはいかかでしょうか。
