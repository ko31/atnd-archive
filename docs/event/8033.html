Asprova ハンズオンセミナ SCP編は、複数の顧客、工場、物流センター、サプライヤをチェーンしてスケジュールする「Asprova SCM」について、Asprova SCMを用いたSCP(サプライチェーンプランニング)のトレーニングです。
資源、品目の拠点(サイト)や購買BOM、計画在庫といったSCP独自の機能や需給表について説明いたします。
Asprova SCMを販売いただくには、Asprova ハンズオンセミナ SCP編にご参加いただき、SCM-APT認定試験に合格することが必要です。 また、ユーザさまへのライセンス形態が月額使用料(年契約)になりますので、販売していただくためには「Asprova SCM販売特約店契約」の締結が必要です。
