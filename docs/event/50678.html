クラウドサービスを提供する「アマゾン　ウェブ　サービス　（英：Amazon Web Service、以下AWS）」のハンズオンセミナーを大阪で開催いたします。

■アジェンダ
17:30～18:00
アカウント作成ハンズオン　※ご希望の方のみ
・AWSアカウントの作成 (30分)
　 http://aws.amazon.com/jp/register-flow/
　※アカウント作成済みの方は、特にご参加いただく必要はございません
　
18:00～20:30
・Amazon EC2概要説明：機能、料金体系等　(15分)
・ハンズオン：EC2インスタンス作成(Linux)、OS設定、タイプ変更等、Amazon Machine Imagesの使い方 (90分)
・Q&A (15分)

■お持ちいただくもの
・Wi-Fi接続可能なノートPC
・参加者個人のAWSアカウント
・クレジットカード(アカウント作成ハンズオンのご参加者様のみ）
・携帯電話（アカウント作成ハンズオンのご参加者様のみ）

■事前にご確認いただきたい資料
・諸注意にもございますとおり、自主トレーニング形式となりますので事前準備資料を必ずご一読下さい

http://www.slideshare.net/AmazonWebServicesJapan/0-aws-basicprepare20120629

・基本的に講師の説明を聞き、ハンズオン資料を見ながらの自主トレーニング形式となりますので、お持ちいただくPCなどお忘れなきようお願い致します

・アカウント作成ハンズオン参加希望の方は、必ずwifiに接続可能なノートPC以外に、クレジットカード、携帯電話をお持ち下さい
