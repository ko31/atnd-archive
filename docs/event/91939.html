description

マーケティングの手法の一つとして、注目を浴びているインフルエンサーマーケティング。様々な場面で広告主が利用している状況になってます。その中で、虚偽の推奨や、関係性明示がない推奨（いわゆるステルスマーケティング）など、数々の問題点も浮き彫りになってきました。
WOMマーケティング協議会としては、現状の問題点を解決するためのガイドライン改定を行います。 2017年のクチコミフェスタでは、インフルエンサーマーケティングを全体のテーマにし、新ガイドラインでのクチコミマーケティングの手法、事例発表、トークセッション、ワークショップを行う予定です。

日程：2017年12月4日（月）13:00〜18:00（予定）
