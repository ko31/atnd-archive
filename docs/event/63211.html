概要

2015年2月14〜15日のバレンタインデーの真っ最中、KDDIは「Fx0 meets mbed」をテーマにハッカソンを実施させて頂きました。そこでは、webを介し『デジタルとフィジカル』をMatchさせた多くの作品を製作頂きました。
二日間という短期間での開催だった為、「もっと作品の質を高めたい」、「出会った仲間と続きを行いたい」などのご意見を沢山頂きました。
そこで、今回『au Firefox OS WoTハッカソン on ホワイトデー』を開催致します。

2月に参加頂いた方は「これまでのアイディアをブラッシュアップする」もしくは「新しいアイディアに取り組む」のどちらかを選択頂けます。加えて新しく参加される方も、新しいアイディアのもとに製作を行って頂けます。

ハッカソンのコンセプトは前回を踏襲し、英国 ARM 社が推進するデバイス開発プラットフォーム 『mbed』と『Fx0』を組み合わせ、Web につながるモノづくりと、モノとヒトをつなげる Web アプリケーションづくりを行います。

2月開催のWoTハッカソンに関しては下記のページを参照下さい。
