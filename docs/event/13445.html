はじめに

転職を考えているけれども、Javaの経験がない・・・。
来春から働くけれども、プログラミング経験がない・・・。

そんな風に考えている人、いませんか？

今回、開発現場に必須となる技術力を伝える場を設けることで、
これからの仕事に少しでも役に立つことはないかと考え、
Java即戦力の基礎を身に着けられるセミナーを企画しました。

第一線で活躍している現場のシステムエンジニアが直接教えます。

これからの仕事にJavaを扱う初心者の方に、お勧めのセミナーです。

ちなみにJava未経験のために、JDKのダウンロードとインストールの
ムービーを以前当社のエンジニアが作成しました。
こちら「初心者のためのJava講座」をご覧ください。
