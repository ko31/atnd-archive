真剣徳大しゃべり場 ～第五弾～ こっからはじまる徳キャンライフ、掘って、しゃべって、繋がろう！～

2012.6.22(金) @徳島大学 4号館402教室

徳大に入学したけれど・・・

徳大に入学して新入生は早2ヶ月、在校生は早数年目・・・。
「今の大学生活って自分が想い描いていた通り！？」

理想の大学生活が送れている人もいるでしょう。
高校時代想い描いていた大学生、大学生活とのギャップを感じている人もいるでしょう。
しかし、理想の生活が送れていてもいなくても、自分の大学生活でもっともっと面白いことをして、
大学生活を充実させたい！と考えている人はたくさんいると思います。

私たち繋ぎcreateは、
そんなあなたの 「大学生活でこんなことしたい！」 「大学生活をもっと充実させたい！」 を考えるための出会い・交流の場を提供します！。

真剣徳大しゃべり場は、学生と教員が同じテーブルにつき、
お互いを あだ名 で呼び合いながら、
フリートーキングを楽しむ企画です。

今回のしゃべり場のテーマは、
「あなたの大学キャンパスライフってどう？」
というものです！

ぜひご参加ください！

真剣徳大しゃべり場ってなに？

真剣徳大しゃべり場は、教員を交えた学生同士のグループトークの場です。
グループは8人前後で構成され、
学生が務めるコーディネーター（進行役）が話を進めていきます。

テーマに沿って、自由に話を広げていきましょう。

【真剣徳大しゃべり場でできること】

1. あなたの「大学生活でやりたいこと！」が見つかる！
2. 徳島大学で頑張っている学生・教員と出会える！交流できる！
3. 人の話を聞く力・人に考えを伝える力が身につく！
4. 互いに刺激を与え合い、日々のモチベーションがアップ！

さあ、学生も教員も、
しゃべってしゃべって、しゃべりまくれ！

対象者

大学生・院生・教職員
