description

普段なかなか目にすることのない、お煎茶のお手前を楽しんで頂く
ワークショップです。

お煎茶のお点前では、玉露や煎茶葉を入れた持ち手の無い急須“宝瓶”に
湯を注ぎ、小さな茶器で頂きます。

玉露や煎茶は、茶葉の量、湯の温度、抽出時間で味が変わります。
味わい深く、まろやかな旨味を感じるお茶を是非味わって頂きながら、
自宅で美味しいお茶を入れるコツをマスターして下さい。
