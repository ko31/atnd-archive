catch 「らしさ」を大事にするアーティストが集う インディペンデントバザール
description

服・バック・アクセサリー・雑貨・絵画・写真など、
アーティストが一つ一つ丁寧に制作したものを展示・販売いたします。

アーティストによっては、その場でのオーダーメイドも受け付けています。

作家同士のコラボレーション
毎日を楽しく彩る個性豊かな作品との出会い
作家とのリアルなコミュニケーション
自分だけのオリジナルをつくる楽しみ
いろいろな灯りが集まります。

１人でも多くの方の心に光が灯せるイベントになるようにと思いを馳せています。

今回の参加アーティストは約20組。
オーダーメイドの服や刺繍Tシャツ、ストール、帽子、手づくりシンプルバッグ、
シルクスクリーンで作ったボタンや布小物、ハンカチ、絵本、ポストカードなど、
春物を主としてたくさんのオリジナル作品が集まります。

今後は3ヶ月に1回開催し、日本全国やドイツをはじめとした海外でも開催予定です。
場所にこだわらず、あらゆる人に素晴らしい作品と出会えるチャンスがあれば良いと願い、
「ちょうちん　〜Cho-chin〜」を開幕いたします。
