いきさつ

4/23に「Windows Developer Days」の前夜祭イベントとして、東京・渋谷の「Bar Android」が、「Bar Windows Phone」として開催されました。
我々、それに参加できなかった一部の名古屋勢は、うらやましく感じていましたが、「それなら名古屋でもやってしまおう」と勝手に有志が企画したイベントが、この「Bar Windows Phone in Nagoya」です。

なお、渋谷の「Bar Android」は、毎週月曜限定のバーですが、それにちなんで、名古屋でも同様に毎週月曜限定の「名古屋Geek Bar」の会場をお借りします。

どんなイベントですか。

勉強会とかではありません。
みんなでワイワイ／ガヤガヤとWindows Phone（WP）について自由に語り合う集いです。

ただし、やや濃い内容になってしまうかもしれません。
たとえば、WP7のユーザーとそうでないユーザー（つまりiPhone/Androidユーザー）が、それぞれの視点でWindows Phoneを語り合う、というイベントになるかもしれません。

また、GeekBarというお店の営業の一環として行われるので、好きな時間にやって来て、好きな時間に帰っていただいてかまいません。（ただしチャージ ￥1,500 のお支払いが必要です）

それでは懇親会と同じでは？

かなり鋭い指摘です！イベント後半は、それに近いものになる（する）予定です。ｗ
しかし、前半の１～２時間では、映像（動画）やPowerPointなどを通して、WPに関するプレゼンテーションも行うなど、少し硬めの内容も用意してあります。
遠隔地の方にSkypeで参加していただく企画も予定しています。

UStream配信されます

名古屋Geek Barの模様は毎週、ustreamで配信されています。
なので、この Bar Windows Phone in Nagoya も配信されます。
遠方で参加できない方でも、このイベントに参加した気分に浸りたいかたは是非ご覧ください！

http://www.ustream.tv/channel/nagoyageekbar
