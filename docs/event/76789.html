こんにちは！
ど田舎である和歌山でオフ会を開こうとしている、まるです。
4/10に自分が和歌山に帰省した時に和歌山勢３名誘ってプチオフ会を開きました。その様子をTwitterに投稿すると、vainglory JapanのRTで多くの方に広まり新たな和歌山勢が見つかったので、オフ会を開こうってことになりました。

〇みんなでお菓子、ジュース飲みながらわいわいプラベ、大乱闘大会！
〇Wi-Fi環境についてですが、こちらの方でポケットWi-Fi借りようと思ってます。
〇参加費はお菓子、ジュース、レンタル代を人数分で割った金額になると思いますが、できるだけ安くなるようにします。

今回はどれくらい参加したい人がいてるか把握したいので、事前アンケート取らせていただきます。何かコメントも書いてもらえると嬉しいです！
