【講座のポイント】
　「思うように部下を教えられない」、職場のリーダーなら、誰しも悩むところです。「教え方」は、単に経験を積めば上手になるものではありません。そもそもコミュニケーションとは何なのか、相手に伝えるとは何なのか、上手くいかないとすればどこに問題があるのか、その問題はどうすれば解決できるのかについてお話します。
　講演では、お話を聴いていただくといった一方通行な内容にならないように簡単なロールプレイなども行います。また、講演内容をわかりやすく理解できるようにビデオ鑑賞もあります。
　部下ができたときに後輩に上手く伝えたい、自分が先輩として恥ずかしくないような存在でありたいと思うようになったあなたにとって参考になる内容です
【プログラム】
１．企業が重視するのはコミュニケーション能力
２．アイコンタクト
　　2-1 予備校の先生方のトレーニングは具体的で実用的
　　2-2 「視線三角形」　自己紹介ワーク
３．コミュニケーションは笑顔が基本
　　3-1 magic wordは、「ウィスキー」「笑顔の作り方」ワーク
４．話上手は、「相手の話を一生懸命に聴く」人のこと
　　4-1 「積極的傾聴」ワーク
５．上手に“間”とることであなたを演出する　　
６．質問力を身に付けよう
　　6-1　「質問力」ワーク
７．７つのプロセスを踏んで教える
　　7-1　「改善点の適切な指示をする（フィードバック）」の経験ワーク
８．ほめ力を身に付けよう
　　8-1 ピグマリオン効果 「ほめる」と「おだてる」は違います
　　8-2 「ほめ力」ワーク
９．アサーションコミュニケーションを身に付けよう
　　9-1 3つのスタイル 「I メッセージ」 ワーク
