クラウド時代を生きるエンジニアのみなさんへ。

「OpenStack入門講座」開催します！

========

クラウド時代に必要とされるエンジニアとして活躍したい方、最新トレンド技術を身につけたい方のために「OpenStack」入門講座を開催いたします。

当講座は初めてOpenStackに触れる方を対象に、OpenStackが提供するコアサービスの特徴を講義で学んでいただいた後、シンプルなネットワークシステムの構築を体験していただきます。
初めての方でも安心してご参加いただける内容となっています！

OpenStackとは、クラウドコンピューティングの基盤を構築するためのソフトウェアの一つ。
「クラウドOS」（cloud operating system）を標榜しており、IaaS（Infrastructure as a Service：サービスとしてのITインフラ）システムの展開に必要な一通りの機能を提供する。

講義内容

【講義】OpenStackの基本を学ぼう　　　　60分
OpenStackのネットワークサービス、コンピューティングサービス、ストレージサービス、認証サービス・操作方法などクラウドコンピューティングの基礎知識等

【ハンズオン】初めて触れるOpenStack　　60分
初めてOpenStackを触れる方を対象に、インターネット上に簡単なWebサーバのシステムを構築する。
