待望の実践編！

日経Linux 2011年4月号から7月号の短期連載『Linuxで始めるデジタル写真処理』を執筆されている
藤田 稜 氏を講師にお招きし、全2回にわたり第1回 撮影基礎(6/11実施)から第2回RAW撮影による写真編集(8月予定)の

今回は大好評だった第1回撮影基礎の実践講演について「もっと実践編を行いたい！」といった参加者のリクエストにお応えして、実践編を特別に開催しちゃいます～！

実践編内容


デジタル一眼 初心者（コンパクトデジカメも可）向けに、水族館を舞台に撮影テクニックを紹介します。
水族館撮影の基礎から、ちょっとした工夫であっと驚くほど綺麗に撮れるテクニックなど♪
