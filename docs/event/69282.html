◆開催主旨ーーーーーーーーーーーーーーーー

「話す」ことは「筆者」にとって切り離せない「技術」の1つです。

本の出版後は「講演」の依頼やワークショップの開催の依頼などが発生します。

そのための「セミ本番」としての合同講演会を開催いたします。

ぜひ「自分の執筆テーマ」で「自分流」の自己表現・講演・ワークショップをお楽しみください。

聴講者の皆様は知人や友人様あるいは応援している表現者の勇姿をぜひ応援ください！ 登壇者の方も、聴講者の方もじおなぎ道場の塾生さん以外の方もぜひお気軽に参加頂けましたら幸いです。

※一般聴講料は￥１０００となっております。

◆登壇メリットーーーーーーーーーーーーーーーーーーー

・講演について良かった点、直したほうが良い点について塾長潮凪洋介が個別にアドバイスいたします。

・聴講者の中に出版関係の方が紛れています。過去にこの講演会から作品が出版採用になりました！

・一般の方も聴講することで自己表現意欲が湧き上がり、自分の表現テーマ、発信テーマについて真剣に考える機会となります。

・講演風景の写真をプレゼントします（プロフィール写真などに使用ください）

・講演の練習体験となります。

・練習会に向けて各自研鑽の機会ができます。

・塾生同士の横のつながりができます。

・講演風景の写真を残すことで以後のブランディングに活用ができます。

・動画素材の撮影も可です。ウエブサイトなどで活用ください。
