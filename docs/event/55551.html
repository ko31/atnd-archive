description

今年もやるよー
キャンプオフ開催いたします。
場所は静かな山の中にあります『西山自然体験交流施設 ゆうぎ』で開催致します。
コテージもあるのでテントなど無くても快適に宿泊できちゃうし、ちゃんとお風呂もあるよ。とっても綺麗でステキなキャンプ場です。
例年通りお昼ころからBBQはじめて、夜までゆるゆるとやりますよ。泊まりたい方はそのまま宿泊しちゃいましょう。参加表明される際はコメントに宿泊を希望されるかどうかもお書き添え下さいませ。不明点は私まで連絡くださいませ。
道具とかテントとか無くても大丈夫ですのでお気軽にどうぞ。

今年こそ・・・晴れてよね！！！

9/2追記
なんとびっくり、ゆうぎが満室とのことで場所変更いたしました。
とりあえず下見行ってきますので、集合場所とか焼くもんとか細かいとこはそれから詰めましょう。
あ、昨年同様のBBSも準備しておきますね。
とりあえず4人定員のコテージを2棟予約しましたよー。

9/8追記
昨日かめこ氏と下見に行って来ました。
大自然の中に囲まれた素晴らしいキャンプ場ですよー。
当日、晴れますよう！

場所の都合により現在の参加者で締め切らせていただきました。
ご了承ください。
