description

日本でフェイスブックの利用人口が１０００万人を突破しました！
もうすでに利用して友達の数も増えている方も多いかと思います。
でも友達申請ってなかなか知らない人に出来ないって人も多いのではないでしょうか？

今回のコラボカフェalikeの交流会は

「facebookをしているけど、もっと友達がほしい！」とか
「facebookのアカウントは作ったけど、
それから特に投稿など出来ていない。」
「facebookしてるけどリアルな友達もほしい！！」と言う方々で
パソコンやスマートフォンを持ってその場でお友達作りをしよう♪

と言う会です。

facebookのアカウントを持っていない方は
その場でアカウント作成のお手伝いもいたします！
またfacebookでわからない事などは
その場で質問して頂けます～。
みんなで解決して楽しいfacebook生活をしましょう♪

※注意・・・ビジネス交流会ではありません。
　　　　　　独身限定の交流会です♪

当日の流れ

入り口のタッチパネルで「alike（アライク）」を呼び出して下さい。
担当者がお迎えに参ります。

１３：４５　　　　　　　受付開始
１４：００　　　　　　　facebookを使った交流
１５：００　　　　　　　休憩
１５：１０～１５：３０　自由に交流して頂けます。
１５：３０～１５：４５　アンケート記入後終了

ノートパソコンやスマートフォンをお持ち下さい。
（デスクトップしかない方は筆記用具などお持ち頂き、お帰りになってから確認頂きます。）

交流会開始しましたら
facebookを立ち上げて頂きます。
参加者全員の一人一人のfacebookの基本データーページを見ながら、
交流して頂きます。

その時に友達申請し合っても結構ですし、
最後に自由に交流していただける時間も設けていますので、
気の合う方同士で友達（FB上でもリアルでも）になって頂けます。

facebookは実名で登録しますので、安心してご参加頂けます♪

★期待できる効果★

異性の友達が出来る・彼氏彼女が出来る・結婚相手が出来る・気軽に婚活が出来る・お得に婚活イベントに参加出来る・facebookの疑問がわかる・facebookの友達が増える・相手の基本データーを確認しながら知り合える

★対象者★

リアルな友達がほしい方・異性の友人作り・婚活している方・結婚相手探し・もっとfacebookでの友達がほしい・お休みを楽しく過ごしたい方・気軽に婚活イベントに参加したい方
