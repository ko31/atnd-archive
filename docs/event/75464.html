概要

北海道の Drupal ユーザーコミュニティ「Drupal さっぽろ」が主催する meetup (勉強会) です。地域の Drupal ユーザーの情報交換と交流を目的に、月に一度、札幌と小樽で開催しています。今回は小樽での開催です。

開催予定は ATND でご案内、Google グループ、Facebook、Twitter を通じて告知します。
https://groups.google.com/forum/?hl=ja#!forum/drupal-sapporo
https://www.facebook.com/drupalsapporo
https://twitter.com/drupalsapporo

北海道の Drupal コミュニティとして情報交換と交流の輪を広げ、さらなる活用につなげていきましょう。

参加料：無料

実施レポートを公開しました。

内容予定

持ち寄りセッション

常時募集中です。発表テーマや他の参加者に投げかけてみたい話題をお持ちの方はお知らせください。随時このページに追加・更新していきます。

勉強会：テーマ「Core Block / Panels / Display Suite」

今回は、Drupal でコンテンツをレイアウトする仕組みを取り上げます。コアのブロックシステム、以前も取り上げた Panels の再入門に加え、Display Suite モジュールの機能を整理し、それぞれの特徴と使いどころについてディスカッションしましょう。

  1. テーマとブロックシステム再入門（白根）
    コアのブロックシステムとテンプレートの関係など
  2. Panels 再入門（今）
    レイアウトを中心に、全文表示のカスタマイズなど
  3. Display Suite 入門（今）
    Views への埋め込み時の適用など
