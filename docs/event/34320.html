当日の勉強会について

忘年会には出たいけど，忘年会だけで静岡に行くのも…… という方もいるかと思います．

WEB 制作系の方は 10th Knock! があります．

開発向けには10人程度入れる場所を確保 できないか調整してみます しました．
店長の店こと photo の 2 階になります．
参加登録は http://connpass.com/event/1549/ からお願いします．

会場について

会場の Bar. D さんですが，いろいろと融通をきかせると言ってくれています．

  • どんな料理がいいか
  • どんな酒が飲みたいか

などあれば，コメントに書いてください．
店員さんに直接見てもらって，うまくやってもらいます．

もちろん，料理と酒はトレードオフです．

二次会について

今年はある程度あたりをつけておきたいと思います．
