description

打合せで描くためのスケッチトレーニングです。 室内空間の場合、デッサンやイラストと異なり90％以上が直線で構成され構図も単純です。 パース（透視図）の図法と異なり、アングルや表現方法など決めておけば素早くイメージを描くことができます。

壁造作、照明計画など立体的な空間構成が把握でき、 お客様や施行業者の方にもイメージ を伝えることができます。 簡単なスケッチを覚えてしまえば ワイドなスケッチも基本は同じですので徐々に描けるようになります。 講習時間は３時間ですが描き方の基本を凝縮してお伝えいたします。 あとはご自宅などで自由な時間に講習レジュメを参考に練習してください。 お客様やコーディネータの皆様の貴重な時間を有意義に使っていただければ幸です。

「tacスケッチ」講習内容

1.線の引き方①(25□・NS法・1,2タッチ)

2.線の引き方②(立方体・NSTD法・洗面台)

3.スケッチ法(基本・応用)

4.陰影・照明テクニック・カラーリング

※製図用0.5シャープペンシル(B芯)ポケットテッシュ・消しゴムをご持参ください。

対象者:インテリアコーディネーター・営業スタッフの皆様

参加費:3,000円(当日会場でお支払いください)

主催：タケダガラスインテリアデザイン事務所
