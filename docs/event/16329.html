この度、青山にて　１対1でお話できる合コン形式の恋活・婚活パーティーを開催することになりました。

仕事帰りにもよれるような、形式ばらない自然な交流会で出会いのキッカケを作りませんか♪

フリースタイルのパーティーではなかなか人と会話するキッカケがつかめない・・・落ち着いて話せない・・・という方、　お酒は苦手、飲み会の雰囲気が苦手・・・という方はぜひわれわれのパーティーにご参加ください。

ここでは1人1人が確実に相手と落ちついて会話ができることを重視した少人数スタイルを開催しております。

大人数が苦手な方、初めての方にも参加しやすい安心できるパーティーだと自負しております。

普通の大人数パーティーが苦手な方や、大手の婚活カップリングイベントは気が引けるという方にこそぜひ参加して頂きたいです。

是非お友達をお誘い合わせの上一度遊びに来て下さい♪

青山　平日夜の少人数制交流会『サポートメントパーティ』とは?

ライフコンサルティングが主催する、パーティーや恋活、婚活に不慣れな方のためのパーティーです。

パーティーになれていない方、知らない人と話すのが得意でない方のための交流の場を目指して、仕事帰りでも気軽に立ち寄れて、一人ひとり確実に交流して頂き、自然な出会いのきっかけを皆様に提供することを目標にしております。スタッフがサポートすることで和やかで安心出来るパーティーにしていきたいと考えております。
