土井高徳(土井ホーム) と 白川美也子 講演が来る2016年11月13日（日）にさわかみ投信 2F 会議室にて「困難を抱えた子供と女性への理解と支援セミナー（協力：公益財団法人 お金をまわそう基金）」を開催します。土井ホーム代表：土井高徳と、白川美也子によるコラボレーションとしての講演会を行ないます

協力：公益財団法人　お金をまわそう基金（代表理事：澤上篤人）

土井高徳 （土井ホーム）と白川美也子の
「子ども問題に対するセミナー」

近年、DVや子ども虐待などの発達期の健全な成長を冒すトラウマや発達課題、アタッチメント課題との兼ね合いによって、複雑な精神症状や問題行動を抱える子どもや大人は大きな困難を抱えています。

これらの困難を抱えた方たちは、医療や心理のみならず、教育、行政、社会福祉、児童福祉、司法、矯正などのさまざまな分野での処遇を受けることになるのですが、その実態やメカニズムはまだ広く世の中には知られていないのが現状です。

このたび、公益財団法人 お金をまわそう基金（代表理事 澤上篤人）協力のもと土井高徳氏（土井ホーム代表）と精神科医・臨床心理士 白川美也子により「困難を抱えた子供と女性への理解と支援」と題して東京（千代田区）にてセミナーを開催いたします。
