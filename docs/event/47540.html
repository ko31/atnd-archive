粉体の基礎と微粒子コーティング・表面改質技術

description

【趣旨】
　粉体へ酸化物やポリマーを均一にコーティングさせるためには系における粉体の分散が最も重要な因子となる。本講では，まず，粉体に関する基礎的な取り扱い方法から粉体物性の各種測定方法を説明した後，単分散微粒子の合成法について概説する。次に，コーティング処理に必要な種々の表面処理方法について説明し，最後にコーティングによる機能性微粒子の調製例をいくつか紹介する。基礎から応用事例まで示すことにより，コーティングに関する知識がより深まることを期待している。

【プログラム】
１．はじめに

２．粉体に関する基礎知識
　　2-1．粉体とは（固体でありながら固体でない）
　　2-2．粉体の大きさと分布
　　2-3．粉体の密度
　　2-4．粉体の表面電位
　　2-5．粉体に働く付着力
　　2-6．流動特性（粉の流れやすさの理論）

３．粉体の計測技術
　　3-1．粒子径，電子顕微鏡（SEM，TEM）
　　3-2．ぬれ性，接触角
　　3-3．ゼータ電位の定義と測定法
　　3-4．比表面積（BET法），細孔径分布（窒素吸着法）
　　3-5．表面分析装置（FTIR，EDX，EPMA，XPS）

４．粉体の調製法
　　4-1．金属アルコキシドの加水分解法（ゾル-ゲル法）
　　4-2．単分散シリカ粒子の合成
　　4-3．高分子ミクロスフェアの合成
　　4-4．粉砕法（遊星ミル，ビーズミルなど）

５．微粒子の表面改質法
　　5-1．DLVO理論の基礎と応用
　　5-2．吸着モデル
　　5-3．界面活性剤の吸着
　　5-4．水溶性高分子の吸着
　　5-5．カップリング剤処理
　　5-6．流体の攪拌特性と攪拌操作

６．微粒子へのコーティング技術
　　6-1．無機物コーティング法
　　6-2．シード粒子成長法（シリカ粒子の大粒径化）
　　6-3．機能性微粒子の調製法
　　　　　導電性微粒子
　　　　　磁性微粒子
　　　　　磁性光触媒微粒子
　　　　　酸化物コーティング金属微粒子
　　　　　酸化物コーティングポリマー微粒子
　　6-4．ポリマーコーティング法
　　　　　水系析出重合法
　　　　　高速気流衝突法（ハイブリダイザー）
　　　　　メカノケミカル法（粉砕法）
７．おわりに
