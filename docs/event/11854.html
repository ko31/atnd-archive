イベント概要

Amazon EC2やAmazon S3に代表されるアマゾンクラウド = Amazon Web Services(AWS)は、ユーザの要望に応じて機能がどんどん追加され、まさにかゆいところに手が届くクラウドサービスになってきています。すでに世界中で数十万のエンジニアがアマゾンクラウドを利用しており、今後は開発者/ITプロフェッショナルにとって必須にクラウドサービスになることは間違いありません。今回は、北陸で初回の勉強会ということで、記念すべき0回目と題して勉強会を開催いたします。

AWSに興味があって使ってみたい方、AWSは触ってみたけれどももう少し深く知りたい方、AWSをもう使われていて周りに仲間が欲しい人、AWSをすでに使いこなしていて他の方にその使い方を広めたい方、AWSユーザグループ北陸のコアメンバーになられたい方、、皆様、すでにAWSユーザ会の一員です。気軽にどしどしご参加ください！！

なお、参加者の方には、$25のAWSサービスチケットが配布される予定です。

開催プログラム

  • 19:10 ～ 19:15 アジェンダ紹介・会場案内
  • 19:15 ～ 19:20 JAWS-UGの紹介
    　　　　　　　　　JAWS-UGコアメンバー 一般社団法人クラウド利用促進機構 荒井氏
  • 19:20 ～ 19:50『アマゾンクラウド概要紹介』
    　　　　　　　　　エバンジェリスト 玉川氏、Amazon Data Services Japan株式会社
    • AWS入門者の方に向けて、アマゾンクラウドの概要を紹介いたします。初期費用不要、従量課金、スケールアウト・インが自在、開発スピードを高めることが可能で、システムレベルの作業から解放されるアマゾンクラウドのメリットをお話します。

  • 19:50 ～ 20:30『アマゾンクラウドのデモンストレーション 既存Webアプリをその場でクラウド化しよう！』

    • 　　　　　　　　　エバンジェリスト 玉川氏、Amazon Data Services Japan株式会社

    • このデモンストレーションでは、典型的なLAMPのWebアプリケーションを例にとって、スケールアウト可能なクラウド・アーキテクチャに変更していく実践例をデモンストレーションを交えご紹介します。フロントエンドにロードバランサ(Amazon ELB)を備え、その後ろに仮想サーバー(Amazon EC2)により並列化されたアプリケーションサーバー、後段にはデータベース・クラスター(Amazon RDS)を備えたクラウドアーキテクチャの魅力をぜひご体験ください。

  • 20:30 ～ 21:00 『RightScaleによるAWS運用管理テクニック』
    　　　　　　　　　JAWS-UGコアメンバー　一般社団法人クラウド利用促進機構 荒井氏
    • AWSの運用管理ツールとして最も有名なRightScaleですが、その協力な管理機能は日本語ではほとんど紹介されていません。ServerTemplates, RightScript, Deployment, ServerArrayといった管理機能について解説し、簡単なデモンストレーションも行います。

  • 21:00 ～ 21:20 ライトニングトーク(※調整中)
    • ライトニングトーク発表者、募集中です。希望者はお名前(ハンドル名可)と題名を記載の上、コメント欄にてご連絡ください。
