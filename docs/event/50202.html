JAWS-UG宮崎勉強会

桜も散ってすっかり暖かくなって参りましたが、5月18日(日)にまたまたやります！ 第2回目のJAWS-UG宮崎勉強会です！
会場は前回と同様 アラタナ さんの会場をお借りして開催させていただきます。
宮崎駅の駅ビルとアクセスしやすい場所ですよ。皆さんふるってご参加を！！

今回はアマゾンデータサービスジャパン株式会社マーケティング本部長の小島英輝氏をお迎えしての開催となりますので、今回もAWSの濃ゆ〜いお話が聞けますよ^^

小島さんはマーケッティングのスペシャリストでこのJAWS-UGを立ち上げた方でもあります。アドビシステムズ在籍時にはFxUG(Flex User Group)の立ち上げを行うなど数々の業績を残されており、今回の勉強会では技術者以外の方も楽しめる内容になるかと思いますので、皆様奮ってご参加下さい！！

LTも発表されたい方がいらっしゃいましたら枠が空いておりますので、ご応募下さい！！
よろしくお願いします。
