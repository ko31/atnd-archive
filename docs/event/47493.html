会場は溜池山王駅から徒歩2分。
迷わず会場までお越しいただけます。

常に新しいカルチャーを発信している港区赤坂にそびえ立つ高層ビルの2階。

～『響 風庭 赤坂』～

会場に一歩足を踏む入れると、そこに広がるのは巨大な大理石が敷き詰められた上質なエントランスが非日常へ誘います。

さらに奥に進むと、ライトアップされた『風庭』を一望できるメインダイニングがお出迎え!!

『風庭』とは？？

専門の庭師により手入れされている竹林。
和の情緒溢れる庭は、夜ライトアップされて、より煌びやかになっております。
研ぎ澄まされた空間で最高クラスのパーティーを開催！

テラスに一歩足を踏む入れると、風が吹く度に揺れる竹の音を感じる事ができ和みます。

お食事は、そんな『風庭』に見合った豪華和食Buffet料理！

そして、ドリンクは鹿児島の有名焼酎『海』などもご用意!!
※パリコレにも登場される日本の唯一のブランド

その他ソフトドリンク・カクテル・ビールなど
沢山ご用意しておりますので女性のお客様、お1人のお客様もご安心ください!

さあ!今宵は「ひな祭り」
都会の喧騒を忘れる上質な空間を眺めながら優雅な出逢いをお楽しみ下さい…☆
