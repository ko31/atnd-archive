東京Basic Technology勉強会[とべとべ]が昨年来の眠りから覚め、第9回のイベントを行います。

テーマは「デスマーチ」と「ブラック企業」。ITエンジニアにとって最大の敵とも言えるこれらとどう向かい合えばいいのかを歴戦の勇士といっしょに考え、学生さんに伝えます。

プログラム詳細はこちらをご覧ください。

終了後、そのまま会場をお借りして懇親会(ビアバッシュ/実費)を行います。参加費は社会人の方は￥2,000くらい、学生さんには割引を予定しています。お申し込みの際、アンケートに参加のご意向をご回答ください。

また、パネルディスカッションのためにアンケートを実施しております。皆様が経験した「デスマーチ / ブラック会社のエピソード」、および「そこで得たもの / 失ったもの」をお聞かせください。

  • アンケートフォーム
  • twitter (ハッシュタグ「#tobe09」をつけてください)
  • kwappa.856 at gmail.com

参加ボタンを押すと、アンケートフォームに「新しい質問」という欄があります。あれおかしいな、消したはずなのに＞＜　せっかくですので、IT業界への疑問・不安、デスマーチ体験、社会人の方は学生さんへのメッセージ、学生さんは社会人の方への質問など、ご意見・ご感想的なものをお書きいただけると嬉しいです。当日ネタにさせていただくかも。

【重要】当日の集合について
会場のセキュリティ上、遅刻なさらないようお願いいたします。オラクル青山センター2Fロビーに、18:55までにお集まりください。ご協力お願いいたします。
