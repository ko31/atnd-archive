PHPソースコードリーディングワークショップ

PHPのソースコードを読むための入門ワークショップです。PHPのビルドとデバッガを使ったリーディング、全体のアーキテクチャ、拡張ライブラリのライフサイクル、などの言語処理系としてのPHPを読むためのとっかかり的な知識やノウハウを、ワークショップ形式で手を動かしながらやっていきます。あんまり流れとかがかっちり決まっているわけではないのでその場の反応みつつ臨機応変にやっていきます。

前準備

ワークショップ参加する方は、以下のような準備をしてきて下さい。

  • PHPのソースコードの取得
    gitやsubversionから取得します。
    git clone git://github.com/php/php-src.git で取得できます。
    svn checkout https://svn.php.net/repository/php/php-src/trunk php-src-trunk でも大丈夫です。
  • PHPをビルド出来る環境
    マニュアルの当該ページを参考にして、すぐにビルド出来るような状態にして下さい。
  • GDBデバッガのインストール
    GDBだけでもいいんですが、ついでにCGDBもインストールしておくことを推奨します。
  • IRCクライアントのインストール
    ワークショップ中は、irc.freenode.netの#yamiphpチャンネルにて資料のURLを記載したり質問を受け付けたりします。
    limechatなんかがおすすめです。
