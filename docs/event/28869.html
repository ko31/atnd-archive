Evernote の User Meetup Spring 2012 を開催します！

今回は、本社 CEO のフィル・リービンが来日し、今まで Evernote を愛し、支えてくれたユーザーのみなさまに直接感謝をお伝えし、交流する時間をたっぷり設けさせていただく予定です。
また、このたび新たに就任した日本担当のジェネラル・マネージャーからもご挨拶をさせていただきたいと思います。

なお、今回のイベントはカジュアルな立食形式を予定しておりますので、Evernote ユーザーのみなさん同士、ネットワーキングの場としてもぜひご活用ください。

フィル・リービンはじめ Evernote 日本オフィスのスタッフ共々、皆様のご参加をお待ちしています。

【Evernote User Meetup Spring 2012 概要】

日　時：2012年5月17日(木) 19:00～21:00（開場 18:30）

場　所：グリーングリル　渋谷店
　　　　東京都渋谷区渋谷1-23-16 cocotiビル3F

参加費：無料

内　容（予定）：
　● CEO フィル・リービンからみなさんへメッセージ
　● 日本担当 ジェネラルマネージャー 着任ごあいさつ
　● 懇親会
　　（ユーザーのみなさん同士や、フィルはじめ Evernote Japan スタッフとのネットワーキング）

定　員：80名
　※ お申し込み先着順とさせていただきます。

募集定員を上回るお申し込みを頂いた場合、キャンセル待ちとなりますが、
繰り上がりのお知らせはATNDの機能を利用してメールにてご連絡させていただきますので、可能であればメールアドレスの登録をお願いいたします。

当日お会いできることを楽しみにしています！
