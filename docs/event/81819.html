description

趣味のオフ会で知り合い、その日のうちに告白されました。
私も少し良いなと思っていたので、巨人倍増少し軽い気持ちでオッケーしてしまったのですが（反省しています）…そこからちょっとずつおかしい事に気付きました。
付き合ってからほぼ毎日一緒にいたのですが
まずすぐにお父さんに紹介されて
ビックリはしましたが、そこまで気にしていませんでした。
普段はとても優しく温厚ですが束縛が激しく、他の男性と連絡を取ることは一切許してくれません。職場の男性ですら、いじけられます。
そして「もし裏切ったら首締めるから、殴るから」などと言われ、そこで私は完全に冷めていましたが別れを切り出せずにいました。
会うたびに何度も過去の彼氏の話を持ち出されていじけられ、そしてついさっき電話したときに今までの不満とか全てぶつけて喧嘩になりました。
その時の言葉遣いと話の通じなさで、シアリス 通販もう完全に無理だと思い別れを切り出しましたが
別れたくない、ごめんねと言われて、結局その時は別れることなくお互いに嫌な所気を付けようねで話がまとまりました。
そして私が休日出勤で疲れていたため、「そろそろ寝ようか」と切り出したら
「は？このまま寝るとか最悪」「寝るってなんだよ」「寝るってなんなんだよぉ！！！！」と怒鳴られました。
結局7時間ほどラインの無料通話で話していました。
29歳会社員♀です。1週間前に、1年お付き合いしていた31歳の彼に別れ話を切り出されました。
彼とのいきさつですが、現在お互い海外勤務、彼から最後の恋にしたいと猛烈アプローチの末恋人に。
彼とは車で2時間の中距離恋愛でも、彼は最初は平日でも私に会いにきてくれるほどでした。
半年前からお互い仕事で忙しく、週末だけとなっていたものの、8月初旬には連休で旅行したり、仲は良好でした。
8月中旬、仕事の激務から心身弱っていた彼は電話の約束を忘れ、その翌日に私が電話で彼に怒鳴ってしまい、それから連絡ほぼゼロ。
それから3週間後、威哥王電話で別れ話をされました。その1週間後に実際に会い、私から冷却期間の提案も、聞く耳は持たず。
時々起こるケンカのたび不安が募り、あの電話が軽くトラウマ、私のカッとなる性格は変えられない、ついに気持ちがなくなった、
仕事と私を同時にハンドルできない、更に2月末で彼は日本、また数か月で次の駐在地に異動することが決定しており、もう無理して付き合いたくないとのこと。
