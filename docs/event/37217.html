3月1日より大阪心斎橋、かに道楽の目の前にあるTSUTAYA EBISBASHI（戎橋）の3階に、AppBankのiPhoneケース販売ブースがオープンいたします！

なんと25坪という非常に大きなブースです！
場所はエレベーターを上がって3階たどり着いたら目の前すぐ！
目玉ケース、注目ケースがたくさんありますぞ！

そんなわけで、3月1日（金）にTSUTAYA EBISBASHI（戎橋）付近のお店でオフ会を開催したいと思います。

場所はTSUTAYA EBISBASHI（戎橋）の近くのお店ウミヤ ハワイアン カフェ。

日付は3月1日（金）。
時間は21時〜23時。

参加費は男性3000円、女性2500円。
2時間飲み放題、食事はビュッフェ形式です。

皆様、ぜひふるってご参加ください！
ぜひ場所はTSUTAYA EBISBASHI（戎橋）を盛り上げてくださいませ！！
