概要

Fireworks と Sketch を中心に、ウェブやU.I.をつくるグラフィックツールの使い方について情報交換しましょう！バリバリに使ってる方はもちろん、「私はデザイナーじゃないぜ！？」って方でも、お気軽にご参加ください。

同時に、Photoshop や Illustrator の猛者募集中！

内容

発表者 内容
@littlebusters Sketchってどんなアプリ？ということで、ザックリと紹介します。
他のアプリとの違いなどもお伝えできれば。
丸山 章 さん FireworksでのRetina対応を考えてみた
モバイルデザイン時に必須なRetina対応。
そこで改めてFireworksでのRetina対応デザイン時の注意点や、制作におすすめの拡張機能などを紹介します。

あとは、参加者同士で「この場合はどう？」とか「こんなことできるの？」とか、ゆる〜い感じで進める予定です。その他調整中。

丸山さんより、Adobeグッズのプレゼントがあります！
