description

ITは、IT業界のものという訳でもなく、家電業界も、自動車業界も、いまや製品にも事業にも、ITを駆使した事業展開をしています。そんな中で、ある意味ITを極めている世界が「ゲーム業界」です。「要望通り動けばいいシステム開発」というのがあるとするなら、そんな世界とは全く異次元の、「利用者に感動を与え」、「販売に直結」できなければ成功とは言えないのがこの世界です。有名タイトルでは、数十億円単位の開発プロジェクトが普通のこの世界で、次々成功をなしえている人は、実はほんの一握りのプロデューサーでしかないのです。

今回の勉強会は、こうした日本を代表するスタープロデューサーの中から、特に幅広い経験をお持ちの郷田さんにお越しいただき、「ゲーム業界の常識」を学んでみたいと思います。

今回は、いつもとは趣向を変え、「座談会」形式で進めたいと思います。ゲーム業界のトッププロデューサーが進めてきたプロジェクト運営手法や、ビジネスで成功させるまでのマーケティングに対する考えなど、みなさんが知りたいことを質問をまじえながら進めていきます。
