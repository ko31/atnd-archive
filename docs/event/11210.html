description

暗闇カフェでは、冬至の「100万人のキャンドルナイト」イベントです。

この日だけ特別に、カフェ店内の電気を消して
ミツロウろうそくの灯りで営業いたします。
カリンバの生演奏のBGMに、美味しいコーヒーとワイン、有機野菜を
使った、パスタやカレーもございます。

暗闇の中、いつもよりゆっくり流れる時間に身をまかせてみたり、

キャンドルの灯りに寄り添いながら、エネルギーのことを考えてみたり。

オレンジ色の灯りとカリンバの音色とともに、あたたかでやさしい夜をお過ごしください。

【カリンバライブ演奏人】
感音Project（かんのん-ぷろじぇくと）
入江規夫　民族楽器ユニット”感謝“のリーダー。

『音により　今　いきていることを感じる』

今、自分が生きている。
それを実感することが容易ではない現代。
自分が自分に戻れる瞬間はどのくらいありますか？

周りに溢れる自然があまりに遠い。
自然の中にいるときに強く自分を感じることができる、と思う。

自然を呼び込む音。 創っていきます。
