第3回立川ボドゲ会(9:30～21:00)

イベント ユーザー

title 第3回立川ボドゲ会(9:30～21:00)
catch 人狼ではなくボードゲームの会です。（現在12名です)
description

はじめまして、立川ボドゲ会の主催のさむすと申します。
今回、第3回目となります。気軽に参加表明をよろしくお願いいたします。

初心者の方もぜんぜんOKです。
部屋を長時間とっていますので、ゆっくりやりましょう！

◆日時
開催日：2015年1月10日(土)
開催時間：9:30～21:00(完全撤収21:30)
・途中参加／退出は自由です。飛び入りでもOKです。
　ただし、定員を超えてしまった場合、参加できない場合があります。
・会場名は「ボードゲームサークル」となっています。

◆場所
場所：立川市錦学習館
住所：東京都立川市錦町３丁目１２－２５　２F第一・第二和室（徒歩約15分)
最寄り駅：JR立川駅

◆定員
32名ほどを予定しています。
現在、参加者はスタッフ(2名)を除いて、５名です。

◆参加料
参加費：500円　学生：300円
※中学生以下は無料です。
※中学生以下は保護者同伴でご参加ください。

参加希望の方は、下要項・注意事項をお読みになった上で、
参加表明していただけると助かります。

◆要項
・ボードゲームやカードゲームを遊びます。（人狼、TCG、3DS、PSPのような電源ゲームはNG)
・要望のあるゲームや持ち込んでもらったゲームを中心に遊びます。
・ゲームの持込歓迎です。
・下記にスタッフの所持ボードゲームリスト（URLなど）を乗せていますので、
　遊んでみたいゲームがあればリクエストもどうぞ。
・手ぶらで参加されても大丈夫です。

◆注意事項
・飲食物の持ち込みは可能ですが、ゲーム中の食事は避けてください。
　(少量の飲食物はスタッフにて持込予定です。)
　食べ物を食べた後は手を洗う等、ゲームが汚れないように心がけをお願いします。
　人からお借りして遊ぶゲームですので、丁寧な取り扱いをお願い致します。
・会場ではゴミを捨てることができませんので、持ち帰りをお願いいたします。
・他の参加者の迷惑になるような行為は禁止です。
・駐車場は6台までスペースがありますが、すぐ埋まりますし、
　できるだけ公共交通機関での参加をお願いします。
・公共の施設を借りています。他の利用者に迷惑をかけないように心がけて下さい。
何か気になったことがあればスタッフ（さむす,U-rey）まで連絡願います。
