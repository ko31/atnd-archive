いきさつ

昨年（2012/7/9）に「Bar Windows Phone in Nagoya」を開催した後、いろんな方から「Windows Azure の Bar もやりたいね！」というお話があました。そこで、単純ですが「それならやってしまおう」と企画したイベントが、この「Bar Windows Azure in Nagoya」です。

そもそもこの「Bar なんとか in Nagoya」シリーズは、東京・渋谷の「Bar Android」が、2012/4/23に「Bar Windows Phone」として開催されたのをヒントにしています。

そして、渋谷の「Bar Android」は、毎週月曜限定のバーなので、名古屋でも同様に毎週月曜限定でオープンしている「名古屋Geek Bar」の会場をお借りします。

どんなイベントですか。

勉強会とかではありません。
みんなでワイワイ／ガヤガヤとWindows Azureについて自由に語り合う集いです。

ただし、やや濃い内容になってしまうかもしれません。
たとえば、「Webサービスは、Webロールで提供した方がいいのか、それともWebサイトのWebAPIの方がいいのか」とか「オンプレミスのActiveDirectoryと連携する場合、PaaSとIaaSをどう使い分けるのか」（注：私はこれは知識不足で答えられません。ｗ）

また、GeekBarというお店の営業の一環として行われるので、好きな時間にやって来て、好きな時間に帰っていただいてかまいません。（ただしチャージ ￥1,500 のお支払いが必要です）
