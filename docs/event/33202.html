『ODD2012f(Ogaki Developer Day)』 の宿泊施設利用
　申し込みATNDです。

・こちらで申込いただいた方の宿泊申し込みはこちらで行います。
・おおよその数字を把握し部屋数を抑えたいので、「泊まるかもしれない」という人も早めに申し込んで
　おいてください。
・利用する宿泊施設は、ソフトピアジャパンの敷地内にある、 ソピア・キャビン です。
・料金は、1,500円(シャワー、トイレ共用)と4,000円(バス、トイレ完備)の２種類から選べます。
　「4000円」の部屋を希望の方は申込に4,000円と記載してください。
・4,000円の部屋は数が少ないため、予約不可となる可能性があります。
・チェックアウトは10:00です。
・キャンセルされる場合はコチラのATNDでキャンセル登録を行ってください。
・タオル、歯ブラシは用意されています。無線LANも利用可能です。
