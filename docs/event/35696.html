開催場所

関東ITソフトウェア健保会館 大久保会議室（http://g.co/maps/ukjnu）

JR大久保駅 北口から徒歩1分

JR新大久保駅 北口から徒歩5分

PHPのWebプログラミングについて勉強します！

実際にプログラムしている画面を見ながら行いますので、書籍よりも分かりやすく学べます。

対象者

  1. HTMLについて学習済みの方
  2. PHPが全く分からない方
  3. PHPがなんとなく分かるけど、もっときちんと学習したい方

学習内容

  1. まず、インターネットでは、どのようにしてブラウザがサーバーにアクセスしているのか、その仕組みを解説します。
  2. その後、サーバー上ではどのようにしてPHPのプログラムが実行されるのか解説します。
  3. 実際に、自分のパソコンに簡単なプログラムを書いてみます。
  4. その後、ページを表示した回数をカウントするプログラムを作成します。
  5. 最後に、フォームを使ったチャットプログラムの作成を行います。

勉強会について

  1. 勉強会はプロジェクターに映した講師の画面を見ながら、自分のノートパソコンにプログラムをします。
  2. 基本的にハンズオン（実際にプログラムを行う授業形式）ですので、ノートパソコンをお持ち下さい。
    （見学だけでも構いません）
  3. ペア・プログラミングという２人１組でプログラムを書く勉強スタイルです。
    なるべく知り合いの方と２人１組でご参加ください。
    １人でご参加の方は、他に１人で参加の方とペアとなります。
  4. １３：１５～１６：１５の約３時間です。
  5. 会議室は１７時まで使えますので、その後は自由時間として使って下さい。帰っても構いません。
    （勉強会の復習や、よく分からなかった箇所の質問など）
  6. 授業料が３，０００円かかります。

XAMPPをインストールしてください

プログラムしたPHPを実行するためにXAMPPというソフトウェアを使います。

これはパソコン上（ Windows, MacOS ）で、Webサーバーを実行するプログラムです。
（基本的にインターネットで使用されるWebサーバーと同じプログラムです）

エディターについて

PHPはHTMLなどと同様に、単なるテキストファイルなので、メモ帳でも編集することができます。
しかし、プログラム専用のエディターを使った方が効率的です。
