description

OpenSolaris に興味があるけれども、何から始めればいいか分からない、または使ってみたものの操作に行き詰まって困っている方を対象に利用方法（インストールやシステム管理の方法、新機能の試し方など）に主眼を置いた初級者向け勉強会です。

内容:



  • 13:00 – 13:30 自己紹介等
  • 13:30 – 15:30 Linuxユーザが最初につまずきそうな事 (瀧　康史)

OpenSolaris には、ZFS や Zone、D-Trace など、凄そうな機能が沢山あります。
使ってみようと、いざ OpenSolaris をインストールしてみたけれども、文化の違いにつまずいてしまった。そんなつまずきそうなことを 1 つ 1 つ、クリアにしていきます。
このテーマでは「Linuxではこうなのに、OpenSolaris ではどうやるの？」という質問はタブーではありません。参加者の方が普段 Linux を使っていて、OpenSolaris でどうやっていいのか疑問があったら、是非参加して、質問してください。

  • 15:30 – 15:45 休憩
  • 15:45 – 16:45 OpenSolaris で使えるデバイス探検隊！(仮) – satokaz

あんなものやこんなものを OpenSolaris で使いたいのだけど、その選別ってどういう風にするの？ぶっちゃけ使えるの？使えないの？
という壁を乗り越えるために OpenSolaris ではどのようにデバイスを認識しているか？どのようにデバイスとドライバを結びつけるか？などを一緒に勉強してみましょう。

更新などありましたらメーリングリスト ug-jposug-at-opensolaris-org でもお知らせします

より上級者向けのトピックを題材にした 東京 OpenSolaris 勉強会を同時開催しています。「びぎなーず」では参加者全員が楽しく勉強出来るように、ワイワイとやっていきたいと考えています。
