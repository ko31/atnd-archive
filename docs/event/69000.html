Web MIDI meets DIY #0

Viling VP 様、 TMCN 様のご共催の下で、Web MIDI Meets DIY #0 を開催します。Web Music（特にWeb Audio API/Web MIDI API）に興味のあり、更にハードウェアに接続をしてゴニョゴニョしてみたい！と日頃から思っている方々向けのMeetupです。

概要

前半はWeb MIDIで接続したオリジナルハードウェア作品のご紹介で、後半は皆様のWeb MIDIと作るハードウェアについて膨らんでいる妄想等をフリーにお話する会にできたらと思っています。

日時と場所
