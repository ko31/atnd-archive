人狼はじめました！

イベント ユーザー

title 人狼はじめました！
catch
description

今や「人狼ゲーム」は世界中で有名なゲームのひとつとなりました。しかし、そんな状況だからこそ、「あんまりルールを知らない……」とはなかなか言い出しにくいのではないでしょうか。そんな方々を対象に、複雑なルールや定石を抜きにして、みんなでワイワイ「人狼ゲーム」を楽しんでみませんか。極力ルールをシンプルにしたものや、オーソドックスなもの、さらにはクロネコキューブ特別仕様のルールを追加したものと豪華3本立てで実施します。経験者の方も大歓迎です。一緒に人狼ゲームを楽しみましょう！
