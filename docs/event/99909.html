世界青少年「志」プレゼンテーション大会

夢は自分ごと、志は誰かのために、みんなのために、 明日のために。 一人一人の中に崇高な思いがある。 それを明確な一文にしたものが志。 そして、志を宣言すると仲間ができる。 応援してくれる人が出てくる。 さらに志は、リレーのバトンのように手渡されていく。

青少年の思いを聞きに、志を応援しに行こう！

■目的：誰もが志でつながる世界を創る！

    青少年が生きる意味を考え、自分で選んで生きる人生の素晴らしさを伝える。
    人や社会に対して貢献して生きようとする自分の思いを世界に伝える。
