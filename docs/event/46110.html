【概要】

近年、スマートデバイスの利用者数増加に伴い、スマートデバイス向けのアプリケーション開発技術者のニーズも高まっています。

また、スマート デバイスの種類も多様化し、技術者もそれらを考慮した上で開発を行うことが求められています。Cocos2d-x はオープンソースの2Dゲーム対応フレームワークで、iOSを始め複数のプラットフォームに対応するアプリケーションの開発を可能にします。そのため、 ゲームアプリ開発現場において最近注目度が高まっているフレームワークの一つです。

本セミナーでは、Cocos2d-xを使ったゲームアプリ 開発と開発に役立つTipsもご紹介いたします。
※ 本セミナーは、iOS対応アプリ開発をモデル に行います。

【参加資格】

iOSアプリケーションの開発の経験を1年程度お持ちの方で、さらに経験を積んでいきたい方。
以下環境設定の完了した、ご自身のPCをお持ちいただける方。

【開発環境】

本セミナーで使用する環境は次の通りです。

Mac OS X v10.8 Mountain Lion
Xcode 4.6 / iOS 6
Cocos2d-x-2.2.1 をダウンロードしておいて下さい。

*Cocos2d-x のダウンロードサイト：
http://www.cocos2d-x.org/download/
