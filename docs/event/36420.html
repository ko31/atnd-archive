イベント概要

Visual Studioの勉強会を開催します！
現在、関西にはVisual Studioの勉強会があまりありません。しかしVSユーザは決して少なくなく、もっと勉強したいと思っている人も多いように思われます。そこで今回VS勉強会を開催します。初回ということで初心者向けです。
今後もいろんなレベル向けに続けて行きたいと思いますので、こちらの アンケート にお答頂ければ嬉しいです。
懇親会3500円ぐらいで会場近くで開催します。参加時に表明ください。

参加費

無料

住所: 　グリー株式会社 大阪オフィス　様　大阪府大阪市北区鶴野町1-9　梅田ゲートタワー10F

  • 会場では食事は厳禁ですが、会場を汚さない程度のお菓子はOKとのことです。また、飲み物はペットボトルのみOKとなります。ご注意ください。
  • ビル入場の際に手続きはありません。10Fまでお越しください。オフィスに入室するにはセキュリティカードが必要になりますので、入り口付近に張り出している連絡先にご連絡ください。
  • 電源が少なめなため、タップを各自ご用意ください。
  • 無線LAN環境はありますが、アクセスが集中すると不安定になる可能性がありますので、なるべく各自で用意されることを推奨致します。

講師ご紹介

名前 自己紹介 発表内容
@moririring ゲームプログラマをしています。わりとC#のことをつぶやきます。わんくま大阪を中心にC#のスピーカーをしたり、C#でTwitterのWinアプリ作ったりしています。コミュニティ「VSハッカソン倶楽部」の代表です。Microsoft MVP for C# Visual Studioをこれからさわって行きたい人、興味があるのでさわった人向けの内容です。但しVSの歴史やC#の基本知識といった内容ではなく、実際にVSをさわりながらどうやって作っていくか、どこにこだわっていくかを紹介します。
