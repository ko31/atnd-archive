title 自分名刺を作る会

description

あなたの全て、会社名刺ヒトツだけで語りつくせますか？

社会人という大人の仲間入りをして数年、自分の中にたくさんの役割が増えてきました。
職場での自分、地域での自分、親としての自分、他にも色々。
新しい出会いもたくさんあるはず。

なのに、初めての人に会うとき、すべて「職場の名刺」でまかなっていませんか。
もしくは、名刺を持っていないので、携帯電話の赤外線通信にてこずったりして
モタモタしているうちに解散時間になってしまうなど、苦い経験はありませんか。

本会では、後藤光正さん著「『2つの名刺』を持ちましょう」の中にある
名刺セラピーと呼ばれる名刺作成ワークショップを実際に行い、
初対面でも話が途切れない、なおかつ人々の印象に残る名刺を作ってみます！
自分の内面や、自分自身も気づかない自分の魅力を見つける機会にもなるはず★

会社で働く人、専業主婦の人、自営業の人、性別年齢問わず誰でも参加OKです。
