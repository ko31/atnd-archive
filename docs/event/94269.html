頼れる子育て社会づくりに取り組むAsMamaがお届けするママのための『育自(＝自分育て)』プログラム。

今回は、大阪で子育てを地域で頼り合える共助のある住まいを提供しているラビーズホームさんとのコラボレーションで開催します。
http://rabbyshome.com/

テーマは、「親子でたのしく英会話 Let’s have fun together♪」
大阪の親子に大人気！の関西イングリッシュクラブによる無料出張レッスンです。
語の手遊び歌や、絵本読み聞かせを通して、身近に使える英語をご紹介♪　親子や地域の仲間とふれあえる、温かくて楽しいレッスンです。

こんな人におすすめ
●子どもと一緒に、英語でふれあう時間を楽しみたい！
●子どもには将来、英語を使える人になってもらいたい！
●子育てを頼れる地域の仲間やママサポと出会いたい！
●コンパクトで広い「充空間」が魅力のモデルハウスを気兼ねなく見てみたい！
●近所の子育て仲間を作りたい！

一つでも当てはまったら、お申込みを♡

黒崎治美

お父さん、お母さん、ごきょうだいも家族での参加、大歓迎♪

★会場は、ラビーズホームさんのおしゃれな1戸建てモデルハウス
http://rabbyshome.com/
ラビーズホームさんの自社ブランド「NICO」は新しいお住まいをお探しの方全てに「快適で、自分らしい、充実した暮らしができる空間」をお届けしたい、
そんな想いを込めた新しい形の住宅ブランドです。
コンパクトでありながら、広い空間のご提案ができ、また人と人との繋がりを大切にしております。

☆AsMamaの託児体験とは・・・？
年齢に応じた制作活動をはじめ、絵本＆紙芝居の読み聞かせ、ダイナミックな新聞紙遊び ふわふわバルーン遊びなど、AsMama 公認子育て支援者ママサポーターによるとっても楽しい集団託児を体験いただけます。
お子様をまだ親族以外に預けたことがない方等も、お友達がいっぱいいる集団託児だったら安心！ママと離れるときはちょっぴり泣いちゃうかもしれませんが、ママサポーターやお友達とのふれあいの中で、だんだん慣れて楽しむことができるようになります。
(1 歳以上のお子様対象。小さなお子様は会場に同伴可。ママサポーターが保護者様の近くで見守り託児いたします)
