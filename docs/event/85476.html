名古屋駅と岐阜駅の真ん中に位置し、
どちらの駅からもわずか10分程の一宮駅で開催する
アート、クラフトに特化したオリジナル作品の展示・販売イベントです。
全国から選考した様々なジャンルのデザイナー、作家が約70名集まります。
他にはない作家オリジナルの作品が勢ぞろい！
小さなお子様から大人の方まで幅広い世代にお楽しみ頂けるワークショップや
クラフト資材の卸販売もございます。

会場は駅直結でわかりやすい場所にあります。
立地も良く、開放的で雰囲気の良い屋根付きテラスでの開催です！

・日時：2017年4月22日（土）10：30～15：30
・場所：一宮駅 シビックテラス
・雨天決行/入場無料
・駐車場：駅周辺近隣のコインパーキングをご利用下さい（※会場直結駐車場有）
