description

防災体験学習施設「そなエリア東京」に、かわいい動物たちが大集合！
うさぎ、やぎ、ひつじ、こぶた、ひよこ達に触れて、エサをあげることもできます。（エサは100円で販売します）家族や友達と遊びに行こう！
動物クイズに答えると、そなエリア東京オリジナルのかわいい動物缶バッチをプレゼント。ぜひ親子で挑戦してみてください。

「ふれあい動物園」
○日時：2011年9月23日(祝・金) ・24日（土）10:00～15:00
※雨天時の実施については公園WEBでお知らせします。
または管理センターまでお電話にてお問合せください。
○参加費：無料、エサやり体験は100円（エサ代）がかかります。

○会場：東京臨海広域防災公園　防災体験学習施設「そなエリア東京」
　　　　〒135-0063 東京都江東区有明3-8-35
　　　　http://www.ktr.mlit.go.jp/showa/tokyorinkai/

○アクセス：
りんかい線「国際展示場駅」より徒歩約4分。ゆりかもめ「有明駅」より徒歩約2分。
※駐車場はございません。

○お問合せ先：東京臨海広域防災公園　管理センター
