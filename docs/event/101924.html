（※19:45～受付開始）

IT業界で正社員としてご活躍されているエンジニアさん＆デザイナーさん向けのセミナーです。

■概要

職場や周りの友人・知人で “フリーランスとして活動中” という方を見かけたり、聞いたりしませんか？
最近よく耳にするフリーランスエンジニア(デザイナー)ですが、「どうやったら、フリーランスになれるの？」「本当に稼げるのか？」などと思ったことありませんか？
本セミナーでは、正社員でご活躍中のエンジニアさん＆デザイナーさん向けにフリーランスになるためのTips勉強会を実施させて頂きます。

＜こんな方にオススメです！＞
・「今すぐではないが、今後の参考までに話を聞いてみたい方」
・「半年、もしくは1年以内にフリーランスになろうと思っているが、情報が少なく困っている方」
・「フリーランスになるか？正社員として転職しようか？迷われている方」
などといったお悩むを、本セミナーで少しでも解消して頂ければと思っております。

■スケジュール

第1部:フリーランスエンジニアになるための勉強会
* ・フリーランス市場
* ・フリーランスになるメリット・デメリット
* ・フリーランスとして中長期的に続けるコツ
* ・フリーランスになる前、なった後にやるべきこと
* ・節税対策、確定申告対策等のお金回りのポイント
* ・お仕事を探すポイント 等
上記をテーマに、フリーランスが知ってた方が得するTipsについてお話しさせて頂きます。

第2部:フリーランス向けサポートサービスのご案内
お仕事紹介、技術教育、開発ツール提供、健康管理、ファイナンス、ネットワーキング等の支援等、
フリーランスエンジニアの方のお仕事と生活面をサポートする各種サービスをご案内します。

第3部:個別相談会＆懇親会
個別にご相談がある方は、セミナー終了後、ご遠慮なくお声掛け下さい。
雑談形式の懇親会も実施する予定です！軽食や飲み物もご用意しております。

■対象となる方

  • フリーランスになろうとしているエンジニア・デザイナーの方
  • フリーランスの働き方にご関心のあるエンジニア・デザイナーの方
  • フリーランスor正社員(転職)としてのキャリアを迷われている方

■費用

無料です！

■今回の特典書籍

参加特典として、参加者全員へオライリー・ジャパン社より刊行の下記書籍のいずれか1冊をプレゼントします！

3.「リーダブルコード －より良いコードを書くためのシンプルで実践的なテクニック－」
