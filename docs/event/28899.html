description

「アジャイルサムライ ― 達人開発者への道」 の読書会を行います。
順番に輪読していき、セクションごとに本の内容についてディスカッションする形式ですすめていきます。
今回はTDD（テスト駆動開発）がテーマです。
普段からできる事を増やしていけるよう読み進めていきましょう :)

開始時間は19時30分です（終了時間は21時予定です）
道場主の都合で申し訳ありませんがよろしくお願いします。

内容・範囲

輪読形式での読み合わせを行います。
読み合わせ後、セクションごとに参加者で疑問や意見を話し合っていく形式を想定しています。

第21回は、以下の範囲を予定しています。

  • 第15章 継続的インテグレーション：リリースに備える

スケジュール

21:00 – 21:10 撤収作業
