今回、2013年初のブロガー伝はあの知る人ぞ知る「西の聖地・竹蔵」で行うことになりました！！

明石にある「魚料理のおいしいお店」竹蔵。
そこは、たくさんの有名な数々の方が訪れるお店。
そこは、ブロガー界隈では有名な方もたくさん訪れるお店。

竹蔵についてはこちら→トップページ ≪ 竹蔵

さて、そんな竹蔵というお店で行う今回のタケゾー伝ですが、いくら竹蔵で行うからといっても目的はブレず！

呑んでブログについて語ろうぜ！

です。

参加資格はただ1つ。「1度でもブログを書いたことがある人」です。

集まって酒呑んでめちゃおいしい料理を食べてブログについてしゃべる。今から楽しみでございます！！

今回、この会を催すにあたってご協力頂いた方をご紹介したいと思います。

iPhoneのことなら何でもござれな「覚醒する @CDiP」の管理人、

@donpyさんです！

どんぴさんにこの会を持ちかけられて実現することができました！本当にありがとうございます！

そして今回、どんぴさんもいるということでいつもの「呑んでしゃべるだけ」に加えて、どんぴさんと我々幹事3人によるプレゼンも少々したいと思います。

テーマは「ブログについて」です。どんぴさんと我々のブログに対する想いというものをお伝え出来ればと思っています。

また、要望があればLTも行えたらと思っていますので、「オレもブログについて喋りたい！！」という人がいれば、是非お教えください！

てなわけで、3/17（日）はタケゾー伝だ！！

日時：3/17（日）13:00（12:45受付開始）〜16:30（17:00完全撤収）

2次会は有志で行いたいと思います♪

人数：20人

場所：竹蔵（兵庫県神戸市西区伊川谷町有瀬766-1 ビサイド大蔵谷102号）

参加費：3000円＋飲み放題（グレードによって800円～2500円に変動）

持ち物：竹蔵のおいしい料理を食べるスペースのある胃、ブログへの想い、あればブログを書く道具

それでは3/17（日）にお会いしましょう！
