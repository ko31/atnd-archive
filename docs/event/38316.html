クラスメソッド開発ブログ課外授業とは、クラスメソッド開発ブログであるDevelopers.IOの記事をメインにした社外勉強会になります。
今回で7回目になりますが、いつもは技術に特化した勉強会をしていますが、SIについて話をさせていただきます。タイトルの"GIGSI" CASE OF CLASSMETHODにもある通り、クラスメソッドの場合のSIを勉強会(GIG)という形でお話しさせていただきます

授業内容

1時間目：「Process Framework CYCRONE ～デザインから駆動するシステム開発～」

ビジネスの課題と、デザインに対する要求、そしてテクノロジーによるソリューション。これらを調和させるクラスメソッド流のシステム開発をご紹介します。

クラスメソッドは、受託開発型のSIを行っている会社です。ここ数年SI業界は厳しい状況が続いています。小さな会社が生き残るため「スピード」「少数精鋭」「情報発信力」に特化し実績を上げているノウハウを公開いたします。
