title 家づくりで大切な知識を学ぶ。
catch
description

安易な選択をしないためにしっかりと情報収集をしてきた。週末にはモデルハウスの見学にも余念がなかった。
それなのに納得のゆく進め方が見えてこない。期待に夢を膨らませて望んだはずの家づくりなのに現実は考えなければならないことばかり。それはなぜだろう。

・自分たちはなぜ家をつくるのか。

・説得されるのではなく、納得できているか。

・自分たちは必要な選択肢の中から決断できているか。

その理由はこんなシンプルなポイントの中に隠れているかもしれません。様々なチャネルを活用することによって容易に情報を手に入れることができ、雑誌やムックなどの媒体の数も増えてきています。そして情報は一般論であって自分たちに向けて発信された情報ではないのです。大切なことは常に「自分たちの最善の選択」をしてゆくことです。

この講座は、漠然と進めがちな家づくりのステップを明確にします。家作りの全体行程、住まいの選び方、資金の計画など、プロデューサー・建築家・ファイナンシャルプランナーが参加して、皆さまがお持ち家作りの疑問にお答えします。
少人数制で、聞かせる講座ではなく参加者の皆さまが知りたいことへアドバイスする講座です。この講座で学ぶことで、それまでの家づくりを冷静に見つめ直す機会となることでしょう。
