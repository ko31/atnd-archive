■ゲストスピーカー

エンジニアのアイドル、アマゾンウエブサービスさんの中の人をゲストスピーカーとしてご招待しております。

堀内 康弘氏

ソーシャルゲーム大手gumiにてAWSクラウドに出会い、スケーラブルでプログラマブルなAWSの可能性に一目惚れ。AWSの素晴しさを日本の全ての開発者に知って欲しいという思いから、2012年3月にアマゾンデータサービスジャパン入社。AWSの普及のために日本中を飛びまわる日々を送る。

大谷 晋平氏（TwitterのID@shot6）

大手SIerを経て、2011年2月AWSへ。これまでオープンソースのJava Webフレームワーク等の開発をしながらHadoop／NoSQLミドルウェアも手掛ける。オープンソースコミュニティでもその存在が知られる。

片山 暁雄氏(@c9katayama)

アマゾンデータサービスジャパンでソリューションアーキテクト職に従事。Japan AWS User Group(JAWS-UG)の活動を通じてクラウドの魅力にとりつかれ、今に至る。メイン言語はJava。好きなAWSサービスはElastic Beanstalk。
