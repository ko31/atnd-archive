Ｗｉｎ7やVistaは捨てるしかない？

Windows7は、来年１月でMicrosoft社 によるサポートが切れます。
以後ウイルスなどには無防備となるので、そのままでのビジネス利用は非常に危険です。

ではWindows7パソコンは、もう廃棄する他に対策はないのでしょうか？
いえ、サポートが切れるのは Windows 7 OSだけです。
OS 以外は、 CPU もメモリもディスプレイもみんな生きています。捨てるのはあまりにもったいないことです。

つまり、ＯＳだけを入れ替えれば、色々な利用方法に転用することができるのです。

ＯＳを手間かけて入れ替えれば使い続けられる？

通常のWindows１０への移行作業は、相応のライセンス費用と時間と技術を要します。
更に、一般にハードウェア能力の小さいWin７パソコンで、Windows１０を動作させるには、
負荷が重くスピード面からみて、ビジネス利用は実用的ではありません。

入れ替え簡単で、ＰＣ負荷が軽い方法がある！

ずばり「変身USB チップ」です。

WinＯＳの入れ替えは「変身USB チップ」を差し込むだけ！
「変身USB チップ」を抜けば、元の状態に安全に戻ります。

そして、差し込むＵＳＢを変えれば、「Win10 like パソコン」や「ＡＩ・ディープラーニング専用機」にも変身できる　という優れものです。
※「変身USB チップ」は、Windows VISTA、８．１～１０についても有効で、同様の動作をします。

下記４種類の変身ＵＳＢのラインナップがあります。
①For ディープラーニング　Python－Pytorch
②For ディープラーニング　Julia
③For Tin Cliant
④For Win 10 likeパソコン
※価格：１パソコン用提供物一式
（ＵＳＢ１個、取扱説明書、主要ソフトウェア取扱説明書）　１万円（税別）

変身ＵＳＢ for DL Juliaって？

高速で動作し、かつ記述しやすいプログラム言語として、近時話題を集めているJulia（ジュリア）を
搭載のＡＩ・ＤＬ専用機に変身します。この変身ＵSBには、JuliaによるＡＩ・ディープラーニングの
教育訓練または開発機・小規模な実用機に必要なソフトウェア一式が組み込まれています。

　そのため、社内で 最先端のAI エンジニアを養成するときでも、新しく専用機器を購入する必要がなくなります。 Win７パソコンにこの変身USBを差し込むだけで、以下のソフトウェアが組み込まれ、起動２分で利用開始できます。

以下のソフトウェアが、カストマイズされ、整合性を取られて収録されています。

  • Customized Ubuntu　 （ＯＳ）
  • Julia　（ディープラーニング用プログラム言語）
  • Jupyter-notebook（対話型プログラム作成、動作確認、結果記録を含む）
  • pandas　（データ分析用ライブラリ）
  • scikit-learn（各種機械学習用ライブラリ）
    　　　分類　classification
    　　　回帰　regression
    　　　クラスタリング　clustering
    　　　次元削減
    　　　ハイパーパラメータの最適化
  • 練習用サンプルデータ 　　　手書き数字の認識　MNIST
    　　　あやめ、しょうぶなどの認識　iris　　など
  • matplotlib　(データの可視化 )
  • Tensorflow（ＡＩ用ライブラリの定番）
  • keras　（ensorflowを簡単に利用するツール）
