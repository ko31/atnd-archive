description

【受講対象】
技術者・研究者・技術営業担当者

【必要な知識】
特になし

【本講座を受講することで得られるスキル】
問題解決力、論理構成力、現状分析力、意思決定力、実行力、が短期間に身につきます

【講師の言葉】
　組織、職場で抱えている問題に対して、自己流で取組んでいる人は意外と多いものです。本セミナーでは、技術者・研究者・技術営業担当者を対象に、問題解決の基本を短期間で習得できるプログラムをご提供します。

　本セミナーには５つの特徴があります。
　　a) 技術者・研究者・技術営業担当者として、これだけは押さえておきたい「問題解決の基本」が学べます
　　b) 自己流の問題解決から脱却ができ、解決の糸口、ヒントを得ることができます
　　c) 目標（あるべき姿）を理解し、行動変容につなげます
　　d) 相手（お客さま、上司、部下、後輩など）の問題やニーズを的確に押さえることができ、相談に乗ることができます
　　e) 職場ですぐに使えるワークシートを活用して、問題解決の理解を深めます

【プログラム】
１．深く考える技術者・研究者・技術営業担当者が活躍する【導入講義、演習】
　　（１）なぜ技術者・研究者・技術営業担当者に問題解決が必要なのか
　　（２）深く考えるとは何か
　　（３）深く考えるための道具はこれだ
　　　　～ロジック・ツリー、マトリックス・チャート、プロセス・チャート
　　（４）深く考えることからの展開
　　（５）ロジック・ツリーを活用した演習
２.問題解決のための現状分析【講義、演習】
　　（１）現状分析から問題を見つける
　　（２）現状分析には“現場力”が必要
　　（３）現状分析の道具はこれだ～SWOT分析法
　　（４）現状分析の道具はこれだ～クロスSWOT分析法
　　（５）現状分析の効果的な進め方
３．問題解決の基本を習得する【講義】
　　（１）問題とは何か
　　（２）組織目標と自己目標の違い
　　（３）問題の３つの見つけ方
　　（４）問題の３つの種類を押さえる
　　（５）問題解決のプロセス（手順）を理解する
　　（６）原因を事実と仮説にわける
４.問題解決を理解するための事例研究【講義、演習】
　　（１）なぜ事例研究を行うのか
　　（２）事例研究の進め方
　　（３）グループ討議、発表
　　（４）講師コメント
５．自分の問題への取組み方【講義、演習】
　　（１）自分の問題を抽出しよう～目標から問題を考える
　　（２）これだけは押さえたい問題解決のポイント
　　（３）技術者・研究者・技術営業担当者は、子供の視点で原因分析をする
　　（４）実践的ワークシート「問題解決シート」の概要
　　（５）「問題解決シート」から「実行計画書」への展開
６．総まとめ講義～問題解決を実践で活用する方法【講話】
