description

秋、いい年になったオトナたちが、一年を振り返り今年もはやかったなぁ、
来年はがんばろうと殊勝な気持ちになるころ。
毎日仕事に忙しい @trico_lour が気晴らしにUstでもやろうかなぁとTwitterでつぶやいていた。

「仕事仕事の毎日では、さらに忙しさが増す年末を乗り切れない。
仕事を忘れて楽しくUstでもしたい！」

これはおもしろそう！と @pyunmaru @yassyama @yuzuki_m が飛びついて、
お茶でも飲みながらのんびりトークでもしようということに。

四人でみそじあたりの人生を振り返り、
マンガ・ゲーム・映画・音楽で「みそじのいっぽん！」
（これまで生きてきたなかでのイチオシの作品）を紹介する
オトナの放課後部室トークをいたします。

オトナもコドモも、お茶でも飲みながらUstreamで一緒にのんびり語りましょう。
そんなわけで、「きっさこ！～オトナの放課後ティータイム～」のはじまりはじまり～。

オトナの放課後ティータイムメンバー

@trico_lour と @pyunmaru と @yassyama と @yuzuki_m です。

みなさんの「いっぽん！」を教えてください！
