自宅警備隊宅外派遣！

新年初の近畿・中部方面隊派遣作戦。

近畿・中部方面隊以外の隊員の参加も無論大歓迎です。

参加人数について

今回特別に設けませんが、参加人数が多い場合は分隊行動も視野に入れています。

なお、以下の項目を守ることが必須条件です。

・自宅警備隊公式wikiの隊員規則を守ること
・コスプレイヤーとしてのマナーや、軍装系の危険性を熟知していること
・こみっくトレジャー21　コスプレ参加の規約を熟読すること
　http://www.aoboo.jp/cosplay.html

タイムテーブル

10:00　　地下鉄弁天町駅改札前集合（※希望者のみ）
以降集団での行動（行動前に点呼・銃器チェック）
14:30頃　更衣開始
以降自由

※希望者は夕凪（@tatusis4848）までリプを願います。

先行入場受付について

今回の会場では先行入場受付はありません。

モデルガンならびにその他装備について

公式の規則に加えて近畿方面隊制定規則に倣い『発射機構のあるモデルガンへの保護キャップの着用義務化』を行います。
当日会場内で移動する際には保護キャップを着用して移動、撮影時も着用を厳守。
保護キャップに関しては各自でメーカーから購入するか、取り扱いのないメーカーに関しては自作も認めますが、必ず『発射口を完全に塞ぎ、色は赤などの目立つ色にする』様にして下さい。

なお、今回の作戦では拳銃と1m以下、ＳＭＧ程度の軽装備でお願いします。（自宅警備隊・武装職安共）

加えて、お掃除装備として最低１つの持参をお願いします。（自宅警備隊のみ）
ほか、ネタ小道具等も持ち込み可能としますが、限度を考えて下さいね。（自宅警備隊・武装職安共に）

撮影時以外はバラクラバ・ゴーグル（自宅警備隊）、サングラス・ガスマスク（武装職安）を外すようにお願いします。

料金

コスプレ参加　1000円
撮影　　参加　 500円
