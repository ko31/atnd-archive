絶対わかる統計的検定・推定【Excel実習付】

description

受講対象・レベル
・製造業での統計解析をこれから始めるかた。
・統計解析をすでに行っているが，解析原理や解析方法に自信のもてないかた。
・ただし，Excel上でのセルのコピーや貼り付け，簡単な数式の作成など，Excelの基本操作ができるかた。
習得できる知識
・解析原理とExcel上での具体的な解析方法が習得できます。
趣旨
製造業での研究開発，生産，品質管理，出荷後顧客対応などで発生した問題を解決するために，統計解析が必要となる場合が少なくありません。本講では，最も基本的な統計解析である，母平均，母分散の統計的検定・推定について，まず，その原理的な考え方を説明します。次に，Microsoft Excelによる演習を交えて具体的な解析手順が身につけられるようにします。
プログラム
１．母集団と標本
　　1.1　全数検査と標本検査
　　1.2　母集団と標本
　　1.3　真値（母数）と推定値
２．平均とばらつき
　　2.1　分布とはなにか?
　　2.2　平均の意味
　　2.3　分散の意味
　　2.4　標準偏差の意味
　　2.5　平均はどのようにばらつくか?
　　2.6　分散はどのようにばらつくか?
　　2.7　ばらつきとかたより
３．統計的検定と推定
　　3.1　検定，推定とは?
　　3.2　第1種の誤り（消費者危険）と第2種の誤り（生産者危険）
　　3.3　母平均の検定: 製品を出荷してよいか?
　　3.4　母平均の推定: 「平均」の値はどのくらい信頼できるか?
　　3.5　信頼区間の意味
　　3.6　母平均の差の検定: 試料Aと試料Bとで，有意差があるといえるか?
　　3.7　母分散の検定: 測定機器の実力は仕様を満たしているといえるか?
　　3.8　母分散の推定: 量産品のばらつきはどのくらいか?
　　3.9　分散の比の検定: 開発品は従来品よりもばらつきが減ったといえるか?
　　3.10 統計的有意差があっても技術的には有意差はないかも
