＊＊＊大好評につき定員10名増加しました！！参加希望の方はお早目に！！＊＊＊

■イベント概要

こんな勉強会が欲しかった！
これから本格的にWebサイトを作りたい初心者の方がつまづきやすいポイントを、受験参考書やお勉強ラップ（http://maji.me/rap/）でおなじみのエンジニアが、余すことなく解説！

MTLのcameran（http://cameran.in）チームのエンジニア( @takahide_h)がお送りする、
今風Webサイトの作り方。
・Facebookと連携したバレンタインサイトの事例を使って、今風CSSの組み方と、Javascriptの基礎まで教えます。
・HTML初心者でも、1日でおしゃれなWebサイトを作って公開できます。

【こんな人におすすめ】
・Web大好き！Tech系の情報サイト読むの大好き！
・HTMLを少しだけ書いたことがある！Javascriptは未経験。
・もっと自分でWebサイトを作ってみたいと思っていたけど、どこから始めていいか分からない

※当日の参加人数や技術レベルに応じて、作るサイトは臨機応変に変わる可能性があります。
※当日の参加人数やノリに応じて、ラップを披露しない可能性があります。披露してもグダグダになる可能性があります。
