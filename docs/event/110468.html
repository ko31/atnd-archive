って思う、印象イケメン、もしくは印象美人っていますよね。

…そして願わくば自分もそんな風になりたい、と思いませんか？

仕事でも高評価！プライベートでも引っ張りだこ！異性からは注目の的！
ってな感じで嬉しいことがいっぱいなはず。人生めちゃくちゃ楽しくなりそう…。

ですが、実際に「印象をよくしたい」「周りからの見られ方を変えたい」
と思う人は多いにもかかわらず、
ここで、ほどんどの人は、大きな壁にぶつかるわけです。

「自分の印象をどういじったらいいか、全く分からぬ」

…（´･ω･｀）ウーン、コマッタ

そこで、そんなみなさんの悩みを解決すべく、今回のGLAMセミナーでは
スーパーセルフプロデューサー、自己ブランディングのプロ✨
をお呼びすることにしました！！！！！

Amazonレビュー評価4.5超え、販売直後に増刷が決定した話題のビジネス書
『「ウケる」は最強のビジネススキルである。』の著者であり、
元お笑い芸人から起業という異色の経歴を持つ、株式会社俺の中北朋宏さんです！

（あのホリエモンチャンネルにも出演されている凄まじい方でございます。）

芸人さんって、自分の顔立ちや体格など元々ある自分の特徴を活かしつつ
「どうやったら売れるか」＝周りから印象に残るキャラを確立できるか
を服装、話し方、立ち振る舞いなど色んな視点から考えるプロ。

その経験をご自身のビジネスで活かしてる中北さんに、
仕事でも、プライベートでも使える自分のブランディング術について教えてもらったら…
明日から人生が30倍くらい楽しくなりそうですよね✨

GLAMのセミナーだからこそできる、NGなしのぶっちゃけトークで
2時間にわたりたっぷりと講演いただく予定です！

ぜひ、自分の変化への期待感と筆記用具を忘れずにお越しください♫
