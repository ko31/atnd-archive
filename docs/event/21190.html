芋煮会＠多摩川河川敷

イベント ユーザー
event_id 21190
title 芋煮会＠多摩川河川敷
catch 東北復興支援と交流会
description

東北の復興支援の一環として、芋煮会を開催します！
目指すは、参加者１００人！ ←最大50人と縮小させていただきます…。
周りを誘って、ぜひ参加しよう！
(直接連絡をもらっている人の分だけ、このサイトでの定員は縮小しますのでご了承ください)

【日時】

2011年11月26日(土)　11:00～16:00

【場所】

二子玉川駅・二子新地駅近くの多摩川河川敷

【会費】

2,000円
(うち500円は入場料として川崎市に徴収されます)

【内容】

芋煮会を通して、様々な人との交流、東北の食材を使っての復興支援、東北の今を伝える報告タイム、その他、個人で告知したいことがあれば復興支援に限らず何でもＯＫな会。

【メニュー】

・芋煮
　- 山形風
　- 仙台風
・石巻焼きそば
・気仙沼ホルモン
・笹蒲鉾
その他、個人の持ち込み物(あれば)

【当日スケジュール】

11:00 調理準備開始
　　　少しは事前に準備しておきますが、参加者全員で芋煮を作ります！
　　　ワイワイにぎやかに作りましょう！
12:00 芋煮完成予定。
　　　みんなで作った芋煮を、みんなで食べる！
　　　以降、石巻焼きそばや気仙沼ホルモン、笹蒲鉾などを順次作っていきます。
15:00 かまどの火を落とし、撤収準備開始
14:00 撤収。

※芋煮会が終わった後は2次会に行くもよしですが、本会は2次会は企画しません。

【持ち物】

11月下旬の河原です。防寒対策はやりすぎと思うくらい準備してください。
特に足回り用に、ひざ掛けなどをお勧めします。
あと、持ち込み大歓迎です！
お酒やおつまみ、手作りのものなど、限られた予算をカバーしてもらえると助かります！

!！重要！!
予算の関係と、お酒の飲む量に個人差があることから、自分の飲むものは各自で用意してもらうようにします。
ご理解とご協力よろしくお願いします。
(一応、多少のお酒はこちらでも用意しますが。)

【会場への行き方】

現地へは二子新地駅からだと近いですが、各駅しか停まりませんので気をつけてください。
なお、隣の二子玉川駅には急行も停まります。
会場は、二子新地駅(川崎市)側の河川敷です。
仮設トイレと橋脚の間の柵で作られた通りの奥に川崎市の入場ゲートがありますので、そこで入場料500円を支払って中に入ってください。
中に入れば、ただの河原です。
運営委員が事前に準備していますので、すぐに分かると思います。
