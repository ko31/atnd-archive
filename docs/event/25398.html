「HTML5」と「スマートフォンアプリ」にピンときたら！

このワークショップは、HTML5を学びながら最終的にスマートフォンサイトの制作を目指します。
スマートフォンサイトの制作の経験のないデザイナやコーディング担当の方々、
HTML5でのサイト制作に興味をお持ちの初心者のかたにもご理解いただけるような入門講座（ワークショップ）です。

ワークショップ内容

・HTML5　スマートフォンサイト用のHTML5
・CSS3　デザインの見栄えを良くする。
・JavaScript　WEBサイトからWEBアプリへ
・canvas　グラフィカルなWebアプリcanvasサンプルはこちら

用意するもの

・PC
　　・safari または chrome のブラウザをインストールしておいてください。
　　・UTF-8の文字コードを扱えるテキストエディタをインストールしておいてください。
