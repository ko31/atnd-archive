毎回好評＆満員のクリエイター・エンジニアのみの交流会【クリのみ】

"クリエイター、エンジニアのみ" が集まる "クリエイター限定飲み会" 【クリのみ】。
2012年から３ヶ月ごとに開催中（1月、4月、7月、10月）、
のべ 2,535人 以上の方とクリエイティブな出会いがありました。

次回クリのみは2019年7月12日（金）です！

申し込みページ からお申し込みください。

過去の参加職種は、WEBデザイナー、DTPデザイナー、イラストレーター、フォトグラファー、映像制作、番組制作、ライター、アニメーター、漫画家、ディレクター、プロデューサー、ゲームクリエイター、プログラマー、フロントエンドエンジニア、HTMLコーダー、3Dデザイナー、CGデザイナー、モーションデザイナー、プロダクトデザイナー、カーデザイナー、プランナー、映画制作、映画監督、脚本家、シナリオライター、編集者、作家、絵コンテ作家、建築家、技術、設計、書道家、造形家、彫刻家、作曲家、研究開発、ファインアート、美大生、専門生など

  • 職種がらシャイな人が多いけど、だからこそ同じ温度で安心して楽しめる会です。
  • 参加者もほとんどが一人参加、半数が初参加なので話しやすく、
  • 初対面がおおいけど、似た仕事なのでとても打ち解けやすいです。

  • 日時：2019年7月12日（金） 19:45〜21:45（開場19:30）

趣旨はこんな感じです。

  1. クリエイター系の人たちと同じ温度でコミュニケーションできる
  2. あまり外に出られない仕事だから新しい縁があるといいね
  3. 他業種のクリエイターがいるから勉強になるね
  4. 学生さんたちにプロの経験や体験を話してあげよう
  5. ゆくゆくはみんなで協力してクリエイティブから日本や世界に貢献してけたらいいね～

なのですが、まず一番大事なのは
「楽しく飲むぜ！語るぜ！」ということなので気軽に来てくださいね。
