description

［講座のポイント］

　アトピー性皮膚炎などの「かゆみ」に有効な鎮痒薬を開発には，「かゆみ」の性質を理解し，動物実験と臨床試験において，自覚的感覚である「かゆみ」を正しく評価することが重要である。
　本セミナーでは，先ず，「かゆみ」の性質を理解し，ヒトと動物における「かゆみ」評価法とその留意点について知ってもらう。また，これまでに報告された「かゆみ」の動物モデルのうちで，前臨床における鎮痒薬の薬効評価に有用と思われるものとその特徴について紹介する。
　自覚的感覚である「かゆみ」を評価することは容易ではないが，新規鎮痒薬の開発に向けた留意点について考察する。
［プログラム］
1．かゆみの性質と特徴
　1-1　かゆみは生体防御感覚
　1-2　かゆみと痛みの類似性相違点
　1-3　鎮痒薬と鎮痛薬
2．かゆみの評価方法
　2-1　ヒトにおけるかゆみの評価方法
　2-2　動物におけるかゆみの評価方法
3．かゆみの動物モデル
　3-1　皮膚のかゆみ
　　3-1-1　起痒物質の皮内注射によるかゆみ
　　3-1-2　アレルギー性皮膚疾患によるかゆみ
　　3-1-3　非アレルギー性皮膚疾患によるかゆみ
　3-2　粘膜のかゆみ
　　3-2-1　アレルギー性結膜炎
　　3-2-2　アレルギー性鼻炎
　3-3　内臓疾患のかゆみ
　　3-3-1　慢性腎不全
　3-4　薬物誘発性のかゆみ
　　3-4-1　オピオイド
　　3-4-2　クロロキン
4．メカニズムに基づいたかゆみ治療のターゲット
　4-1　マスト細胞依存性のかゆみ
　4-2　マスト細胞非依存性のかゆみ
　　4-2-1　表皮ケラチノサイト
　　4-2-2　炎症性浸潤細胞
　4-3　中枢神経系
5．鎮痒薬開発の課題
