description

幡ヶ谷の喫茶店VOCALでおいしいポークカツレツ（もしくはチキンカツレツ）を食べましょう。

※定員はいちおう8人くらいにしておきますが、希望者が多ければもうちょっといけそうです。

集合時間と場所

幡ヶ谷駅北口出てすぐの路地（ラーメン屋、カレー屋などがある）の奥の文華堂という本屋というかレンタルコミック/DVD店の1階で20時20分頃待ち合わせましょう。
遅れる方は現地集合で。道のり→ http://bit.ly/cXpF1g

アクセス
  • 新宿駅から、京王新線（都営新宿線）で2駅め、幡ヶ谷駅下車。
    （京王線とは違う乗り場なので要注意です。南口と西口の間の地下深くから発車します。）
  • 渋谷駅西口から、渋63中野駅行きか、渋66阿佐ヶ谷駅行きバスで、幡ヶ谷バス停下車。
駐車場
  • 幡ケ谷駅（東京） 付近の 駐車場 – Google Map

会場は、甲州街道の北側にあります。
駅付近商店街は一方通行が多いのでご注意ください。
