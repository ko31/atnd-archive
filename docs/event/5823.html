description

質を求めるビジネス交流会です。

真の「農耕型ビジネス」を展開し、「深い信頼関係」が根底にある方々が集まって
より強いネットワークを構築。相互のビジネスを補完・発展させる会です。


＜来て頂きたい方＞


○経営者の方、志の高いビジネスパーソンの方。
○真の「農耕型ビジネス」を運営し、「深い信頼関係」がビジネス・人生の根底にある方。
○信頼関係を築くために、コミュニケーションを重ねることを厭わない方。
○仲間と協力しあって、相互のビジネス、コラボ・ビジネスを発展させる力量・気概のある方。
○ビジネス・人生において、相互に良い時間を共有できる方。



＜ご遠慮頂きたい方＞


○狩猟型ビジネスの方。偽農耕型ビジネスの方。自分のビジネスしか頭にない方。
○表面的な信頼関係しか構築できない方。
○仲間と協力しあって、相互のビジネス、コラボ・ビジネスを発展させる力量・気概のない方。
○ビジネス・人生において、相互に良い時間を共有できない方。
○ネットワークビジネス、宗教勧誘、販売のみが目的の保険・健康食品・スーツ販売などの営業の方
　会の参加者から苦情の出た方。

　

＜今後の展開＞


○定期・不定期に、交流会・懇親会を開催し、相互の信頼関係を深めます。
○もちろん、個別でビジネス交流もOK!　せっかくですので、途中経過を会で発表してください。
　他メンバーの参考になりますし、進行中のビジネスにも、新しいヒントが出てくるかもしれません。
○会のメンバー同士で、セミナー開催も。ご自身のビジネスで参加者の「お役立ち」に貢献するコトはもちろん、
　ご自身のUSPをまわりに知ってもらう良いチャンスになります。


などなど。。。そのほか、面白い企画・楽しい会合も募集します(^_^)。
