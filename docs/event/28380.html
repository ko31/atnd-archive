catch Drupal 7 のデザインレイアウトとテーマを学ぼう！
description

企業のWebサイトとして実績のあるオープンソースCMS、Drupal 7のデザインテンプレートを作成する方法をハンズオンで学びます。　教材は米国Acquia社の “Drupal 7 / Layout & Theming” を使用します。 しっかり学べる2日間コースです。

セッションが概要：

　テーマの基礎
　テンプレートのカスタマイズ
　テーマの開発手順
　先進的なテーマ開発方法の紹介
　レイアウトの制御
　テーマシステムの概要（サブ、ベース、Omega他）
　フロントエンドのセキュリティとパフォーマンス
　レビューまとめ、

デザインテンプレートの概要が理解でき、オリジナルテーマを作れるようになります。　 既にDrupal CMSを使っている方、もしくは構築検討、導入予定の方に最適です。
