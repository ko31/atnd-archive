14:00 ~ 14:40 コバヤシトール
(@toru0325)

Titaniumで実現するリッチHTML5アプリのつくり方 〜Mooklet大解剖〜

昨年のMogSnap、MogSmashに続き、今年の6月にMooklet（http://mooklet.in/ja ）というアプリをリリースしました。Titaniumを使ったネイティブアプリなのですが、実は中身はHTML5なエッセンス満載なことになっていますので、その辺りを実際のコードを見ながら大解剖したいと思います。さらにアプリが完成するまでの流れやデザイナーさんとのコミニュケーション方法、プロジェクトの反省点など、つつみ隠さず報告します（汗）
14:50 ~ 15:30 柳田亜沙美
(@asami_lin)

IT業界で働くキャリアも年齢もバックグラウンドも異なる女子5人組の『Quintech（クインテック）』がアプリを作っちゃいました！女子会を楽しくするアプリ「Roll（α版）」（http://roll.quintech.jp/）

はじまりは、パジャマパーティでシャンパンを飲みながら、「こんなアプリ欲しいよね！」。それから3ヶ月、リリースまでの道のりをお話します！Titaniumのおかげで、10日で開発しました！
15:40 ~ 16:20 うえのまさひろ
(@astronaughts)

Titanium ならここまで出来る。サーバーサイドノンプラミングで作るソーシャルアプリ

日々進化する Titanium Mobile ですが、最近はサーバーサイドのプログラミングをしなくても、簡単なソーシャルアプリが作れてしまうんですってよ、奥さん！
ということで今話題（？）の Appcelerator Cloud Service を利用してアプリを作ってみることにしました。
非常にざっくりとですが、どういう風に作っていくかをご紹介したいと思います（どこまで出来てるかが問題 orz）
