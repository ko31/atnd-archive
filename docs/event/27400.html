大人気すぎて大幅にオーバーしてる iphone_dev_jp 東京iPhone/Mac勉強会 を大阪で見る会です。

東京側の誰かの専用Macでスピーカーとのお話をust中継しつつ、資料はSYNCNELで全員同時に閲覧できるよ！

おしらせ：ビルの呼び出し電話は使えません！ gachasan@gmail.com までご連絡ください。

iPhoneDev-jp 用 ニコ生コミュニティにご参加ください。
放送を觀るのにコミュニティに参加する必要はありませんが、メンバーが増えると放送が安定するのでよかったら参加してください。
当日はこのコミュより東京側で生放送予定です。

※重要※

会場のセキュリティにより、ビルの入退館は社員同伴（当日2名待機）でないと出来ません。13:30・13:55・休憩タイム１・休憩タイム2・終了１・終了２の限られた時間しか原則出入りできないので、当日予定が不安定な方はご遠慮くださいませ。それ以外の時間は@ku_sukeにお声がけいただくと、お迎え/お見送りに参ります

資料・スライドの共有について

SYNCNELの会議機能を利用して資料を共有します。発表者と手元の資料を自動的に同期して閲覧することができます。
ネット中継をご覧になるかた、大阪会場のかたも手元で資料を閲覧できます。
会場に来られるかたも、座席によってはプロジェクターが見にくい場合がありますので、あらかじめ設定しておくとよいでしょう。

SYNCNELのダウンロードはこちら => SYNCNEL – AppStore

参加者のかたの設定手順

  1. AppStoreから「SYNCNEL」のアプリをダウンロードして下さい

    • SYNCNELのダウンロード => SYNCNEL – AppStore


  2. 起動して新規アカウントとして以下で登録→ログインして下さい

    • 表示名 : [任意]

    • ユーザ名 : audience@iphonedevjp.syncnel.biz

    • パスワード : audience


  3. ディスクドライブの一覧が出てきたら画面一番下の「会議」タブをタップして下さい

  4. 「iPhoneDev jp勉強会」をタップして下さい

  5. 後は放っておくと自動的に資料切替やページめくりが追随されます

  6. 画面に触れると自動追尾が切れますので、再度発表者に自動追随する場合はリンクボタンをタップして下さい

SYNCNEL「会議機能」について

  • PDF資料を配信する事ができる
  • 発表者がPDFのページめくりをすると追随してページが開く
  • 発表者がPDFを切り替えると追随して自動的にPDFをダウンロードして開く
