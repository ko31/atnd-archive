■講演内容

★★2018年冬開催！レバテックフリーランス感謝祭！★★

フリーランスの皆様に感謝の気持ちをお伝えしたく、
今回も企画させていただきました！

　★当日はフリーランスの皆様同士の交流会あり！
　★参加者全員にバレンタインプレゼントあり！

◇第1部◇レバテック営業がこっそり教えます！「言語のトレンド・案件動向」

エージェントだからこそ分かる！『言語別案件動向・トレンド』をテーマにお話します！

『最近、キテる言語って何？』
『言語別の案件数の割合って？』
『前年と比べてどういう案件が増えてるの？』..etc

★将来を考える基準になる！
★変化の早いIT業界の波に乗れる！
★スキルチェンジのヒントになる！
★エンジニア仲間にドヤ顔できる！

ここでしか聞けない内容ばかりです！是非お楽しみに！

◇第2部◇ライトニングトーク　「フリーランスとして生きていく術」

今回は『フリーランスとして生きていく術』についてお話しします！

技術面、コミュニケーション術等、様々な視点からノウハウを伝授いただきます。
今回はITフリーランスとしてご活躍されている3名がご登壇予定です。

『フリーランスとして将来何をすればいいか分からない』
『フリーランスって一概に言うけど、他の人はどんな働き方をしているのか？』
という方は必見です！

フリーランスとしての今後の活動についてお悩みの方、
是非イベントに来て・見て・聞いてみてください！

◇第3部◇交流会

フリーランス同士の交流の場として、今回も交流会を行います！
お食事とお飲み物もご用意しますので、当日は飲みながらの参加もOKです。
参加費無料ですので、ぜひ気軽に遊びに来てください！！

※内容は変更になる可能性があります。


■講演者プロフィール

「35歳以降でも技術者として活躍するために考えていること」
フリーランス2年目Androidプログラマー
船津 真一郎（フナツ　シンイチロウ）氏

29歳からプログラマデビューの遅咲き6年目Androidアプリ開発プログラマー。
1年半前にSI系の会社からWeb系に転向するついでにフリーランスに。
現在、業務委託として常駐しながら個人開発の両立を目指す。
最近はプログラマーとして生き残っていくためにKotlin、Clojureなど
複数のプログラミング 言語や開発手法を勉強中。
ニッチを狙いながらもトレンドに強いプログラマーを目指す。

「秘伝！！PJ成功に向けたた担当者とのコミュニケーション術」
フリーランス　2Dデザイナー
石橋　慎平（イシバシ　シンペイ）氏

美術系の専門学校を卒業後、アパレル店員としてアルバイト生活。
その頃漫画家を目指しイラストの勉強を始める。
在宅でイラストの仕事を請け負い
ガラケー時代のソーシャルゲームイラストを経験。
ソーシャルゲーム業界に興味が湧き、上京。
遊技機業界を経て、2015年、フリーランスとして独立し
夢であった大手ゲーム会社へ参画。
並行して友人の紹介でキャラクターデザイン講師として
フランス人専門学生にイラストを教え始める。
現在は2Dデザイナーとして大手ゲーム会社で
人気IPゲームの製作に携わっている。

「ブラウザアニメーションの軌跡とこれから」
フリーランス　開発エンジニア
N氏

2011年にソーシャルゲーム業界に参入。
Flash や CSS/javascript による実装の表現や最適化についての経験を積む。
その後、縁があり、ソーシャルゲーム会社の立ち上げに関わる。
トイレ掃除から始まり、デザイン/演出/プログラムの実装まで
3年ほどガムシャラに走り続ける。
その後、2017年にフリーランスとして独立。
現在はフロントエンドとして、 主に Canvas 周りの実装などを担当。
