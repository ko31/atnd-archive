description

Vyatta とは、オープンソースの技術を集めて作られたルーターソフトウェアです。
一般社団法人LOCAL インフラ部では、 日本Vyattaユーザー会 による “ALL JAPAN VYATTA USERS MEETING 2011” の一環として、北海道地区における第1回目のVyatta勉強会を開催します。
講師に 日本Vyattaユーザー会 運営委員 松本直人 氏 をお招きし、Vyattaの機能・魅力について紹介していただきます。

開催概要

  • 参加費用 : 無料
  • 開催協力 : 一般社団法人LOCAL インフラ部
  • 前提知識 :
    • 初心者、上級者を問わず、Vyattaに興味があればどなたでも参加可能です。
    • Vyattaってなに？という方は、こちらの記事が参考になるかと思います。
      参考記事： @IT – Vyatta ～クラウド時代の仮想ルータ活用術
  • 会場について
    • 18:30より入室可能です。 入口は施錠されていますので、インターホンを鳴らして「勉強会参加者です」とお伝えください。
    • PC持ち込み可能(必須ではありません)、電源を利用可能です。 ネットワークについては提供できませんので必要な方は各自準備願います。

プログラム

Session.1「Vyattaを知ろう！」

  • オープンソースソフトウェアルーターであるVyattaの基本機能と概念、さらにはその中身と使い方のTIPSについて皆様と情報共有をさせていただきます。
  • 資料は こちら からダウンロード可能です。

Session.2 「災害時における情報分析と共有」

  • 3.11の災害時には様々なメディアによる情報共有と分析が行われました。特にコンピューティング資源を使った分析と情報共有の結果について、皆様と共有できればと思います。
  • 資料の公開は行いません。当日のお楽しみです！

講師 : 松本 直人 (さくらインターネット研究所 上級研究員、日本Vyattaユーザー会 運営委員)

勉強会終了後、講師の松本さんを囲んで懇親会を予定しています。(出欠は当日に取ります)
ぜひこちらもご参加ください。

その他不明点・お問い合わせは @yamaki_panda までご連絡ください。

多数の皆様のご参加をお待ちしております！
