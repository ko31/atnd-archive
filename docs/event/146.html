★きっかけ★　

周りの同年代の人たちの結婚や出産が多くなってきました。

それと同時に自分の子どもに虐待したり悲惨な事件が増えてきてる気がします。　

ふと親になるイメージをしてみた時に正直不安でした。

ここでは楽しい体験や交流を通じて子どもを持つことや、幸せな未来のママ＆パパライフを想像してもらえたらいいなぁと思います。 　

子育て真っ最中のママ＆パパさんや妊娠中の方にも息抜き＆交流の場として参加して楽しんでもらえたらいいですね。 　

◆ステキなママ＆パパになるために◆ 　

子どもがいない方は、いろいろな体験を通じて幸せな未来や親子の生活をイメージしてみて☆

子どもがいる方は、先輩ママさんに話を聞いてもらったり、ママ友を作ったり楽しんでね(^o^)

あとは将来のママ＆パパに良きアドバイスなどしてくれたらウレシイです。

▽カテゴリー▽

●妊娠・出産

妊婦体験ジャケットを着用して妊娠体験してみよう！

赤ちゃんの抱き方、お風呂の入れ方など具体的にお世話の仕方を体験してママ＆パパになる準備を！

助産師の方に質問できるコーナーも検討中です。

●子育て

赤ちゃんってカワイイだけじゃない！？

子育ての大変さと楽しさの両方を知ることができます。

先輩ママさんの体験談は貴重ですね。

子どもとふれ合ったらハッピーになれちゃいますよ。

●ママライフ

出産後も仕事を続けたい！でも家事と育児と両立できるかな？

ワーキングマザーの実体験の話が聞けるコーナーを準備予定です。

ママは家事に育児に大変…やっぱり息抜きだって必要！

プライベートの楽しみ方を情報交換してみては？
