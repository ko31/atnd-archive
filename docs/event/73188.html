ボードゲーム忘年会

catch
description

■概要
ボードゲームを持ち寄って皆で遊ぶ会です。
プレイ後に懇親会もします。
プレイ、懇親会、どちらか一方だけの参加も可能です。
プレイ参加費は無料で、ゲームと飲食物を持ち寄り。懇親会は後述。
複数人参加のゲームも個人プレイのゲームも何でもアリです。
人数がそこそこ集まれば「テラミスティカ」をプレイしたいと思います。

■プログラム
13時～　　小規模ゲームをポツポツと
14時～　　自己紹介＋好きなボードゲーム紹介（任意）
15時～　　小規模ゲームと大規模ゲーム（テラミスティカなど）で部屋を分け、各自好きな方へ
18時半～　懇親会

■持ってくるゲームについて
各自持ってるゲームリストをコメント欄に記入し、実際に持ちよるゲームを相談しましょう。

■懇親会
会場で軽食をつまみます。
カンパ制、参加者は一人1000円以上のカンパをお願いします。
持ち込み歓迎。

■会場情報
電気、電波、水は無料提供致します。
部屋を複数とっているため、
一人～少人数用ゲームをプレイする部屋と
テラミスティカやカタンなどそれなりの人数集まってプレイする部屋とを分けて進行する予定です。
どちらでも好きな方にお入りください。
