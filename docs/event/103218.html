＜損をせずに不動産を買う方法とは？公認不動産コンサルティングマスターがお得な情報を教えます！＞

「不動産を買う前に知っておくと得する情報ってありますか？」

最近このような質問をよくいただきますが、もちろん、知っておくと得する情報はたくさんあります！

当セミナーに参加いただくと、下記のようなお悩みを解決できます。

・住宅ローンの金利ってどこまで下がるの？　保証料が安くなる方法はないの？
・フラット35を検討しているけど、フラット35Sが簡単に利用できるようになる方法はないの？
・マンションで築25年を超えていても、住宅ローン控除が使えるってホント？
・住宅取得の非課税制度を利用した、親子間贈与と税制優遇の活用方法とは…？
・物件探しサイトってどうやって見ればいいの？　どういうポイントを見るといいの？
・不動産会社が提案する資金計画に、抜け漏れはないだろうか…？
……etc

本セミナーは、不動産売買において「お客様にとって最善の方法」を提案することを得意とする資格『公認不動産コンサルティングマスター（宅建士の上位資格）』をもつ、不動産売却のエキスパートが開催します。

ぜひお気軽にご参加ください！

▼セミナーにご参加いただきたい方
・これから不動産を購入しようか検討している方
・住宅ローンを借り換え検討している方
・不動産の市場動向について知りたい方
・買い替え時に有効な税制活用方法について知りたい方
・将来のために不動産を購入するか、このまま賃貸でいいか、どちらかで迷っている方

▼セミナーで得られること
・物件（新築・中古）購入時に、住宅ローン控除をかしこく活用する方法を知ることができます
・「どの住宅ローンが自分に適しているのか」を判断できるようになります
・金融機関（大手・地銀）やフラット35の金利動向、融資事務手数料が理解できるようになります
・フラット35を利用する際のメリット・デメリットが分かります
・不動産サイトの見方や、重要視すべきポイントが理解できるようになります
・不動産会社から提案された資金計画について抜け漏れがないかを精査できるようになります

▼その他
・受講に必要な持ち物：ノート・筆記用具
・お子様連れ歓迎です！
・セミナー参加後に個別に質疑応答時間を設けることも可能です

▼講師
公認不動産コンサルティングマスター：井口 忠二

福岡県出身。明治大学商学部卒。グロービス経営大学院経営学修士（MBA）修了。
某大手不動産会社に入社し、売買仲介営業を経験。東京23区エリア営業トップセールスを2年連続受賞。その後、不動産の売買賃貸仲介・管理・コンサルティングをワンストップで対応する株式会社アスパートナーを経営。売買仲介成約件数600件超。
不動産会社経営以外に、下記公益活動も力を注いでいる。2015年医療介護の課題解決を図ることを理念とした一般社団法人ソーシャルアップを設立し、代表理事に就任。
2017年4月から福岡市の「Startup Cafe」に参画し、創業支援における相談員：アンバサダーコンシェルジュを兼務する。2018年4月から福岡市剣道連盟理事に就任し、剣道×ITを導入し、業務効率の改革に取り組んでいる。
