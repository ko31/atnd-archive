趣旨

熱海の掛け流し天然温泉に浸かり、泊まり込みで DNS を学びましょう。 昨年のDNS温泉 は騒ぎになった毒入れに焦点をあてましたが、今回はわかっているつもりでわかっていない DNS について基礎から RFC とインターネットシミュレータ (VITOCHA) でじっくり理解を進めましょう。参加者からのライトニングトーク (時間延長可) も歓迎します。

日程

9月12日 (土) 13:00 – 9月13日 (日) 12:00　(現地集合、現地解散)

会場

  • 伊豆山研修センター
    JR東海道線・JR新幹線「熱海駅」下車　
    →タクシーで約10分(熱海駅より)
    →伊豆東海バス　七尾原循環｢般若院前｣下車→徒歩10分
  • 参加費: 12,000円 (税込)
    • 夕朝食+宿泊(5人/1室)+酒 (酒を飲まない人と学生は10,000円)
  • 女性部屋も用意します。

申し込み/定員

プログラム (仮)

一日目

13:00 始業式

  • オリエンテーション
  • 自己紹介

13:40 1時間目 哲学

  • DNS の基礎を疑い再考する
    • 権威とは何か
    • 委任とは何か
    • ゾーンとは何か
    • 内部名とは何か
    • グルーとは何か
    • 再帰とは何か
      (テキスト: RFC1034, RFC1035)

15:10 休み時間

15:20 2時間目 図工

  • DNS を触ってみる
    • 仮想ネットワークを作る
    • ルートサーバを作る
    • TLD を作る
    • 自分のドメインを作る
    • キャッシュサーバを作る
    • 検索してみる
      (教材: ノートパソコン, VirtualBox, VITOCHA)

16:50 放課 (お風呂など)

18:00 夕食

19:30 3時間目 学級会

  • LT大会 & 懇親会
    • 「nslookup しか知らなかった人がDNS温泉に参加するまで」
    • 「毎日 dig ったら分かったこと 〜某 registry 編〜」
    • 「幽霊を見てみよう」など

22:00 就寝

2日目

7:00 朝食

8:30 1時間目 理科

  • キャッシュサーバの動作を知る
    • 反復検索を理解する
    • 何がキャッシュに入るか
    • キャッシュはどう変化するか
    • 浸透の正体を知る
      (テキスト: RFC2181 / 教材: ノートパソコン, VirtualBox, VITOCHA)

10:00 休み時間

10:30 2時間目 社会

  • DNS の脆弱性を知る
    • 共用サーバの問題を理解する
    • キャッシュポイズニングを理解する

12:00 解散
