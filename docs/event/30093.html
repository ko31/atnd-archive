description

よい日本酒に一番合う食べ物ってなんだろう、、、
間違い無いです。焼き鳥です。

でもちょっと、会社帰りに居酒屋でソレはオヤジくさくないですか、、、
ということで開催されるこのイベント、

スタンディングバーで焼き鳥と日本酒をいただいちゃいます。

日本酒のラインナップについては、
大手日本酒の団体に勤めるO氏によるセレクト。
（ちょっと会社関係の話なので正体は明かせないものの、日本酒に精通している方です！）

こんな魅力的なラインナップになっています。
飲み比べができて楽しいですね。
・本醸造２本
・純米酒２本
・純米吟醸５本
・純米大吟醸５本
・吟醸５本
・大吟醸４本
・古酒１本
（本数は、応募人数により調整）

焼き鳥も専門店から取り寄せまして、
美味しくいただけるように手配しています。

ぜひ、日本酒大好きな方も、
日本酒をこれから始める方も、
飲み比べながら楽しめるイベントです。

年齢、男女に関わらずご応募下さい！
