この度、プログラミングに触れたことのある大学院生の皆さんにお会いしたい
IT企業が集まる、2014年修了予定の方向けのインターンイベントを
【6月16日(土)に東京にて】開催します！

インターンという言葉はよく耳にしますが、はっきりとイメージできない方も
多いかと思います。社会に出る前に企業の中で就業体験をすることは、自分の
将来を決める上で大いに役に立ちます。企業の方と直接お話ができるこのイベ
ントを是非ご活用ください。

プログラミング経験については、
「授業で少し学んだだけ」
「勉強を始めたばかり」
「趣味でかじった程度」
といった方でも、IT企業に関心があれば参加大歓迎です。

昨年開催した同様のイベントでは、企業の人事担当者だけでなく、
現役エンジニア、さらには代表取締役やCTOにご参加いただき、
インターンについてご紹介していただきました。
中には、こんな工夫を凝らしたインターンも…

　○昨年のイベントにて紹介されたインターンの例
　・1週間で新規事業を立案、経営陣にプレゼン！超実践型インターン
　・詳細非公開！無人島を舞台にした合宿形式のインターン
　・シリコンバレーへ殴り込みできるインターン

　※昨年度開催インターンイベントのレポートページ
　　http://www.acaric.jp/event/report/2013/20110618tokyo.html

インターンへの参加を考えていなくとも、まずはIT業界の第一線で
活躍する先輩と身近に接して、皆さん自身の将来像をイメージして
みませんか。

皆さんのご参加を心よりお待ちしております！

※登録者以外の方のご参加も歓迎しております。
ご友人もお誘いあわせの上、ご参加ください！（事前予約が必要です。）

◎事前予約はこちらから⇒　http://goo.gl/xKRb3
※予約者多数の場合は、受付を締切る場合がありますので予めご了承ください。

==　開催日時 ・ 開催場所 ・ 参加対象者 ・ 参加方法 ==

□タイトル：「アカリクITインターンイベント」
■日時：2011年6月16日（土）13:00～19:00
□場所：東京・AP浜松町
地図 … http://www.ap-hamamatsucho.com/info/access.html
■参加費：無料
□対象：
・2013年4月～2014年3月修了もしくは見込の方（2014年4月1日に入社可能な方）
・授業、研究、趣味などでプログラミング経験がある方
・IT企業へのインターンシップ参加を希望する方
　（学部生、修士課程学生、博士課程学生）
※プログラミング能力やIT企業への関心等についてご確認させていただく
　場合がございます。
■定員：70名
□参加企業数：10～11社（予定）

★☆★☆★☆★☆★☆イ ベ ン ト の 特 徴 ☆★☆★☆★☆★☆★

【少人数制なので企業の人事担当者や現役エンジニアとじっくり話せる！】
大人数で行われる一般的なイベントとは違い、少人数のグループで
採用のキーマン他、現役エンジニアと接することができます。企業の話を
じっくり聞いて、積極的に情報を得てください。

・初めに、参加学生の皆さんに対し参加企業が1社ずつプレゼンを行います。
そのプレゼン内容をもとに、「1ターム30分×5ターム」のタイムスケジュール
に従って各企業ブースを自由に回っていただきます。

・話を聞けなかった企業やもっと詳しく話を聞きたい企業については、イベント
終了後の懇親会をご活用ください。

席数に限りがございますので、お早めにご予約下さい。
それでは皆さまのご参加を心よりお待ちしております！
