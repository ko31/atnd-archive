統計解析環境「R」の勉強会、Nagoya.R #18 を開催します。
分野も年齢も職業も問わず、「R」について一緒に学びましょう！

入門者の方も大歓迎です！Rを始めるなら是非ここで！

★メニュー

※Rを触ったことのない方も、Rの基本的な操作ができるようになる内容となっています
　 14:45- 【開会行事】
-主催者挨拶

2部 14:50- 【Lightning Talks・発表】

  1. 尾関修治「R Bootcamp / Python Bootcamp 開催のお知らせ」（5min）
    • 2018年3月に開催される、R Bootcamp ならびにPython Bootcampのご案内です。 サイト

  2. 西村嘉人 「初心者が関数作ってみた」（5min）
    • フォルダの中から任意のファイル（好きな拡張子）を好きな数だけコピーあるいは移動させる関数のお話です。

  3. 川口勇作 「形成モデルの構造方程式モデリング（をしようとした話）」（10min）
    • lavaanパッケージで、形成モデルの構造方程式モデリングを行おうとして悪戦苦闘した一部始終をお話します。

  4. m-yoshino「ビッグデータは古典統計を破綻させる～Rで検証するビッグデータの性質～」（20min）
    • ①ビッグデータは、仮説検定が常に有意になる、②ビッグデータは、次元の呪いがある、③ビッグデータは、予期せぬ線形制約がある
