catch キャンドルの下でしっぽりと詩を読みませんか？
description

Studio Kuraは夏至にあるキャンドルナイトのイベントに参加してきましたが、今年は冬至のイベントにも参戦することに決定いたしました！！最近掃除が完了してリニューアルオープンした第二の蔵にて「詩を詠む会」を開催いたします。

キャンドルを灯し、酒を入れつつ詩うたう、盛り上がること間違い無しのこのイベント！自分の好きな詩人の詩を読むもよし、自作で参戦するもよし、皆様のご参加をお待ちしております。

今年やり残したことはもうないか？私にはありました。詩を詠みます。
