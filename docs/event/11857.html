アプリだけがスマートフォンじゃない！
ブラウザベースのゲーム、メディアも大きな可能性を秘めています。
そこで第1回SmartHacksではスマートフォンのブラウザゲーム、プラットフォームにフォーカスして、テクニカルな勉強会を行います。

HTML5 + CSS3 + Javascriptとその関連技術を使って開発したブラウザゲームを事例にした、
スピーカーの話を中心にインタラクティブな内容を予定しています。
