「HTML5」と「スマートフォンアプリ」にピンときたら！

このワークショップは、HTML5を学びながら最終的にスマートフォンサイトの制作を目指します。
スマートフォンサイトの制作の経験のないデザイナやコーディング担当の方々、さらにHTML5でのサイト制作に興味をお持ちの初心者のかたにもご理解いただけるような入門講座（ワークショップ）です。

ワークショップ内容

  • HTML5　スマートフォンサイト用のHTML5
  • CSS3　デザインの見栄えを良くする。
  • JavaScript　WEBサイトからWEBアプリへ
  • canvas　グラフィカルなアプリ

用意するもの

  • PC(Mac/Windows
  • ブラウザ(safariまたはchrome)をインストールしておいてください
  • UTF-8の文字コードを扱えるテキストエディタ（秀丸、TeraPad、サクラエディタ、miなど）をインストールしておいてください。
  • 参加費5,000円（このコースではリピータ割引はおこなっておりません）
  • お名刺１枚(プライバシーマークの都合上入館の際、ご提出ください。)
  • iPhoneまたはAndroid端末をお持ちでしたらご持参ください（必須ではありません。）

その他

  • Web関連のデザイナー、コーダ、Webに興味のある初心者・企画関係・ノンプログラマ向けです。
  • HTMLなどのコードをガリガリ手打ちします。
  • スマートフォン端末をお持ちでない方は、こちらから。
  • 無線LANを準備する予定です。
  • 定員は８人です。会場の都合でこれ以上は増やせませんのでお早めにお申し込みください。
  • 定員や部屋の都合がありますので、直前でのキャンセルは本当にご遠慮ください
  • この講座は 株式会社ビーグッド・テクノロジー様の協力をいただいています。
