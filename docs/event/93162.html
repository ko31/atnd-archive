Insurtechに興味があり「新しいイノベーションを起こしたい！」と思うU35集まれ！

  • 金融xTECH=Fintech
  • 教育ｘTECH=Edtech
  • 物流ｘTECH=Logitech

など、日本においても多くの新しいTECH企業が生まれています。MoneyFowardのマザーズ上場やDMMによるCASHの買収など、TECH企業のニュースも日夜、茶の間を賑わせています。

しかし！保険ｘTECHであるInsurtechの流れを作り出す新しいTECH企業はほとんど出てきていません。

それには大きく２つの要因があると考えています。 一つは人材の要因。Insurtechに取り組むためには、データ収集や分析に関する深い知見を持つエンジニア人材と、保険業というビジネスに対する深い経験のあるビジネス人材の組み合わせが不可欠です。

いくらエンジニアが優秀であったとしても、実務を知らなければ、新しいアイデアが現実的かどうか見極めることが難しいでしょう。一方保険業界に精通している人は、概して保守的な傾向が強く、新しいビジネスに対して前向きと言える人は多くはありません。

二つ目は規制・実務面の要因です。現在の日本の規制環境や実務環境では、他業界のテクノロジーや商習慣、海外事例をそのまま保険ビジネスに活用するのは難しいと言われています。保険行政の運用面の制約や、過去の新規参入に際して既存業界との調整により作られた複雑なチャネル規制、業界独自のお作法など、新しいビジネスを展開する上で様々な障害があります。

この現状を打破し日本のInsurtechを進めるためには、同じ問題意識を持った若手（35歳以下限定）が交流し、オープンイノベーションが必要不可欠であると考えます。

そこでこの勉強会は保険会社、代理店、既にInsurtech企業に勤める方、エンジニアなど、保険業界に関心のある若手が集まり、保険の未来やテクノロジー活用について話し合い、密度の濃いディスカッションを通して、新しいシナジーを生み出すことを目的としています。

  • 保険業界で働いているが、最近このままでいいのか、もやもやしている！
  • 日本の保険業界を変えたい！
  • とにかく新しいことをやってみたい！
  • エンジニアとしての技術を保険に活用したい！

と少しでも思われたならぜひご参加して下さい。 日本のInsurtechの将来について考えましょう。
