description

・自分にとっての「仕事」の意義を見つけたい
・社会に必要とされる「仕事」をしたい
・社会起業・社会貢献に興味があるが、一歩が踏み出せない
こんな、想いをお持ちなら、社会起業大学の「特別無料体験講座 アクションラーニング編」にお越しください。

本講座は、社会起業大学のソーシャルマネジメントクラス（基礎クラス）の必須授業、アクションラーニング演習の体験講座です。

ご自身のミッションを確立するためには、深く自分を理解し、本当に自分のやりたいこと、誰にも負けない強みを自覚すること。そして、その強みを活かして「行動」を起こしていくことが重要です。
では、どのように自分を掘り下げ、アクションを起こしていけばいいのか？そのエッセンスを持ち帰りたい方はもちろん、社会起業大学の授業の雰囲気を知りたい方にもおすすめです。過去に無料体験授業をご受講された方もお気軽にご参加ください。
