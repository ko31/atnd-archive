札幌で、第1回F#ハッカソンを開催したいと思います。

ゆるふわなイベントですので、恐れる必要はありません。基本的人権は守られます。

ご自由にお過ごしください。

疑問、質問、意見等は気軽にコメント欄にお書きください。(別の日がいい！など)

  • OCamlで何か作りたい型 ← !
  • Haskellのお勉強をしたい型 ← !!
    …など
