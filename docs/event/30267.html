Gradleとは

Gradle (グレイドル)は Groovy ベースのビルドシステムです。2012/6/12に1.0がリリースされました。

Maven とか Ant とか Ivy とかの単語でピンときた貴方。そうですそんな感じのアレです。

アレじゃ分からんという声が聞こえた気がするのですこし特長を挙げるとこんな感じです。
（間違ってたら指摘願います）

Gradleの特長

  • 中の人はGroovy
    Groovyベースの DSL です。Groovyなので色々と柔軟にカスタマイズできます。
    ビルド設定ファイルもGroovyで書けるから、簡単で読みやすくなります。
    ゆるふわ。
  • 既存ビルドツールと仲良し
    Antは大親友です。MavenやIvyの機能もばっちりサポート仲良しさん。
    慣れたところからちょっとずつGradleの記法に置き換えていけます。
    癒し系。
