catch 名誉学長 田坂広志と切磋琢磨しませんか？
description

◆社会起業大学・名誉学長である田坂広志先生の主宰される田坂塾というのがございます。「田坂塾」開塾以来、企業の経営者や組織のリーダーを中心に、現在1000名を超える方々が集まられ、塾長と熟成諸氏が互いの人間成長をめざし、切磋琢磨していく場です。また、ここは、塾長と塾生ひとりひとりとの真剣勝負の場です。

◆田坂塾の塾生である山口校長が、田坂塾での学びを、参加頂ける皆さんと一緒に、田坂広志塾長のお話を考えていきたいと思い、また、田坂広志塾長の言葉や情熱をお伝えしたくて、開講します。まさに、山口校長と一緒に業を積む時間を過ごしてみませんか？
　田坂塾第1回目は、「潜在意識を活かす」をテーマに、21世紀の知性を身につけていきましょう。
全国の田坂広志FUNのみならず、これからのリーダーとして活躍を期待される若き諸侯や企業の経営者や管理者の方、人間成長をともに切磋琢磨出来る方々を対象としています。
　どうぞ、多くの参加をお待ちしています。
