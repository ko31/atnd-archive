過去の開催履歴とこれからの開催予定はこちら

PHPのプログラムの書き方について基礎から勉強します

当勉強会は、"PHP初心者" と “PHP『超』初心者” を分けて行います。

PHPのプログラムの書き方が分らない方は『超』初心者クラスにご参加ください。

PHPのプログラムの書き方を理解された方は、初心者クラスにご参加ください。

詳しくは、こちらのページにまとめてあります。

途中からの参加でも問題ありません

当勉強会は、途中から参加しても理解できる内容で開催していますのでご安心下さい。

前回の内容

  1. 全てのプログラムの基礎となる変数を理解する
  2. 変数を使って計算してみる

今回の内容

  1. 前回の内容の復習
  2. PHPでよく使われる、配列と連想配列について理解する

XAMPPをインストールしてください

講義にはXAMPPというソフトウェアを使います。

これはパソコン上（Windows,Mac）で、Webサーバーを実行するプログラムです。

（基本的にインターネットで使用されるWebサーバーと同一のプログラムです）


Windows版
http://www.apachefriends.org/jp/xampp-windows.html

Mac OS X版
http://www.apachefriends.org/jp/xampp-macosx.html

XAMPPのインストール方法は、以下のURLをご覧下さい。
http://www.apachefriends.org/jp/xampp-windows.html#2619
http://www.apachefriends.org/jp/xampp-macosx.html#2721
