来たる2017年9月1日にソシム社より「無理なく続けられる! 副業でも充分稼げる アフィリエイトのためのブログの書き方講座」が出版されます！

この本の著者は、鈴木利典さん。
（ハンドルネームの鈴木こあらさんと言ったほうが分かりやすいかも知れませんね。）

その鈴木さんが執筆した書籍はどういう内容の書籍なのか？
 
その説明はご本人が書いた紹介記事を読んでいただくとして（少し紹介するならば、ブログでアフィリエイトを行うためのノウハウはもちろん、アフィリエイト界隈で有名な方々の体験談なども寄稿されている読み応えのある本です）、ここでは皆様にお知らせがございます。

2017年9月2日、書籍の出版を祝してパーティーを開催いたします！

開催日時まであまり間がありませんが、皆様、ふるってご参加ください！

日時と開催場所

日時と開催場所は以下の通りです。

申し込み方法と定員

申し込みは以下のATND（このページ）から登録をお願いします。

ATND

但し、参加人数に限りがあり、30名様までとなります。

（正確に言えば、主役の鈴木さんと主催の私を除いた28名です。）

なるべくお早めにお申し込みをお願いいたします。

飲んで騒いで質問をぶつけましょう！

当日はもちろん、パーティーの主役の鈴木さんもいらっしゃいます（主役が来ないパーティーなんてあるわけがないですけど（笑））。

パーティーでは皆様で飲んで騒いで鈴木さんの出版をお祝いするのはもちろん、鈴木さん自身に本の感想や質問、日ごろの鬱憤をぶつけてみるのもよいでしょう。

本人に色々と聞けるチャンスでもありますから、是非書籍を購入してサインをおねだりするのもよいですね。
 

また、パーティーについての質問等があれば、私に遠慮なくご質問ください。

皆様のご参加、心よりお待ちしています！
