『ぷよぷよ！！』大阪オフ会（2011年12月23日）

description

セガ公式イベント「ぷよぷよフェスタ2012/ぷよぷよチャンピオンシップ」の開催決定を記念して、大阪でぷよぷよオフ会を開催します。

シリーズ最新作『ぷよぷよ！！』での対戦を中心に、いろいろ遊びましょう。
真剣に対戦をしたい人も、気楽に交流したい人も、ぷよぷよが好きなら誰でも参加OKです。
途中参加、途中退室可能で、他のゲームで遊ぶのも自由です。

参加費：1500円 （小学生未満は無料）
※ジャンカラのフリータイム料金です。
ジャンカラ阪急東通3号店

持参品：持っていればDS、3DS、PSPの本体と『ぷよぷよ！！』ソフトなど （なくても大丈夫です）
※Wiiを会場のモニターに接続して遊べるので、Wiiの本体とソフトはイノセンス @inosendo が用意します。
※コンセントタップは会場にあるので、充電ケーブルは必要なら各自持参してください。

その他の詳細は未定で、参加者の集り具合を見てから考えます。
ぷよぷよチャンピオンシップに合わせて、ぷよぷよフィーバーでトーナメント形式の大会などもしたいと思っています。
19時に終了した後は二次会として希望者で食事にでもいきましょう。

当イベントに関する問い合わせはTwitterでイノセンス @inosendo までお願いします。
もし二次会だけ参加したい方がいれば、このページでの正式な参加登録ではなく、コメント欄に書くかTwitterで個別に連絡をください。

ちなみに会場は大阪ぷよぷよ通対戦会が行われているアミュージアム茶屋町から徒歩10分くらいの場所です。

主催：ぷよぷよポータル！！ （追記：開催レポート）
