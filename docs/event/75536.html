アニメーション作りで楽しく覚える一日完結型セミナー

脱力系キャラクターを動かしながら、楽しく学べるセミナーです。
プログラムの印象を「とっつきにくい」から「もっと知りたい」に変えるためのカリキュラムを組んでみました。
ワークショップ感覚で気軽にご参加ください。

Webクリエイターの必須スキルとして定着しつつあるJavaScriptですが、
独学するとなるとハードルが高いものです。

「今更、デザイン動かせないなんてクライアントに説明できない；」

「スライダーだけじゃ、フロントエンドエンジニアを名乗れない…」

「溢れ出すイマジネーションにコーディング技術が追いつかない！！！」

・・・そんな声が日々寄せられるアジャストアカデミーでは、
とっつきにくいプログラミングの基本原理を楽しく突破するための
全く新しいカリキュラムを構築しました。

講師は、インターネット黎明期から１７年間プログラマとして活躍し、
職業訓練講師のキャリアも５年を超える“癒やし系”プログラマ安村淳です。

迷えるWebクリエイター救済のため、脱力キャラ「うるめくん」を使って
楽しく学べるJavaScriptセミナーを開催いたします！
