◆はじめに◆
「VOCALOIDは好きですか？」
このページをご覧の皆さんは、この問いにどう答えますか？
好き／嫌い／そうでもない／ミクさんマジ天使……さまざまな答えがあるかと思います。
その「好き」や「嫌い」には程度の差こそあれど、なかにはその「好き」「嫌い」を誰かに伝えたい人もいるかもしれません。
伝える形式はなんだっていいんです。
批評、小説、イラスト、漫画、コスプレ、フィギュア撮影えとせとらえとせとら。
「ともかく、僕の／私の好きなVOCALOIDを伝えたい！！」
そんな想いを、形にしてみませんか？
　
◆概要◆
この企画は「筑波民でVOCALOIDの同人誌を作ろうぜ！」という、ただそれだけの企画です。
初音ミクを筆頭に、いまや社会現象すら生み出しているVOCALOID。
筑波大学にいる人々のなかにももちろん、「VOCALOIDが好きだ！」という人はいるかと思います。
そんななかでも、自分のその想いをなんらかの形で発表できる機会を待っている人だっているかもしれません。
じゃあ、一緒に「VOCALOIDの同人誌（ハッピョウノキカイ）」を作りませんか？
筑波大学にある広範な学問領域と、そこに属する多様な人々の持つ「VOCALOID観」を、いろんな人に伝えて、自分のなかでのそれをさらに深めてみましょう！
　
◆目的◆
・筑波大学でVOCALOIDに関する同人誌を作ること！
・VOCALOID好き同士でつながりをもつこと！
・楽しむこと！

◆参加形態◆
