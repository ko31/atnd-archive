経堂の癒し系もんちゃん@mont_happyといやしい系かおる@kaorun67の「メガネ食堂」、
第2回は経堂のフシギ空間「アクセルイン」で開催します！

国際薬膳師でもあるもんちゃんの「薬膳」の知恵と
かおるがちょっとはまってる「発酵」の知恵をプラスして
旬を生かした身近な食材を使いつつ
ココロにもカラダにもジワジワくるごはんをご用意します♪

+

今年は早々に梅雨入りしましたね。
ジメジメ･むしむし、なんともうっとしいしなんとな～くだるくてスッキリしない…
それはついつい口にしてしまう冷たい飲みもの食べものがカラダを冷やすことで
胃腸の働きや脾臓の機能が低下してむくみ、だるさ、胃もたれ、疲れやすさを引き起こすからなのです。
薬膳メニューでこの「なんとかしたい」体の不快症状を吹き飛ばしましょう！

また、梅雨は食中毒も起こりやすくなります。
体の外から予防する—抗菌･殺菌・除菌—に加えて
体の中に発酵食品をいれて腸内環境を整え、食中毒の原因菌もやっつけましょう！
また発酵食品は放射性物質の排出を促す効果も期待できます。

薬膳と発酵のチカラでもっと元気に！

　+++
