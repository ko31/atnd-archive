■概要

SUPER ILLUSTRATION SCHOOLでは、毎月NUDEデッサン会を開催
してます。参加資格は、SUPER ILLUSTRATION SCHOOLの受講者、
イラストレーター、イラストレーターを目指す方、デザイナー、画家
など、絵に関わる仕事をされている方。

前回は、ムービングの時間をやや長めに取って、ファッションカメラ
マンの本間日呂志さんがモデル撮影をする中、デッサンをするという
コラボレーションを実施しました。

第5回目は、ファッションモデルさんにお願いして、おもいっきりお
しゃれな雰囲気を演出できればと、考えています。

■参加費用

ひとり2000円の参加費用で会場費用やモデル費用を補います。
費用は当日入室時に担当者に収めてください。
（お釣りの無いようにご協力を）

参加者数に限りがありますので、参加できる方には、佐藤より開催場
所のDMをさせていただきます。Twitterの場合はDMをお送りするため
に、必ず佐藤をフォローください。
http://twitter.com/#!/SatoRichman

■禁止事項

告知用や今回のような趣旨以外では基本的に写真撮影はできません。
ブログやTwitterに作品を掲載することは奨励しますが、モデルさんが
特定できるコメントや写真をモデルさんの許可無くブログなどに掲載
しないでください。
スクールのサイトに皆様の写真が掲載されることがありますので、予
めお許しください。
