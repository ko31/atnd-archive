発表内容

Servo (by Tetsuharu OHZEKI a.k.a @saneyuki_s)

現在、Mozillaが開発中の並列・並行性を意識した実験ブラウザエンジン実装Servoについて、開発参加者のひとりから、並列化戦略の概要についての解説となります。（資料）

アクセシビリティオブジェクトが作られない場合（by @Takenspc）

ブラウザーが支援技術に提供しているアクセシビリティオブジェクトは、すべての要素に対して作られているわけではなく、対応するオブジェクトがない要素もあります。どういった場合にオブジェクトが作られないかを、古典的な話から最近の話題まで解説します。（資料）

CSS JIT (by @Constellation)

現在、WebKitにて実装が行われているCSS JITについて、実装者の一人による、セレクタマッチングの最適化手法も踏まえた解説となります。（資料）
