イベント概要

アイ・ファクトリーではWebディレクターを募集中！
フランクな雰囲気でお仕事内容や、会社のことを知っていただける交流会を開催いたします。
Web業界にご興味のある方、インターネット広告にご興味のある方、アイ・ファクトリーにご興味のある方は是非お気軽にご参加ください。

■アイ・ファクトリーってどういう会社？

採用担当者がアイ・ファクトリーについて会社のことや仕事についてご紹介します。

■Webディレクターとしてアイ・ファクトリーでどのように働いているか

アイ・ファクトリーの現場で活躍しているWebディレクターが、 仕事のやりがいや本音をお話しいたします！
こんなこともWebディレクターがやっているのかと発見があるかもしれません！

■現場社員×採用担当×参加者での交流会

Web業界への転職を検討しているけど、なかなか動けていない方、
転職活動をどのようにすればいいかわからない方、 企業がどんな人を求めているのか知りたい方、
そのような皆さまの疑問についてお答えします！
キャリアカウンセラーも参加していますので、お仕事選びや転職のお悩み相談なども承ります。

登壇者プロフィール

三澤　陽介（ミサワ　ヨウスケ）　1986年生まれ　広島県出身

2012年アイ・ファクトリー入社。

大学卒業後、アルバイトとしてWeb制作会社で画像のレタッチ、デバッグ作業、マニュアルの作成等を行う。
もっと大手の案件に携わりたいという想いもありアイ・ファクトリーに入社。
入社後は、Webディレクターとして大型銀行案件のサイト運用をメインで担当し、6年目に至る。

今は銀行案件の他に、クレジットカードサイト、通信事業サイト、医療系サイトも担当。
業務効率化やミスを起こさないような管理等の仕組みを作り、オペレーティブ領域を得意としている。

舟久保　拓哉（フナクボ　タクヤ）　1994年生まれ　埼玉県出身

2018年11月アイ・ファクトリー入社。

大学卒業後、建設会社にて営業を7ヶ月間行うも、以前より関心の強かったWeb業界に興味を持ち、
Webの基礎を学ぶために6ヶ月間のスクールに通う。
スクール期間にKROWLを知り、セミナーやインターンに参加したことで、アイ・ファクトリーに入社を決める。

将来はクライアントの課題に対して、Webを用いた幅広い提案をしていけるWebディレクターを目標に、
プロジェクトの進行管理業務に従事する。

主催：株式会社アイ・ファクトリー

アイ・ファクトリーは博報堂アイ・スタジオグループとして、2011年に設立したデジタル制作×人材サービスの会社です。大手・有名企業のコーポレートサイト、キャンペーンサイト、バナーなどのデジタル制作と、自社の教育システムを活かしたIT/デジタル人材の育成を行っています。また、業界未経験から現場で活躍できるクリエイターをこれまでに多く輩出した実績をもつ会社です。

KROWL（クロール）とは

これからWeb業界でチャレンジしていきたい方を対象に、セミナーや面談、キャリアコンサルティング、ワークショップ等の就業支援を行っています。 Web業界を知っている企業がサポートしているからこそ、就職活動もスムーズに。是非KROWLの就業支援を体験してください。
