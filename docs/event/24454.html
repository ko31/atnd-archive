アドテク勉強会・OpenRTB Mobile

★★★追記 2012/01/20★★★
主催者側の予想をはるかに上回る反響で改めてRTBの注目度の高さに驚いています。
これだけの人数で密な議論を行うことが難しいので、班に別れてワークショップ形式で行わせていただこうと思います。それに伴い上限の人数を増やしました。

また、RTBについてや自社の取組み（うちはこんなすごいDSP作ってるよ、DSPもSSPもXXだから作る気ないよ）、などのLTをお願いしたいと思います。われこそはという方がございましたらtwitterで@ishibashitsまでご連絡いただければと思います。
→LTの応募は締めきりました。
★★★追記ここまで★★★

スマホのADNWは年々進化し、今年は1impression毎に取引を行うRTBが急速に広まりそうな勢いです。

そこで来るmobile RTBに備えて、RTBの実装に広く使われている仕様であるOpenRTBの読書会・討論会を行おうと思います。

OpenRTB
http://code.google.com/p/openrtb/

以下の仕様書について読合せしたいと思います。
Mobile RTB Interface Specification v1.0

読み合わせには日本語に翻訳したファイルがあるのでそちらを使用します。英語が苦手な方も安心してご参加ください。

ひと通り読み合わせした後、議題を主催者側で用意しておくので、参加者全員で議論したいと思います。話し合いたい議題があれば、開始前にお知らせ下さい。
