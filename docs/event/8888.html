description

昨今、スマートフォンは誰もが注目し、
スマートフォンという言葉を見ない日はないくらいの盛り上がりぶりです。
ですが実態は非常にマネタイジングが難しいといわれています。

それはまだユーザー数が少ないから？
技術が日進月歩で開発費がかかるから？
iPhoneアプリは0円～115円ばかりだから？
マーケットが整備されていないから？

まだ立ち上がったばかりの業界なので課題は山積していると思います。

そこで先行して実践されているプレイヤーの方にご講演いただき、
その技術面とビジネス面について勉強し、敵味方関係なく、
ワイガヤやで考えていこうよ、
という集まりをやっていきたいなと思って開催いたします。

勉強会の後は懇親会にて他社の技術者同士の交流を持つ場をご用意致します。
積極的に外部と関わって業界を盛り上げていきたい
という意志をお持ちの方にご参加頂きたいと思っております。

今後、継続的にスマートフォンに関する交流会＆勉強会を
開催してまいりますので
どうぞよろしくおねがいします。
