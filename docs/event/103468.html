イベント概要

「同じフリーランスの方と交流したい」
「フリーランス向けの仕事の情報が欲しい」
「フリーランスになりたいけれど、どんなものなのか詳しく知りたい」
「広告では良い情報ばかり目にするけど、実際の所を聞いてみたい」

そんなフリーランスで働いている方やフリーランスでの仕事を考えている方々向けの交流会を開催します。

フリーランス同士、普段話せないお金の話や、案件ならではの悩みなどぶっちゃけトークをするもよし、
フリーランスになりたいと悩んでいる方は、現役フリーランサーへのお悩み相談も可能です。
弊社で作成しているフリーランスの方向けコミュニティについてもご案内致します。

<img src=“http://www.reedex.co.jp/connpass/hand.jpg” alt=“楽しい交流会”/ width=“50%” height=“50%”>

主な予定内容

  • 自己紹介
  • フリートーク
  • 連絡先交換(名刺交換でも可)

※出入りは自由です。遅れての参加や途中で抜けての帰宅もOKです。

対象者

  • 現役のフリーランスエンジニアの方
  • フリーランスエンジニアになりたい方
  • 転職を考えている方

開催概要

  • 日時：2019/1/26(土) 16:00～18:00（15:30～受付開始となります）
  • 場所：東京都渋谷区南平台町1-10 いちご南平台ビル5階 株式会社リーデックス
    　　　※ビル正面から入っていただき、エレベータで5階までお越しください
  • 主催：株式会社リーデックス: http://www.reedex.co.jp/

注意事項

  • キャンセルする場合はATNDより必ずご一報をお願いいたします。
  • 大人の常識を持って参加をお願いします。
  • 営業活動や泥酔などによる迷惑行為があった場合は、即刻お帰り頂きます。
  • 迷惑行為を起こした方は、今後弊社の勉強会には参加いただけません。
