ビブリオバトル（知的書評合戦）

２年前に開催したビブリオバトルを再度行います。
http://blog.benkyoenkai.org/2013/05/23it.html

ビブリオバトルって何？と言う方はここに詳細に解説されています。
http://www.bibliobattle.jp/

簡単に言うと、1冊5分間で自分の選んだ本がどれだけ面白くてためになるかを発表し合って、最後に全員投票で「チャンプ本」を決めるという集まりです。質疑応答に時間制限がありませんので、技術関連の本は特に面白いです。怖いもの見たさ（？）でご参加ください。

発表者

最大10名とします。10名×10分＋投票20分=2時間とします。初期エントリーは5名です。追加募集中です。管理者宛てにメールして下さい。
hatsanhat＠gmail.com　　　（＠は１バイト文字に変更して下さい）

　※MLへの入り方は次のアドレスに空メールを送って下さい。
“Benkyoenkai+subscribe@googlegroups.com”

現在の発表予定者は次の６名です。
　渡辺幸三さん、杉本啓さん、野村昌由さん、中尾泰治さん、佐野初夫

ルール

1. 発表参加者が読んで面白いと思った本を持って集まる
2. 順番に一人５分間で本を紹介する
3. それぞれの発表の後に参加者全員でその発表に関するディスカッションを行う（Max５分程度）
4. 全ての発表が終了した後に「どの本が一番読みたくなったか？」を基準とした投票を行う
5.最多票を集めたものを『チャンプ本』とする
