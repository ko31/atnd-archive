description

なぜ株式会社Light&Hope代表取締役 高橋尚志は、全く成果の出ないセールスマンから一転し、TOPを取り続けるセールスマンになったのか？

あなたにその秘訣をお伝えします。

このような思いを一つでもお持ちであれば、参加されることをお薦めします。

『達成に満ちた2015年を送りたい』 『将来独立したい』 『自分を変えたい・変わりたい』 『1日でも早く成果を出したい』 『成果を出したいけど、どうすればいいか分からない』

『トップセールスになりたい』
『20代のうちに影響力を与えられる人間になり、突き抜けた成果が欲しい』
『圧倒的に認められ、尊敬され、感謝される自分になりたい』
『チームワークの高い魅力的なチームを創り、仲間を導いてあげたい』

22歳からフルコミッションという世界に飛び込み、数々の苦労を経験した中で培ったこと、成果を出すために必要なことを余すところなくお伝えします。

本セミナーでは、 現場で即座に成果を出す秘訣をあなたにお持ち帰りいただきます。

・なぜ急に成果が出るようになったのか？
・成果が出る人に共通する特徴とは？
・成果を出すために必要な能力とは？
・眠っている力を引き出す秘訣とは？
・爆発的に成果を生み出すマインドとは？
・仲間に応援されるために必要なこととは？
・収入を上げるために今すぐやるべき３つのこととは？

2012、2013年で3社でトップセールスで最優秀賞を獲得しながら
教育教材販売で日本一の実績を作り
一業種一社のビジネス交流組織BNIの立ち上げで殿堂入りを果たす等

関わる組織の結束と一人ひとりのモチベーションを高めるとともに

具体的なスキル、メンタルの向上を図り
組織と個人の成果を引き出してきた株式会社Light&Hope代表取締役高橋尚志が全力で成長のきっかけを伝えます。

高橋 尚志

1987年6月1日横浜生まれ
早稲田大学スポーツ科学部在学中に、就職するよりも自分の可能性を試したいという思いで、起業を志し就活せずに卒業し営業代行などのフルコミッションの営業にチャレンジするも芽が出ない日々が続く中で、2011年6月にSAプログラムを採用してから2012年、2013年と関わる企業数社でトップの成績を残す。

・2012年、2013年(株)Beautiful Days MVP受賞
・2013年 SAプログラム販売実績個人部門日本一獲得。
・2014年 SAプログラム販売実績個人部門日本一獲得。
・2014年 SAプログラム販売実績ＦＣ部門日本一獲得。
