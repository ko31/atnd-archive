2015年2月に２回めの無料公開講座を行います！

HTML5がW3Cから勧告！

HTMLは、1999年のHTML 4.01以来改定がありませんでしたが、10月28日ついに大幅な改定が行われ、新しい規格「HTML5」がW3C勧告されました。静的なWebサイト・Webページからインタラクティブで楽しいWebアプリケーションを作るツールとして、今後期待されています。
それを記念して放送大学に場所をお借りして講演会を開催します。参加は無料です。ぜひお友達とご一緒にご来場ください。

講演の内容と各タイトル

初心者向けにHTML5のできること、HTML5の楽しさをご紹介します。

  1. 10:00-10:20 開会・キーノート「HTML5の基礎知識」（中島）
  2. 10:20-11:00「デモで理解するこれからのHTML5」（山崎）
  3. 11:00-11:30「HTML5が創りだす新たな世界」（成井）
  4. 11:30-11:55「HTML5でアニメーション」・閉会（中島）

講師ご紹介

  • 中島 俊治先生（主催）
    放送大学面接授業「HTML5入門」非常勤講師

  • 山崎 大助先生
    デジタルハリウッド大学大学院・デジタルハリウッドオンラインスクール講師 
    「数々のIT系メディアに登場し、２年連続 Microsoft MVP（Bing Maps Development）に選ばれた業界最前線で活躍するクリエイター」、アジアで唯一（世界９人中の１名）のMicrosoftMVP（BingMapsDevelopment）として、Bing関連のAPIだけでなく、HTML5やWeb関連技術の普及に尽力している。IT系メディアでの寄稿、書籍の執筆など、ヒカ☆ラボなど他のセミナーやイベントでも活動。最近では「HTML5アプリ作ろうぜ！」の連載で人気を集めている。
