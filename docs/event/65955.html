description

唐揚げをもりもり食べながら、VRの未来にときめいている仲間とアツく語りたい——。2月25日に開かれて、120人ものVRクリエイター・ファンを集めて、いろいろ酔いしれたあの興奮から3ヵ月。ついに第2回の開催が決まりました！

会場となる「綱島ラジウム温泉 東京園」は、神奈川県の東急東横線・綱島駅から徒歩1分のところにある天然のラジウム温泉です。創業は昭和21年（1946年）と約70年もの歴史があるこの東京園ですが、な、な、なんと5月17日に閉園が決まっているとのこと（ウェブサイトには載っていませんが現地で確認しました）。

綱島といえば、IT系ではアップルの開発拠点がつくられることで一躍注目を集めたスポットでもあります。そんな歴史が交錯する特異点にてみんなで盛り上がり、その様子をVRで記録しておきたいという思いも込められております。

ちょうど「Oculus Mobile VR Jam 2015」が11日に終了し、「VRCクリエイティブアワード」が17日に締め切りを迎えるというタイミングですので、みなさんぜひ作品を持ち寄ってお互いに見せ合い、貴重なVRコンテンツ開発の情報を交換しあってください。もちろんVRコンテンツを体験したいだけという方でも、東京園大好きという方でもOK。ぜひ奮ってご参加ください!!

追記情報

●開場は17時台になります
当日は17時台から東京園に入場していただけます。宴会場に入れる時間が18時のため、お時間に余裕のある方は先に入場してお風呂に入っていただき宴会中はVRに専念……という流れも可能です。終宴は20時です。

●今回のからあげは「とり多津」
VRまつりというからには、からあげにもこだわりたいところ。厳選の結果、「とり多津」さんにお願いさせていただくことになりました。同店は、日本唐揚協会が開催する「からあげグランプリ」の第三回にて、東日本醤油ダレ部門金賞を受賞したこともある有名店。秘伝というタレの味を存分にご賞味ください。

●THETAをお持ちください
開宴直後にからあげの山を作って、みんなで撮影できればと思います。VRということで、ぜひTHETAなどの360度カメラをお持ちいただき、みなさんで撮影しましょう。

●ワンドリンクのみ含まれます
参加費にはワンドリンクのみ含まれております。なるべく多くの方に来ていただきたいため参加費を低めに抑えつつ、からあげなどにこだわったところ、「やべぇこれ普通に赤字になる(´；ω；｀)」と気付きました……。2杯目よりお買い上げいただけると助かります。

●電源使用は1台につき500円かかります
東京園に確認したところ、電源の利用には1台500円かかるとのこと。大変恐縮ですが、出展される方はご負担いただけると助かります。
