DXRuby Advent Calendar やります→やりました

完走しました！！

参加された皆さまのおかげで無事に最後まで記事を埋めることができました。ありがとうございます。

概要

RubyからDirect Xを使ってゲームを作ることができるライブラリ DXRuby についての記事を書きながら、クリスマスまでの日数をカウントダウンしましょう。

何を書けばいいの？

DXRubyに関することなら何でもオッケーです。
たとえば「DXRubyのこの機能が便利」「DXRubyと一緒に使うと便利なツール」みたいなお役立ち情報もいいとおもいますし、もちろん「DXRubyでゲーム作りました！　みんな遊んでね！」もいい感じです。「DXRubyのソースコードを読む」なんていうのもおもしろいとおもいます。

何でもアリです。ふるってご参加ください。

どうやって参加するの？

  • まず、参加登録します
  • 参加後、日程が割り振られるので確認します（希望日があれば対応できますのでご相談ください）
  • 割り振られた日になったら、ブログやサイトに記事を投稿します
  • 記事のURLをコメント欄に投稿してください

ブログもサイトもないんだよ

GitHub Gist や Qiita みたいなドキュメントを公開できるサービスが便利です。

ひとり一記事？

何記事書いてもらっても結構です。どんどん書いてください。
ぜひ、全日程埋めましょう！
