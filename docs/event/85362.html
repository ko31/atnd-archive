概要

モバイル向けのゲームを作る上で、知っておくと便利なゲームシステム以外の機能（クラウドサービス）を一通り使えるようになるためのワークショップイベントです。

参加費

無料

参加対象者

  • Unityで使うC#が少し理解できる方
  • Unityでモバイル向けゲーム（アプリ）を作っている方
  • これからアプリを作ろうと思っている方
    （中学生以下の方は保護者の同伴をお願いいたします）

  • Unity Collaborate (チーム開発)
  • Unity Cloud Build (ビルド環境)
  • Unity Ads (マネタイズ)
  • Unity Analytics (ゲーム解析)

持ち物

  • Unity （最新版）がインストールされているPC、Mac
  • PC用電源
