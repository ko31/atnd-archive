2014年7月11日、大阪府東大阪市にて、
一般参加可能な講演会があります。
こうぶんこうぞうの講演を聴きた事のない方、聴きたい方は必見！
あまりない一般公開ですので、お時間の許す方はぜひご来場下さい！
ホール併設のギャラリーにて、一日だけの展覧会を開催します。
作品や画集等の販売は一切ございません。

当日、テレビ取材が入るそうで、講演前後、講演会中など
撮影クルーが会場内を行き来致しますが、ご協力をお願い致します。

参加には申し込みが必要ですので、
以下の案内、またはチラシをご確認下さい！

■主 催
東大阪市・東大阪市教育委員会・世界人権宣言東大阪連絡会議
東大阪市人権啓発協議会・東大阪市人権擁護委員会
すすめよう!人権尊重のまちづくり

■日時
2014（平成26）年7月11日（金）
13時（受付開始）14時10分～講演予定。

■会場
東大阪市イコーラムホール
http://www.ikoramu.com/
