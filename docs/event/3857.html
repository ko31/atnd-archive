アイリスキッチンスタジオ(http://airisu-ks.jp)の料理合コンは・・・

料理を通して男女が出会いのきっかけを作る新しいかたちの婚活です。

料理合コンに参加頂いた方は、料理の先生からレッスンを受けながら、レシピに沿った料理を作り、完成後は食事をしながら異性の方との会話を楽しむという通常の料理教室と合コンが合わさった（料理＋合コン＝料理合コン）ひとあじ違ったユニークな体験をすることができるのです。料理合コンでは、参加者全員が料理という共通の話題を持てる為、会話が上手でない方でも安心して異性の方と楽しくお話しができることでしょう。

またこの料理合コンでは、下ごしらえ、調理、盛り付け等の作業を男女共同で行いますので、会話だけではなかなか分からない相手の普段の姿や長所を知ることができます。男女の共同生活の中で、料理は日常的なものです。料理合コンは、自然に相手を知ることができる最高の出会いの場なのです。

料理ができなくても大丈夫！　1人参加でも大丈夫！
