【プログラム】
◆講師の言葉
　歯車の強度設計法は国際規格のISOに提示されているが、素直に従っている会社は少数派である。それどころか、会社によっては5種類、6種類といった多数の設計手法を駆使している。その上、歯車の強度といっても、歯が折れる、歯面がぼろぼろに痛む、焼け付いてしまうといった全く異なる損傷形態があって、それぞれに対する強度設計が要求される。そうした損傷形態も、運転時間とともに変化し、例えば、歯が折れたといっても、その真の原因となる初期の損傷形態が折れた歯からは全く消滅してしまっていることも多い。では、こうした問題を、どのように考え、どのように対策を取ればよいかについて、歯車の基礎から説き起こして問題の解決策までわかりやすく、詳解します。

◆修得知識
　・歯車の強度設計の勘所を把握する。その結果、強度設計の実務可能な状態になり、また現実問題にも対処できる。
　・歯車の強度の考え方、損傷問題の対処法が理解できる 　　

◆プログラム
Ⅰ. なぜ、歯車の強度が問題にされるのか
　　1.設計法で変わる歯車のサイズ　　　　　　　　　　　　　　　
　　2.設計者を悩ませた歯車の強度規格
　　3.不可解な強度試験データ

Ⅱ. 歯車とは何か
　　1.動力伝達機構における歯車の位置づけ　　　　　　　　　
　　2.歯車と歯車機構
　 　 　a.歯車の種類が多いわけ　　　　　　　　　　　　　　　　　　　
　　　　b.歯車装置のマジックショウ
　　3.歯車の生い立ち

Ⅲ. 歯車の基礎
　　1.歯車各部の名称
　　2.歯の形
　　　　a .歯形あれこれ　 　　　　　　　　　　　　　　　　　　　　　　　　
　　　　b .なぜインボリュート歯形か
　　3.歯車の精度がなぜ問題にされるのか
　　4.歯のかみ合い

Ⅳ. ISO 規格による歯車の強度設計
　　1.歯車の強度設計に規格をどう適用すべきか　　　　　　　　
　　2.ISO 規格の形式
　　3.歯車に働く力　　　　　　　　　　　　　　　　　　　　　　　　　　　
　　4.ISO 規格による強度設計

Ⅴ. 歯の曲げ強度の実際
　　1.歯の損傷形態は単純ではない　　　　　　　　　　　　　　　　
　　2.歯元応力の算定
　　3.歯の曲げ強度試験法　　　　　　　　　　　　　　　　　　　　　　
　　4.疲労折損の実験例
　　5.歯の折損形態にも様々ある

Ⅵ. 面圧強度( 接触疲労強度) の実際
　　1.ヘルツ応力という奇怪な応力 　　　　　　　　　　　　　　　　　
　　2.ピッチングが問題の鍵
　　3.マイクロピッチング　　　　　　　　　　　　　　　　　　　　　　　　
　　4.新たな脅威・フレーキング
　　5.スポーリング、ケースクラッシング

Ⅶ. 耐スカッフィング強度の実際
　　1.スカッフィングの発生原因　　　　　　　　　　　　　　　　　　　
　　2.歯面温度とスカッフィング発生確率
