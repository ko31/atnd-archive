プログラムスタートアップ勉強会ハンズオン

～始めようプログラミング～

WebデザイナーやHTMLコーダーの方がプログラムに触れる機会が増えてまいりました。
今回は、特殊なソフトを必要としないJavaScriptを元に「プログラムとは？」と、基本的な内容を抑えたいと思います。
※今回最低開催人数に達さなかった場合は、開催を中止させていただきますことをご了承ください

タイムテーブル

時間 発表者 タイトル
18：30 受付開始
19：00 弊社講師 プログラムの基本を抑えよう

本勉強会では、どのプログラミング言語でも共通して必要となる知識の習得を目指します
※特定の言語がマスターできる勉強会ではありませんのでご注意ください

内容

  1. プログラムとは
  2. 変数ってなに？
  3. 条件分岐
  4. 制御
  5. 配列
  6. 関数
  7. プログラムを覚えてできること（デモ）
