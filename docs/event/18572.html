第１回ジュリサミット開催

　関東ではザンギサミット、アドンサミットなどキャラ別の交流会が開催され、
その流れにジュリも続け！という事で今回のサミットを企画させていただきました。

アーケードでプレイしていない方や初心者の方でも大歓迎なので、ぜひジュリ使いの親交を深めていきましょう。

☆要確認☆
　ここでの参加は、居酒屋で飲み会の席をとるときの人数把握に使います。
　飲み会に参加可能な方は登録よろしくお願いします。

　また、飲み会の人数は２０人で仮予約をしています。キャンセルは前日まで受け付けてもらえるようにしていますので、飲み会に参加できそうな人は、手を挙げておいてもらえると助かります。

　☆現時点で参加の確定を頂いている方☆(敬称略)

　＠ミヤ(グットタイム)、よっさん、洋梨、ツナ
