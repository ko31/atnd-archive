1000000人のキャンドルナイト＠OSAKA CITY2012 Summer

イベント ユーザー

title 1000000人のキャンドルナイト＠OSAKA CITY2012 Summer
catch 茶屋町ナイト
description

大阪では、“街と人のコミュニケーション”を軸に、街に集う人びとや地元民間企業が一体となり、夏至の夜のパーソナルなキャンドルナイトを呼びかけるイベントとして、夏至に先駆けて、６月６日（水）茶屋町ナイト、６月１３日（水）西梅田ナイトを開催。

６月６日（水）茶屋町ナイトでは、会場内の各施設と周辺施設の協力による２０時から２時間のライト・オフを実施。

地元の専門学生・大学生や関西を中心に活躍するクリエイターによるキャンドルを使ったインスタレーション「キャンドル回廊」。
クリエイターと、地元の小学生のおともだちが参加してくれている、日本古来のあかり「行燈」にヒントを得た「ペーパーバッグ行燈展」。
来場者の方が参加して作り上げる「メッセージキャンドル」。

ＮＵｃｈａｙａｍａｃｈｉ３Ｆテラスでは、キャンドルの灯の中での、アコースッティックライブ「Ｔｗｉｎｋｌｅ　Ｌｏｖｅ　Ｌｉｖｅ」を実施。
出演アーティストは、まきちゃんぐ、タイナカ彩智、松井省悟（空中ループ）。

茶屋町ナイトでは、この夏、初めての取り組みとして「ゆかたｄｅキャンドルナイト」－いつもと違う、キャンドルナイトをゆかたで楽しむ－を提案しています。

ベルェベルビューティコミュニケーション専門学校様のご協力で事前予約いただくと、浴衣ご持参のお客様に着付けとヘアセット＆メイクのサービスが受けられたり（※事前予約、先着50名女性限定）、茶屋町バルチケット（１枚）のプレゼントがあったりと、ゆかたで来場いただくと、ちょっとお得な企画もあります。

街なかに一夜限りのキャンドルの森が現れるひとときを、ぜひ体験してください。

詳しくは公式ＨＰをご覧ください。
