description

今年創立102周年を迎える早稲田大学広告研究会は、昨年延べ16万人以上が来場した早稲田祭において「ワカモノ文明開化」と題して、４つのイベントを行います。その１つのイベントである『WAS GAME SHOW』では、週刊「ヤングマガジン」（講談社）にて連載中の人気漫画シリーズ『賭博黙示録カイジ』とのコラボレーションイベントを開催します。

本イベントでは、『賭博黙示録カイジ』の作中に登場するゲームを、より面白く親しみをもっていただけるようアレンジし、ゲストVS来場者約600人の対決をします。ゲストには幅広いジャンルでご活躍されている加藤夏希様をお迎えします。

そして、来場される皆様には普段の生活とは一味違う、異次元のような非日常的世界の中でゲームを楽しんでいただきます。早稲田大学広告研究会が考える、新しいパーティーゲームにおける心理的駆け引きのドキドキ・ワクワク感を実際に感じてほしいという思いから生まれたイベントです。

また、ゲームやアニメなど無類のサブカル好きとして芸能界で定評のある加藤夏希様のトークも必見です。
