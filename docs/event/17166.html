カスタマイズされたWordPressを利用し、オリジナルなサイトを作成します。

title カスタマイズされたWordPressを利用し、オリジナルなサイトを作成します。
catch 【アートセミナー】
description

アーティストの作品紹介 や 大学生の就職活動ツールとしてご利用できます。

情報技術の目覚しい発展により、作家活動や就職現場のニーズも著しく変貌を遂げています。就職活動においても画像や動画を中心とした作品紹介サイトが不可欠となっています。
また一般ビジネスサイトとしてもご利用できるCMSサイトを制作するセミナーです。

講師：後藤孝之（メディアクリエータ）

  • 日本安全学教育研究会会員
  • 株式会社スパイスワークス顧問
  • 知育ラボ　アドバイザー
  • 女子美術大学非常勤講師 など
  • http://career-goto.arttimes.info/

セミナー概要

このセミナーではCMSの利用方法を解説し、当セミナーで用意するサーバー内に、参加者全員が自分自身のサイトを作成し、管理者権限で運営することができます。
サイトの維持・運営に必要なサーバーとサブドメインを、セミナー終了後も利用することができます。（条件有）

  • 受講料：40,000円（一般）
  • 教材費：8,000円
    この教材費には1年間のサーバーとサブドメインの利用料が含まれています。

* 開講日：2011年　8/7(日),8/21(日),8/28(日)の3日間

  • 時間：10:00～17:00（昼食時間含む）

＊お申し込みはこちらでお願いします。

メニュー

日 内容
1日目 WordPressのしくみ、コンテンツの構造（作品紹介サイト）、画像データの色調補正とリサイズについて、
はじめに行うWordPressの基本設定、ヘッダー画像のアップ
2日目 ページ/カテゴリーの設定、記事の投稿/ページの作成、画像をアップロード（レイアウト含む）、
テーマ/プラグイン活用
3日目 基本テーマ・プラグインの操作方法、フォームの作成、具体的な記事を入力など
