catch パクチーハウス３周年（５）
description パクチーハウスは、お子さま連れも大歓迎です。店内を子どもたちが走り回っている日も珍しくありません。日本国内のレストランでは、連れて行きづらいところが結構ありますが、パクチーハウスはすべての世代の方々に来てほしいと思っております。

　店主のKyo paxiが会社を作ろうと思ったのも、子どもが生まれることが分かり、その子と一緒に成長していきたいと思ったから。最初に始めたパクチーハウスという事業は、長男が０歳のときにできました。パクチーハウスの床でハイハイをして、ほとんどすべてのパーティに参加して今、３歳９カ月です。

　さて、６夜連続パーティ５日目のこの日は、いい夫婦（1122）の日だそうです。日曜日と祝日の間なので、ぜひご家族・ご兄弟とお越しください。参加費3000円で、飲み物はキャッシュオンです。午後6時開場。なお、夫婦で来られた方は500円引き（夫婦で5500円）、お子さま（10歳以下）を一人連れてくるたびに500円引きの、ご家族優遇をします。（つまり、夫婦と子ども3人であれば全員で参加費4000円。一緒に来る人数が多ければ多いほど安くなる不思議な仕組みです。）飲み物はキャッシュオンにて承ります。もちろん、おひとりで来られてもOK。この機会にぜひいろいろなお子様と遊んでエネルギーをもらってください。

　子どもも大好きなパクチー麺をメニューとして出す稀有な日でもあります。
手食の素晴らしさを家族で体験していただくため、すべての料理を手で食べるパーティとします。スプーンや箸は用意しません。世界人口の半数近くが行っているというグローバルスタンダード“手食”に、ぜひともチャレンジしてください。
