実装会情報

ZIPの圧縮・解答を実装する勉強会です。
日曜の昼に、5時間ほどかけて、実装・情報交換など、行います。
ZIPを通じて、圧縮のアルゴリズムへの理解を深めようという勉強会です。
開発環境・開発言語は各自での用意をお願いいたします。
当会のC, C++, Java, Common Lispを推奨言語とします。
