「第7回 MongoDB JP 勉強会 in Tokyo」〜今年最後のMongoイベント〜

■ 11月15日（火）に第7回MongoDB勉強会を開催します

MongoDB勉強会も着々と回を重ね第7回目を迎えました。第7回は今年最後のMongoDB勉強会になります。今年最後にふさわしく、海外サービス Server Densityより@davidmytton氏をお招きして講演を行います。今後も#mongotokyoはよりよい場となるための機会をたくさん提供していきたいと思います。

※ また、勉強会の後は会場での懇親会を予定しております。是非ともそちらもご参加下さい。

■Ustream中継とハッシュタグについて

Ustreamによる中継を予定しています。

併せてハッシュタグですが毎回#mongotokyo を使っています。これをツイートに付けて頂いて発言していただければ会場に声が届きます。#mongodb も併せてお使いください。
