WordBench福井 第5回勉強会を2011年7月30日(土)に開催します。前回に引き続き、みんなでPCを持ち込んでのワークショップ形式です。わいわいもくもくWordPressと一緒に遊びましょう :)

プログラム

  • テンプレート階層をカシコク使おう
  • プラグイン作り入門
  • WordPress 3.2の機能紹介＋WordPressお悩み相談

詳細は http://fukui.wordbench.org/archives/119 をご参照ください。

勉強会終了後は「おいしいは、コミュニケーションを加速する」をモットーに、WordBench 福井 もぐもぐ部がプロデュースする懇親会。今回は 『田安鐵工謹製』ブランドのバーベキューグリル を持ち込んでの大BBQ大会です！

定員

10名

参加費

  • 勉強会…500円
  • 懇親会…2000円～2500円を予定（当日かかった実費を割り勘で精算します）
    ※飲みもの・デザートなど持ち込み大歓迎！

懇親会へのご参加はキャンセル待ちとなります ご注意ください

持ちもの

  • ノートPC（OSは問いません）
  • LANケーブル（長さ3mくらいのもの）
    ※PCには事前に以下のソフトをインストールしておいてください
    • アドオン『Firebug』設定済みのFirefox、またはGoogle Chrome
    • UTF-8が編集できるテキストエディタ
      WordPress 日本語 Codexの解説ページ
    • FTPソフト
      ※サーバ環境はWordBench 福井にて準備します
