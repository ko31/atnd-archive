title 【最新政策・業界動向から具体的な求人票の解説まで】太陽光だけでない環境業界転職セミナー[東京・無料]
catch 〜激動する環境ビジネス業界で働くチャンスとリスクを徹底解説〜
description

環境業界への転職に関心のある方に向けて、転職活動をはじめるため、
成功させるためにどうすれば良いかを解説するセミナーを行います。

太陽光やLEDを中心に、成長を続けている環境ビジネス業界。
環境省によると、2020年には120兆円市場に成長すると算出されています。
成長産業であり、社会的な貢献度も高い、この分野への転職に関心を持つ方も増えています。
今回は、『環境業界に関心はあるけど、まだ、具体的には動いていない…。』という方を対象に、
『環境業界』と『転職準備のやり方』の基本を解説致します。


[このような方を対象にしたセミナーです]

『転職に関心はあるけど、時期や希望企業等何も決まっていない…。』
『転職したいけど、どのように進めれば良いかわからない…。』
『転職先の企業で自分のスキルや経験が通用するか不安だ…。』
『環境業界に関心はあるが、安定的に成長する産業なのか気になる…。』
『具体的にどんな求人企業があるの？』

といった声に、環境業界の専任キャリアコンサルタントが徹底的に解説させて頂きます。

[セミナー内容]

1. 120兆円市場に成長する環境ビジネス業界の実際
2. 国内のエネルギー政策と産業支援を理解する
3. マクロでみた環境ビジネス業界
→エネルギー基本計画と、2030年に向けた環境・エネルギー産業の動向や政策を理解し、
　今後の環境ビジネス業界の流れを理解します。
4. ミクロでみた環境ビジネス業界
→太陽光接続保留問題や電力自由化など直近の環境ビジネス業界の動きや動向について解説致します。
5. 人案件・求人内容分析
→現在の求人案件や求人票を基に、
　求人の背景や求人動向の分析や求められる人物・経験・スキルについて解説致します。
