description

本日（ 2月14日 金）にて開催を予定しておりましたが、大雪にて交通機関の混乱が予想されますことから、残念ながら延期とさせて頂きます。

今後の詳細につきまして、後日改めて連絡させて頂きます。何卒、ご容赦のほどお願い申し上げます。

ITを取り巻く環境は、今、大きな変化のただ中にあります。その一方で、経営資源として、ITの戦略的価値は、これまでにも増して高まりつつあります。

この変化は、IT活用の常識を変え、これまでのユーザー企業とSI事業者・ITベンダーの役割や関係をも変えようとしています。

本会議は、こんな問題意識を前提に、IT部門とSIer・ITベンダーの皆さんが、お互いに立場を越えて、議論する場として、開催するものです。

＊ 参加費・無料 ＊

対象

  • これからのIT活用のあり方をよりよいものにしてゆきたいという志を持ち、自らも意見を述べていだだける方。
  • ユーザー企業、ITベンダー、SI事業者に関わりなく、ご参加頂けます。

お願い

  • 当日はメディアの取材が入りますのでご了承ください。
  • 写真撮影などで個人を特定される場合はあらかじめご了承頂きますのでご安心ください。
  • 原則として途中での退出はご遠慮頂きたくお願い申し上げます。
