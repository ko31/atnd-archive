格之進の直営牧場である門崎牧場が、生後約千日もの期間をかけて大切に育てた【いわて門崎丑】。
約10日〜2週間枝肉を吊るしドライエージングにて熟成させ、牛肉の旨さを十分に引き立たせてから
真空パックして冷蔵熟成させるウェットエージングを10日間以上。
一頭買いではなく「一頭育て」ならでは！希少部位の味わいはどこまでも深く。

今回は、すし処さいしょ の お魚のプロ @SushiSaisho さんに
格之進自慢の【生肉】を中心にご堪能して頂こうと企画しました。
冷凍モノは一切なし！通常コースでも、提供するお肉はすべて生で食べられる、厳選黒毛和牛（雌）のいわて門崎丑。
格之進の千葉社長渾身☆スペシャルコースをどうぞご堪能下さいませ♪
