description

毎回、カラオケボックスを借り切ってアニソンカラオケを楽しむイベント「ykaraoke」ですが、今回は都心を飛び出して合宿イベントを実施します。去年に引き続き、2泊3日にての開催！ みなさんの大好きなアニメ・アニソン・声優ネタ／トークを心ゆくまで語り明かしましょう。

10～15名程度を目標にしています。
本企画の内容についてのご質問・ご相談は、 @pen2 までお願いいたします。
