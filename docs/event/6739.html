2010年8月10日(火)19:00〜、SINAP TALK Vol.04を開催します。

SINAP TALKとは、第一線で活躍している方々を講師にお招きし、Webやコミュニケーションデザインについて語り合うSINAPのイベントです。
社外からも聴講者を募り、質疑応答、懇親会ともに、大いに盛り上がっています。

今回は講師に、CSS Niteの主催をされるなどでご活躍中の、株式会社スイッチの鷹野雅弘（たかの・まさひろ）氏をお招きします。
以前CSS Nite 特別版としてご自身でご登壇された際のテーマ、
「プレゼンテーション」にフォーカスしたセッションを行なって頂く予定です。

また「プレゼンテーション」に限らず、鷹野氏に聞いてみたい事を Twitter で募集しています。
ハッシュタグ「#snptk」を付けて「@sinapinc」宛にツイートしてください。
当日会場でご本人にも直接お伺いしたいと思います。

皆様のご参加お待ちしております。

日時：2010年8月10日（火）19:00〜21:00
※講座後に鷹野氏を交えての懇親会を予定しております。

テーマ：「プレゼンテーション」

講演者：鷹野 雅弘（たかの・まさひろ） 氏

会場：株式会社SINAP

受講料：無料
