description

ガラスは建築、自動車、ディスプレイ、あるいはフォトニクスやエレクトロニクスなどの分野で必要不可欠な基盤材料である。これらの分野では、資源・エネルギーおよび炭酸ガス排出の削減、或いは軽量化のために、薄型ガラスが望まれている。しかし、薄いガラスは割れ易いという欠点があり、ガラスの高強度化が強く求められている。本講では、ガラスの破壊の原理を科学的に説明し、高強度化の技術について概説する。

８．まとめ：ガラスの高強度化の設計指針
