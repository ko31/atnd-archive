満員御礼！
【明日9/12（木）19時開催！無料懇親会付き！限定70名様】
90分で身につくスマホウェブ制作術

～UI設計から開発効率アップまで～

◆◆━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

スマホは機種によって挙動が異なり、端末数は全世界に約11,800モデル以上あると
いわれています。そんなスマホ向けサイト制作をする際に

「UI設計はどの点に気をつけたら良いか？」
「デバッグやパフォーマンスをあげるにはどうしたら良いか」
「どういうツールを使えば開発効率があがるか？」

などで迷われた経験はないでしょうか？

本勉強会では、スマートフォン向けのフレームワークを駆使し、Webアプリを
制作してきた高岡氏と㈱まぼろしの西畑氏、木村氏、NTTレゾナントテクノロジー㈱
の森本氏をお招きしてスマホサイト制作の注意点ティップス、役に立つツールなどを
90分間でご紹介します。

もちろん、有識者との懇親会も【無料】でご参加いただけます！！

【限定70名様】となります！

出演者について

■高岡 大介（たかおか・だいすけ）さん

外資系SIerでエンタープライズシステムの開発に従事。その後、独立行政法人研究所へ移りセマンティックWebなどを研究。現在はフリーランスとして、主にモバイル向けWebアプリの研究開発を活動の中心としている。Sencha Touchパーフェクトガイド共著。
AITC運営委員。Japan Sencha User Group共同運営者。

■西畑 一馬（にしはた・かずま）さん　to-R 代表 / ㈱まぼろし フロントエンドエンジニア

JavaScript / Ajaxを利用したWebアプリケーション制作やCSS3 / HTML5を利用したサイト作成を業務で行い、個人でも書籍や雑誌の執筆、Webクリエイター向けの講座 / セミナーなどの活動を行っている。主な著書に『Web制作の現場で使うjQueryデザイン入門』などがある。

■木村 哲朗（きむら・てつろう）さん　㈱まぼろし フロントエンドエンジニア

主にWebサービス・アプリケーションの仕組みや構成についての設計および、HTML/CSS/JavaScript を用いた実装を行う。

■森本 恭平（もりもと・きょうへい）さん　NTTレゾナントテクノロジー㈱　UXデザイナー

HTML5を利用したWebアプリケーションやスマートフォン向けネイティブアプリの開発なども行っている。現在は、スマートフォン向けサービスや開発者向けサービスなどに従事している。
