Force.com開発者も盛り上がってきてるのになぜコミュニティが無いんだ！？ということで、みんなで集まってMeetupしてみましょう…というところから始まって、今回で3回目の開催となりました。

Salesforce.comの提供するCRMの基盤部分を切り出した形で2007年に発表されたForce.comですが、いわゆる十八番であるコーディングレスでの画面自動生成やワークフロー、数式項目などは有名ですが、意外にもApexやVisualforceを利用すればゴリゴリにカスタマイズでき、 コードアーティスト達の活躍の場は多い のです。

「まだForce.comをガチで使ったことはないけど興味はある…」という方の参加も大歓迎です！

また、今回はLTで発表して頂いた方の中から、以下の書籍をプレゼント（3名まで）します！
Advanced Apex Programming
