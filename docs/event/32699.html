東京渋谷 “cocos2d for iPhone” 勉強会(10/17)

cocos2dを覚えよう！初級編②

iPhone対応のゲームエンジン cocos2d の勉強会を行います。
初級編ということで、簡単なサンプルを作成し、機能を学習します。

前回の勉強会の続きとなります。

http://tonosamart.com/blog/cocos2d-for-iphone-2012-09-13/

前回の勉強会に参加してなくても、参加できます！

今回は、キャラクターの動きや
タッチイベントの処理などを中心に
やって行きたいと思います！

対象

cocos2dに興味がある、iPhone開発者（初心者～中級者以上）
前回の勉強会に参加してない人も参加可能です。

ノートパソコンについて

*Mac OS X 10.8 Mountain Lion
*Xcode 4.4.1
*cocos2d for iPhone 2.0

上記インストール済みのノートパソコンをもってくるといいかも！（必須ではありません）
(それ以外の環境でもOKですが、環境の違いにつきましては、原則フォローいたしません。)
