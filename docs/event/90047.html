ゆるゆるJavaScript交流会＃1

ゆるゆるとJavaScriptに関する情報共有や交流を行うイベントです。
JavaScript，HTML，CSSなどがメインですが、この他にもバックエンド、アプリ制作など、自由に楽しく交流していただければと思います。

会場には液晶モニタがありますので、自作アプリやサービスの発表、LightningTalk（以下LT）も大歓迎です！
もちろん、どなたでも登壇していただけます。

フロントエンドのエンジニアやデザイナー、コーダー、ディレクター、マネージャーの皆様はもちろんのこと、勉強を始めたばかりの初心者様など、少しでも興味が湧きましたらぜひご参加ください。
新しい人脈を広げたり、技術やキャリアのお悩みを解消するきっかけになるかもしれません。
ゆるゆるカジュアルな交流会ですので、お一人様もお気軽にご参加ください！

★19:00までに会場への入場をお願いいたします（遅れる場合はご連絡ください）
　入場後の退室は自由です。



▲　プレゼンやLT練習にもどうぞ！

  • エンジニアと交流して成長したい学生さん、求職・転職活動中の方
  • エンジニア、デザイナー、デベロッパー、クリエイターの方
  • JavaScriptを業務や研究等で扱っている企業や団体、大学や研究機関の方
  • JavaScript，HTML，CSSなどフロントエンド開発に関心がある方
