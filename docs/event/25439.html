Kyoto.pmとは

Kyoto.pmは京都で活動するPerlのコミュニティです。作られたばかりですが、これからはテクニカルトークやハッカソンなどを企画しようと思っています。
詳しい経緯などは Kyoto.pmを作成＆開催します！ を参照してください。

日時・会場・参加費

  • 日時 : 2012年3月17日(土) 13:30-18:00(13:00開場)
  • 会場 : 御池ビル8F 株式会社はてな京都本社
  • 参加費 : 社会人500~1000円・学生無料(お菓子代+JPA講師代に使います)。人数によって変動します。もしJPAからの講師派遣が出来なかった場合は、100円にします。

特別ゲストでYappoさんが来ます!

なんとJPA様から支援いただき、App::Ikachanなどの様々なCPANモジュールの作者であり、WEB+DB PRESS での Perl 連載枠の立ち上げなども行なっている Yappoさん が京都まではるばる来てくれることになりました！ありがとうございます！
