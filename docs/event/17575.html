ユースケース駆動開発 Boot Camp

ユースケース駆動開発 Boot Camp（UcDDBC）は、ユースケース駆動開発を実際に手を動かして学ぶ勉強会です。TDD Boot Campと同様に、まずはセミナー形式でユースケース駆動開発について学び、演習としてユースケースシナリオやロバストネス分析・レビューなどを体験します。

札幌市産業振興センター セミナールーム７
住所: 札幌市白石区東札幌５条１丁目１－１（東西線東札幌駅から徒歩１０分）
