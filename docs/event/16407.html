会場

　長崎市立図書館研修室2、3。

　建物内の研修室の場所が少しわかりにくいです。 長崎市立図書館 Nagasaki City Library 図書館について に図がありますが、正面入り口から入場後、1階図書室へ入らずそのまま右へ折れ多目的ホール脇の階段を登ってください。二階の左奥に会場の研修室があります。

　館内飲食禁止との事で食べ物の持ち込みはできませんが、水分補給のためのペットボトル入り飲料の持ち込みは可能です。

備考

　「みんなぎっと」は初級者向けGitハンズオン、第二回目です。先日開催した NAGASAKI-IT hands-on Spring 2011 の続編になります。が、前回の内容を最初にフォローしますし、直接的な続きではありませんから前回への参加が必須ではありません。
　Gitやバージョン管理システムの経験の有無も問いません。むしろこれをきっかけに始めてみてはいかがでしょうか。そういう方にも参加していただけると嬉しいです。
　前回の「ひとりぎっと」の資料が公開されています。興味がある方や前回参加されなかった方は是非どうぞ。
　 ひとりぎっと 〜ひとりで使うバージョン管理システム導入編〜
　 チートシート

　ノートPCなどをお持ちの方は是非持参ください。無線LANアダプタなどがあるとインターネット接続が可能です。
　今回はgitとgit-svnを事前にインストールしていただきたいと思います。Windowsパソコンをお持ち込みの方にはこちらで環境を配布しますが、Mac OS XやLinuxをお使いの方はよろしくお願いします。twitterやSkypeなどでフォローする事もできますので、是非チャレンジしてみてください。
　Mac OS Xではhomebrewでインストールするのがおすすめです。
　Linuxでは……、なんだろう :-)
　それと、Google Chromeもご用意ください。Mac OS Xの方はもちろんSafariでいいですよ。

　見てるだけ、他の人との共有での参加ももちろん可能です。雰囲気だけ味わうのもいいかと思いますよ。

　「暗号技術入門」は東京よりSkypeを使った講演です。主な暗号技術がどう使われているかをストーリー仕立てに紹介します。初心者向けで難しい話はありません、数学的な理論へ深入りもしません、その分軽めの内容で楽しめると思いますよ。

　インターネット接続回線としてWiMAXを用意していますが、ご自分でどうにかできる方にはそうしていただけると助かります。一回線しか用意できないので、あまりに多くなると接続できなかったり極端に遅くなる可能性があります。

　電源タップをお持ちの方も持参していただけると嬉しいです。会場の床に複数のコンセントが埋め込まれていて数としては充分なのですが、あまり使い勝手がよくありません。大きめのアダプタですと刺さらない可能性もあります。

　終了後、この為のというわけではないですが、懇親会を開きたいと思っています。
　参加は自由で、場所や費用なども当日の状況により決めたいと思いますが、できるだけ気楽で安く上がるお店をと考えています。この会と直接関係がないので懇親会だけ参加でもOKです。
　ただし、酒乱の気のある方はご遠慮ください :-)
　この懇親会の前にカフェでお茶するも多分あります。

　懇親会の会場を決定しました。
　前回と同じく「誠」という居酒屋さんで行いたいと思います。住所は 長崎市興善町７−２ です。
　時間は午後6時くらいからを予定しています。懇親会のみ直接お越しくださっても結構ですよ。唐揚げが人気です！

　ご質問などありましたらお気軽にお尋ねください。
