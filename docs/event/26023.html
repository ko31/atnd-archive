楽しく未来を描き、やりたいことを見つけるワークショップを２０代限定で行います！


★自分の未来のことを考えるともやもやする。。
★自分のやりたいことってなんだろう？
★転職などを含めた今後のことを考えているけど、よくわからない。
★３０歳になったときにどんな自分になっていたいか言えない。。
★自分のやりたいことをする自信がない。。

そんな人におススメのワークショップになってます！

人はやることが明確になったとき真価を発揮します。
くすぶったままでは、もったいないです。
自分のやりたいことを見つけてどんどん未来を変えていきましょう！

今回のワークショップでは、
この種のワークショップによくある紙に書き出すのをメインで扱わずに
参加者同士の会話の中で

”自分の制限を外す質問”
（人は無意識に考えに制限をかけています）
”ワクワクする未来を描く質問”
（ネガティブな未来ではなく、ポジティブな質問でこそ理想の未来は明確になります。）
”それらを引き出す聴き手の存在とフィードバック”
（一人で話すよりも誰かに話すと自分の思考はどんどん深まり、新しいものが生まれます）

を通して、
未来を描き、GAPを知る。

そして、

”現状を認めるワーク”

を行います。

それらの流れによって
”自分の中のやりたいことを実現する未来”と”今”がつながり、行動に移しやすくなるので、
やりたいことが見つかるだけでなく、今を変える行動も起こしやすくなります！

人生に変化をおこしたい人におすすめの内容になっております。

黒田洋介
iLassic代表。
大学時代初期まで友達ができず、それを改善するために大学入学後、コミュニケーション関連の本を読みあさり、本の内容を実践することで少しずつ自分を変えていく。
その後、就職活動時代に”コーチング”と出会う。
コーチングの「相手の可能性を信じて、聴く」という考え方に共感し、「このコミュニケーションが広がるともっと素敵な世の中になる」と考え、
コーチングを学び始めると同時に、コーチングを大学生に広げるための学生団体iLassicを設立。
1年間でのべ１０００人以上の大学生にコーチングを伝えるセミナーなどを行う。
また、その過程で「一人ひとりが自分らしく生き、共に生きる社会」を創るという夢を持つ。

卒業後は、「自分らしく生きる社会」を実現するため、教育を変えるという想いで学習塾に就職。
同時に”対話の場”を社外に創るため”夢を語る”ワークショップなどを多数開催する。
のち、独立。
現在は「イノベーションを興す３０歳を輩出する」を実現するために年間60回以上のコーチングなどをベースにしたセミナー、イベントを主催している。

田中良洋
Mr.サプライズ
自分の能力を伸ばすため、IT会社に入社する。
しかし、年を経るにつれて自分のやりたかったことがかすむような違和感を覚える。

社会人３年目にして、これからの人生をどう生きるかを考え、自分の中にあるブレーキに向き合う。
その結果、「遊び心を持って、自由な人生を歩む後押しをする」をモットーに
コーチングやカウンセリングといった手法を、現在学習中。

また、人の想いを伝えるお手伝いとして、スライドショー作成を武器に独立の準備を進めている。
