☆テーブルトークRPGとは？
略称「TRPG」。
サイコロと会話で進める、楽しいゲームがテーブルトークＲＰＧです。
ゲームに参加する人は、自分だけのキャラクターを作り、そのキャラクターとして会話しながらゲームのまとめ役である「ゲームマスター」の用意した物語や試練に立ち向かっていきます。
その場に集った参加者や、参加者の作ったキャラクターによってお話の展開がまったく違うので、同じ筋書きでも遊ぶ人によって異なった展開やストーリーが楽しめることが、なによりの魅力。
創作やゲーム、演じることやじっくり想像することなどが好きな人には、きっと気に入って頂けるはずです。
なお、サイコロは、キャラクターの行動がうまくいったか／いかないかを判定するときに使います。

☆『リアリティショーRPG キルデスビジネス』について
「大金持ちになりたい！」「あいつを●●しちゃいたい」といった強い願い事をもった人や動物、無機物となって地獄のTV番組でバトルを繰り広げるテーブルトークＲＰＧ。
願い事を叶えられるのは、キャラクターたちの中からたった１人だけ！

その座を目指してやることは、「黄金の魂」の持ち主から魂を回収したり、ほかの参加者を蹴落としたり、番組の視聴率を上げるためにサービスシーンを繰り広
げること……！？
プレイヤーVSプレイヤー（PvP）こと妨害要素ありあり、だけど不思議と笑いが止まらない。シンプルなお手軽おもしろさにあふれたゲームです。
