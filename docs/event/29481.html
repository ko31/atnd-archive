インド進出、インドでの起業などインドビジネスに関心が高い方はぜひご参加下さい！

BRICsの一角を担う成長著しいインド市場。しかし他の国と同様にインドへの進出は簡単ではありません。
Gateway to Asiaはこの度、主席アドバイザー廣瀬義一氏とインド支部を担当するAmandeep Taneja氏を迎え、インドビジネス、という共通点のキーワードを持つみなさまの疑問や不安の解消、そして情報交換や交流のための場をご用意致しました。
インド進出にご興味ある方はぜひご参加ください。

【こんな方におすすめ】
□インドに進出を検討している
□既に進出しているがうまく結果を出せていない
□自社の製品・サービスがインドで通用するか知りたい
□日本とインドの両方のビジネスを理解している人に相談をしたい
□インドの商習慣が理解できず、ビジネスパートナーとトラブルが続いて困っている
□インドでマーケティングをしてみたい
□インドに視察に行ってみたいがどこが最適か事前に知りたい
など

【プログラム内容】
講師が一方的に話すセミナーではなく、参加型の座談会方式を行います。
ファシリテーターの司会のもと、みなさんのインドビジネスに関する
疑問点や不安点などをざっくばらんに語って頂き、解消していきます。
