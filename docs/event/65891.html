■参加方法

横浜Apple User Group『AoMacクラブ』の交流会を上記の要領で開催します。
参加希望の方は右カラム“赤地の【このイベントに参加登録する】”をクリックし、
指示に従って参加登録してください。
Facebook でも受付しています。

■Apple User Groupとは？

Apple User Group（アップル ユーザグループ） は、アップルの製品（Mac、iPhone、iPad、iPod等）を
こよなく愛するユーザーの会です。

『 AoMacクラブ 』は横浜北部エリア（主に東急「田園都市線」沿線）居住のメンバーが中心のApple User Groupです。
Macを初めアップル製品や広くIT関連の情報に興味ある方、
興味はあるけど「何にも解らないから教えて〜！」な方、
お住まいが横浜でない方でも、
老若男女、Ladies & Gentlemen, Boys & Girls !!
誰でも参加できるクラブです。
毎月第3土曜日に交流会を開催しています。

アップルの事や広くIT関連の話題をみんなで共有、情報交換して
月に1度大いに盛り上がりましょう！！

http://aomac.blogspot.com/
