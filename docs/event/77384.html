description

BtoB（企業間取引）もECサイトで注文を受け付ける「WEB受注」がトレンドになっています。電話やFAXでの受注体制を脱却しオンラインでの取引に移行しています。
当該セミナーではどうすればECサイト上で成功するのか。売上げアップに軸をおいて有名講師にかんたん分かりやすく解説していただきます。

第一部

ライバルはすでに始めている！？BtoBにおけるEC化。その魅力をたっぷりとご紹介。

・BtoB-ECの圧倒的な市場規模とその特徴を解説
・何故いまBtoBコマースに参入するべきなのか？


株式会社Dai　営業統括マネージャー　鵜飼 智史
国内最大級のメディアを企画/運営していたことから、WEBマーケティングをはじめ、飲食・サービス・教育・介護・流通など幅広い知識と経験を持つ。現在は、BtoBコマースの第一人者として年間100社を超える業務改善や業績アップを支援し、全国規模でのセミナー活動や社外勉強会をこなしBtoBのオンライン化に奮闘している。

第二部

売上アップを続ける ネット販売の施策公開セミナー

EC業界歴17年、最高月間転換率18％を誇るECコンサルタントが『どうすればなるべく早く売上が上がるのか？』『どうやって売れ続けるネット販売サイトを作るのか？』をテーマに、あますところなくお話しします。


株式会社バックキャスティング　代表取締役
株式会社フェイルオーバー　EC運用部
菅原 渉

1999年よりネットショップ業界へ。コンサルタントとして活動しながら全国でセミナーを行い、現状を肌感覚で認識する。大手企業から個人運営ネットショップ、弁護士事務所まで、業界問わず「自社の強みの明確化 → 成約率の向上 →成果」を得意とする。さいたま市産業創造財団認定専門家、ネットショップマスター資格講師、中小機構販路開拓アドバイザーも兼務する。

第三部

運営実績No.1BtoB-ECサイトの決定版「BカートASP」とは？

BカートASPでここまでデキる！BtoBで必要になってくる主要な機能をデモサイトを交えながらご紹介。

BカートASP運営部
BtoBに特化したASP型のカートシステムとして200社以上の導入実績を誇る。BtoBのカート事業としては10年以上の実績あり。
