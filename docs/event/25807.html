ゲームセンターCXを多人数で見るオフ

イベント ユーザー
event_id 25807
title ゲームセンターCXを多人数で見るオフ
catch 課長と同じゲームで先を争って見るオフも併設ｗ
description

複数人でゲームセンターCXを見て楽しんだり、
その放送内容に付随して課長と同じゲームをプレイして課長と先を争って見るオフ会を考えてます。
勿論、フツーにTVとゲーム機持ち込んでゲームをするのもOK！

…と言う、上映会主体のゲーム系オフ会を考えております。

◇発端
　レトロゲーム界隈で話題になってる『ゲームセンターCX』という番組。
まず、この番組を視聴するに当たり、スカイパーフェクトや
対応するケーブル会社と契約し、その上で新たにフジテレビ721+739と
視聴契約を結ぶ必要がある。
　「ゲームセンターCXを見たいんだけど、そこまで契約するのはちょっと…」
と言う人、結構いらっしゃるんじゃないでしょうか。（オイラもｗ）

　又、通常はTVで見るのが普通で１人で見ている事に対し、
色々な人と一緒に見て楽しみたいよね～と言う声もあると思います。

　で…そのゲームセンターCXのDVDも発売されてる事だし、
それなら上映会してみようかな？と思って企画しております。

そんな訳で、どっちかと言うと「ゲームはしこたま遊んでるけど、
ゲームセンターCXは名前だけしか聞いた事が…」と言う方向けの
企画にしてます。勿論、ゲームセンターCXを熟知している方の参加も
歓迎しますよ！（但しネタバレは勘弁してね…ｗ）

◆日時・場所・参加費

◆今回上映する予定の内容

どなたかがDVD-BOXをお持ち頂けるのでしたら、

・『DVD-BOX5』
　Vol.9 ：収録時間232分
　　（ディレクターズカット版超魔界村 / 東海道五十三次 / カイの冒険 完全版）
　Vol.10：収録時間179分
　　（ロックマン / スーパーファンタジーゾーン）

を上映したいと思います。 で、DVDBOXが間に合わない様子ならば、

・レンタル版1.0：収録時間111分
　　　（プリンス・オブ・ペルシャ / 魔界村）
・レンタル版3.0：収録時間138分
　　　（ディレクターズカット版忍者龍剣伝 / ロックマン2 Dr.ワイリーの謎 / 魂斗羅）
・レンタル版9.0：収録時間130分
　　　（カイの冒険 完全版）

の3本を上映したいと考えてます。
