◆今回のテーマ◆
『お金があれば幸せ？
　お釈迦様の教える、
　本当のリア充になれる秘訣』

明るく楽しい毎日にしたいと思っているのはみんな同じです。
でも、そうなっている人と、なっていない人がいるのはなぜでしょう？

「お金さえあれば」
「物さえあれば」
「有名になれたら」
「地位が得られれば」
「家を持てたら」
「恋人が欲しい」
その願いが叶えば、本当に幸せになれるのでしょうか？

多くの人が見落としがちな、本当のリア充になれるポイントをお釈迦様に学びます。

世界の偉人ブッダの教えから、大切な人生観を学びましょう。

◆参加者の感想◆

～ここに来ないと聞けない話がたくさん～
ここに来ないと聞けない話が盛り沢山でした。
”本当の私”が何なのか。もっと知りたくなりました。
自分の心に振り回されて生きているなと思いました。
これから勉強させて頂きます。(30代・女性)

～自分を知ることで幸せになれる～
自分を知ることで、自分が幸せになれるということを改めて
気付かされました。仏教に対してはほとんど、無知でありますが、
これから勉強しようと思います。（50代・男性）

～自分のことなのに、わかっていなかった～
今まで自分のことは自分が一番よく分かってると 思っていましたが・・
私のことなのに、よく分かっていなかったことが分かりました。
初参加でしたが、仏教の言葉を私にも分かる言葉で話されるので、
とてもよく分かりました。（20代・女性）
