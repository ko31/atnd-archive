description

【講座のポイント】
組織のビジョンをベースとし、マーケティング技法を駆使した戦略策定の方法を修得します。　
安易なＳＷＯＴ分析では無く、本質的な活用方法「自社・顧客・市場・競争分析」から戦略を策定する体系的なプロセス」を演習中心で修得します。

【習得できる知識】
１．マーケティングの基礎ノウハウが活用できる
２．自社のサービス・お客様・敵の「見える化」ができる
３．自社ソリューションのトレンド分析ができる
４．お客様の価値に訴求するマーケティングが理解できる
５．業界の環境・競争分析ができる
６．戦略と実施計画が策定できる

【プログラム】

１．自社のミッション・ビジョンの確認
２．３Ｃ（４Ｃ）分析　自社何？敵は誰？
３．商品（ソリューション）分析
４．トレンド分析・ライフサイクル
５．４Ｐから４Ｖ
６．顧客分析
７．業界を取り巻く環境分析
８．競争分析
９．弱み問題点分析
１０．ＳＷＯＴ分析
１１．戦略策定
１２．実施計画とリスク分析
