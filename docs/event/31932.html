description

今年2012年6月20日、国内のレンタルサーバ業者の保有するサービス提供用のサーバに障害が発生しました。
障害の原因は、運用作業に関連する過失により発生しました。

その障害以前にも世界中の多くの業者で、事故は発生していましたが、今回の事故は大手一般メディアにも取り上げられるほど注目される事故でした。
世の中の様々なシステムがクラウド化され、社外に自社の情報を持つことが一般化しています。
また、Webサーバやメールシステムは以前より社外のサービスを利用することはインターネットが普及するころから普通に行われています。

今回の事故では、多くの企業・団体で情報が紛失し、復元できない事態となりました。
また、その事故に関連して復旧作業の段階で二次的な事故も発生しました。

事故後、第三者で構成される事故の調査委員会により調査報告書が作成され、個人情報などを含まない「要約版」が公開されました。

同じような事故を起こさないために
同じような事故を起こした際の対応のために
同じような事故による被害を被らないために
当方はこの調査報告書を読み、皆で気付きを共有することを考えました。

参加ご希望の方は下記サイトに掲載されているリンクから、
第三者調査委員会「 調査報告書 （ 最終報告書 ） ＜ 要約版 ＞ 」
をダウンロードいただき、印刷するかpdfをPCやタブレットなどで閲覧できるようにして
ご参加ください。
ファーストサーバ社プレスリリース「第三者調査委員会による調査報告書の受領について」
