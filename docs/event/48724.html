description

「新規開拓したいけど、口ベタで営業は苦手…」
「営業の成績が悪い…」
「新規営業はやってるけどストレスが多くて疲れる…」

将来への不安を払拭するために、何とか条件の良い新規受注を確保したい！

と誰もが願うところですが、はっきり言って従来型の営業方法では
「怖い」「辛い」「疲れる」の三拍子で、どうしても積極的に行動できません。

そんな悩みを解決するため生みだしたのが、言霊名刺です。

言霊名刺を発明した中野さんは、
以前は自分から見込み客にアプローチすることに必死でしたが、
現在では言霊名刺を配るだけで、相手に興味を持っていただき、
先方から連絡が来るという有り得ない状況が実現しました。

言霊名刺を手にする前も当然名刺を使っていました。

しかし、まったく結果が出なかったのです。

名刺とは、

・驚異の精読率100%を誇る唯一の営業、販促ツール
・ワンチャンスの第一印象を決めるツール
・名刺を覚えてもらうツール
・コミュニケーションのきっかけになるツール
・仕事の獲得ができるツール

です。

しかし従来型の名刺では、もう仕事は取れないのです！

・形状ばかりが目立つ名刺→アピールしたいのは名刺ではない！
・変形名刺→名刺入れに入らない名刺は捨てられやすい！
・情報が少なすぎる名刺→興味があれば、情報は求められる！
・透明名刺→裏面を有効に使えない！
・英語表記の名刺→読むのが面倒。迷惑！
・イラスト名刺→写真が最良。自己満足はNG！
・売り込み名刺→追えば追うほど逃げられます！

「言霊名刺」を活用すると仕事が激増します。
実際に今も名刺のみで新規の仕事を受注し続けています。

「言霊名刺」は単なる自己紹介ツールではありません。

心理学の要素とデザインスキルを組み合わせて作った、
仕事や人脈を確保するための強力な営業ツールです。

口ベタ・人見知りで上手くアピールできない方や、
喋りすぎて商談が失敗する方も、
配るだけでPR活動が完了する楽々営業ツールです。

後はしゃべる名刺「言霊名刺」が大事な事を伝えてくれます。
そもそも優秀な営業マンは、決して売り込みや営業はしません。

あなたが口ベタだったとしても、
決して無理してしゃべる必要ないのです。

この言霊名刺の作り方を1から完全に勉強したい場合には、
ぜひ当日「言霊名刺の作り方セミナー」へお越しください。
