catch 花より団子！ジロジロ桜を見て、飲み食いしよう。
description

◼最終
とりあえず、本日は1330に横浜駅、みなみ西口、タネマキから一番近い改札に集合でお願いします。一品などは買わなくてOKです。買った人ごめんなさい！

天候不順のため、時間を変更します。
11時改め、暫定的に1330横浜にします。明日、再度連絡いたします！

どうやら天気がかなり悪いらしいので、タネマキ、もしくは横浜駅周辺の施設に変更になるかもしれません。日曜日朝くらいに、様子を見つつ再度アナウンスいたします。たぶん室内です…。

タネマキ 主催で花見やります！雨天中止！晴れろー！散るなー！
ということで、花見やりましょう。 花見は諦めて、桜の木を眺める会になりました。

桜を観る以外のイベントはコレといって考えておりませんが、飲んだり食べたりしましょう。友だちやら家族やら、ぜひお気軽に誘ってきてねー！

  • 集合場所：三ツ沢公園 第3レストハウス前
  • 花見場所：その周辺
  • 遅れてくる場合：電話ください。参加者のみ、ここで見えるはず。たぶん。
  • 参加費：1000円くらい＋1人1品持ち寄り（忘れんなもぐー！）
  • 酒、軽くおつまみを準備予定（1000円分くらい）
  • 1人1品持ち寄り
  • 出入り自由
  • 現地集合
  • 雨天中止。その場合、タネマキに場所変更。

タネマキから徒歩20分ちょっとですので、一緒に行く人は、一緒に行きましょう！

横浜駅からバスとかも出てます。

  • 地下鉄三ツ沢上町駅下車徒歩15分
  • 横浜駅西口より市営バス3、6、7番乗り場より34、44、50、87系統バスなど「三ツ沢総合グランド入口」下車

最後に、買い出し手伝ってくれる人、募集中だよ！ビニールシートも足りないだろうから、ある人は持ってきてちょ。遊び道具などもあると、モグ雄喜ぶよ！
