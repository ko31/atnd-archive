イベント主旨

今回は「リソース管理」というテーマで「プロジェクトテーマパーク」「ポイズン」もしくは「ニムト」（予定）というゲームを実施します。

「プロジェクトテーマパーク」はBacklogで有名な株式会社ヌーラボさんが作った協力型ボードゲームで、メンバーの能力やモチベーションを把握しながらプロジェクトを進め、遊園地を完成させるゲームです。

「ポイズン」と「ニムト」はバースト（規定枚数を超えたペナルティ）したカードをなるべく引き取らないように、配られたカードの中から最適なものを場に出していくゲームです。

主催について

いまオロでもっともアツい部活動のひとつ、ボードゲーム部によるイベントです！
オロのボードゲーム部では、毎回10名前後のメンバーでボードゲームを楽しんでいますが「ボードゲームって、ほぼそのまま仕事に活かせるよね」「どうせならもっといろんな人とゲームしたいよね」ということで、今回はその部活を外部解放します！
メンバーは全員気さくなので、すぐに打ち解けること間違いなし。
ボードゲームをやったことのない初心者でも、きっと病みつきになるはず。
ご参加お待ちしています！

こんなかたは歓迎です
  • ボードゲームに興味がある
  • むしろボードゲームが大好きだ
  • ボードゲームは知らないけどオロのことは知ってる
  • ボードゲームもオロも知らないが楽しいことが好き
  • ちょうどその日は暇だ
  • 学びたい気持ちがないわけではないが、主に遊びたい
