【プログラム】
◆講師の言葉
　近接場光学はナノ物体や構造体の近傍で繰り広げられる興味深い光学現象を扱う学問分野です。近接場光学の基礎を習得すると、通常の光学が抱える回折限界の壁を乗り越え、光をナノの世界で操る新しい技術が見えてきます。本セミナーでは、特にイメージング、センシングへの応用を想定し、近接場光学の基礎的な概念を丁寧に説明します。また、時間の許す限り、最新のトピックスもご紹介したいと考えております。
　有意義なセミナーとするため、受講者の方々には事前アンケートにご協力いただき、予備知識やバックグラウンドに応じて、セミナーの時間配分を検討します。また、個別の質問にも時間をかけて対応させていただく予定です。

◆修得知識
　・近接場光学の基本概念
　・近接場光学顕微鏡とその応用
　・近接場光学を利用した分光計測、センシング、イメージング技術

◆プログラム
Ⅰ. 光と物質の相互作用に関する基礎知識の復習
　　1. 電磁波・複素誘電率
　　2. 物質のモデル化
　　3. バンド構造と光学スペクトル

Ⅱ. 近接場光学技術の基礎
　　1. 近接場光学とは
　　2. 微小開口の光学
　　3. 近接場光学顕微鏡
　　4. プラズモニクス

Ⅲ. 近接場光学のイメージング技術への応用
　　1. ナノ偏光イメージング
　　2. ナノ蛍光イメージング
　　3. ナノラマンイメージング
　　4. 特殊環境下でのイメージング

Ⅳ. 近接場光学のセンシング技術への応用
　　1. 表面プラズモンを用いたセンシング
　　2. 金属ナノ粒子を用いたナノセンシング
　　3. 蛍光相関分光を用いたナノセンシング
