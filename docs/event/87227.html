大学生・編集者＆ライター募集　説明会のご案内

　当社が発行する出版物（書籍、雑誌、パンフレットなどの制作物）の業務をサポートしていただける大学生を求めています。実務体験を通じて、プロに近づけるために現場でスタッフが丁寧にレクチャー致します。出版業界、編集業務を志す現役学生が対象で、未経験者を歓迎致します。アルバイト、パート、インターンシップ生など幅広く募集しております。ぜひお気軽にご参加ください。

＜募集対象＞
一、大学生（未経験者可）
　
＜業務内容＞
一、取材、原稿書き、編集業務サポート
一、校正、校閲
一、その他、出版業務全般のアシスト

＜時間＞
９時〜17時　※時間は応相談（自宅での業務も可、現場への直帰直行も可）

※取材の案件による謝礼制あり

＜選考について＞
①説明会に参加　②履歴書提出　③面談　④筆記試験
