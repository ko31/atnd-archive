概要

飯塚から世界へ！をキーワードにソフトウェア技術者たちが集結し、テクノロジーについてディープに語り合う場、e-ZUKA Tech Night。

さて、Vol.7は”mruby Night”を予定してます。
九工大の田中和明先生や広島でmrubyを使ったソリューションを展開されてる株式会社アイ・エル・シー熊野さんにお話していただきます。

そしてなんと！e-ZUKA Tech Night Vol.7 の参加者に限り、熊野さんに紹介していただく mruby学習キット (定価￥17,640)を、特別価格¥15,000で購入することができます！

また、LT希望の方は可能な限り こちら からご連絡下さい。
イベントの告知でも、スライド無しでももちろんOK。
