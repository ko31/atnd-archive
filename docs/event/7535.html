世界最先端企業、Microsoftから学ぶ　企業法務・知的財産セミナー 　～就活へ向けてのFirst step.～
supported by　Microsoft

★就職活動を始める前に、職種の理解は必要不可欠です。
・『知的財産』は文系・理系どちらの分野でしょうか？　・『企業法務』とはどのような仕事をしているのでしょうか？
・IT業界でトップを走り続けるマイクロソフトの知的財産戦略とは？　・また、これらの分野にはどのような人材が求められるのでしょうか？
これらの疑問にマイクロソフト株式会社の現役の執行役　法務・政策企画統括本部長である伊藤ゆみ子氏が答えます。

●今回のセミナーのポイント●
　①就活に向けて、企業法務・知的財産分野の職種内容をしっかり理解できる！
　②世界的企業であるマイクロソフトの現役執行役の話が聞ける！
　③名刺作成やチラシ作成に便利なDreamSparkのDVDがタダでもらえる！

学校の授業ではなかなか実際の仕事内容や、本質の部分を感じることが出来ませんが、これからの時代において
最重要分野である『企業法務』・『知的財産分野』について、１日で理解を深めることが出来ます。
また、企業内弁護士に興味がある方にもおススメの内容になっています。
１回生や２回生の方々も是非ご参加ください。

※参加条件は学部・学年、また大学院等は問わず、『学生』であることです。参加費はもちろん無料。
　マイクロソフトという企業に興味がある、という学生の参加も大歓迎です。

【講師紹介】　伊藤ゆみ子　マイクロソフト株式会社 執行役 法務・政策企画統括本部長 弁護士
経歴：東京大学法学部卒業。1989年弁護士登録（現在、第一東京弁護士会所属）。大阪、東京の法律事務所
で弁護士業務に携わった後、GE横河メディカルシステム株式会社、日本アイ・ ビー・エム株式会社での
インハウス・カウンセルとしての勤務を経て、2007年 3月より現職。早稲田大学非常勤講師、
青山学院大学非常勤講師、在日米国商工会議所競争政策タスクフォース副座長なども務める。
アメリカ合衆国ワシントン大学ロースクール法学修士（LL.M）。 アメリカ合衆国ニューヨーク州弁護士。
