「クリエイターのための成功哲学と使える心理学」

～これだけはおさえたい「失敗法則」と「成功哲学」、使える「心理学」～

https://sfida-workshop5.peatix.com/

クリエイター交流会【クリのみ】が主催する、
デザイナー、エンジニア、フォトグラファー、ライター、 イラストレーター、プランナー、制作などものづくりをするクリエイターのためのクリエイター勉強会（第5弾）です。

◆目的
クリエイターが無駄な失敗や損をしないために何が大切なのか、
成功や幸せを手に入れるために何が必要か学ぶ勉強会です。

◆セミナー内容
私たちクリエイターは、日々自分の専門分野の技術、知識の研鑽を行っていますが、
クリエイターとしての未来や年を重ねたときのこと、将来のポジションについては中々学ぶ機会がありません。

例えば、今の職場・職種での成功、将来の転職、フリーランスや起業などが人生に訪れますが、
そのときにどういう風に考え、行動すればよいのか、学ぶ機会が極端に少ないのが実際のところです。

哲学というととても難しいイメージがありますが（実際は突き詰めると難しいのですが…）、私たちは哲学者ではありませんので、クリエイターにとって必要な、重要な部分だけ知っておけば良いので、難しくはありません。

また、同様に大切なものに失敗の法則があります。
失敗の種類はそれほど多くないのですが、みな同じ轍を踏み消えていきます。
傾向と対策を知っておけばいいだけです。難しくありません。

私たちクリエイターにとって大切なのは技術、知識、経験ですが、クリエイターの人生で考えると、それ以外の知識が必要で、専門スキルと人生スキルの総合点で将来が大きく変わってきます。
例えば、スティーブ・ジョブズは創業から辞めるまで（初期）、その後アップルに返り咲いて（後期）、世に新しいモノをリリースしましたが、初期は厄介者として追い出され、後期は伝説的貢献者となりました。
それはなぜなのでしょう。

失敗しないための、失敗法則、成功哲学、心理学をクリエイターのみんなで学びましょう！

◆こんな人にオススメ
・今よりもっとスキルアップがしたいと思っている人
・できれば成功したいなと思っている人
・夢や目標がある人
・失敗したくない人
・漠然とした将来への不安がある人
・最近ストレスを感じている人
・将来、転職、独立、フリーランス、起業をしたい人

◆期待できること
漠然とした不安が減って、今やるべきことが明確になりやすくなる。
転ばぬ先の杖、失敗の法則を知ってしなくても良い失敗を避けることができる。
同僚や上司に聞けない、仕事や将来の不安や疑問を気軽に質問でき る。
