description

もしあなたが自分らしく生きていけたら、
どんな生活を送っていると思いますか？

—————————————————————————
・自発的に動いて活躍している生活。

・自分の気持ちに素直になれている生活。

・その場しのぎをしない、仕事をしている自分。

・他人の感情に煽られない生活。

・自分で状況を冷静に判断して仕事している生活。

・自分らしさを理解して思いっきり発揮している生活。

—————————————————————————

私もそんな生活ができたらなぁって、

30歳を目の前にした29歳の時に思っていました。

しかし目の前にある現実は、、、、

—————————————————————————

・個人の能力に依存した業務で、休みたくても休めない職場

・指示の出し方が高圧的で、全てが命令に聞こえる上司

・気持ちが萎縮し、ミスを連発してしまう日々

・上司や年下の先輩社員に毎日怒られて、自信を失い、自分を責めてしまう日々

・ミスを改善できず、「足を引っ張るやつ」というレッテルを貼られ、
孤立感を感じる職場

・働き盛りの30歳を目前とした自分が、

　「会社を辞めたい」と思うなんて甘いんじゃないかという自分の声

・仕事のことばかり考え、心身ともに休めず、憂鬱になる休日

・転職活動をしても、紹介される案件は見込残業80時間込みのハードな仕事ばかり。

・転職先の総合的な職場環境が
　現状よりもっと悪くなるのではないか？
　もしかしたらこの状況がまたくるかも、、、という葛藤と不安

・この状況を手放したい、、、でもどうしたら、、、という不安
—————————————————————————

この状況は私が29歳の時に感じていた想いや
体験していた状況です。

この時、私は行き場のない想いを抱えて
自分に無理をさせている
非常に苦しい毎日を送っていました、、、。

毎日朝起きた時に、
会社に行こうか、行かないかを
必死で考えていました。
そして、、会社に行けなくなる日も出てきました。
そんな出口が見えない辛い状況が5ヶ月ほど続き、
耐えられなくなった私は、

現実を変えようと思い切って上司に退職の意思を伝えました。

しかし耳に入ってくる言葉は・・・

「こんなことで逃げていたら、どこに行っても通用しないよ」という言葉。

その言葉を聞いた私は、上司にこの辛さを理解されないことに絶望し、

「やっぱり自分の思いは甘かったんだ」と自責の念に駆られ

悲しみを抱きながら、退職の意思を撤回しました

でも・・・当然のことながら何一つ状況は変わりません。

いつものように神経をすり減らし、心も身体もボロボロになりながら働く日々…

２か月ほど経過したある日の帰り道、

自宅近くの書店に足を運んだ私は、

一冊の本をなんとなく手に取りました。

『「好きなこと」だけして生きていく〜我慢が人生を閉じ込める〜』

（心屋仁之助著、PHP研究所）という本です。

この本との出会いが、私のその後の生活を

大きく変えることとなるとは思いもしませんでした。

読み進めていると、無理をしていると自覚している自分と、
それを分かっていても無理を続けている自分がいることを痛感しました。

そして、辛い思いをしている今現在の自分の状況を前向きな気持ちで受け止められるようになりました。

どんどん力がみなぎってくるのを感じました。

無理している事が馬鹿らしいと思いました。

そして、本当に”自分がなりたい自分”を目指そうと思えました！

好きなことをベースにビジネス構築をして
人の役に立ち、「上司にも、同僚にも誰にも文句を言わせない」生き方をしようと思ったのです。

私の中から力が湧いてくることに気づいてから、

無理していることを止めようと、下記のことを即実行しました。

・有給休暇を使用し（体調が悪いなどと言って）、会社を休むようになりました。

・次の職場が決まってもないのに、再び退職の意思が固いことを上司に伝え、嫌だった会社を辞めました。

・ずっと家に閉じこもり、仕事について考えているのをやめてサイクリングをし始めました。

・好きなことをして、生きて行くヒントを得るためにセミナーに通い始めました。

・読書会に参加するようになりました。

・自分の好きな読書をベースに、読書会を始めました。

・自分の経験を誰かに伝えて役にたててもらおうとセミナー開催やブログ、メルマガを始めました。

などなど、徐々に好きなことができるようになりました。
我慢して無理をしている自分を手放し、好きなことをしている
自分に変化することができたのです。

こんな私の経験から、かつての私のように、

・「今、職場で無理をしている。」

・「休日も仕事のことばかり考えてしまう、、」

・「仕事内容にやりがいを感じられない、、でも転職活動もうまくいかない、、」

・「職場が辛い、、、でも転職活動もうまくいかない、、、」

・「会社を辞めると決めつつある、だけど次にやること、次にやりたいことがない」

という悩みを抱えていらっしゃる方に自身の体験をシェアしたいと思い、

このカフェ会を開催することにしました。

「会社を辞めたい」という気持ちを前向きな力に変えたいという方にとっては

とても役立つカフェ会であると自負しています。

ぜひ、「会社を辞めたい、、でも転職もできない、、、」という悩みを

楽しい希望のある話にしたいと思います！

気楽に楽しむスタンスでご参加ください。
