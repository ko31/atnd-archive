description

文章を上達させるヒントが欲しい方をはじめ、これからブログをはじめようと考えている方や書籍化を目指している方などを対象に「はじめてのWebブログ文章教室」をこのたび初開催します。

開催場所は、梅田スカイビルにほど近い、コワーキングスペース「THE(COMMON)PLACE」。

キャリア20年以上の編集ライターが、新人ライターに語っていることをそのままお話ししますので、かなり実践的な内容も含みますが、気楽な「Webライティング入門」といった趣きの気楽な教室になります。

Webだけでなく、当日は出版業界のお話もいたしますので、ライター志望や書籍化を目指す方に役立つ内容も盛り込みます。

【予定プログラム】

□書きはじめる前の準備
□ネタ探しのコツ、継続するコツ
□文章を書くときのノウハウやチェックポイント
　サンプル原稿を推敲していき、完成原稿を目指します
□ブログに載せるちょっとカッコイイ写真の撮り方
□Webライターなら誰でもなれる
□雑誌・書籍ライターのなりかた
□書くならば、いっそ書籍化を目指しちゃおう
□実践編に向けての原稿課題のお知らせ
□希望者のみ、課題内容の原稿を本格的にチェックします（+1000円）
