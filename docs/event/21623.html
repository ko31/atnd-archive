description

すらすらと読める文章を書いて、ちゃんと言葉で伝えたい。

その想いは、きっとみんな同じです。

正しい文章を書くための講座や、集客やＳＥＯ対策などの文章講座が、
今は、どうやらたくさんあるようですが、
私のセミナーは『わかりやすい文章』の書き方をお伝えするのが特徴です。

正しいだけでは、個性が失われますし、
自由におもしろく書けなくなってしまいますよね。
とはいえ、文法的にあまりにも間違っていると
やはり、とても読みにくいものなのです。

そこで、何に気をつければ読みやすいのか
文法の基礎をふまえつつ、今すぐできる工夫やコツをお伝えする、
文章講座を行っています。

これまで行ってきた、文章セミナーとは別に、
楽しくゲーム感覚で文章が上手になるための
オリジナルメソッドを考案しました。

文法を一からやり直すのは大変！もっと簡単にコツが知りたい。
かといって、一度の講義でお伝えできることには限界があります。


この講座では、実践ワークで
楽しみながら文章力アップすることを目指します。
頭のなかを柔軟にするためのワークですので
文章だけではなく、発想力アップにも役立ちます。

１）読みにくい文章を推敲するワーク

２）主語と述語の関係をチェックするワーク

３）魅力的なタイトルをつけるワーク

４）俯瞰力をつける言葉の連想ワーク

５）表現力アップのための五感ワーク
