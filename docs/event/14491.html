【講座趣旨】
次世代のWeb標準技術HTML5は「Google I/O　2009」で、HTML5の強力支持発表から一気注目を浴びるようになっており、今ではWebの世界での流行語といっても良いほどの過熱ぶりを見 せています。HTML5では、PCでのWeb表示という枠を超え、スマートフォン、WebTVなどでの標準プラットフォームになることも予想されます。
