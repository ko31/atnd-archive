catch ～バリ取り・エッジ仕上げの基礎技術～
description

【趣旨】
　高精度微細部品において、設計技術者が要求する「エッジ機能」を満足させる｢エッジ品質｣を得るには、設計図面上に慣習的になっている「バリなきこと」という曖昧な表示をやめ、JISに基づく「エッジ品質」表示に切り替えなければならない。この講演では、JISによるエッジ品質、バリの生成メカニズムから抑制技術に至る基本的技術について解説する。

【プログラム】
１．バリの残留がもたらす製品のトラブル

２．バリ生成のメカニズム

３．製品性能に応じた表面機能とエッジ機能

４．JISによるエッジの表示とエッジ品質の等級

５．バリおよびエッジの測定ならびに評価

６．設計技術によるバリ抑制方法

７．加工技術によるバリ抑制方法

８．バリ取り・エッジ仕上げ加工の種類と選定方法

９．バリ取り・エッジ仕上げ加工の最新動向

【習得できる知識】
　今日の製品の軽薄短小化および高機能化などにおいて、我が国のものづくり技術先進国として更なる高付加価値化に対して、また航空機用機材に使用されるCFRP材やチタン材をはじめとする難削材料の加工に対して、加工後に生成するバリの除去ならびにシャープ・エッジの形成が大きな課題となっており、永遠の課題と言われるものの現在大きな問題となっている。このような課題に対する解決の糸口を与えるためにバリ取り・エッジ仕上げ技術が必要である。
