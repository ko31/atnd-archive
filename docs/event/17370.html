この度、四国GTUGでは、Androidアプリ開発を推進すべく、

日時：2011年7月2日(土) 13:00頃～17:00頃
場所：サンポートホール高松 ６６会議室
参加費：¥500円（高松会場・会場費に充てます）

おしながき

前菜：アイデアのディスカッション
こんなアプリあったら使ってみたいなを繋げてみよう。

メインディッシュ：Androidアプリ開発環境の構築
どうすれば、Androidアプリを作成できるの？
デザート担当（以下参照）のシェフが、 アプリの
調理方法の初歩から教えてくれます。

デザート：すでにがっつり作っているぜという方
今回サンプルで Gtalk チャットソフトを作ります。
余力があれば、改造を行いますので、追加したい 機能などを
考えてきて下さい。

デザイナー、ブログラマー、SE、アプリに興味がある方、
パソコンを持って是非参加してください。 パソコンは Windows 環境を推奨します。
Mac 、 Linux の方は事前に開発環境を整えてくることを推奨します。

懇親会：瓦町で有志で行います。参加登録で一緒にご登録ください。
