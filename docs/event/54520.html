ITストラテジスト試験対策講座

ITストラテジスト試験の合格を目指す方を対象とした試験対策講座です。
ITストラテジスト以外の情報処理技術者試験で論文執筆に不安がある方も是非お越しください。
「論文試験は決して難しくない！」ということを実感いただけると思います。

午前・午後ともに傾向や試験勉強のコツ、そして論文試験のポイントをわかりやすく解説いたします。

主催：ITストラテジスト協会九州支部
お問い合わせ：岩本(iwamoto@shuta.net) Twitter:@shutanet

会場は福岡市中央区のAIPCafeです
