description

DSゲームカフェ「カルドセプト」応援キャンペーンと連携したオフ会を兵庫伊丹で開催します。
ニンテンドー3DSソフト「カルドセプト」を持っている人なら誰でも参加できます。
※4人戦×2卓を同時に行えるよう、8人で開催することにします。

参加費：1500円（＋入会金200円）
※参加にはワイプの会員登録が必要なので、非会員の人は名前・現住所・生年月日がわかる書類（運転免許証・保険証等）を持参してください。

会場はワイプ阪急伊丹駅前店のダーツコーナーになります。
コンセントは4口あり、延長コード・タップも用意されているそうです。
ニンテンドー3DSの充電器は各自で持参してください。
飲食物の持ち込みはアルコール類以外であれば可能です（店内にフリードリンクコーナーはあります）。

開催日はカルドセプト関連グッズプレゼントの抽選の実施期間でもあるので、参加すれば1人あたり2回ずつ抽選に参加できます（抽選は4時間に1回）。

対戦形式、オフ会開催特典のグッズの分配方法

　＜予選＞
　　　3マップで1試合ずつ、4人戦×2卓を同時に行います。組み合わせは毎回くじで決めます。
　　　ルールはすべて［E禁止］ 時間制限［40秒］ その他の対戦条件はマップの初期設定通り。

　　　使用マップ：
　　　　1戦目：ジャンクション（7000G）
　　　　2戦目：３つの鍵（9000G）
　　　　3戦目：変異手裏剣（7000G）

　　　参加者は1試合ごとに対戦の順位に応じて、以下のポイントを得ます。
　　　　1位：12P、2位：4P、3位：2P、4位：1P

　　　※対戦の途中でAI化した場合も、AIの順位でポイントを得ます。
　　　　ただし、5R以内にAI化した場合は最初からやりなおしとします。

　　　※もし急遽欠員が出た場合、会場の無線LANかイノセンス持参のWiMAXを使用して
　　　　オンラインで参加者を募集することを考えています。

　＜決勝戦＞
　　　予選での総ポイントが多い順に4人選出し、決勝戦を行います。
　　　ボーダーで同ポイントの人が並んだら、くじ等で決めます。
　　　決勝戦で1位になった人が本オフ会の優勝者となります。

　　　ルールは予選と同様［E禁止］ 時間制限［40秒］ その他の対戦条件はマップの初期設定通り。

　　　使用マップ：ラティス（7000G）

　　　※決勝戦に出場できなかった4人も並行して同ルールで対戦します。

　＜グッズの分配方法＞
　　　グッズは缶バッジ型音楽プレイヤー1個、オリジナルタンブラー2個があります。
　　　最初に優勝者が、好きな方を選んで獲得します。
　　　その後は、くじ等で残った7人の中から2人当選者を決めます。

※終了後に追記：対戦結果

希望者がいれば終了後に二次会として近くのお店に食事に行きます。
