健康"感"察

イベント ユーザー
event_id 31686
title 健康"感"察
catch
description

健康に気をつかってはいても、
それを自ら進んで管理したり、つきあっていくことは、
中々難しいことはありませんか…？

本ワークショップは
“Inclusive Design”という、個人を追求することを特徴とし、
人々が気付いていない、しかし共感できるニーズを発見するための
デザイン手法を使い、実際に形にしていきます。

その人が感じる"健康"を大事にしたもの、
楽しみながら健康管理が続けられるものを
一緒につくってみませんか？

【内容】
①発見しよう　(９月１日)
健康ってなんだろう・・・？

実際に健康管理に困ってる人の普段の生活の中から、
何気なく見逃してしまっている、
「実は問題であるもの」を発見します

②アイデアを形にしよう(９月下旬)
①で発見された問いを解決する為に、
コンセプトを設けた上で「アイデア」を考え、
実際に「形」にしてみます

③深めよう(１０月以降)
②で形にしたものを
その可能性と関わりについて、
普段から医療や健康に携わっている人と深めます

※ワークショップは３回に分かれていますが、
もちろん１回のみの参加でも大歓迎です！！
