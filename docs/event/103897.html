ご挨拶

草の根から熊本を盛り上げるべく開催しております「熊本情報セキュリティ勉強会」の第21回を、平成31年3月2日(土)に開催いたします。 地震で色々と影響はありましたが、だからこそ歩みを止めず取り組みを続けて行きたいと考えております。あと、色々運営側の環境が変わって第21回から空いてしまって本当に申し訳ありません。モチベーションのチャージ期間は必須なものでして…

テーマ

さて今回は、国際セキュリティカンファレンスAVTOKYOでスピーカーを務め、犯罪、情報漏えいからBCPまで実社会に即した幅広いネタで様々な勉強会で講演されているAIDO氏を講師としてお願いしております。AIDO氏は、第4回と第11回に講師をお願いし大盛りしましたが、流石にそれから4年強ということで、初めて見られる方も多いかと思いますが、面白さと内容の深さが入り混じった発表をぜひご期待ください。

テーマとしては、タイトル自体は現在準備中ですが、2本仕立てをお願いしており、ひとつは「システムリスクはこれをやっとけ」とある大きな組織から俯瞰したサイバーセキュリティに限定しない"システムリスク"に関するヤバイ解説と、もうひとつは「仮想通貨のはなし」技術面とは異なるやはりこちらも普段では東京であってもなかなか聞けないサイドからの仮想通貨制度に関する話を予定しております。

色々と伏せないといけないものがあるというところを汲み取ってご参加頂ければと幸いです。

勉強会のカラー

「勉強会」っていうと詳しい人の集まりで難しいんじゃないか、と不安に思われる方もいらっしゃるかと思いますが、本勉強会では、「セキュリティに興味があればＯＫ！」をテーマに、難しく考えがちなセキュリティを楽しく学べる場として、ボランティア的に企画しております。熊本に限らず近郊地域の皆様に参加頂き、セキュリティというものを身近に感じて貰える様な、また同じような興味を持った人間同士の交流ができる会となれば幸いです。
