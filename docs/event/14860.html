日本アンドロイドの会 横浜支部の第7回定例会です

前回の第6回はビジネストラック・開発デザイントラック・タッチアンドトライと盛り沢山の内容で開催し、各方面から好評を頂きありがとうございました。第7回は新年度に入り、新しく横浜で活動される方も多いかと思われますので、参加者の交流を主眼に置いての開催です。

横浜には著名なAndroid開発者も多く、コミュニティ活動で貢献されている方や勉強会を主催されている方もおります。
是非この機会に、知見と交流を深めてください！

ビジネストラック

・横浜コミニュティデザイン・ラボ　の御紹介（横浜コミニュティデザイン・ラボ　杉浦 氏）
・sinsai.info の御紹介（sinsai.info 嶋坂氏）
・災害時支援アプリ の御紹介（日本アンドロイドの会 福祉部 飯塚氏）

LTについて（大募集）

是非参加者の交流を、ということで、LTを発表して頂ける方を大募集します。

今回はLT中心でテーマも必ずしもAndroidコテコテである必要ありません。
見てるだけ聞いてるだけの定例会から、ちょっと5分だけ自分も参加する定例会にする大チャンスです。

募集人数は最大20人、100分前後を予定しています。
ノートPC、プレゼン資料は各自でご用意ください。（プロジェクタはあります）
GoogleドキュメントなどWebで表示できると準備が楽です。

申し込みはこちらのフォームからお願いします。
http://goo.gl/xVaXy
締め切りは4/17（日）の24:00とさせていただきます。

下記に該当する方は是非ご参加ください！

・Android Application Award 2011 Springに登録した方など、是非アプリケーションを紹介してください。
・A3申し込んだのにアプリ登録できなかったなんていう自虐ネタも歓迎です。
・アンドロイドに拘らず震災関係や自己紹介ネタなども勿論アリです。
・運営委員もLTしますよ！

発表は基本5分ですがもっと時間をよこせという方はその旨、申し出てください。応相談です。

交流会

ビジネストラック・LTが終了した時点で、その場で交流会を実施します。
交流会は有志による持ち寄りのほか、食べ物飲み物を用意します。
実費では1000～2000円程度かかりますのでご了承ください。

※差し入れ歓迎です。

交流会会場は時間制限ありません、飽きたら適当に帰るなり、連れ立ってお店行くなりとなります。

その場では、椅子・机・プロジェクタが引き続き利用できます。
開発で困ったことを聞いたり、アプリケーションを紹介したりは自由です。
隅でもくもくしててもOK！
