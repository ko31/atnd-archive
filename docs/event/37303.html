概要

大阪で攻防戦CTFを経験してみようという企画です．

CTF is 何

CTF (Capture The Flag)

セキュリティ技術を競うコンテストの総称です．
セキュリティに関する幅広い分野から問題が出題され，正解すると得点を得ます．
競技時間内に得た総得点をチームごとに競うゲームです．

詳しくは ココらへんを参考にしてください

クイズ形式と攻防戦形式

CTFは，クイズ形式と攻防戦形式があります．
クイズ形式は予選で，攻防戦形式は本戦で採用されることが多いです．

攻防戦形式は物理的な制限から経験する機会が少ないです．
今回はその攻防戦形式のCTFを経験してみようという企画になります．

なお，以下の攻防戦CTFのルールはSECCON CTF 全国大会で採用されたルールを参考にしています．

攻防戦CTFルール

攻防戦CTFでは，参加チームがclosedなネットワーク環境下で脆弱性を含んだ問題サーバに対して攻撃を仕掛けます．
次の条件をクリアすることで点を取ります．

  1. サーバに隠されたキーワードを見つける
  2. サーバの指定されたFLAGファイルにFLAGワードを書き込む
    (一定時間ごとにFLAGワードは変化し，その度に得点が加算されます)

おおまかな手順

  1. サーバの脆弱性を調べる
  2. 脆弱性を利用してサーバに侵入し，キーワードを探す
  3. キーワードを参考にしながらFLAGファイルへの書き込み方法を模索する
  4. FLAGファイルへFLAGワードを書き込む
  5. 他のチームがキーワードやFLAGファイルへのアクセスできないように妨害しつつ他のサーバを攻略する

※注意
一般のサーバに攻撃を仕掛けることは犯罪となる恐れがあります．
このCTFで経験したことをネット上で試す等の行為は絶対に行わないでください．

問題について

2013/2/23-24 東京北千住で開催された SECCON CTF 全国大会 で使用した問題サーバを利用します．
この問題サーバはSECCON CTF運営のご好意でイメージを頂けるとのことです．

日時

2013年ゴールデンウィーク以降の休みの日の予定

場所

おそらく大阪大学になります

イベント開催に必要なもの(メモ)

  1. 場所: 朝から晩まで使える会場
  2. 機材: 競技用ネットワーク構成のためのルータ, ハブ, LANケーブル等
  3. 問題: SECCON運営様から頂けるはず．いつ貰えるかはわからない．
  4. チーム決め: 参加者がある程度集まったら考えます
