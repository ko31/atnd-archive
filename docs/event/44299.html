SWWDC 第15弾を開催します！

仙台iOS開発者勉強会 SWWDCも、早いもので15回目の開催となりました。
今回は体育の日にちなみまして、もくもく会(ミニ・ハッカソン)を開催したいと思います。

内容 〜体育の日は元気に屋内で、もくもく作業！〜

内容は各自集まっていただいて、各自やりたいことをやっていただく感じです。一応iOS関連のことをしていただくのが推奨ですけども、テーマは自由です。過去にはこんな事をされている方もいました。

  • もくもくと、iPhoneアプリの作りこみ（正統派）
  • 参加者のiPhoneを使用して、複数台通信のデバッグ（多人数が集まること機会を利用）
  • ブログの執筆（iPhoneあんまり関係ないかも）
  • phpでwebサイト構築（iPhone関係なし）

というように、周りの迷惑にならない範囲で色々やってくださいw。

「自宅にいると集中できないんだよね」とか、「ちょっとやりたいことあるんだけど、まとまった時間を作って作業したい」などの解決に良いと思います！

（※なお、iPhoneアプリ作成に関して「教えたがりオーラ」をお持ちの方が多数参加される予定です。iOS関連で何かお困りの方は、ぜひ参加してみてください。）

タイムテーブル

  • 17:40- 片付け開始
  • 18:00- 終了
