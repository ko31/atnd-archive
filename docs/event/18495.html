【講座のポイント】

　プラスチックを用いた製品において，構造、製造プロセスあるいは使用条件により生じる残留応力・ひずみが製品特性の安定性などに影響する場合がある。これらの問題に事前に対処するため、プラスチックの力学的性質、ならびに残留応力・ひずみが生じるメカニズムとその低減化技術を解説する。

受講対象者レベル

プラスチック製品の設計、成形に関る技術者

【プログラム】

１．プラスチック成形品の残留応力の発生要因

２．残留応力に関わるプラスチック特有の性質

　　2.1　プラスチックの結晶化

　　2.2　プラスチックの物性の異方性

　　2.3　線形粘弾性と残留応力

３．残留応力の測定法

　　3.1　機械的測定法

　　3.2　光学的測定法

４．成形に伴う残留応力と低減技術

　　4.1　熱可塑性樹脂

　　4.2　熱硬化性樹脂

５．低残留応力化成形法

　　5.1　超音波射出成形

　　5.2　冷熱サイクル成形

　　5.3　電磁誘導加熱成形

　　5.4　炭酸ガス溶解成形

　　5.5　多段速度制御定圧成形

　　5.6　射出圧縮成形

　　5.7　成形条件と残留ひずみ，残留応力
