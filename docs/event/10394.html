description

蜜蝋キャンドルのやさしい灯りの下で
チベットで語り継がれてきた宝物のような民話と歌の世界に
今宵ひと時一緒に包まれてみませんか…

◆うた・ダムニェン　：　川辺　ゆか
様々な土地や人をおもい、訪ね、歌う “うた旅行家”
チベット、日本、東地中海、アイルランド等の歌をうたい、関西を中心とした音楽ユニットに参加。
自らの足で訪れ、時間をかけ宝物のように拾い集められた音は、どこか懐かしく、大切なことを思い起こさせてくれる。
また、アムステルダム在住の美術家、渡部睦子のプロジェクトchikahomeに参加している。

◆朗読　：　minae
おとぎの国からやってきた！？不思議な声の持ち主。富田林のパン屋「ぴんぽんパン」
店主。自家栽培の野菜・果物・ハーブなどを使った、季節感のある、身体に優しくておいしいパン・お菓子を作っています。

◆100万人のキャンドルナイト　でんきを消して、スローな夜を。

日時12月23日（木・祝）　18：00～（17：30開場）
