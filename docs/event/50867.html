イベント概要

超汎用2Dアニメーションツール 『OPTPiX SpriteStudio』 の大きめのアップデートである"Ver.5.3"リリースを記念(?)して、初の単独勉強会を開催いたします。

今回の勉強会では、Ver.5.3で追加された 新機能、改良ポイント のご紹介の他、ちょっと使い方が難しい カーブエディタの使い方 解説、それに現在GitHubで公開している SpriteStudio SDK の解説を。
また、ユーザーによる ライトニングトーク も調整していますので、これからSpriteStudioを使おうと考えている方はもちろん、今現在SpriteStudioをお使いの皆さんにも～と、 初級～上級 まで広く対象としている勉強会です。
当日は 『OPTPiX SpriteStudio』のステッカーも配布いたします。
