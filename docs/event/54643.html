description

ホームページやブログを作ってみたいけれども、どのサービスを利用すれば良いかがわからず困っている方、記事を書く上で写真や商品の画像を掲載したいのだが方法がわからず困っている方、様々な立場の方とお話ができればと思い、交流会を開いております。

私は日頃webプログラミングに取り組んでおりますが、
お客様の話を聞いた上で作業に取り組む機会があまり持てていません。
そのため、皆様の意見をお聞きした上で何か役に立てるようなアドバイスができればと思っております。

交流会の中でお話して頂き私からアドバイスをするといった流れだけではなく、皆様から学ばせて頂くことは多々ありますので、ご参加頂けますと幸いです。

また、交流会の前に既存のwebサービスを使ってどのようなことを実現したいか、ホームページやブログを作成する上で何を使おうとされているか等具体的にご連絡を頂けますと、当日の話はスムーズに運ぶと思いますので、ご連絡を頂ける方はお待ちしております。
（まだ、何をしたいか漠然としているよという方は、交流会で当方よりお話を進めさせて頂き何が必要なのか一緒に考えていきましょう。）

私は職場では普段着で出勤していますので、当日は普段着で参加致します。
皆様も当日はお話がしやすいスタイルでお越し下さい。

※また、弊社ではこちらの活動をブログに掲載しておりますので、ご了承ください。
個人を特定できるようなものや、秘密事項については掲載しません。

それでは、ご参加を心よりお待ちしております。
