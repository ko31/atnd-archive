description

こえ部で放送中のラジオ「TRYout」が一周年を迎えます。
こえ部でもラジオをやってみたい！という方がたくさんいるということで、
こえ部で行うラジオをメインテーマにオフ会を開いて見ようと思いました。
ラジオパーソナリティに必要な「アドリブ力」を鍛えていきたいと思っています。

集合時間：　10:30　（途中合流OKです。その場合は事前に連絡をお願いします）
