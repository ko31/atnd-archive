Motoki Boot Camp’2011【夏】

〜いつもの場所でまた逢いましょう〜

概要

初めてキャンプを開催したのは、
2006年Joho大と美専と支笏湖で開催した合同キャンプ。
気づけば過去５回キャンプを開催し、
１度は台風が直撃しても開催した「Motoki Boot Camp」

首謀者が東京(8月現在は名古屋)にいってしまって、
今年は無理だなと思っていた今日この頃ですが。

「やっぱり夏は、支笏湖で過ごしたい！」
と思い、【夏】北海道に戻ります。

  • あのいつもの場所でテント貼りたいよね
  • 今年は誰がメガネを落とすんだ？
  • 夏は水辺でスイカを食べたり花火をしたい！
  • キャンプ後の温泉が好き
  • いつものおそば屋さんもいくんだよね？

….などなど、
色々意見があるかと思いますが、
一緒に【夏】を楽しみたい方のご参加お待ちして下ります。

参加資格は、Motokiの知り合い。
来るモノ拒まず、去る者多少追う感じです。
宜しくお願いします！

会場及び日時について（詳細決定）

集合日時 2011/08/06(土) 10:30
集合場所 北海道情報大学
※参加者は基本的に上記の場所に時間厳守で待ち合わせ。

※自動車は３台、12名が集合予定。
※当日遅れるなど何か連絡事項があれば、オーナミ君までご連絡を。

集合日時 2011/08/06(土) 11:00

※支笏湖キャンプの締めは、ここも毎年恒例の盛朗庵でおそばです。

その他諸々

７月後半に参加希望者が急に増えて、トータル20人になりました。
人数に伴ってレンタカーを１台借りることにしました。ただ人数が人数なので頭数で割ると負担額はかなり少なくなるかと思います。

キャンプ用品に関しては、モトキ、オオナミ、ミッツが中心になって用意しますので、その辺はご心配なく。ただし支笏湖の夜は冷えると思いますので、寝袋や毛布、防寒具を忘れずにお願いします。寝具に関して一人一人への対応は難しいと思いますので各自の対応をお願いします。

テントに関しては２−３つしか確保できないと思われるので、朝まで語るも良いですし、車中泊も想定しておいて頂けると幸いです。ただし女性陣は優先してテントをお使い下さい。
※ただ毎年、朝まで語る方々が多いですね、しっかり寝る人の方が少ない。
