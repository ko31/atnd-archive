イベント概要

福井で毎月行っている炉端の集いですが、今回は新拡張「コボルトと秘宝の迷宮」の実装直後ということで
トーナメント形式の大会(賞品あり)　さらにサービス開始1ヵ月が経った「ドラゴンクエストライバルズ」の大会も併せて行います！

時間があればボードゲーム等も行う予定です。

ウォーロックスキン・ネムシーが獲得できる炉端の集いになっております。

ATND参加申請特典

参加申し込みに関しては当日の飛び入りでも可能ですが、今回はこのページで事前に参加申請しておくと
酒場の主人(くしぃ)自作のハースストーンリアルカード
が貰えるらしいので、参加申請する方は参加時のコメントorこのページのコメント欄に

  • カード名
  • テキスト言語(英語or日本語or他言語)

を書いておくと当日希望したカードが参加特典としてプレゼントされます。

新拡張のカードでも受け付けます。当日までに希望カードの変更することも可能です。
ハースストーンのカードは枠がゴテゴテしているので多少の余白が出るのでご了承ください。レジェンドカードは特にゴテゴテなので要相談。

特に希望の無い方には新拡張の中から適当にピックします。

クオリティに関しては初挑戦なのであまり期待しないでください。当日間に合わない可能性もあるのでご容赦ください。

大会詳細

ハースストーン大会

①二人三脚トーナメント
2人組タッグによるトーナメント形式の大会です。 先鋒と大将を決めてもらって対戦の時は先鋒vs先鋒、大将vs大将の形で対戦します。
チームの2人共勝利すれば勝ち上がり、1勝1敗の場合はお互い勝った者同士で決定戦を行います。所謂早稲田式と呼ばれるものです。

勝ち上がった場合、次の試合までにデッキ交換及びカード入れ替えを行ってもOK

2人組はエントリー時に事前に結成することも可能ですし、主催にくじ引きでパートナーを決めてもらうこともできます。

②炉端の喧嘩トーナメント
12月炉端の喧嘩は「最強のコンビ」です。以前にもありましたが今回は少しルールが変わっていて

2枚のうち1枚は酒場の主人が選んだカードで2種類15枚ずつのデッキが作られます。そのデッキで一発勝負のトーナメントです。

お題は当日までのお楽しみ★

ドラクエライバルズ大会

BO1のトーナメント形式の予定ですがルール変更の場合があります。
何か楽しそうな対戦形式があれば是非ご意見ください。

注意事項

会場には駐車場がありますが、駐車台数が限られているので乗り合わせや公共交通機関の利用を推奨しております。
駐車場所は複数ありますので必ず確認の上駐車をお願いします。駐車場が一杯の場合近くのコインパーキングをご利用ください。
会場内は飲食物の持ち込みは禁止です。ドリンク及びフードオーダーは通常通り行っています。
Wi-Fiは会場内の無線と主催のポケットWi-Fiが利用できますが、接続人数によっては遅延等も発生する可能性があるため自前のインターネット環境はできるだけ用意しておいてください。
開始時間にどうしても間に合わないという方は事前にご相談ください。
対戦中のトラブルが起こった場合は速やかに主催までお伝えください。
