description

当日は、18Fエレベーターホールに直接お越しください。19:15～19:30の間でしたらスタッフが会場にご案内します。

遅れる場合は、20:00までならTwitterでハッシュタグ(#TokyoSNA)をつけてつぶやけばスタッフが案内します。

20:00を過ぎるとニフティ社員の同行なしではビルに入ることもできなくなります。20:00より遅れる場合は事前に @who_you_me までreply/DMください。

オライリーから出版された『 オープンソースで学ぶ社会ネットワーク分析――ソーシャルWebの「つながり」を見つけ出す 』をテキストに、みんなで社会ネットワーク分析について勉強する会です。

日々分析を行なっている方、「TwitterやFacebookの分析やってみたいけどよくわからない」という初心者の方、エンジニアの方、エンジニアじゃない方、社会人の方、学生の方など誰でも大歓迎です。主催者もへっぽこなのでゆるーい感じの会になると思います。気軽にご参加ください！

進め方

1回1章ずつ、隔週ぐらいのペースで読み進めていこうと思ってます。ただ輪読するだけでもつまらないので、以下のような流れでやります。

テキストの輪読（40分）
1章ずつ担当者を決めて発表していただきます。内容は発表者に一任しますが、もちろんテキストの内容から外れ過ぎないように（笑）
↓
社会ネットワーク分析（SNA）に関する発表（30分）
こちらも各回ごとに担当者を決めて、「こんな分析やってみました」「こんなツール使ってます」みたいなSNA関連のネタを発表していただきます。ちょっとでもSNAにかすってればテーマは自由です。
↓
演習（40分）
実際に手を動かして理解を深めましょう。
とりあえず初回は、Pythonのインストールおよび必要なライブラリ（networkX, numpy, matplotlib）のセットアップあたりを課題にしようと思います。2回目以降は未定ですが、どっかから練習問題見つけてきてみんなで考える、とかを考えてます。

内容についても、とりあえず初回はこんな感じでやりますが、みなさんの要望に合わせてどんどん変えていこうと思います。みんなで作り上げる勉強会にしましょう！
