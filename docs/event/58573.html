description

今年創立102周年を迎える早稲田大学広告研究会は「ワカモノ文明開化」と題して、昨年延べ16万人以上が来場した早稲田祭において4つのイベントを行います。その一つのイベントである『厨二妄想概論』では、『文化妄想学部』という架空の学部を創設し、『妄想』をテーマに、様々な分野でご活躍されている安齋肇様をお招きして講義形式の観客参加型トークイベントを開催します。

本イベントでは、来場者に『早稲田大学文化妄想学部厨二妄想学科』に入学した生徒としてイベントに参加していただき、一風変わった素敵なゲストによる『妄想』に関する講義を、来場者も実際に発言する場面を設け、リアルタイムでTwitter等を活用しながら、観客一体となっておこないます。
さらに、より文化妄想学部生として楽しんでいただけるよう、教科書等の配布も予定しております。

――――大人は、空想と現実の区別がつかないといけない――――
そんな言葉をよく耳にします。しかし、子どもの頃の発想はもっと自由だったはずです。
妄想まで現実に縛られる必要はないのです。ユニークなゲストの独創的な感性に触れながら、少年少女時代の『妄想』に、想いを馳せてみませんか?
