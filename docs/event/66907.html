初心者歓迎【学生限定】Firefox OS アプリハッカソン　

ー 音の出るものを作るー

Firefox OS上で動くアプリケーションを作成するハッカソンをTECH LAB PAAKで開催。

今回はMozilla Japan様・KDDI様にご協力をいただき、MozillaエンジニアのサポートのもとFirefox OS端末であるFx0を用いて、Firefox OS アプリケーションの作成を行って頂きます。

創る自由が楽しめると話題のFirefox OSで“音の出るアプリを作る”をテーマに取り組んで頂きます。初心者の方もその他ネイティブアプリエンジニアの方も、この機会にクロスプラットフォーム開発の体験をしてみませんか？ネイティブアプリエンジニアとしてWebプログラミングを経験してみたい、Firefox OSに少しでも興味があるという方に是非ご参加頂きたいハッカソンです。 豪華な優勝商品もご準備いたしましたので、是非チェックしてみてくださいー！

追記：開発ドキュメントデータをアップしました！初心者の方も大歓迎ですのでぜひご参加下さい。
