description

世界人口の約1/4を占めるムスリムの訪日観光客が増えるなか、スポーツの国際大会においても食事を提供する際は、食材だけでなく、調理器具や提供方法なども含めてきめ細かく対応することが求められています。長年ハラール食品を製造販売されている二宮伸介氏に、日本でハラール対応をする際のポイント、国内のハラール食品の製造や販売状況、ハラール認証について、国内レストラン等での対応状況等についてお話ししていただきます。また、ハラール対応のオリジナルのアスリート弁当をご試食していただきながら、弊社代表の橋本より過去の国際大会で提供されたハラール食やアスリートへの提供方法についてご紹介します。
