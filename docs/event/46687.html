ヒカ☆ラボ一覧は こちら！！

★講演内容

最近フリーランスになったばかりの方、
今後フリーランスとして活動したい方、
確定申告について不安や疑問がある方に向けて
最低限おさえておきたい確定申告のポイントを分かりやすくお教えします！

「 初めてだけど、確定申告っていつまでに何すればいい？ 」
「 青色申告決済書ってどうやって書けばいい？」
「 確定申告の際に気をつけなければいけないポイントとは？」

この機会に確定申告に関する不安や疑問を解消しましょう！
