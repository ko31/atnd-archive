インプロとは｢improvisation｣の略で｢即興｣と訳されますが、私たちは｢即興芝居｣という意味で使っています。元々は俳優のトレーニング法として開発されたもので、欧米では学校の授業や企業研修にも取り入れられています。

またエンターテイメントとしても確立しており、台本がないためやっている本人も先がどうなるか分かりません。即興ですから上手くいくことも失敗することもあります。アクシデントが起こるのも魅力のひとつでその瞬間にしか生まれないハラハラドキドキのストーリーが展開します。

現在では日本でも即興でのパフォーマンスのみならず、コミュニケーション教育や人材育成のため教育・福祉現場や企業研修でも利用されています。

3/7(土)のＴＢＣ｢ウオッチンプラス｣でもインプロを紹介していただきました！
　
今回は｢繋がる｣ということをテーマに、仙台駅東口の隠れ家的スタジオ「夜カフェインプロ」を開催します！

第一部は、通常の「インプロワークショップ」18時30分～21時30分予定
意識をしないで自然体の方が繋がれるという事を体験していきます。お芝居や表現活動、コミュニケーションでも大切な要素です。

第二部は、「夜カフェ」21時30分～22時30分予定
いろいろなドリンクを飲みながら、みんなで語り合いましょう。

飲食物の持ち込みや差し入れは自由です。その場で豆から挽く美味しいコーヒーも準備します！

参加費は2000円。
第一部「インプロワーク」のみの参加は1500円。
第二部「夜カフェ」のみの参加は500円。
時間帯は若干変動する可能性があります。
見学もできますので見るだけでも構いません。

会場は参加者にお知らせします。仙台駅東口徒歩８分。
