概要

Effective Java 第2版 を集まって読む かなりゆるい 会です。
第1回 に参加してなくても問題ないです。
Java7が間近に迫ってきた事ですし、現時点のJavaの足場をしっかり固めましょう。

途中参加、途中離脱OKです。
費用は無料です。
エレベータで7Fの会場に直接お越しください。

対象

  • Javaをしっかり勉強したい。
  • 一人で読むと進まない。
  • EffectiveJavaを復習したい。
  • 他の勉強会の敷居を高く感じている。
  • 発表してみたいけど敷居を高く感じている。

Javaの基礎的な文法（ifとかforとか）で詰まったりするのは厳しいかもしれませんが、
突っ込んだ所（Genericsのゴニョゴニョとか）は随時誰かが補足してくれる事を期待してます。

つまり「EffectiveJavaを読もう」と思った時点で、参加条件はほぼ満たしています。

持ち物

  • Effective Java 第2版
    ノートPCは無くても問題ない感じです。
    あってもいいかも。電源と無線LANはあります。

準備

  • 心構え
    • グダグダに耐える必要があるかもしれません。
    • 脱線を許容する必要があるかもしれません。
  • 予習
    • なし（その場で読むため）

すすめかた（暫定）

  1. 適当な人数のチームにわかれます。
  2. 各チームで読む範囲を決めます。
  3. 読みながら雑談します。
  4. 各チームで代表者を決めます。
  5. 読んだ項目について発表して頂きます。（1チーム5分程度）
  6. それについて全体で雑談します。
  7. 余裕があったらループします。
