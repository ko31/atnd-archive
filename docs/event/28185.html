title だれでもわかる財務入門

description

財務はほんとに簡単で、難しくありません。単に財務用語を覚える段階で、理解への障壁となるだけです。 この講座は算数さえわかれば、誰でもわかる内容ですので、財務の苦手な人に最適な入門講座です。
経営者や社会人の中には、財務が苦手な方はとても多くいます。ただ、財務は企業として、社会人として避けて通れないことから、最低限の必要な基礎知識を培って頂くために、この入門講座を開催します。
損益計算書と貸借対照表の『意味』や『つながり』を、『具体的に』わかりやすくご理解頂ける内容です

初心者対象の入門講座ですので、財務や簿記会計の知識は不要です。
過去4回開催し、多くの経営者・経営幹部・大学生の他、70代女性、そして経理を経験したことがない方にもご参加頂き、意味がわかりとても興味がわいてきた等、大変ご好評をいただいた講座です。
超初心者や、木村会長の『BS経営の税金川』が理解出来ない方にもおすすめします。

講義時間　9：45～11：15

こんなかたにおすすめ

・財務の基本的な見方を、カンタンに身につけたい方
・経営者・起業家・独立準備中の方
・経営幹部を目指す方、経営に興味のある方
