Gaprot(ギャップロ)勉強会

ギャップロは iOS/Android エンジニアのための技術情報サイトです。

第 3 回となる今回は、Unity で使用することの出来る Google VR SDK に関しての勉強会を開催します。

今回は Google VR SDK の上手な付き合い方と題しまして、それを実践的に使用した場合困るであろう部分に対しての解答や、知っていると得する情報をお伝えできればと思います。
「使ってみたけど結局挫折してしまった」「なんか変なバグが起きて対処できない」等といった方にとてもためになる勉強会にする予定です。

第1回、第2回と同様にアップフロンティア株式会社の会議スペースで開催します。

対象

  • Unity で Google VR SDK を利用したいエンジニア
  • Google VR SDK を利用していて困った経験があるエンジニア

内容

Unity で Google VR SDK を実際に利用する上で気をつけたほうがいいことや、困った時の対処方法などを実戦ベースで話します。

実際に開発をした上で得た知見が盛りだくさんなので、これから VR をやっていく方は聞きに来てみてはいかがでしょうか。

また勉強会終了後、VR 機器の体験会を予定していますので、そちらの方も楽しみにしていただければと思います。

予定

  • GVR SDKの歴史
  • 視線入力
    • 実装するなら、RayCastを使おう。
  • ある場面で覚えておくと良い関数。
    • どこを見ていても正面にしたい時。
    • 単眼・複眼を切り替えたい時。
    • 複眼にした瞬間の画面の乱れを消したい時。
    • 複眼時の左上のボタンを検知したい時。
  • Manifest設定の注意点
    • 視差設定が反映されなくなる。
    • 複眼時の左上ボタンと右上設定ボタンが表示されなくなる。
  • その他の不具合
    • スリープタイムアウトが設定できない？！
    • Editorと実機で挙動が違う。

勉強会に使用した資料はギャップロサイトにて公開しますのでそちらの方も確認してみてください。
