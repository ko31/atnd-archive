description

ビール（お酒苦手な人はジュース）でも飲みながら、ピザや餃子をつまみながら、ゆる～い感じの読書会にしたいと思います。

くれぐれも、大事な本をピザソースで汚さぬよう気をつけてください。

とりあえず最初なので、3章くらいまでのざっくりとしたまとめはこちらでやっておきます。

必須ではないですが、参加される方は以下の物、ネタを持って来ていただけると嬉しいです。


  • 本、あるいは電子書籍リーダー

  • 自己紹介に使うポジペ（ノートPC持参でも紙持参でも、なければ名刺でも口頭だけでも）

  • よくわからなかったところ

  • みんなと議論したいところ

  • 関係ありそうな悩みごと

  • 関係なさそうだけど酒の肴になりそうな恋の悩みごと、家庭の悩みごと、etc…


ちなみに、ビールサーバがあるのでビール代は（樽に残っている限り）無料ですが、食べ物やおつまみの分は割り勘で行きたいと思いますので、参加費は1000円から1500円程度とお考えください。

20時以降に参加される方は、場合によっては1階の入り口の鍵が閉まっていることもありますので、お手数ですが誰かに気付いてもらえるよう必死につぶやいてみてください。

どうしても心配な場合は事前に @takaesu0 に連絡先を問い合わせておいてください。


また、万が一終電以後も飲みたい場合は朝まででも寝泊まりも可能ですが、翌日は別の勉強会も開催されますので今回は控えめにしましょう。
