description

エニアグラムは、性格タイプを9つに分類し、それぞれの特徴とタイプ間のさまざまな関係性を明らかにしていくツールです。

個人の本質的特性や陥りやすい思考・行動パターンなどを理解するために役立ちます。

自己理解から他者理解が進み、各メンバーが自己理解・他者理解を進めることによって組織活性化を図ることが可能なため、ストレスチェック後の職場環境改善の一方策としても注目されています。

・自分の本質的な強みの発見
・就職・転職活動の自己分析
・今後のキャリア設計・人生設計の際の参考
・友人・恋人・家族・職場での人間関係のあり方を見直すため
・チームビルディングのアイデア
として活用することができます。

今回のワークショップは、初めてエニアグラムを学ぶ方々を対象に開催されています。
レクチャーに加えて、簡単なグループワークを予定しています。

【講師紹介】
竹村基弘
産業カウンセラー・キャリアコンサルタント・THP心理相談員・EAPコンサルタント
2006年よりエニアグラム研究所[日本]の代表であるティム・マクリーン・高岡よし子両氏に師事。
キャリアカウンセリングや心理カウンセリングの現場でエニアグラムを活用。
企業研修や障害者の就労支援の現場でもエニアグアムのワークショップを行い、受講者の人間的成長を支援してきた。
