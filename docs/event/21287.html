description

日本舞踊に対して、みなさんはどのようなイメージをお持ちでしょうか。
着物を着て、扇子を持ってはんなりと踊るイメージ？
間違いではありませんが、日本舞踊とは実はもっと奥の深いものです。

今回のSNCは日本舞踊の成り立ちから日本舞踊における様々な要素（使用楽曲、衣装、小道具、見立て等）を
簡単にではございますがご紹介させていただき、皆さんが日本舞踊をご覧になる際に少しでも楽しんで頂ける
知識をお伝えする機会にしたいと思います。

また、体験コーナーとして扇子を素早く片手で開くコツを教える時間も少し設けたいと思いますので、
ご自宅に扇子がある方は是非お持ち下さい。普通の扇子で結構ですし、もし一本もない方がいらしても
貸し出し用をご準備致しますのでご安心下さい。

せっかくのチャンスですので、
是非とも日本人の方はもちろん、外国人の方も、「日本舞踊」に少しでも興味がある方は是非とも気軽にご参加ください！

また、我々は「日本」を通じた親睦を深めたいという想いもあり、
参加者同士でお話頂ける時間もご用意しております。
茶道好き、日本好き、etcな方たちが集まりますので、同じ志向を持った人と繋がることも出来る場となっております。

ご参加いただける方は、右上の赤いボタンから申し込みをお願いいたします。
