本年も押し迫ってまいりましたが、皆様いかがお過ごしですか。

この度、appC cloudが交流会を開催する運びとなりました！

その名も…

「第１回 appC cloud 年忘れ! もぐもぐ会 !!」

この機会に是非、デベロッパー様やこれからアプリ制作をお考えの方々と親交を深めていただき、

情報交換の場として利用していただけたらと考えております。

とても簡単に言わせていただきますと、

食べて飲んでアプリについて語っちゃおうってことでございます！

もちろん、弊社スタッフも参加いたしますので参加者の皆様から貴重なご意見を
頂ければ幸いです。

皆様のご参加を心よりお待ちしています。

参加方法や開催場所、お問い合わせにつきましては、下記詳細をご確認ください。

＜第１回 appC 年忘れ! もぐもぐ会 !! 概要＞

開催日程：2014年12月22日　開始時間 20:00～(２時間ほどの予定)

参加費：2,400円(当日会場にて参加費をお支払いください)

＜プログラム(予定)＞

①ご自身のアプリを紹介！

紹介したい方は下記連絡先までメールをください。

当日、レビューサイトの方にもお越し頂くので、PRしちゃってください！

②ご歓談

もぐもぐ＆わいわいしましょう！　　　　

③「2015年アプリ業界の展望」

④appC cloud グッズの配布(ステッカー・バッジなど)

お土産としてお持ち帰りください。

定員：30名(最低開催人数は10人)参加希望者多数の場合は増員を予定しております。

※最低開催人数に達しない場合は、延期または中止とさせていただくことがございます。予めご了承下さい。

＜appC cloud とは？＞

アプリ開発者のための「appC cloud」無料サービスを提供させて頂いております。

「appC cloud」は従来のMbaaSをベースに、 アプリのマネタイズに必要な機能をもった環境を全て無料で提供するクラウドサービスです。

今後とも、より多くの個人デベロッパーの方々に知っていただきたいと考えております。 「appC cloud」https://app-c.net/
