WordBenchは WordPress ユーザーのための地域ソーシャルコミュニティです。
京都を中心としたコミュニティとして2013年度 第4回を開催します。
WordPress京都はWordPressの好きな人の集まりであって、リーダもなければ組織でもありません。
気軽にご参加ください。
WordBench京都
