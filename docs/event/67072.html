概要

スタートアップ企業6社ものCTOが福岡に集まり、勉強会を実施します。
各社の経験を元に、スタートアップにおける開発体制、開発フロー、開
発組織のマネジメントに関することなどを第１部のセッションでそれぞ
れ発表させて頂き、第2部ではそれぞれのCTOが一同に介してのディス
カッションを予定しております。

当日はこんな方々と現場の体験談を共有したいと思ってます！！

  • 起業を考えている方
  • スタートアップ企業で開発体制の構築に取り組まれている方
  • etc…
