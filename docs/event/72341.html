■このセミナーで、あなたの結婚力を飛躍的に上げることができます。

 

結婚力を上げると
未婚の方は、幸せな人生を共に生きる理想のパートナーを惹き付け、結婚出来ます。
既婚の方は、夫婦仲を理想な状態に改善し、末永い幸せな結婚生活を送れます。


このプログラムにより、多くの方が幸せな結婚をしています。
単なる結婚ではなくて、幸せな結婚です。

■瞑想で、心の状態を整えて、結婚力を上げる


瞑想が注目されています。
インテルやグーグルの最先端企業が、瞑想を社員教育に取り入れています。

「スタンフォードの自分を変える教室」で有名なケリー・マクゴニガル博士も著書の中で
「瞑想は脳を鍛えて意志力を強化するのに最適な方法」と書いています。

■瞑想で、自分を信頼する力を高め、結婚力を上げる


多くの方が、子供時代の心の傷により、自分には幸せは似合わないとネガティブな感情を抱いていますが、それに気付いていません。

瞑想の状態なると、潜在意識の扉を開け、そこに、プラスの思いを入れることが容易になります。
自分を素晴らしい存在と思えるので幸せな結婚が可能となります。

■幸せな結婚の豊富なノウハウ


ラディアンスの設立の趣意は、「催眠と瞑想で地球を癒す」です。
ラディアンスは多くの方に幸せになって頂くために設立して20年が経ちます。

ラディアンスでは、これまでに
5000名以上の方に4万回の催眠療法を施療し
1000名以上の方に瞑想法を伝え
100名以上のヒプノ・セラピストを輩出してきました。

その経験から、幸せな結婚をしている人、不幸な結婚をしている人、結婚ができない人
等の多くの方々にお会いし、それらの方々の心の記録を綿密にとってきました。

そこで、幸せな結婚をするためのノウハウをたくさん蓄積しています。

■幸せな結婚の実績


私自身も51歳のときに、21歳年下の妻と再婚しています。

そして、出会ってからこれまで一度も喧嘩したことがなく毎日幸せに暮らしています。
結婚相手により、ここまで人生の幸福感が違うのかと痛感しています。

私たちと同じ方法で、多くの方に幸せな結婚をしていただいています。

単なる結婚すら難しく、幸せな結婚は至難の業である今日この頃ですが、
実証済みの正しい方法をとると、幸せな結婚ができるのです。

あなたは、結婚とは何と考えていますか？
そして、人生とは何と考えていますか？

この一連のセミナーで、あなたの結婚観や人生観が変わっていきます

■セミナーで得られること


多くの方が、瞑想というとヨガや座禅という旧来の瞑想法を思い浮かべます。

旧来の瞑想法はどれも、宗教由来なので、修行色がとても強いものです。
今や入門は簡単なのですが、魅力的な瞑想の様々な効果を日常に活かせるようになるには
何十年もの苦行が必要です。

ラディアンスの瞑想法は、宗教とは無縁で、心理学や脳生理学の科学を基にしています。
そこで、短期間で瞑想の本来の効果を習得できる新しい瞑想法です。

今回は、新旧の瞑想法の体験ができます。

さらに、潜在意識の扉を開け、日々のストレスで疲弊してしまっている心の状態を整え、
すべては可能と思える自分への信頼感を高める瞑想します。

結婚力が高まります。
