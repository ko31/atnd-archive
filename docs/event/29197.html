description

Perlをもっとカジュアルに！

Hachioji.pm に対するオマージュです。
最初に全員でLT（ライトニングトーク）をした後に、
ビギナーズセッションで初心者の疑問をみんなで解決したいです。
5, 6人しか集まらない場合は、ビギナーズセッションに時間を回します。

月1開催（第3水曜日）で、気軽に立ち寄れる感じを目指してます。

LT（ライトニングトーク）とは？

短い時間（今回は5分）の中で、スライドを用いて何かお話して頂きます。
ですが、負担になるようでしたら自己紹介だけでも構いません！

例えば？

  • あなたのPerl歴について
    • プログラミングしたことがないレベル？
    • Perlは使ったことがないレベル？
    • Perl初心者を脱したいレベル？
  • Hokkaido.pm Casualでやって欲しいこと
    • Perlでやってみたいこと
    • Perlで困ってること
    • Perl以外で困っていること（フロントエンド、サーバサイド等）
  • 他には？
    • Perlでこんなことできるよ！
    • Perl以外のネタもOK！
    • ビギナーズセッションで話したいこととか！

それって、自己紹介じゃないの？

簡単な自己紹介から、スライドを用いたLTまで何でもありです！！

ビギナーズセッション

今回のテーマは、「Perlの開発環境」

  • あなたの開発環境を教えてください！
  • Macなら？Linuxなら？Win32なら？
  • その便利な機能は、どうやってカスタマイズしたの？
  • perlbrewって、ほんとに簡単なの？

この機会に、もっと便利な開発環境を手に入れましょう!
