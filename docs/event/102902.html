description

旭川市では働き方改革推進の一環である柔軟な働き方がしやすい環境整備のため、多くの方に「テレワーク」や「クラウドワーキング（インターネット上で仕事を受発注できるクラウドソーシングを使用した非雇用型の在宅ワーク）」に触れていただく機会として以下のとおりイベントを初めて開催いたします。

事業主の方、企業の従業員の方だけではなく、在宅ワークやモバイルワークの実施に興味のある方であればどなたでもお気軽に参加可能です。
地域の限定もないので、市外、道外からもお越しいただけます。

出展企業等の詳細については、市のホームページをご覧ください。
http://www.city.asahikawa.hokkaido.jp/kurashi/364/392/work0001/d064521.html

■とき
　平成31年1月31日（木）11時～17時

■ところ
　旭川市神楽４条６丁目
　旭川市地場産業振興センター「道の駅あさひかわ」１階　大展示場

■概要
　以下の取組を連続して開催します。
○テレワークエキスポ（11:00～17:00）
　テレワークやクラウドワーキングを行なう際に使用するテレビ会議、ビジネスチャット、リモートデスクトップ等の製品・サービスに関する展示会です。
　約20社が出展するブースを自由に回り、見て、触れて、各企業の担当の方と直接お話しをしていただける場となります。
　受付はありますが、入退場自由です。

○クラウドソーシング活用セミナー～そうだ！外注しよう！！～（11:00～12:30）
　企業等がクラウドソーシングサイトを使用し、業務を発注するメリット、活用事例、活用イメージワークショップのほか、旭川市クラウドワーカーチームの活動内容の紹介をします。【講師：株式会社クラウドワークス】

○インターネットを使った新しい働き方セミナー（13:30～15:00）
　自宅で空いた時間にできる働き方であるクラウドワーキングの紹介と、先輩ワーカー（旭川市クラウドワーカーチームのメンバー）の体験談の発表等を行ないます。旭川市クラウドワーカーチーム（３期生）への参加のご案内もいたします。
　託児サービス付きです。
【講師：株式会社クラウドワークス】

○働き方改革ロゴ・マークの来場者投票（11:00～16:00）
北海道立旭川高等技術専門学院の学生が作成した地域の働き方改革を象徴するロゴ・マークのうち、6作品を展示します。
　来場者の投票により、最優秀作品を決定し、今後の市の施策等に関連して使用させていただきます。

○テレワークエキスポ出展企業による（11:20～順次開催）
　出展企業のうち7社による30分程度のミニセミナーを連続して開催します。

■同時開催
同日2階の会議室で総務省主催「働き方改革セミナーin北海道」を旭川で初開催

■主催
　旭川市
