CloudFoundry輪読会:

Rubyで書かれたマルチ言語PaaSであるCloud Foundryについての勉強会です。

元々はCloud Foundryのソースを読もう！という会ですが、それだけでなく

  • Cloud Foundry上でこんなアプリケーションをデプロイしてみた！
  • Cloud Foundryの○○のコンポーネントを調べてみた！
  • Cloud Foundryじゃないけど▲▲というPaaSを触ってみた！

などの発表もあったりします。CFマスターな方からPaaSに興味があるのでCFについて知りたい、という入門者な方まで、自由にご参加ください。
