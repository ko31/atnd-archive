祝・Java EE 7 リリース！ いち早く新しい Java EE を知ろう！

「かんたん開発」で脚光を浴びた Java EE 6 リリースから3年半、「もっとかんたん開発」を実現した Java EE 7 がリリースされます。GlassFish Users Group Japan では Java EE 7 リリース直後に勉強会を開き、新しい Java EE の姿をいち早くのぞいてみようと思います。日本全国数ある Java 関連ユーザーグループの中で Java EE 7 の勉強会を開くのは、ひょっとしたら最初かもしれません。

今回は Draft 段階から Java EE 7 の各仕様をウォッチしていた方々にご登壇頂く予定です。新しい Java EE に興味のある方、これから Java EE を始めてみたい方、 どこかの誰かさんのデモ を見て GlassFish 4.0 に興味を持った方、ぜひこの機会にご参加ください。

勉強会の後は懇親会を予定しています。スピーカーと参加者が直接、じっくり話を出来る貴重な機会です。ご都合の付く方は ぜひ懇親会にもご参加ください 。
