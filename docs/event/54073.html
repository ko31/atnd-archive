description

世界文化遺産　
おもてなしの心　
日本人の多くが誇りに思う和食　
身近すぎて、考えたこともなかった"和食"って、なんだろう？
和食は、
美味しく
見た目にも美しく
カラダに優しく健康的
ぼんやりと、イメージではそんな風に思っていたけれど、
今まできちんと向き合ってこなかった。
そんな、貴方(女)がもう一歩深く和食に関われるイベントが
日本で禅が芽生えた地、鎌倉にて開かれます。
□　イベント進行役は、超宗派の僧侶達が集うウェブサイト「彼岸寺」創設メンバーで、料理僧として料理、食育に取り組む、青江覚峰さん。
□　昨年ユネスコの無形文化遺産にも選ばれた「和食」とは何かを青江覚峰さんから学びます。また、和食に関わる座学だけでなく、仏教の感嘆な体験を五感通して感じていただける企画も準備しております。
※　青江覚峰　氏　プロフィール
1977年東京生まれ。浄土真宗東本願寺派緑泉寺住職。カリフォルニア州立大学にてMBA取得。
著書に『お寺ごはん』（ディスカヴァー・トゥエンティワン）、『ほとけごはん』（中公新書ラクレ）などがある。日本初・ お寺発のブラインドレストラン「暗闇ごはん」代表。
上記のとおり、超宗派の僧侶達が集うウェブサイト「彼岸寺」創設メンバーのひとり。
