インスタグラムのショッピング機能「Shop Now」がついに日本でも導入！〜"興味関心で繋がる場"から"購買市場"へ〜

インスタグラムAPIのデータ分析などに基づき、アカウント運用代行やインフルエンサー施策をはじめ様々なインスタグラムマーケティングを専門的にやってきたノウハウと経験から、この新たなショッピング機能「Shop Now」とは何なのか、また、インスタグラムでどのような役割を果たすのかについて、実際に活用イメージができるようにアカウント運用全体の目線も踏まえながらお話します。

開催日程

日程 時間 会場
6/22 (金) 14:00 – 16:00 株式会社パスチャー第2会議室

※申し込みは2名/1社までとさせていただきます。
※開始5分前にはご来場いただきますようお願いいたします。

対象者

インスタグラムの現状を踏まえながら、「Shop Now」の導入とその運用について、初心者でもわかりやすくお話させていただければと思いますので、企業のマーケティング担当者の方のほか、インスタグラムアカウントを持ってない方や個人の方でもぜひご参加ください！

「こんな方におすすめ」
● ECに興味のある方
● インスタグラムをビジネスに取り入れたい方
● Shop Nowの導入について知りたい方
● Shop Nowの活用事例を知りたい方
● インスタグラムのアカウント運用について知りたい方
● インスタグラムのフォロワーを増やしたい方

コメント

日本でも「Shop Now」のテスト運用に参加したブランドから、すでに効果があったとの声が上がり始めており、インスタグラムビジネスと消費者の繋がりに大きな変化の波が起きつつあるのを感じています。国内で月間2,000万以上のアクティブユーザーが興味関心で繋がり、ユーザーが日々新たな発見を繰り返す場所で、シームレスなショッピング体験が可能となることに大きな期待を抱かずにはいられません。セミナーにご参加いただいた皆様が実際にインスタグラムの「Shop Now」を導入した際、アカウント運用全体でショッピング機能を最適化できるようなイメージが掴めるよう、少しでもお役に立てるお話ができればと思いますのでどうぞよろしくお願いいたします！
