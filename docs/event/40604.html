description

あらすじ

2013年3月に以下のようなツイートをしたら、賛同してくれる方が何名かいらっしゃったので、集まる場を作ることにしました。

@xtetsuji : 自分は昨年2012年のYAPCで初トークだったので、同じく昨年初トークだった人達と集まって飲みながら今年どうするかとか雑談する飲み会したい。誰か賛同してくれる人いるかな。
2013年3月12日 – 10:30

内容

  • YAPC::Asia トーク歴1年〜3年くらいまでの人が中心となって、今年2013年のYAPC::Asia Tokyo 2013で何を話したり何を盛り上げたりするか話し合う
  • Perlの話で盛り上がる

飲みながら、雑談の中で今年のYAPCについての展望を語ったり、何をトークしたりLTしたりしようか話し合う場です。

参加対象者

  • 昨年までのYAPCで1回〜3回程度トークやLTをしたけど、今年も何かやりたい。だけどネタに困っている人 (ここが今回の会のメインターゲットです)
  • まだYAPCでトークをしたことは無いけれど、今年は何かしゃべりたい。YAPCでトークした経験者に経験談を聞きたい
  • 「駆け出し」の人達にアドバイスをしてあげるよという「熟練YAPCトーカー」

場所に関しての注意点

会場が入っている新宿の「110ビル」は2013年5月31日にグランドオープンした新しい建物です。普段新宿に慣れている方も迷うかもしれません。リンクの地図とともに、以下のリンクから建物外観などを把握しておくとたどりつきやすいはずです。

  • 新宿西口　110ビル – NAVER まとめ
  • 廣和ビル,116ビル,110ビル｜NIKKOSHOJI CO.,LTD

場所はGoogleマップでは現在空き地になっている以下の場所です。緑の矢印が立っている場所。

新宿の地下1階空間を歩いて、6番か7番の看板の階段から上階に上がるとよいでしょう。道路の対岸にヨドバシカメラが見えるのが目印です。

備品について

現在、お店にプロジェクターを貸してくれるか交渉中です。もしプロジェクターを貸してくれることになれば、VGAケーブルとMac用のMiniDisplay変換コネクタは主催者側で用意しますので、昨年までのYAPCでトークしたスライドを大画面に映しだしたりして盛り上がることができます。 今回用のLTのようなものを用意してくださっても良いかもしれませんね！

→結局、他のお客さんとのプロジェクター争奪戦に負けてしまってお店からはプロジェクターを借りることが出来なかったのですが、 @ytnobody さんから借りられることになりました！ありがとうございます！

  • プロジェクターが借りれるかは6月17日月曜日に判明予定です
  • コースや飲み放題にはせず、各自が好きなものを注文して、幹事がアバウト割り勘をする予定です
  • ここを定期的にチェックしていただくか、 Twitter @xtetsuji をフォローしていただくことで、情報を随時流します
  • 要望や質問等ありましたら Twitter @xtetsuji に Mention か DM ください
  • 盛り上がったら2次会も

  • YAPC::Asia Tokyo 2013 の主催側とは関係の無い、 あくまで有志によるイベントです。
