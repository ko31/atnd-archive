★講演内容

関西に拠点を構えるゲーム企業にご登壇いただきます！
それぞれの企業が導入している技術やサービス向上に対しての
取り組みについてご登壇頂きます。

★ポノス社★
＜テーマ＞
Unity で Firebase™ を使ってみる

＜概要＞
・Firebase の Unity への導入方法
・Firebase Database への書き込み・読み込み方法
・Firebase Functions の使用・登録方法

＜学べること＞
・Firebase を例に挙げた mBaaS の簡単な仕組み
・Firebase の各機能の使用例

★クローバーラボ★
＜テーマ①＞
はじめてのElixir

＜概要＞
関数型プログラムも未経験のペチパーがElixirを使い感動した話

＜学べること＞
未経験でも分かるようにElixirの素晴らしさを布教いたします

＜テーマ②＞
サーバーと端末のリアルタイム連携（Web）

＜概要＞
データベースの情報を変える時点で端末も変えた通知をもらう方法

＜学べること＞
サーバー内部は端末に通知の実装が特に実装せずに楽に情報が同期できる


★講演者プロフィール

■ポノス社 にゃんこスタジオ
エンジニア

2015年新卒入社のプログラマー(4年目)の39歳。
他業種にて就労していましたが、
どうしてもゲーム業界に関わりたくて強引にプログラマーとして転職。
ポノス入社後、一貫してクライアントサイドのプログラムに従事していましたが、
サーバーサイドにも憧れを持っており、
入門的な意味合いでプライベートで Firebase を少しイジってます。
