5月27日（日）に韓国で実施される「AfreecaTV Star League（以下、ASL）Season 5」の決勝戦パブリックビューイングを、東京都中野にあるRed Bull Gaming Sphere Tokyoで実施します。
需要がありそうでしたら、昔のようにSC2の大会「GSLやWCS」などでも継続していければなぁと。RTSのパブリックビューイング楽しいよ！

If you are not Japanese but speak English, don’t worry and just come, event organizer speaks English and there is no reason in this world you should not be enjoying StarCraft. Email me anytime at kang2808[at]gmail.com if you have any questions.

実施内容

直前の告知ですが、5月20日だと思われていたASL Season5の決勝戦が27日であることが発覚したので急遽準備しました。プログラム自体はStarCraft1のものです。StarCraft2勢でも興味があればということで、気持ち程度に8台ほどプレイ可能なPCを現地に用意しています。

  • ASL Season5 決勝戦
    1. パブリックビューイング（みんなで観戦）
    2. Rain (P) vs Snow (P)
    3. 17時スタート
    4. タイトルはStarCraft1です
  • プレイエリア
    1. 15時頃から開放予定
    2. 8台程度のMSIゲーミングノート予定
    3. マウス等のデバイスは持ち込み推奨
  • 飲食
    1. 人数に応じたケータリング（食事）を準備予定
    2. BarCraftという名前ですがアルコール提供は残念ながらなし！（諸事情により）
    3. 会場が会場なのでRedBullは現地で飲めます
    4. 飲食の持ち込みはご遠慮ください
  • 参加費
    1. ケータリングやその他実施経費の補填のためにお一人あたり1,500円を参加費とさせていただきます
    2. 有効な学生証が提示可能な方は1,000円となります
    3. 会場到着時に現金にてお支払いをお願いいたします
