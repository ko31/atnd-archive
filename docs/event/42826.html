独学一週間で二級船舶免許を取得！

バリューイノベーションに入社して二日目。最初の仕事が二級船舶免許を取ることだった小沼です。面接時にはそんな話、聞いてないんだけど・・・。
南さん、中谷さんは約10万円かかるスクールに通って免許を取得。
私小沼は、3万円かからずに免許を取得しました。
最小限のエネルギーで合格した小沼の実体験を伝授する講座です。

■ボートライフは、意外とお安い！？

ボート遊びって、セレブ感ありまくり！ と思っていたのですが、友人とシェアすれば、かなりお安い料金で優雅な遊びが出来るんですよ。
1回居酒屋に行くお金があれば、海の上で優雅な休日を過ごせるんです！

■勉強したいけど、テキストを渡してもらえなかった

新しい会社に入社したばかりの緊張感と、覚えなきゃいけないこと、原稿書きなど気持ち的に切迫した日々を過ごしていた私が勉強期間1週間で合格しました。
どの程度、勉強時間に費やしたのか？ 忙しい人でも簡単に免許をとれるコツを教えます！

■いままで船に乗ったことないけど、実技試験を受けました

いままで小さな船に乗ったことがありません。操縦ってどうやるの？？？
そんな私が、どうしてボートを操縦できたのか。いや〜、初めて操縦するボートは気持ちよかったです！

「ボート=高い」と思っている方。
ボートは憧れだけど、現実的じゃないと思っていた方。
ボートに乗って、可愛い女子にモテたいと思っている方。
簡単に免許が取得できたら、優雅な遊びが待っています。
講座に参加すれば、楽しいボート仲間も出来ますよ〜！
