東京大学 Summer Founders Program 最終応募会 #3

この夏、東京大学 産学協創推進本部の新プログラムとして、技術プロジェクトや製品開発への支援を行う Summer Founders Program を行います。

本イベントは、Summer Founders Program へ応募を考えている人向けに、最後に応募フォームを埋める時間を作る会です。

参加登録せずにお越しいただいても結構です。

ご興味のある方、ご質問のある方、ぜひお気軽にご参加ください！

Summer Founders Program について

Summer Founders Program の仕組みは夏のインターンと似ていますが、大きく異なる点があるとすれば、それはプログラムに採択された皆さんは既にある会社のために働くのではなく、あなたの会社（になるかもしれないチーム）のために働く、という点です。

本プログラムに採択された場合、プロジェクトへ活動資金を提供するとともに、プロジェクトや製品開発を様々な形でサポートします。

詳細は 東京大学 産学協創推進本部 Summer Founders Program を参考にしてください。また 初期の Y Combinator などをご参照いただけばイメージがつきやすいかもしれません。
