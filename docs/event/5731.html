description 監修に面出薫氏(照明デザイナー)、竹村真一氏(文化人類学者/京都造形大学教授)、佐藤卓(グラフィックデザイナー)をお迎えした、学生主体・地域参加のキャンドルナイトです。 毎回イベント内容は少しずつ異なりますが、イベントの主旨をご理解頂いた施設や店舗のスペースをお借りした”キャンドルインスタレーション”や、周辺のカフェやレストラン店舗内でキャンドルを灯して頂く”キャンドルカフェネットワーク”、また神宮前小学校の小学生やご父兄の皆さまにキャンドルを持って表参道を行進して頂く”子供たちのキャンドルパレード”、そして個性豊かであたたかな行灯が表参道に集まる”オリジナル行灯”を企画中です。 東京近郊の美大生を主体とした実行委員を持ち、昨年は当日482名ものスタッフが2時間のキャンドルナイトを盛り上げ、多くの来街者を迎えることができました。 ぜひ当日は足をお運びください!
