★講演者プロフィール

【LT登壇者】
■鈴木 孝之 氏(株式会社Re:Build)

1990年神奈川県生まれ。多摩大学経営情報学部卒。
文系の学部を卒業後、2013年に東京のITベンチャー企業に
システムエンジニアとして入社し、プログラミングを習得。
その後、株式会社フルスピードにwebエンジニアとして入社。
フルスピードでは、大規模な広告配信サービスの管理画面開発を担当し、
サーバサイド(php)からフロントエンド(JavaScript)まで幅広い開発に従事。
2017年11月に沖縄で起業し、株式会社Re:Buildを設立。
東京の企業様からの開発案件を請けつつ、自社Webサービスを開発中。

＜LTテーマ＞
～Laravel5.5で作る管理画面爆速開発～
リソースが少ないスタートアップ企業で、
どのように工夫してWebサービスの開発効率化を行ったかをお話します。


■岡田 正平 氏（株式会社ウィルゲート）

2015年4月、株式会社ウィルゲートに新卒で入社。
コンテンツマーケティング事業の拡大をより加速させるため、
開発チームを技術面でリードする役割を担う。
また、社内のエンジニア組織の文化を作るべく勉強会を積極的に開催中。

参考:http://tech.willgate.co.jp/entry/3475/

＜LTテーマ＞
～チームで「きちんと」 Laravel を使っていくための取り組み～
美しく、シンプルなコードを実現するための多様な機能を提供している Laravel。
それらの機能を把握し、使いこなすには一定の経験と知識が必要になります。
そんな Laravel を、多様な背景を持つメンバーどうしで
「きちんと」使っていくための取り組みをお話します。


■卜部 真一 氏（株式会社オズビジョン）

株式会社オズビジョン 2017年9月入社
Hapitas (hapitas.jp) の開発、保守・運用業務を主に行っています。
主なフレームワークは Codeigniter, Zend Framework, Symfony です。

Blog: https://tech.nuts-choco.com

＜LTテーマ＞
～PHPにおけるWEBセキュリティの話、各フレームワークごとの対策～


■田中 孝治 氏（BASE株式会社）

BASE Product Division サーバーサイドエンジニア
2017年9月入社
WebアプリケーションのPHP, フレームワークのバージョンアップを主に担当しています。
1年くらいCakePHP コードとドキュメントにプルリクエスト送ったりしてます。

Blog: http://tenkoma.hatenablog.com
GitHub: https://github.com/tenkoma

＜LTテーマ＞
～BASE の PHPアプリケーション バージョンアップについて～
サービス開始から6年目の BASE は PHP、CakePHP を利用して構築しています。
当初 PHP 5 & CakePHP 2 で作られたアプリケーションをバージョンアップする
取り組みについてお話します。
