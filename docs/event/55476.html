description

ナノファイバーが注目される中、エレクトロスピニンング法は室温、低エネルギーでナノファイバーを作製することが可能な方法として注目され続けてきた。日本をはじめ、アメリカ、ヨーロッパ、韓国、中国などからも多くの研究が報告されている。講座では初めてエレクトロスピンングに取りかかろうとする人にもわかりやすく、ナノファイバーの作製方法を講義する。そして基礎から最先端までを１日の講義で理解できる内容である。さらに実際にこれからナノファイバー作製に取りかかる場合にどのような装置を用いればよいのか、どのように工夫すれば目的のナノファイバーを得ることができるかを実習を交えて原理から製造までを理解できるようにする。
プログラム
１．ナノファイバーの利点
２．エレクトロスピニング法によるナノファイバー作製の原理
３．世界のナノファイバー研究
４．ナノファイバーの応用
　　4-1　PM2.5対応マスクなどのフィルター用途
　　4-2　人工血管やドラッグデリバリー用途
　　4-3　気体や液体の分離膜 セパレータや透湿撥水膜の用途
