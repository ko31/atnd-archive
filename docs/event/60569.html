各自が自身の作業をする作業会ですが、分からないことやハマったことがあれば参加者間で相談し合ったりして進めて参ります。

PageSpeed Insights は、モバイル向けのユーザビリティ診断等も同時にしてくれるので、スマートフォン対応もきっちりとやっていきたい方にもオススメです。

Webサイト制作に手慣れていない方も、基本的な更新作業ができれば大丈夫です。改善は簡単にできることから難易度の高いものまで色々とあるので、できることからやっていきましょう。

ノートパソコンは各自ご持参ください。Wi-Fi環境は用意しますのでインターネット接続できます。

事前に下記"PageSpeed Insights(Chrome拡張 or Webサービス)"でご自身のウェブサイトをチェックしてみてください。何も指摘されなかった方は、今回のワークショップはする作業がないので参加する意味はないかと思います。

  • PageSpeed Insights for Google Chrome(Chrome拡張) (※ Chrome ブラウザのインストールが必要)
  • PageSpeed Insights(Webサービス)
