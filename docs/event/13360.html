すいません！日付間違ってました！今日です！26日です！

昨日今日で行われているヤングカンヌ サイバー部門の日本予選参加者で集まる会、的な。

なんとなく自分（ @6tn ）の周りの友人が結構な数（１０組とか）予選に参加していまして、
今日、提出後、互いの提出した作品を見ながら、集まって酒でも飲もう的な感じです。

で、せっかくなので、自分たちの知り合いに限らず、今年予選に参加したひとで集まって、
交流しつつ、作品見つつ的な睡魔と闘うおつかれ会でできたら素敵だなと思い。
そんな会に参加したい方がいましたら、是非、合流してください！

新宿南口で２１：００あたりからで考えています！

参加条件
-今年のヤングカンヌのサイバー部門の国内予選に出ている方。今。まさに。
-おつかれ会で、自分たちの作った作品を見せてもOKな方。

パートナーと是非、ご一緒に参加ください！
あと知り合いの出場者とかにも声かけてカモン。

場所は、新宿南口で、友人と集まって借りているシェアオフィスです。
住所を見て来てください。（渋谷区千駄ヶ谷5-33-2）
なんか酒とか一品持ち寄りできたらナイスです。

場所がわからなかったら、 @6tn にmentionお願いします！

あと、遠隔地にいて、リアルには参加できないけど、自分たちの作品見せてもOKで他人の作品見たい的なひとのために、なんか、送ってくれたらまとめてこっちから送る、みたいなのができたらいいな、とか思ってます！
