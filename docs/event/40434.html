#03 Git 入門

Gitは今や欠かせない開発ツールになっていると思っています。まだ使い方を覚えていない方を対象にし、実務でとりあえず対応できるように知識を吸収して帰ってもらえれば。#01で開催したGit入門をブラッシュアップして、県内のプログラマ、デザイナ、コーダーさんが集まりやすい高崎で再度開催します。

またまた講師役は @yterajima です。

ゲスト

デザイナさんgitを実際に使用している フィヨルド の @machida さんが来てくれます。デザイナ視点からのgitの利用方法等についてお話いただく予定です。

内容

Git の基本的なコマンドの使い方、複数人での作業の行い方、github の活用方法をざっくり3時間くらいで実際にCLI(黒い画面)でコマンドを叩きながら、 セミナー形式 で学びます。とりあえず使ってみて残りはググって解決できるようになるのが目標です。

CLIの操作に不慣れであっても問題なく学習できるように、サポートの方を用意する予定です。

Windows ユーザの方

以下のパッケージをインストールしてください。それぞれ今回の内容では問題なく使用できます。

  • Github for Windows
  • msysgit

エディタに vim を使用したい場合には msysgit がついてきます。また, Linux の操作になれているような場合にもいいかと思います。その他の場合には, Github for Windows をインストールしていただければ問題ありません。

Mac ユーザの方

Mac の場合には, Homebrew または MacPorts で Git をインストールしてください。

Linux ユーザの方

パッケージ管理ツール等を用いて git を入れてきて下さい。
