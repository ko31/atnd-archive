“NPO TOKYO SKILL UPs”とは？

　私たち"NPO TOKYO SKILL UPs"は、2010年4月から始まった、まだ若いNPOです。
「若者の 若者による 若者のための『ビジネススキル研修』」の実施を通して、「人材育成」「人材交流」「社会貢献」を達成したいと考えます。

メンバーは大学1年生から、30歳までの若手社会人まで、大学生・専門学校生・会社員(技術職・営業職など職種も多数！)・公務員と、
多種多様な方にご参加いただいています。
一般的な「ビジネススキル研修」は、参加費が非常に高かったり、単純な講義を中心としていたり、同じ業種の人だけが集まったりすることが
多いのではないでしょうか？　私たちは、会員が力を合わせて運営することで、それらの問題を解決しています。

"キャリアスタートセミナー"とは？

これからの就職を目指す方も、社会で活躍されている方も、一緒に学ぶ場として開設された「ビジネススキル研修」です。
この研修では、サークルのような雰囲気の中で、オリジナルのグループワークやロールプレイを使った研修が受講できます。

テーマは"Management"、"ProbremSolving"、"Marketing"、"Communication"の4つのカテゴリーごとに分かれています。
どのカテゴリーのどの講座からでも参加できますので、安心してご参加ください。
また、グループやサークル単位での開講もできますので、ご希望の方は事務局までご連絡いただければ幸いです。
私たちの手作りの研修ですが、自信を持ってオススメします。
一緒にビジネススキルを学んでみませんか？
