description

アンケートは右上の赤いボタンからお答えいただけます！

ゲーム業界への就職を希望する学生や転職希望者が、業務内容や採用情報について、採用担当者から詳しく聞いたり直接質問ができるイベントです。

いますぐの就職・転職をお考えでない場合も、業界や会社についてご興味があるかたは、お気軽にご来場ください。

なお、本アンケートは来場人数把握のため設置したもので、当日ご参加において必須のものではございません。またアンケートの有無、回答などが採用に影響することはございませんので、ご安心ください。

【日時】
2014年3月8日（土）9:30〜12:00　※9:20〜開場、9:30〜各企業オープニングトーク、10:00頃〜ブース説明会

【会場】　
京都市勧業館（みやこめっせ） 地階　第一展示場

【参加対象企業】
ゲーム制作に関わる企業

【対象参加者】　
・学生（大学、大学院、短大、専門学校の2014年・2015年3月卒業予定者）
・既卒者、転職希望者

【参加予定企業】※50音順　
株式会社 エイティング
株式会社 Aiming
日本一ソフトウェア

・京都求人限定ブース
京都エンタテインメントワークス株式会社
株式会社D4A
株式会社モバイル＆ゲームスタジオ

【募集職種】
企画運営、ゲーム開発エンジニア、インフラエンジニア、グラフィックデザイナー、プログラマー、ディレクター、プランナー、営業職、管理（総務、経理）職　等

【備考】
入退場自由。スーツ着用での来場はご遠慮ください。

【担当者より】
上記職種以外にも、様々な職種、雇用体系で採用をお考えの企業様もあります。正社員だけでなく、アルバイトやパートや契約社員、技術職だけでなく、営業や総合職、といった具合です。
本説明会はあくまで、業界を目指して勉強してきた学生さんや、経験者のための就職説明会ですが、未経験だけどとにかくゲーム業界で一度は働いてみたい、というかたも、諦めずにお越しください。
