新開発石窯を使って測定、制御、食べるワークショップ
「おいしいプログラム開発」
RaspberryPiを制御しておいしいおやつをつくろう！

Raspberry Pi石窯 DELiveを使ってパイを焼きます。
DELiveは、断熱性、耐熱性に優れた素材を使って机の上に置けるモバイル石窯。
通常のコンセントの電力で最大500℃。Raspberry Piでモニター&制御して、データはスマホで見える！
この石釜を、自分の好みの言語でプログラムをつくって制御し、お菓子をつくろう！
自分のパソコンから石釜にsshでログインして、自分のプログラムを作ります。
もちろん、おいしく焼けたパイは楽しくいただきながら、ワークショップを行います！
