夏なのでビアガーデンに行きたいなー。
どうせ行くなら高尾山で夜景を観ながらビールを飲んでみたいなーと思って企画しました！

＜諸注意＞

  • 雨天中止(都内で飲みましょう・・・)。
  • 遠いですが、興味ある方はご自由にどうぞ。
  • 遠方から参加される方を考慮して早めに待ち合わせて並ぼうと思います。
  • 夏の山の格好、日差し対策など、各自注意しましょう〜。
  • 入れ次第2時間制なので、時間の指定が出来ませんがご了承下さい。
  • 詳しくはビアマウントのページを見て下さい。

料金は、
男性：3300円
女性：3000円
で、飲み放題・食べ放題の2時間制バイキングです！
＊2時間を超えた時点で500円の超過料金が発生するみたいです。

当日の流れ

  1. 京王線 高尾山口駅に15:00に待ち合わせ
    (新宿からの方は、新宿で待ち合わせて一緒に向かうと安心？かと。)
  2. ケーブルカー 清滝駅に向かい、ケーブルカーに乗ります。
  3. ケーブルカー 高尾山駅で降りて、高尾山ビアマウントへ
  4. 案内され次第2時間満喫し、帰宅します。

面識ある方に、直接お声掛けて企画したので、多少適当なところもありますが
暑い夏に皆で、都心から離れた標高500mの場所で飲むビールは格別だと思ってます！
