description

ひつじ４０人の会、予想をはるかに上回る反響をいただき、
無事開催できそうです。ありがとうございます。
希望者が殺到したことと、管理者がATNDのシステムをよく理解していかったため、多少きつめの配席となるもようですが、ご理解ください。
（キャンセル待ちでご参加いただけない方には、お詫び申し上げます）
なお、入場は４時４５分から準備しております。
場所が若干わかりづらいのですが、
歌舞伎町の大久保公園（バスケコートとかある場所）の東北角です。
とてもレストランなどなさそうな西洋風ビルの２階です。
恐れず２階に上がってください。
それでは楽しみにしています。

＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠＠

羊のお肉って美味しいですよね。独特のうまみがあるし、油もヘルシーだし、毎日でも食べられますよね。
そんなひつじ好きな方を４０人を集めて１頭丸ごと料理して食べるモンゴル料理の会を開催します。
１頭の羊が、チャンサンマハ（羊肉の塩ゆで）、串焼き、スープ、ボーズ（肉まん）など各種料理になって次々と出てきます。
興味のある人には、解体ショーもかぶりつきで見られます。
４時間食べ放題飲み放題で５０００円という４０人ならではの値段設定。
料理に時間がかかるので、５時スタートでメインディッシュが出てくるのは７時ごろと思われます。
「４０人いないと食べきれない」というお店の指摘で、４０人集まらないと１頭準備してもらえません。
なんとか集めたいと思っています。
皆さん、お誘い合わせのうえ、ご登録、ご参加下さい！
