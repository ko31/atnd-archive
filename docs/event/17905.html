【内容紹介】
（7月4日に実施した勉強会を、好評につき再度同じ内容で実施します！）

AndroidとiPhoneの大きな２つのプラットフォームに対して、サービスを開発していきたいと考えている方は多いと思います。
２つのプラットフォームにいかに簡単にアプリを提供するかについてが問題になってきますが、その問題の解決で注目されるフレームワーク”Titanium Mobile”があります。

今回は「スマートフォンのアプリをはじめてだけど作ってみたい」という
ＷＥＢサービス開発者の方を対象に、Titaniumをはじめて使って
デモアプリ開発するまでを、Titaniumのエバンジェリストとして著名な
増井雄一郎氏に指導してもらえます。※勉強会の最後は簡単な懇親会も実施します！

※Titanium Mobileについて・・・公式サイトはこちら
Titanium Mobileはベースで、iPhone/Androidなどスマートフォン向けのネイティブアプリをJavascriptベースで構築できるフレー
ムワークです。

【対象】
Android / iPhoneの開発に興味があるWebサービス開発者全般
※Java/PHPを学んだことのある方
※上記に該当すればWebサービス以外のエンジニアの方も参加頂いても構いません。
※7月4日の第1回に参加された方はご遠慮下さい

【参加にあたり準備頂くもの】
↓↓必ずTitanium studioの以下設定を事前にお願いします↓↓
１．Windows又はMacOSのノートパソコン
２．Titanium studioを事前インストールをお願いします。
　　・http://www.appcelerator.com/ にアクセス。
　　※インストール後、iPhone OS SDK又はAndroid SDKのどちらかをインストールして下さい。SDKのダウンロードは公式サイトからできます。
３．Titanium Mobile Kitchen Sink Demoの事前インストールをお願いします。
　　・https://github.com/appcelerator/KitchenSink にアクセスしてダウンロードをして下さい。
　　※上記の事前準備が難しくてできなかった場合は、会場に15分前にお越し頂ければこちらでサポートすることができます。
　　※参考サイトhttp://d.hatena.ne.jp/kaz_konno/20110614/1308075564
　　http://t.co/02NZYanが推薦図書です。
　※当日会場には無線LAN環境を用意します。
