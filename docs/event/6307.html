title 日本初！Javaチップを使ってTwitterボットを作ろう
catch
description 
みなさんは<strong>Javaチップ</strong>というものをご存知でしょうか？
特別なJVM？アクセラレーション？
いいえ違います、Javaバイトコードをネイティブに直接実行する世界で唯一の技術です！
そんな素晴らしいJavaチップを使った日本で初めてのプロトタイピングボード「<strong>chabaco</strong>」を使って、
一足先に次世代ガジェットクリエーターへの道へ進んでみませんか？
<br> 
講座内容
参加者の皆さんには、「chabaco」を使って「Twitterにランダムな文章をつぶやくガジェット」を作っていただきます。
SDカードの中に書き込んだテキストデータを「chabaco」があなたのアカウント（または別のアカウント）を使って
ランダムにつぶやきます。
加えて、接続された温度センサの値もランダムにつぶやきます。もちろん<strong>OAuthに対応！</strong>
（xAuthではなく、他のサーバーも必要としない、完全に単独での認証です。）<strong>
Javaチップの機能をフルに使った</strong>プログラミングが体験できます。
<br>
参加資格
Javaの知識がある程度ある方。（JavaSEが理解できればOK。JavaMEが解れば尚良し。）
ハードウェアの知識はなくてOK！
<strong>Windows32ビットOSを搭載したノートパソコンを持参できる方。</strong>
（32ビットであればWindows7でも可。または64ビットOS + VMWare上32ビットでも可）
JTAGを使う際のツールの対応状況が理由です。2台までなら貸出します。
Javaチップを<strong>誰よりも早く</strong>使いたい人！（<strong>これ重要</strong>）
