場所が変更(タイムインターメディア→早稲田奉仕園)になりました、ご注意ください!!持ち物にも追加があります!!

会場へのアクセス

  • 早稲田奉仕園への地図と構内案内図 より
  • 東京メトロ東西線早稲田駅より(徒歩約5分)
  • JR山手線・西武新宿線　高田馬場駅より　バスを利用する場合(所要時間約10分)
    「学02 早大正門行」行きのバスにのって「西早稲田」で下車してください。
  • 東京メトロ副都心線　西早稲田駅より(徒歩約8分)

この勉強会について

Zope/Ploneの開発者で集まってもくもくと開発したり色々相談したりとかやろうと思います。
(といいつつ実際は Python 関連を中心になんでもありな感じです)
スプリントのゆるい版みたいな感じで開催しています。6月の第0回から毎月開催しています。参加費は無料です。
もってくるもの

  • PC
  • マイカップ(できれば)
  • LANケーブル(持ってくると安心)
  • 電源タップ(できれば)

多少ですが、飲み物とジュースとおやつが出ます。

会場には無線/有線LANがあります。
Zope/Plone とかの開発に興味の有る方は、是非ともこの機会に一緒に勉強しましょう。
