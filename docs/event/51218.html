description

趣旨
皮膚科学の基礎知識と化粧品コンセプトのつながりを理解し、化粧品の有用性を達成するための基本的な考え方を取得することを目的とする。

プログラム
1．皮膚の構造と機能

2．皮膚の保湿に関連する因子と機能の紹介
　　2-1　角層細胞　コーニファイドセルエンベロープ
　　2-2　第二の表皮バリア　角層細胞間脂質のラメラ構造体
　　2-3　第二の表皮バリア　タイトジャンクション
　　2-4　表皮細胞の水チャンネル　アクアポリン3
　　2-5　天然保湿因子とフィラグリン
　　2-6　保湿効果の評価法

3．皮膚の酸化ストレス
　　3-1　紫外線による活性酸素の産生
　　3-2　SLSによる活性酸素の産生
　　3-3　糖化生成物による活性酸素の産生
　　3-4　赤外線による活性酸素の産生
　　3-5　抗酸化ストレス評価法

4．美白効果を実現するためのメラニン産生メカニズム
　　4-1　メラニン産生の基礎的な情報
　　4-2　紫外線によるメラニン産生
　　4-3　炎症によるメラニン産生
　　4-4　メラノソームの表皮への移送
　　4-5　表皮内部でのメラノソームの分解
　　4-6　真皮からのメラニン産生刺激メカニズム
　　4-6　美白効果の評価法
　
5．抗老化効果を実現するための老化メカニズム
　　5-1　皮膚老化　生理的老化と光老化
　　5-2　コラーゲン線維の変化に焦点をあてたシワ形成メカニズム
　　5-3　エラスチン線維の変化に焦点をあてたシワ形成メカニズム
　　5-4　血管、リンパ管の変化に焦点をあてたシワ形成メカニズム
　　5-5　乾燥によるシワ形成メカニズム
　　5-6　抗シワ効果の評価法
