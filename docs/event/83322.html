イベント概要

Ubuntu 16.10のリリースを記念したオフラインミーティングイベントです。

今回は特にテーマを設けていません。16.10に関する話、17.04に向けての豊富、普段使っているUbuntuに関する話、Ubuntuとは直接は関係ないFLOSSの話などをネタに、普段出会わない人と気軽に歓談しませんか。

前方のセミナー、後方のハッカソンという雑然とした空気とともに、Ubuntuの魅力を語って頂ければ幸いです。

参加費・参加資格等

参加費は無料です。ささやかではありますが、無料の食べ物とからあげと飲み物を用意してお待ちしております。

Ubuntuを使ったことがある、あるいは興味がある場合など、どんな方でも来て頂いて問題ありません。

詳細は、https://wiki.ubuntulinux.jp/Events/OfflineTokyo201612 を参照してください。

会場へのアクセス方法

  • 会場までのアクセス方法： http://corp.gree.net/jp/ja/corporate/location/

会場では入館手続きが必要となります。詳しい手続き方法は、開催日時が近づき次第お伝えします。

セミナーについて

セミナーの登壇者を募集しています。発表したい方は以下の情報を揃えた上で、本イベントのコメント欄に記載するか contact@ubuntulinux.jp 宛までご連絡ください。

  • イベントページのセミナー一覧に掲載する発表者名

  • 希望時間（大雑把で可）
  • 希望するタイミング（イベントの最初の方、後ろの方、など）

原則として先着順となります。

当日のUST配信は、 http://www.ustream.tv/channel/nSAVy95kheb です。
