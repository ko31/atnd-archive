title 癒しフェア2014 in TOKYO
catch 日本中の「癒し」が集まるビッグイベント
description

「癒しフェア」は今年で19回目を迎える、日本最大の癒しの祭典です。
その特徴は、さまざまな癒し関連の商品やサービスが実際に体感できること。
オーガニックコスメやアロマの販売、ヨガなどのパフォーマンス、ヒーリング＆占い
そして癒しに関わる芸能人・著名人のトークショーや講演も開催します。
まさに女性のための“美”と“癒し”が凝縮された2日間。
ぜひ、心も体も癒されてキレイになってください。
