みなさま、こんにちは♪
いきなりですが、イスラエル・パレスチナ関係者さんが集まって一緒にご飯食べるイベントを
5月に都内で行いますよー！
学生さんから社会人さん、留学経験者さんからジャーナリストさんまで、
ぜひ交流＆情報交換＆ホンモスを食べにいらっしゃってくださいませ♪

イスラエル・パレスチナ関係の企画・イベント告知もあるかも…？
オープンマイクしますので、自分が企画中の何かに人を巻き込みたい方もどうぞ。笑

こんな人にオススメ

  • イスラエル、パレスチナに滞在していた。
  • イスラエル、パレスチナを研究している。
  • イスラエル、パレスチナ人と結婚している、付き合っている。
  • むしろイスラエル、パレスチナ人は義兄弟である。
  • っていうか自分イスラエル人／パレスチナ人だし。
  • 主催者、参加者（右リスト参照）の関係者である（ゆるい・笑）
  • 「これから初めて現地に行く予定なんですけれども／興味が有るんですけれども…参加できますか？」という方は、まずは主催者、参加者とコンタクトを取ってくださいませ。関係者になってください。笑

必須の持ち物

1. 参加費500円（ホンモス・ファラーフェル代に回されます）
2. マイカップ、マイ箸、マイ皿（エコ推奨）
3. 参加者の皆さんとシェアしたい食べ物or飲み物1品以上（アルコール、ソフトドリンク）
4. オープンマインド

主催側が用意するもの

  • ホンモス／フムス
  • ファラーフェル（どっかでケータリング予定。誰か作れます？）
  • サラダ（予定）
  • シャーイ／テ（予定）

注意事項

  • 雨天延期（このページで当日朝10時に告知します）
  • 代々木公園の原宿口を入り、すぐ右側あたりの原っぱでどんちゃんします。
    地図はこちら
  • ケンカ、一方通行の議論はご遠慮ください。いろんな考え方があります。念のため。
