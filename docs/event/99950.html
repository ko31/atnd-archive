description

BUDDYZでは、広告・PR・ITデジタル業界で働く皆さまが多くの人と出会い、業界自体がますます発展するように、定期的にネットワーキングパーティーを開催いたします。

今回の開催は、BUDDYZを運営する「株式会社ホールハート」の学生向けサービス「シンアド就活（就職支援）」のイベントです。ベンチャー・スタートアップに興味があり、熱意ある大学生も参加します。「デジタルネイティブ」の世界で育った今の学生は、社会人とはまた違った視点で、同業界を見つめています。彼らのフレッシュな感覚が、ビジネスのヒントになることもあるかもしれません。

WeWork Icebergで初開催

横のつながりをつくるのもよし、マーケティングやリクルーティングの一環として学生とコミュニケーションをとるのもよし。ご興味ありましたらお知り合いをお誘いのうえ、ぜひお越しください。

皆さまにお会いできるのを楽しみにしております。
