description

2月2日（木）から土日の4,5日をはさんで７日（火）まで床張り訓練講座を行います。

場所は和歌山県熊野川町の木造校舎。
世界遺産の熊野本宮大社の近くです。

田舎での必須スキルの一つ、それは床張りです。
なぜなら格安物件を借りるとしてもだいたい床が腐っていることが多いからです。

無傷の家は当然売りにだされるにしても高いです。
そんなんやったら田舎に住む意味は少ない。
家を自前で改修できれば、杉板は手に入りやすいので低コストで
快適な家を保持できます。

場所によっては一軒家の家賃は月千円単位だったりしますので
この時代に床張りができるのは非常に重要な技術になります。
なんなら副業にもできます。

かといってプロの大工ほどの修行期間は必要ありません。
日本家屋を一人で立てるわけではありませんので。
6日間みっちりやれば、もう忘れることはないでしょう。

つまり、床張りは習得するコストに比べてかなりの大きな自由度が
得られる技術である、ということです。
お得である、ということです。

ということで、みっちり忘れられないぐらい床張り訓練を
しますので田舎に住む予定がある方はぜひ参加ください。
むしろ床張り技能の習得なくして田舎暮らしはない！ぐらいの意気込みです。

今回は杉板を使用する予定です。
なぜ杉板なのか？ということですが、柔らかく初心者にも加工しやすく低コストだから。
柔らかく傷がつきやすいという弱点もありますが、逆に言えば当たっても
痛みが少ない、ということで幼稚園の床に使う例もあります。

そして日本全国で植えて生産しているので田舎なら手に入りやすい。
自分で運ぶんなら無料でもいい、ぐらいのところもあります。

この講座では技術の訓練はもちろん、床板張りや簡単な電気工事など
小さな施工業を生業にしている齋藤さんを講師に行うので、
杉板の入手方法や果てはナリワイのつくり方のヒントを
得ることができるでしょう。

ちなみに齋藤さんは、素人から自分で覚えて自分で家を建てました。
田舎で少々農業ができて、大工仕事ができればそれだけで
暮らしを建てていくことはできるようになります。

逆にローン組んで家を建てたりしてしまうとちょっと
大変かも。そんなあたりの実情もワークショップ期間中に
先達に聞くことができるかと思います。
