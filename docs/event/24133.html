description

Wakame-VDCの機能を全て理解できる会です

今回は、GitHubで公開されている最新版のWakame-VDCのコードを皆さんと一緒に追いかけていき、そこに備わっている機能を見ていこうと言う企画です。
テーブルを囲んで、皆様と一緒に、Axshのエンジニアを交えて質問に答えながらインタラクティブにやっていきます。
トイレ休憩なども各自適宜取りながら進めていきましょう。

内容や参加人数の上限などは常時変更されることがございます。

見所

  1. 簡単な自己紹介と、ゲスト紹介、本日のおやつの紹介
  2. あのファイルはシステム構成上のアレ
  3. 謎のデータベーススキーマを解明
  4. ここでまさかのWeb API
  5. 管理者のためのコマン道vdc-manageとは
  6. どうして起動するのか仮想サーバ
  7. どうしてこうなった仮想ストレージ
  8. どうにかなりそう仮想ネットワーク(OpenFlowだとか)

コミットの仕方実演

  1. インストールしてみる
  2. 改変してみる
  3. GitHubでPullリクエストを出してみる

参加資格

今回は、下記いずれかに当てはまる方に限定いたします。

  • Wakame-VDCを一度でもインストールしたことがある
  • GitHubのソースコードは眺めてみたことがある
  • 自力でプロダクトに手を加えられてこそのオープンソースであると信じている
