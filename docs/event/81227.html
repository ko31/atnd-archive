IIJ GIOアカデミー九州初上陸！！

クラウド全般の技術情報の発信を目的に、東京で定期開催しているIIJ GIOアカデミーですが、多くのご要望をいただきまして九州で開催することになりました。 記念すべき初のコンテンツは弊社クラウドサービス「IIJ GIOインフラストラクチャーP2」のご紹介とハンズオンです。本サービスは気軽にオンラインで導入いただけるのが特長で、今回は仮想サーバを中心としたパブリッククラウドのサービスについて、実際に仮想サーバの契約から起動、設定変更などハンズオンで体験していただきます。
IIJ GIO P2や他社クラウドの導入を検討中の方、この機会に是非ご参加ください。

ご参加にあたって

  • 定員を超えていてもキャンセルがあった場合は繰り上がりで参加可能です。
  • ATNDにメールアドレスを登録していない場合、繰り上がりがあっても通知されません。ATNDにメールアドレスを登録するか、時々このページで繰り上がり有無のご確認をお願いします。
  • 当日参加が不可能になった方は、キャンセル待ちの方のためにキャンセル登録のご協力をお願いします。

プログラム

18:30～19:00 30分 受付開始

博多祇園M-SQUARE入館後、3Fにお上がりいただき、セミナー受付へお越しください。受付時にはATNDユーザ名の確認とお名刺を1枚頂戴いたしますので予めご準備ください。
19:00～19:20 20分 IIJ GIO P2サービス紹介

ハンズオンを実施する前に、簡単にサービスをご紹介します。
19:20～19:30 10分 質疑応答
19:30～21:00 90分 いざ！ハンズオン！！

  • オプション付与＆契約変更 など
