【講座のポイント】
　混練機の誕生から現在に至る変遷について説明し，何故現在のような混練機／押出機の構造になったかについて解説する。また，近年，開発されている特殊セグメントについても解説する。
　さらに，現在注目されているナノコンポジットに関して，世の中の動向を説明し，現在進めている研究内容の一端を解説する。

【プログラム】

１．混練の概念

２．混練機・混練技術の変遷
　　2-1．バッチ式混練機
　　2-2．二軸連続混練機（非嚙合い型異方向回転式）
　　2-3．二軸連続混練押出機（嚙合い型同方向回転式）

３．装置の大型化とエネルギーバランス

４．混練押出し技術の最近の動向
　　4-1．ナノコンポジット（現在進行中の研究途中結果）
　　4-2．二軸混練押出技術によるナノコンポジット製造
　　4-3．射出成形技術によるナノコンポジット製造
