description

昨今において多くのユーザーに長く愛されるスマートフォンアプリをつくる事は、簡単ではありません。良いアプリをつくるのはもちろんの事、マーケティング、運用改善、品質管理、カスタマーサポート、ツール活用などさまざまな方法を通じて顧客満足度を上げ、継続率や課金率などを高めることが欠かせません。

本セミナーでは、『すぐに真似したくなるアプリ運営の㊙ノウハウ』をテーマに、アプリ内課金の必勝テクニックについてや、アプリのレーティングを★5にする為にできることについて、事例やデータを交えてご説明いたします。

セミナー内容

■<講演1>購入データから導き出すアプリ内課金の必勝テクニック

登壇者：後藤 崇　- カイト株式会社　代表取締役CEO

itemstoreを提供しているカイト株式会社が、アプリ内課金の必勝テクニックをお伝えします。広告設置以上に難しいと思われているアプリ内課金は、大手アプリ開発会社ではマネタイズの中心になっていますが、小規模開発会社ならびに個人開発者はまだまだの状況です。アプリ内課金の導入に踏み切れない開発者のために課金の魅力と、面白さ、そしてマネタイズを実現するための必勝テクニックを紹介します。

■<講演2>アプリのレーティングを★5にする為にすべき事

登壇者：吉井 文学  -  FROSK株式会社 CMO

アプリの品質にも高い水準が求められるようになった昨今における、品質改善やカスタマーサポートを通した☆5アプリのつくり方についてお話致します。人気アプリの事例を踏まえた運用/品質改善のノウハウをご紹介するとともに、導入アプリMAUが1.6億人にのぼるSmartBeatのデータから、最新のOSバージョンシェアなど、このセミナーでしか見られない貴重なデータも多数ご紹介します。
