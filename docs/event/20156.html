イベント概要

学生劇団の現役生と卒業生の交流イベントです。
小劇場で活躍する先輩方と交流をメインに、テーマ別のトークセッションも行われます！

参加規程

学生劇団の現役生・引退生・または卒業生。

以上に当てはまる方であれば、

劇団を立ち上げようと思っている人。
制作・広報に対して考えを深めたい人。
演劇の感性の幅を広げたい人。
とにかく人と交流したい人。

どんな方でも気軽にご参加いただけます。

トークセッション内容

『まつりの胸騒ぎ』

with 沢大洋 / 藤原治基

京都学生演劇祭の発起人とおうさか学生演劇祭のプロデューサーのお二人に
学生演劇に対する熱意や期待について語っていただきたいと思っております。

『スタンスの取り方』

with 上田一軒 / 川内信弥 / 永井悠造 / hige / 安武剛

演出、舞台監督、役者などそれぞれの活動を中心にしながらも、
全員が劇団・ユニットの代表を務めるこのグループ。
世代も芝居のジャンルもバラバラな中、
それぞれの集団での広報や運営に関する考え方など、
スタンスの取り方を語っていただければと考えています。

『ノンフィクションに近づいて』

with サリngROCK / 土橋淳志 / 三名刺繍 / 山内直哉 / 山崎彬

各劇団の作家の方々により構成されたこのグループには
学生劇団時代の思い出を語っていただきます。
当時の将来への思いや学生劇団あるあるまで
リラックスした状態で話していただければと思います。

こんなところまで隅々見ていただいてありがとうございます。
短冊ストライプってのがなにかご存じないかと思います。
怪しい宗教とか政治団体ではないのでご安心ください。

学生劇団って凄く面白いんです。
演劇に対して興味ない人もある人も混在して演劇をやってる。
高校までスポーツやってましたとか言ってる人もいっぱい。

で、中から本気で芝居やろうって人が出てきて、
小劇場って世界を引っ張ってる人がいる。
そんな人たちに声をかけさせてもらいました。

このイベント、メンバーが物凄く豪華です。
正直自分でもここまで集まっていただけるとは思ってませんでした。
主催側の僕たちがどうなるのか一番ワクワクしてます。

だからって、敷居が高そうだなんて思わないでくださいね。
"結婚式の二次会"とか"新入生歓迎コンパ"にくる感じで来てください。

学生劇団って一般の感覚じゃ分からないもんってあるじゃないですか？
それを共有できる、学生劇団の関係者しか呼んでません。
大阪と京都もまぜまぜして学生劇団の関係者だけ。

ここまで読むほど悩んでるなら、おいでください。
もらった時間の濃度はいつもの倍にしてみせますから。

読んでもらった時間、感謝いたします。
