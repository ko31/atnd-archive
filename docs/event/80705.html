description

自分のやりたい事が分からない、自分らしさって何だろうと考えながら仕事をしている方も大勢いると思います。
人の役に立つことが出来ながら、自分らしく満足のいく仕事をする、そんな仕事が出来れば理想的ですよね。
しかし、そんな仕事はなかなか見つからないものです。特に自分のやりたいことが良く分かっていないときには、一歩を踏み出したくても、何をすればいいのかよくわからなくなってしまいます。
もし、そんな悩みをお持ちなら、私たちと一緒に自分のやりたいことを探してみませんか？
ヒントは意外と身近にありますし、特に、誰かの役に立てる活動を仕事にできれば、ありがとうと言ってもらえるうえに、自分を生かすことも出来て良いことばかりですね。

今回のセミナーでは、自分らしく仕事をしたいという気持ちと、誰かの役に立ちたいという気持ちを両立させている人たちの活動を例として挙げながら、「なぜその活動をしようと思ったのか」「その活動をどのように続けているのか」などについてのお話を行います。

毎回2時間のセミナーでは、内容が盛りだくさんのため、時間が足りないくらいです。
毎回好評を頂いている内容のセミナーですので、少しでも気になった方は是非参加してみてください。
