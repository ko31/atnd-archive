LittleSmallscript(http://ympbyc.github.com/LittleSmallscript/)(ympbycさん：@ympbyc)

Little Smallscript is a dialect of Smalltalk that compiles into JavaScript. Hidden behind the huge Smalltalk Environment, Smalltalk always had a beautiful syntax that is fit for object oriented scripting. Little Smallscript is an attempt to let Smalltalk see the light of day again. (Although I’ve never written Smalltalk before)

※ympbyc さんは日本の方です。
16:55 – 17:30

Amber SmalltalkによるClient-Side Web開発(マエダ：@maeda_)

Amber SmalltalkはJavascriptで実装されたSmalltalk処理系です。Smalltalkの世界とWebの世界が融合した新しいWebアプリケーションの形を紹介します。また、Amberをいじって、clojureやF#などの機能を取り入れた話をします。
17:30 – 18:00

Smalltalk ハッカソン

好きなことやりましょう。 Smalltalkでどうかく？など疑問・質問なども募集します。 Smalltalk初心者用に環境の準備不要なAmber Smalltalkハンズオンも考えています。
