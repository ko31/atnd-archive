14:50 デザインとプログラムを繋ぐ Flash Catalyst（40分）

Flash Catalystは、デザイナと開発者の間の連携作業をスムーズに進めるデザイナー向けのツールです。
Illustratorなどで作成したデザインにインタラクションやエフェクトを追加し、Flexアプリケーションのデザインを最大限に引き出します。
デザイナーの創造力をすぐに形にし、プログラマーの負荷を軽減するFlash Catalystを体感してください。

by 株式会社Modulay 渡邊佳一氏 (@keiitic )

15:40 AIR2のできること（30分）

Flex の開発ターゲットが Web アプリケーションなのに対して、Adobe AIR はデスクトップアプリケーションです。
Flex フレームワークを拡張した AIR は新しい技術でも、そこには既存の Web 開発技術の多くが利用できます。
AIR 2 で Web という枠から飛び出しましょう。
by しかじろう＠渡辺慎二郎氏 (@shikajiro )

16:20 Flexプロジェクトやってみた(仮)（20分）

Flex3で開発を行なった証券会社向け取引ツールを題材に話を展開します。
このプロジェクトは非常に高度な画面デザイン/操作設計に加えて、AIR版・Flash版の同時リリースという困難が多々ありました。
そこで用いられた開発手法やTipsを失敗、成功談を交えてご紹介します。
by 株式会社EC-One 岩尾厚志氏 (@iam100a24 )

16:50 共同編集ドローツール Cacoo（30分）

Cacoo(カクー)はWeb上で簡単にサイトマップやワイアーフレーム、UML、ネットワーク図などの図が作成できるドローツールです。
Cacooのデモンストレーションと裏側の仕組みを参加者の皆さんだけにご紹介します。

by 株式会社ヌーラボ 縣俊貴氏 (@agata )
