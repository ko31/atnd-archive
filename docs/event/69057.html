概要

競技プログラミング初心者が CODE FESTVAL 2015 の予選を突破するための有志による 非公式 の練習会です。
グループでワイワイ教えあいながら、競技プログラミングを始めてみましょう！

※このイベントは競技プログラミング初心者向けです。
競プロ勢はチューターとしてご参加いただければ幸いです。

対象

以下のグループに分ける予定ですので、お気軽に是非ご参加ください！

  1. プログラミングを始めたばかりの人
  2. プログラミング経験はあるが、競技プログラミングの経験はない人
  3. 競技プログラミング経験はあるが、予選を通過できるか不安な人

日時

2015年 9月8日 15:00〜19:00
（初心者の方向けに14:00から講習会も行います）

場所

コワーキングスペース Basis Point
http://basispoint.tokyo/

タイムテーブル

時間 内容 備考

30分問題を解く→10分解説する　というサイクルで行こうかと思ってます。

コンテンツ

プログラミング入門

開始前の1時間で、今回出題する問題に必要な、最低限のプログラミングの知識を身につけます。
資料はこちらから
資料には豊富な練習問題と、その解説が含まれています。
1時間という時間の制約もありますので、参加される方は予め資料を読んで、実際にコードを書き、AtCoder上で実行してみてください。

分からないことや取りきれないバグ、意味不明なエラーなどがあれば、お気軽に @kenkoooo までご連絡ください！

練習会

レベル別に、4人程度のグループに分かれて、指定した問題をオンラインジャッジ（AtCoderを予定）で解いていただきます。
最初は個人で解きますが、グループ内で分からない問題を教え合ったり、バグを指摘しあったりできれば良いと思っています。
チューターもいるので、活用していただければ幸いです。

懇親会

ピザとソフドリの予定です！
