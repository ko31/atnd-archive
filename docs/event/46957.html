Let’s make a game together!

Global Game Jam(GGJ)が今年も1/24～1/26開催されます。

Global Game Jamとは世界中で一斉にゲームを作るという壮大なイベントです。
Global Game Jamの大会概要についてはこちらをご覧ください。
GGJについて　http://ggj.igda.jp

今回「横浜青葉台会場」として登録させていただきました。
Aobadai Jam
http://globalgamejam.org/2014/jam-sites/aobadai-jam

「ゲーム開発は初めて」の方も問題ありません。必要なものはやる気のみです。一緒にゲームを作って学びましょう。ゲームの完成度よりプロセスを通して何か得るものがあればと思います。管理者である私はBlenderを使ってデザイナーとして挑戦予定です。会場が住宅街ですので、一般常識ある方に限らせていただきます。

大会スケジュール

1/24
17:00 田園都市線青葉台駅改札口 集合 会場へ向かいます。
18:00 大会開始 開会式～グループ分け～作るゲーム検討～ゲーム制作開始

1/25
ゲーム制作　（好きなゲームについても話しましょう。）

1/26
18:00 大会終了（その後懇親会もできればと思っています。）

注意事項

長丁場になります。物理的に寝る場所はありますが、人数分の寝具はありませんので、ご持参ください（数人分のマクラなどはあります）。よろしくお願いします。寝るための一次帰宅も大丈夫です。

コンビニは会場から徒歩１０分くらいのところにあります。

Global Game Jamサイトでの参加者登録もお願いします。
http://globalgamejam.org/
