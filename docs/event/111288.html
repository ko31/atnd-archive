『Ateam Design MeetUp』のテーマは、”フロントエンド特集！”　セッション＆交流会形式で開催！

『Ateam Design MeetUp』では、自社サービスや制作会社などでWebデザインの業務に関わるデザイナー向けに、勉強会や交流会など、アウトプットいただける場を設け、新しい気づきや成長につながるような機会を提供することで、技術の向上やデザイン業界のさらなる発展を目指します。自社サービスならではの考え方や業界の技術トレンドに関する共有、交流の場を通じてより深い知識を得て、さらには参加者同士でのつながりも増やせればと考えております。

今回は、エイチームのデザイナーによるトークセッションと懇親会を行います。気軽にお召し上がりいただけるフードとドリンクもありますので、デザインに関する情報交換をしながら楽しんでいただければと思います。 デザイナーとして活躍している方はもちろん、勉強中の方、学生の方や、将来的にWebデザイナーを目指したいと考えている方なども、ぜひご参加ください！

また、『Ateam Design MeetUp』では、参加者のライトニングトークセッション（以下：LT）の時間を設けております。参加デザイナーのアウトプットの機会を増やすことにより、デザイナー同士の交流を促進し、会社や組織などの垣根を越えた横のつながりを深めることを目的としています。積極的に発信していただき、大阪のデザイナー業界を盛り上げていきましょう！

セッション詳細

セッション①『コードで見る「ReactとVueどちらを選択するべきか」からの脱却』

  • 所属：株式会社エイチームライフスタイル クリエイティブ戦略部 大阪デザインTM
  • 担当サービス：「ナビクル」
  • 得意分野：フロントエンド全般

セッション概要

フロントエンドフレームワークは、混乱期を経て多くの比較情報が出てきました。実際に業務でReactとVueを使用し、主にDOMに関わる部分で得た知見やコードを交えながらお話します。

セッション②『ReactのメリットとAtomic Designとの相性』

  • 所属：株式会社エイチームフィナジー カードローン事業部 デザインTM
  • 担当サービス：「ナビナビキャッシング」
  • 得意分野：UI、フロントエンド

セッション概要

現在注目されているJavaScriptライブラリの一つである「React」のメリットと、UIデザインの設計手法である「Atomic Design」と組み合わせることによる効果についてお話します。

ライトニングトーク①『Tailwind CSSでCSSをアップデートしよう！』

  • 所属：株式会社エイチームフィナジーカードローン事業部 デザインTM
  • 担当サービス：「ナビナビキャッシング」
  • 得意分野：フロントエンド、UXデザイン

LT概要

フロントエンド開発において、意外とアップデートされないのがCSSの最新知識ではないでしょうか。今回はTailwindというCSSフレームワークについてご紹介します。
