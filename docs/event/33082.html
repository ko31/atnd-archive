セッション情報、スピーカー紹介

UXドキュメンテーション

「課題解決型ではなくビジョン駆動型」とするこれからのモノづくりとして、ワイヤーフレームやUXジャーニーマップなどの途中成果物を題材に、わたしたちの業務におけるコミュニケーションをどのように円滑に進めていくことができるのか、みなさんと対話したいと思います。

坂本 貴史

2002年より、 ネットイヤーグループ株式会社 にて、IA/UXデザイナーとして活躍。主に、企業におけるインタラクティブマーケティング支援（コンサルティング）や、Webサイト構築におけるクリエイティブディレクションを担当。とくに、Web情報アーキテクチャを設計する専門職インフォメーションアーキテクトとして活躍中で、執筆・寄稿やセミナーの講演なども行ってる。2010年には自治体サイトの標準化を推進する「OpenUMプロジェクト」の立ち上げにも参加し、著書に『 IAシンキング 』がある。

キッチンにはスマートフォンとカレーライス

今日、モバイルデバイスの急速な普及により Web が日常に溶けこみはじめています。モバイルデバイスの普及がもたらす日々の暮らしの変化をデザイナーとしてどうとらえるか、“体験”という言葉を軸に考えてみたいと思います。

いちがみ トモロヲ

15VISION 代表。Webディレクション、Webデザインを中心に活動しており、研修講師やテクニカルライティングなども行っている。近著に『 すべての人に知っておいてほしい スマートフォンサイトデザインの基本原則 』（共著）がある。石川県のWeb制作コミュニティ「 WDF 」、UXコミュニティ「 UX Kanazawa 」をという2つのセミナーイベントを運営している。

Windows 8 UX Guidelines

2012年10月26日、Windows 8がリリースされました。そこに新たに搭載されたタイルベースのデザインには、一貫性があり、洗練された魅力的なユーザーエクスペリエンスを実現する重要な特徴があります。本セッションでは、そのLook and Feelの概要やガイドラインについてご紹介します。

鈴木 孝明

地元上場企業プログラマ。Microsoft系の技術や製品が大好き。勉強会の主催や登壇、Web記事の執筆などで日々活動中。Microsoft MVP for Visual C#。

新人プログラマーが考える、UXと魔法の板っきれの話

モバイルアプリ開発の会社に新人プログラマーとして働き始めて早半年。私自身が普段置かれている「スマホアプリ制作現場」という環境の中で、スマホを利用する上で得られるUXとその質に大きく影響するUIについて、色々と考えてみたこと・学んだことをご紹介したいと思います。 あなたの愛用しているアプリにも、開発側が苦心した様々なUX向上の工夫が隠されている…はずです！

柏 夏美

2012年度、石川高専を卒業し、 株式会社jig.jp にプログラマーとして入社。
最近は Webアプリに強い興味があり、主にJavaScriptを勉強しています。ここからアプリ作りに必要な様々な知識吸収につなげていきたいと思いつつ鋭意修行中です。

Web最適化におけるデザインアプローチ

ユーザーのWebに触れる環境が多様化する中で、訪れるユーザーの情報も複雑になってきています。また、それに伴い解析できるデータも多くなってきており、それらの情報をどのように分析し改善していくかが、Web最適化の大きな課題になります。今回は弊社のWeb改善業務における実例を元に、UXを意識したデザインアプローチについてお話します。

澤田 幸弘

1982年生まれ。 株式会社オールコネクト にて、 デザイン部のマネージメントとWebサイトのディレクションを担当。
アクセス解析を軸にしたデザイン制作に力を入れています。
