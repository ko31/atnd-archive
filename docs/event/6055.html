富士山に登り隊2010を以下の日程で開催いたします。

事前の確認では登り隊隊員がけっこういたのですが、今のところリアル参加者は3～5名!?となっています。

人数が多い方がくじけず登れると思いますので、皆さんのご参加をお待ちしております。



・スケジュール



07月24日17:00 都内出発予定(人数次第でもっと早く？各地でピックアップ)

07月24日21:00 富士宮登山口五合目駐車場

07月25日04:00 山頂着(健脚なら5時間程度ですが、7時間と余裕を見てます)

07月25日11:00 下山開始（この辺は適当に、たぶんもっと早く下山開始か）

07月24日15:00 富士宮五合目に下山（のんびりでも4時間程度)



富士山情報(静岡県)


富士山情報(富士宮市)




・参加申し込み方法



参加を希望される方は、このページ右の「このイベントに参加登録する」からお申し込み下さい。



・参加費用



参加費用は人数次第で。

レンタカーにした場合には割り勘。
交通費はたぶん土日1000円にかかると思うのでガソリン代とあわせて適当に割り勘。
あとは自分に対しての「食料」「水」「装備」「保険」などは各自負担。



・必要装備



夜間登山なのでヘッドランプ。(懐中電灯は禁止)

防寒具。(山頂は日の出前、氷点下前後になります。上に着いてから着るつもりで)

水。(汗かきな人は二リッター以上、高山病対策のためにもそれなりに飲みます)

食料。(山頂までのエネルギー補給。アンパン、おにぎり、ゼリー類がオススメ)



・オススメ



帽子。日の出後は確実に紫外線に被爆します。

日焼け止め。同上。(曇りでもSPF50くらいは必須)

手袋。富士宮ルートは結構ごつごつしています。(軍手でもいいけど速乾性があるほうが…)

ストック。登攀補助以外にも立ったまま休憩で寄りかかれたり。(金剛棒は思いだけなので禁止)

ハイカットな靴やスパッツ。靴の中に砂利が入るのを防ぎます。

着替え。帰りに汗を流すために一式あるといいです。(もちろん持って登りません)





・注意事項



富士山は日本一メジャーですが日本一高い山です。



登山はいかなる状況下であっても全て自己責任です!!



なめてかかると痛い思いをしますが、それなりの装備があれば本格的でなくても何とか登れます。
特に登りは何とかなりますが、降りで足首をひねったり、転んですりむいたり、ひざをやることが多いです。
なのでなるべく長袖長ズボン、また膝はサポータを装着して降りるのがオススメです。
あとは歩きやすい靴と、吸湿速乾の服装なら問題なしです。



富士山は「意外と寒い」ので、こまめに着たり脱いだりできるといいです。
Gパンはやめた方がいいのと、靴は出来ればハイカット系をオススメです。
というのも、富士山は「細かいじゃり山」なので、靴の中に小石が入ってくるからです。
それと、ストックがあると歩くときにも止まっているときにも楽です。
間違えても金剛棒は買わないように。あれは荷物なだけです。
カッパはいらないかもしれませんが、あると防寒具代わりにもなります。



保険ですが、この手の保険に入っていないなら、モンベルの保険をオススメします。

http://hoken.montbell.jp/



こいつは「ヘリ」をはじめ「他人」を使った救護の時に必要なお金を出してくれるものです。
他にも東京都山岳連盟とかでも最近は個人会員を募っててそこで保険も入れたりしますが、まぁそこまで本格的にと言うのでなければ吸湿速乾Tシャツをモンベルで買うついでに保険に入るというのがいいかと思います。
