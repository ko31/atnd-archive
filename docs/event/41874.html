MySQLの実践的な学習

実際にMySQLにコードを入力している画面を見ながら、自分のパソコンにも同じように入力を行います。

書籍では経験者にとっては当たり前のことが省かれていますので、前提となる知識がないと、動かなくても自分のミスなのか説明不足なのか分かりません。

実際に操作の経過を見ながら、自分で行い、分からなければ質問もできるので書籍よりもスムーズに学べます。

勉強会について

  1. 勉強会はプロジェクターに映した講師の画面を見ながら、自分のノートパソコンにプログラムをします。
  2. ハンズオン（実際にプログラムを行う授業）形式ですが、見学だけでも構いません。
  3. １８：００～２１：００の３時間です。
  4. 会議室は１7時から入れます。主催者は１７時４５分頃に到着します。
  5. 参加費として２，５００円かかります。

XAMPPのインストールをお勧めしています

PHPやMySQLを自分のパソコンで実行するには、XAMPPというソフトウェアを使うと簡単です。

これはパソコン上（ Windows, Mac OS X ）で、Webサーバーを実行するプログラムです。
