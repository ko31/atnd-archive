description

すでにビジネスをやっているのに稼げていない方に朗報です。

電子書籍は、専門性と信頼が
一度に手に入る魔法のツールなのですが、、

その魔法のツールを使いこなすための
電子書籍を使ったビジネス戦略勉強会を開催します。

以下のような内容をワークショップ形式でお伝えします。

●　初心者でも失敗しようがないブランディングの方法とは？

●　あなたの著書がAmazonから販売される具体的な方法とは？

●　無名のあなたを見込み客に先生と呼ばせる簡単な方法とは？

●　あなたの商品、サービスに顧客が行列を作る！その秘密とは？

●　知らないと損をする目的別電子書籍の活用方法を大公開！

●　最大限にあなたのチャンスを活かすための戦略とは？

●　電子書籍１冊で６０００万円以上売上げた事例を紹介します！

●　大手出版社の知らない本当の書き方とリスト収集方法とは？

●　バックエンド商品を販売するための集客出版の方法とは？

●　規約違反にならずにアフィリエイトが出来る事例を紹介！

電子書籍出版コンサルタントとして
わたしたちはさまざまなことを今まで試してきました。

例えば、

・電子書籍では何が出来るのか？
・専門家としてのポジショニングはどうすれば取れるのか？
・読者さんに何を伝えたらクライアントさんになってくれるのか？
・先生、お願いします！と読者からメールをもらうには？
・一般の広告と比べてどのくらいの費用対効果があるのか？
・アフィリエイト報酬は発生するのか？

など。

依頼も多く数々のクライアント事例も持っています。

手がけた著者の中には有名人もかなりいて

プロダンサーのＳＵＢＡＲＵ氏や
オリンピックコーチのセット裕一氏。
元ヤクザでプロボクサーの川嵜竜希氏
元Jリーガーの佐伯直哉氏、
若手起業家の春田和樹氏など

３０名以上が名を連ねています。

さらに、驚くことに、
全ての著者をKindleのランキングで１位にしています。

しかも、電子書籍たった１冊で６０００万円を売り上げた。

という事例も持っています。

今年の3月に開催したセミナーでは、
1会場30名のところ

2日間で70名以上の申し込みがあり
3会場での申し込みも170名を突破する
超人気セミナーでした。

今回は少人数制で各会場５名までとなっているので、

おそらく、早期に満員御礼と
なることは間違いありません。

勉強会参加費用も3000円と
かなり格安なので、

今の現状から絶対に脱出したい！
という方は、是非、参加された方が良いと思います。

定員は各会場5名で全12回です。
総勢60名限定ですので
お早めにお申し込みくださいね。
