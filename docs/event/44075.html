AKB48の『恋するフォーチュンクッキー』を、さまざまな企業・団体が一丸となって踊るミュージックビデオがあちこちで公開され、ネット上で話題になっているそうです。

参考：
企業のスタッフが踊る「AKB48恋するフォーチュンクッキー」動画まとめ

：

企業といえば出社。出社といえば「エクストリーム出社」

ということで、
早朝から『恋するフォーチュンクッキー』を踊ってから出社してみませんかという企画です。
みんなで踊りたいひとは、芝公園で。各自で踊りたいひとは、どこでも。
同時多発的にできたらいいなと思っています。

[参加方法]

1. ATNDにエントリーする
2. 準備の様子をTwitter ハッシュタグ #エクストリーム出社 でレポート
3. 開催時間（6:30~8:00）に踊って、Youtubeに動画をアップする。
4. YouTubeのURLをTwitter #エクストリーム出社 でつぶやく
5. 出社の様子を写真に撮って #エクストリーム出社 でつぶやく

・各自で参加する人は…
　→ 自分で踊って動画をアップ＆#エクストリーム出社でつぶやく

・みんなに混じりたい人は…
　→ 10/3（木） 6:30 芝公園に集合！

できれば日本各地や複数の会社で、その地域性や企業性を背景にして踊った映像と、そのあとの出社する写真を、エクストリーム出社協会で編集してつなぎ合わせたいと思っています。
