catch 世界はそれを浴衣と呼ぶんだぜ。
description

　浴衣って花火大会とかお祭りなんかの行事が無いと、着ないうちに夏終わってしまいがちです。でも人ごみを浴衣で歩くと折角着付けた型や髪が崩れてしまいがちで特に女性にとってはあまり嬉しくないですよね。

　だから今回は「浴衣を着てどこかへ行く」のではなくて「何も行事は無いけど浴衣を着る」ことが目的です。着た時点で勝利。あとはビアガで酔っ払って花火でもしてその後はドヤ顔で闊歩して各々好きにしたらいいじゃない！という企画です。ビアガということで費用は各々飲んだ分 + 買った花火代の頭割りのみです(そんなに多くならないように調整します)。また雨天の場合は普通にどこかで飲みます。

参加資格は浴衣(男性は甚平もOK)を着ている人。
