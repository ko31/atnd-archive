概要

2月17日に東京で開催された、 【初心者向け】Sencha いろはの「い」 を札幌で再現します。

HTML5 Webアプリケーションフレームワークの Sencha Ext JS / Sencha Touch、そしてその開発ツール（Sencha Cmd/Sencha Architect）などのご紹介を始め、Sencha Touch を利用した簡単なアプリケーションの構築をハンズオン形式で、0から完成まで一緒にやっていきます！

「Webアプリを作ってみたい」
「使ってみたいけど、何すれば良いの？」

といった初心者の方でも気軽にご参加ください！

コンテンツ

19:00 ~ 19:30: Senchaフレームワークとツール達のご紹介
19:30 ~ 21:00: ハンズオン

講師ご紹介

後藤 恭智 ( @yasutomog )

1983年札幌市生まれ。学生の頃からプログラミングを始めて就職をきっかけに東京へ。主にJavaとJavaScriptを使ったWebアプリケーションの開発で複数のプロジェクトに従事。その後、札幌に戻り、Sencha Ext JSと出会う。情報を模索する中で小堤 一弘 氏や中村 久司 氏とtwitterで知り合い、出会い系みたいと言いながら直接の交流を重ねて、現在、株式会社ゼノフィ札幌事業所にて勤務。これまで以上に北海道のSencha開発を活性化できればと日々模索中です。Sencha社認定トレーナー。
