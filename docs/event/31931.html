今度は実際に手を動かしてみて、その過程であったことなどを発表する会にしてみます。

各自行うこと


  • 何でも良いからパッケージをいじってみる
  • いじってみてハマったところ、よく分からなかったところ、気づきがあった部分（納得した部分）があったらそれもメモしておく
  • 知りたいこと、もメモしておく。出来れば事前にtitanpadに記入しておくこと。
  • pbuilder でビルドできる環境をローカルに整えておいて、実演などできるようにしておく

持ち物


  • ノートPC（可能な限りローカルでビルドできる Debian 環境。unstable推奨）
    ダメならVPSとかに入れてリモートで使えるように。
  • もしあればインターネット接続可能な機器

オープンソース・ソリューション・テクノロジ株式会社様の会議室を提供して頂いております。感謝。
場所は五反田駅（ＪＲまたは都営浅草線、東急池上線） 西口より徒歩３分 東京都民銀行の上 ８階になります。
