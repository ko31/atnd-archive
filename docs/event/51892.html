description

競争が激しくなり、国の方針によって経営を左右される医療介護業界。
来年・平成27年には介護保険法の改正があります。
この波を乗り切るための情報をお伝えするセミナーです。
平成27年度の介護保険法改正についてのいち早い情報を司法書士が、職員の能力をアップさせるシステムを教育マネージャーがお話しします。
セミナー終了後は各講師に個別相談が可能です！

＜セミナースケジュール＞
第一部　経営の守りを固める　13：30～14：20
「Ｈ27年度 介護保険法大改正のツボ」
　介護保険法の改正の前に、出来る限りの情報をお伝えします。
　　講師：司法書士　羽原　孝治

第二部　職員教育による運営支援　14：30～15：00
「ＩＴ利用で医介教育改革を」
　ウェルグループが考案した介護職員の教育システムをご紹介。
　　講師：安井　妃世（ウェルグループ教育マネージャー）

第三部　将来に向けて新たな道を開拓したい方へ　15：10～15：40
「介護業界に新風を吹き込みます！」
　特約店・提携店ビジネスモデルのご提案。
　　講師：引尾　拓志（株式会社ロマン倶楽部）
　
個別相談　 15：00～16：00　
交流会　　 15：50～16：40
