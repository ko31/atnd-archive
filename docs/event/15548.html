title 第36回「WEB TOUCH MEETING」やります！
catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。


■現在予定している内容：

　●セッション１(30分)
　『カラーコードのデザイン方法、さくっと教えちゃいます。
　　YouデザインつくっちゃいなYO♪』カラフルブリック島原さん

　＜内容＞
　デザインを携帯やスマホで読み取ることができる次世代QRコード、
　カラーコードの一番の特徴となるデザインの仕方を中心に、
　そもそもカラーコードってなんぞやの部分も改めまして
　ご紹介したいと考えています。

　以下のページより、無料のカラーコードリーダーを事前に
　ダウンロードしておいていただけると助かります。
　http://www.colorzip.co.jp/cz-wp-3.0.1-ja/Download/

　＜島原さんの紹介＞
　カラフルブリック株式会社(http://colorfulbrick.com/)の人。
　分類的にはシステム系ウェブ屋で、今のメインのお仕事は
　ディレクターさんです。

　趣味はバルーンアート。

　趣味がこじれて土日は結婚式場やイベント会場で活動しています。
　バルーンの中から登場したい…とか、300個のハートバルーンを
　ふらせたい…とか、そんなご希望があれば島原まで♪

　アレ…？ 結局、何屋さんなんだか(笑)


　●セッション２(40分)
　『OpenStreetMap ～ GPS, * による地図 ～』丸市さん

　＜内容＞
　「自由なウィキ世界地図」である、OpenStreetMap を紹介し
　どうやって地図を編集するのかを説明します。

　発表の内容に関しては、以下のものが参考になります。
　http://www.openstreetmap.org/
　http://openstreetmap.jp/
　http://wiki.openstreetmap.org/wiki/Ja:Main_Page?uselang=ja

　＜丸市さんの紹介＞
　FM TOWNS から、Linux ユーザになりユーザ歴18年です。
　当初は、NIFTY の TOWNS 関係のフォーラム(FTOWNSなど)で活動、
　LUCK, LUKY, LILO などのユーザグループに参加しています。
　プログラマ、サーバ管理、ネットワーク関係の仕事をしています。


　●セッション３(40分)
　『androidゲーム制作について』佐々木さん

　＜内容＞
　この夏、androidスマートフォンが竹の子のように出てくる中、
　私はandroid用シューティングゲームを発表します。
　初のandroid用ゲームを開発、リリースしていくまでの道を話します。

　＜もくじ(予定)＞
　1.開発実機を手に入れる——>開発実機を選ぶ
　2.ターゲットマシン(対応端末の動作環境)を決める
　3.ゲームのプログラム
　4.携帯アプリ作成の注意点
　5.アプリの名前の付け方
　6.アクティベーション(不正利用防止策)の処理
　7.アプリの価格
　8.アプリ発表時のその他の連動について
　9.ロードマップ

　＜佐々木さんの紹介＞
　佐々木順昭
　某スクールの非常勤講師でゲームプログラムを教えながら、
　シューティングゲームを作って販売しようとしています。

　趣味はRCとハムスターとその他いろいろ
　日本androidの会　中国支部や、
　他にもいろいろお邪魔させていただいております。

　ホームページ　http://www.lemonte.net
　メールアドレス　support@lemonte.net
　twitter:sasakilemonte
　mixi:LemonteSasaki
　Facebook:Yoriaki Sasaki
