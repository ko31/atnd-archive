このたび、ティードリの企画趣旨に共感して、ティードリの活動を応援してくださる

「ドリームパートナー」として名乗りを上げてくださった岩崎由純先生の講演会を

特別に開催させていただくことになりました。

岩崎先生は、何冊もの本も出版され、全国で講演もされており

また、アスリートやスポーツ関係者のモチベーションをあげる

セミナーも開催されている方です。

今回特別に、通常では考えられない参加費で開催させていただきます！

・自分自身が元気になります！

・周りの人も元気にできます！

・コミュニケーションをとることが楽しくなります！

是非、お時間のご都合がつきましたら、ご参加下さいませ。

〇岩崎　由純さんの『心に響くコミュニケーション ペップトーク』

※ 岩崎　由純さんのプロフィールはコチラ　PDFはコチラ

講演内容　: スポーツ現場のコミュニケーションスキルの一つである ペップトークをご紹介します。

ペップトークの基本であるプラス思考やイメージの共有の考え方は、ビジネスや、 子育ての現場でも応用できます。

加えて一流の選手達が日々行っているアファメーションについてもご紹介します！

・日　時　: ３月１７日(木)　１９時００分～２１時３０分

・場　所　: 新丸ノ内ビル　14階　パソナ ※はじめてお越しになる方は、地図をご参照ください。

http://www.pasonacareer.jp/info/map/188/

新丸の内ビル14Ｆ　エレベータホールから正面入り口 左手奥が会場となります。

（商業ビルスペースではございませんのでご注意ください。）

参加費用　: ３,０００円/人

・人　数　: 限定５０名(先着順)
