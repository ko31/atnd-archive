description

名古屋で活躍中のWebディレクターで飲んでみよう！ということで、「名古屋Webディレクター飲み会（仮）」を開催します。

あまり大きなことは考えていない、シンプルな飲み会です。
Webディレクションについて刺激がほしい方、悩みを共有したい方、Webの将来について熱く語りたい方など…、何かのキッカケになる場になればと思います。
皆さま、よろしければご参加ください。

対象

・Webディレクション（PM、PL、CD、AD、IAなど含む）に専属的に従事している方。

※ひとまず括りを設けさせていただきます。上記に当てはまらず参加希望の方はご相談ください。
※定員数は状況によって枠を増やしますので、補欠登録してください。

参加費用

・5,000円
・コースの食事と2時間のフリードリンク代です。
・参加費用は当日会場にて徴収いたします。
・領収書の発行等はできませんのでご了承ください。

募集期間

・一旦締め切りましたが、ご希望の方は補欠登録いただくかご相談ください。

場所

・場所が決まりました！「くつろぎ食堂 Ami」
・大人の雰囲気でしっぽりです。

申し込み

・本ページよりお申し込みください。

キャンセルについて

・会場を人数で予約していますので、当日のキャンセルは極力控ていただくよう調整をお願いします。
・基本的に一週間前までのキャンセルを承ります。
・近日中に連絡手段をご案内します。

発起人

・岩田玲　http://www.aquaring.co.jp
