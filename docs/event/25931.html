PHPカンファレンス北海道の 懇親会 の参加登録用です。
詳細は、PHPカンファレンス北海道のWebサイトをご覧下さい。
メインイベントの登録は、別途、メインイベントの申し込みからの登録をお願いします。

どうしてもATNDにログインすることができない場合は、Webサイトの「お問い合わせ」から、参加申し込みをお願いします。（お問い合わせ項目は「その他」にしてください。 懇親会の申し込みであることを明記してください ）

  • 会場：いっしん　IZAKAYA　其の壱　南1条店
  • 参加費：3,500円
  • 講師以外の方は、スタッフも含めて全員参加申し込みしてください
  • キャンセルは 4/10 までにお願いします！ 期限を超えてキャンセルをした場合、参加費を請求します
  • メインイベントの参加が必須になります。 懇親会だけの参加はできません。 メインイベントの受付を通っていないと参加できないのでご注意下さい。
  • 懇親会の料金は、イベント会場受付（産業振興センター ルームA前）でのお支払いをお願いしております。
  • 懇親会に参加される方は、当日13:00までにイベント会場受付にて懇親会費をお支払いください。
  • 間に合わない方は、@phpcondo までご相談ください。
    ※当日14:00を過ぎてお支払い・ご連絡のない場合は、キャンセルとさせて頂きます。ご容赦ください。

ATND上で補欠登録になっている方へ

  • ATND上での補欠登録は無効になります。キャンセル待ちをする場合は、お手数ですが以下の項目ををお読みください。

キャンセル待ちをしたい方へ

  • イベント会場受付にて、当日13:00までに、キャンセル待ちリストへ氏名と連
    絡先の記入をお願いします。
  • 当日、懇親会参加予定者にキャンセルが発生した場合、キャンセル待ちリストに記入いただいた順から参加可能になります。参加可能になった場合、スタッフからご連絡いたします。
