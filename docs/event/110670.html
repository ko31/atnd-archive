description

部屋のスペースに余裕があるので5名分枠を広げました。よろしくお願いします。

※前回と同じく刈谷市での開催ですが、施設が異なります。よくご確認の上お越しください。

※部屋の定員は30名ですが、同伴者用に10名分余裕を持たせるため"20名"に設定しております。同伴者の合計人数次第では、申込期間中に定員枠の調整を行う可能性があります。ご了承ください。

ルービックキューブのレクチャーが主になります。わからないことがあれば何でもお聞きください。もちろんひたすらタイム計測をしたり、色々な立体パズルに触れたりなど、過ごし方は自由です。
レクチャー会終盤には大会と同じタイマーを用いた計測会を行っています。大会ルールに則って行いますが、あくまで"体験"ですので堅苦しいものではありません。ご安心ください。
※途中参加・途中退室可です。参加登録時のコメントもしくは下部お問い合わせにて、事前にお知らせいただけますと幸いです。

参加費

500円/人 (ウェブサイト運営費や会場費、その他消耗品の購入等に充当)
同伴で参加される方がいらっしゃる場合は、申込時のコメントにてその人数をお聞かせください。公平を期するため、同伴参加の方もお一人ずつお支払いいただいています。ご了承ください。未就学児は無料です。

開催場所・アクセス

刈谷市産業振興センター 3F 305会議室
JR刈谷駅(北口)・名鉄刈谷駅(北口)から徒歩3分。

持ち物

  • いつも使っている解説書など
