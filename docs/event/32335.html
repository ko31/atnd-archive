セッション概要

Stormの注目の新機能 Trident API（木村 宗太郎）

8/6に、Stormの新バージョン 0.8.0 がリリースされましたが、 本バージョンから、「Trident API」という新機能が含まれるようになりました。

「Trident API」は、ジョイン、統合、グルーピング、フィルタリング等をサポートしており、 これまでのステートレスな処理ではなく、ステートフルな処理が可能となり、 Stormで実現できるストリーム処理も、大きく変わることになります。

本セッションでは、その「Trident API」の概要について紹介します。


単なるキャッシュじゃないよ!? Infinispanの紹介（束野 仁政）

ビッグデータ時代のシステム開発では、NoSQL、KVS（Key-Value Store）といった、 大量データを効率的に扱うための技術が不可欠になっています。

HBase、Cassandra のように、データを永続化するものから、 memcached、Redis のように、インメモリで処理するものなど、 用途によって、様々なプロダクトが存在します。

本セッションでは、キャッシュ、KVSという枠組みを越え、 「インメモリ データグリッド」という位置付けで登場した、「Infinispan」を取り上げます。

「Infinispan」は、分散キャッシュ、MapReduce、Grid File Systemといった、 多彩な機能を提供しており、その概要について紹介します。
