イベントの概要

現在増え続けるスマートフォンアプリ。そして、実はそのスマートフォンアプリ向けに提供されているサービス、ツールも増加を続けています。

この勉強会では、日本ではまだあまり知られていないであろう海外のスマートフォンアプリケーション向けサービス（ツール）の具体的内容について、参加者同士で調べ・発表しあうことによって理解を深めましょうという都合の良いことを考えています。
※スマートフォンアプリ向けサービスとは、主にスマートフォンアプリ開発者向けに提供されているサードパーティーサービス（例：API、SDK、Webサービスなどなど）を想定しています。

基本的に参加者の皆様1人ずつに1サービスの調査と発表をお願いできればと思います。（自社サービス紹介、日本のサービス紹介は原則禁止とします）
1サービスあたりの発表時間は短く、特に高いクオリティは必要ありませんのでお気軽にご参加ください。
（むしろサービス内容を短く簡潔に発表する必要があるかもしれません）

また、各サービスの発表希望者が重複した場合は先着順とさせていただきます。調べたい・発表したいサービスが見つかったら参加表明の際かATNDのコメント欄でつぶやいてください。タイムテーブルに追加していきます。
発表内容で迷っている方は、先に参加表明だけしていただいても結構です。後ほどATNDのコメント欄等でご連絡ください。

比較的ニッチなテーマですので、同じような分野に興味がある方々が集まる中で、有意義な時間を過ごせるような会にできればうれしいです。
