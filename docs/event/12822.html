イベント概要

第0回勉強会からはや3カ月、大阪で再びAWSの勉強会を開催することとなりました。
前回に引き続き今回も初心者を対象としたアマゾンクラウドの概要の説明もあります。
今までにアマゾンクラウドを触ったことがある人には最新の技術の紹介も行います。
初心者から上級者まで興味がある人は気軽に申し込んでください。

さらに今回は特別に AWS Senior Evangelist の Jeff Barr も呼びました。

関西でもアマゾンクラウドを盛り上げていきましょう。
