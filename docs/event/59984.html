description

Kickstarterでのクラウドファンディングから始まり今や日本の全国ネットでも目にするようになったヘッドマウントディスプレイOculus Rift。
今年9月には2013年リリースの初代開発キットの図面がオープンソースで公開されました。
しかし、現在はまだ開発キットの段階であり、来年の一般製品版を前にしてまだまだ開発者は少ない状況です。
プログラマの多いRuby Cityである松江で、Oculus Rift開発を行う方がもっと増えるように今回Oculus Riftの体験を行える交流会を開催致します。
当日はOculus RiftだけでなくLEAP MotionとMyoも用意しています。

タイムテーブル

時間 タイトル
10:00 – 11:30 開場・午前の部
11:30 – 13:30 開場の様子を見て昼食休憩に入ります
13:30 – 17:00 午後の部・閉場

当日は入退室自由ですが、昼食の時間帯は外出のため入室出来ない場合があります。
また、会場の定員が20名のため余りに人が増えた際は入場規制を行いますので、ご了承下さい。
来られる時に twitterアカウント にリプライを飛ばしてもらえると助かります。
