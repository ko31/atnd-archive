仏教には、「幸せ」になる原因と結果の法則が教えられています。

皆さんにとって幸せとは？
幸せには様々なカタチがありますよね♡

今回は「恋愛」をクローズアップ。

3回に分け、仏教の教えから、恋愛にまつわる数々の悩みを紐解いていきます。

第一回 「自分を知る」
第二回 「相手を定める」
第三回 「未来に向かって動き出そう！♡」

毎回のテーマについて深める質問に答えていただく形で進めていきます。
その後、お互いにシェアする時間を設けるので、そこでより深い気づきを得てくださいね♡
その後は、女子トーーク♡で、"本当の自分"をさらに深く！探って行きましょう！

さて、第一回目は「自分を知る」です。

恋愛をして幸せになるためには、私がどんなことに価値を置いていて、どんなことが好きかを知るってとっても大事なんです。

例えば、就職活動するのにも、まず自己分析です。
大学受験をするのにも、まず自分のレベルを知ることが先決です。

何をするにも、自分を知るって大切♡

では、みなさんはどの程度自分のことを知っていますか？
案外、周りの人の方が知っていたりして…。

１人だと意外とわからない自分のこと。
この講座で、一緒に自分を深めましょう♪
