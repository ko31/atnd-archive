【AID勉強会】アプリのデザイン事例共有会！

【今回の目的】
・アプリデザインのノウハウ共有
・デザイナーとアプリ開発者のマッチング


AID勉強会とは

AID定期勉強会は、無料アプリにおける3大要素
「開発」「DL拡大」「マネタイズ」について、情報提供を行っていく場です。

例えば、以下のような点について各専門家の方にお話頂きます。
『もっと効率的にアプリ開発出来ないの？』
『お金をかけずにどうやってダウンロードを増やすの？』
『収益を上げる為に気を付ける点は？』

定期開催致しますが、登壇者の方や発表内容は随時変更致しますので、是非ご都合が宜しい際にご参加下さいませ。

場所

Tech Buzz Space（新宿三丁目E1出口徒歩4秒です！）
新宿オミビル4階　東京都新宿区新宿5丁目18番20号

参加条件

・Android/iOSアプリ開発を行っている/興味がある方
・アプリデザインに興味がある方（学生さんも歓迎！）
※広告会社様は参加NGとさせて頂いております

参加料

無料（受付で名刺を1枚頂戴致します）

【特別講演】
UI for Girls
～KawaiiをThinkingしちゃおう♥～

アプリのUIデザインとしてフラットデザインが主流になりましたが、スタイリッシュでカッコイイ印象のものが多い中、女の子向けのデザインやアプリが少ないなと思ったのがきっかけで今回は女の子がカワイイ！使いたい！と思うようなコンセプトやデザインを
試行錯誤するプロジェクトの第一弾についてお話させて頂きたいと思います。
女の子向けのアプリやWEBデザインの実例も踏まえ、
女子目線ならではのお話ができればと思います。よろしくお願いします。

株式会社スキーマ 　クリエイティブディレクター＆リペアラー　イチハラマリコ 様

＜略歴＞
1985年岐阜県岐阜市生まれ。
早稲田大学在学時に学生ベンチャーでWEBデザイナーとしてデザイン業界に参入。
大学中退後フリーランスのイラストレーターとしてグループ展や個展など積極的に活動を行う。
2010年10月よりスキーマに入社。
主にiOSアプリのキャラクターやUIデザインの制作ディレクションを担当。
ビビッドな色遣いや海外テイストや女子向けのデザインが得意分野。
最近はディレクションだけでなくプランナーとしての知識と経験を溜めこみ中。

＜実績＞
・株式会社ゼンリンデータコム「恋するマップ～女子ちず～」
・株式会社ゆめみ「Nailbook」
・株式会社サイバーエージェントアメリカ「AnimalBakery」
