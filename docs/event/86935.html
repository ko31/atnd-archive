description

このワークショップでは、第一線の現役ビジネスマンが持っている活きた知識・スキルを共有して、新しい発想や刺激、気づきを得ることを目的としています。


設立直後の会社や上場企業の経理財務、IR担当も経験
社会を革命的に躍進させるために努力しております。


今回は第一線で働く財務マネージャーを講師として招き、
決算書について、何の知識が無い方でも分かるように解説いたします。

第2回も参加無料ですので気軽にご参加下さい。
社会人の方はもちろん、学生の方も参加頂けます。

講座内容


決算書、主に損益計算書（P/L）、貸借対照表（B/S）を中心に、
全体像の把握、明日から役立つ分析手法をご説明いたします。

決算書が読めることで、少し先の未来が分かる

読めるとかっこいい、出世、独立、投資に役立つ、いつか読めるようになりたい、、、、
と思っていた方は、この機会に一歩足を踏み出しましょう。

・決算書の全体像が理解できます
・10秒で決算書の最重要ポイントを押さえる方法が分かります
