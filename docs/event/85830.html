　ひかりの輪では、6月29日(土)～30日(日)に、東京本部教室を中心に、2日連続の特別セミナーを開催しますので、ご紹介いたします。

　特別セミナーでは、令和という新しい時代を生き抜くために役立つ新しい思想・人生観・世界観や、安定して柔軟な強い心、健康、知力・集中力を培う術・智恵を学ぶことを目的にしています。
　すでに今、お金・地位・競争に偏った価値観を超え、人生全体を見渡し、心の幸福・解放、心身の健康、良い人間関係などを重視した、本来の生きる目的を取り戻そうとする方が多くなっていると思います。

　こうした皆さんのために、このセミナーで、さまざまなプログラムをご用意しました。



１，セミナーの具体的なテーマ

（１）心の安定・幸福のための仏教や心理学の思想・瞑想

　仏教の思想・哲学や現代の心理学の思想の中で、心の安定・幸福、人間関係の改善、自己実現に役立つものを解説・実習します。

（２）健康・若さ・集中力・自己実現を助けるヨーガなど

　ダイエット目的ではなく、心身の健康・安定・悟りを目的とした本場インドのヨーガや気功法に基づいて、各種の体操、姿勢・座法、呼吸法を解説します。

（３）仏陀（目覚めた人）の説いた精神的覚醒の思想・瞑想法

　仏陀が説いた精神的な覚醒（大慈悲・安定した広大な意識の実現）の思想と実践法に関して、普通の人でも日常生活で実践できるほどに、かつてないほどわかりやすい言葉で解説します。

（４）仏教・ヨーガの東洋思想と最新科学の一致点

　心身医学、脳科学、身体心理学、高齢者の心理学や社会学といった最新の医学・心理学・社会学は、ヨーガや仏教の実践哲学の思想と様々な点で一致しています。こうした最新の科学の視点からも、東西の違いを超えた普遍的な叡智を解説します。

（５）令和新時代の新しい人生観・世界観

　高齢化社会が進む令和新時代は、老化防止を含めた新しい医療技術による長寿化などとともに、新しい人生観が広がり始める気配があります。人生全体を見渡した、新しい時代のための生き方を展望します。

　なお、参加の仕方としては、下記にご紹介する講義、瞑想・行法の指導、個人面談といった、各プログラムに絞ったご参加も可能です。

　ご機会・ご関心がおありの方は、皆さんぜひご参加ください。



２，日程と場所

（１）東京本部教室セミナー（世田谷区南烏山）：2019年6月29日(土)～30日(日)
（２）千葉支部教室セミナー（千葉県鎌ヶ谷市）：6月29日（土）予定であり、変更の可能性あり


３，各プログラムの内容のご紹介

（１）上祐史浩の講義

（２）心の安定・意識の拡大：各種瞑想法の実習

　良い瞑想は、心の安定と精神的な覚醒＝広大な意識の決め手となります。そして、そのためには、①瞑想の前の準備と、②効果的な瞑想の双方を理解して練習することがポイントとなります。そこで、瞑想のための環境、日常の注意点、身体的な準備を総合的に解説するとともに、伝統的なヨーガ・仏教の瞑想に加え、ひかりの輪が作った現代人に合った新しい瞑想法（輪の読経瞑想）を実習します。それは、わかりやすく短い言葉で、仏陀の智恵のエッセンスを表したもので、どなたにでも実践でき、長期的に高い効果があるものです。

（３）ヨーガの行法指導：本場本来の実践法とオリジナル行法も

　心身双方に良い影響のある、各種のヨーガの体操・呼吸法・瞑想法を指導します。その内容には、長年の研究・実践を通して、さまざまな工夫が込められています。





（４）自己内省法「内観」

　内観とは、もともと、古くから伝わる「身調べ」という修行法からヒントを得て、宗教的な要素を取り除いて、誰もができる自己反省法として確立されたものです。学会（日本内観学会）があって、医療にも応用されており、海外にも広がっています。


　その方法は、いたって簡単で、内観専門家の指導のもと、自分がこれまでの生涯で、両親・親族・友人知人などとの関係で、①していただいたこと、②して返したこと、③迷惑をかけたこと—を１つ１つ思い出していくという作業のみを、一定の作法に基づき、丸１日～１週間ほど集中して行なうというものです。


　内観の良さは、感謝の心＝愛が深まり、幸福感が増し、心が安定することです。また、感謝の心が深まれば、人間関係も改善します。 そして、その延長上には、仏教が説く悟り、心の解放、慈悲の体得の土台にもなります。


　今回のセミナーでは、希望者の方に、それぞれ内観に集中できる専用のスペースをご用意して、内観に詳しいスタッフのお手伝いのもと、内観を実践していただけます。

（５）ヒーリングプログラム

　セミナーの期間中、より深い効果を得たい方のために、希望者の方には、以下の特別な瞑想やヒーリングのプログラムを用意しております。
　参加料金は、上記の他のプログラムとは別となります。詳細にご関心があれば、下記の担当者にご連絡下さい。

①仏教法具の聖音ヒーリング
　各種の仏教法具の聖音によるヒーリングです。ドニパトロの深い慈悲の波動、精神的な智恵の目覚めを促すようなティンシャー・ガンターの波動が、あなたの心身を浄化します。聖音以外にも、視覚的なシンボル・瞑想香などを用い、特別に浄化されたヒーリング室で行われます。

②入浴ヒーリング
　体を温め、気の通り道を浄化する特殊な効果を持った石の成分を含んだ入浴ヒーリングです。気の流れを改善し、心身の浄化につながります。

（６）野外での歩行瞑想

　近くの良い自然の中で、歩行瞑想が行われます。歩行と瞑想を組み合わせるとさまざまな良い効果があります。これらの歩行瞑想・気功・ヨーガなどの身体行法は、心と体が深く関係し合っているために、心身を共に浄化する効果があります。



４，セミナーのその他の特長

（１）教室全体がヒーリング空間に

　セミナーが行われる教室の空間は、さまざまなツールによって浄化され、非常に優れたヒーリング空間になります。
　具体的には、①良い瞑想を促すチベット・ブータンの特別な瞑想香、②心を静める聖音を奏でる仏教法具、③聖地の聖水、④仏教の各種の法具や貴石、⑤神聖な意識を引き出す仏像・仏画といった象徴物・ヒーリングツールを用います。

（２）上祐史浩らとの個人面談

　また、講義に加えて、講義の後には、上祐史浩らとの個人的な面談や相談の機会も設けております。
