内容

・環境構築編

iPhoneの開発に必要な環境構築を説明する予定です。
iPhone開発に必要な開発ツールであるXcodeを実際にインストールし、
iOSシミュレータで簡単なアプリケーションを動作させたり、
iPhoneの実機デバッグの方法を学んだり(こちらは説明のみとなります)
など、環境構築編として、iPhoneの開発へのとっかかりとしていただければと思っております。

・入門編

Storyboardを使用した簡単なアプリケーションを作成していくハンズオンセッションです。

参加資格

Mac OS X 10.6(SnowLeopard)以上が動くPC(推奨はOS10.8(Mountain Lion))を持参できる方。
iPhoneのアプリケーション開発に興味がある方。

持ち物

・上記条件のPC
・出来れば：Wifiルータ(会場のWifiが使用できない場合があります。テザリング等ご協力頂ける方、よろしくお願いします！)
・欲しければ：おやつと飲み物

事前準備

・基本的には上記条件のPCがあればOKですが、以下の準備がしてあるとよりスムーズかつ楽しいかと思います。

☆Xcodeのイメージ(.dmg)ファイルをダウンロードしておく
　→ご自分のOSバージョンに合わせたものを必ずダウンロードしてください
　→Lion以降であれば、AppStoreからXcodeが落とせます。
　→それ以前は非推奨ですが、どうしてもというひとは、Apple develper登録(無料)をし、開発者ページからダウンロード可能です。

☆Apple developer program登録(有料)
　→事前に登録が済んでいると、環境構築編での実機デバッグ実行がその場で試せます
　→登録時、住所などは必ず英語で登録しないと、うまくいかないようです。
　→登録は有料ですので、ご自分の責任で行ってください(本勉強会では登録は【必須ではありません】)
　→登録完了まで数日～数週間かかることがあります。
　→登録処理の中で疑問点などあった場合、当日ご質問いただくことは可能ですが、100%返答を保証することは出来ません。
