勉強のコツをつかんで、モチベーションをあげよう！

10月19日(日)にIPAの基本情報技術者試験が実施されます！あと11日。とうとう追い込みです。

ReptAns というアプリを公開した記念も含めて、下記の日程で基本情報技術者試験を題材にした勉強会を行います。
コンセプトは 「約一週間前のこの時期にモチベーションにテコ入れする！」 勉強会です。

イベント概要

日時 10月10日(金)　20:00～22:00
場所 株式会社スタジオ・アルカナ Barスペース
住所：東京都杉並区荻窪4-33-5 ジミーフードビル3F
会費 1000円(ただし、iPhoneユーザーの人はReptAnsをDLしてくれたら500円割引)
ReptAnsのダウンロードはこちら
備考 飲み物・軽食あり

内容

イベント内容は、まず過去問から40問くらいを抜粋したペーパーテストをしてもらって、なんとなく実力を把握してもらいます。

そのあと、「あと1週間あれば逆転できる？勉強のコツ！(仮)」という20分程度のセミナーをしたり、プロジェクターに映し出した問題をみんなでクイズ形式で解いたりとかを考えています。

コンセプトの通り、基本楽しんで、なんかちょっと勉強した気になって帰る、という会を目指してるので、「ガリガリ勉強したい」っていうよりは、「同じ受験者同志で交流したりして楽しもう」のほうが強いです。

お祭り好きなアルカナで、IPAの前夜祭のようになればいいなと思ってます！「IPAは受けないけどね」という冷やかしさんもOKです！
