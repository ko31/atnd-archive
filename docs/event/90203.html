ゾスクエが運営しているサーバーに特別にご招待！

ゾスクエが管理人を務める 『NextFutureサーバー』 略して 『ネクフチャ』 に １２人限定 で特別ご招待！
多分１２人の定員にはならないと思うので、少なかったら運営さんに少し来てもらって一緒にやってもらう予定です！

ゾスクエのYouTubeチャンネルは こちら ！

  • 参加条件
    1. ゾスクエのYouTubeチャンネルを チャンネル登録する
    2. ネクフチャのDiscordに 参加する
    3. Minecraft の正規アカウントを持っている
    4. パソコンを持っている

この４つの条件だけで参加できます！

注意事項

こちらは抽選ではなく、先着順となります。
定員に達した場合に参加を希望された場合、キャンセル待ちになります。
なお、YouTubeのチャンネル登録・Discordへの参加は各サービスのアカウントが必要となります。お持ちでない場合はアカウントを作成し登録・参加をお願いします。
ネクフチャサーバーはPCゲームソフト マインクラフトPC版 にのみ対応しております。スマホ版やWiiU、Switch版などには対応しておりません。
サーバーには割れアカウントなどの正規アカウント以外でのアクセスは出来なくなっております。正規アカウントを持っていないのに参加を希望される方はご遠慮ください。
参加登録をする画面のアンケートの２つ目の文が『ネクタイのDis…』となっているのですが、正しくは『ネクフチャのDis…』です。
