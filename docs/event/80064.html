こちらは「Microsoft協業事例と最新テクノロジーを学ぶセミナー & 機械学習ハンズオン」の案内ページとなります。
詳細・参加申込は下記URLからお願いいたします。
https://training.softopia.or.jp/event/20160810/


公益財団法人ソフトピアジャパンは、マイクロソフト岐阜イノベーションセンターと連携して、セミナー、および、ハンズオンを開催します。
セミナーでは、ソフトウェア開発ベンダー (ISV) とマイクロソフトが新規ソリューション開発面で協業した事例と、今後に役立つ最新テクノロジーの情報を発信します。
ハンズオンでは、最近注目を集めている機械学習分野において、マイクロソフトが提供するクラウド型の予測分析サービス「Azure Machine Learning」を体験学習いただきます。
是非、ご参加ください。

＜内　容＞

＜第1部＞
　13:00～　「新規ソリューション開発協業例と最新テクノロジー紹介」
　14:00　　　本セミナーでは、新規ソリューション開発をしたい方向けに下記のような情報を提供します。
　　　　　　　◆Microsoftが取り組むパートナー企業様とのビジネス協業事例
　　　　　　　◆新規ソリューション開発に役立つ最新テクノロジー情報

＜第2部＞
　14:30～　「Azure Machine Learning ハンズオン」
　18:00　　　Azure Machine Learningを活用し、ユーザーへ映画を推薦するシステムの構築手順を学びます。
　　　　　　◆Azure Machine Learning の概要
　　　　　　◆Azure Machine Learning ハンズオン
　　　　　　　・Azure Machine Learning Studioへ推薦システムのExperimentをインポートする　
　　　　　　　・インポート後のExperimentの解説
　　　　　　　・WebAPIの公開方法の解説
　　　　　　　・WebアプリケーションからのAPIの呼び出し

※ハンズオンの実施にあたり「無料のAzureアカウント」を取得してください。
　■　https://azure.microsoft.com/ja-jp/free/

※ハンズオン用のPCは用意いたしますが、数に限りがありますので、ご自身のPCをご持参いただける方はその旨ご連絡ください。
　PCを持参される場合は、最新のブラウザーをインストールしてきてください。
　（Internet Explorer、 Chrome、Safariのいずれか）

＜講 師＞

大田　昌幸　氏　
日本マイクロソフト株式会社 テクニカルエバンジェリスト
　スタートアップでのWeb開発経験を経て、2011年に日本マイクロソフトに入社。現在はソリューション開発企業向けのテクニカルエバンジェリストとして活動。東京工業大学や慶應義塾大学にて非常勤講師の経験も持つ。AzureなどのMicrosoft製品に加えて、社外の技術も加えたオープンな新規ソリューション開発を企業とともに企画・実現する活動に取り組んでいる。
