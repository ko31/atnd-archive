ウェブデザイン技能検定　3級対策講座
～最終チェック！学科＆実技：ここだけは確認しておこう～
  


2月21日(日)に実施されるウェブデザイン技能検定受検の前に、学科・実技の間違いやすいポイントを徹底解説します。
自己学習の最終チェックに、また、今後、受検を予定されている方には、受検勉強のコツをつかむために、ぜひご受講ください。


※インターネットの仕組みに関する基礎的な知識をお持ちの方、およびHTML,CSSの知識とハンドコーディングができる方を対象にしています。
※配布された講義用データは持ち帰ることが出来ます。ご自身でUSBメモリをご持参いただくか、会場PCからご自身のWebメールサービスをご利用ください。  


【主催】 株式会社クリーク・アンド・リバー社
         プロフェッショナルエデュケーションセンター


【講師】 内田祐生 ( 有限会社デザインフリート )


【日時】 2016年2月6日（土） 13：00～16：00(全1日・3H)


【場所】 C&R社 3号館4F トレーニングルーム
         東京都千代田区麹町３－６ 住友不動産麹町ビル３号館
　　　　　https://www.c-place.ne.jp/0020Access/
　　　　　※●オレンジ色の丸印です


【参加料】7,500円（税込)
教材：「ウェブデザイン技能検定3級ガイドブック」を含む 
※すでに教材をお持ちの場合：参加料5,000円（税込）
※講座では、2014年8月6日発行 第4版 のテキストを使用します。）


【対象】
・ウェブデザイン技能検定3級を受検される方。受検を考えていらっしゃる方。
・どのような対策をしたらよいか知りたい方
・専門学校等でウェブデザイン技能検定3級対策を担当している教員の方
・インターネットの仕組みに関する基礎的な知識をお持ちの方、およびHTML,CSSの知識とハンドコーディングができる方

【備考】     
受検を前提にこれまで自己学習をされてきた方を対象に、受検直前に効率的にまとめや確認を行うことを目的とする講座です。

講座の内容は、下記の知識やスキルがある方を対象としています。

HTML,CSSの知識およびハンドコーディングのスキルが必要となります。


-——————————————

■講座内容に関する注意

◇学科◇
・過去に出題された問題から出題の傾向を解説します。
・これだけは理解しておきたいという内容を解説します。
・問題のとらえ方、考え方を解説します。
・試験までの間に何をすればよいか、アドバイスを行います。
・次回の出題問題の具体的な予想（いわゆる山かけのようなもの）は行いません。
・ただし、ある回を境に突然内容が大きく変わったということはありません。
・過去問題の印刷物を配布することは著作権上の理由から行うことができません。（私的利用の為に皆様がプリントすることは問題ありません）


◇実技◇
・過去に出題された問題からどのようなスキルが必要か解説します。
・模擬問題を例に、作業の手順を解説します。
・試験までの間にこれだけはできるようにしておこうというアドバイスを行います。
・PCを使用した実習作業を行います。
・模範解答のようなデータを配布することはありません。
・学科同様次回の出題問題の具体的な予想（いわゆる山かけのようなもの）は行いません。
・ただし、ある回を境に突然内容が大きく変わったということはありません。

※模擬的なデータを使用して解説します。
※実際の問題の使用や出題予想などは行いません。
