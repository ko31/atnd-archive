■イベント内容

iPhone/Android向けアプリの開発にご興味のある皆さまへ。

アプリ開発に携わる方々の集いが、名古屋で行われます。
弊社開発者を交えた、MeetUPイベントです！

実際のアプリ制作事例をご紹介しながら、開発上での
大切なポイントや、秘話などを語っていきます。
もちろん、開発者同士の交流会として懇親会も
予定しております。

アプリ開発について興味がある・開発者へ直接聞いてみたい・
何らかの解決法をお探しの方など、フランクにご質問をいただき
ながら進めて参りたいと思っています。

お時間のある方は、ぜひご参加ください。
（本イベントは開発・Webデザイン系の方向けです）

その他各種ニュース・勉強会情報をお送りしております。
詳細はこちら → AteamDevelopers
