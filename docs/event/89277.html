内容詳細

  • Git/Github 入門（10：00〜13：00予定）

Gitの入門からGithub上でのプルリクエストの送り方、そのワークフローを学びます

  • Ruby / Ruby on Rails 入門（14：00〜18：00予定）

Rubyの言語の特徴を学びます。他の言語の経験がある方には最初は違和感があるかもしれないループの作法や

# Rubyでは1〜10の合計をこのように計算「しない」
sum = 0
for i in 1..10
  sum += i
end
sum
=> 55

# 代わりにこうする
(1..10).sum
=>55

全てがオブジェクトである

class Integer
  def +(obj)
    send :-, obj
  end  
end  
=> :+
3 + 5
=> -2

ということを学びます。

その後、Railsの基本を学んで、簡単なWebアプリケーションをクラウド上の開発環境を利用して動かせるところまでを学びます。
