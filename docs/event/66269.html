イベントの概要：

モバイルアプリのバックエンド共通機能をAPIで提供するBaaS「アピアリーズ」と、HTML5とJavaScriptでAndroid、iOSを始めとするマルチプラットフォームのアプリを開発できる「アプリカン」をハンズオン体験いただくワークショップです。

このワークショップではiBeaconと連携したO2O/クーポンアプリの開発を行います。また、アプリ開発はせずに、聴講のみの希望者も大歓迎です。開発には自信がないな…という開発者の方、あるいはディレクター、デザイナー、PMの方などもお気軽にご参加ください。ワークショップ終了後の22:00から交流会の場も設けております。もちろん参加費は無料です！

共催：

株式会社アピアリーズ
株式会社ニューフォリア

対象受講者：

- HTML5でのWebサイト/Webアプリの開発経験者。
- ハイブリッド環境とBaaSを利用したアプリ開発に興味のある方。
※デザイナー/コーダー/ディレクターの方も歓迎です。

ワークショップ概要：

ワークショップ形式でアプリカンを用いたアプリ開発の実践講義を行います。
HTML5とJavaScriptでAndroid、iOSで動くアプリが間単に作成できます。BaaSの「アピアリーズ」を用いてデータベースと連携したアプリの実装を行います。
ワークショップ内では、iBeaconと連携したO2O/クーポンアプリを開発。また、プッシュ通知の利用方法についても解説します。

ワークショップタイムスケジュール：

19:00～19:10　アピアリーズの概要紹介
19:10～22:00　Developers Workshop
22:00～23:00　懇親会（自由参加・自由解散）
