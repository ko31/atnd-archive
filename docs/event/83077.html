このレッスンは「自分という存在を丸ごと受け入れて、自分自身を大切に扱う」美容術です。

アロマセラピー・氣・呼吸法などの自然の美容法を用いたスキンケアを通じて、

年齢による生理機能の変化、感情の揺れ動き、肌質の変化に寄り添うことで

心と身体の調和を保っていくことができるようになる美容術。

オリーブオイルクレンジング・洗顔・化粧水・ビューティーオイルのシンプルな4ステップで

肌の美しさはもちろんのこと、心身のアンバランスを整えて内側から健康に美しくなります。

【自分という存在を丸ごと受け入れるってどういうこと？】

自分自身の心と身体のコンディションときちんと向き合い、あらゆる自分にＯＫを出すこと。

年齢による心身の変化や、感情の揺れ動き、肌質の変化といったコンディションと向き合い、

欠点やコンプレックスも全て「自分の一部」として大切にすることです。

【自分自身を大切に扱うとどうなるの？】

自分自身を大切に扱うと、今まで見逃していた心や身体、肌のサインに気づきやすくなるため

無理のない、自然な生き方が出来るようになるため、自然とコンディションが整ってきます。

自然な生き方が出来るということは、自分自身と調和しているということ。

自分自身と調和しているということは、世界と調和しているということ。

そのため、自然に周りが整っていき、周囲の人からも大切に扱われるようになります。

【なぜ、自然のものでケアするのでしょう？】

肌は肌だけで存在するのではなく、心は心だけで存在するのではない、

身体は身体だけで存在するのではない　　　　　―すべては繋がっている。

というのが、この美容術の基本理念です。

そのため、肌はもちろん、脳・ホルモンなどの神経系統にダイレクトに働きかける

アロマセラピー・呼吸法・氣によってわずか3か月間で、簡単に身体の機能を高め、

心を整え、肌を美しく保つことができるようになります。

【このような方におススメです】

●肌質が変わってきたため、根本から改善する方法を知りたい

●時間をかけずに美しくなりたい

●スキンケアだけで、肌・心・身体のコンディションを保ちたい

●極力シンプルなケアをしたい

●ただ、美しいだけでなく「幸せな美人」になりたい
