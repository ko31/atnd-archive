ゲーム業界コミュニティとは

ゲーム業界には横の繋がりが少ないと思ったこと、ありませんか？
所属や経験の垣根を越えて、ゲーム作りに関わる全ての人が自由に交流を広げられるコミュニティ。
ゲーム業界コミュニティは、そんなコミュニティ作りを目指しています。

詳細などを こちら で紹介しておりますので、合わせてご覧ください。

概要

  • ゲームプランナー勉強会は、ゲーム制作におけるプランニングやゲームデザインなどについて、
    知識や経験を共有し技術の向上を目指す勉強会です。

対象者

  • ゲーム制作に関わる以下の方々なら、どなたでも参加できます。
    • カテゴリー
      • 企画
      • プランナー
      • ゲームデザイナー
      • レベルデザイナー
      • ディレクター
      • シナリオ
      • 「自分はプランナーの仕事をしている！」と思われる方
      • 以前はプランナーだった、またはプランナーになりたい方
    • キャリア
      • コンシューマー、ＰＣ、アーケード、オンライン、モバイル、ソーシャルを問わず、ゲーム開発企業で勤務経験のある方。
      • フリーランスでゲームの開発企業と仕事をされている方。
      • その他、ゲームの開発企業とゲーム制作に関係するお仕事をされている方。
  • 学生の方は現在参加をご遠慮いただいております。なにとぞご了承ください。

目的

  • 知識と経験の共有
    ゲームプランニングについては、明確な方法論が定まっていないのが現状です。
    私たちゲームプランナーの一人ひとりが持てる知識と経験を積極的に発信し、
    共有した内容を現場に持ち帰り実践する。そしてまた勉強会などでフィードバックする。
    これを繰り返すことによって、ゲーム業界全体のゲームプランニングレベルを向上させることを目指します。
  • コミュニケーションの輪を広げる
    残念ながら会社や業種の枠組みを超えたゲームプランナー同士の交流は、あまり活発とはいえません。
    同じ目的で集まり一緒になって考え協力しあうことで、コミュニケーションをより深めていただけると考えています。
    参加資格を限定することで多くのゲームプランナーの方々に興味を持っていただき、気軽に参加していただくことを目指します。

内容

  • プレゼンテーション
    ゲームプランナーの仕事にまつわる様々な事例について、発表をしていただきます。
  • ワークショップ
    現場での実践例を元に、ゲームプランナーの仕事に役立つ技術を体験します。
  • ラウンドテーブル
    ゲームプランナーの仕事について生じる疑問や提案などを、参加者全員でディスカッションします。
