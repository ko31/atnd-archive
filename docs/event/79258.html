nRF52勉強会

8/15(月) MakerLab Nagoya にてnRF52勉強会を開催します。

開催内容

  • EBSHCNZXZ Taiyo YudenもしくはnRF52-DKを使用して、nRF5 SDK v11でのプログラミングと、nRF52 / BLEの雑談がメインな内容の予定
  • 64MHz / EasyDMA / I2S / NFC / DSP(Cortex-M4F)など盛りだくさんのボードですので、楽しみながらまったりしましょう！
  • 64MHzです！省電力も良いですが、64MHzなんです！ブン回しましょう！64MHzですから！！（主催者的に）

必要機材及び会場諸注意

  • 太陽誘電のEBSHCNZXZ(Evaluation Board)もしくはnRF52 DK
  • Windowsが動くPC
  • nRF52 DKを使用時は電波出さないようにしてください。
    (nRF52 DKで電波出すならシールドボックスを持参してその中でお願い致します)
  • 参加費は無料ですが、募金をお願い致します。
  • 会場内に冷蔵庫があるので、100円で飲み物が購入できます

注意事項

今回の勉強会ですが、主催者はnRF52をまったくやっておりません！

ですので、nRF52に関して何か発表したいという方がいらっしゃいましたら、下記コメント欄にて絶賛受付中です。

進捗がなくても石を投げないでください。
