何か作りたい人が集まって、各自好きなものを作る会

もくもく会とは？

カフェとか誰かの家とかに集まって、それぞれ各自もくもくと勉強したり作業したり本を読んだりするだけの会のことです。

「家ではどうも仕事や勉強に集中できない、さぼってしまいがち」とか「一人だとついつい遊んでしまうけど他の人が作業してるのを見ると自分もやんなきゃなんないような気がする」とか「一人でタスクに向かう気の重さを人と集まることで紛らわせたい」とかそういう人にお薦めです。（ もくもく会とは？ より引用）

ジャンルは今回、コンテンツ関係（ゲーム、映画、ウェブサイト、イベント　など）に限らせていただきます。

参加費無料。PC持ち込み、飲食可。場内WiFi飛んでいて利用可。募集人数は8名（先着順）です。参加表明の際、できればどんな作業をするかも教えてください。質問などあれば@kawanakakiyoko まで。

参加条件

何かコンテンツ（ゲーム、映画、ウェブサイト、イベント　など）を作りたい人。商用か非商用かは問いません。コンテンツっぽいことであればたいがいなんでもOKです。

例
  1. プログラムを組んだりゲームを作る
  2. ウェブサイトにアップするイラストを描く
  3. DTMで音楽を作る
  4. イベントの企画書や映画のシナリオを書く
  5. 仕事のウェブサイトを作成する
  6. 芸大などの卒業制作
  7. 同人誌の原稿製作
  8. プログラムの勉強のために読書
    など

なお、下記のような作業をされる方はご遠慮ください

  1. 周囲に音が漏れる
  2. 会議や打ち合わせなど喋る必要がある
  3. 飲酒しながら
  4. 独り言が多い
