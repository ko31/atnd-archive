【セミナー詳細】
　　
コンサルティングや講座を販売しようと、セミナーを企画してみたが、お客さんが集まらなかった・・・
　
始めの方は知り合いが来てくれたけど、だんだん人が集まらなくなってきた・・・
　
そもそも、お客さんが来てくれるかどうかわからず、不安でセミナー自体が開催できない・・・
　
セミナー講師をやっていると尽きないのが、この「集客」に関する悩み。
　
これを解決するには、あなたのお客さんが自動的に集まる「集客の仕組み」を作ることが最善です。
　
セミナーや講演、イベントへの集客の仕組みを作ることで、次のようなメリットが得られます。
　
■セミナーが毎回満員御礼となる
■一度仕組みが出来ると、あなたの見込みのお客さんが自動的に集まってくるようになる
■あなたの知名度は積み上げ式に上がっていく
■知り合いに声をかけすぎて、嫌がられることがなくなる
■毎回集客地獄に陥ることがなくなる
■時間にゆとりができる
■お客様に良いサービスを提供できる

集客が安定することによって、結果として、あなたの収入は飛躍的に伸びます。
　
このセミナーでは、セミナーや講演、イベントへ集客するために、
どのようにインターネットを中心に情報を発信していけば良いのか、
またどのように継続して集客をする仕組みを作れば良いのか、
その方法についてお伝えします。
　　
そして、集客を大きくサポートする『最強集客サポートツール』をご紹介します。
　
このツールを活用することで、あなたは集客の仕組みを簡単に構築することが出来ます。

【講師プロフィール】
　
ビジネス構築プロデューサー、㈱スタックアンドスタック 代表取締役
　
１９７４年京都生まれ　東京西国分寺在住
　
「売れる」講座やコンサルティングサービス等を、クライアントと二人三脚でつくるビジネス構築プロデューサー。
　
会社員として働くかたわら、起業を模索し、様々なビジネスに取り組むも全てに挫折する。
　
しかし、2011年に短時間で効率的に稼げるアマゾンを使った輸入物販ビジネスに出会い、多くの仲間の助けもあり、起業独立を果たす。
　　
そして輸入ビジネスの自身のノウハウを、講座やコンサルティングサービスとして、販売しようとするがことごとく失敗。
　
そんな中、売り込まずに、見込みのお客さんを継続的に集める方法を発見し、自身の講座を、再販売後、わずか30日で売上月商300万円を実現する。
　
2016年、自ら成果の出た利益を上げる方法を体系化し、他の仲間たちのプロデュース活動を行う一方、コンテンツの商品設計と販売、Webマーケティング、コミュニティー創りを学ぶ、コンテンツエース塾を、 盟友の安井麻代や池田誠之と共に開始する。
