description 『土の灯/風の灯』キャンドルナイトは、滋賀県高島市の棚田を再生させようというソルビバ農園が主催する地域再生プロジェクトとして昨年に引き行うイベントです。今年も昨年の御堂筋イルミネーションを担当された照明デザイナーの長町氏や関西圏の大学生などの協力により、棚田の持つ美しさを引き立てます。『土の灯』とは棚田が持つ大地の美しさを照らす灯、『風の灯』とはこの地の外から来た人達がこの地の美しさに見せられて照らす灯の事を言います。昨年のキャンドルナイトでは、NHK番組「SAVE THE FUTURE」で中継され、さらに「100万人のキャンドルナイトの楽しみ方」にも掲載され、皆様に棚田や地球環境に対し考えるきっかけを御提供できたと考えております。土の灯と風の灯が混じり合い、琵琶湖を見下ろす美しい棚田の夜の風景、さらに参加者がキャンドルを持ち歩き風のように灯が舞う『風の灯』などを予定しています。
