■会場の情報追加しました！(2013/06/20)

東京駅ちかくの「らっきょう」というお店を予約しました。
http://r.gnavi.co.jp/gcgv200/

個室なのでNFC関連のおもちゃとかデモ機材とか色々もちこんで遊びましょう。
モバイルプロジェクター持っていくので即席LT大会とかも楽しいかも。

NFC対応端末もたくさん出てきて、
いよいよNFC元年ですね！（毎年いってますがw）

NFCラボ界隈からの派生イベントですが、方向性とか正直あんまり深く考えてなくて、
とりあえずNFCネタを肴に飲みましょうみたいな感じです。

4人あつまれば開催決定です。

ゆるすぎる感じであれですが、よろしければ是非ー。
