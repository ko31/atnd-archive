description

3月24日オープンソース・カンファレンス2012@愛媛の出展者および参加者で道後温泉の近所のゲストハウスで合宿を行います。
OSC会場の愛媛大学からは15～20分程度の徒歩圏内です。

場所：ふじ屋あつし

tel. 080-1750-5454
yado-fujiya@hotmail.co.jp

費用は素泊まりで2,000円程度を予定しています。

以下去年暮れの下見旅行で当宿に泊まられた東平さんのレポートの一部です。
;+

ここは WiFi もありますし、各ベッドに電源がありますので充電も可能です。
テレビは共用のが一台ありますが、私と岡野さんが宿泊した時は
誰も見ている人はおりませんでした。

また富士屋には共用のシャワーがありますが、
道後温泉の共同浴場の椿湯や道後温泉本館に歩いていけますので
そちらへ行くという手もあるでしょう。
椿湯も道後温泉本館も営業時間は 6:30～23:00 です。
私は同宿の人と一緒に椿湯へ朝風呂に行きました。

さらに道後温泉では毎月第4日曜日に朝市をやっています。
OSC愛媛は第4日曜日開催ですので、そちらを覗くのもよいかと思います。

ただ、伊予鉄の路面電車やバスは最終が早いです。
松山駅前を22:00に出る電車が道後温泉行きの最終です。
その時間を過ぎるとタクシーしか交通手段がないので
ご注意ください。
