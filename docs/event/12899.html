= イ ベ ン ト の 特 徴 =
(1) PCを持ち込んで自由に自身を売り込める！
このイベントの最大の特徴は、皆さんが主導となり自己PRができることです。
プレゼンテーションの内容では過去に作った作品や趣味や研究等で培ったITスキルを存分にアピールしてください。
また、一日で数回プレゼンを行うことになりますので、他の就活生に負けない経験値を積むことができます。

(2) 企業のエンジニア、社長や人事責任者と対話できる！
企業のキーマンと一対一で会うことができるので、通常の説明会では得ることのできない情報が手に入るはずです。
エンジニアの方が参加する企業もあるので、新らしい技術や新サービスの裏話が聞けるかもしれません。

(3) ITエンジニアを大切にする意欲的な企業と出会える！
就職イベントにあまり出ていない企業も参加しますので、この機会を是非有意義なものとしてください。

ご不明な点はなんなりとアカリクイベント運営事務局までお問い合わせ下さい。
