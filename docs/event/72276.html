結婚報告会をやりまーす！

2015/12/26(土) 伝説の加藤が「そういえば結婚しました」という報告をするのを忘れていた為、忘年会をかねたささやかなパーティを行うことにしました。場所はいつものあそこ！！普段会わないヒトも、この機会にご挨拶できたら嬉しいなッ！！

どんなイベントなの？

●イベントの概要

”結婚報告会”としていますが、結婚式でも披露宴でもないのでドレスコードは御座いません。新郎新婦も親の目も子供の目も届かないところで楽しんでやろうと思っています。ゆる～くご参加頂けます。

また、せっかく結婚報告会なので定番のビンゴゲーム的な何かをしたいと考えております。最初から最後まで楽しんで頂けるよう努めて参りますが、参加頂くみなさんは、カオスな交流会としてお楽しみ下さいませ。

●イベントの形式

立食でのビュッフェ形式となります。会場内に椅子は大量に設置しておこうと思いますが、おそらく参加人数分は御座いませんので譲り合ってご利用下さいませ。また、飲み放題もついておりますので飲み放題メニューからおかわりしちゃって下さい。ただし、帰り道は急な階段になっているので飲み過ぎると帰れません…

●イベント終了後も

イベント終了後も、お店はそのまま営業します。キャッシュオンで追加ドリンク可能ですので、そのまま閉店までお楽しみ頂いても構いません。（※新郎新婦はその時のノリ次第で、サッと消える可能性も御座います）

お申し込みについて

[申込方法]

参加ボタンをポチっと押して参加申込をお願いします。参加申込時にコメント記入欄がありますので、誰なのかがわかるように記載を頂けますと助かります。会費は会場払いとさせて頂きますが、2015/12/12(日)以降のキャンセルではキャンセル料を頂きますのでご注意下さいませ。操作方法がわからない場合はご質問頂きましたらご案内差し上げます。

[参加条件]

新郎新婦の関係者であればどなたでも参加可能です。参加希望者が増えすぎた場合は、なんとかします。恋人や家族、同僚を連れてきたい！という場合はご相談ください。

[お支払いについて]

結婚報告会は会費制で、当日会場支払いとなります。ご祝儀は必要ございません。領収書が必要な場合は受付時にご相談下さいませ。会費は6000円となります。
※参加人数次第では多少の増減の可能性がございます。

[キャンセルについて]

参加申し込み後のキャンセルについては、2015/12/12(日)迄にお願いします。以降のキャンセルはキャンセル料金が発生いたしますのでお気をつけください。

当日について

[会場案内]

会場は加藤主催の飲み会でお馴染みの「CROSS BEAT Hommachi(クロスビート本町)」さんをお借りします。

堺筋本町駅と本町駅の中間地点、中央大通りの北側の道、クロスビートの看板を地下に降りたお店です。

[受付のお願い]

階段を降りていただいた後、会場入り口にて受付をさせて頂きます。受付にて申込名とお名刺を頂戴致しますのでご準備下さい。お名刺が無い場合はノートに記帳をお願い致します。

その後、会費を頂きましたら手にハンコを押します。ハンコが飲み放題のチケットがわりになりますので消さないようにしてください。

領収書が必要な方は、受付でご相談ください。パーティー終了時にお渡し致します。

[お楽しみ下さい]

パーティーは開場後30分程度で開始予定です。最初の乾杯から2時間を飲み放題&立食ブュッフェとなります。

会場内には椅子を設置しますが、全員分足りない可能性も御座います。譲り合ってご利用下さい。

[不明点があれば]

その他、不明点等あればお気軽にご相談ください。

幹事チーム募集

今のトコ、セルフプロデュース(ﾟ∀ﾟ)！パーティーの運営を手伝ってくれる方を募集します。幹事チームの方には素敵な特典を用意するつもりだぞ！！

幹事チームにご参加頂けるという方は、個別にご連絡をお願い致します。
