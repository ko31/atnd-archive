description

「管理画面」は日々の業務に直結するとても重要な任務を担っておりますが、各社ニーズやターゲットが違えば管理したいデータや指標も異なるため、各社の取り組みについてなかなか話題になる機会がありません。

このイベントでは、管理画面を通してどのような取り組みをしているのか、そこに秘められたケーススタディ・エピソード等をゆるく語れる会があれば良いなと思い、今回開催する運びとなりました。

UI/UX、フロントエンド、バックエンド、運用オペレーション、トラブったエピソード等々、管理画面にまつわる話題が中心の LT会 ＋ 懇親会 となります。
