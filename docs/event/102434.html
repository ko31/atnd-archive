12月23日(日)、さいたま市鈴谷公民館にて、モンコレ忘年会2018と題して、モンスター・コレクションTCGメインのイベントを開催します！

※12月17日：モンコレD体験会について追記しました！

◆大会情報
イベント名：モンコレ忘年会2018
日時：2018年12月23日（日、祝）　10時開場10時50分大会開始
会場：さいたま市鈴谷公民館
埼玉県さいたま市中央区鈴谷７丁目５−１１
ＪＲ埼京線南与野駅より徒歩10分、京浜東北線北浦和駅よりバス10分
参加費:300円(メイン大会の参加費です。ランブル、モンコレD体験会のみの参加費はありません)

主催：煙突工房
共催：ろくもんや
協力：株式会社ＫＡＤＯＫＡＷＡ ドラゴンブック編集部

◆イベント内容：
〇10時50分～　モンコレ忘年会大会（メイン大会）
レギュレーション：
モンスター・コレクションTCG 20ｔｈアニバーサリーのみ使用可、スイスドロー戦
2回戦目まではデック交換可能

第21回全日本選手権優勝者の和馬さん、第1回ふーりん杯優勝者のナリさんを招待しています！
2018年の締めくくり、モンコレ勝負をお楽しみください。

優勝賞品として、ドラゴンブック編集部様から提供いただいたデータから作製した、
優勝楯（図柄：魔剣姫ドリブラ）を提供予定です！
また、同等の楯を抽選でお楽しみ賞として提供予定です
（作製が間に合わず、後日のお渡しとなる可能性があります。ご了承ください）。
参加賞で20thパックなど提供します。

〇13時30分～　サブイベント開始：
■モンコレD体験会
ドラゴンブック編集部様の協力により、モンコレD体験会を開催します！

・人数：１回につき６名（３卓）

・開催概要
下記の３回開催
1）13:30-14:20
2）14:30-15:20
3）16:30-17:20

テスト版カードとルールブックをお渡しして、プレイしてもらう方式となります。
参加については、13:20より参加整理券を配布します。

■ランブルファイト
20thのデックはもちろん、A,S,G,オープン、ネオスタンのデックが使用可能です。
使用可能なレギュを対戦記録用紙に記載していただき、運営側で対戦可能な組み合わせをマッチングします。
お互いに使用するレギュを合意して対戦してください。
20th以外のレギュのルールは運営では関知しません。

ドレッドノートもOKです！

1戦ごとに20thパックと20thのプロモからランダムで1枚お渡しします(上限8枚)。
