●●●皆さまはじめまして！●●●

株式会社Arinosの金沢みどりと申します。
　
　
弊社では、アクセンチュア出身の代表のもと、コンサルティング事業を主軸に3か国6事業を展開しております！

弊社社員の半数以上はコンサルタントとして実際に案件に参画！
プロパー社員だけではなく、多くのビジネスパートナーの方に協力していただいております！
　
　
私の所属しているキャリアパートナー事業部ですが、
　
人材と案件のマッチングを行っているということもあり、
フリーランスとして活躍されているコンサルタントの方のお話をたくさん聞くことができます。
　
面談にも同行させていただいておりますが、
会社に所属しないで働くのってなんて怖いんだろう・・・っていつも思ってます。
　
あるフリーコンサルタントの方は、3か月以上は同じ案件に入らないことをポリシーにされている方で、
年に4回以上は面談を行っているとのこと
　
私もたくさんの経験を積んで、成果を出して、独立して、自分の力で業務改善できるような
かっこいいコンサルタントになりたい！と密かに燃えているのですが・・・
　
やっぱり怖い！！
　
「どれだけの成果を上げたら独立してもやっていけるの？」
　
「そもそも独立して案件に参画することなんてできるの？」
　
「全部自分ひとりでできるかな？」
　
「福利厚生や保証など無くなったらどうすればいいんだろう？」
　
こんな不安が時々頭をよぎります。
　
　
どれだけベテランのコンサルタントであってもそんな不安ありますよね？
　
　
　
●●●独立予定のコンサルタント向けセミナー開催したします●●●
　
　
11月1日（水）にセミナーを開催いたします！

《コンサルティングファーム勤務→独立→起業》という道のりを経験してきた
講師陣による独立起業のノウハウたっぷりセミナーです！！

フリーが直面する壁の乗り越え方のコツや

フリーランスを楽しく営むヒントをお伝えいたします！！

もちろんコネクション作りノ来場も大歓迎です！！

フリーになったらコネクションってものすごく重要ですよね！
