「Security Tech Lounge」とは

セキュリティ運用や設計構築開発、解析などサイバーセキュリティの業務に携わる方々の親睦と相互交流を目的としたイベントです。 セキュリティにまつわるTIPSやトリビアなどのちょっといい話から、 いろんな業種、立場の方を交えてのパネルディスカッションやライトニングトーク、 みなさんのセキュリティ知識を活かせる？ ミニCTFを予定しています。

おかげさまで第四回を迎えることができました。
今回のTechLoungeのテーマは「SOC」です。
SOCの現場の方をお迎えし、SOC構築から日々の運用、SOCエンジニアになった理由、会社の愚痴までSOC現場のリアルを根ほり葉ほり聞いちゃいます。

過去のテーマはこちらです。
第一回「インテリジェンス」
第二回「クラウド」
第三回「セキュリティアナリスト/リサーチャーの注目する脅威」です。

参加していただきたい方

- ITセキュリティのお仕事に従事されて最前線で戦っている方
- セキュリティシステム設計、構築に従事されている方
- 日々セキュリティプロダクトと戯れている方
- 日々バイナリと戦っている方
- エンジニア職で強くITセキュリティに興味をお持ちの方
※ 現場でヤラレテいる方を対象としておりますので、学生の方や営業職の方の参加はご遠慮ください。

持ち物

・ノートPC( 必須 )
ミニCTFを実施しますので、ブラウザの動作などに支障のない標準的なスペックのPCをご持参ください。また、CTFを解くためのツール等の環境をご準備ください。
会社PCでは制限がある場合もあり、個人PCをお持ちください。
*バイナリ解析ツール
*ネットワーク解析ツール
*プログラミング環境(python等)
etc

※電源の人数分確保が難しいため充電してお持ちください。

・モバイルWifi
会場ではフリーWifiサービスを利用できますが、同時に多くの方がご利用いただくと、接続できない可能性があります。可能であればモバイルWifi/テザイング機能をご持参ください。
