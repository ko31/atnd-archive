エコリクルート2016

イベント ユーザー

title エコリクルート2016
catch ～環境分野の新卒合同説明会～
description

出展企業はすべて環境事業を展開する企業です。事業内容や業界動向など企業担当者から直接話を聞ける場として好評いただいています。
就職活動に悩んでいる方には環境ビジネス業界専門のマッチングディレクターが就職相談・進路相談を会場で受け付けます。さらに、合同説明会終了後も企業担当者より直接スカウトのご連絡があります！

【参加対象】
2016年卒の理系・環境系学生（大学生・大学院生）
※以下のような希望・悩みをお持ちの方は今すぐ申し込み！
　「大学で学んだ知識・技術を活かした仕事に就きたい」
　「環境に貢献できる仕事がしたい」
　「環境分野で就職をしたいが、どの様な仕事があるかわからない」
　「どうやって募集を見つければいいのだろう？」
　「環境ビジネスの現状ってどうなんだろう？」

【プログラム】
第1部　12：00～13：30　環境企業からのプレゼンテーション
　・環境企業各社からプレゼンが行われます！
　・企業の取り組みやビジョンなどをこれまで知らなかった新しいきっかけが見つかります。

第2部　13：30～17：00　企業との自由交流会　＆　個別進路相談会
　・企業の担当者と直接話をすることができ、知りたい事、気になる事、何でも聞くことができます。
　・会場では、環境ビジネス業界専門のマッチングディレクターに就職・進路相談をすることもできます。
