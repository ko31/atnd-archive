description

【受講対象】
　主に電子材料・部品、デバイス、車載部品の他、電気電子装置を扱うメーカーの方

【講座のポイント】
　改正RoHSでは6物質を製品に含有させないというだけでなく、EU適合宣言とCEマーキングの導入という極めて大きな変更が加わりました。整合化規格など必要な文書類の官報告示など情報は整備されてきつつありますが日本など域外の製造者、あるいは関係者向けの資料はほとんどありません。

　本セミナーではRoHS規制の背景全般から見て、日本の企業関係者が本来どのようなことを求められているのかといった基礎的なところから説明していくこととします。

【プログラム】
１．RoHS指令の基本要求
　　1.1 適用範囲
　　1.2 用語の定義
　　1.3 製造者の義務
２．製品と欧州法の枠組み
　　2.1 新たな法的枠組み/NLF
　　2.2 RoHS指令と加盟国国内法
　　2.3 廃棄物管理に係るRoHS以外の欧州法
　　2.4 モジュールA
　　2.5 必須要求事項と適合性評価
３．適合宣言とCEマーキング
　　3.1 適合宣言と適合性評価
　　3.2 整合化規格/EN50581
　　3.3 技術文書化
　　3.4 適合宣言書/DoC
　　3.5 適合性の推定
４．製品カテゴリー
　　4.1 RoHSにおける除外
　　4.2 カーオーディオはRoHS対象か？
　　4.3 11の製品カテゴリー
　　4.4 カテゴリー8＆9
　　4.5 新たに対象になる製品は？
