description

ディズニーをもっと知る。
そこから見えてくるものはなんだ…！

子供の頃、何も考えずに楽しめたディズニーランド。
大人になっても“何も考えずに”楽しむことのできる場所。
一体なぜこんなにディズニーは人の心を掴むのだろう？






昔から積み重ねてきた「ディズニー知識」と「さえり的ディズニーの見方」、「キャスト経験を得て感じるホスピタリティ」等を含めてお話したいと思います。



気張らずにオシャレなセミナールームで、少人数でゆったりまったりやるかんじにしようと思います。もし分からないところがあればみんなで考える感じも面白いかもしれませんね。気軽な気持ちで参加してください。





★このセミナーに参加したらどうなるのか？

「ディズニーの見方が変わる」とか
「同じ曲をきいて何度も泣ける人になる」とか、
「ディズニーがもっと楽しめる」とか、
「自分の仕事への姿勢が変わるかも」とか、
「自分の夢についてもう一度考え直すきっかけになるかも」とか、
そんなかんじです。







ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー
テーマは三本柱でいこうと思います。


★ディズニーソングから知るディズニーの想い
私たちは普段、ディズニーソングを日本語で楽しんでいます。
これによって本来の歌が伝えたかった意味合いがしっかりと伝わらなくなってしまっているんです。普段聴いている歌に隠された意味を考えてみませんか？


★ディズニーランドのマニアック情報♡
ディズニーにはバックグラウンドストーリーが多く隠されています。その他に、パーク内には様々な工夫が。それらの情報も惜しみなく。


★ディズニーのサービス・ホスピタリティ
こんなところまで気を使っていたんだ！こんなことを考えているんだ！そんな風にディズニーの内部に少しだけせまってみませんか？もちろん、夢が壊れるようなお話はしませんが、ディズニーの“夢部分”だけしか見たくない、「経営」や「サービス」などは聴きたくない！という人にはおすすめしません。




ミッキーがいつも笑っているのは　実はすごいことだ！
なんてことまで触れます。

主催者情報

大宮冴理（おおみやさえり）
幼稚園年長から小学校１〜３年まで東京に住み、その間の約３年間、毎週末家族でディズニーに通う。（１５周年・フィエスタトロピカール・フィールザマジック・ファンテリュージョン時代です）
その後は転勤によりディズニーにはあまり行けなくなる＞＜。でも、その後もディズニーに対する思いは強く、大学は東京にでてきて、念願のキャストを経験。
ディズニーに慣れ親しんで育ったのはもちろん、その他にディズニーの経営・サービスに深く興味を持つ。好きなキャラクターは、まわりまわってやっぱりミッキー。世界中に彼ほどのスターはいない。とか言ってみる。
