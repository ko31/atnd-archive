description

●趣旨等
　何故接着するのかの疑問に応えることを目的に, 接着及び接着剤に関する理論的なことを解り易く説明し, 接着剤を選定するとき, 及び使用するときの指針になるようにする。特に異種材料の接着（金属とプラスチック）について詳細に説明する。
　また接着作業時に起こすトラブルについて触れ, 幾つかの接着剤についてトラブル事例を紹介し, 処理指針を示す。
