CLTT読書会 概要

B.Jacobs “Categorical Logic and Type Theory” (CLTT, ISBN: 0-444-50853-8) をテキストにした読書会です。

要領の詳細はこちらを参照のこと。

今回は1.3節、1.4節の節末問題を適当にチョイスして解きます。


  • 各節毎に、内容のおさらい30分、問題演習1時間という感じで進めます。

  • 時間進行はきっちりめで行います。30分で1から理解するのは大変なので、予習超オススメです。

  • おさらいでの説明係は立候補制です。２セクションを２名でお願いします。希望者は右の参加登録もしくは下記Googleグループにて表明して下さい。（ただ聞くだけよりずっと理解が深まります）

連絡事項

フォローアップのため、参加者はGoogleグループ『圏論飲み会』（こちら）にも参加されることを強く推奨します。
