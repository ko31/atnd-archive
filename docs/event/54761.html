catch 大阪のスリバチの空も広かった
description

東京スリバチ学会と大阪高低差学会による合同フィールドワークのお知らせです。今回の企画は、東京スリバチ学会の皆川会長からお声を掛けていただき実現しました。東京からも何名かが大阪に来られるということなので、大阪らしい場所がいいのではと考え、ルートを四天王寺周辺に決めました。ここは上町台地の真ん中、へそのような場所で、天王寺七名水があることからも水が豊かだった事が伺える地域。水は「スリバチコード」でも重要なワードですので、東京スリバチ学会ならではの視点で大阪の凸凹地形を歩かれるのではないでしょうか。「大阪のスリバチの空も広い！」とか言ってくれないかな…

ということで、今回も参加記念の缶バッチを用意する予定です。途中抜け、途中参加自由、会員以外の方の参加も大歓迎。とてもゆるいフィールドワークですのでお気軽にご参加ください。おそらくまだ残暑厳しい時期だと思われますので、くれぐれも無理をなさらずこまめな水分補給をお願いします。

開催日：9月20日（土）雨天決行（ただし悪天候の場合は中止） 
集合場所：天王寺公園・公園入口（天王寺ゲート） 
集合時間：12時15分（16時30分頃キューズモールで解散予定） 
参加費：300円（記念缶バッチ＋凸凹地図＋保険料）小学生以下は150円。
※天王寺公園の入場料150円が別途必要です。

※参加費はできるだけお釣りがでないようにお願いします。
※昼食を済ませてお集りください。
 ※万一の事故に備えて傷害保険に加入いたします。当日にお名前と住所を書いていただく必要があります。名簿はイベント終了後、新之介が責任をもってシュレッターで破棄します。
※住宅地、寺の境内を通りますので住民及びお参りの方の迷惑にならないようにご注意ください。
※参加登録がなんらかの理由でできない場合は、メールでお名前（ハンドルネーム可）と人数をお知らせください。
（ osakakoteisa@gmail.com ）
※下見に行って予習用の記事をホームページにアップする予定です。８月末頃になるかも…。

【懇親会のお知らせ】
フィールドワーク終了後、少し時間をおいて18時30分頃から皆川会長をはじめ、東京スリバチ学会の方々をお迎えしての懇親会（鯛よし百番、会費制・5000円程度）を予定しています。大阪高低差学会としても懇親会的なものはしたことがなかったのでいい機会かも。東京から来られる方の人数が決まりましたら、大阪高低差学会のホームページ及びFacebookページでご案内する予定です。人数に限りがありますがお気軽にご参加ください。
