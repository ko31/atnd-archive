同年代を繋ぐコミュニティ「85en / はちごえん！」の6月イベントです。
4月に桜丘カフェを貸切り開催した1stイベントは90名弱もの同年代が集まり、沢山の新しい繋がりが生まれました。
5月は30名程度が中野Mags集まり、互いのことをよく知るためのイベント「85jam！/はちごじゃむ！」を開催しました。

第３回目の6月は、より仲良くなろう！ということで代々木公園でピクニック？を行います。

みんなでご飯やお酒を持ち寄って、わいわいしつつ親睦を深めましょう！
（もちろん飲み過ぎたり、人に迷惑かける行為は禁止ですからねー！！）

85enは同年代が集まることで新たなことが生まれる、そういった目的でイベントを開催しています。
このピクニックも新しい出会いや仲を深める場になればと思っています。
