HTML5&JavaScript Hackathon in Fukui 2012

description

2月25日と3月4日に福井県でHTML5とJavaScriptをメインテーマとした開発イベントHackathon（ハッカソン）を開催します。

「Hackathon？レベルの高い人たちの集まりかな？そんなの怖いわー。ついてけない。」という心配はありません。Ideathon（作るもののアイデアを考え出しあう場）からHackathonまでの準備期間が十分にありますので、学生や初心者の方も安心してご参加いただけます。

メインテーマをHTML5・JavaScriptとしていますが、バックエンドにHeroku, AWS, GAE, Azureなどを使っていただいて自由なサービスを作っていただいても構いません。

Webサービス・アプリケーション、スマートフォンアプリ等、各チームで話し合って自由に開発してください。
他社の方とチーム組み、作りたいものを作れる貴重な機会だと思いますので、是非参加をご検討ください。
（Hackathonについて詳しくはこちらをご覧ください →http://www.masahiko.info/it/archives/000958.html ）

■日程（2日構成で、両日ともの参加が必要です）

Ideathon 2012年2月25日(土)　
　日程：9:30受付 10:00開始〜13:00終了
　※ランチタイムを含みます。
　※終了後も17:00まで会場は使えますので、引き続き準備をしていただいても構いません。

Hackathon 2012年3月4日(日)
　日程：9:30受付 10:00開始〜18:00終了
　※Hackathonのみの参加は出来ません。Ideathonにも必ずご参加下さい。

※2週連続の参加が難しい、または遠方で参加を悩んでる方はご相談ください。

■参加費

無料
※但し、ランチ代は各自ご負担となります（出前又は弁当等）。

■目的

  • 開発を楽しむ
  • デザイナー・技術者の交流を通じてスキルアップを図る

■対象者

  • テーマに興味のある学生、社会人ほか。
  • プログラマー、デザイナー、Hackathonに興味がある方。
  • 学生さんや初心者の方向けに、入門チームを用意する予定です。お気軽にご参加下さい。

■持参するもの

  • 2月25日は、名刺、筆記用具だけで結構です。ネットワーク環境を用意しますので、PCをお持ちいただいても大丈夫です。
  • 3月4日は、ノートPCなど開発に必要なものをご持参下さい。
    コンセントは各自1つはご用意しますが、2つ以上要な場合は電源タップをご持参下さい。

■タイムスケジュール

◇Ideathon （2月25日）

09:00 会場設営
09:30 開場・受付開始
10:00 開始
・イベント概要説明
・共同編集に使えるツール/サービスの紹介、デモ
・チーム分け
10:30 Ideathon
・グループに分かれて、作りたいアプリケーション/サービスのアイデアを出し合います。
・途中でランチを挟みます。会場で出前を取るか、弁当を持参してください。
近くにコンビニ、スーパーもあります。
12:30 発表
・グループ毎にアイデアを発表して頂きます。
・ここで決めたアイデアを、必ずしもHackathon当日に作る必要はありません。
13:00 終了
※終了後も17:00まで会場は使えます。引き続き準備をしていただいても構いません。

◇Hackathon （3月4日）

09:00 会場設営
09:30 開場・受付開始
10:00 開始
・Ideathonで決めたグループ毎に、自由に開発します。
・ランチは各自好きな時間に取って下さい。
※近所にご飯を食べる所もあります。出前も可能です。
17:00 発表
・開発したものを発表して頂きます。
・最優秀グループには、チーム全員にささやかなプレゼントがある・・かも？
18:00 終了
18:30〜 懇親会
