description

以前、渋谷と大久保のシステム会社さんの会議室で、分散システムという厚い本の輪読会に参加していまして、
セキュリティーの部分を担当しました。
あれから４，５年がたち、ブロックチェーン技術が注目を浴び、分散システムの古典に立ち返り、
もう一度読み直して、かつ、ブロックチェーンのソースコードの確認も並行してできばとおもいます。

分散システムの方は、こちらです。

分散システム　第二版 単行本 – 2009/1/1
アンドリュー・S・タネンバウム (著), マールティン・ファン・スティーン (著), Andrew S. Tanenbaum (著), Maarten van Steen (著), 水野 忠則 (翻訳), 佐藤 文明 (翻訳), 鈴木 健二 (翻訳)

ブロックチェーンの方は、実際にソースコードを読んでいくのを基本にして、
日本語の本もでていますが、どれも、さらっとしているので、どういう本があるよとかいうのだけ紹介できればとおもいます。
実際のソースコードを、ＰＣと、プロジェクターでみながらやっていく予定です。

曜日日時等、別日が希望あれば、コメントください。
