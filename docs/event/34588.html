HTML5勉強会 名古屋 について

会のご紹介

名古屋近辺で、HTML5を中心としたWebテクノロジーを共に学んでいったり、
一人で追いきれない情報をお互い共有し合ったり、
近くに居るからこそ一緒にできる、HTML5にまつわる色んなことを推進していく会です。

最初は初心者〜初級者向けが中心になるかもしれません。
もちろんある程度レベルの高い内容でも構いませんが、
できるだけ知らない人に分かりやすく伝えてみると、みんなハッピーになれるはずです！

詳しくはこちら（Google グループ）

今回の勉強会について

できれば月一で定期開催をしていって、お互いの情報共有の場にしていきたいですが、
まだまだこれからな面があるので、とりあえず1回開催してみて、
お互い今後こういうことやりたいよ、という意見も出してもらいつつ、
今後の進め方を考えていきたいと思います。

ただ、色んな方の話を聞く限りだと、需要は相当あると思いますので、
無理なく継続できる形を模索していきたいです。
ご協力していただける方はぜひともご参加ください！
