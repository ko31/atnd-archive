10/6 グローバルフェスタで、南アとアフガンをテーマにワークショップをやります！

JVCが支援する10カ国の中から、今回は南アフリカとアフガニスタンをとりあげます。
南アフリカでは、人々の食事をテーマにグループワークをします。
アフガニスタンからは、現地「女子会！」「ヒゲオヤジ会？」でかいま見た、意外な素顔をお伝えします。
昨日まできっと知らなかった南アフリカ、アフガニスタン。現地の人々のくらしを感じてみませんか？

グローバルフェスタ？

グローバルフェスタは、国際協力をテーマに、NGO、国連・政府機関、企業が集まるイベントです。
10/6,7(土日)、日比谷公園で開催されます。
今年のテーマは、「Think Global, Think Green 世界を変えよう。未来をつくろう。」
いろんな団体のブースや、ステージイベント・ワークショップを楽しみながら、国際協力を身近に感じることができます。
飲食ブースでは、各国の珍しい料理が楽しめます。

詳しくはこちら。
http://www.gfjapan.com/

JVCもブースを出します！

ワークショップだけではありません。JVCは今年も会場にブースを出します！
南アフリカのビーズ製品の他、パレスチナやラオスの刺繍雑貨、タイの自然染色の布製品などを用意しました。
ブースには、NGOスタッフやボランティアがいます。JVCは何をやっているの？ボランティアは何ができるの？など、ぜひお気軽にお声かけ下さい。
