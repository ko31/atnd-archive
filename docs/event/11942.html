Arduinoを使った電子工作「8x16ドットマトリクスLEDで遊ぼう！」

description

「ArduinoでLEDチカチカだけじゃつまらない。もっと面白いことをやってみたい！」と言う方にお薦めです。また、「電子工作をやってみたいけど、ハンダ付けって難しそう」と思っている方も大丈夫。誰でも簡単にうまく出来るコツを教えます。

１．部品をハンダ付けして、8x16ドットマトリクスLEDシールドを製作します。
２．文字表示やゲームなどのサンプルスケッチを動かしてみます。
３．サンプルの内容の説明やスケッチの作り方を解説しますので、あなたのアイデアで面白いスケッチを作って下さい。
