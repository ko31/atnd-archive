Cocoa 勉強会広島支部、ついに始動。


CocoaStudy *study = [[CocoaStudy alloc] initWithLocation:@"Hiroshima"];
[study start];

というわけで、広島界隈で Mac OS X や iOS iPhone iPad のアプリ を作るために Cocoa の勉強をする勉強会です。
Mac App Store も発表され、iOS だけではなく Mac OS X 自体の Cocoa アプリにも開発熱が back to the Mac してますね!

この勉強会は、お気楽に「前読書会(後述)」から始めたいと思います。

また、ちょこっとしたトピックなど、どんなものでもライトニングトーク (LT) のようなものを募集しております! お気軽にお声をおかけ下さい。
発表者がおられない場合は Nyoho が「Cocoa 超入門のさわりの初歩の第一歩」をやります。

内容

中心内容は「読んだ本/読みたい本を持ち寄って紹介するコーナ名付けて「前読書会」」にしてみます。

  1. ポジションペーパ
    を使った自己紹介 (可能ならば。関西Emacsで好評らしいので真似する?)
  2. 読んだ本読みたい本を持ち寄って紹介するコーナ名付けて「前読書会」
  3. 発表/LT (募集中!)
  4. 発表者がおられない場合は Nyoho が「Cocoa 超入門のさわりの初歩の第一歩 (軽薄版)」をやります。
  5. (アイディア募集)

持参物

  • ポジションペーパ A4 (可能ならば。事前に Nyoho にお送りくだされば印刷しておきます)
  • 最近読んだ本 (または読みたい本)

無料。しかも超高速 Wi-fi ネットワーク完備 (予定)

場所

広島大学西図書館3階 グループ学習室 (飲食禁止)

住所: 日本, 広島県東広島市鏡山1丁目7-2

とにかく広島大学東広島キャンパスの西図書館の3階を目指して下さい。

交通案内

  • 自動車: 地図の西図書館付近の駐車場に止めるのが best! (土日はゲートが開放されていますのですぐに入れます)→ 西図書館を目指す
  • 公共交通手段の場合: JR西条駅まで来る → ブールバール経由広島大学行きのバスに乗車 → 「広大西口」で下車 →西図書館を目指す

参考: 交通アクセス 広島大学東広島キャンパス http://www.hiroshima-u.ac.jp/top/access/index.html
