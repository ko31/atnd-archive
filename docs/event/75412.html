DNS温泉番外編(2016年春)


「浸透いうな」特論

RFC2181 に焦点をあてて DNS のキャッシュの動作を理解しましょう。
DNS権威サーバ引越し問題から、幽霊ドメイン名脆弱性、移転インジェクションまで。
参加者からのライトニングトーク (5分以上に時間延長可) も歓迎します。

DNS 温泉未経験者歓迎です。過去の DNS 温泉と内容は重複しているところが多いですが、過去の参加者でも疑問点などあればぜひご参加頂いて質問などしていただければと思います。
