【内容】

０．オープニング、会場説明、注意事項（14:00～14:10)

１．「基本をおさらい　EC-CUBEインストールから初期設定まで」　講師：sikaoさん (14:10～14:50)

ホスティングに備わっている簡単インストール等は使用せず、FTPでのファイルアップからインストールを行いショップをオープンするまでの流れと、初期設定として変更しておきたい設定箇所についても触れます。

２．「動画プラットフォームClipBucketを使ってEC-CUBEを効果的に演出」　講師：まいパパ (15:00～15:40)

動画プラットフォームClipBucketを使ってEC-CUBEをより効果的に演出する方法をお話します。

　ClipBucketの導入

　EC-CUBE側のカスタマイズ

３．「EC-CUBE + PHPUnit で実践テスト駆動開発」 　講師：みそごぼうさん(15:50～16:35)

EC-CUBE は PHPUnit を使用してテスト駆動開発できるよう設計されていますが、残念ながら活用しているユーザーは少ないと思われます。

すぐに EC-CUBE の開発に役立てられるよう、テスト環境のセットアップからテスト駆動開発の進め方、コツなどを解説します。 ・

４．質疑応答、ＱＡ、ライトニングトーク等(16:35～16:50)
