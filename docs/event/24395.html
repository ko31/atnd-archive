description

2011年のモンゴル武者修行ツアーの同窓会です。

光の館という天井が開く仕掛けの
ある美術作品である建物に泊まります。

12人まで宿泊できます、というか12人ぐらい
いないと貸切できないのでこの機会は貴重とも
いえる。

ドライバーがいれば東京から車を借りてもよいのですが
バスで越後湯沢まで行きレンタカーを借りて
光の館まで往復するプランが有力です。

宿泊費は一人当たり約5000円です。
※施設利用料金20000円+一人追加ごとに3000円/泊
20000円+3000×12＝56000円、タオル寝具500円/人

これに交通費と鍋買出し費、立ち寄り温泉費などが
つくかんじです。
※池袋－越後湯沢バス片道2700円

日曜日は越後妻有トリエンナーレエリアを巡って
帰還という予定。
車で往復するなら、群馬か埼玉かどこかに
寄るのもいいかも。
