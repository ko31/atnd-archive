※WordBench神戸コミュニティはこちら。今後の開催予定などの最新情報が手に入ります。
WordBench神戸

●どんな人が参加できるの?

初級から上級までどなたでも参加できます。
ただし、今回の勉強会は初級者にはやや難しい内容があるかも知れません。ご了承ください。
おひとりの参加、初めての参加も大歓迎です。
会の最初に自己紹介タイムを設けます。

●今回のテーマ

  1. Debug-Bar-Extenderの紹介 (by @HissyNC ) 中級:60分
    WordPressの処理速度計測や変数チェックなどができる Debug-Bar-Extender プラグインについて紹介してもらいます。
    　
  2. WordPressのバックアップについて
    後述の"[続]プラグインの紹介"のテーマが中止になりましたので、代わりにこのテーマを入れます。
    　
  3. DesktopServerの紹介 (by @pictron2009 ) 初級:30分
    XAMPPのようなAMP実行環境とWordPressがノンストップでインストールできる DesktopServer のご紹介。
    　
  4. [続]プラグインの紹介 (by @01v0101 ) 初級~中級:30分
    3/20の 第9回WordBench神戸 で大量のプラグインを紹介して頂きましたが、個々のプラグインの機能や使い方についてもっと知りたいという要望がありましたので、特定のプラグインに絞ってさらに時間を割いて説明していただきます。
    もし説明を聞きたいプラグインが有りましたらコメント欄に書いておいてください。
    前回紹介した プラグイン一覧はこちら
    　
  5. その他、もし時間が余れば、前回までの勉強会テーマで紹介しそびれた続きなどを。
    　

10:00開始ですが、会場は9:00から利用できます。自習などに利用していただいて結構です。
