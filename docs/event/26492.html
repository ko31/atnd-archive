2012年3月23日開催！cloudpack Night #2

Amazonが提供するパブリッククラウドサービス、Amazon Web Services（AWS）を実運用している各社自慢のTipsやノウハウをお互いに自慢し合い、さらにより良いアイデアを求めてディスカッションする場になればと企画した #1 が思った以上に大好評！調子に乗って #2 を開催いたします。
前回同様、勉強会形式で2時間程度AWSについて語り合った後は、より密な交流を兼ねて田町駅周辺で親睦会を予定しています。

豪華なゲスト陣が来場！人数限定の小規模なAWSの勉強会＆懇親会

ゲストとしてAWSからエバンジェリストの玉川様、堀内様をはじめ、ソリューションアーキテクトの荒木様、片山様、渡邉様の3名、忍者の松井様にご参加いただく予定です。

USTREAMでライブ配信します！

http://www.ustream.tv/channel/cloudpack

※一部セッションを配信しない可能性がございます。ご了承ください。

懇親会はcloudpack夜の男が盛り上げます

JAWS-UGの勉強会よりも懇親会への出席率が高い、cloudpack夜の男 @shouhei が、前回に引き続き、21時から始まる真のcloudpack Night（懇親会）を仕切ります。


注意事項
当日AWSのサービスやプロダクトについてのご紹介や説明はありません。ディスカッションにご参加いただくためにも、AWSのサービスやプロダクトについて必ず事前に一通りご理解いただいた上でご参加ください。

Amazon Web Services（AWS）　→　http://aws.amazon.com/jp/

※前回より開始時間を30分早め、18時30分からの開催となりますので、お間違えのないようにご注意ください。
