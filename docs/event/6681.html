＜セミナー概要＞

今回のセミナーは、Amazon EC2 / Amazon S3 に代表されるAmazon Web Services(以下、AWS)の概要、機能、
操作方法の説明と共に、実際にAmazon Web Servicesの利用・操作まで体験できる「AWSハンズオンセミナー」です。
『AWS』の利用方法を知っていただき、『AWS』の利用機会を増やしていくことを目的としています。

Amazon EC2やAmazon S3に興味はあるがまだ利用したことのない方や、クラウドを利用していかなければと考えているが実際に触ってみる機会が
なかったという担当者の方に最適です。
