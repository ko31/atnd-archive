CAT 15周年記念公演 「眠れぬ森の美女」

東京都町田市で活動するミュージカル＆演劇ワークショップCATの15周年記念公演！世界中で愛されている童話「眠れる森の美女」をモチーフとして、現代のＮＹに舞い降りたプリンセス・ステラの眠れぬ一夜をコメディタッチで描いた歌＆ダンス＆映像のコラボレーションによるファンタジーミュージカル！
