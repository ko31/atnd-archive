概要

Github Kaigi の後、GitHub社の方が京都にて、GitHubやAtomについての講演を行なってくれます。
内容は基本的に英語ですが、通訳付きですのでご安心ください。

主な対象者

  • GitHubを学びたいWebデザイナー、Webディベロッパー
  • GitHub/Atomを活用中の方
  • GitHub/Atomに興味がある方

スピーカー

  • Jordan McCullough (@jordanmccullough) プロフィール: http://training.github.com/trainers/
    • GitHub for Developers
  • Nathan Sobo (@nathansobo) プロフィール: Atomの開発者
    • Atom, the Programmable Text Editor
