大和セキュリティ場阿忍愚（バーニング）CTFの解説会＆パーティ

皆様、場阿忍愚CTFを楽しめていますでしょうか？
問題を解けなくて悔しい！！と思っている方はご安心下さい！この度、各問題の作者が直接解き方を解説いたします。
企画からサーバ管理、運営の裏話、様々な面白い話が沢山あるので、是非戯れにお越し下されね（遊びにお越し下さいね！）。

（情報を更新する場合があるので、このページと @yamatosecurity をご確認下さい。）

参加者

場阿忍愚CTFに参加した方。（ burningctf.yamatosecurity.com )

準備するもの

頭、Wifi付きのパソコン、Kali、セキュリティツール等
（パソコンを飽きた方は、パソコン無しでぼ～と聴いても結構です）

12:30　　オープン
13:00　　オンラインCTF終了・解説会開始
18:00～　パーティ

※初段以上の方は無料！

解説会が終わったら、全問題の解き方をオンラインで公開して、解けなかったサーバの問題等を最後に解けるようにサーバも暫く（一週間？）そのまま運用します。
（※運営等の楽しい話は会場に来て下さらないと聴けません。）
