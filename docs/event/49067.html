description

　「聴くこと」は全ての支援の基本です。しかし、我慢して相手の話に耳を傾けるのはつらいと感じている人も多いようです。聴かなければならないという思いが強いことがかえって逆効果になっているのかもしれません。本来、「聴くこと」は、相手と対話し、相手の世界を知る、という営みです。だからこそ、聴かせてもらうことは、聴き手の側にも、ある種の楽しさが湧いたり、感謝の気持ちも生じてくるものです。

　この講座では、特別な理論や考え方を学ぶのではなく、相手の言葉の一つ一つに目を向け、耳を傾ける実習を行います。そして、相手の心の動きに気づき、相手を理解することをあらためて学びたいと思っています。

　この講座は、既に現場の様々な場面で、相手の話を聞くという活動をされている方を対象としています。話し手と聴き手のロールプレイを検討するという方法ではなく、楽しめるようなワークや演習も活用して、聴くことを学び直していけるように考えております。
