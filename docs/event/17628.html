description

今回は、現在RC(リリース候補)版が公開されている「Rails3.1」をメインテーマにして勉強会を行います。

コンテンツ内容は「Rails3.1 Overview」と題して、Rails3.1はどう変わったのかを RailsCasts を利用して学びます。
そして、HTTP Streaming, Coffie Script, Sassといった、Rails3.1の新機能を実際に使ったチーム開発にチャレンジします。

また、今回の勉強会では、福岡県のRuby開発サーバの環境を利用して行います。
（Ruby開発サーバとはRubyビジネスに取り組もうとする企業等に対して製品開発と技術支援を目的として、福岡県が提供しているクラウドサーバです）

  • Ruby開発サーバに関してはこちらをご覧ください
    • 公式サイト

なお、Ruby開発サーバを利用するにあたって、Linuxの基本的な知識が必要になってきます。
そこで、始めにLinuxの基本的な操作なども学びます。
「Linux扱ったことないよ」という方には絶好のチャンスです。

普段Linuxを扱われている方も積極的に参加していただき、サポートしていただければと思ってます。

是非、ご参加ください。
