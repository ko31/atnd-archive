description

朝活マインドマップ＠池袋とは
リラックスした状態で頭と手を動かしてマインドマップを描き、役立つ情報整理を実践する会です。
対象はマインドマップ初心者の方となります。

■初参加者の感想
「うまくできるか不安でしたがどんどん出てきて新しいことも出てきたので気付いたことをまたテーマに決めてやってみたいです。とても楽しかったです」ともみさん
「描きながら引きずり出てくる感じで、一人の時でもがんばれる自分はどんな時か気づきました。教えてもらいながらやるのが良かったです」はるかさん

コーチング資格を持つ講師が主催、楽しいだけで終わらない役立つ朝活を提供いたします。
ワンコイン(500円)で朝をフル活用して、新しい刺激を取り入れてみませんか？

■こんな方にオススメ
・マインドマップに興味がある方
・企画アイデア力を磨きたい方
・新しい刺激を取り入れたい方

■他の勉強会との違い
・実践的で役立つ方法を学びます
・実践後のシェアの時間があります
・講師がいるので効率的に学べます

■内容
マインドマップの使い方説明→見本参考→ざっくりマインドマップを描く→シェアタイム→連想してひきずり出てくるイメージでマインドマップを掘り下げて描く→コーチン​グ3つの質問→問題と解決法を描く（チェック表になる）→深呼吸→お互いシェア→質問タイム→可視化して見返す・付け足す→日常に応用

■目的
マインドマップを使って情報整理、新しい刺激を取り入れ、日常生活・ビジネスをより良くすること

■対象
基本的にマインドマップ初心者の方になります。経験者にとっては原点に戻ったり今まで描いたものを掘り下げ付け加える場となります。

■日時
金曜日
午前8時15分～9時15分

■場所

池袋駅西口のスターバックス（徒歩3分）
東京都 豊島区 西池袋1-17-10 エキニア池袋

■学ぶ内容キーワード
マインドマップ、情報整理、刺激、傾聴、コーチング、時間を作る、尊重する、話す環境、最後まで聞く、判断しな​い、確認、 客観的、肯定的、ボディランゲージ、沈黙、コミットメント、NLP、質問力、可視化、シェア、連想、企画アイデア力

■参加費　
500円

■参加者感想
●ワクワクすること感動すること好きなことは何だったのか、出していくとそこにあるのは人と一緒にいて出会い学び仲間が増えることが好きなんだということとそんな時は一人の時でもがんばれる自分に気づきました。もっと自分を高めてそこで人に必要とされる、また会いたいと思われる、なりたい自分像がけっこう見えてきてどういう自分になりたい、したい、されたいというのがいっぱい出てきました。一歩踏み出せない、現状維持で満足してしまっている自分もいて、これからいい方向にもっと変わりたいと思いました。文章にすると続かないのですが今日みたいに連想していくのと教えてもらいながらやるのが良かったです。友達がやっているのを見て自分にもできるかなと思いながらだったのですが、引きずり出てくる感じで可視化されて他の分野の所と同じことが出てきてつながったりして、ずるずるって出てくる感じでした。共通点は頭の中だけでは整理できないので同じところに印をつけて見えるようにしてもっと描き足したいなと思いました。はるかさん
●自分の長所を掘り下げてみました。こんな一面もあるんだなと新しい気づきがあり、性格から派生して喜んでもらうことが好きと最終的にそこにつながりました。先延ばしにして止まっていることをやり続けていきたいと思いました。見たことあるくらいのマインドマップでうまくできるか不安でしたがどんどん出てきて新しいことも出てきたので気付いたことをまたテーマに決めてやってみたいです。とても楽しかったです。ありがとうございました。ともみさん
●登山から広げてキーワードをばーっと出していくと海・空につながって、そもそもアウトドア好きだった、友人と一緒にキャンプするのが好きだった、夏に行こうと思いました。他にも昔好きだったことも思いだし、やりたいことありすぎてできていないことがもったいないことに気づきました。今回は描き方のコツやKeep on keeping on.という言葉などを教えてもらいプラスの方向、自分に質問というところからまた描いてみたいと思いました。もりぐちさん
●マインドマップは聞いたことがあるくらいでやったことなく、時間が空いていたので興味があって参加しました。書くというのはいいですね。手帳には仕事のことをよく書くのですが忘れないためのものだったり、やるべきことだったりで今日みたいに性格やプライベートなど掘り下げて考えて書くことはめったにないので、けっこう気づきがありました。退屈せずおもしろかったです。さとるさん
●企画系の仕事をやっているのでアイデアを色々出したくて参加しました。7年前にセミナー行ってそれっきりだったので。手をつけていないやることたくさん出てきました。客観的に見えるようになり、自分のプロジェクトについても視野が広がり、こういったノウハウを吸収していく必要があるなと思いました。マインドマップは一人でやれなかったのでこういう会・環境があると続けられるなと感じました。てらださん
●教えてもらった方が手っ取り早いと思って参加したのですが、書くって大事だなと思いました。考えるだけで今まで終わっていて可視化することがなかったですし、どんどんイメージが出てきたので重要性がわかりました。人に幸せを運びたい、思いやりを持ちたい、信頼されるようになりたい、そのために約束を守る、私生活を変える、自分のくつろげる空間を見つけるなど方向が見えました。こうじさん
●小学校の音楽の先生をやっていて、今日知って活かせたらと思い参加しました。書いて気付いたのはけっこうやりたいことはうまくいっていることでした。そこからこうなりたい未来をもっと掘り下げてみました。認められ自信がついたことから人を認められる人間になりたいと思ったのと、「いつでも感動していたい！」と思いました。心に余裕が必要、時間を大切にすることが課題になりました。本を見た時は枝が広がらなかったのですが、今日みたいにアドバイスもらいながらだと枝が広がっていき参加してよかったです。しょうこさん
●転職について掘り下げていくと自分にしかできないことは何だろうかともっと掘り下げたくなりました。何のためにやっているのか、喜んでもらえることは何か、時間をうまく使いながらやりたいことをやるにはどうすればいいかなど、さらに付け加えていきます。たけうちさん
●もっと書いていく練習をして続けていこうと思いました。自分の長所、趣味、やっていることからつなげていくと人間関係をつなげる、情報やアドバイスを渡して喜ばれることがやりたいことだと思いました。しげるさん
●自分の将来について書き始めたら結構とまらなかったです。計画性が自分は無いなと思っていたのですが、マインドマップの描き方を教えてもらいながらプラスのことを書き連ねていくうちに建設的なブランチが伸びていき、大きな目標立てて、分割して、細部を考えるようになって、結果計画性がずいぶん出てきました。自分との対話でしたが恥ずかしがらずに褒められたことなど思いつくまま書いていけばいいんだなと思いました。今日は脳に刺激になりました。いけださん
