今回のRBCの勉強会では、な、なんとハードウェアの登場です。

軽量Rubyが組み込まれたボードを使用して、あんなことや、こんなことをやっちゃいます。使用するボードはこれ

enzi board

勉強会と前後して発売されます。なんという Good Timing !!
実際にカードにプログラムを仕込んで動作させるところまでやります。

ハンズオン！

さらに特別ゲストに福岡CSKの三牧氏をお招きして、ボードにかける熱い思いを語って頂きます。

  • まずは簡単にシステムの解説を。
  • みんなで作ってみよう。数人のチームに分かれて、実際に製作、発表。
  • 三牧氏による解説。

Ruby と言えば Rails 。とは、ちょっと違った Ruby を体験してみませんか。

対象

  • 組込み系のプログラミングをされていて、実は Ruby に興味が有る方。
  • Ruby on Rails で Web サイトは構築したことがあるけど、実はハードウェアに興味が有る方。
  • コンピュータで何かをコントロールしたり、されたりを Ruby で試してみたい方。
  • 通常の Ruby よりも制限のある 軽量Ruby で自分にプレッシャーをかけてみたい方。

軽量Ruby とは

  • Ruby を組込み開発に適用できるように軽量化し、既存のC/C++アプリケーションと共存できるようにしたものです。
  • 高い生産性、可読性は Ruby の良さをそのまま引き継いでいます。
  • インタプリタ型で動作する他に、コンパイラと仮想マシンによる実行モデルでも動作できます。
  • MITオープンソースライセンスです。

背景

軽量Rubyの維持・メンテナンス、エンジニアのスキルアップ・普及活動を行うために、2013年１月にNPO法人軽量Rubyフォーラムが設立されました。

http://forum.mruby.org/

今回は、軽量Rubyフォーラムの団体正会員である株式会社福岡CSK様のご協力のもと、enzi boardの発売と併せて、RBCの皆さんにも mruby を実際に触っていただく機会を持つことができました。
