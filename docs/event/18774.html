catch ネットで生中継が簡単にできる！ビジネスに活かす！
description

Ustreamとは、スマートフォンのカメラや、PCに接続したWEBカメラを
使って、簡単に世界中へ生中継ができるサービスです。
無料で生中継ができますし、中継に対してリアルタイムでコメントを
することもできますので、双方向のコミュニケーションが活発に
行われています。

川井氏はUstreamの日本での伝道師とも呼ぶべき存在で、
“Ustreamer”と称されるUstreamの利用者を多数養成してきました。

Twitter上で20,000人近いフォロワーを持ち、川井氏が一声
かけると、Ustream上の生中継に数百人が集まるほど
人気がある方でもあります。

@takuyakawai

そして、あのソフトバンクの孫社長に「Ustream中継専用のスタジオを
作りましょう」と提案して、「やりましょう！」と言わせた
熱い人柄。楽しいセミナーになることは間違いありません！

今回のセミナーでは、広告制作会社出身である川井氏が最も得意とする
“Ustreamを活用したプロモーション・ブランディング”について、
存分に語っていただきます。

何といってもUstreamは無料で使えますので、起業家の皆様には
ありがたいプロモーションのチャネルです。
この貴重な機会をぜひお見逃しなく！！
