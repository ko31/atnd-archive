みんなで共に創る「ボードウォーク」

フジロックの森プロジェクトでは、毎年ボードウォークの整備を行いながらボードウォーク周辺の森林の整備～保全につながる活動を行っています。

年間を通してのべ300人を越えるボランティアの方々や地元苗場の皆さんと共に、車いすでの自然散策を可能にする木道をつくっています。

苗場の大自然の中で、参加者のみなさんとのコミュニケーションを楽しみながらボードウォークの補修活動をしませんか？
