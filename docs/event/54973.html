人数が少ないので場所を代々木上原に変更します。
新しい場所はメッセージでお知らせします。

このごろ話題のBLEビーコンを手作りする＆iBeaconとPassbook連携もくもく会をやります。

(注意：まだテスト版なのでバタバタすることを許していただける大きなハートの方の参加を前提としていますｗ）

開催日：8/16
開催時間：13:00-19:00くらい
懇親会ぽいもの　19:00から

参加費：500円＋ドリンク代（400円くらいから。ビールもありますｗ）
部品代：
Raspberry PI版　7000円 限定10セット
Raspberry PI, Bluetooth USBドングル、SDカード(8G)、USBケーブルx1を持って帰れます。

2. Aplix製 MyBeacon 版　1500円　限定8セット
Mybeacon, ボタン電池、ボタン電池ホルダー、スイッチ(2こ)、リード線を持って帰れます。
はんだごてなどはこちらで用意します。

どんな風にやるか

いくつかの島を用意します。自分のやりたいテーマの島で作業します。
なので何時に来ても大丈夫です。

その島のテーマが終わったら、別の島に行けますが、１つ大事なルールが
あります。

「移動するには、その島の誰かを教えること」

教えて移動できるようになるまでサポートしてくださいね。また島がいっぱいの場合は待っていただくことがあるかもしれません。ご了承ください。

1. はんだづけしてビーコンを作るよ島（部品代1500円）同時4名
AplixのMyBeaconを使ったビーコンを作ります。ケースはミンティアを使います。
ミンティアは各自買ってきてください(１つ80円くらい）
参加条件：特になし。はんだづけしても泣いたり、訴えたりしないこと。
所要時間：1時間くらいですが、半田ごてが4セットしかないので順番待ちの可能性があります。

2. ミンティアビーコン設定島　同時4名
１で作成したビーコンを自分用に設定します。
参加条件：１の卒業生。スマートフォンを持っていること。アカウントの作成、設定アプリのインストールが必要です。
所要時間：30分くらい

3. RPビーコン組み立て島（部品代7500円）同時4名
Raspberry PIを使ったBLEビーコンを作ります。
参加条件：特になし。
所要時間：15分くらい

4. RPビーコン設定島　同時１名（＋教える人1名）
３で作ったビーコンを設定します。
参加条件：３の卒業生。Linuxのコマンドライン操作とviなどのエディタが使えること。
所要時間：15分くらいですが、設定用PCが1台しかないので順番待ちの可能性があります。

5. Webサービス準備島　同時４名
Passbook連携Webサービスを構築するサーバーを準備します。サンプルコードのコピーまで行います。
参加条件：すでにApache/PHPが実行できるレンタルサーバーを持っているか、どこかのVPSなどのサーバー申し込みができるクレジットカードを持っていること。Linuxのコマンドライン操作とviなどのエディタが使えること。
所要時間：１時間くらい？

6. Passbook連携Webサービスもくもく島　２Fのカフェ
２，３で作ったビーコンと連携するWebサービスをサンプルコードから構築します。
参加条件：５の卒業生でPassbookとBLEに対応したiOSデバイスを持っていること。前回参加の方はビーコンを持参してもOKです。
所要時間：好きなだけ

参加する方は、申し込み時にビーコンの作成の有無を必ずアンケートで表明をお願いします。表明されない場合、参加できない可能性もあります。

またビーコン台数は制限がありますので、申し込み順にかかわらず、現場での先着順とさせていただきます。ご了承ください。

また「ちょっと手伝ってもいいよ」という方も大募集です。お手伝いの方は参加費が無料になります。こちらは先着5名にさせていただきますので、コメント欄でその旨申し出てくださいね。なお前回参加の方を優先としますのでご了承ください。
