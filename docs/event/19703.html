【講演主旨】
FMEAの帳票は埋めるだけでも相当の工数を必要とするが、その工数にみあった効果が見込めないのとの現実がある。この研修ではDRBFM(Design Review based on Failure Mode)を学ぶことにより、未然防止につながる創造的な手法としてのFMEAを学ぶ。
