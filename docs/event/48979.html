前回に続きアマゾン ウェブ サービス（AWS）の入門ハンズオンを札幌で開催致します。

アジェンダ

第1部セミナー

AWS概要と最新アップデート情報

第2部ハンズオン

第1章 Amazon EC2の基本操作

  • Amazon EC2とは？
  • EC2インスタンスを起動する
  • EC2インスタンスを操作する
  • 固定IPアドレスを割り当てる”Elastic IP”
  • SSHでログインしApacheをインストールする

第2章 Amazon EC2の運用方法 (基本CDP)

  • ある瞬間の仮想イメージを保存する”Snapshotパターン”
  • EC2インスタンスをひな型から作る”Stampパターン”
  • CPUやメモリなどのスペックを変更する”Scale Upパターン”
  • ディスク容量を変更する”Ondemand Diskパターン”

第3章 その他に押さえておくべきこと

  • Amazon EC2の料金体系
  • Amazon EC2をさらに使いこなす
