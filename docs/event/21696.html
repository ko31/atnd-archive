description

　厳しい就職難といわれる中で、学生の皆さんは
　「内定を取るため」の就職活動をしていませんか？
　
　せっかく時間をかけて、苦労をして就職活動に
　取り組むのであれば、同時に学べるビジネススキルや
　ヒューマンスキルを高めながら、内定を取ってしまいましょう！
　
　私たちは、業種や業態にとらわれずに、ビジネススキルを学ぶNPOとして、
　違う学校や志望の方たちとも学びあえる場を提供したいと考えています。
　
　今回は2013年度入社を目指す方を中心に、
　「就職で使える考え方と面接術」を一緒に学んでいきたいと思います。
　単なる座学だけではなく、実際に体験しながら学べるプログラムをご用意しています。
　
　就職活動のスタートを、私たちとみなさんで、一緒に切りましょう！

　《研修会》
　　第一部：就職支援講演会
　　　「誰も言わない、就職試験の正しい見方
　　　　　　　　　　～企業側から見た就職面接～」
　　　　　　NPO TOKYO SKILL UPs　理事長　長谷川啓介
　　第二部：グループワークショップ
　　　「面接官にもなってみよう！」
　《懇親会》
　　　・参加者の方と若手社会人を中心としたスタッフで
　　　　お食事をしながら懇親します。
