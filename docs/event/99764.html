本セミナーは、高度外国人材採用コンサルティング・グローバル研修を提供するK-NET株式会社と、日本最大級のエンジニア・IT企業、案件のマッチングプラットフォーム「アサインナビ」共催による、「成功するアジア圏外国人材の新卒採用」を学ぶイベントとなっております。今回は実際に海外学生を新卒採用した企業のご担当者の方にお越しいただき実体験を語っていただきます！お席に限りがありますのでお早めにお申し込みください！

【セミナー詳細】

こんな悩みをお持ちではないですか？

・ニーズやアイデアはあるものの、技術者が不足していてプロジェクト化できない

・海外人材を含め、優秀な人材を採用する戸口を広げたい

・トップ大学卒の優秀な若手人材を効率よく新卒採用し戦力化したい

経済のグローバル化や少子高齢化が進む中、IT業界は、人材不足という大きな課題にぶつかっています。 国内での人材確保が困難となってきている今、アジア各国から高度な技術や専門知識を持った外国人材を採用し、活用する企業も増えてきています。 一方で、高度外国人材を効率よく採用する方法や就労ビザ取得の流れ、そして受け入れを含め採用した人材の活用方法などが気になっておられる方も多いのではないでしょうか。

本セミナーは、「外国人採用を検討、あるいはすでに進めている企業」、「人材不足に悩んでいる企業」、「海外進出を考えている、あるいはすでにしている企業」において採用を担当されている方や、人事部門に所属されている方向けの内容です。高い潜在能力を持つ優秀な若手人材を採用したいと感じられている方を対象にした内容を、延べ1,000人以上の外国人採用支援をご経験された株式会社K-NETの川村 慶様を講師にお招きしてお話しいただきます。

なぜ海外の優秀な人材を採用するのか

どのようにこれらの優秀な人材を採用するのか

採用後にどうすれば効率よく戦力化できるのか

特にフィリピンからのトップ大学卒業生採用支援のご経験をもとに、上記の質問にお答えいただきます。

また、ソフト開発、ゲーム開発会社、その他 IT 系、組込み系開発、製造業、商社、金融系、コンサル系、教育（英語など）、ホテル・旅館、人材会社など幅広い業界における海外人材活用をうまく進めるためのポイントも、参加者のご希望にあわせてお伝えさせていただきます。

株式会社K-NET

代表取締役 川村 慶様
