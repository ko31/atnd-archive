description

　岩手県立大学の修士，修士を希望されている方を対象とした交流会を開催します．
　様々な方のクチコミを通じて参加者を募っていますので，普段接点の無い研究室の方などと交流を深めていければいいなと思っています．また修士を希望されている方は説明会などでは聞けなかった深い話などをこの場を借りて聞いてみてはいかがでしょうか．
　開催日は大学院入試の合格発表日としておりますので修士入学が決まった方のお祝いも同時に行いたいと思います．

参加応募締め切りは１７日まで
