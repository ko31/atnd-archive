◆セミナーの特徴
年金などに期待できない今、先を見据え、
如何に自己資金を増やそうと考えている人が、少なくありません。

また、既にリタイアし資金に余裕があるものの、
子供やお孫様に対する心配や、
将来待ち受ける介護などに備え考えているシニアの方も、
多くいらっしゃいます。

「まずは貯蓄」ですが、低金利の中、大きく増えることは期待できません。

その中、NISAの影響もあり、株式投資が今、改めて見直されています。
普段から新聞やテレビで多く取り上げられることもあり、
株式に興味を持つ方が増えています。

しかし、
　「何をしたらいいの？」　
　「どうすればいいの？」　
　「難しいそう…」　
　「怖い…」
という声を多く耳にします。

今、お読みになっている貴方。
あなたはツイています！

そんなあなたのために、
業界屈指のプロが、無料で株式投資の始め方を教えます！

これまで、シグマインベストメントスクールでは、
多くの金融機関（証券・銀行）や公的機関（金融庁・財務省他）の方に、
デリバティブを始め金融理論を教えて参りました。

今回、そのノウハウの一部を、これから投資を始める貴方にお教えします。

内容としては、
口座の作り方から、ファンダメンタル、チャートの見方や経済指標など、
簡単に分かりやすく説明いたします。

また、エクセルを活用することも多いので、
投資結果の評価におけるエクセルの活用法も、
簡単にお教えいたします。

証券取引を行ったことがない初心者の方、
パソコンが苦手なシニア世代の方に、
証券投資について一からご説明します。

ご興味のある方は、ぜひご参加ください。
