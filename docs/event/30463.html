Akasaka.scala とは？

隔週で開催しているハンズオン形式の勉強会です。初心者の方もお気軽にご参加ください。

http://akskscala.github.com/

今回の予定

主催者の都合により 7/12（木）から 7/26（木）に延期しました。

Play20 Scala の zentasks サンプル は Anorm を使っていますが、これを他の DB アクセスライブラリで書き換えてみるというハンズオンをやってみましょう。事前にやってみて当日発表、という方も歓迎いたします。

ちなみに主催者（@seratch）が開発している ScalikeJDBC の例だとこのようになります。
