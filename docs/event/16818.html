「今の給料で将来幸せな家庭を築けるのか」

「両親が元気なうちに親孝行をしてあげられるだろうか」

…なかなか給料が上がらないこのご時世、
将来のお金に関する悩みは山ほどありますよね。。

今から何か始めなきゃまずい！

というのはわかっているけど、だからって何をすればいいかわからない！
というあなた、必見です。

今回、
『年収200万からの「お金と人生を真剣に考える」講義』(明日香出版)
http://t.co/fXX8rde
の著者である田口智隆さんをゲストにお呼びして、
お金にストレスのない生活を手に入れる方法をわかりやすく教えていただきます。

今回をチャンスに、将来のお金と人生について真剣に考えてみませんか？
