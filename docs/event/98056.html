『現役で活躍中のクリエーターから、これから目指す方に向けて』

ファッション・グラフィック・アート・ウェブ・フォト・ムービー･･･
様々なクリエイティブな業種が集まる東京。

横の繋がり、情報の交換、今のクリエイティブが知れる場所として
このクリエーター交流会を開催する運びとなりました。

運営チームは有志で集まった現役デザイナーになります。

定期的に開催予定ではございますが、第一回目は代官山のカフェにて行います。
少しの時間で、素晴らしい縁が繋がることを願っております。

【例えばどんなことが出来るの？？】

◆クリエーターとしての参加◆
クリエーター同士だと状況や悩みも近く、一つの出会いが解決や先に繋がるケースが多くございます。

・直近のトレンドなど情報交換
・仕事の紹介や営業（自身の仕事のPR）
・新プロジェクトなどのメンバー探し
・将来への不安など自身が抱えてる課題への相談

◆案件依頼（クライアント）としての参加◆
業務委託や単発での仕事のご依頼など、人材派遣を通さずに知り合うことができます。

・クリエイティブな仕事の依頼
・参加者・主催者の人脈による仕事の紹介

◆クリエイティブな仕事を目指している方◆
クリエーターを目指している学生の方や、現在の仕事からクリエイティブな仕事へのジョブチェンジを
考えてる方など、実際に現役で活躍している方の話を聞くことが出来る貴重なチャンスです！

・クリエーターになる為の相談
・クリエーターの仕事の内容を知る
・自身のPR。仕事をお手伝いできる可能性も？
・将来への不安など、プロの方に相談できる

などなど、参加者の方々によりますが、様々なケースが考えられます。
