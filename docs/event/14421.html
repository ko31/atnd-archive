◆講師の言葉
　今、日本のものづくり産業の将来は世界的経済動向と関連してその浮沈をかけた瀬戸際にある、といっても過言ではありません。また国内的には真摯な技術者人口が、激変しています。世界的かつ歴史的な不況の中で生き残るためには、日本のものづくり・製造業も大きく変わらねばなりません。つまりコストダウンと信頼性・安全・品質を活かした高付加価値製品の開発・ものつくりです。
　本講座では、構造設計の信頼性向上と、ものづくり設計・コストダウンについてわかりやすく詳解します。又日本の産業の再起に心ある諸兄に対して、これからの日本のものづくりに必要な原理原則と方法論についてもお話し致します。

◆修得知識
・物を造るとは、よい設計とは、を分りやすく修得できます
・これからの日本の物造り分野全般をささえる初心の方に対する基礎技術全般について
・設計、開発の実務問題点と全社的コストダウンの要点
