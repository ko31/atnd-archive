description

起業という見えない山を登るあなたへ

自分を変え、やがては世界を変えたい。
人生とは挑戦の連続であり、今より成長したいという欲求が人生に輝きを与えます。

一方で、
◇会社を辞めて起業したい
◇会社の中で新規事業を立ち上げたい

そう思っても、現実には幾つかの壁があります。
家族の反対、上司との軋轢、友人からの好奇の視線。。。
思いを実現するのに、できない理由を探せばきりがありません。

でも少し立ち止まって考えてみてください。
あなたの起業を阻害する本当の壁は何でしょうか？
それは、不安や恐怖心など、自分自身の心が作りだしているものです。

とは言っても、起業や新規事業を目指す人は何も、
鋼の肉体と不安や恐怖心、痛みを感じない心を持った
サイボーグである必要はありません。
むしろ、自分の弱さと向き合いそれを認め、苦しい時は周りの力を借り、
ぼろぼろになりながらも前に進む、小さな勇気を持つことこそ大切です。

そこで今回は、登山家である栗城史多氏をお招きし、数々の体験を基に
限界を超えてもチャレンジする勇気をお伝えいただきます。

栗城氏は、6大陸の最高峰を登り、その後8000m峰4座を単独・無酸素登頂するも、
2012年秋、エベレストで重度の凍傷により手の指9本を失う。
しかしながら、2014年7月にブロードピーク8,047mに単独・無酸素で登頂。
さらには、今年エベレストへの5度目の挑戦を表明。

そこで語られる話は、生まれながら全ての才能に恵まれた
スーパーマンの自慢話でも、過去の栄光の話でもありません。

何度もチャレンジしては倒れ、傷つき、打ちのめされ、
自らの弱さと向き合いながらも、今なおチャレンジし続ける、
等身大の人間の物語です。

生きる意味を探しそれを実現するために、古い秩序に挑み
自ら出る杭となって行動するアントレプレナーの物語です。

道なき道を挑む全ての起業家にとって、今より一歩でも前に進める
勇気の源となる話を聞くことができます。

今後起業を目指されている方
起業して不安な毎日をむかえられている方
新規事業を考えられている方
古い秩序を破壊しイノベーションを起こしたい方、
ぜひご参加ください。

ここでの話が、これからチャレンジしようとするアントレプレナーシップたちの
「一筋の光」になりますように。
