ITベンチャースタートアップ時の夢と現実

イベント ユーザー
event_id 30929
title ITベンチャースタートアップ時の夢と現実
catch
description

近年日本のIT業界にてスタートアップとして自分のサービスを立ち上げていく人が
増えております。
その一方でサービスをクローズするケースも多く見られます。
その原因は、ビジネスモデルなのか、それとも技術力不足なのか・・・。
このセッションでは、数々のサービスの立ち上げのご経験された竹内真氏をお迎えし、スタートアップ時にビジネス面・システム面で何に一番力を入れるべきなのか、①立ち上げ期、②運営期、③転換期の3つのフェーズに分けてお話を伺います。
（①立ち上げ期はビジネス視点・システム視点の2回を予定しており、今回は前半のビジネス視点のお話に
なります）
また、会の最後にVCのライトニングトークを3～5社予定しております。

※講演が終わった後に希望者を募って懇親会を開催予定です。
　ご希望の方はコメントに「懇親会希望」と記載お願い致します。
　（3000～4000円を予定）
