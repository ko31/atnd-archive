【参加条件】

ハースストーンが好きな方(男性、女性、生死を問わず)

【開催内容】

ハースストーンを通じた交流、友達作りをメインにしており、18:00までに集まった方で総当たり戦の大会(賞品もあるぞ！)を予定しています。

また、本イベントは「燃えよ！炉端の集い！」サポートを申請しており、報告書作成のため写真撮影、今後のサポートのためバトルタグの収集をさせていただきます。写真撮影時には必ずお声掛けし、お面等で顔を隠すことも可能です。 バトルタグの収集をスムーズに行うため、参加コメントにバトルタグの記入をお願いします。 (例:HogeHoge#1111)

【開催場所】

フレンズ大宮会議室 埼玉県さいたま市大宮区下町1-43-7 南銀プラザ501号室

南銀座商店街の奥にある茶色レンガの5階建てマンション、南銀プラザの会議室です。

【炉端限定スキンの獲得について】

ウォーロックのスキン「ネムシー・ネクロフィズル」が獲得可能です。
