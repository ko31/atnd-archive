「Google Apps/Maps 導入活用セミナー」

イベント ユーザー
event_id 18824
title 「Google Apps/Maps 導入活用セミナー」
catch
description

このたび福岡にて「Google Apps/Maps 導入活用セミナー」を開催することになりました。社内システムやグループウェアの中核としてGoogle Appsの導入を考えている方や、 新しく地図サービスの導入を考えている方や、現在いずれかのサービスを利用中で内容を見直しを考えている方を対象に、Google Maps 活用の実際を紹介いたします。Google AppsやMapsについて学べ、具体的な活用方法を検討できる、またとない機会となっておりますので、皆さまのご参加をお待ちしております。
