旧東海道を有松から鳴海まで歩こうと思います。
旧東海道の史跡や名所はもちろん、おしゃれなお店を見たり美味しい食べ物を食べながら歩こうと思います。
基本街道筋を中心に歩こうと思いますが、桶狭間の古戦場の近くでもありますので要望が多ければ、桶狭間関連の史跡を含めて歩こうと思います。昔を知り、今を味わい、ウォーキングで健康な企画ですのでぜひぜひご参加ください。
