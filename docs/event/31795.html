みなさんは「夜のピクニック」って知っていますか！？

全校生徒が長時間かけて歩く高校の伝統行事「歩行祭」を通して、主人公たちが心の中に抱えていた人間関係の悩みを解決し成長する物語です。

実際に非日常の体験をすることでより深い人間関係を築けると思いませんか？

そのためにみんなで30km先の目的地までダラダラと友だちとおしゃべりしながら歩きつづける。
もちろんそれだけのイベントではなく、みなさんが最初に仲良くなりやすいようにコンテンツは用意してあります。

数十kmも一人で歩くのはすごいしんどいし、だるいと思います。
でもみんなと一緒なら乗り越えられる！
こんな辛い状況だからこそ、
初めて会った人と仲良くなり、もともとの友人とは親友になれる。

そんなイベントにしたいと僕たちは思っています。
共感した人は是非参加して下さい。
