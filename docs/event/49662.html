description

あなたはホームページから無限に注文や相談を受けとりたくありませんか？

「業者にホームページを作ってもらったけど全然反応がない…」
「営業で仕事を取っていくのはもう疲れた」
「インターネットから新規見込み客を集めたい！」

こういった悩みを解決するため、自動で相談が舞い込んでくるホームページの文章術（コピーライティング）を1から学べる入門セミナーを仙台で開催します。
あなたが自動で注文・相談が舞い込んでくるインターネットの仕組み作りを行いたいなら、ぜひお越しください。

ブログ、メルマガ、ホームページなど、インターネット上のコピーライティングに焦点を当ててお話します。

○コピーライティングで最も重要なマインドセット（心構え）とは？
●コピーライティングで一番気をつけるべき"バランス感覚"について
○お客さんに商品を売るために必ず越えなければならない3つの壁とは？
●あなたが書いた文章を最も美しく見せるテクニックとは？
○いくらコピーが上手くても商品が売れない根本的な原因とは？
●トイレのモップすら魅力的な商品に見せてしまえる強力なテクニックとは？
○私がコピーを書くときに採用している「二段階執筆法」について
●コピーに入れるだけでお客さんが興味を持ってしまう"魔法の数字"とは？
○お客さんの脳を麻痺させるコピーの最終兵器とは？
●お客さんを洗脳して商品を欲しくさせる方法！
○良いコピーか悪いコピーかを科学的に判断する方法とは？
●コピーライティングスキルを伸ばすために必ずやるべきトレーニング術とは？
○他人のコピーを合法的に盗んで商品を売りさばく方法とは？

などをお伝えします。
