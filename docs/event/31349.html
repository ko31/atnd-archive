1, 学生ヨル会議って？

私たち学生ヨル会議は「世代間格差から世代間協力へ」をテーマに世代を超えて様々な人が集まり、話す場所を作っています。
2012年の1月1日にNHKで放送された「ニッポンのジレンマ」を観た大学生たちがソーシャルメディア上で集まり、京都で学生ヨル会議が生まれました。
地域のコミュニティスペース等をお借りして、様々なテーマについて議論し、その様子をUstreamで全国に放送。地域間、世代間を超えた情報共有し、何か新たな気づきを生み出すこと目指しています。

1-2, 医療chについて

医療chでは、若者もオトナ、どちらの視点からも様々な角度から「医療問題」を考えます。実際の会場の議論と特別ゲストによるクロストークを組み合わせた新しいカタチのUstream番組放送です。
