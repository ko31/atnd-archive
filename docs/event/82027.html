開催概要

マスコットアプリ文化祭連動 プログラミング生放送勉強会、DMM.comラボで開催です！

プロ生勉強会は、IT・開発系ノンジャンルの IT 勉強会です。参加資格は、興味があれば OK ！　初めての方も参加お待ちしてまーす。いつも半数ぐらいの方が初めての参加です。

当日は、ニコニコ生放送で配信予定です。オンラインからも参加待ってます！　スピーカーへの質問・感想コメントもお待ちしています。

ニコ生 → http://live.nicovideo.jp/gate/lv280216122
