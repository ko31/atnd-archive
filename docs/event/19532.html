title Python初学者向け読書会＠札幌 #1
catch Pythonで学ぶ、初めてのコンピュータサイエンス
description

Pythonやプログラミングの基礎を学びたいひと向けの読書会を行います。

概要

  • Pythonを使ってコンピュータサイエンスの基礎が学べます
    • 基礎的なところから始めるので、興味があればプログラミングの経験がない方でもOKです
    • 広い内容を扱うので、将来的にPython以外の言語を使う上でも役に立ちます
    • もちろん、Python初心者がPythonを習得するためにもうってつけの内容です
  • 今回は初回なので、ランチミーティングの形式で行います
    • 次回以降は公共施設の会議室などを借りて行うことが多くなるかと思います
  • この勉強会はPython札幌の活動の一環として行って居ります
    • 興味があればGoogleグループ(メーリングリスト)にもご参加ください

教科書

  • 『初めてのコンピュータサイエンス (O’reilly)』を使用します
    • 各自持参が基本ですが、学生などで購入を躊躇される方は購入を急がなくても大丈夫です
    • 教科書の内容に納得した上で、懐に余裕ができた時点で購入してください

持ち物

  • 任意ですが、会場に電源と無線LANがあるので、PCを持参して使用できます
  • Practical Programming: installation instructionsを参考に環境構築すると良いでしょう
    • 当日までにPython環境が用意できない場合は遠慮なく相談してください

参加費

  • 1000円＋α
    • ランチ・飲物のセット
    • プロジェクター使用料(1000円)を人数で頭割り

次回以降について

  • 現在のところ、週末(土日)の午後もしくは夜に2時間程度の方針です
  • 予定はGoogle カレンダーで管理します
    • 個人的な用事や勉強会に参加する予定などがあれば登録してください
      • Python札幌Googleグループの初学者向け読書会トピックで発言した方が編集(登録)できるようにしてます
      • 都合が悪い用事があれば、遠慮なく登録してください

連絡

  • 全体への連絡や質問などは、初学者向け読書会トピックにどうぞ
  • 今回はうさみが代表してとりまとめてますので、個人的に何かあればこちらへ
