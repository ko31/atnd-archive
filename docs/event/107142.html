趣旨

温泉に浸かりながら泊まり込みでとことん DNS を学びましょう。DNS の基礎からセキュリティまで、インターネットシミュレータ (VITOCHA) 上で DNS 権威サーバやキャッシュサーバを動かしながら動作の理解を進めます。参加者からの発表、ライトニングトークも歓迎します。
今回も DNS 超初心者を主たるターゲットとしますので、ご専門の方やリピータの方は退屈かもしれませんが、夜の濃いコースもありますのでどなたでも遠慮なくご参加ください。
