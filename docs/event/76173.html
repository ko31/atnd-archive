業務系や受託系の開発から自社サービスの開発をしたい、 どうやってそれができるようになるのか

ご一緒にトライブの事業をケーススタディにして 情報交換をしましょう。
トライブでは、自社のサービスをWebサービス化
社内のシステムも含めて全てを自社開発運用をしています。

業務系や受託系の開発から自社サービスの開発をしたい、 どうやってそれができるようになるのか ご一緒にトライブの事業をケーススタディにして 情報交換をしましょう。

トライブはWebメディア・コンテンツサービスを行う企業と エンジニア・クリエイターを結ぶエージェント企業です。 トライブでは、 自社のサービスをWebサービス化 社内のシステムも含めて全てを自社開発運用をしています。

エージェントとエンジニアが共存し、新たなサービスを 今のサービスを改善するために日々、システム・サービス改善を進めています。

そこで見えた今後の新たなサービスとエンジニアの可能性を共有いたします。 自分の可能性を広げることを一緒に考えましょう。

弊社の開発環境をご参考までに共有いたします。

【開発環境】

・HTML5、CSS3、javascript、jQuery

・PHP　フレームワークも使用しています。

・仮想環境（VM ware、vagrant、KVMなど）

・redmine

・ソース管理システム（git、github Enterprise、svnなど）

・DB設計

・Slack

・Subversion

弊社は主にLAMP（PHP)環境で開発を行っております
