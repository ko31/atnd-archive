もくもく開発会#2を9月21日（土）にOpen Network Space 鎌倉にて開催します。

前回のもくもく開発会＠鎌倉の様子
普段とは違う環境で、もくもくひたすら1日開発しませんか？

海まで徒歩1分で、足を洗うシャワーも完備しているので、開発に行き詰まったら目の前の海で遊ぶことも可能です。

参加のご連絡をいただければ出入りの時間は自由ですので、みなさんぜひお越し下さい:)

※当日キャンセルはご遠慮ください。

【開催概要】

  • 参加費：無料（別途BBQも予定していますが2,000円程度の食材費等は実費となります。）
  • 参加対象者：Qiitaアカウントを持っている方
  • 定員：20名（※先着順、定員に達し次第締切ります）
  • 但し、10名以上のご応募があった場合開催します。
  • 場所：Open Network Space 鎌倉
  • 所在地：〒248-0013神奈川県鎌倉市材木座5丁目12-8 日時：9月21日（土）10:00〜21:00
  • 鎌倉駅からのアクセス方法
  • 参考：品川駅からの交通費は片道1,000円程度です

【タイムテーブル】

  • 10:00 オープン
  • もくもく
  • 17:00 〜 18:30 BBQ（開発したい方は開発）
  • 20:00 クローズ（予定）

【連絡事項】

  • 出入りの時間は自由としていますが、到着が13時以降になる場合は必ず事前に info@qiita.com までご連絡ください。
  • 海での海水浴は可能ですが、タオルの用意はありません。ご自身でご準備ください。
  • 15分ほど歩かないとコンビニがありません。飲み物の自動販売機はありますが
    食事等は鎌倉駅等で事前に購入しておくことを強くおすすめします。
  • コンセントは自由にお使いいただけます。冷蔵庫の利用も可能です。

皆さんのご参加お待ちしています:)
