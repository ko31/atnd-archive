Herokuのユーザー同士で気軽に情報交換できるような場、ということで開催されるMeetupの第二回目です。

RubyのPaaSとしてスタートしたHerokuですが、現在JavaやNode.js、Python、Scala、Clojureなど利用できるようになったので、「まだHerokuを使ったことはないけど興味はある…」という方の参加も大歓迎です。

ちょうどHerokuの Morten Bagai と Keith Rarick が来日しているので、Keithに話をしてもらいます。
ちなみに僕も最近知ったんですが。Keithは Beanstalkd の作者でもあります。

あとは5〜10分くらいのLTを4〜5枠ほどやる予定です。
