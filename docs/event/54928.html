description

山口県下松市にて、Web業界勉強交流会「ヤマグチ Web勉強虫」を初開催いたします。

参加対象者の方は、Webデザインを勉強中の方をはじめ、Web制作・SEOなどWebで仕事している制作会社やフリーランスの方、ネットショップなどでWebを活用している方など。

参加者が集まって「名刺交換・情報交換・ノウハウ交換 = 新発見！」をコンセプトに、様々なお話が伺えたらと思っております。

なにぶん初開催ですので、フタを開けてみなければ分からない部分もありますが、必ず新発見があると思います。

ちなみに参加者の方には、自己紹介をはじめ、仕事内容や取り組んでいる課題などをお話していただきます。

進行役の自分は、関西地方のWeb業界の動向や電子出版、有名人インタビュー裏技などのお話をさせていただく予定です。

また、当日語りたいテーマや他の参加者に聞いてみたい質問などがあれば、気軽にお寄せください。

・参加者による自己紹介（当日の参加者名簿を作成します）
・仕事内容の紹介
・現在取り組んでいる課題
・各自のWeb業界の持ちネタ披露
・質疑応答
・座談会
