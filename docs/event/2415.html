有限会社アドリブのkabayanと申します。

いつも一人でコードを書いているのはちょっと寂しいので、androidのアプリ開発の小さな勉強会＆ワークショップをしています。
申し訳ありませんが初心者の方は
Androidをイチからやってみる勉強会
2009年12月15日（火）　 http://atnd.org/events/2308
2010年 1月12日（火）　　http://atnd.org/events/2309
2010年 1月26日（火）　　http://atnd.org/events/2310
2010年 2月　9日（火）　　http://atnd.org/events/2311
2010年 2月23日（火）　　http://atnd.org/events/2312

への参加をお願いします。

今回はGoogle Wave(GAE)+androidということで、GAEで作ったロボット経由でWaveへ投稿するところの解説をします。また時間があればatndのAPIを使って検索と参加ができるatndクライアントを作ってみたいと思います。


場所：渋谷の弊社事務所兼サーバールーム
（１階が釣道具や、地下に村さ来があるビルの３階です）

時間：19:00くらいから終電まで（途中入場、退室OK)

参加費：
社会人１０００円(ビール、チューハイ１本つき）
学生 ３００円(ビール、チューハイ１本つき）

飲み物、夕食、お菓子など持ち込み自由ですが、こちらでもある程度用意します。終了後のチューハイ、ビール（発泡酒ですが）が1本ついています。追加は１本２００円です。


開催最低参加人数：なし（１人でもやってます）


※注意
定員は9人ですが、いすに座れるのは5人です。残りの方は申し訳ありませんが床になります。会場に先着した順なので、ご注意ください。
過去の経験上、キャンセル待ち１，２番なら多分入れます。

wifi, 有線LAN両方あります。

注意１：androidの開発環境が入ったノートPCは必ず持ち込んでください。
注意２：喫煙可です。禁煙な人は煙が平気な方でお願いします。
注意３：ゴミは基本持ち帰りでお願いします。
