Moodle講演会(初日)・Moodle管理者ワークショップ(2日目)を開催します。

Moodle講演会では、現在、参加者および発表者を募集します。

また、Moodle管理者ワークショップでは、参加者（受講者あるいはサポータ）を募集しています。

講演会申込は、こちら

ワークショップ申込は こちら　

※両日共に参加される方はお手数ですが個々の参加登録をお願いします。

内容：Moodle講演会（初日予定 10:10受付開始 10:30-12:30, 13:30-18:00）


  • 九州工業大学におけるMoodle活用事例(予定)
  • 利用制限と活動完了・コース完了の使い方と注意点（予定：ヒューマンサイエンス提供）
  • （仮）Webカメラを用いた講義アーカイブ（Moodleとの連携）（予定：佐賀大学、SRA西日本提供）
  • Apache・PHP・PostgreSQLのセキュリティを意識した設定（予定：ヒューマンサイエンス提供）
  • 発表枠(30分)を用意しています（発表者1,2名募集中）、Moodle座談会(内容提案募集中)

内容：Moodle管理者ワークショップ（２日目予定 10:10受付開始 10:30-12:30,13:30-16:00）


  • Moodle管理体験入門編(午前)
  • Moodle管理体験ワークショップ実践編（午後）
  • 上記ワークショップは管理者同士の教えあいを含みます

本イベントは、2013年度Moodleサブグループ企画セッション（AXIES年次大会）の結果を踏まえ、ネットワーク回線の安定した会場にて、再度Moodle管理者ワークショップを開催するものです。また、同時にMoodleミニ講演会も企画し、Moodleに関する情報交換会も行う予定です。


本講演会および管理者ワークショップでは、 Moodle管理者同士の情報交換（オフライン情報交換）とMoodle管理者のノウハウを交換することで、オープンソース Moodleの活用をより深めていくことが目的です。Moodleに興味のある方やMoodle管理に携わる方は経験年数を問わずご参加ください。技術職員の方のご参加も大歓迎です。

会場内では、貸出ノートPCあるいは持込ノートPC から仮想サーバへアクセス頂き、実際にMoodleの管理を実践（CentOS：MoodleのINSTALL環境は用意しておきます）する予定です。
