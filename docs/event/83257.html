description ▼イベント概要

「エンジニアが採用できない」

「エンジニアってどうしたら採用できる？」

「どのように企業ブランディングすればいいの？」

「どの媒体が採用に一番効果があるの？」

多くのIT企業が悩んでいるエンジニアの採用において独自の採用方法を行ってきた人事の方々に登壇していただき、それぞれのツール/媒体/手法によるメリットデメリットなどを根掘り葉掘り聞いちゃうイベントです。


19:00-19:30に登壇者と参加者の交流会を行ってからイベントを開始させていただくので、可能な限り早めに到着されることをオススメします！

TECH∞とは？
インフラ・フロント・アプリ・デザイン・キャリア・テクノロジー・マーケティング・ビジネス の8つのカテゴリーでテクノロジーを題材にイベントを開催するTECH∞(テックエイト) Branding Engineer がエンジニア/学生/フリーランスのために無料で開催する勉強会コミュニティです。 ご興味お持ちの方は是非1度ご参加下さい！

▼登壇企業（五十音順）

小山清和(http://goodpatch.com/jp)　

「大学卒業後は外食、人材、小売、などの現在の経験とは程遠い業界で経験を積む。
2011年からWeb界隈に足を踏み入れ、前職のヒトメディアでは２ヶ月間応募0と言うスタートから、採用上でのブランディングを地道に作り、最終的に年間10名程度の採用が出来る体制を作る。
2016年10月からグッドパッチに入社。現在は色々なキャッチアップに四苦八苦中」

勝股修平(http://www.opt.ne.jp/)　

2013年に新卒で株式会社オプトに入社。新卒採用担当業務に従事。
同年10月より中途採用担当に着任、60名規模の採用を担当。
2015年よりエンジニア組織「Opt Technologies」の立ち上げに参画し、現在に至る。

村上有基(https://monstar-lab.com/)　

株式会社モンスター・ラボ コーポレート本部マネージャー。
前職は衆議院議員秘書。2013年10月モンスター・ラボにジョイン。モンスター・ラボでは採用を中心として人事業務を担当。2015年6月に開かれたWantedly Awardにてモンスター・ラボをWantedly大賞受賞に導く。
これまでWantedlyを活用し120人以上の仲間を採用している。

ファシリテーター紹介

石倉 秀明(https://b-engineer.co.jp/)

早稲田大学からリクルート入社。
アルバイト、中途採用領域の求人広告の営業としてスタートし、チームマネジメント、事業企画、営業企画などを経験。2009年6月に当時5名の株式会社リブセンスに転職し、主力事業であるジョブセンスの事業責任者として入社から2年半で東証マザーズに史上最年少社長としての上場に貢献。
その後、DeNAでEC事業本部で営業責任者、新規事業、採用責任者を歴任し、退職。
2015年より独立し、複数のスタートアップの顧問として活動する傍ら、2016年3月株式会社働き方ファームを設立。株式会社Branding Engineer 顧問も兼任している。
