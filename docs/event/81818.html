半年振りにGMOインターネット次世代システム研究室エンジニア勉強会を開催です。今回は新技術を実サービスに利活用するために行っている内容について、各担当エンジニアからトークを行う予定です。

トークテーマ

Deep Learningによる株価変動の予想

研究活動で得られた知見を元に、最新の解析手法であるDeep Learning（RNN）の紹介し
ます。デモとして、RNNを用いた金融商品の特徴の予測します。さらに結果についての
考察を行い、予測改善に必要なポイントについてもお話したいと思います。
勝田 隼一郎

ブロックチェーンを利用したサービス開発について

Fintech などの分野で注目されているブロックチェーンですが、仮想通貨の管理をするだけではなく、スマートコントラクトと呼ばれるプログラムの実行基盤にすることができます。本講演では、ブロックチェーン、スマートコントラクトおよびスマートコントラクトを利用した開発中のサービスについて説明致します。
宮尾 武裕

Unity3DとOculus RiftでウェブVRコンテンツの表現

Oculus Riftをはじめ、VRヘッドマウントディスプレイが一般の消費者向けに販売されるようになり、2016年はVR元年と言われています。
また、Unity3DのゲームエンジンもVR（Oculus Rift）ネイティブサポートすることになり、VRコンテンツの開発を行いやすい仕組みを備えるようになっております。
今回はウェブサービスと連携しながらVR空間内でウェブコンテンツの表現についてVRコンテンツ開発仕組み・技術を検証したので、その結果をお話したいと思います。
チャン ドック ウィ
