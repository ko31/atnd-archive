認知症公開セミナー
『認知症と嗅覚障害の関連性について』

こんな方におすすめ！==========

※ご自身の周りに、認知症なのでは？と思われる人がいて、
ちょうど興味を持っていた。
※シニア層に関連が強く、商品そのものに興味がある。
※新しいレクリエーションアイテムになるものを探していた。
※会社の年齢層が序々に上がって来ているので、
新しく、介護と仕事の両立支援を検討している。
等々
==============

認知症の発症時に、嗅覚障害が起こる事は研究者の間では
古くから知られておりましたが、この事実は一般の方々には、
まだあまり知られておりません。
今回は、認知症と嗅覚の関連性について研究しておられる
昭和大学講師 神保先生に公開セミナーをして頂けます。

さらに本セミナーでは、
嗅覚テストにより、認知症の早期リスクを発見出来るキット
「はからめ(4,860円相当)」の無料体験も実施させて頂きます。

ぜひ、お誘い合わせの上、
お申込み・ご来場下さい。
