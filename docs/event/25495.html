catch 市場価値の第一人者が語るグローバル基準の能力の高め方
description

ビジネスのグローバル化が進む中で、日本も他国に負けないグローバルなビジ
ネスを展開することが必要不可欠です。そしてグローバルなビジネス展開を行
うためには、世界を視野に入れ、グローバルなビジネスを進めることのできる
「グローバル人材」が必要とされています。

従来のドメスティックな人材育成では、世界に太刀打ちできません。
グローバル基準の能力を開発することが求められます。

本研修では、グローバル人材を育成することの重要性を念頭において、市場価
値の第一人者、㈱企業変革創造代表取締役社長の藤田聰が、グローバル人材を
育成するための「グローバル基準の能力の高め方」を伝授し、グループ討議等
を通じて、「グローバル人材」を、参加者の皆様に体得していただきます。

特に
・グローバル基準での社外レベル・市場価値を知りたい方
・どの企業でも通用するビジネスリーダーを目指す方
・社外人脈を獲得したい方
・次世代幹部候補の方
・キャリアの節目を迎えている方など、必見です。

◆本研修の特長◆ ━━━━━━━━━━━━━━━━━━━━━━━━━━━

弊社では、グローバル人材を「多様性×人間力」で表現し、この２つを兼ね備
えた方こそ、真のグローバル人材として世界で活躍できると考えます。

本研修では、普段お会いすることのない様々な業種・職種の方と、グループワ
ークを通じて、互いの「差異」を受け入れ、「多様性」を認識することからス
タートします。

そして、リーダーシップ能力など、どの業界でも普遍的に必要とされる能力や
性格特性を詳細に測定できる弊社独自の人材評価技術　"市場価値測定テスト"
により、ご参加いただく方それぞれの「人間力」とその強み・弱みを明らか
にします。更に、このテスト結果を用いたグループ編成技術により、グローバ
ル人材として、人間力の多様性が最大限に体感できる研修を実現します。

◆統合型市場価値測定テストとは◆ ━━━━━━━━━━━━━━━━━━━━

「統合型市場価値測定テスト」は、弊社附属機関である市場価値測定研究所で
開発されたテストです。１９９８年のリリース以降、現在では１５０社以上５０
万人を超えるビジネスパーソンに受検いただいた実績の高い人材評価ツールです。

本研修では、自己の気づきと参加者同士の相乗効果をご体験いただくため、事
前にこの「統合型市場価値測定テスト」を受検いただくことが前提となっており
ます。
研修当日にお渡しするテスト結果の報告書は、５０ページ以上にわたり、

・ビジネス基礎能力、
・性格特性、
・ストレス値、
・ベンチャーマインド、
・推定市場価格

の5つの柱で7段階13項目におよぶ能力を事細かに測定した結果の他、これら
13項目の能力向上の指針となるガイドブックとなっております。このガイド
ブックに従い、長期的な研修効果をサポートいたします。

＜以下記事の内容＞
㈱企業変革創造の考える「グローバル人材」とは、単に英語ができるというこ
とではなく、営業・技術職関係なく、あらゆる業務において不可欠となるリー
ダーシップ能力やマネジメント能力といった全てのビジネスの根幹となる「ビ
ジネス基礎能力」が備わった人材であることを前提条件として考えています。
このビジネス基礎能力が高い人材は、国内外問わずどんな業務であっても高い
生産性を発揮し、成果の向上に直結するでしょう。
