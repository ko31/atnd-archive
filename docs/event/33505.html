description

Windows ストア アプリ を開発しましょう。

Windows 8が発売されました！　早速使っているという方も多いかと思います。
Windows 8から登場した新しいスタイルのアプリケーション、Windows ストア アプリ (Metro スタイルアプリ)をあなたも開発してみませんか？
この開発の会では、簡単なアプリの作り方から、アプリのストアへの申請までをレクチャーします。
「プログラムはほとんどやったことが無いけれど、XMLならわかる」というデザイナーの方でもお楽しみいただけます。

対象者

以下の条件すべてに当てはまる方

  • C#という言語でプログラムを書いたことがある人　または　XMLを扱ったことがある人
  • 学生の人
  • Windows ストア アプリ を作ってみたい人！

ゴール

用意されたテンプレートを基にアプリケーションを作り、Windows ストアにアプリケーション掲載の申請をします。

費用

無料

持ち物

必ず下記のものをお持ちください。

  • Windows 8とVisual Studio 2012がインストールされたご自分のPC（Windows 8は仮想マシンにインストールしても可）
  • 学生証（←10/31追記）
  • やる気

Windows 8の評価版はこちらから、<a href-“http://www.microsoft.com/visualstudio/jpn/downloads#d-2012-express”>Visual Studio 2012 Express for Windows 8はこちらからダウンロードできます。
絶対に現在お使いのWindowsからWindows 8の評価版にアップグレードしないでください。ダウングレードも製品版へのアップデートもできなくなります。
また、できれば参加者本人名義のクレジットカードをお持ちください。（支払いは発生しませんが、Windows ストアへの開発者登録に必要です）

場所

株式会社インターネットイニシアティブ(IIJ)様
住所: 東京都千代田区神田神保町1-105 神保町三井ビルディング

タイムテーブル

13:50 – 14:00 休憩
14:00 – 14:30 説明の部 （Windows ストアアプリの概要を説明します）
14:30 – 14:40 開発の部　イントロ
14:30 – 14:40 開発の部 （実際に手を動かしてアプリを作ります）
17:00 – 17:30 成果物発表
17:30 – 18:00 解散

講師

その他

Windows ストア アプリ はC++(/CX), Visual Basic, HTML5+JavaScriptでも開発できますが、今回はC#のみを対象とさせていただきます。
