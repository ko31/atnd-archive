　　　　　　　Facebookのシステムについて


　概要：かつては、Googleのシステムを理解することが、重要でしたが、現在では、Facebookのシステムがどのように機能しているのかを理解することは、とても大事です。講演では、FacebookのリアルタイムBig Data処理の試みである Real Time Insight の処理を紹介します。Scribe → pTail → Puma → HBase → Insight という流れのなかでMapReduce/Hadoopとの比較をまじえながら、AggregatorとしてのPumaの役割を解説出来ればといいと思っています。

　講演者：日本Androidの会　理事長　丸山　先生


　　　　Google I/Oから見るAndroid最新技術動向


　概要：Google Glass, Nexus Q, ADK2.0, Jelly Beans…

　講演者：株式会社GClue 代表取締役　佐々木　陽　様


　　　　スマートフォンテストラボみやぎのご紹介


　概要：公設試験研究機関では全国初となるスマートフォンアプリや機器テストのための実機検証設備について概要をお伝えします。

　講演者：宮城県産業技術総合センター　伊藤　利憲　様
