第20回NSEG勉強会 TDDBC長野0.1

NSEG は主に長野市で毎月IT系の勉強会を開催しています。過去の勉強会については http://nseg.jp/ から見ることができます。

第20回は TDDBC のようなことをやってみようと思っています。題して「TDDBC長野 0.1」です。

TDD

TDD とは テスト駆動開発 のことです。プログラムとして実行可能なテストコード(自動テスト)を書きながらプログラムを作っていく開発手法です。

午前中は TDD に関する発表やデモを行います。

午後の演習では参加者同士がペアになって、出題された演習問題について TDD でプログラミングします。

そのため、何らかの言語でプログラムを書くことができる必要があります。

また、その言語での Unit テスト系ツールの使い方を知っている方が望ましいですが、知らない場合でも、詳しい人が教えてくれるかも知れません。

運営側でサポートできるツールは今のところ以下の通りです。

言語 テストツール サポーター
Ruby RSpec @tmtms
C# NUnit @koty
VB.NET NUnit @koty
JavaScript Jasmine @ktz_alias
