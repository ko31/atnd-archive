第1部　画像局所特徴量・統計的学習手法による物体認識技術

【講演主旨】
　本講座では,近年のアプローチである画像局所特徴量と統計的学習手法の組み合わせによる物体認識のしくみについてわかりやすく解説する.輝度勾配をベースとした画像局所特徴量であるSIFT(Scale-Invariant Feature Transform) やHOG (Histograms of Oriented Gradients)を中心に画像局所特徴量について概説した後,このような特徴量と密接な関係にある統計的学習手法のAdaBoostについて物体検出を例に実践的に学ぶ.さらにKinect等の距離画像からの人体姿勢推定法を紹介し,近年注目を集めている学習手法であるRandom Forestsを用いた物体認識の最新研究動向を紹介する.

【プログラム】

1. 進化する物体認識
　1.1 一般物体認識とは
　1.2 一般物体認識の細分化

2. 画像局所特徴量と特定物体認識
　2.1 SIFTのアルゴリズム
　2.2 SIFTの高精度化(PCA-SIFT, GLOH)
　2.3 SIFTの高速化(SURF)
　2.4 SIFT以降のアプローチ(FAST, BRIEF, etc)

3. 画像局所特徴量と統計的学習手法による物体検出
　3.1 Haar-like特徴量と顔検出
　3.2 AdaBoost
　3.3 HOG特徴量
　3.4 Real AdaBoost
　3.5 HOGによる人検出
　3.6 第3世代の特徴量(Joint-HOG)

4. マルチクラス識別による画像分類
　4.1 Random Forestsのアルゴリズム
　4.2 Random Forestsによる画像分類
　4.3 Random Forestsを用いた人体姿勢推定(KINECT)

第2部　局所特徴量を用いた物体検出
【14:00-16:00】

講師:　オムロン株式会社

1.顔検出、
2.人検出
3.物体検出
4.ブースティング

【講演主旨】
顔や人を対象として、映像中から特定の物体を検出する技術が実用化されつつある。物体検出は、検出対象の向きの変化や照明等の外乱の影響下でも高い検出精度が求められる。さらに、実用化に向けては高速な処理や低メモリコストでの実現が不可欠である。本講演では、物体検出で実用化されている方法として、統計的な学習手法の1つであるBoostingと局所特徴量を組み合わせた方法を紹介する。また、実用化に向けた高速化や省メモリ化には様々な工夫が必要となる。それらの工夫についても紹介する。

【プログラム】
1.物体検出について
　1-1　物体検出の難しさ
　1-2　物体検出の流れ
2.局所特徴に着目した特徴量
　2-1　Haar-like特徴量
　2-2　HOG特徴量
3.物体検出の学習
　3-1　ブースティングについて
　3-2　学習の流れ
4.実用化に向けて
　4-1　高速化の方法
　4-2　メモリ削減方法
