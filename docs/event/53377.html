description

事業にとって大切なのは「人、モノ、金」そして「情報」といわれます。
インターネットが普及した現在、「情報」の価値は更に向上しているかもしれません。

本セミナーは、起業時に極めて有益な「情報」をお送りできればと思っております。
その情報は、人、モノ、金、その全てに適用可能です。

昨年度もっとも人気があった「創業補助金」。
約7,500名の方が、補助金200万円～700万円を獲得しました。基本、返済不要です。
残念ながら、2014年6月30日を持って予算が終了し、無くなってしまいました。

ですが、ご安心下さい。
一方で、補助金・助成金は年間で「3000種類」存在しています。
2014年7月1日には、「結果的に起業時に使える同じような補助金」が新しく発表されました。

補助金とは、3ヶ月毎に適用条件がどんどん変化してしまいます。
起業時期によってその条件はだいぶことなり、まさに「情報」の価値はこの部分にあるのだと思います。

●今回お伝えできる内容
￣￣￣￣￣￣￣￣￣￣￣
　・起業時に申請できる補助金
　　８月１１日〆切：ものづくり補助金（700万円）
　・その為に必要な融資（金融機関）に関する具体的な知識

今年は消費税増税もあり、景気対策の一環として補助金の実施数が大幅に増える見込みです。
弊社は、2013年4月から補助金申請支援事業を開始しました。
現在、月間数百社から、補助金の申請相談を受け付けています。
国内で最も多く申請支援をしているのではないかと思います。

元々が、弊社の「取引先支援」を目的にしたサービスだったため、現在でも完全成功報酬ないしは無料で申請代行を実施しています。

格安価格が理由だと思いますが、おかげさまで非常に好評です（成功報酬だと皆さん100%申請されます）。

※創業融資に関する説明も実施します。補助金を活用するには金融機関の面談を受ける必要があるのをご存知でしょうか。創業融資の仕組みを理解することで面談突破率も大きく向上します。

●本セミナーの主な内容

今年春に、ものづくり補助金が発表されました！
発表後、早々にものづくり補助金セミナーを開催しました。
当日70社の方が参加、その場で36社の経営層の方がお申込いただき、
一緒に申請に向けて頑張ってきました。

参加後の満足度はおかげ様で「100%」。
セミナーでは、単なる書類の書き方ではなく、

・補助金の考え方
・対象になりやすい「業種」になる方法
・補助金クイズ
・補助金を活用した新しいビジネスモデルの組み立て方
・質疑応答

「補助金を自社のビジネスに組み込み、経営を変える」
そんな点を意識してお話しています。

また、今回は、タイトルの通り「創業時に役立つ融資のはなし」も交えてお伝えします。

【Jマッチ事務局よりご挨拶】

Jマッチ事務局の白石と申します。
「日本の中小企業を全て黒字にする」を理念とし、いくつかのクラウドサービスを提供しています。
その中で2013年末より大ヒットしているのが、この「Ｊマッチ」です。
昨年は億単位の資金を、全国の中小企業や個人事業主の方に提供し、経営の安定化や新規事業の活性化に使っていただきました。

2014年は、そのノウハウを更に拡大し、全国の中小企業に「必要な資金」を適切に手配できるよう、各種セミナーにも力を入れて取り組んで参ります。
本セミナーが皆さんの企業経営にとって少しでも有益なものになれば、事務局スタッフ一同、大変嬉しいです。セミナー参加前に遠慮無くご質問等もお送りください。
