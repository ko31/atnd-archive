description

小雨決行。
ただし天候状態で選べるように遊びを持たせておきます。
7末から8月いっぱいの山小屋は既に満席ぽい。
ので、早朝発→夕方帰宅とします。

登山ルートは吉田口から。
http://ja.wikipedia.org/wiki/富士登山
目安は登り5H30、下り3H。

必須装備は以下

  1. 登山靴（捻挫防止のハイカット）
  2. 雨具(傘は役に立たない)
  3. 防寒具（雨具の下に着る）
  4. 懐中電灯（日が落ちると無いと死ねる）
  5. 手袋（スキー用が良いかも）
  6. 水（お茶はトイレ行きたくなるので避ける）

あると便利なもの

  • 帽子（タオル巻いても可。飛ばされないように）
  • アイウェア（紫外線＆砂埃対策）
  • 日焼け止め（空気が薄いぶんダイレクトです）
  • ストック（杖。あると楽）
  • スパッツ（notタイツ。砂走り対策）
  • カメラ（防水が良いと思うよ）

あとはよしなに。質問・相談はいつでも@sigwygまで。
