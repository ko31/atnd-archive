東京大学教養学部附属教養教育高度化機構アクティブラーニング部門・セミナー

■講演：東京大学大学院工学系研究科附属
　国際工学教育推進機構・バイリンガルキャンパス推進センターの試み

■開催趣旨
　東京大学大学院工学系研究科では、国際的な工学教育研究基盤を構築し、国内外の優秀な学生や教員が結集して、世界との国際教育連携を推進していくことを目的として、平成23 年4 月、国際工学教育推進機構を設立しました。バイリンガルキャンパス推進センターでは、教職員・学生への国際リーダー力強化、英語教育や講義の英語化など、国際化環境を組織的に整えており、今回はそれらの活動について、国際化推進部門長の森村久美子先生より、Global Ware Project やスペシャル・イングリッシュ・レッスン、e-Learning プラットフォームのSNOWBALLS など、バイリンガルキャンパスへ向けた多彩な取り組みについてご紹介頂きます。

■主催：東京大学教養学部附属教養教育高度化機構アクティブラーニング部門
■日時：2012 年5 月9 日（水）13:00-14:30 (３時限目)
■会場：東京大学教養学部21KOMCEE ２階 201教室
