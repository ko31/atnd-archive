浅草・田原町のコワーキングスペース
「 Bug Ticket Coworking 」さんで開催!!

東京浅草・田原町にある、隠れ家的コワーキングスペースさんにて実施させて頂きます！
おひとり様ノマド作業推奨とのことで、是非一度、場所の見学も含めてご参加頂けますと幸いです。


■こんな方にオススメです

・アプリの広告でどうやってお金稼いでいくの？ということを知りたい方
・アプリのマネタイズを最近担当することになった方
・広告ってどこに実装すればいいの？運用ってなに？という方


場所

コワーキングスペース Bug Ticket Coworking
東京都台東区寿１丁目１９−７ ※駅から5分！

参加対象

アプリ・Webコンテンツの開発を行っている方、またその予定がある方
※広告会社様は参加NGとさせて頂きます

参加料

無料（受付で名刺を1枚頂戴致します）

進行

ライヴエイド株式会社　GeneralManager 福島 智晴

プログラム

18:40：受付開始

19:00～

-

１．アプリ収益化の基礎知識 1部：用語編（45分程）

【予定内容】

・スマホアプリ/Webのマネタイズに必要な基礎知識

広告を取扱う際の高頻出ワードについて解説します。

・スマホ広告のお金の流れ

広告が誰にどのように買われ、アプリ開発者にお金が入ってくるのか、その一連の流れについて。

・現場の会話例

現場の担当者ベースで交わされているリアルなやり取りの事例を紹介。


-

２．アプリ収益化の基礎知識 2部：実装/運用編（45分程）

【予定内容】

・ページ別広告実装オススメ箇所

ページによってユーザーのニーズや訪問時間・頻度などは異なっているかと思います。
そうした背景からこんなページであれば、こういった広告実装がオススメ、など弊社の知見からお話させて頂きます。

・広告運用って??

主にアドネットワークの運用について。デイリー、時間帯によっても収益性の異なる各アドネットワークを、
どのように配信し収益最大化していくのかについて。

・よくあるトラブル、注意点

広告関連で社内・社外問わずよくあるトラブルや注意点についてお話します。

【こんな用語関連のお話です】

「アドネットワーク」「SSP」「メディエーション」「アプリ起動時」「アプリ終了時」
「アクション完了時」「バックグラウンド復帰時」「CPM」「フィルレート」など


-

３．参加者の皆さんでフリートーク（30分程）※お酒付※

【少人数なのでワイワイやりましょう】

　・どんなコンテンツを作っているのか
　・開発ここで詰まっちゃっている
　・マネタイズ、プロモーションがよく分からないんだが、皆さんどうしてる… など
