日付，定員，費用が変更しましたのでご注意ください

学生(小学生から大学生まで)を対象としたAndroidアプリ開発の入門のための勉強会です．
(ちなみに主催者も大学生です)
平凡な大学生が思いつきでやってみる勉強会なので過度な期待はせず，気楽に来てください

目標としては
・Androidアプリの開発環境の完成
・アプリ開発のおおまかな流れの理解
・Androidアプリが動作する簡単な仕組みの理解
としようと思います．

何かプログラムを書いたことがある人なら大丈夫かなと思います．
主催者も初めて主催する勉強会であり，またそこまでハイレベルなプログラマでは無いのであまり難しいことはしません(出来ません)．

この勉強会の目的は上の目標とプログラミングに興味のある学生のコミュニティーを作る助けとなることですので，Androidアプリ初心者の方でもバリバリ書けるぜって方でも興味があればぜひお気軽に参加してみてください．
(バリバリ書ける人はむしろ手伝ってくださるとありがたいです)

[あると良い物]
・ノートPC
・電源タップ
・ネット環境(WiMaxやイー・モバイルなど)
(当日はPCを使って実際に動作を見てもらえたらと思いますので，出来ればノートPCをご持参いただければと思います．)
また可能ならPCの中に以下の環境を整えてもらえれば助かります．
・JDK (Java Developers Kits)
・eclipse(Java開発の統合開発環境)
・ADT (Android Developer Tools)
・Android SDK
一応当日に開発環境のデータを用意しようとは考えていますが，オフラインで開発環境を構築する準備が出来るかどうかわからないので，揃えてもらっておくと助かります．
