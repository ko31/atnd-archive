株式会社スパイア
http://www.spireinc.jp/
が主催する、スパイアスマートフォンラウンジ（月1回開催予定）の第1回目です。

この「ヒットアプリの作り方!」では、
ヒットアプリを生み出したiPhone/Androidアプリの開発者・担当者の方々が登壇して、人気アプリを作るノウハウやプロモーションの秘訣を講演します。

パネルディスカッションでは、
今後注目されるアプリ内課金・月額課金について、
豪華パネリストが意見を交換いたします。

満席が予想されますので、お早めにご登録下さい！
