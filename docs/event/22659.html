こっちはにほんごよくわからない人に色々書きたい人むけです。

DBIx に一家言あるひとはこちら

俺の win32 トラックが無い！ acme トラックが無い！ あとあんなトラックが無い！とお怒りのお方は、適当に atend 立てといてください。勝手に立てて勝手に github にトラック追加しやがれ。

※YAPCや.PMで発表した事の有る人はハッカーの方に回ってくださいな。去年はカジュアルじゃなさすぎたので、今年はカジュアルにせめまっしょい。

今年も Perl Advent Calendar の季節がやってきました。
ぐぐれびりてぃが低いので JPerl って表記はやめました。
Perl に関連する tips を25本おとどけしちゃいます。
12月1日から毎日一本ずつ tips をアップしていっちゃうという企画です。
くわしくはこのあたりをみてください。

今年も Hacker Track と Casual Track と English Track の三本だてに挑戦しようとおもいます。

Hacker Track では自分の書いた CPAN モジュールや Perl Internal や system call に関わる話についての解説をかいてください。Casual Track では他の人が書いた CPAN モジュールや、他の人があんまり知らないようなゆるふわな Perl tips についての解説を書いてください。English Track では Perl の話を英語で書いて下さい。

我こそは tips 書いたるぞ!という人はどんどん参加表明してください。登録順に1日からまわしていきます。

去年のシステムを使い回しますね。
モジュールとからめるのが辛かったら、他の事でもいいです。Perl Hacks っぽいことならそれでも OK!
