2014年春学期 DEMO DAY のお知らせ

卒業スタートアップによるピッチイベントを開催します

1,000社以上の起業を支援し、卒業生の40%以上が資金調達に成功しているシリコンバレー発の起業家育成プログラム「ファウンダー・インスティテュート関西」

5月14日から始まった春学期もまもなく終了し、3ヶ月のプログラムを経た起業家が卒業を迎えます。

この度、卒業式を兼ねた DEMO DAY を下記の要領で開催させて頂くことになりましたので、お知らせさせて頂きます。

この DEMO DAY には、大学関係者・メディア関係者の方、社内ベンチャー・新規事業担当者、投資家の方、学生の方など、どなたでも無料で参加頂けます。

日時：８月７日（木）１８：３０〜
場所：グランフロント大阪 Global Venture Habitat
地図： http://www.sunbridge-gv.jp/about/access.html#osaka
参加費：無料

卒業スタートアップ＆プレゼン企業：

Up performa：スポーツ業界向けウェアラブル端末
CANVAS COFFEE：最高品質のコーヒー豆定期購入サービス
Crewbase：ルームシェア向け家計簿サービス
BestLogi：輸送業界の価格ドットコム

スケジュール：

１８：３０　受付

ぜひ皆様のご参加をお待ちしております。
