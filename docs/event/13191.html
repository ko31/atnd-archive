激動する世界を牽引するグローバルアントレプレナーを育成する機会の提供と、
輩出した人材をリンクさせ、世界を動かすイノベーションネットワークの形成を
目指し、KBC実行委員会では、今年度からKBC Value Creation Link を始動しま
した。

グローバルアントレプレナー育成の機会を提供するために、
世界を意識したシンポジウム、オープンセッション、ブース展示等の開催をします。
また定期的に参加者同士が集まることのできる交流会も企画運営し、
参加者同士の切れることのない強いつながりの形成することで、
10年後、参加者が講演者として本イベントへ戻ってくるようなイベント作りを目
指しています。

【当日コンテンツ】
★Panel Discussion
「Be a Global Entrepreneur」

登壇者:
國領二郎様(慶應義塾大学総合政策学部教授)
北城格太郎様(日本IBM最高顧問)
中村幸一郎様（Kauffman Fellows Program 16期生フェロー）

コンテンツ内容：
今、グローバル社会では様々なことが起こっています。
Facebookが台頭し国家であれば世界で3番目の人口を持つ国家です。
2013年には80か国以上の国の首相、大統領が変わります。　
世界のトップを牽引してきたお二人が、このような激動の世界において、
Initiativeを取るGlobal Entrepreneurになるためには何が必要かを語り合います。

「What I Wish I Knew When I Was THE 20’s」
登壇者:石倉洋子様(一橋大学大学院国際企業戦略研究科教授)
「What I wish I know when I was 20」この書籍をご存知でしょうか。著しい変
化を遂げる世の中では、スキルや知識ではなく、
振る舞いや心構えではないでしょうか。ダボス会議のフェローもご担当される方
によるお話は今の自分を見直す機会になるのではないでしょうか。

★Open Session
「Gy million Project」
登壇者：小林慎和様、その他4名ほど
※Gy Mission Projectとは：日本を変えるために、グローバルで通用し、
改革を推進していく若者を100万人生み出すために活動しているプロジェクト。

「Keio × Ritsumeikan Case Study of Incubation」
登壇者：立命館インキュベーター/慶應湘南藤沢インキュベーションビレッジ

「Woman’s power」
登壇者：石黒不二代様(ネットイヤーグループ株式会社代表取締役社長兼CEO)
近年、女性の活躍もテレビや雑誌で取り上げられるようになり、女性の社会進出
は当たり前のことになりました。
今回はシリコンバレーの女性起業家をお招きし、グローバルに通じるキャリア形
成ということでお話しいただきます。今後ロールモデルになりうる方な ので、
是非ともお越しください。

「外資系女性バンカーのキャリア【寄付講座：クレディ・スイス証券株式会社】」
登壇者：クレディ・スイス証券株式会社より女性バンカー数名
外資系で働く女性バンカーの生の声が聞けます。普段表に出ることが少ない外資
系金融会社だけでなく
さらにその中で戦う女性バンカー達が描く将来のキャリア像についてざっくばら
んにお話ししていただきます。
