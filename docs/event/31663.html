catch 「ループ」から始まる、WordPressとアナタの恋物語♥
description

WordPress習得のカギといえる「ループ」を、実際にコーディングしながら学んでいくワークショップ。第9回勉強会（2012年7月28日開催）で使った『本格ビジネスサイトを作りながら学ぶ WordPressの教科書』のサンプルファイルをもとに、プログラムを進めていきます。
（サンプルファイル使用をご快諾くださった中村けん牛さん〔プライム・ストラテジー株式会社 代表取締役〕にあらためて感謝）

対象

第9回勉強会に参加された方を対象としますが、飛び入り大歓迎！

定員

10人（開催時間内は出入り自由です　ご都合のよい時間帯にお越しください）

参加費

無料

持ちもの

  • 第9回勉強会で使用されたサーバ環境（ローカル環境・ご自身が契約されているレンタルサーバ・WordBench福井にて準備したサーバなど）
  • ノートPC（OSは問いません）

※PCには事前に以下のソフトをインストールしておくと便利です

  • アドオン『Firebug』設定済みのFirefox、またはGoogle Chrome
  • UTF-8が編集できるテキストエディタ
    WordPress 日本語 Codexの解説ページ
  • FTPソフト

会場のネット環境について

会場常設の無線LANを開放します。ただ、同時使用時の負荷が予想できないため、Wi-Fiルータなどモバイル環境をお持ちの方はご持参のうえ負荷分散にご協力をおねがいいたします。また、有線でのLAN環境をご希望の方は、下記の「お問い合せ先」までご相談ください。

駐車場について

会場には無料駐車場が併設されています（台数制限あり）

お問い合せ

wordbench.fukui [at] gmail.com への直メールでお問い合わせください。
