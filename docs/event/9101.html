description

来る10月27日（水）クラウド時代に身につけるべき
ＩＴスキルを学んでいただく機会を設けました。

来る前と来た後では貴方のＰＣに変化が

いわゆるノウハウ本はたくさん出ているけど
使いこなせていないと人も多いのでは？？？

今回の勉強会では、実際にみなさまが今お使いのＰＣや
i-phoneをお持ちいただき、ソフトやアプリをその場で
ダウンロードして、使い方を覚えたりテクニックを学んだり
していただきます。

この勉強会はカリスマ？ＩＴプロデューサースキルに長けたお客さんに
個人的に学ぼうと思ってたんですが、どうせなら
みんなで一緒に勉強してもオモシロイかも！っていう
発想から生まれた勉強会です。

クラウド時代の激動の波を軽々乗り切りたいという人には
ぜひオススメの勉強会です。

以下、案内です。ご確認ください。
