自分の大切な奥様の事で悩みってありますか？

例えば、

・何で妻はこんなことにこだわるのだろうか？

・何で妻はこんな言い方をするのだろうか？

・言わなくても分かってほしいのに分かってくれない

・え！？そこは誰だってそんなことしないでしょ！！

小さなことで、トラブルに成りかねますよね。

そして大きな問題へと発展したりと・・・

このセミナーでは、長年、人の個性について分類してきた、個性心理学という学問を使って、生まれ持った行動パターン、思考パターンを見ていきます。

個性心理学は、人の個性を動物のキャラクターで表現する、統計学かつ分類学でもあります。
まさに自分と奥様との違いが分かる、取扱い説明書です。

自分と相手との違いを知ればお互いを認め合えることを、お伝えしていきます。

自分が分かれば、自分らしく生きれるし、

自分がどういった適正能力を持っているのか？

自分はどういった人生観を持っているのか？
等も分かってきます。

◆イベント・セミナー内容

ご参加者専用の個人分析シートを見ながら、ご自身の個性（行動パターン、思考パターン）を確認し、講演を進めていきます。

人の個性は、動物のキャラクター12分類に分かれる前に、大きく3分類に分かれます。その３分類のうち、ご自身がどのタイプに当てはまるかを理解していただいた上で、話すので分かりやすいです。

私が用意している、身近な例え話をベースに、自分がそのときどういったことを考えるか？どういった行動をとるのか？

自分と個性が違う他の参加者様とは、全く考え方、とらえ方が違うのでかなり盛り上がります。

悲劇を喜劇に変えて、自分を納得させる。そして自分を認めれるようになったら相手を認めれるようになる。
　
　笑いあり、納得ありのセミナーとなっています。
期待できる効果

夫婦お互いの個性を知ることができ、
個性を知ることで、お互いを認め合うことができる。
認め合うからコミュニケーションが楽しくなる。

自分と相手は全く違うと理解できるからこそ、
区別ができるようになる
