description

『白猫プロジェクト』や『クイズRPG 魔法使いと黒猫の ウィズ』などの
人気タイトルや、『白猫テニス』や『ドラゴンプロジェクト』等、
最新作まで数多くのゲームの基礎を作り、その後の運用を支える
サーバサイドエンジニア・インフラエンジニアにフォーカスを当ててお話しします。

セミナー終了後には、登壇者との懇親会も実施しますので、
ざっくばらんな雰囲気の中で意見交換やご質問が可能です。
セミナーに参加してコロプラに興味を持たれた方、
採用選考を希望される方には、後日面接を実施します。

ゲーム業界での活躍を望む方、ゲームが大好きな方、ぜひご参加ください。

■講演内容
　・講演1：コロプラ流ゲーム開発
　　　コロプラのゲーム開発スタイル、サーバサイドエンジニアの業務に関する
　　　話の他、「エンジニアでも企画に携われるの？」などといった、
　　　コロプラのゲーム開発に関する素朴な疑問にもお答えします。

　・講演2：『白猫テニス』の負荷対策
　　　アプリ事前登録者数約60万人。
　　　アプリリリース前にサーバサイドで実施した負荷対策についてお話しします。

　・講演3：コロプラ流AWSクラウド活用術
　　　『白猫テニス』や『ドラゴンプロジェクト』など最新の人気ゲームを支える
　　　コロプラのインフラについて、その構成や運用、
　　　パフォーマンスチューニングなどをお話しします。
