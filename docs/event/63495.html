description

「各々テキトーな場所に集まって各々勝手に株でもなんでも勉強をしましょう。」っていう会。勝手に来て勝手に帰ってください。コミュニケーション一切不要。

この会を通して集まったっぽい人に声を掛けてどっかに遊びに行ったりするのも可。「誰かと喋りたくてなんとなく来てみた。」ってのも可。

イベント主催者も参加しますが何時に来るか、いつ帰るかも分かりません。

というか館内が広いのでまず会う事はないでしょう。誰がこの会を通して集まった人間なのかもほぼ100%分かりません。

どうぞご自由に。
