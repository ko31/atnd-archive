ハースストーン1周年記念：炉端の集い in福岡

大人気カードゲーム「HearthStone（ハースストーン）」の
オフラインイベントを福岡で開催いたします。

なお、今回はハースストーン日本リリース1周年を記念し、
大阪・愛知・新潟でも同時開催！各会場とのオンライン対戦も実施したいと思います。
今回は賞金や賞品も充実していますよ！

【概要】
日時：11月5日（土）13時～17時（予定）
※12時30分より受付いたします。

場所：サイバック博多駅前店
http://www.kaji-icafe.com/area/kyushu_okinawa/ekimae/

料金：お一人様1000円（税込）
飲物：フリードリンク（アルコールは除く）

定員：50名

参加：事前応募制
※参加応募が定員に達し次第、募集終了となります。予めご了承ください。

【内容】
対戦会を中心に行います。それ以外は参加された方のレベルに合わせて遊んでいただく予定です。
大阪・愛知・新潟でも同時開催しており、他地区とのオンライン対戦も行います。

【賞金・賞品】
成績上位の方に賞金もご用意しています！
さらに！成績上位3名は他地区とのトーナメント戦に挑戦して頂きます。
このトーナメントで優勝すると更に1人10,000円が進呈されますよ！
その他、ハースストーングッズ（ノートやお財布など）もご用意しています！
ジャンケン大会も実施しますよー！

「店内予選賞金]
優勝：10,000円　2位～4位まで賞金あり

[オンライン対戦予選賞金]
優勝メンバー全員に1人10,000円

・スタンダード、フリーデッキ、BO1で行います。

・再試合の際は同じデッキを使用して下さい。
・地区予選はトーナメントまたは総当たり戦を参加人数に合わせて実施します。
・地区予選上位3名が地区代表として、他地区とのオンライン地区対抗戦に出場します。
・地区対抗戦は3vs3の団体戦を行う予定です。
