石巻復興プロジェクトとは？

Web – http://ismk.seesaa.net/
Facebook – http://www.facebook.com/Re.ishinomaki

東北・石巻を応援してくれている、JR大森駅近くのウィロード山王商店街にて8月20日(土曜)、21日(日曜)の11時～18時まで石巻復興プロジェクトが出店いたします。

■ハッシュタグ

#ismkFair (石巻フェア)

■場所

  • 東京都大田区山王3-1-6

■出品内容

  • 石巻味噌やきそば(高砂長寿味噌さん開発の味噌ソースを使った石巻やきそば！)
  • 桃生茶(うるおい茶園）
  • お茶かき氷(うるおい茶園)
  • 希望の缶詰(木の屋石巻水産)
  • 長寿味噌、牡蠣味噌、醤油など(高砂長寿味噌本舗)

となっております。

また、20日は
ウィロード山王商店街様などが主催される盆踊り大会(19時～、石巻復興プロジェクトブース付近のスーパーカドヤ駐車場)も開催されますので、お時間のある方はぜひ楽しんで行って下さい！
