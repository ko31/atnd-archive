description

2月のマーケティング庵は、今注目を浴びている【facebook】をテーマにします。
世界で6億人のユーザーがいると言われているfacebookですが、最近日本でも急速にユーザーが増加しています。
今回は、「facebookって何？」「どんなことができるの？」「これからやってみたい！」という方のために初心者向けの解説から、twitterやmixiとの比較、facebookをビジネスに活かすノウハウといった実用編や、マーケティング的な視点から今後のfacebookの可能性まで、様々な観点からfacebookについてみなさんで紐解いてみたいと思います。
もちろんfacebookにまだ登録していない方もご参加いただけます。

※マーケティング庵は、身の回りに起きる様々な事象をテーマに生活者の視点から素人の感覚で、自由で気軽なフリートークスタイルで掘り下げるユニークなマーケティング研究会です。
ドーナツやオーガニック食品の試食などから激安酒場の研究に出かけて、実体験してみるなど、参加者で体験しながら理解を深める研究会でもあります。
目的は愉快に時間とテーマを共有することです。

※ご参加に関しては、マーケティング関係者や広告関係者、学生、マーケティングに興味のある方など、特に制限していません。
また、直接マーケティングに関わらない方やマーケティングは全く知らないという方、ただ単にテーマに興味があるだけの方でもご参加いただけます。

※見学だけのオブザーバー参加でもOKです。
