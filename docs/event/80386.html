あなたの夢を潜在意識が叶えます

努力しても結果が出ない、上手くいかない、変わらない ･ ･ ･
そう感じた事はないでしょうか。

それは、あなたの努力が足りないから？

いいえ、違います！！

実は、潜在意識の正しい使い方を知らなかっただけなんです。

一般的に意識して使える能力は５％程度だと言われています。

氷山に例えると、海面から出ているほんの一部なんです。　　　

もし、今まで使えていなかった能力をほんの少しでも使う事が出来るようになったら ･ ･ ･

今までの悩みが解決するはずです！！

でも特別な人だけなのでは？

大丈夫です！！

その使い方を学校教育で教えてもらえなかっただけなんです。

誰でも潜在意識の仕組みと使い方を知れば、自分の夢が叶えられます。

是非、そんな体験をして下さい。

私が提供する夢を叶えるメソッドは、潜在意識の仕組みに従って心身をリラックスさせます。

これにより、普段使えていない潜在能力まで発揮出来るようなります。

では、リラックスする事によってどんな効果があるのでしょうか？

リラックスする事によって脳波はα波の状態になります。

α波の状態の時には潜在意識へ直接的に働きかける事が出来ますので、集中力・イメージ力・発想力・直観力・記憶力などが高まり、結果的に夢を叶えられるようになります。

一方、緊張や不安、イライラなどのストレスを感じている時はどうでしょう？

主に脳波はβ波の状態となり、普段の実力も発揮できず、望む結果を出すのは難しいのです。

さらに、このβ波の状態が長く続くと、自律神経のバランスが崩れ、心身に影響を及ぼし病気の原因にもなります。

メソッドによりリラックスする事で、ストレスを解消し、健康維持・健康回復も望めます。
 

【シルバ・メソッド】
アメリカ発祥の潜在能力活用プログラム

西洋の大脳生理学、医学、心理学などの科学的理論を取りいれ、東洋古来のヨーガや瞑想、種々の精神統一法を総合的に組み合わせ、 数多くの実験を積み重ねて体系化したプログラムです。

いわゆる右脳開発やα波による能力開発ブームの基になっています。

既にアメリカでは、GEやフィリップスなどのような大手企業が社員研修として取り入れ、学校では27の大学で正規の学習講座として採用されています。

また、医療機関でも心理療法としてガン治療をはじめとする、様々な分野で活用されている実績があり、いずれも高い評価を得ています。

現在、70年間の潜在能力活用の研究を基に、30の言語に翻訳され世界111カ国で常設セミナーを開催し、1,400万人以上が受講されています。
