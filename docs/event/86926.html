マルチプレイゲームを簡単に作りましょう！

実作業30分でマルチプレイゲーム開発を体験できます！

マルチプレイのゲームを簡単に実現する、ネットワークエンジンの『Photon』。
そのPhotonのハンズオンが大阪で開催です！

今回は、Unityとの相性抜群のPhoton Unity Networkingを実際にUnityで使い始める部分の導入編です。
プロジェクトを作成し、Assetをインストールして、接続設定やキャラクタ設定、実際のキャラクタ同期までを体験できます。
Photonを触ったことがない方でも十分対応できる内容となっておりますので、安心してご参加ください。

Photonの手軽さ、そしてマルチプレイゲームを作る楽しさをぜひ体感してください！

今回は、PlayCanvasの初級編ハンズオンも同時開催しております。こちらもぜひご参加ください。
