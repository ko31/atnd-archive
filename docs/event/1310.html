description

ITが社会基盤化したことにより，利便性と低コスト化はトレードオフの関係ではなく，ITによる社会的費用の削減が可能な状況となってきたことを踏まえ，ITがもたらす安全安心，医療・教育・市民生活等に焦点を当て，豊かな暮らしを実現するITを考察するセミナーを開催します。
参加費は無料で，どなたでもご参加いただけます。

【基調講演】ITを基盤とした生活の安全安心
【講演　１】医療分野におけるIT活用
【講演　２】教育分野におけるIT活用
【講演　３】自治体のIT利活用

タイトルや案内に生活者向けの香り付けがされていますが，「ビジネスイノベーションセミナー」ですので，ビジネスチャンスを見いだす場のご提供を基本として講演を組み立てています。

基調講演では，ITを活用する上でどの分野であろうと必須である『安心』をどう担保すべきかの考察を行います。

講演１では，情報保護に細心の注意が必要とされる医療の分野でも，既にオープンソースソフトとインターネットが活用されている国内外の状況と，将来像をご紹介します。

講演２では，私学からの教育の情報化紹介ですが，講師は民間企業や教育委員会でも活躍された方で，これから国の重点政策として進められる教育分野のIT化について，様々な見識をお持ちです。

ご多忙中とは存じますが，ぜひご参加ください。
