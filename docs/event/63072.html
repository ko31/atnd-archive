　　『 ウェブ売上・問合せ急増セミナー
　　　for 経営者＆ウェブ担当者 （初級編）』

=============

<< この様な方へオススメ >>

・ ホームページを運用しているが、なかなか成果が上がらない。
・ Google Analytis を設置しているが、見方がちゃんとわからない。
・ サイトの改善が必要だが、どこから手を付けて良いのかわからない。
（ ECサイトは、今回のセミナー内容対象外としております。 ）

<< セミナー内容 >>　※セミナー内容は、多少変わることが御座います。

1. ホームページを成功させるたった一つの方法

2. Google Analytics を活用し、改善のヒントを探す
　　⇒ 正しい初期設定方法 / 特に見るべき値の紹介

3. 最も簡単なコンバージョンアップ方法の紹介
　　⇒ フォーム改善のススメ / フォーム診断シートの配布

4. サイト全体の改善として、ABテストのご提案
　　⇒ 定期的な実施方法 / ABテストツールの紹介

5. その他、トレンドのWEB集客手法のご紹介

=============
