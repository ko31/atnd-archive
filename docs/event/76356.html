Cygames UI Night　
　　　　〜Cygames流制作テクニック&秘話！ 〜

概要

ユーザーの満足度が高いゲームを開発するうえで必要不可欠なのが、洗練されたUIデザイン。
本イベントでは、リリースから2年経った現在も常に進化を続ける『グランブルーファンタジー』や最新タイトルの『Shadowverse』等のタイトルを題材に、CygamesにおけるUIデザインのこだわりや、制作工程、裏話などをご紹介します。

また、イベント終了後は軽食をご用意し、チームメンバーも参加する懇親会も実施致します。
登壇者・チームメンバーやご参加の方々との情報交換の場としてぜひご活用ください。
また、希望者を対象にUIデザインチームのメンバーが作品を拝見いたしますので、是非ポートフォリオをお持ちいたければ幸いです。
※作品集としてまとめたものでなくても問題ございません。お気軽にご参加ください。

タイムスケジュール

・21:30　　終了予定
