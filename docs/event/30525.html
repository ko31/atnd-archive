description

今回は実践モデリングの第３回目です。
参加者が実際の業務で抱える問題をモデリングします。
業務で抱える問題の本質部分を残し、その他の部分は参加者の経験や一般的な知識をもとにモデリングを行います。
モデリングを通したコミュニケーションにより対象を深く知ることを体験します。

今回の対象は設備保全業務です。

以前開発した設備保全システムの話を聞きながら、その対象ドメインの形を明らかにします。
設備保線業務で扱う情報構造とはどのようなもので、システム化のポイントはどこにあり、そのためのドメイン構造をディスカッションしながら明らかにします。

　・どのようなコンテキストなのか？

　・代表的なユースケースは？

　・上記ユースケースを意識したドメイン構造とは？

　・などなど　様々な視点をドメインモデルと絡めて議論しながら進めたいと思います。


ドメインモデリングの能力を向上し仕事に活かそう！！　をテーマに今後も進めたいと思います。

興味のある方はご参加ください。
