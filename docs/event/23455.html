Advent Calendar とは

クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーです。転じて、技術系コミュニティで、クリスマスまでの25日間にTipsを1つずつ公開するイベントをAdvent Calendarと呼ぶそうです。

ルール

PHP Advent Calendar 2011（全部俺）を真似して、12月1日から12月25日の間、Redmine に関する話題を参加者=changeworld一人によって毎日一つずつ書いていくという企画です。

テーマ

Redmine に関することなら、何でも構いません。
「こんな使い方してますよ」とか「こんな Tips 知ってましたか？」とか「愛用しているPlugin」、「こんなもの作りました」、「こんなことに使っています」など、 Redmine に関するTipsならどんなことでも構いません。
