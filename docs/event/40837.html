SpriteSheet・AIR for Desktop・CreateJS・Scoutについて語り合おう！

通常のFlashコンテンツだけでなく、GPUを利用するStage3DコンテンツやHTML5コンテンツにもFlashが利用できる新しい時代に突入しました。
そうした新しい時代に求められている「SpriteSheet」「AIR for Desktop」「CreateJS」「Scout」などについて語らいませんか。

　(1) テーマに関する何かを作ってきて披露する。
　　* テーマに沿ったサンプルを作ってきたり
　(2) テーマについて質問する。
　　* テーマに関する不明点とか
　　* これまで経験したテーマの疑問点とか

参加者の方々へのお願い

ひよこの会 について

ひよこの会は「いまさら他人に聞けないFlashの基礎的な何かを勉強する会」です。
プロアマ、学生、ベテラン、年齢問わず、Flashを現在進行形で勉強してる人であればどなたでも参加可能。
Flashを現在進行形で勉強してるFlash好きなら誰もがもってる今さら他人に聞けないFlashの話で盛り上がる会です。
