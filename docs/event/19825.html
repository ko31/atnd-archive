description

Pythonやプログラミングの基礎を学びたいひと向けの読書会を行います。

概要

  • Pythonを使ってコンピュータサイエンスの基礎が学べます
    • 基礎的なところから始めるので、興味があればプログラミングの経験がない方でもOKです
    • 広い内容を扱うので、将来的にPython以外の言語を使う上でも役に立ちます
    • もちろん、Python初心者がPythonを習得するためにもうってつけの内容です
  • この勉強会はPython札幌の活動の一環として行って居ります
    • 興味があればGoogleグループ(メーリングリスト)にもご参加ください

教科書

  • 『初めてのコンピュータサイエンス (O’reilly)』を使用します
    • 各自持参が基本ですが、学生などで購入を躊躇される方は購入を急がなくても大丈夫です
    • 教科書の内容に納得した上で、懐に余裕ができた時点で購入してください
    • 今回は第2章 Python入門 (p.7〜26)から始めます

持ち物

  • 任意ですが、会場に電源があるので、PCを持参して使用できます
  • 無線LAN環境はEMOBILE回線を持ち込んで公開しますが、高速ではありません
  • Practical Programming: installation instructionsを参考に環境構築すると良いでしょう
    • 当日までにPython環境が用意できない場合は遠慮なく相談してください

参加費

  • 利用料金+プロジェクタ使用料を頭割り
    • 530会議室(夜間) 2800円
    • プロジェクター使用料 1200円
    • コンセント使用料 200円

次回以降について

  • 現在のところ、火曜または木曜に月2回程度開催予定です

連絡
