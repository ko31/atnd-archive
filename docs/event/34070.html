Vimperatorユーザーのみんな! オラに力を分けてくれ!

12月1日から25日まで毎日1つずつテーマに沿う記事を書いていくAdvent Calendar。
Vimperator が無いのは寂しいので作りました。

参加の流れ

  1. 参加登録する
  2. こちらで担当日を指定するので、いつかを確認しておく
  3. 担当日にVimperatorに関する事を何か書いて何処か適当な場所(blogなど)に投稿する
  4. 投稿した記事のURLをここのコメント欄に書く

ね、簡単でしょう?

スケジュール

暫定的に、2週目以降の担当予定者名を記入していますが、まだまだ 参加者募集中 です。
2周目以降においては、新規参加者が加入した場合、主催のAmaiSaetaが気づき次第、 次の日 に新規参加者を追加、以降の予定をずらします。これは、既存参加者と新規参加者の投稿が被らないようにする為の措置です
勿論、FAQにある通り、2周目以降の投稿を行うかは各自に任せます。

FAQ

  • Advent Calendarって何?
    元々は、12月1日からクリスマス(25日)までの日めくりカレンダーの事です。
    IT業界分野では、それに倣い、参加者が1人1つずつテーマに沿った文章を書いていくイベントを指します。
  • どんな内容を書けばいいの?
    Vimperatorに関する事ならなんでも!
    Vimperatorプラグインの紹介、Vimperatorと他の類似プラグインとの比較、tips、自分の.vimperatorrcの紹介、Vimperatorへの(勝手な)要望、なんでもどうぞ。
    Thunderbird版Vimperatorの Muttator 、Vimperatorのフォーク Pentadactyl の話題でも。
  • どこに書けばいいの?
    web上で閲覧出来る場所であればどこでも!
    blogを持っているならばそこでも。皆が閲覧出来るのであれば、FacebookやGoogle+などSNS上でも可能です。
  • 参加したいけれど、25日まで埋まってしまっているんだけど……
    26日以降も、なんなら新年超えてしまってもいいのよ?
  • 25人も集まらなかったら?
    2周目に突入する場合も多いようです。年の瀬ですし、2周目を行うかどうかは各々に任せます。
    (前もって2周目を行うかどうか表明して頂ければ、他の人が埋めてくれるかもしれません)
