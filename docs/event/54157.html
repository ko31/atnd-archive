description

アドテクとはアドテクノロジーの略称で、インターネット広告に関するIT技術を意味します。
ネット経済がさらに成長していく中でアドテクの存在感は非常に大きくなっており、広告代理店やネット企業のみならず、SAPやOracleの様なメガベンダーや大手SIerの参入が相次いでいます。
普段、業務系システム開発などをやっているとなかなか馴染みのないアドテクについて、エンジニア目線でその裏の仕組みについて解説させて頂きます。これからネット企業で活躍したいエンジニアの方々、是非気軽にご参加ください！！

※エンジニア向け講座の為、基礎的な技術がわかる方向けです。ご了承ください。
※業界のプロの方の鋭い突っ込は禁止です。懇親会のタダ酒を目的にお越しください。
