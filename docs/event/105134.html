働き方改革推進コンソーシアム ワークショップ 2019 大阪 Vol.1

～会員企業の働き方改革の取組と働き方改革法案への対策 ～

詳細なアジェンダは<a href"http://workstyleinnovation.org/2019/03/31/0423/">こちらを参照ください。
申込締切：4月19日(金)18:00

働き方改革推進コンソーシアム ワークショップ 2019 大阪 Vol.1を開催いたします。

今回のワークショップ 2019 大阪 Vol.1 では、既に働き方改革に取組んでいる会員企業からその状況を公開していただきます。加えて、特別講演としてワークライフバランス認定コンサルタントより、今年4月から施行される働き方改革関連法についての概要と対策について講演いただきます。
自社の働き方改革を実現するためのヒントを見出していただければ幸いです。

本コンソーシアムは、テレワークを含め働き方改革を実現しようとしている企業や導入支援を行う企業が一同に集まり、会員企業間で情報共有をする事により[働き方改革]をいち早く実現するための”場”を提供しております。

日時：2019年4月23日(火) 15:00-18:00 (14:30 受付開始)
会場：日本ユーレットパッカード(株) 西日本支社
住所：大阪府大阪市淀川区宮原3-5-36 新大阪トラストタワー20階
地図：<a href"https://www.hpe.com/jp/ja/japan/about/japan-office-locations.html">詳細

主催：働き方改革推進コンソーシアム
後援：日本ヒューレット・パッカード(株)、(株)テレワークマネジメント、テレワークICT協議会、ビジネスアライアンスコンソーシアム
参加費：無料 (事前登録制)
定員：40名 (申込順で受付：定員になり次第受付終了)
対象：働き方改革を実現しようとしている企業・団体の経営層や責任者ならびに導入支援企業・団体など
(非会員の方も参加可能)

情報交換会：18:30-20:30
会場：会場付近を予定
会費：4,000円/人 (当日現金にて支払：会場の領収書を発行)
注意：開催3営業日前以降にキャンセルした場合は、後日、会費を徴収いたします。

問合せ：office@workstyleinnovation.org
