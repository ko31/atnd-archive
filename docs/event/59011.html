WordBenchは WordPress ユーザーのための地域ソーシャルコミュニティです。
京都を中心としたコミュニティとして2014年11月度を開催します。

WordBench京都はWordPressの好きな人が集まってみんなで作るコミュニティです。
ガッツリしたセミナー形式から情報交換など様々ですがアットホームなコミュニティですので気軽にご参加ください。

WordBench京都

開催日時

2014/11/30(日) 13:50～17:00
　※13:20から会場に入れます。

対象者

WordPressに興味がある方なら誰でもOK！
これからWordPressを始めようと思っている全くの初心者も大歓迎です。
京都住民でなくてもOKです。
WordBenchに加入していない方でも大丈夫です。
2014年11月度ですが毎回独立した勉強会です。初参加OKです。

イベント内容

今回のコンセプトは、WordPressのテーマの質を高めよう！
あるテーマを準備します。そのテーマをみんなでチェックし改善していきます。
目標は公式ディレクトリに登録できるレベル。
このレベルになれば、
　・WordPressの管理画面の設定がテーマに反映されない！
　・古いテンプレートタグを使って動かなくなった！
　・テーマのセキュリティ対策って？
　・テーマのテストツールって？
などが身につくかもしれません。
チームごとに分かれれて作業する方針ですが、初心者でも何か作業できるようにと考えています。

※企画側も初心者ですので、作業しながらみんなで情報共有しましょう！　
　　

どのようにチェックするのかは以下のサイトを参考にします。
　
■THEME REVIEW HANDBOOK >> Theme Review Team
https://make.wordpress.org/themes/handbook/guidelines/

公式ディレクトリにテーマを登録できるかどうかチェックをする、テーマレビューチームのガイドラインです。

■Codex >> Theme Development
http://codex.wordpress.org/Theme_Development

WordPressの公式マニュアル Codex のテーマ開発の説明です。
