第1部　リチウムイオン二次電池用電解液の材料構成と効果を引き出す添加剤の使い方、安定性の評価手法
【13:30-15:00】

講師:群馬大学大学院　工学研究科　応用化学・生物化学専攻　教授 鳶島真一 氏

【キーワード】
1.リチウムイオン電池
2.電解液
3.安全性

【講演主旨】
リチウムイオン電池用電解液と添加剤は実用電池の電流取得性能、充放電性能、容量劣化、保存特性、安全性等に影響する重要な因子である。本講演では基礎、応用、各種特性評価方法、分析手法について概説する。

【プログラム】
1.リチウムイオン電池の今後の展開と電解液 　

2.リチウムイオン電池用電解液に要求される基本特性

3.リチウムイオン電池電解液の種類と特徴　
　3.1　各種電解液システムのイオン伝導機構と安定性
　　3.1.1 有機溶媒電解液
　　3.1.2 ポリマー電解質
　　3.1.3 無機固体電解質
　　3.1.3 ゲル電解質
　　3.1.4 イオン液体

4.電解液各種添加剤の種類と機能

5.効果を引き出す電解液添加剤の上手な使い方
　5.1　導電率向上
　5.2　負極の安定性向上
　5.3　正極の安定性向上
　5.4　難燃性向上
　5.5 高電圧対応
　5.6 過充電防止剤

6.安定性の評価手法
　6.1　交流インピーダンスによる電極電解液界面状態の抵抗解析手法
　6.2　赤外吸収スペトルによる電極表面膜の化学分析手法
　6.3　マススペクトルによる電極表面組成およびガス分析手法
　6.4　熱量測定による熱安定性評価手法

7.まとめ

【質疑応答　名刺交換】

第2部　リチウムイオン二次電池における各種部材の分析評価法
　　～電解液への添加剤使用による分析評価事例を交えて～
【15:15-16:30】

講師:(株)東レリサーチセンター　ご担当者

【キーワード】
1.リチウムイオン電池
2.劣化解析
3.機器分析

【講演主旨】
リチウムイオン電池の分析評価法は、部材・着目点ごとに非常に多くの手法が存在し、新規材料開発・耐久劣化時の現象の解釈など、様々な場面で分析評価が必要となります。
本講演では、弊社が行っているリチウムイオン電池の評価での様々な分析手法の紹介と、添加剤を使用した電解液での劣化評価として、様々な分析法を用いた総合解析事例について紹介します。

【プログラム】
1.はじめに
　1.1　リチウムイオン電池の構成と解体・解析への適用
　1.2　電池評価としての機器分析
　1.3　劣化解析の場合の着目点と分析方法一覧

2.電極活物質の分析評価法
　2.1　電極活物質粒子に着目した分析
　　2.1.1　形状比較
　　2.1.2　活物質の構造解析
　　2.1.3　Liの分布評価
　　2.1.4　Li状態分析
　　2.1.5　元素分布比較
　2.2　電極活物質粒子の表面に着目した分析
　　2.2.1　形状比較
　　2.2.2　活物質表面の元素分布
　　2.2.3　活物質表面の電子状態
　　2.2.4　深さ方向分析

3.電解液の分析評価法
　3.1　電解液構成成分の組成分析
　　3.1.1　溶剤・添加剤成分の組成分析
　　3.1.2　電解質の組成分析
　3.2　電解液変性物の組成分析
　　3.2.1　溶剤変性物の分析
　　3.2.2　電解質変性物の分析
　3.3　電解液不純物分析
　　3.3.1　フッ酸定量
　　3.3.2　水分定量
　　3.3.3　溶出金属定量

4.電解液への添加剤使用による評価事例
　　(電極表面構造の評価、電解液分解物の有機組成変化)
　4.1　添加剤に起因する電極表面構造変化
　4.2　添加剤に起因する電解液分解物の有機組成変化
