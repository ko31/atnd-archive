title のおがたOffCafe
catch 東日本大震災は私たちに何をもたらしたのか ～いま自分の中にあるものに向き合う～
description

2011年3月11日14：46　M9.0の地震、そして津波。そして、追い討ちをかけた原発事故。

多くの尊い命が失われたことに深い哀悼の意をささげるとともに、多くの被災された方々に心からお見舞いを申し上げます。

刻々と伝えられる被害の大きさに、ただ呆然とし、漠然とした不安にさいなまれながら、なすすべもなく情報にさらされつづけました。
今回の震災では、テレビやラジオのマスメディアよりも、インターネットを通じて情報が広がり、様々な情報は、その膨大な量とスピードにより、まさに”情報のつなみ”のごとく押し寄せてきました。
そして、被害の状況と共に、たくさんの支援活動の様子が伝わり、この遠く離れた福岡の地でも、すぐに行動となって現れました。
自分にできることはないのか、という小さな気持ちが集まり、「何をすべきなのか」「何ができるのか」
という大きな動きにつながってきました。
その一方で、自然の猛威に圧倒され、余りの被害の大きさに対してどうしてよいのかわからない。そんな気持ちの整理もつかないままに、支援の波に押されて走り始めた方もいるのではないでしょうか。

復興に向けた取組みは、短距離走ではありません。かといって、長距離走でもありません。
本当に長い道のりであり、いつまでも一人で走り続けらられるものではないでしょう。

今回のOff Cafeでは、「何をすべきか」ということは話しません。
長い復興に向き合うためにも、大震災が自分の中に何をもたらしたのか、自分の気持ちにどんな変化があったのか。
いまいちど立ち止まり、「いまの自分の中にあるもの」に向き合ってみたいと思います。
