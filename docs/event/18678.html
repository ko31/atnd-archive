「せまいドレスコードでムダにあつまりたい！」が、とりあえずのgommの遠足のテーマです。
花火大会も少ない今年。浴衣でお出かけする機会もなかなかないので、みなさんで遠足しませんか？
第一回と同じく、まったりと盛り上がる遠足にできたらと思います。

[ざっくりしおり]

  1. 遠足のテーマ：浴衣でムダに集まりたい。
  2. 遠足の決まりごと：男女問わず、浴衣を着用してくること。
  3. 遠足のおこづかい：花火代として、300円いただきます。晩ごはんに、4000円かかります。
  4. ＜時程＞　
    16時20分くらい　江の電 鎌倉駅 集合。
    16時30分くらい　江の電に乗って、稲村ヶ崎に向かいます。
    17時くらい　「稲村ヶ崎食堂」 (website)
    にて、軽くごはん&お酒。予算は¥4000です。
    19時くらい お店からすぐの浜辺で、手持ち花火をします。
    20時くらい 稲村ヶ崎駅にて、解散。

※ 集合前に鎌倉あたりをうろうろ、おすすめします。
※※　写真はイメージです。
