★講演内容

フロントエンドフレームワーク/ライブラリ戦国時代と言われるほどに、
次々と新しいフレームワーク/ライブラリが生まれたり、
破壊的変更を含んだメジャーバージョンアップが行われるなど、
めまぐるしく変化するフロントエンドフレームワーク/ライブラリの世界。
Angular 2、Vue.js 2、React.jsなど、
まだ比較的新しいフレームワーク/ライブラリをサービス導入してみて、
実際にどうだったのか？
導入準備から運用までのポイント、苦労したことや、導入して何が変わったか、
3社のフロントエンドエンジニアが、
ここだけの話を交えながら、活きた事例をお話させて頂きます。

※ 内容は変更になる可能性があります。
