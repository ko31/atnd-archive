新しいゲームが完成したのでパーティします！

オインクゲームズのカードゲーム新製品「さるやま」が間もなく発売になります。すぐ覚えられるかんたんなルールで、何度もプレイしたくなるこのゲームの発売を記念して、ささやかなパーティ＆ゲーム会を開催しちゃおうかなーっと思っちゃいました！
　
　

▼でもゲームをプレイしないでまったり過ごしてもイイよ！

会場はとても雰囲気のよい優雅なイベントスペースです。ゲームしてる人たちを眺めながら飲んだりつまんだりしましょう！ゲームをプレイしつつ、飲食しつつ、歓談しつつで、まったり過ごしましょう！ゆるっといきましょう！
　
　

▼来たい時間に来て、帰りたい時間に帰る感じで

18:30から開場して、23:00で閉場します。ゲーム会でもあるので長めに時間をとってますが、来たい時間に来て、帰りたい時間に帰る感じでOKです！食べ物はだいたい20時前後に出す予定です。
　
　

▼もちろんオインクゲームズ作品は全部プレイ可能！

会場では新作の「さるやま」はもちろん、オインクゲームズの全作品がプレイできます！気になっていたあのゲーム、ぜひ遊んでみてください！
　
　

▼世界のボードゲーム・カードゲームもあるよ！

オインクゲームズが独断と偏見で選んだ世界のオススメボードゲーム・カードゲームもセレクトして持ち込みますんで、このさい遊んじゃいましょう。持ち込みも歓迎です！
　
　

▼オインクゲームズ製品のイラストレーター、ゲームデザイナーも来ますよ

「さるやま」イラストレーターの岡本かな子さん、「StrayThieves」「Stamps」のイラストレーター菊池洋平さん、「卑怯なコウモリ」の作者の新澤大樹さんもいらっしゃいます！サインをもらうチャンス！
　
　

▼参加費とか

参加費はフード＋ドリンクチケット２枚つきで3500円。
　
　

▼そのほか

ちょっとしたトークイベントなんかも企画中ですが、やるかもしれないしいろいろ間に合わないかもしれません！でも大丈夫、ぜひご参加ください！
　
　

親睦！
