description

医学の目覚ましい進歩によって、これまでたくさんの生命を奪ってきた「感染症」の脅威は大きく後退しました。
しかし、近年、薬に順応した病原体（薬剤耐性菌）の種類や数が世界的に増えてきています。
しかも、こうした病原体に対応する新しい薬もどんどん生まれにくくなっていると言われます。
「薬が効かない感染症の時代」が来ないように、私たちがいまなにを考えて、どんなことに取り組んでいかなければならないのか、専門家を一緒に考えてみましょう。
