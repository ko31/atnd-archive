｢Perl入学式 #1 補講｣について

｢Perl入学式 #1 補講｣は, 1月22日に開催した｢Perl入学式 #1｣に参加できなかった方向けに開催します.
用事があったり, 補欠になっていたりという理由で｢Perl入学式 #1｣に参加できなかった方は是非ご参加下さい.

また, 次回｢Perl入学式 #2｣のATNDはこちらとなります. こちらも参加して頂ければと思います.

PC貸出制度

｢自宅にデスクトップPCはあるが, ノートPCは持っていない｣という場合, 当日会場限定でノートPCをお貸しすることができます. 何台お貸しできるかは毎回変わります(1台もお貸しできない場合もあります)ので, 参加登録をする前に, 貸出可能かどうかを papix2011@gmail.com に確認して下さい.

お願い

#1ではPerlの実行環境を構築する為に, Windows系OSをお使いの方は, “VMware Player”というソフトウェアを用いて, “Ubuntu”というOSを導入します.
これらのダウンロードには時間がかかりますので, もしよろしければ事前にダウンロードしておいていただけますようお願いいたします.

”VMware Player 4.0.1"ダウンロード先:
http://downloads.vmware.com/jp/d/info/desktop_end_user_computing/vmware_player/4_0

”Ubuntu 11.04”ダウンロード先:
http://www.ubuntulinux.jp/News/ubuntu1104-desktop-ja-remix
※”CDイメージ”というリンクをクリックしてダウンロードを開始してください.

既にLinux系OSを導入済みの方, Macをお使いの方(OSがMac OS Xの方)は, ダウンロード･導入をする必要はありません.
そういう意味では, 既にLinux系OSやMac OS Xをお使いの方は, 参加するメリットがあまりない, と言えるかもしれません.

また, ｢どれをダウンロードすればいいかわからない｣, ｢どうしたらいいかわからない｣という方は, ダウンロードして得られるファイルを入れたUSBメモリをこちらで用意します.
