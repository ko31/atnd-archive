description

製造業での研究開発，生産，品質管理，出荷後顧客対応などの各工程においては，様々な問題が発生し，これらの問題を解決するために，実験・解析を行うことになります。本講では，どのような心構えで実験に取り組めばよいか，どのような考え方で実験結果を解析・解釈すればよいか，初心者がなかなか気づかないポイントについて解説します。

１．問題解決プロセス
　　1-1　測定結果の信頼性に関する概念
　　1-2　ばらつき～繰返し性と再現性
　　1-3　真値，測定結果，誤差
　　1-4　ばらつきとかたより
　　1-5　基本的な統計解析
２．標本数・測定回数と信頼性
　　2-1　データの分布とはなにか?
　　2-2　生データの分布と「平均」の分布
　　2-3　平均の信頼性とばらつきの信頼性
　　2-4　何回測定すればよいか?
　　2-5　測定回数が増えると無意味な有意差が出る
　　2-6　測定機器を校正してもばらつきは減らない。
３．実験の3原則
　　3-1　実験の反復
　　3-2　実験の局所管理
　　3-3　実験の無作為化（ランダム化）
４．実験前の心得
　　4-1　どんな数字に着目するか?
　　4-2　予備実験の重要性
５．実験中の心得
　　5-1　実験担当者としての心構え
　　5-2　ばらつくデータは悪か?
　　5-3　実験中または実験直後の判断
６．解析の心得
　　6-1　グラフによる全体像把握
　　6-2　表示桁数を決めるポイント
　　6-3　統計的有意差があっても技術的には有意差はないかも
　　6-4　相関と因果
　　6-5　昨日の標準偏差と今日の標準偏差を平均してよいか?
