description

この撮影会では、
できるだけリラックスした状態で撮影に入る為にも、
毎回撮影会前に参加者みなさんとお茶会やランチ会を開き、そこで写真の話や雑談しながら和気あいあいと楽しい時間を一緒に過ごします。

その中でカメラマンの僕自身のことを知っていただいたり、参加者さん同士お互いのことを分かり合い打ち解けあったところで撮影会に入っていきます。

撮影会自体も話をしながら場所も変え参加者皆さんで楽しみながら進めていきます。

最初はやはり緊張しますが、だんだん慣れてきると
「楽しい！！」
と皆さんおっしゃるようになり特に女性の方は目がキラキラしてきます ^^

その後、写真の選定も他の参加者の皆さんにも見てもらいながら選んでいったりします。

他の方に見てもらうことで、
新しい自分の魅力や自分では気づいていない「自分らしさ」を発見できたりします。
この時間も僕の撮影会ではとても大事な時間です。
もちろん、お時間の都合でランチ会等出席できなく途中参加でも大歓迎です！！
いきなし撮影には入らず、私や他の参加者さんとコミュニケーション取った上で撮影に入ります。

以上のようなことから、僕の撮影会は少人数にこだわっています。
流れ作業的な撮影や参加者お一人お一人に目の行き届かない撮影会はしたくはないからです。
