開催概要

沖縄在住のITエンジニアのみなさま、この夏の台風17号が接近するいまをいかがお過ごしでしょう。

よろしければ、今回の台風もコードの肥やしにしてみてはいかがでしょうか？

台風ボッチソンは、

  • IT 系の読書
  • お仕事
  • 暇つぶしに LL なスクリプト
  • API の勉強
  • ウェブサービスの開発
  • Android/iOS な開発や実験
  • 下の岩盤を掘る作業

などを行い、台風が過ぎ去るまでひたすら耐え忍ぶ台風限定ボッチソンです。
