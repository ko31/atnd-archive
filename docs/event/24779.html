都合により14日に延期となりました。

開催趣旨

全国いろいろなところで行われている、アジャイルサムライ読書会ですが
広島でも誰かやらないかな～と思っていたのですが、誰もやらないのでとりあえず、自分でやってみることにしました。

最近注目されているアジャイル開発ですが、アジャイルソフトウェア開発宣言が2001年に発表されて10年経ちました。

直感的に良いと判っていても、実際に導入するとなるといろいろと問題が起き、効果があるのかと思われる方はもちろん
読んだ事ある方、まだ読んでない方、どなたでも参加可能な会にしたいと思うので、興味ある方は参加してみてください。

今回（第0回）は、どれくらい参加者がいるかの調査も含めての会ですので、実際に読書するのは第１回からとします。
第一回の開始日は準備会で決めます。

場所がカフェなのは、集まるかどうかも不明なので、とりあえず場所代が掛からないこと、場所がいいことを理由に選択しました。
ただし、最初１ドリンクは注文してください、おしゃれなカフェで、ホットコーヒー600円です。

今後の予定としては、２週ごとの曜日19：00から開催（この辺も決めたいと思います）
第一章を読み終えたあとで、各自の感想や意見、疑問を出し合い、共有していければ良いなと思います。
対話のなかで何かが生まれることを期待して・・・
