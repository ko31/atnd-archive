catch インストール編
description

徐々に注目を集めつつあるドキュメンテーションツール Sphinx のインストールや基礎についてハンズオン形式の勉強会を開催します。

OSCの併設イベントとして開催されるSphinxCon JP内で2つのハンズオンを実施しますが、これはインストール編です。
Sphinxをインストールし make html でhtmlを生成できる事を目指します。

既に、 make html を実行しhtmlを生成できているのであれば、同じ部屋で開催されているハッカソンや クイックスタート編 に参加しましょう。

対象
Sphinxハンズオン(クイックスタート編)に参加される方向けに、事前準備を会場で実施して頂きます。

勉強会の進め方/範囲
Sphinx-users.jp の Sphinx をはじめよう というページの内容に沿ってSphinxをインストールしhtmlの生成までを実施して頂きます。

参加条件

  • Sphinxをインストールする為のノートPC/Macを準備して下さい。

注意事項

  • ネットワーク(無線LAN)は会場で提供します。
  • 十分な数のコンセントが用意できない可能性があります。各自、電源タップを持ってきていただけると助かります。
  • 入退場は自由です。
