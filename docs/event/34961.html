2013年2月10日（日）サバイバルゲームインドア戦を行います。（基本雨天決行）

どうも、主催こしがやです(‘ω’)

またもや3連休のど真ん中２月1０日にインドアサバゲを行いたいと思いますので、募集させていただきます。
基本初心者メインのぬる～いゲーム会ですのでその様な感じで宜しくお願いいたします。
一応身内向けのゲームとなっておりますが、集まりの趣旨を理解して空気を読める素敵なお友達がいれば、
お誘いあわせの上ご一緒に参加して下さいましても大丈夫です。
お友達の方で参加希望の方がいらっしゃれば、本ページをお教え願えますようお願いします。

■参加募集締め切り２月８日予定（※延長いたしました）

■場所
埼玉県日高市インドアフィールドZEEK　2階・Bフィールド

■定員
25名　

■ATNDに登録されない（できない）ゲスト参加の方をお連れになられる場合は、
お手数ですがコメント欄に人数とその方のお名前（HNで可）をご記入頂くようお願いします。

レンタル装備ご希望の方は早めにコメント欄に表明してください。

■必要な物、参加費
参加費：4,500円～（参加者10名以上の場合。参加者が10人以下なら使用料の45,000円を頭割り）

　エアガン：電動・ガス・エアーコッキング式のエアソフトガン
（持っていない場合フィールドでレンタルあります。
＞ガン・マガジン・バッテリー・ゴーグル・ＢＢボトルセットで￥3500/日。そのほかハンドガンや各単品のみのレンタル利用も出来るそうです。BB弾は別途用意。）

　※以下フィールド引用：使用銃に関しましては、当日弾速計により測定
　　　しチェックマーキング済みの銃のみの使用可能となります。　

　使用BB弾：６ｍｍバイオＢＢ弾０．２ｇ以下。
　装備：サバイバルゲーム用ゴーグル（シューティンググラス不可）、顔面を保護するマスク推奨。
　（マスク付きゴーグルレンタル有り）
　
　他に、怪我を防止するのにグローブ手袋、ニーパッド、エルボーパッドなどが役に立つので、
　ご用意されると宜しいかと思います。

　フィールドは多量のBB弾が散乱しており非常に滑りやすくなっています。
　急階段等もありますので滑りにくい靴を推奨。

　お名前（HN）が判るような名札をご用意いただければ幸いです。

　昼食に関してですが、事前に申請していただければ弁当の予約が出来るそうです。
　１食400円となっておりますので、予約したい方は2月8日までにコメント欄にて申請してください。

　買い出し等でフィールド外に出られる際は装備、BDU等着用のまま外出されないようお願
　い致します。

　当日、かんたんなシューティングマッチごっこを行いたいと思います。
　つきましては皆様から賞品を提供して頂けると大変有りがたく思います。
　使用しない物や不用品、いやげ物などネタでも本気でも何でも構いません　。
　もちろん任意ですので、お気軽にお願い致します。

■レギュレーション
　パワー制限は銃刀法に定める規制値以下。
　フリーズコールタッチ、ナイフアタック不可（痛くなさそうな末端を撃って下さい）。
　基本的にブラインドファイア（遮蔽物より銃のみ出しての発砲)禁止。
　外部ソースの使用禁止。

セミオート縛り、弾数制限などの全体的な縛りは行わない方向ですが、セミオンリー戦やハンドガン戦などゲーム内容によっての
縛りは行う予定でいます。
初心者の方でも楽しめるような内容をで進めていきたいと思います。
シューティングマッチ戦も可能であれば行ってみたいと思います。

　その他、フィールドレギュレーションに関しては フィールドHP（http://www.zeekasgp.com/regulation/regyu_ware.html） にてご確認下さい。

■ちょっとしたお願い
締め切り等やその他諸事情で参加を伏せたい人も居ると思いますので、終了後ですが
個人単位で「今日サバゲして来た」と言うのは自由ですが、
誰が参加していたかに関しては『触れない』方向でお願いします。
TwitterやMixi等で「○○さんお疲れ様でした」も出来るだけ空気読んで頂けると助かる人も多いと思います。
（自分が参加していた事を表明する事に問題ない人同士のやり取りについては特に問題ありません）
交流等は本ページコメント欄をお使いください

尚、会場の方に駐車場は30台ほど停められるそうです。

公共交通機関をご利用の方はJR川越線・武蔵高萩駅からタクシー利用が便利そうです。おおよ1.7キロから2キロ未満でワンメーター位だと思います（憶測）。
バスだと武蔵高萩駅バス停から２つ目の女影バス停まで道程の約半分までこれます。
徒歩だと小一時間ほどかかりそうです。

■そのほか、質問やご希望等がありましたらこちらのコメント欄にお書き込みください。
