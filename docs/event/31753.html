概要

Windows 8のアプリケーション開発について、幅広く学べる勉強会です。
日本Microsoftのエバンジェリスト 大西彰様が、講師としていらっしゃいます。

Windows 8の特徴の１つが、ストアを通してアプリケーション配布が行われることです。開発したアプリケーションをストアへ掲載するにあたり、これまでとは異なる開発ポイントを知っておく必要があります。この勉強会でそれらポイントをしっかり押さえていきましょう！
