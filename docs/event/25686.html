―――――――――――――――――

神戸市のロボットテクノロジー（RT）を活用した産業創出を推進する
神戸RTビジネスプラットフォームは、2012年3月23日（金）
「高齢者対応コミュニティをつくるために」というテーマで
ビジネスセミナーを開催します。

今回のセミナーでは、センサ技術やデータベースの活用による個々の
高齢者に適した生活設計の提案やロボット技術の活用による高齢者の
生活支援、社会参加の促進、高齢者対応コミュニティのあり方について
議論します。コミュニティベーストロボティクスという新たな視点に
関する基調講演と介護の現場で活躍される専門家の方々をまじえた
パネルディスカッションを通して高齢者コミュニティにロボット技術
がどのように役立つのかを探っていきます。

介護・福祉の現場で活躍される方、今後介護・福祉分野に
進出しようと考えられている方、それに絡むロボット技術の
新しい転用の仕方を考えられている方など、是非、当セミナー
にご参加いただき、ヒントを得て頂きたく、ご案内申し上げます。
