実践アジャイルテスト読書会 02

「実践アジャイルテスト」の読書会を行います。
平日の夜（火曜日）に、隔週で読み進める形式となります。

日時

2012年3月6日（火）19:00 ～ 21:00
※終了時間は目安です

内容

「 実践アジャイルテスト 」をベースにテストに関して学びます。

勉強会の中では書籍の読み合わせ（輪読）は行いません。
あらかじめ該当の章を読み進めて来ることを前提とします。
したがって、書籍の購入は必須です。

1回の勉強会（2時間程度）で、１章（20～40ページ程度）ずつ進めます。

勉強会の当日は次のような流れで行います。

1. 自己紹介
2. 20の質問（アイスブレイク）
3. 担当者による発表
4. ディスカッション

各章毎に担当者を設定し、内容をざっくりとまとめておきます。
この資料を元に参加者は気になった部分や理解できなかった部分などをあげ、参加者全員で解決していきます。

今回の範囲

今回は２章「アジャイルテスターのための10の原則」です（19〜33ページ）。
尚、今回も書籍を未購入または予習無しでも大丈夫な形で進めます。
