description

最新のAndroid端末を無料で触れます

これからAndroidアプリ、テスト検証サービス事業を立ち上げる事を目的に
国内で販売されているほとんどのAndroid端末を揃えております。
今回はテストマーケティングとして
これらAndroid端末を自由に触って頂くイベントを企画致しました。

社内業務フローにAndroid端末を組み入れようと考えているシステム担当者の方が操作感を試していただくとか、Androidアプリを開発されていて、多様な実機が無くてテストに困っている方などにご参加いただきたいイベントです。
勿論、開発者の方は、ご自身の責任の範囲で、ご自分のアプリをDLして、検証いただくことも可能です。
（混雑具合によっては、時間で交代いただくこともございます。）

今後、反響をみて、
様々な検証サービスを展開していく予定でおりますので、
当日はどういったサービスが必要か積極的なご意見もお伺いできればと思います。

イベントの性格上、タイムスケジュールに乗っ取って何かを発表というスタイルではなく
各自、自由に触って頂いて、ご感想をお聞きできればと思っていますので
時間内にふらっとお立ち寄り頂けましたらと思っています
