#yidev 横浜 iPhone 開発者勉強会です。
第２２回目、本年初の開催になります m(_ _)m

開催日は、噂のイベント try! Swift の終了翌日になります。満足して疲れ切っている人もたくさんいらっしゃるとは思いますけど yidev もどうぞよろしくお願いしますね。

勉強会

参加者の中から発表者を募り、みんなで日々の成果を持ち寄って、談笑しながら学びを深める勉強会です。
講習会ではないですからね :D

同じ趣味を持つもの同士、わいわい、楽しく、技術を深めて行けたらいいなって思ってます。iOS や OS X など Apple 系のアプリ開発に興味がある人が主な対象です。

発表枠もあります。iOS に関わることなら、ちょっと見つけたお話でも、ここまで頑張って調べてみた！でも
なんでも大歓迎 ですので、ぜひお話を聞かせてくださいませ。

参加費（会場費）

- ひとり当たり 2,000 円 です。
- 発表してくださった方は 免除 されます。

発表者の募集

発表してくださる方も募集中しています。発表時間は３０分くらいの人が多いですが、２０分くらいから１時間くらいまで、可能な範囲で調整しますのでぜひ声をおかけください。

せっかくの勉強会、発表するには満たないような小さな話題でも、空いた時間で語り合ってもらえたらいいなと思うので、日々の中でいろいろ思うことを暖めながらお越しくださいませ。

懇親会

勉強会終了後にも、希望者を募って 懇親会 を行います。
中華街でお食事しながらみんなで談笑しましょう。

懇親会の参加希望は こちら からお願いします！
懇親会だけの参加もウェルカムです！
