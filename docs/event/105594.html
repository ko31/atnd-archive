感動の新言語　日本初研修！

Julia(ジュリア）って？

「今知っておけば、２年後プログラマ集団のトップに立てる！」 そう言い切ってしまいたい程の感動新言語が登場しました。

その名も「Julia（ジュリア）」。

Juliaは、汎用の一般計算からディープラーニングまで、非常に広い範囲をカバーしています。
そして、Ｃより楽に記述でき、Pythonより50倍も高速です。

まだまだある！感動のJulia優位性

  • Ｃのように、速い
  • Rubyのように、動的である
  • Pythonのように、一般的な書き方ができる
  • Ｒのように、統計処理ができる
  • Perlのように、文字列処理ができる
  • Matlabのように、数学的表記ができ、線形代数の強さをもつ
  • Lispのように、真のマクロをもつ
  • 変数・関数名に日本語なども使える
  • 並列処理と分散計算ができるよう設計されている
  • 対話型である
  • コンパイラ言語である
  • インタープリター型のデバッガがある
  • 上位・後方互換性を保証している　　　　　　・・・・・・・などなど

実は、Juliaは全米屈指のエリート名門校：MIT（マサチューセッツ工科大学）が開発してきた新言語です。
それを聞いて、なぜこれほどまでに素晴らしい言語なのか、納得ですね。

アークは、国内で初めて、この感動の新言語を
理解、体感して頂く場をご提供できることになりました！
