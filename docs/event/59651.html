「社長！ドラえもんになってはいけません！」

イベント ユーザー

title 「社長！ドラえもんになってはいけません！」
catch ～　社長が出社しなくても会社が回る業績UP術　～
description

業績が頭打ちになっていると感じている経営者様へ

社長が会社に来なくても、勝手に業績が上がる人材マネジメント手法を公開するセミナー
突然ですが、こんな悩みを持っていませんか？

◇とにかく社長が社員以上に働いている。
◇社員に仕事を任せるのが怖い。
◇仕事の依頼はあるのに、断っている。
◇寝る間がない。

一つもあてはまらない方は、お読みいただくのはここまでです。
一つでもあてはまる方は、この続きをお読みください。

私たちが、採用戦略を作る際に行なうヒアリングで、
お客さまから出てくる悩みの大半が、上記の４つ。

では、どう改善すれば良いのか？

こう考えても、解決策が出てくるわけではなく、結局、同じことの繰り返し。
悪循環に陥っている社長を何人も見てきました。

今回は・・・

組織に属していた時には営業本部長、人事本部長、経営企画部長、
採用責任者、教育担当という立場、

独立してからは代表取締役、コンサルタントという立場にいた弊社代表 高地が、
自分の経験から、社長が会社に来なくても、勝手に業績が上がる人材マネジメントの手法を
お話しさせて頂きます。

営業本部長という立場から見た“組織とヒト”

人事本部長という立場から見た“組織とヒト”

経営企画部長という立場から見た“組織とヒト”

採用担当者という立場から見た“組織とヒト”

教育担当者という立場から見た“組織とヒト”

代表取締役という立場から見た“組織とヒト”

コンサルタントという立場から見た“組織とヒト”

本セミナーでは、実際に悩みをどのように解決していったか、
事例を含めてお話する予定です。

少しでも皆様の経営のヒントになれば幸いです。
