こんにちは！銭湯部部長のたむらです。
約3ヶ月ぶりに開催します、蒲田銭湯部。
今回は、夏にふさわしく、風呂あがりのかき氷パーリーをやりたいと思います！

今年の夏は、震災の影響もあり、電力が足りないから節電でクーラーを使えなかったりと
いろいろと制約もあったりしますが、本来夏は暑くて汗をかくものです。
暑い中、つめた～いかき氷を食べることによって、暑さもまた美味しくかき氷を食べるソースになるのではないでしょうか！？

今回は、さらに久しぶりの番頭さん体験の第一弾として開催された、お風呂掃除もやります。

暑い中、みんなでお風呂をザッパ～ン！と掃除し良い汗をかき

自分たちで掃除したピカピカの1番風呂でさっぱりと汗を流し

火照った体でつめた～いかき氷をシャクシャクっと食べる！食べる！食べる！

某パティシエが作る美味しいかき氷を食べながら、

この夏の過ごし方や、銭湯の未来、銭湯の活用手段など
ゆるくブレストしていきましょう～！
