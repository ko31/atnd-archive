description

中国古代より、時代を超えて伝わっている重要な書物には３種類があります。
それは、経と訣と呪です。
これらを中国十翼書院設立者の米鴻賓先生は、今回日本にも伝承しようとしています。
特に、奇門・太乙・六壬の三式と呼ばれる占術は、儒学八目の中の「格物」の用法であり、漢代の経学集大成者の鄭玄・三国時代の諸葛孔明・明代の王陽明などが継承して来ました。
さらに三式の難解さを解消し、予測的中率の高さを引き継ぎ、簡潔に未来を予測することができるようにまとめられたのが、米先生の家系が受け継いできた「六壬神課金口訣」です。
金口訣の特徴は、吉凶を判断する課の作成が簡単。そして、判断方法も易しいことです。
その初歩から応用までを、通訳をつけて解説してくれます。
日本に今まで伝わっていなかった金口訣を、中国第一人者が解説する貴重な内容になります。
