description

Unity技術者交流会を開きます。
興味のある方は是非ご参加ください。

大義名分

Unityに関する、あらゆる交流を行う場を提供します。

目的

まだまだ情報が出揃ったとはいえず、業界として未成熟な状況だからこそ、「横のつながり」は得がたいものではないでしょうか。
技術者同士でノウハウを共有するだけでなく、足りないものを補いあえるような場所を提供したいと考えています。

このような方に

・興味はあるけれど、記事やコラムを読むばかりで、実　際に使った事がない方
・何か一つアプリを作って、実績を残したいという方
・いくつかアプリは作ったけれど、他の人のつくり方を参考にしたい方
・足りないと感じている技術分野を得意としている人と知り合いたい方
・企画はあるけど、実現するには力が足りないと思っている方
・案件をお持ちの業界の方で、有望な技術者と知り合うチャンスが欲しい方。
