コーチング・心理学から学ぶ！相手の心を開く話題の選び方・話し方
**************************************

友人や知人と交わす「会話」。
気心の知れた仲であれば楽しく話せたり、特別何も考えなくとも会話が盛り上がります。
しかし、あまり親しくない相手や知り合って間もない相手となると、会話が続かなかったり話題作りに困ってしまったりと、ちょっぴり困ってしまうこともあります。

どんな相手にでも共通する「相手の心の開き方」があります。
相手が心を開いてくれることにより、会話はどんどん盛り上がっていきます。
「相手が心を開く話題の選び方・話し方」を学び、誰とでも楽しく会話が出来る日常生活を過ごしませんか♪

～セミナーの内容～

・自分の「気持ち」を表現するのが鍵
・ごく最近の出来事にネタがたくさんある
・お互いの「クセ」が心の距離を近づける
・自分の弱点を味方につける
・相手の懐に飛び込めば心を開いてくれる

～このセミナーはこんな方におすすめです～

・誰とでも快適な人間関係を築きたい
・みんなに好かれる自分になりたい
・たくさんの人脈を作りたい
・人と話すことが楽しくなりたい
・コミュニケーション力を身につけたい
・上司･部下･同僚との絆を深めたい
・営業成績を良くしたい

etc…
