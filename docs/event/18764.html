イベント概要

開催日時：平成23(2011)年9月11日(日) 13:00-

開催場所：新潟県長岡市島崎(旧和島村) 羽入商会敷地内・岡本学習教室

参加費：大人500円 お子様無料。

持ち物：副菜(後述)。そしてどんな臭いにも引かない心構え。

　「夏のシュールまつり」ではパン、ジャガイモ、玉ねぎ等の副菜はこちらで用意しましたが、今回は全て持ち寄りでお願いしたいと思います！No副菜、No Life。
　というわけで必要な副菜の持ち寄り係を決めたいと思います。係の人が決まったものにはお名前を付けて行きますので、未定ものものに手を上げて頂けるとありがたいです。ツイッター等々で返信下さい。なおマッシュポテトはこちらで用意する予定です。その他、いろいろ持ってきて頂けるとうれしいでっす！包丁、まな板、ガスレンジ、電子レンジ等の簡単な炊事設備有り。ご自慢の一品、余っているお中元、飲みたい方はアルコール類等お持ちよりください！(飲酒運転はダメ絶対。)

  • 玉ねぎ係：1名(1-2個)・・・jinpokoさん
  • トマト係：1名(4-5個ほど)・・・jinpokoさん
  • 食パン係：2名(それぞれ食パン2斤)・・・a_kieさんとその友達
  • クラッカー係：1名(オサレカナッペ風のアレ)・・・momo_canさん
  • チーズ係：3名(それぞれクリームチーズ1パック)・・・nobuchunaさん、gatabutsuさん、noto_yさん ※前回少なめだったので増やしました。
  • 飲み物係：2名(それぞれ牛乳1本とその他ジュース等)・・・bdprさん、
    atsu_atsuさん
  • 係のない人：シュールにあいそうなもの、なんか美味しいもの、飲みたい・食べたいもの持ってきて下さい。

　パン、チーズ系はもっとあってもいいかも？よろしくお願いしまーす☆
