HTML5で重力センサーを使ったゲームを作りましょう！

iPhone4は傾きや回転、重力などのセンサー情報をHTML5で取得することができます。
センサー基礎をワークショップ形式でご体験いただき、端末を傾けて遊ぶゲームを作ります。
サンプル動画はこちら

対象者

・HTML5に興味があれば、プログラマ・エンジニアでなくてかまいません。
　初心者のかたでもOK。まずは触れてみましょう！

・iPhone4以降をお持ちの方
　iPadや、一部のAndroid（標準ブラウザ/FireFox）でもご利用いただけた
　という情報があります。
　http://jsrun.it/nakashimmer/vO7h にスマフォでアクセスしていただき、
　端末を傾けると三角の自機が上下左右に動くようでしたらご参加ください。

用意するもの

・ノートPC
　　・safari または chrome をインストール
　　・UTF-8の文字コードを扱えるテキストエディタをインストール
・iPhone 4以上
・参加費：4,000円(こちらではリピータ割引はございません)
