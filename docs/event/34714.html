こんな時期にやって誰か来るんだろうか…不安を抱えながら開催！（最小催行人数3名）

年末だ、クリスマスだ！　だからどうした、とにかく僕らは作るんだ！

もくもく会とは？

カフェとか誰かの家とかに集まって、それぞれ各自もくもくと勉強したり作業したり本を読んだりするだけの会のことです。

ジャンルは今回、コンテンツ関係（ゲーム、映画、ウェブサイト、イベント　など）に限らせていただきます。

PC持ち込み、飲食可。場内WiFi飛んでいて利用可。募集人数は7名（先着順）。参加表明の際、できればどんな作業をするかも教えてください。質問などあれば@kawanakakiyoko まで。

前会好評につき、今回は2days開催

翌日23日（日）も、最小催行人数（3名）を超えた場合開催します。2日連続でご参加の方は、別途 こちらから お申し込みください。

参加条件

何かコンテンツ（ゲーム、映画、ウェブサイト、イベント　など）を作りたい人。商用か非商用かは問いません。コンテンツっぽいことであればたいがいなんでもOKです。

例
  1. プログラムを組んだりゲームを作る
  2. ウェブサイトにアップするイラストを描く
  3. DTMで音楽を作る
  4. イベントの企画書や映画のシナリオを書く
  5. 仕事のウェブサイトを作成する
  6. 芸大などの卒業制作
  7. コミケの準備
  8. プログラムの勉強のため読書
    など

なお、下記のような作業をされる方はご遠慮ください

  1. 周囲に音が漏れる
  2. 会議や打ち合わせなど喋る必要がある
  3. 飲酒しながら
  4. 独り言がデカイ
  5. 作っているふり

備考

前回の様子は こんな感じ でした。

  1. 館内禁煙です
  2. ポットのお湯、電子レンジ、冷蔵庫はUZUさんにお借りして使用できます
  3. 館内は比較的静かですが、周囲の音が心配な方はイヤホンなどの持ち込みをお勧めします
  4. 暖房は入れますが、足下が冷えるかもしれません。また、快適な作業のためクッション、膝掛け、履き替え用のラフな靴（館内コンクリの土足です。ずっとブーツなどはしんどいかも）、マイカップ　などの持ち込みもお勧めです
  5. まわりにお昼ご飯食べるところは結構あります。コンビニ（セブンイレブン）も徒歩2分くらいのところにあります。前回は大雨だったのでみんなで中華の出前を取りました。
  6. 自転車で来られた方は、ビルの壁にぴったりつけて駐輪してください。専用駐車場はありませんが、コインパーキングは近くに何カ所かあります
  7. 2daysご参加の方のお荷物に関して、ノートPCや重い道具などは場合によっちゃ会場でお預かりできるかも知れません。当日ご相談ください。
  8. pixelさんは今回家庭の事情で欠席が濃厚です。
