ESP-WROOM-02でArduino

4月17日の勉強会はESP-WROOM-02を使った電子工作をテーマに行います。

今回取り上げるESP-WROOM-02と言うWi-Fiモジュールは、単なるWi-Fiモジュールと言うだけでなく、Arduino IDEで開発を行うこともできます。つまりWi-Fi付きのArduinoのように使用できるので、モジュールにセンサーなどをつけて、ネット上のサービスでログを取ったり、ESP-WROOM-02からWeb APIを呼び出すようなこともできます。しかし、今回は入門編として行いますので、ESP-WROOM-02の概要と、Arduinoとしての初歩的な使い方が中心の勉強会になる予定。

用意する物

  • Arduino Software (1.6.8)をインストールしたMac
  • インターネット接続環境(モバイルルータなど)

推奨パーツ

実際に自分で体験したい人は以下のパーツもご用意ください。全てスイッチサイエンスで購入できます。

なお、以下の推奨パーツを使用して勉強会を進めますが、同等品であれば必ずしも同じ物でもなくても大丈夫です。ただし推奨品以外を使う場合は、配線などは各自その環境に置き換えて作業をお願いします。
