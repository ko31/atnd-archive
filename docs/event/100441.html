＜こんな事が出来るようになります＞

１０時開始で１７時終了(お昼は１時間休憩)の６時間になります。
場所的に、近くにはいろいろな飲食店が揃っています。

  2. HTMLで雛形作成
  3. PHPの組み込み

  5. お問い合わせフォーム作成(入力フォーム編)
  6. お問い合わせフォーム作成(確認フォーム編)
  7. お問い合わせフォーム作成(送信・ありがとう画面編)
