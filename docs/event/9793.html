Webフロントエンドな人たちを集めて、
フロントエンドの話をしよう！ …な感じの集まりです。

  1. 自室で作業する気の起きない人が
  2. カフェにノーパソ持ち込んでする作業に
  3. どうせなら人呼ぼうかって程度の
  4. 週末プログラミング＋＋みたいなイメージ。
  5. 作ったものの感想が聞きたいとか、
  6. あのひとはどんな環境で作ってんだろーとか

何がしかネタや質問リスト等を持ってくると、より楽しめるかもしれません。

  • Livedoor Wirelessは100％使えます。
  • Livedoor Wirelessだと、MACアドレス認証やっとくと吉。

場所代として 一人1500円 ほど見といてください。
当日は 現地集合！ ぐだぐだ話しましょう。
