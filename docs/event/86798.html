【起業に必須な3つの準備がデキる！！！】
　それぞれの分野で"結果を出す"ために必要な
　　★カバン：資金、個人の時価総額
　　★ヂバン：ブランディング
　　★ジバン：人脈（顧客、その分野の権威、起業仲間）
　の準備がデキる勉強会です！
　
　起業の仕方が変わる、新しい勉強会です！！！

======
こんな方々にオススメです！
======
　★近い将来、起業を考えている方
　★自分で売るモノを持っていない方
　★多くの著名人との人脈をつくりたい方
　★人に喜んでもらえる事をシゴトにしたい方
　★自分のビジネスを飛躍的に成長させたい方
　★自由な時間を手に入れたい方
　★自分自身をプロモートしたい方
　★手持ち資金はないが、ビジネスを始めたい方
　★すでにビジネスを始めたが集客がうまくいかない方
　★シンプルなビジネスから始めたい方

======
こんなコトが学べます！
======
　起業に必要な3つの準備 である、
　『　資金・ブランディング・人脈 　』
　の準備をする方法として、
　ビジネスパブリシストのノウハウをお伝えします。

　　
　★★ ビジネスパブリシスト゛とは？ ★★
　　　広報のことです。
　　　売るモノ（知識やノウハウ）を持っている人の
　　　勉強会を開催したり、教材販売をします。

　★★ 起業準備に適している理由 ★★
　　　勉強会の開催、教材販売をする事で
　　　起業に必要な3つの準備 が『一気に』手に入るからです。
　　　
　　【準備１：資金】
　　　・勉強会や教材販売による、収益が得られます。

　　【準備２：ブランディング】　
　　　・専門家・技術者が多い一方、
　　　　それを広めるノウハウを持つ人は
　　　　希少価値の高い存在になれます。

　　　・「その分野の権威」である人と
　　　　仕事をしている、繋がりがあるという
　　　　ブランディングが手に入ります。

　　【準備３：人脈（顧客、その分野の権威、起業仲間）】
　　　・その分野の権威との人脈がつくれます。　　
　　　・勉強会参加者や教材販売の顧客リストが
　　　　あなた自身の顧客リストとして使えます。
