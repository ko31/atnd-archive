◆Overview　概要◆

Interested in Data Analysis and Machine Learning? Then this course is for you!

This course will walk you through the basic Python skills and the workflow of Data Science. With 2-hours course, you will improve you understanding of the job of a Data Analyst and grab the basic concept of Machine Learning.

この講座はデータ分析や機械学習に興味をもつ方々にむけて開催いたします。

2時間で簡単なプログラミングコードをPythonで書けるようになり、ビジネス上役に立つデータ分析や機械学習の基礎知識を実例を元に学ぶことができます。

◆Agenda　予定◆

  • Part 1: Introduction to Data Analysis
  • Part 2: Python basics (Pandas, Numpy, Matplotlib, Seaborn)
  • Part 3: Machine Learning Model with business examples
  • Part 4: Real Project - Titanic

第1部分：データ分析入門
第2部分：Python基礎（ライブラリー：Pandas, Numpy, Matplotlib, Seaborn）
第3部分：ビジネスで使用する機械学習モデルの紹介
第4部分：実例：Pythonでタイタニック号乗客の生存を予測
