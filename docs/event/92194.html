★講演内容

PLAN-BのJuicer事業部で取り組んでいるビッグデータ、
機械学習のナレッジを一挙大公開。

ハンズオンも開催するので興味がある方はぜひ参加ください。


★講演者プロフィール

■講演① スピーカー: 箱根 祥雅 ( はこね よしまさ )
株式会社PLAN-B Juicer事業部

Juicerのログ基盤を担当。
徐々に増えていくログと徐々に増えていくサーバを横目に
時折鳴り響くアラートにビクッとなりながらも
日々半ニヤで仕事をしております。
ありえない物が存在する世界がどんな世界なのかを妄想するのが好きです！！


■講演② スピーカー: 粟田 順也 ( あわた じゅんや )
株式会社PLAN-B Juicer事業部

京都大学で物理学の勉強をしていたが、
PLAN-Bでのアルバイトをきっかけにデータサイエンス領域にのめり込む。
入社後Juicer事業部の理系チームとして、
Juicerに溜まっている膨大なデータを日々分析している。
趣味はテニス、好きな食べ物はタコ、得意な手法はRandomForest。


■講演③ スピーカー: 藤本 永一 ( ふじもと えいいち )
株式会社 PLAN-B Juicer事業部

Juicerのインフラやプログラミングの傍ら、統計やデータ分析周りを勉強中。
Juicerの持っている膨大なデータを活用するため日々奮闘している。
（チーム内で唯一、算数までしかわからないのは内緒）


■講演④ スピーカー: 得上 竜一 ( とくがみ りゅういち )
株式会社 PLAN-B 技術統括

データを集めるとこから、使うとこまで全部やりたくて、
インフラ設計、構築から、会社の設立、YARNアプリの実装も。
そんな10年過ごしてきたら、たどり着いたがPLAN-B。
色々なサイトのログが集まるって幸せだなと思ってます。
日々オープンソースのチュートリアルをこなす日々を送ってます。
