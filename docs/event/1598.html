description

毎月最終週の月曜日と第３月曜日に定期開催することが決定しました！！

会の趣旨は

毎月２回　週初めの月曜日　仕事が終わってから
いろんな人との交流と人脈を広げるのはもちろん、他の方からアウトプットを聞いて知識、見識を広め
自分も積極的に仕事や趣味の話などを他人にアウトプットしていきましょう☆

そしてその週の仕事やプライベートのモチベーションを上げていこう！！

そんな会です。

●週一で開催している『早朝アウトプット交流会』は下記のような感じです。

http://mixi.jp/view_event.pl?id=45614093&comm_id=1288825

●前回の第２回は夜の交流会はこんな感じでした
http://mixi.jp/view_event.pl?id=45878682&comm_id=1288825

アウトプットするネタは

◆最近読んで影響を受けた本
◆最近見て影響を受けた映画
◆自分の趣味の話（オタクネタ歓迎）
◆自分の仕事の話（マニアックな話も歓迎）
◆その週（もしくわ前週）の日経新聞の話題をアウトプット
◆テレビ番組、ニュースからのアウトプット

等々

参加者が「へぇ～～！」というためになるような内容のネタ話を
１人自己紹介１分アウトプット３分～５分　で回していきます☆

途中で突っ込みや質問をみんなで入れながら
会話の内容を膨らませながら
と言う感じです☆

全員回し終わってからフリータイムでアウトプットネタや仕事の話など
席移動しながら交流していく流れです。

●開催場所は
東京国際フォーラム内の
ロイヤルカフェテリア

http://r.gnavi.co.jp/g183401/

●会費は
各自個人飲食　（飲み物コーヒーなど380円前後、カレーなど食べ物900円前後）

と参加費１人１０００円

※お金を払うことによって
払ったからには何かを得て帰ろう！と必死になると思います。
より緊張感がある意識レベルの高い交流会になると思います☆

★参加費を貯めてこの会のホームページ、メーリングリストを作ったりの運営費に回していきます★

●時間は
１９時３０分～２２時００分

※途中参加、途中退場　全然OKです

人数は特に設定していませんので
職場の同僚、お知り合いなど連れきても大丈夫ですので
たくさんの方のご参加をお待ちしております♪

★お手数ですが参加する方は、アウトプット時間の短縮もありますので事前に
コメントで

①年齢 （女性で書きたくない人は書かなくてＯＫ）
②現在の住まい、地元
③仕事（何業、何関係の会社か　自分がやっている仕事もできたら）
④自分のバイブル（一番影響を受けた、好きな本）　
⑤自己ＰＲ　頑張っていることなど　

を書いてください（前回かいてくれた方はコピペでOK）

そこそこしっかり書いてもらえない方は
参加資格なし！にしますので
その点よろしくお願いします★

★参加者へのこの会のルールです★

この会の特徴は
良くある

「名刺を相手から出されたらしょうがないから自分も出さないと気まずい・・・」

という良くある　名詞の数は減るけどたいした人はいなかった・・・・と言うような
　『名刺交換会』的な交流会とは明確な違いを作っていますので

『名刺交換を禁止』といたします（特に個人名詞）

名詞を自己紹介のときに皆さんにどこの所属（企業や会社）しているからを明確にしてもらいたいので（胡散臭い人を排除したいので）

１枚だけ用意してください♪
（学生などない人は別にＯＫ）

会が終わってから外で個人的に交換はＯＫですが

『自分が渡したくもない人からの名詞交換しませんか？オファーは拒否してください』

名詞は自己紹介のときに
皆さんにどこの所属（企業や会社、団体）しているからを明確にしてもらいたいので（胡散臭い人を排除したいので）

１枚だけ用意してください♪
（学生などない人は別にＯＫ）
