description

茨城を夢と活躍の地域へ。意欲的な茨城人が集まってます。

＜茨城起業茶屋ってどんな場所？＞
茨城の県南地域において……

起業家が自分の事業アイデアをぶつける場所
経営者が自分の事業を見直す場所
起業準備者が経営を学ぶ場所

そして、
積極的な者同士が仲間となる場所

＜なぜやるの？＞
茨城には気軽に起業の話を自由にできる場所がありません。

これまで起業や経営の話を躊躇していた方、
ここではどうぞ自由に話してください。

そして、
近い将来ここでの話を実現し、
新しい売上を生み出してほしいと思います。

＜こんなことが体験できます＞
起業家･経営者との横の繋がりの意見交換。

答えは無く、こーしたらいいんじゃない!?
あーした方がいい、あーだこーだ…。
で、家帰って参考にして考える。

こんなやり取りが、
その後どれだけの大きな価値を生むことになるか、
想像を膨らませてみてください。

しかもこの会は少人数制。
丁寧なやり取りが繰り広げられます。

＜こんなことを学びます＞
価値創造と組織効率化、地域独自性を軸に
起業･経営のノウハウ、実態を学ぶことができます。

＜本気ならこんなことが出来るようになります＞
開業･設立
売上創出
利益向上
人脈形成

＜こんな風に教えます＞
話を聞いて伝えてを繰り返します。
その中で自由に楽しく学んで頂きます。

主催者の一方的な説明は行わず、
あくまで参加者の皆さんと一体となって会話を進めていきます。

　＜対象はだれ？＞
茨城で新しい売上を生み出したい。
意欲的な起業家、経営者、起業準備者が対象です！
起業家精神をもった一般の方ももちろん参加可能です。

意欲的な方であればぜひお越しください！

＜最後に主催者から＞
百聞は一見にしかず。
素早い判断でお越し頂けたら幸いです。

参加費とは別にコーヒー代（実費）がかかります。
