イベント概要

イスラエルに展開するグローバルなテクノロジー企業から、なぜイスラエルのハイテク系スタートアップはこれだけ「引き」が強いのか、その本質に迫ります。イスラエル事情に詳しい方、実際にイスラエル企業へ投資するVC、イスラエル企業の日本のカントリーマネジャー、日本企業でイスラエル企業の技術を使いビジネスを行う方々が集まります。

１部 : テクノロジーのホットスポットとして世界的に評価されるイスラエルの現状
２部 : 世界展開するイスラエルのIoTスタートアップ
３部 : パネルディスカッション

「イスラエル」について

シリコンバレーの源流、スタートアップ大国イスラエルでは、ハイテクスタートアップは、毎年 800 社程度が生まれ、現在 6000 社程度が存在するとされています。また、テクノロジー企業による M&A も盛んで ２０１５年も活発な状況が続いております。

2015年1月には、たった1週間だけでCloudOn（Dropboxが170億円ほどで買収したモバイルクラウドアプリ）、Visualead（アリババが10億円ほど投資したQRコード系スタートアップ）、Annapurna Labs（Amazonが400億円ほどで買収した半導体スタートアップ）など、合計、900億円のディールが動きました。
今年に入って、Google の pulse.io、Timefulの買収、MicrosoftのN-Trig,equivioの買収、AppleのLynxの買収、PaypalのCyActiveの買収、InfosysのPanayaの買収、電通のAbagadaの買収など総額4000億円を超えております。(2015年6月末時点)

一方、日本に目を向けると、2013年度の日本におけるベンチャー投資総額は約1800億円です。既に、2015年に入っても日本の投資額1年分を超えており、人口800万人の国であることを考えると、イスラエルのスタートアップ界隈の注目の高さが数値面でもわかります。
