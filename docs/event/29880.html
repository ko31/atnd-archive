Scene Builder で FXML

JavaFX は GUI の構造を FXML という XML 形式で記述します。この FXML を WYSIWYG で編集できるのが、Scene Builder です。

Scene Graph を使えば、あっという間に GUI が作れんです。そこで、本セッションでは、実際に Scene Builder を使用して GUI を作りながら、Scene Builder の紹介を行います。
