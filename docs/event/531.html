description

■会費：200円(会場費をみんなで割り勘)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　●セッション１
　『Ruby入門』藤岡さん


　＜内容＞
　Rubyの過去・現在・未来
　Rubyのライブラリ「Nokogiri」などの紹介と応用
　Ruby on Railsのインストール
　付録 「Parrotの動向」

　＜藤岡さんの紹介＞
　TS Network (テキスト＆スクリプト ネットワーク)メンバー
　TSNETスクリプト通信編集委員

　●セッション２
　『Flashを開かずにswfを使いまわす汎用swfの解説と配布』柳谷さん

　＜内容＞
　Flash苦手なコーダーさんも大丈夫！swfを配置するだけで
　決まったサイズにトリミングされてアンチエイリアスもかかる
　サムネイルビューアーを開発しました。

　参考）京都福祉専門学校　資格取得者/内定…エリアの学生写真
　http://www.kyoto-fukushi.ac.jp/

　Flashを開かずにいろんな案件でswfを使いまわせます。
　html・cssではできない「アンチエイリアス」をつけたい場合や、
　決まったサイズにぎりぎり収め、余りは削るトリミングをしたい場合に便利です。
　・機能の説明
　・配置の方法
　・制作工程
　・配布
　を当日行いたいと思います。

　＜柳谷さんの紹介＞
　カンドウコーポレーション　デザイナー
　ウェブデザイン・イラスト・フラッシュアニメーション
　インタラクティブフラッシュコンテンツなどを提案・制作しています。

　●セッション３
　『CakePHPによる高速Web開発！のさわり』竹本さん

　＜内容＞
　PHPの高速開発フレームワーク、CakePHPについてさわりをご紹介いたします。
　CakePHPは以下のような特長を持つオープンソースのフレームワークです。

　・強力でシンプルなデータアクセス機能の提供（SQL要らないよ）
　・モジュール化による冗長コードの削減（コード減らせるよ）
　・設定が簡単、外部ライブラリに非依存（単体ですぐ動くよ）

　CakePHPの概要や特長について、実際にCakeで作成した
　簡単なWebサイトのコードを紹介しながらご説明していきます。

　＜竹本さんの紹介＞
　広島の某IT系企業に勤めるSE。
　日曜Webクリエイターです。

　＜参考サイト＞
　CakePHPを使って制作・運営しているサイト
　　　　↓↓↓
　今週のハイライト – デザイン系ソーシャルニュースサイト インデザ.jp
　http://indeza.jp/

　●セッション４
　『GTDタスク管理ツール“Remember the milk”を徹底解析！』藤本

　＜内容＞
　仕事に限らず、日常的にひたすら発生する
　やらなければいけないタスク。

　自分自身、やることが多くなってくると
　把握できなくなって気持ちばっかり先走り
　目の前の作業に集中できなくなってしまいます。

　そこで、以前よりは少しは改善した？
　GTDという考え方に基づいて、Remember The Milkという
　Webアプリを使った管理の仕方を説明します。

　私自身、Remember The Milkと３年前に出会い
　それからずっと使っていますが
　まだまだ日本語のマニュアルが不足しているため
　一人で一から使い方を覚えるにはとても大変です。
　この機会にぜひ一緒にRTMを使ってみてはいかがでしょうか！？^^

　＜参考サイト＞
　GTD(Getting Things Done) – Wikipedia
　http://ja.wikipedia.org/wiki/Getting_Things_Done

　Remember The Milk
　http://www.rememberthemilk.com/

　＜藤本の紹介＞
　以前フリーランス、今はしがないサラリーマンWeb制作者です(しがないゆうなっ)
