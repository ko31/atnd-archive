  • ブランディングとは何か
  • 自分の人生に置き換えて理解する
  • 中小企業ケーススタディ
  • ブランドを育てる・守る
【7つのことが得られます】
  • ブランドを多角的に捉えられることが出来る
  • 必要な要素・工程を理解することが出来る
  • ミッションやビジョンなどの考え方が解かる
  • 戦略、デザイン、コピーなどの幅が広がる
  • 自分ブランドと向き合うことが出来る
  • 企業ブランドをシンプルにし腹落ちできる
  • ちょっとした熱意
