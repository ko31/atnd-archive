description

全世界300万を超える(もうすぐ400万)ユーザーを達成し、市場評価額500万ドル(400億円)超を誇るHootSuite。
そのHootSuiteを初期から支えるEricがUI/UXについてお話し致します。
またHootSuite Japanese Community Coordinator として約2年間カスタマーリレイションに関わっているTakayaよりHootSuiteが成功した秘訣をカスタマーリレイションを担当した立場からお話し致します。

======講演者情報======

【 Eric Puchmayr 】
HootSuite Interface Developer
http://ca.linkedin.com/in/ericpuchmayr

2008年のHootSuiteの誕生よりインターフェイスデザインの開発に携わり、HootSuiteの発展を３年以上支える。
ITの本場　北米にてウェブインターフェイス開発には５年以上携わる。

【 Takaya Kitazawa 】
HootSuite Community Coordinator
http://jp.linkedin.com/pub/takaya-kitazawa/36/46b/265

2010年のHootSuite日本語化より日本向けカスタマーリレーションに携わり、HootSuiteユーザー数世界第二位を誇る市場を担当。
また日本で70万インストールを誇るiPhoneアプリのカスタマーリレーションにも担当。
