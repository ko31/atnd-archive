description

あなたは今までの人生で誇れるような体験、
一生モノの満足をした経験ってありますか？

それはくじの当選やSNSの「いいね」、ゲームなどから得られたものでしょうか？

人生の充実感には2種類あります。

いつになっても思い出すことができる賞味期限の長い満足感と、
すぐに忘れてしまう賞味期限の短い満足感。

くじの当選やSNSの「いいね」などは、賞味期限が短い満足感であり、
何年も過ぎてからは思い出すことができない満足感になります。

それでは、賞味期限の長い満足感であり、人生で誇れるような体験や、
一生モノの満足はどのような経験から得られると思いますか？

今回、創業者セミナーでは、Pando事業を立ち上げたクインテットの創業者である松下耕三から、人生を充実させるための大事なことについてお話させていただきます。

テーマは、大きく分けて2つ

・ビジョンとは何か、どこからやってくるのか
・一生モノの満足はどうしたら得られるのか

クインテットでは、「後世に素晴らしい時代を残す」というビジョンを持ち、ひとりひとりが人生を謳歌できる社会を創る事を目標に事業を展開しています。

人生を謳歌するというのは、人生の充実を意味していて、人生をかけて成し遂げたいと思えるモノを見つけ、その夢に向かって取り組む事が、人生の充実につながると考えています。

困難に直面したり、苦労しても諦めずに頑張って、
何か達成することで一生続く充実感・充足感を得ていくと考えています。

セミナーを通じて、ビジョンを考える重要性が伝わり、考えるきっかけを与えられればと思っています。
