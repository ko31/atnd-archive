catch 初心者でもOK！全3DCGデザイナー参加OK! 次世代ゲーム制作を体験
description

■講座概要 ・PlayStation4、XboX-ONEなどハイエンドの次世代ゲームが次々に発売され、スマートフォンの3DCGもそれに追従して発展しています。この講座では、そういったゲームコンテンツを今お使いの3DCGソフトから制作できるノウハウを学べる実践的な体験実習です。
・Maya、3dsmax、Blender、Metasequoia、Shade、Cinema4Dなど、いずれかの3Dデザインツールで作成したキャラクターモデリング、シェーディング、アニメーションを最新技術で話題沸騰のゲームエンジン「Unreal Engine 4」の連携させるノウハウを学びます。
・素材テーマは株式会社インフィニットループのイメージキャラ「あいえるたん」を使って、アクションゲーム制作のコンテンツ例を時間内に完成させます。
　http://www.infiniteloop.co.jp/special/iltan/3d-dl.php
・ゲームエンジンのセミナーは、とかくエンジニア視点が多い中で、デザイナーをメインとしたものであるのが特徴です。エンジニア的なノウハウもデザイナー視点で理解できるような授業展開です。
・「映像中心なので、ゲームは関係ない」という方もこの講座はオススメ！ゲームエンジンをリアルタイムレンダリングツールとして使えば、レンダリング待ち時間なしです。これを知らないと損？
・授業で使うオリジンナルの素材とテキストをお持ち帰りできます。その後の質問やサポートも充実対応します！

■開催日時　2014/12/19　金曜日　19:00～20:30
■会場　日本電子専門学校　7号館（校舎が複数ございます。お間違えのないように）
https://www.google.com/maps/d/u/0/viewer?msa=0&mid=zA93plHwcc0Y.k0GSgia8Zr1I

■カリキュラム内容：90分（質疑応答あり）
１．Unreal Engine 4の基本操作、システムを学ぶ
２．MayaのデータからUnreal Engine 4へモデルデータ変換ノウハウ
３．Unreal Engine 4での独自の高画質リアルタイムシェーダー設定でレンダリング
４．キャラクターアニメーションをUnreal Engine 4での再生
５．プループリントによるＧＵＩプログラミング基礎
７．リアルタイムレンダリングを映像として書き出す方法

■料金　3000円（税込）

＊ゲームエンジンに関して初心者のＣＧデザイナー、プランナー、ディレクター
＊何かしらのツールで３ＤＣＧ開発経験がある方。初心者でも歓迎です！
＊学校や独学で２Ｄ・３Ｄデジタルコンテンツ制作を学ばれている方
＊企業での新規事業でゲームエンジンの活用を検討・採用されている企画担当の方
