Global Game Jam 大阪とは

Global Game Jam (以下GGJ) は、ギネス記録を持つ世界最大級のハッカソンです。
全世界に200以上のハッカソン会場が設けられており、全世界で同じ日に同じテーマで、ハッカソンを開催します。

おおまかな流れ

プロ・アマ・大学生・専門学校生が混じり合ったチームを結成
世界統一のテーマが出題されるので、それに沿ったゲームをチーム単位で
ワイワイガヤガヤいいながら楽しく作る
48時間内にサイトにアップロードし、互いにコメントし合う

GGJの面白さ

即興で組んだメンバーでつくり上げるゲーム製作
この高揚感と時間制限によるハラハラは体感してみないと分からない。
テーマについてネタが飛び交い、笑いながらすぎる初日
そして始まるゲーム製作２日目と、迫り来るデスマーチ

48時間の偉大さを心底体感出来るイベントです。
