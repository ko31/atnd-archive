ヒカ☆ラボ一覧は こちら！！

★講演内容

AWSの無料アカウントを作って試してみたけど、そこで止まってしまっている方や
従量課金制であるAWSの実際の運用費がどれぐらいか
知りたい方が多くいらっしゃるのではないでしょうか。

AMG Solution社で運用しているオウンドメディアや
自社サービスの最小構成の運用方法や費用についてお話しします。
これから導入を検討される方にとって、参考になるかと思います。

■導入までの軌跡
　AWSを導入するまで導入していた弊社の導入サーバーの変遷をご紹介します。

■サービスに合わせた最小構成
　WordPressを使ったオウンドメディアやSE能力診断などの自社サービスを支えるAWSの構成や
　構成を決定するにあたり、気をつけたことをお話しします。

■１年間運用して苦労した点や費用
　１年間の運用してきて苦労した点や運用費用について、余すことなく公開します。


※ 内容は変更になる可能性があります。


★講演者プロフィール

■株式会社AMG Solution
山口　博史　様

情報システム系の専門学校を卒業後、

2009年にAMGの創業メンバーとして参加し、
新電力システムの外部接続機能やOpenADR認証の取得を担当。
