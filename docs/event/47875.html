初音ミク生誕６周年、且つミク記念日3月9日ということで、
オンラインでイベント開催してみたいと思います！

名付けて、「ボカロonlineオフ会！」

3月9日　matsuridadiva開始と同時刻より、Skypeアカウント「on_off_party（オフ会online）」でグループ作って開始します！

ボカロ好きならどなたでも！
聞き専さん、見る専さん、歌い手さん、踊り手さん、MMDerさん、Pの方々、調教師さん、等々、やってることカンケーなく、Skypeで集まって、みんなで同期して自部屋飲みしませんか？
（※主催者は過去にも「onlineオフ会」開いてそこそこ幸せな場にはできたかなという体験があります。）

諸事情でボカロ系のオフ会になかなか出られず涙を飲んできた方々、
matsurida divaのチケット取れなかった侘しさを癒したい方々、
ボカロ異業種交流に興味がある方（MMDerと歌い手では層が違うらしいから、とかとか）
皆で集い、盛大に馴れあいましょう！
あ、あと、年齢カンケーなく。主催者は超オサーンです。
（先日、某ボカラオフ会に行ったら若い人たちばかりで浮いてたなぁ…）

Skypeで語り合いつつ、ニコ生でボカロ関連動画を流したり、ときにはカラオケ歌ったりもできるようにする予定。

参加費：無料（当然）

ちなみに主催者は、超底辺MMDerで、最近は我が身でボカロ曲でヲタ芸打ったりしてる者です。近々ぜひ呪音キクのコスで曲で踊ってみようと企んでます。
ニコ動アカ：gekitora

人数はSkype定員の20名以内と思ってます。
マンマンがイチ、それ以上集まられたら、複数PC、複数アカウントで対応しようと思います。

とかって、・・・盛大な　ぼっち　にならないといいなぁ・・・
(´・ω・｀)
