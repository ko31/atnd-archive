description

『お盆』には、自分の行く先が本当に正しいのか、強烈に考えさせられる深いメッセージが込められています。

お盆といえば、忙しい人でも休みをとって家族と過ごすことが多いですよね。
墓参りなどを考えると、仏教に関係しているのかな、とは思いますが、実態はよくわからないのが現状ではないでしょうか。
"とにかく夏休みがとれるからありがたいなぁ"くらいに思っている人がほとんどだと思います。
せっかく休みをとるなら、お盆の由来を学び、心を落ち着けて自分の姿を見つめたいものです。

仏教に関する行事は、お盆に限らず、案外たくさん身の回りにあります。
葬式、法事、読経などは、どんな意味があり、自分とどう関係しているのか。
この機会に学んでみませんか？

◆講座内容◆
・お盆って何？
・お経ってどんなことが書かれてあるの？
・葬式や法事ってどんな意味があるの？
など

◆参加者の感想◆

～ここに来ないと聞けない話がたくさん～ 
　ここに来ないと聞けない話が盛り沢山でした。
　”本当の私”が何なのか。もっと知りたくなりました。
　自分の心に振り回されて生きているなと思いました。
　これから勉強させて頂きます。(30代・女性)

～聞かねば一生わからない“人生の目的”を知れた～

　学校の勉強でも、教えてもらったからこそ、
　今知っているということが多いです。
　仏教で教えられている「人生の目的」も、
　聞けば知れることでも、聞かねば一生分からないことだと感じました。
　続けて聞いていきます。（30代・男性）
