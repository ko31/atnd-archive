description

だれの人生にもアップダウンがあります。

何をやってもうまくいかない時期、
努力が実って何とか結果に結び付く時期,
やること全てが思った以上に上手くいく時期、

などなど、いろいろな時期がありますが、

あなたのこれまでの人生はどうでしたか？
思い通りの人生を歩んでこられたでしょうか？

実は、あなたの人生のアップダウンには、深い意味が隠されています。

そして、人生にはいくつかのスケジュールパターンがあるのをご存知でしたか？
病気・仕事・人間関係・etc

こんな人生を選んだつもりはないけれど…そう言う方が多くいます。
誰もが知らず知らずのうちに、人生スケジュールパターンのどれかを選んでいます。

セミナーでは、心理カウンセラー岩本和也が
あなたの人生アップダウンの意味と知らずに選んでいた人生スケジュールを解き明かします。

過去、現在の人生に不満を抱いていても大丈夫です。

これさえ知れば、きっとあなたの人生は必ず良くなるという
方法もお教えします。

興味のある方はぜひお越しください。
