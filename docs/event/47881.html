イベント内容

マイクロソフト MVP 澤田賢也さんをお招きして、Windows Server 2012 R2 Essentials の勉強会を行ないます。
今回のテーマは、「R2を使ってみよう」です。

前回まで、３回にわたって、SDKの使い方を勉強しました。
そして、最終的に、Essentialsのアドインなどを作成して、かなり奥深い領域まで踏み込んでいきましたが、今回は、一般的なテーマにもどります。

昨年発表された最新バージョンのR2の使い方を勉強していきます。

R2からは、StandardエディションにもEssentialsの機能が増えていたりして、面白いことができそうな、でも混乱してしまいそうな、そんな興味深い製品に仕上がっています。

そこで、さあ、一緒に勉強しましょう。
