【病気や偏見を気にせずに活動できる毎日が手に入ります】

病気や偏見を気にせず、自分らしく行動したいあなたへ

こんな事でお悩みではありませんか？

・病気等の偏見が原因で周りの方から理解されない。
・就職や仕事がうまく進まない。
・一定の配慮すら手に入らない。
・生活にもう少しゆとりが欲しい

もしこんな状況が続いたらどうなるでしょうか。

仕事が無駄に大変になるし、
理解が得られない事が原因で無理して働くことになり、
症状の悪化を招くかもしれません。

今回の講座に参加すると、次のような結果が手に入ります。

　・難病の事を隠さず、行動できます
　・自分の事を理解してもらえるようになります
　・病気、障害の事を理解してもらえて、人間関係が良くなります
　・あなたが扱われたいように周りが扱ってくれます
　・自分の本当に欲しい結果が見つかります

やりたい事があるのに、行動できない。
周りの目が気になる。

そんな状況の改善は、私にお任せください。

参加したい方は
Facebookのイベントページよりお申し込みください。
