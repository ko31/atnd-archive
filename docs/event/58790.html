description

見出しは28文字以内で！ツカミの3行で決めろ！
体言止で文章にリズムをつけろ！

この2行にピンと来なかった、そこのあなた！
そそ、そこで鼻ほじってる人！ そこのあなた！

きっと参加するだけで、たぶん当日深夜からグッと伝わる文章が書けるようになります。
お連れさんも少しだけ偉くなった気がします。

まずはブログに載せる写真のかっちょいい撮り方からはじめて、文章を書く前に準備しておきたい妄想、文章の完成度をあげるためのノウハウ、雑誌ライターデビューの裏技など。

なんだかんだと、Webライティング入門から出版業界の裏話まで、編集者として新人ライターに語っていることをそのまま講義形式でお送りします。

【当日のブログラム】

入門編とうたっておりますが、意外と実践的です。

  • ブログに載せるかっちょいい写真の撮り方
  • ブログの選び方
  • 書き続ければ、プロになれる
  • 人気ブログをはじめる前に準備しておきたいこと
  • 誰に向けて書くのか（ターゲット設定の重要性）
  • 書くことを続けるコツとネタ探し
  • サンプル原稿を題材に、文章の完成度をあげるシミュレーション
  • Webライターなら誰でもなれる
  • ライターのギャラ相場公開
  • 出版社や雑誌編集部に売り込むときの裏技
  • いまなら「電子書籍」がオススメ
