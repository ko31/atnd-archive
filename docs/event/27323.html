description

関西・関東では20年以上の伝統がある屋外ジンギスカンパーティー「J2」を名古屋近郊で「J2東海」として復活させます。これは、参加者全員で作るパーティーで、自主的な協力をお願いしています。

  • 日時: 2012年5月13日(日)→幹事の都合により延期します。日程が決まり次第、このATNDを再利用して募集します
  • 時間: 10時30分ごろ〜15時ごろ。乾杯は正午ごろの予定。場所取り担当者は9時ごろ集合
  • 場所: 新しい場所を求めて検討中
  • 会費: 大人2000円、中学生以上の未成年1000円、小学生以下無料
  • 持ち物: 使い捨てでない食器 (コップ、耐熱の皿、箸など)、会費、「J2スピリット」、その他

J2 東海の公式サイトは http://tokai.j2.org/ です。今回は、持ってくるものの調整などは ATND サイトでやることにします。
