ハッカソンについてのご注意

今回はハッカソンですので各自PCに下記の環境を整えてご参加ください。

  • Windows VistaまたはWindows 7/Windows 8が稼動している事（Win7/Win8推奨)
  • Windows Phone SDK 7.1.1を事前インストールして動作確認済であること
    • Windows Phone SDK 7.1.1は下記URLよりダウンロードできる無償開発環境です
    • Windows Phone SDK 7.1
    • Windows Phone SDK 7.1→7.1.1アップデート
  • App Hubの開発者登録が完了している事
    • App Hubが未登録で参加をご希望の方は事前にお問い合わせください。
  • ペンタブレットやフルキーボード、みなさんがアプリ開発に必要なものをお持ちください。
  • 飲み物、食べ物の持ち込みOKです。手の汚れないお菓子の持ち込みも大歓迎です。
    • ネコがいますので飲み物はペットボトルなど口のしまるタイプをお勧めします。
  • 電源および無線LANは会場にご用意しております。できれば電源タップをお持ちいただけると助かります。
