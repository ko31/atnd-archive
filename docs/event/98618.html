description

こんにちは。田中総一郎です。

このセミナーでは、ブログを使って「あなたの理想のお客様」に興味を持ってもらうための仕組みづくりを学ぶことができます。

理想のお客様が増えてくると、好きな仕事でやりがいを感じることができ、収入も安定していきます。

初心者向けなので、お気軽にご参加ください。初めての方も大歓迎です！

こちらの職業の方に向けた内容です。

・パーソナルトレーナー
・インストラクター
・女性起業家コンサルタント
・経営コンサルタント
・漢方薬局
・物品販売業
・イメージコンサルタント
・結婚相談所
・教室のレッスン
・先生業の方　など

では、なぜ理想のお客様にアピールする必要があるのでしょうか？

例えば、無料にしか興味のないお客様や続かないお客様を集めてもなかな定着してくれません。

お客様が定着しないと、収入が常に不安定です。

さらに、お客様がやめしまうと悲しいですよね。

「また生徒さんがやめてしまって・・・」と落ち込んでしまう方もいます。

そうならないために、ブログを作り込んで、あなたの理想のお客様にアピールする仕組みを作ることができます。

お客様がすぐにやめてしまう原因の１つは、はじめから長く続く理想のお客様を集められていないことにあります。

興味がある人なら誰にでも来てもらおうとすると、結果として続かないお客様が増え、ストレスを感じてしまいます。

だから、はじめから長く続く可能性が高いお客様に興味を持ってもらえるかが重要なポイントです。

続かない方をどうやって続けさせようと考えるのではなく、はじめから続く可能性が高い方に興味を持ってもらうことができます。

では、セミナーに参加した方のお声をご紹介します。

私のアメブロが日記ブログだったのがわかりました。

まったく使い方が違うのですね。

私の先生のやり方をマネしていましたので、日々のことを書くのがブログだと思っていました。

こうやって使うと確かに理想のお客様にアピールできますね。

まずは、ヘッダー画像を作るところからはじめてみます。

ブログからお客様が来るようになりました。

でも、体験だったり、料金の安いレッスンにしか集めることができず、やめてしまう人が多かったです。

続けてもらうために必死に努力もしてきました。

田中さんのブログを読んで、もしかしたらやり方が間違っているのかもと思いセミナーの参加を決めました。

セミナーに来てよかったです。

同じようなやり方でも、ちょっと角度が変わるだけで興味を持ってもらえるお客様の層が違うのだとわかりました。

すぐやめてしまう方ではなく、長く続けてくれる意識の高い方に向けてアピールするためのブログ作りに力をいれます。

会社員をしながら、将来自分でやりたいと考え参加しました。

名前を出さずに準備する方法や、何をすべきかが分かりました。

会社を辞める１年、２年前から準備をしていた方の事例を見れたので参考になりました。

私も準備をはじめます。

女性の方の事例が多かったのセミナーにきました。

いろいろな方のやり方も教えていただけたので参考になりました。

好きな仕事で収入を安定させるというのが私の今の目標です。

そのためには、やることをやらねば、と気が引き締まる思いでした。

やりたいという気持だけで、実際に行動に起こしたり、仕組みを作ったりができていないと反省しました。

私も同じようにやってみます。

何からやればいいのかわからないので受講しました。

１つずつ順序を教えていただいたので、まずは、そのとおりに１つずつコツコツやってみます。

積み上げるように仕組みを作るというのが印象に残っています。

ここ１年半ぐらいブログを頑張ってきましたが、振り返ると何も残っていません。

ただなんとなく日々のことを書いているだけでした。　

主婦という立場ですが、空いている時間を使って、少しずつ頑張ります。

いかがですか？

アメブロをうまくカスタマイズして使うことで、理想のお客様にアピールするための仕組みを作ることができます。

ノウハウを上手に活用した方は、こんな結果が出ています。

・レッスンが満席になった
・仕事の依頼がアメブロから来た
・講師のお仕事につながった
・毎月１０万円はブログから売上になっている
・出版につながった
・生徒さんが増えはじめた

仕組みができあがると、長期的に続く理想のお客様が増え、収入も安定し、やりがいを感じることができます。

例えば、魚釣りで考えると、小さな魚を釣るためではなく、大きな魚を釣るために仕組みを作ります。

大きな魚を釣るためには、それなりの準備が必要ですよね。

ブログも同じで、そのまま記事を書いているだけでは、理想のお客様に興味を持ってもらえません。

理想のお客様は、ほんとうにいい人は誰なのか、自分に会う人は誰なのかを真剣に探しています。

この方に価値を感じてもらうために、ブログを作り込んでいきます。

では、どんなことをすればいいのでしょうか？

セミナーでは、ブログを使ってあなたの理想のお客様に興味を持ってもらうための一連の流れを学ぶことができます。

１、理想のお客様は誰か？

２、どうやって価値を伝えやすくするか？

３、記事はどのように書くのか？

４、値段をどのようにアピールするか？

５、お申し込みのページをどうやって構成するか？

このような流れで、具体的に一連の流れを学ぶことができます。

あとは、１つずつ同じように設定をしていくだけです。

日々の出来事を書く日記ブログとはまったく違うやり方です。

アメブロの仕組みをカスタマイズして、理想のお客様にアピールするためのツールを作りあげます。

それでは、仕組みを作りはじめた方のお声をご紹介します。

田中様

お世話になっております。

月曜日の体験から、またプラスの流れが次々に起こってます。

お教室も3名の方が、続けて予約入れてくれました。

お客様の層が変化しつつあるのを感じます。

11月の予約まで入れてくれました。

キャンセルが少しずつ減ってきました。

以前は、開催日直前になって、キャンセルの連絡が多くありました。

最近は、生徒さんのモチベーションが上がってきたように感じます。

ブログのカスタマイズ有難うございました。

カスタマイズ前は、まだもう少し仕事が動き出してからでも良いのでは、と考えたこともありました。

ですが、やはりせっかく書いた記事が一人でも多くの方々の目にとまるようにするには、プロらしくブログの見た目を整えることも大切だと思い、お願いすることにしました。

色や光の調整なども、何度も私のリクエストに真摯に応えていただきまして、感謝の気持ちでいっぱいです。

お陰様で、読者になってくださった方からも、「素敵なブログですね。」とお褒めいただくことがしばしばあります。

メニュー変更やホームページとのリンクなど、今後もマイナーチェンジをお願いすることもあると思いますが、これからもよろしくお願いいたします。

私は、顔もフルネームも出せない、コンセプトカラーやメニューも決まっていない、という、かなり曖昧な状況でカスタマイズをお願いしました。

しかも私は群馬県で仕事をしていますから、対面での打ち合わせが難しいという不安もありました。

それでもこうしてカタチになったのは、スカイプやメールでの打ち合わせに、田中さんが根気強くお付き合いくださり、なおかつ、時間帯・曜日関係なく、迅速にご対応くださったおかげです。

また、カスタマイズ完了後も、使い方が分からない箇所の動画による説明も、とてもわかりやすくて助かっています。

やっぱり、専門のプロの方にお願いして、大正解でした！

これから、このブログをしっかりと育てつつ、今年10月にはホームページの作成をお願いしたいと考えています。

そして、ブログの育て方やブラッシュアップでも、ぜひお力をお借りしたいと思います。

今後とも、どうぞよろしくお願い申し上げます。

いかがですか？

あなたも、セミナーに参加することで、好きな仕事を軌道にのせるためのアメブロのノウハウを手に入れることができますよ。

一連の流れを知ることで、ブログをどのように使いこなせばいいかが見えてきます。

準備をしかりとして、仕組みができあがれば、好きな仕事で収入を安定させるためのツールを持つことができます。

仕組みを作る理由は、はじめから大きな魚を釣る道具を手にいれるためです。

いい道具を持っていなければ、大きな魚を釣ることができません。

まずは、理想のお客様にアピールするための道具作りからスタートです！
