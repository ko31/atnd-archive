description

ネット上では、TwitterやFacebookでアピールしてるのに反応がない。

リアルの場では、次につながる人脈が築けない。

とお悩みではありませんか？

そういう方に満足していただけるワークショップ・セミナーがあります！

そのまえに、

ネット＆リアルの営業で紹介を受けている人の共通点とは？

リアルの世界では「見た目や特徴的な名刺」が大事？

ネットの世界では「Twitterやメルマガ」が有効？

このようなことを本やセミナーでたくさん学びやってみましたが、なぜか、リアル＆ネットどちらも成果が出ませんでした。あるとき、「営業がうまい人、人から紹介してもらえる人」と「そうでない人」を比較してみました。

そこで発見したことは、彼らはテクニックだけで営業しているのではないということです。

その共通点はお客さんにとって解りやすく、差別化された価値を持っている人。

自分の価値がお客さんに伝わって初めて次につながるのだと確信しました。

このセミナーでは・・・

  1. 差別化されたあなただけの価値を発見すること
  2. リアル＆ネットで営業に使えるプロフィールを作る方法

を学び、実際にプロフィールを作るワークを行います。

巷で人気の集客テクニックなどはお伝えしません。とにかくテクニックでやっていきたいと思っている方にはご満足いただけません。

表面的なテクニック系のセミナーは実に多いです。主催者にとっては稼ぎやすいからです。

だからこそ、このワークでテクニックの前に重要な伝えるべき自分の価値を整理し、成果に結びつけていただきたいと思っております。

営業・集客では、まず最初に「自分を知ってもらう」ことが重要です。その手段として、「ネットを利用する」「特徴的な名刺を作る」という手段を使っている人は多いです。

しかしそれは手段に過ぎず、それだけでは営業・集客はうまくいきません。

ただ覚えてもらうのではなく、お客さんにとっての「あなたの価値」が理解されて初めて次につながります。

このワークショップでは独自の自己分析によって明確に差別化されたプロフィールの作り方を学習し、作成します。

    • ポジショニングをするとお願いされる人になる？
    • おもしろいほど出てくる自分の強みを探す方法
    • ポジショニングマップを使うと○○先生になれる

ワークショップ　1時間30分

  • 理由×強み×機会＝自分の価値を伝える公式！
  • 他人に紹介してもらうと自分の強みがＵＰする！
  • 忘れられないキャッチーコピーを作る
  • テンプレートを使って営業プロフィールを作る

メッセージ

巷で人気のSEO対策やfacebookなどは戦略があってこそ効果が出てきます。

異業種交流会でたくさん名刺交換をしているのに印象に残らない人は多いです。逆に、少ししか名刺交換しないのに印象深く仕事につながりやすい人もいます。

この違いは、その人がどう見られ、何ができるのかがはっきりしていること。つまり、ポジショニングがしっかりとできていることです。

このワークで行うポジショニングが明確であれば見込み客に覚えてもらえる確率は格段に上がります。

Twitterやfacebook、異業種交流会で知り合いを増やす前に自分自身のポジショニングすることをおすすめします。

追伸
