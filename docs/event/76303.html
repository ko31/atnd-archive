Photon勉強会、第3部！

1時間でPlugin開発がわかります！

『FINAL FANTASY GRAND MASTERS』『リトルノア』『ファミスタドリームマッチ』など、大型スマホゲームで多数採用されている、Unityとの相性も抜群のネットワークエンジンである「Photon」。

今回の勉強会では、Photonの新機能であるPluginについて、その仕組みと開発方法を伝授します。カスタマイズの敷居が大きく下がり、手軽にサーバーサイドのロジック組み込みが可能になります。

また大きな可能性を秘めているEnterprise Cloudについても詳細にお話していきます。Enterprise CloudではPluginの利用も可能となっており、クラウドサービスで手軽に運用をしながらPluginでサーバーサイドのロジックカスタマイズも可能です。Photon Enterprise Cloud + Pluginの可能性をぜひ感じてください。

セッション終了後、参加者同士やPhotonのエンジニア、GMOアプリクラウドのスタッフと直接お話できるネットワーキングタイムも設けておりますので、ぜひこの時間もご活用ください。
