概要

　Ruby on Rails 4.0 リリースまでもう少し、仕事ですぐ使うわけじゃないけど、
　軽く見ときたい、だから勉強会に参加したいと思ったけど、Rails 勉強会が全然ねぇ！！
　 東京 ruby 会議 は、ハードルが高いし、開催数も少ない。
　最近は初心者～中級者向けも少ないし、だったら自分たちで勉強会開けば良いんだよ。
　と、考えたが吉日さっそく開催しちゃいます。

弊社の技術力向上のためにもという思惑もありますが、初心者・中級者の方ぜひぜひご参加ください。

　決して上司から命令されてとかじゃないし、参加料無料なので安心してください。
　怪しいツボは売ったりしませんw

参加対象の方

ruby もプログラミング経験も無いけど、アプリ作ってみたい方
プログラミング経験あるけど ruby2.0/rails4.0 勉強しようかなと思っている方
もちろん中級以上の方も歓迎です、ruby の素晴らしさを語らいましょう
