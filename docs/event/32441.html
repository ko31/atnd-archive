第6回WordPressお茶会 WordBench香川を 2012/10/06（土）13:00～ に開催したいと思います。
場所は前回同様「デザインラボラトリー蒼」様となります。

概要

  • テーマ作成ハンズオン（総参加者人数少ない場合トーク班が寂しいのでみんなでトークになるかもしれません）
    第5回で要望がありましたのでWordPressテーマ作成ハンズオンを予定しています。
    1対複数人だと土井のキャパオーバーになるので定員は1、2名程度。
    静的HTMLをWordPressテーマ化といったありふれた内容を検討中ですがその場の思い付きでの進行になると思われます。
    WordPress 3.x (速習デザイン) のサンプルデータをもとに制作していきます。
    @jim0912 さんサンプルデータ使用許可頂きありがとうございます。
    ハンズオン参加希望の方はTwitterにて @show_web までご連絡ください。

　※事前にローカル環境構築もしくは専用ツールにてWordPressが動く環境をご用意下さい。
　・ローカル環境構築 : 第1回WordPressお茶会時のローカル環境構築記事
　・Windows用Webサイト構築ツール : Microsoft WebMatrix
　・Mac用Webサイト構築ツール : BitNami

  • WordPressに限らずその他CMS・HTML5・CSS3等最近のWeb周りに関しての雑談なり意見交換
    その他の方はいつも通りだらだら進行で行いたいと思います。
  • ライトニングトーク
    「jQuery+HTMLで作ったフロントエンドにBackboneを導入」 @_nobinobiru さん
    「逆に考えるんだ！セミナーイベント規模を小さく小さくして個人レッスンに近いかたちにするといいよ」 @styledesign さん

　発表したい方がいらっしゃいましたら大歓迎です。下記お問合わせよりご連絡下さい。
