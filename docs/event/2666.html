本イベントは、ケツダンポトフの@ksoranoさんの協力により、Ustreamによる「ダダ漏れ」（動画生中継）を実施します。

本イベントでは、会場内の写真・動画の撮影を原則として許可し、それらのWeb上での公開を推奨します。撮影されることを希望しない方を当日イベント開始時に確認しますが、途中参加の場合などにチェック漏れが起こりえますので、事前に意思表示可能な場合には、この掲示板か「#nsmc」をつけたTweetでお知らせください。

本イベントは、Poken Meetupとして登録されましたので、Poken用の交換電池10個がアコニ様より提供されます。Pokenの電池が切れた方は当日無料で交換することができます。

Niigata Social Media Club / 新潟ソーシャルメディアクラブは、新しいウェブ事情に敏感な人々をつなげる、新潟のプラットフォームを目指します。昨年のオンライン上でのスタート以降、各種SNSを通じて、緩やかに人々をつないできました。また、独自のイベントとして、新潟県内で4度のフォトウォークを実施しました。約10カ月の準備期間を経て「Niigata Social Media Club / 新潟ソーシャルメディアクラブ」を冠したイベントを、パーティ形式で交流できる形式でスタートすることにしました。新潟のウェブに関心のある人が気軽に集まり、楽しめる場所になってくれたらと思います。

LT希望者募集：LT 希望者募集：上のテーマで5分間のLightning Talkを行っていただける方を数名募集します。第一回は募集範囲を少し広めに設定し、１）新潟県内その他でウェブ関連のイベント、勉強会、SNSなどを実施・運営されている方、２）企業やウェブサイト等の「公式」Twitterを運営されている方からのLT希望を募集します。1月7日までにご応募ください（希望者が多い場合には、早めに締め切る場合があります。また、主催側の判断によりご希望に添えない場合もあることをご了承ください。）
