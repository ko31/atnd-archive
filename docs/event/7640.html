□ 1st Session [ CIとブランドソリューション ]
───────────────────────────────────
◆ クワダ ヤスヒサ (くわだ やすひさ)
yasuhisa kuwada design ofc

http:// そのうちやる。
Twitter : やるひまない。

【1.自己紹介】
デザイナーという枠に縛られずに好きなように、
毎日たのしく仕事をさせていただいております。
故にわがままですいません。


【2.経　歴】
落書きの大好きな、くそガキがオトナになっても、
まだ、落書きが好きだったので、広告の仕事を志す。

大阪デザイナー専門学校卒業(たのしかった)

その後、人生の紆余曲折を経て、フリーのアーティストとして活動再開。
世界中を飛び回り、大活躍する。のは、まだ先のお話


【3.セッション内容】
デザインコンサルティングビジネスにおいて気をつけること
ニーズの掘り出し、デメリットをメリットに変える
ヒアリングの目的、VIの作り方 (オリジナルがどう生まれていくのか)
センスの中身 (テクニックとアレンジ)
精神論 (一目惚れ理論)

□ 2nd Session [ ロゴって大切 ]
───────────────────────────────────
◆ 高橋 伸幸 (たかはし のぶゆき)
ナッツ・デザインスタジオ

http://www.nats-ds.jp/
Twitter : none


【1.自己紹介】
福山市を拠点にデザイン制作活動をしています。
プロダクト・グラフィックを軸に、幅広い分野のデザインワークをカバーして
います。

【2.経　歴】
広島県出身
‘88　大阪総合デザイン専門学校　インテリアデザイナ−学科卒業
’88〜’97　(株)ハーズ実験デザイン研究所
’98〜 　ナッツデザインスタジオ設立

【3.セッション内容】
基本的なロゴマークの考え方
デザインに展開の仕方
主に基本的な部分に重点をおいたセッションです。
