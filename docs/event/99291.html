テーマは
・参加型
・一緒に創っていきましょう
そうすればスタートアップの方はお互いの事業発展に繋がり
仲間もできるよね♪

という考え方のMeetUp会となります！

The First MeetUpでは、起業/独立に必要な4つのプチワークショップ付き
ネットワーキングを行い共創に繋げるというテーマで開催しております。

＜プチワークショップ内容＞
A：セールス力アップ
B：企画力アップ
C：事業計画力アップ
D：自分の軸を創る
※日程によりA～Dのワークショップ内容が違います

また、毎回ではありませんが「創業1〜3年目で活躍されているあなたとの距離が近い経営者」をゲストにお呼びして話を聴くこともあります。

こういった「リアルな起業話」を聴けることで、活動の幅が広がると考えています。
セミナー前後には交流の時間も設けておりますので、積極的に新しい方との繋がりを創ってください。

みなさまのご参加を心よりお待ちしておりますm(_ _)m

株式会社トゥエルブ（Twelve inc.）
代表取締役 近藤圭介

1987年1月31日生まれ。
大学卒業後、大手印刷会社に営業として入社。
26歳で独立するが、最高月収8万円という苦しい状態が半年続く。

この状況を何とかしようと受講した
「世界No.1営業のセールスセッション」や
「ハリウッドスターを顧客に持つ女性社長との出会い」
により一気にセールスが上手くなる。

3ヶ月で売上1億2000万円、利益1000万円等の実績を出し株式会社トゥエルブを設立。

9年間3000人以上との営業経験を基に作成したオリジナルのコンテンツである
【右脳セールステクニック100】や【売れるサービス創り8つのポイント】などを
提供しクライアントのサポートをしている。

ラジオや雑誌出演、計200回以上のセミナー講演も経験。
