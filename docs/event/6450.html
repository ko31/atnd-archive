description

兵庫県の丹波篠山の田舎町で小学校を使ってキャンドルナイトを開催いたします。

幻想的な旧小学校跡地で田舎町の子供たちと一緒にキャンドルを並べます。京阪神から車で1時間ほど。

鹿やタヌキ。イノシシも出てくるかも。

街中にはない特別な場所“雲部”。近くには古墳や篠山チルドレンズミュージアムもあり,多くの子供たちに特別な日を一緒に過ごしてほしいと考えております

篠山市街地をまっすぐ東へ抜けていきます。雲部の夏祭り会場にてやっております。お楽しみに！
