MySQLのバージョンアップ計画を立てませんか？
（好評につき増席を計画中です）

MySQLのサポート期間は ライフタイムサポートポリシー に従って定義されていますが、MySQL 5.6のPremier Supportは2018年2月に終了します。Extended SupportやSustaining Supportは引き続き受けられますが、この機会にバージョンアップを検討してみてはいかがでしょうか？
（MySQL 5.5のExtended Supportは2018年12月に終了します）

本セミナーでは、MySQLバージョンアップの手順、注意事項に加え、最新バージョンのMySQL 5.7、現在開発中でリリース候補版となっているMySQL 8.0の最新情報を合わせてお伝えします。

参考資料： Oracle Lifetime Support Policy: Technology Products 　※"MySQL"で検索

講演概要

MySQLサポート期間のご案内

  • MySQL Standard Edition/Enterprise Editionのサポート期間はオラクルのライフタイム・サポート・ポリシーに従って定義されています。本セッションでは、このライフタイム・サポート・ポリシーとサポートが受けられる商用版のMySQLであるMySQL Standard Edition/Enterprise Editionについてご説明します。

MySQL 5.7 新機能

  • 現在の最新バージョンであるMySQL 5.7の新機能についてご説明します。

MySQL 8.0 RC2 開発最新動向

  • 現在開発中でRC(リリース候補版)となっているMySQL 8.0の開発最新動向をお伝えします。

MySQLバージョンアップの基礎知識

  • MySQLをバージョンアップする際に気を付けた方がいいポイントやバージョンアップ手法などについてご説明します。
