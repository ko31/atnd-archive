第２回 仁福の会　谷町４丁目

谷町４丁目にあるシナジーのオフィスで、第１回目となる落語会を開催しましたが、生憎、大阪地震で取りやめ…とはせず、がんばって開催しました。笑福亭仁福さん、露の吉次さん、笑福亭大智さんも交通機関が混乱する中、来て頂き、演じてくださいました。 今度は第２回となるオフィスで落語、仁福の会　谷町４丁目を開催します。 出演者は笑福亭仁福師匠、笑福亭仁嬌師匠、笑福亭智丸さんの三人です。

月曜日夜、谷町落語で大笑い

    2. 開催場所：㈱シナジー 〒540-0028　大阪市中央区常盤町1-1-4谷町細見ビル3F

  • 料金 当日：１５００円、前申し込み：１０００円

出演者

  • 笑福亭　仁福：笑福亭仁鶴二番弟子。ほのぼの天然な噺家で豪快な爆笑落語を演じる。関西各所で地域寄席を開く、底知れぬ人気者。噺家はもちろん芸人仲間の誰からも愛される存在。

  • 桂　坊枝：五代目桂文枝入門。上方落語協会協　 　会誌「んなあほな」第二代編集長。「上方らくごカルテット」メンバー。阿波踊り「はなしか連」“宴会部長”。

  • 笑福亭 智丸：ふんわり丸っこいところから芸名は智丸。古典落語、新作落語の発想を大事に、古典と新作の双方でお客さんに喜んでいただける噺家を目指す。
