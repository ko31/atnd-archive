梅田ミツバチプロジェクトで採れたハチミツを試食する会です。


※参加登録は不要。開始時刻にピエロハーバーへお越しください。
※17時より開始（ゴリモンは16時くらいから現地にいます）。ハチミツが無くなり次第終了
※参加は無料ですが、ハチミツに合いそうなものを各自オーダーしてください（紅茶とかトーストとかプレーンラッシーとか‥‥かな？）

※プロジェクト公式のイベントではありません。ゴリモンが勝手に企画しているのであしからず^^;



同じ巣箱でも、時期によって採れるハチミツは変化します（蜜源となる花が変わるため）

ゴリモンがサンプルとしていただいているハチミツは3種類
・4/28採蜜分［蜜源は、菜の花や桜？］
・5/10採蜜分［蜜源は、アカシヤやバラ？］
・5/20採蜜分［蜜源は、バラやラベンダー？］


皆で試食会。香りや味の違いを楽しませんか？
実はわたくしゴリモンもまだ封を開けていません。どんな味か楽しみ〜
