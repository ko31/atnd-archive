Pentaho無料体験ハンズオンセミナー 無料体験ハンズオン（入門編）
～2時間でPentahoを体験してみる～

開催日程

2016年11月22日（火）16:00-18:00 （受付15:45～）

開催場所

ベンチャーデスク銀座 5F （東京都中央区銀座6-16-12 丸高ビル）

参加対象者

BI・BA製品を導入予定、利用、比較検討されている企業の方

内容

2時間でPentahoビジネスアナリティクスのレクチャーと、実際のデータの使った分析・レポーティングを体験いただきます。

１．Pentaho製品やサービス概要
２．ビッグデータへのPentahoの取組み
３．ご説明とハンズオン
　1) Pentahoビジネスアナリティクス(BA)
　Pentahoユーザーコンソールからレポートやビジュアライズ機能を体験いただきます。
　2) Pentahoデータ統合(DI)
　Pentahoデータ統合を使用したデータクレンジングやアジャイルBIを体験いただきます。

費用

無料

8名（先着順）
