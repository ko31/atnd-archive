description

皆様のおかげを持ちまして、Latin Bar sonrisaも２周年を迎えます。
あらためまして、ありがとうございます。

２周年を記念して、ここでビッグパーティーを開催、題して
「Latin Salsa Club Sonrisa」.
場所は在大阪ボリビア多民族国家名誉総領事館！
往年のディスコの様な空間で、サルサ聴きながらみんなでパーティーしましょう。

当日は、ラテンミュージックあり、サルサダンスあり、天満の飲食店の出店も….さらに！バーカーウンターを固めるのは、天満・中崎町の有名バーテンダーばかり。しかも、ちょっとしたサルサレッスンも受けれます。

皆様お誘い合わせの上、是非。
パーティーがしたいんだろ？
