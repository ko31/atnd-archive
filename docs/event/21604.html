セミナー内容

本セミナーは、70-659試験合格を目指す方々を対象に開催いたします。
70-659試験では主にHyper-V、System Center Virtual Machine Manager 2008 R2、リモートデスクトップサービス(RDS)について出題されます。
特にHyper-VとSystem Center Virtual Machine Manager 2008 R2が出題の中心となり、インストールや初期設定に関する部分から、System Center Virtual Machine Manager 2008 R2を連携させた奥深い管理の仕組みまで幅広く出題されます。
セミナーでは、「MCP教科書 Hyper-V（試験番号：70-659）Windows Server 2008 R2対応」の書籍を使用して、当書籍の著者が、試験で出題されるポイントを絞って紹介いたします。
