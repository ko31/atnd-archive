description

iOSアプリの開発言語としてリリースされた話題の言語「Swift」に関するイベントを開催します。

本セミナーは注目の言語「Swift」とは何かという説明に加え、既存の言語「Objective-C」とは何が違い、どんなことができるのかについて、詳しくお話します。

講師には株式会社ジブンスタイル代表取締役で「Swift」の書籍「UIKit＆Swiftプログラミング 優れたiPhoneアプリ開発のための UI実装ガイド」著者の斉藤祐輔氏をお呼びします。　

まだあまり情報の出ていない言語ですので、質疑応答も十分な時間を用意しています。 現役のプログラマー、スクリプターなどの参加をお待ちしております。

また、懇親会も開催しますので、この機会に交流を深めてください。
