オープンソースカンファレンス2018 KyotoへBlenderの日本のユーザー会であるBLUG.jpがブースを出展するので、そのついでにBOF的な何か?をやってしまおう…というか、お店番してくれる方を 2～10名程度 募集、ということで。後ふらっと寄ってみて、お店を覗いてみようかなって人も参加登録してってことで。

    • 2-10名程度

    • 269名まで

オープンソースカンファレンス2018 Kyoto

特に深い意味はありませんが、少しの時間でもブースに立ってみたいという方、お気軽にどうぞ。
私の作品を公開したいっという方も、Blenderでのご自身の作品なら得に問題無しです。

BLUG.jpは8月4日(土)の土曜日のみブース出展します。
(オープンソースカンファレンス2018 kyoto は 8月3日(金)、8月4日(土)に開催ですのでご注意ください。土曜のみです)

やること

  • お店番
    • お店番をする( 一瞬でももちろんok)
    • 飴ちゃんをくばる
    • blenderについて説明してみる
  • お客さん
    • お店に訪れる( 一瞬でももちろんok)

    • blenderについて説明してみる

利点

  • blender知ってそうな人、興味ありそうな人と話が出来る(出展側として、会場に来た人と)
  • (ATNDで参加表明することで)盛り上がっている感を演出

注意点?

  • 「少しのぞくだけー」の方も登録しちゃってくださいな。参加はいつの時間でも構いませんので。
  • bofとは言ってもブース前で内輪だけで喋っていることは無いようにね。

ということでよろしくですー。
