イベント ユーザー

title Heroku-ja Meetup #1
catch Herokuユーザーが情報交換し合えるMeetupです
description

Herokuのユーザー同士で気軽に情報交換できるような場が欲しい！ということでMeetupを開くことになりました。Rubyに加えてNode.jsやClojureも利用できるようになったので、「まだHerokuを使ったことはないけど興味はある…」という方の参加も大歓迎です。

今回はセールスフォース・ドットコムの方からHerokuについてお話していただくほか、5〜10分くらいのLTを4〜5枠ほどやる予定です。後半はビール＆ピザで交流しましょう！（受付時に参加費を1,500円いただきます）

※発表してくださる方を絶賛募集中ですので、何かネタのある方はコメント欄にて「発表タイトル」を記入してくださると嬉しいです。
