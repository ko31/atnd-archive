【プログラム内容】

本プログラムは、国内トップクラスの実績*を有するベンチャーキャピタル事務所であるPE&HR株式会社(以下「弊社」といいます)が開く、ベンチャーキャピタリスト養成講座です。弊社の投資方法・育成支援・ファンド運用・資金調達など全てをお見せします。

(*投資を完了した3つのファンドから42社に投資をし、5社の上場、2~74倍で9回の売却、2社が上場準備中)

本プログラムでは、プログラム中からベンチャーキャピタリストとして弊社オフィスにて実際に投資業務を担っていただくため、投資候補先との面談やデューデリジェンス、ファンド組成方法など、プログラムを通じて様々な知見が得られます。

会社員の方が、所属されている会社の就業規則があるために参加できないことが無いよう、講座という形にしています。

投資候補先とのご連絡は、原則弊社オフィスのPCから行っていただきます。
全ての投資交渉に、代表の山本が責任をもって立ち会います。

今回の投資編を修了された方には、成長支援編・エグジット編の開講も検討しています。

【プログラム開始経緯】
弊社では3年前から学生のインターンシップを受け入れており、業務の一環として投資候補先企業の発掘を担ってもらったところ、スポーツアプリ会社、音楽機器会社、グルメアプリ会社、動画メディア会社への投資決定に結びつきました。今では彼らのことをインターン生ではなく、敬意を込めて『学生ベンチャーキャピタリスト』と呼んでいます。
このような実績から、受け入れ先を学生から社会人に広げようと思い立ち、昨年9月に当プログラムを開始しました。

弊社にはスタートアップへの潤沢な投資資金があり、ベンチャー企業の開拓・投資を加速させたいと考えています。プログラム参加者の方々には、ベンチャーキャピタリストとして、投資候補となる有望なベンチャー企業を開拓して頂きたいと思っているため、プログラムの参加に際して料金は一切かかりません。

参加者の経歴は多岐に渡り、経営者として活躍されている方々や事業や経営戦略の責任者を務めるなど専門性を備えた方々など、これまで約20名にご参加いただいており、現在複数の投資案件が具体的に進んでいます。

2017年5月には1期生の方が投資を実行し、毎月の取締役会に出席いただくなど投資先の成長支援も担っていただいております。

※講座参加者間における当日及び後日の、ネットワークビジネス、宗教、その他の営業行為等の勧誘は固くお断りしております。

【講座内容】
・ベンチャーキャピタル事業の本質と全体像
・スタートアップベンチャーの見つけ方と審査の方法
・投資実行、IPOと売却活動
・決算、監査、ファンド組成の方法、組合契約書の概要　　など

第1期では、上記に加え、上場企業経営企画室の元役員で多数のM&A案件を実行された方をお招きしてM&Aについての講義をしていただきました。

第2期では、企業・自治体で事業創出を数多く手がけているインキュベーションの国内第一人者、3社を上場に導いたIPOのプロフェッショナルで、東証一部上場企業のCFOに講義していただきました。

4期・5期では、元投資先の東証一部上場企業の社長と役員の方、資金調達業務や株式上場支援の第一人者の方に講演していただきました。

6期では、より充実した内容をご用意します。

【報酬】

講座を無償で提供することと併せて、ご自身で開拓した企業に投資が行われた場合は1社につき20万円、弊社が提供した案件の審査補助や書類作成を通じ投資が実行された場合は1社につき10万円の報酬をお支払いいたします。
