description

みんなで牡蠣を食べましょう！

猛烈に牡蠣が食べたくなったので、年末に、牡蠣を食べる会を開きたいと思います。
ついったーでフォローしてる方で、あまりお会いしたことない方とも一緒に牡蠣が食べれたら嬉しいな。
皆さん、奮ってご参加ください。

時間

最初考えていた時間よりも30分早くなりました。
1時間50分で次の予約の人が来るそうです。
二次会も考えといた方がいいのかな…。

会場

かき小屋　ジャックポット 祐天寺

その他

炭代　１台　５００円
お通し代　ひとり　３００円
飲み物持ち込み　ひとり　９００円

一人900円を払えば、飲み物の持ち込みがOKなのだそうです。
でも、サワーとか焼酎とかワインとか、普通の飲み物も売っています。
