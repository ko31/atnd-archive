巷でよく聞くようになった、ネットで生放送が出来るサービス「USTREAM（ユーストリーム）」。
単なる思いつきで始めてしまうと、自分の私生活をダラダラ垂れ流すだけの放送になってしまいがち。
そこで、観ている人も一緒に楽しんでもらえる、或いはセールスマーケティングに有効なWeb生放送とは何か、実例を交えてご紹介します。
また、テレビ局の放送機材にも匹敵するMac OS Xのフリーソフト、CamTwistの簡単で効果的な使い方も。
スピーカーにはテレビ局でテロップ制作の最前線に立たれていた四方宏樹さんをお招きして、Ustreamを使った可能性についてお話していただきます。
