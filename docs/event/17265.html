目的：

今年の６月２１日～２３日にかけて、コンピュータビジョンの大きな国際会議の一つ「IEEE Computer Vision and Pattern Recognition (CVPR) 2011」が開催されました。
そこで、今回は「CVPR2011祭り」と題して、普段勉強している「コンピュータビジョン最先端ガイド」から離れて、CVPR2011で発表された内容について勉強したいと思います。

開催場所：

慶応大学矢上キャンパス
http://www.st.keio.ac.jp/access/index.html

発表：20-30分、質疑：10分
