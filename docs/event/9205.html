川上成夫氏

数々の名著の装丁を手がけられ、多くのアワードを受賞されてて
いる装丁家。
真鍋博アトリエにてイラストと装丁を学ぶ。
1968年株式会社スタジオCGSを設立し、装丁、ブックデザインの
仕事に携わる。朝日広告賞、造本装幀コンクール等で受賞。
日本図書設計家協会、JAGDA会員。
http://www.tosho-sekkei.gr.jp/booksfile/member_detail.php?member=A1042

佐藤豊彦

海外スポーツメーカーやMAZDAやFORDなどのWEBサイトのディ
レクションや制作をはじめ、多くのビジネスのブランディング＆
クリエイティブ・ディレクターとして活躍する一方、月刊誌など
の編集統括やアートディレクターなどを担当。
SatoRichmanの名前でイラストレーターとしてもTVCFや雑誌な
どで活動している。
SUPER ILLUSTRATION SCHOOL校長、SUPER PROJECT CEO、
女子美術大学特別講師、AppleやIT情報の番組「Apple CLIP」の
構成やパーソナリティなどを務める。
http://www.superproject.jp/
