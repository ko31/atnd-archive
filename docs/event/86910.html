description

・職場と家の往復の毎日で、新しい出会いがなく、刺激が足りない
・毎日がただ何となく過ぎていき、漠然とした不安を感じる
・最近、自分の方向性がぼんやりしている気がする

こんな方にピッタリ！
世界的大ベストセラー「7つの習慣」を題材にした読書会を開催しています。

読書会というと、みんなで本の読み合わせをしたり、または、本についてディスカッションしたり、そんなイメージをお持ちの方も多いことでしょう。

こちらの読書会では、7つの習慣をあなたの生活に落とし込めるワークを中心に、進めていきます。

ワークでは、あなたにとってプラスになるとわかっていても、一人ではなかなか時間をつくれない、ついつい後回しにしてしまう内容を取りあげていきます。

2012年12月よりスタートし、190回以上、述べ900人以上の方々にご参加いただきました。

最年少は19歳、最年長は75歳。
大学生、会社員、経営者、士業、教員、フリーランス、アーティスト、主婦などなど
年代、職業、ステータスも様々な方々にご参加いただいています。本当に幅が広すぎて、我ながら驚きです！

この読書会は上記のように本当に様々な方がいらしてくださるので、良い刺激をものすごーく得られます！
そして、難しいディスカッションはなく、誰でもできる内容のアウトプットを通じて
自分自身の頭の中を整理整頓できます！
例えて言うなら、必要な本を探すのに時間がかかるぐちゃぐちゃな本棚が、ジャンル別にわかりやすく整理された本棚になる、そんなイメージでしょうか。

ちなみに、この読書会は7つの習慣を読んでいなくても参加OKです。

もちろん何度も読み返しているという方も参加されますが、タイトルは知っているけれど読んだことはない、マンガだけちら見したことがあるという方の参加もとても多いです。

何度も読んでいる方も、まったく読んだことがない方も、楽しく有意義な時間をお過ごしいただいていてますし、有難いことに何度もリピートしてくださる方も多いです。

各回の人数としましては、2名の主催の他に平均10名程度で、約80％の方が初参加です。初めての方も、安心してご参加いただけます。
なお、ご友人やご家族と一緒に参加される方は参加費割引があります。2回目以降の方も割引になります。

刺激いっぱいの読書会に、ぜひお気軽にご参加ください！
主催者一同、笑顔でお待ちしています！
