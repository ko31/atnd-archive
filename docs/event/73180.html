description

いま大流行の人狼ゲームを、水素水を使った、体によく、しかもおいしいコーヒーを召し上がりながら楽しんでいただきます。

今回の会場となるカフェは、日本でここだけと言われる、水素水にこだわったカフェ。
お冷はもちろん、コーヒーや紅茶、そしてお料理に至るまで、あらゆるところに水素が含まれています。

ゲームの方は、基本的な役職（人狼、騎士、占い師）をベースに、各ゲームごとに様々な役職を、１～２程度入れていきますので、いろんなタイプのゲームが楽しめます（※参加人数にもよります）

また、今回は、会の途中の参加、または退場もＯＫです。
「１３時じゃちょっと早いかな、１４時からなら参加できるよ」
「私、１６時から用事があるんだよねえ」
という方も、お気軽にご参加下さい（なお、申し込み時にお申し出下さい）

楽しみながら健康にもなり、そしてもちろん友達までできてしまう。こんな素敵な人狼会は他にはないはず！
皆様奮ってご参加下さい。

日時：2015年12月26日　13時～17時
