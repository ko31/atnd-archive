description

　日本へ一時帰国されている内田さんに縁あって、新宿トラベラーズ・コワーキングにご来訪いただいて勉強会（情報交換会）を開催できることになりました。内田さんは大学在学中でありながら、中国やフィリピンへの留学経験も豊富で、貴重な体験をされています。今後の活躍を期待せずにはいられませんが、アジア方面への正規留学や起業にご興味のある方は、ぜひご参加くださいませ。

■概要

　現在、進行中のフィリピン正規留学ビジネスを立ち上げるに至った留学体験や起業体験を中心に語っていただきます。

　・中国本土（武漢・上海）交換留学
　・香港起業（現地登記。口座開設など）
　・フィリピン正規留学事情（大学・大学院・MBA）～留学ビジネス立ち上げ

■ 日時
２０１２年８月２４日金曜日　午後７－９時
（セミナー入場　午後６時３０分から）

■講師
内田和宏さま（内田集団有限公司 代表取締役）
立命館大学在学中。大学在学中には中国の武漢で半年間の中国語研修、上海交通大学で１年間の経済経営学部での研修を経て中国語を習得。大学３年生が終了した時に２年間の休学を決意。フィリピンの英語学校で3,4ヶ月程の英語研修を経てHoly spirit college of manilaに正規入学。半年後には学校を退学し、タイの現地企業にてインターンシップ生として入社、日本や香港へのネット販売などを担当。その後会社を辞め、今までの生活の場であった中国や東南アジアに対し貢献し活躍したいという気持ちから内田集団有限公司を設立、代表取締役に就任。日本人からの認知度が低かったフィリピン正規大学留学を日本に紹介し、首都マニラの4年制大学、大学院留学、そして博士課程留学などの幅広い留学の形を提供している。

■対象
中国留学、フィリピン正規留学 に興味がある方。
香港での起業に興味のある方。
