＜appC cloud 交流会開催！＞

昨年末に開催いたしましたデベロッパー交流会では、たくさんの方にご参加いただきまして、 誠にありがとうございました。

新年度が始まって1か月経ちましたが皆様いかがお過ごしでしょうか。

ＧＷ直前の4月28日！

第2回デベロッパー交流会を開催する運びとなりました。

デベロッパー様やアプリ制作についてお考えの方々で親交を深めて頂き、 情報交換の場としてたくさんの方にご参加頂けたら、と思って思っております。

お酒を片手に和やかな雰囲気の中、自慢のアプリを披露したり、 開発中の悩みを相談してみたりしませんか？(^_^)

＜弊社スタッフも参加いたします！＞

100万DLのAndroidアプリを開発経験もあり、現在講師として各種セミナーやschoo等で活躍中のエバンジェリストや 弊社の新リリースアプリpeepの開発携わったエンジニアなど 普段は聞けないような話が聞ける機会になるのではないでしょうか

「普段は直接関わることのできないデベロッパー様から貴重なご意見を聞かせていただけるのでは。」 と、弊社スタッフ一同とても楽しみにしております。

皆様のご参加を心よりお待ちしています。 参加方法や開催場所、お問い合わせにつきましては、下記詳細をご確認ください。

＜appC cloud とは？＞

アプリ開発者のための「appC cloud」無料サービスを提供させて頂いております。

「appC cloud」は従来のMbaaSをベースに、 アプリのマネタイズに必要な機能をもった環境を全て無料で提供するクラウドサービスです。

今後とも、より多くの個人デベロッパーの方々に知っていただきたいと考えております。 「appC cloud」https://app-c.net/
