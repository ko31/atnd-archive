セッション概要とスピーカー紹介

1. JavaScriptを使った開発を始めるなら！TypeScriptをはじめよう ～ ステップアップ

もしJavaScriptを扱う開発をこれから新たに始めるのであれば、ぜひ、TypeScriptを習得してください。TypeScriptを使うと、簡単に高品質で扱いやすいJavaScriptのコードを作成することができます。TypeScriptの概要と書籍「TypeScript実践マスター」がカバーする範囲、あるいは本書ではカバーできないその先の内容について紹介します。TypeScriptを新しいプロジェクトに採用できるかどうかイメージしていただけます。

古賀 慎一

アバナード株式会社 マネージャー

Microsoft MVP for Visual Studio and Development Technologies

ALM（アプリケーション・ライフサイクル・マネージメント）と.NET ベースのアプリケーションのデザインおよび構築を専門とする。Visual Studioを使用した開発標準の策定やEVM（Earned Value Management）によるプロジェクトマネジメント、アジャイル開発の推進を担う。

2. TypeScript + Angular で Todo アプリを作ってみよう

これから TypeScript を使って Web アプリケーション開発を始めたい方は、ぜひ Angular を試してみてくだい。Angular の開発は TypeScript で実装するのがデファクトになっているだけあって、VS Code Extension による強力なコード補完や、Angular CLI というコマンドラインツールのおかげで、非常に簡単に TypeScript ＋ Angular のアプリケーション開発環境を構築することができます。簡単な ToDo アプリの作成を通して、TypeScript ＋ Angular の基本的な実装方法を押さえて頂ければと思います。

桐生 達嗣

インフラジスティックス・ジャパン

テクニカルコンサルティングエンジニア

SI企業にてJavaベースWebアプリケーション開発・保守に従事。その後、UI/UX企業にて受託開発および自社サービス開発のWEBフロントエンドエンジニアとして、ビルドツールのセットアップや、フロントエンドアーキテクチャ、CSSアーキテクチャ、Web APIの設計・実装など全般を担当。2015年にインフラジスティックス・ジャパンに入社し、現在は テクニカルコンサルティングエンジニア（Webスペシャリスト）として製品サポートや技術アドバイスの提供を行なっている。

3. Microsoft Bot Framework とTypeScript を使った BOT 開発

Microsoft Bot Framework と LUIS を使用すると、自然言語による対話が可能なインテリジェントな Bot サービスを構築することができます。これらを使用した Bot 開発は Node.js で行うことができ、TypeScript を組み合わせることで堅牢なコードを記述することができます。 このセッションでは、Bot Framework と TypeScript を使用した Bot の開発方法について紹介します。
