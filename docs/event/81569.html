バリューコマース株式会社（本社：東京都港区、代表取締役社長　最高経営責任者：香川 仁、以下 バリューコマース）は、「MA導入前に知っておきたいノウハウを凝縮！MAを「活用する」ための選定ポイントやMAで「効果を出す」ための具体的なシナリオ設計、事例のご紹介」と題した無料セミナ―を2016年9月30日（金）に開催いたします。

マーケティングオートメーション（MA）は、ユーザーの購買意欲や趣味趣向を理解し、それに合わせた接客をメールや広告、SNSなど様々なチャネルを活用して最適なタイミングで自動実行することができます。近年、MAを導入する企業が急増していますが、「MAを活用し、効果につなげること」ができている企業はまだまだ多くありません。

本セミナーでは、MAツール選定中の方、導入を検討している方、導入後の運用イメージが沸いていない方向けに、「活用できるMA」の選定のポイントと、「効果を出すための活用方法、シナリオの設計ノウハウ、活用事例」まで、MAについての具体的な内容をご紹介いたします。

「MAとは？」という基本的な内容から一歩踏み込み、「MAを活用し効果を出すには？」という具体的なノウハウを学んでいただける機会となっております。ぜひ、ご参加ください。
