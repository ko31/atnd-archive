【概要】

　企画書、営業資料、顧客名簿、規程書、仕様書、見積書…スケジュール・タスク・資料化されないノウハウ…
組織の中には様々な種類の情報があふれています。
　溢れ返った情報の上手な蓄積と、共有していく仕組み。それが今、様々な組織、ビジネスにおいて重要な課題のひとつとなっています。

　そこでテラスク365では今季、「情報の整理」というテーマに絞ってセミナーを開催致します。

　12月のセミナーでは、前半にOffice 365の概要と業務に使う「情報」について。
後半はYammerにフォーカスし、社内ソーシャルメディアでであるYammerの特性と
Yammer上で行う情報の整理整頓方法をレクチャーします。

メイン講師として Microsoft MVP for Office 365であり、テラスク365統括部長の“もくだいさん”が
さまざまな不満や不安、疑問をお持ちの方へ向けて優しく、楽しく、しっかりとレクチャー。
また、前半でOffice 365 の概要にも触れるため、「Office 365って何だろう？」「Office 365で何ができるんだろう？」
そんな疑問をお持ちの方にもきちんと学んでいただける内容となっております。
