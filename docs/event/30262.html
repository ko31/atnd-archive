【オンライン座談会】その1：WEBデザイナーの巻

description

Lingrというチャットサービスを使った、オンライン座談会をやってみたいと思います。
以下の項目に該当する方はお気軽にご参加ください。

  • オフ会、オフイベントに行くのがめんどくさい
  • そもそも人に会うのがめんどくさい
  • 名刺を渡すお付き合いがめんどくさい
  • でも人と話してみたい（オンラインに限る）
  • もっとゆるゆるふわふわ話したい
  • 参加費を払うのがいやだ
  • ネカマ技術には自信がある


オンラインイベントなら、なんと移動時間0！交通費0！参加費0！ぼっち0！
電車が遅延しようが、台風がこようが、体調崩そうが、仕事で遅くなろうが問題なし。
オンラインってすばらしい！

Lingr自体はFacebookアカウントでログインすることができますが、もちろん匿名参加でOKです。
※事前にプロフィール設定しておいてください。

今回はとりあえずWEBデザイナーの方に限定させていただきます。

■座談会プログラム

時間 内容 備考
22:00〜22:30 好きなWEBサービスについて語る

・デザインがすき
・機能がすき、なんとなくすき etc…
※ステマ可。アプリOK。
22:30〜23:00 好きじゃないWEBサービスについて語る

・このデザインが気に入らない
・この機能がイライラする etc…
※ステマ可。アプリOK。
23:00〜24:00 これについてみんなに聞きたい！

・自分がつくってるものに対して意見を聞きたい
・いま話題になってる事象に対して議論してみたい
・このデザインってどうやってつくるのか聞きたい

■参加に際して必要なもの

  • Lingrのアカウント（登録はこちらから）
  • 好きなWEBサービス1こ以上　※必須じゃありません
  • 好きじゃないWEBサービス1こ以上　※必須じゃありません
  • なんか聞いてみたいこと　※必須じゃありません
  • オンラインをたのしむこころ

■参加からイベント終了までの流れ

  1. こちらで参加表明をしていただいたら、当日までに参加者に向けてLingrチャットルームURLをお送りします。
  2. 当日、時間になったら適当にチャットルームに集まってください。はや乗り可。
  3. 2時間くらい話す予定ですが、途中離脱・途中参加OKです。
  4. 2時間たったらイベント自体は終了しますが、そのまましゃべっててもらってもかまいません。

※後日、チャット内容の中から公開してよさげなところを編集して、どこかで公開させていただいたりする場合もあります。

■補足

当日しゃべってて、参加者すくなかったらTwitterとかで追加募集する予定です。
もし当日Lingrのサーバが落ちてたら延期か、別の場所考えます。
