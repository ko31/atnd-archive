イベントの概要

夏休みバイナリ集中講座の進め方について議論する会です。

議題

以下のような進め方を考えています。

8/19（日） 13:00～18:00

1. JDKでハローワールドをコンパイル
2. classファイルを分析
3. classファイルを実行するインタプリタを実装

8/26（日） 13:00～18:00

4. JARファイルを作成
5. JARファイルを分析
6. 3で作成したインタプリタにJARサポートを追加

まだ構想段階のため、この会で内容を確定していきます。

資料

  • classファイルの解析
  • classファイルの注釈付きバイナリダンプ

参加費用

  • 無料

会場にはドリンクバーが用意されています。ジュースやコーヒーが飲み放題です。
