description

就職活動の方向性が定まっていますか?
業界ごとの特徴や業界ごとの関連性が解らず悩んでいませんか?

このセミナーでは、小売り業界の歴史から今後の展望までを解りやすく説明いたします。
業界の展望が見えてくることで、いま必要なスキルやいまやらなければいけないことも見えてきます。
今後の皆様の就職活動において絶対に価値のある時間になることは確実ですので是非ご参加くださいませ。

■第一部　全ての業界は繋がっていく
全ての業界が繋がっていく時代、
仕事の細分化やテクノロジーの進化に伴い、業界間の壁はなくなってきています。
時代やライフスタイルの変化が業界にどのような影響を与えたのか、
通信業界でトップを走り続けた弊社の実例をもとにわかりやすく説明致します。

■第二部　パネルディスカッション
接客業やコンサル業など、皆様が業界に対してどのようなイメージを持っていて、
業界同士がどのように影響を与えていくと思うのか、参加者のみなさんで共有していただきます。
普段考えていることをアウトプットしていくことで、
より幅広い視野や知識を得ることができます。
就職活動でお悩みの方には是非ご参加頂くことをおすすめします。

■第三部　今後、業界はどうなっていくのか
今後も全ての業界は、テクノロジーの影響を受け、よりスピーディーに変化していきます。
変化の波をダイレクトに受ける通信業界で培った先見性をもとに
これからの業界の動向と展望について解りやすく説明いたします。
