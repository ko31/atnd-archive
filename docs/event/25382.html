iOS開発者向け勉強会＠仙台 第5弾を開催します！

内容

来たる3月3日（土）に、iOS開発者向けの勉強会を開催します！

いつも若干グダグダ ゆるふわ な進行になりがちですが（スイマセン）、セッション内容はライブコーディングを見つつ話をしたりと、なかなか硬派な内容になっております。

仙台でiOS開発者が集まって話をする機会も貴重かと思いますし、情報交換の時間もご用意しております。是非ご参加ください〜。

対象

  • iOSアプリ開発をやっている方
  • Android/WindowsPhone開発をやったけど、iOSアプリ開発もやってみたい方
  • プログラム経験があり、iOSアプリ開発もやってみたい方
  • プログラム経験はないけど、iOSアプリ開発をやってみたい方（少し厳しいかもしれません・・・）

タイムテーブル

発表内容が揃いましたー。今回は、広告掲出についてとBlocksKitについて解説していただきます！





Session 0 自己紹介・情報交換（雑談 / 世間話 / 井戸端会議） 60 min
Session 1 アプリへの広告掲出について、iAdとAdMobの自動切り替えのサンプルコードの解説を行なっていただきます。by @magickworx_dev さん
こちらからサンプルコードをダウンロードしてください！(AdMobのSDKダウンロードも必要です。下記コメント欄もご参照ください)		 35 min
Session 2 BlocksKitとブロック応用について。h_moriさんのブログでもご紹介されている、BlocksKitについて解説をしていただきます。 by @h_mori さん 60 min
Session 3 仙台産アプリLT大会のご案内・次回の勉強会についてご相談・懇親会のご案内 20 min

参加される方へ

・MacBookなどをお持ちの方は、ご持参下さい。（コーディングを行う時間があるかもしれません）
・会場に無線LANのネット接続環境をご用意いたしますが、念のためemobile等をお持ちの方はご持参下さい。

場所

・今回もデータコム様の会議室を使用させていただきます。いつもありがとうございます。
・今回は通常の土曜日ですので、ビル入り口は通れるようになっているはずです。ビルに入られましたら、エレベーターで6Fにお越しください。

懇親会

終了後には懇親会を考えております（当日、行ける方を募って近場の居酒屋などに行く予定です）。（※学生さんは割引ありにします）
