NFC で始めるイベント出欠管理

ATND などに代表されるイベント管理システムって気軽で便利ですけど、気軽なために「明らかに登録者数と参加者数に差がある……」なんて経験、したことありますよね？　本発表では、そんな問題を解決するシステムを基軸にほんわか NFC とは何ぞやというお話をしたいと思います。
（40分程度 予定）

スピーカー: trebyさん（@treby006） Fukuoka NFC Lab

空のテンプレートから始める Windows ストアアプリ

Windows ストアアプリのテンプレートは便利なんですが、なんだからみんなグリッドデザインになってしまっていませんか？　今回は、Claudia Clock のソースコードを元にグリッドデザインではない Windows ストアアプリの作り方をお話したいと思います。もちろん、テンプレートの使える部分は使って、作りたいものに集中できるアプローチになっています。
（40分程度 予定）

スピーカー: 初音玲さん（@hatsune_） Microsoft MVP / Oracle ACE

Delphi で Win/Mac クロスコンパイル

むかしむかし（2000年前後）至るところで、Delphi が流行っていましたね！　ここ10年で Visual Studio にその座を奪われた感じですが、その Delphi がアツい機能を引っさげ、復活の狼煙をあげようとしています。その１つが Windows/Mac のクロスコンパイルです。デモを見ながら、進化の程を一緒に楽しみましょう。

スピーカー: 大場@tworksさん（@tworks） 新大阪アプリ開発勉強会「楽しいアプリ制作の会」代表

クラウディア窓辺のタダではじめる Windows Azure（入門編）

最新のクラウドサービス「Windows Azure（アジュール）」はもう試してみた？　「ノー」って答えた君のために Windows Azure 概要やオススメの機能を教えるわ。セッション後または懇親会で仲良くなったら名刺交換しましょう！

スピーカー: クラウディア窓辺さん
