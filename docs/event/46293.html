コスプレスタジオBootyにて撮影会を開催します。

2014年3月10日（月）10:00～18:00を予定しています。

こんにちは、主催のさくらだもんです。
この度、まったりと貸しスタジオでコスプレ撮影会をやりたいと思いまして企画立案いたしました。
平日で申し訳ないと思いつつ、スタジオでまったり撮りたい方・土日は仕事な方も
ぜひこの機会にご参加いただけたらなと思います！

※コスプレイヤー・カメラマンで、私と相互フォローの方に限り参加可能です。ご了承ください。
※途中参加・早抜けもOKとしますが、事前にご連絡ください。
※余談ですが、主催はスタジオ・スナップカメラマンをやっています。
　ご希望の方は撮影させていただきますのでお声かけ下さいませ！
※主催が撮影した写真は、後日参加者の皆様にお配りいたしますので、
　自分の写真は入れて欲しくない！という方はご連絡ください。

  • 参加募集は締め切らせて頂きました。

日程に変更が生じた場合は速やかに告知いたします。

用意していただく物（コスプレイヤー・カメラマン共通）

  • 費用
    1. 入場料　一人　3,150円（カメラマンも一律料金です。）
    2. VIPルーム使用料（1部屋4人まで）　一人　790円 ※
      個室のロッカー・メイクルームです。
    3. 撮影ブース貸切料（60分 5,250円）
      利用人数で頭割りとします。
      3F白ホリゾントの貸切を予定しています。
      ※当日申込ですので貸切できない場合もありますがご了承ください。
    4. 貸出機材・グッズ料金
      一覧は下記リンクをご参照ください。
      Item
      こちらは任意で完全に個人持ちとします。
      ちなみに主催がリファーライトを借りる予定です。（リファーは1グループ1つまで）
  • 撮影機材
    自前の撮影機材をお持ちの方は是非お持ちください。
  • 飲み物・食べ物
    飲食可能な場所に制限がございますのでご注意ください。

※VIPルームですが、参加人数が男性女性含め少人数なため、
今回は通常のロッカールーム・メイクルームで着替えとします。

コスプレイヤーさんに用意していただく物

  • 衣装・ウィッグ
    何着でも可。時間内で着替え自由とします。
    露出の多い衣装は必ずインナーを着用するなどして、
    極力肌の露出を控えてください。
  • 小道具
    周りの方の迷惑にならない様、常識の範囲内でお願い致します。
    血糊など、設備を汚損する可能性のある物は持ち込み禁止とします。
    これはいいの？と言う物に関しては主催まで確認をお願い致します。
  • 撮影用の靴
    スタジオ内は土足厳禁（入場時にスリッパに履き替え）ですので、
    必要があれば靴底の綺麗なものをご用意ください。

カメラマンさんに用意していただく物

  • 撮影機材
    自前の機材をお持ちの方は是非お持ちください。
    バッテリー・メディアは各自必ず用意してください。
    貸出機材もございますが、費用は個人持ちでお願いします。
    貸出機材については上記の「Item」リンクをご確認ください。

注意事項

  1. スタジオのルールに従い、周囲の迷惑にならない様行動してください。
    スタジオのルールは以下を参照してください。
    Rule
  2. コスプレのまま来場・退場しないでください。
  3. スタジオ内は土足厳禁です。
  4. 貴重品の管理は各自で行ってください。
    紛失・盗難の被害に遭われましても、主催は責任を負いかねます。
  5. 当日撮影した写真について
    インターネット上に公開する際は、必ずモデルさん・撮影者の許可を得てからお願いします。
    トラブルの元になる場合もありますので、厳守してください。
  6. その他
    質問・ご要望等ありましたら、コメント欄に書き込んでいただくか、
    主催まで直接ご連絡ください。
