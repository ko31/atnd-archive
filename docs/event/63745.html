description

仕事のみならず、プライベートでも利用頻度の高いGoogleのサービス。 検索やGmail、カレンダー機能などは多くの方が利用しているかと思いますが、Googleで提供しているサービスは実はもっとあります。しかも、基本的には無料で利用可能です。

但し、主要サービス以外の機能の存在は知っていたとしても、使い方や活用方法がわからないということもあるのではないでしょうか。

そこで、今回は元VACANCY OFFICE月額会員で、システム開発会社 (株)シーケンス http://www.e-sqc.co.jp 代表取締役の篠原康行氏に、Googleが提供しているサービス「Googleドライブ」についての概要・使い方・活用方法を伝授いただきます。

【アジェンダ】

  • Googleドライブとは
  • 利用アカウントとデータ容量について
  • 利用できるサービスについて
  • Googleドライブ（クラウド）Officeのメリット
  • Office機能の説明
  • Googleドライブの活用方法
  • 利用の具体的事例
  • 質疑応答
