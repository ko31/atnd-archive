技術情報の保護と管理

イベント ユーザー
event_id 36023
title 技術情報の保護と管理
catch
description

【受講対象】
・研究開発部門のリーダー、担当者（特にプロセス系のエンジニア）
・製造技術部門のリーダー、担当者
・知的財産部門の担当者、契約担当者

【習得知識】
・情報管理に関する基本的な考え方を理解できる
・技術を特許で保護するべきかノウハウで保護するべきかの判断ができる
・特許とノウハウ、それぞれの方法で技術を保護するための具体的な手法を理解できる

【講座のポイント】
　他社との共同開発事業や、外国企業による社員の引き抜きなどの増加により、「技術情報」や「技術者」が社外に流出していくケースが増えています。このような状況の中で技術的な競争力を保つためには、技術情報（アイデア、ノウハウ）を、「社員の頭の中にある暗黙知」から「文書・図面などの形式知」に変換した上で、「会社の資産」として適切な管理をすることが不可欠です。
　本セミナーでは、開発現場や製造現場で生まれる「暗黙知」を「形式知」に変換する方法を学ぶとともに、特許・営業秘密などの保護方法を活用して「会社の資産」として適切に管理する方法を学びます。

【プログラム】
１．はじめに
　本セミナーで取り上げる内容のポイントを紹介します。
２．技術情報の流出が引き起こす問題
　技術情報流出の現状と、情報流出が引き起こす問題を紹介します。
　　2-1.技術情報流出の現状（流出の実態、流出する経路）
　　2-2.技術流出のケーススタディー
　　2-3.技術情報の流出が引き起こす問題

３．技術情報の流出対策
　技術情報の流出を防ぐための具体的な対策を紹介します。
　　3-1.技術情報へのアクセス自体を防ぐポイント
　　3-2.技術情報へのアクセスを許した上で、情報流出を防ぐポイント

４．技術情報を保護するためのポイント
　技術情報の保護する際に考えるべきポイントを紹介します。
　　4-1.保護するべき技術情報の把握
　　4-2.技術情報を開示する相手への意識
　　4-3.「情報の内容」と「情報を開示する相手」の関係

５．技術情報を保護するための「特許」と「営業秘密」
　技術情報の保護方法として、「特許」と「営業秘密」を紹介します。
　　5-1.某飲料メーカーの秘密情報管理
　　5-2.特許で保護する場合のメリット、デメリット
　　5-3.営業秘密で保護する場合のメリット、デメリット
　　5-4.技術の特性に合わせた保護方法を選ぶコツ

６．特許で技術を保護する
　特許で技術を保護する際のポイントを紹介します。
　　6-1.必要な権利範囲で特許を取得できるか
　　6-2.秘密の情報を公開してしまわないか
　　6-3.他社の権利侵害を発見、立証できるか

７．営業秘密で技術を保護する
　営業秘密で技術を保護する際のポイントを紹介します。
7-1.秘密情報として管理し続けることができるのか
7-2.他社が独自に同等技術にたどりついた場合にどうするか

８．まとめ
　セミナーの内容を振り返ります。
