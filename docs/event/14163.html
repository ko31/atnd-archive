title マーケティング庵
catch スマートフォン
description

４月のマーケティング庵は、今ユーザーが急増している【スマートフォン】をテーマにします。

そのスマートフォンの中でも今回は用途についてフォーカスし、マーケティング視点からユーザーの方々が日頃どういう使い方をしているのか、年齢等様々な属性別に紐解き、そこから生活者へのアプローチの方法をみなさんで考えてみましょう。

また、現在スマートフォンを使用されている方に、ご自分の主な用途や、こういった機能があれば便利なんだけど…といったご意見も交わしていただく予定です。

もちろん現在スマートフォンを購入されていない方もご参加いただけます。

＊ マーケティング庵は、身近な話題をテーマに生活者の視点から素人の感覚で、自由で気軽なフリートークスタイルで掘り下げるユニークなマーケティング研究会です。
スイーツやオーガニック食品の試食などから激安酒場の研究に出かけて、実体験してみるなど、参加者で体験しながら理解を深めています。
目的は愉快に時間とテーマを共有することです。

＊ご参加に関しては、マーケティング関係者や広告関係者、学生、マーケティングに興味のある方など、特に制限していません。
また、直接マーケティングに関わらない方やマーケティングは全く知らないという方、ただ単にテーマに興味があるだけの方でもご参加いただけます。

見学だけの、オブザーバー参加も受け付けます。
お気軽にご参加ください。
