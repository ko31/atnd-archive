＜BRUSHセミナー＞
使える！クリエイターのためのエクセル講座 ~第2回 / 5月17日(火) ~
powered by クリーク・アンド・リバー社

4月から新しい職場、新しい職種でスタートを切った方も多いと思いますが、どんな環境でもクリエイターの必須使用ソフトとなっているExcel(エクセル)。

エクセルに入力したり簡単な表を作ることは出来る方も多いと思いますが、活用できている、という方はまだまだ少ないようです。

今回は皆さんのお仕事をもっと効率的にし、クリエイティブに専念する時間を増やすために、押さえておくと格段に業務効率がアップするエクセル活用術をレクチャーします。

■この講座の特徴

★少人数制のクラスなので、わからないことはすぐ講師に聞きながら進めて行くことが出来ます。

★聴講型のセミナーとは違い、皆さんにも実際にエクセルを使っていただきます。

★講師はクリエイターとしても活躍している方ですので、クリエイターが実務で使う場面を想定しながらエクセル術を学ぶことが出来ます。

★全５回になっていますが１回ずつ完結された授業ですので、ご自身のレベルに合わせて参加することが出来ます。

■開講スケジュール

第一回：表作成・四則演算・ワークシートの使い方

★第二回：データ集計・順位や割合の算出・複数ページの印刷など

第三回：少ないステップで帳票を作る・関数の活用

第四回：グラフ作成・高度なデータ集計関数

第五回：ピボットで大量データを一発集計・マクロで自動処理

■下記思い当たる方、是非ご参加ください！

  • エクセルを使ってもっと業務効率をあげたい！
  • 簡単な表作成まではできるけど、それ以外のエクセル活用術がわからない…
  • エクセルで何ができるか知りたい！

今後のキャリアアップやステップアップにお悩みの方にはキャリアカウンセリングも行っております。
