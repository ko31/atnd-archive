CS-Cart勉強会を開催します

内容

高機能ECサイト構築パッケージ「CS-Cart」の勉強会です。

  • 2014年5月下旬リリース予定のCS-Cart Version 4.1の紹介
  • 今後のロードマップ
  • マーケットプレイスを構築できる「CS-Cartマーケットプレイス版」の紹介
  • 「CS-Cartマルチ決済」の紹介

対象者

CS-Cart（無料版を含む）を利用中、または導入を検討している以下の方を対象としています。

*ショップオーナー
*Webデザイナー
*Webデベロッパー
*CS-Cartとの連携を検討している各種ソリューションプロバイダー

日時

2014年5月24日(土) 14:00～（開場 : 13:30。途中退席可）
