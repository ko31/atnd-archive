description

あなたも、新電力に切り替えてないならすでに２年、○○円も損してますね。
２０１６年４月から電力が自由化されましたので。

９０％の人がいまだに新電力に切り替えておらず電気代を損してる？

でもわかります。
何事も新しいことというのは少なからず抵抗があるものですよね。

「新電力ってホントに安くなるの？」
「お金がかかる工事が必要なんでしょう？」

「手続きとかがめんどくさそう」
「頻繁にブレイカーが落ちるようになったりしないかしら？」

「新電力ってどういう仕組みなのかよくわからないから、とりあえず様子見」
「新電力って色々ありすぎて選ぶのが大変でめんどくさい」

「そもそも新電力ってなに？」

保証します。
上記に一つでも当てはまるならば当セミナーに参加する価値は大いにあります。

当セミナーではこういった新電力の疑問や不安にさっくりお答えします。
新電力について何も知らない方でも大丈夫です、ご安心ください。

あなたもセミナー参加後にはこの新電力の問題に関してさぞかしすっきりできることでしょう。

私事ですが、弟が某大手電力会社である東〇電力に技術者として勤めており、私自身も新電力の取次店をしているためにこういった話に少なからず理解があります。

こんなにお得で安心できる電気なのに、９０％の人がいまだに新電力に切り替えていないのは日本人の心理なのでしょうか？

もったいない、そんな思いからせっかくのお得な電気を広めるべく当セミナーを開催しています。

あなたも当セミナーで新電力の疑問や不安を解消して頂ければ幸いです。
今回、手続きが簡単で安心できるおすすめの新電力もいくつかご紹介しますので楽しみにしててください。

※私が取次している新電力も入ってます。

今回限りあなただけの特別オファー
セミナー参加者特典１
温かいコーヒーとチョコレートをお出しします

セミナー参加者特典２
おすすめ新電力をまとめたリスト（A４一枚）を差し上げます

セミナー参加者特典３
電気代の検針票を持ってきていただいた場合、
新電力に切り替えたらどれぐらい安くなるのかシュミレーション可能

セミナーの時間は30分から40分ほど、
あなたのご参加お待ちしてます。
