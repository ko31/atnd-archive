皆さんは「margin」というCSSプロパティをご存知でしょうか？

marginは、「ボックス要素の余白を指定するプロパティ」と説明されがちですが、CSSで組むレイアウトにおいて重要なプロパティの1つでもあります。
floatとclear、相殺、ネガティブマージン、これらは実制作におけるクロスブラウザ対応で必須ともいえる知識です。これらと密接に関係するmarginを正しく理解することで、ブラウザごとのデフォルトスタイルシートの理解や効率の良いコーディングを考える一歩となります。

今回はそんなCSSレイアウト手法を、CSSプログラミング等を行うこじか氏に、Emmetなどのコーディング開発環境を踏まえて、ライブコーディングを行っていただこうと思います。各レイアウトの例題を解きながら解説する、初級者〜中級者から一歩進むための内容となっています。

書籍では分からない、Reset CSS、Normalize.cssなど、
実制作におけるCSSテクニックを余すところなくご提供します。

Web Developer Selection（WDS）とは

Web Developer Selection（WDS）は、「数多くある技術から、Web Developerが最適な知識や技術をセレクション。セッション+ワークショップで提供する、とことん実践主義にこだわった技術力向上を目指す場」です。
「レスポンシブWebデザインの実践ワークフロー」でDesign Spiceの小川さんや、「FireworksユーザーのためのPhotoshop講座」をStocker.jpのなつきさんにお話頂いております。
