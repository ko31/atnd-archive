■イベント概要

2泊3日の合宿型KINECTハッカソン。マイクロソフトのエンジニアとサムライインキュベートが参加者を全面サポート。最終日に、合宿の結果をプレゼンして頂きます。

また、当イベントにご参加された方々は「Kinect for Windows Contest 2012」へもご参加頂けます。
http://k4wa.com
Kinect for Windowsを活用したサービスやアプリケーションの普及を目指して、
アイデアを募集するコンテストを開催いたします。ぜひふるってご応募ください。
主催：Kinect for Windows Contest 実行委員会（事務局：株式会社ゴーガ）
協賛：日本マイクロソフト株式会社、株式会社サムライインキュベート、東京エレクトロン デバイス株式会社
グランプリ賞金：100万円

■日程

・事前オリエンテーション　　　　6月1日(金)19:00～21:00
・開発合宿期間　　　　　　　　　6月15日(金)19:00～6月17日(日)18:00（※帰宅自由）
・最終プレゼンテーション　　　　6月17日(日)18:00～21:00

■対象者
下記に当てはまる方
・KINECT技術を採用したサービス開発に関心がある
・本気でサービス開発をする意思がある

（※合宿中の飲食費は各自ご負担ください。）

■持ち物

　・最終プレゼンテーション：　6月17日（日）18:00～21:00
　　サービス概要および作成したデモ版を発表して頂きます。 （各社プレゼン10分＋フィードバック5分）

S/W；

・Visual Studio 2010
・NET Framework 4.0
・KINECT for Windows SDK
*KINECT for Windows SDKを使用しており、その上で動作するミドルウェアを含む他のソリューションは使用しても問題ありません

KINECT；
基本KINECT for Windows センサーデバイスを使用してください
他社製のセンサーデバイスは不可となります

FAQ
1.Open NIを使用してもいいのでしょうか？
KINECT for Windows センサーはOpen NIを使用した場合は動作しません

2.他社からリリースされているセンサーデバイスを使用してもよいでしょうか？
今回はKINECT のアプリケーション開発です。KINECTを使用するようにしてください

3.仮想環境上で動作しますか？
動作しません
