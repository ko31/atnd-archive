description

「海外で働く」「英語を使って働く」ことに興味がある方向けの交流会イベントです。
UI/UXデザイン・ブランディングなどに携わるWEBデザイナー、ディレクターの方や、そういった仕事に興味のある学生の方を対象としています。
専門領域だけでなく、自分の可能性を広げるためにも海外に挑戦したい！…でも、いきなり働き方を変えるのは少し怖い。

そんな方にピッタリな環境があります。まずは身近にあるグローバルな環境で、自分のキャリアを形成しませんか？
フィリピン・セブなどに海外事業を展開する「LIG」 & 訪日外国人向けサービスを展開するベンチャー企業「Huber.」
海外と日本を行き来する両企業のデザイナー、エンジニア、採用担当が新しい働き方についてお話します。
当日は、日本 x セブ x フランスを中継し、参加型のパネルディスカッションも行う予定です。
コーヒー片手に、気軽にお話しましょう！

【こんな人におすすめ】
- 英語を活かした仕事に興味があるけれど、一歩踏み出せない人
- スキルを身につけたデザイナーやディレクターとして、一段上に進みたい人
- 海外など、今と違う環境で働いてみたい人

【スケジュール】
19:00　受付開始
19:30　LIG 「【中途 / インターン】「LIGの海外支社ってどんなことやるの？面白いの？」について話します！」
19:45　Huber.「【鎌倉ときどきセブ】デザイナーkiwaの働き方」
20:00　パネルディスカッション「日本 x セブ x フランス」
20:30　交流会
21:00　終了

【登壇者】
・堀口誠人（LIG Phippines代表）
2012年にLIG Inc.にジョイン。 エンジニア、デザイナー、ディレクターなど幅広く経験した後、右往左往あって現在はLIGフィリピン支社長。 好きなものは猫、嫌いなものはバナナの皮、ポケモンで最初に選ぶのはゼニガメ。

・三浦貴和子（Huber. デザイナー)
高校卒業後、単身渡米。大学在学中、大手ブランディング／デザイン会社でインターンを経験。 American Advertising Awardsにおいて金賞を２つ受賞。卒業後、WEB制作会社でデザイナーとして従事したのち帰国。Huber.のサービスサイト、販促物全てを担っている。
