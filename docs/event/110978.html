English

参加について

事前登録が必要です。イベントに参加登録する ボタンからお願いします。
キーボードでのプレイ不可。使用するコントローラー（USB）は持参してください。

トーナメント表について

イベントへの参加締め切り後、トーナメント表を公開します。
（Challongeを利用します）

大会の形式

  1. ダブルエリミネーション（2回敗退で除外）
  2. 全試合2セット先取、グランドファイナルは3セット先取
  3. 大会が遅れている場合は進行を完了することを最優先とし、状況に応じて大会ルールを変更する場合がございます。
    （14:30までには次の主催者が控えている為、完全に撤収します！）

大会のルール

  1. ターボ2（デフォルト設定）
  2. ランダムステージ選択（プレイヤーの合意があれば自由に選択できます）
  3. 勝者はチーム変更不可
  4. 勝者と敗者はチーム順番の変更可
  5. すべてのキャラクター使用可
  6. オプション設定にてポーズ状態が不可に変更されます。

大会の進行について

  1. まずキャン太2の試合を行うPCに持参のコントローラー（USB）を差し込みます。
  2. 初期画面でコントローラーを割り当てし、画面でキーボードのF5キーを押し、ボタン設定を行います。
    （間違えた場合はF2キーを押すと初期画面に戻ります）
  3. 1P、2P側のボタンチェックをお互いが終えた後、ローカル対戦を選択し試合を開始してください。
  4. 勝者は主催者に報告をお願いします。トーナメント表を更新します。
    （ダブルエリミネーションなので２度敗退したら大会から除外です。ご注意を！）

その他、注意事項

  1. 大会の進行にパソコン（A、B）を２台使用し、AはTwitch配信に乗ります。TOP8から配信に乗せるためにAのパソコンでの進行になります。実況（配信画面）席はウェブカメラを使用していますので、ご了承ください。
  2. 横並び対戦なので、出来る限りのプレイヤーモラル・マナーをお守りください。
  3. ゲームやコントローラーのトラブルが起きた場合、すぐ挙手をして主催者を呼んでください。状況に応じてすぐに対応いたします。
  4. いたずら・悪質・禁止行為と判断した場合は、細かく状況をお伝えください。場合によっては行われているセットをペナルティ側は敗北とし、次のセットを開始します。
  5. 上記ルールについて、予告なく追加、変更される可能性がございます。
  6. 体調不良や、何らかの理由で途中退場する場合は遠慮なく主催者に伝えてください。不戦勝・大会除外の処理を行います
  7. 主催者は注意欠如・多動症（ADHD）で、忙しくなるとミスが目立つかもしれないので、もしお気づきになられたら一言頂けたら助かります！
  8. 記述にミス、ご質問・ご指摘があれば、どうぞお気軽に下記のコメントまで書き込みをよろしくお願いします。
