平日の旅イベントは珍しいのでいかがですか♪
日本一の富士をバックに満開のかわいらしい桃の花を写真に撮りましょう♪(足湯もあります）
下は夜景、上は星空、周りはカラフルな湯船とフルーツ温泉ぷくぷくもオススメ！
なかなか食べれないスイス料理のお昼を食べて、ハイジになってヨーゼフと村を駆けましょうｗ
ワイナリーは可愛い洋風の建物で旅行会社さんにコッソリ教えてもらった秘蔵の場所です☆
最後にみんなで桃源郷でお花見をしながら花火を見ます！！！！

2015年4月10日（金）10:00出発
集合場所
東京駅
