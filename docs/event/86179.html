CREACT-クリアクト- 3月4月新規受講生募集

役者の役者による役者の為のトレーニングスタジオ

CREACTについて

数日間のワークショップではなく、定期的・継続的に役者としてのエンジンを温め続けるという趣旨の元、2月にスタートしたばかりのCREACT-クリアクト-

劇団なら、そこの演出家の言う通りに動けば劇団員としては正解でしょう。
ただ、一役者として舞台・映画・テレビ・CM と様々な仕事をしていくには、あらゆるニーズに答えていかねばなりません。
その為に、決まった講師、決まった価値観の中で自分を高めるのではなく、
様々な監督や演出家と出会い、それぞれの要求に答えて、一役者としての価値を高めていかなければならないのです。

クリアクトでは、映像系の監督から、舞台の演出家、ドラマの脚本家などをお呼びして、様々な講師の元で「演技」を深く学ぶ事ができます。

また、現役の俳優として活躍されている方をお呼びして、俳優だから教えられる、台本の読み方、役の作り方、現場での対応の仕方など、を指導してもらえます。もちろん、役者同志だから話せる悩みなども聞いて頂きます。

そして、定期的に「特別ゲスト日」というのを設け、特別講師の方をお呼びしてレッスンをしてもらったり、映画、ドラマ、CMなどのキャスティング権を持つ方にもお越し頂きます。そこでの評価次第では、仕事の場に繋がるチャンスになります。（スタジオ参加者は特別料⾦での参加が可能）

映画監督　倉田健次

シナリオライター/CMディレクター/小説家。
次世代を担う新しい映像作家の発掘/支援、映像文化への貢献の為、全世界から受賞者が選出されるサンダンス映画祭「サンダンス・NHK国際映像作家賞」にて『彼女のSpeed 』グランプリ受賞。新作劇場長編『藍色少年少女～Indigo Children～』が2016年より渋谷ユーロスペースを皮切りに日本全国にて絶賛展開中。

脚本家　ますもとたくや

フジテレビ系アニメONE PIECEのテレビシリーズで脚本家デビュー。 以降、テレビ東京系深夜ドラマ『宇宙犬作戦』や、MBS『コドモ警視』、TBS月曜ミステリーシアター『ホワイト・ラボ〜警視庁特別科学捜査班〜』など、テレビドラマを中心に、実写作品の脚本を手掛けることが多い。俳優ムロツヨシとのプロデュースユニット紳士スラックスを結成。2014年にアメリカンコミックシアターを旗揚げ。2016年、小学館ガガガ文庫にてマイダスタッチ 〜内閣府超常経済犯罪対策課〜が出版された。

俳優　川岡大次郎
