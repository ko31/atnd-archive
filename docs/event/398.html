catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：200円(実費)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　●セッション１
　『AIRの構築デモンストレーション』藤田さん＆板垣さん


　＜内容＞
　とりあえず、フラッシュでAIRガジェット（時計）を作ってみよう！
　イラストの取り込みから、簡単なスクリプト、背景の透明化など。
　デスクトップにのっけチャオ！

　＜トライポットの紹介＞
　ホームページ・フラッシュ作成、システム開発してます。
　楽しい仕事がしたーい！

　藤田：デザイン、サーバ構築、データベース設計、しゃべる担当。
　板垣：フラッシュ、ＰＨＰプログラム担当。
　とりあえず、２人でなんでもやってしまう会社です。

　●セッション2
　『Adobe Dreamweaver Developer Toolboxに関して』keisoさん

　＜内容＞
　プログラミング無しでも比較的高度なウェブアプリケーションが
　作成可能なAdobe Dreamweaver Developer Toolboxの紹介と
　デモンストレーションを行います。

　デモンストレーションでは、PHP+MySQLの環境で、掲示板か、
　ブログの様なモノを作り、その過程を見ていただければと思っています。

＜keisoさんの紹介＞
　フリーランスでPHPやMySQLを使ったウェブページ制作のお手伝いをしています。
　たまにFlashとかFlexでActionScriptを書いたりもしてます。

　●セッション３
　『Google Androidについて(仮)』重村さん

　＜内容＞
　2007年にGoogleより発表され、じわじわと広がりを
　見せているGoogle Androidについて、最近調べた情報について
　まとめて紹介したいと思います。
　現状、以下の構成で説明する予定です。
　※予定はあくまで予定です＾＾；ただいま考え中ですので、変更になるかも・・・
　１．Google Androidって？
　２．Google Androidで出来ること
　３．Google Androidのこれから

　＜重村さんの紹介＞
　仕事はSEということで、コーディングもしなくなってきていますが、
　元来のプログラマ気質で日曜プログラマをやってます
　（サボり気味ですが＾＾；）

　目標は大きく！
　ということで、世界に通用するようなサービスを構築すべく、
　活動を広げようとしている変わり者ですが宜しくお願いします。
