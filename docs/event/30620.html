はじめてのアマゾン ウェブ サービス(AWS)【セミナー形式】

アマゾンクラウド = Amazon Web Services(AWS)は、2006年のサービス開始以降、Amazon EC2やAmazon S3、Amazon Relational Database Service (RDS)など、開発者の皆様が、スケーラブルで高度なアプリケーションを従来より早く、簡単に、しかも手頃な価格で構築するための様々なウェブサービスを提供しています。
本セッションではAWSの概要とそのメリットを活用事例を交えてお話します。

Amazon EC2 & S3 ハンズオン

Amazon EC2、S3を実際に使っていただく実践形式のセミナーです。
ご自身のAWSアカウントを作成し、実際にEC2インスタンスを構築できるようになること、オンラインストレージサービス Amazon S3を使ってウェブサイトをホスティングできるようになることをゴールとします。
