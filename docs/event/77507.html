■Twitter広告って使える？？そんなあなたに最適のセミナーです！

ビジネスユースではPPC広告や最近ではFacebook広告、そしてInstagramの広告までもが出てきています。情報を集めるツールとしてもまだまだTwitterは顕在ですが、こと広告についてはまだまだ知られていないことが多いと思われます。

そこで、今回のセミナーでは、企業の宣伝・マーケティングご担当者様や個人事業主のみなさま向けに、売上・集客・会員獲得増に直接貢献するTwitterの活用・運用方法、広告の投資価値についてご紹介する、カジュアルなセミナーです。

商品・サービスのEコマース　会員獲得　実店舗集客　アプリプロモーション
アフィリエイト　B2Bビジネス　フリーランスの営業・・・等、
幅広い業種における成功・失敗事例や、最初に踏むべきステップについてご紹介します。

■このようなみなさんに最適です！！！

  • Twitterビジネス活用の概要・仕組みを知りたい（意外な機能をご紹介します）
  • 始めたいがどうしたらいいかわからない（5分で始める方法があります）
  • 稟議や予算の取り方がわからない（稟議書の設計図をお見せします）
  • ツイートするのが難しそう（成功・失敗事例をたくさんお見せします）
  • どのくらいの効率で獲得できるのか（業種別の実数字をお見せします）
  • B2Bの商材だから合わなさそう（よいターゲティング方法があります）
  • 一度試したことがあるがうまくいかなかった（成功のセオリーがあります）

■セミナー内容

『Twitterビジネス活用　最短距離で効果最大化するための最適化テクニック』
簡単にできる最適化施策や、トラブルシューティングについて

『Twitterでモノが売れる！セルフサービス式Twitter広告とは』
Twitter広告のセルフサービス運用の特徴、わずか5分でスタートできる、掲載の手順など

『Twitter広告　直近６ヶ月の業種別掲載レポートとその成功ノウハウを公開！』
業種別のクリック単価・獲得単価等の平均値や上限・下限値や、優れた活用事例をご紹介

※セミナー終了後、個別のご相談も承ります。

■スピーカーご紹介

森田謙太郎 ( Kentaro Morita )

  • Twitter Japan株式会社/ 中小ビジネス シニアマーケティングマネージャー

ソフトバンククリエイティブと宝島社で雑誌広告の営業を経験後、アイレップでリスティング広告のコンサルティングに従事。その後、ウォルト・ディズニー・ジャパンにてモバイル・ゲームコンテンツのマーケティングを担当。2015年よりTwitter Japanに入社し現職。

中小ビジネスが展開する、特に売上や会員獲得などのマーケティング活動にTwitterがどのようにサポートできるかに注力し、より短期的な効果を生み出すための解説・事例コンテンツの制作や配信を担当。
