description

2011年の未曾有の大震災から今年で4年が経とうとしています。
被災地の東北ではいまだ家族や大切な人が見つからず、「復興」という言葉の裏で立ちどまったままの人たちが居ます。

震災後、被災地に足を運んだ方、そうでない方、・・・いろいろな方がいる中で今回、関東で「現地の声を聞く・お話に耳を傾ける」機会はとても重要じゃないかと思いました。

東北に3年半もの間、足を運び、取材を続けてこられたノンフィクション作家の矢田海里さんによる写真展とお話会です。風化してはならないことや命の大切さ、失われてはならないものについて、この貴重な機会に皆さんで考えてみませんか。

小さなことでも、何か自分にもできることがあるー一人一人がそう考えるきっかけになればと思います。
平日の開催ですが、ご都合のつく方・またお子さん連れでも大丈夫です。たくさんのご来場をお待ちしております！

＜日時＞　2月26日（木）12：00～14：00（11：30開場）

質問タイム、など

【作家プロフィール】
矢田 海里（やだ かいり）ノンフィクション・ライター
1980年生まれ。慶応義塾大学総合政策学部卒。東日本大震災以後、ボランティアとして現地入りし、宮城県名取市を中心に取材活動を続ける。放送批評雑誌『GALAC』にて「東北再生と放送メディア」を連載中。また、旅人や冒険家をextremerとして紹介するサイトdocue!で9.11以降のアメリカの紀行文、Across-Americaを掲載中。活動の様子はHPをご覧ください。
