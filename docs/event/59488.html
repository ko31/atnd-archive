#dendai sie; Talk Live!
東京電機大学情報環境学部 パソコン部がお送りするちょっと変わったLightning Talk発表会。
技術的な話、ゲームの話、所属、興味、環境…
それぞれが個性溢れる話題を持ち寄り、
Lightning Talk形式で発表を行います。

#What is “Lightning Talk”？
Lightning Talkとは、カンファレンスやフォーラムにおいて、プレゼンテーションの合間に行う5分間のショートプレゼンテーションのことを指します。
このイベントでは、あえてその「短いプレゼンテーション」に着目し、自分の話したいことを5分間にまとめて発表しあう場を提供します。

#Theme
今回もテーマは「自由」！
自分の話したい内容を5分間という制約の中で
創意工夫を凝らしながらオーディエンスに伝えましょう！

#Present
全プレゼンターのLT終了後、オーディエンスを含めた全員で投票を行い、
最も優れたプレゼンテーションに選出された方にささやかなプレゼントを用意しております！
(プレゼントは決定次第この場で発表します…今しばらくお待ち下さい！)

#Ticket
参加は、登壇・視聴ともに無料となっております！
また、当日参加出来ない方のためにUstreamにて配信・タイムシフト録画を行います。

プレゼンターチケット :
登壇希望者用のチケットです。時間の都合上、今回は30枚とさせて頂きます。

オーディエンスチケット :
視聴のみの方はこちらのチケットです。

また、例によってLT終了後に懇親会を予定しております。
チケット申込時に必要事項と共にチェックを入れる欄がありますので、
参加希望の方はそちらにチェックをよろしくお願いします！
