労基法改正対策セミナー

イベント ユーザー
event_id 95738
title 労基法改正対策セミナー
catch
description

政府主導の「働き方改革」が進み、法改正、助成金等の環境整備の支援も整ってきました。中でも、「長時間労働の是正」については喫緊の課題として挙げており、「労働時間管理適正化指導員」や「時間外及び休日労働点検指導員」の大幅増員が予定されています。
本セミナーでは中堅中小企業に焦点を当てた、今後予定されている法規制ついて、ITソリューションでの対応を含めたご案内いたします。

日時）2018年4月18日（水）　13：30～17：30（受付13：00～）
場所）情報創造コミュニティー
詳細）http://www.t-itc.jp/fair20180418/
申込）http://www.t-itc.jp/fair20180418/mail.cgi
※セミナー内容・時間につきましては、予告なく変更となる場合がございます。
―――――――――――――――――――――――――――――――――――――――
【第1部】13:30～14:30
労基法改正と平成30年度の最新助成金情報について　（講師）汐留社会保険労務士法人
2019年4月に実現される労働基準法改正。労働時間の上限規制や有給休暇の取得義務化など、押さえておくべき改正内容に加え、具体的に企業が対応しなければならない36協定の見直しやより一層厳しくなる労働時間の適正把握のための対策について解説します。
―――――――――――――――――――――――――――――――――――――――
【第2部】14:50～15:35
勤怠管理のクラウド活用で実現する働き方改革　（講師）株式会社オービックビジネスコンサルタント
長時間労働解消をしていくためには、まず「現状を把握すること」です。労働時間を社員が確認できる環境や超過残業に警告できる仕組みはありますか？本セミナーでは、過重労働と労働時間管理の効率化を追求した「勤怠管理サービス」による対策例を詳しく解説します。
―――――――――――――――――――――――――――――――――――――――
【第3部】15:55～16:35
働き方改革ソリューションを活用した総務業務の改善事例とIT補助金の活用方法
（講師）協立情報通信株式会社
働き方改革が注目されている現在、特に総務業務の改善・効率化は不可欠となってきております。本セミナーでは、OBCパートナーAward最優秀賞６回受賞の協立情報通信が、業務改善に成功したお客様事例を交えて、ICTソリューションの活用ポイントをご紹介いたします。また、2018年4月から実施予定のIT導入補助金や活用プランについて具体的にご説明いたします。
―――――――――――――――――――――――――――――――――――――――

新奉行クラウドによる業務生産性と製品ロードマップについてご説明

奉行シリーズは、小規模から中小・中堅・成長企業まで累計56万社から高い評価を
いただいています。そのすべてのお客様が、クラウドで安全・快適に運用いただくために、
高セキュアな業務プラットフォームとして「奉行クラウド」は誕生しました。
クラウド化による業務効果と製品のロードマップについてご説明します。

個別相談会、実機体験も実施いたします。
