description

伝えたい想いが、ここにある。
全国の大学を巡り、4年生が学生生活で培ってきた「想い」を伝える、
「ソツセイ・講演キャラバン」第2弾は、慶応大学日吉キャンパス。

何かしてみたいけれど、何を頑張っていいかまだ分からない、
自分の可能性をもっと信じてみたい、
講演キャラバンは、想いを持ちたい、育てたい大学生に贈る企画です。

サークル活動に打ち込み、100人以上を運営に巻き込んだ英語劇を統括した学生と、
学外のNGOで3年間走り回ってきた学生、
まったく異なる道のりを来た二人の「想い」、「想いの育て方」を伝えます。
