  • 「OAuth 2.0 update」 by @nov
    　ドラフトの段階で facebook や twitter など既に様々なサービスにサポートされている OAuth 2.0ですが、先ごろついに成人 (IETF I-D version 20) を迎えました。仕様の最新状況について、@nov さんが分かりやすく解説します。(当日までには draft 取れてるかも!?)
  • 「OpenID Connect update」 by @_nat
    　次世代のOpenID、"OpenID Connect" に関する仕様の策定状況やこれにより実現できること、Google や facebook, Microsoft, salesforce.com などの動向などについて、米 OpenID Foundation Chairman の @_nat さんが説明します。
  • 「 Cloud Identity Summit 2011 TOI」 by @tkudos
    　クラウド上の ID管理テクノロジーに関する著名なイベント、CIS 2011 が7月下旬にコロラドで開催されました。当イベントに参加してきた @tkudos さんが、OpenID や OAuth に限らず、クラウド・アイデンティティの技術トレンド全般についてお話します。

技術者向け無料セミナー OpenID TechNight #7 を開催いたします。仕様が策定中の OpenID Connect や OAuth 2.0 の最新状況や、海外のクラウド・アイデンティティの動向などについて、分かりやすく解説します。
