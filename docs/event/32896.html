Quintechプロデュース ☆はじめてのAPI

API、なんとなくは理解しているけれど自分で使ったことはない、というTech女子のためのワークショップです。

MA8のAPIパートナー7社のご協力をいただき、APIごとにチームに分かれて少人数でハンズオンします。

ワークショップ後にすぐアプリを作れるようになる！・・のは難しいと思いますが、マッシュアップ（複数の Web サービスの API を組み合わせ、新しいWeb サービスを作ること）の楽しさを発見できる3時間になると思います！お勉強の後は、お酒と軽いフードを食べながらワイワイ交流会も予定していますので、お楽しみに！

＜先生プログラマーの皆さま＞

ホットペッパーBeautyAPI（リクルート）
感情解析API　（メタデータ）
トラベル空席検索API　（楽天）
天気予報API　（アップフロンティア）
画像認識API　（PUX）
tabAPI　（頓智ドット）
boudioAPI　（KDDIウェブコミュニケーションズ）

＜よくある質問＞

■何をするイベントなの？
前半で、APIについての講義とTwitter APIを使った簡単なプログラミングを行います。後半は、身近なWebサービス会社7社が提供するAPIごとにチームに分かれて、ワークショップを行います。

■何が出来上がるの？
APIを使った簡単なプログラムを書き、Webブラウザ上で実際に動かしてみます。

■LANや電源は使えますか？
会場ではゲスト用無線LAN・電源が利用可能です。

■開発できなくても平気なの？
もちろん大丈夫です！後半のワークショップは1チーム最大5名の少人数制なので、気軽に先生に質問してください。

■友達と参加してもいいですか？
大歓迎です。お友達と同じチームになりたい場合は、当日お申し出ください。
※お友達の方もATNDで参加登録をお願いします。
