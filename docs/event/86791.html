FMS5期生の皆さん、ご入学おめでとうございます。

B1(1年生)向けに、昨年開催して好評だった、自己紹介LTを今年も開催したいと思います。

先輩が大学でどのようなことをしているのか発表してもらうので、どういう学科なのか、体験しに来てください。

LTとは？

超簡単に言うとミニプレゼンです。

ライトニングトーク（英: Lightning Talks）とはカンファレンスやフォーラムなどで行われる短いプレゼンテーションのこと。様々な形式があるが、持ち時間が5分という制約が広く共有されている。(wikiより)

タイムテーブル

基本、参加者は何かしゃべるということでお願いします。

  • 自己紹介
  • 最近したこと
  • 自慢したいこと
  • 興味があること
  • 今年度の目標
  • やりたいこと
  • なんでも可
  • パワポ可
  • デモ可
  • 動画プレゼン可(もし参加出来ない場合、事前に動画データをもらって当日流します)

サークル・団体の紹介は別途時間を設けます。個人の自己紹介の時間には行わないようお願いします。

atnd登録した人から下のタイムテーブルに追加していきます。B1(一年生)以外はコメントで以下の項目を教えてください。

基本はM1→B4→B3→B2→B1という順番でやります。

B1以外は一人3分程度、B1は1分程度 （参加人数により変わる場合があります。）
