＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

ご存知ですか？　要介護認定者が増加する中で、
「　リハビリテーション需要　」が増加しています。
「2025年問題」つまり団塊の世代の方々が後期高齢者になる為、
医療やリハビリテーション需要が拡大する事は、間違いありません。

訪問してリハビリテーションを提供できる事業はたった２つ。
その中で唯一民間参入が可能なのは、「　訪問看護ステーション事業　」です。 その１つに参入してみませんか？

厚生労働省も「在宅リハビリが不十分」であると言っているように、
サービス提供者の新たな担い手として期待されているのが、
訪問看護ステーションが提供する、リハビリサービスなのです。

まずは、需要拡大が続く「在宅リハビリ」の現状を理解し、
そのサービスが本業と、どのような相乗効果を生むことができるか、
是非一度お問い合わせ下さい。

現在のところ、全国３００拠点の開業支援をさせて頂いています━━━━━

　私共からご案内させて頂く『ケアーズ 訪問看護ステーション』は
　開業支援は利益率の高い介護事業で成長したい方におすすめです。

事業参入から最短３ヶ月～６ヶ月で以下のような数値を達成が可能━━━━
　
　個人宅に出向くスタッフ（資格者）を少しずつ増加させ、１０名体制が
　組めた時点でおおよその月商は８００万～１０００万円前後。
　スタッフ（資格者）にかかる人件費は基本３５０万円、
　歩合でプラス１５０万円程。その他諸経費で１００万円程度ですので
　営業利益２００万円～３００万円は単月で達成可能な数値です。

◆医療・介護ビジネスの経験がない、異業種の企業の方でも、
　 安心して参入できる開業支援を提供しています。
