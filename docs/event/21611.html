風呂敷ワークショップ開催！

11月3日、国民文化祭京都2011で「風呂敷ワークショップ」を行います！
3.11の東日本大震災以来、エネルギーや資源、環境のことについて考える機会が多くなったと思います。
そこでこの機会に私たちは、この京都の知に根付く「風呂敷文化」を見なおしてみました。
するとそこには環境にとても優しい部分や、興味深い歴史など様々なものが散りばめられていたのです！
そんな素晴らしい風呂敷に実際に触れてみませんか？
京都の庭園でお茶をご用意してお待ちしております！

講演形式

魅力的な風呂敷の歴史の紹介 と 風呂敷を使った包み方講座


素敵なおみやげ！

参加者にはとても素敵な風呂敷を1枚プレゼントしております！
