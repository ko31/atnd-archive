Raspberry Piを使っていろいろやってみよう

えれくら！TypeRは、Raspberry Piを主題とした勉強会です。

今回は、「Windows10 IoT Core for Raspberry Pi」以外にも、各種Linux系ディストリビューションや、ほかのOSも話題に含め、RaspberryPiについての、広い範囲での情報交換を行います。
RaspberryPi にご興味のある方、どんどんご参加くださいませ。

ご用意いただくもの

  • Raspberry Pi各モデル
    RaspberryPiには現在、モデルが数種類ありますが、持っているモデルをお持ちください。
  • Micro SDカード
    SDカードは、OSをインストールするのに使います。8GB以上の容量があるClass 10以上のものをご用意ください。
  • ノートPC
    できれば、ノートPCをお持ちくださいませ。試してみた結果をそのまま持ち帰ることができるので便利です。

    開発環境や実機へのOSインストールで困ったことになっちゃった方は、午後が始まる前に、ぜひお助けタイムへご参加くださいませ。

    ※Windows10 IoT Core for RaspberryPiに関するインストールについて
    RaspberryPiは、RaspberryPi2またはRaspberryPi3が必要です。OSインストール用MicroSDは、8GB以上の容量があるClass 10以上のものが必要です。
    開発用のPCは、 Windows10 がインストールされているもの が必要です。HDDやSSDには30GB以上の空き領域が必要です。開発環境がかなり重いので、快適に開発作業をおこなうためにはPCはなるべく高性能なものをご用意ください。
    また、PCには VisualStudio 2015 Community Update 2 と、各種SDKをインストール しておく必要があります。

    Windows10 IoT Core 開発環境のインストールに関する参考資料はこちら：
