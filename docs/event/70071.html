フットサルでリフレッシュするのが目的のエンジョイフットサルです。

最小催行人数は15名です。詳細は下記参照ください。

※時期的にお子様の運動会にかぶるかもしれませんので事前のご確認をお願いします。
※ここ以外でも集めていますので随時定員を調整していきます。ご了承ください。

15:00から120分

フットサルポイントMESSE天下茶屋

15名

・コート代1500円。現地で集金させていただきます。

　各自フロントでお支払いください。

・IT関係でなくても良いです。
　男女問いません。

　未経験でもOK。上手くなくてもOKです。
・お友達との参加やチームで参加される方はその旨コメントに書いてください。
　会社チームで参加もOKです。
・チーム分けをなるべく偏りがないようにしたいので
　サッカー・フットサルの経験の有無や年数を書いていただけると幸いです。

服装と持ち物

  • 必須
    • フットサルができる服装（ジャージ、Tシャツ等。運動できればなんでもOK）

  • あると良いもの

備考

  • フットサルシューズは裏がアメ色のもの。または運動靴に限ります。
    フットサルシューズでも裏がイボイボのシューズやスパイクも禁止です。
    お持ちでない方はレンタルできます（300円）。
    フロントで各自レンタルしてください
  • コート内での飲食・喫煙は禁止です。指定の場所をご利用下さい。
  • 到着した人から更衣室でフットサルができる服装に着替えてください。
  • 着替えた荷物は預けられないので、コートに持って入ってください。
    置き引きが多いので荷物の管理をしっかりとお願いします。
  • 雨の場合でも屋根がありますので開催します。
  • なんらかの理由で中止になる場合はここに掲載します。
    前日には確認するようにお願いいたします。
  • ご質問や連絡がありましたら ひとのみち（@twitter） までご連絡ください。

Q&A

Q.フットサルシューズは普段のくつのサイズで良いですか？
A.フットサルシューズはレンタルする時に試し履きできるので、余らないよう痛くない程度のきっちり目をお勧めします。

Q.靴下は薄め厚め長め短めどれがよいですか？
A.脱げないものであれば大丈夫です。保護の観点からではくるぶしまであった方が良いとおもます。

Q.キーパーの装備はありますか？
A.キーパー用のグローブも長袖も用意しませんので、怪我しないよう気をつけてください。
