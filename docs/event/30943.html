ハッカソンとは

みんなで集まり、アプリの開発・発表をする会のこと。
「『メトロ・スタイル』ってなあに？」という方も大歓迎です。

参加するとこんな良いことが！

  • みんなと一緒にアプリ開発ができる
  • アプリ開発者との交流ができる
  • 自分の知らない、技術・情報が共有できる
  • 初心者の方なら、今後もプログラミングを継続していく自信がつく

イベント内容

  • Metro初心者の方向けに、簡単なワークショップを開催します。
  • 経験者の方は、お題に沿って自由にアプリケーションを作成していただきます。
  • 最後に、全員でアプリケーションの披露をしていただきます。

参加条件

特にございません。初めての方、初心者の方、大歓迎です！
デベロッパーの方、デザイナーの方、興味のある方ならどなたでもご参加頂けます。
会場のPCを利用できるので、Windows8などの環境を構築して持参する必要はありません。（もちろん、マイPC持ち込みOKです）

場所

用意していただくもの

  • 開発用PC（用意できる方）
    • OS Windwos 8 RP 又は Windows 8 RTM
    • VHDブートでもかまいません。
    • 仮想環境でも大丈夫（のはずですｗ） Macを使う場合はBootcampでの動作を推奨します。
    • 持参できない場合は、会場のPCを使用できます。（ただしタッチパネルではありません）

※開発用PCを持参する場合は、あらかじめ下記のツールをインストールしておいてください。

Visual Studio 2012 RC
