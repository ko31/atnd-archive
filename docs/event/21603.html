いよいよ就職活動がスタートする時期となりますが、業界研究・企業研究を
これから始める皆さんにぴったりの就職イベント。
IT業界に関するセミナーや、成長中IT企業の話をじっくりと聞くことができる
【アカリクIT企業交流イベント】を、11月26日(土)に東京にて開催いたします。

アカリクイベントだからこそ実現する、他では聞くことのできない
【業界研究セミナー】も実施しますので、業界や企業について理解を
深めることはもちろんのこと、企業の人事担当者他、現役エンジニア
と密に話のできるまたとない機会となるはずです。

皆様のご参加を心よりお待ちしております。

ご友人もお誘いあわせの上、ご参加ください！（事前予約が必要です。）

= イ ベ ン ト の 特 徴 =

【他では聞くことのできない「業界研究セミナー」】
参加企業のうち3社による「業界研究セミナー」により、業界理解・仕事理解を
深めることができます。他の就活生と差をつけるチャンスです！

【企業の人事担当者や現役エンジニアと密に話せる！】
大人数で行われる一般的なイベントとは違い、事前予約制・定員100名なので
採用のキーマン他、現役エンジニアと密に接することができます。企業の話を
じっくり聞いて、積極的に情報を得てください。

【効率良く企業の話を聞くことのできるタイムスケジュール】
参加企業のうち3社による「業界研究セミナー」＆1社5分の「全体プレゼン」の
あと、「1ターム30分×5ターム」のタイムスケジュールに従って各企業ブースを
自由に回っていただきます。
話を聞けなかった企業やもっと詳しく話を聞きたい企業については、イベント
終了後のフリータイムをご活用ください。

席数に限りがございますので、お早めにご予約下さい。

ご不明な点はアカリクイベント運営事務局までお問い合わせ下さい。
