＜講座のポイント＞

　製造業における研究開発，生産，品質管理，出荷後顧客対応などの各工程においては，様々な問題が発生します。これらの問題を解決するためには，以下のような作業が不可欠です。

　１．問題の整理，分析と再定義

　２．原因や因果関係に関する推定と仮説構築

　３．仮説検証方法の検討

　４．検証方法の実行（実験，測定）

　５．実験データの解析

　６．実験結果に基づいた，結論の論理的導出（必要に応じ，仮説の修正と再検証）

　　本セミナーでは，初心者がなかなか気づかない実験，測定のポイントや統計的な観点に基づくデータの解析・解釈方法について，説明します。

＜受講対象者・レベル＞

　・製造業での実験や測定にこれから携わるかた。

　・実験や測定にすでに携わっているが，実験方法やデータ解析に自信のもてないかた。

＜プログラム＞

１．問題解決プロセス

　　1.1　問題の再定義の必要性

　　1.2　問題の再定義のポイント

　　1.3　原因推定と仮説構築

２．測定結果の信頼性に関する概念

　　2.1　ばらつき～繰返し性と再現性

　　2.2　真値，測定結果，誤差

　　2.3　ばらつきとかたより

　　2.4　最低限着目すべき統計量

３．実験の3原則

　　3.1　実験の反復

　　3.2　生データの分布と「平均」の分布

　　3.3　何回測定すればよいか?

　　3.4　実験の局所管理

　　3.5　実験の無作為化（ランダム化）

４．実験前の心得

　　4.1　何を目指して実験するか?

　　4.2　何に着目しなければならないか?

　　4.3　予備実験の重要性

５．実験中の心得

　　5.1　実験担当者としての心構え

　　5.2　ばらつくデータは悪か?

　　5.3　実験中または実験直後の判断

６．解析の心得

　　6.1　グラフによる全体像把握

　　6.2　表示桁数を決めるポイント

　　6.3　100回測定しないとわからないような有意差は本当に有意差か?

　　6.4　相関と因果

７．Microsoft Excel活用のポイント

　　7.1　グラフ

　　7.2　データベース

　　7.3　オートフィルタ
