description

昨今、オープンソースソフトウェア（以下OSS)は、様々なシステム開発において
欠かせない存在となっています。それに伴い、オープンソースライセンスも複雑化する中で、OSSを安心して安全に活用できる環境づくりが重要となっています。本セミナーでは、このような背景を踏まえ、オープンソースライセンスについて理解を深め、その未来像について情報発信を目的に開催致します。
