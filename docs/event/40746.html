今回のテーマは、「HTML5とANDROIDの未来です。」各分野の第一人者が会津に集結し、HTML5とANDROIDの未来について語りあいます。

講演内容:

17:00-17:50 「Packaged Web Appについて - AndroidとChromeの統合」

日本Androidの会 丸山不二夫


現在、アプリ開発のスタイルに大きな変化が起きようとしています。HTML, CSS, JavaScriptといったWebの標準技術をクライアント側のアプリに応用しようと言う流れが大きくなっています。そうしたクライアント・サイドのWebアプリは、現在、Packaged Web Appというスタイルでまとめられています。講演では、Packaged Web Appの代表的な取り組みとして、Google Chromeのアプローチを紹介し、Googleのこうした動きが、AndroidへのPackaged Web Appの導入を大きな目標としているという予想を述べたいと思います。

18:00-18:50「HTML5とCoronaSDK」

HTML5Caravan 山本直也


HTML5でのアプリ開発の最新動向や、CoronaSDKでのアプリ開発手法やトレンド等に関して解説します。

19:00-19:30「Google Glassが変える世界」

GClue 佐々木 陽


Google I/O 2012でそのコンセプトが発表されたGoogle Glass。2013年の春についに一部ユーザに出荷が始まりました。Google GlassはOSにはAndroid 4.0を搭載し、Google Glass専用のアプリケーションが起動しています。Google Glassでは、Glasswareと呼ばれるアプリケーションが登録可能なうえ、Androidアプリを独自にインストールしたりRootを取得したりする事も可能になっています。Google Glassの構造やGoogle Glassの未来について述べたいと思います。

19:30-20:00「HTML5とAndroid以外のOS」

株式会社デザイニウム 浅井 渉


Firefox OSやTizen、Ubuntu Touchなど、HTML5ベースのプラットフォームの普及がWeb制作者にもアプリ開発への門戸を開く可能性についてお話しします。
