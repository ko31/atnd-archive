description

オープンソースカンファレンス2010 Kansai@Kyotoが7/9(金)-10(土)に開催されます。その参加者で宿泊所を借り切って、一夜限りのギークハウスにしませんか。


日時

2010年7月9日(金)

場所

東九条文庫センター

  • 夜通し Hackathon やるも良し、明日のために英気を養うも良し、各自自由。
  • 駐車場は近所のコインパーキングを利用。
  • 15～20名を上限として募集、今のところは16名で。
  • ネット環境はe-mobileルーターを用意します。電源はあります。
  • 素泊まり。食料などはみんなが持ち寄る。(台所もあります)
  • ふとんは10組ぐらい。足りない人は雑魚寝！

会場

東九条文庫センター


京都市南区東九条南河原町


地図: http://www.mapion.co.jp/m/34.9777677777778_135.768923333333_9/


青い壁に農楽の絵がある2階建ての建物で、その2階が会場です。
3つドアがありますが、中央にある引き戸を開けてください。



とても分かりにくい場所で、カーナビで「東九条文庫センター」と
入れても辿りつけません!! 「九条須原通の1本北」で探してくださ
い。

各自用意するもの

  • 誰かプロジェクタ持ってきて！
  • 自分が食べたいもの
  • 宿泊所へのカンパとして1人2000円

畳12畳ぐらいのスペースに長机を置けば20人くらいは談笑できます。
2段ベッドが2つあるので、雑魚寝を含めて16人くらいは寝られます。


厨房があります。夜食や朝食も作れます。調味料はある程度揃って
いますが、できれば持参して余りを寄付してもらえると助かります。
400m 離れた場所に大黒屋があるので、そこで食材を調達してもいいでしょう。
食器は一通り揃っているので、紙コップ、紙皿、割り箸などは無用
です。


風呂設備もあります。タオル、石鹸、ドライヤーは持ってきてくだ
さい。


利用される方には、1晩2000円のカンパをお願いします。この場所を
維持・管理するためにご協力をお願いします。

当日の連絡、移動と翌日

多分、OSCの懇親会が終わってみんなで移動になると思います。懇親会に白衣で参加している人がnanbuwksです。終わったら集まってね！

OSC の懇親会場からは、地下鉄二条城前→烏丸御池→九条、という経路
で行けますが、タクシー数台に分乗した方が楽で早いでしょう。
