初期ロット：10枚
値段：7000〜8000
リリース：MFT2013のある11/3を予定
渡す方法：MFT2013 or 郵送

FPGAとは：Field Programmable Gate Arrayの略でIC内のハードウェアロジックを自在に書き換えることができるデバイスのこと。HDL(ハードウェア記述言語)と呼ばれる言語でハードウェアロジックを記述することができる。
ハードウェアの高速性を持ちながら、柔軟にプログラマブルロジックを組むことができることが特徴。

※Arduino IDEでプログラミングするものではなく、HDLコーディング、コンポーネント開発をするためのハードです。

Physicaloid FPGA Peridot（フィジカロイド FPGA ペリドット）

特徴
Arduino型FPGAボード
  • Arduinoピン互換ボード
  • そこそこ遊べるサイズのFPGA Altera Cyclone IV EP4CE6E22 (6,272 EL)を搭載
  • 64Mbit SDRAM 搭載
  • Arduino用シールド使用OK（ただしIO電源3.3Vに限る、アナログ入力なし）
  • USBケーブル1本で動作可

PERIDOT の回路図、HDLコード等 https://github.com/osafune/peridot New!

Macでもいけるエントリ向け開発環境（予定）

MacだとQuartusIIインストールできないから開発できないんだよﾈｰをなくしたいので。。。

  • FPGA初心者用にコンパイル済データを提供。
    　「とりあえず動かしてみる」ができるようMac,Linux,Win Chrome/AndroidからUSBケーブル1本でFPGAに即コンフィグレーション、即動作。
    　元となっているHDLも公開
  • もちろん通常のQuartusIIで開発、USB BlasterでコンフィグレーションもOK
  • Webコンパイル環境構築予定（QuartusIIが使えないMac,Linuxでもコンパイル可）

Chromeブラウザ、Androidと連携（動作テスト中githubにてコード公開中）


FPGAの処理結果をPC/Android上で表示する仕組みを提供

  • Chrome, Androidのjs/Java開発ライブラリを提供、オープンソースで公開
  • Chrome/Android ⇔ FPGA間データ通信のためのシンプルなAPIを提供
  • Win, Mac, LinuxのChromeブラウザ、もしくはAndroidからコンフィグレーション

Chrome js コード https://github.com/osafune/Physicaloid.js New!
Android Java Library コード https://github.com/ksksue/PhysicaloidLibrary New!
