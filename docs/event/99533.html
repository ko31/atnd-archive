description

プログデンスでは転職活動中で運用から上流工程にキャリアアップしたい方や、一部構築に携わってはいるが、設計要素を学ぶ機会が少ない方々を対象として、8月26日(日)にセミナーを実施いたします。

「どこも上流工程を謳っているし、見分けがつかない…」
「設計・構築業務を目指すために必要な、環境と勉強方法とは？」
「20代のうちに意識すべきキャリアステップって何があるの？」

そのような方のために、一般的な IT 企業の入門研修によく見られる問題点をお伝えした上で、当社の実業務に活かせる教育カリキュラムの一部をご紹介します。原理原則となる基礎習得のコツや、実際に実機を利用して、設定内容の設計と、物理接続や設定投入等の環境構築も行い、設計・構築業務の一部も体感していただけます。

また転職先企業を選ぶポイントとして、エンジニアとして成長し続けるために必要な、会社の教育内容や環境、周囲のエンジニアのレベルの高さ等に焦点をあてて解説をいたしますので、是非この機会にご参加下さい。

＜対象となる方＞
20代のNWエンジニアで次のような経験と、転職をお考えの方々を対象にしています。
・CCNA を取得されている方
・ネットワーク運用業務を2年間以上経験された方
・ネットワーク構築業務の一部でも従事された経験がある方
・プログデンスへの応募を検討したい方
　※面接を同日に実施希望の方は、履歴書・職務経歴書をご持参ください。
　※選考プロセスにご応募いただいた方は対象外とさせていただきます。

※定員10名を超えた場合には、締切を待たずに早期に申込みサイトをクローズさせていただくこともございますのでお早めにお申し込みください。
