HCCC「学生作品展示会」

今年発足された「北陸クリエイティブコンテンツシティ」によるイベント、
「学生の学生による学生のための作品展示会」を９月２０日(土)、２１日(日)に行います。
当日はプロの方たちによるセミナーやトークセッション、
また懇親会も行いますので是非ご参加ください！

プログラム内容

13:00-14:00　Unity Technologies Japan による公開セミナー

　　3F 大会議室 / 参加費：無料
　　ユニティコミュニティエバンジェリスト 小林様による公開セミナーを行います。
　　今話題のUnityについて、現状や、今後の展開などお話しいただきます。
　　乞うご期待！！

　　＜登壇者＞
　　Unity Technologies Japan コミュニティエバンジェリスト
　　小林信行（こばやしのぶゆき）さま
　　＜プロフィール＞
　　最近はMayaを使うことが多い元コンシューマゲーム系クリエイティブディレクター。
　　現在はUnityを通じて、ゲーム作りを楽しくしたいコミュニティエバンジェリスト。
　　多くの原作付きキャラクターゲームの企画＆開発に関わり、アニメ業界にも精通しています。
　　代表作：『涼宮ハルヒの追想』『涼宮ハルヒの約束』『とらドラ・ポータブル！』

14:30-15:30　学生による作品プレゼン大会

　　3F 大会議室 / 参加費：無料
　　各学校から、我こそは！と手を上げた全１２組による５分間のピッチプレゼンです！！
　　ゲームやアプリ、デザイン、映像など、様々なジャンルの作品に込められた
　　生徒たちの熱い想いをお聞きくださいー！
　　どんなプレゼンが飛び出すか、乞うご期待！！
　　※ピッチプレゼンとは短い時間の中で自分の作品や個性をアピールする
　　　プレゼン方式のことです。

16:00-17:30　コンテンツ系企業経営陣によるトークセッション

　　1F 談話室 / 参加費：一般 1,000円・学生 無料
　　定員：50名（一部立ち見）
　　コンテンツ系企業の経営陣によるトークセッション
　　参加者には「限定ユニティちゃんステッカー」プレゼント！

　　登壇者：
　　１）ウィザードソフト株式会社　村田さま

　　２）ヒロイズムラボ　北さま

　　３）ラノベ作家　荒川さま
　　４）Unity Technologies Japan　小林さま（ゲスト）
　　　セミナーに引き続き登壇していただきます。
　　５）株式会社C8LINK竹田（司会進行）

　　以上の方々が、コンテンツ業界のこれまでとこれからについて語ります！
　　見学者の方々や学生からのきわどい質問にも答えます！
　　どんな応答が繰り広げられるのか、乞うご期待！！

18:00-20:00　立食パーティー懇親会

　　3F 大会議室 / 参加費：一般 2,000円・学生 1,000円
　　定員：50名
　　プログラムの登壇者の方々を交えての懇親会になります。
　　名刺交換・登壇者の方々への質問など自由に交流していただけます。
　　オードブルや簡単につまめるものを用意します。
　　会場の都合上、飲み物はソフトドリンクのみになりますのでご了承ください。
