IINAlab スマートフォンの世界をまるっと変える！
Project Yを8月22日に発表
秋葉原HUBにて、記者会見兼飲み放題兼トークショーを開催

報道関係各位

暑中お見舞い申し上げます。

さてこの度、IINAlab株式会社(以下弊社)では、8月22日金曜日、(多分世界初)となる、飲み放題記者会見を行う運びとなりました。
発表するのは、これも世界初となるProject Yの全貌をこの記者会見でプレゼンします。(もちろん、記者会見で正式名称を発表します。)

会見で初公開となる、同プラットフォームおよび、ハードウェア規格は、C2CおよびO2O、B2B2Cなど、なんでも来い！兎に角スマートフォン環境を [まるっと変えて] しまう、という目論見で特許申請を行っている技術及びサービス群です。

本記者会見では、発案者である弊社代表でない方の取締役の飯嶋の発表スピーチの他、同分野で著名ライターである本田雅一氏をお招きしてのトークショーを行います。

日本発のペンチャー企業が世界に打って出るプラットフォーム発表に世界の趣都、秋葉原を選びました。

是非、ご参加、取材をお願いします。
