日本以外の注目スタートアップについて発表しあう会です。
一人5分程発表で20人くらいを予定しています。（自己紹介は短めに）

　
【スケジュール】
2010年11月26日（金）19時～21時位＋懇親会

19:00-19:10：
　　@ono_masatoさん「最近の海外スタートアップの投資状況について」
　　http://d.hatena.ne.jp/masaono777/
　
19:10-20:50:
　　5分間×20人程、注目スタートアップのプレゼン
　
20:50-21:00：
　　スペシャルゲスト： 百式 @taguchi さん
　　http://www.100shiki.com/
　
【場所】
Open Network Lab http://onlab.jp

グローバルに活躍することを目標に、インターネットを利用したビジネスでの起業を志すエンジニアを支援するプロジェクトであり、「Seed　Accelerator」「Laboratory」「Community」と呼ぶ三つの活動を通じて、多角的に起業を支援します。海外の有力ベンチャーキャピタリストや起業家とのネットワークを活用して、海外進出を支援するゲートウェイとしての役割も担います。

現在、第2期SeedAcceleratorに参加する個人やチームを募集中です。募集期間は2010年10月27日～12月1日までです。
　　
【会費】
・参加費無料
・飲み会代のみ（2000円～3500円くらい）
　
【申込方法】
・登録時に発表するサービス名を書いてください。
・Twitterの@nobot_kをフォローしてください。（連絡方法としてDMを利用するため。）
・サービス発表は以下の点を参考にお願いします。

　・どんな人が創業しているのか？　
　・どんなサービスの内容なのか？
　・どうしてそのサービスができるのか？
　・どうして注目されているのか？
　・どのように今後展開していくのか？
　・自分がならどんな工夫をするか？
　
　※国内のサービスや誰もが知っている有名なサービスは、
　　発表内容を変更してもらうことがあります。
