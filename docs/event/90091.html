■参加方法

See you NeXT meeting !

横浜Apple User Group『AoMac』のmeeting（交流会）を上記の要領で開催します。
参加希望の方は、
Mac や PC でこのページをご覧の方は→右カラム
iPhone や iPad でこのページをご覧の方は→下段　にある赤地の
【イベントに参加登録する】をクリックし、
指示に従って参加登録してください。
Facebook でも受付しています。

■Apple User Groupって？

Apple User Group（アップル ユーザグループ） は、アップルの製品（Mac、iPhone、iPad、iPod、Apple WATCH等）が大好きな皆んなの会です。

『 AoMac 』は横浜北部エリア（主に東急「田園都市線」沿線）居住のメンバーが中心のApple User Groupです。
Macをはじめアップル製品や広くIT関連の情報に興味ある方、
興味はあるけど「何にも解らないから教えて〜！」な方、
お住まいが横浜でない方でも、
老若男女、Ladies & Gentlemen, Boys & Girls !!
誰でも参加OKです。
毎月第3土曜日にmeeting（交流会）を開催しています。

アップルの事や広くIT関連の話題をみんなで共有、情報交換して
月に1度大いに盛り上がりましょう！！

■My Creation by Apple

meetingでApple製品から産まれた「あなたの作品」を披露してみませんか？
iMovieで編集したムービーやiPhoneで撮影した写真集、iPadのアプリで描いた絵画・イラスト、はたまたオリジナルのiOSアプリなど何でもOKです。作成のコツや苦労した事などもお話ししていただけると嬉しいです。

■初めて参加される方へ

初参加の方は・・・、

https://aomac.blogspot.com/
