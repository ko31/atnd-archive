交通アクセス

[市内各所から]

徒歩 : 田中里の前交差点，ローソン向かいです．隣にリカーマウンテンがあります．
叡山電車 : 元田中駅から徒歩3分です．
京阪電車 : 出町柳駅から徒歩15分程度です．少し道がわかりにくいです．
（なお，出町柳駅から歩きたくない方は，叡山電車が利用できます．叡山電車では，一駅だけ乗車して，元田中駅で降りてください．）
バス : 市内各所から，市バス1本で向かうことも出来ます．市バス206系統か3系統を利用できます．バス停 叡電元田中で降りてください．徒歩3分程度です．ただし，3系統の一部は，叡電元田中まで行きません．確認して乗ってください．手前の飛鳥井町まではすべてのバスが行きますので，そこで降りるのも一つの手です．
[遠方から]

JRや近鉄でお越しの場合 : 京都駅をご利用ください．烏丸口 市バスD2乗り場から祇園・清水寺方面の206系統に乗車してください（逆向きに乗らないように注意）．50分程度かかります．お急ぎの場合は，京都駅からJR奈良線で東福寺駅まで行き，京阪電車に乗り換え，出町柳行きにご乗車ください（出町柳からは，上の「京阪電車」をご覧ください）．
阪急でお越しの場合 : 阪急 河原町駅で下車後，京阪 祇園四条駅で京阪電車に乗り換え，出町柳駅までお越しください（出町柳からは，上の「京阪電車」をご覧ください）． バスの場合は，祇園・百万遍方面（田中大久保町経由）の市バス3系統にご乗車の上，バス停 叡電元田中で降りてください．ただし，四条河原町の市バス乗り場は大変複雑な上，3系統の一部は叡電元田中まで行かないので，オススメできません．
場所がよく分からない方は，京阪電車/叡山電車 出町柳駅までは迎えに行くことが出来ます．そこから徒歩で現地まで移動しようと考えています．これを希望する方は，必ず事前に幹事までご連絡ください．

その他

未成年者の飲酒・喫煙はご遠慮ください．車を運転する方の飲酒はご遠慮ください．
飲み物はヱビスビール以外にもウーロン茶等がありますが，なにがあるかは誰にもよく分かっていません．たぶん気分です．
幹事への連絡はTwitter（@highemerly）へのreplyでお願いします．
