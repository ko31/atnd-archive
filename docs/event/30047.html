Network Forensics Puzzle Contest に挑戦

今回の「ネットワーク パケットを読む会(仮)」でも前回に引き続きオンライン上で開催されているパケット解析コンテスト Network Forensics Puzzle Contest のPuzzle に挑戦します。
今回は第2回の出題「Puzzle #2: Ann Skips Bail」の解析を試します。参加される方は、可能な範囲で出されている問題の解析を行ってください。なおこの Puzzle は過去の出題ですので、既に正解が公開されています。勉強会では「どのような手法で解析できるか」を検討していきます。

解析手法とその結果を当日発表できる方は、参加申込時のアンケートでお申し出いただくか、または後からコメントでお知らせください。ぜひふるってご応募ください

以下に、出題の概要を掲載しておきます。

保釈後、Ann Dercover は行方をくらましました。幸にも、彼女が町から居なくなる以前に、調査員は慎重に彼女のネットワーク活動をモニタリングしていました。
「我々は、Ann が居なくなる前に秘密の愛人 Mr.X と連絡を取ったと確信している」と警察署長は語っています。「パケットキャプチャには彼女がどこに行ったかの手掛かりがあるはずだ」
貴方はフォレンジック調査員です。貴方の任務は Ann が何をメールしたか、彼女がどこに行ったかを示し、含まれている証拠を明らかにすることです。

  1. Ann のメール アドレスは?
  2. Ann のメール パスワードは?
  3. Ann の秘密の愛人のメール アドレスは?
  4. Ann が秘密の愛人に持っていくとくるよう伝えた 2 つのアイテムとは?
  5. Ann が秘密の愛人に送信した添付ファイルの名前は?
  6. Ann が秘密の愛人に送信した添付ファイルの MD5sum は?
  7. 彼らのランデヴー場所の都市名と国名は?
  8. 文書に埋め込まれたイメージの MD5sum は?

証拠のパケット (evidence02.pcap) は以下からダウンロードできます
http://forensicscontest.com/contest02/evidence02.pcap
MD5 (evidence02.pcap) = cfac149a49175ac8e89d5b5b5d69bad3

  • 会場にはインターネット アクセス環境がありません。必要な方は自前で G3/LTE/WiMAX などの環境をご用意ください。
