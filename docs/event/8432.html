description

爽秋の候、益々ご清栄のこととお慶び申しあげます。
　さっそくですが、私ども「阿蘇ペンクラブ」は、皆さま方の温かいご支援により、今秋で創立５周年を迎えることができました。ほんとうに、ありがとうございます。つきましては、これまでのご厚誼に感謝し、会員・会友と周囲の支援者の方々の親睦をより深めるため、記念の小宴を開催させていただきます。
　何をするにもよい季節柄、ご多忙のこととは存じますが、阿蘇ならではの秋景も広がっていることであります。どうぞ、ご家族・ご友人・ご同僚お誘い合わせてご参加いただけますよう、ご案内申しあげます。

会次第　開会→代表会員あいさつ→記念講話（熊本保健科学大学学長・小野友道様）→祝電披露→
