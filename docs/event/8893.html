Scalaを遊ぶ!会 第3回参加者募集のおしらせ

Scalaをみんなでいじって遊ぶ会です。
第3回はいよいよ本題、 Play! framework で遊びます。

公式サイトの Starting up the project（Scala version） を教材に、インストールから始めて実際にサンプルのアプリケーションを作りながらPlay!の使い方を学んでいきます。

おそらく１回では終わらないだろうと思うので、このテーマは２回に分けて実施する予定です。

1. 準備について

Scala2.8.0 の実行環境がインストールされたPCを用意してきてください。

さらに 公式サイトトップ からPlay! frameworkの最新版、1.1RC1のバイナリパッケージをダウンロードしておいてください。

なお会場では公衆無線LANが使えます。有料のサービスに加入する必要がありますが、 ヨドバシワイヤレスゲート や Wi2 300 《どちらも月額380円》が安くておすすめです。
また、emobileの電波も入るようです。

2. 会費について

会場は有料施設です。そのため 会費が発生します 。
最小参加者数（5名）の場合が1人1,600円で、それ以降人数が増えるごとに安くなる計算です（第1回は1人1,000円でした）。なお会費にはドリンク1杯分が含まれています。

3. キャンセルについて

有料施設を少人数で使用するため、直前のキャンセルは他の参加者に少なくない金銭的な負担をかけます。
また施設側からもキャンセル料を請求されるため、キャンセルを受け付けるのは基本、前々日までとします。
当日のドタキャンは、発生した損失分をキャンセル料として当人から後日何らかの手段で徴収します。場合によっては前日キャンセルも同様です。
