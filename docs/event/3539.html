お花見したいな、っていうのと

また女子で集まってわいわい喋りたいな、って思っていた折に、

丁度目黒川のお花見クルージングなんていう素敵イベントを見つけたので

ドボク女子お花見会を企画しました。



天王洲アイル駅を降りて、フジテレビ別館の方から乗船して、

目黒川をのぼってゆく工程です。

時間は1時間半の予定。



休日で運行してる日がもう22日しか空いてなかったんでこの日にしました。

そしたら偶然にもカルカルの車両基地ナイトも同じにやるじゃないですか！！すごい！！

というわけでクルーズの後はカルカルへ移動です。



お花見はメールで仮予約をしている段階なので、確定ではないのですが

みんなで楽しみましょう♪♪



ちなみに料金はお一人様３０００円です。

お茶とお菓子が付くそうです。持ち込みは・・・出来るのかな？？



※このお花見クルーズは、@hirataroさんのつぶやきから情報を得ました。





変更！！！！

第一希望で出してた15時半にハズレました（泣）

第二希望の13時半になってしまいました････

15時半なら行けるって事だったお二人ごめんなさい。。

13時半のは14時出航らしんだけど、間に合う？？





杉浦さん、22日はこちらのイベントでお仕事だそうです。。。

↓東京湾横断サバイバルツアー↓

http://boatpeopleassociation.org/project/lob100322

午前中から13時までのイベントがあるから

強行軍は行くといいよ！！
