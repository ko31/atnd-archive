　　わずか4年で1億人の利用者を獲得したTwitter。

　　Twitterを利用したマーケティングは驚くような成功事例が続々と出てきています。

　　しかし、せっかくTwitterを始めてもほとんど活用できていない企業もあります。

　　
　　　　　　　　「今、流行のTwitter活用できていますか？」

　　　　
　　Twitterはソーシャルメディアの１つです。

　　このセミナーでは、
　　ブロガー３０万人を抱え、ブログというソーシャルメディアを
　　活用した口コミ・バズ施策を成功させてきた弊社の経験を踏まえた、

　　成功する【Twitterのマーケティングノウハウ】を惜しみなくご紹介します！

　　今回は、中小企業、個人店舗の方向け。
　　無印良品や、東急ハンズのようなネームバリューのある企業でなくても
　　Twitterをうまく活用すれば、売り上げを上げることができます。

　　その秘訣とは…？
