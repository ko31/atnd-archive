もうすぐ平成も終わりですね。
皆さまいかがお過ごしでしょうか。

今回の学び舎は、
聖マリアンナ医科大学病院 精神療法ストレスケアセンターより
臨床心理士の木下直紀先生にお越しいただきます。

臨床心理士、と聞いてどんな仕事を思い浮かべますか？
「精神分析」、あるいは「心理療法」のとき、
専門家はどんなことを考えながら話をしたり、聞いたりするのでしょうか。

当日は、精神分析の歴史や考え方を簡単にお話ししていただき、
最後に、架空のケースを題材にして、
一人ひとりのクライアントにどうやって臨床心理学を応用していくかを考えます。

もちろん、
notame名物のおいしい晩御飯も楽しんでいってくださいね。
