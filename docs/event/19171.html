description

露出狂の変態の集まりです。

おのおのが持ち寄ったFlash作品をプロジェクターに映して、それを酒の肴に飲む、変態の集まりです。
10年近く前に（細々と、でも確かに）発足した、「京都Flasherの会」をリスペクトした集いです。

どんな集まり？

学生さんも会社員さんもフリーランスさんも、交友関係を広げようと一生懸命に交流会や勉強会やオフ会に思い切って顔を出した結果、せっかくの交流の場なのに緊張するだけして誰が何やってる人なのか全く覚えてない、自分ももちろん何してるのか全然わかってもらえなくって残念な思いをする事が多いよね。
Flasherの会は簡単。
「Flash作品を酒の肴に飲む」会です。
どうせ緊張するなら、自分の作品を見てもらってからにしよう！
上級者初心者、プロアマ、商業作品実験作品、その他ありとあらゆる形態問わずFlash作品を発表してくれる変態を随時募集の会です。

どんなレベルの作品が集まるの？

いくらFlash作品を持ち寄る集まりだと言われたって、初めての参加者にとって、「一体全体どんなレベルの作品が集まるのだろう？自分の作品のレベルは発表のレベルに見合っているのか、心配」という不安は至極自然な事と言えます。
その答えは簡単です。
これは「Flash作品を酒の肴に飲む会」です。
もしレベルが低いとか高いとか、そういう次元で発表作品を評価する人が居たとしたら、その人こそ会にふさわしくありません。
レベルとか関係なく、「Flash作品を酒の肴に、飲む会」なのです 。
ちなみに飲みながら進行するので、レベルがどうこうというより、終始グダグダです。

Flashとか言ってるけど自分Flash全然知らんけど参加して良い？

もちろん良いに決まってます。
むしろ参加してください。
Flasherは（露出狂なのに）シャイなのです。つまりFlashを知らないあなたとの交流を望んでいるとも言えます。
発表する作品も、別にFlashでなくても良いです。そのほうがより、露出狂の会に近いとも言えますよね。

とはいえ最初は様子見で、作品とか無くても参加して良い？

もちろんです。
大歓迎です。
次の会にはきっと作品見せたくなります。
Flasherでない方の参加も絶賛お待ちかねでございます！！！

つまり…、どういうことだってばよ？

プロジェクターに、持ち寄ったいい感じのデジタル作品流すからみんなでいっしょに見ながら飲もうってことだよ言わせんなよ恥ずかしい。

まだ詳細が決まっていなくて参加しにくい。

参加者枠は最大30人まで拡大する予定なので、ゆっくり検討してください。
いちおう、決めていることは、
9月3日（土曜日）19:00〜21:00
大阪梅田あたりで
プロジェクター使える場所で（たぶんカラオケボックス）
やりますよーということだけです。

8月中に参加表明もらえれば凄く有り難いです。

まずは、一人でも参加者居れば決行で！
基本的に最初から最後まで酒飲む会なので、酒飲みなら誰でもウェルカムな形です。Flashとか作品のレベルとかそういうの関係なく。
参加はのんびり待っていますので、是非是非みなさまお誘い合わせの上ご参加を！
