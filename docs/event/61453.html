Photon を使って 「リアルタイム」×「マルチプレイ」を「短時間」で実現

　

　

ワークショップ内容

オンラインゲームを作ってはみたい。

でも「でも何からはじめていいかわからない！」「サーバーを設置するのが手間だし、知識もない。。。」

そんなハードルを一気に解消するのが Photon

今回のワークショップでは、 Unity と　Photon Realtime を使い、この日のワーク時間内で簡単にオンラインゲームを作ってしまいます。

しかもPhoton を使うことで 「 リアルタイム 」×「 マルチプレイ 」 を実現。手間がかからず、簡単で、すぐ始められる。オンラインゲームの新たな構築方法をぜひ体験してください。

　

■持ち物　 ： Unity ( version 4.2 以上 )インストール済みのノートPC ( Windows / Mac )

　　　　　 ： お名刺　1枚
