第1回 Japan.Rを開催します！！

Japan.Rとは

  • Japan.R は、全国各地の R コミュニティが集まって行われる全国規模の勉強会です。
  • 午前は初心者向きのワークショップを行い、午後は R ユーザーの交流を目的とした企画が催されます。
    • 午前の部はこちら

内容

地域コミュニティーの紹介

  • 各地のコミュニティが、どんなことをやっているのかの紹介をします！
  • また、勉強会運営の生の話とか裏話とか、そういうのもアリ！

ジャンル毎のブース

  • 各ジャンル毎にブースを開き、気軽に情報交換出来たり、普段聞けないような話を生で聞けるような場になればと思っています！
