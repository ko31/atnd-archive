description

ホームページ担当者様向けのセミナーです。ホームページ初心者の方からワンランク上のホームページ作成を求める方を対象としています。
ホームページの知識がほとんどないのに会社のホームページ担当になってしまった方、ホームページ制作を外部に発注している会社のホームページ担当者様、起業にあたってホームページをこれから作る方、そんな方におすすめのセミナーです。
堅苦しいものではなく、ドリンクや軽食をとっていただきながら勉強会形式で行なう予定です。ホームページのスキルアップに是非お役立てください。

●第一部 ホームページ制作から運営に必要なスタッフとは？
講師 ： ジャストプレーヤー株式会社 　代表取締役社長 瀧 康史 氏
いざ、会社のホームページを作りはじめようと言っても、最初に何をするべきかなかなか難しいものです。
やり遂げるためには仕事の棚卸しが重要。ホームページ担当者のあなたに、ファーストステップを教えます。
ジャストプレーヤー株式会社 HP ＞http://www.justplayer.co.jp/

●第二部 ホームページ更新システム（WikiPlus）実演
講師 : 株式会社 TOKAI
ホスティングサービス「WikiPlus（ウィキプラス）」のでもサイトを用いて、30分以内にホームページを作成する実演を行ないます。
株式会社TOKAI HP ＞http://biz.tnc.ne.jp/
主催： 株式会社Teable
共催： ジャストプレイヤー株式会社　　株式会社 TOKAI

【 主催者 (株)Ｔｅａｂｌｅ より】
この度、首都圏イノベーションセンターMICAN（http://www.webmican.com）を運営しております、(株)Teableでは、お客様の販路開拓・全国的へ向けての営強化支援の一環としまして、企業様のホームページ作成担当者向けの「ホームページ実践セミナー」を開催することとなりました。
企業や製品のPRにおいて、昨今ホームページがきちんと作られていることも、企業や製品の安全性や信頼性を伝えるのに重要な要素となっております。この機会に是非、自社のホームページを見直し、営業活動に効果的なホームページにしてみませんか。
ホームページの制作を専門で勉強されていない方、企業様の広報ご担当者様、ホームページを自作していてワンランク上のホームページを作りたい方、知識はないけどホームページを作りたいという方など初心者の方が対象となっております。
ホームページ制作のスキルアップにお役立てください。
今回のセミナーは、ＭＩＣＡＮ会員以外の方でも参加可能ですので、興味ある方はどなたでもお問合せください。ただし、定員がございますので、お申込はお早めにお願いいたします。
