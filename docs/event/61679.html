チャイルドアートディレクタートレーニング講座
http://www.geijyutuniyoru.com/

子どもの絵と心を学ぼう。
そして、子どもをよく知ろう、自分を知ろう。
子どもの造形と絵画の指導者を養成する講座です。

絵に表れる子どもの心理を通して、子ども達への接し方を学び、
芸術が子どもの成長にどのような役目を持っているのか理解していきます。
美術指導者を目指す方、指導に困っている方、必見の講座です。
