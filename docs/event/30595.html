2012/08/28 丸の内MongoDB勉強会 #2 隣の人とシャーディングやってみない？ – ATND

概要

ずっとNoSQLが気になってたけど、情報収集する時間が無かった方。
以前からMongoDBに興味はあったけど、触る機会が無かった方。

いっしょに入門してみませんか。

勉強会ではローカル環境にMongoDBをインストールして、実際に動かしてみるハンズオン式で行おうと思っています。

対象

MongoDB初級者

各自準備するもの

・ノートPC
・OSに合わせたMongoDBバイナリ
・emobile、WiMAXなどモバイルネットワーク
（必須ではありませんが、会場に設備はありませんので必要でしたら各自準備をお願いします。）
7/30修正
資料をgithubに公開しておりますので、インターネットにつながる環境をご用意ください。無理な場合は現地でご相談ください。なんとかします。

懇親会について

勉強会後に懇親会を行う予定です。懇親会費は割り勘でお願いします。

会場協力

NRI OpenStandia

7/26追記

会場ですが、丸の内北口ビル9F（野村総合研究所）会議室 10 で行います。

また、当日はRubyを使ってMongoDBを触ってみる予定です。
事前にRubyとmongo-ruby-driverのインストールをお願いします。
ruby download

mongo-ruby-driverはgemが入っていれば、
gem install mongo
でinstallできます。

MongoDBのバイナリのダウンロードも合わせてお願いいたします。
mongodb download

会議室の関係で多数の参加希望者をお断りする形となりまして、申し訳ございません。改めてアナウンスいたしますが、次回は8/20（月）開催予定です。（まだATNDは作っていません）
