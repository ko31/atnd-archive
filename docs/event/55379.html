PostgreSQL Night について / About PostgreSQL Night

PostgreSQL Night は、PostgreSQLのユーザ、サービス提供者、開発者などが、お酒と料理と楽しみながら PostgreSQLについて熱く語るイベントです。今回、初開催となります。

PostgreSQL Night is a casual event for PostgreSQL users, PostgreSQL Service suppliers and PostgreSQL hackers. Let’s talk about PostgreSQL with beer and food!

SRA OSS, Inc. 日本支社 が、企画運営を行っています。
This event is organized by SRA OSS, Inc. Japan.

今回の目玉 / Special Feature

PostgreSQL開発者でロシアのモスクワ大学に所属する Oleg Bartunov 氏が初来日します。Oleg氏は、1996年から PostgreSQL の開発に参加、ロケールサポート、GiST、GIN、SP-GiST、全文検索、半構造化データ(hstore、jsonb)、intarray、pg_trgm、ltree などに貢献しています。 現在は、インデックスシステムである VODK と jsonb の改良に取り組んでいます。

Oleg氏を囲んで、PostgreSQLな夜を楽しみましょう！

同日日中に、Oleg氏が講演する無料セミナーもあります。
よろしければ、 こちら もご参加ください。
