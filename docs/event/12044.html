description

○内容

  • jQueryってよく聞くけど結局コレはなんなの？
  • ライブラリーを落として使うくらいしかできない。どんな風に動いているのか分からないからいじれない。
  • 落としてきたjQueryが動かなくて断念することがある。どうしたら解決できるの？

みたいな、jQueryの入り口でつまずいている人向けにできるだけ分かりやすく解説
また、個別相談やjQueryの活用法についての即席プレゼンなど考えています

○対象
Webデザイナー、マークアップエンジニアで、jQuery初心者、javascriptもよく分かってない人向け
大阪・十三でコワーキングするメンバーが来ると思いますので、同業者とのつながり作りに、すでに使っている方もお気軽にご参加ください。

アクセス
阪急十三駅から会場までの御案内

案内１．
阪急十三駅東口改札を出ます。
ケンタッキーとミスタードーナツの間の商店街を直進します。
商店街をぬけると正面に水交ビルが見えますので右手にある歩道橋を渡ってすぐのビルです。
ビルの看板が見えにくいかもしれませんが１階がバーのビルです。

うっかり「阪急十三西口改札」に出てしまった場合は、すぐ左後ろ側に見える陸橋をくぐると東口改札の方に出ますので後の道中は同じです。

案内２．
１）阪急十三駅東口改札を出ます。
２）ケンタッキーとミスタードーナツの間の商店街を直進します。
３）商店街の途中一つ目の十字路を右手に見るとセブンイレブンと交差点が見えますのでその方向に右折します。
４）交差点を渡り左に直進すると陸橋とビルの看板が見えにくいかもしれませんが１階がバーのビルがすぐにあります、そこが会場のビルです。

案内１．と２．の違いは歩道橋をわたるか交差点を渡るかだけで
道中約５分ほどです。
