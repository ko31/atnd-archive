description

「音」と「香り」をテーマにした展覧会です。音楽家、サウンドアーティストの安井麻人による音と香りのイメージを体験するインスタレーション。匂いのテクニカル・ユニット last note とのコラボレーションにより実現。
展示期間中はイベントとして作家本人によるトークセッションや、レセプションパーティーでは安井自身によるDJも行われます。また関連イベントとして香道イベント（組香）も企画しています。
お気軽にご来廊下さい。
