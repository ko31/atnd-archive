「Android/Javaでペイントアプリ開発」セミナー[6時間マスター / 座学 + ハンズオン] #4

■勉強会内容

第四回の勉強会になります。

Android「ペイントアプリケーション」の作成から完成までの一連の手順を通して、開発の基本を学びます。
※Javaの基礎をおさえている方であれば、スムーズに開発可能です。
※セミナー後に復習ができるよう、開発手順のドキュメントをお渡し致します。

〜 セミナーの学習タイムテーブル 〜

時間 内容
12:00-13:00 •概要
　Androidの概要について（4要素+αについて・シェア・マネタイズなど）
　Eclipseの操作方法（ショートカット・デバッグなど）
　プロジェクト作成方法
　APKファイル作成方法
　セキュリティ関連について
　XMLファイル概要
13:00-18:00 •ペイントアプリ開発
　アプリの仕様について
　描画機能作成
　オプション作成（undo、clear、save）
　　Android特有の機種依存によるファイル保存方法について
　

※時間帯は進捗により前後する可能性もございますので、ご了承ください

※セミナー開講人数は2名となります（セミナー実施の有無は当サイトからご確認下さい）

  • Android開発者

■当日お持ち頂くもの

1.開発環境が整ったPC(Windows/Mac)

下記全て行い環境を構築してお持ち下さい
①JDKのインストール
②Eclipseのインストール
③AndroidSDKのインストール
④EclipseプラグインとしてAndroidSDKインストール、設定
☆重要：Eclipseに取り込むAndroidSDKのバージョンは2.3.3でお願いします。
⑤プロジェクトの動作確認
プロジェクトを作成しエミュレータ(AVD)起動を行い、プロジェクトのデフォルトの「Helloworld」が表示される所まで確認をお願い致します。

各ダウンロードサイトは以下となります。
①JDKのダウンロード
JDKの「Download」よりダウンロード下さい
http://www.oracle.com/technetwork/java/javase/downloads/index.html
②Eclipseのダウンロード
最新の「Eclipse Classic4.2.2」をダウンロード下さい
http://www.eclipse.org/downloads/packages/release/juno/sr2
※他のバージョンでも構いません
③AndroidSDKのダウンロード
「Download the SDK for Windows」よりダウンロード下さい
http://developer.android.com/sdk/index.html
環境構築手順については、「Android 環境 構築」などのキーワードで検索をして頂けると、各サイトで紹介されていますのでご確認ください。
