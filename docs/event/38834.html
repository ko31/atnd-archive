■内容

ウェブ制作の単価が低下する中で、成果が明確に見えるECサイトへの投資意欲は高まる一方です。ウェブ制作会社で、最近になって積極的にECサイトを作り始めたところも多いのではないでしょうか。 
しかしながら、ウェブ制作者の多くはECサイトを非ECのサイトと同じスキームで制作し、失敗しています。ECサイトはネット上でコンバージョンさせるという明確なゴールがあり、それを達成するにはターゲットユーザーの絞り込み、そしてそのターゲットがコンバージョンするためのコンテンツ、サイト設計が必要になります。 
当セミナーでは、ウェブ制作のプロフェッショナルの立場にある方を対象に、以下の内容についてお話しします。

  • 成果の出るサイトとは？ ・ウェブサイトの5階層概念モデル
  • ECサイトと一般的なサイトの設計の違い
  • ニーズとウォンツ（ECサイト4モデル）
  • 成果の出るECサイト制作のワークフロー
  • 事例紹介：近江牛.com
  • 質疑応答
