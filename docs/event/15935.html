description

近畿方言。それは、、近畿地方のうち、兵庫県但馬と京都府丹後西部を除き、福井県嶺南を加えた範囲で用いられる日本語の方言の総称です。
（wikipedia参照）

日本の方言として、近畿方言の認知度と影響力は計り知れないですね。近年では上方落語や芸人の台頭で、ポピュラーな言葉となっています。

そんな知名度の高い近畿方言で語られた絵本は、結構あります。
話は変わりますが、私は東京生まれの東京育ち。方言とは縁遠い環境で子どもから大人までを過ごしました。（もちろん、東京の言葉も方言といえますが、「標準語」という観点からみてそれは除外とします）

そのために、ポピュラーな近畿方言に淡い憧れを感じており、近畿方言の絵本を購入するのです。

しかし、ネイティブのように上手く読めない。そんな負い目から、今回のテーマがうまれました。

近畿方言を巧みに操るネイティブの皆さんのお越しをお待ちしております。もちろん、方言に憧れる東京っ子の皆さんもどうぞお越しください。

私の国の言葉も聞いて！という地方出身の方もお待ちしています。

皆さんで、一般的な近畿方言を身につけられたら、最高だなと思います。
飲みながら近畿方言の絵本を語り合いましょう！

今回は、人数が確定したら場所取りを開始します。つきましては、開催日前の6月3日（金）を締め切りとさせていただきます。場所は、新宿か下北沢の居酒屋を予定しています。
