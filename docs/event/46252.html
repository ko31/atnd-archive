概要

今回のテーマは「HTML5に関わる通信プロトコルたち」です！WebRTC, WebSocket, SPDY, HTTP2など、注目キーワード満載でお届けします。

先日カンファレンスを終えたばかりのため、今回のコンテンツはカンファレンスの再演がメインになりますが、どうかご容赦下さい。ただ、講演者の皆さんは「ただの再演じゃ終わらせない」とおっしゃってますので、ちょっとだけ期待しましょう:-)

  • WebSocket, WebRTC, Socket API, … 最新Webプロトコルの傾向と対策（45分 小松健作さん、カンファレンスの再演）
  • HTTP/2.0がもたらすWebサービスの進化（45分 大津 繁樹さん、 林 達也さん、中島 博敬さん、カンファレンスの再演）
  • 休憩（10分）
  • LT大会（20分）
    • 「きっとこうなる IE12 大予想」（村地 彰さん）
    • 「IEless JavaScript」（辻 亮佑さん）
    • 「見せて貰おうか、最近のweb開発の性能とやらを(for mobile)」（酒巻 瑞穂さん）
    • 「クリエイティブなWebGLでの3D表現（仮）」（竹ヶ原 旭さん）

場所は、年末恒例のソフトバンクさん！クリスマスも間近なこの時期に、素晴らしい夜景を臨めるソフトバンクさんオフィスで、今年最後の勉強会を楽しみましょう！
勉強会中、離席も飲食もOK！併設されている売店／バーで各自ドリンクやおつまみをお買い求め下さい。

会場についての注意事項

今回の会場では、無線LANが使用できません。
なので、講演中のTweetなどはお手持ちのケータイ、スマートフォンで3G回線などを使って行なっていただくことになります。
