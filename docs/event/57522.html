2014年秋から冬の為替マーケットはどう動く？
"FX天気予報士"太田二郎氏が2014年"秋～冬"のＦＸ空模様を鋭く解説します！
彼が今もっとも注目する通貨とは？ドル？ユーロ？ポンド？
そして、為替市場を取り巻く"次なるシナリオ"とは？
為替取引の自由化以前から業界に携わり培ってきた、キャリアと知識が導き出す太田予報士の今年秋から冬にかけてのFX天気図は！？

また、初心者の方にも安心してご参加いただけるよう、為替相場の変動要因や材料、
値動き、トレード手法を解りやすく説明します。

業界歴半世紀以上！！
CXの重鎮がファンダメンタルから見た相場の読み解き方を伝授！
アメリカでは金融緩和終了で、新興国から先進国へ戻る投資資金。
一方、ヨーロッパでは、「ゼロ金利」からついに「マイナス金利」の時代到来！？
日本企業もアベノミクスによって稼ぐ力を取り戻し始めたかに見える現在。

しかし、消費税10％への引き上げや、国民年金の株式の運用比率の引き上げ検討など、
足元ではこれまでの歴史上類を見なかった政策が、次々と行われています。

商品市場を分析して50年。あまたの時代の転換点を目撃してきた古川昭治氏が

今何を考え何に投資するべきか、熱く語ります！！
半世紀もマーケットと向かい合い続けている古川昭治氏の知見は、
これからのあなたの資産運用を考えるうえで、きっとヒントになります。

講師プロフィール

太田二郎氏 外国為替取引の自由化前となる１９７９年来、FXに従事し、
ポジション・テークから始まり、後には法人、企業間とのFXを担当。

FX個人向け取引との関わりは１９９８年と古く、ロイヤルバンクオブスコットランド銀行で、日本の業者向けにマージン取引の営業を開始し、FX個人向け取引のフロンティアとして活躍、２００２年以降は日本、米国のFX業者で、日本の個人投資家向けの営業や、FXストラテジストとして活躍。

現在、ヘッジファンドバンキング㈱のFX担当アナリストとして、
FX取引における個人投資家向けの投資教育を行っている。

古川昭治氏 株式会社商品データ代表取締役。

1962年東京商品新聞入社。17年間の商品業界紙の記者を経て、
1977年に株式会社商品データを設立。国内商品の市況分析を専門とする。
週刊商品データを始め、多数の専門誌に記事を掲載、講演、イベント、コーディネーターなど幅広い分野で活動。
特に商品先物業界の発展に尽力している。
