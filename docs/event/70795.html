アメリカやヨーロッパで話題のダイエットにも効果的でワインのように長期熟成ができるお茶
詳細
アメリカやヨーロッパで今ダイエットや投資対象、またワインのような性質を持つことから大変注目され、多くの資産家やセレブ、中国茶愛好家に人気の「プーアル生茶」。
ワインのように長期熟成することができ、ヴィンテージ物になると大変高額な値段で取引されている高級プーアル生茶。今回は100g3万円もする日本では大変貴重な高級プーアル茶を試飲会にて無料で試飲体験していただけます。
インテリアとしても飾ることができ、好みの発酵度合いで飲んでいただける大変珍しいお茶です。
また、試飲会参加時には無料でお茶菓子も出てきますので、お茶会感覚でお越しください。
プーアル茶専門のティーコンサルタントによる中国茶の淹れ方を試飲の際にその場でお教えいたします。自分自身のスキルアップとして、一生の趣味としても中国茶はこだわりのあるあなたにぴったりです。
お友達と、ご家族と、そしてお一人でもお気軽にご参加ください。
試飲は予約制になりますので、予約希望者は下記の希望時間枠をご記入の上、メールまたはお電話にてご予約ください。
当日の参加も可能です。お越しの前にご連絡ください。
