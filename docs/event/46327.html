なんと開催も3年目となりました、Androidっぽい新年会
今年もAndroid界隈で繋がっていきましょう

Androidアプリ開発等ってまだまだ面白いよね、新年もAndroidっぽい話題で攻めていきましょう、という感じで。
飲もう飲もう！

開催趣旨とか注意とかっぽいもの

  • 元々は2年前に日本Androidの会横浜支部な人の「塩辛で日本酒を飲みたいね～」なんて話から始まった新年会です。
  • お酒も肴も【完全持ち寄り】でやってます。
  • なので、日本酒や食べ物をぜひともお持ち寄りください。美味しいワインなんてのでもいいですね。
  • お正月に実家に帰ったときに買ってきた地酒！なんてのを超期待しています。
  • もちろん旅先で買ったとか近所で買ってるお気に入り、知り合いに貰ったなんてのでも全然OKです。
  • 昨年までと会場が異なります。アットウェア様の新オフィスで【みなとみらい】です。
  • 定例会やハッカソンなどいつも会場をお貸し下さっている 株式会社アットウェア様 ありがとうございます。
  • ただし新会場（会議室）は20人位しか入れないっぽいので早めの登録をお勧めします。
  • 参加可能人数が少ない為、申し訳ありませんがAndroidっぽい方限定でお願いいたします。もちろん横浜在住限定ってわけではありません。ドロイド君っぽい体型の人限定ってわけでもありません。知ってる人が1人でもいるっぽくて日本酒が好きっぽい人は参加するといいと思うよ。
  • お気に入りのぐい呑みまたはグラスを持ってくるとよろしいのではないでしょうか。
  • ちなみにこちらが 昨年のATND と 一昨年のATND です。
