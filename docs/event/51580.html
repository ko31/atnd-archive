第4回生田塾　開催決定！

6月21日（土） 10：00～18：30（9：30開場）

生田塾は、Web業界で働く若者、またWeb業界を目指す若者に向けて、キノトロープ創業者の生田より、
熱いメッセージとノウハウをお届けする1日完結型講座です。

多くの制作者や企業担当者に、高い評価を受けているメソッドを事例を交えて講演します。
インターネット草創期より20年以上、お客さまに選ばれ続ける理由が明らかになります。

セミナー履修者で希望があれば、インターンシップで実践の現場体験も可能です。

制作会社様はもちろん、企業のWeb担当者様、Web制作に興味のある学生の方も是非ご来場ください。

講師プロフィール

株式会社キノトロープ　代表取締役社長　生田昌弘

1985年に生田写真事務所を設立し、カメラマンとして活動を開始する。
1993年12月にキノトロープを設立。一貫した方針で数々のWebソリューションを築き上げる。
現在もネットエバンジェリストとして布教活動を行い、積極的に講演も努める。

著書に「Webブランディング成功の法則55」（翔泳社）、「Webデザインワークフロー」（ソフトバンクパブリッシング）、「CMS構築成功の法則」（技術評論者）、「新・Webデザインワークフロー」（ソフトバンクパブリッシング）、「Webサイト運用成功の法則」（ソフトバンククリエイティブ）、「次世代Webサイト構築ワークフロー」（インプレスジャパン）等。
