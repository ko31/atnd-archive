エンジニアなら知っておきたい　「仮想マシン」のしくみ

長谷川　猛 / Twitter: @hasegaw

最近「クラウド」という概念が広く認知され、その上で様々なWebアプリケーションが動作しています。クラウドは様々な技術の結晶ですが、特にそ の中核をなす物は「仮想マシン技術」だと言えるでしょう。
今日、多くのエンジニアが「VMware」などの仮想化ソフトウェア、「さくらのVPS」などのサービスを利用していると思います。しかし、「ど うして、これら仮想マシンが動くのか」を理解しているアプリケーション開発者は少ないのではないでしょうか？
今回は、Linux標準の仮想マシン技術KVM(Kernel Virtual Machine)を中心に、仮想マシンを実現する様々なテクノロジについて、かいつまんでご紹介します。

■内容（予定）
1. 現在のx86における仮想マシンの方向性
・x86 仮想マシンが提供する基本機能・x86 仮想マシン技術のトレンド概要
2. いまさら聞けない仮想マシン技術のしくみ
・仮想CPU・仮想マシンのメモリ管理・仮想ハードウェア・virtio 仮想デバイス
