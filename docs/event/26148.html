  3. 資産運用に対する講師の信念
  4. 金融業界について（仕事内容や問題意識など）

＜セミナーの魅力＞

金融系のセミナーってそつのない理論だけで終わっちゃうんでしょ？
どうせ講師が自分の成功例を元に後付けの理論をゴタゴタ話すだけでしょ？

いえ、本セミナーは違います！

本セミナーでは、金融業界で活躍する現役のデイトレーダー（兼ファンドマネージャー）とファイナンシャルプランナーという全く対極な2人を講師として迎えます。
そして、この2人の話を一度のセミナーで聞けるのです！
このセミナーに参加すれば、お金や自分の収入、仕事をしていくことについて幅広く学ぶことが出来ます。
しかし、他のセミナーと違って、「これが正しい！」という方法は提供しません！
あなた自身が、グループワークや質疑応答を通して、この2人の話から何が自分にとって正しいかを考えるのです。
その上で、変化の激しい現代に本当に役立つ“自分流”の生き方を見つけてください。
我々はそのためのきっかけを提供します。
また、本セミナーの後には講師陣を囲んだ親睦会を予定しています！
お酒やお食事を楽しみながら、講義や質疑応答の時間に十分理解できなかったことだけでなく、普段疑問に思っていることなどもこの機会を活かして解決していってください！

＜セミナー開催団体について＞

Ex-Lives

「人に教えたい！」、「人に伝えたい！」、「自分の技術や能力を世の中に役立てたい！」という声と、
「こんなことを知りたい」、「あんなことを学びたい」という声を拾い上げ、彼ら/彼女らを結びつけている
これまでに、料理教室や金融セミナーなどの企画をサポートしてきた
