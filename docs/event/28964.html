★ユーストリーム見学会（参観費1,000円／定員10名　要予約）
　　2012年5月25日（金）開場19:30 開演20:00 終了21:45

　また冒チャンネルは、トークコーナーと音楽ライブの2部構成です。
　音楽スタジオを利用せず、池袋にある通常の貸し会議室において
　ビデオカメラ1台で放送を行っています。

　ユーストリーム中継にご興味のある方に、放送開始までの準備の様子、
　中継中のドタバタも含めてご覧いただければと思います。

　特に、個人的にユーストリームを始めたいと考えている初心者の方に
　お勧めです。『ユースト名刺』というビジネスモデルもご案内できま
　すのでお気軽にご参加ください。
