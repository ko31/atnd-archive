= イ ベ ン ト の 特 徴 =

▼1. PCを持ち込んで技術について自由に自己PRが出来る！
このイベントの最大の特徴は、皆さんが主導となり自己PRができることです。
PCを持ち込んだプレゼンを通じて、自身のITスキルやプログラミング経験を
視覚的に企業の担当者にアピールすることができます。研究で構築した
プログラムや趣味で開発したゲームやアプリなどの開発成果物について
紹介することで、より効果的なPRをすることが可能です。
資料を作ってアピールするもよし、熱く語るもよし、最大限にこのイベント
形式を有効活用してください。
また自己PRノウハウにつきましては、アカリクスタッフから事前に
細やかな指導を受けることができます。

▼2. 企業の若手エンジニア、CTOなどの開発者と濃い話が出来る！
通常のイベントでは人事の方が参加することが多いですが、本イベントでは
現場の開発者や技術系執行役員（CTO）、採用責任者といった企業の
担当者と1対1でお会いいただきます。
最先端の技術やサービス開発の話を含め、通常の説明会では得ることの
できない貴重な情報が手に入ります。

▼3. プレゼンが上手くなる！
1企業との面談時間は30分となり、うち冒頭の5分間で自己PRの
ライトニングトークを行っていただきます。
残りの25分間は、企業から事業内容に関する説明や皆さんへの質疑、
また皆さんから企業担当者への質疑に使っていただくことが可能です。
企業担当者に対して1日で数回のプレゼンを行うため、その場で
フィードバックがもらうことができますので、イベントに参加しながら
自然にプレゼンの魅せ方が身につくはず。そうなれば、今後の面接試験
対策にもつながっていくでしょう。
まずは自身を存分に語りつくしてください。

皆様のご参加を心よりお待ちしております！！
