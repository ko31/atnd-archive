大分マッピングパーティー 2011/9

event_id 18892
title 大分マッピングパーティー 2011/9
catch みんなで作る地図「OpenStreetMap」に触れてみよう！
description

OpenStreetMap上に地図を描く方法を経験者とともに体験するイベントです。
経験者・未経験者を問わず歓迎致します。ペンさえあれば参加できますのでお気軽にお申込みください。

1.主催
大分県オープンソースソフトウェア研究会
（同日午前中に開催される
OpenStreetMap講演
との連動企画です。講演にもぜひご参加ください）

2.日時、集合場所
9/10(土)13時00分
NS大分ビル1F（大分市）

雨天決行（歩ける程度なら）

3.募集人員
・12名(目安ですのでこれを超えても申し込めます)

4.タイムスケジュール（当日の状況により多少変わる可能性があります）
13:00 集合(NS大分ビル)
13:00～13:30 ガイダンス
13:30～14:30 街歩き、マップ情報収集
14:30 再集合(ハイパーネットワーク社会研究所 会議室)
14:30～15:30 マップ編集
15:30 解散

5.持参するもの
＜必須＞
・筆記具
＜任意＞
・デジカメ
・スマートフォン
・GPSロガー
・モバイルなPC

6.費用
・参加費無料
・交通費等は各自でご負担ください。

7.目標
・どういうものを地図に描くか（マッピングオブジェクト）の理解
・OSM上に点(ノード)を打ったり線（ウェイ）を引いたりできる

8.当日までの事前準備
＜必須＞
・事前にOpenStreetMapアカウントを作成しておいてください。（無料）
手順はこちらのビデオを参照
＜任意（可能な方のみで構いません）＞
・以下の関連情報をご一読ください。

10.マッピング（マップの描き方）の例
例えばコンビニをOpenStreetMapで表現する場合には以下のように描きます。
1)まずコンビニの位置にノード（点）を打ちます。
2)そのノードに以下のようなタグ（属性を表わすキーと値の組み合わせ）を書き込みます。
shop=convenience
（コンビニ）
name=FamilyMart （名称）
