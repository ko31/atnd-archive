・背景

Origamiのオフィスにお招きして、働いている人はどんな人間で、何を考え、どんな環境でサービスを創っているのかを少しでも知ってもらいたいという思いから、開催を決定しました。


・ 開催概要

お酒とお食事をご用意させていただきます。

カジュアルなホームパーティーのようなイメージで気軽にご参加ください。

オフィス見学だけや、働いている人の話を聞くだけでも全然かまいせんので、お気軽にお立ち寄りいただければと思います。何時から参加して頂いても、ご参加可能です。ぜひ、遊びにくる感覚でお越しください。

 

日時: 2015/2/26(Thu) 18:30 – 21:00  

場所: 東京都港区南青山7-8-14 ( Origami Inc. )

参加費: 無料

 

  

またOrigamiでは、一緒にサービスをグロースしてくれるメンバーを募集しています。

ぜひ、ご興味ある方はご参加ください。
