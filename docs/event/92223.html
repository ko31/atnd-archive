UX dubについて

UX dubは、UXデザインに縁のある方々が、会社の垣根を越えて集まる知識と人脈の交流の場です。
日々の業務の中で行うサービスデザインや、事業の成長に役立てていただくことを目的としたイベントです。

UX dub #2では、UXデザインコンサルティング専門会社のえそら合同会社とコラボし、UXデザインのプロセスを用いたアイディア発想ワークショップを開催します。

イベント概要

サービスやプロダクトのデザインを変えることで満足していませんか？
わたしたちは、表面的なデザインを変えることだけではなく、ユーザーを理解し、共感し、体験をデザインすることがUXデザインであると考えます。

今回は、ワークショップでは削られてしまいがちな「ユーザーを知り・共感する」部分から、アイディアの発想までを体験していただきます。
ここで体験していただいたUXデザインのプロセスはきっと、普段の業務でいかしていただけるはずです。

ワークショップ終了後には懇親会を開きます。 素敵なフィンガーフード等の軽食、ハートランドのビール等のお飲み物をご用意する予定です＊

ワークショップについて

ワークショップではリアルな場面でのUXデザインを考えます。
お題は「図書館の課題を解決するアイディア発想」。
現在、図書館では民間企業とコラボレーションする等、様々な施策が行われています。
既に読書習慣のあるユーザーの図書館の利用者数を増やすべく、「本を読む」という体験をリデザインします。

こんな方におすすめ

・UXデザインのプロセスを体験してみたい方
・事業運営に携わっている方
・プロジェクトマネージャーの方
・事業に悩みがある方
