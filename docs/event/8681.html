アパレル、テキスタイル業界関係者必須！！
素材＆カラーを中心に、ファッショントレンドの理解を 深めていただくセミナー。
なかなか、取材が、難しいと、いわれるプルミエール・ヴィジョンの取材、撮影歴10年近くという
韓明世による9月に、開催されたプルミエール・ヴィジョンをはじめパリ取材での臨場感溢れる
インタヴュー＆バラエティ豊かな写真、映像などを使い、わかりやすく解説いたします。
世界で、もっとも需要で、シーズン最初に開催されるプルミエール・ヴィジョンのレアな情報を
中心に、201１/12年 秋冬期の提案トレンドを　いち早く紹介、早めの期日設定でのセミナー開催。
来るべきシーズンのアパレル～商品企画やバイイングのために活用していただけるセミナーです。
ぜひ皆様のご参加 賜わりますよう、ご案内申しあげます。
