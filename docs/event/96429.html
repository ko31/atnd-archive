「みんなのラズパイコンテスト2018」に応募しようぜ！！

「みんなのラズパイコンテスト2018」に応募したい人向けの勉強会です。
応募に関してや、RaspberryPi、電子工作、ネタ作りなどに関する情報交換を行います。

えれくら！名義で応募した作品、その時の作品の作り方、資料なども紹介いたします。
「すごく応募したいのに、何をどうしたらええのか皆目見当がつかんのじゃー！！」という方も、ぜひお越しくださいませ！

ご用意いただくもの

基本的には「応募に必要な材料をお持ちください」ということになりますが、大抵は以下のようなものが材料として必要になります：

  • Raspberry Pi各モデル
    RaspberryPiには現在、モデルが数種類あります。応募するために必要な機能を持ったRaspberryPiをご用意ください。
  • SD/Micro SDカード
    RaspberryPiは、一部のモデルを除き、SDカードにOSをインストールして使います。お持ちのRaspberryPiに合ったSDカードをご用意ください。
  • ノートPC
    ノートPCをお持ちくださいませ。RaspberryPiのOSをインストールしたり、デバッグに使ったりと、いろいろなことに使います。
  • その他工作材料と道具
    電子工作と組み合わせる方は、部品やニッパー、はんだごてなど。
    その他（たとえばケースなど）の工作をする方は、板、ハサミ、カッターなど。

コンテストに興味があって、応募してみたいけど現状何も持ってない、 という方でもご参加いただいてかまいません。
当日、各種材料・道具などの紹介と貸し出しを行います。
