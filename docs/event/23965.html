内容

2012年最初の技術ひろばの勉強会は MS 安納さん、SQLAzureJP さんを迎えての開催です。ハードウェアから開発系、IT Pro 系と広いジャンルのテーマを集めました。得意な内容も不慣れなジャンルもみんなで楽しみながら視野を広げましょう。

  • 「AD FS を極めよう～要求規則言語 講座」 （MS 安納さん）
    AD FS はもうご存知ですよね。AD FSを使いこむと必ず思うこと。それは「もっと複雑な要求規則は書けないのだろうか？」。
    大丈夫です。書けます。複雑な要求規則を記述するには、要求規則言語と呼ばれる専用のスクリプトを使用します。と言っても、WEBには情報が極端に少ない！ということで、本セッションでは、要求規則言語を使用したカスタム要求規則の作り方について解説します。あまりにもニッチすぎてMS開催のセミナーでは絶対に聞けません。お待ちしております。
  • 「knockout.jsとASP.NET MVC で始めるMVVMパターン」 （Honda Tetsuyuki さん）
    今まではSilverlightの実装で使われていたMVVMパターンがknockout.jsを使うことによりHTML、JavaScriptのみを使用するWebアプリでも実装できるようになりました。
    今回はASP.NET MVC 4 とknockout.jsを使ったMVVMパターンの初心者向け解説をしてみたいと思います。
  • 「ＡＲＭ時代の基礎」 （西澤さん）
    気がつくと、いつの間にやら世間に浸透し、今後も長い付き合いになりそうなＡＲＭ。
    その戦略やそこから見えてくること等を、ゲーム屋の視点を入れつつ語ります。
  • 「SQL Server 2012 (仮)」 （SQLAzureJP さん）
    （詳細未定）
