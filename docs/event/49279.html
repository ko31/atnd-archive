description

あなたの「今月」の目標は何ですか？
「今週」、そして「今日」は？

この朝活は、描いたものを得るために、目標達成の習慣形成を通して、仲間と共に自己成長する為の会です。
目標・計画達成に向け、ＰＤＣＡサイクルを廻す為のエッセンスを実践していきます。

Plan 計画（一週間の目標を立てる）
Do 実行（行動・実践する）
Check 評価（振り返り、進歩確認）
Act 改善（参加者全員で改善案を考え再実行する）

会社や他人から言われている目標だけでは無く、自ら目標設定し、持っている能力と可能性を最大限に発揮して道を切り開き、達成する楽しみを体感しながら自己成長していく。
そんな『成功のサイクル』を身につけ、生涯を通して人生をより楽しいものにしてもらう事が目的です。

意思を形にしたいけれど、具体的にどうすれば良いか判らない方。
志を持つ人と触れ合い、刺激を受けたい方。
そして今持つ目標を何としても達成したい方。

貴方の力を貸してください。
本気のアドバイスを貴方の力にしてください。

日時

4月5日（土）　（毎週土曜日）
第一部　9時～10時30分
第二部　10時30分～12時
第三部　12時～13時
（御参加は第一部のみ等、部分参加でも結構です。また毎週の参加が難しくとも、来れる週のみの参加で問題ありません）

進行

第一部
毎週、今週あった良いこと（Good&New）と、先週の目標とその達成度合い及び翌週の目標をシェアしてもらいます。
もし目標を立てていなければ、参加を機に立ててもらっています。

第二部
毎月第一週は、長期目標を定める為に、五年後の自分になりきってトーク。
第二週以降は、交流会か参加者や招待者による無料セミナー。
毎月最終週は、翌月の目標設定を致します。

第三部
ランチ会。
お昼の時間です。食事は活力。
ご一緒に美味しいランチは如何でしょうか？

費用

500円（ドリンク料込み。ただしランチ費用は実費）

参加資格

夢や目標を具現化したい方。夢や目標を作りたい方。
20～30代の方（それ以上の年代の方は紹介のみ）
保険、宗教、ネットワークの方はご遠慮ください。

こんな方が参加されています。
会社経営者、政治家志望、エンジニア、薬剤師、商社マン、看護師、会社事務、ストレッチインストラクター、ネイリスト等。

ザ・タワー・グランディア

右側に目的地のタワーマンションの入り口がございます。
