統計解析・データマイニングの方法論を用い、蓄積されたデータを有効活用していく方法を学ぶ勉強会です。
今回は、機械学習・解析・セマンティックウェブ祭りです！

※今回も、USTREAM配信を行います。詳細は本ATND、#TokyoWebmining タグでアナウンスを行います。
※当日、迷ったり、入れないときは#TokyoWebminingだけでなく #nifty_engineer や @nifty_engineer を追加してツイートするとニフティさんが助けてくれます。


目的：
データマイニング・統計解析の手法を用い、蓄積されたデータを有効活用していく方法、および、WEB上に蓄積されたデータの活用に関し、皆で学び共有していくことを目的としています。
※詳細:
　・「データマイニング+WEB 勉強会＠東京」紹介@Tsukuba.R #7 (技術評論社さん撮影の動画つき)
　・Slideshare: 創設の思い・目的・進行方針

概要：
各データマイニング・統計解析の手法、各種WEB APIの活用に関し、基礎的な内容から学べる場にしたいと思います。
各方法論・関連トピックについて、分野の初心者方でも分かりやすい内容を意図し、進めていきます。
