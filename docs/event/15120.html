こんにちは！sumyappです。
相変わらず変なアプリかしょぼいアプリか、そんなのしか作らない私です。

いろいろなアプリに感化されて、アプリの企画を考えてみました。
女の子がみんなを元気にする写真集iPhoneアプリです。
その名も、「ねっ、デートしよ！」

一言で言うと、
・いろんな素人な可愛い普通な女の子の街中での姿を集めた写真集
・「デートに誘う文言」が一言付いている
って感じのものです。
「デートに誘う文言」が一言付いている だけでわくわくすると俺は思います。
街中で「あー、あの女の子が声掛けてくれたらな＞＜」って思うことありませんか？

つまりこれはしゃいな男心をくすぐる、逆ナンされる的な展開を妄想出来るアプリなのです。

アプリ自体はもう出来上がっているんですが、写真がまったくありません。
なにせ、私女の子の知り合いとかいないんで。

なので、 4/22(土)に渋谷でいろんな方に声をかけさせていただき、
写真を撮らせて頂いて、その場で作って、
後日プロの方にPhotoshopで写真を見やすくして頂いて、
AppStoreにて公開します。

有償公開の予定ですが、アプリの売上による収益は全額、日本赤十字社に寄付させて頂く予定です。
(※少なくても7月末までの収益は全額、以降、面倒になってやめるかもしれませんが。。。)

というわけで、手伝って頂ける方を超募集しております！
具体的には以下のようなメンバー！
・写真を撮らせてくれる女の子
・カメラマン等、撮影に協力頂ける方
・女の子に声をかけて頂ける有志
結構長く頑張る(心が折れなければ)いる予定ですので、
ぜひご興味があれば sumyapp@gmail.com
もしくは http://twitter.com/sumyapp までぜひ！
当日急な参加でもOKです！お願い致します！
