＜「相続」ってどんなことをしたらいいの？実例を紹介しながら丁寧に解説します！＞

「相続問題」は、みなさん耳にしたことがあるかと思います。
そして「相続」は、突然やってくる場合がほとんどです。

相続のうち「不動産相続」は、相続開始を知った日（被相続人が死亡した日）から10ヶ月以内に相続税の納税をしなければなりません。同時に煩雑な手続きも数多く生じます。

スムーズに手続きが進められることが望ましいですが、もしかしたら親族間で問題が発生する可能性も…？

今回のセミナーでは、不動産相続に関する基礎知識をはじめ、下記にリストアップしたような生前から相続発生後までの疑問点に対する対策を解説。
実際の事例も紹介しながら、わかりやすくご説明します。

・相続税の基礎控除について知りたい
・相続した不動産を、売却するためにはどのような手続きが必要？
・遺産分割協議をする際に気を付けるべきこととは？
・代償分割ってなに？ 物納はできるの？
・相続対策における税務上の注意点とは？

身近な人たちとのトラブルを極力なくし、スムーズで円滑な相続を行うための「事前準備」について、しっかり学んでおきましょう。

本セミナーは、不動産を中心とした相続の事前準備およびサポートに特化した資格「公認不動産コンサルティングマスター（宅建士の上位資格）」をもつ、不動産のエキスパートが丁寧に解説します。
