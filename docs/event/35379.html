● 非エンジニアな人のためのWebセキュリティの基本（進藤龍之介）

クラウドの普及やWebを表現する技術の多様化などにより、Webコンテンツはより複合的な技術の集合体となり、これにともなうセキュリティの懸案も多くみかけられるようになりました。
この講演では、WebシステムやWebサイトの製作において知っておかなければいけないセキュリティの基礎知識について、サーバー・インフラからコンテンツに至るまでわかりやすく解説したいと思います。
時間があれば（資料作成が間に合えば）、XSS（クロスサイトスクリプティング）などの脆弱性の実証などもサンプルを作ってお見せしたり、ワークショップやディスカッションみたいなこともしたいと思います。
非エンジニア対象ですので技術的な知識は不要です。
