2009.3.13追記：
多数の方の申し込みありがとうございます。54名なら入れると思いますので、補欠の方もご都合がよろしければ是非お越しください。お会いできることを楽しみにしています。

　Android SDK WG とは、日本Androidの会（ http://www.android-group.jp/）のワーキンググループのうちのひとつで、Android SDKの理解を深めながら、開発者同士仲良くなろうぜ！を目的としたフランクな集いです。定期的に週末の午後を使って「セッション」と言う名の｛勉強会｜交流会｝を開催しています。
　Android SDKについての会話をしようという共通のマインドがあれば、サンデープログラマー、上級者、iPhoneプログラマー、どなたでも参加可能です。ぜひお気軽にご参加ください！


内容：

　日本Androidの会のメンバーで、「Software Design 2009年 3月号（技術評論社さん刊行）」の特集記事「Android アプリ開発「匠たくみ」への道」を執筆しました。この特集の「4章：Androidアプリを作ってみよう！～サンプルゲーム／マッシュアップ／拡張現実感～」には、いくつかのサンプルアプリが掲載されています。皆でこれらのサンプルアプリのコードを読んでみませんか。

サンプルアプリの種類
・サンプルゲーム「ソート中毒」
・お天気情報アプリ
・Android ＋ AR（拡張現実感）で広がる現実世界

　サンプルの作成者も何名か参加します。不明点に関する質問や、紙幅の都合上削らざるをえなかった話、実際のアプリで応用する上でのポイントやノウハウなどを聞くことも可能です。

　また、SD誌のコードリーディングだけじゃつまんないから何か話をさせろと言う方がいらっしゃいましたら、是非コメント欄などでお知らせ下さい！！


事前準備・注意など：


　Android SDK WGは、日本Androidの会のワーキンググループですので、参加される方は日本Androidの会のメンバーになっておいて下さい。Google Groupsに参加するだけです。以下から登録出来ます。
