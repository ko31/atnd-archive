学生のみなさま！

東北太平洋沖地震 被災地支援のため、一緒に献血しましょう!!!!!!!!!

＜400L for Hope＞

ぼくたちは、東北太平洋沖地震に際し、被災地復興のために学生1,000人の献血と義援金100万円を募るプロジェクトを進めています。　

・目的

東北太平洋沖地震の被災者のために、「学生だからこそ」できるアクションを起こす。

・目標

学生1,000人 の 400ml献血　合計 400L
合計100万円の義援金集め

・期間

2011年3月15日(火)　〜　2011年5月31日(火)

・関連リンク

公式サイト http://www.littleshotaro.com/400LforHope/index.html
公式Twitterアカウント @400LforHope
公式facebookページ http://www.facebook.com/400LforHope

＜献血ツアー＞

400L for Hope［学生1,000人献血キャンペーン］のイベントです。
今回の震災に際して「なにかしたい！」と思っている学生のみなさま。ぜひこの機会に一緒に献血することで、被災地のあしたの希望につなげましょう！！

日時：3/28(月) 10:30-12:30

集合：am10:15 紀伊國屋書店 新宿本店 前

新宿東口献血センター近くです！

参加対象：学生

大学生でしたらどなたでも！お友達をお誘いあわせのうえ、ぜひぜひお気軽にご参加ください(^-^)/！

内容：学生20名で献血センターに献血しにいきます！

団体予約をするので、スムーズに献血を行えます。説明/検査/献血/休憩を含め、予定終了時刻は12:30となっています。

ピクニック：献血終了後、代々木公園に行きましょう！笑

献血の終わった後、時間に余裕のある方で、ぜひお昼ごはんをかねて代々木公園へ行きピクニックしましょう！地震の影響であらゆるイベントが中止になってしまっていますが、気分転換にぜひこちらもご参加ください(o^^o)
