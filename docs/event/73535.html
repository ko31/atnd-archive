内容

Single Page ApplicationにおけるPreloadJSの活用事例

リクルートマーケティングパートナーズ 山田 直樹
リクルートが2015年11月30日にリリースした『英語サプリWeb版』は、これまでのリクルートのサービスには類を見ない大規模かつ本格的なSPA (Single Page Application)です。当アプリでは、大量の画像や音声といったメディアファイルを動的かつスムーズに読み込み管理するための仕組みとしてPreloadJSが使われています。

今回は、そんなSPAがネイティブアプリに劣らないユーザー体験を提供するためにPreloadJSをどのように活用しているのか、その仕組みをご紹介します。

事例から見るCreateJSの使いみち

世路庵 (ceroan) 沖 良矢
「CreateJSの概要は知っているけど、現場での使いみちを知りたい！」という方のために、プロモーションサイトを中心とした様々なプロジェクトで実際にCreateJSを活用した事例を紹介しながら、そのメリットと使いこなしのポイントについてお話しします。

EaselJS 0.8.2の改訂項目とgskinner labの作例「Arc Rainbow」

野中 文雄
まず、2015年11月26日付ビルドのEaselJS 0.8.2で改訂されたおもな項目を簡単にご紹介します。つぎに、gskinnerチームの習作が公開されている Lab から「 Arc Rainbow 」を採り上げ、その中で用いられているGraphicコマンドなどの技術について解説します。

CreateJSで開発するクロスプラットフォームなアプリ開発〜ElectronとPhoneGapで広がる可能性〜

ICS 鹿野 壮 / 野原 のぞみ
昨今のWeb技術の発展により、HTML5製のコンテンツをデスクトップやモバイル向けにアプリとしてリリースできる環境が整ってきています。今回はAdobe Illustratorのようにイラストを描くツールをCreateJSで開発し、デスクトップとモバイルのそれぞれに向けてElectronとPhoneGapでアプリケーション化を行った実例を紹介します。
