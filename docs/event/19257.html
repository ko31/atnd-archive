第02回 本当のバックアップってものを見せてやる

台風真っ盛りですが、開催します。

今回の ストレージ友の会 では、グループディスカッションを通じて、本来、バックアップはどのようにあるべきか、実は今まで考えていたバックアップは、そもそもバックアップじゃないのではないか、みなさんで話し合っていきたいと考えています。

できるだけ奇抜で、コストの安いソリューションが出てくることを期待しています。

「でも、あたし、具体的なソリューション作れるようなスキルないし・・・」なんてことは考えないでください! 知識あるメンバーとともに一緒に考えることがスキルの向上につながります。むしろ、こんな良い機会はありません。是非、手に手を取り合って、ご参加くださいますようお願い申し上げます。

バックアップへの要求事項

  • 24/365 連続稼働予定のデータベースサーバー
  • 3TB 以上のデータ蓄積があり、年率 20% 程度の伸張率でデータが増えていきます。
  • ディスクは専用装置
  • 0時を静止点とするデータがバックアップには必要です
  • 毎日、テープへのバックアップが必要です
  • 常にオンライン業務で利用しており、バックアップ取得時に極限まで I/O への負荷がかからないようにデザインする必要があります

なお、こちらは「 一般枠 」となっております。女性･学生の方は 女性･学生枠 へ登録をお願いいたします。
当日はニフティさんへ名刺を提示することになります。必ず、名刺に Twitter ID を付記してお持ちください。
