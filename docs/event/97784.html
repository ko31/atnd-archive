description

世界で最も優れたアイディアは、私たちの日常生活、ハマってるゲーム、好きなキャラクターなどに
ヒントがあるかもしれないです。
「アイディアがあるけど、どう実現するのがわからない」、
そのアイディアを少しでも実現できるようにお手伝いします。
「どこからスタートすればいいかわからない」
ちょうど良かった、ここからですよ。

オプションA: プログラム知識がなくてもドラッグ＆ドロップ
（ビジュアルスクリプティング）で簡単なモバイルゲームの作り方を教えます。
オプションB: 大ヒットパズルゲームのくるくるパズルの部分の作り方を教えます。（プログラミング知識が必要）
オプションC: 集まって自分のやりたいことをする、もしくは自習します。
※A〜Cのいずれかを選べます
プログラミングをしている方もいれば、それ以外の方（イラストレーターさん、サウンドクリエーターさん、ライターさん）も大歓迎です。
※ゲームエンジンはUnityです。
