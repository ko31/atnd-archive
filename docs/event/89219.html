２人の大学生が訪問したネパールとバングラデシュの現場の話を聞いて語る集い

アジア保健研修所（ＡＨＩ）でインターンをした大学生が、その後、ＡＨＩの研修生を訪ねて現地へ飛び立ちました。
国際協力に関心を持つ２人が、ネパールとバングラデシュのNGOの活動現場で、インターンとして見てきたこと、感じたこと、今考えていることを語ります。　国際協力に関心のある学生の皆さん集まれ！　一緒にお話しましょう！！

話し手

  • 明坂裕子さん…大学４年生。2015年８月~2016年２月、マイクロクレジットについて理解を深めるため、バングラデシュのNGO、Jagorani ChakraFoundationへ。
  • 福井ゆたかさん…大学４年生。2017年２月~３月、心のケアを必要とする人たちのための活動に関心があり、ネパールのNGO、Kopila Nepalへ。
