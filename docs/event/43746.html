9月21日から23日まで米国で開催のMySQL Connectで発表された最新情報をご紹介！

MySQL Connectに参加された方から、イベントの雰囲気や面白かったセッションなどをご紹介いただく予定です。

紹介予定の内容

  • MySQL 5.7 DMR 2 ( ダウンロード )
    MySQL 5.7 DMR：準同期レプリケーションに新しいオプション追加、InnoDBのトランザクション処理性能の大幅な向上、一時表にInnoDB利用
  • マルチソース レプリケーション ( Labリリース )
    複数のマスターのデータを集約、準同期レプリケーションおよびマルチスレッドスレーブにも対応
  • Fabric ( Labリリース )
    シャーディング(データ分割)による拡張性の向上、Fabricが分割キーと構成を管理してコネクタはその情報をキャッシュ、キーに応じて該当キーを持つMySQLを選択
  • MySQL JSON UDF – libmy_json_udf ( Labリリース )
    基調講演では触れられずにひっそりとLabsのサイトに掲載されたUDF (ユーザ定義関数)。MySQLに格納された JSON ドキュメントを検索または更新できるSQL関数群。
