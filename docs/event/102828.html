第一回スマブラオフ「クリスマス？なにそれおいしいの？杯」

クリスマスに集まってスマブラ大会を開催したいと思います！

ロータスに集まって最強を決める熱い戦いが始まる(　ﾟдﾟ)

優勝賞品も用意していますので、ぜひぜひ参加して下さいね！

＊主催たかちゃんが本体ｘ２台＋GCコントローラー８つ＋マルチタップを用意しています。モニターは会場のLOTUSに３枚借りています！

対戦卓を３卓程立てたいので、本体＋コントローラー、マルチタップの持込が可能な方はＤＭかリプで連絡を頂けると助かります！

GAME BAR LOTUS

トーナメントの他にも、サブイベントやフリー対戦などで一日中遊べたらと思っています。

swicth版のスマブラを買ったけど、一緒に集まってわいわいしたい方はぜひ集まって遊びましょう！
