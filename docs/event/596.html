まずはじめに、よくある誤解について。
どうも「激しい運動を伴うので、遠慮します」という声をお聞きするのですが、このイベントの趣旨はあくまで「技術者同士の会話」です。
すると、「走っている間に、喋る余裕があるのか？」と反論されますが、これまで経過を見ている限り気がついたら喋っていたというケースが殆どです。
走ると言っても、ジョギングですのでペースはかなりゆっくりです。
自分の体力に自身がない。しかし、技術に関しては語りたいことが山ほどある方。これはあなたのためのイベントです！


えーと…、本題に入ります。
なんと、今回は国民の休日の朝９時からの開催です！！


今回のお題は「Webアプリケーション・フレームワーク」です。
基本的なトピックスですが、意外と今までの参加者内でも、どのフレームワークを使ったことがあるか、又は興味があるかはまだ話したことがありません。
実際の現場ではレガシー・コードの保守をしていると言う方でも、どうすればこの先改善出来るのか、など話せるネタはあるかと思います。
我々の会話の中に、未来のフレームワークはきっと見つかるでしょう。勿論、フレームワーク否定派という立場であっても歓迎致します。
さあ、Let’s Jog!!!


※イベントの概要については後述の「Jogについて」をご一読ください。

[ 公式タグ ]：Jog (Jog系のイベントが隆盛してきたらoopjogにするかも？？)

[ はてなグループについて ]：まだまだ未整備ですが、近々Jog関係の連絡や情報集約のため、はてなグループを利用していこうと思っています。
参加に登録は不要ですが、ご興味があればこちら
に気軽に登録どうぞ。

[ 参加資格 ]：オブジェクト指向言語の使用経験がある方、もしくは使用経験が無くても興味の有る方

[ 走行コース ]：こちらをご覧下さい。（@takiuchi 再利用させて頂きます。）

[ 前回走行記録 ]：6.52km, 41分48秒（平均1km/6分24秒)


[ 集合場所 ]：東京メトロ日比谷駅A10番出口から出て、ナチュラルローソン、ロッカーがある付近、又は出口を出てすぐの交番前あたり(開始時間が近くなった場合移動しています)。
遅れた場合はある程度待ちますが、走り出している場合は皇居を反時計周りに走り、走者と合流することになっています。

[ 準備運動 ]：日比谷駅A10出口を出て日比谷公園に入ってすぐ、ゴミ箱が近いベンチの辺りで最低20分前くらいから準備運動を行うつもりです。

[ 持ち物 ]：基本的にジャージなどの服装、ランニングシューズ等は各自ご用意ください。着替え場所として駅構内のトイレを使用していますが、着替えたまま現地にいらっしゃる人も多いです。
また、走行後、体を冷やすといけないので汗を拭くタオル等を持参することをオススメします。
持ち物は駅構内ロッカーに預けることになりますので、財布／携帯等の貴重品を持参したい場合は、携帯可能なポーチやショルダーバッグのご利用を検討してください。


[ 名刺交換について ]：集合後、軽く自己紹介を行う時間を設けたいと思います。その場でpokenや名刺などを用いた名刺交換を行ってください。

[ 設備利用について ]：今回はアートスポーツ　ランニングオアシス日比谷を利用し、走行後シャワーを借りようと思っております。詳細はサイトをご覧下さい。

[ 懇親会について ]：健康を目的としたイベントでもあるので、走行後の公式な居酒屋などでの懇親会は予定しておりません。
しかし、今回は休日の朝からの開催ということで、「Jogらしい」懇親会を予定しております！予算は決めていませんが、2000円程度持っていればシャワーも含めて十分かと思います。

[ Jogについて ]：takiuchiさん発祥の企画です。MerbJogからフォークして定期化したイベントという位置づけです。
詳しくはtakiuchiさんの勉強Jogのすすめ – s21g::Hello, world!や、私のOOPJog#1完走！！などのエントリをご覧下さい。
