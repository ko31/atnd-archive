募集要項(良く読んでください)

Androidのセキュリティを勉強する会 では、ライトニングトーク(LT)の発表者を募集します。

  • 持ち時間は5分です。
  • 応募者は、コメント欄に発表のタイトルと概要を書いてください。
  • 応募の中から、発表内容をみて主催者側で発表者を選考します。
  • 選考結果は8/31までにATNDのメッセージでお知らせします。
  • 当日の連絡用に、ATNDのメッセージで主催者の携帯番号をお送りします。ATNDに必ずメールアドレスを登録してください。メールアドレスの登録のない方は選考できません。
  • 申し訳ありませんが、会場の定員の関係で、当日10:00amからの「Androidのセキュリティを勉強する会(9/3)」に応募され既に定員に入っている方以外の方の席はありません。
  • 発表者の控え室は用意します。また、終了後の懇親会にはご参加いただけます。
  • 発表者は14:30までに現地入りお願いします。発表は15:00からです。
