◆◇概要◆◇
このセミナーは参加者さん同士でキャッシュフローゲームを行います。
ゲームを通じて初めの一歩を踏み出したい人同士の交流を深めつつ、お金の知識と感性を磨いてもらうことを目的としております。

◆◇キャッシュフローゲームとは◆◇
キャッシュフローゲームとは『金持ち父さん貧乏父さん』の著者であるロバート・キヨサキさんが考案したゲームです。
本ゲーム会を通じて実際にゲームを体験し、参加者どうしでの気づきをシェアします。

◆◇どんな人がくるの？◆◇
これまで参加された方は学生からサラリーマン、主婦の方まで様々です。
特に職業、性別にゲーム上の優劣はございません。
進行中、ルールや不明点は講師がその都度丁寧に説明します。
もちろん、初めての方も安心してご参加頂けます。

◆◇ゲーム会の流れ◆◇
１．講師によるゲームの説明
２．参加者同士で自己紹介
３．ゲーム開始 途中で一度休憩を挟み、気づきをシェア
４．ゲーム終了 参加者同士による気づきのシェア、講師まとめ

◆◇ゲーム会を通じて得られるもの◆◇
◾︎お金の流れがわかる
◾︎何に投資をすればいいのかわかる
◾︎お金持ちが考えていることがわかる
☆ゲーム会の最後に、ゲームの中で得た気付きと現実をつなげるための大事な【3つの価値】をお伝えします！
そのうちの1つは…【環境】
あなたはどうして「日本語を自由自在に話す事ができる」のでしょうか？
誰もが日本語を話す環境に身を置いているからです。
もし、アメリカに生まれていたらどうでしょうか？
…残りの2つは当日ご説明致します！

知っているだけでゲーム会の効果が10倍違いますよ・・・_v＾-＾

◆◇講師から◆◇
希望通りの会社に入れた！
ところが現実は思った以上に厳しい…。
早朝出社、深夜残業する上司の姿を見て、
「このままではいけない！」「でもどうしたらいいんだ！」

そんな風に思っていませんか？
大きな決断や行動は必要ありません。
《ヨチヨチ歩きの一歩を踏む出す》だけでいいんです！

まずは今のあなたの「お金の使い方」について見直してみませんか？
そんな機会を提供しているのがこのキャッシュフローゲーム会！
毎回好評を頂き今では『満足度No.１』です！
ゲーム会後は講師と無料で個別面談も可能です。
ぜひ一度お試しあれ！！
