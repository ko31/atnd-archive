description

IS03はauが発売する、初めてフューチャーフォンの日本独自機能（おサイフケータイ、ワンセグ、赤外線通信）を搭載したAndroidスマートフォンです。
今回はそのIS03のタッチ＆トライを中心に、Androidスマートフォンの可能性についてのセミナー、トークセッションも開催します。

今までスマートフォンに興味があっても、なんだか難しそうで躊躇していた方も、お気楽に参加ください。
Androidアプリ開発者、既にスマートフォンをお持ちの方の実機確認も大歓迎です。

「未来に行くなら、アンドロイドを待て。」
そこまで言われるんだったら、実際どんなものか体験してみましょう。

今回はKDDIからIS03の担当の方もいらっしゃいますので、IS03に関する質問も出来る時間を設けています。

内容
IS03の特徴とau ONE Marketについて
IS03が与えるスマートフォン業界への衝撃
IS03タッチ＆トライ
Androidスマートフォンの可能性についてのトークセッション
