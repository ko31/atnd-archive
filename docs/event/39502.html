title はじめてのデザイン講座
catch イラストレーター・フォトショップに触れてみよう！
description

未経験〜初心者向けのイラストレーター・フォトショップセミナーを開催します。
グラフィックソフトに興味がある方、デザインの勉強したい方、
デザインのお仕事就きたい方など、どうぞお気軽にご参加ください！

■こんな方が対象者

  • デザインに興味があり、グラフィックソフトに触ってみたい方。
  • デザインの仕事に興味がある方。
  • フォトショップで写真加工がしてみたい方。
  • パワーポイントより綺麗に資料を仕上げたいと思っている方。
  • 自社でデザインを行い、広告の外注費を抑えたい事業主の方。
  • デザイナーとのコミュニケーションを円滑にしたい営業の方。

    • ■セミナー内容

  • イラストレーターの基本操作〜図形の作成
  • フォトショップの基本操作〜色調補正
  • 簡単なポストカードの作成
  • コワーキング施設の利用が無料（3ヶ月分）になるオンデマンド教材のご紹介

    • ■セミナー費用

    500円（税込）

    ■注意事項

    ハンズオン形式：Adobe イラストレーター・フォトショップがインストールされたノートPC（Windows/Macどちらも可能です）をお持ちください。お持ちでない方は体験版をインストールしてご参加ください。

    ■その他

    ※学生、個人の方もぜひご参加ください。
    ※簡単なアンケートをお配りしますのでご協力をお願い致します。
    ※道に迷ったり、入り口が分からない場合は、03-5213-4567までご連絡ください。
