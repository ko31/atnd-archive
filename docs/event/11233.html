概要

ARアプリが簡単に作れる「ウキウキView Web API」を使ったハッカソンです。
デベロッパやデザイナの方々で集まって
AndroidとかHTML5とかARとか、わいわいと面白いものを作りましょう。
自分で作成したアイコンや画像をコンテンツとしてAR(拡張現実)で表現したり、
画面レイアウトを工夫したりも面白そうなのでデザイナさんも大歓迎です。

誰でも簡単にARを作れるこのAPIを使って、
一緒にARの世界を作って見ませんか？

準備物

・ノートPCなど開発用の道具
・APIキーの取得と準備
電源、ネット環境は会場側で準備いたします。
