ディベート初心者を対象にした、ディベートの基本を学ぶ1日講座です。

○ディベートとは
ディベートとは、一つのテーマについて賛成と反対に分かれ、第三者である審判を説得する競技です。
明確なルールの下、審判を説得するという目的を持って議論することにより、以下のような様々な能力を身につけることが出来ます。
・論理的思考力
・物事を多面的に捉える力
・説得力
・傾聴力（相手の話を聴く力）

○こんな方におすすめです
・自分の考えを相手にうまく伝えられない方
・「何を言っているかわからない」と言われたことがある方
・議論、反論する力を身につけたい方
