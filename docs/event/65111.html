◆春は何かを始めるキッカケになる季節です。今までと違った自分を見つけ出すために、今までと違った働き方を目指すために、自分自身の心の中でモヤモヤと抱き続けている想いを形にしてみましょう。「ありがとう」と言われる働き方を考えているあなたに、大きなヒントになる体験授業です。

◆仕事を通じて社会貢献のできる、ソーシャルビジネスと呼ばれる分野をご存知ですか？例えば、子育てと仕事の両立が難しいお母さんのために、病児保育という分野で支援をする団体。個人事業主や主婦がなかなか健康診断を受診できないことを改善した団体。発展途上国の貧困問題を教育の視点で変革する事業。など、さまざまな社会のお困りごとを解決し、今までに無いあたり前を作りだすことで、受益者から「ありがとう」と言われる働き方に取組まれる方が多くいらっしゃいます。そんな、自分らしい働き方や生き方、困っている人たちのために何かをしたいと考え、行動するあなたの姿を想像してみてください。こんなに素敵なことはありませんよね。

社会起業大学の体験授業では、そんな働き方をする人たちの事例を含め、どのようにすればそのような働き方が出来るのか。自分の想いは特別なことではないということを実感できるものとなっています。

やりたい！やってみたい！と思っていても、なかなか踏み出せずにいる方も多いかと思います。でも、どうしようかな？と迷っているなら、まずは行動してみることが大事です。行動しなければ現状は変わらず、何も始まりません。

どなたでも無料でご参加頂けますので、まずは一歩、踏み出してみるところから始めてみませんか？
