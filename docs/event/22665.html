概要

第3回スマホアプリHands Onは、
Windows Phone 7 アプリの開発入門講座です。

今回は、日本マイクロソフト株式会社より講師の方をお招きした、
Windows Phoneアプリ開発講座を開催します！
第1回、第2回と同様に、「スマートフォンのアプリを作ってみたい」
という学生の方、ぜひご参加ください！

第1回、第2回の様子はこちら

Windows Phone は、ゼロベースでの開発が行われた、
すべてが新しく生まれ変わった最新型のスマートフォン OS です。
誰もが簡単に使いこなせる「メトロデザイン」やWindows Live、Facebook などの SNS連携機能等が主な特徴です。

Windows Phoneに関する情報はこちら

開発講座では、Windows Phone 7 の概要から、アプリ開発の基本と流れ、
上級者向けのヒントまでをカバーします。
また、今回学生向けの講座を実施するにあたり、下記3つの特典があります！

  • Windows Phoneマーケットプレースへの登録に必要な、
    App Hub年間のメンバーシップ料金は 9,800 円 (税込) が、”無料”になる DreamSparkを、もれなくプレゼント
  • Windows Phone 7 の実機端末を用意します。
    Windows Phone 7 をお持ちでない方も、実際に端末に触れながらアプリをテストできます。
  • Macユーザの方もご参加可能です。
    対応策をご用意していますので、Macで参加希望の方は Facebookイベント から参加申し込みの上、ウォールにコメントください。別途ご案内いたします。

スマホアプリ開発に興味のある方、ぜひご参加ください！
