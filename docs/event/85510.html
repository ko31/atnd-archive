企業紹介：株式会社キュービック

株式会社キュービックは「ヒト・オリエンテッドなデジタルマーケティングでみんなの明日が変わるキッカケを生み出し続ける」をミッションとし、比較サイトを中心とする自社メディア事業や成果報酬型の集客代行事業を手がけています。 デジタルだからこそ、データだけでなくフィールドワークをベースとしてニーズを掘り下げ、真にユーザーへ寄り添うことを重視しています。ユーザーにもクライアントにも「目に見える価値」を提供し続けるテクノロジーを磨き、今後とも事業を拡大してまいります。

社長紹介:世一英仁（よいちひでひと）

1981年、埼玉県生まれ。東京大学法学部卒業後、弁護士を目指して司法試験に打ち込むかたわら、ひとりで事業をスタート。2006年には司法試験に区切りをつけて事業を法人化し、株式会社キュービックを設立。代表取締役CEOに就任。現在に至る。

人事紹介：森實泰司（もりざねだいじ）

リクルートエージェント（現 リクルートキャリア）で採用コンサルタントに重視し、インターネット業界を中心に、様々な業界の採用課題に向き合う。求職者の深層心理に向き合ったアプローチで功績をあげ、多数MVPを受賞。 2016年4月よりキュービックにジョインし、HRチームの立ち上げや戦略的な社内制度の設計など経営課題の解決に従事。ジョイン後3ヶ月で約30名の採用を成功させる。

企業紹介：株式会社アイタンクジャパン

2007年より、長期実践型インターンシップ求人サイト「キャリアバイト」を運営。サービス開始10年目に突入し、インターンシップ導入支援企業は1,700社以上。圧倒的な集客力と導入コンサルティングに定評があり、日本最大級の支援実績を誇ります。2015年に人材大手のエン・ジャパン株式会社のグループとなり、新卒採用領域にもサービスを拡大中。

社長紹介：野村勇（のむらいさむ）

1981年大阪生まれ。関西大学卒業後、エン・ジャパン株式会社に入社。大阪支社の立ち上げから参画し、500社以上の採用支援に携わる。リーダー、マネージャーを経て、2013年にグループ会社である株式会社シーベースの取締役に就任。経営戦略立案、商品企画、人事企画、評価制度の構築など多岐にわたる業務を経験。2016年、同じくグループ会社である株式会社アイタンクジャパンの代表取締役社長に就任。『長期インターンシップを文化にすべく奔走中』
