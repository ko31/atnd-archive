実務におけるWebデザインのTips for Illustrator

description

ワンランク質感を上げ、フロントエンドとのやり取りをより円滑に。

IllustratorというとDTPやグラフィックデザイン出身の方が、以前と同じ環境での制作方法でデザインするため敬遠されがちではありますが、現在のマルチデバイスに対応するWeb制作の実情に非常にマッチしたツールになりつつあります。

Illustratorは基本的にベクターデータを扱うため、モバイルやタブレットを中心とした解像度の幅が広い世界での制作に適しています。
また、グラフィックパーツの作り方もCSSでの表現に似ている部分が少なからずあり、先人達の挑戦により複雑高度に見えるものでもCSSでの描画が可能になってきています。

小原氏にはWebのフロントエンドとの連携や、デザインの概念部分をメインに、鈴木氏にはアプリのUI制作に通ずる話しをメインにお話いただきます。

前述したようにIllustratorは現在のWebの実情にマッチはしているものの、フロントエンドエンジニアにとって、AiデータはPSDと比較して扱いづらいデータ形式です。正しく、効率的なデータのやり取り、管理方法も今回はお話させていただきます。

Web Developer Selection（WDS）とは

Web Developer Selection（WDS）は、「数多くある技術から、Web Developerが最適な知識や技術をセレクション。セッション+ワークショップで提供する、とことん実践主義にこだわった技術力向上を目指す場」です。
「レスポンシブWebデザインの実践ワークフロー」でDesign Spiceの小川さんや、「FireworksユーザーのためのPhotoshop講座」をStocker.jpのなつきさんに、また、「CSSレイアウトの理解と実践」ではKOJIKA17のこじかさんにお話頂いております。

講師プロフィール

小原 司 氏

フリーランス、SAUCER（ソーサー）として活動後、2013年4月より、株式会社ピクセルグリッドにアートディレクターとして参加。
Webデザインだけでなく、CSSやJavaScriptなど複数のスキルを習得しており、最終的なフロントエンドの形を意識したデザインを得意とする。

slideshareで公開されている「ノンデザイナーのための配色理論」は小原氏によるもの。
http://www.slideshare.net/saucerjp/ss-14902681

鈴木 ともひろ 氏

その卓越したIllustratorの技術力で日本最年少でDribbbleへ招待される。
質感の描画力、Illustratorへの理解は群を抜いており、情報設計にも明るい。
現在は株式会社ナナメウエのUIデザイナーとして勤務している。
