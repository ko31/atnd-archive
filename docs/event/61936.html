2015年スタートに、iOSアプリを作ってみませんか?!

本セミナーは未経験者や初心者の方でもiOSアプリ開発を体験することができます。初心者の方でもベテラン講師がアプリ作成をレクチャー・サポートするのでご安心ください。

スマートフォンアプリ市場は、国内外において最も成長している分野です。このビジネスチャンスを逃さず、最大限に生かしていくためにアプリ開発に関する知識やスキル習得は必須のものとなっています。

スマホアプリを作る楽しさを体感することで"やってみよう””試してみよう””自分でも学んでみよう”といった最初の一歩を踏み出しましょう。

＜iPhone無料体験コース＞

下記の開発機器を御持参頂くことが条件になります。
受講に必要なMacやiPhone(以下iOS端末)は受講生ご自身でご用意下さい。
※代々木本校では、教室に備え付けのMacをご利用頂けます。
1、MacBook (Air・Proどちらでも可)
※Xcodeを使って開発するには、Mac OS X v10.7.3 Lion以上を搭載したIntelベースのMacが必要です。
※最新のOSにバージョンアップして頂く必要があります。
→ OSのバージョン確認方法
2、iPhone または iPod touch または iPad
※iOSのバージョンはなるべく5.0以上でお願いします。最新版はiOS 7です。
3、iOS Developer Programへの登録
※年会費(8,400円/年)がかかります。
