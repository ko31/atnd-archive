クラウド電話API Twilioのハンズオンイベント
Twilio API 勉強会 Vol.7を開催致します。

■ Twilio API 勉強会？
クラウド電話 API Twilioが好きなかたもしくは興味をもっている方エンジニアの皆さんが集まるイベントです。
初心者の方はもちろん既にTwilioを使っている方々、Twilioをテーマで情報交換ができればと思います。
内容も難しくないため初心者の方々も気軽に参加できます。
今まで考えたことのない「電話(SMS)」の可能性を共有できればと思います。

■ 今回の勉強会では？
簡単なフォーム(WEB)を作って画面のボタン操作で電話をかける（Outbound）部分を試してみたいと思います。
サンプルコードは準備しますが、みなさん独自のものでも構いません。Twilioと繋ぐ部分を中心に一緒に試してみたいと思います。
・各自のアカウントを利用して電話の受信 – 約30分
・簡単なフォームから電話の発信 – 約1時間30分
・その他、TwilioのQueue機能を使った通話待ちシステムも試して見たいと思います。
※ イベントで使うサンプルコードや環境は主催側で準備しますのでお気軽に参加してください。
　もちろんみなさんの環境やコードでも問題ございません！
