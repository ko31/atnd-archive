■「ELITES(エリーツ)」および「Webエンジニアコース」について


「ELITES（エリーツ）」とは、本気でプログラミングを学びたい人向けの「実務経験カリキュラム有り」の中期型プログラミングスクールです。当スクール「Webエンジニアコース」においては、インターネット上のWebサービスを作れるエンジニアになるための必要な知識を全て学べ、 実践経験を積むことができるカリキュラムを提供しております（学習内容： HTML / CSS / Bootstrap / JavaScript / jQuery / git / PHP / CakePHP / MySQL / Linux(CentOS) / Apache / Vagrant）。又、ELITESは実務での通用力を最重視しているため、他のプログラミングスクールでありがちな「お勉強」レベルのカリキュラムとは一線を画するクオリティを確保しております。 　 　



■「春のWマック祭」で「Macの壁」と「空腹の壁」を打破


弊社は日々「プログラミング学習を検討している方」と面談を重ねておりますが、「Macを持っていないのですが、Windowsでもプログラミングはできますか？」という質問に数多く遭遇しております。又、「Macを買った方が良いということは分かっているが、新たに購入するのには躊躇してしまう」という方が多いことも認識しております。
そこで弊社では、この「Macの壁」ともいうべきハードルを打破すること、イコール、プログラミング学習を始める大きな後押しになるのではないか？との仮説を抱くに至りました。加えて、プログラミング学習では、高度の集中力を要するため大量のカロリーを消費します。そして、空腹になることで集中が切れてしまう、いわゆる「空腹の壁」によって、勉強のリズムが阻害されてしまう現象も多数確認しております。


これらプログラミング学習者に対する「Macの壁」と「空腹の壁」への直線的ソリューション。並びに、これから本気でプログラミングを学ぶ方への弊社からの応援の気持ちを込めて、このたび「春のWマック祭」開催することを決定いたしました。本キャンペーンに応募いただくことで、「学習用の新品MacBookAir」と「マクドナルドハンバーガー365個分相当のマックカード」を進呈いたします。
