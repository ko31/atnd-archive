“内定直結型イベント”として毎年大好評の【アカリクITイベント】を
今年もついに第一弾として、12月17日(土)に東京にて開催します！

このイベントは11月26日(土)にアカリクが開催する「IT企業交流イベント」
のような企業説明会の形式（企業がブースを構えて学生が回る）とは異なり、
プログラミング経験のある学生がブースを構え、【1対1】で自身を最大限に
アピールすることができる【逆求人形式】のイベントです。

▼【逆求人形式とは】
・事前に作成いただくプロフィールシートをもとに、企業が学生を指名します。
・企業の採用担当者やエンジニアが皆さんのブースを個別に訪問してきます。
・訪問企業に対し自分をプレゼンテーションし、最大限にアピールできる
　イベントです。

= イ ベ ン ト の 特 徴 =

▼1. PCを持ち込んで技術について自由に自己PRが出来る！
このイベントの最大の特徴は、皆さんが主導となり自己PRが出来ることです。
PCを持ち込んだプレゼンを通じて、自身のITスキルやプログラミング経験を
視覚的に企業の担当者にアピールすることが出来ます。研究で構築した
プログラムや趣味で開発したゲームやアプリなどの開発成果物について
紹介することで、より効果的なPRをすることが出来るでしょう。
資料を作ってアピールするもよし、熱く語るもよし、最大限にこのイベント
形式を有効活用してください。

▼2. 企業の若手エンジニア、CTOなどの開発者と濃い話が出来る！
通常のイベントでは人事の方が参加することが多いですが、本イベントでは
現場の開発者や技術系執行役員（CTO）、採用責任者といった企業の
担当者と一対一で会うことが出来ます。
新らしい技術やサービスの話を含め、通常の説明会では得ることの出来ない
情報が手に入ります。

▼3. ITエンジニアを大切にする価値ある企業と出会える！
一般広告媒体にあまり求人を出さない企業や就職イベントに出ていない企業も
参加しますので、今回しか会えない企業もあるかもしれません。

▼4. プレゼンが上手くなる！
1企業との面談時間は30分となり、冒頭5分間で自己PRを進めていただきます。
残りの25分間は、企業から事業内容に関する説明や皆さんへの質疑、また皆さん
から企業担当者への質疑に使っていただくことが出来ます。企業担当者に対して
1日で数回のプレゼンを行いますので、その場でフィードバックがもらうことが
出来、次第にプレゼンの魅せ方が身につくはず。そうなれば、今後の面接試験
対策にもつながっていくでしょう。まずは自身を存分に語りつくしてください。

皆さまのご参加を心よりお待ちしております！！
