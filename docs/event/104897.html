5/25(土)クリエイター MEETUP in 池袋

【クリエイター交流会を開催します！】※現在参加者17名


この交流会はクリエイターやアーティストの繋がりを作り、
情報交換やビジネス発展の場として活用して頂く為に開催しています。

フリーランスの方を中心に会社員・学生さんも含め、
「人脈を増やしたい」・「ビジネスパートナーを探したい」と
思っている方も是非参加してください。


【参加資格】

職種は問いません。自分がクリエイティブな仕事だと思えば参加OKです


【主にこのような方々】

デザイナー、webデザイナー、イラストレーター、カメラマン、ライター、
エンジニア、プログラマー、 画家、ミュージシャン、工業デザイナー、
劇団員、作曲・作詞家、工芸作家、出版系、映像系、ゲーム関係、
アパレル関連、セラピスト、コーディネーター、学生、ets…☆


【この交流会のスタイル】

●初めての方でも参加しやすく、堅苦しく無い交流会を目指しています。
服装も普段着のままでOKです。気軽にご参加ください♪
●立食形式にて、カフェスペースをお貸しいただきました「CAFE SELECT」様
より、日本で唯一味わうことができる拘りのコーヒーや珍しいナチュラル食材を
使ったお食事もご提供いただきます。お酒等もご用意いたします。


【交流会の特徴】


●開催時間は2.5時間。その間にマッチングや企画提案等はいたしませんので
自由にコミニュケーションをとり、沢山の繋がりを作ってください。

●ご自身をアピールできるチラシ等がありましたら、ご持参ください。

 ●勉強会やセミナー、仕事案件等も随時発信していきますので、
ご参加時にその都度ご案内させていただきます。

作品・宣伝チラシ等でA4サイズまでであれば、お預かりさせていただき
次回以降の開催時にも展示いたします。ご本人様が交流会に参加されていなくても
アピールさせていただきますので、ご希望の方はお声お掛けください。


【詳細】

【開催日時】5月25日（土曜）18:00〜20:30（受付開始17:50）
　　　　　　※開催時間内であれば、入・退場自由です。(参加費は変わりません)
【参加人数】約20名（※別サイトでもイベント案内を掲載している
　　　　　　関係上、早めに応募を締め切る場合がございます。）
【参加費】　3,000円(前払い) (※ドリンク・食事込み)
【開催場所】CAFE SELECT 
【住所】　　東京都豊島区東池袋3-11-9 1F
【アクセス】池袋駅東口から徒歩10分、
　　　　　　東京メトロ有楽町線東池袋駅から徒歩5分、
　　　　　　池袋サンシャインシティ、
　　　　　　東急ハンズ池袋店から徒歩2分。
　　　　　　乙女ロードと春日通りの東池袋3丁目交差点角地にある
　　　　　　明るく開放的なロースタリーカフェです。
【服装】　　自由な格好(普段着)でOKです。


【注意事項】

・フロアは"完全禁煙"ですので、指定の喫煙スペースにてお願いします。
・ネットワークビジネス、宗教、保険などの"勧誘行為”は禁止いたします。
・キャンセルの場合は必ず3日前までにご連絡お願いします。
・3日前を過ぎてのキャンセルは返金不可となりますのでご了承ください。
・無断キャンセルにつきましては、今後参加をお断りさせていただく場合も
　ございますのでご容赦ください。


【 交流会の趣旨】

クリエイター MEETUPは交流会を通して、ただ「横の繋がりを作る」事だけを
目的とするのではなく、交流会の中から自立した発展を
遂げられるようにしたいと思っています。クリエイターさんそれぞれの
個性や技術を活かし企画を考え、小さな事からでも実行に移し発信していく。
ビジネスを受け身だけでなく、自分達で作り上げる事を互いにしていければと思います。
