catch 腐食のメカニズムを詳解し、損傷事例から防食技術や耐食材料の選定法を学ぶ！
description

［講座のポイント］
　金属材料の腐食とは何か、基礎を理解した上で、実際に遭遇する様々な腐食現象について原因を解明するにはどうしたらよいか、どんな腐食防止法があるかを学ぶ。ステンレス鋼や銅合金の耐食性の本質を理解した上で、使い方を学ぶ。

［習得できる知識］
金属腐食の機構の理解、金属材料の局部腐食、ステンレス鋼や銅合金など耐食材料の
選定法、および防食技術の考え方を学び、腐食問題の解決法を習得する。

［プログラム］
１．金属腐食の基礎知識
　1-1.金属腐食へのアプローチ
　1-2.腐食の電池作用及び局部電池
　1-3.表面皮膜の役割と不働態皮膜

２．金属腐食の腐食形態
　2-1.金属材料の局部腐食（異種金属接触腐食、微生物腐食、通気差腐食、電食）
　2-2.炭素鋼、ステンレス鋼、銅及び銅合金、アルミニウム合金の特性
　2-3.ステンレス鋼の不働態特性と局部腐食（応力腐食割れ、孔食、すき間腐食）

３．腐食性環境とは何か
　3-1.水質と腐食性因子（溶存酸素、塩化物イオン、遊離炭酸）
　3-2.スケール生成と条件

４．腐食防止法
　4-1.塗装と塗膜
　4-2.カソード防食と適用
　4-3.インヒビターの種類と機能
