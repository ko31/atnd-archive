内容

ゲームエンジン・アーキテクチャ読書会

第1回は、ゲーム製作に必要な基本的な用語や技術や数学などを学ぼうということで、「ゲームエンジン・アーキテクチャ」の読書会をやります。なお、読書会ですが、初回は解説を中心にやろうと考えているので、本は持っていなくても参加可能です。

今回は以下の章を読んで、7章で紹介されている「ポンゲーム」の JavaScript での実装とソースコードの解説をやります。

  • 第1章 イントロダクション
  • 第3章 ゲームのためのソフトウェアエンジニアリングの基本
  • 第7章 ゲームループとリアルタイムシミュレーション

上記以外のこの章が読みたい、解説したいという方はこのページのコメント欄に書いていただくか、twitter で@hakoberaまで mention または DM してください。
