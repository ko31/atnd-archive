description

神戸のiPhoneアプリ開発の勉強会です。

当日、会場代実費(500円)を集金します。
参加にあたって、Xcodeのインストールをお願いします。
MacAppStore : Xcodeのサイトへ

会場情報

会場最寄りのコンビニは地下鉄県庁前駅近くのセブン-イレブンです。必要な物がある方は事前に買ってきてください。会場にも自販機、コーヒーサービスがあります。
会場にはFREESPOTが設置されています。FREESPOTの登録をすると一日じゅう使うことができます。
プロジェクター、変換アダプタ（VGA ←→ディスプレイポート、VGA←→ライトニングコネクタ）は準備しているので、困ってることや発表でプロジェクターを使えます。

場所

生田文化会館　第5会議室
住所:兵庫県神戸市中央区中山手通６−１−４０

タイムスケジュール

13:00〜 　発表：byようにゃん
・最近作ったもの
・iPhone6/6+について

14:00〜　フリートーク・フリー質問・フリー開発
発表が終わったら、参加者同士で情報交換する時間にします。アプリ開発で、困っていること・悩んでいることなどを共有して解決して行きましょう。

懇親会

みんなで飲みながら、iOS談義しましょう。会場は元町駅周辺ですが、当日決めます。予算は3000円程度です。
