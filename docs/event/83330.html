description

「デジタル×てしごと」 をテーマに革製品を制作しているクリエイター、じろうさんを講師としてお招きし、上質な栃木レザーに想い想いのデザインを彫刻。世界に一つの本格レザーカップスリーブを作ります。大切な人へのクリスマスギフトにもぴったり！
​
【つくるもの】

上質な革である栃木レザーを使ってオリジナルカップスリーブをつくります。お好きなモチーフで、イラスト、文字を自由に描いていただき、データを作成し、レーザーカッターで加工します。革の色はキャメル、赤、オレンジ、緑の4色からお選びいただけます。

【ワークショップ内容】
​
参加者の皆様それぞれのお好きなイラストや文字をテンプレートに合わせて、グラフィックデザインソフト「Illustrator」か手書きで作成していただきます。手書きのものはスキャンしてデータ化します。
※データ作成はスタッフがサポートいたしますので、Illustratorの使用経験のない方でも安心してご参加いただけます。

データができたら、レーザーカッターを使って革に彫刻、加工をします。その後、コバと呼ばれる切断面を磨く作業を行い、ボタン金具をつけて完成です。

【体験できること】
​
革を使ったものづくりとデジタルなものづくりを体験できます。栃木レザーのこと、革のものづくりの流れ、レーザーカッターの使い方を学べます。
