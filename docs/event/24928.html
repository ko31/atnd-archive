Android Application Award 2012
h3. A3アイディア部門に応募しようアイディアソン！

と言うことで今回は開発者に限らず
ユーザーサイドからの意見を集うユーザーグループ枠を設けました。

  • 開発者グループ（アプリ開発経験者
  • ユーザーグループ（スマホユーザー、これからユーザー
    の2チームに分かれます。

ユーザーグループは

日ごろ使っているスマホにあったらいいなと思う機能やまだスマホ使っていないけど
こんな機能があったら是非使ってみたいなどの意見をどんどん出してもらいます。

スタッフ(チュ－ター)が参加ユーザー皆さんの意見を纏めのお手伝いをし
グループ企画として応募いたします、途中経過などは勉強会G＋ページで公開します。
また個別での応募を希望される方はお申し出ください。

日ごろ溜まっている鬱憤をここでぶつけましょう！

開発者グループの方はもくもくと作業お願いします。
アプリ/Webサービス部門への応募も歓迎です！（既にマーケット出展のアプリも可。
また、ユーザーグループのアイディアをアプリにしてくださる方大歓迎です。

初心者から超ベテラン、ユーザーの方まで参加大歓迎です!
アイデアに自信がある人、技術に自信がある人、デザインができる人、
絵や音楽ができる人、とにかく役に立ちたい人、とにかく何か作りたい人…
お待ちしています。どうぞふるってご参加ください。
