FxOSコードリーディングミートアップ＃１３

　第１３回目のFxOSコードリーディングミートアップを開催します。
　今回もゆるふわやります。はじめてでも安心！
　Fx0も出たことなので通ぶった常連さんを困らせるぐらい質問しまくりましょう！

開催概要

　いつものとおりゆるふわとソースコード読みます。
　読みたいところを読みます。聞きたいことを聞きます。話したいことを話します。
　Fx0もでたことなのでいっぱい質問しちゃいましょう！

懇親会について

　途中でスタッフが飲み物と揚げ物を買いに行きます。
　一口1000円ぐらいでカンパ募ります。

FxOSコードリーディングって？

　Firefox OS (FxOS) のソースコードを読んで何かしら言い合ったり発信しあったりするグループです。

　大体毎月 Mozilla オフィスでミートアップという名のオフ会をしています。
　半年に一回ぐらいは休日開催でセミナーやりたいなー。

　当コミュニティではGMOインターネット様よりコミュニティ支援としてConoHa VPSの提供を受けてます。
　ConoHaについてはこちら→http://www.conoha.jp/community
