中部クリエイティブ業界フェスタカンファレンス2016 - GAIRA関連企業様によるポートフォリオ指導・添削（デザイナー及びアーティスト向け）

GAIRA（中部ゲーム産学協議会）は、愛知県、名古屋市の後援（予定）を受けて下記の日程でクリエイティブ業界フェスタを開催します。今年のクリエイティブ業界フェスタは、中部圏のデジタルアニメーション、ゲームなどデジタルコンテンツ制作を行う学生そして一般のクリエイターに向けて技術カンファレンスを実施します。

タイムテーブル　セミナールーム2(20名)

時間 イベント内容 内容
10:30 - 12:20 GAIRA関連企業様によるポートフォリオ指導・添削 指導・添削

ポートフォリオ指導・添削指導

デザイナ及びアーティスト向け作品指導・添削指導を行います。作品を持参してください。 GAIRA関連企業様から直接作品の指導や添削を受けることができます。
・１人当たりの指導・添削時間は1０分です。
・来場された方に整理券を配布します。

ご連絡

事前申込者の人数が少ないので、時間内ならいつでも良いので、会場に作品持ってきてください。 混雑している場合も、事前申し込みしていることを伝えてもらえれば、優先的に行います。
