世の中は街コン流行り、ならば開発コンがあってもいいのでは？　2月9日！御茶ノ水でHTML5の勉強会を開きます！

御茶ノ水駅から徒歩３分のとても交通の便のよく、近くに明治大学もあるアカデミックな環境の中、無線LANも使えるとても綺麗な教室でHTML5の勉強会を開催します。

対象はシングルな開発男子に制作女子！

いつもIT開発に忙殺されている男子や、Webデザイン制作女子、もちろん逆もOK、企画、運営、カスタマーなどWebにかかわる人たち、異業種だけどHTML5ということばにピンときた人など、どなたでも結構。ただしシングルな初心者に限定です。

今回はみんなでハートマークを作りましょう！

Canvas 2DはHTML5の一番の機能で、ホームページに自由に絵を描き、動かすことができるすごい技術。これを応用すればスマフォゲームも作成可能！2月14日のバレンタインもまもなく。ハートマークを描きましょう！

特典！

会場を提供してくださった「楽読」様が、勉強前に頭がスッキリする秘訣を教えてくださいます。
また、終了後、すぐ近くのマクドナルドにご案内しますので皆さんでおくつろぎください。

持参するもの

・しっかり充電したノートPC
　・無線LANを使いますので受信できること
　・ChromeかSafariをインストールすること
　・テキストエディタ（TeraPad，秀丸、サクラエディタ、miなど）をインストールすること
