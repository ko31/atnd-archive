Security.GS in仙台

〜ウェブセキュリティ勉強会〜

ゲヒルン株式会社は、2013年2月19日（火曜日）にアンデックス株式会社と共同主催で「Security.GS in仙台」を開催します。開催場所はAER 8F 会議室となります。

この勉強会では、主にWebサービスの開発を始めたばかりの方や、スタートアップ企業で自社サービスのセキュリティをより強化したいエンジニアの方、または最近VPSに触り始めた方など初級〜中級者を対象としていますが、どなたでもご参加いただけます。

講座内容

  • クラッカーの心理
  • XSSの話
  • JavaScriptの話

※ 講座内容は変更する場合がございます。

タイムスケジュール

18：30〜18：45　自己紹介等
18：45〜19：30　クラッカーの心理、XSSのおさらい、JavaScriptの話
19：30〜19：45　質問タイム
19：45〜20：00　「Gehirn Web Services」の紹介
20：00〜20：30　ライトニングトーク枠（発表希望の方はcontact@gehirn.co.jpまでご連絡ください。）

場所・日時

  • 2013年2月19日（火）
  • 18時30分〜20時30分（2時間）
  • AER 8F 会議室
  • 参加費：無料
  • 参加定員：30名

講演者の紹介

  • ゲヒルン株式会社　代表取締役　石森大貴
  • Twitter：@isidai
  • 平成２年生まれの２２歳。小学６年から独自にサーバーを作り始める。高校時代に経済産業省の主催する「セキュリティキャンプ」に参加。
    　その後、セキュリティキャンプの講師補助を２年間務め、ＴＢＳが放送したドラマ「ブラッディ・マンデイ」のハッキングシーンの技術考察記事で話題を集める。
    　筑波大学へ進学していたが、休学し独立、起業した。
