世界で戦ったトップアスリートから学ぶ
いのちを大切にする心と思いを実現するために必要なこと

「いのちを守る いのちを大切にする心を育む」をスローガンとする ATHLETE SAVE JAPAN より特別講師を招き 「いのちの教室」 を開催します。
今回は 巻誠一郎さん にお越しいただきます。

・・・・・・・・・・・・・・・・・

◇いのちをまもるAED実習

心臓突然死はいつでも誰でもどこにでも起こる可能性があります。
AED（自動体外式除細動器）実習 では、いざという時誰もが助けることができるよう心肺蘇生法を学びます。実技だけでなく巻さん自身の経験や実感していることなども講習の中でお話しいただきます。

◇トップアスリートが見てきた世界

巻誠一郎さんのトークセッション では、思いを実現してきた軌跡を伺います。サッカーを始めたキッカケから現役時代のこと、し烈を極めるアスリートの世界で戦いつつ社会貢献事業といった幅広い経験から感じたことをお話しいただきます。

・・・・・・・・・・・・・・・・・
