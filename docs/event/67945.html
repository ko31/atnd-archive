19:40〜20:10 セッション 『WebRTC Apps Testing and debugging』
It has been notoriously difficult to debug or trace webRTC applications. This presentation will go through some methods of testing that have been learned from the 3 years experience of building Skylink.io and the Temasys webRTC plugin. How to test in the browser, on-the-wire, how to test the webRTC stacks themselves, and how to test end-to-end calls will all be presented, along with the corresponding tools, and illustrated with screenshare and examples（原文）/ WebRTCのデバッグとWebRTCアプリケーションのトレースはすごく難しい。このプレゼンテーションでは、WebRTCのテストを行うためのいくつかの方法を、Skylink.ioとTemasys webRTC pluginの３年間の開発経験を基に振り返る。どのようにブラウザでテストをするのか（ローカルではなくて実際のネットワーク環境において）、どのようにWebRTCスタックのテストをするのか、どのようにEnd to End Callsのテストをするのか、対応するツールと共にスクリーンシェアなどのサンプルを例に、全て紹介する。（意訳）		 Mr. Alex Gouaillard
