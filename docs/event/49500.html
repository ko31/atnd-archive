★各社、講演内容と登壇者

①株式会社じげん

＜会社説明＞

基本理念
生活機会の最大化を目指し、インターネットを通じて宇宙（せかい）をつなぐ『場』を提供することで、社会との調和を図り、共に持続的発展を追求していく。

経営理念
OVER the DIMENSION ― 次元を超えよ！
圧倒的に突き抜けたサービス、圧倒的に突き抜けた会社を創り、世の中の常識や価値観を覆す

＜講演内容＞
「スクラム開発の広め方」
じげんでは、ここ一年間で開発環境が大きく変わりました。一年前には、Githubさえ使っていませんでした。スクラム開発の導入、社内への広まりを軸に社内に開発環境がどのように広がっていったのかをお話しします。

＜登壇者プロフィール＞
田口 哲也
インフラ、スクラムマスター


②株式会社ウェルセルフ

＜会社説明＞
株式会社ウェルセルフは、インターネット上で知識・スキル・経験を可視化し流通させることで「個人のエンパワメント」をする CtoC マーケットプレイス「coconala」を運営しています。
coconala

＜講演内容＞
スタートアップに求められる、少人数でスピード感あるサービス開発を可能にする技術や開発手法、またココナラでのそれらの実践による経験をご紹介します。

＜登壇者プロフィール＞
株式会社ウェルセルフ Software Engineer
恵比澤 賢
大手インターネットサービスプロバイダ、ソーシャルゲーム事業会社等を経て、株式会社ウェルセルフにて「coconala」の開発全般を担当。


③Sansan株式会社

＜会社説明＞
導入社数も2000社を目前に控えた導入法人向けクラウド名刺管理サービス『Sansan』。
50万人以上が利用している写真を撮るだけで名刺をデータ化できる人気の無料サービス『Eight』の企画・開発・販売をおこなう。

＜講演内容＞
「エンジニアのコミュニケーション」
Sansan のサービスを支えるテクノロジーと、組織が急成長していく中でエンジニアがどのようにコミュニケーションを図っていったのかについて、弊社での事例をお話しさせて頂きます。

＜登壇者プロフィール＞
Sansan株式会社　開発部長　
藤倉 成太
中央大学理工学部精密機械工学課卒業後に株式会社オージス総研入社。
プロダクト関連ビジネスの事業部にてミドルウェア製品の導入コンサルティング業務を経て
OGIS International, Inc. に出向。
シリコンバレーではR&Dや現地ベンチャーとの共同開発等に従事。
Sansan株式会社に転職後は BtoBの名刺管理ソリューションである「Sansan」の開発部門にてアーキテクトを担当。現在はSansan事業部にて開発部長を務める。
