description

来る 3月1日〜2日にかけて、沖縄高専xギークハウス沖縄の2会場で遠隔ハッカソンを行います。お互いの技術力向上を目的としたハッカソンです。

今回初の試みとなります。上手くいけば継続的にやりたいと思います。

会場提供について

会場提供協力: 辺野古IT工房seed 様
<img src =“https://dl.dropbox.com/u/5458377/%E3%80%90%E7%A2%BA%E5%AE%9A%E3%80%91seed-logo.gif”>


沖縄高専生への会場は 辺野古IT工房seed 様を利用させて頂くことになります。営業時間もこのハッカソン参加者に対してご配慮いただいております（3月2日は通常より2時間早く利用することが可能です)。この場の借りて御礼申し上げます。

タイムテーブル

時間 内容
3/1 18:00 – 18:15 開幕、ハッカソンのテーマ発表（会場:ギークハウス沖縄、辺野古IT工房seed)
3/1 18:15 – 18:45 各チームの作るもの紹介（会場:ギークハウス沖縄、辺野古IT工房seed)
3/1 18:45 – 21:00 ハッカソン（会場:ギークハウス沖縄、辺野古IT工房seed)
3/1 21:00 （沖縄高専組）会場の閉館時間より中断 （会場:ギークハウス沖縄 )
3/2 10:00 – 20:00 （沖縄高専組）ハッカソン再開（会場:ギークハウス沖縄、辺野古IT工房seed)
3/2 20:00 – 20:30 成果物発表（会場:ギークハウス沖縄、辺野古IT工房seed)
3/2 20:30- 20:45 投票（会場:ギークハウス沖縄、辺野古IT工房seed)
3/2 20:45 – 21:00 投票結果、閉幕（会場:ギークハウス沖縄、辺野古IT工房seed)

沖縄高専組は 辺野古IT工房seed さんを利用できます。1日は18:00~21:00まで利用可能となっております。2日目はハッカソン参加者のみ通常の営業時間より2時間早い10:00から利用可能となっております。

ギークハウス沖縄組は利用時間に制限はありませんので24時間使用可能です。

ルール

  • 各自のターミナルを screenxtv を使って配信しようと思いますので、当日までに配信できる状態にしておいてください。

* 当日はUstreamで会場の様子、会場間のやりとりを行いたいと思いますのでそちらの準備もお願いします。

  • ギークハウス沖縄の方はUstreamの配信を行いますが、沖縄高専の方は現在テスト中です。
  • 作成したプログラムは外部から閲覧できるようにしてください。

テーマの決定方法について

テーマは開催日当日の18時に @geeoki_hack から投稿された単語(2単語)を使った（または連想した）ものを作成していただきます。テーマは当日18時まで発表しません。

@geeoki_hack への単語登録方法

@geeoki_hack にリプライを飛ばすと単語の登録が完了します。どんどん登録して当日のテーマ決定を待ちましょう!!!!

当日の配信について

ギークハウス沖縄は当日の開発の様子をUstreamを使って配信 いたします。

また、各参加者は screenxtv を用いてターミナルを配信致します。インストール方法は、 こちら から確認できます。

Twitterのハッシュタグは #geeoknt で行います。

その他情報

ギー沖ハッカソン のページにて掲載していきますが、重要な情報はATNDにも反映いたします。

お問い合わせ
沖縄高専、ギークハウス沖縄ともに kimihito@gmail.com 宛にご連絡ください。
