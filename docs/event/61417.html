ヒカ☆ラボ一覧は こちら！！

★講演内容

・えふしんさん×山崎講師のパネルディスカッション
　良いWebサービスとはなにか？
　なぜサービスの解るエンジニアが必要なのか？
　文系の人がエンジニアになるためには？

・「文系でも解るプログラミングの学び方」
　ジーズアカデミーが開発した新たな学び方とは

～開催趣旨～
・新たなエンジニア養成スクール　G’s　ACADEMY　TOKYOの開校を記念して、
　TOPのエンジニアとエンジニア教育の第一人者が
　『全くの初心者からエンジニアになるためには？』をテーマに、対談形式でその方法を公開します。

アイディアをカタチにしたい方、起業を考えている方、プログラミングを始めようと思っている方、
ぜひご参加ください。

※内容は変更になる可能性がございます。

★講演者プロフィール

<img width=“200” src=“http://at-agent.jp/images/htmlMail/96af7dac13f3ad0e5b337bb2dc882a61.jpg”alt=えふしんさん />

■藤川真一（えふしん）氏

FA装置メーカー、Web制作のベンチャーを経て、2006年にGMOペパボへ。
ショッピングモールサービスにプロデューサーとして携わるかたわら、
2007年からモバイル端末向けのTwitterウェブサービス型クライアント『モバツイ』の開発・運営を個人で開始。
2010年、想創社を設立し、2012年4月30日まで代表取締役社長を務める。
その後、想創社（version2）を設立しiPhoneアプリ『ShopCard.me』を開発。
2014年8月1日からBASE（ベイス）株式会社のCTOに就任

<img width=“200” src=“http://at-agent.jp/images/htmlMail/33e0f75694f3022cd95669c1f2858bac.jpg”alt=山崎さん />

■山崎大助　氏

アジアで唯一（世界9人中の1名）のMicrosoft MVP（Bing Maps Development）であり、
数々のIT系メディアに登場し、2年連続 Microsoft MVP （Bing Maps Development）に選ばれた
業界最前線で活躍するクリエイター。大学院講師。
Bing関連だけでなくHTML5やWeb関連技術の普及に尽力しつつ、
IT系メディアでの寄稿、書籍の執筆、「ヒカ☆ラボ」や「パソナテック」などのセミナーやイベントでも活躍中。
最近では@itの「HTML5アプリ作ろうぜ！」の連載でも人気。
著書として「レスポンシブWebデザイン「超」実践デザイン集中講義」（ソフトバンク クリエイティブ）、
「jQueryレッスンブック jQuery 2.x/1.x対応」（ソシム）
連載：連載インデックス「HTML5アプリ作ろうぜ！」 がある。
