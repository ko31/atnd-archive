『Actions』
〜The 秋間勇人の想い 是非ご一読を！！〜

This event is…
「社会人と学生とが介する場所」

学生の皆さん、このまま日々を過ごしていて大丈夫ですか？
社会人の皆さん、毎日仕事に追われて新鮮な目で日々を過ごしていますか？
日常に新しい刺激と接点をもたらし、自分自身についてもう一度考えてはみませんか？

学生達は毎日楽しいと思います。
気楽な仲間に囲まれて、気楽な毎日を過ごす。

さて、このままでいいんですか？
世の中にはまだまだ広く熱い人々がいるということを、ここで気付いてみませんか？
少し視点を挙げるだけで世の中の見え方、変わりますよ？

今回社会人の方にはそんな学生達の素晴らしいメンターになっていただきたいと思います。
貴方達が過ごしてきた日々、経験、世界を若い世代に伝えていただけませんか？
その言葉の一つ一つが僕たちを大きく揺れ動かし、大きな成長へと変わるのです。

もちろん、この場は学生×社会人の他に、社会人×社会人、学生×学生の交流も可能です。
お酒を片手に見知らぬ者達が語り合う
少し異質な空間に、いらっしゃいませ
