iOS/androidのアプリ開発がワンソースで可能な『corona』。

日本コロナの会では地道にこのcoronaを口コミで伝えてきました。

【 日本コロナの会 Japan Corona Group 】
[ WEBサイト ] http://www.japancoronagroup.jp/
[ グループサイト ] http://groups.google.co.jp/group/coronaja
[ twitter ] http://twitter.com/JapanCoronaGrp

iphoneが盛り上がり、android端末の発売ラッシュを控え、
今年はいよいよ日本corona元年になりそうです。

これまではメーリングリストを中心に情報交換頂いておりましたが、
一度オフラインで集まってキックオフミーティングを行いたいと思います。

顔合わせや今後やりたい事の話し合いなどが出来ればと思っておりますので、
ご参加を希望される方はこちらからエントリー頂くか、
日本コロナの会のグループサイトからご連絡をお願いします。

また、懇親会なども併せて行いたいと思います、
coronaビールでも飲みましょうか（笑）

事務局的には今後、勉強会、アイデアソン、ハッカソン、地方支部設立など、
アプリ開発を盛り上げる色々な事を企画したいと考えておりますが、
これから一緒にcoronaを盛り上げていきたいと思っている方、
coronaを勉強したいと思っている方、
coronaでビジネスがしたいと思っている方、
都内在住・地方在住、個人・法人を問わずお集まり頂いて、
みなさんで一緒に楽しいことを始めましょう。

また、会の運営を手伝ってくださる方も大歓迎です！

それではみなさんのご参加希望のご連絡をお待ちしております！！
