第1回HS技術部総会

参加条件

・ハースストーンが好きな方

開催内容

ハースストーン及びそれに関連する創作活動を通しての交流を行う会です。
14:30よりトーナメント又は総当たり戦の大会を行います。
優勝者には自作のハースストーングッズを賞品として提供します。

また希望者を対象に以下のどちらかの講座を実施します。
参加コメントに希望する講座を書いていただけると助かります。

  • 「ボブでもできる！DiscoardBot作成講座」
    • Discordのテキストチャットで会話ができるBotの作成講座です。
    • 事前にノートPCを持参する必要があります。
    • 難易度が少し高く、時間がかかる講座です。
    • 主催者側の準備の都合により開催されない可能性があります。
  • 「創作のモチベーションの維持の仕方」
    • 主催者の創作活動に対する個人的見解を語るだけの講座です。
    • 持参が必要なものは特にありません。
    • 上の講座が開催できない場合は、こちらを実施する可能性があります。

場所

kurenai会議室
高田馬場駅早稲田口付近の第一大島ビル4階の406号室です。

諸注意

  • 予備の端末は所持していないため、各自で端末の持参をお願いします。
  • 電源、充電コードの用意はございますが、数に限りがあります。ご自身でご用意していただけると便利です。
  • wi-fiは会議室のwi-fiを使用できる予定ですがトラブル等で利用できない可能性もございますので、できるかぎり各自での通信環境のご用意をお願いします。
  • 位置情報を使用しての開催になりますので位置情報の許可をよろしくお願い致します。* 飲食は可能ですが、匂いのきつい食べ物や飲酒は禁止です。
  • 勧誘等、他の参加者の迷惑になる行為は禁止です。
  • 上記のような迷惑行為が発覚した場合、イベントから退出いただき、今後の参加をお断りさせていただくことがございます。
  • 盗難や紛失については主催は一切の責任を負えません。貴重品等の管理はご自身でお願いいたします。
  • 忘れ物があった場合、取りに行くのにもう一度会議室を予約しなくてはならなくなってしまうので、忘れ物のないようにお願いいたします。
  • 詳細の確認・質問・連絡等はTwitter（@incarnation_0）の方にお願いします。
