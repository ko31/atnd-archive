title 私、処女膜破ります！嬉し恥ずかし沖縄ツアー2泊3日
catch
description

1. 期　日　平成22年9月25日（土）〜27（月）
※雨天決行
2. 集　合　●:● 沖縄現地集合
3. 行き先
・初日
　齋場御嶽、残波岬、アメリカンヴィレッジ
・２日目
　ちゅら海水族館、ビーチロックビレッジ、瀬底島
・３日目
　首里城、牧志市場
・陣地
　国際通り

4. 目　的
・友達と協力しあおう。
・自然とふれあい、体を動かそう。
・規則を守り、仲良く、楽しい沖縄にしよう。

5. スケジュール
【初日】
AM●●
羽田空港第●ビル集合着後レンタカーゲット。
AM●●
高澤到着までドライブ→高澤拾ったら南の齋場御嶽へ（３日間のパワーをもらう）。→どっかで沖縄ずば（昼飯）→北へ行く。恩納までいって万座毛→残波岬→アメリカンヴィレッジと下り適当にふらつく→アルデック（沖縄のでかい酒屋）で３日間の酒買い出し→ホテルチェックイン（※買った酒を冷やすのを忘れないこと）→国際通りで宴→ホテルで宴→就寝

【２日目】
美ら海水族館→ビーチロックビレッジでお茶か昼飯（この辺要相談）→海中道路を通って瀬底島→国際通りにて宴→ホテルにて宴→就寝

【３日目】（※レンタカーいつ返すかにもよる。）
ホテルチェックアウト→首里城で赤く染まる→国際通りでおみやげ→牧志第一公設市場にて昼飯→空港

6. 持ち物（リュックサックに入れること）
水筒（お茶、さ湯、酒）　おやつ（ 300 円まで）　
しき物　ビニール袋 1 〜 2 枚（ゴミ袋）
紙袋（エチケット袋）　ふきん 1 枚　雨具（かさは持ちません）ハンカチ　ちり紙　しおり　

7. 服装
動きやすい服　はきなれたくつ

8. 注意事項
・飛行機の中では大声を出さない、窓から顔や手を出さないこと。
・気分が悪くなったら、がまんしないでスッチーに知らせること。
・勝手な行動をとらないこと。
・自然の草木を大切にすること。
・集合の合図があったら速やかに集まること。

9. その他
・雨天の場合は、朝 6:30 に連絡網をまわします。お弁当と水筒を持って登校してください。
・当日欠席の場合は 7:30 までに学校に連絡してください。
