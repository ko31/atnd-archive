イベント概要

Hamamatsu.rbは、静岡県浜松市周辺のRuby技術者やRubyに関心がある人が集まって、Rubyに関する何かをする予定の地域コミュニティです。キックオフ・ミーティングでは、メンバーのRuby愛を確認し、これから何をやるかを軽く飲み食いしながら相談します。

プログラミング言語Rubyに興味がある人、Ruby on Railsでウェブアプリを作りたい人、これからRubyを始めたい人、何か良く判んないけどプログラマーの集まりに参加してニヤニヤしたい人なんかが集まるとイイと思います。

Hamamatsu.rb とは

昨年の2月に開催された 浜松ITカンファレンス の講演で、Matzが「浜松はRuby誕生の地といっても過言ではない」と宣言してから早1年、Ruby誕生の地にもかかわらずRubyに関する地域コミュニティが無いのは寂しいので作ってみました。

今回のキックオフ・ミーティングを皮切りに、しばらくは毎月第2水曜日の夜に定期開催する予定です。今回はキックオフ・ミーティングで飲み食いしますが、2回目以降は定期的に集まって勉強会や読書会をベースに、発表したい人がいれば発表するぐらいのユルい感じになると思います。
