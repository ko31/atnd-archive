description

　本講座では、国際オリンピック委員会（IOC）のワーキンググループがまとめた「アスリートの栄養ガイドライン」や海外の栄養士に実施したヒアリング結果を基に、アスリート食の考え方から国別にみえる食生活の違いについて解説します。
また、本年で移転が決まっている横浜F・マリノスの練習場「マリノスタウン」の選手食堂にて、トップ選手が食べているオリジナルの「アスリート食」をご試食いただきながら、海外で学ばれ、国内外の様々な選手をサポートされてこられたストレングス&フィットネスコーチの渡部龍也氏に、外国人選手の身体作りや食生活の様子、日本人選手との違いについてお話しいただきます。
どなたでもご参加いただけますので、是非この機会に皆様お誘い合わせの上、ご参加ください。

Food Guide講座2015の7月以降の講座は順次Facebook及びwebsiteにてご案内いたします。
