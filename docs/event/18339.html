概要

Wolfram Research社による数式処理システムかつプログラミング言語であるMathematicaについてみんなでわいわいやる勉強会です。
今回はMathematica未経験者を対象とし、Mathematicaの概要と使い方、Tipsなどを紹介する予定です。
Mathematicaを持ってない方でもぜひご参加ください。
現在LTをしたいという方も募集しています。Mathematica / Wolfram Alphaの話題だけでなく、技術系であればOKですので、興味がある方はぜひ@aomoriringoまでご一報ください。

不明な点はTwitterアカウント@aomoriringoまでお願いします。
