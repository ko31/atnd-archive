概要


『Web標準の教科書─XHTMLとCSSで作る“正しい”Webサイト』や『伝わるWeb文章デザイン100の鉄則』の著者であり、数多くの連載を持つ“あの”サイバーガーデンの益子貴寛さんを迎え、来る2009/12/12に開催される「CSS Nite in TAKAMATSU 」のプロローグ版を開催します。
秋の大型連休の初日は大いにインプットしましょう！
当日は益子さんのCSSをテーマにしたロングセッションの他、地元スピーカーのショートセッション、ライトニングトークなどを企画しています。
Webridge（ウェブリッジ）に興味ある人も、益子さんに合いたい人も是非参加してください。

参加費用


会費は一般2000円　学生1500円（学生証をご用意ください）
イベント終了後は益子さんを囲んでの懇親会（別途有料）も予定しています。

CSS Niteとは


2005年10月、アップルストア銀座にてマンスリーイベントとしてスタート。当初は文字通り「CSS」にフォーカスしていましたが、現在は“Web 制作者にとって有益であろうトピック”を広く扱っています。
名古屋、大阪、青森、福岡、沖縄、秋田、札幌、福井、仙台、福島、広島、新潟、京都など首都圏以外での“地方版”なども開催しています。10月に石川版、12月に高松版、2010年2月に富山版を開催予定。茨城、静岡版も構想中です。
これまでに100回以上の関連イベントを通し、のべ17,000名以上が参加しています。
CSS Nite オフィシャルサイトはこちら＞＞

Webridge (ウェブリッジ) KAGAWA とは


香川県で活動するWeb制作者や、Web制作者を目指す学生・生徒などが、互いに刺激し合い感性を磨き、情報・知識・モチベーションを高めていくため、CSS Nite in TAKAMATSU 実行委員会を中心に発足したコミュニティです。当日はWebridge (ウェブリッジ) の参加申し込みも受け付けます。※他県からの参加も大歓迎です。
