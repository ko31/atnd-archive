起業、副業…などなど、ビジネスをし始めると

気になるのが売り上げですよね！
　
大切なのは
実績です！！売上です！！

では、質問です。
あなたは、どれだけの売上、つまり収入が必要だ。欲しい！と思っていますか？
10万ですか？30万ですか？50万ですか？１億ですか？

あなたの周りを今、見渡してみてください。
あなたの理想の収入を取れている人がどれくらいいるでしょうか？
残念ながら、あなたが今一番よく会う人10人の平均収入以下の収入しかあなたは得ることはないのです。

それが、あなたが今いるステージです。

そのステージを上げ、ビジネスで成功するには
圧倒的に実績と売上を上げ続けている人の側にいるしか
実は、方法がない。

「自分よりできる人と一緒にいるから成長できる。ステージを上げられる」

売上アップするために
必要な7つの経営戦略を
実績あるステージの高い東京の社長様を招いて
講演していただきます。

この講演をご受講くださった方が、
・起業後、初めて月商７桁に突入！
・わずか２週間で１２０万円のコンサル契約を取得！
・主婦から、経営者のコンサルタントへ転身！
・複数の会社からヘッドハンティングを獲得！
・会社の人間関係の悩みから解放された！
・自分のやるべきことが明確にわかって、毎日楽しくてしょうがない！
・法人化し代表取締役社長になった！

などなど売り上げを上げ続けるための原理原則を
2人の講師より伝授していただきましょう。

第1部【グループカウンセリング】
あなた自身に自立、起業のために
どんなチカラをすでに持っていて
どんなチカラが必要なのか自分で知る

第2部【講師：東景子】
｢落ちこぼれからの復活！
　スピリチュアル×成功法則｣

第3部【講師：久道さおり】
「借金1300万円からの大逆転！

《講師紹介》
【東　景子】
Ｋstyle株式会社 取締役
物心ついたときから妖精や宇宙・占いなど目に見えない世界に興味を持ち、のめり込むように勉強をスタート。
2015年 マヤ暦鑑定を始める。

鑑定の的中率の高さだけでなく、
カウンセリング技術に対する評価も高く、アドバイスが適確でわかり易いと噂があっという間に広がり10代から60代まで幅広い年齢層の方から依頼が殺到。
2017年 起業し、仕事として本格的に活動を開始。全国を飛び回る日々を送っている。

スピリチュアル業界では異例の半年で月商７桁を達成し、1年半で売り上げ実績が4000万円を超えるなど起業家としても注目を集めている。
現在手掛ける仕事はスピリチュアル鑑定を含め、子育て事業、WEB・ネット構築、スクール運営など１０種を越える。

【久道さおり】
日本女性支援プロジェクト 代表理事
４年前の私は、多額の借金を負ってしまった独身のアナログアラフォー女性でした。切羽詰まった状況にもかかわらず、どこか現実味がなく、人生どうにかなるだろうと楽観的に思っていました。

しかし、あることがきっかけで考えを改め、自分の人生をやり直そうと思ったんです。

結果、この４年で女性を対象にしたキャリアアップ事業を立ちあげて、元々人前で全く話せなかった恥ずかしがり屋が、5,000人以上の方の前で講演を実施（全国22都道府県で開催）。
自社開発をした教育プログラムの採用実績も200社を超えました。また昨年春には、東久邇宮記念賞※を、秋には褒章を受賞させていただくことができました。

４年前の私は、今の状況を全く想像できていませんでした。
でも、実現しているということは、「理由」があるんですね。

私には、「なぜ自分がここまで変われたのか？」という理由をたくさんの方にお伝えして、多くの女性の皆さんにご自身の「幸せのカタチ」を掴んでいただきたいという思いがあります。

そして、その思いをコツコツと伝え続けてきたおかげで
私を含め月商７桁を軽く超える女性起業家や、
副業で成功する会社員など、自らの力で自分自身の
「幸せのカタチ」を実現する方が続出しています。
