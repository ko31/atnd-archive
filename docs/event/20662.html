生産の３要素(３Ｍ)で捉える中国工場の製造現場における品質管理の問題点と対応策

イベント ユーザー

title 生産の３要素(３Ｍ)で捉える中国工場の製造現場における品質管理の問題点と対応策
catch ～失敗等具体的事例、改善手順、指導のポイント～
description

【習得できる知識】
・今まで見えなかった中国工場の問題点が明確になります
・自社中国工場や取引先中国工場の品質改善、指導のポイントが理解できます
・中国人従業員への教育をどのように進めたらよいかがわかります
・中国人の考え方や中国人とのコミュニケーションの取り方が理解できます

【受講対象者】
・日本人品質管理担当者で中国工場・企業に勤務されている方
・中国に行って品質管理指導に困っている方
・これから中国展開を考えている製造業の方及び中国進出して間もない製造業の方
・中国で部材調達をして、その品質改善に苦労されている方

【講座の趣旨、ポイント】
・中国工場の品質がよくならない要因を生産の3要素の面から考えます
・具体的事例、品質課題の解決手順、部材購入先の品質指導法など中国工場の現場で活かせます
・ディスカッションやワークで新たな気付きが得られます

※参加者特典
・参加者特典として、講師が参加者の抱えている中国工場の問題点に関する相談を無料でお受けします。
　1社様1回限定です。講師が貴社訪問して行いますので、講師の交通費のみご負担ください。
　詳細はセミナー当日説明いたします。

【プログラム】

１．中国工場・品質トラブル事例
　　1-1.A社－人の問題
　　1-2.B社－設備の問題
　　1-3.C社－材料の問題

２．生産の3要素(３M)の面から品質問題を捉える
　　2-1.人（Man）
　　　a.作業者　　　b.管理者 c.経営層 d.日本人駐在員
　　2-2.設備・機械（Machine）
　　　a.日本と同じ設備なら大丈夫か b.ノウハウ伝承と維持・標準化 c.教育のしくみ
　　2-3.材料（Material）
　　　a.部材調達の変遷 b.中国部材を使う側の問題点 c.部材品質と対処法

３．中国を理解する（異文化コミュニケーション）
　　3-1.中国人を理解する
　　3-2.中国で注意すべきこと

４．中国工場の品質管理・改善で押さえるべきポイント
　　4-1.中国工場のABC
　　4-2.自社問題点をわかっているか
　　4-3.課題解決の順序　　－工場のレベルで順序は異なる
　　4-4.課題解決の方法は日本と同じ、でも・・・
　　4-5.顧客監査も改善のチャンス

５．取引先工場を見るとき、指導するときのポイント
　　5-1.意識のずれを解消
　　5-2.基本の徹底
　　　a.識別・表示　　　b.不良品が発生したときの処置　　　c.異常発生時のルールと対応
　　5-3.不具合対策書のチェックポイント
　　5-4.どこまで求め指導するか
　　5-5.新規開拓ではISOをうまく活用する

６．まとめ
