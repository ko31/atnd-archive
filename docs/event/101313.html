ルール詳細

大会全体について

  • 大会はスイスドロー方式、5回戦で行います。
    • 対戦相手の勝率(オポネント・マッチ・ウィン・パーセンテージ)は日本選手権と同様考慮しません。 同じプレイヤーと複数回マッチングすることもあります。
  • ゲームは4 人戦で行われます。ただし、参加人数が4 で割り切れない場合、3 人戦テーブルがいくつか発生する場合があります。
  • 各試合の1位に6pt,2位に3pt,3位に1pt,4位に0ptの勝ち点が与えられます。
    • ゲーム終了時に同点のプレイヤーが存在した場合、消化した手番数が少ないプレイヤーを上位とします。
    • 消化した手番数が同じ場合、それぞれのプレイヤーにポイントを分割して配分します
      • 例えば、一位と二位が同点だった場合は、(6+3)/2 でそれぞれのプレイヤーが4.5ポイント獲得します。
  • 5回戦終了時点での勝ち点により、順位を決定します。

  • 勝ち点が同点のプレイヤーが存在した場合、以下の順で判定を行い成績上位者を決定します。

    • 一位の回数＞二位の回数＞三位の回数
    • ドミニオン日本選手権と異なり、各試合の獲得勝利点数は順位に影響しませんのでご注意ください。

  • プレイヤーのマッチングは、運営が発表します。

  • 各試合間の休憩時間は最低5分間設けます。

各試合の進め方

  • 各テーブルの手番はランダムに決定されます。
  • 試合時間は45分とします。
    • 45分以内に試合がおわらなかった場合、45分経過時点でのプレイヤーから最後の手番のプレイヤーまでゲームをプレイし、その時点の勝利点の点数で勝敗を決めます。
    • 時間切れ後最後の手番プレイヤーのターンが回る前に通常のゲーム終了条件を満たした場合は、その時点でゲームを終了します。
    • 時間切れ後に最後の手番プレイヤーが前哨地, 使節団, 寄付などターン終了後に効果を発揮するカードを利用した場合、そのプレイヤーのターン中に通常のゲーム終了条件を満たさなければ各カードの効果は発動します。
  • ジャッジについては基本的にテーブル内でのセルフジャッジになります。
    • プレイミスをした場合の処理は、テーブル内で話し合って決め、テーブル中のプレイヤーの過半数が認めたやり方に従うようにしてください。

サプライの決め方

各試合、以下の割合でランダムにカードを選択しサプライを決定します。

  • 一回戦：『基本第二版』（第一版は使いません）+『陰謀』＋『海辺』(基本4枚 + 陰謀3枚 + 海辺3枚)
  • 二回戦：『錬金術』＋『繁栄』(錬金術4枚 + 繁栄6枚)
  • 三回戦：『異郷』＋『収穫祭』(異郷6枚 + 収穫祭4枚)
  • 四回戦：『暗黒時代』＋『ギルド』(暗黒時代6枚 + ギルド4枚)
  • 五回戦：『冒険』＋『帝国』(冒険5枚 + 帝国5枚 ランダムで選ばれた「イベントカード」、「ランドマークカード」をそれぞれ1枚ずつ使用)

サプライは各ラウンド3パターン用意されます。いくつかのテーブルでは同じサプライを使用しますが、全てのテーブルで同じサプライにはなりません。具体的には以下のようになります。

  • ランダムにサプライA,B,Cの３種類のサプライを上記ルールで作成したあと、下記のような割り当てで試合を実施します。
  • サプライAは1,4,7番テーブル
  • サプライBは2,5,8番テーブル
  • サプライCは3,6番テーブル

時間制限について

  • 45分終了時点でプレイしているプレイヤーから2回目の4番手のプレイヤーのターンが終了するまでゲームを進行し、その時点での獲得VPで順位を決定します。
    • (例1) 2番手のプレイヤーのターン中に試合時間が終了した場合、2->3->4->1->2->3->4 の順でターンを実施します。
    • (例2) 4番手のプレイヤーのターン中に試合時間が終了した場合、4->1->2->3->4 の順でターンを実施します。
    • 45分終了後に通常のドミニオンの終了条件を満たした場合は、その時点で通常通りゲームは終了します。
    • 最後のターンのプレイヤーが寄付や峠など、「ターンの間」に発動する効果は通常のドミニオンの終了条件を満たした場合は次のターンがないため効果を発動できませんが、通常のルールであれば1番手のプレイヤーのターンがあるが、時間切れ終了にともなって4番手のプレイヤーが最後のターンのプレイヤーになり、寄付や峠を使った場合には効果が発動することとします。
    • 時間制限があるルールですので極端な長考(何のアクションも購入もしない状態で1分以上考えること)は避けてください。
    • 逆に他のプレイヤーを極端に急かすことも禁止します。
    • 露骨な時間稼ぎはやめてください。
