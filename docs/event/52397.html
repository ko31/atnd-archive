catch 資金調達・補助金獲得のための収支計画を描く１時間
description

◆スタートアップ支援に特化する税理士から直接レクチャー
20代で銀座に会計事務所を開業され、高校生や大学生の起業家を含め、20代、30代の経営者の企業を中心に財務・法務・労務・税務など幅広いサービスを提供されています。2年間で100社以上のクライアントを口コミのみで獲得し、年間100社以上の経営者に事業計画書の指導、資金調達・補助金獲得の支援をされています。

◆１時間で税理士と１VS１で収支計画を作り込む
税理士の先生と１対１で収支計画書の作成を行うため、１時間ごとに１名限定の予約枠を設けております。お好きな時間をご予約頂き、税理士から直接レクチャーを行います。
【時間枠】
①18:00〜19:00
②19:00〜20:00
③20:00〜21:00
④21:00〜22:00

◆収支計画を「学ぶ」ではなく「創る」
本イベントは座学ではなく、実際に手を動かして、収支計画書を作成するためのイベントです。実務で使えて、資金調達に活用できる収支計画書をその場で作成して帰って頂きます。

◆起業準備中の方・経営者の方向け
これから事業を行っていこうと考えている方や、既に経営を行っている方で、収支計画を創ったことがない、資金調達を考えているという方を対象としています。

※参加者の方は、お名前、メールアドレス、事業の内容、希望する時間を記入した上でお申し込みください。
