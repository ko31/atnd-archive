【講座のポイント】

　近年の地球環境・資源問題のなかで注目を集めているバイオプラスチックの存在意義と分類・特徴について先ず概説し、次に最も実用化の可能性の高いポリ乳酸の基本特性、高性能・高機能化技術、材料設計技術、成形加工技術と製品・用途開発の現状と課題について、初心者～中級者にも分かり易く説明する。本セミナーは、25年間に渡りポリ乳酸の基礎・応用研究から技術開発・事業開発までを世界に先駆け成し遂げた実績のある高分子学会フェローによる渾身のセミナーである。

【プログラム】

１．ポリ乳酸の基本特性

　　1.1　今、なぜバイオプラスチックなのか…地球環境・資源問題
　　1.2　バイオプラスチックの分類
　　　a)　日本バイオプラスチック協会（JBPA）識別表示制度
　　　b)　グリーンプラとバイオマスプラの違い
　　1.3　なぜポリ乳酸がベストの選択なのか？
　　　a)　生分解機構の分類と特徴…酵素分解タイプと非酵素分解タイプ
　　　b)　バイオリサイクル材と耐久性構造材料の両立可能
　　1.4　ポリ乳酸の製造法と基本特性
　　　a)　製造プロセス…乳酸発酵からポリ乳酸への化学合成
　　　b)　ポリ乳酸の物理的性質（熱的・機械的性質、透明性）、化学的性質
　　　c)　ポリ乳酸のヒトと自然環境に対する安全性…WHO勧告、食品衛生法
　　　d)　ライフサイクルアセスメント（LCA）による環境負荷の客観的・定量的評価
　　　e)　長所（透明性、力学的強度、安全性等）と短所（耐熱性、耐久性、耐衝撃性等）

２．ポリ乳酸の高性能化・高機能化に関する最新技術開発動向

　　2.1　耐熱性向上技術
　　　a)　ポリ乳酸そのものの耐熱性向上…結晶化速度の飛躍的向上
　　　b)　石油系エンプラ（PC,　ABS）とのブレンド／アロイ化による耐熱性付与
　　2.2　長期耐久性付与技術…耐加水分解性（耐湿熱性）向上
　　2.3　耐衝撃性付与技術…可塑・柔軟剤、フィラー
　　2.4　難燃性賦与技術
　　2.5　発泡成形技術…押出し発泡とビーズ発泡
　　2.6　ガスバリア性付与技術…フィルム、シート、ボトル

３．ポリ乳酸の最新成形加工技術

　　3.1　結晶性高分子の成形加工プロセス…成形加工性とは？
　　3.2　成形加工工程における結晶化促進…成形品の耐熱性向上
　　3.3　ポリ乳酸の成形加工プロセス
　　　a)　融点からの降温過程におけるmelt　crystallization…射出成形
　　　b)　室温からの再昇温過程におけるcold　crystallization…真空・圧空成形
　　3.4　これまでの製品化と実用化例
　　　a)　繊維・不織布…農林・園芸・土木、生活・インテリア、衣料
　　　b)　２軸延伸フィルム…窓張り、ラミ袋、紙ラミ、青果物包装
　　　c)　ブローンフィルム…生ごみ袋、農業用マルチ、レジ袋
　　　d)　真空・圧空成形…ブリスター、青果物容器、食品容器、カップ
　　　e)　射出成形…リターナブル食器、電気・電子・OA機器、自動車内装材、雑貨
　　　f)　発泡成形…食品容器、緩衝材、魚箱
　　　g)　ブロー成形…ボトル

４．ポリ乳酸製品事業成功例に見るキラリと光る事業戦略…ポリ乳酸繊維エコバッグ

　　4.1　はじめに卓越したデザイン戦略ありき（産官連携）
　　4.2　地場産業の技術力の結集
　　4.3　単なる加工メーカーからの脱却と最終製品志向
　　4.4　したたかな販売戦略（国内外有名美術館、欧米のセレクトショップでの販売）

５．ポリ乳酸に残された技術的課題とその解決策

　　5.1　残された技術的課題
　　　a)　射出成形などの成形加工性向上（成形サイクルの短縮、金型温度の低減）
　　　b)　寸法安定性の向上（低熱収縮率、耐クリープ性、二次結晶化等の経時変化なし）
　　5.2　上記課題解決への二つの技術的アプローチ
　　　a)　ステレオコンプレックス型ポリ乳酸（sc-PLA）の可能性と問題点
　　　b)　高L組成ポリ乳酸（highly　LLA-rich　PLA）への期待
　　5.3　高Ｌ組成ポリ乳酸の成形加工工程における結晶化ダイナミクス
　　　a)　結晶化速度、結晶化度、融点のD-乳酸含量依存性…耐熱性向上と成形サイクル短縮の可能性
　　　b)　結晶化温度依存性…低温成形と成形加工性向上の可能性
　　　c)　非等温結晶化…冷却速度依存性
　　5.4　マルチ機能改質剤…少量添加で耐熱性と耐衝撃性の同時改良は可能か？
　　　　　ポリ乳酸用造核剤（耐熱性は向上しても耐衝撃性は低下する）との比較
　　　a)　射出成形
　　　b)　真空・圧空成形

６．バイオプラスチックのこれから

　　6.1　原料ソースの多様化…食料資源から非食料資源へ
　　6.2　新規バイオプラスチックの胎動…バイオポリオレフィン、バイオポリカーボネート、完全バイオPET、バイオポリアミド
