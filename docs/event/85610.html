登録について

・右上の赤いボタンよりご登録ください。捨てアカウントでも結構です。
・ボタンを押したあとにアンケートが出ますので、ご協力お願いいたします。
・ご参加前に公式サイト JobJamKyoto の注意事項とQ&Aをご一読ください。
・キャンセル、欠席処理は、本ページから行ってください。
・本登録は必須ではございませんが、人数把握のため必要なものです。ご協力お願い申し上げます。
・海外ではプランナー（企画職）のことを「ゲームデザイナー」というらしく、そう書かれると企画か絵の方なのか曖昧なので、アンケートには、企画のかたは「プランナー」、デザインや絵のかたは「○○（例：キャラクター、UI…etc）デザイナー」等、書いていただけると助かります。
・アンケートに「お住いの都道府県」とあるところ、ご住所を番地まで全部書いていただかないで大丈夫。都道府県がわかればOKです。

当日について

・募集内容やタイムテーブルなどは パンフレットpdf （当日配布します）をご覧ください。
・荒天中止等の場合はこのページでアナウンスしますので、来場前にご確認ください。
・お昼食べる場所は会場周辺に、王将とかいろいろあります。コンビニで買ったものは飲食スペースをご利用ください。
