description

■イベント趣旨
今、日本は人口の減少と共に、大都市と地方の差が拡大しているようです。
そこで、大都市が持つ人材・資金を活用し、地方の課題を解決するためのハッカソンを開催します！

地域の声を聞き、集めた悩みや課題、それらをハッカソンを通して解決します。果たして、今の地域格差・社会の問題自体を解決することができるのでしょうか？！

ギフソンを通して、今の現状を少しでも多くの方に知って頂き、一人でも多くの方に協力して頂けたらと思います。

■仕組み
1. Facebookを活用して、地域の悩みや課題、要望など収集します。
2. 集まった課題に対し、ハッカソンで解決に挑みます。
3. ハッカソンで作られた作品は、それを必要としている地域に届けられます。
4. このプロジェクトで人と人とがつながり、より多くの人に幸せと感動を届けることが、私たちの目標です。
