概要

飯塚から世界へ！をキーワードにソフトウェア技術者たちが集結し、テクノロジーについてディープに語り合う場、e-ZUKA Tech Night。

Vol.0, Vol.1と立て続けに海外ゲストを迎え、幸先の良いスタートを切ることが出来ました。今回からはコミュニティ内の交流に力点を移しながら、さらに内容を充実させていきたいと思います。

さて、新入生や新社会人を迎えるこの時期、未経験のプログラミング言語に取り組むという方も多いと思います。
ということで、Vol.2のテーマは 「私を魅了する言語」 です。
ソフトウェア技術者にとってプログラミング言語は、道具であり成果物であり創造性の発現でもあり、いわば思考の一部と言っても過言ではない存在です。各スピーカーがなぜその言語を選択したのか、そして何に魅力を感じているのか。その語りを聞いて、未知の言語に挑戦してみるきっかけが生まれれば嬉しいです。

LTを募集中です！テーマに関係ないもので構いません。
