title 第28回「WEB TOUCH MEETING」やります！
catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。


■現在予定している内容：

　●セッション１(35分)
　『Twitterおもしろ活用講座』光本さん

　＜内容＞
　Twitterをつぶやくためだけに使うのはもったいない！
　Twitter廃人によるTwitterのちょっと変わった面白い(かもしれない)使い方をご紹介。
　拙作の「世話焼きさんbot」についてもしゃべらせていただきます。

「世話焼きさんbot」についてはこちら↓
　http://www.mitsumotocs.com/twitterbot/sewayaki/

　＜光本さんの紹介＞
　フリーランスのイラストレーターとして活動するかたわら
　ウェブ開発なんかもする器用貧乏な人間です。
　http://www.mitsumotocs.com/


　●セッション２(40分)
　『ウェブならではの仕組みを盛り込んだゲーム』カンドウ柳谷さん

　＜内容＞
　今まで作ったものの中から、今回は主に二つを紹介します。

　・「貫通狙いシューティング」のウェブならではのマウス操作感について
　　http://sharefl.jp/projects/shoot/
　・ツイッターで結果報告機能を追加してみる

　・オリジナルパズルゲーム「三次シフト」の開発課程を振り返る
　　http://www.miyoshi-townnavi.jp/blog-parts/
　・モバイル対応Flash Liteへの移植してみる

＜柳谷さんの紹介＞
　Flashでは、キャンペーンゲームコンテンツ、ブログパーツ、Flash Lite1.1、2.1、
　ブログ等のデータをビジュアル化、twitterのビジュアル化するコンテンツ。
　その他、大規模コミュニティー、学校のウェブサイトのデザイン、
　イラスト等の仕事を受託しています。

　カンドウコーポレーション
　http://www.can-do.co.jp/


　●セッション３(40分)
　『WEB開発に役立つかもしれないゲーム講座』佐々木さん

　＜内容＞
　WEBとゲーム。
　そのゲームはどのようなしくみで、どのように作られているのか。
　さわりの部分を案内をしていこうかと思います。

　＜佐々木さんの紹介＞

　某ゲームスクールでゲームプログラムを教えながら、
　androidやその他のプラットフォームでゲーム開発を行っています。
　なお、シューティングゲームしか作れません。
　http://www.lemonte.net/
