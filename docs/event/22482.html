テーマ「楽しいLTの会 2011」

2011年もあと僅か、ということで2011年締めとなる「たのアプ」を開催します。

今回は趣きを変えまして、大阪や全国で活躍されている勉強会主催者さんを始めとする方々に「楽しい ほにゃらら」をテーマとしたLightning Talksをして頂きます！題して「楽しいLTの会 2011」。

参加者全員で一番面白かったLightning Talksを選んで頂き、見事選ばれた方に粗品を差し上げようと思います。また参加者全員の中から抽選１名様にも粗品をプレゼントする予定です！

みなさんとワイワイガヤガヤ楽しい時間を過ごせればと思います。お気軽に、ぜひぜひご参加ください。
