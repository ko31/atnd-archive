description

神戸のiPhoneアプリ開発の勉強会です。

最近やっていること、困っていること・悩んでいることなどを共有するのスタイルの勉強会です。ストーリーボードやコード見ながら話すイメージです。

会費はかかりませんが、会場代実費を集金します。
(参加者人数によりますが、だいたい700円程度です)

参加条件はXcodeをインストールしていることです。
MacAppStore : Xcodeのサイトへ

会場情報

飲食店
セブン−イレブン 神戸下山手通５丁目店が会場最寄りのコンビニです。
会場には、コーヒーサービス(350円)、喫茶店、自販機があります。

通信環境
会場はFREESPOTがあります。登録するとしばらく使えます。

会場備品
プロジェクターがあります。
変換アダプタは主催者側で準備しています。（VGA ←→ディスプレイポート、VGA←→ライトニングコネクタ、またはAppleTVによるAirPlay）

場所

生田文化会館　第5会議室
住所:兵庫県神戸市中央区中山手通６−１−４０

タイムスケジュール

13:00〜
・自己紹介など

発表
・MapKitなど最近のできごと by @windblume

フリートーク・フリー質問・フリー開発
参加にあたってなにか発表する必要はありませんが、困っていること・悩んでいること、最近やっていることなど共有しましょう。誰かが知っていて解決できるかもしれません(できないこともありますが)。また、プロジェクターを使ってストーリーボードやコードを映してレビューとかも面白いです。

懇親会

こっちが本番！みんなで飲みながら、Apple、iOS談義しましょう。そろそろ発表されるっぽいAppleWatchやiOS8.3について語りたいですね。
会場は元町駅周辺で、予算は3000円程度です。
