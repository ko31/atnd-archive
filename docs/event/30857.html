エフェクト作成ツール「Effekseer」の勉強会です。

Effekseerとは

Effekseerは主にゲームを対象としたエフェクト作成ツール＆ランタイムです。
オープンソースで開発されていて、無料で誰でも利用することができます。

  • 配布サイト（Google Code）
  • 公式Wiki

勉強会内容

  • 13:20 Effekseerチュートリアル
  • 14:00 Effekseer機能解説
  • 14:20 作例紹介
  • 15:00 自由時間

自由時間は参加者がエフェクトを自由に作って見せ合ったりする時間です。
開発者も来ますので、要望なり感想なりを伝えることもできます。

もちもの

  • ノートPC
  • Effekseer

あるといいもの

  • マウス
  • Photoshop、GIMP等。画像編集ソフト

備考

何か発表ネタがある方はコメントで書いていただけると嬉しいです。
会場にプロジェクターが無いようなので、持っていて貸してくれる方大募集。
→貸してくださる方が見つかりました。

参加費

参加費は、ルノアールの使用料が\1,100 * 4h = \4,400 + 一人最低ワンドリンクとなっていますので、それを参加人数で割ります。
いろいろ飲み食いして大体\2,000くらいです。参加者が多くなるほど安くなる仕組み！
参加者が増えて、主催者のミスにより1時間短くなった(5h→4h)ので\1,000くらいになる見込みです。

場所

開催はルノアール（喫茶店）の奥にあるマイスペース会議室で行います。
JR秋葉原駅 電気街口から徒歩5分くらいの所にあります。
「エフェクト勉強会」で予約してあります。開始の10分前から入場できるようです。
電源は使用可能です。延長タップを借りる予定です。
