概要

JAWS-US沖縄主催にて、朝からクラウド勉強会、お昼からはビーチパーティをしよう、というのがCloud on the BEACH!という企画の趣旨です。
今回は完全未経験者向けの企画ですので、クラウドってなんだろう？というあなたやAWSって聞いたことあるんだけど使ったこと無いんだよなーというあなたなど、どなたでもご参加頂けます。

ゲストスピーカーにアマゾン堀内様、サーバワークス大石様、クラウドパック後藤様をお呼びしています

特に話題沸騰サーバワークス大石様の「AWS専業SIerの舞台裏」は、県内SIerの皆様も是非、お聴き逃しなく！

ところでAmazon Web Services(AWS)とは？

Amazon EC2やAmazon S3に代表されるAWSクラウド = Amazon Web Services(AWS)は、ユーザの要望に応じて機能がどんどん追加され、まさにかゆいところに手が届くクラウドサービスになってきています。

個人でも瞬時に必要なサーバー、リソースが安価な料金で調達でき、データベースやロードバランサまで使うことができます。企業でも、業務で使えるクラウドとしてはAWSが一番だと市場の評価が確立しています。
