##【ベンチャーの働き方】第1弾!!!!!!
>新卒採用のイベントに参加すると必ず聞かれることとして「リクルートで働いているときと比較して、、、」という質問があります。
多くの新卒学生が大手で働くことと、ベンチャー企業で働くことについて考える機会があり、さらにどちらかというとベンチャー企業に惹かれる学生の多くがリクルートのことも気になっている傾向が多く見受けられます。
ならばいっそのこと私だけでなくリクルートに関わりのあった人たちから、フラットな目線で「働き方」についてお話できればと思っています。
大手で働くことって？ベンチャーで働くことって？さらにはリクルートではどう働いてきたのか、就活生なら考えておくべき今後の「働き方」について新たな視点を入れてもらえればと思います。
