大好評につき、またまた増席！

Oculus Game Jam in Japan 関西 とは？

関西でUnityとOculus Riftを使ってゲームを作ったり、
VR内に自分の嫁や夫を召喚させてみようというイベントです。
もちろんプログラム未経験者やデザイナの方も参加可能です！！

質問や「こんなことがやりたい」「こうすればもっと良くなる」といったコメントもお待ちしています。

概要

※内容は随時更新予定です

日時：12/14 (土) 10:00～18:30
場所：IIJ 様 関西オフィス

※10:00に1Fのシアトルズベストの前に集合して下さい。
遅刻や当日欠席などの際は、
・Twitterでハッシュタグ #Oculus関西 でツイートするか、
・@l___1cm___l (syyama)までDM下さい。

当日は、グループでOculusに関するゲームを作ります。
最後に各チームの制作したゲームを発表して解散となります。
もちろんゲームじゃなくても結構です。

アイデア帳を作りました。
当日までに書いておけば、チーム分けだけで済むのでスムーズに動けます。
参加の可否に問わず、どんな些細なことでも書いて頂ければ幸いです。
Oculus 関西 ネタ帳

注意事項

休日で閉館のため、出入りが不自由となっております。
入退室の際はIIJ様の社員の方にご協力を頂き、
裏口を開けてもらう形になりますので、ご理解よろしくお願いします。
