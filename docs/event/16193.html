Vyattaの開発者、ステファンを囲んでVyatta について語って呑もう！

米Vyattaのプリンシパル エンジニア、ステファンを囲んで Vyatta について語って呑もう！という会です。

Stephen Hemminger は Vyatta開発の他、Linux カーネルのネットワーク関連のコントリビューターとして “netem” network emulator や各種ドライバの開発に貢献しており、bridging と iproute のメンテナでもあります。

Vyattaはもちろん、ソフトウェアルーターやそのパフォーマンス・チューニングに興味がある方はぜひご参加ください。

また、昨年来日時には日本Vyattaユーザー会、第一回メンバーミーティングにて講演頂いてます。
※ 資料はこちらから：「ソフトルーターの概要と今後の展望」
http://www.vyatta-users.jp/2010/11/vyatta-4.html
