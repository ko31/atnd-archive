日本マイクロソフトの戦略

講師：日本マイクロソフト株式会社　
　　　クラウド部門ご担当者

クラウドを取り巻く最新トレンド、同社の「Cloud OS」戦略、その核となる
パブリッククラウド「Windows Azure」で加速するビジネスイノベーション、
そして、日本市場における同社のクラウド事業の最新動向について講演します。

クラウド最新動向セミナー

講師：株式会社FIXER
　　　代表取締役社長　松岡清一

今や「クラウドファースト」を超えて「クラウドマスト（必須）」の時代！

今回のセミナーはマイクロソフト パートナーアワードのクラウドサービス部門
において最優秀賞を受賞したFIXERの社長による講演！

マイクロソフトのクラウドプラットフォーム「Windows Azure」の導入設計から
運用、保守、監視までをサポートするフルマネージドサービス
『cloud.config（クラウド・コンフィグ）』を提供する、
株式会社FIXERの代表である松岡が、圧倒的な導入実績で培った技術とノウハウ
を語ります！

クラウドの新しい技術トレンド、市場動向、これから身につけるべきスキルにつ
いて講演予定です。クラウドの最前線情報について学べる絶好の機会です！

ぜひご参加下さい！！

講演者の概要

株式会社FIXER　代表取締役社長　松岡清一

  • プロフィール
    • 証券会社システム、自治体情報化コンサルティングを経て2009年11月にWeb Consulting Firm FIXERを設立。Webサイト構築に関わる企画・戦略立案～運用・更新までのプロジェクトマネジメント業務をワンストップで実現し、Webコンサルティングファームとして、そしてCSV（Cloud solution vender）として、良質なコミュニケーションを提案。2011年８月には、Windows Azureのフルマネージドサービス『cloud.config（クラウドコンフィグ）』をリリース。同スピンオフサービスとして、GREE開発者向けソーシャルゲーム開発支援・運用サポート『cloud.config for GREE』、動画プラットフォーム配信サービス『cloud.config動画』など、Windows Azureの柔軟なプラットフォームを活用したクラウドソリューションを提供し続けている。
