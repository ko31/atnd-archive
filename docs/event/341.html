■会費：200円(実費)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

　『WebアクセシビリティJIS(2004年制定)の改訂版となるJIS X 8341-3:2009について』FUMINGさん


　＜内容＞
　・Web Content Accessibility Guidelines(WCAG)について
　・JIS X 8341-3:2004について
　・WCAG2.0とJIS X 8341-3:2009の関係について
　・JIS X 8341-3:2009はどうなる予定？（公開レビュー版を元に）
　・Webアクセシビリティってなんだろう？

＜FUMINGさんの紹介＞
　今は、プーに近い状態なテストエンジニア。
　2000年にWebユーザビリティに興味を持ち始める。
　その流れでWebアクセシビリティにも興味を持つようになりました。

　ちなみに私はホームページ・ビルダー使い(^^;

　ホームページ・ビルダーの解説もやっていたりします。
