description

今年度は起業・創業する方にとって、とても魅力的な「創業補助金」という制度が設けられています。この補助金の補助率は3分の2で上限額は200万円（第二創業は500万円・海外需要獲得型は700万円）、原則的に返済不要です（一定以上の利益が発生した場合は返済の必要あり）。

今年度はすでに2回の採択結果が出ており、その採択率は約7～8割と非常に高い採択率となっています。この勉強会では、「創業補助金」の申請にあたり、これまでの採択案件の傾向をもとに、スムーズな申請の進め方、申請提出書類の書き方のポイントやコツについて、ざっくばらんに、そして具体的にお話しします。

また、勉強会終了後は個別相談もお受けしますので、「自分の起業が創業補助金の対象にフィットするのか」など申請にあたっての不安や悩みについても解消していただけます。
