“The book “Mastering the Unpredictable” is about a new approach to supporting work which we call “Adaptive Case Management” and it is for handling work of knowledge workers, who think on the job, and directly plan each instance of a process. Each instance of a process might be different, so instead of investing a lot of up front energy in designing the perfect process, one will provide the worker with a lot of information resources that they use to determine at run-time what to do next."
