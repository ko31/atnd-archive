概要

昨年12月に KDDI さんから Firefox OS 端末「Fx0」が発売されました。アプリケーションを web技術だけで作ることが出来るというおもしろいデバイスです。自分達でいろいろ作ってみる事にフォーカスした端末だと思いますので作り方を一緒に学びましょう。

過去の関西 Firefox OS 勉強会はガチ組み込み系の内容が多かったと思いますが、今回は主に webフロントエンジニアさん、デザイナーさんに向けた内容で開催したいと思います。

各自で用意するもの

ノートPC ( Firefox ブラウザをインストールしておいてください）
可能であれば Firefox OS の実機（ Fx0 や Flame など）
