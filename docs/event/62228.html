話すことに悩みがある方限定！！

・声が小さいといわれる
・朝礼で声が震える
・会議で振られると言葉に詰まる
・言いたいことがわからないといわれる
・話の輪に入るタイミングがわからない
・面接で思っていることを話せない
・プレゼンで一方的に話してしまう
・緊張しないで話せる自分になりたい

こんな方にお役に立てるアドバイスになっています！

話し方アドバイザーとして、多くの方のお悩みをお聞きしてきました。

この「話力チェック＆アドバイス」では、気楽にお話していただけて、

直接マンツーマンであなたの話力について、客観的なアドバイスを受けられます。

そして、今まで、あなたが意識していなかった改善点に気づくことができます。

★アドバイスの内容について

　次のようなアドバイスをさせていただきます。

　・緊張やあがり症をなおし、自信を持って話すためのアドバイス
　・コンプレックスや苦手意識をなくすアドバイス
　・発声や滑舌をよくするアドバイス
　・あなたに最適なトレーニング方法のアドバイス

　机上論ではなく、実践で使えるものをお伝えします。

【今までこのような方が参加されています】

・自分の話し方について具体的なアドバイスをもらえて参考になった。
・職場や飲み会などで会話を続ける方法がわかった。
・異性との付き合いで自分に何が欠けていたかわかった。
・自分のあがり症がトレーニングによって治るよう希望が持てた。
・面接官に好意をもたれる話し方を身につけるためのプログラムがあった。
・滑舌や歯切れのいい言葉で話すために何をすればいいかわかった。

ヒューマン話し方教室は、学校ではありません。
場かず、場馴れする場として、現在のあなたの能力をトレーニングするジムです。

あなたの「緊張とあがり症」の悩みを克服し、
相手に「伝わりやすい」話し方を身に付け、
あなたが「話に自信を」持てる。

そんな「適切なコーチ」が受けられる「話しのジム」です。

忙しい方は、必ずしもコース入学の必要はありません。
あなたの目的に必要なレッスンを必要なだけ受講することができます。
まずは、「話力チェック＆アドバイス」でご相談ください。

★話力チェック＆アドバイスの流れ

１．あなたのお悩みやお困りごとを伺います。
２．あなたの話し方に関して、発声、姿勢、視線、表情、メンタル、内容の観点から具体的なアドバイスをさせていただきます。

　　★トーククリニックシート
　　　終了時に現在の話し方のレベルがわかる「トーククリニックシート」を差し上げています。

　「話力チェック＆アドバイス」をきっかけに話し方だけでなく、よりよい人間性を発揮して自己実現への道を進んでいかれるよう願っております。
