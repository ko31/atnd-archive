★講演内容

～税制優遇を味方につけて、かんたん・かしこい資産形成術を学ぼう！～

個人が活用できる税制優遇制度の代表格「NISA、確定拠出年金、保険料控除」は、
資産形成を考える上で必ず活用しておきたいものです。
ただ、それぞれのメリット・デメリットを押さえたうえで活用しなければ
税制優遇制度も十分に力を発揮しません。
必要なのは、貯めながら増やす、貯めながら守ることです。
なにもしない人と実践している人では10年後、20年後に差がついてきます。

本講演では、ファイナンシャルプランナーにご登壇いただき、
「NISA」制度の変更点や、今年1月より対象者の範囲が拡大される「確定拠出年金」など
最近の税制変更のトピックスを交えながら、
フリーランス初心者向けに賢い活用法をお伝えします。
講演後には無料相談会も行う予定ですのでお気軽にご参加ください。
