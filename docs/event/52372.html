Google I/O でサンフランシスコへ行ったついでにおいしいカニを食べようというイベントです。

念のため書いておきますが、日時は現地の時間です。自動的にカレンダーに登録してる場合などに日本時間として解釈されると思うので注意してください。

イベントURL: http://atnd.org/events/52372

場所は毎年恒例の Crustacean Restaurant SF です。

これ とか これ の 2014年版です。

お店は 19:15 に15 名で予約してあります。

ドレスコードがあるので、エリのついている服(シャツとか、ポロシャツとか)でないと、食べさせてもらえないらしいです。

18:00 に Marriott Marquis に集合してその後みんなで移動します。直接お店に合流する方は連絡をお願いします。

過去の実績だと、予算は$80-$90 くらいです。みんなたべすぎ!!
