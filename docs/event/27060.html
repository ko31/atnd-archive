title 第47回「WEB TOUCH MEETING」やります！
catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：300円

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション1(40分)
　『自社ブランディングのすすめ』トライポット藤田さん

　＜内容＞
　自社の活動をアピールし、いい仕事環境を構築していくための方法を
　自社の３年間の活動を通して公開します。
　営業苦手だけど仕事を取りにいかないといけない人、
　自分のやりたい仕事をしたい人は是非、１つでも実行してみてください。
　ベラボー儲かってはないけど、楽しく仕事できてます。デスマよさらば！

　１「自社ホームページのリニューアルと更新」
　２「出せる実績つくり」
　３「コンテストに参加」
　４「ロゴマーク・商品を作る」
　５「好きな・得意な仕事が入ってくる体制づくり」
　６「手を挙げてもらえる営業体制」
　７「年賀状を出す」
　８「先生になる」
　９「ノウハウを出す」
　１０「イベントを主催する・手伝う」
　１１「チーム作り」
　１２「その他」

　
　＜トライポット藤田さんの紹介＞
　広島で、iPhone・iPad・Android・WindowsPhoneアプリ開発。
　デジタルサイネージ、ホームページ、Flash制作、システム開発とかやってます。
　デジタルサイネージアワード２０１０でインテル賞いただきました。

　●萌えとゆるキャラでスマートフォン（iPhone・iPad・Android・WindowsPhone）を盛り上げる！
　「萌えゆる」 http://moeyuru.com/

　●ITのテクノロジーで「楽しい」をつくる。
　「Augment8」 http://augment8.org/

　トライポット
　http://tripot.net/

　

●セッション2(40分)
　『「撃墜王ゲーム」のしくみとインスタレーションへの取組みについて』
　　ココノヱ丸山さん

　＜内容＞
　カードに描いた絵をモニタ上で戦わせる
　「撃墜王ゲーム（ http://gekitsuioh.jp/ ）」の制作に関する話と、
　デジタルインスタレーションへの取組みと可能性についてです。

　・撃墜王ゲームの紹介と開発の経緯
　・しくみ
　・演出とデザイン
　・アイデア
　・まとめ
　・今後の展開

　＜丸山さんの紹介＞
　　
　丸山 祐司（まるやま ゆうじ）
　ココノヱ
　ディレクター
　http://9ye.jp/
　ウェブサイトの制作・運営業務を中心に、
　現在はインスタレーション制作などにも取り組んでいます。

　

●セッション3(30分)
　『オンラインウェブサイトクリエイター「Jimdo」のご紹介』エコーワークス野崎さん

　＜内容＞
　誰でも簡単にWebサイトを持つことができる「Jimdo」昨年Googleが始めた
　「みんなのビジネスオンライン」で採用されて話題になりました。
　Jimdoの特徴や、Web制作者が利用する上で知っておくといいこと、
　カスタマイズの事例などをご紹介させて頂きます。

　＜プロフィール＞
　愛知県名古屋市を中心に活動する、Web制作のフリーランス。
　製造業の社内SE、自社サービスのサイト制作・管理などの経験を経て、
　2008年にフリーの制作として独立。2009年にKDDIウェブコミュニケーションズの
　Jimdoに触れ、衝撃を受ける。より多くの人にJimdoの便利さ、
　楽しさを伝えるべく解説ページ「Jimdoでつくる簡単ホームページ！」や
　制作＆カスタマイズサービスの「SITENEXT（サイトネクスト）」を運営。
　Jimdoに関するセミナーやイベントなども多数主催。

　JimdoExpert/JimdoEvangelist

　エコーワークス
　http://www.ewn.jp/

　Jimdoでつくる簡単ホームページ
　http://jp-m.jimdo.com/

　

●セッション4(40分)
　『KDDI ウェブコミュニケーションズの
　　デザインチームにおけるモチベーション復活劇』
　　KDDIウェブコミュニケーションズ高畑さん

　＜内容＞
　　2年前、当社のデザインチームはモチベーションが壊滅的で、
　　退職者が絶えませんでした。

　　体制を一新し、現在は好循環にまわっています。
　　何を変えたらモチベーションがあがったのか？何が諸悪の根源だったのか？

　　今だから話せるモチベーションとアウトプットの関係をお話します。

　＜高畑さんの紹介＞
　　高畑哲平（たかはた てっぺい）
　　KDDIウェブコミュニケーションズ
　　SMB事業本部　事業本部長

　　2006年、CPI（現KDDIウェブコミュニケーションズ）に入社。
　　マーケティングと新規事業立ち上げに従事し、携帯で会社のメールを
　　送受信する「モバイル＠ドメイン」を立ち上げる。
　　2007年、中小企業をベースに活動するSMB事業本部長に就任。
　　2009年、ドイツ発のWebサイト制作ツール「Jimdo」の日本独占販売権を取得し、
　　同サービスの立ち上げを行う。
　　2011年、Google、KDDIと共に“みんなのビジネスオンライン”を立ち上げる。
　　現在、レンタルサーバーCPI、Jimdo、CloudCore、boundioの事業責任者を努める。
