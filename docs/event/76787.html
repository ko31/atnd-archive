◆　クリプトン・フューチャー・メディア(株) (http://www.crypton.co.jp/)　：
"音で発想するチーム"をスローガンに、世界に百数社の提携先を持ち、日本国内においては1,000万件を超すサウンドコンテンツをライセンス販売する。DTMソフトウェア、サウンド配信サービス、音楽アグリゲーター、3DCGコンサートシステムなどの開発・運営も手掛け、DL販売サイト｢SONICWIRE｣(http://sonicwire.com/)がある。バーチャルシンガー『初音ミク』の開発会社であり、若者の音楽シーンに大きな影響を与えている。
　2015年から、１つのファイルから音声を分離するサービス｢APS｣(オーディオ・プロセッシング・サービス)も開始。マルチトラックで音声が残っていないような映像作品の音声からセリフだけを抜き出し、分離することができるほか、BGMとセリフが同一トラックになっているファイルからBGMだけを除去したり、音楽作品からボーカルと伴奏を分離させることも可能。セリフと背景音が分離できず、多言語化を諦めていた古い映像作品の再活用や、放送コンテンツでDVDにする際に著作権問題が生じるBGMの除去などができ、新たなビジネスチャンスを創出する可能性がある。
　当日は、APSを中心に、他のプロジェクトやAPSを可能にする"ADXテクノロジー"を用いたソフトウェア｢TRAX PRO｣などの紹介もする予定。
　＜プレゼンター＞
　　　　　SONICWIREチーム　チームリーダー　 田名部　茂　氏

◆ (株) フェリシモ (http://www.felissimo.co.jp/)：
　1965年創業の神戸に本社を置く大手カタログ通販・EC企業。自社企画のアパレル商品、ファッションアイテム、生活雑貨、セレクト商品やサービスなどをメインに急成長を遂げている。20代から30代の女性に強い支持がある。
東北での過去5年間の活動（http://www.motzutkit.com/）を踏まえながら2016年3月から、東北から「スター商品」を生み出すための新事業「Startline(スタートライン)」を開始。
これは東北の事業者が主役になって、未来につながる魅力的な商品を生み出せるように、ものづくり、ひとづくりをサポートする。ユニークな商品コンセプト」を生み出すためのトレーニングを行い、フェリシモの商品企画ノウハウを提供。大手企業との取引に必須となる品質基準や企画中の商品へのワンポイントアドバイスが受けられる個別相談会も実施する。全国に販路を持つ流通・小売の協力企業のバイヤーやプランナーを招き、定期的に「スター商品誕生オーディション」を開催し、会員が開発した新商品が全国デビューできる環境を作っている。
　当日は、これまでの東北での事業とStartlineを中心に、フェリシモのユニークな活動とビジネス戦略について語る。

◆　ヤマハ(株) (https://net.vocaloid.com/) ：
　歌声合成技術と応用ソフトウェアであるVOCALOID(ボーカロイド)を開発・提供している。 2015年12月からは、新たなアプリ、ゲームコンテンツ向け展開として、世界で大きなシェアを持つゲームエンジンであるUnityの開発環境で、VOCALOIDの歌声を使ったコンテンツが制作できるソフトウェア開発キット「VOCALOID SDK for Unity」の提供を開始した。これにより、誰もが自由にUnity上でVOCALOIDの歌声を利用した音楽ゲームやインタラクティブなコンテンツを開発できるようなった。
　ゲームのシチュエーションやユーザーの操作に連動して、リアルタイムに歌声を合成し、キャラクターに歌を唄わせたり、その歌い方を変化させたりするようなインタラクティブ・コンテンツを容易に制作できる。そのコンテンツには歌声合成エンジンが自動的に組み込まれるため、コンテンツの配布時に特別な対応は不要となる。VOCALOID SDK for Unityにインポートすることで、コンテンツ上の歌声音源として利用できる歌声ライブラリ「Unityランタイム版VOCALOID Library unity-chan!」も提供している。
　当日は、こうしたVOCALOID技術の新たな展開を中心に、最新動向とさまざまなコラボレーションの可能性について説明する予定。
　＜プレゼンター＞　事業開発部　VOCALOIDプロジェクト
　　　　　　　　　　　　　　　　　　　リーダー　　石川 克己　氏

◆　(株) 角川アスキー総合研究所 (http://www.lab-kadokawa.com/)：
　膨大なデータ量のTwitterツイートをリアルタイムで分析し、トレンドを視覚化する技術「Realtime trend analytics」を東京大学生産技術研究所喜連川研究室と共同開発。長い年月をかけて研究・開発した超高速言語解析に関する技術と、KADOKAWAのコンテンツ情報をもとに角川アスキー総研が開発したフィルター用ジャンル別辞書を組み合わせることで実現した。
設定された解析期間内での該当ワードの出現数をカウント、グラフ化して、ランキング形式で表示する。レーティングも算出。該当ワードが含まれるツイートのうち、リツイートとオリジナルツイートの割合も検出。ツイート単位で感情割合を判定する。特定のワードと一緒に書き込まれるワードを識別できる。今後は伝播経路分析、 未来予測機能を実装していく予定。
この技術を活用して、ツイートから「今」No.1な話題のエンタメ情報をリアルタイムにランキングで届けるスマートフォン向けアプリ「ついラン」を提供している。
当日は、Twitterのリアルタイムトレンド解析サービスを中心に、最新の話題について語る予定。
