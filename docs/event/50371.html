title 『常識を疑え！！』出版記念講演
catch 〜わたしにはどうしても伝えたい想いがあります〜
description

こんにちは、高木純です。

わたしには長年、どうしても伝えたい想いがあり、
今回ようやく、書籍を出版することによって、
その想いをかたちにすることができました。

わたしの伝えたかった想いとは、
「一つひとつの出逢いを大切にしたい」ということ、
そして「常識を疑い、非常識を選択する」ということです。

その想いの丈は、わたしの書籍
『常識を疑え！！上巻：日本国内立志編』
『常識を疑え！！下巻：海外武者修行編』に綴りました。

みなさんが、
失敗や挫折を恐れて、次の一歩を踏み出す勇気を失ってしまう前に、
年老いて、人生に感動がなくなってしまう前に、

わたしがこれまでの人生で得た「大切な想い」を伝えたいーー

ある意味それは、わたしに与えられた役割なのだと思っています。

　
これまでお会いした方にわたしの想いすべてを伝えるには限界がありました。

そんな時、本田健さんの「ユダヤ人大富豪の教え」を漫画化した
今谷鉄柱さんとの運命的な出会いにより、

これまでお会いした方に、そしてこれからお会いする方に、
もっとわかりやすく、わたしの想いを伝えられる方法として、
『常識を疑え！！』の出版が実現したのです。

　
まずは、東京と大阪で『常識を疑え！！』出版記念講演を開催します。

人生は有限であり、一度きり。
だからこそ、ひとつひとつの出会いを大切にしたいーー

そして、常識を疑い、非常識を選択することで、
人生を輝かしい成功へと導いてくれるーー

書籍を読んでくれた方、そうでない方、
もっと多くの方に直接お会いして、わたしの想いを伝えたいと思っています。

みなさんとお会いできるのを楽しみにしています。
それでは会場でお会いしましょう。
