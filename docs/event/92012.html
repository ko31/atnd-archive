参加費無料のワークショップ (先着20名)を、2017年11月15日(水)に東京・飯田橋にて開催いたします。スキルアップの 絶好の機会になると思いますので、ふるってお申し込みください。

主催 : 株式会社サポータス (FBA Platinumメンバー)
協力 : 株式会社ワークスペース (FBAリセラー)
後援：ファイルメーカー株式会社

テーマ : FileMaker Cloudで カスタム App を共有体験してみよう

前回開催後の参加者の皆さんからいただいたアンケートから、「目的がみえにくい」、「内容が広範囲過ぎる」等の感想をいただきました。いただいたご意見を踏まえ、「FileMaker Pro で作成した カスタム App をFileMaker Cloud にアップロードする」ということに重点をおいたワークショップを開催いたします。

「そもそも FileMaker プラットフォームって？」、「クラウドってなんなの？」という方のために、最初にFileMaker 製品のラインナップや FileMaker Cloud についての説明をファシリテーターが行います。まったくの初心者の方でも安心してご参加いただけます。

具体的には、以下のような内容でワークショップを展開する予定です。
(当日の進行状況により内容を一部変更する場合がございます)

==========
■ 開催要項
==========

FileMaker Cloud を立ててみよう。
参加者の皆さんで試行錯誤しながら、 FileMaker Cloud の立て方を体験していただきます。もちろん、ファシリテーターが皆さんをサポートいたします。

カスタム App を作成してみよう。
ファシリテーターが事前準備または当日ライブで作成いたします。参加者の皆さんは、FileMaker データベースの基本型や作成手順などをみていただきます。

カスタム App を FileMaker Server で共有しよう。
ファシリテーターが用意したカスタム App を、 FileMaker Server で共有し、FileMaker Go をダウンロードした参加者全員の iPhone や iPad で共有したカスタム App を開くことを体験していただきます。

FileMaker Cloud 上で少し高度な操作をやってみよう。
ファシリテーターのサポートのもと、参加者の皆さんで共有したカスタム App のカスタマイズを体験していただきます。フィールドを追加したり、セキュリティの設定をしたり・・・ FileMaker プラットフォームの可能性を感じていただけるはずです。

==========
■ 参加対象
==========
　初心者 〜 初級・中級の方を対象とします。
（初級、中級の方は、進行をサポートしていただくと、ワークショップに一体感が生まれます！）

==========
■ 参加者の皆さんにお願い
==========
iPhone や iPad をお持ちでしたら、事前に FileMaker Go を App Store からダウンロードしてご参加ください。ワークショップ当日は、ご持参のデバイスでも接続できるかテストしたいと思います。
