12/4～12/12　話題の映画『エクスペリメント』の特別割引鑑賞券　無料配布のお知らせ！

話題の映画『エクスペリメント』の特別割引鑑賞券を無料で配布します！！

１２/４(土)～１２/１２(日)の９日間だけ渋谷限定で使えます！

特別割引鑑賞券を無料で配布しちゃいます！！

どなたでも１０００円ポッキリで観れちゃいます！！

割引券の枚数も無制限です！！

割引券をGETするには　experiment1000yen@yahoo.co.jp　に「エクスペリメント割引券希望」という件名で空メールを送るだけ！！

★メールを送って頂いた方に割引券の画像データを返信いたします。
※割引券はプリントアウトして劇場にお持ちいただくので、メールの方は携帯メールよりPCメールのほうが良いかもしれません。

★画像をコピー、プリントアウトしてヒューマントラストシネマ渋谷に直接お持ちいただいてもOKです！！

★プリントアウトした割引券や画像のデータをご友人に配布してもOKです！！

※２０１０年１２月４日(土)～１２日(日)ヒューマントラストシネマ渋谷でのみお使い頂けます。
