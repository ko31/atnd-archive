【コミュニケーション能力向上カフェセミナー】

・会話が盛り上がらない
・取引先の人と何を話したらいいかわからない
・自分のコミュニケーション能力に自信がない
・営業でなかなか成績がのびない
・コミュニケーションに自信を持ちたい
etc・・・

このようなお悩みはありませんか？
今回は、カフェでまったりしながら、コミュニケーションのコツをつかんでいきます。

そもそもコミュニケーションって何？
というところから、
コミュニケーションのコツまでお話します。

コミュニケーション能力に自信がないのに、何もしないままでは、
残念ながら、何も変わりません。

無料開催ですので、えいっと一歩踏み出してみませんか？

カフェで開催するアットホームなセミナーですので、肩肘はらずにご参加ください(^^)

【担当講師】
社会伝達心理士　藤永晴人
これまで、5万人以上の人と出会い、コミュニケーションをとり、コミュニケーションのクセやコツについて研究してきた。
2014年から社会伝達心理士として、活動を開始。
学生や社会人問わず、様々な人に、コミュニケーションのコツを伝え、大学で講師としても活躍している。また、6月中に、大手生命保険会社でのセミナー講師も担当する予定である。
