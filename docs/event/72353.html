description

全国100万人のガジェットファンのみなさま！ 今年もKickstarterなどからモリモリとガジェットを購入したと思いますが、意外と最初だけ触って放置状態になってしまっているモノも多いはず。そんなお宝をみんなで持ち寄って、個人で売買し合おうというのが、ガジェットに特化したフリーマーケット「ガジェットフリマ」になります。

今年のうちに使わないものを「断捨離」してスペースをつくっておき、来年のガジェットをお迎えするのもよし。レアアイテムを「非売品」として陳列して自慢するもよし。ぜひご参加いただいて、買い物ついでにガジェットファン同士で好きなアイテムについて熱く語り合いましょう。

さらにガジェットファンが楽しめるトークセッションも開催します。

1つ目は日本のVR業界のパイオニアで、ガジェットを買いまくっているGOROmanさんをゲストに、「本音で語る！ Kickstarterベスト3、ワースト3」というテーマで熱く語ります。

2つ目は、日本一電化製品を買いまくっているであろう週刊アスキー編集部からACCNさんこと矢崎飛鳥さんをお迎えしてお送りする「週アス的、今年盛り上がったガジェットはコレだ！」。スマホを中心に、話題になったガジェットをゆるゆるとおさらいしていければと思います。

最後は、なぜそのアイテムを買ったのか、そして手放すに至ったのかを会場の出品者に聞いて回る「そのアイテムなぜ買ったの？」をお送りします。
