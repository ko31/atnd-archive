第3回千葉支部初心者向け勉強会

日本Androidの会千葉支部では第3回勉強会を開催いたします。

概要

第1回千葉支部勉強会で決定した年間スケジュールを元に、「Android開発を行いたいけれどもどうすればいいかわからない」方へ向けたスキルアップの場として勉強会を開催します。

開催日程

開催日：2011年6月18日(土)　13:00～17:45(受付12:45～)
場　所：流山市生涯学習センター A102会議室（最寄り駅：つくばエクスプレス流山セントラルパーク駅徒歩3分）
[会場には駐車場がございますが、できるだけ公共交通機関のご利用をお願いいたします。]

タイムテーブルと内容

前回の勉強会で要望の多かった内容を取り込んだ形で実施します。

  1. Android概要：Android開発を行う上での構造等の概要説明を行います。
  2. Androidの画面制御に関して：画面状態遷移に関する概要の説明を行います。
  3. Eventに関して：Button、ListViewなどからのEvent処理の説明を行います。
  4. 情報のＤＢ更新に関して：画面より入力した内容をSQLite3に格納・照会する方法の説明を行います。
  5. 画面遷移に関して：複数画面間でのパラメータ渡し、画面起動に関する説明を行います。

4.～5.については進行状況により進めたいと思います。

！注意！
当日は実際にプログラムを動作できる環境（※）をご用意ください。運営側でノートパソコンを用意することはできませんので、ご注意ください。

※開発できるスペックのノートパソコン
CPU：デュアルコア程度(インテル製CPU搭載)
メモリ：2GB以上(4GB程度以上が望ましい)

(Eclipseを使用した開発環境はあらかじめ構築されていることを前提に進めます。 開発環境構築手順書 )
