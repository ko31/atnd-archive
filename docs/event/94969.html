Microsoft Azure Stackの技術が半日で学べる -本物のハイブリッドクラウドの姿とは

いま押さえたい、Azure Stackの全貌

日本マイクロソフト×シスコ×三井情報×F5　4社ジョイントでお届けします。

デジタルトランスフォーメーションの促進に向けて多くの企業が求めるもの、それは迅速かつ柔軟なアプリケーション開発を実現するハイブリッドクラウド環境。

中でもいま、従来のパブリッククラウドの多くが抱えるセキュリティや法規制、パフォーマンス面での課題を解消する環境として、マイクロソフト社が提供するハイブリッドクラウドソリューション・Microsoft Azure Stackに熱い視線が注がれています。

Azure Stackは、Azureとの一貫性を前提に作られたプライベートクラウドとして、仮想マシンの構築のみにとどまらず、アプリケーションの開発環境やネットワーク・セキュリティなどの統合管理をオンプレミスで実現するシステムです。

本セミナーでは、主に、Azure Stackの基盤実現の要となるインフラエンジニアの方々に向けて、F5ネットワークス、マイクロソフト、シスコシステムズ、三井情報株式会社の4社が連携して、Azure Stackの全体像をわかりやすくお伝えします。

明日のより良い基盤の実現に向けて、ぜひ参加をご検討ください。
