description

横浜赤レンガでマスのジョッキをがっつんがっつんやりあう集いです。

「参加条件」
20歳以上（ビールイベントのため）
企画者とツイッター、メール、電話である程度リアルタイムに連絡がつく手段がある人
海外ビールが苦手でない人

「予算」
飲む量と銘柄に依存しますが、3000円あれば満足に飲めると思います。
がっつりいくなら、4000-5000円程度を推奨します。

「集合時間、場所」
14:00　JR桜木町駅改札周辺
途中参加となる場合は、個別に前述手段にて連絡をお願いします

「その他」
開催時間を遅めに設定しています、昼食は各自判断でお願いします（個別に取ることを推奨します）
2次会は自由としますが、要望があるならそのまま中華街になだれ込みます

「参考資料：会場イベント開催概要（公式より引用）」
4月26日(金) ～ 5月6日(月・祝)

＊荒天の場合中止する場合があります
＊グッズ＆カルチャーゾーンは4月27日(土)より営業
＊アトラクションゾーンは天候によっては休業する場合があります。

■時間 11:00～21:30
（アトラクションゾーン・グッズ＆カルチャーゾーンは11:00～18:00）
＊4月26日(金)のみ17:00～21:00
(アトラクションゾーンは15:00～18:00)
