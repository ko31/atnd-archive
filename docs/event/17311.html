完全合法SIMロック解除の旅　2011

アメリカでは携帯電話のSIMロック解除は、公式・非公式に限らず合法とされています。
一方日本では、孫正義（通称禿、ハゲのバーコードちゃん、つるピッカ、ハゲゲの鬼太郎、ハゲおやじ）が首を縦に振らないなどの理由で違法とされる場合があります。

そこで、SIMロック解除のためだけにアメリカへ渡航し、SIMロック解除を行って日本へ帰国します。
これにより完全合法でSIMロック解除が行えます。
