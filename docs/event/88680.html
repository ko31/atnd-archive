description

予算が限られている中小企業こそ、
求人戦略も選択と集中。

あらかじめコツを抑えておけばホームページから欲しい人材を集めることができます。
実際に弊社が製作した採用ホームページで成果のあった事例を紹介し、狙った人材の心をつかむ方法を伝授いたします。
ぜひ、御社の採用活動にお役立てください。

オープン形式のセミナーとなっておりますので、他社に発注をお考えの方も、お気軽にご参加くださいね。

立石 大樹（株式会社フォノグラム 経営企画部 人事担当）

システムエンジニア、経営管理などの職を経験後、フォノグラムに転職。
10年にわたり、経営企画・人事採用に携わっている。
