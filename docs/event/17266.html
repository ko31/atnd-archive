静岡県浜松市を拠点に活動するオタク向けパーティ団体「Project H.A.P.」主宰

1999年頃にアニソンとユーロビートを主軸としたDJのキャリアをスタート。
過去には東京・名古屋・大阪等、様々な地域のパーティにDJ出演。
アニソン原曲から4つ打系まで幅広いDJがプレイ得意。

そのDJプレイのルーツは関東地区のコスプレダンスパーティにおけるディスコスタイルDJであり、アニソンを1コーラスカットインでガシガシつなげるDJプレイでは抜群の安定感とスピード感でオーディエンスをロックする。

現在は地元のヲタク文化活性化のため、地元静岡県での活動が中心。
Project H.A.P.としては、アニソンDJ＆LIVEイベントの「HAMAMATSU A-POP PARTY!」や、
東方クラブイベント「東方御阿礼響」等を企画し、人気を博している。

その他にも、静岡市のアニソンクラブイベント「だん☆れぼ」や
富士市のコスプレダンパイベント「DANCE NAITION」にも参加中。

最近はMacBookProを購入しPCDJにもついに参入。
今後の活躍にも目が離せない。
