ご新規の方ようこそ！まずは立川人狼ゲーム会ウェブサイトをご覧ください。

　はじめまして。ここは新規参加の方専用のページです。過去に本ゲーム会主催のイベントに参加したことがある方は、立川人狼ゲーム会ウェブサイトより、通常の参加登録ページをご利用ください。
　なお、新規参加の方は、14:00までにお越しください。午前中からの参加も大歓迎です。

■年齢による時間制限について

　参加者の皆さんが安全に遊ぶことができるよう、年齢による時間制限を設けています。
　・小学生まで…必ず中学生以上の方による引率が必要です。
　・中学生まで…18:00まで遊ぶことができます。
　・高校生まで…19:30まで遊ぶことができます。

■参加費について

開催14日前までのお申し込み 開催13日前からのお申し込み
新規の方 無料 無料
学生の方 300円 500円
一般の方 700円 1000円

　一人でも多くの方にご参加いただきたいので、参加登録後にご都合が悪くなられた場合には、できる限り早めのキャンセルにご協力をお願いいたします。

みなさまのご参加を心よりお待ちしております。　　　立川人狼ゲーム会　スタッフ一同
