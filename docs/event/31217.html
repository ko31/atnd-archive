イベント内容

  • リポジトリ登録について
    [発表者：木谷さん] (20分)

    過去資料を掘り起こしていたらWordPressとの出会いはWordBench滋賀・京都がOSCで発表されたイベントに参加したことが切っ掛けと発覚。そのあたりも面白そうなので１ページぐらいで紹介する予定です。で概要としては
    WordPressとの出会いを紹介しながら、プラグインのリポジトリ登録をする過程の具体的な作業内容について紐解きます。後半は、リポジトリ登録に必要なコマンド説明など若干技術的な説明になりますが、出来るだけ分かりやすく説明します。


  • 静的ページからテーマ作成への実演
    [発表者：山本さん @_mayurin] (130分　※間に休憩あり)

    今回発表側になりました、まゆりん（@_mayurin）です。
    普段の仕事はグラフィックデザイン会社で、WEBデザインやコーディングを1人でさせていただいています。
    MovableTypeや、EC-Cubeの構築経験は少しあるのですが、WordPress構築の経験はありません。
    そんなWordPress初心者の私が、WordPress構築経験者のアドバイスを受けながら、ライブコーディングに挑戦します。

    要は、「あらかじめ静的にHTML化されたデータ（フラダンス教室のWEBサイト）をWordPressに組み込んでいく作業」です。

    全ページを対象にすると時間が足りないので、

    目標を立てます。

    1. ロリポップの簡単インストールでインストールしたWordPressの管理者画面より、初期設定方法（デフォルトプラグイン設定など）
    2. デフォルトテーマ[Twenty Eleven]を、子テーマのCSSだけで、見た目をカスタマイズ！（TOPページ、下層1ページのみ）
    3. 新しいお知らせをWordPressで（HTML知識の無いお客さまでも）簡単に投稿できるようにする。
    4. パンくずリストの設定方法
    5. SEO対策→ページ毎に異なるkeyword,descriptionの設定方法
    6. WordPressのプラグインを使ってお問い合わせフォームを作成
    7. グローバルナビゲーションの現在位置表示ページの部分だけ、表現を変える方法

    を実践しながら紹介できればと思っています。

    実際に静的にコーディングしたサイトは配布する※予定ですので、
    一緒にWordPress構築していただくことも可能です。
    ※今回の勉強会で、配布するサイトデータは、2次配布、再利用しての公開は禁止（JavaScriptや、jQweryはライセンスに基づく）とさせていただきます。

    初心者の方は一緒に、質問を交えて疑問点をクリアにしていきましょう。
    中級者、上級者の方はやさしく「あーだ、こーだ」アドバイスお願いします！

    できるだけ時間内に紹介できるようにしますが、全部紹介できない場合は、ごめんなさい。
    その場合はWEBミーティングで続きができたらとも考えています。

    どうぞよろしくお願いします！
