内容

ディレクション講座です。
社内に向けた講座を、せっかくなのでオープン参加にしよう、という流れになりました。

数回に分けて行う可能性があるので、初回はワークフローの説明・各フェーズにおけるディレクターの役割など、概要がメインになる予定です。

半分講座、半分ワークショップを行う予定で、ワークショップでは、数名のチームでWBS（制作ガントチャート）を実際に作ってもらい、タスクのブレイクダウンと日程読みのポイントを押さえたいと思います。
