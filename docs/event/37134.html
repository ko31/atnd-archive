title GIFカソン
catch みんなでGIFアニメを作りましょう
description

みんなで集まってGIFアニメを作るイベント「GIFカソン」を開催します。

いろんな物でコマ撮りしたり、お互い被写体になって特撮映像を撮ったりしましょう。
基本的なGIFの作り方も編集部メンバーがお教えします。初心者の方も遠慮なくどうぞ。

もうすでにGIF作ったけど容量が大きくてアワードに応募できない！なんていう人も、作品データ持ってきてもらえれば石川ができる限り小さくします。

参加希望の方はこのページで参加登録してください。

  • 途中参加、途中退出もOK。仕事帰りの方もどうぞ
  • 以下の どちらか一方 をお持ちください（両方でもいいです）
    • スマートフォン
    • デジカメとノートパソコン
  • 自分で被写体にしたいものがあればお持ちください
  • ごはんは出ません （おなかが空きそうな人は持参してください。食べている様子をGIF化しましょう）→軽食が提供できることになりました！

参加登録された方には当日までにATNDメッセージをお送りします。ご来場前にこのページを必ずチェックしてください。

※メッセージを見ないと入り方がわかりません。Twitter、メール等でのご連絡はいたしませんのでご注意を！

質問等はこのページのコメント欄へどうぞ。
