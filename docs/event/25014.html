    • R-bloggers
    • The R Journal

    • parallel
    • foreach
    • bigmemory, ff …

      • BLAS/LAPACK
      • Eigen

    • NumPy/SciPy/Matplotlib/IPython

    • MATLAB/Octave/Scilab
    • Mathematica/Maxima
