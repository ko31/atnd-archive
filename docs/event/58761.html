description

肩こりの原因と現状の身体の把握に特化したセミナーです。

●肩こりの要因・原因が分かります。
●肩こりに関係した、自分の身体の状態(どこが硬い)を把握できます。
●肩こり解消に何をすべきか分かります。

＊本セミナーではエクササイズまでは行いません。
肩こり解消エクササイズをご希望の方は、本セミナー後、19:30～20:30に行われます
「肩こりをストレッチで解消！あなたに合ったエクササイズで今すぐ楽に！」のセミナーも、合わせてご参加ください。
２つのセミナーを連続して受講されることで、更に効果が高まります。

･････････・・・・・・・・・・・・・・・

肩こりには、実に様々な要素が関係します。
・ストレス
・仕事環境(イス・机・パソコンのモニターの高さ、ウデの位置）
・姿勢(立位及び座っているときの姿勢)
・身体の使い方(正しく身体が使えているか)
など、実に様々な要因が考えられますが、
本セミナーは、身体の問題で肩がこっている人を対象にいたします。
（ストレスなどが原因のこりは対象外とさせていただきます。）

「肩こりを解消する」というゴールのためには、「現在地＝現状の身体の状態」の把握が必須です。
現状の身体の把握なしに、最適な「ルート＝トレーニング方法」の実施はあり得ません。

本セミナーは「現状の身体の状態」を把握するため「動作テスト」と呼ばれるチェックを行います。
そのチェックを行うことで、身体のどこが硬いのか把握できます。
どこが硬いのか把握すれば、なんのエクササイズを行えば効果が出るのか分かります。

肩こりの原因と現状の把握に特化したセミナーです。

＊＊動きやすい服装(寝転がっても大丈夫な服装、ジャージ大歓迎です）筆記用具をお持ちください。
