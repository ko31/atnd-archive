description

Zope/Ploneの開発者で集まってもくもくと開発したり色々相談したりとかやろうと思います。

スプリントのゆるい版みたいな感じで開催します。基本、毎月開催の方向で。参加費は無料です。

会場は無線LANと有線LANがあります。

Zope/Ploneの開発に興味の有る方は、是非ともこの機会に一緒に勉強しましょう。

当日のタイムテーブルはだいたいこんな感じのつもりです。

開発と飲んでいる時間の長さが同じくらいな気がしてきたので、開発の時間を1時間延ばしてみました。

  • 11:00-11:30 自己紹介とか
  • 11:30-12:00 やること決めたり準備とか
  • 12:00-13:00 昼食
  • 13:00-18:00 もくもくと開発
  • 18:00-19:00 成果発表(できたとこまで)
  • 19:00-19:30 後片付け
  • 19:30- 打ち上げ(いける人で)
