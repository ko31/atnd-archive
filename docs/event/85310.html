ついぴの会について

ピアノが好きな方であればどなたでもご参加可能です。ピアノ連弾や練習中の曲、暗譜でなくてもOKの「ゆる〜い弾き合い会」です。ピアノが弾けないけど聴いてみたい、という方も、ピアノ以外の楽器の方も、ぜひ気軽にご参加ください！

初めての方は「ついぴとは」もご覧ください。

開催概要

日時：2017年2月18日（土） 17:00〜21:00

会場：ヤギヤ（金沢市寺地1-5-21）

参加費：演奏者 3,500円、聴くだけ 無料（人数により前後します）

定員：18名

おなじみクラシックライブを楽しめるカフェ、ヤギヤさんでの開催です。ベヒシュタインの音色、ピアノ仲間との交流を楽しみましょう！

  • お腹のすく時間帯だと思うので、いつものように持ち寄りでの飲食を考えています。詳細は開催が近づいて人数がある程度分かり次第お知らせします。
  • 当日は開場時間の17:00の少し前までお店が営業しています。チャージ無しのティータイムコンサートも14:30から行われるので、コーヒーを飲みながらお店の雰囲気を楽しみたい方はそちらもぜひ。

参加方法（聴くだけの方もお願いします）

  1. 本ページの「イベントに参加登録する」ボタンからエントリーしてください。 Twitter, Facebook, Google+ のいずれかのアカウントが必要です。
  2. 2月11日中までにアンケート回答をコメントボードに記入する。（アンケートは下記参照）
  3. 当日、元気についぴに参加する。

アンケート（2月11日までにお願いします）

  1. お名前（Twitter, Facebook, 本名いずれも可）
  2. 演奏される方は曲名、作曲者、演奏時間（8分以内）
    聴くだけの方は「聴くだけ」とお書きください
  3. 自己紹介または演奏する曲についてのコメント
  4. ひとこと質問：あなたの雪の思い出を教えてください
  5. その他
    • 8分を超える場合は幹事にお気軽にご相談ください
    • 駐車場に限りがありますので、車で来場を希望される方はその旨をお書きいただけると助かります
