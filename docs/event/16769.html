この度、BoPInnovationLabでは、今回初の試みとして、世界中からラボメンバーが参集するウェビナーを開催します。

ウェビナーとは：ウェビナーとはWeb（ウェブ）とSeminar（セミナー）を組み合わせて出来た造語のことで、インターネット上で行われるセミナーのこと(ref)

実はラボメンバーは３５０人を超えましたが、皆、世界中に散らばっていて、なかなか会う機会がありません。東京にいるメンバーは既にface to faceの活動をいくつか続けていますが、そのアップデートもなかなかシェアされていないのが現状。みなさんの悩み事から生まれた発想で、オンラインでいいので、色々経験や疑問をシェアして、とにかく話をしよう、という動きを作り出すことにしました。

そこで今回は、メンバーの一人、紀古鮎美さんにお願いをして、彼女がケニアのECOTACTという社会的起業家の下で、数ヶ月のインターンをしてきたことを中心にお話をしていただくことにしました。

今回のウェビナープロセス：

  • Skype参加者として申し込みをしていただいた場合、事前の資料URLなどが送付されます。
  • Skype参加者として申し込みいただかなくても、Twitterのハッシュタグを追うことで参加できます。
  • 当日は、紀古さんの経験に対してBoPinnovationLabの槌屋からインタビューする形でスタート。
  • 参加者の皆さんにお願いし、ツイッターでも適宜内容を流していきます。
  • 今回は音声はSkypeで実施。同時接続で23人参加可能なので、定員は23名になります。
  • Skype参加の23名の方は、質問をどんどんしてください。
  • その後、Skype参加ではない皆さんからツイッターや事前のメールなどで頂いている質問を投げます。
  • 終了後、Skype参加の皆さんにはアンケートなどに答えてもらったり、グループメンバーに入っていただき、その後のウェビナーや勉強会の告知も優先的にさせていただきます。

以上、もし興味があれば、是非、登録してください！
