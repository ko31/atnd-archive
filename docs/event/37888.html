title 第55回「WEB TOUCH MEETING」やります！
catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション1(30分)
　『e-learning CMS Moodleについてと、大学での活用事例』森下さん

　＜主な内容＞
　　・e-learningについての簡単な復習
　　・moodleについて
　　・大学における活用事例の紹介

　＜森下さんの簡単な紹介＞
　　広島の大学生で大学内外でもくもくと勉強中。
　　Rubyが好きで広島Ruby勉強会によく出没。
　　エディタはSublime Text 2を使用。
　　新しい刺激を求めるために近くHaskellを学ぶ予定。

　　Facebook : https://www.facebook.com/tomohiro.m0219
　　Github : https://github.com/moriC
　　Twitter : https://twitter.com/CentBoss

　

●セッション2(40分)
　『もう怖くない！？WEBアプリケーションのDB運用』曽根さん

　＜主な内容＞
　　名著「SQLアンチーパターン」
　　を題材にしてWEBエンジニアの困ったあるあるネタの解決案をご紹介します。
　　またOSSなRDBのPostgreSQLとMySQLを比較しながら実務的なテクニックから
　　ちょっと驚きのテクニックまでご紹介します。

　＜曽根さんの紹介＞
　　岡山Python勉強会主催、オープンラボ備後、日本PostgreSQLユーザ会所属
　　日々、WEB 系エンジニアとしてインフラからプログラミングまで
　　設計・実装を幅広く担当。

　　曽根 壮大(そね　たけとも)
　　日本PostgreSQLユーザ会　中国支部長

　　http://soudai1025.blogspot.jp/

　　twitter@soudai1025

　

●セッション3(40分)
　『Passbook　～O2Oとボクと、時々、CM～』ブリコラージュの山城戸さん

　＜主な内容＞
　　ふざけたタイトルですみません。
　　最近耳にする事がある「O2O　（Online to Offline）」の説明をします。
　　その中で、iOS6からの新機能であるPassbookがどのように使えそうかを話します。
　　モバイルITアジアのカンファレンスで行った発表をアレンジした内容です。
　　なので時々、宣伝が入るかも。

　＜山城戸さんのプロフィール＞
　　家電屋さんに修飾語、組み込み屋さんでIT業界にデビュー。
　　その後システム屋さんとして暗躍。
　　日本Androidの会で勉強会に参加するうち、持ち前の胡散臭さを買われて、転職。
　　しかし転職先はiPhoneとWEB技術のエンジニアを求めていた！
　　どうなる山城戸さん！

　　株式会社ブリコラージュ
　　http://www.bric.jp/

　　会社のFacebook
　　https://www.facebook.com/bric.jp
