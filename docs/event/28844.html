概要

  • 数学の愛好者たちが集まってやさしい入門書をゆっくり読む、初心者向け読書会です。
  • 大田春外 『高校と大学をむすぶ幾何学』 から第4章「円周率」を読む予定です。
    ※ 当日までにテキストの用意が難しい方は、ご相談ください。
  • やさしく丁寧に解説された本なので、特に予備知識は必要ありません。
  • 学生の方も社会人の方も、お気軽にご参加ください。
  • 今回で 4回目となりますが、初参加の方も大歓迎です。飛び入りでもどうぞ。
  • これまでの勉強会の板書（写真）です。
    • 第1回「ピタゴラスの定理」
    • 第2回「三角形の基本定理」
    • 第3回「タレスとユークリッド」
