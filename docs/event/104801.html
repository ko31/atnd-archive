description

独立起業して成功したい！
誰もが思っているけど、
実際に成功できる人はあまりいません。

そもそも実際にビジネスで成功するには
何が必要だと思いますか？

  • 資金
  • 人脈
  • 経営力
  • 人間力
  • 営業力

もちろん、これらも大事です。
でも一番必要なものは、

お客さんに売る、モノやサービス。
つまり『コンテンツ』が一番大事です！！

成功した人に共通するのはヒットコンテンツを作り出し持っていることです。

そのヒットコンテンツを数多く生み出し、
実際に成功した人から直接学べるのが今回の講座です。

ヒットコンテンツ以前に、

  • 突き抜けたアイデアなんて思いつかない。
  • アイデアマンでもなんでもない私には難しいよ。

そんな風に思っていませんか？

たしかに最初から優れたアイデアをポンポン出して、それを行動に移せる天才もいるでしょう。
でも、天才だけができるビジネスではありません。

現に多くの人がビジネスを始める前は、いわゆる普通の人でした。

どのような考え方に変化したのか？
そこからどんな努力をしたのか？
何を学んできたのか？

それをしっかり抑えることが大事です。
学校で勉強する学問もそうですが、ものごとには順番があります。

簡単に言えば、
ビジネスを成功するためには、努力する順番が決まっている。
そう言って過言ではありません。

ほとんどの人がその努力する順番を間違ってしまったから、
失敗してしまっているだけなのです。

努力する順番がちがっても成功する人もいます。
しかし成功するまでには10年20年と長い時間がかかってしまいます。
そんな努力する順番を間違いながらも、どうにか頑張って成功までたどり着いた。

それはそれで素晴らしいことですが、
わざわざ大変な道を私達が同じ道を通らなくても良いとは思いませんか？

努力することに変わりはありませんが、
可能であればできるだけカンタンなほうを選びたいと思うのが人情です。
まず知ってほしいことは
ヒットコンテンツを生み出すには、

努力する順序を間違えないことが大事！！

起業や独立、副業でもなんでも
自分の力でやっていこうという人の悩みは
本当に多いです。

  • これから起業しようとチャンスをつかみたい
  • 一度失敗してしまったけど、諦めきれない
  • サラリーマンを抜け出したいけど、何から勉強していいのかわからない
  • 独立したけど、毎月毎月ギリギリの状態。そこから抜け出したい
  • 独学では限界が見えているし、人と比べて突き抜けた自分のウリが欲しい

でも、実はこのような悩みはヒットコンテンツに関わる３つの極意があれば、
すべて解決してしまいます。

その極意とは

●　コンテンツを作ること

●　コンテンツを人に伝えて喜んでもらう

●　コンテンツを提供し続ける

たったコレだけです。

多くの人は

「作り方がわからない」
「上手な伝え方を知らない」
「続けていくために必要なものが抜けている」

このベースとなる部分をしっかり抑えていくのが今回の講座のメインテーマになっています。

講師を務める古谷氏は、コンサルが本業ではありません。
『正しい知識と意識の持ち方が、日本の幸せ指数向上に繋がる！！』
を理念としてビジネス活動されています。

ビジネススタートにあたっては、コネ、専門知識、大きな資金はありませんでした。

そんな状態から脱サラして故郷の湘南にオープンした、
一軒の格闘技＆フィットネスジムから始まり、
「ダイエット検定」「ルーシーダットン」「和ハーブ・プロジェクト」
などなど、様々なヒットコンテンツをゼロから立ち上げ、
それを全て全国メジャーレベルに引き上げてきました。

継続にあたっては、借金、赤字もありません。

そして活動当初からブレることがありませんでした。

このような人物からコンテンツの極意を教わる機会は他にはありません。

特に美容業界、健康業界で独立・起業を考えている人には、
すでに同じ業界で成功している古谷氏との接点をもつだけでも、
人生が大きく変わっていくことでしょう。

『本当突き抜けた結果をに実現してきた人』から直接学べるチャンス

『発想』を逆転させたビジネス思考法をメソッドとして多くの起業家、経営者へ提供しています。

常識や固定概念に捉われがちな世間において、相反する概念を並行することで、
他の人が発想・追随できない『トンガリ』が確立されます。

誰もが出来る。

でも、そもそも誰も発想することができない。
だからこそ突き抜けられる。

まさに究極の実践ビジネスノウハウを学べます。

これはビジネスだけにとどまりません。

人の生き方・幸せの作り方といっても過言ではありません。
自己啓発に希望を持ち、コンサルに食い物のされるのは終わり。

講師の古谷氏は実はコンサルが生業ではありません。
古谷氏が実際にビジネスしてきたことを直接学べます。

本来は古谷氏と身近で一緒にビジネスをやっている人間しか共有されることのないノウハウ。
それが今回、ついに直接指導を受けられるチャンスがやってきました。
社会に感謝され、自分にブレない、１０年継続する事業。

幸せな仕事人生を実現したい人には、必ず来て欲しい、体験して欲しい講座です。
