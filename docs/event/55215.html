【ゆるゆるカラオケオフ】は、こんな人にオススメ～


会社終わって、帰宅前に軽～くカラオケしたい

お酒もジュースも飲み放題がイイ

ヒトカラよりも、みんなで楽しくワイワイしながらカラオケした～い

今夜の予定がいきなりなくなった～どうしよ～っ

突然カラオケに行きたくなってしまった～

カラオケは苦手だけど、みんなとワイワイした～い



【ゆるゆるカラオケオフ】は、こんな感じでやりますよ～

途中参加、退室

好きな時に演奏停止

前の人の歌パクり

人の歌聞かない

１曲しか歌わない

歌うジャンルはなんでも

カラオケ苦手な人は初回は飲みだけの参加も
