description

卓越した企画力・表現力をもつアドバイザーと
「企画」を考え、体験する4時間。
思考が澄みゆく爽快感が、あなたの脳を駆け抜ける。

売り上げをあげる、話題をつくる、商品開発する…。
仕事のあらゆるシーンで必要とされる、企画をつくる力。
課題の本質を見極め、的確な手法を提案し、実行していくよりよいプロセスは、
どのような状態で生み出されていくのか。

ボケない大阪移住プロジェクトの3人のアドバイザーをゲストに迎え、
各プログラムに参加する参加者や一般参加のみなさんと一緒に、
企業課題を解決するための企画公開会議を開催します。

ボケない大阪移住プロジェクトにエントリーし、面談を通過された方以外にも、
本イベントや大阪移住に興味関心があれば、どなたでも参加可能

ぜひお気軽にご参加ください！
