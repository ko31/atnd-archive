アドバンテック株式会社と日本ラッド株式会社は、2018年07月05日(木)に新大阪駅前、および2018年07月12日(木)に虎ノ門・共同通信会館ビルにて、「IoT ソリューションカンファレンス」を開催致します。
◆セミナー名：「IoT ソリューション カンファレンス」： http://iot-sense.jp/event/201807solutionday/
製造業のIoT化ニーズが、もの凄い勢いで高まっている中、ソリューションベンダー各社の方向性が出揃ってきて、皆様のニーズに対応できるものとできないもの、難易度の高いもの低いものそれぞれに階層が出てきたように思う昨今、我々はオープンなプラットフォームを駆使して、本当に皆様が進めていきたいIoT化の実現を小規模から進められるように支援できます。

産業用PCのリーディングカンパニーであるアドバンテックとハードウェアとソフトウェアを組み合わせたシステムインテグレーションに定評のある日本ラッドが提供する製造現場で活用できるIoTのソリューションを、よりたくさんの方々にご覧いただけるよう、社員一同、皆様のご来場を心よりお待ち致しております。

ただのプレゼンレクチャーに留まらず、
実際にIoTの現場で取り扱われている最新技術の機械を皆様の前で実演し披露いたします。
動的なマシンを通じて、Factory Automation （FA＝工場の自動化）、Building Automation（BA＝建造物の自動化）、
そしてMachine Automation （MA＝機械の自動化）を、参加者が実際に目で確かめて学び、感じ取って頂く事ができます。
