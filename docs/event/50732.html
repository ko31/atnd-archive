セッション概要

セッション1

  • タイトル
    • Firefox OS アプリ開発クイックツアー
  • 講演者
    • 一般社団法人 Mozilla Japan　清水 智公　: @chikoski
  • 概要
    • Firefox OS のアプリは全て Web 技術、HTML＋CSS＋JavaScript で作成されます。
      普通の Web サイトとどこまで一緒で、どこが違うのか。
      本当に JavaScript だけで書けるのか。API には何があるのか。
      このようなことを30分で概観します。

      あなたをモバイルプラットフォームの中の世界へご案内します。
