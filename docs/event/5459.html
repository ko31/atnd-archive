title トウキョウミルキーウェイ
catch キャンドルナイト２６日(土)19:00～21:00＠ギャラリーツープラス
description

◆銀座エリアでは、有志画廊がキャンドルナイトの呼びかけに共鳴しました。

いっとき照明を消し、キャンドルライトの素朴な灯りにココロをゆだねます。

また、２６日(土)は、人気企画「銀座・夜のギャラリー巡り」。

トウキョウミルキーウェイ参加ギャラリーをツアー・コンダクターの先導でまわります。ツアー最終地点のギャラリーにて、パーティーを開催。

詳細は、http://tokyomilkyway.org/　をご覧ください。

◆さて、銀座１丁目のギャラリーツープラスでは、「ノスタルジア・アンティーク・マーケット」を開催。

カナダを仕入先とし、ヴィンテージの洋服、服飾雑貨、食器などを販売する店「シュープリーム」(銀座１丁目)がプロディースするプチ・アンティーク・フェアです。
アンティーク雑貨店を中心に、９店舗が集まり展示即売。

フランス、アメリカ、北欧、カナダ、日本のアンティークや、レトロ物。また、廃棄金属をリユースしたアクセサリーを販売。カフェも併設します。ゆったりとしたひとときを過ごしていただけます。

◆会期：6月19日(土)～7月1日(木)
※会期中無休 ※12:00～19:00

☆26日(土)は、キャンドルナイトです。21:00まで営業します。

お問合せは、会場のギャラリーツープラス(加藤)までどうぞ。
