これからのアプリ・WEBサービス作りに必要な「ノウハウ」×「UX（ユーザーエクスペリエンス）」×「ゲーミフィケーション」について、各分野のエキスパートから学べるイベントが大阪で実現！貴重な機会ですので、お見逃しなく。

※予定・演題は直前まで変更になることもございますので、ご了承ください。

横井太輔： 株式会社エルイズビー代表取締役

演題と内容

1.「お笑いUX道場～人の行動を刺激するUX（ユーザーエクスペリエンス）のツボはここだ」：坂井とわ子

人間はほんとうにおもしろい。人の行動を刺激するUX（ユーザーエクスペリエンス）のツボはここだ

  1. スタートアップになぜデザインやデザイナーが必要なのか
  2. ７つの大罪を刺激するUXとは
  3. UX技法（ウルトラエックス）
    • 人の言うことを聞きすぎてはいけない
    • お金や時間をかけようとしてはいけない
    • 仲間を増やしすぎてはいけない
    • 理想のユーザーをつくりこんではいけない
    • UXを勉強しようとしてはいけない
  4. デザイナーの扱い方
    • スロースターター型
    • ビッグドリーム型
    • 天然型
  5. これからきそうなUX
    • ノンバーバル（感覚情報のコミュニケーション）
    • ノンタッチ（手を使わない）
    • ノーエイジ

2.ゲーム企画の立て方とゲーミフィケーション：定元邦浩

  1. FBアプリの歴史を振り返り、成功要因を分析する。
  2. 日本発で世界を攻める場合のゲーム企画の立て方。
  3. ゲーミフィケーションの功罪について考える。

3.アプリ開発秘話（トークセッション）：横井太輔＆安達貞雄

  1. アプリのご紹介
  2. 発想のきっかけ
  3. 開発中のエピソード
  4. Japan Nightを経験して
  5. 資金調達のポイント/体験談
  6. これから目指すもの
  7. サービス作りにおけるノウハウ
  8. 質疑応答

4.「How to change the World 拡張現実で世界を変えよう！"世界頓智の旅"」：井口尊仁

2008年にシリコンバレーで産声をあげた頓智はよりオープンマインドな世界を実現する為、セカイカメラを始め拡張現実技術を通じたグローバル展開を進めています。 ソーシャル・ネットワークの次世代にどういった時代を創造するべきなのか？拡張現実でモバイルインターネットがいかに進化を遂げて行くのか？ 現実の体験、試行錯誤を経て体得したリアル・ストーリーをお届けします！
