“知育アプリを作ろう！ Kwikで作るデジタル絵本” 勉強会

簡単なデジタル絵本アプリの作り方を学習します。

Kwiksher社が開発したPhotoshopプラグイン
Kwikを使って開発を行います。

Photoshopを使って、iPhoneやAndroid用の
デジタル絵本がつくれちゃいます！

プログラマーはもちろん、デザイナーさん向けの環境なので
アプリ出したいけどプログラムはちょっと・・・
っていう方も参加してみてください！

今回は、導入編ということで、環境を用意していないかたでも
参加できる形式にしたいと思います！

もちろん、環境を用意していただければ
ワークショップ形式で実際に試してもらえるので、用意できる方は
ノートパソコン持参でお願いします！

対象

デジタル絵本、知育アプリ開発に興味がある方
Photoshopを使用しますので、Photoshopをお使いの方が対象になります。
プログラミング知識をほとんど必要としないため
デザイナーさんに最適です！

Kwikとは？

Kwik2

Kwiksher社が開発したPhotoshopプラグインです。
Photoshopがあれば簡単にスマートフォン用のデジタル絵本アプリを
作成することができます。

プログラミング知識が要らないため、デザイナーやイラストレータが
インタラクティブ要素のあるデジタル絵本を作るのに適しています。

※Kwikの動作には、PhotoshopとCorona SDKが必要です。

ノートパソコンについて

ノートパソコンの持ち込みをしていただくと、
実際に試しながら勉強会に参加出来ます。

※講師はmacで行います。Windowsのサポートは行いません。

下記ソフトのインストールが必要です。

・Photoshop
※CS5以降（エレメンツ不可）

・Kwik2
http://www.kwiksher.com/download-kwik2/

・Corona SDK
https://developer.coronalabs.com/user/login?destination=downloads/corona-sdk

※Kwik2とCoronaSDKのインストールについては解説しますのでインストールしていなくてもOKです。（macのみ解説）
