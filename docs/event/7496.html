和菓子・洋菓子・パンの魅力が満載、おいしくて楽しい文化祭です！学生たちが心を込めておもてなしします。
『第２回　学生祭』が１０月３日（日）、午前１０時から午後４時まで開催されます。
東京製菓学校と製菓製パン業界がタイアップした盛大なイベントです。もちろん入場は無料。
展示コーナーでは、関東近県の和菓子店・洋菓子店・パン店・ホテルが出店予定で、伝統ある老舗から今話題の人気店まで約６０店舗が集まります。
また、各学生のイベントでは実演や体験、さまざまなテイストを盛り込んだカフェなど学生自身が企画・運営し、趣向をこらした内容となる予定。
その他、本校教師による「アメ細工」や「上生菓子」のデモンストレーションや、有名店のお菓子を特別価格で購入できる？！
子供から大人までお菓子好きにはたまらない催しものとなること間違いナシのイベントです。
