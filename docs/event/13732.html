【プログラム】
■講師の言葉
　本格的な医療機器が開発されてから100年余りが経過しました。我が国においては、現在でも、「医療機器法」のような独自の法律がなく、薬事法の中で規制されているのが現状です。この薬事法の本質を知らずして、医療機器開発が不可能であることは、いうまでもありません。したがって、その本質や特質を知ることによって、新しい医療機器開発参入の道も開けてきます。
　本セミナーでは、医療機器独自の特殊性に触れながら、新しい医療機器を開発するにはどうしたらよいか、という観点からわかりやすく解説したいと考えています。薬事法の基本的なことが理解できれば、医療機器開発の方向性も明瞭となります。これから医療・福祉機器業界に参入したい方、すでに参入していても、薬事法の規制にとまどっている方などに対して、このセミナーがその指針・ヒントを与えられることを願っています。

◆修得知識
・医療機器の本質が理解できます
・医療機器開発の方向性が明確になります
・薬事法をクリアするにはどうすべきか、という基本が理解できます
・各企業が独自の医療機器事業を展開するに当たって、その目標が明確になります

◆プログラム
Ⅰ. 医療・福祉機器概論
　　1.医療・福祉機器開発の歴史
　　2.医療機器開発の特徴
　　3.最新の医療機器情報
　　4.医療・福祉機器市場と将来の需要

Ⅱ. 薬事法の概要　
　　1.薬事法とは何か
　　　　a.薬事法の基本
　　　　b.海外での規制との相違

　　2.医療機器関連の標準規格
　　　　a.日本標準規格
　　　　b.ISO
　　　　c.IEC

　　3.薬事法申請手続き全般
　　　　a.業許可と品目認証・承認
　　　　b.医療機器のクラス分類
　　　　c.医療機器申請のための要件

Ⅲ.医療機器開発における薬事法対応
　　1.薬事法の特徴とその対応策
　　　　a.薬事法と特許法との大いなる違い
　　　　b.申請時の留意事項
　　　　c.審査時の留意事項
　　　　d.QSM対応の要点

　　2.どう対応すれば良いのか
　　　　ーそれぞれの企業に合致した事業化をー
　　　　a.販売業としての事業
　　　　b.OEM生産-製造業としての事業
　　　　c.本格的生産-製造業・製造販売業
　　　　d.海外製品の取り扱い

　　3.ポイントは何か

　　　　b.必要条件の理解とその対処
　　　　c.商品企画時からの対応
