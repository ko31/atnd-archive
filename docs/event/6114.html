手続き型言語のプログラム経験者を対象にした Erlang の基礎勉強会の第二回です。
今回も少人数で行います。

今回は Erlang プロセスの基礎から応用まで、一緒に考えて行く予定です。
gen_* が今ひとつ解らない方は、是非、ご参加ください。

実際にコードを書いて頂きますので、Erlang VM がインストールされたノート PC をお持ちください。

第一回の内容を把握していると理解が早いと思われます。第一回の内容は、下記の URL をご参照ください。
http://github.com/cooldaemon/BasicStudyForErlang

今回利用するサンプルコードはこちらです。
