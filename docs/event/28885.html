第21回勉強会を開催します。

今回は丹所さんに講師をお願いして、AndroidでOpenGLです。

・内容：OpenGL/ESハンズオン〜AndroidでOpenGL/ESを体験しよう
　　　　講師　スマート・ラボ　丹所　育男　氏
　　　　・OpenGL/ESとは
　　　　・AndroidのView
　　　　・OpenGL/ESハンズオン
　　　　　プロジェクトの作成・初期化
　　　　　三角形の描画
　※OpenGL/ESの説明から、画面の初期化、簡単な描画までを行う予定です。
　　　また、Android特有の問題についてもお話しできればと思います。
　※OpenGL/ESは、携帯端末などの組み込み向けの3Dコンピュータグラフィクス用のAPI群ですが、今回は2Dを扱います。

・準備：各自のPCに、最新のAndroidの開発環境（EclipseやADTなど）の準備をお願いします。
・端末について：動作確認はエミュレータでも可能ですが、実機があれば持参ください（Android 1.6以上）

　よろしくお願いします。
