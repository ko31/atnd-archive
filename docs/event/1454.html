description

mixiアプリの一般公開で日本でもかなり盛り上がり始めているSocialWeb。

東京でたびたび勉強会が開かれていますが、このたび名古屋でも勉強会を開催することになりました！

今、日本で一番ホットなOpenSocialを中心に、知識の共有・議論ができればと思います。

対象

SNSのビジネス、OpenSocial、mixiアプリ、OpenID、OAuth、Friend Connect、etc…
興味のある方ならWEBプロデューサ・エンジニア・学生など、どなたでもOKです。
名古屋近辺でWEBに興味がある人達と触れ合える機会なのでぜひご参加ください。

参加方法

このページの右上にある「このイベントに参加登録する」から参加登録してください。

※ 当日はルーセントタワー内の高層階用エレベータで32Fにお越しください。

タイムテーブル

  • 19:00～ 開場・受付
  • 19:10 ～ 開会の挨拶
  • 19:20 ～ 勉強会
  • 21:00 ～ 懇親会

講演内容

  • ソーシャルウェブの可能性、OpenSocial(株式会社エイチーム 川渕貴志)
  • 携帯SNSのオープン化(N氏)
  • 韓国のOpenSocial(J氏)
  • OpenSocial入門 (株式会社エイチーム 川渕貴志)

服装

お堅い勉強会ではないのでカジュアルな服装で構いません。
会社帰りの方も多いかと思いますので、もちろんスーツでも問題ありません。
