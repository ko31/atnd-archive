『ローカリゼーションの視点から日本製品の海外進出戦略を考える』

title 『ローカリゼーションの視点から日本製品の海外進出戦略を考える』
catch 〜「マルちゃん」はなぜ海外で国民食になったのか？〜
description

「自社製品を、どうにかして“海外”に売り出したい。」
　そう考えている社会人の方は多いのではないでしょうか？

“日本の魅力を発信する”活動をする和えるが、この度、
「いかにして日本製品を海外進出させるか？」という趣旨で、実際にイタリアで活躍されているビジネスプランナー、安西洋之氏と弊社代表取締役の矢島里佳の公開対談セミナーを開催いたします。
　
第１部では実際にイタリアで活躍するビジネスプランナー、安西洋之氏と共に海外進出の本質を考えます。安西氏の専門分野であるローカリゼーションの視点から、実際のビジネス経験を交え、海外進出の考え方からその事例まで、わかりやすくお話します。
第２部では少し視点を変え、ご活躍なさっている安西氏がどのように育ってきたのか、自らの子供のころを振りかえながら、異なった文化に関心をもつ子どもに育てるにはどのような環境をつくると良いのかを考えます。

【講座会詳細】
第1部 「日本の製品の海外進出〜ローカリゼーションの本質〜」

グローバル化が進む世界の中でMade in Japanの商品をどう海外に広めるか？　ビジネスプランナー安西洋之氏は、いかに海外の文化や文脈に合わせた製品デザインをするかが大事だと考えていらっしゃいます。その考え方や主な事例、本質に迫ります。

第2部「輝く子どもの育ち方研究ラボ〜輝く大人はどうやって育ってきたのか〜」

このような考えを持つ安西氏の幼少期にフォーカスします。ご活躍なさっている安西氏がどのように育ってきたのか、自らの子供のころを振りかえながら、異なった文化に関心をもつ子どもに育てるにはどのような環境をつくると良いのかを考えます。

【講師紹介】
ゲスト
安西洋之（あんざい　ひろゆき）氏
1983年上智大学文学部仏文学科卒業後、日本の自動車メーカーに勤務。1990年よりイタリア在住。ビジネスプランナー。1995年よりメトロクスのヨーロッパにおける商品開発コンサルタントでもある。著書に『ヨーロッパの目、日本の目—文化のリアリティを読み解く』（日本評論社）『「マルちゃん」はなぜメキシコの国民食になったのか？ 世界で売れる商品の異文化対応力』（日経BP社）がある

モデレーター
矢島里佳（やじま　りか）
2011年慶應義塾大学法学部政治学科卒業後、伝統を次世代に繋ぐ株式会社和える代表取締役就任。高校時代TVチャンピオン2「なでしこ礼儀作法王選手権」優勝（テレビ東京）。2010年2月ビジネスコンテスト「学生起業家選手権」優勝（東京都・財団法人東京都中小企業振興公社）。著書に『その常識もしかして非常識？！自分を魅せる本当のマナー』（高陵社書店）、他2冊有り。 現在、慶應義塾大学大学院 政策・メディア研究科修士課程１年在学中。
