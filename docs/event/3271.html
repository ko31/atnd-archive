DTrace に関する話題を集中的に扱う勉強会です。

DTrace が稼働するプラットフォームは、Solaris 以外にも　FreeBSD/MacOS X と広がりを見せていますので、これらのユーザも巻き込んで皆で DTrace を使い倒そうぜ！という意図で、今回はイベントタイトルから OpenSolaris の文字を抜いております。

他プラットフォームユーザの方も、是非ご参加ください。

以下のような話題を扱います。

  • pid プロバイダを用いたユーザプログラムからの情報採取
  • 独自プロバイダを用いたユーザプログラムからの情報採取
  • DTrace によるシステム解析
  • DTrace による性能チューニング
  • その他
