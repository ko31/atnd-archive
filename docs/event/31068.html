今回はいよいよCNCを使った実習です。　基板用CNCであるオリジナルマインドCIP100を直接使用できます！　CIP100の開発者も講師に迎える豪華内容！

自分のPC

作成する基板について

主催者が用意した回路図を元に、マイコン回路を作成します。
実習時間はCADでのパターン作成、CNCでの基板加工、パーツのはんだづけとなります。
CNCでの基板加工の実習は必須となりますが、CADとはんだづけは自信が無ければ座学のみでもOK。

作成する基板のパターンについて

ミニミニArduino互換マイコンキットの回路図を用意します。これを元に自分で回路のパターンを作ります。意欲のある人は回路の拡張設計も試してみてください。

自分の希望の回路をつくりたい人

希望により、完全オリジナルのパターンを作ることができます。切削時間の制限やパーツの用意などもありますので、事前にご相談ください。

見学について

参加者以外の見学大歓迎です。

講習時に使用できる機材

はんだごて、はんだごて台、はんだ、CNC、ニッパー、ラジオペンチ、フラックス、テスター、すずめっき線、
ベークライト基板など

ご注意

  • 実習中で使用する回路CADはKICADとなります。他に自分で使い慣れた回路CADがある
    場合は各自ご用意ください。
  • 持ち込みPCは回路CADを動かすのに使います。KICADはWindows/Linuxに対応してい
    ます。Macにも実験的に対応しておりますが、ご希望の場合は事前にご相談ください。
  • 加工可能な基板はベークライト基板です。
  • 加工可能な線幅は0.8mmまでです。
