<href=“http://office.microsoft.com/ja-jp/lync/”>Microsoft Lyncを構築して遊ぶハンズオン大会です。
有志が勝手に機器を持ち寄って、無慈悲な仕様に頭を抱えるイベントです。

やること

Lyncが他のIP-PBXと繋がるかどうかを検証します。
Lyncなんてサッパリなので、ハンズオンで触って慣れ親しんでみます。

  1. Microsoft Lyncを構築しよう
  2. CiscoのIP-PBXで内線網を構築しよう
  3. Microsoft LyncとCiscoの内線網をつなげてみよう
  4. 会議通話をしてみよう
  5. Microsoft LyncとAsteriskの内線網をつなげてみよう(オプション)
  6. Microsoft Lyncと黒電話をつなげてみよう(オプション)

会場および集合場所

国際産業技術様のセミナールームをお借りする予定です
会場に変更がある場合は別途お知らせします

注意点など

  • パケットの挙動に悩まされるので、ネットワークパケットを読む環境があると便利です。
    (ダムHUBやtcpdump採れるPCなど)
  • Cisco電話機の持ち込み大歓迎です。
  • 音声系の機器なら大体持ち込みOKですが、Lyncと接続できなくても泣かないでください。

スケジュール

機材がたくさんある人は11時入場のうえ、機材を設置します。
機材が少なめな人は13時開始となります

13:20 ～ 構築開始
17:30 ～ 後片付け
18:00 ～ 撤収完了
