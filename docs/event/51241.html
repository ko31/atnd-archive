description

NTTドコモなど携帯３社のeco検定対策の公式サイトの運営や各種環境ビジネスセミナーなどで環境教育に取り組んでいる「環境教育フォーラム21」は第16回eco検定(環境社会検定・7月27日に実施）合格に向けて、6月20日東京・渋谷など全国各地で「重要ポイント集中講座」を開催します。
◎試験によくでる重要テーマ別に講義
　「重要ポイント集中講座」では地球温暖化、循環型社会の構築、エネルギー問題、環境と経営などeco検定にでやすい問題を2月に発売された公式テキスト第4版に沿ってテーマ別にわかりやすく講義します。eco検定で増える傾向にある時事問題もテーマにあわせて解説します。
◎忙しい社会人の皆様から高い評価
　当フォーラムのeco検定講座は講義内容が初心者にもわかりやすく、配布する重要ポイント暗記ノート、重要ニュース集など豊富な教材も自宅での学習に役立つと大変、好評です。特に、仕事などで日々お忙しい日々を送られている社会人方から講義、資料ともに合格に効果があったと高い評価をいただいています。
ホームーページは　http://ecoken.newsmedia.jp/
