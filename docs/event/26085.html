スポットライトを浴びよう！

市民が作る舞台　座・大阪市民劇場　第9回公演出演者キャストオーディション

　～新しい自分に出会えるチャンス！！～
幅広い世代間交流が最大の魅力！！

　半年ごとにメンバー集い、公演後解散する、市民参加型劇団秋９月公演キャスト募集！
5歳～80歳の色んな職業、世代の仲間と作る舞台。
実行委員は、プロがボランティアでバックアップ。 普段の生活では、出会うことのない幅広い世代の仲間とふれ合い、つながる人の繋がりもこの舞台ならではの特徴。

説明会&面接は2012年3月25日（日）15時～ ※無料
「中津芸術文化村ピエロハーバー」にて開催。

●応募資格 5歳～80歳までの男女、経験不問。
大阪市民以外でも、稽古場（大阪中津）に通える方であれば応募可。
未成年者は、詳細ＵＲＬを保護者様にもご覧頂いた上でご応募下さい。また、キャリアをお持ちの方も、こんな趣旨なら参加してみたい♪という方は沢山います。是非一緒に楽しみましょう。舞台の原点は「夢を共有」そして「観る人も出演する人も一緒に楽しめる」ということですから

公演参加者は毎週日曜稽古を行い
第9回　座・大阪市民劇場公演に出演※公演終了後、解散

●公演日　2012年9月15、16、17日（3日間）

●備考　稽古場代として　月/1万円

◆稽古日程 ４月ー９月の毎週日曜
（13時～15時※会場の状況で曜日変更あり）
◆公演日 2012年9月15,16,17日に舞台上演後、解散です。
◆演目 「新釈竹取物語　婿取りの編」（仮）演出は元梅田コマ劇場の仲風見先生

◆応募方法

「座・大阪市民劇場の件」と題し、名前、連絡先、年齢、職業ご記入の上メール、ＦＡＸまたは郵送でお送り下さい。

〆切　3月20日
