Facebookやブログで書くネタが増える映画鑑賞法

「映画の見方がひとつやふたつ増える2時間を目指します！」

なにげに映画を見ていて、こんな経験はないでしょうか。

面白かった映画の感想を書こうと思っても、面白かった理由が自分でもよく分からない。あるいは、いい映画だと思うけど、うまく友だちに説明できなかった……。

誰もが一度は経験したことがあると思います。

しかし、その論理的なヒントが映像テクニックとシナリオにあったりします。

例えば、映画の成り立ちと密接に関わっている“映像技術”が少し分かれば、好きな映画監督のどこが好きなのか気づくこともあります。

またシナリオの基本が分かると、セリフがウマいかヘタかを論理的に説明できるようにもなります。

というわけで、映画を見る基準でモヤモヤしている方、映画の成り立ちの基礎について知りたい方、シナリオを書いてみたい方。

そんな方々を対象に「Facebookやブログで書くネタが増える映画鑑賞法」をキャッチフレーズに、いろいろお話しします。

たぶん基本的な映像テクニックやシナリオをご存じない方には、目からウロコ。映画の見方がひとつやふたつは増えます。

【講義内容（予定）】

■「映画の日」が神戸キッカケの理由

知っておきたい映画発祥のあれこれ。キネトスコープって映画ですか？ 素人ながらお話しします。

■映画好きなら知っておきたい映像技術の基本

映画好きでも、意外と知らない「メリエス」「エイゼンシュテイン」「グリフィス」がやったことは知っておきたい。

そのうえで、ジル・ドゥルーズ「シネマ論」など。
※小ネタ「魔女の宅急便」は黒澤明組しか撮れない。
グリフィス作品の撮影監督ビリー・ビッツァーはスゲーな。

■映画の小ネタ

映画を作っている人は大画面で映画を見ていない。
マスコミ関係者も大画面で映画を見ていない。
ゆえに、映画館で映画を見る必要なし？

■シナリオの基本知識

シナリオは映画のすべての基本です。
その書き方や、日本と海外とのシナリオの違い。
デビューできるシナリオ勉強法（野島伸司のノウハウ）をご紹介。

■シナリオの小ネタ

洋画のシナリオは結構ネットに転がっています（その探し方）。
シナリオを読むと脚本家が本当に描きたかったことを再確認できたりします。

■映画宣伝・企画の小ネタ

映画「ダイハード」が変えた（宣伝会議参照）。
ディズニー赤字で、今後数年はオリジナルなし？

■映画企画のはなし（時間があれば）

映画をもっとも楽しむ究極の手法は、自分で作ること。
コネなしのシナリオの売り込み方など。
