自動車触媒の構造・機能と劣化抑制技術

イベント ユーザー
event_id 14586
title 自動車触媒の構造・機能と劣化抑制技術
catch ～触媒技術の進化と触媒開発のヒント、劣化抑制技術、今後の研究開発の方向性～
description

【受講対象】
触媒開発・設計・関連部品、燃料電池など触媒に関わる広い分野の研究者、エンジニア。さらに、石油、天然ガスのクリーン化を目指す燃料メーカーの技術者・研究者の方々。

【習得知識】
自動車触媒に使われた、および現在も使われている材料や技術、これからの触媒開発の ヒント・方向性を探るための知識を知ることができる。

【講座のポイント】
　自動車触媒に関する研究開発の歴史は約40年、実用化されておよそ36数年しか経過していませんが、その技術は、その時々の環境浄化や資源問題を克服しながら、ナノテク技術を駆使して年々進歩の一途を辿ってきました。
　本セミナーでは、自動車触媒の更なる技術革新を促進すべく、過去に開発された軌跡を顧みます。ガソリン用三元触媒、リーンバーンガソリンエンジン用NOｘ吸蔵還元型触媒、ディーゼルエンジン用触媒それぞれの構造と機能・劣化抑制技術について解り易く解説することによって、これからの触媒開発のヒント、方向性を提示いたします。

【プログラム】
１．はじめに
　　1-1.触媒反応の立場から見た自動車排気の特質
　　1-2.自動車触媒開発の必要性－工業触媒との相違点－
　　　　1-2-1.希薄な反応ガス系における貴金属系触媒作用
　　　　1-2-2.自動車触媒の基本技術
　　1-3.排気規制の変遷と動向
２．ガソリンエンジン用三元触媒
　　2-1.三元触媒の設置環境
　　2-2.三元触媒の構造、機能（反応・呼吸作用）と劣化抑制
　　2-3.基材、触媒（貴金属）、担体、助触媒
　　　　　2-3-1.貴金属
　　　　　2-3-2.担体
　　　　　2-3-3.助触媒
　　2-4.新しい技術
　　　　2-4-1.ハニカム基材の改良
　　　　2-4-2.HC吸着材料としてのゼオライト利用
　　　　2-4-3.大量酸素貯蔵放出型三元触媒技術
３．リーンバーンガソリンエンジン用NOx吸蔵還元型触媒
　　3-1.リーンバーンガソリンエンジン用NOx吸蔵還元型触媒の設置環境
　　3-2.リーンバーンガソリンエンジン用NOx吸蔵還元型触媒の構造、機能（反応・呼吸作用）と劣化抑制
　　　　3-2-1.NOx吸蔵・還元
　　　　3-2-2.基材、触媒（貴金属）、担体、助触媒
４．ディーゼルエンジン用触媒
　　4-1.ディーゼルエンジン用触媒の設置環境
　　4-2.ディーゼルエンジン用触媒の構造、機能（反応・呼吸作用）と劣化抑制
　　　　4-2-1.基材、触媒（貴金属）、担体、助触媒
　　　　4-2-2.HC吸着・浄化
　　　　4-2-3.PM濾過・浄化
　　　　4-2-4.NOx吸蔵・還元
　　　　4-2.5.尿素選択還元
