description

　寒さも厳しくなりました今日この頃、皆様はいかがお過ごしでしょうか。我が、天王寺アップルクラブはお陰様をもちまして２０１４年も発展の年となりました。これも皆様からのご支援のお陰と会員一同、心より感謝いたしております。

　今年も様々な活動を行ってまいりました。会員数も少しずつですが増えております。また、ＴＡＣの本拠地である大阪でのＡＵＧＭ大阪へのスタッフ派出、今や第２の本拠地と言っても過言ではない東京でのＡＵＧＭ東京へのスタッフ派出、はたまた全国各地のＡＵＧＭへの参加などなど、小さな所帯ですがアップルＵＧでの存在感は大きくなっております。

　そこで、２０１５年を更なる飛躍の年とする第一歩として、ＴＡＣ活動の柱である、飲み会イベントを以下の要領で企画させていただきました。その名も「ＴＡＣ新年（餃子）会２０１５」です。

　ＴＡＣ会員はもちろんのこと、ＴＡＣにご興味がある（なくても結構ですｗ）方のご参加をお待ちしております。公私ともにお忙しい正月とは存じますが、是非ともご参加くださいますようよろしくお願い申し上げます。

ＴＡＣ（天王寺アップルクラブ）代表　佐竹宏康
