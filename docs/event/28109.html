主催

名古屋アジャイル勉強会。本イベントに関するお問い合わせ等はこちらからお願い致します。

イベント概要

書籍「インターフェイス指向設計」や「プレファクタリング」の著者 Ken Pugh 氏(@kpugh)が来日されます。名古屋にも訪れてトヨタ工場や産業技術記念館の見学もされるそうです。

  • Ken Pugh氏の経歴
  • 著書
    • プレファクタリング
    • インターフェイス指向設計
    • Lean-Agile Acceptance Test-Driven Development

名古屋滞在中に地元のIT技術者と意見交換のミーティングを行いましょうという話がでており、豊田でお会いしましょうということになっています。
つきましては、参加者を募集します。

イベント日時・会場

参加を希望される方はこのページでお申し込みの上、以下の日時に集合場所までお越しください。

  • 日時：4/24(火) 19:30
  • 集合場所：愛知環状鉄道線 新豊田駅 改札前

上記集合場所でおちあった後、近くのお店（その場でPugh氏の意向も聞いて決めます。今のところ居酒屋を想定）に移動して歓談の予定です。

  • 案
    1. 株式会社オージス総研様 豊田オフィス 会議室
    2. 馳走屋 てんごく 豊田店
    3. 北海ダイニング 康家
    4. 琉球和食 わらい場

なお、通訳的な人はいませんのでご了承ください。
それから、ITコンサルタントの原田 騎郎さんも参加される予定です。
