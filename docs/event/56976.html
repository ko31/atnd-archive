description

オープンソースハードウェアの勉強会をはじめます。第一回目は　RaspberryPiを使って Python をはじめて触るというぐらいにしようかなと。

内容

簡単なPythonのレクチャーと、簡単な電子回路の実験をやって二時間ぐらいのライトな勉強会。

対象

どなたでも。小学生あたりからチャレンジできる内容です。

日時

2014/9/28 (日) 15:00-17:00

場所

ギークハウス神田秋葉原
http://geeaki.github.io/
1Fが薬品会社の黒いビルで、狭い路地を入ったところに入口とインターホンがあります。近くにホルモン屋のでかい看板があります

費用

必須ではありませんがギークハウス神田秋葉原にカンパを！

プログラム

  1. 自己紹介
  2. Python3.2を使ってみる
  3. RaspberryPiを使ってPythonからLEDやらスイッチやらでif文、while文をほげる
  4. 上級者でつまんないヒトはADコンバータMCP3008を使ってアナログセンサーを作ってみる

使用機材

PC , Raspberry Pi, ブレッドボード, 簡単な電子部品, USBケーブル, フラットケーブル , テキスト , おいしいもの , LANケーブル

PCとおいしいものだけお持ちください。その他は全部貸出しします。

顔出し

ご注意。勉強会の風景は写真撮影します。顔出しNGの方は事前にお申し出ください。

会場について

ギークハウス神田秋葉原2Fで開催します。秋葉原近辺でオープンハードウェアのファブリケーション＆交流施設をたちあげようとしており、その準備としてギークハウス神田秋葉原の一角でいろんな活動を行おうとしています。

なお、家族での参加やベビーの持ち込みもWelcomeです。

主催

秘密結社オープンフォース
