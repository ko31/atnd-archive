■概要

　動画サービスの発展により、インターネット上で活躍する映像制作者が急速に増えています。
しかし、まだまだ引き出しが足りない。もっと技を磨きたい。周りに動画界隈の友達が少ない。などと思ってる人も少なくはないはず。
そんなCGクリエイターを対象にした、関西ベースの勉強会・交流会です。

元々は関東で行われている「AEオフ」というイベントを、関東AEオフ主催であるむらしん様に許可をいただき、今まで東京で行われていたために参加が困難だった関西の人たち向けに開催することになりました。

　今回で4度目の開催となり、第一回は2013年8月17日、第二回は2014年08月10日、第三回は2015年7月26日、第四回は2016年7月17日に開催されました。

　また参加が困難な方でもご視聴ができるようYou Tube Liveでの配信も予定しております。

今回は会場をお借りしている立命館大学映像学部様が10周年ということで特別講演を予定しています。

■会場・日時
