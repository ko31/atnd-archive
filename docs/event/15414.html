catch ～日本全国の美味しいものを麻布十番にお取り寄せ♪～
description

皆さんは美味しいスイーツと言うと、どこをイメージしますか？
銀座？六本木？田園調布？
確かにおっしゃる通りです。

しかしスイーツの穴場は他にもあるのです。
それが千葉県船橋市。

かつてテレ東で放送されていたテレビチャンピオンで
ケーキ職人王、和菓子職人王で3連覇した方々の店が船橋市周辺にあるのです！

今回用意するスイーツのお店はケーキ職人王3連覇した横山知之さんのお店で、
「ここを知らねえ千葉県民はモグリだ！！」とまで言われる人気店『ル・パティシエ・ヨコヤマ』です！

ル・パティシエ・ヨコヤマ
http://p-yokoyama.jp/

こちらのケーキはどれも絶品で、開店して10年経ちますが200個限定のシュークリームは
今でも午前中で売り切れるほどの人気っぷりです。

その中でも私のお勧めはロールケーキ。もう最高に美味しいです！
ここのロールケーキを食べて以来、ロールケーキの代名詞となった『堂島ロール』が
「うん…、まあ、美味しい…かな」と思ってしまうくらいに！

地元雑誌では取り上げられていますが、TVではほとんど取り上げられることがない穴場の店ですので
この機会にご体験ください！

今回場所をお借りするHiddenLoungeさんの料理も絶品ですので、
食事・スイーツ共に大満足の内容です！是非ご賞味あれ！！
