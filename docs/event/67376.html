BSoD Version 2.0 連携！ Unity ワークショップ

（BSoD Version 2.0 のATNDはこちら）

BSoDに出演したい！
でも、VJって何すればいいかわからない、かっこいい映像作れない、Processingじゃ物足りない…

そんなあなたに朗報です！
Unityさんにご協力いただき、連携Unityワークショップを開かせていただきます！

今回のワークショップは、 音と同期したテクニック が主となりますので、実際にVJにも応用しやすい内容となっています！

事前準備

事前に

  • Unity（起動することを確認しておく）
  • Data Racket

を導入しておいてください。

ワークショップ内容

名雪さん

  • Data Racketを用いてUnityに音声を送る
  • 音声に合わせたリアルタイム3Dオブジェクト生成

石橋さん

  • 音に同期したパーティクル制御
