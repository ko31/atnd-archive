イベント概要

北海道の小樽と札幌で開催している Drupal CMS（オープンソースのコンテンツ管理システム）の勉強会です。今回は札幌での開催です。名前は「勉強会」としていますが、セミナーのような堅苦しいものではなく、Drupal ユーザーが自由に情報交換できる場にしたいと考えています。気楽な集まりですので、お気軽にご参加ください。北海道の Drupal ユーザで情報交換と交流の輪を広げ、さらなる活用につなげていきましょう。

参加費：無料

内容

今回は、開発作業に関する題材も取り上げてハンズオン的にやってみたいと思います。パソコンをお持ちいただける方は一緒に操作しながら、そうでない方は他の人の操作を見ながら雰囲気を感じていただき、Drupal 活用のヒントやテクニックを共有しましょう。

(1) Ampps を利用したローカル Drupal 環境構築
Drupal の環境が手元にない方向けに、Ampps ベースのローカル環境構築手順を紹介します。
(参加者の方にニーズがなければ省略します)

(2) NetBeans とDrupal プラグインを利用した開発環境構築
Java の統合開発環境として有名な NetBeans は、PHP ベースの Web 開発でも広く利用されています。ここでは (1) の Ampps ローカル環境を前提に、NetBeans で Drupal 開発環境を構築する手順を紹介します。IDE の基本設定、デバッガ（xdebug）の設定、Drupal プラグインのインストール、テーマ/モジュール作成での利用など。

(3) 住所の自動入力フォーム
EC サイト等で欠かせない、郵便番号から住所を自動入力するフォームを Drupal7 で作る方法を紹介します。
