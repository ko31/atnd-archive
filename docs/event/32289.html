　とのコラボレーションで、WEB TOUCH MEETING の
　スペシャルバージョンを開催します。

　ますます注目される今、Web 開発者向けの Mozilla のイベントを
　中国地方でも！という声を受け、本イベントの開催が実現しました。
　HTML5 はもちろん、Web 標準技術がもたらすその先の未来、
　実際の開発現場で役立つレスポンシブツールの話題など、
　盛りだくさんの話題でお送りします。
　参加費無料。ぜひお早めにお申し込みください。

■現在予定している内容：
　

●セッション1(40分)
　『IAから見た、Webの世代遷移とHTML5時代のIAについて』
　ナナイロデザインワークス 薬師神裕樹さん

　＜主な内容＞
　wwwが生まれ、html、XHTML、そして時代はHTML5と移り変わっていく中で
　最近「UI」や「UX」はたまた「IA」といったワードを良く目にするようになりました。

　プログラマーでも、デザイナーでもコーダーでも無い
　インフォメーションアーキテクトという立場の視点から
　これまでのWeb標準の振り返りと、HTML5時代について
　レスポンシブやUXのお話も織り交ぜつつ
　IAについても触れられる内容をお話したいと思います。

　▼レジュメ
　・IAから見た、Webサイトの『世代』のお話。
　・レスポンシブのIAな話。
　・HTML5時代のIAについて。

　＜薬師神さんの紹介＞

　2000年頃より趣味でweb制作をはじめて、東京でwebのお仕事してましたが、今年6月広島に移転。
　EC制作、ネットベンチャー、大手SIer、フリーと渡り歩いた中で
　情報設計フェーズ、プロモーションでのディレクション経験が最も長く、
　今後も広島を拠点にIA、ディレクションでの立場でWebやデザインに関わっていきたい34歳。

　U理論やコーチングの手法を取り入れたモダンなファシリテーションアプローチを用いた
　戦略立案と、情報設計が得意です。

　

●セッション2(50分)
　『HTML5時代のWebデザインワークフロー』こもりまさあきさん

　＜主な内容＞
　閲覧デバイスの多様化や回線環境のカオス化により、
　これまで良しとされていた制作の常識がいま問われています。
　本セッションでは、ワークフローのことから
　ラフやモックアップ、素材の作り方と管理にいたるまで、
　これまでの常識では通用しない新しい世界に備えるために、
　いま覚えておきたい最新のコンテンツ制作技法を紹介します。

　＜こもりさんの紹介＞
　1972年生まれのフリーランス。1990年代前半に都内のDTP系デザイン会社にて
　アルバイトをはじめる。大学卒業後そのまま正社員となり、入出力業務、
　デザイン業務、ネットワーク関連業務に並行して従事。2001年、会社を退職し
　そのままフリーランスの道へ。
　案件ごとに業務内容や立ち位置が異なるため、職域的な肩書きはなし。

　### こもりセッション　アウトライン

　* デバイスの多様化に備えるコンテンツ制作
　* 変わりゆくWeb制作ワークフロー
　* Photoshopはもう古い？スタイルガイドとは
　* 未来を視野に入れた素材制作と管理（CSS3/Webフォント/ベクター）
　* モバイルファーストなレスポンシブWebデザインの実装（デモとか）

　

●セッション3(40分)
　『インスタレーションイベントのその後のウェブ再現』
　　シェアフル/カンドウコーポレーションの柳谷さん

　＜主な内容＞
　3000ページ自動生成化までの道のりのお話です。
　AIR SHODOU (http://sharefl.jp/projects/airshodou)
　AIR SHODOU VIEWER (http://www.sharefl.jp/projects/airshodou/data/)

　AIR SHODOUという赤外線センサー(KINECT)を使った
　体験型フィジカルインスタレーションを現代美術館で1ヶ月展示させてもらいました。
　1回1分間のプレイデータ、3000プレイ。
　たくさんのデータをウェブで誰でもみれるようにしたら楽しいはずとの思いから、
　コンテンツを作ることにしました。
　その過程についてのお話が、マークアップについてと
　イベント運営についての視点からみて面白い内容になりましたので、
　ご紹介させていただけたらと思います。

　普段のウェブ制作のノウハウを使い、普段とはちょっと違った頭をひねることで
　自動化を実現した、AIR SHODOU VIEWERコンテンツの
　自動生成化までの道のりのお話です。

　＜セッションの流れ＞
　・AIR SHODOUというKINECT赤外線センサーインスタレーション
　・1分再生しなければキャプチャーできない3000ファイルサムネイル化
　・最低限AnalyticsとFacebook OGタグを入れたページ3000ファイル一括生成
　・大きなプロジェクターでAIR SHODOU

　＜柳谷さんの紹介＞
　CANDO CORPORATIONに所属。
　そのほか、SHAREFLという非営利クリエイティブの場をつくり、
　企業、ビジネス、クライアントワークのしがらみのないものづくり活動も
　行っています。
　WEB TOUCH METINGでは、ウェブと写真、ウェブとロゴ、AIRならではのアプリ開発、
　JIMDOセミナー、KINECTについて、FlashMediaServerについて、
　ウェブならではの仕組みを盛り込んだゲームについて…など
　デザイナー目線でのウェブやプログラミングの考え方や
　ノウハウについてのお話でよく登壇させてもらっています。
　SHAREFL (http://www.sharefl.jp/)
　YAMPUU DOT COM (http://www.yampuu.com/)

　インタラクティブとデザインとイラストのお仕事。
　問題解決提案やユーザーへの印象付けについて提案する
　コーポレートサイトやポータルサイトをよく受注しています。
　山陰ディスティネーションキャンペーン (http://saninji.jp)
　ひろしま清盛(http://hiroshima-kiyomori.jp)

　

●セッション4(50分)
　『HTML5 と Web 技術の最新動向』Mozilla Japanの浅井智也さん

　＜主な内容＞
　"HTML5" は Web ブラウザ上はもちろん、モバイルアプリやデスクトップ
　アプリなどあらゆる環境で使われるアプリ開発の共通プラットフォームとなってきています。
　ではその “HTML5” では何ができ、いつからどこでどのように使えるのでしょうか？　HTML5 を中心とした Web 技術の標準化と実装について最新動向をまとめてお話しします。

　＜浅井さんの紹介＞
　学生時代に Web 技術文書を翻訳、Firefox 1.0 から Mozilla 製品の
　日本語リリースを担当。インターネットをすべての人々の利益になる方向へ発展させる
　Mozilla の目標に共感し、大学院卒業後 Mozilla Japan に参加。
　テクニカルマーケティングとして Mozilla の製品や技術の他、最新 Web 技術や
　業界動向などを解説している。

　Twitter: @dynamitter (https://twitter.com/dynamitter)
　Facebook: http://www.facebook.com/dynamis
　SlideShare: http://www.slideshare.net/dynamis
