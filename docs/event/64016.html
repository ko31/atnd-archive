概要

Spring Boot, the new convention-over-configuration centric framework from the Spring team at Pivotal, marries Spring's flexibility with conventional, common sense defaults to make application development move swiftly and enjoyable. It's ideally suited for building microservices and distributed systems and makes streamlining devops practices simple. Join Spring Boot committer Stéphane Nicoll for a look at what Spring Boot is, why it's turning heads, why you should consider it for your next application (REST, web, batch, big-data, integration, whatever!) and how to get started.

During this session, we will build an application from scratch and show how easily it can be deployed in the cloud. We will also cover some awesome features of Pivotal Web Services (PWS), the public Cloud Foundry offering from Pivotal.

[和訳] Spring Boot (PivotalのSpringチームが開発した「設定より規約」中心の新しいフレームワーク)はアプリケーション開発を迅速で楽しいものにするため、Springの柔軟性と従来の常識的なデフォルト設定を結びつけます。Spring Bootはマイクロサービスや分散システムの開発に理想的に適しており、合理化のためのDevOpsの慣習をシンプルにします。Spring Bootとは何か、なぜ次のアプリケーション(REST, Web, バッチ, ビッグデータ、インテグレーションなど何でも!) のためにSpring Bootを検討すべきか、どうやって始めるかを知るために、是非Spring BootコミッターのStéphane Nicoll のセッションに是非参加してしてください

本セッションでは、スクラッチからアプリケーションを構築し、いかに簡単にクラウドでデプロイできるかを示します。また、Pivotal Web Services (PWS, Pivotalが提供しているパブリックなCloud Foundry)の素晴らしい機能ついても紹介します。
