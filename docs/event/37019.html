温故知新、アセンブリ言語を触ってみる

高性能で手軽な言語が良く使われる中で、今更感がありますが、プログラマの古典として今や実用性はないかもしれませんが、そのさわりの部分を少し紹介して、プログラムがCPUの中で動作していくという様子をイメージしてもらえるきっかけになるような、主に昔話とほんの少しのサンプル実演を行います。私のようなIT業界高齢者には、昔話のきっかけに、若い人たちにはこんな事もあったんだよ。程度のお話としてお聞きいただければ光栄です。（40分程度）

スピーカー: 堀本　正文さん（@mhorimoto）

Arduino で"ディスプレイよりこっち側"をプログラミングする

限りなくユーザーに近い「ディスプレイよりこっち側」をプログラミングするフィジカルコンピューティングをやっちゃいます。実際の事例を踏まえ、入門的な Arduino プラグラミングを紹介します。

事例（予定）

  • 玄関をプログラミングする
  • チェックインをプログラミングする
  • ゆっくりぬいぐるみをプログラミングする

（40分程度）

スピーカー: 渡辺慎二郎（しかじろう）さん（@shikajiro）　Android 専門フリーランスプログラマー、コミュニティ: Physical Computing Lab、日本 Android の会福岡支部

FPGA でおれおれコンピュータつくるよ

FPGA で CPU とかディスプレイプロセッサとか自分好みにつくるよという野望について。（40分程度）

スピーカー: きしだ なおきさん（@kis）　福岡でフリーでプログラマやってます　Javaコミュ＠九州、JJUG　http://d.hatena.ne.jp/nowokay/

Android でテストってどないすんねん（仮）

Android 開発では、ただでさえ面倒なテストがもっと面倒になります。そんな中で費用対効果の高いテストを行うためには、ツールやテスト項目の取捨選択が必要になります。共著で書いた「Google Android プログラミング入門 改訂2版」での内容を元に、上記を考察していきます。（40分程度）
