第1回目の開催！

横浜（あたり）に住むブロガーのためのミーティングイベントです。
みんなで集まり、ブロガソンをしつつ、ブログ（SNS）運営に関する情報を共有するなどのトークをしましょう。

  • すでにブログ（WordPress）を書いている方のみならず、
  • これから書きたいと思っている方でも、
  • ブログをやっていなくてもツイッタラー・インスタグラマーの方まで、

幅広くご参加いただけます。

途中入場・途中退場OKです。

会場施設詳細

横浜市西地区センターに入ると、奥に2階への階段があります。
2階に上がると、小会議室への方向掲示がありますので、それに従って進んでください。
※施設内の掲示板には“プログラミングの会”と掲示されていますが、それがこの会を指しています。

もしも施設の方に呼び止められた場合には、“小会議室の”・“プログラミングの会”と言えば大丈夫です。
会場施設は比較的古いですが、会自体は明るく盛り上げていきますので、ご容赦ください。

会場の小会議室内では飲食が可能です。
ただしゴミは各自で持ち帰ってください。
室内以外のオープンスペースでは飲み物のみの許可となります。
1階には自動販売機、会場施設近くにはコンビニもあります。
