ある国はにwifi ジャマーの使用を合法化する

description

バスや電車の中、携帯電話で大声で話している人がいます。映画館や劇場などで通話している人がいます。携帯電話での会話は、第三者には相手の話す内容が聞き取れないため、会話の内容が中途半端にしか判らないので不快に思うという心理学的なデータがあったと思います。電車に乗っている人同士の会話なら完全に内容も判るから不快にならないんだとか。
みなさんこんな方に遭遇し、不快に思ったことはありませんか？こういうモラルの無い方っています携帯電話のマナーを守らない人が増えています。そんなときに、妨害機を使用することで携帯電話を使えなくすることができます。しかしこれがあったらもう安心です。この携帯電話ジャマーのスイッチを押せば、彼らの悪行を防げまカモフラージュされているし、ポケットサイズなのでバレる心配もありません。

使い方は簡単でスイッチを押すだけです。インストールやアップデートの必要も無いので管理がとても楽です。そこで、パーツを買って、ジャマーを作ろうという大胆な企画を計画しました。
ノコギリ派を出すだけなので、簡単に作れるだろうという安易な発想から生まれたハンズオンイベントです。携帯電話によるいやな騒音を気づかれずにカットします。静かな環境をお約束します。もちろんネットへのアクセスもできなくしますから、イベントなどでも出し物を集中して見てもらえるようになります。

携帯電話ジャマーが働いている場所では、携帯は「圏外」状態になる。多くの人はこの対策に賛成しており、この夏には実際に妨害が実施される見通しだが、世界の携帯電話業界は当然黙ってはいないです。


携帯電話会社や電話機メーカーは、責任ある携帯電話使用の奨励に懸命だ。「映画館では通常、電話会社がスポンサーになって、鑑賞中は携帯電話の電源を切るよう求める広告を流している」とヤナス氏。「携帯電話のイメージ向上は、結果的にこれらの企業の利益につながる。一方、通信妨害装置が悪用される可能性についての懸念もある。悪徳小売業者が装置を不正に使用するかもしれないのだ」

フランス政府は昨年12月にwifi ジャマーの使用を合法化し、現在は電気通信規制局が実際の使用に向けて技術基準の策定にあたっている。通信妨害装置はこの夏までに実用化されそうです。

いいかげんな作りの超強力妨害機 なら携帯電話の周波数以外の周波数にも広範囲に電波をばらまいて他の通信を妨害する恐れもあります。

他人の通信を妨害するということは、他人の家の電話線をちょん切って電話がかけられないようにするとか、他人の家の郵便受けから入っている郵便物を盗んでくるのと同じことです。 
また、仕事で使っているトランシーバーなどの通信を妨害すると業務妨害ということになるでしょう。
