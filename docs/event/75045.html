コマンドについて初めから比較的に優しくやって ワイワイ楽しい勉強会にしたいと思っております。

PCの環境

Windowsの場合は、VirtualBoxもしくはVMwearPlayerにUNIX環境を入れればいいです。

日本語対応の端末と日本語入力はあったほうがいいです。

GNU版のsed、awk、grepなどを使います。Linux系ならまず大丈夫。

GNU系のコマンド

MacはBSD系なので、homebrewで下記をインストール。

$ brew install coreutils

$ brew install gnu-sed

$ brew install gawk

$ brew tap homebrew/dupes

$ brew install homebrew/dupes/grep
