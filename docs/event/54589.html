リーダーのコミュニケーションスキルアップ実践セミナー

イベント ユーザー
event_id 54589
title リーダーのコミュニケーションスキルアップ実践セミナー
catch ～メーカーの技術者・研究者および営業部門・管理部門のリーダークラス対象～
description

◆受講対象・レベル：
　技術者・研究者および管理部門・営業部門のリーダー

◆習得できる知識：
　コミュニケーション力、説明力、プレゼンテーション力、論理構成力、問題解決力が短期間に身につきます。

◆趣旨：
　組織のリーダーとして必要なコミュニケーションスキルを体得します。年上の部下といった、多様なメンバーとの接し方、言いにくいことを伝える技術、信頼感を得る相手の話の聴き方、モチベーションアップの仕掛け等、コミュニケーションを通じて出来る職場によりよい影響力を発揮できるようなスキルを、多くの演習を通じて学びます。

◆プログラム：
１．トップ・リーダーは“ビジョン発信”と“周囲を巻き込む”ことが不可欠である【導入講義】
２．トップ・リーダーに必要なコミュニケーションスキル【講義、演習】
　（１）コミュニケーションスキルとは何か
　（２）多様なメンバーとの効果的な接し方
　（３）言いにくいことを伝える技術
　（４）信頼を得る話の聴き方
３．職場のコミュニケーションに関する事例研究【講義、演習】
　（１）問題とは
　（２）問題解決の手順
　（３）コミュニケーションに関する事例研究
　（４）グループ討議、発表
　（５）講師ワンポイントアドバイス
４．メンバーのやる気を引き出すコミュニケーション 【講義、演習】
　（１）モチベーションアップの仕掛け
　（２）やる気を引き出す３つの方法
　（３）コミュニケーションゲーム
５．メンバーへの効果的な働き方【講義、演習】
　（１）ビジネスコーチングのポイント
　（２）コーチングの事例研究
　（３）ロールプレイング、発表
６．現場実践に向けた行動宣言【演習】
７．総まとめ講義と質疑応答【講話】
