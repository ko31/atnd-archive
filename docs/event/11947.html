毎月開催しているPython mini Hack-a-thonの合宿バージョンを開催したいと思います。
実際の募集は こくちーず の 2月25日 Python mini Hack-a-thon 雪山合宿 で行っています。

開催概要

  • 日時: 2011年2月25日(金)から27日(日)までの2泊3日(途中参加(1泊など)も可能)
  • 会場: マウンテンパパ
  • 参加費: 20,000円(2泊3日4食付き・おやつ付き・暖房費込)(交通費・レジャー費除く)
    1泊の場合はだいたい半額
  • 移動: 現地集合現地解散(参加者ごとに調整を行う予定)
  • 持ち物: 開発用PC(いざというとき用ネットワークあればなお可)
  • 定員: 10人(先着順)
  • 申し込み終了: 2011年01月24日 00時00分まで

峰の原と菅平スキー場にアクセスできます。滑る人は昼に滑って夜開発、そうじゃない人は一日中開発できます。
