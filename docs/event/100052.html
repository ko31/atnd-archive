~一年間AIを学んできたDIVE INTO CODE 卒業生3人によるトーク会（プレゼン）になります。

以下の人を参加対象としています。

  • AIを学ぼうと思って，スクールを探している方
  • 独学でやってきたが，一人での限界を感じている方
  • 弊社スクール関係者(受講生、メンター)
  • スクールの受講を考えている方

年齢制限などはございませんので，お気軽にお越しください．

卒業生にどのようなことを発表していただくか？

  • バックグラウンド（以前の職業）
  • AIエンジニアになるために必要なこと
  • 自分が現在どのような技術を持っているか

そのほか，「本当に未経験者がAIエンジニアになれるのか？」「DICで勉強するというのはどういうことなのか？」などもお話ししていただく予定です．
