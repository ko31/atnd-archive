PHP 『 超 』 初心者勉強会の名刺交換会をしようと思います。
基本は名刺交換会ですが、懇親会も兼ねています。
毎回来て頂いているので、顔は分っても名前が分らない方も多いと思います。
この機会に懇親を深めようと思います。
場所はいつもの大久保の会議室の前にある白木屋を考えています。
20人程度を想定していますが、人数が大幅に増えるようであれば、白木屋以外も考えています。
会費は人数割りで2000～2500円前後と思います。
途中参加、途中早退も可能です。その場合は一律2000円予定です。
白木屋の場合、19時半開始予定ですが、19時から座席を確保しています。
基本的には勉強会に参加したことがある人だけ参加申込をして下さい。
ご質問はphpbeginners@gmail.comまで。
