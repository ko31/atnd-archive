description

7/18、8/11、8/13の３日程あります。

今、大学では「キャリアデザイン」が単位の出る講義として行われています。内容もただの就活対策ではなく、自分自身の内面を見つめ将来のライフキャリアを考える、より人生を深く見つめられるものになってきています。

「キャリアを考える」は大学の就活の時で終わるものではなく、一生続くものです。
今回は、大学で単位認定制のキャリアデザイン講義を担当する講師が、普段の講義の中から２０代社会人向けに内容をセレクト＆アレンジし、皆さんが今後のキャリアを考えるヒントをご提供します。

☆こんな方にオススメ！☆
・今後のキャリアに迷いがある
・今後に迷いはないが、自分を見つめ直すことは大切だと思っている
・今の大学でどんなキャリア教育が行われているか興味がある

３回に分けて開催します。どれか１回のみの参加も可能です。

第１回：７月１８日（月・祝）
「発想力を鍛えて今後のキャリアをデザインする」

第２回：８月１１日（木・祝）
「２０代、今だからこそやりたい　この自己分析」

第３回：８月１３日（土）
「チャンスを活かす、偶然を活かす」

いずれも時間は１３：３０～１６：３０
WithYouさいたま　セミナー室３（さいたま新都心駅徒歩３分）
さいたま市中央区新都心２－２
参加費：１回１５００円、全ての回参加の方は最終日に５００円キャッシュバック

この夏、改めて自分を見つめ、今後について考えてワクワクしませんか？
