Tジョイ京都
「映画プリキュアオールスターズNew Stage〜みらいのともだち」世界最速上映記念
「第６回関西プリカラオフ」開催けってーい！！

０時００分からプリキュア映画が公開されるという「マジキチイベント」
もちろん終電などあるわけなく、どうしようもないとお嘆きの方！

朝までカラオケ歌ったり、映画のネタバレ全開トークしませんか？

さらに今回は「サウンドジョイ木屋町店様」のご好意により仮眠用としてカラオケルームを２部屋（男女別）別に用意していただけました！
「ウチ、明日仕事やからちょっとでも寝やなあかんねん・・・」
という方はそちらで４時間、しっかり眠れます！

【概要】
参加費：一般１７８０円、学生１４８０円（学生証の提示必須）✩ドリンクバー付
集合時間および場所：１時３０分「Tジョイ京都ロビー」
エントリー、キャンセルリミット：３月１６日２３時３０分
（当日、空きがあり、かつ時間内でしたら映画館内でのエントリーも可能です）

集合次第、タクシーにてカラオケ会場へ移動します
京都駅前→四条河原町まで一人あたり５００円程度です

【注意事項】
映画イベントと同じく、開始時間が23時00分を超えますので「18歳未満の方」の参加はご遠慮ください
✩今回は「映画公開記念」ですので、曲は「プリキュア縛り」です
他のジャンルの曲は次の機会で☆彡

是非、手にとってくださいね！

大変長らくおまたせしました！
