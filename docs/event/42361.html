description

ビッグデータ俯瞰分析の簡単な説明（概要や変遷など）を行い、ビッグデータ俯瞰分析技術
の事例をいくつか紹介した後で、下記(下記はあくまで予定です)に示すようなSNSビッグデータ俯瞰分析技術を
演習していただき、受講者にビッグデータ俯瞰分析技（一部）等を体得いただく。

  1. Twitter APIを用いたツイート情報及び位置情報の取得
  2. 形態素解析及び係り受け解析を応用したツイート情報分析
  3. 引用分析及びクラスタリングを応用したツイート情報及び位置情報の分析
  4. 力指向アルゴリズムを用いた可視化

本演習には、PC（Windws7以降やmac OS X）と
以下（RとPython）の環境が必要ですので、事前に準備しておいてください。
なお、この事前準備が出来ていなくても、演習はできませんが
受講すること自体は可能です。
