title 本の交換会「ブクブク交換」
catch ～武蔵小山で温泉に入って本を交換しよう♪～
description

日本全国２５都市、フィレンツェ、ロサンゼルス、上海で開催されている本の交換会「ブクブク交換」が武蔵小山の清水湯で開催決定！
テーマに合った本を持ち寄って交換しよう！
当日は、武蔵小山に本社をかまえるわるならハイサワー♪でおなじみの博水社の田中秀子社長と清水湯すぐ近くのブックスタマの加藤社長をゲストに迎えて、最高の温泉と本とハイサワー♪を楽しみます。
本を交換して楽しんだ後は、温泉に入り、ブックスタマへ行く予定です。

５月１１日（土・昼）
１２時〜１４時まで
場所：清水湯（１２時に２階休憩所に本を持って集合して下さい。）
http://www.shimizuyu.com/
ゲスト：田中秀子さん（博水社３代目社長）　加藤勤さん（ブックスタマ社長）

【持ち寄って交換する本のテーマ】
『酒／酒場』『街』『春』
※このほかテーマ以外の本でももちろんウエルカム！
１冊から３冊で交換できる本をお持ち下さい。
