description

こんにちは、 @mainamiki こと並木麻衣と申します。
ドロップアウトした若者支援団体「NPOキズキ」をやってるイケメンな友人 @yasuda_yusuke が
「合コンしたい！」とのたまうので、
せっかく若い元気な男女が集まるなら…（そして私貧乏だし…ｗ）と思って今回はこんな企画をしました。

「異国料理×青空プレゼン合コン」。

要は、ちょっと珍しい料理を持ち寄ったり皆で作ったりして和気あいあいと食べながら、
自分のやってる面白いプロジェクトや活動について青空のもとでプレゼンしてみようぜ！
ついでにかっこいい／可愛い子ゲットしたいな♥という何とも欲張りすぎじゃね、的交流会なのであります。
既婚・未婚問いませんが、狩場であることは一応ご了承ください。笑
あ、入退場自由です。飛び入りも歓迎です。

こんなひとに来てほしいな♥

  • 「皆のやってることに興味あるなぁ」
  • 「めずらしいもの食べたいな～」
  • 「未知なもの大好き！」
  • 「むしろ誰かの熱い活動に巻き込まれたいぞ！」
  • 「自分の活動に人を巻き込みたい！」

※もしもこのイベントで収益が発生した場合は、主催者の所属団体もしくはプロジェクト（6/5現在では「NPOキズキ」「スーダン障害者教育支援の会」）に均等に寄付させていただきます。なお、主催者様をあと2人ほど募集中です。ご興味のある方は並木 @mainamiki までご連絡ください。具体的なコミットなど、ご相談させていただきます♪

プレゼンについて

NGO、マイプロジェクト、学生活動、旅の報告などなど、とにかく「活動」を伴うものなら何でもござれ！
紙媒体からデジタルツールまでお好きに駆使して、
一緒にピクニックしてる皆さんのハートをがっちり掴んでください。いろんな意味で。
一人15分まででお願いします。ワークショップ風もありよ。駆けっこもできる公園が会場ですから。

プレゼン開始時間ですが、14:00／15:00／16:00／17:00の4回スタート目安で。
タイムテーブルがもし決まったら、こちらで公開しておきます。

注意書き

雨天の場合は、場所を個人宅に移します（中目黒駅から徒歩10分程度）。
天気が怪しい時は当日9:00までに本ページで移転の旨を告知します。チェキラ。
場所は目印を木の幹か何かに貼りますが、@mainamikiにメンションいただけると反応早いです。
中目黒の場合は、案内係が札を持って改札の前に立ちます。
