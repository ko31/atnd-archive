第３７回勉強会のお申込みページです。本ページのＡＴＮＤを利用しますので、慣れない方もおられるかと思いますが、何卒ご協力をお願いします。どうしても登録出来ない方は、管理者柴田までメールでご連絡ください。

勉強会に関する内容については、これまで通り、下記のオープンＣＡＥ初心者勉強会のＷｉｋｉページにまとめます。今回は、質問会：実費1000円＋勉強会：実費300円＋懇親会：実費＊円と３部構成なので、アンケートの方もよろしくお願いします。

午前の質問会では、参加者が持参したオープンＣＡＥ環境において各自の質問を皆で情報交換しながら、解決のヒントを得るための機会です。流体関連は講師確認中・構造関連は柴田がご支援します。

なお質問会に参加される方は、このATNDのコメントにおいて「目的とする分野（流体・構造）と質問したい内容」を必ず記入してください。

詳細は 第３７回勉強会：H270221 をご覧ください。

第３７回では、午前：質問会・午後：勉強会の２部構成ですが、変則的に以下の構成で準備を進めています。

午前９時から１２時：OpenFOAM質問会＋Salome-Meca質問会＋FrontISTR構築講座　を平行して臨機応変に
午後１時から２時半：「FrontISTR講習会」（演習参加は任意ですが全員が受講）
午後２時半から３時：「使えるスパコン産業利用」（FrontISTRの応用発展の話題）
午後３時から５時：　通常の勉強会（構造流体の各種の事例報告や質疑応答）
