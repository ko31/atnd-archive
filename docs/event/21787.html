講談社刊行の月刊誌「クーリエ・ジャポン」をネタに，飲み物片手に気軽に気楽に話してみましょう．

「朝食会」が多い中，逆張り（？）で「夕会」を実施してみたいと思います．「夕会」の開催は多分，きっと，少なくとも名古屋では初だと思います．おそらく．
仕事などが終わって，少し早い時間から集まって，日常とはちょこっと離れた話をしてみませんか？

開催日：2011年11月9日（水）
時間：午後6時48分～8時くらい
場所： cafe＆bar PERMS （地下鉄・久屋大通駅近く）
　　　　　　〒461-0001 愛知県名古屋市東区泉1丁目23-36 NBN泉ビル 1F
定員：10名（くらいかな？）
参加費：不要（但し，カフェ・バーですので，1オーダーお願いします．）

2つだけルールをお願いします．

  1. 酔っ払って正体を無くさないこと．ノンアルコール飲料もございます．
    UKのパブでビール片手に談笑，ってイメージでお願いします．コーヒー片手も勿論アリ．
  2. ちょこっと遅れても参加したい，という方はその旨，アンケート回答ないしコメントお願いします．

以下は大歓迎です．

  • 事前に読んでおいて，関連ネタを持ってくる
  • 自分だったらこうかなぁ，なんて意見を持ってくる

それでは当日，お会いできることを楽しみにしております．
