【講演主旨】
　近年、画像認識(コンピュータ・ビジョン)への期待は益々高まっている。セキュリティ分野での監視カメラ、生産分野での各種外観検査、医療分野での組織検査や各種画像認識、スポーツ分野での動作解析や評価、インターネットでの画像検索、さらにはロボットの視覚など、多岐の分野にわたっている。これらのニーズに呼応して画像認識の研究が盛んに行われているが、個別のアドホックな手法や専用の高価なシステムとなっていて、未だ自動化や実用化は難しく、実際の現場では依然として人の能力に頼っているのが現状である。そのため、PCベースでの簡便で安価な、しかも高速で汎用性の高い柔軟な視覚システムの実現と普及が強く望まれている。講師らは、これまで高次局所自己相関(HLAC)特徴と多変量解析手法を合わせた並列学習型の汎用認識方式を提案し、画像や動画像等の種々の認識課題に応用してきた。
　本セミナーでは、その概説とともに、基礎としてのパターン認識の基本的な枠組、画像認識への一般的なアプローチ、具体的な応用事例について、わかりやすく解説する。

1　画像認識のニーズとシーズ
　1.1　ニーズ:応用分野の概観
　　1セキュリティー:監視カメラ
　　2検査の自動化:外観検査、医用画像認識
　　3画像検索とアノテーション
　1.2　シーズ:現行手法の概観
　　1逐次手順型 vs 並列学習型
　　2モデルベース vs 非モデルベース

2　画像認識の基礎
　2.1　画像処理・認識の一般的枠組
　　1表現方式(関数、ベクトル、記号)
　　2処理方式(直接手順、前向適応、後向適応)
　2.2　基礎としてのパターン認識
　　1一般的な枠組
　　2特徴抽出(幾何学的側面と統計的側面)
　　3類別・識別・判別(各種手法)
　2.3　基礎としての多変量解析手法
　　1多変量データの種類(量的と質的、n-way data)
　　2線形代数の簡単な復習
　　3確率と統計の初歩的な基礎
　　4各種手法(PCA, MRA, ARA, DA, CCA, 数量化手法Ⅰ～Ⅳ)
　2.4　高次局所自己相関を用いた学習型汎用画像認識方式
　　1画像特徴としての高次局所自己相関(HLAC)　
　　2HLAC特徴の拡張としての立体高次局所自己相関(CHLAC)
　　3学習型汎用画像認識方式(ARGUS)
　　4通常からの逸脱としての異常検出

3　画像認識の具体的応用事例
　3.1　多変量解析手法の適用
　　1線の当てはめ(PCA)
　　2画像圧縮(SVD)
　　3自動閾値選定(DA)
　　4学習型画像処理(MRA)
　　5形(輪郭)の認識(複素ARA)
　3.2　HLAC特徴を用いた画像認識
　　1複数対象の同時認識・計数
　　2図形の位相的特徴の認識
　　3顔認識と表情認識
　　4異常検出の応用:癌細胞検出、外観検査
　　5画像検索とアノテーション
　3.3　CHLAC特徴を用いた動画像認識
　　1ジェスチャ・動作認識
　　2移動体追跡
　　3gait(歩様)認識
　　4異常検出の応用(監視カメラ)
　　5その他、時系列データ解析・認識への応用
