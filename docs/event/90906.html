description 昨年インターネット広告媒体費が初めて1兆円を超え、広告媒体としてのインターネットの役割が年々増しております。一方で、Webサイトの集客においては自然検索経由での流入割合が高く、SEO施策をほどこすことで、安定的に流入を獲得できるサイト構成にすることが必要不可欠と言えます。

そんな中、Google検索結果の順位付けの基準となる評価軸は年々広範化・複雑化しており、SEOで成果を上げ続けるためにはより高度なノウハウと、日々起こる変化に追従し続けることが求められております。そのためにはGoogleの考え方を把握し、それに沿ってWebサイトに施策を行わなくてはなりません。

本セミナーでは、SEOにおいて把握しておくべき基礎的な知識や最近のトレンド情報、サイト構成において意識すべきポイントなどを分かりやすく解説いたします。

■このような方におすすめです

・Webサイトの強化/集客に悩まれているWeb担当者様

・新規顧客を開拓し営業リストを増やしたい営業担当の方

・自社の情報をより広めたいとお考えの広報の方

・クライアント様等の集客を強化したいWeb制作会社の方

・新しい販路を開拓したいとお考えの経営者の方

・顕在/潜在顧客への適切なアプローチ方法

・コンテンツ作成時の考え方や実践方法

・SEOの適切な運用方法
