クラウド電話API Twilioのハンズオンイベント
Twilio API 勉強会 Vol.6を開催致します。

■ Twilio API 勉強会？

クラウド電話 API Twilioに興味を持っているエンジニアの皆さんが集まり毎回ハンズオン形式で開催しています。
初心者の方はもちろん既にTwilioを使っている方々、Twilioをテーマで情報交換ができればと思います。
いつも変わるデータで皆さんとTwilioの機能を試しています。
基本テーマ以外でもいろんなTwilioの機能を試しながら参加の皆さんとTwilioで楽しむ勉強会です。

■ 今回の勉強会では？

今回もクラウド電話API Twilioで遊びましょう。
電話はもちろん2013年6月27日、日本での提供を開始したTwilioのSMS機能の紹介とハンズオンの時間ではみなさんと電話の発・受信とSMS送信を試してみたいと思います。

■ LTタイム

イベントのハンズオンで皆さんが作った物を発表する時間です。
どなたでもどんな内容でも構いませんので、気軽にみなさんのコードを紹介してください！
事前申請もオッケーですし当日申請もオッケーです。

■ 参加対象

Twilioに興味を持っているエンジニアのみなさん（言語、分野は問いません。）
