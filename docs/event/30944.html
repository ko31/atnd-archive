内容

Arduino相当の基板を自分ではんだづけして作成します。
完成したら簡単なプログラムを書き込んでみます。
はんだづけがはじめての方でもWelcome!
玄人の方も自分に合わせたArduinoが作れる魅力たっぷりのワークショップです。
作成したArduino互換基板は次回以降のワークショップ参加時に使用できます。

実習機材紹介

Liblo AVR …Arduino互換の組み立てキットです。安価で初心者にも組み立てができるように
なっています。また慣れたら、基板だけを購入して自分で必要なところだけ部品を追加するこ
とで更に安く組み立てができるようにもなります。
以下のうちひとつをご購入ください。当日購入可能です。

  • Liblo AVR フルキット ￥3000 (スタンダード＋ACアダプタ、書き込みアダプタ)
  • Liblo AVR スタンダード ￥1500(本体基板＋組み立てに必要なパーツ)
  • Libro AVR 基板のみ ￥250 (自分で必要なパーツを別にご用意ください)

工具類は会場側で用意しますが持ち込みもOKです。また、ACアダプタや書き込みアダプタに
ついても希望者へ貸出致します。
Arduinoについての相談などもご遠慮なくどうぞ！

河野先生のわくわくワークショップ紹介

Arduinoワークショップ三部作

第一部（7月)は自分でArduino互換機を組み立て、環境構築
はんだづけはじめての方から、Arduinoマスターして量産したい人まで幅広い方がターゲットです。

第二部(8月)はセンサーと自動制御
温度と扇風機、照度と照明、騒音とボリュームなど、便利な制御小ネタを通してフィジカルコンピューティング。

第三部(9月)はネットワークとウェアラブル
電池駆動と無線通信を試します。
