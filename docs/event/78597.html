description

”偏愛マップで新しい出会い”とは、よくある合コンや街コンとは全く違います！

自分の好きなこと、そして相手の好きなことや興味があることを『偏愛マップ』を見ながら話題にできることで、お互いの共通点や関心があることを簡単に見つけ出すことができる、内面重視のパーティーです。

－初めに偏愛マップの作成をします。

あなたが好きなことや興味があること、今までやってきたことやこれからやりたいこと、テレビや本や映画、旅行した場所、ペットや音楽、好きな人物・・・何でも１枚の紙に書き出していきます。

例えば、旅行が好き！温泉が好きで伊豆や草津がお気に入り♪など。

作成した偏愛マップをお互いに見せ合い、相手のマップから興味を引かれたこと好きなことを中心に話すことができるので、共通点や気になる所などいくらでも話のタネが見つかります。

あなたを知ってもらい、そして相手に興味を持つことで、短い時間の中でもより深く共感し合える相手を見つけることができるでしょう。

なかなか出会う機会がない人、話が合う人に出会いたい、初対面で話すことが苦手、出会い系のパーティーに行ったことがあるが楽しくなかった・・・
出会い系で知り合った人はいるけど何か違った・・・

『偏愛マップ』があれば、どんな人なのか探ることもなくすぐに打ち解けられますので、気になったあなたはぜひ参加してみてください。

内容

当日の進行

１．主催者あいさつ

２．偏愛マップの説明（全く知らなくてもここで説明しますので大丈夫です！）

３．偏愛マップを作りましょう！（３０分位で作成します）

４．偏愛マップを使っての自己紹介

５．休憩（マップの見直し）

６．フリートーク

７．お相手選び（もっと話したい人を３人紙に書き出します）

８．回収・発表

９．感想・主催者あいさつ

受講対象

・アラサー・アラフォーで出逢いが欲しい方
・内面から理解してほしい人
・共通の趣味を探している人
・自分のことをうまく話せない人
・今までパーティーで出会えなかった人
・面白そうだと思った人

プレイベントですので気楽に参加してみてください。

期待できる効果

・気の合うパートナーと出会えます。
・自分の好きなことなので自信をもって話すことができます
・また相手の好きなことを話題にできるので自然と会話が弾みます。
・楽しめるパーティーになります！
