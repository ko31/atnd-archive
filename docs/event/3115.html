本イベントでは、会場内の写真・動画の撮影を原則として許可し、それらのWeb上での公開を推奨します(Ustream中継も検討中です）。撮影されることを希望しない方を当日イベント開始時に確認しますが、途中参加の場合などにチェック漏れが起こりえますので、事前に意思表示可能な場合には、この掲示板か「#nsmc」をつけたTweetでお知らせください。

本イベントは、Poken Meetupとして登録されましたので、Poken用の交換電池10個がアコニ様より提供されます。Pokenの電池が切れた方は当日無料で交換することができます。

Niigata Social Media Club / 新潟ソーシャルメディアクラブは、新しいウェブ事情に敏感な人々をつなげる、新潟のプラットフォームを目指します。新潟のウェブに関心のある人が気軽に集まり、楽しめる場所になってくれたらと思います。第一回イベントは、2010/1/29に開催しました。
