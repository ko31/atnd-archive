description NPO法人かものはしプロジェクト学生部ゆるかもです！

アンコールワットで有名な、東南アジアの国、カンボジア。
皆さんはこの国に、どんなイメージを持つでしょうか？

ゆるかもが初めて知ったカンボジアは、
貧困や、人身売買といった、
深刻な問題を抱える国でした。

どんな国なのか自分の目で確かめたい。
昨年夏、スタディツアーを企画、実際に現地を訪れました。

そのとき、
現地に行って”自分の目で見たカンボジア”は、
”それまで思い描いていたカンボジア”と、全く違っていました。

あたたかくて、また行きたくなる、
素敵な国。

今回の訪問に際し、 私たちも何かできることしよう、と思い、

カンボジアで最貧困の女性たちのために、パーティをプレゼントしてきました。

そこで、出会った人々、湧き起こった感情は、
何にも代えられない経験となりました。

自分で行くと決めたからこそ、 知ることができたカンボジアの一面がありました。 自分たちで考え抜いてやり遂げるからこそ、 感じることができた達成感がありました。

ゆるかもスタディツアーから見えた
カンボジアの魅力、たくさんの人に伝えたい！
そんな想いで企画したイベントです。

ゆるかもに興味がある人、
カンボジアに行ってみたい人、
海外やバックパッカーに興味がある人、
大学で何かに打ち込んでみたい高校生も大歓迎！

当日は、現地で民芸品として作られている小物も製作します★

お楽しみに！！
