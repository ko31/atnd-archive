サラリーマンＶＳ起業家　この先の未来、どちらを選択すべきか？徹底検証会

日程：１０月９日（日）１３：３０～１６：３０
場所：東京都内セミナー会場（参加者に別途お知らせ）
参加費：３０００円

サラリーマンとして企業で一生働く人生と
起業して好きなことを仕事にして働く人生

はたして、5年先、10年先を考えた場合
どちらを選択すべきなのでしょうか？

サラリーマンも今や「安定した職業」とは言えません。
終身雇用制が消滅し、『能力主義』となり、成果を
出し続けなければ、生き残れない時代となりました。

だからといってサラリーマンを辞めて起業しても、
収入は不安定だし、生活できるか心配。

たとえ起業で成功しても、一時的なモノが多く
一つのビジネスモデルが長期的にかつ安定的に
稼ぎ続けることは非常に困難な時代となりました。

このように、サラリーマンも起業家も
能力やスキルがなければ、生き残れない時代なのです！
安定した時代は終焉を迎えようとしております。

サラリーマンで働くのも、起業して働くのも、
両者とも難しくなると、どちらを選択すればいいのか？

そこで、多くの社会人が悩んでいる
サラリーマンとして働く人生と
起業家として働く人生の
メリット、デメリットを洗い出し

・どちらが自分にとって向いているのか？
・どちらのが自分の将来にとって有利なのか？

転職10回以上、会社倒産３回、起業４回を経験した私と、
某一流企業の元ヘットハンディング、現在はサラリーマン向けに
キャリアアップコンサルタントで活躍中の清水コーチが、
これまでの経験で「成功した事例」や「失敗した事例」を
お話し、皆様と一緒に徹底検証会を行います。

 

 

【プログラムの内容】

・10年後の未来を予測。サラリーマンと起業家の優位性は？

・サラリーマンで働くメリット・デメリット

・起業するメリット・デメリット

・やりたいこと、好きなことで起業して本当に稼げるようになるの？

・副業ってホントに稼げる？副業で稼げる人、稼げない人の違い

・起業に最低限必要な能力やスキルとは

・どのような人が会社で生き残り、昇り続けることができるのか？

・サラリーマンとして生き残りために最低限必要なスキルとは

 

 

【こんな方にオススメ】


・起業・副業を考えている方

・転職を考えている方

・今の収入に不安を感じている方

・収入アップ・キャリアアップを目指している方

・すでに起業、副業しているけど、稼げていない方

・自分の能力やスキルで、どのお仕事に就けばいいのか？悩んでいるいる方
