DAM★ともLIVE On stage! Vol.11　”W-mode”

title DAM★ともLIVE On stage! Vol.11　”W-mode”
catch 出演者＆観客参加募集中です！
description

本格的な音響＆照明があるステージで、ライブ感覚で気軽に歌える！聴ける！カラオケライブイベント！

本イベントは、今まで「人前やステージで歌ったことがない」という初心者の方からライブ活動をされている方まで、誰でも気軽に本格的なステージで歌ったり、他の参加者の方の歌を聴いて楽しむことができるライブ形式のカラオケイベントです。

今回は、「Ｗ＝笑い」、つまり思わず笑ってしまうようなネタ曲限定での開催となります。元々ネタ曲として出ている曲はもちろん、パフォーマンスや衣装との組み合わせなどやり方は自由です。目的はひとつ、「お客さんを笑顔にすること」。
いろいろと世知辛い世の中ですが、こんなライブがあってもいいんじゃないかと思って企画しました。

イベント概要

  • 本イベントは、主に DAM★とも ユーザー向けに、実際にライブ形式で歌を歌ったり聴いたりする場を提供するカラオケライブイベントです。
  • 出演者はプロのアーティストのように、本格的な音響や照明のあるステージで歌うことができます。
  • 歌い手としての 『出演参加』 と、観客としての 『観客参加』 の２つの参加方法があります。（観客参加はDAM★ともをやってなくても問題ありません）
  • 歌い手として『出演参加』する場合は、最初に自己紹介を兼ねて全員１曲ずつ、その後続けて２～４曲と合計で４曲歌うことができます（1組あたり合計で２５分が目安）。
  • 選曲や順番だけでなく、ＭＣや振り付け・衣装も含めて自分なりのステージを考えて、楽しんでください。
  • 音源は基本的にプレミアＤＡＭのカラオケ音源を使用します。歌詞もモニターに表示されます。
  • まったくの初心者という方でも、リハーサル含めてしっかりとスタッフがフォローしますので安心してご参加ください。

参加方法

  • 『出演参加』を希望する場合は、第一興商の無料カラオケ録音・公開サービス「 DAM★とも 」で、自分の歌った曲を公開していることが条件となります。
  • また、連絡をスムーズに行うため、ATNDでのメールアドレス登録をお願いします。
  • 複数名で参加される場合は、参加申し込みは代表の方1名でかまいません。また出演者同士のデュエットやコラボも推奨です。

１.出演参加を希望する方は、以下の注意事項を読んだ上で、参加ボタンを押してください。
　→先着順に「仮参加」扱いとして一旦枠をおさえます。
２.主催者から折り返し確認メールをお送りしますので、必要事項を3日以内にご返信ください。
　→返信確認後に、正式に参加扱いとなります。3日以内に返信がなかった場合は、一旦「仮参加」を取り消しますのでご了承ください。
３．決められた期日までに当日のセットリスト４曲を決めて、主催者に連絡してください。
４．あとは当日参加費を払って、9時からのリハーサルに参加し、本番を迎えるだけです！
５．終了後、ステージ映像を動画にて出演者のみ公開します。

  • 『観客参加』を希望される場合は、特に事前の申込は必要ありません。当日直接会場にお越しいただき、参加費を払ってご参加ください。

  • 『観客参加』→1500円　1ドリンク（500円）込み

注意事項

  • 今回は「笑えるネタ曲」限定です。選曲のジャンルは特に限定しません。ただし設備の関係上、マイクは５本までが限界です。（ワイヤレス２本、有線３本）
  • 正式参加決定後のキャンセルや当日の遅刻については、基本的に無しでお願いします（キャンセル費用をいただくことはありませんが、今後参加をご遠慮いただく場合があります）。
  • イベントの安全かつスムーズな進行のため、主催者および会場スタッフの指示にはご協力いただけますようお願いします。
  • 本イベントは、『ライブ形式』です。選曲やトーク、衣装や振付などその場に観客がいるということを意識して、自分ならではのパフォーマンスで通常のカラオケオフとは違う楽しさを体験してください。
  • 参加費は代表1名が「出演参加費」、その他の方は「観客参加費」となります。
  • 出演者用の控室および着替え用スペースがあります。
  • 歌う順番は、出演者からいただいたセットリスト４曲を元に、構成やジャンルを考慮して主催者にて決定します。基本的に最初に全員が1曲ずつ歌い、その後２～４曲目を連続して歌っていただく流れは変わりません。
  • 公演中の来場者によるビデオやカメラによる撮影は、出演者のプライバシー保護のため禁止とさせていただきます。もし撮影している方を見かけた場合は、スタッフよりお声掛けをさせていただき、指示に従っていただけない場合はご退場いただくこともあります。（終了後などの撮影については上記の限りではないですが、ご本人の許可を得たうえでお願いします）
  • 本公演はUSTREAMによる配信を行います。出演者のプライバシー保護のため、はっきりと顔までは映らないようにしていますが、実際の過去の配信動画を参考にしてください。
