第1回、2回とやってきたソーシャルアプリ勉強会、第3回を企画する余裕がない！すでに3か月近く経過してしまった！

ということで、勉強会のかわりに飲み会どうでしょう？という企画ですｗ

サイテックでも、この3カ月でmixiアプリをリリースし、運営中なので、こういう席でしかできない話もできるかと思います。

◇ソーシャルアプリを作った人

  • 作ったけどユーザーが伸びない！
  • どうやってマネタイズしよう！？
  • アクセスが増えすぎて処理しきれない…
  • ひとりじゃ限界だから誰かヘルプ！

なんて悩みを、飲みながらうだうだ語りませんか？

◇ソーシャルアプリを作ってみたい人

  • 作りたいけど何を作ろうか？悩み中
  • どうやって作ったらいいのか分からない
  • そもそもどのプラットフォームがいいんだろう？
  • AmazonEC2ってどうよ？
  • 一緒につくる仲間がほしい

なんて疑問を、多分解決できるかも。
