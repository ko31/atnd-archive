■内容

最近人気のテキストエディタ「 Sublime Text 」の勉強会を開催します。

sublime text2に興味ある、触ってみようかなと思っている人向けの初心者向けの内容です。また、時間があるようならsublime text2をいろいろ触って自分好みのカスタマイズをする時間もとりたいと思います。

PCは無くても大丈夫ですが、あったほうがその場で色々触ってみれるので、持参いただくことをオススメします。また、基本macでの操作説明になります。windowsでもそんなに変わらないと思いますが、ご了承下さい。
