description

「海なし群馬に沖縄の風を！」を合言葉に、沖縄の音楽・文化・食を通じて多くの方に沖縄を知って笑顔になっていただけたらと思います。

飲食ブース出店・物販ブース出店・地域住民参加ライブ・プロの三線アーティストの三線ライブ・エイサー・その他パフォーマンスあり！！
観覧無料！！
飲食、物販ブースにおきましては有料（通常販売）となります。

また、今回は認定NPO法人メッシュサポートさんへ、飲食・物販の売り上げを寄付させていただくチャリティーイベントとなっております。

メッシュサポートさんや沖縄タイムスさんからの資料提供いただいたものはメッシュサポート特設ブースにて展示させていただきます。

沖縄の素晴らしさだけでなく抱える問題、今回は医療格差や医師不足などを多くの方に知っていただきたいと願っています。 群馬だけでなく沖縄にも笑顔になっていただけるお祭りにしていきます。 沢山の人を巻き込み大きなイベントに成長するようにと願いを込めて『琉球の嵐』というタイトルにしました。

8月28日は笑顔の嵐が吹き荒れるよう出演者、スタッフ一同心よりお待ちしております！
