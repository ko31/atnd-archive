description

参加費無料でFacebook、Twitter、アメブロ、youtubeを使って初心者の方でも簡単に収入アップさせるセミナーを開催します。参加される方には、過去に講師が販売していた15万円相当の豪華特典（テキスト・自動集客ツール・解説動画）をプレゼントいたします。

SNSやネットの知識ゼロの方でも基本から解説しますのでご安心くださいませ。

将来が約束されないご時世に、自分で独立してお店を出したり、ネットでサイドビジネスをしたり、お勤めをしながら週末起業・副業に挑戦する方たちがかなりの勢いで増えてきています。

しかし、無料ツールであるSNSを使ってサイドビジネス、
副業が出来るならチャレンジしたいが使い方がわからない方、アフィリエイトなどの商材を買って見たものの売れない状況が続いているetcという方もたくさん見受けられます。

そんな方に対して、本セミナーでは、SNSを使って年間5000人集客を起こした講師より、

・インターネットでビジネスを起こせない人の特徴と対処方法
・Facebook、Twitter、アメブロの特徴の説明と収入を得るための使い方について
・Facebook、Twitter、アメブロで自動集金出来る仕組み作り
・youtubeに動画をアップさせる際の簡単なテクニック集

などなど、ポイントを抑えて無料でセミナー開催をさせて頂きます。

冒頭に記載の通り、参加された皆様全員に過去にセミナー講師が販売していた、合計15万円相当のFacebook、Twitter、アメブロの集客テキスト（全200ページ）、Twitter自動集客ツール、youtubeにアップした動画にアクセスを集めるテクニックをまとめた動画6本を無料で差し上げます！

SNSを使って収入を自動で得る仕組みを理解頂いた後に、参加希望者様のみに2016年にSNSで大きく拡散していくプロダクトのご紹介もさせて頂きます。

■以下一つでも当てはまる方はご参加ください！
・起業したいが方法論がなく、リスク少なく実践できる方法を探している
・現在の会社、お仕事内容、人間関係、お給料に漠然と不満があり今のままでは嫌だと思っている
・夢や目標などがあるが、どう踏み出していいかわからない
・アフィリエイトで販売する様に商材を買ってみたが思ったより売れなかった
・欲しいものなど物欲などがあるが今とこれからの収入だけでは賄えない
・Twitter、アメブロやFacebookの効率のいい使い方を知りたい
・インターネットが苦手な自分でもインターネットで収入を得れるようになりたい
・youtubeに動画を掲載しているが、全くアクセスが起きない
・セミナーや本でいろいろ勉強してみたが、結局実践できていなかったり、失敗に終わった
・資格などを取ったけど仕事が少ない
・ブログを書いているけど、アクセスが少なく収入が得れない
・高いお金をかけてブログをカスタマイズしたのに、なんの変化もない
・Facebook、Twitter、アメブロの自分に適した使い方を知りたい
・youtubeに動画などを今後掲載してビジネスをしてみたい
