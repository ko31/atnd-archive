description

お休みの日に16時から21時まで、まったり飲み食いしながらアナログゲーム（ボードゲームやカードゲーム）で遊ぶ会です。
初心者の方大歓迎です。私も初心者です。初めて参加したゲーム会もドキドキしながら一人で行きましたが一回でボードゲームにハマってしまいました。その楽しさを知ってもらえたらなーと思います。というより単純に面白いですよ！

会場
「セルフキッチン」
東京都中野区東中野4-9-1　第一元太ビルB1F
JR総武線　東中野駅東口２　徒歩１分／地下鉄大江戸線　東中野駅A1　徒歩２分／地下鉄東西線　落合駅　徒歩１５分

「清貧」という名前で有名だったお店がリニューアルオープンしたところです。

参加費用
とくにありません。タイムチャージのないCafe Styleという形式です。
なので、ドリンクやフード（軽食）を各自でオーダーしていただければと思います。

主催者で用意するゲーム （ゲームの持ち込み大募集！！）
「パンデミック」「ニムト」「藪の中」「エセ芸術家ニューヨークへ行く」「HANABI」「レジスタンス」「ワードバスケット」「お邪魔者」「ヒットマンガ」あたりは持ち込む予定です。
これ以外持っていないので、他のゲームを持ち込んでくれる方超絶大募集！
