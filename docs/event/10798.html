description

日本は現在、非常に多くの政治経済・社会問題を抱えています。「政治経済・社会問題をみんなで議論する会」では毎回様々なテーマを取り上げ、日本社会の今後について参加者同士が真剣かつ真面目に建設的議論を行い、各自の考えを一層深めることを目指します。

■今回のテーマ■　「自治体の首長の育児休業取得の是非」――　あなたは賛成？　反対？

■日時■　2010年12月9日（木曜日）午後7時～午後9時

■場所■　銀座ルノアール　ニュー銀座店　マイ・スペース　http://www.ginza-renoir.co.jp/renoir/040.htm

■人数■　６名程度

■費用■　（会議室代2200円÷参加人数）＋ドリンク代

■参加資格・目的■
政治経済・社会問題に強い関心があり、議論を通じで自分の考えをより深めたい方（当会は、特定の宗教・政党・団体などとは関係ありません）

■申込方法・締切■
▼①氏名、及びフリガナ　②所属　③年齢（何歳代かのみ）④連絡先メールアドレス　を記載の上、下記メール宛てにお送りください。
omusubiyama7【アットマーク】gmail.com　 （送信時は【アットマーク】を@に変えてください。）
▼申込みは先着順（活発な議論を行うため、少人数定員制）
▼締切は開催日の前日（ただし、それまでに定員に達した場合は、その時点で締切）

■流れ■
①簡単な自己紹介
②担当者が参考資料・関連書籍を簡単に要約し、テーマの論点をいくつか提示（約30分）
②参加者が論点について本気で議論（約1時間30分）

■注意点■
▼相手を論破することが主たる目的ではありません。
▼意見のぶつかり合いは大歓迎。ただし、罵声・暴言・誹謗中傷など礼節のない言動、暴力、宗教活動、営業宣伝活動・ネットワークビジネスなどは禁止。
▼参加前に関連書籍を読むなど、議論を掘り下げて進められるようにして参加すること。
