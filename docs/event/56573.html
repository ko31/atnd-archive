四ツ谷のシェアカフェ・コワーキングスペースのFrohでは、アットホームな雰囲気で話しつつ、新しい出会いを提供する場所を目指しており、定期的に学生主催のイベントを開催しています♪

今回のテーマは、、、ずばり【食】です！

食欲の秋…

食べることも、料理を作ることも、とにかく「食」が好きな方、秋の夜長に美味しいイタリアンを食しながら、「食」について語りあってみませんか？

みなさまのご参加をお待ちしております♪

当日は現役大学生でありながら、シェアカフェでもあるFrohカフェで金曜日に店長をしている竹村さんと、お友達の近江さんにお料理を作っていただきます。

参加者の皆様には、食について思うことを一人3分以内でプレゼンしていただきます！

【テーマ例】
・私の得意料理
・現代の食文化について
・好きな食べ物
・私のおふくろの味、おばあちゃんの味
・外食の思い出
・今興味のある料理、食材
などなど、なんでも好きなことを熱く語ってください( ω )

＊タイムライン＊
19：00 開場、受付
19：30 フローコワーキング・シェアカフェの紹介
19：35 自己紹介と、食についてのプレゼン(一人3分以内)
※プレゼンを聞きながら、全員に一言フィードバックコメントを書いていただきます
20：15 お料理を食べながら、交流会
21：00 写真撮影、解散

＊参加費＊
学生1000円、社会人2000円

四ツ谷駅からすぐなので、ふらっとあそびにきてくださいね。
お友だちと一緒でも、お一人様でも、大歓迎します！

※当日、記念撮影をするので撮影がNGな方はお気軽にご相談ください☆
