description

日本におけるキリスト教の布教は、1549年フランシスコ・ザビエルが鹿児島に渡って来たときから始まります。
そして、秀吉による26聖人殉教やキリシタン禁制下の鎖国時代の徳川幕府の厳しい弾圧に耐えながら、キリシタンは、密かに信仰を守り続けました。そして数代、二百数十年を経て、フランス人神父との再会をここ大浦天主堂で果たしたのです。長期間の重い弾圧に耐え、信仰を代々守り伝えたことは世界史的にも稀有なことで、キリスト教を信仰する人たちにとってばかりでなく、日本の歴史としても誇るべき、輝かしい出来事です。
このようなキリシタンの強靭な精神と歴史を可能にしたのは何であったのか、それを考えてゆくことは、混迷と不安が覆う今日の日本を生きる私達にとっても多くの示唆を与えるもとなるでしょう。
