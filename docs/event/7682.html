catch 今、Web制作者の間で話題のCMSのセミナーを北九州市で開催します。　開催協力：ガレージインク・アットウィル
description

Web制作に関わる方のためのセミナーイベント「CSS Nite LP6（CMS リベンジ編 2009年）」での参加者投票 1位の実績を持つ、話題の国産CMS「a-blog cms」の紹介セミナーを北九州市で開催します。

a-blog cmsの特徴って？

  • 複雑なプログラミングの知識は不要
  • コンテンツはもちろん、管理・投稿ページのカスタマイズが自由自在
  • クライアントも分かりやすい更新画面
  • 静的サイトからの導入が容易
  • フォーム機能は標準搭載
  • SEO・LPOを考慮した機能

詳しい内容は、 a-blog cms公式HP を是非一度ご覧下さい。
