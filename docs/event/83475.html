経済的な自由や時間の自由、精神的な自由を手に入れたいと思いませんか？

お金がないことで取り組みたい事に制限されたり、
買いたいものを制限されたり、大切な時間を切り売りしたり、
わたしたちは様々な自由を奪われ生活をしています。

今回ご紹介する先生は
インターネットビジネスで半月150万円稼ぐ男の子をプロデュースしたり
セミナー活動で月商1200万超を超えるプロモーションのプロデュースしたり
20万円をたった8時間で130万円に変えたありえない投資法をプロデュースしたり
ファイナンシャルプランナーとしては異色の経歴を持つ宮先生を招待してセミナーを行います。
（保険の話しはいっさいしませんｗ）

先生は22歳の若さで飲食店を経営されましたが
ご家族のご事情で店舗を売却することになったそうです。

それまで社長を務めていた人生から一転して
金なし、コネなし、学歴なし。
社会に通用する能力もなければ、資格なんて持っていない、
さらには社会人経験さえないという人生のどん底を味わったようです。
周りを見渡せば
昇進や起業の話、景気のよさげな話ばかりが耳につき
ご自身は当時雇っていた従業員からもそっぽを向かれる始末だったようです。

そんな状態からファイナンシャルプランナーの資格を取得し、
某最大手の損害保険会社からヘッドハンティングを受け、
その経験を活かして再度起業を果たされました。

お金で失敗されている方なので、
わたしたちの悩みや解決したいことを馬鹿にせずに快く受け止めてくれて
一緒に解決策を考えたり相談に乗ってくれる素晴らしい先生です。

そんな先生が何年もの時間とお金をかけて学んできたことを
「お金の4つの力」という先生のオリジナルカリキュラムに変えて説明してくれます。

お金の使い方、稼ぎ方、増やし方、守り方をビジネス初心者向け
に分かりやすくお話ししてくれるのでどんな方でもご参加いただけます。

当日は懇談会もあり、
お金のことが楽しく学べるような構成になっています。

お時間に余裕のある方はぜひ一度ご来場ください。

会場でお会いできることを楽しみにしています。
