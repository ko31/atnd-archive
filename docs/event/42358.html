description

［講座のポイント］
　本講座では、真空システムの設計・製造・運転上、何に気を付けるとトラブルと失敗を避け、無駄なくそのシステムの性能が発揮できるようになるか、その背景にある真空の基礎を学ぶとともに、具体的なトラブルや失敗の原因および対策を解説する。

［受講対象者・レベル］
　主に初級・中級の現場担当者向け

［プログラム］
１．はじめに
　　1.1　基礎の無理解とブラックボックス化がトラブルと事故を起こす！　
　　1.2　真空作業が絡む大事故例に見る基礎の無理解
　　1.3　真空システムとリスクマネジメント
２．無駄なトラブルと事故を避けるため、3時間で学ぶ真空物理の基礎のキ！　
　　2.1　気体分子運動論と気体の性質
　　2.2　気体の流れ
　　2.3　排気の方程式
　　2.4　コンダクタンスと排気速度
　　2.5　有効排気速度
　　2.6　ガス放出速度とは
　　2.7　真空排気の理想と現実
　　2.8　気体原子・分子や荷電粒子と真空材料表面との相互作用
３．真空材料と加工
　　3.1　材料とその特性
　　3.2　材料特性を無視した材料選び（使用して後悔する材料）
　　3.3　機械加工・接合
　　3.4　低真空用でも無視できない表面処理・表面改質
　　3.5　低真空用ならベーキング・プリベーキングは不要？
４．真空機器構成部品
　　4.1　各種真空ポンプの原理と特徴
　　4.2　各種真空計と残留ガス質量分析計の原理と特徴
　　4.3　その他、真空システムアクセサリー
５．真空システムの設計・製造・運転・保守におけるトラブル事例と解決法
　　5.1　真空容器・部品材料の選択ミス
　　5.2　排気ポンプと真空計の選択ミス
　　5.3　真空ポンプが無駄になる真空配管
　　5.4　オイルフリー排気系の落とし穴
　　5.5　そのオペレーションではせっかくの真空システムが汚れる！
　　5.6　ケチってはいけない真空システムアクセサリー・周辺機器・設備
　　5.7　出荷梱包から組立調整の間で生じる瑕疵
　　5.8　間違ったヘリウムリークテスト法と解釈
６．万事うまく行くと・・・　
　　6.1　ベーク不要！極高真空への高速排気
　　6.2　不純物フリー低真空装置への高速排気
７．まとめ
