・入室の方法
ビル入口とエレベーター前で、それぞれ
「呼」ボタン＋「2300」と押す→ラウンジ内の呼び鈴鳴り、ロック解除
→エレベーターで23階へ

翌日に帰国するアメリカ人大Perfumeファン、thispenguinの送別会です。
（日本語ベースなので、英語喋れない方も大丈夫★）

★氷結とお酒とピザとスナック付きの3時間で、会費はたったの2,000円★
★締切は、2/2開始直前まで★
★未成年の方もソフトドリンクで参戦下さい★

代々木以来、Perfumeのライブの度に来日して今回が5度目。
沖縄遠征を躊躇している自分を殴りたくなるくらいの
素晴らしいアメリカ人Perfumeファンと、
Perfumeまみれの3時間を過ごしてみませんか？

初対面の方も、というか、こそ、来て頂きたい位です。

マンションのラウンジを貸し切って、
ホームパーティーのようなあたたかい送別会にしたいと思っています。
遅刻も早退も全く問題ないですよ。

しかし、お料理が少ないかもしれないので、差し入れ大歓迎です☆
thispenguinに食べさせてみたいもの、なんかも是非。

ミュージシャンでもある彼に、アコギでココスポやthe best thingを歌ってもらう予定です。
そして、彼の伴奏で皆で歌ったり踊ったりも！
さいたまの感動も分かち合いたいですね。

振りコピ（あまりドスドスできませんが）や、曲のコピーが出来る方、
是非是非いらしてパフォーマンスして下さい！
