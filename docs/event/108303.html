お久しぶりの、マーメイド基礎体験会です。

9/22、BumB(夢の島・東京スポーツ文化館)を17時〜21時のロングで取りました。
ぜひ遊びにいらしてください。

施設の都合で次回開催は11月以降、そして来年はオリンピック関連で施設が長期閉鎖のため、開催可能性自体がかなり怪しいです。
今回是非遊びにいらしてください。

●やること
マーメイド基礎練習会、水中撮影練習会とか。
すでに参加の意向を表明頂いた方の中に水泳ガチ勢経験者が多そうなので、ちょっとしたリレー遊びとかも考えています。

●参加費
フル参加の方は2000円。今回4時間コースなので、いつも(2時間)の倍になります。
前半のみ、後半のみ参加の方は1000円です。

●参加方法
こちらのイベントをぽちってください。
メールアドレスかTwitterアカウントの登録を頂いて、主催グループからコンタクト可能なようにお願いします。

●持ち物
必須：水着、タオル、キャップ
ほぼ必須：ゴーグル
持って来てもOK：フィン(マーメイドやモノフィン含む)、シュノーケル、ノーズグリップ、水中カメラ、その他もろもろ水中で使いたいもの
持ち込みNG：水着系やマーメイド系じゃない衣装、ウィッグ、その他水を汚しかねないもの
