一般社団法人グローバル･ベンチャー協会ならびにＩＴサービス企業の 株式会社うえじま企画は、「ＩＴを活用して社会に貢献する」という経営理念のもと、 共催で、オープンイノベーションを実現する活動を推進致しております。

今回の東京ビジネス交流会では、下記内容をテーマに実施致します。

「ＩＴエンジニア採用の現状・課題と取り組み」

当日は、採用支援サービスを展開しているベンチャー企業にプレゼン頂く予定です。

また、ベンチャー企業から求めるＩＴエンジニアにつきましてプレゼン頂く予定です。

その後、参加者の皆様と意見交換会を実施予定でございます。

＊ビジネス交流会の過去ご参加企業の一例

タイムスケジュール

　　　　　　　・グローバル・ベンチャー協会理事長挨拶

　　　　　　　　（企業紹介、商品サービス案内、資金調達希望など）

　　　　　　　・各支援部のご紹介

２０：００～　ビジネス交流会　

　　　　　　　・上場企業とベンチャー企業のマッチング・名刺交換
