iPhoneアプリ開発マスターコース（RainbowApps名古屋栄校）

本格的な１０個のサンプルアプリからiPhoneアプリ開発をマスターします。

　　プログラム未経験者でもOK 受講生のほとんどが１講義目でアプリ開発成功
　　日本最大のiPhoneアプリ開発スクール 受講生１５００名以上の実績
　　受講生限定のメーリングリスト、掲示板、ソース共有など受講後のサポートも万全

　　毎回１個ずつベースアプリを講師の懇切丁寧な説明のもと実際にコーディングする形で進行します。
　　全コースを通じて、あなたの作りたいアプリを開発し、AppStoreに登録、公開するのがゴールです。

受講対象者

　　iPhoneアプリ開発のスキルを学びたい方
　　iPhoneアプリ開発に対応できるエンジニアを目指す方

受講料

　　98,000円　 申込はこちら

持参していただくもの及び、受講に必要な準備

　　Mac(ノートPC）及びiPhoneまたはiPad/iPodTouchはご自身でご用意下さい。
　　iPhone SDKを使っての開発には MacOS X SnowLeopard またはLion を搭載したIntel ベースのMacが必要です。
　　また、Apple社のiOS Developer Programへの登録には別途費用がかかります。
　　Mac Book Airはイーサネットアダプタが必要な場合があります。
