description

●受講対象者
消費者の潜在ニーズを把握し、商品・サービスに必要な提供価値の習得方法について学習したいかた。EXCELの操作に不安のないかた（テキストで操作説明もあり、実際に操作を行いますが、EXCELで関数入力を行ったことがあればベストです。）

●習得知識
・消費者の潜在ニーズ把握に必要なポイント
・潜在ニーズ把握に必要なアンケート作成の事例
・潜在ニーズを中心としたSTP（誰に・何を・どのようにして）を把握するための分析方法（アンケート調査を事例として）
・デモデータによる統計解析手法の習得

●講座の趣旨
　現在国内の市場は成熟かつ縮小しているため、競合商品との競争がより激化しています。また、口コミ評価サイトを活用して、商品・サービスに関する情報収集も簡易に出来ることから消費者の情報優位性もより高まっております。
　そのような環境の中、新しい商品・サービスの市場動向を見るには、消費者の価値観を実現するために必要な課題、特に簡単に表現しづらい“潜在ニーズを把握”し、課題解決可能な商品・サービスの実現を目指す事がより重要となってきております。
　本セミナーは潜在ニーズとその潜在ニーズの実現に必要な視点把握をするためのアンケート作成方法と分析方法についてフォーカスした講座にしております。

●演習内容
デモのアンケートデータをEXCELの関数などを用いながら、統計解析手法を学びます。
画面操作がEXCEL2013・テキストがEXCEL2007・2013を考慮して作成されているため、
Microsoft Office EXCEL（2007以降）がインストールされているPCをご持参下さい。
またUSBメモリなど、外部データのダウンロードが拒否されないように設定をお願いします。

●プログラム
１．潜在ニーズとは？（講義）
　　1-1．ニーズと潜在ニーズの違いは？
　　1-2．なぜ、潜在ニーズの把握が必要か？

２．潜在ニーズを把握する調査プロセス（講義・一部ワーク）
　　2-1．現状把握の分析方法
　　2-2．課題抽出方法
　　2-3．アンケート作成方法

３．アンケート作成の注意点（講義）
　　3-1．質問項目の性質と注意点
　　3-2．質問文作成上の注意点

４．潜在ニーズ視点のＳＴＰ分析事例（講義）
　　4-1．潜在ニーズを把握し、市場をセグメントする
　　4-2．顧客の価値観から市場をセグメントする
　　4-3．顧客の購入目的から潜在ニーズを把握し、市場をセグメントする

５．潜在ニーズ抽出のための分析方法（デモデータを活用したＰＣ演習）
　　5-1．ベネフィットの束からニーズを把握する方法
　　5-2．ニーズの束から潜在ニーズを把握する方法
　　5-3．プロファイルを含めて、潜在ニーズを持つ方々の姿・形を明確にする方法

６．まとめ
