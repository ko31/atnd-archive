description

朝カフェの会は、当日参加大歓迎、途中参加、途中退出自由のゆる～い交流の場です。
事前申込みは不要ですので、お気軽にご参加ください。
参加費は、各自当日の飲食代のみです。
机の上に青い旗を立てていますので目印にして下さい。

【開催日時】第４金曜日　7:00～9:00

【開催場所】モスバーガー荒川遊園前店（東京都荒川区西尾久6-30-6）

【参加費目安】珈琲 230円～/朝食各種有

【会場の特徴】都電の電停からあらかわ遊園へ向かう道の入り口付近にあるモスバーガー。地元に人に愛されている地域密着のお店です。
客席は１階、２階に分かれていますが、１階奥の席で歓談しています。
主催者からのメッセージ 地域の方との交流の場であることに加え、あらかわ遊園を見てみたいという区外からの参加も大歓迎です。 平日ですが、お時間がある方は朝９時から開園するあらかわ遊園もお楽しみ下さい。 動物や乗り物、釣り堀などを楽しむことができます。 また、都電が走る下町の風景もお楽しみ下さい。
