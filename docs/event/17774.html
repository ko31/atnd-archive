実装会情報

ZIPの圧縮・解凍を実装する勉強会です。
日曜の昼に、5時間ほどかけて、実装・情報交換など、行います。
ZIPを通じて、圧縮のアルゴリズムへの理解を深めようという勉強会です。
開発環境・開発言語は各自での用意をお願いいたします。
当会は、C, C++, Java, Common Lispを推奨言語とします。

会場情報

会場には10分前から入れます。
会場は貸し会議室を利用しますので、一人あたり 1,000円の会費を負担していただく予定です。
1ドリンク制で、会費はドリンク代込みです。
