どうなるMovable Type

2013年7月10日に、Movable Type 6ベータ版が発表となりました。

http://www.sixapart.jp/movabletype/news/2013/07/10-1000.html

これに伴い、ユーザー数限定版ライセンス、およびオープンソース版（MTOS）の提供終了が発表されています。
今後2年間のセキュリティアップデートはあるものの、新規案件へのMTOS導入は悩ましいものがあります。

http://www.movabletype.jp/blog/movable_type_Life_Cycle_Policy.html

a-blog cms？

国産CMS「a-blog cms」はMovable Type5のコマーシャルパックに近いライセンス料金と機能を持ち、無償版も提供されています。

http://www.a-blogcms.jp/

a-blog cmsはMovable Typeの空白をサポートする存在となりうるでしょうか。また、導入の際は、どこが改善点・懸念点となるのでしょうか。

「Movable Typeコマーシャルパックの提案が難しい案件の代替として、a-blog cmsを考えてみる」

というシビアな内容にフォーカスして、久しぶりにa-sapを開催します。

テーマ

MovableTypeからの移行について、真剣に考えよう

リリースから4年、バージョン1.7になったa-blog cmsの現状と、MTOS・Movable Typeコマーシャルパックから移行した場合の違い、考慮すべきことについて、セミナー形式で紹介します。

こんな人におすすめ

- これまでMTOSを主力にしていて、今回の発表で困っている制作者さん
- CMS採用の決定権を持っていて、MT対応に迷っている、ディレクターさん
- だいぶ前にa-blog cmsを触ってみたけれど、最近のことはよく知らない制作者さん
