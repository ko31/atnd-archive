description

ネイティブアド、キュレーション、アプリ化・・・。
近年メディアの在り方が急速に移り変わっていくなかで、各メディアで実施している先進的な取り組みや、チャレンジしている取り組みをこの場限りで一挙大公開するイベントを開催します。
リアルタイムの現場で起こっている成功事例や失敗事例を共有することで、メディア各社様の新しい取り組みへの心理的障壁を下げたり、失敗の事前回避に役立ててもらえることを期待しています。

■登壇者



  • エキサイト株式会社
    スマートフォン推進室プロデューサー　池村修 様

    1981年沖縄生まれ沖縄育ち（サトウキビとシーサーはだいたい友達）。2005年、エキサイト株式会社入社。音楽サービス「エキサイトミュージック」のプロデューサーとして、音楽業界にも携わる。2012年、スマートフォン事業に異動。以降、「エキサイトニュース」「おとえほん」「寝たまんまヨガ」等のアプリを担当。世界中のユーザの生活を楽しくするアプリサービスの提供を目指して、チームメンバーと日々邁進中。趣味はサーフィン。

  • ヴォラーレ株式会社
    執行役員　土居健太郎 様

    1984年横浜生まれ。東京大学を中退し、2009年ヴォラーレに入社。同社のSEO事業立ち上げ人であり、現任の事業責任者。コンサルティング業務監修のほか、自社メディアApplivのSEOの監修も行う。

  • 株式会社LIG
    LIGブログ編集長　朽木誠一郎 様

    1986年生まれ。国立大学医学部在学中にフリーライターをはじめる。卒業後、医師にはならず、2014年4月に（株）LIGに新卒入社。同年9月より月間400万PVのオウンドドメディア『LIGブログ』編集長として企画・編集・執筆を担当。2015年2月より、LIGがはじめる人材サイト『Poole』編集長、3月より広報戦略室室長を兼務。

  • 株式会社ゲーム攻略完全図鑑
    マネージャー　津田　大樹　様

    1989年生まれ。早稲田大学在学中、メディア事業で学生起業。その後、投資会社を経て、アドネットワーク国内大手のアイモバイル社に入社。主にゲーム攻略メディアを担当した。メディアのマネタイズの課題に直面したことから2014年10月よりゲーム攻略完全図鑑へ入社し、メニュー企画、営業、マーケティングを担当。


    • ■場所

    Impact HUB Tokyo（東京都目黒区目黒2-11-3 印刷工場１階）

    Impact HUB Tokyoは、現在世界70都市に展開する世界最大規模のグローバルなコワーキングスペースの国内初の施設として、2013年2月11日に東京・目黒にオープン。"志あるチェンジメーカーが世界とつながる秘密基地”をコンセプトに、起業家が集まる”コワーキングスペース”という枠組みを超え、世界に変革をもたらし、互いに学びや知識、経験、ネットワークのシェアを実現する有機的なコミュニティを創ることを目指す。
