普段何気なく食べているコンビニスイーツとパティシエが作ったスイーツは何が違うのか・・・・・。

違いのわかるスイーツ会です。

パティシエ歴１２年現役パティシエが語るコンビニスイーツと手作りスイーツの違いは必見です。

今回用意するパティシエスイーツはナチュラルスイーツです。　

もちろん手作りです！

一緒に食の真実を知りましょう！

ちょっとしたタメになる話からマニアックな話までジャンジャン話しちゃおうと思います！

堅苦しくなくアットホームにみんなで楽しみながら学んじゃおうっていう企画です！！

なので今回は限定６名！

スイーツを食べながら楽しく勉強会＆交流会です！
