PHPの 『 超 』 初心者向け勉強会です。

難易度は 『 全く難しくない 』 です。

対象者は 『 プログラム未経験者 』 です。

PHP初心者勉強会は定期的に開催しています。
http://sites.google.com/site/phpbeginners/

定期的に開催している初心者勉強会は、PHPは未経験でも他のプログラムの経験がある人を対象としているため、
プログラムそのものが初めての方では、ついてくることができませんでした。

『 超 』初心者勉強会は、プログラム未経験者の方を対象にしていますので、誰でも参加できます。
実際にサンプルプログラムを作りながら解説を行います。

ご案内

  1. MySQLというデータベースの使い方について勉強します
  2. MySQLの基礎の基礎を勉強します
  3. XAMPPというソフトについてるphpMyAdminというソフトを使います
  4. 時間は午後7時から午後9時までの2時間です
  5. ノートパソコンをお持ち下さい。（なくても参加できます）
  6. 飲食物の持ち込みはできません
  7. 受講料が2000円かかります

XAMPPというソフトウェアを使うので、以下のURLからXAMPPをインストールして下さい。

Windows
http://www.apachefriends.org/jp/xampp-windows.html

Mac OS X
http://www.apachefriends.org/jp/xampp-macosx.html

XAMPPのインストール方法は、以下のURLをご覧下さい。
http://www.db.is.kyushu-u.ac.jp/rinkou/mysql/xamppinstall.html

インストールがうまくできなかった場合には、インストールをお手伝いしますので、少し早めに会場に来て下さい。
