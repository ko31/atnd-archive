エンタープライズ風なVoIPネットワークを自作してみる。という試み

音声ネットワークもインフラの一つなんだし、自作して遊ぼうぜ!!という
ハンズオン大会です。
今回のイベントは闇鍋てっく様に便乗する形で会場をお借りしています。

やること

  1. サーバーにAsteriskを入れてみよう
  2. IP電話をAsteriskに繋げてみよう
  3. 他の参加者と拠点間通話してみよう
  4. PBX持ってきてVoIP-GW経由でVoIPしてみよう (オプション)
  5. IPv6でVoIPしてみよう (オプション)

持ってくると盛り上がるモノ

手ぶらでの参加でもOKです。
でも、以下の機器を持ってくると盛り上がるかもしれません。

Asteriskが動いているPC
OSを問いません。轟音サーバーOK
IP電話機
自宅のオブジェクトになっているかもしれないIP電話機が蘇ります。
併せてPoEに対応したSW-HUBを用意できるとなお良しです。
Asterisk以外のPBX
CUCME（CiscoのIP-PBX）などといった、Asterisk以外のIP-PBXで通話検証をやってみることで、
面白おかしく電話のシステムが学べます。
レガシーPBXを持ってきてもOK。なお100V以外の電源はありませんのでご注意ください。
VoIP-GW　(BRIインターフェース付きのものを推奨)
レガシーPBXを持参する場合は必須になります。
その他小物
OAタップとかUTPケーブル(5mくらい)とかあると、大変助かります。

会場とか

調整して頂いてます。 ←New

喫煙所は利用できませんので、ニコチンの補給は外でお願いします。

注意点

  • IP電話機スキー。でも電話は大の苦手。というコミュ障が胴元やります。
  • 秋葉で現調してもOKですが、箱は持って帰ってください。
  • 外線はさすがに用意できないので勘弁してください。
  • 懇親会は別途企画中です。
