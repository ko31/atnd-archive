マニフレックス大阪ショールームでは、質のよい眠りに関心のある女性のためのアカカツ（セミナー）“ボンジョルノ！マニフレックス”を開催しています。朝の時間を有効活用していただけるよう、朝7時30分にスタートする体験型セミナーで、参加費500円は全額寄付されます。おいしい軽食と飲み物をご用意して、皆様のお越しをお待ちしています。

マニフレックス朝活 11月・12月プログラム

11月18日（金）朝7:30～8:30

骨盤矯正ピラティス

『骨盤矯正ピラティス』とは、身体を整える美容整体とボディメイクができるピラティスを融合させたオリジナルレッスンです。身体の土台である骨盤には、肩こりや腰痛などの不調を改善するためにとっても重要な役割を果たします。骨盤は開きすぎても閉まりすぎてもよくなく、スムーズな骨盤の開閉リズムをキープすることが大切です。骨盤が整えば、自律神経も安定し質のよい眠りにも繋がるはず。
毎日をキラキラと元気で輝くために、骨盤リセットを体験してみませんか？

講師：三矢 秀子 先生
トータルバランストレーナー　美容整体・ピラティスサロン「マハロハ」代表　整体ピラティスの普及に努める
