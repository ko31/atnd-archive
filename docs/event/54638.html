W3Cが2014年7月24日に公開した Techniques for WCAG 2.0 (Public Editors’ Draft 24 July 2014) にコメントを送ります。

Techniques for WCAG 2.0（WCAG 2.0 実装方法集）は、WCAG (Web Contnet Accessibility Guidelines) 2.0の達成基準を満たす実装方法や失敗事例をまとめた資料です。このドキュメントはWCAG 2.0そのものを置き換えるものではなく、WCAG 2.0を補う資料であり、定期的に更新されています。

このイベントでは、先月公開されたTechniques for WCAG 2.0のPublic Editor’s Draftの実装方法や失敗事例の内容を確認し、必要があればコメントを送ります。
（今回のPublic Editor’s Draftに対するコメントの募集期間は2014年8月12日までです)

このイベントの参加者は次のいずれかを行います。

  • Techniques for WCAG 2.0に対するコメントの提出
  • 新たなTechniqueの作成、投稿

このイベントでは次のようなことは行いません。

  • WCAG 2.0の解説
  • Techniques for WCAG 2.0の解説
