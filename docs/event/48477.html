description

株式会社アドライフ（東京都新宿区　代表　井本弘巳）は、平成26年3月29日（土）二子玉川ライズ・オフィスにて、俳人堀本裕樹さんを主宰に元チャットモンチ―・ドラム担当の高橋久美子さんをゲストに迎え、俳句が初めての方でも安心して楽しんでいただける俳句イベント「カタリストBA句会」を開催します。

「カタリストBA句会」では、「俳句をつくったことがない」、「句会に初めて参加する」、という方にも安心して楽しんでいただけるよう、俳句のつくり方や句会の楽しみ方をわかりやすくレクチャーします。

自分の俳句をつくって参加するもよし、他の参加者の俳句から優秀作品を選ぶ「選句」のみで参加するもよし、どなたでもお気軽にご参加いただけるイベントです。

また、ゲストの高橋久美子さんとのトークや、ご本人がつくった俳句もお楽しみいただけるほか、参加者の俳句の中から優秀作品「特選」を堀本裕樹さんと高橋久美子さんが選出、特選に選ばれた方には、素敵なプレゼントもご用意しています。

堀本裕樹プロフィール

いるか句会・たんぽぽ句会主宰。第36回俳人協会新人賞、第2回北斗賞受賞。『すばる』にてピース又吉直樹と「ササる俳句笑う俳句」、『波』にて穂村弘と「俳句と短歌の待ち合わせ 17×31」、『Webちくま』にて「富士山五七五」連載中。主著に『十七音の海俳句という詩にめぐり逢う』、句集『熊野曼陀羅』。
