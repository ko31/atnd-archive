収益を最大化させるアプリ広告実装セミナー

そもそも売れてるアプリって何でそんなに稼げる訳？どうやって稼ぐの？
作ったアプリをみんなに見て欲しい。売れて欲しい。
広告入れるからには稼げなきゃ意味が無い！収益を最大化させるappC cloudの種類豊富な広告と今流行のマッチアップ広告の説明および、広告の入れ方のアドバイスと共に実装してみましょう！
初心者から経験者まで幅広くこの機会に広告実装を体験して、開発したアプリをより魅力的な稼げるアプリに。前回好評につきアンコール開催いたします。前回参加したけどもっと知りたい。前回参加できなかったけど今回は参加できそう！皆さんいらっしゃいませ、またやりますよ！

対象

iPhoneアプリ開発者（よちよち歩きの開発者からバリバリの開発者まで）
ノートパソコン（mac）の持ち込みが必須となります。
開発環境のインストールおよびappC cloudの会員登録をお願いします。
※開発環境（XCODEなど）については最新のものでお願いします。バージョンが古い場合のフォローはできかねます。

appC cloudの事前登録が必要です

appC cloud

※無料で登録できます。事前に登録しておいてください。

ノートパソコン以外に持参いただくもの

受付の際にお名刺を頂戴しますので受付用・名札用の２枚ご用意下さい。
*名刺をお持ちでない場合は、学生証等の身分証明書の提示をお願いします。
飲み物の持ち込み可能です。こぼれないよう各自ご注意ください。
（食べ物の持ち込みはできません）

会費

無料

会場について

マルチメディアスクールWAVE様のご厚意により無料で貸していただいております。Wifiおよび電源が利用できます。

注意事項

※席数が少ないのでお早めにご応募ください。
今回も懇親会はありますが、参加希望者のみ参加費をいただきます。参加費は500円です。
※当日の進捗や準備が必要な部分もあるため、講師だけ実装する箇所もあるかもしれませんがご理解ください。

キャンセルについて

前日までに必ず、キャンセルをお願いします。
万が一当日のキャンセルの場合は、ご連絡くださいますようお願いいたします。
