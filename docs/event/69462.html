ヒカ☆ラボ一覧は こちら！！

★講演内容

昨今注目度を増しているインフラの自動構築・オーケストレーションが可能なAnsibleを
自身の経験もふまえながら、明日からすぐに使えるAnsibleをご紹介します。
入門的な内容を多く含みますが今後の使われ方、また他のツールや
エンタープライズでの活用も意識した内容も含めてお話させていただきます。

Agenda

- What’s Ansible
- Ansible Design Principles
- How to use Ansible
- Ansible for Enterprise



※内容は変更になる可能性がございます。

TIS株式会社 IT基盤技術本部
IT基盤技術推進部 OSS推進室
主査　倉持健史 氏

SIer、ソフトウェア/ハードウェアベンダーでのサポートやソリューション開発、
エバンジャライズな経験を活かし、現在はTIS株式会社OSS推進室にて、
自社のサポートサービスやクラウド時代におけるOSS活用を社内外にてプロモーション中。
インフラ寄りのスキルセットが主だが、
ここ数年は統計学、Rにも興味を持ち何かに活かしたいと考えている。
