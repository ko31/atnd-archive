Interface 2014年12月号「USB探偵団」オフ会やります！

日時：2014/11/21(金) 19:00～21:00
定員：25人★好評につき若干数ですが定員増やしました★

特典：USB通信まるはだかアイテムUsbStudy主要部品セット
　　　（詳細はInterface2014年12月号特集「目指せ小型でハイパー！USB探偵団」参照）
会場：CQ出版社 セミナ・ルーム（東京都豊島区巣鴨1-14-13 炭七ビル1階）
URL：http://interface.cqpub.co.jp/

　身の回りにあふれるPC向けのさまざまなUSBデバイスを，300円ていどで入手できるワンチップ・マイコンPIC24FJ64GB002から操ってみます．難しいと思っていたUSBドライバは，300行ていどから，非常にシンプルに作れます．方法はInterface2014年12月号特集「目指せ小型でハイパー！USB探偵団」で紹介しました．特集筆者：中本 伸一氏に，実演＆解説いただきます．

【補足情報】
　会場にはPICマイコン用の書き込み器PICkitを用意しますので，普段PICを使っていない人でもその場でPIC24FJ64GB002に書き込みができます．

■ こんな人大歓迎！
●特集で紹介していないUSB機器用ドライバを作ってマイコンから操ってみた
●PIC24FJ64GB002以外で特集で紹介した会話型USBホストUsbStudy/USBドライバを動かしてみた
●いいかげんUSBをブラックボックスじゃなくてちゃんと理解したい
