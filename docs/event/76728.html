description

屋外ジンギスカンパーティー「J2」の関東支部が久しく開催されていませんでした。現在、連絡メーリングリストと告知用ウェブサイトが停止しており、通常開催が困難な状態です。なんとか今夏に開催を予定していますが、まずは集まって肉食しましょう!! 場所はあの「北京」を確保しました。平間の本店ではなく向河原の支店ですが「予約できる」「追加注文できる」という北京初心者にもやさしい店のため、北京の魅力をじっくり堪能頂けます。

  • 日時: 2016年4月29日(金・祝) 17:30-20:00 ※いわゆる「肉の日」です
  • 開催場所: 北京向河原店 (川崎市中原区下沼部1905-13)
  • 集合場所: JR南武線向河原駅改札口 (JR横須賀線からは、武蔵小杉駅横須賀線出口より徒歩が便利です)
  • 予算: 3000〜5000円 (飲まない人に差を付ける割り勘を予定)

注意

  • 申し込み時は直近に参加したJ2の開催回もしくは年・季節をご記入ください (関西支部、新潟支部、東海支部の回でも構いません。回数はだいたいで結構です)
  • ATND で申し込みたくない場合は別途連絡をください。
  • 家族連れで申し込む場合はコメント欄に書くか別途連絡をください (残り枠数を調整します)
  • ドタキャンは店からキャンセル料を取られるため「行けたら行く」は困ります。前日20:00に人数をfixしますので、以後にキャンセルされて欠員が出た場合はキャンセル料をもらいます。(キャンセルしても補欠で補充できれば問題なし)
  • 補欠が多数出るようなら予約人数を増員しますので、満員でもあきらめないでください。
