Twilio API 勉強会 Vol.8を開催致します。

■ Twilio API 勉強会？
クラウド電話 API Twilioが好きなかたもしくは興味をもっている方エンジニアの皆さんが集まるイベントです。
勉強会ではTwilioを作りながらいろいろ技術的な情報交換をしています。
