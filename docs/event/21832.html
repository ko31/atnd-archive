description

「達人プログラマー システム開発の職人から名匠へ道」 の読み合わせを行います。
読み合わせとディスカッションを通して、pragmaticなプログラマーを目指すためのアプローチを学んでいきます。
「興味はあるけど〜」「読んでみたけど難しかった〜」という方、一緒に読み合わせをしてみませんか？

前回 の続きですが、初参加でも大丈夫です。お気軽にどうぞ！

内容

  • 自己紹介タイム
    参加していただいたみなさまに自己紹介していただきます
  • 「達人プログラマー」読み合わせ
    輪読形式での読み合わせを行います。途中15分程度の休憩を取ります。
    予定している範囲は次の通りです。
    • 第6章「コーディング段階」、セクション31 「偶発的プログラミング」からセクション33「リファクタリング」前後まで

なお、書籍については各自で用意をお願いします。

会場について

エスプランニング様のご好意により会場を提供していただきました。いつもありがとうございます！
