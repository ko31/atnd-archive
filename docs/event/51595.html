description

明るく、華やかに見える格好を知りたい！
好きな人に見違えたねって言ってもらいたい！
男性とお話するのが苦手！

そんな、不器用な女性を応援したいと思っています！
私自身、今までと違う自分になりたいと思ったのをきっかけに自分を変える努力を始めました。

その結果、３年ほど経って、雰囲気が変わって素敵になったと言われるようになりました。
知っていた方には別人のようだと言われたり、知らない方には以前が想像つかないなんて言われます。

そんなに周りの方がほめて下さるなら、自分がして効果のあったことや知ったコツを、お伝えすることで女性らしく自信を持つことをお手伝いできたら素敵だなって思っています。

流行りを追うつもりも、個性的な突飛さを出すつもりもありません。
素敵に見せるには、そういうことが必須ではないからです。
あなたが可愛く見えて、より自分を好きになれればそれでいいんです！
少しの勇気で何かが変わります。
変わりたいと思っている方を思う方向へ進むお手伝いをしたいと思っています。
気負わず、どうぞお越し下さい。

内容

  1. 講師自己紹介とビフォーアフター。
  2. 女性らしく見せるために、最低限したほうが良いこと。
  3. 大人の女性が素敵に見えるファッションってどんな格好？
  4. 合コンで、男性をにこやかにする鉄板言葉とは？

場所

住所: 大阪府大阪市浪速区湊町１－４－１　OCATビル４F

藪田　瑛について

  • 幼いころより３年前までは、女性らしく綺麗にする必要性を感じず、可愛いものや綺麗なものは自分には似合わない、むしろ恥ずかしいとさえ思っていました。
    婚活をきっかけに、今まで避けていた女性らしさと向き合うこととなり、分からないなりに日々取り組むことで徐々に苦手意識を克服していました。
    そして、初めてすることを楽しむうちに、見た目も女性らしくすることが普通になり、マインドも変わり、楽しく安定した毎日を過ごせるようになりました。
