・セミナーの内容：2018年は“AIの民主化元年”と呼ばれるように、誰もがAIを使える社会が到来しようとしています。こうした潮流を後押ししているのが、市民データサイエンティストと呼ばれる人々の活躍。そして、こうした人たちが本職のデータサイエンティストのような高度なスキルがなくてもAIの機能を使ってデータ分析・活用できるツールの登場です。言わば、Excelを使って資料を作成するのと同じように、ツールを使って画像認識や音声認識、異常検知といったAIの利用が可能になってきています。
こうした状況の中、最右翼のツールとして注目されているのが、ソニーが開発した「Neural Network Console」（以下NNC）です。データサイエンティストのようなスキルを持たない普通のビジネスパーソンでも、ドラッグ&ドロップ操作でAIの機能を易しく利用できます。
本セミナーでは、このNNCに関する国内唯一の書籍である「ソニー開発のNeural Network Console入門」の著者と、NNCの開発者が適用領域から利用方法、活用のポイントをわかりやすく解説します。

・セッションの詳細
第1セッション：「Deep Learningの活用を加速するNeural Network Console」
＜内容＞Neural Network Consoleは、効率的なDeep Learningの応用技術の開発、迅速なDeep Learningエキスパートの育成のために大きな効果を発揮するDeep Learning専用ツールです。
本セッションではNeural Network Consoleの概要、応用事例のご紹介、および基本的な使い方と特徴的な機能のデモンストレーションを行います。
★講師のご紹介　小林由幸氏
1999年にソニーに入社、2003年より機械学習技術の研究開発を始め、音楽解析技術「12音解析」のコアアルゴリズム、認識技術の自動生成技術「ELFE」などを開発。近年は「Neural Network Console」を中心にディープラーニング関連の技術・ソフトウェア開発を進める一方、機械学習普及促進や新しいアプリケーションの発掘にも注力している。

第2セッション：「あなたも市民データサイエンティスト！　一味違うNNCの使い方をデモを交えて解説」
<内容＞NNCを使って時系列データから異常検知モデルを作成する方法、テキストデータから分類モデルを作成する方法をデモを交えて紹介します。NNCは、マウスのドラッグ＆ドロップだけで簡単にAI（ディープラーニング）モデルを作成できる、市民データサイエンティストに打ってつけのツールです。今日から皆さんも、AIを活用する第一歩としてNNCを使い始めてみませんか？
★講師の紹介　足立悠氏
メーカーで働くデータサイエンティスト。ユーザー企業においてデータ分析・活用を推進、ベンダー企業において国内企業のデータ分析・活用を支援した経験があり、両方の立場からデータサイエンスに携わっている。データサイエンスの普及を目的に、Webや雑誌へ記事執筆してきたほか、国内各地でセミナー講師を務めている。主な著書に「初めてのTensorFlow─数式なしのディープラーニング」、「ソニー開発のNeural Network Console 入門─数式なし、コーディングなしのディープラーニング」がある。
