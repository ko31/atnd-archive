本イベントは主に東京西部、神奈川西部のエンジニアの交流を目的としたものです。
普段エンジニア交流会は都心部、平日に多くおこなわれており、それらの補完を目指しています。

pmとついている様に、Perlユーザーが多くいますが、話題はPerlに限定されているわけではありませんし、主催を始め、PHPやJS、HTML等をPerlよりも使っている人多くいます。
エンジニアであれば、お気軽に参加いただければと思います。

特に若い人、お気軽に参加ください。

—
いつものHachioji.pmではなく、今回はTakao.mt、高尾山でミーティングです。
今回は基本高尾山ビアマウントでビールを飲む会です！！！

—
高尾山ビアマウントは、土曜は予約不可能、整理券方式なので、
・全員同時にビアガーデンにはいれないかもしれないです
・大丈夫だとおもいますが、仮に当日整理券の発行が終わってしまっていたら、急遽八王子のどこかに移動します
・整理券を入手後、時間が空いた場合は山に登ります。（といってもせいぜいハイキングです）
・雨天は居酒屋開催に変更になります、予約もないのでキャンセルOKです。
という事をご理解ください。

今回普段の集合の18時ではなく、13時です（14:30開催なので並んで待ちます）。

集合は京王線高尾山口駅です。
高尾山口駅についたら、Twitterなどで連絡を取り合いましょう。
（ハッシュタグは#hachiojipm）

会費制ではありません。高尾山ビアマウントの成人男性価格は¥3,500です。
http://mount-takao.com/beermount/
途中ケーブルカー、またはリフトにて上りますので、その分の料金往復¥930が必要になります。
http://www.takaotozan.co.jp/timeprice/#h3_02

歩いて登る人はもちろん無料ですね！（ビアマウント周辺に、本イベント開催時間+20分くらいにはついていてください）
（できれば、登る人はコメントで登るって表明してください）

—
高尾山薬王院に祈願を予定しておりますが、大体いつも土壇場までどうなるかはわかりません。
ここ数年、「暑くて無理だな？」という判断で諦めるということが続いております。

—
おそらくは終わった後、八王子駅前で温泉にはいって、さらにどっか吞む予定ですが、これも予定は未定です。
