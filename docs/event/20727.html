description

ICTソーシャルアクションミーティング（略称ISAM）は、
東日本大震災における各種震災復興支援に携わる方を中心に、
エンジニアやビジネスリーダー、起業家、投資家などが
組織やサービスの垣根を超えてお互いに協力できることを探し、
支援成果の最大化を目指す会です。

★ISAMの狙い★

1. ICTを用いた復興支援を手掛ける事業者や個人間の横連携をうながす
2. 今後求められる支援の形を皆で模索し、さらなるソーシャルアクションを醸成
3. IT×ビジネスの人材交流で、支援活動を継続していくためのナレッジ共有を促進

【第2回の開催概要】

今後の自立運営、支援拡大を目指す【復興支援団体】【支援企業】の運営課題を
「ワークショップ」形式で有志が集まって議論し、
運営課題の解消や横連携に向けたアクションを応援していきます。

【Point-1】 ワークショップで運営課題を発表したいプレゼンター募集中！

プレゼンターとして、5団体が参加決定!!（2011.10.26現在）
各団体の活動にご興味をお持ちの方、ぜひご参加ください。

■放射性物質測定マップ＆除染支援『HOPE-Japan』（ http://hope-japan.tv/ ）
■ボランティアマッチング支援『ボランティアインフォ』（ http://volunteerinfo.jp/ )
■南三陸 戸倉復興支援団（ http://www.shien-togura.jp/ ）
■南三陸の事業者支援ネットショップ『南三陸町 de お買い物』（ http://www.odette-shop.com/ ）
■必要物資・支援要求マップ『311HELP.com』（ http://311help.com/ )

ワークショップでナレッジを提供くださる各種専門家の顔ぶれも決定！

■A.T. カーニー　プリンシパル　國分俊史氏（ビジネスコンサルタント）
■グロービス経営大学院　講師　加藤剛広氏（マーケ・ネットビジネス戦略）
■『助けあいジャパン』『sinsai.info』『Hack For Japan』等で活動中の方々

【Point-2】参加費を、被災地から来場されるプレゼンターの交通費に全額フィードバック！

今回は、ご参加いただける方々から【参加費1500円】を頂く予定です。
集まった参加費を、全額、被災地から参加いただくプレゼンターの交通費に充てることで、
被災地で活動している支援団体（人）が抱える課題を解消するお手伝いをしていきます。

【ご参加いただきたい方々】

■各種復興支援に取り組む企業・ボランティア団体の方

■支援の自立運営／事業継続／ヒト・モノ・カネの確保を助言できる方
（コンサルタント、アントレプレナー、ビジネススクール講師、社会起業家など）

■復興支援事業への参画やサポートに興味を持つ個人の方
（エンジニアや各種ビジネスパーソン、学生、企業のCSR担当者など）　etc.

震災復興を支援するあらゆる人たちの「悩み解消と再創造のプラットフォーム」を形成するためにも、より多くの人にお集まりいただき、知恵とスキル、そして復興に向けたパワーを持ち寄りましょう！
