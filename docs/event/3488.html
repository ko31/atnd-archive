□ 1st Session [高確率な新規開拓営業の追求]
───────────────────────────────────
◆ 田口 雅浩 (たぐち まさひろ)

株式会社快的空間
営業部

氏名：田口 雅浩 (35歳) 1975年1月2日生まれ Ｂ型 やぎ座
家族構成：妻 娘2人 (長女6歳、次女3歳)
趣味 : 野球 (バリバリの体育会系です。)
経　歴 :
地元の工業高校卒業後 (もちろん野球部) →広島の専門学校へ
職歴 : 広告代理店勤務 新規開拓営業に携わる。その後、医療機関への衛生管理商材の新規開拓営業に携わる。
昨年11月より快的空間にてＷｅｂ関連新規開拓営業に携わる。

セッション概要 : 今までの営業経験の失敗事例や成功事例などを発表させて頂き、皆さんで『高確率な新規開拓営業を追求』についてディスカッションし
ましょう

□ 2nd Session [業界裏事情もふまえたサーバーとCMSの選定]
───────────────────────────────────
◆ 高畑 哲平 (たかはた てっぺい)
KDDIウェブコミュニケーションズ
SMB事業本部 事業本部長

Twitter : http://twitter.com/teppeitakahata/

自己紹介 :　老舗のレンタルサーバーブランドCPI、2009年に立ち上げましたCMS Jimdoの事業統括をしております。
経　歴 :
2001年、Canadian International Collegeを卒業後、ケンウッド・ジオビットに入社し、オーディオアプリのマーケティ
ングと戦略を担当。
2003年、経済誌専門の広告会社に入社し、同年東京支社長に就任。
2006年、CPI（現 KDDIウェブコミュニケーションズ）に入社。マーケティングと新規事業立ち上げに従事。
2007年、中小企業をベースに活動するSMB事業本部長に就任。
2009年、ドイツ発のWebサイト制作ツール「Jimdo」の日本独占販売権を取得し、同サービスのカントリーマネージャーに就任。
現在、レンタルサーバーCPIの事業統括、Jimdoの事業統括を行う。

セッション概要 : 敬遠しがちなサーバーの選定、偏りがちなCMSの選定方法をシンプル且つ裏事情もふまえてお伝えします。
また、低価格案件（制作費、サーバー代込みで10万～50万円）にいかに手離れよくウェブサイトを構築するか、
いかに効率よくウェブ制作を行うかを紹介致します。
グラフィックデザイナーさんが簡単にウェブ制作に参入できてしまう方法も。

□ 3rd Session [提案時のワイヤーフレームとかスケッチとかについて]
───────────────────────────────────
◆丸山 祐司 (まるやま ゆうじ)
株式会社ココノヱ
取締役・ディレクター

Twitter : http://twitter.com/yujixxx/

自己紹介 :
こんにちは。岡山の制作会社、ココノヱの丸山です。
今回もお邪魔いたしますのでよろしくおねがいいたします。
がおー。
経　歴 :
岡山県出身。
専門学校卒業後、大阪の設計事務所でプラント設計等に従事。
2003年創業時、退職し放浪していたところを拾われ参加。
社では企画提案業務を主に担当。

セッション概要 :
初期提案時に、がっつりカンプつくる時間やデザイナー動かす余裕が
無い時にフレーム的なものやスケッチなどで（絵なんて描けないのに）
イメージを伝えなければならない時があるかと思いますがそういうのの
事例とかツールとかを紹介してみます。
