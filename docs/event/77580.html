【得する保険・損する保険】
難しい保険もゼロから分かる簡単保険セミナー

国民の８割が加入しているといわれる身の保障を守る『保険』

しかし、現状は
もしかしたら、過剰の保険内容かもしれない。
もしかしたら、子育て世代なのに万が一の時の保障がない。
など、

毎月の生活費を圧迫したり、
そもそも本当に自分に合ったプランなのか？
心配ではないですか？

また、
「何がいいのか分からず、営業マンに勧められるがまま加入した」
「友人知人の営業で断り切れず加入した」
という方も実際にいます。

【20代の方】
「自分は病気もしないし、元気だから、保険は必要ない」と
思っていませんか？
保険料が安く済む若いうちに加入すれば、その後も
保険料が高くなる事もなく、終身までご契約できる
メリットはかなり大きいですね♪

【30代の方】
結婚や出産という大きく環境が変わるこの世代。
忙しさに追われて、肝心な家族を守る保障の見直しは
されましたか？

【40代の方】
10年以上前に加入したままのものではありませんか？
どういった保険の内容かご存知ですか？？
働き盛りのこの世代、万が一の時の保障は万全ですか？

【50代、60代の方】
高額な保険料の割には、契約期間が70歳までとかの内容
ではありませんか？ご自身の保障内容は把握されて
いますか？

「保険は難しい・・・」と、思い込んでいませんか？
いいえ、難しい事はありません!(^^)!
きちんと説明を聞けば、理解出来ます！
今回はそんな保険に対して、
全く分からない方でも理解しやすく、
そしてあなただけに合った保険のセミナーを開催させて頂きます。

『保険』は全ての年代の方に当てはまるはずです！

〜〜〜こんな方にオススメ〜〜〜

・難しい保険を簡単に理解したい
・もっと得する保険を知りたい
・調べたり、勉強しなくても保険の事を知りたい
・見直しして自分に合った保険にしたい
・お金の事をもっと知りたい
・過剰に保険金を払うのは嫌だ

〜〜〜〜〜〜〜〜〜〜〜〜〜〜〜

既に保険に加入されている方も、
これから検討するという方も
まずは聞いてみてご判断下さい。

保険営業マンに任せるのではなく
お客様と一緒に相談できる存在でありたいと思っております。
