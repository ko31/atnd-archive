title 第16回iWest勉強会
catch iPhoneアプリを作ろう！(Xcode入門)
description

　福岡Appleユーザグループ「iWest（アイウエスト）」の勉強会開催のお知らせです。今回のテーマは『iPhoneアプリを作ろう！(Xcode入門)』です。実際にMacを操作しながらアプリ開発の手法を初歩から学ぶ、非常に実践的な勉強会となります。

　講師は福岡県糸島市の観光スポット紹介アプリ「いとしまっぷ」(iPad専用)や、ショッピングサイト比較アプリ「お買い得.net」をリリースされているひだけん（@hidaken）さんです。

　2012年7月にもひだけんさんに講師をお願いして開催しましたが、当日は満員御礼。勉強会の様子を収録した動画も再生回数が1,600回を超えるなど、大変ご好評をいただいている内容です

　また、勉強会終了後に懇親会も予定しております。都合のよい方はこちらにも引き続きご参加ください。

　皆さまのご参加をお待ちしております。


【日　時】2014年5月17日(土) 13:30〜16:30
　　　※本会終了後18:00から懇親会を開催予定

【会　場】ふくふくプラザ（福岡市市民福祉プラザ）501研修室

【テーマ】iPhoneアプリを作ろう！(Xcode入門)

【参加費】無料（懇親会参加者は実費）

【受講に必要なもの】
　　　OS X 10.8.4 以降と最新版のXcodeをインストールしたMac

《2014.05.08追記》

参加される方は次の3点をお読みになり、ご準備をお願いいたします。

　1.Xcode の最新バージョンをインストールしてきてください
　　（ならびに mac を対応バージョンにアップデート）

　2.Xcode をインストールしたら一度起動して、
　　他のオプションもインストールするように求められた場合は、
　　それもインストールしておいてください

　3.当日は、マウス（できれば右クリックができるもの）があった方が
　　作業しやすいので、持っているかたは持参してください
