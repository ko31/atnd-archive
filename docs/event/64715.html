昇龍蓬莱の会
日時:5月17日(日) 17時開始　※16時45分開場予定
会場:土と青 (京王線 調布駅より徒歩4分)
会費:7000円
ゲスト:大矢孝酒造 菊池譲さん

昨年に引き続き今年も調布「土と青」さんでお酒を楽しむ会を開催させていただくことになりました。

今回は神奈川県の大矢孝酒造で「残草蓬莱（ざるそうほうらい）」「昇龍蓬莱（しょうりゅうほうらい）」を醸す菊池譲杜氏をお招きします。

菊池さんの造るお酒の特徴は、まずは使用している酵母がすべて７号酵母ということです。
パイナップルやバナナを想起させるような香りのする旨みのあるジュシィ―なお酒が多いです。

流行の最先端を行くであろう12度の低アルコール原酒や、白麹の使い酸を活かしたお酒など「冷」で飲むことを意識した残草蓬莱。

今期より全て生酛（きもと）の造りとなり、冷だけでなく「燗」向きの昇龍蓬莱。

同一酵母を使用していますが、酒米や造りの違いでバラエティー豊かな味わいを楽しませてくれます。

私と菊池さんの出会いは2013年の春。ある居酒屋さんのカウンターでたまたま席が隣同士になったご縁で、そこから仲良くさせていただいています。

会場となる「土と青」さんの最寄駅は京王線の調布。
新宿から特急に乗れば、2駅、20分かからない近さです。

店主・松本さんの造る料理は明石から仕入れている魚介類を中心とした日本創作料理。
決して派手な味わいではなく「淡さ」を感じさせる仕事で、しみじみと「旨いなぁ～」と唸りたくなります。
店内はレトロな雰囲気で落ち着きがあります。接客が心地好く、ついつい長居してしまう素敵なお店です。

土と青さんでは普段から残草蓬莱、昇龍蓬莱を扱っていて、先日松本さんを蔵へ見学にお連れして、今回の会を開催することになりました。

翌日が月曜と言うことで、開始時間を17時にしました。

だんだんと夏に近づく5月の日曜日、土と青さんで菊池さんと一緒に楽しく過ごしましょう。
