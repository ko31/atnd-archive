description

システムエンジニアからITコンサルタントへのキャリアアップセミナーを行います。
以下のような考えをお持ちの方、是非当社セミナーへお越しください。

「コンサルタントになる方法を知りたい」
「コンサルタントの考え方を仕事に取り入れたい」
「キャリアアップで収入を上げたい」

ITエンジニアとしてプロジェクトに参画していると現在の業務をこなすことが精一杯で、これからのキャリアや今後の収入についてじっくり考える機会はあまりないかもしれません。
このセミナーでは、大手外資系コンサルティングファーム出身の当社トップコンサルタントが、コンサルタントになるためのスキル・考え方などの秘訣を、特別に公開します。
BTC社内のコンサルティング研修で実際に行われているカリキュラムを基に再編成した実践的なセミナーです。

■参加費■
　無料

■対象者■
・今後コンサルタントとして活躍していきたい方、もしくはITコンサルタントとして活躍されている方(35歳以下限定)

※以下の方は是非ご参加ください！
・現在のキャリアのままではキャリアアップのイメージが湧かないかた
・自分のスキルや仕事に対して満足できていない方
・エンジニアからITコンサルタントへのキャリアモデルを持ちたいかた
・現在の収入に満足していないかた

■セッション一例■
・エンジニアとコンサルタントの違い
・コンサルタントとしての考え方、働き方
・フレームワークの紹介・実践

※セッション内容については若干変更させて頂く可能性がございます。
