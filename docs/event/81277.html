description

東日本大震災や熊本地震をはじめとした災害。猛暑日の続く日々に、ゲリラ豪雨や異常行動する台風といった異常気象。また、障害者施設での殺傷事件など、殺伐とするニュースが日本中を流れました。一方では、リオオリンピック一色だった今年の夏。メダルラッシュのニュースに始まり、そして2020年のTOKYOオリンピックに向けた盛り上がりへと余韻を残しました。
そんな様々なニュースを耳にしながら、「今の社会って何かおかしくない？」と疑問をお持ちになったり、自分自身に問いかけた方もいるのではないでしょうか？中には、「なにか社会に役立つことをしたい」「この世の中をもっと良くしたい」と考えている方もいるでしょう。しかし、自分では何も社会を変えることができないことに、憤りを感じていらっしゃるかもしれませんね。
では何を、どうすればよいのか。果たして自分に何ができるのか。とあれこれ考える前に、行動を起こさなければ、何も変わらないことを皆さんはご存知だと思います。まず一歩進むことがとても大切です。自分の身の回りのことや自分自身が自分らしく生きることから行動することで、ほんの少しでも社会を変えることが出来るのではないでしょうか？
そのような行動や活動をされる方を、「ソーシャルアントレプレナー（社会起業家）」と呼びます。そんな方々の活動の事例や皆さんの想いを少しでも後押しさせていただく、体験授業を開催いたします。

体験授業では、
①社会起業家やソーシャルビジネスについての講義
②実際に社会起業大学に通う学生のビジネスプランの説明
③参加者全員でビジネスプランについてディスカッション
という構成でお話を行います。

参加される方は、年齢・性別・ご職業などが様々です。しかし、さまざまな背景をお持ちの参加者に共通しているのは
「社会に貢献し、社会をより良くしたいという意思」です。
世代の異なる方や他業種で務めている方からの何気ない一言が新たな行動の種となることもあります。
そんな仲間と出会える無料体験授業にあなたも参加してみませんか？
初心者向けの内容ですので、社会起業家やソーシャルビジネスについてよくご存じでない方でもお気軽にご参加頂けます。
また、起業を考えている方だけに限らず、社内の新規事業や企業の社会的責任(CSR)活動をご担当されている方にも、何か気づきになるような内容となっています。お気軽にお申し込みください。

※申込者が、最低実施人数に満たない場合は、受講日の変更をお願いすることがあります。あらかじめ、ご了承願います。
