【Bohnanza（ボーナンザ）】
ボードゲーム大国のドイツでカード部門1位になった
優良カードゲームです！
http://ja.wikipedia.org/wiki/ボーナンザ

交渉には三つの肝があります。
手法や理念といった「知識要素」
準備を軸にした「プロセス構築」
そして相手を目の前にした時の
状況分析や判断、話術などの「アプローチ」

カードゲームでは場面によって変わる展開に
準備できる要素はごく限られていますが
どう状況判断しアプローチしていくか、
そんなトレーニングにも。

そしてこのゲームの肝は「交渉」
価値交換を心掛けて交渉を行わないと
決してゲームに勝つ事が出来ません。

アクションに取り入れられた
交渉フェーズをどう活かすか、
正に「遊びの中にある学び」です(^^)/

また、一緒に同じことを体験した仲間は
その印象も繋がりも濃くなります。

参加者同士のコミュニケーション（交流）と
交渉のトレーニングを遊びながら行う
一石三鳥の企画です。

聞いたことがない・・・
全く問題ありません！！

ゲームガイドは勿論！
私達、交渉のプロがゲームマスターとして
ガイドラインの解説を致します。

ぜひ遊びに来て下さいね！
