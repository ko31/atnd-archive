description

～仏教を説かれたお釈迦様（ブッダ）の智恵から、
　人生で大事なテーマを学べます～

日常で、こんなことを思われたことはありませんか？　

「私らしく、迷いなく生きていきたい。
　でも、そもそも私って何だろう？」
　
「毎日、会社と家の往復で同じことの繰り返し。
　いったい、何のために生きているのかな？」　

「イライラして、仕事もプライベートもうまくいかない。
　怒りっぽい自分がイヤになった…」

「努力はしているのになかなか報われない…
　もうがんばるのをやめてしまおうかな。」

「職場の人間関係に、もう疲れた…。
　どうすればいい人間関係をつくっていけるのかな？」

職場でも家庭でも、
このような悩みを人知れず抱えている人は少なくありません。

それらの疑問に対し、
仏教（ブッダの智恵）ではどのように教えられているのでしょうか。

◆講座内容◆

○心理学よりずっと深い！ 仏教で説かれる「本当の私」とは？

○「毎日同じことの繰り返しに、どんな意味があるの？」

　仏教でハッキリわかる“生きる意味”

○もう悩まない！ イライラをしずめる仏教の教え

○努力は必ず報われる！　～勇気がもらえる、因果の法則～

○仏教が説き明かす　幸せになれる６つのタネまき
