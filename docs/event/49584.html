初夏の川越で、ビールを飲みながらライトニングトーク大会

小江戸らぐ(小江戸 Linux Users’ Group)を主宰しているはとちゃんです。

東急東横線が地下にもぐり、地下鉄副都心線や東武東上線とつながって横浜中華街から川越まで一本で来れるようになったことを記念にはじめた、スピンオフ企画の Open Source UN-Conference 川越 を、今年も小江戸らぐの本拠地"川越"で開催します。

小江戸らぐといえば、ゆるく活動を続けて今年で12年目になるので、それにあわせて小江戸らぐが春に開催するBBQパーティで出している生ビールを会場に持ち込み、これを飲みながら ライトニングトーク をやります。みなさんが日頃やっていること、これからやろうとしていることなど、思いの丈を 5分間 にこめて語ってください。

Ustreamで中継

当日、Ustreamで13:30〜 中継する予定です。会場に来られない方は、ぜひ Usteram でご視聴ください。

http://ustre.am/ICGw

ビール飲み放題と地の食べ物

ライトニングトークをゆるく聞きたいので、 飲み放題 の 生ビール と地の食べ物を用意します(なくなったらごめんなさい)。

ライトニングトーク

各自 5分間 でLinuxやOpen Source Softwareに関して日頃やっていることや、これからやろうとしていることなど思いの丈を語ってください。

発表者は、受付に発表リストを用意しておきますので、そこに記入してください。また、参加表明時のコメントに、ライトニングトークしますとか、話す内容を書いていただけると、進行の目安になりますので、よろしくお願いします。
