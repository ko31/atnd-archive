Salesforceのオンライン学習教材Trailheadのもくもく会です！


https://trailhead.salesforce.com/ja
Trailheadは、Salesforceの学習のために大変に有用な教材ですが、業務内ではなかなか学習する時間がとれない場合もあるとおもうので、みんなでTrailheadでモクモク勉強する場を設けてみました。
Salesforceの学習という同じ目的で集まりますので、わからない部分など教えあったり、情報交換したりといったコミュニケーションもとれればと思っています。
注意：誰かが講演するタイプの勉強会ではありませんのでご注意ください。

懇親会

勉強会後は、会場で出前でも取ってお酒を飲みながら懇親会をできればと思っています。
(参加費500円、参加者３名以上で実施)
同じテーマで集まっていますので、今後情報交換できる仲間づくりにお役立ていただければ幸いです。
是非積極的に参加していただけると嬉しいです。

グループ

勉強会の告知やSalesforceの学習に関する情報交換を目的としたFacebookグループを作りました。
覗いてみてもし気に入ったらグループに参加していただけるとうれしいです。
