要はゆる～い忘年会なのですが、あの「みちしお」で開催しますよ～♪

おかげさまで、下関では記念すべき「第10回」を先月無事に終了し、
宇部では、「第3回」にして、フィールドワーク＆オンパクと連携し、
地域性を保ちながら、着実に歩んできました、井戸端サークル@しものせきが展開する「哲学カフェ」。
【過去の開催レポート】⇒ http://bit.ly/9ALtrQ

この度「望燃会」と称し、記念すべき【下関＆宇部の合同例会】を開催／哲学カフェに参加したことがない方も大歓迎!!

今回はいつもの井戸端会議のように、気軽な「哲学カフェ」スタイルではなく、
ドライブインで、食事やお酒、そして、温泉を楽しみながら、ざっくばらんに楽しみましょう！！！

開催日時／2010年12月10日(金) 20:00～23:00？

  2. 下関方面から「埴生駅着」＝19:29着

  4. 終電…下関行・23:22埴生駅発／新山口行・23:16埴生駅発

会場／ドライブイン みちしお（山口県山陽小野田市埴生）

参加資格／「人間」であれば誰でもOK（笑）

  1. 年齢制限なし、世代を越えた交流をしましょう！
