ボードゲームに関するツイートでAdvent Calendarをやってみませんか！？

初めての試みなのでやってみないとどうなるか予想できないっ！！

Advent Calendar とは

クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーです。転じて、ネット上のコミュニティで、クリスマスまでの25日間にTipsを1つずつ公開するイベントをAdvent Calendarと呼ぶそうです。

ルール

Advent Calendarはブログに記事を書く形で進めることが多いようですが、初めてなのでお手軽にtwitterでのツイートでやっちゃいましょー！

例）「.@△△から（の続き）。☓☓というボードゲームおすすめだよー！コンポーネントもかわいくて超すてき！！明日は@○○ ですー #advent12bg」

  • イベントに参加した順で 12/1 から 1 日ずつ参加者がツイートする
  • 12/1 はイベント参加 1 人目，12/2 は 2 人目，… 12/n は n 人目 … 12/25 は 25 人目 … という流れです
  • ツイートには「明日は@○○○ さんです」という形で翌日の参加者にリンクする
  • ツイートじゃ物足りない！って人は思う存分ブログに記事を書いてそこへのリンクをツイートしてください

テーマ

ボードゲームに関することならなんでもOK！

  • あのボードゲームが好き！
  • あのボードゲームをやりたい！
  • おすすめのボードゲームを紹介！
  • ボードゲームの思い出。
  • こんなボードゲーム作ったぜ！
