イベント概要


NTTデータ、Panasonic、DENSOの3社の出資からなる当社。

流通・製造業各界トップカンパニーのパートナーとして
コンシューマ向けモバイルアプリ、ITソリューション開発を、
最上流工程から行っています。

“NTT DATA MSE Tech Career Talk”は、
私たちと技術力とアウトプットに触れていただき、
エンジニアとしてのキャリアのヒントを
提供するイベントです。

今回のテーマは、”Edge AI Solution”。
Deep Learningによる画像認識システムを、
Edge Computing技術を使って
車載にて実装したケースをお話しいたします。

※ CES 2018にてデモンストレーションも公開。

　当日はお見せいたします。

コラボレーションを加速し、
イノベ−ションを生み出すことを目的としたこの拠点で、
あなたの知的好奇心を刺激してみませんか。
