名古屋で２回目のUnreal Engine 4 ミートアップです

第１回Unreal Engine4名古屋勉強会に続いて、２回目の勉強会です。今回から正式にUnreal Engine 4ミートアップとして開催します。今回は、2015年10月に開催されたUNREAL FEST 2015 YOKOHAMAの報告会がメインになります。前回参加された方も、参加していない方もぜひご参加下さい。

また、IGDA名古屋では2015年1月29日から31日の予定で、Global Game Jam2016という国際的なゲームジャムの名古屋サテライト会場を開催予定です。前回の福島GameJam2015でもUnreal Engineを使ってゲーム制作したチームが名古屋でもありました。

なお、諸事情により変更等があった場合はご連絡を差し上げますので、できるだけメールアドレスをご設定の上、参加申し込みをしてください。
