description

福岡にて第1回 ネットワーク勉強会を開催します。

  • 当勉強会はネットワークに関連した様々な内容を学ぶ勉強会です。
  • 基本的にネットワークに関連するものであれば、レイヤーを問わず様々な内容を扱います。
  • 「技術」だけに限定しているわけではありませんので、ネットワークにご興味がある方はお気軽にご参加ください。
  • 特に学生の方は、普段社会人と接する機会が少ないと思いますので、是非ご参加ください。
  • なお、当勉強会は 「ネットワークビジネス」 の勉強会ではありませんので、その界隈の方はご遠慮ください。
