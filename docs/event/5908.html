title 第一回 人類ガラクタ博覧会
catch リアル脱出ゲームのSCRAP「加藤隆生氏」トークイベント＆交流会
description

人類ガラクタ博覧会とは、
斬新で面白いアイデアをガラクタ呼ばわりされようが、
チャレンジし続けて成功を収めた方々を講師としてお招きし、
参加していただくお客様に
少しだけ“自信”と“勇気”を分けてもらう
面白起業家勉強会＆交流会です。
いろんなナレッジ、ノウハウだけでなく、
おそらく一番必要であろう“熱意”も教わります。

第一回目のゲストは、
全国でリアル脱出ゲームを開催している
株式会社SCRAPの代表「加藤隆生」さんをお招きして
現在の活動から苦労談など笑いあり涙ありのお話をして頂きます。

◆講師：株式会社SCRAP 加藤隆生氏
◆日時：2010年7月4日（日）
　　　　16:00～18:00　一部：トークイベント
　　　　19:00～21:00　二部：交流会
◆会場：アートセントー：女湯（元・天神橋温泉）
　　　　大阪市北区天神橋7-13-17
◆参加費：トークイベントのみ：500円
　　　　　トークイベント＋交流会：3,000円

【PROFILE】
SCRAP　加藤隆生（代表、編集長）
フリーペーパーSCRAP編集長、ロックバンド「ロボピッチャー」ギターボーカル、リアル脱出ゲーム発案者、くるりARGプロデューサー。
空間を使った謎解きイベントに卓越したセンスを発揮する。あるとき、ある場所をまるで物語の中のように演出するゲームイベントを得意とする。
普段は思いつくままにフリーペーパーを編集したり、陽気に曲を作ったり、飲み明かしたりしている。趣味は働くこと。遊ぶように働いて、あそびを仕事にするのも得意。
website:株式会社SCRAP
