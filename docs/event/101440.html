あなたは自分のゴールに対する自分の位置は分かりますか？

自分の人生のゴールを持っている際、自分の現状を正確に把握することが必要です。

正確に状況を理解すれば、

◆起業でも、

◆ビジネスや人間関係でも、

◆日常生活や趣味でも、

あらゆるゴールに向けて何を行えばいいのかわかります。

自分の長所は何なのでしょうか？

自分の短所は何なのでしょうか？

その答えがこのテストに正確にあらわれてきます。

「自分の能力を、自分がどう評価しているか」が、実際に分かります。

そして、「なぜ」、自分がこの状態なのか？

この「なぜ」が正確にわかるのも、このテストの特徴です。

はっきりと言えることが１つあります。

それは、“自分の大きさを決めるのは自分自身”だということです。

このテストがその大きさを変えるチャンスをつかむ一歩です。

あなたの今後の人生を大きくする為のチャンスを、自分の大きさを変えるチャンスをあなたは手に入れたいですか？

～　200問の簡単なYes・Nｏテストです　～

自己分析テスト、アンケートご記入の後、自己分析結果を行います。時間は約1時間30分～2時間程度を予定しております。

テストにより、自分の現在・過去の経験がどのように現状の能力に影響を与えているかが分かります。

結果はグラフで表示され、１０項目の自分の能力の状態が分かります。

結果については、スタッフが説明いたします。

何が原因でその結果が出ているのか理由もちゃんとあるので、それを一緒に見つけていきます。

このテストで自分の能力を更に発揮して人生をより良く過ごすためのチャンスを手にしましょう！

～～～グラフで表示される能力の例～～～

○安定性／不安定：あなたは人生の目的が明確か？人に流されやすいのか？

○幸福感／抑圧：自分の目標に向かっているのか？誰かにストレスを受けているのか？

○確実性／不確実：自分の考えなどに確信を持っているのか？着実に行動できるのか？

○責任感／無責任：物事について意欲的にコントロール出来るのか？まあいいやということをしているかどうか？

○積極性／抑制的：自分の考えを発して行動するのか？それとも受け身になって言われて行動するのか？

○コミュニケーションできる／できない：あなたは相手に自分の考え等を伝えられるかどうか？

などなど…。

自己分析結果の説明後、ダイアネティックスR・カウンセリング、本などのご紹介をさせていただきます。

●個人の現状についてとその原因・理由を知ることで、自分をより理解出来、未來をどう進むかの視点が得られる。

●自分で分かっている自分のことだけでなく、潜在的に隠れた自分の部分も知ることが出来る。 こちらのテストの所要時間は約2時間です。

お申し込み後、事前に行っていただくと、所要時間を短縮することができます。
