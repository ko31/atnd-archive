イベントの概要

GC アルゴリズムの1つであるマーク＆スイープの実装過程を追いながら勉強する会です。GC やスタックの理解を深めたい方向けです。説明には C++ を使用します。

はじめに概要を説明してから、参加者の皆さんで実装に挑戦していただきます。日曜工作のような感覚で GC を作ってみるという趣旨です。

Mono 開発者の @atsushieno さんによる Mono for Android の GC 周りについてのセッションも予定しています。

ご参加をお待ちしています。

準備する物

実際に開発を行いますので、ノートパソコンをご用意ください。説明に使用する環境は以下の通りですが、説明を読み替えできる方であれば必ずしも合わせる必要はありません。

  • CPU: IA-32 に依存した説明をします。
  • OS: 64bit OS でも 32bit コードが動けば構いません。説明には Windows 7 64bit を使用します。OS にはあまり依存しないため UNIX 系 OS でも構いません。
  • コンパイラ: 説明には MSYS＋MinGW を使用します。i386 向けの gcc なら他の OS でも大差はありません。VC++ をご利用いただいても構いません。
  • デバッガ: 説明には Insight を使用します。gdb 直接や他のフロントエンドが使い慣れている方はそちらをご利用いただいても構いません。VC++ をご利用の方は内蔵デバッガをご利用ください。

※ これらの使用方法は勉強会で解説します。

Insight

Insight は以下からダウンロードしてください。

- insight-6.6-mingw.tar.bz2

適当なフォルダに展開して bin/insight.exe を起動すれば使えます。起動しない場合、バイナリエディタで 00DC を 02→03 に書き換えてください。
