内容

※まだ仮の内容ですが、大きくは変わらないと思いますので、公開しておきます。

IoTあるじゃんでは、IoTの実現に、主にMicrosoftの技術を使っています。
このため、本ワークショップでは、クラウドとしてMicrosoft Azureを使い、IoTデバイス側では、.Net Micro Frameworkを使います。

    PinKit(GR-PEACH＋センサーボード)を使って、取得した温度をクラウドへアップロードしてみます。

事前にご準備いただきたいもの

以下の2点は、ワークショップの内容を実践するためには必須となります。必ず準備をお願いいたします。

  1. ノートPC
    PCには、インストール手順資料を参考に、以下のソフトウェアを事前にインストールしておいてください：
    • Visual Studio Community 2015 Update2（Professionalでも可）
      Visual Studio のインストールは、かなり時間がかかります。当日にインストールするとインストールだけでワークショップの時間全部を使ってしまいかねないので、 必ず当日までに インストールをお願いいたします。
    • Azure SDK
    • .Net Micro Framework SDK
  2. Microsoft Azure の契約
    https://azure.microsoft.com/ja-jp/free/ から「無料のAzureアカウント」作成で契約を行なうと、 契約後１ヶ月間 、20500円分の無料枠が付いています ので、ワークショップ中の作業で課金されることは、まずありません。
    また、契約を行なっても、Azureの有料機能を使用しない限り、課金はされませんのでご安心ください。
    詳しくは、 https://azure.microsoft.com/ja-jp/free/ をご覧ください。

    ※Microsoft Azure アカウント作成の際の注意点
    • 無料期間の関係上、 ワークショップになるべく近い日 に、アカウント作成手続きをしてください。
    • アカウント作成には Microsoftアカウントとクレジットカード（プリペイドタイプも可能だそうです）(プリペイドタイプは使えません)が必須 となります
