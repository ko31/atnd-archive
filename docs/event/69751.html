description

IPAが実施する情報処理技術者試験のうち、最も専門性が高いレベル4の試験に該当する「ITストラテジスト試験」の試験を受講するかたに向けた試験対策講座です。

主に以下のような方を対象としています
・実務経験がないが合格を目指している方<
（経験がなくても合格することは可能です）

・午後試験の勉強法がわからない方

・論文試験に不安がある方

・受験の意思がある方

※試験の研究や経験によるノウハウをお伝えすることで合格の可能性を上げることを目的としています。

アジェンダ（予定、変更になる可能性があります）
１．各試験区分のポイント
２．情報処理技術者試験トピックス
３．午前試験対策
４．午後（記述式）試験対策
５．午後（論述式）試験対策
６．まとめ

※筆記具を持参ください

※昨年度開催し、好評いただきました講座の内容をさらにブラッシュアップした内容を予定しています。

講師については日本ITストラテジスト協会の会員で情報処理技術者試験を何度も受験、合格し熟知した者が担当いたします。ご期待ください！

参加費：無料
※Peatexでお申し込みの方はこちらのでお申し込みは不要です。

日本ITストラテジスト協会について
日本ITストラテジスト協会は、ITストラテジストおよび、ITストラテジストの資格取得を目指す方で構成されるプロフェッショナルコミュニティです。

本試験対策講座は、日本ITストラテジスト協会九州支部により、資格取得を目指す方の支援、ITストラテジストの認知度向上を目的に無料で開催されます。
