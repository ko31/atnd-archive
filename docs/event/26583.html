日本で英語を話す機会がないと思いませんか？

留学から帰って来たけど、英語を話す機会が減ってしまった！

これから海外へ行きたいけど、気軽に英語を話す時間を作りたい！

朝の時間を有効に活用したい！

そんな皆さん。

留学仲間が主催する『英語de朝活！』に参加し、英語での会話を一緒に楽しみましょう！

〜英語で朝活！とは？〜

出勤前や登校前の朝食の時間を有効に活用し、英語での会話を気軽に行います。

英会話のレベルは関係なし！

朝食を食べながら、参加者と共に英語での会話を楽しみましょう。

〜内容〜

皆が話せる機会を持てるよう、少人数制で行います。
※参加者同士、英語で話します。

まず英語で自己紹介を行い、その後簡単なトピックについて参加者同士英語で話します。

※会話の進行役として海外大学卒業生が参加します。

〜参加者・参加人数〜

留学経験が無くても、英語での会話を楽しみたい方ならどなたでも参加OK。

英会話のレベルは関係ありません。一緒に練習しましょう！

1グループ5~10名の少人数制で行います。

〜参加方法〜

〜参加費〜

無料
※ご自身の朝食代のみお支払いください。

〜メッセージ〜

留学経験者の方で、留学先では日本人同士でも英語で話していたという人はたくさんいるのではないでしょうか。
英語上達の為には、そういった心がけが大切だと思います！
また、私はアメリカの大学と院に合わせて５年間留学していましたが、日本に帰国後は英語を話す機会が減り、覚えるよりも何倍もの速さで忘れていっています。

朝食の時間を使って、気軽に英語を話せる機会を設けるために『英語de朝活』を始めます！！！
