description

『未病を改善する』とは…特定の病気になってから治療するのではなく、普段の生活において「心身の状態を整えて、より健康な状態に近づける」という意味があります。今回、季節の旬のお野菜を中心に体に取り入れるものから健康を考えるメニューをご用意致します。

MENU
・有機白ワイン、赤ワイン
（ソルビン酸ｋ・亜硫酸塩、無添加）

・野菜の温野菜サラダ
(有機農業ネットワーク神奈川)

・有機切干し大根、押し大豆の煮物

・湘南の豊かな環境で育った「みやじ豚」のソーセージ

・発芽発酵玄米の地養卵に照り焼き鶏肉をのせた和風オムライス（横浜橋通商店街、（有）大野屋高木商店の、ヨモギと海草を食べて真心こめて作った安心の地養卵を使用）

・季節有機野菜のスープ

・季節の果物ゼリー

・森のコーヒー　or　有機紅茶
（銀座パウリスタの有機珈琲）
