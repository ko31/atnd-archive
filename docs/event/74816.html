内容

初心者から経験者まで、Live2Dに興味がある方で集まって自主勉強会を開催いたします。
初心者の方は公式チュートリアルを、チュートリアルを終えてる方は作りたいものを
「質問や情報交換を交えつつ主に黙々と作業する会」を予定しています。

必要なもの

  • ノートPCもしくはタブレット
    • Live2Dが初めての方・・・無料版をあらかじめ入れてきていただけるとスムーズに作業できます。
    • 経験者の方・・・作りたいデータの、できればパーツ分けまで済ませた状態でお持ちください。
      　（１からパーツ分けすると、それだけでほとんどの時間が終わってしまう可能性があります）

※今回はvol.0ということで、実験的な意味も踏まえ無料開催となります。
今後定期開催を予定しており、次回より会場代（料金未定）が必要になりますのでご了承ください。
