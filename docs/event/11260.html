【講座のポイント】
　プラスチックを難燃化する場合、ハロゲン、リン、水酸化物しか使えなければ、それらの配合を検討する以外に方法が無い。そこで、この３つの方法以外の難燃化方法を２つ提案すると共に、環境問題と難燃という見地から、今後、注目される製品安全性、環境適合性について整理をする。

【プログラム】
１．プラスチック難燃化技術の基礎
　　1.1 プラスチック燃焼の基礎理解

　　1.2 難燃化するには？

２．ハロゲン、リン、無機酸化物、ナノテクによる難燃化の整理
３．燃焼しないプラスチックの構造
　易燃焼プラスチックでも、難燃剤を添加しなくてもまったく燃焼しない領域があること、
　そのような領域を作り出す新しい方法を述べる。

４．微量添加の方法と原理（新しい炭化層形成方法）
　微量添加でＵＬ試験がＶ－０になるケースがあり、その原因が新しい炭化層の形成によることが
　明確になった。　
　その原理を応用してこれまでの添加量を激減させる可能性があるかについて解説を加える。

５．アロイの燃焼、着火と延焼などと製品安全設計
６．新しい方法の失敗と成功　
　 ハロゲンとリン以外に、本当に「難燃化する」という方法はあり得るのか、
　それをナノ粒子を用いた研究の失敗例と、最近の新しい研究を参考にして考える。
　ＵＬ試験のレベルだけに偏した難燃化材料開発に他の面からも考えてみる。

７．逆転の発想・・・高分子材料は可燃物か？
