description

2010年12月24日に、日本でのオープンストリートマップ（OSM）の活動を支援する法人組織として、
一般社団法人オープンストリートマップ・ファウンデーション・ジャパンがOSMコミュニティの中から生まれました。

設立から半年、まだ活動が始まったばかりですが、Yahoo! JAPANからの地図データ寄贈や
2011年3月11日の震災からスタートした sinsai.info 等のクライシスマッピング・プロジェクト等
多くの取り組みを、日本をはじめ世界中のOSMコミュニティのみなさんとすすめることができました。

しかしあくまでOSMFJは、OSMコミュニティを支援する裏方役。
主役は、OSMのユーザとして活動に参加するみなさんです。

このイベントは法人としてのOSMFJのお披露目と、日本で活躍されているOSMユーザのみなさんの顔合わせを狙いとしたもので、
誰でも無料で参加することができます。
もちろん、これからOSMを始めてみようという方も大歓迎です。

共催：(社)OSMFJ x 東京大学CSIS

参加費：無料
