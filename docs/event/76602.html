PSoC組込みボード勉強会 Rev2

イベント ユーザー
event_id 76602
title PSoC組込みボード勉強会 Rev2
catch
description

PSoCボードを使ってプログラミング体験してみるワークショップの2回目です。
もちろん初めての方もOKです。
外付け部品なしでアナログ回路・デジタル回路を組むことが可能です。
PSoCの手軽さと開発のしやすさを体験していただけます。

日時

2016年5月21日（土）10:00～17:00
※途中、昼食タイムと休憩をとります。

場所

ファブラボ北加賀屋（コーポ北加賀屋1F）
大阪市営地下鉄四ツ橋線「北加賀屋駅」4番出口から徒歩10分

料金

お一人500円(ワンコイン)
※料金は当日会場でお支払いください。

必要機材

1.　ノートPC (Windows 7,8.1,10)
2.　PSoC 5LP Prototyping Kitボード
　　※秋月電子参考リンク PSoC 5LP Prototyping Kit
3.　WindowsPCに Cypress PSoC Creator3.3(最新版)を導入
　　※CYPRESSサイト PSoC Creator 3.3 for Windows
　　※レジストレーション(無料)を済ませておいてください。
4.　I2Cキャラクタ液晶モジュール（ACM1602NI-FLW-FBW-M01,AE-AQM0802,SB1602BW等）
5.　温度センサー（LM61CIZ,AM2320,AM2321等）
6.　ブレッドボード（ジャンパ線、その他お手持ちの電子部品をお持ちください）
7.　USB延長ケーブル(必須)
8.　USBマイクロケーブル(必須)
9.　延長コンセントタップ

ワークショップの概要

1　PSoCって何？（他のマイコンとの違い、特徴など）
2　PSoC Creatorの使い方
3　前回のおさらい（Lチカ等）
4　I2C液晶モージュルを使う
5　アナログ入力の仕方
5　お部屋の湿度と温度を測ろう
6　質疑応答
※適宜、休憩をはさみます。
※実施内容は都合により変更になる場合があります、ご了承ください。
