こちらとの２箇所で告知しております。

９月３日（土）にオフ会を開催しようと思います。

同人ゲーム、ゲーム制作に携わる方、
これから作りたいと思っている方々で集まりましょう～。
あまりかたいものというよりゆるっとした会にできればと思います。

最初なのでささやかな会になるかもしれませんが
よろしくお願いいたしますm(_ _)m

ネットワークが広がったり、いろいろなゲームの話ができる会になればと思います。
