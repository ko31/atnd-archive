『小さい夢から始めよう』

title 『小さい夢から始めよう』　
catch 早川周作・出版記念感謝セミナー＆交流会
description

２００９年１２月に発売された
『人生が変わる！「夢実現力」』は、おかげ様をもちまして
紀伊国屋新宿本店・ビジネス書ランキング第１位を獲得することができました！


その勢いが衰えぬまま、いよいよ第２作目となる
『小さい夢から始めよう。』（生産性出版）が
２０１０年２月２７日に発売されました！
前作からの読者の皆様、
日頃よりセミナーや講演会にご参加いただいている皆様、
また、早川の活動にご支援いただき、応援してくださる皆様、
そして、まだ早川のことをよく知らない皆様・・・

すべての方に…

「夢は必ず実現する！」

という思いを届けるため、早川から感謝の気持ちを込め、
更にパワーアップした内容でセミナー＆交流会を開催致します！


縁もコネもカネもない状態から…


これまでどうやって夢を叶え、また、立て続けに出版を重ねる著者となることができたのか？


このセミナーに参加すれば…


様々な逆境をプラスに転じさせ、

日々進化を遂げている早川のパワーを間近で感じられるとともに、
その秘訣を知ることができます！


ひとつひとつ、確実に夢を叶えていく秘訣や、
夢を諦めそうになったときの自分自身との向き合い方、
そして、著書の中では伝えることのできなかった早川自身の体験談など、


夢実現の８つのステップ

を踏まえた盛り沢山の内容でお届けします。



また、前回の出版記念感謝セミナーでも大好評を博した
ご参加の皆様全員による交流会も開催します。


ここでの新しい出会いは、必ず皆様の心強い『夢実現のサポーター』となるでしょう！

今、上昇気流に乗っている早川とともに、「夢・実現」を宣言してみませんか？


今回、ご参加いただいた方全員に、早川が毎日欠かさず書き溜めているものと同様の

「夢実現・シミュレーションシート」をプレゼント致します！


さらに・・・

ご参加者の中から抽選で

・早川が直接指導する「夢・実現」への個人コンサルティング
（通常３万円相当）を３名様へ！


・『２１世紀の幸せ、夢、お金の考え方』
～本田健×澤上篤人×早川周作～
（５，２５０円相当）を３名様へ！


・サイン入り書籍『小さい夢から始めよう。』
（１，３６５円相当）を３名様へ！


当日抽選会開催にて、豪華プレゼントをご用意しています！

夢実現のワンツールを手に入れるチャンスです。
皆様のご参加を心よりお待ちしております。
