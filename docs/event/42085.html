Get Great Git #1 by Qiita

8/27(火)にGitに関する勉強会を開催します．

（※8月13日追記：定員を60名から100名にしました。それに伴い、会場を株式会社ミクシィさんに変更しましたのでご注意ください！）

Qiita内でとても人気のあるGitに関する情報を、リアルの場でもシェアしていただくイベント「Get Great Git #1」 を開催します。

Gitは、より知ったら便利になるけど、その存在をしらなかったり非効率なやり方をしていることに気づいていないことって多いですよね？Qiitaでは皆さんの開発効率向上のために今後、Gitに関するイベントを行っていきます。今回はその第1回目となります。

勉強会のあとは、ビール、飲み物、軽食を用意していますのでGitやプログラミングについて皆さんで楽しくおはなしましょう :)

    • Git+GitHubを使った開発に興味がある方 などなど
