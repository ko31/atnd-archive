description

「色の共通言語」と呼ばれる世界共通のカラーシステムPANTONE®毎年、発表しているカラートレンド。国際カラーデザイン協会では、パントン社より"カラー戦略のスペシャリスト"マリアンを迎え2012年秋冬の流行色を予測。インテリア、ファッション、コスメ、プロダクト、インダストリーの多分野の流行色について詳しく解説いたします。 また、今回特別に一流ブランドのカラー戦略を仕掛けてきたマリアンとのトークセッションも開催。様々なビジネスに活用できるだけでなく、プライベートの色選びにも役立つ最新カラー情報やカラー戦略などの話が無料で聞けます。

＜Profile:マリアン＞
MexxやGap、ジョルジオ・アルマーニによるアルマーニエクスチェンジのスパなどから、キャリアをスタートさせ、さらに、洋服とアイウェアの小売部門において、カジュアル＆ジーンズウェアのCLOSED、ヴィヴィアンウエストウッド、クリツィア、ユナイテッド・カラーズ・オブ・ベネトン、シスレーのアイウェア部門など数多くの有名ブランドを販売するブランドマネジャーとしてキャリアを積む。2003年、パントン社に入社。ファッション、インテリアのカラーシステム分野において、最も重要な役割を担っており、マリアンの豊富な経験は、あらゆる顧客のニーズを理解し、対応していくことを可能とする“カラー戦略のスペシャリスト"。
