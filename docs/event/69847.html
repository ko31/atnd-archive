スマホで簡単！LINEスタンプ講習【無料】

description

紙に描いた絵をLINEスタンプにする無料講習です。

スマートフォンのカメラやアプリを使い、オリジナルキャラやお子さまのラクガキをLINEスタンプに仕上げます!

【イラストが苦手でも・LINEスタンプに興味のある方必見】

初心者からOKの講習会です！
PCのお絵かきソフトも敷居が高い･･････
自分でイラストをうまく描けるかどうか不安、という方でもスマホだけで簡便にLINEスタンプを作る講座をご紹介します。

LINEスタンプに興味はあるけれど、スタンプ化の仕方がわからない方、必見です。

  • オリジナルキャラクターを、LINEスタンプにしたい！
  • 子供の描いた絵をLINEスタンプにしたい！

スタンプ化の作業を0から一緒に行いますので、初心者の方もLINEスタンプが作れるようになりますよ！

【対象者】

既にLINEをご利用中で、ご自分のオリジナルスタンプを作ってみたい方。
無料アプリのダウンロードが可能な方。
スマホの操作ができる方。

【日時】

2015/9/25（金）　10：30～11：30（講習会は45分）

【開催場所】

東京都渋谷区代々木2-4-9　NOF新宿南口ビル4F
新宿駅南口より徒歩３分

地図

【定員】

15名
先着順とさせていただきます。

【内容】

LINEスタンプの制作
手描きしたものを、そのままスタンプとして完成させます。

  • イラスト作成（紙と鉛筆/ペン）
  • スマホ内での画像の加工

【講座】

全4回の講座となっております。
初回のみ無料
2回～4回は有料のご案内となります。
※初回のみの参加可能
講習会ページをご確認下さい

【持ち物】

  • スマートフォン（普段使用しているもの）
  • 筆記用具（鉛筆かシャーペン、消しゴム、サインペンは必須となります）

※スマホのカメラ機能が使用できない場合、受講することはできません。

【講習会開催のいきさつ】

LINEクリエイターズスタンプでは、無料で自分のスタンプを登録したものが、有料で販売できます。

LINEスタンプって、面白いですよね。
コミュニケーションも円滑にできますし、何よりユーモアがありますね。
でも、自分の気に入ったスタンプがなかなか見つからないこともしばしばです。
「こういうのあったらいいな」→「自分で作ってしまえば！？」

お友達に、家族に、自分が作ったスタンプを送れると思うと、ワクワクしませんか？
しかも、販売したスタンプをいろいろな人が使ってくれる･･･それも、嬉しいですよね。

普段スマホを活用されている方であれば、空いた時間にちょこちょこと気軽にLINEスタンプを作ることができることをお伝えしたく、開催の運びとなりました。
ご興味のある方、是非ご参加下さい！

【注意点】

※本講習では「カメラ機能の使用」「アプリのダウンロード」が必要となりますので、ご注意下さい。
※登録にはイラストが40個必要であるため、講習期間中に登録できるわけではありません。
※当講習は、LINEスタンプの作り方をお教えする講習であり、売れるスタンプのノウハウをお伝えする講習ではありません。
　また、LINEの審査に通過することを保障するものでもありません。
