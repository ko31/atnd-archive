イベントの概要

Japarin Xamarin User Group Conferenceは Japan Xamarin User Group 主催の初めてにして最大規模の Xamarin イベントです。
今回は、Xamarin の第一人者の伊勢さん、あめいさんにお話いただきます。日に日に増える、日本のXamarinユーザーが集まり新しい Xamarin を皆で勉強しましょう！その後は Xamarin 3 Celebrate Pary を兼ねて懇親会です！大きなXamarinケーキを食べながら、ドリンクを飲みながら、日本におけるXamarinの夜明けを皆さんと御祝いしたいと思っています。
