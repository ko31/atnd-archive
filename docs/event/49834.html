Ubuntu 14.04 LTSのリリースパーティを兼ねた「オフラインミーティング」イベントです。

前方のセミナー、後方のハッカソン（withよっぱらい）、という雑然とした空気とともに、Ubuntuの魅力とクラウド環境での利用方法をお届けします。

詳細は、 https://wiki.ubuntulinux.jp/Events/OfflineTokyo201405 を参照してください。

  • テーマは『14.04 LTSリリースおめでとう』『クラウド/VPS環境でUbuntuを使おう』です。
  • 開催側の意識合わせのための裏タイトルは 「そろそろ仕事でもUbuntu使ってみるか」です。
  • セミナーに登壇いただくクラウド/VPS事業者の方を募集中です。「クラウド/VPSでUbuntuを使うならウチで！」という事業者の方、がっつりテクニカルな話から「こうやると簡単に使えますよー」という話まで、さまざまなものを歓迎します。
    • 趣旨上、各事業者様の商業的な話題といいますか、宣伝をして頂いて問題のない場です（Ubuntu関連であれば）。

発表予定（順不同、随時追加予定）：

  • Azure概要セッション by 戸倉彩 (またはクラウディア窓辺)
  • Ubuntu on Azureセッション (JAZUG; Japan Windows Azure User Group)
  • ConoHa by GMOインターネット
  • さくらのVPS＆クラウド by さくらインターネット
