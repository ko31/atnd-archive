▼講演概要

１．グロースハックの絶対条件であるプロダクトマーケットフィットとは一体何か？



グロースハックを実際に開始して良い状態と言われているプロダクトマーケットフィット。それって実際どういう状況のことを指しているのかご存じでしょうか？この定義が理解できていないと、どのようなツールを使ったとしてもサービスはなかなかグロースしてくれません。今回はサービスにおける４つのフェーズを使ってプロダクトマーケットフィットをご紹介していただきます。
講演者：株式会社グロースハックスタジオ　取締役　広岡一実 氏


２．黒船＝Facebook広告(Audience Network)襲来！
「アドジェネ」でアプリ向け広告をグロースハック！

　
アプリ内の広告の導入において、広告収益を最大化するために活用されるSSPサービス。
そんなSSPサービスの中で、facebookの広告の配信も可能にしてしまったという、業界でも話題となった交渉を実現した「Ad Generation（アドジェネ）」
アプリ内の広告といえばバナー広告がまだ主流ですが、Facebook広告（Audience Network）によって何が変わるのか、その可能性について豊富な事例を交え、生々しい情報を語っていただきます。
講演者：Supership株式会社 ジ・アドジェネ　池田 寛 氏


３．Push通知改善のイロハ


Pushサービスを提供する会社は何社もありますが、PushのABテストまでできるツール「LogPush」を無料で提供しているpLucky。そんな彼らのPushツールを活用したグロースハックの最新の事例を交えつつ、Push通知改善のイロハをご紹介いたします。
講演者：株式会社pLucky 代表取締役CEO　林 宜宏 氏
