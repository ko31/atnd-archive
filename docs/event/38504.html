■内容

これは、iPhoneアプリを作って楽しむ、ハンズオンの体験型勉強会です。

「Mac」と「楽しむ心」を持っていれば、実は誰でもアプリを作ることができるんです。iPhoneアプリを自分で作る、なんていうと、面白そうだけど、本当に作れるんだろうかって不安な気持ちになったりしますよね。そういうときに大切なことは、まず、小さいものでいいから、作って完成させること！

「ものを完成させたときの感動」というのは、想像以上に作った本人に「夢」と「快感」と「パワー」を与えてくれます。見よう見まねで作ったとしても、作れたときの感動は忘れられないものです。

この勉強会では、プログラムをやったことがなくても楽しくiPhoneアプリを作れることを体験してもらいます。
中高生にもわかるやさしさで、キット感覚に手作りアプリを作れるので、きっとあなたにも作れるはず。
むしろ、私はWEBデザインの仕事をしてるからあんまり関係ないかなー、なんていう人にこそ試しに作ってみて欲しいのです。業務でアプリ作りをするわけではなくても、作れたよ、っていう体験はきっと新しい世界を垣間見せてくれますよ。
