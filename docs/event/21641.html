Windows Phone Arch です。

大阪でWindows Phoneのハッカソンやりますよ！。

ハッカソンとは、ハック＋マラソンの造語で、チームになって当日アプリケーションを開発することです。

当日は興味関心に合わせて、チームを作成し、アプリ開発、プレゼンテーションを行います。
開発者でなくとも、デザイナさんや、こんなアプリがあればいいなというアイデアマンの参加をお待ちしています！
また、初心者グループで、「今日でここまで出来ました！」というのも嬉しいです。

費用：参加費として300円

参加テンプレート(チーム分けなどの参考にさせていただきます。)
・プログラマ寄り or デザイナ寄り
・使用ツール（VS, Blend, Ps, Ai….）
・作りたいアプリ（カメラ系、メディア系、ソーシャル系、ゲーム系、GIS系、etc..）
