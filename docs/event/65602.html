【参加者多数につき増枠！】i-mobileが語る「App×アドネットワーク」のマネタイズモデルはこんなに進化している！！？と題しましてRainbowApps大阪校に初登壇！

■概要

  • Appの過去と現在(カテゴリ／プロモーション／マネタイズ／個人開発者の成功例）
  • i-mobileの広告SDKについて
    　
    など、実際の事例を元にApp業界のプロモーション、マネタイズの話しから、
    広告SDKの導入方法まで、業界の先頭を行くアイモバイル社員が最新情報をお届けします。

■こんな方々におすすめ

「Appの現状ってどうなの？」
「みんなどうやってマネタイズしてるの？」
「プロモーションってどうしてるの？」
「既存Appの収益をもっと増やしたい！」
「個人開発者でも稼げてるの？」

上記のような悩みや、課題のある方におすすめのイベントです。

      現在はAppのマネタイズ担当として活躍する
      広告主／App側両方の知見を兼ね備えた頼れる存在！

2．App業界の流れ～過去と現在～

  • 内容
    • 流行りのカテゴリ、プロモーション、マネタイズ
      App界隈の過去と現在から読み取る、今鉄板の広告マネタイズ方法について
      また、激動の中で成功した個人開発者の例など。
      今後、アプリ開発をしていく方々必見です！
  • 登壇
    • アドネットワーク事業本部

  • 自己紹介
    • 誰よりもAppを遊ぶおてんば娘、
      最新のAppの情報やトレンドを把握して、
      お客様と同じ目線でご収益化をさせて頂きます(ハート)

3．SDKについて

  • 内容
    • Appで収益化をするために必須となったSDKでの広告配信について
      導入方法やメリットなど分りやすくお伝えいたします。
  • 登壇
    • アドネットワーク事業本部
