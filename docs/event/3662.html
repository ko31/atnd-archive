description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。


■現在予定している内容：

　●セッション１(25分)
　『はじめてのUstream(ユーストリーム)』日本Androidの会 中国支部 Ust担当 tkt(高田)さん

　＜内容＞
　誰でも無料で利用できるリアルタイム動画配信サービス、　Ustreamについて
　はじめての方向けに配信方法をデモを交えながら簡単にご紹介します。

　＜高田さんの紹介＞
　日本Androidの会 中国支部、WebTouchMeeting、　hiroshima.rb等の勉強会で
　電源、無線LAN等のインフラ準備をいつの間にか担当。
　本職は某通信会社のISPサーバ担当。
　Ustreamは昨年(2009)12月から始めてまだ4ヶ月程度なのであまり期待しない様に。

　●セッション２(45分)
　『サーバー選定　業界裏事情も踏まえたサーバー選びのツボ』KDDIウェブコミュニケーションズ高畑さん

　＜内容＞
　 何故にサーバー各社によってこんなに値段が違うのか？落ちないサーバーはどこ？
　普段お話できない裏事情も含め、ざっくばらんに暴露致します。自社のPRはしません。
　正しい知識を基にサーバーを選定することで、契約してから後悔することなくウェブサイト運営が可能になります。

　＜高畑さんの紹介＞
　1978年、愛知県生まれ。2001年、Canadian International Collegeを卒業後、
　　　　　ケンウッド・ジオビットに入社し、オーディオアプリのマーケティングと戦略を担当。
　2003年、経済誌専門の広告会社に入社し、同年東京支社長に就任。
　2006年、CPI（現 KDDIウェブコミュニケーションズ）に入社。
　　　　　マーケティングと新規事業立ち上げに従事。
　2007年、中小企業をベースに活動するSMB事業本部長に就任。
　2009年、ドイツ発のWebサイト制作ツール「Jimdo」の日本独占販売権を取得し、
　　　　　同サービスのカントリーマネージャーに就任。

　Tech Crunch寄稿：「あのギーク達がまたやってくる！
　Start up Weekend meets Geeks on a Plane」
　など

　●セッション３(20分)
　『デイリーユーズは便利だな。-よくつかうJS（?）まわりのこと-』aki☆さん

　＜内容＞
　「ちょっとこういうとき便利なのでいつも良く使う」JS（8割JQuery+ライブラリ）まわりの話です。

　JSライブラリは沢山あって、組み合わせてすごいこともできたり、
　クロスブラウザで部分的に補いながらインタラクティブデザインを可能にしたりしますが、
「JQuery」「ライブラリ」の説明とか、「すごいこと」の紹介、

　ではなく、

　「ここちょっと、こうしたい」「ここを少し楽に」という、
　ミニマムニッチなこと。


　普段つかう = よく使う = 便利

　便利 = 楽である = 効率がいい


　じゃないのかなぁと思うので、
　普段使っているJQueryライブラリの紹介や、
　使うときに気をつけていることなどを。

　とってもミニマムな話です。
でも、個人的に、他の人が便利だと思って
よく使うものを知りたいなぁと思うし、

　最近ちょこちょこ、「あ、それもいいね」とゆったりいわれたりすることがあるので、
　この題材にしてみました。

　20分と短いので、さっくりざっくりゆきたいです。

　＜aki☆さんの紹介＞
　CSS Nite in HIROSHIMA　実行委員
　市内のWebシステム会社でWeb屋をしています。
　コーディングとgiraffeとラーメンズとクラムボンがすきです。

　まだまだですが、なまあたたかい目で見守ってください。


　●セッション４(20分)
　『２０分でEmacsの魅力をどれだけ伝えられるか挑戦してみる』えいるさん

　＜内容＞
　UNIX界の２大エディタのひとつ GNU Emacsの紹介を
　短い準備期間と短い発表時間で試みる。

　- Emacsの紹介
　- Emacsの魅力などなど(内容は未定)
　- 時間があればモダンな Emacs の拡張機能を紹介( anything.el, auto-compelet.el など？)

　GNU Emacs – GNU Project – Free Software Foundation (FSF)
　http://www.gnu.org/software/emacs/

　Emacs – Wikipedia
　http://ja.wikipedia.org/wiki/Emacs

　＜えいるさんの紹介＞
　一日の半分をコマンドラインとEmacsで過ごしているらしい。
　Hirshima.rbなるものを立ち上げてRubyを盛り上げる活動をしているけど、あまりうまくいっていない。
　最近はLinuxのカーネルとかHaskellとかiPhoneとか勉強したいらしいけど、時間がとれていない。
　身近にemacs使いがいなくて寂しいらしい。
　dvorak配列使い。
　バンホーテンココアが主食。
