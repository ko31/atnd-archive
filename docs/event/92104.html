福岡市のビズ・コミュニケーション・ライブラリ－ BIZCOLI におきまして、ファイルメーカー Pro 九州ユーザーズミーティング (以下、FM-Kyushu) の勉強会 FM Talkers を開催します。勉強会は、BIZCOLI 会員でない方でも、どなたでも参加できます。

今回は、あまり知られていない FileMaker Pro (ユーザ接続用) のご紹介と、さまざまな API を FileMaker から利用してみようということで、下記の 3 点を予定しています。また、詳細が決まり次第、このサイトに掲載していくように致します。

1. FileMaker Pro (ユーザ接続用) のご紹介
2. 歯科医院向け診療予約 Web サービスが提供する API を利用して FileMaker Go から予約を行うカスタム App の事例紹介 (トラスト・センス株式会社様)
3. 日医標準レセプトソフト ORCA の API を FileMaker から利用してみる

その他、お悩み相談や、自分で作ったソリューション自慢など FileMaker 話で盛り上がりましょう。

・FileMaker ユーザ及び FileMaker にご興味をお持ちの方は是非ご参加ください。
・ソリューションを持ち込まれる方は当日の持ち込みでも構いません。
　ノート PC 等をお持ちいただけると、より盛り上がるかと思います。

■参加申込み方法

ログイン後にページ右側の「このイベントに登録する」ボタンよりお申し込みください。
