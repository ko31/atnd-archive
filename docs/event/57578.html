description

伸び悩んでるシンガー、ボーカリストのために！

「声量を上げたい！」「どうやってトレーニングしていいかわからない」「とにかく歌が上手くなりたい！」など、
シンガーやボーカリストなど声の悩みもっている方へ、現役のプロのボーカリストであり、ボーカルトレーナーでもある梶原茂人さんによる目から鱗の"声"の技術を披露。今より、もう一段階ステップアップする技術をお教えします。

例えば、

  • 声量を上げるコツ
  • 声帯を鍛えるとは
  • 「口を大きく開ける」は正しくない!?
  • 実は腹式呼吸は重要ではない!?
  • 普段、話す時に自然と行うことをやるだけで声の響きが変わる方法
  • 2～30曲歌っても声がれしない法
  • 喉が痛くならない法
  • たった3分のトレーニングで音程が良くなる
    など、

シンガー、ボーカリストの方だけではなく、俳優、声優の方、カラオケがうまくなりたい方など、プロ、アマ問わず、"声"をうまくコントロールする術を知りたい方は、奮ってご参加ください！
「ボーカルトレーニングは必要ない」という方でも話を聞くだけで、きっとタメになりますよ！

講師の梶原茂人さんって、ボイストレーナーとしてはどんな方？

経歴は以下に記載しておりますが、
ボイストレーナーとしては、一声聞いただけで、「少し前まで風邪を引いていた」「お腹がすいている」など、その人の体調を探れるといった能力と、一般的なボイストレーニングとは一線を隔した技術を持っており、真摯かつ誠実にボーカリストの悩みにお答えしてくれます。
