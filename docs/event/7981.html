【講座主旨】
センシング機能の実用化においては、システム要件に伴うセンサ素子の選択のみならず、センシング対象をいかにモデル化するかが重要である。本講演では、「モデル化とは何か?」を基点にし、モデル化の手助けとなる数理的・信号処理的アプローチの紹介を経て、モデル化プロセスの実践例紹介へとつなげる。
　また、モデル化のレベルやプロセスは,たとえセンサ素子が同じであっても、どのような対象をどのような目的でセンシングするのかによって大きく異なる。そこで本講演では、いくつかの代表的なセンサを例にとり、それぞれの適用シーンに応じたモデル化プロセスの違いを、具体的手法やモデル化における最新トレンドを通して紹介する。

1　センシング入門
　1.1　センサとは?センシングとは?
　1.2　センシングにおけるモデル化の役割
　1.3　モデル化における数理的アプローチの重要性
　
2　モデル化入門
　2.1　センサのモデル化とセンシングのモデル化の違い
　2.2　信号処理的立場からみたモデル化手法
　2.3　確率統計論的立場からみたモデル化手法
　
3　モデル化の実例
　3.1　モデル化のトレンド
　3.2　画像センシングにおける実用化例
