セッション詳細


Denali CTP3 AlwaysOn Availability Groups概要
　　SQL Server の次バージョンでは高可用性環境の構成として
　　AlwaysOn Availability Groups という設定が追加されます。
　　7月に提供が開始された SQL Server Code Name “Denali” CTP3 を
　　使用してデモを交えながら AlwaysOn Availability Groups の概要を
　　お話しします。　　[Speaker : SQLTO　小澤 真之]

System Center User Group Japan 発足のお知らせ ～System Center Configuration Manager 2012 新機能概要
　　System Center User Group Japanを発足しました。
　　本セッションでは、立ち上げたばかりのコミュニティのご案内させて頂くとともに、
　　既にBeta版がリリースされているSystem Center 2012から、System Center Configuration Manager 2012の新機能についてお話しさせて頂きます。
　　[Speaker : System Center User Group Japan 齊藤 和雄]

初めてのSQL Azure Federation入門 – クラウドDBの神髄ここにあり
　　クラウドデータベースSQL Azureが真の効力を発揮するのが、SQL Azure Federation。
　　入門編と称して、わかりやすく丁寧にFederationを紹介します。
　　[Speaker : SQLTO 大和屋 貴仁]

SCVMM も 2012 がやってくる！
　　(作成中)
　　[Speaker : System Center User Group Japan 杵島 正和]

SQL Server ユーザーも System Center を使ってみよう！
　　SQL Server ユーザーも System Centerを使ってみましょう！
　　今回は System Center Orchestrator , System Center Data Protection Manager ,
　　System Center Advisor の3つの製品をご紹介します。
　　[Speaker : 日本マイクロソフト 高添 修 , SQLTO 小塚 大介]
