やきに駆動 に向けてダイエットしたい人が集まってみんなで競い合い、助け合い、支え合い、時に貶し合い、足の引っ張り合いをしながらダイエットをしましょう。

ということで、みんなのダイエット促進のためにレースをしましょう。

細かいルール案はコメント欄に書き込んで下さい。
ざっくりとした、ルール案は以下のとおり

ルール

  • やきに駆動までの目標を宣言
    • 絶対値でも相対値でもOK
    • 例1:絶対値ver) やきに駆動までに65kgまで減量する
    • 例2:相対値ver) やきに駆動までに-5kg減量する
  • 毎日進捗をつぶやく
    • これも相対値でも絶対値でもOK
  • 罰ゲーム
    • 目標達成できたなかった人で目標達成した人におごる
    • 当日体重計の上に土下座して、その様子をtwitterにうｐ(自分の体重表示がほんとに目の前にくるorz)

とりあえず、参加者希望だけど、参加者以外からのルール案も歓迎！

レース開始のメドは、やきに駆動の丁度1ヶ月まえの9/27を想定しています。
