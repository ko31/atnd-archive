スペシャルゲスト「AWS新着任日本人スタッフ小島さん」を囲む会
（参加登録は先着順です。補欠となった方は入場できません。ご了承下さい）

先日、11月12日に米Amazon Web Servicesから以下の発表が
ありました。
「2010年上半期にはシンガポール、下半期にはアジアのほかの
場所にもデータセンターの拠点（Availability Zones）を構える。」

このタイミングで、皆さん待望？の「AWS日本人スタッフ小島さん」が着任しました。

なんか、わくわくしますね。何が始まるんでしょう。
今回、渋谷のAmazon のオフィスをお借りしてAmazon Web Services
についての意見交換の場を設定しました。
小島さんは、マーケティング的なご担当なので、今回は、特に市場動向や、営業、マーケティング系の情報交換をしたいと思っております。

開催概要

  • 日時：　2009年12月25日（金）19:00開場、19:30開始～21:30終了
  • 開場：　アマゾン ジャパン株式会社

  • 定員：　40名
  • 費用：　無料
  • 主催：　Amazon EC2ユーザ会（担当　学びing株式会社：五十嵐）
