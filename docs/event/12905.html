description

ハレ着で梅とレトロ建築！！

６回目の「てくてく着物」は、初めてのドレスコード！
普段あまり着る機会の無い“ハレ着”で、初春の香りの梅と、昭和初期の素敵な会館を観て巡ります。
見学のあとは、カフェでまったりお茶タイムも♪

日時・場所

２月２６日（土）　１２：３０ｐｍに 天満宮 境内へ集合。
天満宮にて梅を観賞後、公共交通機関で移動。
１４：３０～ 綿業会館 を見学。（※綿業会館見学からご参加の方は、１４：２０迄に現地へご集合下さい！）
移動後、カフェでまったり。

参加費

参加費は無料。
但し、各交通費・綿業会館見学料（１人￥５００、高校生以下無料）・飲食代は各自ご負担下さい。

ルール

前述していますが、今回は初のドレスコードのイベントです。
是非、レトロ建築にお似合いのハレ着でご参加下さい^^
盛装やパーティ着の感覚、ご自分が「晴れやかな気分になる」お着物でどうぞ！
勿論、正装でも♪
また、いつも通り洋装でのご参加も勿論ＯＫ！（今回は洋装でもお出掛け着がお勧めかも。でもカジュアルなお洋服でも勿論ＯＫ^^→写真を撮る時などに浮いちゃうかも？という程度の理由ですので^^）

補足

普段の「てくてく着物」は、申し込み不要のイベントなのですが、今回は見学に定員枠がある為、事前にお申し込みが必要です。
お申し込みは当ページでお願い致します。
また、小さなお子様についてですが、（会館に問い合わせたところ）２歳児くらいまでは人数に含まれない感じです。ですので、３歳以上のお子様については念の為、大人と同様に当ページでお申し込み下さい。
また、天満宮のみご参加の場合、お申し込みは不要です。

お申し込み時の当ページの操作がわからない場合は、きもの結び contact@kimonomusubi.com までお問い合わせ下さい。
※参加お申し込み後にキャンセル、変更される方は、なるべくお早めに上記までお知らせ下さい。
