description

チョーアナログ営業＆アナログコンサルタントであった森岡氏。１日１社が限界だったアナログコンサル手法に限界を感じて、デジタル化を図り、受注キャパを50社に増やす。アナログであれば、６人体制でする業務量を、チャット・スカイプ・google appsをフル活用して、社長とパート２名体制で業務をこなす。当然、それに伴いクライアントの業務効率も鰻のぼりに改善。今回は、このアナログからデジタルコンサルスタイルへの変貌するビフォアー、アフタァーのノウハウの全貌を明らかにしていきます。

こんな方にお薦め

→企業の教務効率を提案するコンサルタントの方、または、それに順する方
→業務効率化して、業績を上げたい企業の社長や役員の方
→各種システムを導入したが、イマイチ業務効率ができない企業の方

森岡 寛（株式会社M’s FACTORY 代表取締役）
