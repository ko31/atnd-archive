目的

サービスデスク、サポートデスク、カスタマーサポートなどの業務に特化した
オープンソースのOTRS（Opensource Ticket Request System）を知るため／研究するための勉強会です。

また、海外ではAsteriskを始めとしたIPテレフォニーと組み合わせて
コールセンター向けのソリューションとしても広く使われており
回を重ねるごとに事例や使い方など様々な角度から勉強していきたいと思います。

OTRSはITSMを運用するための機能もありITシステムの運用管理も行えます。

主催

日本OTRSユーザ会（OTRS調査兵団）
