主催：iPhone-dev.jp

<セッション１>
発表者：(株)FEYNMAN 川畑（ku_suke）
テーマ：発売時と発売後のDL数の伸ばし方講座。弊社の携わったアプリを実例に、アプリのDL数を伸ばすには何をすればいいかを解説します。生々しいデータをお見せできるかもしれません。
★スペシャルゲストより動画メッセージを頂きました。

<セッション2>
発表者：リクルートMTL
テーマ：調整中：ホットペッパーアプリをiPhone/iPad/Androidの３媒体でリリースされたご経験から、それぞれのストアの感触の違いなどをお話しいただく予定です

<セッション3>
発表者：ダイヤモンド社 加藤様 常盤様
テーマ：DReader成功のカギはエンジニアとのコラボレーション
「適当日記」「もしドラ」などランキングトップに君臨するヒット作連発の電子書籍アプリケーションエンジン、DReaderの開発秘話をお話し頂きます。


<セッション4>
発表者：株式会社ノボット
テーマ：調整中：海外で活躍されている個人開発者の方にご登壇頂きます。
