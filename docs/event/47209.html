description

名称は映画好きな人、ライターですが
航空機が出てくる映画が好き、映画「プレーンズ」「カーズ」が好きな人
から戦闘機、零戦が好きだと言う人でもかまいません
ライター、物書き、ＳＯＨＯライター、自称作家、電子書籍出している人
交流できればと思います

流れとしては自分の好きな映画や自分が書いた作品を持ち寄ったりという
フリートークとなっています

日時　　２月１４日　　１９：３０～２１：００

場所　　新宿エルタワー　サイゼリア内

　　　　新宿駅西口側、小田急ハルクそば

参加費　　飲食費のみ

こんな人にオススメ

・初参加である
・自分はライター（もの書き）だ
・将来は作家を目指している
・ランサーズやグランドワークスでチームを作りたい
・映画は好きだ
・映画は最新作が来ると見に行く
・自称シナリオ、脚本家、作家だ
・趣味で絵、小説を書いている
・航空機、鉄道が出てくる映画が好き
・カーズ、プレーンズファンだ

当日の流れ

〇自己紹介

〇自分のプロット、作品（あらすじ、箇条書きのもの）見せ合う

〇フリートーク

〇この交流会に来てよかったと解散

参加表明はコメントかメッセージをお願いします

当日のキャンセルはご遠慮ください

禁止事項　参加される方はよく読んでからお願いします

営業、販促、宗教勧誘、情報交換目的の方の参加は固く断りします
勧誘行為とは営業、宗教、営業契約 、また付随する商品の販売、保持
情報交換などが対象となります。
また婚活、ナンパ目的の参加もお断りします
その他不快な行為。例えば騒ぐや誹謗中傷などです
これらが判明しだい、カフェ会から退場して頂きます。

主催者は不快な思いをしていますので退場していただきます

初めての方、初心者の方でも参加しやすいようにアットホームな雰囲気で行いますので気軽
にご参加ください
