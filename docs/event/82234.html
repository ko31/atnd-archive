description

男性に質問します。

「私と仕事どっちが大事なの！？」

とパートナーや恋人に言われた場合、あなたは本当の意味を理解してますか？
女性に質問します。

あなたのパートナーや恋人はこの言葉を理解できると思いますか？

恐らく多くの方が理解できない、もしくは理解しているつもりになっていることでしょう。

実はこの何気ない問いかけに大きな問題が潜んでいます。

いわば、ほとんどの方が、今回のパッション・イン・ラブの対象者となるでしょう。

あなたのパートナーや恋人と良縁と感じていますか？

現在パートナーはいないが良縁なパートナーが欲しいと望んでますか？

じつはパートナーとの関係性は幸福度やビジネスで非常に重要な位置を占めることを知ってましたか？

よくビジネスの現場で経営者と会話していると「パートナーがいると仕事の妨げになるからいらない！」と言ってる人がいます。

特に女性の活躍されている方から多く聞く機会があります。

果たしてそれでいいのでしょうか？

世界のビッグビジネスを手掛けているビジネスマンは必ずパートナーを大切にしています。

世界ナンバー１コーチと名高いアンソニーロビンズは「パートナーがいないだけで人生は欠落している」とまで言っています。

じゃあパートナーがいないと幸せや成功は無理なの？と思う方も見えるでしょう。

もちろんパートナーがいない状態でもビジネスの成功はあると思います。

しかし、なぜかほとんどが短期的なものになってしまうのです。

経営者だけでなく、サラリーマンの方もパフォーマンスに大きく関わってきます。

あなたはパートナーと本当に良縁でしょうか？

「うちはパートナーと意外にうまくいってるよ。」

こういう方もいると思います。

一見うまくいっているように感じるのですが本当にそうでしょうか？

ここで注目したいのが「本当の良縁か？」ということです。

よくあるのが、男女互いが本当の自分から逆転してしまっているから合ってるように感じてしまっていることが多いのです。

この状態が続くとお互い情熱がどんどん無くなっていき、一緒にいることが出来なくなっていきます。

あなたは男性と女性を本当に理解していますか？

男女、学名は同じサル目ヒトです。

しかし、男女で全く違う考え方や行動パターンを持っています。

この違うパターンに気づいていないと誤解が誤解を招きどんどん泥沼にハマっていくのです。

誤解のまま突き進む状態と、分かり合えた状態で進む違いがどれほど違うか想像できるでしょうか？恐らく180度進む方向が違うといってもいいでしょう。

人生の最大化を手に入れたい方はパートナーとの関係性を是非見直してください。

パートナーとの関係性が幸せな人生を掴むのに必要とお伝えできるのはパッション・イン・ラブだけでしょう。

また、これから素敵なパートナーを見つけようとしている方は必ずパッション・イン・ラブを学んでください。

自分らしさ、男性、女性らしさを取り戻すと出会う人が変わります。

それは理想のパートナーになることでしょう。

仕事で上司、部下で異性は絶対関わりがあります。

その時に男性、女性が解っていればどれほど組織が円滑に動くことでしょうか。

理想の職場環境を望む経営者は必見です。

なかなか男性女性のコミュニケーションがうまくいかない方はもちろん、いつも割とうまく立ち回れる人は特に聞いてみてください。

危うい関係性であることに気付けることでしょう。

今までなかなか明かされなかった男女の関係性からくる人生の最大化を自己投資額2000万円を越え、なおも学び続けている後藤氏が2時間セミナーで伝えきれなかった部分を1日じっくり時間をかけお伝えします。

学びは1回でなかなか習得できるものではありません。

そのため再受講が非常にお安くなっているのがこのセミナーの特徴です。

最後まで付き合う後藤氏の覚悟が感じられます。

そして彼は一つの学びにとらわれず、様々な学びの中から自分でためし効果のある物をチョイスすることで個人コーチングやセミナーで多くのクライアントを変化させてきました。

今回その能力を余すことなく発揮し1日しっかり時間をかけ、名古屋で行います。

非常に濃い1日になると思います。

・パートナーの行動、言動が理解できない。

・パートナーと触れ合っているがイマイチ喜んでくれない。

・異性と話がどうもかみ合わない。

・当たり前のことを言ったつもりがパートナーを傷付けてしまった。

・パートナーにプレゼントを渡しても喜んでくれない。

・家事など手伝っているのに奥さんがいつも不機嫌。

身に覚えのあるあなた！お楽しみに！
