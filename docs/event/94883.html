【SKYARC】セキュアなCMSへの乗り換え相談会

description

最近、「WordPress」をはじめとしたオープンソースのCMSで作成されたウェブサイトのセキュリティ事故に伴う、CMSやサーバーの乗り換えのご相談を多くいただいております。そのため、脆弱性診断の実施や、CMSやサーバーの乗り換えをご検討されている方々を対象に『セキュアなCMSへの乗り換え相談会』を開催します。

相談会では、CMSやサーバーの乗り換えに伴う注意点はもちろんのこと、金融機関や大手企業の情報セキュリティご担当者様にもご納得いただける攻めのセキュリティ対策について余すことなく解説します。

実際にインシデントが発生してしまったお客様に限らず、会社のポリシー変更によりオープンソースのCMSを利用できなくなった企業様のご担当社様、セキュリティが心配だが乗り換え作業が大変そうなため二の足を踏まれているご担当者様、ひとまずは現行ウェブサイトの脆弱性診断をしてみたい方、この機会に是非ともご参加ください。

相談会概要

開催日：2018年3月15日（木）15:30 - 17:30（受付15:15〜）

定員：先着10名 

会場：〒108-0071 東京都港区白金台2-26-10 スカイアーク東京 セミナールーム

　　　（都営浅草線 高輪台駅 A2出口 徒歩1分／JR 品川駅 徒歩10分）

受講料：無料・完全予約制

受講料：無料・完全予約制

セッション内容

1）CMSやサーバーを乗り換える際の注意点

2）CMSやサーバーに関する攻めのセキュリティ対策

3）情報セキュリティのご担当者様を納得させる上申方法

4）セキュリティに特化したCMS「MTCMS」のご紹介

こんな方にオススメです

・日々の脆弱性対応に追われ業務に支障が出ているご担当者様

・フリーのCMSの利用が禁じられている企業のご担当者様

・ウェブサイトの運用を任されているシステムご担当社様

応募方法

こちらのフォームからお申し込みください。

みな様のご応募を楽しみにお待ちしております。
