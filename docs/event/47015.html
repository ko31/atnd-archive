今後どんどん広がっていくと思われるAWS(Amazon Web Services)について！
AWSでアプリケーションサーバを構築したいと思っているけど、こんな事ありませんか？
・インストール作業が手間なので、最初からアプリケーションがインストールされているサーバが欲しい。
・GitやRedmineなど興味のあるツールをとりあえず手軽に試してみたい。
・社内向けに説明や評価するために使ってみたい。
・AWS無料枠を使ってローコストで運用したい。

今回は、インストール作業無しですぐ使えるアプリケーションサーバについて、
3つの構築事例を紹介しながら講演者とともにハンズオンで体験いただきます。
