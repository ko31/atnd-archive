夢や目標を口にすると、どんどん実現する!

「私、これやります!!」
一度口に出した事で
やる気が湧いた経験ありませんか?

「やろうか、やめようか迷うな…」
そんな時も、やると宣言する事で
迷いが吹っ切れた経験もあると思います。

品川コミットメント朝活は、

朝の異業種交流タイムの中で、
一週間後までに達成したい
目標についてコミットしたり、
前週のコミットメントについて
報告し合う朝活です。

あなたの背中を押してくれる
仲間が朝活で待ってます!!

品川コミットメント朝活・詳細

◆ 日程: 毎週水曜
◆ 時間: 7:00～8:30（途中退出OK）
◆ 定員: 15名
　↑
( 20代〜30代男女、夢や目標がある
方が参加しています。
気持ちは20代の方も大歓迎です❗️ )
