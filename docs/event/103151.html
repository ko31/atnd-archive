過去2回開催してきた「WEBメディア関係者・マーケター向けデジタルナイト」ですが、第3回を開催いたします。

本イベントを通して、メディアやマーケティングに関わる方々に情報共有を通して、業界に貢献していければと思います。

申し込みはこちらからお願いいたします。 https://media-incubate.peatix.com/view

【第一部】
日本人クリエイターの中国進出支援を行うVstar Japan株式会社の番匠達也社長より「中国KOL市場と施策事例のご紹介」を講演いただきます。

具体的には、
①中国KOL市場の最新情報
②中国で人気のある日本関連コンテンツ
③中国KOLプロモーションを行う上での注意点
④中国KOLプロモーション事例について
についてお話いただきます。

次に、メディア特化のインフルエンサーの組織化＆事業化支援も行っている他、ワイン・LGBTQ・子どもをテーマに事業展開を行う株式会社メディアインキュベート 副社長 / 株式会社Omnibus 代表取締役社長の鬼澤絵美氏より「インフルエンサー業界の裏側。そして、未来とは」 を講演いただきます。

具体的には、
①インフルエンサー業界構造と実状
②インフルエンサーについて
③事例から見るインフルエンサーマーケティングの失敗と成功
④インフルエンサー業界の未来について
⑤これからインフルエンサー施策や事業の取り組みを始めたい方へ
についてお話いただきます。

【第二部】
第一部の講師に加え、
朝日新聞社の人気サイト 「withnews」編集長・奥山晶二郎氏と、
広告主・媒体向け営業や大手インフルエンサー事務所の営業TOPを経て、タレントの育成支援なども行う杉村 侑大 氏の4名で、

「日中におけるインフルエンサーマーケティング とWEBメディア」をテーマに

・インフルエンサー市場のこれまで。そして今後どうなる？Tik Tokの未来的可能性はある？

・失敗しない為に知っておくべき！失敗するインフルエンサーマーケティング とは？
・今後日本でも増えるであろうSNSダイレクトショッピング

・WEBメディアとインフルエンサーの理想的関係性は？

などについて、

日本と中国それぞれの経済、流通事情の違いも踏まえて

実際に、インフルエンサーマーケティング 最前線にいた4者によるトークセッションで

WEBメディアとインフルエンサーマーケティング の可能性を探っていきます。

セミナー後には交流会を開催します。

・日時：2019年1月31日（木）19時(18時半受付開始)～21時 終了後懇親会（終了22時）

・場所：朝日新聞社メディアラボ渋谷分室（渋谷駅から原宿方向へ徒歩５分：渋谷区神宮前6-19-21 BCG神宮前PROPERTY4F）

・対象：webライター・編集者・マーケター・広告代理店営業の方など（定員50名）

・参加費：3,500円（交流会含む）
