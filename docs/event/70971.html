長野県にUIJターンした女史としたい女史に向けた講演会・トーク&交流会

鮭女の会とは？

鮭女とは、鮭の遡上になぞらえて、就職や進学で一度都会に出てから地元に戻ってきた女史のこと。
鮭女になりたい方や先輩鮭女が集まり、仕事のこと、暮らしのこと、Uターンのことなどを語り合うのが「鮭女の会」です。鮭女への一歩をなかなか踏み出せなかった方、鮭女としてもっと地元で楽しみたい方など、情報交換をしつつ鮭女仲間を増やしましょう。

こんな方はぜひご参加ください！

  • 何年か都会で修行したら地元に戻ろうと思ってる女性
  • もう既に信州にカムバックしている女性
  • Uターンについて全く考えた事も無かったけど、ちょっと話を聞いてみたい女性
  • 信州に限らず自分の地元に戻りたい
