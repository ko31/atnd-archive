description

１年後も予測できない未来。今もっとも必要とされている
「アイデアが出る仕組み～出す方法」
をたった３時間で体験ができる。

５年後も１０年後もアイデアに困らない技術を手に入れませんか？

・もっとこんな風にしたら面白くない？
・これを組み合わせたらいいかも。
・こんなことができたらいいいな。
・こんなことを知りたい、やりたい！

等を話し合い、そこから
「こんなサービスがあったらいいよね」を皆で一緒にアイデア出しをしていきます。

・将来起業したい人が、起業アイデアを見つける場。
・中々行動できない人が、一歩踏み出せる場。
・イイとこないかな、イイとこないかなと伸ばす場。

「アイデアは、みんなで、考えよう！」

これから何かをはじめるスタート時の方は１人で考える環境って多いと思います。
中々周りに相談したり、他の方の意見を聞ける環境を作れている人
は少ないかと思います。

当然、アイデアを１人で考えても、どうしても発想にも限界があるわけです。

このワークショップでは毎回色々なテーマでアイデア出しを実施しています。

どんなプロのスポーツ選手も日頃の練習は欠かせません。
起業家も同じです。日頃から鍛えていなければなりません。

毎回参加しアイデアの量を出せば出すほど組み合わせのパターンも
多くなってきて、そこから質の高いアイデアが生まれやすくなり
他とは違ったアイデアが出てくるようになります。

※ブレインストーミングとは？
複数人でアイデアを出し合うことにより
相互交錯の連鎖反応や発想の誘発を期待するのが目的です。

簡単に言うと、
「１人ではアイデアとか発想に限界があるからみんなでいろんなアイデアを
自由に出し合っておもしろくて実現できそうなものを作りだす」
ということです。

ワークワークショップを通じて生み出されたさまざまなことが
あなたの行動や起業に繋がっていく。
そんな「場」を参加者の皆さまと一緒に
つくることができたらいいなと思っています。

まずは気軽に楽しく、ご参加ください。

=========
■参加する３つのメリット：

① 独自性の高いアイデアがだせるようになる。
毎回参加しアイデアの量を出せば出すほど組み合わせのパターンも
多くなってきて、そこから質の高いアイデアが生まれやすくなり
他とは違った独自のアイデアが出てくる脳になってきます。

② コミュニケーション力が鍛えられる
色々業界の人たちが集まってチームになりアイデア出しをします。
まったく違う業界の人の発想は、自分では思いつかないアイデアが出てきます。

③ 自分の起業アイデアの新しいヒントが得られる。
いいアイデアが出たら起業しちゃいましょう。

■対象：
・これから起業したい方
・起業には興味あるが、何をやったらいいか迷ってる人
・起業のテーマを見つけた人

など、起業に興味がある人であれば誰でも参加OKです。

■【期待効果】
・いろいろな角度から自分では思いつかないアイデアが出てくる

・自分の起業アイデアの新しいヒントが得られる。
