description

普段、友達や知人など、色々な人と会話をします。
通常の会話だと楽しく話せても、会社の同僚や部下・上司が相手だと、どうも会話がぎこちなくなってしまったり、言いたいことをうまく伝えられないことがあります。
プライベートでは、苦手な相手と距離を置くことが出来ても、会社の同僚が相手だと、距離を置くのもなかなか難しい場合が多くあります。
職場ならではの、コミュニケーションを上手にとる技法がたくさんあります。
コーチング＆コミュニケーション学から「職場のコミュニケーション向上技法」を習得し、職場内でのコミュニケーションを明るく楽しいものにしませんか♪

～セミナーの内容～

・職場のコミュニケーションが良いとこんなに得をする
・相手に何でも話してもらうための「聴き方」
・自分の思いを理解してもらえる「伝え方」
・歳の離れた同僚とのコミュニケーション
・苦手な同僚とのコミュニケーション

～このセミナーはこんな方におすすめです～

・職場や団体での人間関係を良いものにしたい
・苦手と思っていた上司や部下との関係を良くしたい
・相手を不快にさせずに断ることが出来る話術を身に付けたい
・世代の違う相手とスムーズに話したい
・相手がたくさん話したくなる話術を身に付けたい
・部下や上司に気に入られる会話をしたい
・嫌いな相手の良いところに気付くようになりたい
・コミュニケーション力を身につけたい

etc…

《講師》
エグゼクティブコーチ／心理カウンセラー　上前拓也

社会福祉法人を母体とする北海道内大手企業に１０年以上勤務する傍ら、２００６年よりインターネットを中心とした心理カウンセリングとコーチング、ライブトーク活動などを行なう。
独立前に携わった営業職では、コーチング手法を用いた営業方法により12ヶ月連続新規開拓トップ、過去に例を見ない成績を残す。（年間新規契約数450軒）
２００９年に起業し、北海道内を中心に企業向けのコーチング研修や講演、パーソナルコーチング、コミュニケーション講師など幅広く活動を行なっている。
講座でのモットーは「頭で分かりやすく・心に入りやすく」。終始和やかなムードで行なわれる講座だが、参加者のハートをしっかり掴む。
