初心者向けのASP.NET勉強会を大田区 池上会館で開催します。

ASP.NETはMicrosoftのWebアプリケーション開発環境です。
Webアプリケーションを作ってみたいと考えている方向けに初心者でも分かるように
分かりやすくご説明します！

勉強会の内容はVisual Studio Web Developer（フリーの開発環境）を使って簡単なWebアプリケーションの作成を行います。データベースにはSQL Server Express Editionを使用します。
人が集まれば定期的な実施も考えておりますのでよろしくお願いします。

  • プログラミング未経験者
  • Webプログラムの基礎知識を知りたい方
  • PHP、Javaは分かるが.NET技術に興味のある方
