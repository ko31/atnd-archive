見える化でわかる開発段階の製品原価管理

イベント ユーザー
event_id 16738
title 見える化でわかる開発段階の製品原価管理
catch ～高付加価値製品の創出ノウハウを提案します～
description

【習得できる知識】
・付加価値の考え方から顧客要求の掴み方
・目標原価の設定手順
・製品の適性機能・品質追究と原価の作り込みのポイント
・機能と品質を作り込んだ後のコストダウンの実践方策
・製品原価管理の具体的な進め方と実践例　など

【講座のポイント・趣旨】
　付加価値のある製品を生み出すには、顧客要求の絞り込みから始まる製品の開発段階における製品原価管理の役割が重要です。そのキーポイントは、開発段階の製品原価管理の見える化と図面段階での機能・品質と原価の作り込みにあります。

　このセミナーでは、製品原価管理の基礎知識と具体的な進め方について、わかりやすく実例を交えながら解説します。セミナーにご参加頂き、顧客の価値を向上させた高付加価値製品が生まれてくれば幸いです。

【プログラム】
１．製品の付加価値とライフサイクルコストを見える化する
　　1-1　付加価値の高い製品を開発する
　　1-2　顧客が価値を感じる製品とは
　　1-3　高付加価値製品の設計・開発手順を標準化する
　　1-4　製品原価はライフサイクルで考える
　　1-5　どれくらいライフサイクルコストは低減できるか
２．原価企画で出図前の原価を管理する
　　2-1　目標売価から原価を作り込む原価企画の考え方と進め方
　　2-2　売価と利益を決定する要素を見える化する
　　2-3　原価企画で必要な見積原価のレベルとは
　　2-4　目標利益を満たし達成可能な製品の目標原価を設定する
　　2-5　目標原価をユニットや部品に割り付ける
３．製品の適正機能・適性品質を追究して原価を作り込む
　　3-1　製品機能を原価で見える化する
　　3-2　価値ある機能にお金をかける
　　3-3　固定概念を打破してアイデア発想する
　　3-4　品質を上げるとコストアップになる誤解
　　3-5　適正品質追究の進め方
４．製品のばらつきを低減すると原価は激減する
　　4-1　コストダウン施策を見える化する
　　4-2　製品設計段階でどれくらい原価コストが下がるかが分かる
　　4-3　材料費に含まれる基本機能と補助機能のロス
　　4-4　1級品と2級品の適用を原価で判定する
　　4-5　原価を考慮して図面交差を決定する
　　4-6　最適なコストダウン技術で目標原価を達成する
５．開発段階の製品原価管理の実践例を見える化する
　　5-1　究極の製品設計は試作レス
　　5-2　ＣＡｘ（Ｄ、Ｅ、Ｔ、Ｍ）で原価を作り込む
　　5-3　最適モデルを試作して手戻りをゼロにする
　　5-4　ＣＡＤ見積で最適コストを作り込む
　　5-5　設計・開発リードタイムも短縮する
