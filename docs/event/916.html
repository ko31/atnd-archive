description

「iPhone OS 3.0をごにょごにょする会」は@ku_sukeと@sumihiroのチャットからスタートした企画、堅苦しい勉強会形式じゃなくて、ゆるーくオフ会形式がいいよね！というので始まりました。
今回はその第３回（いや２回？）@sumihiro多忙のゆえ@m_hashが幹事で進める事になりました。
あの懐かしの303号室はありませんが、気楽にわいわいやろうぜ！という主旨は変わりません。
という事でみなさんご参加そして色々と発表などよろしくお願いします。
なお、勉強会のあと懇親会・座談会になだれこむ予定です。

肝心な勉強の中身ですが「GameKitのデモ」「MapKitのデモ」「APNSのデモ」などが出て来る予定です。


集合は大平ビル２階のタンデムシステムズで、会場はキネトスコープさんから３階をお借りしています。
設備はこんな感じ。
電源タップ、無線LAN（有線なし）、発表用に30インチ液晶モニタ、灰皿はたぶんあり、トイレ・給湯室有り、電気ポット・
電子レンジ・小さい冷蔵庫は２階にあり、近所にセブンイレブン
最寄りの駅は地下鉄南森町・北浜・天満橋のどこからでも７〜８分


懇親会は近所の雲州堂さんを予定、どうも当日お芝居があるそうで早めの予約が必要な感じです。参加の方はコメントに記入しておいてください。

勉強会そのものはもちろん無料でみんなでカンパしながらおやつでも買いに行きましょう。懇親会は割り勘でよろしく。

当日の予定を追記します。
会場には１２時から入れる様にしますが集合は１時、頃合いをみて２時までには始めようと思います。
懇親会がお店の関係で６時からになりますので、勉強会は５時頃に終わる予定。途中に休憩も適当に入ると思います。
その後は適当に歓談して過ごして下さい。

重要
コメント欄にもありますように、椅子が全然足りません。で、処分しようとしていた本を束ねておきますので、それをご利用下さい。
床に直の方には段ボールを支給します。ご協力のほど、よろしくお願いします。

懇親会情報
会場は雲州堂さんで、６時から。
2500円のコースを頼んでおきました。プラス1500円でフリードリンクにできますが、これは当日に決めようと思います。
