扇町ビジネスサロンはコワーキング・レンタルスペースのROUGH LABO TECHとセカンドチャンスが共同で行う、集まり・イベントです。「フリーランスや中小企業がより稼げるようになる」ため、役立つセミナーやイベントなどを開催してまいります。

今回は初のイベントということで、ざっくばらんな「たこ焼きパーティー（タコパ）」を開催したいと思います。参加者の方とたこ焼きを焼きつつ、ドリンクを飲みながら楽しく語らうパーティーです。そこで仲良くなった方と仕事をしてもいいですし、相談をすることもできるでしょう！

フリーランスの方や中小企業の方、制作パートナーを探している方、日頃の愚痴を発散したい方、どんな方でもご参加可能です！参加していただく方は入退場自由です。いつ来ていただいてもいつ帰っていただいても構いません！奮ってご参加くださいませ。

  • 日時：2019年11月22日（金）19時～21時
  • 場所：ROUGH LABO TECH扇町
    （大阪市北区末広町３−21　扇町センタービル７階707 ）
  • 参加費：1,000円（受付にてお名刺1枚頂戴いたします）
  • 対象：フリーランス・個人事業主の方
    中小企業の代表者様
    企業に勤めている方
  • お申し込み：イベントページご確認の上、お申込みください。
