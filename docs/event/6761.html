過去の開催履歴とこれからの開催予定はこちら


Webプログラミングの基礎の基礎の勉強会を行います

PHPのプログラムの書き方や、フォームの使い方を重点的に勉強します

PHP 『 超 』 初心者向け勉強会です。

難易度は『全く難しくない』です。

対象者は『プログラム未経験者』です。

PHP初心者勉強会とPHP中級者勉強会は定期的に開催しています。


定期的に開催しているPHP初心者勉強会は、PHPは未経験でも他のプログラムの経験がある人を対象としています。（C言語、JAVA、Perlなど）

そのため、初めてプログラムを学習する方は、内容を理解することができませんでした。

PHP 『 超 』 初心者勉強会は、プログラムが初めての方を対象にしています。

勉強会の進め方は、プロジェクターに映した画面を見ながら、一緒に操作を行って進めて行きます。


内容

  1. Webプログラムについて
  2. PHPについて
  3. PHPの書き方
  4. 文字について
  5. 変数について
  6. 配列と連想配列について
  7. フォームのプログラム
