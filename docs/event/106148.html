description

　自動車雑誌、カタログ等で数多くの実績を持つマルチカメラマン庄野正弘氏によるフォト＆ムービーセッション！
　箱根の自然を背景にした愛車のプロモーションビデオを制作します。

　複数台のカメラを駆使して様々な角度の走行シーン、
　ドローンで人間の目線では見られない空からの姿、
　一生に残る愛車とオーナーとの姿を捉えます。

　プロモーションビデオのほか、当日に撮影したすべての画像と動画をご提供。
　他の撮影業者のような1枚単位の販売形態とは異なり、撮影からデータ（DVD、ブルーレイ、web便をお選びいただけます）のお渡しまで追加料金はありません。
　
　愛車とのメモリアルに、
　愛車と家族の思い出に、
　カスタムカーなど…
　
　撮影した画像や動画は、ホームページやブログ、SNS、販促など様々な用途に自由にご利用いただけます。
　メイクアップ、ランチ、ティータイムは無料でご提供致します。
　お申込み先着順となりますので、ご希望の方はお早めにお申し込み下さい。
