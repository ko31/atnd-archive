3人の「きっかけナビゲーター」が再び登場！

人生ハッピーそのもの！毎日楽しくてしょうがない！！
この先、なんの不安もない！！
という方は、この先を読んでいただく必要はありません。

でも……。

◯毎日忙しくて、いっぱいいっぱい。
　雑用に追われてばかりであっという間に夜が来る。
　何も達成感がない。
◯仕事や家庭で、いろいろ期待されてることが多くて、正直重荷。
　考えるとなんだか眠れなくて、毎日本当に疲れてる。
◯慢性化した肩こり、腰痛持ち。歩くのもおっくうで、面倒くさい。
　健康診断の数値が悪いことわかってるし、病院行きたくない。
　このままじゃダメなことわかってるけど、どうしたらいいのかな？
◯子どもの進学・就職・結婚とか、いろいろ考えることが多すぎ。
　つい、後回しにしてきたけど、そろそろそうもいかなくなってきた。
　夫や両親との関係もこのままで大丈夫かな？
◯友達はいるけれど、価値観が合わないことが多い気がする。
　わたしのことをわかってくれている人なんて、いないんじゃないかな。

いろいろ考えると、自信なくなっちゃいますよね。

今回、問題の裏の裏を読む3人の超異色のスペシャリストが、
これからの行き方で悩むあなたをチームで応援するために再結集しました。

「きっかけナビゲーター」

◯ 稲岡義昭 板橋からだ回復院、整体）
　体の不調の原因は筋肉の緊張です。体全体をリラックスさせて、
　あなたの人生、健康面で全面的にサポートします。　

◯ 藤澤和斉 （日本語研究家）
　キャリア、ワークライフバランスなどの考え方は外国の考え方です。
　日本語を母語とするあなたの感覚に合わせたライフスタイルをご提案します。

◯ 宮腰育幸 （人生のシナリオReライター）
　心の奥底に眠る思い込みが自由に生きることを邪魔しています。
　潜在意識の思い込みを書き換えて、あなただけの幸せな人生をプロデュースいたします。

このセミナーに参加すると・・・

  1. 毎日の生活に生き甲斐と達成感を「こころ・からだ・ことば」で感じるきっかけがつかめます。
  2. 疲れた感覚を「こころ・からだ・ことば」でやわらげるきっかけがつかめます。
  3. 肩こりや腰痛・・・実は肩や腰以外の筋肉や、はたまた心のあり方に原因があることをご存知でしょうか。「こころ・からだ・ことば」で健康になるきっかけがつかめます。
  4. 家庭で仕事でも人生を見直す時期。「こころ・からだ・ことば」から解決の方法がみえることで後回しにせず、前に進むことができます。
  5. 自分がわかってもらえないな、と思うとつらいです。「こころ・からだ・ことば」から気持ちが楽になるきっかけがつかめます。

変わって幸せになるあなたを想像してください！
お友達と一緒に参加してみませんか！
