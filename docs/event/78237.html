出演者プロフィール

■GUEST DJ

saki.k

2011年に大阪でキャリアを開始したDJ/オーガナイザー

名古屋、福岡、熊本など多数の都市に招聘され、2014年には台湾で自身初となる海外出演を果たした。

オーガナイザーとしては「東方×クラブミュージック」をキーワードにしたパーティ"FarEast"、全国11の東方クラブイベントを集めた"AJTF - ALL JAPAN TOHO FES -"などを企画。

全国の東方DJシーンに多大な影響を与えている。

'15年より活動拠点を東京へと移すと共に、心機一転名義を変更。

聴き手の内面に訴えたいという軸をぶらすことなく、東京での活動に挑んだが、その実態はエニタイムハウスタイムというよりサムタイムハウスタイムであった。

東京生活も苦節2年目、コアタイムとの折り合いをつけながら最高のハウスタイムを目指すSAKI.Kの挑戦は続く。


sai.ko ( DISCOVER / Lunatic Love )

自身が主催する同人音楽パーティDISCOVERでは、開催当日朝にタイヤのパンクに見舞われるなどの活躍により、多くの注目を集めた。

実家は暴力団特別警戒区域。


HIRO ( 東方萃楽宴 )

東方萃楽宴主催。2009年9月頃より萃楽宴立ち上げと同時にDJ活動を開始。

活動開始以前、たまたま遊びに行ったアニソンクラブイベントで東方アレンジと出会い「試しに1枚」と会場物販で買ってしまったが故に、気付けば東方projectにどっぷりハマり込んでしまう。

自イベント「東方萃楽宴」の他に同県内外問わず東方系のクラブイベントやアニソンクラブイベントに多数出演しており、東方クラブ系アレンジ楽曲を中心にElectro House寄りの楽曲を好んでいる。


かことん ( T.M.R. )

2010年に悪友に唆されてDJを開始、即売会で買った東方アレンジでプレイをする。 色々あって現在【T.M.R.-Toho.Miracle.Revolution.】を主催、東方アレンジとクラブミュージックなどを主体としたパーティを開催している。

イベントのオーガナイズがメインながら、独自の解釈で音を捉え、ひとつの世界を作り上げるDJプレイを目指してDJ出演も行っている。


■Guest VJ

VJ Akinashi ( T.M.R. / 一畳未満 )

生まれも育ちも愛知県名古屋市。公用語は名古屋弁。

2012年よりVJ活動開始。現在は名古屋東方クラブイベント「T.M.R.」にてレジデントVJおよびフライヤーデザインを務める。

過去には「音盤夜行」出演や「AJTF」のVJ統括等、T.M.R.以外にも精力的に活動しており、名古屋を中心にカルチャー、サブカルチャー問わず色々なイベントに出演。

さまざまな映像をかき集め、機材の隅々まで使い、あらゆる手法を駆使して音に合わせた映像を展開して、いかにお客さんを楽しませる事が出来るか日々試行錯誤している。

また、映像制作技術を駆使しjingleを制作、DJの転換に合わせて演出することを得意としている。


■DJ

Mi-Yan ( modify / Reverie )

大学時代に東方永夜抄を紹介され、優曇華にホイホイされる。同時に様々な東方アレンジに興味を持ち、CDを収集し始めた。また、同時期からクラブへ通い、HOUSE,TECHNOなどのクラブミュージックに触れる。

2011年3月頃よりDJを開始。

ジャンルは、東方アレンジの他、Progressive House・Tech Houseなどを中心にプレイ。これまで「A-Crash!!」「ツンデレーション」等のアニソンイベントや「Far East」「T.M.R」「音盤夜行Ex」等に出演を果たす。

2011年7月より「STEP!」をオーガナイズ。東方とクラブミュージックの融合と普及に微力ながらも尽力している。

最近メキシコ出張より無事帰還。


Pyrovers

大都会岡山出身。

都内で受託開発に従事する傍ら、時折DJ活動しているあなたの町のWeb屋さん。クラブに遊びに行くうちに、気がついたらDJブースに立っていたとかいないとか。

STEP!の遠すぎるレジデントを務めつつ、音を求めてあちらこちらへ右往左往。Progressive House、Trance主体の叙情的なグルーヴを好む。

そんなにやかましくないスクリレックス亜種…だったのはもはや過去。近年は日本庭園を作ったり、吹雪の中ゾンビと死闘を繰り広げたり、文明を育てて繁栄を謳歌したりと卓上での活動は多岐に渡る。


しゅれ ( ミクノビーツ / A-Crash!! )

何かに導かれヒロストにてDJデビュー。

その後、A-Crash!!をメインの場として、チキンナイフ、VOCALOID LABORATORYなどのイベントに出演、ボカロクラブイベント「ミクノビーツ」を主催。

東方Project作品には東方紅魔郷のときに出会う。

東方妖々夢のEXはぎりぎりクリアできた程度の元へたれシューター。 酒好きが高じて伊吹萃香を敬愛。（ただしロリコンではない）


りふ ( animazione!山口 / ミクノビーツ / AniTunes! )

中学２年でレミリア・スカーレットに出会い東方厨そのまま厨二病へと足を踏み入れた

紅魔館の屋敷で迷ってしまい扉を開けるとそこは大きな図書館だった

暗いのと寂しい思いで泣きそうな僕に優しく声をかけてくれた彼女

少しめんどくさそうに僕の手を取って歩き出す彼女

後ろから着いて行く僕、いい匂いがした

雑談をしているうちに増える彼女の顔を横目に楽しい夜は更けていった

「そういえば名前を聞いてなかった、、、あなたお名前は？」

「あ、ぼ、ぼぼ、僕はりふって言います、、、あなたは、、」

「りふ、、、不思議な名前ねわたしはパチュリー・ノーレッジよろしくね」

僕とパチュリーの恋物語が今、始まる


Kenryu ( Feel The Beat!! / ΣXITENZ / ミクノビーツ )

2012年、KONAMIの音ゲー「SOUND VOLTEX」との出会いによりボカロと東方、同人音楽の沼に沈み、DJを志す。広島音ゲークラブイベント「Feel The Beat!!」主催。

音ゲーメイン機種はSDVXと弐寺、プレイジャンルは主にハードコア。


デラグルーヴ ( やばきゅーぶ )

はじめて遊びにいったクラブイベント「STEP! with 東方埜麟祭」で衝撃を受けDJ沼へとハマる。翌年DJデビュー。

東方アレンジは8年前より収集しており、自宅は同人CDで溢れている。House,Bassアレンジが好き。万年ノーマルシューター。

東風谷早苗さんが好きすぎて一周回ってエロ本が買えません。東風谷早苗さんの来場お待ちしております。


■VJ

みはる ( ミクノビーツ / A-Crash!! )

2011年「VOCALOID LABORATORY」スタッフとして活動と同時にVJを始める。

その後も「A-Crash!!」「ツンデレーション」など様々なイベントにVJで参加し、日々勉強中。

普段の姿は人妻、ネトゲーマーであり、ただの酒飲みである。

東方Projectとの出会いは原曲を耳にして心奪われ、CDを買い漁る日々。

シューティングが苦手なのでゲームには手を出せないでいる。

嫁は火焔猫燐。おりんりんランドに住みたい。にゃーん。
