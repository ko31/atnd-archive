description

突発でとりあえず集まって何か作ろうぜ！

■コンセプト

・見た目が面白い料理（例：オウムライス、ブルーカレー）
・話題だけれど食べたことがない料理（例：タモリのカレー、シュールストレミング、まんが肉）
・調理方法が独創的（例：自作料理デバイス）
・料理としてオルタナティブ（例：電気味覚）
・単純に普段食べてる、美味しい（例：紅茶、気合いが入った角煮、cookpadで話題のレシピ）

■内容

・11:00にあつまり、おもむろに料理を作り、ワイワイと食べる
・食材持ってきてくれるひと歓迎

■決定している企画

・あの「片栗粉X」を作って食べます。　＞注：下ネタ　片栗粉X-3写真解説　http://t.co/n7JPGGmZz9
・食紅で超色とりどりのクリームをやって、アメリカンケーキを作ります。
・ニコニコ技術部abaさんが、記録可能な炊飯マシンを持ってきてお米を炊いてくれます。
・とりあえず伊予柑が紅茶出します。いれかた教えます。
・ドリアンを買ってきて食べます｡
・上野の地下にある謎の海外食材店で買ってきた謎のものをとりあえず食べてみます（動画必須）
・ガリガリくんコンポタを溶かして飲む
・ニコ生する

■参加方法

・このatndで参加ボタンを押す
・11:00に「きゅりあん」に集まる

■募集

・ドリアンだれか買ってきてください
・上野で誰か怪しい食材買ってきてください

日時：4月13日(土）11:00-17:00(完全撤収)
施設料：16600円（使うスペースが少ないと多少下がる）

参加費：０円（会場代・食材費のみカンパいただけるとうれしいです）

■設備

・会場自体は09:00からあいてます
・和洋中一通り作れる設備がある。包丁、ミキサー、計量器などがある。
・ガステーブル、ガスオーブンがある。
・プロジェクター用スクリーン（無料）がある。プロジェクターは２０００円。
・超簡易PAが借りられる（無料）　小学校で校長先生が使うようなちゃちいやつ
・各テーブルのコンセントは20A。
・ゴミはゴミ箱１個まで捨てていける
・商業利用可
