description

このワークショップでは、第一線の現役ビジネスマンが持っている活きた知識・スキルを共有して、新しい発想や刺激、気づきを得ることを目的としています。


現役コンサルタントが実践している
生きたスキルが学べるワークショップです。


今回は第一線で働く現役ITコンサルタントを講師として
「明日から毎日1時間早く家に帰る」を目標に
コンサル的思考を実践した「時短」仕事術の研究講座を行います。
第1回ということで参加無料ですので気軽にご参加下さい。
社会人の方はもちろん、学生の方も参加頂けます。

講座内容


コンサル的思考で仕事の時間を短縮し、
毎日1時間早く仕事を終わらせる仕事術を身に付けます。

「毎日1時間早く家に帰る」＝「余暇が1日1時間増える」

1日1時間増やすことができれば、
1週間で5時間、1ヶ月で20時間の余暇を得られることになります。

講座の限られた時間内で、現役コンサルが実践する
速効性の高い活きたスキルを吸収していきます。

Excel、アプリ活用などのITリテラシー向上をはじめ、
データに基づく方法論、コンサルが現場で使う手法など、
第一線で働く現役コンサルタントのスキルを凝縮した1時間です。

簡単なサンプルや事例を交えて進行しますので、
新卒や実務経験の浅い方でもご心配ない講座内容となっております。
