description

石田三成公茶の井の伝承がある竹林の竹でキャンドルホルダーを作り点灯します。

滋賀県彦根市古沢町にある仙琳寺（せんりんじ）は、井伊家と深い関わりがありがあるお寺です。裏山の竹薮には、今も井戸があり「石田三成公の茶の井」との伝承が残っています。ひこねキャンドルナイト実行委員会は、竹薮を整備し、切り出した竹でキャンドルホルダーを作り、石田三成公と今を結ぶスローな夜を演出します。

「仙琳寺の竹灯」プレゼント
キャンドルナイト終了後、夢京橋宗安寺まで取りに来ていただければ、仙琳寺の竹で作ったキャンドルホルダーを差し上げます。ひこねキャンドルナイト2010・石田三成家紋入りロゴシール付き。
また、石田三成家紋入りロゴシールは、宗安寺にて、100円募金で配布しております。募金は2011年のキャンドルナイトの資金とさせていただきます。
