description

この度の震災におきましては、被害にあわれた方々に心よりお見舞い申し上げます。

  • 開催趣旨
    震災の混乱がいまだにおさまらないこの時期ではありますが、日々暖かくなり、桜の花も咲きほころびはじめました。
    震災以降、皆さん緊張のとけない日々をお過ごしのことと思いますので、この桜に少し甘えて、ちょっとだけ肩の力を抜いてひと時を過ごしましょう、という趣旨の元、お花見会を開催致します。
    先日開催された震災支援ブレストの内容や本会で推進している震災支援のとりくみにつきましてもこの場でお話しできればと思いますので、多くの方々のご参加をお待ちしております。
  • 会場
    飛鳥山公園（王子駅）
  • 開催日時
    4月10日（日）　13:00～
  • 各自ご用意いただくもの
    飲食物は各自で用意してお持ち下さい。
    経済活性化にほんの少し貢献できるよう、思い思いのもを持ち寄りましょう。
    敷物は10人分程度は用意しておきますが、参加人数によっては足りなくなることも想定されますので、できましたらご用意下さい。
    会場となる公園は小高い丘になっており、風が強かったり少し寒いときもありますので、念のため防寒のご用意もあるといいかもしれません。
  • グループ未参加の参加者の方は、以下URLよりGoogleグループへの登録をお願いします。
    http://groups.google.co.jp/group/android-saitama
