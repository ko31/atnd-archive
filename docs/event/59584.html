Ganbare Tigers Jr.として、IIJ関西支社の一部をお借りし、SECCON 2014 オンライン予選（英語）に参戦します。
本申込みは、12月6日分です(7日分は別途準備)。

2014 オンライン予選（英語）の概要は、下記ページを確認してください。
http://2014.seccon.jp/winter2014quals.html

オンライン予選は、12月6日午前9時から7日17時までの32時間にわたり、開催されます。
12月6日、7日とIIJ関西支社に集まるかたは集まって、一緒にチャレンジしましょう。
遠隔地から参加されるかたも、一緒に頑張りましょう。
情報共有方法は改めて連絡します。

なお、12月6日は、PCを持参の上、8時30分までに、IIJ関西支店が入っているビルの1F 裏口(コーヒーショップ辺り)に集合してください。
汚さない、ごみを持ち帰ることを前提に、飲食可能です。
途中で買い出しも可能ですが、昼食や軽食を準備される方は、事前に用意してきてください。
