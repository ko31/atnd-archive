11月20日(木) に FI 関西が主催するピッチイベント「Startup Pitch Bootcamp」に、あの話題のVR プロダクトの開発を手がける FOVE Inc. の小島由香 CEO の登壇が決まりました！

https://www.youtube.com/watch?v=aPxNK24zzqg

今年の TechCrunch Disrupt で世界発となるアイ・トラッキング技術を持つヘッドマウント・ディスプレイをお披露目した FOVE 社は、Micorosft Ventures London のアクセラレーター・プログラムに採択され、現在はロンドンで活動を行っています。

多忙を極める小島 CEO ですが、今回ロンドンからのライブ中継という形でイベントに登壇頂き、ご自身の起業体験やハードウェア分野で起業する際のアドバイスついてお話頂きます。

また、今回のイベントでは参加者のスタートアップ・アイデアのピッチタイムを設けます。世界の最先端を行く小島 CEO や FI 関西のメンター陣から直接フィードバックを受けられる非常に貴重な機会ですので、ぜひご参加ください。

IoT・ハードウェア・VR に関心のある方はもちろん、起業を検討されている女性の方も、この機会をお見逃しなく！
