description 100万人のキャンドルナイトという呼び掛けに、たんぽぽのうちも3年前からこの呼びかけに応えてきました。 ひとつの灯りからスタートした岐阜市の中心街、“柳ケ瀬”で、このようなイベントが出来ることを、心から感謝しつつ、皆様と一緒にエコや平和を願いながらスローな時間を過したいと思います。当日はお子様にエコカードを持っていろいろなブースを回ってもらいながら、楽しんでもらう企画をしています。又、色々なお店の出店(パン屋さん・雑貨屋さん・等など)や、演奏、19時からはいよいよ商店街の灯りが消され、キャンドルナイトの時間。大切な人と一緒に素敵な時間を過しませんか!皆様のお越しを心からお待ちしています。。。 尚、当日のエコラリーカードは申し込みが必要です。1枚800円ですがご兄弟がいらっしゃる方は2枚目から500円で購入いただけます。
