description

高分子分散剤としての両親媒性高分子の微粒子表面処理とその分散メカニズム，効果について基本原理を理解することを目指します。特に、両親媒性高分子の精密重合法と分散効果を得るための設計指針，その効果について概説します。

プログラム

１．帯電コロイド粒子の動電現象
　　ゼータ電位の基本的考え方
２．DLVO理論の概要と分散の考え方
　　分子間相互作用とからコロイド粒子間相互作用へ
３．高分子の吸着と分散・凝集作用
　　3.1　分散剤の種類
　　　　 高分子精密重合を使った分散剤の合成
　　3.2　高分子分散剤の分子設計
　　　　 分散剤と微粒子表面との相互作用と溶媒和効果
　　3.3　高分子分散剤の分散機構
　　　　 分散剤の表面状態が分散に及ぼす効果
