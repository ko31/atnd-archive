2014年第１回目の統合開発環境 Xojo 東京勉強会を開催いたしますので、ご案内申し上げます。今年は Xojo Cloud のサービス開始や iOS への対応など、賑やかな年になりそうです。情報交換もしていければと思いますので、ご参加よろしくお願いいたします。
