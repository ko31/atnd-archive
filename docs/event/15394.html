関西での第1回の組み込み勉強会です。

現時点ではまだ具体的な内容が決まっていません。
ですので、参加者の皆様で発表して頂ける方募集です。(プロジェクター利用可)
ちなみに私は、最近携わった組込みの仕事での経験(失敗談)等を
まとめて発表してみたいと考えております。

関西での第1回目の勉強会になりますので、組込みに関わらず、自己紹介
(自社で取り組まれていること等)的な発表でも問題ないかなと思います。

グダグダ進行になるかもしれませんが、皆さんと一緒に盛り上げていければと
思いますのでどうぞよろしくお願いします。

不明点等ありましたら本ATNDのコメントまたは、
hiroyuki.miyazaki13@gmail.com
までお願いします。
※組込みこぞうの会にも投稿しておりますのでそちらでも大丈夫です。

■アクセス
大阪市営地下鉄、堺筋本町駅より徒歩5分です。

詳細は下記の会場アクセスURLにてご確認ください。
http://www.sansokan.jp/map/

http://www.sansokan.jp/map/#kouiki

■会場設備
・ネットワークは会場の無線LANが利用できます。
・プロジェクターも利用できます。

■参加条件
組み込みに興味がある方でしたらどなたでもご自由にご参加ください。
なお、発表して頂ける方も募集しております。
会社での取り組み、電子工作、遊んでみたボードの紹介、RTOS等々、
組み込みに関わる内容であれば基本的になんでもOKです。

■参加費
会場利用料と備品利用料を参加者で割り勘させて頂く予定です。
目安ですが、10人程度の参加で一人500～600円程度になりそうです。

皆さんと一緒に楽しく学んでいけたらと考えておりますので
どうぞよろしくお願いします。

【追記・連絡】
2011/6/4

■時間変更につきまして
勉強会時間を18:15 ～ 21:00に変更したいと思います。
※18時には会場のカギを開けられると思いますが、準備等を考慮して開始時間を15分ずらしました。

■発表につきまして
参加される方で何か発表頂ける方は本ATNDのコメントまたは組込みこぞうの会のMLで、
・発表する内容のタイトル
・発表の目安時間
を投稿頂けると嬉しいです。

私の方も内容は薄いですが、「H8とSH2での最適化について」
というタイトルで発表させて頂きたいと考えています。(10分～15分程度)

■懇親会(飲み会)について
当日の懇親会についてですが、終了時刻が21：00予定ということもあり
現時点ではお店の予約などは考えておりません。
当日終了後にいけそうな方何人かいらっしゃったら、
集まってどこかお店にという流れを想定しています。

それではよろしくお願いいたします。
