Docker 触ってみませんか？

昨年ぐらいからインフラ業界で話題になっているDocker。
東京では勉強会やハンズオンが行われているけれど、ここ名古屋では…orz

そんな（私の含む）人のために、名古屋で集まってDocker触ってみませんか？

私もDockerについて詳しいわけではありません。
みんなで色々情報共有できればいいなと思います。

対象

  • Dockerという名前だけを知っている人
  • Dockerを触ってみたい人
  • 仮想化環境に興味がある人
  • 手軽に開発環境が欲しい人

持ち物、準備

  • 無線LANが使用できるノートPC
    • OSはWindows、Mac OS問いません
    • ハンズオンは各自のPCにDockerサーバを構築して行います
  • DockerのインストールはしていなくてもOKです
  • Vagrant (64bit版), Gitを予めインストールしておいてください
    • わからない方は早めに会場へ来ていただければお手伝いできるかも
