XAMPPをインストールしてください

講義にはXAMPPというソフトウェアを使います。
これはパソコン上（Windows,Mac）で、Webサーバーを実行するプログラムです。
（基本的にインターネットで使用されるWebサーバーと同一のプログラムです）

Windows版
http://www.apachefriends.org/jp/xampp-windows.html

Mac OS X版
http://www.apachefriends.org/jp/xampp-macosx.html

インストール後、http://localhost/にアクセスして、インストールが成功したか確認して下さい。

インストールに失敗した方は、http://twitter.com/phpbeginnersまで、ご相談ください。
