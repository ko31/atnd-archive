プレゼンター

桑島 洋

東京支社 / 電子基盤先行開発室　担当係長

2008年大阪大学大学院 工学研究科 修士課程修了。2009年 株式会社デンソー入社。2013年から2015年までStanford University客員研究員。2016年から人工知能の品質基盤の構想と技術開発に従事。

曽田 尚宏

基礎研究所 / 先端研究部 / ＡＩ研究課

2012年 東北大学 情報科学研究科 博士後期課程修了。博士（情報科学）。2013年株式会社デンソー入社。ＡＤＡＳや自動運転システムのためのニューラルネットワークの研究に従事。

佐藤 育郎

株式会社デンソーアイティーラボラトリ / 研究開発グループ　リサーチャ

2005年 米国メリーランド大学にて博士号取得（理学）。2005年から2007年まで米国ローレンス・バークレー国立研究所にてポスドク。2008年 株式会社デンソーアイティーラボラトリ入社。コンピュータビジョン。機械学習に関する研究開発に従事。2013年画像の認識・理解シンポジウム(MIRU2013) MIRU長尾賞受賞。
