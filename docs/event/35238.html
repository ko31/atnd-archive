第27回勉強会を開催します。

　平成25年最初の勉強会は、タオソフトウェア株式会社　谷口社長に講師をお願いし、
Androidアプリが行う通信の最適化の手法「ARO（Application Resource Optimizer）
ツール活用方法」について講演をいただきます。
　また、この日（1月18日（金））は勉強会の前15:00～17:00に「Androidアプリのセキュアな設計」セミナー
を同じ会場で開催いたします。こちらも是非ご参加ください”無料”です。

内　容：Androidアプリが行う通信の最適化の手法
　　　　「ARO（Application Resource Optimizer）ツール活用方法」
　　　　講師：タオソフトウェア株式会社　代表取締役　谷口　岳　氏

　　　　・AT＆T社が開発したARO(Application Resource Optimizer)というツール
　　　　　を用いたAndroidアプリケーションのプロファイリングを通じて、Androidアプリ
　　　　　ケーションが行う通信の最適化の手法を紹介します。
　　　　　通信の最適化を行うことで、ＵＸの向上や電力消費量の低下を期待することが
　　　　　可能となります。
