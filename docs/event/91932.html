基礎から学ぶブロックチェーンと暗号通貨(仮想通貨)セミナー　暗号通貨を使った話題のサービス

ブロックチェーンや暗号通貨の基本的な仕組みや理論の理解と、その技術や活用事例を暗号通貨のソフトウェア開発者が基礎から解説

初心者を対象としたブロックチェーンと暗号通貨(仮想通貨)の基礎を学ぶセミナーです。 また暗合通貨を使ったビジネス活用事例や話題のサービス※リリース前から注目のREAL BOOST、そして暗合通貨を管理するウォレットアプリHB Walletのお話もしていただけます。

登壇者

奥田 雄馬 bacoor株式会社 代表取締役
2009年に関西学院大学経済学部へ入学しマクロ経済学を専攻。 IS/LMモデルを中心に財市場や貨幣市場の構造を研究する。
2017年2月9日にbacoor株式会社を設立し代表取締役に就任する。
