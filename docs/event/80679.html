description

プロの写真家がスマートフォンで料理写真を上手に撮る方法を教えます！

はじめて作った料理だから、写真に残しておきたい。
綺麗にできたお弁当、スマホで撮って記録しておきたい。

でも、撮ってみると、どうも美味しく見えなくて、せっかくの料理が台なし。
綺麗なお弁当が台なし。

そんなことはありませんか？

飲食店を経営していて、自ら料理写真を撮っている方も、同じような悩みを持っているかもしれません。

美味しいお店で、料理写真に撮って、ＳＮＳにアップしても、その美味しさがいまいち伝わらない。
そんな写真になってしまうとはよくあります。

料理写真（テーブルフォト）って、結構奥が深いのです。

もちろん、写真に収めるために、綺麗な料理を作っているわけではありません。
色鮮やかなお弁当にしているわけではありません。

色鮮やかなのは、栄養満点だから。
綺麗なのは、家族や子どもに喜んでもらいたいから。

だから、
「美味しそう！」
「綺麗～」
という声ほど、作り手にとって嬉しい言葉はありません。

もし、実際と同じような美しさを写真で表現できたとしたら、どうでしょう。

あなたのお料理を食べてみたいという人が増えるかもしれません。
同じお弁当を作ってみたいと思う人も出てくるでしょう。
あなたが作ったお料理やお弁当に、ファンがつくかもしれません。
教えてほしいと頼まれるようになる可能性だってあります。

そう。
美味しそうなテーブルフォトを撮れれば、あなたのお料理・お弁当が広まります。
そして、あなたのお料理・お弁当を食べてくれる人が増えるのです。

そんな方にオススメしたい講座が、「スマホで美味(うま)いテーブルフォト講座」です。

色鮮やかな野菜中心のランチを食べながら、料理写真をスマホで綺麗に撮影する方法を学べます。

講師は新進気鋭の写真家・野口健吾さん。
東京藝術大学大学院を卒業後、様々なカメラマンに付いて修行を重ね、現在はネパールや北インドを中心に世界を飛び回っています。
昨年は、若手写真家の登竜門のひとつ「ニコンサロン」で写真展「Family Affair」を開催。
各方面から注目を集めています。
専門学校でも教鞭をとるなど、丁寧な教え方には定評があります。

ランチは表参道「海月」の料理人・毛利奈津美さん。
近茶流茶懐石料理を習得した彼女は、2012年、最短期間でシニア野菜ソムリエを取得。野菜のうまみを引き出す調理法「＋１vegeメソッド」を考案。
COOKPAD料理教室ではつねにランキング上位の人気を博しています。
野菜はルコラステーションの畝田謙太郎さんが選び抜いたもの。ドレッシングは毛利さんが考案し、スタイリストのソニア・パークさんのお店「DOWN THE STAIRS」に提供しているものを使用します。

そんな栄養豊富な絶品料理を食べながら、料理写真の撮り方も学ぶことができるのです。

開催日は8月末。
「味覚の秋」の直前です。
これからもりもり食欲がわいてくるという時期に、スマホで美味しそうな写真を撮って、あなたの料理を広めませんか？

あなたにご用意いただくのは、スマホ一つ。

表参道でランチを楽しみつつ、テーブルフォトを学びましょう。

写真家・野口さんは8月末より、ポーラ美術振興財団の助成を受けて、1年間インドに行くため、この講座は少なくとも今後1年は開催されません。
この機会に是非、ご参加ください。

野口健吾（写真家）

東京藝術大学大学院美術研究科先端芸術表現専攻修了。
