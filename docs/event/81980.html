講師の清水宗己（しみずもとき）氏はコピーライティング、クリエイティブディレクション、編集企画、NHK World(国際放送）の放送作家、マリンジャーナリスト等 幅広く活躍し、毎日広告賞、朝日広告賞、消費者が選ぶ広告賞など、多くの受賞をされています。彼のその多才なバイタリティの原点は、自然と向きあう、美しく過酷なスポーツ《セイリング》でした。清水氏曰く 「海や風、強い陽射しなど、懐の深い自然がもたらしてくれる環境は、何物にも代えがたく、陸にあがると、すぐに潮の香りを求めてしまう。｣とのこと・・・
その《セイリング》のお話を通して、ダイナミックな地球を感じてみてください！

◆講 師 :
清水 宗己 （しみず もとき）1947年鳥取県生まれ。
コピーライター兼編集者。
株式会社清水宗己事務所代表。
東京コピーライターズクラブ会員。
日本セイリング連盟会員。
マリンジャーナリスト会議メンバー。
広告制作会社に合計14年間勤務後、1985年自身の事務所を設立。
T.C.C.新人賞、毎日広告賞、朝日広告賞、消費者が選ぶ広告賞受賞他。
