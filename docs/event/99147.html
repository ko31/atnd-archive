description

おそらく誰もが不安に思っている老後生活ですが、老後の資金や介護のことを甘く見てしまったり、また何を対策をすれば良いかわからないまま時が過ぎ、いざ現実の問題になって途方に暮れてしまう方がたくさんいらっしゃいます。

今回のセミナーの前半は、老後の諸問題への対策を行う事によって、老後や相続問題が解決するだけでなく豊かな老後生活にできた実例や、贈与に頼らない相続税対策の実例など、ほかでは聞けない相続対策現場でのお話を、わかりやすく事例を元にお伝えします。

またセミナー後半では、近年、新しい相続対策の手法として注目を集める不動産小口化商品について、そもそもどういったものか？なぜ注目されているのか？という点をわかりやすくご説明します。

（相続や老後対策についてあまりしらない初心者の方でも、ご参考になる内容となっています）

　

アイディ（株）合同　相続対策セミナー・相続相談会　in 大井町きゅりあん　

お盆のあとに親子で考えたい、老後問題と相続対策（無料）

老後対策も万全。節税・争族対策しつつ、親を大切にする対策 と 最先端の対策手法

　　講師： アイディ株式会社　経営企画室　佐藤公洋　宅地建物取引士、不動産コンサルティングマスター
