catch まずはインドビジネスの可能性を知ってください。
description

「海外進出を考えているが、どの国がいいの？」「海外に販路拡大したいけど、どの国がいいの？」
ぜひインドにも注目してください。なぜ 今インドビジネスが 魅力なのか 、
そして成功のための実践的ノウハウとインドビジネス最新成功事例をお伝えします。

日時：２０１４年８月２５日（月）　１５：００～１８：００

１．これだけは知っておきたい、インドの基礎知識
●インドの「大きい」「多い」（土地、人口、交通事故件数など）
●宗教（頭にターバンを巻くインド人は、多い？少ない？/仏教発祥の地インド、仏教
　徒はどれくらいいるの？など）
●人口ピラミッド（日本、インド、中国を比べてみると一目瞭然！）
●平均年齢（日本の平均年齢は、インドの○倍！）
●日本企業に吹く“政治的な追い風”と”経済的追い風”

２．日本で働くインド人だからこそ見えるインドの可能性！
●インド人からみた日本企業（こうすればもっとインドでのビジネスが上手くいく。
　インド人とのビジネスの進め方）
●インドで今、こんなビジネスがチャンス！（今伸びている産業、これから伸びる
　産業。日本企業だからこそのチャンスとは？）
●インドと日本、企業体質の違いとは？

３．インド専門家と日本在住インド人経営者に質問！

４．懇親会

インドのみならず、グローバルビジネスに興味のある方は、必聴です。
