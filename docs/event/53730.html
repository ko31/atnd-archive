【Q&A】

こちらのコーナーとあわせて、ハッシュタグ「#DIS勉」もご覧ください。

Q-01.話ができないと参加NGですか？
A-01.いいえ。聞くだけでも参加OKですが、みんなが情報を持ち寄る場にしたいと考えています。自分ではたいしたことないと思うことも、実は重要な情報だったりする場合がありますので、遠慮せずに発言をお願いします。

Q-02.差し入れは、必ず必要ですか？
A-02.いいえ。必ずではありませんが、大歓迎です。ただし、ご自身の分は必ずお持ちください。（人の差し入れを当てにして、手ぶらで来るのはダメダメ）

Q-03.飲み物はアルコール類もOKですか？
A-03.はい。OKです。ただし、未成年の飲酒は固くお断りします。

Q-04.飲み物は冷やせる環境はありますか？
A-04.はい。冷蔵庫の用意があります。大きさは一人暮らしの方が使う程度の大きさです。入り切らなくなった場合は、別の場所にも冷蔵庫がありますので、一時お預かりすることは可能です。

Q-05.ライトニングトークの際に自分のパソコンを使うことはできますか？
A-05.はい。可能です。ただしRGBケーブル（D-sub15pin）での接続となります。macご利用方や、最近のパソコン（HDMI出力のみ）をご利用の方は、変換ケーブルをご準備ください。

Q-06.ライトニングトークの追加枠はありますか？
A-06.現在調整中です。

Q-07.みんなのライトニングトークを聞いてたら、自分も話したくなった！当日突参ありですか？
A-07.時間にあまりがあればご案内します。また、延長戦での発表時間も検討中です。

Q-08.インターネット環境はありますか？
A-08.発表者用のインターネット環境は用意する予定です。無線LAN接続を希望される場合は、設定が必要になるので、早めにお越しください。
参加者用のインターネット環境はありません。ご自身でご準備お願いします。

Q-09.箸、皿、コップなどは必要ですか？
A-09.@mtakeshiさんのご厚意で、すべてこちらで準備します。

Q-10.差し入れで出たゴミの処分は？
A-10.会場で捨てられます。

Q-11.当日の、録音・撮影はOKですか？
A-11.私的利用での「録音・撮影」はOKとします。ただし講演者が拒否した場合を除きます。「録音・撮影」した物の公開については、各講演者の指示に従ってください。

Q-12.両手を使いたいのですが、マイクスタンドはありますか？
A-12.マイクスタンドはありませんが、ピンマイクがございます。

Q-13.会場へは何時から入れますか？
A-13.スタッフ以外は、12:30～を予定しています。

Q-14.13時以降、遅れても会場へ入れますか？
A-14.できるだけ入れるように検討いたしますが、ビルセキュリティの関係上、非常に手間のかかる作業になります。できるだけ遅れないようにお越しください。
