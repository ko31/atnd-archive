description

　渋谷で毎週木曜日夜に開かれている「東京ビットコイン」ですが、それを真似て船橋でも開催いたします。
　今回のお話を頂く　あん共育　代表の桝田良一氏は、精力的にビットコインの情報を調査されており、金沢と徳島でもビットコイン勉強会を既に開催済みです。
　今回はビットコインの価値を知る学びの場であり、全国でもまだまだ少ない状態です。

　お金のこれから・ビットコインのこれからを皆さんと一緒に情報交換しながら勉強しましょう！

1.ビットコインって何
2.何故、今ビットコインなのか
3.ビットコインで何ができるのか
4.実際にビットコインを使ってみよう

　当日は500円分のビットコインをお渡しいたしますので、スマホもしくはパソコンをお持ちください。環境としてメールアドレスの受信確認も必要となりますので、メールアドレスもメモするなり持ってきてください。
