description

当校では、IT業界で就職や転職を目指す20代～30代のITエンジニア、学生、社会人の方（毎回20名様限定）を対象に、「IT業界就転職セミナー」を開催いたします。

セミナー終了後は隣接する「惣菜ばる　おためし屋」で昼食をとりながら、個別相談会を行います。IT業界に興味をお持ちの方は、ぜひご参加ください。

スケジュール：

11:00～12:00　説明会
12:00～13:00　個別相談会・昼食

参加費：

無料

資料：

履歴書、職務経歴書をご用意ください（学生の方は成績証明書もご用意ください）
※ご提出いただけない方は昼食のご提供をお断りさせていただきます。
※転職希望以外の同業者の方のご参加はご遠慮ください。
