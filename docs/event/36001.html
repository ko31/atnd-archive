■ WordBench鹿児島とは

WordBench鹿児島は、鹿児島県内のWordPressユーザ・コミュニティーです。 まだスタートしたばかりですので、まずは無理のない範囲で勉強会・情報交換会を開催し、徐々に活動の幅を広げたいと考えています。 WordPressに興味のある方なら、どなたでも参加できます。一緒に鹿児島のWeb業界を盛り上げましょう！

via : http://wordbench.org/groups/kagoshima/

■ そもそもWordPressってなに？


WordPress (ワードプレス) は、オープンソースのブログ／CMS プラットフォームです。セマンティック Web、コードやデザインの美しさ、Web 標準、ユーザビリティなどを意識して開発されており、無料でダウンロードして使うことができます。

via : http://ja.wordpress.org/

■ 各セッション概要と出演者のプロフィール

1 ) 設計書から制作物への具体的な落とし込み方法（GOWASの場合）

「設計(ディレクション)」から「制作(クリエイティブ)」に落とし込む際、なかなか言葉に出来ない「モヤっ」とした部分をクリアにするGOWASなりの方法をご紹介いたします。わかりにくいですね。。
例えば、デザイン。デザインには物事をわかりやすく伝えるための問題解決としての役割があります。ですから、デザインには感覚的な要素だけではなく理屈的な要素も含まれています。このような点を実際に調査、ヒアリングを基に作成した設計書から具体的な形(デザイン)に反映する工程を実例を交えてご紹介できたらと思います。

3 ) ぬるま湯トーク・初心者コース　〜 Wordpress・WebのことならなんでもOK! 〜

・ ぬるま湯トーク — 最近気になるWebトークや、使っているアプリ、それぞれの開発環境など聞いてみたい事を何でも語り合うコーナーの司会のみ担当します。主役はあなたです!!

・ 初心者コース — WordPressの使い方や、インストールの仕方、プラグインの事など、僕らも通ってきた様々なお悩みを熱狂的に解決します!!どんなことでも気軽に聞いて下さいね。実際に触りながら、優しく丁寧をモットーにお伝えします!!

- 工房 舞麗歩（コウボウ ブレイブ） http://www.webrave.jp/


鹿児島県霧島市隼人町見次屈指のWeb製作会社。
山登り、ミニ四駆の製作、モンスターハンターを得意とし、県内で行われる数々のプロジェクトに参加を希望している。
BRAVE（ブレイブ）とは「勇敢、勇者」の意味。ドラゴンクエストの勇者の様に、「霧島をWebで切り開いていこう」という決意が込められていることにしようと、最近決まった。
Web業務以外にも、写真撮影・映像コンテンツ制作・イベントサポートなどなど、幅広く活動している。地域密着で皆様と共に歩んで行く。そんな企業を目指している。
