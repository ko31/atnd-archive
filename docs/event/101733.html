『今、映像業界は深刻な人材不足！あなたの活躍のチャンスです！』

Youtubeなどのネット動画が一般化してきて、誰もが気軽に動画を見て楽しむ時代になりました。
もはや誰もが情報を動画で発信できる時代です。

これからは今まで以上に映像制作のニーズが高まってきますが、その担い手が不足している状況です。

TMS東京映画映像学校は20年以上、映画・映像業界へ人材を輩出してきました。
映像制作の流れを知りたい、業界への就職や実際の現場の状況を聞いてみたい、そんな方はぜひ体験講座にご参加ください。

業界で活躍しているプロが講師を務めますので、きっと有意義な時間になるはずです。

・映画やTVが好き
・映像制作に興味がある
・仕事上で映像を制作したい
・映像業界の仕事について知りたい

そんな方におススメです。

【1日☆ジョブトレーニング】映像クリエイター入門講座

　　　　東京都新宿区新宿1-34-15　新宿エステートビル7F
