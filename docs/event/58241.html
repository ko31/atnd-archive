description

　第６回湖畔舞闘祭―――滋賀天則・心綺楼オフの参加申し込みのページになります。
　オフ会は運営含め、参加者全員の協力や理解があってこそです。重ねてになりますが、ご理解ご協力よろしくお願いいたします！！

　このオフ会の詳細は、上記のＵＲＬ先で確認できます。参加申し込み前にご一読ください。また、二次会の参加申し込みはこのサイトとは別となっています。注意してください！！

期限となりましたので、参加申込を締め切りました。

１２月になってからの参加申込希望者は運営に連絡をお願いいたします。

参加申し込みについて

　参加申し込みの方法はページ右上にあるボタンを押し、アンケートを記入してください。
　分からないことがあれば、運営聞いていただくか、このサイト下部のコメント欄を利用し、質問してください。

参加費について
　参加費は６００円、ＰＣ持込の方３００円となっています。

　サイト上部の「参加費無料」となっているのは、サイトの仕様上変更できないので、ご注意とご理解お願いいたします。

参加申し込み期限
１１月３０日（日）　　オフ当日の一週間前となっています。

　参加希望の方は１１月３０日（日）までにこのサイトで参加申し込みをお願いいたします。
参加申込方法は右上のボタンをクリックして、アンケートに回答してもらい、参加確定となります。
　１１月３０日以降の参加表明に関しては、基本認めない方針です。期限を過ぎてからの参加表明は、まず運営に連絡をお願いします。運営への連絡はこのサイトへのコメント欄か、メインサイトのほうの連絡先、また運営のツイッターから可能です。

アンケートについて

　アンケートは名札の制作やオフの管理のためです。また。運営の人間にしか見ることが出来ない仕様になっています。
　ですので、キャラ対の心配などは不要ですので、アンケートへの未記入はご遠慮ください。
　ただ、アンケート最下部のコメントについては全員が閲覧可能という仕様になっています。

アンケートでは、
　名札に表記するお名前、対戦時のプロフ名、大会のどちらに参加するかどうか(天則・心綺楼・もしくは両方)、使用キャラ(非想天則・心綺楼それぞれ)、ＰＣの持込可かどうか(また心綺楼が動くかどうか)をお聞きします。
　参加申し込み時にご回答のほど、よろしくお願いします。

参加者上限

　参加者の定員は５４名となっています。これ以上の増員はありませんので、ご了承ください。この５４名には主催２名の分も含まれています。
　参加者の方々の宣伝は大歓迎です！　参加者の増加に協力のほど、よろしくお願いします。

プロフのアップロード

　プロフのアップロードは、こちら（http://ux.getuploader.com/kohan_butou6/）にお願いいたします。
　参加者は適宜、アップロードするようにしてください。当日、持ってきてくださったＰＣに参加者全員のプロフ入れないといけませんので、オフ前日までにはプロフをあげるようにしてください。

打ち上げ（二次会）について

　運営ではオフの二次会の開催を予定しています。ですが、二次会の参加申し込みや詳細は、管理の都合上、こちらの参加申し込みとは別のサイトで行います。
　二次会の参加ページはこちら（http://twipla.jp/events/116160）となっています。
　参加される方はご確認をよろしくお願いいたします。
