description

国際市場の新製品導入戦略マネジメントでイニシアチブをとってきた自己体験をもとに講座を構成します。社会環境や消費者意識の変化を見極めた上で、自らの強みを生かし新価値を創造する視点を伸ばすプラクティスを行います。また創出した価値を最大化し戦略の一貫性を具現化する全体最適マーケティングの考え方へのヒントにより参加者が自ら考え方をまとめていくプロセスを提供します。
