description

中国ビジネスの成功を考える上でこれまで見落とされてきた視点、多くの日本企業が進出で失敗に終わった理由と原因を、中国人と中国社会の隠れたルールに照らし合わせることで、中国の理解を深められます。

拙著の『中国市場　攻略のルール』をもとに、日本と中国の価値観の違いを丁寧に解説し、価値観をどのように受け入れ、共有するべきか、それを実行するためのルールを解説します。

本書の各章から要点を抜粋し、書ききれなかったポイントを補足します。

１．夢の中国13億人市場で成功する！とっておきのビジネス情報
２．過去の失敗から得た教訓を企業の未来に活かす！
３．中国人と関係（グワンシ）を築けば、ビジネスはうまくいく
４．相手の勢いに丸めこまれない絶妙な交渉テクニック
５．優秀な人材を確保する！雇用＆育成のポイント
６．上海・北京だけではない！狙い目の新興都市はココだ
