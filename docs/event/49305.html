description

Voice
普段、大きな声を出す事ってありますか？？
声と共にストレスもアウトプット！！
声を楽に出すためには？基本の発声練習！
歌が上手くなるコツを伝授します！プロのちょっとしたテクニックであなたのカラオケもワンランクUP☆
また、ヨガでリラックスした身体で発声する事により、驚くほど声が出しやすくなります！！！ぜひ体感してみてください！
気持ち良く、カッコ良く、楽しく歌えるようになりましょう！！

Yoga
リズムヨガとは？！
ヨガのポーズをとる前に
楽しくリズムを刻みながら素早く代謝を上げ、
肩甲骨周りの筋肉を柔くしていきます。

肩甲骨には１０種類以上の筋肉が集まっていて
その周りには、体の熱を作り出す「褐色脂肪」細胞といい、脂肪を燃焼する働きがあります。
この肩甲骨をリズムヨガで柔軟に動かしていくことで、体全体の熱を作り出し、全身運動にも！
背中の余分な脂肪を落として綺麗な背中美人に！！

ヨガ経験者はもちろん、カラダが硬い人や
初心者でも楽しめる！
音楽を身体に染み込ませ楽しい時間を！！

マクレーン有美

日本大学芸術部・声楽科に進学。大学で声楽の他、ピアノ・フルート・チェロ・音楽理論を学ぶ。
大学卒業と同時にクラシックから離れクラブイベントを中心に音楽活動を開始。趣味のツーリングがこうじて、バイク雑誌のツーリングライター、モデルとして日本各地を旅する。その後日本のみならず、その興味は世界各国へ。ニューヨーク、ロサンゼルスにてボイストレーニングを受ける。
子供の頃からの２つの夢。CDデビューと海外移住。念願叶って、2005年　shakabone　の初代ボーカルとしてavexよりメジャーデビューし、ボーカリストとしての才能を高くかわれる。翌年2ndアルバムリリース後、2007年12月、２つ目の夢、海外移住を開始！2009年秋、帰国。

座右の銘は『なりたい自分には必ずなれる！！！』
