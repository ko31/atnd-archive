　ソーシャルメディアとは、ユーザーが情報を発信し形成していくメディアです。

　ブログからＳＮＳへ、インターネットは情報伝達力や繋がる力が強く速く進化してきました。
　また、昨年から爆発的にユーザーが増加しているツイッターは、情報伝達スピードの速さと日本人との親和性の良さで今後も伸び続けると予想され、ネットのソーシャルメディア化が急速に進んでいます。

　これらの変化はインターネット黎明期の再来を感じさせるところがあり、変化を先取りし積極的にビジネスに活用していくことは、中小製造業者が勝ち残るための大きな武器になるはずです。

　本イベントは、わが国でも珍しい製造業者自身による、ソーシャルメディア・マーケティングの勉強会です。
　ツイッター、Ustream、ipadなどを最新トレンドを体感できる仕掛けが満載のこのイベントで、新しい時代をカラダで感じてみませんか？

（参加にはメールがやり取りできる程度の知識があれば大丈夫。必要なのは新しい時代にチャレンジする意欲です。）

「体験からソーシャルメディア活用の一歩を踏み出してほしい。」

ツイッター・IPhone・スマートフォン・IPAD等。
我々地方の中小製造業にも「ソーシャルメディア」の大きな波が押し寄せてきているのをひしひしと感じます。

この感覚はちょうど７，８年前のインターネットの波が押し寄せてきた時のような感覚とよく似ています。
７，８年前、我々中小製造業のインターネットに対する考えといえば。

「インターネット？？あーネットね。」
「小売業ならわかるけど、我々は商品を売っているわけじゃないから」
「インターネットで仕事なんてとれないよ」
なんて、当たり前に言われえていた時代でした。


それが、いまでは
「ホームページもっていないの？」
「ブログやってる？」
「ホームページ持ってなかったら信用できないよ」
というように、インターネットに自社のホームページを持っていない事で時代から取り残されたような感覚に陥ることさえあります。

まさに「やらないでは済まされない時代」にへと環境が変化しているのは皆様も共感していただけるのではないかと思います。

そして今。

「ソーシャルメディア」という新たな波が「モバイルインターネット」というインフラに乗って打ち寄せてきているんです。
また、同じような大きなパラダイムシフトが起きようとしています。

この変化もまた「やらないでは済まされない時代に」へと変貌を遂げていく事が予想されます。

我々地方の中小製造業の中では
「ツイッター？？スマートフォン？？ユーストリーム？？やってみたいけどよくわからない」
「近頃よくきくけど、身の回りにやっている人がいなくて、わからないんだよー。」
といった方は、まだまだ数多くいらっしゃるはずです。

そんな皆様に向けて
「ソーシャルメディア」が起こす変化の波を体験して頂こう！」
「体験からソーシャルメディア活用の一歩を踏み出してほしい。」
との思いからこのセミナーを企画させていただきました。

新たな時代の波を共に体験し、次代へと向けて一歩を踏み出していきましょう！
