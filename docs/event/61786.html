2015.7.23追記: 好評につき第2回目の開催決定しました! 詳しくは こちら

島根県松江市では、Ruby 関連の勉強会が豊富な反面、ネイティブアプリやウェアラブルといった技術に触れる場はまだまだ限られています。島根をITでもっと盛り上げたいフェンリルでは、島根支社開設二周年を記念し、島根のデベロッパーのための勉強会を開催します。

日々現場で活躍するフェンリルのエンジニアや、業界をリードするゲストスピーカーが登壇。Ruby、Swift、Apple Watch、ウェアラブル、AWS、開発手法など、ホットなテーマを数多く取り上げ、幅広いデベロッパーに楽しんでもらえる内容を予定しています。

テーマは “Beyond” です。いま現場で活躍している技術と、まったく新しい潮流の両方に触れることで、この時代の先にある世界を感じとってみてください。

  • 終了後懇親会あります! 参加申込は こちらから!

セッション概要

  • ウェアラブルや IoT 開発で田舎を盛り上げたい件について （ゲスト:村岡正和／バスタイムフィッシュ）
    日本ウェアラブルデバイスユーザー会の活動や直近のウェアラブルデバイスの話題を紹介。 また個人的なウェアラブル、IoT 開発やそこから得られた知見などをお話いたします。
  • Swift ではじめる iOS アプリケーション開発（木村友哉／フェンリル）
    Objective-C を知らなくても大丈夫。
    Ruby、Java、PHP 開発者にも分かる iOS アプリ開発の仕方を紹介します。
  • Ruby によるプッシュ配信機能の省エネ開発（福田寛之／フェンリル）
    Ruby による iOS, Android へのプッシュ通知実現方法を紹介。 最近増えてきているプッシュ通知のために使用したプラグインなどを取り上げます。
  • Fast Deployment Time By AWS（下田祥嗣／フェンリル）
    短期間で AWS に環境を構築し、アジャイルプロセスによる開発実績をもとに、オンプレではできない高速リリースのための活用方法を紹介します。
  • GitHub Enterprise による社内ソーシャルコーディング（田部祐樹／フェンリル）
    フェンリルで実践している GitHub を使った開発やコードレビュー、 作業の自動化など、最新開発手法を紹介します。
  • Apple Watch で出来ること出来ないこと（図子達也／フェンリル）
    アップルが発売する Apple Watch に関する内容です。 (※ 発売スケジュールによっては内容変更の場合があります)
  • 座談会・質問コーナー(スピーカー全員)
    質疑応答・フェンリル島根支社開設二周年について・島根を IT で盛り上げるには? など、いくつかのテーマについて話し合います。
  • 懇親会（終了後 18:00 より）
    セミナー終了後、懇親会を開催します。セミナー中に気になったけど聞けなかったことや、島根でITで盛り上げていくアイデアなど、いっしょに盛り上がりましょう! 参加申込は こちらの ATND から!

ゲスト紹介

村岡正和（バスタイムフィッシュ）
1975年生まれ。鳥取県出身。
神戸を拠点としてWebアプリケーションの開発、IT技術コンサルティングを行っている。
Webを有効活用したビジネスモデルの設計から、それを駆動させるためのアプリケーション開発、セールスマネージメントまで。事業の特性に合った最新技術を選定し、企業に導入するための指導を行っている。
次世代WEB技術であるHTML5に早くから着目し2008年より技術者向けの講演活動を開始。
全国各地でHTML5技術に関連した数多くの公演を行い最新Web技術普及に努めている。
HTML5-WEST.jp代表、HTML5Experts.jpメンバー、日本ウェアラブルデバイスユーザー会コアメンバー、神戸ITフェスティバル実行委員
