～日本Androidの会 四国支部 第９回勉強会 in かがわ～

Androidレイアウトの記述方法を教えます！

Androidアプリのデザインを描いても、自力で設定することができない！でもAndroidってEclipseじゃないといけないし、開発環境の準備しないといけないし…。XMLで書かないといけないらしいし、よくわからない！（＞＜）

そんなデザイナーの方が多いらしいという情報をキャッチした日本Androidの会 四国支部が遂に動いた！
やります！Androidデザイナー勉強会！

準備について：

  • ノートPCは持参してください
    （CPU Core2Duo以上, メモリ 2GB以上推奨）
  • 開発環境は事前に構築しておいてください
    • Windows 7(64bit版)の環境構築方法はこちら
    • MacOS Xの環境構築方法はこちら
  • Android実機を持っていない人は、エミュレータの設定を行ってください
    • Win エミュレータの設定の仕方（ページ中盤より）
    • Mac エミュレータの設定の仕方

午前の部

  1. Androidのレイアウトの種類の説明(HTMLっぽく)
    • レイアウトはdivタグのように使え！
    • 押さえるべきレイアウト2つ。
      1. LinearLayout
      2. RelativeLayout
  2. LinearLayoutだけでここまでできる！
    • 横方向に均等配置
    • 縦方向に均等配置
    • スクロールするViewの配置
    • フッターを作る方法
  3. 中央揃えしたければRelativeLayoutを使え！
    • 画像を中央表示するときに便利
    • Viewを重ねるときなども便利

午後の部

  1. 実践！XMLでレイアウトを書く！
    • 実在するアプリ「コスメマネージャー」のトップを作ってみる
    • まずはレイアウトファイルだけを編集
  2. スタイルを使ってみる
    • XML属性を使い回してコードをスッキリ
    • 実践！XMLでスタイルを定義！
  3. テーマ
    • アプリに統一性を持たせる
    • マニフェストで指定する
    • 実践！XMLでテーマを定義！

備考など（必ず読んでください）
