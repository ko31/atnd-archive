開催情報

日時: 2011年 6月18日(土) 15:00 〜 19:00（予定）
会場: 情報科学類ラウンジ （筑波大学 第三エリア 3C棟 2階 3C213）

概要

筑波大学第三エリア辺りで開催している，
情報科学類(情報学類)生を中心とした
有志によるプレゼン発表の練習場です．

発表のテーマは情報，計算科学，数学，工学など，
技術や理科学的なものが主で，内容は自由，レベルは問いません．
この会議の主な目的は参加者の発表練習と交流を行うことですので，
どうぞお気軽にご参加下さい．

当日の発表者(TwitterID)と内容

次の順番で発表を行います:

1. @Flast_RO (未定) 2. @tadanotokki (未定) (15分休憩) 3. @pi8027 「PostScriptメタプログラミング入門」 4. @erutuf13 「Coqで数独」
  • 座長: @t_teruya

※順番と内容は未定です．決定次第更新します．

一つの発表時間は「20〜30分+質疑応答」になります．

アクセス

  1. TX つくば駅を出てつくばセンターへ
  2. 6番のりばから「筑波大学循環(C10)」に乗車
  3. 「第三エリア前」で降車(つくばセンターから“右回り”に乗って約10分)
  4. バスから降りたところから大学の内側を向く(ほぼ東の方角)と，茶色の最も高い建物(12階建て)が見えます．その股をくぐった目の前の建物が「3C棟」です
  • “路線バス系統別時刻表（つくばセンター<つくば駅>発着系統）｜関東鉄道オフィシャルサイト”:http://www.kantetsu.co.jp/bus/rosen/timetable/tsukuba.html

「第三エリア前」のバス停の近くに地図があるので，それを見て現在位置と「3C棟」の位置を確認すると，より確実に会場を目指すことができると思います :-)
