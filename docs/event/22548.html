小樽市内でIT事業に従事する方々による飲み会を開催！

小樽のIT活性化、情報交換などを目的としています。
今回で第4回目の開催になりますが、今回は忘年会も兼ねて開催したいと思います！
初めての方もどうぞお気軽にご参加ください！
開始時刻、場所など随時掲載していきますので、よろしくお願いします。
（会費は割り勘・コース設定等で考えています）

小樽市内にお住まい/お勤めじゃなくてもOKです。

小樽にお住まいじゃなくても、小樽・ITにご興味のある方なら参加OKです。ぜひぜひ！
また、当初趣旨として「小樽でIT関連に関わっている人ってどれぐらいいるの？」から始まっているので、「小樽に住んでるけど、それ以外の場所で働いている」や「小樽で働いているけど、それ以外の場所に住んでいる」「いや、小樽に住んでもいないし、働いてもいないけど、小樽や小樽のIT業界あるいは関わっている人達に興味がある。早い話がこれを口実に小樽で飲みたい」なども全然OKです！

会場決まりました！

今回は「焼肉　しちりん」さんにお邪魔します！
120分食べ飲み放題4,480円を予定しております。
（90分3,980円ってのもありますが、500円なら良いですよね^^;）
所在地等はマップをご確認ください！
