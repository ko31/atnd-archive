イベント概要

「Cloud Express Roadshow」は、AWS のエコシステムを支える各都市にオフィスを構えるコンサルティングパートナー、テクノロジーパートナーとAWS が一緒に地方にて開催するプログラムで、四国では今回が初の開催となります。

本セミナーのプログラムはは2018年5月30日〜6月1日に開催され参加者2万人以上という大盛況のイベント [AWS Cloud Express Roadshow 2018] の内容をAWSの担当者が全国に届けることを目的としています。

また、セミナー終了後にはAWS担当者ととコンサルティングパートナーである株式会社えむぼまが、AWSの導入、運用支援についての様々なご相談や質問にお応えいたします。

みなさま、この機会にぜひご参加ください。
