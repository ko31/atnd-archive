ActionScript3.0開発オンライン会　第0回の続きです

皆さん始めましてツイッターアカウント@sakuriverと申します。

直近のお仕事だと、ソーシャルゲームでFlashlite1.1を触っていました。
また、お仕事でPHPを触ることが増えてせっかくなのでAS3でPC用ゲームを作ろうと活動しています。

毎週水・土の2回実施で、いつから入ってきても各自が自由にやりつつわからないところは質問したり調べたりしながら作っていこうというスタンスです。

時間は夜9～11時を予定しています。

ツイッターのハッシュタグは#AS3DEVでやります

AS3を勉強してみたい人や、Webでちょこっとやっているという人も歓迎です。
自分は過去に4回実施した勉強会で作った素材を使ってゲームを作っていく予定です。

企画が思いつかない方はキャラクターや軽い企画書であれば、そちらをお渡しするのでそれを改良なり何なりして作ってみましょう

＊単純に企画書だけとか、ネタを提供したい方も絶賛募集中です。
