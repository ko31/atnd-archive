大病院から愛を込めて

どうも、不肖コマチ只今入院しております…まあもうすぐ退院なんすけどね。右足以外は元気なので、3食しっかり食べて積ん読を消化し、飽きたら院内のコンビニに揚げパンを買いに行き…回診時の医師陣に漂う(この人もう退院してもよくね？)という空気が居た堪れない今日この頃、皆様如何お過ごしでしょうか。

2016年は遅めの立ち上がりとなったドリミではございますが、巻き返すようにもう夏の回。例年通り全体的に夏っぽくやります。まあ何処がって言われるとアレですけど。あと、人によっては冬の認識になってる曲が流れる場合がありますが、そこら辺各人の裁量に委ねてるので何卒ご了承ください。まあなんだ、どうせアガるじゃん？
とゆことで好事家の皆さまに於かれましては、舞浜で水浴びなど興じてからお越しいただくとよいのでは。またそうでない方も、街遊びがてらにお立ち寄りいただけますと幸甚に存じます。何卒〜！

ドリミとは何ぞや

…と思ってる方も少なくないかと。前回に続き今回も説明致します。
世界中に溢れる「D系音楽」の沼からドリミクルーがそれぞれ独自の目利きで掘り起こし、キュレーションを経てお届けする事により、好事家もそうでない方でもD系音楽の新たな魅力に気付いてもらいお楽しみいただこうという主旨のイベント。基本グルーヴ解釈(実際できてなくても)で脱・プレイリスト！あと「D系は飛び道具ではない」をスローガンに、あちら側に警鐘を鳴らしたい…(いつから決まったんだよ)ということで徹頭徹尾D系オンリー。年に一度はスペシャル回として、特段にビッグなゲスト様をお招き致します。本年度も画策中！

今回の企画

●山の日を我々なりにお祝いしたい
学生さんにはそれ程恩恵ないものの、社会人的には喜ばしかったりそうでもなかったりする新しい祝日「山の日」…今回はそれに因んで、各人の持ち時間内(研究生は自由参加)に必ず最低1回山の曲が流れます。なんだそりゃ。

●ぼくらのかんがえたさいきょうのDフェス2016
昨年開催時好評を博した(のか？)フェス企画、D系でフェスといえばコレジャナイでおなじみのあの曲を口火に、今年は研究生全員が手分けして担当します。何があっても所定時間経過であの曲リプライズにより強制終了。今年はどんなフェスになりますやら…乞うご期待！

●レギュラーコーナー
・8月が誕生月のお客さんをこぞってお祝いします。
・今や過去の遺物となってしまった、あの丸が重なった形の「光る棒」…ご自宅に眠ってたら持ってきてください。だんだん本数が減って寂しいのう。高円寺に光の海アゲイン…とはいえ万が一上手くいかなくてもご笑納ください。
