毎年恒例入力メソッド飲み会です。
毎年懇親会の参加希望を書き忘れる人がいるので、今年はデフォルトで懇親会参加希望とします。懇親会のみの参加、あるいは昼の部のみ参加の人はコメントに残してください。昼のセッション（ライトニングトーク含む）で発表ネタがある方は、Google Groups 等で小町まで連絡ください。どうぞよろしくお願いします！

以下のような方々の参加を特に歓迎します。

  • 過去の IM 飲み会に参加されたことのある方
  • 過去の入力メソッド関係のイベント（uim conference, Workshop on Advances in Text Input Methods 等）に参加されたことのある方
  • 現在入力メソッドの開発をしている、または過去に入力メソッドの開発に携わった経験のある方
  • 以上の方からの紹介

昼のセッションの会場は常時施錠されており、入り方は Google Groups の im-core で告知するので、Google Groups に入っていない人は小町まで連絡ください。その際、本名あるいは紹介した人などの情報をリクエストに含めてください（アカウント名だけから分からない場合があるためです）。
