description

2012年5月30日は2011年12月に天王洲銀河劇場で行われた坂本真綾LIVE 2011「in the silence」の発売日でもあり、最新アルバム「Driving in the silence」をはじめ、3枚のコンセプトアルバムを引っ提げての前代未聞のLIVE SHOWだったことは銀河劇場に参加された方にとっては記憶に新しいかと思います。


そのライブ映像が5月30日に発売されるとのことで前代未聞のLIVE SHOWを映画館クラスの大画面で、5.1chによる大迫力サラウンドで、去年行われた銀河劇場のような座席に座って坂本真綾LIVE「in the silence」を堪能することを一夜限り再現したいと思います。


前代未聞のLIVE SHOWを真綾さんファンの方でしたら、お一人でもグループでも参加が可能です。


なお、今回のイベント開催にあたっては一個人での使用となり、全く営利目的での開催ではありませんのでご了承願います。


・日時：6月2日 (土)  18:00〜21:00頃 (受付は17:30〜)


・会場：参加される方のみお伝えします (山手線沿線内です)


・参加費：1500円以下 (参加人数によって変わります。)
(飲食はありませんのでご自由にご持参ください。)


・定員：参加者5名から開催決定 〜 40名くらいまで


・その他：21:00〜希望者の方のみ二次会を開催します(3000円前後)


主な内容
・5月28日(月)の時点で参加者5名以下の場合は開催を中止となりますのでご了承願います。
・前代未聞のLIVE SHOW＋αを約3時間に渡ってお楽しみいただけます。
・飲食はご用意してませんので各自持参していただいて構いません。
・参加される方は何でもいいのでグッズを持ってきてください。
(in the silenceパンフレット片手に観るのも良いでしょう。)

・モラルに反する行為や迷惑行為、不快感を与える行為等についてはご遠慮願います。

・注意事項については更新を行ないますのでご了承願います。
・質問などに関しては主催者までメッセージをお願いします。
