description

　日本人に根付いている「完璧な英語でないと話したくない」という考えから脱却しましょう。そして、拙い英語でいかに相手に伝えるか、高校生レベルの英語でも堂々と主張するマインドの獲得を目指すセミナーです。意欲さえあればどなたでも参加出来ます！

　チーム毎のロールプレイングも交えながら、ITの現場で使われる生の英語を、シチュエーション毎に基礎からわかりやすくトレーニングします。 　ネイティブの講師からも講義を受けられるため、実践的な英語を学ぶことが可能です！

　会議編では、セミナーの最後に英語でミーティングができるようになることを目標とし、会議でよく使用される表現等を学んでいきます。 また、セミナーの中では、効果的なミーティングの準備方法、実施方法、ミーティングでの役割分担についても学び、英語だけではなく普段のビジネスの中ですぐに活きる考え方を身につけることが可能です！
