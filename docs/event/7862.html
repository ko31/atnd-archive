Mashup Camp ってなに？

Mashup Camp は、参加者それぞれノートPC を持ち込んで、 みんなでわいわい開発しちゃおうっていうイベントです。
キャンプという名前ですが基本的に日帰りです。
MA5 では Hackathon という名前で1度だけ開催しました。

MA6では、たくさんの人に気軽に参加してほしいという思いから Mashup Camp という名前にし、定期的に開催していきます。

推奨テーマ

本 Mashup Camp では、次のテーマを 推奨 します。

  • ゲームを作ろう
    • みんなで楽しいソーシャルゲーム(アプリ)を作ろう
    • スマートフォンや携帯電話ならではのゲーム(アプリ)を作ろう
  • ガジェットを作ろう
    • 生活を便利にするガジェットを作ろう
    • 仕事に役立つガジェットを作ろう
  • 協力会社が賞をあげたくなる作品を作ろう

プラットフォームやウェブサイト、PC やモバイル(スマートフォンや携帯電話)などの動作環境や提供方法などは定めません。自由にアイディアを練って、自由に作品として表現してくださいね。

推奨 API

本 Mashup Camp では、次の API を 推奨 します。 これらの API を使わず、他の API を使ってもかまいません。

モバゲーオープンプラットフォーム
http://ma6api.mashupaward.jp/entry/70712/
モバゲーオープンプラットフォームおよびモバゲーAPIは、DeNAがソーシャルゲーム開発・提供で培ってきたノウハウをもとに開発されました。OpenSocialに準拠しており、154社、350アプリが公開されています（7月27日時点）。

OpenSocial Host JavaScript API
http://ma6api.mashupaward.jp/entry/70576/
mixi アプリ、モバゲーアプリ、CREYLE アプリ等、OpenSocial アプリケーションの開発／管理を支援する、多数の API を提供しています。

HeartRails Capture API
http://ma6api.mashupaward.jp/entry/70579/
任意のウェブサイトのサムネイル画像を簡単に取得できる API です。サムネイル画像のサイズを自由に設定できる他、影付きや縁付き、角丸等の装飾を施すこともできます。

HeartRails Express API
http://ma6api.mashupaward.jp/entry/70578/
線／駅名／最寄駅等の地理情報を XML、JSON （P） 形式で取得できる API です。Google Maps 等の地図サービスと組み合わせてご利用いただけます。

HeartRails Graph API
http://ma6api.mashupaward.jp/entry/70577/
キュートな円グラフを簡単に作成できる API です。レビュー記事やアンケート結果にいかがでしょうか？

天気予報API
http://ma6api.mashupaward.jp/category/up_frontier/
「天気情報API」は、天気予報をはじめ、花粉情報や紫外線情報、洗濯指数、ビール指数など様々な指数情報を、ウィジェットやmixiアプリ、ブログパーツ、Flashなど様々なアプリケーションで利用するためのAPIを有償で提供するWEBサービスです。

mashmatrix OpenSocial Dashboard
http://ma6api.mashupaward.jp/entry/70684/
ブラウザ上で OpenSocial ガジェットを自由に配置し、互いに連携させることが可能なダッシュボードサービスです。開発者の方は、標準仕様にのっとったガジェットを開発するだけで、さまざまなWebサービスやガジェットと組み合わせてマッシュアップすることが可能です。

infoScoop OpenSource
http://ma6api.mashupaward.jp/entry/70692/
nfoScoop OpenSourceは大手企業で実績のあるオープンソースのOpenSocial対応パーソナライズドポータルです。ガジェットを組み合わせることで、様々なサービスを組み合わせることができます。特にMASHUP AWARDS向きの機能として、簡単にOAuth対応ガジェットを作ることができ、さらにpubsub APIによりガジェットとガジェットを簡単にマッシュアップすることができます。

OpenSocial Pages
http://ma6api.mashupaward.jp/entry/70672/
OpenSocial Pages は Google Apps と連携して動作する OpenSocial プラットフォームです。Google Apps のユーザとグループを活用して、同僚とコラボレーションするビジネス向けの OpenSocial アプリケーションを開発できます。開発したアプリケーションは、PC に加えて、iPhone や iPad、Android といったスマートフォンからも利用できます。
