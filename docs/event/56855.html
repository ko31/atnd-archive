description

これから20年間働くイメージがありますか？

これまで「長崎 ITキャンプ」と銘打って2009年から4年に渡って4回、「IT系業種」にフォーカスしたワークショップ型イベントを行ってきました。
しかし、現在の「職場」はIT系かどうかに関わらず、ほとんどの場で当たり前のようにPCやインターネットが使われています、むしろ「ITスキル」は「IT系業種」にだけ必要なものではありません。
また、ITに理解がない事による弊害もかなり多く散在し、早急にいろいろな人へ理解を広げる必要性があることにも気が付きました。

そこで、今回は 「IT Business BootCamp 2014」 と題し、これから20年間働くイメージを個々が持つことを目的としたイベントを開催することにしました。

IT系に興味がある人達には、ITとそれに関わる「非IT系」の話を。
IT系に興味がない人達にも身近なところにある「IT」の重要性を。

そんなことを伝えられたらいいと思っています。
