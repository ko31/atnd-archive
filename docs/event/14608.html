府中公園でサクラを撮るぜ。

写真に興味があるとか、何となく暇だから、暖かいので外でてみよっか。
などなど、ゆるい感じで自由にサクラを撮る活動です。
現地までまとめて車で移動して、サクラ撮りまくりましょう！
自分の世界に入れる人大歓迎！公園までまち歩きしながら撮ってもいいよ。

カメラ＆菓子持参・基本自由・みんなカメラ初心者だから！

そんな感じでほのぼのと行きましょう。普通に日向ぼっこしてもいいよ。
当日は色んな花見イベントと、かぶりまくりなので、短時間でハイフォーです。

現地集合（府中公園）

当日の時間の都合や、撮影場所も花見混雑が予想されること、
また他の花見イベントとの、重複もあるので短時間集中型です。

雨天中止

雨が降ったら中止します。その時はほんとにゴメンナサイ。

ポーケン持ってる人は、ハイフォーしましょう。

事前にPOKENのパソコン同期や、電池交換とかしといてね！
