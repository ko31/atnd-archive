description

Swift 言語のプログラミングの勉強会です。

東京では盛んな勉強会ですが、地元ではなかなかなくて、かねてより地元で小さな勉強会を開いてみたいと思っていたので、思い切って開催してみることにしました。

横浜市青葉区、青葉台という駅から徒歩５分以内のところにあります。

勉強会

目標は「カジュアルに Swift プログラミングを楽しみたい」です。

参加者の中から Swift に関する話題を募って、それについてみんなで談笑しながら学びを深める勉強会です。少人数で気兼ねなく語り合えたら理想的です。

言語仕様に関する話題が多いですけど、iOS/OSX/Server Side Swift とかの事柄でも、言語の仕組みに関することでも、別の言語のとある考え方を Swift に取り入れると面白そうでも、Swift 言語にまつわることならなんでも、何かをみんなと共有できたらいいなと思ってます。

対象

プログラミングが好きな人ならどなたでも歓迎です。あまり気張らず、みんなと語らい・ときに相談できる場所であれたら理想的です。講習会ではないですけれど、プログラミングが好きな 初心者さんも大歓迎 です。

持ち物・会場設備

必須ではないですが Swift を動かせる環境があると、より有意義に過ごせるかもしれません。

  • MacBook / Note PC
  • Mac OS X / Ubuntu Linux
  • iPad / Swift Playground
  • Xcode / Swift Snapshot
  • IBM Swift Sandbox

飲み物は、会場のすぐ手前に 100 円の自販機があります。

参加費

  • 当日、参加費として 1,500 円 を頂きます。

タイムテーブル

発表時間と人数枠は完全に目安です。ゆっくり語らいながら進められるように余裕を持って設定しています。

時間 内容 話題提供 テーマ
13:30-13:45 準備・談笑
13:45-14:00 自己紹介タイム
14:00-15:00 発表・談笑 @es_kumagai AnyObject について！
15:00-16:00 発表・談笑 tokyoyoshida さん さよなら Open Swift ！
16:00-17:00 発表・談笑 （どなたかが発表！）
17:00-17:30 みんなで談笑
17:30-17:50 片付・談笑

終了後に希望者を募って懇親会も行きたいなと思ってます。

話題提供者の募集

参加者の中から、この会でみんなと共有する話題を提供して下さる人を募集中です。

たっぷり１時間ほどの枠にしてありますけど、発表時間は 数分から１時間まで 好みに合わせて使ってください。しっかり整った「発表」にこだわらなくても『ここまで出来たけど後が分からなかった』でも『こんなことを思ったけれどどうだろう』みたいなのでも OK です。質問を１つや２つ持ってくるだけでも、みんなとけっこう談笑できる感じです。

未完成でも、初歩的なお話でも、他の勉強会で話したことでも。他の人と同じテーマでも大丈夫ですので、興味のある人はお気軽に下のコメント欄やツイッターなどでお知らせくださいね。

話題提供枠はあくまでもしゃべれる優先枠ですので、表明されてなくても話題があれば当日にでも聞かせてください。自身が興味ある話題ならそれだけ有意義な時間になると思いますので、ぜひぜひ活用くださいませ。

交通

東京方面の方は、渋谷駅から田園都市線、長津田・中央林間行きの電車でお越しください。急行電車も止まります。所要時間は、急行なら 25 分ほど、各駅停車で 38 分ほどになります。
