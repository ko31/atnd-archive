□主旨
人口減少社会の日本。都会では急激な高齢化、地方では消滅可能性、このままではいずれ破綻することは明白です。まちづくりだけでもダメ。ひとづくりだけでもダメ。からだづくりだけでもダメ。ここはひとつ、多分野で協力して、乗り切ろうではありませんか。
「実践！健康のまちづくりアカデミー」では、
健康のまちづくりに関する机上の理論学習と地域での実践学習の融合
地域づくり（社会学）、まちづくり（工学）、ひとづくり（教育学）、からだづくり（医学）などの多分野にわたる広域多職種協働/連携教育
により、まちの気持ちがわかるまちの救世主、つまり、健康のまちづくりおよび地域存続可能性の向上に寄与する人材を、効率的・効果的に育成して参ります。
 
□対象・参加資格・定員
Ⅰ．全国の学生および（若手）専門職　※定員１５名
①地域づくりに関心のある社会学系学生および専門職
②まちづくりに関心のある工学系学生および専門職
③ひとづくりに関心のある教育学系学生および専門職
④からだ（健康）づくりに関心のある医学系学生および専門職
⑤とにかく地域・まち・ひと・からだづくりに関心のあるあらゆる学部の学生および専門職
Ⅱ．全国の行政関係者　※定員５名
地域づくり、まちづくり、ひとづくり、からだ（健康）づくりに関心のある行政関係者
 
※受講後も引き続き無理のない範囲で高浜町の健康まちづくりや本事業に関わって下さる意思のある方
 
□履修による高浜町からの「健康のまちづくりプロバイダー」認定について
Ⅰ．「健康のまちづくりプロバイダー」の認定（ファースト・ステージ）
１年間のコース履修により、高浜町より「健康のまちづくりプロバイダー」として認定いたします。
Ⅱ．「健康のまちづくりプロバイダー」の活動（セカンド・ステージ）
２年目より、実際に高浜町を舞台に健康のまちづくり活動を実践し（自由参加・無期限）、また、その効果の評価も積極的に行います。年ごとに参加者・功労者を認定／表彰いたします。
