クラウド電話API Twilioに興味があるエンジニア向けイベントです。

Twilioに興味を持っているエンジニアの皆さんが集まり毎回ハンズオン形式で開催しています。
初心者の方はもちろん既にTwilioを使っている方々、Twilioをテーマで情報交換ができればと思います。

■ クラウド電話API Twilioとは。
・電話とウェブをつなぐ、クラウド型の電話APIサービスです。
・アプリケーションにAPIを組み込むだけで、電話機能を利用することができます。
・Twilioはクラウド型サービスなので、 API はいつでも利用可能、継続的にアップグレードされます。
・多額のハードウェア投資や長期契約は不要、使った分だけのお支払いで必要なだけスケールアップが可能です。

■ Twilio API 勉強会でのLTについて
参加する皆さんの時間です。Twilioの簡単な使い方でも構いませんので、是非みなさんのお話を聞かせてください。
当日申請もオッケーですし、事前応募も可能です。気軽にお声かけください。

■ 今回の勉強会では。
・Twilioの各名詞や動詞の使い方
・Twilio API の開発についての情報共有が目的です。
・Twilio APIを直接触りながら（コードを書きながら）Twilioを試します。
・LTも用意されていますのでお楽しみにしてください。
・AWS様より AWS利用券 $25を参加のみなさんにプレゼント致します。
・勉強会では個人のサーバーの利用も問題ないですが、AWSをセットアップしTwilioを試してみませんか？
・AWSエンジニアの皆さんよりAWSのセットアップについてサポートしていただけます。
