★講演内容

■■講座の特徴■■

レバテックキャリア WEB Core コース

WEBのスキルアップに特化した
UX・UI、マークアップ、プログラミングが、
無料で学べます！

幅広くより高度なスキルをもったWEB業界のゼネラリストを育成します。
さらにWEBデザイナーとしての経験と強みを活かし、
デザインとマークアップのスキルアップや、
ダイナミックページへの理解を深めることで、
さらなるキャリアアップをサポートします。

少人数制ならではの、マンツーマンスタイルで
スキルを習得できるチャンスです！

■■転職成功者特典■■

本学校を卒業し、レバレジーズを通して転職が成功された方には、
受講料分の金額をキャッシュバックさせて頂きます。
※別途弊社サービス利用規約もございます。

★講演者プロフィール

西澤 修二　氏
shuji nishizawa

工学部卒業後、IBM関連会社でエンジニアとして就業。
インターフェイスデザインとシステム開発に携わる。
その後、専門学校やクリエイタースクールにて、
カリキュラム開発や講師としてクリエイター育成に携わりながら、
さまざまなメディアのコンテンツ制作を手がける。
2011年、target media communicationsを起業。

Webメディアをコントロールするスキルを習得することにより、
きっとあなたの仕事の幅は飛躍的に広がるでしょう。
あなたの新しい一歩を踏み出すお手伝いをさせていただきます。
