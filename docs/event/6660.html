description

シュールストレミング１缶（300g）を手に入れました。
東京湾大華火祭が開催される8/14(土)の昼から、葛西海浜公園にて開封します。

ご参加者は以下をお持ちになり現地にご集合ください。


  • 会費：500円/人

  • 各自、シュールストレミングと一緒に食べたいもの（付け合せ）をお持ちください。
    例：
    • パン
    • 茹でてスライスしたポテト（またはマッシュポテト）
    • 赤タマネギのみじん切り
    • ゴートチーズやサワークリーム、バターなど
    • ビール

集合場所：葛西海浜公園内（海側の島になっているところ）
開催日時：2010年8月14日(土) 12:00-
会費：500円（付け合せは各自持参）
連絡先：Twitter: ch1cala
facebook: Chikara
