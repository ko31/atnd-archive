description

HTML+CSS+JavaScriptによるアプリ開発ができるというPhoneGap。モバイルアプリケーションのマルチプラットフォーム対応を容易にする、ハイブリッドアプリ開発のためのフレームワークとして注目を浴びています。
そのメリット・ディメリット、パフォーマンスや移植性など、実際に触ってみたて感じたもろもろを発表します。
PhoneGapやハイブリッドアプリに興味がある方は、お気軽にご参加下さい。

※多数のご応募ありがとうございます。今後の補欠者の増え具合で定員増を検討致します。ご希望に応えられない場合もございますが、ご希望の方はひとまずご登録ください。
※定員を１６名にしました(2013/07/23追記)
