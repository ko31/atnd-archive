Rfm Code Reading #16

2012年4月21日（土）午前10時よりRfm（FileMaker API for Ruby）のソースコード読書会「Rfm Code Reading #16」を開催します。

Rfm Code Readingの概要

FM-Tokyo（FileMaker Pro 東京ユーザーズミーティング） の分科会活動の一環として、Rfm Code Readingでは、Rfmのソースコードを読み解くことにより、FileMaker ServerのカスタムWeb公開やRuby、およびRuby on Railsを使った開発のスキルアップを目指します。今回はFileMaker 12 製品ラインが発表・発売されたので、FileMaker Server 12 XML APIの変更点について確認していく予定です。

RubyおよびFileMaker ServerのカスタムWeb公開に興味のある方のご参加をお待ちしています。
