謙信IT勉強会

中小企業、小売店等を対象としたネット活用の勉強会です。特に決まったテーマはなく、各自が持ち寄った問題や新しいサービスの習得など自主性を重んじた勉強会です。
なかなか一人では解決できないITのギモンをみんなで集まって解決してみませんか？

過去に実施したテーマ

  • HTML + CSSを使ったホームページ制作
  • Movable Type、WordPress、JimdoなどのCMS活用
  • Googleサービスの活用
  • Twitter、Facebookなどのソーシャルメディア活用

など。

なお、セミナー参加企業の情報を集めたポータルサイトも運営しております。
→ 謙信eあきない
