濃！Know！NoSQL ～春の陣～

満を持して大阪開催！！

KVSなど最近注目を集めるNoSQLをテーマにした勉強会/セミナー”濃！Know！NoSQL！”
第一回の東京開催では増員！さらに増員！の大反響を頂きました。
最新動向、活用方法など、最先端で活躍されている方に登壇して語っていただきます。
主に開発者、エンジニア、企業情報システム部門の方向けのセミナーとなります。お気軽にご参加ください！

大阪梅田　毎日インテシオ4階　E会議室

予定プログラム

1. ＜ゲーム開発現場から見た、NoSQLを使う上での苦労話や期待すること＞
　　株式会社インテリジェンステクノロジー　取締役　笠原 大輔氏

　　　　KVSをはじめ、様々なNoSQLを使ってみて苦労したことから
　　　　”もっとこうならいいのに”や”まだまだこんなことやってみたい”まで、
　　　　ゲーム開発現場からお伝えします。

2. ＜エンタープライズシステムでも活躍する「ReflexWorks」の紹介＞
　　有限会社バーチャルテクノロジー　代表取締役　竹嵜 伸一郎氏

　　　　エンタープライズ利用するアプリケーションではトランザクションが必須だったり
　　　　容易なスケールアウトが要求されたりします。
　　　　ReflexWorksが解決してきたエンタープライズな課題を、
　　　　事例やデモも交えながらその仕組みについて説明します。

3. ＜KVSを使った web システムパフォーマンス改善手法を一挙に紹介！＞
　　株式会社 神戸デジタル・ラボ　先端技術開発事業部　山本 浩生氏

　　　　サーバの高負荷によるパフォーマンス劣化へどう備えるのか。
　　　　分散 KVS を導入することでシステム全体に大きく修正することなく
　　　　容易に負荷対策が実現できることを、事例やデモを交えながら説明します。
