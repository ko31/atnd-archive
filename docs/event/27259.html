JavaScriptWEBフレームワーク
概要：
　"Knockout.js"をご存知ですか？Knockout.jsは最近注目されている、高機能なオープンソースのJavaScriptのUIフレームワークです。HTML/CSS/JavaScriptであたかもクライアントアプリケーションを作るかのようにUIを構築することが可能になります。
　今回はこのKnockout.jsについて、コアコンセプトである『宣言的バインディング』『自動UIリフレッシュ』『依存性追跡』『テンプレート』の基本をお話します。
　Knockout.jsをうまく活用するとコードの生産性とメンテナンス性を向上させるだけでなく、デザイナさんのHTML/CSSの編集とプログラマさんのJavaScriptのコーディングの分業の作業効率を上げる事も可能です。
　より効率的なWebアプリケーションのUIコーディングをお求めの方はとりあえず聞いてみてください！
　また、最新のASP.NETの状況についてもちょっとだけお話します！（実は、Knockout.jsはASP.NETの今後にも大きくかかわってくるかも！？）		 きよくらさん(@kiyokura )
