TDD Boot Camp 札幌 1.5

TDD Boot CampはTDD（テスト駆動開発）を実際に体験しながら学ぶイベントです。
これまで東京を初め北陸や名古屋でも開催されており、札幌で2011年1月に開催しています。
そして、TDDBC札幌2.0が6月4日に開催される事も確定しましたが、その前に少し予習をしたい！という要望に応えるべく、TDDBC札幌1.5を開催します。
TDDBCでは多くの事を学び感じることができますが、それだけでテスト駆動開発を自分のスキルにすることはできず、何度も繰り返して実践していく必要があります。
TDDBC札幌1.5は、TDDBC札幌02の前哨戦としてJavaを使いユニットテストに慣れる事を目標としたイベントです。

日時

2011年5月15日（日）10:00 ～ 16:30

タイムテーブル

09:30〜10:00 開場
10:00〜11:30 テスト駆動開発とユニットテスト（講師 @shuji_w6e）
11:30〜12:00 チーム作り、お題の発表、テストリストの作成
12:00〜16:00 テスト駆動開発演習（途中でお弁当）
16:00〜16:30 振り返り

・時間配分等は目安です
・演習は３〜４名で１チームとします。
・演習は１台のPCを使って、TDDの１サイクル毎に実装者を交代する乱取り形式で行います。
・演習ではノートPCを使用しますが、３〜４名に１台あれば良いので全員が持ってくる必要はありません。
・演習で対応する言語はJavaのみです。
・Javaの基本構文が解り、数十行程度のプログラムが書ける程度のスキルがあれば参加可能です。

後半ではデータベースと接続してのテストを行うなどの予定です。
