【紹介するサイドビジネス】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
内容が不明確なセミナーが多いですが、参加してみた結果、既にご存知でしたら大切な時間を無駄にしてしまいますので、概要を記載致します。
ここに記載のできない貴重なお話も盛り沢山ですが、まずはこの中でもご興味のあるものがあれば、ご参加ください。

●座談会・アンケートで収入
企業がマーケティング目的で行う座談会やアンケートに参加する方法と
どのようにすれば、オファーを貰えるのか。案件の多い座談会業者を具体的に紹介。
（座談会は時給4000円～8000円程度で1～2時間程度のものが多いです。）

●ネットオークションでの転売・せどり
難しいインターネットの知識はいりません。
商品の見つけ方、出品のコツやノウハウ、具体的な商品の紹介、ネタの見つけ方などをご紹介
（3,000円～10万円/月程度　→　やり方によっては上限なし）

●アフィリエイトの秘密
流行っているアフィリエイトやネットビジネス。
インターネット業界出身だから、業界のビジネスについてはかなり的確。
今まで試したけど上手く行かなかった方もその謎がわかります！

●転職・人材紹介
転職支援ができて、喜ばれた上で紹介料も頂けるお仕事です。
紹介先は大手や上場会社、ベンチャーなどもあり、職種は様々。
（採用決定で5～30万円/名 程度、面接時点で5,000円の案件もあり）

●代理店ビジネス（コスト削減がメイン）
市場よりも割引して提供できるため相手から喜ばれ、
自分も収入につながる様な案件を取り揃えております。

・通信関連の割引
・交通費の削減
・自動販売機の設置
・クレジットカード決済代行
他にも紹介できる限りの案件を紹介していきます。
（毎月利用されるサービスは、毎月の収入が見込めます）

●交流会・イベント・セミナー主催
「参加費×参加人数-会場費雑費＝収益」というシンプルな仕組みです。
（例：参加費4,000円×40人－会場費10,000円＝150,000円）

毎月のように案件は増えるので、その時旬なものをご紹介していきます！

海外にお住まい、海外ノマドで生活したい方向けの案件などもご用意！
しかしコチラは貴重な案件だけに、当日は詳しくはお話できませんが
概要をお聞きになり、ご興味ある方は個別にご案内する形になります。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

現在は、インターネット事業、代理店事業、人材紹介事業など、ストック型にこだわった収入口を複数持っている。
都内在住の30代後半の2児のパパです。
