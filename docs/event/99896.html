今回はハンズオン勉強会！
Firebaseに追加されたIn-App Messagingを使ってみよう!!

先日「Firebase」に多くの新機能が発表されたことはご存知かと思います。
中でもアプリを活発に利用しているユーザーへのメッセージ機能である「In-App Messaging」は、
ユーザー体験を高める手段として非常に価値ある機能だと考えています。

※ In-App Messaging参考：https://firebase.google.com/products/in-app-messaging/?hl=ja
今回はこのテーマを元に実装まで行おうというものです！

Firebase Japan User Group 村本 章憲 氏の登壇が決定しました！


村本 章憲 氏（@1amageek ）
半導体エンジニアからiOSエンジニアにコンバート、クックパッドでサーバーレス開発を推進し、
フルFirebaseのプロダクトを手がける。
独立後もFirebase Japan User Groupのオーガナイザーとして、精力的にFirebaseの普及に尽力している。

当日行うコンテンツ

・イベントに応じたメッセージの出し分け
・メッセージからの遷移（アプリ内の特定ページ、Webページなど状況に合わせた先へ）
・メッセージ自体のカスタマイズ（ポップアップ or バナー）

＜使うサービス＞
● Firebase In-App Messaging
● Firebase Analytics

＜言語＞
● Kotlin（Javaの方も歓迎！）

持ち物

・Android Studio3.1がインストールされたPC
・Googleアカウント（Firebaseを使用するため）
・Android端末（可能であれば）
※Wi-fiはありますが、セキュリティや速度を気にされる方はご持参ください。
