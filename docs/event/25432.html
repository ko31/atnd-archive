Monacaスマートフォンアプリ開発体験セミナー(午前)

Monacaは最新のWeb技術であるHTML5＋JavaScriptを採用した、誰でも簡単にアプリが開発できるクラウドサービスです。今回のセミナーでは、「Monaca」とその内部で使用されている「PhoneGap」に焦点をあて、その可能性についてお伝えします。その後、サンプルアプリの制作を通じて実際にスマートフォンアプリ開発を体験していただきます。

タイムテーブル

  • 14:30 受け付け開始
  • 15:00 はじめに(無線LANセットアップ・環境確認)
  • 15:15 講義開始
    ・Monacaによるアプリのビルド
    ・PhoneGapデモ
    ・リンク集アプリ
    ・ラブコールアプリ作成
    ・PhoneGapによる連絡先取得
  • 17:30 おまけ：iUI/zeptoの解説

※セミナー内容は余儀なく変更される場合がございます。

持ち物

・ノートパソコン
・スマートフォン(Android2.2以上)
