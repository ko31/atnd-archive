description

同じ世代の仲間と、お好み焼きをメインに料理を作りながら楽しい交流会をしませんか？
職場以外での新しい交流を目的に、知り合いばかりで気軽に、そして楽しく時間を過ごすことが出来る会にしたいと思います。

料理が得意な人はアピールチャンス到来です。みんなの前で自慢の腕前を披露しませんか？
みんなに得意の料理を振舞ったり感想を聞いて盛り上がりましょう。もちろんそれを食べにくるだけもオッケー！
メインのお好み焼きを焼いたり、料理上手が作った料理をみんなで食べながら、いろいろしゃべり合って過ごす．．こんな楽しい交流会になったら素敵です。
今回は、ちょっと一人で料理をつくるのは不安という人も大丈夫、ちゃんとサポートいたします。

料理が得意な人も、食べるの専門の人も、普段からお好み焼きを家でつくって食べて人から、いままで作ったことがない人も気軽に参加してください。みんなで楽しくつくって、歓談しましょう。
