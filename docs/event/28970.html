OSC北海道併催企画！Webラジオペースメーカー特別出張編

来るオープンソースカンファレンス 2012 北海道にて、Webラジオペースメーカー特別出張編の公開生放送をニコニコチャンネルで行います。

生放送ニコニコチャンネル「OSS.CH!」

→ http://ch.nicovideo.jp/channel/oss

オープンソースカンファレンス2012 北海道

オープンソースカンファレンスは、オープンソースの今を伝えるイベントです。東京だけでなく、北は北海道、南は沖縄まで全国各地で開催しています。
オープンソース関連のコミュニティや協賛企業・後援団体による、セミナーやプロダクトの展示などを入場・参加料が無料でご覧いただけるイベントです。

詳しくはオープンソースカンファレンス北海道のページをご覧下さい。

→ http://www.ospn.jp/osc2012-do/

入場について

・9時30分より整列（ATND申し込み順）
・9時45分より入場開始
