手をつないだ瞬間ホッとした経験、体に痛みがある時手で体をさすったことはありませんか？
これはココロをつかさどる【脳】と【手】に密接な関係があるからです。
「ハンドリフレクソロジー」を行うことによって、脳がリラックスするため、
体の緊張がほぐれ、心地よい安心感が得られるので興奮状態や不安感、痛みの緩和につながるといわれています。

本講座はそんな、ハンドリフレクソロジー手技とそれを伝える講師技術の体験会となっております。

イベント・セミナー内容

「力は弱く、体力も無いのにマッサージの様なことが出来るんだろうか？」
「簡単に身に着くと謳ってはいるけど、実際はもっと大変なのでは？」
「そもそもリフレクソロジーの効果が疑わしい」

様々な疑問を持って、是非お越しください！
講師・スタッフがお応えいたします。

  2. ハンドリフレで期待できる効果

  4. ハンドリフレの活用方法
  5. ハンドリフレの基本のテクニック（実践体験）

期待できる効果

手をつないだ瞬間ホッとしたり、体に痛みがある時手で体をさすったことはありませんか？
「ハンドリフレクソロジー」は手の「反射区」から「自律神経」を経て主要臓器や筋肉に働きかける、身も心もまとめて癒せる手技です。
本講座でその効果を体感してください。
