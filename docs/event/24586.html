■参加方法

  • デザイナ応募は、美大在学中、美大卒、美大受験、美大的、美大ノリな方に限定しております
  • 応募方法
    • 参加コメントに「スタートアップ応募」「デザイナ応募」「閲覧」「メディア」のいずれかを記入
    • 「スタートアップ応募」の方々は、以下までご連絡ください。詳細お伝えします
    • 「スタートアップ応募」は、主催者による内容判断の上、参加可否を決定します
      • みなさん、「はてぶ」「facebook like」「twitter tweet」に協力おねがいします！
      • 採用、営業目的の方は、美大系であっても参加することはできません

■『スタートアップ応募』の皆さまへ

  • 参加条件は、下記条件をみたしているスタートアップのみです
    • 本イベントで出会ったデザイナーに20万円お支払いください（実際にスタートアップのサービスに利用する対価となります）
    • 弊社への連絡をせずに、ATNDで『スタートアップ応募』と参加登録している方は参加できません。必ず弊社へ連絡ください。
    • 連絡後、弊社から参加許可がおりたスタートアップのみ参加可能です
    • 弊社への連絡なし、許可なしでの、デザイナー探しは禁止です
