第５回 アプリ【禁煙駅伝】のオンライン交流会（飲み会）を予定しました。
🍎🍏🍎🍏🍎🍏🍎🍏🍎🍏

アプリと共に、LINEアプリでお会いして励ましあえば、もっと禁煙が楽にできると思います。

2019/11/23(土曜)20:00〜22:00

※参加費は無料です
※途中参加、途中退場、いつでも自由です
※顔出しNGの方は無しまたは好きな画像で声だけでも参加OKです
※事前に私（運営）のLINEのIDをアプリ内タイムライン（またはメール）でお伝えし、登録してグループに追加させていただきまして開始します
※オンラインですので最大5名までとさせていただきます

アプリでコメントしたことのない方も、たくさん使っていただいている方も、アプリを使ったことのない方も
励ましあって禁煙を成功しようとされる方はみんな「仲間」です。

ジュースやお酒を飲んだり食事をしながら禁煙話で盛り上がりましょう♪

話題は豊富ですので退屈しません。
例：
・禁煙に何日目なのか
・何年吸ってきたのか
・きっかけ
・銘柄
・健康や家族のお話など
お話されたいことだけお話して、用事が出来たらいつでも入退室できるのでとても気楽な雰囲気です。

お気軽にご参加ください。

皆様とお会い出来るのを楽しみにしています。

※参加登録をする際につきまして
もしよろしければ禁煙駅伝でお使いのハンドルネームを記入いただけますと助かります。

※TwiterやFacebookのアカウントがなくここで予約出来ない方でも飛び入り参加OKです。
以下の方法でご連絡お願いいたします。
iOS: 禁煙駅伝アプリ→左上メニュー「三」→問い合わせから運営に「参加希望」と書いてメール送信
Android: 禁煙駅伝アプリ→右下アカウント→設定→問い合わせから運営に「参加希望」と書いてメール送信
