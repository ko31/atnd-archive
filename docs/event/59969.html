正規の方法で発作的にバイスサワーはあうめえしたくなる酒屑の集いです。

【参加要件】
・nrtt-住民、もしくは住民予備軍であること。
・20歳以上であること。

【会場】
・加賀屋　上野広小路店
・17時の開店と同時に入店します。がまんできねえ。

【集合場所】
・12/6（土）16:50ぐらいまでに、JR御徒町駅南口改札へ集合下さい。

【その他】
・予約はしませんので満席でも泣かない。
・二次会以降もあるかもしれません。
・もしかしたら集合時間前に軽いラジオ体操（意味深）もあるかもしれません。
・無理をせず自分の肝臓や終電と相談しつつ、愉しく酒屑をトゥギャザーしましょう。
