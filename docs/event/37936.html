データベースは圏なんだよ！

David Spivak （ http://math.mit.edu/~dspivak ）が唱える 関手データモデル ( “Databases are Categories” だの functorial data migration だの )を眺めてみる会です。

注意事項

部屋は『圏論勉強会』の名義で借りています。

また一応ですが、ある程度圏論の知識は前提で進めるつもりでいます。
「関手圏」（functor category）が頭に入ってるといいかもです。
