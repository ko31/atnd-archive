Amazon「Kindle」電子書籍出版のキホン技2017夏

description

なにかと気になる「電子書籍」ですが、
2017年の電子書籍市場予想は、2280億円（雑誌含め2660億円）になるといわれています。
http://www.excite.co.jp/News/it_g/20170103/Economic_69988.html

2016年の日本映画界の総興行収入額は2360億円なので。
もしかすると2017年、電子書籍業界が映画市場を抜くのではないかともいわれていて、
電子書籍って、意外と無視できないメディアなのです。

しかも、映画や音楽と違って、
誰もが気軽にデビューできるのが「電子書籍」の世界。

例えば、Amazonの電子出版「Kindle」を利用すれば、
最短2週間以下（ブログを書きためていたら）で世界作家デビューできます。

じつはこの出版方法が驚くほどカンタンなのですが（入門書も多数あります）。

問題はどうすれば、Kindleで売れるのか？
その裏技を含めて、ご紹介する第4回目のセミナーとなります。
紙の書籍としての「企画」売り込み方の裏技もお話しします。

-——————————————

神戸・元町映画館開催では、2017年3月「電子書籍Kindle入門・3ヶ月後に作家デビューする方法」以来の開催となりますが、その後、大阪、山口で開催し、最新ネタも含めた改訂版となります。

というわけで、元映版「電子書籍Kindle入門・3ヶ月後に作家デビューする方法」参加以降、すでにKindleデビューされた方の参加も大歓迎（ぜひその後の体験談を伺いたい、前回元映版参加者さんは、出版の有無にかかわらず1000円引き）。

また、今回は今後いっしょに仕事をしていけるライターさんや編集者さんとも知り合いたいです。

いわゆるビジネスパートナー探し。

文章教室受講者（近日スケジュール発表予定）もあわせて受講された方には、プロの現場を紹介する予定です（もちろん課題原稿など実力チェックはありますが、デビューまでフォローいたします）。

個人的には、日々映画評を書いている方や文章を書くことが好きな方とも出逢いたいとも熱望しておりますが、地元情報に詳しい方も編集部に紹介しやすいと思います。


また、今回は参加者全員が今秋までに、Amazon「Kindle」作家デビューできるワークショップも検討中です（参加費500円くらい）。
というわけで、講師の話を聞いて終わり……ではない、その後も続くセミナーを今回は目指します。
