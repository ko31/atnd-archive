MySQL Casual Advent Calendar 2011

こんにちはこんにちは！

昨年 有志による技術系Advent Calendar が盛り上がった事は記憶に新しいかと思います。
今年も MySQL Casual は Advent Calendar を行います！
昨年は こんな感じ でした。

今年は25日全部書くのはしんどいので、有志を募ってみんなで書く形式にしたいと思います。

参加方法について

参加者各自の担当日にご自身のblog等で記事を上げて、このatndのコメント欄にリンクを書いてもらう形式でお願いします。

各々書いていただいた記事は MySQL Casualブログ からリンクを張って取りまとめていきます。

もし 「blog持ってないけど記事書きたい!」 という方が居ましたら id:myfinder まで @とか付けて連絡ください。
MySQL Casualのブログ に直接掲載します。

もしくは参加者の誰かにはてなブログの招待をお願いすると、すぐにblogを作れるので大変Casualですね！

記事の順番

このイベントの参加順に1日から25日までの担当とします。

投稿基準

特にありません。
MySQLに関連することであればCasualに書いていただきたいです。
ある言語に特化した使い方等でもno problemです。

昨年は こんな感じ で大変Casualでした。

お問い合わせ等

id:myfinder にmentionを付けていただくか、IRC(#mysql-casual@freenode)で呼びかけていただくかしてくだされば反応できるとおもいます。
