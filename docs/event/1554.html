title ねじ山＠高尾山ビアマウント
catch 佐藤ねじ君とろっかくさんを囲んでボルトの新記録を祝おう！
description

要は、みんなで山登って、山でビール呑もうっていう健康的な企画です。
山かねじが好きな方はどなたでも参加することができます。
「山が好きすぎて逆に海」と答えるユニークな方も気軽に参加ください。



佐藤ねじ君とろっかくさんは100％来るので、ねじ山の如くエッジの利いた彼らから色々なアイディアを聞くことができます。例えば冒頭のタイトルイメージはお二人の光速コラボによる仕事です。これは信頼できますね。AR三兄弟+SCHEMAも駆けつけるので、AR家族とかも駆けつけてもらえると嬉しいです。ユニークな面々の参加が予想されるので、参加者同士、うっかり交流を深めていただければと思います。



「天狗を含めてでも、何とか37人集めて昼間の開催としたく」・・・と意気込んでおりましたが、この路線だと直前まで時間がFIXできないので、ビアマウント開場前に皆で並んで普通に入ることにしました。ご家族お誘いあわせのうえ、是非ご参加ください。

＊天狗にからまれた場合、コンチうちわが有効です

昼と夜の間のねじ山

12：00 京王線/高尾山口駅/改札前集合

12：15 みんなで登山（登山服着用推奨

14：30 みんなでビール＆バイキング！

16：30 みんなでロープウェーで下山（下山服着用推奨

17：00 解散！（家に帰るまでがねじ山です



＊登山服は山に即した格好で、下山服はユニークな格好で参加いただけると面白いと思います

料理
和・洋・中のバイキングです。ケバブとかセルフクッキング（鉄板焼き）とかデザートもあるらしいです。
ドリンク
ビール・ハイボール・ワイン・カクテル・サワー・かめ樽焼酎・ソフトドリンク
＊生樽ビールはアサヒスーパードライ・キリン一番搾り・ハイネケン・黒生・ハーフ＆ハーフがあるそうです
