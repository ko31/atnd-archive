「自分の人生は、自分で創ればいい」

イベント ユーザー

title 「自分の人生は、自分で創ればいい」
catch 安藤 美冬 ノマドライフセミナー
description

グローバル化、少子高齢化・・・
テレビ、新聞、といったマスメディアにおいて、
あちこちでそんな言葉を耳にする。
 
日本の人口が減り、国内の市場が低迷すると言われている。
でも、僕たち若者は「それがどうした？」「自分の生活に何の影響があるの？」って感じ。
 
幸せに生きていきたい。ただそれだけなんだって思う。
 
学生なら3回生の終わりから就活をして、大企業に行く。
安定が欲しくて公務員になる。
 
それがいけないとも正しいとも思えない。
 
でも、どこかで不安に思ってる。
 
自分にとって、幸せってなんだろう？
 
声には出さないけれど、なんとなく葛藤したり、不安を感じている。
 

自分にしかできない何かがあるって思っている。
 
自分らしい生き方って？
自分らしい働き方って？？
 

今回、社会や組織の中において
自分らしい働き方、自分らしい生き方を見出し実践している
安藤美冬さんをお招きしセミナーを開催いたします。
 

安藤さんは、
TwitterやFacebook、ブログなどのソーシャルメディアを駆使し、
仲間との出逢いやご縁を生かしつつ、予想もしていなかったチャンスを創りだし
複数の仕事、複数の肩書きを持って仕事をしています。
 

■セミナーについて
本セミナーは安藤さんの強い希望もあり、
学生をメインターゲットとさせて頂いております。
社会人枠が少ない点はなにとぞご了承ください。
 

すでにある選択肢の中から選ぶだけじゃない、自分の人生は自分で創りたい！
と思っている学生にぜひ、参加いただきたいと思います。
 

安藤さんから学生に向けて
セルフブランディング、セルフメディア、プラットフォーム化を軸足とすることによる
・自分で主体的に選択/決定することの素晴らしさ
・会社員としての生き方だけでは無い多様性
をテーマにお話し頂きます。
 
また、今回セミナーの目玉として、
普段滅多にお聞きすることの出来ない、
お仕事とお金（ギャランティー）について
赤裸々にお話し頂きます！！！！
なおこちらは、
「非公開」であり「完全なオフレコ」
となっております！
 
・お金にならない仕事/そうでない仕事の違い
・ダンピングの是非
・自分の価値と値付け
といったこと、すなわちお仕事とギャランティーの関係から
お金だけではない、お仕事のオファーがあってもお断りすることもある
セルフブランディングについても詳しくお話し頂きます。
