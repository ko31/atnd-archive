★コーチングとは何か？★
━━━━━━━━━━━━━━━━━━━
書店などに行くと”部下を伸ばすコミュニケーション”など、管理職のマネジメントとしてのスキルという位置づけが多いですが、
コーチングを簡単に言うと”人を伸ばすコミュニケーション”です。
だから、マネジメントの分野だけでなく
学校教育、恋愛、営業など人が関わるさまざまな分野で用いられて成果を出しているものです。
コミュニケーションなので、日常的な
・恋人との会話
・上司との会話
・仲間との会話
などでもそのエッセンスを用いることができます。

要は”人を伸ばすときにどんなコミュニケーションをとったらいいか？”

という要素を集めて体系化したものがコーチングです。
そのための”聴く”スキルや”質問”のスキルなど様々なスキルがあります。

━━━━━━━━━━━━━━━━━━━━━━━
★コーチングを身につけるメリットって？★
━━━━━━━━━━━━━━━━━━━━━━━

何度も書いてますが、コーチングは”人を伸ばすコミュニケーション”です。
人と関わって仕事をしたり、人との関係性が多様化している今の時代において
コーチングを身につけることによるメリットはかなり大きいです。

例えば、
・将来、部下を持った時に部下の可能性を伸ばす上司になれます。
・人から信頼される人物になります。
・客観的な視点で自分や周りを見つめ、行動を促すことができます。
・人の感情に寄り添う聴き方が身につきます。
・自分の伝えたいことをうまく相手に伝えられるようになります。
・周りの意見を踏まえて物事を進めるリーダーシップを発揮できるようになります。

ここで挙げたのはあくまで一例で
基本的には”コミュニケーション”であるので、他にも色々あります。

副次的な効果ですが、
人にもてるようになったり笑。
自分にコーチングをすること（セルフコーチング）で自分の成果も向上します。

コミュニケーションをしっかり鍛えて、日常生活で活用すると周りの反応や出会う人も変わっていきます。

★講義の後の実践という形式
コーチングはコミュニケーションなので、実際にやってみないと身につきません。
そして、コーチングの本は世の中にたくさん出ていますが多くの人が本を読んだだけだと実践しようとしません。
実践していく中でコーチングの意義やスキルの存在、
どういうコミュニケーションをとると相手が気付くのか
など様々なことが体験を通じて感じ取れます。

★実践へのフィードバックと振り返り
コーチングを実際にやってみて、
そこで相手から生のフィードバックをもらえます。
そのフィードバックから普段のコミュニケーションに立ち返って気付くことも多く。
ここで得た気付きを多くの参加者が実際に活かしています。
またフィードバックをもらった後にもう一度実践ワークがあるので
振り返ってすぐさま自分の改善内容を試せます。
普段はコミュニケーションに対してフィードバックをもらえる機会はほとんどないと思うので、おススメです。

　　　　　　★講師紹介★

黒田洋介
iLassic代表。コーチ/講師。

大学時代初期まで友達ができず、それを改善するために大学入学後、コミュニケーション関連の本を読みあさり、本の内容を実践することで少しずつ自分を変えていく。
その後、就職活動時代に”コーチング”と出会う。
コーチングの「相手の可能性を信じて、聴く」という考え方に共感し、「このコミュニケーションが広がるともっと素敵な世の中になる」と考え、
コーチングを学び始めると同時に、コーチングを大学生に広げるためのiLassicを設立。
1年間でのべ１０００人以上の大学生にコーチングを伝えるセミナーなどを行う。
また、その過程で「一人ひとりが自分らしく生き、共に生きる社会」を創るという夢を持つ。

卒業後は、「自分らしく生きる社会」を実現するため、教育を変えるという想いで学習塾に就職。
同時に”対話の場”を社外に創るため”夢を語る”ワークショップなど多数開催する。
のち、独立。

現在は「イノベーションを起こす30歳を輩出する」を実現するためにコーチングをはじめとした年間70回以上のセミナー、イベントを主催している。
