九州近辺のみなさん、一緒に勉強しませんか？

5月12日（土曜日）に第1回勉強会を開催します。

「MS Management Day in Fukuoka」というタイトルで、Microsoftの管理系製品やテクノロジーについて勉強したいと思います。興味がある方、ぜひご参加ください。
当日は日本マイクロソフトのエバンジェリスト・安納順一さんお越しいただいて、Windows PowerShellの講義をしていただき、参加者でハンズオンも行う予定です。

参加条件※重要※

今回の勉強会ではマイクロソフトのラボにアクセスしてハンズオンを行うため、「RDP7.0でリモートデスクトップアクセス可能なPC」と、PocketWifiなどの「モバイルルーター」をご持参ください。

Windows 7はそのままでRDP7.0が利用可能です。
Windows XP（SP3）とWindows Vista（SP2）についてはリモートデスクトップクライアントのアップグレードが必要なため、事前のご準備（ダウンロード・インストール）をお願いします。

Windows XP SP3、Windows Vista の SP1、および Windows Vista の SP2 のリモート デスクトップの接続 7.0 クライアント更新リモート デスクトップ サービス (RDS)

Win.tech.q について

「Win.tech.q （ウインテックキュー、通称ウインク）」は、 九州を中心にWindows Serverなど、MicrosoftのITPro系のスキルアップを目指すコミュニティです。
「Win.tech.q」の「Win」は「Windows」、「Tech」は「テクノロジー」、「Q」は「九州」や「(技術を)吸収」や「Quest(探求)」といった意味があります。
