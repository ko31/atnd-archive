イベント概要

Ubuntu 18.10のリリースを記念したオフラインミーティングイベントです。

過去のオフラインミーティングの様子は以下のレポートをご覧ください。

  • 18.04リリース時
  • 17.10リリース時
  • 17.04リリース時

前方のセミナー、後方のハッカソンという雑然とした空気とともに、Ubuntuの魅力を語って頂ければ幸いです。

今回は会場のサイズの都合上、参加上限が少なめです。はやめの登録をよろしくお願いいたします。

参加費・参加資格等

参加費は無料です。ささやかではありますが、無料の食べ物（からあげ含む）と飲み物（からあげ含む）を用意してお待ちしております。

Ubuntuを使ったことがある、あるいは興味がある場合など、どんな方でも来て頂いて問題ありません。

イベントの詳細は、 https://wiki.ubuntulinux.jp/Events/OfflineTokyo201812 を参照してください。

会場へのアクセス方法

今回の会場も六本木です。 前回と同じく土曜開催です！

  • 会場までのアクセス方法： https://corp.gree.net/jp/ja/corporate/location/

最寄り駅は「東京メトロ日比谷線六本木駅」「都営地下鉄大江戸線六本木駅」になります。

会場では入館手続きが必要となります。詳しい手続き方法は、開催日時が近づき次第お伝えします。

セミナーについて

セミナーの登壇者を募集しています。Ubuntuに関連している発表内容であれば、テーマに沿っていなくても問題ありません。

発表したい方は以下の情報を揃えた上で、本イベントのコメント欄に記載するか contact@ubuntulinux.jp 宛までご連絡ください。

  • イベントページのセミナー一覧に掲載する発表者名（ハンドルネームで可）
  • タイトル（仮タイトルで可）
  • 希望時間（大雑把で可）
  • 希望するタイミング（イベントの最初の方、後ろの方、など）

原則として先着順となります。当日の様子を動画配信するかどうかは未定です。

申し込み上の注意

このページで参加登録を行うと登録完了です。

また料理を用意する都合から、事前に参加登録が必要です。参加する予定の場合は登録しておき、都合がつかなくなったらキャンセルをお願いします。

重要：申し込みの締め切りは12/6（木） 20:00です。 これ以降の申し込みはシステムの都合上、行えませんのでご了承ください。
