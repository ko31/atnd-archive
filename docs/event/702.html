必要事項：


  • 英語が読めること

  • Skype で参加できる or ねっとらじおが聞ける

  • Skype で参加する人は日本語で話せること

  • html と pdf が読める環境が整っていること

  • C++ を知っていること

  • たとえば「N2857 を見てください」と聞いただけで何をどうすれば良いか分かること

肴の候補：


  • N1377, “A Proposal to Add Move Semantics Support to the C++ Language” (html)

  • N1610, “Clarification of Initialization of Class Objects by rvalues” (html)

  • N1821, “Extending Move Semantics To *this (Revision 1)” (html)

  • N2080, “Variadic Templates (Revision 3)” (pdf)

  • N2081, “Concepts (Revision 1)” (pdf)

  • N2342, “POD’s Revisited; Resolving Core Issue 568 (Revision 5)” (html)

  • N2346, “Defaulted and Deleted Functions” (html)

  • N2535, “Namespace Association (”inline namespace")" (html)

  • N2634, “Solving the SFINAE problem for expressions” (html)

  • N2640, “Initializer Lists — Alternative Mechanism and Rationale (v. 2)” (pdf)

  • N2761 “Towards support for attributes in C++ (Revision 6)” (pdf)

  • その他，「これを肴にして欲しい」的な希望があればコメントで（WG21 の paper に限りません）
