イベント概要

StartupDatingとWeb CAT Studioがコラボして、新しいビジネスパートナー探しのプログラム
【Find Partners】を開始することになりました！
第一回目として、4月24日（火）にパートナーを探している企業・個人・プロジェクトによるピッチと、
ご参加者とのミートアップの場を開催いたします。

【主催】
StartupDating

【共催】
Web CAT Studio （株式会社リクルートエージェント）

オーダーピッチ企業

boundio（KDDIウェブコミュニケーションズ）
4月10日に発表したクラウド電話APIプラットフォームのboundioを紹介、このプラットフォーム上でアプリケーションやサービスを展開してくれるパートナーを募集します。

FxCamera × Cellar（ビットセラー）
4月11日にFxCameraの買収というニュースでステルス状態から姿を現したスタートアップ、ビットセラー社が仲間を探しています。

iQON（VASILY）
iPhoneアプリをリリース後、3月のユニークユーザー数100万人、投稿数が一気に20倍に跳ね上がったソーシャルファッションサイトの「iQON」がデザイナーを求めて参加です。

Conyac（エニドア）
ちょっとした翻訳を世界中にいる翻訳者に気軽に頼めるサービス「Conyac」は現在世界展開を本格的に進めています。エンジニア、デザイナー、マーケターを募集しています。

trippiece（トリッピース）
みんなで旅をつくるwebサービス”trippiece（トリッピース）が、 旅好きなエンジニア・デザイナーを求めて参加です。
