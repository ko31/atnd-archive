description

あの「餃子ナイト」が帰ってきた！

１７時あたりからゆるゆると。チャージ１５００円(餃子食べ放題／ドリンク別)
※皮を伸ばしたい方は麺棒を各自ご持参ください(100均でも売ってます)。何本かはこちらで用意してますのでお使いください。

下北沢スローコメディファクトリー で人気の「手作り・ものづくり系」のイベントが経堂のユル飲みBAR笑和堂で鮮やかに復活！餃子を皮からこねてもよし、包むだけでもよし。ただひとつのルールは「作った餃子は責任持って食べるべし！」食べ残しのないようお願いします。

こだわりのタレ、餡などを持ち込み、味を比べてみるのも楽しいものです♪ラー油も大歓迎！ビールでカンパーイ！といきましょう♪

タイムテーブル

5月21日（土）17:00～餃子の皮作りからゆるゆるとスタート。
※5/18（水）の時点で参加人数10人に満たない場合は中止となります
　早めの参加表明をお願いします！

手作りの皮は格別。自分で作ってもよし、ビール飲みながら眺めるも良し。もちろん餡も作ります。試したい具材があればお持ちください。餡ができたらいよいよ「包み」に入ります。
焼いたり茹でて出来たてアツアツを頬張りましょう～♪
なお、笑和堂の生ビールはあの エーデルピルス！ 最高のひと時が味わえます。もちろんビール以外もワインやカクテルなどもイロイロあります。
店主ダムッチョ、サポート山ちゃん、共に濃味風味ですが、楽しくゆるくラー油の芳しく香る、そんなステキな夜にしてまいります～
