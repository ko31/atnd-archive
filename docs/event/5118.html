description

World Café By OKWave は世の中の様々な悩みや課題に対してみんなで考える場所です。
毎回選りすぐりのゲストに登場いただきテーマにそったトークをしていただきます。
そしてトークタイムの後にゲストの人が考えている抱えている課題を投げてもらいます。
その課題に対して参加者のみなさんと解決策を探ります。
課題と発表いただいた内容はOKWaveコミュニティでも公開しOKWaveユーザーからも解決策を募ります。


前回は「グローバル」にフォーカスしましたが、今回のテーマとして取り上げるのは

『ワーキングプア』。



社会の問題をビジネスで解決する社会起業家が現在の取り組みと
現在の課題について投げかけます。

ゲストはNPO法人NEWVERY代表 山本繁さん

日本の教育、若者の就業問題について様々なアプローチで
取り組んでいる山本さんがこの度上梓された『日本中退白書』を元に
ワーキングプアが起こる構造、教育システムについて問題提起します。

なぜ問題が起こる？どうしたら解決する？

Q&AサイトOKWaveでも働くことの質問や悩みは数多く寄せられます。
皆様のアイデアや問題意識が社会の「ありがとう！」に変わるように
楽しく活発な意見交換ができればと思います。



～当日の流れ～

19:30-20:30 ゲストスピーチ＆テーマ投げかけ
20:30-21:30 World Cafeにて参加者の皆様のディスカッション
21:30-22:00 ネットワーキング


◆スペシャルゲスト：山本繁さん（NPO法人NEWVERY理事長）
◆参加費：2,000円（飲み物と軽食をご用意致します）



教育や働くことについて関心のある方、
また社会的な問題を扱う仕事に興味のある方ぜひご参加ください！
