description

【講座のポイント】
　従来は既存顧客の要望にひたすら『すりあわせ』を行うことで事業を「受動的」に行ってきたが、今後、国内外のグローバル競争でサバイバルするためには、市場への「能動的」な組織的取り組みが必要となる。
　当セミナーでは、生産財という特徴を踏まえたマーケティングの考え方の理解を目的とする。マーケティング戦略の策定の考え方やポイントの理解も深める。

【プログラム】
１．生産財マーケティングの事例紹介
　　企業例：東レ炭素繊維、コマツ、キーエンス、シマノ　など
２．生産財マーケティングとは
　　製造業の戦略の基本：技術―製品―事業を１セットで考える
　　マーケティングとは
　　消費財と生産財の違い
　　製品戦略：製品コンセプト企画、製品アーキテクチャー企画　など
　　価格戦略：価格設定における主な変数、単一製品・複数製品の価格設定　など
　　広告プロモーション戦略：広告から営業のマルチチャネル戦略　など
　　販売チャネル戦略：直接販売、間接販売、電子商取引　など
３．日本製造業の生産財マーケティングの課題
　　そもそも市場環境変化を認識できているか？
　　顧客セグメントとターゲティングは明確か？
　　顧客価値＝製品・サービスの差別化戦略は明確か？
　　顧客の経営・事業ソリューション提案が出来るか？
　　製品・技術アーキテクチャーとビジネスモデル戦略は連動しているか？
　　サービスや情報を組み込んだビジネスモデル戦略であるか？
　　エコシステムからの差別化発想が出来ているか？
　　競合企業ベンチマーキングが行われているか？
　　新興国における価格戦略は適切か？
４．生産財マーケティングの戦略モデル
　　ドメイン変革の戦略モデル
　　プロダクトライフサイクル戦略モデル
　　アンゾフの成長ベクトルの戦略モデル
　　アライアンス・Ｍ＆Ａ戦略
　　オープンイノベーション戦略　～オープン＆クローズ戦略～
　　エコシステム戦略
５．生産財におけるグローバルマーケティング戦略
　　国・地域×産業マトリックスによる市場セグメンテーションと優先度付け
　　海外の有望市場における顧客企業とのパートナーシップ構築
　　国・地域や産業の横断による共通化戦略
　　海外の競合他社のゼロベースのベンチマーキングと事業として勝つ競争戦略
　　生産財におけるグローバル・ブランド戦略
　　ビジョナリーな顧客企業を巻き混んだ未来シナリオ構想
　　結果重視型の戦略実行計画の立案、そして経営資源の集中
６．生産財マーケティング戦略の策定
　　戦略策定のステップ
　　　ステップ１．戦略検討の与件整理と戦略仮説の想定
　　　ステップ２．戦略コンセプト企画
　　　ステップ３．業界構造分析
　　　ステップ４．市場・用途セグメンテーションと有望市場・用途の特性分析
　　　ステップ５．顧客特性分析
　　　ステップ６．競合ベンチマーキングと自社分析
　　　ステップ７．生産財マーケティング戦略の検討
　　　ステップ８．生産財マーケティング戦略の計画立案　～成果直結型ブレークスループロジェクトとは～
