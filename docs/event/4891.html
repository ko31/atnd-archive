【概要】
『デジタルネイティブ世代・・・生まれた時からネットやモバイルのある生活環境の中で育ってきた世代。1986年生まれ前後。』


現在ネット業界の中心いるのが、76世代であることは間違いないです。
GREE田中社長、mixi笠原社長、ドリコム内藤社長、ウノウ山田社長、ひろゆきさんなど毎日色々な情報、サービスを触れていて、76世代の名前を聞かない日はありません。
彼らにはもう到底勝てないのでしょうか？


いや、

時代は変わります。



中学・高校の頃にWindows98が発売し、i-modeでモバイルインターネットを楽しみ、
大学生の頃にmixiでネットワークを広げ、ブログで情報を発信し、
社会人になった今はTwitter・Ustreamでリアルタイムの情報を発信・取得し、
ソーシャルアプリで離れた場所の友だちとコミュニケーションを取ったりしています。


そんな我々は、今後のネット・モバイル業界を引っ張っていく日は、もうすぐ目の前に来ています。

次のネット・モバイル業界は、我々デジタルネイティブ世代が中心に創っていきます。

そんな熱い心をお持ちのデジタルネイティブ世代が、情報交換・ネットワークを広げるための交流会を開催します。


今回のテーマは、

【76世代からSmartPhoneを学ぶ】

です。

ゲストはモバイル業界で最も有名な6,000人参加の勉強会『モバイル夜間大学』を主催(現在は別の方が主催しています。)し、
モバイルのプロ2,000人が読んでいる『いきべん通信』を発行している
Eagle株式会社代表取締役藤永真至社長です。
