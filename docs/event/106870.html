今スグできる！「オフィスワークスタイル改革」セミナーとオフィス見学ツアー

弊社では６月６日に”今スグできる！「ＯＷＳ改革２」”と題して、オフィス・ワークスタイル改革に関するセミナーを開催させていただきましたがご好評につき、７月にも同テーマ（働き方改革／健康経営／ペーパレス）でセミナーを開催いたします。
セミナーは、『紙を使わないと文化が変わる 「働き方改革をデザインする」』とイナバインターナショナル株式会社様の『「働き方改革」を進めるためのオフィス創りのご紹介 』の２つをご紹介いたします。
また、セミナーと合わせて、当社オフィス「Ｌｉｂｅｒｔｙ ＆ Ｃｒｅａｔｉｏｎ Ｏｆｆｉｃｅ」の見学ツアーも開催いたします。社員一人ひとりが自由と未来を創造し、自らが働き方を変えることによって作り出される新たな働き方の確立へ向けた活動をご案内いたします。
セミナー終了後に個別相談会も行いますので、合わせてご利用ください。（受付終了　17：30）

・開催日：7/9(火)　14：00～17：20　(13：30 受付開始)

・会場：東洋通信工業株式会社　本社３Ｆ　会議室

・定員：20名

・受講料：無料

・セミナーお申込み締切　開催日前日まで
