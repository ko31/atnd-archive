description

「家や車を買ったら一人前」
「社会人になったら保険には入りなさい」
「投資はギャンブルだから、手を出してはだめ」
これらは、親世代の価値観です。
生涯年収が減り、地価も上がらず、年金が期待できない現状で、こうした価値観を鵜呑みにしてしまうと、周りに勧められるままに
”持ち家に生涯賃金の２〜３割をつぎ込み”、
”手数料の高い保険に多額の保険料を支払い”、
”ほとんど利息のつかない預金に残ったお金を預け”、
結果かなりの確率で破綻してしまいます。
これを回避するためには、お金に対する考え方を根本から見直す必要があります。
講師は、都内の電機メーカーに勤める極めて普通のサラリーマンですが、家族の将来と自分の老後の安全、そして自分の夢の実現のため、経済的自由を目指して活動しています。
本セミナーでは、現在の社会情勢と人生の５大費用を確認した後、皆さん自身が幸せな人生を送るために叶えたいことと、必要なお金をイメージしていただきます。あわせて、自分の経済状況の健康状態を把握するために有効な手法をご紹介いたします。
本セミナーは、お金を通して皆さんがご自分と向かい合うための気付きを得ていただくためのセミナーです。特定の投資手法や金融商品を斡旋するためのものではありません。セールス目的の方や、マルチ商法等の勧誘目的の方の参加はお断りさせていただきます。

これまでお金から目を背けていた皆さんも、まだ間に合います！自分がどんな人生を送りたいのか？それにはどれくらいのお金が必要なのか？講師と一緒に考えてみましょう！
眉唾ではなく真剣に取り組んでみたい方のご参加をお待ちしております！！

•対象
-自分を取り巻く社会情勢に漠然と不安を感じているが、将来に向けて何をすれば良いか分からずモヤモヤしている20代〜30代前半の方
-人生を通して叶えたい夢がある方
-独自の投資スタンスが確立できていない方

•アジェンダ
-社会状況の把握　〜このまま行くとどうなる？〜
-まずは、食べて行くためにはどれだけ必要？　〜人生の５大費用について〜
-目標設定　〜あなたの夢にはいつまでにいくら必要？〜
-自分の経済状況の把握　〜PL、BSを書いてみよう〜
-まとめ
-雑談

•参加して得られること
-人生で必要な５大費用の明確化と改善のポイントを知ることができる。
-人生で叶えたい目標と、そのために必要な費用をイメージする切っ掛けが得られる。
-経済状況の健康状態を見える化する手法を身につけることができる。
