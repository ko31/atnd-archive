Cygames Tech Fes
～Cygames流！Unity × スマホ3Dゲーム開発のすべて！～

概要

Cygamesがゲーム開発者の皆様に向けて、勉強会を開催いたします。
スマートフォンにおける3Dゲーム開発の現状を、Unityを活用した開発実績を通してご紹介。
現場における生産効率の向上を実現するツール活用、メモリ管理、Unityチューニングによる最適化まで、
「最高のコンテンツ」をつくるための技術のすべてを制作現場のプログラマーが直接解説いたします。

さらに、CEDEC2015にて発表した内製ゲームエンジン「Cyllista Game Engine」の最新情報も公開。
イベント終了後は懇親会も予定しておりますので、登壇者やご参加の方々との情報交換の場としてぜひご活用ください。

・14:00　　開場

　0.　Opening
　1.　メモリ管理について
　2.　生産効率を上げる内製ツール

　4.　Unity開発現場を支えるJenkins
　5.　Unity＋スマホで3Dゲーム開発/最適化するための考え方
　6.　Cyllista Game Engine
　7.　Ending
