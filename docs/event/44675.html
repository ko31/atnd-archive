description

【講座のポイント】
　機能部品に使用されているゴム材料は様々な環境下にさらされて問題（トラブル）を発 　生する場合があります。自動車・ガス・水道用ゴム部品の開発段階及び市場でのトラブル事例とその対策（配合技術を主とした）について説明いたします。

【受講対象者・レベル】
　ゴム材料開発に携わる技術者及びそれに関係されている方々、各種ゴム部品のトラブルを経験しているので、その原因の究明と対策について学習したい人など

【プログラム】
１．はじめに（既存のゴムのトラブル事例と対策資料の紹介）
２．自動車用部品（燃料系）でのトラブル
　　2-1　サワーガソリンによる硬化劣化と軟化劣化
　　2-2　アルコール含有ガソリンによる膨潤大
　　2-3　バイオディーゼル燃料による硬化劣化
　　2-4　ＬＰＧ燃料による硬化劣化
　　2-5　ＣＮＧ燃料によるブリスター現象
３．ガス（都市ガス・ＬＰＧ）用部品でのトラブル
　　3-1　他部品から抽出した可塑剤による膨潤大
　　3-2　ＬＰＧボンベのドレンによる膨潤大
４．水道用部品でのトラブル
　　4-1　塩素水接触による黒粉現象
　　4-2　ＦＫＭ（フッ素ゴム）の膨潤大
５．おわりに（トラブルをとらえた上で材料開発をめざしたい！）
