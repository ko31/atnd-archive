title ＠特勉（＠IT 特集連動勉強会）
catch #1 ログ、データ解析、インフラ技術
description

ユーザー行動分析と次の施策を裏付ける判断材料として、オンラインで収集するログデータの重要度は増しています。ログデータを基にした過去データの分析、将来予測は、Webやソーシャル、モバイルゲームの世界だけでなく広い領域で注目されています。セミナーではこうしたデータを扱うためのインフラおよび周辺技術の最新動向を紹介します。

ログ解析の現場で注目を集めるFluentdのメンテナとしても知られるトレジャーデータ　中川氏、Hadoopディストリビューションの1つを提供するClouderaの嶋内氏らによる技術解説と、実際にそれらを活用する現場の最前線にいる須田氏らによる技術解説を生で聞けるチャンスです。
