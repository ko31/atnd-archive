怒涛のリリースラッシュだった2012年もあと少しです。
System Center User Group Japan は結成から1年が無事にたち、来年に向けていろいろ準備をしています。

さて、少々遅くなりましたがSCUGJ結成1年の感謝と忘年会をかねまして第5回勉強会を12/1(土)　日本マイクロソフト 品川オフィス VIPルームにて行います。
今回のテーマは「Windows Server 2012 Hyper-V ウォームアップ」です。

Windows Sever 2012もリリースされ、SystemCenter2012の対応も見えてきました。Hyper-Vについては相変わらずの注目度ですが、今回SCUJGの視点で、いろんな情報をお伝えしたいと考えています。

【開催日】
2012年12月1日(土)

【場所】
日本マイクロソフト 品川オフィス　VIPルーム

【スケジュール】
13:00　受付開始
13:30～13:40　ご挨拶　
13:40～14:20 Session 1 – Service ManagerとOrchestratorで作る簡単PrivateCloud
14:20～14:35 休憩
14:35～15:35 Session 2 – 「Network Virtualization Packet Dive! Returns with SC2012VMM SP1 Beta」
（Session 2 には、注意事項があります。参加前にリンク先をご確認ください）

15:35～15:50 休憩
15:50～16:20 Session 3 – Hyper-V ホストの展開
16:20～16:35 休憩
16:35～17:35 Session 4 – 「Effective Hyper-V – 久しぶりエディション」

17：40 終了（予定）

懇親会は別途、会場近くで行います。
