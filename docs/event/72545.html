KROWL（クロール）とは

Web業界への “ いりぐち ” となる場を提供する、アイ・ファクトリーの事業です。これからWeb業界でチャレンジしていきたい方を対象に、セミナーや面談、キャリア相談、インターンシップ等の就業支援を行っています。

KROWLの由来
自由形の泳法クロール【Crawl】、Webサイトを巡るクロール【crawl】、そして働く【work】の3つの言葉を掛けあわせ、Web業界中をたくさんのクリエイター達が目標に向かって自分らしく自由に巡り、業界全体を発展させる、そんなきっかけを与えられる場所を築きたい想いからKROWLは生まれました。

KROWLのセミナーとは

毎月1回、参加費無料のWeb業界への就職に関わるセミナーや相談会を実施しています。第1回、第2回、第3回は下記の内容を実施しました。イベントをきっかけに就業が決定したというご報告も複数いただいております。

▼第1回　就職率アップセミナー
職業訓練校に在学中もしくは卒業した方を対象に、元インテリジェンスの起業家が就職について熱く語りました。ファシリテーターに元IT系トップキャリアコンサルタントを向かえ、就職活動の裏側もお話させていただいたセミナーでした。

▼第2回　職業訓練校出身者が語る就業支援セミナー
職業訓練校卒業から現在にいたるまでの実体験を語るトークセッションや今後のキャリア形成の話は、参加者からも参考になったという声を多数いただけました。

▼第3回　Web業界に本気でチャレンジしたい “あなた” のための就職相談会！！
Web制作会社の経験を持ったキャリアアドバイザーによる就職相談会を実施し、今後の就職活動について一緒に考えさせていただきました。
