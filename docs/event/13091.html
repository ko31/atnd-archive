札幌圏のiPhoneデべロッパーコミュニティ、iPhone Dev Sap主催の勉強会です。
当日、会場代実費のみ集金します。（1000円以内・22人以上で500円）

前半は発表形式。
後半はフリートーク＆フリー質問＆フリー開発。

発表者も併せて募集。
終了後に懇親会も予定しています。
懇親会の参加登録はこちら→ http://atnd.org/events/13092

MacBookなどをお持ちの方は、ぜひご持参下さい。
iOS SDKをインストールされていない方は
事前にインストールしておくと面白いと思います。
(SDK自体は無料でダウンロードできます）

参加者同士で情報交換などする時間もあると思いますので、
なんらかのネタ（いま作ってるもの、いま困っていることetc）があると有意義に過ごせるかと思います。

なお、当日はUstreamによる中継を行う予定です。

予定されている発表は、
「独りでもチケット駆動開発 〜 Xcode + Subversion + Redmine の利用例」@keneo さん
「iOS4プログラミング本について」@notoroid さん
「こくばん！エンジンによる簡単お絵かきアプリの開発」@palnyan さん
「サルでもできるTowerでGit」@frnk
