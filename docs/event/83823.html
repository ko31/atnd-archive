スマートデバイスの性能向上や普及、アプリケーションソフトの発達、インターネット回線の速度向上などの環境変化によって、質の高いUGCがSNSや動画サイトなどにあふれるようになりました。本セミナーでは、UGC活用の事例や実績を元に、消費者と共に作るマーケティングの“いま”をご紹介してまいります。

※UGCとは…
User Generated Contentsの略称で、Webサイトのユーザー（消費者）によって制作・生成されたコンテンツ（文章や写真、動画など）を指します。

■プログラム
第１部 UGCの概要とUGCを使ったマーケティング事例
講師｜株式会社オプト ソーシャルメディア事業部 青山 嘉晃
UGCが、なぜ近年注目されてきたのか。そして企業として、そのUGCにどう関わっていくべきなのか。UGCを活用する意義について事例を交えてお話します。

第２部 最先端のUGC活用法とその実績
講師｜シャトルロックジャパン株式会社 代表取締役 金光 展意様
より幅広い手法でUGCをマーケティングに活用する最先端の手法について、概念とともに実績データを用いてご紹介します。

第３部 自社でUGCを使うには
講師｜シャトルロックジャパン株式会社 青柳 洋平様
理論だけでなく、実際に自社でUGCを使ってマーケティングやプロモーションをする際の「考え方」や「手順」を、例を用いて具体的ご説明します。

■登壇者プロフィール
第１部講師｜株式会社オプト ソーシャルメディア事業部 青山 嘉晃
オプト入社直後からユーザーコンテンツチームのメンバーとして新規事業の立ち上げに携わりShuttlerockの日本市場への参入から拡販までを経験。大手企業を中心に多くのクライアントへUGCを活用したオウンドメディア支援を行なっている。

第２部講師｜シャトルロックジャパン株式会社 代表取締役 金光 展意様
バリュークリックジャパンにて営業に従事した後、ガリバーインターナショナルにてオンラインマーケティング部門の査定件数増加施策を担当。その後グルーポンジャパンオンライン広告営業部門長として、主に金融系企業の新規会員獲得や、ECサイトの集客・新規会員獲得、売上拡大施策の立案を行い、現職。

第3部講師｜シャトルロックジャパン株式会社 青柳 洋平様
エンタメ業界のベンチャーでインターンをした後、新卒でシャトルロックジャパンに入社。大手クライアント、代理店をメインとしたサービスの導入・運用を中心に、予算に対するメンバーの進捗管理・クライアントの声をまとめたサービスの設計や企画の進行を行う。

【シャトルロックジャパン株式会社について】
インターネット上にある無数のUGCを活用し、企業やブランドのデジタルマーケティングを支援する会社です。テクノロジーを通して企業と消費者の繋がりを深めることで、新たな価値や可能性を創出し、世の中の人々に貢献していきます。
