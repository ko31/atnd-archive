開催概要

2017/2/11（土・祝）、プログラミング生放送勉強会第46回を、今回も DMM.com ラボの会場をお借りして開催します！　マスコットアプリ文化祭連動のイベントで、受賞作品の発表と表彰も兼ねています。

プロ生勉強会は、IT・開発系ノンジャンルの IT 勉強会です。参加資格は、興味があれば OK ！　初めての方も参加お待ちしてまーす。いつも半数ぐらいの方が初めての参加です。

当日は、ライブストリーミング配信予定です。オンラインからも参加待ってます！　スピーカーへの質問・感想コメントもお待ちしています。
