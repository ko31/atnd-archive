第171回目の朝ごはん会のテーマは
「大恋愛のススメ！」です！！！
大恋愛と聞いてあなたはどんな事をイメージしましたか？

恋愛に、大の文字がつく意味とは？
求めるのは愛情を確かめあえるパートナー？
人生の仕事をともに創り出す存在？
大恋愛をしたら、何が起きるの？
そもそも、なぜ人は異性を求めるの？

恋愛に関する問いは、次々と出てきます。
人間にとって、興味が尽きないテーマでもあります。

朝ごはん会は、毎回違うテーマを
美味しいごはんを食べながら、和気あいあいと
自由に、話してみる場です。

話すって、凄い！わたしはいつも感じます。
初めて会ったのに昔からの友人のように感じること
があるのです。

「いってらっしゃい！」「行ってきます！」と言葉を掛け合い、職場へ向かう。
そんなアットホームで楽しい時間を一緒に過ごしませんか？
