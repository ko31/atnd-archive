【講演主旨】

スクリーン印刷用インク,リチウムイオン電池電極材料あるいは導電材料などでは,粒子の分散性が製品の可否を左右するといっても過言ではありません。有機溶媒中における分散安定化では,粒子表面のぬれおよび高分子(樹脂・分散剤)の相溶性や吸着性などに関する情報が不可欠です。どのような実験を行いどうデータを処理すればよいか実例を踏まえて平易に説明します。

【キーワード】

1.有機溶媒
2.微粒子・ナノ粒子分散
3.溶媒・分散剤の選択

【プログラム】

1.　有機溶媒中における分散安定化の特徴

2.　粒子表面におけるぬれと広がり
　2.1　接触角と表面自由エネルギー
　2.2 接触角の測定法
　2.3　表面自由エネルギーによるぬれの評価と溶媒の選択

3.溶解度パラメータ(SP値)で読む高分子の相溶性
　3.1　溶解度パラメータSP値およびHSPとは?
　3.2　溶媒,高分子および固体表面のHSPの求め方
　3.3　相溶性に基づく溶媒および高分子の選択

4.酸塩基特性で分かる高分子の吸着・付着特性
　4.1　酸塩基性とは?
　4.2　溶媒,高分子および固体表面の酸塩基度の求め方
　4.3　酸塩基性を利用した溶媒および高分子の選択

5.高分子吸着層による立体反発作用と分散安定化
　5.1 立体反発作用の原因　-混合効果と体積制限効果-
　5.2 高分子の良溶媒あるいは貧溶媒とは?
　5.3 高分子の吸着特性と吸着層構造
　5.4 新規分散剤の開発動向

6.粒子の表面改質
　6.1 表面改質で何が変わるか
　6.2 物理的および化学的表面改質法

7.分散安定性をどう評価するか
　7.1 沈降特性による評価
　7.2 レオロジー特性による評価
　7.3　薄膜のAFM,SEM,TEMなどによる観察
