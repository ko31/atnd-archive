こんな研修です

熟練したAIエンジニアでも１ヶ月以上かかるAIシステム構築作業を、一般ビジネスマンがたった１日で構築作業を体験する日本初の研修です。

Windows10の一般的なPCをご持参頂き、AIシステムを実装し、実践操作を体感します。

※持参パソコンの条件

①ソフトウェア　Windows10

②ハードウェア　 + CPU 64ビット + メモリ　4Ｇ以上 + ハードディスク空き容量　10Ｇ以上

体験できるＡＩデータ
  • ①株価・有価証券データ（株価の未来予測）
  • ②CIFAR-10（googleの猫で有名な画像の認識）
  • ③MNIST（ＡＩシステムの標準的な性能評価用データ）
  • ④上記③のサブセット（0～9の数字認識） ※研修中は１種類を選択して頂き、セミナー後に残りを持ち帰り検証も可能です。
