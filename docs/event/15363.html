トーストマスターズクラブ(Toastmasters Club)とは、参加型のスピーチ学習会(例会)を通して、話し方やプレゼンの能力を楽しみながら向上させることを目的とする非営利教育団体(米国のNPO)です。世界で12,500以上の支部(クラブ)があり、110カ国以上、26万人以上が参加。著名経営コンサルタントのトム・ピーターズ氏が著書『ブランド人になれ!』にて紹介しているほか、米国を中心に政治家や起業家の参加実績も数多くあります。18歳以上ならどなたでもご参加できます。日本では全国100以上のクラブが存在し、日経Associe、東洋経済オンライン、AERA、地方紙各紙に掲載されるなど、時代のニーズに応える形で現在急速に発展しつつあります。

ビジョナリーズ・トーストマスターズクラブ(Visionaries Toastmasters Club)は、武蔵小杉を主な拠点とする、英語中上級者向けクラブです。ある程度英語に慣れ親しんでいるビジネスパーソンや大学生が主な対象で（TOEIC600以上が目安）、幅広い年齢の会員が参加しています。ビジネス英語、話し方やプレゼンの上達はもちろん、会議の進め方を学ぶにも効果があります。

例会は月2～3回、2時間（原則毎月第1・第3・第5土曜日夜）の開催です。見学は無料で、運営はすべてボランティアのため、会費も月1,000円と大変リーズナブル。まずはご見学にいらしてみませんか? 明るく親しみやすい雰囲気の会です。
