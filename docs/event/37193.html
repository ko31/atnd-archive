■欠席の場合キャンセルを！
欠席の場合はお早い目にキャンセルをお願いします。

Flashのオーサリング経験を生かして、HTML5/Canvas上のアニメーションを作り出せます。

  • PC/iOS/Android対応 → OK!
  • Flash CS6でオーサリングして書き出し → OK!
  • AdobeやMicrosoftがスポンサーについていて（ちょっとは）心強い → OK!
  • ASライクな記述でJSが書ける → OK!
  • FlashDevelopでの開発 → OK!
  • 商用案件での実績 → OK!
  • これまでの経験を生かせる → OK!

優れたライブラリではありますが、日本語での情報が少なく手を出しづらい面がありました。この勉強会の活動を通して、ノウハウの発掘と共有を行います。

CreateJSに興味ある方は是非ご参加ください！
