【プレリリースパーティーとは？】

拡張版の正式リリース前に、先行購入したパックを開封出来たり新カードで遊べる炉端の集い（イベント）です。

以下、公式サイトより引用

「突撃！探検同盟」プレリリースでは、いち早くパックを開封できるってだけじゃなくて、正式リリースよりも早く新しいカードを使える「炉端の喧嘩」で遊べるんだ。

「突撃！乱戦同盟」は、プレリリースイベント中に開封したカードを使ってデッキを構築できる喧嘩だ。クラスにも、同じカードを入れる枚数にも一切制限なしだ、ド派手に行こうぜ！

もう一つは構築済みデッキでプレイする「ランダム地獄のウルドゥム」、最新拡張版のレシピを一足先に使ってプレイできる喧嘩だ。こちらの喧嘩は、カードパックを先行購入してなくてもプレイできる。だがまあ、プレリリースを存分に楽しみたいなら、先行購入の利用をおススメするぞ！

※イベントの参加にパックの先行購入は必須ではありません。先行購入パックがあるとよりイベントを楽しめます！
