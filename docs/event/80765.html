イベント概要

Polycountにある SubstanceChallange にあるお題のサブスタンスファイルを事前に作成していただき、
作ったものを当日に見せあう勉強会です。
ハンズオン形式の勉強会ではありません。

うまくできなかった処理や、ここはこういう処理で表現してみた、など1人ずつプレゼンをしながら、
他の参加者からさまざまなアドバイスをもらいつつSubstanceDesignerに対して理解を深めていくことを目的としています。

ゆるく、お菓子を食べながらまったり、"サブスタンスお茶会"みたいな雰囲気で進行します。

※SubstanceDesigner初心者向けの操作説明などはないため、ある程度どのようなソフトか理解している方推奨です。

ガイドライン

作成するサブスタンスファイルに対してのガイドラインです。

  • 作成に苦労したら、サイトや動画など様々なチュートリアルを参考にしてみましょう。
  • 完成できたらベストですが、完成しなくても構いません。
    躓いたポイントをみんなに聞いてみましょう。
  • Height/Displacementマップの使用は歓迎です！
  • プロシージャルのみで作成しましょう。
    インポートしたカラーマップ、ノーマルマップ、マスクマップは使用しないでください。
  • サブスタンスファイルの共有は大歓迎です。
  • 当日プレゼンしやすいようにノードは整理しておくことをおススメします。
  • 可能ならパラメータを設定してみましょう！

お題

Week1　コンクリートタイル （ SubstanceChallange のWeek1を参照）

必ずしもサイトにある画像と同じ物を作成する必要はありません。 大体お題に沿っていれば自分が作成してみたいものに挑戦してみてください。 その場合、参考にした画像を持参していただけるとなおよし！

当日持ってくるもの

  • サブスタンスファイル（必要なデータを一式USBなどに入れて持ってきてください）
  • SDが動くノートPC（持参歓迎。ない場合は他の方のPCにデータを入れてプレゼンしてもらいます。）
  • ちょっとしたお菓子（持ってこなくても問題ありません）

1台は私のを貸せるのですが、今回2テーブルで進行するので、 もう1台ほかの人に貸してもいいよという方がいらっしゃると助かります（でないと、進行不可に…）
