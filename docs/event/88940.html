32回目のSWWDC(Sendai World-wide Developers Community)は、 毎年恒例のWWDCキーノートふりかえりをします。

開催概要

6月5日(火)から行われた、Appleの開発者向けイベント「WWDC2017」の復習会を行います。2時間20分の基調講演のビデオを見つつ、内容をあれこれ語り合います。

  • 「眠いので基調講演を見れなかった」という方も。
  • 「起きてたけど英語がわからんかった」という方も（※）。

みんなで集まれば、バッチリ内容が理解できるのではないかと思います。発表を見て、今後のApple関連の技術をおさらいしましょう。

(※6/18確認したところ、ビデオに日本語字幕が付いていました。ですので、英語がわからなくても大丈夫そうです)

タイムテーブル

  • 18:20- 開場
  • 18:30 - 20:50 キーノート鑑賞。内容についてあれこれ語り合います。
  • 21:00- 終了

参加費

  • 参加費は無料です。

  • ソシラボ（SOCiAL Laboratory）
  • 〒980-0811 宮城県仙台市青葉区一番町2丁目2-8 IKIビル10F 会場の地図、案内はこちらです

懇親会

平日開催のため、懇親会の開催は未定です。閉会後に希望を聞いてみて、時間がある人が多いようであればやります〜。
