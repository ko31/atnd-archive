マニフェストのオープンデータ化で政策の比較・活用・検証が簡単に！。

早稲田大学マニフェスト研究所では、2015年4月の統一地方選挙を契機に「マニフェストの共通フォーマット」を作成し、全国の候補者・政治家の政策をデータで公開する「マニフェストスイッチ」を開始しました。

マニフェストスイッチについて

マニフェストの標準化により、各人の政策や方針を見やすく形で比較検討することが可能となり、また集めたマニフェストを「オープンデータ」（CC-BY）にすることで、テレビや新聞などの商業媒体から、個人のブログに至るまで、あらゆる人がデータを使って自由に利活用できるようになります。

折しも選挙権の18歳への年齢引き下げがニュースになっていますが、学校での授業での活用などにも期待できます。

マニフェストスイッチ公式サイト

説明会について

今回、県議・政令市議の告示が始まり、前期日程の選挙戦が本格化する4/3に合わせ、この「マニフェストスイッチ」の実際のスタートの模様と、プロジェクトが目指す「変わる選挙」の未来像、現状の課題と解決策について、説明させて頂きます。
