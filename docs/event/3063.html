【集合方法】当日現地集合

【開催にあたって】

  • 参加時間は目安です。途中参加・途中退出OK。
  • PC（＋開発環境）は持ち込みオンリーです。
  • ネット環境はありますが、館内用のプロキシ設定が必要です。
    　　※http プロキシなので http, https, ftp くらいに接続が限られます。
  • 室内飲食可、自販機もあります。
  • ゴミは各自持ち帰りでお願いします。（飲み物の缶程度なら問題ないです）
  • お昼にランチミーティングを実施します。美味しいご飯で楽しいお話をしましょう。
    （近くのiiASが候補地。）
  • 「開発オフ文庫」があるのですが、担当の方の会社においてあるのを受け取ることができないため、今回はお休みです。
  • PCをお持ちでない方もぜひご参加ください！
  • 各自参加者により写真撮影をすることは自由ですが、撮影されたくない人もいます。
    配慮をもって行動しましょう。

その他質問事項も本コメント欄にて。
多数の参加者をお待ちしております！！
