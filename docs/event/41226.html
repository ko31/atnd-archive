description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)

■袋町の広島市まちづくり市民交流プラザ５Ｆ　研修室
　広島市中区袋町6-36

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション1(40分)
　『夫婦で自営業と子育て』トライポット 藤田さん

　＜内容の紹介＞
　　独立して１３年、夫婦で５年、子どもが生まれて２年。
　　夫婦で一緒に仕事をしてく中で、仕事や家事や子育てを
　　どのようにしてるかなどの話し。

　・自営業と周りの理解
　・夫婦で一緒に仕事をするメリット・デメリット
　・仕事の役割（上司と部下）
　・仕事と家事と育児の分担
　・子どもがいる生活の一日
　・子どもを横に座らせて仕事ができるか？
　　など、、、

　おまけ
　・地方でも仕事はできるか？

　トライポット紹介

●セッション2(40分)
　『女子デザイナー的ライフサイクルシフトチェンジ』
　Bleunote Designのくどうさん

　＜主な内容＞
　東京と地方、独身生活と結婚生活、夫婦生活と子育て生活などなど、
　ライフサイクルの変化は仕事に少なからず影響を与えます。
　それらとどうつきあってきたか、そしてこれからどうしていくべきか…、
　デザインネタもちょっとだけ交えつつ、お話しします。

　＜くどうさんの紹介＞
　フリーランス・デザイナー。
　Bleunote Designを屋号に活動中。
　
　CSS Nite in AOMORI実行委員会 広報
　CSS Nite in HIROSHIMA実行委員会 スタッフ
　Android女子部 メンバー

　青森県黒石市生まれ。山口県在住。
　東京都内のデザイン専門学校卒業後、青森県内の建築会社にて勤務。
　2003年10月より東京都内のモバイルコンテンツ企画会社、Web制作会社、
　Webサービス会社でデザイナー勤務し、2010年よりフリーランス。
　主に、Webサイト制作、グラフィックデザイン、iOS/Android向けアプリケーションのデザインを行う。
　ほか、様々なプログラマーと共にアプリ制作、勉強会での講演活動をしている。

　keikudo.com
　http://keikudo.com
