description

受験、資格試験などのため、学校でも社会でも勉強は避けては通れないことです。
多くの方は勉強を楽しくないこと、嫌なことだとお思いでしょう。
それは私も同じです。
だからといって、嫌々勉強をしていると、効率的な勉強ができていないせいで、時間ばかりかかってしまいます。
勉強は目的を達成するための手段であって、勉強自体が目的ということはまれです。
なので、勉強に多くの時間を費やすのは望ましくありません。
効率的な勉強法を習得して、【時間】と【スキル】の両方を手にいれましょう。

本セミナーでは、短時間で効率的に勉強をして、資格と自由な時間の両方を手に入れる勉強法の習得を目的にしています。
私もこの勉強法を駆使して、短時間の勉強で、資格を取得しています。

私は地方の公立高校を卒業後、現役で薬学部に進学し、一回で薬剤師国家試験に合格しました。
その後も大学院修士課程、博士課程で効率的に勉強したことで、所定の年数で修了しました。
そして現在は都内の大学で学生の指導をしています。

学生の指導をしている中で気づいたことがあります。
それは、もの覚えがいい学生と悪い学生とでは、勉強法で決定的に違う点がいくつかあることです。
その点を、もの覚えのわるい学生に指摘したところ、その学生の記憶力は非常に伸びました。
つまり、記憶力に能力の個人差はなく、やり方を知っているか知らないかの違いだけなんです。

本セミナーでは、それらの相違点を説明しつつ、効率の良い勉強法を紹介していきます。
資格試験、昇進試験などを控えている方、お子さんに勉強させたい方、ぜひ聞きにいらっしゃってください。
