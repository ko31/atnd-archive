2010年12月7日(火)に開催したMySQL勉強会in大阪の第２回を有志で開催します。

今回は日本オラクルのMySQLエヴァンジェリストの梶山さんに来ていただいて「MySQLのレプリケーションとMySQL Cluster 7.2」というテーマでお話してもらいます。
また、宗近の方からPHPで使えるMySQLのロードバランサ「mysqlnd_msについて」講演します。
場所は前回と同じ日本オラクルの西日本支社をお借りすることが出来ました^^
ご興味のある方は是非ご参加ください！

申し込みはATNDからお願いします。
http://atnd.org/events/25437

開催概要は以下の通りです。

  • MySQL勉強会in大阪(第２回) *

演題:「MySQLのレプリケーションとMySQL Cluster 7.2」
日本オラクル　梶山隆輔　氏

演題:「mysqlnd_msについて」
株式会社あすか　宗近龍一郎

ソーシャルゲームのような高付加がかかるサイトにおいては、データベースをレプリケーションやクラスタによって可用性を高める必要があり、アプリケーション側においてもロードバランス等の負荷分散が必要となります。

今回のセミナーではレプリケーションについてのおさらいと、新しくなったクラスタの紹介、またPHPから利用するMySQLのロードバランサ「mysqlnd_ms」について説明します。
講演終了後別会場にて懇親会を実施する予定です。
