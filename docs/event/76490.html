第4回　Unreal Engine4　ミートアップセッション　in九州

4月24日（日）にUnreal Engine4ミートアップセッションin九州を開催します

2016年も多くの方にUE4の素晴らしさを伝えるべく、どんどん活動していきますっ！

今回は注目度No.1とも言える、「VR」の体験会を予定しております。
ゲームだけではなく、様々な業界で注目されているこの技術を体験していただき、
みなさまの今後の開発などに役立てていただければと思います。
セミナー後には名刺交換会を行いますので、お気軽にご参加くださいね。

※現地までの交通費等の諸費用はご参加者様の負担となります。

※当日の録音、録画、インターネット上への配信（SNS含む）は固くお断りいたします。
　イベント開催の拡散は大歓迎です♪
