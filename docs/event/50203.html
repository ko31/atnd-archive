catch 地域医療の現場でICTを効率的に活用し、医療の最適化から産業の活性化につなげる秘訣をご紹介します。
description

地域医療の現場でICTを効率的に活用し、医療の最適化から産業の活性化につなげる秘訣をご紹介します。政府の国家戦略特区に指定された大阪・京都・兵庫の関西圏は、規制緩和と先端医療開発促進、そして国際ビジネス拠点を目指します。次世代の医療環境をICTで実現する取り組みを、神戸大学の杉本真樹医師とファルメディコ株式会社の狭間研至先生にお話頂きます。関西から医療の未来を切り拓く新産業イノベーションを共有しましょう。
http://www.art-center.jp/seminar/index2.html
