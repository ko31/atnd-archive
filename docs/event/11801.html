コンセプト

2011年4月は統一地方選挙。今から街角には候補者のポスターがたくさん張られています。うーん、写真だけ見てもなぁ。キャッチフレーズもきれいごとばかりだし、なんだか違いがよくわからん。そもそも、候補者って、なんだか違う世界の人のよう・・・。でも今私たちの生活は不安だらけで、今後のこの地域での生活を考えると、実はこういった選挙も人ごとではないんですよね。

少しでも、私たちの心境を理解して、同じようなことを考えている人が当選して京都の今後を託したい。もしくは、多少でも共感するところがある人が市会議員になったら、これから私たちの意見も言いやすいんじゃないか。・・・といった動機から、市会議員候補者とお話しする場を企画することにしました。そんな企画者の想いに共感してくれた候補者の皆さんをお招きして、和やかながら本気で話し合う場を作りたいと思います。

ポスターでいつも見ている候補者はどんな人なのか見てみたい人、当選したら何をしたいのか聞いてみたい人、候補者と一緒に話し合いたい人、ちょっとどんな場か覗いてみたい人・・・皆さん大歓迎です。選挙をちょっと楽しくしてみませんか？
