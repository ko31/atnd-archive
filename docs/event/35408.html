概要

飯塚から世界へ！をキーワードにソフトウェア技術者たちが集結し、テクノロジーについてディープに語り合う場、e-ZUKA Tech Night。

2013年最初の開催となる、Vol.8は ”新春LT大会” 。

今年の抱負（野望）やお正月を利用して作ったもの、などなどをLTしちゃいましょう。
LT希望の方は、 こちら からご連絡するか、参加コメントに書いて下さい（飛び入りも歓迎！）。

それから、e-ZUKA Tech Nightで何度か顔を見かけているけど、名前を知らない人、いたりしませんか？後半は簡単なワークショップをおこなって、否応なしにきっかけを作ります！？
