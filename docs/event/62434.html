次世代のグローバルリーダーを輩出する海外インターンシップ事業GlobalWingを運営しているGLナビゲーション株式会社による【☆海外に興味がある女子学生限定15名☆】Global Wing Cafe を2月15日(日)開催します。

ーーーーーーーーーーーーーーー

こんな人におすすめ!!

☆周りとは1歩進んだ海外経験をこの春休みにしてみたい！

☆長期海外留学/インターンは不安なので、短期でもキャリアに繋がることをしたい！

☆成長して就活で語れる実績を創りたい！

ーーーーーーーーーーーーーーー

こんな人が来ます!!

☆ベトナムローカル向けのカフェを2週間で全て自分好みにデザインした、人気者ビール売り子。

☆中国の日本人経営者、起業家にインタビューし、本格フリーペーパーを作成。学生時代、海外経験がなかった社会人一年目の女子社員。

＊詳しい登壇者プロフィールは下記をご覧ください。

ーーーーーーーーーーーーーーー

学生全体のわずか3%のみという海外インターンシップ経験者に、ぜひ会いに来てください★

＊彼女達が参加したプログラム、体験談はコチラ。
