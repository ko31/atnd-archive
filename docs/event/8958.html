description

本格的な就職活動シーズンの開始に際し、ソーシャルメディアを使うことにどのようなメリットがあるのか？　また、どのようにソーシャルメディアを使っていけばいいのか？といった初歩的な基礎知識を学ぶ【基礎篇】から、具体的な操作方法・使用方法に至るまでを網羅した内容です【応用篇】。これらの知識・技能 を身につけることで、就職活動において最も重要である「情報」ソースの種類・数・質において、他の学生と比べて飛躍的に差がつくことを指向いたします。
2011年より更に厳しくなると言われている2012年卒生にとって、就職活動の成功確度を高める手助けに努めて参ります。 .
『Facebook&Twitter就活セミナー』シリーズの主な骨子

【基礎篇】

●就職活動とソーシャルメディア概論
ソーシャルメディアの基礎知識および就職活動におけるソーシャルメディアの位置づけ、メリットとデメリットを学びます。

●Facebook、Twitterの使い方
学生にはあまりなじみのないFacebookやTwitterの操作方法を習得します。

【応用篇】

●ソーシャルネットワーク・コミュニケーション
企業やＯＢ・ＯＧとソーシャルネットワーク上でコミュニケーションをとり、またインターネットという空間上でいかに自己アピールしていくのか、といった手法や注意点などを学びます。

●ソーシャルサーチ
業界研究や企業研究、または先輩社員の声や学生動向など就職活動に必須の情報を、このソーシャルメディア時代においてどのように入手していくのか。従来の検索エンジンによる検索の限界を超えた、ソーシャルサーチの手法と活用方法について学びます。

インターワークスでは、世界の趨勢であるソーシャルリクルーティング（ソーシャルメディアを用いた人材採用・人事業務）に関する日本国内および米国や欧米での事例研究等を通じ、ソーシャルメディアを用いた就職活動の支援、また日本企業におけるソーシャルメディアを用いた採用戦略の構築支援を積極的に推進して参ります。
