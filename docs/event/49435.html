プログラミング言語 Python を使って, プログラミング入門講座を行います.
主な対象はプログラミングを始めてみたい方と, Pythonを始めてみたい方です.

この講座だけでプログラミングが出来るようになる訳ではないですが,
これからどのように学んでいけば良いかも含めてお話しする予定です.

皆さんの参加をお待ちしています!

ノートPC をお持ちの方は事前に Python をインストールして持参されることをお勧めします.
OS は Linux, Unix, Mac, Windows 等どれでも構いません.
既にインストールされていることがありますのでご注意ください.
なお, 講座では Python 2.7.6 を使用予定です.

ダウンロードはこちらから: http://www.python.jp/download/
(パッケージマネージャ等を利用しても構いません)

CAMPHOR- HOUSE 内は電源と Wi-Fi を利用出来ます.

講座内容 (予定)

  1. イントロダクション
  2. プログラミングとは?
    初心者向けにプログラムやプログラミング言語などの言葉について説明します.
  3. Python とは?
    他のプログラミング言語との比較を交えつつ, Python がどのような言語なのか説明します.
  4. Python チュートリアル
    公式ドキュメントのチュートリアルの内容をベースに Python を使ってみます.
  5. まとめ
    プログラミングを今後も行なっていくにあたって, どのように勉強するのが良いかなどを経験者の意見も交えつつ考えます.

会場の場所が少々分かりにくいため、お困りの場合は@CamphorKyotoにリプを飛ばして頂くか、090-2944-8820 (森川)までご連絡下さい。
