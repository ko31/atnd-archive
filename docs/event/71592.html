PHP勉強会@相模原 の第2回を2015年11月10日(火)に相模大野(町田の隣)で開催します。

はじめに

PHP勉強会@相模原はPHPに興味のあるエンジニアを対象とした勉強会です。
延べ2～3人の発表者による10分～20分程度の講演と質疑を行う形式で進めます。
発表内容は決まり次第このページでお伝えしていきます。
聞きたい内容をコメントに書き込むと話したい発表者が現れるかもしれません！

参加条件

PHPに興味のある方ならどなたでも。
別に参加費は取りませんし、*途中入場/退席もアリ*ですのでお気軽にご参加下さい。

タイムテーブル

  • 20:00　開場・準備
  • 20:15　オープニング・自己紹介
    • PHP勉強会の説明 @yoya
      • https://speakerdeck.com/yoya/phpsagami2
  • 20:30　発表セッション開始
    • 20:30-21:30 PSR-7 – これは何だ @sasezaki
  • 21:20　LT発表開始
    • 21:30-21:40 PHP でバイナリ処理Tips @yoya
  • 21:45　片付け
  • 21:55　完全撤退・解散

講演 (10～20分発表)

  • PSR-7 – これは何だ @sasezaki
    • http://psr7.net/sasezaki/slide20151110/#/

PHP の話で聞いた人の実になりそうな事であれば何でも構いません。
PHP で何か作ってみた。フレームワークの復習、パフォーマンス改善、PHP 拡張うんちく、内部構造の話し、 etc 何でも。

LT (5分発表)

  • PHP でバイナリ処理Tips @yoya
    • https://speakerdeck.com/yoya/phpbinarytips
  • PHPパフォーマンス事例 (仮)
  • ゲーム開発者の集まり (仮)

PHP で話したい事があれば何でも。
