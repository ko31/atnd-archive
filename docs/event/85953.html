イベント内容

Web系技術全般のもくもく会です。

  • Web技術を勉強したい！
  • Web技術を使って何かつくりたい！
  • プログラミングの相談したい！

など、どんな目的でもOKです。

基本的に個人でもくもく。わからないことは聞きあったり、意見を求めたりして、楽しくもくもくしましょう。

成果の内容は問いませんが、最初にやることを宣言し、終わり際に一日の成果報告を発表し合いましょう。
途中、志願者によるLT（ライトニングトーク）を挟みます。

どんな言語でもOKです！！


  • Ruby on Rails

  • PHP

  • Python

  • Swift

  • JavaScript（AngularJS, React.js）

  • HTML

  • CSS

などなど、他の言語でも大歓迎です。

参加対象者

  • エンジニアやデザイナーになりたい人

  • プログラミングが好きな人

初心者の方もベテランの方も大歓迎です。
