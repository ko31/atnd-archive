title 【IDCフロンティア】 第3回 NOAHユーザー会
catch ～ クラウドの最新動向を勉強しよう ～
description

IDCフロンティアのクラウドサービス「NOAH」をご利用のお客様や利用を検討されているお客様を
対象に、クラウド技術の最新動向やNOAHの開発状況などをお伝えし、お客様同士の交流を深める
ユーザー会を開催いたします。

今回、CRMや電子ブックソリューションに強みを持つスターティアラボ株式会社 取締役の小友様を
お招きしてデジタリンクActiBookについてお話していただきます。デジタリンクActiBookは
導入事例が1,000社以上で、誰でも簡単に電子ブックが何冊でも何ページでも制作可能なサービスで、
そのインフラとしてクラウドサービスNOAHをご活用いただいています。

また、IDCフロンティアからはサービス開発部 部長の大屋から11/10に発表したNOAHとRightScale
の連携及び今後のNOAHの開発やUSのクラウド最新動向などを、また新基盤開発部　開発エンジニア
の横山からアプリを作る方の立場に立って必要な機能やカユイところを分かりやすくご説明いたします。
