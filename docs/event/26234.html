ぼんくらカンファレンスとは、

「我はぼんくら」

と名乗り上げた連中が、言いたいことを言いたいだけ話せるイベントです。

テーマは自由、発表時間も自由(アンカンファレンス形式)を予定しています。

本イベントは、昨年の開催が予定されていた「高専カンファレンス for speaker」が中止となってしまったため、そのやり直しです。ただ、発表予定の内容を見たところ、高専と関係ない話ししか無い気がしてきたので、高専カンファレンスの看板というかタガを外し、企画時の開発コード(？)だった「ぼんくらカンファレンス」に改名しました。

日本に広がる無差別系ぼんくら野郎どもの参加をお待ちしております。

前回の発表予定だった内容の一例

  • 湾岸ミッドナイトについて
  • ビールについて
  • 温泉について
  • 除草剤を1回撒いただけで農家を継がされそうになった件について
  • 等々

上記のように、「誰得なのかわからないネタを持ってるが、とりあえず話してみたい」参加者を募集しています。発表内容からわかるとおり、堅苦しいイベントでは有りませんので、「みんなが得する発表」をしても構いませんし、「今度、ちゃんとしたところで発表するので、練習代わりに使いたい」というのも有りです。正に自由自在です。

発表者募集

発表者の募集と、発表内容の運用は当日にアンカンファレンス形式で行う予定です。

アンカンファレンスについては 高専カンファレンス 014tokyoのアンカンファレンスの説明ページ が綺麗にまとまってるので参照ください。

発表枠は 「一般発表 20分」 と 「LT 5分」 の2枠が用意されています。複数枠組み合わせて発表時間の調整が可能です。また、一般発表で20分をフルに使用する必要は有りませんので、自由に20分をお使いください。

当日、会場にて12:00から発表の受付を開始します。ただし、「高専カンファレンス for speaker」にて発表のお申し込みをいただいた方は、優先的に発表を受け付ける事とします。

おねがい

発表の受付は、ぼんくらカンファレンス当日に行います。当日の参加・発表受け付けも可能ですが、会場の都合が有るため、参加希望者は事前にATNDにて出席の意思表示をお願いします。
