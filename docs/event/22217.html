description

たくさんのご応募ありがとうございます。
定員のため申し込みを締め切りさせていただきました。

この度、原宿ウェブ研究所では、起業とファイナンスの入門セミナーを開催いたします。

今回は、ベンチャー企業のファイナンスの専門家でブログisologueや『起業のファイナンス』で有名な磯崎哲也氏と、 投資銀行やFacebookを経て起業し、ソーシャルリクルーティングサイト「ウォンテッド」を立ち上げた仲暁子氏にお話をうかがいます。

磯崎氏からは、GREEやmixiなどの実際のベンチャー企業を例にして、起業から資金調達、上場への道のりをお話ししていただきます。現在は大きく成長した企業が、創業後に増資やストックオプションなどをどうやって実際に行なっていったのかを見ていきます。

仲氏からは実際の起業、サービスの企画・制作などの実例のお話をうかがいます。 ベンチャーをつくる際に必要なお金の話とビジネスや企画、そして仲間をいかに獲得するのかといったことについて、お話をうかがいたいと思います。

早い段階で定員数に達することが予想されますので、
参加希望の方はお早めにお申し込み下さい。

日程：　12月14日（水）19:30～22:00

場所：　東京都渋谷区渋谷3丁目28-8 久我屋ビル4階

料金：　3,000円（当日受付にてお支払いください）

イケてるベンチャー企業のつくりかた
