サイボウズ東京オフィスにて、kintone無料セミナーを開催いたします!!!!

【現場目線から学ぶ】kintoneをもっと便利に使うkintoneプラグインと地図のkintone活用事例セミナー

～kintoneは会議室で使うんじゃない！現場で使うんだ!!～

kintoneって色々出来るけど、具体的にどんな事が出来るの？ プラグインや連携サービスって色々あるけど、どう使えばいいの？

kintoneの様々な疑問を、現場目線から実際の導入事例や運用ベースに、顧客管理や点検管理などのアプリ構築方法から連携方法をご紹介。

またkintoneをさらに便利に使いこなすためのkintoneプラグインの使い方や、 kintoneと地図の組み合わせて出来る仕事革命方法など、ノンコーディングで実現できるkintone使いこなしテクニックを伝授します。

申し込み

セミナー特設ページ内にあるお申込みフォームよりお申し込みください。

セミナー内容

第一部：チームと自分の働き方を救うサイボウズの情報共有のリアルとkintone活用事例

[サイボウズ株式会社]　ビジネスマーケティング本部　丸田哲也

チームと自分の働き方を救うサイボウズの情報共有のリアルと多様性についてと、kintoneを活用した情報活用を事例を交えながらご紹介します。

第二部：【現場目線から学ぶ】kintoneをもっと便利に使うkintone運用方法と活用事例

[株式会社ノベルワークス]　下村佳穂

  • kintoneってどんな事が出来るの？
  • 実際の現場では、どのように活用しているの？
  • kintoneで簡単に業務システムが作れる？
  • 本当に３分で作れるの？？ kintoneの開発ベンダーだから分かる「kintoneの得意なこと・苦手なこと」を実際の作業者目線からお届けします！

実際の現場で利用されている顧客管理から、案件管理、バックオフィス業務でどう業務効率が出来るのか、kintoneの活用術と今後のkintoneの可能性をお伝えします。

第三部：「マップ × クラウド × タブレット」で屋外での仕事を変えよう!

[あっとクリエーション株式会社]　代表取締役　黒木紀男

「カンタンマップ」でkintoneのデータを地図化しよう 「カンタンマップ for kintone」シリーズは、kintoneのデータを地図で表示することで、業務効率化を目指すサービスです。 【カンタンマップ for iPad版 / カンタンマップ for iPhone版 /カンタンマッププラグイン】

これに、他社さんのプラグインを連携させることで、さまざまな業種で活用頂いている事例をご紹介します。

  • カンタン点検マップ
  • カンタン選挙マップ
  • カンタン顧客管理マップ
  • カンタン不動産マップ

ユーザさまからは、「現場での作業をこれを使うことで残業もミスも減った！」とのお声を頂きます。 kintoneと地図・iPad・iPhone・プラグインを組み合わせることで、業務を変えることを考えるきっかけにしてみてください。 また、現地でのiPad・iPhone活用ノウハウや注意点なども、時間の余す限りお話致します。
