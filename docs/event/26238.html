主催：ニジボックス（http://nijibox.jp/）

ソーシャルゲームでよく出てくるFlashアニメーションをレクチャーとワークショップの形式で、
ソーシャルゲーム制作の現場で利用されるスキルが半日で学べます。

●費用

無料

●日時・場所

3月 17日 (土), 13:30～18:30
ＭＴＬカフェ
(東京都中央区銀座7-2-6 リクルートアネックス1ビル B1F Google Maps：http://bit.ly/ x51vgP )

●タイムテーブル

第一部： レクチャー　13:30～
13:30～　「キャラクターに関節をつけて動かしてみよう」　講師：齊藤光一
内容：弊社タイトル「妖女大戦」「ヤン極志」でも重宝されているFlashキャラアニメの手法をご紹介。
14:00～　「衝突判定を使ったミニゲームを作ってみよう」　講師：寺井周平
内容：決定（エンター）キーで敵を避けるゲームをゼロから作り、スクリプトへの恐怖を和らげたく思います。
第二部： ワークショップ　14:30～
初心者の方にはベースとなる.flaを提供します。
二つのネタを元に、ご自身の経験にあわせて選んでください。
第三部： 懇親会　18:00～
随時解散

●当日の持ち物

ワークショップがありますので、無線LAN機能のあるノートPCをお持ちください。
Flashの開発環境をお持ちでない方は、体験版をインストールしてください

●申し込み方法

定員：20名

●対象となる方

画は描いているけど、アニメーションを業務でやったことは未経験という方から、Flashバリバリの方まで広く募集しています。
