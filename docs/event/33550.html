秋の京都で MobiRuby をつつく会 in はてな

秋の京都でちょっと濃い iOS プログラミングのお勉強はいかがですか。

最近、iOS プログラミングの多様性が増しています。従来は Objective-C で iOS SDK をダイレクトに扱うこと以外にほとんど方法がなかったのですが、今回とりあげる MobiRuby や PhoneGap のような開発支援システム（実際何と呼べばよいのでしょう）が増えつつあり、また HTML5 アプリも実用範囲に入ってきています。

今回は、株式会社はてな京都本社に増井雄一郎さんをお迎えして、MobiRuby の解説をしていただこうと思います。それもぐっと内部構造に突っ込んだ議論ができれば、と思っています。
また、はてなの浅野慧さん (id:ninjinkun) に、iOSアプリ新サービス開発へのチャレンジと、そこでのさまざまなアプリ開発についてお話していただきます。

iOS プログラミングに興味のある方、Ruby に興味のある方、言語処理系に興味のある方、MobiRuby や PhoneGap のようなシステムに興味のある方、あるいは はてな、浅野さん、増井さんに興味のある方などなど、どなたさまも是非お越しくださいませ。
