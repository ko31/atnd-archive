『超々』としましたとおり、「え？！そんなところから？？」というくらい初歩的なところから解説します。

Webプログラムでは HTML,　CSS, JavaScript など色々な技術がからみ合って実現されているため、初心者にはごちゃごちゃになっていると思います。

変数・配列・関数・条件分岐について詳しく説明したいと思います。

今回は10人席の丸いテーブルなので、わからない所があれば簡単に質問できる雰囲気だと思います。

XAMPPをインストールしてください

講義にはXAMPPというソフトウェアを使います。
これはパソコン上（ Windows, MacOS ）で、Webサーバーを実行するプログラムです。
