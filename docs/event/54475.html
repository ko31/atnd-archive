description

すべてのお客様に快適な花火観覧をご提供するため、定員206名の船をあえて120名に限定して運航します。
大型船からの花火観覧は、邪魔な建物や木、窮屈な場所取り、熱を帯びたアスファルトなど、不愉快な要素が一切ありません。
視界をさえぎる物が一切ないデッキで手すりにもたれかかるのも、空調の効いた涼しいボックス席でゆったりと座って窓から見るのも自由です。
疲れとは無縁の海上観覧で、夏の風流を優雅にお楽しみください。

2014年8月5日 第29回神奈川新聞花火大会 観覧クルージング
