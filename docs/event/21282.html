Re:Earth OpenAir実行委員会では、10月29日（土）～10月30日（日）にかけて岩手県奥州市の種山高原「正座の森」にて野外音楽イベント「Re:Earth OpenAir」を開催します。地元主導による皆で集まれる場所の創出と地産地消を目的として実施します。

■ 目的
新たな「イベント」の創出を「地元の若い世代」主導で行い、岩手県を代表するようなイベントに育て上げること。またフードのエリアは地元の店舗に出店を依頼、地産地消を推進すること。それと共に県外からの来場者に岩手県の認知度を向上させること。

■ Re:Earth OpenAir　開催要項
日時 2011/1029-10/30
OPEN 10/29/16:00　START 18:00 – END　10/30/09:00 CLOSE 12:00
会場：奥州市　種山高原「星座の森」　
〒023-1551　岩手県奥州市江刺区米里字大畑66-21

■ 実行委員会設立の背景
2011.3.11を境にして多くの友人・知人が亡くなり、
思い出のあの場所も記憶の中のものだけになってしまいました。
多くの同世代の皆は大きな音も出せない仮設住宅での生活が始まっています。
もともと遊ぶ場所が少なかった街には今や集まる場所すらありません。

皆で集まれる「場所」を「皆」で創れないだろうか？
その想いが全てのきっかけになっています。

「Re:」には「再び，さらに，新たに; …し直す; 原状に復す」という意味が、
「Earth」には「地球」という意味があります。

故人や家族、親しい人々、そして震災、街への想い。
仲間それぞれの「Re:」として自分たちで創り、仲間が毎年楽しみに思うようなイベントに育てていく事。
そうすることで「私たち」の「Re:」が世界に響いていくのだと信じています。

Re:Earth OpenAir　実行委員会　岡本 翔馬
