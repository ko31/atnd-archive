■会費：300円-予定■

■セッション内容■
——————————————
【iOS SDK デザイン-セッション】3回目の予定(時間内にすべて完了しない場合が有ります。)
・開発ツールの操作方法概要
・iPhoneのインタフェースデザイン規定
・今後の方針など
　(運営方法とか開発との連携など)
——————————————

☆★【3回目の注意事項】★☆
実際に開発環境をインストールして使い方の基本を行います。
開発環境である「Xcode」の入手は「iPhone Dev Center」で登録する必要があります。
http://developer.apple.com/jp/

■内容■

iPhone/iPad/iPod touchのデザインベースの開発勉強会を開催します。
iPhone/iPad/iPod touch専用のWebPage/アプリケーションを作成するときに必要な情報交換を実際のWebPage/アプリケーションを作成しながら習得することを目標に出来ればと考えてます。

通常のWebPageデザイン/デザインとは大きく異なる部分があります。
それは、iPhone/iPad/iPod touchの画面デザインを行い実際の機器にインストールして評価することまでを実施します。

■背景■
現在、iOS SDK 開発勉強会は毎週1回のペースでプログラミング関連の勉強会を開催しています。
一方、iPhone/iPad/iPod touchを開発/運営する上で重要な画面デザイン/サポート用WebPageなどの部分はiOS SDK 開発勉強会では全くサポートしていない状況です。

新しい「iOS SDK デザイン-セッション」では、プログラミングを主体に行う方向ではなく画面デザイン/対応するWebPage制作に主体に開発環境であるXcodeなどを使用して、iPhone/iPad関連の画面デザインなどを実際に組み込み評価する方法を情報交換し模索したいと考えています。


■参加資格■
・iPhone/iPad/iPod touchの画面デザインを実際に勉強し実施したい方。
・参加は個人/企業どちらでも可能ですが、Macを所有し1ヶ月に2回前後時間を共有出来る方。
・WebPage(出来ればHTML5)のページを制作できる方。(サポートページとかアプリケーション内で使用するWebPage向け)
・iPhone/iPad/iPod touchを使用している方。

■参加資格に合致しないなーーーと感じる方■
・ま、とりあえず見学でも

■参加資格補足■
・「iOS SDK デザイン-セッション」では、プログラム開発は基本的に行われません。
　ロジック的な部分は「iOS SDK 開発勉強会」などの他の開発メンバーからの支援を受けようと考えてます。
・iPhone/iPad/iPod touch用アプリケーションにデザインを組み込む場合Macによる環境は「必須」です。
