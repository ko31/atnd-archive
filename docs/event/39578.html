description

沖縄のゲーム制作者のみなさん。テストプレイのメンバー集めにお困りではありませんか？
自作ゲームを持ち寄り、お互いにテストプレイをしあおう！これはそんな主旨のイベントです。
もちろん、(ゲームを作ってないけど)テストプレイだけしたい！という方も大歓迎です。

概要

持ち寄った自作ゲームを順にテストプレイします。
プレイの後は、感想を述べあったり、ルールの抜けをチェックしたりしましょう。

今回テストするゲーム

『アリジゴク』 by fullkawa

  • 一番最初にゴールしてしまった人が負けという、通常とは逆のルールの双六。
  • あるいは、他人を先にゴールへ送り込むゲーム。

『ndqz』 by fullkawa

  • 四則演算でモンスターを攻撃し、倒した数を競うカードゲーム。
  • 試作が間に合わなければ別のゲームになるかも…

※テスト希望ゲームの連絡が来しだい、追加を行っていきます。

全ての参加希望者へ

  • 会場であるクリエイターズレジデンスの入場料として800円(ドリンク付き)が必要です。
  • 会場についての詳細は下記ページをご覧下さい。
    http://summer-time-studio.com/CreatorsResidence/index.html

制作者の皆様へ

  • 申込時のアンケートに、テスト希望ゲームの情報をお書き下さい。本ページに掲載し、参加を検討されている方へのアピールとしたいと思います。
  • 楽しくテストプレイができるよう、お菓子(手を汚さないで食べられるもの)やドリンク等の持ち寄りに協力して頂けると幸いです。

プレイのみ希望する方へ

百戦錬磨なヘビーゲーマーの意見も参考になりますが、ふだんあまりゲームをしない方の意見や“反応”も同じくらい重要です。なので、参加資格のようなものはありません。お気軽にご参加下さい^^

  • 申し訳ありませんが、謝礼はありません。
  • プレイ後に感想をお聞きします。
  • “未完成の”ゲームをプレイするという性質上、面白くないゲームにあたる可能性が少なくありません。
