description

身の回りの事柄をフリートークスタイルでお話するマーケティング庵では、過去に取り上げたテーマを再度取り上げ、市場がその当時と比べてどのように変化しているのか検証しています。
先月の“音楽業界”に続いて第2弾として、今回は【コンビニスイーツ】を取り上げます。
このテーマは2004年10月に取り上げました。
mixiにそのときのまとめを掲載しています。11～13番目のコメントです。
http://mixi.jp/view_bbs.pl?id=21993267&comm_id=2522544
（mixiに登録されていない方は、当日印刷してお渡しします。）

2009年のコンビニスイーツ市場は、前年度4％増の950億円で、2010年は1000億円市場に迫る勢いを見せています。このご時世の中、拡大している市場と言えます。
今回は特にコンビニ各社が取り組んでいるオリジナルブランドを中心に、前回取り上げた6年前と比べ、どのように進化してきたのかまた今後の展望を、コンビニスイーツを試食しながら、生活者の視点から検証してみたいと思います。

参加される方は、コンビニでスイーツを購入して持参していただけると有り難いです。

※マーケティング庵は、身の回りに起きる様々な事象をテーマに生活者の視点から素人の感覚で、自由で気軽な対話スタイルで掘り下げるユニークなマーケティング研究会です。
ドーナツやオーガニック食品の試食などから激安酒場の研究に出かけて、実体験してみるなど、参加者で体験しながら理解を深める研究会でもあります。
目的は愉快に時間とテーマを共有することです。

※参加資格者は、マーケティング関係者や広告関係者、学生、マーケティングに興味のある方など、特に制限していません。
また、直接マーケティングに関わらない方やマーケティングは全く知らないという方、さらにコンビニスイーツの試食だけが目的の方（笑）の自由なご参加もお待ちしています。
特に今回はスイーツ大好き！という方は大歓迎です。

※見学だけのオブザーバー参加でもOKです。
（オブザーバーの方でもスイーツの試食はできます(笑)）
