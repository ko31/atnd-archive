ゆるりとワードプレス勉強会＠つくば

※※受付方法※※

お申し込みは 専用フォーム からお願い致します。

＜概要＞

ホワイトハウスやSony Music、ウォルトディズニー、TOYOTAなどのサイトで使用されている無料CMSソフト、ワードプレスの勉強会を行います。

WEB制作会社であるオフィス江口株式会社がWEBの業界に育ててきてもらった恩返しとして毎月開催していこうと思っています。初めてなので、拙い点もあると思いますが、暖かい目で見守っていただければ幸いです。

全４回シリーズの勉強会ですが、勉強会の後半には単発のテーマで勉強会を行います。LT歓迎します。（内容は下記に）

＜対象＞

  • 事業でWEBを活用されたい経営者の方
  • フリーランスのWEBで仕事を取りたいデザイナーの方
  • フリーランスのコーダー（プログラマー）を目指している方
  • ワードプレス好きな方

＜目的＞

この勉強会はワードプレスの仕事を納品できるレベルになることをきっかけに、各々がwebやワードプレスに関する知識を深めるための勉強会です。

＜到達目標＞

  • WEBの基本を理解し、ワードプレスを事業に活用できるようになる。
  • ワードプレスを使用したWEBの制作案件を仕事として受けられるようになる。

〜具体的には〜

  • ワードプレスで自社サイトを作り、営業やブランディングに活用できる
  • WPのデザイン案件、コーディング案件を受託してもビビらないようになる
  • セキュリティ等も考慮されたWEBの運用ができるようになる
  • WEBだけでなくバックボーンのWEBの知識を手に入れる ことを目標としています。

＜講座の内容＞

※番外編の内容は受講者の方の状況に合わせて変える可能性がありますがおおよそ下記の予定です。

【第１回】2/23に実施しました。

＜基本編＞

  • 講座のための準備
  • WEBの基本（html,css,javascript,サーバ,WP）
  • wordpressの基本
  • サイトを運用するまでに必要なもの

＜番外編＞

様々な自動化を学ぼう

  • 情報収集の自動化
  • 投稿の自動化
  • 提携作業の自動化

【第２回】3/30に実施しました。

＜基本編＞

  • WPの開発環境の用意
  • テーマの作成（アーカイブページ）

＜番外編＞

  • １歩先へ、プロの考える最強のワードプレス開発環境

【第３回】4/27の内容です。

＜基本編＞

  • テーマの作成（個別ページ）
  • ループの使用

＜番外編＞

  • 人工知能で遊ぼう

【第４回】5/25の内容です。

＜基本編＞

  • カスタム投稿の作成と表示
  • カスタムタクソノミーの作成と表示
  • カスタムフィールドの作成と表示

＜番外編＞

  • ワードプレスってセキュリティどうなの？管理しているサイトが改ざんされないために。

＜スケジュール＞

14:15〜　受付開始
14:30〜　基本編
16:15〜　番外編
18:00〜　完全撤収&つくば駅近くのお店で懇親会

＜講師＞

wordpressに精通する江口。 制作実績、 https://inouehiroyuki.com/ や https://pusyuke.com/ など。 これまでpythonによる人工知能開発やweb開発の講座を行ってきました。受講生はすでにweb関連の仕事で独立したり、データ解析関連の企業にインターンしたりしています。

＜参加費＞

１人およそ1000円（会場費割り勘方式）

＜注意＞

今回初めての勉強会になります。代表の江口自身WEBを中心としたコミュニティに育ててもらいましたので、恩返しの意味も込めての開催となります。 食べてよし、飲んでよしのゆるっとした勉強会を想定しています。 がっつり教えてくれー！というMな方にはお答えできるかわかりませんが頑張ります。
