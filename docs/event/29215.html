案内文 ver201205

「ボードゲームサークル響」は、
沖縄でアナログボードゲームで遊ぼうという会です。
主催（RX）が遊びたいがために勝手に開催しています。

人生ゲームやモノポリーは日本では定番ですが、
最近海外で出てる現代のボードゲームは
子供だけじゃなく大人も遊べるものが多いのでとても楽しいのです。

遊んでる写真：http://goo.gl/cSnAW
RX所有ゲームリスト：http://d.hatena.ne.jp/nakarx+hibiki/20100810/1281448862
ブログ：http://hibiki-gamers.info/

毎月、オープンに参加者を募集するゲーム会を開いています。
RX個人分でも100種近くゲームを揃えておりますので、
飽きない程度に皆で遊び倒してます。 持ち込みがあると喜びます。

海外製アナログボードゲームとか、未知もいいとこですが
常連も交えて、最初にルール説明するようにしてますので
知らないゲームでもそれなりに遊べるようにしています。
戦略と運が乱れ飛ぶ妙味のある楽しさへ、興味ある方はどぞ。

初心者大歓迎！…ですし来てくれたら色々と配慮はしますが
卓が別れたりするとRXの目が届かないことも多く
メンバーによっては至らないこともあるかもしれません。
笑って流してくれるかRXまでこっそりチクると善処します。

ここだと名前もあまり聞かずにゲームやりたがる人が多いかもです。
雑談ぐらいはもちろんありますが、合わない人いたらごめんなさい。
同じゲームで囲む人同士、うまいとこ遊んでくれると嬉しいです。

参加者は学生が多めですが、大人も結構来ます。 女性もたまに。
特に制限は無いので、沖縄で対戦する面子が
なかなか集まらないといった人にもオススメ。
一回顔だしてみて、それから合うかどうか決めて下さいな。

Twitter, skype, mixi, facebookなど、
各種ネットツールにて告知してます。詳しくは上記ブログで。

また、オープン会とは別にSkypeにて突発で募集かけて
夜な夜なボードゲームで遊んでたりするのでSkype確認オススメです。

ぜひ、ご友人などお誘い合わせの上、一緒に遊びましょ。
