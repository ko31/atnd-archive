絵を描かずにイラストを簡単に作成できるパソコンソフト「3Dイラストデザイナー」 を使うと、年賀状/名刺用の3Dイラストを簡単に作成できます。
さらに iPhone の無料アプリ「3Dイラスト AR」で年賀状や名刺のイラストが現実空間に飛びだす!

本イベントではこの「3Dイラストデザイナー」と「3DイラストAR」の使い方をバッチリ説明します。

3Dイラストは、あらかじめ用意された3000点データを組み合わせるだけで簡単に作成できますよ。 さあ、あなたもAR対応の3Dイラストが現実世界に飛び出す年賀状や名刺を作ってみませんか?
