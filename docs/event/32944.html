■参加方法

横浜Apple User Group『AoMacクラブ』の交流会を上記の要領で開催します。

■Apple User Groupとは？

Apple User Group（アップル ユーザグループ）は、アップルの製品（Mac、iPhone、iPad、iPod等）をこよなく愛するユーザーの会です。
『AoMacクラブ』は横浜北部エリアのメンバーが中心のApple User Groupです。
アップル製品や広くIT関連の情報に興味ある方、
興味はあるけど「何にも解らないから教えて〜！」な方、
お住まいが横浜でない方でも、
老若男女、Ladies & Gentlemen, Boys & Girls !!
誰でも参加できるクラブです。
毎月第3土曜日に交流会を開催しています。
アップルの事や広くIT関連の話題をみんなで共有、情報交換して
月に1度大いに盛り上がりましょう！！

■初めて参加される方へ

初参加の方は・・・、
上記の【URL】をクリックして当クラブ公式ブログも併せてご覧ください。

※※登録なしで、交流会当日に見学、飛び入り参加でもOK！大歓迎です。
