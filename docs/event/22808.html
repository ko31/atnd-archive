title ビジネス×パーソナルFacebook活用

description

企業のソーシャルメディア活用を支援するトップブランド「ループス・コミュニケーションズ」許直人さんと、NHK「会社の星」などでFacebookを活用した人脈術など「個人のFacebook活用」を紹介する佐々木和宏さんにお越しいただき、ソーシャルメディア・Facebook活用の現状とネクストステップについてお話いただきます。 またトークセッションではSocialMedia360.jpを運営し、この度「Facebookページ制作 パーフェクトガイド」を出版された加藤洋氏が加わり、個人と企業の両方の側面からFacebookを語って頂きます。

目的

企業のFacebook活用がすすみ、ファン獲得目標など設定されるなか（ファン数が目的として本質的ではないにせよ）今、Facebookがどういう状況で今後どうなっていくのか、ビジネスと個人利用それぞれで日本のトップを走っておられるお二人にお話をお聞きします。
