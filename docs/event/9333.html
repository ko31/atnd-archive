概要

「久留米プログラミングフリークス　スマートフォン分科会」とは、スマートフォンやタブレットPC上で動作するアプリケーションの開発や、
それらの利用についての情報交換を目的とする、「久留米プログラミングフリークス」を母体としたコミュニティです。

アプリケーションをバリバリ開発されている方から、プログラミングなんてした事もないけれどもやってみたい方、こんなアプリケーションがあったら便利なのに……というようにアイデアをお持ちの方、その他スマートフォンに興味をお持ちの方を募集しています。

iPhone, Android, Windows Phone, iPad など、機種は問いません。

今回は第1回という事で、イントロダクションなどを行ないたいと思います。
