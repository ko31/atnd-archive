成果に繋がるパフォーマンス・クリエイティブ習得ワークショップ

title 成果に繋がるパフォーマンス・クリエイティブ習得ワークショップ
catch Webクリエイティブの効果を把握し、向上させるための具体的な手法を学ぶ！
description

クリーク･アンド･リバー社は、リスティング広告を中心とした運用型広告のリーディングカンパニーであり、
多くの広告主のパフォーマンス向上に携わってきたアイレップと共同で、
いまや多くの企業がWebクリエイターに求めるスキルとなっているマーケティング視点を取り入れた
クリエイティブを習得するためのワークショップを開催する運びとなりました。

ワークショップでは、驚きの成果をあげるクリエイティブ施策や、顧客の心を掴む制作テクニックについて解説してまいります。
また、その解説に基づきクリエイティブを実際に制作する実技の時間もご用意しています。

これから成果を出すクリエイティブを作りだすことで活躍をされたいWebディレクター、 Webデザイナーにとって
必見のワークショップです。

【開催概要】
■ 内　容
・クリーク･アンド･リバー社からのご案内
・アイレップによる『成果に繋がるパフォーマンス・クリエイティブ』講演
・パフォーマンス・クリエイティブ制作実技
・実技講評
※実技の使用ソフトについてはPowerpoint（ディレクター向け）、Illustrator
　もしくはPhotoshop（デザイナー向け）になります。

≪見どころや学習ポイント≫
・レスポンス率を確実に高める企画・制作テクニックが習得できる
・見込顧客を“成果”へと結びつけるデザインやコピーが学べる
・クリエイティブ成果改善のポイントを知ることができる
・業種やターゲットに合わせた最適なコミュニケーションシナリオの設計を理解することができる
・長年の消費動向に基づく分析・検証結果から弾き出された豊富な事例や広告サンプルが見られる

■ 対　象
1．より高度なランディングページやバナーの制作、デザインを学びたい方
2．決裁者やマネージャーにクリエイティブの根拠を論理的に説明できるようになりたい方
3．グラフィックなどでレスポンス広告に携わってきており、Webにも興味を持っている方
