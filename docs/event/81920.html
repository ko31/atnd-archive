自分がバイなのかレズなのかノンケなのかわりません
好きになるのはノンケのボーイッシュな女性、中身が男性のボーイッシュな女性、中性な女性です。
H経験は女性とありますが、巨人倍増挿入以前までなら男性ともあるし嫌でもないです。ただしイケメンに限りますが。
映画館へ行ったり、仕事終わり遅い時間帯だけどドライブで夜景をみせに連れて行ってくれたり、目的もなく話しながらドライブしたり、水族館、カラオケと何度かお互いの時間を合わせて会っていました。
今まで同い年か年上でしたが、初めて年下に好意をいだきました。

その子が可愛くて。。運転してるときがカッコよくて。。とにかく優しくて思いやりがあります。

お互いの今の仕事や恋愛話になり、あたしはその子しか会いたくなかったので”会うんだったら〇〇に会えないかなと思うよ。”と勇気をだして伝えました。
色々話してたら告白みたいな感じになり、シアリス 通販彼からは好きは好きだけど、中途半端に考えたくないから少し待ってほしいと伝えられました。
付き合って、キスして、その後があって、と一部の男は１つのゴールになるんだけど、俺はそう考えたくないから答え出るまで待っててもらえる❔と。。
同棲中の彼氏からの暴力や暴言が辛いです。
暴力・行動…腕や頭を殴られ、アザやたんこぶができる。グラスや扇風機を投げつけられる。携帯を投げつけて壊される。真冬に外へ閉め出される、ものを窓から投げ捨てられる等
暴言…バカ、クズ、ゴミ、ウザイ、頭が弱い、頭がおかしい、障害があるんじゃないか、どんな育てられ方をしたんだ等
別れようと何度も思い、別れ話をしますが、そのときは途端に優しくなったり、頭がいいので、言いくるめられてしまったりします。
普段は優しいので、情がわいてしまい、いつも別れられません。
別れなければいけないのは分かっているのですが、威哥王自分に自信がなく、別れたらこのままずっと独りなのではと怖くなってしまいます。
彼氏に精神的に依存している弱い自分もいけないのですが、このまま一緒にいたら、おかしくなってしまいそうです。(もう十分おかしいとは思いますが)
