このセミナーで明らかになること
株式、英語、金（ゴールド）など、
幅広い分野での著作を持つ林則行氏。
このセミナーでは、金（ゴールド）を扱います。
金の買い時売り時はいつか
地金購入のリスクは何か
銀、プラチナ、石油、穀物といった他の
商品よりなぜ金がいいのか
大きな値動きがある前にはどのような兆候があるか
金価格に作用する長期の力とはなにか
最近の下げをもたらした原因はなにか
インターネットから最先端の専門家と
同等の情報を無料で得る方法

このセミナーではみなさんが聞きたいと思うことを
存分に語ってもらいます。

なぜいま金(ゴールド)なのか。
このセミナーで明らかになります。

　　林則之とは
サンワード貿易でははじめてのセミナーとなる林則行氏。
いったいどんな方なのでしょう。

林氏の経歴として一番最初に出てくるのは、
まずなんといっても、
世界最大の政府系投資ファンドと言われる
「アブダビ投資庁」で日本株式の
運用部長をされていたという事実です。
当時は、中東でただ一人、
日本人としてオイルマネーを運用していた
という実績を持つ、まさに投資のプロ。
欧米の運用機関でもファンドマネージャーや
アナリストを歴任されてきました。

その後、相場ノウハウ修得を目指し、
投資の本場、アメリカへ渡ります。
コロンビア大学MBAに学ばれ、
あの著名な投資家であるジム・ロジャーズから
時代の読み方、変化の見出し方を伝授されたそうです。
現在は、アシュクインベストメント代表。
米国公認会計士の資格もお持ちです。

毎年3月に東京ドームプリズムホールで行われている
投資戦略フェアには2年連続で登壇。
満員の観客をうならせました。

その林則行氏の講演を直接聴ける貴重な機会。
是非ご参加ください。

著書
『金はこれから２倍になる』(宝島社)
『伝説のファンドマネージャーが教える株の公式』
『伝説のファンドマネージャーが実践する　株の絶対法則』(ともにダイヤモンド社)
『銅メダル英語」をめざせ！』(光文社新書)
『TOEIC500点でOK！ビジネスエリートを動かす英語』(宝島社)など 　
金（ゴールド）以外のコモディティは？
セミナー後半では、ここ最近市場で
注目を集めている原油安について、
その影響なども含めて弊社ディーラー山口が
林氏に伺います。
コモディティ市場の今後の展開について考える、
貴重な機会となるでしょう？！

弊社山口が展望する今後の見通しは、 林氏の意見と一致するのか？ 見所満載のトークセッションをお見逃しなく！
