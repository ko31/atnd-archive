title tokyo Linux Study #004
catch ビール片手に分散 KVS と Munin を学ぶ会 〜年末年始のサーバ負荷は超速で乗り切れ〜
description

年末スペシャルとして年末年始に向け、
予想されるアクセス増によるサーバ高負荷やパフォーマンスの劣化に対し、
分散KVS「okuyama」や
オープンソースのリソース監視ツール「Munin」を導入し、
簡単にできる対策を紹介します。

また、パネルディスカッションでは、登壇者と参加者がビールを片手に、
参加者の困った体験やその解決策をまじえながら語り合う時間を予定しています。

参加無料なので、お気軽にご参加ください。
