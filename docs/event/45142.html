なんとかスマス ＆ むにょちゃんとチャーハン

昨年のブリスマスアゲインじゃないけど、時期的にまた幸せのはし貸しきってパーティーやります。

料理について

今年は鱒料理を用意してもらえることになっています。飲み物は飲み放題です。

会費について

総予算として、一人あたり6千円です。時期的に鱒が高くつくため、通常の貸切より千円ほど高くなります。

会費は年齢や収入レベルに応じて3千〜1万円くらいの間で傾斜をかけます。財布に自覚のある方はよろしくお願いします。

ドレスコードについて

店長 と おかやん にドレスコードに関する仕切りをお願いしてあります。何か連絡があったときには協力をお願いします。

プレゼント交換

せっかくなので、プレゼント交換ができればと思います。賛同する方はご用意ください。未成年の学生もいるので、あまり高価なものは控えてください。千円程度を目安に。

キャンセルポリシー

18日の時点ですでに料理の仕込みが始まっています。人数が減らないようにしたいので、以後のキャンセルは代役をたてるか、お金だけ置いていくようにお願いします。また、そのさいは おびなた まで一報ねがいます。
