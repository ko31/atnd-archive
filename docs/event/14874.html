description

オープンソースソフトウェア（以下OSS）を利用したシステム開発が一般化しつつある今日、一足早く企業に導入され企業の情報化戦略（BI）に活用されているのがJasperSoftとTalendです。これらのOSSを有効活用する事を目的として結成されたのがJJSUGなのです。
九州新幹線も開通した2011年5月にJJSUGの九州における拠点としてJJSUG Qshuが設立されます。これを記念して『JJSUG Qshu 設立記念講演　福岡セミナー』がOSSベンダー及びJJSUG管理人を招き開催致します。
オープンソースの利用をお考えの方や興味をお持ちの方はぜひご参加ください。（一般の方でも可能です）
