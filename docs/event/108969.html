description

突然ですが、あなたは自分の満足いく人生を送れていますか？

・特にやりたくない仕事をしている…
・もっと好きなことを仕事にしてみたい！
・そもそも、自分の好きなことって？
・好きなことを仕事になんてできないよ！

そう思っている人も多いのではないでしょうか。

しかし、あるんです。好きなことを仕事にして、理想のライフスタイルを送る方法が。
そのために必要なスキルは、次の5つがあります。

・自分を知るスキル
・コミュニケーション力
・営業力（影響力）
・目標達成力（導く力）
・考え方

これらのスキルは、ひとつずつ学んでいくのは大変で、時間もかかります。
しかし、それを全部学べるのが「コーチング」というものです。

この講座では、あなたがどんな理想を持って、どんなライフスタイルを送りたいのかを発見し、あなたが自分で気づかなかった価値観を見出すことができます。
また、多くの人が知らない「目標を叶える考え方」を知ることもできます。

人間は、「変わりたい」と思っても、明日には忘れてしまう生き物です。
この講座に興味を持っても、少し考える時間を作ってしまうと明日には忘れてしまうでしょう。
本当に変わりたいなら、このページを見たいま、申し込んでみてはいかがでしょうか？
会場でお待ちしています！
