昨年初めて開催したSSH同窓会。

イベント概要

  1. ビッグバンド
  2. セッション(自由参加)
  3. 名簿 大人版

イベントについて

  1. 18〜30歳くらい向けの同窓会です。
    現役生の参加もOK! 懐かしい先輩や後輩と楽しく飲んだり話そう！
  2. 年に一度のぶっつけビックバンド「オトナSSH」による演奏や有志によるコンボ。（オトナSSHは現役時にやった曲を中心に去年とほぼ同じ曲をやります。一曲から気軽に参加OK！）
  3. 自身のコンボやビックバンドも募集中。ステージ時間とりますよ！
  4. 名簿（冊子）で近況報告！
    現役時代に作成した名簿を復活！大人の力でコンテンツも充実させます♪
