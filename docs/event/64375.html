創業支援セミナー

日本政策金融公庫（国民生活事業）との共同開催！

■第1部： 創業時に利用できる借入金

■第2部： 創業時のチェックポイント

■第3部： 個別相談会・懇親会及び情報交換会

場所

大阪産業創造館 6階 会議室A・B
住所: 大阪市中央区本町1-4-5　

講演概要

第1部： 創業時に利用できる借入金

  • 新規開業資金など、創業時（事業開始後 税務申告を2期終えていない方）だからこそ借りることができる借入金の内容と借入の手順をわかりやすくご紹介。
    （通常より低金利! 無担保・無保証の融資のご案内）
  • 株式会社日本政策金融公庫（国民生活事業）守口支店 融資第二課長　炭田 勝利 氏

第2部： 創業時のチェックポイント

  • 創業時に知っておきたい税務に関する項目をピックアップしてご紹介致します。
    知っているか知らないかで経営判断が大きく分岐することになります。
    低金利で無担保・無保証の融資を受ける為の創業計画書・事業計画書の書き方を詳しくご紹介いたします。
  • 講師：久保総合会計事務所 創業・経営革新アドバイザー 奥村 高也　

個別相談会・懇親会及び情報交換会

  • 特別に時間を設けて皆様の疑問にお答え致します。日本政策金融公庫 融資課長にも
    引き続き参加して頂きます！この機会を最大限にご活用下さい！
    （懇親会では軽食・アルコールをご用意しております！）
    ※懇親会参加者のみ2000円頂戴致します。
