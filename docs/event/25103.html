都合により本プレイベントは中止になりました

ITS Japan では「人と物の移動に役立つＩＴＳ防災アプリアワード」を開催するのに先立ち、プレイベントとして「カーナビ連携アプリ NaviCon のタッチ＆トライとアイデアソン」を開催します。

NaviConは、カーナビとBluetoothもしくはUSBで接続し、地図操作や目的地設定をiPhone, iPad, iPod touch(以下、iPhone等と表記)から行うアプリで、AppStoreから無料でダウンロードできます。
http://itunes.apple.com/jp/app/navicon/id368186022?mt=8

NaviCon はスマートフォンからカーナビをコントロールできるアプリです。
NaviCon 自体からカーナビをコントロールするだけでなく、
自分たちでコーディングしたアプリから NaviCon を呼び出し、
カーナビをコントロールできます。つまりカーナビ連携の API として機能します。

2/11のプレイベントでは iPhone 版でのタッチ＆トライとなりますが、
Android 版も今春登場予定です。

NaviCon の開発情報はこちら
http://www.denso.co.jp/ja/products/aftermarket/info/navicon2/index.html

開発元のデンソーさんより講師をお招きし、NaviCon の機能や開発方法を説明をいただきます。
また当日は NaviCon に対応したカーナビを準備しますので、その場で使ってみていただいたり、自分で作った iPhone アプリの動作確認を行っていただけます。
