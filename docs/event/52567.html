description

節約だけで乗り越えられる時代ではなくなってきてしまった世の中ですが、
給料も、景気も上がらないのであれば、増やした方が早いですし、
いずれダブルワーク、トリプルワークが当たり前の時代になるのであれば、
前もって認知・準備・経験しておくのもいいのではないでしょうか。

そのきっかけづくりとして、こちらのセミナーを役立てて頂ければ幸いです。
スピーカーの方にもセミナーの重要性を感じて頂き、
通常5,000円の内容を貢献という意味を込めて1,000円でお願い致しました。
ネットで検索すると自己アフィリエイトの説明だけで20,000円が相場のようです！
少し大げさに聞こえるかもしれませんが、人によっては人生のターニングポイントになると思いますので、この機会を是非ご利用ください！
今回は、「私のやってきた副業」というテーマで話して頂きます！

今回のスピーカーの方は、サイドビジネスについて勉強され
様々なビジネスをご自身でしっかり経験し、各ビジネスの良し悪しを自身の目で判断され
社会的に認められた『稼げるビジネス』のみを見つけ、今でも継続して稼いでいる方です。

その経験から、空いた時間でやりたい方から、一度に稼ぎたい方、手堅く稼ぎたい方、
いろいろな方向けの内容をお話しいただけます。

知る人ぞ知るビジネスのお話です。
「なんであの人は余裕があるんだろう？」
「なんであのお店は潰れないんだろう？」
という疑問も解消されて、視野も広くなると思います。
※怪しい物、法に触れるもの、グレーなものはありませんのでご安心ください。

【こんなモノが手に入ります】
・自分にあった副業の見極め方
・上手くいっている副業とは
・帰ってすぐにはじめられる知識
・世の中のお金のしくみを知ることができる

【プログラム】
世の中の・これからのお金事情についても紹介しますので、
お金を増やすために必要な知識や考え方が学べます。

【スケジュール】
1部：お金の勉強　～　副業紹介（前編）
休憩
2部：副業紹介（後編）～　質疑応答の時間

【紹介するサイドビジネス】
内容が不明確なセミナーが多いですが、参加してみた結果、既にご存知でしたら大切な時間を無駄にしてしまいますので、概要を記載致します。
ご興味のあるものがあれば、ご参加ください。
●座談会・アンケートで収入
企業がマーケティング目的で行う座談会やアンケートに参加する方法と
どのようにすれば、オファーを貰えるのか。案件の多い座談会業者を具体的に紹介。
（座談会は時給4000円～8000円程度で1～2時間程度のものが多いです。）
●自己アフィリエイト
スマホ1台数あれば、空き時間にできます。サイトの紹介、間違えやすいポイントなども紹介。
また、基礎知識ではありますが、amazonや楽天などの通販サイトを常に割引価格で購入することができるようになります。（10～20万円程度+α（割引購入ができるようになります））
●ネットオークションでの転売・せどり
難しいインターネットの知識はいりません。
商品の見つけ方、出品のコツやノウハウ、具体的な商品の紹介、ネタの見つけ方などをご紹介
（3,000円～10万円/月程度　→　やり方によっては上限なし）
●転職・人材紹介
転職で困っている方の紹介で謝礼が頂けます。
転職支援ができて、喜ばれた上で紹介料も頂けるお仕事です。
紹介先は大手や上場会社、ベンチャーなどもあり、職種は様々です。
周りにそういう方がいらっしゃれば、紹介だけしてみるというものです。
ヘッドハンティング案件は特に報酬が高めです。
（採用決定で5～30万円/名 程度、面接時点で5,000円の案件もあり）
●代理店ビジネス（コスト削減がメイン）
市場よりも割引して提供できるため相手から喜ばれ、
自分も収入につながる様な案件を取り揃えております。

・通信関連の割引
・交通費の削減
・自動販売機の設置
・クレジットカード決済代行
他、30種類以上の案件をご紹介していきます。
（毎月利用されるサービスは、売上の1部が毎月収入になるものが殆どです。継続収入が見込めます）
●交流会・イベント主催
「参加費×参加人数-会場費雑費＝収益」というシンプルな仕組みです。
イベントの企画を行い、募集ページと集客を行うだけです。
今ですとfacebookなどを利用すれば気軽に主催可能です。
人数と参加費の掛け算によってはかなり大きなモノになります。
（例：参加費4,000円×20人－会場費3,000円＝77,000円）
●セミナー主催
得意分野を活かして、その知識や技能を発表します。
その対価として、参加費を頂きます。
社会にでているほとんどの人が、実はできる可能性を秘めています。
なぜ、殆どの人が出来るのかをご説明します。
切り口や視野の広さでいくらでも広げることが可能です。
また、自分で喋れなくても喋れる方にお願いすることで幅が広がります。
（例：参加費8,000円×20人－会場費3,000円＝157,000円）
●その他
希望者には、下記の案件もご紹介可能です。
・座談会の案件多めの会社の紹介
・専門職の方にはフリーランスとしてのお仕事の提供
・治験モニターの紹介（新薬ではなく既に世の中に出ているもの）などもあります。

【スピーカーの紹介】
通信系の代理店からインターネット業界に転身し、WEB広告全般を学ぶ。
今では良く聞くアフィリエイトも10年前から携わっており、アフィリエイト会社にも勤務。
現在は、アフィリエイト、代理店事業など、ストック型にこだわった副収入を複数持っている。
東京都出身の34歳

※他人に迷惑をかけるようなものはありませんのでご安心ください。

自分のやりたいもの、興味のあるものだけメモして、わからなければ質問してください
楽しい雰囲気で行いたいと思いますので、リラックスして気軽にお声掛けください！

質疑応答は最後にたっぷり受け付けますが、
それでも足りない場合は、別途お茶でもしながらお話できればと思います。
ご覧の通り、初期費用を掛けずに始められるモノばかりです。（インターネット環境がないなどはその準備費用だけはかかりますm(__)m）
さらに、コレ以上のものが知りたい方は、別途お声がけください。
会場は別の目的で使用する予定でしたので少し広めですが、
少ない人数を考えております。
