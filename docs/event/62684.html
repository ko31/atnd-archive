description

「日米欧の金融政策と原油価格暴落の本質」！
昨年10月29日、米国が量的緩和政策の終了を決定。
その2日後、日本も追加の金融緩和を決定。
さらに先月22日には欧州が追加の金融緩和を決定しました！
米国の量的緩和政策の終了に対して
追加の緩和を続ける日・欧。
ドルの独歩高はいつまで続くのだろうか？
全世界が注目する米連邦公開市場委員会（FOMC）の利上げに踏み切る可能性は？
また昨年11月27日には、石油輸出国機構（OPEC）が石油の減産見送りを決定。
その後、半値以下にまで暴落した原油価格の背景にはいったい何があったのか？

今年の3月以降には原油価格を巡るイベントが目白押しとなっています！

3月17日はイスラエルで総選挙があり、3月22日にはエジプト議会選挙があります。
さらに、6月のOPEC総会や欧米イランとの核開発協議などもあります。
今後の原油価格、主要国通貨の動きから目が離せません！！

一部を除いて名前もあまり知られていない永山氏ですが、
経済評論家”副島隆彦”氏の 会費数十万円、少人数制の経済塾で副塾長を務めたという経歴があります。

そんな永山氏ですが、
今回特別にサンワード貿易のセミナーに出演していただけることになりました！
当日、永山氏との懇親会も予定しております。
普段なかなか会うことの出来ない永山氏に、ぜひ色んな質問をぶつけてみてください！

プロフィール
日本大学文理学部史学科卒業。
岡藤商事株式会社に入社し企画調査部調査課に配属。
主にマクロ経済や金融市場を担当し、取材その他による分析活動やレポート執筆、情報誌の編集を手掛ける。
同社を退社後、中堅クラスの商品先物取引会社に入社し、従来のマクロ経済や金融市場に限らず、商品先物取引の各銘柄も含めほぼ一人で調査活動を手掛ける。
2001年（平成13年）6月　有限会社ナリッジ・クリエイション設立　代表に就任。
フリーの立場で従来のマクロ経済や金融市場分析に限らず、その背後の政治的な権力闘争に至るまで調査活動に従事。月刊誌『商品・証券・金融先物市場』とその後継の『フューチャーズ・マーケット』や投資日報社などの大部分のインタビュー記事の作成をはじめ、他の情報媒体の市況執筆や著名評論家の著作本執筆の補助などの業務に携わる。
副島国家戦略研究所主幹　副島　隆彦　氏が開催する『経済塾』で副塾長を務めた。
現在、有料メルマガ配信サービス「フーミー」にて『永山卓矢の経済・金融・市場の本当の話』を配信中。
