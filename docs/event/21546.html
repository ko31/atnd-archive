Windows Phone勉強会＠四日市とは？

今年9月から日本でも販売が開始されたWindows Phoneについて、利用ユーザ・アプリ開発者のそれぞれの視点から、みんなで一緒に勉強して行く会です。

東海地方でのWindows Phoneの勉強会が少ないことから、このような場を立ち上げました。

2011/11/5 開催日と会場の部屋が変更されました。ご注意ください。
2011/12/8 セッションのタイトル・内容を更新しました。

セッション概要

「はじめてのWP7」

Windows Phone7の世界観から開発のイロハについて、デモコーディングを交えながら浅く広く紹介しようと思います。

「Windows Phone”らしさ”の作り方」

Windows Phoneのコンセプトと、それを実現するための方法概要をお話しさせてもらおうと思います。キーワードは「UX・センサー・クラウド・ソーシャル」で行こうと思います。

「Windows Phoneアプリとネットサービスとの連携」

Windows Phoneは各種クラウドやネットサービスとの連携に主眼が置かれています。またマーケットプレースでも外部のサービスと連携するアプリが多く公開されています。

このセッションでは、Windows Phoneアプリケーションがネットサービスと連携するために使える技術として、2011/9に//build/で発表されたLive ConnectによるWindows Liveとの連携を紹介します。具体的にはLive IDによる認証、Hotmailサービス、SkyDriveサービスへのアクセス方法を紹介する予定です。
