description

個人で19インチラックを使いこなしてみよう。という勉強会。

  • ついカッとなって小型19インチラックを買った。42Uラックを買うべきだったと反省している。

    そんな自宅ラッカーを応援するための小規模イベントです。
  • 無秩序な配線や場当たりなサーバー設置で自宅ラックが「ケーブルの樹海」にならないよう
    しっかりした配線を実施するための、初級者向けハンズオン勉強会となります。
  • ネタで終わらせようかと思っていましたが、開催の要望をいくつか頂きましたので、
    ちゃんとしたイベントとして開催することにしました。

やること

  • LANケーブルの敷設、成端
  • LANパッチパネルの成端
  • 19インチラックへの、ネットワーク機器取り付け作業
  • ケーブル包縛および丸札とりつけ作業

会場および集合場所

個人住宅(アパート)が会場となります。京急梅屋敷駅 駅前にて集合。
集まり次第、会場まで移動します。

※Ciscoロゴの服きた人が目印です

注意点

  • 会場には机がありません。その代わりシャーシ型SW-HUBを使います。
  • 冷蔵庫はありません。その代わり19インチラックがあります。
  • 暖房がありません。各種機器の廃熱で間に合います。
  • 少々轟音気味ですが問題ありません。
