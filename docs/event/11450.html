ある日友人がこんな夢をみたといいました。

「マックやモスやロッテリアなどのファストフード店で
【１００人が同じものを頼んだらどれくらいの処理能力を発揮できるのか】をはかる調査（いたずら？）をしていた。」

私はそれを聞いた瞬間、

やべえちょーーーーやりたい!!!!!!
ファストフードの帝王マックと戦いたい!!!!

といてもたってもいられなくなりました。

なので、実際やってみちゃおうと思いまして、参加してくれる方募集します。

ちなみにこれを主催するのは私の会社、つまり法人です。

こんなことを仕事でやる会社は日本広しといえどなかなかないだろーと自画自賛しておきます。

思いついた本人は徹夜してこんなサイトまでつくっていました。

http://tricky.enjoyn.jp/

一晩で即席でつくったこのサイトには夢が詰まっているそうです。

人はこんなことをやる人のことを暇人と呼ぶ。
しかし私はこう思う。

大人が本気で遊べばそれは仕事になる！！！

大人が集団でイタズラして、世間の注目を浴びたらそれは、新たなPRになる！

ちなみに頼むのはチーズバーガーです。
チーズ嫌いな方、ごめんなさい。

参加したい方、ひやかしたい方、セレブの街麻布でいたずらしてみたい方
平日休みでヒマな方、マックを愛してやまない方、なんかおもしろいことやりたい方
きちゃいなよ♪
