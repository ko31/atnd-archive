MHP3 オフ会のお誘い

MHP3オフ会開催のお知らせ

この度、MHP3のオフラインイベントを開催させていただくことになりました。

今回は特別な催し物の企画はございませんので、お気軽にご参加いただき、思う存分狩りを楽しんでいただきたいと思っています。どうぞご友人等お誘い合わせの上、お気軽に遊びに来て頂ければと思います。

参加可能人数について

今回は施設の収容人数の関係上、参加人数はスタッフを含め 60名 までとなっております。お手数ですが人数把握の為、参加希望の方は参加表明をお願いいたします。参加希望人数が60人を超える場合は先着順とさせていただきます。どうかご了承ください。

参加方法について

参加表明はページ右上の「このイベントに参加する」ボタンをクリックすることにより行って下さい。同伴者がいる場合は、参加表明時に「友人が○名一緒に参加します」などのコメントを記載して頂きますようお願いいたします。コメント記入部分はページ下にございます。

ログイン時のOpenIDの使用について

参加表明する際にログインが求められますが、このログインにはOpenIDを使用します。Twitterやmixi、はてな等のIDでログインして頂きたいと思います。しかしながら、もしこれらのIDとパスワードの入力に抵抗がある場合、汎用的なOpenIDを作成することで代替して頂いて構いません。

具体的には OpenID.ne.jp ( http://www.openid.ne.jp/ )にてOpenIDを作成して下さい。そちらで任意のOpenIDを取得の上、ログインをお願いいたします。なお、OpenID.ne.jpでは「ユーザーID」と「OpenID」は異なるものですので注意下さい。「ユーザーID」が abcde として、「OpenID」は http://abcde.openid.ne.jp/ になります。

イベント詳細

日程

2010/12/12（日）

時間

10:00～21:00（受付は9:30から開始予定）

場所

赤羽北区民センター2F 第2ホール（JR北赤羽駅下車 徒歩1分 「アクトピア北赤羽六番館」内）

  • JR北赤羽駅の赤羽口（最も赤羽寄りの階段）を降りて下さい
  • 赤羽口を出てすぐ右手にセブンイレブンが見えるかと思いますが、その横の脇道を通って建物に入って下さい
  • 建物目の前の広い道路の方からも入ることができます（正面入口）
  • 正面入口そばにあるエスカレータを上がって、2Fの会場にいらして下さい
  • 2Fに上がってからは案内板がありますので、それを参考にご来場ください
  • 当日はスタッフが10:00までは誘導のために改札前に立っておりますので案内に従ってご入場下さい

会場近辺の地図

会場近辺の地図は http://goo.gl/maps/8612 を参考にしてください。「アクトピア北赤羽六番館」内に会場があります。

注意事項

  • トラブル防止の目的により、18才未満の方の参加をお断りさせていただいております
  • 途中入場および途中退場は自由です。一時外出希望の際は、受付にて再入場証をお受け取りください
  • 会場内は全面禁煙です
  • 飲食は可能ですが、ゴミは各自でお持ち帰り頂きますようお願いいたします
  • いわゆる悪魔アイルー等の改造行為は禁止です。使用された場合はご退場いただきますのでご了承下さい
  • 会場には駐車場はありませんので、お車でのご来場はご遠慮下さい
  • 持ち物の盗難については責任を負いかねますので、貴重品はご自身で管理して下さいますようお願いいたします
  • 電源タップが不足していますので、可能ならば電源タップを持参して下さいますと非常に助かります
  • その他、不明な点等あればスタッフにお尋ね頂くか、コメント欄にご記入下さい。コメント欄では書きづらい内容につきましては gregminster@gmail.com までお尋ね下さい
