11/22(日): つながりの時は今　
〜ボブ・フィックスと過ごすハートフルな1日〜

この日は祝詞奉納、コラボトーク、生演奏のプロセス瞑想、パーティーと盛り沢山の内容で 皆様とハートが響き合うつながりの時を思い切り楽しみたいと思います。
詳しくはイベントサイトのページをご参照ください。
こちらから。

概要

【1部 : 講演会】 13:00〜16:00
　　いにしえの叡智とのつながり　そこにハートを開く鍵がある
　　祝詞奉納(古神道家 礒正仁×ディジュリドゥ奏者 KNOB)
　　ボブ・フィックス×古神道家 礒正仁トークセッション
　　「日本の叡智に秘められた啓発の時代への鍵とは」
【2部 : 講演会】16:00〜18:00
　　愛に目覚めハートを開くグループ瞑想
　　愛に目覚めハートを開くグループ瞑想〜プロセス瞑想新曲発表会〜
　　True Heartのライブ演奏によるプロセス瞑想会
【3部 : パーティー】18:00〜20:00
　　ボブと過ごすハートフルパーティー
　　　*ディナーバイキング
　　　*フリードリンク(アルコール、ソフトドリンク 2時間制)
　　　*ボブと出演者からの贈り物 〜プレゼント抽選会〜

出演者

  • ボブ・フィックス(Bob Fickes)
    • 1946年、アメリカ・ペンシルバニア州生まれ。 TM瞑想の創始者マハリシ・マヘッシュ・ヨーギに師事。後にフルフィルメント瞑想を創始し瞑想の普及に務める。現在タイのチェンマイに在住。
  • 礒　正仁
    • 熊野をベースに古神道を学びながら、実践を重ねる。世界各地への旅を通じて、太古からの宇宙や 大自然・精霊への祈りの中から生まれ紡がれてきた叡智や儀式の神髄を現代社会に統合し活用する 試みを行っている。
      一方で、香港をベースに投資家としても活躍。証券会社を経営、環境と地球 再生のための私募ファンドを主宰している。
  • KNOB
    • 芸能界で活動後25歳の時にオーストラリアにて先住民アボリジニの人々の楽器ディジュリドゥに出会う。
      2007年映画「地球交響曲第六番」虚空の音の章に出演。 世界の聖地や日本全国の神社仏閣にて奉納演奏を行なっている。 ディジュリドゥという木の響きを日本の伝統楽器にとの目標を持ち、日本人の精神・文化を学びながら、空洞の木の音、響きの本質と共に世界に発信している。
  • True Heart
    • スコットランドのエコビレッジ、フィンドホーンで出会ったミュージシャン、 Miri（ミリ）と、Joost（ヨースト）の2人が、2013年より、 音楽ユニットTrue Heart として活動し始める。 Miriのルーツである、ボサノバ、ジャズ的な曲や、Joostの多岐に渡る音楽活動から、 ケルティックな曲、即興的な作曲、複数の映画音楽など、様々なジャンルの音楽を手がけている。 2014年、アルバム「リーラ」をリリース。同年夏に、Miriは、フルフィルメント瞑想教師になり、 近年は、2人とも、FF瞑想実践と共に、瞑想音楽も手がけている。 現在はスコットランド、イギリスと、日本にて活動している。
