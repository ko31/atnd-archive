description

ABCとは「Android Bazaar and Conference」の略でAndroidに関わる人々のお祭りです。

今回のコンセプトは 「頑張れ日本、頑張れAndroid!!」

我々もAndroidに関わる一員です。我々にも何か出来るはずです。何ができるか一緒に考えましょう！

目的

来る7月17日(日)に開催されます、Androidバザールカンファレンス 2011 Summerへ出展する内容をリアルに議論したいと思います。

今回のABCには間に合わない場合は次回のABCで出展しようなど、短期から中長期的なスパンでABCをターゲットに行いたいことを議論したいと思います。

開催日程

開催日：2011年6月4日(土) 16:30～17:45［受付16:15］
場　所：千葉市生涯学習センター（JR千葉駅東口または北口から徒歩8分）
[会場には駐車場がございますが、できるだけ公共交通機関のご利用をお願いいたします。]

その他

・参加費
会場費は300円を考えております。( 第２回千葉支部Android勉強会のフォローアップ へご参加の場合は会場費は不要です。)

・勉強会について
当日は同一会場にて 第２回千葉支部Android勉強会のフォローアップ を開催しています。ご興味のある方は是非ご参加ください。

※当日は ビジネス / プライベート どちらでも結構ですので名刺をご用意頂くと交流が深まると思います。

お願い

今回ご参加頂くみなさまで、日本Androidの会千葉支部のメーリングリストへご参加頂いていない方がいらっしゃいましたら、是非ご参加をご検討ください。
