第18回Android勉強会in札幌

イベント ユーザー
event_id 41094
title 第18回Android勉強会in札幌
catch 今回はハンズオン&ハッカソン
description

今回は、初心者企画&ハッカソンということで、目覚まし時計アプリを主題とした初心者ハンズオンを考えています。
同じ会場を、初心者向けのハンズオン対象者と、ハッカソン対象者に分けて、平行に開催します。

初心者向けの内容:
初心者レベル(Hello world程度は自分で試したことがある人)を対象としたハンズオンを予定しており、目覚まし時計プラグインの開発キットを元に、実際にみなさんのパソコン上でオリジナルの目覚まし時計を作ります。
ボタンの設置方法や、画面の切替方法などを、スライドを見ながらみなさん一緒に作業します。

ハッカソン参加者向けの内容:
特に内容に縛りはないのですが、目覚まし時計プラグインを作るとみんなと共有できて楽しいのではないかと思います。
プラグイン構造を持った目覚まし時計は本体は、今年の2月頃にハッカソンで生まれたもので、目覚まし時計は本体が無いと、そのプラグインも動かないのですが、今のところクローズドソースなものの、勉強会までにはアプリ本体をGoogle playにアップ予定です。

以下のURLがサンプルとして公開しているもので、tamesarerualarmの方は天気予報の情報をWebサービスから読み込んで動作します。(いずれも完成度が低いので当日までにブラッシュアップしておきます)

https://github.com/maimuzo/pluggable_alarm_plugin_tamesarerualarm
https://github.com/maimuzo/pluggable_alarm_plugin_example
