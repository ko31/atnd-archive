●持ち込みについて
基本的に飲食物の持ち込みは自由です。
当日は富士金華豚や海鮮や地元の野菜やお米を使ったBBQや朝食をご提供します。
ビールやソフトドリンク類は300円で販売予定、氷も出来る限りフリーでご提供しますので、どちらかというと軽食や強めのお酒を持ち込まれた方が良いかと想います。また、火気や危険物等の持ち込みについても十分にご留意いただき、出来る限りお控えいただきますようお願いいたします。

●当日の気温について
当日は5月中旬ですが、高地のためすごく冷え込むことが予想されます。真冬並みの防寒着をお持ちください。また、天候が悪い場合、お手数ですが「タープ」をお持ちいただけるとすごく助かります。

●宿泊について
宿泊は基本的に持参いただくテントになります。どうしても用意できないという方は、別途主催者までご連絡ください。
佐藤：08039246358／yoji.of.the.clock@gmail.com

●会場について
こちらから様子は御覧いただけます。LINK→http://goo.gl/2uCsJ
「フロア」という概念はあまりなかったりしますので、地面の芝生で存分にゴロゴロしていただいて構いません。椅子やテーブルやシートもご持参いただき、ご自由におくつろぎください。

当日は20日12：00からは撤収になります。ゴミ袋はこちらで準備いたしますので、ご自身のゴミはご自身で片付けていただきますようお願いします。ゴミはこちらで処分いたします。
