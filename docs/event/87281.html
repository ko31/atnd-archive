屋守の会　第一部
日時：5月27日（土）　13時開始 ※12時45分開場
会場：吉祥寺　手打ち蕎麦　中清 （吉祥寺駅から徒歩10分）
会費：6000円
募集人数：22名
ゲスト：豊島屋酒造株式会社　田中孝治さん

第一部 13時開始の募集です。第二部 17時00分開始ではありません。
申し込みの際は注意事項をしっかりと確認ください。

屋守（おくのかみ）の会を開催するのはこれで5回目です。

屋守を醸す豊島屋酒造さんは東京の東村山にあります。

今年も蔵に伺い、田中さんから会への参加を快諾いただくことができました。

例年、非常に人気が高い会で、1日で満席になってしまった年もありました。

より多くの方に参加いただくために昨年に引き続き、2部制にして1日に2回の会を行うことにしました。
今年は曜日を土曜日に変更しましたので、より多くの方に参加いただけらと思っています。

昨年から屋守はラベルがリニューアルされ、生酒と火入れとより見た目で楽しめるものになりました。
味わいは今年も間違いないものに仕上がっています。美味しいです！！

会には屋守に加え、他にも豊島屋酒造のお酒を揃えます。田中さんが試験的に作り「参加者の皆さんから感想をいただきたい」というお酒も特別に出品します。

田中さんから今期の造りについてたっぷりお話しをいただきます。

会場は今年も中清さん。屋守は中清さんで大プッシュの銘柄です。中清さんの蕎麦屋ならではの蕎麦前やシンプルですが飽きのこない料理、〆には東京を代表する味わいの蕎麦を屋守とともにお楽しみいただきます。

初めての方も大歓迎です！！屋守ファンの方（通称・ヤモラー）、お酒の会が初めて方など…心地好い雰囲気の中で、皆さんに楽しく飲んでいただくことを一番大切にしている会です。

土曜日の昼に、田中さんと一緒に楽しく飲みましょう！！
