catch SFS IBIZA presents ETNICAーMax Lanfranconiによる「DJワークショップ」と「ディナー＆トークギャザリング」
description

サイトランスの創世記より業界のリーダー格としてシーンを引っ張って来たETNICAのMAX。

イビザにて設立した音楽映像スタジオSFS (Sound Farm Studios Ibiza)にて、 全世界のミュージックラバーに向けて、１ヶ月間の音楽制作集中コースを企画。講師陣には、現役で活躍するトップアーティスト達を起用。第一期生が昨年春、イビザを訪れ、１ヶ月間、音楽を集中的に学び、フレンドリーな講師陣やイビザパーティー関係者らと親交を深め、音楽の知識だけでなく、貴重な経験を積みました。

このSFSミュージックスクールの創始者＝MAXの日本ツアー中に、授業内容には含まれないDJトラクターとコントローラーの使い方について、また、希望者にはライブのセッティングなどのコンサルタント、音楽制作プライベートレッスンなどを行います。現在のシーンにおいて、もはや音楽制作することとLIVE/DJすることは切り離せなせません。音楽活動をしたい方は、是非、この機会にご参加下さい。

このワークショップの後、場所を移動して、食事座談会を行います。イビザのSFSスクールの事、今後の音楽ビジネス、シーンについてなど、MAXから話を聞いたり、質問に答えてもらったりしながら、一緒にお食事とお酒（飲み放題）を楽しみましょう。
