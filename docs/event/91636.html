メイドさんとプログラミング！
Unityでゲームを作ろう【初心者レベル】
VR/AR技術に必要な、Unity、C#プログラミングの講座です。

プログラミングの勉強をしたいけれど、何から勉強したらいいのかさっぱり分からない方や、
1人だとなかなかやる気が出ない方におすすめです。

プログラミング講師の他に、メイドさんが講座に参加します。
メイドさんたちは、一緒に学ぶ仲間として生徒の皆さんにやる気を注入してくれます！
ゲームを完成させて、メイドさんたちと喜びを分かち合いましょう！
2回目以降のご参加時には、前回の続きから勉強する事ができます。

〈交流タイム〉がありますので、IT、VR、関係の情報感度の高い方や、経営者の方など、様々な人と繋がる事ができます。
