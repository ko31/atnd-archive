description

MyDNS.JPを使った自宅サーバーの仕組みとその運用について勉強する会を、10月23日(土)に広島(広島県情報プラザ)にて開催したいと思います。



内容ですが、これから自宅サーバーをやってみようかなという方や、とりあえず動かしているけれど実際のところどういう仕組みなのか理解したいな、というような初心者向けとなります。

自宅サーバーはじめたいという方や、IT系社会人や学生さんで一年目の方など、ダイナミックDNSや自宅サーバーの仕組みや運用、またよく混乱してしまう点などを判りやすく解説したいと思います。



また午後からはVineLinuxでの各種サーバー設定の方法(LAMP環境の構築)を易しく解説すると共に、Windows7を使った自宅サーバーの実演(WAMP環境やメールサーバーなど)もしたいと思います。
