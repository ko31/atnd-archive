■内容
立案～予約はファンコミュニケーショズって会社にいる、さくま（ http://twitter.com/sakuma1210 ）です。
社内から連れて行くメンバーは恐らく10名～15名くらいですが、恐らく皆社交性があるヤツだと思っております。
今回も同業者～メンバーのお知り合いの方を大歓迎します

初心者歓迎、恐らく半分以上は初心者の可能性もあります。
道具はレンタル可能。
玄人も勿論歓迎。

質問連絡は、twitterにて sakuma1210 もしくは sekine_san まで。
ryo1210@gmail.com（さくま）まででもおｋ。

■持ち物
・銃・ゴーグル
無い方はレンタルできます。
法規制範囲内でお願いします。
（0.2gBB弾で初速95m/s、0.25gBB弾で初速85m/sくらい？）

・迷彩服
※今回はレンタル迷彩服が無いです！
普段着でもかまいませんが、迷彩系の服装推奨（木々に紛れるとホントに見えません）
スーツ、ドレス、着ぐるみ等ネタに走るのも有り。ただしアウトドアなので自己責任。
途中雨が降るかもしれませんので、下着着替えor雨具があるとベター。

・帽子
なんでもいいです。迷彩にあうヤツとか地味な物？
でもなんでもいいです。
防災用ヘルメットでもOK。

・手袋
手に当たった際の痛み軽減用です。
地味な物が目立たないです。軍手でもOK。
滑り止め付きじゃないと、銃がつるつる滑るかも。

■お昼等
現地で、昼飯をお願いしようと思います。500円
ただ、少ないかもしれないので、コンビニ等で足しになるものを買っていくことを推奨します。
疲れるのでチョコレートとかあるとイイかも。

■こまかいルールは当日説明します。
レギュレーションはふつーなかんじです。
味方打たれはHIT、跳弾はOK、ゾンビは気をつける。「今当たったはずだろー」とかは言わない。
ナイフアタックは基本無し。フリーズコールは紛らわしいからやめよう。
という紳士的なゲームです。
そんな感じ。
