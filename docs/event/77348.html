はじめまして、Webデザイナーで運営の石原です。
8年間フリーター。Web初心者だった僕が、勉強し初めて3ヶ月で"未経験"からWeb業界へ転職できました。
転職できた一番の理由は、"現役で働いてるクリエイターと出会えた"からです。
なので、そんな"Web初心者"の方と、現役クリエイターの方が繋がるイベントを企画しました。

【応募対象：以下で悩んでる方は応募対象の方です。】

・Web業界で現役で働いてるプロの人と繋がりたい
・パソコンのスキルを学んで、手に職を持ちたい方
・将来在宅で働く事も視野に入れたい
・Web系のイベントやセミナーに参加したことがない
・Webのスキルが欲しい
・Web業界に転職（就職）したいが、相談できる人がいない
・Web業界で働き始めたけど、悩んでる。あともっと会社以外で繋がりが欲しい
・やりたい事を形にできる仕事がしたい
・クリエイティブな仕事をしてみたい
・頑張れる環境や仲間が欲しい
・そもそもWeb業界がどんなところかわからない

以上の「Web業界に転職したい人」「Web業界で働き始めた人」の為のイベントです。
※ベテランの方も参加可能ですが、初心者向けコンテンツになってますので、そこだけご了承いただけたらと思います。

【概要】

Web初心者の方とプロの方が繋がるように、
現役で働いてるクリエイターの方をゲストに迎えて話が聞ける「トークセッション」。
ゲストとクリエイターであるスタッフと参加者を交えた「グループワーク」。
イベント後、参加希望者とゲストおよびスタッフで軽食を食べながらの「交流会」を予定しています。
その他にも、イベントだけでは物足りなかった方の為に、初心者クリエイターやベテランクリエイターが所属してるコミュニティ（facebookグループ）にご招待します。
そこでは、情報共有やその他イベント告知、定期的な交流会を予定しています。
その他、特典もご用意しておりますので、イベント詳細ページにてご確認ください。（ thecreative.jp/svh-beginner/ ）
