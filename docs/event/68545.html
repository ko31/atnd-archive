description

“島根で働きたい”というエンジニアと、県内IT企業との交流イベントを開催！
近年、都市部からIT企業の進出が増えていたり、地元の面白い企業に注目が集まる島根県。
「実際、地方でのキャリア形成ってどうするの？」「自分のスキルを生かした仕事はあるの？」などの疑問や不安を解消します。
ゲストスピーカーには伊藤直也氏が登壇。
エンジニアの、「地方と都市でのキャリア形成」について語っていただきます。

あなたにぴったりの一社に、ぜひ出会ってください。

プログラム

イベント参加はこちらからも

【第一部】基調講演（13:00～14:00）

伊藤直也氏が語る
「エンジニアにとっての東京と地方、あるいはエンジニアのキャリア形成に環境が与える影響について」（仮）

【第二部】交流会（14:00～17:00）

「エンジニア×島根県IT企業」交流会
島根県に立地するIT企業が都内に集結。県内IT企業の情報収集や、気になることを採用担当者から直接聞けます。
