description

金持ち父さん貧乏父さんを読んで、「投資の勉強をするぞ！」
と思うけど

まずは何から始めたらいいのか分からない!

株投資や不動産の参考書は難しすぎる。。。

そんな方のはじめの一歩として

オススメなのが
このキャッシュフローゲーム！

これは
金持ち父さん貧乏父さんシリーズの著者である
ロバートキヨサキ氏が開発した

ファイナンシャルリテラシー（お金に関する知識）を
高めるためのボードゲーム型教材です。

モノポリーと人生ゲームを足したようなゲームで
あなたが貧乏父さん世界（ラットレース）で
株や不動産でお金をふやして
金持ち父さんの住む世界（ファーストトラック）に

行くことが目標のゲームです。

当会は
キャッシュフローゲームとロバートキヨサキ氏の本を通して
学校や通信教育では学べないお金の勉強を、
いろんな職業や年齢の方と一緒に学んで頂けます。

今回は初心者向けにゆっくり進行しますので
ルールを知らなくてもご安心ください。

しかし、一度キャッシュフローゲームをやっただけで
現実世界で経済と時間を手に入れるのはほぼ難しいでしょう。

その為に
ご希望の方には、実生活でどう活かせば良いのか
一度だけ無料の個別相談も承ります。

あなたが踏む出すべき第一歩がここにあります。

【内容－流れ】
1．自己紹介
2．キャッシュフローゲーム

【こんな方にオススメ】
・金持ち父さんの本を読んで刺激、影響を受けた方
・金持ち父さんの本を読んで行動したくなった方
・本は読んだけれど、何をすれば良いのか悩んでいる方
・不労所得を増やして、経済と時間の自由がほしい方

【参加対象】
勉強会を通じて人生をより豊かにすることを目的とした、下記項目のどれか1つでも当てはまる方
・自分の将来の経済状況について真剣に考えている方
・起業における情報を得たい方
・様々な価値観に触れたい
・人脈を広げたい

ゲーム会の流れ
１．金持ち父さんの考え方について説明
２．キャッシュフローゲーム説明
３．ゲーム開始
４．終了後内容のシェアリング

※終了後、希望者のみで懇親会を行います。
