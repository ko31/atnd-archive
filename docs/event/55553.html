description

マーケティング庵は、みなさまの暖かいご支援のおかげで、今月で設立10周年を迎えることができました！大変感謝しております。

そこで、10周年を記念して特別イベントとして、この会を長年にわたって主催してきた私レフティーとSatoRichimanさんによるトークセッション【10年間の振り返り】を開催します。

これまで10年間で62回の研究会を開催してきました。この10年間でどういうテーマを取り上げ、そこからどういうことがわかったのかを検証していきます。

「あのテーマの会に参加したかったのに…」という方、是非お勧めです。
また、今回は普段開催しているフリートークの会ではなく、講演形式で進めますので、過去ご参加いただけなかった方も、気楽にご参加いただけます。

休憩時間には、スイーツの試食ができるかも…？

トークセッション終了後は、10周年記念パーティーを開催します。

みなさまのご参加をお待ちしております。

尚、会場はいつもの渋谷ではなく原宿ですので、ご注意ください。（下記参照）

＊ 「マーケティング庵」は、身近な話題をテーマに生活者の視点から素人の感覚で、自由で気軽なフリートークスタイルで掘り下げるユニークなマーケティング研究会です。

今回は講演形式で開催しますので、どなたでもご参加いただけます。
お友達をお誘いいただいても、お一人でも気楽にご参加ください。
過去にご参加されたことがある方も、初めての方も、この貴重な機会をお見逃しなく。
お申し込みはお早めに！
