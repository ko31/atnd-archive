こんな想いを持ってるITエンジニアの方が対象です

  • 札幌に戻って仕事したい
  • 東京以外で仕事したい
  • 自然があるところで暮らしたい
  • スノボ大好き
  • 北海道で暮らしたい

そう、Uターン/Iターンです。そんな人達の情報交換の場所にしてください。

30分間だけクラスメソッド札幌の紹介をさせて下さい。それ以外はご自由に情報交換をして頂く形になります。
現地札幌と中継をして、先にUターン/Iターンした人達の生の声もお届けします。軽食と飲み物（アルコール/ソフトドリンク）を用意いたします。

申し込みはhttp://dev.classmethod.jp/news/job-fair-2015-0204/でも受け付けています。（こちらは申し込み者が公開されません）
