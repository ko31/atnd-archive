title 一歩先のケータイを作ってみる会
catch （別名：カスタムROMハッカソン）
description

ケータイの「こうだったらイイのに」を形にする勉強会です。

普段使っているケータイ、特にスマートフォンのちょっとした不満点、改善点を共有し
実際にケータイ自身を生まれ変わらせてしまおう！という事を目的にしています。

プログラマだけでなく、デザイナの方のご参加もお待ちしています。
機能を足したり変えたりするだけが改善ではありません。

画像の差し替えや、UI部品の色合いの調整をするだけでもケータイの雰囲気は一気に
変わります。

アイデアの大小にこだわらず、今より楽しいもの、カッコいいもの、可愛いもの、
ワイワイ話しながらアナタの空想を形にしてみませんか？

【補足】

(1)
実際にアウトプットを形にするため、改善のターゲットとなるスマートフォンはAndroid機を
想定しています。なお、端末はご持参下さい
（ただし持ち込み必須ではありません。端末のない方の参加も大歓迎です。GMO様の
　ご厚意により、数台の端末をお貸出し頂けるとのこと）

(2)
端末に書き込むOSは、Androidでなくとも構いません。

(3)
GMO様のご厚意により、端末に書き込むためのROMイメージをビルドするVPS環境を
お借りすることが可能です。
各自作業用のPCは必要ですが、高性能なものをご持参頂かなくても問題ありません。
（事前の環境整備がどこまで可能か？は検討中です）

■参加費
無料です。
