description

仕事に縛られた人生を終わらせたくないあなたに贈ります。
もっと自分らしく生きてみませんか？

私は保育士として都内で勤務しています。
子どもも好きでやりがいも感じて勤務していました。
しかし、安月給・残業・持ち帰り業務・・・。
年数を重ねるごとに責任も重くなり、頑張れば頑張るほど仕事は増えていきますが、給料は増えず。。。
頑張る程損をする日々に仕事へのやる気もどんどん薄まっていきました。

もちろん、子どもとの関わりはとても楽しくやりがいも感じていますが、お金もなく、時間もない。将来に不安を覚え、趣味も多くプライベートも充実させたいという考えの私は、自分の大事な人生を仕事のためだけに使ってしまって良いのかと疑問を感じ始めました。

そんな時に紹介してもらったのが、ロバート・キヨサキさんのベストセラー、
「金持ち父さん・貧乏父さん」という本でした。

その本を読んでみると、まさに衝撃Σ(゜.゜)！！
Ｂクワドラント！？
こんな働き方があったんだ！

Bクワドラントにはどんな人がいるの？
もっとお金の勉強をして、Bクワドラントの働き方について学びたい！

そこで、本にも書いてあったキャッシュフローゲームというものをどこかでやっていないかとネットで検索したところ、見つけたのがこのゲーム会でした。

はじめは何もわからず不安も多かったですが、一歩ふみだしたことで、実際にBクワドラントになって時間もお金もある生活をしている人と会うことができたり、意識の高い方々やいろんな知識を持った方々とも知り合うことができ、自分の視野がどんどん広がっていきました。

ここで得られた経験を、自分と同じ想いを持つ人々にも伝えたい。
そんな想いで今ではゲーム会を運営しています。

初心者の方にも分かりやすく解説しながら進めますので、
女性やお金の知識がない方でも安心して参加できます♪

勧誘などは一切ありませんので、お気軽に参加してくださいね♪

■ゲーム会の流れ
 １．金持ち父さんの考え方について説明
 ２．キャッシュフローゲーム説明
 ３．ゲーム開始
 ４．終了後内容のシェアリング
 ５．カフェにて交流会（自由参加）

 ＜ゲーム会のご案内＞
 参加費：1000円（会場にて申し受けます）
 持ち物：筆記用具、電卓
