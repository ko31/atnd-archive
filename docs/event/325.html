2009/2/4：非常に他数のお問い合せを頂いており、ありがとうございます。ATNDの参加人数に加え、直接事務局に連絡が来ている分を合わせると、会場の座席数（84席）に届く勢いとなっております。ご来場頂く時間によっては、お座りになれない可能性があることを、ご了承ください。大変申し訳ございません。

共通のユーザIDを複数のWebサービスで使える話題の技術、OpenID。今や全世界で5億以上のOpenIDが発行され、対応サイトも増え続けています。


その仕様や周辺技術を学べる無料セミナーOpenID Tech Night Vol.4を開催します。


新規にOpenIDに取り組みたい方や、もう一度基礎から技術を学びたい方のために、OpenIDへのファーストステップとなる、よりベーシックな部分についても解説します。


OpenID Tech Nightには、どなたでも参加いただけますので、皆様、是非お気軽にお越しください！

●スピーカー

  • tkudo氏：「ID技術　最新動向2009」
    デジタルアイデンティティ業界屈指のエバンジェリスト。
    某外資ベンダーにて、主にアイデンティティ管理に関する営業支援・構築を経て、
    同分野の事業開発に従事。昨年より心機一転？新天地でもアイデンティティ・ビジネス
    にひた走る。様々な雑誌寄稿や、リバティ・アライアンスなどでの多数の講演に加え、
    IPA情報セキュリティ技術動向調査タスク・グループ委員といった幅広い活動を行う。
    OpenIDに関わらず、国内外の様々なアイデンティティ技術動向を俯瞰できる数少ない人物。

  • ZIGOROu氏：「OpenID 認証 2.0 入門」
    バーテンダーからプログラマに転職した唯才の人。Perl Monger。大規模コミュニティサイトの開発・運用を経て、Shibuya Perl Mongersイベント等、多数の講演活動を行う。ITmediaや@ITなどに連載記事を寄せる傍ら、自らのブログ「Yet Another Hackadelic」でOpenIDへの熱い思いを書き連ねている。最近はモバイル分野でのOpenID利用の実現へ向けて活発な情報発信を行っており、その動向が注目されている。
