◆　(株)シンクパワー（http://www.syncpower.jp/）： 　
2006年の設立当初から、「歌詞」に特化した事業を展開している。特に、歌に合わせて流れる歌詞である同期歌詞サービスに力を入れている。2009年10月からソニーの“ウォークマン”向けに曲を再生すると歌詞が自動スクロールする「歌詞ピタ」サービスの運営をスタート。
2011年にはスマートフォン向けに同期歌詞表示プレイヤーアプリ「プチリリ」をリリース。「出ない歌詞はユーザーが投稿」できるユニークなシステムも話題を呼び、160万ダウンロードの人気アプリとなっている。
2012年からプチリリ機能を他社アプリにも短期間で実装できるSDK（Software Development Kit）の提供を開始し、大手CPを中心に多くのアプリに採用されている。
自社でも100万曲の歌詞データベースを生かし、複数のスマートフォン向けアプリを展開。2014年5月には音楽に合わせてスクロールする歌詞(同期歌詞)を作成できる無料アプリ「プチリリメーカー」のiOS版とAndroid版をリリース。対応曲数は国内最大級の約100万曲。併せてプチリリWebサイトを「歌詞投稿コミュニティ」としてリニューアルした。当日は、ユーザー、アーティスト投稿も含めた｢歌詞サービスのエコシステム｣について、そのきっかけと経緯、現状、そして海外展開も含めた今後の展開について説明する予定。
＜プレゼンター＞　　　　　代表取締役社長　　　　冨田　雅和　氏

◆　㈱ファミマ・ドット・コム (http://company.famima.com/)：
　国内・海外店舗数1万5000店超と拡大をつづけるコンビニエンスストア、ファミリーマートの店舗ネットワークを基盤に、そのEリテール事業を担う会社として2000年に設立。同年ファミリーマート店舗に、“オンライン”と“リアル”（店舗）の間に位置し双方の誘導を図る目的でマルチメディア端末「Famiポート」を導入した。
この端末を通じ、企画した各種興業チケットやプリペイド、EC収納などの各種サービスを来店客へ提供する。2010年に伊藤忠商事(株)、(株)ファミリーマートと協力し「Famiポート クーポン」をリリース。このサービスでは、来店客にID を発行し、IDを入力することで「Famiポート」を通じてクーポンを発行、店舗商品との交換や値引きサービスを提供することができる。2014年6月からは、直営店で「Famiポート」の刷新を開始。同9月にはフランチャイズチェーン(FC)店でも入れ替えを始める。
また、2014年5月29日よりファミリーマート店内設置のマルチコピー機から、アニメキャラクターのイラストなどの画像を印刷できる「ファミマプリント」サービスを開始した。このサービスは各社のコンテンツサイトからお気に入りの画像を選択し、プリント番号をファミマプリントに入力してブロマイドなどが購入できる。当日は、新型Famiポートとコピー機のファミマプリントサービスを中心に最新動向について説明する予定。

◆ Ustream Asia (株) (http://www.ustream.tv/)：
TVバンク(株)とインターネットを利用したライブ動画の配信サービスを提供する米国のUstream, Inc.の両社が共同で、アジアでの事業展開を目的に設立した合弁会社。日本では、2010年4月にUstream日本語版サイトの提供を開始。これまで、Ustreamは不特定多数の一般ユーザーが視聴するという使われ方から、さらに企業の中で活用されたり、特定の目的を持ったユーザーが使用したり、さまざまなシステムにUstreamを組み込んで使用する事例が増えてきた。
コンサートやイベントのライブ中継も一般的になり、アイドルグループ・ももいろクローバーZが5月17～18日にかけて生放送した24時間特別番組「Ustream大賞受賞記念 24時間いただきますっTV 2014 ～美しく 楽しく～」の総視聴者数は約165万人となり、　国内過去最高記録を更新している。2014年2月に発売されたソニー・コンピュータエンタテインメント（SCE）の「PlayStation 4」（PS4）ではUstreamの配信機能が搭載され、Ustream上で映像を配信するユーザーが急増した。PS4ユーザー1人あたりが配信したゲームプレイの平均時間は40分に達している。
多様化しているUstreamの利用方法とともに、PS4を使った映像配信の事例など最新の動向を説明する。
＜プレゼンター＞　営業部　営業支援課　マネージャー　早川　弘之 氏
