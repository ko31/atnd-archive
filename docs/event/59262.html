第1４回目となる2015年8月公演に向けて出演者募集！！
半年に1度、5歳～80歳迄の一般公募で集まったメンバーで結成し、公演後解散する 市民参加型・期間限定劇団
「座・大阪市民劇場」201５年８月公演に向けてメンバーを募集！
毎回、約30名～40名のメンバーが300人の前で舞台に出演。
実行委員は、プロがボランティアでバックアップ。 普段の生活では、出会うことのない幅広い世代の仲間とふれ合い、つながる人の繋がりもこの舞台ならではの特徴。本気で役者を目指す方もプロの演出家の指導の元、舞台をふめるチャンス！ もちろん大阪以外の方も参加OK
■応募資格
5歳～80歳迄の男女（８割以上稽古に参加出来る方）
　
■応募方法
まずは2014年11月30日（日）14：00～の説明会へご参加下さい。
やってみたい意思を下記連絡先へ(名前、年齢、職業、連絡先）を記入の上、メール、郵送、電話にてお問い合わせ下さい。
会場場所他詳細は申し込み後、担当者より連絡致します。

■説明会＆オーディション

日程：2014年11月30日（日）14；00（受付１3：30～）

場所：大阪総合生涯学習センター　第3研究室　（大阪第2ビル　５．６階）

持ち物；筆記用具、顔写真（指定サイズなし）　参加費：無料

※この市民劇場説明会は合否を決めるものではなく、みなさんの持っている個性を、演出の先生にみて頂くオーディションになります。

■合格後
色んな世代の方と交流しながら稽古を行い201５年８月に行なう
「第14回　座・大阪市民劇場公演」に出演

◆稽古日程　 １２月ー８月の毎週日曜（14時～16時※会場の状況で曜日変更あり）

◆公演日　201５年８月８日（土）９日（日）に舞台上演後、解散です。

◆演目「また逢う日まで」（仮）参加者の個性をみて変更有 演出は元梅田コマ劇場の仲風見先生
※公演制作費、スタジオ維持費として月1.5万円必要（その他費用無し）
