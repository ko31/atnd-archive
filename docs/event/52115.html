デザイン」という定性的な分野だからこそ WEBデザイナーは活発に情報共有して学び合いによってレベルアップしよう

※第1回は小規模に10名で行いますが今後拡大予定です。

　

第一回のテーマ「バナー祭り」

WEBデザイナーなら一度は関わることになるバナー制作。

メインのワークショップの時間にて、全員で「あるテーマ」に沿ったバナーを制作し、チームで良い所・改善したほうが良い所をディスカッションします。

人それぞれの"感覚の違い"を受け入れて、バナー制作のレベルをアップしよう！
また、第二部のコード編ではCSS3を用いてバナーをアニメーションさせる方法についても勉強します。

　

勉強会の概要と構成

勉強会では、有志による発表を聞く時間と、全員で参加するワークショップ（ハンズオン）が必ず含まれる構成にしています。

WEBデザイナーの能力は大きく分けてデザイン、コードの2種に分かれるため、２部構成で双方に発表とワークショップを含めるようにしています。

　

第1部:デザイン編

  • 発表・プレゼン
  • ワークショップ

第2部:コード編

  • 発表・プレゼン
  • ワークショップ

知識を仕入れること、実践形式のワークショップを両方盛り込むことで、幅広い参加者に満足してもらえる形を目指します。

発足の背景

  • 関西にデザイナーの勉強会が少ない
  • 勉強会の目的は情報共有かレベルアップである
  • レベルアップは大きく分けて「新技術の習得」「既存技術の効率化」である
  • たいていの勉強会は「新技術の習得」や「個人的発表」に偏っているので、このバランスがとれるものがほしい
  • バランスをとってワークショップを入れれば、定性的な分野であるデザインなら初心者も上級者も学べる
  • 定期開催で関西のWEBデザイナーの「基地」を目指す
