description

太古より吉事凶事大きな謀をする際には行われてきたアナログゲーム会ですが
僕らも大人になってあんまり集まる事がなくなったので
こういう場を作って集まってやればみんな楽しいんじゃないかなーって
思って作りました。

八王子スクエアビルの上に立ち愚民を見下ろしながら全員ワイン片手にひざにシャムネコでやるプレイというのも一興グフフでありますな。
初めてやるので進行に不具合があるかもしれませんが
ご参加される方どうぞお付き合い下さいませ。

昼~夜の二面取れたので　12時から20時か22時ぐらいまでガッツリやれます
サウンドルームなので半防音、BGMを流してもよし、
ちょっとのご飯ならその場で食ってもよしだそうです。
持っていくゲームは

ドミニオン
カタン
あやつり人形
アーカムホラー(未定）
ラストナイトオンアース(ゾンビのボードゲーム

等々を予定しております、
皆さまもこれ持って行ってみんなとやりてえ！ってのがあったなら
持って来てくださいませ僕もやりたいです。

バトスピデッキ持ってきてもいいよ。

あとアレだ16名ぐらい入る部屋なので誰か8人マリオカート持ってきてれば
8人でマリオカートができますよ!プロデューサーさん!
アナログじゃないけど。
