Google Cloud Platform 活用法 ～本格利用の前に知っておくべきこと～

description

急速に拡大し続けるクラウドコンピューティングに対し、業種業界を問わず様々な企業が何らかの対応を求められています。

「Google Cloud Platform 活用法 ～本格利用の前に知っておくべきこと～」と題した本セミナーでは、グーグル・クラウド・ジャパンからの登壇者も含め、GCP利用の最初の一歩から本格利用に役立つノウハウまで、最新情報をご紹介致します。

是非ご参加いただきたく、ご案内申し上げます。



■セミナー概要
日時：11月21日(水)　14:30-17:20（受付開始：14:00）
会場：ヒルトンプラザウエスト（第二吉本ビルディング）・オフィスタワー8階　大阪市北区梅田2-2-2
主催：株式会社オージス総研／協賛：グーグル・クラウド・ジャパン合同会社
定員：40名　※参加費無料（事前登録制）

■プログラム
1）Google Cloud 最新情報
　講演者：グーグル・クラウド 西日本営業統括 中島 謙二氏
【講演概要】
　Google Cloud の日本での取り組み、および Google Cloud Summit の大阪初開催をはじめとする西日本エリアでの取り組みなどGCPの概要と最新動向をご紹介します。

2）Google Cloud Platform はじめの一歩 ：エンタープライズ分野での利用のために
　講演者：株式会社オージス総研 プラットフォームサービス本部
　　　　　　IT基盤ソリューション部 クラウド推進第二チーム　村上 希夢
【講演概要】
　GCP が提供するセキュリティモデルや考え方、ユーザ側で実施する必要のあるセキュリティ対策などについてご説明します。

3）次世代EDIサービスを実現するための GCP 活用のコツ
　講演者：株式会社オージス総研 プラットフォームサービス本部
　　　　　　クラウドサービス部 クラウドサービス第一チーム　小林 将也
【講演概要】
　当社が描く次世代EDIサービスと、サービス提供に GCP を選んだ理由、アーキテクチャをご紹介します。

4）Google Cloud Platformの強みと国内事例
　講演者：グーグル・クラウド カスタマーエンジニア　森永 大志氏
【講演概要】
　他社クラウドとの比較検討の際に、GCP を高くご評価して頂く機能をご紹介いたします。また国内でのユーザー事例とエンタープライズ利用にあたってのポイントをご説明します。
