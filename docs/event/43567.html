MozCafe とは？

MozCafe (もずかふぇ) とは、Firefox など Mozilla の製品が好きな人、開発やドキュメント作成やユーザサポートなどに協力してくれる人、ユーザ、オープンソースが好きな人などが気軽にオフラインで集まって情報共有するオフ会イベントです。

Firefox, Firefox OS, Thunderbird などを提供している Mozilla では、ソフトウェア開発や日本語対応はもちろん、ユーザや開発者向けのヘルプやドキュメント執筆や翻訳、ユーザサポートまで、すべてオープンにコミュニティ (有志のボランティア) と共に行っています。

現在 Mozilla で活躍している人が、実際どんな感じで一緒に活動しているのか、どんなところから参加・協力いただけるか、困った時は誰に聞けば良いのかなどを紹介して、皆さんの疑問に答えたり実際に興味を持って頂いたことを一緒にやってみます。

Mozilla の活動やコミュニティに興味がある方、一緒に何かしてみたい方、みなさん気軽にお越しください。

スケジュール:

    • Firefox などの日本語化
    • Firefox などの開発
  • 20:30 – 22:00 一緒に何かしてみよう＆何でも疑問質問お答えします

    • 疑問や質問があれば何でもお答えします
