description

事業を始めて成功する一番の方法は、なりゆきに任せることです。

計画から始めることは悪いことではありませんが、今はどんどん流れが変わって行く時代。 なりゆき起業の経営者は、最初からマーケットの動向を見て対応を変えます。時代の流れに逆らわず、変化に対応できる人しか、これからの起業経営はできません。
成功の秘訣は、「成功者の知恵をパクる努力」。よい考えがない時は「だまって人の言うことを聞いておく」。

なりゆき起業のすすめセミナーでは、税理士法人スプラウト・株式会社スプラウトビーンズ代表中野由里が 起業後も成功し続けたい人のために、成功事例を交えて、自らの起業の経験や 税理士業務を通じて学んだ、ありとあらゆる知恵をお話します。 ただ聴くセミナーではなく、ワークショップを交えた参加型セミナーです。 ぜひ、独立開業をした方・独立開業予定の方はご参加ください。 そして、成功するためのコツをつかんでいってください。
