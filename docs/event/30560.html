description

　8月29日に発売される『ミクの日大感謝祭コンプリートBOX』と、8月30日に発売される『初音ミク Project DIVA f』の発売を記念して、皆で盛り上がろうという趣旨のオフ会です。

公式イベントではありません。有志による手作りのオフ会です。
途中入退出は自由です。Blu-ray上映会だけ、ゲーム会だけでも問題ありません。
再入場については、まねきねこディノス札幌中央店の都合により、10分以内に戻っていただく必要があるそうです。長時間の一時外出は、再度料金が発生しますので、ご注意下さい。

【オフ会の概要】

1.Blu-ray上映会
　開催メンバーの方にプロジェクターをお持ちの方がおりますので、その方からプロジェクターを借りて、大画面でBlu-rayを上映する予定です。
　2012年3月8日～9日、奇跡の二日間の記憶を掘り返りつつ、皆で盛り上がりましょう。

2.『初音ミク Project DIVA f』ゲーム会
　『初音ミク Project DIVA f』を皆で持ち寄って、ゲームを遊ぼうという企画です。
　難しい曲を上手いプレイヤーはどのようにクリアしているのかなど、見せてもらえるかも？
　前作同様に名刺交換機能があれば、名刺交換もぜひどうぞ。
　AR機能を用いて、ミクさんと撮影会なんてのも面白いかも知れません。

3.『初音ミク Project DIVA ドリーミーシアター extend』ゲーム会
　今のところ発売日未定ですが、オフ会開催日までに発売された場合、『初音ミク Project DIVA ドリーミーシアター extend』のゲーム会も行います。
『初音ミク Project DIVA ドリーミーシアター extend』が未発売の場合は『初音ミク Project DIVA ドリーミーシアター 2nd』のゲーム会となります。
　幹事がPS3と3D立体視対応ディスプレイを持ち込みます。開催メンバーの方にもPS3を持ち込んでいただき、2台体制でお待ちする予定です。
　3D立体視未体験の方には、お試し頂くチャンスです。
