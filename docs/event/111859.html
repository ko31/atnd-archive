description

妊活を始めると、妊娠できない問題点と向き合うがために、気になることが、どんどん増えてしまいがちではありませんか？

このセミナーでは「すぐ妊娠する人」を徹底検証した結果、

授かるために

・「やるべきこと」
・「やめるべきこと」

を明らかにしていきます。

◇ すぐ妊娠する人の特徴は？
◇ 妊活がうまくいくカップルの習慣は？
◇ 効果的なサプリや漢方の活用法は？
◇ 排卵日は気にしていた？
◇ 妊活中の食事は？
◇ 授かるための心の整え方
◇ 授かるためにやるべきこと・やめるべきことは？

結局のところ、授かるためにやるべきことは何で、やめるべきことは何なのでしょうか。
その答えを明らかにし、シンプルな妊活の方法をご提案します。

【講師】
伊藤優子
元看護師、妊活カウンセラー。看護師時代、大学病院ならではのハイリスク出産、流産、死産など、出産という喜びだけでなく、悲しい場面にも数多く立ち会う。10年ほど看護師として臨床経験。産婦人科での勤務を経て、2006年、2010年、2016年にそれぞれ女児を出産。娘を重症仮死出産した経験を生かし、2013年10月からカウンセラー活動を開始。65名の妊娠実績。マタニティケア、マタニティセラピスト育成講座も開講。著書に『72時間! みんな変われる魔法の妊活』『半年以内にママになる妊娠セラピー～ママになるための12のレシピ～』がある。
