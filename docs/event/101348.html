概 要

企業活動においては、社内・社外向け問わず、顧客満足度調査や市場調査などアンケートを実施することがよくあります。

LimeSurvey は、アンケートの作成、公開、集計、解析の作業をブラウザ上で簡単に行うことができ、様々な用途に対応できる世界的に有名なオープンソースの高機能WEBアンケートシステムです。 　
本セミナーを通じて、LimeSurveyを使って、オンラインアンケート作成から、回答、回答結果確認まで体験していただきます。

【アジェンダ】

　1.LimeSurveyとは？ 　　　　　　　　　
　2.現在のバージョンとこれまでの歴史 　　　　　　　　　　
　3.動作条件について 　　　　　　　　　　　
　4.LimeSurveyのインストール 　　　　　　　　　　　
　5.LimeSurveyでのアンケート作成 　　　　　　　　　　　
　6.アンケート回答 　　　　　　　　　　　
　7.アンケート結果確認 　　　　　　　　　　　
　8.クローズアンケートの実施方法について
