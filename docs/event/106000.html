description

本は読んだら実践しないと意味が無い!

「○○する方法」 などのノウハウ本を
読んだ後、どうしていますか？

何かの知識を得たくて本を買ったんだから
もちろん実行しているよ!
という方もいると思いますが、

色々な方の話を聞くと
「本を読んだは良いけど、何も実行せず…」
という方が多い事に気づきました。


ダイエット・英会話・片付け・自己啓発…
どんな本であれ

本の内容を実践しなければ、本を読んでないのと
結局は同じ。

　↑
( あと、本代がもったいない )


そこで、

「本の一部 (一章) だけでも良いから、
『実行した!』『やってみた!』と報告しあえる
場を作ろう」
と思い、この読書会を始めました。

本のジャンルは問いません。

● 本を持ってきて紹介する
● 本の一部で良いから、何か実践する
● 再び読書会に来て、実践結果を報告する

こんな流れを参加者みんなで作りたいと
思います!!

読書会の流れ

(2) ２回目の方: 前回参加後、実行した事
