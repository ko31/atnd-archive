description

いま大流行の人狼ゲームを、水素水を使った、体によく、しかもおいしいコーヒーとスイーツを召し上がりながら楽しんでいただきます。

今回の会場となるカフェは、日本でここだけと言われる、水素水にこだわったカフェ。
お冷はもちろん、コーヒーや紅茶、そしてお料理に至るまで、あらゆるところに水素が含まれています。
また、スイーツもオーガニックなこだわりの食材を使用しており、おいしいだけでなく、体にも良いと評判です。

ゲームの方は、基本的な役職（人狼、裏切り者、騎士、占い師）をベースに、各ゲームごとに様々な役職を、１～２程度入れていきますので、いろんなタイプのゲームが楽しめます（※参加人数にもよります）

また、ゲームは、犠牲者が出た時、その場で役職を公開する、「オープンルール」で行いますので、初心者にも優しいゲーム会となっています。

楽しみながら健康にもなり、そしてもちろん友達までできてしまう。こんな素敵な人狼会は他にはないはず！
皆様奮ってご参加下さい。

日時：2015年11月28日　14時～17時
場所：健康カフェ　i-cafe
定員：15名（8名以上参加で開催）
会費：3,000円（飲み物・スイーツ付き）
※飲み物は飲み放題です！！

なお、ゲーム終了後、会場を移して希望者のみ懇親会を行う予定です（近くの居酒屋を予定。会費別）。この時、参加者様のご希望があれば、懇親会場で、ゲームの「延長戦」を行う事も検討しています（ワンターンで決着が付く「ワンナイト人狼」をやる可能性もあります）
