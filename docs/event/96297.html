description

手に持って使えるサイズと発熱量、バッテリーによる長時間駆動など Android スマートフォンには厳しい要求が課せられています。
昨今では高性能なフラッグシップモデルだけでなく低価格なエントリーモデルを手にするユーザも増えており Android スマートフォン向けアプリケーションの実行環境はより複雑になったと言えるでしょう。
このセミナーでは、Arm ベース SoC を搭載した Android スマートフォン向けにアプリケーションを最適化するツールをご紹介します。
