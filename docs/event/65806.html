未経験からスマートフォンゲーム開発エンジニア（正社員）を目指したい方を募集中！

ゲーム業界未経験のプログラマ・エンジニア志望の皆さんが抱く疑問を、
実際に働くメンバーと語らってもらう中で、解決していきたいという思いを込めて、
実際にマイネットで働くエンジニアによるLTやフリートーク、
自由交流会でお答えしていく、会社説明会を2015年5月15日（金）に開催いたします。

  • 下記に一つでも当てはまるエンジニア志望の方はぜひご参加ください。
    • 社内勉強会などでエンジニア同士で議論したい方
    • ネトゲ廃人をやっていた方
    • 新しいゲームを始めたらとりあえず課金してしまうような方
    • SIerでエクセル仕様書を作るのに疲れてしまった方
    • 新しいことをやりたいけど社内政治で頓挫してしまった方
    • スーツを着て働きたくない方
    • プログラミングしまくって人生ダメになった方
    • ユーザー向けサービスの開発をやっていきたいと考えている方

当日、選考はございませんのでお気軽にご参加ください。
エンジニアをはじめとして、マイネットで働く
プランナー、デザイナーとわいわいお話をしましょう！
