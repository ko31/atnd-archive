勉強内容

フレームワークの使い方を基礎から勉強します。

フレームワークを使うとWebアプリケーションの作成に必要な作業を簡単にすることができます。

主に３つの機能がありますが、（フォーム作成、データベース接続、テンプレート）今回はフォームの作成について勉強したいと思います。

  1. 勉強会はプロジェクターに映した講師の画面を見ながら、自分のノートパソコンにプログラムをします。
  2. ノートパソコンをお持ち下さい。
  3. 飲食物の持ち込みはできません。
  4. 費用が3,000円かかります。
  5. 高校生・大学生を対象とした無料参加枠を1席用意しました。
    参加希望の方はtwitterにてお問い合わせ下さい。

XAMPPをインストールしてください

講義にはXAMPPというソフトウェアを使います。
これはパソコン上（Windows,Mac）で、Webサーバーを実行するプログラムです。

Windows版
http://www.apachefriends.org/jp/xampp-windows.html

Mac OS X版
http://www.apachefriends.org/jp/xampp-macosx.html

XAMPPインストール手順

インストール後、http://localhost/にアクセスして、インストールが成功したか確認して下さい。

インストールに失敗した方は、http://twitter.com/phpbeginnersまで、ご相談ください。

エディターについて

PHPはHTMLなどと同様に、単なるテキストファイルなので、メモ帳でも編集することができます。
しかし、プログラム専用のエディターを使った方が効率的です。
PHP初心者おすすめのエディター集をお読み下さい。
