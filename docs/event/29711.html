お待たせしました。好評(?)のうちに終了しました第1回に続きまして第2回ファイターズ観戦オフの開催が決定しました!

今回は二軍戦です。

「え? 二軍? 稲葉とか出てこないからつまんないんじゃないの?」

と思ったそこのあなた!
ファイターズの二軍戦はちょっと他と違うのです。

1.カビーがいる

ファイターズの二軍にはマスコットがいます。
通称「鎌熊」ことカビーです。
横浜にいたレックの引退で二軍に専属マスコットがいるチームはファイターズだけになってしまいました…

カビーは（いちおう）4歳ですが…
・食いしん坊です。
・おなか出てます。
・落ち着きがありません。
・全員集合、水戸黄門、七人の侍、クレイジーキャッツなどなぜか昭和のものが好きですｗ

でもかわいらしさは数あるプロ野球界のマスコットでもトップクラスです。
そんなカビーと遊べるめったにないチャンス!

ちなみに当日私はカビー中心に動きますのでご注意ください（←

2.選手が近い

地方球場ならではの魅力ですが、とにかく目の前で選手が見られます。
未来のファイターズをしょって立つ若きファイターズ戦士にエールを送りましょう!

3.屋外でのんびり観戦

雨の心配のないドーム観戦もいいんですが、屋外でさわやかな風に吹かれながらの野球観戦もいいものです。
ビール片手に観戦というものこれまた乙かもｗ
（車を運転される方の飲酒は厳禁です!）

＊開催要項＊

試合の開催要項に関してはファイターズ公式サイトをご覧ください。

・集合
基本的に現地集合、現地解散です。
車に乗り合わせてくる方はあらかじめ当事者で打ち合わせをお願いします。

・車で来られる方
駐車場完備です。
ただし混雑が予想されるため早めの来場をお勧めします。

・公共交通機関で来られる方
JR新十津川駅は便数が少ないのでJR滝川駅周辺からバスやタクシーの利用をお勧めします。
なお、新十津川町役場からピンネスタジアムまでシャトルバスが運行されます。

・飲食
当日は売店も出ますが、混雑が予想されるのである程度は持参しておいた方がいいかもしれません。

・チケット
「外野芝生席」を購入願います（前売大人1,000円、当日大人1,500円）。
なお、オフィシャルショップや札幌ドームまで買いに行けないという方は代理購入を承ります。
（一次締切6月2日・二次締切6月6日）
代理購入された方のチケットは当日引き換えとなります。

・その他
熱中症、日射病、日焼け、虫よけは各自で対策願います。

＊中止の場合＊

雨天、荒天、その他の理由などで試合が中止になった場合、この観戦オフも中止とします。
チケットの払い戻しは各自でお願いします。
ただし、代理購入された分は当方で払い戻しますので特に手続きは必要ありません。
