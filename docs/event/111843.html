第一回、転職合宿の説明会開催を決定いたしました！

２人に１人が転職をする現代、転職活動も新しいスタイルへ突入しています。
「収入」を追い求める転職から、自分の「ライフスタイル」にあわせた転職へ。
この転職合宿で、自分流キャリアを見つけるための5日間を体験してみませんか？
わたしたちは最新のキャリア理論をもとに、超短期集中型の転職合宿を提供します。

働きたい場所・仕事が見つかる
転職合宿の費用は当社が負担
強いキャリアをピボットし続ける

こんなお悩みをお持ちではないですか？ 
東京で働いてみたいけど転職活動の方法がわからない
東京で転職活動する費用が無い
求人情報が足りず動き出せない
正社員として働きたいけど、これまでやってきたことに自信が持てない
応募書類の作り方がわからない、面接対策がわからない。

人材のプロが主催する
短期集中型の合宿が
あなたのキャリアライフを変えます

キャリアチェンジ・キャリアアップを成功させるために必要なことは「準備」です。
ゴールを設定し、ロードマップを決めて歩み出す。そのための情報を充分に集めて進めていくのが正しい転職活動です 。

Jokio！なら・・・

人材のプロが持つ独自情報 情報の整理も徹底サポート
予想外のキャリアもご提案
プロだからこそのキャリアをプランニング

キャリア・ライフの心理学
メンタルコントロール法を完全マスター
職場バレのリスクヘッジ
