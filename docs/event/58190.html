ヒカ☆ラボ一覧は こちら！！

★講演内容

「もっといろんな仕事をやってみたい！」
「もっとお仕事が欲しい！」

皆さん、いつもこんなことを考えてはいませんか？
実は、自分の好きなことを仕事として継続するには
ちょっとした考え方のコツがあるんです。

クラウドゲート×ヒカラボ　セミナー第二弾として、
今回は評価されるイラストレーターになるための
指摘されないと気づかない「仕事が取れるデザイナーの考え方」を
余すところなくお伝えします！

＜講演メニュー＞
「キャラクターをデザインすること」
「コンテンツを構成する要素」
「コンテンツをデザインする」
「デザイナーに求める考え方とは」
　　　　　　　　　　　　　　　etc…

また、第一弾で募集したオンラインワークショップに
参加していただいた方の作品を使用したフィードバックも
イラスト学校講師とプロのキャラデザイナーからアドバイスさせていただきます。

※もちろん事前課題を提出頂かなくても、参加頂くことは可能です。

今回ご用意したコースは2つ！！

①クラウドゲートが運営するイラスト学校の講師になれる!?「赤入れ」コース
　課題作品の赤入れをして、
　人物デッサンの勉強や魅力的な構図の作りかたなどを
　“教える”立場から勉強してみましょう！

②需要が高いデフォルメイラストを勉強する「デフォルメ」コース
　現在、キャラのデフォルメ化需要が高まってきているのはご存じですか？
　課題キャラのデフォルメを制作して、デフォ技術をアップしていきましょう！

素敵な作品は、3DCGになって12月のセミナーに登場するかも！？

＜ワークショップは既に受付を終了しました＞

※内容は変更になる可能性がございます

★講演者プロフィール

■クラウドゲート株式会社
　企画・イラスト学校講師・授業企画担当
　蛯名結花（えびなゆか）氏

とにかく文章を書く仕事がしたいとゲームライターになったのをきっかけに、
ゲームシナリオ、イベント企画、システム開発、プロモーション活動などに携わり、
現在は企画提案～デザイン制作まで、幅広く担当するプランナー。
クラウドゲート入社後は、クリエイター支援を中心に、
専門学校と連携し産学協同の支援として、
講演会や進路アドバイス、ポートフォリオ品評などを行っている。
