catch 神奈川県民が国内外で撮影した世界遺産の写真を約200点展示しています。【月曜休館】
description

本展では、神奈川県民が国内外で撮影した世界遺産の写真を約200点展示しています。世界遺産の輝きを写し出した見応えある写真が揃いました。世界遺産登録を目指す「武家の古都・鎌倉」の特別展示もお楽しみいただけます。
※「県民が見た世界遺産写真コンテスト」（2011年10月11日～12月6日実施）にご応募いただいた作品を展示いたします。

会期：2012年1月20日（金）～3月18日（日）１０：００～１７：００(入場16:30まで)
