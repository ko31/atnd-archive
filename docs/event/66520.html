不動産投資の鉄則とは「良い物件を適正な価格で入手すること」に尽きます。
おそらく、不動産投資をしたことが無い人でもわかる理屈でしょう。

しかし、不動産投資で失敗する人は後を絶ちません。何故なのか？　
最大の理由は、どんな物件が本当に良いと言えるかを理解していないからです。

また、購入する人の不動産投資の目的や資産状況などによって、良い物件は変
わってきます。相続税対策を考えているシニア層と、年金に不安があり将来の
生活を安定させる目的で不動産投資を考える30代、40代とでは、投資対象とし
てふさわしい物件は異なるのです。

私たちは、常日頃、相談者のさまざまなニーズに向けたコンサルティングを行
なっています。定期的に開催しているセミナーでは、特に、不動産投資である
程度の収入を得て、リタイヤ後の生活を安定させたいと考えている30代、40代
の方々に、本当に良い物件とは何かについてアドバイスをしています。

その答えは明快です。東京都心部の単身者向けのワンルームマンションが最適
なのです。

セミナーでは、なぜその条件の物件が最適なのかについて、不動産投資のリス
クを含めて、じっくりと解説しています。投資に使える資金があまりない、と
いう人も参加して頂きたいと思います。

単身者向けのワンルームマンション投資は、大きな元手は必要ありません。
また、良い物件の条件を知ることで、不動産投資に対する意識も高まります。
真剣に不動産投資を、リタイヤ後の生活を考えるきっかけとなれば幸いです。
