JAWS-UG Yamaguchi 【第２回勉強会】

会場では無線・有線LAN環境完備です
　※有線用のLANケーブルは各自でご準備願います。

第二回目となります、JAWS-UG山口勉強会を開催します。
「何もしらないけど興味がある」「使ってみたいけどよくわからない」
「使っているけどもっといろいろ知りたい」そんな皆様を大募集です。
今回は、ゲストスピーカーとして株式会社ヌーラボでご活躍の山本竜三氏を招き、AWSの利用・開発に纏わるでぃーぷな部分？も分かりやすく講演いただく予定です。
またAmazon玉川氏とは、Webinar×Ustream×Skypeコラボの新形式での講演です。
皆様のご参加をお待ちしております！
