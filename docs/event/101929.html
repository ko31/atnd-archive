「クリエイター（デザイナー、エンジニア）のキャリアアップに大切な３つのこと【転職、独立、会社設立編】」

デザイナー、エンジニア、フォトグラファー、ライター、イラストレーター、プランナー、制作などものづくりをするクリエイターの方に特化した勉強会です。

　

◆目的

クリエイターが将来キャリアアップしていく時に、無駄な失敗をしない、損をしないために何が大切なのかを学ぶ勉強会です。

　現職 → キャリアアップ、収入アップ 　現職 → 転職 　社員 → 独立、フリーランス 　フリーランス → 法人化 などで大切なことを学びます。

◆セミナー内容

この先、訪れる可能性の高い人生の転機、 代表的なのは「転職」や「独立」そして「会社設立（法人化）」です。 特にクリエイターは終身雇用が少ない職種なので、なおさらです。

とはいえ、よくわからない初めてのことであり、どんなことをすればよいのか、どんな落とし穴があるのか不安だと思います。 講師の白井は、クリエイターであり実際に　転職→部下マネージメント→転部→独立・フリーランス→ジョブチェンジ→法人化（経営）の経験から、また、現在はクリエイターの育成、経営者スクールの講師をしており、数多くの受講生の失敗と成功を見てきた経験から、クリエイターのキャリアアップに大切なことをお伝えします。

◆こんな人にオススメ

・今よりもっとスキルアップがしたいと思っている人

・今の職種からジョブチェンジしたいと思っている人

・今の職場、人間関係がちょっと嫌だなあと感じている人

・今、漠然とした将来への不安があって寝付きが悪い人

・将来、転職するつもりだけど、今からどんなことをすれば良いか知りたい人

・将来、絶対独立する！フリーランスで活躍する予定だけど、極力失敗したくない人

・将来、会社設立（法人化）して、会社として大きくor しっかりしていきたい人

◆期待できること

漠然とした不安が減って、今やるべきことが明確になりやすくなる。 転ばぬ先の杖、失敗の法則を知ってしなくても良い失敗を避けることができる。

転職、独立、会社設立のしかた（手続き）は本を読めばわかるが、それ以前にしておくこと（特にクリエイターとして）がわかり、しっかり準備ができるようになる。

会話が得意な講師なので、同僚や上司に聞けない、転職や独立の不安や、疑問を気軽に質問できる。
