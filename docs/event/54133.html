description

社会人基礎力やビジネス基礎力など、
キャリア教育でよく言われる主体的に考え、行動できる力が、
必要なのは充分理解できるけど…、
一体どうすれば、教えることができるの？

セミナーに参加する学習者のほとんどが、
すでに自分で問題意識を持っていて、
比較的優秀な学習者ばかり参加するけど、
本来は、それができない人にこそ受けてほしい…。

ゼミ以外にも、アルバイトやサークル、ボランティア、インターンなど、
いろんな経験をすることを勧めるけど、学生自身に委ねるのではなく、
きちんと動機付けを与えることはできないものか…。

そんな悩みを持っていませんか？

このセミナーでは、
学習者が、以下のようなことができるようになるための方法を紹介いたします。

◆自分なりに考えるということを、積極的に取り組む
◆正解のないことに対して、自分なりの最適解を考える
◆自分なりに考えたことを、他人に伝える
