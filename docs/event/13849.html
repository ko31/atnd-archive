Appleアカデミックユーザ会では下記の通り第1回ミーティングを開催します．
Apple製品を活用した教育・研究にご興味をお持ちの教育機関関係者のご参加をお待ちしています．

プログラム

時間 タイトル 登壇者

16:20 – 17:00 教育用計算機システムとネットワーク環境について（仮） 伴 好弘（神戸大学 情報基盤センター 准教授）
18:00 – 場所を変えて懇親会（実費5000円程度）

講演概要

1. iPad の使い方講習会

登壇者

清水康行

概要

アップルジャパンの専門講師による，iPadの授業や研究に活用できる機能についての講習会を開催いたします．実際にiPadに触れて頂きながら，教員の方向けに教育用アプリケーションや日常業務活用のノウハウをご紹介いたします．その他，実際のデモ授業も織り交ぜながら授業への活用の可能性をご説明いたします．

2. Mac OS X Serverによる学内ネットワーク運用

登壇者

京都女子大学 現代社会学部 准教授 宮下 健輔

概要

京都女子大学では2007年にそれまでのSolarisを中心としたサーバ群をMac OS X Serverに転換しました．その導入から構築までとそれから4年間の運用で得られた知見についてご紹介します．またご希望があればサーバ室を見学していただけます．

3. iPhone/iPad向けコンテンツ作成・配信・閲覧サービスのご紹介

登壇者

パナソニック電工インフォメーションシステムズ株式会社

概要

今や教育機関に広がりつつあるiPhone/iPadなどのデバイス．今回はその中でも，講義資料の配信・共有，学習教材の作成，e-learning等のコンテンツを手軽に作成・配信・閲覧できる統合サービスをご紹介．実際の教育現場での活用方法など事例と合わせてご案内いたします．

4. 教育用計算機システムとネットワーク環境について（仮）

登壇者

神戸大学 情報基盤センター 准教授 伴 好弘

概要

神戸大学情報基盤センターで今回導入した教育用計算機システムに関する概要と，前年度更新した学内ネットワーク環境の中で，特に無線LAN環境の集中管理化について紹介します．
