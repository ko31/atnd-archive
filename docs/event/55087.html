皆さんは、どんな部分に拘って同人誌を作っていますか？

表紙？　タイトル？　ページ数？　出だし？　絵？　写真？　紹介文？　

前書き？　後書き？　サークルスペースの飾り付け？

上記はかなり要約しておりますが、並ならぬ拘りが皆様にはあると思います。

普段中々語れない『作品作りの拘り』をここぞとばかりに語り合いませんか？

そして、普段は聞けない事を他の同人作家さんに聞いてみませんか？

==主催プロフィール==
謎探りは紅茶代表：闇野夜美

2006年に文芸サークル ツキヨトイカイを立ち上げる。
ヤンデレ専門サークルとして小説、四コマ漫画、童話などの作品を作成。
普通とは少し違うヤンデレ作品を作ることを拘りにしており、かつイベントスペースで
一般参加者が本を手に取った時にどのようなインパクトを与えるかを常に考え続けてた。
2014年に評論情報サークル　謎探りは紅茶を立ち上げ、同人誌の研究並びに
同人イベントでのサークルスペースの研究を行いつつ、秋葉原にあるお店の紹介や
秋葉原活性化に向けた活動を行っている。
