description

近年拡大しつつあるサブスクリプションサービス。通常の物販とは概念の異なるサービス提供にも対応した、クラウド型ECパッケージの「ecOrigins」とクラウド型販売課金プラットフォーム「KaKing」をご紹介します。当日はこれからのサブスクリプションビジネスを低コストで実現するクラウドシステムの活用についてご説明いたします。

参加費は無料ですのでお気軽にご参加ください。

ご希望の方はセミナー終了後に個別相談会を実施いたします。
