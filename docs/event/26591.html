～Unity + PHP + MySQL で作るスマートフォンゲーム開発～
　
h3. 今回は、Aiming からリリースされたばかりのスマートフォン向け本格シミュレーションRPG、「Lord of Knights」の技術的な裏側を公開します。

==============

→広めの会場で開催できることになった為限定枠を50席→80席に変更しました。(2012/3/27)

==============

Lord of Knights は、クライアントサイドが Unity + C# 、サーバサイドが PHP + MySQL によって構成されています。今回のセミナーでは、サーバ/クライアントの開発現場のリーダーが、設計・ハマりポイント・tips 等について、惜しみなくお伝えします！

ゲームを開発に携わっているエンジニアのみならず、 iOS / Android のマルチプラットフォームアプリケーション開発、PHP + MySQL によるサーバサイドプログラミング、Unity を使った開発事例等に興味がある方は是非ご参加ください！

■細田 幸治 氏　＜ 株式会社Aiming リードソフトウェアエンジニア＞

筑波大学大学院 生命環境科学研究科 修士課程修了
山梨県韮崎市出身

1979年生まれ、東京都在住のプログラマ。
2005年2月にコミュニティーエンジン株式会社に入社。 gumonji の開発を通じてエンジニアの卵となり、その後10本程度のオンラインゲーム/ツールの開発に携わる。仕事をする傍ら、ブラウザ三国志にハマり、廃人プレイヤーになる。
2010年5月にブラウザ三国志の開発会社である ONE-UP 株式会社に転職、マルチプラットフォーム通信ライブラリの開発や Yahoo! モバゲーでサービスした英雄クエストのクライアント開発などを行う。 廃人プレイの成果を大きく仕事に生かした。
2011年6月に株式会社 Aiming に転職、Lord of Knights のクライアント開発チームのリーダーを務める。
「楽しく」働くために日々考えながら仕事しているので、基本的に楽しそう。
