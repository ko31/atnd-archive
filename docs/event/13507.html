『日曜日のお昼に 今聞きたい気分の音楽 と 素敵な仲間との心地良い時間』

をテーマに、スタートしたイベントです。

まだまだ若い試み故、皆さんの意見を取り入れ成長し、より心地良い時間になっていければと考えています。
「ただ来て見る」より「参加してもらったり、体感してもらえる、そこに【自分】の居る。
新しい可能性を見いだしてもらえる」様な空間として皆でこの空間を使っていけたらと考えます。
少しの想いと少しの勇気があれば、僕たちも出来る限りのバックアップはさせてもらいます。
様々なワークショップやライブ、フードの出店や自己表現の場として皆で楽しみましょう！
