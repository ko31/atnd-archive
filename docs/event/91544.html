description

世界一の都市として十分な国際競争力と高い磁力を維持していくために必要不可欠な要件や課題について、世界の都市との比較から指し示し、持続可能な開発目標（SDGs）に則り、多様性と調和の重要性を認識し、新たな価値を生み出し成長し続ける国際都市東京を目指す必要があります。
我々、青年経済人が「日本人としてのアイデンティティ」を自ら育むとともに、「世界と日本を繋げる磁力ある国際人」のロールモデルを示し、参加者の意識改革を図ることが求められていると考えます。
本事業をきっかけに、東京を世界に誇れる美徳溢れる国際都市に押し上げる基盤を作って参りたいと考えております。
　概要は以下の通りとなりますので、是非、一人でも多くの方のご参加をお待ちしております。
