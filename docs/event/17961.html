DBFluteユーザの集い2011

■ 9月25日（日）に「DBFluteユーザの集い2011」を開催します

今年は、特に(恐らく)Seasarカンファレンスもなく、 特に何もしなければ、DBFluteを公にしゃべる機会もなく、 DBFluteユーザ同士が顔を合わせる機会もなさそうです。

ということで、９月にDBFluteは五周年を迎えるにあたって、 DBFluteの勉強会(jfluteのセッション、持ち寄りのセッション) を兼ねたDBFluteユーザの集いをやります。

■参加費

  • 参加費：1500円（お昼代　ケータリングを考えています）
    ※会場でそのままみなさんで懇談会としましょう。

■持参していただくもの

  • 名刺2枚（ネームプレートと出席確認用です）
  • Wifiルータ（ネット環境を利用したい方）
  • 飲み物（お茶は用意いたしますが、建物内に自販機はありません。飲みたいものがある人は持参してください）

■開催日時

2011年 9月 25日（日） 11:00-16:00
