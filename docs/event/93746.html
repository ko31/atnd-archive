15:00〜　ハンズオン　AVTOKYO配布バッジを使って学習リモコンを作ります！

去る11月に行われたセキュリティイベントでの配布バッジ。 もらったバッジをHACKしたい人、Welcome! また、バッジやパーツも追加販売します。

AVTOKYO 2017 バッジについて

STM32F103C8というARMチップを使ってミニゲームなどを走らせました。 STM32F103CはSTMicroの32bitマイコン。ARMコアで Cortex-M3 USBやCANが使える。

マイコンチップ単体だと100円台、 電池駆動したうえでちょっと複雑なプログラム走らせるには最適。

今回のHackはバッジボード上の赤外線通信機能を使ってArduino開発環境でリモコンにチャレンジ！
