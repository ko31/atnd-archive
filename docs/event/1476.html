description

9月のIT勉強会は、9月11日（金）19時～21時30分、ネスト赤羽２階会議室にて、行います。（会場には、18:30～入れます。早めにお越しの方は、名刺交換など交流を楽しんでください。）
勉強会の後、お時間のある方は、そのまま同じ場所（ネスト赤羽２階会議室）にて、１時間ほど交流会を行います。（21時30分～22時30分を予定）

皆さんのお仕事が優先ですので、当日の突然の出席、当日の突然の欠席も、OKですが、参加希望の方は、事前に参加申し込みいただけますと、嬉しいです。
だいたいの人数把握と、勉強会に参加される人が何を求めているかをプレゼンターが事前把握することが、主な目的です。


勉強会は、以下の２構成です。

（１）質疑応答も含めて1時間30分前後を予定
＜「動画共有サイト（phpmotion）」について＞
インターネットもブロードバンドでの生活が当たり前になってきています。
ニコニコ動画やYoutubeも皆さんご存知かと思います。こんなサービスを自分たちでももてたらいいな。既存のホームページと連携できたらいいなという方のために動画初心者セミナーをさせていただきます。
動画っていろいろ編集やファイルの変換が難しいと思っている方、そんな方は是非、試していただければと思います。

（２）1時間前後を予定
＜オープンソースを使ったWEB制作方法や、メールの処理方法についてのディスカッション＞
オープンソースを使ったWEB制作方法や、メールの処理方法についてディスカッション形式で話し合う予定です。
他の人の使用しているツールや方法を参考にして、WEB制作やメール処理の効率化を図り、有効な方法を考えましょう。
もし現在お困りのことがありましたら、そのことについて、皆さんでお話する中で、情報共有ができて、解決方法が見つかるかもしれません。


19時～21時30分の勉強会は無料ですが、21時30分～22時30分の交流会は実費で500円～1,000円ほど掛かる予定です。勉強会だけでなく、交流会にも参加される方は、当日に実費部分をお支払いください。

http://www.nest-akabane.net/study/695.html
をご覧ください。
