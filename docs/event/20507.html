description

北陸でWindows Phoneの勉強会を開催することになりました。

App Hubのメンバーシップ登録料9800円を支払っていなくても参加できます！

アプリケーションを作成することがメインの勉強会ですので、Windows Phoneのアプリを作ってみたい、SDKをインストールしたのはいいけどなにからやってよいのか分からない、Windows Phone楽しそうー、という方には最適だと思います。

そして今回はマイクロソフトの大西 彰さんに講師をしていただき、半日でアプリケーションを作成していきます。

ただし、今回の勉強会への参加には条件を付けさせていただきます。

条件

・当日はアプリを２つ作りマーケットに登録すること（テンプレートあり）

・後日、各自最低１つはアプリを作ってマーケットに登録すること

・出来る限り実機を持っている方（エミュレータでも開発はできますが、実機とは異なりますので）


※当日作成するアプリケーションはあらかじめこちらからテンプレートを用意させていただきます。
※また、マーケットに登録していない方でも参加できますのでご安心を。
※また後日マーケットに登録してもらえる方だけに限らせていただきますのでご了承ください。

タイムテーブル

時間
13:00 – 18:00 ひたすらアプリを作ります

準備

-Windows Phone SDK 7.1 をインストールしたパソコンを持参してください。実機は必須ではありません。
-App Hubのアカウントを作成しておいてください。 APP HUBはこちら
