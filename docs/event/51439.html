セッションの目的

「自分らしい働き方」「あたらしい働き方」「ワークシフト」
それは会社を辞めて起業する、フリーランスになることと捉えられがちですが、
組織に属しながら実現するには、どうすれば良いでしょうか。

個人の働き方や組織の在り方は、世界規模で大きく変わっています。
しかし、多くの日本人が潜在的にそう感じながらも、
組織に属することへの息苦しさを受け入れているようにも思います。

一人一人が「ジブンゴト」で働き方の未来を考え、
組織との関わり方を見つめ直し、主体的にワークスタイルを選択する。
そんな働き方の未来を創りたいと思っている方は多くいらっしゃると思います。
あなたが考える「働き方の未来」が誰かを勇気づけたり、
丁寧に想いを繋ぐことで後押しされるかもしれません。

このセッションでは、働き方の未来を考えるだけではなく、共に創ることがテーマです。
一緒に未来を創りたい仲間もお誘いの上、是非ご参加ください。

対象者

- 組織に属している方、今後予定のある方、企業のパートナーとして仕事をされている方
- 「働き方の未来」を主体的に創りたいという想いのある方
- 働くということを真剣に考えてみたい方（学生、休暇中の方、転職活動中の方、どなたでも！）
- フューチャーセッション、対話の場を体験してみたい方
