【プログラム】
◆講師の言葉
　本講義は、GPS/GNSSの衛星測位を利用するユーザが、都市部で抱えているマルチパス誤差の問題について言及している。ここ5-10年での可視衛星数の増加は期待できる状況にある中、残存するマルチパス誤差を低減することは非常に重要なテーマである。
　本講座では、代表的なGPSを取り上げ、GPS測位の概観と測位誤差要因についてまず述べる。次に、衛星測位におけるマルチパス誤差の発生メカニズムを、受信機内部の信号処理段階から実際の観測データへの影響を通して詳しく説明する。
　さらに、周囲の環境に応じたマルチパス誤差の実例を挙げる。最後に、最近のマルチパス誤差低減技術について述べる。また、近未来期待できるであろう、マルチパス誤差を低減する技術や衛星側の改善についても言及する。

◆修得知識
・衛星測位におけるマルチパス誤差の発生メカニズムをイメージすることができるようになる。新たにマルチパス誤差低減技術の開発を試みる方の基礎を習得できる。

◆プログラム
Ⅰ. GPS/GNSSの基礎と取りまく環境

Ⅱ. GPSの測位
　　1. 信号処理部
　　2. 観測データ算出部
　　3. 測位計算部

Ⅲ. 測位誤差要因
　　1. 衛星側の誤差要因
　　2. 衛星配置による誤差要因
　　3. 電波伝搬経路における誤差(マルチパス)
　　4. 受信機側の誤差要因

Ⅳ. マルチパス誤差
　　1. マルチパス誤差の発生要因
　　2. 各種観測データにおけるマルチパス誤差の発生メカニズム
　　　　a.擬似距離
　　　　b.搬送波位相
　　　　c.ドップラー周波数
　　3. マルチパス誤差の実例
　　　　a.周囲の開けた場所の例
　　　　b.中層ビル街の例
　　　　c.高層ビル街の例
　　4. マルチパス誤差低減技術
　　　　a.アンテナによる低減技術
　　　　b.受信機による低減技術

Ⅴ. まとめ
