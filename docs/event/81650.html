・今の仕事に不満や不安がある

・なんとなく現状に満足できずモヤモヤしている

・起業に興味はあるけど、何から始めたらいいのかわからない

・起業準備中のまま、なかなか先に進めていない

・自分なりのやり方に行き詰まりを感じている

・何か新しいことにチャレンジしたい

・仕事とプライベートを両方充実させたい

このように、現在の働き方や生き方を変えたいと望んでいるあなたへ・・・。

【仕事もプライベートもバランスよく両立したい女性の幸せ起業術】

「最近仕事ばかりで、プライベートが後回し・・・。このままで、本当に幸せ？」

「会社に雇われない自由な働き方に興味があるけど、不安定な生活が心配で一歩が踏み出せない。」

「もっと自分にしかできないやりがいのある仕事がしたい。」

こんな葛藤を抱えながら、日々を過ごしてしている女性は多いのではないでしょうか？

女性はライフスタイルの変化も多く、仕事とプライベートの両立が難しくなりがち。

ですが、女性として「仕事もプライベートもバランスよく満たされた生活を送りたい！」という望みは、あなたの選択次第で叶えることができます。

そして、そういったあなたの幸せな未来を叶える手段として、私は“起業”という選択肢をご提案しています。

起業すれば、全ては自分で決めることができます。

•どれくらい収入を得て、どんなことに大切な時間を使いたいのか。
•誰と一緒に働くか？
•どんな人に囲まれて生きていきたいか？

そういったことはすべてあなた次第です。

とはいえ、「そんな簡単にうまくいくわけない。」という気持ちもありますよね。

そう。もちろん、誰もが簡単にできることではありません。

ですが、あなたが『本気で生き方を変える』という決断をすれば、
願った通りの生活は必ず手に入れることができます。

あなたが今すべきことは、ただ決断するだけ。

決断し、正しい方法で実践を続ければ、必ず結果はついてきます。

できないのはやり方を知らないから。
不安なのは、まだ経験したことがないから。

やったことのないことは想像できないし、怖いのも当たり前。
だからこそ、“知ること”・“想像すること”これが非常に重要なのです。

まずはこのセミナーで、あなたの理想の生き方を想像してみましょう。
そして、その理想に向かう道筋を明確にしましょう。

それができれば、あとはその道を一歩一歩進むだけです。

最初の一歩を踏み出すときが一番エネルギーを必要とします。
だからこそ、その道筋を示し、「迷っているあなたの背中をそっと押すことができる」
そんな勇気とたくさんの気づきに出会えるセミナーをお届けします。

【仕事もプライベートもバランスよく両立したい女性の幸せ起業術】

＜セミナーでお話する内容の一部＞

・どうやって自分の知識や経験をビジネス化するの？

～売れるビジネスを作るための４つの要素～

・起業って具体的にどうやるの？

～自分らしさ起業を叶えるための具体的な５ステップ～

・ビジネスモデルってどうやって作るの？

～ビジネスの本質を知れば起業はとてもシンプル！ビジネスを作る３つの要素～

・起業準備には何が必要？

～起業するために必要なたったひとつのこと～

・起業して失敗しないためにはどうすればいい？

～起業が失敗する３つの理由。ノーリスクで起業するための商品の作り方～

こんな方にオススメです！

✔　１年以内に起業を考えていて、現在情報収集中。

✔　まずは副業から、小さく何か始めてみたい。

✔　現在の働き方から脱却し、新しい可能性をさぐってみたい

✔　やりたいこと、学んできたことはたくさんあるけれど、自分の商品・サービスつくりに活かしきれていない。

✔　今までの知識や経験がバラバラで整理できておらず、何をすればいいのかわからなくなっている。

✔　起業してはみたものの、ビジネスモデルがうまく作れておらず、売り上げが伸び悩んでいる。

こんなものが手に入ります！

✔　自分の強みを生かしてノーリスクで起業する方法がわかります。

✔　散らばっていた頭の中がスッキリと整理され、次に何をすればいいかが明確になります。

✔　売れる商品・サービスを作り上げるまでの具体的な道筋がわかります。

✔　起業するために必要なことがわかり、起業に対する不安が解消されます。

✔　うまくいっていない原因、見直すべきポイントが明らかになります。
　

【たった３時間で、やるべきこと・進むべき道筋が明確になり、一歩を踏み出す勇気が手に入ります！】

ほんの少しの勇気から、未来を変えるきっかけが生まれるかもしれません。

動き出さなければ、未来を変えることはできません。

勇気をだして、まずは一歩踏み出してみてください。
