title くにたちコミュニティビジネスシンポジウム
catch くにたちではじめよう！　あなたにもできるコミュニティビジネス
description

ちょっとの勇気で、まちは元気になるんです！

北海道帯広で全国的に有名なまちおこし事例である「北の屋台」をプロデュースした後藤健市さんをお招きして、「まちの元気のつくり方」を熱く！力強く！語っていただきます。

さらに、大学生の空き店舗活用プロジェクトから、ハチミツづくり、アートを活かしたまちづくりまで、国立市内のコミュニティビジネスが大集結！！

こんなにある、こんなのもある。あなたにもきっとできます。

お気軽にご参加ください。
