出版記念につき無料開催！
「見える化経営で何が変わるのか？」

年間１０万人が来店しているお店の“見える化”経営セミナー
〜２０年間、増収増益しているお店の舞台裏大公開！〜

 

◆たった数年でお店が消える・・・！？
━━━━━━━━━━━━━━━━━━

飲食店を筆頭に多くのサービス業が、かなりの金額を投資して新規出店をしては閉店・廃業を繰り返しています。20年経っても既存店売り上げが伸びている店舗と、たった数年で消えてしまう店舗との違いは何でしょう。

当日は書籍に書かれていることはもちろんのこと、書ききれなかったお話もしていただきます。

どの内容もご自身の店舗の「見える化」とサポートしたお店で成果を出した結果に基づいたリアルな内容となっております。。

 

◆リピーター獲得に繋がる実践的方法 “おもてなし電話”
━━━━━━━━━━━━━━━━━━

想像してみてください。

あなたは以前利用したレストランを予約しようと電話しました。
電話が繋がった瞬間「高橋様、以前のご利用ありがとうございます。お元気でいらっしゃいますか？」と言われたら、驚くと同時に嬉しいですよね？

シンカCTI「おもてなし電話」は着信と同時に顧客情報がわかるので、各スタッフの対応サービスが向上し、顧客満足度を高め、圧倒的な信頼を生むことで、リピーター増加や客単価増加に貢献しています。

現在、100の業種業態を超える導入実績がありますので、当日は具体的な活用事例を交えながらお話させて頂きます！！

※CTI（Computer Telephony Integration System:電話とコンピュータの統合システム）

講師紹介

株式会社玉海力　代表取締役　河邉幸夫
http://www.tamakairiki.co.jp/

1980年日本相撲協会片男波部屋へ入門し力士となる。1995年、大相撲引退。翌年広尾に「どすこい酒場玉海力」を開業。現在、都内4店舗、ほか上海、プノンペンにも店舗展開中。飲食店ではあまり見かけないクラウドを使った社員への情報開示や顧客管理を行う。現在では飲食業以外にも自店で培った「見える化コンサルティング」で飲食店やサービス業の開業や改善の支援も行っている。
