キュアパッション聖誕祭２０１５開催！

今年も熱い夏がやってきました



今年は６年ぶりにフレッシュプリキュア！が放映された日程と同じスケジュールでＧｏ！プリンセスプリキュアが放映され、ニチアサがやってきています

そして、６年前の２００９年７月１２日の日曜日、キュアパッションが聖誕しました

今年は『映画フレッシュプリキュア！おもちゃの国は秘密でいっぱい！？』のブルーレイも発売されます

大画面でフレッシュプリキュア！を語りつつ振り返りながら見たい…

そこで、副主催のフォロワーさん先にたまたまある１５０インチの大スクリーン、体まで響く迫力のある音を出力するスピーカー、そして映画館ばりのゆったりとしたシートを用意いたしました！

みなさんでわいわいとフレッシュプリキュア！を語りながら楽しみつつ、聖誕祭をささやかにお祝いしましょう！



【参加者で選ぶフレッシュプリキュア！ベスト３】

今回は主催の思いつきですが、皆さんにイチオシのフレッシュプリキュア！の話を３つ選んでいただくことにしました

アンケートの集計可能期間は終了しました
アンケートの結果は反映されませんが、参加申込は当日12時まで可能です


『映画フレッシュプリキュア！おもちゃの国は秘密がいっぱい！？』を超大画面で見るオフを併催します！

Blu-ray版「映画フレッシュプリキュア！おもちゃの国は秘密がいっぱい！？」の発売を記念して、みんなで超大画面で見るオフを併催します
フレッシュプリキュア！の４人の活躍をもう一度、一般のご家庭では見られない大きさのスクリーンでみましょう！
ミラクルライトをどのシリーズのものでも大丈夫ですので持参すると盛り上がれます！
エンディングパートは大人の人が立ち上がってダンスしても大丈夫です！
