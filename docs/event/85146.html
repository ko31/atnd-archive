第３回富山ブロックチェーン研究会講演会

－地域社会でのブロックチェーンの実証－

地域応用の先駆的な取り組みとして、会津地域を中心にブロックチェーン技術を活用した地域通貨やアプリケーション実験が行われています。
実験メンバーをお招きして、地域社会でのブロックチェーン利活用の意義や応用アイディア、また、最新技術の実証状況についてご紹介いただきます。

日時　　　2017年2月6日（月）
　　　　　16:00～18:00（受付15:30～）
場所　　　タワー111　21階会議室

対象　　　ブロックチェーンについて関心のある方

講師プロフィール
　　　藤井靖史氏
　　　1977年生まれ。京都府出身。国内企業、外資系企業、海外ベンチャーと渡り歩き、
　　　帰国のタイミングで現在の奥さんと結婚するために東北へ移住。
　　　2009年、仙台にて株式会社ピンポンプロダクションズを設立。代表取締役就任。
　　　2012年にKLab株式会社とのM&Aを行い、EXIT。
　　　現在は会津大学にて新しい産学連携の形を模索中。
　　　内閣官房オープンデータ伝道師、CODE for AIZUファウンダー、
　　　Code for Japan理事を務める。
　　　岡田隆氏
　　　1983年生まれ。千葉県出身。有限責任監査法人トーマツで上場支援に従事。
　　　その後、南米・アフリカなど途上国を中心に20カ国程度周遊。
　　　帰国後、コンサルティング会社を経て、数社のベンチャー立上げ。
　　　2015年ごろからブロックチェーンの魅力に取り憑かれ、
　　　2016年にソラミツ株式会社を仲間3人で創業。共同最高経営責任者。

富山ブロックチェーン研究会について
　　地域におけるブロックチェーンの応用案創出、およびこれに関する
　　人的ネットワークを形成することを目的に、産学の協力のもと、研
　　究活動を進めます。一般より参加者を募集し、今年度内に3回の意
　　見交換会の開催を予定しています。持ち寄ったアイディアをまとめ、
　　ブロックチェーン技術への関心が高い企業・団体に実証実験を提案
　　することを目指します。
