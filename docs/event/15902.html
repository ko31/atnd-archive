『ドラッカーを超える！佐川急便の「マネジメント」』

catch ～「ゆとり」「スイーツ」「ぶらさがり」の「やる気スイッチ」をONにする～
description

ドラッカーのブームによって「マネジメント」という単語の意味が
改めて見直されていますが、理論だけではなかなか現場は回りません。

また、「ゆとり社員」「スイーツ社員」「シュガー社員」「ぶらさがり
社員」などに対して「今時の若い人は…」などと愚痴っていても業績
は上がりません。

では、従業員ひとりひとりの「やる気スイッチ」を入れ、実際に現場
を機能させるためには、具体的にどうしたらよいのでしょうか？

解決の選択肢のひとつとして、他社では「問題あり」とされそうな社員
に対してもモチベーションを上げ戦力化する佐川急便のマネジメントに
ついてお伝えいたします！

＜講師＞
大重 寛 （おおしげ ひろし）
1967年大阪出身。 運送業経営支援コンサルタント。
佐川急便に20年在籍し、管理職・社内研修講師として1200名以上の
セールスドライバーを教育。
独立後も現場主義のコンサルティングを展開中。
