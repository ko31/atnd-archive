description

キム兄さん(※1)がパンチのある飲み会を！とわがままを言うので開催したいと思います。
とにかく生とハイボールが150円なので貧乏な殿方にもおすすめです。どんなに飲み食いしても一人3000円いきません。


わさび巻きは完全に罰ゲーム的な扱いですが実は相当美味しいです。馴れるまでちょっとキツイですがすぐ馴れます。


そしてどうけさん(※2)が幹事を手伝ってくれるに違いない、と。
新入社員の堤さん(※3)も連れてきてくれるに違いない、と。

※2:面白い法人に勤めてる酒好きな人
※3:面白い法人に最近入った本当に面白い人
