description

全国中小企業団体中央会より新卒者就職応援プロジェクトのコーディネート機関として業務を受託しているヒューマンリソシアではこの度、梅田のヒルトンプラザウエスト・オフィスタワーにてマッチングイベントを開催します。インターン生を募集する企業が25社以上来場します。実習の内容や職場の雰囲気など直接聞くことができるチャンス！ぜひご来場ください。
（予約不要）

新卒者就職応援プロジェクト事業とは・・・。
2007年9月以降の大学等卒業の就職未内定者の方を対象に中小企業の仕事現場に触れる機会を提供するとともに、中小企業で働く上で必要とされる技能・技術・ノウハウ等を習得してもらうための長期間の職場実習（いわゆるインターンシップ）等を実施するプロジェクトです。
インターンシップを通して中小企業の魅力を発見し、企業で必要とされるスキルやノウハウ等を習得して社会人へのスタートをきりましょう！

　　　　
　　＜イベント内容＞
　　・新卒者就職応援プロジェクト紹介
　　・企業ブースにて企業担当者よる事業紹介
　　・キャリアアドバイザーによるカウンセリング
　　・インターンシップに関する相談
　
　　＜イベント参加企業（25社以上の企業が参画予定）＞　　　
　　・教育サービス　・服飾メーカー　・雑貨卸　・情報サービス ・服飾小売
　　・人材サービス　・機械商社　など
