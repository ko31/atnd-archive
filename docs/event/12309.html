高尾山に登ってみたい

内容

独身男女で山登りをしましょう！合同ハイキングです☆
寒い！つらい！でもその中で「あいのり」的な何かが生まれるかもしれません。そこに期待！

  • 寒さ対策だけはしっかりしましょう！カイロ！首周り！手袋！厚手靴下！
  • 足元は歩きやすい靴で。裏がつるつるの靴はＮＧです。雨具の用意必須。
  • 魔法瓶に熱燗入れてくと幸せらしいです…
  • コース＆スケジュールは検討中。オススメなどあれば教えてくださいー。
