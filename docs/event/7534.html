IT系エンジニアがお金を稼ぐことについて真面目に考えるための集い、rake:moneyの第10回を開催します。
当日のスケジュールは以下の通り。
（※ボランティアで会場準備を手伝ってくださる方は、19:30にロビーでお待ちください）


  • 19:55　　　　集合＠オラクル青山センター2Fロビー

  • 20:00～21:55 勉強会＠オラクル青山センター13Fセミナールーム

  • 22:00～　　　二次会（懇親会）


さて今回のお題は「エンジニアとしての出口戦略を語ろう」です。


バイアウト、株式公開等のベンチャー企業の出口戦略が語られる昨今、個々のエンジニアが自身のキャリアパスにおける出口戦略を語ってもいいじゃんないか。
エンジニアとしての人生の展望、さらにゴールのビジョンを明確に持つことは日々の生活を充実したものにするためにも大事なこと。そして他のエンジニア仲間がどんな出口戦略を描いているか、知りたいですよね？

今回はUstreamもなしで、多少の気恥ずかしさは承知の上にお互いの出口戦略を語り合おうという回です。
気楽に参加していただいて、ざっくばらんに話し合えればと思います。


また恒例の推薦図書の持ち寄りを今回も行います。
お題に沿っていてもいなくても構いませんので、お勧めの本を各自最低1冊ずつは持参してください。
なお、これまで誰かが持ってきた本と重複してもかまいません。


なお二次会の参加を希望される方は、必ず登録フォームの「二次会に参加する」のセレクトボックスで「○」を選択してください。

二次会の場所は恒例のお店です。
