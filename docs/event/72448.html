catch
description

 
日々の制作業務において取り扱う制作物そのものの権利や、制作物を生み出すクリエイター自身が有している権利について、ビジネス著作権検定を通して、再度基本的な「著作権」の知識を確認してみませんか？

ビジネス著作権検定はビジネス実務、日常生活において、必要とされる著作権に関する知識および関連する知識について、その基礎的な理解、具体的な裁判例・ビジネス実務における慣例を基準とする事例判断での応用力を測定する検定です。


対策講座では、初級を受験される方を対象に、問題文の読み解き方や、間違いやすいポイントなどを解説していきます。（講座はベーシックの内容も含みます）

※2016年1月19日（火）に当社会場にて検定試験を実施いたしますので、是非対策講座と合わせてお申込ください。


【検定内容】
〈ベーシック〉日常生活において著作物を扱う際、トラブルを起こさないために知っておきたい著作権問題の初歩的、入門的な知識について多肢選択問題として出題。

〈初級〉ビジネス実務、日常生活において必要とされる、（1）著作権に関する基礎的な知識、（2）著作権法および関連する法令に関する基礎知識、（3）インターネットに関連する著作権および情報モラルについての基礎的知識について多肢選択問題として出題。

※検定内容の詳細は、ビジネス著作権検定のホームページおよびBASIC専用ホームページをご覧ください。
