講演概要

トークセッション：いつまでも自分らしく働き続けるために、4つの不安にしばられない過ごし方

  • ゲスト
    • 株式会社ACT3 代表取締役 堂薗稚子さん

自分自身、何をしたい・すべきかわからない／どんな職場で働いたら幸せなの？／どんな旦那様と結婚したら自分らしく働ける？／親とはキャリアに関する価値観が違う・・など、①自分②会社③パートナー④家族の4つの不安を抱えながら過ごす女性が多くいらっしゃると思います。

第1回LiB Role Model Café では、そんな働く女性が直面する4つの不安をテーマに、7月1日に著書「「元・リクルート最強の母」の仕事も家庭も１００％の働き方」を上梓された、株式会社ACT3 代表取締役 堂薗稚子さんをゲストに迎え、トークセッションを行います。

堂薗稚子さんは、ワーキングマザーでありながら当時の組織で最年少にして女性唯一のカンパニー・オフィサー（事業部長）にまで出世した人物。しかしそこまでの道のりは育休明けの降格やリーマンショックの影響、不妊など決して平らなものではなかったそうです。ご自身も様々な問題を乗り越え、数多くの部下の悩みを聞いてこられた堂薗稚子さんが考える、4つの不安の解消法やアドバイスをお話いただきます。

抽選で20名様に、堂薗稚子さんの著書「「元・リクルート最強の母」の仕事も家庭も１００％の働き方」をプレゼント！

今回のイベント参加特典として、堂薗稚子さんの著書である「「元・リクルート最強の母」の仕事も家庭も１００％の働き方」を、抽選で20名のご来場者様にその場でプレゼントいたします。
