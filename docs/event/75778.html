description

　紙媒体としてのCourrierJaponも、ついに今回で最後！お気に入りの雑誌が次々と廃刊されていくことに、ある意味慣れっこになったとはいえ、寂しさは隠せません。５年以上続けてきたこの朝食会＠神戸も、一つの節目ということで、今回は拡大版スペシャルです。

さて、最後である４月号の特集は、「想像以上に面白く、明るい『未来』へ！」

　定期購読者には届く編集長からの手紙にあったように、今回は意図的に明るい話題をピックアップしたとのこと。そのワクワク感がみなさんには伝わったでしょうか。シェアリングエコノミーや、ベーシックインカムなど、経済価値の変化から、脳の進化や心の探究など内面的なことまで。正解のない時代だとよく言われますが、よく言えば好きなように正解を描ける時代。私自身は明るい未来を信じてなりません。

　今回は、拡大版スペシャルとして夕方からスタートし、軽く飲み食いしながら雑誌内のトピックだけでなく、ざっくばらんに今後の会の進め方やスピンオフ企画などについても、語りあいたいです。

　とはいっても、いつも通り、真剣な話から、軽いノリの話まで、いろいろな話題でおしゃべりしています。 初めての方も、どうぞお気軽にご参加下さい！

　Byたけ（@lohasway）

クーリエ・ジャポン DE 朝食会＠神戸　とは？ クーリエ・ジャポン朝食会とは、世界1500のメディアから厳選した記事を紹介している講談社の『クーリエ・ジャポン』という雑誌について、朝ごはんを食べながら、感想や意見を語り合う読書会のこと。読者が主催となって、東京、名古屋、大阪、福岡など各地で行われています。

神戸の朝食会では、会の中でも記事を紹介していきますので、クーリエの読者はもちろん、まだ読んだことがないという方にもご参加いただけます。

  • グローバルな視点を求めている人
  • いろんな国のこと、世界から見た日本に興味のある人
  • 自己成長のヒントを探している人
  • 面白い人とつながりたい人
など、多くのご参加をお待ちしています。学生さんも大歓迎です！クーリエという共通の話題を通じて、新たなつながりをつくり出し、お互いの見識を深められる場としていきたいと思います。
