山陰コンピュータ部とは

山陰コンピュータ部は、島根、鳥取の若年層(U-25)をターゲットとしたIT系の勉強会です。若年プログラマの交流と学習の場の提供を目的としています。

講師を招いてみんなで同じ事を学ぶわけではなく各々が持ち込んだ好きなことをしています。 その中で興味のあることについて共有したり、「これどうやるの？」といったことを参加者同士で教え合ったりしています。

対象

  • 中学生、高校生など(25歳以下)
  • プログラミング初心者もOK!
  • プログラマとの交流を楽しみたい人！
  • WEBサービスを開発してみたい人！

などなど、やる気のある方であれば初心者でも歓迎します！
