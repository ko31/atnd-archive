description

◆「もっと社会のためになることをしたい」「人の役に立つ仕事ってなにがあるの？」「こんな活動をしたらどうか？」などの想いを持ったことはありませんか？また、そういった自分の中のアイディアから起業を考えたことはありませんか？もしくは、今までと違う働き方を考えたことはありませんか？そういった方はぜひ一度、社会起業大学・九州校の無料セミナーにお越し下さい！自分の想いを形にする方法を学んで頂けます。

◆昨今、社会課題（福祉、環境、地域活性、高齢者、教育、生活、過疎、農業など）といった問題や、格差や困窮者の増大などが身近な問題となっています。そんな中で、自分にできることが何か無いか？そんな課題を解決したい！といった想いをお持ちの方が増えてきています。さらに、価値観、労働感などが多様化する中で、自分らしい生き方や働き方をしたいと考えている人が増えています。会社に縛られず、企業の成果にとらわれず、人の役に立つ、「ありがとう」といわれる仕事に注目が集まっています。自分の価値観が合うのであれば、農業でも、過疎の村での生活でも、自分らしさを大切にしたいと考える方も多くなっています。自分らしさを仕事にする！その結果、社会の役に立つ！
そこで注目されているのが「社会や地域が抱える課題をビジネスの手法を用いて解決する【社会起業家】」です。

無料体験授業では、そんな社会起業家の事例をもとに、その事業の社会的な課題の解決方法と収益方法をどのように両立させながら活動されているのかを紹介し、皆さんの想い描く働き方とを重ね合わせてみる時間を提供しています。さらに、個別の質問や参加者同士の意見交換により、なかなか日ごろはそのようなことを口に出して話すことができない自分の想いを語れるセミナーです。自分の想いを実現するためのきっかけが見つかるのではないでしょうか？
あなたの中に社会貢献のアイディアや熱き想いがあるならば、まずは一度、社会起業大学・九州校の無料セミナーでビジネスプランとしてブラッシュアップしてみませんか？また、社会起業家がどういったものなのか、一から知り合いという方もぜひお越し下さい！どなたでも無料でご参加頂けます。
