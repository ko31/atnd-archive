勉強会の概要

ゆる〜い勉強会は、(1)同じ課題や関心を持つもの同士が繋がる場になること、(2)普段着手できずにいたり解決できずにいる課題にチャレンジするきっかけを作る、を目的としたカリキュラムの無いゆる～い勉強会(もくもく会、ワークショップ)です。本会は、毎回異なったテーマで開催されます。

今回のテーマは「Go言語入門」です。Go言語を使っている人、もしくはこれから始める人を対象に、勉強会を通じて、(1)Go言語に関心を持つもの同士が繋がる場になること、(2)普段着手できずにいるGo言語にチャレンジするきっかけ作り、を目的としています。

Go言語の書き方やGo言語の始め方などを経験者が実地で教えますが、参加される方は↓

  • 「とにかくGoでHello Worldを実現する！」
  • 「Goでコマンドラインツールを作ってみる！」
  • 「Goを実行できる環境を構築する！」

などの目的、目標を持っての参加をお願いします！

そうすることで、なんとなく参加してなんとなく終わった…とか、なんとなく分かった気がするけど家に帰ったらやっぱり分からない、ということにはならないかと思われますし、目的や目標を明確にしておいたほうが断然進みは良くなるはずです！

（事前にカリキュラムを組んだ教室というわけではありませんので、ご理解頂ければと思います）

Go言語をやったことがない方や、これからGo言語を始めようとしている方などの初心者でもお気軽にご参加ください。そういった方が参加される場合は基礎から教えます。
