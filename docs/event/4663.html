お手軽にソーシャルアプリを作ることの出来るサービスOpenSocial Host の三回目の勉強会です。

下記のような内容で開発者が説明をしたり質問に答えたりする会です。前回までの内容も含むので、初めての方でも大丈夫です。

  • OpenSocial Host とは？
  • OpenSocial Host の初夏のバージョンアップ内容
  • OpenSocial Host で PC のソーシャルアプリを作ろう！
  • OpenSocial Host でモバイルのソーシャルアプリを作ろう！
  • OpenSocial Host での高速開発に役立つ Tips （ローカル環境の利用等）
  • Lightning Talk

また、参加者の方々のLTも募集しています。自作のOpenSocialアプリの紹介やライブラリの紹介など、OpenSocial関連であればどんな内容でも構いません。LTしたい方はぜひその旨をコメントしてください。

勉強会終了後に近くの居酒屋にて懇親会を予定しております。参加を希望される方はアンケートフォームにチェックをお願いします。
