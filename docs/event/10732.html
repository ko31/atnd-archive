【講座の課題と狙い】
環境問題としての騒音対策用制振材料、吸音材料、災害対策として建築分野用免震アイソレーター、制振ダンパー用制振材料、電子電気機器用振動減衰材料等、制振材料の応用が注目されている。最近の制振材料は、振動減衰特性と共に耐熱性、難燃性、耐久性を兼ね備えた材料特性が要求されている。今回は、制振特性向上の材料設計技術の基本と、各応用分野での要求特性、それに適応した材料設計を基本から解りやすく解説し、代表的な製品の加工技術におけるポイントを説明する。

【プログラム】

1.高分子制振材料の種類と応用分野
　1.1　種類
　　　　ゴム、エラストマー、熱可塑性樹脂、熱硬化性樹脂
　1.2　応用分野
　　　　自動車用、建築用、電気電子機器用、
　　　　車両用、船舶用、宇宙産業用

2.高分子制振材料の設計技術の基本
　2.1　高分子粘弾性特性と制振機構
　2.2　ハイブリッド制振材料の制振機構
　2.3　制振効果向上の有効技術
　　　　ポリマーアロイ、IPNポリマー技術、架橋構造の影響、
　　　　制振性付与剤、ナノコンポジット化の可能性
　2.4　機能性向上
　　　　耐熱性、難燃性、耐候性等

3.代表的な製品の要求特性と設計技術
　3.1　自動車
　3.2　建築用(免震アイソレーター、制振ダンパー)
　3.3　電気電子機器
　3.4　船舶用
　3.5　その他

4.制振材料の製造、製品加工技術における課題

5.最近の技術動向　
　5.1　最近の特許動向から考察
