description

初めて見る方も、何度も見た方も、
映画館並のクオリティでテレビシリーズを見てみませんか？

プリキュアシリーズを途中から見始めたけど、過去のシリーズを見たことがない…
プリキュアシリーズは全部知っているけど、見ることができなかったシリーズがある…
プリキュアシリーズは全作品見たけど、改めてもう一度見てみたい…

今回の上映会は、全てのシリーズを見始めるきっかけになれたり、今まで気づかなかった新しい発見が出来たりする、そんな思いを込めて全てのシリーズの『第一話のみ』を上映する上映会です。

「ふたりはプリキュア」から「スイートプリキュア」まで、全てのシリーズの第一話を、当時の新番組宣伝ＣＭやプリキュア関連の貴重なＣＭ、エンドカードまで大きめのスクリーンと映画館並の音響と音質で見せちゃいます！

上映会開催日時

２０１１年９月２４日（土）０９時００分から１３時００分まで

上映会料金

２０人くらいの場合は２０００円程度です。
