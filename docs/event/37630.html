イベント概要

アジアのテクノロジーとクリエイティブに関わる人々のフェスティバルである『明星和楽』や
各種コミュニティの活性化など、福岡のWeb界隈は引き続き盛り上がりをみせています。
本セミナーは、そんな福岡の生活環境やシゴトについてのリアルなお話を、
福岡市、福岡と東京で事業展開をしている企業、転職エージェントの3者からお伝えする企画です。

地方活性化やU/Iターン、福岡でのWebの仕事やキャリアなど、
少しでもご興味をお持ちいただけましたら、お気軽にご参加ください。

セミナー終了後は無料の懇親会も開催しますので、講師や参加者との情報交換・交流を深めて下さい！

【主催】
福岡市 経済観光文化局
Web CAT Studio （株式会社リクルートキャリア）

【協力】
株式会社アイフリーク

【会場】
リクルートGINZA8ビル 11F（東京都中央区銀座8-4-17）

対象者

・福岡でのWebの仕事にご興味をお持ちの方
・福岡出身で、いつかは福岡に帰ろうと検討されている方
・東京でWeb事業を展開されていて、地方への出店をご検討中の方
・行政による誘致・事業活性にご興味をお持ちの方 など

　※本セミナーでは主にWeb関連のシゴトやキャリアを中心にお話しする予定です。

講演者

中井 大 氏

株式会社アイフリークにてスマートフォン関連事業や新規事業全般を担当。
福岡市へは2011年1月～2013年1月まで赴任し、スマートフォン向け
カジュアルゲームの企画開発や、地元大学・専門学校との産学連携を実施し、
2月に当月帰任。
