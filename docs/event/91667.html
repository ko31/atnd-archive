日本語or韓国語を話す機会がない？！

日常生活で韓国語を話す機会がない…そんな方は、是非ご参加ください！
当言語交換会を通して日本語を勉強中の韓国語ネイティブの方と、気軽におしゃべりしちゃいましょう！きっとお友達の輪も広がると思います♪
2~4名ずつの少人数席での言語交換ができるので、大勢の前で話すのに自信がない方も話やすいかと思います＾＾

日本人5名、韓国人5名の10名で開催予定です。
(当ページは日本人用の参加申請ページです)

*対象者

  1. 日本人：韓国語初級以上、韓国人：日本語初級以上
  2. 年齢性別不問

*参加費
日本人：700円
韓国人：500円

*準備物
特に必要ありません。
話したいと思う積極性だけお待ちください♪
(こちらでドリンクをご用意しております)

*開催場所
福岡市中央区大名を予定(天神駅or赤坂駅から徒歩圏内)
詳しい場所は追ってご連絡致します。
