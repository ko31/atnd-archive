花ちゃんと筋肉hshs IoT勉強会 #1

Raspberry piを使って、花ちゃんの筋肉を眺めながらhshsして IoTっぽくやるハッカソンな勉強会です。

内容

　・Raspberry pi で Lチカやってみる

　・Raspberry pi とセンサーをブレッドボードで繋いでみる

　・プログラムをデバッグしてみる

　・Raspberry pi を WiFiでインターネットに繋いでみる

　・その他 当日やってみたいことがあったら聞いてみてください

　・全部は出来ないかも知れませんが、出来るところまでやってみましょう

注意事項

Raspberry piはB+ 推奨ですがなんでも良いです。各自持ち寄ってください。

当日会場ではご用意できません。

また会場ではハンダ付けはしません / できません。

今回Raspberry piで使用するOSはRaspbianです。

Windows IoT Coreは分かりません。やれる人は自由にやってください。

まったりやりましょう

必要機材

  • PC or Mac
    1. WindowsでもMacでも良いです。
    2. その他Raspberry piと繋ぐUSBケーブル, WiFiドングル, センサー, ジャンパーワイヤー, ブレッドボード
    3. 電源ケーブルなど
