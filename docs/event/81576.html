#【10/11開催！】Ruby on Rails Meetup

#▼講演内容
Ruby on Railsを活用し、自社プロダクトを開発している企業各社からエンジニアの方に登壇して頂き、自社プロダクトでRuby on Railsを取り入れた理由・取り入れた事で何が良くなったのかをそれぞれ15~20分プレゼンして頂きます！

※内容は変更になる可能性がございます。
※会場キャパシティの関係で、立席でのご参加をお願いする場合がございます。
※参加者多数のため、イベント開始直前(19：20～19：30)は受付が大変混雑することが予想されますす。お時間に余裕を持ってお越しくださいませ！
※20時以降はビルの入場に係の案内が必要になります。
※イベントへのキャンンセルの申し込みを一週間以内に行った場合は次回以降のイベント参加をお断りする場合があります。

##橋本 吉治氏（株式会社ジモティー）

▼【略歴】

株式会社ジモティー開発部マネージャー。大手シンクタンクにてシステム構築、技術コンサルティング、新規事業立ち上げ等々の業務を経験後、B2B向けSaaS提供企業にて人工知能ボットサービスを立ち上げ。2016年より失われつつある日本人同士の関係性を再構築して子供達によき日本を残したいという強い想いからジモティーにジョイン。
