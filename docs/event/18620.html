description

東日本大震災から５か月。僕らはどう震災と向き合ってきたか。
これからどう向き合っていくのか。

あの日、日本中が多かれ少なかれ、何もできない無力さに心を痛めた。
少しずつ、復興へ向かっていく東北。
少しずつ、薄れていくあの日の衝撃。

今、改めて僕は思う。
このままでいいのだろうか？
僕らが年を取った時、子供たちにあの時大人が何もしなかった。
そんなことを言わせないために。

今、もう一度考えよう。これからどうするか。
単なる「復興」ではなく、「新しい時代」へ向けて。
想い続け、つながろうとし続けること。それが今、ぼくらがするべきこと。

【内容】
1．この5か月、どう向き合ってきたか共有。
こうした。こうしたかったけど、できなかった。
これからこうする、こうしたい。
2． 寄付をキチンと考える。寄付ワークショップ。
3． 寄付するってこういうこと。（永田氏、野崎の対談形式）
　① 寄付税制の話。
　② ワークショップ結果を踏まえて、考察。社会をちょっと変えるお金の使い方。

できること＠福岡～Beside～主催
　
　福岡出身。さすらいのｲﾍﾞﾝﾀｰ。
　現在はNPOシーズで武者修行中。
　震災直後、いち早く「できること＠福岡～Beside～」を開催。
　福岡中の何かしたいを受け止める場の創出に奔走する。
　再上京後、日本を代表する社会起業家をゲストに招き、
　「社会を変えるをとっても具体的に考える」シリーズを主催。
　他、様々な東北支援ネットワークづくりや、イベント主催、講師として活躍中。
　日本の新たな社会づくりを担う男の一人。
