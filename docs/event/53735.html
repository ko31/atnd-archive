description

　人生を豊かにするには心の持ちようが大切です。
日々さまざまな事柄に追われ、目の前のことをこなしていかなければならない中で、自分の内面を見つめることはたやすくはありません。
瞑想によって、根源の力と知恵、愛につながることで、よりいっそう人生を豊かにし、何より楽に、人生を思いのままに生きていくことができます。　　　　　　　　
　瞑想やヨガ修行の究極の状態であるサマディに到達しインド政府に公認され、現代社会で２人のみのシッダーマスターとなった相川圭子さんが、その方法をお話しします。

【プロフィール】
ヒマラヤ大聖者　ヨグマタ相川圭子 (あいかわ・けいこ)
女性、外国人として史上初めて死を超える究極の意識状態である「サマディ」に到達したヒマラヤ大聖者。
5000年の伝統をもつヒマラヤ秘教の正当な継承者。
ヒマラヤの秘境にて死を超える修行を重ね、神我一如の「究極のサマディ」（究極の悟り）に到達。
1991年より18回、世界平和と真理の証明のための公開サマディを行う。
世界各都市で講演を行うとともに祝福を与え、人びとの意識の進化と能力開発をガイド。
