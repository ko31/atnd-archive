トークテーマ

リクルートとサイバーエージェントGの活躍女性ゲストたちに、プログラミングスキルを保有していることで仕事に役立つこと保有しているメリットなどについてお話いただきます。社内で活躍する秘訣や、企画職の仕事、プログラミングスキルの習得で広がるキャリアの可能性に興味のある方は興味のある方は是非ご参加ください。

TECH LADY HOLIDAYとは

休日のお昼に開催される女性限定のトークセッション・交流会・プログラミング体験会イベント。 ITスキルを活かして活躍している女性ゲストをお呼びし、毎回新しいテーマでITスキルの魅力を発信します。 ITスキルの習得で広がる未来の可能性を感じる休日にしませんか？ → http://lady-holiday.tech-camp.in/

こんな方にオススメ

ITスキルの習得で将来の可能性を広げたい方
登壇されるゲストは皆氏、プログラミングやデザインなど、ITスキルの習得よって将来の仕事の幅を拡げたり、働き方を自由にコントロールしている方です。そんな方々のお話を聞くことで、ITスキルを身につけることの魅力が理解できます。また、そのスキルの習得具体アクションプランを知ることもできます。
ITスキルを活かして「好きなこと」を仕事にし活躍している女性の話を聞き行動したい方
登壇されるゲストは皆氏、ITスキルを活かし自分の好きなことを活かして活躍しています。そんな方のお話を聞くことでどのように「自分の好きな仕事で活躍できるのか」が学べます。また、ゲストの方はもちろん、イベントの参加者の方も「ITスキル習得を通じ、自分の可能性を広げたい」という志向をお持ちの方々です。当イベントに参加することで、興味関心がマッチし、一緒にアクションに踏み出せるような女性たちとの交流を深めることができます。
IT(プログラミングスキル)習得を具体的に試してみたい方
イベントの後半では、「ライブコーディングセッション」「プログラミング体験会セッション」を設けております。実際にプログラミングの動きを理解し、その後自分で手を動かしてプログラミングを体験いただく時間があります。当イベントに参加することで、具体的なプログラミングスキルの習得体験もできます。

TECH LADY HOLIDAYの特徴

ITスキルを活かして活躍している女性ゲストの話を聞ける
TECH LADY HOLIDAYでは毎回新しい女性ゲストをお呼びしてトークセッションを行います。ゲストは皆氏、ITスキルを活かして自分の好きな分野で活躍している女性です。 IT分野に興味のある方や、自分の好きな分野で活躍したいと思う方には最適なイベントとなっています。
同じような考え方を持った女性と交流し、コミュニティを作れる
TECH LADY HOLIDAYに参加される皆氏は、ITスキルに興味があったり仕事の幅を拡げたいと考えている方たちです。同じような志向の女性と交流することで、お互いに刺激を受け、コミュニティを形成することが出来ます。
実際にプログラミングスキルの習得を体験できる
TECH LADY HOLIDAYでは、実際に実際にプログラミングの動きを理解し、その後自分で手を動かしてプログラミングを体験いただく時間があります。学ぶ意欲がわいたところで、すぐに具体的なプログラミングスキルの習得体験もできるイベントになっています。

運営

TECH LADY HOLIDAYは「世の中にサービスを生み出す人を増やす」を理念にしたプログラミングスクールTECH::CAMPが運営しています。
TECH::CAMP（https://tech-camp.in）は未経験からサービスを作れる技術力を身につけるプログラミングキャンプです。非効率なプログラミング学習を無くしたいという想いからスタートし、開始わずか1年余りで3500名以上の卒業生を輩出しています。
この度、女性のみなさんがITの力で活躍する場を広げるお手伝いをさせていただければと思い、TECH LADY HOLIDAYを開始しました。分からないことがありましたらどんな些細なことでも構いません。気軽にご相談ください。
