イベントの概要

F#の入門勉強会です。C#との比較に重点を置いて説明しますが、C#未経験の方でもお気軽にご参加ください。

配布資料は C#で学ぶF#入門ネタ帳 で作成中です。ご意見・ご要望は @7shi までお寄せください。

参加費用

  • 一般: 1,000円
  • 学生: 無料

内訳は場所代とドリンク代です。会場にはセルフサービスのドリンクバーが設置されています。

懇親会

特に予定していません。当日の流れ次第です。

準備する物

実際に開発を行いますので、ノートパソコンをご用意ください。

Visual Studio 2010 Professional以上をお持ちの方はそちらをご利用になれます。お持ちでない方は、以下をインストールすれば無料で環境が構築できます。

  • Visual C# 2010 Express Edition
  • Visual Studio 2010 Shell （Integrated）
  • F# CTP （InstallFSharp.msi）

Windows以外のOSでは Mono がご利用になれます。
