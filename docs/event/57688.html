池澤あやか部長とアプリを作ろう！

MA10より設立されたGeek Girls部に、Rubyの女神・池澤あやかさんが部長として就任しました。

池澤あやか部長と一緒にアプリ開発の基礎を学んで、7つのプラットフォームで簡単に公開するノウハウを身に付け、最終的にはマネタイズができるようになるまでを目指しましょう！

今回のワークショップではアプリ開発の基礎（開発ツール導入とプラットフォームでの公開の仕方）を学びます。

MA10にすでに応募していている方はさまざまなプラットフォームでの展開を、まだ手をつけていない方には、スタッフが用意した雛形アプリを使って開発することも可能です♪

マイクロソフト社 2F受付で、スタッフが入館証を配布します。

当日必要なもの

ノートPCをお持ちください。
（wifi環境はあります）

事前準備をお願いしたいこと

今回の講義内容を池澤あやかさんが体験したレポートが公開されていますので、ぜひご一読を！
https://codeiq.jp/magazine/2014/10/17484/

以下の作業を事前に行っていただけると、当日の作業がスムーズになります。

Windows ストア アプリを開発される方は Windows 8.1 の OS にVisual Studio 2013 のインストールをお願いします。

Visual Studioの無償版は以下から入手できます。
http://bit.ly/win8devtools

Mac の人は、開発環境を構築する方法については以下のURLにアクセスしてください。
http://bit.ly/vs_mac

Windows 8をもっていない人は以下のURLにあるいずれかの方法で無償で使用することができます。
http://bitly.is/YGpzcY

Firefox OSのスマホ向けにアプリを作る方は、Firefox Auroraをダウンロード＆インストールをお願いします。
https://www.mozilla.org/ja/firefox/channel/#aurora
