この夏一番気になる、21 世紀で最もクールなモバイル OS、Firefox OS。(※脳内調べ)

開発用端末 Flame の国内発売も間近に迫り、技適取得済み Firefox OS を堂々と使える日もすぐそこまで来ていますね！これを機に Firefox OS に興味を持った人、誰かと Firefox OS の話をしたい人、とにかく集まってワイワイやりませんか？

7月2日に 関東 Firefox OS 勉強会 7th がありますが、そちらに参加して、もっと身近に Firefox OS に触れたくなった、そんなあなたにもオススメです。

まだ Flame を触る機会がなかったあなたも、ミートアップで Mozilla オフィスに来れば触り放題ですよ! (…のはずだ！)

何をするの？

Firefox OS のソースコードを好きなペースで読んで、分からないことを聞いたりする会です。でも、構えなくても大丈夫。場の雰囲気を味わって帰るだけでも OK です。Firefox OS が好きな人、Firefox OS をもっと知りたい人達がゆるく集まって交流できる「ゆるふわ」な集まりが、このミートアップの持ち味です。

初めての人も大歓迎。最近、Firefox OS を知りたくなった、そんな「あなた」をお待ちしています。Firefox OS のエキスパートに気軽に質問できるチャンスです。話の輪に加わるだけでも Firefox OS の新しい発見があるかもしれませんよ！

何が要るの？

Firefox OS への興味とノート PC だけで OK です。
ちょっとした調べ事なら FxOS コードリーディングで運営している OpenGrok が使えます。

http://reading.fxos.org/source/

※ GMO インターネット様よりコミュニティ支援として ConoHa VPS の提供を受けています。

「FxOSコードリーディング」って何？

Firefox OS のソースコードを読んでいる/さわっている人達のコミュニティです。
Firefox OS を深く知りたい人達がコードを読んだりビルドしたりして意見交換しあう場です。

読んでみたいけどどうしていいか分からない。
読んでみたけど分からないことを聞いてみたい。
読んでみたから誰かに話したい。

・・・そんな人達をつなぐ場です。この機に一緒に Firefox OS を探求してみませんか？
