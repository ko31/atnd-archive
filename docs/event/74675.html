◤ゲーム交流会 ＆ GGJアフターパーティ in 札幌　◢

他所の一部で囁かれる
北海道ってディベロッパーさんいない？
→んなわけ！！

Global Game Jamで制作したゲームをみんなで持ち寄り遊んでみよう！
GGJに参加されていなくても、作るだけでなく、
これまで作ったゲームを持ち寄ったりして輪を広げよう！

◤ 交流場のご提供・軽食・Unityビール出ます ◢

お互いのゲームをプレイしてみて、感想やフィードバックなどの意見交換をするもよし、 今回のゲームジャムの作品をチェックしたりただゲームをプレイしにくるだけでも大丈夫です！ 作ったゲームをGooglePlay/AppStoreへリリースするためのサポートブースやそれをマネタイズするための相談ブースも開設します！
開場時間内（15:00〜18:00）でいつでも好きな時間帯に遊びにきてください。GGJに参加していない方でもご参加下さい。

（※持ち込むゲームに制限はありません！ただし年齢制限があるものはご遠慮ください）
（※持ち込まれるゲームが少ない場合はよくわからない会になりますので、GGJに参加された方ぜひご参加ください！）
（※Unityはビールの会社ではありませんがUnityビールはあります。）

◤ アセットストアのバウチャーカードでスキルアップを！ ◢

下記に当てはまる方には Unityアセットストアで $10分まで無料でアセットをゲットできるバウチャーカードをプレゼント！

  • Unityアセットストアのアセットを使って開発した作品
    （有償・無償や元々所有していたか期間中入手かを問わず）
  • 当アフターパーティで展示する際に使ったアセットの正式名称を明示していただけること
    （付箋に手書きでも構いません）

◤ Unityお困り相談所開催 ◢

プログラミング・マーケティング・マネタイズお困りな部分をUnityの中の人がお答えします。

タイムテーブル

その他

  • 会場から出る時間は自由です
  • 試遊会に作ったゲームを持ってくる方は、延長コードや電源タップなどをお持ちください
  • GGJ以外のゲームジャムで作られたゲームでもOK (VRゲームジャムの作品も是非！)

ぜひみなさんのご参加を心よりお待ちしております！
