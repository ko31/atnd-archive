Coronaキャラバンとは

米 Ansca 社が作成している、スマートフォンアプリケーション開発ツール「 Corona SDK 」は、TitaniumやUnity, Adobe AIRなどと並んで、AndroidとiPhoneのアプリを同時に開発できるツールとして、最近話題になっています。
米国では中学生のロバート・ネイ君がCorona SDKを使ったゲーム「Bubble Ball」で1000万ダウンロードを達成し、ツールとしての名前が知れ渡りました。

  • Android版Bubble Ball
  • iOS版Bubble Ball

本イベントでは、Coronaのサンプルソースを読みながら、文法や関数の使い方をなんとなく掴んでもらうことを目的にしています。

Corona SDKネタで新潟県を練り歩く勉強会。だからキャラバン。

Coronaで新潟県のスマートフォンアプリ産業に新たな風を吹き込むため、定期的に県内を巡ってCoronaについての知識を増やすお手伝いをします。

今回は第１回ですので、環境設定と基礎的な部分を中心に行います。
WindowsかMacのノートPCを持ってお集まりください。
