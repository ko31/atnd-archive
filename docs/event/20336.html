もう一回WebデザイナーのためのGit勉強会をやります。

前回（ http://atnd.org/events/19400 ）大好評だったWebデザイナーのためのGit勉強会、人数がいっぱいで参加出来なかった方のために、@satococoa さんがもう一度講師をしてくれることになりました。今回は小さなオフィスで小じんまりとアットホームな感じでやります。前回、参加出来なかった方を優先的に声をかけさせていただいたのですが、まだ若干枠が余りましたのでATNDで募集しました。

========

バージョン管理システム Git が使えるデザイナーをもっと増やしたい！と思うエンジニアさんたちが集まって、丁寧に Git が使えるまでデザイナーに教えるというイベントを開催します。

新しいサービスを立ち上げる際、デザイナーがバージョン管理システムを使えることで一気に効率がよくなります。つまり、これからのデザイナーには必須なスキル！

バージョン管理？って方も心配いりません。新しいスキルを身につけたいデザイナー、エンジニアの知り合いが欲しいデザイナー、エンジニアたちに自分のデザインスキルをアピールしたデザイナーは是非、ご参加ください。

講師

@satococoa さん

参加条件

申し訳ないのですが、ちょっと厳しい参加条件があります。

Webデザイナーである（デザインが出来る、コーディングが出来る）
Macを持っていて、Macを持ってこれる（windowsでもVMwareでubuntuを動かせればOK）

以上の条件を満たしている方限定になります。

用意するもの

自分のMac（アダプタを忘れずに）

参加費用

無料（飲食OKなので、各自飲み物などを用意してください）

開催日時

2011年10月01日（土曜日）
14:00 〜 18:00（懇親会は18:15より開始予定）

開催場所

合同会社フィヨルドオフィス
http://fjord.jp

行き方

１、初台駅、国立劇場の出口から出ます
２、牛角、マクドナルドが左側に見える方向でまっすぐ進むとガスト（一回はセブンイレブン）が見えます
３、一階がセブン、二階ガストの建物の右側に細い道（本町小路という名前）があります。そこを直進。
４，200mくらい歩いた左側に「ドエルヤマト」というマンションがあります。そこの203を呼び出してください。

注意点

１、OSはMac OS X 10.6 Snow Leopard以上をインストールしておいてください

もし古いOSを使っている方はVMwareかVirtualBoxを使ってUbuntuをインストールをお願いします

２、Xcodeを事前にインストールしておいてください

※これのインストールに1時間以上かかります

インストール方法はこちら
https://github.com/prog4designer/p4d-git/wiki/Git-Install

３，テキストエディット以外のエディタをインストールしておいてください

Coda、Espresso、CotEditor、mi …など

特にXcodeはインストールに時間がかかりますので、忘れずにインストールをお願いします。

４，Gitのインストールに挑戦してみてください

インストール方法はこちら
https://github.com/prog4designer/p4d-git/wiki/Git-Install

質問はこちらで

もし、上記注意点について不明な点、質問がありましたらこちらで質問して下さい。

http://lingr.com/room/prog4designer

Gitがインストール出来なかった方

Xcodeのインストール以外の作業でわからないことがあり、Gitがインストールできなかった方は、当日13:30にご入場ください。インストール作業をお手伝いします。

フィヨルドオフィスの側の居酒屋で行います（参加自由）。
