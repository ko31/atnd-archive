description

アコースティック楽器などを持ち寄って花見やりましょう。


  • 【重要】このイベントは音楽フェスなどではなく、あくまでも『お花見』です。ですので、楽器についてですが

    • 電源やPAなどはありません

    • 基本アコースティック楽器でお願いします

    • 電池駆動式のエレクトリックなどでもOKです

    • 音量は近隣のメイワクにならない程度で各自コントロール願います

    でやりましょう

  • いる場所は当日Twitterでpostします。ついったー→ @gonoverdrive

  • 屋外なので雨天中止にします。

  • 開花してなくてもやります。

  • 食料は各自調達されたし。飲みものも同様。ブルーシートは持っていきます＆そのへんのものも持ってきてくれると助かります！

  • もちろん楽器もってない人も参加OKですが、なにかしら持ってきた方が楽しいかもしれません

  • 当日は、ハピモ（=盛岡を対象にした民営の地域SNS）のOFF会と合流する可能性があります。ハピモのサイト

  • 【追記】当イベントの終了後、会場近くの飲食店にて『夜桜見物は寒いので屋内で花見だin盛岡』が開催されますので、こちらもあわせてどうぞ。なお「夜桜見物は寒いので屋内で花見だin盛岡」の参加申し込みについては、今ご覧のページではなくこちらのページでお願いします。
