description

最近、チームで仕事をしていて、イラッとしたことはありませんか？
チームメンバーから
「やる意味あるんですか？」
「○○なので、できませんでした」
「私の担当ではないので」
といった声が聞こえてきた時。

自分には関係ないはずなのに、発言を聞いた途端に集中できなくなる。
「とばっちり」を受けるのではないかと思い、心がザワッとする。

チームで仕事をしていれば（意識しているかどうかにかかわらず）
何らかの形でお互い影響し合っています。
人間である以上、周囲の人の言動で心が動いてしまうのは自然なことです。

ですが、あなたの中にある「マイルール」に気づき、これにうまく対処することで、
他人の言動にイラッとすることが劇的に減っていくのです。

イラッとした感情にフタをすることなく、
「その人はその人なりに大変な思いをしているんだな」と容認する。
そのうえで必要な働きかけをしていく。

そうすることで仕事と人間関係に余裕が生まれ、
チームのパフォーマンスがグッと上がるのです。

私自身、企業の人事・教育部門に28年間携わり、
社員教育やチーム運営に関する悩みも数多く見聞きしてきました。

まとまっているチームのリーダーは、メンバーの心の動きを察する能力に長け、
これを大切にしていますし、
メンバーも「チーム」にふさわしいマインドで、
目標を達成するため積極的に考え、行動しています。

本セミナーでは、
私が会社員時代に培ったOJTトレーナーの教育や
チーム力向上のノウハウをもとに、
独自の理論を採り入れ体系化した
「最高のチームづくり」の鍵をお渡しします。

★プログラム内容
・自分の価値観を知る
　　カードゲーム形式のワークで、楽しみながら価値観を明確にしていく

・一生の関わり合いを持つ
　　相手との一体感を作る「魔法のワーク」

・自分軸の作り方
　　あなたの過去と現在を再確認する「言葉のワーク」
　　自分軸を作るために必要なパーツを埋め込むワーク
　　自分軸ができた、その先にあるもの

・自分の磨き方
　　決断力、行動力、責任力を磨くテクニック

★こんな人のお役に立てます
・チームの中に不平不満ばかり言う人がいる
・つい相手を責めてしまう
・決断力、行動力、責任力を身につけたい
・「〇〇でなければならない」のに、メンバーが従わない
・他人に左右されることなく、自然体でいたい

この他、本セミナーに興味のある方も大歓迎いたします！

★受け取れるもの
・他人に左右されることのない、揺るぎない自分になる
・決断力、行動力、責任力が身につく
・チームの生産性向上に貢献できる
・規律あるチームの雰囲気が醸成される

★講師プロフィール
石津恵子（いしつけいこ）

チーム活性化ナビゲーター。

東証一部上場製造業会社人事・教育部門勤務28年
新人教育の企画・運営（対象者12,000人）
およびOJTトレーナー（年間150名）に対する教育業務に携わる。

OJTトレーナーのサポートを通じて、現場の教育に関する悩み、
チームづくり・再構築への悩み解決に関与。

この経験から、やる気のない社員のモチベーションをあげ、
チーム全体の目的遂行力を向上させる独自の理論
「５ワード３セット～最高チームのつくり方～」を確立。
チームの立て直しはもちろん、活性化にも大きく貢献する。

これを世の中に広めるために2018年独立。
発表と同時に数々の企業から、チームづくりに欠かせない
新たな手法として話題を呼んでいる。
