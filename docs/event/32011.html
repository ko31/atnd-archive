【講演要旨】
　無線技術の新しいパラダイムとして、マルチユーザMIMO技術がLTEや次世代無線LAN規格IEEE802.11acに採用された。同じ時間・周波数で信号を多重化できるビームフォーミング技術であり、伝送レートの高速化すなわちチャネル容量の飛躍的な増大が図れる。本セミナーでは、幾つかの国際規格動向を踏まえて、マルチユーザMIMO技術の基礎からチップ設計の現状まで分かりやすく解説する。

【プログラム】

1　マルチユーザMIMOとは?
　1.1　ユースケース
　1.2　システム構成
　1.3　LTE、IEEE802.11acでの採用動向

2　MIMOデコーダの基礎
　2.1　ZF/MMSEアルゴリズム
　2.2　Lattice Reductionアルゴリズム
　2.3　MLDアルゴリズム

3　マルチユーザMIMOシステム
　3.1　必要な線形代数の復習(QR分解、特異値分解)
　3.2　線形プリコーダ
　3.3　非線形プリコーダ(Dirty paper/Tomlinson-Harashima/Vector Purturbation)
　3.4　チャネル情報の圧縮とそのフィードバック
　3.5　システムパフォーマンス
　3.6　次世代無線LAN規格IEEE802.11ac対応チップ設計例の紹介
