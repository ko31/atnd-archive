＊お店決まりました！


阪急京都線烏丸駅より徒歩2分
地下鉄烏丸線四条駅より徒歩5分
の　Daniel’s　ダニエルズ　です。

(京都府京都市中京区錦小路通高倉西入ル グラマラスフードビル２Ｆ)

人数減るとちょっと困りますが、
増える分には構いませんので
興味ある方ご連絡ください！

以前の東京デザイン飲みでお話していた、関西版です。
デザイン学んでいるいろんな人たちでわいわいやろうって会です。４月から同期で働く人も大学院へ行く人もデザイナーになる人も実習で仲良くなった友達に再会したい方も他大学のデザイン学生と話してみたい方もどうぞ誰でもご気軽にご参加ください！

＊現社会人の方でも興味ある方なら誰でも大丈夫です！

あとから参加の最終締め切りは２月２３日です。

Twitterない方は、FacebookかこちらのURLからでも登録できます。
