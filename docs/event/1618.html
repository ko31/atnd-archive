title RCMS実践勉強会
catch テンプレートカスタマイズまでを含めた制作者向け勉強会です。
description

RCMSは、RDBMS（リレーショナルデータベース管理システム）を利用した、あらゆる要望に応える最強のCMSです。
守備範囲の広さは他CMSの追従を許さず、予算のない小規模Webから、大がかりなDBサイトまでをカバーできます。
当日は、テンプレートのカスタマイズを中心に、独学では理解しがたい部分を学んでいただきます。
まずは無償版をお試しください。

RCMS
http://www.r-cms.jp/
