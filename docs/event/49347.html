description

CMSインサイドは「OSSコンソーシアム CMSビジネス部会」が運営する、CMSの機能を徹底比較する無料セミナーです。WEB制作者／担当者の方がCMSの選定をする際の参考資料としていただく事を目的としています。CMS導入をお考えの企業、大学、システム開発会社様など、どなたでも参加していただけます。各回で取り扱うCMSは一種類で、各CMSを代表する方に2時間半の講習をお願いしています。

「CMS導入を検討する際、検証や比較をもっと深く行いたい」という声にお応えするため、2014年はインストールや設定をワークショップ（ハンズオン）形式で開催します。
PCをご持参いただき、毎回ローカル環境にインストールして使い込んでみます。

　※新しいお知らせはFacebookページをご覧ください。

第20回CMSインサイドは4/21(月)16:00～18:30の開催

今回のCMSは最強のエンタープライズ型CMSの声も高い「Alfresco」です。

WEBコンテンツ管理のみではなく、文書管理システムとして高名なAlfrescoは最高峰のCMSと言われています。

数々の導入実績を持つ株式会社イージフ杉本琢磨様に、ローカルPCへのインストールをお手伝いいただき、インストールしたAlfrescoを一緒に操作してみる150分です。
【ご注意】
・セミナー当日はPCをご持参ください
・見学のみのご参加も可能です
・追加で必要なツールのダウンロードをお願いすることがあります。
　事前にFacebookページでご確認ください。

※4/21のワークショップのために用意していただく環境

http://wiki.alfresco.com/wiki/Community_file_list_4.2.fより Alfresco Community Edition のインストーラをダウンロードしておいてください。現時点での最新：バージョンは4.2.fです。

すでにその環境でTomcatが起動していると、ポートがバッティングする可能性がありますので、あらかじめ止めておいてください。（インストールされているだけなら問題はありません。起動してる場合のみ問題が発生します）

インストール自体はとても簡単ですので、Alfrescoの豊富な機能をたっぷり体験していただきます。
