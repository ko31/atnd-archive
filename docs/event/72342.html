PHP勉強会@相模原 の第3回を2015年12月22日(火)に相模大野(町田の隣)で開催します。

はじめに

PHP勉強会@相模原はPHPに興味のあるエンジニアを対象とした勉強会です。
延べ2～3人の発表者による10分～20分程度の講演と質疑を行う形式で進めます。
発表内容は決まり次第このページでお伝えしていきます。
聞きたいテーマをコメントに書くと話してくれる発表者が現れるかもしれません！

参加条件

PHPに興味のある方ならどなたでも。
参加費は取りませんし、*途中入場/退席もアリ*ですのでお気軽にご参加下さい。

タイムテーブル (仮)

  • 20:00　開場・準備
  • 20:15　オープニング・自己紹介
  • 20:30　発表セッション開始
  • 21:20　LT発表開始
  • 21:45　片付け
  • 21:55　完全撤退・解散

講演 (10～20分発表) (募集中)

  • 自作 IO_〜 系パッケージ(IO_MIDI や IO_SoundFont) の紹介 (yoya)

PHP の話で聞いた人の実になりそうな事であれば何でも構いません。
PHP で何か作ってみた。フレームワークの復習、パフォーマンス改善、PHP 拡張うんちく、内部構造の話し、 etc 何でも。

LT (5分発表) (募集中)

PHP で話したい事があれば何でも。
