イベント ユーザー
event_id 62426
title あなたのビジネスアイデアを、2時間で商品化するワークショップ
catch 大きすぎるビジネスモデルは結局、動き出せません。小さく始めて、あなたが明日から動ける商品/サービスづくりを一緒に行いましょう。
description

「将来、自分の好きなこと、やりたいことで起業してみたい！」
「いつか会社を辞めて独立して…」
「副業なんかして、収入を安定できたらなぁ…」

そう思ったアナタ、ですが実際に、独立起業や副業に向けて「動けて」いるでしょうか？

当ワークショップでは「やってみたいという気持ちはあるけれど、一歩を踏み出せない」という方に向けて

「アナタの強みを引き出し、アイデアを商品化する方法」
をお伝え致します。

また
・自分のアイデアが本当に売れるのか？を確かめる方法
・適正な価格設定の方法
・アイデアを派生させ、更に売上を増やす方法
・他人と十分に差別化できているかのチェック方法
などを知ることが出来ます。

講師自身、サラリーマンから脱サラして独立起業をしている人間です。
いまでは
webマーケティング/事業コンサルティング/心理カウンセリング
を生業として活動しております。

サラリーマンや多忙な学生に共通する悩みは
「時間がない」、「リソースがない」
ということです。

本ワークショップは、そんな方々でも独立起業や副業へ踏み出せるよう
「いかに小さく」
「いかに早く」
「いかにリソースを使わずに」
ということに主眼を置いたものとなります。

そのため、大規模なビジネスモデルや
設立起業をお考えの方には
本ワークショップは向いていないかもしれません。

逆に、
「小さく始めたい」
「ゼロからスタートしたい」
「明日から動き出したい」
という方にはオススメのワークショップになります。

本セミナーの形式は、少人数の「ワークショップ形式」です。
つまりこのセミナーを受け終わったときには
・あなただけのオーダーメイドで
・小予算で
・明日から動き出せる
ビジネスアイデアが商品化されることとなります。

本ワークショップは6,000円と
やや高めの設定にはなりますが、上記のメリットを考えると
十分元は取れるのではないか？と思います。

また本ワークショップで商品化されたアイデアが
適正な価格にて売れることで
参加費用は回収することは可能です。

最後に、本セミナー開催はあくまで試験的なものであり
今後、本ノウハウをこのようなセミナー形式にて公開する場面は
なくなってしまうかもしれません。

そのために私としてもフィードバックを受けたく
少人数形式とさせて頂いております。

ぜひ、このチャンスを逃さないうちに
あなたからのご参加をお待ちしております。
