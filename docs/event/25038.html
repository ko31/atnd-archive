catch べ、別にノマドブームに乗ったわけじゃないんだからね！
description

目的

個人で仕事をしているエンジニア(主にIT系)が、お互いの仕事について知ることを目的としています。

趣旨

フリーで仕事をしていると、他の案件で手一杯で仕事が請けられなかったり、あるいは一人でやるには規模が大きかったり、あまり自分が得意でないスキルを要求されたり、と言ったことがあると思います。かといって、従業員を雇うのはリスクだし、、、。
そこで、フリーで仕事をしているエンジニアの横のつながりを持とう、という目的で当勉強会を企画しました。
また、違うスキルを持ったエンジニア同士でコラボしても面白いだろうなと思ってます。
尚、あえて「交流会」ではなく「勉強会」としたのは、「異業種交流会」などに漂う「とにかく名刺配って人脈を増やそう」的な軽薄な雰囲気が嫌だったので（笑）

場所

喫茶店ルノアール新宿区役所横店 6号室。
http://www.ginza-renoir.co.jp/myspace/mys006.htm

参加費

10,400円 ÷ 参加人数　＋　飲み物代
（懇親会費は除く）

参加資格

  • 個人事業主、または小さな会社で経営している/働いているエンジニア
  • そんなエンジニアに仕事を出すことを検討されている方（大歓迎ｗ）
  • フリーになることを検討されている方

プログラム

参加者全員に５分間のLightning Talkをしてもらおうと思ってます。
例えば、自己紹介/仕事内容/得意技術/チャレンジしたいこと/実績/困ったこと/悩み、など好きな事を発表して下さい。
発表形式は、口頭だけでも良いですし、プレゼンしても良いですし、デモをしても構いません。

LTの後はブレストやディスカッションを予定してます。
また、懇親会も行う予定です。
