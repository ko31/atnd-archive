Candle Night Live!!第１夜
「インドネシアの秘宝」ジャワ舞踊公演

日時：6/19（土）16:30～18:30

参加費：3000円
（ドリンク＆玄米＆雑穀のインドネシアンプレート付）

公演：サンガール・パムンガス

http://sanggarpamungkas.blog60.fc2.com/blog-entry-43.html

夏至は、1年で一番陽が長く、夜の短い日です。

都会にいると忘れがちな、夜の暗闇や太陽の恵み。
電気を消して、キャンドルの自然な灯りの中で過ごしてみる…
普段とはまったく違う、ゆったりとした時間の流れを感じます。

キャンドルの暖かな灯りの中でおいしいお食事を楽しみながら、暗闇に溶け込むような幻想的なガムランの音色と、優美なジャワ舞踊を堪能する‥

キャンドルが織り成すゆったりとした夏至の夜をお届けします。

【公演者プロフィール】

飯島かほる
ハワイ大学音楽学部民族音楽科卒業。在学中よりジャワガムランと舞踊の基礎をH.Susilo氏に学ぶ。その後ソロ在住のジャワ踊の大家S.Ngaliman氏に師事し、ハワイとインドネシアにおいて、ガムラン、舞踊の学習と公演活動を続ける。インドネシア国費留学生として、1993年から97年までジャワに滞在し研鑚を積む。帰国後はジャワ舞踊グループ「サンガール・パムンカス」を設立。後進の指導や公演活動、ワークショップ等を行う。最近では音楽、舞踊、美術などを融合したアートプロジェクトの企画や製作を行う一方、ジャワ舞踊を学んだ経験に基づいて
、東洋的な身体調整を研究中。http://sanggarpamungkas.blog60.fc2.com/

マライア・イレト
幼少の頃からバレエやモダンダンス、フィリピン舞踊など様々な踊りを学ぶ。２００２年にインドネシア・ジョグジャカルタ滞在中にジャワ舞踊を始める。２００３年よりサンガール・パムンカスのメンバーとして研鑽を積む。フェアートレードや人権団体などでの仕事を経て、現在はフリーランス翻訳家。

稲留暁子
音楽療法士養成校卒業後渡英。エマーソン・カレッジでAnthroposophy(人智学）を学ぶ。帰国後、障害者支援施設などにて音楽療法を実践する傍ら、サンガール・パムンカスでジャワ舞踊を学ぶ。音、動き、身体の調和に興味を持つ。
