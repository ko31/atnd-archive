description

　社内を活性化させるための方法、「評価制度」「表彰制度」あるいは「提案制度」、一度は導入したことがありませんか？
　そして、何の効果もないままに、マンネリ化、あるいは立ち消え、そして最悪の場合かえって人の意欲を失わせる結果に終わっていることも少なくありません。
　これらの制度を十分に社内の活性化につなげるには、実は心理学の知見に基づいた原則を踏まえなければなりません。
　行動分析学～少し応用編～として、「表彰制度」と「提案制度」を例にとって、上手な社内活性化の方法を考えます。
