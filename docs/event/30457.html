description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション1(30分)
　『HTML5でノンプログラミング×インタラクティブ・コンテンツ制作
　　ーオーサリングツール・HypeとAdobe Edgeの紹介』iNDeVisualの浅野さん

　＜内容＞
　Webまわりでの「Flashはずし」が加速する中、
　HTML（Web標準）で制作された、動きのあるコンテンツは
　より求められる場面が多くなっているのではないでしょうか。
　ところが多くの場合、HTMLでこうしたコンテンツを
　制作しようとするならばJavaScriptへの理解は必須で、
　またヘビのようなコードとのにらみあいは避けられず、
　Flashのように直感的な制作環境が整っているとは言いがたいものでした。

　このセッションは、
　HTML5に準拠し、より視覚的／直感的に
　インタラクティブ・コンテンツを制作できるツール、
　「Hype」「Adobe Edge（プレビュー）」を使用して、
　手軽な例を作りつ発表しつつ、Web制作歴３ヶ月ちょっとの発表者が
　初の大舞台で右往左往しているさまを
　ニラニラと楽しめる内容になっております。（あれ？趣旨変わっとる。。）

　「Hype」
　http://tumult.com/hype/

　「Adobe Edge」
　http://www.adobe.com/jp/products/edge.html


　＜浅野さんの紹介＞
　映像作家。
　「iNDeVisual」という任意団体で、
　映像制作や良い（けど大きな資本がつかない）映像作品を
　よりフリーに発表できる環境作りなどに取り組んでおります。
　（代表以外に人材がないため、バカでも肩書きは「エクゼクティブ」）
　目下、法人化が最初の目標。

　「食えない映像作家」を自称しておりましたが、
　いよいよ食い詰めてきたので、Web方面に活路を求め、
　五日市のスペースコントロールというところで職業訓練を受講中。
　その甲斐あってクライアント側からサーバ側(PHP)まで、Webまわりのことは
　一通りできるようになりました。
　足りないのは、代表のセンスと人望だけ＞＿＜

　http://indevisual.jp/

　

●セッション2(40分)
　『ウェブと写真』カンドウ/シェアフルの柳谷さん

　＜主な内容＞
　子供が生まれてから一眼レフを買い1日1枚正面写真を
　撮ることにして2年8ヶ月になりました。
　20歳にモーフィングアニメにするのが夢。
　何も知らない人に商品等の特徴を伝えるためには、
　どういう写真が有効なのか、デザイナー視点で考えながら
　日々写真の勉強中です。

　アジェンダ
　・自己紹介
　・こんな仕事してます
　・写真を撮れるようになりたいとおもったこと
　・家族と写真と加工と共有
　・ウェブだからこその写真
　・一般の人におすすめのPixlrでバナーをつくってみる(実演)
　・PixlrとPhotoshopとの違い
　・まとめ

　Photo editor online
　http://pixlr.com/

　＜柳谷さんの紹介＞
　イラストとデザインとインタラクティブのお仕事をよく受注しています。

　http://www.yampuu.com/
　http://www.sharefl.jp/

　

●セッション3(40分)
　『WEBの表に見えない世界のお話』エスの大林さん

　＜主な内容＞
　先日、とある業者がデータを飛ばして大騒ぎに
　なったのは記憶に新しいところです。

　しかし報道や Twitter のテキストを見ていると、
　解釈の間違いや文字の誤用が散見されました。

　そこで WEB には切っても切れないレンタルサーバは
　どんな仕組みで動いているものがあるのか、
　そしてそれがどういう名前のサービスなのかを
　インフラエンジニアの視点から解説します。


　＜大林さんの紹介＞
　2004年くらいからサーバを触りはじめ、
　2007年からWEBの世界に入りました。

　WEB でユーザーの目に触れない部分はだいたい守備範囲なので
　インフラエンジニア、プログラマー、稀にディレクターなど
　できることは頑張ってやってます。

　専門分野は Linux と ネットワーク。
