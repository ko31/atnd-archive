■概要

Pepper for Bizの概要とSDK（Choregraphe）の説明を行います。
Pepper for Bizの開発してみたいデベロッパーやクリエーターの方や導入を検討されている方向けの基礎的なワークショップです。

直感的に開発ができる SDK を利用するため、プログラム経験のない方でも受講できます。

このワークショップでは Pepper for Bizの以下の機能を体験いただけます。

  • Pepper一般販売モデルとの違い
  • しゃべる
  • 動かす
  • タッチセンサーに反応して動かす
  • 言葉を認識して動かす

■対象

  • Pepperロボアプリ開発に興味のある学生または社会人の方
  • Pepper for Bizに興味のある方

■持参品

  • ノートPC（WindowsまたはMac）
    　※事前にPepper開発に必要なChoregraph 2.4.3のインストールをお願いします。
    　　　アルデバランデベロッパープログラム
  • 社会人の方は名刺を1枚ご持参下さい
  • 飲食物の持込可

注意事項

  • 入場は開始時間の30分前からとなります。
  • 基本的にPepper開発未経験の方を対象としております。
  • 開発用PC（Mac)がありますが、数に限りがございますのでお貸し出来ない場合がございます。
