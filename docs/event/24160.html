catch 従業員一人当たり一日13,890円がもらえる助成金制度
description

世界的な金融危機や景気変動などの経済上の理由による企業収益の悪化から、売り上げや生産量が減少し、事業活動の縮小をよぎなくされた事業主が、その雇用する労働者を一時的に教育訓練（社内教育・社外教育）をさせた場合に、教育訓練に係る手当て若しくは賃金等の一部が助成されます。
当助成金は、バブル崩壊後最大規模の支給額・助成率であるとともに、支給された資金の使い道も自由なため、非常に使い勝手が良く、企業の発展に必要な投資を効率的に行なうことができます。

御社での助成金活用が可能かどうか、ぜひ説明会に足をお運びいただきご確認ください。
