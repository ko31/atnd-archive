description

日本百名山にほんめでもある群馬県の赤城山へ登ろうと思います。

赤城山（あかぎさん(あかぎやま)）の情報 – 登山・ハイキングコース – MAPPLE 山と高原地図

タイムテーブル（詳細は未定です。）

時間 場所 タイトル
早朝に都内を出発し、午前中に登り始め、お昼頃山頂で昼食を摂り、 夕方下山し、温泉に入って、地元で夕食を食べたいと思っています。 都内に戻ってくるのは夜になると思います。

詳細が決まり次第、随時アップしていきます。

僕は平日休みなので、火または水曜日が希望です。
週末でも可能です。

集合場所

車と電車と人数により決めようと思います。
5名までは車でビジターセンターまで行きます。
新宿になると思います。

服装

富士登山よりは軽装で構いません。
登山靴とザックは忘れずに。
服装は長袖シャツとトレッキングパンツが望ましいです。
山の天候はすぐに変わるので雨具は必ず持ってきてください。
念のためヘッドライトと防寒着（かさばらない程度）も。
水分（計1L）と昼食と糖分。

赤城山について

初心者向けの山ですので、登山初心者でも楽しんで登れると思いますし
山頂の景色も晴れていれば絶景のようです。

登山に興味のある方、是非！
