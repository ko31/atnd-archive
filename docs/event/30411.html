description

【講座のポイント】
　ヒートアイランド対策の１つとして定着しつつある高反射製品（太陽熱高反射材／遮熱塗料／日高反射率塗料）の開発に必要な基礎知識，評価技術を伝授いたします。その基本性能である日射反射率の測定方法を中心に，実測例を示しながら，測定装置の原理から具体的な手順，検査機関に依頼測定する際のコツや，測定値の正しい解釈など，高反射製品を適切に評価，選択する際に必要となるポイントを分かりやすく解説します。
　また，近年制定された関連工業規格（JIS K 5602「塗膜の日射反射率の求め方（2008年制定），JIS K 5675「屋根用高日反射率塗料（2011年制定）」）を詳解し，今後の展望も紹介します。

【プログラム】
１．はじめに
　　1-1　地球温暖化，ヒートアイランド，省エネ対策
　　1-2　太陽熱高反射材が何故求められるのか
２．太陽熱エネルギー
　　2-1　基準太陽光
　　2-2　全天日射と直達日射
３．太陽熱高反射材の基礎原理
　　3-1　用語について（遮熱，断熱，高反射）
　　3-2　基礎原理
４．日射反射率の測定
　　4-1　なぜ分光反射率測定が必要か
　　4-2　分光反射率に基づく日射反射率測定の実際 − 使用する装置，反射標準，測定手順 −
　　4-3　測定値は一体何を表しているか − 実測例の紹介，誤差要因の検討 −
　　4-4　分光反射率測定以外の評価方法（温度測定，アルベド測定）
５．工業規格
　　5-1　JIS K 5602「塗膜の日射反射率の求め方」は材料開発過程において有効か？
　　5-2　JIS K 5675「屋根用高日反射率塗料」の考え方
６．今後の課題
　　6-1　太陽熱高反射材の用途展開
　　6-2　海外動向，高機能化
