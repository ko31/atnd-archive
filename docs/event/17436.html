Titanium Mobileを使ってアプリを作ろう
Webデザイナー必見！

学生・初心者　大歓迎!!!

JavaScript・Titanium Mobileの基本的な解説を予定しています。
初心者の方でも、スマートフォンアプリ開発に興味がある方は、
奮ってご参加ください。
また、下記に該当する方もご参加ください。

・サーバサイド開発者や、Webデザイナーの方で、JavaScriptを活用して、
　手軽にスマートフォンアプリの開発を行いたい方
・手軽に、マルチプラットフォームアプリ開発を行いたい方

Titanium Mobileとは

Object-CやJavaを用いずに、JavaScriptでAndroid/iPhoneアプリを
簡単に開発できるツールキットです。
Object-CやJavaでの開発とどう異なるかを本ハンズオンで実体験してください。

参加資格

以下の条件を満たしていればＯＫです。

  • 開発に使用するＰＣを準備できること
  • Macをお使いの方はMAC OS X10.6以上でお願いいたします。
  • 今回はAndroidアプリの作成に限定させて頂きます。

    ☆TitaniumMobileの開発環境を整えておいてください。
※実機テストの端末、ネットワーク環境などは準備します
