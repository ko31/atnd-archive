■イベント概要

「HTML5って……」実は良く分かっていないけど、相槌を打ってしまっているあなたのための勉強会です！

HTML5は、まさに今、リアルタイムで進化し続けている膨大な仕様と実装のカタマリ。
引用と参照を最大活用して、60分でひと通り解説してみたいと思います。
さまざまなライブラリが充実してきているので、面白そうなところからツマミ食いして楽しみましょう！

広義HTML5＝ほぼ何でもアリのライトニングトークの機会もありますので、
トークしたい方はメッセージをお送りください（5分×数名程度）。

内容（予定）
・HTML5にまつわる、いままでのあらすじを振り返る
・マークアップからAPIまで、HTML5の歩き方
・今日から始めるためのライブラリを、PONPONPONと紹介

・参加者によるライトニングトーク
