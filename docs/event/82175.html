description

　ららぽーとEXPOCITYにおいて、
　皆様の「働く」を応援するイベント「おしごと万博～ジョブエキスポ～」を開催します。
　
　赤ちゃん本舗さんによる「子育てと仕事の両立支援セミナー」や、
　無印良品さんの「眠り」をキーワードにしたセミナー、
　手作り雑貨のワークショップ体験など、様々なイベントをご用意しています！

　大阪府広報担当副知事の「もずやん」も応援に駆けつけますよ～！

　皆様のご参加お待ちしております！

【コンテンツ】
