description

『施策に対する成果が、数値としてハッキリ見える』ということで、Webマーケティングの領域ではダイレクトマーケティング的な手法に重きが置かれています。そのため、短期的な成果に主眼が置かれ、長期継続的な顧客との関係構築はおざなりにされる場合も少なくはありません。

しかし、自社の持つ顧客接点での強みを活かし、ある程度の時間をかけて多くのユーザーを獲得し、日々活用されるサービスへと成長する事例も見られるようになりました。例えば、日本マクドナルドの『トクするケータイサイト』もその例の1つです。また、無名の小さな企業でも、同様のプロセスを踏んで成功している事例が複数あります。

今回は、トリムタブジャパン代表・中谷氏をお呼びし、短期施策に終始しない『モバイル施策戦略』についてお話し頂きます。

『企業のWeb／モバイルのご担当者』『広告代理店のプランナー』『制作会社のプロデューサー／プランナー』を対象とし、大きく次の2点に関して学んで頂けます。

1）自社の顧客接点での強みをモバイル施策へとつなげる方法。
2）モバイルの特性を活かした“顧客接点の作り方”。
3）接点を作った“顧客とのより良い関係の築き方”。

講義の流れ）
1．成功企業が実践する、モバイル活用の戦略とは？（事例紹介）
2．成功企業の共通項から読み解く、成功の法則とは？顧客との絆づくりを促進する手法とは？
3．まとめ
4．質疑応答／名刺交換会

受講頂くことで、学んだことを実際の業務で活用できるようになります。
