金曜朝、朝活でブログを書く時間を確保

ブログを書いている皆さん、
ブログは定期的に更新できていますか？

「ブログ継続ってなかなか難しいよねー」

とブロガー仲間と話していて、

ブログ記事を書く時間を朝活で確保したら
自ずとやるようになるのでは? とひらめきました。



<< こんな方、お待ちしています! >>


● ブログを書いている方
( 継続するのが苦手な方、大歓迎 )

● ブログ仲間と出会いたい方

● ワードプレスやアクセスアップに関して
　 知識を得たい方
( ミニ講義タイムあり・質問OK )


毎週継続して1記事作成するという点に
フォーカスして、ブログ作成に慣れ親しむのが
この朝活の目的です。

○ 起業・集客の一環でブログを書いている
○ プライベート・趣味でブログを書いている
など、
ブログを書く目的は何でも構いません。

一緒に朝の時間を有効活用しましょう!
