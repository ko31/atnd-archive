第1回デバイスガジェッターの会　関東支部　平日の部

皆さんが日常的にお使いのスマホやフィーチャーフォン。
電話をするために買う人も、メールをするために買う人も、出会い系サイトでワンクリック詐欺に引っかかるために買う人も中にはいるでしょう。
あるいは、転売目的の不正契約をしてなんとか確定申告の要らないよう収益が20万円以下になるように上手く調整しながら副収入を得ている人もいるでしょう。

端末をどう使うかはその人の自由です。
端末があれば当然ガジェットもお持ちでしょう。
外部バッテリーやケーブル類など。

それらのガジェット、間違って買ってしまった。でも開封済みで返品できない。
あるいは相性大丈夫かしら？と思ってなかなかBuyなうできない。
そんな経験は一度や二度はあるでしょう。

そこでこのイベントでは、それらのガジェットを持ち寄り、
物々交換や売却、あるいは他のユーザのガジェットを拝借して適合するかどうかチェックするという実に画期的な交流会です。

仕事や学校で忙しいという方も考慮し、
平日の部、休日の部をそれぞれ設け、
関東一都六県と近畿二府四県でそれぞれ開催します。

今回は関東平日の部を開催しますが、
他の日程も決まり次第ATNDします。

参加資格

  • 土日は都合がつかないが、平日なら都合がつき参加見込みが立つ方
  • 間違って買ってしまったガジェットがあるが開封済みで返品できない。捨てるのもったいないので有効活用したいと思っている方
  • 相性大丈夫かしら？と思ってなかなかBuyなうできない。一度試してみたい方

お持ちいただくもの

  • ありったけのガジェット
  • ありったけのデバイス

参加費

Priceless（決して高すぎないようにします）

参考資料（関東支部主催者Spaghetticodeなうがお持ちする予定のガジェット）

  • Z3X Sumsung Editon（サムスン製端末のSIMロックを不法解除するツールです）
  • microUSB-USBホストケーブル（ハイブリ用に買ったもののプランGのSIMだけしか活用してないため眠っています）
