==
Vungleについて：
2011年の設立より、アプリ内HD動画広告プラットフォームをグローバルでの提供を行っており、現在では米国を拠点に７カ国に拡大。Vungleのクリエイティブ及びKPI最適化テクノロジーを活用し、多くの広告主様の優良なユーザー獲得に尽力している。同時にマネタイズ方法としても活用されており、Vungleから配信される動画広告から高い収益を生み出すことに成功している。

BitMangoについて
2011年7月に韓国で設立したゲームアプリデベロッパー。現在配信中のゲームは「Block! Hexa Puzzle」 や 「Roll the Ball」をはじめ20以上に上り、日本を含め欧米圏や東南アジアでの世界展開に成功。総インストール数は2017年の第1四半期で6,200万を超えている。
