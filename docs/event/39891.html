a-blog cms Summer Camp 2013

イベント ユーザー
event_id 39891
title a-blog cms Summer Camp 2013
catch 今回、7回目の合宿です。前回はベースキャンプ名古屋でしたが、やっぱり海の見えるところで！
description

初心者の方についてはアップルップルからの初心者向けのセッションを用意しますし、仕事を持って参加頂ければ悩んでいる部分についてを解決する事ができると思います。

上級者のためには、 アップルップル の実制作の案件をベースにCMSで実装する際のサイト設計についてを学んで頂く予定にしております。

参加される方の10分程度のTips等の発表も募集しております。

合宿のいいところは、他の人が作ったサイトの普段見る事の無い管理側をどう作っているのかを知る事ができる数少ない機会です。是非、クライアント様の紹介許可を得て参加下さい。

遠方から愛知入りされるの参加者（飛行機・新幹線）中から抽選で1名様に a-blog cms のベースライセンスをプレゼントさせて頂きます。

費用

  • 1泊夕朝食付　15,000円　通常
  • 日帰り夕食付　8,000円　自分で帰る方法をご用意下さい

※領収書を発行致します。
※キャンセルは4日前(7月15日)までであれば無料です。

会場

  • 知多半島最南端 師崎荘

集合

今までは名古屋駅集合でしたが、人数も多く車の手配も大変だったり、翌日寝不足での運転も危険という事もあり 中部国際空港（セントレア)にバスで迎えに来て頂く事になりました。

名鉄 中部国際空港駅 改札出口　12:00 集合 / 12:15 出発

1日目のお昼は、潮蔵というお店に行くのが定番でしたが、セントレアには多くの名古屋っぽい食べ物が多くありますので、予算にあわせて食事も可能です。駅から空港は繋がっておりますが、飲食できるお店までは徒歩5分くらいはかかると思います。
