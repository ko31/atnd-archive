「 映画＆本で旅するサルデーニャ」フード＆1ドリンク付レクチャー
2016年4月23日（土）17:30-19:30

イタリア通でさえ訪れる事が少ない神秘の島、サルデーニャ島。悠久の歴史、独自の文化と伝統が色濃く残る秘境とも言われています。SAAではプレゼンター独自の視点で、現地を紹介するレクチャーを開催します。オリジナルサルデーニャ料理と合わせてお楽しみ下さい！

FM802「Ciao! MUSICA」での映画紹介やコラム執筆にも定評のある野村雅夫さんと、サルデーニャ島に馴染みのある二宮大輔さんをプレゼンターに迎え、4月23日の「本の日」にちなんで、サルデーニャ島を舞台にした映画や本、マンガ、関連作を紹介します。映画や本を通して、イタリア半島とは異なる島の歴史や文化背景にも迫ります。

プレゼンター 野村雅夫（FM802DJ、京都ドーナッツクラブ） 二宮大輔（京都ドーナッツクラブ）
京都ドーナッツクラブ／2005年発足。「イタリアの文化的お宝発見」をキーワードに、イタリア映画の配給、字幕翻訳の他、書籍出版、日伊通訳など行っている。また、イベントスペース「チルコロ京都」の企画・運営もおこなっており、ライブ・トークショー・映画上映会・演劇など、そこではイタリアのものに限らず多様なイベントを毎月開催している。
http://www.doughnutsclub.com/
http://circolokyoto.jimdo.com/

野村雅夫／1978年、イタリア、トリノ生まれ、滋賀育ち。イタリア語は大阪外国語大学で学び、映画理論を専攻。大阪大学大学院博士後期課程単位取得退学。言葉と映像の関係性について、パゾリーニの映画論をベースに研究。アゴスティの訳書に、『誰もが幸せになる 1日3時間しか働かない国』（マガジンハウス）、『罪のスガタ』（シーライトパブリッシング）がある。他の訳書として、ラジオドラマ風の電子絵本『剣を持ったクマ』がある。ラジオDJとして活躍中。
担当番組：FM802 『Ciao! MUSICA』毎週金曜日 12：00-18：00（大阪）
InterFM 『Mondo Musica』毎週月曜日～木曜日　18:00-20:00（東京）
読売テレビ 『音楽ノチカラ』MC（大阪）

二宮大輔／ローマ第三大学で勉強するかたわら、翻訳などに従事する。専攻は現代イタリアの言語学と文学。最近の興味はモーロ誘拐事件をはじめとする1970年代の社会史。

About SARDEGNA サルデーニャ島とは

イタリア特別自治州の１つ。仏領コルシカ島の南、シチリア島の北西に位置する地中海で2番目に大きな島。世界最古のメソポタミア文明やスペイン・カタルーニャ、アフリカの影響も色濃く、内陸部には特に石の砦「ヌラーゲ」が多数存在する。イタリア語の方言とは異なるサルデーニャ語を話す人々も多数。紀元前の頃から、神の最初の足跡の島という伝説があり、「サンダル」の語源にもなったとも言われている。村毎に異なる特有の文化や食、民族衣装があり、年に大小2000ほど開催するお祭りは、民族衣装を一同に見られる絶好の機会。ヨーロッパ中のセレブ達に人気の高級リゾート地「コスタ・ズメラルダ（エメラルド海岸）」が有名。長寿の島とも言われる島の料理は、ユネスコの無形文化遺産「地中海の食文化」の一部も形成している。本フライヤーの表紙はサルデーニャ島の州旗がモチーフ。
