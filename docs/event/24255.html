毎年恒例になりました静岡キャンプですが、今年は場所を熱海に設定しました。

勉強会のテーマ/内容が確定・訂正されました。2月8日更新

テーマと内容

今回のキャンプでの勉強会のテーマは、参加有志による Solaris 11 や OpenIndiana ユーザ向けのパッケージレポジトリサーバの立ち上げとレポジトリに載せるパッケージを作成することにあります。

内容は、レポジトリサーバとパッケージ構築のためのハッカソンとなります。(温泉でリフレッシュしながらの２日間　強行突貫工事となりそうです）

開発経験がなく、build は難しいという方も参加できます。構築したリポジトリからのインストール/確認してくださる担当も募集しています。

キャンプでレポジトリ立ち上げプロジェクトの初期作業を行いますが、キャンプ以降もユーザ会の有志で開発と運営を継続していきます!!

キャンプ参加のBenefit(利点/成果)

  • OpenSolaris ユーザによるユーザのための世界初?のレポジトリサーバの立ち上げに貢献できます。
  • Solaris 用のレポジトリサーバ、IPS パッケージ作成のノウハウを習得することができます。
  • 自作の OSS (OpenSource Software) をリポジトリに載せることができます。
  • Solaris 11 のパッケージ buildでお悩みの方、経験者/エキスパートからでアドバイスを得られます。
  • Solairs 開発未経験者、初心者の参加も歓迎します。キャンプの間、経験や知識が豊富な他の参加者が Solaris の「いろは」から親切にアドバイス致します。
  • 自分がメンテナーになりたいソフトウェアを持ち込めば IPS パッケージに build してもらえます。

参加者が用意する物

必須

  • SSH できる端末 (Windows, Mac, Linux, Solaris なんでもOK)
    ネットワーク (有線/Ether, 無線/WiFi どちらか必須)
    　　各自にハッカソンの時に利用する「開発用Zone」を割り当てます。
    Solaris/OI環境を用意する必要はありません。

推奨

  • Solaris 11 or OpenIndiana 151a がインストールされた PC/Note

事前準備

  • 予習/復習
    瀧さんの１月の勉強会の資料
  • Build に参加する方は、
    SourceForge.JP
    でアカウントを作成して、jposug プロジェクトに参加申請をしてください。
  • 秘密鍵を登録しておくことで、 commit ができようになります
  • 自分がメンテナーになりたいと思うソフトウェアを tar ball, gz で用意
  • RedHat でパッケージ作成の経験がある方は、RPM の spec ファイルもあるといいそうです。

尚、現地のインターネット/ネットワークの状況によりますが、インターネットからのリモート参加も計画しております。twitter ハッシュタグ #osoljp で当日の状況（可能かどうか）を確認してください。リモートで参加される方も上記の事前準備をお願い致します。
