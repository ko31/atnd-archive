イベント概要

私たちマーケットエンタープライズは「Webマーケティング」「IT」「オペレーション」の３つを高いレベルで融合させた「ネット型リユース事業」を展開し、リユース業界でオンリーワンビジネスを展開するベンチャー企業です。

2006年の設立以来、9期連続で増収増益を達成しており、また、事業拡大に伴い2012年に大阪、2013年に名古屋、横浜、2014年に福岡、2015年には埼玉、神戸、そして2016年には仙台と物流の拠点を次々と展開しました。

3年間で自社システムをすべてAWS化し、たった2名でPythonを使ってゼロから書き起こした当社CTOがあまり外部に公開することのない自社のシステムについてお話します。

・9拠点の物流センターで動く当社のシステムとはどのようなものなのか

・サーバレスアーキテクチャの先端である AWS Lambda はどこに用いられているのか

・どうすれば変化に強いシステムが構築できるのか

今回はそのあたりをお話します。

フリードリンクと軽食をご用意してお待ちしておりますので、ぜひお気軽にご参加ください！

※時間の都合がつきましたら、当社CEOの小林をはじめ、他の役員も参加しご挨拶をさせていただきます！
