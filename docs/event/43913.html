今回はコーディングの会です

先日 Symfony Live London 2013 が開催されました。興味深いセッションがたくさんあり、どれもネタにしたいのですが、その中でライブコーディングのセッションがありましたので、今回の Meetup ではそのライブコーディングを見て真似するということをします。

  • 大まかな内容
    • ドメインモデルの実装コードを、ドメインの言葉を表現するようにリファクタリングしていく
  • 真似する対象
    • Extract Till You Drop
    • スクリーンキャストで使われているコード
    • スクリーンキャストでは PHPStorm が使われていますが、Meetup ではIDEやエディタは特に指定しません。
  • 当日はスクリーンキャストを見ながら手を動かしますので、一緒に手を動かしたい方はプロジェクトやテスト（PHPUnit）が動作する環境までは事前に準備しておいてください。
    • 追って詳細追記・ご案内します
    • 不明点は #symfony_ja ハッシュタグや Symfonyユーザー会ML 等でご質問ください。

Engine Yardへの道案内

参加条件

  • スクリーンキャストを見てマネをする会なので、普段Symfonyを使っていない方でも参加可能です
  • 見ているだけという方の参加も歓迎します。
  • 当日はSymfonyセットアップ、IDEのセットアップのサポートは行いません。
  • 時間帯的に晩御飯の時間なので、何か食べ物を持ち寄って頂く形になります（運営側では用意しません）

事前準備

  • プロジェクトソースコード をclone
  • cloneしたディレクトリで composer install
    • これで、プロジェクトローカルに PHPUnit が入ります
  • IDE等で準備
    • プロジェクトの PHPUnit を実行できるように
      • PHPStorm なら、vendor/phpunit/phpunit/PHPUnit/Autoload.php をロードするように設定
  • キー操作
    • 以下のような操作を行えるようにキーバインドを調べておくと良いと思います
      • 補完による入力
      • リファクタリング rename（変数名、メソッド名変更）
      • リファクタリング move （クラスの位置を移動）
      • クラスに対するテストの作成
      • プロダクションコードから、テストへ移動
      • 定義へ移動
      • 最近使ったファイル一覧から、ファイル選択
  • コードカバレッジ等の動きも試したい場合はxdebugも必須です
