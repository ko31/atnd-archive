description

今年はクラウドコンピューティングの真の元年としてずいぶんと話題の多い年でした。
様々なクラウドプラットホームの登場、という事で主として技術主体インフラ事業が動き出す中、日本の市場ニーズに合った、ジャパンクラウドが形成されつつある、という印象を受けています。
しかしながら、来年、２０１１年は、さらに躍動の年になる、と読んでいます。
一つは、いよいよビジネス主体のクラウド市場、特にAWS等のUSクラウドが本格上陸し、クラウド市場がさらに広がりを見せると期待されています。プライベートクラウドを構築するツールも充実し、ハイブリッドクラウドを実際に導入する会社が急増するもの、と考えられます。
一方では、このようなマルチプラットホームの市場の中、標準化は遅々として進まない、というのも現実として見る必要があります。
それだけ市場のスピードが早くなっている、という事です。

今回の勉強会のテーマは、来年に向けてだんだんと見えてくるクラウドの「エコシステム」そしてその中でどのようにクラウドソリューションを導入すべきか、提案をしながら、ご意見を伺う場として盛り上げることが出来れば、と思います。

例のごとく、懇親会も企画しますのでこぞってご参加をお願いします！
