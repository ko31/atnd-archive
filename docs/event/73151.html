description

表題の件でお知らせです。

幼児園の年長の一部の保護者さまからインタビューをさせて頂いておりますが、その際に小学校に上がった際にどういう生活になるのかイメージが涌かない、という声をよく聞きます。

そこで、小学校一年生になった時にどういう生活になるのか、現在小学生一年生のお子さまを持つ保護者さまから、お話をして頂こうと思います。

是非この機会に幼稚園や保育園の年長の保護者さまに参加頂き、小学校生活のイメージを掴んで頂ければと思います。

開催概要は下記の様になります。

【参加対象】
現在幼稚園・保育園年長のお子さまを持つ保護者さま

【日程】
12/19(土)　10:00～11:00

【場所】
バディスポーツ幼児園長津田校　http://www.y-buddy.jp/kinder-garden/nagatuda
横浜市緑区長津田みなみ台4-3-1

【当日の内容】
長津田第二小一年生のお子さまを持つ保護者さまを囲んでの座談会形式で、小学校一年生の生活についてレクチャーをします。年長の保護者さまからの質問時間も設けます。

【参加費】
無料

【持ち物】
・筆記用具
・飲み物
・お菓子(みんなで食べられるようなもの)

【申込み・お問合せ先】
バディスポーツ幼児園都筑校（担当：小池）
