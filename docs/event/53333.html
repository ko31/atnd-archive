オンラインゲームの開発を一気に加速させる Photon

Unity と Photon を使って 「リアルタイム」×「マルチプレイ」 を短時間で実装するためのデモンストレーションをご覧に入れますので、是非PCを持ち込んでその場でオンラインゲームを作っちゃってください！

　
　　　　　　　 【特別編】ワークショップ内容

< 導入 > 複数のキャラクターを同一ルーム内でリアルタイムに同期させる
< 応用 > キャラクター同士が弾を撃ち、衝突判定を同期させる

　

◆実装に必要なUnityの操作やスクリプトの解説を中心に行います。
◆参加者のスキルに応じて、内容は部分的に変更します。
　※< 導入 >となるキャラクター同期は必ず実施します。

　　　

■持ち物　 ： Unity ( version 4.2 以上 )インストール済みのノートPC ( Windows / Mac )

　　　　　 ： お名刺　1枚

■参加条件： Unity の基本操作ができる

■入館案内： 大阪・梅田・ブリーゼタワー 23F　GMOクラウドWEST株式会社　会議室

　　　　　　　　ブリーゼタワーへの　 アクセス

会場までの道のり

①ブリゼータワーのオフィスエントランスで「GMOクラウドWEST」の
　「Photonワークショップ参加者」とお申し付けください。

②エントランス通過後、ガラス戸の奥にあるエレベーターを使って23階までお越し下さい。

③エレベータを下りて、左側奥にお進みください。

④GMOクラウドWESTのインターホンを取って、ご来場の旨お伝えください。
