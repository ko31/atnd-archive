description

“人生はバランス”です。

この講座に関心を持たれたあなたは
すでに様々な行動に出て、
ある程度の結果を出されているにもかかわらず、
さらにレベルUPしたいと考えていらっしゃる方々だと思います。

素晴らしい思考と行動力です。

世の中には、いわゆる「成功者」と言われる方がいます。
どの分野においても、
尊敬に値する方はおります。
ただ、そうした「成功者」と言われる方を分析すると、
以下の【７つの条件】のバランスが、
全て整っているライフスタイルを送れている方は
本当に少ないのです。
【７つの条件】とは・・・

①心の充実
②体の健康
③経済力
④時間のゆとり
⑤家族・仲間との絆・信頼
⑥人格力UP（人格教育・道徳教育）
⑦スピード成幸

「パーフェクトライフバランス（PLB）実践学」とは・・・
これらの【７つの条件】をバランスよく獲得することによって、
人生のバランスをよく保つために確立した実践学です。

以下のいずれかに当てはまる方は今すぐ参加されて下さい。

①お金はある　But 時間がない
②時間はある　But お金がない
③時間もお金もある　But 出逢いが少ない（人脈が少ない）
④時間もお金もある　But 心から信頼出来る仲間が少ない（またはいない）
⑤時間もお金もある　But　不健康な生活になっている
⑥そこそこ満足している　But　もっとスピードUP出来ないのか？
⑦そこそこ成功出来た　But　後に続く成功者を育成出来ない（少ない）
⑧そこそこ成功出来た　But　もっと価値ある人生を模索している（社会貢献度を上げたい）

これからの世代のリーダーとして
社会、日本、世界で活躍したいと思う方々にお勧めしたい内容です。

※時間もお金も人脈もないという方は、
　この講座の成果は得にくいため、
　参加はご遠慮頂いております。

“人生はバランス”
バランスの取れた、素晴らしい人生を獲得致しましょう。

=========
現在、19事業を展開し、グループ年商3億円突破。
17年前、約１億の負債があるところから起業。
お金・人脈・時間すべてがない中からスタート。
事業立ち上げ当時、過酷な業務により入院。
「自分がいつどうなっても、会社が回る状況を作っておかなければ結局のところ安定などしない！」と強烈に感じ、“ひとり起業家”としては日本初の「＠Megaフランチャイズ事業」を体系化。

現在、加盟店約400社の教育に独自の「VRT」を開発・導入。
7年半前より『親孝行プロジェクト事業』として、
「人生史映像制作」及び「写真整理サービス」事業開始。
現在、信頼する仲間と共に
「次世代教育環境建設プロジェクト」を推進。
2011年10月より農地開拓・新教育によるフリースクール開校。
2012年7月より、
「MHD 男塾」開塾。
同年9月より、「MHD女塾」開塾。
同年12月より札幌校開塾。
2013年10月より「MHDビジネススクール」開校

「事業収益は次世代教育のために」を企業理念に掲げ、
人格教育・食育・起業家志援等、
未来に向けて必要な教育全般に携わる。
2002年8月より開始した「誰も教えてくれない生き方講座」は、
開催回数430回を超え（2014.6現在）、
のべ1000名超が参加され、
“未来の自分らしい生き方の気付き”を与え続けている
