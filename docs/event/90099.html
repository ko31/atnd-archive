2017年8月19日、お楽しるび会　改 お楽しるび祭開催！

概要

お祭りじゃ！そこでゲットしたポイントを集めて最後にくじだ！

はい、コンセプトは前回と基本的に同じ、しかし今回はゲーミング倉庫と会場が広くなったため規模を拡大！

会場写真

今回はFIFA17のコミュニティである東京FIFA会とイベントを同日開催。
他のコミュニティも巻き込んでの夏祭り！

ビデオゲームやPCゲームに限定せず、ボードゲームやダーツなど色々なゲームでポイントを競いおう！

前回と同じくポイント上位者から最後にプレゼントくじを引く形式。
細かいルールや開催タイトルは後日詳細発表。

プレゼント協賛、提供募集

参加者全員分に何かしら行き渡るような形でプレゼントを用意。
前回のプレゼント（一部）

プレゼントを提供してくれる方、企業募集中です。

詳細は stoicfps@yahoo.co.jp までメール下さい。

その他質問などありましたらDiscordでも対応します。

お楽しるび会Discord : https://discord.gg/HndTYJr

日程

8月19日（土曜） 12:00 ～ 20:00 （現在調整中）

会場

会場へは最寄り駅から車で運搬予定です。

Cyac千葉倉庫
最寄り駅 : 稲毛海岸

参加費と募集人数と参加登録方法

最大50名

高校生 : 1000円
大学生 : 2000円
社会人 : 3000円
中学生 : 無料　※
中学生は保護者同伴なら可能。
中学生は18:00以降の参加は禁止とします。

参加費はプレゼント代、お菓子、軽食、機材調達、買い出しなどに使用します。
これぐらいの設定にしないと全員配布プレゼントはもとより赤字になってしまうのでご協力ください。

参加登録方法

人数を早くしたいためお早い登録をお願いします。

飲酒喫煙飲食について

喫煙は指定された場所でお願いします。
飲酒は禁止。

飲食は持ち込み歓迎。

参加者はおやつや皆で食べられる軽食持参推奨。
ただしナマモノや開封済みのものは禁止（魚や果物など加工が必要なものもNG）
開封済みの缶詰やお菓子持ってこられても処理に困るためご遠慮ください。

長時間のイベントのため、当日買い出しか途中で食事休憩などの時間を作ろうと考えております。

規約、禁止事項

お楽しるび会

お楽しるび会は皆で集まってワイワイゆるく楽しむイベントです。
皆が楽しめるように下記を必ず読んで頂き、イベントの運営にご協力をお願い致します。

禁止事項

本イベントの運営を妨害する行為、また主催者が不適切と認める行為。
犯罪行為、または犯罪に結びつく行為。
公序良俗に違反する行為。
火薬・花火・刃物・酒類を持ち込む行為。
主催者や参加者の著作権・財産権・名誉・プライバシー・その他の権利を侵害する行為。
第三者になりすまして本イベントに参加する行為。

参加者

酒気帯び状態での参加はできません。
未成年者の参加は保護者の同意が必要です。また中学生以下の方は保護者の同伴が必要です。
主催者の意図とは異なる場合、本イベントの開催に支障を来す恐れのある場合は入場のお断り、もしくは退場いただきます。
近隣への迷惑になるような行為はおやめください。
貴重品や個人所有物は各自で管理してください。盗難・紛失・事故等に関して主催者は責任を負いません。

運営

主催者の判断によって入場の際に持ち込み禁止物確認の為、手荷物のチェックをさせていただく場合があります。
会場内で主催者の指示及び注意事項に従わずに生じた事故や損害等については、主催者は責任を負いません。
本イベントが中止、延期となった場合の交通費や宿泊費等の補償は致しません。
会場を破損・汚損された場合は実費にて請求する場合があります。

撮影

主催者により本イベントは撮影・録音・録画等される場合があります。その場合、参加者が写り込む場合があります。
写り込みを拒否される方は本イベントが開催される前に主催者に申し出てください。

本イベントでの撮影・録音・録画等の記録行為を行う場合は主催者に許可をお取りください。
撮影・録音・録画等を行い、動画配信・アップロード等を行う場合は主催者・参加者の許可をお取りください。
