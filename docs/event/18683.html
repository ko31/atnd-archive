本イベントは主に東京西部、神奈川西部のエンジニアの交流を目的としたものです。
普段エンジニア交流会は都心部、平日に多くおこなわれており、それらの補完を目指しています。

pmとついている様に、Perlユーザーが多くいますが、話題はPerlに限定されているわけではありませんし、私を始め、PHPやJS、HTML等をPerlよりも使っている人多くいます。
エンジニアであれば、お気軽に参加いただければと思います。

特に若い人、お気軽に参加ください。

—
会場は過去にも開催した沖縄料理屋のあじこ屋になりました！
http://www.hotpepper.jp/strJ000025044/
予約は石田です！

ご予算は割り勘制で、通常３〜４０００円くらいを想定しております。

—
hachioji.pm #1、#2、#3、#4、#5、#6、Takao.mtに引き続き、1枚LTを行いたいと思います。
前回同様ですが、「必須ではありませんが」楽しいので是非作製してみてはいかがでしょうか。
基本的に、席の端っこから端っこの人まで話そうぜ！というコミュニケーションを目的とした物です。

ガイドラインは
・発表する事は一ページ（A4など）にまとめること
・発表時間は１～２分（銅鑼とかないのでｗこだわりませんが）
・自分の名前（ハンドルでもいい）を載せること（コミュニケーションが目的なので）
・質疑応答をやればいいので、書ききらなくてもOK
・ネタ発表歓迎します
・このガイドラインはあくまでガイドラインなので、破ったっていいです。
・目的は皆に発表してもらうこと、コミュニケーションを推進する事です。

今回のテーマはnorry_gogoさんの案で、「夏休みの自由研究」と題して、GWの時の続きネタでもいいし、夏休みに使ってみたものみたいもの、作ってみたものみたいもの、いいと思ったものを発表ください！！

資料は最終的にPDF、またはHTML（どこかにアップしてください）にして頂ければ幸いです。
当日印刷してもってきていただいてもOKです！（印刷枚数は参加人数分ではなくてもOKですが、１枚だと辛いですね）

ウェブページや動画などをデモるときは、PC等をお貸しいたします。（会場によりますが、多分）
