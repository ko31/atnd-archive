今回、yurayuraが輝石（パワーストーン）、スピリチュアル、神社が好きな皆さんとの交流イベント主催させていただきます✨

輝石が好きな人やスピリチュアルな話が好きな人と話をしたい❗️❗️

輝石（スピリチュアル、パワーストーンなど）の疑問がたくさんある！輝石を持つことに心配な点がある❗️

などなどの経験はありませんか？

今回は、そういった参加者の皆さんが集まっていただけることで、より輝石を好きになってもらえる。

気持ちを新しく、曇りのないこころで
この日が皆さんにとっての新たな世界感の出発点になるればとの思いでご案内しています。
