じゃばすく製作所（仮）ふたつめ。

　運営自体もJavaScriptを使ってはいますが素人です。みんなで一緒に学びを楽しんで行きましょう。よろしくお願いします！

会場

　長崎市立図書館研修室3、4。

　会場の場所が少しわかりにくいです。 長崎市立図書館 Nagasaki City Library 図書館について に図がありますが、正面入り口から入場後、1階図書室へ入らずそのまま右へ折れ多目的ホール脇の階段を登ってください。二階の左奥に会場の研修室があります。

　館内飲食禁止との事で食べ物の持ち込みはできませんが、水分補給のためのペットボトル入り飲料の持ち込みは可能です。でも、一応こっそりと飲んでください……。

備考

　時計アプリはWebブラウザ上で動作する簡易的なアナログ時計です。ちょっとだけHTML5のCanvas要素を使います。JavaScriptへの理解も然程必要ありません。とりあえず作ってみるを目指します。

　ノートPCなどをお持ちの方は是非持参ください。無線LANクライアントがあるとインターネット接続が可能です。
　JavaScript実行環境としてのブラウザはWebKit系（SafariやGoogle Chrome）とFirefoxを対象としています。Internet Explorerの新しめのバージョンだと動くかも知れませんが確認していません。古めのバージョンだと動かない事は確認しております。
　各々お好みのテキストエディタもお持ちください。

　見てるだけ、他の人との共有での参加ももちろん可能です。雰囲気だけ味わうのもいいかと思いますよ。
　先着一名さままで私とマンツーマン的に嫌がらせを受けながらも可能かも知れないです。

　インターネット接続回線としてWiMAX回線を用意していますが、ご自分でどうにかできる方にはそうしていただけると助かります。一回線しか用意できないので、あまりに多くなると接続できなかったり極端に遅くなる可能性があります。

　電源タップをお持ちの方も持参していただけると嬉しいです。床埋め込み式のコンセントが複数接地されていますが、アダプタなどは挿し難いです。こちらでも何本か準備しておりますが、何とぞご協力ください。

　終了後、この為のというわけではないですが、懇親会を開きたいと思っています。
　参加希望の方はコメントなどでお知らせください。人数により予約を入れようかと思いますが、できるだけ気楽で安く上がるお店をと考えています。この会と直接関係がないので懇親会だけ参加でもOKです。
　ただし、酒乱の気のある方はご遠慮ください :-)

　懇親会の会場を決定しました！
　やっぱりココしかないでしょ。という事で、今回も居酒屋「誠」さんにお世話になろうと思います。住所は 長崎市興善町７−２ です。費用は3000円程度を予定しています。当日割り勘でお願いします……。
　時間は午後5時過ぎくらいからを予定しています。懇親会のみ直接お越しくださっても結構ですよ。唐揚げが嫉妬するくらい大人気です！

　ご質問などありましたらお気軽にお尋ねください。
