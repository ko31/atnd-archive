description

2014年、国内で本格的に展開されはじめた「ベアメタルクラウド」というインフラ形態は、様々なメディア・勉強会等でも注目されるキーワードのひとつとなりました。
既に導入しているユーザー企業の現場担当者が率直な意見を語ってくれた最新事例と、最新版の管理コントロールパネルで見せるデモンストレーションを通して、ITpro EXPO AWARD 2014 特別賞にも選出された注目のサービスを知るイベントを開催します。

特に導入事例は初公開となるものも！

ベアメタルがまだピンとこない方も、具体的な事例を知りたい方にも、本イベントを通じて2015年ますます注目・研究されているベアメタルクラウドを体験ください。
