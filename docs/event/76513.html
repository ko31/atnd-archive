これまでのあらすじ

去る5月に行われた青空文庫アイデアソン
「Code for 青空文庫」アイデアソン #1」 https://atnd.org/events/66230
にて、

エンジニア向け啓蒙イベントや工作員向け入力イベントを行おうという提案がありました。
「イベント開催計画」

それを受けて、「シャッカソン」（写経：入力イベント）を開催します。今回は第4回目であり、はじめての平日開催となります。

入力だけではなく、今後青空文庫に関わっていきたいエンジニアなど興味ある方と今後のお話もできればと思います。

開催概要

  • 日時： 2015/03/21(月・振替休日) 19:00-22:00
  • 会場：サイボウズ株式会社 東京オフィス 東京都中央区日本橋2-7-1 東京日本橋タワー 27F（受付：7F）
  • 主催：青空文学部
    • 「青空文学部」とは、青空文庫のイベントなどを自主的に行うためのカンムリです。
  • 費用：無料
  • 内容：ハック発表：青空文庫をハックした内容を発表しよう！
    シャッカソン：「底本」を決めて、当日分担して入力、校正を行います。
    当日に作業が完了しなかった場合は、持ち帰っての作業となります。
  • 途中まで、途中からの参加もＷｅｌｃｏｍｅです！

底本

現在、選定中です。

  • ご希望、ご提案などがあれば、コメントなどでお教えください。
  • 当日、申請手続きなどができるかどうか、青空文庫の仲の人と調整中です。
    • できるようであれば、の流れも見せる目的で、「入力受付システム」 http://reception.aozora.gr.jp/accept.php
      から「第3回シャッカソンメンバー」という名義で底本登録します。
    • 当日申請手続きが難しそうであれば、「入力／校正」に重きを置いた内容で進めます。

プログラム

19:00-19:30 オープニング
19:30-20:00 ハック発表！

  • ライトニングトーク形式でミニ発表を募集中！

20:00-20:30 工作員マニュアルの説明、底本登録、チームメイキング
20:30-21:00 入力・校正作業 #1
21:00-21:30 入力・校正作業 #2
21:30-22:00 まとめ、今後について、クロージング

ご参加される方へ

  • ハック発表希望される方は、概要をコメントにご記入ください
  • 一部の時間のみ参加される方も、コメントに。
  • 「青空文庫作業マニュアル」をあらかじめ目を通しておいてください。
    http://www.aozora.gr.jp/aozora-manual/
  • 当日はPCをお持ち下さい
