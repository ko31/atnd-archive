description

都会の喧騒を離れて、クリエイターやミュージシャンなど感度の高い人達が集まる代々木上原に、今年「きんはる」が誕生しました。このお店では、オーガニックな根菜類を中心にしたヌーベル・キュイジーヌ（新解釈）の素晴らしいお料理がいただけます。

SUPER ILLUSTRATION SCHOOLでは、このお店を貸切り、2012年に大きな潮流となりそうな若いイラストレーターをフィーチャーした「イラストレーションの前夜祭」と題して、クリスマスパーティを開催します。
「きんはる」の美しく美味しいいお料理やとっておきのワインや日本酒の他に、2012年に気になるイラストレーターやSUPER ILLUSTRATION SCHOOLのOGOBや受講中のイラストレーターのポートフォリオやポストカードを並べます。

参加作家はSUPER ILLUSTRATION SCHOOLのSupervisor 佐藤（元illustration illustration FILE AD　現illustration FILE web CD）がキュレーションします。
現場ではライブペインティングやillustrationFILE編集長片桐氏他のインタビューなども開催予定です。
特に参加資格は問いませんので、興味のある方は早めにお申し込みください。

2011年12月23日金曜日（天皇誕生日）
19:00〜22:00　22:30以降はキャッシュオンデリバリーのBAR Time
