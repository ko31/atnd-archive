ところで、LTってなによ？

ライトニングトーク（英: Lightning Talk）とはカンファレンスやフォーラムなどで行われる短いプレゼンテーションのこと。様々な形式があるが、持ち時間が5分という制約が広く共有されている。(Wikipediaより)

第一回は大盛況で幕を閉じたdendai sie LT。
今回は時間を「基本5分」と変更して開催したいと思います。
もちろん聞き専参加も大歓迎！
Ustream配信もする予定ですが、学内の人はぜひ直接聞きに来てください！

是非登壇して話したい！って人は こちらに記入してください

今回もテーマは自由。
卓球のコツから彼女自慢、エディタの宣伝、アルゴリズムについてなどなんでも大歓迎！
