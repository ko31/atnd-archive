description

★自己分析が大切なのは理解できたけど・・・
・何をすればいいの？
・テクニック本を読んだけど・・・よく分からない？？
・本を読んでやる気がしない・・・
・何が本当の自己分析？
不安になりますね！

★書面では伝わらない部分を直接お伝えします。
・具体的にどのようにすればよいのか書面では分かりません。
・それは、情報がワンウエイだからです。
・人によって不明確な部分が違います。
・また、実際にやっても合っているのか・・・確かめようがありません。
・不明確な部分を質問できません。

★過去の棚卸しを行い、自己分析を行います。
①過去の成功、失敗体験から「繰り返し行われた経験」
　⇒成功に結びついた「強み・弱み」を抽出
②他者からの意見⇒自分の知らない「長所・短所」を抽出
③性格検査⇒採用担当者の使用する適性検査から自己を探る
④キャリアアンカー診断、適職診断から「思考・行動」を探る
