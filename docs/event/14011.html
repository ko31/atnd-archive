description

青山ソウルオブソウルが送るチャリティーイベント！　４月２日（土）に、8回目を迎える、『こい☆こい』は、イベント収益金を東北地方太平洋沖地震の義援金として被災者の皆様への支援に活用されるよう収益の中から日本赤十字社を通じ寄付いたします。

今回８回目を迎える、エキセントリックパーティー『こい☆こい』はパフォーマンスアート＆チャリティーイベントといたしまして、今話題の３人の女性アーティストと革命的なネオ・フュージョン！！　

まず、ライヴペイントに、ペインティングアーティスト／踊絵師の神田サオリ氏、そして、サウンドセッションには音楽クリエーターの宇田川優氏を迎え、そこに融合するのは、あらゆる空間を植物素材で対応する、いけばな作家の横井紅炎氏。そこに、最強のコラボレーションとして、河鍋暁斎記念美術館と！、今度のこい☆こいは、クリエイターの紡ぎ出すその迫力の前にただ言葉を失います・・・

もちろん、今回もBlack Eyes TATTO による、本彫り Live TATTO に加え、実力派プロDJにYONOSUKE氏
フュージョンダンサー AYAKO ＆ MANA のユニットによる、花、月、音をテーマにしたコンテンポラリーダンス！

サポート陣は、ZIMA、Red Bull、による２本柱を筆頭に、アートペイントには欠かす事の出来ない老舗メーカー、ターナー色彩株式会社のショーサポートが実現！！そして、今回目を見張るのが、ナイトイベントでは、初となる河鍋暁斎記念美術館のバックアップ！

今回、表紙を飾るのは天才絵師・河鍋暁斎の傑作美人画『文読む美人』×『こい☆こい』の時代を超えた究極のコラボレート！！

新境地を迎えた " こい☆こい " ワールドを、是非とも体験してください。

今回もオールナイト！朝までやってます！
