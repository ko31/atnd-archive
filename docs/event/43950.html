RFC Viewer開発を通して学ぶ!! iOS開発のパターン化!!

ヒカ☆ラボ一覧は こちら！！

★講演内容

■講演内容
☆初心者大歓迎☆
RFC Viewer開発を通して学ぶ!! iOS開発のパターン化!!
～iOS開発がもっとよく分かる!!今こそスキルアップのチャンスです!!～

本イベントでは、初級者を対象にiOSアプリケーション開発に必要な基礎知識を学んでいただきます。
シンプルな通信機能を使ったドリルダウン形式UIのアプリを題材に、設計や画面構成から実装まで、
一連の流れを俯瞰しながら各技術を確認していきます。
イベントは、講師が解説するプロジェクターの画面を使って進行します。
本イベントの受講者へは、当日までに予習が出来るよう、サンプルコードを事前にご案内いたします。

■サンプルコード
https://github.com/murakami/RFCViewer
