創業者のためのビジネスプラン作成のポイントについて、税理士法人広瀬 上村勝
規 様と京都信用金庫・企業成長推進部 様に解説していただきます。今回は特に
資金ニーズが旺盛な創業期における金融機関からの新規創業融資について、着実
に融資実行を得るために創業者が整備すべき、新規創業計画の策定手順について
理解いただくことを目的としています。
