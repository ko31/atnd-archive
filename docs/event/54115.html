title 今錦JAMBOREE
catch 【日本酒試飲会】渋谷区にて、長野県の日本酒「今錦」を蔵人と共に味わえる１日限定イベントが開催されます！
description

８月１日（金）に長野県上伊那郡中川村にある米澤酒造株式会社の作る日本酒「今錦」を蔵人と一緒に堪能できる試飲イベントを開催致します。

場所は渋谷区神宮前にあるニューロカフェ東京。
当日は蔵人であり、ラベルデザインも手がけた伊沢広海さんを迎え、実際に様々なお話を伺いながら、５種類以上の日本酒をお楽しみいただけます。

シェフの作る今錦に合う料理もワンコイン５００円でご用意してお待ちしております。

東京ではまだあまり出会う機会の少ない「今錦」、この機会にこの美味しさを味わってみてはいかがでしょうか。

『今錦JAMBOREE』
日時：2014年8月1日（金）19:30～22:30

会場：NEURO CAFE TOKYO（外苑前）

交通：銀座線・外苑前駅／総武線・千駄ヶ谷駅／大江戸線・国立競技場前駅よりそれぞれ徒歩10分

会費：￥2,500（５種類の利き酒付き）
※チケット以外のお酒およびおつまみは全て500円キャッシュオンでお楽しみいただけます。
※お酒の数に限りがございますので予めご了承願います。
