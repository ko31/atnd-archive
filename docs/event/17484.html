Hack For Japan アイディアソン 7/23 仙台会場

“東日本大震災に対し、自分たちの開発スキルを役立てたい”
―IT開発者の皆さんのこうした声をもとに、Hack For Japanは生まれました。

このHack For Japanのアイディアソン/ハッカソンが再度仙台にて行われます。
7/23 アイディアソン ( 懇親会 )
7/30 ハッカソン ( 懇親会 )

今までも複数の有意義なプロジェクトがHack For Japanから生まれています。皆様是非ご参加ください。

イベント概要

アイディアソンについて

前回5/21の開催では非常に好評をいただいたアイディアソン。今回は仙台会場独自の試みとして、仙台が誇るアイディア創発の専門家である石井力重さんにアイディアソンの進行をお願いします。

また、アイディアソンはIT開発者以外の方もお気軽にご参加ください。前回もボランティア団体等のIT分野以外からの参加が半数を占めました。

  • 震災復興でやりたいことがあるけど仲間が欲しい方
  • 復興に手を貸したいけどアイディアがまだ浮かんでいない方

こういったかたの参加も歓迎いたします！

  • 講師
    • 石井力重さん(アイデアプラント)
  • 概要
    • 仙台でおなじみとなったアイディアワークショップの技法を今回はアイディアソンに応用していただきます
