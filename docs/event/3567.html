description

つくば近辺で30～40分(ウォーキングの場合1～1.5時間程度)、スローペースでジョギング(またはウォーキング)しながら雑談する感じで。参加者がある程度集まるようなら、定期開催する事も考えます。ジョギングとウォーキングのどちらにするかは、参加者層に合わせて決定したいと思います。

日時

とりあえず未定ですが、自分以上の参加者が1名以上集まった段階で参加者全員の都合がいい日時にしたいと思います。

参加資格

特に無いですが、主につくば近辺にお住まいの方で、以下のような方の参加を募集しています(それ以外の場所にお住まいの方でも参加可能ですが、その場合、つくばまで来てもらうことになります)。

  • ジョギング or ウォーキングするのが習慣だが、一人でジョギングするのは退屈だと最近感じている方、ジョギング仲間を探している方
  • ジョギング or ウォーキングを始めたいが、きっかけがなかなか無くて…という方
  • ダイエットのためにジョギング or ウォーキングを始めようと考えている方
  • (kmizuと雑談したいという奇特な方)

コース

コースに関しては参加者の都合に合わせて柔軟に対応したいと思いますが、基本的には筑波大学付近のコースになると思います。


必要なもの



  • ジャージなど運動に適した服装

  • 運動に適した靴(特にジョギングを普段する習慣が無い人は、ジョギング用シューズの方が良いかも)
