謎解きイベントって何だ？？

これは、参加者自身が様々な謎や暗号を実際に解いてゲームを進めるイベントです。

最終的には『信州の秘宝』を見つけ出してもらうわけですが、入手する方法は全て、部屋に仕掛けられた謎を解くことで解明できるようになっています。

それらの謎を解くヒントは会場（torinoco）内に散らばっています。正しく謎や暗号を解いていけば『信州の秘宝』にどんどん近付くことができます。

制限時間内にクリアできれば達成感を！ できなければ、悔しさを！いずれにしても日常では味わえないワクワクを提供します！

※ANTDのシステムの都合上、日時は13時から18時と表記されていますが、実際は①13時〜②16時〜の2公演です。
※ANTDのシステムの都合上、定員は40名と表記されていますが、実際は1公演につき20名が定員です。

ゲームの世界観

—信州に眠る「謎の秘宝」。あなたはそれを狙う盗賊団のひとりです。これまでの調査によって、秘宝には“とある魔法”がかけられていることが分かりました。いったい、その魔法とは何を意味するのか？ 制限時間内に謎を解き明かして“正しく魔法”を使い、「信州の秘宝」を手に入れましょう！
