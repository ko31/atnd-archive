現在開催中のウェブアプリケーション開発コンテスト Mashup Awards 6 (MA6) に連動して、
今年より実践形式の開発イベントMA6 Mashup Camp（マッシュアップ・キャンプ）を開催します。

このイベントは男女ともご参加大歓迎です。ややこしいタイトルですみません！

Mashup Campってなに？

Mashup キャンプは、参加者それぞれノートPCを持ち込んで、 みんなでわいわい開発しちゃおうっていうイベントです。
キャンプという名前ですが基本的に日帰りです。
MA5ではHackathonという名前で1度だけ開催しました。

MA6では、たくさんの人に気軽に参加してほしいという思いからCampという名前にし、定期的に開催していきます。

女性視点ですてきなWebサービスを

MashupAward5まで、応募者や応募作品のコンセプトはどちらかというと男性・男性視点のものが多い印象でした。
「自分が便利」「自分がおもしろい」という基準はとても重要ですが、
今回は発想を転換して自分以外の人が気にいるWebサービス、作ってみませんか？

パソコン、インターネットに詳しくない人でも簡単に操作できるユーザビリティや、
女性ならではの不便さを想像してみると新しいアイデアにつながるかもしれません。
女性の方は是非ご自分のアイデアを披露してみんなで開発してください。

女性スピーカーからのお題

とは言えいきなり異性の視点にたつというのも厳しいですよね。

というわけで、今回はMashup CaravanでTalkしていただく女性スピーカーから「こういうのがほしい！」というお題をいくつか出してもらう予定です。

自分たちで設定したお題でもかまいません。自由にアイディアを練って、自由に作品として表現してくださいね。

スカイミント 岩崎 様 (@kaoru_iwasaki)

- タクトモAPI を使って、タクシーを使いやすくするアプリやサイト
- ライドシェアジャパンのAPI を使って、相乗り募集情報を地図上にプロット表示するアプリ
- 上記のAPI と他 イベント系 API などをマッシュアップして、各種イベント会場まで行くために、車やタクシーの相乗り仲間を探すアプリかサイト

ユーザーローカル 閑歳 様 (@kansai_takako)

- mixi チェック / GREE のいいね / facebook のいいね / はてブ / Twitter がマルチポストできるエクステンション
- Twitter でフォローしている人が参加するイベントを通知する BOT
- Twitter の流速を計測する BOT / 2nn の「平均投稿時速：XXXres/h 」みたいな

連携イベント Mashup Camp – Girls Talks

当日作るWebサービスのネタ出しやチーム決め・顔合わせを兼ねた事前ミーティングを、 9/15（水） に行ないます。
9/15（水）は、2部構成となっており、

  • 第1部 18:30〜20:20でAPI提供企業より地図に関する講演を行っていただきます。
  • 第2部 20:30よりMash up Campの事前ミーティングを行ないます。
  1. 9/15（水）18:30 Mashup Caravanを聞いて構想を練る。
  2. 9/15（水）20:30 Mashup Camp 事前ミーティングで構想を企画にする。
  3. 9/18（土）10:00 Mashup Campで企画を実際に開発する

という流れになりますので、ぜひMashup Caravanからご参加ください。

Mashup Caravan – Girls Talksの詳細・参加登録はこちら ⇒ http://atnd.org/events/7694
