朝カフェの会は、当日参加大歓迎、途中参加、途中退出自由のゆる～い交流の場です。
事前申込みは不要ですので、お気軽にご参加ください。
参加費は、各自当日の飲食代のみです。
机の上に青い旗を立てていますので目印にして下さい。

【開催日時】第３金曜日　7:30～9:00

【開催場所】あいるとんかふぇ（東京都荒川区南千住7-2-15）

【参加費目安（飲食代実費）】トースト220円/ピザトースト330円/コーヒー350円（食事とセットで-100円）

【会場の特徴】
地元商店街にあるほんわかした雰囲気のお店。
南千住駅 徒歩3分

【主催者からのメッセージ】
地元の方はもちろん、常磐線沿線、日比谷線沿線の方も気軽に立ち寄っていただける場所です。パフェも美味しいです！
