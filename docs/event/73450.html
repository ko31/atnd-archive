（Android Studio 要件）Windows の場合

・Microsoft Windows 8/7/Vista/2003（32-bit 版または 64-bit 版）
・2 GB 以上のシステム メモリ（RAM）、4 GB 以上を推奨
・400 MB の空き容量のあるハードディスク
・Android SDK、エミュレータのシステム イメージ、
　　キャッシュ用には少なくとも 1 GB 以上
・1280 x 800 以上の画面解像度
・エミュレータ アクセラレータ向け（任意）
　　IntelR VT-x、IntelR EM64T（IntelR 64）
　　Execute Disable（XD）ビット機能対応の IntelR プロセッサ

　※上記が必要な要件ですが、他に追加してインストールするものもありますので、
　　ハードディスクは、10G以上の空き領域を作成しておいてください。

　※お持ちいただいたパソコンで動作できるようできる限りサポートしますが、
　　万一、開発環境、エミュレータが動作しない場合はご了承ください。
