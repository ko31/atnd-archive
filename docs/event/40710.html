NFCといえばAndroidばっかりなので、たまには別の開発環境も触ってみたいので開催。

Win8＋NFC関連をぬるく(場合によってはカッチリ）勉強します。

東京都内にて開催予定。
豆蔵様に会場をご提供いただけることになりました。大感謝！
会場予約が別イベントと日程が衝突しましたので、１週間後ろにずらして7/13(土曜日)になりました。

  • 要 Win8環境です。
  • MacでもVirtualBox + Win8で動かせます(※1)
  • VisualStudio Express for Win8 を事前インストールしておくのを推奨

※1 1CPUで動かさないとRC-S380が認識しない不具合あり

勉強内容(予定)

WinRT APIを使ったNFCアプリ作成(XAML+C#)
SNEPを使ったWin8/Android間での連携方法

事前インストールのお願い

Visual Studio Express 2012 for Windows8 のインストールをお願いします。
以下のURLから、「Visual Studio Express 2012 for Windows8」を選択してダウンロードしてください。
http://www.microsoft.com/visualstudio/jpn/downloads#d-2012-express

インストールから30日間試用できますが、その後はライセンス申請することでプロダクトキーを入手して引き続き利用することができます(無料)
