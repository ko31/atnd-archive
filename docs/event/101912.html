description

　近年、各国の産業政策により、世界中のモノやコト、そして人を繋げて、新たなビジネスを創り出すオープンイノベーションが注目されております。この流れの中で、OPCの基本コンセプトである「つなげる・つたえる・安全に」は、各産業の変革を迅速に実現する上で、大きな役割を果たすと信じております。

　そこで、今回のOPC Dayでは、オープンイノベーションの創出に向けて、OPCが、産業界の未来像に対し、「何に貢献できるか？」、そして、「どのような効果もたらすことができるか？」に焦点を当て、日本とドイツに於ける産業政策の最新動向を元に、OPCの先進事例やコラボレーション活動を紹介いたします。

　本カンファレンスには、OPC会員以外の方々も自由に参加できます。より多くの方にOPC UAを知って頂くため、冒頭のセッションではOPC UAの概要も紹介します。この機会に、OPCに興味のある方は是非足を運んでいただき、OPCの可能性を実感してください。　皆様のご来場を心よりお待ち申し上げます。
