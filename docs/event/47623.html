・・・と、まぁ、東京で オープンマジカ をやるので、名古屋でも祝っちゃわね？的イベントです。

祝・日本DCオープン★Windows Azure 最新情報

  • スピーカー
    • 大森さん（日本マイクロソフト）
  • 概要
    • 2/26よりWindows Azure の日本データセンターが開設となりました。日本でのサービス提供を開始してから4年、当初に比べると格段にサービスの幅が広がりました。日々進化を遂げている Windows Azure の最新情報 (直近1年くらいを中心に) をお届けします。

Windows Azureにおけるアプリケーションアーキテクチャデザイン

  • スピーカー
    • 酒井さん (JAZUG)
  • 概要
    • Windows Azureを利用する際のアーキテクチャデザインで考慮すべきポイントを紹介します。
      また、先日公開されたCloud Design Patternsのパターン適用例も併せてご紹介します。
