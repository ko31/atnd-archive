かながわシニア起業セミナー

本セミナーは、神奈川県の「シルバーベンチャー創出促進事業」を受託し、銀座セカンドライフが実施するものです。

セミナー概略

本セミナーは、最近の起業の動向・特徴や、起業のアイデアの発掘方法、アイデアをビジネスプランに組み立てる方法について、事例を交えながらお話しします。起業アイデアを考えるために、会社員時代の得意分野を自己評価する「自分の棚卸し」や、自分のやりたいこと、抱いていた夢などを洗い出す「やりがい発見」のワークを実施します。さらに具体的な起業アイデアがある方は、「事業計画書」の作成に取り組んでいただきます。

後半では、実際に地域で活躍する定年前後で起業した方の講演を聞いていただき、起業の具体的なイメージを掴んでいただきます。ご希望の方は、個別相談時間があります。ぜひご予約ください。

講師プロフィール

銀座セカンドライフ株式会社 代表取締役　片桐　実央

行政書士、1級ＦＰ技能士、インキュベーションマネージャー。 学習院大学法学部卒業後、花王株式会社 法務・コンプライアンス部門法務部、大和証券ＳＭＢＣ株式会社引受審査部に勤務。祖母の介護をきっかけに「ゆる起業Ⓡ」を支援する、銀座セカンドライフ株式会社を2008年7月に設立。起業支援サービスを含んだレンタルオフィス「アントレサロン」を、銀座・東京・新宿・池袋・横浜駅周辺に8店舗運営。創業セミナーの講演は年間100回を超え、毎月150件の起業相談を受け、これまで7,000件を超える。

・著書　講談社新書「シニア起業」で成功する人・しない人。
・著書　同友館　片桐実央の実践！ゆる起業Ⓡ－シニア起業の成功書－
・毎週水曜連載　産経新聞「片桐実央のゆる起業のススメ」
・連載月曜連載　夕刊フジ「定年起業への挑戦」
・連載金曜連載　夕刊フジ「定年起業　私の事業計画書」

講演者プロフィール

株式会社ソーシャルウィル 代表取締役　平山　雄三　氏

東京大学工学部卒業後、株式会社東芝に入社。世界最高速の半導体レーザを開発したのち、MIT客員研究員となる。その後、東芝にて世界初の裸眼立体テレビの製品化に成功し、発明協会全国発明表彰 21世紀発明賞、文部科学省文部科学大臣表彰科学技術賞、東芝社長表彰などを受賞。工学博士。2013年に（株）ソーシャルウィルを設立。科学を学べる子供向けプログラミング教室を新宿、横浜にて開講し、現在4期生が学んでいる。また、ベネッセコーポレーションが提携する世界最大級のオンライン講座Udemyなどでも映像教材を配信している。

株式会社Triple Win 代表取締役　朝倉　博行　氏

中央大学　法学部卒業後、1973年（株）第一ホテルに入社。ベルボーイからスタートし、ホテル開発部門では10軒のプロジェクトに参画。その後、1993年に三菱地所（株）ホテル事業部に転籍。2000年に分社した、（株）ロイヤルパークホテルズアンドリゾーツでは、開発部門の担当役員として7軒のプロジェクトを主導。2008年から運営会社の（株）ロイヤルパークホテルマネジメントの代表取締役に就任。2015年6月末で、両社の役員を退任後、同年7月1日付で、株式会社　トリプル・ウィンを設立。これまでの経験と人脈を活かした、「ホテル・旅館などのコンサルティング、アドバイザリー業務」に従事
