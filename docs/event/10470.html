第1期Social & Smartphone app school(SS app school)をスタートいたします。

第1期は「mixiアプリ開発」講座です。（全5回）
講師にSAPトップ企業のエンジニアさんをお迎えして実施します。
今回は記念すべき第１期ですので参加費は無料です！
ぜひお気軽にご参加ください！
参加者全員でmixiアプリを作りましょう！！

■内容
　－全5回の参加でmixiアプリを実際に作れるようになるのが目標です
　－講師にSAPトップ企業のエンジニアさんをお迎えし、ケーススタディとし て、実際のアプリを題材にして学んでいきます
　－講義間には課題がありますので短期間でしっかり力をつける事がでます
　－第1期目の開催ですので初めての方ばかりの参加です。お気軽に参加下さい

■日時
※大変申し訳ありませんが、講師との打ち合わせの都合上年明けからの開催予定となりました。
以下仮日程となります。
　第1回：1月17日（月）講義形式･･･「アプリ開発のポイント」他
　第2回：1月24日（月）講義形式･･･「開発のポイント」他
　第3回：1月31日（月）ハッカソン＆質問形式
　第4回：2月14日（月）ハッカソン＆質問形式
　第5回：2月21日（月）アプリ発表LT
　※各回：19：30～22：30
　※日時は変更する場合もございます

■場所　都内調整中
　※詳細決まり次第ご連絡いたします

■対象者：35歳までの現役エンジニア

（１）サーバサイド：PHP Ruby Perl Python Java C++の経験 もしくは
　　 クライアントサイド：Javascript HTML5 Flashの経験を
（２）3年以上(30代)、2年以上(20代後半)、1年以上(20代前半・学生)もっている方
