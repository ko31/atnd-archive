2011年5月5日午後12時27分、UCSの全漢字のグリフが作成されました。

GlyphWikiにて、グリフを作成していただいた皆様、本当にどうも有難うございました。

これを祝して、5月14日の午後18:30頃から、東京・新宿周辺にて簡単な打ち上げをしたいと思います。

また、会場では、正式な花園明朝フォントのリリースや、今後のロードマップの発表も行います。

場所はまだ未定ですが、参加者の人数を把握したいため、参加希望者はATNDにて参加表明をお願いします。場所は5月10日までに決定し、このページおよびTwitter (@kamichikoichi, @kawabata) にてアナウンスする予定です。

電子書籍の普及に伴い、外字に頼らない、大規模なでフリーな漢字フォントの登場が強く望まれてきています。GlyphWiki＆花園明朝は、世界最大の文字数の漢字を収録した漢字フォントとして、今後の電子書籍の発達に貢献できると思います。

今後とも、皆様の協力をお願い致します。

祝賀会は通常の食べ物屋で行います。特にグリフウィキには若手の参加者が多いので、学生の方には補助いたします。是非ご参集いただければと思います。

(2011/05/12 09:30)
