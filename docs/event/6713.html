description

エンジニアライフでコラムを書いてる人と書いていた人と携わっている人などが集まりLT大会を行います。


LTのテーマは「コラムニスト」です。

たとえば

「コラムニストとして」とか「コラムニストの○○さん」とか「コラムニストとブロガーの違い」とか

キーワードが入ってればOKです。

なお、「真面目具合」はおまかせしますので笑いの調整はお願いします。

なお、当日は

「実際に会ってイメージとギャップがあったコラムニストBest5」

「実際に会って想像通りだったコラムニストBest5」

を投票で決めます。

1位になった方にはなにかあるかも？
