【毎月第２月曜日開催】ナポレオン・ヒル読書会～「思考は現実化する」～

「それが何であれ、心に描いたものは現実化できる」／ナポレオン・ヒル

８０年の歴史と実績を誇る、人類普遍の成功哲学であるナポレオン・ヒルの成功哲学。
彼の書籍に書かれている哲学は、１７の原理原則に分類され、さまざまなケーススタディに対応し、高い汎用性を持つことで誰でも実生活で応用実践可能なものばかりです。
そんな世界中で多くの成功者を生み出してきた哲学書、「思考は現実化する」をナポレオン・ヒル財団の上級指導員によるファシリテーションで、読み解く読書会を開催致します。
