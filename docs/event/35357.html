2013年もゆるーくボルダリングの魅力を広げて生きたいと思います。
どちらかというと、登った後の反省会がメインになりそうです。

ハッシュタグは　#マルダリング　で変わらずです。

2012年、月一開催してきたマルダリングが無事終わりました。
参加してくださったみなさん、ありがとうございました。

一年間で終わりのはずでしたが、勢いで2ndシーズンとして2013年も行うことになりました。

酔った勢いで決まった話なのでどこまで本気か怪しいところですが、マルダリングの一環として次の2大イベントも行う予定です。

①クライミングシューズ買っちゃう会(2013/01/19(土)　目白　Calafate)
　そろそろ靴買っても良いかも？と思っている人はこの際買っちゃいましょう！
　マイシューズ買ってテンション上げていきましょう。
　　⇒○山が仕事のため中止します。1/25(金)のマルダリングの時に買いましょう！

②外岩行っちゃう会(2013/04/13(土)　御岳ボルダー)
　ボルダリングの本来の姿である、自然の岩場での登りを体験しましょう！！
　ジムとは一味違った雰囲気で、登らなくてもピクニック気分で行ってもOK！
　※登る人はクライミングシューズが必要です。(レンタルはありません)

※突然ですが延期のお知らせです。
直前の変更ですいません。
1/18(金)は秋パンの休業日のため、1/25(金)に日程を変更させれもらいます。
これにこりずにまた参加お願いします。
