概要

オープンなやりとりのなかで、OSHW（オープンソースハードウェア）の在り方などを議論したりするオンラインセミナーイベントです。

各地で活動するハードウェア制作者の知識・知恵を共有しながら、ソフト／ハードの垣根を越えたさまざまなモノ・コト・ヒトのつながりを模索する集まりとして定期的に開催を考えています。ご興味のある方は是非ご参加ください。

参加費用： 無料

大阪会場にお越しになる方に
開催当日、会場となる大阪イノベーションハブでは「全国Startup Day in 関西」が開催されています。50人超の参加者が以下の会場レイアウト図のコラボレーションスペースにいますので、ご注意ください。
大阪イノベーションハブのレイアウト 使用可能スペース
オープンハードセミナー2014 3Qは、会場入って左奥（会場レイアウト図では右下）の相談室になります。入り口受付にてご確認ください。

以下の拠点でオンライン中継を行います。

  • 東京： オープンハードカンファレンス 会場： MONO /青海
  • 大阪：日本橋でんもく会 大阪イノベーションハブ 梅田
    • 参加申込み（ このページです！ ）
    • 共催：大阪イノベーションハブ

お互いに自己紹介やセミナー交流・質疑応答を行い、各地域ならではの情報を共有できればと思っています。
