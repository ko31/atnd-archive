概要

　Windows 8に導入される新しい種類のアプリケーション、Windows ストアアプリの実習形式のセミナーを講師の方を招いて行います。
　新世代のアプリケーション開発を学べるチャンスですので　是非ご参加ください。

日時： 2012年 9月22日（土）13:00～17:00
場所： 東海大学熊本キャンパス 新1号館 N1205教室
講師： 服部 祐一 様 （MSPフェロー）

　参加費は無料となっておりますので、Windows 8に興味がある方でしたらどなたでも参加いただけます。

タイムテーブル

12:30～　　　　受付開始
13:00～13:10　 前説
13:10～14:00 　Windows ストアアプリ解説
14:00～14:10 　休憩
14:10～14:50 　C#によるWindows ストア アプリ開発入門
14:50～17:00 　Windows ストア アプリ制作実習

会場へのアクセス

熊本キャンパス案内 | キャンパス案内 | 大学概要 | 東海大学

電車

「熊本」駅からJR豊肥本線で５つ目（熊本駅より約13分）の「東海学園前」駅下車、駅正面

駐車場

　大学の駐車場が利用できますが、事前申請が必要なため車で来場される方はイベント参加登録時のアンケートにて御回答ください。
　なお、登録後に駐車場をご利用されたい方は下部のコメント欄にその旨をお書きください。

注意事項

　当日はVMware PlayerなどにWindows 8 RTM と Microsoft Visual Studio Express 2012 for Windows 8 がセットアップされたPCを各自ご持参ください。
　会場での貸し出しなどは行いません。

懇親会

　セミナー終了後に懇親会を予定しております。
　セミナー会場より徒歩圏内の会場を予定しておりますが、人数によっては市内へと懇親会会場が変更になる場合があります。
　参加をご希望される方はイベント参加登録時のアンケートにて参加表明をよろしくお願いします。
