description

HTML5 Japan Cup の開催を記念して、html5jマークアップ部のHTML5マークアップ勉強会を開催します。

開催概要は以下のとおりです。

2014年6月10日（火） 19:00～21:00

場所：ファーストサーバ株式会社 会議室 （大阪市中央区安土町1-8-15 野村不動産大阪ビル3F）

参加料：無料

主催: html5j.org マークアップ部員＠大阪
協力: HTML5-WEST.jp

タイムテーブル(予定）

時間 発表者 タイトル 内容
30min 村岡正和 HTMLのコトバ 新卒の歓迎会も一段落して、社員研修の時期だと思います。今さらですが、HTMLに関連する当たり前の専門用語を解説してみたいと思います。
ちょっと当たり前すぎて玄人には面白くないかもしれませんが、まあ春なのでカンベンしてください。
30min 森和恵 『突撃！隣のマークアップ』 Markup Cafe に参加する方なら、「HTML5？もち、デフォルトだよ☆ミ」みたいなマークアップを日頃されていることだと思いますが、まだまだなサイトもたくさんありますよね。
新規案件で、HTML5を使おうとしても、それを知らないインハウスのWeb担当の方に「まだ、早いんじゃないですかね。」とか言われて断念したり…。
そこで、いま企業がどんな具合にHTML5のコーディングをしているか？を調査してみたいと思います。その結果、おおよその平均値がみえてるくるかも？（いや、これから調べるので、結果は何ともいえないんですけれど。）
一般的な企業へのHTML5導入の実情を形にすることで、今度の案件から、HTML5の導入がスムーズになるかもしれません。（もしくは、本当にまだ早いという結果が出るかも？）
ヨネスケ さんよろしく、「突撃！隣のマークアップ」をしてみたいと思います。
30min Q＆A・LT みなさん LTしたり、質問しあったりします。
