山崎朋新作ダンス公演『窓の外』

窓の外

へだたり について考えています。
壁ではない。
からだが感じ取っている、
あるいは信じ込んでいる。

悲劇を生きられず茫漠とした
疲弊感を重ねつづけるものたち。

溶解と凝固。

世界の一切が風景になるとき、
へだたりは侵され、私は淵にいて、
その滅形を眺めていた。

profile
-————————————————————

山崎朋（やまざきとも）
1989年長野生まれ。東京藝術大学音楽環境創造科四年在学。
マイムや舞踏などを学んだ経験をもとに、独自のスタイルの作品づくりに取り組んでいる。今回はドラマトゥルクの横堀応彦と共に初の長編ソロ作品制作を行う。
最近の出演作に矢内原美邦演出「桜の園～最後の実験～」など。
