Faster Harder!!

title Faster Harder!!

description

■どんなイベント？

東方曲のクラブアレンジや、他のクラブ音楽もいっぱい流れる、音楽好きにはたまらないイベントです。
東方ファンやクラブ音楽が好きなお友達と一緒に、おしゃべりしたり音楽に身を任せたりしましょう。

■どんな音楽が流れるの？

今回はトランスやハードテクノ、ハードコア、ガバ、FUNKOTなど、イベント名の通り比較的速くてハードな曲が中心です。ニコ動やようつべで大人気の東方アレンジと一緒に、同じジャンルのクラブアンセムも一緒に楽しめます！

■音楽はPCのスピーカで聞くから結構です。

それはとてももったいないことです。
PCやコンポで聞くのも結構ですが、大音量で聴くクラブアレンジは、自宅では味わえない気持ち好さを味わうことができます。この機会に、普段聞いてるあの曲やこの曲が本当はどんな音楽なのか、体感してみてください。

■クラブって怖くない？大丈夫？

大丈夫だ、問題ない。暗かったり、会場がススキノだったり、様々な報道やドラマの影響で「クラブは怖いところ」というイメージがありますが、そんなことはありません。フロアに集まる人達は、東方アレンジとクラブ音楽が大好きなフツーの人たちです。クスリや未成年の飲酒とナンパ/痴漢行為は法律と主催（ハゲ）が許しません。

■未成年なんですが…

アルコール無しのドリンクもあります。あとはタバコを吸わないだとか、22時以降の外出をしない（18歳未満）だとか、未成年としての社会ルールと法律を守ってくだされば、何も問題はありません。繰り返しますが、クスリや未成年の飲酒とナンパ/痴漢行為は法律と主催（ハゲ）が許しません。

■ところで→の「参加する」は押さなきゃいけないの？

僕も最初はそう思ってましたが、幼くても、もとい押さなくても大丈夫です。ふらっと気軽に来て東方好きやクラブ音楽好きの方とお話ししたり、お酒飲める人はお酒飲んだりして、終電に遅れないように帰ってくださーい。
