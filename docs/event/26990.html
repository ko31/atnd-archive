description

お知り合い（＆紹介者さま）繋がりな、お花見的な呑み会をやろうと思ってます。

場所：尼崎市立青少年いこいの家

兵庫県川辺郡猪名川町万善字東山6-1
http://www.aspf.or.jp/ikoi/

施設は公共のキャンプ場なので、大音量は無理ですが
他に迷惑がかからない程度の音楽は大丈夫と思います…
（電源も備わったところなので、そのあたりもなんとかなるかと）

バーベキューとメインでやろうと思うんですが、
飯ごう炊さんとかカレー作ったりとかも出来るので
参加者へアンケートしたいと思ってます。

交通手段ですが、基本的に車で移動となります。
最寄り駅は、能勢電鉄日生中央駅　＞路線バスも出てます
http://www.aspf.or.jp/ikoi/access.html

出来れば事前に待ち合わせて、乗り合いで向かえるようなのがベストかもです。

＊飲酒に関してですが、ドライバーはもちろんダメですが、
　公共施設なので、オトナらしさをふまえて、控えめに、という建前で宜しくお願いします。

＊定員ですが、ひとまず先方へは6人〜って話で場所取りをしてます。
　ただ、途中から増えても何も言われなかった実績があるのでw
　そのあたりお誘い合わせ上、集まっていただければ嬉しいです。

では、ご参加お待ちしています。
