【概要】
オーム社から発売されている「情熱プログラマー ソフトウェア開発者の幸せな生き方」という本があります。皆さんはお読みになりましたか？

  • 情熱プログラマー ソフトウェア開発者の幸せな生き方｜Ohmsha
  • 楽天ブックス｜情熱プログラマー – チャド・ファウラー : 本
  • Amazon.co.jp： 情熱プログラマー ソフトウェア開発者の幸せな生き方: Chad Fowler, でびあんぐる: 本

この勉強会は、「情熱プログラマー」を読んで内なる情熱を掻き立てられたプログラマー / 非プログラマーが、その情熱について語るために開催されます。

スピーカーには持ち時間5分でLT(Lightning Talks)をしていただき、そのあと5分以内の質疑応答の時間を設けます。アツいメッセージを聞き、アツい気持ちを投げ返す。スピーカーと参加者、みんなで作り上げ、温度を上げて行く場にしたいと思っています。

【発表者募集】
「情熱プログラマー」について情熱的に語りたいぜ！という発表者を募集します。「情熱プログラマー」からエピソードを一つ選び、前述の通り5分のLT + 5分以内の質疑応答、という構成で発表をお願いします。

5分のLTにつきましては、スライドを用意してのプレゼンであっても、担当のエピソードを音読するだけでもかまいません。

発表いただける方は、申し込みフォームにチェックを入れ、希望のエピソードをご記入ください。応募者多数の場合はこちらで抽選させていただき、後日ご連絡いたします。なお、発表されるエピソードが片寄ってしまうのを防ぐため、可能であれば第2希望・第3希望ぐらいまでお書きください。

なお、発表希望者が多数の場合は質疑応答の時間を少し減らすことも検討しています。
