第1回「ゆるふわ愛されシェルスクリプト」につづき、第2回「もてカワ bash」の開催が決定しました。

エンジニアのみなさんはコンピュータを使って、さまざまな環境や言語で開発をおこなっていると思います。一流のスポーツ選手が道具を大事にするように、エンジニアも普段業務で使用している OS や開発言語を大事にすべきでは？と思い、このセミナーを開催するに至りました。

第1回ではシェルスクリプトの基礎を紹介しました。第2回では第1回の内容をダイジェストで紹介したのちに bash の機能である「条件分岐」や「繰り返し」などを応用したより実践的な内容となっています。

ところでみなさんは黒と白のコマンドラインを使った Linux のサーバ運用って難しいそうと思っていませんか？しかし、シェルスクリプトを使うことで「繰り返し作業の自動化」や「テキスト処理の省力化」、「プログラム監視の効率化」などをパワフルに実現できます。
本セミナーでは、初めてシェルスクリプトを扱う方、ちょっと発展した使い方をしたい方のために簡単なコツやテクニックを伝授しちゃいます！

やってみれば”意外と簡単”だったりと何でもできちゃいます！！
少しでも興味のある方はお気軽にご参加ください。
