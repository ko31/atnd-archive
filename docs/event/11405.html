前回決定した通り、今回より名称をHachioji.pmに変更しました。
新年会的に皆さん気軽にご参加ください。
—
本イベントは主に東京西部、神奈川西部のエンジニアの交流を目的としたものです。
普段エンジニア交流会は都心部、平日に多くおこなわれており、それらの補完を目指しています。

pmとついている様に、Perlユーザーが多くいますが、話題はPerlに限定されているわけではありませんし、私を始め、PHPやJS、HTML等をPerlよりも使っている人多くいます。
エンジニアであれば、お気軽に参加いただければと思います。

特に若い人、お気軽に参加ください。

—
会場は八王子駅周辺のどこかになります。詳細は未定です。
参加人数を見つつ、予約を入れたいと思います。
前日までに会場をこのATNDに再設定するので、確認をお願いいたします。
（もし八王子で良い店をご存じであれば、是非教えてください）

基本的に会費制ではありませんので、当日の飲食割り勘になります。
前回は3000円ほどでした。参考まで。

—
hachioji.pm #1においては、一応まじめな顔も作っていく為に、＃0で話した通り、超簡単なLTをできればと思います。
これはできれば皆さん全員にお願いしたいです。
ただ、これは必須ではありません。
基本的に、席の端っこから端っこの人まで話そうぜ！というコミュニケーションを目的とした物です

ガイドラインは
・発表する事は一ページ（A4など）にまとめること
・発表時間は１～２分（銅鑼とかないのでｗこだわりませんが）
・自分の名前（ハンドルでもいい）を載せること（コミュニケーションが目的なので）
・質疑応答をやればいいので、書ききらなくてもOK
・ネタ発表歓迎します
・このガイドラインはあくまでガイドラインなので、破ったっていいです。
・目的は皆に発表してもらうこと、コミュニケーションを推進する事です。

テーマはまったく自由ですが、何もでてこなければ、技術と関係なくてもいいので、hachioji.pmを盛り上げる為の何かでお願いします（私が聞きたいので！）

とにかく、難しく考えず、思いつきや１ネタをまとめる感じのもので十分です。

当日までに私にPDF形式にしてお送りいただければ幸いです。
（適当に印刷するだけなので、印刷していただけるのであれば持参でOKです）
