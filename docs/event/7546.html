description

日本の「和の心、侍の精神」を未来産業化とさせ、全く新しい日本式経済発展モデルによって、日本の危機、世界の危機を大反転させていくムーブメント、JAPAN MISSION　PROJECT(以下JMP)
そのJMPの全国実行委員長を務める、内海昭徳氏の著書、「新世界への航路 – 時代の羅針盤を求めて」が2010年8月17日（火）にAmazonにて販売が開始され、
みごとAmazon本の総合ランキング１位を獲得したことを記念し、「内海昭徳出版記念　スペシャルトークライブ」を開催する運びとなりました。

「人を悟らせること」が仕事になる時代になる！

これまで特別視、タブー視されていた人間ならではの知性、『悟り』の世界が、21世紀、客観的な論理とイメージによって学問化、技術化、商品化、産業化され、一般常識化していくロードマップを高らかに宣言する、この内海昭徳のトークライブ。
大注目間違いです！
