「健康的な食生活を送りたいけど、ストイックなのはイヤ」とか「近所のスーパーで手に入る食材で、簡単で体にいいおかずを作りたい」と、お望みの方向けの講座です。

イベント概要

薬膳というと、どの料理も苦くて、高級食材が使われていて、しかも、色々と決まり事が多くて、作るのが面倒というイメージが付き物。

でも、本当は、季節の食べ物を食べる人の体調・体質に合わせて選び、美味しく食べながら、体と心を健康に導く食生活を指します。

この講座では、薬膳を日々の食生活に取り入れる上で知っておきたい自分の体調・体質に合う季節の食材の選び方について、ポイントをお伝えします。

さらに、“手間なし・安価・苦くない” の3拍子が揃ったオリジナルレシピ集がテキストなので、講座を受けてからすぐに、しかも、だれでも“ゆる～く”薬膳をベースにした食生活を続けられるようになります。(調理実習はございません。)　

講師

伊藤なお美

国際薬膳師（中国食医薬研究促進会）／ヨガインストラクター

ヨガを続けていく中、季節ごとの体調の変化に氣が付いたことをきっかけに、ヨガと平行して体調を整えるための方法として薬膳を学び、作り置きおかずやお菓子作りにその知恵を生かしています。

『その人の”はじめの一歩”に寄り添う』ことがインストラクター活動のモットーです。
