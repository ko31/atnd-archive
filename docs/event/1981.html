PHP 中級者 勉強会では、少し難易度の高い勉強を行います。

新しい勉強会は、PHP初心者勉強会ホームページにてご確認下さい。

Google Maps APIsなどを使いながら、オブジェクト指向などについても学びましょう。

勉強内容：Google Maps APIsのおもしろ機能


  1. IPアドレスから現在地を求める

  2. 緯度と経度から住所を求める

  3. 地名やランドマークから緯度と経度を求める

  4. ポリライン描画

参加費用は無料ですが、貸会議室の代金がかかります。


  1. 前回までの課題をプログラムしてきた人：500円

  2. 前回までの課題をプログラムできなかった人：1000円

前回までの課題（簡単）


  1. GoogleMapを表示して、中心にマーカーを立てる

  2. マーカーをクリックして、吹き出しを表示する

  3. 静的XML(緯度経度情報)を非同期で読み込み、マーカーを立てる

前回までの課題（少し難しい）


  1. 吹き出しをタブ化する

  2. 緯度経度情報をPHPで動的にXML化し、非同期で読み込む
