Esolangとは

Esoteric ( Programming ) Languageの略で難解な（そして奇妙な）プログラミング言語の事を指します。

( 参照 : http://esolangs.org/wiki/Main_Page )

Esolangに関することならなんでも歓迎です！Esolangを作って見せびらかすもよし、とっておきのEsolangを紹介するもよし。
皆でEsolangネタを持ち寄って12/25まで繋ぎましょう。
