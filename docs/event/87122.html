■このようなお悩みはございませんか？
・BtoBマーケティングを体系的に学びたい・・・
・デジタルマーケティングが本当に効果があるのかわからない・・・
・Webマーケ、展示会、電話営業、どの取り組みに注力すれば良いか迷っている・・・
・活用しきれていない見込み客リストがたくさんある・・・
・営業の人数が少ないので、もっと効率化したい・・・
・インサイドセールスの効率的なやり方を知りたい・・・
・営業がクロージングだけに集中できる体制を作りたい・・・

経営者に寄り添った独自のコンサルティング・スタイルを貫く船井総研と、
BtoBマーケティングを支援するアイアンドディーは、最新のマーケティング手法を公開するセミナーを共催します。
本セミナーでは、マーケティングに特化した最新デジタルマーケティングとインサイドセールスによる
新規開拓手法やナーチャリング手法を、BtoBに範囲を絞って公開いたします。
本セミナーを受講いただくと、マーケティング活動における有益なヒントをつかみ取ることが出来ます。

今回は当日お越しいただいた参加者全員に豪華特典「ナーチャリングを成功させる極意」小冊子を進呈いたします。
また、勉強としてのセミナーのように成功ポイントだけをお伝えするのではなく、実際に成果のあった生の成功事例をたっぷりとお伝えします。

尚、人気セミナーのため、すぐに席が埋まってしまう可能性があります。
ご興味のある方は、下記のURLよりすぐにお申し込みください。
http://www.iad.co.jp/seminar/20170516/

BtoB企業のマーケティング手法と成功事例＆１０倍成果を出すデジタルマーケティングとは
講師：八角 嘉紘氏
1.BtoB企業の戦略とマーケティング（Web、セミナー、展示会、テレアポetc・・・）

3.見込み顧客の数を10倍にするデジタルマーケティング
4.顧客を行動に駆り立てる！コンテンツ作成のコツ
5.カスタマージャーニーマップって本当に必要？マーケティングオートメーションの第一歩

第二部　15:20～16:20
BtoBマーケティングを成功させる効果的なインサイドセールスとは？
講師：矢作　英樹
1.デジタルマーケティング×インサイドセールスのシナジー効果
2.インサイドセールスを成功させる5つの下準備
3.デジタルマーケティング×インサイドセールスによる効果的な新規開拓手法
4.意外と知らない！リストマネージメントの重要性
5.購買客を作り上げるナーチャリングの成功事例

最後に　16:20～16:40

◆講師紹介◆
講師①
八角 嘉紘氏
株式会社船井総合研究所　ITチーム　チームリーダー　チーフ経営コンサルタント
プロフィール(略歴)
徹底した調査を前提にしたマーケティング戦略・戦術の策定を得意としている。
IT企業に特化したWebマーケティングを軸にコンサルティングを展開。
「船井総研IT企業コンサルティングチーム」のWebマスターであり、マーケティングオートメーション、インバウンドマーケティングノウハウを中心にFacebook・ブログを絡めた独自のWebノウハウを数多く有している。

講師②
矢作 英樹
株式会社アイアンドディー　カスタマーリレーション部 部長
プロフィール(略歴)
BtoB企業の新規顧客開拓を支援するテレセールス事業の責任者。
業務委託によるテレセールス事業では、自身も大手ITメーカーに常駐してテレセールス業務に従事した経験もある。顧客の課題分野を深耕する聴取型テレセールスの手法、インサイドセールスの立ち上げ支援など、セミナー講師を多数努める。
