description

大阪国から、ぢるのといっせーが東京に殴りこみです。

  • 日時:3月22日(木)　夜19時～夜21時の2時間
  • 場所:池袋　ヴィーボ・サウンドスタジオ（ Vivo Sound Studio ）
  • 予算： 一人500～600円程度を見込んでいます。
    参加者全員でスタジオ代（二時間5700円）を割り勘致しますので、参加すればするほどお安くなります。
  • 持ち物： 楽器。スタンダード本（青本・黒本等）、演奏する人数分のリード・シートを持参していただければどんな曲でも大丈夫です。特にボーカルの方はリード・シートの持参を強くオススメ致します。
  • 演奏曲目： 主にジャズスタンダード。なんでもありです。先に言ったように、楽譜さえあれば例えアニソンでもやります。

セッション初心者の方や見学のみの参加も、もちろんオッケーです。オフ会です！！
みなさん奮ってご参加下さい！！

参加、ご質問はTwitterで、@jazzcoltrain (ぢるの)　または　@n_enot（ねの） にまでご連絡ください。
もちろんここでの参加表明もよろしくおねがいします。

  • 集合時間は 18時30分頃に池袋駅東口の「いけふくろう」前（周辺）に集合 です。
    18時50分ごろになればスタジオに移動しますので、遅れる方は必ず連絡して下さい。
    ※主催者は大阪の人なのであてにしないで下さい。もし何かあれば@n_enot　までお願いいたします。
  • セッションが終わった後は、スタジオ周辺の居酒屋で打ち上げをします。（場所は未定）
    むしろこっちが本命です。さぁ呑むぞおめぇら！！
    こちらの方も是非ご参加下さい。

みなさまのご参加お待ちしております。
