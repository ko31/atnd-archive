title 第24回「WEB TOUCH MEETING」やります！
catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。


■現在予定している内容：

　●セッション１(35分)
　『Webデザイナーでも作れるGoogle Androidアプリ』日本Androidの会 中国支部長 重村さん

　＜内容＞
　Androidアプリの開発手法はJavaを用いた開発しかないと思っていませんか？
　今回は、HTML & CSS & JavaScriptでAndroidアプリ開発を実現するアプリケーション
　開発ツールキット「WebSDK」について紹介させて頂きます。


　＜重村さんの紹介＞
　IT関連企業に勤める傍ら、Androidの普及啓蒙活動でどたばたしています。
　趣味でAndroid Marketにアプリケーションの公開など行っています。
　宜しくお願いします。

　日本Androidの会（日本アンドロイドの会）
　http://android-group.jp/


　●セッション２(45分)
　『事例紹介 オハヨー乳業サイトリニューアル』ココノヱ丸山さん

　2010年3月に公開した乳飲料メーカーのサイトリニューアルについての解説です。
　プランナー、ディレクターさん向けの話になると思います。

　＜内容＞
　・与件整理と企画の立案
　・デザイン/Flash実装について
　・運用/更新について
　・今後の運用について

　オハヨー乳業株式会社
　http://www.ohayo-milk.co.jp/

　＜丸山さんの紹介＞
　岡山県のウェブ制作を中心としたプロダクションで、ディレクターをしています。
　よろしくおねがいいたします。

　株式会社ココノヱ
　http://9ye.jp


　●セッション３(35分)
　『iphoneアプリのリリースから、３ヶ月たった現状報告』トライポット藤田さん

　＜内容＞
　今年の１月にリリースした「虎卓」や「GALジャン」の３ヶ月の順位の推移やダウンロード数を公開。
　アプリをリリースしての実感や、AppStore市場の簡単な分析。
　今後の可能性などを話します。
　技術的より、マーケティング的な内容です。営業マン、プランナー向きな内容です。

　＜藤田さんの紹介＞

　トライポット　藤田健一
　トライポット代表
　Augment8メンバー「今年はいろいろ面白いことをしたい！」
　最近は、iPhoneアプリや、mixiアプリの開発を研究中、Androidもやりたいな。
　その他、デジタルサイネージ、Flash、webシステム、ホームページの作成など。

　トライポット
　http://tripot.net/
