【AID勉強会】最新技術・ガジェット勉強会

【今回の目的】

「Oculus」「ジオフェンス」「iBeacon」といった、
最新技術、ガジェットの見識を深め、今後の企画の参考にして頂ければと思います


AID勉強会とは

AID定期勉強会は、無料アプリにおける3大要素
「開発」「DL拡大」「マネタイズ」について、情報提供を行っていく場です。

例えば、以下のような点について各専門家の方にお話頂きます。
『もっと効率的にアプリ開発出来ないの？』
『お金をかけずにどうやってダウンロードを増やすの？』
『収益を上げる為に気を付ける点は？』

定期開催致しますが、登壇者の方や発表内容は随時変更致しますので、是非ご都合が宜しい際にご参加下さいませ。

Tech Buzz Space（新宿三丁目E1出口徒歩4秒です！）

・Android/iOSアプリ開発を行っている/興味がある方
※広告会社様は参加NGとさせて頂いております

【１部】
Oculusを始めて14ヶ月。良いこと悪いこと普通のこと

Oculusを始めてみて解ったこと、良かった事を実体験をもとに解説しつつ、同時に悪かった事も包み隠さず話すことで最終的にはVRソフト開発に参加してくれる人を増えることを願った講演内容となっています。

OcuFes 　代表　桜花一門 様

ゲーム業界でのメインジョブは企画者。サポートジョブはCGデザイナーとプラグラマー。

【２部】
誰でも簡単に！？「ジオフェンス」でスマートフォンを一番身近なコンシェルジュに。

スマートフォンが急速に普及し、O2Oビジネスがより身近なものに。その1つである「ジオフェンス」（仮想的な境界線）を利用したサービスの企画や開発のポイントをご説明。
企画はあるけど作るのは大変……。そんな方々にも耳寄りな情報も。

株式会社ゼンリンデータコム
モバイルサービス本部　サービス企画部　ディレクター　中西紀子 様

業務システム開発等に従事。5年前から地図サービスに関わり、2013年10月より現職に。
地図が苦手な女子でも利用したくなるサービスを目指し、女子向けの地図サービス「恋するマップ～女子ちず～」を2014年3月にリリース。

【３部】
iBeaconが気になる企画者向けのビーコン入門

iPhoneのiOS7より標準搭載された「iBeacon」の発表から半年が経過し、国内でも事例が出始めました。いくつかの事例を交えながら、ビーコンを使ってどんなことが実現できるかを非エンジニア向けにお話します。
『ビーコンってなに？』と聞かれた時に説明ができたり、今後のサービス企画に活用できるようになります。

こんな方に！
・ビーコンを使った企画を提案・導入したいプランナー/ディレクター
・店舗や施設にビーコンを設置したいとお考えのご担当者

株式会社ゆめみ
マーケティングソリューション事業部 プロデュースチーム　有泉雅彦 様

株式会社ゆめみにて、フィーチャーフォンからスマートフォンまで幅広いモバイルコンテンツの企画・プロデュースに従事。現在は少数精鋭部隊のプロデュースチームにて、未来を創るサービスを担当。

21:30～22:00頃終了予定（自由解散）
