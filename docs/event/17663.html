BoinxTV Homeの紹介

ニュース、生放送の情報番組、通販番組などのような映像をお手軽に作れてしまう、BoinxTVの廉価版、BoinxTV Homeの紹介を行います。Home版ではカメラが一台しか使えないなど制限がありますが、少人数でセミナーや宣伝番組を作るにはピッタリなのではないかと思います。他のツールと組み合わせるとUstreamの配信にも使えます。単にカメラで配信するだけの映像ではない少しリッチな画面のUstream配信が行えるので、その辺の事に関しても少しご紹介します。

BoinxTV Home

Quartz Composerハンズオンセミナー

Macの開発環境をインストールすれば無償で使用出来るQuartz Composerのハンズオンセミナーを行います。実際に参加者に試して貰いたいので、ノートパソコンを持ってきて参加出来る人は、是非開発環境をインストールして来て下さい。開発環境のインストールは、Xcode 4でも良いのですが、デベロッパープログラムの契約をしてない人は有料に成って仕舞うので、Xcode 3でも良いです。

2011年7月21日追記：Mac OS X 10.7 Lionの場合、Mac App StoreからXcode 4.1が無料でダウンロード出来ます。そちらのインストーラーを使ってXcodeインストールして頂ければ、Quartz Composerが使用出来る状態になります。

Quartz Composerとは

Quartz Composerは、命令をテキストで書いて行うプログラミングとは異なり、機能毎のモジュールを作業エリアにマウスで配置して、それらのモジュールを接続していく事で、2Dや3Dの映像を作成するツールです。スクリーンセーバーやリアルタイムに音に反応する映像も作れるのでiTunesのビジュアライザーも作成可能です。

何よりも、Quartz Composerは無料で使えるツールなので、Macを使っていて、映像に興味がある人は是非とも使って貰いたいツールです。
