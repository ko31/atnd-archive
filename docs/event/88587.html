【HADOとは？】
HADO（ハドー）は、体を動かして技を発動させ、フィールドを自由に動き回り、味方と連携して楽しむ「テクノスポーツ」。
エナジーボールとバリアを駆使して戦います。

【参加方法】
参加ボタンを押すだけ！
当日ご都合の良いタイミングでお越しください。

【HADO WORLD CUP】
今年もやります、HADOの世界大会！
4月20日にはHADO SPRING CUPが大盛況で幕を閉じました。
８月にはHADO SUMMER CUPが開催されます！
そして年末には賞金総額300万円のWORLD CUPを開催します！
この大会もぜひご参加下さい。
詳細はこちらから
http://meleap.com/worldcup/
