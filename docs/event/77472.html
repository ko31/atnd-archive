セガ×あいちゃれ

サービス開発はじめの一歩！
～現役エンジニアが語る、セガ流モノづくりの極意。～

あいちゃれ 2016のキックオフイベントとして、セガゲームス セガネットワークス カンパニーの渡辺 京介様をお迎えし、【サービス開発はじめの一歩！～現役エンジニアが語る、セガ流モノづくりの極意。～】をテーマとしてご講演いただきます。また懇親会では、セガのスマホゲーム大会も開催します！

渡辺 京介 様（セガゲームス セガネットワークス カンパニー IT統括部 統括部長）

あいちゃれ とは？

ジャンル不問、開発言語不問、実行環境不問のソフトウェア創作コンテストICT Challenge+R（正式名称： あいちゃれ）は、 高校版（小学生・中学生・高校生） と 大学版（大学生・大学院生） の二本立てで同時開催します。
あいちゃれは、立命館だけに限らず「 全国の小学生・中学生・高校生・大学生・大学院生 」も応募可能です！
