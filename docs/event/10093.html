勉強内容

  1. XAMPPについて
  2. インターネットの仕組みから、サーバーでどのようにPHPのプログラムが実行されるか説明します。
  3. PHPのプログラムの書き方の初歩から解説します。
  4. デバッグの仕方
  5. 勉強会はプロジェクターに映した講師の画面を見ながら、自分のノートパソコンにプログラムをします。
  6. ノートパソコンをお持ち下さい（貸し出し用のノートパソコンが１台だけあります（要予約））

  10. 領収書の発行ができますのでお問い合わせ下さい。

XAMPPをインストールしてください

講義にはXAMPPというソフトウェアを使います。
これはパソコン上（Windows,Mac）で、Webサーバーを実行するプログラムです。
（基本的にインターネットで使用されるWebサーバーと同一のプログラムです）

Windows版
http://www.apachefriends.org/jp/xampp-windows.html

Mac OS X版
http://www.apachefriends.org/jp/xampp-macosx.html

インストール後、http://localhost/にアクセスして、インストールが成功したか確認して下さい。

インストールに失敗した方は、phpbeginners@gmail.comにご相談下さい。
