BES勉強会とは

BES勉強会は、以下のような建物エネルギーシミュレーションを行うソフトウエアの勉強会です。

  • ESP-r
  • EnergyPlus (DesignBuilder, OpenStudio)
  • TRNSYS
  • AE-Sim/Heat
  • BEST
  • Ecotect
  • LCEM
  • その他

なお、幹事は東京大学大学院工学系研究科建築学専攻の今野雅です。
