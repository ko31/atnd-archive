11月24日（金）の16時～、フューチャースピリッツ様のセミナールームをお借りして、大阪グランフロントにて「第一回クリエイターと未来をつなぐセミナー～私がクリエイターになったわけ」を開催致します。

「クリエイターと未来をつなぐセミナー」はWEBデザイナー・WEBコーダー・プログラマー・システムエンジニア・ライター・イラストレーター・フォトグラファーなど、様々クリエイターの方々及びクリエイティブに携わるすべての方々のためのイベントです。

  • クリエイターとクリエイターを探している企業・個人をつなぐ
  • クリエイターとクリエイター同士をつなぐ
  • クリエイターと新しい知識・ノウハウをつなぐ
  • クリエイターと人材採用をしている企業をつなぐ
  • クリエイターが活躍できる未来・将来をつないでいく

このような目的を持って行われるのが「クリエイターと未来をつなぐセミナー」です。

第一回はクリエイターのキャリアや人生について考えるイベントを行います。漫画家・イラストレーターとして活躍しながらも就労支援事業所で働きながら、ご自身で講演活動も続けられているオサさんをお招きし、お話をしていただきます。

第一回クリエイターと未来をつなぐセミナー～私がクリエイターになったわけ詳細

当日プログラム

15時30分～ ：開場

16時00分～ ：セミナー開始
オサ（漫画家・イラストレーター）

私は漫画家です。そして身体障害者です。15歳で若年性関節リウマチを発病。体全身の関節に激痛が走り体が不自由になった事で私の人生は1度、15歳で止まりました。その後『仕事』や『生活』に関してはとても難しい大きなハードルばかりでした。

みんなと同じでないといけないとか、ダメージケアにばかりに力を入れてその人の良いところを全くいかさずに生きてませんか？見る角度を変えてやり方を変えたらやりたい事をやって生きていける！と私は思っています。

身体障害者でも自分のやりたい仕事、夢に挑戦する事、なんか出来るよ？って事を実践し、病やケガ、心が苦しんでいる人に伝えたいのです。『万策尽きたと思ってもまだなんかあんで！？』という勇気が出るお話を私の経験を元にお話させて頂きます。

気負わず、気張らず、ゆるゆるとお話を聞いて貰えたら嬉しいです。是非ご参加ください！

障害者本人、ご家族、教育関係、福祉関係者に限りません。興味を持っていただけた色んな方に聞いていただきたいです！

17時30分～ ：講師・参加者同士の交流

～18時00分 ：イベント終了
