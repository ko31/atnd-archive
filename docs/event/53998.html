description

2014年１０月1３日（月･祝）藤岡Ｆ２プラントにてサバイバルゲームを行います。（全面使用予定）

どうも、主催こしがやです(‘ω’)

再びですが栃木の藤岡にあるＦ２プラントを借り切ってゲームしたいと思います。
今回も全面使用なので皆さんで遊べたらなと思います。

基本的に身内と初心者メインのぬる～いゲーム会です。
一応、主催の面識のある方メインの身内向けのゲームとなっておりますゆえ、ご友人を誘われてご参加なされるようでしたら、集まりの趣旨を理解してご参加いただけますようお願いいたします。

尚、基本的にセミオートでのゲーム進行となります。

■参加募集締め切り　第一次：2014年９月２9日第二次:2014年10月11日
　　　　　　　　　　
■場所：栃木県　F2プラント

■定員：４０人

■ATNDに登録されない（できない）ゲスト参加の方をお連れになられる場合は、
お手数ですがコメント欄に人数とその方のお名前（HNで可）をご記入頂くようお願いします。

■必要な物

参加費
　男性：５､000円
　女性･18歳未満4,000円
　(※18歳未満の方は保護者の方の承諾（必須）と主催との「交流・面識」が必要です。）

エアガン：電動・ガス・エアーコッキング式のエアソフトガン
　　　　※持っていない場合フィールドでレンタルあります。
　　　　（レンタル料金は こちら ）

昼食：お弁当の注文が可能。一食500円

※レンタル装備ご希望の方は募集締め切りまでに
　コメント欄に表明してください。

使用BB弾：６ｍｍ完全バイオＢＢ弾0.25ｇまで。
　
装備：サバイバルゲーム用ゴーグル（シューティンググラスは自己責任で）、顔面を保護するマスク等推奨。

　他に、怪我を防止するのにグローブ手袋、ニーパッド、エルボーパッドなどが役に立つので、
　ご用意されると宜しいかと思います。

　フィールド廃工場を利用した施設で機械等があり、砂やＢＢ弾で非常に滑りやすい場所も存在します。
　滑りにくい底の靴を推奨。

　任意ですがお名前（HN）が判るような名札をご用意装着いただければ幸いです。

■レギュレーション

パワー制限は銃刀法に定める規制値以下で、

エアーガンは、0.20g BB弾　初速98m/s未満、0.25g弾　初速87m/s未満の初速のもの
※減速アダプターの使用禁止
※レーザーサイト原則使用禁止

フリーズコールタッチ、ナイフアタック不可（痛くなさそうな末端を撃って下さい）。
　基本的にブラインドファイア（遮蔽物より銃のみ出しての発砲)禁止。
　外部ソースはデフォルト仕様以外は

※跳弾狙いでの天井や壁へ向かっての発砲禁止
※過度の上方曲射禁止（M203やM302等のグレネードランチャーは除く）
※トルネードグレネードは遠投禁止。軽く転がす程度の運用推奨

ゲームは基本セミオートで進行させていただきます。
弾数制限などは行いませんが、当日のゲーム種類内容によっての制限はあります。

■諸注意

　買い出し等でフィールド外に出られる際は装備、BDU等着用のまま外出されないようお願
　い致します。

■ちょっとしたお願い
諸事情で参加を伏せたい人も居ると思いますので、終了後ですが
個人単位で「今日サバゲして来た」と言うのは自由ですが、
誰が参加していたかに関しては『触れない』方向でお願いします。
TwitterやMixi等で「○○さんお疲れ様でした」も出来るだけ空気読んで頂けると助かる人も多いと思います。
（自分が参加していた事を表明する事に問題ない人同士のやり取りについては特に問題ありません）
交流等は本ページコメント欄をお使いください

フィールドまでのアクセスは こちらのほうを参照

鉄道ご利用の方は東武日光線藤岡駅から徒歩30分くらいです。
バス・タクシーご利用されたほうが楽かと思います

バスは藤岡駅より市営のふれあいバスが近くまで運行しているようです。
http://www.city.tochigi.lg.jp/hp/page000008000/hpg000007733.htm
藤岡線の谷中湖方面行きに乗り原向高間公民館前バス停で下車すると目の前のようです。
http://www.city.tochigi.lg.jp/ct/other000025000/time_table%28fujioka%29%28h25.10%29holiday.pdf

■天候中止に関して
基本的にはインドアフィールドなので雨天でも決行いたします。
