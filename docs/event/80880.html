イベント概要

エンジニア目線の求人サイト「Forkwell Jobs」主催イベント第22弾！

今回はユーザ数160万人を超えるポイントサイト「ハピタス」を運営している、株式会社オズビジョンとのコラボイベントを開催します！

出勤日数、時間、場所など可能な限り社員の自主性に任せるという『完全自立型勤務』制度を導入しているオズビジョン社。

その中でエンジニアはどのような開発環境で成果を出しているのか、エンジニア以外の職種のメンバーとの関わり方など、 オズビジョン社ならではの事業会社での働き方に迫ります。

また、SIerから事業会社へのキャリアチェンジを考えたときに、 自社サービスの改善・運用の経験を求められることが多くあります。

これらの経験を現状持っていない場合、
どうやってその文化や考え方を身に付ければ良いのか。
また、今までの経験をどのような形で活かすことができるのか。
面接でのアピール方法は？　など

元SIer出身のエンジニアより、SIerから事業会社へ転職する際に気をつけるべきことについてもお話いただきます。

これから事業会社側への転職を考えている方にとって、 『ユーザ目線を持つことはどういうことか？』を実感でき、あらためてご自身のアピールポイントも振り返ることができる内容を予定しております。

▼こんな方におすすめ

  • 時間や場所にとらわれず、自分の能力を発揮したいと考えている方
  • 実際に顧客の声を聞いてサービス開発に活かしたい方
  • 今後自社サービスを開発したいと考えている方
  • 現在PHPでの開発をしている方

コンテンツ内容

第一部：オズビジョン社の開発環境

  • オズビジョン社の中でのエンジニアの働き方
  • どのような場面でエンジニアが関わっているのか？
  • エンジニア以外（デザイナー、インフラ、ディレクター、営業）との仕事上の関わり方について　など

第二部：実際に働いているエンジニア達によるパネルディスカッション

  • ユーザヒアリングをして得られた経験、気付きは？
  • 今までの経験の中で何を御自身の得意分野として活かせているか？ 　
  • 入社してみて感じた良かった事、前職での経験が活かせる事は？ 　　
  • この会社に入社したからこそ得られたと思う経験について　など

第三部：懇親会

オズビジョン社のエンジニアとお酒や軽食をお召し上がりいただきながら気になることを個別に聞ける会です！
