description

面白ビジコンとは？

「今までにないユニークで画期的なビジネスか？」「ビジネスをする自分たちがワクワクするか？」「他人に話したくなるビジネスか？」私たち面白法人カヤックは、こんな点にこだわって仕事をしています。学生の皆さんとも同じテーマで議論する、そんなビジネスコンテストです。

当日は「飛び入りプレゼン枠」もあります！

●飛び入りプレゼン枠、「下克上プレゼン」とは？
つまりは「敗者復活枠」です。
面白ビジコンの予選で落ちてしまったチームはもちろん、申込しなかったチームの参加もOK！
何が起きるかわからない、当日の「おたのしみ枠」です。

●応募資格
小学生から大学生、その他専門学校生などすべての学生が応募可能です。
もちろん、東軍・西軍の最終プレゼンテーションに惜しくも敗れてしまった、今回の面白ビジコン参加者も応募可能です！。

●下克上プレゼンの流れ＠7月14日（東西対決当日）
・13時15分～13時45分：課題資料受付
・14時55分～15時：審査結果発表
・15時～16時15分：6番目のプレゼンテーターとして発表

●申し込み方法
（１）
まずは、ATENDで東西対決への参加をしてください。
http://atnd.org/events/30659

（２）
「面白いビジネスアイデア」を1つ
5～15ページ程度でA4用紙に印刷して会場に持参ください

（３）※下克上プレゼンに合格した場合
面白ビジコン・最終プレゼンテーターとして発表していただきます。
そのため、提出資料の他に、当日プレゼンテーション用資料をご用意ください。
使える機材は、windowsのPowerPoint2010かmacのoffice2011のどちらかです。
配布資料やレーザーポインタなどはありません。

●審査基準
下克上プレゼンに登壇できるのは１チームのみです。
審査基準は次の通りです。自分たちが面白いと思うアイデアで、かつ、それを聞いた周りの人が友だちに伝えたくなるような、面白いアイデアを募集しています。
もちろん、そのサービスが実際に使われるイメージが沸くか、ビジネスとして、どんな人にどういう価値を提供し、どれくらいのリターンを得るのかなども総合的に考慮いたします。

1.ユニークか？
2.人に伝えたくなるか？
3.ビジネスとして価値換算できるか？
