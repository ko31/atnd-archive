description

＜講座のポイント＞
　大気圧グロー放電技術を用いれば任意表面のクリーニング、プラズマ改質、薄膜堆積等が真空機器を使用せずに可能になる。言いかえればこの技術を使えば、表面処理装置のインライン化が容易に達成できる。本講座では日本発のプラズマ技術として世界中に広まりつつある、本技術の紹介と表面処理への応用例を多数挙げて、分かりやすく紹介する。

＜プログラム＞
１．大気圧グロープラズマの基礎
　　1-1低圧グロー放電と大気圧グロー放電
　　1-2大気圧グロー放電の成り立ち
　　1-3希ガス以外の大気圧放電
２．大気圧グロープラズマ技術
　　2-1各種大気圧プラズマ装置
　　2-2平面へのプラズマ表面処理
　　　2-2-1異種ポリマーフィルム同士の無接着剤型接着法
　　　2-2-2 OH基付与の為の薄膜堆積
　　　2-2-3　他
　　2-3．粉体表面処理への応用
　　　2-3-1 有機顔料および酸化チタン粉体へのシリカ堆積と分散性
　　　2-3-2 ポリマー粉体への化学修飾
　　　　(1)酸化および親水化
　　　　(2)臭素化
　　　　(3)アミノ化
　　　2-3-3ポリマー粉体のPEG薄膜コーティングと分散性改善
　　　2-3-4バイオ粉体処理、他
