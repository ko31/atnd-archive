description

あなたの周りに、こんな人はいませんか？

　　自信に満ちあふれ、
　　苦しい思いで頑張っているように見えないのに、
　　目標を次々と達成している人。

目標を達成できる人とそうでない人は何が違うのでしょう？
その人に揺るぎない自信が感じられるのなら、その人は
「価値観」が明確になっているのかも知れません。

価値観を明確にして言語化すると、目標達成がグッと近づきます。
明確な価値観に従って行動していくことで、
目標達成のハードルを下げることができるのです。

私自身、企業の人事・教育部門に28年間携わり、
社員教育やチーム運営に関する悩みも数多く見聞きしてきた中で
つくづく感じるのですが、

組織（チーム）のリーダーは特に、自分の価値観を
明確にしておく必要があります。

揺るぎない自信がメンバーに伝わり、
チームのパフォーマンス向上に好影響を与えるのです。

本セミナーでは、カードゲーム感覚で、
楽しみながらあなたの価値観を明確にしていきます。
相手の価値観もリアルでわかり、自分との違いに気づくことができます。

自分のために、チームのために。
あなたの価値観を明確にしていきましょう！

★プログラム内容
・圧倒的な行動力を生み出す「３つの要素」について考える

・「勝ち」について、改めて考える

・価値観発見ワーク
　　カードゲームの要領で、楽しみながらあなたの価値観が明確になる

・チームのパフォーマンスを低下させる「５つの言葉」とその対策
　　こんなセリフはチームの危機？５つの「無」
　　価値観との相乗効果でチームを元気にする「３つのツール」

★こんな人のお役に立てます
・自分にとって大切なものを知りたい方
・揺るぎない自信と信頼感を手に入れたい方
・身近な人とのコミュニケーションを円滑にしたい方
・影響力を発揮したいリーダー、マネージャー
・行動力をもっと高めたい方
・会社の研修とは違った切り口で、チーム運営について学んでみたい方

この他、本セミナーに興味のある方ならどなたでも大歓迎いたします！

★受け取れるもの
・自分の価値観が明確になる
・メンバーの価値観を理解し、寛容になれる
・他人の目が気にならなくなる
・あらゆる選択に迷いがなくなる
・自分と他人は違っていい、と心から実感できる
・友人や家族、同僚とのコミュニケーションが円滑になる
・勝利（目標達成）の確信が高まる

★講師プロフィール
石津恵子（いしつけいこ）

チーム活性化ナビゲーター。

東証一部上場製造業会社人事・教育部門勤務28年
新人教育の企画・運営（対象者12,000人）
およびOJTトレーナー（年間150名）に対する教育業務に携わる。

OJTトレーナーのサポートを通じて、現場の教育に関する悩み、
チームづくり・再構築への悩み解決に関与。

この経験から、やる気のない社員のモチベーションをあげ、
チーム全体の目的遂行力を向上させる独自の理論
「５ワード３セット～最高チームのつくり方～」を確立。
チームの立て直しはもちろん、活性化にも大きく貢献する。

自らの、プロジェクトにおけるチームビルディングの失敗経験を活かし、
メンバーに寄り添いながら「動いてもらう」ためのノウハウを提供している。
