Webデザイナ・Webプログラマの為のAndroidアプリ開発入門

HTMLやCSS、JavaScriptでのウェブサイト制作を知っている人向けに
「PhoneGap」フレームワークを使ってみながら開発環境の構築から
Androidのネイティブな機能を使った簡単なアプリケーション開発までをやってみるハンズオンの勉強会です。

補欠登録になってしまった方へ

今回残念ながら、定員オーバーで補欠になってしまった方にお知らせです。
この度アバンドさんよりお声を頂き、同趣旨でパワーアップした第二回PhoneGapハンズオン勉強会を開催する事になりました。

「JavaScriptでAndroidアプリ開発」ハンズオン！
～PhoneGap + HTML5 + jQueryMobile + enchant.js～で作ってみるAndroidアプリ開発入門
