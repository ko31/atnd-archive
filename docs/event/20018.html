Windows Phone 7ハンズオンin広島

開催概要

　日本でも発売されたWindows Phone 7.5搭載スマートフォンのアプリケーションをハンズオン形式で実際に触りながら学ぶセミナーです。
　今年中に開発の基礎を身につけ来年のブレーク(？)に備えましょう！！

参加費：無料

持参するもの

　Windows Phone SDK 7.1 RC 版のインストールされたPC。

　*SDKのバージョンは今後新しくなる可能性があります。
　
　会場はネット環境がございませんので、Wifi機器などがあると便利です。
　Wifiなどの接続環境を開放いただけるかた歓迎！！

定員：8名

　ハンズオンという形式とがっつり身につけて欲しいので少人数で行います。

想定するスキル

　C#の開発経験者ないし、他の言語での開発経験がある方。
　スキルはないけどWindows Phone 7が好きすぎる方。
