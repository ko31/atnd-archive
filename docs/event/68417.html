◆第１部：キャリアデザイン手法◆

技術教育制度が存在する会社は多数あります。しかし、それ以前に重要なことがあります。それは 今後このＩＴ業界で一人のエンジニアとして自分自身をどのような方向性にもっていくのか、そのためにはどのような技術や知識を身に付けて、目標に辿り着くのかを描けなくてはなりません。

しかし、そのことを教えている会社は皆無といっても過言ではありません。一度現場に入って仕事をすると、アッという間に3年、5年が経ちます。具体的な目標のないまま、方向性を見失っているエンジニアの方も多いと思います。これは明らかに『会社の責任』です。

まだ遅くないです！今後のキャリアデザインの重要性について、一緒に考えていきましょう。次世代を担うITエンジニアの皆さんに、より成長していただきたいために、キャリアデザインの事前準備や具体的な方法を伝授致します。

◆第２部：成長したい人が次々転職してくる会社とは◆

成長したい人が次々と転職してくる会社では一体何が行われているのか。それは、エンジニア個々の目標や方向性を最優先に仕事をアサインしています。エンジニアの希望にそぐわない案件には一切アサインしません！他社では上からの業務命令により、次に担当するプロジェクトや作業が“勝手に”決められてしまいます。

なぜそんなことになるのか。それは、エンジニアの希望や方向性などに関係なく、会社利益や会社都合が優先されてしまっているからです。成長したい人が次々転職してくる会社では、まずエンジニア自身にキャリアデザインを描いてもらい、それを軸に自己実現に向けて、エンジニアと営業と経営者とが『三者間連携』することで、より確実でより早く、目標達成できるような体制を整えています。

また、ブラック企業で横行している「残業代を全額支給しない」という問題もその成長企業にはありません。

◆講師紹介◆

フリーのシステムエンジニアとして20年以上の経験をもつ。自身のキャリア設計の重要性を知り、キャリアアップにつながる仕事を自分で掴み取ってきた。その経験を活かして、いかにITエンジニアとしてのキャリアを形成していけばいいのかを研究してきた。
