セッション内容

  • YAPC::Asiaにて発表内容について
    Fukuoka.pmとしてYAPC::Asiaにて発表することが予定されています。みなさんの経験談などを元に作成できたらと思います。
    • Fukuoka.pm活動内容について
    • 福岡のPerl事情について
    • Perl活用例などお聞かせください。
  • 今後のFukuoka.pmの運営について
    • 主催者の明確化について
    • サイトなどの管理についてご説明とご確認
  • 従来のPerl Workshop
    >山本さん
    テスト用データベースをクリーン＆クールに起動する Test::mysqld と Test::postgresql の紹介と実際に私が現場で使っている例についてです。（ Test::Httpd::Apache2 もあるよ ）

発表希望の方はMLにてお知らせください。
※時間が足りない場合又は発表者がいない場合には中止いたします。
