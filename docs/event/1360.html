description

福岡および九州近隣に住む、携帯電話やスマートフォンを中心としたデジタルデバイスが好きな人を対象に、酒を片手に楽しく、そして熱く語り合う機会を持ちたいということで「スマートデバイストーキング九州(S.D.T.Q.)」というイベントを実施しています。
入手したデバイスを互いに披露したり、一緒に問題を解決したりできる機会として、また同好の方々と気軽に話を出来る機会となればと思っております。
各プラットフォーム（Windows Mobile、Symbian OS、iPhone OS、Android、他）のアプリケーション開発者の方とそのユーザーの方との意見交換の場にもなれたらと思います。


スマートフォンやデジタルデバイスに対する深い知識、所有している端末・機器の数や珍しさ、年齢や職業、性別、一切を問いません。
携帯電話あるいはスマートフォンやデジタルデバイスが好きだという方であればどなたでも歓迎します。
福岡市内での開催となりますが、他県（九州各県をはじめ日本全国、あるいは全世界？）からの参加を歓迎しています。


今回は先日まで中国に赴任されていた方が帰国なさいましたので、中国の様々な事柄についてご教示いただけることになりました。
携帯電話などのデジタル関連分野以外にもいろいろとお伺いすることが出来るかと思います。
講演というかたちではなく、酒の席で様々なお話を伺うという緩いかたちになりますので。念のため。


こちらでも参加表明承りますが、URLリンク先の参加表明手順に沿ってご連絡いただけますと非常に助かります。



参加場所、集合場所の詳細はある程度の参加人数固まり次第逐次お知らせいたします。
会費は3500円を予定しています。


たくさんの方のご参加をお待ちいたしております。
