以下、流用です。

「愚直に組み合わせるのが創造性。失敗してもそれがそれ。」ということをよく考えてます。そこで今までやっていた4つくらいのネタなどを無理やり一度にやってみようと思いました。破綻するかもしれませんしこれ何の会だよっていわれるかもしれませんね。バランスなんてとりません。

== 1. A DIS-COMMUNICATION NIGHT ========

「〇〇様

パーティーギバーという言葉がございますが、
およそこれほどお前らに似つかわしくない言葉は御座いません。

そこで、君たち二人で4月の東京カーネルパニックのパーティーを
オーガナイズしてみていただけませんでしょうか。

よろしければ１を、そうでない場合は２とご返信ください。

パイ投げの機運が高まっておりますので、できたら実施していただけると助かります。
日程的には5月21日でいかがでしょうか。

基本的にはみなさん定期的にパーティーに来てもらえる人がいるとおもうので
それなりにまとまると思いますが、君たちならやってくれると信じています。」

（ちなみに、今のところの反応として「ディスコミュニケーションナイト」みたいな
ものが打診されてしまっており、それは大変に面白そうだと思っています。
最初の返信は「あ、あたしできない絶対、無、無、無理、ひいっ」といった
ホイミかけられたゾンビみたいなものでした。期待できそうです。

== ネタ２　アクションペインティング・パイ投げ =========

【 パイ投げ？ 】
童心にかえり、参加者が口々に「見ているだけではまったくその人生を意味あるものにしかねない、あの体感を味わえないよ！」と興奮して話したり、「自分を汚さず観てるだけでは、あの楽しさは分からんよ。」と哲学命題の静かさで語ったりしてしまう器の広いイベント。会の一企画として1時間ほどやります。パイ投げの開催時間は20-21時くらい。パイは個数に限りがあるので、参加者は必ずパイ参加の旨をatndへお知らせください。

というのを連綿と7回もやってきました。今回は食紅等を駆使して、佐藤雅彦研などで活躍する才能ある若者に、色つきホイップを作ってもらい、ポロックなどの「アクションペインティング」に近いなにかにならないかなーという実験を行います。世界発の試み。キャンパスは闘技場であるそうですが、パイ投げは文字通りそうです。。
アクションペインティングとは、ペンキをどろっと床に落としたり筆に絵の具をつけてピットふるやつです。厳密には違うけど、そういうジーンズとかみたことないですかね？アレです。

手ぶらで参加してください。費用については後述。

　== ネタその３ フリーランス的な交流会 =========


今探している物、提供できるもの、こころの中に用意する。紙を用意するのでそれに書いてもらう。席替え的に時間決めて話ししたりする、的なノリが１時間に１０分位、掛け声でやられたりします。なんかこう「出会いとかうぜー」という人たちが出会うところに集中したいです。パーティーとは本来そういうものだと。

あとは、フリーランスで、１０年やってきた人、１年目の人、なってみたい人、就職しようとしてる人などで情報共有したりサバイブすることについて話すなど。仕事はどこからくるのか。儲けはどこからくるのか。資金繰り、営業、作業、集中コントロール、自由の意味、場所、人脈などについて。起業、スケール（サイズ）、投資、などについても。仕事はどこからくるのか、なんていうけど、あなたはどこからきてどこからいくのか、というときにそれはフリーランスという言葉の周りにおいて話されると、それは仕事はどこからくるのか、にもとても近い。

※こういう企画とDIS-COMMUNICATION NIGHTをぶつけるのはとても面白そうではないですか。初対面ですが失礼します。といって一回必ずDISってもらおう。すげーやりずれーのがいい。

= ネタ 4 次世代情報流通云々　aka 電子出版　===========

ご存知か、会場の東京カーネルパニックはスタジオボイスさんの件でにわかに盛り上がっている
Publishing Guevara、出版ゲバラという計画の基地になっており、世界最先端の情報が終結され
アメリカのギークアンカルトアンドビジネスのスタートアップが熱狂の中で革命を起こしていったような
ベンチャーをやるぜということで、その表象的な「騒いでいる」イメージ作りとは裏腹に
電子出版ということに取り組んでいる日本の集団のもっとも深く緻密に設計され、経済的にもシステム
的にもとても事業として王道的かつ革新的な取り組みを行っています。おそらく僕たち以外に
日本の出版を変えられる主体はないでしょう。お話をしましょう。

出版ゲバラの子らの顔合わせも行われるようです。
