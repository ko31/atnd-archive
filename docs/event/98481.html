医薬品業界の再編に伴い、MR職の採用状況にも変化が出てきています。
2000年代のMR職は転職を通じてキャリアアップを望めましたが、現在はそこまでの勢いを感じられません。
そこで、MRとしてご活躍されながらも“次のチャレンジ”をお考えの20代・30代の方へセミナーを開催します。

■MRからキャリアチェンジできる職種
■求められる要件
■仕事内容
■面接のポイント など

当日、お集まりの方と情報共有しながら、より前向きになれる場をご提供できればと考えております。
是非、お気軽にご参加ください。
