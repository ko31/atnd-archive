毎回好評のクリエイター交流会【クリのみ】も25回目！

"クリエイターのみ" が集まる "クリエイター限定飲み会" 【クリのみ】。
2012年から３ヶ月ごとに開催中（1月、4月、7月、10月）、
のべ 2,160人 以上の方とクリエイティブな出会いがありました。

次回クリのみは2018年7月27日（金）夏の宴です！

クリのみ公式サイト

過去の参加職種は、デザイナー、イラストレーター、フォトグラファー、ライター、アニメーター、ディレクター、プロデューサー、ゲームクリエイター、3Dデザイナー、プランナー、シナリオライター、編集者、作家、漫画家、アニメーター、絵コンテ作家、映像制作、番組制作、映画制作、映画監督、脚本家、建築家、技術、設計、書道家、造形家、彫刻家、作曲家、研究開発、プログラマー、ファインアート、美大生、専門生など

  • 職種がらシャイな人が多いけど、だからこそ同じ温度で安心して楽しめる会です。
  • 参加者もほとんどが一人参加、半数が初参加なので話しやすく、
  • 初対面がおおいけど、似た仕事なのでとても打ち解けやすいです。

  • 日時：2018年7月27日（金）　19:45〜22:00（開場19:30）
    ※仕事柄、遅刻は可ですがキャンセル払い戻しはできませんので頑張って来てくださいね。とーっても楽しいですから！

  • プチ講演：19:50〜20:10予定（講演者未定）

  • 場所：新宿区歌舞伎町 　

  • 飲み代
    　クリエイティブ系学生：3500円
    　クリエイター社会人：　5500円

  • 飲み放題（22時まで、ビール、ハイボール、カクテル、ソフトドリンク等色々）と軽食（パスタ、ポテト、サラダ等、おつまみ程度ですので、食いしん坊は歌舞伎町でがっつし食べてからの参加を。） 　

  • 定員：90人（第24回は満員90名参加）

  • 参加資格：デザイナー、イラストレーター、フォトグラファー、ライター、アニメーター、ディレクター、プロデューサー、ゲームクリエイター、3Dデザイナー、プランナー、シナリオライター、編集者、作家、漫画家、アニメーター、映像制作、番組制作、映画制作、映画監督、建築家、技術、設計、書道家、造形家、彫刻家、作曲家、研究開発、プログラマー、ファインアート、美大生、専門生などクリエイティブ職

    ※以下職種を兼任の方、以下行為目的の参加は固くお断りしております【ダメ絶対】：各種保険営業、不動産業営業、ネットワークビジネス・MLM・マルチ商法・ネット商材などの勧誘、オーナービジネス勧誘、先物取引・投資勧誘、宗教勧誘、スーツ・宝石の販売、交流会主催者＆関係者、ナンパ・スカウト

趣旨はこんな感じです。

  1. クリエイター系の人たちと同じ温度でコミュニケーションできる
  2. あまり外に出られない仕事だから新しい縁があるといいね
  3. 他業種のクリエイターがいるから勉強になるね
  4. 学生さんたちにプロの経験や体験を話してあげよう
  5. ゆくゆくはみんなで協力してクリエイティブから日本や世界に貢献してけたらいいね～

なのですが、まず一番大事なのは
「楽しく飲むぜ！語るぜ！」ということなので気軽に来てくださいね。
