主催：株式会社FUNPLE STREAM

第10回のFANBATはいま話題のPUBG、「PLAYERUNKNOWN’S BATTLEGROUNDS」で実施します！
ラフにみんなでパーティを組んでスクワッドで遊ぼうと思います。
PUBG仲間を作りたい人、オフラインでスクワッドを遊びたい人
初心者の方ももちろん大歓迎です！

日時：2017年6月10日（土）13時～17時(12時30分受付開始)
場所：アイカフェAKIBAPLACE店
http://www.kaji-icafe.com/area/kanto/akiba/
会場利用料：1,300円（税込）
参加定員：40名

【内容】
会場ではランダムに4名チームをつくり、それぞれのチームにはスクワッドモードでプレイしていただきます。参加した方にはいろいろな方とチームを組んでスクワッドを遊んで頂きたいと思いますので、イベント中何回かチームシャッフルをしながら合計5～6戦ほどする予定です。当日はバトルの成績をポイント化してランキングにし、上位の方には賞金や賞品を進呈いたします。

※カスタムマッチは行いません
※参加者ご自身のアカウントを使用します
※当日ランダムで4人1組にチームに分け
スクワッドモードでプレイし、順位によるポイントを競います
※チームでの参加は不可
※マウス・キーボード持込：可（再起動が必要なものは不可）

【賞金・賞品】
当日獲得ポイント成績上位者に賞金を進呈。
1位　5,000円
2位　3,000円　
3位　2,000円
Webrootアンチウイルス for ゲーマーほか協賛企業様賞品もご用意しています。

※賞品提供：株式会社FUNPLE STREAM

※重要
当日はネットカフェのPCを使ってPUBGをプレイしますが
Steamで新しいPCにログインする際にはメール認証が必要になります。
当日steamに登録しているメールアドレス・パスワードが分らないとログインができません。
事前にメールアドレスとパスワードを控えるなどお願い致します。
