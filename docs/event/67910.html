 新規事業検討の経営者様・起業をお考えの皆様

今回は、新規事業を検討している経営者様や起業をお考えの皆様から非常に相談が 多い内容のため、このセミナーを企画致しました。

皆様、次の事実をご存じですか？

　起業の約35%は、3年以内に倒産又は解散　

　起業の約85%は、5年以内に倒産又は解散

　新規事業の成功確率25%～50%と言われている

つまり、起業も新規事業も5年以内に残っていることが非常に少ないということです。

私共も起業家の方や新規事業立ち上げの支援を行っておりますが、中には支援することがもう少し早ければと歯がゆく感じることも多いのも事実です。

 

その失敗事例を検証すると実は、失敗の理由は５つのポイントであることがわかりました。

そこで、今後、起業される方や新規事業を立ち上げられる方には、そのような失敗を起こして頂きたくないと考え、このセミナーを企画致しました。

 

今回は、ドリームゲートアドバイザーの5名が福岡・四国・大阪・東京の4箇所でセミナーを開催致します。ドリームゲートとは2003年に経済産業省の後援で立ち上がった、起業家や経営者様の経営相談を専門家がオンラインで解決するサイトであり、全国の各分野の厳選された専門家が相談にあたっております。

↓↓↓ドリームゲートサイト（詳しくはクリック）↓↓↓

今回は特別にその専門家の5名が集まり、ネットだけでなく、リアルの場でもお客様にノウハウ提供し、個別相談も行い、悩みを解消するセミナーを開催致します。

今回の4会場限定というコラボ企画であり、今後の開催は現在予定されておりません。

 

この機会に皆様の悩みを解決したいのです。

このセミナーは次のような方にピッタリの内容です。

　何をしていいかわからない

　ブランディングの仕方がわからない

　資金調達のやり方がわからない

　人の採用や教育のやり方がわからない

　どのように売っていったらいいかわらない

 

セミナー受講のメリット

 

　起業・新規事業の重要な５つのポイントを理解できる

　集合個別相談会では、悩みを専門家に相談できる
