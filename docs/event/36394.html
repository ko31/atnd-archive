Perl でも Python でも Ruby でも何でも良いので、新潟の開発者が自分の好きなLLについて発表するイベントです。

どの言語がLLなのかは各自の判断でお願いします。C++とかが来ても驚かないと思います。

発表枠は5秒から30分まで、お好きな枠をお使いください。
「Python で Hello World 書いてみた！」や「自分とPerlの出会いを一言で」、「最近作ったプロダクト」など、どんなネタでも構いません。

発表なしも可です〜が、できるだけ多くの参加者に発表していただきたいです。どんな内容でもウエルカムなので、ワイワイやりましょう！ワイワイ！新潟にこんな人がいるんだ〜とお互いに知ることができたら良いなと思います。

詳細な予定は随時追加します！質問や発表の希望等は @jewel_x12 までお願いします！
