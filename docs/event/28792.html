気負い・気取り・着飾り不要！
でも新しい出会いにドキドキ♡わくわく☆な農作業の体験をご一緒に。
「知ってるつもり」から「体験する理解」へ。
眠気まなこな五感をフルに使って小さな営みの世界を直に触れてみませんか？
ご案内したいのは『笑い溢れる畑の生きものたちの暮らしぶり』です。
受け身だらけの農業ファンタジーイベント？ノンノン。

〜 農っテク友活 〜
【日時】
５月２７日 　日曜日

＊ネタ仕込みに奮闘するため１０時半より早い到着はお控えください。
イベント開始 : １１時（終了予定 : １５時〜１５時半頃）
【参加費】
お一人：３８００円

楽にしゃがんだりできる, 汚れても大丈夫な靴と服。
天候が怪しい場合, 雨合羽も持参ください。
＊要着替えの方は１０時半〜１０時５０分までにお越し下さい。

♡軍手, タオル, 暑さ対策グッズなど
