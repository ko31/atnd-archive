　１. セミナー内容
　───────────────────────────────────────────────────────

　本セミナーでは、日本マイクロソフト社のエバンジェリストである講師とともに
　Visual Studio Professional 2012をプリインストールした、
　「お名前.com デスクトップクラウド for Windowsアプリ」を使って、Windowsアプ
　リ開発を体験していただきます。

　また、Windowsストアへの申請方法もレクチャーいたしますので、アプリ開発～スト
　ア登録までの方法が学べます。

　さらに、「お名前.com デスクトップクラウド for Windowsアプリ」なら、セミナー
　終了時の状態から、ご自宅や会社から開発を再開することもできます。

　このセミナーをきっかけに、今後盛り上がっていくWIndowsアプリ開発の世界に参入
　しましょう！
　Windowsアプリ開発にご興味がある方は、是非、ご参加ください！
