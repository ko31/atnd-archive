相続のこと、後回しにしていませんか

相続のイロハを相続専門ファイナンシャル・プランナーがわかりやすく解説します。以下の項目に1つでも当てはまる方は、是非受講してください。

  1. そもそも相続問題って何？
  2. 相続が「争族」にならないか心配だ
  3. できるだけ相続税を抑えたい
  4. 老後のお金がいくらかかるかわからない
  5. 相続のことを誰に相談すればいいかわからない
  6. 生前贈与したいけれど、子・孫に浪費癖がつかないか心配だ
  7. 遺言書の書き方がわからない
  8. 空き家問題が心配だ

講師：羽生信彦（はしょう・のぶひこ）

たすくFP事務所代表。ファイナンシャル・プランナー、宅地建物取引士、相続診断士。
広島大学卒業後、スポーツ記者志望で日刊スポーツ新聞社に入社するも、配属されたのが総務部で、いきなり挫折を味わう。同期が阪神タイガースやJRA、海外サッカーなどの華やかな取材現場で活躍する中、給与計算や社会保険、クレーム電話対応、アルバイトの契約解除など、新聞社の「陰」を担当する。
2009年、当時最年少で次長に昇格するも、給与カットや出張費カットで溜まった社員の不満のはけ口となる。2014年、人の役に感謝される仕事をしたいと、自ら作った早期退職制度に応募し、ファイナンシャルプランナーとして独立。相続や確定拠出年金（401ｋ）のセミナー・相談業務を中心に活動している。
