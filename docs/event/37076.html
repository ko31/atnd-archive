■What’s“Extract“?
クラブミュージックと同人音楽を同じフロアで聴ける機会が増えてほしい…
普段、ストリーミング配信を視聴することはあっても中々現場で見る機会がない…
画面の向こう側で見てみたい。仕事や学校で夜のイベントへ行くのが難しい…
そんなあなたへ、ひと癖もふた癖もある出演者たちがお贈りするクラブパーティー!!!

“音同人×クラブ”をコンセプトに、アツい音をお届けします！クラブに行ったことのある人もない人も、みんなで家とは違う、フロアの大音量を体感しよう！ 音楽好きなら、絶対楽しめる！

※※※Extract一周年企画として、来場していただいた方々に出演者による限定MixCD(出演者全員収録予定)を配布させて頂きます！是非ご来場ください！※※※

※未成年者の飲酒・喫煙を防止するため、受付にて年齢確認を行います。ご来場の際は年齢を確認できる身分証をお持ち下さい。
※前回と違いオープンが13時となっておりますのでお間違えの無いようご注意ください。

■FAQ
Q:Extractって、どんなパーティーなんですか？どんなジャンルが流れるんですか？
A:ハウス / トランスを中心にエレクトロ / プログレッシヴ / テクノなどの４つ打ちクラブトラックや同人音楽（東方アレンジ、オリジナル）がメインとなります。
当日、気になった曲があれば出演者に質問してみましょう！

Q:未成年でも遊びに行って大丈夫なの？
A:喫煙、飲酒をしなければもちろんOKです！もし未成年と思われる方が飲酒、喫煙を確認された場合、公演が中止になりますのでご協力のほどよろしくお願いします。

Q:１人で遊びに行ってもだいじょうぶ？
A:もちろん大丈夫です！開催時間もお昼からなので気軽に遊びにきて頂ければと思います。

Q:クラブとか遊びに行ったことないけど…
みんな揺れたりJumpしたりしてるけど、音楽聴くためだけに遊びに行ってもいいの？という方もいらっしゃいますが、自分なりに気持ちのいい楽しみ方や聴き方を見つることができればと思います。 大音量の音楽に耳を傾けるのもいいと思いますし、DJがクラップをしていたらクラップするのもいいですし、回りの人と音楽に体を任せて揺れてみるのも楽しいかもしれません。 ただし回りの人に迷惑をかけるのはNGです。

Q:入場制限ってあるの？
A:当日は会場側のキャパシティオーバーの場合は入場制限となります。予めご了承下さい。

Q:USTREAMでの配信はありますか？
A:今のところ配信する予定はありません。
