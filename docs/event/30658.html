【イベント概要】
これから会社を設立しようと考えている起業家のために
基本的な法律の知識や手続き、また節税できる会社設立の方法を説明します。

【イベントの趣旨】
会社設立には非常に多面的な知識が必要です。
たとえば法的に有効であることはもちろんのこと、節税対策も考えていかなければいけません。
さらには起業後に助成金をもらえるかどうかや、さらには銀行からの融資の可能性も検討していかなければいけません。
それらを実際によくある失敗例などを交えながら解説し、実際の会社設計をしていきます。
