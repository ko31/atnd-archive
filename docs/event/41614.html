  1. iOS, OS X での開発に関する勉強会です。
  2. Cocoa のあれこれについて、テーマを決めての発表・聴講／質疑応答、が主な内容です。
  3. 発表内容の規模は問いません。何か喋っていただける方大募集。ライトな内容・ゆるーい内容大歓迎です。お気軽にどうぞ。
  4. 今回は大阪のサイバーエージェントさんのオフィスで開催です！
  5. 勉強会の様子は Ustream の Cocoa 勉強会関西チャンネル で中継する予定です。
    1. 今回はUstreamはありません。申し訳ありません！
    2. すみません、やはりUst出来るようになりました！ Cocoa 勉強会関西チャンネル でよろしくお願いします。
  6. 勉強会終了後、懇親会あり。
  7. 参加登録時に発表予定が決まっていない方は、決定次第メッセージまたはコメント欄でご連絡ください。
  8. 参加費:無料
