title メタル資源開発の闇
catch 過熱する資源争奪競争とその影にある腐敗・環境・労働・人権問題
description

私たちの生活に不可欠な携帯電話やパソコンには多くの金属が使用されています。
その鉱物資源採掘のために環境汚染が進行し、先住民の強制移住や児童労働といった
人権問題を引き起こしています。
企業の社会的責任(CSR)が問われている今、メタル資源のサプライチェーンの最上流で
起きている諸問題について、現場*での取材を基にお話いただきます。
*最近の取材先：ニューカレドニア(Ni/Co)、ルーマニア(Au)、インドネシア(Au/Cu)、
オーストラリア(U)、中国内モンゴル自治区(希土類)等
(Ni:ニッケル、Co:コバルト、Au:金、Cu:銅、U:ウラン、希土類:レアアース)
