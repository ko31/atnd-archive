イベント内容

マイクロソフト MVP 澤田賢也さんをお招きして、Windows Server 2012 R2 Essentials の勉強会を行ないます。
今回のテーマは、「R2を使ってみよう～その5」です。

昨年発表された最新バージョンのR2の使い方を勉強していきます。

前回から、R2の新機能ということではなく、R2の全体像から細かな機能まで順に勉強しています。
ですから、Essentialsにまったく触ったことがない、という方も気軽にご参加ください。

そこで、さあ、一緒に勉強しましょう。
