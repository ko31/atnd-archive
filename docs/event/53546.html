鹿児島でもLinuxの勉強会がやりたい!

ということで鹿児島Linux勉強会第03回目です．
LinuxやPC UNIX，OSS関連の話題で盛り上がりたいと思っています．

今回は前半読書会，後半発表にしてみます．

鹿児島らぐ

第00回で鹿児島らぐ(Kaoshima Linux User Group)を立ち上げました．
オフに参加できなくてもオンラインもあるのでMLの登録をぜひ

- 鹿児島らぐ

読書会

Linuxをこれから始めるって人が多いようなので前半は読書会を行います．
教材はLPI のLinux標準教科書 を使います．
PDF/ePub/Android/Kindle版があります．前もって入手しておいてください．
※iPad版は現在利用できないようです．

- Linux標準教科書 無料ダウンロード LPI-Japan LPICレベル1対応｜Linux技術者認定機関　LPI-Japan [エルピーアイジャパン]

発表を行わない人優先で輪読していく予定です．

発表

後半はLTのような感じで発表したいネタを持った人が発表できるようにしたいと思います．
発表ネタはLinux やPC-UNIX などにかすってればOK
相談事とかもOK
(Ubuntu 14.04LTS 入れてみたよとかRaspberryPi 買ってこんなことやりたいけどどうやるのとか XPマシンに Linux 入れてみたけどこれまでXPでやっていたこういうことはLinuxではどうするのとか…)

ネタ(こんなことやってみたよ，これ教えて，ここがわからない等々)がある人は参加コメントや下のコメントなどに発表したいネタを書いてください．
ネタがない人も是非参加してください．

会場

今回は鹿児島中央駅前のダイエーの入っているビルの7階の「よかセンター 第2会議室」で行います．
