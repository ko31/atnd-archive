第1回　第1回　Windows 7 Phone（F-07C）分解改造ハッカソン 2011 Summer!

Windows Phone 7.5(Mango)の発売が噂されるなか、なぜか日本市場に突如現れたWindows 7 Phone。
発売日にゲットした方を集めて情報共有をしましょう。

実施内容

変態端末F-07Cでやっていること、これは動いた、動かなかったアプリの報告をしましょう。
また、端末をバラしてパーツを換装したり、オーバークロックして故障する寸前の状態まで試します。

もし万が一端末が壊れても、保証書片手に「初期不良」と言い張り無償交換したり、
ケータイ補償お届けサービスを活用すれば痛くもかゆくもありませんよね？

ちなみに余談ですが、香港版iPhoneをJailBreakし煉瓦になった際、アップル香港へ「初期不良」と言ったら本体無償交換になりました。
また、SC-02BにマイクロSIM突っ込んで抜けなくなり、強制的に抜いてSIMスロット破損させたときも、「初期不良」という魔法の言葉で無償修理になりました。

また、au版XOOMにカスタムROM焼いて煉瓦になったときも、
「初期不良」という魔法の言葉で本体新品交換になりました。

さぁ、メーカーはこんなに優しいのです。
何処までできるのか試してみたくありませんか？

諸注意事項

いろんな実験を行う予定です。

  • 万が一に備え、ケータイ保証お届けサービスへのご加入を強くお勧めいたします。
  • 万が一に備え、初期不良という魔法のことばで無償修理にするために保証書を紛失なきようお願いいたします。
