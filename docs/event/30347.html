description

様々なメーカのL3/L2スイッチ，ルータ，FWなどを用いて，ネットワークの設計，構築，テスト，撤収までの作業を行い，最後に設定したコンフィグレーションの良し悪しについての反省会を行おうという内容です．集まった人数によって1から2チームで作業をしてもらうことになります．競技ではないので，皆でいろいろ相談しながら作業を進めて行きます．

また，朝から作業を始めたいので，出来れば前日入りして下さい．宿泊場所としては鈴鹿高専のすぐ近くにあるGH鈴鹿を予定しています．GH鈴鹿でネットワーク構成の検討ができればと思っています．

なお，GH鈴鹿を利用される方はそのことコメントに残して下さい．社会人1000円，学生500円の負担をお願いします．

現在用意できる装置
FireWall:
・NetScreen NS-204
・Fotigate FG-60C

ルータ:
・NEC IX2015
・YAMAHA RTX1100x2台
・YAMAHA RTX810
・YAMAHA SRT100
・YAMAHA RT100i

L3/L2スイッチ:
・Allied Telesis CenterCOM 8724SL(v2)x3台
・Alaxala AX1240S
・Cisco Catalyst 2950×2台

他にも面白そうな装置をお持ちの方は是非お知らせ下さい．

いろいろやりたいですが，1日で収めることを考えるとIPまでにしておいて，サーバ周りやDNSは今回は無しにしましょう．たぶん，VLAN tagを振るだけでも結構苦労するだろうと思います．あと，デュアルホームもどきが可能なら試してみたいところです．

ハッシュタグは #nethackathon でお願いします．
