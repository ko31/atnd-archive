「関西ゼロからはじめるR言語勉強会」開催します

※定員になりましたのでさらに定員枠を増やしました！

データマイニングというバズワード的表現から脚光を浴びてきた統計解析用であるR言語。
私もRを使って業務をしていますが関西でRを使って統計解析の業務に従事
している方、興味のある方はどれぐらいおられますでしょうか？

この勉強会は

  • R言語について今からはじめたい。
  • 周りのみんなはどうやって勉強をはじめたのだろう？
  • どんなデータ分析をしているのだろう？

など情報交換の場です。
又、発表者も同時に募集していますので是非ご連絡ください。
勉強会後に懇親会予定しています。
参加表明時に懇親会への参加・不参加もコメントお願いします。
