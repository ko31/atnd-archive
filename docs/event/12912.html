Flashのシーン遷移について語り合おう！

Flashコンテンツを作る上で避けて通れない「シーン遷移」
そんな「シーン遷移」について語らいませんか。

　(1) シーン遷移に関する何かを作ってきて披露する。
　　* シーン遷移ライブラリを作ってきたり
　　* Progressionを用いたサイトを作ってきたり
　(2) シーン遷移のサンプルをいじって披露する。(※)
　　* シーン遷移のサンプルをいじってみたり
　　* シーン遷移のサンプルのここが分からないと質問してみたり
　(3) シーン遷移について質問する。
　　*　これまで経験したシーン遷移の疑問点とか

勉強会の詳細については、こちら を参考にしてください。

　[AS3.0] シーン遷移に挑戦！ (1) / [AS3.0] シーン遷移に挑戦！ (2) / [AS3.0] シーン遷移に挑戦！ (5)


ひよこの会 について

ひよこの会は「いまさら他人に聞けないFlashの基礎的な何かを勉強する会」です。
プロアマ、学生、ベテラン、年齢問わず、Flashを現在進行形で勉強してる人であればどなたでも参加可能。
Flashを現在進行形で勉強してるFlash好きなら誰もがもってる今さら他人に聞けないFlashの話で盛り上がる会です。

時間 部会
18:00〜20:00 勉強会
20:00〜20:30 相談会
21:00〜23:00 懇親会

3部構成になっていて、どの部会に参加していただいても構いません。
ただし、懇親会のみ実費で、あとは無料です。

懇親会会場は「炭火焼肉 心斎」
(4,000円飲み放題コース)

3回目の開催なので、まだまだ雰囲気ができあがってはいません。
初参加の方もどんどん積極的に絡んできてください。


参加表明 について

twitterを通して、告知・連絡など行いますので、twitterアカウントを必ずお知らせください。

人数については、多少のゆとりがあるので、とりあえず、参加登録しておいてください。
また、キャンセルによる繰り上がり参加などもありますので、是非、ぽちっと。

勉強会・相談会・懇親会のいずれの部会に参加されるかを明確に。

懇親会については、できるだけ人数を把握したいので、
3/4(金)までに、↓下のコメント欄に出欠の確認を必ず書き入れておいてください。
参加者で、特に欠席の連絡のない方は、そのまま出席の頭数に入れておきますので、ご注意ください。
勉強会・相談会の出欠についても併記していただけると有り難いです。

[例]　勉強会・相談会・懇親会 全参加です！

また、懇親会だけ参加、というのもぜんぜんありなので、
勉強会は行けないけどとか、お友達も懇親会には誘いたいといった場合も、
↓下のコメント欄にその旨を書き入れてください。

ちなみに予算の予定は、前日までに正確に人数が把握できた場合は、飲み放題で4000〜5000円あたりで予約、
そうでない場合は当日にどこか適当な居酒屋を探すことになります。ご協力よろしくお願いします。

その他ご質問などもお気軽に↓下のコメント欄かTwitterにて！
