ネコワーキング、オーナーの広瀬です。
今週日曜に行なわれるイベントの告知です。

ネコワーキングの内装も監修して頂いている
起業家のmorikawaさんに起業したばかりの
広瀬が公開相談するというイベントです。

ついつい忘れがちな起業後に必要な事や、
起業後の生々しい話しが聞けると思います。

ご興味ある方は、
ぜひお越し下さい。

　　　　　　　　　　　　　　　

＜＜＜＜＜＜　こんな人にオススメ　＞＞＞＞＞＞

・起業を考えている方。
・起業したての人がどんな課題を抱えているか知りたい方。
・起業するまでではなく、した後に必要になるコトを知りたい方。

□■□■□■　イベント概要　□■□■□■

＜＜イベント内容＞＞

株式会社や、スペースを立ち上げた広瀬ですが、
細々とした雑務や初めて行う事に忙殺される日々です。

仕事量が多いのに、
初めてのことが多くて進まない！
でも、お客の対応、新規顧客の獲得は外せない！

そんな迷える新米社長が、
先輩起業家であり、建築家のmorikawaさんに
アドバイスを求めます。

一円の重み（経理）、領収証管理、
業務報告、スタッフとの付き合い方。、
税理士の見分け方。
銀行との付き合い方。
お客さんの見つけ方、利用し続けてもらうには。
メディアとの付き合い方

など
ざっくばらんに相談します。

ビジネスプランでも、ビジョンでもなく、
見落としがちな泥臭い話し。
「起業した後に必要になるモノ」を教えて頂きます。
