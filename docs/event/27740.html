共同研究でモメないための基礎知識

description

【受講対象】
・対外的業務の発生する技術系担当者
・社外機関と共同研究を行う人
・開発系の中小/ベンチャー企業の技術者、リーダー、経営者
・技術移転担当者（公的機関など）

【習得できる知識】
・共同研究において、陥りやすいトラブルを回避できる
・あらかじめ社外機関との契約を想定して、業務を推進できる
・委託業務やサンプル取得にまつわるトラブルを回避できる
・研究開発における対外的なやり取りを、自信を持って推進できる

【講座のポイント】
　開発委託や共同研究をはじめとして、企業の開発活動は単独では完結しない場合が多々あり、それには他社との契約がつきものです。そして、契約相手との関係は技術者から始まるケースが多いのが現状です。
　しかし技術者が相手方と協議をして話を進めた後に、契約のために法務部門に相談してみると、「それは認められない」「相手を説明・説得して訂正してください」などという強力な「指令」により技術者が板挟みにあることが多々あります。
　本セミナーでは、そのような板挟みを回避するために、共同契約をはじめとする６つの典型的ケースを題材として、触れてはならない危険ポイント、明確に決めるべき必須ポイントを整理し、技術者が安心して対外的な関係を構築するための基礎的な知識を整理します。

【プログラム】

１．契約意識と基礎知識
　　1-1.契約の果たす役割
　　1-2.契約書を作る意義
　　1-3.契約チェックポイント７つ道具

２．情報保護と秘密保持契約
　　2-1.事例：情報交換に際して交わされる「秘密情報」
　　2-2.情報の定義
　　2-3.相手方の特定
　　2-4.事例：秘密保持義務を負う期間の設定
　　2-5.事例：契約締結後の管理
　　2-6.事例：現場見学と情報保護

３．サンプルとサンプル提供契約
　　3-1.事例：サンプル提供契約の重要性
　　3-2.サンプルの秘密保持
　　3-3.サンプルの用途
　　3-4.成果の取り扱い
　　3-5.事例：サンプル授受がもたらすリスク
　　3-6.サンプル提供と取引の強要

４．外注と業務委託契約
　　4-1.事例：業務委託契約の重要性
　　4-2.対価と成果
　　4-3.事例：役立つ成果のために
　　4-4.仕様決めの注意ポイント
　　4-5.委託業務の終了

５．共同研究・共同開発と契約
　　5-1.事例：成果の帰属の問題
　　5-2.保有技術の取り扱い
　　5-3.契約終了後の成果
　　5-4.事例：想定外のケース
　　5-5.役割分担の重要性

６．産業財産権と共同出願契約
　　6-1.事例：貢献度と特許の持ち分
　　6-2.共同出願の対象
　　6-3.事例：勝手に特許出願されてしまった
　　6-4.改良発明の取り扱い
　　6-5.利益配分の調整
　　6-6.事例：大学との共同出願に関する問題点

７．総合ケース
　　7-1.学会へ参加、大学の展示に興味を持ち、情報交換したい
　　7-2.展示会へ参加したところ興味ある技術を発見！サンプル提供を頼みたい。
　　7-3.製品開発に際し、部分的に他社へ開発を委託したい
　　7-4.製品開発に際し、他社の技術を利用したい
　　7-5.共同研究から新たなアイディアが生まれたので、特許出願しよう
