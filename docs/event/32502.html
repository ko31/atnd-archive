SAMPLE: JUnit4.11を読み解く

Javaでプログラム組んだ事はあるけど、 もうちょっとレベルアップしたいんだという中級者向けに Parser役を@nagiseにお願いして、Javaに関する勉強会を行っています。

第参回目となる今回は、"JUnit4":http://www.junit.org/ のソースコードリーディングを行います。

形式

ビアバッシュでだらだらソースを読み解きます。
準備費用として一人1000円ほど徴収させてください。

参加形式

  1. 食べます
  2. プロジェクタにソースを写しながら読み解いていきます。
  3. 実際に動かして挙動を確認します
  4. 1にもどる。

必要なもの

Java6 or Java7のソースとJavaDoc、今回のソースコードリーディング対象となるJUnit4.Xのコードの持参を推奨します。

環境

ネットワークはWiFi環境を準備します。

PCを持参してもOKですが、
持たない人向けにプロジェクタとスクリーンは会場にて準備します。
