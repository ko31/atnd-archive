description

スマートフォンでのゲーム利用の拡大を背景に、アイテムの不正取得やゲームの不正施行などによる被害が拡大しています。これらの“チート”と呼ばれる不正行為は、ある程度の知識を持っていれば容易にできてしまうものであり、ゲームアプリを提供する企業にとって売り上げの減少や信頼性の低下など、重大な事業リスク要因となってきています。

　年間100本以上のゲームアプリ診断や、大手ゲーム事業者様のセキュリティガイドライン策定に携わるサイファー・テックでは、このたびゲームアプリのチート対策に関する相談会を実施します。当日はチートの実演を交えながらその危険性と対策に関してご説明させていただき、その後専門のスタッフが皆さまのチート対策に関する疑問にお答えします。

　チート行為をはじめとして、ゲームの不正行為へご懸念をお持ちのアプリ企画・開発ご担当者様、ぜひこの機会に足をお運びいただき、アプリの企画・開発段階からチート対策をお考えいただければと思います。

＜プログラム＞
・チート行為の脅威とその対策に関するご説明
・チート行為のデモンストレーション
・チート対策コンサルティングメニューのご紹介
・ゲームアプリのチート対策に関する質疑応答
