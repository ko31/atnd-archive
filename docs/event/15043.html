いよいよピクニック・シーズンがきましたね◎

こんにちは！ greenz.jp の @whynotnotice と @ono_chi です。

さっそくですが 今週24日（日） に green drinks 春の特別編 ということで急遽PICNICを開催することにしました！場所はアースデイ東京が真っ只中の 代々木公園芝生広場 にて。（雨天時は中止です！こちらに極力書き込めるよう努めますが、現場に誰もいなかったら中止とお考えください。）

24日には、13時から自然エネルギーの語り部として注目されている 田中優さん の講演や、greenz.jpも応援している行進 エネルギーシフトパレード もあります。

そちらにご興味がある人は、集団で渋谷を散歩するもよし。会場に残ってお酒を酌み交わしたい方はそれもよし。ピクニック会場でのお遊びも、シャボン玉からドッチビー（仮）まで、信頼のピクニックプロデューサー集団 東京キャンプ にもご協力いただいて盛り上げていく予定です。

ふらりと入れ替わり立ち代わりでOKです。春の風を感じながら、地球の日を考えるアースデイに、みなさんと呑めることを楽しみにしています◎
