description

夜中（11時頃？）に集合。

その後、ラーメンでも食べてから、夜中の京都を写真を撮りながら歩きます。

簡単な写真のワークショップも開催予定。

カメラはデジイチでもコンパクトカメラでもiPhoneでもOK！（もちろん、iPadでも）それぞれに合わせた撮り方で、深夜の京都を楽しみます。

当初、出町柳→銀閣寺→哲学の道→若王子→南禅寺→知恩院→円山公園→清水寺というルートを考えていましたが、イベント当日、下鴨神社で御手洗祭が午前5時30半から行われているので、そちらの行ってみようかと考えています。

その後、朝のカフェ（イノダコーヒーあたり？）でお互いの写真を見せあいながら朝食。

今のところ参加費など無料です。
Maxで、12人位を見込んでいます。
詳しいルート等は、決定次第お知らせいたします。
雨の場合は、多分中止です。
