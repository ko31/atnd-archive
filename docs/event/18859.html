第1部　プラスチック加飾技術の概要と最近のトピックス
【12:50-13:40】

MTO技術研究所　所長　桝井　捷平　氏

【講演趣旨】
プラスチックの加飾技術の概要と最近のトピックスの紹介をし、今後の展開の方向性を予測する。

【プログラム】

1.プラスチック加飾技術の概要
　　　加飾の意義、種類、分野別採用状況、棲み分け、環境対応等

2.国内外プラスチック加飾技術の動向のまとめ

3.加飾の種類別の最近のトピックス
　1)最も活発な動きのあるフィルム転写・貼合加飾
　　　国内外の加飾フィルム、意匠表現、成形方法等
　2)特別な表面層を付加しない低コスト加飾
　　　バリオサーム射出(電磁誘導加熱、レーザー加熱等)
　3)着色材不要の多層構造色加飾
　4)視覚、触覚を満たすソフト表面加飾
　　　軟質表皮材貼合、ソフト/ハード二層成形、ソフトフィール塗装
　5)環境に優しいインモールドコーテイング、リアクティング
　6)少量多品種に適する二次加飾の新たな動き　
　7)地球環境対応のバイオマス材料利用の加飾
　8)可能性を拡げる組合せ加飾

4.加飾技術の今後の方向

第2部　真空・圧空成形から誕生した3次元加飾工法(TOM工法)(Three dimension Overlay Method)
【13:40-14:55】

布施真空株式会社　代表取締役社長　三浦　高行　氏

1.真空・圧空成形概論

2.次世代成形法NGF (Next Generation Forming)

3.いろいろな3次元加飾工法の特徴とTOM工法の位置付

4.「TOM工法」の詳細および実用化例 (Three dimension Overlay Method)

5.発展する「TOM工法」

6.TOM工法」における課題
