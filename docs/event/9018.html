description

第１回開発プロセス　「開発工程を組み立てよう！」のノウハウを使って、札幌市で採用した包括フレームワークの開発プロセスを分析します。

工程毎に調達をかける「札幌市の包括フレームワーク開発プロセス」は大変野心的であり、うまくいけば画期的であり、下手に運用すると下工程でとんでもないことになる開発プロセスです。

そのプロセスを題材に工程毎に調達をかけるリスクとその回避策をグループディスカッションしてみたいと思います。

私も含め包括フレームワークに詳しい人がいるわけではありませんので、あくまでも公開されている情報をもとに開発プロセスとして分析し、工程を有効に活用していくノウハウをディスカッションして共有できればいいと考えています。

興味のある方は是非ご参加ください。
