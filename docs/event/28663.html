catch
description

facebookやtwitterなどのソーシャルメディアで、やっぱりカッコいい写真を掲載してみたいですよ、ね。しかし、プロに依頼すると、それなりの値段になっちゃいます。「そこまでは、費用をかけれない！けど、それなりの写真は撮影したい」という方のための相互撮影大会を開催したいと思っております。この機会に、奮ってご参加くださいませ。

イベント内容
30分　カッコいい写真を撮影するポイントは、ここ！

座学によるカッコいい写真を撮影するポイントをレクチャします。

60分　相互撮影大会
座学をもとに、参加者相互に写真の撮影を実施します。
（こちらの指定するサーバにアップロードして、写真を共有します）

30分　評価会
アップロードされた写真を見ながら、それぞれの写真を各自で評価しましょう。
