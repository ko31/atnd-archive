description

　京都ブランド推進連絡協議会(京都府、京都市、京都商工会議所)では、京都の魅力を全国に発信するため、全国の主要都市でフォーラムを開催しています。
　今回は、歴史的建造物や城下町を有し、世界遺産登録でも注目を集める萩市と、1200年の悠久の歴史を持つ京都の両都市でご活躍のパネリストより、各都市の多彩な魅力についてお話いただきます。
　参加者にはお帰りの際、もれなく京都の大人気スイーツをプレゼント！さらに、お着物でお越しの方には、手ぬぐい＆あぶらとり紙セットをプレゼントいたします。
　心をこめたおもてなしで、みなさまのお越しをお待ちしております。

◆日　時　平成２７年１０月３日（土）　１４：００～１６：００
