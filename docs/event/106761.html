■障がい者の方へお願いする業務がない
■作業指示をどのようにすればいいのかわからない
■体調に不安がある

すでに障がい者の方を採用されている企業様も、これから採用をお考えの
企業様も上記のようなお悩みをお持ちのことと存じます。
同様のお悩みをお持ちの企業様向けに、障がい者就労支援事業所様と
業務マニュアル作成についてのセミナーを開催させて頂き、大変ご好評を
頂きました。
ご参加頂いた企業様から個別に「同様のセミナーは他に開催しないのか」、
「うちにきて個別にセミナーをやってくれないか」 などのお言葉を頂き
今回ご案内させて頂きます無料体験会セミナーを開催する運びとなりました。

障がい者雇用にお悩みの方、マニュアル活用にご興味のある方には
ぜひご出席いただきたいセミナーです。
