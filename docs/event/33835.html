あの、伝説のアドベントカレンダーが帰ってきた！！

2011年。IT業界は空前のアドベントカレンダーブームであった。

  • アドベントカレンダーとは
    • 12月1日から12月25日まで、特定のテーマで複数人でリレーブログを書くこと
      参考：<a href= “http://www.otchy.net/20120106/tacj2011-finished/”>今こそまとめ読み！技術系アドベントカレンダー総まとめ 2011【完結】

そんな中、綺羅星のごとく現れた伝説、それが変態アドベントカレンダーだ！！

変態アドベントカレンダー2011

意識の高いこのカレンダーは、11月から始まって12月には息切れしているという有様。今年は、ちゃんと12月1日スタートで完走しよう！！

ちなみに去年のは技術評論社さんから電子書籍化されました。
<a href = “https://gihyo.jp/dp/sp/advent2011/G11C03”>変態アドベントカレンダー 2011 電子書籍版

調子に乗って夏にもやってました。
<a href = “http://atnd.org/events/29918”>変態アドベントカレンダー in Summer

ルールは「変態」というキーワードがあれば何でも良いです。
ITに関する技術的な話題でも良いし、エッセイ的な内容でもかまいません。

書いたらここのコメント欄に記入して、あとはTwitterに放流するなりFacebookでシェアするなりしてください。

都合で一日間が空いたりした場合は、続きの日程で日をずらしたり、入れ替えたりしていただいて構いません。

みなさんの遊び心をおおいに発揮して、楽しいカレンダにしましょう！！
