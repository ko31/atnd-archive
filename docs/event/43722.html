description

たった4行で電話がつくれる、クラウド電話API“Twilio”を用いたハッカソン。
Twilioではさまざまな言語やフレームワークに対応したHelperライブラリーが提供されているため、自身が得意とする環境で開発することが可能！
Twilioのもつ「着信、転送、録音、カンファレンス通話、SMSの送受信」の機能をつかってチームでアイデア～コンセプトを考え、実際にサービスを生み出していきます。

優秀者には賞品もあり☆
“電話”という概念にとらわれず、新しいコミュニケーションツールを一緒につくってみませんか？

＜実施内容＞
Twilioの「着信、転送、録音、カンファレンス通話、SMSの送受信」などの機能を使い、あるテーマに沿って、新しいサービスをつくっていただきます。
※テーマはアイデアソン当日は発表いたします

＜対象＞
・本ハッカソンに興味をお持ちの開発者/サービス企画者/デザイナーの方
・Twilioをこれから始めたい・興味がある方
・無線LANを搭載したノートPCを持参できる方
※開発ができない方でも、アイデアソンのみ参加ＯＫ！
