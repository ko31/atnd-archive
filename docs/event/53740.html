　福岡の皆さん、こんにちは！

　8月9日（土）と10日（日）、ウエアラブルデバイスの可能性を皆で考え、作ってみるハッカソンを開催します。

　今回の開発テーマは「ライフスタイルを変えるウエアラブル」。ソニーモバイルの最新デバイスSmartBandおよびSmartWatch2のほか、ビーコン、NFCタグ、さらにはGoogle Glassも会場に持ち込みます。まずは、これらのデバイスとクラウドサービスをつないで何ができるか、参加者全員でディスカッションします。

　さらに、2日目には、アイデアをもとに実装を進めます。実アプリを作っていただいた方は、Android Application Award 2014に優先登録します！

　イベント参加費は無料。初日、2日目、1日だけの参加でも構いません。皆さんのご参加をお待ちしています！

（注）
　本イベントは、先日ご案内した「Xperia™ Wearable Hackathon Fukuoka」をリニューアルして開催するものです。地元の皆さんのご意見をうかがい、ソニーモバイルの端末だけでなく、Google Glassなども体験できるようにしました！
今回のハッカソンでは、未公開のSDKを限定公開します。このため、ご参加いただく際、SDK利用に関するNDA（秘密保持契約）にご署名いただくことになりますのでご了承お願いします。
