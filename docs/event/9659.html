description

＜講座のポイント＞
　めっきに関する研究のほとんどは、浴の開発や電気化学的観点から行われてきているが、講師は、めっき膜は金属材料そのものであるということから、これまで40年以上にわたって、めっき膜の構造と物性に関する研究を金属学的観点から行ってきた。 それら膨大な実験結果から、めっき膜の構造と物性制御に関する講師独特の理論を構築することに成功した。それらは上記の著書にまとめ、すでに研究者や技術者に利用していただいている。

　講演会では、参加者が現在抱えている問題点や知りたい点を中心にディスカッション形式で進めます。本講演で特に重要な内容は、めっき膜の密着性についてであり、これまでのものと全く異なった理論で、密着・不密着の現象を明快に説明する。また、金属の腐食現象とその対策についても、独特のものをお話しする。

＜プログラム＞
１．めっき膜の構造と物性制御理論（金属学的考察）
　　1-1 結晶学的構造の制御
　　　　　ａ．結晶質
　　　　　ｂ．アモルファス
　　　　　ｃ．金属間化合物
　　　　　ｄ．固溶体
　　　　　ｅ．混合組織
　　1-2 表面形態の制御
　　　　　ａ．平滑（光沢性）
　　　　　ｂ．半光沢（表面凹凸）
　　1-3 結晶粒径(形)の制御
　　　　　ａ．微結晶
　　　　　ｂ．繊維状結晶
　　　　　ｃ．粗大結晶
　　　　　ｄ．柱状晶
　　1-4 密着性の制御（含、エピタキシャル）
　　　　　ａ．金属基板
　　　　　ｂ．非金属基板（Siウエハー、セラミックス、プラスティック）などとの密着
　　1-5 配向性の制御
　　　　　ａ．電析めっき膜
　　　　　ｂ．無電解めっき（含、置換めっき）膜
　　1-6 膜の残留歪（残留応力）の制御
　　　　　ａ．引っ張り歪（応力）
　　　　　ｂ．圧縮歪（応力）

　　　　　ａ．ノジュール

　　　　　ｃ．ピット（ピンホール）

　　金属の腐食に関する、これまでと全く異なる理論の提案と、その理論による実際の腐食現象の説明
　　3-1 全面腐食

　　3-3 エッチピット
　　3-4 粒界腐食
　　3-5 応力腐食
　　3-6 ガルバニック腐食など
