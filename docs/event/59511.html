description

あなた自身で選んだ色には
あなたの本当の気持ちや想いが隠れています。

今の自分に合う色をみつけたら
ラッキーモチーフのパーツに取り入れて、
毎日身につけたくなるアクセサリーを作りましょう。

ちょっとした息抜きの
カラーセラピー×プラバン・コットンパールアクセサリー作り
ワークショップです。

■ワークショップ内容■
◎アクセサリーについて
セラピーで選んだヒーリングカラーを用いて、
ラッキーモチーフをプラバンで焼き、コットンパールと合わせて
世界で１つだけのピアス又はイヤリングを作りましょう。

◎カラーセラピー体験について
カラーセラピーというと、スピリチュアルで不思議なものとして捉えられがちですが全てがそうではありません。今回は、セラピーの本質（色彩心理）にちょっと触れてもらい楽しみながらセラピー体験をして頂きたいです。
