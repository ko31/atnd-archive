description

こんにちは。シンメトリー・ジャパン代表の木田知廣です。

ビジネスだって、「ちゃんと聞いてたのかよ～」ってボヤきたくなることありますよね？

部下への指示や社内で仕事を頼んだ時、はたまたお客さんとの商談だって、あれほど
詳しく説明したのに、アウトプットのイメージが共有できていなくて、トンチンカンな
ことって、けっこうあります。

「まったく、もう…」なんてぶつくさ言いながら、改めて説明し直すという経験は誰しも
あるはずですが、ちょっと待ってください。

それって、実は、自分の側のコミュニケーションの問題であることが多いのです。

考えてみれば、「効果的なビジネス・コミュニケーション」って、ちゃんと教えてもらった
ことってないですからね。多くのビジネスマンは。

損な問題にぶち当たったことがある人に学んで欲しいのが、「ファシリテーション」です。

プレゼンテーションとも違う、コーチングとも違う、今ビジネスマンがもっとも必要と
されているコミュニケーションのスキルです。

このファシリテーション勉強会は、ビジネス・コミュニケーションの第一人者をゲスト
スピーカーにお迎えして、問題意識を共有する仲間と一緒にスキルアップをして
いこうというものです。

第1回のゲストスピーカーは、株式会社シナプス代表取締役の家弓正彦氏 (http://kayumi.jp/)。

シナプス・マーケティング・カレッジを主催しているので多くの方がご存じでしょう。

もちろん、ファシリテーションでも多数のセミナーを企画されていて、まさにビジネス・
コミュニケーションの分野では、日本を代表するビジネスパーソンの一人です。

そんな家弓さんの話を少人数で聞くチャンス、ぜひお見逃しなく！
