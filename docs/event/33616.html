2dに特化した、iPhone対応のゲームエンジン cocos2d の勉強会を行います。
導入編ということで、cocos2dの機能説明および簡単なサンプルを作成し、学習します

※パソコンで実際に作るハンズオン形式の講座です。

講師プロフィール

西田 寛輔 氏（tonosamart）
ハイパーメディアクリエイター
数々の制作会社でメディア制作、スマホアプリ制作、WEBサイト制作に関わる。
自身でも多くの勉強会を主催。その広く深いスキルに裏打ちされるわかりやすい講師スタイルは広く学習者から支持を受ける。
とのさまラボ（facebook）

対象 / 持ち物

cocos2dに興味がある、iPhone開発者（初心者～中級者以上）

ノートパソコン（Mac）の持ち込みが必須となります。
　※xcode や objective-C 自体の解説はおこないません。

ノートパソコンについて
・Xcode 4.5.1
・cocos2d for iPhone 2.0

上記インストール済みのノートパソコンおよびアダプターを持参ください。
(それ以外の環境でもOKですが、環境の違いにつきましては、原則フォローいたしません。)
※cocos2dについては、当日インストール手順を説明します。cocos2dのインストール方法がわからない場合は、インストールしていなくてもかまいません。
※Xcodeのインストールは必須です。
