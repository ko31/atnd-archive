femme入会説明会（無料）
～femmeならではのお茶菓子付き～

男女共同参画」や「ワークライフバランス」と言った言葉が定着し、女性の生き方も多様化してきました。女性の社会進出が叫ばれる中、「自分はどう生きたいのか？」を考え、悩む女性も増えています。そんな女性をサポートしたいという願いから、ウィメンズライフデザインコミュニティ“femme”は誕生しました。

femmeは、『一生輝くわたしになる』を合言葉に、自分らしい生き方を模索し、どのライフステージでも輝ける女性を目指 していくための会員制のコミュニティです。

femmeでは、Study-Salonや交流座談会。そしてさまざまなセミナーやイベントを通じて、現在目の前におかれたハードルを低くしたり、将来目の前におかれるであろうハードルに備えていき、自分らしく活き活きと生きていけるようなスキルや知識を楽しく身につけるこ とを目指しています。

【会員になるとお得なこと】 平日Study-Salonが何度でも受け放題！

femmeでは連日さまざまなサービスを開催しています。
女性の「あったらいいな」の声を取り入れて、積極的に新しいコンテンツをご用意しております。
毎回参加メンバーが変わるので、内容も幅が広がります。
すべてのStudy-Salonが無料で何度でもご参加頂けます。

スペシャルスイーツ付きのfemmeサロンも無料！

femmeでは会員同士の交流の場としてfemmeサロン（美味しいスイーツ付き）を提供しています。会員の方はいつでも無料でご参加いただけます。

その他のサービスもすべて会員価格で！

femmeではセミナーやワークショップ、講演会など、様々な企画をしていきます。
これらすべてのサービスを会員価格でご参加頂けます。

たくさんの方にfemmeの魅力を知っていただき、会員になって頂けたらと思います。
もっともっと大規模なコミュニティにしていきたいと思いますので、是非ご参加くださいね♪
参加者の方の疑問・質問にお答えします。
femmeならではのお茶菓子をご用意して、みなさまのお越しをお待ちしております。
