agatsuma.survive#02

agatsuma.survive は群馬県内のプログラマ, デザイナー, マークアップエンジニア, コーダーがこれからの群馬で生き残っていく為に知識やスキルをシェアする勉強会です。

#02 モダンすぎる静的サイトの作り方

第2回は、テンプレートエンジン(Markdown, Haml, Slim) と CSSプリプロセッサ(Sass/Scss,compass) を用いた静的サイトの作り方について勉強します。テンプレートとCSSプリプロセッサの効率的な変換を行うため静的サイト生成ツール「Middleman」も触れる予定です。

前回 git について勉強しましたが、今回その知識は無くても一応大丈夫です。今回も先生役は @yterajima です。

内容

各種テンプレートエンジンとCSSプリプロセッサ, compassを組合せた静的サイトの作成方法について学びます。よりメンテナンス性の高いサイトを目指して構造化、共有化、DRYに作成する方法。テンプレートとCSSプリプロセッサの変換にはコマンドラインを少し使います。

対象

  • プログラマ, デザイナ, マークアップエンジニア, コーダー
  • HTMLをそのまま書くのがダルい人
  • CSSをそのまま書くのがダルい人

必要なもの

  • ノートPC(できれば Mac または Linux)
    • できれば Mac, Linuxまたは仮想マシン
    • Rubyが動作する環境
    • 環境設定がわからない場合早めに来てください
  • テキストエディタ or IDE
  • 折れない心
  • 寒さに負けない服装

資料

資料については当日PDFまたはHTMLでの配布を予定しています。
その他Webサイトを閲覧しながら進める予定です。

会場情報

  • 無線LAN, 電源, プロジェクタあります
  • 飲み物はコンビニで買ってきてね！
  • 電車で来る人は言っていただければ迎えに行きます(市城駅12:30着)
  • 駐車場はMax5台なので電車で来れる人は電車または乗合せだと助かります。

その他

  • 一部で需要があるようなので、どこか別の日程でgit/github会を再度開催する可能性があります。
