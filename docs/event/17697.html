Android女子部とmixiさんと共同開催のHackathonをやりま～す！！！

今回は「mixi API SDK for Android™」を使ったハッカソンを行います。
「mixi API SDK for Android™」を使えば、ソーシャルアプリを作るときに面倒な認証処理も簡単に実装できて、しかもmixiで利用されているさまざまな機能のAPIが簡単に使えます！
あなたが作ったAndroidアプリも簡単にソーシャル化できるんです！

講師としてmixiの「mixi API SDK for Android™」開発者の方が詳しい説明をしてくれます。
チューターにはmixiとAndroid女子部から数名ご協力いただいて、皆さんのアプリ作成をサポートします♪

女子同士楽しくおしゃべりしながら、Androidのソーシャルアプリアプリをつくりましょう☆☆

mixi API SDK for Android™については こちら をご覧ください。

対象OSは2.2～です。
