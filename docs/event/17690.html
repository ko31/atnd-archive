WordPress 3.2 リリースパーティー in Tokyo

イベント ユーザー
event_id 17690
title WordPress 3.2 リリースパーティー in Tokyo
catch WordCamp Tokyo 2011 を先取りっ!!
description

2010年7月5日 (日本時間)、WordPress の新バージョン 3.2 “Gershwin” がリリースされました。WordCamp Tokyo 2011 を先取りしたイベントとして、新バージョンを祝うパーティーを行います。

※27日に95人→105人に増やしました。現在補欠の人は、当日ギリギリまで参加可能になる可能性がありますので、登録状況の確認をお願いします。

  • 日時: 2011年7月30日(土) 17:00〜19:30
  • 場所: マイクロソフト本社カンファレンス会場 (品川)
  • 参加費: アルコールを飲まない人1500円、アルコールを飲める人は2000円(飲まない人の500円アップ)。※7月21日18時に決定しました。当日受付で徴収いたします。
  • 対象: WordPress ユーザー、WordPress を仕事に使っている人、WordPress に興味のあるブロガー等、誰でも可
  • 食べ物・飲み物の持ち込み歓迎 (参加登録時およびコメントで持ってくるものを記載願います)

関東地区の WordPress ユーザーや開発者、ウェブ制作者などが一斉に集うお祭りです!! WordPress コミュニティーで活躍する方も多数来られます。「WordCamp Tokyo まで待てない」という方、「WordBench 勉強会は敷居が高そうで……」という方、気軽に参加できる今回のパーティーにぜひ行きましょう。

プレイベント

なお、当日はパーティーに先立って以下のイベントも開催されます。全部通しての参加、途中からの参加、勉強会だけの参加 (パーティーは出ない) も可能ですので、ぜひとも万障繰り合わせの上ご参加ください。

  • 11:00〜12:00 Weekly CMS 公開生放送 (見学無料。10時から入場可能)
  • 13:00〜16:00 WordBench 東京勉強会 (参加無料)

※勉強会に参加の場合は、コメントにその旨を書いてもらえば WordBench 東京での申し込みは不要です。(なお、WordBench 東京での参加表明をしても構いません)
※勉強会だけ参加でパーティーに出ない場合は、当ATND を使わず WordBench 東京サイトで申し込みをお願いします。
※ (2011-07-09 10:30 追記) 勉強会は満席になりました。パーティーの方も、勉強会だけ先に申し込んだ方を優先してパーティーへの参加を案内しておりますので、これから補欠登録されても参加できない可能性がございます。ご了承の上、登録してください。
※ (2011-07-11 21:10 追記) 勉強会およびパーティーの増枠 (60→95) を行いました。ともに受付を再開しましたので希望者は早めの登録をお願いします。

二次会

パーティー終了後、以下の通り二次会を開催します。

  • 場所: プロント品川インターシティ店
  • 時間: 20:00〜23:00 閉店まで貸切 (早く帰るのは自由)
  • 参加費: 2000円 (料理4品+飲み放題)

注意

  • マイクロソフト品川本社は、入場時に当日パスが必要です。イベント開始時および休憩時 (10時〜11時、12時〜13時、16時〜17時) にはエントランスフロアにスタッフが待機してパスを配布しますが、途中から参加する場合はスタッフがいませんので、入口付近に記載した電話番号まで連絡願います。
  • アンケート項目 (飲酒／二次会) は修正できませんので、事情が変わった／記入間違えた、などの場合は ATND のメッセージ機能でイベント発起人に連絡するか、全体向けコメントに修正内容を記載願います。

準備に協力したい

準備に協力したい方は、WordBench のアカウントを取得した上で、東京グループでの掲示板スレッドに Reply 願います。
