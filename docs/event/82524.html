出　演 杉浦 太陽 ・ 富永 美樹 ・ 村上 実樹 ・ 縄嶋 智徳

【第1部】 未来予想図 「グランドデザインの発表 !」　 → この地域のワクワクする未来予想図を発表します！

私たち公益社団法人平塚青年会議所は、平塚・大磯・二宮でまちづくりに携わる活動をしている、20歳〜40歳までのメンバーが所属する団体です。私たちが描く「グランドデザイン」は、青年らしい壮大な夢をたくさん詰め込んだものになっております。今回「グランドデザイン」を描くにあたり、多くの地域の方や行政の方の意見をお聴きし、その上で私たちの夢を取り込み制作させていただきました。「ずっと住みたい、願いがかなうまち」。これから何十年後も、夢を描きながら暮らしていける、そんなまちづくりを考えるひとつのきっかけとなれば幸いです。

【第2部】 未来×子どもフォーラム 「このまちが変われば、子どもが変わる」 → 杉浦太陽氏、富永美樹氏をお招きし、「未来・子ども」について特別対談を行います。

「このまちが変われば、子どもが変わる」

富永美樹さんをコーディネーターとして、杉浦太陽さんに、このまちの未来を、子ども、海、地域のきずなというテーマに沿ってお話いただきます。また、杉浦さんの子育てや家族・趣味のお話、ご出演中のＮＨＫ番組「キッチンが走る！」での体験談などを語っていただく予定です。そのほか、公益社団法人平塚青年会議所の取り組み、さらにこの地域の未来についてお話しさせていただきます。
