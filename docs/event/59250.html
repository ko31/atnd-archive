■福岡城 CULTURE&HEALTH WALKING TOUR 開催決定！

福岡城「軍師官兵衛」ドラマ展を出発地点とし勾配のある立地を活かした福岡城を健康的にウォーキングしながら、
歴史と文化を楽しむウォーキングイベントです。
ゲスト講師には今村式メソッドペットボトルウォーキングやスタードラフト会議等のメディアでも
お馴染みのあの「今村大祐」がプロのウォーキングをレクチャーします。
福岡城で歴史を学びながら参加してみてはいかがですか？
NHK大河ドラマ「軍師官兵衛」の最後の舞台となる福岡城の歴史を見ながら
健康的に歩いてみませんか？
