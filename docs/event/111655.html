今、大活躍されている怪談師8名がRan Theatre Kyotoで最恐の怪談を語ります。

開催場所であるRan Theatre Kyotoはオーナーや従業員が不思議な体験をしたり、地下でお化け屋敷×リアル型脱出ゲーム「ヤミツキ★ヤミチカ」が営業されていたりと、怪談にぴったりな場所…

雰囲気のある店内とオーナーが厳選した人気怪談師の語りをごゆっくりとお楽しみください。

怪談を聞き終わった後には出演者との懇親会や閉店後のお化け屋敷ツアーも予定しています。

懇親会は簡単なお料理、おつまみ、お酒、ソフトドリンクをご用意します。

出演者全員が参加しますので、怪談師との交流をお楽しみください。

他では絶対に体験できない怪談イベントですので、この機会を逃さずにお早めのご予約をよろしくお願い致します。

※怪談イベント後の懇親会、お化け屋敷ツアーは希望者のみの参加です。

※懇親会は準備の都合上、ご予約のみとさせて頂きます。

※料金は現地でお支払いください。
