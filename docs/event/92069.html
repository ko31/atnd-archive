仙台でハースストーン炉端の集いを開催します！

ハースストーン日本語化二周年のこのタイミングでウォーロックスキンが炉端の集いで入手可能になるなど、
炉端の集いが各地で開かれ盛り上がっています！
東京や大阪といった都市圏で炉端の集いが開催されていますが、ここ仙台でも炉端の集いを開催します！

ハースストーンコミュニティ：ハースストーン愛好会が運営として参加しイベントを盛り上げます。
本イベントでは各種企画を用意しておりますが、ハースストーンプレイヤー同士が集まって交流するのがメインとなっております。
ゆるい雰囲気をお楽しみください！

※重要※
参加希望の方は人数が埋まってしまっても諦めず、補欠に登録しておいてください。
なるべく全員参加できるよう色々検討します。

参加者多数のため、開催場所を変更いたしました。ご来場前にご確認ください。
