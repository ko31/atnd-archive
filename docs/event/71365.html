description

弊社はスクールアルバムを長年手がけてきた九州の上場企業です。このたび新事業としてネット事業を始めるのにさきがけ、無料のオンラインアトリエソフトを開発し、無償で提供することになりました。

イラスト、写真、てづくり雑貨などなどオリジナル作品を展示してみませんか。自分だけの「オンラインアトリエ」が無料で持てるんです。そしてｅコマース対応もできているため、そのまま販売も可能なのです。カメラや絵画に興味のある方はぜひご活用いただきたく思います。

　大阪梅田で無料の説明会が開催されますのでご興味ある方、ぜひご参加ください。ソフトもセミナーもすべて無料です。ほかにも、当日の参加特典があります。
　またオンラインアトリエの解説以外にも、当日、プロによる撮影セミナー（無料）もありますので「自分の作品、うまく撮れるかなあ」なんて方もでも安心です。プロのカメラマンさんから、スマホしかカメラはもっていないという人まで、ぜひこの機会にお集まりください。

　なぜ無償で提供しているのか？そうお思いの方も多いでしょう。それは当面無料でご活用いただき、それらの写真や絵画を１冊にまとめたいなんて要望もあるでしょう。こんな時は、オンラインアトリエに付加機能としてついている「アルバム化希望」ボタンを押していただければ、ここからは有料になりますが、記念アルバムとして完成品を郵送で送付させていただきます。弊社はもともとアルバムの会社ですので、仕上がりは任せておいてください。
　この機能を活かせば、たとえば友人の結婚式、新郎側、新婦側の友人が写真を撮りあって、ひとつのオンラインアトリエにのせます。そして全部集めた写真の中から、一番いい写真をレイアウトして、記念アルバムにすることもできます。これをご結婚のお二人にプレゼントする。こんな使い方も出来てしまうのです。そのほか活用方法はたくさんあります。
　まずはぜひ無料でオンラインアトリエをお楽しみいただき、より上手い使い方を是非発見していただければ幸いです。
