クインターネット業界で働くクリエイターのスキルアップを支援し、未来のインターネットサービスを創るクリエイターを育成するサイバーエージェントの教育プログラムです。

  • スキルアップして、スマートフォン業界で活躍したい
  • 同じクリエイターともっと交流を持ちたい

そんな皆さん、ぜひお気軽にお越しください。

■ セッション概要と出演者のプロフィール

「モバイルウェブ開発ベストプラクティス」

モバイルウェブ開発が本格化するとともにモダンブラウザも台頭しはじめてきました。それと時を同じくしてBackbone.jsなどのJavaScript MVCライブラリやSaSSなどのCSSプリプロセッサなど様々な開発ツールやHTML5 Boilerplateなどベストプラクティスが多く公開され始めています。
今回はモダンフロントエンド開発に欠かす事ができないツールとベストプラクティスを紹介します。


- 斉藤 祐也

ウェブデザイナー／フロントエンドデベロッパー
1979年生まれ。ネバダ州立ラスベガス大学、プリントジャーナリズム学部卒。株式会社All Aboutオールアバウトで6年、2012年2月からは株式会社サイバーエージェントへ。個人では、フロントエンドデベロッパのためのミニブックというコンセプトのブログ CSS Radar 、海外のウェブ開発情報をキュレートしているメールマガジン CSS Radar Newsletterを運営。


「アプリクリエイター採用Special講座 説明会」

クリエイティブアカデミーのプログラムでは

  • 若手クリエイターの技術力向上支援を掲げ、無料開講いたします
  • 休日開講により、現在の仕事を続けながらの通学が可能です
  • より専門的な知識や高度な技術を学ぶことが可能なプログラムです
  • 最終課題および面談に合格時にされた場合、弊社の選考ステップの一部をスキップできる特別選考パスを付与します。

- CA人事部

クリエイティブアカデミーとは…

クインターネット業界で働くクリエイターのスキルアップを支援し、未来のインターネットサービスを創るクリエイターを育成するサイバーエージェントの教育プログラムです。

その第3回としてFlashをメインに活躍されていた方々や、HTMLコーダーとして働いていた方の、キャリアチャレンジを応援するHTML5、CSS3、JavaScriptの養成講座を開講します。「Flashには詳しいが、HTML5やCSS3などの最新技術を身につけて、スマートフォンアプリを開発したい」「AndroidもiOSも両方開発してスマートフォン業界で活躍したい」とお考えの方を募集します。

2012年8月25日(土)～2012年9月25日(土)　全5回

  1. 8月25日(土) 「HTML/CSSの基本」
  2. 9月1日(土) 「JavaScriptの基本とインタラクションを実現するための知識」
  3. 9月8日(土) 「HTML5 API / Canvas Animation」
  4. 9月15日(土) 「正しいHTML5マークアップとブラウザ対応について学ぶ」
  5. 9月22日(祝土) 「アニメーションを中心にしたCSS3での表現力の幅を広げる」
