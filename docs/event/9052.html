ジオメディアサミット西日本のミニイベントを開催します。

昨年に引き続き、FOSS4G（Free and Open Source Software For Geospetial）の中の１コマをお借りして、交流をメインとした大ライトニングトーク大会を行ないます。
このFOSS4Gも関西オープンソース２０１０の会場内で行われます。いつものジオメディアサミットと違って、GISの人や、オープンソースのコミュニティと交流する、絶好の機会です。

イベントの後は、会場から徒歩３０秒のビュッフェにて昼食を兼ねた懇親会を行ないます。この懇親会はノンアルコールですが「きちんと交流できる」と昨年も好評でした！

ミニイベントなので定員が少ないのですが、奮ってのご参加をお待ちしています！

（第三回ジオメディアサミット西日本については現在準備中です。詳細は後日発表します。）
