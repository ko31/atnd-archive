Unityでゲームを作る！

Unity、もしくはゲーム作りに興味がある方に向けた勉強会です。
実際に手を動かしながらゲームをひとつ完成させて、Unityの魅力に存分に触れましょう！

懇親会を含め、参加費は無料です。

どんなものを作る？

こちらからは特に制限は設けません。Unityで作ったゲームであれば何でもOKです！ 3DアクションゲームやRPG、2Dのシューティングなど作ってみたいものを自由に作っていきましょう！
また Oculus Rift や Leap Motionといったガジェットをこちらの方でご用意する予定ですので、そちらの方も是非ご活用下さい！

※ 開発時間が限られるため、アイデアはイベント当日考えるよりも事前に考えて頂けるとスムーズにご参加頂けます。
以下のサイトを見ながら考えるとイメージがつきやすいかと思われます。

Unity入門 (全26回) - プログラミングならドットインストール
初心者向け公式チュートリアル
中級者向け公式チュートリアル

対象者

Unity、もしくはゲーム作りに興味のある方。
プログラミング経験のない方や、Unityを触ったことがない初心者の方も大歓迎です！

持ち物

ノートパソコン
Windows または Mac OS X が入ったもの。

Unityはダウンロード、インストールに時間がかかるので、あらかじめ最新版のUnity 5をインストールしておくようお願いします。
また、初回起動時にアカウントの登録が求められるのですが、そちらの方も済ませておくようよろしくお願いします。

当日のスケジュール

・10:00-11:00 Unityやエディタの触り方等の説明
・11:00-17:00 開発&リリース (昼食は各自で)
・17:00-18:00 成果発表
・18:00-20:00 懇親会

景品

イベント終了時に優秀な製作物を決め、制作者には株式会社リクルートホールディングス様より以下の景品が贈呈されます！

・Leap Motion + お食事券
・ゲームパッド + お食事券

両方の景品に人事の方にご飯をご馳走して貰えるお食事券が付いてきます！

メンター

当日はメンターとして CAMPHOR-メンバー が会場に居ます。解らない事があった際にはお気軽にご質問下さい。

その他

本イベントは、京都IT系学生コミュニティ CAMPHOR- 主催、株式会社リクルートホールディングス後援により開催します。
CAMPHOR- HOUSE 内では、電源と Wi-Fi をご利用になれます。
会場の場所が少々分かりにくいため、
お困りの場合は @CamphorKyoto にリプライを飛ばして頂けると対応致します。
