2015年も FI 関西はイベントを随時開催します！

今年の第１弾は、2014年春学期でウェアラブル端末での起業を果たした Up Performa 代表の山田修平氏にスピーカーとして登壇頂きます。

FI 関西の卒業からわずか４ヶ月で、ラスベガスで開催された世界最大の家電見本市 CES でのデビューを果たした山田氏。

FI で得たノウハウをどのように活かし、短期間で急成長を成し遂げたのか。その秘訣を伝授して頂きます。

ハードウェア・IoT・ウェアラブルに興味がある方は、ぜひこの機会をお見逃しなく！
