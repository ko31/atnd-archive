33回目のSWWDC(Sendai World-wide Developers Community)は、 恒例のApple Special Eventふりかえりをします。

開催概要

9月12日深夜に行われる、Appleの新製品発表イベント「Apple Events - Keynote September 2017」の復習会を行います。基調講演のビデオを見つつ、内容をあれこれ語り合います。

  • 「眠いので基調講演を見れなかった」という方も。
  • 「起きてたけど英語がわからんかった」という方も。

みんなで集まれば、バッチリ内容が理解できるのではないかと思います。発表を見て、今後発売されるApple製品をおさらいしましょう。

タイムテーブル

  • 18:20- 開場
  • 18:30 - 20:50 キーノート鑑賞。内容についてあれこれ語り合います。
  • 21:00- 終了

参加費

  • 参加費は無料です。

会場

  • 株式会社ナナイロ セミナールーム
  • 〒980-0014 仙台市青葉区本町2-3-10 仙台本町ビル3F
  • 注意1 移転のため、以前とビルの場所が変わっています
  • 注意2 18時以降は正面玄関が閉鎖されます。ビルの北側入り口(StreetView)からお入りください。
  • 注意3 20時以降は北側入り口も施錠されてしまいます。20時以降に来られる方は、@totottiまでどうにかして連絡ください。

懇親会

平日開催のため、懇親会の開催は未定です。閉会後に希望を聞いてみて、時間がある人が多いようであればやります〜。
