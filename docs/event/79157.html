高知ゲームアプリ勉強会(cocos2d-x,Unity)#4@カフェ＆レスト アイランド

高知市にてゲームアプリの勉強会を開催します。
自分の時間を使って自分のアプリを作ります。
初心者の方でも丁寧に教えますので大丈夫です。
各人の進捗に合わせて、説明しますので初回の方でも安心して参加ください。
ゲームアプリ以外でもプログラミングで困っていれば参加可能です。
対応可能な言語は以下です。
　C,C++,Java,PHP,Python,Objective-C,JavaScript

目的

プログラミングが出来るようになる。
企画が出来るようになる。
ゲームアプリを作り収益を得る。

コース

３つのコースがあります。
それぞれ自分で選んで進みます。

プログラマー育成がっつりコース

　CとC++を学んでcocos2d-xを使いゲームを作ります。
時間はかかりますが、ゲームプログラマとしての基礎が身につきます。
　

プログラマーお手軽ゲームアプリ最速リリースコース

　難しいことを飛ばして、Unityを使ってさくっとゲームアプリを作ります。

プランナー・デレクター・シナリオライターコース

　絵も描かないし、プログラミングもしません、アプリの企画と進行をします。
ゲームのシナリオのこちらです。

自己紹介

25年ぐらいC言語でプログラミングをしていてその後6年以上、AndroidとiPhoneのプログラミングをしています。
cocos2d-xとUnityでゲームアプリを作っています。
高知市にある株式会社テラシュールウェアのプログラマーです。
