回を重ねるごとに議論が深まる『マネタイズナイト』、ついに12回目の開催です！
今回もメディアマネタイズを熱く議論します。

「TOPコンテンツプロバイダ各社が選ぶスマホのマネタイズ手法とは」

スマホではアプリのダウンロードの実に約9割が無料とも言われる中、フィーチャーフォンで月額課金をフル活用していたCP各社は、スマホでどのようにマネタイズしようと考えているのか。

キャリアやマーケットの月額課金が進みつつある現状において、スマホでも月額課金が定着するのか、はたまた広告モデルが主役なのか、今こそ改めて考えるべきタイミングです。


本イベントはいわゆるセミナーではなく、参加者の皆様にも積極的に議論に参加して頂く座談会的スタイルで実施しています。会場はカジュアルな雰囲気ですので、
メディア運営者様同士の交流や情報交換の場としても活用ください。
