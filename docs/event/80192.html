description

社会起業家育成のプロフェッショナルが、ソーシャルビジネスに興味・関心のある方、NPO法人やボランティアグループで活動をされている方、起業やプロジェクトの立ち上げを目指している方などを対象としたソーシャルビジネスセミナーを開催いたします。
現代では多様性に富んだ社会問題が多く存在し、社会全体が閉塞感を抱えながら、先の見えない時代を切り開こうとしています。
この環境を突破するためには、本気と志、そしてイノベーションを起こす力が必要です。
そのために、社会をより良い方向へけん引するイノベーションを起こす力を持った、「ソーシャルイノベーター」が求められています。
本講座では「ソーシャルビジネス」のメカニズムを取り上げることで、どうすればあなたがソーシャルイノベーションを起こす担い手となれるのか、そのヒントとなる社会起業大学を卒業し、実際に事業を行っている団体などの取り組みを、具体的な事例を交えてお伝えします。特に、300名以上にもなる社会起業大学の卒業生たちは、東京をはじめとする全国で活躍をしています。そんな彼らの活躍や都市部でのソーシャルビジネスの最新事情を余すことなく、東京本校の学長である田中勇一氏から直接話を聞く機会など、滅多にございません！
ソーシャルビジネスに興味、関心がある方向けの入門講座ですが、既にNPO法人などの団体での経営を行ってる方にも活動のヒントを得ることが出来る内容となっています。
まさに社会的課題の解決を目指す人、すなわちソーシャルイノベーターのための特別公開講座です。
参加費無料の講座となっておりますので、この機会にぜひご参加ください。

■講師　田中　勇一
京都大学理学部卒業後、住友銀行（現三井住友銀行）入社。
米国カーネギーメロン大学にて MBA 取得後、帰国して2年間、銀行のALM業務携わる。
その後、ビジネススクール運営会社に転職し、人材・組織開発コンサルティング会社の立ち上げに参画。
経営人材の紹介をコアとする成長事業支援会社を経て、新銀行東京設立プロジェクト草創期に2年間携わり、採用統括責任者として 5,000 人以上の応募があった公募採用で陣頭指揮をとり、銀行立ち上げに大きく貢献する。
現在は、リソウル株式会社を設立し、大手 GMS による銀行設立の人事業務コンサルティング、経営相談、起業支援、転職支援、カウンセリング等に取り組む傍ら、社会起業大学を設立し、社会起業家育成にも従事。卒業生の数は、300名を超える人数です。また、2013年4月より、多摩大学大学院客員教授に就任。
