ドミニオンの本格派大会！

先日発売された拡張「夜想曲」を使った大会形式のイベントです！ あなたもドミニオンの腕試しをしてみませんか？

ここで勝てれば日本選手権優勝も夢じゃない！

ぜひ奮ってご参加ください！

概要

  • 日時

    • 3/9(土) 12:30-20:00
      • 12:30-13:00 受付
      • 13:00-20:00 ドミニオン 基本第二版と夜想曲を利用したスイスドロー5回戦

  • 場所

    • ハロー貸会議室 新宿曙橋 会議室（2F）
    • 道案内
      • 都営新宿線　曙橋駅　A4出口徒歩3分
      • JR中央・総武線　市ケ谷駅　徒歩10分

  • 参加人数

    • 32名

  • 参加費

    • 1500円
    • ドミニオン基本第二版と夜想曲(スリーブ入り)をセットで持ち込んでくださった方は500円引きします。
      • 英語版でもokです。
    • 学生の方は500円引きします。
      • 大学生、大学院生も可。学生証の提示をお願いします。
    • 持ち込み割と学生割は重複okです！

  • 参加資格

    • ドミニオン基本第二版のルールを把握していること
      • 細かいルールや注意点、ルールの疑問点などについては確認の時間を設けますので、大体知っていればそこまで神経質にならなくても大丈夫です。
      • 夜想曲については新しいエキスパンションであることを考慮し、遊んだことがなくても参加していただいて問題ありません。

  • ルールサマリ

    • ドミニオン日本選手権の予選に準拠したスイスドロー5回戦
    • 基本第二版＋夜想曲からランダムに作られたサプライを利用(全テーブル同じサプライを利用)
    • 日本選手権と同様の1位6pt,2位3pt,3位1pt,4位0ptの勝ち点形式
    • 試合時間は45分とし、決着がつかない場合は45分終了時点から2週目の4番手のプレイヤーまでの手番で打ち切りとし、その時点での勝利点で順位を決定します。

  • カードの取り扱いは丁寧にお願いします。
  • ユーザー大会ですので、参加者のみなさまのご協力が不可欠です。楽しい大会にできるよう努めますので、参加者のみなさまも円滑な大会運営にご協力お願いします。
  • 運営をメインで行うメンバーも大会に参加します。予めご了承ください。
  • 勧誘行為、迷惑行為は禁止です。
  • 当日の告知などは会場＋主催Twitterで行います。差し支えなければ主催のアカウントをフォローしていただけると便利です。
  • 不明な点があればコメントお願いします。

ルール詳細

  • ゲームのマッチングについて

    • ゲームは4人戦で行われます。ただし、参加人数が4で割り切れない場合、3人戦テーブルがいくつか発生する場合があります（3人戦テーブルは、獲得VP を3/4 して計算します）。
    • プレイヤーのマッチングはドミニオン日本選手権と同様のスイスドロー形式で行います。
      • 対戦相手の勝率(オポネント・マッチ・ウィン・パーセンテージ)は日本選手権と同様考慮しません。
      • 同じプレイヤーと複数回マッチングすることもあります。

  • サプライについて

    • 当日はすべてのテーブルで同じサプライを利用します。
    • 基本第二版と夜想曲を利用し、ランダムな方法で決定したサプライでプレイします。
    • 10種類の王国カードについては以下の割合でランダムに決定します
      • 1回戦 基本第二版5枚、夜想曲5枚
      • 2回戦 基本第二版5枚、夜想曲5枚
      • 3回戦 基本第二版4枚、夜想曲6枚
      • 4回戦 基本第二版4枚、夜想曲6枚
      • 5回戦 基本第二版3枚、夜想曲7枚
    • 一度使用した王国カードは以降の試合では使いません。

  • ルールについて

    • 試合の前にルールについて質問をする時間を設けます。細かいルールで不安な所があればその際に質問をお願いします。
    • 基本的にドミニオン第二版、夜想曲の公式ルールに従いますが、大会進行の都合上、1試合45分+アルファの時間制限があります。(詳しくは後述)
    • シャッフルは必ずディールシャッフル＋他のプレイヤーにカットをしてもらうようにしてください。
    • ジャッジについては専用のジャッジはいませんので、基本的にテーブルで決めていただくことになります。ゲーム中プレイミスをした、ルール解釈ミスをしたなどがあった場合はテーブルで相談し、当事者以外の過半数が認める方法で解決してください。

  • 時間制限について

    • 45分終了時点でプレイしているプレイヤーから2回目の4番手のプレイヤーのターンが終了するまでゲームを進行し、その時点での獲得VPで順位を決定します。
      • (例1) 2番手のプレイヤーのターン中に試合時間が終了した場合、2->3->4->1->2->3->4 の順でターンを実施します。
      • (例2) 4番手のプレイヤーのターン中に試合時間が終了した場合、4->1->2->3->4 の順でターンを実施します。
      • 45分終了後に通常のドミニオンの終了条件を満たした場合は、その時点で通常通りゲームは終了します。
    • 時間制限があるルールですので極端な長考(何のアクションも購入もしない状態で1分以上考えること)は避けてください。
    • 逆に他のプレイヤーを極端に急かすことも禁止します。
    • 露骨な時間稼ぎはやめてください。

  • その他

    • 当日遅刻した場合、ゲームに参加できないことがあります。運営の都合上、途中からの参加はできません。
    • 当日途中で都合などで途中で大会からドロップする場合は必ず運営にお知らせいただいてから離脱をお願いします。
    • ゲーム中に他のプレイヤーにアドバイスしたりアドバイスを求めたりすることは禁止します。
      • ゲームが早く終わった際に他のテーブルの観戦も自由にしていただいて構いませんが、ゲーム内容に口出ししたりすることは禁止です。
