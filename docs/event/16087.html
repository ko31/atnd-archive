地理学と隣接分野の若手有志による「初心者向けGIS勉強会」と、考古学の若手有志による「Archaeo-GIS Workshop」が初のコラボレーション。東京西部の武蔵野台地を流れる野川で巡検を行います。参加表明、お待ちしています。

以下、仕掛け人のツイート。

@emirts : 野川巡検しましょう、Geoらしく!! 崖線、野川仙川、武蔵小金井駅前再開発、河川改修＆修景、中央線、庭園、あ、調布飛行場も良いですね!!

@yaskondo : 野川流域には旧石器時代と縄文時代の遺跡がたくさんあります。

@tosseto : 野川は先史時代ももちろんですが、中世の六郷用水や戦後の宅地開発という、幾つかのインパクトという点で面白いですよね。環境的にもドブ川から、1990年代後半にかけて回復する様子を間近に見られた川の一つです。

巡検案

7月31日（日）午前9時　JR西国分寺駅集合　小雨決行

同日午後5時 東急二子玉川駅にて解散予定（途中参加・離脱OK）

  • 大まかなルートは、西国分寺駅から姿見の池に行き、お鷹の道沿いに歩き、北上し、日立の野川源流を外から眺め、そこから野川沿いに歩いて途中にある公園や地形、植生、遺跡を見ながら、最下流部の二子玉川まで行きたいと思います。学芸大野川巡検で必ず行く、東京パミスの露頭があるポイント、通称「レンタカー屋の裏」も行きたいと思っています。もしかしたら、みんな採集して行くのでない可能性もありますが、東京パミスの上部に縄文時代に人が住んでいた証拠でもある黑ボク土壌とさらに縄文遺跡もあるので、 @yaskondo さんが解説。
  • 上流部で湧水が発生するメカニズム、河川保全活動、国分寺から小金井以降の景観の違い、京王線国領駅から二子玉川までの範囲で現在行われている改修工事の概要とそれに対する市民の動き、小金井の貯水池について、 @erimts さんが説明します。
  • 天気が良ければ、 @ta_niiyan さんの自転車チームが出動します（二子玉川発着）。
  • 巡検中、GPSログを取ります。解散後、希望者がいれば、東京工業大学（大岡山）に移動して、ログの解析を行います。
