大変長らくお待たせ致しました！
日本Androidの会の関西支部主催で、Androidのセミナーを行います。

参加資格はありません。
Androidに興味・関心のある方は、
ぜひ参加してみてください。

※1アカウント、1名までとさせて頂きます

※6/30追記
既に定員オーバーですが(嬉しい悲鳴です)、
立ち見(or床に座ってもらう)でよろしければ、
参加して頂いて構いません。

ただし、電源を確保できない可能性が高いので、
電源タップをご自身の名前を書いて持参してもらえると
非常に助かります。

ご自身のノートPCにAndroidのSDKをインストールして持参して頂くと、
よりハンズオンの講義が楽しめるかと思います。

インストール方法はこちら

2009年6月現在の最新版(Android 1.5 SDK, Release 2)をインストールしてください
インストール済みの方で、SDKのバージョンが1.5でない場合は、
1.5にアップデートしてください。
(バージョンが1.5であることを前提に話す講義もあります)


皆様にお会いできることを楽しみにしております。

*講義内容
・有限会社シーリス 有山圭二
「XML画面デザインのノウハウ」
XMLで画面デザインをする時に気をつけること。レイアウト種類毎の得手不得手や、
複数のXMLレイアウトファイルをどのように扱うのか、
実践の開発で使っている泥臭いテクニックを一挙大公開。
資料は後日配布予定です。

・日本Androidの会　杉本礼彦
「Androidは本当に流行るのか？ビジネス的観点、技術的観点で分析」
ハンズオンではありません。

・日本Androidの会　宮川彬央
「AppWidgetでWidgetを作ってみよう！」
ハンズオンです。
パソコン、SDK 1.5が必要です。

・株式会社ブリリアントサービス 藤井洋祐
「AndroidとSQliteと、ときどきオトン」
ハンズオンではありません。

・エモーションプラス 原秀樹
「Canvas使ってスタンプポンポン！」
タッチパネルのイベントによって画面にスタンプのパターンを
描画するアプリをハンズオンで制作してもらいます。
Androidアプリを作ったことがまだ無い初心者を対象としています。
＜解説する題材＞
・タッチパネルのイベント onTouchEvent
・描画用クラス Canvas, Drawable
・描画パターンを扱うリソース res/drawable
