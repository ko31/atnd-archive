“Oh!いい～町会”（大井町会）Vol.10

title “Oh!いい～町会”（大井町会）Vol.10
catch 〜大井町会新年会〜オオイ町で、オオイにイイ事を考え実行する会
description

あっという間に新年を迎えますが、昨年は震災というかつて経験のしたことがない出来事がありました。

震災以降、今年の一言にもなりました「絆」を大切に、大井町界隈の方、また、その友人・知人など幅広い方々と、オオイにいいこと考えながら開催してきた「オオイ町で、オオイにイイ事を考え実行する会”Oh! いい～町会”（大井町会）」ですが、新たな年を迎え10回目の開催となります。

毎度参加の方も、一度だけ参加された方も、来る機会が無かった方も、お気軽に新年の喜びや叫びを分かち合いつつ交流できればと思っています。

新年会は20時からですが、パトロールマラソンを終えてからの参加者は、21時から合流となります。

新年にふさわしく爽やかな汗を流したい方は、パトロールマラソンからの参加お待ちしてます。



＜全体スケジュール＞

第1部：18:00～　旧東海道パトロールマラソンで爽やかに汗を流そう！（任意参加）
第1.5部：～21:00　北品川温泉 天神湯で爽やかな体になって新年会に行こう！（任意参加）
第2部：20:00～　大井町会新年会 大ジョッキで新年大盛り上がり！


＜第1部〜旧東海道パトロールマラソンで爽やかに汗を流そう！〜＞（任意参加）

人力車の佐藤さんが行なっている「旧東海道パトロールマラソン」は毎週水曜日に行われています。
1月11日は、ちょうど水曜日ということもありますので、新年会前に走れる方は任意で新年早々に爽やかに旧東海道を走り抜け、地域安全のパトロールをしてみてはどうでしょうか？

走らずに、夜の旧東海道をちょいと粋に散歩なんぞはいかがでしょうか。

※受付で専用ビブスを身につけてますので、旧東海道交流館にお集まりください。

■大井町会とは？

「大いに“イイコトする”町！大井町！」

を合言葉に、大井町から日本全国、東北、そして地域へ元気と支援の輪を広げるための活動を目指して行きます！

オオイに語り、オオイに呑みながら自分達に出来る事を考え、そして実践して行きましょう！

オオイにかはちょっと…
まずは何かしたいのだけど、何をして良いか…
やりたい事はあるのだけれと、1人では…

という方オオイに歓迎します！

（あと、決して大井でなくても大歓迎です。）
