「 .NET Framework 4.5 MVC4 OAuth認証アプリケーションのススメ」
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　株式会社テイルウィンドシステム　岡田 裕二

　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　株式会社テイルウィンドシステム　坂本 純也

　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　株式会社テイルウィンドシステム　佐藤 昌和

　　　その対応した内容を解説します。
