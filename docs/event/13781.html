MovableType5 カスタマイズ勉強会

4/16(土)に名古屋千種でMTDDC Meetup Nagoyaを開催します。
Movable Type 5のカスタマイズにフォーカスを当てた勉強会です。

2008年ぶりの、MTに特化した勉強会になります。
前半は講演が3本、後半は120分集中ハッカソンの構成で、1日かけて
MTについてじっくり学べる形式となっています。

      小林 晋也

      北海道帯広市出身。Movable Typeの専業ベンダーとして北海道帯広市に 株式会社スカイアークシステム を創業。
      2005年11月、Movable TypeをCMSとして活用するプラグイン「MTCMS」をリリースしてCMSの低コスト導入を推進。SKYARC Solanowaや多数のプラグインを開発し、企業の情報発信を「かんたん」にするお手伝いをしています。

      講演内容「Movable Typeの可能性について」

      Movable Typeはテンプレートの設計によって様々なウェブサイトを実現することができます。この講演では、どのようにテンプレート設計すべきか、さらにプラグインの活用でどこまでMTの可能性が広がるかお話しします。

      120分集中ハッカソンの内容

      120分間、MovableTypeに関することをがっつり作業しましょう。
      内容はMovableTypeに関する事であれば何でも構いません。

      例）

      • ベータ公開中のMovable Type 5.1をひととおり触ってみる
      • Movable Type 5のテーマを作ってみる
      • Movable Type 5 のプラグインを開発してみる
      • 他の開発者やデザイナの意見を聞いてみる
