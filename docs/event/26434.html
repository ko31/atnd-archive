現在のクラウドコンピューティング会を牽引する「Amazon Web Services」
Amazon Web Service（以下、AWS）は、「コンピュータのリソースを初期投資無しで、しかも従量課金で利用できるサービス」です。
このサービスには、仮想サーバや永続的なストレージ、データベース、ネットワーク、メール、DNS、監視を提供しており、ユーザーの要望に応じて機能がどんどん追加され、まさに痒い所に手が届くクラウドサービスになってきています。

すでに国内外問わず多くの企業で採用されており、さらにはスタートアップのインフラとしてAWSが採用されています。
世界中で数十万のエンジニアがAWSを利用しており、今後は開発者/ITプロフェッショナルにとって必須なクラウドサービスになることは間違いありません。

導入事例も数多く発表され、その中でも昨年の3.11 東日本大震災直後にシステムインフラがダウンした際に、
既存システムをAWSに移行することにより短期間で復旧させた事例もあります。

今回は、長崎で初めての勉強会開催致します。
初回と言うことで不慣れな点もございますが、皆様の導入・活用事例及び初めて利用されるかたもAWSが提供する
クラウドサービスについて理解して頂ける会になるように努めます。

AWSに興味があって使ってみたい方、AWSは触ってみたけれどももう少し深く知りたい方、AWSをもう使われていて周りに仲間が欲しい人、
AWSをすでに使いこなしていて他の方にその使い方を広めたい方、気軽にどしどしご参加ください！！

ちなみに九州地区はAWS熱が高いので、残り大分UGが発足すれば九州制覇です！！
AWSを使って九州を盛り上げてみませんか。
