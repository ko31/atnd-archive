title STOICな休日
catch 〜寝て過ごしても騒いでも同じ一日の巻〜
description

あの人気企画がパワーアップして帰ってきた！
日本一ストイックな読書会として有名なSTOICREADINGが
自信をもってお届けする企画、

「第二回ストイックな休日」

開催決定です！！

■第2回ストイックな休日■

【開催日時】
10月10日(月)
13:30〜17:00

【開催場所】
大阪市立大淀コミュ二ティーセンター
(大阪市北区本庄東3-8-2)

【集合】
13:30に現地集合

【アクセス】
大阪私営地下鉄堺筋線・谷町線
「天神橋筋六丁目」11番出口より 徒歩５分

【コンテンツ】
1.ワールドカフェ
今ちまたで注目されている会議手法です。
ある普遍的なテーマについて話合います。

2.ストイックリーディング
普段行っているストイックリーディングのエッセンシャル版です。
読んできた本の学びをプレゼンします。
どんな本でもかまいません、オススメの本について３分間プレゼンしてもらいますので準備してきてください。

3.シークレット
みなさんにわくわくをお届けするコンテンツです。
お楽しみに！

【費用】
５００円(会場費・軽食費)

【持ち物】
プレゼン用の本
筆記用具
家に余ってるお菓子とジュース

参加ご希望の方はストイックリーディングのメンバーか、
もしくは高橋(cube_@docomo.ne.jp)までご連絡ください。
あなたのご参加をストイックにお待ちしています！
