概要

書籍「基礎から学ぶ facebookアプリ開発」をもとに勉強会を行います。
http://www.amazon.co.jp/dp/4863540949/

第2回目では、～P107まで進みました。
第3回目では、Chapter3 (～P191) 各種 Web API の利用について学びます。

参加者は、書籍掲載の各種 Web API の中から興味のあるものを選択し、
サンプルアプリを作成してきてください。
コードをスクリーンに映しながらレビューをしていきます。

書籍に書かれている内容以上の機能を付けれるようであれば、さらに勉強になると思います。
書籍に掲載されていない Web API (e.g. Twitter) でも構いません。

書籍掲載の各種 Web API
YouTube API
Amazon API
face.com API
Google Earth API
Google Ajax Feed API
Ustream API

途中参加を検討される方へ

現在輪読中の本は、PHPでの開発経験がある方にとっては、比較的やさしめの内容です。
準備にさほど時間は要さないと思いますので、途中からの参加でも全く問題ないと思います。

Facebook API は、もちろんのこと、YouTube, Ustream, Google, Amazon 等の API 群、
CakePHP や、 enchant.js を使ったシューティングゲームアプリ等、
色々なものに触れることができる内容となっておりますので、興味がある方は是非ご参加ください。

Facebookアプリを動かすために必要なWebサーバの条件は、「PHP5」「MySQL」「SSL」です。
グローバル環境が必要になりますので、レンタルサーバ等を借りられるのがよいと思います。
活動報告 に目を通していただけると、よりスムースかと思います。

第2回目までの内容のフォローアップも行っておりますので、興味をもたれた方は、まずはそちらにご参加ください。
SwimmyStudy #2.1 【フォローアップ】基礎から学ぶ facebookアプリ開発 vol.2

対象者

  • PHP を使ってある程度の開発を行った経験のある方
  • facebookアプリの開発に興味のある方
  • 技術の話をできる仲間・環境を渇望している方

他言語でのウェブサービス開発経験があれば問題はないかと思いますが、自助努力が必要です。

持ってくるもの・準備してくるもの

  • 基礎から学ぶ facebookアプリ開発
  • 開発が行えるPC
  • 開発環境（PHP5, MySQL, SSL が利用できるグローバル環境に置かれたサーバ）

Wi-Fi環境をこちらで準備します。もしPocketWi-Fi等をお持ちでしたら持参していただくと助かります。
