引っ越しを考えている、親の家がちらかっていて心配、自分の家を片付けたい…、「片付けたい」と思う対象や目的は人それぞれ。どんな目的であれ、毎日暮らしている家だからこそ、家の片付けは人生を見つめなおすことにもなります。これまでに2500軒以上の家の片づけに携わってきた「片付けのプロ」による特別講座。あなたもこれを機会に、片付けと向き合ってみませんか？

講師　野口　幸恵
株式会社サマンサネット専務取締役
日本ホームステージング協会理事　公認講師
現在までに伺ったお宅は2500軒以上。片づけに悩んでいる方の救世主、建設会社・リフォーム会社・不動産会社等でのお片づけセミナー講師多数、ＴＶ朝日『お願い！ランキング』『関ジャニの仕分け∞』出演
