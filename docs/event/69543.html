description

シナリオライターや小説家を志望されている方、朝の出勤前や休日の朝にお話しましょう。
将来の展望や勉強の仕方をざっくばらんにお話できればと思います。ぶっちゃけ聞きにくいお金事情など、プロの世界に飛び込んだ後でないとなかなか聞けない本当の業界事情などもお話します！

サラリーマンをやりながら、文筆業を目指している方に私の経験や注意すべき点、プロのなり方などのお話するのを趣旨としています。

■参加資格
・シナリオライターや小説家を志望されている方（プロ禁止）
・２０歳～３０代
・社会人経験２年以上
・ライターとしての勉強期間は問いません。初心者歓迎です♪
※申し訳ありませんが、学生さんはお断りしています

■ご注意
時間が限られたカフェ会になりますので、テクニカルなお話はしません。専門的な学校や弟子入りするなどして、学んでくださいね！

■主催者
３０代の既婚男性。大学卒業後にアニメ業界へ就職。制作部で働きながら脚本の勉強を続けて、2005年にゲーム業界でシナリオライターデビュー。その後約10年にわたって、家庭用ゲームのシナリオライターとして携わり、ゲームファンからの賞賛と罵声を浴びながら、携わったタイトルは累計で250万本以上のセールスに達する。ドラマCD、短編小説などの仕事もあり。
現在ライター業は休業し、企業に所属して生計を立てている。自身の経験をムダにするよりはと思い、朝活を開催。

カフェ会なので、お気軽にご参加ください♪
