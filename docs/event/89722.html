【概要】

今後更なるクラウド普及に伴うセキュリティ技術の重要性やサイバーセキュリティの現状等、具体的な事例を合わせご説明させて頂きます。

クラウドは利便性があり初期コストが安く、拡張・縮小が簡単に出来るというメリットがありますが、環境が変わることで利便性に乗じたセキュリティリスクがあるためクラウド特有のセキュリティリスクを意識することやそれに対応するセキュリティソリューションを知っている必要があります。

今回のセミナーでは、セキュリティ運用の現場から見て見逃すことのできないクラウドにおけるセキュリティインシデントの事例とともに、グローバルなパブリッククラウドサービスでパーソル プロセス＆テクノロジーが多数の実績を有しているMicrosoft 社の Azure AD におけるID管理の事例やそのMicrosoft 社がグローバルに調査したサイバー攻撃の実態などについてご紹介させて頂きます。
