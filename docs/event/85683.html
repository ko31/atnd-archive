Unreal Engine Sapporo Meetup!

2017年2月25日(土)に札幌にて、Unreal Engine 4 (以下UE4) についての Meetup を開催します！

UE4 は エピック・ゲームズ社が開発している、マルチプラットフォーム向けのゲーム開発エンジンです。
近年AAAゲームタイトルでの採用や、ハイクオリティなモバイルゲーム、VR開発から、非ゲーム分野、エンタープライズ領域での活用も行われている、注目の環境です。

Meetup は共通の興味関心や問題意識などを持つ人たちの集まる場です。
経験の有無を問わず、UE4に興味のある方は大歓迎です。
札幌、北海道でも UE4 を盛り上げたい。そこから新しい何かが生まれるきっかけになれたらいいなぁと思っています。

開催概要

今回はショートセッションと Lightning Talk を行い、その後、参加者全員で交流の場を設けます。
4-5名のグループに別れて、お互いの関心事や UE4 についてざっくばらんにお話できればと思います。
特に話題に制限は設けませんが、きっかけとしてのお題や話題も用意する予定です(もちろん用意している以外のお話も歓迎です！)

Meetup を通して、エネルギーをもらったり、何かをはじめるきっかけになったり、新たな発見があれば幸いです。

参加費及び人数

  • 25名
  • 無料

運営について

札幌での UE4 Meetup は今回で4回目の開催となります。
2015年から、なつのやさん主催、近未来ガジェット研究所の主催で行っていました。
今回は個人主催の Meetup となります。至らぬ点もあると思いますがご指導・ご協力よろしくお願い致します。
