description

「３Ｒ推進関東大会ｉｎさいたま」の中で、キャンドルナイトに使う”エコキャンドル”や、”マイ箸袋”づくりのワークショップを行います。

エコキャンドルは、今年の７月７日に川口駅西口／リリアパークで開催された川口市主催のクールアースデーイベント「キャンドル・ナイト２０１０」で初めてお目見え！このイベントに使うロウソクを、環境に配慮したものにできないかということから、川口市役所が調整役となってみなさんの協力（協働）で生まれました。使用済みのロウソクを溶かし、もう一度新しいものにリメイクしてできあがったキャンドルです。

今回のワークショップでは、この「エコキャンドル」と「マイ箸袋」を作る体験をします。
