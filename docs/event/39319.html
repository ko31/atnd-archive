■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション1(30分)
　『AR技術の紹介とiPhoneを用いたアプリ開発』
　　デジタルソリューション株式会社の山根さん

　＜内容の紹介＞
　近年急速に普及しつつあるAR技術について、基本的なキーワード、
　メリット、使用されている技術や分野のさわりを紹介します。

　時間があればiPhoneとARライブラリVuforiaを用いた
　ライブコーディングも行う予定です。

　＜山根さんの紹介＞
　気がつけばMacユーザになっていたCGエンジニア。

　今年に入ってから広島でのIT系勉強会に参加させて頂いております。
　出席するだけでなく、コミュニティに何か貢献出来ないかと考えていたところ、
　しゃべり手のお誘いを頂きました。当日はよろしくお願い致します。

　現在はDS ANATOMYをはじめとした医療系学習用ソフトウェアを開発しています。
　OpenGL、Unity、ARなどのCGに関連する技術を日々勉強中です。

　- DS ANATOMY: http://ds-anatomy.com/
　- Facebook　: https://www.facebook.com/masato.yamane

　

●セッション2(40分)
　『おとのさまをすくいだせ！』大林さん

　＜主な内容＞
　旧市民球場跡地で出会ったおとのさまはとてもキュートでした。
　ですが彼もほんの数週間前、大ピンチな状況におかれていました。

　アクセス数が多いサイトというのは普段みえない
　様々なトラブルが降りかかります。
　某イベントもそんなトラブルに巻き込まれて
　開幕前に大変なことになっていたのを
　記憶されている方も多いのではないでしょうか。

　開幕をひかえアクセスが急増したとき、既存サーバが悲鳴を上げました。
　中の人たちの奮闘も実らず事態は悪化の一途。
　彼らがサジを投げたその瞬間、外部でヤイヤイと煽りまくってたインフラ屋が
　気づけば中の人になっていた！　何を言っているのか（AA略）

　やる事になったからには繋がるようにしなければならない、ということで
　どんな対策をうったのかを、お話できる範囲で大雑把に解説します。

　なお AWS に関係する話はしません。

　■レジュメ
　・はじめに
　・WEBサーバ
　・高負荷状態
　・なんとかしたい
　・正しく判断しよう
　・実施した対策
　・まとめ

　＜大林さんの紹介＞
　　2004年くらいからサーバを触りはじめ、
　　2007年からWEBの世界に入りました。

　　2012年末からフリーランス。屋号準備中。

　　WEB でユーザーの目に触れない部分はだいたい守備範囲なので
　　インフラエンジニア、プログラマー、稀にディレクターなどやってます。

　　クリエイターの能力を引き出した瞬間が一番の仕事のやりがいを感じる瞬間。

　

●セッション3(40分)
　『ウェブサイトを使用しての起業とコミュニケーションの重要性』
　株式会社ピーエイチジャパンの吉田さん

　＜主な内容＞
　創業に至るまでの経緯から、創業後のウエブサイトの運用についてと
　それに必要不可欠なコミュニケーションの重要性を説明します。

　＜吉田さんの紹介＞
　株式会社ピーエイチジャパンの代表取締役

　公私ともに、良い結果を得るにはどうするかを極めつつ、社会貢献をすることを使命とし、
　業務に関わる専門知識と、広島市の英語ボランティア通訳、ウエブサイトの企画等の経験をいかし
　約9年前に創業し今に至ります。　

　昨年の秋から紹介で、累積で軽く10回を超える、起業と仕事の悩み等の相談に応じています。
　本業ではありませんので、料金を一切頂いておりませんが、
　今回、藤本さんよりご依頼がありましたので、少しでもお役にたてばと思い今回のセッションでお話しします。

　株式会社 ピーエイチジャパン
　http://phjapan.jp/
