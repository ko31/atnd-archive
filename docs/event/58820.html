読書の秋！

今回は、大好評☆今巷で話題のビブリオバトルの第３弾です。

ビブリオバトルとは本を５分間でプレゼンしあって，みんなの投票でチャンプ本を決める書評ゲームです。

チャンプ本発表者には今回も豪華商品券をご用意しておりますので、みなさん奮ってご参加ください！

ルールは簡単。

1. 発表参加者が読んで面白いと思った本を持って集まる。
2. 順番に一人5分間で本を紹介する。
3. それぞれの発表の後に参加者全員でその発表に関するディスカッションを2～3分行う。
4. 全ての発表が終了した後に「どの本が一番読みたくなったか?」を基準とした投票を参加者全員一票で行い、最多票を集めたものを『チャンプ本』とする。
— ビブリオバトル普及委員会、ビブリオバトル公式ルールより

今回のテーマは、『本当におすすめしたい本』とします。

ジャンル問わず、みなさんの好きな本、おすすめしたい本、みんなに読んで欲しい本などなど、なんでもけっこうですので、あなたのとっておきの１冊を皆さんとシェアしてみませんか？

我こそはという方、学生・社会人問わずご参加をお待ちしております！ ＊テーマ＊
