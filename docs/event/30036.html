人生で成功する人、そうでない人、
その鍵は「EQにあります」


EQ（EmotionalIntelligence）は「心の知能指数」とも呼ばれ、
誰もが持っている能力で、誰でもその能力を高める事が出来るものです。

この「EQ」とは何かを理解し、感情能力と行動の関連に気づき、
自己理解を深めて頂く事を主眼としたセミナーです。

自分と自分が持てる能力を知り、明日からの人生を
「自分らしく」歩んで頂くための第一歩として頂きたいと
願っております。

ちょっとした”コツ”を知ることで、
「現状分析」と「能力開発」の進め方のヒントを理解いただけます。

【こんな方のご参加をお待ちしております】



  • いつも前向きにがんばっているはずなのに、報われた実感がない。

  • 日々の努力に違和感やむなしさを感じる。

  • 何をやっても続かない。

  • そもそも、何をやればいいのかがわからない。

  • 『就活や転職がうまくいかなかったらどうしよう』と少し不安に思っている人

  • 自己PRがあまり上手ではないと思う。自分の強みや弱み、自分らしさを発見したい人

  • EQを効果的に使ってコミュニケーション力をアップしたい営業マン

  • 社内チーム内で部下の指導に悩んでいる中間管理職の方

  • つい感情的になってしまいの子供との接し方に悩んでいるお母さん
