論理思考で人を「ワクワク」させるプレゼンテーション

イベント ユーザー
event_id 48770
title 論理思考で人を「ワクワク」させるプレゼンテーション
catch シナリオ作成と演出を行うための手法、プレゼンテーションの完成度を上げるための技法、実施と評価・改善のポイントを習得！
description

◆受講対象・レベル：
・自身のプレゼンテーションを見直し、飛躍的な向上を行いたい方
・特に、理系脳の人で、ビジネスで訴求できるプレゼンテーションノウハウを習得したい方

◆習得できる知識：
（１）プレゼンテーションの目的と「ツボ」が理解できる。
（２）自身行っているプレゼンテーションの現状を認識し、改善指針が策定できる。
（３）プレゼンテーションの基本と極意が習得できる。
（４）論理的で、訴求力の強いプレゼンテーションを行う手法を習得できる。
（５）人を感動（感情＋論理）させ、動かすプレゼンテーション手法が習得できる。

◆趣旨：
　プレゼンテーションの目的は、「聞き手のキーマンに、意図したアクションを実行させる」ことです。
　本講座では、自分自身の現状認知から、成功するプレゼンテーションに必要なノウハウを体系的に習得します。特に、シナリオ作成と演出を行うための手法、プレゼンテーションの完成度を上げるための技法、実施と評価・改善のポイントを習得します。

◆プログラム：
１．本日の目的
　　1-1．アイスブレイク
　　1-2．プレゼンの基本と現状確認（行動心理と認知科学）
　　1-3．自己紹介プレゼン演習（感情・論理・訴求力）
　　1-4．映像による現状確認と評価（改善指針の明確化）
２．キーマンの確認とシナリオ作成
　　2-1．テーマの確認（ゴール設定と評価尺度）
　　2-2．キーマンモデルの策定（理解と意思決定の特性）
　　2-3．シナリオマップとシナリオ作成（明確な論理構成）
３．コンテンツ作成
　（スピーチのシナリオ構成と演出設計）
４．リハーサルと評価・改善（明確な評価指針）
　　・グループリハーサル（３名）
５．グループプレゼン（6名）
　　5-1．代表プレゼンテーション（ＶＴＲ撮影）
　　5-2．映像再生・評価シート記入
　　5-3．全体フィードバック
６．効果的なパワーポイント資料の作成方法
　　・サンプル事例（ひな形）をベースに解説
７．まとめ＆フィードバック
