講演者プロフィール　/　登壇内容

◆土濱 健太郎氏
flexエンジニアとして金融機関向けコンテンツの開発を行ったのち、2012年1月より株式会社CROOZに入社。
CROOZ入社後モバイルFlashの開発を始める。入社後1年で8タイトルの開発に関わる。
現在はflash開発チームディレクター、アヴァロンの騎士プロジェクトに所属。

ソーシャルゲーム業界におけるflash開発の現状
SAPにおけるflashの開発手法、様々なOSへの対応方法やモバイル独自の考慮点等を、事例を元に説明いたします。
※アヴァロンの騎士を題材に説明いたします。

◆原 一成氏
株式会社サイバーエージェント所属のWeb Developer。Amebaブログ、Amebaピグを担当後現在はスマートフォン向けAmebaを担当している。主な開発言語はHTML, CSS, JavaScript(node.js含む)。まゆまろコンテストでは新しい機能を使った、どんな面白いアプリが現れるか楽しみです！
Ameba

スマートフォン時代のWeb制作術 Vol.2
HTML5やCSS3が策定され、ブラウザには様々な機能が実装されました。特にスマートフォンの環境では今すぐ使える機能が数多くあります。では実際にはどのように使ったら有効的なのか、面白いのか、実例をふまえながらお伝えします。


◆菊本 久寿氏
2006年フラクタリストに入社。エンジニアとしてモバイルのSIやメディア事業、広告事業に携わり、2011年にngi group技術開発部部長に就任。RTBを含めたいくつかの広告サービスの立ち上げに従事した。
2012年12月にモーションビートを退職し、現在はフリーランスのエンジニアとなり、レンタルCTOという名義でエンジニアリングからスタートアップの支援を行なっている。
・サービス紹介urlなど
「シャチクのミカタ」
「告白の行方」

ひとりでできるもん。〜個人サービスの作り方〜
クラウドやソーシャルといったものが当たり前になった現在では個人でサービスを作って運営することが可能になりました。
そこで開発、運営、プロモーションなど全てを一人でなるべく簡単に効果的に行う方法を体験談からお話したいと思います。
