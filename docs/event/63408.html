新規ネットサービス起業ネタ考案ブレスト会 Panpx

■Panpxについて

Panpxはインターネットサービスの新規起業ネタをみんなで頭絞って考えるマジメで実践的でオープンなブレスト会です。
世界で戦えるベンチャー起業家を輩出して日本を元気にしようと毎週可能性を模索しています。
起業に興味のある個人がネタを見つけ、チームを集め、起業し、資金調達するまでをフォローします。

起業に興味が有るけど何だかよく分からない人 起業に参加してみたいエンジニア 一緒に起業してくれる仲間が欲しい人 起業のテーマが欲しい人 ブレストで頭を柔らかくしたい人

など、起業に興味があれば誰でも参加OKです。
また、panpxでは

・投資用資金1500万円
・ベトナムにプロトタイプ開発ライン

を整備しています。いい起業ネタが考案されたらチームを組み、資金を注入し、ベトナムでプロトタイプ開発ができてしまいます。単にアイディアだけで終わらずに実際に起業して開発開発まで行う勉強会として実践力を上げていきます。

参加費は無料です。
すでにPanpxのブレスト会から2社スタートアップが登場しました。
また、3,4社目も立ち上がりそうな雰囲気です。
ネタ作りを頑張って新たな未来を切り開きましょう。
