内容

初心者向けのAndroidアプリ開発方法をハンズオン形式で行います。
ハンズオン形式とは、実際にPCを使いながら行う勉強会の形式です。

  1. Hello Worldを出してみる
  2. カスタムビューを作ってみる
  3. JUnitで単体テストを行う
  4. MonkeyTestを行う
