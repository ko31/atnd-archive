Windows 8 Developer Camp in かごしま

～Windows 8 アプリ開発に興味のある鹿児島県民のための技術セミナー～

概要

Windows 8 アプリケーションの開発について学べる勉強会です。
日本マイクロソフトのエバンジェリスト　大西彰先生をお迎えします。
鹿児島でなかなか聞くことのできない、最新OSの情報を入手するまたとないチャンスです。
Windows 8 アプリ開発を始めたい方、ぜひご参加ください。
参加申し込みは、右上のログインボタンから！

対象者

Windows 8 アプリケーション開発に興味のある一般の方（学生・社会人問いません）
※何かしらのプログラミング言語(C++,VB,C#,HTML,JavaScript等)の経験があると良いです

参加費

３００円 ／１名（※会場使用料として）

講師

大西　彰 先生（日本マイクロソフト　エバンジェリスト）

スケジュール

時間 内容
10:00 – 11:30 Windows 8 アプリ開発のポイント part 1
開発ツール、デザイン、実装上の注意点
11:30 – 12:30 休憩
12:30 – 14:30 Windows 8 アプリ開発のポイント part 2
メディア機能、HTML/JSでの開発、ストア申請前の注意点
14:30 – 14:45 休憩
14:45 – 16:30 Windows 8 アプリ開発ワークショップ
テンプレートを利用したアプリ開発など
16:30 – 16:45 Ｑ＆Ａ
16:45 クロージング、撤収
18:00 – 懇親会 （希望者のみ、 別途参加登録 をお願いします）
