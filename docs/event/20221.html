震災復興＋designとは

issue＋designの主催、神戸市の共催のもと、全国各地で、同様のテーマで同じ志を持った仲間と対話し、
東北の未来を考えるワークショップが開催されています。
被災地の現状を学び、議論し、被災地に求められるアイデアを発想する場です。
皆さんの1つひとつのアイデアの積み重ねが被災地の未来をつくります。

issue＋designとは

博報堂イノベーションラボ / hakuhodo + design / studio-L / FELISSIMOにより運営されている、
『社会の課題に、市民の創造力を』を謳ったプロジェクトです。

デザインには問題の本質を捉え、そこに調和と秩序をもたらす力がある。
美と共感で人の心に訴え、社会に幸せなムーブメントを起こす力がある。
社会課題を市民の創造力で解決し、
安心して市民が暮らせる社会を実現すること。
神戸から日本に、世界に、アイデアを示していくこと。
それがこのプロジェクトの狙いです。

主役となるのはデザインの専門家、ではありません。
一人一人の生活者です。社会の課題をリアルな不安として
感じている生活者だからこそ、捉えられる「問題の本質」がある。
提案できる「実質的なデザイン」がある。
市民を救えるのは市民の力なのだ。
私たちは、いまそう信じています。
（http://issueplusdesign.jp/より一部抜粋）

ワークショップ内容・日程

このワークショップは上記の震災復興＋designのコンペティションに応募する方をバックアップします。
http://issueplusdesign.jp/project/06_fukkou/oubo

震災復興に関心のある方や、デザイン思考や未来に関心のある方との交流を目的としているため、
参加費も材料費とシェアおやつのみとなっています。
4人1グループで協力し、全4回でプロダクトのプロトタイプか
プロジェクトのストーリーを完成させ、参加者全員で共有します。
コンペティションへの応募は各チームで行っていただくのですが、
あなたの頭の中に眠っているアイデアを絞り出すところから始めて、
異なるバックグラウンドを持つ人たちで協力しつつ、新しいアイデアを作り上げるところまでサポートします。
既にもやもやっとあるあなたの頭の中のアイデアを、頭の外にすっきり放り出してみませんか？
