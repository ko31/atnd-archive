description

2013年9月14日(土)に千葉のCKFでタクトレ会を催したいと思います。

今回のメインはタクトレ会ですが、折角フィールドを貸し切るので、タクトレとは別に空いてるスペースでエアガンのプリンキング（適当に的を撃ったりするやつ）を楽しめるようにしたいと思いますので、気軽にエアガンを撃ちたいだけの方の参加も募ります。（10名集まらないと、結構困る（切実）

■タクトレ内容（予定）
大きくわけて3ステージを組む予定です。
それにあわせて参加者も2〜3チームに分け、ローテーションでステージを練習して頂きます。
　1.基本的な射撃姿勢、立射、膝立ち撃ち、伏せ撃ち。
　2.バリケードを利用した射撃。
　3.動きながらの射撃。
基本的にトレーニングではセミオートで撃ってもらいます。
セミが効かないエアガンしか無い場合は仕方ありませんが、出来ればセミ撃てるのを持って来てください。
装備品については、サバゲに役立つ実戦的な練習会にしたいので、普段のサバゲと同じ装備を推奨しますが、季節的にも暑いですし、軽装でもかまいません。
ただしリロードの練習もするので、マガジンの数は最低2本ご用意ください。

■プリンキング内容
フィールドの2/3はタクトレで使用していますが、他の余った部分にスチールターゲットや空き缶等を並べて、持ち込んだエアガンで思う存分撃ちまくってください。　安全が守れていれば特に注意事項はありません。

■サバゲ初心者の為に。
とりあえず、撃つ目的で参加してみませんか？
横でトレーニングしているので興味が湧いたら参加してみてください。
トレーニングなんて難しい。　なんて思うかも知れませんが、ただただ弾をバラまくだけのサバゲから、一歩踏み込んだサバゲの楽しみ方が見えてくるかもしれません。

■時間の流れ

時間は10：00~17：00の予定です。
0940頃にはフィールドゲートが開き、駐車場に入れるようになります。
トレーニング開始は11：00前後を予定しております。
終了は17：00に完全撤収

■必要な物、参加費

タクトレ、プリンキング共通
・参加費4500円（予定）
・エアガン
・バイオBB弾（プラ、セミバイオ不可）
・ゴーグル（撃ち合いにはならないのでシューティンググラスでもOKです。）

タクトレの方は、さらに…
・長物エアガン　および　長物マガジン2本以上（スプリングマグ推奨）
・ピストル　および　マガジン
・普段のサバゲ装備推奨。軽装の場合も腰にマグポーチ使って弾倉を携行してください。

■お昼ご飯について

お昼ご飯は各自で用意して頂きますが、吉野家とガストのデリバリーが利用出来るようです。（前回使用したとき、便利でした。が、今も使えるかは不明）
近所にコンビにあります。

■交通手段について

基本的に現地集合現地解散ですので、自家用車の乗り合いは個々に打ち合わせをしてください。
今回はフィールドの送迎車はありません。
