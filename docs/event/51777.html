description

食には、人を癒す力があります。
毎日の生活で疲れた心と体を、食によって癒すことができます。

しかも、そのために特別な食材を買い求めたり、特殊な料理法を使ったりする必要はありません。
日常の食事を「正しく」とることで、無理なく心と体を癒すことができるのです。

ヒーリングフード・ジュニア・インストラクター養成講座は、
名から10名程度のアットホームな雰囲気のなかで
「日常食を通じて心と体を癒す」
ためのもっとも重要なポイントを具体的にお伝えする認定講座です。

＜たとえば、こんなことが学べます＞
・心と体を癒す食生活の8つのメソッド
・コンビニ総菜の選び方
・絶対避けたい添加物とそうでない添加物
・日本と世界の食文化に秘められた健康の知恵
