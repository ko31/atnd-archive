第6回Android勉強会in札幌

イベント ユーザー
event_id 8379
title 第6回Android勉強会in札幌
catch 今回はハンズオンです
description

北海道支部では、しばらく飲み会が続いていましたが、今回はマジメな勉強会です。
今回はハンズオン形式となりますので，参加される方はAndoid開発環境がセットアップされているノートパソコンを持参してください。
内容としては、初心者向けというよりは、中級者向けとなります。少なくともAndroidでのHello world程度の知識は必要となります。(なにかしらのAndroid本を一冊読んだ方、程度を目安としてください)
また、(必須ではありませんが)NDKを使用しますのでC/C++の経験があると理解しやすいと思います。
なお、今回は株式会社インフィニットループさんの協力により，参加費無料となっています。

日時: 2010/10/16(土曜) 13:00〜17:00
場所: 北海道札幌市中央区北2条東1丁目4-9 アーバンコート武蔵野
　株式会社インフィニットループ 会議室(部屋番号803)　玄関で803をコールしてください。
地図(ストリートビュー): http://maps.google.co.jp/?ie=UTF8&hq=&ll=43.064093,141.357522&spn=0.001499,0.003425&z=19&brcurrent=3,0×5f0b2976f37e6c4d:0×7cef839646d93f71,1&layer=c&cbll=43.064112,141.357677&panoid=1CZ-6LcQw3zUo_hGUrQZyg&cbp=12,341.16,,0,7.47
参加費: 無料
参加に必要なもの: Android開発環境が入っているノートパソコン1台。必要に応じて電源タップ、LANケーブルなど。
定員: 最大10人ぐらい(要相談)　いつも直前にキャンセルがあるので、キャンセル待ち狙いでもOK
参加方法: このATNDより申し込み。(定員が少ないため早めの申込を!)

スケジュール:
　13:00　開場

　　　3. 見栄えをソニエリのカスタムビューに変えてみよう。（ハッカソン）
　14:20〜15:20
　　　「NDK+OpenCVでできること」講師:まいむぞう
　　　AndroidへのOpenCVのポーティングはいくつかありますが、今回はGitHubにて
　　　公開されているものを使って、NDKとOpenCVへの理解を深めます。
　　　参加者はお使いのOSに対応したNDKをインストールしておいてください。
　　　http://developer.android.com/sdk/ndk/index.html
　　　また、こちらからOpenCV一式をダウンロードしておいてください。(Zip版でOK)
　　　http://github.com/billmccord/OpenCV-Android
　15:20〜17:00
　　　「フリータイム」

　17:00　撤収
　
　なお、今回は懇親会を予定していません。(飲みたい方はその場で募ってみてください)
