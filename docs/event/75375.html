description

たまにはDVDが観たいなって思いませんか？

社内で大人気だった、映画大好きサーバーサイドエンジニアまっさんによる
DVDを観ながら映画の裏話を聞けるというイベント。

裏話や小話が多いので私達の中では尊敬の意味も込めて「副音声」と呼んでいるこの裏話。

あまりにも面白いので自分たちだけで楽しむのはもったいない！これは他のエンジニアさんとも共有したい！
そんな思いから開催を企画しちゃいました！

そして今回まっさんの副音声付きで鑑賞会をおこなう作品は

2014年11月15日公開され記録的大ヒットを出した
「魔法少女まどか☆マギカ」の虚淵玄が脚本を、「機動戦士ガンダムOO」の水島精二が監督を務めたSFアニメ。

楽園追放　Expelled from Paradise です。

DVDを観る時には必需品であるこれらも用意させていただきます！

  • DVD鑑賞の必需品(社内調べ)
    1. ビールもしくはコーラ
    2. ポテチ コンソメ味(リクエスト受付中)
    3. ピザ
    4. その他お菓子いっぱい

まっさんについて

スパイラルセンスで働くサーバーサイドエンジニア
得意な言語はPHPだが、Unityでのフロント開発、イラスト制作までなんでもこなします。
豆知識が凄いのでDVDを観る際には副音声として社内から一目おかれている。
もちろんエンジニアとしてもすごい人。

スパイラルセンスについて

国立競技場の近くにオフィスを構えるゲーム開発会社。
主にcocos2d-xでの開発が得意。
イベントをやるならたくさんの人と楽しみたい！と考えているので
突発的に色々なイベントを開催する。たまに勉強会とかセミナーもやってます！
