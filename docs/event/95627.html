前回の情熱的な嵐からはや数か月…

人々はエモい音楽とエモいビールが楽しめる時間 ～エモ散らTime～ に飢えていた…

…なんてことはないと思いますが、今回もM3勝手にアフターイベントとして開催いたします！

今回の開催日は春M3の翌日・4/30(月･祝)
つまり前回は買い物帰りに寄れなかった人も、今回は安心して遊びに来られる!

さらに今回は関西や名古屋を中心に哀愁感溢れるエモ散らかしＤＪで人々を魅了するDJ、Snafkinさんをゲストにお迎えいたしました！
果たしてどんなプレイが見られるのか…！？

エモい音楽とエモいゲストビール
最高のエモ散らかしTimeをお過ごしください！

エモいゲストビール

今回も2種類のエモいゲストビールをご用意いたしました！

1種類目は、長野県は喜久水酒造さんより、Kikusui Cidre！
…前回と同じじゃないか？って？

　いいえ、違うんです。

前回提供いたしましたのはStandardですが、今回はSweetを提供いたします。
本来ならば前回のエモ散ら会ではこちらを提供する予定でしたが、あまりの人気に入手出来ず…
それが今回は手に入りましたので、提供出来ることとなりました！

Sweetの名の通りジューシーで甘く飲みやすいので、飲み過ぎにはご注意を…(笑)

喜久水酒造ホームページ

続いて2種類目ですが、湘南は茅ヶ崎にあります熊澤酒造さんより、湘南ビール アルト！

前回の試飲会でIPAに次いで人気だったビールです。
IPA同様に生で飲める機会はブリュワリー直営のレストランに行かないとなかなか難しいです…
それを今回もエモ散ら会クルーの「どうしても生で飲みたい…！」という欲望(?)の元、生で提供することとなりました！

味は…当日飲んで確かめてください！

[VOCAtraVERS / ボカロ・東方喫茶　２人でお入りください / 映像犯]

Aly & Fila Japan Tourへの出演、
vocaloid × trance イベント「VOCAtraVERS」のレギュラーを務めながら
後進の育成にも力を注ぐ

Prog.house / Trance / Upliftingを主体に
大箱から喫茶店、スナックまで場所を問わず
哀愁感溢れるエモ散らかしＤＪを得意とする。
今、日本で１番富士山に近いDJである。

プレミックス(α)はこちら

ニン[ 神籬 ]

※全力必死に顔隠してます( ；´Д｀)

velfarre Cyber Trance全盛期に端を発す、10年来のクラブ沼。
EDM好きらしき客がTranceをこき下ろすのを聞き「これはよくない」と勝手に奮起。タイムラインの流れに便乗し、今回の事件に至る。
六本木トランス、外タレから音ゲ、東方まで。東京近辺でゆるりと活動中。


ごれなす

2015年1月 mixcloudに投稿する形でDJ活動を開始。
2017年2月 東京の神籬で現場デビューを果たす。
琵琶湖県在住、交通の便を活かし各地のクラブで自分の好みの音を浴びることを趣味としている。
Trance,Progressive Houseを中心にDeepHouse,Tecnho,TechDanceなどBPM145以下の4つ打ちを得意とし、サブカルチャーとクラブミュージックの融合をテーマとしたプレイを行っている。

プレミックス(α)はこちら

zi-ma [ T.M.R. ]

民放2局という試される大地高知に生を受け、オタクとは縁の遠いの幼少期を過ごすも、
2007年東方Projectとの出会いを機に同人音楽に傾倒。
その溢れる情念とリビドーが具現化する形で家にDJ機材が生える。
2011年から1年ほどの地下DJ活動の後、転勤の拍子でうっかり名古屋の東方系パーティーT.M.Rに参画。
家系ミュージックを主体とした選曲でフロアを大いに盛り上げるべく日々吶喊（とっかん）中。


wakana [ circle-rw / 東方酔音庵]

幼少期からクラシック・J-pop・ハイパーテクノ・ユーロビートとその時流のジャンルに正しくハマる。
その後テレビから流れたDarude/Snadstormに大きく影響を受けトランスに傾倒する。
そして東方Projectと出会い、クラブとゲームを融合させて布教する生活が始まり今に至る。
自らトランス系楽曲を作るcircle-rwを立ち上げ、その活動は10年を迎えた。
これからも彼女（？）の活動から目が離せない。


ミサキナガサキ [ Extend!! ]

“Extend !! / EXphantasm” 主催にしてたまにコスプレしてDJとかやってる妖怪。
オールジャンルやるけど実はゼロ年代トランスが自分のクラブミュージックの原点だったり。
今はすっかり社畜ですが #2018年はトランスがくる の言葉を信じて、イベント当日までなんとか生き延びます。

VJ Akinashi [T.M.R. / AJTF ]

生まれも育ちも愛知県名古屋市。公用語は名古屋弁。
2012年よりVJ活動開始。名古屋を中心にカルチャー、サブカルチャー問わず色々なイベントに出演。
さまざまな映像をかき集め、機材の隅々まで使い、あらゆる手法を駆使し音に合わせた映像を展開して、いかにお客さんを楽しませる事が出来るか日々試行錯誤している。
また、映像制作技術を活かしjingleや素材を自作したりもしている。

過去に制作したjingleはこちら

Az [ 東方酔音庵 ]

お金にものをいわせ機材を買い集めるVJ
流れる曲や周囲の反応のネタを拾い、フロアを沸かせることに全力。
