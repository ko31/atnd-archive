歯科医院経営をしながら、大学で教鞭をとり、日本コンサルタント協会認定コンサルタントとして企業の相談にのり、毎年数冊のベストセラー本を出版している井上裕之氏。
経営に関する基本的なことから、モチベーションの高い組織作り、院内システムの構築方法、収益を伸ばす仕組みの作り方まで、円滑な歯科医院経営をするために必要なことを全て知ることができるセミナーです。

さらに地域密着ビジネスを知り尽くした妹尾榮聖氏。
この２人が組んで、歯科医院経営に革命を起こすための「経営論」「組織作り」「院内システム作り」「マーケティング」をお伝えする。
それが『歯科医院　経営革命セミナー』です。
