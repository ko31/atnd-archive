イベント内容

概要

2014年を最高の年始にするために、ちょっぴり新春 Firefox OS ハッカソンを開催します！

制限はひとつ。WebのAPIを用いてマッシュアップアプリにすること。
APIに制限は設けません。

会場に来れば席と電源とインターネットはあります。とりあえず書きましょう！

必要なもの

  • Firefoxの入ったPC
  • Firefox OS端末 (あれば)
    • (Mozillaさんで数台用意いただけるようです。)

タイムテーブル

時間 内容
9:30-10:00 開場、受付
10:00-10:30 オープニング
10:30-19:00 みんなで開発、ランチ (チームなどでご自由にお取りください)
19:00-20:00 成果物発表会、クロージング

懇親会

クロージング後21:00まで、会場をそのままに懇親会を予定しています。
当日、参加の有無をお聞きします。費用は1人1000円程度を予定しています。

注意事項

  • 会場にネットワークを用意しますが、接続台数に制限があるためなるべくご持参ください。
  • Firefox OSならびにJavaScript初心者の方は予習の上ご参加ください。
  • 参加できなくなった場合はATND上でのキャンセル処理をお願いいたします。
