description

11月2日に千葉北フィールドでサバゲ会を催します。（仲間内のゲームです。）
フィールドへのアクセスはココをクリック

■ゲーム時間

時間は13:00~22:00の予定です。
平日開催と言う事もあり、いつもの8時〜17時開催よりも車の移動が楽なのと、折角夜も使えるフィールドなので、暗がりでも遊んでみたいので、こんな時間にしました。

■参加表明

基本的にここで参加表明してください。
ただしまだ日付変更の可能性もあるので、11/2以外なら参加出来る…と言う方は、参加表明後にコメント欄でその旨をお書きください。
11/2に確定いたしました。

■ゲスト参加の場合

基本的に参加の方はココで表明していただきますが、ATNDを使用できない方なのどの参加の場合、同行する方がコメント欄にゲストの名前表記して参加表明してください。
リストを作る都合上、ハンドルネームでもOKですのでお名前を頂きたいです。

■必要な物、参加費

・参加費5000円
・軽食（ゲーム中、お腹が空く事もあると思いますので、各自で用意を。）
・エアガン
・BB弾（バイオのみ）
・ゴーグル（シューティンググラス不可）
・顔面を保護するマスク、四肢保護のニー、エルボーパッドや手袋があるとベター。

■レギュレーション

・セミオート・フルオート問わず全ての銃でパワーは0.98Ｊ以内
・使用BB弾はバイオのみ（プラスチック、セミバイオは禁止。）
・減速アダプタの使用は禁止
・フリーズコール禁止
・基本的にブラインドファイア禁止
・シューティンググラス使用禁止（フルフェイスゴーグル推奨）
・NV非推奨
・ライトは123Aが2セルまでの物。
・レーザーサイトはすべて使用禁止
・ゾンビ行為禁止
・フラッグへのダッシュアタック禁止（周囲制圧後にプッシュ。）]
と、基本的にいつも通りです。

■装備と夜戦用装備について

・主催の経験では、結構頭をぶつける構造になっていますので、キャップ程度でもいいので、頭部を守る物を被る事をお勧めします。
・AGITOよりも狭く、撃ち合うレンジも近くなりがちなので、1Jギリギリよりも優しさを持ったエアガン推奨。
・ハイパワーな方はセミオート等の自主的な優しさを見せて頂けるとみんな幸せ。
・CKFは夜でも照明がつくので、真っ暗にはなりません。
暫くは照明有りでゲームをしますが、最後の方は照明の量を減らしますので、相当足元が見えにくくなりますご注意ください。
・ライト等の装備をお薦めします。（レギュレーション以内）
・フルオートトレーサーとか賑やかしアイテム大歓迎！（w

■ゲーム内容

基本的にはフラッグ戦メインです。
通常のフラッグ戦のほか、ペットボトルフラッグ戦（ブザーの代わりに3本のPBを倒す事でフラッグゲットとする）を交互に行ないます。
さらに、この2パターンに加え、セミオート縛りや弾数制限を設けてのゲームになります。

かなりの回数をこなすと思いますので、ノマグオンリーやリアカン、アイアンサイトのみなど 個人的な縛りを設け、自分を追い込む様なプレイに挑戦する のも良いと思います。

詳細は後日発表のタイムシートにて。

■セーフティでのBGM■

当日は、セフティゾーンにスピーカー等を持ち込むので、CDやipodをBGMに出来ます。ゲーム前にテンション上げたい方は、お気に入りの曲を持って来てくださって結構です。
