2012/07/06(金)のICPC国内予選に向けた勉強会です。
内容は下記のとおりです。


LTの部 : 1時間(1枠10分程度)
ペアプロの部 : 2時間(当日集まったメンバーで2人,3人でグループになって練習)
交流の部 : 30分~1時間(気になる人に思いを告げるチャンスです！)
AtCoderの部 : 1時間半(時間が被ったと思ったけど、オフをやっちゃえばいいじゃない)

LTについては、「今年は乱択がくる！」とか「神デバッグ方法」のようなテーマを想定しています。
発表をしたい方は、コメントなりメールなりtwitterなりで、名前と発表タイトルを伝えてください。
対象はガチ勢onlyではないので、気軽なもので構いません。むしろ発表することで色々と教えてもらえるかも？
*発表中に関連した問題を記してあるとみんな嬉しいと思います。

ペアプロについては、数人でグループを組んで練習する形となります。
集まったグループでどういう問題に取り組むか決めていただいて構いません。
(LTの部であった問題や過去問等でよいと思います)


日時：6/30(土) 17:00-21:00
場所：チームラボ株式会社(http://www.team-lab.com/)
持ち物：ノートPC、名刺など
対象：ICPCに参加したい方、競技プログラミングしたい方
あると嬉しい：回線、電源タップなど(こちらでも用意します。)

なお、ノートPCはペアで1台使用するのでなくても構いませんが
誰も持っていないと困るので持参できる方は持って来るようお願いします。

今回は会場を提供していただいたので参加費はありません。
uwiさんおよびチームラボのみなさんに感謝！


21:00-22:30はAtCoderオフをします。
交流の部の20:00-21:00までを食事休憩として、21:00からAtCoder#005をやります。
もちろん、途中退場・途中参加構いません。

長丁場になりますが、こちらも是非ご参加ください。


LTの内容(参加表明はコメントか@mitsuchieまでお願いします) 残り2枠です！
