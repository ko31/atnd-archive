今回の勉強会の詳細が決まりました。
今回はちょっと趣向を変えて、PCを使わずに手書きで行きたいと思います。

まず数人ごとにチームに分かれてもらい、
予めこちらでデザインを用意したので、
それに対して自分だったらどういう設計をするか、というのを話しあってもらいます。

そして最終的にチーム単位でプレゼンしてもらい、みんなに発表してもらう、という形で進行したいと思います。
-————————————-
さてさて、前回から結構時間があいてしまったので
そろそろ第3回 CSS Holicを開催したいと思います。

ちょっとまだ方向性は確定していないんですが、
運営のほうのトピ でもちらほら話していますが、
「ID/classの命名方法」や、
それら決定、チェックのワークフローなど、
今まで以上に実務に直結する内容でやりたいと思っています。

まだ少し先なので、まとめてから改めて内容を記載したいと思いますが、
とりあえず日付を決めないと、ということで7/17（土）に予定しました。

内容についてはできるだけ早めに決めたいと思います。
よろしくお願いします。
