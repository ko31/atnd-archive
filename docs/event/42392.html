FUR EAST Web勉強会

description

概要

  • 『Webデザイン受発注のセオリー』 と 『IAシンキング』 の2冊をテキストに用います。
  • テキストはあった方が望ましいですが、後日に購入を検討頂いても構いません。
  • レクチャーを受けた後、テキストを題材にディスカッションと演習を行います。
  • 見学、新規の参加も歓迎します。
  • 参加費は無料です。

対象

  • Web制作に関わる人、Web屋になりたい人
  • プログラマ、デザイナ、ディレクター、職種は問いません
  • 印刷物メインのデザインに関わっていて、これからWebデザインへの移行を考えている方

使用テキストについて

『Webデザイン受発注のセオリー』 [ AA ]

フリーランスに限らず、個人の経験だけで学んでいくには限界があります。
こうした書籍を活用して皆でディスカッションし、意見を交換しましょう。

『IAシンキング』 [ AA ]

Webサイトを構築する上で必要なスキルは様々ですが、その場限りでない、より本質的なものの一つがIAに基づいた思考力だと思います。
演習問題も豊富なので、実際に取り組んでみましょう。

会場について

  • 13:00から入れます。
  • 飲食物は持ち込み自由です。
