『 Microsoft Office 365 セミナー！～多数の実績を持つ最前線のエンジニアが語るクラウド移行と運用に必要な要素～』
難易度：★★★☆☆

サーバエンジニアとしてご活躍中の方、また運用・保守業務から更なるキャリアアップを目指されている方を対象に、
『 Microsoft Office 365 セミナー！～多数の実績を持つ最前線のエンジニアが語るクラウド移行と運用に必要な
要素～』および会社説明会を、7月25日(水)に実施いたします。

本セミナーでは、レガシー Exchange などのオンプレメールシステムから Exchange Online へ移行する際の
ハマりポイントや、実際に活用できる移行テクニックを Microsoft MVP がご紹介いたします。

また、クラウド運用に欠かせない ID 管理分野において、オンプレ AD や Azure AD の効率的な運用管理手法を
実例に基づいてご紹介し、クラウド時代に求められる ID 管理の設計・実装について解説します。

ユーザー企業が求める要求仕様を熟知している当社視点と、クラウドとオンプレミスの ID 管理を知り尽くした
当社エンジニアが、最適なソリューションの一例をご紹介しますので、是非、ご参加ください。

＜対象となる方＞
・サーバ構築のご経験がある方
・サーバ運用業務・保守業務のご経験が1年以上ある方
・インフラエンジニアとしてキャリアアップしたい方
※毎回、サーバエンジニアとして活躍中の 20代半ばから 30代半ばの方々を中心に好評いただいております。

※定員人数を超えた場合には、日程調整及び早期に申込みサイトをクローズさせていただくこともございます。
お早めにお申し込みください。
