夏は駆け足で過ぎ去り、秋の気配が濃くなっていく今日この頃。秋空にひときわ映えるオレンジ色といえば、そう、Firefox OS の季節です。

Flame とともにひと夏を過ごされた人も、そうでなかった人も、夏の思い出を懐かしみつつ Firefox OS のコードを読むのも一興かと思います。Flame を機に Firefox OS に興味を持った人も、とにかく集まってワイワイやりましょう！

何をするの？

Firefox OS のソースコードを読んだり、語らったりする会です。構えなくても大丈夫。Firefox OS が好きな人、Firefox OS に興味を持った人達がゆるく集まって交流できる「ゆるふわ」な集まりが、このミートアップの持ち味です。

今回は、「コミュニケーションスペース」と称して、とりあえず何をしていいか分からない初心者の方や、コードを読んでいるだけで物足りない初心者() の方が、エキスパートと話せる、コミュニケーション中心のテーブルを設けます。

なお、恒例の無料懇親会は、○○えもんの四次元ポケットから出ていたひみつ道具「ポケットマネー」がなくなりましたので、有志の持ち寄りで何とかしたいと思います。

有志による軽食・お菓子などの持ち寄り歓迎します！！！（ソフトドリンクは用意します）

何が要るの？

Firefox OS への興味とノート PC だけで OK です。
ちょっとした調べ事なら FxOS コードリーディングで運営している OpenGrok が使えます。

http://reading.fxos.org/source/

※ GMO インターネット様よりコミュニティ支援として ConoHa VPS の提供を受けています。

「FxOSコードリーディング」って何？

Firefox OS のソースコードを読んでいる/さわっている人達のコミュニティです。
Firefox OS を深く知りたい人達がコードを読んだりビルドしたりして意見交換しあう場です。

読んでみたいけどどうしていいか分からない。
読んでみたけど分からないことを聞いてみたい。
読んでみたから誰かに話したい。

そんな人達をつなぐ場です。この機に一緒に Firefox OS を探求してみませんか？
