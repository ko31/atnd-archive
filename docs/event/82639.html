イベント内容

将来は海外で働きたい！
働きたいけど、実際どんな感じなんだろう？

そんな方の疑問を解消するため、フィリピン・セブでビジネスをしている起業家3人を集めてのパネルディスカッションを行います！
イベント当日は以下のテーマの中から時間の許す限り、面白おかしく話していきます！

  1. 登壇者のみなさん、どんな仕事してるんですか？
  2. 海外就職のメリット・デメリット
  3. フィリピンのエンジニア＆スキル事情
  4. 海外（フィリピン）の生活スタイル
  5. 海外で働き始めたら、英語は伸びるの？
  6. 海外（フィリピン）での就職に向いてる人の特徴
  7. 海外に来てビジネスをやる意義って？

懇親会もあるアットホームな会なので、お気軽にご参加ください！

こんな方にオススメ

  • 海外、フィリピンで働くことに興味のある人
  • 自由な働き方をしたいWebエンジニア、Webデザイナー、Webディレクター
  • オフショア開発に興味のある人
  • プログラミング教育ビジネスに興味のある人

登壇者プロフィール

山口竜矢（MonstarLab Philippines Cebu inc. / CEO）

東京のシステムインテグレーターにてプログラマー兼システムエンジニアとして7年勤務後、フィリピン留学を経験。
株式会社InfinitePointsのフィリピン法人立ち上げの発起人兼取締役として参画。現地責任者としてブリッジSE業、バックオフィス業、新規事業を担う。
自身の原体験からフィリピン人材の潜在力とラボ型オフシェア開発の可能性に気づき、FreeMight Philippines ,Incを設立。代表取締役に就任。
2016年9月をもってMonstar Lab, Inc.と資本提携を結び、同社のフィリピン拠点マネージャーとして参画。

横田猛夫（Kredo IT abroad / CEO）

2012年 国費にてマケレレ大学院（アフリカ）へ留学。
その後、2013年 セブ島留学を経験し、QQEnglishへとインターンとして入社。
のちに正社員となり、マネージャーへ昇格。2014年 同社でIT留学を立ち上げ、2016年8月Kredo IT abroadを開校。

堀口誠人（LIG Philippines inc. / President）

2012年LIG.incにジョイン。
これまでに上場企業での研修、ワークショップ、LTイベント、オンライン学習サイト『Schoo スクー』などで多数登壇し、主な著書に『最強のCSS設計（SBクリエイティブ）』がある。
好きなものは猫、嫌いなものはバナナの皮、ポケモンで最初に選ぶのはゼニガメ。
