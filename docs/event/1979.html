description

【勉強会内容】
・チームラボが、今回新しく研究開発した情報推薦アルゴリズムのご紹介。

チームラボが新しく開発したレコメンデーションアルゴリズムに関しまして、弊社CTO田村がご紹介します。

・レコメンデーションエンジンの精度の向上について
現場でレコメンデーションエンジン精度向上のため研究開発を行っている弊社テクノロジスト　水谷 有也よりアルゴリズムについて話します。（予定）
