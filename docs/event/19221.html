ぐゎらん洞に行きましょう！

8月28日 惜しまれつつも閉店予定の唯一無二の和風スープカレー
ぐゎらん洞さんのスープカレーを食べに行こう！というだけの会です。

できれば宴会をしたかったのですが、難しいようですのでただ、ただスープカレーを食べに行きましょう。

ぐゎらん洞が大好きな方も、まだ食べたことがない初めての方もせっかくなので食べたくなったら行きましょう♪
