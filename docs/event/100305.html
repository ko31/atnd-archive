極真空手をやりながら臨床心理士として社会活動をする『玉川真里さん』と、
剣道六段の称号をもちながらIT社長であり地域教育活動をする『武藤晴彦さん』とのコラボセミナーです。

キーワードは「人を動かすには、心を動かせ」

異職種に見える2人が武道と地域貢献という共通点もあるところから見た仕事と生きるをバランス良く楽しむ方法を語り合います。

以下のような方にオススメです。

●仕事でパフォーマンス良く、成果を出したい方
●私生活で自分の時間を持ち、充実させたい方
●仕事とプライベートでの役割を、家族の支障なく円滑かつ同時実行できるようになりたい方
●リーダーシップ、コミュニケーション技術に興味のある方
●PTA等地域活動に取り組まれている方
●剣道や空手といった格闘技やスポーツに取り組まれている方
など

後半は座談会方式で皆様にもご参加頂きながら、ワークライフバランスや生きることについて熱く語り合えればと思います！

■主催者プロフィール

玉川真里（たまがわ　まり）

元自衛隊の臨床心理士、NPO法人ハートシーズ理事長

1973年岡山生まれ。アルコール依存症の父と、自殺未遂を
する母のもとに育つ。
自分の身を守る強さを求め、1991年に陸上自衛隊に入隊。
女性初の大砲部隊野外通信手として活躍する。
2008年、陸上自衛隊において現場初の臨床心理士として、最も自殺率の高い職業と言われる自衛隊の自殺予防対策を任される。

より多くの人の心を救済したいとの思いから自衛隊をやめ、資産全てを投入してNPO法人を設立。

「災害・事件・事故に対する心の危機管理分野」では、日本で数少ない有職者の一人として活躍し、自衛隊・海上保安大学校・福島県立医科大学など延べ100件以上の実績を持つ。

クライアントからの信頼も厚く、年間延べ2000件を超える相談があり、累計3万人以上の心を解放してきた。

TV出演、ラジオ番組は「玉川真里の心の話」放映中、webコラムや女性誌美stの掲載、日経新聞、Japan Timesなど記事掲載と取り上げられている今注目の臨床心理士。

複雑な生育歴やDV、自殺未遂、離婚、結婚詐欺などの実体験をもとにし、当事者の辛さを面白く楽しく説明し、どのようにすれば心が鍛えられ潰れないかの技術を具体的に提示している。

現在は、海上保安学校、消防学校、福島県立医科大学、近畿大学附属広島高等学校、中学校東広島校での非常勤相談員および講師、自治体主催の研修や企業・学校から依頼された講演会などメンタルヘルス、ハラスメント、人権、子育てなど講演多数。

武藤晴彦(むとう　はるひこ)

外資系製造業向けIT会社の日本法人社長、教育地域活動家

自営業の家に生まれ、社長である父親の頼もしい背中を見つつも、下請け業者の厳しい現状も体感。
10人兄弟の下から2番目で生まれた父は経済的な理由で高校進学叶わず。中学卒業後、東京の自動車整備工場で働き始めた。
この時、父は、自分の子どもの教育には金を惜しまないと強く思った。

厳格な父の教育方針のもと、偏差値60以上の中学校のみの受験を許可される。
習い事である剣道との両立を図り、早稲田中高等学校に合格および入学。

2000年頃、バブル崩壊などの影響で、父の自営業の収入が激減、その結果、事業がうまくいかなくなる。
2005年、父の末期がんによる余命宣告から長男として実家に戻り、息子達と共に地域との関係を構築するために地域活動を始め、2007年からの町会の子ども会会長を経て、2010年小学校PTA会長に。
2011年3月には東日本大震災を経験。
陸前高田市出身のおやじの会会員がきっかけで、おやじの会、PTA、学校共同で、お金や衣類のボランティアを募り、渡す活動も実施。
2012年には「働くと生きるのバランス」の題で市の地域活動団体主催の講演会で講師を務めた。

PTA会長や東日本大震災の経験などから、母子家庭、父子家庭、低収入家庭などの厳しい、現状/実態を知る。
その結果、教育 地域活動に目覚め、今後の人生は教育地域活動に時間を使おうと考え、2013年、脱サラして 「学力よりも人間力」をテーマに、オンライン学習塾「わらふく塾」を起業。
2017年には外資系IT企業の日本法人社長に就任。

お客様、協力者(自社社員、他社関係者)、自分が、心からの喜びとしてWin-Win-Win の関係になるように推進するをモットーに社長業を邁進中。

ボランティアで教育・地域活動も精力的に実施中であり、現在以下の役割を務める。

中学校おやじの会会長 (越谷市立西中学校)
自治会(町会) 広報･書記 (越谷市新川町2丁目)
地区主任児童委員 (越谷市出羽地区)
地区コミュニティー推進協議会 委員 (越谷市出羽地区)
地区音響･放送 責任者 (越谷市出羽地区)
出羽剣友会副会長 (越谷市、剣道六段)

地域・教育活動の経験により、様々なコミュニケーションスキルを学び、それらをビジネスに適用・活用。
複数タスク/役割を並行・同時に実行する方法や利点について、セミナー・イベントを通して伝えている。

修行歴40年である剣道は、六段の腕前。
