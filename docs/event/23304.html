description

ZCBは、WindowsマシンのデータをAmazon S3に低価格でバックアップできるソリューションです。月々4.95ドルと転送コスト(0.15ドル/GB)の料金で、お手軽にディザスタリカバリが実現可能で、Windowsのファイルのみならず、SQLサーバやExchangeなどのバックアップを取得することができます。 また、ZCBは、プロプライエタリなフォーマットを使用していないので、オープンなフォーマットでのバックアップを可能にします。毎月の料金は、Amazonを通じてクレジットカードでお支払いいただけます。

ZCBには、多くの便利な機能を実装し、非常に柔軟性に優れています。今回は、これまでに頂いたお客様からのご質問やを参考に、ZCBの主な機能とその利用例、そして弊社の料金設定を、わかりやすくご紹介したいと思います。
