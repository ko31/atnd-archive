IoT の普及や自動運転の開発が進むにつれて、エッジデバイスへの機械学習の導入に期待が高まっています。
本セミナーでは、学習済みニューラルネットワークの推論処理を、Arm 搭載デバイスに実装し最適化する方法をご紹介します。
また、エッジ推論をどのようにサポートしているか等の仕組みついて、Arm のプロセッサ IP とソフトウェア開発ツールの観点からお話しします。

今回、Arm 社からエンジニアを講師に招き開催します。Arm 社のテクノロジーを知ることができるチャンスをお見逃しなく！
セミナーでは Arm 搭載デバイスを用いたデモなども予定しております。
