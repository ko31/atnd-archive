description

クリーク･アンド･リバー社では、12月4日（月）、3DCG関係者やVFXに興味のある方、ＣＧ業界への就職を目指す学生を対象に『Houdini × VR×デザイン塾 ～3DCG、VFX、アニメーション、ゲーム制作の「Houdini」を使ったVRの制作現場～』を開催します。

今回のデザイン塾では映画、3D、VFXツールとして名高いツール「Houdini」をVRで使用する現場作業についてご説明します。

データの作り方の詳細や、実際にHTCのVRゴーグル「VIVE™」を使用し、データ作成後の実演を行います。
「Houdini」ならではのデータ作成を徹底解説します。
この機会に最高峰のツールを体感しましょう。

内容
・Houdini および SideFXに関して
・HoudiniでのVRデータ作成
・「VIVE™」デモ実装体験

Houdiniとは
最高峰の映画の実写系VFXから、ゲーム、遊技機などの映像表現で幅広く使われている3Dアニメーションツール
https://www.sidefx.com/ja/
