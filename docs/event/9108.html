元学校の先生であり、カードゲームの本も出している
草場純さんからカードゲームの面白さを教えてもらっている会です。

教え方が丁寧で分かりやすく、そして、教えてくれるゲームも
面白い。

ビールやチューハイの飲みながらでもOK。

とにかく楽しい！！

忙しく煩わしい仕事が終わったら、頭をリセットしませんか？

といわけでさっそく第４回目を開催します。

カードゲーム界のカリスマ草場さんと
カードゲームで盛り上がりましょう

　http://www42.tok2.com/home/q/access.html

　国道２４６を道なりにまっすぐに渡ってください。

　そのまままっすぐに進むと「ドトールコーヒー」があります。
　手前にビルのエントランスがあるのでの階まで上がってください。
