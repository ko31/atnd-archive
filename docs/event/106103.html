久しぶりのミートアップとなりますが、今回は、いろいろとご縁がありまして、株式会社トラストインベストメント様のプロの不動産投資の方々と関わる機会があり、不動産投資に関しての分析をどのようにグラフィカルモデルを活用して分析できるかについて、挑戦したいと思います。

また、不動産投資の中でも、購入する側のオーナー目線で見た場合の”騙されない為の分析のポイント”を紹介しながらお話しします。

tech blog : https://graphicalmodeling.com

github : https://github.com/GraphicalModelLab/GML_SaaS
