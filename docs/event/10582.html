数多くある就活イベントと何が違うのか


中大生がお送りする、１００％中大生の為のイベント

「合同説明会に行ったけど、いい事しか言わない。一方的に話を聞かされるばっかりで、つまらない。」

「俺らが欲しいのはいい事も悪い事もあるREALな情報！企業が出来ないなら、俺たちで作ってやる！」
このイベントは、そんな想いで作られました。

違う点①
イベントコンテンツは、企業の話を聞くときにでも
一方通行ではなく、みんなで楽しく考えながらきける企画になっています。
◇ＯＢ一人につき、5人で双方向のコミュニケーションが取れる仕組みになっています。

違う点②
こうした形式のイベントを開催しているのは、実は中央大だけで、他にはないイベントで、
中央大学に特化したイベントです。
◇OBに中央大学が社会からどのように見られているのかなど、絶対に他のイベントでは聞く事が出来ない内容です。

違う点③
様々な業種のＯＢと密に話せるイベントです。
人気大手のＯＢ・ＯＧを本当に様々集めたイベントはこのイベントだけ！

このイベントでは、中央大学の学生と社会人の先輩方が交流することを目的で、
イベントを通して、学生の就職活動の不安や悩みを少しでも解消したいというのが私たちの願いです。

コンセプトは

【自分の道を進むためにこれからすべきこと】

【社会で活躍する先輩たちから直接お話をすることで自分の進む道を夢ではなくRealにする事！】

そして、中央大学生と中央大卒の先輩によるイベントという強みを活かして、就職活動や社会に出てから

”自分が中央大学卒だと感じる瞬間”を実体験を通じて話してもらいます。

今回は、【100人限定】で行います！！

また、参加者、スタッフ、協力してくださる社会人の先輩を募集中です！

スタッフ参加ご希望の方は、twitterにDMください！！

タイムテーブル

15:00～：第一部 パネルディスカッション

１部はパネルディスカッション形式。社会人の先輩がパネラー６名で
　就活に関する振り返りやアドバイスを行う
　フラットフロアでシアター形式にて学生が聴講＆質疑応答。
　

16:00～：休憩

16:30～：第二部　 学内ＯＢＯＧ訪問

２部はテーブルを８脚ほどの椅子で囲んだ島を１0島ほど作成。
　１テーブルに社名を立て、先輩一人に対し学生5名位という超少人数。
　学生は15分ごとに入れ替えがあるが自由着席

17:30～：閉会

参加希望者は、右上の「参加登録」ボタンより、
お申し込みください。
