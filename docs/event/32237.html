[84関西] オオサカオクトーバフェスト 2012

昨年大阪初開催だったオクトーバフェストに乗り込んで、大変盛り上がりました。今年も同い年のみんなと美味しいビールを味わいながら、真夏の太陽の下で楽しくわいわい盛り上がりましょう！

出展ビールの予習はこちら！

ハチヨン関西って？

関西のハチヨン世代（1984年4月〜1985年3月生まれの男の子・女の子）が集まってできたコミュニティ、「ハチヨン関西」。

ウェブマガジン「84ism」を中心に関東で頻繁に集まっている同世代を見て、「関西でも集まろうぜ！」ということで、2010年7月に最初の集まりが開催されました。

それ以来、同い年ならではの楽しい気兼ねのない感じで、不定期に飲み会などイベントを開催しています。
