高等教育改革の諸前提

  1. 高校求人件数の激減
  2. 高校求人件数の激減はなぜ生じたのか
  3. 失業問題は若年層問題
  4. 高度教育の必要 ― 「それなり教育」からの脱皮

カリキュラムとは何か

  1. 3つのポリシーの中核としてのカリキュラム
  2. 大学カリキュラムの現状 ― 選択制の弊害
  3. なぜ、選択制カリキュラムは蔓延するのか
  4. 選択制カリキュラムの実際

必修制カリキュラムとは何か

  1. 必修制の科目数・単位数・期構成
  2. 必修制のシラバス構成― アメリカ型シラバス・日本型シラバス・カリキュラム型シラバス
  3. 必修制の教員構成
  4. 高等教育のグランドデザインと必修制カリキュラム

参加希望の方へ

本講演会は広島修道大学人文学部人間関係学科心理学専攻に与えられた「教育成果指標の開発支援事業」の予算により実施されるものです。対象者は本学心理学専攻のスタッフですが、学内・学外を問わず、参加したい方はどなたでも無料で参加できます。懇親会もしたいと思いますので、懇親会に参加されたい方はその旨コメントに書いてください。

会場にたどり着く方法

アストラムラインをお使いの方は、「広域公園前」駅で下車し、改札を出て「広島修道大学」の看板が出ている方 (改札を通ってすぐ左) で降りてください。階段を下りて、左に曲がって上って行くと正門にたどり着きます。正門を通り過ぎてずっと上がってください。途中ロータリーが2つあります (最初のロータリーにはバス停があります。バスでお越しの方はこの1つ目のロータリーのバス停で降りることになります)。2つ目のロータリーの辺りに電話ボックスと郵便ポストがあります。郵便ポストを超えて、横断歩道のあるあたりを左折してまっすぐ行くと右方向に上る階段があるので、そこを上ってください。右側に生協書籍部のあるところで、左に入ります (ここらへんに喫煙所があります。ここは若干分かりにくいです)。まっすぐ行くと2103教室が見つかります。
