description

経済活動全体が停滞する状況下で、企業は売上を伸ばすことが難しくなり、コスト削減などの業務改善に取り組む企業が増えています。
業務改善と聞いてどのようなイメージをお持ちでしょうか？
「後ろ向きな仕事？」「他にもっとすべきことがある？」「現場の仕事！」。過去に失敗経験があり業務改善には良いイメージを持っていない人もいるのではないでしょうか？
業務改善がうまくいかない原因として “現場の無関心” “未完全な支援体制” “経営課題の未解決” などが挙げられます。せっかくスタートした現場の業務改善が、経営課題がネックとなり頓挫することも少なくありません。
業務改善の最初のステップは業務の見える化です。そして次の業務分析を正しく行うことで、業務改善だけでなく経営課題の解決糸口まで見えるようになります。業務改善には「自分の会社、自分の職場は自分で良くしていく」という現場の自発性が重要です。見える化では自らの業務を自ら書かなければ、他人事となり良い結果は得られません。そのためには、“現場が簡単に使えるツールと正しい書き方”が必要です。
これまでの枠組みのカイゼンから脱却し、経営指標にプラス影響を与える「効果」と現場の「効率」を同時に実現し、かつ業務改善に関わるメンバーが主体的に取組むためには何をどうすれば良いのか？ 理論と事例、可視化ツールを交えながら、わかりやすくお伝えします。

■対象
　・企業経営者、役員、管理職の方

　・業務改善に経営者層を巻き込みたいと考えている方
　・経営課題・業務改善で悩む本社・管理部門の管理職以上の方

　　『業務の見える化で進める業務改善と経営改革へ』

業務の見える化は重要ですが、それ以上に「見える化によって何ができるか」をお伝えします。特に、業務改善が具体的にうまく動く仕組みづくりだけでなく、現場への動機づけ、経営層の役割、経営改革との両立などをお話しします。
何社もの業務改善、経営コンサルティングに入った経験とノウハウより、綺麗な理屈よりも臨場感ある実例を交えながらお伝えいたします。

　　『業務プロセス可視化を効率的に実現するために』

業務プロセス可視化を効率的に実現するためのポイントを実例を交えてご紹介。国内企業2,000社に選ばれたBPMツール「iGrafx アイグラフィックス」を使用し、どれだけカンタンにフローを作成・編集できるかをデモンストレーションで見ていただけるセッションです。
