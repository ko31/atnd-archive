九州でMicrosoftのITPro系コミュニティ「Win.tech.q」を立ち上げました！

「Win.tech.q （ウインテックキュー、通称ウインク）」は、 九州を中心にWindows Serverなど、MicrosoftのITPro系のスキルアップを目指す目的で立ち上げたコミュニティ・勉強会です。
九州近辺のみなさん、一緒に勉強しませんか？

3月20日（祝日・火曜日）に第0回勉強会・準備会を開催します。

まずは、九州でのこの分野についての現状の共有や、みなさんのお考えなどお聞きしたいと思っています。興味がある方、ぜひご参加ください。
当日は日本マイクロソフトのエバンジェリスト・高添修さんお越しいただいて、マイクロソフトの今後の動きなどについてもお話していただく予定です。
また、同じく日本マイクロソフトのMVP事務局・北端智さんにもお越しいただいて、「MVPアワードプログラム」について紹介していただきます。

Win.tech.q について

「Win.tech.q （ウインテックキュー、通称ウインク）」は、 九州を中心にWindows Serverなど、MicrosoftのITPro系のスキルアップを目指すコミュニティです。
「Win.tech.q」の「Win」は「Windows」、「Tech」は「テクノロジー」、「Q」は「九州」や「(技術を)吸収」や「Quest(探求)」といった意味があります。
