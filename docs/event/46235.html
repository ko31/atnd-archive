Titanium もぐもぐ会の髭怪人こと @k0sukey です。

Titanium もくもく会の後はいつもさみしく解散してしまうので勝手に作りました。もくもく会場の近場でおいしいごはんが食べられるところへ、会が終わったら有志で行きましょうということです。

今回は年末ですから忘年会といきましょう。会場は特に決めていないので、ぶらぶらしながら居酒屋を探しましょう。ぼくはお酒が飲めないので、フライドポテトがあるお店ならどこでも OK です。オススメのお店があったら是非教えて下さい。

もくもく会の終わる 21：30 から会場を出て、みんなでぞろぞろともぐもぐ会場まで移動します。
