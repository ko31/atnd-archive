title 「 愛媛Android勉強会 part.１」
catch 　　　　　～Android端末に何かをつないで動かしてみる～
description

８月２２日開催された「日本Androidの会四国支部第二回勉強会inえひめ」参加の方々からのご要望にお答えし、実際にＰＣ上でアプリを作りandroid端末を動かしてみる勉強会を企画してみました。

アジェンダ：

１３：３０～１４：３０　【開発環境の構築】（未インストールの方対象）

※android SDK+Eclipsのインストールを一緒に行います。

（インストールしたけどうまくいかなかった方、質問がある方も対象）

インストール済みの方はこちらから

１４：３０～１６：００　【アプリ開発】（インストール済みの方対象）

※ＰＣ上でサンプルアプリをハックしandroid端末にインストールします。

１６：００～１６：３０　【外部機器接続】

※android端末に手作りの模型の車を接続しコントロールします。

１６：３０～１７：００　【意見交換会】

※勉強会に対する要望などを話し合い今後の方向性を考えます。

参加条件：

◇Javaプログラム経験者

◇開発ＰＣをご持参ください。

◇android SDK+Eclipsのインストールは事前に行ってきてください。

（未インストールの方は【開発環境の構築】からご参加ください）

◇駐車場がありませんので公共交通機関をご利用ください。

◇ネット環境はありません。

◇出来ればandroid端末もご持参ください。　余ってないです　＞＜）
