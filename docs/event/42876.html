開催概要

Visual C# Express 2010で開発できるオープンソースハードウェア『Netduino』『FEZ Hydra』等を使って、Java屋さんやC#屋さんでもできるんだ！ということで、電子工作をして遊んでみようと思います。

■募集期限

飛び入りOKです

■開催日時

2013/08/31(土) 17:30 ～ 21:30(最大)

■集合場所、時間

2013/08/31(土) 17:25
地下鉄新栄駅 1番出口を出たところに集合、遅刻された場合は070-5641-2248まで電話をお願いします。
なお、いつもの会場になりますので、場所をご存知の方は直接会場にお越しいただいてもOKです。

■会場

公共施設を借りて実施
(最寄駅：地下鉄新栄駅 徒歩3分程度)

■募集定員

10名程度まで

■参加費

実費相当（会場施設代等）として、500円をお願いいたします。
電子工作に必要な部品は各自、手配をお願いします。

なお、ネットワーク環境は開催者から提供いたします。

■内容

.NET Gadgeteerを試してみましょう。
FEZ Hydra、Ceruberusを2台ずつお持ちしますので、お持ちでない方にもお貸しできます。

初学者がいらっしゃる場合は、個別にプチハンズオンも入れる予定です。

■注意事項

会場利用の都合上、名簿作成が必要になります。
当日、参加時に氏名・連絡先をお伺いいたしますので、ご協力をお願いいたします。

■必要な事前準備

以下の電子工作に必要な部品をご手配のうえ、当日ご持参ください。

しばらく.NET Gadgeteerで遊んでいく予定ですので、続けて遊びたい方は、以下の電子工作に必要な部品をご手配のうえ、当日ご持参ください。
2,3個であれば貸し出しは可能です。

  • FEZ Hydra Basic Kit あるいは FEZ Cerberus Basic Kit

あとでカメラ、液晶と接続して遊んでみたい方には、Hydraの方をおススメします。

以下の開発環境を準備のうえ、当日ご持参ください。

  • Windows XP, Vista, or 7 (32 or 64 bit)
  • Visual C# 2010 Express
  • Microsoft .NET Micro Framework 4.2 QFE2 SDK
  • ”GHI Software Package v4.2”：http://www.ghielectronics.com/support/dotnet-micro-framework
    • ダウンロードにはGHI社へのユーザー登録（無償）が必要です。
