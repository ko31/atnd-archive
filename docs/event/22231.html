今回はなんと外国人が多く住んでいる２つのゲストハウスとGaitomoのコラボ国際交流パーティーを開催致します。

会場は渋谷と恵比寿のちょうど真ん中「グータンヌーボ」や「中居正広の金曜日のスマたちへ」等の撮影で使用されてたお洒落なダイニングバー。

クリスマスまでに素敵な出会いを見つける大チャンスです！ゲストハウスにお住まいの外国人が多く来ますので、たくさんの異文化交流を楽しんで頂けると思います！ また、いつもよりアジア圏の方が多いと思われます！韓国人、中国人が好きな方に大変おすすめです！

また、今回のワンコインブースには「カラーボトル占い」と「似顔絵ブース」も登場します！この機会に是非お楽しみください。

外国人は皆日本語が話せますので、英語に自信が無くても安心してご参加頂けます。ヨーロッパからアジア圏まで幅広い国の外国人たちが参加されます。 たくさんの新しい出会いをこの機会に楽しんでください。

今回はより良いパーティーを実現するために、定員制となります。Gaitomo経由でのご予約は日本人に限り定員100名様で打ち切らせて頂きますので、ご予約はお早めに！
当日参加も受け付けますが、入場は予約者優先、入場制限がかかる場合もございますので、ご了承ください。
