概要

最高のLinuxディストロGentooと最強のOSセキュリティ対策GRSecurityをインストールするハンズオンをして勉強します。
仮想マシンでゼロからGentoo＋GRSecurityを構築します（してみます）。

Gentooとは？

GentooはLinuxのディストロの一つでプログラムをインストールする時にソースコードからコンパイルされます。最もカスタマイズできるディストロで最もパワーフルですが、その分設定が難しくなります。Hardened Gentooは様々なセキュリティ対策が導入されているGentooです。

GRSecurityとは？

GRSecurityはSELinux、TOMOYO、AppArmor等のLSM(Linux Security Modules)の中でも最もセキュリティの高いカーネルパッチだと考えられています。

参加者

LinuxやOSセキュリティをより理解したいサーバ管理者・Linuxユーザのための勉強会です。内容が若干難しくて早いペースで進む予定なので、基本的なリナックスコマンド・管理方法、ファイルシステム・パーティションの概念の知識、仮想マシンの経験が必要ですが、セキュリティやカーネル設定の知識は要りません。

期待できること

Linuxカーネルのマニュアル設定・コンパイル、Hardened Gentoo・GRSecurityの基本的なインストールの体験

期待できないこと

Linuxカーネルの設定・Gentoo・GRSecurityのマスターになること

必要な物

Gentooではプログラムを全てソースコードからコンパイルするので、できるだけハイスペックのマシンと十分なHDD空き容量（４０GB＋？）が必要です。パソコンが遅過ぎると時間内に構築が終わらない可能性があります。（終わらない場合は宿題として持ち帰っても結構です）
Virtualbox若しくはVMWareを事前にインストールして下さい。物理的なハードウェアで挑戦したい方も大歓迎ですが、うまくインストールできるかどうか保証できません。https://www.kali.org/からkaliのisoファイルをダウンロードして、可能であればlive cdで起動できるように仮想マシンを作っておいて下さい。
