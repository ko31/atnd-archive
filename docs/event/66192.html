description

　マッピング・パーティとは、街歩きをしながらオープンストリートマップのようなフリーの地図に、新たな情報を付け加えるイベントです。
今回は、愛媛で初の取り組みで、愛媛大学と松山大学近辺の自動販売機の設定場所を調べて、その位置情報をオープンストリートマップに付け加えます。



【テーマ】
生活を便利にするため多機能型自動販売機の設置場所を調べよう！
（くじ付き、電子マネー対応、災害対応、Wi-Fi、おしゃべり機能付き等）

【調査項目】販売物の種類、メーカー、緯度経度、設置場所の住所など



参加費／無料

主　催／ Code for DOGO

集　合／愛媛大学・総合情報メディアセンター・メディアホール

スケジュール／
・13:00
　オープンストリートマップ（OSM）とは何？
　どのようにマッピングパーティを行うのか？

・13:30
　グループに分かれて街歩き開始。
　とにかく自動販売機を見つけたらデータをゲット！

・15:30-17:00
　パソコンを使い調査したデータをアップロードし、
　地図に書き込みます。最後に、新しく更新された
　地図を見て確認してみて終了です。



本イベントは、平成27年度 情報通信月間の参加行事のひとつです。
