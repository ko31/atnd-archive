description

　オープンCAE（OpenFOAM，Salome-meca，FreeCAD，ParaViewなど）に関心を持つ技術者，研究者，学生などが集まり，気軽に情報交換・相談する場所を作ることを目的に，勉強会を開催します。
　すでに使用経験のある方はもちろん，興味はあるけれど使い方が分らないという方も大歓迎です。お気軽にご参加ください。
