◇講義内容

『AWS』は、世界中の企業が利用する、信頼性の高いパブリッククラウドです。
本セミナーでは、AWSでのシステムを構築するために、仮想サーバEC2や、ストレージS3、ネットワークVPCといった主要サービスを紹介し、ハンズオンを通して概要を理解します。
さらに、ユーザ管理などのセキュリティ機能であるIAMや、データベースサーバであるRDSなども紹介します。

◇カリキュラム

　・AWS とは
　・AWS 各種基盤サービスの紹介（ EC2、VPC、S3、EBS ）
　・AWS のセキュリティ（ IAM ）
　・AWS のデータベース（ RDS ）
　・AWS 管理ツール （ Elastic Load Balancing、Trusted Advisor ）
