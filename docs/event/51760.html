若者憲法集会

イベント ユーザー
event_id 51760
title 若者憲法集会
catch 声をあげよう私たちの命と自由のために
description

　海外での武力行使を可能にする集団的自衛権の行使容認、知る権利と表現の自由を奪う秘密保護法、「公益」「公の秩序」の名で国民を縛ろうとしている自民党改憲案――。勝手に憲法解釈を変え、私たちを戦争に動員しようとしている安倍政権に若者が怒りの声をつきつける。日本国憲法の価値を若者の言葉で発信する集会がついにＯｐｅｎ！

◆１１：００～１２：３０　分科会　【会場】渋谷区、目黒区、千代田区周辺各施設

１．秘密と弾圧だらけの社会でいいの！？
　　ビラまき弾圧事件や冤罪の実態、秘密保護法の危険性を知り、　「表現の自由」の大切さを考えます。
　　【会場】　大岡山東住区センター（目黒区碑文谷3-15-5、東急東横線「都立大学駅」より徒歩15分）

２．アジアの若者と考える平和主義と安全保障
　　憲法の平和主義はアジア地域の安全保障構築に役立つのか？中国、韓国の若者とともに討論します。
　　【会場】　心障害者センター「あいアイ館」（目黒区八雲1-1-8、東急東横線「都立大学駅」より徒歩7分）

３．憲法でつくろう「健康で文化的な生活」
　　憲法が保養している生存権、労働者の権利について学び、憲法が暮らしに生かされる大切さを考えます。
　　【会場】　渋谷勤労福祉会館（渋谷区神南1-19-8、ＪＲ線「渋谷駅」より徒歩7分）

４．どうしたらできる？女性が生きやすい社会
　　ベビーシッター事件や待機児童の問題、仕事や生活での女性差別についての解決の方向を考えます。
　　【ゲスト】弁護士・長尾詩子（東京南部法律事務所）
　　【会場】　八雲住区センター（目黒区八雲1-10-5、東急東横線「都立大学駅」より徒歩7分）

５．歴史の真実を知りたい、伝えたい――教育と自由を考える
　　教育への政治介入の危険性があるのか、次の世代に何を伝えるのか、学びや教育のあり方を考えます。
　　【ゲスト】渡辺雅之さん（立教大学講師、『いじめ・レイシズムを乗り越える「道徳」教育』著者）
　　【会場】　渋谷区勤労福祉会館（渋谷区神南1-19-8、ＪＲ線「渋谷駅」より徒歩7分）

６．靖国神社フィールドワーク
　　靖国神社をガイドつきフィールドワーク。単なる追悼施設ではないことを体験学習します。
　　【会場】　靖国神社大鳥居前（千代田区九段北3-1-1、地下鉄「九段下」より徒歩5分）

７．大学生企画 「紛争解決のプロと話す集団的自衛権」
　　東京外国語大学教授・伊勢崎賢治さんとのお話で、集団的自衛権と紛争解決の道を学びます。
　　【ゲスト】伊勢崎賢治さん（東京外国語大学教授）
　　【会場】　めぐろパーシモンホール（目黒区八雲1-1-1、東急東横線「都立大学駅」より徒歩7分）

８．高校生企画 落語で憲法を楽しく学び交流しよう
　　憲法9条って何だろう？憲法の解釈を変えるねらいは？弁護士による落語で憲法を楽しく学び交流します。
　　【ゲスト】八法亭みややっこさん（八王子合同法律事務所・飯田美弥子弁護士）
　　【会場】　中根一丁目会議室（目黒区中根1-10-22、東急東横線「都立大学駅」より徒歩5分）

◆１４：００～１６：００　全体会　【会場】めぐろパーシモンホール（東急東横線「都立大学駅より徒歩７分」）
　高遠菜穂子さん（イラク支援ボランティア）、黒澤いつきさん（明日の自由を守る若手弁護士の会）をゲストに、いまこそ力を発揮する日本国憲法の価値、安倍政権が進めている「戦争する国づくり」の危険性について、熱いトークを繰り広げます。
