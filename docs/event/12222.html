手続き型言語のプログラム経験者を対象にした Erlang の基礎勉強会の第三回です。
毎回の事ですが、少人数で行います。(キャンセル率を考慮し 15 名に増やしました)

会場は、ジェミナイ・モバイル・テクノロジーズ株式会社様の会議室をお借りします。
※お飲み物の持ち込みは可能ですが、お食事はご遠慮ください。

今回は、簡易的な Key Value Store を作成しながら、Erlang でミドルウェアを作成する方法を学ぶ予定です。
実際に皆さんにコードを記述して頂く時間を確保いたしますので Erlang VM がインストールされたノート PC のご持参を推奨いたします。

第一回と第二回の内容は、下記をご参照ください。
https://github.com/cooldaemon/BasicStudyForErlang

Twitter のハッシュタグは、#TEWB3 でお願いいたします。
