ご好評のMacに続いてiPhoneについてもイベント開催！

学生、教職員の方は無料で招待します

iPhone/iPadの使い方を学べるデモンストレーション形式の講習会です。
講師がプロジェクター画面を使って丁寧に説明します

街中で見かけるiPhone/iPadに興味はあるけど使い慣れたケータイやAndroidが動いているスマホやタブレットから乗り換えてみたいとは思いませんか？

いざiPhone/iPadを買ったはいいけどWebやSNS、ゲームくらいにしか使ってないので有効活用してみたくはありませんか？

設定方法が分からなかったことはありませんか？

説明書がないので何か分からないことがあるたびにネット検索していませんか？

iPhoneの動きが重くなったり空き容量が一杯になってしまって困ってはいませんか？

新しいiPhoneに買い換えたいけど、今までのデータはどうやって移行するのか分かりますか？

ケータイやAndroidからもiPhone/iPadへデータを移行したいとは思いませんか？

そんな方々を対象にデモンストレーションを中心にiPhone/iPadの魅力をお伝えいたします。

iPhone/iPadを使っていて以下の項目に１つでも当てはまる方はぜひご参加ください。

きっと今まで以上にiPhone/iPadを活用できるようになります。

・そもそもタップやスワイプなどの基本操作方法がわからない

・買ったばかりで設定方法が分からない

・マナーモードがない？

・着信音をCDの音楽やダウンロードしたオリジナルの音にしたい

・撮った写真を周りの人と共有したい

・買ったばかりのときよりも動作が重くなってしまった

・容量がいっぱいになって減らすにはどうしたらいいかわからない

・Siriを使ってみたい

・Facebook、LINEやTwitterなどのSNSアプリを活用したい

・スケジュール管理をiPhone/iPadでしてみたい

・iPhone/iPadの大事なデータのバックアップを取得したい

iOS10やiPhone7などの最新情報についてもお話しします。

iPhone/iPad初心者の方でも安心してご参加ください。講習中や後に講師への質問や相談もできますし、交流タイムに参加者同士で悩みを共有することで一人では気付かない発見があるかもしれません。知っておくと便利な知識もお伝えします。既に使っている方でも意外と知らない機能もあるかもしれません。
