【ゆう活セミナー】HTML5が創り出す新たな世界

開催日時

2016年1月28日（木）18時～（60分程度）

場所

品川ラーニングセンター 12階ラウンジ
住所: 東京都港区港南2-15-2 品川インターシティB棟12階

講演概要

「HTML5が創り出す新たな世界」をテーマにビジネス及び人財の観点から解説をいただきます。

対象：クライアント系ソフト開発者、Webサイト設計者、マルチデバイス開発者、最新技術動向・HTML5にご興味のある方

  • セミナープログラム
    1. 基幹系システムにおけるHTML5の重要性
    2. WebにおけるHTML5の重要性
    3. Apple社の強みとHTML5
    4. HTML5が創り出す新たな世界

講師紹介

特定非営利活動法人　エルピーアイジャパン　理事長　成井 弦様
