ついに大阪でもやります！Sublime Text 2 勉強会

選挙寸前の金曜日のこの日、テキストエディタの第三極 「Sublime Text 2」について語り明かす朝まで生 … もとい、勉強会です。

とにかく話題の Sublime Text 2 ですが、気になってはいるけどまだ使ってない、使っては見たものの良さがわからない、そういう方たちのために今回は Sublime Text 2 入門的な内容を企画させていただきました。

東京、岡山、福岡でもすでに勉強会が開かれて、今後ますます Sublime Text 2 は人気を伸ばしていくことでしょう！

ご興味ある方はエンジニア、コーダー、デザイナー、その他問わずだれでもご参加くださいませー
