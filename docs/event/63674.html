description

野外好きのみなさま、いよいよ春ですね！今年も長かった冬を死んだ魚のような目でなんとかやりすごし、お外遊びのシーズンにたどり着きました。フジロックも何故か既に第6弾発表まで進んでいて、気分だけは夏になってる方々も多いことでしょう。というわけで冬眠後1発目の野外大宴会に、カドヤ花見なんていかがでしょうか？ノッてるかね！（雄三）

12月京都での大忘年会、2月東京でのカドの市と来まして、4月はやはりお花見ですね。ありえない規模とテンションで開催されるカドヤ花見は、河原に立ち飲み屋台や野外スナックが出現するとのこと！畳の座敷ステージ（グリーンステージ）には弾き語りや落語からラッパーまで登場します。設置されたコタツ群（コールマン製）でぬくぬく観ましょう。その他、焚き火、鍋、餅つきなどなど、紅白幕や立ち並ぶ灯籠などのデコまでするそうで（キャンドルなんちゃらプロデュース）、これはもはやフェスと言えるのでは。オールナイトでの開催で、テント持ってきたら張れそう〜〜とあいかわらず謎な企画ですが、またしてもひどい酒呑みが集まって盛り上がること間違いないでしょう…。

場所は京都の桂川・松尾橋。阪急松尾大社駅すぐなので、大阪方面からも電車で行きやすい場所となっております。今回は年末のようにフジロックな人だけの集まりではないので、どなたさまもお気軽に。24時間開催ですが、勿論昼間だけ参加も可能です。野外で定員もありませんので、事前の申し込みもいりません。当日の気分でふらっと飲みにおいでくださいませ！
桜を見るには一週遅いかもですが関係ありません！飲んだくれましょう！

■持ち物など
お酒やつまみなど持ち込み自由です。屋台がいくつも出て、ビールや日本酒や焼酎、おいしい焼き鳥やおでんやパエリアやなんだかんだも買えるとのことなので手ぶらでもOKです！
あと、レジャーシートや椅子など使えそうなものあるといいかもです。この季節、夕方はまだ肌寒いので寒さ対策もしっかりお忘れなく！
