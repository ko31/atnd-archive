2016年5月5日（木）に開催されるゲムマ2016春で発売する『ピクテル』の予約受付となります。 （ピクテルLight funnyの予約はこちら / ピクテルLight HORRORの予約はこちら）

  • その他要望などあれば、参加コメント欄に記入ください。
