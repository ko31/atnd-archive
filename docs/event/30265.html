実践! Android/Javaで作るペイントアプリケーション開発！

■セミナー内容

まずは、Android「ペイントアプリケーション」の作成から完成までの一連の手順を通して、開発の基本と楽しさを体感していただきます！！

※Javaの基礎をおさえている方であれば、スムーズに開発可能です。
※セミナー後に復習ができるよう、開発手順のドキュメントをお渡し致します。

  • 開発フローについて：
    ・プロジェクトの作成
    ・ペイントアプリケーションの土台作成(描画)
    ・ペイントアプリケーションのオプション作成（undo、クリア、保存）
    ・ペイントアプリケーションのさらなるオプション作成[時間がある限り]
  • アプリケーション作成で学ぶこと：
    ・ペイントアプリケーションのアルゴリズム
    ・レイアウト、メニューファイルの作成方法
    ・Strings.xmlの使用方法
    ・ファイルの外部メディア保存からギャラリーの反映方法
    ・Eclipseの便利なショートカット
    ・Eclipseデバッグ・LogCatの使用方法
    ・実機端末上の実行方法
    ・apkファイルの作成方法

■対象者

  • Javaの基本構文・Eclipseの基礎を理解している方
  • Android始めたばかりの初心者の方
  • Android開発者でアプリ開発を行いたい方

■費用

8,000円

■用意頂くもの

開発環境が整ったPC端末(Windows/Mac)

下記全て行い環境を構築してお持ち下さい

①JDKのインストール
②Eclipseのインストール
③AndroidSDKのインストール
④EclipseプラグインとしてAndroidSDKインストール、設定
☆重要：Eclipseに取り込むAndroidSDKのバージョンは2.3.3でお願いします。
⑤プロジェクトの動作確認
プロジェクトを作成しエミュレータ(AVD)起動を行い、プロジェクトのデフォルトの「Helloworld」が表示されることまで確認をお願い致します。

各ダウンロードサイトは以下となります。
①JDKのダウンロード
JDKの「Download」よりダウンロード下さい
http://www.oracle.com/technetwork/java/javase/downloads/index.html
②Eclipseのダウンロード
「Eclipse Classic3.7.2」よりダウンロード下さい
http://www.eclipse.org/downloads/
③AndroidSDKのダウンロード
「Download the SDK for Windows」よりダウンロード下さい
http://developer.android.com/sdk/index.html

インストール手順については、「Android 環境　構築」などのキーワードで検索をして頂けると、各サイトで手順については紹介されていますのでご確認ください。
