・いつかは起業して、社会に役立つ事業をしていきたい！
・アイデアを事業にしたいが、何から手をつけたらよいかわからない！
・個人事業主（フリーランス）として独立するための稼ぐ力を手に入れたい！

もし、こんな想いをお持ちなら、社会起業大学の「ビジネスマネジメントクラス（応用クラス）無料体験授業」で体験してみませんか。

本講座は、社会起業大学ビジネスマネジメントクラス（応用クラス）の必須授業「経営戦略論（全12回）」の担当講師、横田洋一先生の授業を体験できる貴重な機会です！
実現可能で持続性があるビジネスプランは具体的であり、シンプルで、何をすべきかが明確になっているものです。
担任の横田先生自身も、今リアルに新たなビジネスを創っている最中でもあります。今戦っている真剣勝負の横田先生だからこそ、授業も臨場感があり一人ひとりに真剣です。
（横田先生のクラスからグランプリファイナリストも出ております！）

当日は横田先生が考えているビジネスプランも聞くことが出来ます。
講義とワークショップを取り入れた授業の一部を体験できる内容を予定しておりますので、入学を検討されている方、過去に体験授業を受講された方も、ぜひお気軽にご参加ください。

【日時 】 2016年3月12日（土） 15:00-17:00

【場所】社会起業大学 (東京都千代田区二番町2番平田ビル1階)

【参加費】無料
