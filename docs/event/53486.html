イベント概要

2014/7/12にオープンハードカンファレンスが行われます。
http://oshwc.project2108.com/event/oshwc2014tokyosummer/

このイベントの懇親会では、新たにオープンハードウェアとして開発した珪藻土石窯のデモンストレーションを兼ねて、無限ピザパーティが行われます。

今回は、珪藻土石窯の試作2号機の試運転を兼ねて、予行演習としてのパーティーを行います。

日時

2014年7月9日　夕方～夜

場所

渋谷区代々木2-36-6 ギークハウス新宿　裏庭
同一住所で複数建物がありますが、

・代々木よりの角
・レンガっぽい壁
・一階が駐車場
・ポストにギークハウスのシールを貼ってる

あたりをヒントにお越しください。

最寄り駅は小田急の南新宿。
新宿線の駅（都庁寄り出口）からなら5分弱。
山手線、新宿と代々木双方から歩いて7,8分です。

参加費、持ってくるもの

参加費はありませんが飲み物、トッピングは持ち寄りです。カンパ大歓迎！

ルール

・あくまで新型試作機の調整／試運転の人体実験です。
・失敗しても恨みっこなし、過剰な期待は禁物。
・ベース部分、生地とソースは主催者が容易するので、チーズを含むトッピングは持ち寄りです。
・雨が降ったら適当に中止です。
・ドリンクは、ハウス内の冷蔵庫から買うまたは、持ち込みです
・泥酔禁止。
・騒音禁止。
・試作機調整は夕方頃からはじめます。夜に入ったころに本格的に焼きはじめ、20時頃解散の予定。

※ご近所の迷惑にならないように声のトーンを落としてね

珪藻土石窯とは

断熱性能、耐熱性能に優れた素材である珪藻土ブロックを用いることで、本来部屋を占領する大きさのピザ石窯をコンパクトに。また優れた熱効率の特性のために100V電源環境でも500℃まで加熱できる性能を達成しました。
珪藻土は自然素材であり、お酒の濾過などに使われるように人体に無害な特性を持ちます。また加工も簡単であり、廃棄時にも崩せば土に戻る、小容量熱源を高効率で利用できる、など環境負荷が著しく低い素材として更なる利用シーンが期待できます。
