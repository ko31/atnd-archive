★講演内容

☆★☆★☆★他媒体を含め合計200名超のご応募を頂いております！★☆★☆★☆

普段エンジニアとして「AWS」についてぼんやりとした概念で捉えている方は
まだまだ多いのではないでしょうか。

自分のキャリアにAWSの経験を取り入れたい。
自社サービスにAWSを導入したい。
個人的にAWS環境を構築してみたい。

だが、具体的にはどうすればいいのだろう。

そんな漠然とした思いを持つ皆様に対して、
今回はAmazon Web Services様にご登壇いただき、
より具体的なAWSの世界についてお話していただきます。

世の中におけるAWSプロダクトの現状は何か。
実際に使われているプロダクトのシェア、
技術的な解説を通してエンジニアとして勉強しておいた方がよいことは何か。
どの程度の規模感のサービスで、どのようなマシンセットが使用されているか。
具体的事例を通して、導入前と導入後のパフォーマンス変化を可能な限りご紹介。

今回のイベントを通じて、
「AWS」というぼんやりとして言葉を使うことはやめて、
「EC2」「S3」「Cloudwatch」等、具体的なプロダクト名で会話できる方が少しでも多くなれば幸いです。
