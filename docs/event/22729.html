今回のハンズオンのテーマは

Lightboxdroid: Picasaウェブアルバム対応。

現在公開されている0.1.1のソースを元に、
ギャラリーから直にPicasaウェブアルバムを共有できるようにします。

必要な物: Eclipse、Android SDK (r14〜)。
扱う分野: antプロジェクトのインポート、AsyncTask、AndroidHttpClientの基礎。

アプリ本体
Lightboxdroid 0.1.2(android market)
http://bit.ly/tu3jhV
