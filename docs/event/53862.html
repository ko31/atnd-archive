title 【CCIEを目指す方必見！】
catch 意外と知られていないCCIEを取得するために知っておくべき4つのポイント
description

【概要】

CCIEの取得を考えている方のために、ラボ試験を少ない回数で受かるための勉強法をお教え致します。取得までの道のりは想像をはるかに超えるものではありますが、無駄な時間とお金を使わなくていいように、4つのポイントをNWを熟知した現役CCIEホルダーである落合氏よりお話頂きます。

『CCIE R&S取得を目指している方』『CCIEに興味を持っている方』『受験しても合格できないと悩んでいる方』等を対象に、ラボ試験の対策から実際にかかる勉強時間や費用、どのようにすれば合格出来るのかまで、これまでに得たノウハウ・勉強方法を現役のCCIEホルダーが生の声で語ります。

一方的に講義するスタイルではなく、みなさんの疑問に全てお答えしますので、是非この機会にご参加ください。

【スケジュール】

13:00～13:30　開場、受付開始　

13:30～13:50　CCIEとは　:　市場の資格価値、難易度、試験概要からCCIEを認識頂きます。

13:50～14:20　CCIE筆記／ラボ試験対策　:　使用した教材、勉強方法、勉強時間／環境、費用について具体的にご紹介します。

14:20～14:50　CCIE ラボ受験記　:　合格までの道のりと重要なポイントを熱く語ります。

14:50～15:05　CCIEを取得する意味　:　CCIEを取得したことにより感じたこと、見えたことをお伝えします。

15:05～15:30　質疑応答
