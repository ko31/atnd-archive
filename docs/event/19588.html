■アウトドア社会人サークルFEAD 町田飲み会イベント■

東京町田にある おしゃれなバーでの飲み会パーティーです！

カクテル・ビール・ワインetc…
フリードリンクでバーテンさんが作るお酒と料理が美味しいお店です。

【時間】19:00～（予定）
【費用】3,000～5,000円（男女差あり）
食事つき・飲み放題のスタイル(ｏ’∀’ｏ)ﾉ

毎回多くの方にご参加いただく、人数限定の飲み放題パーティーイベントです！
ワイワイ飲んでおしゃべりして、みんなで楽しいひとときを過ごしましょう☆

社会人サークルFEAD（フィード）は東京・神奈川・千葉・埼玉、関東近郊を中心にアウトドアイベントや飲み会パーティーを
休日だけでなく平日にも沢山企画開催している非営利活動のアウトドア社会人ｻｰｸﾙです。

毎回初めて参加される方や１人参加の方が多数いらっしゃいます。
あっという間にうちとけられる明るい雰囲気のサークルです。
ｻｰｸﾙFEADのイベントにはたくさんの出会いと笑顔があります。
社会人サークルでの婚活・恋活、友達作りはオススメですよ〜！
是非お気軽にご参加くださいね☆
