そんな学生エンジニア大注目！
サイバーエージェントグループ株式会社ハシゴが、有名企業でCTOなどを務めている方のお話を聞ける学生向けイベントを開催します。

実際にエンジニアとして社会で活躍している人から、皆様が今聞きたいであろうことについて実際の経験を元にパネルディスカッションを行います。

また、パネルディスカッションの後には、誰でも質問することができる「Q&Aタイム」も設けております。お話を聞いてみて気になったことや、普段から気にしていることなどに対して、登壇者に回答してもらいましょう！

▼登壇紹介

・岸田 崇志様　株式会社LITALICO 取締役CTO
- 経歴-
2006年3月博士（情報工学）取得。大手ネットワークインテグレータを経て、2009年5月グリー株式会社に入社。エンジニア兼事業責任者を経て日本スタジオ統括部長、開発本部副本部長を歴任。2013年10月同社執行役員に就任。その後、新規タイトル開発やクリエイター育成に従事。2015年11月株式会社LITALICOに入社、執行役員CTOに就任。

・桂大介様　株式会社リブセンス 取締役
- 経歴-
“史上最年少の東証一部上場”としても話題になった株式会社リブセンスの創業メンバー。
中学時代にプログラミングの楽しさにのめりこみ、高校時代には個人事業主として独立。
大学時代に友人とともにリブセンスを創業し、現在まで取締役として技術面と経営面のどちらも担当されています。
