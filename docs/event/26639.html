description

ソーシャル時代の『働き方』塾では、GTDの勉強会を定期的に開催しています。GTD？？　ちょっと、とっつきにくいですが、要はクラウドを利用した情報整理術についてです。

なんか小難しそうですが、ITバリバリの話ではまったくありません。基本的に入ってくる情報をどのように処理するかという整理術の話です。

対処すべきは、
・仕事で使うメール
・口頭でやってくる指示
・その他洪水の様に押し寄せる情報
をどのように処理するかの方法論を知ることによって、

感じられる恩恵としては、
・なんだか忙しい状態から脱してスッキリした状態になる
・しなければいけないことが明確なので集中力が増す
・仕事への満足度が上がる
が挙げられます。

実際の方法論のベースとなるのは、デビッド・アレン氏が提唱している『Getting Things Done』というもので、2000年あたりに紹介された方法論で、もうすでに10年以上が経過していますが色褪せることなく、クラウドコンピュータの登場により、ますます当時ではできなかった方法で、情報処理が可能になって便利になっています。

対象としては、
・仕事でパソコンはよく使う
・何が忙しいのか分からない
・仕事の満足度があがらない
・スマートフォン買ったはいいけどゲームだけって言うのも
っていう方々に響く内容にです。

使用するツールは、
・Evernote
・GTDツール（OmniFocus、Nozbe、etc）
です。

講座の基本構成として、基礎編と実践編の２部構成をとっています。前回の基礎編で体系論を学んだ方々を対象に、今回の実践編では、EvernoteやOmniFocusといったツールの使い方をご紹介いたします。
