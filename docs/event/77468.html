ゲームスタジオ合同説明会2016春＠東京

今年も全国のゲームスタジオ6社による
ゲームクリエイターと話せる合同企業説明会 を開催いたします。

【注目】

  • クリエーター兼社長が自ら会社をプレゼンする本気舞台です！
  • グラフィックデザイン、ゲームプランニング、ゲームプログラムなどゲーム製作に第一線で活躍するプロのクリエイターの話を直接聞ける！
  • スマートフォン、家庭用ゲーム機、オンライン、アミューズメント施設向けゲーム、インディーズ、
    グラフィックデザインや映像が得意な企業まで。
    起業間もないゲーム関連企業が大集合！

皆さまのご参加をお待ちしております！
