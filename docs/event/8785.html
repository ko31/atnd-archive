■会費：300円(※会場費を人数で割っています、参加人数によって下がります)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。


■現在予定している内容：

　●セッション１(35分)
　『これからの「技術」の話をしよう――いまを生き延びるための実装』高橋さん

　＜内容＞
　ここ数年でホットキーワードになっている技術を取り上げ紹介します。
　プログラム寄りになると思いますが、
　なるべくプログラマ以外の方にもわかってもらえるよう、
　技術が利用されているサービスの紹介等もする予定です。

　* Google App Engine
　* Key Value Store
　* Protocol Buffer
　* MessagePack
　* Thrift
　* 位置情報サービス
　* 拡張現実サービス
　* SNS アプリ
　* Amazon EC2
　* Twitter

　＜高橋さんの紹介＞
　「大規模データ処理で新しいユーザー体験を」という目標をもって、

　* 新規サービス構築
　* プロトタイプ開発

　を行っています。

　http://d.hatena.ne.jp/h0x10
　http://twitter.com/jp_h0x10
　http://h0x10.sakura.ne.jp

　最近は Twitter 向けの画像検索サービスを作ったり、
　MessagePack-RPC の コミッターをしたりしています。


　●セッション２(40分)
　『FlashMediaServerでリモート共有コンテンツの開発』カンドウ柳谷さん

　＜内容＞
・FMS(FlashMediaServer)でなにができるのか
・作った共有コンテンツをみんなで触ってみる
　当日無線LAN対応PC(win,mac)を10台接続できます。
　多くの人と繋げれたら面白いので、PC持参できる方はおねがいします。
・実績
　・チャット
　・共有黒板
　・共有カメラ
　・共有キャラクター
・鬼ごっこサイトをローカルで復刻

＜柳谷さんの紹介＞
　Flashでは、キャンペーンゲームコンテンツ、ブログパーツ、Flash Lite1.1、2.1、
　ブログ等のデータをビジュアル化、　twitterのビジュアル化するコンテンツ。
　その他、大規模コミュニティー、学校のウェブサイトのデザイン、イラスト等の
　仕事を受託しています。

　カンドウコーポレーション
　http://www.can-do.co.jp/


　●セッション３(40分)
　『FlashCS5「Packager for iPhone」でつくるiPhoneアプリ開発』トライポット藤田さん

　９月初旬、アップルの規約が変更され、使用不可となっていた
　「Packeger for iPhone」が開発環境として利用出来るようになりました。
　今回は、Flashを使って、どのようにして作ったらいいか、出来ること出来ないことや、
　どんなアプリに向いているかなどを解説します。

　iPhone（iOS）の専用開発言語である、Objective-Cを覚えるのは大変です。
　「Packager for iPhone」なら、使い慣れているFlashのAS3で開発出来るようになります。
　web系のお仕事をされている方は、可能性が広がるのではないかと思います。

　また、Android2.2からは、FlashやAIRが採用され、同じソース（若干は変更が必要かと思いますが）から、
　iPhoneとAmdroidの両方に対応したアプリをリリースすることも可能になりました。
　両環境をサポートしたい方も注目です。（今回は、Androidについては解説しません。）

　＜主な内容＞
　・Developer登録
　・開発方法（AIRの設定）
　・ネイティブアプリとの違い（どんなアプリ開発に向いているか）
　・使える機能
　・AIRの書き出しからIPhoneアプリへの変換
　・証明書の発行
　・AppStoreへの登録

　＜トライポット藤田さんの紹介＞
　広島で、iPhone・iPadアプリ、デジタルサイネージ、ホームページ、Flash制作、システム開発とかやってます。
　デジタルサイネージアワード２０１０でインテル賞いただきました。
　Augment8メンバー
　WebTouchMeeting
　CSS Nite in HIROSHIMA実行委員

　http://tripot.net/
