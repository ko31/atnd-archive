description

―6秒に1人が、いま、世界で人身売買されています。

なぜ、人身売買は起こるのか、そして、人身売買をなくすために、今、私たちに何ができるのかを、スライドシアター、トークセッション、そしてシンガー・山根麻以によるコンサートを通して考えます。

たいせつな人と、でんきを消して、スローな夜を過ごす「100万人のキャンドルナイト」のシーズンにちなみ、当日は会場にキャンドルを灯します。そして、人身売買の火を消して、人身売買と対称にある自由と愛のキャンドルに火をともしていきます。

■プログラム

14：00～ スライドシアター「天の浮舟～森を失った国の少女と森を消費した国の少年の出会い」

東南アジアの日本人学校に通う少年は、ある日、人身売買の被害から逃がれてきた少女に出会う。ひょんなことから、気球に乗って少年と少女、そして謎のピエロが旅に出ることに…。

「天の浮舟」HP：http://rainbow.gr.jp/ukifune/　　　　

14：50～ トークセッション「たいせつな森と水と人をまもる～人身売買をなくすために」

タイ北部で、住民による持続可能な農村開発をサポートするNGO、エイズ、DV、子どもの人身売買に取り組む企業からの報告から、人身売買をなくすために、わたしたちができることを考えます。

ゲスト：木村 茂（NPO法人Link・森と水と人をつなぐ会 代表）×藤田紀久子（ザ・ボディショップ コミュニケーション部部長）

コーディネーター：百瀬圭吾（てのひら～人身売買に立ち向かう会）

Link・森と水と人をつなぐ会：http://www.geocities.jp/link_chiangmai_forest/
ザ・ボディショップ：http://www.the-body-shop.co.jp/values/stop_traffic/stop_trafficking_01.html

15：40～ 山根 麻以 ミニコンサート「ともそう、自由と愛のキャンドルを」

自らの音楽を「目覚めの音楽」と呼び、人間の本質を謳った唄を作り続ける山根麻以さんが、キャンドルナイトにふさわしい選りすぐりの楽曲を演奏。聴く人の心をやさしくつつみこむ歌声です。
山根麻以HP：http://www.futsuunouta.net/
