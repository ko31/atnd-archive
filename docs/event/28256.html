description

２０１１年３月。。。

勝間和代さんの私塾「勝間塾」が開校という案内をtwitterで知りました。

面白そうなので申し込み、３月３０日の初回オリエンテーションの日を心待ちにしていた矢先。。。。

２０１１年３月１１日１４時４６分。。。。
未曽有の大震災。。。。

あれから１年。

世の中の価値観が「つながり」や「コミュニティ」へとシフトしていき、一人一人の力を発揮して結集していくことが求められる時代へと急速に変化しているような感覚があります。

そんな中。。。

勝間塾に通ってたくさんの出会いとたくさんの学びをしてきたけれど、 いまいち自分の成長に実感が持てない、 もっと自信を持ちたい、 ２年目を迎えるにあたって新たな自分の可能性を見出したい。。。。

そんな思いをもっているあなた！

お待たせいたしました！
『勝間塾収穫祭』のお知らせです！
１年の成長をみんなで分かち合い、今までの成長を収穫したうえで、未来へとつなげていく、、そんな思いで企画しました。

今までの自分を振り返りながら、ほかの人と対話をしつつ未来を語りあいませんか？
これからの未来のためにほんのちょっとでも自分が行動していくことは何か、リラックスした雰囲気の中で考えていけるよう、ワールドカフェという手法を使うことにしました。

最後にはチームごとにプレゼンを行い、勝間和代さん、上念さんに直接聞いてもらいます。その場でフィードバックももらえるため、普段の月例会とは一味違った「濃い」場になること間違いなしです！

勝間塾歴の長短は不問です。

今までに勝間塾で学んだことのある方ならどなたでもご参加いただけます＾＾

初対面同士でも気軽にお話をいただけるよう、心地よい場を用意しますので、今まで勝間塾生との交流があまりなく、お友達がまだいなくて不安だ、という方もこれをきっかけにぜひ参加いただければ嬉しいです。

そしてそして今回イチオシのテーマは 「家族応援企画」！

お子様、ご主人（奥様）も誘っていただけます！
普段とは少し趣向を変えた休日の過ごし方に家族の絆が深まる事間違いなしです＾＾

普段、ご家族を気にして休日はなかなか外に出てこれない方、ぜひご家族をお誘い合わせの上、一緒にまるごとご参加いただければと思います。

小学生以上のお子様はワークに参加できます。
小学生未満のお子様をお連れになる場合でも、会場に保育士さんを呼んでいますので、ワークの最中はお子様をお預けになり、安心してご参加ください。

お子様の前でお母様お父様も童心に帰り、お子様の天才性にびっくりする。

自分が今までやってきたことを気軽に振り返り、笑顔で次の一歩を踏み出す。

そんなエネルギーあふれる場になることを夢見て・・ １年の締めくくり、収穫祭で新しい自分に会いましょう！
