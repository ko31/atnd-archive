概要

Haskell 界の大御所 Simon Peyton-Jones さんが、来日します。ご好意で、お話しを聞く機会を設けることができました。日本側からも2つ発表をする予定です。東京近郊の Haskell 好きは、全員集合しましょう！

プログラム

  • 18:00- 開場

    • Haskell本の販売、サインもねだれるかも:)
  • 18:30-18:40 主催者 和田英一先生からの挨拶
  • 18:40-19:20 Simon Peyton-Jones さん

    • 「Haskell and Software Transaction Memory」

      • スライド
    • 以下のどれかで予習してきましょう！

      • ビューティフルコード の24章「美しきかな、並列」
      • Real World Haskell の28章「ソフトウェア・トランザクション・メモリー」
      • スライド
  • 19:20-19:40 山本和彦

    • 「Experience on implementing a Web server in Haskell」
    • 「Haskell で Web サーバーを実装してみました」

      • スライド
  • 19:40-20:00 山下伸夫 さん

    • 「Yet Another Purely Fuctional Expression for Interaction」
    • 「もうひとつのIOの表現」

      • スライド
  • 20:00-20:01 O’Reily からのお知らせ
