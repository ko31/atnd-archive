Web 音楽プレイヤーの「君のラジオ」、写真で一言ボケる「ボケて」など数々の人気WebサービスをつくってきたWebエンジニア和田裕介氏(通称：ゆーすけべー)が、遂にその長年のノウハウをまとめた本『Webサービスのつくり方』(技術評論社)を出版しました。
http://gihyo.jp/book/2012/978-4-7741-5407-7

今回、その出版を記念して、先日Yahoo! Japanと提携したことでも注目の「暮らしのレシピサイトnanapi」の創業 者古川健介氏(通称：けんすう)を招き、書泉ブックタワー（JR・東京メトロ秋葉原駅）にてトークイベントを開催します。

今回のトークイベントでは、エンジニアゆーすけーべー・経営者けんすうの二人が、来場者からWebサービスに関する質問に"基本全部"答えます。
共に1981年生まれ、アプリ全盛期のこの時代にWebをつくり続ける二人に聞きたいことを持って、是非ご来場ください!

※このイベントのtwitterハッシュタグは、#web_essay です。

イベント開始は19:00からになります。開始10分前迄にはご来場いただけますと幸いです。なお、参加される場合は、『Webサービスのつくり方』(2,289円：税込)を書泉ブックタワーでご購入いただくことになります。その時間も考慮してご来場お願いします。
