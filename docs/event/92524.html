『LDAP Manager 実装例紹介セミナー！運用目線の Office 365 連携システム定番解説』
難易度：★★☆☆☆

サーバエンジニアとしてご活躍中の方、また運用・保守業務から更なるキャリアアップを目指されている方を対象に、
『LDAP Manager 実装例紹介！～運用目線の Office 365 連携システム定番解説～』、および会社説明会を11月25日(土)に実施いたします。

本セミナーでは、よくある Office 365 との連携に関する実装をご紹介し、オンプレミスとのシームレスな連携に必要な ID 管理の設計について解説します。
ユーザー企業が求める要求仕様を熟知している当社視点と、クラウドとオンプレミスの ID 管理を知り尽くした当社エンジニアが、最適なソリューションの一例をご紹介しますので、是非、ご参加ください。

＜対象となる方＞
　・サーバ構築のご経験がある方
　・サーバ運用業務・保守業務のご経験が1年以上ある方
　・インフラエンジニアとしてキャリアアップしたい方
　・当社への応募を今後検討したい方（※本セミナーは選考会ではありませんので、お気軽にご参加下さい）
　※毎回、サーバエンジニアとして活躍中の 20代半ばから 30代半ばの方々を中心に好評いただいております。

　※定員人数を超えた場合には、日程調整及び早期に申込みサイトをクローズさせていただくこともございます。
　お早めにお申し込みください。

■詳細・申込ページ https://www.progdence.co.jp/seminar/LdapManager_seminar_2017-11.html
