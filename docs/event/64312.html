description

このセミナーでは同人誌の「印刷」「レイアウト」の基礎を解説します。
「文章は書ける」でも「同人誌は出したことがない」という人に向けたものです。
このセミナーが終わるころには「自分の書いたものを同人誌即売会に持って行く準備」ができるようになっています。

誰もが最初は「右も左もわからない」ところからスタートします。
でも、時々、「右も左もわからないから諦める」人もいます。
それはとってももったいないことだと思い、このセミナーを開講することにしました。

「同人誌を作ってみたいけど、どうすれば良いかわからない」
「身近な人にはちょっと質問しにくい」
といった、誰もが抱えがちな小さな課題を解決します。

二人の講師と三人のスタッフが質問を受け付けながらゆっくりと説明していきます。

当日の概要
①ごあいさつ
②印刷業者の使い方
(1)印刷業者に委託するメリット
　お金を払う価値はどこにあるの？
(2)印刷業者の選び方
　金額？納期？業者の選び方って？
(3)見積もり
　どのくらいお金がかかるの？
(4)印刷業者に発注する手順
　実際にどのくらい発注に手間がかかるの？
③レイアウトの考え方
(1)印刷ソフトの使い方
　どんなソフトを使えばいいの？
(2)レイアウトの基本的な考え方
　工夫をしたらどんな風に見栄えが変わるの？

*今回は主に文章系の同人誌を制作する際のセミナーとなっています。
