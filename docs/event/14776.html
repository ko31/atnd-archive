description

熱いWeb業界と話題と人気のサービスを盛り上げるべく、業界人、いわゆる「中の人」と、
ユーザー、ファンのみんなが楽しく、心ゆくまで交流できる、ネットワーキングパーティー
を企画いたしました。
SocialWeb2.0Party(そーしゃるうぇぶつーぽいんとおーぱーてぃー)です。
第1回は、2009年7月24日に緊急開催され、60名の参加者が集い盛り上がりました。
「ぜひ次回も」の声に答えて、さっそく「第2回目」を、9月3日に開催し、80名の参加
者を集め、「特別編：Twitter国内関連サービス「中の人」ミーティング」、「第3回」、
「大忘年会」と開催。2010年は、さらにパワーアップして開催してきました。。
そして、2010年12月6日開催の総括イベントに続いて、2011年は、3年目へ向けて開催し
てまいります。
2011年の第2回目は、震災に際して、Twitterやfecebookなどのソーシャルメディアが
「ソーシャルインフラ」として機能したことをテーマに取り上げ、震災復興支援企画
として開催します。

後援は、NPO法人 日本ITイノベーション協会／スマートモバイルコンソーシアム様です。

各分野の識者の方々をお招きしてのスペシャル講演を予定しております。

みなさまのご参加をお待ちしております！

※収益の一部は、日本赤十字社を通して義援金として寄付いたします。
