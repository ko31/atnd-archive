仙台iOS開発者向け勉強会 第6弾を開催します！

内容

来たる6月23日（土）に、iOS開発者向けの勉強会を開催します！　

進行は、いつも若干ゆるふわです。セッション内容はライブコーディングを見つつ話をしたりと、なかなかギークな内容になることが多い、わりとガチ系の勉強会です。

仙台でiOS開発者が集まって話をする機会も貴重かと思いますし、情報交換の時間もご用意しております。是非ご参加ください〜。

※いつもは参加費無料で実施しておりましたが、今回は会場費(ドリンク付き)として1人1000円をご負担頂きます。何卒、ご協力をお願い致しますm(__)m

対象

  • iOSアプリ開発をやっている方
  • プログラム経験があり、iOSアプリ開発もやってみたくて、本を1冊くらい読んだことがある方

タイムテーブル

現在Session発表募集中です！（出来れば、WWDCの内容をベースにした発表があると良いななんて思ったりして・・・）

2012/06/17追記：
Session内容を追記しましたー。

Session 0 自己紹介・情報交換（雑談 / 世間話 / 井戸端会議） 60 min
Session 1 storyboardが便利らしいのでその辺やってみます。 by @h_moriさん 60 min
Session 2 UIViewのアニメーションを使ってみるテスト by @tototti。UIViewのアニメーション機能で、いろいろ動かしたり回したりして遊んでみます。 60 min

場所

・コワーキングカフェBeehive+（びーたす） さんを貸切で使用させて頂きます。
・定禅寺通りと晩翠通りの交差点近くのローソンの北隣です。
・今回は、勉強会の会場使用代として、1000円（ドリンク付き）を頂戴します。何卒、ご協力をお願い致します。

参加される方へ

・MacBookなどをお持ちの方は、ご持参下さい。（コーディングを行う時間があるかもしれません）
・会場には無線LANのネット接続環境があります。

懇親会

・終了後、勉強会と同じ場所で懇親会を行います。
・懇親会参加費は、「800円(チャージ・オードブル)＋飲み物代金(ソフトドリンク350円〜 アルコール550円〜)」です。
・懇親会参加の方は、申込時のアンケートにて「懇親会に参加する」を選んでください。
