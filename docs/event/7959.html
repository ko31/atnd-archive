description

インターンシップに周りが参加する中、
就職活動といえば、企業研究をしているあなた。
その研究の仕方は本当にあっていますか？
本当にほしい情報が得られていますか？

「と言われても、どうやったらいいかわからないから、とりあえず調べてます。」
その気持ち、わかります。私たちも当時はそうでした。

就活ってどうやるの？
そもそも、仕事ってどう選ぶの？
でも、流れに乗って就活するのも何か違う！

始めなきゃ、と思っているけれど何をしていいかわからない。
そう思っている方に、必ずプラスの時間になります。
みなさんの一番知りたいことを知っているのは「人事」です。

今回は、「就活を通じて自分のやりたいことを実現する人生を送ってほしい。」
そんな想いを伝えたく、就職活動時代の仲間であり、あるベンチャー企業の人事である
社会人4年目の同士で就活サロンを開くことにしました。←無料です。
みなさん一人一人とじっくり話したいので、あえて15人限定で開催します。
しかも口コミだけの告知で、近い友人関係にある人事が開催するから安心です。

このセミナーに参加すると 　
・仕事選びの軸がわかります
・企業を選ぶポイントがわかります
・数百人の面接をしてきた現役人事に就活のホンネが聞けます
（知っていること、何でも話します。）

就活に本気で取り組みたい！と思っているみなさんのご参加をお待ちしています。
ウェブや他のイベントでは得られない、生の情報をお伝えします。

□メンバー
・廣田　正史
新入社員で入社以来、のべ1000人の面談・面接を行い、キャリアコーディネートを行う。
入社6ヶ月で支店の立ち上げメンバーの一員として運営基盤の確立に貢献。
個人予算2000万円/月、支店運営の全般を任される。
現在は人事として採用・研修を主に担当し、のべ400人の学生と接する。

・松本　貴子
学生時代から「企画すること」「人に機会を提供すること」に興味を持ち、
就職活動支援団体での活動や女子大生による女子大生のためのキャリア予備校の運営にのめり込む。2007年ベンチャーに入社。新卒採用プロジェクトリーダーとして採用に携わり、年間1,000名を越える学生と会い、10名の学生を採用。現在はコーディネーターとして勤しむ。
