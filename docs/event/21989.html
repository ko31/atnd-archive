AWS概要

クラウドとは、Amazon Web Services とは何か、全般的にお話しさせて頂きます。
今までの社内やデータセンターを利用したインフラと、クラウドとどう異なるのか。AWSでどう実現するのか。
また、AWSには仮装サーバ以外の有用なサービスも多数展開されています。併せてお伝え致します。
http://aws.amazon.com/jp/

S3解説

Amazon Simple Storage Service (S3) は、格納するデータ量に従って課金されるストレージサービスです。
非常に高い堅牢性と可用性を提供しており、データが失われる心配はないといっても過言ではありません。暗号化もできますし、各オブジェクトには一意のグローバルなアドレスが割り当てられるため、すぐに公開できます。
このストレージサービス S3 について、解説致します。
http://aws.amazon.com/jp/s3/

EC2解説

Amazon Elastic Compute Cloud (EC2) は、従量課金制で利用できる仮装サーバを提供するサービスです。
ものの数分でサーバが立ち上がり、スペックや台数の増減も容易です。
また Amazon Machine Image (AMI) というOSイメージをもとに、Linux から WIndows まで様々なOSを利用できます。
より最適な仮装サーバを構築できるこの EC2 について、解説致します。
http://aws.amazon.com/jp/ec2/
