WordPressサイトSEO強化ワークショップ

初心者必見！本質的なSEOを学びWordPressで即実践!! あなたのサイトをアクセスアップ！［ハンズオン講座］

　

開催概要

WordPressをインストールしただけでそのまま使っている方など初心者の方を対象にSEOの基本とWordPressサイトの機能を強化する方法をお伝えします。
検索エンジン上位表示やアクセスアップに繋がる最適なサイト運営の基盤を構築するワークショップです。

☆次のような方におすすめです！

  • WordPressをインストールしただけでそのまま使っている方
  • WordPressでとりあえずサイトを構築してみたが，次に何をしたらよいかわからない方
  • プラグインのインストールや設定の方法がよくわからない方
  • 本質的なSEOやWordPressサイトの改善方法を知りたい方
  • SEOを意識したコンテンツ制作やキーワードの選定方法を学びたい方
  • サイトの検索エンジン上位表示やアクセスアップの為の基盤を構築したい方
  • より多くの方に情報を届けたい方，ビジネスを提供したい方

アジェンダ

◆イントロダクション

◆SEOの基本と重要ポイント

  1. SEOの概要(SEOとは何か？)
  2. 検索エンジンのアルゴリズムとSEOの重要ポイント
  3. サイトのディレクションとコンテンツの方針
  4. キーワードの選定と各種ツールの活用方法
  5. サイトの内部構造の最適化
  6. サイトの被リンク

◆WordPressサイトのSEO機能強化

  1. パーマリンク設定(URL構造)の最適化
  2. プラグインによるWordPressのSEO強化(インストール及び各種設定)
    ー メタタイトル/メタディスクリプション/メタキーワードの設定
    ー パンくずリストの設定
    ー サイト表示の高速化
    ー ソーシャルメディア(SNS)の連携
    ー ウェブマスターツールの連携
    ー アクセス解析ツール(Google Analytics等)の設置
  3. SEOに有利なWordPressテーマの紹介
  4. 記事投稿の最適化
  5. 更新通知サービス(ping送信)及びXMLサイトマップの送信設定
  6. サイトの表示速度及び動作テスト
  7. サイトのSEO解析ツールの紹介

◆ その他 (質疑応答，個別相談 等)

☆ハンズオン形式で解説しながら進めていきます。
