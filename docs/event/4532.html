description

Web制作ではさまざまな知識が求められますが、求められる知識が多岐にわたるようになり、ひとりで知識のすべてを身につけ、ひとりですべてを作るのは難しくなってきました。また、内容も高度化し自分ひとりの中で消化するのが難しくなってきています。そこで、近隣のWeb制作者同士で集まって、いろいろな技術や知識について考え情報交換して、明日からの業務に役立てていきませんか、という小規模な勉強会です。

単純に、まったり楽しく集まって、自分が作ったものを披露したり、ほかの人の制作環境を知ったり、消化できなかった／知らない知識を知ったり、みたいな集まりにしたいです。

今回は窓から差し込む海のきらめきが気持ちのよい部屋で行います。人数分、高速インターネット回線／オフィス・制作ソフトがインストールされたパソコンがあります。
