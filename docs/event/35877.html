第15回 FreeBSD勉強会

インストーラを使わないでFreeBSDをインストールする方法

講師
後藤大地
経歴
BSDコンサルティング株式会社取締役 / 有限会社オングス代表取締役 / FreeBSD committer

FreeBSDのインストールでは通常、インストーラが使われることが多い。インストーラはディスクのパーティショニング、プロダクトの展開、基本的な初期設定などを代替でおこなってくれる。実際にはコマンドを実行してパーティショニングやプロダクトの展開を実施しているだけなので、インストーラを使わなくてもFreeBSDをインストールすることができる。

本勉強会では、実際にFreeBSDを手動でインストールする方法を実演しながら、FreeBSDの仕組みや、インストーラが実施している内容を深く理解することを目的とする。また、インストール後の基本的な設定についても、どのコマンドを使って何を設定するのかを説明する。FreeBSDを管理することになったビギナー管理者にもオススメの内容。
