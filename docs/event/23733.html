CSS Holic vol.8開催

CSS Holic vol.8 開催のお知らせです。
今回は今までとだいぶ内容を変更したものになります。
ですので、以下の内容によく目を通して参加・不参加をご検討ください。
その内容とは、（すでにサブタイトルに出てますが）ずばり「 コーディングバトル！ 」です。

いきなりなに言ってんだって感じですがｗ
今回の流れは↓の感じになります。時間割についてはまだ検討中なので細かいところは修正が入るかもしれません。

今回の大きな目的としては、

「他の人がコーディングしている姿を見る」

というものです。

コーディングバトル

どういうことかというと、運営側で用意したデザインを元に、
4チームに別れて、それをどう組み立てるのか、普通の案件のように打ち合わせをしてもらいます。
その上で、そのチームの中からひとりを選びその代表者がコーディングをする、というものです。

さらに、コーディングしている風景に関しては2台のプロジェクタを利用して
ふたり同時にコーディングを開始してもらい、完成度、完成スピードなどを競ってもらいます。

制限時間は（今のところ）30分を想定しています。
つまり、30分以内でどこまで組めるのか、30分でより多く組むにはどうしたらいいのか、などを
コーディングバトルの前の戦略会議で決めてもらう、ということです。

コーディング以外の戦略も大事

なので、コーディングバトルだけではなくその前の会議も重要になってくるので
言わばチーム戦ということになります。

と、バトル推しですが、実際のところは他人がコーディングしている姿をまじまじと見ることはあまりないと思うので、
その風景を見ながら、なにかしらの刺激が得られたらいいなーというのが目的です。

最終的にはみんなに投票してもらって勝敗を決めてもらうところまではやろうと思っています。
評価の軸・根拠については人それぞれだと思うので特に判断基準は設けません。
（スピードが早いからという理由で高得点でもいいですし、綺麗な構造だからということで評価しても構いません）

ちなみに、コーディングバトル中は解説者を立てて、
ちょっとした演出を考えています。
なのでコバさん、よろしくお願いしますね( ´⊇`)
