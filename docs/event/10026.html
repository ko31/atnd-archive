description

「Facebookってどんな使い方すればいいのかわからない？」
「上司からビジネスで使えないかチェックするように言われたけど、よくわからない」
「ファンページ作ったんだけど何かよくわからない？」
「Facebookを利用してキャンペーンをしたい会社があるんだけど、色々と教えて！」

こういう相談が先月からとても増えてきて、来年あたりにはブレイクしそうな予感いっぱいのFacebook
いち早く、あなたのウェブ施策に取り込んでみませんか？

まだまだ国内では成功事例も少ない中で、いち早く活用し、あなたが成功事例として取り上げられるようになりましょう！

今回のセミナーでは、Facebookがこれから何故必要になってくるのか？
そして、Facebookを活用しないとウェブ施策上、大損をしてしまう可能性について
更に、その運用のポイントまでをわかりやすく解説いたします。
（基本的なFacebookの利用法なんかも説明しますよ）

セミナー後には、希望者全員で飲みに行って、懇親を図りましょう。
