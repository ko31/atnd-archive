Start recruit programとは？

Ptech Lab主催の就職活動プログラムです。10月頃から本格化する就職活動に備えて、就職活動で必要となるノウハウを無料にて提供致します。就職活動の準備からエントリーシート、面接対策、そしてエンジニアになるために必要なアプリ開発を通じて、エンジニアになるための第一歩を全面サポート致します。もちろん、エンジニアを現時点では希望していない方でも、少しでもIT業界に興味がある方は是非、参加を検討してみてください。
本プログラムは、学生が主体となり、企業、他団体、社会人等の協力を得て実施するプログラムですので、知識、知恵が豊富に吸収できるプログラムです♪

第一弾　就活の流れを知って、友人よりも一歩先へ

★就職活動って何？やりたい仕事が見つからない、どうしたら良いかわからない。先輩には大変だけど。。と言われるけど何が大変なの？
★就活を進めていくコツとは、漠然と流れに乗るのではなく、能動的に行動するということ！だけど、そのために必要な基本とは！？
★まだ誰にも聞けずにいる疑問がある…だけど友人には聞けない…こんな問題を解決したい！

などなど、就職活動はやってみなければわからないこと。
今回の第一弾では、就職活動の全体の流れやスケジュール、そしてどのような行動をしていくべきなのかという点に焦点を当てたセミナーを実施致します。

第1弾は初歩的の内容ようです。恥ずかしくて聞けないようなことでもドンドン質問可能です！
就職活動における全体像のイメージを持てるか持てないかで大きく行動に影響します。まずは、基本となるイメージをつかみ、就職活動が始まった段階で、友人より一歩€先に進んでおきましょう。
