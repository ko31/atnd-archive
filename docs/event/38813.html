iPhoneアプリを作って見ませんか？
本セミナーは未経験者や初心者の方でもiPhoneアプリ開発を体験することができます。初心者の方でもベテラン講師がアプリ作成をサポートするのでご安心ください。

スマートフォンアプリ市場は、国内外において最も成長している分野です。このビジネスチャンスを逃さず、最大限に生かしていくためにアプリ開発に関する知識やスキル習得は必須のものとなっています。

スマホアプリを作る楽しさを体感することで"やってみよう””試してみよう””自分でも学んでみよう”といった最初の一歩を踏み出しましょう。

■下記の開発機器を御持参ください。
・Mac（MacBook AirまたはMacBook Pro、Mac OS X 10.7.3以上）
・【任意】iOS端末（iPhoneまたはiPadまたはiPod touch）
