熊本サイクリングオフ(仮称)

イベント ユーザー
event_id 31018
title 熊本サイクリングオフ(仮称)
catch ちょっとそこまで行ってくる
description
  • 熊本周辺で主にIT系の自転車乗りが増えたような気がするので、ちょっと皆でツーリングしませんか？
  • 熊鹿ファミリーロードを往復するコースになります休憩を挟みながらゆったりと進むサイクリング(往復70km強)です
  • 初心者参加を前提にしているので、玄人な方は、荷物持ちや風除けを頑張りましょう
  • 雨天中止ですし、予約などもないただゆっくりとしたサイクリングですので、趣旨をご理解のうえお申し込みください。

    【詳細】
  • 「熊鹿ファミリーロード」の開始地点にある「ハローグリーンエブリー島崎店」前に集合し、簡単な説明、出発となります
  • 距離が50kmを越えるため流石にママチャリはキツいかと思います。ロードバイク、クロスバイク、マウンテンバイク等のスポーツ自転車を推奨します
  • 片道休憩込みで2.5h ～ 3hを見積もっています。
  • 帰ってきたら希望者は最寄の温泉で汗を流しましょう

    【注意事項】
  • 夏です！非常に暑いです！体調管理に気をつけてください
  • 塩分、水分はこまめな補給を心がけましょう
  • コースは自転車専用道がメインのため、コンビニ等は多くありません。事前にある程度の携行が必要です
  • 出発前に日焼け止めを塗っておくと吉(3時間ぐらい毎に塗りなおしましょう)
  • ライト必須、負担がかかるため「ヘルメット」「手袋」「サングラス」を推奨します。特に「ヘルメット」については走行上の安全のため可能な限り確保ください
  • 井芹川沿いを走るため虫が多いかと思います。フェースガードをつけるか、覚悟を決めるかしておいてください。(主催は覚悟側でｗ)
  • 道路交通法に則り安全なサイクリングを心がけましょう
  • 救助車等の準備はありませんので、調子が悪い場合は、早期に申し出てください。なるべく自走での帰還を目指します。
