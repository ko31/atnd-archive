【キュウショクについて】
キュウショク≒九州の食単なる言葉遊びではありません。
食事の供給だけでなく、「食育」の役割も担うようになった「給食」のようにキョウショクでの経験から、九州への関心につなげ、九州好きをもっと増やしたい、育むのは、「LOVE 九州」の輪。
キュウショクなりの「ショク育」に取り組みます。
「給食」に感じるある種のノスタルジーは、九州で育ったぼくらにとって、それは、九州の記憶。いまは九州を離れて暮らすオトナになり、「給食」をとおして九州を感じている、そんなぼくたちが、まずはキュウショク当番となって、「LOVE 九州」の輪を広げます。

はじめは、定期的なイベントから。そして、九州の縁でつながった円を囲んで語れる場所、「KYUSHOKU CAFE」のオープンを目指します！
