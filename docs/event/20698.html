description

自分の自分による自分のためのイベント、新宿 Book-a-thon をやります。
やることはシンプルで集まって黙々と本を読みます。
読書会ではないので、参加者で読み合わせるなどは行わず、
各自がそれぞれ好きな本を持ち込んで読みます。

☆こんな人におすすめ！！
・技術書を買うけどつい積んでしまう人
・どうせ積むだけと技術書を買わない人
・空いてる時間があるとついコードを書いてしまう人

☆Book-a-thon はあなたの読書ライフをサポートします！！
・電子書籍に対応したイベントです。
・単に集まって本をもくもく読むだけです。
・参加者はみなばらばらの本を読むので、サポートは特にありません。
(他の参加者に質問してみたりするのは自由です)
・読書が主であれば何をやっても構いません

☆前回やっていたこと
・自己紹介 + 読む本の紹介
・もくもくと読書
・わからないこと、興味のあることについて雑談
・本を読みながらコードを書いてみる

というわけで、普段自分では本を読まないので、
たまには読書でもしてみるか、という試みです。
同じような境遇の人は参加してもいいんじゃないでしょうか。

会場は特に決めておらず、当日になってから空いているお店を探します。
(ルノアール、ドトール、マクドナルド等の喫茶店に向かいます)
混雑具合によっては場所が確保できずに右往左往する可能性がありますがあしからず。

※ 直前に Twitter の #bookathon タグで連絡を取り合いながら店に移動します。
