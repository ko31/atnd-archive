■セミナー内容

【都市の景観】

平木啓一先生による【都市の景観】をテーマにしたセミナーです。
「セミナー概要」
　ヨーロッパ諸国では、1600年初頭頃から神聖ローマ帝国の版図に入る小国を中心に、自国の主要都市の景観を描く貨幣発行を行う傾向が見られた。 これらは出来る限り忠実に市の中心部を描いたことにより、その面影を今日に伝えてくれている。 　神聖ローマ帝国――ドイツ諸邦は32箇国という多きに達しており、イタリア、ポーランド、スイスなどに較べて、圧倒的に多かった。 ターレル銀貨など大型の貨幣だけでなく、1/4グルデン、1/3ターレルなどの小型銀貨、ゴールトグルデンやドゥカット金貨など小型金貨にも、市の景観を描いたのである。
「お食事内容」
日本各地で育まれた食文化を映した料理のコースをお楽しみいただきます。

近海で捕れる魚介や契約農家から無農薬野菜など、旬でこだわりの素材を活かしたシンプルながらの技の光る逸品です。

厳選した食材を一品々を調理人が心を込めて調理したランチコースです。
