description

xCCKは、汎用データベースモジュールの一種で、フィールドを自由に定義し様々な用途に使うことができます。
http://jp.xoopsdev.com/modules/wiki/?Menu%2Fxcck

factory は、cubson/tubson をベースにXCL2.2向けの機能も取り込んだモジュールを簡単に作ることができるモジュールジェネレータです。
http://jp.xoopsdev.com/factory

■対象：
・モジュールのインストールなど、XOOPSの管理やブロック、テンプレートなどXOOPSの基本的な概念についてある程度理解されている方。
・XOOPS Cube Legacy を自分でインストールできる方
・思い通りのサイトを作りたいという意欲のある方
・モジュールを作ってみたいという方（PHPとか知らなくてもOK）
・テンプレートは多少触れるほうがいいです。

■必要なもの
XCL2.2が動作するテスト環境
＃ノートパソコンにXAMPPをいれるなどしてご用意ください。
参加無料です。
