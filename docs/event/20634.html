来る 2011/10/22(土)に Okinawa SuperHappyDevHouse を開催いたします。懇親会も予定しておりますので、ご興味のある方はぜひご参加ください。

[会場]

〒904-2234　沖縄県うるま市字州崎14番17 IT津染パーク

★地図など

入り口は駐車場側の自動ドアから 。1枚目(外側)の自動ドアは自動で開きます。2枚目(内側)はセキュリティ上、内側からしか開きませんので、どーにかして中に居る人に気づいてもらってくださいw
また、14:00 から 15:00 の間は基本的にフリータイムです。今回からは津梁パーク 5F にあるレキサス会議室で実施する方向で考えておりますので、必ずフリータイムの間に入館しておいて下さい。
基本的には 14:00 から 14:50 の間に 1F 入口近くで待機しておきますので、その時間帯に来ないと館内に入れませんので、その旨ご承知置き下さい。

[内容(予定)]

iPhoneやandroid、サーバサイドでもデザイナでも、誰でも参加できるし、誰もがやるべきことをやる勉強会

時刻内容
14:00-15:00会場準備、フリータイム
15:00-15:10開会、席決めなど
15:10-16:10
  1. 環境構築、scaffold の先に進んでみよう
  2. テニスのエントリシステムを考えてみよう
  3. “大会” has_many “種目” だよね
  4. User と "種目"は 多対多の関係だよね( user has_many item through => :entry)
  5. ActiveRecord, rails console の話(テニスエントリの概念投入完了[C や Vの部分はこの時点でコーディングしていない])
  6. TDD の話（私はRSpec 知らないので概念的な話）
  7. controller と view を実装しよう
  8. ダブルスって皆さんはどう設計します？
  9. deploy(Capistrano や heroku とか)
  10. アプリはリリース後の改造が楽ちん＆楽しい＆夢が膨らむ(アプリの一人歩きする!?)
16:10-16:20休憩
16:20-17:00fluxflex meetup #1 で僕が見聞きした事を発表 (予定)
