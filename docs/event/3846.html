◆初心者のためのカジュアルなワイン会を開催します。

『赤坂 カジュアル ワイン サロン』

－第1回 ワインの品種を飲み比べて知ろう!－

【開催内容】

日々の食卓を豊かにするワインライフを提案。

ワイン初心者の方に　「食べる」という生きることに欠か

せない 大切な瞬間に　ワインで彩りを加え　当たり前の

時間に 楽しい価値をアップさせるためのシンプルで簡単

な知識を 実際にテイスティングしながら身につけていた

だきます。

★日時：4月17日(土) 15時～17時

※14時40分より受付開始。

★会場：Akasaka NOTE

東京都港区赤坂2-17-55サイバーフォースビル5F

(地下鉄千代田線赤坂駅下車5番出口より徒歩5分)

地図→ http://akasakanote2009.syncl.jp/?p=custom&id=2355211

★会費：4,000円(ワインと料理付)

★募集人数：30名様限定、着席スタイル

※定員になりしだい締め切ります。

★主催：インタラクティブエンゲージメント合同会社

【ワインコーディネート&ミニセミナー講師】

美人ライフコンシェルジュ　川原愛子

http://ameblo.jp/bijin-no-recipe/

【講師略歴】

ライフワークとして「美人のレシピ」を主宰。

「美しく生きる」をテーマに様々な活動を展開しています。

日本ソムリエ協会認定　ワインアドバイザーを1999年に取得。

パーソナルコンサルタントとして　お一人ひとりのビジュア

ルから 内面までより素敵に生きることをコンサルティング

しています。

【ワイン】

赤・カベルネ・ソーヴィニヨン、メルロー、シラー、(ピノ・ノワール) 白・シャルドネ、ソーヴィニヨン・ブラン、シュナン・ブラン

【お料理】
