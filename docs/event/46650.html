分析化学における測定値の信頼性

イベント ユーザー
event_id 46650
title 分析化学における測定値の信頼性
catch 検出限界と定量下限の考え方、不確かさ、信頼性を判定するための検定など、「はかる」上で必要となる知識を学び、演習を交え実務に活かす！
description

【講座のポイント】
　「はかる」という行為は分析化学に限らず我々の生活に深く浸透している。しかし正しくはかることは易しいことではない。また測定した値は最終的な分析値ではない。測定値を分析値にするのは実験者の仕事である。本講演ではそのために最低限知らねばならない基礎的な事項を解説する。講演は分析化学を基軸とするが、計測はすべての自然現象の数値化に共通の行為であるため、他分野の方でも概ね理解できるように講演を行う。演習も取り入れて、確実な理解を助けるよう工夫したい。
【プログラム】
Ⅰ「はかる」とはなにか？
　1.1　測定値とは何か
　1.2　分析化学において正しく「はかる」ことの意味
　1.3　計測値の信頼性を保証するシステム
　1.4　化学分析における測定値の信頼性
Ⅱ　有効数字
　2.1　有効数字の桁数とその意味
　2.2　数値の丸め方
　2.3　演算に伴う有効数字の処理
　2.4　分析方法による有効数字の具体例
　2.5　有効数字の観点から見た分析値の報告例
Ⅲ　検出限界と定量下限の考え方
　3.1　「検出限界」「定量下限」「感度」とは何か
　3.2　検出限界に関する2つの考え方
　3.3　原子スペクトル分析法における検出限界の算出
　3.4　定量下限を見積もるには
　3.5　検出限界や定量下限付近の結果をどのように表記するか
Ⅳ　信頼性の用語
　4.1　信頼性にかかわる概念や評価方法の推移
　4.2　化学計測領域における信頼性用語
　4.3　物理計測あるいは数理統計における信頼性にかかわる用語
　4.4　電子工業における信頼性にかかわる用語
　4.5　濃度に関わる用語と国内外の規格における濃度単位
Ⅴ　不確かさ
　5.1　「不確かさ」とは何か
　5.2　不確かさの見積もりの基礎（Ａタイプ・Ｂタイプ・合成）
　5.3　不確かさを見積もる前にすべきこと
Ⅵ　不確かさの実際の見積もり例（秤料、溶解、希釈から測定・データ整理まで）
　6.1　定量分析において取り扱う数値
　6.2　分析手順に関する標準不確かさの見積もり
Ⅶ　信頼性を判定するための検定
　7.1　信頼区間
　7.2　代表的な検定法とその考え方
　7.3　実際の検定例
Ⅷ　よりよい分析値を提示するための工夫
