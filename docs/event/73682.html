■発表者

・株式会社 SUPINF はら やすひろさま
大企業向け ERP パッケージベンダーでの会計製品開発を経て、ゲーム開発/MSP 事業を手がける株式会社 SUPINF に SRE として入社しました。スマホ向けネイティブアプリケーションから Web アプリケーションの開発、AWS インフラ設計や運用、開発環境の構築など幅広く担当しており、特に最近は Docker の布教と運用にはまっています。JAWS-UGコンテナ支部という勉強会の運営もやってます。
Twitter: ＠tori_cluster

発表テーマについて：
サーバーにSSHでログインして svn update でデプロイしていたチームが、なぜ Docker を本番環境でクラスタリングすることを目指したのか、そこに至るまでの経緯と実践してきたことを紹介します。

・日本IBM株式会社 田中ゆういちさま
アナリティクス事業部　テクニカルリード
金融系・組み込み系・コミュニティサービス・ゲーム・広告系・オンライン・ショップ系と、多様なビジネスでの重要な機能を担うエンジニアとして豊富な経験を持つ。サーバーサイド、フロントエンド、ミドルウェア、インフラと幅広く担当できるフルスタックエンジニア。Aerospikeのコントリビュータでもある。最近はHadoop/Sparkエンジニアとして活動してます。

発表テーマについて：
昨今ビッグデータへの注目のたかまりもあり、spark/hadoopエンジニアの不足やデータサイエンティストの重要性が増しています。今回はApach の中でも人気の高いSparkを使った処理を体験してみます。
