38歳の田口が教えてくれる貯金ゼロからのお金の授業

catch ～自己破産寸前から立ち直った貧困脱出経済マインド～
description

仕事にせよ投資にせよ、うまくいき、かつその状態が続いている人の共通点として,ストイックなほどに自分を律し、目標や自分との約束事に対してブレずに努力していることが挙げられます。
頑張ってよい結果を出し、経済的成功を手にするも、そこに気の緩みが生じて、生活態度が一変して身を持ち崩し、お金も人も去り、結果として振り出しよりマイナス状態に陥る人も多い中、ピークの状態をキープし続けることは、ひょっとするとそこに至るまでの道のり以上に、高いハードルの意志の強さが求められるのかもしれません。

今回登壇の田口智隆氏は、自己破産寸前の状態から、わずか数年で経済的に不自由のないセミリタイア状態を実現しましたが、濡れ手に粟で浮かれまくったりするわけではありません。
傾きかけた人生を建て直したのは、ある決断と実行。
そして現在もそれは継続されています。
「金持ち父さん」に憧れて実現できる人とできない人の大きな違いはここにあります！
素手でトイレ掃除をすることや、その時々の経済状況に最適な投資方法や、これから伸びる銘柄がどうこう言う以前の、もっと大切なことがあるのです！

弱い自分や誘惑に負けずに、
経済的に幸せになるための心構えをお伝えします！

★こんな方におすすめ
■お金が貯まらない■つい無駄遣いや浪費をしてしまう■投資失敗■儲かったことがあるが続かない■家計の収支をどうにかしたい■飲み歩き貧乏■ギャンブル貧乏■見栄っ張り■老後の生活費が不安■カード使いすぎ■毎月赤字…といった方に効果が見込まれます！

★講師：田口智隆（たぐち ともたか）
1972年さいたま市出身。
株式会社ファイナンシャルインディペンデンス代表。
貯金なし・自己破産寸前の状態から、6年で経済的ストレスフリーの生活を実現。
現在は、多くの人の経済的自立の支援をミッションに、個別相談・カウンセリング、講演・セミナーなどを積極的に行なっている。
