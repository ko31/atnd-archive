title 注目市場ベトナムをトップランナーが語る！
catch
description

スタートアップも注目する、成長著しい東南アジア市場。そこで、東南アジアセミナーの第１弾として、急成長中のホットなベトナムマーケットをテーマにアクセルビート株式会社が「注目市場ベトナムをトップランナーが語る！」と題してセミナーを開催致します。実際にベトナムで活躍される方々をお招きし、ベトナムの魅力とさらなる可能性、また現在のビジネスモデルに囚われない新しい挑戦の方法等に関してキーノートプレゼンやパネルディスカッションを行います。アジアマーケットやベトナムマーケットに関して興味がある方、生の声による情報を知りたい方、新たなパートナー等を探したい方等のご参加をお待ちしております！セミナー後には軽く懇親会も行いますのでこちらも是非ご参加ください！

★開催日時★
3月14日（木曜日）19:00 ~ 22:30

★開催場所★
西新宿界隈（後日決めさせて頂きます）

★主催★
アクセルビート株式会社（アクセルマーク株式会社の海外戦略子会社、ベトナムを始め東南アジア諸国へ向けた検索エンジン提供、クラウドソーシング事業、メディア事業、広告事業、を展開中。）
