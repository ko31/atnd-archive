はじめまして！
この度、東京日本橋にあるレンタルオフィス『ウィズスクエア』様主催のもと交流会を開催いたします。

私、この会のマスターを務めさせていただきます、INCNT株式会社の西村と申します。
よろしくお願い致します。

いつもの交流会・・・ただの名刺交換で終わってませんか？

そんな交流会ではないお酒を飲みながら、ビジネスの話をできる友人を作りませんか？

友人を作り、楽しみや苦しみを分かち合い、コミュニケーションできる場所。
そんな時間を作って行きます！

夢を語れる人が少なくなっている昨今。大いに夢の話をしましょう！


第一回は皆様のおかげで盛り上がりました。

今回は第2回です。前回は焼きそばでしたが、今回は別の軽食をご用意いたします。その場で作って皆で食べる食事は格別です。

皆様のご参加お待ちしております。
