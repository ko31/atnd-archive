講演内容

大学進学率の上昇,入学者の多様化に伴って大学教育にはこれまでのような知識や技術中心の教育よりも幅広い内容が求められるようになっている。そうした要請のキーワードに「○○力」があり,中教審の提唱する「学士力」はその代表的なものである。こうした「○○力」概念は社会の要請をわかりやすく示すものとして有効であるが,その反面それがあくまでも「比喩的な表現である」点で教育実践への使用にはさまざまな問題がある。この講演では,そうした「○○力」概念の持つ問題点を明確にした上で,そうした概念によって社会が示す大学への要請に大学がどのように応えうるかを考察する。
