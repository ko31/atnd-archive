description

近年注目されている「NOSQL」（Not Only SQL）と総称されるデータベースをご存知でしょうか？

最近では、GoogleやFacebookなど、数兆ページ・数億人もの大量データの高速処理が必要なウェブサービスで採用されるなど、注目が高まっています。

GoogleやFacebookの規模まででなくとも、多くのWebサイトが抱える大量アクセスやサイト内検索等のレスポンス対策に効果が期待できます。

また、Webサイトに限らず、抜本改善が困難な企業の保有する顧客データや商品情報・各種ログデータといった大量データの管理・活用においても、改善策につながる可能性があります。

KDLでは、これまでのNOSQLに独自の機能を豊富に追加した"okuyama"を独自開発し、2011年9月より企業利用に適した商用ライセンスの下、国内初の企業サポートサービスを開始いたしました。

http://okuyama-project.com/

okuyamaの企業サポートサービス開始に伴い、9月22日に大阪・梅田でNOSQLとokuyamaについての無料勉強会を開催させていただきます。

「NOSQL」がなぜ高速・大量データ処理に優れているのかの説明に加え、国内初の企業サポート付きNOSQL「okuyama」についても、その特徴と機能を詳しくご紹介いたします。

特別ゲストとして国内有数のサーバサービス「at+link」の文屋宏氏をお招きします。文屋氏は、okuyamaの持つ圧倒的なポテンシャルに早くから気付き、ソーシャルアプリ運用に特化して提供する「at+link アプリプラットフォーム」にokuyamaを組み入れ、サービスの差別化を実現されました。サーバ周辺事業だけでなく、okuyamaがもたらす大きな可能性について、具体的にお話しいただきます。

内容

■Session 1 「NOSQL/okuyama入門」

（講師：株式会社神戸デジタル・ラボ　岩瀬高博）

キー・バリュー型のデータ構造を持つNOSQLデータベース"okuyama"について、設計・開発に携わった岩瀬本人から分かりやすく解説いたします。

■Session 2　「at+linkがokuyamaを採用した理由」

（講師：株式会社リンク　ディベロッパーサポート　マネージャー　文屋宏氏）

at+linkが2011年3月1日より開始された、ソーシャルゲーム特化型専用サーバパッケージ「at+link アプリプラットフォーム」には、キャッシュサーバ機能として"okuyama"が採用されています。本セッションでは、同パッケージにokuyamaが採用された理由や、サーバ事業者様から見えているokuyamaのメリットなどについて、お話しいただきます。
