概要

ソフトウェアテストコンサルタント・秋山浩一さんの新著 『事例とツールで学ぶHAYST法』
を教材とした勉強会です。
ゲストとして参加してくださる秋山さんと一緒に、ソフトウェアテスト技法を学びましょう！

本書のタイトルには 「HAYST法」 という言葉が入っていますが、いきなりHAYST法の話
から始まるわけではありません。
前半には、HAYST法の前提となる 状態遷移図/表、 直交表 、そして PathCombo法 の
解説があります。第1回は、このPathCombo法を理解することを目標にします。

なお本イベントは、 @niftyエンジニアサポート 様のご協力で、会場をお借りしています。

全体の流れ

  1. 本書の2.2節「応用的なテスト技法」の解説
  2. 演習問題を解く
  3. 演習問題の解説
  4. 秋山さん質問・コメントコーナー
