ドールをテーマにしたスマフォアプリについて、ご意見を募集します

ドールをテーマにしたスマフォアプリを企画し、
実際にゲームを作る準備のためのリサーチを行っております。

ご意見をいただきたく、下記のような方にお話をうかがう機会を設けたいと思います。

  • ドーラーの方（ゲームに興味あり/なし は問いません）
  • 「ドールのゲーム」に興味のある方
  • 女性向けドール企画ですが（その趣旨をご了承いただける）男性の方の参加も歓迎です

どうぞお気軽にお立ち寄りのうえ、ご意見をお聞かせください。
アットホームな雰囲気で進めたいと思います。

注意事項

進行中の企画ですので、秘密は守っていただきますようお願いします。
（SNSなどに、内容や当日の様子をアップしないでくださいね）

交通費をお出ししますので、ご参加いただいた際に
往復分の額面をご申請ください。
