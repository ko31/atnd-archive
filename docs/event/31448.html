catch from legacy to modern
description

一からパッケージ作成を覚える…のではなく、既存の古めのパッケージをいじってモダンなパッケージに「なんということでしょう！」とビフォーアフターしてみようという集まり。

キーワードは以下。


  • dh7-9/cdbs (でも主催者はcdbs分かってません）
  • source format 3.0 (quilt) (= drop dpatch)
  • build hardening
  • xz compression
  • lintian warning/error
  • multiple upstream source
  • new upstream release

各自行うこと


  • リファクタしたいソースパッケージを宣言（2個ぐらい？）
  • どのあたりが気になるのか、があったらそれもメモしておく
  • 知りたいこと、もメモしておく
  • リファクタしたいパッケージを pbuilder でビルドできる環境をローカルに整えておく

持ち物


  • ノートPC（可能な限りローカルでビルドできる Debian 環境。unstable推奨）
  • もしあればインターネット接続可能な機器

日程と場所が確定しました。。オープンソース・ソリューション・テクノロジ株式会社様の会議室を提供していただけることになりました。感謝。
