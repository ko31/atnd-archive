無料ポイント解説セミナ

この講座は「HTML5プロフェッショナル認定資格」受験を考えていらっしゃるかたがたの最初の一歩としてのガイダンスです。

「HTML5プロフェッショナル認定資格」は、NPO法人LPI-Japanが、次世代のWebプロフェッショナルのスキル向上のため、HTML5を駆使したWebページ・アプリのデザイン・設計・構築に関する体系だった知識・スキルを備えた「HTML5プロフェッショナル」を中立的に公平・原生に認定する資格制度です。

さらに、Webデザイナ・技術者が取得したいと思う「JavaScript」「マルチメディアコーディング」「HTML5」をすべて含んでいます。⇒ http://html5exam.jp/

東京アプリ・ワークショップは、NPO法人LPI-Japanの認定するHTML5アカデミック認定校として、皆様の資格取得のための、最初の一歩としての「解説セミナ」を開催します。
