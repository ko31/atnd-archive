Google Cloud Platformをもっと知ろう！GCPUGに参加して盛り上げよう！支部を立ち上げよう！

【概　　要】
いよいよGoogle Cloud Platformのユーザー会、GCPUG(Google Cloud Platform User Group)が立ち上がりました！GCPUGはGCPを全国に広めるためにGCPの良さを皆様にお伝えします！初めての方は是非この機会にGCPUGに参加してGCPの素晴らしさを享受しましょう！そして一緒にGCPを盛り上げていきましょう！
GCPが初めての方のためにGCPの全体概要の解説や、アカウント開設からのハンズオンもあります！（※無料枠が使えますがクレジットカードが必要です）
後半ではビールや軽食を交えての交流会を行います。お気軽にご参加ください！

【こんな方に向いています】
GCPって何？美味しいの？という初心者の方。
GCPはちょっと知ってるけどもう少し詳しく知りたい方。
GCPを一緒に盛り上げたい方。
とにかくGCPが大好物な方。
