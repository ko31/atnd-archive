description

【講座のポイント】
　企業において技術情報を収集する必要が発生する。それを効率よく収集し、しかも内容が高度なものでなければなりません。そのテクニックをお伝えします。

【受講対象者・レベル】
　企業の調査・企画・研究開発・新製品開発・新事業開発・技術提携・投資・M&A・経営戦略・マーケティング・人事などの部門の方々、およびこれからベンチャー・ビジネスを創業したいと考えている方々、調査会社で調査報告書の作成速度を上げ、内容を充実させたい方々に最適です。レベルは初心者から、部長クラスまで。

【プログラム】
１．技術調査の必要性・重要性
２．技術調査の種類
３．技術調査の進め方
４．技術調査の方法
５．海外技術調査の方法
６．技術情報の集め方
７．特許による技術調査
８．技術者の採用育成のための技術調査
９．技術経営（MOT）の動向
１０．技術調査マン十戒
付録
　　A.技術調査のためのインターネット情報源
　　B.技術調査の参考書
　　C.日本が世界経済のなかでいきていくには「技術立国」しかない
　　D.アイデア実現研究会
　　E.戦争なしに技術進歩を加速する「世界技術大賞」の提案
　　F.著書
　　G.自己紹介
