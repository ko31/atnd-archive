title 倒産可能性を引き下げるキャッシュマネジメント
catch キャッシュはストックこそが重要
description

　企業のマネジメントは、資金繰りが逼迫するまで、資金の問題を気にしようともしない。
　それでは、満足な仕事をするには手遅れとなる。
　ドラッカーが「現代の経営」の中で語っている言葉です。
　資金＝キャッシュは、フローだけを見ていても十分な管理はできません。たまたま一時期キャッシュフローが良好だからといって、財政状態が良いとは限りません。
　そうです。安定経営を目指すなら、常にフローと合わせてストックを見ておかなければ成りません。
　しかし、残念ながら、キャッシュのストック面を正しく分析できる人は専門家でも多くはありません。
　この経営ゼミでは、短時間でフロートとストックとをバランスよく管理し、そして倒産しにくい体質作りのために必要なことをお伝えします。

１．キャッシュはフローだけ管理してもダメ
２．専門家も間違っている貸借対照表の読み方
３．キャッシュに注目してこそ安全性は保てる
４．フローとストックを一表で管理する目から鱗の方法
５．あるべき未来のキャッシュから描く目標
