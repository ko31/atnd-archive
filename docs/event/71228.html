撮影企画 「わたしのチャレンジ」について

スポーツ少年団、クラブ活動、部活動
個人競技、団体競技、文化活動 など 何でも結構です。
あなたが日常、取り組んでいる活動をポートレイトで記録します。
年齢、性別は問いません。何かに夢中になっていることがあれば、ぜひご参加ください。
グループとして何人かでお集まりになってもＯＫです。

継続は力です。毎日コツコツと続けていくうちに、ものすごいことができるようになっていきます。
一生懸命努力する姿は、とても素敵なものです。

辛い時もあると思います。やめてしまいたいと思うこともあると思います。
でも、続けていたからこそ、できるようになったことがあるはずです。

今までのがんばり、これからの決意、目標、記念、そんなことを含めて
ポートレイトを撮ってみませんか。きっといい思い出になると思います。

撮影について

撮影料は、無料です。一切の費用はかかりません。
当店スタジオにて撮影を行います。
活動がわかるようなものを持参してください。
道具、楽器、ユニフォーム など
こんなポーズをしてみたいな、といったことを考えると
より一層、撮影が楽しめると思います。
ベストショットをＡ４サイズ程度の写真プリントにしてプレゼントいたします。
後日郵送、または店頭にてお受け取りいただけます。
撮影期間

2015年10月12日（月）体育の日
2015年10月18日（日）
2015年10月25日（日）
なお、10月31日まで、平日17:30以降でしたら撮影ができる場合がありますす。
下の申し込みフォームより、随時お申し込みください。
日時につきましては、先着順ですので、希望日時をうかがい、調整をさせていただくことがございます。

プレゼント

ベストショットをＡ４サイズ程度の写真にしてプレゼントいたします。
後日郵送、または店頭にてお受け取りいただけます。

参加条件

本人に撮影参加の意志があること。
本人が、撮影されたいという積極性を持っていることが大切です。
お子さんの場合は、保護者の方が無理やり企画に参加させるのはご遠慮ください。

活動報告やＰＲにお写真を掲載させていただきます。
時期は未定ですが、写真展を開催する予定です。
活動の輪を広げていきたいと考えています。ご協力ください。
