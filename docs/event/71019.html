来る 10/24 の Kalafina の LIVE 後にオフ会を開催致します。
ライブには全国から有志が集結されることと思いますので
全国のKalafinaファンと交流を深めませんか？

1人からの参加も大歓迎です！
私も初のオフ会は1人での参加でしたし遠慮せずご参加頂ければと思います。
まだライブ後の予定が決まってない方は是非ご参加ください！
　
　

【日時】
　　10/24（土）21:30～23:30（予定）
　　集合場所は会場入り口付近。
　　前回と同じ場所、つまり会場を出て右手側の広場です。
　　詳細な場所はtwitter等で当日に画像をアップします。

　　時間はライブの終了時間によって前後する可能性がありますが
　　20:30に終わると仮定して、集合～参加者の確認の後
　　お店までの移動時間もかかりますので1時間後の開始を想定しています。

　　ライブの終了時間が想定と異なった場合でも
　　集合場所へお集まり頂けますようお願い致します。
　　アンケート記入や物販などの都合もあると思いますので
　　集合場所にすぐ集まれそうにない方は早めに
　　幹事までご連絡くださいませ。
　
　

【場所・定員】
　　まだ決まっていませんが、会場から徒歩圏内にはお店が少ないので
　　梅田方面へ移動する可能性が高いです。
　　できるだけ早めにお店の確保は行いたいと思っています。
　
　　スムーズに開始するため、１杯目のドリンクはビールかウーロン茶でお願いします。
　

【会費】
　　4000円程度を予定しています。
　
　

【注意事項】
　　何らかの事情によりキャンセルされる場合必ずご連絡をお願いします。
　　連絡のないキャンセルは、お店にも他の参加者の皆様にも迷惑ですから、
　　絶対になさらないようにお願いします。
　　3日以内のキャンセルは会費の徴収をさせていただく場合がございます。

　　会場では、一般常識、社会的モラルに反する行為は禁止です。
　　（暴力、暴言、ナンパ、セクハラ、未成年飲酒、飲酒強要など）
　　喫煙者の方には、禁煙へのご協力をお願いします。

　　未成年の方の参加はお断りさせて頂くことがございます。
　　深夜までの会であることと、飲酒等トラブルが発生した場合に責任が取れません。
　
　

【連絡】
　　原則として個別での連絡はせず、このイベント内容に追記を行うか
　　私自身のtwitterやmixiでのつぶやきによる連絡をする予定です。
　　どちらかのアカウントをお持ちの方は ID を教えてください。
　　どちらもあまり使われていない方はこちらのサイトにて
　　詳細を把握して頂けますようお願いします。
