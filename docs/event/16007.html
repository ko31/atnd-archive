TDD Boot Camp 札幌 2.0 SP1

本イベントは、前日に同会場で行われる TDD Boot Camp 札幌 2.0 の２日目となります。１日目に参加していなくとも参加可能ですが、開催要項と趣旨を確認の上、ご登録下さい。

日時

2011年6月5日（土）9:30 ～ 16:40

タイムテーブル

09:00〜09:30 開場
09:30〜16:40 自主トレ
16:40〜17:00 撤収

・ノートPC用の電源は用意します
・入退出は原則として自由ですが、前日に参加者同士で約束している場合はちゃんと時間にきましょう
・講演やお題などは用意しません
・各自、自由に前日のお題をやったり、疑問点を相談したりしましょう
・別の言語で同じお題にチャレンジするのも良いと思います

概要

TDD Boot Campの２日目は、完全フリータイム（自主トレーニング）です。
前日のお題でやりのことした事、一晩寝て思いついたアイディアを試してみる事、他の仲間と相談する事など、自由にお使いいただけます。
1日目で消化しきれなかった事などを、緩い雰囲気の中でやってみましょう。

趣旨

TDD Boot Campは非常に体力を消費するイベントです。
２日開催がデフォルトになっていますが、1日目に朝から夜まで慣れないペアプロや乱取り形式、テスト駆動などを行い、懇親会まで参加すると疲労困憊です。しかし、１日目の感想を聞いてみると「なんとなく解ってきた、でも、そこで終わってしまった」と感じている人が多数います。しかし、２日目も参加できるかと言えば、体力的に厳しいことも事実です。

そこで、２日目は大きなお題やセミナーは実施せず、緩い雰囲気で講師・ゲストと話したり、一晩休んで出たアイディアを試したり、中途半端であった１日目の課題を進めたり、全く他の言語で再チャレンジしてみたりと、自主トレーニング的な場所を提供しようと思います。
