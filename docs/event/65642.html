3月に開催されたFacebookの開発者向けカンファレンス f8。注目されたのは、何と言ってもMessengerプラットフォーム！

…ただ今回のf8では、Messengerプラットフォーム以外にも【アプリマーケターにとって必ず知っておくべき発表】がたくさんあったのはご存知でしょうか？ 技術的や目に見えてイノベーティブな内容ではなかったこともあり、日本国内のテック系メディアでは、残念ながら大きく取り上げられることはありませんでした。

実際は、アプリマーケティングや広告に関わる私達にとって、大きな転換となる可能性がある内容が含まれていました。

という訳で、今回はアプリマーケターの皆さんに向け、【Facebook Messenger以外の】Facebookカンファンレンスのまとめセミナーを開催させていただきます。

講演内容/講演者

19:30-19:55 IOTや第三世界マーケットについて / 株式会社Phybbit 代表取締役 大月聡子

講演概要

ParseのIOT用SDKや開発キットが公開され、Facebookがいよいよ本格的にIOTのマーケットに参入してきました。また、internet.orgの発表で、今後の第三世界のマーケットの可能性も広がってきました。 まだappデベロッパーの方々には馴染みのないお話かもしれませんが、今後のマーケットの可能性や広がりをお話させて下さい。

講演者略歴

20:00-20:25 LiveRailとFacebook SDKのNativeAd対応について / 株式会社トーチライト Sherpa事業部 プロダクトマネージャー 三原 敬太

講演概要

F8で発表されたビデオ広告配信プラットフォーム「LiveRail」のディスプレイ広告への参入と、Facebook SDKのNativeAd対応について、その意味をあらためて紐解くとともに、それがもたらすモバイルアプリ広告への影響とその意味を考えます。

講演者略歴

20:30-20:50 アプリ向け解析ツール Facebook Analytics for Apps / アドイノベーション株式会社 メディア事業部 部長 細川竜馬

講演概要

「えっ！無料でここまで！？」という広告あるあるクリエイティブですが、Facebookがリリースしたアナリティクスツールは、冗談抜きでそんな言葉が出てくる出来です。 今回はFacebook Analytics for Appsの概要と、簡単な機能をご紹介し、遠大なFacebookのたくらみ(私の予想ですが。。。)をお話させて頂きます。
