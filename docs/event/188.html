description

金持ち父さんになる男Ｏちゃんです。

あなたは金持ち父さんを目指してますか？
それとも超えようとしてますか？

まわりに同じ道を目指す仲間はいますか？
いるならその人達と一緒に行きましょう。

もし、探しているのなら、この会に参加してみてください。
きっと同じ方向を目指す仲間が出来ますよ。

わたしもキャッシュフローゲームに出会い、
このゲーム会に出会った事で、
明るく前向きな仲間をたくさん作る事が出来ました。

そして今度は仲間を求めて転々とするのではなく、
自分が仲間を集めることを出来るようになりました。

正直こういう会を主催するのが苦手だったわたしに
やり方やノウハウを教えてくれた多くの方がいらっしゃいました。

自分の目指すモノがあり、一緒に目指す仲間が欲しければ
ここに来てみてください。

自分の目指すモノがない…そういう方はここに来てどんなモノが
あるか確かめてみてください。

今のご時世「お金」の話は「怖い」と感じるかもしれませんが、
今のご時世だからこそしっかり覚えておきたい話でもあるんです。

人生お金だけではありません。
でも、やりたい事を実現しようとすれば必ずお金がついて回ります。
今の収入で実現出来る事であるなら、気にしなくても良いでしょう。
もし「実現したい事、やりたい事が、今の収入では出来ない」とお考えのひとは、
一緒にどうすればそれが出来るか考えてみませんか？

あなたのお越しを心からおまちしています。
