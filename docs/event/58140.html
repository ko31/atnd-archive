description

「Live2D」は『ガールフレンド(仮)』 (サイバーエージェント)、『pepita』(井上雄彦氏)など国内外の 100 以上の商用作品で採用されている高品質な 2D キャラクター表現技術です。 スマートフォン、3DS・PSPなどのコンシューマーゲーム、電子書籍など多くのプラットフォームで実績があり、UnityやWeｂGL、AfterEffectsなど多彩なのゲーム開発環境との連携が可能です。 このイベントでは、Live２Dのデータ素材がつくられる工程をLiveで実演します。 まったく初めてLive２Dに触れる方にはその楽しさを、またプロのゲーム・アニメ制作者には開発スピードの速さやわかりやすさを実感していただきたいと思っております。 そして、Live２Dの表現の幅を大きく高める次世代技術をご紹介いたします 。制作者の皆さんとキャラクターアニメーションの次のステージについて語り合えることを楽しみにしております。
