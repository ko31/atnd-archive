description

HTML5はいろいろな機能がてんこ盛りの次世代の標準規格です。

特にゲームに最適な「Canvas」という機能は、四角や三角、円を簡単に描くことができ、それらを組み合わせれば様々な図形を描くことができます。しかもアニメーションにして動かすことも。
この講座では、基本的な図形の描画からアニメーションまでご案内し、最終駅にクリスマスツリーを描くことを目標としています。
プログラムでアートすることを「アート・スクリプティング」と言います。またアニメーションを発展させれば「ゲーム・アプリ」を作ることもできます。ぜひそのきっかけにしてください。
