ゆるい初めてのゲームジャム「ゆるふわゲームジャム」

ゆるふわゲームジャムって何？どこがゆるふわなの？

ゆるふわゲームジャムとは

ゆるふわゲームジャムとは、「ゲームジャムに興味はあるけど、本気のゲームジャムにいきなり参加するのは怖い」
という方や、「本気のゲームジャムは疲れる、もっと気楽に楽しみたい」という方を対象として開く

ゆるくできて、ゲームの完成より楽しむことに重点を置くゲームジャムです

どのへんがゆるふわなのか

  1. 開催時間は短め、ゲームを無理に完成させなくてもＯＫ
  2. 持ち帰っての作業は極力なし！体に無理をしない！
  3. 運営側も企画は初！gdgdにならないようにがんばります!

どんな人に向いているの？

  1. ゲームジャムに興味がある方。
  2. 普段とは違う役職をやってみたい方。
  3. ユニティちゃんゲームジャムに出られなかった方。

経験豊富な方には是非講師、リーダーとして参加してほしいです！

つまり"ゲームを作ること"自体を楽しみたい方向けです。。

ゲームジャムのテーマは？

テーマは「ユニティちゃん」

このイベントを企画した理由が、６月のユニティちゃんゲームジャムに参加できなかったことだったので、
テーマはユニティちゃんゲームジャムと同じ「ユニティちゃん」にしました。
ユニティちゃんゲームジャム番外編のようなものとして開催できたらと思っています。
