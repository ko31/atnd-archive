Lifelog Meetupはライフログをテーマに参加者のみなさんとディスカッションを行うイベントです。
毎回、プレゼンテーション１本とディスカッションという構成の予定です。


第一回目なのでひとまずテスト的に開催します。ライフログと言っても、あまり堅くならないライトな会にしたいと思っていますので、お気軽にどうぞ。


第一回


プレゼンテーション：テーマ「Photomemo」
作品集ではなくライフログとしての写真共有サービス。開発当時のスタッフが設計コンセプトと裏話を話します。


ディスカッション：
よりよいライフログツールは何か。そもそも何でログしたいのか、今回のテーマ「Photomemo」の活用法などについて参加者にみなさんとざっくばらんにお話しできたらと思います
