札幌圏のiOSアプリ開発者コミュニティ、iPhone Dev Sap主催の勉強会です。
会費は無料です。

iOS実機をお持ちでMacにXcodeをインストール済みの方に限らせていただきます。

参加者の方は全員、自己紹介を兼ねて簡単な発表をお願いします。
（発表内容は技術的なものに限りません。こういうものを作りたい、勉強したい、などでも構いません。時間は数分程度でも構いません）
また、アプリ制作をおこなう上で、聞いてみたい質問を各自1個以上ご用意下さい。

当日は交流しやすいよう、名札をご持参ください（名札の形式は自由です）

前半は発表形式。
後半はフリートーク＆フリー質問＆フリー開発。

終了後に懇親会も予定しています。
懇親会の参加登録はこちら→ <a href = “http://atnd.org/events/28116”>http://atnd.org/events/28116

参加者同士で情報交換などする時間もあると思いますので、
なんらかのネタ（いま作ってるもの、いま困っていることetc）があると有意義に過ごせるかと思います。
