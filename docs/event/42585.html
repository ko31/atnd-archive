概要

2013年8月26日(月) に FabLab Kannai が正式にオープンします。
それに先立ち、24日(土) に Night Party を開催します。
正式オープン前の FabLab 0 Kannai に参加して頂いてる方々や、
これから FabLab Kannai に参加したい方々との交流の場にしましょう。
