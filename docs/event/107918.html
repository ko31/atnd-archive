講演概要

企業の魅力発信をイノベートさせるセミナー

  • 概要、講師コメント

今から3年前に
「いわてで働こう推進協議会」が
岩手県内の学生にアンケートとった結果、
『県内企業を
1社も知らないと答えた学生が3割、
5社以下しか知らないと答えた学生は9割』
という話を聞いた。

就職時に学生の県外流出が多い岩手で、
個人的には学生が岩手から出ようが、岩手に
残ろうがどちらでも良い派だけど、
『岩手の企業を知り、残ることと、
残らないことのメリットを知った上で学生が
選択する』方が学生にとってはより良い人生の
選択になるよなと思ったのが3年前。

でも、そもそも企業側にも責任があり、
『学生への魅力発信の仕方が
上手くないんじゃないのか？』
『学生に魅力があると思ってもらえる企業に
果たしてなっているのか？』とも思い、
だったら自分が学生に面白い企業の魅力を
伝えよう！との想いに至る。

その後、
採用担当でもないけど
『ふるさと発見！大交流会』に参加し、
第一回目は最優秀賞、そして第二回目は
優秀賞と2年連続学生からありがたいことに
高評価を頂いた。

そして遂にこの取り組みや、
裏エピソードを交え、社外に発信することに
至りました。

3年分の想いをプレゼンします！

まずは一ノ関で9月5日にやります！

そして、
盛岡でも8月30日に富士ゼロックス岩手の
本社でやります！
