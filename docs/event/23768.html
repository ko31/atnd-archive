description

これだけは知っておきたい！！家づくりを成功に導く為の知恵をお伝えします！

家づくりの基本と押さえておきたいチェックポイント、自分らしい家を創る工夫など、家づくりの成功する姿がイメージ出来ます。
家族の為の間取りの作り方など、成功する秘訣をお伝えします。

個別相談の時間も設けています。

家づくりの進め方や、土地探しの方法、工法・技術、お金の話、法律の話から、気を付けた方が良いポイント、自分らしい家を創る工夫まで知っている様でいて、中々知らないものです。

家づくりが成功する為の知恵をお伝えします。

震災以降懸念の、地盤や構造的な配慮、エコな住宅の作り方もお話します。

ちょっとしたワークをして貰ったり、和気あいあいと、 行っていますのでお気軽に、ご参加ください。

会場：新宿文化センター　第５会議室
