名古屋でパターン認識（PR），機械学習（ML），コンピュータビジョン（CV）について勉強します．

概要

第1回を予定していましたが，どうなるか分からないため，顔合わせ・雰囲気を掴む意味を込めて第0回としました．
第0回では，今後の活動内容に関する意見交換や各自が話したいことを持ち寄る自由形式としたいと思います．
PR・ML・CVに関することならば特に制限は設けません．

お知らせ

  • 当日は台風が通過するため，強い風雨が予想されます．あまり無理をしないでください．来れない場合は@miyabiartsにツイートして頂けると助かります
  • 事前受付なしの当日参加も歓迎です．
  • 途中参加の場合，@miyabiartsに連絡頂けると助かります．
  • 参加者の方は，簡単で良いので，自己紹介をお願い致します．現在行っている研究，興味あることなどを話していただけると有難いです．
  • 説明のための資料などは各自印刷でお願いします．
  • 会場には，インターネット環境はありません．

内容

  • 自己紹介（形式は自由）
  • 今後の活動内容に関する意見交換
    • 開催日時
    • 方向性（輪読・研究紹介・サーベイ発表・デモなど）
    • Twitterのハッシュタグとか勉強会の名前（通称）とか
  • その他，話したいことがあればなんでも

今回は特に事前に決めていることがほとんどないので，当日やってみない点も多いです
