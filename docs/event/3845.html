= 4/23（金）開催！就 活 イ ベ ン ト の お 知 ら せ =

この度、SE/PG志望学生の皆様および参加企業から高い評価をいただいている就職イベント「IT逆求人フェスティバル」を東京にて開催いたします。前回開催分では、一次・二次選考を免除される参加者が続出！有名IT企業が注目しているエクセレントイベントです
是非、この機会をご利用下さい。

■参加費：無料

□対象：2011年新卒の学部・修士・博士の学生またはポスドクで、研究・趣味でプログラミング経験のある方

（経験対象例：C, C++, JAVA, PHP, Python, .net, Perl, UNIX, Linux）

□参加企業数：10社

「逆求人」形式で自分をアピールする経験は、今後の就職活動の場面で大いに活きてくるはずです。一度限りの新卒就職を、是非有意義なものとして下さい。
