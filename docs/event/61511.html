description

・Unreal Engine 4をOculus Riftで使ってみよう！ということでゆるーく勉強会を開催します。

・ビギナー向けに導入や使い方みたいなことが出来ればとは思っていますが、
　具体的に色々と決めているわけではないので、みんなで集まって雑談しながら
　いろいろと遊んいければと思います。

・Oculus Riftを体験してみたいという方もどうぞ。

・事前準備として可能であれば、Unreal Engine 4をインストールしたPCを
　用意していただければと思います。
　
　〈UE4推奨スペック〉
　・Windows 7 64 ビットまたは Mac OS X 10.9.2 以降
　・クアッドコア Intel または AMD プロセッサー　クロック周波数 2.5 GHz 以上のCPU
　・NVIDIA GeForce 470 GTX または AMD Radeon 6870 HD シリーズ以上のGPU
　　※ただし、Oculus Riftをしっかりと動かすためにはGPUのスペックが
　　　より高い方が望ましいです。(GeForce 9XXシリーズ etc…)
　・8 GB以上 の RAMメモリ

・会場となるフィーバー柳橋ですが、少々たどり着きにくくなっています。
　こちらの動画を御覧ください。
　　　→https://www.youtube.com/watch?v=zZgP2Durgpk

　動画ではわかりにくい場合もありますので、ストリートビューのリンクです。
　画像真ん中に見える部分が入り口です！
　　　→https://goo.gl/maps/QZv0p

　尚、日曜日はビルのセキュリティロックがかかっております。
　申し訳ありませんが、到着された方は入り口でフィーバー柳橋（052-485-5335）
　までお電話ください。
