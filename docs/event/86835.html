イラレ / フォトショ / WordPressの制作作業や学習において誰かに訊きたい。教えてほしい。そんな方向けのフリー時間です。

2時間半、自由に作業していただけます。
もくもく会と異なるのは、私は講師役として作業中の皆さんの疑問・質問にお答えします。（対応可能範囲は後述）
デザイン・Web制作を進める場・学習の場として活用してください。

･･･････････････
■開催背景
･･･････････････
これまで イラレ / フォトショ / WordPressをテーマに勉強会を行ってきました。

私の勉強会では後半に自由な作業時間を設けることがありますが、この時間を頼りに来られる方もいらっしゃったり、「同じテーマを行うときはまた参加したい」「続きを自宅で進めたときに出てきた疑問点は質問したい」というお声を頂くこともあります。

また、単純にイラレやフォトショの使い方を教えて欲しいというお話をいただくこともあり、自由に訊ける場があれば便利に活用していただけるかと思い、今回の企画となりました。

もちろん、今回が初参加でも歓迎いたします。
イラレやフォトショ、WordPressをはじめてみたい方、初めたばかりの方、疑問にお答えしますので、よろしければご参加ください。

需要がありそうなら、通常の勉強会とは別に時折開催できればと思っています。

･･･････････････
■自己紹介
･･･････････････
社会人5年目、転職して3社目なウェブの人です。

デザインは専門学校でグラフィックデザインを。
ウェブ制作は独学で学び、アルバイトから業界に入りました。

1社目は小さなウェブ制作会社。ここでWordPressと仕事の基礎を学びました。

2社目.. というべきでしょうか。フリーランスをしていました。
とある会社の常駐がメインだったので2社目ということで。
WordPressを使ったウェブ制作（デザイン/コーディング）、名刺/ロゴデザインなどを手がけていました。

3社目は現在の会社。
大手IT系企業でウェブディレクターとして勤めています。

･･･････････････
■こんな方向け（対応可能範囲）
･･･････････････
初心者〜中級者序盤向けとなります。
これから学習していきたい、学習を初めたばかりという方の疑問・質問を想定しています。

=
・Adobe Illustrator / Photoshop
　　└本などで学習しているが１人ではわからない部分がある
　　└名刺を自分で作りたい、ブラッシュアップしたい
　　└ロゴデザイン、バナーデザインなど、デザイン作業を相談しながら進めたい

・WordPress
　　└基本的な使い方を知りたい、教えて欲しい
　　└持っているWordPressサイトの改修・改善相談をしたい

・HTML/CSS
　　└コーディングの学習で躓いた所を訊きながら進めたい
=

※私は疑問・質問にお答えし、あなたの作業サポートを行います。作業代行は基本的に行いません。（それをすると１人の方にかかりきりになってしまうためです）

※私のWordPress / コーディングスキルについて
・HTMLで作成したページをWordPress化する経験があります。
・1からWordPressテーマを作成した経験があります。
・PHP開発（WordPressプラグイン開発）はできません。
・JavaScript を含めたプログラミング言語についてもお答えできません（私も勉強中です）。
