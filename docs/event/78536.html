- わずか3日で300人を見込み客にできる出版集客術!-
起業して3年以内の経営者のためのはじめての電子書籍出版セミナー

電子書籍って聞いたことはあるけど、実はよく知らない人も多いはず。
このセミナーは電子書籍をまだよく知らない人、出版には興味はあるけど自分にはムリだと思っている人、電子書籍を自分のビジネスに活かしたい人に、電子書籍のはじめの一歩がわかるセミナーです。
まずは、電子書籍を知ることから始めましょう！

電子書籍は
『求めているお客様がやってくる魔法のツールです!』

高い広告費を払って集客するよりも、電子書籍を出版した方が効果的な方法だってご存知でしたか?
実は、電子書籍は最強の販促ツールなんです!
電子書籍ならお客様が自らあなたのことを探して
感謝されながらお客様を作ることができるんです!

■こんなことで悩んでいませんか?

集客に悩んでいる。　→　たった一つの仕掛けで自動的に集客ができる!
営業が苦手。　→　たった3日であなたのお客様が見つかる!
無名なので信頼度が低い　→　一瞬で信頼されてメジャーになれる!
毎月広告費をかけられない。→　無料で作って、収益を上げながら宣伝できる!
HPやブログで効果が得られない。　→　何もしなくてもお客様からお願いされる!

一つでも当てはまるお悩みがあれば、まずはセミナーにご参加ください!
電子書籍出版は新しいチャンスを掴み取れる場所です!

ビジネスを広げるには電子書籍から!

■こんな方に向いています。
・電子書籍をまだよく知らない方
・出版には興味はあるけど自分にはムリだと思っている人
・電子書籍を自分のビジネスに活かしたいと思っている人
・お金をかけずに集客したい人

このセミナーでは、
電子書籍の基礎知識、印税のヒミツ、
どうして電子書籍で集客ができるのか、
出版に必要なものが丸分かりのセミナーです。

多くの個人作家さんが知らない
『出版に必要な3つのモノ』がわかります!

セミナー参加者限定
売れる書籍に必要な
『3つの魔法のシート』をプレゼント!

■このセミナーで習得できること。
・電子書籍の今がわかる!
・電子書籍を出版するメリットがわかる!
・印税のヒミツがわかる!
・ネタの探し方がわかる!
・出版に必要な3つのモノがわかる!
