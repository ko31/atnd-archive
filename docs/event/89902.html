高度なIT化が進み、クラウドサービスの導入、リモートワークなど
働き方改革が行われつつある現代。情報セキュリティ対策は切っても切り離せません。
…と、私たちは一応理解してはいます。

しかし、現実のセキュリティ対策はどうでしょう。
皆さんの会社では対策が過剰になりすぎて
働きにくくなっていることってありませんか？

情報セキュリティ対策のせいで業務に不都合が生じていることを挙げて
今後どういった取り組みをすべきかを議論しませんか？

セキュリティ初心者の方でもお気軽にご参加ください。
本当に必要なセキュリティ対策って何なのか、ゆるく楽しくお話しましょう。
