ゲームプランナーといえば「好きなゲームを企画できる花形職業」というイメージがありますが、実際にはどうなのでしょうか。
今回は、フリーランスのプランナーによる実体験をふんだんに交えた講演と、プランナー職経験者によるトークセッションから、抱かれがちな理想、そして憧れだけでは勤まらないゲーム業界の現実を紐解きつつ、ゲームを作る環境や働きかたについて模索します。
プランナーやゲーム業界への就職・転職に夢を抱かれている皆さんや、ゲームプランナーってどんなこと考えて、実際どんな仕事をしているんだろう…等、ご興味のあるかた、お気軽にご参加ください。

■内容（予定）

・クリエーター職を諦めない 職務経験ゼロからの挑戦
・企画書とは企業へ送るラブレター
・バランス調整とは目に見えぬ悪魔
・プランナーなのに、ゲーム開発に関われない？
・コンシューマでなければゲームじゃない？
・世界を広げる「ゲームとの関わり方」を知る　など

■講師

小川浩史（おがわ　ひろし）さん：
フリーランスのゲームプランナー。2002年デジタルクリエイターカレッジWAO!を卒業後、Irem Software Engineering、SNK PLAYMOREなどを経て、2014年より独立。コンシューマー、PC、スマートフォンと、プラットホームを問わず、ゲームプランニング業務に携わる。
また個人では、福島ゲームジャムで作成した『JumpGun!』が、協賛企業・Unity Technologies Japan賞を受賞するなど、公私問わずゲームシーンに関わる。

■トークゲスト

藤井美樹（ふじい　みき）さん：
京都のディベロッパーにてDSタイトルを中心に、プランナー職を8年経験後、ゲーム業界で開発サービスのソーシャル営業職に従事。

高橋　功（たかはし　いさお）さん：
ゲームとお酒が大好きな３２歳。ＫＣＧ卒業後、カプコンなど数社のゲーム会社でプランナー職を経験。地元大分でゲームクリエイター育成のＮＰＯ団体などで活動しながら、母校の講師として勤務。
