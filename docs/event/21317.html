description

Android Make Daysの1イベントとして，Android OS上で動くものを作るコンテストを行います．
開発時間は24時間，チームもその場で決めます．
賞を幾つか設けています．受賞者には，zeemote, googleグッズ，アメーバクッション等，豪華賞品が贈呈されます！

１日目は「GuildCafe Costa」で一晩開発し，2日目は「Android Make Days」が開催されている
「Ruby・コンテンツ産業振興センター」場所を移して開発を行います．

ドキドキ，ワクワクの24時間！皆さんのご参加お待ちしています！
プログラミング初心者のかたも，Androidで開発したことないけど興味ある人も，
遠隔地からSkypeで参加したいという人もみんな大歓迎です！気楽にお祭りを楽しみましょう！
イベントの詳しい内容は，こちら

※開発中は，拘束時間を設けてませんので，途中で家に帰っていただいてもかまいませんし，Android Make Daysの講演を聞いていただいても構いません．基本的に出入り自由です．

※Android端末は運営側でお貸ししますので，Android端末を持ってない方でも参加可能です．

※13日（日）に，「明星和楽」本体の方で開発した作品のプレゼンテーションを行う時間がありますが，プレゼンテーションへの参加不参加は自由です．コンテスト的には24時間で終了となります．
