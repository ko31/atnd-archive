モバイルコアにてアプリコンテストへの応募を目的としたアイデアソンを行います！！

iPhoneキャンプ、Androidキャンプ、Aiキャンプ、クラウドキャンプ、普段ばらばらに研修をしている方々と交流しながらブレインストーミングするチャンスです。

参加費は無料、筆記用具をご持参ください！

この機会に是非交流を深めましょう♪

「エッグプラント」の由来
英語の「茄子」って意味ですが、事を「成す」とかけつつ、岐阜県大垣市でのスマートフォンアプリ開発者、及びクラウド技術者育成事業の研修生の横のつながりを強化する活動で、僕たち開発者の「エッグ（卵）」を育てる「プラント（工場）」としての今回の事業に感謝しつつ、情報共有・ブレインストーミング・勉強会・交流会などを行う活動です。

また、賛同いただける既にアプリをリリースされておられる方や企業の参加やご協力も歓迎致します。

質問等ございましたら、お気軽にコメント下さい。
