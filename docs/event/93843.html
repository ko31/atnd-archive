冬といえば、メガデモだ!!

メガデモ発祥の地である北欧では「寒すぎて外出たくないけど暇だからコンピューターで遊ぶか」といった感じで全てが始まりました。

Tokyo Demo Festの開催時期変更で冷え切った身体とマシンを暖めるぞ!


概要

メガデモに関連する技術の勉強会を行います。
メガデモが発表されるイベント(デモパーティ)は日本でも毎年行われ、
2018年は秋あたりの開催が予定されています。
http://tokyodemofest.jp

参加費 : 無料


参加対象者

  • メガデモに興味がある方
  • Tokyo Demo Festに参加予定の方
  • リアルタイム3DCGなどの技術に興味がある方
  • 自身でクリエイティブ活動を行なっている方
  • これからメガデモを作ろうと思っている方
  • AMIGAが好きな方
    など誰でも気軽にご参加ください。


タイムテーブル

注意: 入場は開始30分前からです。早く到着しても入れませんのでご注意ください。また、最寄り駅の新宿駅からは徒歩１５分程度かかりますのでご注意ください。
