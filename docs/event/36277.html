Security.GS in北海道

〜ウェブセキュリティ勉強会〜

ゲヒルン株式会社は、2013年2月17日（日曜日）に北海道の株式会社インフィニットループ様　会議室にて「Security.GS in北海道」を開催します。

去年は、「ゲヒルンウェブセキュリティ勉強会」として各地で開催していました。
2013年は「Security.GS 」として勉強会を開催していきます。

この勉強会では、主にWebサービスの開発を始めたばかりの方や、スタートアップ企業で自社サービスのセキュリティをより強化したいエンジニアの方、または最近VPSに触り始めた方など初級〜中級者を対象としていますが、どなたでもご参加いただけます。

講座内容

  • クラッカーの心理
  • XSSの話
  • JavaScriptの話

※ 講座内容は変更する場合がございます。

タイムスケジュール

15：00〜：受付開始
15：30〜16：20　クラッカーの心理、XSSのおさらい、JavaScriptの話
16：20〜16：45　質問タイム
16：45〜17：00　「Gehirn Web Services」の紹介
17：00〜17：30　ライトニングトーク枠（発表希望の方はcontact@gehirn.co.jpまでご連絡ください。）

場所・日時

  • 2013年2月17日（土）
  • 15時30分〜17時30分（2時間）
  • 株式会社インフィニットループ　会議室
  • 参加費：無料
  • 参加定員：30名

講演者の紹介

  • ゲヒルン株式会社　代表取締役　石森大貴
  • Twitter：@isidai
  • 平成２年生まれの２２歳。小学６年から独自にサーバーを作り始める。高校時代に経済産業省の主催する「セキュリティキャンプ」に参加。
    　その後、セキュリティキャンプの講師補助を２年間務め、ＴＢＳが放送したドラマ「ブラッディ・マンデイ」のハッキングシーンの技術考察記事で話題を集める。
    　筑波大学へ進学していたが、休学し独立、起業した。

会場についての注意事項

会場の都合上、平日以外は正面玄関よりの入場が行えません。
ビル正面に向かい左手奥側にある通用口から入っていただくことになります。

１．通用口のあるビルの左側におまわりください。
２．薬局側の角をぐるっと曲がります。
３．ビルの左側をずーっと進むと、ここにインターホンがあります。
４．インフィニットループに通じる　「601」を押して呼び出してください。

詳しくは「土日や夜間のIT系勉強会で会議室をご利用になる方へ」をご参照ください。

その他・注意点等

ノートPCをお持ちの方は是非お持ち寄りください。
なお、会場では無料のWi-Fiが提供されていますが、公衆無線LANとなります。

また、ホスティング会社や情報セキュリティ会社等の同業者の方の参加はご遠慮くださいますようお願い申しあげます。
