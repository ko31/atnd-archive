磁歪材料を用いた振動発電技術とその応用

catch
description

【受講対象】
振動発電の事業化を検討している企業
（自動車，インフラ，電子機器，産業機械，ホームオートメーション，医療関連）

【習得できる知識】
磁歪振動発電の基礎原理と構造，磁歪材料の基礎知識，
振動発電やアクチュエータ技術の利用法，分野

【講座の趣旨】
将来の実用化が期待される磁歪式振動発電について，磁歪材料の基礎から，具体的な発電素子の構造と原理，そのメリット（シンプルで堅牢，高出力，高効率，高耐熱性）を実験結果と試作機による実演を交えて解説する。振動発電デバイスの簡単な設計法からその可能性や応用技術（電池に要らないワイヤレスセンサシステム，リモコン，照明，再生エネルギー発電）と事業化の現状について解説する。振動子や球面モータ，骨伝導イヤフォンなど磁歪材料の優れた特徴（小型，堅牢，高出力，低電圧駆動）を生かしたアクチュエータについても解説する。

【プログラム】
１．磁歪材料を用いた振動発電
　　1-1.一般的な振動発電の背景とメリット
　　1-2.従来技術（圧電材料，エレクトレット，磁石可動型）
　　1-3.磁歪材料と逆磁歪効果，鉄ガリウム合金
　　1-4. 素子の基本構造と原理，特徴
　　1-5. 特性評価と設計
　　1-6. 試作機の実演（小型，大型デバイス，発電スイッチなど）
　　1-7. 応用と市場性，課題など

２．磁歪材料のアクチュエータの応用
　　2-1.鉄ガリウム合金の特徴（製法，加工性，温度特性など）
　　2-2.アクチュエータ（マイクロデバイス）のメリット
　　2-3.研究開発例（振動子，球面モータ，骨伝導イヤフォンなど）
　　2-4.試作機の実演
　　2-5.求められる材料の特性
