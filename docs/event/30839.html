9/8　CKF　ナイトサバゲ大会

title 9/8　CKF　ナイトサバゲ大会
catch 日の光を浴びると灰になっちゃう方も参加可能！
description

コミケ後のお財布に厳しい時期ですが第四回のゲームを企画いたしました！
場所は今までと異なりアウトドア！千葉北フィールド（CKF)での開催です。

しかも今回は「夜戦！」です！！！

開催時間も午後からとなりますので早起きが苦手な人も安心。
22：00には完全撤収なので終電まで余裕があります。

原則的に身内ゲームとなっておりますが 趣旨をご理解頂けるご友人が
いらっしゃればお誘い合わせの上、ご参加下さると幸いです。

■参加表明に関しまして
最終締め切りは9/5（水）とさせていただきます。
ATNDを使用できないゲスト参加の方を お連れになられる場合はお手数
ですが リストを作る関係上、コメント欄に人数とその方のお名前
（HNで可）をご記入頂くよう願いします。
またお連れの方には大会のレギュレーションをかならずお伝え下さい。

■必要な物、参加費
・参加費：6000円
・BB弾：６ｍｍ　バイオＢＢ弾０．２ｇ以下
・装備：ゴーグル（シューティンググラス不可）、顔面を保護するマスク、
　　　　ニーパッドや手袋があるとベター(装備各種レンタルあり)
　　　　お名前（HN）が判るような名札をご用意いただければ幸いです。

　　　　夜戦ですのでサイリウムが必須となります。赤と黄色を二本づつ
　　　　くらいお持ち下さい。当日一本150円にて販売も致しますが事前に
　　　　事前にご準備をいただきますようお願い致します。

・昼食：各自でご用意下さい。
　 　　 または吉野家、ガストのデリバリーが可能です。
　　　　徒歩5分程度の所にコンビニもありますがフィールド外に出られる
　　　　際は装備、BDU等着用のまま外出されないようお願い致します。

■レギュレーション
・パワー制限は銃刀法に定める規制値以下（０．９８J以下）
・フリーズタッチ、ナイフアタック不可
　（痛くなさそうな場所を撃って下さい）
・基本的にブラインドファイア（遮蔽物より銃のみ出しての発砲)禁止
・外部ソースの使用禁止
・レーザーサイトの使用禁止
・ライトに関しては123Aを2セル仕様のものまで

弾数制限、フル/セミ制限等、装備によるハンデも設けません。
ただ、ハンドガンオンリー戦、セミオート限定戦等も開催する予定し、
シューティングマッチも計画しておりますのでハンドガンのみの方も
お楽しみ いただけると思います。

■ちょっとしたお願い
今回からコミュ以外の方の参加も予定しております。
諸般の事情により参加を伏せたい方も居らっしゃいますので大会参加者の
個人名をご自分のブログ等で上げるのはご遠慮いただきますようお願い
致します。
個人で「今日サバゲして来た」と言うのは自由ですが誰が参加していたかに
関しては「触れない」方向でお願いします。
TwitterやMixi等で「○○さんお疲れ様でした」等も出来るだけ空気読んで
下さると助かる人も多いかと思います。
基本的にATNDページコメ欄使って頂きますようお願い致します。

■その他
・フィールドの各詳細確認はCKFのHPをご参照下さい。
・レギュレーション、ゲーム内容等に関しては随時、更新をして参ります。
・主催者は12時過ぎに会場入りしております。
　最終的な装備確認（弾道調整等）、諸注意を簡単に行い午後二時を目処に
　ゲームを開始しようかと考えております。

■■雨天の場合■■
前日より確実な雨天（大雨や嵐）と分る場合は9/7のお昼頃にアナウンス致します。
ただし、フィールドは屋根あり屋根無しが半々位でセーフティは屋内なので、
小雨程度ならば開催いたします。
