description

ＮＰＯ法人みんなの夢をかなえる会は、夢の実現に役立つ場となること、まだ夢が見つからない人が 夢を見つけるためのヒントを得られる場となることを目的に、みんなをワクワクさせ、みんなに夢を与えるすてきな夢に贈られる「みんなの夢アワード2010」を開催します。
　９月８日（水）から11月15日（火）まで公募した「みんなの夢アワード2010」へのエントリーには、209件の応募があり、１次選考会にて選出された５名を本日公式サイト（http://www.miraimeishi.net/award）にて発表いたしました。

（世界共通のお祭りを創り、大切な人を大切にするのが当たり前の世界を実現したい、マサキさん／バングラデシュで貧困層がチャレンジできる国づくりを目指して教育支援を行う、税所篤快さん／悩み相談リレーで人と人とをつなぐＷＥＢサイトを運営する、野口亮太さん／「泊まることで世界のためになるホテル」を開発中の鶴岡秀子さん／世界一夢を与えるマジシャンを目指す、ナカノ・マクレーンさん）

　上記５名がイベント当日（12/21）、参加者の前で「夢」を語ります。最優秀賞には、渡邉美樹が夢を全面バックアップ。どんなサポートかはその夢次第。夢を夢のままにしたくない、かなえたいという強い思いと、サポートする気持ちがここで交わります。その夢のかなう瞬間に立ち会うことが出来ます。

　報道関係者の皆さま、イベントの事前告知のご協力よろしくお願いします。写真素材など必要な場合は、事務局までお気軽にお問い合せ下さい。当日取材も大歓迎です。事務局までお申込下さい。

■■■みんなの夢アワード2010全体概要・流れ■■■
　
「みんなの夢アワード2010」は、当会が企画運営を行う“５年先の夢を名乗って、未来の名刺を作ろう”というコンセプトのホームページ『未来の名刺』【http://www.miraimeishi.net/ 】において、エントリーされた夢の中から、注目の夢を選んでレポートし、一般からの応援投票を受け付けます。さらに、会期の最後には、みんなに夢を与えるすてきな夢に「みんなの夢アワード」を贈ります！

１．エントリー　2010年９月８日（水）〜11月15日（火）
専用フォームからエントリーを受け付けます。応募者数は、209件でした。
２．一次選考　2010年11月22日
一次選考では選考委員たちがひとつひとつの夢を見て5人の通過者を決定しました。
３．一次選考結果発表＆応援投票（ホームページ投票）受付
公式ホームページにて一次選考の結果を発表し、一般ユーザーからの応援投票を受け付けます。
４．最終選考　2010年12月21日（火） 17時〜20時　日比谷公会堂
最終選考は12月21日（火）、日比谷公会堂。選考委員はワタミグループ会長・渡邉美樹。スペシャルゲスト・テリー伊藤さんも応援に駆けつけます。当日の様子は、Ustreamでの中継も予定しています。
