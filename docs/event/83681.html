description

玄米に含まれる栄養素が優れていることは皆さんご存知の通り。しかし、「おいしくない」「炊くのが面倒」･･･という声もよく聞きます。

本講座では、本格的な長時間炊き玄米から、いろいろな炊き方、玄米や玄米粉を使ったアレンジ料理までを一日で学べます。

実演、試食を交え、会話を楽しみながら講義を進めますので、初めての方でも安心してご参加いただけます。玄米に興味のあるあなた様からのご応募をお待ちしています。
