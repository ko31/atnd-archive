もくもく会とは？

もくもく会 とは、カフェとか誰かの家とかに集まって、各自もくもくと勉強したり作業したり本を読んだりする会のことです。

今回は京都駅から徒歩圏内にあるオシャンティーな コワーキングスペースcotoさん と、 太秦 の2カ所で同日開催。たぶん今年はこれが最後の開催になります。次は来年の春くらいになります（京都寒い）。

自主的に進めたいことがあるけれど、自宅では誘惑が多くて集中出来ない、カフェや図書館では長居するのに気が引ける……そんなかた、お気軽にご参加ください。

ジャンルはなんでもOK。PC持ち込み、飲食可。場内WiFi飛んでいて利用可。

参加条件

もくもくと作業するものであれば、ジャンル、商用か非商用かは問いません。趣味活動やお仕事、卒業制作、読書、お裁縫、コミケの準備などなんでもOKです。

なお、周囲に音が漏れる／お喋りする／飲酒しながら／作っているふり…など、もくもくの主旨に反するとこちらが判断した場合は参加をお断りする場合がございます。

参加料金

500円／お一人様1日（当日徴収）
※お水とコーヒー・お茶・紅茶はフリードリンク（コーヒー・お茶・紅茶はhotのみ）でご利用いただけます。
