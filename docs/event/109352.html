概要

FRONTL1NEが運営するBBQイベントです。

FRONTL1NE関係者はもちろん、そうでない方々も是非ご参加ください。

(参加キャンセルは2019/11/01(金)の23:59までにお知らせください)

前回の様子


開催日時

2019年11月03日(日曜日) 小雨天決行

9:00から順次スタート (自由参加・自由解散 | 18:00解散予定)

(大雨などの場合 @FRONTL1NE_NET 及び FRONTL1NE Discordにて開催なのか中止をおしらせします。)


参加費

当日会場で徴収します

  • 過去にFRONTL1NE BBQへ参加して機材費(合計4,000円)を支払っている方 : 4,000円

  • 機材費を合計で4,000円分支払っていない方 : 4,000円(参加費) + 2,000円(機材費) = 6,000円

FRONTL1NE BBQは有志の集まりで運営されているため、機材の費用がいくらかかかってしまうことにご理解&ご協力をよろしくお願いします

個人スポンサーも募集しています!! ページ下部にあるスポンサーの欄をご覧ください


場所

市川市 妙典河川敷 [妙典小学校及び妙典保育園付近]

Map : https://goo.gl/maps/WW5nkPrzJc1pxBda9


アクセス

  • 電車の場合 : 東京メトロ 東西線 妙典 下車後徒歩10分

  • 車の場合 : 会場の公園に駐車場があります (一台 1000円) (最寄IC : 首都高速湾岸線 千鳥町IC)

  • バイク・自転車の場合 : 会場の公園に駐車場に無料で停められます。

(車・バイク・自転車でお越しの方及び未成年者の飲酒は法律で禁止されております、お控えください。)

(また会場近辺への路駐なども近隣の住民のご迷惑となるのでおやめください。)
