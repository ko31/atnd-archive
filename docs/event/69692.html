description

※店頭から５名様お申込みいただいているため定員３名と表示されます。

「自宅でおいしいコーヒーを楽しむのはむずかしい。」

これ、実は大きな誤解です。

「コーヒーの味と香りのちがいが良く分からない、酸味が苦手、飲むと胃もたれ胸焼けがする…」

それにもちゃんと理由があります。

美味しいコーヒーを楽しむのはとっても簡単。
抽出の職人技や、ちがいを感じるための味覚の訓練もいりません。
必要なのはほんの少しの知識と品質の良い素材だけ。
たった２時間で、あなたのコーヒーライフに劇的な変化をもたらします。

苦いだけでも酸っぱいだけでもない、素直に美味しいと思えるコーヒーを飲みながら
楽しく学んでみませんか？

受講料：通常3,240円→移転記念価格2,160円

内容：講義1時間
　　　ペーパードリップ実習0.5時間（1人3回ずつ）
　　　色々なコーヒー飲み比べ0.5時間（6種類）

場所：もとえ珈琲店舗(神奈川県川崎市木月３－１３－２)

持ち物：筆記用具

講師紹介
もとえ珈琲店主　大沢征史（おおさわまさふみ）
1979年生まれ。料理学校卒業後、東京都・神奈川県内のカフェやレストランに勤務。
2005年に堀口珈琲入社。堀口俊英氏に師事。焙煎の責任者を務める。
2013年にもとえ珈琲を開業。
2015年店舗を移転。
