description

［講座のポイント］
　樹脂・ゴム材料を用い工業部品を成型した後、経時的に成型体表面に意図せぬ模様が 発生したり、成型品表面の触感が変化したりすることがある。これらの現象の見分け方・発生原因の特定法と発生メカニズムとについて解説し、合わせて個々の材料や添加剤の種類別に、発生防止策を提案する。

［受講対象者・レベル］
　初心者～中級者向け、プラスチックス・ゴム部品の開発設計・材料選定技術に携わる研究者、樹脂による部品でトラブルを経験し、その未然防止策を習得したい人など

［プログラム］
１．ブリードアウトとブルーミング
　　1-1　成型不良とブリード現象の見分け方
　　1-2　ブリード・ブルーム発生機構の理論：拡散と溶解度
　　1-3　ブリード成分の同定と解析：分離抽出と機器分析
２．樹脂・ゴム材料におけるブリード現象の制御
　　2-1　添加剤の種類と処方
　　2-2　樹脂・ゴム材料におけるブリード・ブルーム防止技術の変遷
３．ブリード現象の制御と活用
　　3-1　ブリード制御による機能発現と商品設計
　　3-2　モルフォロジー（ポリマーアロイ）設計理論の応用によるブリード・ブルーム制御策の提案
　　3-3　Customer　Solution型　対策（個別ポリマー種に関するコンサルテーション）
［質疑応答・名刺交換］
