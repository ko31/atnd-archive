概要

FAG Studio による Unity を使ったゲーム作りのハンズオン勉強会です。
一緒にiPhoneやAndroidアプリ開発を体験してみよう！

「Unityって何？」という方のために少しだけ説明。
Unityを使って開発をすると次の様な良いことがあります！

  • 開発環境がすばらしい。
    1. 初心者でも実際にゲーム画面を見ながら感覚的に開発できる。
    2. Assetsを使って素早く開発できる。
    3. プログラミングすることで細かい処理も実装できる。
  • マルチプラットフォームに対応しています。
    Unityで作ったゲームはボタン1つで色々なデバイス上で動作させることができます。
    1. スマートフォン(iPhone, Android, WindowsPhoneなど)
    2. パソコン(Windows, Mac, Linux)
    3. Web(ブラウザのUnity Web Playerプラグイン)
  • 無料で始められます。
    開発環境は Unityダウンロードページ より無料ダウンロードできます。
    (※より高機能な有料版もありますが無料版でもかなりのことができます。）

まとめると、
「無料でリッチな開発環境を使いながら、スマホ・パソコン・Webブラウザ向けのゲームが作れてしまうツール」
です。
