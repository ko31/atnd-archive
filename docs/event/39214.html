第1部　多層カーボンナノチューブの分散法とその応用
【11:00-12:30】

講師:保土谷化学工業(株)

【講演主旨】
カーボンナノチューブは製造方法によりまったく異なる形状を有する材料です。適用したい分野に合った材料選択をしないと、機能が発現しません。
本講義では、製造法の異なる様々なカーボンナノチューブについて紹介します。また、多層カーボンナノチューブ(MWCNTs)に絞り、各媒体(熱可塑性樹脂、熱硬化性樹脂、有機溶剤、水溶液)中への分散方法を、分散機と分散剤に分けて説明を行います。多層カーボンナノチューブの応用分野としては、熱可塑性導電樹脂分野、電波吸収分野、リチウムイオン電池(LIB)電池分野、強化樹脂分野を中心に紹介します。適用したい分野に合った材料選択と分散方法を解説します

【プログラム】

1.カーボンナノチューブについて
～多層カーボンナノチューブの種類及び形状による特性の違いなど～
　1.1　単層カーボンナノチューブ
　1.2　二層カーボンナノチューブ
　1.3多層カーボンナノチューブ

2.多層カーボンナノチューブの分散方法
～分散機・分散剤による各媒体への分散手法～
　2.1　熱可塑性樹脂
　2.2　熱硬化性樹脂
　2.3　有機溶剤
　2.4　水

3.適用分野とそれに応じたMWCNTsの選別方法・応用例
　3.1　熱可塑性樹脂分野
　3.2　電波吸収分野
　　3.2.1　誘電性電波吸収体
　　3.2.2　抵抗皮膜型電波吸収体
　3.3　リチウムイオン電池(LIB)分野
　　3.3.1　負極膜への添加効果
　　3.3.2　正極膜への添加効果
　3.4　強化樹脂分野

第2部　カーボンナノチューブ技術の基礎、材料技術、分散・凝集、市場課題(仮題)
【13:30-16:00】

講師:(有)スミタ化学技術研究所

【キーワード】
1.カーボンナノチューブ
2.カーボンナノチューブの分散
3. CNTデータ集
4. CNT実用化推進研究会

【講演主旨】
今やカーボンナノチューブは完全な商用化時代に入っており、海外、特にヨーロッパやアジアでは本格的に実用化されている。このような時代を迎えて、多面的な特性を有するCNTの用途開発を如何にして立ち遅れている日本市場に定着させるか、基礎から応用、安全性、普及策まで含めて詳細に説明する。

【プログラム】
1.カーボンナノチューブの魅力
　1-1　CNTの種類
　1-2　CNTの特性
　1-3　CNTの期待される用途

2.カーボンナノチューブ実用化に向けての弊社の取組み
　2-1　液相分散からプロトタイプ開発
　2-2　産官学のネットワーク構築
　2-3　CNT選別のための「CNTデータ集」の作製
　2-4　「カーボンナノチューブ実用化推進研究会(仮称)」の設立

3.界面科学的観点からみたカーボンナノチューブの分散
　3-1　液相分散技術(分散機から分散剤まで)
　3-2　固相分散技術(超臨界炭酸と分散剤)
　3-3　分散の評価

4.応用開発事例(プロトタイプ)
　4-1　透明・静電防止塗料
　4-2　高導電性インク
　4-3　透明導電膜・フィルム
　4-4　面状発熱ペーパー
　4-5　その他

5.カーボンナノチューブの毒性・安全性情報
　5-1　NEDOのCNTリスク評価結果
　5-2　CNT飛散量の測定事例

6.まとめと将来展望
