概要

「enchant.js meetup!」とは「enchant.js でゲーム作りをはじめたいけど、どうしたらいいか分からない」「enchant.js でゲーム作りのスキルを磨きたい！」「enchant PRO ってどんなもの?」という皆さまのためのイベントです。

初心者の方から、すでにゲームをいくつも作ったことがある方まで、enchant.js に興味のある方ならどなたでもお楽しみいただけます。お気軽にお越しください！

応募者多数のため、会場を180人まで収容できるように拡大しました。
当日の飛び入り参加も歓迎。お友達を連れて来ても大丈夫だと思います。

enchant.js について、詳しくはこちらのリンクから。

  • 公式サイト http://enchantjs.com/
  • Developers Blog http://blog.enchantjs.com/
  • github https://github.com/wise9/enchant.js

FAQ

  • 何をするイベントなの?
    みんなで enchant.js についてもっと知ろう！ユーザ同士で仲良くなろう！というイベントです。
  • 電源や回線はあるの?
    人数が増えてしまったため、電源はあまり用意できません。バッテリーを満充電してきてください。机はあります。回線についても人数が多いので十分な数を用意できなくなってしまいました。各自無線ルータの持参を御願いします。
  • enchant.js や JavaScriptの開発経験がなくても大丈夫?
    大丈夫ですが、ちょっとだけ予習しておいた方がよりお楽しみいただけること間違いなし！です。 http://enchantjs.com のサンプルやチュートリアルを参照してください。
  • その場でゲームの開発はできるの?
    今回は 「9leap game programming camp」とは異なり、開発タイムはありません。直接聞きたい質問・ぶつけたいご意見などありましたら、開発チームにお声掛けください。
