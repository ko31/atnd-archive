description

なぜ、「目標」を「結果」に移す事が上手い（上がり目）人とそうでない（下がり目）人がいるのか？
出来ている人もそうでない人にも共通の要素、「共通点」があるそうです。

もっと高いレベルで仕事をこなし、結果を出したい！
「ビジネス書」や「成功本」をいくら読んでも何も変わらない。
「心構え」すら変えられない。
思い通りに「結果」を出せない、「目標」を達成出来ないあなたに伝えたいことがあります。

人間はこうなりたいと心の中で描いていても、思い通りの「目標」はそんなに簡単に達成出来ないものです。

もう、あなたも気づいているではないでしょうか？

実際、本屋で売られている「ビジネス書」や「成功本」を読まれた方の中で、実際何名の方が成功したのでしょうか？

目標を紙に書いただけで達成したでしょうか？

本を読んで成功された方は3％もいないと言われています。

うまくいく人もそうでない人も共通点があるのです。

頭では理解して思い描いていても、具体的に計画したり行動ができない、自分が考えていた通りにならない。

この「本で学んだ知識」というものを「結果」に移すには、少しコツがあるのです。

【車の運転】に例えると分かりやすいかもしれません。
免許を持っていない方が運転の仕方の本を読んで、「さぁ！運転してみて下さい！」と言われてもはたして「安全」に運転出来る方っているでしょうか？
おそらくほとんどの方は出来ないはずです。

目標を確実に達成するだとか、人生で成功する為には本やセミナーだけで「知識」を増やすだけでは、ほとんどの方は達成出来ません。
知識を行動に、目標を達成する為に行動に移す為のトレーニングやコツが必要なんです。

成功者はそのコツを知り、実践してきました！

今回はその成功者が実践してきたように、講師から成功の秘訣を学びながら行動計画を立てるセミナーです。

皆様の人生を変えるきっかけにお役に立て頂ければ幸いです。

前回東京セミナーは満員御礼でした！
多数のお申込を頂き、ありがとうございます。

講師紹介

有限会社 ポシビリティー・アンリミテッド　
代表取締役社長　早崎　郁之 氏
1967年生まれ
大学時代は英米文学を学び、オックスフォード大学ワッタム・カレッジに短期留学する。
社会人となってから、まずオーストラリアの高校で日本語教師をつとめた後、帰国し、親が経営していた150人近く入る飲食店の店長を、30歳までつとめ売上を驚異的にアップさせる。
多くの方を成功に導くため、30歳頃より十数年の時間と3,000万円以上の経費をかけて自己投資し、国内のみでなくアメリカなど海外でもセミナーを受講する。世界30カ国近くを旅し、特にユダヤ人の国イスラエルには3回行き、ユダヤ人のお宅を訪問した経験がある。世界最高峰のありとあらゆるプログラム、セミナー、コーチング、カウンセリングを学び、人生成功の原理原則、哲学を学ぶ。
1999年　ナポレオンヒルの代理店として活躍する。
2005年～2007年　ブライアントレーシーの販売代理店として3年連続日本一となった。
※ブライアントレーシー：世界優良企業FORTUNE Global 500のうち210社が導入している業績アップの教育プログラム
現在は、SMI代理店として活躍、精神面、経済面、教養面、健康面、家庭生活面、社会生活面一つ一つに目標設定を行い、トータルパースンを目指している。最愛の妻と横浜の高級マンションに住む。
