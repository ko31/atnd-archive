description

弊社は国分寺に事務所を構え、 20年以上地域に密着した不動産の仲介を行っている会社です。

この度、地域にお住いのお子様がいらっしゃる世帯を対象に、 無料のセミナーを7月23日（日）に開催することになりました。

子育ての世代の学習環境を調査・研究し、 「頭のよい子が育つ家」を執筆した 著者：四十万先生の日本各地で大人気のセミナーです。

少子高齢化が進む日本を将来担っていく、 子供たちにとって良い教育環境を整え、 21世紀の国際社会で活躍出来る若者の育成していきたいと、 著者の四十万氏の考えに共感し、 エステート白馬は四十万氏と業務提携を結ぶことになりました。

国分寺をはじめ小金井や府中・小平など近隣にお住いの 子育て世帯の方に地域貢献を考えております

その一環としてまず7月にセミナーを開催しますので、 ご興味あるご家庭がありましたら、 是非、お声がけ頂けますと幸いです。

将来的には国分寺周辺地域の教育環境が向上し、 この地域に暮らしたいといった子育て世帯が増え、 人口・児童数の増加により地域が活性化していく ことを望んでおります。

また、このような施策にご協力頂ける幼稚園や企業・団体の方 いらっしゃれば、是非、ご協力お願い致します。
