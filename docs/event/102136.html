先日も参加いただいた方にご好評を頂いた、発達障害のあるお子様も参加できるプログラミング体験会ですが、引き続き12月に2回開催させていただきます。

12月1日（土）と12月15日（土）のそれぞれ13時30分～、先着5名の方に無料で体験いただける内容となっています。発達障害のあるお子様もぜひ保護者の方とご一緒にお越しくださいませ。

【イベント詳細】

今回は2つのプログラムを用意しております。1つはタミヤ社のロボットを使ったもので、こちらは初心者向けでパソコンなどを使わなくてもできるプログラミング教育ロボットです。

2つ目はパソコンを使ったプログラミングです。簡単なコードでキャラクターを動かし、ゴールを目指すというプログラミング教育ソフトを利用します。

どちらでもお好きな方をお選びいただけますが、パソコンに馴染みがないお子様にはロボットプログラミングをおすすめいたします。ぜひ発達障害のあるなしかかわらず、体験会にお越しくださいませ。


  • 日時：12月1日（土）12月15日（土）13時30分～（開場：13時15分、終了14時30分）

  • 場所：就労支援事業所ディープレイス（大阪市北区本庄東2-2-30ウイッシュ大阪ビル9階）
    ＊天神橋筋六丁目駅　徒歩5分

  • 定員：各日5名

  • 料金：無料

  • 持ち物：筆記用具（アンケートにご協力お願いします）
