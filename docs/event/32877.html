上司が採用したくなる企画・プレゼンテーション

イベント ユーザー
event_id 32877
title 上司が採用したくなる企画・プレゼンテーション
catch 上司に報告、提案する際に、論点がまとまった明確で、シンプルな企画・プレゼンテーションができるため、仕事の成果が向上する！
description

【講座のポイント】
上司が採用する企画・プレゼンテーションを作成のポイントを理解します。　そして、実際に企画書作成テンプレートをベースに、企画書を作成します。（自身のテーマで、骨子を策定します。）　さらに、上司が採用するプレゼンテーションを、「キーマンモデル」、「シナリオマップ」をベースに策定し、行動心理学のエッセンスを取り入れたプレゼンテーションノウハウを習得します。

【習得できる知識】
１．ロジカルシンキングを実践するポイントが修得できる。
２．上司の意思決定のプロセスを理解し、適切な対処方法を修得する。
３．わかりやすさの本質を理解・実践できる。
４．自分しか知らない情報を的確に表現・伝達できる。
５．社会・会社・組織の状況・競争・戦略が鳥瞰できる。
６．論理的な企画・プレゼンテーション資料が作成できる。
７．上司が採用したくなるプレゼンテーションができる。

【プログラム】

１．ロジカルシンキングのポイント
　　1-1.論理的提案演習（論理構成とわかりやすさ）

２．行動心理と受け入れのメカニズム
　　2-1.人は「必要なもの・欲しいもの・好きなもの」を受け入れる
　　2-2.上司の価値観・特性の分析（キーマンモデル）
　　2-3.報告・相談・連絡（上司に報告する時のポイント）
　　2-4.会社・組織の「ベネフィット」の「見える化」

３．ロジカルコミュニケーション演習
　　3-1.自分しか知らない情報を表現する
　　3-2.適切な企画書を作成するポイント
　　3-3.論理的な表現と質問力強化演習

４．上司が採用したくなる企画・プレゼンテーション
　　4-1.付箋を活用した論理の見える化（カード型ブレーンストーミング）
　　4-2.問題点の「原因・影響の見える化」
　　4-3.論理的なシナリオ構成作成演習（シナリオマップ）
　　4-4.プレゼンテーションの実施と採用へのポイント
