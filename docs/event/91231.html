description

EC市場は右肩上がりに拡大を続ける一方、悪意を持ったユーザによる不正利用や取り引きで生じるトラブルなどEC事業者を標的としたトラブルが数多く報告されるようになってきました。
このような状況の中、EC事業者は安心安全に事業に取り組む為の環境構築が急務となっています。
今回のカンファレンスでは、ネットショップ、ECサイトを運営する方を対象に、安心安全なEC事業運営のための被害事例や対策事例をご紹介します。

EC安心安全カンファレンス2017詳細

  • 日時：2017年10月3日　14時から16時15分（開場　13時30分）
  • 場所： トスラブ山王 （東京都港区赤坂２丁目５−６　2階）
  • 主催：一般社団法人セーファーインターネット協会
  • 参加費用：無料（事前申し込み制）
  • 主な対象者：ネットショップ、EC企業にお勤めの方
  • お問い合わせ先：一般社団法人セーファーインターネット協会
    　　　　　　　 メールアドレス：ec_anshin@saferinternet.or.jp
  • お申し込みは こちら から

本カンファレンスの参加者は、EC事業者様に限らせていただきます。何卒、ご了承ください。
