第13回のワークショップですが下記の様に開催したいと思います。

アンドロイドマッシュ！ #1 キックオフミーティング

眠っているアイデア、そろそろ一緒に形にしてみませんか？
これまでもくもくと各人で勉強・交流という感じでやって来ましたが…

  • 頭にあるぼんやりとしたアイデアを形にしてみたいけど、
    　技術に自信がなくてそれとなくしょんぼりしていませんか？
  • 技術はあるけど、いいアプリの案が浮かばず日々苦々しい思いをしていませんか？
  • 自分で創作した絵や音楽を使ってなにかアプリを作りたいけど、
    　どこから始めていいかわからなくてぐぬぬぬぬなどはありませんか？

そこで、今回のワークショップに来てくださった方々全員で…

1. 作りたいアプリの案を出し（アイデアソン）、
2. 相性のいい人とユニット（3～4人）を組み、
3. 定期的にミーティングを行ない、
4. ３ヶ月かけて完成を目指してみましょう。
5. そして最後に全員でコードレビューを行います。

このキックオフミーティングではユニットの編成までしたいと思います。
