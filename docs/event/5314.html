title 越後長岡珍スポット巡り
catch 第一回　『居酒屋　夕暮れ』～『コスプレ居酒屋　瑠夢』
description

ご要望があったので長岡の珍スポットを梯子する飲み会を企画してみた。

まず一軒目は、ご存知持ち込み自由なってゆうか持ち込まないとつまみがピーナッツしかない『居酒屋　夕暮れ』

まあなんか適当にみんな持ち寄って飲みにいこうよ。

事前に駅に集合してつまみ買って行ってもいいし、隣の焼き鳥屋で買ってもいいし。

お風呂に入りたい人は入浴の準備をしてくること。
てか、誰か入れ。

その方が面白いから。

でもこの店、ちゃんと予約できるかわかんないからもし予約取れなかったら勘弁ね。

そんときゃ代わりに天風行ってカエルとかスズメとか喰えばいいと思う。

二軒目は『コスプレ居酒屋　瑠夢』

まあコスプレ居酒屋ぐらいなら全国的には珍スポットの範疇に入らないかも知れないが、リクエストされちゃったからにはしょうがない。

長岡じゃあ十分珍スポットだしね。

こちらは一応予約していくつもり。

一応予定では18時30分～19時30分『夕暮れ』→20時～21時45分ぐらい『瑠夢』こんな感じかな？

あくまでも現段階の予定なので、仕事上都合等、もっと開始時間早くしてくれとか遅くしてくれとかあったらご意見聞かせてね。

まあとりあえず関原クラスタがちゃんと22時のバスに乗って帰れるような感じでは行きますが。

そんなわけでみなさんよろしこ！
