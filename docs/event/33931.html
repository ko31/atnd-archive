title Android 勉強会（入門編）
catch Eclipseを使用したAndroid用Javaアプリ開発の第一歩をサポートします。
description

Android開発者が参加者をサポートします。

参加費：

  • 無料

前提スキル：

  • プログラミングの経験のある方。（JAVAでなくてもOK,独学可）

参加者が持参する物：

  • 名刺（無くても可）
  • ノートPC（JDKとEclipse、AndroidSDK［Android 2.2 API Level 8］をインストールしてきてください）
  • もっている方はアンドロイド端末
    ※１、希望者にはノートPC貸出（数台）
    ※２、Eclipse、AndroidSDKのインストールが難しい場合は、当日午前中に別途時間を取ってサポートしますので、下記の問い合わせ用アドレスまでメールにてご連絡ください（11/15必着）。
    その他、開発環境のインストールで不明点がある場合もメールにてお問い合わせください。
    前回までの勉強会で数名の方がインストールを完了出来てませんでした。

開催条件：

  • 最低3人以上参加が在る場合開催します。
    参加者が3人未満の場合は中止します。

予定：

  1. Eclipseの基本操作方法説明
  2. プロジェクトの作成
  3. レイアウトエディタを用いた画面作成
  4. ボタンアクションの作成
  5. 画面遷移
  6. じゃんけんアプリの作成
  7. ソースコードを見ながらの技術紹介
    • タッチセンサーを用いたマルチタッチの実装
    • アプリケーション公開時に行う、難読化の注意点

★入館の都合がありますので、参加希望の方は下記アドレスまでメールを下さい。
　また、開始時間を過ぎてからの入場は原則的に行えませんのでご注意ください。
