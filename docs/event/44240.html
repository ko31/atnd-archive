title 生命保険の基礎セミナー
catch 無駄・漏れのない賢い保険の入り方
description

あなたは自分が入っている生命保険の内容を理解していますか?
多くの方がご自身の加入している保険の内容を理解していない現状に出くわします。

生命保険の効力が発揮されるのは病気やケガ・死亡など人生におけるマイナスの時期。
きちんと理解していないと“万一の時役に立たない”事態にもなりかねません。

『保険はよくわからない』『専門用語が多く難しい』という声をよく聞きますが、基本を押さえ、
シンプルに考えればけっして難しくありませんよ。

セミナーでは生命保険を基本からわかり易く説明します。
必要な箇所はきちんと押さえ、無駄なく賢く保険に入りましょう!

[セミナーの内容]
セミナーではみなさんが抱いている生命保険の疑問やお悩みを解決します。

・生命保険の仕組みを知りたい
・どんな保障があるの?
・どうして病歴を申告する必要があるの?
・掛捨型と貯蓄型どちらがいいの?
・パンフレットを見る時のポイントは?
・合理的で無駄のない保険の入り方は?
・主契約と特約ってなに?
・保険の更新ってなに?
など

『生命保険は住宅に次ぐ高価な買い物』と言われています。
総支払保険料が数百万円になることも珍しくありません。
高額商品の割に安易に加入していませんか?

講師は銀行・保険会社・保険代理店で働いてきたファイナンシャルプランナー。
長年金融業界で働く中で『金融知識があるかないかで大きな差が出る』ことを体感してきました。

生命保険も重要な金融知識の一つ。
基本を学んでおくと、きっと今後の人生のお役にたちますよ!

セミナーは少人数制。参加者のみなさんと雑談をしながらすすめます。
金融商品の勧誘・販売は一切行いません。お気軽にご参加ください。

【無料個別相談】
セミナー終了後、希望者のみ個別相談を実施しています。(後日予約制)
・今入っている保険の内容を詳しく知りたい。
・ライフプランシュミレーションをしてほしい。
・自分に必要な保障は何か知りたい。
などのお悩みにお答えします。

席に限りがありますので、お早めにお申し込みください。
多くの方にお会いできることを楽しみにしています。
