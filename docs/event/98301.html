description

本セミナーでは、ベトナムオフショアを検討している、もしくは既にオフショア開発に着手しているがマネジメント上の悩みがあってそれを解決したいという方に、実際にすでにオフショア開発の発注を４年間行っている発注者様の「生の声」をお伝えして行きます。

イントロダクションでは、オフショア開発を行う際に陥り易い「間違い」についてまとめてお話をしていきます。その後、実際の発注者様をゲストにお呼びして、発注者が最初にどんな悩みを感じたか、それをどうやって解消していったのか？について赤裸裸にお話をして頂きます。


＊＊＊＼＼こんなお悩みを抱えている企業様にオススメ／／＊＊＊
★ベトナムのオフショア開発を検討しているが不安もある
★ エンジニアの月単価が上がり、想定コスト内での開発が難しい
★エンジニア不足により必要な人材を確保できず、機会損失がある
★新しいオフショア開発先について情報収集をしたい
＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊

16:00 第一部（登壇者：株式会社Grrow グローバル事業推進室　統括責任者　三上健太郎）
———————————————————
『ベトナムオフショア開発をはじめようと思った時にぶつかる５つの壁』
———————————————————
・ベトナムでのオフショア開発は、比較的他国と比べればやり易いと言われています。
・しかし、そこに甘んじてしまうと大きな問題が発生してしまうこともあります。
・ここでは、オフショア開発でぶつかってしまい易い「壁」を５つにまとめました。
・そのそれぞれの「壁」の解決方法についてお話をしていきます。

———————————————————
『ベトナムオフショア開発が上手くいったのはこんな企業』
———————————————————
・ベトナムオフショア開発が上手くいっている企業様の共通点は幾つかあります。
・ここでは、それらの共通点に関しまとめ、お伝えしていきます。
・これからベトナムオフショア開発をはじめられる場合には、自社と異なる点について確認してみてください。

———————————————————
『受託開発とラボって良く聞くけど、どっちがいいの？』
———————————————————
・ベトナムオフショア開発では、「受託開発」と「ラボ」という言葉を聞きます。
・そのそれぞれの言葉の意味、特長等についてまとめました。
・また、実際に御社がどのような開発を志向しているかで「受託」なのか「ラボ」なのかは決まります。
・それらを上手く判断できる基準を簡単にお伝えしていきます。
