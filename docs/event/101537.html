< セミナーについて一言 >
コンテンツマーケティングはWEBを通してユーザーと対話をすることです！
サイト観覧者の「知りたい」「やりたい」ことをコンテンツとして発信することが成功に繋がります。

< 経歴 >
●株式会社エイチームでは引越し侍SEO責任者としてコンテンツマーケティングに従事。

●その後リッチメディアに入社。
ライターチームへのSEOライティングの指導、記事作成オペレーションの構築、リライトによるセッション回復に従事。

●2018年にCROCOにジョイン。
マーケティング部門にて社外のSEO講師、SEOコンサルタント、SEO用ツール企画、サイト設計などに携わる。

< 実績 >
「引越し」28位→2位「引越し　見積もり」8位→1位「引越し費用」5位→1位「引越し準備」83位→1位「引越し　単身」27位→1位。

< ワークショップについて一言 >
サイトの規模や部署の人数に関係なく、明日からすぐに実践できるノウハウを出し惜しみせずにご紹介いたします！

< 経歴 >
●日本大学経済学部を卒業後、約10年間に渡り証券業界にてセールス・運用・運用コンサルに携わる。

●その後IT業界へ転身。
不動産系の比較サイトを運営する企業で、企画・マーケティング担当としてサービスのWEBマーケティング戦略の全体設計・制作ディレクション・構築後の改善運用・セールスコ―チング・ブランディングまで多岐にわたる業務を行う。

●2012年にサムライファクトリーに入社。
事業本部長として、複数のサービス開発・組織づくりに従事する一方、コンテンツマーケティングの支援・推進だけにとどまらず、クライアントのチームビルドも実現するなどコンサルティング業務も牽引。

●2015年にCROCO株式会社へ参画し取締役COOに就任。

この機会に、いまのやり方を見直すヒントになれば幸いです。

< 経歴 >
●2008年より、10年間「お名前.com レンタルサーバー」のマーケティングとプロダクト戦略に携わる。
