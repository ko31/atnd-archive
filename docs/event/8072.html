秋の a-blog cms 合宿が開催されます！

春の合宿では、宮城・東京・静岡・愛知・岐阜・兵庫・福岡と各地からの人が20名集まりました。前回の模様は 日本各地から集まった a-blog cms Power User Training Camp 2010 Spring が終了しました をご覧下さい。

今回の夕食は 天然トラフグコース です。が、懇親会のための合宿ではありません。真面目なスキルアップのための合宿となっております。

費用

  • 1泊夕朝食付　20,000円　通常
  • 1泊朝食のみ　8,000円　金曜の夜から参加する方

※領収書を発行致します。
※キャンセルは4日前(11月15日)までであれば無料です。

会場

  • 知多半島最南端 師崎荘（今回は無線LANが、しっかり使えるようになるかも！）

その他

  • 遠方からの参加の方についての移動手段は別途ご相談させて頂きます。
  • 決定事項については随時追記していきます。

遠方から愛知入りされるの参加者（飛行機・新幹線）中から抽選で1名様に a-blog cms のベースライセンスをプレゼントさせて頂く事になりました。
