第1部　塗料に適した紫外線吸収剤・HALSの上手な使い方
【11:00-12:30】

講師:クラリアントジャパン(株)　添加剤ビジネスユニット ポリマー添加剤チーム　チームマネージャー　田中　浩二　氏
講師のホームページ

【キーワード】
1. 紫外線吸収剤
2. HALS

【講座のポイント】
1. 紫外線吸収剤の基本的な作用機構による添加時の留意点を解説する
2. 状況に応じた紫外線吸収剤、HALSの選定

【講演主旨】
屋外で使用される塗料は、太陽光に含まれる紫外線によってダメージを受け、その光沢を失い、時には下地材料の保護膜としての機能も失ってしまう。この塗料を紫外線のダメージより保護するために添加されるのが、紫外線吸収剤とHALS(光安定剤)である。現在自動車塗料が、10年以上に渡って、その光沢を保持し続けられるのは、もちろん塗料メーカーの材料自体の開発、改良と共に、これらの添加剤なくしては、そのような耐久性は得られない。
今回、それぞれの特性を紹介すると共に、配合設計における留意点を説明する。

【プログラム】
1.塗膜の劣化機構
　1-1　高分子材料の劣化機構
　1-2　塗膜の光劣化

2.安定剤の機能別分類
　2-1　安定剤の機能別分類
　2-2　高分子材料の自動酸化機構とそれを防止する添加剤

3.紫外線吸収剤の作用機構と種類
　3-1　紫外線吸収剤の作用機構
　3-2　主な紫外線吸収剤の特性
　3-3　主な紫外線吸収剤の吸光カーブ

4.光安定剤(HALS)の作用機構
　4-1　HALSの基本構造と機能
　4-2　HALSの作用機構

5.耐光安定剤、配合設計時の留意点
　5-1　紫外線吸収剤とHALSの特性
　5-2　配合設計時の留意点(耐熱性、相溶性、他の添加剤との相互作用、エナメル塗膜の配合設計、ラジカル重合塗膜へのUVA、HALSの影響)

6.まとめ

2.濡れに関する問題

　2.2　基材湿潤と、液体と基材の表面張力
　2.3　基材湿潤剤の構造と働き
　2.4　静的表面張力と動的表面張力;測定方法と添加剤の種類による差
　2.5　添加剤の種類と選択
　2.6　使用事例と効用;スプレー塗装での微粒化とハジキ防止効果

3.UV・EB処方向け有機変性シロキサン添加剤
　3.1　UV・EB硬化型塗料・インキについて
　3.2　シリコン添加剤に求められる特性
　3.3　反応硬化型シリコン添加剤について;特性・化学構造
　3.4　TEGO反応型製品群の設計・諸特性;
