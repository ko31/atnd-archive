description

太陽光発電所や工場、商業ビル等の電気設備の保安監督業務で
近年注目されている「三種電気主任技術者」試験合格対策セミナー

太陽光発電所をはじめとした自然エネルギー業界や、
工場や商業ビルの省エネなど、業界全体が成長しているため、
この業界でのスキルアップ・キャリアアップ・新規事業参入にご興味をお持ちの企業様も多いのではないでしょうか。

これまで専門誌「環境ビジネス」の読者ユーザーからの開催要望リクエストにより開催実現にいたりました。

※今回は、過去実施（講師:石原鉄郎先生）の生講義をビデオ収録したものを使用したビデオ講義になります。
　生講義ではございませんのでご注意ください。配布資料は同じものを参加者皆様に無料でお配りします。
　また質疑応答も、会場ではアンケート形式で集め、後日返答という形で実施いたしますので、好評頂いております。

　当セミナーは資格合格を保証するものではございませんので、予めご了承ください。

[このような方を対象にしたセミナーです]

これから成長業界『環境分野』でのスキルアップ・キャリアアップに
ご興味のある企業様や個人の方（業界未経験者歓迎）

[セミナー内容]

1. 三種電気主任技術者はこうやって対策すれば合格できる、その具体策
2. 三種電気主任技術者でできる環境エネルギーの仕事
3. 三種電気主任技術者をとるとどういう新規事業参入、スキルアップ、キャリアアップできるのか
