GREE Tech Talk #10

テーマ

GREE Tech Talk 10回目のテーマは「データ可視化」です。
今回は、データ可視化についてみなさまと一緒に学びを深めたいと思います。

GREE Tech Talkとは

進化し続けるインターネット技術を、特定のジャンルにとらわれることなく追求していくために、グリーが年2回開催する技術勉強会です。

最前線で活躍する国内外の技術者をお迎えし、毎回異なるテーマについて、みなさまと一緒に考え、議論する場を提供しています。
