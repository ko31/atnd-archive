関東や関西では良く目にする VAPE オフ や VAPE イベント 。
北海道ではなかなか見掛けませんよね。

北海道には VAPE の文化がまだまだ浸透していないこともあり、ユーザー間の交流が
中々とれずにモチベーションを保てない方も少なくないのでは？と常々思っています。

そんな中、自ら勧めた友人や、SNS を通してお知り合いになれたユーザーさんと

これを書いている主催者自身は（ @ns_p_v ）結構なお歳を召されておりますので(笑)、どちらかと言うと、クラブ系やファッション・アパレル系の流れからくる VAPE ではなく、あくまで同じ趣味趣向を持った同士の語らいの場になればと思っています。
