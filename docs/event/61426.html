ヒカ☆ラボ一覧は こちら！！

★講演内容

WEBアプリケーションの負荷試験のやり方、勘所からアプリケーションのプロファイリング方法を
数千万ユーザが利用するサイトを開発・運用してきた強者がお話します！！

※内容は変更になる可能性がございます。

★講演者プロフィール

■ 株式会社ゆめみ

デジタルマーケティング領域において、企業とお客様とのタッチポイントをオムニチャネルでデザインし、
価値ある顧客体験を新たに創造する企業です。
最近ではファーストフード、ファストファッションなど大手ナショナルクライアントの
モバイルサービスを一手に引き受け、大規模ユーザのシステム構築・運用経験が豊富です。

■仲川　樽八氏

2000年頃から主にLAMP構成のWeb系アプリケーションの開発を行っており、
近年はクラウドを前提としたシステムを中心に高負荷アプリケーションの設計・開発を手がけている。
数千万ユーザの利用を前提とした大規模負荷試験を異なる複数のシステムで実施するという貴重な経験をしている。
