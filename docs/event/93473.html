株式会社サポータス（東京都千代田区飯田橋3-11-13 FORECAST飯田橋9F　以下サポータス）は、2018年1月11日（木）に株式会社U−NEXUS（長野県長野市若里4丁目8番40号）主催による「 iPad と FileMaker を使った業務の自動化・効率化で業務時間を大幅に削減するセミナー」（無料）を開催いたします。
働き方改革、業務改革、残業削減、待ったなし!
社内の改革が進まないとお悩みの経営者の方へ

iOSやAppleが分からなくてもOK!
社内システムがWindowsでもOK!

⻑野県内初の Apple のコンサルティングパートナーおよび FileMaker の開発パートナーである U-NEXUS様を講師にお招きし、独自性に溢れる注目企業(U-NEXUS様のお客様)のIT先進事例をご紹介する耳寄りのセミナーをサポータスで開催いたします。
iPad を使った業務改革や業務の標準化、スタッフ間の情報共有、業務の徹底、物の管理、ミスやクレームの対策、在庫管理や資材管理などに悩む方々にオススメの内容となっています。
また、業務アプリの開発(初心者OK!) や FileMaker を使ったシステムの内製化、さらには基幹システムのカスタマイズにお悩みのお客様にもオススメの内容となっています。
iPad と FileMaker を使ったシンプルかつ優れたビジネスソリューションで仕事のやり方を変えたノウハウにぜひ触れてください。
日本経済新聞やマイナビなどにも取り上げられた事例をご紹介します。ご期待ください!

【セミナーについてのお願い】
　本セミナーは以下の方を対象としたセミナーになります。
　　経営者（経営幹部）、IT担当者
