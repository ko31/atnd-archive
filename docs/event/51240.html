description

　NTTドコモなど携帯３社のeco検定対策の公式サイトの運営や各種環境ビジネスセミナーなどで環境教育に取り組んでいる「環境教育フォーラム21」は第16回eco検定合格に向けて、6月16日大阪市など全国各地で「重要ポイント集中講座」を開催します。
　地球温暖化、循環型社会の構築、エネルギー問題、環境と経営などeco検定にでやすい問題を2月に発売された公式テキスト第4版に沿ってテーマ別にわかりやすく講義します。eco検定で増える傾向にある時事問題もテーマにあわせて解説します。
　当フォーラムのeco検定講座は講義内容が初心者にもわかりやすく、ご自宅での学習用にご用意した重要ポイント暗記ノート、重要ニュース集など豊富な教材も大変、役だったと好評です。学習法やセミナー参加者のご意見などはホームページをご参照ください。
ホームーページは　http://ecoken.newsmedia.jp/
