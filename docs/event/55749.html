title 狙われる企業の機密情報
catch ～社内の不正から守るための備えとは～
description

本セミナーでは、最近の重大な情報漏えい事案を含む、多くの企業不祥事の解決に携わってきた塩崎彰久弁護士（長島・大野・常松法律事務所）が、こうした情報流出事案に関する組織の備えや初動対応などの最新実務について、実例を交えてわかりやすく解説いたします。
また、株式会社エアーからは情報漏えいを抑止防止するための具体的なソリューションについて、株式会社UBICからは情報漏えい検知時の初動対応と予兆把握のための仕組みについてご紹介をいたします。
パネルディスカッションでは、最近の情報漏えい事件の傾向や危機管理実務と共に、平時の情報漏えい対策について議論をする予定です。
