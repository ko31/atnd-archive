第13回Android勉強会in札幌

イベント ユーザー
event_id 29474
title 第13回Android勉強会in札幌
catch 今回は電子工作祭り!
description

今回は、Androidと電子工作をテーマに、ADKを使ったArduino連携や、その応用(戦車のラジコン制御)についてのセッションをまとめてみました。
また、神戸支部(「Android ADK プログラミング&電子工作バイブル」の共同著者の一人である吉田さん)と、北海道のArduino勉強会とのタイアップ企画となっています。

今回からGoogle+のHangoutsを使ったネット越しでの講演や、それらの放送を別のエリアで集まって視聴するサテライト会場も設ける予定となっています。
今のところ、北海道繋がりということで旭川にサテライト会場を設ける予定ですが、会場とネットワーク設備とWebカメラだけ揃えて頂けるなら、他のサテライト会場と繋ぐことも可能です。
特に道内に限定しませんので、興味ある方は連絡下さい。
(Hangoutsは参加できるのが10人までなので、繋げるのは会場間だけとし、個人間は考えていません)
