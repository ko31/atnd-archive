筑波ゲーム作るよ！

イベント ユーザー
event_id 4465
title 筑波ゲーム作るよ！
catch 「筑波大学関係者だけで同人ゲームを作る企画（仮）」の説明をする会
description

主催の『筑波大生による筑波大生のための同人ゲームを作ろうよ！』という思い付きから、
出来るだけ多くの学生に参加していただけることを第一に考えたゲームの企画を作ってみました。

制作に必要な技術や経験のある方は勿論ですが、
今までゲーム制作に関わる機会のなかった方にも是非協力していただきたいと考えています。

今回は、そんな企画の内容や、募集したいメンバーについての説明、顔合わせなんかをするための集まりです。
（人員募集については後ほどサークルHPにも掲載する予定です。この集まりに来なければ制作に参加できないということではありません。）

■内容
・はじめに　…同人活動って？
・ゲーム内容（企画書という名のなぐり書きをお配りします。）
・今後の予定
・募集したいこと等について
・雑談とか相談とか

■参加条件
・筑波大学に在学中～の方で、
　同人ゲーム及び当企画に興味のある方。
　実際に制作に参加する・しないに関わらず、お気軽に。
　(飛び入りOKですが、参加登録しない場合は椅子とかコップがないかもしれません）

■サークルメンバー
　*制作の進行の上でコアになるメンバーです。あれこれまとめたり最終決定したりします。

　　ぽち。（http://twitter.com/lizhi3）：言いだしっぺ、キャラクター原画、各種デザイン関係

　りだい　（http://twitter.com/s0711470）：システム関係

　　AknEp　（http://twitter.com/AknEp）：WEB、広報関係
　　

■注意■
　当企画で制作するのは、あくまで「同人ゲーム」です。

　商業用のゲームとは違い、採算は度外視（収益を得るために作るものではない）です。
