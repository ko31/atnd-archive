2015/7/24 追記：

会場変更となりました！　
東陽町駅近くの「江東区文化センター」です。ご注意下さい。

これまでのあらすじ

去る5月に行われた青空文庫アイデアソン
「Code for 青空文庫」アイデアソン #1」 https://atnd.org/events/66230
にて、

エンジニア向け啓蒙イベントや工作員向け入力イベントを行おうという提案がありました。
「イベント開催計画」

それを受けて、第一回目のイベントとして「シャッカソン」（写経：入力イベント）を開催します。

入力だけではなく、今後青空文庫に関わっていきたいエンジニアなど興味ある方と今後のお話もできればと思います。

開催概要

  • 日時： 2015/7/25(土) 13:00-17:00
  • 会場：江東区文化センター　第一和室
  • 主催：青空文学部
    • 「青空文学部」とは、青空文庫のイベントなどを自主的に行うためのカンムリです。
  • 共催：秘密結社オープンフォース
    • 会場の手配をこの団体名義で行っています。
  • 費用：会場費￥3100を人数割
  • 内容：「底本」を１冊決めて、当日分担して入力、校正を行います。
    当日に作業が完了しなかった場合は、持ち帰っての作業となります。

底本

現在、選定中です。

  • ご希望、ご提案などがあれば、コメントなどでお教えください。
  • 当日、申請の流れも見せる目的で、「入力受付システム」 http://reception.aozora.gr.jp/accept.php
    から「第１回シャッカソンメンバー」という名義で底本登録します。

プログラム

13:00-13:30 オープニング
13:30-14:00 工作員マニュアルの説明、底本登録
14:00-15:00 チームを作って入力・校正作業 #1
15:00-15:15 休憩
15:15-16:30 入力・校正作業 #2
16:30-17:00 まとめ、今後について、クロージング

ご参加される方へ

  • 「青空文庫工作員マニュアル」をあらかじめ目を通しておいてください。
    http://www.aozora.gr.jp/aozora-manual/
  • 当日はPCをお持ち下さい

アクセス

2.都営新宿線東大島駅より　門21系統「東大島駅」⇔「門前仲町」 1.2.とも「江東区役所」下車徒歩3分
3.「亀戸駅通り」より　都07系統「錦糸町駅」⇔「門前仲町」「東陽町駅」下車徒歩5分
