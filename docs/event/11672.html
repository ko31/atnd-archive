catch ～Rubyで構築されたＣＴＣリモートオペレーションセンターとその運用サービスの全貌～
description

平素は格別のご高配を賜り誠にありがとうございます。

最近、話題の「エンタープライズRuby」ですが、この度、Rubyの開発者である まつもと ゆきひろ氏をお招きし、「エンタープライズRuby」の現状と将来についてご講演いただきます。
またＣＴＣではこの度、2011年1月中旬に自社リモートオペレーションセンター（ROC）を開設予定ですが、ROCの機能を構築するにあたり約80%近くをRubyで自社開発いたしました。今回はその開発に何故Rubyを採用したのか、その経緯や開発プロセスでの苦労話など、今回のプロジェクトに携わった開発担当メンバーがお話しさせていただきます。

さらに、この最新のROCから提供される仮想化インフラ運用のシステムマネジメントサービスやミドルウェアのシステムマネジメントサービスなど、お客様の運用業務効率向上とコスト削減を実現するＣＴＣの運用サービスを実例をまじえながらご紹介いたします。

ご多忙の折とは存じますが、皆さまのご参加を心よりお待ち申し上げます。
