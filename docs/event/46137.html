《JDK（インストールされていない場合）》

以下のリンクよりJDKをダウンロードし、インストールします

《Scalaインストール手順（使用するバージョンは2.10.3）》

Windows

1. 以下のリンクからScalaをダウンロードし、インストールします
http://www.scala-lang.org/files/archive/scala-2.10.3.msi
2. コマンドプロンプトでscalaを起動し、プロンプトが表示されればインストール完了です

C:\> scala
Welcome to Scala version 2.10.3 (Java HotSpot(TM) Client VM, Java 1.7.0_45).
Type in expressions to have them evaluated.
Type :help for more information.

scala >

Mac

1. homebrewでインストールします

$ brew install scala

2. ターミナルでscalaを起動し、プロンプトが表示されればインストール完了です

$ scala
Welcome to Scala version 2.10.3 (Java HotSpot(TM) Client VM, Java 1.7.0_45).
Type in expressions to have them evaluated.
Type :help for more information.

scala >

マニュアルインストールの場合

1. 以下のリンクからscalaをダウンロードし、任意のディレクトリに展開します
http://www.scala-lang.org/files/archive/scala-2.10.3.tgz

2. binディレクトリへ移動し、起動確認します

$ cd /path/to/extracted/dir/bin/
$ ./scala
Welcome to Scala version 2.10.3 (Java HotSpot(TM) Client VM, Java 1.7.0_45).
Type in expressions to have them evaluated.
Type :help for more information.

scala >

《sbtインストール手順（使用するバージョンは0.13）》

Windows

1. 以下のリンクからsbtをダウンロードし、インストールします
http://repo.scala-sbt.org/scalasbt/sbt-native-packages/org/scala-sbt/sbt/0.13.0/sbt.msi
2. コマンドプロンプトでsbtを起動し、プロンプトが表示されればインストール完了です

C:\> sbt
...
[info] Set current project to ...
>

Mac

1. homebrewでインストールします

$ brew update
$ brew install sbt

2. ターミナルを起動し、インストールされたバージョンを確認

$ sbt --version
sbt launcher version 0.13.0

3. sbtを起動し、プロンプトが表示されればインストール完了です

$ sbt
...
[info] Set current project to ...
>

マニュアルインストールの場合

以下の手順に従い、インストールします。
http://www.scala-sbt.org/release/docs/Getting-Started/Setup.html#manual-installation
