ゲーム開発ワークショップ@仙台

4月17日に開催され、好評を頂いた「9leap Game Programming Camp @東京」に続き、次は仙台でゲーム開発ワークショップを開催します。前回は定員20人の募集に対して50人以上の応募があり、急遽会場の規模を変更し定員を30人の方々に参加していただき、ゲーム開発についての講義のあと、4時間の短い開発時間の中で1人1本のミニゲームを開発しました。

概要

「JavaScriptを使ったことはあるけど、スマートフォンで動くアプリを作ったことはない…」
「ゲーム製作に興味があってプログラミングの勉強をしてるんだけど、何をやったら良いのか分からない…」
そんな方々に向けて、 9leap ではHTML5+JavaScript を利用したゲーム開発ワークショップ「9leap Game Programming Camp @ SENDAI」を開催します。

ゲーム開発といっても、CanvasやJavaScriptを理解してイチからミニゲームを作るにはとても1日では足りません。そこで今回は、ミニゲームなどを簡単に製作することができる、HTML5+Canvasを用いた開発支援ライブラリ 「enchant.js」 を用意させていただきました。HTML5を用いたアプリケーション開発の実践、また、本格的なゲームプログラミングへの入門としてお使い頂けます。

ゲームデザイン・ゲームプログラミングに関する講義と、ゲーム開発支援ライブラリ「enchant.js」の解説のあと、参加者の方々には実際のミニゲーム開発にチャレンジしていただきます。最後に開発したゲームの発表、講師からの講評という流れです。

今回のワークショップで製作いただいたゲームは、5月1日から応募受け付けを開始するミニゲーム開発コンテスト「 9leap 」にも投稿していただけます。
