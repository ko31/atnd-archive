description

弊社ライトアップでは、Webマーケティング人材育成を支援しております。

その一環としまして、
・Web人材を社内で育成したい育成責任者・経営者
・Web業界にキャリアチェンジしたい方
・Webマーケティングを今後徹底して学びたい方

を対象として、Webマーケティング人材育成説明会を開催することとなりました。
より交流を深めて頂くため、ランチ交流会　軽食付きです。
ランチ交流会については、初めてご参加頂いた方にも参加しやすい雰囲気作りのため、交流会プランナーがアシストしますので、安心してご参加ください。
※ランチ交流会のみ参加の方もいらっしゃいますので、人数は50名規模となることが予想されます。

Web育成も学べて、ランチ交流会でフラットに普段お会いしない方と交流ができる
そんな時間になるようにします！

◆Webマーケティング人材育成説明会※参加費無料

具体的には、Web解析からSEO、リスティングまでのWebマーケティング運用・提案に必要な研修を弊社で一貫して受講が可能となります。
※弊社ライトアップでは、Webマーケティングのインハウス化の支援として、
クラウドサービスだけでなく、研修までご提供しご支援をしております。

特に、Webマーケティングでは解析部分を強みとしており、
またSEO分野については、ツール販売からコンサルティングまで中小規模から大規模まで対応しております。リスティングについては、弊社親会社オプトのシニアコンサルタントが担当させて頂いております。

◆ランチ交流会※参加費500円（税込）

Webマーケティング担当者人材育成と業界間の横繋がりのお名刺交換会として、
ランチ交流会を実施する運びとなりました。
※ご参加者の方は恐れ入りますが、参加費500円（飲食代・軽食付き）を当日頂戴します。
※前半のWebマーケティング人材育成勉強会・後半のランチ交流会のみの参加も可能です。

◆Webマーケティング人材育成勉強会とは

Webマーケティング人材育成勉強会少しでもWebマーケティング人材育成にご興味をお持ちでしたら、ぜひご参加頂きたいプログラムとなっております。※Webマーケティングに必要な解析手法、コンテンツSEOテクニック、助成金ネタなどもこっそりお伝えします！！

また、今回は導入編となりますので、まずはお気軽にご参加頂き、そこから本編に入るかをご選択頂ける形となります。
本編については、研修サービスですので費用は発生しますが、助成金を活用すれば無料でご参加頂けます。
助成金活用については、条件がございますので、そちらも合わせて御説明いたします。

◆当日のプログラム
当日のプログラム
（今回は凝縮版となりまして、1時間30分ほどがレクチャーとなり、最後個別相談会となります。）

(１)　WEBマーケティン戦略に必要なスキル
・Googleアナリティクスの重要性
・WEBマーケティング戦略研修の内容

(２)　SEOの社内化「インハウスSEOとは」
・インハウスSEOに必要な3つのスキル
・インハウスSEO研修の内容
　　　
(３)オプトが教える、リスティング研修
・ネット広告国内No.1のオプトのノウハウを吸収できる研修の内容

(４)研修費用が無料になる国の「助成金」について
・御社が受給できる助成金診断
　
(５)　個別相談会（ご希望に応じて）
・セミナー後に15分程度お時間を設けますので、
個別でのご相談があればお気軽にお申し付けください。

※今回は、完全無料の勉強会ですので、ご安心ください。※

（６）ランチ交流会　11時45分～13時30分
・入退室自由となっております。
※ランチ交流会のみのご参加も承っております。
（飲食代・軽食付き）
