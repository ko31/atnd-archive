参加方法

  1. このページの[このイベントに参加登録する]から登録してください
    • ユーザープロファイルか参加コメントで自分の tumblr の url がわかるようにしてください
  2. [JOIN TO MEMBER]にて RAC5 特設 tumblr サイトにメンバー登録してください
    • join page by mamemomonga
  3. コメント欄に自分が担当したい日付を宣言してください
    • ただし 12/1から12/23までとします。早いもの勝ち
    • 12/24,12/25は下記のルールを参照

ルール

  1. 11/29,30は自由投稿を許可
    • 5周年記念として、コースインスペクションを許可します。本来12/1から1名ずつに許可されているコースの走行が可能です
    • 各自、ツールの動作確認などを実施してください
  2. 12/1から順番に、一人一日を担当
    • たくさんリブログしても良し
    • 濃い一発をかましても良し
    • それぞれがそれぞれのスタイルで
  3. 当番がいなくなった時点または当番がいない日、そして12/24 0:00からはフリー
    • 「今日はみなさんに、ちょっとリブログしてもらいます。 最後の一人になるまでです。反則はありませーん」
    • つまりバトルロイヤル
  4. 12/25 24:00 の鐘が鳴り終ったら
    • Members | Tumblr にて “Leave this blog” します
    • 最後の一人が “Leave this blog” した瞬間に reblogadventcalender5dot1.tumblr.com は消滅します
    • “Leave this blog” しなくても、強制的に消滅させます
