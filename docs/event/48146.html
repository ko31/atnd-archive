【概要】

2014年3月8日(土)
18:30～20:30
新宿で起業家交流会を開催いたします。
【特典資料付き】

・人脈が欲しい。交流を広めたい。
・仕事はあるが人手が足りない
・人は足りているが、仕事が欲しい
等々。

名刺交換・情報交換を通じて、新規お取引先の開拓や人脈作りにお役立てください。

【詳細】

日程：03月08日（土）
時間：18:30～20:30
場所：新宿GSハイム佐藤ビル 会議室
参加費：¥1,000
（当日会場にてお支払いください。）
持ち物：名刺
（受付時2枚ご提出いただきます。）

【内容】

1.
まず、入口にて下記説明があります。
・交流会についての注意事項
・参加者名簿（資料）の配布
・幹事挨拶
（1社毎、直接ご挨拶いたします。）

2.
ご自由に名刺交換会
・起業時の悩みを双方で交換！
（同じ境遇だからこそ、話せる内容もあるかもしれません！）
（悩みを解決できる糸口が見つかれば、なお良し！！）
・ビジネスパートナーのご相談
（どんどん輪を広めていってください！人脈は最大の武器になります。）
・途中参加、途中退出ももちろん可能です。

3.
19:00に希望者5名、アピールタイムを設けます。
・発表者は当日決めます。
・ご自身の理念、やりたいこと、どんなことを求めているか存分にアピールしてください。
・アピールタイム後は再び歓談です。

4.
20:00、 再び別の希望者5名にアピールタイムを設けます。
・発表者は当日決めます。
・ご自身の理念、やりたいこと、どんなことを求めているか存分にアピールしてください。
・アピールタイム後は再び歓談です。

5.
20:30に終了
・二次会、三次会・・・以降はお好きに！
・21:00完全退出の為、会場は時間厳守でお願いいたします。

【特典】

参加者には、以下をプレゼントいたします。
・起業前にこれだけは押さえたい5つのポイント
・起業後も忘れてはいけない3つのポイント
※当日、どちらかをお選びください。

【注意事項】

下記に当てはまる方は即退場していただきます。
・ネットワークビジネス勧誘目的の方
・宗教勧誘目的の方・先物取引勧誘目的の方
・能力開発セミナー勧誘目的の方・その他、交流会の趣旨に沿わない目的で参加希望の方
