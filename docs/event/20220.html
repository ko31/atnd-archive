初となるワイン国際交流パーティーがついに恵比寿で開催が決定しました。会場は、なんとあの結婚式場で有名なロビンズクラブ。恵比寿の閑静な住宅街を徒歩数分、姿をあらわすのは、オフホワイトの優美な曲線で構成された外観が印象的な一軒家レストランでの国際交流パーティーとなります。セレブな会場でワインを楽しみながら、外国人との会話を楽しんでください。

業界初となる『セレブ国際交流パーティー』はGaitomoだけ！いつもと違う国際交流パーティーを楽しんでください。外国人は皆日本語が話せますので、英語に自信が無くても安心してご参加頂けます。ヨーロッパからアジア圏まで幅広い国の外国人たちが参加されます。このパーティーは、某国際交流パーティー3社の特大コラボ国際交流パーティーとなりますので、300名近くの方が参加される予定となります。たくさんの新しい出会いをこの機会に楽しんでください。
