萩野酒造の会
日時：9月24日（土）18時開始
会場：酒と魚 mocchi （京王線 神泉駅よりすぐ）
会費：6500円
ゲスト：萩野酒造　佐藤曜平さん

～自分で考え、自分で決めろ。自分が今、何をすべきなのか。ま、後悔の無いようにな～

萩野酒造。宮城県の最北、岩手県との県境の栗原市に蔵は位置します。

東京農大をご卒業後、蔵に戻られた佐藤曜平さんが18BYより指揮をとり酒造りをしています。

メインブランドは「萩の鶴」と「日輪田」の2種類です。
25BYから萩の鶴は速醸、日輪田は山廃という形になりました。

曜平さんの造るお酒は、正直言っていい意味で「とらえどころ」がありません。

新酒から味がしっかり乗っていてガンガンに旨いお酒。
熟成にも耐える燗にしてしみじみと旨いお酒。

どのお酒をとっても、高いレベル。 共通して「クリア」な印象です。

近年は「火入れ」に力を入れていて、搾ってから加熱処理までの時間を極力短くすることでフレッシュさを保ったまま、安定した酒質を実現させています。

「火入れを制するものは日本酒を制す」と曜平さんは語ります。

3.11の震災で大きな被害を受けて、蔵は新しく建て替えられて、4年目を迎えます。
2012年に初めて蔵を訪れました。そこからのご縁で会を開催させていただいています。

今回はmocchiさんでの開催。
店主の望月さんは曜平さんと一緒に釣りに行く仲です。

望月さんの料理は出汁を大切にしたもの。お酒の相性は抜群で、杯と箸が自然と進んでしまいます。

会では、mocchiさんの料理と曜平さんのお酒を存分にマリアージュいただきます。

暑さが少しゆるんだ夜に、曜平さんと一緒に楽しい時間を過ごしましょう(^_^)/
