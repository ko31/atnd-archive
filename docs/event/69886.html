description

自社ECサイトを運営される企業様にとって、市場の拡大、売り場・ニーズと共に多様化が進む中、
今まで通りの施策では売上を上げることが難しくなってまいりました。
とは言え、どのような施策を打って良いか悩まれている企業様も多いのではないでしょうか。
成功している企業では、ECサイト内に様々な「おもてなし」の仕掛けを取り入れ、
また訪れたくなるサイトを作り上げることで継続的な売上に繋げています。

今回のセミナーでは、顧客育成に成功している企業様の事例を中心に、
既存顧客をファンにする実践的なノウハウをご紹介してまいります。

セミナープログラム
-—————————————————————————-

【第一部】
『徹底した顧客視点で新たな価値を生み出すEC戦略』
登壇：株式会社コマースニジュウイチ　田村信博氏

O2O、オムニチャネル、DMPと様々なキーワードが飛び交う昨今で弊社のお客様が
EC事業で成功している様々な事例をご紹介します。
課題があり、それが解決されなければ成功はありません。
今後のEC事業を進める、検討される中でお役に立てる内容をご用意いたしております。

【第二部】
『一度、購入したお客さまをファンにする3つの方法』
登壇：株式会社Sprocket　深田浩嗣氏

1度、購入したお客さまでも、
ファンになってもらいリピーターにまで育成することは困難です。
EC事業者さまが本当に伝えたいメッセージが伝わらなかったり、
メッセージが書いてあるページに辿り着かない事が多くあります。
今回の講演では最適なメッセージの伝え方やファン育成のための
仕掛けを事例と共にお話します。

【第三部】
『顧客の育成プロセスまで考慮したデータドリブンマーケティング』
登壇：アライドアーキテクツ株式会社　津下本耕太郎

SNSの出現によりタッチポイントが増加し分散している顧客接点において、
どのようにデータを管理し、どのようなコミュニケーションをとっていく必要があるのか、
その方法について事例を交えて解説いたします。

【第四部】
パネルディスカッション『新たな視点でEC事業の課題解決を考える』
登壇：株式会社コマースニジュウイチ　田村信博氏（ファシリテーター）
株式会社Sprocket　深田浩嗣氏
アライドアーキテクツ株式会社　津下本耕太郎

事前にお客様より頂いたEC事業の課題やご質問について、
登壇者がパネルディスカッション形式でお答えしていきます。

【名刺交換会・ご相談会】
第四部終了後、名刺交換会および各社への相談や質問ができるお時間を用意致します。
お気軽にご相談、ご質問ください。
