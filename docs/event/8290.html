Scalaを遊ぶ!会 第1回参加者募集のおしらせ

Scalaをみんなでいじって遊ぶ会です。
第1回は、REPL（Scalaのコマンドラインインターフェース。Rubyで言うirbにあたる）で遊びます。

First Steps to Scala を教材にREPLで動作を確認しながら、アプリケーションを構築するのに最低限必要なScalaの基本的な文法を身につけます。

なお会の方針や進め方等は、 こちらのブログ記事 を参照してください。

1. 準備について

Scala2.8.0 の実行環境がインストールされたPCを用意してきてください。
なお会場が地下のため、emobileやUQが入る可能性は低いと思われますので、必要なソフトウェアは事前にインストールしてくるようお願いします。
（※喫茶フロアでは公衆無線LANが入るようなので、いざとなったらそこまで移動するという手もあります。 ヨドバシワイヤレスゲート や Wi2 300 《どちらも月額380円》等のサービスに加入しておくと使えます）

2. 会費について

会場は有料施設です。そのため 会費が発生します 。
最小参加者数（5名）の場合が1人 1,500円で、それ以降人数が増えるごとに安くなる計算です。なお会費にはドリンク1杯分が含まれています。

3. キャンセルについて

有料施設を少人数で使用するため、直前のキャンセルは他の参加者に少なくない金銭的な負担をかけます。
また施設側からもキャンセル料を請求されるため、キャンセルを受け付けるのは基本、前々日までとします。
当日のドタキャンは、発生した損失分をキャンセル料として当人から後日何らかの手段で徴収します。場合によっては前日キャンセルも同様です。
