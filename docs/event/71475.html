第47回 FreeBSD勉強会

ライトニングトーク×FreeBSD忘年会’15！

講師
発表したい方みんな

FreeBSD勉強会も今回で47回目を迎えます。2015年も終わるということですし、ここで一つ、2年ぶりに、大盛況だったあのライトニングトーク大会を開催したいと思います。今やっていることがある、実はこんな感じで運用している、困っていることがあるので助け求むなど、FreeBSDに関することを5分から10分程度にまとめて、その時発表したい方が発表するスタイルです。

いつもの勉強会とはことなり、ビールを飲みつつざっくばらんに行こうと思います。後藤からは発表の冒頭、5分〜10分くらいでbhyveを使った仮想環境構築のデモンストレーションを行いたいと思います。CentOSかOpenBSDあたりをon bhyveで使う方法をご紹介します。

ひとりあたりの発表時間は、全体の発表人数を見つつ当割でいきましょう。時間に余裕があれば現場で飛び込みライトニングトークも大歓迎です。日程も休日の前をチョイス。思う存分語らいましょう。Let’s talk frankly!
