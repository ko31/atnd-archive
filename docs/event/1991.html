11月の WordPress 勉強会＠福岡を開催します。

今回は参加費無料です。19時くらいからどこかのお店で懇親会を予定していますので、懇親会まで参加できる方はコメント欄にその旨書き込んでください。なお、今回の懇親会は同日に開催される XOOPS ユーザー会との合同開催になる、かもしれません。お楽しみに。

今回の勉強会の内容ですが、1〜2時間程度、プラグイン作成を実習形式でやりたいと思います。PC にローカルの WordPress 環境を作って持参されるといろいろとよいことがあると思います。その他の時間は未定ですので、何か話したいことがある方はぜひお願いします。コメント欄に書き込んでください。

初めて参加される方へ: 場所がとてもわかりにくいので注意してください。地図に載っているローソンは実際にはありません。通りの向かいにジョーキュウ醤油の店舗があります。
