2014年11月15日（土）に男木島（香川県高松市）で「WordBench香川 in 男木島」を開催します。

概要

午前中は島散策、午後からWordPressプチ勉強会を行います。
備考の項にも記載しておりますが今回は本格的な島散策・勉強会ではありませんのでカメラ・ノートパソコンは必要な方のみご用意ください。

@nukaga さん
　以下よりいずれかのテーマでアンカンファレンスできればと提案（要望により変動）
　・香川のオープンデータ状況とWordPress（JSON REST API）
　・WordPress+grunp
　・公式ディレクトリテーマ登録について
　・WordPressとアクセシビリティ

@dr_cova さん
　日本Androidの会香川支部で開発した『イケ麺時計』についてLT（フロントサイドはJavaアプリでバックエンドをWordPress使ってコンテンツ管理）

また時間があればVagrant+VCCWでWordPressが動くところまで+αをやりたいと思います。
事前にVagrant,VirtualBox,git（Windows環境の場合）をインストールしてください。
・ VirtualBox
・ Vagrant
・ git

ライトニングトーク行いたい方がいらっしゃいましたら大歓迎です。下記お問合わせよりご連絡下さい。
