自宅ラック友の会、
2013年の1発目は「自宅ラックパーティー」です！

自宅ラック勉強会の残念な人、@tadkinosです。

どうしてそうなった？

　* 仕事熱心な妻が、職業柄ついカッとなって計画的におせち料理をyumしてしまった。
　　稀によくあるので折角だから誰かにinstallしてもらおうか。
　* つまり新年なのに妻がバイトなので子育てが大変な某牛乳屋さんが開催する、小規模イベントです。
　* ついにリビング使用の許可が下りました。

何をやりたいの？

　* 今後の「自宅ラック勉強会」ネタを考えたいです。
　* ポータルサイトについてもネタを考えたいです。ついでに構築できるか？
　* 娘（6歳）も一緒に遊べるような何か（ゲームとか）を期待しています。
　* 仮想化とかNASの検証をやって遊ぶというのも勿論アリです！

持ってくるモノ

　* PCとかスイッチとかサーバとかIP電話機とかゲーム機とか
　* セブンイレブンのおせち料理他をおいしくinstallするための大容量ストレージ（空腹的な意味で）。
　* 今回は前回(http://atnd.org/events/33726)以上のお値段なのでnullデバイスへの
　　リダイレクトは本当に勘弁してください。
　* 実際に量的には多くないかもしれませんので、そこは適宜考えます。

参加条件

　* 自宅ラックな人をメインに、配偶者様、お子様、恋人やお友達など参加可能。
　* 娘が部屋にいても嫌じゃない人。
　* お年玉は不要ですが、参加費として500円だけ徴収させてください。

会場および集合場所

　* 個人住宅(マンション内の一室)が会場となります。
　* 今回は私が駅に集合できませんので、住所は事前にお教えします。
　* というわけで、駅からは各自でタクシーで会場まで移動してください。(割り勘でお願いします)

注意点

　* マンション共用スペースでの私語は控えて頂けますようお願いいたします。
　* 開催者はただの残念な人なので技術力がありません。
　* その代わり周りの皆様が助けてくれるような気がします。
　* PCやサーバ等の持ち込みは可能ですが、電源容量は期待できません。（30Aです)
　* 上記理由により、ブレードサーバなどの持ち込みとか本当に勘弁してください。
　* 駐車場は近くにありません。
　* もちろんですが、自宅にラックが無くてもご参加いただけます。
　* 開催者の職業柄、別途おやつが用意される可能性があります。
