TensorFlowチュートリアルに学ぶディープラーニング入門 ハンズオン勉強会

講座内容概要

本講座では、TensorFlowチュートリアルを参考に、ハンズオンすることで、ディープラーニングの理論を実践的に学んで頂きます。
講座の前半はハンズオン時に理論を理解するための知識について説明します。後半では、TensorFlowを使ってMNISTモデルを構築し、モデルの評価までをハンズオンします。
深層学習（ディープラーニング）におけるMNISTは、プログラミングにおける"Hello World."の出力に比するものですが、概念を理解することで、より高度なディープラーニング利用への道が開けます。

こちらを事前に受講いただけると、よりスムーズに理解できるように講座を設計しておりますが、
前提知識がなくても問題ありません。

機械学習・深層学習のための数学入門 #1
https://skillupai.connpass.com/event/69892/

この講座でみにつくスキル

・機械学習とディープラーニング（深層学習）の違いが説明できるようになります
・ニューラルネットワークとディープラーニングの背後にある核心的概念を理解できます
・TensorFlowの基礎的な使い方が理解できます
