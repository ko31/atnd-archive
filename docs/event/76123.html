ライフ＆インフォ女子の会

ライフ＆インフォ女子の会は、ライフサイエンスに携わる女子のライフスタイル、インフォマティクスを支援するコミュニティです。スキルアップと情報交換、ソリューションの構築などを目指しています。第一回女子会では、プチ勉強会とお食事会を開催します。参加者は研究や仕事でバイオインフォをやっている、またはやってみたいなと思っている女子を対象としています。コミュニティと言っても発足したばかりです。お気軽にご参加ください！

ライフ＆インフォ女子の会　代表
（キアゲン宮本＆イルミナ長井＆サイキンソー竹田）

どんなことやっているの？

  • バリバリのバイオインフォマティシャンに最先端のお話をしてもらう
  • RとかPerlとかphythonだとか…
  • バイオインフォで用いられるプログラム言語を実際に触ってみる
  • 課題を解いてみる
  • わからないことを相談してみる
  • わいわいしゃべりまくる
  • 美味しいごはんとお酒で盛り上がる
  • ワークライフバランスについてアイデアを出し合ってみる
  • キャリアを相談してみる

RとRStudioのインストール

Rは3.2.4を想定しています。sessionInfo()にてインストールされている
バージョンが確認できます。

またRStudioも最新版の0.99.896を想定しています。RStudioのHelp > About RStudioより
ご確認ください。

パッケージのインストール

  • 基礎編
    isnstall.package(“ggplot2”)
    install.packages(“ggthemes”)
  • 孫さん講義分
    install.packages(“RColorBrewer”)
    install.packages(“reshape2”)
    install.packages(“edgeR”)
    install.packages(“clusterProfiler”)
    install.packages(“org.Hs.eg.d”)
  • 露崎さん講義分
    install.packages(“reshape”)
    install.packages(“GGally”)
    install.packages(“pairsD3”)
    install.packages(“scatterplot3d”)
    install.packages(“animation”)
    install.packages(“rgl”)
    install.packages(“plotly”)
  • 注意
    edgeR, clusterProfiler, org.Hs.eg.dbあたりは、install.packageだとnot availableになったのでbiocLiteでインストールしてください
