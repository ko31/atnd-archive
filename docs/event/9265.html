description

１２月１２日（日）に 御舟かもめさんを貸し切って 中之島１週のナイトクルーズを開催します
ルートは　若松の浜船着場から　堂島川を西に向かい中之島西端で土佐堀川に入り東へ
中之島東端から大川をのぼり大阪城付近へ　そこで折り返し　若松の浜船着場へ戻ります
時間は　１６：２０～１８：２０　料金は３０００円（１０名集まった場合）です
当日の日没は１６：４７　月没は１７：０７（ただし新月なので月明かりは無し）
ちょうど 夕焼けの時間から　マジックアワーを経て暗闇へと　街の表情がドラマチックに変化する時間帯です
当日は ＯＳＡＫＡ光のルネサンスが開催されており　橋のライトアップと合わせてかなり見ごたえのあるクルーズになると思われます　
また　あひるちゃんも帰ってきてますので　中之島バンクス付近でのご対面となります

当日はＵＳＴＲＥＡＭ配信も実施予定です　ＶＡＩＯとポケットＷｉ－Ｆｉを 持参します　
また電源をお借りできることになったので アイフォーンの充電もできますよｗ
クルーズの後は　京阪中之島線に乗って あひるちゃんをバックに記念写真を撮りましょう
そのあと 食事ができればいいなと考えてます

当日のメンバーは　ｓｅｅｍｏ・ゴリモン・カオス・バネヤ・ニシキン・イッコウ・ミズ００１・コロクン・ムツキ・ｔｅａ ｒｏｏｍ（敬称略）の十名です

それでは　よろしくお願いします
