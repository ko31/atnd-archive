description

ウェブ技術にテーマを絞り、参加者同士がネタを持ち寄って発表し合う形式の勉強会です。

PHP、Perl、Ruby、JavaScript、MySQL といったガチなWebプログラミング系の話はもとより、 CSSを使ったデザインテクニック、HTML5、Flash、Webサーバ運用、クラウド活用法、ネットセキュリティ、SEO など、ウェブに関するテクニカルな話なら何でもOK。

趣味や仕事でウェブに携わっている方はもちろん、まったく初心者の方も大歓迎です！

一般：500円
学生・発表者：無料

発表予定

  • Webプログラミング入門
    特定プログラミング言語での入門を話すつもりはなく、 最近のフレームワークの主流ってこんな風（MVCとか）になってるけど、 なんでこういう構成が好まれるのかとか、 そういう初歩的な所に焦点を当てて説明をできればと思っております。
    （colun、30分）
  • IE10とCSS3
    IE10で対応して欲しかったCSS3をWindows8 Developer Preview 上で対応状況を調べてみた。
    （田中克之、30分）
  • あなたの知らないウェブサーバー「Yet another web-server」
    Javaで動作する謎のウェブサーバーを紹介し、ローカルに実演する。
    （＠ヒデ、30分）
  • jQuery Mobileでスマートフォン用サイトを構築しよう
    jQuery Mobileの簡単な説明と実際のサイト構築してみた流れ
    （スギミス、30分）
