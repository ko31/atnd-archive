description

リクルートでは、各グループのサービスで生まれる大量のデータを日々解析し、ビジネス上の施策へと接続しています。

データ利活用の必要性は日に日に高まり続けており、中でも昨今は

・テキストや画像といった非構造データの収集・活用
・Deep Learningをはじめとした新しいアルゴリズムの適応
・人的作業の一部を機械代替するような新たな利活用シーンの開拓

といったテーマを重点的に推進しています。

今回は上記の取り組みの一環である、リクルートグループでの利用を目的に開発されたAPI群
A3RT（Analytics & Artificial Intelligence API Via RECRUIT Technologies）の概要を中心に、
各ソリューションで用いられるアルゴリズムや利活用シーンを紹介したいと思います。

また、現場で実際に開発・施策接続を推進しているメンバーだからこそわかる、施策接続にいたるまでに現場で起きる苦悩や課題、それらをどう乗り越えてきたかといった内容を赤裸々に語っていきたいと考えています。

「AI」という華やかなイメージの裏側にある「リアル」。これを皆さんにお伝えしていきます。
