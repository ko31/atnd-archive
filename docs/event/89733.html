description

飲食店の出店をお考えの皆様。

儲かるお店のつくり方では、一般的にはこんなことを言われますよね？

「初期投資を抑えよう」　「集客に余計なお金をかけない」

「人件費を抑えよう」　「安価な仕入先の確保」

しかし、儲かる商売の本当のポイント・本質は別にあります。

それは読んで字のごとし！

「儲」　と書いて「信者」と読む。

信者と呼ばれるような、「熱烈なファン」を作れるかどうか。

そんな信者（ファン）を生み出すお店を作るには、

単に美味しい・不味い・高い・安いだけでは、「全然」足りません。

・ファンとアンチが存在し、ときおり良し悪しについて議論になる。

・駅前の良い場所ではなくて、むしろ路地裏や駅から遠い不便なところにある。

・その店の成り立ちや店主の経歴にストーリーがある

そのように適度に「尖った」店が信者（ファン）を生むのです。

※適度に、というのが重要ではありますが。

その「尖り」を最も巻き起こしやすい商材の一つが、「ラーメン」。

例えば、ラーメン二郎をこよなく愛するジロリアン。

例えば、博多っ子はお気入りのとんこつラーメン店について２時間議論。

そのような信者（ファン）を「狙って」生み出し、

５０店舗全てを黒字にしてきた経営者がいます。

直営は食べログ３．５以上。（３．５点を超える店は全飲食店の上位４.４％）

全店でも平均３．４２。

それらはほぼ名前がそれぞれ違い、出しているメニューも少しずつ違う。

「チェーンには全く見えない、チェーン店」です。

そこで今回は、

～路地裏でこそ儲かる！熱烈ファン（信者）がつく飲食店の作り方～

について解説。

知り合い・紹介以外で、初めてFC加盟店を募集します。

路地裏で儲かるお店作りに関心がある方。

画一的なチェーンに、ウンザリしている方。

にはオススメです。

あ、あと「店での労働時間を４ｈくらい減らしても運営できる」業態に関心ある方にも。

満席になることが予想されます。お早めに申し込みください。

本セミナーでは「フリーネーム」鶏白湯・煮干しラーメンのFCのお話を中心に

儲かるお店づくりのノウハウをご紹介させていただきます。

これまでは紹介や既存店の増店のみで３年で５０店舗以上を展開。

全店黒字を継続的に実現！

「低コスト開業」　「広告費用の削減」

「一等立地はお断り！路地裏出店のコツ」

「人材の疲弊を抑え、離職率を激減」

「食べログ高得点を取るための秘訣」

などを高いレベルで実現しているラーメン店。

実は今回わたくしどもから無理を言って、

「月１店舗の出店ペースであれば」

という条件ではじめて、

紹介以外での募集をさせて頂くことが可能になりました。
