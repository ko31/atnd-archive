description

“みんなで技術を共有・披露・暴露しあっちゃおうぜ！”

今回の勉強会は講師一人がセミナーするのでなく、
全員が、実案件または自分のWPブログなどを例に、話(暴露)しあってWP技術を共有しあう勉強会です。

※別件で、同7日夜にデジハリスタッフ上船さんの退職送別会があります。(こちらは50人以上は参加するかも…？)
北新地駅近く「Chinese Dining まりんてい」19:00～￥3500です。
勉強会・送別会、両方参加か片方参加か4月末までにコメント欄に書いてくださいませ。

【目的】：みんなが今後、難しいWP案件受注する時のウツになる悩みを減らす。

●イイネその１　勉強会中「あのやり方・プラグインめちゃいい・・・私も使おう！」情報ゲット！
●イイネその２　勉強会中「(発表者の)その案件なら、俺やったらこうやったなー」自分のWPに対する頭の体操＆相手のアドバイスに。
●イイネその３　勉強会後日「さっき来た案件、、そーいやWP勉強会であの人が同じようなのやった言ってたな…真似てみよう（いや、やっぱあの人に聞いて頼ってみよう）」

って仕事のお悩み解決法がぽぽぽぽ～んってなるはず。



ということで、参加者全員が発表者！！

持ち時間は、1人(1組チーム可)発表8分＋質疑応答5分くらいかな。人数次第。
一番最後にも長めに時間とって、復習議論しましょう。


「えぇぇ…ぼく・わたしそんなに発表するほどWP技術ない…」って思う人、発表するターゲットレベルは自分。
だから自分より詳しい人がいたとしても気にしないで、皆に自分の持ってる限りの技術を提供する。
（どちみち時間そんなにないから、全部提供しようとも無理やけどねｗ）



皆から教えてもらう代わり、自分もチョビっとでも皆に役に立ててもらおう！
ギブ＆テイクのスタンスでやりましょ！

■どや系（自慢）
・自分の案件・ブログでこんな便利プラグイン使ってるよっ。どやっ！
・この実案件でやっかいな注文あったんやけど、こうやって組んで完成させたで。どやっ！
・WPにこんな機能・プラグインあるの知ってた！？どやっ！
・これは誰もやった事ないはず！どやっ！

（同じ参加者の誰かから聞いてやったことでもOKす！）


□どーしょ系(質問)
・今ややこい案件やってんのやけど、組み方さえどうしたらえぇかわからん…どーしょ？
・これここまでやってみたけどうまくいかんねん、なんでなん…どーしょ？
・この案件のとき私こうやったんやけど、皆ならどーしたっしょ？


なるべく「どや系」で、どーしても時間埋めれないかったら「どーしょ系」を足すのが良い。
