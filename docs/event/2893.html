「オフ会とか出てみたいけど人見知りだからムリ…。」
そんな方のための飲み会、やります。

広い会場のどこかで餃子を食べながらツイッターで会話。
お互い顔を合わせる必要は特にありません。
例えば「どこそこの餃子が美味しいですよー」「へー、買ってこよう」みたいな、
同じ場所で同じことをしているというだけの一体感で盛り上がり(?)ましょう。
参加者っぽい人に声をかけることのできる勇者ももちろんOK。
あいさつが終わって沈黙タイムになってもツイッターがあります！
だいじょうぶ怖くない怖くない。（注・主催者が一番怖がっています）

ツイッターをやってない方はこの際だからぜひやりましょう。http://twitter.com/
アカウントを取得したら@beetonをフォローしてもらえるとありがたいです。
もちろんツイッターやってなくても参加可能ですよ。

会場はナムコ・ナンジャタウン内にある池袋餃子スタジアム。
入場料300円がかかりますが飲食はキャッシュオンですので、各自のペースで気の済むまで飲み食いしてください。
21時までなら入園可能なので、ふらっと来て食べてささっと帰って全然構いません。
会場の奥の方は携帯（特にSOFTBANK）がつながりにくいので入り口はいってすぐにある広場をメイン会場とします。
人見知りにとって携帯やiPhoneは命綱ですからね（笑）
十分なバッテリーを各自ご用意ください。

当日は土曜日なので開始時間より前に来てほかの施設でひとしきり遊んでからの参加も可能です。
途中でデザート共和国に移動したりもできるので飽きないと思いますよ。
ハッシュタグ「#shygyz」をつけてゆるく会話しましょう。

このページでの参加表明は別にしてもしなくても構いません。ドタキャンとかも全然おk。
とにかくなるべく縛りのないゆるいオフにしたいです。

※このオフはかつて湯川鶴章さん（@tsuruaki）が開催されたイベントを下敷きにしています。超リスペクト。
ちなみに主催者beetonはガチで人見知りなため、前回の同様オフの会場で
和気あいあいと盛り上がる参加者に声をかけることができず遠巻きに眺めただけで帰り優勝しました。（なんのだ）
