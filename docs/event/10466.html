参加と記事アップロードの仕方

ATNDに参加した順番で12月1日から12月24日までsymfony / Symfony2に関する
tips書いていきます。 1番目に参加された方は12月1日にブログで記事を公開してATNDのコメント欄に告知。2番目に参加された方は12月2日に…という流れでやっていきます。

このルールだと参加が難しい方もいらっしゃるようですので、日付指定されない方は
空いている日付から自動的に割り当て、日付指定したい方は参加時にコメントで記載
、もしくはATNDのコメントに書きたい日付を書いてください。

日程については 継続的に更新 しておりますのでそちらを参考にお願い致します。
