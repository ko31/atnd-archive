description

　欧米諸国では小麦に含まれるたんぱく質「グルテン」にアレルギー反応や不耐症を示す人が増加しており、アメリカ食品医薬品局（FDA）ではグルテンの免疫疾患を持つセリアック病患者は全米で300万人に上ると推定されています。これを受けて、アメリカの食品業界では、グルテンフリー市場は2012年までの5年間に年平均28％成長し、グルテンフリー食品の市場は今後、益々拡大するものと思われます。日本政府観光局（JNTO）によると、2013年のイギリス、アメリカ、フランスからの訪日外国人の割合は、それぞれ10.3%増、11.5%増、18.8%増と、欧米諸国からの訪日観光客も増加しており、今後、日本でもグルテンフリーに対応したサービスが求められることは必至です。日本ではまだ馴染みのない食品ですが、小麦粉や小麦を原料とする醤油を多くの料理に使用する日本にて、グルテンフリーを求める外国人に安心して食事を召し上がっていただくためには、どのような対応が望まれるのでしょうか。
　本講座では、グルテンフリーについての基礎知識から、国内外の市場、グルテンフリーを求める方への具体的な対応方法までお伝えいたします。

【海外の食と健康トレンド2015〜注目食品(グルテンフリー)と健康事情・健康食品の動向〜】

◼︎講師　：二神さやか（株式会社セシリア　代表取締役）
慶応義塾大学大学院理工学研究科化学専攻修士課程および米国johns Hopkins大学大学院バイオテクノロジー専攻修士課程修了。一般社団法人グルテンフリー協会代表理事。学生時代よりずっと理系畑を歩んできたが、母親が還暦を目前にして突如小麦アレルギーになったことから、一念発起して国内初のグルテンフリー食品専門店をはじめる。別途、グルテンフリー協会を設立し、国内におけるグルテンフリー表記のガイドライン策定などの活動を行っている。
