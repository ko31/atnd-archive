会場は六本木の閑静な通りに佇む、
様々なパーティーオーガナイザーに大人気の一軒家寛ぎの私空間「ASTRANCE」

豪華かつスタイリッシュなメインフロアに加え、
メゾネットになった中2階フロアが素敵な出逢いを演出します。

プランはASTRANCE特製ビュッフェ&フリードリンクと大満足の内容です。

邸宅風のスタイリッシュな造りとプライベート感あふれる上質のホスピタリティと
会場のゴージャスな雰囲気が、出逢いと交流を盛り上げること間違いなしです。
