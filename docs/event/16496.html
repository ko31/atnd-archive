たとえば…
お客様向けの帳票テンプレートをExcelで作るとき、ご満足いただけるものを効率よく作れていますか？
プロジェクトで利用しているExcelの帳票から品質データを収集するとき、うまく自動化できていますか？
品質データを分析した報告書をExcelで作成するとき、見やすく分かりやすい資料をテキパキと作れていますか？
大量の品質データを効率よく分析し、改善につながる分析ができていますか？

この勉強会では、ソフトウェア品質に効かせたい「小ワザ」をお互いに紹介し、わいわいやりながら楽しく学びます。
小ワザは、ショートカット操作やマクロ、ワークシート関数などさまざまです。
Microsoft Excelの小ワザが中心ですが、上に書いたような場面で役立ちそうなツールの小ワザも歓迎です。

＊＊＊＊＊＊

タイトルに「ソフトウェア品質に効かせたい！」と書いていますが、それは、ずーっと先の目標です。
まずはExcelの小ワザをたくさん身につけ、仕事を楽に、楽しくしたいというのが今回のねらいです。

なので、参加してみたけど「ソフトウェア品質の話がない／少ないじゃないか！」と思ってしまいそうな方は、
今回はご遠慮ください。

＊＊＊＊＊＊

紹介するネタがない方や、学生さんの参加もウェルカムです。気軽にご参加ください。

小ワザから出発して、いつか、明日のソフトウェア品質技術をみんなで創りましょう（ちょっと大袈裟ｗ）
