LTとは？

超簡単に言うとミニプレゼンです。

ライトニングトーク（英: Lightning Talks）とはカンファレンスやフォーラムなどで行われる短いプレゼンテーションのこと。様々な形式があるが、持ち時間が5分という制約が広く共有されている。(wikiより)

タイムテーブル

基本、参加者は何かしゃべるということでお願いします。 立ち見はご遠慮ください。

  • 自己紹介
  • 最近したこと
  • 自慢したいこと
  • 興味があること
  • 今年度の目標

  • なんでも可
  • パワポ可
  • デモ可
  • 動画プレゼン可(もし参加出来ない場合、事前に動画データをもらって当日流します)

サークル・団体の紹介は別途時間を設けます。個人の自己紹介の時間には行わないようお願いします。

HDMI端子・VGA端子は使用可能です。それ以外のアダプタを使用する場合は変換器を持参してください。

B1(1年生)で「スライドを使いたいけどパソコンを持っていない！」という方は、当日USBにスライドを保存して持参してください。
