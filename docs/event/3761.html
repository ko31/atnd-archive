= パイ投げ？ ========
童心にかえり、参加者が口々に「見ているだけではまったくその人生を意味あるものにしかねない、あの体感を味わえないよ！」と興奮して話したり、「自分を汚さず観てるだけでは、あの楽しさは分からんよ。」と哲学命題の静かさで語ったりしてしまう器の広いイベント。利口な人たちの枠に入ったやんちゃな精一杯の可愛い悪ふざけだなんて行ってる魂のアーティスト気取りのあなた。やったことあんのか？
パイは個数に限りがあるので出欠や、怪しといった情報もできるだけ欲しいです。これないのはしょうがないのでご協力をー。
・定員は、atndからは15人、ほか名くらいでだいたい２０名強での開催予定で、すでにだいたい声をかけている人でそのくらいになる見込みです。
・レインコート・タオル・お風呂・お手拭き・食事・お酒は運営で用意します。
・準備３０分、実施１０分、おしゃべり４時間です。
= 東京カーネルパニック =====
開催場所は、シェアオフィス的ななにかです。このシェアオフィスは基本的にはプログラミング・デザイン・クエイテイティブを生業にしているフリーランスの人たちの基地になりつつあります。上述のような海外からの旅人の宿になれるアットホームな雰囲気？住み込んだりできるスペースと、遊び心とクリエイティブな仕掛けと調理道具一式が楽しげに、時に混沌と行われております。「最後のオルタナティブスペース」なるか？たぶんなりません。 ※いろいろ書いたけど普通にホームパーティーです。


ENTRANCE FEE【会費】パイ投げ参加者3500円、非参加で2500円。楽しい持ち込み歓迎。プロジェクターやネットワークなどITっぽい機材は一通りあるので即席告知とか上映会・ライブなどもありです。データやCD等を持参ください。「パパになりました」とか「うちの犬が可愛いから見て」とかはよしてね！キリないから！冗談です。そういうのでもいいです。


※参加は、お知り合いとか誰かとなにかのご縁があるといいとは思うのですが、twitterとかtumblrでみかけたよというような方でもかまいません。
