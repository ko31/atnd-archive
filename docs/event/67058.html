　　　　　世界１３６カ国で、約４００万人が勉強する私たちの「日本語」！
　　　　　　　　　世界があなたの「日本語」を待っています！！

「外国と関係がある何かをしたい・・・」と考えたとき、外国語を勉強するのも一つの方法です。

しかし、実は、今日まで皆さんが毎日使ってきた「日本語」を活かす方法があります。
それが「日本語教師」、「日本語ボランティア」です。

私たちは毎日日本語を話し、日本語を理解しているのだから、日本語を教えるなんて、
カンタンじゃない？と思うかもしれません。

しかし、「日本人」であるだけでは、「日本語がわかる」だけでは、
外国人に日本語を教えるのは大変難しいのです。

つまり、「日本語を外国人に教える」ということは、
そのための「知識とノウハウを持つ専門家」だからできることなのです。

この講座で「日本語教育の基礎知識」を身に付け、「あなたの日本語」を活かした国際交流の

一歩を踏み出しましょう！

語学力、年齢、経験は一切問いません。 「興味がある！」が参加資格です。
