description

個人的にメールくれた人含め、定員に達したのでいったん〆切ります。要望によっては枠増やすかもしれません。ビリヤニ協会さんと相談します。

開業にあわせて、屋上でビリヤニパーティ

いよいよ5月22日火曜日に、スカイツリーがオープンしますので、それにあわせていつも夏にやってる屋上ビアガーデンをちょっとはやいですけど、やります。
点灯初日となるスカイツリーのライトアップを眺めながら、ビールの宴。
しかも！今回は日本ビリヤニ協会さんとコラボレーションし、ビリヤニパーティとなります。
うん、きっとビールがうまいぜ。

開催概要

時間

2012年5月22日火曜日　20時開始

場所

東京庚申堂本堂屋上
墨田区向島3-7-4
半蔵門線、浅草線押上駅より徒歩10分ぐらい

備考

ビリヤニ以外は持ち寄りとします。
お酒か食べるもの一品持って来てください。

雨天の場合は、４階で決行します。
屋上じゃなくてもまあ、ビールはいつだってうまいさ。
