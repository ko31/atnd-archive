鯖江マッピングパーティー

オープンデータで注目されている鯖江市にて、OpenStreetMap(OSM)のイベントを開催致します。
OpenStreetMap は、誰でも手軽に活動に参加でき、地図を通じて社会へ貢献できるオープンデータプロジェクトです。
お店やポスト、公衆電話、道路など、様々な地物情報を簡単な操作で追加編集でき、世界中の人々と共有できます。
また、名称だけでなく電話番号や喫煙可否、バリアフリー状態など、あらゆる情報を付加することが可能です。
是非、あなたの知っている情報を地図に追加して下さい。

本イベントでは、鯖江市内を散策し、皆で地図を編集します。

例えばバリアフリー情報に着目しながら町歩きをしていただくと Wheelmap にも即時反映され、鯖江市のバリアフリーマップの充実に一役買います。

Wheelmapの例 福井駅前

前日にもOSMのイベントを開催予定です。OSMの概要について学んだり、懇親会を行う予定です。
こちらの参加も是非御検討下さい。
オープンストリートマップアイデアソン in 鯖江
