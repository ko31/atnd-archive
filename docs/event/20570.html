【勉強会の概要】
XOOPS Cubeを使っている人、もしくはこれから使おうとしている方向けのゆる～い勉強会です。
XOOPS Cube本体やモジュールのインストール方法、設定方法、テーマを変更して
オリジナルなサイトにしていく方法などを経験者が実地で教えます。

お気軽にご参加ください。

※テーマとはWebサイトの見た目（ヘッダの画像やサイト全体の色など）を決定するものです。

【必要なもの】

  • ノートパソコン
    　　実地で教えますので、ノートパソコンを持参してください。
    　　Wi-Fi接続可能なパソコンであれば会場の無線LANが利用できます。
    　　もしくはイーモバイル等の無線インターネットが使える
    　　モバイル端末をご用意ください。

　　また、既にレンタルサーバを契約していたりドメインを取ったりしている方は
　　ノートパソコンからFTPやSCPで繋がるように設定してきて下さい。
　　⇒　まだサーバの契約をしていない方はパソコンでXOOPS Cubeを動かすことができる
　　XAMPP（無料）、MAMPなど（無料）があります。これらを利用する場合は事前にダウンロードしてきてください。

  • ホダ塾ディストリビューション(XOOPS Cubeを使いやすくしたパッケージ)
    　　XOOPS Cubeの本体として、使いやすいパックになったホダ塾ディストリビューションを利用します。
    　　こちらのURLからバージョン1.0.4をダウンロードしてきてください。
    　　URL: http://sourceforge.net/projects/hodajuku/files/Package_hd%20for%20XCL2.1/HD-1.0.4/
    　　⇒　hd_full_1_0_4.zip をダウンロード！

・Windowsを使用している方
　　XAMPPが利用できます。
　　下記のページへ行き、バージョン1.6.8をダウンロードしてください。
　　URL: http://sourceforge.net/projects/xampp/files/XAMPP%20Windows/1.6.8/
　　⇒　xampp-win32-1.6.8.exe
・Macを使用している方
　　XAMPP for Macが利用できます。
　　下記のページへ行き、バージョン～をダウンロードしてください。
　　URL: http://www.apachefriends.org/jp/xampp-macosx.html
　　⇒　XAMPP Mac OS X 1.7.3
