ポイノ（安野太郎：作曲家）と危口（悪魔のしるし主宰：演出家）によるイベント企画。

かつて好評を博した噂の企画「クリエイティブ・クラスターに50の質問」が装いも新たに復活。危口のライブペインティングを添えつつ、新興ソーシャルメディア(Twitter、ユースト)を含む各種メディア技術を駆使しトークイベントの可能性を無駄に拡張。

クリエイティブクラスタ代表岡田智博氏を味わい尽くす濃密かつ無益なひとときをお届けいたします。
