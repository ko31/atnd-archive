現在開催中のウェブアプリケーション開発コンテスト Mashup Awards 6 (MA6) に連動して、
今年より実践形式の開発イベントMA6 Mashup Camp（マッシュアップ・キャンプ）を開催します。

Mashup Campってなに？

Mashup キャンプは、参加者それぞれノートPCを持ち込んで、 みんなでわいわい開発しちゃおうっていうイベントです。
キャンプという名前ですが基本的に日帰りです。
MA5ではHackathonという名前で1度だけ開催しました。

MA6では、たくさんの人に気軽に参加してほしいという思いからCampという名前にし、定期的に開催していきます。

Coolな地図サービスを作ろう

Mashup Camp 第1回目のテーマは地図。
地図を用いたWebサービスは、Mashup Awards1から多数応募されており、世の中にもたくさん存在しています。
今年は一味違うCoolな地図サービスを作りましょう。

当日は、チューター・アドバイザーとして Yahoo! Open Local Platform 、 高速.jp API 、 ライドシェアジャパンあいのり情報API それぞれのエンジニアの方にもご参加いただきます。

うまい使い方や不明点はその場でどんどん解決させて、CoolなWebサービスを開発しましょう。
