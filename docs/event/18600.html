トラ技オフ会｜超お手軽ARMマイコン基板「STM32ディスカバリ活用！」

catch USBに挿すだけで使えるデバッガ付き1000円マイコン基板を使いこなす
description

トランジスタ技術2011年3月号で紹介した「STM32 Value line discovery」（以下STM32ディスカバリ）は，超お手軽なARMマイコン基板です．マイコン・メーカのSTマイクロエレクトロニクスが入門向けに開発したもので，デバッガを搭載しており，USBに挿すだけで簡単にARMマイコン・プログラミングを試してみられます．LEDやスイッチなども備えているにもかかわらずパーツ・ショップなどで1000円強で購入でき，たいへんお得です．

そこで以下のような方に集まっていただき，ワイワイガヤガヤSTM32ディスカバリ電子工作を持ち寄って，活用方法についてお話してみたいです．

・こんなガジェットを作ってみました！
・夏休みにちょっとARMマイコンで節電装置でも作ってみようかなー？
・STM32ディスカバリで分からないことがあるんだけど…

基本的に，参加いただく方には，何かちょっとした発表をお願いします．製作物自慢からちょっとした疑問まで何でもOKです．製造元のSTマイクロエレクトロニクスのSTM32/STM8マイコン担当者も参加されますので，表の質問から裏話まで聞いちゃってください．

もしかしたら参加するといろいろいいことがあるかも…？
