RightScaleユーザ会(RightScale勉強会) – 第2回

Amazon Web Services(Amazon EC2/S3)やRackSpace Hostingなどの
クラウドサービス用の管理ソフトウェアとして最も有名と言えるのが RightScale(http://www.rightscale.com/) です。
併しながら、国内では日本語での機能説明の資料がなく、知名度に比較してRightScaleのSeverTemplateや
RigthScriptを始めとする各機能の利用方法についてほとんど知られていません。

今回は、RightScale社と雲屋株式会社の協力により、RightScaleの概要や機能についての勉強会を行うことになりました。
他にも、今話題のcloud.comが提供しているオープンソースのIaaSクラウド環境構築用ソフトウェア『CloudStack』の紹介や、
11/4-7Cloud Expo(SantaClara,CA)レポートなどを予定しております。

RightScaleのTシャツや帽子もプレゼントしてもらえると思います！
この機会にクラウドについて深く学んでみましょう！
