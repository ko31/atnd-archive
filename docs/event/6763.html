Android デザイン部の第2回ミーティング

　2010年8月20日（金）

　19:00スタート、21:00終了予定

　会場：nifty さん会議室（http://www.nifty.co.jp/company/access.htm）を貸していただけることになりました！

　集合時間　　18:45
　集合場所　　大森ベルポート1F 北口の喫煙室前

　＊注意事項！！！
　　20時を過ぎると、ビルの北東にある社員通用口以外の入館ができなくなります。
　　20時以降に来た方は twitter でハッシュタグ #andc をつけて @yanzm に mention してください。

　初参加の方は Android デザイン部の Google Group

　http://groups.google.co.jp/group/android-design-club?pli=1

　に参加の上、自己紹介シートに記入してください！！

　自己紹介では

　　　・名前（ハンドルネーム可）

　　　・職種（プログラマー or デザイナー or その他）

　　　・プログラマーなら Android で何をしているのか。アプリを作っている のか、個人的にか、業務なのか。

　　　・デザイナーなら、どの分野なのか、イラストレーターとか、UI設計とか。

　　　・Android のデザインに関してどこに興味があるか

　　　・デザイン部でやりたい！！こと

　など書いてください。

　＊＊＊＊＊＊＊＊＊ 重要 ＊＊＊＊＊＊＊＊＊＊

　　当日までにAndroidアプリ 「KsLauncher」を

　　いじり倒しておくこと！！！

　　デザインの観点(UI,UX,配色などもろもろ）から

　　いい点、改良すべき点など意見を各自まとめて

　　おくこと。

　　当日は、4，5人のグループで各自の意見を

　　ディスカッションしてもらいます。

　＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊

アジェンダ

　1. @95kugo さんから KsLauncher のこだわりについて

　2. グループに別れてディスカッション

　3. ディスカッションの結果をグループごとに発表

　4. （時間があれば）Android UI Design Tips http://www.slideshare.net/AndroidDev/android-ui-design-tips　の和訳をしたので紹介します。
