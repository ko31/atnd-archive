講師からのメッセージ

元々、制作業界にいた私は作って納品したWEBサイトが運用されずに放置されているだけの状況を見て、
運用したい想いにかられ、フリーを辞めて客先に押しかけ採用をお願いしたことがあります。

入社後（結果はまたセミナーでお伝えしますが）、社内から挙がってきた言葉は「ホームページから問い合わせって来るんだね」でした。やり方さえ知っていれば、問い合わせやアクセス、ひいては、売上にも貢献出来るんです。
そういったお話をさせていただくセミナーとなっておりますので、ご興味あればご参加をお待ちしております。

お客様の声

  • 初めて自社サイトに携わり、全てが初心者という状態のため、広くご教示いただき勉強になりました。無料で使えるツールなど、すぐにできそうなものは是非実践していきたいと思います。　女性：ECサイト運用担当


  • アクセス解析の重要さを改めて感じました。分析回数を増やしたり、ユーザー視点に立ったコンテンツ、記事作成などで情報提供をしていこうと思う。　男性：自営業


  • WEB担当としてやらなければいけないことがわからず、会社から評価されるためにどうしたらいいのかもわからなかったが実体験を交えてお話いただけたので分かりやすかった。　女性：WEBサイト運用担当


  • 数字の活かし方をもう少し知りたい。PDCAの回し方はある程度わかってはいるが、もっと具体的な実例で話を聞きたい。　男性：ECサイト事業部マネージャー
