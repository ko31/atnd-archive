iPhoneアプリマスターコース、10,000円割引！

2000名が受講した日本ナンバーワンのスマホアプリスクール
キャンペーンコードで10,000円OFF！

全くの初心者、アプリ開発が初めてでも受講OK！休んでも 安心！
振替受講とオンライン講座のサポート付き！受講生から続々とヒットアプリリリース中！

受講生の7割が未経験者で、初心者にもやさしくアプリ始めるならまずここから。
とにかくアプリ開発が楽しいってことをiPhoneは教えてくれます。

毎回一つずつベースアプリを講師の懇切丁寧な説明のもと実際にコーディングし、そのあとは各自でオリジナルアプリに拡張するという形で進行します。全コースを通じて、あなたの作りたいアプリを開発し、App Storeにリリースすることがゴールです。人気アプリの解説講座、アプリのフォーマット事例講座、面白いアプリの企画方法などのブレイクタイム講座も毎回ご用意します。

  • 10コマ分の教室での講義
  • 20コマ分のアプリ開発最新テキスト
  • 20個のアプリのサンプルコード
  • 動画による予習、復習サポート
  • 2000名以上の開発者コミュニティ
  • 現役エンジニアによる講義

少人数生の授業

さいたま校では少人数での開校をしております。分からない事も気軽に講師に質問でき、一人一人に親切丁寧にご指導致します。またご自分のリズムに合わせたペースで学ぶ事が出来ます！

アレンジ方法の提供

20個のサンプリアプリを元にアレンジしたアプリのご紹介と、
そのソースコードも無料でご提供いたします。

  • 01：My Very First Apps（初めてのiOSアプリ）
  • 02:カウンターアプリを作ろう
  • 03:ドル円換算機アプリを作ろう
  • 04:手拍子アプリを作ろう
  • 05:クイズアプリを作ろう
  • 06:ジェスチャーフラッシュアプリを作ろう
  • 07:RSSリーダーアプリを作ろう
  • 08:ブラウザーアプリを作ろう
  • 09:万歩計アプリを作ろう
  • 10:My Tweetアプリを作ろう

プログラミングがいらない！GameSaladaコースの無料提供

GameSalada社が提供している「GameSalada」を使って簡単なゲームが作成できる「iPhoneゲームアプリコース」のテキストをお申し込み頂いた方全員に無料でご提供いたします。ノンプログラミングでアプリを作成することができます！
