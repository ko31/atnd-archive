description

八耐は８時間でゲームやCG、映像を制作し、参加者全員で発表するイベ​ントです。グループ、個人などの制限はありません（制作物はアナログ、デジタル関係ありません）。IGDA Fukuokaの皆さんや本学、周辺の学生さんなど様々な方々が参加し、ゲームやCGなどの作品が発表されています（毎回、３０名程度の参加があります）。 開発用器材は基本的には持ち込みでお願いしていますが、本学の演​習設備を利用して頂くことも可能です（環境への質問はメールでお願いします。メールアドレスは 弘中研究室 を参照して下さい）。 見学だけの参加もＯＫです。入退場も自由となっております。

　９：３０　開場

　　｜

　　　　　　（参加者の方々に作品を２～３分ほどで紹介してもらいます）

制作途中に他の参加者のみなさんと交流できるよう、お昼休憩を入れています。是非、初めて会う方々と交流してみてください！
