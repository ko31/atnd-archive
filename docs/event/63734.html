大好評☆ビブリオバトルを親しみやすくアレンジした会第６弾です！

ビブリオバトルとは本を上限５分間で紹介しあって，みんなの投票でチャンプ本を決める書評ゲームです。

チャンプ本発表者には、豪華図書券をご用意しておりますので、みなさん奮ってご参加ください！
参加者が多いほど図書券の金額が上がっていきます。
ぜひご友人とお誘い合わせの上、ご参加ください。

今回でFrohビブリオバトルは、最終回となります。
ビブリオバトルに興味のある方、気になっている方はこの機会に是非ご参加ください！

ルールは簡単。

1. 発表参加者が読んで面白いと思った本を持って集まる。
2. 順番に一人上限5分間で本を紹介します。
3. みなさんの発表の後に全員で発表した本に関してフリートークをを15分行います。
4. 最後に「どの本が一番読みたくなったか?」を基準とした投票を参加者全員一票で行い、最多票を集めたものを『チャンプ本』とします！

今回のテーマは、「フリー」です！

ジャンル問わず、みなさんの好きな本、おすすめしたい本、みんなに読んで欲しい本などなど、なんでもけっこうですので、あなたのとっておきの１冊を皆さんとシェアしましょう！

Ｆｒｏｈビブリオバトルは、今回が最終回です！！

この機会に、ビブリオバトルに興味があるという方、我こそはという方、学生・社会人問わずご参加をお待ちしております！
