【Ｅ資格の研修に関してのご注意】

1.Ｅ資格は、協会認定のＥ資格研修を修了することが受験要件となっています。

2.弊社も認定申請をしておりますが、本日（H30.3.23）現在、弊社を含め認定された研修機関はありません。

3.試験までの実期間と研修時間の関係で、弊社の研修開催は１回のみとなります。

4.研修にPythonの実機演習を含み、また申請中の研修機関は3～４社程度しかないため、全体としての研修受け入れ人数は多くは出来ないものと考えられます。

5.受講者の知識レベルに相当の分布があることが予想されています。「ＡＩ基礎知識」「Python文法」「ＡＩで利用される数学（微分など）」の力量を事前確認し、対応を図る予定です。

6.正式な研修申込み受付は認定後となります。万一認定されなかった場合は、仮に研修に申込されていても撤回となり無効ですので、あらかじめご了解ください。
