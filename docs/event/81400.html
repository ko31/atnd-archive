「AWSでのストリーミング配信（動画配信）を大幅にコストダウンする方法」

ストリーミングの勉強会Streaming Dayの第６回を開催します。今回は「AWSでのストリーミング配信（動画配信）を大幅にコストダウンする方法」になります。

AWSの従量課金は、"AWS/EC2+WowzaCDN"で大幅にコストダウンする事が可能です。
第６回の勉強会ではその方法をテーマにしました。

ご要望にお応えして「AWSでのストリーミング配信を大幅にコストダウンする方法」をテーマに開催いたします

今、ストリーミング配信のインフラとして、多くの企業様がAWSを利用されています。その構成は「S3 + CloudFront」が主流ですが、データボリュームによって、より大きなコストを負担せざるを得ないことも事実です。

今回の勉強会では、この課題に「EC2 + WowzaCloud」で大きなコストダウンを図る方法についてお話します。

**■従量課金(データ転送の料金)を減らす方法
