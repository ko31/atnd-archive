アイロンビーズでいろんなものをつくってあそびませんか？

アイロンビーズとは？

ワークショップについて

アイロンビーズ・エバンジェリストのヌン先生（@nun_）に、
アイロンビーズの楽しみ方をおそわりつつ、
みんなでわいわいアイロンビーズであそぶワークショップです。

かわいいキャラクターを作ってみましょう！

手ぶらでOK！

材料や道具はこちらで用意します。
※材料費および道具代として800円が必要です。
※飲み物の持ち込み可能です。食べ物はご遠慮ください。

つくった作品はお持ち帰りOK！

おきにいりの作品を作ってもちかえりましょう！

つくりたいものが決まっていなくてもOK！

図案と見本をご用意しています。

注意事項

無断欠席・遅刻・途中入室はお断りします。

18:00開場となります。
19:00に開始しますので、５分前くらいには集合ください。

参加費用

材料費および道具代として800円を当日いただきます。
おつりがないようにおねがいします。
