Docker 再入門 2016 update @長野県塩尻市

- 最新のLinuxコンテナ技術について学ぼう！ -

今から2年前の2014年9月にDocker入門の勉強会を地元で開催した時は、一部の技術者が試していたLinuxコンテナのDockerも、現在では徐々にビジネスの本番環境でも使われるようになって来ました。また、コンテナ技術はLinuxに限らず最新のWindows環境でも採用されるようになって来ました。
2016年もDockerには様々な要素が新規開発や変更もされています。
基本的な事は過去資料のupdateとして、2016年の最新動向を踏まえて再入門の勉強会を再び開催します。

対象OS環境

Dockerは最新の64bit OSが必須です。
LinuxはKernel 3.10以降が対象です。
Windows 10 Proが対象です。
　他のバージョンのWindowsは条件によっては動作可能です。
macOS 10.10 Yosemite以降が対象です。
　MacOSについては別の担当講師が準備しています。
