description

こんな人が集まるイベントです

CakePHPを使っている人、CakePHPに興味がある人を対象にしたイベントです。
最新の技術動向に触れつつ交流を深めましょう。技術系の勉強会に参加した事がない方や学生さんでも気軽に参加してください。連絡先を交換する為の名刺やiPhoneアプリなどがあると良いと思います。
プレゼンテーションやライトニングトーク（5分間のミニプレゼン）を聞いて情報収集をしつつ、CakePHPを実際に業務で使っているような人と交流するチャンスです。

イベントの後半は軽食とドリンクをつまみながらになります。費用の一部として500円〜1000円程度の費用を頂きます。（参加人数次第で変動しますので、当日ご案内します）
学生の方は懇親会費用は無料でOKです。
