78±とは「未来公益」を追求する78会が年に数回開催する大交流会イベントです。
±（プラスマイナス）というのは、78世代＋上下の世代の皆さま、異業種、異世代交流を
コンセプトに、良いご縁の広がりから日本を元気に出来れば考えています。

今回は、その2011年の大新年会。
来る2011年1月16日日曜日、六本木ヒルズの「新」で開催させて頂きます。
少しでもご興味をお持ち頂いている皆さま、これをキッカケに、是非お越しいただければと思います。

この度は、「リゴレット「ダズル」などを展開しているHUGEグループ代表の新川さんのご厚意で、
六本木ヒルズの「新（あらた）」を貸し切り。仲間と串ＢＢＱに来た感じのパーティーをコンセプトに、
新川さんの方で、お料理と飲み物のご準備を頂いています。
美味しい料理と飲み放題で、ゆっくりとお楽しみ、語らいの時間を持たせて頂ければと考えております。
また、新年ですので、、、こういうときしかチャンスの少ない、和服での参加、大歓迎です！

早稲田大学バンザイ同盟による「バンザイ」から華々しくスタート、
美味しいお料理とともに、参加者同士の新たなつながりを持って頂ける趣向もご準備しております。
お一人でも、またご友人とお誘い合わせででも、是非。
