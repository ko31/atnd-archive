description

新年度を迎えたことですし、浮かれ気分で美味しいお酒を呑みましょう。

予算

プレミアム飲み放題コース 4800円(2時間制、おつまみ4～5品＋約150種の酒呑み放題) を頼む予定です。
「普通の店の飲み放題は何であんなに酒が不味いのか。最高にうまい酒の呑み放題をやろうじゃないか！」
という非常に良い心意気を持つお店なので、そこを存分の楽しみましょう。

備考

・「お酒はそんなに強くないけど、色々飲んでみたい」って方や、「上京してきたばかりで飲み仲間がいない！」って方も大歓迎です
・一応高専カンファ界隈中心なので、幹事もしくは参加者と面識ない方は申し訳ないですがご遠慮ください。
