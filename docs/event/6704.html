description

（特）長井まちづくりNPOセンターでは、本年度、やまがた社会貢献基金「ネッツ山形『自然を大切に』活動支援事業」に取り組むこととなりました。これはネッツトヨタ山形様のご協力のもと、長井市内を流れる撞木川の周辺環境を、子ども達も参加して楽しみながら保全活用することを目的に環境整備活動や体験教室を行うものです。

　撞木川は、今春、自然環境に配慮した工法で改修されました。そのお披露目も兼ねて、来る７月31日（土）に川の周辺をろうそくでライトアップする「撞木川キャンドルナイト」を実施致します。

　当日は大町弁財天まつりの開催日で、子供達の獅子が撞木川に沿って渡御する一場面もあり、キャンドルナイトで演出させて頂きます。

　一夜限りのイベントですが多くの方々にご覧頂きたいと思います。
（雨天の場合、８月１日に変更）
