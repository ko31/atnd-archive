クリスマスまでのアドベント期間，日替わりで R に関する記事を公開していきましょう！ クリスマス当日まで毎日 R に関する新しい記事が産まれる素敵なイベントです。

R Advent Calendar は昨年も行われ，大盛況でした。昨年度の R Advent Calendar の様子は 昨年のカレンダー をご覧ください。

ルール

  1. まずはこのイベントに参加しましょう。
  2. 参加した順番に日替わりで R に関するブログ記事を書きます。最初に登録した人が 1 日目，次に登録した人が 2 日目，…という具合に進行します。
  3. ブログ記事を書いたら，このページのコメント欄にその URL を書いてください。 HTML 形式でリンクを張ることもできるようです。

そもそも R って何？

R はフリーでオープンソースな統計解析環境およびその記述言語です。 R をはじめて知ったという方は，ぜひ公式サイト www.r-project.org (英語) からダウンロードして触ってみてください。

どんな内容を書けばいいの？

  • R はじめました
  • R を使う前に注意すべきこと
  • R でこんな解析が簡単にできるよ
  • R でこんなグラフが描けたいんだけどどうしたらいいの
  • R の example がわかり辛いのでいろいろ試して理解を深めた
  • R を使うには Windows, Mac OS X, Linux どれが良いの？
  • R はオワコンだから○○やろうぜ！

…などなど，とにかく R に触れる内容なら 何でもあり です。気負わず楽しく。

もちろんこれら以外のトピックも大歓迎です。 昨年のカレンダー も参考にしてみてください。

ブログがない

ブログというのはあくまで 1 つの形式であって，本当にブログである必要はありません。例えば RPubs という R に特化したドキュメント公開できるサイトの利用がオススメです。

参加したいけど定員オーバーしちゃった…

定員オーバーした場合でも参加可能です 。 Advent 期間ではありませんが， 26 日以降の日程で割り振ります。クリスマスを越して，ついでに年も越したらいいんじゃないかな！ 躊躇せず参加してください。

逆に定員に達しない場合は？

人数が集まらなかった場合は，有志がコメント欄に「○日は自分が書く」と宣言して書いてくれると嬉しいです。それでも足りなければ「オレ達の R は始まったばかりだ！」と， R のこれからの発展を願いつつ終了いたします。
