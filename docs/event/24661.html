description

2011年、標的型攻撃は国防に関わる大手製造業や公的機関に止まらず、ITに強いはずの大手ゲーム企業などにも及び、攻撃の対象は価値ある情報を持つ全ての企業になりうるという現実を世間に知らしめました。そして2012年、年初から標的型攻撃は引き続きニュースを飾り、今後こういった傾向が増すことはあれど、下火になることは無いことが示されました。

このような状況に対し、様々に提案されるソリューション・セミナーを見ても、各社の提案はまちまちであり、現時点でこの攻撃に対する決定打は無く、結局は社員の啓蒙活動と攻撃後の検知「出口対策」の運営のみが現実的な対策のようにも見受けられます。
しかし、目の前に経営リスクが迫る中、またITの高度化とコスト削減が同時に求められる中、企業のシステム担当部署にその初期投資と運用のコスト(手間)をかける余地は残されているでしょうか？

本セミナーでは、既存のネットワーク構成には手を加えないまま、セキュリティ運用を専門家にアウトソーシングすることで、すぐにでも始められる画期的な標的型攻撃対策を、具体的な５つの理由と共にご提案します。デモを交えた分かりやすい内容となっておりますので、是非お誘い合わせの上ご来場ください。
