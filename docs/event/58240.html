「ホントに数時間で3Dコンテンツを作れるようになった！」

jQueryさえわかればたった1日の講習でWebGLクリエイターになれる驚異の勉強会【jThree Class】と初心者でも使いやすいと話題のクラウドサービス【 Microsoft Azure 】が夢のタイアップ！！！
ブラウザのJavaScriptだけで簡単に扱えるAPIとWebGLを組み合わせた作品をAzureでウェブ公開するところまで無料でできちゃいます。
第1回のツイートまとめ

参加条件はただ１つ、「jQueryが好きである」ということです。
jQueryが好きで最低限、
$( “div” ).animate( { width: “500px” }, 1000 );
の意味さえわかれば技量に関係なくjThreeでWebGLを動かせるようになります。

タグで描く。jQueryで動く。世界初インラインWebGLライブラリ『jThree v2』

さらに今回は以下のAPI提供企業からもエンジニアが来てくれるので「WebGL+API」をみっちり学ぶことが出来ますよ！

企業名 API 活用例
日本マイクロソフト Kinect for Windows v2 リアルタイムにモーションを検知して3Dモデルを動かす
テクニカルロックスターズ milkcocoa 世界中からリアルタイム参加が可能なブラウザFPSゲーム
エクシング IMINOS ユーザーの入力内容に応じて3D空間をデコレーションする

さらに今回は CodeIQ さんがお弁当を、マイクロソフトさんがドリンクを提供してくれることになりました！
高層ビルを出て品川駅周辺でお昼を取るのはなかなか苦労するところなのでとても嬉しいですね。

ランチスポンサー

そのうえなんと！ハコスコさんからお手軽VRビューワー「 ハコスコ 」をドドーンと50個提供していただけることに！！！
サクッとスマホVRコンテンツが作れます。
