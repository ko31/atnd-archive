モノづくり・ガジェットが好きな人必見です。

ガジェットの祭典であるWEEKEND BASE AKIHABARAでは、BASEを始めとした「個のモノづくり」を応援するプラットフォーマーたちが一同に介するトークセッションをご用意しております。

最新のモノづくり市場から、人気のガジェットの動向までの情報を一気にキャッチできるチャンス！

登壇するのは最先端のモノづくり応援プラットフォームたち

今までの企業、大型マシーンを使ったモノづくりが3Dプリンターやクラウドファンディングの登場によってどのように個のモノづくりに変化をしてきています。 起業支援、クラウドファンディング、ネットショップ機能提供のプラットフォーマーであるGUGEN、TSUMIKII、ABBAlab、BASEがそれぞれのモノづくりのあり方について最新の情報を交えながら熱く語ってい来ます。
