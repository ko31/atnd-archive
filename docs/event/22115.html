[地下鉄新栄徒歩5分]
地下鉄新栄から広小路通りに出て広小路葵の交差点を渡り広小路通り沿い栄方向に向かい歩く。CBC中部日本放送の手前角(広小路新栄1丁目)を左に曲がりすぐ1本目をまた左に曲がる。そのまま30m程歩くと左側に第2Mビル(1階が空海という中華レストラン)があり、そこの2Fがシバで6Fがクラブブッダとなります。

[千種方向からお車の場合]
広小路通りを千種方向から来た場合、CBC中部日本放送を左手にこえた1本目の角(福文堂書店)を左折する。すぐ1本目をまた左に曲がる。そのまま東にまっすぐ走行して路地を渡り30m程左側に第2Mビル(1階が空海という中華レストラン)があり、そこの2Fがシバで6Fがクラブブッダとなります。

[栄方向からお車の場合]
広小路通り「東新町」の交差点を南へ、1本目を左折して２本目の路地を左折し、1本目を右折したすぐ左側に第2Mビル(1階が空海という中華レストラン)があり、そこの2Fがシバで6Fがクラブブッダとなります。(CBC中部日本放送から東に30m)
