Atom in Bit outをプロトタイプで体感するイベント

総合研究所の横浜ラボ(通称 浜ラボ）は、ITを使った車の革新を研究している小さな研究室です。今回、この浜ラボでは研究中の革新的なプラットフォームを使ったハッカソンを開催します。
WebAPIを通じて車を自由にプログラムできたら、人と車の関係がいったいどういう風に変わるのか、実際にプロトタイプを作って語り合いましょう。

日時：2016年9月 9日(19:00-)チュートリアルセッション

参加対象：WebAPIが使える人、できればDBも使える人。９月９日チュートリアルセッションだけの参加も可です。
申し込み方法： （本Webサイト）
その他：当日はささやかですが、軽食をご用意いたします。

内容：
WebAPIと簡単な操作デバイス、ナビを模したディスプレイ、を使って
限られた時間でアイデアとそのプロトタイプを実装します。
インターネット環境、サーバ環境は用意させていただきます。
開発マシンはご持参ください。
まだだれもつかったことがない車のAPIに触ってみたい方、是非参加ください。
※このプラットフォームはカーライフのカスタマイズに関するものです。
車の制御や走行に関するAPIには触れません。
