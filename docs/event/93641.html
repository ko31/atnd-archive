■講演内容

3万6700人に選ばれたフリーランス専門エージェント
『レバテックフリーランス』が主催する連載キャリアセミナーの第6弾。

今回は、IT未経験から正社員を経て、
現在フリーランスとして活躍中のエンジニア阿部氏にご登壇いただきます。
一度はエンジニア以外へのキャリアチェンジも考えた阿部氏が
なぜフリーランスを選んだのか？
フリーランス初心者が持ちやすい疑問や悩みについて
ご自身の経験も交え、等身大でお話し頂きます。

当日は懇親会も設けておりますので、登壇者に直接質問や相談することも可能です。

・今後のキャリアについて不安がある…
・フリーランスに興味があるが具体的にどうしたらいいの？
・実際にフリーランスとして働く人からアドバイスがほしい！

…などなど、フリーランスに興味・関心のある方はもちろん、
今後のキャリアを検討したい方は必見です！

《トークテーマ》
・正社員からフリーランスに転向した意外なきっかけとは？

■講演者プロフィール

◆阿部 龍治 氏

フリーランスエンジニア。
文系出身かつIT未経験のフリーターだった26歳のときに、
ITスクールでJavaを勉強し、システム開発会社に就職。
ネットバンクシステムの開発～保守、
金融機関向けシステム開発支援ツールの開発プロジェクトに携わる。

33歳でチームリーダーやサブリーダーとしてマネジメントを1年経験。
その後、自身で主体的にキャリアを選択したいという思いで
フリーランスエンジニアに転身することを決意。

2016年11月にレバテックを通し、フリーランス契約で独立系SI企業に参画。
現在は地図データの生成を行うアプリケーションの開発に携わりながら、
ITスクールに通い技術力アップにも努めている。
