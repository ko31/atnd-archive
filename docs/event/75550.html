Live2Dセミナー　中級コース

Live2Dエディタを学ぶ中級者向けの講座になります。
一通りモデルを作ったことはあるけど、更にクオリティアップしたい方やLive2Dを使ったお仕事に興味がある方にお勧めです。

本セミナーは、公演中のディスカッションも含み、かなりラフなスタイルで進行させて頂きたいと思います。
わからないことや、日ごろ聞いてみたかったことなど、どしどしお受けしたいと思いますので気軽にご質問いただければと思います。
更には、今後のエディタ開発への新機能のご要望やご提案もぜひお聞かせいただきたいと思います。
※後半にディスカッションの時間を設けております

なお、今回は中級コースという事で、ある程度のLive2Dに関わる知識が要求されます。
使い始めて間もない方や、まだちょっと自信がない方は次回「初級コース」も準備しておりますので、そちらへのご参加をご検討頂ければ幸いです。

なお、多くの方に参加して頂きたい為、ATNDキャンセル数が多いユーザーさんに関しましては参加申請をお断りする場合がございます。
あらかじめご了承頂ければと思います。

懇親会

セミナー終了後に近くの会場で懇親会を予定しております。
Live2Dに関わるいろんな方々の情報交換や交流の場として有意義な時間にしていきたいと思いますのでご興味のある方はぜひご参加ください。

※懇親会会場の確保の為、3月27日で懇親会参加受付を終了させて頂きます。
