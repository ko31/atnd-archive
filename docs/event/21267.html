TDD Boot Camp 福岡 2

TDD Boot CampはTDD（テスト駆動開発）を実際に体験しながら学ぶイベントです。
TDDBCは全国各地で開催されており、福岡でも2011/03に開催されました。
TDDBC福岡2 Day2ではリファクタリングかレガシーコード改善に焦点をあてる予定です。

2011/11/27(日) 10:00〜18:00

09:30～10:00 開場

11:10〜12:00 デモ

17:30〜18:00 ふりかえり,総括

午後の演習に関して

  • 2人1組で行います。
  • 1台のPCを使って、ペアプログラミングで取り組みます。
  • PCは各チームに1台あれば良いので、全員が持ってくる必要はありません
  • スタッフ側でサポート可能な言語はJava(JUnit)、Ruby(RSpec)、JavaScript(QUnit) です。
    • 当日の受付時に、改めて第２希望までをエントリーシートに記入していただきます。
    • ペアが組めない場合などは、希望に添えない可能性があります。
    • 上記以外の言語でも、自分でテスティングフレームワークの導入が出来て、参加者間でペアを組めるなら使用可能です。
  • 各言語、基本構文・ファイルI/O程度を扱えるとスムーズに演習が可能です。
  • 会場にネット環境がない(後述)ので、PCを持参する場合は可能な限り前日までに環境構築しておいてください。

詳細は随時TDDBCポータル TDDBC福岡2にて更新します。
