アメリカやヨーロッパの各地で注目されている、JavaScriptのリアルタイムウェブプラットフォーム node.js のハッキングをしている皆さん、楽しく飲みましょう！

（まだ触れていないけどもっと知りたい開発者も、興味があればどうぞ。）

１０月のnode.js飲み会は大成功でしたので、これからも定期的にnode.js関係のイベントを開催したいと思います。今回の場所は渋谷のセンター街にある だいぶつころころ という「隠れ家和食」です。コースは飲み放題付きで４５００円前後になります。

あと、補欠者が寂しいほど多かったので、人数を前回からほぼ２倍にしました。ですが、参加をしたい人がまだ残っているみたいなので、参加できなくなった参加者がいれば早めのキャンセルをお願いします。

不明な点があれば幹事の ジェッド に連絡してください。よろしくお願いします。
