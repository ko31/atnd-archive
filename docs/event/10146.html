寺田 佳央

エンタープライズ Java をリードする GlassFish v3.1
新しい飼育・繁殖方法のご紹介

GlassFish v3.0 (Java EE 6 の参照実装) のリリースから 1 年が経過し、今まさに v3.1 がリリースされようとしています。GlassFish v3.1 では WebSocket への対応、配備時のアプリケーションバージョニング等様々な機能が盛り込まれていますが、最大の変更はクラスタ機能が組み込まれた事です。SSH を利用したクラスタノード管理で、GlassFish を大量繁殖させる事ができるようになり、飼育 (運用・管理) も簡単にできるようになります。
本セッションでは GlassFish v3.1 で提供されるクラスタの新機能をデモを交えながら分かりやすく紹介します。

20:05-20:25

櫻庭 祐一

Oh, JavaFX 2.0!!

JavaOne 2010 にて発表された JavaFX 2.0。今までの JavaFX からの大変身を遂げてしまいました。なんと JavaFX Script のサポートが終了してしまい、Java から使用する新しい GUI ライブラリとなりました。本セッションでは、JavaFX 2.0 がどのようなものか、どこに行こうとしているかを考えてみたいと思います。

20:25-20:45

中野 靖治

ポスト JavaFX Script? ～Groovy の Builder とは～

Groovy に標準バンドルされている SwingBuilder を引き合いに出しつつ、Groovy における DSL 実装の一形態である Builder という仕組みについて説明します。
そして、JavaFX に対して DSL 記法を提供する試みについても簡単に紹介したいと思います。
ポスト JavaFX Script をお探しの方へ。
