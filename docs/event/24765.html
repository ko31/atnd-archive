BeagleBoard について

　BeagleBoard は Beagoeboard.org が開発する ARM Cortex-A シリーズのCPUを搭載した小型組み込みボードです。
BeagleBoard の改良版である Beagleboard-xm が現在の主流であり、CPUがARM Cortex-A8 が600MHzから1GHz、RAM容量も256MBから512MBへとパワーアップしています。また、LANやUSBポートなどのペリフェラルも-xmの方が豊富になっています。（現在の最新版は、BeagleBoard-xm Rev.C4 です。）
この勉強会でも主に Beagleboard-xm を扱います。　多分。　きっと。　わかんないけど。

Beagleboard は海外電子部品販売業者 Digi-key か日本の輸入代行業者から購入可能です。

どちらのボードも Ubuntu や Android 、MeeGo といったLinuxディストリビューションをポーティングして遊ぶことができます。
