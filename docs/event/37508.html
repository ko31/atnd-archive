description

【講座のポイント】
　世界の市場はかってのプロダクトアウトの時代からマーケットインの時代を過ぎ、いまやコンセプトアウトの時代とさえいわれています。製品やサービスを通し、顧客に価値を提供する時代、顧客に感動を提供する時代と言われています。私たちのお客様は誰かを常に問いかけ、そしてお客様が真に求めているものは何かを考え、それを提供できたときにマーケットは開けます。そのために、マーケティング理論の理解とその実践は必須となります。
　本講座では、自社の状況を見直すとともに、変化していく経営市場に適応した自社の進むべき道をあきらかにし、業績を伸ばすための実践可能で、かつ具体的なマーケティング戦略策定の方法と戦術立案の方法を学習し、失敗の無い新事業・新商品開発が出来るよう指導いたします。

【プログラム】
１．マーケティング基本
　1-1.顧客満足の仕組みを知る
　1-2.顧客の言葉から、真の要求を知る
　1-3.顧客要求への対応から顧客満足度を把握する
　1-4.自社の強み、弱みを分析してみる
　1-5.時代の変化への対応度合いを確認してみる
　1-6.分析手法のいろいろ
　1-7.顧客満足向上をベースとした自社課題分析
　1-8.課題解決に向けた戦略立案方法

２．実践ポイント
　2-1.課題を設定して、目標を設定する
　2-2.実践を可能にするプロジェクト運営の方法
　2-3.目標達成のための解決ステップ
　2-4.解決策の創出と評価の方法を学ぶ

３．まとめ　演習
