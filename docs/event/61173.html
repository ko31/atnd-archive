好きな風景、好きな物、ふとした瞬間、好きな人、いろんな場面で偶然撮れた写真をニットで編んだらどうなるだろう？
改造した電子編み機を使って写真を編むワークショップをします。デジタル編み機を紹介し、写真をニットサイズに変換するレクチャー、A4サイズくらいを実際に編み機で作ります。

日時　2014．2．28　各回5 名、計10人
1部　10:00～13:00　定員5 名
2部　14:00～17:00　定員5 名
・1，2部で同じ構成のワークショップを行います。
※こちらは1部の参加フォームになります。

会場：コーポ北加賀屋 大阪市住之江区北加賀屋5-4-12
地図 : http://fablabkitakagaya.org/access_page/
大阪市営地下鉄四ツ橋線「北加賀屋」駅4番出口より徒歩10分
http://coop-kitakagaya.blogspot.jp/

講師 : 宮田明日鹿

＜ワークショップメニュー＞
1. 編み機紹介と、ワークショップの流れを説明
2. ペイントで写真をニットサイズに変換デモ
参加者様の写真は事前にメールにて送付して頂きこちらで加工して当日はスライドで変換後の写真を順に紹介致します。（ 画像を二値化後、解像度を100ピクセルx100ピクセルに変換することをここではニットサイズに変換すると意訳します。元の画像サイズから、解像度を下げることで、元の画像とは違った味になることを楽しみます。)　　
3. ニットソフト、Asuamiソフトにデータをインポート
4. パソコンから編み機にデータ送る
5. 二色の糸を選ぶ
6. 編み機で写真を編む
7. 仕上がった作品の撮影（終了時間が少し押す可能性があります。）

費用　￥4,000円(当日受付、材料費込み)
必要機材　なし

参加者必要スキル

2.Emailにて写真データを送付出来る方。（開催日前日までに送付願います。）
