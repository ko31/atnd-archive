Niigata.pm tech talk #1

Niigata.pmとは、新潟県内のPerlユーザによるコミュニティ活動です。今までは「居酒屋LT」や「居酒屋meet up」などの活動を行ってきました。Perlをお仕事で、趣味で使っているひと、Perl使ってないけどPerlに(あるいはPerlプログラマに、その周辺技術に)興味があるひとは是非ご参加ください。

tech talk #1では何をやるの？

主に新潟に住んでいるのPerlユーザやPerl技術に興味のあるひとたちが集まって、それぞれがPerlやその他の技術について発表をします。コミュニケーションの促進のため、参加する人は是非発表もお願いします。発表のレベル、内容は問いません(なるべくPerlに即した発表だとうれしいな、くらいです。過去にはcssのことを話してくれた人もいます)　。基本的に初参加の方は大歓迎されますので、「アウェイ感」のようなものはないと思います。とにかくdisるみたいな怖いひとはいないので、是非ふるってご参加ください。懇親会のみの参加もOKです(その場合はコメントに「懇親会だけ出たい」という旨を記入してください)。

タイムテーブル

15:00 ~ 18:00 で発表
18:30 ~ おいしいお店にいきましょう

持ち時間はひとり20分程度というあたりで。5〜20分くらいでなんでもいいのでしゃべってください。

場所

貸会議室にいがた （新潟市中央区米山４丁目２０番１１号）で行います。新潟駅徒歩10分くらいです。黄色の「のぼり」が目印だそうです。
