RICOH THETA Advent Calendar 2015 ( http://qiita.com/advent-calendar/2015/theta ) ができて、たくさんの人に参加いただき、興味深いネタも多く投稿されていて毎日楽しく過ごすことができました。作成者の @itachin さんをはじめ、参加者のみなさまありがとうございました！
せっかくこれだけネタが集まったので、実際に見たり交流したりしてみたいと思い立ち、ゆるくLT大会をやってみようと企画してみました。

主催や会場の提供はリコーITソリューションズ(株)の社員であります設樂ではありますが、とくにTHETA公式イベントという位置付けではなく、有志のイベントとなります。その点はご了承ください。
