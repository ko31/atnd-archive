description

Adobe Illustrator をWeb制作用にカスタマイズするワークショップ。
情報交換をしながら、デザインの効率化を図りましょう。

ワークスペース、プラグイン、文字/段落スタイル、シンボル、スウォッチ、グラフィックスタイル、ブラシ、アクション、アピアランス、プラグイン etc.

Adobe Illustrator の初期設定を整えておくことで、Webデザインの効率をアップすることができます。

今回の勉強会は、参加者各々の各種設定を見せ合い(可能なら素材の交換なども)、互いに参考にしながら制作環境を整えるワークショップです。

勉強会終了後、セミナールーム近くのオープンスペースで茶話会をします(無料、参加自由)。
