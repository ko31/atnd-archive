10月1日に許可申請を頂いて営業開始する予定の「就労支援事業所ディープレイス」の見学会を行います。

10月14日・21日・28日の13時～、16時～、19時～から行います。無料で見学できますので、ご興味のある方はぜひ起こしくださいませ。

【就労支援事業所ディープレイスとは】

ディープレイスは発達障害・精神障害の方を中心に、就労のためのサポートを行うスクールです。「放課後等デイサービス」と「就労移行支援」の2つのサービスを提供する多機能型事業所で、学生の方から社会人の方まで広く利用することが可能です。

障害福祉サービスの受給者証をお持ちの方でしたら、誰でもご利用可能ですので、お気軽に見学にお越しください。

【就労支援事業所ディープレイスの特徴】

・ディープレイスの放課後等デイサービス
「就労・進学を考える高校生が使いやすい設計をめざしました」

  • 月曜日～土曜日まで開所
  • 21時まで開所と学習塾と似た営業時間帯
  • IT・WEB業界で働く施設長から学べる

・ディープレイスの就労移行支援サービス
「福祉業界への就職を目指せるカリキュラムを設計しました」

  • IT・WEB業界で働く施設長から学べる
  • 13時から開所なので、朝に弱い人でも利用可能
  • 障害福祉サービスで就職する、新たな可能性を提供

  • 9月21日（金）13時～、16時～、19時～
