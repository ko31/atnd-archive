description

「お金はそこそこ稼げるようにはなったが、
果たしてこのままで順調にやっていけるのか、
将来は不安なままだ・・・」

とお悩みの方は、高年収の方でも意外と多いものです。

これをお読みのあなたもそうなのではないでしょうか？

そんな中、投資に回せるお金もできたことで、
もう一つの収入の柱を作ろうと、
株や投資信託、不動産といった資産運用を考える方も増えてきています。

国自体も、

「貯蓄から投資へ」

というスローガンのもと、NISA等の
優遇措置を受けれる投資制度を作り、
資産運用をどんどん促す傾向にあります。

ただ、そんな状況にあっても、
もしあなたが、

「投資なんてこれからもやる必要は全くない」

とお考えの方なら、もうこれ以上この文章は
お読み頂く必要はありません。

では、そうではなく

「今のままじゃ不安なので、収入の柱を増やしたい」

「自分が働くだけでなく、お金にも働いてもらえたらどんなに楽だろう」

「資産運用についても本格的に取り組んでみたい」

とお考えの方なら、今回のこの機会は
大変有意義なものになるかと思います。

実は、こういう風に資産運用も考えてらっしゃる方は、
そこから更なる富裕層へステップアップできる
高い可能性を秘めています。

大富豪の実態や行動について
分析しているあるデータアナリストによると、
年収が高い人であればあるほど、
自分が働いて稼ぐ以外に
資産運用でお金にも働いてもらっているという
傾向がどんどん強くなっていってるとのことです。

しかし、
それでも資産運用には
なかなか足を踏み出せないという方

「リスクは怖いのでできるだけ安定的な資産運用方法を知りたい」

という方は多いでしょう。

そこで、そんな方々のご要望にお応えするために、
実際に数々の富裕層をコンサルティングし、
自身も豊富な投資実績を持つ、
お金のプロ「ファイナンシャルプランナー」さん
を講師に迎え、その資産運用術を公開する
というセミナーを開催することに致しました
