【講座のポイント】

　防大で40年間研究してきた圧電セラミクスの開発、プロセス、応用化に関する全てを熟知した講師が、圧電セラミクスの破壊と劣化のメカニズムから寿命時間の評価法を解りやすく講演する。実験結果をもとに説明するので簡潔で理解し易い内容である。

　特に、圧電セラミクス特有の劣化特性では空間電荷効果、内部応力、非線型圧電効果などが寿命時間に影響することはあまり知られていないが、研究成果とモデルを用いて詳しく講演する。

　最後の質疑応答では圧電セラミクスの全般に関する質問も歓迎する。圧電セラミクスの応用化に携わっている研究者には、ぜひ聴講して欲しい。

【プログラム】

１．圧電セラミック素子の破壊条件と寿命時間の評価法
　　1.1 圧電セラミック素子の駆動法と破壊モデル
　　1.2 電気的破壊と劣化のメカニズム
　　1.3 機械的破壊の劣化モデル
　　1.4 加速読験による寿命時間の評価法

２．圧電セラミック素子の絶縁的破壊と微細構造
　　2.1 製造条件が影響する絶縁破壊特性
　　2.2 絶縁破壊特性に及ぼす微細構造の影響

３．圧電セラミック素子の誘電的破壊
　　3.1 不純物，イオン，酸素欠陥の移動と誘電特性
　　3.2 誘電的劣化に及ぼす結晶粒界の影響

４．電気的劣化に及ぼす空間電荷の影響
　　4.1 強誘電体の空間電荷モデルと発生メカニズム
　　4.2 圧電セラミクスの絶縁破壊に及ぼす空間電荷の影響
　　4.3 圧電性およびエージング特性に及ぼす空間電荷の影響

５．圧電セラミックス素子の機械的破壊と内部応力
　　5.1 機械的強度に及ぼす微細構造の影響
　　5.2 圧電セラミクスに発生する内部応力と機械的強度

６．圧電セラッミク振動子の非線形現象
　　6.1 圧電振動子のハイパワー励振時の挙動
　　6.2 圧電素子の圧電線形現象と非線形現象
　　6.3 機械的破壊に及ぼす非線形現象の影響

７．積層圧電セラミック・アクチュエータの寿命時間の評価法
　　7.1 寿命時間の加速試験法と評価法
　　7.2 圧電アクチュエータの寿命時間の概算
　　7.3 ＡＣおよびＤＣ印加時の破壊メカニズム

８．圧電トランスの寿命時間の評価
　　8.1 圧電トランスの性能に及ぼす材料定数の影響
　　8.2 圧電トランスの破壊モデルと寿命
