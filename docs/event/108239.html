クリエイター、エンジニアの独立、起業に大切なこと勉強会


～独立・起業で失敗しないためにこれだけは知っておこう！～

申し込みページ → https://sfida-workshop10.peatix.com/

クリエイター交流会【クリのみ】が主催する、
デザイナー、エンジニア、フォトグラファー、ライター、 イラストレーター、プランナー、制作などものづくりをするクリエイターのためのクリエイター勉強会（第10弾）です。

◆目的

クリエイターが将来、独立（フリーランス）・起業（法人化）する時に、無駄な失敗をしない、損をしないために何が大切なのかを学ぶ勉強会です。

　・独立前（会社員の時）に絶対しておいたほうが良いこと
　・フリーランスになった時にどんな事がおこり、何をしなければいけないか
　・すでにフリーランスの人は今からでも遅くない知っておくと良いこと
　・法人化した後に待ち構える落とし穴を避けるには

などを学びます。

◆学習内容

この先、クリエイターに訪れる可能性の高い人生の転機、
「独立（フリーランス）」そして「起業（法人化）」です。
特にクリエイターは終身雇用が少ない職種なので結果的にそうなることも多いと思います。

とはいえ、よくわからない初めてのことであり、どんなことを準備すればよいのか、どんな落とし穴があるのか不安だと思います。
講師はクリエイターであり実際に　インハウスデザイナー → 独立・フリーランス → 法人化（経営）の経験から、また、現在はクリエイターの育成、経営者スクールの講師をしており、数多くの受講生の失敗と成功を見てきた経験から、クリエイターの独立、起業に大切なことをお伝えします。

◆こんな人にオススメ

　■現在インハウスクリエイター
　・いつかはフリーランスになって活躍がしたいと思っている人
　・将来、独立するつもりだけど、今からどんなことをすれば良いか知りたい人
　・今は独立も起業も興味ないけど、どんな世界かのぞいてみたいと思っている人
　・今の職場がちょっと嫌だなあと感じている人

　■現在フリーランス
　・フリーランスになってみたけど、専門以外の業務がよくわからない人
　・将来、会社設立（法人化）したい人

　■起業１〜２年目
　・従業員を雇うことにとても不安がある人
　・5年後、10年後が想像できなくて不安な人
