マサラ料理教室とは

マサラ料理教室は「スパイス料理をもっと気軽に簡単に」を目指した、初心者やお子様でも参加できる、簡単で楽しく美味しいスパイスワークショップです。
お子様に限らず、大人の方でもOKなのでお気軽にご参加下さい♪

今回作るカレーは？

今回はスパイスセットで販売中の「かんたんチキンカレー」「キャベツのクミン炒め」「ラッシー」を作ります。
近所のスーパーで手に入る材料で簡単に作れるレシピです。
調理のコツを聞きながら、みんなで楽しく作りましょう。

  • 「かんたんチキンカレー」は基本中の基本。お子様向けに辛味無しのマイルド仕上げにすることも、大人の酸味と辛味仕上げにすることも可能です。このレシピをベースに、野菜カレーやココナッツカレーなど色々アレンジすることができます。
  • 「キャベツのクミン炒め」はニンニクとクミンが香るスパイス野菜炒め。歯ごたえと香ばしさがたまらない、カレーの付け合せに最高の一品です。大人ならビールのアテにも♪
  • 「ラッシー」はミキサーを使うのが一般的ですが、空ペットボトルを使ったレシピをご紹介します。
