UnrealFridayNight -1st-

別名：UE4に関係したことを呑みつつ語って交流しよう！会

内容

以下のような、個人個人での交流を助ける場として、今回の企画を立ち上げました。

  • 技術的な情報交換
  • 作成物の公開・試遊

技術的な情報交換

正直なところ、普段飲みつつやる人はやってるじゃねーかっていう話ですが、 こういう機会でしか出来ない、UE4の濃い話をする手助けが出来ればと考えています。

作成物の公開・試遊について

UE4で作成中のゲーム・映像などのコンテンツについて意見が欲しい方、 第5回UE4ぷちコンで作成したものを遊んでもらいたいという方など、積極的に活用いただければと思います。

その他

  • 基本的には立食形式(椅子も用意されている)を考えています。
  • ぷちコンに参加された方々の投稿された動画を大画面で流す予定です。
  • 一度始まれば、入退場は自由です。
  • こちらでは、以下の食事・飲み物を手配させていただきます。
    • ピザ
    • パエリア
    • ビール
    • サワー
    • ソフトドリンク

飲食物の持ち込み歓迎！！

申し込む際にアンケートをとっておりますので、そちらから、どちらにするのかを選択してください。

  • 2500円：食事等持込無し(ドリンク代、食事代、スペース利用代込み)

最後に

ぶっちゃけて言うと、各種UE4のイベント後に行われる懇親会に、各々の制作物の公開・試遊を合わせたようなイベントです。 海外のMeetupイベントや、Demo体験会みたいなことが出来たらと思い、今回企画させていただきました。 皆様の交流等の手助けが出来れば幸いです。 当日は楽しんでいきましょう！私も楽しみです！
