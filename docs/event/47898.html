うずらインキュベータ

豊橋にて C#.NET および Ruby の勉強会を開催致します.

日時

  • 2014年2月26日(水) 19:00 – 21:00

場所

  • 豊橋市民センター カリオンビル 中会議室
    • ※会場に駐車場はございませんので、お車でお越しの場合は近隣の有料駐車場をご利用下さい

概要

豊橋の開発会社（株）マップクエストと（株）ウェブインパクトが合同で勉強会を開催致します.

時間 発表者 タイトル
19:00 – 19:05 主催者 主催者からのご挨拶
19:05 – 20:00 奥山様 第一部 : Xamarinによるクロスプラットフォームモバイルアプリ開発
20:00 – 20:45 ウェブインパクト 小峰様 第二部 : Code Kata（Rubyプログラミング演習）
20:45 – 21:00 主催者 主催者からのご挨拶＆片付け

第一部について

  • 出演
    • 奥山様
  • 概要
    • Xamarinによるクロスプラットフォームモバイルアプリ開発
    • C# で Android/iOS アプリが開発できる Xamarin の紹介、および Android/iOS/Windows のクロスプラットフォーム開発についてお話します。
  • 対象
    • C#.NETでの開発経験のある技術者
  • その他
    • 座学形式で約45分講義のち、質疑応答

第二部ついて

  • 出演
    • ウェブインパクト 小峰様
  • 概要
    • Code Kata（Rubyプログラミング演習）
  • 準備物
    • Ruby実行環境(できれば1.9以上)
  • その他
    • 5分程度で主旨説明、25分演習のちプログラムレビュー
    • 課題は 練習問題 – プログラミングスレまとめ in VIP より適宜取り上げる

    • 第二部のハンズオンに参加される方は事前に環境をご準備の上お持ち下さい
