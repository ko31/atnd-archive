description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。


■現在予定している内容：

　●セッション１(35分)
　『iPad 紹介 & 実機によるデモ』iPhone OS SDK 開発勉強会

　＜内容＞
　アップル社「iPad」の日本発売前の実機を約1ヶ月間
　使用した感想/アプリケーションの紹介などをします。

　iPadを実際に触れて、その素晴らしさと可能性の高さを
　実感できると思います。

　対象となる端末
　・iPad
　http://www.apple.com/jp/ipad/


　(以下の内容は変更になる場合が有ります、ご了承ください)

　1.『iPhone OS SDK 開発勉強会』の紹介
　　5月から毎週行っている開発者向け勉強会の紹介
　　(勉強会の宣伝は最初少しだけ)

　2.『iPadアプリケーションの紹介など』
　　iPadのアプリケーションを実際にテストした感想、
　　約1ヶ月間の使い方などを紹介します。
　　アプリケーションは無料/有料アプリを120本程度テスト。

　　＜iPhone OS SDK 開発勉強会の紹介＞
　　実際にiPhone OSを使用して
　　ビジネスで開発を行っている方の集まりです。
　　5月からは毎週offline/onlineで情報交換を実施中。

　　ささやかなプレゼント有り(^^;
　　具体的にはiTunesで1曲購入できるカード
