WP初心者必見！WordPressセキュリティ&バックアップ講座

すぐできるWordPressサイトのセキュリティ対策，バックアップ方法など初心者向けに解説します！[ハンズオン講座]

　

開催概要

WordPressでサイトを制作・運用していると，セキュリティや何かトラブルがあった時のことが不安になります。

セキュリティは大丈夫なのか？ バックアップはどうするのか？ 万が一サイトがおかしくなった時の復旧はできるのか？など気になることかと存じます。

今回，そのような疑問やお悩み又は全くそんなことは考えていなかったという方を対象にサイトのセキュリティと運用のポイントをお伝えすると共にすぐ実行できるセキュリティ対策とバックアップ方法をハンズオン形式で解説しながらシェア致します。

PCとWordPressサイトをご準備頂ければ，その場で即実践することが可能です。

☆次のような方におすすめです！

  • WordPressをインストールしただけでそのまま使っている方
  • WordPressでとりあえずサイトを構築してみたが，次に何をしたらよいかわからない方
  • WorｄPressサイト(データベース)のバックアップ方法や復旧方法を知りたい方
  • コピーサイトやサーバ変更などWorｄPressサイトを移転する方法を知りたい方
  • とにかくすぐにWordPressサイトのセキュリティを強化したい方
  • サイトの乗っ取りや改ざん，情報漏洩，コンテンツの喪失を防ぎたい方
  • とにかく万が一の事態に備えたい方

アジェンダ

◆イントロダクション

◆WordPressサイトの運用とセキュリティについて

  1. WordPressサイトの運用とセキュリティ対策 (とりあえず押さえておきたいポイントとは？)
  2. WordPress本体，テーマ，プラグインの更新方法
  3. ユーザーセキュリティ及びログインパスワードの強化
  4. サーバ側でのセキュリティ対策
    ― IPアドレスによるアクセス制限とファイルの改ざん防止
    ― ログイン試行回数制限
    ― コメント・トラックバック制限
  5. プラグインによるセキュリティ強化
    ― スパムコメント防止
    ― 管理画面への不正アクセス監視
    ― 管理画面ログイン認証のセキュリティ強化
    ― セキュリティプラグインの紹介
  6. WordPressサイトのセキュリティ診断

◆WordPressサイトのバックアップについて

  1. バックアップの方針について
  2. 投稿ページや固定ページのコンテンツのバックアップ
  3. プラグインによるデータベースの自動バックアップ
  4. WordPressサイトコンテンツ及びデータベースの復旧
  5. WordPressサイトのコピー及びサーバ移転(引越し)
  6. 外部サービスの利用について

◆ その他 (質疑応答，個別相談 等)

☆ハンズオン形式で解説しながら進めていきます。

持ち物

  • ノートパソコン(WiFi接続)，電源コード/電源アダプタ(又はバッテリ)

    (WordPressサイトをお持ちでない方は，WordPressデモ環境をご用意させて頂きます。)
