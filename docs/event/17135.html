満員御礼

『格之進の直営牧場である門崎牧場が、生後約千日もの期間をかけて大切に育てた【いわて門崎丑】。約10日〜2週間枝肉を吊るしドライエージングにて熟成させ、牛肉の旨さを十分に引き立たせてから真空パックして冷蔵熟成させるウェットエージングを10日間以上。一頭買いではなく「一頭育て」ならでは！希少部位の味わいはどこまでも深く。。。』
今回はそんな門崎丑の巨大な肉塊を切りわける様からご覧いただける一大スペクタルショー。門崎丑の美味しさについての熱い語りもお楽しみいただきたく。お店にご協力いただき、通常のお店の開店時間より早く始まります。すべてはおいしいお肉をまったり、ゆったりとご堪能いただきたいためです。


  • 16:30頃より受付を開始いたします。17:00頃より本編開始を予定しております。
  • 21:30頃には終わる予定（希望的観測）ですが、長丁場のイベントですので、体調管理を万全にされる事をお勧めします。飲酒量も多めになろうかと思いますし、満腹中枢との熾烈な戦いも予想されます。
  • このイベントは一種のエクストリーム・スポーツです（笑）。

お品書き的な何か
ヒレ・サーロイン的な秘密？部位を解体して、厚切り、ブロック焼き、超低温焼ステーキなどの様々な焼き方で提供していただく予定です。当日のノリと雰囲気と勢い（笑）で変わることがあります。
