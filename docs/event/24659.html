description

そのまま帰っちゃもったいない。

「よりみち」は、単に家にそのまま帰ると勿体ないから、みんなで集まってワイワイやろうぜと言う。定例開催のゲーム会の名を借りただべりイベントです。

非常にユルい集まりにしていきたいと思いますので、是非とも皆様宜しくお願い致します。もちろん、ガッつりゲームを楽しんでもOKです。アリアリです。

みんなが寄り道した結果、楽しい事が起きる場になりますように。
