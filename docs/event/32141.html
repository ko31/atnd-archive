description

スマートフォンアプリ開発者様向けに、アプリ成功の秘訣を語るセミナーを開催します！

国内外でのプロモーションから広告マネタイズまで幅広くお話しすると共に、実際にアプリ開発をされているデベロッパーの方を交えてのディスカッションを行い、100万DL越えのアプリの企画の出し方や、面白アプリの企画の出し方まで成功エピソードを探っていきます。

※30名⇒70名⇒100名様に増枠しました。
※受付でお名刺を2枚頂戴します。
※ご参加はスマートフォンメディア（サイト/アプリ）の開発・運営ご担当者様のみとさせて頂きます。
