description

2011年　引き続き今年も「米づくり」、農と食を考えつつ地域活性化を考えたい。

78会では過去2年、延100人を超える参加者の皆様と、年間を通じて農業体験を実施、「農」と「食」のつながりを体感するとともに、子育てやコミュニケーション活性化など、自然と向き合うなか期待以上の効果を得ることが出来て来ております。

昨年は本格的な無農薬農法に挑戦、公的検査機関で『250項目の残留農薬０』　という貴重な実績を築き上げるとともに、食味テストでも優良に値する『78点』の評価を得ました。本年は、田んぼだけでなく畑作業、周辺農家の方々の協力も得て、
更にヴァージョンUPしたプロジェクトとして、皆さんと御一緒出来ればと考えています。積極的なご参加、是非宜しくお願い致します！

■78米　参加要項
□　1口=7,800円 （複数口ももちろん可能です）※全80口を募集します。
　　出資頂いた方には、美味しいお米と野菜＋素敵体験を！！

①農業体験に、いつでも自由にご参加頂けます
※基本スケジュール外でもご利用になられたい方はご相談）
②自分たちで作ったお米を、2011年秋にお届け　
　800kgの収穫を目標、最終収穫量を口数で配分致します。
③国内有数の無農薬農法を体験頂けます。（武藤農法）
④田んぼでの作業に加え、畑での作物づくりもご準備。収穫した野菜も口数で分配いたします。
※収穫量は天候により、変動いたします。上記は収穫量は目安であり、
　保証するものではございません。

【申込の際に必ず、mailアドレスの登録をお願いします】

■ロケーション
2011年の米ＰＪは、甲府盆地の北側、富士山を一望できる最高のロケーションです。
加えて、田圃での米作りだけではなく、無農薬の野菜作り、の準備も万端。
「生きている」という力と味のある野菜も是非お楽しみに頂ければと思います。
