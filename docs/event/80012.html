現役コスプレイヤーによる、コスプレ衣装+コスプレメイク講座が名古屋で!

本や解説サイトはあるけれど、少し難しそうなコスプレ。
どんな素材を選んだらいいの?型紙はどうやって作るの?縫い方は?お裁縫苦手だけどできるかな・・・
そんな方でも大丈夫。現役のコスプレイヤーさんが、初めてコスプレ衣装を制作する方にこの講座をお届けします。
まずは普段使いもできるコスプレ小物、手袋を作ってみましょう。
次の日は、ご自身のメイク道具を使ってコスプレメイクに挑戦します。
コスプレ造形だけ、メイクだけ、または両方の受講も可能です。
