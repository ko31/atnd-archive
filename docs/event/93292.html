description

初めまして。

この勉強会を開催させていただきます、青木です。
今回は、仕事の悩みの9割を占めるともいわれる
『人間関係』をテーマに勉強会を開催します。

・ビジネスでの人間関係がうまくいかず
仕事を任せてもらえない
・職場にどうしても苦手な人がいてストレス
・上司から頼まれた資料を作っても
「意図と違う」といわれ評価してもらえない

今回はそんな悩みを解決するために、
皆さんとビジネスにおける人間関係を
うまく構築する方法について考えていきたいと思います。

○セミナー内容
【1】自分の悩みを自覚する
【2】ビジネスにおける人間関係
-ビジネスの人間関係の特徴
-‎ビジネスの人間関係で悩む理由
【3】人間関係改善のためのコミュニケーションスキル
-仕事をうまく進める「実務コミュニケーション」
-‎人間関係を円滑にする「印象コミュニケーション」

○開催場所：ＣＡＦＥ英國屋 心斎橋本店

○参加費：2500円(ワンドリンク付き)

○講師プロフィール
数千人規模の会社で商品企画・開発業務に従事。
ニーズのヒアリングから始まり、企画、開発、PR、運用まで、幅広く担当。
様々な業務を通して多くの人と関わるなかで、ビジネスにおける人間関係のポイントについて考えを深める。
