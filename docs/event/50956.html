セッションの目的

「シナリオ・プランニング〜未来を描き創造する」の原著者 Woody Wade氏と、その本の監修者であるKIT虎ノ門大学院教授の野村恭彦が、シナリオ・プランニングの世界の潮流を解き明かし、その上で参加者の皆さんと一緒に、そのまた先の「シナリオ・プランニングの未来」を描き出すフューチャーセッションです。

対象者

未来を創ることにチャレンジしている方々。
・起業家　・企業の長期計画・戦略部門　・人材育成部門
・プロボノ　・NPO　・行政　・政治家　・学生　・ワーキングウーマン
などなど、多様な「未来づくりの担い手」を歓迎します。

Program
問いを共有する
19:00 〜 19:15 ストーリーテリング（Storytelling）
「シナリオ・プランニングの未来」について、問いかけ。

多様な知識を持ち寄る

新しい問題の構図を描く

「シナリオ・プランニング」の可能性を皆で挙げる。

アクションを生み出す

「未来づくりに向けての第一歩」を皆で相互応援する。
