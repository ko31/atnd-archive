概要

2014年末、ついに日本国内でもFirefox OS端末が発売され、つい最近Raspberry Pi 2 も発売されました。 そんな注目を集める二つのデバイスを、Web技術でつなげるハンズオンを開催します。

Firefox OSのアプリケーションの作り方と、Raspberry PiをNodeJSで制御するノウハウをハンズオン形式で体験していただければと思います。 Firefox OS、RaspberryPi共に実機が無い方でも参加できるよう、端末は用意されています。

自分の作ったアプリケーションをFirefox OSとRaspberryPiの実端末で動かしてみましょう。

時間は12:00 - 19:00となっていますが、ハンズオン自体は２時間で終わる内容を予定しております。 来ていただいた方から随時個別に対応いたします。

※ 当日使用するRasberry Piは2ではありません。
