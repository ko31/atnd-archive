PHP『超』初心者勉強会

PHP『超』初心者勉強会では、プログラムをはじめて勉強する方に向けた勉強会を開催しています。
対象者はプログラム未経験者ですので、どなたでも参加できます。
難易度は全く難しくありません。
プロジェクターの画面を見ながら、少しずつプログラムを作っていきます。
未完成のプログラムを少しずつ作りながら、最終的に完成したWebプログラムになります。

チャットプログラム

チャットプログラムの作り方を通して、Webプログラムの作り方を勉強します。
フォームの使い方、ファイルのアクセス、セキュリティ対策など。

勉強内容

  1. Webプログラムのテクニック
  2. ファイルにログを保存する方法
  3. ファイルフォーマットの考え方
  4. デバッグの方法
  5. ファイルの不正アクセスを防ぐ方法
  6. 勉強会はプロジェクターに映した講師の画面を見ながら、自分のノートパソコンにプログラムをします。
  7. ノートパソコンをお持ち下さい（貸し出し用のノートパソコンが１台だけあります（要予約））

  10. 費用が2,000円かかります。

XAMPPをインストールしてください

講義にはXAMPPというソフトウェアを使います。
これはパソコン上（Windows,Mac）で、Webサーバーを実行するプログラムです。
（基本的にインターネットで使用されるWebサーバーと同一のプログラムです）

Windows版
http://www.apachefriends.org/jp/xampp-windows.html

Mac OS X版
http://www.apachefriends.org/jp/xampp-macosx.html

インストール後、http://localhost/にアクセスして、インストールが成功したか確認して下さい。

インストールに失敗した方は、http://twitter.com/phpbeginnersまで、ご相談ください。
