早川周作「私とパフォーマンス力」セミナー

title 早川周作「私とパフォーマンス力」セミナー
catch ◆マイク一本で夢を次々とかなえてきた男・・・そのパフォーマンス力に迫るセミナー開催！◆
description

早川周作と一度でもお会いした方は、
そのイメージと彼の口から
流れるように出てくる会話のギャップに
一様に驚かれるようです。

また、早川のことをよく知る経営者達からは
「一瞬にして人の心を掴む天才」
とも称されています。

それは、早川が苦労時代の経験から独自に得てきた
『プレゼン能力』と『相手の心を読む力』を駆使し、
様々な場所で場数を踏んで
実績を残してきたからに他ありません。

・学生時代より資格予備校にて教鞭を振るい、
　人気ナンバーワン講師となる。

・出会って三週間で現在の妻と入籍。

・元総理大臣へ身体とハートを使った
　体当たりのアプローチをした結果、
　政治への道を掴み２０代で衆議院選挙出馬へ。

・実業界を代表する数々の大物とのセミナーを開催。
　スピーカーとしてだけでなく、
　ファシリテーターとしても高い評価を得ている。

その他数限りない結果を、口先だけではなく形として
出してまいりました。

日常生活はある意味プレゼンテーションの
繰り返しだといっても過言ではありません。

どんな些細なことでも、自分の意志を誰かに伝え、
共感を得て仕事や人脈、
そして信頼と富を勝ち得ていく。
そして、その総合的な力を「パフォーマンス力」と呼びます。

あなたは、自分のパフォーマンス力に自信がありますか？

この度、早川が初めてパフォーマンスにフォーカスした
セミナーを開催することとなりました。

「どうしたら早川さんのように、人前でも堂々と話ができて、
　なおかつ仕事や人脈に繋げることができるのですか？」

皆様から多く頂いた、早川が魅せる
パフォーマンス力について、
経験から得たテクニック、そしてワークを通じて
たっぷりお伝えしてまいります。
