「プログラミング・ブートキャンプ」体験会

福岡ではまだまだ数が少ないキッズ向けのプログラミング教室を、他へ先がけてこの秋開講しました。社会人講座で実績のあるイデアルITスクールで、遊びながら楽しく学んでいきましょう！

体験内容

ブロック状のビジュアルプログラミング言語 AppInventor 2を使い、パズルゲームのような感覚で遊びながらandroid アプリを作っていきます。

作成アプリ例

  • じゃんけんゲーム
  • もぐらたたきゲーム
  • 簡易メモアプリ
  • お絵かきアプリ

※作成例は変更する場合があります。

使用設備

ハードウェア：Android4.2　以降のスマートフォン・タブレット
ソフトウェア：App　Inventor2
