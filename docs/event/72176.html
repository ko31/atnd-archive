開催内容

Web制作者としてフリーランスを視野に入れている方、実際にフリーランスとして動いている方へ、活躍の場を広げ、収益を安定化させるための考え方を 会話形式のセミナーでお伝えいたします。

Web制作のセミナーではあまり見かけない「フリーランスとしての仕事の仕方・考え方・マインドの持ち方」について話し合う場となります。

技術スキルと共に、フリーランスの仕事への向き合い方をしっかり作ってこそ、継続して仕事を得ていくことができます。

一緒に、フリーランスの働き方、考えてみませんか？
