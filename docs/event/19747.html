Facebookイベントにて定員に達したため閉め切ります

概要

AndroidとiPhoneの2大スマートフォン向けにアプリを開発してみたいという方は多いと思います。

そこで今回、「スマートフォンのアプリをはじめてだけど作ってみたい」という大学生・大学院生の方を対象に、
Titaniumのエバンジェリストとして著名な増井雄一郎氏をお招きした、
スマートフォンアプリ企画/開発勉強会を開催します！

Titanium Mobileは、iPhone/Androidなどスマートフォン向けのネイティブアプリを
Javascriptベースで構築できるフレームワークです( 公式サイトは こちら )

勉強会は、

  • Titanium Mobileが初めての方向けに、デモアプリ開発までを増井さんからレクチャー
  • 技術だけでなく、どんなアプリを作ったらいいかというアイディアブレストも実施
  • 当日はニフティ株式会社のエンジニアと企画担当もアプリ開発/アイディアブレストをサポート
  • 勉強会の最後は懇親会
  • 無料

と至れり尽くせりの内容となっております。

スマフォアプリ開発に興味のある方、ぜひご参加ください！

対象

  • Android / iPhoneの開発に興味がある大学生・大学院生の方（文理問いません）
  • 何等かのエディタを操作できる方(eclipse、emacs、秀丸エディタ、Dreamweaver等なんでもいいです)
  • Titaniumの環境設定を自分のPCにセットアップし持参できる方(下記参照)

参加にあたり準備頂くもの

↓↓必ずTitanium studioの以下設定を事前にお願いします↓↓

  1. Windows又はMacOSのノートパソコン
  2. Titanium studioを事前インストールをお願いします。
    http://www.appcelerator.com/ にアクセス。
    ※インストール後、iPhone OS SDK又はAndroid SDKのどちらかをインストールして下さい。SDKのダウンロードは公式サイトからできます。
  3. Titanium Mobile Kitchen Sink Demoの事前インストールをお願いします。
    https://github.com/appcelerator/KitchenSink にアクセスしてダウンロードをして下さい。

※ 上記の事前準備が難しくてできなかった場合は 、
会場に15分前にお越し頂ければこちらでサポートすることができます。
※参考サイト http://d.hatena.ne.jp/kaz_konno/20110614/1308075564
http://t.co/02NZYan が推薦図書です。
※当日会場には無線LAN環境を用意します。
