連休はAndroidしようぜ！　- 合宿編 -


9/18(日)に
大原温泉　湯元のお宿　民宿大原山荘 (楽天トラベル)
で、Androidな合宿をしましょうー！

宿泊内容

  • 宿泊プラン
    【伝統の味】全7品★自家製味噌鍋プラン！
    　→夕食は自家製樽出し味噌がベースで京赤地鶏と地野菜をふんだんに
    　　使用したレギュラーのお鍋、とのことです。
    　　朝食もあります。
  • 山のお風呂
    　源泉は弱アルカリ。疲労回復はもちろん、刺激が少なく、
    　肌の角質を軟化させツルツルにすると言われています。
    　山のお風呂は檜と石風呂の２タイプ。男風呂と女風呂
    　を朝と夜に入れ替えますので、両方をたっぷり楽しんで
    　いただけます。
    　また、露天風呂に併設された信楽焼の壷風呂は源泉
    　そのまま１００％で半掛け流しです。
    　→とのことです。ただし、入湯税150円必要だそうです。

おおまかな計画
　朝からコースとお昼からコースを考えています。
　朝からの人は11時に集合してカフェでお昼も取りつつ、
　開発します。
　お昼からの人は各自でお昼は取ってきてもらって、
　13時に集合して現地に向かいます。

　宿は
　チェックイン　15:00から
　チェックアウト　翌10:00まで
　なので、そこらへんで入ろうと思います。

　現地についてから15:00までは近くのカフェで
　まったり開発しようと思っています。
