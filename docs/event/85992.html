年に一度のPFLab活動報告

PFLabは東京大学情報理工学系研究科コンピュータ科学専攻・理学部情報科学科に所属する研究室です。オペレーティングシステムや並列分散システムに関する研究を進めています。本フォーラムではその研究活動の一部を報告します。

研究室配属を考えている3年生や進振りを控えている2年生の皆さんにとっては有益な情報満載だと思います。また、PFLabの研究活動に興味を持っている企業の方、一般の方、外部の学生の皆さんにとっても楽しめる内容になっています。

加藤真平准教授による自動運転とベンチャー起業のお話

PFLabを主宰する加藤真平准教授はオペレーティングシステムや並列分散システムの研究だけなく、自動運転ソフトウェアのオープンソース開発やベンチャー起業も手掛けています。特に2015年8月にオープンソース公開したAutowareと2015年12月に創業したティアフォーは大きな注目を集めています。ほかの講演では聞けないお話もありますのでお楽しみください。

PFLab学生の研究紹介

今年度は4名の学生が新たにPFLabに加わりました。自動運転に必要となる3次元地図作成や運転データストリーム処理、Deep Learningのスパコンによる高速化、次世代メニーコア向けオペレーティングシステムなどに関する話題を提供します。

豪華ゲストによる講演

記念すべき第1回PFLabフォーラムのゲストはスクウェア・エニックスの三宅陽一郎さんとマイクロソフトの千代田まどか（ちょまど）さんです。三宅さんはゲームAIの達人として業界で知らない人はいないでしょう。ちょまどさんはツイッターのフォロワー数が2万人を超える人気のエンジニア兼漫画家です。偶然にもお二人は「はしれ！コード学園」の連載開始に際して、三宅さんの発案でちょまどさんが漫画にしたという経緯があるそうです。講演が楽しみです！
