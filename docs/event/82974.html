競プロerの皆さん、オフ会がてら皆さんで非公式のオンサイトをしませんか？

JOI2016の予選が12月11日に予定されています。この予選は、誰でもが参加できます。そこで、本選参加資格のある方も、そうでない方も、皆さんで集まって問題を楽しみませんか？終わった後にはその場解説も行い、皆さんで集まってコンテストを楽しみたいと思っています。

オンサイトの後には、忘年会も企画しております。オンサイトを楽しんで、忘年会になだれ込みましょう！！(オンサイトのみの参加も可能です)


参加費

会場費を割り勘する予定です。

  • 学生　　　 500〜1000円程度
  • 社会人　　 1000〜1500円程度

を予定しています。(参加人数により変動)

なお、会費決定のため、参加登録時のアンケートフォームにて上記のどれに該当するかお聞きしています。ご回答をお願いします。


集合時間・場所

2016年12月11日 12時15分
阪急うめだ ビッグマン前 (昼食はお済ませください)


会場

ニュー共栄ビル 地下会議室


タイムテーブル

  • 12:10　　 集合→移動
  • 12:20　　 会場到着・自己紹介タイム(一言ずつぐらい)
  • 13:00　　 コンテスト開始
  • 16:00　　 コンテスト終了・小休憩
  • 16:15　　 解説会
  • 17:15　　 片付け・撤収準備(ご協力をお願いします)
  • 17:30　　 撤収・忘年会会場へ移動

※ 忘年会は別途募集していますので、ご参加いただける方はそちらにもご登録をお願いします。

注意事項

  • 集合場所・時間に間に合わない場合は、Twitterにて、@tatuyan_edson, @yurahuna, @threepipes_s, @laycrs のどなたかにご連絡ください。(直接会場までお越しいただくこととなります。)
  • 無線LAN環境は用意していますが、お持ちの方は持参いただけると幸甚です。
  • 予選参加には前もっての参加申し込みが必要になりますので、お忘れのないようお願いいたします。
  • 本オンサイトは情報オリンピック日本委員会とは無関係に、有志で設定したものです。本企画については同委員会にお問い合わせなさらないよう、お願い申し上げます。
