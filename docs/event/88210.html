講演概要

MySQL 8.0 最新開発動向

  • 現在開発中のMySQL 8.0では、日本語に適した照合順序の実装、Invisible Index、トランザクショナル・データ・ディレクトリ、など、多くの新機能が実装されています。このセッションでは、MySQL 8.0の最新開発動向をお伝えします。

MySQL 8.0 CTE ＆ Window関数

  • MySQL 8.0では、Common Table Expressions（WITH句）が実装されており、今まで実行できなかった再帰的なクエリーも実行できるようになっています。また、集計系の処理で役に立つWindow関数の実装も予定しています。このセッションでは、これらのCTEとWindow関数についてご紹介させて頂きます。

MySQL 8.0 Replication改善点

  • MySQLで人気の高いレプリケーション機能は、MySQL 8.0でも引き続き改善を進めています。このセッションでは、MySQL 8.0でのレプリケーション改善点についてご紹介させて頂きます。
