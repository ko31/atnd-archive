スタートアップウィークエンド愛媛vol.6プレイベント

コワーキングスペースで、事業計画をモクモクとつくるイベントです。
日々の業務に追われて、先の戦略や戦術を考える時間がとれてないことが、ありがちです。
このイベントでは、この先のことを考えたり、アウトプットしていきます。

※参加者に応じて、ちょっと相談できる専門家にも来てもらいます。いま、税理士はいます。

こんな方にお勧め

  1. 起業前後の方
  2. 日々の実務に追われて、戦略を考える時間がとれてない方
  3. 新規事業をはじめる方

このイベントの提供する価値

  1. まとまった時間といつもと違う環境で集中的に計画をたてることができる
  2. 他の人の話しを聞くことでヒントになることがあるかも
  3. 税理士がいるので会計税金、融資等の相談も可能
  4. 事業計画で使うテンプレートなども要望により提供可能
