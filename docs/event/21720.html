故 佐藤惠美（旧姓：津々見めぐみ） 儀 2011年11月1日 23時57分に39歳にて永眠致しました。
つきましては下記の通り、通夜・葬儀告別式を執り行いますのでご案内申し上げます。

通　夜：2011年11月5日（土曜日）18時00分〜19時00分頃
告別式：2011年11月6日（日曜日）11時30分〜12時30分

当日は雪ヶ谷総合式場全館を貸切で執り行います。
# ただし告別式の当日は一部の部屋を法事でご利用される方がおります。

宗旨は無宗教形式です。また妻@megsatoがこの半年の闘病生活の間、iPhoneで聞いていた音楽（Rock’n Roll!）の中なら特に本人が気に入っていたものをBGMに、故人を忍びながら皆様から最後のご挨拶をしていただくお時間を設けさせて頂ければと思います。

当日は、息子@aoisato115の三歳の誕生日でもあるため、あまり暗くならない程度に、服装なども形式張らずに平服で構いません。黒タートルセーターにジーンンズ大歓迎です。
（ただしランニングシャツ、ひざ上半ズボン、サンダルなどはご遠慮ください）

●電車を利用する場合
五反田・蒲田より東急池上線を利用「石川台」駅より中原街道に向かい、徒歩5分程度

●お車をご利用の場合
都心方面より首都高速2号線、または第二京浜を経て中原街道を、多摩川「丸子橋」方面へおります。
東名高速・第三京浜・環八利用の場合、中原街道を都心に向かいます。
式場に最大36台まで収容可能です。
＃通夜・葬儀時間は中原街道の混雑が予想されますので、電車でのご利用をお勧めします。

●ご供花・その他、お問い合わせ先
