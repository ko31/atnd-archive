第2部　今後の新産業・新市場創出への攻めの方策　～課題解決型産業と産業技術戦略～
【14:45-16:15】

【キーワード】
1.経済産業省技術戦略マップ
2.素形材技術戦略
3.新産業・新市場

【講演主旨】
本講演では、今後の我が国の新産業・新市場の創出に向けた攻めの方策について解説を行います。今後期待されている新産業は、ライフ・グリーンイノベーションを活力とした課題解決型産業です。また、これら経済活動を円滑に進めるためにはクリエーティブ産業の潜在力や中小・小規模企業の潜在力を強化することが必要であり、これらを実現する産業技術戦略等について解説を行います。

【プログラム】

1.新産業・新市場の創出～攻めの方策～

2.課題解決型産業～ライフ・イノベーション
　2.1　診断と治療の一体化
　　2.1.1　サンプル提供システム、早期診断法、根本治療薬
　　2.1.2　バイオインフォマティクス技術・診断技術
　　2.1.3　創薬力強化
　2.2　生体負担の少ない治療
　　2.2.1　ポスト抗体医薬
　　2.2.2　内視鏡化手術支援システム
　2.3　再生医療
　2.4　生活支援ロボット
　2.5　ヘルスケア

3.課題解決型産業～グリーン・イノベーション
　3.1 再生可能エネルギーを高効率活用技術
3.1.1　エネルギー転換効率の極限へ
　　3.1.2　太陽光や風力等の再生可能エネルギー、原子力エネルギー
　3.2　高効率送電・蓄電システム
　　3.2.1　電気抵抗ゼロへ～超電導・光
　　3.2.2　高効率送電・蓄電システム
　　3.2.3　超電導線材を用いた機器開発・実用化
　3.3　AC(交流)からスマートDC(直流)へ
　　3.3.1　新材料パワー半導体・インバータの効率改善、小型化
　　3.3.3　住宅・ビル全体への直流配電技術
　3.4　有機・複合材料ベースに転換できる無機材料
　　3.4.1　無機から有機・複合材料へ
　　3.4.2　半導体プロセス、電子回路・情報通信機器の有機・複合材料への転換
　　3.4.3　有機半導体、導電性ポリマー
　　3.4.4　インクジェット・印刷技術を利用したエレクトロニクス技術
　　3.4.5　炭素繊維複合材料
　　3.4.6　カーボンナノチューブ・新規超高強度素材
　3.5　常温常圧(に近い)プロセス要素技術
　　3.5.1　高温高圧から常温常圧プロセスへ
　　3.5.2　ガラス溶融プロセスの転換
　　3.5.3　新規触媒・セメント焼成プロセスの省エネ化
　3.6　レアメタル

　3.7　グリーンケミストリー

　　3.8.2　流通・輸送

　　3.9.3　太陽光発電、蓄電・蓄熱システム
