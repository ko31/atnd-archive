☆Talk Field第13弾
　テーマ「自分の心の声を聞いてみよう」

Talk Fieldは直訳すると「おしゃべりする場」ということで
20代～30代の同世代の人たちと、普段は話さないけど
本当は話したい、自分の考えや、思いを
たくさん話したり、考えたりするワークショップです。

コミュニケーションワークを通して、毎回違ったテーマについて話し、深めていきます。

今回のTalk Fieldのテーマは
「自分の心の声を聞いてみよう」です。

日々の生活に追われ、自分が本当にやりたいことを無視してしまう。
「何か違うな…」と思いつつ、そのままにしてしまう。
やりたいことをやっているけど、果たして本当に自分がやりたいことが出来ているのだろうか。

日々様々なことを感じながら過ごしていると思います。
ここいらで、ちょっと立ち止まって、ゆっくりと会話をしながら、自分の心の声を聴いてみませんか？

今回はリラックスバージョンということで、アットホームな雰囲気で行いたいと思います。

よろしければご参加ください。
