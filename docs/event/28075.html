過去一度たりとも裏切らない『Hiroki mode』は大人のファンタジーな空間をテーマにした恵比寿の隠れ家一軒家★
砂漠をイメージして作られたスタイリッシュな１階フロアと東洋をイメージした地下一階のオリエンタルなスタンディングバーは圧巻！Hiroki modeは古今東西の「童話」「神話」の世界観を体感できる空間となっており、日常生活で身についてしまった窮屈な価値観や時間の感覚をリセットされることでしょう。
そして、外壁にはアートが描かれ、各階ごとに空間が全く異なります。GW半ば、時間を忘れて心地よいひとときをお約束いたします。
都内でも有数の婚活企業様とのコラボレーションパーティーです！お仕事帰りに是非この機会をお掴みください・・・
