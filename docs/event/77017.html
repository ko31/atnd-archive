ボードゲームオフ会

ご覧いただきありがとうございます

こちらは初心者＆２０代３０代限定で
楽しくボードゲームをするオフ会です

子供のころに皆で集まって
ボードゲームや
トランプ・UNOなどを
プレイしたことがある方が多いと思います（＾＾）

しかし大人になってからは
なかなかプレイする機会がない（－－）

このオフ会では
あの雰囲気を楽しみたい方が集まって
皆でボードゲーム・カードゲームをいたします

近年、人狼ゲームなどが流行っており
専用のプレイルームなどもあります

その流行りに乗っかってｗ
皆でゲームしたいなと思い、企画しました

こんな方にオススメ

初心者＆２０代３０代限定なので同年代で楽しめます

◆友達や人脈を増やしたい方
◆他の業種の方と知り合いになりたい方
◆職場や学校とは違う友達をつくりたい方
◆つまらない毎日にスパイスを加えたい方
◆コミュニケーションスキルを向上させたい方
◆ボードゲームを体験したい方
◆皆でゲームして楽しみたい方
◆ＰＳやスマホゲームとは違うアナログゲームを体験したい方
◆時間を有効活用したい方

ゲーム内容について（１位景品あり）

ゲームは十数種用意しております
当日の人数に合わせて
数種類選んで持っていきます

最少２人～最大１２人までです
２人専用ゲームなどもありますので
おひとりでの参加もお待ちしております

基本的に１プレイ１０分くらいで
終わるゲームを何回かプレイし
オフ会全体で
２～３種類のゲームをやります

個人戦・チーム戦など
人数に合わせてプレイしていきます

総合１位になられた方には
ちょっとした景品も用意していますので
皆でワイワイ楽しみましょう
