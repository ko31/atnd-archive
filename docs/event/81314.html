理想のパートナーを引き寄せる、心と意識のあり方を学び体験します。それは、自分に素直に生きる＝自分の気持ちを裏切らないことでもあります。

７－０.はじめに

７－１.絶対にやらないことを決める

７－２.パートナーとの関係について、あなたの本質を知る

７－３.理想のパートナーを明確にする

７－４.どんな付き合いをしているのか、イメージを拡大させる

７－５.拡大したイメージを口に出し、他人にシェアをする

７－６.理想のパートナーと出会うための行動に移る

７－７.限界を越えて、次のステージへ行く

７－８.おわりに

上記の７ステップに基づいて、セミナーを進めていきます。

持っているかを表現し、シェアをすることによって
まだ気付いていない、あなたの本質に沿った
理想のパートナーを明確なものへとしていきます。
