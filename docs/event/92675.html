Alibaba Cloud 特集！

2017年最後のMasterCloudは、「AlibabaCloud」だらけでお送りします！ 日本ではまだ馴染みないと思いますが、世界では急速な勢いで伸びているAlibabaCloud。まもなくGCPを抜くとも言われています。
AWS, Azure, Alibabaの「3A」と呼ばれる日も近いかもしれません。

◆MasterCloudとは

MasterCloudの目的はクラウドコンピューティングの最先端の技術や知見を共有・ディスカッションし、ビジネスやエコシステムの構築に寄与することです。

◆Great speakers

吉田真吾 (セクションナイン)

AWSウルトラクイズ初代チャンピオンでAWS SAMURAI 2014&2016はもはやレジェンド。 サーバーレスのイベント開催やAWSのコンサルティング、書籍執筆などの活動を行なっている。

森 真也 (SB Cloud)

もーすけ(@mosuke5)です。Alibaba Cloud日本サイトの中の人やってます。 主に、お客様へのAlibaba Cloudのソリューション提案や、クラウド関連の技術検証・セミナー開催などやっています。

ZHIMIN TANG (AlibabaCloud / Geek Engineer)

I'm Product Manager and Technical Lead of Alibaba Cloud Container Service focusing on containers, microservices and cloud native applications. Prior to Alibaba Cloud, I was an architect working on middleware and application infrastructure inside Alibaba.

浅尾 元 (スカイアーチ)

新卒でスカイアーチに入社。テレアポ営業からスタートし、 現在は担当顧客のビジネスをインフラ面から支える提案を主に行っている。 目標は技術もできる営業でサーバー業界の大谷翔平を目指してます。
・Twitter ID：@G_Asao110

砂原 健佑 (Mobingi)

MobingiのAlibabaCloud担当APIエンジニア
