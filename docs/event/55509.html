イベント名:
～Unite 2014(Seattle) を踏まえて～
Windows Phone8.1アプリ開発とUnityゲーム開発の夕べ

概要

8/20-22 にシアトルで開催された Unite 2014 にはマイクロソフトも協賛し、技術セッションや Unity 開発者向けの特別特典なども発表させていただきました。本社レベルでもイベントだけでなく技術面でもご一緒させていただいていて、既存の Android や iOS 向けに制作された Unity ゲームを Windows Phone 8.1 や Windows 8.1、Xbox One などのマイクロソフトのプラットフォーム向けに移植しやすくする方法なども提供しています。今回のセミナーでは、その Unite 2014 で発表された内容や Windows Phone 8.1 向けのアプリ開発手法の概要もご紹介します。
これまで Windows を対象としたアプリやゲーム開発をしたことが無い方にも理解いただける内容となっていますので、ぜひ気軽にご参加ください。皆様のお越しをお待ちしています！

【追記】ポケット・クエリーズの佐々木様にもご登壇いただけることになりました！

講演概要

Windows Phone 8.1 アプリ開発概要

  • 概要
    • スマートフォン・モバイルデバイスプラットフォームとして注目を集めている Windows Phone。最新版の Windows Phone 8.1 OSの基本的な構造と、アプリケーションプラットフォームとアプリケーション開発環境の紹介、さらに Windows Phone 8.1 向けのアプリ開発の方法をデモを交えながらご紹介します。

Unite Seattle を踏まえて～Unity ゲーム開発概要 on Windows

  • 概要
    • Unite Seattle でご紹介したWindows Phone 8.1 や Windows 8.1、Xbox One を対象とした Unity ゲームの開発/移植方法を中心に最新情報をご紹介します。Visual Studio Tools for Unity （旧 UnityVS）を使用した開発方法やすでにWindows向けに移植されている開発者の方から実体験に基づく話もご紹介します。
