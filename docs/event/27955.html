イベント概要

ひとつのソースコードでマルチプラットフォーム展開ができる、
スマートフォン対応のオープンソースソフトウェア「Titanium Mobile」に関する無料の勉強会です。

Titanium Mobileのエヴァンジェリストである増井雄一郎さんから、
Tiの紹介と、Tiによるアプリ開発をライブコーディングで実施いただきます。
（もちろん、先日リリースされたTitanium Mobile2.0対応です！）

また、LTではサイバーエージェントの新規ビジネス担当の方から、
賞金総額500万円の「モックプランコンテスト」のご紹介もあり。
Tiで開発したオリジナルアプリで一攫千金を狙いたいという方も、ぜひお気軽にご参加ください！

勉強会終了後は、同会場にて懇親会も開催。
講師の増井さんや参加者との情報交換・交流を深めていただければと思います。

※なお、本イベントはWeb CAT Studioが主催する勉強会です。
　サイバーエージェント社の「モックプランコンテスト キックオフイベント」
　とは別企画ですので、ご注意ください！
　（コンテストへは、キックオフイベントに不参加でもご応募いただけます。）

※モックプランコンテストの概要はこちらをご覧ください

講演者

増井 雄一郎 氏　　@masuidrive

2009 年までAjax、Ruby on Railsなどの技術を使った Web アプリケーションの構築や雑誌・書籍への執筆をフリーランスとして日本で活動。PukiWikiなどのソーシャルアプリケーションの企画・開発を得意とする。2008 年より渡米し iPhone、iPad関連のアプリケーション開発に従事。主な著書:「Ajax 実装のための基礎テクニック」(技術評論社)、「PukiWiki入門まとめサイトをつくろう!」（翔泳社）2010年12月に帰国。
現在は、Appcelerator Inc. テクニカルエバンジェリスト。
