description

小さい画面のなかじゃない、目と目をあわせて本気で遊ぶ。

「遊びは学び」　大切なのは、楽しいことです。

普段お子さんと向き合って遊んでいますか？
自分が子どものころ、家族で遊んでいた記憶はありませんか。
年に何回かの家族イベントではなく、日常の記憶。
お子さんとの遊び方が分からない親が増えているようです。
だったら大人も楽しく遊べるゲームで子どもと遊びませんか？
対等に、本気で。
画面じゃなく顔を見て、会話をしながら、家族で遊ぶボードゲームを体験してください。

★　　★　　★　　★　　★

ボードゲームひろばは、乳幼児から中学生までそれぞれのお子様にあったゲームを選んで親子で・友達で遊んでもらうのが目的です。

ボードゲームって何？やったことない、という人も大丈夫。
ゲームの世界の案内人、ゲームマスターが丁寧に楽しみ方を伝えます♪

おうちでの遊びの幅を広げたい、
遊びを通してコミュニケーション能力を伸ばしていきたい、
そんな方はぜひ一度ボードゲームひろばに遊びに来てください♪

また、こどもに関わる職業の方や、
こどもと遊ぶことが好きな大人のみでの参加も大歓迎です。
ここのところ学童・塾でのボードゲーム導入事例が増えています、
その場での相談・販売もできますので興味のある方はお気軽にお越しください。
おもちゃコンサルタントでもあるボードゲーム販売店店員がお待ちしております。
