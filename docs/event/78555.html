description

3DCGで漫画を作ることを試みる会を開催いたします。

第2回はスカルプトリスとblenderをテーマに話し合います。

3Dの知識が全くない方も大歓迎ですのでお気軽にご参加ください。

主に無料ソフトを使った方法になりますので、３ＤＣＧを趣味でやりたいという方も大歓迎です。

対象者

漫画に興味がある方・描いている方・3D(CG・CAD・モデリング)に興味がある方

参加無料ですのでお気軽にご参加ください。


内容

①作るものをみんなで話し合う。
＊候補をいくつか出して、その中で何を作るかをみんなで決める。
②勉強会 スカルプトリス
③勉強会 blender
です。

①で今候補となっているのは
・VR系コンテンツ
・動く漫画
・360度から観れる漫画
・動く絵画
・などなど
　＊忘れていたらごめんなさい…

話し合いと自主製作の報告が主な活動になります。
自主製作を互いに支えあうことがコンセプトの会になります。
今回はスカルプトリスとblenderを使ったワークフローを説明します。
初心者の方には3DCGの基本的なことから説明します。
作品を制作することが第一目的なので、最終的に3Dを使わないという選択もありです。

＊内容の詳細は検討中です。
