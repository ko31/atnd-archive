2016年5月、キャリア志向の働く女性のための無料カフェイベント「Me Cafe」が誕生します!

このイベントは”MICA COCORO”がプロデュース・出資し、NEXT STAGE PROJECTメンバーが発案・発信する参加費無料イベントです。

「Me Cafe」は、「私のためのカフェ」。 「自分らしく過ごしたい! 誰にも真似できないスキルとキャリアをクリエイトする !」というテーマのもと、NEXT STAGE PROJECTが企画・運営するカフェイベントです。 他人の考える枠にはまる人生ではなく、自分らしさを活かした人生を生きるためのヒントを発見できる、そんなイベントを目指しています。

イベント当日は、NEXT STAGE PROJECT第2期生のYUKO(38)・CHIKA(25)が、世代間ギャップを越えて進行を務めます。事前にお答えいただくアンケート・お寄せいただいた相談についてのトーク、ワークショップを予定しております。

お問い合わせフォームやLINE@も設けており、事前質問、ご相談も受け付けております。 http://nextstageproject.com/wp/event-01/

なお、このイベントは様々な場所で告知をしております。 合計10名を超えましたら募集は停止となりますので、ご了承のほどよろしくお願い致します。
