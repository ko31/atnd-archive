昼の部と夜の部は別々に予約が必要ですので、ご注意ください。

ライトニングトーク ルール

いつもの5分で話が終わらない「名古屋式？ライトニングトーク」とは形式が異なるので注意してください。

  • 1人の発表時間はジャスト5分です。
  • 5分を超えた時点で、発表途中であっても打ち切ります。
  • 5分未満で発表が終了してしまっても、5分経つまでそのまま放置されます。
