この勉強会について

Unityや、3Dモデリング初心者さんを対象に、実際に触りながら勉強していく会です。ゆるい感じでやりますよー
知り合いのデザイナーさんに頼まれて開催することにしましたので、初心者さん歓迎です！

  • Unityを起動したことがない
  • プログラムを書いたことがない
  • モデリングしたことない

上記のような人が理解できるように進行しますので、ご気軽にご参加ください！

持ち物

ノートパソコン
　Windows または Mac OS が入っているもの
マウス
　Blender操作時に便利なのでできれば持ってきてください

事前にやっておいてもらいたいこと
・Blender2.7のインストール(インストールは こちら から)
・Unity5のインストール (インストールは こちら から)
Unityは起動時にアカウント登録がありますので、そちらも済ましておいてください。
手順についてはこのサイト等が参考になるかと思われます。
http://hiromart.com/blog/from_installation_to_s
