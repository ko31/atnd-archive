catch 今回もまったり系もくもく会
description

「技術」「サービス」「コード」などなどに恋をしてしまった・・・
そんな技術者やデザイナーさん等が集まって、自由に作業をする会です
いわゆる「もくもく会」です

どんな作業をするのも自由です
コードを書くのも、サービスのコンセプトを練るのも、
調べ物をするのも、積んでいた本を読むのも、全て自由です

一人でひたすら作業するのも、他の人に聞きながらでもOK
同じ場所で一緒に作業する空気感を楽しんでください

他の方と技術情報の交換をしたり、
デザインについて話し合ったりといった、
「一緒に作業するからこそできること」も楽しめます


今回は特に二次会を用意していません
終わった後、希望者の方達で食事をする感じになると思います
