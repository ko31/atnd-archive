タイトル：【0から始めるbootstrap 4 〜基礎からモバイルアプリへの応用まで〜】
現在のフロントエンド開発でよく耳にする 「Bootstrap」。名前は知っているけど、そもそも何なのか？ どんなメリットがあるのか？どうやって使うのか？
今回は機能紹介から基本的な使い方、また応用までを具体的なアプリ開発の事例を通してご紹介します。

・川田 寛(ふろしき)さま
Webとモバイルを扱うエンジニア。ここ最近は、パフォーマンス関連のWeb技術のトレンドを追いつつ、コミュニティ活動、オンラインメディアや雑誌に寄稿を通じて、日本国内への普及・推進を行う。
NTTコムウェア入社時、オープンソースソフトウェア推進部に所属し、OSSとコミュニティ文化、ビジネスの関わりを追求。その後、開発プロジェクト、OSS技術者育成などを経て、現在のWebの世界へ。Microsoft MVP Internet Explorer 受賞、html5j パフォーマンス部やApache Cordovaユーザ会といった、コミュニティの設立と運営にも関わる。

タイトル：【ブラウザのPerformance API の話】
最近あんまり勢いがない(と噂されている)Webの技術の話をします。とはいえ、内容はHTML5全盛の時と同様、最新のことに特化します。Webのパフォーマンス関連の進化とかの話をかいつまんで、わかりやすくざっくりと。あと、この手の話で悩んだことある人にちょっとした小ネタの話をします。ちょっとだけ、得した気持ちになれるかもしれない。
