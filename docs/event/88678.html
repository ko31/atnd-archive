description

実績14年、5000件以上の相談を受けてきた私たちが失敗と成功から学んだノウハウを全て公開します！

これを考えないままでは、どのホームページ制作会社に頼んでも思うような成果にはつながらない？！

「費用対効果」は？「集客方法」は？

予算が限られている中小企業だからこそ、
是非とも知っておきたいホームページの戦略ポイントをまとめました。
他社への発注を検討中の方も知ってて損はありません。

費用に見合った効果を得るために

かかる費用が同じなら、より効果を出せるホームページにしたいですね。
逆に、同じ効果を出すのに、もっと低コストな方法があるかもしれません。
それもこれも、まずは戦略ありきです。

あるいは予算どりの時期で、今はホームページの相場や費用感だけを知りたいというご担当者さまも
「費用対効果」のお話なので、きっと今後にお役立ていただけると思います。

オープン形式のセミナーとなっておりますので、
この機会に、あなたの会社のホームページ戦略を見直してみませんか？
他社に発注をお考えの方も、お気軽にご参加くださいね。

講師

長田 匡平（株式会社フォノグラム 取締役）

10年以上にわたりウェブ戦略コンサルティング業務に従事。
地元銀行さまと共同で、ウェブを活用したビジネスを進めるなど、ホームページ以外のウェブ活用方法にも明るい。
