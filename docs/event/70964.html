■13：05～13：50
検索におけるコンテンツの役割と利用の仕方(仮)

アユダンテ株式会社
SEOコンサルタント
村山 佑介氏

ネットを利用して商品やサービスを提供するさいに、ユーザーにとって検索という行為は重要な位置づけとなります。それは事業主にとっても同じであり、検索を利用したマーケティング施策を選択肢から外しづらい商材も多く存在します。事業主にとって、今できる検索における施策はどのような手段があるでしょうか。SEOにおけるコンテンツというテーマでご紹介します。

■14：00～14：50
顧客を魅了しつづけるシナリオ（CRM）の作り方とは（仮）

株式会社 売れるネット広告社 東京オフィス長
コンサルティング部 部長 チーフコンサルタント 兼 メディア部 部長 チーフメディアプランナー
樋口 慎司

製品での差別化が難しくなり、新規顧客の獲得コストも年々増加…　通販を行う事業主にとって厳しい環境が続いています。そういった厳しい競争環境の中で、儲かる通販を構築するために最も大切なことはなんでしょうか。
その答えのひとつがCRMです。
本講座では、実際にCRMを構築するためのノウハウを事例を使ってご紹介いたします。

■15：00～16：00
事例でご紹介！提携・稼働数をアップさせるキャンペーン施策！アフィリエイト実践講座

株式会社売れるネット広告社
メディア部課長　シニアメディアコンサルタント
鈴木珠世

アフィリエイトは"費用対効果"の高い広告手法と言われています。
ですが、多くの広告主が取り入れようとして失敗し、アフィリエイトから撤退していきます。
それはなぜか？「アフィリエイトは"運用"が必要である」ということを、知らずにスタートしてしまうからです。
今までの通販会社様向けアフィリエイトセミナーは、管理画面の使い方が中心の『ASPのシステムを効率よく使い、運用する方法』でした。
当講座は、通常広告とアフィリエイトの違いを理解し、アフィリエイトプログラムを利用することによる可能性、成果を上げるための施策など、具体的に、どのように取り組むべきかを学べます。
一般アフィリエイトサイト向けキャンペーンを中心に、アフィリエイターの提携・稼働数を伸ばすための施策を過去事例を用いてご紹介します。実際にやるべきことが整理され、明日から"すぐ使える"情報をお持ち帰り頂けます。

対象：アフィリエイト広告を既に利用されている通販会社様

■16：10～16：40
売上が上がるのはどっち？　【A/Bテスト】から学ぶ売れる「最強のバナー」事例

株式会社 売れるネット広告社
東京副オフィス長 営業部 課長 チーフコンサルタント
荒木 達成

ウェブサイト(ランディングページ)の集客の入り口となるバナー。
バナーを制作するとき、デザイン構成やキャッチコピーに悩んでいませんか？
ただ、目立っている、カッコいい、美しいなどの基準値でバナーを選んでいませんか？
当講座では、“200億円”以上の広告費を使った【A/Bテスト】の結果から、目からうろこの本当にクリックされる、売上が増加するバナーをクイズ形式でわかりやすく紹介します。
明日からすぐに使える具体的なノウハウが満載です。

対象：広告出稿をされている通販会社様
