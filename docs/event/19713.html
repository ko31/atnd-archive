『キャンパスライフ』を見つめ直してみませんか？

あなたは、自分の大学生活に満足していますか？。

「ただ何となく大学に通ってる・・・」
「大学生活に、新しい刺激が欲しい！」

私たちは、
そんなあなたが新しい 『きっかけ』 を見つけられる出会い・交流の場を提供します！。

キャンパス・ビジョンでは、あなたの大学生活がより実り多いものになるような、
普段の生活では得られない素晴らしい 『出会い』 『きっかけ』 が見つかります。
そして、自分自身の 『キャンパスライフ』 を見つめ直すことができます。

ぜひご参加ください！

キャンパス・ビジョンとは！？

キャンパス・ビジョンは、皆さんがグループトークの中で
コミュニケーションする面白さを味わいながら
新しく何かを始める 『きっかけ』 を見つけるイベントです。

①「学習する」こととは？
②「活動する」こととは？
③「就活する」こととは？
という3つのテーマから1つを選択していただき、
テーマ別のグループに分かれて、グループトークを進めていきます。

あなたの『キャンパスライフ』を見つめ直してみませんか？

【キャンパス・ビジョンでできること】

1. あなたの新しい『きっかけ』が見つかる！
2. たくさんの頑張っている学生・教員と出会える！交流できる！
3. 人の話を聞く力・人に考えを伝える力が身につく！
4. 互いに刺激を与え合い、モチベーションがアップ！！

グループトークのテーマ

①「学習する」こととは？
皆さんは今、大学で何を学んでいますか！？そしてその学びに満足していますか！？
一口に「学ぶ」といっても内容は様々です。
学生にとって「学習する」という意味を再確認する必要があるように思います。
その意味を一緒に見つけてみませんか？

②「活動する」こととは？
「ただ大学に行って授業を受ける」・・・
それだけでは面白くない！という人も多くいると思います。
そんなとき、部活、サークル、バイトなど、やりたいことは人によって様々です。
大学の授業以外での活動が重視される現在、大学での4年間を充実させるためのに「活動する」意味を考えてみませんか？

③「就活する」こととは？
就職氷河期と言われる昨今、エントリーシートや面接対策などについつい意識がいってしまいます。
就職活動は、皆さんにとって将来を考えるとても大切な機会となります。
そこで、本当の意味での「就活する」とはどういうことなのか、改めて考えてみませんか？
