Keio.rb KEIO rails 開催のお知らせ

京王沿線で rails の勉強会開きたいなとつぶやいたら何人かから賛同いただいたので開催したいと思います。

■ 内容
勉強会というと内容を考えるのが主催する側の負担になります。そこで、Rails のちょっとしたテクニックや新しい機能などをスクリーンキャストで紹介する http://railscasts.com という非常に良質なコンテンツ（英語）があるので、これのエピソード（現在 224 まであります）をひたすら追っていくという形ですすめていこうと考えています。

とりあえず初回は最新の

Controllers in Rails 3
http://railscasts.com/episodes/224-controllers-in-rails-3

から始めて、昔のエピソードへ順にもどっていく感じかなと思います。

一方同時進行で「デザイナーのための Rails 勉強会」も進めてみたいと思っています。

プログラマー1人 + デザイナー1人 でサービスを開発するというゴールを想定していて、そのためにはデザイナーには 1) 基本的な開発環境の構築 2) svn や git といったバージョン管理の理解 3) Rails のビューやヘルパーを実装できる知識 4) prototype や JQuery といった Javascript ライブラリを使いこなせる知識、が必要だと思っています。

こうしたスキルをプログラマと一緒に勉強することで身につけられ、一方プログラマも基本的なデザインの知識が身に付けられる場になればいいなと思っています。

自分のブログエントリーで僭越ですが、

デザイナーのための Ruby on Rails + Windows 環境構築ガイド
http://tsukurusha.com/2010/07/ruby_on_rails_setup_guide_for_designers/

あたりを題材に、初回は環境構築あたりを一緒に勉強できればと思っています。
