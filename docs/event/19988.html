一日に4つの保育園をバスで巡る、就活イベント！
『第6回 ココキャリバスツアー』が10月15日に開催致します！

『ココキャリバスツアー』はバスで1日に4つの保育園を巡り、
現場の雰囲気を感じ、法人説明会を通して就職活動も行えてしまう。
保育学生のための一日限定の就活イベントです！

“施設見学”＋“法人説明会”＋“α”の
普通の就職活動では味わえない特別な時間を通して、
あなたの「理想の保育園」を見つけよう！

過去にはこの『ココキャリバスツアー』がきっかけで、就職した先輩もいます。
ただ悩んでいるよりも、
みんなで現場を見て、説明会を聞いて、気になる点はしっかり質問をしよう！

………………………………………………………………………
　　　　10月15日（土）ココキャリバスツアー
………………………………………………………………………

　今回は「東京都内」の4つの園を巡ります！
　1日に都内の特徴的な保育園を4つも見学できるので、
　魅力的な理想の保育園に出会えるかもしれません。
　内定が出ている方も、まだ就職先を探している方も、
　他にはない、園見学ツアーなので是非、ご参加下さい！
