イベント概要

couchbase-jp は今年1月に行われた CouchConf Tokyo 2012 をきっかけに誕生した、米Couchbase社認定の日本向けコミュニティです。

Couchbaseから発信される情報を日本語で展開する場所であり、
Couchbase Serverや周辺プロダクトについて、日本語で議論する場であります。
そして日本の声をCouchbaseに届けるのが目的です。

本イベントでは、couchbase-jpの第一回 meetup として、メンバの自己紹介やCouchbase Serverに期待するところなど、ざっくばらんに議論したいと思います。

初めての方がほとんどですので、お気軽にご参加いただけると思います。

Couchbaseって何？

簡単に言ってしまえば、次の様になります。
memcached → 永続化、スケールアウト → Membase Server → ＋CouchDBのインデックス ＝ Couchbase Server

Couchbase Serverは、memcached互換のメモリキャッシュで、データも永続化でき、CouchDBのインデックスによって、キー以外のデータでも検索が可能なNoSQLデータベースの1つです。
