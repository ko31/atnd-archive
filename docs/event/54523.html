description

「世界の恥」と言われながら、遅々と進まなかった電線類地中化。何としても国が動
かねば、と訴えてきたところですが、ついに明るい兆しが、「無電柱化議員連盟」の
決議、政権政党の専門委員会の中間とりまとめ、そして「～上を向いて歩こう～無電
柱化民間プロジェクト」の始動と、続いています。2020年という目標も掲げられてい
ます。
この際「日本の地中化はなぜ遅れているか」を明らかにし、早く、大きく前進させるた
めに何が必要か？技術・コスト・意識・制度そして各アクターの行動計画を具体的に問う場を設けます。
