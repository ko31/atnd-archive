catch 高周波ノイズによるトラブルの事例とその解決策を詳解し、望ましい高周波システムの設計条件を習得できる！！
description

【習得できる知識】
・高周波回路のまわりにできる電磁界がイメージできるようになる
・電磁波ノイズの発生原理がわかるようになる
・電磁波ノイズのトラブルシューティングの勘どころがつかめる
・望ましい高周波システムの設計条件がわかるようになる

【受講対象】
・高速・高周波回路、高速デジタル回路、ワイヤレスシステムなどの設計、開発、評価業務に携わる方
・同上のシステムの設置、管理、評価、運用などの業務に携わる方

【講座のポイント】
　コンピュータや電子機器の回路は小型・高速化が進み，機能も高度化してきました．また，周囲の電磁環境によっては「高周波ノイズ」によるデジタル回路などへの悪影響も問題となっており，電気自動車をはじめ，とりわけ人命に関わるシステムは，細心の注意が必要です．
　本セミナーでは，電磁波ノイズ発生の原理を詳しく調べ，高周波ノイズによるトラブルの事例とその解決策（トラブルシューティング）を，基本からわかりやすく解説します．

【プログラム】
１．高周波回路設計と電磁波ノイズ問題
　　1-1.高速と高周波の問題
　　1-2.パルスは撒き散らす
２．電磁界シミュレータでわかること
　　2-1.シンプルな線路のシミュレーション事例
　　2-2.Ｓパラメータの評価とクロストーク
　　2-3.グラウンド面の電流からわかること
　　2-4.基板がアンテナになってしまう？
３．電磁界シミュレータの手法とその特徴
　　3-1.時間領域の手法と周波数領域の手法
　　3-2.離散化と解析誤差の関係
４．プリント回路基板からの電磁波ノイズ
　　4-1.差動線路のシミュレーション
　　4-2.多層プリント回路のEMIシミュレーション
　　4-3.筐体に実装した状態でのEMIシミュレーション
５．高周波ノイズの対策
　　5-1.基板からのＥＭＩとその対策
　　5-2.配線からのＥＭＩとその対策
　　5-3.筐体からのＥＭＩとＥＭＣ
６．トラブルシューティングの勘どころ
　　6-1.最適な解析方法の選定
　　6-2.有効活用のポイント
７．まとめ　Q&A
