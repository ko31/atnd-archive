本イベントは主に東京西部、神奈川西部のエンジニアの交流を目的としたものです。
普段エンジニア交流会は都心部、平日に多くおこなわれており、それらの補完を目指しています。

pmとついている様に、Perlユーザーが多くいますが、話題はPerlに限定されているわけではありませんし、私を始め、PHPやJS、HTML等をPerlよりも使っている人多くいます。
エンジニアであれば、お気軽に参加いただければと思います。

特に若い人、お気軽に参加ください。

—
会場は八王子駅周辺のどこかになります。詳細は未定です。参加人数を見つつ予約を入れたいと思います。
前日までに会場をこのATNDに再設定いたしますので、確認をお願いいたします。
（もし八王子で良い店をご存じであれば、是非教えてください）

会費制ではありません、当日の飲食割り勘になります。
前回、前々回は3500円ほどでした。参考まで。
（上記は５時間分くらいです）

—
hachioji.pm #1、#2に引き続き、1枚LTを行いたいと思います。
前回同様ですが、「必須ではありませんが」楽しいので是非作製してみてはいかがでしょうか。
基本的に、席の端っこから端っこの人まで話そうぜ！というコミュニケーションを目的とした物です。

ガイドラインは
・発表する事は一ページ（A4など）にまとめること
・発表時間は１～２分（銅鑼とかないのでｗこだわりませんが）
・自分の名前（ハンドルでもいい）を載せること（コミュニケーションが目的なので）
・質疑応答をやればいいので、書ききらなくてもOK
・ネタ発表歓迎します
・このガイドラインはあくまでガイドラインなので、破ったっていいです。
・目的は皆に発表してもらうこと、コミュニケーションを推進する事です。

テーマは「バッドノウハウ」です！
たとえば…
「Perlの改行コードがCRLFの場合になんとかスクリプトを動かすには、
ln -s ~/perl5/perlbrew/bin/perl ./perl^M（Ctrl+Vの後にCtrl+M）
としておけばいいよ！」
等と言ったネタをお願い致します。
勿論、上に書いた通りガン無視して好きなネタ発表してOkです！

資料は最終的にPDF、またはHTML（どこかにアップしてください）にして頂ければ幸いです。
（Macなら印刷でPDFにできますし、WindowsでもPrimoPDFなどフリーのPDF出力がありますので、それらを利用頂くと簡単です。）
データをいただければ私が印刷します。当日印刷してもってきていただいてもOKです！

ウェブページや動画などをデモるときは、PCとプロジェクタをお貸しいたします。（会場によりますが、多分）
