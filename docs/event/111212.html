■開催主旨

2012年以降、GoogleはSEO対策においてコンテンツが最重要と提唱され、

さらに2019年からコンテンツSEOの重要性が一気に加速しました。

京都においてWEB上でサービスやプロダクトを提供するためには、

本来ブランド力が強い特性を生かしたコンテンツを提供することが重要です。

本イベントでは流入元から基づいたデータをキーとしたSEO対策

最適な「コンテンツSEOの取り組み」 / 「コスト削減」 / 「アクセス＋流入元」の3点を軸に、

京都の事例をご紹介し、CVを繋げるためのSEO施策に特化したイベントを開催致します。

■対象者【こんな方にオススメです】

・SEO対策についてある程度の知識を持っている方

・自社が提供しているサービスの潜在検索ニーズについて知りたい方

・最新のSEOトレンドが知りたい方

・今後注目されるであろうSEOトレンドが知りたい方

・アクセスアップ、売上アップに繋げるための道標が欲しい方

・自社で改善施策を企画する為の効果検証のやり方を知りたい方

・コンテンツマーケティングの運用方法

・アクセス流入を持続性させる運用方法
