description

Box竜之介は、動物愛護活動の一環として、
ノラ猫ちゃんの不妊手術を実施いたします。

減らない殺処分と、猫の習性の不理解からなる
善意のエサやりによる野良猫の増加という
状況を打破するために、不妊手術運動(TNR)を大々的に行います。
野良猫を対象に、不妊手術を行うことで
個体の増加を防ぎ、殺処分を減らし、
ノミ駆除薬により地域の衛生を考えます。

＜期間＞
2015年11月22日（日）〜2015年11月29日（日）
※この期間以外は通常通りの金額となります。

＜必須条件＞
1.野良猫である
2.ノミ駆除（実費必要）をする
3.耳カットする
4.キャンペーンの趣旨に賛同する
5.日帰りが原則となります
※体重1kg未満の子や、体調が悪い子は手術ができません

＜費用＞
不妊手術(無料)　+　外部寄生虫駆除代（実費必要）

＜お申し込み方法＞
竜之介動物病院まで、お電話で事前にご予約をして 下さい。詳細をご説明さしあげます。
