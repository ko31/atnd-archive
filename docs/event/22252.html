description

「神戸ロボット工房」は、神戸市が長田区の旧二葉小学校の校舎（神戸市立地域人材支援センター）を活用して、平成２２年１１月１９日にオープンさせました。
ロボットについて気軽に楽しんで頂ける「ロボット展示室」や、工作教室やＲＴ関連ビジネスの打合せなどに活用いただける「ロボット工作室」を設けました。

さて、この度、神戸ロボット工房は1周年を迎えました！

この1周年を記念して、11月27日（日）に「神戸ロボット工房」1周年イベントを開催します！

このイベントでは、レスキューロボット（災害対応ロボット）のデモや東日本大震災での活動紹介があったり、
モーターカーを自作し走らせたり、SEGWAY（セグウェイ）に試乗できたり、二足歩行ロボットやおしゃべりできるロボットと触れ合ったりもできます。
その他にもいろいろありますよ！

また、この日、地域人材支援センターでは、『まちの文化祭2011』も開催されております！
地元商店街を中心とした屋台が出ていたり、舞台パフォーマンスもあったりしますよ！
http://www.city.kobe.lg.jp/information/press/2011/10/20111024070402.html

お子様からご年配の方まで1日楽しめます☆

是非お越し下さい♪
