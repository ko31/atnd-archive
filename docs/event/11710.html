統計解析・データマイニングの方法論を用い、蓄積されたデータを有効活用していく方法を学ぶ勉強会です。
今回は、大規模解析・機械学習・クオンツ 祭りです！

※Twitter タグは #TokyoWebmining です。
※当日、迷ったり、入れないときは#TokyoWebminingだけでなく #nifty_engineer や @nifty_engineer を追加してツイートするとニフティさんが助けてくれます。


目的：
データマイニング・統計解析の手法を用い、蓄積されたデータを有効活用していく方法、および、WEB上に蓄積されたデータの活用に関し、皆で学び共有していくことを目的としています。
詳細:
Slideshare: 創設の思い・目的・進行方針

概要：
各データマイニング・統計解析の手法、各種WEB APIの活用に関し、基礎的な内容から学べる場にしたいと思います。
各方法論・関連トピックについて、分野の初心者方でも分かりやすい内容を意図し、進めていきます。

■大規模解析：
1. Mahout Canopy Clustering (講師：@hamadakoichi)(発表30分＋議論60分)
　Canopy Clusteringは通常の多くの手法と異なり、クラスタ数指定を必要とせず、指定距離 離れたクラスタ算出を実現する。
　Hadoop上で動作する大規模データマイニング・機械学習ライブラリ Mahoutでの実行法も含めお話しします

2. 機械学習＝機械の代わりに人間が学習 (講師：@shuyo))(発表20分＋議論40分)
　Gihyo.jp でも機械学習の連載し裾野を広げる活動をされている @shuyo さん。
　今回、機械学習の歴史や専門外エンジニアがどのように学んでいけばいいかに関し、お話し頂けます。

3. 安全にスケールするログ解析システム構築の勘所 (講師:@yamaz)(発表20分＋議論40分)
　大規模ログ解析のシステム構築には実は隠された 技術・勘所がある。
　今回、大規模広告配信エンジンを開発されている @yamaz さんにその技術・勘所に関しお話し頂けます。


■クオンツ：
4. コピュラと金融工学の新展開 (講師：@teramonagi)(発表20分+議論40分)
　コピュラ関数は、多変数の分布関数と周辺分布関数の多様な依存関係の記述を新たに可能にする。
　今回、クオンツ @teramonagi さんにコピュラ活用、金融工学の新展開に関しお話し頂けます。

5. 賃料相場のデータマイニング (講師：@rinzo_rinrin)(発表20分+議論40分)
　大規模賃貸データの解析をされている @rinzo_rinrin さん。
　賃貸データ解析の勘所に関し、お話し頂けます。

※ペアトレードを実装してみた (講師：@yokkuns)(発表15分+議論15分)は 2nd Week へ延期となりました。
