description

キラリと光る地域には、地域を引っ張る頼もしきリーダーが存在する。

「ヒムカレッジ」は、地域の課題と真っ向から向き合い、
先駆的な取り組みを実践しているリーダーたちから、地域にある課題に対する
アプローチの仕方を学ぶ月１回のカレッジ（定期講座）です。

地域の「今」を知り、これからの宮崎、そして地域づくりについて
ちょっぴりマジメに、だけどフランクに語り合いませんか？

講座の概要

  • 日　時：2013年2月8日（金）19:00-21:00（18:45開場）
  • 会　場：みやざき県民協働支援センター
  • 定　員：40名（参加費無料）
  • 申込み締切り：2013年2月6日（水）

講演概要

地域ブランドを魅せるチカラ！
綾町の地域活性化アドバイザーが語る情報発信のコツ

講師：中島 淳さん（株式会社カルチャーアットフォーシーズンス 代表取締役）

講師紹介：1988年株式会社リクルート入社。主にマーケティングの基本と組織活性化に向けた人材育成を担当。民間のシンクタンクにて地域づくりに関するコンサルティング等に従事。2001年実践に近い地域活性化マーケティングを目指し独立 。現在、綾町倉輪地区活性化のアドバイザーを担当している。
http://www.soumu.go.jp/main_content/000027261.pdf

こんな人におすすめの講座です！
