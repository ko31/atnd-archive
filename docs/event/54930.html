description

文章を書くうえで、ヒントが欲しいという方のための教室を「画材屋R」にて初開催します。

これからブログをはじめようと考えている方をはじめ、文章をクオリティアップさせるためのノウハウ、ブログに載せる写真のちょっとカッコイイ撮り方など、編集者として新人ライターに語っていることをそのまんまお話しします。

そのぶん実践的な内容も含みますが、あくまでも初心者の方を対象に「Webライティング入門」といった趣きの教室になります。

出版業界のお話もしますので、書籍を出したい方、ライターになりたい方に役立つ情報も盛り込みます。

また参加者で、映画を題材にした電子書籍を年内に発刊しますので、
いきなりライターデビューの可能性もあります。

また、当日用意します教材準備の関係で、なるべく参加申込のうえ、おいでいただければ幸いです。
もろちん飛び込み参加も歓迎です。

ただし、教材の準備にお時間をいただくことがありますので、ご了承ください。

今回は、以下のプログラムを予定しております。

■書きはじめる前の準備
■ネタ探しのコツ、継続するコツ
■文章を書くときのノウハウやチェックポイント
　サンプル原稿を推敲して、完成原稿を目指します
■ブログやSNSに載せる、ちょっとカッコイイ写真の撮り方
■Webライターなら誰でもなれる！
■雑誌・書籍ライターになる方法
■書くならば、いっそ書籍化を目指しちゃおう（書籍出版の企画書の書き方）
■電子書籍（Amazon Kindle）をみんなで出そう！（即デビュー）
■開催日時：8月30日（土）15:30〜17:30

■開催場所：画材屋R
