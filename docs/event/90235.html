■ セミナー詳細

思い出してみてください。
もし仮に、あなたの人生で最もモチベーションが高かった時があったとするならば、それはどんな時だったでしょうか？

受験に向けて猛勉強していた時でしょうか？
それとも、仕事に打ち込んでいた時でしょうか？
趣味に没頭していた時かも知れません。

そんな、モチベーションが高かった時には、必ずモチベーションが向かう対象があったはずです。例えば、受験であったら「○○大学合格」とか。目標と言ってもいいかもしれません。

多くの場合、モチベーションがあるときには目標が明確になっています。目標が無いことがモチベーション欠如の原因になっています。

この講座では、科学的に立証されている方法で、あなたのモチベーションの対象となる目標設定をし、燃えるようなやる気とモチベーションを作り出していきます。

当講座の大きな特徴は再現性の高さです。
なぜならば、
①ワークを通じて、実際にモチベーションを上げてもらうプログラムだから（跳ねたり、叫んだりはしません）
②科学的なモチベーションアップの方法を体系化した、誰がやっても同じ結果が出るプログラムだから
③数多くの人に体験してもらい、改善を繰り返してきたプログラムだから

多くの方が、モチベーションが上がらない為に、自分の能力を発揮できずにいるのではないでしょうか？とても、もったいないことだと思います。

・自分の能力を思う存分発揮していく為に
・毎日を充実感に満たされて送る為に
・自分らしく、堂々と生きていく為に

当講座を、そのきっかけとなる大切な機会としてお役立てください。
