□　「開運方法」「神社参拝　「占いの本質」について学べます。 バス内のセミナー後　「三嶋大社で昇殿参拝」を体験します ＜バス内特別セミナー＞ ＊三嶋大社への車中でセミナーを開催致します。 この　体験講座では ＊＊＊＊＊＊＊＊＊＊＊ ●　「究極の神社開運方法とは」 ：「神社参拝開運法」 ＜神社開運について＞ 日本各地にパワースポットと言われる地が沢山ありますが、 実際に、その地に出向きパワーを受け取るのは簡単ではありません。 パワーを受け取るために感応しなければならないからです。 しかし、パワースポットを神社と捉えるのなら、 誰もが手軽に神社参拝の功徳という事ででしょうか。 本当の神社開運について詳細に解説します。 実際に「三嶋神社」で昇殿参拝を体験します。

●「三嶋大社のひみつ」とは ＜三嶋大社について＞ 富士山のお膝元にある三嶋大社。 三嶋大社は源頼朝が源氏再興を祈願した神社でもあります。 何故、三嶋大社に源頼朝が源氏再興を祈願し、 そして見事に再興したのかをじっくり考えていきたいと思います。 個人レベルでは、人生、生活の土台が出来るというそのわけとは。 なぜ　今回は「三嶋大社の参拝」なのか・・・

●　「占いと占い師について」 ＜占い鑑定について＞ 「鑑定法伝授会」 数多ある占いは、三つに分類されます。命・卜・相。 命・卜・相の捉え方と活用の仕方を伝授。 そして、占い鑑定を受けるコツ。 さらに今回は特別に、占い師側からの正しい占い鑑定の仕方について 細かく解説致します。 占い鑑定を正しく行えば徳を積み、間違って行えば劫を積む。 その分岐点とは何か。 おまけに、占い師のなかには運が悪そうに見える その理由もお知らせしましょう。 ＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊ ＜スケジュール＞ ２０１６年　７月１８日（月祝） ９：００　新宿発　車中セミナー １２：３０　三嶋大社参拝（正式昇殿参拝） １３：００自由行動 １５：００三嶋大社発　直会（車中呑み） １８時００分　新宿着 １DAY　参加費　10580円 　税込み

□　講師プロフィール　□ 日本大開運協会 代表 遠藤裕行 大学卒業後、運命学に強く関心を抱き気学、姓名判断を学び始める。 研鑽を積み、占い店の専属占い師としてデビュー。さらにタロットの 専門家、手相専門家にそれぞれ師事し、占術の幅を広げる。 その後有名店を歴任し、会社経営者、芸術家、芸能人、漫画家等、 会社組織に属さない自分の運勢で人生を賭けている人達に絶大な人気を誇る。 現在までの占い師活動で５０００人以上を鑑定。 その経験から得た占いの素晴らしさを一人でも多くの方に伝えたいと 日本大開運協会を発足し、プロの占い師を養成している。
