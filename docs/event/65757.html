生徒さん達の要望に答えてできた、実践的なWordPress講座。

既存のテーマを利用するWordPressの講座はたくさんあるけど、オリジナルのテーマを使ってサイトを完成させる講座やセミナーが無いので、やってほしい！という声を聞き、キヅキが早速作りました。

今回はHTML/CSSはわかるけど、WordPressはよくわからない！phpとかムリ！MySQLって何ですか？という方のためのハンズオン講座です。

二日間の講座でオリジナルデザインのテーマの制作、ブログの作成からカスタムをしてコーポレートサイトを講師と一緒に作成していきます。
WordPressを勉強したい人にも、WordPressでサイトを作りたい方にも大満足の二日間です。


▼こんなことが学べます

・WordPressのインストール
・主要テンプレートタグ
・オリジナル静的ページの組込
・パーツを部品化していこう
・記事を投稿してみよう。
・テーマやプラグインのインストール
・個別ページと固定ページの制作
・カスタムメニュー（ナビゲーション）を作ろう



▼こんな方にオススメ！

・HTML/CSSの知識を踏まえてスキルアップしたい方。
・WordPressでコーポレートサイトを制作したい方
・ビジネスでWebサイト制作をしていきたい方
・オリジナルデザインのブログを作りたい方



▼kizukiPlusのWordPress講座はココが違う！


・基礎からカスタマイズまで

オリジナルテーマ（テンプレート）の作成が理解できます。

・サーバーを借りなくてもOK！

仮想サーバーを使用しますので、サーバー契約などを事前にお願いすることはありません。

・コーポレートサイトを作ります

一般的にWordPressといえば、時系列で表示をするブログを想像する方が多いかもしれませんが、

この講座では、企業のWebサイトを題材に制作をしていくハンズオンセミナーです。
もちろんブログページも作れますので、個人のブログを構築したい方にもオススメです。

・ノンエンジニア/ノンデザイナーを対象としています。

専門用語は少なめに、Web制作初心者でもわかりやすい講義で進めていきます。
講座で講師二人体制でサポートしますので、わからなくなったり、困ったときにも安心です。

・もちろん作ったサイトはお持ち帰りOK！

商用/個人問わずに制作していただいたページを、そのまま運用していただくことができます。



▼参加資格

▼講師紹介

石川健太 講師


WEBの可能性と面白さに惹かれ、複数のWEB制作会社で、デザイン、コーディング、システム、フラッシャー等の業務に従事。
その後、「WEBの面白さを多くの方に伝えたい」という思いからWEB講師となる。
これまで約300名近くのWeb未経験者を、短期間でWeb制作現場で働けるレベルまで指導。
これまでの指導実績はXHTML、HTML5、CSS、CSS3、Photoshop、Illustrator、Flash、Javascript、jQuery、PHP、MySQL、wordpress、SEO等。
初心者に必要なWeb制作スキル、ほとんどの指導を得意としている。

▼kizuki Plus（キヅキプラス）とは？

厚生労働省認定の職業訓練校を運営しているキヅキWEBスクール主催の、WEBデザイナー、クリエイターを目指す社会人、学生向けのセミナーです。

初心者から「現場で生きる技術」を身に付けることをモットーに、「質実剛健」な授業を行います。

HTML+CSSの基礎から、スマホサイトやレスポンシブWEBデザインで必須のHTML5+CSS3、jQueryやjavascript、PHP、Wordpressまでの初心者向けの基礎から実践で生きる応用スキルまでを学べる講座を今後随時行う予定です。
講座情報については随時、Twitter、Facebookで告知していきますので、是非フォローしてみてください。
