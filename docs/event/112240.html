ED治療薬を服用するタイミングについて
ED治療薬を服用する際に気を付けなければならない事の一つに服用のタイミングがあります。
よく、カマグラはやっぱり効果がない、副作用も気になる、、、、やっぱり海外通販で買ったからな？

等という口コミを目にしますね。
ですが、効果がないと感じる前に注視て欲しいのが、
①空腹時に服用してください。
　　胃の中に食べ物がある状態で服用しても効果が激減します。
　　　食前30～60分前が良いです。
②ミルクなどで服用しない。
　　胃に膜ができるのでカマグラの効果を阻害する可能性があります。

また、副作用を強くするとしてグレープフルーツジュースと一緒は厳禁です。
一般的なお薬によくある注意事項と同じです。
柑橘系のフルーツの皮には薬の効きを強くする働きがあります。
皮を剥いたらカマグラと一緒に食べても大丈夫です。

1日/錠で安全にカマグラでED治療とパートナーとの深いコミュニケーションを楽しんでください。

カマグラでED治療をしたいなら、厚生労働省も許可している個人輸入代行の三越をご利用ください。
