description

初めての試みだった前回のダンス練習オフはお陰様で大好評！！
ということで、2回目を開催いたします！
都内のダンススタジオを借りて、みんなで一緒にプリキュアダンスを練習しよう！という企画です！
目標は 『参加者みんなが一曲通して踊れるようになろう！』
カンペキではなくても、皆で通して踊ることができればきっと最高の充実感を得られるはず。
全く踊ったことがないという初心者から、自信を持って踊れるという上級者まで、皆で集まって一緒にダンスしてみませんか！？

日時と待ち合わせ場所

12月11日（日） 14:45 に西武池袋線 江古田（えこだ）駅 改札前に集合
会場までは徒歩6分程です。集合次第、幹事がご案内します。
※当日の遅刻・キャンセルのご連絡は、幹事@speedmodまでお電話もしくはメール、DM、リプライなどの方法でお伝え下さい。

練習内容

対象曲

「H@ppy Together!!!」（フレッシュプリキュア！後期ED曲、約1分30秒）

前回の参加者の方々の間で要望が多かった、フレッシュプリキュアの後期ED曲を今回の練習対象としたいと思います。
2年前のアニメとは言え、お隣台湾では現在放映中＆アニマックスではニチアサの時間帯に2話ずつ放映中だそうですね。だってそれが永遠不滅プリキュアよ♪（それ５）

プログラム予定

  1. 準備運動
    体育の前の準備運動です。
  2. 模範演技
    ダンス映像をプロジェクタで流しながら、講師陣＋既に踊れるという方に見本として、前でダンス映像に合わせて踊ります。
  3. パート練習 (1時間程)
    パート毎に分けて練習を行います。講師が前でポイントを伝えつつ模範演技→皆さんで練習、という流れになります。ここで体の動かし方のポイントを掴んでもらいます。
  4. 休憩(10分)
  5. 通し練習（スロー）
    全体を通してスローの音楽に合わせてダンス　パート毎の練習が終わったら、一度曲をゆっくり流して踊ってみます。
  6. 通し練習（通常スピード）
    通常スピードで皆で一緒に踊ります。
  7. *動画撮影、鑑賞*（ネット等に出回らないよう、動画はその場で削除いたします）

※変更することがあります。
