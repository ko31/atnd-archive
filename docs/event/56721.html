２０１３年に北海道内の学習塾経営者など教育に関わるメンバーで、『教よりも育』を重視し、人財を育てる新しい枠組みを生み出していくと取り組みを続けている北海道教育ユニットＥｄｕ。

学ぶ人の可能性を広げる「きっかけ」を生み出す・学ぶ人の「やってみよう」を全力で後押しする・学ぶ人のためにできることを広げるために、まずは自ら行動する教育者ユニットです。北海道から、自ら道を切り拓く学ぶ方を応援し、育つ環境を生み出す活動を行う教育ユニットです。

http://edu-hokkaido.com/

そんなＥｄｕメンバー間の交流、そして一緒に活動したいなと考えておられる方に集まって頂き、ちょっと気軽に、飲みながら楽しく自由に語る日があってもいいよね、ということで企画しましたその名もわかりやすく『Ｅｄｕ札幌交流会』。

教育パワーランチやＥｄｕの様々なプロジェクトに参加しているメンバー、またＥｄｕのこれからの活動に興味を持っている方でちょっとゆったり飲んだり鍋でもつつきながら語りませんか？

～こんな方は参加すると楽しいかも？～
●Ｅｄｕの様々なプロジェクトにピンッと来た方
●『やってみたい』ことがあるけどきっかけを探している方
●Ｅｄｕにはどんな人が集まってるのか、飲みながら腹を割って話してみたい方

今回の会場は人財を『育てる』ことに力を入れるコワーキングスペース『育てるコワーキング札幌』。Ｅｄｕの活動拠点のひとつであるこの場所はＥｄｕのプロデューサーがプロデュースするコワーキングスペース。コワーキングに興味があったんだ、という方もこの交流会を機会に一度訪れてみませんか？
