description

薄毛や白髪は実は年齢のせいだとあきらめてはいませんか？それらの原因は実は加齢だけではなく、日々の食生活やストレス、また間違ったヘアケアが頭皮を不健康にしている原因のひとつになっています。

そこで、国際毛髪科学研究会では毎年、頭髪の日を記念し、当会長井上哲夫のセミナーを開催させていただきます。

1.セミナー概要
頭髪をスタイル、美しさだけで無く、健康のバロメーター的角度からの頭皮ケアも追求したセミナー。セミナーのポイントは以下3点を予定。
1)シャンプ－の必要性、方法
2)髪の発生メカニズム
3)継続の力 ※マイクロスコープを活用したデモも実施予定

【参加者特典】
・参加者全員に「ヘッドスパトリートメントカラー」を提供
・男性参加者全員に「ＴＩスカルプコントロ－ル　頭皮ケアコ－ス」無料券を提供（有効期限2012年12月末）
・プフレ頭皮ケアシャンプ－を女性参加者全員に提供

2.セミナー講師
髪の悩みを機材・器具・育毛剤といった外的要素に捉われない生活アドバイスを含めた育毛アドバイスの研究・活動を積極的に行っている『国際毛髪科学研究会』の会長である井上哲夫によるセミナー。（プロフィール、活動実績等は http://www.moukaken.jp/profile.html をご参照ください）

井上は、東洋医学を美と健康に応用する「理論漢方」のオーソリティとして、現在までに7000店以上の理美容店で指導実績を持つ。育毛研究25年・専門分野に取材多数。

“健康で美しい髪を育むのは健康な頭皮”という持論の元、健康な髪がつくられるしくみや正しいホームケアの仕方（頭皮マッサージ、シャンプー、ドライヤー、ブラッシング、市販の育毛剤の使い方）など、美しい髪を育む頭皮マッサージの方法も伝授。頭皮の血行を促進し、美しい頭皮&髪を手に入れてください！
