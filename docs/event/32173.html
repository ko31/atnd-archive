title 第50回「WEB TOUCH MEETING」やります！
catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション1(40分)
　『いまさら実演で見てみる、Illustrator CS5とCS6の新機能　ほか』
　　川谷制作の川上さん

　＜主な内容＞
　AdobeのサイトなどでIllustrator CS5、CS6の新機能の一覧を見た人は多いと思いますが、
　実際に使っている所を見た人は多いのだろうか……
　そんなあやふやな「かもしれない人達」に向けて、実際にその場でその新機能を使って
　「へー」とか「ふーん」とか言ってもらう内容です。

　あと、IllustratorやFireworksに入れて便利だった機能拡張なども紹介していきます。
　ネタが溜まっていれば「あと5分寝るためにTips」をやるかもしれません。

　＜川上さんの紹介＞
　川谷制作のWTMによく顔を出す方です。
　DTPが長かったのでイラレ歴はそれなりにあるのですが、
　最近はFireworks触ってる時間の方が多いです。
　だからといってFireworksに詳しいわけではないので
　(特にコード書き出し部分)、あまり振らないでー

　川谷制作
　http://kawa-tani.com/

　Adobe Illustrator CS6
　http://www.adobe.com/jp/products/illustrator.html

　

●セッション2(30分)
　『SNS「github」で遊ぼう』ひむらさん

　＜主な内容＞
　普段使う SNS といえば、そう github ですよね。
　え？ エンジニアなのに github のアカウントをもってないって？
　いますぐ作りましょう。
　使い方がわからない？ 気にしたら負けです。
　Facebook なんかでもアカウントをとってから1年ぐらい放置しませんでした？

　そんなわけで
　- github で どんなことができるのか
　- 私が日々どんな風に github を利用しているのか
　を紹介します。

　＜ひむらさんの紹介＞

　プログラムとかかいてます。Linuxとかいじってます。
　iOSアプリとかつくってるときもあります。
　gitとかHaskellとかEmacsとかRubyとかがマイブームです。
　あと Hiroshima.rb の会場確保係とかもしています。

　http://eiel.info/
　http://hiroshimarb.github.com/

　

●セッション3(40分)
　『Windowsストアアプリのデザイン攻略
　　-Windows8公開前にアプリをストアに載せる方法-』
　　システムフレンドの西村さん

　＜主な内容＞
　発売まであと一か月ちょいとなったWindows8。
　本セッションではWindows8のデザインの仕事が来た時に役に立つ知識と、
　アプリをWindowsストアに載せるためのノウハウをお伝えします。
　もうすぐ発売のWindows8のアプリケーションに
　いち早く着手してアーリーアダプターを目指しましょう！！

　＜西村さんの紹介＞
　広島でWindows Phone、Windows 8の開発勉強会をやっています。
　EC-CUBE公式エバンジェリストもやってます。　

　株式会社システムフレンド
　http://www.systemfriend.co.jp/

　Windows 8 Release Preview
　http://windows.microsoft.com/ja-JP/windows-8/release-preview
