description

・ブログを使って自分の商品の見込み客を増やしたい

・ブログで集客をしたいコーチ・コンサルの方

・自分のお客様になってもらえるための文章力がほしい


という自分の商品を売りたいコーチ・コンサルの方が

ブログで集客するためのライティングセミナーです

＜特にこんな悩みを持った方へオススメです＞


● 書くことが苦手で、１つの記事を書くのに４時間もかかってしまう
● やっと記事を書いても、コメントも全く来ず反応がない
● 毎日一生懸命記事を書いているが、読者が増えない
● ブログやメルマガで告知を出しても、まったく反応がない
● アクセスが全然伸びない・・・
● ブログを続けても結果が出ず、すべて無駄な時間になってしまうかもしれない
● わかりやすく、読みやすく書けない
● 表現の仕方がどうしても同じになってしまう
● 読者がよろこぶ書き方が分からない


＜セミナー内容＞


● あなたのブログからお客様になるまでの３つのカベ
● 読まれるブログと、読まれないブログの違いとは？
● コメントやお問い合わせがじわじわ増える “３ステップの書き方”
● 殆どの人がつまずく書き方の“落とし穴”とは？
うまく文章を書けない４つの原因とその対処法
● 事例を使った４つの対処法の活かし方
● なぜ多くの人が、書き方を学んでも文章力が向上しないのか
● このセミナーでお伝えする文章力が
「あらゆるビジネス上の問題を解決することができる」その理由とは？


＜このセミナーを受けたらこうなります＞


● あなたのブログに読者が集まり出します
● ブログをスラスラと書けるようになります
● その結果、お問い合わせ、セミナー申し込みの数が増えるようになります
● 何度もリピートしてくれる強力なファンを作ることができます
● 時間短縮と響くブログが書けることが同時に叶います
● 言葉一つ一つに繊細な感覚が身につくので、
その時々に合わせて読者に響く言葉を選ぶことができるようになります
● セミナー告知文を打てば、たちまち申込みがやってくるようになります
● アクセスやコメントが増え出し、セミナーや無料相談
といった商品にも問い合わせが来るようになります
● たちまち読者をあなたのブログの虜にしてしまいます



今回なぜ僕がこのセミナーを開催するに至ったのか




今回のセミナーは僕の全てが集約されています。

ですので、
これから僕が伝えていくセミナー内容を作っていった
その背景について少し話をさせてください。



僕は当時、少し変わった子供でした。


中学生ぐらいまで友達ができず、苦しい時代がずっと続いていました。

そうなってしまった理由は主に２つあります。

１つめは、自分の感情を抑えられず
周りの人とよく衝突してしまっていました。

上面を良くするだけではこの問題は直りませんでした。

友達がほしくて、気を引きたくていっぱい嘘もつきましたが
それでも結局衝突してしまって友達ができなかった過去があります。

本当に自分自身を変えていくしかなかったので
目の前に出された問題を深く見つめる癖がつきました。

その時、無自覚にあらゆるイメージを
言語化していくことに努めていきました。

ここで得たその言葉にする感覚は
セルフマネジメントや自分を正確に伝えるときなど、

あらゆる場面で大きな土台となりました。

もう１つの理由を挙げると、
相手の顔の表情から感情を
読み取ることができませんでした。

これを言うとよく驚かれるのですが、
実は、それは今でもできません。

しかし、それを克服するために考え抜きました。

相手が使っている言葉に感覚を研ぎ澄ませ、
その人が使っている言葉から
その考え方や性格を分析するようにしました。

その独自で磨いた言葉の感覚から、
相手の気持ちを理解できるようになり、
沢山の人と繋がれるようになりました。

更には
様々な方々との信頼関係を築いていき

当時からは想像もつかないくらい
色んな人と出会うことができて本当に幸せです。


僕は、言葉の力には人生を変える力があることを知りました。

この時点ではまだ言葉の力が
僕の才能だとは気付いていませんでしたが、

この人生が変わったことをキッカケに
人の人生をより良くしていくためのお手伝いがしたい

という想いが芽生え、起業を決意しました。



最初はどうやって起業すればいいのかわからなかった


しかし、この時はまだ「どんな手段で」
人に貢献していけばいいかわからず

自分のビジネスモデルを作れないまま
ただもんもんと、成功したい想いだけが先行したまま
先が見えず不安と闘う日々が続きました。


しかし、ひょんなことから
僕の文章力が評価されました。

「ライティングスキルを本気で勉強すれば
人に教えられるようになるのではないか？」

と思いました。



ライティングスキルで起業することを決意


衝撃が走りました。

「どんな手段で」人に貢献すればいいのか分からなかった
当時の僕に一番ほしかったアドバイスでした。

この才能を誰かの役に立てることはできないかと
勉強と工夫を重ね、試行錯誤を繰り返しました。

そうすることで蓄積された僕のノウハウを活かし、

様々な文章に関わる悩みを持った方々に対しコンサルをした結果、
それぞれが抱えていた様々な問題を解決していきました。



これまでのお客様からこんな声をいただきました。


そして、これらは一部になりますが、
これまでのクライアントからこんなお声をいただきました。

「ブログを書く際に、どう書けばいいかが分かった。思い込みが取れ、あまり難しく考えずに書けるようになった」（山田さん・男性・コーチ）

「文章力がすごく上がった！思考整理され時間短縮にもつながった」（原田さん・女性・イラストレーター）

「タコの足とかの話を聞いて納得した！これからブログなど、文章がちょっと書ける気がした！」（岡本さん・女性・恋愛コンサルタント）

「感情表現ができていないという指摘は以前から受けていたが、実際にどのように表現していけばいいのかが分からなかった。なので、実際に川畑さんと話すことで、どのように書いていけばいいかが分かった。」（Ｓさん・男性・難関資格浪人生専門「合格請負人」）


僕が作り上げた独自のメソッドは多種多様な方々の
文章力を底上げできるものだと確信しました。

さらに多くの人を救いたい。

ブログの書き方が分からなくて困っている方々を
１人でも多く救うためにセミナー開催を決意しました。

コーチ・コンサルの個人事業主の方が
困ることなくブログを書けるような
ライティングスキルを伝授します。


セミナー参加者だけの特典


●無料の６０分コンサルティング（２０，０００円相当）
●記事作成時間を大幅に減らし、アクセスが増えてくるブログ記事作成テンプレート配布（１０，０００円相当）



僕が一番やりたいこと、
それはライティングを教えることではありません。

ではいったいなにをしたいのか？

それは、
あなたに最高の人生を送ってもらうことです。

そのキッカケとなる存在でありたいのです。


それを実現できる方法が
僕の場合、人に文章を教えることでした。


文章の書き方が変わるだけでも
あなたのビジネス規模は全く変わります。

その理由は、
これだけインターネットが発展し続けている中で
その多くは文字というデータで成り立っています。

ライティングスキルと呼ばれるものは
今後ますます重要になっていきます。

生涯でかせぐ金額が「億」単位で
変わってしまう可能性すら秘めています。

そのかせぐ額が変わることの背景には、

あなたを慕う人がどれだけいるのか
あなたがどれだけの人を救ってきたのか

ということの指標になります。


想像してみてください・・・

このセミナーを受けた後
あなたのブログの“書き方”が変わります。

そして読者は少しずつ増え始め
セミナーやコンサルへの申し込みが出始めます。

その後、アクセスもお問い合わせも増え続け
あなたの助けを必要とする方がどんどんやってくるようになるのです。

あなたはやりがいと充実感に満たされていきます。

あなたを縛っていた不自由なものは
ひとつひとつなくなっていきます。

そして、
あなたを縛っていたものすべてが外れた時
本当の自分を手にすることになります。


そのすべての基盤となるものが“文章”なのです。

セミナーを開催する時にも
コンサルをするときにも

初めての人ほど必ず
セリフを書き起こしますよね。


その時に文章力があれば
あなたの商品ひとつひとつが、
間違いなく価値のあるものへと進化させていきます。


何をするにしても、文章力は必要になってくるのです。


ぜひ、あなたの人生を変えるための
お手伝いをさせてください。

僕の全てを詰め込んだセミナーを
あなたにお届けさせていただきます。
