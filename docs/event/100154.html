description

ゲームと聞くとスマホでゲーム、パソコンやPS4、Nintendo Switchなどでオンラインゲームが主流となっております。現在はゲームは一人でやるか、複数人でやるといっても直接会うことはなさそうです。

しかし！みんなで集まってワイワイゲームしてた時代があったんです。

家庭用ゲームとしてファミコンが登場し爆発的にヒットした1980年代、友達の家に集まってゲームをやったり見たりしたことを懐かしく思っている人も多いと思います。

そこで、この度、レンタルスペースを借りてあの頃のように集まってワイワイゲームする企画を用意しました。

使うハードはスーパーファミコン クラシックミニです。星のカービィ スーパーデラックスやF-Zero、マリオカート、スト2など盛り上がるゲーム満載です。

そういえばあの頃はゲームを見てるだけでも楽しめましたね。やるのも良し、見るのも良し、ゲーム画面見ながら談笑するも良し。
昔ながらのゲームの楽しみ方を満喫しましょう。

終了後は有志で食事会も予定しています。
