description

Webは改善していくものです。
でも自分だけで、いつものメンバーで考えていては見えてこないこともあるかもしれない…

そこで、とあるサイトをモデルとして、参加者みんなで改善について話し合おうという企画です。

モデルになるサイトは
この勉強会に常連で参加していただいている方の勤務先のサイトで、サービスのコンバージョンを目的として実際に運用されているサイトです！
（許可をいただいております）

・今まで一人で改善点を考えていた方
・Webサイト改善の視点について知りたい方、経験したい方
・普段とは違うメンバーでディスカッションしたい方
・新たな気づき、他者の視点を知りたい方

是非一緒に話し合いましょう！

･･･････････････
★企画の狙い
･･･････････････
全員で一つのWebサイトについて改善点を話しあうことを通して
Webサイト視点や気づきを得ること。

･･･････････････
★企画詳細
･･･････････････
数人のグループに分かれ、ディスカッションを行います。
なるべく多くの視点を得られるように途中で班替えをします。
2回目の班で視点の発表を行います。
