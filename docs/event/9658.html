GaN系半導体レーザの材料物性と素子高性能化

イベント ユーザー
event_id 9658
title GaN系半導体レーザの材料物性と素子高性能化
catch 高性能緑色レーザの実現に向けて克服すべき問題点やその解決策についての議論を行う！
description

＜講座のポイント＞
　高密度光ディスク用光源として実用化されているGaN系半導体レーザは、ディスプレイ用途の純青色や緑色レーザへの展開も期待されている。こうした展開も含め、GaN系レーザのさらなる高機能化には、GaN系材料（特にInGaN量子井戸）の物性の理解が不可欠である。

　本セミナーでは、GaN系材料に特徴的な物性についての基礎的な解説を行い、レーザ特性との関連について詳説する。そして、特に高性能緑色レーザの実現に向けて克服すべき問題点やその解決策についての議論を行う。
