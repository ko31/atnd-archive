そんな中今注目される投資実践法は？
一時は大きく注目されていたが
低迷し続けている金・プラチナ。

貴金属ディーリング第一人者の池水雄一氏が色々な角度から今後の金・プラチナの行方を鋭く答えて行きます。

世界一の金取引量を誇るスタンダードバンクの支店長である池水雄一氏が2015年の貴金属市場の動きをズバリ読み解く！！

次から次へと降って沸く世界中の経済問題、 「株だけではなく、やはり安全資産といわれる金に投資しておいた方がよいのかな～」とお考えのあなた！ 豊富なデータを基に池水雄一氏がわかりやすく2015年の貴金属の動きを読み解きます。 「私は貴金属に関してまったくの素人だから」とおっしゃっているあなたもこのセミナーで貴金属を語れるようになってしまうかも！？
