description

あなたの１週間は、確実にあなたの夢に近づけるものですか？

このセミナーでは『７つの習慣　最優先事項』のエッセンスを学ぶことで、 従来の時間管理とは全く違ったスケジューリング能力を身につけられます。

【概要】

ノート、ペン、持っている人はスケジュール帳を持ってきてください。主催者のプレゼンに沿って実践ワークを行い、翌週のスケジュールを作成します。

こんな人にオススメ

・もっと時間を有効に使いたい

・仕事と私生活のバランスをもっと良くしたい

・ストレスを感じる事なく毎日を過ごしたい

・職場や家庭に置いて信頼関係を築きたい

・人生の目的や目標を達成したい

・『７つの習慣』に興味はあるけど分厚すぎて読むのが面倒

【『７つの習慣　最優先事項』推薦の声】

・本書を読んでためにならないという人など皆無だろう

（ラリーキング：CNNキャスター）

・ついに、従来の時間管理学を超える本が登場した。

本書はまさに、「時間の投資」を人生の最重要事項に結びつけるための

うってつけの一冊である。啓発的であり、なおかつ実践的な本書こそ、

何よりも優先して読みに値する本といえる。

（チャールズ・ガーフィールド：ベストセラー作家）

・本書は、私たちが自分の願っているとおりの人間になることを教えてくれ、

さらに真の成功者となるには良心をコンパスとしなければならないことを

斬新な形で示してくれている。本書を読み、もう一度人生の意義を見いだすべきである。

（アンソニー・ロビンズ：啓発書の権威）

【参加者の声】

・自分の１週間の過ごし方が全く変わってしまった。今まで如何に会社に流された生活を送っていたかが分かった。（２０代男性・メーカー営業）

・夢のために何をすれば良いのか分かっていたようで分かっていなかった事に気づいた。（２０代女性・銀行員）

・本当にやりたかった事を思い出せた。やりたくない事にも意味付けができ、つらかった仕事が楽しく思えてきた。（３０代男性・出版社勤務）

・今までは頭に浮かんだ事を何となくやっていたが、自分の行動を見える化したことで、何を変えたら良いかが明確になった。（２０代男性・経営コンサルタント）
