一年間AIを学んできたDIVE INTO CODE 卒業生3人によるトーク会（プレゼン）&交流会になります。 先日開催したところ，好評だったので2回目を開催することにしました！

以下の人を参加対象としています。

  • AIを学ぼうと思って，スクールを探している方
  • 独学でやってきたが，一人での限界を感じている方
  • 弊社スクール関係者(受講生、メンター)
  • スクールの受講を考えている方

年齢制限，持ち物などはございませんので，お気軽にお越しください．

卒業生にどのようなことを発表していただくか？

  • バックグラウンド（以前の職業）
  • AIエンジニアになるために必要なこと
  • 自分が現在どのような技術を持っているか

そのほか，「本当に未経験者がAIエンジニアになれるのか？」「DIVE INTO CODEで勉強するというのはどういうことなのか？」などもお話ししていただく予定です．

スケジュール

12:45~ 会場 13:00~ 開始、及びご挨拶、ご紹介 13:10~ 卒業生による発表（20分×3人） 14:30~ 交流会 15:00 クローズ

交流会

カジュアルかつざっくばらんに卒業生，メンターと話せる貴重な機会となっております．ぜひご参加ください．
