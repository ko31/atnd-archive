description

手をかざして操作する Leap Motion というデバイスがおもしろそうです。
ゲームだけじゃなくどんなことができるんだろう…!
そんなアイデアを出し合うアイデアソンをしませんか。

  • Leap Motion のデバイス実機は（参加者次第ですが）2〜3台用意できると思います。
    公式チュートリアルを体験すると何かひらめくかもしれません。
  • Leap Motion のSDKは公開されていて こちら からダウンロードできます。（要Sign Up）
    今回はアイデアソンなのでSDKは必ずしも必要ありませんが、どんなことができるか知りたかったり
    実際に何か作ってみようとされる方は参考にしてみてください。
  • 時間も場所も未定でふわふわしています。9月中には開催したいなー。
    興味がある方はコメント欄やTwitter等でお声がけください。
    「やりましょう」と言った3人の日程が合う日取りでサブスレッド社（定員9名）で開催、、が濃厚です。
    9月11日(水) 19:00-21:00 を仮設定しています。
  • アイデアソン（Ideathon） は Idea と Marathon を合わせた造語です。（参考）
    教えを受ける勉強会とは異なり、自ら発言・提案しないと何も前に進まないし得られるものもありません。
    みんな初めて触れるオモチャにワクワクしているだけなので、臆せず意見を発信していくとよいでしょう。
