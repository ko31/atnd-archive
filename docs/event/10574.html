description ■「本旅」プロジェクトとは あなたの大切な本が、人から人へ渡り、本を読んだ人同士が１年後に出会う実験企画。

あなたの蔵書の中からお気に入りの１冊を選び、あなたの大切な人に本を貸しましょう。
本を読み終えたら、あなたの大切な人が大切にしている人に本を貸すようにお願いしましょう。


1年後の2011年12月3日、都内某所にて「本旅会」を開催！
あなたが旅に出した本を読んだ人が集まり、語らいましょう。12月3日はみかんの日なので、みかんでも食べながら。

1冊の本から生まれる出会いはどのようなものでしょう？

大切にしている人が、大切にしている人と出会い、さらにその大切にしていいる人とも出会えるかも。

１年後の出会いの種、どんな芽がでるのでしょう？　一緒に実験に参加してみませんか。
