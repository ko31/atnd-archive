　　　　一年に一度の星の輝く日にだけ会う事のできる
　　　　　　　　　　　超絶ロマンチックリア充、織り姫と彦星。

　そんな彼らにあやかって、願いを込めた短冊を笹につるす例の行事ですが、
　　　　彼らはいよいよタダで願いを叶えることに飽き飽きしている模様。
　そこで、何か人のためになる事をした人から優先的に願いを叶える事に決定！
　　　　　　　　　みんな、募金をしてから願い事をしよう！！　

:..｡o○☆ﾟ･:,｡:..｡o○☆*:ﾟ･:,｡*:..｡o○☆ﾟ･:,｡*:..｡o○,｡*:..｡☆ﾟ･:,｡*:..｡o

7月に入り震災から4ヶ月経とうとしているけれど、
これからも出来る範囲の支援、楽しい募金を重ねていくことを誓い
一日でも早い復興を皆で祈りましょう。という企画です。

またまた、年半ばでモチベーション下がりがちなこの時期に
同世代のみんなで悩みや愚痴や色々思ってる事を話してすっきりして、
7/8からまた心新たに頑張ろう！
という気合を短冊にぶつけましょう。というイベントです。

皆さんの短冊は笹につるし、代々木公園にさします！（回収します。）
記念撮影しましょう＾＾

願いもかない、心もすっきりし、被災地の方のささやかなお力になれるかもという
一石三鳥なこの企画！

途中入場途中退場可！
短冊のみの参加可！
