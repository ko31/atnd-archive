description

マッチングアプリに興味があるけれど、不安や疑問があり使えていない人や、
少し使ってみたけれど、うまく活用できていない人を対象に、
マッチングアプリの基礎から出会い方、出会えるコツまでを対面で解説していくセミナーとなっています。

開催日は、2020年2月26日（水）14時より横浜みなとみらいにて行います。

「マッチングアプリや婚活サイトに興味はあるけれど、実際登録する勇気が出ない、不安がある」
「マッチングアプリでの婚活・恋活って実際どうなの？」
「マッチングアプリを使ってみたい！教えて欲しい！」

こう思っている方のために、マッチングアプリを研究し続けてきたマッチングアプリ大学の運営メンバーが、安全性から出会い方、攻略法まで丁寧に解説します。
