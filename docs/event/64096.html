イベント概要

SmartNews主催のTech Event “SmartNews Tech Night” 第二弾です!
第一弾はAdTechについてでしたが今回はインフラ!

サブタイトルは 「クラウド事業者に就職する以外の インフラエンジニアの生きる道」 です!

クラウド関連の技術が大いに注目を浴びる中、最近私の知り合いのエンジニアもクラウド事業者様に就職する人が増えてきています。
もちろんSmartNewsでもクラウドを最大限活用してサービスを提供させて頂いていますが、

  • インフラエンジニアとして生きていくためにはクラウド事業者に就職するしか無いのか?
  • サービス事業者内でインフラエンジニアとして価値を発揮し続けていく道とはどういうものなのか

そんなことを社内で議論していて「そうだこれを次のSmartNews Tech Nightのテーマにしよう」と決まりました。
インターネット業界の重鎮の方々にご参集していただきdeepな話が聞けると思います!
