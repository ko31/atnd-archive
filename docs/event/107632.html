description

小規模事業者が国の企業全体の9割を占める時代。
大企業は価格競争ができても、小さなお店には価格競争に限界があります。
他の競合他社と比較をされたとき、あなたの商品・サービスが消費者に選ばれる理由はあるのでしょうか。
今回のセミナーでは、高級な「匠の傘」のEC販売で成功した大阪・心斎橋の傘専門店「心斎橋みや竹」の実際のイノベーション事例を特別にご紹介いたします。
-—————————
輸入雑貨店として明治29年（1896年）に大阪・心斎橋で創業したみや竹。
2代目から傘を扱うようになり、4代目の代表である宮武和広氏の時代に傘専門店として平成8年（1996年）にはネットショップも立ち上げました。
競合が少なかったネット通販の黎明期なら、商品の魅力だけ伝えれば売れた時代。
しかし在庫の豊富なブランドによる直販サイトの出現や、実際に見て触れられる百貨店と比較されるようになり、売り上げは激減しました。
そのような中で、「AB3C」という新しい思考法を用いて、みや竹が「選ばれる理由」を見つけ出し、あした。
そして業界の常識にとらわれず、社会の課題に向き合っていくことで、赤字からの脱却を果たしたイノベーション事例をご紹介します。

マイクロな市場のニーズに応えることは、小さなお店だからこそできる強みです。
これからの時代、あなたの事業を生き残らせていくには何が必要か。
何を変え、何を変えないべきか。
事業に行き詰っているものの、何をどうしていけばよいかわからないという方は、ぜひ一度セミナーにご参加ください。
