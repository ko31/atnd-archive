内容

本セミナーでは、オープンソースのデータ分析ソフトを体験しながら、生データを使った実践的なデータ分析を行います。

ノンプログラミング分析ソフト”RapidMiner”、世界標準の”R”でプログラミングによる分析、その商用版である”Revolution R”をご紹介します。

さらに日本発のオープンソース”NYSOL”を使って、初心者でも簡単に大規模データを加工する方法をお伝えします。

データ分析に関心の高いビジネスユーザー、分析ソフトの乗り換えを検討するデータ分析者、データがあっても分析方法分からない方にオススメです。



■詳細は弊社Webサイトをご参照下さい

注意事項

ノートPCをご持参ください（OSはWindows、Macを問いません）。

分析環境はOracle VM VirtualBoxでご提供いたします。

お渡しする分析環境は、オープンソースの分析ソフト3製品が事前インストールされており、サンプルプログラムや各種環境設定がされています。

分析環境はセミナー開始時にUSBメモリで配布するため、USB接続禁止の場合は申込時にその旨をご記載ください。

セミナー当日の数日前に別途FTPサイトのご連絡を差し上げます。
