今回から始まる「GIT GUD（ギットグッド）」は、みなさんで遊びたいゲームを好きに持ち寄ってゆるく遊べる、カジュアルなゲームイベントです。

記念すべき第1回目の会場は、プロゲーミングチームである「サイクロプス大阪」様の地下イベント施設「Osaka esports basement.」で開催します。

持ち込めるゲームは何でもオッケー！コンシューマー、PCゲーム、携帯ゲーム、TCG、ボードゲー、その他、デジタル、アナログ問わず遊びたいゲームやゲームハードなど何でも持ってきてください。持ち込みするとより楽しめますが、もちろん、手ぶらでの参加も大歓迎しています。

みんなでワイワイとゆるめに遊ぶも良し、一人でのんびりとまったり遊んでも良し、積みゲーをクリアするために遊びに来るのも良し、自分の好きなゲームを布教するも良し、ゲーム大会や対戦会などプチイベントを開いても良し、疲れたら寝たり途中で帰っても良し、ゲームよりも居酒屋で飲み会だ！なんてのも全然アリ。

イベント中の過ごし方は基本的には自由なんでオッケーです。

会場の最寄り駅の近くには、コインパーキングが数件、居酒屋や焼肉屋など飲食店が多数、コンビニやスーパーもあり、銭湯（営業時間は昼14時～深夜1時まで）などもあるのでめちゃくちゃ環境は良いです。

◆機材等

ゲーム用の液晶PCモニター、スピーカー、PS4は8セットほどこちらでご用意が出来ます。また、ご希望があれば、配信用にPC、マイク、webカメラ、LANケーブル、HUBなど機材もありますので借りたい場合は言ってください。ネット環境もあります。

このイベントのTwitterのハッシュタグ「#gitgudgaming」で遊びたいゲームや持ち込むゲームをツイートすると、もしかしたら他の人たちと一緒にゲームしやすいかも？

ハッシュタグを付けてどんどんツイートしましょう！

GIT GUDのDiscordサーバーもあります！

https://discord.gg/myAz8qw

イベントに全然関係がない話とかしても大丈夫なので、参加者同士の雑談、交流や質問などに好きに使ってください。

◆その他、注意事項など

・会場施設内は全面禁煙です。

・食べ物、飲み物は持ち込み自由です。お酒の持ち込みも大丈夫です。ただし、出たゴミは各自自分たちで持ち帰ってください。

・会場にあるゲーム機材の多くは「サイクロプス大阪」さんのご厚意で貸し出して頂いてるものですので、丁寧に扱ってください。もし壊したら弁償でお願いします！

・会場の周辺は住宅街となっているので、会場外で騒ぐと近隣の住民の方に大変ご迷惑となりますので、外出される際は出来るだけ静かにお願いします。

・貴重品の管理は各自徹底してください。もし何かトラブルが発生してもこちらは責任が負えません。

・会場内では、スタッフによる写真や動画の撮影が行われることがあります。ご了承ください。

・節度ある行動をしましょう。みんなで楽しくゲーミングがしたい！
