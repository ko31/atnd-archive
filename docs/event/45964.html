dendai sie; LTとは

東京電機大学の秘境の一つ、千葉NTキャンパスで行われている不思議な仲間たちの会合。
Lightning Talkという「一人5分」という制約を用いた形式で行われている。





いつもUstreamをご視聴いただき、ありがとうございます！
お待たせいたしました！今まで学内向けだったdendai sie; LTを、学外を巻き込んだ形で開催致します！


今回はLightning Talk形式ではなく、持ち時間を「15分」として開催したいと思います。
もちろん聞き専参加も大歓迎！
Ustream配信もいつも通り行います。


テーマについては「自由」とさせていただきます。
自分の興味があること、のめり込んでいること、研究していることなど何でも歓迎！
