title 家づくりの基本セミナー
catch 知っておきたいコトを、わかりやすくお伝えします。
description

一級建築士による、家づくりの基本セミナーを行います。
家づくりに関する、気を付けた方が良い事、基本的に知っておきたい事などを、
公的機関含め多数の家づくりセミナーを行って来た講師が、わかりやすく解説します。

また、夢と希望を整理するワークショップも行い、
家づくりの知識習得以外に、自分の家づくりを具体的に考え纏めて行く手助けをします。

（個別の質問時間（相談会）も設けます。）

開催日時：平成２２年１１月６日

場所　　：東京都北区滝野川会館４０３集会室
料金　 ：無料

　 こんな方にオススメ

家づくりに興味のある方
家づくりを具体的に考えている方

家づくりの予定が無い方も、一緒に家づくりについて勉強しましょう！

期待効果
基本的に知っておきたい知識が学べます。
初めて家づくり、その進め方が理解できます。
失敗しない為の注意点を知る事が出来ます。
自分の家に対する、潜在的な想い・夢・希望を具体的に想像し、
どうしても必要なモノ、無くても良いコトを、整理する事が出来ます。

主な内容
１）家づくりのプロセス
　　　　・手順・誰とつくる？・ハウスメーカー、工務店、建築家それぞれの利点
２）住まいの対する夢と希望の整理
　　　　・家族の為・子供の為の住まい、心地よい住まい
３）土地の選び方
　　　　・土地選びのチェックポイント・法律上の規制
４）費用について
　　　　・家を建てるのに必要な費用・ローコストな作り方・費用を抑えるプランニング
５）機能性と作り方
　　　　・工法・構造について（木造・鉄骨造・RC造）・仕上・温熱環境・採光・眺望

○ワークショップ（夢と希望の整理）
○相談会

特典
個別の質問時間（相談会）を設けています。
具体的な質問の有る方は、資料等をお持ちください。
