description

ここ数年、超売り手市場と言われる新卒市場
において、優秀な「金の卵」を獲得するためには？

超売り手市場の中、経営人材候補－「金の卵」を
長年発掘してきた市場価値測定研究所長による、
優秀な学生の採用ノウハウを伝授いたします。

■当セミナーの概要

①新卒採用市場、最近の動向

②今後活躍する新卒学生の人物像

③経営者が新卒採用に関与する重要性

④蓄積された人材データをベースとした
　HRテックを用いた「金の卵」の発掘方法とは？

⑤質疑応答、意見交換、参加者の人脈形成

日　時：2017年10月19日(木)
　　　　10時～11時30分(最大12時）or
　　　　16時～17時30分(最大18時）
同一内容なので、ご都合が良い時間帯で
お申し込み下さい。

会　場：恵比寿ビュータワー 1F会議室
*恵比寿ガーデンプレイス内です。

対 象：経営者・人事部長・採用責任者の方々

定　員：午前・午後ともに10名様まで（完全予約制）

講　師：市場価値測定研究所所長　藤田　聰氏

費　用：無料

■お申し込み方法

お申込みを希望される方は、

「氏名、会社名、部署・役職名、ご連絡先のお電話番号、
ご希望の時間帯」をご記述の上、

info@v-change.co.jp

にご連絡ください。

「参加確認証」のコピーを受付してお渡し下さい。

*10名を越えた時点で受付終了となります。
*同業他社様の参加をご遠慮いただく場合があります。
　ご了承下さい。
