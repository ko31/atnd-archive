description

Unityを使って、2Dシューティングゲームぽいものをハンズオン形式で作って行く勉強会を開催します。

Unityをはじめてさわる人向けです。
内容的には以下のような感じ
　AssetStoreからモデルをインポート
　プレイヤーモデルの配置
　カメラの設定
　ライトの追加
　背景の設定
　敵のモデルの配置
　敵モデルのアニメーションを設定
　敵のを移動するようにする
　敵のモデルが自動生成されるようにする
　弾を打つ
　あたり判定をつける
　パーティクルで弾がぶつかったときに爆発させる
時間余ったら、そのときは＋αなんか説明します

ハンズオンなので、なるべく時間に遅れないようにお願いします。
勉強会終了後、希望者がいれば懇親会をしようと思います。
