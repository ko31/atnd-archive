description

《レーベンス・マイ》は音楽家と音楽愛好家が友達になる場。会員同士が互いの信頼と友情を深める為の様々な活動を推進する中から、日本におけるクラシック音楽支援のパワーを生み出し、ひいては優秀な音楽家を育成することを目的とします。

《レーベンス・マイ福岡》は平成22年２月21日創立、平成23年６月26日（日）で９回目の例会を数えます。今年４月、桃原健二（会長）大畑惠三（幹事）のもとに新体制をスタートいたしました。

【例会進行】
１）点鐘（開会）、２）今月の「歌」、３）会長報告、４）幹事報告、５）音楽演奏、６）小パーティー（ワイン・ビールにオードブルやお弁当を準備しています）。＊パーティーと同時進行で、委員会の報告や議題に基づいての意見交換を行います。
７）点鐘（閉会）

［例会費］３,０００円
