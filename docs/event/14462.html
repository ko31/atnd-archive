【プログラム】
◆講師の言葉
　レーザー光源の著しい進展により、レーザーディスプレイの研究開発が活発化している。半導体レーザーをベースとした赤色、青色、緑色のレーザー光源が実 用レベルになったこと、および課題であったスペックルノイズ低減技術が提案されたことによる。2008年には世界初の民生用レーザーTVが発売された。ま た国内で障害となっていた消安法の改正により、超小型の携帯レーザープロジェクタおよびヘッドマウントディスプレイが実用化となる。レーザーディスプレイ は、従来のsRGB規格に対し色再現範囲を約2倍程度広げることが可能であることに加え、光源の高効率化により超低消費電力化もねらえるという特徴も加わ り、高出力レーザーを用いたレーザーリアプロジェクションTVや高輝度プロジェクタ、そして超小型レーザーを搭載した携帯型プロジェクタやヘッドマウント ディスプレイという異なる2方向に分かれて研究開発が加速している。ここでは、現状機器を凌駕するだけでなく究極のディスプレイや照明となり得るレーザー ディスプレイについて、その構成、特徴、課題および応用展開も含め解説する。

◆修得知識
