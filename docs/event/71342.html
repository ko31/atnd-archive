電子書籍(EPUB)とは？

EPUBは、電子書籍の為のファイルフォーマットです。EPUBフォーマットは、個人用途、商用問わず、誰でも、特定の団体や個人に許諾を得る必要なく、自由に、無料で利用できます。世の中で電子書籍と言われているものの最も一般的なフォーマットです。

KindleやGooglePlayBooksって何？

Kindle(キンドル)は、米国アマゾンドットコム社が運営する電子書籍ストアのことです。電子書籍ストアでは最も規模の大きいストアでアマゾンのアカウントがあればどなたでも電子書籍を購入、販売できます。Kindleで販売された本はKindleペーパーをはじめ、PC、Andoroid,iPhone / iPod touch / iPad（iOS）で読むことが可能です。

GooglePlayBooksはgoogleの運営している「Google Play」の中にある［書籍］メニューです。Androidのみならず、iPhone / iPod touch / iPad（iOS）、ウェブブラウザで見ることが出来ます。

KindleとGooglePlayBooksで電子書籍を出版の方法がわかれば電子書籍の出版のほぼ全ての出版方法がわかるといえます。

特殊なソフトが必要なの？

作成に有料のソフト等は使用しません。原則、無料のソフトとオンラインエディタ―での作成を考えています。

紹介する作成方法はwordから始めますが、一太郎などの文章作成ソフト、PDFからでも同じ手順で作成が可能です。

対象のOSはWindowsを想定していますが、使用するソフトが多少違うだけでMacでも作成可能です。
