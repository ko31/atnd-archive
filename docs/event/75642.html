フルコミ営業マン卒業セミナー
＜特別招待状＞

起業をしたくても、できないもしくはできなかったと思っているあなたに、
「こんなに簡単だった」んだと口からこぼれる情報をセミナーで一般公開します。

業界では一部の人しか知らない秘密をあなたが知った時
なんで今まで起業に踏み切れなかったんだと心の底から
叫ぶことになりますが、このセミナーを聞いた直後には、
あなたの今の収入を２倍以上に跳ね上げることが
簡単にできるようになりますのでご安心ください。

今までの常識を根底から覆す

あなたがフルコミッションの営業を選んだのは、他にもなく稼ぎたいからだと思います。
確かに売れる営業マンである以上フルコミッションは、おいしいですが今の成績で
さらに２倍以上の収入を得るには、起業するしか方法はありません。

しかし起業には多額の資金が必要と考えている人がかなり多く存在します。
さらには、信販会社の提携ができないとか、仕入れ先をしらないなどで
あきらめてしまっている人が圧倒的に多いのです。

そのあなたが考えている常識は、私のセミナーに来ることによって
根底から覆されることとなります。

実際、私は太陽光発電のフルコミッションで営業をやっていましたが、
売り上げの１０％しか収入がありませんでした。

フルコミで営業をやっているあなたであれば、わかるとおもいますが
２００万円の契約に対して２０万円の収入です。経費を引いたら
当然１件では食べていけません。

最低月に２件以上の契約が必要です。

そんな私がフルコミッションで得た、たった50万円の月の
収入の中から30万ほどの資金を使っただけで起業ができ
さらにはフルコミの２倍以上の収入を得ることが出来たのです。

あなたも、フルコミで営業しているなら今すぐ卒業して
起業してください。収入の面だけではなく、あなたのやりたい
事業展開も簡単にできるようになります。

実際、現在の私は太陽光発電のフルコミッションを卒業し
起業したことによって

太陽光発電販売・施工・保守
リフォーム販売・施工・保守
ハウスクリーニング
フルコミ営業マンコンサルティング
webコンサルティング
水素水販売・施工・保守
光回線取り次ぎ

などの業種を自由に選択し商売できるようになりました。
当然、収入もフルコミ時代の3倍以上の収入を得ています。

はじめまして！フルコミ営業マン卒業セミナー講師の島尻と申します。

私たちはフルコミ営業マンのコンサルティングはもちろんですが
「webマーケティングコンサルタント」のグループを作っています。

そして今回はフルコミ営業マンを卒業して頂き、どのように
起業して、最大の収益化にするのかを日々研究しています。

そして私、島尻を含む国内最高レベルの起業ノウハウを持った
コンサルティングチームが今回ゼロからどうやって
あなたの収益を2倍以上にするのか？をお伝えします。

信じられないかも知れませんが、今まで私どものセミナーに参加した
人の中には収入をフルコミ時代の5倍にした人もいます。

彼はフルコミの営業マンを5年ほどやってましたが、どうやったら
起業できるのかと悩んでました。

それは、あなたも同じかもしれませんが、信販会社の提携が
できないのと仕入先や工事の職人等のつながりがないから
ということでした。

しかし、信販会社のつながりがなくても契約を上げる方法を
知り仕入先や工事の職人も簡単に探すことができ、今では
億を超す会社の社長になってます。

島尻さんのおかげだと今でも感謝されます。
（私より大きな会社になったのはビックリしてますが・・・(笑)）

こんな話をすると、「ほんとにそんな方法あるの？」と
思われるかもしれません。

事実、リフォームや太陽光発電等の高額な商材を
扱う職業では起業するのに多額な資金がいると
思っている方がほとんどだからです。

なのでフルコミの営業マンから抜け出せないのです。
フルコミの営業マンの行方

フルコミの営業マンは自由です。しかし起業家以上にプレッシャーがあります。
それはなぜかって、入ってくる収入が安定しないにも関わらず、経費は出ていき
成績の悪い時が続くと辞めなければならないからです。

しかも起業家よりも入ってくる利益が少なく経費も掛かります。
スーパー営業マンでも継続するのは至難の業です。

私もフルコミッションの経験が10年ほどあるので痛いほどわかります。

自由なことが邪魔をして稼ぐ額が年々減っていく人も
少なくないと思います。

結局モチベーションも続かずに辞めてしまうという事態になるのです。

あなたは、会社の中で固定給をもらいコツコツとやっていくのが
嫌でフルコミの世界を選んだのではないでしょうか？

なのに継続ができずに辞めていく人が後を絶たないのです。

「おれは違う！今のままでいいんだ！」っというのなら
今のままで良いと思います。しかしこのレターをここまで
