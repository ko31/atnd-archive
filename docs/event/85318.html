参加対象者：ブラウザゲーム制作者、ブラウザゲームが好きな方 はじめて話す方も歓迎です。

目的：時間目標を決めて制作することでゲーム制作の進捗率を上げる。
　　　制作中のゲームを公開し、クオリティを上げるアイデアをもらう。

内容：ブラウザゲーム制作者が自分の制作ゲーム（制作途中）を公に公開し、他の人はそれに文句垂れたり面白いこと言ったりする会です。
21:00～21:30 各自自己紹介・現在制作しているゲーム紹介・本日の制作目標を公開。
21:10～23:00 各自ゲーム制作（雑談や相談など自由にしてください）
23:00～23:30 進捗報告・解散

場所：スカイプ (ユーザー名：hothukurouで検索して友達登録しておいてください。時間になったらグループ集合しましょう。)
