description

先日お知らせした、7人倶楽部親睦七尾ツアーを開催します。
主宰のヤマグチケンスケさんはじめ、7人倶楽部の楽しくも濃いメンバーと
交流していただければと思います。
参加される方は、コメント欄にオプション（金沢散策）もつけるか否かを
記入してください。

なお、大阪からはサンダーバードの和倉温泉行きなら乗り換えナシでアクセスできるので、
意外と気軽に参加できます。

◆主なコース(暫定)
＞16(土)
和倉に14時ごろ集合。
(備考：夜行バスだと朝9時ごろつくので合流まで自由)
和倉温泉街を散策。（七福神めぐり）
和倉の安い大部屋の温泉旅館に宿泊。（みんなで七並べをする）

＞17(日)
七尾市街に移動。
食祭市場の朝市へ。朝食。
一本杉通り散策。昼食
一次解散(土日限定組)

ここからオプション—-
金沢へ移動。市内で夕食。
金沢のシティホテルに宿泊。

＞18(月)
金沢市内散策。
