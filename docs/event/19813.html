イケテル “Hadoop MapReduce” 勉強会 in 北九州

今回のテーマは、「 Hadoop MapReduce 」です。

勉強会では流通業を例にし、テストファーストで開発を進めて行くことを学びつつ、agile開発と、Hadoop MapReduceについて学びます。

Hadoop 拡張の Hadoop Streaming を使って Ruby で MapReduce処理を作成します。
そして、RSpecを使用してテストファースト開発も体験します

使用する環境は、福岡県のRuby開発サーバの環境を利用して行います。
（Ruby開発サーバとはRubyビジネスに取り組もうとする企業等に対して製品開発と技術支援を目的として、福岡県が提供しているクラウドサーバです）

  • Ruby開発サーバに関してはこちらをご覧ください
    • 公式サイト

今回の会場及びPCは、北九州市の外郭団体である「ヒューマンメディア財団」のご協力により、お借りしております。
