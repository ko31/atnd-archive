catch 海外で人気のボードゲームをカジュアルに楽しみましょう！
description

海外のボードゲームをアイデアの源泉として、テレビ番組企画やソーシャルアプリのヒットを飛ばす人たちが世界中で現れてきています。
しかし、日本ではまだ人生ゲームやモノポリーぐらいしか遊ばれていません。これはもったいない！
ゲームは、コンピュータ相手よりも人間相手のほうが絶対に面白い。脳が心地よく疲れる感覚をもっとみんなに知って欲しい！
お酒を片手に、ボードゲームを通して人間同士の駆け引きやコミュニケーションを楽しむ大人を増やしていくことを目的に、私たちUrventは日々活動しています。

オフィス新大陸さんに名作・新作のボードゲームをご提供頂き、ルールなどを教えてもらいながら楽しむ会です。
だから、海外のボードゲームで遊んだことが無い人でも大丈夫！大歓迎！ぜひ遊びにきてください。
