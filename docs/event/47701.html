JAWS DAYS 2014

  • JAWS DAYS 2014参加はこちら

3/15（土）に JAWS DAYS 2014 が開催されます。
今回は地方でも参加しやすいように、行きだけ無料バスが用意されています！
しかし、名古屋出発です。。

「JAWS DAYS 2014 に参加するために深夜バスで移動しよう！」 というイベントに参加するイベントです。

出発
3月14日(金) 20:00くらいにどこか集合。
名古屋のバスに間に合うように行く。

※上記イベントに必ず参加してください。
なお、上記イベントに書いてありますが、名古屋→東京（片道）が無料となります。
イベント当日のホテルや帰りの便は別途自身で予約してください。

料金：名古屋までの高速道路代＋ガソリン代＋名古屋での駐車場代をワリカン。

概算 -———————-

■高速料金
軽の場合、1650＋350＋750 = 2,750円
普の場合、2050＋350＋750 = 3,150円

■駐車場代
マユミパーキング
2,000 × 3日 = 6,000円

■ガソリン代
距離 約150km 1リットル → 150円

燃費 5km → (150 × 150) / 5 = 4,500円
燃費 10km → (150 × 150) / 10 = 2,250円
燃費 15km → (150 × 150) / 15 = 1,500円

もし1人（軽）で行く場合、
電車で行きましょう。
20:55出発22:35到着
福井 (しらさぎ)→ 米原 (こだま)→ 名古屋 = 片道5,190円

もし2人（軽）で行く場合、
電車で行きましょう。
20:55出発22:35到着
福井 (しらさぎ)→ 米原 (こだま)→ 名古屋 = 片道5,190円

もし3人（軽）で行く場合、
2,750＋6,000＋2,250(燃費10km) / 3 = 3,700円 / 人

もし4人（軽）で行く場合、
2,750＋6,000＋2,250(燃費10km) / 4 = 2,750円 / 人

もし5人（普）で行く場合、
3,150＋6,000＋2,250(燃費10km) / 5 = 2,280円 / 人

もし7人（普・ファミリーワゴン）で行く場合、
3,150＋6,000＋4,500(燃費 5km) / 7 = 1,950円 / 人

※行きのみです。帰りも名古屋→福井という方は要相談となります。
