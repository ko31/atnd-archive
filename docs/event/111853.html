description

「チョークアート」はオーストラリア生まれのおしゃれな黒板アート。
発色の良いクレヨンのような画材（オイルパステル）を使って、カラフルにえがきます。
手描きのぬくもりある黒板絵で、春気分を盛り上げましょう。

イベント概要

「オイルパステル」を使ってカラフルな絵を描く、オーストラリアスタイルの黒板アート「チョークアート」が体験できます。

今回は、15cm×15cmの小さな黒い板に、写真の桜のイラストを描きます。
黒板には、あらかじめ下絵がかいてありますので、絵を描くことが苦手な方も楽しく作品づくりができます。

「オイルパステル」は、発色の良いクレヨンのような画材です。
やわらかく色が伸びていきますので、描き心地がよく、つい夢中になってしまいます。
