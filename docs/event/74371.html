今、エンジニア市場ではスマホアプリ開発者が極端に不足しています。 アプリ開発ではSwift、Javaといったネイティブ言語が基本ですが、最近企業内ではHTML5を使ったハイブリッドアプリに注目が集まっています。 従来のWeb技術を使えるのに加え、iOS/Android両方に対応できるのが魅力です。

そこで今回はWeb技術をお持ちの方々にハイブリッドアプリ、さらにmBaaSを体感してもらうことによってその可能性と今後の開発に役立てもらうハンズオンを企画します。 ぜひアプリ開発にチャレンジしてください！

Monaca×mBaaSで簡単ハイブリッドアプリ開発！

講演

講演1.ニフティクラウド mobile backendについて

ニフティクラウド mobile backendの概要説明と使い方について説明。

ニフティクラウド mobile backend　エバンジェリスト　中津川氏

オープンソース・ソフトウェアを毎日紹介するブログMOONGIFT、およびスマートフォン/タブレット開発者およびデザイナー向けメディアMobile Touch 運営。B2B向けECシステム開発、 ネット専業広告代理店のシステム担当を経た後、独立。多数のWebサービスの企画、開発およびコンサルティングを行う。2013年より法人化。2014年04月より企業の外部開発者との繋がりをサポートするDevRel事業を開始。ニフティ株式会社のDevRelチームの一員として活動中。

講演2.Forkwell/Forkwell Jobsについて

エンジニアによるエンジニアのための求人サイトForkwell Jobsの説明。厳選された自社サービスを運営している企業様を中心に求人を掲載しておりスマホアプリエンジニアの採用市場についても説明させて頂きます。

grooves Forkwell事業部　執行役員　中込氏

海外大学卒業後、2005年に株式会社groovesへ入社。05年からクラウドソーシング事業の立ち上げに携わり、主にIT業界を担当しエンジニアから営業部長、人事部長などの転職支援。 その後、新規事業の立ち上げを経て、エンジニア向け求人サイトを運営するForkwell事業部を担当。

ハンズオン

ニフティクラウド mobile backendにあるチュートリアル（フィードリーダーにお気に入り登録機能を追加しよう）を元にハンズオンを行います。基本はドキュメントを各自読みながら行ってもらう形式です。受講者が分からないところがあれば、Monaca/mBaaSそれぞれにサポートをします。
