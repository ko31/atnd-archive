この度、
三崎町7-4から
三崎町9-9（元EBISUYAさん跡地）へ移転します！

また、広々とした2Fフロアもあり
女子会・ママ会・誕生日会・結婚式二次会 etc…
少人数から貸切まで、パーティーにも最適です♪
