概要

今注目のクラウド、スマートフォン、各種ボードを体験してみましょう。

第一部　クラウド
Google ドキュメント の「文書」、「スプレッドシート」、「プレゼンテーション」、「図形描写」の基礎講習会です。この講習を終了するとクラウドを使って文章を作成する基礎が身に付きます。
Google ドキュメントはOSを問わず、ブラウザが動けば使う事が出来ます。つまり、Mac,Win,Linuxなどで同じように使う事が出来ます。

第二部　スマートフォン、各種ボードなど

  1. SkyDrive
  2. アマゾン ウェブ サービス（AWS）
  3. iPhoneハッキング
  4. Arduinoの紹介
  5. SAKURAボードの紹介

第一部だけでも第二部だけでもお申し込み下さい。

IT 勉強会スタンプラリー に参加しています。参加している方は台紙を持って来て下さい、初回の方は配布します。
