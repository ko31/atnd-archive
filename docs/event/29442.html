description
「Barcode KANOJO」「俺の妹がこんなに可愛いわけがない」「ときめきメモリアル Girl’s Side Premium」など、iPhone / Androidの大人気アプリやPSPゲームへの搭載が話題の「Live2D」。

「Live2D」は魅力あふれるキャラクターや原画を2Dのまま立体的に動かすことが出来る世界初の技術ですが、今まで限定公開のみだった「Live2D Cubism」の概要を本セミナーにおいて初公開いたします。

Live2Dの概念から当社オリジナルツールによる実際の制作過程、採用作品の紹介など幅広くご紹介。
質疑応答などLive2Dを深く知る絶好の機会ですので、ご興味のある企業様はこの機会にぜひお越しください。

セミナー終了後に近くで簡単な懇親会を予定しておりますので、
お時間がありましたらぜひご参加ください。
