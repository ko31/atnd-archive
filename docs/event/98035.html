「未来テーマパーク会議2018」

主に三つのテーマで語り合います

・舞浜&大阪の大規模拡張計画！どうなる新エリア

みんなで新エリアのアトラクションを考察しましょう。ファンタジーなエリアからゲームのエリアまで幅広く語り合おう！

・俺に紹介させろ！テーマパークLT！

一人5-10分間で自由にテーマパークについて語ってください！

・俺たちの考えるテーマパークのありたい姿！

お集まりの皆さんで理想のテーマパークやまだ見ぬ斬新なテーマパークを創造し新たな夢へと向かいます。

自由なプレゼンテーション(LT)に参加したい方はコメントに「LTへの参加希望」と記入お願いします！
