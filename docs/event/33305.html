概要

最近話題のテキストエディタ 「Sublime Text 2」についての勉強会です。
Sublime Text 2 をちょっと触ってみたいという方からバリバリ使いこなしている方まで、Sublime Text 2 について情報交換・交流しませんか？まだ、第0回ということで、なるべくゆるい形でできればと思います！皆さんの参加をお待ちしております。 : )

宜しければ、Sublime Textについて日本語で情報交換している Sublime Text Japan Users Group にもご参加ください。
