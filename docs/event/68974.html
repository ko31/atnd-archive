イベント概要

　マイコンの開発にRubyを使う動きが始まっています。その名もmruby。RubyはWebの世界で有名なスクリプト言語ですが、そのモダンで書きやすく理解しやすい記述をマイコンに持ち込もうという試みです。エンジニアの育成が容易であることや開発速度向上だけではなく、最近はIoTで注目が集まってきました。この講座では、参加者が作りたいものを作るハッカソン形式とします。わいわいしながら楽しく電子工作してみませんか？

LTについて

　もしLT（ライトニングトーク）で発表したい方がおられましたらご連絡下さい。1人10分ほどの時間を作りたいと思います。

持って来て頂きたいもの

次のものを事前にご用意下さい。

  1. 作業用PC（OSはWin/Mac）
  2. enziマイコンボード（4つは貸出可能）
  3. ArduinoをはじめようキットのArduino本体以外と同等のもの（ブレッドボード、ジャンパワイヤなど）
  4. 使ってみたいセンサ

　また、インターネットで検索・ダウンロードを実施するため、テザリングやポケットWi-Fiをご用意下さい。もし事前にご要望があれば、mbedのデバッガ付きArduino互換機も貸し出せます。こちらで開発されても構いません。さらに、作りたいものがはっきりしていてArduinoを使ってみようという方も歓迎です。「Raspberry PiをOSなしで使ってIoTするんじゃ～」という剛の者も大歓迎です。(笑)
