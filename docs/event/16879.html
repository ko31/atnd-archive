Twitterやfacebookで知り合った仲間と交流を深めるイベント「神戸twitter交流会」を、7月10日(日)に開催いたします。
会場はなんと「電車の中」！神戸を走る貸切電車「ツイッタートレイン」の中で、楽しい仲間とともに楽しいイベントで盛り上がりましょう！

○日時：2011年 7月10日(日)　14:00集合～17:30ごろ解散
○場所：北神急行電鉄　貸切列車内（集合・解散は北神急行電鉄新神戸駅となります）
○参加費：お一人様3,000円（運賃・飲食(軽食)代込み）
○イベントの内容
　・フォロワーさん大集合！twitter・facebook交流パーティー
　・電車でライブ！？iPad2などを使ったミニライブを開催！
　・前代未聞？「土足禁止」の車両で床に座って真剣勝負の「かるた大会」
　・車内イベントの模様はUSTREAM放送にて中継予定
　・特別企画：関西USTREAM界の巨匠・今川敦史氏(@imajp)のバースデーをお祝いするスペシャルセレモニー
　・その他、様々なイベントを計画中（企画持ち込みも大歓迎！）
　　（イベント内容は変更の可能性があります）
○イベントの詳細及びお申し込み・お問い合わせは、こちらのページでお申し込みください。
○参加申し込み期限：2011年7月9日(土)
○主催：ツイッタートレイン神戸運行委員会
