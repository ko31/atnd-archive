The GitHub poweredby Agile渋谷

イベント ユーザー
event_id 30016
title The GitHub poweredby Agile渋谷
catch 日本のSOCIAL CODINGの今を見る
description

オープンソースソフトウェア開発に革命をもたらしたGitHubがテーマです。

使い方の事例を中心にメリットや現状の開発との違いを感じ取り、現場に導入するキッカケをつかんで頂ければ幸いです。

世界中のプログラマが利用しているGitHubは、もはやプログラマであれば使えなくてはいけないほど普及してきています。
まだ使いはじめていないプログラマもこれを機会にSOCIAL CODINGの世界を体験しましょう。
