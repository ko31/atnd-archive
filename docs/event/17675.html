description

Cloud.com社のクラウド基盤ソフトウェア「CloudStack」は、Tata Communicationsや、Korea Telecomをはじめとした多くの クラウドサービス事業社で採用されています。また「Citrix XenServer」はRackspaceや ソフトバンクテレコムをはじめとした多くのクラウド環境で最も実績のある ハイパーバイザーです。これらの２つの製品を組み合わせることにより、 スケーラビリティのあるオープンなクラウドを素早く構築する事ができます。

本セミナーでは、CloudStackとXenServerを利用したプライベートクラウド構築を実機によるハンズオン形式で体験することができます。
