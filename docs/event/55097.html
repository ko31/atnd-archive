■概要

9/20(土)のオープンソースカンファレンス@広島の開催に合わせて、第22回ユーザー会 in 広島 を開催します。

CloudStackは、Apache Software FoundationのOSSプロジェクトとして開発されており、IaaS型クラウドを実現するソフトウェアとして、多くのクラウド事業者や 北海道大学アカデミッククラウド をはじめとした学術機関などで実績があるソフトウェアです。

こんな方にオススメ

・クラウドを作ってみたいインフラエンジニアの方
・オープンソースプロジェクトに興味のある方
・CloudStackとかOpenStackとか、最近よく聞くけど何が違うの？という疑問をお持ちの方
・クラウドの中の人にいろいろ質問してみたい方

初心者の方にもわかりやすいコンテンツをご用意しております、ぜひお気軽にご参加ください！！
