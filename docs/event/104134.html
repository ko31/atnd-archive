Ingress First Saturday

地域のコミュニティからエージェントを集めて、ソーシャルな環境でIngressの経験を共有し、新しいプレーヤーに機会を提供するイベントです。 浜松市内外からエージェントが集まります。 初心者の方もお気軽にご参加ください。

内容

ブートキャンプ(BC)を行います。 アノマリー前のBCになります。 戦いに向けて皆さんレベルアップしていきましょう！ また、普段会うことのないエージェントと顔を合わせるチャンスです！ 楽しく交流しましょう！

参加方法

  1. 本サイトへの登録(必須)
  2. 参加フォームへの登録(必須)
  3. fevgamesのRSVP(参加登録フォーム)への登録(必須)
  4. telegramへの参加(後日公開。任意)

参加フォームと当日の実績登録フォームに登録していただく情報は以下の目的で使用されます。

  • fevgamesへの実績報告
  • メダルが付与されなかったエージェントの参加の証明
  • 参加者の来訪元のヒートマップ作成

実績登録について

当日はBC前とBC後の実績登録を行っていただきます。

当日実績登録でうまくいかないトラブルが何件か発生します。

FS前日まで実績フォームを公開しますので、事前に練習していただくと、当日、スムーズにイベントを楽しんでいただけます。

実績フォームはメンテナンスする場合があります。その為、一時的に動作しない場合があります。 ご了承ください。
