AWSで、簡単に高速なWordPressサイトを作ろう！

  • 今話題のクラウドってなんだろう？
  • 運用中のWordPressサイトが遅くてなんとかしたいんだけど・・・
  • 急なトラフィック増に困ってしまった！
  • AWSに興味はあるものの、どんな風にはじめたらいいのかな？

そんな不安や疑問をお持ちの方、
１２ヶ月の AWS 無料利用枠 を使って、まずははじめてみませんか？

概要

Amazon EC2という仮想サーバーのサービスを使って、
簡単、すばやくWordPressのサーバーを立ちあげちゃうハンズオンセミナー！
クラウドならではの使い方で、サーバの運用がいままでより簡単で柔軟だよ!

え？なに言ってるかわからない？
そんな頭の中のもやもやしたクラウドみたいな不安をさ、パーッと、五月晴れに変えようよ！

実は主催者もAWSは初めてなので、初心者でも安心して来てください。
WordPress中・上級者でも、これをキッカケに今日からクラウドを初めてみましょう。
