2012年の Advent Calendar は終了しました！

ご参加いただいた、江尻さん、マナカさん、アービンズさん、澤田さん、檜田さん、小犬丸さん、UXAnalytics Labさん、ありがとうございました！

Advent Calendarって？

元々は、12月25日のクリスマスを楽しみに過ごすため、12月1日から24日までのそれぞれの日に色々な仕掛けがあるカレンダーです。
数年前から技術系のAdvent Calendarとして、何らかのルールにそってブログに記事を書いていくというイベントが各所で開催されています。

WEB解析 Advent Calendar 2012 を行なうにあたって

こんな内容があればいいなと思っています。もちろん、下記に限りません。

  • アクセス解析全般
  • ソーシャル分析
  • SSA（サイトサーチアナリティクス）
  • ウェブマーケティング分析
  • Google Analytics 等分析ツール
  • 視聴率調査
  • WEB解析の関係書籍の書評
  • ウェブ解析士の資格取得
  • ユーザビリティやコミュニケーション設計まわり

・・・などなど、WEB解析にまつわる内容の記事を幅広く募集します。

参加することに意義がある、と思えるようなものにしたいので、最近ウェブ解析を始めてみたという初心者の方も大歓迎です。（主催者はまだまだヒヨッコです）
もちろん、本職の方の参加もお待ちしています！

自サイト：WEB解析 Advent Calendar 2012 をやります

参加方法

ATNDで参加表明していただければOKです。
希望日があれば参加表明時に記入してください。特にないようでしたら、こちらで早いところから埋めてしまいます。
ご連絡いただいた方から、こちらで日程表に下記の内容を記載します。

12/5 変更：少しずつ参加表明をいただいていますので、若干変えますね。

日にちから決める場合

ATNDで参加表明時に、希望日時をご記入下さい。その日までに、ブログのURLなどをお送り下さい。（参加表明例：「12/12にしたいです。よろしくお願いします」）

記事を先に書く場合

記事URLを @mekemoke か mekemoke[at]gmail.com にお送り下さい。希望日時があればその日に掲載いたしますが、いただいた日に掲載してしまいます。

参加に必要なもの

  • お名前（公開していいもの）
  • 公開アカウント（ツイッターやFBなどなんでも。なくてもOKです）
  • ブログURL（記事投稿後は記事のURLになります）

ATND は Twitter や Facebook のアカウントで参加できますが、直接申し込みたい、とか記事を書いたから載せて！という方は、@mekemoke へ上記内容をご一報ください。こちらで入力します。

参加申込は空いてる日の15時頃まで受け付けます。たくさんの方に広めていただけるとすごく嬉しいです！（人数足りない場合は主催者責任で何とかガンバリマス…）
