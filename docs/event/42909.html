title WordBench長崎 勉強会01
catch オリジナルテーマで学ぶWordPressの基本
description

架空のWeb制作事務所を想定したオリジナルテーマを教材に、WordPressの基礎知識とテーマの基本的な構成を解説します。

参加者の皆さんが実際に手を動かして学ぶハンズオン形式です。
ご自分のマシンをご持参ください。

＊参加費無料

メイン講師
北島卓(モンキーレンチ)

こんな人向けです
・WordPressを勉強し始めた
・既存のテーマのカスタマイズのみでオリジナルテーマを作ったことがない
・独学なので基本を押さえたい
・WordPressを愛してる(運営側でサポートしてください！)

参加条件
1.htmlとcssは理解できる人
（さらにphpが分かってると理解が早いです）
2.開始時間までにローカル環境にWordPressをインストールできる人
(これが難しい人は13時に集まってください。サポートします)
3.マシン持参可能な人

その他
1.WordBench長崎は、WordPressユーザーコミュニティの長崎版です。
2.電源とWi-Fiはこちらで準備します。
3.会場では水のみご用意できます。他の飲み物はご持参ください。
4.終了後に懇親会を予定してます。

会場へのアクセスはこちらをご参照ください。
