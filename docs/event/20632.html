description

記念すべき１周年を迎える、こい☆こいは、『詫び・寂び的？』『禅的？』『ニホン的？』再構成プロジェクト！ということで“華道×書道×水墨画×ジャパニメーション”のトップクリエイターによる、体感型コラボレーションアートをお届けします！。華道は、活け花のオートクチュール性を空間デザインに積極的に反映させ、その自由な創造性は古典アートの革新スタイルとして定評の高い『横井紅炎』氏。書道は、今や『美人書道家』という金字塔を打ち立て、その確かな足跡においても多くのファンを虜にし「書は机上のみで作られる芸術ではない」というスタイルを確立した『涼 風花』氏。水墨画は、イメージを視聴化する独特の画法で、その筆遣いは墨画に命を吹き込む気鋭のアーティスト『CHONTO』氏。そこにオマージュされるアニメーション作品は、制作期間7年、作画枚数10万枚。CG全盛の時代にあえて手描きにこだわった究極の 日本が世界に誇るアニメーション『REDLINE』！
そして、今回もゴールデンメンバーのDJ陣に加え、以前、最先端のトラックに心躍らずにはいられない魅力あるプレイで、フロワを席巻させたDJ、Vincent.Bも再登場！もちろん、Black Eyes Tattooによる、LIVE TATTOOや深夜の複合屋台「ソウル食堂」も健在です！！
