description

自分のやりたいことをとことんやりたい！
自分の好きなことを仕事にしたい！
どんな仕事が自分にピッタリなのか知りたい。
ワクワクしながら仕事がしたい。
思い描いた人生を歩みたい。

そんな風に考えたことが１度もない人はいないでしょう。
しかし、そこでつい歩みを止めてしまう原因は、お金、環境、不安などの現実。
特に、多くの情報が目に飛び込んでくる現代では、どのやり方が自分に合っているのか、判断がつかないことが多くあります。

スポーツも勉強も同じですが、目標にたどり着くための「やり方」を知らないと、
ただやっているだけになってしまいます。それではあまり意味がありません。

もしあなたが一生懸命に資格をとろうと学んでいたり、
仕事で成果をあげるために、セミナーや勉強会で日々、学んでいたり、
本を読んで、知識を得ても成長を実感できないということであれば、
それは「やり方」を知らないからです。

今回、ゲストで来ていただく井元龍太郎さんは、
月間4000万PVをもつ大手旅メディアのTABILABOで、
「每日4時間だけ好きな仕事をして食べていく、生涯現役フリーランスの生き方。」というテーマで寄稿。情報収集術を生かすことで1日の仕事時間を驚くほど短くしています。

井元さんは福井県在住にて普段は東京におらず、
現在ではそもそも勉強会やセミナーなどの機会がめったにないんです。
今回は特別に、主催のやっちの呼びかけでこの日のためだけに来ていただけることになりました！
この貴重な機会を見逃さず！！

井元さんの代名詞でもある「情報収集術」。
これはビジネスに限らずすべてのことにおいてまずやることですね。

場所を確認　→　路線情報、グーグルマップで調べる
ご飯を作る　→　料理アプリを見る
旅行に行く　→　旅行サイト、アプリを見る
特別なものを食べたい　→　飲食店サイト、アプリを見る

この最初のアクションがずれていて検索に時間がかかってしまったり、
本当は安く買えるものを高く買ってしまったり、
1時間で終わることを１日がかりでやってしまうことがあります。

この差は、情報格差によるものです。
情報収集術さえ身につければ、すべての行動において応用が効きます。

何を始めるにしてもすべての出発点だからこそ、
しっかり身につけて日常の中で生かすことができるようになれば、
誰かに聞く必要もなく、何も困ることはありません。

ワークを交え、皆さんからの声を一つ一つ聞きながら、
明日から具体的に使えるやり方を身につけます。

正しい情報を、正しいやり方で最大限に活用し、
自分のやりたいことを最短・最速で実現させませんか？

◎得られるもの

・自分の好きなことを情報収集してまとめることでコンテンツにして売る方法
・情報収集術とその応用の仕方
・コピーライティングに関する実践的な学び
・得た情報の見本を見ながら、足りないものを修正して実践していく具体的な方法

◎こんな方におすすめ！

・好きなことを仕事にしたい
・やりたいことを改めて考えてみたい
・将来に漠然とした不安がある
・副業を考えている
・正しい情報をつかむ力を身につけたい
・得た情報を応用して使うための知識がほしい

◎登壇者情報

　▶井元龍太郎氏

早稲田大学卒。大学院卒業間際に少しでも無理をすると免疫異常になる難病が発覚し、その後、療養も兼ねて東京から福井県へ移住。

現在は、福井を拠点に1年の1/3以上を全国各地で旅しながら、病気の経験からたどり着いたゆとり世代だからこその「每日4時間だけ、好きな場所で好きなことを仕事にして生きていく」を講演やオンラインサロンなどで伝えるフリーランスとして活動中。

専門は、情報収集×スモールビジネス×効率化で、特に情報収集に関しては、好き・得意スキルを販売できる副業のパイオニア「ココナラ」の祝3万商品出品記念で、公式に発表された売上ビジネス部門で、オリジナルで編み出した「情報収集術」が第1位を獲得。Yahoo知恵袋公認アドバイザーにも就任。

これまでに500人以上への指導をし、満足度98%以上と高い評価を得ている。情報収集術を教える専門家として、日本No1の実績を持つ。21万人以上の生徒が在籍するオンラインスクール「schoo」、スルガ銀行と共催で湘南T-SITEなどでも講演をおこなった。

「情報収集術」だけでなく他にも、大手韓国ファッション企業へのマーケティングコンサル、人生の開花プログラム、0から始めるスモールビジネスの作り方、など枠にとらわれず、10以上もの多種多様なサービスを手がけている。
