トライボロジーの基礎講座

title トライボロジーの基礎講座
catch トライボロジーのメカニズムを理解し、性能改善が可能か？対応策があるのか？改善効果をどう評価すれば良いか？を経験豊富な講師が詳解！
description

【講座の趣旨】
　トライボロジー現象は、複雑で対応策を立て難いという印象を与えています。しかし、基礎メカニズムを理解することで、どのような性能改善が可能か？また、そのためにはどのような方策があるのか？さらに、改善効果をどのように評価すれば良いのか？ということを整理することにより、対応策も立て易くなります。また、機械のメンテナンスにおいても、どのような点に注意しなければならないか？ということは、原理を知らなければできません。セミナーでは、実際への応用を考えて、基礎知識をわかりやすく解説します。

【習得できる知識】
・トライボロジーの基礎メカニズム
・トラボ特性改善を目的とした表面改質技術の概略と実例
・表面の機械的物性評価技術
・トライボロジー特性の標準試験
・トライボロジー技術の最近の動向

【プログラム】
１．トライボロジーの基礎
　1-1　トライボロジーとは
　1-2　固体表面と接触
　1-3　摩擦のメカニズム
　1-4　摩耗のメカニズム
　1-5　潤滑のメカニズム

２．必要とされるトライボロジー特性
　2-1　潤滑状態と支配因子
　2-2　必要とされる特性と特性発現メカニズム
　　2-2-1.省エネルギー
　　2-2-2.高耐久・高信頼性
　　2-2-3.新しい機能や性能

３．表面処理
　3-1　表面改質概論
　3-2　技術動向と具体例
　　3-2-1.DLCコーティング
　　3-2-2.表面テクスチャリング

４．トライボロジー表面の分析・評価方法
　4-1　表面性状評価
　4-2　表面の機械的特性評価
　　4-2-1.ヤング率と硬さ（粘弾性）
　　4-2-2.密着性
　4-3　摩擦・摩耗の標準試験方法
　4-4　トライボ表面の機器分析
