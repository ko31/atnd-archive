====講演概要====

Part1 『フリーのウェブアナリストとしての振り返りと今後の取り組み』 ウェブアナリストとしてフリーになり約2年経ちました。2016年はセミナー・勉強会の講師回数が100回近くありました。また数多くのお客様と分析・改善に取り組んできました。

講演では継続的にお仕事をするために行った取り組みや、効率化手法などを紹介いたします。具体的な成果や、2017年以降に向けての取り組みについてもあわせてお話しいたします。ウェブアナリストとして生きていくためのヒントや考え方などをお伝えできればと考えております。

Part2 『eMatrics Summit @ London 報告会　「ラディカルな解析アプローチ」』 2016年10月にロンドンで開催されたアナリティクスのイベント「eMetrics Summit」の本編と1日のワークショップに参加してきました。そこで学んできて考えた「ラディカル（根本的・急進的）」な解析アプローチ及び参加セミナーレポートを紹介します。
