■GSGLとは

・スマホ、ソーシャルメディア時代に、SAP海外進出を質の高いグローバル人材育成により支援!
・デジハリと日本MSが運営主体となり、その他複数の協賛企業の支援のもと、GSGLを開催し、即戦力となるグローバル人材輩出を目指します。

■登壇企業

株式会社gloops
Mobageを中心に『大熱狂！！プロ野球カード』『大進撃！！ドラゴン騎士団』など、累計登録者数1250万人を超えるソーシャルゲームを提供しているgloops。
国内だけではなくグローバル展開も進めており、アメリカ・ベトナムへの子会社の設立や、Facebookへのソーシャルゲームの提供もスタートしています！
一番のヒット作は会員数290万人を突破し、MLB開幕戦の冠スポンサーとして協賛するなど、快進撃を続けている同社にご講演いただきます！

株式会社ソーシャランド
株式会社ソーシャランドの事業ドメインはFacebookなどのソーシャルメディアを活用した一般ユーザー様向けサービスの開発運用。そして企業様向けFacebookを中心としたソーシャルメディアマーケティング活動の支援事業を行っている。今回の発表ではソーシャランド社がFacebook活用事業第一弾として手掛けた、インドネシア向けソーシャルマッチングサービス「menurut anda? （メヌルッアンダ？）」（ユーザ13万人！）について発表頂きます。スーパー先行事例です！
