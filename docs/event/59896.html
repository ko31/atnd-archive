description

シリコンバレー発の起業家育成プログラム「ファウンダー・インスティテュート関西」からのお知らせです。

日本全体で起業に対する意識が高まりつつあるなか、関西でも起業を支援する団体・企業・行政によるサポートが、以前と比べてだいぶ増えてきました。

しかしながら一方で、これらの支援リソースの存在を知らない方がまだまだ多いことも現実です。

自分の身の回りにどういった支援リソースがあるのか？関西のような地方都市で起業して、成功する確率を高めるにはどうすれば良いのか？そういった疑問にお答えさせて頂きます。

今回は「関西で勝てるスタートアップを作るには？」と題して、関西で活動されている起業支援団体の代表をお招きし、起業を志す人が利用できる支援リソースついてのパネルディスカッションを開催します。

これまで数多くの起業家の支援に携わられてきた以下の方々に登壇頂きます。

・大崎 弘子　オオサカンスペース代表
・嶋内 秀之　West Booster / 株式会社アントレプレナーファクトリー代表
・長川 勝勇　大阪産業創造館スタートアップ支援チームリーダー

普段なかなか揃うことがないスピーカーの方々ですので、このイベントで自分が利用できる支援リソースについて色々とご質問頂ければと思います。

■イベント対象者

・関西でスタートアップの立ち上げを検討している方、または立ち上げ中の方
・関西で利用できる支援リソースについて詳しく知りたい方
・アクセラレーター、コワーキングスペース、起業支援団体への参加を検討されている方
