description

本セミナーは、難解な数式やプログラミング言語を一切使わず、図と例え話だけでAIの基礎から応用まですべて解説します。

「えっ、そんなことできるの？」と一見、滑稽に感じるかもしれません。

たとえば、難しい専門書を読み込んだあと、頭の中でいろいろ咀嚼しながら整理し、ああ、そういう意味だったのか……ガッテン！」と思わず納得し、感動する瞬間があると思います。

つまり、時間をかけてAIの難しい理論や技術を学ばなくても、明確なイメージを掴むことによってその本質とノウハウを理解することができれば、AIビジネスをスタートアップさせることができます。

本セミナーでは、講師歴30年のGVA理事がAIのエッセンスをわかりやすく解説します。

＊イベントのイメージ

背景

平成２９年２月経済産業省「イノベーション・ベンチャー政策について」で べンチャー企業に関して下記の課題と政策が報告されています。

　(1)世界のトップ企業について、⽶国では約1/3が1980年以降に設⽴されている

　　⼀⽅、⽇本では約1/6に留まっており、グローバルで通⽤するベンチャー

　　企業の創出・育成も課題。

　(2)国内市場がある程度の規模があるため、⼩さくまとまるビジネスモデルが

　　多い。また、グローバルな市場や拠点とのつながりが⼩さく、グローバルな

　　視野を持った⼈材が少ない。

　(3)我が国社会においては、起業が職業の選択肢となっておらず、

　　新しいことに挑戦する起業家精神が低調。また、⼤企業を含め、

　　新たな事業を⽣み出すノウハウを持つ⼈材の層が薄い。

　(4)シリコンバレーと⽇本の起業家・企業をつなぐことで、世界市場で

　　打ち勝つベンチャー企業等の創出及びグローバルに通⽤しロールモデルと

　　なるイノベーターの基礎集団の形成及びグローバルで活躍するベンチャーの

　　創出を促進する。

　(5)今後の⽇本経済を⽀え、グローバルにインパクトを⽣み出す起業家や

　　ベンチャー企業、イノベーション企業が⾃律的・連続的に⽣み出す社会を

　　構築し、より便利でより豊かな国⺠⽣活を実現するため、様々な政策を

　　統合的に実施する。
