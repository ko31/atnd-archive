技術ひろば.net 2011年5月 勉強会＆お茶会

以下の要領で「技術ひろば.net 2011年5月 勉強会＆お茶会」を開催します。
「技術を楽しむ、技術でつながる」の理念の元、人と技術を大切にする活動を行っています。
どなたでもお気軽にご参加ください。

開催概要

■日時 : 2011年5月21日(土) 14:30～17:30
■会場 : 首都圏コンピュータ技術者株式会社 2F 大会議室
　http://mcea.jp/tokyo#maptitle
■会費 : 無料
■定員 : 10名

※ 会場が変更 になりました。ただし 同じフロア内 での変更ですので迷子になることはないと思います。

内容

久しぶりの開催となる 5月の技術ひろば.netでは、Silverlightでマルチタッチアプリケーションを開発する際のTipsを紹介します。
また、毎回好評なフリートーク、改めお茶会も開催します。「懇親会（＝飲み会）まで付き合うのは時間的、金銭的にちょっと」という方も、おしゃべりや情報交換をしましょう。

  • 「Silverlight で作るマルチタッチアプリケーション 」 （セッション形式：瀬尾）
    Siverlight はマルチタッチを処理することはできますが、Windows ネイティブや WPF とは異なり、
    マニピュレーション（＝回転、拡大縮小）のための API を持ちません。
    そこで自前で演算することで Silverlight でもマルチタッチマニュピレーションを実現する方法を紹介します。
    ※このセッションは Developers Summit 2011 での 17-D-3 ライトニングトークセッションの内容を
    改めて解説するものです。また 技術ひろば.net 2011年3月の勉強会 で実施予定だったものです。
  • 「projectぐぐる失敗談」 （セッション＆ディスカッション形式：早川）
    技術情報、LifeHack、ぐぐってますか？じゃぁ、自分たちのプロジェクト情報は？
    大規模、長期案件にて後から参加した場合にExcelに纏めていたファイル保管場所の表が古かった状況での格闘談、失敗談を話します
  • お茶会
    お茶でも飲みながら、自由におしゃべりや情報交換をしましょう。
    各自お好みのノンアルコール飲料を持ってきてください。
    （アルコールはお茶会の後で）
