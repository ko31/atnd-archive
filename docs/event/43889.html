description

最近、「スマートフォン」と呼ばれる端末が多くなってきました。 好きなアプリケーションを入れて、自分に最適なカスタマイズができる、いままでのケータイとは異なるところの多いスマートフォン。これらスマートフォンの懇親会を行いたいと思います。
これは、スマートフォンについて、自分の思い入れや自分の使い方など、みんなでわいわい話しあうことを目的としたイベントです。

概要

対象

スマートフォンをお持ちの方・興味がおありの方。

ライトニングトーク

当日は、ライトニングトークも開催します。

ライトニングトークとは、一人5分程度で行われる短いプレゼンテーションです(こちらでは特に明確な制限時間は設けません)。

テーマは、スマートフォンに関わることなら何でもOK。興味のあること、発表したいこと、是非みなさん発表してみましょう。

ライトニングトークスピーチ希望の方は、以下のコメント欄や参加時コメントにスピーチ内容を記入してください。

LTリスト

当日のライトニングトークのリストです。現在一人がスピーチ予定です。

  • ゆねばゆうた「Facebookでイベントページを運用してみて」(Ustreamなし)
  • 高見知英「買い物2.0」

USTREAM

本イベントは、ライトニングトークのUStreamによる中継を行います(ただし、会場の都合などにより上手く行えない場合がありますのでご了承ください)。当日来られない方は、こちらもご覧ください。

スマートフォン懇親会 Ustreamチャンネル

カメラはプレゼンテーションが写るように配置します。プレゼンテーションが写る範囲に入りますと、自分も撮影されますのでご注意ください。

会場情報

参加費など

無料です。ただし、会場はハンバーガーショップですので、ワンオーダーをお願いしています。
メニューについては お店のサイト をご確認ください。

電源と無線LAN

会場は電源および、無線LANが使用可能です。
必要であれば電源ケーブルなどをご用意ください。
