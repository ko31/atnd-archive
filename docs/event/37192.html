＜講座内容＞
1. シェル、スクリプト、およびデータ管理
　• シェル環境のカスタマイズと使用
　• 簡単なスクリプトをカスタマイズまたは作成する
　• SQLデータ管理

2. ユーザインターフェイスとデスクトップ
　• X11のインストールと設定
　• ディスプレイマネージャの設定
　• アクセシビリティ

3. 管理業務
　• ユーザアカウント、グループアカウント、
　　および関連するシステムファイルを管理する
　• ジョブスケジューリングによるシステム管理業務の自動化
　• ローカライゼーションと国際化

4. 重要なシステムサービス
　• システム時刻を維持する
　• システムのログ
　• メール転送エージェント(MTA)の基本
　• プリンターと印刷を管理する

5. ネットワークの基礎
　• インターネットプロトコルの基礎
　• 基本的なネットワーク構成
　• 基本的なネットワークの問題解決
　• クライアント側のDNS設定

6. セキュリティ
　• セキュリティ管理業務を実施する
　• ホストのセキュリティ設定
　• 暗号化によるデータの保護

＜講師からのメッセージ＞
お世話になります、（株）クリエイティブワークスの成瀬です。

資格は取るだけでは何の意味もないかもしれません。
それでも、資格があるからこそ新しい案件にアサインされたり、
転職活動時等に前向きな姿勢が評価されることがあったりします。
未来の扉を開けるために必要な鍵の１つが、資格であると私は考えています。

しかしながら、資格試験受験においての第一歩はどこから手を付けたら
いいか難しいですよね。
実機環境がない。どう作ったらいいかわからない。
教わることができる先輩がいない。
実際、どのように資格試験に資格を取ったらいいのかわからない。
会社がまだ小さくて、研修制度が整っていない。
この講座は、そんな人にこそ受けて頂きたい講座だと考えています。

もちろん、4時間という短時間ですので、若干駆け足での進行かと思います。
ただ、実機環境もお持ち帰りできますので、
復習していただければ合格圏内にたどり着けるはずです。

努力家の貴方とお会いできることを、楽しみにしております。
