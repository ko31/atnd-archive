description

昨今のスマートフォンアプリ市場では、アプリをリリースしたのは良いものの、長くアプリを運営することはとても難しい環境となっています。長くアプリを運営するためには一体何をすれば良いのか、効果的な施策とは何なのかを見つけるのはとても難しいことです。

本セミナーでは、『長くアプリを運営する為に追うべきKPIについて』というテーマで、アプリの品質を改善するための指標であるクラッシュ率というKPIをご紹介いたします。 クラッシュ率を改善することによってアプリの品質を向上させ、既存ユーザーの継続率を上げるノウハウをご紹介いたします！
セミナーにご参加頂いた方には、自社アプリの品質が現在どうなっているかを知ることができる『アプリの健康診断』も行なっておりますので、ぜひお気軽にご参加くださいませ。

セミナー終了後には個別相談会を設けておりますので、課題をお持ちのご担当者様がおりましたら講師陣より個別に情報提供をさせて頂きます。

※　こんな方にオススメです！　※

セミナー内容

■長くアプリを運営する為に追うべきKPIについて

登壇者：吉井 文学  -  FROSK株式会社 CMO

クラッシュ率を改善しアプリの品質を向上させることによって、ユーザーの継続率を高められる手法を具体的なデータをもとにお伝え致します。人気アプリの事例を踏まえた運用/品質改善のノウハウをご紹介するとともに、導入アプリMAUが1.7億人にのぼるSmartBeatのデータから、最新のOSバージョンシェアなど、このセミナーでしか見られない貴重なデータも多数ご紹介します。
