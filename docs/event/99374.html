去る7月2日、ネットプロテクションズは事業拡大に伴い東京本社オフィスを麹町へ移転しました。
新しいオフィスのコンセプトは 「Activity Based Talking～働くというより話そう～」 。
オフィス内にコンタクトポイントを多く設置することでコミュニケーションを活性化し、「つぎのアタリマエをつくる」ためのアイディアが、これまで以上に生み出せる環境を目指しています。

この度、移転を記念しまして新オフィスのお披露目イベントを企画いたしました！

本イベントでは、新オフィスを実際に見てまわっていただけるオフィスツアーのほか、
移転プロジェクトにご参画いただいたパートナー企業様をお呼びしての対談企画を実施する予定です。
お酒や軽食もご用意しての交流会も行いますので、お気軽にご参加ください。
参加を希望されます方は、下記の参加申込フォームよりご応募お願いいたします。
