講師紹介

垣花 顕 (Akira KAKINOHANA)
URL : http://ja.kirameister.net/
Twitter : http://twitter.com/kirameister_ja

高校は沖縄県立の浦添工業高校へ。高校在学中に運良く国費の奨学金を当ててアメリカに留学する。
帰国後高校を卒業し、琉球大学の情報工学科へ入学。
大学在学中にドイツへ遊びに行く。その後色々あったが、最終的に DAAD (ドイツ学術交流会)という
ドイツの政府機関からお金を貰って、ドイツの Saarland University で Computational Linguistics を学ぶ。
修士課程を修了後、日本にある NTT研究所発の企業へ就職。1年経たないうちに日本にいてはマズいと考え、
海外へ転職しようとする。その数か月後にウィーンにある今の企業へ転職し、以来ウィーン在住。

好き : 英語配列キーボード、面白い人との会話、下ネタ、読書
嫌い : 車の運転、飛行機による長距離移動 (と乾燥した空気)、寒さ

講演の目的・講演者より

現在私 (垣花) はウィーン (オーストリア) で働いています。情報工学科を卒業し、自分の勉強 (研究) したかった Computational Linguistics をドイツで学び始めたのは良かったのですが、そこでのコースを修了したあと日本の企業で働き始め、しばらくすると「これで良いのだろうか?」と考える様になりました。自分自身を労働市場という視点から視た時に、どのような選択が自分にとって望ましいかを考え、現在ウィーンで技術翻訳者として働いています。

この発表では留学の方法論や国外で働くメリット、または英語学習の方法などの「ハウツー」を言うつもりはありません (少なくともそれらはターゲットとしたいトピックではありません)。それよりも、「なぜ」私が日本国外に出るような決断をしたのか、その決定のベースとなる私の思考 (方法) について主に述べたいと考えています。私の様な情報工学科にいながらドイツ語を学習し、卒業後修士にてドイツへ正規留学し、日本で働いた後、再度ヨーロッパ (それも非英語圏) で半分技術者として働きだすという日本人は、私の知る限り皆無です (いることにはいると思いますが、数は少ないでしょう)。従って私が今まで辿ってきた軌跡が他の人の参考になるとはそこまで思えませんが、そのような行動を行うことになった思考法は、少なくとも (他には無いという意味で) 参考になるところが幾分かはあると考えています。

発表のトピック・流れ

  • 私の簡単な紹介 (with ice-breaking)
  • どこへどのように留学したのか? 留学後どこに行ったのか? 何故また日本国外へ?
  • 労働市場における個人の価値とは? そもそも労働市場とはどこにあるのか?
  • 人生を戦略的に考える
    • 人間は必ず死ぬ (そして死ぬときは簡単に逝く)。ここに今いる人間は全員、恐らく100年後には死んでいる
    • その人にとって何が重要なのか? 捨てるものと捨てないものを見極める
    • 人生は長期戦。数十年というスパンで考える必要がある (i.e., 何故公務員なのか?)
    • 「何」の為の人生かは人によって異なるが、「誰」の為の人生かは恐らく殆どの場合一致するだろう
  • 私の思考の核となる過去の体験
    • アメリカでの体験 : キリスト教原理主義の家庭で約１年間の滞在 : 「話せば分かる」などという言葉は時として非常に虚しい & 人間は簡単に死ねる
    • 日本への飛行機の中 : (私のように) 日本語が話せない人間というのは少数派だった : 限られたリソースの中で何ができるのか? 何をしたいのか?
    • ドイツでのクリスマス体験 :もう１回死を強烈に意識して、何がしたいのかを考える。墓場までお金は持っていけないし、死ぬ直前に「もう少し職場にいたかった」という人も恐らく少数派だろう。
  • 日本の外の常識
    • 「分かり合えない」のがデフォルト (e.g., メルケル独首相の発言)
    • それでも世界は狭くなっている (e.g., 電話相手の localtime を考える)。どのように社会生活を営むのか?
    • ソフトスキルとは? ハードスキルとは? どちらが重要なのか?
    • 世界規模の頭脳獲得合戦 : 国家の仕事が変わってきている
  • 日本のこれからと個人
    • 英語を公用語とする日系企業や、日本の採用を減らしながら海外のそれを増やす企業。
    • ３大都市で産まれる人の10%は両親の片方か両方が非日本人。
    • 40年後 (つまり今の20歳が60歳になる頃) まで英語 (または中国語) を話さないで生きていく事はできるのか?
  • 私から学生への質問
    • 留学を考えた事はあるか? あるとすれば具体的な行動はどのようなものか?
    • 大学での１コマ (または１時間) あたり、いくら払っているか計算した事はあるか? (何故大学にいるのか?)
    • (公務員になりたい人は) 何故今公務員になりたいのか? それは目的に対して妥当な手段なのだろうか?

学生への質問（事前に各自考えてみて下さい）

  • 卒業後沖縄に残りたいか? 残るとしたらいつまで残りたい? その理由は?
  • 90分の授業１コマに自分がいくら払っているか計算した結果は?
  • 自分は何歳まで生きると考えているのか?
