①アシアル株式会社

＜講演内容＞
　今、最もイケてるPHPフレームワークLaravel4
　このフレームワークへ行き着いた経緯とLaravelの今後の流れの紹介します。

　対象：PHPフレームワーク選定に悩んでいるエンジニアとプロマネージャー

＜会社説明＞
　株式会社インテリジェンス キャリアDiv マーケティング企画統括部　
　総合人材サービス会社「株式会社インテリジェンス」のマーケティング部門に今年から設立された技術部門です。
　「社内業務システム」「新規Webサービス」を中心に開発を行っています。
　企画・開発、継続的なサービス改善までをカバーする、技術に特化した組織です。

＜講演内容＞
　「スピード」と「品質」を実現するPHP開発チームの取り組み
　～AngularJS + FuelPHP + AspectMock～

＜登壇者プロフィール＞
　株式会社インテリジェンス
　マーケティング企画統括サービス開発部
　清田 馨一郎氏

　独立系SIにて、基幹系システムからソーシャルゲーム開発まで幅広く手がける。
　データ分析、マーケティングに関わりたいと現職へ。
　HR/HMを愛するメタル系エンジニア。

③ランサーズ株式会社

＜会社説明＞
　日本初・日本最大のクラウドソーシングサイト「ランサーズ」を運営。
　2014年8月現在、ランサーズ上での依頼総額は300億円を突破、
　また、実績とスキルのある全国各地の登録者が36万人を超えています。

＜講演内容＞
　Behatで行う、E2Eテスト入門。
　PHPのBDDテストフレームワークを利用して行うE2Eテスト。

＜登壇者プロフィール＞
　堀川 豊氏（WEBアプリエンジニア）
　公式携帯サイト運営やソーシャルゲーム開発を経て、2014年ランサーズにJOIN
