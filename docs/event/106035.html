イベントの説明

弊社の運営しているプログラミングスクールCodeShipの無料体験会となっております。体験会の内容は、CodeShipで使うカリキュラムの一部を実際に使い、WEBサイトのレイアウトを作成していただきます。レイアウトの作成をするなかで、学べる点と致しましては下記に記載されております。

・Webページの基礎知識

・Visual Studio Codeの活用方法

・HTML5/CSS３の基礎的な実践学習

・Emmetを使ったコーディング方法

CodeShip（https://code-ship.wemotion.co.jp）について

CodeShipは池袋にある短期集中学習型のプログラミングスクールです。

就活を成功させたい学生の方、キャリアアップやキャリアチェンジを目指す社会人の方が多く通われています。身につくまで学べる「無償延長保証」、目的に合わせて学べる「様々な学習領域のコース」、就職・転職で活用できる「オリジナルのポートフォリオ制作」などの様々な特徴があります。
