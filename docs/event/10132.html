description

ボードゲームときいて思い浮かべるのはどんなものですか？
すごろく、オセロ、人生ゲーム…。
ではボードゲームに対するイメージはどうでしょうか？
新鮮味に欠ける、白熱しない、ツマラナイ…。
もしこう思っているなら、そんなあなたに朗報です。
時間を忘れるようなオモシロいボードゲームを体験できる機会を用意しました。

海外では次々とボードゲームが生み出されています。
そのため、ボードゲームの賞が設立されたり、
展示会が開かれるなど盛り上がりをみせています。
そこに集まるボードゲームは戦略と運、駆け引きのバランスが
絶妙なものばかりです。
それぞれのボードゲームには世界観が
しっかりと作り込まれていて非日常な世界を体験できます。

おいしいごはんとボードゲームがあるラボへ参加してみませんか。
