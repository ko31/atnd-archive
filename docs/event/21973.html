description

12月18日　日曜日に二子玉川の河川敷で個々にお弁当を持ち寄って
簡単なピクニックと撮影会をしたいと思います。
この時期寒いかもしれませんが写真を撮るには
なかなかの楽しいところでした^^
予定としては１１時に現地集合で日没後の５時くらいから
近くの居酒屋で忘年会をはじめたいと思っています
ピクニックだけの参加でもＯＫ！忘年会だけの参加もＯＫ！
１１時からですが途中からでもＯＫです。
時間に関係なくお好きな時にお越しくださって結構です^^
（ぜひお友達も誘ってお越しください♪）

忘年会の場所は人数にもよりますので後日発表させていただきます
日中ピクニックもあるので簡単な感じで予算も控えめに抑えたいと
思っています。

※　参加ボタンをポチッとする際はコメント欄に
　 忘年会へ参加か否かもよろしくお願いします。

※　お弁当や飲み物は各自でご用意ください
　 缶やペットボトルを捨てるゴミ箱は公園内にございます

※　場所は東急田園都市線の二個玉川駅を下車して徒歩５分ほどで
　 東急田園都市線の陸橋した付近です。

■　忘年会の募集は終了させていただきました。

忘年会会場は「うまいもの市場ＴＡＰＡ 二子玉川店 」
会費は３時間飲み放題で２９８０円となります
掘りごたつのフロアー貸切となります。
http://r.gnavi.co.jp/g066104/

明日は１１時からスタートですが１１時から１７時の間で
お好きな時間に来てくれて構いません、
場所は二子玉川の河川敷にある兵庫島公園です。
東急田園都市線の陸橋した付近にいます。
場所への簡単な説明ですが東急田園都市線の二子玉川の改札を出て
左へ行くと直ぐに多摩川通り（２４６）になります。そこを渡らずに
左へと曲がり進むと信号があるので渡ってください
信号を渡ると陸橋の側道奥にジャイアントと言う自転車メーカーの
お店があるのでそこを右に曲がって陸橋をくぐって直ぐ左に
河川敷に下りる入り口がありますのでそこを下りて小さな橋を渡って
左手に東急田園都市線の陸橋があります。
明日は総勢５０人越えと大人数なのでひとりでは対応しきれないので
わからなければご自身で聞くなり調べて来てくださると助かります。
それでは明日は寒さも厳しいらしいので寒さ対策は万全にして
お越しくださいね^^
