Windows Phone Advent Calendar : Consumer Track って？

クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーがあり、そこから技術系コミュニティで、クリスマスまでの25日間にTipsを1つずつ公開するイベントをAdvent Calendarというそうです。今年もいろいろなAdvent Calendarが企画されると思います。

というわけで、本イベントは Windows Phone ユーザー版です。12月1日から25日までの間、参加者が日替わりでblogを更新していきます。お題はWindows Phoneの「利用」に絡めばなんでもありです。アプリ紹介とかでもいいと思います！

  • Windows Phone 8 SDK は こちら → Windows Phone 8 SDK Advent Calendar

ルール

  • Windows Phoneに関するTipsを順番に1つずつブログに書いていきます。
  • 書く順番はこのイベントに参加した順番です(最初に参加した人が12/1、最後に参加した人が12/25)
  • 記事を書いたらこのイベントのコメントにURLをはってください。
