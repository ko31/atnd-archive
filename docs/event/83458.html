■イベント概要

第１部：フリーランス講座

・フリーランスエンジニアとして働くメリット・デメリット
・お金について(会社員時と比較してどれくらい上がるの？)
・ワークスタイルについて(どれくらい自由度が上がるの？)
・リスクについて(案件はそんなにあるの？今は良くても将来は不安じゃない？一人で孤独じゃない？)
・事務手続きについて(独立までの流れ、税金、確定申告、その他役所への手続き等)

第２部：座談会 （現役フリーランスエンジニアを交えてのざっくばらんな座談会）

・フリーランスになったきっかけと動機
・フリーランスになって良かったこと
・フリーランスになって苦労したこと
・フリーランスで活動していく上でのTipsなど

■対象となる方

・フリーランスになりたてのエンジニアの皆様
・フリーランスになろうとしているエンジニアの皆様

■費用

無料です！

■その他

お腹が減る時間帯ですが、軽食(ピザ等)とドリンクもご用意してますので、 お仕事帰りに是非お立ち寄りください。
