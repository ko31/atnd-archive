ヒカ☆ラボ一覧は こちら！！

★講演内容

従来の家電をスマート家電へと変化させるWebサービスを提供する東大発のベンチャー”株式会社Pluto”。

フロントサイド・サーバーサイドだけのソフト面だけでなく、
ハードウェアまでをほぼ2人で開発した当社ですが、
今回はCTOの市東が、プログラミング言語の本質について語ります！

主な講演内容は、以下の3点です。
・Plutoが思い描くスマート家電の未来のお話
・PlutoというWebサービスの裏側に隠された立ち上げ秘話
　～今に至るまでの苦労とPlutoを支えた技術とは？～
・ソフトウエア開発者に向けた、ハードウエア開発の簡単なイメージと考え方
　～コンパイルするとはどういうことか？～

ハードウエアからプログラムを始めた関係上、
普通のプログラマーとは違った目線で物を見ている事があり、
ハードウエアからみたプログラムについても解説したいと思います。

※内容は変更になる可能性がございます。
