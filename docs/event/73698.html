【CTO Night！】BtoCサービス開発の裏側対談！エアークローゼット社をはじめCTO３名が登壇！

テクスタLIVEイベント一覧はこちら：https://tech-hub.doorkeeper.jp/

講演内容

本講演は、各企業でCTOとしてご活躍されているお二方をお呼びして、
「BtoCサービス開発の裏側」について熱く語っていただきます！

登壇していただくのは、株式会社airCloset（エアークローゼット）でCTOとして月額定額制のファッションレンタルサービスの開発を行っている辻 亮佑さん。
株式会社モトアップのCTOとして、ウェブインテグレーション事業やシステム開発事業を手がけている井出新吾さんの２名です。

それぞれのCTOの独自の視点から、様々なお話をお伺いしていきます。モデレーターは、Branding EngineerCTOの浜田が務めます。

ここがPOINT！

・経験豊富なCTO３名が登壇
・各分野、異色の３視点でのパネルディスカッション
・CTOならではの話が豊富

登壇者


モデレータ　浜田 卓


1989年生まれ。2014年に東京大学大学院情報理工学研究科コンピュータ科学専攻を卒業した。
その後、世界一の求人検索エンジンを持つ米企業、Indeedに日本人初の新卒社員として入社。

同社を退社後、自らでエンジニアのための理想の組織作り、データ・技術ドリブンの視点でエンジニアの雇用問題を解決するため、Branding EngineerにCTOとして参画した。
現在はBtoB/BtoC両視点が必要な「Tech Stars」の開発を中心に手がける。


辻 亮佑氏


2009年、エンタープライズ向けの開発会社AISにてキャリアをスタート。インフラ、DB、バックエンド、フロントエンドなど多岐にわたる開発に従事し、開発の面白さにのめり組む。

2012年から楽天株式会社にて、フロントエンドに特化して新規立ち上げビジネスの設計、開発、チームマネジメントを担当。その後、株式会社ノイエジークCTOを経験。

現在は株式会社エアークローゼット(airCloset)CTOとして活躍中。

井出新吾氏

77年生まれ。大学在学中にアメリカに短期留学中にインターネットの素晴らしさに触れ、プログラマーに。大手コミュニティサイト運営会社に派遣で入社その際に独立。

その後Livedoorと契約し数十台のサーバー運用・負荷分散・アプリケーションリニューアル等対応を行う。その経験を活かし受託で大手サイト(PV7億)/新規サイトの立ち上げから7000万PVまでのサイトスケールアップなどを行い、その時の負荷分散の経験を用いて現在は自社製品の開発及新規受託案件の拡大を行っている。
