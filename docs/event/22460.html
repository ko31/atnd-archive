TDD Boot Camp 福岡 2

TDD Boot CampはTDD（テスト駆動開発）を実際に体験しながら学ぶイベントです。
TDDBCは全国各地で開催されており、福岡でも2011/03に開催されました。
1日目の夜に、講師と参加者の皆さんで懇親会を行いたいと思います。
焼肉の食べ放題(飲み放題付き)で、TDDの黄金の三角形に負けないような、焼肉の三角形を回しましょう！
(懇親会のみの参加も可能です)

会費

予算は3,500円を予定しております。
