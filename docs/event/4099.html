■■■チベットのうたごえ「バイマーヤンジンコンサート」

チベット出身のバイマーヤンジン。来日して15年、豊か
な日本への驚き、日本でみつめなおしたチベットのこと
などを軽妙な大阪弁で語り、歌います。
　　
バイマーヤンジン(四川省アバ・チベット族自治州出身
（四川大地震震源のすぐそば）:中国国立四川音楽大
学で西洋オペラを専攻後、同大学専任講師。94年来日、
チベットを紹介するため全国でコンサート、講演活動
を行っている。また、チベット族の文化を守るための
学校建設活動をチベット地域で継続中。現在、拓殖大
学客員教授。　http://yangjin.jp/ )
