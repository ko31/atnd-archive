description

　みんなでネタを持ち寄ってお話しませんか！？何かしらBlenderを絡めたネタを持ち寄って発表していただき、それを元に軽く質疑応答をして、意見交換やアドバイスの場になればと思います。少人数を予定しているので、発表初心者の方の練習の場にもなればと思います。どんな些細な事でもかまわないので、 肩の力を抜いて気軽にご参加下さい！特に初〜中級者の方、「これどうやろな〜」とか、「ここ分からんねんけどな〜」という困りどころがあれば、シェアしてみれば解決できるかもしれませんよ！もちろんガチな発表も歓迎です！

参加費
　コワーキングスペースの利用料金（ 1,200円+税 ）を各自負担して頂きます。会員未登録の方は早めに来て登録願います。(要身分証)

参加条件・注意事項
・Blenderに絡めた何かを発表してもらいます。(スライドや資料無しでもOK!ソフトを操作するだけとか、物理的に作ったものを見せて貰うとかでも大丈夫です )
・参加登録時、コメントに大体で構わないので発表内容を記載して下さい。
・持ち時間は1人10分＋質疑応答5分ぐらい（±5分？）を予定しています。
・発表内容が当日までに変わる時はこのイベントページ下のコメントに追記して下さい。
・繰り上がりの都合などもあるので、参加が難しくなってきた場合は前々日まで(27日中)を目処にこのページにてキャンセル処理をお願いします。
・プロジェクターを私が借りておきます。接続はアナログVGA＋HDMIです。(HDMIへの変換アダプタを私が持って行きます)
・その他設備の詳細については Co:Labo なんばさんのWebサイト を参考にして下さい。

発表内容例
・初めてBlenderを使ってここまで出来た！
・機能を追加してみたので紹介
・気になる機能を使ってみた感想
・制作中の作品の紹介
・etc…
Blender＋α大発表会 さまを参考に企画させてもらいました。そちらも合わせて見てみて下さい。（というか首都圏の人は是非行きましょう！)
