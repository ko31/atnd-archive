[再演版]Web制作者のための黒い画面（ターミナル）入門勉強会 : ATND

概要

  • Web制作者向けの記事を読んでいて「以下のコマンドを実行し…」などと書かれていても、どうすればよいかわからない
  • なんとなく「黒い画面（ターミナル）のことを知らないとまずいんじゃないか…」と思っているけど勉強できていない

という方に向けた勉強会を 6/9（月） 20:00 から渋谷 Stocker.jp / Space にて開きます。
こちらの勉強会は無料です。

主な内容

この勉強会では、ターミナルに慣れていただくための第一歩として、下記のような基本のファイル操作を中心に行います。

  • ターミナルの起動
  • ファイル移動、コピー、削除
  • 空のフォルダーやファイルを作成する
  • 複数の空のフォルダーやファイルをまとめて作成する
  • Vi, Vimテキストエディターを使う
  • ファイルをダウンロードする
  • ZIPファイルを解凍する

いきなり自分の Mac 上でターミナルを使ってファイル操作を行うのは不安がある方もいらっしゃる方もしれませんし、Windows をお持ちの方はそのままではターミナルを使うことができませんので、前半は Javascript PC Emulator というWebアプリを使い、Webブラウザー上の仮想の Linux を使用して基本的なファイル操作の練習をします。

後半は Mac 上のターミナルアプリを使い、より効率のよいファイル操作と、Mac 特有の Finder とターミナルの連携について学びます。

講師

講師は、庄崎大祐（Webデザイナー）です。
Stocker.jp / diary というブログや「プロになるためのWebデザイン入門講座」や「神速Photoshop [Webデザイン編]」という書籍を執筆しています。

必要なもの

  • ノートPC（MacBookまたはWindows）

この講座は説明だけでなく、実際にターミナルを操作してみることでより理解を深める講座です。
Windows の場合、後半の Mac 特有の操作については見学となりますがご了承ください。
