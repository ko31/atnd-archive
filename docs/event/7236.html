description

夏から秋へ季節の移り変わるころ。
過ぎゆく夏に感じる名残。
迎える秋を想うこころ。

「灯り・空間・食」でかたちにしました。

Atelier Romeo Styleのスクール生による作品展と、やわらかな灯りとともに過ごすはるはカフェでの寛ぎのひとときを、また、夜は優しいひかりに包まれたキャンドルナイトをどうぞお楽しみ下さい。

◇オープン日の夜は「はるは」プロデュースのフィンガーフードパーティーです。どうぞお気軽にお越し下さい。
◇キャンドル制作のワークショップ・作品の販売もございます。
◇はるはカフェでは、素材や季節にこだわった「マクロビオティック」でのお食事をお楽しみ下さい。

【詳細・お問い合わせ】
食空間スタイリングはるは
WEB　http://haruhacafe.blog88.fc2.com/
MAIL　haruhacage@gmail.com
担当　増本・半田
