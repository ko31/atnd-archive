情報処理推進機構(IPA)、内閣府 内閣官房番号制度担当室、福岡エレコン交流会、応研株式会社の後援・協力による無償セミナーです。
　
中小企業・各種法人の経営者・責任者・IT担当者様、個人事業主様に向けて、最新の実例紹介や、映像の視聴等で情報セキュリティについて基礎的な理解を深めていただく内容となっております。
後編では、マイナンバー関連の基本的な理解と関連する法令について確認し、更に中小企業向けの正しく、かつ最小限必要な対応方法について解説させていただきます。
　
経理やパソコンを担当されておられる方ではなく、経営者や管理職の方に理解して頂くための分かりやすいセミナー内容となっております。

詳細は弊社Webサイトをご覧ください。
お申し込みについては、FAXもしくは電子メールにて受付をしております。
