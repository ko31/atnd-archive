OVAがアドテクノロジーを用いて生きづらい若者を特定して支援をする
活動を始めて4年、NPO法人を設立してから3年が経ちました。

私たちが始めた
「テクノロジーを用いて声なき声に支援やその情報を届ける（アウトリーチ）」
という考えは様々な領域で、全国的に広がりをみせています。

当日は、OVAのこれまでの歩みとこれからの展望のご説明に加えて、
最先端のテクノロジーを活用したソーシャルグッドについて、
現在、人工知能（AI）を用いて活動している稀有な2名のゲストと共に
来るべき未来について皆様と一緒に考えたいと思います。

▼登壇者紹介

髙岡 昂太さん
（産業技術総合研究所人工知能研究センター研究員／NPO法人Child First Lab.代表）
東京大学大学院教育学研究科博士課程修了（教育学博士）、臨床心理士。
ブリティッシュコロンビア大学への研究留学を経て、
2017年3月より産業技術総合研究所人工知能研究センター確率モデリング研究チームにて、
生活現象における人工知能を用いたリスク予測と
意思決定支援プラットフォームの研究開発に従事。
NPO法人Child First Lab.では、子ども虐待における現場支援モバイルアプリ開発を進めている。

浅見　淳さん（株式会社LITALICO　ライフネット支援室長）

精神保健福祉士。株式会社LITALICO　ライフネット支援室　室長
2009年、リタリコに入社。
拠点長、エリアマネージャーを経て、就労支援研修センターを設立、
支援者育成のパッケージ化を進める。

2014年、ライフネット支援室を立ち上げ室長に就任、同社就労移行
支援サービス利用者の自殺予防に本格的に着手。2016年には人工知能
エンジン「KIBIT」を用いた自殺の予兆を早期発見する新たな仕組みの
開発にかかわり、現在も運営に尽力している。

伊藤 次郎 (NPO法人OVA　代表)
精神保健福祉士。人事コンサルティング会社（EAPプロバイダー）を経て、
精神科クリニックにてビジネスパーソンの復職支援を行った。
2013年より若者の自殺予防のために
アドテクノロジー用いた「夜回り2.0」の手法を開発・実施し、NPO法人OVAを設立。
日本財団「ソーシャルイノベータ―」選出（2016）
朝日新聞「ひと」、ＮＨＫ「ハートネットＴＶ」などメディア掲載多数。
