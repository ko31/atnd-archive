九州大学で勉強会を開きます．

授業では習わないことは自分たちで勉強するしかない．
でも，一人では始めにくいし，どうやって勉強すれば・・・
何が流行ってる技術なのかわからない・・・
自分の勉強をしてることをもっと他の人に知ってもらって仲間を増やしたい・・・
都市部では勉強会やら流行ってるみたいだけど，学校は天神・博多から遠いし・・・

そんな人たちのために，伊都キャンパスで勉強会の場を用意します．
メインは交流会ですが，自分の製作物を発表したいという方には別途お時間をご用意いたします．
発表時間は50分，20分もしくは5分と発表の長さで選んでもらって構いません．
発表したい人が発表する形式です．学生が発表するという場は少ないので「人にわかりやすい発表」を目指す意味でもどうですか？
自分の興味ある回に参加すればよく発表者以外は当日キャンセルなど自由です．
学年は関係ありません．学部1年であろうと修士だろうと関係なく皆さん勉強しましょう．

発表したい方はTwitterアカウントまたはこちらのページにぜひぜひご連絡ください．
参加されたい方はATNDアカウント作成後に参加のボタンを押してください．

1月23日の18：30～21：30を予定しています．
部屋は伊都キャンパス　ウエスト4号館　312号室です．
学外からお越しの方は下記のURLに学内地図をご用意しております．
学内地図
※室内は飲食禁止です．借り部屋ですので今後のためにも飲食はお控えください．
