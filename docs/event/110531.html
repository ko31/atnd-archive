【ハンズオン講習 & ハッカソン】Node.jsでLine or Slackのbotを作ってみよう #Cording owl


Cording owls by KOMAZAWA_THON × Code Village

イベント概要

Node.jsを使ってLINEやSlack、TwitterなコミュニケーションアプリのBotを作りましょう。ハンズオンで形式でNode.jsの使い方の講習を行った後、実際にハッカソン形式でチャットボットを作成していただきます。コミュニケーションツールやボットの形式、内容は自由です。様々なアイディアを形にしてみてください。

イベント終了後は、希望者でバーカウンターのある会場で懇親会を行います。軽食やお飲物を用意しております。奮ってご参加くださいませ。

（別途参加費 1000円）

Node.jsは一般的にフロントエンド開発に用いられるJavaScriptでサーバーサイドの開発ができるようになる環境です。同時に大量のトラフィックを捌くことができ、最近の動画配信サービスやチャットアプリ、オンラインのゲームの開発で広く使われています。メモリ消費量が少なく、小さな規模の開発・運用時に、特にパフォーマンスを出すことが可能です。
