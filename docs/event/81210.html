イベント概要

フルHD対応、複数サービスの同時配信、録画機能など多数の新機能を搭載したLiveShellシリーズの最新モデル「LiveShell X」の体験イベントです。LiveShell Xの動作デモをご体験いただけるほか、ビデオカメラやレコーダ、スイッチャーなどご自身の機器をお持ちいただいて試すことも可能です。



ライブ配信機器「LiveShell X」の特徴

  • パソコンいらずで、6時間の安定配信を実現
  • 最大解像度1080/60p。コーデックは最新のH.265も対応
  • 最大3ストリームまでの同時ライブ配信
  • 配信と同時にmicroSDカードへの録画機能を搭載
  • 携帯性の高い手のひらサイズの筐体
  • 映像や音声の配信状況を確認できるモニタリング機能
  • ネット回線不要のローカル配信など多彩な配信モードに対応
  • 自動配信再開機能やアナログ音声入力など配信に便利な数々の機能を搭載
  • スイッチャー不要、本体のみでロゴやテロップを挿入・切り替えが可能
