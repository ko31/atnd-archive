つくらぐ勉強会は、つくらぐの活動目的の内「Linux, FOSSに関する知識・ノウハウの学生間での共有と相互サポート」の実現を担うものです。 主にテクニカルな内容の発表及びそれに関する議論を目的とします。


== 開催日時等 ==

  • ’’’日時: 2010年 03月 31日 (土) (開場 16:30) 17:00 – 19:00’’’
  • 開催場所: 筑波大学 第3エリア B棟 302室 ‘’’[3B302]‘’’ [http://maps.google.com/maps?f=d&source=s_d&saddr=36.111184,140.100236 地図]
    • ‘’’[[筑波大学へのアクセス]]‘’’
  • 費用: 無料
  • 定員: 20人
  • 対象: GNU / Linuxに興味のある方ならどなたでも。初心者大歓迎。筑波大学関係者である必要はありません。
  • 備考: ノートパソコンを持ってくるといい事があるかもしれません。(電源、無線LANは極力用意致します。emobile電波入ります。)
    ‘’’ 注意事項: Ustream や YouTube で動画配信する可能性があります。参加者の顔を映す予定はありませんが、後ろからちらっと映ってしまう可能性はあるので、ご了承ください。’’’
