「読書会」というと、
事前に本を読まなければならないとお考えではないでしょうか？

しかし、この読書会は違います。

このしつもん読書会は、

「事前に本を読まずして、

用意されたしつもんを使い、

本を読み解くことで、

自分自身と深く向き合うことができ、

なおかつ一冊の本のエッセンスが理解できる読書会
なのです！

そこで、今回は『質問は人生を変える』をテーマに読書会を行っていきます。

著者のマツダ ミヒロの集大成の本でもあり、
質問力をマスターする最適な本なのです。

質問力が身につくと・・・

・話下手でもコミュニケーション上手になる
・部下のやる気を引き出すことができる
・子どもの能力と可能性を引き出すことができる
・売上がどんどんあがる
・営業力が高まる
・やりたいことや夢が見つかる
・運命のパートナーが見つかる
・夫婦関係が劇的に良くなる

等、様々な成果があります。

【ファシリテーター紹介】
古川真理
ウェルスダイナミクス認定プラクティショナー
しつもん読書会ファシリテーター

不動産広告営業において、トップセールス、2度の月間MVP受賞など、数多くの実績を残す。そして、転職した国内大手証券会社においても、幾度となく部署内でダントツの営業実績を残し、半期に一度の役員賞を通算3度受賞。

しかし、それにもかかわらず、2017年3月に会社都合により解雇通知を言い渡される。一時は自殺を考えるほどの失意のどん底に陥るが、そこから立ち直り、資格取得の末に同業他社への転職に成功。とはいえ、会社員としての職務を果たしつつも、これまでのように会社だけに頼る生き方ではいけないという、強い危機感を抱くようになる。

そこで、人や環境や状況といった外的要因に振り回されない自分となるべく、ウエルスダイナミクス認定プラクティショナーとなる。ウエルスダイナミクスの理論を学ぶことにより、これまで学んできたどんなセミナーや能力開発のツールでも教えてくれなかった、「自分が持って生まれた才能」と「その活かし方」を知るようになる。

今では、会社員としては圧倒的な実績を残しながら解雇されたという経験を踏まえ、

「もう、私のように一生懸命頑張ったのにもかかわらず、路頭にさまよう人生を送る羽目になった人をなくしたい。」

という強い使命感のもと、自分自身が生まれ持った才能やその活かし方を提示し、会社に縛られることのない自身の価値の高め方を伝えるという活動に邁進している。
