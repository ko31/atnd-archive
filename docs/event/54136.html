第7回目の WordBech 長崎を開催します!

WordBench はWordPress ユーザーのための地域コミュニティです。WordBench 長崎では月1回程度で勉強会を行っています。
WordPress を勉強したい方、ユーザ同士で交流したい方、情報交換したい方、ぜひぜひお気軽にご参加ください。

毎月ネタ切れ気味なので何かやりたいという方、◯◯について話したいという方も大歓迎です。WordPress以外にもWebに関することをいろいろやっています。

今回の内容

今回はプラグイン・テーマを作って公開しようということで、各々で WordPress のプラグインやテーマを作って、あわよくば公式ディレクトリに申請するところまで持って行こうという内容です。基本的に参加者の方々がそれぞれ作業するもくもく勉強会的な感じになるかと思いますが、世話役のほうで色々サポートできると思いますので、勉強したい、誰かと一緒に開発したい、自分の作ったプラグイン・テーマを見てもらいたい、などなど興味のある方はぜひぜひご参加ください！
