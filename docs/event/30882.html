企画系！CEDEC

イベント ユーザー
event_id 30882
title 企画系！CEDEC
catch CEDEC2012最終日の夜に開催するゲームプランナーやゲームデザイナーなど企画系の方々向け非公式飲み会
description

集まれ！ゲームプランナー・ゲームデザイナー・レベルデザイナー・ディレクター、プロデューサー、シナリオライター、スクリプター、って多いよ！

この交流会は、CEDEC2012最終日の夜にゲームプランナーやゲームデザイナーなど企画系の方々が集まって、
交流を深める非公式の飲み会です。

例年、プログラマーやアーティスト、サウンドクリエイターの方々が集まる非公式な交流会が実施されてきました。
しかし、ゲームプランナーやゲームデザイナーなど企画系の方々が集まる単独の交流会はありませんでした。
企画系って、あまり集まって交流するイメージが無いのでしょうか？いえいえ、そんなことはありません。
むしろ企画系の方々ほど、積極的に集まって交流を深めていきたい人達はいません。

今年のCEDECで企画系の方々が何を見て何を感じたのか。
横浜でゲーム業界の未来を熱く語りあいましょう！

当日について

参加される方は、開始時間までに直接会場までお越しください。
CEDECの会場からは若干距離がありますので、ご注意ください。
万が一遅れそうな場合や、道に迷ってしまった場合などは、Twitterで主催者向け（ @uehara1974 ）にリプライを飛ばしてもらえれば対処致します。
それでは皆さま、当日をお楽しみに！

募集を締め切りました

会場の都合により、8/19 24:00 までに受け付けられた方までで募集を締め切らせていただきました。
参加をご希望の方は、「 TwiCEDEC/ゲームのお仕事2012 」さんが同じ会場で開催されますので、そちらの募集をご確認ください。

対象

企画系のお仕事をされている方、されていた方、したい方、興味がある方など、我こそは企画系と思う方はどなたでもwelcomeです！
CEDECに参加された方はもちろん、参加できなかった方も大歓迎！
プラットフォームも関係ありません。フリーランスの方も学生の方も、ぜひご参加ください！
自分が参加してもいいのかどうか…迷う方は幹事までご相談ください。多分大丈夫です(笑)

（職種の例）
企画、プランナー、ゲームデザイナー、レベルデザイナー、ゲームクリエイター、
ディレクター、プロデューサー、シナリオライター、スクリプター、など。

開催要項

今回は「 TwiCEDEC/ゲームのお仕事2012 」さんと会場をシェアさせていただく形式で開催します。
（「TwiCEDEC/ゲームのお仕事2012」に参加される方々と同じ会場になります）
※初回から厳密には単独開催ではありませんが…予想をはるかに超える人数が集まったら来年は頑張ります！
