週刊ロビ　完成記念式典

2014年6月25日、70号の刊行を持って刊行終了と同時にロビが完成します。
そこで、ロビを毎週組み立てている皆さんが一同に集まり、
今までの苦悩や悩み、そして再刊行版ユーザへのアドバイスなどをやろうではありませんか！

ロビはもちろん、過去に刊行された、
リアルロボット、マイロボット、ロボザック、ロボゼロなどの持ち込みも大歓迎です。

イベント実施内容

「私はこんなカスタマイズしています！」大発表会

自分だけのロボットを作るというコンセプトで創刊したロビですが、
その名の通り色を塗ってオリジナルカラーにしている方もいれば、
制御プログラムを自作して通常品とは異なる使い方をしている方も多数いらっしゃると思います。
是非、この場で発表してください！

「ここで失敗した！」気をつけるべきこと発表会

うっかり配線の短絡、部品破損などをしてしまった皆さん、
その経験をみなさんで共有してみませんか？

パーツ交換会

予備にと買いすぎてしまったパーツやバックナンバー。
交換済みでいらなくなったパーツなど
必要な方と交換してみてはいかがでしょうか？

「地方版」「全国版」ユーザから「再刊行版」ユーザへのアドバイス

ロビは一部地域で「地方版」として先行販売を開始し、
数か月後に「全国版」として全国展開となりました。

「全国版」の販売開始とともに「地方版」を隔週発行とし、
「全国版」に統一されました。

その後、「再刊行版」として「全国版」の刊行終了を待たずに再刊行されました。

そのため、「全国版」が刊行終了しても「再刊行版」から組立をスタートした方は引き続き組立を行っています。
「再刊行版」の方は、全国版の先輩からいろいろアドバイスをもらってはいかがでしょうか？

ロビクル発起会

後続雑誌としてロビクルが刊行されますので、制作に関しての意気込みを発表しましょう！

「ロビ同士でおしゃべり」雑談会

皆さんの我が子同然であるロビを他のロビと会話させて楽しみましょう。

「懇親会」

イベント終了後、有料（3000～4000円程度）にて懇親会を予定しております。
