【講演主旨】
　導電性ポリマーコンポジットの開発に当たって導電性フィラーのポリマー中での分散性制御は最も重要な課題です。本講演ではまず、導電性フィラーの分散性と関係が深いパーコレーション現象について紹介し次いで、低いパーコレーション閾値を得るフィラーの表面処理法、配合・加工法および分散性評価法について具体的に紹介します。第Ⅱ部の各論では、最近注目を集めているカーボン系ナノフィラーおよびπ共役系導電性高分子をフィラーとして用いたコンポジットのパーコレーション挙動の特徴について紹介します。

【プログラム】

第Ⅰ部　導電性コンポジットの導電機構
1.パーコレーション機構
　1.1　統計的パーコレーション
　1.2　熱力学的パーコレーション
　1.3　動的パーコレーション
2.導電性フィラーの分散性制御
　2.1　フィラーの表面処理
　2.2　フィラー選択的局在化
　2.3　ダブルパーコレーション
　2.4　2段階パーコレーション
3.画像統計解析法による分散状態の評価

第Ⅱ部　新しい導電性コンポジットの製造と特性
1.カーボン系ナノフィラーを配合した導電性コンポジット
　1.2　気相法炭素繊維(VGCF) 系
　1.2　カーボンナノチューブ 系
　1.3　グラフェン系
2.π共役系導電性高分子を配合した導電性コンポジット
