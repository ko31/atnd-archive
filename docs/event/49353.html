米国Microsoftで、4月2日より、BUILD 2014が開催されました！

BUILDの様子は、デベロッパー向けの動画配信サイト 「channel 9」 で配信されていますが、この映像をプロジェクターで見ながら、Microsoftの新しい技術ネタで一緒に盛り上がりましょう！！

  • AM10:00～PM5:00 まで。
  • デバイス（ノートやタブレットやフォーン的な何か）・昼食・おやつなどは各自持参。
  • 無線LANは提供されます。会場設備のPCの利用可否は当日次第です。
  • 基本、ストリーミング流しっぱなし・勝手ディスカッション・即興ハッカソン・オレオレLTありのゆるゆる進行です。某BarのノリでOK。
  • 参加費無料　（当初おやつ代500円としていましたが、おやつは準備しない事にしましたので、各自で用意お願いします。会場にはドリンク自販機はあります。また、近所にローソンがあります）。
  • 会場を名古屋ソフトウェアセンター様にご厚意でお借りしています。ゴミの回収や設備の取り扱いについてはご協力をお願いします。

BUILD軽いまとめ

  • Windows 8.1 Update1のリリース
  • Windows Phone 8.1の限定無償化
  • Windows 8/Windows Phone 8.1での設定ローミングなど
  • Windows Phone 8.1アップデートで、学習するアシスタント「Cortana」の追加
  • Visual Studio 2013 Update2 RCの公開、RTMへのアップグレードパスあり
  • .NET Framework Nativeコンパイラ
  • Internet Explorer 11にEnterprise Modeを追加
  • Microsoft Azure、PuppetやChefに対応
  • Microsoft Azure、新しいポータルインターフェイスに刷新。Visual Studio Onlineも統合へ
