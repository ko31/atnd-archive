概要

　Windows8アプリの開発ノウハウが学べるBootCampです！！
　実際にPCを利用したワークショップも行いますので、体験しながら学ぶことができます（PCを持参できない方も参加可能です）。

　日本マイクロソフトのエバンジェリスト大西さんが講師として参加します。Windows8について聞きたいことがある人もこの機会に色々聞いてください。

　Windows 8のアプリケーションを開発を開始するためのまたとない情報入手の機会です。この機会をお見逃しなく。

当日特典

　Windows ストアへの近道は、Application Excellence Labです。
Application Excellence Labの情報: http://www.facebook.com/5Metro#!/5Metro/app_384971174888643

　イベント当日に発表しますが、参加した人が得をする企画になっています。
