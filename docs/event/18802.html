2011/08/20に、仙台でiOSアプリの勉強会を開催します！

先月開催したSendai.iPhone #1 勉強会では、「開発者向けのディープな勉強会を！」というご意見をいただきましたので、そのような会を開催してみようと思います！

Sendai.iPhone 勉強会は、今後３ヶ月に一回を目安に行いますので、その合間合間に月一回でディープ勉強会を開催したいと思います。今回はその第一回目です。

内容

今回は、実際に開発を行っている方向けの内容を考えています。

タイムテーブル
  • @h_mori さん：facebook APIについて（1.5時間〜2.0時間）
  • @tototti：iOS5 でのtwitter連携機能について（? 時間）

現在決まっている内容では、TweetMe/SOICHAの @h_mori さんが、facebook API についてセッションを行ってくださります！さらに、 @yuuhic さんと、 @tototti のセッションもあるかもしれません。

とりあえず第一回目の開催ですので、時間の中で今後の内容も検討したいと思っております。やりながら内容を考える風になりますので、若干まったりゆるふわの進行になるかもしれません・・・。ご容赦くださいませ。

他に考えている内容案は、以下のような感じです。
・iOS5の新機能について
・テーマに即して継続的にライブコーディング

参加される方へ

・開催日が土曜日のため、会場ビルの入口が閉鎖されています。中に入れない場合は、フライトシステムコンサルティング 仙台事業所 022-225-4041 までお電話下さい。
・MacBookなどをお持ちの方は、ご持参下さい。（コーディングを行う時間があるかもしれません）
・Pocket-Wifiにてネット環境を用意しますが、接続数に限りがあります。emobile等をお持ちの方は、ご持参下さい。

懇親会

勉強会終了後に、懇親会を実施予定です。場所などの詳細については、当日会場で決定致します。
