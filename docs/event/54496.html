description

古座川町は紀伊半島の先端にほど近い山と川の町です。
日本の中でも世界の中でも雨の多い熊野地域に属し、その多雨が豊かな山と川とそこに住む人々・動植物たちを支えています。

また紀伊半島の南端に位置すること、半島の先を流れる暖流黒潮のおかげで比較的温暖な気候に恵まれています。

豊かな山・川、少し行けば海もあり、おまけに気候も温暖で雪もめったに積もることもなく、住み心地のよい地域かと思います。

当日は、短期滞在住宅見学、先輩移住者と地元の方々とのバーべーキュー交流会、農業体験、川遊び体験、ダッキーカヌー体験など内容の濃い1泊2日の現地体験会になっております！
