【プログラム】
◆講師の言葉
　最近、材料力学や構造設計の基礎的な知識が不足して起きる事故がめだっている。
　材料力学に関しては多くのテキストが出版されている。しかしなかなかうまく材料力学を実際の設計に活用できない。材料力学の知識が活用できない。活用するためには基礎をよく理解することが重要である。また、コンピュータを使えば何でも解いてくれると考えるのも間違いである。し　かしこれらは材料力学の基礎をよく理解すればよく見えてくる。
　本講座は今までのテキストと違った観点で材料力学の基礎と、また材料の破壊とつながる材料力学の基礎を基礎的な演習、実験も含めて学んでみよう。またこれまでのの種々の破壊事例から、材料力学の基本を考え、役に立つ、生きた知識として材料力学を分かりやすく学んでいただく講座です。またこれまでのの種々の破壊事例から、材料力学の基本を考え、役に立つ、生きた知識として、材料力学について実験と演習を交えて修得していただきます。

◆修得知識
・実際の設計で役に立つ材料力学と強度設計の基本

◆プログラム
Ⅰ.　はじめに
　　　1.材料力学を実際の設計に活用できない事について
　　　 　・コンピュータを使えば何でも解いてくれると考えるのは間違い　
　　　2.材料力学の基礎をよく理解すればよく見えてくる。
　　　　　a.今までのテキストと違った観点で学ぶための必要性について
　　　　　b.材料力学の基礎
　　　　　c.材料の破壊と関連した材料力学の基礎について　
　　　　　d.破壊事例から学べる生きた知識を学んで材料力学を整理してみる

Ⅱ. 材料力学の基本は力の流れ
　　　1.機械の部品、また構造物の部材のなかは力が流れている
　　　2.力の伝達状況を把握することが材料力学の基本
　　　3.部品はどこで、どのように固定されているのかを考える

Ⅲ. 応力の性質を理解する
　　　1.物体内の内力は応力と呼ばれこの応力状態を評価するのが材料力学の目的
　　　2.垂直応力とせん断応力は役目が異なる

Ⅳ. 材料の破壊は応力と材料の性質によって決まる。
　　　1.材料の破壊に関係する垂直応力とせん断応力の違いを理解する
　　　2.荷重の性質も破壊に関係する

Ⅴ. 軽量構造は棒、板、かくなどの薄肉部材が使われる
　　　1.薄板、細長い棒、かくには特有の問題が生じる
　　　2.応力集中や不安定問題が生じる。
　　　3.はり理論がよい答えを出す問題、はり理論が成立しない問題を実験する

Ⅵ. 高温機械では熱応力も深刻
　　　・熱応力は拘束が大きいと大きな応力になる。

Ⅶ. 回転機械の軸は曲げとねじりが生じる。
　　　・回転機械の軸はねじりのみでなく曲げ応力を考える。

Ⅷ. 機械・構造設計のための材料力学の4大基本原則
　　1.遊んでいる部分は無くそう
　　2.面内力(軸力)で荷重を支えよう
　　3.応力集中に気をつける、とくに継ぎ手部は問題が多い
　　4.公式を使いこなす

Ⅸ. 非線形現象も設計では必要になる
　　1.非線形材料力学も有効に活用する
　　2.座屈の大変形の意味を簡単な実験から学ぶ

Ⅹ. 事故例は材力の知識の宝庫
　　1・事故例から材料力学を学ぶ
　　2.機械・構造設計のための材料力学の4大基本原則を守っていないのが事故原因である
