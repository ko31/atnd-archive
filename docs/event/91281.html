MySQLのプロダクトマネージャーであるMorgan Tockerの来日に合わせて、MySQL最新情報セミナーを開催します！！

　
このセミナーでは、MySQLの最新情報をお伝えします。MySQLプロダクトマネージャーのMorgan Tockerによる講演などを通して、Oracle OpenWorld 2017で発表された最新情報をフィードバックします。

MySQLプロダクトマネージャーの話を直接聞ける貴重な機会ですので、是非ご参加下さい！！

Morgan Tockerについて

Morgan TockerはMySQLプロダクトマネージャーとして、MySQLサーバーのロードマップ策定や開発管理を担当しています。以前はOracleのMySQLコミュニティチームの一員として、MySQLコミュニティのイベントやセミナー、ブログでの情報発信、各種の交流機会を通じてMySQLの普及拡大とコミュニティの発展に貢献してきました。

講演概要

State of the Dolphin

  • Oracle OpenWorld 2017の基調講演であるState of the Dolphinで発表された最新情報をフィードバックします。

What’s new in MySQL 8.0

  • MySQL 8.0では、データディクショナリでのシステムメタデータの管理、日本語に適した照合順序の追加、Window関数の追加など、多くの新機能/改善点があります。このセッションでは、State of the Dolphinで伝えられなかったその他のMySQL 8.0の新機能/改善点についてご紹介します。

MySQL InnoDB Clusterによる高可用性の実現

  • MySQL Group Replication, MySQL Router, MySQL Shellを組み合わせ、高可用性とスケーリング機能を包括的に備えた単一製品として、使いやすいエンドツーエンドの統合ソリューションをMySQL InnoDB Clusterとして提供しています。本セッションでは、構築～障害発生時の挙動をデモを通して確認して頂きます。

MySQL Enterprise Edition紹介

  • MySQL Enterprise Edition(商用版)では、MySQL開発ベンダーからの強力なサポートが受けられるだけでなく、運用監視ツール、バックアップツール、セキュリティ強化ツールなど、MySQL Community Editionでは使えない追加機能も使え、TCO削減、セキュリティコンプライアンス対応などに役立ちます。本セッションでは、このMySQL Enterprise Editionの概要をご紹介します。
