★★★よくある質問と答え★★★

Q.パーティーの流れを教えてください。
A.受付・支払い→ドリンクを注文→フリータイム交流といった流れです♪
お見合いパーティーのような司会や進行はなく、参加者様とご自由に交流していただけます(^^)
↓こちらのURLに今までのイベント風景を載せていますので、ぜひご参考にしてください♪
http://ameblo.jp/ever-party/entry-11566194800.html

Q.初参加なので不安なんですが…。
A.パーティー参加者様の7～8割程度の方が初参加です。人数も多く初参加の方メインで開催しているので、内輪ノリ等は一切ありません！
皆さん少し緊張しているのは同じです。本当は話しかけたくても恥ずかしくて話しかけられずにいる方も多いので(特に女性の方に多いです)、気になる方がいたらぜひ自分から話しかけてみてくださいね(^^)v
また、会場にはスタッフがおりますので、「あの人と話したいんだけど…」なんてことがありましたら、お気軽にご相談ください♪

Q.一人でも参加できますか？
はい、できます♪お一人参加の方も初対面の方たちと楽しく交流されているので、ご安心ください(^_^)/

Q.どんな服装で行けばいいですか？
A.男性はスーツスタイルやシャツスタイル、女性はワンピースなどをおすすめしています(^^)

Q.どんな人が来てるんですか？
A.常識と良識のある20～30代の社会人の方がメインです。明るすぎる髪の人や過度な露出をしてる人・ギャル・ギャル男などは皆無ですので、ご安心ください☆

Q.料金はどうやって支払うのですか？
A.当日、現金でお支払ください。なるべくお釣りが出ないよう準備していただけると、お支払いがスムーズにできます♪

Q.会場に行くまでに道に迷ったらどうすればいいですか？
A.下記に記載している会場まで、直接お電話でお問い合わせください☆

Q.変な人はいませんか？
A.今までにそういったご報告を受けたことはございません。会場内にはスタッフがおりますので、しつこい勧誘行為や営業行為等何か困ったことがありましたらスタッフまでお申し出ください。
