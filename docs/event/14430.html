【プログラム】
◆講師の言葉
　放熱設計はご存知の通り電子機器にとって重要な要素です。最近ではとくに電子部品が多機能高密度化され、部品のレイアウトも高密度化が進んでいます。こ のような背景で、放熱設計については今まで十分冷却できていたものが冷やしきれなくなることが多くなっています。また、そのような課題への取り組み方も、 対症療法で運良く済ますことができなくなってきているのはお察しの通りです。ではいったいどうすべきでしょうか?開発のなるべく早い段階から放熱について 考慮しておくことが必要です。放熱設計が後手にまわりがちだ、思った通りにコミュニケーションがうまく行かない、そんな状況には理由があります。いったい 何が問題なのか、どうやったら改善できるのでしょうか?最初から熱くなることが無いように、初期の段階から放熱を開発プロセスに組み込む方法、その後の開 発渦中のトラブル対応や製造移管時にすべきことまで順を追ってご紹介します。 また、熱流体シミュレータは開発スケジュールにも大きなメリットを与えてくれるので、この効果的な活用方法も解説します。

◆修得知識
・放熱設計についての経験の有無を問わず、プロセスの視点から解説を行います。特定の放熱テクニックを修得するのとは違い、開発のフェーズごとに何をすべきかわかるようになるのが目的です。
