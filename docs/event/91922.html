バックエンドエンジニアクエスト：Bull Session

縁の下の力持ち、サーバーサイドエンジニアさんが集まって、
あるテーマについて、ざっくばらんに情報を共有し、
技術レベルや横のつながりをレベルアップするための会です。

▼ 2ダンジョン構成です

  • ダンジョン1：64の戦い方
  • ダンジョン2：ワークショップ

＜ダンジョン1＞：バックエンドエンジニア 64の戦い方

人知れず、障害や問題をなぎ倒していくことも多いと思います。

過去、どういう敵がいたか、どれくらいの壁だったかと、
それにどう立ち向かったか、結果どうだったかを、
1ファイト 10分ほどのライトニングトークで発表し合います。

今回は、5つ程度/64。
最終的に、64種の技が載っているバックエンドエンジニア冒険の書ができあがります。（といいなーw）

＜ダンジョン2＞：テクニカルワークショップ

力技で盛り上がった後は、業界のトレンドや注目の技術など、ひとつのテーマについて。
聞きたいことや話したいことがあれば、コメントまで！
何もなければ、どういう時間にするかなど、ネタのヒアリング時間にします。

▼ ご参加にあたって

  • 今までぶち当たった壁とどう壊したかを持ってきてください
    • 愚痴や自慢、悩み相談を兼ねてもOKです！！
    • 工夫したポイント、苦労したところ、などに焦点をあててください。
    • 問題の重量やテーマは問いません。（テクニカルではなく工程や私生活などでも可）
  • ワークショップ
    • テーマ選定中です。
    • 聞きたいことや話したいことがあれば、コメントまで！
    • 何もなければ、ネタのヒアリング時間にします。

▼ 参加条件

  • タダより安いものはない
  • たまには話したいけど最初の一歩が出づらい
  • 質疑応答の時間が短いイベントはもったいないと思う
  • 明日役に立つ話を聞きたい＆したい
  • 外の刺激が欲しい、井の中は飽きる
  • 横のつながりを広めたい
  • 盛大に愚痴りたい

この中で 2つ以上 当てはまる人は参加いただけます！

▽ Bull Session について

毎日への退屈を「たまに」どうにかしたくなりませんか？
セミナー等の参加モチベーションに「ムラ」がありませんか？
でも何とかしたい時がある。

そういう時だけ集まれて、次の日からちょっと元気になれる集まりができればと思い、
ざっくばらんに雑談という意味の Bull Session をやっています。

ライトな気持ちで、たまには脱線しまくったりして、
でもその場だけではない、次に繋がる会を続けれるよう、モデレーター頑張りますので、
お気軽に参加してもらえると嬉しいです。
