はじめに

エスノグラフィ、ユーザインタビューという手法を知っている人は多いと思います。
しかし、検証するための予算や時間が取れない、という理由により
実際の業務において使う機会は少ないのではないでしょうか。

やらないよりやった方がいいし
経験を積んでいた方が仕事に生かせるのならば
UX Fukuokaでやってしまおう、と考えて
サービス提供者のご協力を得て
体験型ワークショップを企画しました。

実際にリリース予定のサービスを題材としてエスノグラフィー・ユーザーインタビューを行います。

第1回では、ユーザーインタビューで使う質問をみんなで考えていきます。
第2回は、参加者みんながインタビューアーとなってユーザーインタビューを行います。

本やネットの知識の枠から一歩脱却したい方
是非、一緒にユーザーインタビューづくりしてみませんか？

専門の方はいませんので、みなさんで一緒に勉強していくスタイルです。

内容

第1回（9月24日）
(1)UXを検証するツールとしてのエスノグラフィ、ユーザインタビューについて軽くおさらいします。
(2)実際に手法を試すため、リリース予定のサービスを使って検証します。
　（題材提供者によるサービスの説明があります）
(3)ユーザーインタビューに使うための質問を、みんなで考えていきます。

第2回（未定。第1回参加者の多くが参加できる日程で調整）
(1)第1回で作成した質問集を使ってユーザーインタビューします。
(2) (1)で得られた情報と、サービス提供者側が想定していた
　ユーザーが求めるものの差や共通点などを検討します。
