リバくま交流会#3

～被災地での活動を支える情報ネットワークの有効活用を考える～

産官学NPO連携によるICT支援ボランティアチーム「リバイブくまもと」では、関係者間の情報交換による支援連携を目的とした交流会を継続的に開催しています。第3回目は、熊本地震で甚大な被害を受けた益城町で、現地での支援活動に取り組まれている団体や被災された地域の方々との意見交換や情報共有を行います。 
現地での復興に向けた様々な取り組みと課題についてご紹介しながら、ICTを活用した解決策や支援活動を中心に、今後の取り組み方を一緒に考えてみたいと思います。

当日は東京でもSkypeを通じて同時中継予定です！（ビューイング形式にて参加可能・参加費無料）

「リバイブくまもと」について
熊本地震の被災地では、今後、復興に向けた長期的な取り組みが必要となる中で、被災者の支援や被災地の復興に関わる組織や団体等におけるICTの利活用促進や、ICTを活用した地域防災への継続的な取り組みが課題となっています。 
今回、一連の地震発生を受けて、熊本県内を中心にICTを活用した地域活性化やまちづくり等の活動を行ってきた有志が集まり、様々な支援団体からの協力をいただきながら、熊本市の指定避難所へのタブレットPCの展開やOffice365による情報共有（くまもとRねっと）などを支援するITボランティア活動を行ってきました。 
私たちは、リバイブくまもとのスローガン「Revive KUMAMOTO with ICT ＝ ICTで元気な熊本を再興」を形にするため、避難所における情報共有の支援、被災者支援団体等へのリユースPC無償提供、ICTを活用した仮設住宅等での生活支援、地域防災情報共有システムの提供、ネットを活用した地域防災学習などの活動を通じて、ICTを活用した被災地の救援や長期的な復興支援に取り組んでいきます。
