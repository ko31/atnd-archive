description

習得できる知識
・アナログ・デジタル混在回路に於けるAD／DA変換回路の動作原理、設計技術の基礎を学ぶ。
・ΔΣ型AD／DA変換回路の動作原理を理解するとともに、効果的な活用のポイントを習得する。
・AD／DA変換回路の実装時のポイントを習得する。

プログラム
1. AD変換回路設計
　1.1．AD変換の動作原理
　　・標本化、量子化、符号化
　　・折り返し雑音、オーバーサンプリング技術、ΔΣ変調方式
　1.2．周辺回路の設計
　　・入力フィルタ回路、S/H回路、基準電圧源、基準クロック
2.　DA変換回路設計
　2.1．DA変換の動作原理
　　・再生化
　　・アパーチャ効果、ディザー、オーバーサンプリング技術、ΔΣ変調方式
　2.2．周辺回路の設計
　　・出力フィルタ回路、基準電圧源
3.　ΔΣ変調方式AD／DA変換回路
　3.1．ΔΣ変調方式の動作原理
　3.2．ΔΣ変調方式のメリットとデメリット
4.　AD／DA変換回路の実装時のポイント
　4.1．アナログ・デジタル混在回路に於ける設計時のポイント
　4.2．基板レイアウトのポイント
