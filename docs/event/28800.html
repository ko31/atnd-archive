description

大好きなジャンクフード、バックや引き出しにはチョコレート。
そんな自分を変えたおいしくて、バリエーション豊かなマクロビオティックのお料理との出会い。
「安心、安全、健康の食」＝自然食を知っているのと知らないのとでは大違いです。
生活習慣病やガンなどの疾病が増加し、自分自身もモチロンですが、このままでは子どもたちの将来、地球の未来は大丈夫でしょうか。
少しでも食、健康について興味ある方に、正しい知識を身につけて、賢くなって、本物選びができる目をもっていただくために聞いていただきたい講座です。

【美味しい自然食を始めましょう！（オーガニックワイン試飲付）】
1.今なぜ自然食がいいと言われているのか
2.日本の農業は今、、、
3.健康は食事からはじまる（薬膳、ローフード、マクロビオティック）
4.玄米ごはんは何がいいの？
5.自然食を社会の中で上手く、賢く取り入れるために
6.未来の子どもたちへ

□講師プロフィール

【講師】久保千尋氏
【プロフィール】オーガニックワイン専門店　マヴィ新高円寺店　店長。
岡山県出身。神戸女学院大学卒業後、大手コンサルティング会社で自然食をテーマに、自然食レストラン、カフェ、自然食品店、惣菜店、素材型和食店などの業態開発、出店開発、店舗売上げアップ支援を多数経験。現在は独立し、「自然食は地球と人を蘇生化する、これから大切になるライフスタイル」と考え、マクロビオティック、ローフード、薬膳、オーガニック食材の生産者支援、自然食レストラン・カフェの出店支援、メニュー開発、販促支援、人材育成といった、総合的な業績アップコンサルティングを展開している。また、生活者サイドからの「本物の食」への意識変革が必要との思いから、2009年からオーガニックワイン専門店＆食育サロン「マヴィ新高円寺」で自然食について学ぶ「おいしい楽しい自然食スクール　Food Paradaise35」や本物の食と自然食ライフに関するイベントなどを運営し、明るく、素敵に、おいしく自然食ライフを提案することをモットーに奮闘中。
