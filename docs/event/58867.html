開催にあたり

KDDIは、 webを通じて様々なモノが繋がっていくWoT（Web of things）の世界の実現に向けて、「Open Web Board」と「Gluin」のリリースを発表致しました。
（10/3発 プレスリリース ）

現在、IoT（Internet of things）の世界はウェアラブルデバイスや家電製品との連携サービスを初めとし、大きな広がりを見せています。 私たちは、この流れにオープンなwebの技術を加え、汎用的なデバイスとサービスの組み合わせを実現させる事で、新しい体験を生み出す事が出来ると信じております。

今回のイベントでは、まず「Open Web Board」「Gluin」の紹介をさせて頂き、KDDIの考えるWoT プログラミングを皆さまに体験して頂きます。そして、どのようなモノとモノ、モノとサービスを組み合わせると良いのか皆さまと共に考え、実機を用いて皆さまのアイデアを具体化するワークショップ（アイディアソン＆ラピッドプロトタイピング）を行います。

また、当日は参加された方にOpen Web Boardの配布を行います。
本環境を自宅に構築することで、皆さま自身でアイデアを磨き上げて頂くことができます。
皆さまのご参加をお待ちしております。

※11/1に東京で開催したイベントと同内容となります。
11/1にご参加頂いた方へのOpen Web Boardの再配布はできかねますのでご注意下さい。
