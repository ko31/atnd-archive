概要

今年の三月の東京リージョンの開設以降もその勢いは衰えることなく、着々と新サービスやサービスの改善を打ち出し続けている Amazon Web Services (以下、AWS)。
その AWS について、具体的な利用方法から設計、運用の考え方まで幅広く情報共有及び議論する場である JAWS-UG 京都勉強会が約半年の沈黙を破り (?!) 第二回目の勉強会を開催したいと思います。

今回は東京より Amazon Data Services Japan ソリューションアーキテクト片山さんをお迎えし、最新動向などをおうかがいするほか、実際の利用事例、LT 大会等、盛り沢山の内容でお送りしたいと思います。

  • AWSに興味があって、使ってみたい方、
  • AWSは触ってみたけれども、もっと深く知りたい方、
  • AWSをすでに使いこなしていて、その使い方を広めたい方、
  • AWS の設計方法について、ディスカッションしたい方、
  • JAWS-UG 京都の運営メンバーになられたい方、

などなど、是非お気軽にどしどしご参加ください。
