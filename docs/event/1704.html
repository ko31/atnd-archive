description

11月下旬または12月上旬ごろに予定している「Twitter 水曜どうでしょうサミット（ついどうサミット）」に先駆け、事前に小規模なオフ会を開催します！

オフ会の開催日は水曜どうでしょうの放送13周年記念日でもある10月９日（金）。

本来はこの記念すべき日にサミット本番を開催すればいいのですが、やはり平日ということでなかなか難しいかなという判断をしました。ですので、この10月９日には小規模なオフ会を開催します。

参加人数は何人になるかまったく不明ですが、話題が「水曜どうでしょう」ばかりになるでしょうし、人数が少なくても話は尽きないでしょう！！なお、オフ会は単なる飲み会ではなく、盛り上げるためのちょっとした準備もしておくつもり。

■参加資格
Twitterユーザーであること、「水曜どうでしょう」ファンであること

■日程・場所・予算
予算は3000円～4000円程度を想定。
普通の居酒屋でやります。
日時は10月９日（金）、時間は20時～22時
会場は新宿駅周辺の居酒屋
※詳細は後ほどご連絡します

■Twitter

さぁ、Twitterにいる「どうでしょう藩士」のみんな、どんどん参加してくれ！！
