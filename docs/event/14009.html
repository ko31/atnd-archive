description

青山ソウルオブソウルが送るチャリティーイベント！　４月２日（土）に、７回目を迎える、『こい☆こい』は、イベント収益金を東北地方太平洋沖地震の義援金として被災者の皆様への支援に活用されるよう収益の中から日本赤十字社を通じ寄付いたします。今回の内容は、水墨画の大壁画ライブパフォーマンスと即興音楽（インプロ）の裏打ちナシの本気即興コラボレーションアートワーク！！

そして、この完成した大壁画をバックにバーレスクダンスの饗宴と、まるでアッサンブラージュのような祭典です！

巨大水墨画実演は水墨画イラストレーターCHONTO（ちょんと）氏。即興演奏はNHK・TV「ふえはうたう」「趣味悠々」講師でお馴染みのリコーダー奏者の吉澤実氏、あのEXILEやMONKEY MAJIKもカヴァーする、ゴダイゴのベーシストでギタリスト吉澤洋治氏。
バーレスクダンサーには現役セクシーアイドルで、本年度のスカパー大賞最優秀新人女優賞を受賞したばかりの、羽田あいちゃんが登場！。

ハイレベルな即興の音楽と、それが下書ナシの水墨画でどのように具現化されるのか、そんなスリリングな即興を楽しみながら、再構築された、バーレスクダンスが、危うげな境界線上に広がる夢を演出します。

もちろん、今回もBlack Eyes TATTO による、本彫りLive TATTO に加え、実力派プロDJにYONOSUKE氏！

サポート陣は、ZIMA、Red Bull、による２本柱を筆頭に、バーレスクダンス衣装提供は、あのプレイボー社も絶賛するポーランド発のセクシーランジェリーブランド『オブセッシブ』のショーサポートが実現！！

ハイアートからロウブロウと見事にボーダレス化された" こい☆こい " ワールドを、是非とも体験してください。

今回も、オールナイト！朝までやってます！
