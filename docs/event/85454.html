メンタルヘルスを扱える人をもっともっと増やしたい。

最近元気ないように見えるあの人。心配だけど、どう接したらいいんだろう。何を言ったらいいの？そう迷っている間に会社で姿を見なくなってしまう…。そんな寂しい事態も、正しい知識で解消できるんです！

心の病が増える現代。 　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　ご自身のケアはもちろん、周りの方のメンタルヘルスを理解する事も必要とされます。 この機会に、一生もののメンタルヘルスの資格を取得し、多くの方の心に関わっていきませんか？

協会ではメンタルヘルスカウンセラー養成講座を東京、大阪、名古屋で定期開催していますが、更に全国に広げていく一環として福岡、札幌でも講座を開催いたします。

詳しくは当協会HPをご覧ください。
