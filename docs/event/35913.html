title 第53回「WEB TOUCH MEETING」やります！
catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション1(30分)
　『PHPとフレームワーク〜いまさらCodeigniter』大林さん

　＜主な内容＞
　　PHP って何？　から始まってフレームワークはどんなものか、
　　最後にPHPのフレームワーク Codeigniter の簡単な紹介までを
　　駆け足でお送りします。

　＜大林さんの簡単な紹介＞
　　2004年くらいからサーバを触りはじめ、
　　2007年からWEBの世界に入りました。

　　WEB でユーザーの目に触れない部分はだいたい守備範囲なので
　　インフラエンジニア、プログラマー、稀にディレクターなど
　　できることは頑張ってやってます。

　　専門は LAMP とネットワークなWEB全般の裏方まわり。

　

●セッション2(40分)
　【通販】３ヶ月で、過去最高売上を約7倍更新
　『売上拡大に向け、ウェブ解析士が取組んだ、仮説から検証までの実例紹介』
　広島ウェブ解析士認定講座運営事務局の岡山さん
　ナナイロデザインワークスの薬師神さん

　＜主な内容＞
　　昨年、ECサイト（広島）の売上アップの為に
　　広島のWeb解析士が集まり、プロジェクトを結成。
　　限られた予算と時間の中で売上拡大の施策を行い過去最高売上を更新しました。

　　Webで成果を拡大する事を検討する上で、
　　参考になる部分もあるのではないかと思い
　　実際に取り組んだ内容を発表させて頂きます。

　　■プロジェクト
　　　目的：通販サイトの売上アップ
　　　予算：数万円（制作費＋プロモーション）
　　　期間：2ヶ月間

　　■レジュメ
　　　1)ビジネスにおけるWebサイトのポジションとデータ分析
　　　2)Web解析士による通販サイト売上拡大に向けての実例
　　　3)まとめ（成功のポイント）

　＜薬師神さんの紹介＞
　　ナナイロデザインワークス：http://7dw.jp/
　　IA、デザイナー、ディレクター、上級Web解析士

　　2000年頃より趣味でweb制作をはじめて、東京でweb屋のお仕事してましたが、昨年広島に移転。
　　EC制作、ネットベンチャー、大手SIer、フリーと渡り歩いた中で
　　情報設計フェーズ、プロモーションでのディレクション経験が最も長いです。
　　広島で唯一「情報設計」を専門に取扱うIA（インフォメーションアーキテクト）としても活動中。

　

●セッション3(40分)
　『Windows8の復習と書籍を出す方法指南』システムフレンド西村さん

　＜主な内容＞
　　Windows8というOSの機能について再度確認しながら、
　　書籍出版するための道筋と、苦労話を紹介します。
　　今一部で話題の最新携帯電話Windows Phone8もチラ見せします。

　＜西村さんの紹介＞
　　広島でWindows Phone、Windows 8の開発勉強会をやっています。
　　EC-CUBE公式エバンジェリストもやってます。　

　　株式会社システムフレンド
　　http://www.systemfriend.co.jp/

　　Windows 8
　　http://windows.microsoft.com/ja-JP/windows/home
