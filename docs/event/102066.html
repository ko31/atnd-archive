【イベント概要】

SNS などで目にする「最近流行りのライブラリ」「人気急上昇のフレームワーク」、
気になっているけど実際に触る機会が無い…。どう手をつけていいか分からない…。
そんな人たちで集まって、みんなが気になるライブラリをハンズオン形式でつまみ食いするグループです。

第3回は TensorFlow.js で機械学習やっちゃいます！

発表者が用意した実際に動くサンプルをもとに、
ハンズオン形式で見ながら・触りながら旬の JS を気軽につまみ食いできるイベントです。
途中でつまづいたり疑問が出ても、その場で質問してサポートが受けられます。


【タイムスケジュール】

13:30〜14:00 受付開始

14:10～14:30 テーマ説明(TensorFlow.js のご紹介)
14:30～16:30 ハンズオン・あいまで質疑応答
16:30～16:50 参加者みんなでふり返り
17:00～18:30 懇親会

登壇者の発表を聞きながら実際に JavaScript を書いてもらうハンズオン形式で進行します。
各自プログラムを書き始められる状態の PC を持参のうえご参加ください。

ハンズオンの後に懇親会を行います！
軽くお酒を飲みながら興味のある技術やクリエイティブについて語り合いましょう！
