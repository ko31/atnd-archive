description

「超高速開発ツールで、短期間で業務クラウド化」をテーマに、安心・短期間・低コストでクラウドシステムへのリニューアルを成功させる手法について、 業務をさらに効率化するオプションサービスの紹介を交えてわかりやすくご紹介します。

◇主催企業：株式会社ソフトウェア・パートナ、株式会社CloudPayment

こんな方におすすめです

・業務改善のためのクラウドサービス導入を検討している方
・低コストかつ短期間で業務クラウド化を実現させたい方
・クラウドサービスを活用した業務効率化と生産性向上に関心のある方

セミナースケジュール

16:00～ 開演

16:00～16:45　第1部
経理のミカタとも容易に連携！企業成長に合わせてシステムも成長させる理想的な開発手法「超高速開発」とは？</>

超高速開発ツールWagbyを使用した顧客管理システムの概要と優位性を御紹介すると共に、
「経理のミカタ」と連携した「販売管理システム」の実例を御紹介いたします。

株式会社ソフトウェア・パートナー　営業企画部　滝澤好道

16:45~17:30　第3部
請求業務のオートメーション「経理のミカタ」による効果的なデータ活用を実現する業務の仕組み作り</>

Salesforceと連携した請求クラウド『経理のミカタ』による売上・売掛金管理業務の効率化をはじめ、経営データを活用した収益改善の具体的な方法について、 導入事例やデモンストレーションを交えてお話します。

株式会社Cloud Payment　営業推進本部　本部長　青木誠

17:30~18:00 希望者個別相談会
※業務の課題や導入に関する相談に講師が個別対応します。
