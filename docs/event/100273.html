9/22 ブレストバー「睡眠ブレスト」

description

ブレストバーは、身近なテーマでお酒を飲みながらブレインストーミング（ブレスト） を楽しむ。
もっともゆるく、もっともハードルの低いブレスト交流会を目指しています。

ブレストのルールは一つだけ。発言されたアイデアを否定しないこと。
トンチンカンでも、的外れでも、自由な発言を。

さぁ、みんなでゆるく盛り上がりましょう。

第二回目のテーマは
「睡眠ブレスト」

「最近寝つきが悪いんだよなぁ」
「3度の飯より睡眠が好き♡」

毎日とる睡眠。日常のことだから睡眠のことをじっくり考える機会って実は少ないのかも。

この機会に就寝前から睡眠、目覚めまでをブレストしてみませんか？

会場は小川町駅や淡路町駅から徒歩3〜5分の「神田グランピング」を予定しています。
グランピングの雰囲気でリラックスしながらブレストしましょう。

※お酒が飲めなくても参加OK。ソフトドリンクを用意しています。

◾️タイムスケジュール
・15:00（開場:14:30）
はじまりの挨拶は乾杯から
アイスブレイク（場がほぐれるお遊び）をしながら
軽くお酒（またはソフトドリンク）を引っ掛けましょう。
乾き物もご用意しています。

・15:30
「羊が1匹、羊が2匹、、、」
最初のブレストは、そろそろ羊の数を数えること以外を考えてみませんか？

・16:00
ここからはブレストラッシュです。
子テーマを複数用意しているので、選択式スタイルで進めます。
例えば…
就寝前〜
「寝付けない時に快眠につながるアイデア」
「お家にある道具を使って気持ちよく寝るアイデア」
「パートナーと良い睡眠をとる方法」
「旅行先や飛行機で快眠する方法」
「寝具メーカーが嫉妬。次世代の快眠アイテムを考えよう」

睡眠中〜
「見たい夢をみるアイデア」
「いびきがうるさい..どうにか気にならない方法は？」

目覚め〜
「目覚めたい時間に起きる」
「寝てる人を機嫌よく起こす起こし方」
「おはよう。以外の第一声を考えてみる」

昼寝〜
「オフィスに最適な昼寝スタイル」
「シエスタ制度の導入の仕方」

などなど。時間が許す限りブレストしていきましょう。

・17:00
今後のためのアンケートご記入後、お開きとなります。
