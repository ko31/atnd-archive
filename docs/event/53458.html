description

三河iOSアプリ開発の会は、iOSアプリをつくっている、または、これからつくりたい人たちの共通の学びの場、情報交換や人脈づくりのためのコミュニティです。

公開を前提としている高い意識の仲間が欲しい方、ずっとアプリを作りたいと思っていたけどなかなか時間を作れなかった方、実際にアプリを作り始めているけど行き詰っている方、アプリは作れるけどWebシステムとの連動に敷居を感じている方etc…情報交換などしませんか？

基本は独習で、それぞれ自分の開発を行います。開発環境は各自でご用意ください。
Xcodeだけでなく、Flash、Flash Builder、その他による開発も歓迎です。

会場にあるiPhoneアプリ作成の為の参考書籍のご利用は自由です。
自作アプリの宣伝、発表も歓迎します。教えることに興味がある人も大歓迎！

キッズITの円

（※iOSアプリ開発ではありません。キッズITの円のみのご参加でもOKです。）
教育用に開発された、子ども向けのプログラミングの"Scratch"の体験ができます。小学生高学年から参加できます。
マシンはこちらで用意したものをお使いください。（もしも足りない場合は順番で）
※キッズITの円にお子さまが参加の場合、参加されるお子さまの人数を入れて、コメント欄に「キッズITの円に参加する」ことを入れてください。
参加は自己責任でお願いしております。行き帰りの交通など、十分にお気をつけください。
