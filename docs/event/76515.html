日本Androidの会秋葉原支部ロボット部　第46回勉強会を開催します。

今回は今年初めての青空勉強会です。無限ピザを楽しみながらら各自の発表を行います。

開催概要

日時

2016/04/10(日) 10:30-16:00 ぐらい

場所

和田堀公園
https://www.tokyo-park.or.jp/park/format/map095.html
地図では、宮下橋の辺りのバス停から、橋渡って左折、河川工事の塀ぞいに進み、地図の「べ」の字の所、灰色四角の所です。
車なら駐車場あります。徒歩なら高円寺か永福町からバスが出てます。

場所取りの都合もありますが、公式バーベキュー所の奥側のブランコの更に奥、桜の木の下あたりで予定しています。

京王-井の頭線 永福町または西永福駅より徒歩10分ほど。

無限ピザパーティー

エアロジェルピザ窯

モバイルピザ窯DELiveの新バージョンで無限にピザを焼きます。

今までの断熱耐熱セラミックブロックを使った石窯に対し、1/10に軽量化したエアロジェル窯。今回は新開発のSUS/酸化珪素/SUS複合装甲を装備したアーマードエアロジェル窯を用います。

その他、七輪、コッヘルなどが持ち寄られます。

焼きたい物持ってきてね !!

青空勉強会

紙で資料を印刷して配布します。
または動作デモンストレーションを行います。

参加に必要なもの

・かかった費用を人数割（学生半額、小学生以下無料）にします。（下に別記）

・発表するもの

・食材は適当に買い出しに行きます。また、持ち寄れるものがありましたらエントリー時にコメントでお書きください。

プログラム

10:30 開場、買い出し、調理開始
11:30 乾杯オープニング
13:00〜 研究発表

なるべく全員発表しましょう。パソコン発表、紙発表 or 実演デモなど形式は自由。

今のところ以下の発表が決まっています。

  • 人工知能発表　日本Androidの会秋葉原支部ロボット部部長　清水
    • （１）自己意識の進化論的説明（ダマシオ説）　15分
      　原自己、中核自己、自伝的自己の進化論的な説明をする予定です。
      　神経学者アントニオ・ダマシオが主張している説です。
    • （２）TensorFlow をWindowsマシンで実行してみた　15分
      　つぎの環境でTensorFlowのチュートリアルを実行してみました。
      　・HW 4GB 3.4GHz 64bit Corei7(4コア,8スレッド)
      　・Windows7(32bit) 3.4GB!
      　・Oracle VM VirtualBox
      　・Ubuntu(64bit) CUIモード、2796MB!、8プロセッサ
      　・TensorFlow(Ubuntu/Linux 64-bit, CPU only)
      　メモリ不足で実行できないチュートリアルも有りますが、8スレッドで動きMacBookより２倍高速です。
      　MacBook 8GB 2.4 GHz 64bit Core 2 Duo(2CPU,2スレッド)
