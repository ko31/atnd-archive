主宰 ぞの。

「悩み」ってそもそも何？

悩みとはどのラインから悩みとして指定されるのか？
自分が悩みと感じることの共通点とは何でしょうか？
悩みと認定する基準は？着眼点は？
他者と自分を比較し悩みの条件や特長を知り、「悩みごと」そのものに悩んでみる会です。

歳を重ねるごとに「悩むことは面倒なもの」として避ける能力が自然と働きます。
その回避方法とは？あなたの悩みを具現化するとどんな形？
などのテーマで他者と共に考え談義し、悩み事だらけな人も、悩みを感じない人も、
「悩む」ことに「悩んで」これからも楽しんで「悩みと付き合ってほしい」なぁと思います。

※相談会ではありません。
※このイベントは悩むという行為自体の特長や傾向を認識して、「悩むこと」への理解をより深めるものです。
※参加しても、あなたが今もつ悩みが解決するわけではありません!!

■悩みごとナイトの内容■

１．お悩み傾向カルテ
　突然「あなたの悩みの定義を発表して！」と言われても答えられないですよね。
　まずはこの紙に傾向を書き出してみましょう。

２．お悩みを具現化して説明してください
　ここに四角、三角、まる、箱、やじるし、ひも等があります。
　あなたに悩みがうまれた時に見える心のイメージを説明してください。
　もしかしたら同じビジュアルの人がいるかもしれないし、一人もいないかもしれない。

３．悩みごとナイト
　ある程度意識共有したところで、「悩み」がどうやってできているのか考えていきます。
　・悩みの大中小
　・共通点
　・悩みと認めるとき
　・相談するときされるとき
　・悩み解決の決定打
　などなど、そのときの感じで内容増えたり減ったり変わります。

４．どうだった？
　ちょっと総評。

☆☆主催者コメント☆☆

私自身には『悩みが無いのが悩み』というほど、悩みのある生活とは縁が無いです。
私が「悩んでいる」という状態を指すときは、
「ある程度の答えと道筋があり、それをどのような方法で進めるか」を「選びかねてる」状態なのです。
なので、「悩みすぎてわからない」と言うひとの気持が私はわからない･･･。

答えがわかってるのに、なんで悩むの？
答えが分からないという状態になったことがない。
しかし、毎日なにかと思考を巡らした生活を送っているのですが、ふと、
『もしかしたら、この考え事は誰かの基準では“悩み”に相当するのではないか･･･？』
と感じてしまいました。
これは自分の基準だけではわからない！比較したい！他の人の悩みの基準が知りたい！！

と、突発的に人の思考を覗いてみたくなり、こんなイベントを開催することになりました。
これはアカデミックな賢いイベントではありません。
居酒屋よりは、ちょっとディープな話をするだけです。
なにかの偶然でこのページをごらんになったあなたでも、どなたでもご参加ください。

また、私にとって「相談する」というのは、選びかねてる選択を確信的に選択するため
「すでに分かっている答えをより質の高い答えに導きくため」の理由付けなので、人に相談しまくる人の心理もよくわからず、それも知りたい。
どこかで聞ければなと思っております。
