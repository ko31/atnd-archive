【ゲストトーク講演、LT発表者の皆様へ】

ご講演誠にありがとうございました。
発表資料をスライドシェアなどにアップロードされた際には、
是非、コメント欄にご記載をお願いたします。

JAWS-UGサイト(http://jaws-ug.jp)からリンク致します。


Amazon EC2やAmazon S3に代表されるAmazon Web Servicesは、
ユーザの要望に応じて機能がどんどん追加され、まさにかゆいところに手が届くという言葉が当てはまります。
第5回勉強会では、これら機能の詳細・利用方法や、様々な活用事例について紹介します！
