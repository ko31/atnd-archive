title 西海岸射撃ツアー
catch ラスベガスで銃を撃ちまくって、LAやシスコで遊ぶツアー
description

現地集合・現地解散ＯＫなツアーです。
シスコで半日観光した後にＬＡ軽油でレンタカー使ってラスベガスまで行くのも良し、
飛行機でラスベガスまで行くも良しです！

シスコで半日観光の場合はオイスターバーでシーフード食べたりしにいきます。
その後に飛行機でLAまで行ってからレンタカーを借りて移動します。
LAではチノの航空博物館、アイオワ戦艦博物館とかに行ったりします。
ラスベガスではカジノやシューティング、米国的伝統舞踊鑑賞したりします。

なお渡航期間は決定していません。
参加される方々の多数決で決めたいと思います。
現地集合・現地解散OKなので各々の好きなスケジュールでも行くも良しです！

また、2月21日～３月初旬までアリゾナに行きますので、それに合わせても問題ないです。
ただし、その時はLAよりもアリゾナメインの観光になります。
飛行可能なB17がある航空博物館、飛行機の墓場とか見に行けます。

なお、渡航費宿泊費は自腹でレンタカー代やガス代は割り勘で！
