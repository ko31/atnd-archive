概要

Microsoft Azureの日本データセンターが2月26日にオープンしました。それを記念して北陸でクラウド関連イベントを開催します！Microsoft Azureの概要から少しディープな内容まで。Microsoft Azureに興味がある方だけでなく、Amazon Web Servicesを使っている方々、開発者からインフラ担当者、学生さんまで幅広くご参加頂けます。お誘いあわせの上、奮ってご参加ください！

現在登壇者を募集しております！やってみたいと思う方は、@xin9le までご連絡下さい！

セッション内容

松崎 剛 (日本マイクロソフト株式会社)
はじめまして？日本マイクロソフトでエバンジェリストをしています。最近は、Windows Azure と周辺技術との連携系の話を主にしています。札幌に続き、久しぶりの北陸でワクワクしてます。よろしくお願いします。

Office, Exchange, SharePoint, Yammer, … – クラウド ＆ モバイル時代の最新の “アプリ” 開発テクニック
3 月に実施された SharePoint Confernece での発表をはじめ、Microsoft のプラットフォームを活用したクラウド アプリ開発は日々進化しています。本セッションでは、マイクロソフトにおけるさまざまなプラットフォームに対応したアプリ開発の最新情報を、Windows Azure と絡めつつ解説していきたいと思います。
芝村 達郎 (Microsoft MVP for ASP.NET/IIS)
最近は Windows Azure Web サイトを弄るのが好きな普通な子です。

明日からすぐに使える Windows Azure Web サイト開発のヒント
日本データセンターの開設でより速く、気軽に使えるようになった Windows Azure Web サイトですが、実はとてつもない拡張性を秘めています。
本セッションでは、今まで以上に Windows Azure Web サイトを使いこなすためのヒントをご紹介します。
木村 秀敬 (JAWS-UG 福井支部)
Hokuriku.NET の皆さんはじめまして。JAWS-UG (AWS User Group – Japan) の方から来ました。今日はお手柔らかにお願いします。

密林から見た碧い空
密林での暮らしに慣れた住民が、あるとき空を見上げた時に見たものとは……
このセッションは、密林系エンジニアの赤裸々ドキュメンタリーです
