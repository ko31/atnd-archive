概要

  • 桜満開の代々木公園で、酒を飲みながら花見や勉強をする会です
    • 花見をするのも、勉強をするのも自由です
    • 酒やつまみ、勉強道具は持参してください
    • 周りの花見をしている方々に迷惑をかけないようにしてください
  • こんな方にオススメです
    • 勉強したいけど、家だと勉強できない
    • 花見をしたいけど、一人で花見をするのはつらい
    • 休日だけど予定がない
  • 初参加の方も歓迎です
