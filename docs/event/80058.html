イベント・講演概要

超汎用2Dアニメーションツール 『OPTPiX SpriteStudio』 勉強会！

今回は、これまでのSpriteStudio勉強会とちょっと趣向を変えて、2Dアニメーションリソース制作の 受発注 がキーワード。実際のプロジェクトと照らし合わせながら、発注側の思惑と期待、受注側の覚悟と経験則を、それにツールベンダーの立場からお届けします。

「アニメーションの発注を依頼したいけどどういう仕様で？」

「受注するにしても責任の範囲は？」

「要望に合わせたツール側のカスタマイズって？」

などなど、SpriteStudioとその制作案件を取り巻く数々の事例、制作委託をする側もされる側も、知りたい情報が盛りだくさん！ また、当日はGTMFで配布して以降、大変好評な 『OPTPiX SpriteStudio ビギナーズガイド』を配布 いたします。

更に！　上記テーマに加えて今回は、ブラウザ上で動作するゲームエンジン、<a href = “https://playcanvas.com/” target = "_blank">『PlayCanvas』について、PlayCanvas運営事務局さんからお話頂きます。

全セッション終了後、懇親会＆名刺交換会を実施いたします。

参加対象

・2Dアニメーション・デザイン制作の 発注でお悩みの方
・2Dアニメーション・デザイン制作の 受注/営業でお悩みの方
・ゲーム業界（特に デザイナー・アーティスト・テクニカルアーティストの方
