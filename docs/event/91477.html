description

～世界中で“知的ゲーム”として広く認知されているポーカーの一種である「テキサスホールデム」のNo1企業を決める大会を開催します。～



ポーカーの一種である“テキサスホールデム”には、「リスクに対して得られるリターンを、感情に流されず、冷静に判断出来るマインド」が必要とされており、このマインドは企業経営にも通じるところがあります。


マイクロソフトのビルゲイツや投資家のウォーレン・バフェットなど、ポーカーで企業経営のマインドを学んだという経営者は多数存在します。


各企業における選抜メンバーによってテキサスホールデムの強さを争い、テキサスホールデムのNo1企業を決定したいと思います。


また、本イベントを異なる業種の方々と交流する場としても利用して頂き、新たなビジネスチャンスを見つける機会としてご活用いただければと思います。
