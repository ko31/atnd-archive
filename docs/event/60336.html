年末だから飲もう

好評だった xtcup #01 xtcup #02 に続くイベントです。

似た名前に xtnight ってのがあって、今回も退職したし似たような状況だったんだけど、"night" は最近のエンジニア界隈では違う使われ方をしているので、まぁ単なる小規模飲み会には cup つけておこうかという感じ。

今回の場所も江古田

西武池袋線江古田駅 の周辺です。

会場は

  • 仲屋

を予定しています。

もし店が休みだったり満員だったりで入れなかったら、キャパシティのある

  • わさび

に行く予定です。主催者 ( @xtetsuji ) はなるべく20時前に行って、仲屋の様子をツイートします。どちらも当日は営業しているはず。

西武池袋線江古田駅方面への行き方はいくつかあります。

  • 中央線ユーザは、中野駅北口から関東バスの「中12」「中41」に乗って終点まで向かうとよさそう (6分おきくらいに発車していて、かかる時間は20分くらいです)
  • 山手線ユーザは、池袋駅から西武池袋線の各駅停車に乗って数駅です (豊島園ゆきでもOK)
  • 副都心線ユーザは、小竹向原→練馬→江古田の切り返しが意外に時間がかかるので、雑司が谷駅で降りて、駅の3番出口から出て、高田一丁目バス停で都営バスの「白61」に乗って江古田二股バス停まで行くのが良さそう (10分おきくらいにはきます)

仲屋への行き方は

  • 西武池袋線江古田駅の南口に出る
  • そこから左側(東側)に歩く
  • すき家と江古田コンパの間の路地に入る
  • 仲屋の入口は相当狭いので、右側に注意しながら歩く

です。

冒頭の写真が仲屋入口だけど、暗くてわからない場合は黄色いでかでかとした看板のラーメン屋の手前くらいだと思ってもらえれば行き過ぎることはないはず。

よくある格安居酒屋とかではないので、大人の飲み方食べ方でいきましょう。
