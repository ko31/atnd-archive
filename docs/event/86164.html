プラグインの詳細な解説と、開発のデモをお見せします！

スクウェア・エニックス、Gungho、SEGA NETWORKS、ワンダープラネットなど、数多くのパブリッシャーさまに採用されている、Unityとの相性も抜群のネットワークエンジンである「Photon」。

今回は、Photonの新機能であるプラグインについて、その仕組みと開発方法を伝授します。プラグインを使うことにより、手軽にサーバーサイドへのゲームロジック組み込みが可能になります。
説明だけではなく、開発を始めるまでの方法を実際にお見せします。また、PCをお持ちいただければ、デモと合わせて皆様ご自身でもプラグイン開発を体験できます。これであなたもすぐにプラグイン開発ができちゃいます！！

また大きな可能性を秘めているEnterprise Cloudについてもお話していきます。Enterprise Cloudではプラグインの利用も可能となっており、クラウドサービスで手軽に運用をしながらプラグインでサーバーサイドのロジックカスタマイズも可能です。Photon Enterprise Cloud + プラグインの可能性をぜひ感じてください。
