◆概要

ソーシャルアプリ勉強会第2回！

大人気のゲームアプリ「ぼくのレストラン」をご存知ですか？
国内初の、位置情報と全国の飲食店情報とが連動したケータイゲームとして2009年10月28日にスタートし、更にmixiアプリモバイル版の公開を行った11月26日からわずか13日で、なんと登録数50万人を突破した、今大注目のアプリです。

第2回のソーシャルアプリ勉強会では、そんな「ぼくのレストラン」を企画・開発した公文氏を講師に迎え、技術者に向けた、ソーシャルアプリ開発のリアルなお話をうかがいます。

  • socialグラフ→アクティビティ→ソーシャルプラットフォームと進化してきたSNSの変化の振り返り。
  • みんなに使ってもらえるソーシャルアプリケーションの作り方
  • 「ぼくのレストラン」の開発・運用・運営の中で学んだこと、苦労していることなど
  • 大規模サービスならではのエピソード

などなど、
「たくさんの人に使ってもらえる、人気のアプリを作るにはどうしたらいいのか？」
「SNSならではの特徴をいかしたソーシャルアプリって？」
と悩んでいる人には必ず役立つ内容です。

◇公文善之氏　略歴

青山学院大学大学院を1999年に卒業し、携帯電話向けのwebサービスを行うベンチャー企業のピー・アイ・エム株式会社の設立に技術担当の取締役として参加。2000年には、同社とヤフー株式会社が合併となり、ヤフー株式会社に入社。モバイルサービス、教育関連のサービス、地域サービス、オークションと様々なサービスに技術・企画として参加。2009年の4月に同社を退職、6月に株式会社Synphonieを設立し、同社代表取締役に就任し今日に至る。
