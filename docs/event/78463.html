「介護」をキーワードにした、地域交流イベントです。
介護職の方だけでなく、医療・看護関係・福祉関係の方、家族の介護に関わっている方、また今は介護をしていないけれど将来の介護が何となく心配…といった方まで、多様な方が自由に情報・意見交換する場を提供しています。

だれもが介護を受けたり、おこなったりする可能性が高い時代。
このカフェイベントは、日常生活におけるちょっとした悩みや不安、疑問も気軽に解消できるような場を目指しています。

普段の仕事で悩んでいる介護職の方、業界の新しい情報に触れたい方、家族の介護についての相談をしたい方など、「介護」というキーワードに興味を持たれた方なら、どなたでもご参加いただけます。
参加希望の方は、下記ページの申し込みフォームからどうぞお気軽にお越しください。
http://www.sydecas.jp/contents/regional-life-design/care-friends-cafe/

◆参加対象◆
介護職・介護関連職の方、医療・看護・福祉関係の方、家族を介護中の方、将来の介護に何となく不安を抱えている方など、お気軽に！

◆参加費◆
各回500円。飲み物付き。「みんなでシェアできる一品（お菓子でも、お惣菜でも！）」をお持ち寄りください。

◆注意◆
名刺交換などはご自由に行っていただけますが、強引な営業活動、マルチ商法・宗教等の勧誘活動は固くお断りいたします。
場合によっては主催者判断にて退去・出入禁止とさせていただきます。
皆さんが気持ちよく過ごせる場を提供するため、ご協力ください。
業界のお話など、話題提供は大歓迎です。
