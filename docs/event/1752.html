description

2008年に11月に開催した「第一回ジオメディアサミット関西」では、「ジオメディアの熱気を関西へ」と言うことで、（東京での）ジオメディアサミットの熱気をそのまま関西へ持ってこようというテーマで開催しました。
今年は「ジオメディアサミット西日本」と改題し、関西圏に限定せず、西日本でユニークなジオメディアの取組みをされている方々にお越し頂き、西日本から発信するジオメディアサミットを作ります。東京から足を運んで頂いても十分に価値のあるイベントです。

形式としては昨年同様、ライトニングトークがメインの構成となります。西日本のユニークな「ジオメディア」の取組みを、スピード感一杯に紹介していきます。
