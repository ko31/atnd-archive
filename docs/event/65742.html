ゲームデベロッパー9社と大学・専門学校が、渋谷ヒカリエに集うゲーム業界一大就活イベントを2014年5月16日（土）に開催します。
各社・各学校が、ブースを設け個性をアピールするだけではなく、各社の個別説明会はもちろんのこと、 数社の社長たちによるトークセッション、各社のデザイナーによるポートフォリオ講評会を実施いたします。
各社ブースや、別途会議スペースを使っての面接/面談も実施されます。
ゲーム業界を目指す学生さんは必見のイベントです！

◆参加エントリーはこちら◆http://urx2.nu/fQYS
※事前エントリー制ですので、必ずお申込みください。

◆開催イベント
【1】社長トーク（17:40-19:00）
各社の社長が、ゲーム業界に、必要とされる人材と育成方法についてトークセッションいたします。
登壇者：アールフォース・エンターテインメント 横山裕一社長、ヴァンガード 杉山智則社長、リズ 磯野貴志社長、ランド・ホー 塚本昌信社長
モデレータ：DeNA 馬場保仁プロデューサ兼採用担当
【2】企業説明会（13:10-17:10）
各社の説明会を30分間実施します。
【3】ポートフォリオ講評会（13:10-17:20）
デザイナー志望の学生向けに、お持ちいただいたポートフォリオを各社の現場デザイナーがコメントをします。
【4】企業・学校ブース（13:00-19:00）
企業、学校がブースを設けて、展示、説明を実施します。

◆日程：2015年5月16日（土）13:00〜19:00（途中入退場可/最終入場18:00まで）入場受付：12:30から

◆参加予定企業（ブースあり）：アールフォース・エンターテインメント / アクセスゲームズ / ヴァンガード / ヘキサドライブ / マトリックス / モノビット / ランド・ホー/リズ / ワンダープラネット / DeNA

◆参加資格：大学生、院生、専門学校生（学年は不問）、ゲーム業界、企業関係者さま
