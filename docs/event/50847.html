やる気に関係なく自然と勉強できる短期合格らくらく勉強法セミナー

title やる気に関係なく自然と勉強できる短期合格らくらく勉強法セミナー
catch 働きながら９ヶ月で社会保険労務士試験に一発合格した本質的な学習メソッドをたったの２時間で学べます
description

あなたはこんなお悩みを抱えていませんか？

「残業続きで疲れてしまい、勉強しようと思っているのになかなか勉強が進まない、、」
「１日２時間勉強する！と紙に書いて壁に貼っているのに、続けられない、、、」
「不合格続きで、モチベーションがなかなか上がらず、毎日勉強しなくちゃって思うけど、集中できない、、」
「休日は、家事や育児に時間を取られてしまい、思うように勉強時間が取れない」
「勉強しようと思っているけれど、勉強以外のことをついしてしまう・・・」
「成績が思うように上がらず、焦ってばかり。自分の勉強法は正しいのだろうか、、」

はじめまして。横澤有季と申します。
私は難関資格合格に関する勉強法の個別アドバイスを行っております。
私のもとには勉強に関する数多くの悩み、相談が寄せられます。

中でも「モチベーションを上げるにはどうすれば良いですか？」というご相談を非常に多くいただきます。

多くの方は、勉強を続けるためにモチベーションを上げることが大事だと思っています。
あなたもそう思っていませんか？

でも、モチベーションを上げることよりも、資格合格に直結するもっと重要な事があります。
そもそも「モチベーションを上げること」、それ自体に個人差が大きく、定着させることが難しいのです。
なぜなら、モチベーションは「成績が上がってきた」など、結果が出てきて初めて上がるものだからです。

資格合格において本当に重要なこと、、、
それは・・・

モチベーションに頼らずとも、淡々と勉強を続けることができる仕組みを学び、
正しい仕組みを習慣化していくことです。
それが難関資格試験に合格するためのたった一つの秘訣なのです。

この秘訣に気づいたのは、私が資格試験を勉強していた時に遡ります。
私は社会保険労務士の勉強をしていた時に資格の学校で行われたテストの成績がなかなか上がりませんでした。そのため、モチベーションも上がらずダメダメ受験生で、苦戦した経験がありました。

「過去問を解いても、やたらと時間がかかってしまうし、なかなか勉強が進まない・・・。」
「仕事も忙しいし、家事もしないといけない・・・。」

この暗いトンネルから抜け出せる日が本当に来るのかな？と、毎日落ち込んでばかりで、途方に暮れていました。

そのような状況の中で、藁をもつかむ思いで、必死に合格体験記や勉強本を読みあさり一発合格した人の勉強法を学びはじめました。他には、ある資格合格の達人に出会い、勉強法をヒアリングし、スキルとして落とし込んだりし、やれることは全て取り入れました。
その中で、モチベーションに関係なく自動で勉強ができる勉強法を編み出しました。
どんなに仕事が忙しくても、家事が忙しくてもスイスイ勉強できるようになりました！
そしたら、今まで勉強していた知識が組み合わさり、一気に成績も伸びたのです。
そして、９ヶ月の勉強で社労士試験に一発合格することができました！

たった１つの勉強法を確立するだけで、資格勉強がこんなに簡単になるなんて・・・。
その想いをより多くの方に伝えるためにコーチングを学び、
受験経験とコーチングの手法を取り入れた独自のメソッドを確立しました。
その独自メソッドを、資格勉強でモチベーションが下がっている方にアドバイスしたところ、
自然と勉強できる仕組みを構築し、最後まで全力で勉強することができました。
結果的にその方も１回で社会保険労務士試験に合格することができました。

再現性のある私の独自メソッドをより多くの方にお伝えし、目標達成のサポートをしたい・・・。
そのために、このセミナーを企画しています。
このセミナーを受講することであなたは一早く自然と勉強できるようになり、
合格に近づけるようになります。

【こんな人にお勧めです】
　・残業続きや家事で、資格試験の勉強時間がなかなか取れない方
　・不合格続きで、「私は受からないのではないか？」と思っている方
　・「絶対に合格したい！」と思っているけれど、今の勉強法で本当に良いのか分からない方
　・勉強している割には成績が伸びず、いつも焦っている方
・合格すると周りに言っているけれど、密かに自分は合格しないのではないかと内心思っている方
　・モチベーションが上がらず、資格試験の勉強が続かない方
　・今年の「資格試験に絶対に合格する！」と決意していて勉強法が知りたい方
　・短期で一発合格したいと思っていて勉強法が知りたい方

【対象の資格試験】
　社会保険労務士、行政書士、中小企業診断士、税理士、公認会計士、司法書士、司法試験、
　不動産鑑定士、宅地建物取引主任者、FP、TOEIC、英検などその他の難関資格試験

【講座内容】
　・モチベーションに頼らずに自然と勉強ができてしまうたった一つの秘訣とは？
　・不合格になってしまう落とし穴とは？
　・合格するために必要な勉強法７つのステップ
　・独自の合格チェックリストで、あなたの現状の学習状況を徹底的にチェック！
　　どこを改善すれば、合格に近づけるのかを知る。

　

【講座で得られるもの】
　・残業続きで勉強時間ができないと思っていた方が、勉強時間の捻出ができるようになり、
　　毎日勉強が続くようになります。
　・残業も家事も勉強も、１日の中でやることが沢山あり、どれも中途半端でパニックになりかけていた方
　　が、今やっていることに集中できるようになります。
　・正しい勉強法を探していた方が、このメソッドを学ぶことで、他の学習法を学ぶ必要がなくなり、
　　エネルギーを集中させることができます。
　・自信を持ってスイスイ勉強できるようになります。
　・自分のどこにつまずいているのかが分かり、次に取る行動が分かるようになります。

【講師紹介】
　難関資格専門　時短学習法アドバイザー＆合格引寄人：横澤有季
　大学卒業後、１３年間にわたり、政府系金融機関、民間金融機関にて、役員秘書、総務、人事に所属し、
　労務関係、教育研修、採用などの業務に携わる。
　２００９年に社会保険労務士試験を受験し、なかなか成績が上がらずに苦戦しながらも学習法を身に付け
　たことで、９ヶ月間の勉強で１回で合格をする。
　その後、コーチングを学んだことにより、社会保険労務士の受験経験とコーチングを掛け合わせた独自の
　メソッド「時短学習法とモチベーションらくらく管理メソッド」を確立する。
　社会保険労務士を初めて目指す方や不合格だった方から合格のための秘訣を教えて欲しいと聞かれるよう
　になり、その秘訣を伝えて、学習方法のサポートしたところ、その方も社会保険労務士試験に１回で合格
　する。２０１４年に難関資格試験の学習法アドバイザーとして独立し、数多くの方に独自のメソッドを伝
　えている。
