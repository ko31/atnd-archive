■最終的に雌雄を決するのはメンタル力です。

メンタル力は、様々な修羅場を乗り越え、百戦錬磨によって鍛えられると、多くの方は勘違いしています。

百戦錬磨によって得た自信は、試合で優勢なときは、得点が進みますが、一度劣勢になると挽回が出来ません。

雌雄を決するのは、最後まで自分を信じ切る自信です。
最後まで自分を信じる自信があれば、最後の土壇場で挽回できます。

この、最後まで自分を信じる力は、自己肯定感と言います。

■ここに、自己肯定感を上げる特別な方法があります。

ソニー（株）の創立者井深大氏の命令で研究した「潜在意識活用法」を用いると下記が可能となります。

・成功者となることを妨げる「心の癖」を発見し改善出来る
・様々なメンタルブロックを外すことが出来る
・あなたの持つ諸能力を高めることが出来る
・意識の力を自在に駆使することが出来る


これでメンタル力を上げ、人生の成功者になれます。

■今回は、自己催眠状態になって、潜在意識の中を探っていきます。
そして、内なる叡智に繋がり、人生の最終ゴールを設定します。

自分で人生を決められると分かると、最後まで自分を信じきる自己肯定感が高くなります。


■セミナーで得られるもの

今回のセミナーでは、実習と共に、自己肯定感を高めることができます。
更には、自宅でも繋がれるよう、誘導音声の聞けるメンバーサイトに1ヶ月間ご招待します。

この回は、８，内なる叡智に繋がり、人生の最終ゴールを設定する　です。

■あなたは、生まれてから死ぬまでの、自分の人生をしっかり見つめたことがありますか？

通常の意識状態では、既成概念が邪魔をして、真正面か見ることができないばかりか、雑念がわいてきて、すぐに気が散ってしまいます。

しかし、催眠の意識状態である「変性意識状態」になると、集中力、洞察力、ひらめき力が高まります。
さらに、イメージ力も高まってくるので、人生の様々な場面をありありとイメージし、真正面から観察することができます。


■自己催眠状態で、あなたの「人生での体験」を観察する

これまでの人生で出会った重要な人々、
これからの人生で出会う重要な人々、
それら、一人一人と出会う意味が分かります。


■自己催眠状態で、あなたの「人生の気づきや学び」を観察する

人生での様々な体験を感じ取り直すと、そこから様々な気づきや、学びを見いだせます。


■自己催眠状態で、あなたの「人生の目的と、その実行方法」を見いだす。

人生での体験や、気づきや学びを元に探っていくと、人生の目的が分かります。
そして、内なる英知とつながり、人生の目的のより良い実行方法が分かります。


■自己催眠状態で、「人生の最終ゴール」を設定する

あなたは、人生の最終ゴールをどのように設定していますか？

ビジネスで成功して、○○億円企業を複数持つオーナーになっている。
○○に貢献して、○○賞を受賞している。
幸せな家庭を持ち、幸せな人生を全うする。
・・・・・・・

人生の目的とその実行方法が分かると、自分自身の最終ゴールまでの道筋がより具体的になり、の設定にも自信と確信を持てます。



■最近は多くの医師達が自己の専門分野から実体験した、魂の存在を肯定する本を出版しています。

東大医学部教授の矢作直樹氏の「人は死なない」

魂のビジョンから、人生を見つめるとさらに多くのことに気づきます。
