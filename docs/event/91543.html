【講座概要】

hands-on形式でサーバを2台構築し、負荷分散させるところまで。

【講座対象者】

・サブネットマスク等の基本的なネットワーク知識を有している方
・AWSを未習得の方

【事前準備】

1)AWSアカウントのログイン、AWSコンソール画面が表示できるところまで
以下の手順でAWSを利用できます。

《手順》
①AWSサイトへアクセス( https://aws.amazon.com/jp/ )
②メールアドレス入力
③ログイン認証情報入力
④連絡先情報入力
⑤クレジットカード番号入力
⑥電話による本人確認
⑦サポートプランの選択
⑧メール送信
⑨ログイン確認(ログイン後、AWSコンソール画面が表示される)

2)事前に必要な環境

①WindowsPC/Macともに必要なもの
Cyberduck(ファイル転送ツール)
Chrome
②WindowsPCの場合
・TeraTerm

【ご注意点】

事前検証は全てMacで実施しているため、 WindowsPCの動作環境は担保できないところがあります。
勉強会会場は無線LAN配備であることを想定

【参加者持ち物】

・名刺　２枚（お持ちの方のみ 1枚は名札用）
・ノートPC（上記、必要環境を設定済みのもの）
