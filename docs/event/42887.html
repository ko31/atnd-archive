Androidアプリを作って見ませんか？
本セミナーは未経験者や初心者の方でもAndroidアプリ開発を体験することができます。初心者の方でもベテラン講師がアプリ作成をサポートするのでご安心ください。

スマートフォンアプリ市場は、国内外において最も成長している分野です。このビジネスチャンスを逃さず、最大限に生かしていくためにアプリ開発に関する知識やスキル習得は必須のものとなっています。

スマホアプリを作る楽しさを体感することで"やってみよう””試してみよう””自分でも学んでみよう”といった最初の一歩を踏み出しましょう。

■===無料体験参加条件===
無料体験コースに参加される方は開発機器をご持参頂く必要があります。
※見学だけを希望の方は開発機器は必要ありません。

＜iPhone無料体験コースの場合＞
下記の開発機器を御持参頂くことが条件になります。
・Mac（MacBook AirまたはMacBook Pro、Mac OS X 10.7.3以上）
・【任意】iOS端末（iPhoneまたはiPadまたはiPod touch）

＜Android無料体験コースの場合＞
下記の開発機器を御持参頂くことが条件になります。
・Mac（MacBook AirまたはMacBook Pro）またはノートPC
（Mac OS X 10.7.3以上、Windows 7 64bit メモリ2GB以上必要）
・【任意】Android端末（iPhoneまたはiPadまたはiPod touch）
