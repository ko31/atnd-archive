description

経営者が著者として電子書籍を出版し、セルフブランディングし、売上に結びつける。 そんな、電子書籍を使ったマーケティング戦略が実績を上げ、注目されています。 本セミナーでは、Kindleで知名度ゼロから19部門ジャンル1位のベストセラーを獲得し、顧客から選ばれ、売上アップを実現し、個人起業家、経営コンサルタント、電⼦書籍プロデューサーとして数々の実績を持つ、前原泰子氏を講師にお招きしました。以下、前原泰子氏からのメッセージです。

「頑張ればいつかきっと報われる。」 そう勘違いし、方向性を見失い、 悩み続けた時代がありました。 が、ある信頼できる先生との出会いから、成功には近道があることを知り、やり方やる順番が大事だということもわかりました。 そして私のビジネスが大きく拡大し安定して売上をあげることができるようになったのは、電子書籍を出版し19部門1位になったからです。 同業者に埋もれ知名度ゼロだった私が、お客様にお願いされながら高単価商品が売れるようになりました。 いま、よい商品を持って頑張っている起業家やこれから起業を志しているあなたには、 私のように頑張り方を間違って、遠回りして欲しくないのです。 あなたが最短で成功する秘訣「電子書籍ランキング1位×売れる仕組みづくり」をお伝えします。

かつては、書籍を出版することはハードルが高く、自費出版でも数百万の費用がかかることもありました。 Kindleなど電子書籍が普及したいま、非常に低コストで出版ができ、誰もが自分の持つ専門知識や考えをまとめ、伝え、世に問うことができるようになりました。 このように変容した「出版物」としての電子書籍を、ビジネスを拡大させるツールとして活用し、成果をあげたノウハウを共有する場として、このセミナーを開催します。 電子書籍の企画の立て方から書き方、Amazonへの公開の仕方、公開後のフォローアップまでをともに学びます。 出版がより身近になり、ビジネスのツールとしてより手軽に効果を手にできることを学ぶことができたらと思います。
