名古屋でも「第13回UE4ぷちコン」ゲームジャム

現在、株式会社ヒストリア様主催による、Unreal Engine 4作品コンテスト 「第13回UE4ぷちコン」が
2月14日（金）～3月29日（日）の期間で開催されています。

このコンテストに合わせて 東京でゲームジャムが行われますが、
名古屋でもあればいいな、ということで今回も名古屋でゲームジャムを開催いたします。

自主開催のゲームジャムですのでぷちコンゲームジャム名古屋への問い合わせ等は
このページのメッセージにてお願いいたします。

・今回のゲームジャムの目的は「ぷちコンに出すゲームを作成」することとなりますので、
必ずぷちコンの詳細をご確認の上、 ぷちコン用のゲーム制作を目的としてご参加ください。

また、この期間内での完成を目指すかどうかや、チーム分けをするかどうかは
当日の状況をみて決定・進めていく予定となります。

・ひとまずその場での開発開始を想定していますが、
事前に考えた企画の持ち込みや
すでに制作開始されているものを進める形もOKです。
途中参加・飛び入りもOKです。

会場

名古ゲ部にて行います。

注意事項

2日間開催となります。
1日目と2日目の間は各自のご都合の良いタイミングで
帰宅・再開していただければと思います。

よほどのことがない限り大丈夫ですが、
ゲーム制作の空気をあまりにも乱す場合は大変申し訳ありませんが
ご退場いただく場合がありますのでご了承ください

スケジュール

*時間は目安となります。
1日目終了時間以降も各自ご都合の良い時間まで作業OKです。 *
