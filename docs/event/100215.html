はじめまして、YANA.と申しますm(._.)m
タイトルのとおり、シルドラ様でスプラトゥーン2交流会を開催させていただくことになりました！
(過去の開催先駆者様に敬意を払って第3回とナンバリングしました)
こういった企画は高ウデマエの方が開催するのが相場ですが、今回は(ほぼ)初心者の私が主催いたします。
そこで、今回は初心者・低ウデマエ・Switchを持ってなくて不定期にしか遊べない・オフ会怖い…といったイカさん大歓迎の回とさせていただきます！
「弱いから…」と二の足を踏むのはもったいないです。このゲームはリアルの交流ができると世界が広がります。
(もちろん高ウデマエの方も大歓迎です。)

【第3回スプラトゥーン2交流会詳細】
■日時
2018/10/06（土）12:30～18:00
■場所
カフェ&バー シルドラ（宮城県仙台市）
・HPリンク
【https://twitter.com/cbsyldra】
・会場までのアクセス
【https://goo.gl/zHhtDz】
■参加者定員
32名（プレイヤー枠24名+見学枠12名〜）
※プレイヤー枠は最大24名です。
※先着順となります。期限が近づいたら4の倍数で締め切る場合があります。
※「プレイしないで見学だけ…」もOKとします。
※止むを得ず当日キャンセルや遅刻がある場合は、見学者から立候補を募ります。
※見学希望の方はコメントにその旨の記載をお願いします。
※プレイヤー枠の上限に達した後は、自動的に見学枠となります。
※見学の方はスプラトゥーン2の見学のみで、他のゲームプレイはご遠慮いただきます。
■参加費
1,500円
（一部アルコール、ソフトドリンク、月変わりフード食べ飲み放題付き）
※見学の場合も参加費は必要です。
※見学希望の途中入店は、一時間500円(上限1,500円)。
■設置物
モニター8台、プロジェクター、電源タップなどをお借りします。
■参加条件
・スプラトゥーン2の経験者
・ガチルールのウデマエC-以上
・みんなで楽しくできる方(味方や相手の批判は禁止)
■イベント内容
4人でチームを組み、以下の内容で対戦していただきます。　
※チームは当日抽選で決定。
※ウデマエが偏らないよう調整(入場時にウデマエアンケートを実施)。

第1部 ガチルールの部
第2部 甲子園ルールの部
第3部 初心者マッチ&個別指導の部

【各ルール】
第1部 ガチルールの部(想定所要時間3時間)
・6チームを2グループに分け総当たり
・下記ガチルール2本先取
①エリア (ステージはランダム)
②ホコorヤグラ ①の負けチームが指定(ステージも指定)
③ホコorヤグラ 残ったルール(ステージはランダム )
・各グループ1位同士で決勝戦(決勝も2本先取)
※同率の場合…得失セット差
※それでも同率の場合…じゃんけん

第2部 甲子園ルールの部(想定所要時間2時間)
・6チームによるトーナメント(第1部のグループ1位はシード)
・甲子園ステージからランダム、ナワバリ2本先取(決勝も2本先取)
・ギアパワーはメインのみ
甲子園ステージ…
・バッテラストリート ・ザトウマーケット
・ショッツル鉱山 ・コンブトラック
・海女美術大学 ・スメーシーワールド
・マンタマリア号 ・ハコフグ倉庫

◎初心者チーム編成◎

第3部 初心者マッチ+ウデマエX勢からの個別指導の部(想定所要時間15分)
・ウデマエが低い方から8人を2チームに分け、ガチエリアバトル
・ステージランダムの2本先取
・1プレイヤーにつき1人、うしろにXプレイヤー(高ウデマエ)が付きます
※試合中試合後に、ルール関与や打開、対面について優しくアドバイスしてあげてください。
※時間の都合により第3部は中止になる場合があります。

■最後に
初心者の方・オフ会に二の足を踏んでいた方、絶対に後悔させませんのでご検討ください！今回は見学枠もありますので、まずは見てみる…だけでもオッケーです！
そしておなじみの方々も遠慮なくご参加くださいm(_ _)m
