description

Reproでは、当社の持つアプリの改善ノウハウや、「Repro」のツールの使い方をもっと知りたいというご要望にお応えし、アプリの改善をこれから始める方に向けた少人数セミナーを定期的に開催しております。

「アプリをDLしたユーザーが継続的に利用してくれず、すぐ離れて行ってしまう」 「解析ツールなどで離脱した箇所は特定できるが、離脱している原因がわからない」 「プッシュ通知やアプリ内メッセージを活用した効果的な施策を行いたい」 「自社アプリの無課金ユーザーと課金ユーザーの利用方法の違いを知りたい」

こういったアプリ運用の悩みはReproを使うことで解決することができます。

Reproとは？

https://repro.io/

Reproはモバイルアプリの課題の発見から解決の方法までワンストップで提供しているアプリの成長支援（グロースハック）ツールです。

アクセス分析やユーザー行動の録画機能など課題発見の機能から、プッシュ通知やアプリ内メッセージなど課題解決のための機能まで1つのSDKで提供しています。

※本グループは、グループ・イベント管理サービス「peatix」のReproページからも集客を行っております。

セミナーの対象者

  • アプリのディレクター、マーケター、エンジニア

※ Reproの実装方法ではなく使い方に関するセミナーなので、コーディングなどの知識は不要です。

※ すでにReproを利用している必要はありませんが、ツールの使い方がメインのセミナーとなっておりますので、アプリ担当者ではない方、Reproを導入する予定の無い方のご参加はご遠慮ください。

こんな方におすすめ

  • アプリの解析ツールを探している方
  • プッシュ通知ツールを探している方
  • アプリのグロースハックに興味がある方
  • ターゲティングプッシュ通知やアプリ内メッセージなど最先端のアプリ内マーケティングを体験してみたい方
  • Reproのアカウントをすでに持っているが、活用方法に迷っている方

セミナー内容

セミナーはワークショップ形式とし、Reproの管理画面とデモアプリをお渡しし、ユーザーをターゲティングしたプッシュ通知やユーザー行動の録画機能を使ったアプリの課題発見などについて参加者に実際に体験してもらいます。

最後に個別相談会も設けておりますので、アプリの改善に関するものから技術的なご相談まで、ご質問がある方はぜひお越しください。スタッフ一同お待ちしております。
