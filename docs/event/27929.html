iPhone、iPadのビジネスアプリとこれから流行るアプリ一挙紹介

description

Apple好きがこうじて３万人のリスナーを抱えるAppleNewsRadioワンボタンの声をiTunesから配信し、関西ウォーカー誌にもiPhoneアプリの紹介記事の連載を持つ山村氏。その山村氏が長年の経験と実績で、iPhone、iPadを使いこなす為に、これは絶対外せないアプリ、これはお薦めできるアプリ、そしてこれから流行りそうな新しいSNSなどのアプリを一挙紹介。少なくとも2007年にiPhoneが発表されてから５年間の山村氏のノウハウを、ギュギュと２時間に凝縮してお伝えする、とってもお得なセミナー。iPhone、iPadユーザーは必見です。

こんな方にお薦め
・iPhone、iPadを購入したけど、ビジネスあまり活用できていないんじゃないか？と感じている方
・iPhone、iPadで、もっと楽しい使い方ができると感じている方
・iPhone、iPadを、もっとトコトン使い込みたいと思っている方
