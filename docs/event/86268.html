3月の川岡大次郎さんのクラス受講生募集

CREACT-クリアクト-では現役で俳優として活躍されている方々にお越し頂き、役者だから聞きたい事や悩み、台本の読み方、仕事の仕方、撮影現場でのお話など、普段では聞けない貴重なお話をしてもらいます。
今回は、岸田戯曲賞受賞のヨーロッパ企画原作の映画「サマータイムマシン・ブルース」（本広克行監督）や、蜷川幸雄の舞台「わが魂は輝く水なり」、フジテレビ「救命病棟24時」第三シーズンで小栗旬の兄役、大河ドラマ「龍馬伝」福山雅治演じる龍馬のいとこ高松太郎役、朝ドラ「カーネーション」など様々なフィールドで活躍の俳優川岡大次郎さんのワークショップを開催します。

川岡大次郎さんからのコメント

川岡さんからコメント頂いております。
「僕は今年でデビュー２０年目です。
その間、様々な現場で仕事をしてきました。様々な役者を近くで見てきました。そして様々な演出家とも出会いました。いろんな人を間近で見てきて思う事は、どんな才能豊かな人でも努力しています。苦しんでます。己の壁と戦ってます。それを芝居という形でどう表現できるのか？が役者さんです。

「台本を読んで、声に出すまでに何をしていますか？何を考えてますか？
ちゃんと色んな準備ができてますか？」

自分が経験してきた、色々な状況や芝居する上での大切な事など、伝えられる事が多いと思います。あと、折角のこういう場所なんで僕自身も色んな方と出会って、刺激を受けたいと思ってます。」

川岡大次郎さんのプロフィール
1997年映画「タイムリープ」でデビュー、その後、フジテレビ月9「ビーチボーイズ」（反町隆史＆竹野内豊）、TBS「最後の恋」（中居正広＆常盤貴子）、フジテレビ「救命病棟24時」（江口洋介＆松嶋菜々子）、NHK大河ドラマ「龍馬伝」（福山雅治）、映画では、行定勲監督デビュー作「OPEN　HOUSE」、ヨーロッパ企画原作映画「サマータイムマシン・ブルース」（瑛太＆上野樹里）、その他、舞台、CM、ナレーション、等々、様々な分野で活躍。

詳しいプロフィールは
http://www.kawaokadaijiro.com/prof/index.html

なかなか、聞けない話や、一流の現場に出ている役者にしか感じられない体験のお話など、色々吸収する事はあると思います。

「参加者からの声」

・本当に毎回のレッスンで熱心な指導をしていただける人は中々いらっしゃらないです。
もっと大次郎さんから演技や生き方等の話を聞いてどんどん吸収したいと思っています。
・川岡大次郎さんは、自身の経験から、私たちに必要な事や大切な事を教えてくださるので、すごく説得力がありますし、現実味があり、意欲向上に繋がります。
川岡大次郎さんに役者の術を教えて頂くようになり、これまでの自分の人生がどれだけ薄っぺらかったかを痛感しました。役者はその人の生き方が出る。アンテナを張るという事の意味を具体的に考えないで、やっているような気になっていました。
・俳優目線で語ってくださるので共感できることが多く、説得力がありました。
