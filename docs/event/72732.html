JAWS-UG宮崎勉強会

宮崎県 JAWS-UG の勉強会を開催します！！ エバンジェリスト 高岡氏をお迎えし、AWS最新の情報についての講演をしていただく予定となっております。 また、実際にAWSの導入時の体験談や、導入後での変化した点、ノウハウ蓄積等を紹介するLTの予定も組んでおります。 高岡氏やLT発表者へ直接ご質問もできる時間もありますので、興味のある方、AWS導入を検討されている方、是非ご参加ください。
