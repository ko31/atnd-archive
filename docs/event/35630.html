Windows Phone ハッカソン in 名古屋

イベント ユーザー
event_id 35630
title Windows Phone ハッカソン in 名古屋
catch キャンペーンに応募してデバイスをゲットしよう
description

テーマ

  • ハッカソン（Hackathon）とは、ハック（Hack）とマラソン（Marathon）を組み合わせた造語ですが、このイベントでは、ハックとは「プログラミングする」という程度の意味で使っています。つまり、みんなで集まり、アプリの開発・発表をする会のことです。
  • 今回は、マイクロソフト主催の 「Windows Phone アプリケーション トライアスロン 2013」 に応募して、Windows Phoneのデバイスをゲットしようというのが目的です。
  • もちろん、このキャンペーンとは関係なく、情報交換したいとか、ひたすらプログラミングに打ち込みたい、という方の参加も歓迎します。

イベント内容

  • 初心者向けには、アプリの作成方法からストアに公開する方法までをワークショップ形式で解説します。
  • 経験者や開発者登録をお持ちの方は、参加者どうしで情報交換しながら、自由にアプリケーションを作成していただきます。

注意事項

  • トライアスロン・キャンペーンに応募するためには、事前に開発者登録（年間9800円）を完了しておく必要があります
    • なお、トライアスロン達成の結果として、デバイスではなく、次回の開発者登録の更新費用が無料になるトークンを選択することもできます。（この場合でも、今回の応募のための開発者登録は必要です）
  • トライアスロンの応募は、任意です。（強制ではありません）
  • このハッカソンは、トライアスロンの完走を保証するものではありません。
  • 今回のキャンペーンは、デバイスの数に限りがあり、先着順でデバイスが配布されるようなので、早めの応募をお勧めします。

参加条件

  • 特にございません。初めての方、初心者の方、大歓迎です！
  • デベロッパーの方、デザイナーの方、興味のある方ならどなたでもご参加頂けます。
  • 開発環境をインストールしたPCを持参してもしなくても、会場に用意してあります。
    • 自分のPCを持参した方が開発しやすいという方は、ご持参ください。
    • ただし、会場のPCは、SLATに対応していないため、WP7のシミュレータのみとなります。つまり、WP8のシミュレータは動作しませんので、ご注意ください。
  • Windows Phone 7 デバイスは、所持していなくてもかまいません。（エミュレーターで動作確認します）

用意していただくもの

  • 開発用PC（用意できる方のみ）
    • OS Windwos Vista 又は Windows 7 / Windows 8 (WindowsXPは使用できません)
    • 仮想環境は非推奨です Macを使う場合はBootcampでの動作を推奨します。
    • 持参できない場合は、会場のPCを使用できます。（ただし、WP7向けの開発のみ）
  • Windows Phone（用意できる方のみ）
    • 持参できない場合は、会場のPCのエミュレーターを使用できます。（ただし、WP7向けの開発のみ）
  • 開発用ツールは、 Windows Phone SDK からダウンロードして、インストールしてください。
    • ［ダウンロード］ボタンを気軽にクリックしてはいけません。そうすると、英語版がダウンロード（インストール）されてしまいます。
    • 自分のPCが日本語環境の場合「詳細情報および別の言語を入手する」から、日本語のバージョンを選択してください。
    • Windows 7/Vista の場合は、［SDK7.1］と［SDK 7.1.1 更新プログラム］の両方をインストールすることを推奨します。
    • Windows 8 の場合は、［SDK8.0］をインストールすることを推奨します。
    • ISO形式のセットアップ・ファイルが必要な場合も「詳細情報および別の言語を入手する」を選択し、画面の下の方にある「.iso」へのリンクを利用します。
