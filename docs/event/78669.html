description 　環境NGOの環境教育フォーラム21（東京・千代田）は、第20回eco検定に1回での合格を目指して、東京、大阪、名古屋で2日間の集中講座を開催します。

・一日目は地球温暖化、生物多様性、循環型社会、エネルギー問題、環境と経営などeco検定に出題される問題をテーマ別にわかりやすく講義します。
・二日目は模擬問題形式での学習で、試験に出題されやすい問題を解きながら、重要ポイントを習得します。
パリ協定などeco検定で出題されやすい時事的な問題もテーマにあわせて解説します。
開催スケジュールは下記の通りです。
※開催時間は原則午前10時－午後4時半。
◎東京
■6月25日、26日　文京シビックホール（東京メトロ、都営地下鉄最寄り駅直結）
◎大坂
■7月1日、7月2日　大阪市 大阪産業創造館（地下鉄堺筋本町駅、本町駅などから徒歩7分）
◎名古屋
■7月 8日、9日　　名古屋市中小企業振興会館（吹上ホール、地下鉄桜通線吹上駅徒歩5分
◎参加費　9260円（消費税込で10000円）
