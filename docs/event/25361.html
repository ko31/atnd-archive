■イベント概要

最新のiOS5のAPIと Xcode4.2を使った、開発手法を体験しましょう。

また、ライトニングトークの機会もありますので、トーク希望の方はメッセージをお
送りください。

2008年にリクルート初のiPhoneアプリである「ホットペッパーアプリ」の開発から携
わり、2010年にはiPadの国内発売にあわせて「じゃらん沖縄アプリ」をリリースした開発者
が講義します。

内容（仮)
・ARC
・Storyboard
・Segue
・Table View Static Content
・iCloud
・TwitterAPI
・Newsstand
・Custom Appearance
等々、開発効率があがる機能がたくさんあります。サンプルコードとともに、ご紹介します！

最新の実装方法を体験して、アイデアをスマートに実装に結びつけましょう。

日時：2/24（金） 17:00～20:30（19:00～20:30は懇親会）

費用：無料
主催：リクルート メディアテクノロジーラボ（MTL）

■応募条件

(1)学生であること
(2)MacBookまたはMacBookProまたはMacBookAirを持参できること
(3)Xcode4.2がインストールできていること
(4) Objective-Cの文法は、理解できている。
