【給料以上の不労所得を得る方法】個人でできる副業・ビジネス情報2018（初心者向け）

今も昔も、お金に直結する最も価値のある要素は、「情報」と言って過言ではありません。

ただしネットが普及した今、「情報」を取りに行って得られる量があまりに多すぎるのが実情。

なので現代においては、「情報」をすばやく、正しく判断できるための「取捨力」も同時に必要になってきます。

でないと、運良く良質な情報に出会えたとしても、「そんなうまい話があるわけがない」と自分から拒否してしまったり、
または、良い情報だと思って喜んで飛びついた結果、騙されてしまうことにもつながりかねません。

副業解禁元年と言われる2018年ですが、会社勤めの人のマインドはまだ「勤め人マインド」のままの人がほとんどです。

辛い仕事を一生続けていく人生もあれば、自分が楽しい、有意義、もしくは楽に感じる仕事を自分でチョイスして、
給料以上の活動所得、不労所得を得ることは実は難しいことではないのです。

「知っているか、知らないか」「知っていたら、やるか、やらないか」だけなのです。

このセミナーでは、

・マインドの転換（仕事が人生を楽しく豊かにするマインドセット）
・情報の取捨能力を上げる
・今始めると有利なビジネス情報の最新情報

を初級編として、あなたにわかりやすい言葉でざっくりとお伝えしていきます。

難しく考えずに、コーヒーを飲みながら気軽な気持ちで聞いてもらえるセミナーですのでご安心ください♪

セミナーの後には講師やスタッフとの相談（雑談）機会を設けることも可能です（自由参加）

楽しく仕事をすれば、もっと幸せな毎日にできます！
