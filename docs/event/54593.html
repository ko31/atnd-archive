組織活性化のためのリーダーシップと成果直結型プロジェクトの進め方

title 組織活性化のためのリーダーシップと成果直結型プロジェクトの進め方
catch 「個」と「組織」を活性化し、飛躍的な事業成果を生み出す！
description

◆受講対象・レベル：
・戦略の実行、成果に責任を持つ幹部、管理職
・継続的な業績結果を出せる組織体質をつくりたいと考える幹部、管理職
・結果が出せる人材を育成したい幹部、管理職

◆趣旨：
　多くの製造業が中期経営計画に基づき実行フェーズへ移行している頃だろう。グローバル化が進む今日においては、海外市場も視野に含めてチャレンジングな戦略・計画を立案しているケースも多い。しかし、それらの進捗は果たして順調であろうか。
　もし順調でないとすれば、そこにはひとつの原因が考えられる。それは現場のモチベーションの低下という問題だ。戦略・計画は論理的には良いが、戦略実行の担い手である肝心の現場がイマイチ活性化していない。地頭は良く基本的に真面目な人材が多いのだが、どうも組織全体となると元気がない。そんな壁にぶつかっている製造業も少なくはないのではないだろうか。
　当セミナでは、組織活性化のためのリーダーシップのポイントと成果直結型プロジェクトの進め方を紹介する。コンセプトは「組織の理念・ビジョン・目標と、個人の価値観・キャリアビジョンを重ねることによる自律性の欲求の充足」「プロジェクトにおける自己成長や創出した成果への他者からの評価による有能感の充足」「プロジェクトというコミュニティの形成による関係性の欲求の充足」の３つである。

◆プログラム：
１．リーダーは「経営者」という認識をもち、組織メンバーをモチベートする
　・そもそも企業や組織とはなにか？　
　・企業は経済的成果だけなく、それを超えた目的をもつ
　・財務成果・戦略成果に責任を負うリーダーは、「経営者」である
　・成果創出のために組織メンバーの能力・行動を引き出す動機付けが必要
　・働くことのモチベーションはどこから来るか？　～３つの基本欲求「自律性」「関係性」「有能感」の充足～

２．理念を軸にしたリーダーシップとは
　・経営者は経営の「軸」を明文化し、一貫性をもって組織リードしていく
　・理念とは、理念の役割とは
　・戦略やマネジメントにおける理念の位置づけ
　・理念を軸にしたリーダーシップとは　～変革機会の把握力、ビジョン構築力、価値共創、創発力、優先課題の突破力、加速度力、展開力～
　・組織の理念・ビジョンと組織メンバーのキャリアビジョンを重ねることによる「自律性」の欲求の充足

３．ポジティブ心理学によるリーダーシップとは
　・ポジティブ心理学とは
　・ポジティブな状態で脳は活性化される
　・適切なストレッチ目標・ビジョンの設定により「フロー状態」をつくる
　・人の心のつながりとしての業務プロセス
　・リーダーは「モチベーション・クリエーターになる」
　・ポジティブフィードバックによる組織メンバーの「有能感」の充足

４．理念を軸にした組織活性化の進め方
　・理念を軸にした組織活性化の全体像　～対話による理念浸透～
　・理念を軸にした組織活性化の基本ステップ
　・ステップ１．企業エピソードの作成と今後継承すべきDNAの抽出
　・ステップ２．外部・内部環境分析と今後追加すべきDNAの作成
　・ステップ３．ＤＮＡ・理念・ビジョンの再定義
　・ステップ４．ＤＮＡ・理念・ビジョンの浸透ＰＪの企画、実施
　・ステップ５．理念・ビジョン・戦略をベースにした各組織の課題の再設定と「ブレークスルーゴール」の検討
　・ステップ６．各組織における「ブレークスループロジェクト」の実施

５．「ブレークスループロジェクト」とは
　・なぜ大きなプロジェクトは失敗するのか？　
　・ブレークスループロジェクトとは　～連続的な成果を通じた人材の意識と組織文化の変革～
　・プロジェクトというコミュニティの形成による組織メンバーの「関係性」の欲求の充足
　・ブレークスループロジェクトの【スタート～３０日】のポイント
　・ブレークスループロジェクトの【３０日～６０日】のポイント
　・ブレークスループロジェクトの【６０日～９０日】のポイント
　・ブレークスループロジェクトの実施事例
