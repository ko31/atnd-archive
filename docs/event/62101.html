普段の生活の中で簡単に出来る、朝から始めるダイエット！美容の小ワザをご紹介いたします。取り入れるだけで、他人と差がつく！ ワンランク上の自分と出会うための魅力アップセミナーです。

キレイナビ代表であり、JDHA認定インストラクター飯塚美香が、朝活ダイエット法をはじめ、食べ過ぎた時の対処法や簡単スリム術をご提案。そしてフェイシャルヨガインストラクター仲里あやねが、フェイシャルヨガレッスンで小顔エクササイズを実践！
　
また、ご希望の方には13:00よりタイチの60分レッスンに無料でご参加頂けます！（通常4,100円）

参加者全員にコスメなどキレイになれる美容アイテムをお土産としてプレゼント致します！

お子様連れ、男性も大歓迎！
みなさまのご参加をお待ちしています！
