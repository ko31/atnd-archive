description

Raspberry Pi は Linux で※1 動いています。
ネットワークを経由して Raspberry Pi に ssh でログオン、Linux の簡単な操作をしてみましょう。
Raspberry Pi ならではの電子工作の初歩の初歩もチャレンジしてみます。

株式会社XSHELLの社内勉強会として企画しましたが、誰でもWelcomeです！！

やること

Raspberry Pi の配線、起動、ネットワーク上でのRaspberry Piの見つけ方と識別。
SSHでログオン、Linuxのオペレーション。
フラットケーブルとブレッドボード、WiringPiを使ってLEDを光らせる実験。

対象

  • 文系チャレンジャー
  • 理系だけどLinux知らない
  • RaspberryPi買ったけど動かしてない
  • Linux 触ってみたい
  • ssh ってアコガレ
  • クラウドでLinux操作したいけど、はて？

のようなひと

学生や個人の方も大歓迎です！

XSHELLとは

シングルボードコンピューターを使って、プロトタイピングから、プロダクションモデルまでをシームレスにつなぐ、IoT向けのPaaSを開発しています。
WEBアプリケーション技術、ハードウェア技術、3Dモデリング技術を組み合わせて、IoTの事業開発支援も行っています。
オフィスはレーザー加工機、3Dプリンタ、小型CN旋盤、オシロスコープなどがあって、ミニFablabみたいになってるよ！
