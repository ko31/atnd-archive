概要

Amazon EC2やAmazon S3に代表される Amazon Web Services(AWS)は、さる 2011 年 3 月に東京リージョンが開設され、今後日本でもますますその利用が広まっていく事が予想されます。

この度、その AWS について、具体的な利用方法から設計、運用の考え方まで幅広く情報共有及び議論する場を京都でも設けたいとの思いより、JAWS-UG 京都支部を立ち上げるに至りました。

その、立ち上げ初回の勉強会、記念すべき第1回目の勉強会を開催したいと思います！

  • AWSに興味があって、使ってみたい方、
  • AWSは触ってみたけれども、もっと深く知りたい方、
  • AWSをすでに使いこなしていて、その使い方を広めたい方、
  • AWS の設計方法について、ディスカッションしたい方、
  • JAWS-UG 京都の運営メンバーになられたい方、

などなど、是非お気軽にどしどしご参加ください。
