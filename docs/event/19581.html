無論、Haskellの秋

Haskell入門者のための勉強会 スタートHaskell の 第2回 ATNDです。

スタートHaskellは、入門者から初心者の方を対象に、実際にコーディングをしながら１からHaskellを学んでいく勉強会です。月1回ペースで全5回を予定しています。

第2回は プログラミングHaskell の5章から7章までの範囲を取り上げます。
