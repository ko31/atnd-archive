(仮)Sendai.swift を開催します。

今年はそろそろ本格的にSwiftをやろうかと思っていますので、勉強会的なものをやってみようと思います。

概要

今回は、WWDC2015のSwift関連セッションを復習します。

WWDC 2015 - Videos - Apple Developer にて一般公開されているビデオやスライドを見て、Swiftについて学び、理解を深めていきたいと思います。

内容

取り上げるセッションは、Protocol-Oriented Programming in Swiftです。ざっくり言うと、「Swiftの継承はClassじゃなくてProtocolを使うと良いよ」という内容です。

参考資料

  • Protocol-Oriented Programming in Swift - WWDC 2015 - Videos - Apple Developer
    • こちらがWWDCの資料になります。
  • Swift 2で提唱されているProtocol Oriented ProgrammingをWWDCセッションから学ぶ - Qiita
    • わかりやすい解説記事です。ぜひ、この記事を予習してから参加されることをお勧めします！

タイムテーブル

  • Session 1 (60h).
    • 公開されているビデオやスライドなどを見て、WWDC2015のおさらいをします。
    • 注) 資料はすべて英語です。
  • Session 2 (60h).
    • セッションについて、参加者の間で質疑応答などを行います。疑問点などを話し合いたいと思います。
