Windows Phone 全国ハッカソン祭り in 福岡 + 初心者向け勉強会


■ハッカソンとは

「Hack(ハック)」と「Marathon(マラソン)」を合わせた造語。特定の開発テーマの技術に興味のあるプログラマーたちが会議室やソファーがある場所などにノートPC持参で集まり、みんなで一緒にソフトウェアをハックし、最後に開発したアプリケーションやサービスを参加者全員の前でプレゼンするというイベント。


今回の勉強会 + ハッカソン祭りに参加すると！

  • Windows Phone アプリ作成から申請までをレクチャーを通して体験出来る
  • アプリ申請時に必要なApp Hub（年会費9,800円）をプレゼント（※1）
  • 他のWindows Phone アプリ開発者との交流が可能
  • スキルアップ・新たな技術の発見がある？
  • 高橋さん（※2）に技術的な質問が出来る！

※1・・App Hubに関しては、その後、複数のアプリを登録する事が約束との事です。
※2・・日本マイクロソフト（株）から、デベロッパー・エバンジェリスト高橋 忍 様がレクチャー講師としてお見えになります。


【　内　容　】

勉強会
最初に初心者向け勉強会を行ないます。
勉強会では、Microsoft高橋様によるレクチャーを受けます。
レクチャーでは、アプリの作成から申請までを実際に行ないます。
『Windows Phoneってどげんとかいな？』と思っている方も、一通り体験する事で全体を知る事が出来ます。

ハッカソン
Windows Phone 7をテーマに自由に開発を行ないます。
初心者の方は、新たにアプリを1つ作るのは難しいかも知れませんが、出来る所まで開発を行ないます。

ハッカソンといえば、チームを組んでお菓子を食べながら開発を行なうイメージが強いのですが、今回は、Windows Phone 開発未経験者が多数なので、チーム制では無く、個人でアプリを作る可能性が高いと思います。（もちろんチームを組むのもOKです）


【　参加要件　】


特にございません。誰でもOKです(^^)

当日高橋様のレクチャーを行ないます。出来ればWindows Phone開発に必要な以下の環境が入ったWindows Note PCをご準備下さい。

  • Windows Phone SDK 7.1
  • Windows Phone Icons Maker
  • Zune Software

後、参加表明時に、懇親会参加可否のコメントもよろしくお願いします。

※会場スペースにも限りがございます。先着順ですのでお申し込みはお早めに。九州各県からのご参加、是非お待ちしています。
