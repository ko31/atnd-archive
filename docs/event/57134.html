XMLデータベースで柔軟なデータ管理

イベント ユーザー
event_id 57134
title XMLデータベースで柔軟なデータ管理
catch 近そうでいて異なる、XML DB と RDB
description

Webアプリケーションでの標準データフォーマットとして用いられるXML。そのほか、HTML5を始め、XBRL、流通BMS、DITA等、各業界でも今が旬のXML。それらを格納するデータベースエンジンとしてXML DBという存在は聞いたことがある！という方は増えてきていますが、最初の一歩を踏み出すにあたり、以下のような疑問をお持ちだからではないでしょうか？

・「具体的にどういった時に使われるの？」
・「RDBとの違いは？」
・「今流行りのNoSQLやKVS（キーバリューストア）との違いは？」
・「XML DBは遅いという噂を聞くけれども、本当？」

本セミナーでは、XML DBについて基本的な解説を行わせて頂いた後、ここだけでしか語れない事例を月替わりでお話させていただきます。

XMLデータベースの情報収集目的や、顧客に提案してみよう、という方から、自社システムのデータベース選定を行う方まで、少しでもご興味をお持ちの方は気軽にご参加下さい。

■対象

一般的なRDBを用いたシステムとは異なった分野やスタイルとなりますので、以下の課題をお持ちの方にはうってつけのセミナーです！

・既存顧客の取引拡大を目指されている方
・新しいアイテムで新規顧客の開拓を検討されている方
・過去取引顧客にアプローチしたいけれどもネタがない方

また、エンジニアリングの観点から、以下のような課題をお持ちの方にもぴったりのセミナーとなります！

・現在データベース選定を行っている方
・顧客にXML DBを提案しようと考えられている方
・XMLデータのハンドリング方法を検討されている方
・複雑なデータのハンドリングに困っておられる方
・変更が多いデータのハンドリングに困っておられる方
・DITAなどの半定型文書※ の利活用を検討されている方
※ある程度整った、テキスト中心のドキュメント文書。画像などの差し込みも可能。
・最新のXML DBに関する動向を知りたい方
・XML DBを活用した事例に関心をお持ちの方

皆さまのご参加を心よりお待ち申し上げております。
