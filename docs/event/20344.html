今回は「App Inventor」を使った勉強会です。
「App Inventor」を使えば、Javaを書けない非開発者でも手軽に短時間でAndroidのアプリケーションを作ることができます。
「App Inventor」はちょっと前まではGoogle Labsで運用されていたツールでしたが、現在はMIT Media Lab（メディア・ラボ）に移管されました。

皆さんは学習用言語Scratchをご存知ですか？これは子供用教育用プログラミング環境で、レゴのようなブロックを積み重ねていくことによって画像をアニメーションさせるようなプログラムが作れてしまう大変面白いツールです。本イベントの管理人であるkaoの娘（当時6歳）でも、一日でそんなプログラムができてしまったくらい簡単です。
「App Inventor」はこのScratchの思想を大きく受け継いでいるAndroidアプリ版のツールです。

この勉強会に参加して、作り方がわかったら早速おうちに帰ってお子さんやお友達などにも教えてあげてみてください。自分が作ったアプリがスマートフォンで動いたら感動ですよね！？

ノートパソコンさえ持っていれば誰でも参加することができます。
今回もAndroid女子部ならではのゆるりと楽しい勉強会にしましょう！

App Inventorについては こちら をご覧ください。
