今月は 「イケテルGit勉強会」 と題して、
バージョン管理システム のスタンダードとなりつつある Git について、みなさんと一緒にハンズオン形式で勉強しましょう!!

今回の勉強会は、Git のホスティングサービスである github を使います。
後述の 「必要なもの」 をご確認ください。

日時

3月24日(土) 13:00〜18:00
(受付は12:30〜)

場所

住所: 福岡市博多区博多駅東1丁目17番1号　福岡県Ruby・コンテンツ産業振興センター 5階

タイムテーブル

時間 内容
12:30 – 受付
13:00 – Gitってなに？
13:30 – 基本操作のハンズオン
14:30 – Ruby開発サーバの紹介
15:00 – Githubってなに？
15:15 – チームプレイをやってみよう
17:00 – クロージング、会場片付け
17:30 – 公開スタッフミーティング

対象

  • Gitを使ったことがある方、もしくはSubversionなどのバージョン管理システムを使ったことがある方

必要なもの

  • ノートPC
    • 要無線LAN
  • Github のアカウント
    • https://github.com/
  • また、ノートPCでGit が使える状態であること
    • Windowsユーザーの方は hibiki さんのブログエントリーを参考にGitのインストールとgithubに鍵登録をしていただくと良いです。(hibiki さんに感謝!!ありがとうございます!!)
      • Github を Windows で使ってみる 2011 年版
