  • 「 Raspiで百葉箱をつくりたい　第一回目　カメラ周りの復習」
    • 平澤俊継

超ベーシックですが、備忘録的にmotionとかraspistillとかの使い勝手をまとめてみます。

14:00-14:30 セミナー2

  • 「子供向けプログラミング入門からフィジカルコンピューティングへのステップ」
    • ちっちゃいものくらぶ　とも
    • http://tiisai.dip.jp/?p=2591

近年、子供向けプログラミングが広がり始めています。この流れは、PCに閉じたプログラミングから、フィジカルコンピューティングや電子工作にも拡大しています。
なのぼ～どを使った、Scrachでのフィジカルコンピューティングワークショップを二年間実施してきた経験から、新作のArduino互換ボード、「ちびでぃ～のBB」を開発しました。
子供たちの利用形態に最適化したハードウエア、ソフトウエアについて紹介いたします。

14:00-14:30 セミナー3

  • 「ちっちゃい3Dプリンタを考える」
    • 秘密結社オープンフォース　河野
    • http://twitpic.com/dnbejs

冬コミに向けて一万円台で購入できる小さな3Dプリンタのキットを企画しています。
外形サイズ15cmx15cmx15cm程度、ワークエリアが3cmx3cmx3cm程度、精度0.5mm程度のおもちゃですが、自分で様々に手を加えて機能・サイズアップが可能となります。
歩行・走行ロボットに代わるメカトロ入門教材を目指し、小学生程度でも作成OKなレベルで想定しています。
今回は、このプロジェクトの目指すもの、および小サイズという要件等から導き出される技術について説明します。
