技術ひろば.net 2011年10月 勉強会

以下の通り「技術ひろば.net 2011年10月 勉強会」を開催します。
「技術を楽しむ、技術でつながる」の理念の元、人と技術を大切にする活動を行っています。
どなたでもお気軽にご参加ください。

開催概要

■日時 : 2011年10月29日(土) 13:00～17:45　(開場 12:30)
■会場 : 首都圏コンピュータ技術者株式会社 2F 大会議室
　http://mcea.jp/tokyo#maptitle
■会費 : 無料
■定員 : 30名

内容

久しぶりの開催となる 10月の技術ひろば.net は、

　System Center Virtual Machine Manager 2012
　お客様の希望を叶えるために覚えておきたいこと
　Windows Azure Toolkit for WP / iOS

の3本立てです。
インフラ、開発、ビジネスとバラエティーに富んだ構成です。自分の得意分野・興味分野とあわせて、他のジャンルにもぜひ触れてみてください。

お茶会（フリートーク）も開催します。「懇親会（＝飲み会）まで付き合うのは時間的、金銭的にちょっと」という方も、おしゃべりや情報交換をしましょう。

※当初予定していた LightSwitch セッションは中止とします。11月に他の勉強会で同じテーマのセッション予定です。12月以降の技術ひろばでの実施も検討中です。

  • 「はじめての SCVMM 2012」 （玉井さん）
    来年リリース予定である System Center Virtual Machine Manager 2012 についてデモを交えてご紹介します。新機能について紹介します！
    複数の Hyper-V を管理している、Hyper-V Manager だけで管理しているなどの場合には SCVMM を導入するとハッピーになれるかもしれません。
  • 「要望をかたちに ～ ニガテなデザインについてお客様と語るために知っておきたいこと ～」 （セカンドファクトリー 新谷さん）
    「お客様に振り回される」「完成してから仕様が違うといわれる」「仕様通りに作ったのに使えないと言われ、案件が終了しない」など、実際に開発を行っていると技術を磨くだけでは対応できない、多種多様な問題が発生します。
    要望の裏に隠れている真のニーズを考えること、開発しやすい機能に落とし込むために必要な考え方など、お客様の希望を叶えるために覚えておきたいことはいくつもありますが、その中から今回は、エンジニアが苦手とする「デザイン」について考えてみたいと思います。
    とはいえ、私自身もデザイン苦手なエンジニアです。そんな私が、クライアントの要望をデザイナーに伝えるためにどんなことを聞いているのか、どうやって機能とデザイン（的なこと）を考えているのか、デザインセンスがなくても覚えておいて役に立ったことなど、デザインについて他の人に助けてもらうのに役立つことを共有したいと思います。
  • 「Windows Azure Toolkit for Windows Phone / iOS による開発」 （MS 鈴木さん）
    Windows Azure Toolkit for Windows Phone / iOS を使って、スマートフォン+クラウド(Windows Azure)のソリューションを開発する方法をご紹介します。
  • お茶会
    お茶でも飲みながら、自由におしゃべりや情報交換をしましょう。
    各自お好みのノンアルコール飲料を持ってきてください。
    （アルコールはお茶会の後で）
