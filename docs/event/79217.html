ハースストーン究極Wiki主催のデッキコンテストです。 概要は上記URLでご確認ください。

参加費

  • トーナメント参加費：無料
  • 会場場代：600円前後(最初の1時間:300円、以降1時間ごとに100円)

ルール

  • 第一回デッキコンテストに投稿されたデッキからデッキを1つ選んで戦います。
  • トーナメントの配置を最初にくじ引きで決定します。
  • 大会開始前に自分が使うデッキを申告し、大会を通してずっと同じデッキを使用します。
  • 勝負は、1回勝負です。
  • 試合中の回線切れは試合継続します。
  • 引き分けの場合は再試合とします。
  • 大会中、デッキ編集画面を開くことは反則行為となります。ご注意下さい。

進行

  • 第一回デッキコンテストに投稿されたデッキからデッキを1つ選び、作成しておきます。
  • デッキ名は「デッキコンテスト」とし、デッキリスト内に「デッキコンテスト」という名前のデッキが複数存在しないようにしておいて下さい。
  • 大会運営に使用するデッキを申告し、デッキチェックを受けます。
  • デッキチェックを受けたあとは、デッキ編集画面を開くことは反則行為になりますのでご注意下さい。
  • デッキコンテストのページやデッキレシピを印刷した紙を見ることは可能です。運営のほうでデッキレシピを印刷した紙を持って行きますので見たい方は言って下さい
  • 組み合わせくじが出来たらくじを引きます。
  • 発表された組み合わせに従い、対戦相手を見つけます。相手が見つからない場合は運営に言って下さい。
  • 試合開始前、デッキをすぐに選ばずにデッキ選択画面をお互い見せあい、お互いに確認しあいながら「デッキコンテスト」という名前のデッキを選択します。
  • 試合を進めます。
  • 終了後、主催に勝敗を申告し、勝てば次のトーナメント相手の決定を待ち、初回同様にデッキ選択画面を見せ合ってから次の試合を進めます。
  • 1位〜4位までを決定します。惜しくも負けてしまった人は、進行の妨げにならない範囲で観戦や応援をしても、負けてしまった人同士で自由に遊んでも構いません

注意事項

たまり場(AKIBA トイ&ラウンジ)のWifiを利用できますが、15人くらい同時接続すると不安定になるので、Wifiを使うか自前の回線を使うかの判断は自己責任でお願いします。
