食品サンプル製作体験　in　道具屋筋
(雨天決行です！！)

きもの結び のイベント 「いろいろむすび」 第７弾は、
フードサンプルのお店で、食品サンプル製作体験 をします。
時間があれば、 店舗 のある道具屋筋を散策、その後、お茶でもしましょう❤

＊ 日時・場所＊

６月１２日（日）　１２時 なんばガレリア集合

  • 「なんばガレリア」 :元「ロケット広場」。吹き抜け階段が目印です。
  • 製作体験が１２時半から始まります。
    遅れる方は直接、 Food Sample　Ｒ＆Ｍ さんに来て下さい。
    また、その際は contact@kimonomusubi.comに必ずご連絡下さい。

＊参加費＊

製作体験費

  • Ａコース：（レタス・天ぷら３品）￥1,575
  • Ｂコース：（レタス・天ぷら５品）￥1,980
    （ＡコースかＢコースかは、当日　各自で決めて頂きます。）

別途、交通費・お茶代等は各自お支払い下さい。

＊その他＊

  • 服装について
    1. 汚れる可能性が考えられますので、汚れてもよい着物でお越しください。
    2. 洋服での参加も、もちろん大歓迎です！！
    3. 着物の場合、袂が邪魔にならないよう、必ず　たすき、又は割烹着をご持参下さい。
    4. エプロンは貸出して下さいます。ただし前が隠れるだけの物です。
  • お子様について
    1. お子様は小学１年生から体験可能です。
    2. 小学生未満のお子様については同伴可能ですが、参加者様の責任の下、
      安全に気を付けて下さいますよう、お願いいたします。

  • 参加ご希望の方は、このページからご登録ください。
  • 親子でご体験される場合（又はお子様のみ、保護者様のみ）は、その旨ご記入ください。
    操作がわからない方は、 “きもの結び” contact@kimonomusubi.comまでお問い合わせください。
