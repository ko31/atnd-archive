title 「concrete5 公式活用ガイドブック」活用ワークショップ
catch 「ブロック型」の CMS、concrete5を使ってみよう！
description

3月2日（日）に、「concrete5 公式活用ガイドブック」の著者である concrete5 Japan の菱川 拓郎さんをお迎えして、concrete5 のインストールからサイト制作まで行うワークショップを行います。

concrete5 は、WordPress のようなブログ型ではなく「ブロック型」の CMS（コンテンツ管理システム）です。

編集の自由度の高さや使いやすさ、デザインカスタマイズの容易さ、機能拡張の柔軟さなどが注目され、ユーザー数を伸ばし続けています。

このワークショップは、「concrete5 公式活用ガイドブック」をお持ちの方であれば無料でご参加いただけます。
本をお持ちでない方も、会場で冊数限定で割引販売しておりますので、会場でご購入いただけます。

※会場での購入希望の方は、2月25日までにお申込みが必要です。会場では ¥3,000 で販売いたします。
※書籍は、Kindle 版をご持参いただいても大丈夫です。

ワークショップの内容

このワークショップでは、HTML, CSS, Javascriptで制作したWebサイトをもとに、concrete5 を使用したWebサイトを制作します。

HTMLモックアップ
http://book.concrete5-japan.org/mockup/

concrete5サイト
http://book.concrete5-japan.org/concrete5/

ご参加頂ける方

・HTMLとCSSが自力で書ける方
・ノートPC（MacBookまたはWindows）をお持ちの方
・「concrete5 公式活用ガイドブック」を既に購入されているか、会場で購入される方
