シャーマニズムワークショップ/ベーシック・コース

1Day Workshop / Basic Course

シャーマニックサークル　イーグルトライブ

初めてシャーマニズムに触れる方に合わせた、シャーマニズムの基礎となる技法を体験するワークショップです。
このワークショップでは、シャーマニズムの技法を通して私たちが忘れてしまった感覚を取り戻します。
これまでとは違う世界の捉え方が身に付き、これから続く旅の大きな糧となるでしょう。

シャーマニズムの概要から技法の実践までを、集中して体験していただけます

    3. シャーマニック・ジャーニー　～アッパー・ワールドへの旅 / ティチャーに出会う～
    4. シャーマニック・ジャーニー　～パートナーのための旅～
