【ここがポイント】

Java 11 が正式リリースされました。本バージョンはJava 8 以来のLTS(長期サポート)対象なので、企業で使って行く中で今後重要なバージョンになっていくことが想定されます。
また、今月は世界最大のJavaイベントがサンフランシスコで開催されます。そこでの注目セッションのレポートなどもきっと聞けるはずです。
ということでJDKの細かい新機能の話からイベントレポートまで幅広くキャッチアップしたい人はぜひ参加してみてください。
解説：ANKETテクニカルアドバイザー　吉田真吾氏
