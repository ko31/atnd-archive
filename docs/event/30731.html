【講演主旨】
　 Smart PhoneはPC並の機能を有しており、技術的にはさまざまな可能性が考えられている。しかし、各社類似のハードキーをほとんど持たないフルタッチのパネルを有するSmart Phoneは、「デザインでの差異化」を図ることが従来品以上に難しくなっている。このような背景の中でデザイン的な差異化を図るため「薄型化」が重要なTrendとなっている。その実現には、タッチ機能付き表示パネルを薄型化・筐体の薄型化技術などが必須である。さらに、Smart Phoneの市場拡大にともないCloud Computing環境が本格化し、その結果Camera Module Businessにも大きな変遷を強いる状況になっている。
　本講演では、Smart Phoneの市場拡大に伴うTrendの変遷に対し、Camera Moduleが技術的・ビジネス的にどのように変化していかねばならないかを説明する。
