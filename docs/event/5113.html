■1st Session [藤本壱 (ふじもと はじめ) ]

「Movable Type 5のテーマ機能を活用する」

Movable Type 5の新機能の1つに、テンプレート等をパッケージ化する「テー
マ機能」があります。非常に便利で協力な機能ですが、まだ十分に使いこなされ
ていないようです。
そこで、テーマの基本から、テーマの設定ファイルを手動で書き換えるやや高
度な手法まで、テーマ機能の活用について解説します。

【略歴】
1969年4月生まれ。兵庫県伊丹市出身。
1992年4月に神戸大学工学部電子工学科を卒業。
パッケージソフトメーカーに就職後、フリーライターとして独立し、現在に至る。
MovableType5実践テクニック著者


■「私も出来たよMovableType」
six apart PronetパートナーによるMovableTypeについてのトークディスカッション


MovableTypeに関わらずCMSに関わる人は必聴コンテンツ。
MovableType導入のきっかけや、みんなが聞きたくてもなかなか聞けない現場の生の声を聞いてください。

■2nd Session[野田 純生 (のだ すみお) アルファサード有限会社]

「Movable Type on Business」〜地方の小規模組織が勝ち抜くWeb構築ビジネスの戦略と施策〜

『つーか、マーケット的には東京中心になるのは止む終えないわけだけど、むしろ地方でそれをやるってところで何かが巻き起こせたらなぁと思う。だから僕は今回敢えて東京で話さないことを広島で話そうと思った。』

【略歴】
2003年 11月アルファサード有限会社 設立。
アクセシビリティを重視したWebサイト企画・設計・制作を行うかたわら、Webアクセシビリティを支援するフリーソフトウェア等を公開。
Movable TypeやWebアクセシビリティに関する講演、セミナー、書籍の執筆等を行う。
