description

仕事で使うTouchDesignerというテーマで行うワークショップです。

TouchDesignerに慣れてもらい、仕事で使えるようなフォーマットを作成し、
映像ソフトとの連携などを説明します。

ユーザーインターフェースの説明から、TouchDesignerでのプログラミングを簡単に説明し、付属のサンプルファイルを用いて、幾つかのフォーマットを組んでいきます。

例えばこのようなものになります。
フルスクリーン表示（インスタレーション・デジタルサイネージ用途向け）
ユーザーインターフェースの作成（ボタン、スライダーなど）
VJソフトとの連携（Syphon/SpoutやOSCを使って）

参加は20名までとなっていますが、見学だけというかたは10名程受け付けます。
その場合も参加費は発生します。

■参加に必要なもの
TouchDesignerがインストールされたPC（Windows / Mac）
TouchDesigner099（ベータ版を使います。正規版の088ではありません！）
https://www.derivative.ca/099/Downloads/Default.asp

3ボタンマウス（必須です）

PCのスペックはこちらに記載されています。
http://derivative.ca/wiki099/index.php?title=System_Requirements

インストールについては、こちらをご覧ください。
http://derivative.ca/wiki099/index.php?title=Install_TouchDesigner

インストール後、登録が必要になりますので、
以下のページで必要事項を記入してID/パスワード設定をしてください。
http://www.derivative.ca/Login/RegisterForm.asp

■スケジュール（休憩は適宜挟みます）
13:00〜13:30 イントロダクション
13:30〜15:00 TouchDesignerでのプログラミング
15:00〜16:00 フルスクリーン・フォーマット作成＋簡単なインスタレーション制作
16:00〜17:00 付属のサンプルプログラム説明
17:00〜18:00 他ソフトとの連携説明（Syphon/Spout/OSCなど）
18:00〜19:00 実習＋検証

■懇親会
