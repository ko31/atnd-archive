　お馴染み『怪獣酒場』の予約を取る事ができました。

　　　　　　（時間制限付き／オーダーストップは 18:30 になります）
　開催場所：怪獣酒場（川崎駅東口地下商店街「アゼリア」直結：ただし階段あり）

　集合場所：店舗写真の右手、エレベータ側の壁付近
　参加条件：年齢、関係ナシ（未成年でも「席料」取られるけど「おみやげ」付き）
　総低予算：大人 5000 円
　　　　　　中人 3000 円（中学高校生）
　　　　　　小人 1500 円（小学生）
　　　　　　幼児 1000 円（未就学児）

　　　　　　子供を連れて参加される方が多いので、ご遠慮いただければと（汗）

　　　　　　ただし、店舗の入り口に段差があり、予約席は座敷なので要注意
　参加条件：「ウルトラ」シリーズの怪獣が好きじゃないと面白くないと思う
　　　　　　年齢制限はありませんが「未成年は必ず保護者同伴」でお願いします

　おねがい：参加登録時のコメントで「同伴：〜名」と入れて頂けると助かります。
