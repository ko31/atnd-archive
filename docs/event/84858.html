　購買履歴や仕入履歴、来店履歴、営業履歴、WEB上でいうならアンケートや閲覧履歴など、さまざまなビジネス行動データを安価に残せるようになったことにより、これらのビジネスデータを上手に活用することに注目が集まっています。これまでマーケティングとよばれていたユーザのためのビジネス改善活動に対して、現在、より多くの量と種類をもつデータへの適切な処理が求められています。〜2014年度　弊社・データ活用セミナーより〜
-———————————————————————————————————————————————-

【概要】

・今回のセミナーは2014年に弊社・里洋平が話したものとほぼ同じ内容です。

・ビッグデータを取り巻く環境は２年で変わった部分もあれば、ほとんど変わっていない核心もいくつかあります。

・これからデータ活用に本格的に取り組みたいと考えている方を対象に、これまでのいくつかの事例をご紹介します。

・セミナー中に適宜、質問時間を設けます。終了後、直接お話ししたいという方もお待ちしております。

【参加対象】

ビッグデータに関心がある（が活用のために何からはじめてよいのかわからない）方
データをすでに活用しているが、他の事例をお知りになりたい方
※受付時にお名刺を１枚お渡しください

【会場について】
DATUM STUDIO ８階会議室にお越しください。ビルの１階はメインオフィスとなっているのでご注意ください。

【講師】

里洋平

R言語の東京コミュニティTokyo.Rの主催者。ヤフー株式会社で、推薦ロジックや株価の予測モデル構築など分析業務を経て、株式会社ディー・エヌ・エーで大規模データマイニングやマーケティング分析業務に従事。 その後、株式会社ドリコムにて、データ分析環境の構築やソーシャルゲーム、メディア、広告のデータ分析業を経て、DATUM STUDIO株式会社を設立。
