[ 今回のテーマ ]：「失敗したプロジェクト」。
属社的な話しになりそうですが、そこはグッとこらえて何故プロジェクトは失敗したのかを分析し、様相を並べ共通点を探りたいと思います。
勿論、実装レベルにまで還元して、OOPと結びつけると具体的で良いのではないかと思います。
