オンラインゲームを作ってはみたい。

でも「でも何からはじめていいかわからない！」「サーバーを設置するのが手間だし、知識もない。。。」

そんなハードルを一気に解消するのが Photon Cloud

今回のワークショップでは、 Unity と　Photon Cloud を使い、この日のワーク時間内で簡単にオンラインゲームを作ってしまいます。
