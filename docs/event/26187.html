description

楽しかったり、美しかったり。
そんなことを感じさせてくれるものを、
見たり、聞いたり、知ったりすると、
ついつい誰かに伝えてみたくなってしまいます。
感じたことを共有し合うと気持ちが良いものです。

2012年3月24日の東京で、出会った人々と一緒にアートを散策してみませんか。

「Sharing Walk」は国籍関係なくその日に出会った人々と一緒に散歩をし、
そこで感じたことを共有し合うイベントです。
今回はアートに焦点をあて、「TOKYO SPRING」を楽しみます。

TOKYO SPRINGは、東京の春を盛り上げるクリエイティブ・イベントのプラットフォームです。
http://tokyospring.jp/
