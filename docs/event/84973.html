内容

Pythonを使用して、機械学習による英語ツイートのポジネガ分類を行います。
今回は、機械学習モジュールとして、Pythonのsklearnを使用します。
分類器にはSVMを使用します。

[注意]

講座会場にはネット環境がございません。予めご了承ください。

受講する方は、以下のものが使用できるPCをご持参ください。
　　python3
　　numpy
　　scipy
　　sklearn

本講座では，環境ごとの不具合を考慮し，英語ツイートを使用します。
講座に参加される方は以下のファイルをダウンロードしておいてください。
Tweetデータです。
http://fast-uploader.com/file/7040525549700/
pass:0204seminar

日本語の文字コードに対応できる方(mecab等のスキル)は日本語にも応用できますが，本講座ではサポートしません。

デモをお見せしながら、進めていくので、比較的わかりやすい講座となっております。機械学習に興味のある方は、ぜひご参加ください。

本講座の動作確認はVMware Workstation 12 Player + Ubuntu 14.04 LTSの仮想環境で行っております。
こちらは無料で利用できるので，同じ環境をおすすめします。

http://www.vmware.com/jp/products/player/playerpro-evaluation.html
VMware Workstation 12 Player

https://www.ubuntulinux.jp/News/ubuntu1404-ja-remix
Ubuntu 14.04 LTS

上記と同じ環境であれば，ターミナルから以下のコマンドで必要なものがインストールされます。
sudo apt-get install python3-all-dev -y
sudo apt-get install python3-setuptools
sudo easy_install3 pip
sudo pip3 install numpy
sudo pip3 install scipy
sudo pip3 install sklearn
