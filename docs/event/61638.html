ニコファーレで頂上決戦。 CODE VS決勝トーナメント！

相手の隙を衝くアルゴリズム活用力と、的確なコーディング技術を競い合うプログラミング・コンテスト。

今回テーマとして与えられるのは「戦略ゲー」。
資源を集め兵を生産し敵を倒すシミレーションゲームです。
アルゴリズムで超多数のユニットを操り、相手の城を攻め落とす。

ゲームを攻略する「最強のAI」を作り、プログラマ日本一に輝くのは誰だ？？

司会・実況

司会・実況
“きゃんち”喜屋武ちあきさん

きゃんちのニックネームで親しまれる。マンガ、アニメ、コスプレなどを愛するグラビアアイドル、タレント。
作詞家としても活躍中。「CodeIQ MAZAINE」で技術とプログラミングを学び、ギークガールを目指す。

司会・実況
高橋大輔さん
フリーアナウンサー、ラジオDJ、ナレーター

学生時代は、プロゴルファーを目指すも怪我により断念。
クラブをマイクに持ち替えてプレイヤーから伝える側へ。「ハッスル」、「ZERO1」、「シュートボクシング」、
「戦極」など、格闘技・プロレスなどを中心に数々のスポーツ実況をつとめる。
2009年からは、東京ドームの読売ジャイアンツ応援スタジアムDJを担当。

特別ゲスト
池澤あやかさん

女優　1991年大分県生まれ。慶應義塾大学環境情報学部 卒業。
2006年の第6回東宝シンデレラオーディションで審査員特別賞を受賞し、芸能活動を始める。
映画、テレビ、ラジオ、CMなどで活躍する一方、 Rubyが得意な「GEEK系アイドル」として
エンジニアの人気が急上昇し、「Rubyの女神」とも呼ばれる。
現在、雑誌MacPeopleで「池澤あやかのギーク道」連載中。

解説
高橋直大さん
AtCoder株式会社

Imagine Cup 2008年アルゴリズム部門世界3位/TopCoderOpen四年連続決勝進出、準優勝１回/最強最速アルゴリズマー養成講座著者。
「CODE VS 4.0」では決勝の実況解説を担当。
Twitter : @chokudai

解説
田村哲也さん
チームラボ株式会社

2000 年 12 月有限会社チームラボ設立、取締役に就任。主に、新技術の研究開発を担当。
毎年「CODE VS」では、高橋さんと共に決勝の実況解説を行っている。
