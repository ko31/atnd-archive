description

Webを中心とした編集領域の最適化を事業の柱とするアウルが最新のコンテンツマーケティングセミナーを行います。
ソーシャルメディア上でコンテンツが拡散されるかどうかは多くの要因に左右されます。
コンテンツの中身、トレンド、配信方法・タイミング等々。
それらを複合的に検討し、最適化させたときにコンテンツは拡散します。
アウルでは過去1万件以上の実績をもとに導き出したコンテンツ制作・配信のノウハウを持ち合わせています。
どうすればソーシャルメディア上でコンテンツを拡散させられるのか。
そのメソッドをアウルの事例とともにお伝えします。

【アジェンダ】
・なぜ、コンテンツマーケティングが注目されているのか?広告と編集の違い。「ネイティブアド」の役割と効果。
・ここが違う！４大コンテンツレコメンデーションエンジンの使い方と特長。
・フィードランクとソーシャル流入率から読み解くこれからのメディアの選び方。
・人工知能を活用したPRトレンドの読み方、バズるコンテンツの作り方。
・″出た感”に惑わされない、絶対的なKPIの設計とその測り方。
・ABテストで分かったよく読まれるコンテンツに共通する「タイトル」の付け方、「写真」の選び方。

【スピーカー】
瀬戸　大輔
株式会社オプトにアカウントエグゼクティブとして2003年入社。大手企業のWebサイト構築から集客に至るまでWebマーケティングを一気通貫で実施。同社でMVPに輝き、最年少で管理職に昇進。
2006年、取締役 兼 COOとしてアウル株式会社を創業。
WebマーケティングとPRを融合した統合型のマーケティングプランニングに定評があり、イケアをはじめ、ソフトバンク、ネスレなど幅広い企業を担当し現在に至る。

山口　秀夫
ネットレイティングス株式会社(現ニールセン株式会社)に技術者として2000年入社。日本初のインターネット視聴率データ提供システムの開発に従事。マーケティングデータを集積するためのWebクローリング技術やビッグデータ解析を得意とする。2004年よりシステム技術部 部長に昇格しニールセンを10年にわたって牽引。本国アメリカとの窓口もこなすマルチパフォーマー。執行役員CTOとして2015年、アウル株式会社に参画、現在に至る。
