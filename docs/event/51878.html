Code for SAITAMA アイデアソン

Code for SAITAMA のアイデアソンを開催します。今回のテーマは
秋の自転車レースや世界盆栽会議など、内外の観光客向けにガイドマップを
作っていこうという趣旨です。

前提条件

今回はOpenstreetMapを利用した観光客向け地図の作成です。
ベース地図にOpenstreetMapを利用します。
事前にさいたま新都心でマッピングパーティを行い地図データを
増やしています。もちろん、足りないところは補完できます。
