description

この講座シリーズでは、デバッグ実行まで無料で全機能利用できる統合開発環境 Xojo で OpenGL 上で動作するゲームの作成を通じて、アクション／パズル系ゲームのシステムデザインとゲームデザインの基礎を学習します。
第１回目として、まずは画面上にオブジェクトを表示することから始めます。OpenGL の一番の基本となる、オブジェクトの表示、タイムループ管理などを取り扱います。

・マルチプラットフォーム統合開発環境 Xojo は Windows、Mac OS X、Linux の各OS上でご利用いただけます。ビルドをしない限り、デバッグ実行まで全機能を無料でご利用いただけます（ビルドにはライセンスの購入が必要です）。セミナー開始までにあらかじめパソコンにインストールされることをおすすめします。
