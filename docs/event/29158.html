description

　ＮＴＴドコモなど携帯3社のエコ（eco)検定対策の公式サイト「エコ 実践塾」の運営などで環境教育に取り組んでいる「環境教育フォーラム21」は、第12回eco検定の合格対策に、東京・新宿で６月８日（水）と１３日（金）、大阪市で６月１５日に「eco検定帰宅前のポイント学習講座」を開催します。昼間は仕事でお忙しい社会人や大学生の皆様に帰りがけにeco検定合格の重要ポイントを習得していただきます。
　実質３時間強の学習ですが、eco検定試験の対策、公式テキスト第３版のポイント、eco検定にでやすい環境時事問題の解説をコンパクトに説明します。講習後のご自宅での学習に役立つ豊富な資料を用意しており、仕事でお忙しい皆様の効率的な学習に役立ちます。概要は下記の通りです。

６月８日（金）、13日（水） 東京・新宿文化センター
時間:午後6時30分から午後10時00分
住所:交通:新宿区新宿6-14-1(JR、東京メトロ、都営地下鉄など最寄り駅から徒歩数分)

６月１５日（金） 大阪市総合生涯学習センター
時間:午後6時15分から午後9時30分
住所:大阪市北区梅田1-2-2-500 大阪駅前第2ビル(JR、地下鉄、私鉄最寄り駅から徒歩数分)

参加費:5000円
