水響亭平均動員数200名～300名様前後

メディア掲載実績多数！銀座の地下に広がる海底美岩城。

エントランスを入ると壁一帯に映し出される海の映像が眩しく照らし続けています。

そして、長いスロープをメインダイニングへ進むまでの間に
「こんな素敵なところあったんだ」と思わずにはいられない。

地底に潜った幻想的空間は、天井が高く、まるで海の底にいるような別世界の空間へ引き込まれます…

水が流れる通路を歩き、テーブルの真横にも水、水、水！
そこは、まるでディズニー映画のように主役を引き立てる幻想的な空間。

会場全体に広がる水槽の熱帯魚達が今にも歌いだし、踊りだし、パーティーを一層に盛り上げます！

ただただ素敵な出会いをご想像頂き、楽しみにお越し下さい♪
