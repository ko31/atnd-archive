2012LET中部秋二次会

description

2012年度外国語教育メディア学会中部支部秋季研究大会（＠名古屋大学）の夜に、学会非公式の二次会を開催します。
　
概要

ツイッターといったSNS上で今年のLET中部秋にお越しなる先生・学生達で懇親会の後に二次会を催したら、というお話がございました。つきましては、
以下の要領で学会非公式ながら二次会を開催させていただきたく思います。是非中部地区のみならず、全国的な語学教育に関する学術的交流の場としてご利用頂きたいと思っております。

タイムテーブル

時間 予定
19:30頃 学会懇親会よりなだれ込み（草薙がご案内します）
20:00 – 21:30 二次会
21:30 解散

場所

萬鳥商店

名古屋大学近くのお店（名古屋市営地下鉄、名城線・東山線、本山駅すぐ近く、学会場より徒歩5-10分）を予定しています。会場から近い上にお帰りは栄や名駅まで乗り換えなしの好アクセスです（福田君チョイスです）。

料金

おひとり様3000円でよろしくおねがいします。

ご参加の方法

原則として、このサイトにて11/29までにご予約いただくか（右上の「参加登録」、申し訳ありません。ツイッターなどのアカウントが必要となります）、または直接、ツイッターやemailでご連絡頂きたく思っております。ご連絡先は名古屋大学国際開発研究科の草薙邦広(@KSNG92 )か、email: kusanagi [at] nagoya-u.jpまでよろしくお願いいたします。
