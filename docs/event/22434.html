今年も Advent Calendar の季節がやってきました。

Advent Calendar が何なのかは ここらへん を見ていただくとして、Java をネタにしたブログを公開してしまいましょう。

12 月 1 日から登録して、人数が多ければクリスマスがすぎてしまってもかまいません。ぜひ、みんなでブログを更新しましょう。

Java といえば、サーバーからクライアント、クラウドに組み込みと、さまざな分野で使われており、そこで使われている技術もたくさん。

Java の Tips でもいいし、ライブラリの使い方でもいい、自作のツールの紹介など、Java に関していれば何でもかまいません。ぜひこの機会に、ブログ書きましょう。

ブログ書いたら、コメント欄にブログの URL を書いてください。ここを見れば、すべてのブログを見られるようにしましょう。

最後ですが、いちおう… Sponsored by JJUG です！！

クリスマスを過ぎたとしても全然 OK。正月を過ぎてしまっても、全然かまいません。補欠になっている方もぜひブログ書いてください！！
