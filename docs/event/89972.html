TECH VALLEY(テックバレー)とは？

ギークスが送る、「エンジニアの好奇心を満たす出会いが見つかる」をコンセプトにしたイベントです！

東京にて大反響を頂いているこのイベントですが、福岡でも6月に第一回目を開催し、大いに盛り上がりました！

そして、好評につき…早くも第2回目を開催いたします！

今回もまた、新しい情報、新しい働き方、そして新しい仲間との出会いを、

リアルコミュニケーションを通して、ご提供します。

今回のイベントについて

「エンジニアにとってのキャリア選択～フリーランスとかってぶっちゃけどうなの？～ 」と題し、

フリーランスを検討している方や、キャリアに悩んでいる方の疑問を、このイベントで解決しちゃいます！

お呼びする2名のゲストの方に、ご自身の体験談や、他では聞けないような「ここだけの話」

などもお話していただきますので、これからのキャリアを考えることができる良い機会です。

特に

・フリーランスエンジニアになろうかと思っている方

・今は具体的に考えていないけど、今後のキャリア選択の為に色んな働き方の話を聞いておきたいという方

にはピッタリです！

是非ご参加をお待ちしています。

講演会後は、交流会も行いますので、軽食を片手に、皆さん同士で情報交換していきましょう！

登壇者と講演テーマに関して

株式会社スカイディスク 　最高技術責任者CTO　大谷 祐司氏

株式会社スカイディスクの技術部門を管掌。
サイバーエージェントでインターネット広告のシステム部門を立ち上げ、子会社の技術責任者に就任。
その後インテリジェンスで技術責任者としてエンジニア組織を立ち上げる。
2017年1月より福岡に移住し、CTOとしてスカイディスクに参画。

フリーランスエンジニア　佐藤誠氏

フリーランスのインフラエンジニア。
派遣会社に社員として11年、フリーランスとして1年、計12年のエンジニア暦をもつ。
某大手派遣会社に入社後、広島に勤務するも、すぐに東京へ転属となり、主に東京で活動。
12年の経歴うち、前半は主にネットワーク、後半はサーバ、仮想化基盤の構築がメイン。
そして、一念発起し、2016年にフリーランスへ転向。
現在はギークスの紹介案件にて、金融系のインフラ案件に携わる。
なお、趣味で運用しているブログは月間20万PVを超え、副業での収入も得ている。

登壇企業情報

株式会社スカイディスク

福岡にスタートアップ企業として、2013年10月に設立。
IoTサービスを簡単に実現するための、デタッチャブル（着脱式）センサと、
そのセンサを取り扱うためのクラウドサービスを展開してる。
また、AIによる様々なデータ分析や、業界では初となる低消費電力・低価格な無線通信規格「LoRa」
を採用した通信環境の構築を手掛け、IoTのソリューションに欠かせないサービスをワンストップで提供している。

ギークス株式会社

2007年に、フリーランスエンジニアの支援事業であるIT人材事業で創業した、総合インターネットカンパニー。
案件の紹介からフリーランス向け福利厚生プログラム「フリノベ」の提供など、
フリーランスエンジニアに向けた幅広いサービスを提供している。
IT業界の盛り上がりを背景に、2016年に福岡支店を立ち上げる。
運営メデイアにフリーランス専門案件検索サイト「geechs job（ギークスジョブ）」や
ITクリエイターのための気軽な情報収集メディア「geechs magazine」がある。
