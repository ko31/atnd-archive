description

みなさん、Apple Watchはもう手に入れましたか？周りの人に話を聞くと『まだ買う気にはなれない』『買う気はない』『時計に5万は・・・』『買っても1年で買い替えでしょ？もったいなくて買えない』などイマイチな反応。ん？こんな状況、今までにもあったぞ？
あーそーだ、iPhoneの発売のときじゃありませんか。『オサイフケータイ機能も付いてないし』『テレビ見れないし』
うーん、そのあとはご存知の通り。

それに、よく聞いてみると、Watch使っていない人の声ばかりだぞ？Apple Watchは、本当にそんな評価なのか？
ならば、使っている人の話を聞いてみよう！
使っている人の話は含蓄があるもの。

iPhoneので始めの頃、その良さが伝わるかどうかの頃、早くアプリ作りに手を出した人が大儲けしてるじゃないか。食わず嫌いよりも食ってから文句を言うのもありじゃないか？（人のApple Watchを前にして文句はキレられるので注意ね！）

というわけで、買った人からその魅力を聞いたり、Watchアプリを作った人に作ってみた体験談を聞いてみましょう。そして、来たるべく、新しいWatchのステージを前に可能性を夢見てみましょう！（あるのかはわからんけど・・・いやきっとある！）

また、今回は、SpriteKitの書籍の著者陣も発表者として登壇されます。

『SpriteKitではじめる2Dゲームプログラミング Swift対応』
著者:山下佳隆、村田知常、原知愛、近藤秀彦（敬称略）

よりディープなSwift開発の話を聞きたい方にもオススメの内容です！リクエストのあったWebAPI関連の話もあります。

そんで、夜は楽しく懇親会で美味しいもの食べたり飲んだりして、アプリ開発の未来を語りましょう！発表者の裏話や昼間は話せない話はこちらで聞けるかも？（発表者もほとんど参加予定。初参加の方も是非！）
