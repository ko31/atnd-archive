テレビチャンピオンのラーメン王選手権優勝者 @cova1026 とコラボしておおくりする第一回グルメ謎解きイベント『ラーメンハンティング〜幻のラーメンを見つけ出せ！〜』

謎解きあり、
007並（！？）のミッションあり、
ラーメン王おすすめラーメンあり！
ラーメン王とのラーメン談義あり。

ラーメン好きな人、謎解き好きな人、体動かしたい人、ビビっときたら、ぜひっと参加をー。

＜STAGE1＞
●概要：
この日1日だけ、みなさんは隠密行動をとる忍者に変身します！合言葉はいつだって「ニンニン！」

味方が見つけた幻のラーメン。しかし、そこにはあなたを行く手を阻む見えない敵がうじゃうじゃ！
時には敵から身を隠し、時には俊足を飛ばし、時には仲間からの暗号を解き明かす！
渡された隠密書をもとに、すべての指令をクリアし、目的地を探し出した人だけが幻のラーメンの在り処に辿りつけるのです！

●開始時刻：12月10日 20：00
●集合場所：JR 高田馬場駅 早稲田口
※20時以降に来られる方でも、到着次第はじめることが可能です。帰宅できるのであれば何時からはじめても構いません。
●料金：食べたいラーメンの料金＋200円くらい
●イベントの流れ：
・イベント参加ボタンを押した後、miyoccoアットgmail.com送られてくるメールを待ちます。（twittetでフォローしてくれている方はDMの場合もあります）
・高田馬場駅 早稲田口に行きます。
・事前に受け取っている「隠密書」（DMかメール）を読みます。
・指示に従ったり、指令をこなしたりします。
・最終的にラーメン王おすすめのラーメン屋を見つけます。
・好きなラーメンを食べます。（終わり・・・？）
●注意事項：2人以上の参加を推奨します。（ひとりでもできなくないですが、ちょっとたいへんかも）5人以上で参加するときはちょっと時間を置いて、2つの班にわかれるといいかもしれません。

＜STAGE2＞
●概要：
お腹がいっぱいになったあとには、ラーメン談義に花を咲かせましょう♪
高田馬場駅から徒歩8分くらいのところに私の家があるので、そこで、プチ飲み会しまーす。

「ラーメン王」がみなさんをお待ちしておりますよー。ラーメンについて聞きたいことがあったり、自分ちの近くの美味しいラーメン屋さんを知りたかったりする人はぜひぜひご参加をー
（強制参加ではありません。ラーメン食べ終わって時間がない人は帰宅しても大丈夫です）

●開始時間：ラーメン食べ終わり次第順次
●終了時刻：みんなの終電まで
●場所：ご存じの方は各自。ご存じない方は、ラーメン屋につけばわかります！
●定員：20名くらい（早い者勝ちだーい）
●予算：1000円（お酒とおつまみ代）
※高校生はタダ。大学生は500円で。
