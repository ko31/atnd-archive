『Ogaki Developer Day 2014 Summer』の宿泊施設利用申し込みATNDです。

今年から貸し切りとなりますので、20名は泊まってほしいです。
料金は20名を想定した料金になっていますので、それ以上宿泊があると安くなります。
宿泊申し込みが増えてきたので、もう少し増えるとお安くできそうです。

・利用する宿泊施設は、ソフトピアジャパンの敷地内にある、 ソピア・キャビン です。
・料金は、Mタイプ（2,000円：シャワー、トイレ共用)とLタイプ(3,000円：バス、トイレ完備)の２種類から選べます。
　※Lタイプ（3,000円）の部屋を希望の方は申込に”Lタイプ”と記載してください。
・Lタイプは全１０室のため、先着順とさせてていただきます。
・タオル、歯ブラシ、パジャマみたいな服は用意されています。無線LANも利用可能です。
・チェックアウトは10:00です。
・こちらで申込いただいた方の宿泊申し込みはこちらで行います。
