「Perl入学式って?」

「プログラミングに興味があるけど, ちょっと難しそう…」という貴方!
「他の言語使いだけど, ちょっとPerlも使ってみよっかな?」という貴方!
「仕事や研究でPerlを使い始めたけど, ちょっと自信ないな…」という貴方!

プログラミング未経験者からPerl初心者を対象としたワークショップ「Perl入学式」で, 一緒にPerlで学びましょう!

どんなことをするの?

Perl入学式は1月から12月まで, 毎月1回開催する予定です.
簡易版twitterのような｢一言掲示板｣の完成を目標にして, その為に必要な知識を学んでいこうと考えています.

今回の｢Perl入学式 #1｣では, あなたのPCでPerlを実行する為の環境を構築します.
OSにWindowsを使っている方は, “VMware Player”という無償で使えるソフトウェアを使って, こちらもまた無償で使えるLinux系のOSである"Ubuntu"を導入します.
そのあとは, Ubuntuの簡単な使い方や, コマンドラインの基本的な操作方法などをチャレンジします.

参加条件

基礎的なパソコンの操作(クリックやドラッグといったマウスの操作, キーボードでの文字入力など)ができる方なら, プログラミング経験やPCに関する知識の有無は一切問いません.
ただし, ｢Perl入学式｣では, 毎回ご自身のPCを使って, 実際に手を動かしながらPerlを学んでいこうと考えていますので, 毎回必ずご自身のノートPCを持ってきて下さい.

PC貸出制度

｢自宅にデスクトップPCがあるけれど, ノートPCは持っていない｣という場合, 当日会場限定でノートPCをお貸しすることができます. 何台お貸しできるかは毎回変わります(1台もお貸しできない場合もあります)ので, 参加登録をする前に, 貸出可能かどうかを papix2011@gmail.com に確認して下さい.

お願い

“どんなことをするの?”にも書きました通り, #1ではPerlの実行環境を構築する為に, Windows系OSをお使いの方は, “VMware Player”というソフトウェアを用いて, "Ubuntu"というOSを導入します.
これらのダウンロードには時間がかかりますので, もしよろしければ事前にダウンロードしておいていただけますようお願いいたします.

“VMware Player 4.0.1”ダウンロード先:
http://downloads.vmware.com/jp/d/info/desktop_end_user_computing/vmware_player/4_0

“Ubuntu 11.04”ダウンロード先:
http://www.ubuntulinux.jp/News/ubuntu1104-desktop-ja-remix
※”CDイメージ"と書かれているリンクをクリックしてダウンロードを開始してください.

既にLinux系OSを導入済みの方, Macをお使いの方(OSがMac OS Xの方)は, ダウンロードしておく必要はありません.
また, ｢どれをダウンロードすればいいかわからない｣, ｢どうしたらいいかわからない｣という方は, ダウンロードして得られるファイルを入れたUSBメモリをこちらで用意しますので, ご安心下さい.
