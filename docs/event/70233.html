【新規限定】【学生限定】 Pepper 開発体験 ワークショップ（SDK 基本編 #1 -喋る、動かす、会話する）

Choregraphe 2.0.5をインストールしたノートパソコンをご持参ください。今回学生のみの対象とさせていただきます。

【初心者向け】

Pepper の概要と*SDK「Choregraphe（コレグラフ）」*の説明を行います。
Pepper で開発してみたいデベロッパーやクリエーター方向けの基礎的なワークショップです。

直感的に開発ができる SDK を利用するため、プログラム経験のない方でも受講できます。

このワークショップでは Pepper の以下の機能を体験いただけます。

  • しゃべる
  • 動かす
  • タッチセンサーに反応して動かす
  • 言葉を認識して動かす

【Choregraphe の事前ダウンロードについて】

Pepper の開発に必要な Choregraphe は事前にダウンロードし、PCにインストールした状態でご来場ください。

Choregraphe は「デベロッパープログラム」に登録することで無料でダウンロードできます。

デベロッパープログラムの詳細

※Choregraphe 2.0.5 に対応した Pepper をご利用いただけます。インストールする Choregraphe のバージョンも2.0.5 （Software Suite 2.0.5 (Pepper)）としていただけますようお願いします。

　ダウンロードURL は こちら です。
　

【ご注意】

  • 多くの方に Pepper の開発を体験していただくため、ワークショップを受講経験のある方はキャンセルさせて頂く場合があります。
  • ワークショップの様子を撮影し、ブログなどの媒体で公開する場合があります。撮影がNGな方はスタッフにお声掛けください。
