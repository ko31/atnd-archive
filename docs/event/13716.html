【プログラム】
◆講師の言葉
　設備・装置の維持管理や製品の耐久設計において、金属材料の腐食の抑制・防止が重要な課題とされています。腐食損傷の中でも、応力腐食割れ (Stress Corrosion Cracking、SCC)は、潜伏期間が長く一旦発生すると破壊的なダメージを与えるため、対処が難しい腐食形態といえます。
　本講では、応力腐食割れに対処するために必要な金属腐食の基礎について述べます。次に、応力腐食割れにおける材料・環境・応力の観点からの要因と、発生・成長メカニズムを解説します。また、実務経験に基づいて、評価試験や損傷調査の方法と事例への応用を紹介し、防止対策とその適用例を紹介します。

◆修得知識
・応力腐食割れに関して、「実務レベル」で対処するための基本知識。

◆プログラム
Ⅰ. 金属腐食の基礎
　　1.金属の腐食形態
　　2.湿潤腐食とガス腐食の違い
　　3.全面腐食と局部腐食の違い
　　4.局部腐食の種類と要因

Ⅱ. 応力腐食割れのメカニズム
　　1.応力腐食割れとは
　　2.応力腐食割れの要因とメカニズム
　　3.応力腐食割れと水素脆性の比較

Ⅲ. 応力腐食割れの試験と調査　
　　1.各種の応力腐食割れ試験の長所・短所
　　2.応力腐食割れ損傷の調査方法
　　(各種の試験片および損傷品を使用)

Ⅳ. 各種金属材料の応力腐食割れ事例
　　1.炭素鋼の応力腐食割れ
　　2.ステンレス鋼の応力腐食割れ
　　3.非鉄金属の応力腐食割れ

Ⅴ. 応力腐食割れの防止対策と適用例
　　1.材料からの対策
　　2.環境からの対策
　　3.応力からの対策
