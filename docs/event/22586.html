description

DevCafeとは

自分が作りたいものや、現在開発中のものをもくもく開発していきます。
最後に何か成果があれば発表をします。
カフェでもくもく開発しているのを、みんなで集まってやるという感じです。

本気でもくもく開発してもいいし、
誰かと雑談交じりに和気あいあいと開発するのもOKです。

他の人がどんなことしてるのかを見てみたり、
詰まっているところを誰かに教えてもらったり、
新しいお友達ができたり、などなどが期待できます。

一応Androidですが、じゃなくてもOKです。
また、お昼からとか途中参加でも問題ありません。

初心者歓迎です！
開発環境まだって方でもこれを機に整えてしまいましょう！

諸事項

  1. 吉田先生のご好意で貸して頂きました！
    ありがとうございます！
  2. ネット環境が無いので、必要な方は各自ご用意ください。
    僕の方で1回線取れれば、無線で5台まで拡散できます。
  3. 電源あります。タップがあると安心かもしれません。
  4. お昼からなど開始時間に間に合わない方は、参加登録のコメントに一言書いて頂けると助かります。
  5. お昼ご飯は各自自由に取って頂いて構いません。
    お勧めは「 一悶屋 」というラーメン屋さんです。
    いつもここですけど、ここの塩ラーメンがうまいんですよ。
  6. 飲食OKです。ペットボトルなど、ごみの分別にご協力ください。
  7. 片付け後の退出は速やかに行おうと思っています。ご協力お願い致します。
  8. 主な持ち物としては以下になります。(11/27追記)
    ・開発用ノートPC(必須)
    ・タップ、延長コード(推奨)
    ・ネット環境(推奨)
  9. 費用はかかりません。(11/29追記)

その他何かあれば、またここに追記していきます。
