description

OpenStreetMapというみんなで作る地図を描く方法を経験者とともに体験するイベントです。千葉県成田市の成田山新勝寺に向けて、かつての成田街道を７kmほど歩きます。経験者・未経験者を問わず歓迎致します。お気軽にご参加ください。

成田山新勝寺は不動明王をご本尊とし、平将門を調伏するために寛朝大僧正が東国に遣わされたことがきっかけで開山されました。江戸時代に当時の人気役者市川団十郎が信仰したこともあり、江戸をはじめとする近郊から３泊４日程度の手軽な参詣ルートとして人気を博しました。参詣者は年間２００万人ほどいたそうです。
江戸からの経路は複数あったようですが、今回はそのひとつをごく一部だけ、酒々井からゴールの新勝寺までの約７kmを歩きます。古道/旧道に興味ある方は事前に農研機構さんの 農業土地利用変遷マップ で明治初期の地図（迅速測図）と現在の地図（国土地理院の基盤地図）を、透明度を変えたり位置を合わせながら見比べて当時の成田街道の経路を推定しておくと、当日歩きながらの検証をより楽しめると思います。
お好みでどのようなPOI（興味深い対象物）をチェックしても構いませんが、今回は特に経路検証の手がかりとなる石碑や古い建造物に着目します。
