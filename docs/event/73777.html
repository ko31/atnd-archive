description

環太平洋パートナーシップ（TPP）交渉が合意に達しました。TPPが成立すれば、世界の約4割のGDPを占める巨大な自由貿易経済圏となります。このようなTPPの動向も含め、海外における「売れ続ける仕組み」を企画する力はますます重要性を増しています。 本講座は、主に海外取引に携わる方を対象に、マーケティングの基本とその策定プロセス、および海外進出日系企業のマーケティングにおける成功要因を理解し、海外における「売れ続ける仕組み」つまり海外進出マーケティングの基礎知識を習得することを目的としています。
マーケティングの基礎と戦略手法を学んだうえで、実在企業の海外でのBtoC（消費者向け）マーケティング事例を用いて、個人演習やグループ演習を通じて成功要因を分析します。分析から得た知識やポイントを自社製品に適用することで、社内会議や海外顧客との商談時に活用できるマーケティング理論を踏まえた企画分析スキルの習得を目指します。

■講座で学べる内容　
１．マーケティングとは
２．環境分析
演習①環境分析事例演習「スズキ自動車のインド進出」
３．マーケティング戦略（セグメンテーション・ターゲティング・ポジショニング・マーケティングの４Pなど）
演習②マーケティング戦略事例分析演習「ユニ・チャームのインドネシアマーケティング戦略」
演習③マーケティング戦略事例分析演習「ヤクルトの中国マーケティング戦略」
４．まとめ
