実施(実習ワーク)で学ぶ！Webサイト制作におけるマーケティングとプランニングの極意

セミナー内容-OUTLINE

技術や社会の変化にともない、あらゆる市場に激動の時代が訪れています。中でもWeb制作の業界は、日々情報をキャッチしてアウトプットすることが求められるとりわけ変化が激しい市場です。そこで、現在、Webサイト制作の現場で必要とされているのは「マーケティング」に基づいた「プランニング」による本質的な企画力や提案力です。

今回は、知識やノウハウのインプット学習による講義と、実際に体験しながら学ぶ実習ワークを一部交えながらお話しします。

こんな人におすすめ

・webディレクションの基礎は理解できているが、企画やプランニングを勉強したい
・webサイトのディレクションをスムーズにしたい
・本質的な企画提案ができるようになりたい
・Web関係の仕事をしていて、マーケティングやプランニングに興味のある方

セミナープログラム-PROGRAM

(１)サイトプランニング3つの軸
(２)コンバージョンポイントの設定
(３)ターゲットユーザーの設定
(４)ペルソナ設計
(５)カスタマージャーニー
(６)ユーザー導線設計

    株式会社デスクトップワークス（https://d-w.jp） 代表取締役
    Webディレクタースクール（https://webdirection.jp） 主宰

1999年、フリーのWebディレクターとして独立後、株式会社デスクトップワークスを設立。
企業サイトの企画～設計～制作を手掛ける傍ら、独自手法のディレクションを題材にした実践型の研修講師として全国各地での講演活動を実施。

また、Webディレクター向けのライブ配信番組
『ザ・ディレクション（https://webdirection.jp/live/）』や、セミナーイベント『エンタミナ（https://webdirection.jp/event/）
』の主催など幅広く活動。

2017年、世界初となるWebディレクター向け支援ツール『Webディレクター手帳（https://webdirection.jp/tools/）』を開発。2018年、書籍『ディレクション思考（https://webdirection.jp/book/）』を執筆、あわせてWebディレクションを学び身につけるオンラインサロン『4LDK（https://webdirection.jp/4LDK/）』をスタート。

株式会社スクー「2014年度最優秀公認団体賞」受賞。CSS
Nite「ベストスピーカー」を2年連続受賞（2015年、2016年殿堂入り、2017年ベストセッション選出）。
