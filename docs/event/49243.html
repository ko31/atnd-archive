プラスチック材料・製品設計・成形加工の三位一体のものづくり

イベント ユーザー
event_id 49243
title プラスチック材料・製品設計・成形加工の三位一体のものづくり
catch 適正な設計および経済的な成形加工を考慮した、プラスチック製造セミナーです！トラブル対策および劣化に伴う対策も重点において説明します！
description

●受講対象者
　素材メーカー、成形加工の高分子の開発・研究や製造業務に携わる若い技術者や、製造現場にかかわりを持つ経験者でプラスチック成形のトラブルなどの品質向上のプロセスを認識されたい方

●習得知識
　プラスチック、エラストマーの材料特性、物性の把握、成形加工基礎技術、樹脂メッキ技術（湿式，乾式）、トラブル対策の手法。

●趣旨等
　三位一体のものづくりとは、プラスチック材料の説明をする際、設計の注意点、成形加工の注意点を合わせて、ものづくりを解説する。したがって、適正な設計および経済的な成形加工も同様です。ダブラないように説明します。また、成形加工工程でのトラブル対策および劣化に伴う対策も重点において説明します。

●プログラム
（目次）
１．樹脂成形加工・基礎技術
　　1-1．最適製品設計
　　1-2．成形材料の選択
　　1-3．流れと評価試験
　　1-4．射出成形
　　1-5．押出成形
　　1-6．押出ブロー（中空）
　　1-7．圧縮成形
　　1-8．熱（真空）成形

２．押出・射出成形応用技術（ブロー成形含む）
　　2-1．肉厚押出成形（厚み50mm以上、気泡の無い）
　　2-2．2軸押出成形（材料ブレンドから成形と一貫性）
　　2-3．射出ブロー成形 （高速成形）、RIM成形
　　2-4．多層ブロー（化粧瓶）・押出（射出）

３．金型技術（現場を主に）高流動
　　3-1．金型と機構・ゲートレス、ランナーレス
　　3-2．低コストカセット金型、段取り時間短縮化技術
　　3-3．真空成形の真空技術

４．樹脂成形2次加工技術
　　4-1．ホットスタンプ、スパッタリング
　　4-2．樹脂メッキ（生産性向上への注意点）
　　4-3．接着（樹脂と他材料との接着）

５．射出成形・不良現象と対策
　　5-1．成形加工におけるトラブル対策（ヒケ、肉盗み、焼け、バリ、ウェルドフローマーク対策）

６．新成形加工・ナノ材料設計
　　6-1．ハニカム構造形成加工（高強度、軽量）
　　6-2．超臨界射出成形（低コスト、軽量化）
　　6-3．カーボン繊維、ナノチューブ、ナノコイルの材料設計と応用
