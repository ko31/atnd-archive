description 内閣府「地域社会雇用創造事業」のビジネスコンペを通過した社会起業家に対して、グループで議論することで事業のアドヴァイスをしてあげたり、プロボノ支援して下さる方を募集する催しです。

12月1日（水）18時45分〜21時（社会起業家４組プレゼン＋ワールドカフェ）

詳細に関しては別途upします。どの分野、どんなテーマのプランなのかも可能な範囲で事前にお知らせします。

当日の流れ

1. 「プロボノユニット」、および「プロボノOpen IDEO」の説明

2. 社会起業家が事業プランをプレゼン（５〜10分間）

3. 社会起業家から解決したい課題を提示、もしくは組織の問題は何かを参加者に考えてもらう

4. ワールドカフェ形式で課題を探求する

5. 参加者から社会起業家へフィードバック

6. 社会起業家からの感想

7. サポーター、およびプロボノユニット参加者を集う
