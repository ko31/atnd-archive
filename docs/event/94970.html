description

　リクルートライフスタイルが提供する、0円でカンタンに使えるPOSレジアプリ『Airレジ』が、店舗オーナー向けのミニセミナーを開催！第7回のテーマはこれまでの参加者の方からの要望も多かった、「お店の売上を伸ばす！シンプル・カンタン・スマートなAirシリーズ活用セミナー」。Airシリーズのデザインや機能性向上をミッションとするチームのマネージャーとして、いくつもの店舗のAirシリーズ活用方法を見てきた、鹿毛雄一郎（かげ・ゆういちろう）氏が登壇します。自身の店舗運営経験などもふまえて、「お店の売上を伸ばす！シンプル・カンタン・スマートなAirシリーズ活用セミナー」をお届けします。

【こんな方におすすめです】

・カフェや飲食店を開業しようとされている、または経営されている方
・美容室やサロンを開業しようとされている、経営されている方
・雑貨やアパレルショップを開業しようとされている、経営されている方
・ネットや本などでの情報収集にかける時間があまりとれない方
・AirレジなどのAirシリーズの導入を検討している、またはすでに導入している方
※営業や採用目的の方は参加できません（受付時に確認いたします）

【セミナーで学べること】

・お店の売上を伸ばすためのAirシリーズ活用術
・実際にお店の売上を伸ばした、他のお店のAirシリーズ活用事例
・クレジットカードなどのキャッシュレス決済対応方法
※セミナー後は講師や参加者同士での交流会を予定。軽食もご用意しております。
Airシリーズの活用や導入に向けたご相談も個別に行うことができます。

名前：鹿毛 雄一郎（かげ ゆういちろう）
株式会社リクルートライフスタイル　ネットビジネス本部 Air事業ユニット UXデザイングループ グループマネジャー

1987年生まれ。大学院卒業後、2011年に株式会社リクルートに入社。ポンパレモール、じゃらん観光ガイドなどのUIデザインや立ち上げを経験し、2014年よりPOSレジアプリ『Airレジ』のUXデザインを担当。現在は『Airレジ』をはじめとする業務支援サービスのUXデザインを担当するグループのマネジメントを行う。
