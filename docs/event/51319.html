凡人の起業成功メソッド『10の法則』

イベント ユーザー
event_id 51319
title 凡人の起業成功メソッド『10の法則』
catch 才能も実績もない凡人以下の人間が起業家として年収1000万円の継続収入を確立する最短（平均して1年以内）のステップを公開
description

私は2010年に2年弱勤めた会社を飛び出して独立・起業しました。

が、私には知識もスキルも人脈も才能も経験も実績も資金もなかったため、
起業後2年近くは月に2～3万円しか売り上げが立たない
落ちこぼれ経営者となってしまったのです。

当時は100万円以上貯めていた貯金が徐々に減り始め、
500円のコンビニ弁当が贅沢と思えるほどの貧乏生活でした。

しかし、私の尊敬する起業家さんから

『凡人が成功する10のメソッド』

を教わり実践してから、一気に人生が変わり始めました。

合計で10個のステップを乗り越えていくだけで
凡人以下の才能しか持たない人間でも
年収1000万円になれるというロジックだったのですが…

そのステップを1つ1つ乗り越えていくたびに
自分の行動力・思考力・集中力が
限界以上に発揮されていくのを肌で感じられました。

その結果、月商2～3万円の男が
たったの11ヶ月で年収1000万円を突破できたのです。

私はしばらく、その収入で満足していました。

が、周囲を見渡せば3年前の私と同じく全然売り上げが立たず、
極貧生活に苦しんでいる方が山ほどいます。

「今のままビジネスを続けて本当に成功するのだろうか？」
「たまに理由もなく気持ちが暗くなってしまう」
「どうしたら今より収入を増やせるのか…」

そんなご相談を日々いただいてきました。

そこでこの度、たった1人だけでも貧困の生活から脱出できればと思い、
3名限定で『10の成功法則』を伝授することに決めました。

今回は1人1人のお客様とじっくり向き合って
セミナーを進行しなければならないので、
たった3名だけの募集となります。

あなたが10のメソッドで『年収1000万円』という
高みに登りたいなら、ぜひご参加ください。

・起業した直後に必ず行わなければならない『引越し作業』とは？
・90%以上の起業家が見落としている『環境』の概念について
・1年間で月収83万円以上をキープするために必ず発見しなければならない○○○○○先を徹底解説
・人間の記憶力を3倍以上にアップさせる、科学的に証明されたアプローチとは？
・iPhoneを充電しているときに偶然発見した、世界一効率的な知識やノウハウの学習法とは？

・成功者の思考、収入、ライフスタイルを完全にあなたのものとするために必要な『たった1つ』の習慣を公開します
・成功者の思考法を丸ごとゲットする方法！
・現Panasonicの設立者、松下幸之助さんから学んだ本質的な成功法則とは？
・他人に依存して成功を手に入れる具体的テンプレートを公開
・あなたの眠っている潜在能力を引き出すフォーカススキルの秘密

・億万長者が心がけている『クレジットカードの使い方』とは？
・起業で最悪の失敗をしても絶対に破産しない保険のかけ方があります…
・あなたのアイデア力を24時間365日間鍛え上げていく『借金の法則』とは？
・私が11ヶ月で年収1000万円を達成したときに1日何度も唱えた『魔法の言葉』を公開
・成功者がテレビや新聞を見ない驚愕の理由とは？

などを解説します。
