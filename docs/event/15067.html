description

植物の多くは花びらの数、葉の大きさ、枝の長さなど黄金比よってデザインされています。
『黄金分割って何？』『どうやって使うの？』と言う質問に
『Street Flowers』の写真をサンプルに解説します。
セミナー概要：
1_数学者と美術評論家の黄金分割の見解の違い。
2_『ダヴィンチ・コード』に出てくる黄金分割とフィボナッチ数列
3_構図だけではなく配色、音楽、にも使える黄金分割。etc.
4_写真、デザインにすぐに使える黄金分割の活用法。

『ダヴィンチ・コード』を見て暗号解読がよく解らなかった人は
「そうだったのか」と納得するはずです。
