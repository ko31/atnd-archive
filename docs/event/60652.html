description

東北で宮澤賢治の「農民芸術概論」に共感する仲間が集まり、平成25年１月に設立した農民のオーケストラです。
練習、演奏会は冬の「農閑期」。11月頃から10回程度の練習を行い、年明けの２月頃に毎年東北のいずれかの県で順番に演奏会を行います。

【曲　目】
J.シベリウス／交響詩「フィンランディア」（合唱つき）
牧野時夫／管弦楽と合唱のためのオラトリオ「農民・宮沢賢治の祈り」より「雨ニモマケズ」
L.v.ベートーヴェン／交響曲第6番「田園」

【指　揮】出井則太郎、牧野時夫、折居周二
【入場料】一般1,000円、高校生以下500円、未就学児無料
　　　　　東日本大震災の被災者証明をお持ちの方は無料

【チケット販売】
盛岡市／チケットぴあ盛岡、川徳プレイガイド
花巻市／花巻市文化会館、なはんプラザ、正時堂本店、アルテマルカン桜台店、砂田屋（石鳥谷町）、いせかん商店（大追町）、産直あおぞら（東和町）
北上市／さくらホール
遠野市／とぴあプレイガイド

【主催】東北農民管弦団第2回定期演奏会花巻公演実行委員会
【後援】花巻市、花巻市教育委員会、宮沢賢治学会イーハトーブセンター
