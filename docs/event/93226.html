3DCGもくもくしましょう！

ジャンルもソフトウェアを問わないもくもく会です。
自分の作業の傍ら、相談するもよし、作品発表するもよしです。
興味のある方、進捗出していきましょう！

内容

  • 基本的には、自分の作業を進めながら雑談したりします。
  • その日の目標を決めて、終了前に簡単な発表をします（任意参加）。
    なお、会は18時までですが、会場は20時まで使用できます。閉会後引き続き作業することも可能です。
  • 途中入退室は自由です。

会場について

  • コワーキングスペース Biz Library 様の会場を一部お借りして行います。このため、会自体の参加は無料ですが、 一般参加（Biz Library非会員）の方は、会場の1日利用料（2,000円)が必要になります。 ご注意ください。
  • 会場にてWifiの利用は可能ですが、必要と思われるソフト類などは事前にインストールを済ませておいてください。

持参物について

  • ご使用になる3DCGソフトウェアをインストール済みのノートパソコン
  • マウス
    ※電源タップなどは会場に用意があります。
