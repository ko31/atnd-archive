cocos2d for iPhone 勉強会

2dに特化した、iPhone対応のゲームエンジン cocos2d の勉強会を行います。
入門編ということで、cocos2dの機能説明および、簡単なサンプルを作成し、機能を学習します。
メドレー初の勉強会になります。是非皆様ご参加ください！

対象

cocos2dに興味がある、iPhone開発者（初心者〜中級者以上）

ノートパソコン（ＭＡＣ）の持ち込みが必須となります。
※xcode や objective-C 自体の解説はおこないません。

ノートパソコンについて

*Mac OS X 10.7 Lion
*Xcode 4.1.1
*cocos2d for iPhone 1.0.1

上記インストール済みのノートパソコンおよびアダプターを持参ください。
(それ以外の環境でもOKですが、環境の違いにつきましては、原則フォローいたしません。)

※cocos2dについては、当日インストール手順を説明します。
インストール方法がわからない場合は、インストールしていなくてもかまいません。

ノートパソコン以外に持参いただくもの

*受付の際にお名刺を頂戴しますので受付用・名札用の２枚ご用意下さい。
*名刺をお持ちでない場合は、学生証等の身分証明書の提示をお願いします。
*iPhone実機については必須ではありませんが、お持ちの場合は持参いただければと思います。
