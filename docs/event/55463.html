愚痴・不平・不満を建設的に語る会

イベント ユーザー
event_id 55463
title 愚痴・不平・不満を建設的に語る会
catch 聞き屋・本音・感情爆発で何かが変わる？！
description

これはセミナーではなく講師もいません。
そして何かを売りつける、洗脳するなどのような姑息なバックエンドなどありません。
自分自身の勉強会です。
主役はあなたです！

ここは名前も出さずにただ自分の心の中を思いっきり話してみる場です。
そのあと何が残ります？
何が出てきますか？
何を発見できるでしょう？
普段出せない、抑えている感情を全て開放してみましょう！

カラオケで発散するよりも、お酒で紛らわすよりも、もっと気持ちよくなれます！
もっと自分に自信が持てます！
心の底から元気が沸いてきます！
ひょっとしたら自分が思いもしないことが出てくるかもしれません。
そんな発見があれば大チャンスです！

目的は自分を解放してあげることです。
話す内容や脈絡などあまり気にしないで下さい。

でも興味はあっても自信がない方もいるでしょう。
何を話せばいいか分からないかもしれません。

思いっきり話すことに不安を感じるかもしれません。
そんな方には簡単な質問に答えて頂きます。答えていくことで気持ちが開放されるでしょう。

主役はあなた、自分自身です！！

当日の項目
1.主催者からご挨拶
2.自己紹介（各自順番に）
3.思いっきり話そう！（話すことなくても大丈夫です）
4.感想発表（あくまで自分の気持ちです。人の話の感想ではありません）

知らない人前でいきなり本音で話しましょう！といっても絶対に戸惑うと思います。
話したいことがある方は存分に語って頂いて結構ですが、自信が無い方（ほとんどの方はそうだと思います）は私が隣に一緒に立って質問していきます。
質問といっても簡単に答えられることです。（昨日の夕飯は何ですか？みたいに）
人前で答えていくことで少しづつ自信が出てきて、少しづつ自分の気持ちが開放できたらいいなと考えています。
ですので質問の途中でも質問に関係なく話したいことがあればその時点で話していただいて結構です。
その中身や構成などかまいません。
支離滅裂で十分です！
本音で話しましょう！心の底から！

もちろん不満や不平でも大丈夫です！
思う存分に語ることです。

でも思う存分話した後は他の人の本音もちゃんと聞きましょう。
反対の立場、聞き屋として徹してください。
これは他人の普段言えない本音を聴く事で共感・共鳴があるかも知れないからです。
自分が気づいていないことをもしかしたら話しているかもしれません。
気づいてないことを引き出してくれるかもしれない、大きな気づきになるからです。

本音で話す、聞き屋になる、その後に残ることはどれだけ本音で真剣にこの時間を使うかで得るものも大きく違ってくることでしょう。

参加にあたっての注意事項があります。
1.決められた時間で話しましょう（ずっと話されるのは迷惑です）
2.人の話は聴き入りましょう
3.会場内ではニックネームとしますので呼ばれたい名前を考えてきてください。
4.人の話に感想や意見は言わないで下さい。ただ聴くだけです。
5.話している方に失礼ですので途中退席は原則厳禁です。
6.名刺交換はその場ではしないで下さい。会場出てからは自由です。
7.不平不満でも話してＯＫですが、反人道的な言葉はＮＧとします。
注意しても変わらない場合は退席していただきます。
（聴いている方も真剣ですので気分を害する言葉はダメです）
8.主催者の指示には従って下さいますようお願いします。
