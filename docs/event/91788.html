イベント概要

週５日以外のフルタイム以外の仕事、リモートワークの実情などについて学ぶ勉強会です。 週３、４日の仕事の特性、実際に携わっていける人はどのような人なのか、学んでいきます。

ＷＥＢではあまり語られることのない ＩＴＷＥＢ業界においての、フルタイム以外の仕事について、参加者の皆様と一緒に学んでいきましょう。

【先着順で定員は５名とさせていただきます】

内容

・週３、週４の仕事はどんな仕事があるのか

・週３、週４の仕事の特性

・リモートワーク（在宅勤務）の特性

・実際にやっている人はどんな人か

・ディスカッション（参加者と一緒に）

・今後の案件情報のご案内

・その他

※プログラムは変更する場合もございます。あらかじめご了承下さい。

参加対象者

・現在フリーランスの方（フリーランス経験のある方）かつ 経験年数が約半年以上の方

・外注契約（業務委託契約等）で仕事をしたことのある方（雇用であるアルバイト契約、社員契約ではない形）

・現在、週５日、および週５日以外の仕事を探している方

・エンジニア（プログラマー）、人事（採用）、ＷＥＢマーケッター、ＷＥＢデザイナー　いずれかの職種での業務経験のある方
