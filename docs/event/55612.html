■異能のアナリストと言われる陳満咲社氏

大手情報ベンダーや証券、FX会社の誘いを断ってきた と言われる彼のセミナーが、またやってくる！
第1弾、第2弾と満員御礼の大好評！
「わかりやすい」
「今後の見通しが決まった」
と、多くの方からご意見をいただいています。

・2014年後半の売買ポイントは？
・来年以降の中長期の投資スタンスを聞きたい
などの答えを探している方、是非ご参加ください！
また、為替だけでなくコモディティについても陳氏の分析が冴え渡ります。
まさに、ここでしか聞けない話！！
第1弾『スーパーサイクル理論』
第2弾『エリオット波動論』
…さて、今回のテーマは！？

彼の経験により確立された”高度な戦略“と”実践的なトレード手法“をこのセミナーで学んでください！！
無限のチャンスがあなたの手に！

■弊社ディーラー山田高志によるファンダメンタル分析

タイミングを知る上で必要なのが"テクニカル"。
相場の方向性を知る上で必要なのが"ファンダメンタル"。

前回講演では、
「ファンダメンタルは信じない」という陳氏に対し、
当社屈指の人気ディーラー山田高志がファンダメンタルから相場分析を行いました。

来場のお客様の方々も聞き入っておられ、
好評だったため、再び山田高志がファンダメンタルの観点から解説いたします！

自らもテクニカルアナリストの資格を持つ山田高志と陳氏とのファンダメンタルとテクニカルが”融合”したトークセッションが繰り広げられるかも！？

商品先物取引の仕組みの説明から 最新の相場情報、旬なニュースまで幅広くお届けいたします！
２つの要素を一度に学び、両面から2014年の相場動向を読み解きます。
