CAT 1DAY体験ミュージカル・ワークショップ2016 December開催

「ミュージカル＆演劇をもっと身近なものに」というコンセプトのもと神奈川県相模原市を中心に活動するミュージカル＆演劇ワークショップCATでは、ただいま１日体験ミュージカル・ワークショップを開催しています。
ずっとやってみたかったけど思っているだけのあなた、自分には無理だと思っているあなた、そんなことはありません。舞台経験の有無は問いませんので、どんな方でも、何も出来なくても情熱さえあれば大丈夫です。学生の方もお仕事をお持ちの方も、ミュージカル＆演劇を通じてあなたの個性を磨いて見ませんか！

★日程　2016年12月4日(日) 19:00～21:00　8日(木) 19:30～21:00　17日(土) 19:00～21:00　
★会場　橋本公民館 大会議室　（JR横浜線・京王線「橋本駅」下車徒歩1分　イオン橋本6F）
★募集人員　各回10名
★参加資格　小学生以上～50歳まで
★参加費　　各日一回500円

★カリキュラム内容（一部内容が変更する場合もございます）
～声を出してみよう！ミュージカルの世界に入っちゃおう！～　講師：郡　陽二郎　　
ウォームアップ体操、発声、簡単なセリフと動き、実践！ミニ・ミュージカル
★参加当日は動きやすい服装・運動靴・飲物などを用意してください。
★ワークショップの収益金は東日本大震災の義援金として全額寄付させていただきます。
