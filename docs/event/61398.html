講演概要

13:15～14:05 Javaのこれまでとこれから

Javaが1995年に発表されてから20年が経とうとしています。その間、Oracleによる買収などJavaをとりまく環境は大きく変わりました。昨年はラムダ式が導入されたJava SE 8もリリースされ、言語の性質としても変わりました。
今後、Java SE 9や10での変化につながる、ValharaやPanamaといったプロジェクトも発表されています。
そこで、このセッションでは、Javaのこれまでをおさらいし、今後のどのような動きにつながるかを紹介します。

きしだ なおき

14:15～15:05 Migration from Swing to JavaFX

Java SE 8 から標準 GUI ライブラリとなった JavaFX。新規に GUI アプリケーションを書くとしたら JavaFX 一択ですが、まだまだ Swing で作られたアプリケーションが巷にはあふれています。

そこで、本セッションでは Swing から JavaFX へのマイグレーションについて、具体的な例を挙げながら説明していきます。

櫻庭 祐一

15:20～16:10 REST with Spring Boot

モダンなJavaアプリケーションを簡単に作成できることで人気のSpring Bootの説明と、
RESTアプリケーションに関する説明を行います。
RESTの説明ではRichardson Maturity Model/JSON Patch/Securing REST Serivcesをテーマに、Spring HATEOAS/Spring Data REST/Spring Sync/Spring Security OAuth/Spring Sessionなどを説明し、これらをSpring Boot上でいかに簡単に使えるかを紹介します。

槙　俊明(@making)
JJUG幹事、「はじめてのSpring Boot (工学社)」(http://bit.ly/hajiboot)を執筆しました。

16:20～17:10 Spring Boot + Doma + AngularJSで作るERP 〜JavaQneバージョン〜

仕事でERPサービスの開発・運用をやってます。
2007年にローンチしたサービスで、フレームワークはSeasar2(Teeda) → Java EE 6(JSF)と移行してきましたが、
2014年からサーバサイドにSpring Boot、O/RマッパーにDoma、フロントエンドにAngularJSを使っています。
ERPというミッションクリティカルかつ、ライフサイクルが長いシステムでAngularJSを採用した理由や、
開発や運用お話などを事例として発表します。

松崎 学/江原 稔雄
