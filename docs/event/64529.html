現役WEBディレクター生講義 『【高橋流】ウェブサービスの作り方』

ウェブサービスを立ち上げて起業したい人 、今後ウェブディレクターとして活躍したい人必見です。

こんなことを学びます

＜こんなことを学びます＞
・WEBディレクションの方法
・WEBサービスで独立起業する方法

＜こんなことが出来るようになります＞
・WEBディレクション
・自分で企画したサービスをリリース、運営、収益化

＜講座の内容＞
・今回は『企画編』を行います。
WEBディレクターとして、企画に厚みを持たせる方法（ヒアリングのコツ）や、起業に向けて企画の立て方が学べます。

＜こんな風に教えます＞
少人数制グループで丁寧に教えます。
レクチャーとワークを組み合わせた二部構成です。
講師の一方的な説明にならずに参加者の皆さんと対話する形式で、
自由に楽しく学んで頂きます。
