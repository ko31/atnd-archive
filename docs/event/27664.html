第1部:透明複合材の黄変・着色対策
～UV硬化樹脂を中心とした光学樹脂の設計と低着色化について～
【11:00～12:15】

【講座の趣旨】
光学用途にいられる高分子材料は、着色が少なこと必須である。着色の原因となる物質は一般に吸光係数が大きく、微量でも樹脂の着色影響を及ぼすため原因を特定できないことも多い。本講座では、UV硬化樹脂を例として、化学構造と着色の関係、不純物の影響、熱や光による着色を防ぐ手法を紹介する。

【プログラム】

1.UV 硬化樹脂の着色と濁り
　1.1.着色や濁りはなぜ起こるか?
　1.2.モノマー・オリゴマーの化学構造と関係
　1.3.モノマー・オリゴマーの不純物の影響

2.添加剤による着色と濁り
　2.1.光開始剤による影響
　2.2.重合禁止剤・酸化防止剤による影響
　2.3.その他の添加剤による影響

3.硬化プロセスや使用条件による着色と濁り
　3.1.硬化条件による影響
　3.2.使用条件による影響

第2部:SiO2を用いた新規複合材の開発と耐熱・透明性向上
【13:15～14:30】

【講座の趣旨】
近年、ポリマーとシリカなど無機酸化物を複合化した新規複合材料(ナノハイブリッド材料)の開発が盛んに行われ、成形材料、塗料やハードコーティング剤として事業化されている。複合材料の基本的特性はシリカの粒径、分散性や微細構造などハイブリッド構造によって制御される。本セミナーでは、ポリマーとシリカの複合化による新規複合材料の設計、代表的な合成法とその構造および透明性を低下させず、耐熱性や機械的特性を向上させた高機能・高性能複合材料の開発と応用について概説する。

【プログラム】

1.高性能・高機能複合材料の創製
　1.1.複合材料の材料設計
　1.2.複合材料の合成法
　　1.2.1.層間挿入法(層剥離法)
　　1.2.2.ゾル-ゲル法
　　1.2.3.In-situ重合法
　1.3.シリカの表面改質(コアシェル構造型ハイブリッド材料)

2.複合材料の構造、特性解析
　2.1.構造分析:FT-IR、29SiNMR、XPS
　　　(シリカの表面状態、分散状態、微細構造、ネットワーク構造)
　2.2.特性分析:熱分析、透明性(光学特性)、力学測定、表面硬度　　
　2.3.形態(モルホロジー)観察:SEM、TEM　　　

3.種々な複合材料の合成と特性
　3.1.汎用(熱可塑性)樹脂(PMMA、PC、PSなど)
　3.2.耐熱性・熱硬化性樹脂(PI、エポキシ樹脂など)

4. 複合材料の応用
　4.1.成型材料
　4.2.塗料・コーティング材料

5.複合材料の現状と今後の展望　

【質疑応答】
