MakeboxLIVEとは

「Makebox」というVRモデリングツールを配信しています。 そのMakeboxでモデリングしている人や、作っているモデルをHoloLensで見ることができるアプリが「MakeboxLIVE」です。
このような感じで人のモデルと製作中のモデルが転送されます

HoloLensをかぶってネットワークにつながっていればどこからでも参加可能です。

参加方法

  1. HoloLensを持っている
  2. DevicePortalを利用できる状態にする
  3. HoloLensに"MakeboxLIVE"のアプリをインストールする
    インストール手順はこちらから
    ※第1回でインストールしている方はインストール不要です。
  4. 開催時間になったら"MakeboxLIVE"を起動する
  5. モデリングしている姿が見える

アプリについて

・人をドラッグで移動させることができます。好きな場所に置いてみてみましょう。
・アプリを実行した時点からのモデリングした物しか見ることができません。
・今回、テスト配信になりますので、アプリの動作が不安定になる可能性がありますが、ご了承ください。
