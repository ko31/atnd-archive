会の雰囲気について

・男女比は７：３から６：４くらいです。
・人狼ゲームを楽しむというよりは、人狼ゲームをダシにおしゃべりを楽しむという趣が強くなってきているようです（笑
・人狼ゲームの合間のおしゃべりの時間も回を追うごとに長くなってきているようです。
・差し入れも毎回増え続け、食べ切れなくて参加者におみやげとして配っています。
・参加者は年齢も職業も性別もバラバラですが、皆分け隔てなく和気あいあいと楽しいひと時を過ごしています。
・２次会は２１時頃から立川駅付近の居酒屋で開催しています。毎回遅い時間帯にもかかわらず多くの方に参加いただいています。

場所について

東京都多摩教育センター 地図
住所: 東京都立川市錦町６－３－１

・JR南武線 西国立駅 から 徒歩１０分
・JR中央線 立川駅南口 から 徒歩２０分
・多摩モノレール *柴崎体育館駅*から 徒歩１５分

※ネットで調べると 「東京都多摩教育センター」 とか 「多摩社会教育会館」 とか 「都立多摩図書館」 とか出てきますが、 全て建物は同一 です。

活動方針について

・人狼ゲームをまったくやったことが無いという方にも イチ から ルール をお教えします。
・人狼愛好家が少しでも増えてくれればという思いから、 初心者 や 女性 にも楽しんでいただける会を目指していきます。

参加費について

・ 学生 と 新規 の方は 無料 です。
・ その他 の方は、会場レンタル料＋その他雑費の名目で 一人500円 いただきます。

途中参加、途中退出について

会への途中参加、途中退出については基本的に 自由 です。
ただしゲームの性質上、人狼ゲームへの途中参加や途中離脱は出来ませんのでご了承ください。

会場内での飲食について

会場内での飲食は基本的に 自由 ですが、 お酒 と ニオイのキツイもの に関しては 禁止 とさせていただきます。

その他

・人狼ゲームの他にも、軽目の ボードゲーム や カードゲーム をいくつか用意しています。
・人狼ゲームで早々に退場になされた方はこちらでもお楽しみいただけます。
　（勿論、残りの人狼ゲームを観戦していただいても構いません。）
・立川近辺にお住まいの方のご参加がまだまだ少ない状況です。（多摩地区、特に八王子市、国立市、国分寺市、昭島市、日野市、小平市、東大和市、武蔵村山市、福生市、あきる野市、青梅市、多摩市、稲城市、府中市等その近隣にお住いの方のご参加をこころよりお待ちしています。）
