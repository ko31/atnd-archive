description

OpenStreetMap( http://www.openstreetmap.org/ ) という、
Wikipediaのようにみんなで自由に使える地図データを作ろう！というプロジェクトがあります。

地図といえば真っ先に思い浮かぶものはGoogleMapsですが、GoogleMapsには色々と制限があります。
また、自分が描いたものが地図に残るというのは非常に面白いものです。

今回は土日2日間の開催であり、
初日は勉強会、2日目はフィールドワークで地図を作ります。
初日の本イベントでは、GoogleMapsとOpenStreetMapの違い・活用事例などについてご紹介させていただきます。

もちろん、どちらか一方のみのご参加も大歓迎です。
