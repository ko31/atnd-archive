JAWS-UG ( AWS Users Group – Japan : http://jaws-ug.jp/ ) 主催の
ハンズオン・セミナーとなります。

Amazon Web Servicesを利用してみたい方 ／ 初めて利用する方を
対象にチュートリアル(概要, 機能, 操作方法の説明)を行います。
講師陣がアカウントの取得方法からゆっくり説明します。

0. クラウドコンピューティング & AWS概説
1. AWSアカウントの取得方法説明
2. EC2インスタンスの種類と課金の考え方
3. Simple Monthly Calculator での月間使用料見積もり方法
4. Management Consoleを使って、EC2インスタンス立ち上げ
5. セキュリティポリシーの設定とインスタンスへのログイン方法説明
6. Webサーバの設定
6. インスタタンスの消去
7. 質疑応答

※なお、アカウント取得はクレジットカードの番号入力が
必要となりますので、当日は流れのみ説明いたします。
可能であれば、事前に取得されることを奨励しております。
アカウントを取得された方はノートPCもお持ちください。無線LANと電源は会場にて手配予定です。
