夢の同時開催が大崎にあるNaverさんの協力で実現します。

日時は7月23日(金)18時開場で、18時25分からスタート！

Yokohama.pmトラックではいつも通りのメンバーによる自由なPerlについてのトークが
聞けちゃいます。

Perlに興味のある方、レベルに関係なくお越し下さい！
