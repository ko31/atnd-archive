description

＜講座のポイント＞
　基板上に実装された電子部品、LED製品、また小型化が進むモーターなどの高温部からの放熱には、熱伝導、対流、そしてふく射が複雑に関与しています。このような電子・電気機器の合理的な熱設計を行うためには、まず熱伝導による熱移動をしっかりと理解しておく必要があります。本講座では、熱伝導ならびにこれと密接に関連する接触熱抵抗および熱回路網法の基礎事項をわかりやすく説明いたします。また、Excelによる熱伝導シミュレーションを、ノートパソコンを用いた演習により習得していただきます。

＜受講対象者レベル＞
　本講座では、初心者の方でも容易に理解いただけるレベルでの説明を行います。また、Excelによるシミュレーションでは、プログラミングを一切行いませんので、Basic、CあるいはFortranなどのプログラム言語に関する知識は不要です。

＜セールスポイント＞
　電子・電気機器の熱解析を行う上で重要な熱伝導、接触熱抵抗そして熱回路網法に関する基礎的な事項を、具体的な例に基づいて解説します。また、Excelの基本的な参照機能を用いるだけで、実用レベルの熱伝導シミュレーションが容易かつビジュアルに実行できることを、ノートパソコンを用いた実習により体感していただきます。
