description

地中レーダー（GPR）探査は、遺跡情報を広域にかつ迅速に取得できる方法として利用が広がりつつあります。遺跡の範囲の把握や効果的な調査区の設定、広域の調査への利用など、

今回は遺跡探査に焦点を絞り、遺跡調査に必要な地中レーダ(GPR)の基礎知識について講義を行った後、史跡平城宮においてGPRデモンストレーションを行います。
GPRを使ったことが無い方、教育委員会などで今後利用を考えていらっしゃる方も歓迎いたします。
飛び入り参加も歓迎ですが、会場や資料の都合上、人数超過の際はお断りすることもありますので、是非登録ください。

また、探査の利用を考えている方との、情報交換、機材・技術のシェアなどのネットワーク作りも目的としています。

なお、２４日・２５日午前には研究発表、２５日午後にはより詳細なチュートリアルを予定しております。奮ってご参加ください。
