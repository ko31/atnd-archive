Android ADK ハッカソン

ADKを使って、温度や照度などといったセンサ、サーボモータやLEDを動かしてみましょう。ADKボードといくつかのデバイスが動く環境は事前に準備したものを用意するので環境がなくてもOK。

Hackathonは3-4人で1チームになって行います。Androidアプリを中心としたHackathonを想定してますが、サーバ連携したりADKボード側をいじったりするのももちろんありです。

本イベントはハッカソンです。

ハッカソンとは
http://www.masahiko.info/it/archives/000958.html

講師がいて講義をするタイプのセミナーやハンズオンではないのでご注意下さい。

対象者

  • ADKって言葉は知ってるけど何が出来るのかよくわかってないAndroidアプリ開発者
  • ArduinoとかPICとか知らなくてもOK
    *

持ち物

  • 【必須】Android開発環境(API Level10以上)の入った開発マシン
  • 【推奨】ADK対応Android端末。 → http://adis.pigmal.com/accessory_support.html
  • ADKボードは持っていなくても大丈夫です。

FAQ

Q：ハッカソンとは何ですか？
A：「Hack」と「Marathon」を合わせた造語で、短い期間集中して少人数でモノを作るイベントです。

Q：アイデアソンとは何ですか？
A：ハッカソン前に「何を作るか」を話し合うというミーティングです。

Q：当日何が必要ですか？
A：当日はAndroidアプリを作るためにノートパソコンが必要です。

Q：ADKボードは必要ですか？
A：必要ありません。すでにADKでハードウェアを制御しているものがある場合は持ってきてもらってもOKです。

Q：インターネット回線を自分で用意する必要はありますか？
A：会場内の無線LANを開放しますが、多数接続すると安定しない場合もあるようなのでモバイルルータを持っている人は持ってきてもらえると助かります。

Q：電源はありますか？
A：会場内のコンセントをマルチタップで全員に行き渡るよう準備します。

Q：お昼ごはんはでますか？
A：提供の予定はありません。

Q：会場での飲食は可能ですか？
A：軽食程度であれば構いません。

Q：Ustream配信はしますか？
A：検討中です。
