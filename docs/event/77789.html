好評発売中のオニオンたん・びほろあすぱら娘のLINEスタンプ制作者であり、びほろアスパラ娘の原作者でもある八吉(やつきち)さんにマスコットキャラクターをパソコンで描く方法を習ってみよう＆実際にLINEスタンプを世に送り出すまでも学ぶことができる講座が、北見に引き続き札幌でも開催です。

☆オニオンたん☆
https://www.facebook.com/oniontanproject/

◎今回の講座の主な内容
・ＬＩＮＥスタンプを世に送り出すまでの流れ
・企画発案から実際に絵を描くまでの流れ
・どんな絵を描きたいか、まずは描いてみよう。
・『CLIP STUDIO PAINTO』を使って描いてみよう。

※絵を描く部分は、個別指導・サポート方式で進めます。

オニオンたんプロジェクトでＬＩＮＥスタンプデザインを担当し、びほろアスパラ娘の原作者である八吉さんにアドバイスを受けながらまずは自らのオリジナルキャラクターを描き、そのキャラクターを実際にイラスト制作ソフトの使い方を教えてもらいながらパソコンで描いてみることができる講座。

今回使用するイラスト制作ソフトは『CLIP STUDIO PAINTO』です。講座の時間的制約の都合上、事前に以下のサイトから無料体験版をパソコンにダウンロードした上でご参加下さい。

※無料体験版のダウンロードが難しい方は会場でインストールサポートも可能です。またパソコンがないけれど参加したい、という方も数台用意がございますのでご相談下さい。

http://www.clipstudio.net/
http://www.clip-studio.com/clip_site/download/clipstudiopaint/csptrial/index?net_top_head
