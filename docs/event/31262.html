GlassFish ユーザー・グループ・ジャパン 勉強会 年４回化計画 始動！

前回 (６月) の勉強会から、毎年恒例の年末忘年会＋勉強会まで少し期間があるので、今年は秋にも勉強会をやります！
これから春夏秋冬と１回ずつ、年４回勉強会ができたらいいかなと勝手に思っています。

さて今回は、Eclipse ユーザーにはお馴染みのプラグイン日本語化プラグイン「Pleiades」のコミッタである伊賀敏樹氏をお迎えして、JSF 2.0 の新機能である Facelets について発表頂けることになりました。また、前回に引き続き久保田祐史 (@sugarlife ) 氏と久保 智 (@megascus ) 氏にも登壇頂きます。
また、最近発表された JavaFX ベースの GlassFish 監視ツールである LightFish を支える REST 管理チャネルについて、その導入部分を蓮沼 (@btnrouge )＠事務局より発表します。

勉強会終了後は会場周辺での懇親会を予定しています。発表者をはじめ参加者が対等に語らえる場です。ぜひ、懇親会への参加もご検討下さい。

主催

  • GlassFish ユーザー・グループ・ジャパン (GlassFish Users Group Japan)

発表者・発表内容について

  • 伊賀 敏樹 氏 — Eclipse 日本語化プロジェクトの中心人物のひとりとしてご存じの方も多いはず。今回は「おれおれ Practical Facelets」と題して、SIer という立場で、Facelets を利用した場合の現場的・現実的な話題／ノウハウについてお話し頂きます。
  • 久保田 祐史 氏 (@sugarlife) — 前回に引き続きの登板、GlassFishとプロファイラーについてお話しし頂きます。
  • 久保 智 氏 (@megascus) — JPA に造詣の深い久保氏が「金魚本に載っていないJPQLのお話」と題して、JPA の神髄をお話し頂きます。
  • 蓮沼 賢志＠事務局 (@btnrouge) — JavaFX による GlassFish 監視ツール「LightFish」―そのキモとなる REST 管理チャネルについてご紹介します。

※当初予定していた槙 俊明氏 (@making) のセッションは都合により次回 (12月予定) 以降に延期となりました。あらかじめご了承下さい。
