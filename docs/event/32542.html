イベント概要

Windows 8に導入される新しい種類のアプリケーション、Windows ストアアプリの実習形式のセミナーを講師の方を招いて行います。
　新世代のアプリケーション開発を学べるチャンスですので　是非ご参加ください。

日時： 2012年 10月14日（日）13:00～17:00
場所： 出島交流会館２F
講師： 服部 祐一 様 （MSPフェロー）

参加にあたって
参加費は無料です。
Windows8に興味のある方、Windowsストアアプリの開発に興味がある方であれば学生・社会人問わず参加することができます。
ふるってご参加ください。

タイムテーブル

12:30～ 受付開始
13:00～13:10 前説
13:10～14:00 Windows ストアアプリ解説
14:00～14:10 休憩
14:10～14:50 C#によるWindows ストア アプリ開発入門
14:50～17:00 Windows ストア アプリ制作実習

交通アクセス

路面電車電停「長崎駅前」から1番系統「正覚寺下」行に乗車し、「出島」で下車。徒歩3分。

参加にあたっての注意事項

当日はVMware PlayerなどにWindows 8 RTM とMicrosoft Visual Studio Express 2012 for Windows 8がセットアップされたPCを各自で用意されていることが望ましいです。
また、用意ができなかった参加者用にサーバー側にVMを用意し、参加者にはリモートデスクトップでWin8を操作して頂くことのできる環境の整備を現在進めておりますが、それほど多くは用意できません。どうしても用意ができない場合は理由と共にコメント欄にてお申し出ください。(9/21)

(追記)また、Windows8のセットアップを行いたいが自力で出来ない！といった方向けに
-前日13日18:30〜の会場設営時-

前日のサポートですが、当日午前中からのサポートのみにさせて頂きます。連絡が急になってしまい申し訳ありません。


当日14日の9:30~
に開催スタッフによるWindows8アプリ開発環境構築サポートを実施する予定です。
構築した環境は勉強会で使うことができます。参加したいけど環境が用意できないから参加を見送っている…という方も大丈夫です。(元のOSを消す事は一切致しません！)一緒に勉強しましょう！(9/26)

Windows8の環境構築の為に
メモリ容量は3GB以上
CPUはCore2以上のものを搭載したPCをお持ちください。
