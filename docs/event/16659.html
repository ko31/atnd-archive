【ご講演趣旨】
リチウムイオン二次電池は、エネルギー貯蔵デバイスとして発展を遂げている。このデバイスの主要素材である正極活物質は、現状において優れた材料であるが、更なる高性能化が期待されており、このために表面処理技術がある。この技術は、表面のみの僅かな改質で粒子全体の特性が改善でき、弊害が少なく大きな効果が得られ、今後の電池の発展を支える重要技術である。
本講演では、正極活物質の基本的な特性評価技術を理解すると共に、これら特性評価を通して、表面処理技術の有効性について解説する。

【プログラム】
1.正極材料概論
　1-1. 層状岩塩構造正極
　1-2. スピネル構造正極
　1-3. オリビン構造正極

2.正極材料評価技術
　2-1. 粉体特性評価
　　2-1-1. 粒径:一次粒子と二次粒子、粒径分布と平均粒径、
　　2-1-2. 密度:真密度、X線密度、かさ密度、タップ密度
　　2-1-3. 比表面積、空隙、細孔分布
　　2-1-4. 水分、炭酸分
　　2-1-5. 安息角、吸液量(吸油量)
　2-2. 電極特性評価
　　2-2-1. 塗料・塗膜形成
　　2-2-2. 塗料流動性、粘弾性、
　　2-2-3. 塗膜評価、面積密度、体積密度
　2-3. 充放電特性評価
　　2-3-1. セル形成
　　2-3-2. 充放電特性
　　2-3-3. サイクル特性、レート特性
　　2-3-4. インピーダンス
　　2-3-5. イオン溶出と安全性

3.正極材料の表面処理技術
　3-1. LiCoO2の表面改質
　　3-1-1. 高充電圧化による容量向上
　　3-1-2. 活物質による被覆処理 ( Li(NiCoMn)O2, スピネル系活物質 )
　　3-1-3. 金属酸化物による被覆処理 ( ZrO2, Al2O3, MgO, TiO2, SiO2, ZnO )
　3-2. LiNiO2の表面改質
　　3-2-1. 活物質による被覆処理 ( Core-shell型活物質 )
　　3-2-2. 金属酸化物による被覆処理 ( ZrO2, TiO2, La2O3, その他 )
　3-3. LiMn2O4の表面改質
　　3-3-1. 活物質による被覆処理 ( LiCoO2, スピネル系活物質 )
　　3-3-2. 金属酸化物による被覆処理 ( SiO2, ZrO2, ZnO, CeO2 )
　　3-3-3. 導電性材料による被覆処理
　3-4. LiFePO4の表面改質
　　3-4-1. 導電性向上技術の位置付け
　　3-4-2. 炭素質導電層の形成処理
　　3-4-3. 非炭素質導電層の形成処理 ( 金属層, リチウムイオン導電層 )
