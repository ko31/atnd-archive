description

　いらっしゃいませ！
　今回のオフ会を主催させていただきました。「はねうま」と申します。

　大阪の中心街「梅田」で「でぃばついったぁず大阪組」で集まりませんか！？
　2011年5月15日（日）に以下の内容にて開催します！
　5月5日に参加者募集を締切ました。ですが、主催者である自分に相談していただければ追加の手配を行います。
　
【イベント名】
　初音ミク-ProjectDIVA-　でぃばついったぁずオフ会　～2011年大阪春の陣～

【日程】
　2011年5月15日（日）12:00～20:00～

【集合場所・会場】
　大阪府大阪市（梅田）

【内容】
・12:00~14:30
　ナムコランド梅田店でゲームやらないか！？
　DIVAACと、後ろには広いスペースもあります。DIVAACやるも良し！他のゲームやるも良し！雑談するも良し！
　都合がつかない方や、ゲーセンにはあまり行かないという方はカラオケからの参加も可能です。

・15:00~18:00
　カラオケブギ梅田店（http://club.pep.ne.jp/daitoyo/amuse/boogie/boou.html）で3時間歌いまくろう！
　ボカロ曲はもちろん、なんでも歌いましょう！
　PSPに曲データを入れればカラオケの機器と接続して歌や動画を流せます！未収録のボカロ曲もこれで歌える！歌いたいけどまだ入っていない、という曲を持ち寄って歌ってみませんか？
　PSPや曲のデータは各自準備をお願いします。

18:30~20:00
　プライベートダイニング GALLERIA梅田（http://r.gnavi.co.jp/c980122/）で飲んで食おうぜ！
　飲み放題、食べ放題の1時間30分。みんなでワイワイしましょう。
　内容はこちらのページの『プレミアムクラスＧプラン』となります。
　http://r.gnavi.co.jp/c980122/menu1.html

20:00～
　企画している中ではここで解散となります。
　もちろん各自でこの後の時間を楽しむのもokです（企画主はゲーセンに舞い戻るかも？）。

【参加費】
　5000円くらい？

【必要なもの】
　やる気！
　楽しむココロ！

　はねうま
