description

ゲームクリエイターを対象にデザインについて勉強するセミナー「デザイン塾」。今回のデザイン塾ではNUKEを用いたVFXコンポジットの第一人者山口幸治氏を講師にお迎えします。
VFXや画像合成の基本、Adobe After EffectsではできないNUKEならではのコンポジットなどについて、実際にNUKEを触りながらご説明します。

VFXやコンポジット、NUKEにご興味のある開発者の方や、既にNUKEを使用しており、よりクリエイティブを高めたい方を参加対象としています。

なお、イベント終了後には開発者や参加者同士の交流や技術共有の時間も設けています。ご興味のある方は、是非、ご参加ください。

NUKEとは

NUKEは最高峰のノードベースの動画合成ツールです。
Foundry の NUKE製品は、厳しいスケジュールの中でも、映画やコマーシャル、TV、広告、デザインなど、ハイクオリティな静止画や動画を作成する全てのプロダクションのワークフローにおいて、スピードとクオリティ、効率性のマネージメント要求を満たすソリューションです。
https://www.foundry.com/ja/products/nuke
