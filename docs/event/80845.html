description

※仮の要旨です。自分の現場以外のデータ分析について軽く話しあったり相談したりできる会にしていきたいです。

初回ということで、外部からあまり聴衆を呼ぶことは想定していません。可能であれば興味ありそうな友人・知人を数名こえかけていただければ幸いです。

昨今のデータマイニングというとアクセスログ等のWebマイニング文脈が主流ですが、近年センサーデバイスの発達、スマートフォンの普及により人間・動物・物流のデータも容易に扱えるようになりました。

しかし、入力データが不完全であったり、機械的に取得できるデータと取得できないデータの差、計画を立てることの難しさなど、活用に至るまでには様々な障害も明らかになってきました。そこで、実世界の非定型データとセンサやスマホから取得できる構造化データを組み合わせて価値を生み出すことをフィールドデータマイニング(仮)とし、フィールドデータをいかに活用し、価値を生み出すかということを考える勉強会です。
