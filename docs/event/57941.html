catch 絶対取りたい補助金・助成金の申請セミナー
description

本セミナーは、起業時に極めて有益な「情報」をお送りできればと思っております。
その情報は、人、モノ、金、その全てに適用可能です。

昨年度もっとも人気があった「創業補助金」。

今月からは新しい「創業者補助金」の説明も実施できる予定です。
まだ正式発表前ですが、既に中小企業庁関連のサイトにも掲載開始されました。
スタードダッシュで、
合格率が一番高い最初の〆切に間に合わせたいところです。

（１）創業者・第二創業者向け補助金
新たな需要を創造する新商品等を提供する創業希望者や創業者に対して、創業に要する費用の一部を支援します（補助上限200万円、補助率2/3）。
また、事業承継を契機に既存の不採算部門を廃業し、新分野に挑戦する等の第二創業者に対して、人件費や設備費等（廃業登記や法手続費用、在庫処分費等廃業コストを含む）に要する費用の一部を支援します（補助上限1,000万円、補助率2/3）。

（２）創業支援事業者向け補助金
産業競争力強化法における創業支援事業者が、認定創業支援事業計画に基づき行う創業者支援の取組（創業者への継続的な経営指導やビジネススキルアップ研修、コワーキング事業など）に対して支援します（補助上限1,000万円、補助率2/3）。

さらに、補助金・助成金は年間で「3000種類」存在しています。

補助金とは、3ヶ月毎に適用条件がどんどん変化してしまいます。
起業時期によってその条件はだいぶことなり、まさに「情報」の価値はこの部分にあるのだと思います。

●今回お伝えできる内容
￣￣￣￣￣￣￣￣￣￣￣
　・起業時に申請できる補助金
　・その為に必要な融資（金融機関）に関する具体的な知識

今年は消費税増税もあり、景気対策の一環として補助金の実施数が大幅に増える見込みです。
弊社は、2013年4月から補助金申請支援事業を開始しました。
現在、月間数百社から、補助金の申請相談を受け付けています。
国内で最も多く申請支援をしているのではないかと思います。

元々が、弊社の「取引先支援」を目的にしたサービスだったため、現在でも完全成功報酬ないしは無料で申請代行を実施しています。

格安価格が理由だと思いますが、おかげさまで非常に好評です（成功報酬だと皆さん100%申請されます）。

※創業融資に関する説明も実施します。補助金を活用するには金融機関の面談を受ける必要があるのをご存知でしょうか。創業融資の仕組みを理解することで面談突破率も大きく向上します。
