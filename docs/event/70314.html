コンセプト

「ベンチャーDive!」のコンセプトは“より深くベンチャーのリアルに触れる”です。
2つの切り口でベンチャーのリアルに深く触れる機会を提供いたします。

  1. 大企業からベンチャーの世界に飛び込んだベンチャー経営陣によるリアルな体験談に触れられる「パネルディスカッション」
  2. ベンチャー企業のリアルな経営課題や仕事に触れられる「企業出展ブース」

仕事終わりに3時間ほど、ベンチャーの世界に飛び込んでみませんか？
