ダライ・ラマ法王来訪と世界難民の日(6月20日)が重なる機会に、長野オリンピックの際に世界平和を祈って梵鐘を鳴らした善光寺の前で、ダライ・ラマ法王に灯していただいた火を使用して、世界人権宣言と世界中の犠牲者を象徴するキャンドルを囲み、世界中の難民の人びとのために祈るキャンドルナイトを開催します。

プロデュースはCandle JUNEさんです。

長野在住の歌姫えびはらよしえさんのミニライブも予定しています。
