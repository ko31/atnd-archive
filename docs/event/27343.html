第一回　節電サバゲ！GBB＆エアコキ限定サバゲ会！

イベント ユーザー

title 第一回　節電サバゲ！GBB＆エアコキ限定サバゲ会！
catch 暖かい時期にしか使えないGBBガンを今この旬な季節に使い倒そう！！
description

2012年6月30日に千葉北フィールドでGBB＆エアコキ限定のサバゲ会を催します。（仲間内のゲームです。）
フィールドへのアクセスはココをクリック

■レギュレーション　今回は特殊ですので必ずお読みください。

・使用可能エアガン　：　ガスブローバックのライフル、SMG、ピストル（リボルバー含む）等
　　　　　　　　　　　　エアコキのライフル、ショットガン、ピストル等

　　　固定スライドガスガンはいかなるカテゴリーも使用不可
　　　外部ソースのエアガンはCo2に限り固定レギュレターで銃刀法以内収められていればOK
　　　(まぁ、違法な物でなければ相談してください…と言う事で。)

・セミオート・フルオート問わず全ての銃でパワーは0.98Ｊ以内
・使用BB弾はバイオのみ（プラスチック、セミバイオは禁止。）
・減速アダプタの使用は禁止
・フリーズコール禁止
・基本的にブラインドファイア禁止
・レーザーサイトはすべて使用禁止
・ゾンビ行為禁止
・フラッグへのダッシュアタック禁止（周囲制圧後にプッシュ。）
と、エアガン本体の縛り以外は基本的にいつも通りです。

その他サバゲアイテムの手投げグレネード、グレネード、クレイモアも使用可能。

参加の間口を広げる為の特例

GBBはピストルしかないよ！と言う方も多いと思いますが、それで十分楽しめるように工夫をいたします。
例えば、ピストルしかなく、予備マグも少ない方には普段禁止している「フィールド内へのガスとBBローダーの持ち込み」を許可しますので、リロードに時間は掛かりますが、弾切れで終了と言う事は無くなると思います。
また、ARやSMGの高火力の方を両チーム均等になるようにチーム分けをしたりします。

■ゲーム時間

時間は08：00~17：00の予定です。
0740頃にはフィールドゲートが開き、駐車場に入れるようになります。
ゲーム開始は09：30前後を予定しております。
終了は17：00に完全撤収

■参加表明とゲスト参加とお昼ご飯

参加表明の締め切りは6/23頃とします。
基本的に参加の方はココで表明していただきますが、ATNDを使用できない方なのどの参加の場合、同行する方がコメント欄にゲストの名前表記して参加表明してください。（ただし、お連れの方にも今回のレギュレーションをしっかりお伝えください。特に今回は特別なので必ずお願いします。）
リストを作る都合上、ハンドルネームでもOKですのでお名前を頂きたいです。

またお昼ご飯は各自で用意して頂きますが、吉野家とガストのデリバリーが利用出来るようです。（前回使用したとき、便利でした。）

近所にコンビニもありますが、迷彩服での利用はやめてください。

■必要な物、参加費

・参加費6000円（予定）
・エアガン
・BB弾（バイオのみ）
・ゴーグル
・顔面を保護するマスク、四肢保護のニー、エルボーパッドや手袋があるとベター。

■ちょっとしたお願い

締め切り等やその他諸事情で参加を伏せたい人も多いようなので、終了後ですが
・個人単位で『今日サバゲして来た』と言うのは自由で。
・誰が参加していたかに関しては『触れない』方向でお願いします。
・TwitterやMixi等で『○○さんお疲れ様でした～』も出来るだけ空気読んで下さると助かる人も多いかと思います。
（自分が参加していた事を表明する事に問題ない人同士のやり取りについては特に問題ありません）
・ですので、基本的に本ページコメ欄使って頂くのを推奨します

■ゲーム内容

基本的にはフラッグ戦メインです。
通常のフラッグ戦のほか、ペットボトルフラッグ戦（ブザーの代わりに3本のPBを倒す事でフラッグゲットとする）や、他にちょっと新ルール考案中です。

かなりハイペースで回しますので、「補給が間に合わない」「疲れた動けない」等は遠慮せずに休憩してください。　20分もすれば次のゲームに入れます。

■■雨天の場合■■

前日より確実な雨天（大雨や嵐）と分る場合は6/29のお昼頃にアナウンス致します。
ただし、フィールドは屋根あり屋根無しが半々位でセーフティは屋内なので、小雨程度ならば開催いたします。
