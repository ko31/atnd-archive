Javascriptでアプリ作成！Titanium実践講座上級編
大阪私茶塾(Osaka IT School)

急激な拡大を見せているスマートフォン市場。今後ますます増えていくタブレット端末。
「スマートフォンアプリをプロモーションで使用したい」
「タブレット端末で、動画・音声などのデジタル技術を最大活用した営業ツールに使用したい」など、スマートフォン・タブレット端末は、企業のマーケティングやプロモーションを大きく変える可能性を秘めています。

しかし、iPhone・iPadアプリのネイティブ開発環境であるXcodeはObjective-Cという、なじみのない言語で開発を行うため、敷居が高いのが実情です。
そこで、プログラマでない方でもアプリ開発が行えるツールとして登場したのがTitaniumです。TitaniumはWebデザイナーにもおなじみのjavascriptでアプリ開発を行います。また、TitaniumはiOS・Andoroid両方に対応したアプリを開発できるということで、人気上昇中のアプリ開発ツールです。

「Titanium実践講座」はTitaniumを実際に操作していただきながらのハンズオンセミナーです。今回の上級編では中級編で作成した写真投稿アプリにソーシャル機能などの追加を行います。Cloudはもちろん、TwitterやfacebookなどSNSとの連係の実装なども行います。
中級編を受講された方はもちろん、アプリ制作に興味のある方の参加をお待ちしています。

【大阪私茶塾(Osaka IT School)】とは

とかくセミナーといえば堅苦しい雰囲気になりがちです。大阪私茶塾では、美味しいお茶やお菓子をいただきながら、ゆったりとした気分で学んでいただけるセミナーです。6人までの少人数でコミュニケーションをしっかりと取りながら進めて行きますので、わからない事があればすぐに質問することができます。
会場の船場ビルディングは大正時代に建てられたレトロ感漂う雰囲気のあるビルです。
