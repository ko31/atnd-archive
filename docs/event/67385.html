勉強会概要

「データ解析に取り組みたいけれど、そもそもデータ解析がどういうものなのかわからない」
「分析手法やツールについて説明してる本はよくあるけれど、
　具体的に第一歩目をどう踏み出せばいいかわからない」
そんなデータ解析に興味のある初心者向けに、データ解析で価値を得るためにどのように取り組めば良いか、
実践例を交えて解説する会です。

また、サイバーエージェントの若手社員による分析事例とデータ解析の社内研修についてもご紹介します。
統計学や機械学習未経験の新人達が、どのようにしてデータ解析のプロセスや各種分析手法を習得したのか、
データ解析の結果を現場で活かせるような土壌作りを出来たのか、
などなどを実際に研修を受けてきた若手社員たちがご説明します。

会場ではデータ解析の実務プロセス入門の販売も行っております（20部）。

各発表者と発表概要

あんちべ：データ解析の実務プロセス入門

  • プロフィール
    統計屋。@AntiBayesian。
  • 発表概要
    データ解析を実践して失敗に終わらせないようにするにはどのようなプロセスを踏めばよいか、
    また、「良いデータ」とは何かについて解説します。

柴田龍男・山田伸也：サイバーエージェント分析事例＋研修事例

  • プロフィール
    ともにサイバーエージェント新卒2年目のアナリスト。
    柴田がボーイフレンド(仮)、山田がガールフレンド(仮)を担当。
  • 発表概要
    担当サービスの取り組むべき課題の発見から施策提案、実施を通して、
    サービスの品質や売上改善に至るまでの実際の流れをお話いたします。
    さらに、データ解析未経験の新人が社内研修を経て、
    様々な分析手法を実践出来るようになるまでの道のりについてもご紹介致します。

小川卓：「結果を出せる」ワンランク上のウェブアナリストになる方法

  • プロフィール
    ウェブアナリスト。主な著書に「ウェブ分析論：増補改訂版」「Webサイト分析・改善の教科書」など。サイトの分析・改善提案・解析ツールの活用・教育などを得意とする。
  • 発表概要
    業務でサイトやサービスの分析をする時に、分析者として求められるのは結果に貢献することです。
    解析ツールを使う時間が多くても、沢山のグラフやレポートを作ってもその間の売上貢献は0です。
    ウェブアナリストとして、成果を出すためにどういうポイントに気をつけなければいけないのか？
    どういった知識や行動を大切にすればよいのか？
    私自身の体験を交えてお話いたします。
