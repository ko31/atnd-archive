description

流山田園プロジェクトで広報・会計を担当している湯村です。

つくばエクスプレスで都心から30分という場所で
里山や広大な農地が広がっているのが、「千葉県流山市」です。
都心から30分とは思えないローケーション！
夕日がとても綺麗で心が洗われます。

そんな流山市の「新川耕地」という広大な農地で
昨年は無農薬でのお米作りを成功させました。
そして今年から畑で無農薬の野菜作りに挑戦します！

田園プロジェクトの野菜作りのテーマは…
・自給自足ってどうなの？
・みんなで一緒に学びながら！
・楽しく野菜作り！
・コミュニケーションを重視して活動そのものを楽しもう！

という感じで、単に「生産」というだけではない農業の様々な魅力を感じながら
農作業を通じてよりよいコミュニケーションをすれば
人間本来の心地の良い生き方ができるんじゃないかと、
そんな思いで集まったメンバーで構成されています。

気軽に農業に触れたい、土に触れたい
これから本格的に農業を始めたい
食に興味があって野菜を作ってみたい
仲間と一緒にコミュニティ作りに参加したい

などなど、参加理由は様々でも、一緒にやってみたいと思う方は
ぜひ畑へあそびに来て下さい。

通年、色々な企画を考えています！
＜企画案一例＞
　・作った野菜で花見をしよう！
　・夏野菜を自作石窯を作ってピザを焼こう！
　・無農薬のお米と野菜でパエリアを作ろう！
　・バーベキュー！
　・竹切り大会！
　・自家採種して固有種を育てよう！
　・自作無農薬小麦でパンを作ろう！
　・麦でビール作っちゃう？
　・大豆作って味噌作り！
　・流山散策！
　・流山の素材を使った飲食店での食事会
　・田んぼイベントなどと連携
　・コンパニオンプランツ栽培
　・不耕起栽培への挑戦
　・農家さんなどを招いての講習会
などなど。。
