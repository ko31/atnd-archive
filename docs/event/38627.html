イベント ユーザー

title 作業員が納得するクリーンルーム管理　
catch ～ゴミの見える化・清掃の仕方・改善の仕方～
description

【受講対象】
クリーンルームに携わる中級、初級技術者
生産技術、品質管理、設備、管理

【習得できる知識】
クリーンルーム作業員の教育、ゴミNGの軽減（良品率向上）、
製造ラインのクリーン化、クリーンルームの改善、運営管理

【講座の趣旨】
最新の実験映像を多数使用して、分かりやすい講演を目指します。
途中、講演内容を元にした簡単な演習問題を行い、終了前に答え合せをします。
参加の皆様からのご質問を事前に頂き、当日解説いたします。

【プログラム】
１．クリーン化の基礎～ここを抑えれば、クリーン化がグングン進む～
　1-1．クリーンルーム技術とクリーン化のメリット
　1-2．クリーンルームの定義と規格
　1-3．クリーンルームの設計思想
　1-4．粒子の大きさと挙動について
　1-5．気流について
　1-6．ゴミはどこからくるのか？
　1-7．粗大粒子対策とは？
　1-8. 粒子の見える化
　1-9. クリーンルームライトの活用

２．異物の付着と除塵について～大切な製品に異物を付着させないために～
　2-1．付着のメカニズム
　2-2．交差汚染
　2-3．作業員からの発塵　
　2-4．クリーンルーム内の発塵
　2-5．付着持込み対策
　2-6. 除塵について
　2-7. 再付着の問題
　2-8. クリーンルームの4原則
　2-9.　付着させない技術

３．「見える化」機器について～クリーン化の秘密兵器
　3-1. クリーンルームライトの活用
　3-2. 落下塵カウンターの活用
　3-3. 表面清浄度の管理
　3-4. 気流可視化装置の活用

４．クリーン化機器について～クリーン化機器の有効活用法と問題点を理解する。
　4-1．エアシャワーの効果（髪の毛・繊維くず・ポリアミド系異物・パーティクル）
　4-2．エアシャワーで落としたゴミはどこへ行く
　4-3．エアシャワーと静電気吸着
　4-4．クリーンルームの清浄化メカニズム（換気回数について）
　4-5．拡散板の効果
　4-6．局所クリーン化

※演習問題

５．作業員からの発塵・対策
　5-1．クリーンウエアからの発塵
　5-2．レーザー光を使った動作発塵の動画
　5-3．粗大粒子用タンブリング試験装置
　5-4．ブラックライトで浮き上がるゴミの正体
　5-5．作業員教育と入退室管理
　5-6. クリーン化文化を作ろう（習慣づけ）

６．まとめ
※演習問題　答え合せ
