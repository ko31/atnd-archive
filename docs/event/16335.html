第2回船橋ツイッター飲み会 ～震災復興編～

さて、多くの方にご参加いただいた 第一回船橋ツイッター飲み会ですが、その後多くの方に「またやらないの～？」というお声をいただいておりました。
そのため、1年越しになってしまいましたが、第２回船橋ツイッター飲み会を開催いたします。

また、今回は 『震災復興編』 といたしまして、東北のお酒や食材を中心にコース料理を組んでいただく予定となっております。
さたに、会費の10％を日本赤十字社を通じて義捐金として寄付をしたいと考えておりますので、ぜひご参加ください。

よくある質問

  • 誰でも参加できますか？
    はい。概要や開催趣旨にご賛同いただけれる方であれば、どなたでも参加いただくことが可能です。
  • ツイッターのアカウントが無いと参加出来ないの？
    ATND では、Twitter等のソーシャルメディアのアカウントがないと参加表明ができませんが、アカウントをお持ちでない方のために、メールを直接お送りいただくことでの参加表明も可能とさせていただいております。
  • ツイッター初心者だけど・・・？
    ツイッター初心者の方でも参加可能です。
    ぜひこの機会に地元のツイッター仲間を増やしてお互いにフォローしあってください。きっと新しい発見が待ってるはずです。
  • 船橋に住んでいなくても参加可能？
    「船橋」とついていますが、船橋に住んでいなくても参加可能です。船橋に住んでいる人たちや、船橋で働いている人たちが多く集まるかとは思いますが、ただ飲み会に参加したいだけの方も大歓迎です。
  • 女性一人での参加でも大丈夫でしょうか？
    前回も数多くの女性に参加頂きましたが、ほとんどの方は女性一人で参加された方々です。
    ナンパ行為や強引な勧誘を見かけた場合、スタッフが厳重に対応いたしますので、安心してご参加ください。
