FMS(先端メディアサイエンス学科)7期生の皆さん、ご入学おめでとうございます。

B1(1年生)向けに、毎年恒例行事となりつつある、自己紹介LTを今年も開催したいと思います。

先輩が大学でどのようなことをしているのか発表してもらうので、どういう学科なのか、体験しに来てください。

LTとは？

超簡単に言うとミニプレゼンです。

ライトニングトーク（英: Lightning Talks）とはカンファレンスやフォーラムなどで行われる短いプレゼンテーションのこと。様々な形式があるが、持ち時間が5分という制約が広く共有されている。(wikipediaより)

タイムテーブル

基本、参加者は何かしゃべるということでお願いします。

  • 自己紹介
  • 最近したこと
  • 自慢したいこと
  • 興味があること
  • 授業または個人で制作したもの
  • 今年度の目標
  • やりたいこと
  • 推しについて語る
  • なんでも可
  • パワポ可
  • デモ可
  • 動画プレゼン可(もし参加出来ない場合、事前に動画データをもらって当日流します)

サークル・団体の紹介は別途時間を設けます。個人の自己紹介の時間には行わないようお願いします。

HDMI端子・VGA端子は使用可能です。それ以外のアダプタ(USB Type-C、Thunderboltなど)を使用する場合は変換器を持参してください。

B1で「スライドを使いたいけどパソコンを持っていない！」という方は、当日USBにスライドを保存して持参してください。

atnd登録した人から下のタイムテーブルに追加していきます。B1(一年生)以外はコメントで以下の項目(研究室、サークル・団体、その他)について教えてください。わかり次第@kenty_fmsまでDMで知らせていただくという形でもかまいません。

基本はM2→M1→B4→B3→B2→B1という順番でやりますが、進行の都合上、主催陣は先に発表させていただきます。

B1以外は一人2分30秒程度、B1は1分程度 （参加人数により変わる場合があります。）
