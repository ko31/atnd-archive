拝啓　盛夏の候　ますますご清栄のこととお慶び申し上げます。

この度、私たちは日本と海外を繋ぐことを目的とした会社「Goodpatch Inc.(株式会社グッドパッチ)」を9月1日に設立することになりました。

つきましては、お世話になった皆様方への感謝の気持ちをこめて設立記念パーティーを下記の通り催させていただきます。

日時：8月19日 19：30〜21：30
場所：デジタルハリウッド大阪校
会費：2,000円
定員：未定（ある程度キャパはあるので多人数でも大丈夫です）
形式：立食形式（ささやかながらお食事とドリンクを用意させて頂きます。）
講演：サンフランシスコ・シリコンバレーで見てきた日本とアメリカの違い
　　　Goodpatch代表 土屋尚史　
　　　ブログ：Like a Silicon Valley

・Goodpatchとは

事業内容

Co-working spaceの運営
モバイルデバイス向けUI/UXデザイン・設計・コンサルティング
日本企業の北米進出支援
Webサービスの企画

創業者プロフィール

・土屋尚史 / 代表取締役兼CEO
年間600サイトのWebサイト制作実績を持つWeb制作会社のディレクターとして、約3年間で100サイトを越えるWebサイトのディレクションに関わる。デジタルハリウッド大学院に在籍し、京都錦市場Webサイトリニューアルプロジェクトを総指揮し多数のメディアに掲載。2011年4月に会社を退社し、サンフランシスコのWebコンサルティングカンパニーbtrax Incにてインターンとして働き、日本企業の海外進出を支援するイベントSF New Tech JapanNightの企画・運営に携わる。帰国後、Goodpatch Inc.を起業し、代表取締役に就任。
http://twitter.com/naofumi83

・谷拓樹 / 取締役
関西大学を卒業後、デジタルハリウッド、フリーランス活動を経て株式会社EC studioに入社。Webサイトの診断コンサルティング、自社サイト・サービスのディレクション、デザイン・コーディング業務に携わる。2010年11月に独立し、inkdesignの屋号でフリーランスとして活動を始める。
2011年9月より、Goodpatch Inc. に取締役として就任。
著書に「HTML5+CSS3で作る 魅せるiPhoneサイト」（Amazon.co.jp 2011年上半期「コンピュータ・IT部門 6位）。
その他、セミナー・勉強会活動を活発におこなっており、Re:Creator’s Kansai、dotplusなどを主催している。
http://twitter.com/hiloki

・Brandon Hill / 社外取締役
アメリカ人の父・日本人の母の下、北海道札幌市で生まれ育つ。日本の高校を卒業後、アメリカのサンフランシスコへ渡航。San Francisco State University デザイン科卒。在学中にプロのフリーランスデザイナーとして、数々の日米関連企業のWebサイトを手がける傍ら、大学でのWebデベロッパーとしても活躍。2004年に米国・アジア間ビジネスの架け橋となる会社・btraxを設立。
http://twitter.com/#!/BrandonKHill

・今回のパーティーの目的

9月1日の設立を待たずに代表の土屋が東京に引越すため、関西でお世話になった方々にご挨拶をさせて頂きたく急遽このような場を設けさせていただきました。2週間後という急なスケジュールで大変申し訳ございませんが、2人への激励と土屋の送別も兼ねて、都合のつく方はぜひお越しいただければ幸いです。

直接2人に面識がない方でも、ブログを読んでたや面白そうだから行ってみたいという方がいらっしゃいましたら、遠慮なくお越しください。
