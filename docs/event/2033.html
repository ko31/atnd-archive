今週末に（11/7 土曜日）焼き芋をします。
と、いうのは、焼き芋を求めている人が多いように感じたので。

小田急線登戸駅から歩いていける範囲で焼き芋のできるポイントを探し、落ち葉を集め、焼き芋をします。
もしかすると、和泉多摩川駅側になるかもしれません。場所が未確定なので、どちら側がいいか教えてもらえると助かります。

※和泉多摩川側の河川敷にしました

焼き芋を楽しむために皆さんに募集
・落ち葉を集める係（すごくたくさん必要です）
・焼き芋マスター
・窯作りマスター
・焼き芋を食べる人
・たき火が好きな人

あったほうがいいもの
・軍手

ボクと、もう一人が用意するもの
・さつまいも
・アルミふぉいる
・点火剤
・落ち葉を集める袋

以下のものはオプションです
・サツマイモ以外のものを食べたい場合は各自お持ちください
・トッピングも適当にお持ちください

雨天中止

注意事項
多分、直火で火をおこす事はこの川原周辺であれば大丈夫だと思いますが、確認をとっていないため、やり始めてだめだといわれる可能性がないわけではありません。
落ち葉があるかどうかがわかりません。
ない場合は流木などが視野に入りますが、うまく火がつかない場合は焼き芋が食べられません。
ボクは特に焼き芋に精通しているわけではないので、うまくできるかどうか確証はありません。

当日は寒いと思いますので、たき火ができたらみんなで暖まりましょう。
いろいろ懸念事項はありますが、都内で焼き芋をしたいと考えている皆様、一緒にホクホクしましょう。
