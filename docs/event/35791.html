出会い系だらだら読書会、 箱文庫 を開きます。

いつもとはちょっと違う本を読んでみたいという方。
休日に畳の上でごろごろ寝転がりながら読書しませんか？
まったりとした時間を過ごしましょう。

箱文庫ってなに？

みんなが持ち寄った文庫本を、その場でランダムに交換して、普段読まないような本を読もうという企画です。
持ち寄った本は誰が持って来たか分からないように交換することで先入観なく出会った本に没頭することが出来ます。

また最後に感想を言い合う時間を取っています。
その際に持って来た本人による簡単な解説をお願いしています。
誰かに読んでもらいたい、紹介したい本がある方はぜひお持ちください。

こんな人におすすめ

  • 普段読まない本を読んでみたい
  • だらだらと本読みたい
  • 好きな本を紹介したい
  • カフェにいきたい
  • 新しいことに出会いたい
  • 昼寝したい
費用
  • 一人 1500円
    ※カフェのおいしいコーヒーor紅茶(×2)＋場所代になります
持ち物
  • 文庫本
    参加される方は文庫本を一冊お持ちください。
    ジャンル等の指定はまったくありません。(文庫がなければその他お好きな本を)
備考
  • 飲食物についてはカフェのイベントスペースでの開催のため、持ち込みはご遠慮ください。
  • カフェではスイーツ等頼めるのでおいしいパンケーキ食べたい方はそちらでお願いします。
  • 今回お借りするカフェのイベントスペースは和室なのでごろごろできます。

本が好きな人、面白い本に出会いたい人。
参加お待ちしております。

東京では開くの初めてです！
初参加の方、大歓迎なので「ちょっと本でも読もうかな」「ごろごろさせろ」という方はぜひお越しください。
