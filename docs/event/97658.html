パーカッションの募集は締め切りました！

  • ありがとうございました。

あの「ぎじんオケ」が帰ってきました

  • 過去に何度か中止ということを繰り返し、もうやらないつもりでおりましたが、結局やりたくなっちゃったのでやることにしました
  • 今回は公募という形は基本的にはとりません（弦楽器は除く）
  • 曲目は「マーラー1番」一曲のみ
  • 2018年7月16日（月）海の日
  • 森下文化センター レクホール
  • 詳細は公式Wikiを参照ください

【重要】参加費について

  • 一人3000円で確定とさせてください
  • このあと人数が増えていけば、当日もう少し金額下げられると思います

このページはパーカッションパートのエントリを行うページです

  • 参加希望の方は以下の内容を確認の上、このページから申し込みをお願いします
  • 不明点は公式Twitter https://twitter.com/gijin_okeへメンションかDMでお問い合わせください
  • 手弁当での開催のため、できる限りドタキャンは避けていただきたいと思いますが、どうしてもという場合は仕方ありませんので、事前にご連絡いただければと思います
  • 今回募集する人数は「3名」です
  • 以下を募集しています
    (1)大太鼓（with シンバル）
    (2)シンバル（合わせ、吊り）
    (3)トライアングル＋ドラ
  • エントリーされる方は上記3つのうち希望も記載願います
  • ただし重複の場合は相談させてください
  • 楽器はレンタルする予定ですが、もちろん持ち込みもOKです！

当日のタイムスケジュールについて

  • 2018/7/16の9:00〜17:00までで練習及び本番。
  • 会場は森下文化センター レクホールです。
  • 曲目はマーラーの交響曲第1番。（花の章はやりません）
  • 当日のタイムスケジュールは未定です。
  • 今回は一曲だけなので、午前中に1,2楽章、午後に3,4楽章で最後に通しかな、と。
  • お昼は当日各自で摂っていただく必要がありますのでよろしくお願いします。

参加費について

  • 一人3,000円です。
  • まだ人数の確定を見ておりませんが、当日の参加人数に応じて減額も考えています。

配置について

  • 対向配置を予定しておりますが、ちょっとまだわかりません。
  • 対向配置ならば、指揮者から見て、左手から1stVn,Cello（後ろにCb）,Viola,2ndVn。
  • 管打楽器ハープについては当日の状態を見ながら決めます。
  • 会場の広さとかちょっと把握出来てないので、朝ちょっとバタバタしたらごめんなさいm(__)m
