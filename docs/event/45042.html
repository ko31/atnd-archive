今年はAdvent Calendarで盛り上がりましょう。

Advent Calendarってなに？

クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーです。転じて、技術系コミュニティで、クリスマスまでの25日間にTipsを1つずつ公開するイベントをAdvent Calendarと呼ぶそうです。
( Ruby Advent Calendar jp: 2011より )

このイベントはなに？

PHP Advent Calendar 2011（全部俺）を真似して、12月1日から12月25日の間、Clojureに関する話題を参加者（全部俺）によって毎日一つずつ書いていくという企画です。Clojureに関する話題なら何でもOKです。

ルール

参加者（全部俺）は、日付の順番に、Clojureに関する記事を書いていきます。一番目なら、1日目、二番目なら2日目、といったように。もし参加者が25に満たない場合は、番号の倍数の日に、またClojureに関する話題を改めて書きます。
