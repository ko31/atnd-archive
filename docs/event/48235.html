description

教養のある人ならすぐに分かります。

夢は見るだけでは何も現実は変わらないということを。

頭の中で想像しただけ、

お金が舞い込むなんていうことはなく、

目標を達成するために行動することだけが、

本当に成果を挙げる方法だという事実を。

こんにちは。

副業で月収10万円以上稼がせる専門家、

副業ソムリエの秋田俊幸です。

あなたはこんな悩みがありませんか？

・セミナーで学んだだけで、全く実践できていない

・行動しようとしたが、三日坊主で終わってしまっている

・行動しても、途中で心が折れてやめてしまった

自分自身も3年前まで、ずっと悩んでいました。

でも、ある方法を学んだことで解決できました。

どんな環境であっても、

進み続ける行動力を得ることができました。

予言します。

あなたがこのセミナーに参加したら、

お金を稼ぎ続けるために行動し続けるという習慣が身に付きます。

どんな環境にいたとしてもです。

このセミナーに参加すれば、

環境に関係なく、自分の夢や目標のために、

進み続けることができます。

このセミナーは、昨年たった一度しかしなかったセミナーです。

前回参加した方の中に、

この目標設定の正しい方法を使った結果、

仕事で大きな成果を上げた方がいます。

実績のある方法です。

どんな内容か少し紹介すると、

・よくある目標設定で間違ってしまうこと。

・本当に行動できる正しい目標設定の秘密

・目標設定でしてはいけないこと

等です。

このセミナーでは、

ただ学ぶだけではない、ワーク形式で、

実際に取り組んでいただきます。

そして、行動したくてたまらなくなる、

正しい目標設定をします。

そうすることで、

あなたは自分自身の仕事面での目標達成に動き続けることができます。

ただし、このセミナーは今後開催をするかわかりません。

なぜかというと、

強力すぎる目標設定だからです。

設定する方向性を少しでも間違うと、

全然違う方向にいってしまい、

後戻りができなくなるまでになってしまう可能性があるからです。

そういうことが起きる可能性があるため、

人数を12名に限定して、今回公開することにしました。

参加者の方には4月からスタートダッシュを切って、

今年を飛躍の一年にしてもらうきっかけにしたいと思います。

このセミナーに参加された方には特典として、

どんな人でも絶対に行動させることができる

「門外不出の秘密のPDFファイル」　をプレゼントします。

さらに、

参加者のみの専用のグループにも招待する特典もつけます。

追伸：

今回限りの案内です。

早いもの勝ちです。

人数になり次第、申し込みを打ち切ります。
