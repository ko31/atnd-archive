description

オンラインゲーム業界でひそかに注目を集めつつある、ゲームの移管・協業・買取によるコンテンツの二次流通マーケット。"セカンダリ市場"とも呼ばれるこの分野に早くから注目し、さまざまなパブリッシャーのゲームの運営に携わってきたマイネットが、ゲームプランナー・ディレクター、デザイナー、エンジニアを対象とした中途採用説明会を実施します。

当日は、実際に某ゲームタイトルの移管に携わったゲームプランナー、エンジニア、デザイナーによるパネルディスカッションや、各職種ごとにブースを設けてLTなどを実施します。

また、会の終わりには現場で活躍する ゲームプランナー・ディレクター、 デザイナー、エンジニアとの自由交流会を予定しています。お酒・ソフトドリンク、軽食などもご用意しておりますので「まずは話を聞いてみたい」という方、ぜひご参加ください！


※オンラインゲーム業界への転職をご検討されている方により多くご参加いただく為、
　転職をご検討されていない方のご参加はご遠慮ください

プランナー・ディレクターブースは参加者が最低催行人数に足りなかったため、
　今月の発表は中止させていただきます。ご容赦ください。
