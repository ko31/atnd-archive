■NEC Cyber Security Cafe

NECグループでは、サイバーセキュリティをテーマとしたトークイベント「NEC Cyber Security Cafe」を定期的に開催しています。

今回は学生の皆さんを対象に、NECのCSIRTと、セキュリティインテリジェンスに関する講演を実施します。ご興味がある方は是非ご参加ください。

日時：2017年9月30日(土)　東京　14:00 ～ 17:30

会場：NEC本社ビル
　　　　東京都港区芝五丁目7-1
　　　　(JR田町駅西口より徒歩5分、都営三田線/浅草線 三田駅A10出口より徒歩1分)
　　　　　http://jpn.nec.com/profile/branch/headquarter.html

内容：＜講演＞
　　　　・激化するサイバー攻撃の実態とNEC-CSIRTの活動
　　　　　～フォレンジック解析・マルウェア解析を中心として～
　　　　・NECにおける Cyber Threat Intelligence の取組み紹介
　　　＜懇親会＞
　　　　・NECのサイバーセキュリティ技術者との交流会

※ ATNDの「イベントに参加登録する」ボタンを押しても、登録は完了しません。
　お手数ですが参加登録後に csevent at cyber.jp.nec.com まで
　・ATNDのID名
　・メールアドレス
　・氏名
　・所属大学名（学部・研究科/学科・専攻）
　・学年および課程（学部／修士／博士）
　をメールにてご連絡いただくことで参加申し込みとさせていただきます。
　入館の都合上、匿名でのご参加はできませんのでご了承ください。

参加費：無料

参加資格： サイバーセキュリティに関心がある大学生・大学院生
※年齢の確認できる身分証（学生証等）をお持ちください。

その他：
・途中退出はできません。

・交通費は支給されません。ご注意ください。
・懇親会は満20才未満の方の参加はご遠慮いただいております。予めご承知おきください。
