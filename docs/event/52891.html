持参物として，以下を推奨します．
1. 「計算機プログラムの構造と解釈第二版」（サスマン他，和田英一訳）
ピアソン・エデュケーション版（２０００），または，翔泳社版（２０１４）
または，日本語ダウンロード版
http://sicp.iijlab.net/fulltext/
または，英語版
http://mitpress.mit.edu/catalog/item/default.asp?ttype=2&tid=3305
または，英語ダウンロード版
http://mitpress.mit.edu/sicp/
2. 予習したい方は，
http://ocw.mit.edu/courses/electrical-engineering-and-computer-science/6-001-structure-and-interpretation-of-computer-programs-spring-2005/
その他，SICPで検索すると，日本語のページも多数あります．
3. ノートパソコンに，Scheme処理系をインストールしておいてください．
例えば，Dr.Racket(Dr.Schemeの後継)があります．
http://racket-lang.org/
おそらく，他のScheme処理系でも大丈夫と思います．（サポートに限界はあ
りますが．）
4. 様々な質問
2章の途中までなら，お答えできるのではないかと思います．
