description

小学生時代、楽しい夏休みと共に、嫌々ながらやった宿題。

最初に終わらせて思いっきり遊んじゃう！という方も、

最終日に泣きながらやった…という方もいると思います。

先生たちはそれだけでは飽き足らず、自由研究という課題も突きつけてきたはずです。

あの頃の私達にはただの苦痛…！

しかし、大人になった今、知識や行動範囲も増え、

それぞれ調べてみたいこと、作ってみたいものがあると思います。

「今の自分のスペックで自由研究したい！！」

そんな少年少女の心をお持ちの皆様、

やりましょう！

時期もほぼ夏休み期間にあわせてみました。

あの頃の夏休み気分をもう一度味わってみませんか？

参加条件：20歳以上の大人であること。
