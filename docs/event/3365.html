catch BaserCMSテーマ制作のハンズオンセミナーです。
description

３月27日（土）、BaserCMS の初の勉強会を開催いたします。
BaserCMS の導入から、テーマ制作までができるようにハンズオン形式での勉強会を予定しています。
かなりぐでぐでとなる可能性が高いです。
高度な勉強会は期待されないようお願いします。（ある意味アットホームな感じで）


日時: 2010年3月27日（土）15:00 – 19:00
費用: 無料
募集人数: 20名
会場: AIPカフェ
住所: 福岡県福岡市中央区大名1丁目14-28 第一松村ビル3階 302
参加資格: BaserCMS に興味がある方なら誰でも可
用意するもの:
次のものはなくてもいいですがあった方がより理解を深めれると思います
・ノートパソコン
・PHPが実行できる環境（XAMPP、レンタルサーバーなど）
※ 当会場にインターネット接続の用意あり


懇親会: 笑福や（福岡市中央区今泉2-4-35　今泉セントラルビル４階）
