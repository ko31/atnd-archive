■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

　それと今回は、a-blog cmsを開発されている
　アップルップルさんがゲストとして来られます。

　次の日の27(日)に本格的なa-blog cmsセミナーを
　やる予定です。

　a-blog cms紹介セミナー広島
　http://atnd.org/events/1496/


■現在予定している内容：
　●セッション１(25分)
　『Google Analyticsの基本的な使い方と見方のおさらい』藤本


　＜内容＞
　Google Analyticsを解説してある書籍
　「できる100ワザ Google Analytics 増補改訂版
　　SEO&SEMを強化する新・アクセス解析術」
　この本を元に、Google Analyticsの基本的な使い方、
　見方のおさらいと、100ワザのうち特に知っておいた方が
　いいと思うものをピックアップして紹介します。

　＜藤本の紹介＞
　・Google Analyticsが出始めの頃、使い始めてみると
　　他のアクセス解析よりあまりにも機能が優れていて
　　これで無料というのは信じられないと思いながら
　　今まで使っていたアクセス解析を一気に乗り換えたのを今だに覚えています。
　・WEB業界は５年目で、わりと広く浅くやっています。
　・基本的にWEB技術とか仕事とか関係なく、人間が好きです。
　　人類みんな兄弟だと思っているので初対面の人でも
　　３年ぶりに会った友人かのようにいきなり慣れ慣れしく
　　話すと思うので注意してくださいw こちらの心は常にフルオープンです。

　●セッション２(45分)
　『そろそろRubyについて言っておくか』赤木さん

　＜内容＞
　Rubyってなんだ？
　ノンプログラマへ
　マニアックじゃないプログラミングの話
　Rubyを使った事がないプログラマへ
　Rubyの魅力

　＜赤木さんの紹介＞
　名前：赤木（旧姓：にしゅう）
　blog：http://d.hatena.ne.jp/nisyu/
　あいかわらずFlex使ってますが、Rubyは楽しみに使ってます。
　この夏は、サマーウォーズで感動しました。泣けますよ。
　あとジンジャーエールがマイブーム。
　元DTP出身。
　CSS Nite in HIROSHIMA 副委員長

　●セッション３(45分)
　『Web制作者のための a-blog cms』やまもとかずみちさん

　＜内容＞
　2004年にリリースした「a-blog」は、現在2000ライセンスを超える有償での
　利用実績があり、その後継のシステムとして、2009年6月に「a-blog cms」
　がリリースされました。45分で全てを紹介する事は難しいので、今回は
　テーマやカスタムフィールド等についてを紹介させて頂きます。

　＜やまもとさんの紹介＞
　2000年秋に最初の「a-nikki」という日記 CGI を公開し、
　a-news,a-blogを開発、そして今年 a-blog cms をリリースしました。
　また、名古屋のWeb業界を盛り上げるための活動として、
　WCAN という Web 制作者のための勉強会の主宰を2000年より行っております。
