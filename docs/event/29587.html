場所は："MACORE"っていうとこです。マコーレって読むらしい。

JR（中央/総武線）西荻窪です。
方角のない「荻窪」ではなく「"西"荻窪」。

平日です。プチNEET期間なのでとりあえず、勉強したいです。
一人だとトイレに行った時の盗難とかが怖いのでみんなでモクモクします。
電源ありますが、限りがあるのでタップとか持っていったほうがいいかもしれません。
禁煙席に　2人用席に1電源×4席　で電源は4つ確認してます。
喫煙席は隔離された部屋がありますが、嫌煙なので確認してません。

Wifiはありますが、契約する必要あります　※私がもっていくので使っていいです。
朝から晩までモクモクしたいですが、疲れるまでいます。
※途中で抜けるかもしれません

私は気分で以下のどれかしますが、何してもいいです（犯罪行為、店内で鬼ごっこはダメ）。タイ語とかMBAの勉強などIT以外とかもOK

enchant.js, Django, WebSockets, Unity, 読書, iOS開発, cocos2d-x、PHP資格取得
