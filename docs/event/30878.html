AIR 3 から搭載された「Native Extension ( ネイティブ拡張 ) 」のラボです。
AKABANAセミナーの「Native extension Lab」シリーズの大阪版です。

ANEとは…

AIR Native Extension ( ネイティブ拡張 ) の略称。
AIRは、Flashのテクノロジーをブラウザ上だけでなく、
デスクトップやスマートフォン上で動作するアプリケーションも作成可能です。

しかしながら、各プラットフォーム固有の機能にアクセスする事までは、
できませんでした。

そのような機能を実現する為に生まれたのが、AIR3からの新機能、
AIR Native Extension ( ネイティブ拡張 )、通称ANEと呼ばれるものです。

ここでは、ANEをテーマに、今までできなかった
プラットフォーム固有の機能にアクセスするライブラリ開発、
ANEを使ったアプリ開発、ANEに関わる情報などを参加者同士で共有していく場となっています。
