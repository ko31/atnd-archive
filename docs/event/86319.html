description

『共感力』を武器に、
ブログもメールマガジンも持たず、
セミナーやイベント開催は初心者で、
自主開催イベントに、
たった2名の知り合いしか
呼べなかった私が、
そのわずか41日後の平日の夜に

『広告費は0円』

で、114名もの方々に
ご参加いただけたのは、
インターネットのおかげなんです！

　
今回はwebマーケティングの
初心者の方や苦手としている方、
創業期の方々に対して、
カネもコネも実績も無かった私が
webを中心にリアルとの融合で、
どのようにインターネットを使って
イベントの告知を行い、
イベントページを育て、
リアルでお会いした方々に
その内容を魅力的に、
尚且つ参加してみたい！
と思っていただける様に
ご案内をしていったかなどを
ご紹介させていただきます。

逆も然りですが、
webでウマくいったコトを
リアルでも実施することで、
相乗効果で集客と売り上げアップに
繋がっていくのです！

インターネットを利用して
お取り扱いをなさってある
商品やサービス、
手にしてある知識やスキルを告知し、
お客様集めや売り上げアップを考えてある
初心者の方や苦手な方、創業期の方々は
お気軽にご参加ください！！

ーーーーーーーーーーーーーーーーーーー

趣味と実益を兼ねて
というコトもあったのですが、
私は昔から頼まれもせずに
コンパや結婚式の二次会の幹事を
買って出ておりました。

結婚式の二次会は
そもそも人集めをする必要がなく、
またコンパも出会いを求める
男性や女性は私の周りに
たくさんいらっしゃったので、
イベントごとに人を集める
苦労をしたことはございませんでした。

今考えれば当たり前なのですが、
そもそも需要のあるコトを
企画しているから来てくださる人が
すぐに集まるのは当たり前のことでした。

　
そして時は2013年の9月6日。

情報7daysにも出演されてあり、
たくさんの著書をお書きになられている
齋藤孝さんの書籍で
「偏愛マップ」という、
コミュニケーションツールが
ご紹介されてあり、
どれだけ盛り上がるか大学生を相手に
斎藤さんが実験した結果、
実験の場が予想以上に盛り上がって
話し足りなそうだったので、
二次会を提案したら全員が参加するほど
お互いの距離が縮まったり、
どちらかと言えば苦手にしていた上司と
偏愛マップを交換して話しをしてみたら、
意外な共通点が見つかって、
以降は色々と話がはずむ様になったとか、
極め付けは初対面の
40代と思しき男性同士が、
出会って5分ほどしかったってないのに
立ち上がって握手をし始めるほどの
共通の話題が見つかるなど、
非常に魅力的な
コミュニケーションツールと思え、
これを使って
イベントをしてみたいと思い、
盛り上がれば継続的なイベントにしたり、
コンパや会社の懇親会などで
利用しようと思って、
生まれて初めて自分が良いと思える
コンテンツを使ってお金をいただいて
イベントを開催することにしたのです。

結果は冒頭に書いている通り、
ご参加いただけたのはたったの2名で、
しかもお二方とも親しい方で、
来てくれるよね！　
予定ないって言いよったよね！！
と半ば強制的に
ご参加いただいた状態でした。

このコトを通して、
自分が如何にそれを良いと思っても、
聞いてくださる方がそのコトに共感して
参加したい！　と思っていただく
告知の仕方やお誘いの仕方をしない限り、
親しい方ですら
お金と時間を使ってくださるのは
難しいコトなのだと実感いたしました。

　
そしてそこから41日後。

ご縁があって「わらしべ長者の会」という
マーケティングセミナーの主催を
させていただく機会をいただきました。

講師の方は当時2時間で50万円の
コンサルフィーが
発生する方だったのですが、
自分の講演料はいらないから条件を
一つだけ満たしてくれと言われたのです。

その条件というのは、

「セミナーに100名以上ご参加いただく」

ということでした。

　
前述の通り2名集めるのに
ひーこら苦労していた私です。
さて、どうしたものか。
そんな私がご参加してくださる方を
100名以上も募るなんて……

ということで、
まずフェイスブックで
イベントページを立ち上げ、
以前にこの会の主催をなさった方々に
どのようなコトをなさったのかを聞き、
同じ様な規模のイベントを
開催なさったコトがある方に
お話しを聞きに行かせていただいたり、
同規模のイベントに参加したりしながら、
イベントの告知をさせていただいたり、
そのイベントで親しくなった方に
ご案内をさせていただきました。

どのようなコトを行ってきたかの詳細は、
今回のセミナーで
お伝えさせていただこうと思いますが、
一つだけお伝えするとすると、
フェイスブックのイベントページに
関してお伝えすると、
イベントページを作って繋がりのある方を
イベントページに招待して終わり。
という方が非常に多いのですが、
4年前の当時はイベントページへの
招待の仕方が分からない方が
いらっしゃったので、
その方法のスライドを動画にしてみたり、
会場までの道順を
これまたスライド動画にしたり、
先に行われた他県での
わらしべ長者の会にご参加なさった方の
感想をアップさせていただいたりと、
参加人数の経過やセミナーに期待感を
抱いていただける様な投稿を行うことで
イベントの露出と感心・興味を引く
仕掛けを行っておりました。

その様にコツコツ積み重ねてきたことと、
多くの方々に共感・応援・ご協力を
いただけたおかげで、
私がイベントページにご招待した人数は
400名程度だったのですが、
最終的にイベントページへ
ご招待された方々の人数は
1,632名にまで増え、
無事に114名の方々に
ご参加をいただくことができました。

ーーーーーーーーーーーーーーーーーーー

これをキッカケに多くの方々と
ご縁をいただくことができ、
その後自分でセミナーを開催したり、
ご依頼をいただいて
伝えたいスキルや知識のある方々の
イベントやセミナーを
プランニングしたり、
プロデュースさせていただくようになり、
その方々のイベントやセミナーも
もちろんインターネットを使って
広告費0円で告知を行い、
イベントやセミナーを
行ってまいりました。
　
上記の様なコトを
コツコツと続けてきたおかげで、

「一緒にやりましょう！」

とお声かけいただく機会が増えて、
今では先日の2月27日に
西日本新聞の一面に取り上げていただけた

一般財団法人
サイバーセキュリティ財団

で幹事の役目もいただいております。

　
上記を自慢したいのではなく、
私ができるコト、
持っている知識やスキルを使って、
インターネット上で
情報発信をし続けてきたからこそ、
その様なご縁や機会と巡り会うことが
できたということを、
皆様にお伝えしたいのです。

皆様がお持ちの商品やサービス、
知識やスキルを必要となさってる方が
世の中にはたくさんいらっしゃいます。

でも必要となさってる方々に、
そのお悩みを解決できること、
より快適な状態になっていただけること
などを発信して知っていただかない限り、
皆様がお持ちの商品やサービス、
知識やスキルは世の中に無いのと
同じことなのです。

私の実績紹介としては
イベント集客が主になっておりますが、
その内容がご参加いただいた方々に
多くの気づきや学びを与えるという、
イベントの魅力や参加していただく
意義をお伝えする情報を発信することと、
取り扱うモノ・コトが商品やサービス、
知識やスキルがお悩みを解決できること、
より快適な状態になっていただける
という情報発信をするのは、

「消費行動を促す」

という点では全く同じであり、
発信する情報の質と量に見合う
リアクションが必ずあります。
　

ですので今回のセミナーでは、
皆様が取り扱ってある商品やサービス、
知識やスキルの魅力や強みを
どのように情報発信を行えば良いのか、
webマーケティング、インターネットの
初心者の方や苦手な方にも分かりやすく、
創業期の方々にはどのようなことを
意識して情報発信を行えば、
その情報を必要となさってる方に
届きやすくなるかを
お話しさせていただきます！

販促物のプランニングと
