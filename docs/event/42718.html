description

もくもく会とは、みんなで集まって、各自が黙々と作業をする会合です。

自宅などで一人で作業しても集中できずに効率が落ちたり、考えに詰まったり悩んだりして進まないことがあります。もくもく会であれば、周りに刺激を受けて集中できたり、他の参加者に質問したり一緒に議論したりできますので、そんな問題も解決できるかと思います。

みんなで楽しくもくもくと作業しましょう。

作業内容はIT関連であれば何でも構いません。アプリを開発したり、書籍を読んだり、勉強したり、デザインしたり…などなど、他の参加者の邪魔にならなければ好きなことをしていただいて構いません。

タイムテーブル

時間 内容
12:00 会場オープン(集合は都合のよい時刻でOKです)
12:00 – 17:50 各自で作業。任意で作業内容の紹介をしていただいてもOKです
17:50 – 18:00 撤収作業

終わったら、伏見・栄あたりで懇親会をできればと思います。
