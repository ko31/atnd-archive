映画『ラ・ラ・ランド』をもっと楽しむための映画交流会

元町映画館の招待券付き!

ミュージカル映画の歴史や参照映画を30本以上紹介予定！

 

2017年の大注目！

みなさん、もう『ラ・ラ・ランド』をご覧になりましたか？
語りたいこと満載ですよね。

例えば、2016年大ヒット映画『シン・ゴジラ』的に、過去映画への“オマージュ”満載で、元ネタ探しも楽しい。

そのため奥深く語れるし、
最高に楽しくて、猛烈にドキドキで、極限に切ない。

いますぐ語りたくて仕方ない。
ほかの人の感想も聞きたい！

そこで「元映シネマトーク」と題した第1弾として、大注目映画『ラ・ラ・ランド』を軸に、元ネタとなる、いま見ておきたいミュージカル映画や関連映画の紹介、演劇も含むミュージカルの基礎知識や歴史など、この機会にご紹介。

100%『ラ・ラ・ランド』鑑賞に役立つ2時間を目指します。
きっともう一度『ラ・ラ・ランド』を見たくなります！

それだけでなく、週末にレンタルしたくなるDVDが一気に増えること間違いなし。
もちろんジャズも聴きたくなりますよ。

元町映画館での近日公開の、デヴィッド・ボウイ映画『ジギー・スターダスト』についてのお話もあるかも。
http://ziggystardust.onlyhearts.co.jp
http://www.motoei.com/film.html

元町映画館の招待券付きなので、かなりお得です。

 

【参加対象】

映画『ラ・ラ・ランド』を語りたい方が前提ですが、ネタバレを気にしない映画好きならどなたでも。
ネタバレでも楽しめる映画です。

 

【開催内容（予定）】

・2017年大必見『ラ・ラ・ランド』のご紹介
　シナリオの面白さもちょっとだけ解説します（たぶん日本初！）
・映画『ラ・ラ・ランド』とあわせて見ておきたいミュージカル映画のご紹介
・ミュージカル映画＆音楽映画の基礎知識（歴史や必見映画の紹介）
・ジャズ映画も押さえたい
・参加者同士で『ラ・ラ・ランド』とオススメ映画について語り合います
　ぜひオススメ映画を語ってください

参加すれば、すぐに見たい「ミュージカル映画＆音楽映画」が一気に増えて、もう一度『ラ・ラ・ランド』を見たくなります。

参加費：1500円（元町映画館の招待券付き）
