概要

シェル芸勉強会にリモート参加します。今回は頭の体操が主な内容みたいですね。

プログラミングというか情報処理の基本中の基本を、改めて考えてみませんか？情報を使うだけでなく活用するということは、IT専門家だけでなく誰でも可能性を持っています。

情報処理をする上で、UNIX系の世界はとてもシンプルでわかりやすいのではないかと思います。今更と思われがちな、シェルやテキスト処理を中心とした世界が忘れ去れていくのは悲しい。

バリバリのプログラマな方はもちろん、コンピューターを情報処理に活用するという面で、IT専門家と呼ばれる人以外にもUNIX文化を伝えることが出来るといいなと思います。

下記リンクの本家勉強会に、午後からの部でUST遠隔参加します。会場が前回と違いますのでご注意ください。

参加費は無料です。無線LAN有。

  • 持ち物
    1. UNIX環境が利用できるノートPC（OSXなMacも可）
    2. おやつは300円まで

Windowsの場合は、VirtualBoxにUNIX環境を入れればオケ。Cygwinはあまりお勧め出来ないかもしれません。日本語対応の端末と日本語入力はあったほうがいいです。（日本語データを扱う場合がある）
GNU版のsed、awk、grepなどを使うことが予想されます。Linux系ならまず大丈夫。
また集計などで便利なOpen usp Tukubaiコマンドもオススメ。

  • GNU系のコマンド
    MacはBSD系なので、homebrewで下記をインストール。
    $ brew install coreutils
    $ brew install gnu-sed
    $ brew install gawk
    $ brew tap homebrew/dupes
    $ brew install homebrew/dupes/grep
    高速化したGNU grepをインストールする
    FreeBSDの場合はportsからインストールお願いします。
  • Open usp Tukubai
    入手先は下記のリンクを参照。わからない場合はサポートさせていただきます。
    FreeBSDの場合はportsからインストール可。
