今回のあらすじ

1st , 2nd に引き続き、気楽に参加できるテクニカルお茶会mixi unwired 3回目の開催です。

今回はさらに趣向をかえて、Mac OS Xに標準装備されているグラフィック・プログラミング環境 Quartz Composer をつかって、VJ的にいろんなモノを投射してみるセッションを行います。


まずはVJって何? ビデオ・アートって何? というお話から、
Quartz Composerでグラフィック・オブジェクトを作り、
それに動きを仕込んで、組み合わせていってみるという実演をおこないます。

何か動いてるものを見るのが好き、ビジュアル・プログラミングってどんなもの?
Mac OS Xでひととひと味ちがう何かおもしろいことをやってみたい! と思ったみなさん、ぜひお気軽にご参加ください!

当日は、エレポップ・テクノを打ち込んでいたらいつのまにか25年後のいまはループ即興系にQuartz Composerで映像をあわせていた、という @shakuji ( mrmt.net ) と、コラージュ好きな工場長 @imai_factory ( imai-factory – flickr ) が解説・セッション・パフォーマンスをお送りします。

場所

会場となるJELLY JELLY CAFEは、ふだんはみんなでワイワイ楽しく「co-working」できるカフェです。
毎日夜18:00〜はBarとしても営業しており、ドリンク・フードALL500円です!
今回もJELLY JELLY CAFEのスペースをお借りして開催させていただきます。

募集定員は、今回も仮に15名とさせていただいていますが、
完全貸切ではありませんので当日ふらりとご参加いただいてもOKです。 (満席の時はゴメンナサイ!!)

いままで同様、ATNDからお申し込みの上参加された方にはワンドリンクを株式会社ミクシィからオゴリとさせていただきます!

コワーキングスペースってどんな所か見てみたい、という方も
Quartz Composerを触ったことがないという方も
仕事帰りや仲間との遊び帰りになど、気楽な形で結構です。ぜひ足を運んでみてください!
