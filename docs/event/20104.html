description

阿佐ヶ谷の小さいお花屋さん「フラコッタデコ」が、ご近所のお茶のお店とスィーツのお店を誘って、
フラワーレッスン＆お茶会を開きます。

第１回目のテーマは、「ハーブ」。
ハーブを使いながら、テーブルを素敵に演出するお花のレッスンを楽しみながら、
お客様をお招きした際の美味しいハーブティーの淹れ方、楽しみ方をハーブティー専門店店主からレクチャーいただきます。そして、お待ちかねのスィーツは南阿佐ヶ谷の知る人ぞ知る名店「Heartstrings」のレモンシフォンケーキ。ハーブティーは　はぐら茶屋が担当いたします！

盛りだくさんの内容でお届けいたします！！
場所は、JR阿佐ヶ谷駅徒歩2分にある細田工務店・杉並リボン館。
ゆっくりゆったり流れる時間をご一緒しましょう。

参加費：　3,000yen(花材、レッスン費、資材、お茶代、スィーツ含む）

持ち物
・お気に入りのカップ・茶器
・花きりはさみ（お持ちの方）※お持ちでない方は貸し出しします。
・お持ち帰り袋
