公益財団法人品川文化振興事業団（東京都品川区／理事長：中尾根剛）は、品川ゆかりのアーティストを支援するとともに、様々な方がアートを身近に感じられるイベントを開催します。
品川アーティスト7人展ではその名の通り、品川に縁のあるアーティスト7人が写真、イラスト、ガラス工芸、模型制作などの作品を展示。アーティストと身近に触れ合えるアットホームな空間が魅力です。

■日程　　平成29年11月3日（金・祝）、4日（土）、5日（日）
　　　　　午前10時～午後5時　最終日は午後3時まで

■会場　　スクエア荏原　４Ｆ展示室　
　　　　　（品川区立荏原平塚総合区民会館／品川区荏原4－5－28）
　　　　　東急目黒線　武蔵小山駅　徒歩約10分
　　　　　東急池上線　戸越銀座　徒歩10分

■内容　　アート作品の展示　（ガラス工芸、イラスト、模型、写真）
　　　　　　　ワークショップ（似顔絵スケッチ）
　
■主　催　　品川区／（公財）品川文化振興事業団
