「興味がある」「使ってみたい」という方は多いはず！

そんなハンズオンから早3ヶ月・・

ご好評につき、第2弾を開催します！

前回同様、講師やサポーターに質問をしながら、
Amazon EC2 のサーバを立ちあげてみましょう！

今回はEC2をさらに便利に使える機能、
AMIとEBSについてもハンズオンを行います!

・Amazon EC2を便利にご利用したい方
・インフラに興味のある方

この機会に、ぜひ体験してみてください！

■ 対象

Amazon EC2 を利用してみたい方・インフラに興味のある方

Amazon Web Servicesをご利用いただくためには、
事前のアカウント取得が必要です。
アカウントの取得はこちらをご参照ください。

http://aws.amazon.com/jp/register-flow/

※アカウント取得はクレジットカードが必要となります。

セッションに参加される方はクレジットカードをお持ちください。
