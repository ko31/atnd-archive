　環境NGOの環境教育フォーラム21（東京・千代田）は、12月実施の第19回eco検定合格に向けて、全国主要都市で「重要ポイント集中講座」（スケジュールは下記参照）を開催します。重要ポイント集中講座では地球温暖化、循環型社会の構築、エネルギー問題、環境と経営などeco検定に出題される問題をテーマ別にわかりやすく講義します。eco検定で出題されやすい時事的な問題もテーマにあわせて解説します。
◇詳細はホームページ　http://ecoken.newsmedia.jp/

※開催時間は原則午前10時－午後4時半。10月30日広島市は午後0時半から午後6時半となります。
