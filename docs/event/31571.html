どんなことするの？

セッションはご参加の皆さま同士のお話を持ち寄る会です。

WordPress に関係していることでしたら、サーバー保守のことや取引先とのコミュニケーションの問題など、何でも構わないと思います。いま取り組んでいること、分からないことや、漠然としていても構いません。これから学びたいことをご自由にお話下さい。

具体的な疑問・課題などその場で解決できることは少し解決を試みて（あまり皆でムリをせずに）、午後からの WordBench ご参加の皆さまとも課題を共有していけましたらうれしいです。

朝、とても早いです。ご入場が遅くなっても構いません。たとえ遅刻しそうになられてもあまり走らずに、ゆったりした気分で会場にたどり着いて下さい。ボクは遅れるワケにいきませんので、近くのホテルに泊まります。＾＾

午前中のゆるゆるカフェは、午後からのWordBenchをみなで盛り上げていきたいという願いを込めて、今回せんえつながらお申し出させて頂いております。

少し眠気が取れないとお感じになられたら、ご遠慮なく途中退場されて、午後のWordBench に向けて体調を整えられてほしいです。私もお昼の時間は少し皆さまと離れて仮眠などさせて頂きますこと、予めご容赦を下さい。

長く書いてしまいましたけれど、お集まり頂ける人数や皆さまのご興味・関心に応じてセッションの時間配分や組み方は柔軟に変えていきます。またお休みタイムも適時取っていきたいと思います。皆さまからのリクエスト・ご意見なども、コメント・メールなどでお気軽にお伝え下さい。

お目にかかれる皆さま、楽しみにしております。
