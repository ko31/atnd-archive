description

現代の日本では、会社員の給料やボーナスは下がる一方。転職やリストラも茶飯事に起こっています。
働いているにも関わらず貧しいという方、つまりワーキングプアの人がどんどん増えていっています。

ちなみに日本で年収が200万円以下の人口は約1069万人です。

正社員でもギリギリの生活さえ維持が困難、給料が少ないにも関わらず拘束時間が長い派遣社員、
生活保護の水準以下の収入しかもらえない社員、契約社員などなど。

「いい大学を出ていい会社に就職すれば安定・・・。」
この時代はもうとっくに終わりました。

今の時代は、収入源を複数確保するということが必須になります。
会社から給料をただただもらっているだけでは危険です。

「今の収入では納得がいかない、将来が不安。」
「今の状況から脱出して収入を増やしたい！」

そんなあなたへ向けてセミナーを開催します。

「現代の日本が実際どういう状況なのか。

これから世の中はどうなっていくのか。

それを把握し私たちはどう対処していけばいいのか。」

このことについて今回詳しくお話していきます。

※参加条件は20歳以上の無職、フリーター、学生、会社員などの、雇われの方限定となります。未成年は参加できません。

メモ帳と筆記用具はお持ちして頂くことをおすすめします。
