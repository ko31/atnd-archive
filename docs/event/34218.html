趣旨
さわってみよう Firefox OS meets HTML5＋α @福岡

Mozilla Japanの浅井智也さんとFirefox OS にパッチを貢献しているmasapさん (@masap) にお越し頂き、11月に東京で行われた「さわってみよう Firefox OS」の福岡版を、ギルドカフェコスタにて開催致します。

「OS」と聞くと難しい！との印象があるかと思いますが、登場したばかりの新しいOSを、デザインができる人とプログラム書ける人が一緒になって、「ココを変えたら、こんな表示になった！スゲー！カッコイイー！！」みたいにワイワイ触ってみる楽しいイベントにできたらと考えています。

Firefox OSとは？

Firefox OSでアプリを作ってみよう！－勉強会「さわってみよう Firefox OS」 (マイナビニュース)
第1回　ブラウザから生まれた「Firefox OS」とは何か？ (gihyo.jp)
Mozilla Firefox Mobile OS Exclusive Preview (YouTube)
Firefox OS カスタムROMの作成 (YouTube)
Firefox OS カスタムROMの作成 (slideshare)

なぜ今Firefox OSをさわる必要があるの？
今まで、アプリの作成というと、それなりに高いハードルがありましたが、このFirefox OSにおいては、そんな難しいことは必要なくて、HTML5がその架け橋になります。Webの知識でアプリが作れるという、こんなに下がった敷居を黙って見とくのはもったいないと思いませんか？
