勝手にプロモーション甲子園とは「ソーシャルメディアを活用したマーケティング企画力を競うアイディア会議イベント」です。

多くの企業が取り組み始めたソーシャルメディアマーケティング。しかし成功事例はまだ少なく、ソーシャルメディアをどう扱ってよいか戸惑っている企業が多いのが実情です。

「 ソーシャルメディアを使った方が良いのはわかるけれど、何をしたらよいのかわからない。」
「 Facebookページを開設したけど、プロモーションにどう活用できるかわからない。」

そんな声が企業の担当者や、代理店から多く聞こえてきます。

そこで、私たちは情報収集だけではなく仮想実践をすることで経験値を貯める取り組みを考えました。 職種や会社の枠組みを越えて実際にプロモーションを企画して練習をすることで、いざ仕事をうけた時により熟練度の高い企画を考えることができる…いわば、企画の練習試合！

より実践的な内容にするために、依頼主としてモデル企業から課題の提供をして頂き、チームを作って参加者が企画をの提案をします。

職種や会社の枠組を飛び越えて、一緒に明日のマーケティングをつくりませんか？
きっと素晴らしい発見があるはず。ぜひご参加ください！

◆対象者

Facebookページの運用をされている方、 Facebookでプロモーションをしたいけれど何をしたらよいか悩んでいる方、会社でソーシャルメディアを使う予定はないけれど、どんな活用ができるか興味のある方。

職種は問わず。とにかくソーシャルメディアを活用したマーケティングに興味のある方が対象のイベントです。

◆モデル企業

第二回目のモデル企業は株式会社東急ハンズ様。
当日はソーシャルメディア運用担当の方にもお越しいただきます。

◆企画テーマ

今回は株式会社東急ハンズさんを招いて、ユーザーもハンズも楽しめる企画をテーマに企画提案を行います。

「ソーシャルメディア」「ソーシャルメディアと店舗」「ソーシャルメディアとWebサイト」など様々なメディアを活用して、「思わず誰かに言いたくなってしまうような」拡散性のある楽しい企画をみんなで考えてみましょう。

※テーマや東急ハンズ様の過去のキャンペーンなどに関する情報は5/30（水）ソシャマノートブログに掲載予定の予習記事で改めて詳しくご紹介します。
