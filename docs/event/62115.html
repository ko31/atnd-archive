METユニオン（半面）　サバゲ会

title METユニオン（半面）　サバゲ会
catch 新感覚フィールドらしいぞ！(当会初貸切)
description

METユニオン（L側フィールド）でのサバゲ会を2015年2月14日(土)に開催いたします。　今回の運営進行は、朝木とけろよんです。

基本的にはいつも通りのぬるい感じにゲーム内容ですので初心者の方も歓迎です。一日のゲーム数は14ゲーム前後。結構回していきます。

レギュレーションはごくごく普通で、弾数制限等はございません。詳細は以下の通り。

・セミオート・フルオート問わず全ての銃でパワーは0.98Ｊ以内
・使用BB弾はバイオのみ（プラスチック、セミバイオは禁止。）
・減速アダプタの使用は禁止
・基本的にブラインドファイア禁止
・レーザーサイトはすべて使用禁止
・ゾンビ行為禁止

その他サバゲアイテムの手投げグレネード、グレネード、クレイモアも使用可能です。

法定内での道具でしたら、基本的に使用可能ですが、安全面に配慮をした運用をお願いいたします。

参加申し込みですが、とりあえず2月12日いっぱいまでの受付とさせて頂きます。
ご友人の方で参加希望の方いらっしゃれば本ページをお教え願えますようお願いします。

こちらのサイトに登録されない（できない）ゲスト参加の方をお連れになられる場合
お手数ですがコメント欄に人数とその方のお名前（HNで可）をご記入頂くようお願いします

■ゲーム時間と第一ゲームまでの流れ

時間は08：00~17：00の予定です。
08：00頃にはフィールドゲートが開き、駐車場に入れるようになります。
終了は05：00、完全撤収。

■必要な物、参加費

・参加費　3500〜4000円（昼食カレー付き。　昼食不要の人はコメ欄に書いて下さい。）
・エアガン　（電動ガンレンタルあり。2500円。バッテリー、多弾マガジン1本セット)
・BB弾（バイオのみ）
・ゴーグル
・顔面を保護するマスク、四肢保護のニー、エルボーパッドや手袋があるとベター。
・結構、頭ぶつけると痛い所もあるので、帽子やヘルメットもあった方がベター。

■交通手段と送迎バスについて

基本的に現地集合現地解散ですので、レンタカーや自家用車の乗り合いはコメント欄等を使い個々に打ち合わせをしてください。

■ゲーム内容

基本的にはフラッグ戦メインです。
ゲームルールによっては、セミオート縛り等もあります。

全部で14ゲームは回す予定でいます。スムーズな進行にご協力ください。
「補給が間に合わない」「疲れた動けない」等は遠慮せずに休憩してください。30分もすれば次のゲームに入れます。

■食事について。

基本としてはフィールドでカレーライスが出ます。（参加費に含まれてます。）
ですので、昼食のカレーが要らない方はコメント欄に書いて下さい。
コンビニ弁当持参もOK
飲み物はセーフティに自販機がございます。

■ちょっとしたお願い　…けっこう大事。

締め切り等やその他諸事情で参加を伏せたい人も多いと思いますので、開催中や終了後は、
・個人単位で『今日サバゲして来た』と言うのは自由で。
・誰が参加していたかに関しては『触れない』方向でお願いします。
・TwitterやMixi等で『○○さんお疲れ様でした～』も出来るだけ空気読んで下さると助かる人も多いかと思います。
（自分が参加していた事を表明する事に問題ない人同士のやり取りについては特に問題ありません）
・ですので、基本的に本ページコメ欄使って頂くのを推奨します

■■雨天の場合■■
前日、午後0：00の天気予報にて降水確率60％以上の判断いたします。
