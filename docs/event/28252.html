description

「てれねこ」は照れたようにほっぺがじんわり染まるねこのブローチです。

最初はみんなで同じてれねこをつくります。LED基板はできているので、簡単な裁縫をしてもらいます。
できあがったら、他の布生地や自分で持ってきた材料を使って、好きな動物をつくって照れさせてみましょう。

日時：5/12(土)　14：00集合、2時間程度
定員：6名
場所：FabCafe渋谷
　　　　東京都渋谷区道玄坂1-22-7 道玄坂ピア1F
　　　　http://www.fabcafe.com/access
持ち物：自分で作りたい動物の材料(オプション、先生も何種類か持っていきます)
　　　　　LEDの光が透けるものがよいです。
　　　　　薄手のボアや明るい色の羊毛フェルトでも大丈夫です。
会費：1500円

ワンドリンク(400円まで)付です。
てれねこのLEDは赤、オレンジ、ピンクの3色から選んでもらいます。
