description

　電子書籍出版のインテリジェント出版（北海道虻田郡豊浦町幸町87-20　代
表　岡野英紀）は、『電子書籍出版無料セミナー＆ワークショップ』を11月
24日に札幌にて開催いたします。

　本セミナーは、最近話題の電子書籍について初心者やまだ触れたことがない
方でも分かりやすく電子書籍の利用方法や出版方法を解説し、電子書籍専用端
末やタブレット端末での実演も合わせて行います。

　また、前回10月28日に札幌で開催した「電子書籍出版入門セミナー」でご好
評の電子書籍出版公開コンサルティングをグループワークとして参加者全員で
行うワークショップも同時に実施して、参加者の方の電子書籍の出版を推進い
たします。

　そして、弊社のモットーである「地方から全世界への電子書籍の出版による
情報発信のパイオニアであれ。」という思いから、セミナーと同時に懇親会も
開催して、北海道の皆様と電子書籍の出版に関するノウハウを共有できるよう
なイベントといたしました。
