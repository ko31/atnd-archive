description

停電しても照明を自動的に確保したい，なるべく点灯時間を長くしたい，でもやり方がイマイチわからない，という人はいませんか.そこで，みんなでLED点灯回路の勉強会を開きます．

おもしろLED回路を思いついちゃった人から，高効率化のチューンアップを目指すアナログ屋さんまで，みなさんに参加して頂きたいです．

基本的には，参加頂く方は何かの発表をお願いします．改善点を残した作品，ちょっと自慢したい作品，思いついたアイディアの相談などなんでもOKです．
