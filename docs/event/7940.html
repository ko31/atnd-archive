■ふろしき王子による小鳥のためのふろしき活用術

ふろしきの結び方の基本から…
キャリーケース包み、小物入れ包み、ブランコ包みなど
小鳥飼いのために役立つ風呂敷の使い方を“ふろしき王子”こと横山功さんに学びます。
横山功さんもセキセイインコとジュウシマツを飼っている小鳥男子です。

横山功プロフィール
1979年浅草生まれ。武蔵野美術大学在学時、自分の部屋から出るたくさんのゴミに息が詰まり、エコロジーとして風呂敷を使い始める。ゼミのテーマは「風呂敷の研究」。毎日の生活、国内・海外旅行も風呂敷を結んだバッグ一つで出かけ、何をするにも風呂敷を使ってみる。使い続けていくうちに、次々とより便利で新しい風呂敷の技を発見し、単なるエコロジーを超えた万能ツールとしての風呂敷は頼もしい体の一部となっていく。数年後に風呂敷ブームが訪れるが、ただの流行で忘れられないような実践的な練習を主とした風呂敷ワークショップを各地で開催し、便利と安心を備えた和の心を伝え続けている。全国でのワークショップほか、テレビ、雑誌など出演多数。
