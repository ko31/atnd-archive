description

ビジネスをスムーズに進めるうえでは、顧客や関係者のニーズや顕在化していない前提条件を引きだすことが不可欠です。

本セミナーでは、有名超大手IT企業で営業から開発、事業企画に携わり、社内外のあらゆる関係者に対してコミュニケーションを
図ってきた女性中小企業診断士が、本当に使える傾聴のノウハウをお伝えします。

顧客のニーズを引き出し最適な提案のヒントにしたい時、関係者や部下へ自身の目的のために動いてもらいたい時、
課題解決のためのヒントを得たい時…社内外のあらゆるビジネスシーンで傾聴力は重要になる能力のひとつになります。

しかし、相手の思いや考えを汲み取り適切に引き出すことを苦手としている方も多いのではないでしょうか。

そこで、このセミナーでは傾聴の基本姿勢から、非言語的コミュニケーションによる信頼関係の構築、
どのような質問で快く回答を引きだすかまで、分かりやすく実践的なメソッドでお伝えします。

【講師紹介】
古川里奈　中小企業診断士。
大学卒業後、有名超大手IT企業で営業・開発・事業企画業務に従事。
社内外におけるさまざまな場面で、部署やロケーションなどの枠組みを超えた
コミュニケーションを図り課題解決を実施。

東京都中小企業診断士協会中央支部公認の大人気マスターコース
「売れる!人気プロ研修講師・コンサルタント養成講座」の6期生。
東京都中小企業診断士協会中央支部青年部のワンコインセミナーなど、
複数のセミナーに登壇。
