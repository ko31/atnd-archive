発表内容



■１時間目　ぱゆ

モデラーステップアップ　Blenderでできるこんなこと

メタセコイアでのモデル造型はＭＭＤモデラーさんの中ではもはや当たり前。今回は高機能かつまだまだＭＭＤユーザの少ないBlenderならば、こんな感じでモデリングができる！という部分の紹介。

メタセコと比べていい所悪い所を確認しながら、とりあえず踊ってみる所までいってみます。


■２時間目　包囲Ｐ

もうちょっとBlender知りたい？知りたくない？

ぱゆさんの発表からのステップアップとして、より具体的な操作方法や手順などを紹介。特殊な操作が手になじまないと言われがちなBlenderですが、難しくなく手にかけられるような設定などを駆使して、「フリル」などめんどくさそうな造形物を実際に作ってみたいと思います。


■３時間目　蛍の光Ｐ

IKと連動を使った 苦しんで覚えるボーン制御入門編

標準モデル構造でも「足」や「足首」に使われているIKですが
今回はそのIKボーンと、回転連動ボーンを使った例を紹介しながら説明していきます。
標準ボーン準拠のリギングはできた、じゃあ服やアクセサリは？動かすのはボーン手付け？
そんな、脱初心者のためのプラスアルファの手助けになればと思います。
去年出てきた「でもこれは時計じゃないですよね？」呼ばわりのアイツも一年ぶりに再登場。
