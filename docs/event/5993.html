description

２００８年１月にスタートしたベンチャーマッチング交流会。
これまでに僅か２年半で東京・大阪・神戸・名古屋・浜松・横浜・
大宮・高崎・宇都宮・沖縄で延べ５０回以上の開催、
そして５,０００社以上の皆様にご参加頂き、
名実ともに日本最大級の交流会として成長して参りました。

又、本交流会は、新聞・雑誌等の各種メディアでも多数取り上げられ、
今もっとも注目を浴びている交流会と言っても過言ではございません。

そしてこの度、「第５８回ベンチャーマッチング交流会」を
浜松で開催する運びとなりました。

しかも！今回は、浜松で今もっとも注目されている
ＩＴベンチャー企業２社よりスペシャル講演を頂き、
今後のビジネスへのヒントとチャンスを
掴んで頂ける絶好のチャンスでございます！

講演後も講師の方々を交えた懇親会を開催し、
参加者同士の直接的な情報交換の場をご提供し、
浜松地域の中小企業やベンチャー企業間の活性化に繋げて参ります。

「やらまいか精神」を持つベンチャー企業、
そしてベンチャー企業との交流から新たなビジネスチャンスを
掴みたい経営者の皆様に是非ご参加頂き、共に浜松を熱くして参りましょう！

皆様のご参加を心よりお待ち申し上げております。
