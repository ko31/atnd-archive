【ネットワークエンジニア向け無料転職セミナー】
『90分でマルわかり！SDN/SD-WAN 実例解説セミナー』
～実績紹介からみるプログデンスの技術力と取り組み～
難易度：★★★☆☆

ネットワークエンジニアとして転職活動中の方を対象に、90分でマルわかり！SDN/SD-WAN実例解説セミナーを、2019年2月20日(水)19:00より開催します。

当社では、オンプレミス環境からクラウド環境への転換や、レガシーなネットワークから SDN（Softwear Defined Network）への転換を積極的に実施し、SDN/SD-WAN についても黎明期から携わり、現在では国内 TOP クラスの豊富な経験があります。

本セミナーでは、SDN/SD-WAN の基本的な概念(考え方)やメリットについてのポイントや、実際に講師が Cisco ACI を操作し、操作画面をスクリーンに同時投影しながら実績の一例をご紹介します。 実績紹介を通して、当社エンジニアの技術力の高さや専門担当フェーズ、また IT 業界内での立ち位置など、一部ではありますが、魅力をお伝えしたいと思いますので、どうぞお気軽にご参加下さい！

【対象となる方】
・30代後半までの、ネットワークの運用、もしくは設計/構築経験がある方
・SDN/SD-WAN に興味がある方
・株式会社プログデンスに興味がある方
　※選考希望者には『面接確約』を保証させていただき、日程調整の受付をその場でさせていただきます。
