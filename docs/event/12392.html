description

現代美術の勉強会
現代美術に興味ある人なら誰でもウェルカム。
知識や情報・技術の集積をし、
美術の幅広さ、奥深さを知覚していきます。

かたちとしては流動的だが、大きな柱は３つ。

・作品のプレゼン、作品についての講評、合評、クリティーク。
・テーマをきめて次回までの宿題とする。各自予習。テーマに関する講師を呼んでレクチャー。
・札幌のアート関係者のネットワーク形成。

他にも、
道内外の展覧会を見たらレポートしてそれについて意見や感想をだしあう。
売れてる作家の作品を皆で研究して、なぜ売れてるか探る。
最終的に勉強した成果を冊子にまとめる。
専門用語解説や、展示の技術、ポートフォリオの作成法、制作する道具の情報交換などなど
勉強会の方向性や内容も含めてみんなでブレストしていく勉強会です。

初回は参加費500円。
