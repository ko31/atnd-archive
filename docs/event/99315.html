データドリブンマーケティングセミナー開催

ベントのお申込みはこちらのページからお願いいたします。

https://www.keywalker.co.jp/news/data-driven-marketing-seminar-2nd.html

国内においてビッグデータをマーケティングに活用できている企業はほとんどありません。Web上に存在するデータはさながら油田のように次から次へと湧き出てくる資源であり、これを活用することは競合他社に一歩先んじる大きなチャンスです。

ご参加をオススメしたい方

  • 「マーケティングを実施するうえで、データの活用の仕方が明確でない」
  • 「これからデータ活用に取り組みたいが、何から始めたらよいかわからない」
  • 「WEBクローリングで収集したいデータがある」
  • 「業務の自動化やデータ可視化ツールに興味がある」

登壇者紹介

株式会社キーウォーカー 代表取締役
真瀬 正義

2000年にキーウォーカーを創業し、人工知能、自然言語処理、検索エンジン、WEBクローラ、 等の研究開発を推進。現在では不動産業を含む幅広い業界の企業に対してデータ収集・可視化、業務の自動化ソリューションを通じた経営支援を行っている。

株式会社キーウォーカー　取締役
山本 喜士朗

IT業界での業務に30年以上従事し、技術面とビジネス面の両サイドでの幅広い知識と経験を持つ。キーウォーカーではソリューション営業の統括を行い、直近では不動産業界に対して、相場情報の提供、物件情報の可視化・分析など多岐に渡った提案を行っている。
