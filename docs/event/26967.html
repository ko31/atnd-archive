4/24追記 : 会場が (株)ネクサス北海道支店 に変更になりましたのでご注意ください。

北海道パケットキャプチャ勉強会「Hokkaido.cap」の第10回勉強会です

  • 本勉強会は平日2時間程度のナイトセミナー方式で実施しています。
  • 参加費は 【無料】 です。
  • 実際に操作しながら勉強しますので、各自Wiresharkインストール済みPCの持ち込みをお願いします。
    PCの持込が出来ない方は、事前にご相談いただければPCの貸し出しを行います。
  • 後援 : 一般社団法人LOCAL インフラ部

第10回のメニューは以下のとおりです

  1. パケットキャプチャ・解析に便利なツール
  2. パケット解析 CTF
  3. ライトニングトーク
    　- 「NAT配下からIPv6でアクセスするWebブラウザ「EzoScape」はTeredo通信だった」 : SIN さん
  • 会場は18:30より入室可能です。
  • 内容は初心者向けです。これまでパケット解析をしたことのない人でも大歓迎です。
  • 実践パケット解析 をベースに実施しますので、本書を持っているとより理解が深まると思います。
    今回は本書第11章「推薦文献」に掲載されているツールや、その他パケット解析に便利なツール類を実際に動かしながら紹介します。(本を持っていない方向けに簡単なスライドは準備します)
  • 後半は、CTF (Capture The Flag) 形式で、パケット解析を題材にしたクイズにチャレンジします。
  • 演習では、オープンソースのLANアナライザ Wireshark を使用します。
    初参加、またはWiresharkを使った事がない方は 過去のHokkaido.capのスライド を事前に参照しておくとよいと思います。
  • わからないところがあれば、みんなで教え合いながら楽しくやりましょう。

LT発表者を募集中です！

  • 5～10分程度で、パケット解析の経験談、ネットワークセキュリティに関する話題、イベントの宣伝や自分で作ってみたプログラムの紹介など、内容はなんでもOKです。「10分じゃ足りない、もっと話したい！」という方もお気軽ご相談ください。
  • LT発表をご希望の方、および不明点・問い合わせは @yamaki_panda または @smokeymonkey までご連絡願います。

多数の皆様のご参加をお待ちしております。
