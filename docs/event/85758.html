Facebookページが数多くある中、メッセージ機能までを有効活用している企業はまだまだ少ないのが実情かと思います。

今回はメンバーズの福島享之氏よりFacebook Messengerのビジネス活用について、Facebook Messengerならではの利点を整理し、マーケティングの全体施策の中で有効活用する為の方法をお伝えします。

アディッシュからは、チャットボット・AIの長所・短所を踏まえつつ、上手に導入するためのコツを、その後の運用体制も見据えてお話しします。

チャットボットは盛り上がりを見せる一方で、実務者として何から始め、その後どうしていけばよいのか？の知見が十分に共有されず、知識ギャップが大きいサービスとなってます。

運用サービスを提供してきた企業ならでは切り口で「利用ユーザー目線」と「企業の運営担当者目線」の両面から、自社なりの一番良いチャットボット活用を実現するためのベースとなる知識をお伝えします。

株式会社メンバーズ SNSコンサルタント　福島享之

2008年より旅行代理店にてWEBサイト制作、ディレクション、広告出稿管理、プロモーションの担当を経て、2014年にメンバーズに入社。
Facebookを中心とした企業のマーケティング支援担当。各企業が掲げる戦略に基づいた、コミュニケーションを整理し、SNSを活用した戦略策定、コンテンツ企画、広告のプランニングなどを実施。主にコンサルティング領域での業務に従事。
2016年9月にAIを活用したFacebook Messenger運用支援サービスを立ち上げ、ファンとのより強いエンゲージメント向上につながる施策を広めている。

アディッシュ株式会社 取締役　池谷昌大

2000年より国内・海外半導体商社、外資系半導体メーカーを経て、2009年に株式会社ガイアックスに入社。
企業向けSNSモニタリング事業、カスタマーサポート事業の営業及びサービス構築を担当。
2014年よりアディッシュ株式会社の取締役。
