勉強会概要

株式会社リクルートコミュニケーションズ（RCO）や株式会社リクルートライフスタイル（RLS）のエンジニアが自由に話す勉強会です。
今回は、リクルートライフスタイルとリクルートコミュニケーションズの共催で機械学習の活用事例や活用を支えるエンジニアリングの話です。

前回の開催内容は、

  • RCO Study Night 「D-Waveが切り開く、量子コンピュータを活用する未来」
    でした。

内容

1. データ分析基盤チームにおけるDevOps

リクルートライフスタイルのサービスを横断したデータ収集・分析・機械学習を行っているデータエンジニアリングチームにおけるDevOpsについてお話します。

具体的には以下のような開発・運用の自動化や効率化についてになります。

  • TerraformやAnsible等を用いたInfrastructure as Code
  • AWS LambdaやCloudWatchEventsを用いたFluentd Aggregatorのデプロイ自動化
  • GoogleStackdriverを使ったシステム監視

発表者：宮井 康宏 (Twitter: @mia_0032)
リクルートライフスタイル　データエンジニア

大阪大学大学院 経済学研究科経営学系専攻および工学研究科ビジネスエンジニアリング専攻を修了後、株式会社ドワンゴを経て現職。

2. GCPを活用した機械学習バッチ運用環境

リクルートライフスタイルのサービスを横断したデータ収集・分析・機械学習を行っているデータエンジニアリングチームにおける機械学習バッチ運用環境についてお話します。

本トークはQiitaにて公開している記事

  • GCPを活用した機械学習バッチ運用環境

に基づいたものとなります。

発表者：高柳 慎一 (Twitter: @_stakaya)
リクルートライフスタイル データ愛好家

北海道大学大学院　理学研究科物理学専攻修士課程修了。材料科学系財団、金融系シンクタンクを経て現職。

3. Sparkを用いた広告配信の最適化

以下のトピックについてお話しします。

  • RCOの広告配信システムに利用されている機械学習システムの紹介
  • 高速なレスポンスを行う広告配信を実現するための仕組み
  • SparkのUnit Test, チューニング

発表者：棚橋 耕太郎
リクルートコミュニケーションズ　エンジニア

4. 業務に役立つプログラミングコンテスト

アルゴリズムを駆使して問題を解く速さを競う「プログラミングコンテスト」について紹介します。プログラミングコンテストの面白さ、RCOでのプログラミングコンテストへの取り組み、業務への活かし方などについてお話します。

発表者：中村 謙弘 (Twitter: @kenkoooo)
リクルートコミュニケーションズ　エンジニア

東京大学 理学部化学科を卒業後、ニート、国立情報学研究所を経て現職。

5. 見た目検索プロトタイピング

画像の見た目で情報を検索できるアプリケーションのプロトタイピングについてお話します。

発表者：堀澤 健太 (Twitter: @horiken4)
リクルートライフスタイル　アプリケーションエンジニア

(LT) ngx_mrubyを用いた柔軟なABテスト基盤

ngx_mrubyを用いて開発している、webサイトのABテスト基盤について簡単にお話しします。
複雑なnginx.confをngx_mrubyで簡略化していく実装や設計を主にお伝えします。

発表者：平塚 迪久 (Twitter: @techeten)
リクルートライフスタイル　データエンジニア
