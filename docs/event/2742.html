Asakusa.rbプレゼンツ!!

jpmobileをruby 1.9対応させていくにあたって、m17nに詳しい成瀬さんに相談してみてはどうかという試みです。

参加してほしい方

  • jpmobileの今後の発展に協力したいと思っている人
  • ケータイの絵文字変換をRuby1.9のm17nフレームワークで実現することに興味のある人
  • Asakusa.rb のmeetupに参加したことのある方
  • Asakusa.rb にいままで参加したことなくて「一度のぞいてみたいなあ」と前から思ってはいたものの、なかなか都合をつけられなかったけれど、今回たまたま1/12(火)の夜に上野御徒町あたりに来ることができる人

当日の内容(予定)

基調講演:「"なりたいアタシ" と "イマのアタシ"」, darashi

jpmobile
founderにしてbuzztterとsoupcurry.infoの中の人。えにしテックのCTOである
darashi
にjpmobileについてトークしてもらったあとは、Rubyのm17nとm17n以外のことに詳しい
nurse
にm17nのことをいろいろ聞いたり、Rails3やRails2.3.xへの対応などについて議論してもらえるといいなと思っています。

Asakusa.rbのこれまでの実績だと、だいたい22:00頃に終わるんじゃないかと思います。途中退出は自由です。
