越後湯沢ソーシャルメディア交流会

イベント ユーザー

title 越後湯沢ソーシャルメディア交流会
catch 小寺信良さん・津田大介さんを迎えて、「情報セキュリティワークショップ in 越後湯沢」プレイベント
description

10/7からの情報セキュリティワークショップ in 越後湯沢のために、インターネットユーザ協会（MIAU）の小寺信良さんと津田大介さんが湯沢町においでになります。今回は、地元湯沢町の皆さん、さらには新潟県内のソーシャルメディアユーザの皆さん、ワークショップのために前泊する情報セキュリティ業界の皆さん、一堂に会したプレイベントを開催したいと思います。

主催：越後湯沢ソーシャル倶楽部
共催：新潟ソーシャルメディアクラブ、「情報セキュリティワークショップ in 越後湯沢」大会実行委員会
同時開催：情報セキュリティ・ワークショップ in 越後湯沢（10/7-8)

ゲスト：
小寺信良（AV機器テクニカルライター/コラムニスト@Nob_kodera)

津田大介（ジャーナリスト @tsuda）

日時：10/6 19:00-
場所：保よし（新潟県南魚沼郡湯沢町湯沢328-1）
会費：4500円（学生4000円、社会人学生を除く）:食事つき、飲み放題

プログラム（予定）

ゲストからソーシャルメディア最新事情についてスピーチ

Lightning Talk（５分）
湯沢など地元での活動を報告するLTを募集します。
ほか、インターネットユーザ協会（MIAU）、新潟ソーシャルメディアクラブ（NSMC）情報セキュリティワークショップ in 越後湯沢、の紹介なども予定しています。

問い合わせ：、田村 恵一（@soba_rock、湯沢関連）、一戸信哉（@shinyai、ワークショップ、NSMC関連）
