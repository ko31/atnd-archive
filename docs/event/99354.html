◆概要

ネットワークの業務に従事している、若しくは勉強中の方向けの勉強会です。
参考レベル：CCNA R&Sの知識を有している

◆勉強会の内容

PC上の仮想環境（シミュレータ）上に、仮想のネットワーク環境を構築して頂きます。(問題形式)
その後、問題の解説と質疑応答時間を設けます。

【注意事項】

・参加される方は、Windows/MacOS搭載のノートPCを当日持参下さい。


事前に、主催から仮想環境のインストール手順等をご案内致します。


当日持参するPCに予めインストール頂くようお願い致します。


・連絡用のメールアドレスをATNDに登録お願い致します。


・当日の会場は、無線LANによるインターネット（Wi-Fi）が使用可能です。

◆タイムスケジュール

時間 内容
10:30～10:45 概要説明
10:45～11:15 問題演習(Switching)

11:45～12:00 休憩
12:00～12:30 問題演習(Routing)
12:30～13:00 解説(Routing)・質疑応答
13:00～13:10 片付け・撤収

※撤収後、希望者のみでランチを予定しています！
