description

Zope/Ploneの開発者で集まってもくもくと開発したり色々相談したりとかやろうと思います。

スプリントのゆるい版みたいな感じで開催しています。6月の第0回から毎月開催しています。参加費は無料です。

もってくるもの


  • PC

  • マイカップ(できれば)

会場には無線/有線LANがあります。

Zope/Ploneの開発に興味の有る方は、是非ともこの機会に一緒に勉強しましょう。

当日のタイムテーブルはだいたいこんな感じのつもりです。


  • 11:00-11:30 自己紹介とか

  • 11:30-12:00 やること決めたり準備とか

  • 12:00-13:00 昼食

  • 13:00-18:00 もくもくと開発

  • 18:00-19:00 成果発表(できたとこまで)

  • 19:00-19:30 後片付け

  • 19:30- 打ち上げ(いける人で)

ついったーのハッシュタグはちょっと長いけど #zopeplonedev です。
