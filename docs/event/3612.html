何をするの?

Immutable Object(http://www.amazon.co.jp/dp/6130631111/)を人が集まって読もうという会です。

Immutable Object　この本には何が書かれているか分かりませんが、きっと楽しい本に違いないのです！

他にも、ImmutableなObjectについての四方山話をすると良いのではないでしょうか?
ImmutableなObjectなら計算機とか関係無しで今回の話の対象に成る気がします！
ので、Immutable Objectな話のネタやImmutable Objectなモノを持って来て貰えると面白いはずです。
