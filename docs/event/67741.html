伊佐市に住み込み、企業だけでなく地域の魅力にも触れる

今年で創業45年を迎える大口酒造。

長年に渡り、地元鹿児島で愛されてきた焼酎「黒伊佐錦」。

昨年は技術と品質の向上と新製品の開発を図る鹿児島県本格焼酎鑑評会にて「総裁賞代表」を受賞するなど、鹿児島の焼酎業界の中でも注目されている大口酒造。

そんな大口酒造から「営業職」と「生産工程オペレーター」、「研究職」を募集します。

先輩インターン生とのご飯会とは？

インターンシップのこと、会社のこと、仕事のこと、そして大学生活のこと。
手づくりのご飯を一緒に食べながら、一足先にインターンを始めている先輩とワイワイ語り合う場です。

インターンシップフェアでは聞けなかった、あんなことやこんなことを
コーディネーターと一緒にズバッと質問してみましょう！

もちろん、あなた自身のことも存分に語ってくださいね。

経営理念
一、大口酒造は、最高品質の本格焼酎づくりめざします
一、大口酒造は、和を大切にし、人を育てる企業をめざします
一、大口酒造は、地域に貢献し、地域に愛される企業をめざします

豊かな自然と水に育まれて最高品質の焼酎を造り、その商品を安定価格で市場提供すること。
また、和をモットーとし、「企業は人なり」を念頭に人材の育成を行い、社会貢献できる人、企業をめざします。

地域に愛され、育まれている企業であることを理念とし、人、企業ともに地域に貢献できる企業を目指しています。
