[S43-01] ハンズオン: 「PKIハンズオン」

桂木様

Mac OS X / Mac OS X Serverで使用される公開鍵基盤の基礎を理解する為のハンズオンを行います。暗号化、復号、公開鍵暗号、証明書、ハッシュなど、基本的な技術をハンズオンで身近に体感できます。また、時間があれば、キーチェーンアクセスやメールへの証明書の添付、メールの暗号化についても簡単に触れます。MacBookなどを各自で持参の上での参加をお願いします。
