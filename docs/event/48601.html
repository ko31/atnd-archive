主催者側の都合で延期になりました。申し訳ございません。

明治大学・専修大学の学生主催による、競技プログラミングの練習会です。
ACM-ICPC国内予選突破を目標に、定期的な模擬コンテストの開催を企画しています。
初回は、「ACM-ICPC国内予選の出題形式に慣れる」ことを目的として、AOJ Virtual Arena によるICPC過去問題を用いた模擬コンテストを開催します。

1. 内容

  • ACM-ICPC国内予選の模擬コンテストを開催します。
  • 出題問題難易度は、Ａ問題〜Ｃ問題相当を中心とします。
  • 今回は、ACM-ICPC国内予選過去問題セットを使用します。(AOJ Virtual Arena による開催)
  • 当日の流れは、次のような内容を予定しています。(案)
    1. 自己紹介
    2. ACM-ICPCの紹介
    3. AOJ Virtual Arena の使用方法解説
    4. 模擬コンテスト
    5. 結果発表・解説
    6. (希望者のみ)懇親会 (参加費1人500円程度)
      • 会場では、ソフトドリンクとお菓子を準備します。
      • 併せて、情報科学全般に関するLT大会を行おうと思います。発表希望者は@kazh98 まで連絡ください。

2. 持ち物

  • 無線LAN接続可能なノートPC
  • 筆記用具
  • その他、練習会参加において必要なもの。

3. 備考

詳細については、ウェブサイト
　http://www.mo.cs.meiji.ac.jp/kaz/procon
をご確認下さい。
