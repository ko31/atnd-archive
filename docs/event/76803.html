★イベント内容

転職をお考えの方、どうやって企業様を選んでいますか？
どうしても見ている業種が重なっていたり、「あの会社は無理だろうな～」などお悩みではありませんか？
そんな中、今回下記4社の方にお越し頂き、一気に企業説明（＆同時に一次選考も！）を行って頂きます！
今までに考えたこともないような企業様と出会えるチャンスです！
是非お気軽にお越しくださいませ！！

※内容は変更になる可能性がございます。

★ご来場企業プロフィール（※五十音順）

・株式会社AZITO様

http://www.azito.co.jp/
「城姫クエスト」を代表とした、新しいゲームを続々と世の中に発信しているAZITO様。
3月にオフィスも移転し、日々進化を続けている企業様です。

・兼松グランクス株式会社様

http://www.granks.co.jp/
「Clipbox」をはじめとする、世の中の人が「あったらいいな」を続々と生み出しています。
企業内の雰囲気もすごく明るく、エンジニアのみなさんが働きやすい環境が自慢です。

・株式会社システナ様

http://www.systena.co.jp/
幅広い領域の案件に携わっているので、最新の技術や業界ノウハウを身に付けたい人にはすごくおすすめです。
さらに、ほぼエンドユーザーがクライアントなので、やりがいを持って仕事をされたい方にはぴったりな企業様です。

・株式会社プログデンス様

http://www.progdence.co.jp/
日本で20社程度しかない、『Cisco Gold Partner』としての認定資格している企業です。
インフラの領域でスペシャリストを目指す方は、是非ご検討してみてください。
