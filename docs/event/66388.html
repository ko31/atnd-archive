■ネットショップ活用セミナー vol.2 2015年5月23日（土）

「ネットショップ構築ツール BASE の基本的活用法」と「ネットショップ構築ツール BASE のWebマーケティングの基本」

■ネットショップ活用セミナー vol.2

開催日時：2015年5月23日（土）　13:00～17:00 場所：滋賀 南草津駅前 フェリエ１F 「Plan for You」 主催：ミズホネット(MizuhoNet) 持参するもの：WiFiにつながるノートパソコン（必須） 第1部・第2部、別々でも両方でも受講できます。 申込みはこちら： https://netshopcafe.doorkeeper.jp/events/24368 ※お申込みの際、第1部、第2部、両方を明記ください。

場所

フェリエ南草津 住所: 滋賀 南草津駅前 フェリエ１F 「Plan for You」

セミナー概要

【第1部】13:00～14:50（110分）講習料：3000円

  • 講師 *山本一男
  • 概要
    • 「ネットショップ構築ツール BASE の基本的活用法」
    • ●BASE（ベイス)
    • -ネットショップを無料で簡単に作成できるネットショップ構築プラットホーム-
    • ネットショップ作成が30秒で。しかも完全無料。ネットショップをつくるのに、もう「勉強」と「お金」は必要ありません。今までECサイトを作りたくても時間・お金・技術さまざまな理由で始められなかった、そんな全ての方の悩みを解消するサービスです。

【第2部】15:00～16:50（110分）講習料：3000円

  • 出演
    • 山本一男
  • 概要
    • 「ネットショップ構築ツール BASE のWebマーケティングの基本」
    • BASE（ベイス）で構築したネットショップをへの集客方法やSEO対策、SNS活用法など、ネットショップの運用法を学びます。

■ネットショップカフェ - NetShop Cafe-

  • オンライン上で構築できるネットショップ構築ツールが何種類か登場してきました。その内のWix（ウィックス）、Jimdo（ジンドゥー）、BASE（ベース）などを学べる場が「ネットショップ構築カフェ NetShop Cafe」です。 定期的（月に1回）に、ネットショップ構築ツールの1つ1つを具体的に紹介し、分かりやすく基本的な操作方法を学びます
  • 開催日時：毎月第4土曜日（原則） 13:00～16:00 (3時間)
  • 場所：滋賀 南草津駅前 フェリエ１F 「Plan for You」
  • 主催：ミズホネット(MizuhoNet)
  • 開催予定：4/25 5/23 6/27 7/25 8/29 9/26 10/24 11/28 12/26
