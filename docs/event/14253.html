予定として、土曜日は色んなアナログゲームを、
日曜日がカタンだけをやるという方向で動いてますねー
カタンも含め、ボードゲーム持ち込み大歓迎です。

以下、補足になります。

今回は、沖縄のボードゲームサークルの連絡会的な
「沖縄 Analog Gamer’s Union」主催の初イベントで、
現在のカタン発売元である「株式会社ジーピー」様の
CPCとしてのイベント活動の一環という位置づけになります。
なので、これをきっかけにカタン専用大会など開いていく予定です。

Game-shelterガネコさん主催以外にも色々あるので、
良ければ顔出してみてくださいな。
