description

クラウドソーシングサービスの成長に伴い、発注側の企業とそれを受注するコワーカーは「プロジェクト型」の働き方へシフトしており、ビジネスパーソンの働き方が変わろうとしています。

その流れをうけ、パソナテックのクラウドソーシングサービス「Job-Hub」は、ChatWork株式会社が提供するクラウド型ビジネスチャットツール「チャットワーク」を無償で利用できるサービスをスタートしました。

本セミナーでは、Job-Hubサービス内で、「チャットワーク」のグループチャット、タスク管理、ファイル管理などの機能を活用することで実現する『新しい働き方』について、ご紹介します。
