description

要望があったので立てました。
北九州にのりおなたとろが来襲、五反田さんも旅の帰りに寄るとのことで迎撃体制へ移行、集合後速やかに酒屑に移りたいと思います。

会場は焼肉工房 牛の恵。
宮崎産Ａ５黒毛和牛食べ放題のお店です。
コースの設定や飲み放題のON/OFFが当日に設定できるので、全員のお財布の状況から通常コース\3980～希少部位食べ放題コース、最上級食べ比べコース（最大\5500）の切り替えを決めます。
飲み放題\1500はのりおさんの要望でつける予定です。

二次会は串カツ＆お寿司、海鮮がある屑処へご案内の予定、焼き鳥等要望があればその都度案内するってことで。
行き当たりばったりで申し訳ないのですが、初となるお顔が２名いるので全員で当日に相談後移動でよろしいかと。

集合場所は11/2、18:00に小倉駅2F改札口で（改札が1Fと2Fしかないので間違うことがない）。
当日は九州の幸が目白押しなので、お腹空かせて体調管理はしっかりとｵﾅｼｬｽ！
