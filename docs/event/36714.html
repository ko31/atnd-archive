description

春も近づいてますがまだ寒い季節…中華で日本酒を楽しみましょう。
ＮＨＫ（日本昼酒協会）活動ということでランチタイムに開催します。
第一回は好評につき満席です。

主催のＮＡＯがお手伝いしている
高井戸のチャイニーズバールにて。
独創性のある本格中華をバルスタイルで楽しめます。

■場所　CINNABAR 高井戸
　　　　　http://www.cinnabar.jp/

■日時　3月10日（日）
12：15集合
12：30開始

■会費　3500円（コース料理、日本酒持ち込み料金こみ）

■募集　12名　（＋私含め13人。シェフも少し試飲させていただき参加予定）

■日本酒　中華にあわせてみたい日本酒を各自1本。（1本以上でもかまいませんｗ）
　　　　　　12名でシェアして飲むので出来れば4合瓶。
　　　　　　古酒や貴醸酒など小瓶しかない場合は事前にご相談ください。
　　　　　　持ちよるお酒はイベント前日の夕方までにコメント欄に宣言。
　　　　　　早い者勝ち。かぶるのはNGです。

■持ち物　お猪口
　　　　　　※現在、日本酒を常には提供していないため数がありません。
　　　　　　　こちらでも用意しますが可能でしたら御持参ください。

■当日、料理のメニューをくばります。
　そこで、みんなの日本酒を発表したうえで順番を決めて飲みたいと思います。
燗酒も出来るようにします。

よろしくお願いします！
