WCAN プレゼン ワークショップ 2011（土曜版）

イベント ユーザー
event_id 20389
title WCAN プレゼン ワークショップ 2011（土曜版）
catch Web業界で誰よりもプレゼンを見ている鷹野さんを先生にワークショップです
description

久しぶりに WCAN ワークショップイベントを開催致します。過去、写真やアクセス解析、ディレクションといった内容で行ってきましたが、今回はプレゼンです。Web業界で誰よりもプレゼンを見ている、 CSS Nite の鷹野さんをスピーカーに5時間ものワークショップを企画しました。

仕事で客先でプレゼンをしないといけない人や、社内でのプレゼン、セミナーや勉強会（WCANのライトニングトークも歓迎）でプレゼンするかもしれない人、このような機会はあまりありませんので、是非ご参加下さい。

なお、鷹野さんが利用するアプリケーションは Apple の Keynote です。 PowerPoint の方も参加できない事はありませんが、アプリケーションの話は Keynote ベースになりますので、ご了承下さい。

定員

30人（10人以下の場合には中止になる可能性もあります）

参加費

5,000円（領収書も発行します）
  • 当日集金致します。
  • 学生さんも有料になります。

■宿題

  • テーマ「私がオススメするもの」時間は3分以内（自己紹介は省略）
  • 事前に作成してくること。

■内容

まだ、案の段階ですが、どのような事をするのか分かりやすいように紹介しておきます。

＜ロジック＞

  • プレゼンについて（スライドを作ることではない）：20分
  • ワークショップ：ロジックの組み立て方：30分

＜スライド＞

  • スライド制作について（型、スライド、アニメーション/トランジション）：40分
  • スライド添削（ライブ）：30分

＜デリバリー＞

  • デリバリー（トーク）について：20分
  • ワークショップ：グループでのロールプレイイング、全員の前での発表＋講評：90-120分

■持ち物

  • ワークショップではノートパソコンが必要になります。

■講師紹介

鷹野 雅弘 ( Masahiro Takano / @swwwitch )

  • 株式会社スイッチ
  • CSS Nite
  • blog

Webサイトの構築やコンサルティングを中心に、WebやDTPに関しての講演やトレーニングのほか、スクールなどのカリキュラム開発も手がける。テクニカルライターとして10冊以上の著書を持つほか、書籍の企画や編集なども行っている。

2005 年からCSS Niteを主宰。主な著書に『 効率的なサイト制作のためのDreamweaverの教科書 』（毎日コミュニケーションズ）、『 できるクリエーターFlash独習ナビ 』（インプレス）、『 Illustrator CS4 完全制覇パーフェクト 』（翔泳社）。
