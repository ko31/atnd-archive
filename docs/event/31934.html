概要

Jenkinsを実践で使うための勘所を押さえていくことを目的とした勉強会です。
初心者と初中級者のグループを分けて勉強会を進めます。
詳しくは下記を参照してください。

対象者

  • Jenkinsや継続的インテグレーションを知らない方・興味のある方
  • Jenkinsは知っているが、実プロジェクトでどのように使えばいいかわからない方
  • 技術の話をできる仲間・環境を渇望している方

基本的には特定の講師や発表者がいるような勉強会ではありません。皆が参加者であり発表者です。

LT大会

持ち時間5分のLT大会が開催されます。発表者は随時募集中です。
内容は変更になる可能性があります。

初心者向け勉強会の内容

最終的には初中級者と合流して、Jenkinsの運用・チーム開発を学ぶことを目標としています。
Vol.3 では、1名が初心者コースを卒業し、初中級者コースに合流しました。

第5回勉強会について

書籍「Jenkins実践入門」 や、ブログ等を参考に、まずは自身でJenkinsに触れてみてください。
勉強会当日までの自習の中で、困ったこと・質問を都度解決していくといった雰囲気になります。
準備にさほど時間は要さないと思いますので、途中からの参加でも全く問題ないと思います。

現在、PHPをメインとしている方が半数くらいですので、PHPが話題の中心となっておりますが、JavaやRubyがメインの方もいらっしゃいます。
参考サイト： PHPでJenkins

初中級者向け勉強会の内容

開発の現場を想定し、実際にプロジェクトを運用していきながら、CIのあれこれを模索します。
SwimmyStudy – GitHub にあるサンプルアプリをもとに、git-flowでブランチ管理をし、アプリケーションを改修していきましょう！
後日、説明ページを作成し、公開する予定です。

参考になるサイト

Jenkins + PHP環境のセットアップしたい → PHPでTDD＆CIワークショップ、Jenkins + PHP の各種プラグインパート資料
Jenkins + CakePHP環境をセットアップしたい → CakePHP2.0+Jenkinsで継続的インテグレーションを行う方法
Gitについて知りたい → Git Pro
GitHubについて詳しく知りたい → WEB+DB PRESS Vol.69
git-flowについて知りたい → ゼロからわかるgit-flow
