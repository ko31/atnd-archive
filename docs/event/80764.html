作家川端康成や俳人久保田万太郎の定宿だった台東区浅草のホテルニュー魚眠荘で毎月開いている「そば会」が、今回で298回目を迎えました。最初は会津そばを広めるのが目的でしたが東日本大震災の後は復興支援も含めて続けています。
全国でも有数のそば粉の産地、南会津産のそば粉を使って打つ会津そばは、透明感があり、腰が強く、口に含むとほんのりとした甘みが広がります。そば会では、数量限定ですが会津の地酒も出され会費五千円。そば好きだけでなく、会津出身者など登録会員だけで四百人を超えます。会津が好きな方または応援して下さる方ならどなたさまでもお申込みいただけます。是非お誘い合わせの上、お越し下さいませ。
