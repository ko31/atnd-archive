12/24、12/28にディスカッション（集団討論）練習会を行います。
入学試験に個人面接のほかディスカッション（集団討論）が課される看護医療系学校があります。決められたお題で自由に意見を出し合うものですが、実際は思うように発言が出来なかったり、沈黙してしまうこともよくあります。

アルファゼミナールでは、以下の日程で看護学校入試向けディスカッションの練習会を行います。こちらで『看護師の資質について自分が思うこと』などのお題を指定しますので、それにしたがって皆さんに意見を出し合っていただきます。入学試験にディスカッションのある方はふるってご参加ください。

【対象】：京都中央看護保健大学校や大津市民病院附属看護専門学校など、入試に集団討論が課される学校を受験される方
【参加費】：無料
【日時】：
①12/24（木）15:00～二時間程度
②12/28（月）19:00～二時間程度
①と②のご都合のよい方にご参加下さい。両方へのご参加も可能です。
