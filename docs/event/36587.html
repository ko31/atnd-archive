概要

こちらは，みけくん（@mikenov22）が主催する焼肉パーティになります．

みけくんがひたすら参加者のために焼肉を焼き続けるだけのパーティです(自分も食べます)．

目の前にあるお肉をどんどん食べて頂きます

ただ，至らない点が多いため，学生は安く…みたいな金額設定をするだけの余裕はございませんので，年齢に関係なく参加費（食事代）は同じになります．

私にできるのは会合の機会と場所を用意することだけになります．あらかじめご了承ください．

企画情報

詳細

  • 3/2 18:00~20:00
  • 牛角（秋葉原電気街口店）
  • 集合時間：18：00
  • 集合場所：JR秋葉原駅 昭和通り口改札前
  • 食べ放題だと最低3000円
  • 個別に3000円/90分のメニューで食べられるかは分からんです．
  • 飲み放題をつけると，400円ぐらい高くなります（ノンアルコール）
  • アルコールも飲み放題にすると+1500円になります．お金に余裕のある人は是非．

準備するもの

  • お肉を沢山食べるために必要な空腹感…Priceless
  • 皆さんの美味しそうに食べてる時の笑顔…Priceless
  • せっせと焼いてるみけくんに，食べる分のお肉を分けてあげられる優しさ…Priceless
  • 最低限お肉の代金と交通費が払えるだけのお金が入った財布…Priceless…?

2次会情報
