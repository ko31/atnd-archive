Windows Phone Arch 東京アイデアソン#1

Windows Phone ArchはWindows Phoneのコミュニティです。
http://wp-arch.net/

2012/04/28(土)に開催する Windows Phone Hackathon in Tokyo#1 の関連イベントとしてアイデアソンを開催します。

今回のアイデアソンは次のような参加者を募集しております。
関連イベントですがハッカソンへ参加しなくても参加可能です。

  • ハッカソンの環境を事前に整えたいけれどどうやれば不安の方
  • アイデアはあるけれど実現方法を相談したい方
  • ハッカソン当日は予定が合わないけれど日程が合う方
  • ハッカソンに参加したかったけれど参加表明が間に合わなかった方
  • ハッカソン当日に一緒に開発してくれる人を探したい方

費用

アイデアソンについてのご注意

今回はアイデアソンですのでPCは必須ではありませんが、PC持参の場合は次のような環境を推奨します。

  • Windows VistaまたはWindows 7が稼動している事（Win7推奨)
  • ペンタブレットやフルキーボード、みなさんがアプリ開発に必要なものをお持ちください。
  • 飲み物、食べ物の持ち込みOKです。手の汚れないお菓子の持ち込みも大歓迎です。
    • ネコがいますので飲み物はペットボトルなど口のしまるタイプをお勧めします。
  • 電源および無線LANは会場にご用意しております。できれば電源タップをお持ちいただけると助かります。
  • Windows Phone開発にはWindows Phone SDK 7.1が必要です。
    • Windows Phone SDK 7.1は下記URLよりダウンロードできる無償開発環境です
    • Windows Phone SDK 7.1
