WEB☆超初心者勉強会

イベント ユーザー
event_id 20128
title WEB☆超初心者勉強会
catch 超初心者向け。イマドキのWEB制作について勉強しましょう。
description

WEB制作者目線による基本的な技術の話から、
コーディングの話を中心にすすめます。
課題として、主流となりつつあるHTML5&CSSでのコーディングを実践してみましょう。

費用

1,800円

下準備

ノートパソコン（なくてもOK）電源あります。
事前に使用するフリーソフトをインストールしてください。
インストール方法がわからない方は、twitterで @cheepint へご連絡ください。
・テキストエディタ：terapad(windows), mi(mac)
・ブラウザ：Firefox ※firebugアドオンを追加してください。
※アドオンの追加方法
1.Firefoxを開き、上部メニューバーから、ツール＞アドオンを選択
2.右上にある「アドオンを検索」にfirebugと入力、検索。
3.黄色い虫マークの「firebug1.8.1」というアドオンがありますので、インストールボタンをクリック
4.firefoxを再起動して、ブラウザの右上または右下に虫のマークが表示されていればOK!

こんな人にオススメ！

・WEB系の仕事に興味がある。経験はこれから積みたい
・アプリ頼みでなく、ちゃんとコーディングを勉強したい
・共通の興味を持つ人たちと情報交換したい

勉強会メニュー

・WEBの基本の「キ」
　技術は日進月歩。どうして今HTML5とCSS3が話題なの？
　「いまさら聞けない（泣）」なんて言わないで、しっかり解消しましょう。
・WEB業界での役割分担、最近のトレンド
　一口にWEBの仕事と言っても色々あるよ、というお話。
　コーダー、マークアップエンジニアの役割とは。
・やってみよう
　HTML5＆CSS3でコーディングをしてみましょう。
・終わったら、せっかくなので近くのカフェで交流会など。
