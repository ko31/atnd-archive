内容

エイチーム大阪オフィスにて、エイチームスタッフとの座談会を開催いたします。
会社の雰囲気を見てみたい、エイチームに興味があるなど、気軽にご参加いただける座談会です。
エイチームで活躍しているゲームプログラマより、会社の雰囲気、開発環境などお話させていただきます。

どんな会社か知りたい、ちょっと興味があるという程度でもＯＫです！
事前予約不要！持ち物不要！途中参加OK！の座談会です。
お仕事帰りなど、お気軽にお越しください。

対象となる方

エイチームに興味がある方
現在転職を考えている方
将来的にもっと自分にあった活躍できる場を探したい方
ゲームプログラマ経験のある方（ゲーム業界未経験可）
選考を受けるかどうか迷っている方

※ゲームプログラマ、エンジニアの方を対象とした会です
※中途向けイベントのため、学生の方のご参加はお断りさせていただいております

大阪オフィス拡張移転について

2012年9月に梅田阪急ビル オフィスタワー26Fに開所した大阪オフィスは、好調に業務拡大・人員増加を続け、2017年5月に、同オフィスタワー内の35Fへ拡張移転いたしました。新オフィスは以前と比べ約2.4倍のフロア面積となり、和室オープンミーティングスペースをはじめ、ワイガヤエリアを多く設けることで、社員同士のコミュニケーション活性を促進し、クリエイティブな発想が広がるような工夫を凝らしています。

エイチームについて

エイチームは、名古屋に本社をおく、総合ＩＴベンチャー企業です。
スマートデバイス向けゲーム・ツールアプリを提供するエンターテインメント事業、人生のイベントや日常生活に密着した様々なウェブサービスを提供するライフスタイルサポート事業、利便性の実現をテーマに大型商材のEC化に参入し、自転車通販サイトを運営するEC事業の3つの軸で事業を展開しています。

なかでもエンターテインメント事業で手がけるゲームタイトルは実に様々です。
