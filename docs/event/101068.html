クリーク･アンド･リバー社（C&R社）は10月26日（金）、3Dアーティスト、UIデザイナー、アニメーターなど主にグラフィックに携わる方を対象に、ゲームをより豊かにするエフェクトや演出の基本についての勉強会「デザイン塾」を開催します。

勉強会では、エフェクトの基本概念、演出方法について解説した後、スマホゲーム開発のツールとして最もポピュラーとなったゲームエンジン「Unity」の中にある標準パーティクル制御システム「Shuriken」を使ったエフェクト制作の実演・解説を行います。
また講義終了後には懇親会も開催予定です。お気軽にご参加ください。

Shurikenとは

ゲームエンジンUnity内にあるパーティクル制御システム「Shuriken Particle」のことhttps://docs.unity3d.com/jp/460/Manual/class-ParticleSystem.html

■講義内容■

・エフェクトの基本概念
・演出について
・Shurikenでできること
・Shurikenの使い方
・Shurikenでより良くに見せる方法
実際にShurikenを使いながら説明します。

■登壇者■

雨宮 悠介氏
【経歴】
VFXデザイナーを志し、コナミスクールとデジタルハリウッドに入学。
卒業後は株式会社ビースタックや、株式会社ポリゴン・ピクチュアズでアニメ、映像制作に従事。
その後ゲームエフェクトに強く興味を持ち、株式会社コーエーテクモゲームスに入社し、複数のゲーム制作に参加。
株式会社コロプラではエフェクトディレクション・セクション運営コアメンバーとして、エフェクトチームを立ち上げ複数プロジェクトを同時に担当。
現在ではエフェクト制作のプロフェッショナルとして活躍している。

PS VITA：『討鬼伝』
Wii U：『ZELDA 無双』
PC：『信長の野望 ONLINE』
スマートフォン：『バトルガールハイスクール』
アニメ：『TRON-Uprising』

井出 千尋氏
【経歴】
アニメ業界でのコンポジッターを志し、ユーフォーテーブル有限会社に入社。
入社後に参加した案件にて、コンポジット、背景美術、特殊効果、絵コンテ等幅広い業務に対応。
さらなる可能性を求め、遊技機業界でのコンポジット、ゲームエフェクト制作に従事。
現在では、あらゆる業界で演出の提案から最終的なエフェクト制作まで対応可能。
