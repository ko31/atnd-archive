日本最大のプロのイラストレーター団体であるJIA(日本イラストレーター協会)の本格的なイラストレーターを志す方へのセミナーです。　5/13（土）より受付開始いたしました！

今回の講師はJIA会員でもあり、まさに大ブレイクして活躍されている
「花もようアート」のイラストレーター滝川真由美さんをお招きします！！

人気テレビ番組「イチゲンさん」や「行列のできる法律相談事務所」で紹介＆出演され、昨年はイギリスのアートイベントにも出展された滝川さん、ガラスペンで緻密に描く「花もようアート」について色々とお話をうかがいます。

滝川さんがテレビ出演された瞬間、ガラスペンの問い合わせが殺到したとの事です。
先月は銀座の伊東屋さんでもデモンストレーション＆原画販売会を行った滝川さん、イラストレーターとしてブレイクするまでの大変な道のり、作画技法などを講演していただきます。

・人気テレビ番組で紹介された滝川さん！その出演の時のエピソードなども楽しく聞かせていただきます。
・イギリスのアートイベント出展し、海外のアート事情を肌で感じられた滝川さん、そのエピソードも大変興味深い内容です。
・今回は実際に会場に原画を持ってきていただき、ガラスペンを使った超絶技法を目の前で見せていただきます！！

手描き系イラストレーターにはこの上なく勉強になること間違いなし！
