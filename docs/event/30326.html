【Tizen Web App について調べたよ】

  • @TNarutoさん
  • 概要
    Tizen IDE の使い方や、実機へのサンプルプログラムの転送方法などを発表しますよ！

【ARM Linux の最新】

  • @Aliratsukamoto さん
  • 概要
    ・Linaroで取り組んでいる次世代ARMのbig.LITTLE対応カーネル開発の紹介
    ・LinaroでのTizenの情報など

【Qt のコミュニティと日本語での情報発信】

  • @luyikei さん（今年の LinuxCon の学生セッションの優勝者）
  • 概要

【Qt 5 の master で遊ぼう】

  • @task_jp さん
  • 概要

    QtQuick2/QtQuick3D/QtMultimedia/QtWayland/などなど
