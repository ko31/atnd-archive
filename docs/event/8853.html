ネットワークの 超初心者向け勉強会です。

ネットワーク をはじめて勉強する方に向けた内容になります。
CiscoのCCENTベースで勉強していきます。

難易度は 『 全く難しくない 』 です。
対象者は 『 これからネットワークを勉強したいという人 』 です。

勉強会の進め方は、プロジェクターに映した資料を見ながら、勉強会を行います。

[全体のスケジュール]
　1.ネットワークの基礎とOSI参照モデル
　2.TCP/IPプロトコル
　3.イーサネットLAN
　4.ネットワークセキュリティの基礎
　5.Cisco IOSソフトウェアの操作
　6.IPv4アドレスとサブネット化
　7.Catalystスイッチの起動と基本設定
　8.Ciscoルータの起動と基本設定
　9.Ciscoデバイスの管理
　10.ルーティングの基礎
　11.ディスタンスベクタールーティング
　12.WAN接続
　13.Cisco SDM
　14.無線LANの基礎

上記の内容を複数回に分けて行います。
１回に１～２章のペースを予定しております。

　2. 参加費が2000円かかります。(場所代等)
