description

Wizcorp incのハンサリ・ギオーム氏を講師に迎え、Node.jsとCouchbaseを使ったゲームアプリ開発の実践事例を中心に開発スキームでの想定される壁やその突破方法、実際に発生した問題の解説など、アプリ開発に関わる開発者や管理者にとっては必聴のセミナーとなっています。

こんなことにお困りでは？
・キャンペーンを実施するとゲームのDBがトラフィックに耐えられない
・ユーザが多くなるとデータのシャーディングをせざるを得ない
・3G回線を利用するため、画像のクオリティを落とさなければいけない
・オフラインプレーを実現したいのになかなかできない
・エンジニアの育成が大変

また以下でお困りのプロデューサー、ディレクターの方もご参加下さい！
