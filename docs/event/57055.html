スマートフォンアプリのUI,UXデザインのノウハウを、
1日で効率的に習得できる若手デザイナー向けワークショップです。
※参加費、懇親会ともに無料です。お気軽にご参加ください。

▼学べることは以下になります。
・ターゲットの設定、ペルソナ設定、ユーザーシナリオの考え方
・スマホならではの、プロトタイピング作成ノウハウ
・作成したプロトタイプをデザイン評価する手法
・時間が限られた中での、効率的なブレスト手法
・ビジネス上のゴール設定ノウハウ

講師は、DeNAで様々なプロダクトを制作しているUI/UXデザイナーが務めます。
ぜひ、エントリーください！

※応募者多数の場合は、抽選とさせていただきます。

【日時】
10月19日（日）13:00〜18:00

【優先対象となる方】
・ウェブデザイナー、UIデザイナーの20歳代の方で、スマホのデザインを学びたい方
　→デザイナーとしての実務経験が1年以上ある方
　→iPhone, Androidスマートフォンをお持ちの方

【タイムスケジュール】
・13:00〜14:00：座学
　→テーマの説明
　→ペルソナ設計の手法
　→ユーザーシナリオ設計の手法
　→プロトタイピングとその作り方
　→アクティングアウトとは

・14:00〜14:30：グループに分かれてのブレスト

・14:30〜16:30：プロトタイプの作成
　→サービスのワイヤーフレームの作成
　→プロトタイピングツールの説明
　→ペーパープロトの作成

・16:30〜18:00：アクティングアウト
　→制作物の発表と講評会
　→全員での振り返り

・18:00〜19:00：懇親会
