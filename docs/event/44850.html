html5j 英語部とは

  • 英語の情報を日本に届きやすい形に
  • 日本語の情報を世界に届ける

この2点にフォーカスし、 グローバルな舞台で活躍するWeb技術者を増やす ことを目標に、オープンなコラボレーション活動を行っていきます。

今回はhtml5 英語部の方向性共有、webplatform.orgの翻訳など

  • html5 英語部の設立意図と目的、方向性について (部長・斉藤祐也によるプレゼンテーション)
  • グループワークによる翻訳: WebPlatform.org の Beginersセクション から翻訳していく予定です。

翻訳してみたいけど難しそう…というのは気にせず気軽に参加を！

html5 英語部の翻訳ワークフローでは、翻訳者以外に監訳者がいます。翻訳に自信が無くても、ドキュメント翻訳の経験者たちの監訳・アドバイスを受けられるので大丈夫です。みんなで成長していきましょう。

ハンズオンイベントだけでなく、オンライン上での翻訳活動もおこないます。

html5 英語部はGithub上にグループを作成し、 オンライン 上で翻訳の進行をおこないます。
残念ながら今回のハンズオンイベントに参加できなかったという方も、こちらのGithub上での翻訳活動に参加することができます。

参加にあたっては、下記のページをご覧ください。
