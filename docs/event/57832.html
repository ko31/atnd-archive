大阪（梅田）のIT企業であるシナジーマーケティングとフェンリルの仕事現場をもっと知ってもらうためのエンジニア向けイベントです。それぞれの企業で活躍するエンジニア4名が、ふだんどういう仕事をやってんのかを熱く語ります。

私服OK、途中参加・退室もOK。参加無料です。
お気軽にお越しください！

登壇者紹介（あいうえお順）

■ 伊藤 伸裕（いとう のぶひろ）

フェンリルのスマートフォンアプリエンジニア。iOS / Android / Windows 8 など幅広いスマートデバイスでのアプリ開発に精通している。最近は Xamarin を活用したクロスプラットフォームアプリ開発と普及活動に取り組み、講演などもこなす。スマートフォン勉強会の主催、各種イベントにおける登壇など、ITコミュニティで幅広く活躍している。

■ 岩瀬 高博（いわせ たかひろ）

フェンリルのチーフエンジニア。サーバサイド開発を得意とし、Java を中心に データベース、PHP など幅広い技術を熟知する。データベースの専門知識をもとに NoSQL データベース「okuyama」を開発、オープンソースソフトウェアとして公開した。Web+DB PRESS 等技術誌への寄稿多数。

■ 河野 健太朗（かわの けんたろう）

シナジーマーケティングのサーバサイドエンジニア。Ruby での Web アプリ開発だけでなく、社内ライブラリの一部を gem として公開したり、VMware のホスト構築自動化、Gitlab や Hubot を導入するなど開発環境の改善に取り組んでいる。Python の勉強会を初めとしてコミュニティへの参加も積極的に行っている。

■ 鈴木 圭（すずき けい）

シナジーマーケティングの言語大好きプログラマ。技術サイト TECHSCORE の編集長。C# で Windows アプリを作成したり、Ruby で Web システムを開発したり、直近では Java でプロダクト開発に携わる。より良いプロダクトのために奮闘中。Ruby ライブラリの公開や、Docker などの新技術導入にも取り組む。 Twitter: https://twitter.com/suzuki6b6569 GitHub: https://github.com/suzuki-kei
