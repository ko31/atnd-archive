description

　MYSCONが終わって以来、ミステリを読んだり、ボードゲームを遊んだり、すっかり日常生活に戻ってしまいましたが、読書会も継続してやっていきますよ！
　と言うわけで、スタッフの秋山です。
　ミステリを読んで感想を交換したり、意見を出しあったりして新たな発見を楽しめる読書会。
　その名も、MYSDOKU。年内は10月に第10回を、12月に第11回の開催を目指しています！

　さて。
　今回の課題図書は北山猛邦先生の『猫柳十一弦の後悔』と『猫柳十一弦の失敗』の2冊です。
　一緒に読みませんか？

日時：2013年10月13日（日）13:15受付開始 13:30-16:30

参加方法：右上の「このイベントに参加登録する」からお願いします。登録時のコメントは参加者リストに記載致します。
二次会：大森駅近辺の居酒屋にて二次会を予定しております（3500円程度）

【ぷちMYSRECについて】
　会場予約担当だった秋山の不手際で、うっかり夜間まで部屋を抑えてしまっていました。
　16時半で即座に撤収というのは、なんだか、もったいないので、16時半より「ぷちMYSREC」と称して、ミステリをテーマにゲームを遊びたいと思います。内容は当日発表しますが、続けてご参加ください。

【MYSDOKUボードゲーム部について】
　当日は朝10時から会議室を抑えておりますので、一緒にボードゲームを遊びませんか？
　参加希望者は早めにご来場ください。親切丁寧にルールを説明します。
　4人集まったら宮内悠介「象を飛ばした王子」に登場する『チャトランガ』というゲームを遊びたいと考えています。
