catch ビール片手に学ぶ Webサービスで失敗しないための品質担保の裏側〜ベンチマーク・負荷対応・コスト・監視などを実例をもとにお見せします〜
description

普段明かされることのない、webアプリ・webサービスの品質担保についてとりあげます。どのようなことをしているのか、すべきか、ベンチマーク・テスト・負荷対応・コスト・監視・運用体制などについて実際の開発・運用に役に立つヒントを実際のサービスを例にお話しします。

また、パネルディスカッションでは、登壇者と参加者がビールを片手に、
参加者の困った体験やその解決策をまじえながら語り合う時間を予定しています。

参加無料なので、お気軽にご参加ください。
