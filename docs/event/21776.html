title CoronaSDK 勉強会 in 掛川 (12/3)
catch スマートフォンのゲームや絵本を作ってみよう!
description

前半は初めて参加される方へのCoronaSDKの紹介、HelloWorldハンズオン、APIやツールについて解説。後半は、Coronaでの画面デザインや表示について解説します。Photoshopからスマホアプリの作り方も解説します。
最後の２０分くらいは、お題は自由のLTにしようと思います。
主催は、CoronaSDKの開発者コミュニティの日本コロナの会の山本です。

入場 (13:00〜)

前半 (13:15 – 14:30)

  • コロナSDKの紹介(@ymmtny)
  • APIについて
  • ツールについて
  • 環境構築とHelloWorld

    • ※環境構築の話もしますが、お手数でなければ CoronaSDKを http://anscamobile.comのトップページからTry Coronaをクリックして、インストールしてきて下さい。

    後半(15:00〜16:00)

  • コロナで画面を作ってみよう
  • デザインツールについて(Photoshop, Flashを使ってみる)

    • -16:30

  • Lightning Talk (参加者 募集中）

    • コロナ以外でもよいので、なにか５分程度お話しませんか
    パルアルトのアンスカ社にいってきました。(@ymmtny)

    退場 (16:45)

    参加資格
