今回の勉強会は 「Agileな計画と見積り」 についてみんなで考えたいと思います。

みなさんにAgaileな計画立案を体験してもらい、Agileの良さを学んでいきます。
また、実際にAgile開発を行なっているEC-One monkey magicプロジェクトによる事例の紹介も行います。

福岡県のRuby開発サーバーを利用し、プロジェクト管理ツール「Redmine」の紹介デモも行います。
