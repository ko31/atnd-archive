description

1990年代のインターネット登場から急速な発展を遂げたIT。クラウドやIoTも普及し、人々の暮らしや企業のビジネスにITは欠かせないものとなってきました。 今回はインフラ技術の世界を渡り歩いてきた3名の講師をお招きし、OpenStackやBlockChain等のIT最新トレンドやビジネス戦略についてお話いただきます。講師の方々のバラエティ豊かなエピソードの数々を聞くことができる貴重なこの機会を、ぜひお見逃しなく！

パブリッククラウドと同等の機能をオンプレミスでも利用することを実現するOpenStack。その活用はあらゆる企業において急速に進んでいます。本セッションでは、最新のトレンドを利用事例を含めてご紹介いたします。

講師：ブロガー/プログラマー/投資家　小飼 弾 様

「働いたら負け」。オン・ザ・エッヂ上場はそれが正真正銘の事実であるということを講演者に突きつけました。実は誰もが突きつけられているこの事実を前にこの先、生き残るためにはどうしたらよいのでしょうか？
20:05～20:10 5分 休憩
20:10～20:40 30分 BlockChainの実際と今後
～使ってみたら何かがみえた～

講師：株式会社オルトプラス CTO　嶋田 健作 様

仮想通貨をはじめ、様々な分野で期待されるBlockChainの技術。実際に顧客への提供を考えて、チャレンジしてみた活用方法と今後の課題などをご紹介します。
20:40～21:00 20分 質疑応答

■ 輿水 万友美

レッドハット株式会社

■ 小飼 弾

ブロガー/プログラマー/投資家
1969年生まれ。元オン・ザ・エッヂ創業時CTO。 カリフォルニア大学バークレー校を経て、後のライブドアとなる株式会社オン・ザ・エッジの取締役最高技術責任者を務め同社の上場に貢献。 2004年、ブログ「404 Blog Not Found」開始。人気No.1のアルファブロガーであり、プログラミング言語Perlの標準添付最大のモジュール「Encode」のメンテナンスを担当するカリスマプログラマーとして活躍。 著書に『弾言』、『「中卒」でもわかる科学入門』、『未来予測を嗤え！』など。


■ 嶋田 健作

株式会社オルトプラスCTO / NHN テコラス株式会社特別顧問 / 株式会社情報工場取締役
株式会社ライブドアを経て、株式会社データホテル、テコラス株式会社（現、NHNテコラス株式会社）を経て、現在は、株式会社オルトプラスなどで新技術の導入などを推進。
