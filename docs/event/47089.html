html5j自動車部について

昨今、W3Cにおいて車載機器とWeb技術の融合に関して様々な動きが活発になってきています。
自動車部では、そのようなW3Cの動きをウォッチするとともに、QNX Cars、TizenIVI、Webinosなどの車載機器に関係するプロダクトの最新情報の発信、Webエンジニアと車載機器等の組み込み系エンジニアの方々との技術交流を促進していきたいと考えています！

勉強会の見どころ紹介！

セッション１：「2014 international CESから見た自動車ＩＣＴトレンド」

形式：講演＋意見交換会

映像配信：無し（会場でしか聞けません！）

2014 International CESが1月7日～10日、米国ラスベガスで開催されました。
今年は自動車関連の展示が増え、CESの”C”が、Carではないかと思えるくらい、クルマが、家電・モバイルサービス等、他産業を巻き込んでいくプロダクツであると印象づけるイベントになりました。
勉強会では、自動車メーカ9社及びサプライヤの出展やセッション発表等を中心に、そこから見えるトレンドを紹介するとともに、これらを踏まえ、勉強会に参加された皆さんと、自動車業界とＷｅｂ技術の方向性についてディスカッションを行います。

セッション２：「車載組込ブラウザの過去7年と今を40分でまとめてみる」

形式：講演＋デモンストレーション

映像配信：有り

Web と車の中間に位置する車載組込ブラウザ。
登場から今まで劇的な進化を遂げているスマートフォンを例に取り、スマートフォン代表として初代 iPhone が発売された2007年から今の時間軸を横軸、HTML、ブラウザ、OS、SoC(System on Chip)を縦軸に、車載組込ブラウザにこれまで何が起こり、今何が起こっているのかをまとめ、車載組込ブラウザの未来についても軽く触れます。
セッションの後は、ボードに、車載半導体世界シェアNo.1 の Renesas Electronics 社の、車載情報端末向けSoCとして9つのCPUで業界最高性能の R-Car H2、ブラウザに、世界で10億台の組込実績がある NetFront Browser の最新 Ver である NetFront Browser NX を用い、まさに今鋭意開発中のブラウザのパフォーマンスが確認出来るデモを行います。
