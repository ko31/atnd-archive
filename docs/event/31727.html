description

openSUSE の開発で使われているパッケージ開発ソリューションの OBS の使い方と openSUSE のパッケージングについて学ぶ会です。OBS は openSUSE だけではなく、Ubuntu / Debian, Fedora などのパッケージも管理でき、Tizen などの外部のプロジェクトでも採用実績があります。

お願い

会場でパッケージをビルドしようとすると、依存するパッケージをネットワーク経由でダウンロードしてしまうため、ネットワークの帯域が不足してしまいます。WiMAX 等をお持ちの方はできるだけ持参して下さい。

OBS のコマンドラインクライアントを事前にインストールしておくことを推奨します(ディストリビューションは不問) 。実際にビルドするには、パッケージによりますが、1 GB ほどのディスク空き容量が必要です。

OBS クライアントを自宅サーバーなどにインストールして SSH でアクセスできればベストです。帯域の問題も回避できます。

予定する内容

  • OBS の概念と openSUSE の開発プロセスでどのように活用されているか
  • OBS の基本操作 & コマンドラインクライアント
  • RPM パッケージの作成

対象者

  •  openSUSE の開発に参加したい人
  • パッケージを各ディストリビューション向けに配布したい人
  • 他のディストリビューションの開発者で OBS を知りたい人
  • なんとなく興味のある人

前提知識

  •  野良ビルド (例 ./configure, make, make install) をしたことがある

参加費

会場費を割り勘する形となります。目安としては4人集まった場合で500円です。(学生無料)

夜の部

おそらくどこかでやることになると思います。幹事としてはすぐ近くのベトナム料理屋が気になっていますが、人気店なので厳しいかもしれません。参加人数を見て調整します。
