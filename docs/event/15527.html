description

海外で人気を集めているボードゲームをいろいろ集めて、みんなで遊ぼうというイベントです。

ゲームで疲れた脳をゆっくりクールダウンしてもらえるように、ドリンクチケットと軽食をご用意してお待ちしております。お酒を片手に、ゲームを楽しんだ仲間と感想などを語り合ったりしてください。

Urventとは、
オフィス新大陸さん に名作・新作のボードゲームをご提供頂き、ルールなどを教えてもらいながら楽しむ会です。
だから、海外のボードゲームで遊んだことが無い人でも日本語しか分からなくても大丈夫！大歓迎！ぜひ遊びにきてください。
