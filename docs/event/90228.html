【ご注意ください】

以下の場合、「不参加」とみなします。

(1) 正規の手順以外で投稿された方

  • YouTub や ファイル共有サービスを使いたく無い方、参加をご遠慮下さい。
  • ツイッターやLINEからの動画投稿は「不参加」として扱います。

(2) 途中で途切れている動画を投稿された方

  • 最初や最後が切れてる動画を投稿したい方、参加をご遠慮下さい。
  • 途切れていると、リレー動画に映像が入らない可能性があります。
  • その場合、「不参加」とさせて頂きます。

(3) 無断で締め切り以降に投稿された方

  • 期限が守れない方、参加をご遠慮下さい。
  • 期限を守っている方のご迷惑になります。

(4) 再生位置を指定された方

  • 再生位置を指定されたい方、参加をご遠慮ください。
  • 再生位置はクジで決めます。
  • 「XXを使って」あるいは「XXを使って欲しい」といったご要望は一切受け付けません。

■参加方法

  1. ダンエボ筐体の動画を最大５本ご投稿頂けます。
  2. 動画をYouTube 又は ファイル共有サイト (DropBoxなど) へアップロード下さい。
  3. 出来るだけ撮影した動画そのままご投稿願います。
    ※「KINEMASTER」で編集するとロゴが入るため
  4. 投稿動画は、2017年1月1日以降に撮影したものに限ります。
  5. 下記の「必要事項」をアンケートにご記入下さい。

＜必要事項＞

  1. 投稿者
    　※無記名でもOK
  2. 店舗名
    　※必須
  3. 動画のアドレス
    　※必須

■編集した動画について

■注意事項

  1. 下記の場合、「不参加」として扱います。
    1. 正規の手順以外で投稿された方
    2. 途中で途切れている動画を投稿された方
    3. 無断で締め切り以降に投稿された方
    4. 再生位置を指定された方
  2. お一人様１回限り最大５本、ダンエボ筐体が映っている動画をご投稿頂けます。
  3. 撮影する動画に本人や、お友達が映ってもＯＫ。
    ※記念に是非映って下さい。
  4. 万が一、無関係な方が映った場合、こちらでモザイク処理しますのでご安心下さい。
  5. 既に動画を投稿された方が別の方の動画に映ってもＯＫです。ただし、意図的に何度も繰り返し登場する場合は、参加をご辞退頂く場合があります。
  6. 編集は、たけぷ～ が実施します。
  7. 投稿頂いた元の動画に対し、大幅な改編は行いません。
  8. 公開後、諸事情により編集した動画を削除する可能性があります。
