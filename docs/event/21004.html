catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション１(30分)
　『Androidが変えたもの』山城戸さん

　＜内容＞
　　OHAからAndroidが発表になって３年。
　　日本でAndroid端末が発売されて２年。
　　これまでに、Androidは何を変えて、何を変える事が出来ていないのか。
　　そして、スマートフォン大流行の中で、これからAndroidは何を変えて行くのか。
　　肩の力を抜いて考えてみようと思います。

　　※意外に時間が余った場合にはオマケとして
　　　「FOMAってもう新機種でないの？でもAndroidってauなんでしょ？
　　　iPhoneはタダらしいんだけど」と聞かれたときの１０の対処方法
　　　をお送りします。

　＜山城戸さんの紹介＞
　　・日本Androidの会中国支部からの刺客。担当は、受付。
　　・組み込み系のプログラマー。WEB系はちゃきちゃきのド素人。
　　・Androidが発表されたとき、組み込みを駆逐してしまうのではないかと
　　　恐怖におののいて以来、Google様のお膝元で生きて行こうと決意。
　　・勉強会で話すときには、聴講者参加型を目指す。

　

●セッション２(40分)
　『Google Apps for Business、Google Appsの紹介』西田さん

　＜内容＞
　　独自ドメインで利用できる「Google Apps」を導入してみました。
　　導入のきっかけは、書籍「iPhone+iPad*Googleでビジネスを加速する」を読んだことから。
　　無料版「Google Apps」と有料版「Google Apps for Business」を紹介します。

　＜西田さんの紹介紹介＞
　　広島市工業技術センター所属。
　　Mac、iPhoneユーザー。
　　仕事ではウェッブ関連のことはしていないのですが、個人的にIT利活用しています。
　　WordPressを利用していますが、Drupaも使ってみたい。

　

●セッション３(40分)
　『スマートフォン向けwebサイトを作ってみよう』藤本

　＜内容＞
　　ここ最近スマートフォン対応を余儀なくされているので
　　スマートフォン向けのサイト制作とはどういったものなのか話します。
　　まだ自分自身勉強中の身ではありますが
　　復習もかねてまとめてみます。
　　HTML5やCSS3を使った技術など
　　まだまだこれから勉強していかないといけないので
　　始めるのに遅いということはありません。
　　一緒に作れるようになりましょう！
　　すでにばりばり作られている方は温かく見守っててね！

　＜目次(予定)＞
　　・スマートフォン対応サイトとは？
　　・PC版とは似てるようで違うコーディング
　　・iPhone/Androidの仕様の違い
　　・スマートフォンサイトを設計する時のポイント
　　・とまどいやすいiPhoneコーディング
　　・実例を見ながら覚える良いサイトの画面設計
　　・スマートフォンでは重要なJQuery Mobileの基本
　　・おまけ：便利なサイトの紹介

　＜藤本の紹介＞
　　・WEB業界は６年目で、広く浅くやっています。
　　・基本的にWEB技術とか仕事とか関係なく、人が好きです。
　　　人類みんな兄弟だと思っているので、
　　　初対面の人でも３年ぶりに会った友人かのように
　　　いきなり慣れ慣れしく話しかけると思うので注意してください。
　　　こちらの心は常にフルオープンです。
　 ・CSS Nite in HIROSHIMA 実行委員長
　 ・WEB TOUCH MEETING のさりげなく主催者
