catch 第５回　八時間耐久作品制作回＠北陸先端科学技術大学院大学
description

八時間耐久作品制作会とは８時間という短時間の間に，ゲームやCG，アプリケーション、デザインなどの作品を制作し，発表するイベントです。

また、必ずしも制作を終了する必要はなく、途中から制作することもできます。チームでの参加（ペアプログラミングなど）もOKです。

参加者は本学の学生や周辺地域の学生・社会人の方々を想定しています。

前回のゲームサークルのイベント「Global Game Jam 2013@JAIST」については、 ブログ や 告知サイト を参考にして下さい。
ちなみに前回の開発環境はWindows XNAやUnity、Flash、HTML+JavaScriptといったものでした。ゲームの企画やFPGAなどでも大丈夫です。

当日は、運営側で無線LANを用意致します。開発用器材は基本的に各自の持ち込みでお願いしますが，こちら側で用意できる備品（モニター等）は事前にご連絡頂ければ，用意できることがありますので気軽にご連絡ください。

なお，参加費は無料となっています。

当日のスケジュールは以下になります。
　　９：３０ 　:　 開場
　１０：００　：　諸注意・制作開始
　１８：００　：　制作終了
　１８：１０　：　作品報告
　１９：００　：　解散
（当日、大学施設は施錠されています。外部からの参加者の方への入館の対応については後ほどお知らせします。）

石川・北陸地方をゲーム開発で熱くして行きましょう！
