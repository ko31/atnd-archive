description

昨年１２月にさいたまスタジアムで開催された第１回電動スクーター乗り比べ試乗会に引き続き、第２回目の試乗会を開催します。今回は自然豊かな筑波山のふもと、茨城県筑西市に場所を移しての開催となります。今回も前回と同様、ボイジャー・モトのパートナーである電動スクーター・メーカー５社の車輌を一堂に展示し、全ての車種を試乗する事が出来ます。
今回はVECTRIX Japan様の協力により、国内で最大級の普通２輪電動バイク「VX-1」も展示・試乗が可能となりました。　都内でも希少な最先端のスーパー電動バイク　VX-1を是非とも見て、乗って体験して頂ければと思います。
