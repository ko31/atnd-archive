来たる８月１８日（土）、札幌ドームで「 日本ハム対ロッテ 」戦が行われます！

がこの試合・・・　試合終了後にホンモノの花火大会もあわせて行われるという一大イベント！
(*´∀｀)＜（しょぼい花火じゃないですよ、おっきい花火ですよ…）

せっかくの花火大会・・・みんなで見たほうが楽しい！に決まってますよね！！

ってことで野球観戦×花火観覧あわせてしちゃおう！オフの開催をここに宣言いたしますっ(・∀・)

・長時間イベントのため２部制で行いたいと思います。

第１部～　 13；00～　日本ハム×ロッテ戦観戦
（注：１）『ライト外野自由席』において参加者皆で日ハム応援を予定しています。
第１部から参加される方は、事前に『ライト外野自由席』のご購入をお願いいたします。

第２部～　 試合終了後～21：00（予定）　札幌ドーム花火大会観戦　
昨年の花火大会が15時プレイボールで20：15分打ち上げ開始だったので、今年もおそらく
同じ時間設定だと思われます。

試合終了後～花火大会開始まで約２時間のタイムラグが予想されますので、
買い出し部隊を募って、食べ飲みしながら花火が見れれば・・・
というのが個人的な提案なのですが、
ここの部分については参加者の皆様からアンケートでご意見をいただいたうえで、
再度決定、ご案内させていただきたいと思います。

・参考までに去年の札幌ドーム花火大会の概要ページです。
http://www.fighters.co.jp/news/detail/2205.html

それでは、みなさまのご参加お待ちしております！
