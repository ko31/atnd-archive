～これから恋愛は学ぶ時代に!～

社内恋愛・結婚ができない方は

それは恋愛も同じです。
「恋愛も学校では教わらないもの」のひとつです。

では、誰に教わったらより効果があるのでしょうか?
逆に、世の中に恋愛テクニック本は沢山出ているのに効果がないのは何故でしょう?

例えば野球を習うときに、野球をあまり知らない人に習うのと、イチローに習うのでは、当たり前ですが

イチローに習った方が上手くなると思いませんか?

恋愛を教わるときも同じです。
講師のバックグラウンドとはとても重要。

この講義では、元歌舞伎町No.1ホストだった安彦氏が
2万人以上の女性と出会い蓄積した知識を活かし

これまでの経験から実践立証済みのテクニックと
そこから90日間で彼女を作る方法を参加者に特別に伝授させていただきます!
