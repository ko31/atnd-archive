Team UE4 Start !!

～ コンテンツ開発・制作の未来を担うUnreal Engine職人集団「チームUE4」を結成します ～

光の表現が他ゲームエンジンより手軽に美しい表現ができ、Blue Printの採用により、非プログラマでもゲーム作成が容易なことで有名な「Unreal Engine4（UE4）」。

これまで、UE4に関するイベントや勉強会を開催してきましたが、どの講座も満員御礼。UE4に対する参加者のみなさまの興味、熱意はすさまじいものだと感じました。

そこで、クリーク・アンド・リバー社ゲームスタジオでは、「もっとUE4を活用してゲームクリエイターのみなさまと実践的な取り組みをしていきたい！」と考え、企業や肩書きを超えたフラットな取り組みができる組織、通称「チームUE4」を設立しようと考えております。

そんなわけで、「チームUE4」のメンバーを募集を目的とした、「チームUE4」についての説明会を開催します！

「説明会参加＝即加入」ということではなく、普段UE4を使っている方やUE4に興味がある方の「UE4でこんなことをしてみたい！」という声もお聞きしたいと思っているので、ゲームクリエイターやエンジニアのみなさん、ふらっとお気軽にご参加ください！

 

■ チームUE4でこんなことがしたい！（あくまでも希望）

クリーク・アンド・リバー社ゲームスタジオでは、これまで数多くのコンシューマーゲーム、モバイルゲーム、遊技機の開発・制作などに取り組んできました。

そんな私たちにとって、新たなる武器となるのがUE4だと考えています。

これまでの実績と設備を活かし、UE4を使ったゲーム開発ができる環境を整え、さらに発展させていく予定です。

これらのリソースは社内だけではなく、現在フリーでお仕事をしている方はもちろん、企業に所属しているクリエイターやエンジニアのみなさまにも開放し、ともにUE4を使ってゲーム業界を盛り上げていきたいのです。

現在考えている「チームUE4」の活動はこのようなものを想定しています。


  • UE4に関する定期的な勉強会

  • 弊社スタジオにてモックやサンプルの制作

  • 副業したい方・個人事業の方向けのUE4案件紹介

↑これらはあくまでも私たちのやりたいことです。参加者のみなさまと一緒にチームを作り上げていきたいと考えています。

■ 「チームUE」加入条件（下記のいずれか）

  • UE4未経験だけど、実務を通して習得したい「エンジニア」の方
  • UE4で仕事の幅を広げたい「アーティスト」の方
  • UE4の可能性に未来を感じている「プランナー」の方
  • UE4のコンテンツ開発を担える「ディレクター」の方

「UE4はまだ勉強中！」という方でも、やる気意欲のある方であればOK！
フリーランスの方はもちろん、企業にお勤めのクリエイターさんのジョインも大歓迎です！

※弊社の社員としての採用ではありませんので、お気軽にご参加いただけます。
