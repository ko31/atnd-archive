そんなことよりHaskellしようぜ

Haskell入門者のための勉強会 スタートHaskell の 第3回 ATNDです。

スタートHaskellは、入門者から初心者の方を対象に、実際にコーディングをしながら１からHaskellを学んでいく勉強会です。月1回ペースで全5回を予定しています。

第3回は プログラミングHaskell の5章から7章までの範囲を取り上げます。

お願い

名刺を１枚ご用意ください。

ニフティでは、入退室管理および機器破損、怪我、トラブル発生時のために、受付時に名刺（または氏名、連絡先）をいただいております。いただいた個人情報はニフティにて適切に処理しますので、上述以外の目的や本勉強会のために使うことはありません。また、会場内での紛失やトラブルに対しては責任が取れませんのでご了承下さい。
