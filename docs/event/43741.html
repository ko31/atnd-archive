【Vol.1】活用術から上級テクニックまで、WordPressの裏技完全マスター

WordPress (ワードプレス) はオープンソースのブログ／CMS プラットフォームであり、全世界で圧倒的なシェアを誇っています。単なるブログツールの枠を超え、企業サイトや商用サイトを構築するためのプラットフォームとしても理想的であることから、現在も頻繁にバージョンアップされています。利用が標準化していく中で、WordPressは今、Webクリエイターに求められる重要なスキルの一つとなっています。今回は、HTML/CSSの基礎知識がある方を対象にWordPressの可能性や実践的なノウハウを学ぶセミナーを開催します。

今回は、HTML/CSSの基礎知識がある方を対象にWordPressの可能性や実践的なノウハウを学ぶセミナーを開催します。講師はWordPressの第一人者である星野邦敏さんとanygraphicaプログラマー谷口允さんをお招きし、Webクリエイターであれば知っておきたいWordPressの技術や裏技をご紹介します。

WordPress日本語公式サイトのイベントカレンダー更新、WordPressイベントにスタッフやスピーカー、オープンソースCMSである「WordPress」の普及に貢献。WordPressの技術関連書籍の著書・情報協力という形で携わっている書籍も多数発売され、WordPressの第一人者として活躍中。
