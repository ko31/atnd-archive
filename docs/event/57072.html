会場案内

今回は、GMO インターネット株式会社 の GMO Yours をお借りして開催します！　ページトップの写真が会場の写真です。GMO Yours は、世界一のサービスを提供するために世界一の人財があつまる場として作られ、24時間無料で利用できると評判のスペースです！

入館方法・諸注意（よく読まないと入館できません）

  • ＊名刺＊を持参してください
    • 下記 セキュリティカードの返却が確認されるまで GMO インターネット で保管されます。
