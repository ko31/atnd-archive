今、Webサイトを更新できるような機能やスマホの対応は普通に求められる時代ですが、みなさんはどんなCMSをご利用でしょうか？

a-blog cms は、2009年にリリースされ6年で400社、1900サイト以上の商用サイトで利用されている国産のCMSです。PHPとMySQLの環境で動作しますが、カスタマイズにPHPのコードを書く必要がなく、HTML+CSSが書ける人であれば自由にカスタマイズが可能な仕組みになっております。

無料

ざっくばらんに聞いてみようは、実際の案件で今困っていることに対する相談でもいいですし、まったく触ったことがないという人からの質問や、インストールで詰まっているなど、時間の許す限りの相談にお答えしたいと思います。 前日のCMS夏祭りでわからなかったことや、十分話せなかったという方は、ぜひ詳しく話を聞きにきてください。

参加特典：

1.Web制作会社向け特別ライセンスを提供

ライト版を無料で利用する事は可能ですが、フル機能を試してみたいと思った時にすぐに利用できるように「Web制作会社向け特別ライセンス」を提供します。詳しくは Webサイト の方でご覧ください。

2.さくらインターネット様よりサーバを1年間提供

さくらインターネット 様の協力により、a-blog cms の勉強会に参加される方に1年間、CMS が動作する環境「さくらのレンタルサーバ スタンダード」を無料提供頂けることになりました。

3. 初めて利用される方にはライセンスの割引販売
