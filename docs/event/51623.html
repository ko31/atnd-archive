来年2015年は空海（弘法大師）が高野山を開創されて1200年。

これからますます盛り上がる高野山上を体験しに行きませんか・・・

高野山というとお寺の立ち並ぶ荘厳な雰囲気のイメージですが、そこにはたくさんの
人々が暮らしています。自然豊かな、歴史と文化が融合した山上の町なのです。

今回の現地体験会では、高野山ならではの数珠作り体験や美味しい精進料理、先輩移住者や地元の方々とふれ合う交流会を予定しています!!

「わかやま田舎暮らし現地体験会 in 高野町」

【開催日】：6月7日（土）定員20名・先着順

【参加費】：3,000円（税込・体験代・昼食代・保険代含む）

※集合場所までの往復交通費と交流会の飲み物代は別途お願いします。
