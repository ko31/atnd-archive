■概要

7/4-5のOSC名古屋開催に合わせて、OSC1日目の夜 7/4(金)に第19回ユーザー会 in 名古屋 を開催します。

CloudStackは、IaaS型クラウドを実現するソフトウェアとして、多くのクラウド事業者や 北海道大学アカデミッククラウド をはじめとした学術機関などで実績があるソフトウェアです。

CloudStackは、正式にはApache CloudStackという名前です。その名のとおり、Apache Software Foundation傘下のOSSプロジェクトとして開発されています。また、CitrixがApache CloudStackをベースとしたCitrix CloudPlatformを商用製品として提供しています。

名古屋では初めてのユーザー会開催となります。
会場はOSCと同じ名古屋国際センターですので、OSCに参加された方はぜひ併せてご参加ください。
今回は初心者の方にも優しいコンテンツをご用意しました！初めての方もお気軽にご参加ください！！

こんな方にオススメ
・クラウドを作ってみたいインフラエンジニアの方
・CloudStackとかOpenStackとか、最近よく聞くけど何が違うの？という疑問をお持ちの方
・オープンソースプロジェクトに興味のある方
・クラウドの中の人にいろいろ質問してみたい方
