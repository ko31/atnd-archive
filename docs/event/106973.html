Scale Model Builders Fukuoka モデラー交流会

福岡市および近郊のスケールモデル製作を趣味とされている皆様、またはスケールモデルに興味がある皆様との交流の機会として、下記イベントを開催致します。
今回初めての開催形態となり、会場・収容人数を絞っての開催となります。悪しからずご了承ください。

なお今回は本イベントを第一部とし、本イベント終了後会場を移し 第二部『 第2回模型呑会 』 の開催を予定しております。
どうぞ奮ってご参加ください。

開催日時・開催場所・参加費について

日時：2019年8月17日（土） 13:00～17:30
　会場は12時から押さえており、準備のため主催者が入室しております。早く着かれた方は入室いただいて結構です。
　（準備中はバタバタしているかもしれませんが）
17:30閉会後、後片付けを済ませて第二部会場へ移動致します。

場所： リファレンスリバーサイド貸会議室　R-3会議室
住所： 福岡市博多区博多駅東3丁目15-16-2F

参加費用：1名につき100円（高校生以下無料）
　※参加費用を改定しました。

イベント概要

  • モデラー同士の歓談
  • 作品展示（※ 室内の机の一部を展示用スペースとして利用します）
  • 模型製作（※ 下記注意をご参照ください）
  • 【特別企画】モデルキット999（ウォルターソンズジャパン）製作体験会

主にモデラー同士のコミュニケーションを目的としています。従来開催してきました「模型茶会」の延長線上のイベントとなります。
今回はさらに模型製作も行えるよう、喫茶店ではなく貸会議室を借りてみました。

また今回は特別企画として、ウォルターソンズジャパン様より発売されている「モデルキット999」シリーズの製作体験会を実施致します。
普段スケールモデルはあまり作られない方、普段作っているものとは違うジャンルにチャレンジしてみたい方に、入門キットとして是非お試しいただきたいと思います。
キットは弊サークルにて少数ですが用意させていただきました。作成されたキットはそのままお持ち帰り頂いて結構です。

Scale Model Builders Fukuoka(SMBF)について

Scale Model Builders Fukuoka(SMBF)は、AFV、航空機、艦船、自動車などのスケールモデル製作を趣味とする方々同士の情報交換やコミュニケーションを目的としている模型サークルです。

ポータルサイト「Scale Model Builders!!」
SMBFサークルご案内サイト

福岡市および近郊にお住いのスケールモデラーの皆様と実際にお目にかかり、模型についての話をする機会として、 模型茶会 （ 模型 をネタにお 茶 でも飲みながらダべるオフ 会 ）や 模型呑会 （ 模型 を肴に酒でも 呑 みながらダべるオフ 会 ）を実施しております。
実際に作成された作品を持ち寄ったり、技法や製作中の悩みについての相談、情報交換や、様々なお話ができる機会になればと思っております。
これまでひとりで黙々と製作を行っていたという方がほかのモデラーの方と出会う機会になれば幸いです。

諸注意

  • 飲料の持ち込みは可能です。
  • 室内禁煙です。喫煙は指定場所（2Fバルコニー）でお願いします。
  • お隣へ響くような大きな音出しや発声はご遠慮ください。
  • ゴミは各自にて処分をお願いいたします。
  • 今回イベントは室内での模型製作が行えますが、以下ご注意願います。
    1. 組み立てはOKですが、塗装は不可です。
    2. 臭い（溶剤臭）対策のため、リモネンセメントの利用を推奨致します。（主催者側でも若干数ですが貸出品を用意します）
    3. 汚損防止のため机上は養生していますが、接着剤は指定のトレイ内で使用いただきますようご協力願います。
