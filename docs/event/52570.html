「mbedマイコンプラットフォーム上での関数型プログラミング」Metasepi project 主催、Japan ATS User Group 会長 master_q

http://metasepi.org/
http://jats-ug.metasepi.org/

ATSは関数型言語であり、また定理証明によるプログラミングも兼ね備えて、ポインタやCライクな文字列操作を本質的に安全に行うことができます。
セキュリティ対策やメンテナンスコストという観点からこのような概念は必須ですが、エンタープライズ用途のみならずGCを使わないメモリ管理、フットプリントが小さいという特徴はカーネルレベルや組み込み開発にも多大な恩恵をもたらすことでしょう。
今回はこの特徴を示すために組み込み向け ARM マイコンプラットフォームであるmbed 上のアプリケーション ATS を適用した事例についてお話いただきます。
