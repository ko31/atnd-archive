title 「実務的・防災マニュアル」作成のためのガイダンスセミナー
catch 貴社は次の震災への準備はできていますか？
description

リスクマネジメント協会のアンケート調査によると「3.11東日本大震災」の経験で、自社の防災マニュアルは不十分であったと応えた企業は80％におよびました。しかし、実際にマニュアルを再度構築するとなると費用や時間の問題でなかなか着手できていないのが現状です。本セミナーでは防災マニュアルの柱となる最低限必要な準備と効果的なマニュアル作りの手法を専門家がわかりやすく解説します。

講師：　黄野　吉博
　　　　社団法人日本工業技術振興協会　BCM室長
　　　　リスクマネジメント協会　評議員

★セミナー内容
防災対応は企業の所在地・規模・経営者の考えで大きく異なります。
対応は「完璧」を求めるものではありません。災害を想定し準備できるものは準備し、できていないものはできていないと認識することが重要です。
