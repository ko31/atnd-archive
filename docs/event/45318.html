description

ニコニコ技術部とはソーシャルメディアのニコニコ動画にて自分の発明や工作を発表する人達の集まりです。特に運営主体が存在せず、ジャンルを問わず、職業も環境も関係なく、ニコニコ動画にて動画を発表をしています。また、オンラインだけでなく、東京、京都、名古屋、福岡、金沢など全国各地で自発的にオフラインの集まりを開いて実際に作ったものを見せ合うこともしています。札幌はこれまで要望があったものの実現には至りませんでした。しかし、今回、大規模イベントNT札幌に向けて、初のオフラインミーティングを開催します。ご興味のある方はぜひご参加ください！
