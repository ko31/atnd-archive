description

雨のため中止になりました。

ウェブサイトを制作するにあたって重要な『写真』・・・。
これ一つで商品の価値やそのサイトのイメージまで善し悪しが
変わってしまうというこれから我々にさけては通れない存在です。
フツーはプロのカメラマンが撮りWEBなどで販売されてたりしますが、手間、コスト等を考慮し自分で撮るという事になるのは今後想定されます。カメラマンにプロがいるというのにも納得するぐらい、それっぽく奇麗に撮るのは難しいと思います。のでー、実際にパシパシ撮り、構図等を意識しながら少し勉強しましょうという内容です。カメラは一眼レフでなくてもデジカメでもいいと思います。
ご気軽にご参加ください！

※尚、この会はデジタルハリウッド福岡校の在校生、卒業生、その他関係者に限定させて頂きます。ご了承下さいませ。
