書籍発売記念！フィスコ仮想通貨取引所で始める「ビットコイン取引」超入門

近年、金融とITを融合させた新サービス「フィンテック」に関心が集まっていますが、そのなかでも、4月から法制度の枠に組み入れられた「ビットコイン」など「仮想通貨」に対する注目度が高まっています。今回のセミナーはその「ビットコイン」にテーマを絞り、フィスコのアナリストやレポーターが初心者にも分かり易く解説！ビットコイン投資の基礎からビットコイン投資法までを解説します。

株式市場においても足元で注目が集まっている「ビットコイン」への投資法について理解を深める絶好の機会です。当日は仮想通貨取引所の口座開設方法もレクチャーいたしますので、ぜひお聴き逃しなく！

参加者特典：

フィスコ仮想通貨取引所 に 口座開設して1万円以上の日本円入金が確認できたセミナー参加者 には、ビットコインアナリスト「田代昌之」が執筆した書籍 “フィスコ仮想通貨取引所で始める「ビットコイン取引」超入門” をプレゼントいたします！（取引所にログインして日本円残高を見せていただければ差し上げます。セミナー参加時に口座未開設、または未入金の方は、確認後に郵送させていただく予定です。）

なお、最新のビットコイン情報は FISCOウェブ の 「アルトコイン評価ニュース」 で定期的に配信中です。併せてご参照ください。
