ここ数年、O2O/オムニチャネル市場は爆発的な伸びを見せており、いち早くこの業界に参入したアイリッジは、2015年7月に東証マザーズ上場を果たしました。

▼『アイリッジ、上場セレモニーに密着！』 URL : http://iridge.jp/news/201507/9076

そんなアイリッジが、10月8日（木）にO2O業界への転職を考えている方向けのセミナーと交流会を開催いたします。 位置情報やプッシュ通知などのテクノロジーで大手企業の集客・マーケティングを支援するO2Oは、SIerやシステム開発会社で開発業務に従事するPM/PL/SEの方が、それまでに身につけた技術的知見をベースに次のキャリアアップにチャレンジするにはもってこいの業界です。 今回は、アイリッジという急成長中のO2Oベンチャーで働くSIer出身のメンバーが、

「なぜO2O業界に転職してみようと思ったのか？」 「現在どんな仕事をしているのか？」 「どんなスキルが身についたか？」 「O2O事業の魅力は？」 といったことをお話させていただきます。

大企業やSIerでは味わえないチャレンジングな仕事やスピード感のある職場にご興味のある方は、ぜひご参加ください！ みなさんのご参加をメンバー一同お待ちしています。
