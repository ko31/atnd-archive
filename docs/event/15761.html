【講座趣旨】
Google App Engine(appengine)はPaaS型クラウドとして最も有名なクラウドコンピューティング基盤です。appengineは検索機能として世界 NO.1のGoogle検索で培われた技術を応用して構築されていますが、“実質無限に近いスケールアウトを可能とする、データストア(分散キー・バリューストア=分散KVS)”、“全世界130万台を超えると言われるサーバインフラを使用した巨大分散システム環境”をベースとして稼働するクラウド環境で、最近では日本での適用事例も増えており、またappengien自体の機能も年々進化しています。
　本セミナーでは、“appengineでのデータストアとして採用された分散KVSの特徴とアクセス方法”、“appengineサービス機能の中でも実践的で有効なサービスの利用方法”について、実際のサンプルプログラムを用いて解説します。また、最近リリースされたappengine新機能の中でも、最も特徴的な機能と言える、Channel APIでのサーバプッシュ技法についてもサンプルプログラムを使用して解説を行います。なお、USBメモリ持参の方には、テキスト用書籍掲載のサンプルプログラムを進呈致します。

【プログラム】
1　App Engineのプログラミング環境設定
　
2　最初のプログラム(Hello World表示)
　
3　データストア(Bigtable)プログラミング
　3.1　Bigtableでの登録・参照・更新・削除
　3.2　ソフトスキーマ(任意のプロパティ並び)でのプログラム処理
　3.3　ソフトスキーマ(List形式のプロパティ)でのプログラム処理
　3.4　所有・被所有関係とトランザクション処理
　3.5　条件検索プログラムサンプル
　
4　サービス機能プログラミング
　4.1　Google認証機能(Google Accounts Java API)
　4.2　メモリキャッシュ(Memcache Java API)
　4.3　メールの送受信(Mail Java API)
　4.4　画像編集(Images Java API)
