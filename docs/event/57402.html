読書の秋ですが、今回はみなさんの読書を応援するために、３名様の方に無料で弊社の英会話学習書「英会話イメージリンク習得法」を進呈する読書の秋キャンペーンを実施致します！

応募方法は簡単。英会話イメージリンク編集部の公式twitterアカウントをフォローするだけです。たくさんのご応募お待ちしております！

応募期間

2014年10月3日（金）17:00～10月9日（木）17:00

応募方法

該当期間に編集部の公式twitterアカウントをフォローしてください
公式twitterアカウントはこちら→https://twitter.com/imagelinkeditor

当選の流れ

公式twitterアカウントからのダイレクトメールをもって当選とさせて頂きます。その返信に「送付先住所」と「氏名」をご連絡いただければと思います。

※お預かりした個人情報は、ほかの目的には使用いたしません。送付後の個人情報は破棄致します。
