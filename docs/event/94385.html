Nordic モジュールで遊んだ話

秘密結社オープンフォース　河野

micro:bit がはやっています。これで発明したものを製品にするには？
micro:bit に入っている BLE チップ Nordic nRF51 を使うには技適をクリアする必要がありますが、最近はNordic nRF51/52とアンテナ、ファームウェアを搭載したうえで技適クリアしたモジュールが安価に売られています。
これらを使ってカンタンにHackするやりかたを調べてみました。

「Echo 、LED を ON にして」と言ってみた

日本Androidの会秋葉原支部ロボット部部長　清水

信号の流れは、

(1)Amazon Echo Dot　(AIスピーカデバイス)
(2)AVS:Alexa Voice Service
(3)Smart Home Skill (Node-RED Alexa Home Skill Bridge)
(4)Node-RED (ラズベリーパイ)
(5)LED (ラズベリーパイ)

デモを予定しています。
Node-RED is a visual wiring tool for the Internet of Things.
ビジュアルツールNode-REDはまるで大人のスクラッチ！　楽しめます。
https://nodered.jp/
https://nodered.org/
Node-RED はハードウェアデバイス/APIおよびオンラインサービスを接続するための
ビジュアルツールで、ラズベリーパイ用OSであるRaspbianに標準で搭載されています。
