セッション紹介

順不同。

フッと思いついたスマホアプリを即座に実機で試せる Web サービス紹介！

ツール系の iOS アプリになりますが、プログラミング無し！申請も無し！で、アプリのラフ案（図・画像など）だけで、すぐにスマホ端末で試すことが可能になる海外の Web サービスを紹介する。当日は、実際にパソコンとスマホを使いながら、使い方やレビューなどを実践いたします。手間が掛からない分、アプリの試作品を大勢の人に試してもらえるようになるので品質向上にもメリットがありますよ。（40分程度）

スピーカー： まさとらんさん （@0310lan）　アプギガ（IT系メディアで副編集長やってます）、ネット生活人（コミュニティ）

Team Foundation Server 入門

まだソース管理に VSS を使っている所もあると思いますが、いい加減 VSS とはお別れしましょう。入門用の Express やクラウド版である Service が提供され、TFS 利用のハードルも大分下がってきています。VSS の後継というだけでなく、アプリケーションライフサイクル管理ツールという支点で TFS ならではの機能も紹介できればと思います。（40分程度）

スピーカー： なかじ（@nakaji） プログラミング生放送勉強会 四国支部長

Team Foundation Service 入門

昨年11月にベータ版から正式サービスとしてリリースされた Team Foundation Service を皆さん使っていますか？　この Web サービスでどんな事が出来るのか、そして ALM（アプリケーションライフサイクル）管理ツールという視点でどのように活用出来るのか紹介できればと思います。（40分程度）

スピーカー： You＆I（@you_and_i）　プログラミング生放送勉強会 名古屋支部長、名古屋アジャイル勉強会

WebMatrix と ASP.NET Web Pages（仮称　―― C# というウイルスで Web 掲示板でも作ってみる

ほとんどタイトル通り。Web アプリ＝PHP と思っている人、多いと思います。でも、意外と C#/.NET Framework が強力でオススメ。Razor 構文やデータベースを使いながら、サンプルアプリを Windows Azure Web Sites でホストするところまで、なるべく実例重視で頑張っていきたいなぁ、と思います。（40分程度）

スピーカー： daruyanagi（@daruyanagi）　http://daruyanagi.net/
