クエステトラ・Lang-8・ヌーラボ 合同勉強会 (#QLN)

皆様、ご参加ありがとうございました。OST の議事録を以下にアップしておりますので、ご参考ください。

  • OST議事録

クエステトラ、Lang-8、ヌーラボが三社合同で勉強会を開催します。

この三社は、ともに世界を目指すウェブサービス、ソフトウェアサービスを自社で開発、提供しており、最近では、ユーザの投票によって決定される「Asia Top 50 Apps」に各社のサービスが選出された事などが ニュース として取り上げられました。

このたび、その三社がサービスの開発や運営、サポートなどの様々な局面において、直面した困難をどのように克服しているか、というノウハウや、まさに現在進行系の悩みなどを 共有 し、 話しあう 場を作りたいと考えました。

そこで今回、活発なディスカッションをするために以下のような、二部構成にて勉強会を実施いたします。

  • 第一部
    • 各社サービスの紹介
  • 第二部
    • オープンスペーステクノロジー 「世界に通じる良いソフトウェアやサービスを作るには」

前半の第一部は三社のサービスを各々簡単にプレゼンテーション形式でご紹介いたします。

後半の第二部で行うオープンスペーステクノロジーについては、初めて聞くかたもいらっしゃるかと思いますが、簡単に言えば、参加者が皆で、テーマに沿って議題そのものを決める事からはじめ、それについて関心のある他の参加者とともにディスカッションをする、というミーティングのスタイルです。海外では数日をかけて数百名規模で、このスタイルでのカンファレンスが開かれることもあるとのこと。概要は後述しますが、当日にも進め方の説明の時間は用意いたしますので、初めてだから、といって心配する必要はありません。

今回の勉強会では、日頃から「世界に通じる良いソフトウェアやサービスを作る」事を考えている三社のスタッフ、エンジニアと参加者の皆様で、是非、活発な議論、情報交換をしましょう！

事前の準備は必要ありません。それでは、みなさんの参加をお待ちしています !!

オープンスペーステクノロジーとは

オープンスペーステクノロジー (Open Space Technology) とは小さなミーティングの集まりです。
今回は5トラック×2セッション、合計10個のミーティングが開かれます。
議題は当日集まった参加者がその場で出し合います。日頃の問題をみんなで話すことで新しい発見が得られます。

テーマは 「世界に通じる良いソフトウェアやサービスを作るには」 です。
たとえばこんな議題が考えられます。

  • EC2 本当に使えるの？！
  • サイトの翻訳プロセスどうしてますか？
  • 有料と無料の境目の勘所
  • そもそもなんで世界なの？
  • 2010年グローバルな UI のトレンド

これはあくまでサンプルです。
実際はみなさんが興味のある話題を話し合います。

スピーカ紹介

▶▶ 畠中 晃弘（はたなか あきひろ）

株式会社クエステトラ 取締役 兼 CTO。1975年生まれ、2000年京都大学大学院情報学研究科修了。

大学院修了の時にITベンチャー（株式会社四次元データ）の創業に参加。
2008年にスピンアウトし、株式会社クエステトラを創業。

ビジネスプロセス管理システム　Questetra BPM Suiteの開発に携わる毎日。

▶▶ 西嶋 悠貴（にしじま ゆうき）

2008年に木更津高専情報工学科を卒業、学生時代からいくつかのベンチャーでプログラマーとして働く。
昨年、英語学習のために8ヶ月間フィリピンへ英語留学。現在は、Lang-8 の開発に勤しむ。

▶▶ 染田 貴志（そめだ たかし）

株式会社ヌーラボにて、自社製品 Backlog や Cacoo の開発、運用、サポートにたずさわる。
ソフトウェアとロックと家族を愛する開発者。 ブログ なども時々綴ったり。
