そば小屋でちょっと早い年越し蕎麦オフ！

title そば小屋でちょっと早い年越し蕎麦オフ！
catch そば喰おうぜー
description

そば小屋さんのご好意で、酒とか料理とか持ち込み自由で場所を貸していただけることになりました。
だけどせっかくの機会だから、そば小屋さんの蕎麦も食べたいので年越し蕎麦オフにします。
その他の料理や酒は別に持ち込みます。
会費は当日きれいにワリカンでよろしくお願いします。

写真は「つけ鴨せいろそば」（ニガさん撮影）

開催日は23日。
クリスマス？そんなの関係ねえぜ！日本人なら七面鳥より鴨南蛮だろ！

そんなわけで基本的にお子様を除く人数分「つけ鴨せいろ」を事前に注文しておこうと思います。

蕎麦不要な人はここのコメント欄に書きこんでください。

みんなでちょっと早い年越し蕎麦喰おうれー。

駅から遠いけど交通手段は、長岡駅大手口１３番線よりエコトピア寿行きのバスの終点まで。

宝町バス停に行く予定でしたが、バス停からそば小屋さんまで遠いのでエコトピア寿バス停に変更しました。

バス時刻等の詳細は↓

長岡駅大手口（駅前）１３番線　

１４：１０分長岡駅発～立川病院～精神医療センター～１４：３０分終点・エコトピア寿着

帰りはエコトピア寿バス停を１８：４０発の便が最終になります。

バス時刻表（PDFファイル）
http://www.echigo-kotsu.co.jp/contents/diagram/rosen/01nagaoka/west/1301.pdf

念の為、宝町バス停時刻表
http://www.echigo-kotsu.co.jp/contents/diagram/rosen/01nagaoka/west/1302.pdf

バスで向かう人は１４時ちょうどに長岡駅前バスロータリー横の地下道入り口に集合しようと思いますので、ＤＭで僕のところまでバスで行く旨連絡ください。

車で行かれる方はお店の前は３台くらいしか止められませんが、近くのエコトピア寿の駐車場はたぶん止めても大丈夫だとのことです。

蕎麦は要望のあった量喰えない人や肉喰えない人を除いて、基本的にみんな「つけ鴨せいろ」です。

その他に宅配寿司を２人前×３、ピザはとりあえず集合してからその他の持ち込み分を考慮しつつ種類と量を決めて宅配してもらうことになりました。
あんまり多く用意し過ぎても食べられないので。

お酒は飲む人飲まない人がいるので、基本的にお酒飲む人は自腹で持ち込みでお願いします。
近くにコンビニもあります。

なお会費は３千円くらいの予定で、最初に徴収して余ったら返却するとのことです。

それではよろしくお願いしまーす！
