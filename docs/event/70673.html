description

SEO対策のセミナーや書籍は多数ありますが、なかなか自分のサイトに応用するのが難しいもの…。

弊社では、よくご提案の際に先に内部施策の一部やコンテンツ案を紹介し、お客様に修正のイメージをつけたり、工数の計算をしたりしてからご発注の可否など決めていただいております。

「わかりやすい」、「初めて具体的にやることがイメージできた」とお声をいただくので、無料セミナーにしてしまおうと開催を決めました。

こんな方にお勧めです。

・時々SEO対策の相談を受ける広告代理店様やウェブ制作会社様
　→自分で施策のイメージを先に作れるようになるので、契約の確度を高めたり、提案を幅を広げたりできます。

・自社サイトのSEOを担当しているが、正しく施策できているのか不安な方や何をして良いのかわからない方
　→数サイト限定ですが、その場で施策の方針や修正ポイントを指摘させていただきます。
　※希望の方は先にお知らせください。

・コンテンツが重要なことはわかったが何を書いていいのかわからない。書こうとしても書けない。

イベント概要

・定員：20名
・参加費：無料
・時間：19時～20時半まで
・内容
　・SEOの内部施策修正点の出し方。チェックシートをお渡しします。
　・コンテンツマーケティング、コンテンツSEOの考え方、記事の作り方
　・数サイト限定でその場でSEOコンサルティング
・終了後に希望者の方を募って懇親会も開催します。
