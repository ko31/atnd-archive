Community Open Day 2012 とは
コミュニティに所属している方はもちろん、未だコミュニティに所属していない方々にコミュニティの活動内容（勉強会）を披露し、興味があればそのコミュニティに参加してもらう機会を構築するオンライン/オフラインイベントです。

名古屋会場では、3コミュニティ共催で勉強会を開催します。セッションは、最新の Windows 8 開発や、Windows Phone、クラウド関連など盛りだくさん。知識は問いません。興味があれば OK です。たくさんの参加 お待ちしています！

ライトニングトーク募集中

5分のセッションを募集しています。内容は IT・開発系であれば特に問いません。
やってみたい！ という方は、@jz5 まで連絡してください。

  • @lundih 6/23開催のWindowsPhoneハッカソンの宣伝＋案内

  • @tocynet Win 8 か、Live SDKネタで。いや、Android が混じるかも？

参加費・参加資格

参加費：　500円 （社会人学生を除く学生は無料）
懇親会費：　1000円程度を予定。当日会場でハッピーアワーを行います。　
参加資格： 興味があれば誰でも歓迎です。未成年の方は保護者の方の許可を得て参加してください。
