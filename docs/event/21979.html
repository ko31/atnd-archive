JavaScript Advent Calendar 2011 (Node.js/WebSocketsコース)

12月の1日から25日まで、参加者が日替わりでブログを書く毎年恒例のイベントです。

「JavaScript Advent Calendar 2011 (Node.js/WebSocketsコース)」では、Node.js または WebSockets に関連したネタを書いてください。クライアントサイド、サーバーサイド、JS じゃないけど WebSockets 関係等です。

参加方法

  1. このイベントに参加します（先着25名まで）
  2. 先着順に、12月○日の午前0時から午後12時までの24時間の間にブログを書きます

告知方法は、ブログを書いたらここのコメント欄でお知らせ下さい。（もっと良い方法を考えるかもしれません）

参考 → 去年の JavaScript Advent Calendar

今年の JavaScript Advent Calendar は4本立て

去年は書きたい人の数に対して枠が少なかったため、今年は枠を増やしました。

  • フレームワークコース : jQuery, dojo, ExtJS, Sencha など
  • WebGL コース : WebGL や Three.js など
  • Node.js/WebSockets コース : Node.js または WebSockets に関連したネタ
  • オレ標準コース : 上記以外、次期ウェブ標準、ブラウザ限定標準、忘れ去られた標準、オレの中では標準的に使ってるテクニックなど何でもアリ

それぞれオーバーラップしてる部分もあるので、カテゴリーが曖昧な場合はまずエントリーしてみてあとは強引に押し切っちゃってください。

ゆるいイベントなので気楽に楽しみましょう。
