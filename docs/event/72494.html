###登壇者3###

■登壇者：
クラスメソッド株式会社　藤本 真司

■講演タイトル：
Amazon Elasticsearch Serviceを活用したAWSのログ活用

■講演内容概要：
AWSの各種サービスは様々なログを出力します。出力設定は非常に簡単ですが活用となると頭を悩ませます。
ログを活用するには一般的に収集、集約、監視/可視化の流れがあります。
AWS re:Invent 2015でAmazon Elasticsearch
Serviceが公開され、Elasticsearchにより集約が、Kibanaにより可視化が可能となりました。更にre:InventでリリースされたAWS
Lambda for Pythonを利用して、収集されたログをAmazon Elasticsearch
Serviceへ集約する流れを作り、ログを活用しましょう。

■ご登壇者プロフィール：
2006年4月：独立系SIベンダーに入社。Webアプリ開発、ホスティングサービスの構築/運用/保守、IaaSのサービス開発に従事。
2015年7月：クラスメソッド株式会社に入社。AWS環境の導入支援、構築を担当しています。ブログ頑張ってます。
