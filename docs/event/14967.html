第13回R勉強会＠東京（#TokyoR）を開催します！

ATNDは、連絡する手段がないため、グループへの参加をお願いします！

  • http://groups.google.com/group/r-study-tokyo

当日、ISMSと怪我や病気による緊急対応のため、ニフティさんに渡す名刺と名札用の名刺2枚を用意していただきたいとのことです！
よろしくお願いします！

【追記】
今回、 Rパッケージガイドブック の著者の皆さんをお呼びして、サイン会を実施することになりました！！

内容

  1. 自己紹介タイム（30分）
    • 1人1分くらいで簡単に
  2. @isseing333 : 線形判別分析（30分）
    • Rによるデータサイエンス第Ⅱ部 第9章
  3. @kos59125: R と .NET Framework（30分）
    • .NET Framework 環境から R を呼び出すライブラリ R.NET の紹介
  4. @sfchaos : Rで学ぶロバスト推定（30分）
  5. 酒井 : Rで地理空間分析　地理データのとりあつかい（30分）
  6. @yokkuns : YjdnJlpの紹介とか（20分）
    • YjdnJlpというパッケージを公開したのでその紹介
    • 間に合えば、もう一個パッケージ公開してそれの紹介もしたい
  7. @mokjpn : Rパッケージガイドブックの紹介
    • Rパッケージガイドブック の紹介
  8. Rパッケージガイドブック著者サイン会
