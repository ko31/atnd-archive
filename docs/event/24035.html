大阪『ぷよぷよ！！』練習オフ会（2012年01月22日）

description

セガ公式イベント「ぷよぷよフェスタ2012/ぷよぷよチャンピオンシップ」での優勝を目指して、大阪でぷよぷよ練習オフ会を開催します。

…という趣旨ですが、参加するのに実力もぷよぷよチャンピオンシップに出場するかどうかも問いません。
ぷよぷよが上手くなりたい人なら誰でもOKです。

シリーズ最新作『ぷよぷよ！！』のDS版とWii版での対戦がメインになりますが、好きに遊んでもらって構いません。
途中参加、途中退室可能です。

参加費：1500円 （1050円になるかも。小学生未満は無料）
※ジャンカラのフリータイム料金です。
ジャンカラ阪急東通3号店

持参品：持っていればDS、3DS、PSPの本体と『ぷよぷよ！！』ソフト、充電ケーブルなど （なくても大丈夫です）
※Wiiを会場のモニターに接続して遊べるので、Wiiの本体とソフトはイノセンスが用意します。

予定している内容
DSのワイヤレス対戦、Wiiのオフライン対戦の感覚に慣れる。

Wiiのクラシックコントローラーに慣れる。

ぷよぷよフィーバーがうまい人ふたりにWii版で対戦してもらって、イノセンスが解説（参加者のレベル次第）。

「ペアでぷよぷよ」の練習をしたり、出場ペアの相手を見つけたりする。
開催日の1/22がエントリー締切日です。

くまちょむ著『ぷよぷよ通を学ぶための本』鑑賞会。

その他
19時に終了した後は二次会として希望者で食事に行きましょう。

当イベントに関する問い合わせはTwitterでイノセンス @inosendo までお願いします。
もし二次会だけ参加したい方がいれば、このページでの正式な参加登録ではなく、コメント欄に書くかTwitterで個別に連絡をください。

会場は大阪ぷよぷよ通対戦会が行われているアミュージアム茶屋町から徒歩10分くらいの場所です。

主催：ぷよぷよポータル！！ （追記：開催レポート）
