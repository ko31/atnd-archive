description

２００８年１月にスタートしたベンチャーマッチング交流会。
これまでに僅か２年半で東京・大阪・神戸・名古屋・浜松・横浜・
大宮・宇都宮・高崎・沖縄で延べ６０回以上の開催、
そして５,０００社以上の経営者の皆様にご参加頂き、
名実ともに日本最大級の交流会として成長して参りました。

また本交流会は、新聞・雑誌等の各種メディアでも多数取り上げられ、
今もっとも注目を浴びている交流会と言っても過言ではございません。

更に今年は、「この勢いを全国に広めたい！」
そして何より「全国のベンチャー企業を元気にするんだ！」
と言った熱い使命感に駆られ、全国展開を驀進中でございます。

そして今年３月、遂に「ベンチャーマッチング交流会」の埼玉県支部を立ち上げ、
大宮で初開催を果たしました。
お蔭様で毎回、ご参加頂いた皆様には大変ご好評を頂いております。

日本で９番目に人口も多く、政令指定都市であり、業務核都市にも指定されており、
新幹線をはじめ周辺各地の鉄道路線をも集結しているさいたま市（大宮）。

ベンチャーマッチング交流会は、熱いこの街の更なる発展を目指し、
この度、第６１回目の開催を大宮の地で開催させて頂く運びとなりました。

昨今は、不況の煽りで暗いニュースばかり目立ちますが、
これまでに実に５,０００社以上の皆様と交流を深めてきた
「ベンチャーマッチング交流会」が商圏の拡大・代理店開拓等、
埼玉県の不況に歯止をかけるきっかけとなるに違いありません。

是非、共に大宮を！埼玉を！盛り上げて参りましょう！

皆様のご参加を心よりお待ち申し上げております。
