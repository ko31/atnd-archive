★★★当日の進行について★★★
当日は以下の流れで会を進行します。人数多ければ2グループに別れてディスカッションを行い、途中でグループを入れ替えます。
ディスカッションの中では皆さんに記事を1つずつ紹介していただきます。

1. 自己紹介
2. 第1グループでのディスカッション
3. 第2グループでのディスカッション
4. 振り返り

●あの「有名企業」で働いてみたら
ニュースでよく耳にする会社や、就職活動で気になった外資系企業。
そんな有名企業の社風には、いったいどんな特徴があるのか。
実際に働いた人だけが知っている、気になる“会社の内側”を見てみよう。

●君は「スーパーヒューマン」を見たか
人間は数百万年の年月をかけて、驚くほど身体能力を発達させてきた。
さらにいま、技術の進歩が私たちを「限界」の先へと飛躍させる。

●トヨタが走る「復活への道」
大量のリコール問題、災害による供給網の麻痺、そして記録的な円高……。
茨の道を歩んできたトヨタだが、ついに回復の兆しが見え始めた。
