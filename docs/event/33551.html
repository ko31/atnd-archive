第2部　溶媒浸透法を用いたPETフィルムにおける耐熱性・熱収縮付与
【15:15-16:30】

【講演主旨】
　PETボトルは一般にはプリフォーム成形とブロー成形の2つの手順で製造される。
　基本的に結晶性プラスチックであるPETは成形中の伸張や加温により結晶化が進行し、容器として必要な剛性が付与される。すなわち、伸張と同時に100℃以上の温度に保持することで生じる結晶化により、上述の100℃に近い内容液充填時に問題となった大きな熱収縮に対する耐熱性が発現される。
　これまで良く行われてきた方法では、ブロー成形の最終段階において金型内で100℃以上の温度に保持して結晶化を促進させ、耐熱性を与えるという手法がとられてきた。しかし、この方法では、熱処理に必要な時間の分だけ製造効率が落ちるというデメリットがある。　他には、多段ブローと呼ばれる方法も開発実用化されている。ブロー成形した後に熱収縮させ、この成形と熱収縮を繰り返すことによって、内容液充填時の耐熱性を持たせている。この方法は大量生産に対応できるが、複雑な成形手順であり、専門的な工程が必要である。
　このようにPETフィルムの伸張、成形による結晶性や機械特性の改善に関する研究は多く発表されているが、様々のメリット、デメリットを有する。
　我々は、これら従来の方法とは違った様々な新しいやり方でPET ボトルの結晶性を高める方法を模索している。その一つの方法として、プリフォームに化学物質を浸透させて結晶化度を高め、そのプリフォームを唯一度だけ延伸、ブロー成形するという、極めて単純な方法を見出した。しかも、ガラス転移温度よりごく僅かに高い80℃程度の比較的低い温度での成形お可能性を検討してきている。このようにして成形したPET ボトルに大きな耐熱性を付与させることを目標としている。
　本研究では、これまでに我々が行ってきたプロセスを基礎科学の立場から詳細に眺め、より高い耐熱性を獲得することを目指して溶媒浸漬、伸張そして熱処理と様々の処理を施したPET フィルムの結晶化挙動に関する
詳細を赤外スペクトルにおける結晶性バンドを利用して明らかにしたい。本講座ではこのあたりの具体的手法について講演する。
