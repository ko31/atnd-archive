description

ゼロスタートでも使っているAmazon EC2 、
「興味がある」「使ってみたい」という方は多いはず！

出来るだけ多くの方に体験してもらうため開催した
過去2回のAmazon EC2 ハンズオン。
非常に多くの方にご参加いただき、通算200台近くの
EC2のサーバの立ちあげることができました！

ご好評につき、12月3日(月)は、第3弾を開催いたします！

講師やサポーターに質問をしながら、
そのまますぐに使える実践的なVPC環境を構築してみましょう！

なお、今回は一歩先ゆくAWSの使い方、
「VPCトレーニング」の内容をベースに、進めていきます。
AmazonVirtual Private Cloud-Amazon VPCの説明

サーバやインフラの詳しいことが分からなくても、
AWSのエバンジェリスト堀内康弘氏のレクチャーと
サポートメンバーがわかりやすく解説いたします。

この機会に、ぜひ体験してみてください！

■ 対象

・インフラに興味のある方、理解を深めたい方
・Amazon EC2を便利にご利用したい方
・EC2について質問したいことがある方
・ユーザー同士での交流をしたい方
