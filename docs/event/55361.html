description

確定拠出年金は大企業では普通に導入されていますが「選択制…」はあまり知られていません。

この「選択制…」とは

①国が定めた企業年金制度で給料の一部を積立にまわします。
②役員だけや従業員の希望者だけで加入できます。
③積立した金額は全額が社会保険料・所得税・住民税の計算対象外なので、個人は約３０％もお得に積立でき、会社にとっては社会保険料削減になります。

こんなに有利な積立制度なのに、実は中小企業には全くといっていいほど広まっていません。
会社が新たな原資なしで導入できるこの「選択制確定拠出年金」を、中小企業の経営者にこそ知って
いただきたいと思い、今回具体例を用いてわかりやくすくご紹介いたします。

【具体例の一部】
１、月給100万円の役員の場合、社会保険料・所得税・住民税が合わせて年間約30万円減ります。
（給与に比例して減額できる金額は変わります）

２、社員が10人の場合、会社の社会保険料負担が平均で年間約36万円減ります。
（社員数、給与によって変動します。社員も社会保険料負担は減ります。）
３、月給60万円の幹部社員なら、社保・所得税・住民税が年間約25万円削減可能。
★他にも制度上、多数のメリットがありますが、紙面の都合上セミナーにてお話します。

セミナーでは、会社が節約できた社会保険料を⇒役員個人へ、従来の半分の税金で移転させる方法も
お伝えしてますので、できる限り代表者のご出席をお勧めします。
