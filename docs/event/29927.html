INTRODUCTION

13日の金曜日は、JSONの日です。（断言）
参考： 13日の金曜日 JSON – Google 検索

というわけで、JSONに関する勉強会をしましょう。（主催の都合で新潟です）

※今回を逃すと、2013年9月13日まで13日の金曜日がありません

概要

Webサーバーと通信する際のデータ形式として過去のXMLに代わり、さらにMongoDBなどの普及によって単なる通信用データ形式を超えた立ち位置を獲得しているJSON。
Webを扱うエンジニアならば避けては通れないJSONについて、便利な使い方やハマりやすいポイントなどについての情報共有を行いましょう！
