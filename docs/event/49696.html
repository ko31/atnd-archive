ニコニコ大百科を色んな形で愛する人たちで大勢集まってワイワイやろう！

どなたでもご参加いただけます。
ひゃくらぼメンバーももちろんいます。

  • 「あの記事ってこの方が！？」

などなど、よそでは話題にも上がらないような話で大盛り上がりできちゃう貴重なチャンス！！

スマートに自分の記事等を自薦すれば「ほめる」も倍増！？


会費は3,500円です。


皆様のご参加を心よりお待ちしております。
