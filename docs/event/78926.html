OS自作をやっている皆さんで集まりましょう！

このイベントの対象は次のような方々です。

  • 自作OSに興味があり、OS関係の本を読んでいる
    • 「30日でできる！OS自作入門」を含め、OSの本ならなんでも可
    • 今日から読み始めてもOKです。
  • 既に自分のOSを作り始めている
  • 自作OS入門者にいろいろ教えたい上級者

※既存OS（Linux、BSD等）の改良や、組み込み系をやりたい人でもOKです！ただし、企画メンバーがそちら方面に詳しくないので、サポートは出来ないかもしれません。

当日は「OS自作入門」を読んだり、自分のOSを作ったり、分からない部分を他の人に質問したり、自作OSに関する事なら何をやってもOKです！

強制ではありませんが、最後に各自の当日の活動内容をLTで報告する時間を設けます。奮ってご参加ください。

当日のスケジュール

当日は東京日本橋タワー 27F に直接お越しください。入館証の取得に必要なバーコードは後ほど皆さんにメッセージで渡します。

注意

7F から 27F につながるエレベータの入り口がロックされています。送付されたバーコードを使って入館証を取得し、ゲートで読み込ませてください。 遅れる方は Twitter @uchan_nos まで事前にご連絡くださると幸いです。

電源および無線 LAN があります。電源タップをお持ちの方はご協力をお願いします。 無線 LAN はセキュリティのために特定ポートのフィルタがかかっています。 ssh 接続などしたい方はモバイルルータ等をご準備ください。
