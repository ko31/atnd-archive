※申し込みはKSKアナリティクス問い合わせフォームまで

計画・分析・レポート – Excelによるデータ分析ツール　Jedoxの無料体験セミナー

開催日程

【東京】2015年7月2日（木）13:30-16:30

開催場所

株式会社KSKアナリティクス 東京オフィス　＜東京都中央区銀座8-14-11 ワイ・エヌ銀座ビル5F＞

参加対象者

ビジネスアナリティクス製品を導入予定、利用、比較検討されている企業のシステム担当者、ビジネスユーザー様。

内容

Jedox 体験ハンズオン（入門編）として、Jedoxの概要をご説明します。Jedox for Excelを実際に使ってレポートを作成しながら、デモや操作を体験します。

費用

無料

定員

4名

お申し込み

お問合わせフォームより「（ご希望日付）Jedoxハンズオンセミナー申し込み」と記載いただき送信ください。
お問い合わせフォーム：https://ksk-anl.octaba.org/mail/contact.html

注意事項

１．参加者よりWindowsPCをご用意ください。（ノートPC メモリ：最低2GB以上、HDD：最低5GB以上の空き容量 / OS：Windows 7 以降 / Office:Excel 2007以降）
２．テキストやデータ等はトレーニング会場でご提供します。
３．Jedox Premium体験版を事前にインストールしてください。（体験版の入手方法及びインストール方法などは別途ご連絡いたします）
