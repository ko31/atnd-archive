数学をもういちど学んで、数学検定をとろう！

title 数学をもういちど学んで、数学検定をとろう！
catch 〜足し算から微分・積分まで〜
description

学校で習った数学の再入門講座をおこないます。内容には、算数の簡単な復習から始まり、中学校の数学、高校での数学までが含まれます。もう忘れてしまったけど、楽しかった数学をこの機会にもう一度学んでみたい、学生時代に苦い思いをした数学をこんどこそ理解したい、そんな方々におススメの講座です。

また、受講者の中で希望される方を募って、来年三月に、日本数学検定教会主催で行われる数学検定を団体受検します。（数学検定での目標級は２級～準１級を想定しています。）

期待できる効果
１）　数学を学びなおすことで、数学的思考が身につく
２）　理科系の解説書が読めるくらいの、数学の基礎が身につく
３）　難問を解決することで、頭がスッキリする
４）　数学検定を取得することで、進学・就職で有利になる
