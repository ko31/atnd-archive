description

普段の生活をしていて、「したいけどできていないこと」ってありませんか？

私は、溜まっているタスクを処理すること。（とくにまとめて時間を取ってやりたいこと）をこのGWにやってしまおうと思っています。

ただ、１人でやってもいいのですが、誰かと一緒ならより、強制力が、働くかなと思っています。

特に連休中にやることがない人は、この機会を使って、半ば強制的に"時間のかかる"ことを一緒にやりませんか？

　◯将来に向けた人生設計
　◯ブログの立ち上げ
　◯趣味を広げる勉強
　◯夢を語ること
　◯今年の振り返り
　◯会社の仕事以外での企画書作成
　◯NPOやNGOのための活動

などなど、仕事以外の活動で溜まっていることがあると思います。溜まっていることを片付けて、心身ともにスッキリするGWにしてみませんか？
