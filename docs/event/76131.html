待望の第一期プロアスリートネイルトレーナー養成講座が開講いたします。

第一期プロアスリートネイルトレーナー養成講座

開催日：　５月１０日～６月２８日（毎週火曜日全８回の集中講座）
開催場所：東京都渋谷区渋谷エクラート渋谷ビル　５F
受講時間：10:00-13:00 14:00-17:00 / 13:00-14:00 ランチ休憩

アスリートネイルトレーナーとは、アスリートの爪を専門にケアする新たな職業です。

アスリートの過酷なトレーニングによって弱った爪のケアやメンテナンスについて総合的に学べる養成講座で、アスリートネイルの知識・技術は、アスリートを怪我やトラブルから守り、パフォーマンスアップにつながるアドバイスができるようになります。
アスリートの「爪」「医学」「食」「メンタル」を総合的にサポートできるよう短期間で集中して学んでいただきます。

「プロアスリートネイルトレーナー養成講座」はプロとして活躍いただくための講座です。アスリートネイルトレーナーになりたい方はどなたでも受講可能です。

2020年東京オリンピックに向けて、一緒にアスリートをサポートして行きましょう！！
