description

芸術家が集まるひとつの街。そんな街、ご存知ですか？京都の西の山の頂上にそんな街が存在します。
陶芸・絵画・彫刻・木工・金工・硝子・染織・和紙。
伝統工芸作家はもちろん、現代美術家など様々な分野の作家が集まっています。
そんな不思議で楽しい街へGWに遊びにきませんか。参加申し込み不要、誰でも参加可能！
日時は4月3日、4日の2日間で10時から17時まで。
見るも良し、触れるも良し、体験するも良し、食べるも良し。なんと会場は街全体！
ところどころにアーティスティックな看板や椅子、旗などもあり、歩いてるだけで楽しめます。
また、山の頂上ということで、空気がとっても澄んでいて、リラックスした気持ちにもなれます。
普段街中の排気ガスなどの空気に嫌気がさしている方、リフレッシュになりますよ＾＾
体験はしゃぼん玉に身体がすっぽり入る子供に大人気の体験、そして自分だけのオリジナル器が作れる絵付け体験や、楽焼体験、そして紙すき体験も。楽焼体験はその日に持って帰れます。
駐車場あり、送迎バスあり！自然に触れて、アートに触れる。子供から大人まで、老若男女誰でも楽しめる最高に粋なアートフェスティバルへ！
ぜひぜひお越しください。西山高原アトリエ村一同心よりお待ちしております！
詳細は写真、インスタグラム、ホームページにて。
