description

AS/400(IBM i)やz/OSのDB2のデータ連携や移行でお悩みの方必見です。
本セミナーでは以下の課題を解決するための最適なツール「DB2 Connectivity」と「DBMoto」についてご紹介します。

・DB2のデータをWindowsやJavaのアプリケーションから参照・更新したい
・DB2のデータをOracleやSQL Server等、異種DBへ移行・連携したい
・DB2のデータをAWS RDS等クラウドへ移行・連携したい
・DB2のデータを災害対策(BCP)したい

AS/400とz/OSは文字コードがEBCDICと特殊ですが、DBMotoでは内部ですべてUnicodeへ変換してレプリケーションしますので文字化けの心配は不要です。

●DB2 Connectivityとは
DB2 Connectivity DB2 ConnectivityはAS/400やz/OSを含むIBM DB2への接続ドライバです。ODBC/OLE DB/.NET/JDBCの各種ドライバを提供しており、WindowsのアプリケーションやJavaからプログラム経由で、またExcelなどのODBC対応アプリケーションから直接、DB2のデータを参照・更新することができます。

●DBMotoとは
DBMoto DBMotoは異業種データベース間のリアルタイムなレプリケーションを可能とし、AS/400(IBM i)やz/OSのDB2をはじめ、DB2 LUW, Oracle, SQL Server, MySQL, PostgreSQL, Amazon Aurora, Amazon Redshift, Azure SQL Database等メジャーなDBをサポートしています。リアルタイムなバックアップ、データ連携、災害対策、移行などの課題を簡単に解決できます。
