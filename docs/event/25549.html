description

3月4日に、神戸市立婦人会館にてSphinxに興味を持ち始めた方を対象とした勉強会を開きます。

昨年12月の「Sphinx朝会@神戸」「Sphinx朝会@十三」に続いて3回目の開催です。

内容は、前回、前々回行った「Sphinx朝会」と基本的には同じで、Sphinxを使ったドキュメント作成のスタートラインと言える、環境の設定と実際の出力を、演習問題をこなす形で体験するというものです。

ハンズオン形式で進めていきます。

実施内容

  • Sphinxのインストールについて
  • はじめての’make html’
  • 演習問題を使ったドキュメント作成体験
  • 他(S6テーマを使ったプレゼン資料の作成についてとか)

なお、実際に体験していただきたいのでノートPCを持参されるとよろしいようです。またその際は、一通りインストールされた環境があるほうが演習問題をこなす時間が増えてよろしいかと思います。
Sphinxをはじめよう を参照して環境を整えちゃいましょう。

もちろん、インストールで躓いてたらその場で解消できるかもしれませんね。

カバンに余裕があるなら、電源タップを入れておいていただける と何かとはかどります。
