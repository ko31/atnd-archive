=トークイベントです=

2000年代突入以降、CDセールスの減少やフェスの増加・定着……などにより、
音楽を素肌で感じる場としてだけでなく、新たなアーティスト／サウンドとの邂逅の場として、
音楽シーンにおいて以前にも増して重要な位置を占めるようになったもの、それは｢ライブ｣。

その｢ライブ｣を、ステージから最も近い位置で見続けてきた ５人のフォトグラファーにとって、
＜00年代ロック・シーン＞とは果たしてどのようなものであったのか？
自らの作品と、豊富な経験を通して振り返ってもらう。
さらに第二部では、雑誌編集者を交えてクロストークを展開。
＜00年代ロック・シーン＞について、より一層掘り下げ語ってもらう予定。

第一部 from Photographer’s Eyes / Photo Slideshow & Talk
ゲスト：畔柳ユキ、 久保憲司、 佐藤哲郎、 三島タカユキ、 半田安政
第二部 Photographer × Editor / Cross Talk
ゲスト：金光裕史（『音楽と人』編集長） + フォトグラファー
