description

これから留学やワーキングホリデー に行ってみたい方、
予定している方、また英会話の世界に浸ってみたい方を
対象に英会話セミナーを開催します。

レッスンでは例えば
空港に着いてからすぐに使えるフレーズをはじめ、
トラブルが起きた時に役立つフレーズなど事前に
学んでおくと現地で困ることが無くなるであろう
内容を厳選してご紹介。　

　
そして、参加期間中は生徒同士でも日本語の使用は
原則として禁止。

常に英会話ができる環境に浸れるので
国内にいながら海外の雰囲気をそのまま体験でき、
スムースに現地の生活に入ることができるでしょう。
