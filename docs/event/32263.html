Akasaka.scala とは？

隔週で開催しているハンズオン形式の勉強会です。初心者の方もお気軽にご参加ください。

http://akskscala.github.com/

今回の予定

拙作で恐縮ですが xsbt-scalag-plugin という Rails Generator にインスパイアされた sbt プラグインをつくりはじめました。

このプラグインは本体に手を入れなくても project 配下に sbt.Plugin を extends したちょっとした Scala のコードを書くだけで拡張することができます。

今回はこのプラグインを使って、いろんなものを生成するコマンドをつくってみるというテーマにしたいと思います。ぱっと思いつくところだと Play20 の scaffold や Twitter Bootstrap のひな形を指定されたパスにつくったりといったあたりでしょうか。

もちろん xsbt-scalag-plugin 以外の sbt プラグインをテーマに参加されるのも OK ですので、お気軽にどうぞ。
