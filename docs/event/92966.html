『90 分でマルわかり！SDN / SD-WAN 無料セミナー』
”Cisco ゴールドパートナー認定企業”が解りやすくご説明致します！
難易度：★★★☆☆

ネットワークエンジニアとして活躍されている方、運用・保守業務から更なるキャリアアップを目指される方を対象に
『90 分でマルわかり！SDN / SD-WAN 無料セミナー』及び会社説明を、12月6日(水)、12月9日(土)に実施致します。

本セミナーでは、『SDN/SD-WAN の基本的な概念(考え方)』から、『どのようなメリットがあるのか？』等のポイント
をご説明し、Cisco ACI を利用した『SDN/仮想化のデモ紹介』から、SDN/SD-WAN のメリットを解説いたします。
どうぞお気軽にご参加下さい！

【対象となる方】
・ネットワーク構築のご経験がある方
・ネットワーク運用業務・保守業務のご経験が1年以上ある方
・インフラエンジニアとしてキャリアアップしたい方
・プログデンスへの応募を今後検討したい方（※本セミナーは選考会ではありませんので、お気軽にご参加ください。）
　※毎回、ネットワークエンジニアとして活躍中の 20代半ばから 30代半ばの方々を中心に好評いただいております。
　※本セミナーは選考会ではありません。過去に当社セミナーに参加された方、選考プロセスにご応募いただいた方は
　　対象外とさせていただきます。

　※定員人数を超えた場合には、日程調整及び早期に申込みサイトをクローズさせていただくこともございます。
　　お早めにお申し込みください。

■詳細・申込ページ https://www.progdence.co.jp/seminar/aci_seminar_2017-12.html

　　※上記申込ページ経由での登録をもって、正式受付となりますのでご注意ください。
