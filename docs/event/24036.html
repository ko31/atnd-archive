やること

翻訳プロジェクト合同で薄い本を作って、2012 年の夏コミで売る

なぜ夏コミ?

期限を切らないと、いつまでたってもできないと思うから。
(OSC 締切駆動開発と同じ?)

配付方法など

  • 売れるネタとも思えないので、少部数コピー本
  • サークル参加する知合いなどに委託
  • コピー代実費程度または無料配布
  • 別に web で公開しちゃったり
  • 執筆者が自分の記事を再利用するのは自由に

ということで謝礼とかはできませんが、それでも書いてくれるひと募集 (委託を受けてくれるひとも)。
