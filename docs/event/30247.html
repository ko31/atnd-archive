講演概要

はじめよう JavaFX 2.x

  • 出演
    • a_know氏( @a_know )
  • 概要
    • 最近、密かに盛り上がりを見せているJava GUI開発プラットフォーム「JavaFX」。
      今回、「自作SwingアプリケーションをJavaFX 2.xで1から作りなおす」という経験をしたこともあり「JavaFXとは何か」というところから、ごく簡単なJavaFXアプリケーションを作る流れまでを、簡単にご紹介します。

GlassFishの監視ツール、LightFishの紹介

  • 出演
    • 岡山Javaユーザ会　吉田貴文氏( @zephiransas )
  • 概要
    つい先日リリースされたばかりのGlassFish監視ツール、LightFishについて。
    LightFishの導入方法と、機能の説明、またその仕組などをご紹介します。

JSF (JavaServer Faces) 2.0 の詳細について

  • 出演
    • 日本Oracle株式会社　シニアJavaエバンジェリスト　寺田佳央氏( @yoshioterada )
  • 概要
    JSF (JavaServer Faces) は Web のビュー・テクノロジーとして Java EE の
    標準技術に取り込まれました。JSF は JSP (JavaServer Pages) から Facelets
    による開発へ移行し、以前に比べ開発効率が格段に向上しました。
    本セッションでは JSF による開発の詳細をご紹介すると共に、
    JSF 2.0 で新たに追加された機能について詳しく紹介します。

ジャバ語の次はスカラ語

  • 出演
    • 天領倉敷Scala/岡山Javaユーザ会　角田裕樹氏( @razon )
  • 概要
    Scalaは、オブジェクト指向と関数型の両面をもつマルチパラダイムのプログラミング言語です。
    JVM上で動作することから、近年Javaプログラマを中心に注目を集めてきています。
    本セッションでは、Scalaの概要、特徴的な関数型言語としての機能、取り巻く環境など、
    様々な面からScalaの世界をご紹介します。
