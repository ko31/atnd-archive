現状日本のスマートフォン業界は
iPhone / Android携帯、この二つに分断されつつあり、
アプリ開発者は、両方のプラットフォームに提供することが余儀なくされようとしています。

その問題点を解決する一つの手法として、
現在注目を集っているのが、Webアプリです。

Webアプリとは言葉の通り、Web(ブラウザ)上で動作するアプリ(のようなもの)です。

Web(ブラウザ)上で動作するため、iPhone / Android 間もシームレスとなり
開発コストを大幅に下がる事もできますし、
またWeb開発で得た知識を利用することでアプリを開発できるため、
より多くのエンジニア/デザイナーが参入しやすい利点もあります。

そこで今回ウェブ会では、
web制作者のための、スマートフォン向けwebアプリ開発セミナー
と題し、位置情報を用いたチェックインアプリの作成を一つのゴールとし、開発手法や開発における注意点、メリット／デメリットなどを解説致します。
