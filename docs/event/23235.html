【学生対象】
みなさん。
カメラを買ったものの
なかなか日常的に写真を撮る習慣がないですよね？

そんなあなた方のために第1回photographic work shopを開催いたします。

今回は
3人一組になりそれぞれテーマを決めてもらいそれを基にそれぞれの場所に写真を撮りにいくというもの。
またその後全員集まってもらい、写真の共有およびフィードバック、
photoshopなどによる写真加工・編集などを開催する予定です。

自分で撮った写真を他の人と共有して、フィードバックをもらったり、
他の人の写真を見てこういう撮り方もあるんだという発見があればと考えております。

今回は第1回ということで外部の方はお呼びせずに学生10名だけで
自由にカメラのフィルターを通して表現してもらい、
写真および編集の技術を少しでもつけてもらえたらと思い、
企画いたしました。

こんな方大歓迎です。
・一眼レフを買ったもののより詳しい使い方などがわからず困ってる方。
・また一眼レフをもってるけど、なかなか撮る機会がない方。
・これから一眼レフを買って少し力をいれて写真に挑戦したい方。
・photoshopなどで写真の編集・加工をしてみたい方。
・一眼レフなどはもっていないけど写真に興味ある方。
・写真を通して自分を表現したい方。
などなど

なお今回は
定員を12名にさせていただく関係で、
定員に達し次第締め切らせていただきますので、予めご了承ください。

ではみなさまのご参加をお待ちしております。
