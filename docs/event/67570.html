ETロボコンとは

若手エンジニア、学生に技術教育の機会を提供します！
組込みソフトウェア技術教育をテーマとするロボットコンテスト。

ETロボコン

5年後、15年後に世界をリードするエンジニアの育成を目指しています。
若手、初級エンジニア、および中級エンジニア向けに、分析・設計
モデリング開発、製品サービスの企画開発にチャンレジする機会を
提供するユニークなロボットコンテスト。

LEGO MINDSTORMSによるロボットレースで競う「デベロッパ部門」、
製品・サービス企画で競う「イノベータ部門」があります。

参加チームはそれぞれの部門でMDD( Model Driven Development )
に取り組みます。当日のレース／パフォーマンスに加えて、事前に
「モデル分析シート」を提出、総合得点を競います。
