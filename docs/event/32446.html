健康"感"察　〜02 アイデアを形にしよう〜

title 健康"感"察　〜02 アイデアを形にしよう〜
catch
description

健康に気をつかっているつもりでも、
それを自ら進んで管理したり、つきあっていくことは、
中々難しいことではありませんか…？

本ワークショップは
“Inclusive Design”という、個人を追求することを特徴とし、
人々が気付いていない、しかし共感できるニーズを発見するための
デザイン手法を使い、実際に形にしていきます。

その人が感じる"健康"を大事にしたもの、
楽しみながら健康管理が続けられるものを
一緒につくってみませんか？

【内容】
①発見しよう　(9月1日)
健康ってなんだろう・・・？
普段の生活の中で何気なく見逃してしまっている、
「実は問題であるもの」を発見します

②アイデアを形にしよう(９月３０日)
①で発見された“もんだい”
その解決のために「アイデア」を考え、
実際に「形」にしてみます！

③深めよう(１０月以降)
②で形にしたものを
その可能性と関わりについて、
普段から医療や健康に携わっている人と深めます

※ワークショップは３回に分かれていますが、
もちろん１回のみの参加でも大歓迎です！！
