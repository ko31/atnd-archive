description

環境問題、高齢化、地域の衰退、貧困、自殺などの社会問題、エネルギー・資源の枯渇、食品の安全や食料自給率の低下・・・。


生活をとりまくあらゆる問題があるなかで、日本各地・世界各地には、これらの社会的な課題に立ち向かい、よりよい未来を、国や誰かに頼るのではなく自分たちの手でつくっていこうと活動する人々がいます。アミタは、このような未来を創造するために立ち上がろうとしている人々と共に、発信型のイベント、映像トークセッション「未来の設計者たち」を開催いたします。


■イベント開催の目的
本イベントの開催によって、社会的な課題の解決に向けすでに活動を展開している人々や、これから何か行動したいと考えている人々と共に社会課題の現状を共有し、それぞれの力を結集して向かうべき未来の社会をデザインしていきます。


■イベントの様子はwebを通じて発信
Twitter®（新しいウィンドウで開きます）やUSTREAM®（新しいウィンドウで開きます）といったwebサービスを利用して、未来を描く第一線で活動する方々を交えたセッションの様子を映像、音声、テキストにより中継発信します。当日会場にお越しいただけない方でも、web上にて参加が可能です。東京・京都の各会場及びweb参加者から出された意見や提言はリアルタイムでセッションの内容に反映し、多くの方々にご参加いただき活発なセッションを行います。


■開催概要
【テーマ】
　・守るべき未来の象徴としての、「森林」
　・自分たちの手で未来をつくる、「社会起業」
　・それらの活動を支える、「市民金融」


【対象】

　・未来を自分たちの手でつくりたいと考える方
　・森林、社会起業、市民金融といったテーマに興味のある方
【参加費】無料

【プログラム】
①Disconnected World　スライド上映
アミタ（株）主催の写真投稿キャンペーン「Disconnected World―関係性を失った世界」に投稿された写真をスライド上映し、社会課題の「現場」を共有します。


②映像トークセッション
「森林」「社会起業」「市民金融」3つのテーマにおけるそれぞれの活動を、映像上映を通じて紹介し、ゲストと一般参加者が共に未来を自分たちの手で描くには今何をすべきかを話し合います。


③座談会・交流会
それぞれのテーマごとにブースに分かれ、交流会を行います。
