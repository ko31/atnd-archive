２．OpenBuildService道場 in 大阪

OpenBuildService(OBS)とは、Linuxディストリビューションパッケージを開発するためのWebアプリケーションであり、openSUSEを中心に盛んに活用されています。今回はOBSを使い、オープンソースの利点を生かした「自分Linux」を作ったり、openSUSE等の開発に貢献するための、初めの一歩を体験しましょう。

OBS道場の内容

  • OpenBuildServiceってそもそも何？
  • アカウントをつくって手を動かしてみよう！

その他

  • OBS道場の参加は、Linux （ディストリビューション不問）のインストールされたノートPC をご持参ください。勿論仮想PC に Linux をインストールしても OK！
  • ノートPC には予め、OpenBuildService クライアントを こちら からインストールしておくと当日楽ちんです。
  • OBS クライアントを自宅サーバーなどにインストールして SSH でアクセスする方法もあります。
  • Web UI からも OBS が利用できます。クライアントコマンドは不要という方は非力なネットブックでも OK！
