VRDM: VRコンテンツ開発者の雑談会

RiftやGear VR、Vive、Cardboard用VRアプリの開発でお困りではありませんか? VRアプリの開発ノウハウはまだまだ発展途上。相談する相手もままならい状況だとも思います。実はみんな同じ悩みを抱えているのでは？

と言うことで、同じ悩みの方たちが集まって雑談が出来るVRアプリ開発者のための雑談会、題して「VRDM」を定期開催することにいたしました。

今回は@k0rin さんとスパイシーソフト株式会社のご好意により会場を変えて特別版として開催いたします。

NDAとかはありませんので秘密裏に開発しているようなものを持ち込まれると困りますが(汗)、気軽にご参加いただければと思います。

会場設備について

  • 机(長テーブル)と椅子
  • 電源ケーブル(この会場はあります!)
  • プロジェクタ、スクリーン
  • 無線LANはあります!

タイムテーブル

  • 入退場は自由ですので時間割は目安です。

12:00 - 18:30 開場&雑談会(出入り自由)

Studio Squint / スタジオ スクイント 住所: 東京都新宿区四谷4-7小林ビル201

注意事項

  • VRコンテンツ体験会ではありません。あくまでも雑談会です。
  • 初めて参加するときは、自作のコンテンツを持ってくると良いと思います(そこから雑談が始まる)。
  • テーマ特にありません。その時の流行りの雑談でもしましょう。
  • 唐揚げは出ません！！！！！！！！。
  • 飲食は制限しませんが汚さないように注意してください。
  • 終了後、会場を現状復帰しますのでご協力をお願いいたします (- -)(_ _)ﾍﾟｺﾘ
