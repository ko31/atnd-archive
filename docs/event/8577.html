概要

プログラムのソースコードやドキュメントのバージョン管理を行うためのシステムの1つで、「分散バージョン管理システム」として昨今人気が高まっている「Git（ギット）」について、みんなで手を動かしながら試して身につけようという勉強会です。

  • Gitは試したけど挫折した
  • Subversionを使っているけど、Gitを試したい
  • 会社でチーム導入したいが、なかなか踏み切れない
  • 1人で使っているが、チーム開発のノウハウがない
  • GitHubで他の人のリポジトリへプルリクエストを送るのが怖い

そんなあなた！
是非この勉強会に参加して、一緒にGitをモノにしましょう！

参加条件

  • 普段使っているプログラミング言語などは問いません
  • プログラマである必要もありません
  • 勉強会ではその場でGitを操作しますので、ノートパソコンを勉強会にお持ち頂く必要があります
  • 会場は無線LAN環境なので、可能であれば無線LANに接続できる環境をご用意ください
    • どうしても有線LANでしか接続できない方は事前にご相談ください

参加費

無料です。
飲み物はご持参ください。

事前準備

  • あらかじめGitHubのアカウントを作成しておいてください→ GitHub
  • 可能であれば、ノートパソコンにGit環境を準備しておいてください。
    • linux/macではれば、パッケージで手軽にインストールできます
    • Windowsの場合は、 gitextensions がよいということが前回分かったため、gitextensions での環境構築をお願いします。（当日もある程度サポートします）
    • 当日は、Git環境構築のヘルプから始めます

内容予定

  • 今回は、初級の方と中級以上の方で、それぞれにあわせたタスクを実行していく予定です
    • 中級以上の方は、複数のメンバーでGitHubを経由した共同開発（のイメージ）を行う予定です
  • まだ環境構築ができていない方は、初級チームとしてある程度サポートします
