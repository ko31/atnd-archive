description

趣旨
　シャンプー、リンス、アウトバス製品、ヘアカラーの使用感が実感できる頭髪化粧品の開発について述べる。　
　頭髪化粧品の処方開発は、どのような目的とコンセプトで、製品を作る上で損傷毛の状態を知り、損傷毛への対応方法と髪に吸着する成分を検討することである。実際のシャンプー、リンス、アウトバス製品、ヘアカラーの処方は、成分の選択と組み合わせで目的の使用感をもつ製品を作ることである。使われる成分の有用性と特徴が製品の使用感に影響を与える。具体的な成分とその機能をまとめる。

プログラム
１．頭髪化粧品
２．損傷毛の状態
３．損傷毛への対応
４．損傷毛に吸着する成分
５．シャンプーの処方の組み方
６．リンスの処方の組み方
７．頭髪用化粧品の処方の組み方
　　（ヘアミスト、セット剤、ヘアワックス等）
８．ヘアカラーの処方の組み方（Basic,HCカラー）

１５．成分の安定性

１８．防腐剤フリーの対応
