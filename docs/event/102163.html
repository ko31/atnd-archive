いとも簡単に稼げる物販ビジネス！

★株式会社オフィス・ケー代表取締役
熊谷 洋平

平成28年4月に法人化
平成28年11月に楽天市場へ出店、たった８商品（アクセサリー・オリジナルデザインTシャツ）のみで楽天市場出店初月から売り上げ１２０万越えを達成し、
楽天セミナーにて紹介された実績
平成29年4月から楽天で売り上げたノウハウを教えて行きたいと思い、物販スクールを開講
これまで15人の受講生にノウハウを教える
平成30年5月に、物販で成功したいと思っている人たちへバトンを渡したいと思い、これまで運営していた楽天などのショップを退店し、本格的に裏方に回る

★塩見 あつし
高校卒業後、サラリーマンとして働き、職を転々とし中でも営業の仕事では1日80件以上の契約を達成し社内業績ナンバーワンになる

初年度の年収は800万にも登り、今もなお右肩上がりの売り上げを更新し続けている
