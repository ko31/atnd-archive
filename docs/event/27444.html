いよいよ春が訪れました♪ウキウキ陽気(の予定)でウルトラハッピー！

さらに『スマイルプリキュア！ウルトラハッピーカーニバル』でもっとハッピー！
さらにさらに天保山でお花見したり、オールスターズ映画を見たり、
美味しいイタリアンを味わったりして、
もっとも〜っとウルトラハッピー☆になりませんか!?
　

ツアーの概要

  • 開催日時: 2012年4月8日(日)
  • 催行人数: ~8人
  • 主な行き先：大阪南港(ATC)、天保山公園、梅田ブルク7 etc.
  • 会費: ATCまでの交通費、ウルトラハッピーカーニバル入場料(650円)、昼食代(1,000~1,500円前後?)
    (チケットはヴェルナが事前に用意しますの。)

集合場所&時間

  • 集合場所: ヨドバシ梅田地下1階 BARISSIMOのまえ。
  • 集合時間: 11:00　
  • 主催者(ヴェルナ)は場所取りにつき、11:45にウルトラハッピーカーニバル現地で合流です！
    　　

花見に関して

天保山公園で13-15時、8名で申し込みしてありますの！
場所取りですが、一足先に私が赴いてキープしておきます！がんばる。

お花見の準備についてです。食べ物は基本的に、
「天保山マーケットプレースのフードコートでそれぞれ好きなものを買って、みんなで取り分けて食べる！」
というホーム(じゃないけどw)パーティスタイルです♪
(こんな感じ とか あんな感じ で、テイクアウトできるお店たくさんあります)
もちろん、ご自宅から何かお持ち頂くのもすごく素敵です★

花見に必要な物ですが、以下の物はヴェルナが持って来ますの。

  • プラスチックカップ 20個
  • 紙皿 20枚
  • 箸&フォーク&おしぼり十分数
  • ブルーシート
  • クーラーボックス極小
  • とあるお酒 1本
  • ごみ袋
  • キュアドール

　ブルーシートはとりあえずなレヴェルのものなので、携帯可能なレベルのクッションとかを持参した方がいいかもです！

…以下のものがもしあったら嬉しいかも！　

　

  • 携帯用テーブル
  • お酒、ソフトドリンク(現地調達でもいいか)
  • その他、上記で漏れてる「お花見で要りそうなもの！」
