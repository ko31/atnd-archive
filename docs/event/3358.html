こちらは『PHP 初心者 勉強会』 から分割した、PHP 中級者 勉強会です。

PHP 中級者 勉強会では、少し難易度の高い勉強を行います。

PHPフレームワーク勉強会は、勉強方法を少し変更しました。

まずCodeIgniterを勉強して、それからCakePHP→symfonyと勉強していきます。

参加費として300円かかります。

趣旨

PHP フレームワーク勉強会では、代表的なPHPフレームワークを使って、実際にWebアプリの作成を行い、それぞれのフレームワークの使い方や特徴を学びます。

理想は、すべてのフレームワークの使い方に熟知することです。

ギブ＆テイクを忘れずに。自分が得た知識は参加者全員と共有しましょう。

また、ドキュメントを残すことも忘れずに！（ソースコードが最善のドキュメントになるよう心がけましょう）

対象フレームワークは以下のとおり。

  • CakePHP
  • CodeIgniter
  • symfony
  • Zend Framework
