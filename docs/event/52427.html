「2020年、わたしたちが世界に示したい日本の姿」を映画にしよう！~映画「未来シャッター」ネットワークプロジェクト~

イベント ユーザー

title 「2020年、わたしたちが世界に示したい日本の姿」を映画にしよう！~映画「未来シャッター」ネットワークプロジェクト~
catch
description

セッションの目的

2020年、世界から訪れたいと思われる日本はどんな国？その姿を映画で可視化しましょう。
自分たちであるべき、ありたいまちの未来を描き、具体的に映画のストーリに描いていく。　　
その脚本で2014年秋から映画クランクイン！
そのためのキックオフセッションです。
今回で終了するものではなく映画製作のためのセッションを定期的に開催していきます。

対象者

まちの未来を一緒に描きたい方
まちの未来を描くストーリーを描きたい方
映画製作に関わってみたい方
映画に出てみたい方

詳細説明

2020年東京オリンピックが決まり、この6年間は日本の良さを世界中の人にアピールするチャンスです。東京だけでなく国を挙げて観光立国を目指しています。しかし本当に重点的になすべきは市民ひとりひとりが幸せを感じて生きられる社会を創ること。2020年のまちはどうなっている？地域は？商店街は？人はどのように働いているているのでしょうか？教育は？健康は？6年後のエネルギー、食の問題は？
ありえなさそうな　ありたい未来を語り、それを映画の脚本にして、その脚本で映画を創るプロジェクトが立ち上がりました。

Program
問いを共有する
13:00 〜 13:10 サークル（Circle）
本日のゴール
映画「未来シャッター」ネットワークプロジェクトについて説明

関係性をつくる
13:10 〜 13:30 ストーリーテリング（Storytelling）
４人組になって、一人ひとりが持っている「2020はこう変わる」「ありたい未来」を
3分づつ語り合います。
自己紹介を兼ねて、セッションのアイデアを3分ずつ語り合います。

ビジョンを共有する
13:30 〜 13:40 サークル（Circle）
一番ありえない未来を描いた人の思いを共有します。

13:50 〜 14:10 マグネットテーブル（Magnet Table）
自分が一番興味あるテーマを書いて、同じようなテーマの人と5~6名とグループを創ります。

未来シナリオを共有する
14:10 〜 14:40 ワールドカフェ（World Cafe）
10分×３
・あり得ない未来を話し合う
・それを具現化するために必要なもの
・それを実現するために自分ができること

ビジョンを洗練化する
14:40 〜 15:10 金魚鉢（Fishbowl）
各テーマの代表者が真ん中に並びグループの内容をシェア。

アクションしたいことを発見する
15:10 〜 15:30 クイックプロトタイピング（quick prototyping）
個人で自分で描きたいテーマ映画のワンシーンを思い描いてをビジュアル表現する。
それにどのように自分がコミットするかを文章で表現する。

15:40 〜 15:50 ドット投票（Dot Voting）
一番気に入ったシーンに投票。

アクションを生み出す
16:25 〜 16:50 オープンスペーステクノロジー（OST）
上位６つの代表者が自分の未来をプレゼン。
そこに興味がある人が集ってより具体的に映画に描くストーリーをあげていく。
わくわくするような　ビジョンが明確に見えるストーリーに仕上げる。

16:50 〜 17:00 サークル（Circle）
映画への応援エールを一言ずつ皆でシェアする。
