description

若手の経営者が集まり、代表者数名が最近の自社ビジネスを語り、その後懇親会をします。

業種などは特に制限してませんのでふるってご参加ください。
20、30代の経営者が30名前後集まってます。
毎回半数ほどは新規に参加する方ですので、はじめての方もお気軽にどうぞ。

申し込みの際にかんたんな自己紹介をお願いします。

本会・懇親会どちらかだけの参加も可能です。

【本会】

19:00-20:15 事業概要インターバル
事業についてお互い知りたい方も多いと思うので、
5社限定、1社10分程度で事業概要をインターバル形式で
話してもらう会をやります。
（5社集まりましたので発表者の募集は締め切りました）
