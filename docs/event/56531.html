description

どっぷりHTML5にひたってみませんか？

WEBページを作るためのHTMLは、今年「HTML5」として大きく様変わりしています。単なるサイトから、よりリッチなデザインで動きのあるWEBアプリケーションへ。そんなHTML5に一日どっぷりなワークショップ。ぜひご参加ください。

このコースは初心者のかたを対象に、HTML5のマークアップ・CSS3・JavaScriptのうち、基本中の基本「マークアップ（タグの記述）」について一緒に学ぶ６時間です。

初心者向けにHTMLの書き方から始まり、HTML 4.01での「タグ」のいろいろの紹介、そしてHTML5での新しいWebページの作り方や入力フォームのバリエーション、そしてマルチ・メディアの扱いかたなどをどっぷり体験していきます。

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

このコースは、LPI-Japan主催のHTML5プロフェッショナル認定資格（レベル１）のマークアップに関する試験対策としてご利用いただけます。

HTML5プロフェッショナル認定試験オフィシャルサイト

CSS3に関する試験対策は「どっぷりCSS3」へどうぞ

試験の概要を知りたい方は「ポイント解説セミナ」へ

※試験にかかわらず、HTML5の全体を体験したい方は「HTML5入門」へ
