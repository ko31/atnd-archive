内容

エイチーム東京オフィスにて、エイチームスタッフとの座談会を開催いたします。
会社の雰囲気を見てみたい、エイチームに興味があるなど、気軽にご参加いただける座談会です。
エイチームで活躍しているマーケティング担当者より、会社の雰囲気、お仕事内容などお話させていただきます。

どんな会社か知りたい、ちょっと興味があるといった方のご参加もお待ちしております。
持ち物不要！途中参加OK！の座談会です。
お仕事帰りなど、お気軽にお越しください。

対象となる方

エイチームに興味がある方
現在転職を考えている方
将来的にもっと自分にあった活躍できる場を探したい方
マーケティングまたはプロモーション経験のある方（ゲーム業界未経験可）
選考を受けるかどうか迷っている方

※ゲームマーケター、プロモーション担当の方を対象とした会です
※中途向けイベントのため、学生の方のご参加はお断りさせていただいております

エイチームについて
