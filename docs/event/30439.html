名古屋周辺を中心とした地域のFOSS4G（Free Open Source Software for Geospatial）ツールの勉強会です（中止となりました）

※第3回FOSS4Gツール勉強会＠名古屋ですが、発表者・参加者ともに集まらなかったため今回は中止（延期）とさせて頂きます。すでに参加登録された方がいらっしゃいましたら申し訳ありません。

この日は他にもイベントが多いとのことで、日程調整が良くなかったのも原因の一つだと思っております。楽しみにしていらしたたみなさまには申し訳なく思っております。

また秋ごろにでも、今度は事前に発表者をある程度確保した上で今回やるはずであったハンズオンを含めて企画をしたいと思っておりますのでその際はよろしくお願い致します。
