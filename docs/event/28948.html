Rails for Beginners Vol.0

ビギナー向けRails勉強会の第0回です。
来てくれる人、サポートしてくれるエンジニアなど、どういう人がどれくらい集まるかも分からないので、とりあえず実験的にやってみます。

Rails for Beginnersについて

  • 主な対象：
    • エンジニアとRailsで一緒に開発したいデザイナー
    • 学校でRubyを触ってRailsでもっとWebアプリとかを開発したい学生
    • もっとRails力を高めたい経験の浅い駆け出しエンジニア
  • やること：
    1. 環境構築テーブル：Ruby、Rails、Git、GItHub、Herokuなどを自分のPCで使えるところまで
    2. RailsNewテーブル：rails new〜rails g scaffold〜omniauthログインぐらいまで(環境構築が済んでいる人)
    3. HerokuPushテーブル：GitHub、Herokuをいろいろ使ってみる(環境構築が済んでいる人)
  • サポート対象：
    • Mac、Git、Ruby、Rails、GitHub、Heroku
      ※Windowsはサポート対象外です。
