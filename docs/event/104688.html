description

　BCP（事業継続計画）とは、企業が自然災害（豪雨災害、土砂災害、地震）や大火災など緊急事態が発生した場合、損害を最小限に抑え、事業の継続や復旧を図るための計画のことです。
　このセミナーでは、2018年に多発した大規模災害のような災害に直面した時のために企業が日頃から検討し、計画しておくべき対策について分かりやすくポイントを解説します。
　これからの防災、減災、災害対策に必要なノウハウを知ることで、会社も社員も守り、事業継続を可能にします。
　また、BCP策定ステップを理解し、実際に策定を体験することで自らの防災意識を高めるとともに企業としての取組むべき対策の明確化に役立てます。

【開催日】2019年05月21日（火） 13:00 〜 17:00 於：岡山県

【会場】
　キャリアプランニング岡山本社　セミナールーム（岡山駅東口　徒歩5分）
　岡山市北区本町6-30　第一セントラルビル2号館5階

【講師】
　日本マネジメント総合研究所合同会社　理事長　戸村智憲（とむら とものり）
　講師プロフィール： https://www.jmri.co.jp/Profile.tomura.pdf

【受講料】　金5,000円/人

【プログラム】

１．防災/減災・事業継続計画(BCP)はやわかりポイント解説＜講義＞
・天災か？人災か？わが身も会社も守り抜くポイント
・なぜBCPが必要か？BCPの３原則と陥りがちなワナ
・備蓄も対策もカネをかけるより創意工夫で乗り切る　など
・A3用紙１枚・色鉛筆３本でやるリスク管理の意思決定術
・災害対策マニュアルに入れるべき命を守る「四字熟語」
・これからのBCPに必須の「チェックリスト７項目」
・スマホ・100円均一ショップでカンタン防災術　など

２．無料でデキルＢＣＰ策定ステップ解説と策定体験・改善指導 ＜講義、ワーク＞
・無料で誰でも使えるBCPひな形に沿ってステップ解説
・その場でデキルBCP策定体験演習
・その場で聞いて学んで策定に活かす講師からの改善指導
・災害予測ハザードマップと手軽に情報収集する方法
・BCP策定がゴールでなく訓練を通じて精度を高める　など

３．質疑応答・まとめ
