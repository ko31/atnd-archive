ライブハウストゥワー＠高知 千秋楽のチケットを手にすることができた、全国の幸運なPerfumeファンの皆様★ ライブの余韻を一人酒で噛み締めるのもまた一興ですが、折角なので皆でかしゆかんぱいしませんか？

★ お店に現地集合でお願いします。22:00くらいから、「中村ゆうこ」で予約しています。予算は4～5千円くらい（未定）

★ 一応ライブ後会場外で集合しますが、メンバー確認せず、適宜移動しますのでご了承ください。
>> 集合場所：ライブハウス入り口向かって左の建物、はらみ海事事務所前あたり。（グッズ売り場等でふさがってたら、とにかく向かって左辺りで）

サブタイトルが意味不明と思われる方、どうぞお気になさらず。怪しい団体ではありません。

ん～、それにしても待ち遠しいですね、千秋楽。Twitterでネタバレ地雷を踏みまくった私に、もう怖いものはありません。
そして、Zepp東京辺りで、「あぁ、もう、早めのライブにしておけばよかった。」と思ったり思わなかったり。しかし、千秋楽は必ず神公演です！待てば海路の日和あり、残り物には福がある。（どちらも不適切）

では、高知でお会いしましょう♪

なお、居酒屋での深夜開催なので、未成年の方はごめんなさい、ご遠慮くださいませ。
