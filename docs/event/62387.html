FxOSコードリーディングミートアップ＃１４

２月のFxOSコードリーディングミートアップです。今回もゆるふわです。各自のペースでコードを読んだり話したりします。初めてでも大丈夫！！十分楽しんで頂けます。

Open Web Board の発表と Fx0 の発売、各種セミナー・ハンズオン、 バレンタインデー WoT ハッカソン とガンガン攻めている Firefox OS。そんな Firefox OS の内側に一歩踏み込みたい、色々知ってそうな人達に気軽に聞いてみたい、そんなあなたピッタリのイベントです。

何をするの？

Firefox OS のソースコードを読んだり、語らったりする集まりです。ゆるふわだから初めてでも安心。Firefox OS に興味がある人なら誰でも楽しめます。少人数なので、Firefox OS について聞きたいことがあれば、詳しい人や中の人に直接聞ける良いチャンスです！

必要なものは Firefox OS への興味とノート PC だけ。

ちょっとした調べ事ならこちらの OpenGrok でコードが見られます。

http://reading.fxos.org/source/

懇親会

21:30 位から、希望者に一口1000円でカンパ募ります。
スタッフが飲み物と食べ物を買いに行きます。

FxOS コードリーディングって？

Firefox OS (FxOS) の実装やアプリのコードに興味がある人が、ソースを読んで情報交換するグループです。興味があれば、是非、こちらの Facebook グループに参加して下さい。

FxOSコードリーディング@facebook

毎月 Mozilla オフィスでミートアップという名のオフ会をしています。半年に一回ぐらいで、 休日開催のセミナー やってます。

当コミュニティではGMOインターネット様よりコミュニティ支援としてConoHa VPSの提供を受けてます。
ConoHaについてはこちら→http://www.conoha.jp/community
