description

留学前に知っておいたほうがいいことがあります！

フィリピンの語学学校を多数体験して（約30校）、生徒のリアルな体験談（約200人）を集めていろいろ分かったのでアウトプットします。イギリス、カナダ、マレーシアの学校も経験したので、興味ある方がいたらそれぞれの良さを比較することもできます。

初めてフィリピンに行った2012年から、もう何回行ったか分かりません。軽く10回以上行ってます。スタッフみたいになってる学校もあるし、先生の採用の同席したり、カリキュラムを一緒に作ったりしてたら、すごい詳しくなってしまった。

僕自身は留学を経験する前は本当に何も分かってなかったです。だから、これから留学を検討してる方へ、英会話を学ぶ道のりや、効率よい勉強方法、そして上手な留学の使い方を伝えたいです。英会話に興味あったり、留学を検討してる方は是非来てほしいです。絶対知ってたほうがいいことがあるから。

みんなが考えている以上に留学を失敗してる人はいます。でも、そういう声は表に出てこないです。失敗した人は学校を変えることがありますが、ほとんどの場合が完全返金にはならないです。そういうことにならないように学校選びのポイントは押さえていたほうがいいです。
