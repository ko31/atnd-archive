大阪朝食会です。
場所は「サンマルクカフェ」。ヨドバシカメラ梅田1Fです。電源付き。
禁煙ですが外の席は吸えます。ただ、この時期なので寒いです。


ボク(<a href="http://twitter.com/pirokichi">@pirokichi</a>)は、07:15～10:00にいる予定ですが、
ホットな時間帯は、08:00～09:00です。
遅刻・早退・ドタキャンは自由ですが、ホットタイムのどこかで顔を出されるのがベターかも～
だれもいないときは、ボクはパソコンでオシゴトしてます。
開店すぐに来られる方が増えてきてますよ。


参加者制限はないので、だれでも参加できます。
iPhoneを持ってなくても、twitterやってなくても、大丈夫ですよ。
みんなスーツ！ってわけでもないです。9時を過ぎるとスーツ率が激減します。


参加される方は、ここで表明していただくか、 <a href="http://twitter.com/pirokichi">@pirokichi</a> か <a href="http://twitter.com/#search?q=%23osakamorning">#osakamorning</a> まで。
朝になって突然参加になっても店から溢れなければ大丈夫ですよ～
突然来られるのもアリなんですが、
「だれ？」ってなるかもしれないので、
何らかのカタチでコンタクトをとるほうがいいかも？


一番のメリットは、出かける前に交流することでモチベーションがUPすることです。
あとお別れのときに「いってらっしゃい」で終わるところもなんかいい感じ。
