第４回ダンパWEBを開催します！

開催に先立ち、審査員を募集します。

審査員10名集まれば、開催決定となります。

詳細は、「第４回ダンパWEB開催概要」を参照ください。

開催概要説明動画はコチラ。

■応募資格

  • 審査期間中に、下記が実施可能である事。
  1. 投稿された最大２０本（予定）の投稿動画すべて目を通す事が出来る。
  2. 目を通した動画に採点する事が出来る。
  3. 結果を、 @danpa_web へメールまたは DMにて連絡する事が出来る。
  4. もし、複数の方が１位となった場合、決戦投票を行います。決戦投票も可能である事。
  • 大会参加／不参加を問いません。
  • 大会参加者は、自分の動画に投票してもOKです。

■Q&A

Q : 審査員は、具体的に何をやるのですか？

A : 審査員のやる事は次の通りです。

  1. 審査員は、１部門につき５点与えられます。
  2. 各部門毎に基準となる動画「基準動画」をご用意。基準動画に対し、良かったか、悪かったかを審査頂きます。
  3. 良かった動画に対し、１～５点配点できます。
  4. 最大５本の動画に配点可能です。
  5. 自分の動画に配点してもＯＫ。
  • （例）
    • 動画Ａに５点配点
    • 動画Ａに３点、動画Ｂに２点配点（合計５点）
    • 動画Ａ，Ｂ，Ｃ，Ｄ、Ｅに１点づつ配点（合計５点）
  • 基準動画も審査対象です。
