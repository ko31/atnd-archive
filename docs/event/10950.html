はてな技術勉強会 Hackathonを開催します。

今回は特別ゲストに Titanium Mobileを提供しているAppcelerator 社の Platform Evangelist 増井雄一郎(@masuidrive) さんをお迎えします。

Hackathonのテーマは、Titanium Mobile と はてなのAPI を使った iPhone/Android アプリ開発です。

京都本社の8Fのセミナールームで実施します。開発は個人又はチームに別れて行い、最後に発表していただきます。

開催概要

  • 日時: 2010年12月19日（日）09:30開場、10:00開始〜17:00 終了
  • テーマ: Titanium Mobile と はてなのAPI を使った iPhone/Android アプリ開発
  • スケジュール:

    • 10:05 – 10:20 増井雄一郎さんによる Appcelerator 社についての講演
    • 10:30 – 開発開始
    • 16:00頃から 発表
    • 17:00 終了
    • 17:30 – 懇親会 (5,000円程度を予定)
  • 定員: 30名
  • 会費: 無料
  • 備考: 各自で Titanium Mobile の開発環境を整えてからお越しください。
  • 公式ハッシュタグ：　#hatenatech
  • 参考資料

    • Titanium Mobileで作る！ iPhone／Androidアプリ
