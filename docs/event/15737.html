description

東日本大震災を乗り越えて仙台クラスタ今ここに集う!!

以前からTwitterでオフ会やりましょう！という話がありましたが、OSC2011仙台で多くの方が一堂に会するという絶好のタイミングでもありますので、この機を逃さず一気にオフ会やっちゃいましょう！

会場・会費

会場となる「いただきコッコちゃん」から1人500円の割引券をゴッソリいただきましたので、これを活用したく存じます。
ただし震災の影響で未だ仕入れが不安定なようで、現状ではコース料理が出せないそうです。
この点については後日確認しますが、会費は3,000～4,000円あたりで何とかなればと思います。

参加申し込み

Twitter、mixi、Yahoo! など各種アカウントでATNDにログインし、画面右上の参加ボタンを押してください。
「Twitterオフ会」といいつつ特に制限はありませんので どなたでもお気軽に参加OK です！
アカウントをお持ちでないお友達も連れて行きたいというハートフルな方は、コメントに参加人数を書きこんでいただければと思います。

締切り

なるべく流動的に対応できるように致しますが、一応 5/17(火)締切り にいたします。
お店を予約する関係上、参加表明はお早めにお願いできれば助かります。

その他

不確定要素が多々ございますので内容に変更がある場合があります。変更があればTwitterでも情報は流しますが、ブックマークやRSS登録するなどして定期的にご確認いただければと思います。

5/19追記

お店を「芳賀」で12名予約いたしました。お店についたらその旨伝えていただければ案内してもらえるはずです。
キャンセルは前日までなら可能だと思います。ドタキャンはご勘弁ください…
コースにはしないで席のみです。飲み放題￥1,500＆お料理代のワリカン制になります。単価が安いので割引券を使えば￥3,000くらいになると思います。
遅れる場合などはDMかreplyを飛ばしていただければと存じます。

では皆様とお会いできるの楽しみにしております！！
