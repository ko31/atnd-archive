title D&AD Talk Event
catch Creative Directors in Conversation vol.5
description

世界で最も厳しいと定評のある審査を経て選考される国際的なデザイン・広告賞、D&AD賞。最も多くの国からエントリーを募った2014年度より、輝かしい賞を受賞したクリエーターおよびに審査員による体験談、審査の現場についてお話しいただきます。海外のデザイン・広告賞への関心・参加が増える今、出演いただくクリエーターたちのリアルな言葉とともにD&AD賞を分析します。海外のデザイン／広告賞を目指す方にとって、必ずヒントが見つかるはずです。

出演：
佐野研二郎（MR_DESIGN／代表）
2014年度パッケージデザイン部門審査員、2014年度ブランディング部門イエローペンシル受賞

菅野薫（電通／クリエーティブ・ディレクター、クリエーティブ・テクノロジスト）
2013年度デジタルアドバタイジング部門審査員、2014年度クラフトフォーアドバタイジング部門ブラックペンシル受賞

平林奈緒美（アートディレクター / グラフィックデザイナー）
2014年度グラフィックデザイン部門審査員、2001年度グラフィックデザイン部門イエローペンシル受賞
