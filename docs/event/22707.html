イベント ユーザー
event_id 22707
title オーダーメイド型フォトレッスン
catch 写真を始めたけど、いまいち良い写真が撮れない・・・という方へ
description

・せっかく綺麗な景色（花）を見つけたのに、撮ってみるとなんかいまいち！
・赤ちゃんや彼女・彼氏を魅力的に撮りたいのになんか顔色が冴えない！
・ 料理を美味しそうに撮りたいのに、なかなか美味しそうに撮れない！

と思っていたら、逆に

・ なぜか今回は上手に撮れた♪
・ なんか雰囲気のある写真が撮れた！

なんていった経験ありませんか？
こういった経験（結果）には全て理由があるのです！
そしてその理由は驚くほど簡単なことだったりするのです。

ちょっとしたコツを知れば誰でも1ランク上の写真が簡単に撮れる・・・
のに、その「ちょっとしたコツ」を知らないばっかりに上記のような
いわば運試し状態になってしまっているのかもしれません。

フォトレッスンではその「ちょっとしたコツ」を伝授いたします。
また実践編では、実際の被写体を前にしながら、
自分で撮った写真と、その問題点・改善点をその場で随時指摘させていただきますので、
すぐに自分の写真が「変わった！」のを実感できるはずです。
習うのと慣れるのを同時にこなすことが出来るレッスンはプライベートレッスンならではです。

これから長い人生、沢山写真を撮る機会があると思いますので、
きっと皆様の素敵な財産になるはずです。。

一人でも、複数人でも。屋内でも、屋外でも。
撮りたい被写体や撮りたいシチュエーション、また、お持ちのカメラの合わせた
完全オーダーメイドの資料と内容でお届けいたします。

カメラの種類も、コンパクトカメラ、ミラーレス一眼、一眼レフ、と
全てのフォーマットに対応しています。
