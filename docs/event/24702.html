こんにちは、Memolane の庄子です。

この度2月13日（月）に、Memolane CEO Eric Lagierを招きユーザーミートアップイベントを開催します。

Memolane(メモレーン)は、ソーシャル・メディアに投稿したコンテンツを一元化し、タイムラインを作ることができるサービスです。ソーシャル・メディアに投稿した思い出をMemolaneで振り返って楽しむことができます。

Memolaneは2010年春Copenhagen Startup Weekendで優勝し、そのまま起業という短期間で立ち上げた会社です。
起業後、デンマークからサンフランシスコに移住し、Memolane version2.0を2011年11月にローンチ。更なる飛躍へ向けてプロダクト改善に取り組んでいます。
当日はEricから「Memolaneを作った経緯と、これからの展開」に関して語って頂きます。

「Eric Lagierのプロフィール」
Memolaneの共同創業者兼CEO。
Skype Mobile、Oracle等モバイル、インターネット領域で15年のキャリアを積む。Memolaneの前は、世界初の無料"忘れ物預かり所(Lost-and-found)"ネットワークを提供したTigertag.comを創業。
