description

みんなで集まってappengineを触ります。統一テーマなんかは設けず、各自で好きなことをやります。

趣旨：
集まってhackathonするとか結構面白いので継続的にやりたいと思っています。
今回の開催は試験的なもので、趣旨なんかは会を重ねるごとに決めていこうかなと思っています。なので、主催者の会場の下見もかねたゆるめの集まりと思ってもらえるとちょうどいいです。

やること：
基本的に個人ベースで作業して、休憩タイミングで情報交換したり、終わりくらいにライトニングトークをしたりとかいう流れを考えています。
あまり根を詰めた作業を続けるのは休みの日だしあれなので、休憩は強制的に入れちゃおうかなと。二時間×4ショット集中して作業して、あとは楽しく談話する感じで。
ハンズオン的なものなんかは考えていないので、当日作業がないひとはやることを考えててください。

当日スケジュール：
9:00/開場
10:00/開始
12:00くらい/昼飯でも食べながら情報交換
15:00くらい/お茶でも飲みながら情報交換
17:00くらい/お菓子でも食べながら情報交換
19:30くらい/ライトニングトークとかで情報交換
20:00/閉場

会費：
会場が有償なので、一人参加費だいたい500円徴収します。適当な金額で割り勘予定です。

懇親会とかもやろうと思ってます。人数がそんなに多くならないと思うので、当日の勢いで決めます。
