description

「伊藤みどりのフィギュアスケート・ライフ」ほか、「生きる達人」をテーマとしたジュニア向けDVDソフトを製作・販売する株式会社ジュニア・ドリーマーズ・インターナショナル（東京都港区/代表：室希太郎）は、広く中学生～高校生から、新しいビジネス・アイディアを募り、堀江貴文氏が審査を行う、ネット配信＆DVDソフトの連動企画「堀江貴文のワンダフル・ライフ」プロジェクトをスタートさせる。
本企画では、IT界の風雲児としてその名をはせ、現在は宇宙事業や、出版物の大ヒットで、今なお信奉者も多い堀江貴文氏が、日本の未来を背負うジュニアたちに、世の中やお金の本質を説く、珠玉の講義シーンや、ジュニアたちから、ビジネス・アイディアを募り、評定を行う。
優れたビジネス・アイディアがあった場合、実際の起業も視野に入れおり、展開していく画期的、かつ壮大な映像企画である。
配信後、DVD化を想定。スタートは2010年冬からを予定しており、現在は、主に公式サイトを中心に、応募者を募っている。
（応募受付：2010年9月18日～10月5日）
