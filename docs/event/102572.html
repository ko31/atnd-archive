来場について(重要)

【受付】

  1. 参加費は無料です。名刺を一枚ご用意ください。
  2. 12:30から13:00までの間にご来場ください。
  3. 受付はエスカレータを上がって２階の「セルリアンタワーオフィス専用ロビーフロア」です。渋谷駅からの経路はここをご覧ください。
  4. 受付で名刺をお渡しになり、ATNDの【受付番号とユーザ名】をお伝え下さい。（受付番号はログイン後のPCだとユーザ名の横、スマホだとページ上に記載されていますからご確認ください。）
  5. 13時を過ぎると、スタッフ含め全員イベントに参加するため2階入口はクローズします。

【遅刻】

  1. やむを得ない事情で遅刻した方は、ATNDページの「コメント欄」から「到着予定時刻」をお知らせください。スタッフがお迎えに行きます。（お迎えまでに相当時間がかかることがありますからご了承ください。）
  2. 入り口は地下1階の駐車場入口内にある「オフィス専用B1ロビー入り口」になります。遅刻の際の経路はこちらです。

【名刺】

  1. 会場管理の都合上、お名刺を１枚お持ちください。
  2. 名刺のない場合は、指定用紙に「お名前(本名)」「メールアドレス or 電話番号」「勤務先 or 学校名(ない場合は「なし」)」を記入していただきます。
  3. いただいた個人情報は会場提供元(GMOホスコン様)と共有します。

その他のご注意

  1. 会場は椅子のみ、机はありません。
  2. 登壇者から許可いただいて「写真OK」としますので、メモしたいスライドはスマホで撮影してくださって結構です。
  3. PCでのメモ取りはキータッチ音が講演の迷惑になりますのでご遠慮ください。
  4. 「#MicrosoftMVP」でtwitterやFacebookで拡散していただけるととても嬉しいです。
