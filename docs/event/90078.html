人気イラストレーター潤宮るかさんに学ぶ！
デジタルクリエイターの働きかた #01

今年の山の日は働きかたイベントへ行こう！

2017年8月11日（金・祝）に、イラストのお仕事をしているデジタルクリエイター（イラストレーター、ディレクター等）のための働きかたイベントを開催します。

『 Project LayereD 』 （バンダイナムコエンタテインメント）や『 ボーイフレンド（仮） 』 （サイバーエージェント）などを代表作とし、顧客やユーザーが求めるキャラクター設計・デザイン・イラスト監修を得意とする人気イラストレーター 潤宮るか さんをお迎えして独立から初受注に至るまでのポイントやノウハウ、実際の受注後をイメージとしたライブペインティングのほか、

第２部では、イラストレーターへ発注される企業様もご登壇して頂き、「発注側×制作側」の両面視点から今求められる・求めているイラストレーター像や自己プロデュースの仕方について、潤宮さんと共にお送りするトークセッションも行います。

当日は夏コミ１日目と重なっておりますが、コミケ後にふらりと立ち寄ってみようかなという方も大歓迎です！
皆様のお越しをお待ちしております。
