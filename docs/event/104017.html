試合形式と当日の流れについて

受付方法

・参加申込者は大会当日18:00までに会場入り口にて受付してください。
・入場受付時に参加費用500円の支払いをして頂きグループ分けのくじ引きを行います。
・受付時に当日使用するポケモン６種類を記入していただきます。以降の使用ポケモンの変更はできません。

※会場キャパシティに限りがあるため、大会参加者以外の方の見学はご遠慮ください。
※17:30から会場を予約しており、その後会場設営となるため受付開始は17:45ごろとなる見込みです。
※スケジュールの都合上、18:15から対戦開始のため、以降に到着された場合の参加受付はできかねます

予選リーグ

・各グループ内での「総当たり戦」を行い、各グループの勝利数上位2名が決勝トーナメントに進出となります。
・3本勝負でバトルを行い、先に「2本先取」した方が勝者となります。

※予選終了時に同率順位となった場合は、「負け試合数」の少ない人が上位となります。それでも同一となった場合は対象者とグループ内でトーナメントを組んでもらい１本勝負（サドンデス）にて決勝トーナメント進出者を決めて頂きます。（試合をする時間がない場合はじゃんけんとなります）

決勝トーナメント

・予選から勝ち上がった16名によるトーナメント戦
・3本勝負でバトルを行い、先に「2本先取」した方が勝者となります。
・準決勝ののち3位決定戦を行い、決勝戦を行います。
・決勝トーナメントのバトルの模様は会場内のスクリーンにて放映します。

※当日の参加人数次第で、決勝トーナメントに参加する人数は変更になる場合がございます

基本共通ルール

・大会で使用できるポケモンは計6種類とし、同じ種類のポケモンを同一パーティに組み込む事はできません。

・対戦を始めるまでの手順は、以下のように統一します。

①トレーナーカードを見せ合う前に、QRコードにて対戦申し込みを行います。
先攻後攻に関しては「じゃんけん」で順番を決定します。
じゃんけんで「勝った方」がQRコードを差し出し、「負けた方」がカメラで読み込みます。

②パーティの書かれたトレーナーカードをお互いに見せ合います。

③使用するポケモン3体を2分以内に選出し、対戦を始めます。

④2戦目以降は、お互いが「再対戦」を押して対戦を始めてください。
選出時間は2分です。
トラブル時の対応について
1. 両者のアプリがフリーズした場合は、まずフリーズしている事を互いに確認してください。アプリ再起動後、すぐに再戦して頂きますが、この時の選出は同じポケモンにしてください。違った場合は敗退となります。その後の試合展開は自由です。

2. 一時的にアプリがフリーズしても、その後続行できた場合はそのまま試合を続けていただきます。

3. 連戦時に間違えて抜けてしまった場合や、タスクを切る必要がある場合は、アプリを再起動後すぐに対戦申し込みを行なってください。

4. 大会開始時点で参加者全員のスマホを「おやすみモード」に設定をお願いします。これにより電話などによるアプリの強制タスク切り替えを防ぐことができます。これらの設定をせずに強制タスク切り替え等が発生した場合は、その人に非があるものとしてその１戦は負けとなります。

使用するポケモンについて

1. 6種の登録パーティに入っていないポケモンを使用した場合は、いかなる理由であっても敗退となりますので、ご注意ください。
2. 大会中に技マシンによる技の変更や、別の技を覚えた同種ポケモンを使うことは可能です。
3. ただし時間の都合上、同一相手と連戦中に技マシンを使用することはできません。選出個体の変更に伴って技構成が変更することは認められます。

選出時間について

1. お互いのパーティを見せ合ってからの選出時間は毎回2分としています。
2. 予選リーグでは各自のセルフジャッジにて時間管理をお願いいたします。
3. 選出時間中に外部からの情報を取得する行為は全て禁止としています。（スマホでの検索やタイプ相性表などの持ち込みも不可です）理由は、運営遅延につながりうることと、持ち込んだものによる不公平を除外するためです。
4. 選出時間を計測する目的でのスマホの利用は認められています。
5. メモのための無地の紙とペンの利用は可能です。
