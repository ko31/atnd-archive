宿泊：ゴルフ場隣接のコテージ(12人の宿泊棟と4人の宿泊棟)をおさえてあります

チェックイン：宿泊棟へのチェックインは、15時から。荷物は朝、フロントで預かってくれるそうです。

送迎：原則、東武線上三依からの送迎のみですが、2名程度なら、営業氏が通勤途中にＪＲ那須塩原駅からのpiku-upも可能。

＊送迎の希望の有無を教えてほしい
＊飲み物は持ち込みOKなので、「自分の分は自分で」を原則とします。
＊初日の新ぺリアコンペの賞品をひとつお持ちいただくこと
＊当地は高地なので、朝晩想像以上に冷え込みます。部屋着は長袖長パンツがあったほうが良いし、上着はあったほうが良いかも

☆コースコンディション
5月に本格的にオープン、雪解けして間もないですが、月末までにはコンディションも良くなると思います。ただし、山間地で野生の鹿が生息しているので、コース内に「鹿の糞」が非常に多いと予想されますので、ある程度は覚悟をしてください＾＾；
