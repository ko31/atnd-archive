ハースストーン 天下一ヴドゥ祭 プレリリース

12月5日の「天下一ヴドゥ祭」正式リリースに先駆けて、プレリリースパーティを開催します！先行でパックを開封したり、新しいカードで対戦を楽しみましょう！

認定酒場になりますので、炉端の集い限定スキン「ネムシー・ネクロフィズル」取得も可能です。炉端初参加の方もこの機会にぜひ遊びに来て下さい！

約三年間炉端を開催していますが、ほとんどの参加者は最初はお一人で参加されています。暖かい炉端の集いコミュニティでハースストーン仲間を作りませんか٩( ‘ω’ )و

◆会場の様子
GAME BAR LOTUS

イベント概要

当日遊べる特別な炉端の喧嘩で交流戦を行います。お酒を片手に、新カードで盛り上がりましょう！

・炉端の喧嘩「ラスタカンのバーリトゥード！」
プレリリースイベント中に開封したカードを使ってデッキを構築できる喧嘩だ。クラスにも、同じカードを入れる枚数にも一切制限なしでデッキを組める、グルバシ闘技場ならではのメチャクチャな喧嘩だぞ！

・炉端の喧嘩「王家のレシピ」
最新拡張版のレシピを一足先に使ってプレイできる喧嘩だ！こちらの喧嘩は、カードパックを先行購入してなくてもプレイできるからな。
