title 夜フェス

description

少し近づくと暖かくて、もう少し近づくと熱いという
当たり前のことに気づいたのが昨年の、紅葉が彩るくらいの時。


揺れる火の先を見ていると、何かそれが生きているかのように思えて
火という現象を、こうして操れているのが、そもそも訝しくも感じてくる。


この匂いは、眩しさは、温度は、魅力は
認知し得ない、招かざる手のような気がして
くだらない探究心が、じりじりと脳の中央奥に迫る。


この距離感が、大事なんじゃないかと
人とか、社会とか、知り得ないこととか
そういった物事に差し替えてみたりして


何かを得たような気にもなったけど
その実、火を見て耽る、この時間の心地良さに
諸々は回収された。



リバ邸横浜というこの場所は
来たことがある人にも、来たことがない人にも伝えると
ただそこに存在して、人が生活をしている、シェアハウスだ。


なにもしていないけど、周りから見たら
ここはリバ邸という状態で居続けているんだろうな。
でも本当にうちはなにもしていない。


相変わらず庭は鬱蒼としているし
重厚なカーテンが遮光するリビングの床だって
４月の半ばのくせにまだ暖かい。



ここでサイレントフェスというものをしてみようと思う。


サイレントフェスとは


専用の発信機から専用のワイヤレスヘッドホンへ
音源を飛ばす仕組みの、音楽イベントで
周りから見たら無音の状態なので
サイレントということだ。


元は騒音対策としてヨーロッパで生まれ
現在はNYで流行中とのことだ。


普段はアップテンポな曲を流すことが多いのだが



今回は、



リビングに、大小様々なキャンドルを置いて



火を灯そう。



そしてついでだから



ホットワインでも用意して



体の内側から暖かさが滲み出てくるような状態で



ゆったりした、いい音楽を聴いて



贅沢に夜を過ごそう。



サイレントフェス用のヘッドホンはとても音がいいので



きっとあなたが



「これでもう十分だ」



と思ってしまうのに最高のサポートをしてくれるでしょう。
