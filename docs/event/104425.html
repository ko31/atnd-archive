受付方法

・参加申込者は大会当日18:00までに会場入り口にて受付してください。
・入場受付時に参加費用500円の支払いをして頂きグループ分けのくじ引きを行います。
・受付時に当日使用するポケモン６種類を記入していただきます。以降の使用ポケモンの変更はできません。

※会場定員に限りがあるため、大会参加者以外の方の見学はご遠慮ください。
※スケジュールの都合上、対戦開始時刻に間に合わない方の参加受付はできかねます。

予選リーグ

・各グループ内での「総当たり戦」を行い、各グループの勝利数上位2名が決勝トーナメントに進出となります。
・3本勝負でバトルを行い、先に「2本先取」した方が勝者となります。

※予選終了時に同率順位となった場合は【①勝ちバトル数が多い人②負けバトルの少ない人③直接対決で勝った人④じゃんけん】の順で順位をつけていただきます。

決勝トーナメント

・予選から勝ち上がった16名によるトーナメント戦
・3本勝負でバトルを行い、先に「2本先取」した方が勝者となります。
・準決勝ののち3位決定戦を行い、決勝戦を行います。
・決勝トーナメントのバトルの模様は会場内のスクリーンにて放映します。

※当日の参加人数次第で、決勝トーナメントに参加する人数は変更になる場合がございます

基本共通ルール

・大会で使用できるポケモンは計6種類とし、同じ種類（図鑑番号）のポケモンの重複使用はできません。

・対戦を始めるまでの手順は、以下のように統一します。

①はじめに「じゃんけん」をしていただき「勝った方」がQRコードを差し出し、「負けた方」がカメラで読み込みます。

②パーティの書かれたトレーナーカードをお互いに見せ合います。

③使用するポケモン3体を2分以内に選出し、対戦を始めます。

④2戦目以降は、お互いが「再対戦」を押して対戦を始めてください。選出時間は2分です。

使用するポケモンについて

1. 6種の登録パーティに入っていないポケモンを使用した場合は、いかなる理由であっても負けとなりますので、ご注意ください。（アローラやフォルム違いは別種として扱います）
2. 大会中に技マシンによる技の変更や、別の技を覚えた同種ポケモンを使うことは可能です。
3. ただし時間の都合上、同一相手と連戦中に技マシンを使用することはできません。選出個体の変更に伴って技構成を変更することは認められます。
（「シャドークロー」を覚えたギラティナと「りゅうのいぶき」を覚えたギラティナを育成し、連戦時に使い分けるのはOKです）

選出時間について

1. お互いのパーティを見せ合ってからの選出時間は毎回2分としています。
2. 予選リーグでは各自のセルフジャッジにて時間管理をお願いいたします。
3. 選出時間中に外部からの情報を取得する行為は全て禁止としています。（スマホでの検索やタイプ相性表などの持ち込みも不可です）理由は、運営遅延につながりうることと、持ち込んだものによる不公平を除外するためです。
4. 選出時間を計測する目的でのスマホの利用は認められています。
5. メモのための紙とペンの利用は可能です。

トラブル時の対応について

1. 両者・または一方のアプリがフリーズした場合は、まずフリーズしている事を互いに確認してください。アプリ再起動後、すぐに再戦して頂きますが、この時の初手・３体の選出は同じポケモンにしてください。違った場合は負けとなります。その後の試合展開は自由です。

2. 一時的に両者のアプリがフリーズしても、その後続行できた場合はそのまま試合を続けていただきます。
3. 交代バグやその他のバグについてはお互いの判断の元、再戦・続行を決めてください。
4. 連戦時に間違えて抜けてしまった場合や、タスクを切る必要がある場合は、アプリを再起動後すぐに対戦申し込みを行なってください。
5. 大会開始時点で参加者全員のスマホを「おやすみモード（常に知らせない）」に設定をお願いします。これにより電話などによるアプリの強制タスク切り替えを防ぐことができます。これらの設定をせずに強制タスク切り替え等が発生した場合は、その人に非があるものとしてその１戦は負けとなります。
6. 引き分けは再試合となります。選出は変えても問題ございません。
