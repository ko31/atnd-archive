スマホアプリ開発スタートアップセミナー

セミナー概要

普段スマホやタブレットを利用し、「アプリを作ってみたい」「iPhone/Androidってどう違うの？」などの身近な疑問から、アプリをリリースしたい。使うだけでは満足できない方。まずはアプリってどんなものなのか？から始めてみませんか？

こんな方が対象

*スマートフォンを普段使っている人
*デザイナから一歩踏み出したい人
*アプリを作りたいなーと漠然と思っている人
*アプリ開発をいずれ仕事にしたいと考えている人
*アプリの企画（案）は実は頭にある人
※何かしらスマホに興味をお持ちであれば、役に立つ情報をお伝えします
　今回は、ハンズオン形式ではございません
