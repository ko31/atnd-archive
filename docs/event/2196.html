講演内容

櫻庭 祐一

アニメーションの勘どころ

JavaFX で簡単に記述できるようになったアニメーション。その書き方や、使い方など初心者でも分るように、デモを交えて説明します。また、効果的な使い方なども紹介します。

岡崎 隆之

MaiTai のインタラクティブデザイン

JavaFX Script を知らなくても始められる Project MaiTai を使って、対話的なデザインを始めましょう。本セッションでは、MaiTai を使って簡単なムービーを作る過程を紹介します。
