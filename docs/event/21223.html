■場所

◎渋谷駅から徒歩の場合はまず109まで向かって下さい。
渋谷駅から歩いて109に着いたら道が2つに別れているので左手の道(道玄坂)に進んで頂き
そのまままっすぐ進むと右手に(カラオケインターネットカフェカラNET・ケンタッキー・カラオケ歌広場)
があります。
カラオケ歌広場を少し越えて廻るすし台所家を右に曲がり(廻るすし台所家とファミリーマートの間の道)
しばらく突き当たりらへんまで真っ直ぐ進むと左手にSTAR・CRESCENTがあり右手に道玄坂カフェが入ってるビルがあります。
道玄坂カフェが入ってるビルには2階にテーマパークレストラン(アルカトラズ刑務所付属渋谷病棟)があり地下1階に
和ダイニングRAKUと道玄坂カフェがあります。
