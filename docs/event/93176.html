最新のサイバーセキュリティ事情を学ぼう！

第1回 と 第2回 に続き第3回目を今年も開催します。

毎年2月1日から3月18日まで官民連携の サイバーセキュリティ月間 として各方面で、サイバーセキュリティに関する啓蒙活動が行われています。

地方では最新のサイバーセキュリティ事情を身近に接する機会が少ないですが、信州塩尻で各分野の専門家の方々から学べる貴重な勉強会を開催します。

パソコンに限らずスマホなどでインターネットを利用する事が一般的に普及し、今後はスマート家電や自動車、各種センサーのIoT機器などインターネットを使う事が増えていく事でしょう。
しかしインターネットは瞬時に世界中と繋がる利便性がある反面、サイバーセキュリティについて世界と同じレベルで考えて行かなければ、年々巧妙化するサイバー犯罪から身を守る事が出来ません。

また参加者の皆さんと共に意見情報交換し、この勉強会に参加された方が職場や学校、家族や友人にその知見を伝える事が出来れば幸いです。

目的

  • サイバーセキュリティに関する技術者の人材育成や人材交流と人材発掘
  • プログラミングを学ぶ方へのサイバーセキュリティの学習
  • システム・アプリケーションの技術者・開発者との情報交換
  • 最新のサイバーセキュリティについて一般利用者への啓蒙活動

講演内容

  • 対象
    プログラミングを学習される方（中学生以上）
    WEBサイトやアプリケーションの開発者・SE・プログラマーなどの技術者
    クラウド・サーバーやIT機器を管理運用されている管理担当者や技術者
    サイバーセキュリティに関心のある一般の方（中学生以上）
    ビジネスイベントではありませんので私服でお気軽にご参加下さい。
  • 講演内容
    産学官の各分野で活躍されている専門家の方々から、実際の事例を交えた最新のサイバーセキュリティ事情についてご講演をお願いしました。
    一般的な教育教材にあるような内容では無く、リアルで興味深い内容が聞ける機会になると思います。
  • 注意事項
    本イベントはサイバーセキュリティというテーマである為、講演内容には機密事項など取扱に注意すべき情報が多分に含まれます。
    動画写真の撮影やSNS等への投稿は許可されていない場合は制限される事があります。
    後日一部資料が公開される場合もありますが基本的にはイベント会場にてとなります。ご理解ご了承下さい。
