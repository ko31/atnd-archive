title 目覚めのリラックスヨガ
catch ～土曜日の午前中を有効活用して充実した週末を～
description

寒さやストレスで縮こまった体を呼吸に合わせてゆっくりほぐしていくことですっきり疲れをとり、充実した週末を過ごしましょう。

1.当日の流れ

10:00 開場
お着替えをしたらヨガマットの上で自由に体を動かして開始をお待ちください。

10:10～10:55　体を緩めるベーシックヨガ
体の関節と筋肉を緩めることでスッキリと疲れを取ってリフレッシュします。

10:55～11:05 休憩
水分補給をしたり、タイマッサージを応用した簡単なマッサージを各自で行います。

11:05～11:50　デトックスヨガ
主にねじる動きを行っていくことで体の中に溜まった老廃物が外に出るのを促していきます。

11:50 終了
お着替えをしていただき終了です。

ヨガマットは会場にあります。ゆったりとした着替えやタオル、飲み物をお持ちください。

2.　料金
2,500円
（お時間の都合で前半か後半の1つしか参加できない場合は1,500円）
