description

Flashのボタンについて語り合おう！

Flashを作る上で絶対に避けて通れない「ボタン」
それだけにいろいろなボタンや作り方が存在します。
タイムライン派もAS派も、他にもハイブリッドなボタンもFlexコンポーネントも、
なんでもいいので「ボタン」について語らいませんか。

  1. ボタンを作ってきて披露する
    • 「こんなネタ考えたんだけど」など面白いボタンを作ってきたり
    • ちょっと変わったボタンの作り方の紹介などなど
  2. ボタンについて質問する
    • ここまで作ったボタンのここがわからない

などなど、ボタンにまつわるエトセトラを皆さんと共有できればと考えています。
勉強会の詳細は にゃあプロジェクトさんのページ を参考にしてくださいね。

ひよこ会について

いちおうひよこ会についてざっくり説明しときます。
ひよこの会は「Flashにまつわる今さら聞けないような何かをみんな勉強する初心者の会」です。
プロアマ、学生ベテラン、年齢問わず、Flashを現在進行形で勉強してる人であればどなたでも参加可能。

だいたい

時間 タイトル
18:00～20:00 勉強会
20:00～20:30 相談会
21:00～23:00 懇親会

の3部構成になっていて、
*懇親会のみ実費*、あとは無料です。

もともと勉強会＋交流会を目指して立ち上げたので、Flashの話題を共通にしてみんなで交流しようといった目的が大きいです。
会の雰囲気としては、まだ2回目なので幹事の僕も含めて皆さん若干緊張気味ですが、逆に言うと変ななれ合いができあがってる風でもないので、初参加の方もどんどん積極的に絡んできてください。

懇親会について

できるだけ人数把握したいので、10/1までにできるだけ↓下のコメント欄に出欠の確認を書き入れておいてください。
参加者で、とくに欠席の連絡の無い方はそのまま出席の頭数に入れておきます。
あと、懇親会だけ参加、というのもぜんぜんありなので勉強会はこれなかったけどとか、お友達も懇親会には誘いたいといった場合も↓下のコメント欄にてその旨お伝えください。

ちなみに予算の予定は、前日までに正確に人数が把握できた場合は飲み放題で4000〜5000円あたりで予約、
そうでない場合は当日にどこか適当な居酒屋を探すことになります。ご協力よろしくお願いします。

その他ご質問などもお気軽に↓下のコメント欄かTwitterにて！
