description

世代を超えて、多くの人々に愛されているムーミン。2014年、ムーミンの原作者であるトーベ・ヤンソンの生誕100周年を迎えました。
ムーミン、ムーミンママ、ムーミンパパ、ミイ、スナフキン、そしてフローレンなど、ムーミンの物語に登場する人物たちは、自由に個性にあふれ、その言動に思わず共感を覚えずにはいられません。
ムーミンポストオフィス(フィンランド郵政ムーミ郵便局)は、手紙の楽しさ、魅力を再発見していただくため、メッセージフェスタ2014 in KITTE に協力し、ムーミン切手の販売、ムーミンの記念押印サービスを実施します。
ぜひ、この機会に、切手になったムーミンに会いに出かけてみませんか。
