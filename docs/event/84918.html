概要

  • 新年が始まった新しい季節に、カフェに籠ってストイックに読書や勉強をする会です
  • 読書や勉強のテーマは自由です。各自、本や教材を持参してください
  • 複数人で集まりますが、基本的に自習をします。会話は休憩時間に雑談をする程度です
  • こんな方にオススメです
    • 勉強したいけど、家だと勉強できない
    • 外に出ても、一人で勉強するのは精神的につらい
    • 休日だけど予定がない
  • 実施場所はカフェです
    • Wi-Fiは使えます
    • 電源もあります
  • 初参加の方も歓迎です
