参加対象者

・R言語を勉強している方

内容

※講義ではありません。

R言語やデータ分析等に関する課題を各自で持ち込み、座学・ディスカッションを行います。
今回はこちらでいくつかの課題を事前に用意します。
もちろん話し合いたい課題があれば、持ち込み大歓迎です。

課題の例：
・R言語のここがわからない
・このようなデータをとるときのプログラミングの方法は？
・こういう統計データをとりたいんだけど、いいデータ解析のソフトウェアない？
・こういうデータがあるんだけど、どのような統計データを取得すべき？
・こういう統計データがあるんだけど、どのような分析ができてどう結論できる？

アジェンダ

・一般的なデータの相関の求め方とその手順
・ggplot2によるグラフの作り方
・テキストマイニングにおける疑問文の抜き出し方
・その他（持ち込み課題）

ゆるゆるとやりますので、議題に変更が生じる場合があります。

紹介

・主催者
R言語や統計学については今年から勉強を始めた 初心者 レベルです。
IT技術者としては20年ほどの経験があります。

・参加者
行動経済学を専攻されている山梨英和大学助教後藤晶氏が参加します。
R言語や統計学については エキスパート です。
