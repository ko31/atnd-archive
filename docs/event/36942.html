■概要

　クリエイターを中心に世界的に利用されているメディアミックスブログサービスに Tumblr（タンブラー）があります。ソーシャルメディアとの親和性も高く、海外はもとより国内でも利用者が多いのに対して、そのユニークさ故、なかなか活用できていない方も少なくないのではないでしょうか。

http://www.tumblr.com/

　今回の勉強会は、「はじめての Tumblr ワークショップ」と題しまして、企業内などへの導入や利用などに豊富な実績を持つ高橋佳代子さまを講師役にお招きしました。

0）　Tumblrとは？　Tumblrのいいところ　実例紹介
1) 　自己紹介＆発信テーマを決める
2)　 TumblrのIDの取得と、設定（PC、モバイル）
3) 　実際に記事をアップする＆リブログ
4)　 みんなのTumblrをシェアしよう
5） 感想や質問ご意見

// Tumblr（タンブラー）

　ブログとミニブログ、そしてソーシャルブックマークを統合したマイクロブログサービスである。Twitterのように他のユーザーをフォローできだけでなく、リブログ（Twitterのリツイート機能に相当、後述）したり、「スキ！」（Twitterの「お気に入り」に相当）登録したりすることができる。
（Wikipedia より抜粋および一部追記。）

　また全世界では Tumblrは現在月間160億ページビュー（PV）、1億3000万人のユニークユーザー（UU）を誇り、サービスを展開している Tumblr 社は最も速く成長したIT企業の１つとして知られています。
（ Social Media Week 2012年2月14日）
