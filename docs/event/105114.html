description

5月から土曜日に渋谷にて「スマートフォン時代のWebコーディングスクール」を開催します。 これからWeb制作者になりたい方向けに以下の内容を全7回で学べる社会人向けのスクールです。

  • Web制作は初めての方のためのHTML 5.2・CSSの基礎知識
  • スマートフォン・タブレット・パソコンなどさまざまな端末で表示できるWebサイトの制作方法
  • 1つのページをさまざまな端末で表示させるための「レスポンシブWebデザイン」の基礎知識
  • 今時のWebコーディングに必須のCSS拡張言語「Sass」を使用したWebコーディング
  • JavaScript・jQueryの基礎実習
  • 他のスクールでは学べない、効率の良いWeb制作の方法

授業内容

全7回で、以下の内容を予定しています。

  • 最新のHTML 5.2・WHATWG HTML Standard・CSS入門
  • 2019年のWeb制作、スマートフォン対応事情
  • ブログ風のWebページコーディング
  • Sassを使用するメリット
  • CSSの命名規則による破綻しにくく運用しやすいWebサイトの作り方
  • 旅館風Webページコーディング
  • JavaScript・jQuery入門

初回体験レッスンの内容と受講特典

全7回の講座のうち、最初の1回目を「初回体験レッスン」として1,000円でご参加頂くことができます。 初回体験レッスンでは、以下の内容を予定しております。

  • スライド: 「文章をマークアップする」とはどういうことか
  • スライド: HTML・CSSの役割
  • スライド: 「レスポンシブWebデザイン」とは何か
  • 実習: HTML・CSSを書いて、Google Chromeでライブプレビューしてみよう

受講特典として、制作方法のテキスト（教科書）をご覧いただけるようにいたします。 動画も撮影予定ですが、動画の閲覧は正式お申し込みの方のみの特典とさせていただく予定です。
