description

21世紀は「知る」時代。
最近、テレビ番組や本をみても科学をわかりやすく伝える志向のものが多くなってきていると感じます。

そして、宇宙、素粒子、遺伝子、奥深い科学のテーマが大分身近に感じられるようになってきました。

パーソナリティを形成する要因として、遺伝と環境はおおよそ50%ずつと一般的に言われています。

DNAのタイプ別により、
アフリカから始まった長くて辛い15万年の旅の歴史が体の特徴や性格として刻みこまれています。

私たちサイコロカフェでは、
宇宙誕生～生命誕生～人類誕生、
こうした長い長い歴史の意味を科学的に紐解いて人間とは何かを探求し、 これ以上無い自分探しのツールと言えるDNA解析を通じて自分自身のパーソナリティを知ることの価値を伝えています。

今回、国立科学博物館で「人類のルーツ」をテーマに展示を観てまわり、
館内のオシャレなカフェで感想をシェアしつつ、私たちが研究しているDNAによる自己理解と他者理解について楽しく解説しちゃいます。

文系、理系関係なく、知識の深さも関係なく、楽しく有意義な時間となると思います。

同世代での雰囲気で行きたいので、20～30代で宜しくお願いしますm(_ _)m
