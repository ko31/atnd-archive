第２回 ドメイン駆動設計読書会@名古屋
　今回は宿題あるよ！

開催概要

今回はワークショップ形式で、以下の様なお題を元に、自分で作成したモデルをみんなであーだーこーだ言いながら発表していきます。

・Amazonの配送会社決定モデル
・マクドナルドのドメインモデル
・自動販売機モデル
・仮面ライダーモデル
・電気ポット・炊飯ジャーモデル　
　などなど、自分で『これをモデル化するの面白そう！』っていうもので結構です。

モデルは、紙で書いたものを写真で撮ってもらってもいいですし、astah*などのモデリングツールで作成したものでもオッケーです。（GoogleSiteで共有するつもりなのですが、主催者がサボってますごめんなさい）

場所

株式会社ニューキャスト セミナールーム

日時

2012/06/08(金) 19:00～21:00

事前準備

開催概要で書いたようなお題でモデルを作成して持ってきてください。（GoogleSiteで共有するつもりなのですが、主催者がサボってますごめんなさい）
紙のままならみんなで膝つき合わせて発表してもいいし、モデリングツールも自由にしてもらってオッケーです。

astah* なら、CommunityEditionがあります。

また、本読書会では『エリック・エヴァンスのドメイン駆動設計』を読んでいくことになっていますが、とってもお高いので、InfoQに要約版のPDFダウンロードできる ページ があるのでまずそちらを読んでみてください。

注意事項

参加費はありません。
ニューキャスト様ありがとうございます！
