開催概要

Visual C# Express 2010で開発できるオープンソースハードウェア『Netduino』『FEZ Hydra』等を使って、Java屋さんやC#屋さんでもできるんだ！ということで、電子工作をして遊んでみようと思います。

主に.NET Gadgeteerで遊んで行きたいと思いますので、ご興味のある方は是非。

2013/02/18 (月)
上限を超えている場合は別途調整します。

■開催日時

2013/02/23 (土) 14:00 ～ 19:00(最大)
※最大5時間程度の予定

■集合場所、時間

2012/02/23 (土) 13:50
カラオケ ウイングスター前で集合、遅刻された場合は 070-5641-2248 まで電話をお願いします。
いつもの会場とは異なりますので、ご注意ください。

カラオケ ウイングスター No.4ルーム（和室、掘りごたつ）
(最寄駅：名古屋駅 徒歩3分程度)

■募集定員

最大8名まで

■参加費

5時間フリータイム＆飲み放題の予定ですので、実費のご負担をお願いします。
電子工作に必要な部品は各自、手配をお願いします。

■内容

ノミクイで懇親を深めつつ、.NET Gadgeteerを試してみたいと思います。
会場はカラオケハウスですが、歌うのは必須ではありません。

初学者がいらっしゃる場合は、個別にプチハンズオンも入れる予定です。

■注意事項

人数確定の連絡が必要なため、募集期限までにATND、もしくは、Twitterで参加連絡をお願いいたします。

■必要な事前準備

以下の電子工作に必要な部品をご手配のうえ、当日ご持参ください。

しばらく.NET Gadgeteerで遊んでいく予定ですので、続けて遊びたい方は、以下の電子工作に必要な部品をご手配のうえ、当日ご持参ください。
2,3個であれば貸し出しは可能です。

  • FEZ Hydra Basic Kit あるいは FEZ Cerberus Basic Kit

あとでカメラ、液晶と接続して遊んでみたい方には、Hydraの方をおススメします。

以下の開発環境を準備のうえ、当日ご持参ください。

  • Windows XP, Vista, 7 or 8 (32 or 64 bit)
  • Visual C# 2010 Express
  • Microsoft .NET Micro Framework 4.2 QFE2 SDK
  • GHI Software Package v4.2
    • ダウンロードにはGHI社へのユーザー登録（無償）が必要です。
    • 当日USBメモリ経由で渡すことは可能です。
