近未来は、全てがオンライン化になると言われている昨今、世界的にすでに数千万人に利用されている世界最大規模の革新的オンラインビジネスプログラムを持つDubLiがまもなく日本に参入して日本での市場展開をします。

DubLiは、DubLiサイトを通して国内や世界の有名デパート、有名スーパー、有名ショップの公式オンラインショップや有名トラベル関連の公式予約サイトから予約やお買い物ができ、さらにすべてのお買い物にキャシュッシュバック付きの世界最大規模のキャッシュッバックオンラインショッピングポータルサイトです。
さらにDubLiには、音楽を無制限にストリーミングできる音楽サービスや無数の無料フラッシュゲームなどDubLには、多数の無料エンターテイメントサービスもあります。

全世界のトップ企業の数千社が名を連ねているDubLiの日本参入につき日本語サイトと日本国内ショッピングポータルも着々と準備が進んで日本ビジネスがスタートです。

誰でもスタートできる新規オンラインビジネスに興味ある方、日本参入につき国内ショッピングモールに無料で加盟店掲載したい方など日本参入のPre-LaunchならではのBIG情報満載のDubLiの日本参入Pre-Launch説明会です。（参加無料）
