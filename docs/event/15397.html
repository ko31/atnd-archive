description

印画紙の上に被写体を置き露光をした手法を『フォトグラム』と言います。
『スキャングラム』とは従来、医学用語でX線、CTスキャンで撮影された画像の名称ですが、スキャナーを用いて作った画像も『スキャングラム』と呼ばれています。
『フォトグラム』と異なり色彩表現も可能で、写真とは違う立体感があり
新しいデジタル表現として注目されています。
Street Flowersの写真はこの技法を用いています。

今回は花を素材に『スキャングラム』の作り方のセミナーです。説明します。
