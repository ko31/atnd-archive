■内容について

・基本的にアナログゲーム（電源を必要としないゲーム）をみんなでワイワイ楽しむ会です。
・ゲームをだしに、色々な人（年齢、職業、性別問わず）が交流出来る場を目指してやっていきたいなと考えています。
・私自身が今現在、 人狼ゲームサークル を主催しているのですが会のメンバーの中でも人狼以外のアナログゲームを
　もっとやってみたいという声が高まってきたため、このような会を開催する運びとなりました。
・ ゲーム の 持ち込みは大歓迎 です。
・勿論、手ぶらで参加していただいても誰かしらゲームを持ち込んでいると思いますので、何ら問題ありません。

■場所について

東京都多摩教育センター 地図
住所: 東京都立川市錦町６－３－１

・JR南武線 西国立駅 から 徒歩１０分
・JR中央線 立川駅南口 から 徒歩２０分
・多摩モノレール *柴崎体育館駅*から 徒歩１５分

※ネットで調べると 「東京都多摩教育センター」 とか 「多摩社会教育会館」 とか 「都立多摩図書館」 とか出てきますが、 全て建物は同一 です。

■参加費について

・ 学生 と 女性 の方は 無料 です。
・ その他 の方は、会場レンタル料＋その他雑費の名目で 一人700円 いただきます。

■途中参加、途中退出について

会への途中参加、途中退出については基本的に 自由 です。
出来れば、事前に参加表明していただきたいですが、*当日の飛び入りも歓迎* です。

■会場内での飲食について

会場内での飲食は基本的に 自由 ですが、 お酒 と ニオイのキツイもの に関しては 禁止 とさせていただきます。
