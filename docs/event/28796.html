description

あなたは、1日だいたい何時間の睡眠時間をとっていますか？
平日の睡眠不足を補うために、休日の寝ダメをしていませんか？
夜遅い時間のジョギングを行っていませんか？
食事をしてから寝るまでの時間は、何時間あいていますか？

先進国の中でも睡眠時間がとても少ない日本人。
欧米をはじめとする海外の先進国では、初等教育で眠り方を学ぶ「スリープエデュケーション」が組み込まれ、専門的な講習まで行われています。
日本人は、眠り方の基礎さえ学んでいません。情報が溢れている今、眠りについての情報もたくさんあります。
でも、何が正しい情報なのかを選択、判断するための正しい基礎知識を教えてくれる機会はほとんどありません。

「正しい眠り方」とは？

「正しい眠り方」とは、「眠りのメカニズム」を理解して眠る力を身につけることです。
現代人にもっとも必要とされている能力の一つです。
この講座では、友野先生の体験談を織り交ぜ、「眠りのメカニズム」についての解説、日常生活ですぐに取り入れられるテクニックなどをお話いただきます。
この機会に、正しい眠り方を知って、ONの元気と最強の美を手に入れましょう！

【友野なおの「眠活」基礎講座】
１．眠りのメカニズム
２．睡眠前の眠活
３．睡眠中の眠活
４．起床後の眠活
５．眠らないとどうなるか？

□講師プロフィール

【講師】友野なお氏
【プロフィール】14歳から6年間、単身イギリスへ渡り、帰国後、法政大学在学中にバルセロナ大学へ留学。2004年度ミス日本受賞。野菜ソムリエ、はちみつマイスター、睡眠改善インストラクターの資格を持つ。「食」と「睡眠」の専門知識を活かし、体の内側、つまり細胞レベルから徹底的に健康と美を極めるインナービューティアドバイザーとして活躍。

正しい睡眠の知識を知り、睡眠の質をあげることで、より多くの人が健康な生活をおくることができるように、多くの方の人生が豊かになるようにとイベントやセミナー・講演、企画や商品開発、雑誌、テレビ、ラジオ、WEB連載など、多岐にわたる分野で活動を展開し伝えてきている。「友野なお式インナービューティー塾 基礎編・応用編」を主宰し、これまで約2000名に体の中から綺麗になれるインナービューティーテクニックを伝授。
さらに、女優やモデルに大人気の美容クリニック渋谷DSクリニックにて、睡眠療法を行う専属のカウンセラーとして勤務。
