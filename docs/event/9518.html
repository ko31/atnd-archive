KDDIデザイニングスタジオとドコモスマートフォンラウンジを一日で巡って端末を触ってみようという試みです。

基本的にそんなにしっかり予定を立てて巡れるわけではないのですが、一緒に移動したい人がいたら確認したいので、今更ですが募集開始します。
予定は以下のように考えてます。

  • 11時KDDIデザイニングスタジオ着(現地集合)
  • 12時お昼(KDDIデザイニングスタジオ内)
  • 13時イベント→ IS03をいち早く試せる!! – KDDI デザイニングスタジオ)
  • 15時イベント終了見込み
  • 16時移動開始
  • 16時30分ドコモスマートフォンラウンジ着
  • 17時30分イベント開始→ 楽しく広がるWebの世界、Androidで遊ぼう (予約が必要です。 一緒に見たい方は予約は各自取ってください )
  • 18時イベント終了見込み
  • 20時ドコモスマートフォンラウンジクローズ

一応予定は立てていますが、大いに押す可能性があります。ひょっとしたらドコモスマートフォンラウンジに行けない可能性すらあり得ます。

KDDIデザイニングスタジオをもっと堪能したい人、メインがドコモ目当ての人は、わたしについて行かないで自分で移動することをおすすめします。
