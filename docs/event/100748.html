description

あらゆる印象を受け入れ、自分の内側から感情や欲望を伴う力「感性」。
概念や価値などが目まぐるしく変化するこの時代を生きる力として
世界中の企業や大学、そして保育教育現場から注目されています。

効率、数字、他者の評価などが基準の暮らしのなかで
凝り固まってしまった感性をやわらかくし
自分の感じたことを、感じたままに表現する
ありそうでなかった、おとなのためのアトリエができました。

【umumアトリエがお届けすること】


1.アートを楽しむ
umumが大切にしているのは「上手な絵をかくこと」ではなく「感性をやわらかくすること」。
「絵の才能がない」「うまく描かなきゃ」などの自信のなさや固定概念から解放し
表現することの楽しさや心地よさをお届けします。


２.アートを学ぶ
毎回あるテーマをもとに、アートの専門知識や方法論、画材の使い方などを体験を通して学びます。
気になるテーマの回のみの参加も大歓迎です！


３.感性をやわらかくする
umumアトリエのプログラムには、決められた制作工程やお手本はありません。
五感を刺激するモチーフや世界中で集めたたくさんの画材から
イメージにフィットするものを選び
あなたが感じたことを、感じたままに表現できる環境をご用意しています。


その４：アートでコミュニケーションを深める
年齢や立場関係なく、様々な人と一緒に表現したり、作品を対話型鑑賞したり。
人との「違い」をポジティブにとらえる感覚やアートがもたらす人とのつながりを体験します。


umumは4つの体験を通して「あなたとアートの関わり」をお届けします。




【企画詳細】

2018年6月から東京都内の様々なエリアで定期的に開催しているumumアトリエ。

今回のテーマは….

「 デザイン×見せ方」

幼児教育でも注目される手法、無作為の形からイメージする「見立て」と
形や文字を画面の中で構成するデザインの基礎「平面構成」。
2つの手法で作品を作り、デザインの考え方の基本と
他者にイメージを伝えやすい見せ方を学びます。
