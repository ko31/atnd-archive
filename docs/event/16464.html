第3期「Seed Accelerator」応募説明会を6月7日(火曜日)に開催します。

どんどん質問にお答えしたいと思いますのでぜひお越しください！

また、Open Network Lab の雰囲気を直に味わえる絶好の機会でもあります。

1期生、2期生のメンバーが参加する可能性もありますので、
もしかしたら彼らの経験を聞くことができるかもしれません。

そして、不明な点や心配な点をこの時にクリアにして頂いて、
ぜひ募集してみて下さい！
