【お知らせ】
開催日当日（10月6日）となりました。
本日の開場・受付開始は、14:50を予定しております。（昨日ご案内した時間は誤りでした）
なお、事前申込をされていない方の参加、イベント途中からの参加も可能です。
1人でも多くの方のご参加をお待ちしています。
※翌週10月13日開催の「Part2」については こちらから ご覧ください。

開催概要

  • 日時：2012年10月6日（土）15:00〜17:00
  • 場所：熊本市国際交流会館 2階ミーティングルーム（ 会場詳細 ）
  • 主催：ほっとけない熊本プロジェクト
    • 当団体は「 STAND UP TAKE ACTION 」を主催するNGOネットワーク「 動く→動かす 」の加盟団体です。
  • 参加費：無料
  • 定員：20名（※要事前申込→詳しくは「 参加申込方法 」節を参照ください）

内容

  • ワークショップ『身近なもの「油」から考える貧困問題』
    • 熊本市国際交流会館を拠点に活動する「 SMILE Station 」の高校生メンバー3人をファシリテーターとして、
      私たちの生活に身近なものである「油」から貧困問題について考えるワークショップを行います。
  • STAND UP TAKE ACTION
    • 貧困問題をなくそうと立ち上がる「STAND UP TAKE ACTION」を行います。
  • ミレニアム開発目標（MDGs）の紹介
  • 当団体の活動紹介

参加申込方法

  • 参加無料・当日参加可能ですが、参加者数把握のために下記のいずれかの方法での事前申込をお願いいたします（ 事前申込は、10月5日21時で締め切らせていただきました ）。
  1. 「 参加申込フォーム 」からの申込
  2. 右上の「 このイベントに参加登録する 」ボタンによる申込
    いずれの場合も申込いただいた後2〜3日以内に当団体事務局よりご連絡をさせていただきます。

備考

  • 本イベントの翌週10月13日（土）には、市民会館崇城大学ホール（熊本市民会館）にて
    「STAND UP 2012 in くまもと Part2」を開催します。こちらにもぜひご参加ください。詳しい内容は＞ こちら
  • この企画に関する質問などのお問い合わせは、このページの「 コメント欄 」もしくは、ほっとけない熊本プロジェクト公式HPからお願いいたします。
