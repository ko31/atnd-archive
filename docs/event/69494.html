description

多くの企業や組織において、コスト削減や柔軟なリソース割り当て、運用効率の向上など、様々なメリットを期待して、サーバの仮想化への取り組みや検討が進んでいます。

しかし一方で、導入設計の検討や、稼働後の運用管理、仮想化基盤の拡張時の再検討など思った以上にコストや業務負荷が発生しているケース多く存在しています。

本セミナーでは、従来型の仮想化インフラが抱える課題を根本的なポイントから解決する、Nutanixのご紹介と活用事例、ならびにそれを実現するテクニカルな詳細に至るまでエンジニアによるデモを交えてご紹介して参ります。

仮想化基盤の構築・運用を行われている情報システム部門のお客様、ならびに仮想化基盤を提案されるシステムインテグレータのお客様のご来場を心よりお待ちしております。

お申し込みが必須となりますのえ、本サイトよりお願い申し上げます。
