私たちConsulting Collegeは、我々と春期プロジェクトに携わってくれる新規メンバー募集の説明会を行います！

『Consulting College』とは…

大学生と世界4大会計系コンサルファームが2012年に設立した、NGO/NPO専門の学生コンサルタントチームです。
国内最大規模のNPO/NGO団体に対して、コンサルファームと協働し、コンサルティングサービスを提供しています。過去3年間で、11件の支援を行ってきました。

私たち「学生コンサルタント」は…
クライアントが抱える問題・悩みに対する解決策を、コンサルティングファームのノウハウを元に、学生ならではの発想力で考え、クライアントにポジティブな変化を起こします。

（詳細はFacebookページをご覧下さい）

《Consulting Collegeで学生コンサルタントとして働くメリット》
実際に働く中で、コンサルティングについて学び、スキルが得られます

●インターン中
・コンサルタントとして働く経験が得られ、自分のキャリアを考える上で役に立つ。また、就職に役に立つ
・NPO/NGO団体の運営に関わる貴重な体験ができ、社会貢献につながる

・社会人とかかわることで、自分の考えが生かせる点、改善点が発見でき成長を感じられる
・学生ならではの発想を生かし、社会に通用するアイディアを作り上げることができる
・論理的思考力など物事の考え方、パソコンスキル、プレゼンテーション能力などが向上する

●インターン後
・Consulting Collegeの運営に携わり、広報、採用、財務、営業といった、様々な業務を経験できる
・その結果、社会で役立つ運営スキルが身につき、自分の成長が実感できる
・良きライバルと切磋琢磨しながら、一味違う充実した学生生活が送れる
