【Story】

『僕をMichaelにさせて。』

舞台はオリンピックセンター。

出会いのフォーラムに集まったのは、日本、そして世界中から集まった舞台芸術を生業とする方々。

そういった人達を一目見ようと、会場には子どもたちや、その家族で溢れかえっている。

そんな会場内で、どこからともなく現れたのは、なぜか固まってしまっている2人のダンサー。

1人は普通の服を着ていて、もう1人はMichaelの衣装を着ている。

そして、普通の服を来ている人の前には、Michaelの衣装やカツラ、ハット、が置かれている…。

「どうしてこの人たちは動かないんだろう・・？」
子どもたちがダンサーの前に集まり興味津々に見ていると、どこからともなくこんな声が…

「僕をMichaelにさせて…」「僕をMichaelに…」

子どもたちは、可哀想な彼をMichaelにさせてあげようと、隣のダンサーを見ながら目の前に置かれた衣装を当てはめていく。

悪戦苦闘しながらも、見事2人のダンサーの衣装が一致したその瞬間…！

会場全体からあの名曲「Beat it」が流れ始め、2人のダンサーが踊りだしていく！！

自分のおかげでダンサーが動いたと子供たちが喜んでいるのもつかの間、気づくとダンサーは一人ではなく、
周りにいた人たちまで踊りだしていき、最終的には会場全体でBeat itを踊りだすことに…！！！

【一緒にフラッシュモブをしてくれる人大募集】

今回の企画のコンセプトは「子どもたちの日常にもエンターテイメントを」です。

出会いのフォーラム2014のコンセプトでもある「かんじる」「ふれる」「かんがえる」を
盛り込んだものにしようと考え、上記のようなフラッシュモブに決定しました。

そして、今回のフラッシュモブを行うにあたり、
マイケルジャクソンのBeat itを一緒に踊ってくださる方を大募集します！！！

・ダンスフラッシュモブをしてみたかった！
・とにかく面白いことがしたい！
・Michaelのことならおれに任せとけ！
・この夏まだ予定ないんだけど！

そしてなにより、子どもたちの日常にエンターテイメントをちりばめたい方！

この条件に一つでも当てはまる方がいたら、ぜひご参加いただきたいです！

・ダンスが全く踊れないんですよね・・・
⇒全く問題ありません！ダンス初心者の方でもこちらから練習用動画をお送りしますので、
