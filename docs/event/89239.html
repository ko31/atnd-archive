子育て世代☆応援イベント！

≪ 個育てあかもちセミナー ≫

「あかもち」ってなぁに？
お子さまを３つのタイプに分類して、タイプ別の個性の違い、ほめ方、叱り方などをお伝えします。

ママ自身のタイプも知ることで、子育ての悩みやイライラを軽減し、子どもとうまく付き合うヒントを見つけられます。

堅苦しいセミナーではありません♪
参加者さん同士の交流も楽しいですよ！
