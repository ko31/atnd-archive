概要

先日、東京リージョンの初年度の成長率が世界でナンバーワン！との発表がされた Amazon Web Services(AWS)。

どちらかといえば開発者が使うもの、そんなイメージだったクラウドコンピューティングも、こういった成長の背景には、沢山のウェブの制作案件の実績があります。

今回の勉強会では、「ウェブの制作で活用できるクラウド」をテーマに、多数の実績を誇る Cloudpack 後藤さんに事例などを中心にご紹介いただき、また数ある AWS のサービスの中でも、特にウェブ制作をされる方にホットなサービスについて AWS 片山さんにスベラないお話をお願いしております。

また、恒例の事例紹介では、世界に向けてまさに大躍進中の ChatWork の舞台裏についてチャットワーク藤原さんにお話頂きます。色々な立場の人が関わるウェブ制作のお仕事ではかかせないチャットツール。その裏側がどうなってるのかを聞けるまたとない機会です。

ウェブ制作をされている方で

  • クラウドって何？どういうところが嬉しいの？
  • いつも使っているホスティングとどうちがうの？
  • 開発者の人が AWS っていってるけれど、それってどんなもの？

といった疑問をお持ちの方、是非お立ち寄りください。

また、いつもの通り

  • AWSに興味があって、使ってみたい方、
  • AWSは触ってみたけれども、もっと深く知りたい方、
  • AWSをすでに使いこなしていて、その使い方を広めたい方、
  • AWS の設計方法について、ディスカッションしたい方、
  • JAWS-UG 京都の運営メンバーになられたい方、

もどしどしご参加ください。
