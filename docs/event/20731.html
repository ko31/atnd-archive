じゃばすく製作所（仮） ひとつめ

　JavaScriptの事をちょっと基本的な事から学んでみたり、JavaScriptを使って何かモノを作ってみたりする勉強会です。
　今回は初回という事もあり概要や入門的な事を学んだ後、ブラウザ上で実行する電卓アプリを作ります。
　運営自体もJavaScriptを使ってはいますが素人です。みんなで一緒に学びを楽しんで行きましょう。よろしくお願いします！

会場

　長崎市立図書館新興善メモリアル会議室2（洋）。

　会場の場所が少しわかりにくいです。 長崎市立図書館 Nagasaki City Library 図書館について に図がありますが、正面入り口から入場後、1階図書室へ入らずそのまま右へ折れその奥にもある出口から一旦外に出て左へ少し進むと入り口があります。そこで靴をスリッパに履き替えて右奥の方に会議室の入り口があります。

　館内飲食禁止との事で食べ物の持ち込みはできませんが、水分補給のためのペットボトル入り飲料の持ち込みは可能です。

備考

　JavaScriptきほんの’き’ はスライドを使った基本的な事のおさらいになります。

　電卓アプリはちょっとだけjQueryを使いますので、jQueryの概要と今回使う部分の説明が入ります。

　ノートPCなどをお持ちの方は是非持参ください。無線LANアダプタなどがあるとインターネット接続が可能です。
　JavaScript実行環境としてのブラウザはWebKit系（SafariやChrome）がお勧めですが、基本的には問いません。ただしInternetExplorerの古いバージョンだと上手く動かない可能性があります。
　各々お好みのテキストエディタもお持ちください。

　見てるだけ、他の人との共有での参加ももちろん可能です。雰囲気だけ味わうのもいいかと思いますよ。

　インターネット接続回線としてWiMAXを用意していますが、ご自分でどうにかできる方にはそうしていただけると助かります。一回線しか用意できないので、あまりに多くなると接続できなかったり極端に遅くなる可能性があります。

　電源タップをお持ちの方も持参していただけると嬉しいです。今回の会場は部屋が広い上に四隅にしかコンセントがなく延長して分け合うのが必須となります。こちらでも何本か準備しておりますが、何とぞご協力ください。

　終了後、この為のというわけではないですが、懇親会を開きたいと思っています。
　参加は自由で、場所や費用なども当日の状況により決めたいと思いますが、できるだけ気楽で安く上がるお店をと考えています。この会と直接関係がないので懇親会だけ参加でもOKです。
　ただし、酒乱の気のある方はご遠慮ください :-)

　懇親会の会場を決定しました。
　「誠」という居酒屋さんで行いたいと思います。住所は 長崎市興善町７−２ です。費用は3000円程度を予定しています。当日割り勘でお願いします……。
　時間は午後5時過ぎくらいからを予定しています。懇親会のみ直接お越しくださっても結構ですよ。唐揚げが人気です！
