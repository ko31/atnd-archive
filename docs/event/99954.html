ノウハウ公開!ゼロからはじめるライブ配信セミナー


セミナー企画・運営ご担当者様、営業企画、マーケティングご担当者様向け

近年、Youtube LIVEやFacebook LIVEの利用状況は急速に増加し、
手軽なライブ配信のビジネス利用ニーズが高まっています。
BtoBにおいても、オンラインセミナーは一般化されつつあり今後も増加していく見通しです。

このような市場背景を受けて、本セミナーでは、
これからオンラインセミナーを始めたいと考える企業様、既にオンラインセミナーを開催されている企業様にむけて、
ライブ配信の基礎をレクチャーするセミナーを開催いたします。

▼このような課題をお持ちの方におすすめです。

  • 社内リソースでオンラインセミナーを検討中
  • 遠隔地営業用として、商品紹介のオンラインセミナーを行いたい
  • セミナー集客のを強化や開催コストを抑えたい
  • 配信ツールを導入したが、コンテンツの見せ方に課題を感じている
  • 過去のオンラインセミナーコンテンツを有効活用したい
  • web経由での新たな集客施策を導入したい

ご参加いただいたお客様には、もれなくライブ配信の基礎知識をまとめたホワイトペーパーを差し上げております。
オンラインセミナーをご検討のご担当者様や、新たな集客施策をお探しのご担当者様はこの機会に是非ご参加ください。

【講師の紹介】


秋山工(あきやま こう)
1992年クロスコ入社　ライブテクニカルディレクター。

キー局で報道に携わり、PKO法案後のカンボジアや阪神淡路大震災取材などで活躍。
その後テレビ番組制作に携わり音楽チャンネル開局にも助力。
映像伝送や配信に多く携わり、放送コンテンツから、企業・団体等の配信を含め年間50本以上の実績を誇る。

弊社は30年以上にわたり、TV番組の制作やスポーツ中継の技術サポート、 企業様の配信におけるテクニカルアドバイザーとして広くサービスをご提供しています。
