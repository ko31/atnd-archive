★★★当日の進行について★★★
当日は以下の流れで会を進行します。人数多ければ2グループに別れてディスカッションを行い、途中でグループを入れ替えます。
ディスカッションの中では皆さんに記事を1つずつ紹介していただきます。

1. 自己紹介
2. 第1グループでのディスカッション
3. 第2グループでのディスカッション
4. 振り返り

★★★ディスカッション時のお願い★★★
・一方的に話し続けたり、聞き続けるのではなく、他の方の意見を聴くようにお願いします。
・自分の得意分野の記事があれば専門知識を活かして記事よりも分かりやすく説明してください。不得意な分野の記事があれば積極的に質問して場を盛り上げてください。
