概要

これからのPRパーソンはどのようなキャリアを歩んでいけばよいのでしょうか？

PR会社、事業会社の広報部、広告会社のPR局など、少しでもPR業務に携わっている方必聴のセミナーを開催します！

メディア環境や消費行動が激変している今、マーケティング業務に携わる私たちの仕事の範囲も大きく変わってきています。

特にマーケティング活動においてのPRの存在感はこの10年で増していると感じる方も多いでしょう。

しかし一方で、PR業務に携わる人たちからは
「自分の仕事は、本当にその事業／商品の課題を解決しているのだろうか？」
「本質的に“人を動かすこと”　“売上に寄与すること”に貢献できているか疑問だ」
という声もしばしば聞こえてきます。

単なるパブリシティ（露出獲得）ではなく、いかに消費者、ステークホルダーに意識変容、行動変容を促すことができるか。
PRパーソンとして消費者視点＆Public Relations視点を中心としたストーリーを描くことができるか。

そこが重要な時代になってきているのです。

本セミナーでは、マーケティングPRの進化と今後の在り方について解説すると同時に、事業会社出身、広告会社所属、PR会社出身と違う出自でPR業務に携わる3人が、それぞれ今の立場からPRとの関わり方、これからのPRパーソンのキャリアビジョンについて語り合います。

エントリー時に登壇者に「聞きたいこと」を設けており、その内容の一部も質疑応答形式で行わせていただきます。抱えている疑問などをエントリー時にご記入ください。

貴重な機会となりますので、是非ご参加くださいませ。

PROFILE

株式会社インテグレート 代表取締役CEO
藤田康人（ふじた・やすと）氏
慶應義塾大学を卒業後、味の素株式会社に入社。
1992年、ザイロフィンファーイースト社（現ダニスコジャパン）を、フィンランド人の社長と2人で設立。1997年にキシリトールを日本に初めて導入し、素材メーカーの立場からキシリトール・ブームを仕掛けた。この結果、ガムを中心とするキシリトール製品市場はゼロから2000億円規模へと成長。
2007年5月、IMC(統合型マーケティング)プランニングを実践するマーケティングエージェンシー 株式会社インテグレートを設立、代表取締役CEOに就任。

著書に『ヒットを育てる！食品の機能性マーケティング』（日経BP社）、『カスタマーセントリック思考』、『The Real Marketing』（以上、宣伝会議）、『99.9％成功するしかけ』（かんき出版）などがある。

株式会社博報堂 PR戦略局 シニアPRディレクター
橋本真人（はしもと・まさと）氏
1994年に入社。企業や団体のコーポレートブランド領域、あるいは商品・サービスブランド領域に関する統合情報戦略の企画立案、実施、成果測定に至るPDCAの開発、運用、管理業務等に20年以上にわたり従事。最近はアカデミアとの産学官連携や社会実装事業等にも関与している。広告会社のPR戦略セクションに所属するため、狭義のPRから広義のPRまでを取り扱う。イシュー・リスクコンサルティング、トップ広報マネジメントから、デジタルデータドリブンマーケティングに至るまで、「PR発想」と「ブランド」を重視して、多岐にわたる事案に対応している。対外的な教育活動としては（公社）日本PR協会研修の講師等。

株式会社インテグレート 執行役員／統合ソリューション部 マネージャー
赤坂幸正（あかさか・ゆきまさ）氏
1969年宮城県石巻市生まれ。1992年大学卒業後、PR会社の株式会社コムデックスに入社。長崎県のハウステンボスを中心に、福岡や佐賀などを巻き込んだデスティネーション企画を推進するなど、マーケティングPRの基盤を築く。さらに、2000年初頭に「アセロラ」「ココア」「機能性ヨーグルト」など食品分野における機能性PRのプロジェクトに数多く参画。
2007年株式会社アイ・エム・ジェイとインテグレートのジョイント企業3iへの参画を経て、2008年からインテグレートに所属。アカウントプランナーとして、マーケティング課題解決に向けた戦略シナリオとコンテンツ制作、統合型エグゼキューションプランニングに従事。2009年より日本PR協会の講座で講師も務める。
