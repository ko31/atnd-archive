みなさま、ごきげんよう。
『青いバラ会』のお知らせです。
　
2012年11月11日(日)、京都にて『プリキュアイズム』が開催されます。
関東などからご参加予定で前日に関西入りされる予定の方々、
よろしければ梅田で素敵なビアレストランでおトークなどいかがでしょう？
もちろん関西プリクラの方々も。
　
「ワインのようにビールを楽しむ店。最高の焼鳥とともに。」なこのお店。
今回の『青いバラ会』は、こちらで関東・関西酒部交流をご提案させていただきます♪
　
　　
　
＜集合場所・時間＞
集合場所：阪急梅田駅 CO-BIGMAN下のコインロッカー前
(紀伊国屋書店のとなりです)　
集合時間：2012年11月10日(土) 18:20　
　
　

＜予算・コース＞
　
4,000円程度になるのでは、と思います。
個々人の好みで色んなビールを楽しめるよう、コースは無し、フリーで注文にする意向です。
これで均等に割るのは不公平なので、多めに飲んだ人ほど多めに払うと言う
べるな式気まぐれフレグランス勘定です！(✿ฺ-ω-)
　
　
＜参加登録〆切＞
　
新規登録およびキャンセルは11月9日(金)21:00までにお願いします。
　
　
＜その他＞
　
・個室を予約する予定です。それには最低6名必要です。
・個室が取れたらプリグッズを展開するのアリと思いますが、取れなかった場合、フロア席はプリグッズを出す雰囲気ではありません…(笑)
・焼鳥の大変おいしい所ですので、お酒がニガテな方、未成年の方(当然飲む・飲ませるはダメよ！)もいらっしゃいませ！
