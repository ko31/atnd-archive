今年もいよいよフェスティバル／トーキョーの季節がやってきます！
F/T12の目玉はなんといってもエルフリーデ・イェリネク三作連続上演！

そこで、イチゲキはなんと！無謀にも！イェリネクの勉強会主催しちゃいます！

イェリネク関連の作品（上演、小説、戯曲、映画etc）をひとつでも観賞した方だと、なお楽しめますが、興味があるけど、何から観たら良いかわからない、でも知りたい！という方も大歓迎です！

先だって発売されたイェリネク戯曲集『光のない。』がF/T上演作品を網羅しておりおすすめです！！

F/T12開幕直前の付け焼刃勉強会ですが、皆でイェリネクの情報や知識をちょっとずつ持ち寄って共有できたら、けっこう大きな知識になるのでは？と思います。

難解と言われているイェリネク作品ですが、今回もお酒片手にわいわい楽しくやりましょー。お気軽にご参加ください！

（※ちなみにイチゲキ内には普通イェリネクを語る際に必須である独文やらドイツ文学に造詣の深いような人はいません。専門性は正直ゼロです。素人なりのおもしろがり方をしたいと思っていますが、参加者にそういう方がいらっしゃったら、こんなに心強いことはない！ということで、いらっしゃればいいなぁ…募集中。）
