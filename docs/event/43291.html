追記情報：テーマ「出郷者に関する課題を解決する」に決定！

シビックハッカソンでは、テーマは技術的なものではなく、地域が抱える課題の解決を設定します。
今回のハッカソンのテーマは、「出郷者に関する課題を解決する」と決まりました。
近年多くの地域で課題になっている高齢化を形成する原因の一つに、若者が都心部へ出郷してしまい、地元に帰ってこないという問題があります。
出郷せざるを得ない状況には、地元には仕事が無い、刺激が少ないなどといったことがありますが、働き手が減ることにより、税収減やコミュニティの希薄化、活力不足といった問題が引き起こされます。
しかし、地元に仕事を作ったり、Iターンを促進する以外でも、出郷者に対して地域とのつながりをより感じてもらう、帰りたくなった時に受け皿を用意する、遠方からでも地元に何かできるようにする、などといった工夫により、状況を改善することができるかもしれません。

本ハッカソンでは、こういった問題を、テクノロジーの力で解決することを目指します。
こういったテーマに馴染みのない方でも問題はありません。イベントでは、地域でのコミュニティデザインを通じた課題解決に多くの実績がある、スタジオエルの出野さんに話題提供をいただき、皆で解決方法を考えるアイデアソンも行います。

＜出野紀子さん＞
　東京生まれ。日本大学芸術学部写真学科中退。2001年より英国ロンドンへ渡り、テレビ番組制作の仕事に携わる。NHKやフジテレビの情報番組、英国BBC、ITVのドキュメンタリーや音楽番組などを制作し、2006年よりTV番組制作ディレクターとして独立。
2011年にstudio-L参画後は島根県海士町の自主運営テレビ局「あま・コミュニティ・チャンネル」で放送局の運営に携わり、住民による番組づくりをサポート。issue+designでは「声による発想」のための取材インタビュー・映像制作を担当した。
現在は、九州、東北、中国地方で、中心市街地活性化や防災緑地建造のための住民会議運営などを担当している。2014年４月から東北芸術工科大学コミュニティデザイン学科着任予定。

イベントについて

Code for Japan と Mashup Awards のコラボにより、２日間のハッカソンを名古屋にて開催します！

シビックハックとは、市民(Civic)の持つ課題を、最新技術と、みんなの力で
解決することです。みんなで身近な課題を解決するITシステムを一緒に作ってみませんか？
Code for Japan では、日本全国で、そのような取組を支援します。

また、本ハッカソンは、Mashup Awards9 とも連携しており、優秀作品には、MashupAwardの１次予選免除の権利が与えられます。
つまり、Mashup Awards9 に応募するための作品を週末を利用して作ってしまおう、というイベントでもあります。その場で集まったメンバーでチームを作って頂くので、チームメンバーを募集されている方等はぜひご参加下さい！

良くある質問

  • 何をするイベントなの？
    アイデア・発想力を欲しているエンジニアやデザイナーの方と、
    技術力を欲しているプランナーの方の出会い（チーム結成）を生むイベントです。
    具体的には、参加者同士でチームを組んでいただき、街の課題を解決するサービスや
    楽しいアプリのアイデア出しから開発まで行っていただきます。
  • 開発できなくても平気なの？
    もちろん大丈夫です！企画・アイディアやデザイン・イラストなどでご参加ください。
  • LANや電源は使えますか？
    会場ではゲスト用無線LAN・電源が利用可能です。
  • どのような職種の方が来ますか？
    エンジニア・開発者・デザイナーの方が多いと想定しております。
  • 友達と参加してもいいですか？
    ご友人との参加も大歓迎です。同じチームになりたい場合、当日お申し出ください。
    ※ご友人の方もATNDで参加登録をお願いします。
  • 持ち物は？
    普段利用されている機材や開発ツールをお持ちください。
