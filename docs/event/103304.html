【イベントの説明】
Vue.jsを業務で使っている方、これから導入を検討している方同士で情報交換を行うための勉強会を開催します。
Vue.jsを導入してみて改善されたこと、うまく使うために開発したノウハウ、ちょっとした tips、今困っていること.. etc.
なんでもよいので気軽にお話できる場にしていければと思います。
基本はLT形式で発表ですが、ちょっとした質問や議論などを雑談形式で進めるのでもOKです。ぜひお気軽にご参加ください。
(初回は開催者のIGSエンジニアがLTします)

【内容】
夏：
- jquery をやめた理由について
- vue に関して構文、ドキュメント、学習コスト、フレームワークの豊富さ
- パフォーマンスについて react と angularと比較して
- 最近 typescript 対応した vuex を使った vue 開発

帯川：
- Vue開発のベストプラクティスとAtomicなコンポーネントデザインの極意 〜6ヶ月分の学びを5分で〜
