※ プレゼンテーション及び登壇者が以下に決定しました! 乞うご期待!!
  • 「eoID の OpenID技術活用」 by ケイ・オプティコム 山縣憲一さん
    　ケイ・オプティコムの統合ID＝eoIDの光ファイバーサービスを展開するケイ・オプティコムがユーザー向けの様々なIDを統合したのち、OpenID技術を活用したサービスを提供するまでの事例を主にビジネス面での切り口からご紹介します。
  • 「OAuth 2.0 update」 by @nov
    　ドラフトの段階で facebook や twitter など既に様々なサービスにサポートされている OAuth 2.0ですが、先ごろついに draft version 21 を迎えました。仕様の最新状況について、@nov さんが分かりやすく解説します。
  • 「OpenID Connect update」 by @ritou
    　次世代の OpenID、"OpenID Connect" に関する仕様の策定状況について @ritou さんが説明します。OpenID Connect はすでに Google や Salesforce.com, Microsoft などが次期アイデンティティ連携API技術として採用を決定しています。
  • 「Microsoft の RESTful Identity」 by @phr_eidentity
    　Microsoft のクラウド基盤の一つである Windows Azure AppFabric ACS を利用し、Facebook連携、OpenID Provider連携、OAuth を利用した REST Web Service連携を簡単なデモを交えて Microsoft MVP の @phr_eidentity さんが紹介します。

技術者向け無料セミナー OpenID TechNight Vol.8 in 大阪 を開催いたします。Facebook や mixi をはじめとする最近の Web サービス API を利用する上で必須となる「OAuth 2.0」、そして Salesforce.com や Google が相次いで実サービスに採り入れている OpenID の次期仕様「OpenID Connect」の最新動向や、ID プロバイダーの取り組みについてご紹介します。

8回目を数えます今回、初めて大阪で実施することができました。関西の Identity 愛好家、API 愛好家の皆さんにお会いできることを楽しみにしてます。

※ 会場キャパシティは70人程度ですが、ATND では40名の定員枠とします。
