筑波大学の辺りで『密告中』っぽい鬼ごっこのような遊びをみんなでやります。交通法規を守りながらやります。

ルールは後述。9/12 13:00-15:30の予定。人集まらなかったらずらします。

[ルール]

  • 参加者とGM（ゲームマスター。裏方）がいます。
  • 参加者は2人組を作ります。
  • 色つきの腕章と、数桁の番号が書いたゼッケンが配られます
  • ペアごとに散らばったあと、定刻になったらゼッケンをつけてゲーム開始
  • 誰かのゼッケンの番号を見たら、そいつの腕章の色とゼッケンをGMにメール
  • 色と番号が一致してたら、GMからメールがきて逮捕が告げられます
  • 逮捕されたらおとなしくGMが引き籠ってる本部までいきます
  • 15:30まで生き残っているか、ほかのペアが全滅したら勝ちです
  • 勝ったらうまそうな棒か何かがもらえると思います
  • 何か質問とかあったら言ってください

[参加条件]


  • ケータイを持ってる

  • 交通法規を守れる

  • 反社会的な思想を持たない

  • 転んでも泣かない

[日時・場所]

日時: 9/12 13:00-15:30の予定。人集まらなかったらずらします

場所: 筑波大学の辺り。詳細とか集合場所は後日
