第３５回勉強会を開催します。

今回は、前回のAndroidStudioに引き続き、ウェアラブルデバイスとして注目を集めている「AndroidWear」でのアプリ開発に関する勉強会を開催します！！
　AndroidWearをお持ちの方もそうでない方も、まずはどのようにアプリを作るか、どんなことができそうかなど学んでいただければと思います。
　講師の木下さんがAndroidWearを2台ほどお持ちのようなので、おさわりできると思います。

日　時：平成２７年２月５日（木）　１９：００～２１：００
場　所：ソフトピアジャパン　ドリーム・コア１F　モバイルコア
定　員：２０名
参加費：無料
内　容：「Android Wearでウェアラブルアプリをつくってみよう！」
　　　　　講師：木下　照章　氏　（株式会社グルーヴシステムズ　代表取締役社長）

　　　　　1. Android Wear 概要
　　　　　2. 仮想デバイスを作成する
　　　　　3. Android Wear の基本操作
　　　　　4. Android Wear用のプロジェクトを作成する
　　　　　5. アプリを実行する
　　　　　6. 通知を表示してみる
　　　　　※AdroidWearアプリ開発をハンズオンで学習します。

　＜事前準備＞
　ご自身のPCにAdroidStudio1.0以降をインストールしてお持ちください。
　PCのない方は申込みの際にその旨記載ください。モバイルコアのiMacを準備します。

　★ダウンロードサイト：https://developer.android.com/sdk/installing/studio.html

　＜環境設定＞
　Android Studio の SDK Manager で次のものがイントールされている状態にしておいてください。
　・Tools > Android SDK Tools Rev.24.0.x
　・Android 4.4W.2 （インストール可能なもの全て）
　・Extras >
　　Android Surpport Repository
　　Andrpod Surpport Library
　　Google Play services
　　Google Repository
