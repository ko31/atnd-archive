CakeBeer at Co-Edo（コエド） #cake_beer

イベント ユーザー

title CakeBeer at Co-Edo（コエド） #cake_beer
catch @cakephper さんと、コワーキングスペース茅場町 Co-Edo（コエド）で Let's #cake_beer!
description

CakePHPユーザーおよび、それ以外の方が集まってビールを飲むイベントです。

今回は、福岡の @cake_phper さんが東京に来ているタイミングで実施されます。
飲み会後、名古屋へ行くということで、少しだけ早めの時間帯から集まり次第始めます。

日時

2013年1月18日（金）18時〜

  • 参加できる時間にいらしてください

場所

コワーキングスペース茅場町 Co-Edo（コエド）
東京都中央区新川1-3-4 PAビル5F

  • 上記の地図ですとずれて表示されますが、永代通り沿いのビルです

東西線・日比谷線茅場町駅から徒歩2分。永代通り沿いのビルです。
茅場町駅3番（5番）出口をスターバックス方面へ約100mです。
（カフェベローチェの隣のビルです）

水天宮前駅や日本橋駅、八丁堀駅も徒歩圏内。
東京駅へも約1.5km。

http://blog.coworking.tokyo.jp/p/access.html

飲み会の内容

COEDOビールや缶ビール・サワー・シャンパン等を飲み、ピザ・お菓子等を食べながら、PHPに関係あることないこと話しながら、ゆるーく交流しましょう！

  • COEDOビールは早く来た方限定となると思います

@cakephper さんは21時半までのようですが、会場自体は終電過ぎても使えます。

会費

3,000円程度（会場代込み）
※ 当日早めにきてコワーキングスペースとして使っていただいて構いません

ご連絡

  • 飲食店ではありませんので、ゴミ捨てや片付けにご協力ください
  • facebookでも募集 しますが、参加の申し出はどちらか片方だけでも構いません（両方登録してもらえると助かります）
  • PAビルの地下に深夜まで開いているバー( coal black )があります。併せてどうぞ
  • 同時開催 JELLYWEEK Friday night Fever（中継飲み会）
