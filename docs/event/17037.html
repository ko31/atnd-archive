MITの 6.828: Operating System Engineering という授業の講義資料を読み解きながら、OSへの理解を深めようという会です。

この授業では、UNIXv6を参考にx86アーキテクチャ上でgccを使って新たに実装したxv6というOSを用いてOSとハードウェアの基礎を教えています。
このOSや授業資料についてのまとまった日本語資料というようなものは世の中に存在していないので、日本語で資料をまとめてOS学習の敷居を下げようというのが最終的な目標です（が、取り敢えずは読んだり動かしてみようというスタンスで挑んでいます）。

今のところ、東京工科大学 田胡研究室内で細々とした輪講として行っていますが、そのうち外部でやりたいなと思ってます。
