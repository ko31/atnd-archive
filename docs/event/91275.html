MySQL 8.0 RC リリース記念

2016年9月にMySQL 8.0.0がリリースされてから一年を経てリリース候補版になりました。MySQL 8.0.3 RCのリリースとMySQL開発チームのMorgan Tockerの来日に合わせてMyNA(日本MySQLユーザ会)会を開催いたします。

MySQLプロダクトマネージャーの話を直接聞ける/ディスカッションできる貴重な機会ですので、是非ご参加下さい！！

MySQL 8.0について

MySQLサーバーの次期メジャーバージョンとなります。
変更点や追加機能の一部は下記です。

  • データディクショナリでのシステムメタデータの管理
    • .frmファイルなどのフラットファイルやMyISAMテーブルでのシステムデータ管理廃止
  • ロールの追加
  • キャラクタセットと照合(COLLATION)の改良
    • デフォルトのキャラクタセットがutf8mb4に
    • 日本語用の照合追加
  • MySQLドキュメントストアがデフォルトで有効に
  • CTE、Window関数の追加

Morgan Tockerについて

Morgan TockerはMySQLプロダクトマネージャーとして、MySQLサーバーのロードマップ策定や開発管理を担当しています。以前はOracleのMySQLコミュニティチームの一員として、MySQLコミュニティのイベントやセミナー、ブログでの情報発信、各種の交流機会を通じてMySQLの普及拡大とコミュニティの発展に貢献してきました。
