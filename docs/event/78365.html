オフラインでゆるくオーバーウォッチを遊ぶ会です

PS4版 オーバーウォッチを遊びます。
モニタ、PS4はこちらで用意しますが、ゲームソフトについては参加者の方にご用意いただきます。
※お持ちでない場合は、主催者のアカウントで、順番に遊ぶことになります。

初心者から上級者、オーバーウォッチを遊んだことがない方まで、気軽にご参加ください。
※ちなみに主催者はまだまだ全然な初心者です。

参加者が6名以上の場合は、2チームに分けそれぞれでネット対戦、もしくはプライベートマッチをおこなう予定です。

また、休憩スペースにスト5も用意する予定です。
主催者は超初心者ですが、こちらも一緒に遊んでくれる方を募集いたします。

■概要

日時: 6/11 (土) 13:00 ～ 17:00ごろまで

イベント終了後、希望者がいれば、近場で食事に行きたいと思います。

場所: JR 山手線 高田馬場駅 より徒歩10分
　　　東京メトロ 副都心線 西早稲田駅 より徒歩5分
　　　東西線 早稲田駅 より徒歩5分

定員: 10人前後

参加費: 無料
