★講演内容

TVCMでおなじみの日本最大級の不動産・住宅情報サイト
『HOME’S』を運営する株式会社LIFULL。

現在『HOME’S』は、総掲載物件数・使いやすさにおいて
国内No.1の住まいの情報サイトです。※1※2

人生の中で重要な選択肢の1つである住まい選び。

『HOME’S』は、最上の住まいに出会えるよう、
最大の情報と最良の情報品質、最適な手法によって
住まい探しをお手伝いするプラットフォームです。

ユーザーへより良いサービスを届けるために
新技術の導入から新サービスの開発、
新機能の追加なども積極的に行っております。

今回は、Google Play「2016年ベストアプリ」に選ばれた
Androidアプリの開発の裏側についてお話します。

現場で活躍するエンジニア社員2名が
・開発環境や体制
・マテリアルデザイン対応
・マイクロサービス化（API Gateway＋Lambdaを活用したサーバレス化）
についてお話させて頂きます。

Androidアプリ開発に携わっている方、
『HOME’S』に興味がある方、
など様々な方のご参加をお待ちしております。

※1 産経メディックス調査（2017.1.26）
※2 Gomez賃貸不動産情報サイトランキング 2017年2月

※ 内容は変更になる可能性があります。


★講演者プロフィール

■株式会社LIFULL
　寒川 明好 氏

2010年　エンジニアとして新卒入社。
入社後、HOME’Sのウェブサイトの開発を担当。
その後すぐにAndroidアプリやiOSアプリ、
Windows Phoneアプリの開発に従事。

現在は、技術マネジャーとして
HOME’S Androidアプリの開発・運用を推進。

■株式会社LIFULL
　衛藤 剛史 氏

2014年　SIerでの開発経験を経てエンジニアとして中途入社。
入社以降、HOME’SのAndroidアプリ開発を担当。
Androidネイティブアプリの開発を行うと共に、
アプリ用APIのマイクロサービス化による
AWS Lambda / API Gateway等を活用した
バックエンドの開発も行っている。

今回はAPIマイクロサービス化について詳しく紹介します。
