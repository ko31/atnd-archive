真剣徳大しゃべり場 ～第四弾、夏～

2011.7.14(木) 16:20～18:00 @徳島大学 4号館302教室

この夏、あなたは何がしたいですか？

いよいよ、夏休みが目前に迫ってきました。

2ヶ月間の長い大学の夏休みを、あなたはどのように活用しますか？

「この夏、新しいことに挑戦したい…！」

私たちしゃべり場企画チームは、
そんなあなたの 「この夏、やりたい！」 を見つけられる出会い・交流の場を提供します！。

真剣徳大しゃべり場は、学生と教員が同じテーブルにつき、
お互いを あだ名 で呼び合いながら、
フリートーキングを楽しむ企画です。

今回のしゃべり場のテーマは、
「この夏、あなたは何がしたいですか？」
というものです！

ぜひご参加ください！

真剣徳大しゃべり場ってなに？

真剣徳大しゃべり場は、教員を交えた学生同士のグループトークの場です。
グループは8人前後で構成され、
学生が務めるコーディネーター（進行役）が話を進めていきます。

テーマに沿って、自由に話を広げていきましょう。

【真剣徳大しゃべり場でできること】

1. あなたの「この夏、やりたい！」が見つかる！
2. 徳島大学で頑張っている学生・教員と出会える！交流できる！
3. 人の話を聞く力・人に考えを伝える力が身につく！
4. 互いに刺激を与え合い、日々のモチベーションがアップ！

さあ、学生も教員も、
しゃべってしゃべって、しゃべりまくれ！

対象者

大学生・院生・教職員
