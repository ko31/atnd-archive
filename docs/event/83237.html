description

　Web開発エンジニアの採用強化を目的に、名古屋へのUIターンを検討されているWeb開発エンジニア向けの中途採用説明会＆面談会を東京スタジオ（品川）にて開催致します。

　当日は、エイチームのWeb技術統括より、事業内容、エイチームが提供するサービス、実際の開発環境などをエンジニア目線でお伝え致します。また、希望者はそのまま面談を実施させて頂きます。

　エイチームにご興味をお持ちの方は、この機会にぜひご参加ください。

■「将来は名古屋で…」とお考えの方へ
　今回は、名古屋へのUIターンをご検討されている首都圏在住の方を対象に、中途採用説明会・面談会を開催いたします。

＜こんな方におすすめ＞
・“将来”名古屋へ移り住もうと考えている方
・東京で生きること・働くことを見直したい方
・家族ができたので、首都圏を離れて子育てをしたい方
・名古屋・東海出身の方
・学生時代に名古屋に通っていた方
・大事にしたい人（家族・恋人）が名古屋にいる方
　※イベント当日は、皆さまが安心してUIターン頂けるよう、名古屋での働き方や暮らし方についてお話させて頂きます。
