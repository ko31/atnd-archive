description

日本語教師は、国内＆海外共に、今が売り手市場！
「日本語を日本語だけで教える方法」があるから、語学力を気にせず、
やる気と勇気があれば、どなたでもできます。

みなさんは、日本に住んでいる外国人が たった１年で１１万人も増えている ことをご存知ですか？

また、海外では、 世界１３６か国で、約４００万人の人たち 日本語を勉強していることを知っていますか？

一方、日本語教師はというと・・・、 約７０％の教育機関が「教師不足」 と答えています。

そうです！
今「日本語の教師」が、日本で、海外で求められているのです！

そこで、この講座では、外国人に日本語を教えることにご興味を持たれた方を対象に、
「日本語教育の基礎知識」から「日本語教師の頭の中」まで、日本語教師を 目指す前に
知っておきたい「４つのこと」 を、現役教師が分かりやすくお話します。

「日本語教師」は、日本人である「自分を活かした」国際的な仕事です。
あなたも、この夏、この講座で第一歩を踏み出してみませんか？

語学力・年齢・職業は一切問いません。「興味がある」が参加資格です。
