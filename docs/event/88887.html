Google社と共催で、少人数制のGoogle Maps API 活用ワークショップを開催します。
本ワークショップでは、Google Maps をベースに、どのような仕組みがあればもっと効率よくビジネスを進めることができるか、参加者同士でアイディアを出し合って、イメージを膨らませていただきます。
Google、ゴーガの担当も一緒に検討していきますので、Google Maps APIは初めてという方も安心してご参加ください。
また、弊社の開発事例の紹介や、Google Maps APIs Premium Planについても詳しく解説します。

ぜひ、Google Maps APIで位置情報の活用をはじめてみませんか。
皆様のご参加を心よりお待ちしております。
