====== あなたには叶えたい夢はありますか？ ======

もし叶えたい夢なんて持っていない、夢は夢だから叶うはずがないと信じ続けたいなら、この先は読まないで下さい。

『思考が現実化する』、『ザ・シークレット』、『マーフィーの黄金法則』などの夢や目標を叶えるための本は、数多く出ています。あなたも本屋に行けば、その類の書籍をたくさん見掛けたことがあるのではないでしょうか。

叶えたい夢を持っているあなたも既にそうした本を何冊か読んだことがあるかもしれません。

様々な成功法則の本を著名な方たちが執筆しています。色々な体験談を基にした内容のものが多いようですが、そのほとんどの本に共通して書かれていることがあります。

それは夢（目標）を明確にするということ。
-————————————————————————————

夢（目標）を明確にすることは、本当に大切なことです。

『夢は何ですか？』

『達成したい目標は、どのようなものですか？』

この質問に答えられない人は恐らく、夢や目標を叶えることは難しいでしょう。

例えば、歩いて近所のスーパーまで歩いていこうと思えば、その近所のスーパーが『目標』となります。そして何時までにいくかという『期限』を決めるのです。
日常であるこの程度のことならば特に意識せずとも出来てしまうので、人は真剣に考えることはありません。日常的には、これらのことを無意識のうちに脳が処理しています。

しかし、叶えたい夢や達成したい目標は、非日常的なものであったり、今の自分自身では現実的では無いものが多くを占めています。

日常では何とく行動していても、いざ、夢や目標が非日常的なことや現実的だと行動することが出来なくなってしまいます。
人は未だ経験していないことには無意識のうちに恐れを抱きます。

人は、自分がこれから行動することが必ず出来ると言う自信、そして勇気が無ければ行動を躊躇うように脳が信号を発信します。

その自信、そして勇気を持つことがあなたの夢（目標）を叶えるための鍵なのです。

そしてその夢（目標）を叶えるための鍵とは、あなたがあなた自身をどう思っているのかというセルフイメージなのです。

====== 同時にアクセルとブレーキ！？ ======

いつも『いつか幸せな結婚をしたい』と言っている男性がいます。もう年齢は40半ば・・・そして、その男性の話をよくよく聴いてみると・・・

「結婚すると自由が無くなる」

「私は結婚には向いていないかも・・・」

「そもそも、私には結婚相手が現れるはずがない・・・」

幸せな結婚という目標に進みたいのに、これでは、もう一人の自分が後ろから服を引張っているようなものです。

まるで、

「お前には無理だ」

そんな声を発するかのように・・・

車もアクセルとブレーキを同時に踏んでいては前に進むことが出来ません。それどころか、スピンしてしまうこともあります。

このブレーキを踏んでいる足の力を緩め、アクセルを踏み込むために多くの方がＮＬＰを学んでいます。

======
ＮＬＰを学ぶとどうなるのか！？
======

ＮＬＰを学ぶキッカケは人それぞれですが、「自分自身を変えたい」という思いが共通してあるようです。

ＮＬＰの学びの中に「モデリング」という優れた他者の考え方や行動を力を自分の中にインストールしていくワークがあります。まさにお手本となる人物と同じことが出来るようになっていき、自分のセルフイメージも変えていくことが出来ます。

そして自分のセルフイメージを書き換えることで、自分の望む夢に向かってアクセルを踏み込むことが出来るようになっていくのです。
そのほかにもＮＬＰを学ぶことで、以下のようなことが可能になります。

□思考・感情のコントロール　　　
□他者との関係性の向上　　　　　
□夢を引き寄せる行動力　　　　　
□自分の信念・ビジョンの構築　　
□周りへの卓越した影響力の向上　

また、自分の心理の内面にもアクセスしていくことも可能になります。自分のビジョンをイメージし、実行することが容易になる、自分の価値観を見つめ直し、ブレない自分を作り上げることが出来たという人もいます。

幸せな結婚を望んでいた彼・・・今では心優しいステキな女性と結婚し、待望の子供も生まれ、幸せがいっぱいあふれ出るような家庭を築いています。

さらに自分の夢だったビジネスを立ち上げ、キラキラと輝いた人生を歩んでいます。

そして、その彼とは・・・実は私自身のことです。

ビリーフ・チェンジ・トレーニングでは、信念・価値観の強化をすることに焦点を絞り、叶えたい夢や目標をつかむためのセルフイメージ、自己信念の強化、価値観の掘り起こしを促進します。

【こんな方におススメ】
・人間関係にストレスを感じている　
・自分に自信が持てない　　　　　　
・もっと行動力をつけたい　　　　　
・ブレない信念を持ちたい　　　　　
・叶えたい夢（目標）がある　　　　
・影響力のある人間になりたい　　　

繰り返し申し上げると、ＮＬＰを学び、日常で使うことにより、自分自身への自信をより強固に持てるようになり、行動力が身に付きます。行動力が身に付くと、夢や目標を叶えると言うよりも、向こうから歩み寄ってくるような感覚を持てるようになります。
