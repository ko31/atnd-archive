デバイス・通信 を用いたUnityゲーム開発 （とクエリちゃん情報）

  • 発表者 佐々木宣彦さん(株式会社ポケット・クエリーズ)
  • 概要
    kinectやOculusなどのデバイスを用いたUnityゲームの製作方法を、実装サンプルを元に説明します。併せてNetworkViewを用いた同期通信の仕組みについての説明、活用事例の説明を行います。
    また、最近リリースしたSD版クエリちゃんアセットやクエリちゃんの今後の計画について紹介する予定です。なお、時間があれば、Unity社が最近提供を始めたサービス（EveryPlayやUnity Ads）の実装についてもお話します。

Microsoft Azure の Iaas 機能メインで一言語っておくか ～色んな誤解を解く編～

  • 発表者 割と普通さん(@normalian)
  • 概要
    Azure に限った話ではないですが、クラウドサービスの進化は非常に早く、「昨日できなかったことが今日できるようになる」なんてことは良くあることです。Azure 以外のクラウドサービスをキャッチアップしている方々にお話を聞くと「Azure は○○が弱い（それもうできるよ！な機能、特に IaaS 系に多い）」というお話を聞くことが多々あります。今回は未だによく聞く誤解を解きつつ、Azure の素敵な使い方を紹介します。

プロ生を支える生放送技術

  • 発表者 みゆっきさん(@toriimiyukki)
  • 概要
    プロ生勉強会でも欠かせない生放送、スマホから本格的な生放送まで。そんな生放送を支える裏側を配信者、サーバ、視聴者など各視点に着目した最近の生放送技術と共に解説します。また、余裕があれば最近ハマっているFPGAの話題も交えながらお話します。

DMM.com ROBOTSで扱うロボットプログラミング入門

  • 発表者 株式会社DMM.comラボ
  • 概要
    DMM.comロボットで扱う最新ロボットのプログラミングについてご説明致します。
    ロボットのプログラミングの歴史について60年代から最新のトレンドまでをご紹介し、ロボットプログラミングに関して実例を交えてご紹介いたします。
