description

このような時代だからこそ輝く、年齢や性別、職業を超えて役立つスキル – それは「コミュニケーション力」と「リーダーシップ力」です。
世界110ヶ国、26万人以上の会員が存在する、米国のスピーチ教育NPO「トーストマスターズクラブ」では、参加型のスピーチ学習会を通して、誰もが手軽にこれらのスキルを楽しく学べる環境を85年以上にわたり提供しています。

東京メトロポリタントーストマスターズクラブは、東京・都心地域のクラブで、英語と日本語の両方で開催(第2土曜 – 日本語メイン、第4土曜 – 英語メイン)。英語スピーチによって、英会話教室にはない、英語を実践的なツールとして用いる練習ができます。さらに、母国語(日本語)のスピーチ練習により、論理的な話の構成の仕方をはじめ、言語を問わず人前で魅力的に話すための基礎体力を効率的に磨くことができます。また、多種多様な職業や年齢層の会員が集まるため、仕事とは違った出会い、コミュニケーションの場にもなります。

月2回土曜朝、千代田区(麹町・九段・神保町エリア)の公民館で開催。例会後、任意参加のランチもあります。ご見学は無料、入会金4,000円、会費月1,000円で、土曜午前を有効活用したい社会人や学生の方を主な対象としています。

詳しくは、ホームページをご覧ください。
