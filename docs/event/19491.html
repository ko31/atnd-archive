開催概要

関西のJavaエンジニアのためのコミュニティ「関ジャバ」による
ナイトセミナーを開催します。

今回は 「BTS/ITSツールのRedmineの導入事例」 や、
「ペアプログラミングを実践投入した成功事例の共有」 など、
開発プロセスに関連したセッションを中心に実施します。

その他、 東京に一週間滞在して勉強会に参加した苦労話（？） や
おなじみJava界隈の動向が分かる 「Javaニュース」 も継続して実施します。
Javaエンジニアの方も、そうでない方も、ぜひ奮ってご参加ください。


また、セミナー終了後に、セミナー会場にて簡単な懇親会を予定しています。
こちらも、ぜひご参加ください。

セッション概要

1. Redmineの実業務における活用事例紹介

中村 洋 ( @yohhatu )

これまでいくつかのPJでRedmineをチームで使ってきました。
Redmineはあくまでツールで、それぞれのPJやチームに特徴があり、
それに合わせた工夫が必要になってきます。
今回は実際のPJ、チームで使う際に工夫したこと、得た成果や課題をお話します。
補足：「RxTstudy」のほぼ再演です。

2. 勉強会にいこう。東京の。

野崎 啓史 ( @irof )

東京の勉強会が気になる、行ってみたい。けど二の足を踏んでる。
そんな方の背中を押してみます。

3. ぺあぷろ！

稲富 研人 ( @inaton )
谷口 光 ( @tanigon )

数あるアジャイル開発のプラクティスの中でも
特にペアプログラミングに焦点をあてて、
導入から8年になる自社での導入の歴史、近況をふまえつつ、
効果や導入方法、政治的問題などについてお話いたします。

4. Javaニュース

この1～2ヵ月の間に起きたJava界隈のニュースを、
技術ネタ、宣伝、ゴシップ織り交ぜてお伝えします。
今回は、懇親会をやりながらのセッションとします。
