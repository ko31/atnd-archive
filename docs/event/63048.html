学生主導によるAzureの勉強会を名古屋で開催します。

AzureでWebアプリやWebサイトを公開してみたいという学生の方、ぜひ参加してみてください。

Azureのことを知らない方も大歓迎です。

※参加費、懇親会参加費は無料!

内容

今回の勉強会ではAzureに関する講義のほかAzureのクーポン券[※1]を利用した演習を行います。

今回はAzureの機能のうち仮想マシンとWebsitesを中心に取り扱い、Azureの機能を理解するとともに、自作のWebサイトをAzureで公開する手順を理解することを目的とします。

※1Azureのクーポン券について

マイクロソフト社様からAzureを毎月15500円分(最大3年間継続可能)を無料で利用できるクーポン券を提供して頂いています。

Microsoft Azureとは

マイクロソフト社が提供を行っているクラウドプラットフォームです。

最近では日経BP社主催「第10回クラウドランキング」のクラウド基盤サービス(IaaS/PaaS)部門で1位に輝きました。 [リンク]

こんな人におすすめ

●Webサービスを作成・公開したい方

●Azureに興味がある方

※今回の勉強会は学生（高校生、高専生、専門学校生、大学生、大学院生）を対象にしています。
