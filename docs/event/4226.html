中之島まつりで、ヒーローショーが登場！

毎年、ゴールデンウィークに大阪の中之島公園で開催される「中之島まつり」
今年で、39回目を迎えます。
そして、「Ａ-yan!!　関西をアートで盛り上げるＮＰＯ」が参加させていただくことになりました。

【タイトル】第39回　中之島まつり
【日時】2010年５月３日(月・祝)、４日(火・祝)、５日(水・祝)　雨天決行
【場所】大阪市北区　中之島公園一帯
(大阪市庁舎周辺から天神橋剣先公園まで東西約１kmの公園全域と
車道部分を歩行者天国として開催)

【内容】
毎年、５月３日から５日に中之島公園で行われる、日本最大級の市民フェスティバルです。
ライブや模擬店、芸術系企画が満載です。

【最寄り駅】
大阪市営地下鉄または、京阪本線「淀屋橋」駅「北浜」駅
京阪中之島線　「大江橋」駅「なにわ橋」駅

【料金】入場無料
【主催】中之島まつり実行委員会　実行委員長　藤尾律
【ＨＰ】http://www.nakanoshima.net/

私たち、「Ａ-yan!!」も参加させていただくことになりました。
「アート戦士エーヤンダー５」公演行います！

「アート戦士エーヤンダー５」公演
【場所】青空トライランド　あそび銀行エリア
【時間】13:50から(約30分)
(中之島中央公会堂から少しだけ離れた場所です)

「アート戦士エーヤンダー５」は
道徳ヒーローショーして、関西で100箇所以上で公演しております。
今年はさらに、パワーアップして、大人も子どもも存分に楽しめるショーを行います。
