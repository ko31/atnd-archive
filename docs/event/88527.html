TDD Boot Campとは

「TDDとは何か？」という基本を学ぶ座学と、実際に手を動かすワークショップを行うことにより本を読むだけでは掴むことが難しいTDDのリズムを体験し、TDDに対する理解を深めていただくためのイベントです。

次のような方々がTDDについて"学びあえる"場を提供します。

  • テストコード書く意味がいまいち理解できていない
  • TDDに興味はあるが、なかなか実践できない
  • 自動テストを書けるスキルを身につけたい
  • テスティングフレームワークを十分に活用できていない感がある
  • 実際にTDDでソフトウェアを開発したい・見てみたい・体験したい
  • 他のエンジニアはどうやってテスト書いているのか見てみたい・気になる
  • 実際の現場で使えるTDDに関するヒントを見つけたい
