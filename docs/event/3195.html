「ブラウザー勉強会」は東京近辺で開催する Web ブラウザーについての勉強会です。
プレ開催として以下の内容で勉強会を行います。

時間表
- 10:00 会場準備開始 (手伝っていただける方、大歓迎)
- 11:00 ころから
　　ブラウザー勉強会の今後を考えるディスカッションみたいな
- 12:00 ～ 12:30
　　お昼休み (たぶん会場には誰もいなくなります)
- 12:45 受付開始
- 13:00 開会
　　 主宰者あいさつ「ブラウザー勉強会の開催にあたって」
　　 参加者自己紹介
- 14:00ころから
　　セッション 1 : (元)コミュニティメンバーから見たMozilla/Firefoxの歴史と展望 (松澤太郎 aka btm (id:smellman))
- 15:00ころから
　　セッション 2 : 「IEBlog から見える IE の現在と将来」 (hebikuzure)
- 16:00ころから
　　セッション 3 : ライトニング トーク セッション
　　    * 「プライベートブラウジングとメモリ解析」Oro さん
　　    * 「ブラウザのひみちゅ」ととろさん
- 17:00 閉会
- 18:00ころから 懇親会

参加費としてお一人 1000 円のご協力をお願いしています。
参加費は会場費、備品利用料その他の開催経費に充てます。

参加申込をされる方は、差し支えなければブラウザー勉強会グループ (Google グループ)にもご登録ください。
　http://groups.google.com/group
今後のお知らせはブラウザー勉強会グループで行います。

hebikuzure がウォッチャーとして Internet Explorer についての話をし、btm さんに Mozilla/Firefox についての話をしていただきます。
またライトニング トークで登壇していただける方も募集しています。テーマはブラウザーに関するものであれば何でも結構です。
ご協力いただける方は参加申し込みの際にコメント欄にご記入ください。

当日午前 10 時から会場の準備と簡単なリハーサルを行います。ご協力いただける方はその時間に直接会場にお越しください。また準備後に時間があれば今後のブラウザー勉強会の開催についてのディスカッションをしたいと思います。

なおイベント終了後に金春本館にて懇親会を予定しています。参加希望の方は以下から登録をお願い致します。
