開場：14 : 30/開演：15 : 00（予定終了時刻17 : 30）

バガヴァッド・ギーター　勉強会

【ギーター』は宇宙論、人生論を含む高度な哲学書です。
インドの叙事詩『マハーバーラタ』の一部であり、ヒンドゥー教徒の間で最も敬われている聖典の一つです。
バガヴァッド・ギータの教えに従うならば、人は人生におけるあらゆる不幸や心配から救われる。人はこの世ですべての恐怖から解放され、来世は精神的なものとなるだろう。と、言われています。
バガヴァットギーター　勉強会はスワミジが私達に分かりやすい言葉で、ギーターの素晴らしい教えを説いてくれます。

バガヴァッドギーター抜粋ーーーーーーーーーーー

バガヴァッドギーター　6章16節
あまり多くを食べすぎてはいけない
また小食にすぎてもいけない
眠りすぎも睡眠不足もいけない

バガヴァッドギーター　6章9節
自分に好意をよせるものも冷淡なものも
友人も敵も嫉妬深いものも
修道者も罪人も　どこにも属さない人も
みな平等に観る人は誠に進歩している
