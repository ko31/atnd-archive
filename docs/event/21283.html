iOS開発者向け勉強会＠仙台 第三弾を開催します！

内容

来たる11月3日（木・祝）に、iOS開発者向けの勉強会を開催します！今回はiOS5リリースにつき、OS5の内容を中心にやりたいと思っております。

勉強会と言いつつも、情報交換（雑談）時間が1時間もあるのが特徴です。セッションもライブコーディングを見つつ話をしたりと、ユルめの会になっております。お気軽にご参加ください。

タイムテーブル

現時点で、タイムテーブルはおおむね以下のようになっております。


  • 自己紹介・情報交換（雑談 / 世間話 / 井戸端会議）（1h）

  • Session1：iOS5 twitter APIについて ライブコーディング by @tototti （1h）

  • Session2：未定・発表希望者、募集中です！ もし発表がなかったら、参加者の方のアプリ紹介をしていただこうかな、と考えています^^

  • 次回の勉強会についてご相談・懇親会のご案内（10分）

参加される方へ

・MacBookなどをお持ちの方は、ご持参下さい。（コーディングを行う時間があるかもしれません）
・Pocket-Wifiにてネット環境を用意しますが、接続数に限りがあります。emobile等をお持ちの方は、ご持参下さい。

懇親会

今回の懇親会は、Androideaもくもく会のメンバーさんと合同の飲み会です！ぜひご参加ください！
