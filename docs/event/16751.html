セッション情報

セッション１：Android compatibility Fragment API

  • 講演者

    • 寺園聖文 (@yokmama) – 株式会社Re: Kayo-System


  • 概要
    • Fragmentは、最新バージョンのAndroid 3.0(Honeycomb)で新しく追加された要素です。
      本来であれば、2.3などの2系のバージョンではFragmentを扱うことが出来ませんが、Googleから提供されているcompatibility Fragment APIライブラリを使うことで、3.0以前のバージョンでもFragmentを使うことが出来ます。
      本セッションでは、Fragmentの基礎と、compatibility Fragment APIを使う方法を解説します。

セッション２：AOSPにContributeしてみた – 妄想編

  • 講演者

    • 有山圭二 (@keiji_ariyama) – 有限会社シーリス


  • 概要
    • Androidは、ソースコードを公開しているオープンソース・ソフトウェアです。
      AOSP（Android Open Source Project）にサブミット（送付）したパッチの内容と、それらのマージ状況を振り返りながら、コミッターとのやりとりを通じて得られた知見や、各社のサブミット、マージ状況について、個人的な考察を交えて解説します。
