はじめてのAndroidとクラウド連携（@kponda）

Androidとクラウド連携を行うにあたり、クラウド側の使い方についてGoogle App EngineとAmazon EC2のはじめ方を少々。

Android 4.0でのアプリの作り方（@tappuri_460ml )

Android 4.0 (IceCreamSandwich)になって、スマートフォン用のOSとタブレット用のOSが統合されました。Android 4.0で使えるようになった機能など、アプリの作り方についてのお話。

GoogleMapとGPS機能を使ってみた (つるつる)

GooglePlayで公開中のアプリ「原発からの距離は？」で利用している、
”GoogleMap”と”GPS”機能について整理してみました。

ピコピコゲームエンジンを作ってみた（@yamagame）

Lua/Box2D/FlMMLを使って遊べるゲームエンジンのデベロッパープレビュー。
Luaのソース一つでiOS/Android/Windows/MacOSで遊べるゲームが作れます。

Androidの画面デザインとかグラフ描画とか（@kponda）

ボタンに画像を貼り付けたりして、グラフを書いてみたりする方法についてのお話。
