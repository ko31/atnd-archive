【東京】アピアリーズ × アプリカン Developers Workshop for ハイブリッドアプリ

イベント ユーザー
event_id 60466
title 【東京】アピアリーズ × アプリカン Developers Workshop for ハイブリッドアプリ
catch ～BaaS「アピアリーズ」と、ハイブリッドアプリ開発環境「アプリカン」で簡単アプリ開発！～
description

イベントの概要：

モバイルアプリのバックエンド共通機能をAPIで提供するBaaS「アピアリーズ」と、HTML5とJavaScriptでAndroid、iOSを始めとするマルチプラットフォームのアプリを開発できる「アプリカン」をハンズオン体験いただくワークショップです。

共催：

株式会社アピアリーズ
株式会社インターネットイニシアティブ
株式会社ニューフォリア

対象受講者：

HTML5でのWebサイト/Webアプリの開発経験者。
※デザイナー/コーダー/ディレクターの方も歓迎です。

ワークショップ概要：

ワークショップ形式でアプリカンを用いたアプリ開発の実践講義を行います。
HTML5とJavaScriptでAndroid、iOSで動くアプリが間単に作成できます。
またBaaSサービスの「アピアリーズ」を用いてデータベースと連携したアプリの実装も行います。

ワークショッププログラム内容：

1. IIJのクラウドサービス「IIJ GIO」のご紹介

2. アピアリーズの概要紹介

BaaS「アピアリーズ」を利用するメリット
アピアリーズの利用事例

3. Developers Workshop

＜第一部＞　アプリカンのアカウント作成、概念の説明、環境設定

1. アプリカンとは
2. アプリカン入会
3. アプリカンで新規プロジェクト作成
4. アプリカンシュミレーターのDLとAPI Kitの動作確認
5. サンプルアプリソースファイルのDL
6. ワークショップで作成するアプリの説明と実機デモ

＜第二部＞　アピアリーズのアカウント作成、操作

7. アピアリーズのアカウント作成
8. アピアリーズ管理画面の操作、概念解説

＜第三部＞　アプリカンとアピアリーズを利用したアプリ構築

9. アプリカン：スライドメニューの利用
10. アピアリーズの接続設定
11. アプリカン：カメラ機能の呼び出し
12.　JSON Data API／ファイルAPI でのデータ登録
13.　JSON Data API／ファイルAPI でのデータ表示
14.　各種アプリカン機能の紹介
15.　質疑応答

　また、当日の進行状況によってはすべての内容を網羅できない可能性がありますのでご了承下さい。
