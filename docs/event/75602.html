description

マンガを中心に電子書籍市場が急成長を続けている今、その背景を担うテクノロジー領域において何が起こって いるのか。 グローバル化の進む近未来の電子書籍業界について、メディアドゥの役割とともに解説します。

かつて活版印刷という革新的な発明によって、人類は多くの情報を記録・共有する事ができるようになり、その 媒体として「書籍」が出版されるようになりました。 昨今の高度デジタル潮流の中で「書籍」もまた「電子書籍」として大きな技術的転換点を迎えており、その中で 電子書籍流通の中核を担う当社がどのように現状を捉え、未来をどう作り上げていくのか、戦略・技術の両面か ら解説します。

前半は、
・電子書籍はこれまでどのように成長してきたのか
・電子書籍が今後どうなって、あなたの生活をどう変えていくのか
など電子書籍のトレンドを当社の戦略等を交えながら解説します。

また後半は、
エンジニア視点から考えた「電子書籍」として、利用者・開発者という立場からの考察と、当社が月間1.6億件 という決済・購読を支える技術の解説などを踏まえながら、当社の技術的バックボーンを語ります。

★講演内容★

１．『多様化する電子書籍の可能性と未来』
株式会社メディアドゥ 取締役 事業統括本部長　溝口敦

2000年NTTドコモに入社。
音楽・映像配信などデジタル著作物の流通プラットフォームの企画及びコンテンツ開拓に携わる。
2008年より、メディアドゥに参画。同社の根幹ビジネスである電子書籍配信事業を主導し、
OverDriveやScribdとの提携による海外展開も担う。
ユーザの余暇時間・隙間時間を獲得するため、電子書籍サービスは多様化の時代を迎えています。
多様化の流れと、多様化を支えるテクノロジーの重要性について、マーケティング視点からお話します。

２．『月間60億配信に向けたインフラ基盤の構築』
株式会社メディアドゥ　技術本部　技術部運用課　課長　山田　祐市

同社所属のインフラエンジニア。朝日大学経営学部情報管理学科で経営情報学を学ぶ。
2008年に入社し、現在は運用責任者として電子書籍システムのインフラ構築・運用に従事。
常にインフラ基盤の最前線に立ち、情熱を持ってチャレンジし、年間システム稼働率99.999%を達成。
現在当社の基幹システムは月間約2億件のダウンロード配信をしており、昨今の急激な需要増加に対応
する為に昨年度「Oracle Exadata X5」を導入しました。
これによって当社は3年後に目標とする「60億配信」を実現できるインフラ基盤を構築しており、
これまでの経緯と今後の展望を技術的観点から解説します。

３．『電子書籍システム9年間の歩み』
株式会社メディアドゥ　技術本部　技術部開発課　課長　森川　晃

同社所属のフロントエンドエンジニア。中京大学経営学部経営学科で経営情報学を学ぶ。
2005年に入社し、着うた配信関連のシステムを開発。その後、電子書籍事業の立ち上げ当初より
約9年に渡りストア構築システム「MDCMS」のプロダクト責任者として開発業務に従事。
ここ最近は、最先端のフロントエンド技術に興味があり、自社システムに積極的な導入を進めている。
自身が担当するストアシステムや電子書籍ならではの話題もさる事ながら、これまで約9年間継続稼働
し続けているシステムを立ち上げから現在に至るまで関わり続けた経験をもとに、工夫してきた事、
苦労話、今後の課題など、「システムの継続性」というキーワードを掘り下げてざっくばらんにお話し
ます。
