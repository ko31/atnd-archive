OS自作をやっている皆さんで集まりましょう！

今回はいつもの自作 OS もくもく会と趣向を変えて、オープンソースカンファレンス 2018 Tokyo/Fall での特別開催です。 各人が作った OS の展示をしますので、ぜひ応援しにきてください！

そして、もし部屋が確保できれば「OSC 出張版・自作 OS もくもく会」も併せて開催するかもしれません。もくもく会の開催が決まったらこのページでお知らせします。

このイベントの対象は次のような方々です。

  • 自作OSに興味があり、OS関係の本を読んでいる（または今から読み始める）
  • 既に自分のOSを作り始めている
  • 既存のOSを学んでいる・改造している
  • 低レイヤ技術に興味があり、情報交換したい。

自作 OS もくもく会の過去の発表資料は osdev moku2 presentations にまとまっています。

当日のスケジュール

今回はサイボウズ東京オフィスではなく明星大学日野キャンパスですので、ご注意ください。

時刻 内容
1 日目 10:00 - 11:00 展示準備
1 日目 11:00 - 17:30 展示
2 日目 10:00 - 16:00 展示
2 日目 16:00 - 17:00 片付け

皆さんの OS をこの機会に展示しませんか？展示してもいいな、という方は @uchan_nos までお知らせください！
