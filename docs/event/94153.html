description

弊社は多言語Webサイトの企画制作、Webコンサルティングを行っているクリエイティブ会社です。国内向け、海外向け双方のコーポレートサイトの制作・運用・分析などを、多数手がけて参りました。コーポレートサイトのコンテンツは各社同じようなものになってしまいがちですが、IRランキングで高評価を得る企業と、そうでない企業は分かれます。

「そもそも良いコーポレートサイトとはどういうものなのか？」
「コーポレートサイトの最新トレンドは？」
「IRサイトランキング上位の企業はどういう工夫をしているのか？」
「海外向け広報。何からはじめればいいのか？」
「SNSは、本当に企業活動上意味があるのか？」

弊社の経験とノウハウに基づき、上記のことを紐解く無料セミナーを開催いたします。
もし、ピンと来ることがあれば、是非、今回の無料セミナーにお申込ください。

■セミナー概要

・評価の高いコーポレートサイトの特徴
・CSRコンテンツの最新トレンド・ESG投資への対応
・日英コーポレートサイトの違い
・企業としてのSNSの活用方法
・海外向け広報活動について

※応募者多数の場合は抽選とさせていただきます。

■講演者プロフィール

シトラスジャパン株式会社
Webグループ　執行役員　渡辺紀章

講師略歴

ITベンチャー企業で創業から上場まで経験。役員ならびにWebコンサルタントとして活躍後、2010年シトラスジャパン株式会社入社。現在は、最先端の海外マーケティングとこれまでの経験をもとに、日本企業が世界中に広がる潜在顧客開拓をどのように実現するかをテーマに、コンサルティング・執筆・講演などを行っている。
