description

新しい年が始まりました。この時期に今年の抱負や目標を決めている方は多くいらっしゃると思います。その中には今年をこれまでとは違った変化のある年にしたいと考える方もいらっしゃるのではないでしょうか？

いつも自分の中で何かモヤモヤしているものがある。
今の自分と本当の自分のギャップを感じている。
これからの社会や自分の生き方を考えると、息苦しさを感じる。
やむにやまれぬ思いに押しつぶされそうだ。
今働いている仕事にやりがいを見いだせない。
いつか起業したいと思っているけれど、なかなか行動に移せない。
自分が何をしたいのかよくわからないけれど、もっと社会に認められたい。

そんな想いを抱えながらも行動を起こせない現状を変えるチャンスです！
私たちは、あなたのこれまでの人生の振り返りや体験などを掘り下げていくことで、あなたの持っているスキルや知識、経験だけでないあなたらしさを再発見することが出来るのでは？！心の奥底に眠る原石を一緒に探していきます。そして、あなたらしい、あなたしかできない何かを形にすることで、これまでとは違った生き方を見つけませんか？！
例えば、医師という仕事は、患者さんに対して病の原因であるガンという病気を発見し、告知することは出来ます。それを治療するために精一杯の努力をしていきます。しかし、ガンを告知された患者さんやご家族は、その一言でそれまでの生活が一変します。医師としての一言の重大さを感じた時から、医師として出来ることは何か、人として出来ることは何かと悩み続け、社会起業大学での学びを通じて、新しい働き方を見つけた方がいます。
体験授業では、そんな事例を含め様々な働き方や生き方をしている人たちを紹介し、皆さんの想いにシンクロさせることが出来ればと考えています。
社会起業大学・九州校では、今年もそんな「変わりたい！」という気持ちを持つあなたのチャレンジを応援します！

まずは体験授業でその第一歩を一緒に踏み出してみませんか？
あなたの第一歩を踏み出す勇気をお待ちしています。
