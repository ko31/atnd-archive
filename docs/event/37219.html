またやるんですか？

ラック内の機器よりもチーズをハックする勉強会、再び。

自宅ラック勉強会でお世話になっております、@tadkinosです。
今回は #自宅ラック勉強会 タグをつけておりませんが、自宅ラック勉強会も宜しくお願いいたします。
正式な開催の際には自宅ラック系のタグがつくかもしれません。

どうしてそうなった？

  • 職業柄ついカッとなって大量のチーズをdownloadしてしまった。
  • 稀によくある、ので折角だから誰かに「 インストール(I) 」してもらおうか。
  • つまりチーズ大好きになってもらいたくてしかたがない某牛乳屋さんが開催する、小規模イベントです。
  • 今回もリビング使用の許可が下りました。

何をやりたいの？

  • 架空の中小企業を想定した環境を作って、好きなように遊べばいいじゃないかと思いました。
    こんな勉強会をやってみたい
  • この機会に、自分が知らない技術を「 共有(H) 」できれば。
  • 今回は検討会のプレという位置付けですので、開催に向けての実験という意味合いがあります。後日検討会で議題として挙げるべき要件とか課題が出てくると思いますのでご意見をいただきたいと思います。

持ってくるモノ

  • PCとかスイッチとかサーバとかIP電話機とかアプリケーション評価版CD/DVDとか
  • Windows7または8の試用版を事前にインストールしておくと良いかもしれません。
  • PRIMERGY TX100s3にWindows Server 2012 Essentials(評価版)をインストールした状態で皆様をお待ちしております。
  • PRIMERGY TX100s3がもう一台用意できます。メモリは少なめです。
  • メモリ2Gで良ければ自由に使えるサーバは3台用意できます。
  • それでも足りなければ、Poweredge 2950が1台使用できます。
  • 大容量ストレージ（空腹的な意味で）。
  • 今回も前回(http://atnd.org/events/35030)ほどではないですが、それなりのお値段なので「 削除(D) 」は本当に勘弁してください。

参加条件

  • チーズ食べたいひと。
  • あまり騒がしくしないひと。(常識の範囲でお願いいたします)

今回のチーズ

  • Gérard Sélection Fromage Roux(フランス産)→前回好評だった
  • アメリカンクリームチーズ(アメリカ産)
  • ロトラ バジル(アメリカ産)
  • グリュイェール(スイス産)

会場および集合場所

  • 個人住宅(マンション内の一室)が会場となります。
  • というわけで、駅からは各自でタクシーで会場まで移動してください。(割り勘でお願いします)
  • 初参加の方のサポートはできる限りいたしますのでご安心を。

注意点

  • マンション共用スペースでの私語は控えて頂けますようお願いいたします。
  • 開催者はただの残念な人なので技術力がありません。
  • その代わり周りの皆様が助けてくれるような気がします。
  • PCやサーバ等の持ち込みは可能ですが、電源容量は期待できません。（30Aです)
  • 上記理由により、ブレードサーバなどの持ち込みとか本当に勘弁してください。
  • 駐車場は近くにありません。
  • もちろんですが、自宅にラックが無くてもご参加いただけます。
  • 開催者の職業柄、別途おやつが用意される可能性があります。
