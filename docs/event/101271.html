今回の『ATEAM TECH MeetUp_vol.03』のテーマは「フロントエンドのモダナイズ」！

今回は、入社1-2年目の若手エンジニアより「フロントエンドのモダナイズ」をテーマにセッションを行います。若手ならではの新しい学びを実務にどのように活かして成果を挙げたのか、実際の開発事例を交えながらお話します。

また、『ATEAM TECH MeetUp』では、エンジニアのアウトプットの機会を増やし、交流の活性を目指しておりますので、エイチームエンジニアと参加者のみなさま、また参加いただいたエンジニア同士の横のつながりを深めるために、参加者のLTセッションや懇親会もご用意しております。名古屋のエンジニア業界を盛り上げていきましょう。

「ATEAM TECH」とは

ゲームやインターネット業界で働く技術者向けに勉強会や交流、アウトプットいただける場を設け、新しい気づきや成長につながるような機会を提供することで、技術力の向上や業界のさらなる発展を目指します。業界の技術トレンドに関する共有や交流の場を通じてより深い知識を得て、さらには参加者同士でのつながりもできればと考えております。

セッション登壇①：エンジニアの個人サービス開発のすヽめ

発表者：タニー

エイチームライフスタイル 名古屋開発部

エイチームライフスタイルが運営するサービス、くるまDELI（クルマデリ）の開発業務を担当

得意技術分野はバックエンド（Ruby on Rails）

セッション概要

これまで2年間程プライベートでWebサービスを運営してきました。 そのサービスで jQuery から React に書き換えた結果。その経験が業務に役立ちました。今回はその成功事例をもとに、プライベートでWebサービスを開発することで得られるメリットをお伝えします。

セッション登壇②：Nuxt.jsを使った非エンジニアでも更新出来るサイト構築について

発表者：MRP

引越し侍　デザイン開発部

現在は、引越し比較・予約サイト引越し侍を運営する株式会社エイチーム引越し侍にて、エアコンお買い得ドットコムやエアコンサポートセンター、らくらくネットサポートなどの開発を行なっています。

得意技術分野は、フロントエンド（HTML/CSS, JavaScript）

セッション概要

引越し侍のサービスでNuxt.jsを使ったページがリリースされました。 エンジニアの手を借りずにSEOサイトの新規記事をリリース出来るようにする為にnuxt.jsを導入しました。 新卒２年目のフロントエンジニアが社内で活用事例を踏まえてご紹介します。
