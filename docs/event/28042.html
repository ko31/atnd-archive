■登壇企業

株式会社gloops
Mobageを中心に『大熱狂！！プロ野球カード』『大進撃！！ドラゴン騎士団』など、累計登録者数1250万人を超えるソーシャルゲームを提供しているgloops。
国内だけではなくグローバル展開も進めており、アメリカ・ベトナムへの子会社の設立や、Facebookへのソーシャルゲームの提供もスタートしています！
一番のヒット作は会員数290万人を突破し、MLB開幕戦の冠スポンサーとして協賛するなど、快進撃を続けている同社にご講演いただきます！
