内容

インストラクター主導のオンサイト（お客様先）またはオープン（集合会場）研修

■達成スキル
このコースの完了時に、次のことができるようになります。

ステップやホップを含む基本的なデータ変換の作成、プレビュー、実行ができる
・データ変換の結果をメトリクスビューやログビューで確認できる
・基本的なセキュリティを含むPentahoエンタープライズレポジトリを設定できる
など

詳細は弊社案内ページをご参照ください。
