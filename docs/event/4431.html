共通のユーザIDを複数のWebサービスで使える話題の技術、OpenID。今や全世界で5億以上のOpenIDが発行され、対応サイトも増え続けています。

その仕様や周辺技術を学べる無料セミナーOpenID Tech Night Vol.6を開催します。

今回は、OIDF-J翻訳・教育ワーキング・グループによるOpenID関連文書の翻訳公開に際して、技術者が OpenIDについて理解を深めることができる機会を提供したいと思っています。

代表的な翻訳文書
– OpenID Authentication 2.0 – 最終版
– The OAuth 1.0 Protocol – draft-hammer-oauth-10 (RFCドラフト)

※OIDF-J翻訳・教育ワーキング・グループ
http://openid-foundation-japan.github.com/

つい先日、TwitterのBasic認証停止のカウントダウンも始まったことでご存じの方もいるはずの"OAuth"。当イベントでは、OpenIDに限らず、OAuthなど、Web Identity技術について広くエンジニアの方々に分かりやすく解説していきます。



OpenID Tech Nightには、どなたでも参加いただけますので、皆様、是非お気軽にお越しください！
