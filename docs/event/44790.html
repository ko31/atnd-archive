久々やります、「おしどりプレゼンツ日本酒の会」

場所はいつもと同じく菜庭酒菜　櫓仁です。
なんと櫓仁は改装工事中です（平成25年10月21日現在）
予定では10月29日に改装オープンするのですが、改装後の櫓仁でみんなでワイワイ飲みましょうっという趣旨です

すでに参加されたことのある方はご存知かと思いますけど一応…
ここは、月替わりのお任せ懐石コースが楽しめます。
旬のものをいただきながら日本酒を呑むという至ってシンプルな会です。
今回は改装記念で今までのベストメニューを集めてベリーベストオブ櫓仁な構成だそうです。
いつもは周年メニューの際にベリーベストオブ櫓仁なのですが、今回は特別だそうです。

日本酒の詳しいことを知らない方でも大丈夫！！
私も知りません。

料理にあわせてお酒も選んでいただきます。
