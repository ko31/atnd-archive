description

ビジネスでも就活でも成功の最大の秘訣は、情熱をさらけ出し、
相手にぶつけ、心を動かし鷲掴みにすること

これは当社代表の児玉が「落ちこぼれ→三菱商事→ハーバードMBA」という異色の経歴の中で見つけた必勝方程式です。「情熱」というと大袈裟に聞こえるかもしれませんが、実は誰の心にも潜んでいるものです。まず大切なのは、己に潜む情熱をしっかり認識すること。そこを起点として、情熱に向き合い、深め、効果的に相手にぶつけることできるようになります。
今回のイベントでは、これから就活に取り組む皆さんに児玉の生の言葉をお聞きいただき、対話していただくことで、皆さんの情熱をがっちり掘り起こし、アツアツの情熱を持ち帰っていただきます。夢を捨てたり、何か違う自分を演じたり、そんな嘘くさい就活とは一歩も二歩も違う「熱い就活」をしていただきたいという想いで開催させていただきます。
