その日、新潟大学がリアルとネットの交差点になる(ハズ)

趣旨

新大祭の日、新潟大学五十嵐キャンパスに集まって、ついったらーのお祭りをしよう!!

居場所や服装、イベント情報などをつぶやいて新大祭を満喫しつつ、できるだけ多くのついったらーを見つけよう!

＊このイベントは、 @scixer 発案の非公式なものです。実行委員会などには問い合わせないようにお願いします。

ルール

①新大祭1日目に、新潟大学五十嵐キャンパスにくる。

②新大を楽しむ。

③ついったらーを見つける。

  • 基本的な手順
    1. 逆かくれんぼ大会のハッシュタグを追う。
    2. 会いたいついったらーに、リプライを飛ばす。
    3. 場所や服装など、見つけるヒントになる情報を教えあう。
    4. チラッと目があったら、勇気を出して"逆かくれんぼやってる人ですか?"
    5. 当たったら出会えた奇跡を喜ぶ。外れたら趣旨を説明し謝る。
    6. 名刺交換など、連絡先を伝えたりすると幸せになれるかも。
    7. 後は、一緒に学祭を楽しむのも、他の参加者を探すのも自由

④状況によっては、終了後にオフもあるかも!?

注意事項

  • 楽しむのが目的なので、他人が嫌がることはしない。何が該当するかは自分で考えよう。
  • 現在のところ、新大祭一日目(10/22)のみで計画しています。
  • 新潟大学には、旭町キャンパス(新潟市中央区)と五十嵐キャンパス(新潟市西区)がありますが、逆かくれんぼ大会は五十嵐キャンパスで行います。
  • アクセスは、JR越後線の新潟大学前駅から徒歩20分位でキャンパスに到着します。
  • 極力、一発で当てるように努力する。蝶のように舞い、蜂にように刺す!!(刺さないけど)
  • 非公式なイベントなので、新大祭本部などには問い合わせない。@scixer にお願いします。
  • 無関係な人にアタックしてしまったら、逆かくれんぼ大会の趣旨を説明し、非公式であることを分かってもらった上で、ごめんなさいする。
  • 途中参加・辞退自由。ただ、実態把握のため、ATNDやツイッターに一言いただければ嬉しいです。

だれかが悲しいイベントで、みんなが楽しいはずがない
