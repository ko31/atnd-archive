中止のお知らせ

地震の影響による電力・交通事情を踏まえ、中止いたしました。 次の開催日時は未定ですが、決定後に改めて募集予定です。

イベントの概要

加賀ハイテック株式会社より発売されているタブレット型端末 Meopad を活用するための勉強会です。開発元の協賛により人数分の実機をご用意しますので、実際に触れていただけます。Meopad実機のアウトレット販売も予定しています。

Windows CEをインストールした後のアプリケーションの使い方や、無料環境（Visual C# 2010 Express）による開発方法について説明します。

当日はUstreamによる配信も予定しています。

参加費用

勉強会の参加費用は無料です。

勉強会終了後、希望者で懇親会を予定しています。懇親会は実費精算となります。予想価格は3,000円です。懇親会の参加有無につきましては、参加登録時に表示されるアンケートにご記入をお願いします。

準備する物

実機は勉強会でご用意しますので、特に必要なものはありません。お仕事帰りなどお気軽にお立ち寄りください。
