description

国民文化祭・京都2011では、大学が集積する京都から、地域の文化力向上に
つながる事業企画を若者が主体的に実施することで、地域文化発掘や京都文化を
見つめ直すきっかけとすることを目標に、
「まゆまろチャレンジ事業 次世代・若者企画」への申請事業を募集しています。

本事業では、国民文化祭・京都2011期間中において、学生の自由な発想による
文化芸術事業を選定し、若者が活躍できる場を提供することを目指しており、
この審査・運営もまた、学生・若者を中心に実施されています。

採択された事業企画には、実施期間の10月末まで、資金・施設・人脈紹介など、
京都府を中心とする主催団体より、事業実現に必要なサポートが受けられます。

このイベントでは、「まゆまろチャレンジ 次世代・若者企画」に事業申請を
希望する学生・若者によるプレゼンテーションと、担当者による意見交換を
実施いたします。

このまゆまろチャレンジ事業へ企画を申請しよう！と思われる学生・若者の方は、
【1月11日（火）23:00】までに、必要事項を記載した資料を送付してください。
（必要事項→ http://on.fb.me/dG4nKI）
（プレゼンテーション資料の送付先：cotolink+pop@gmail.com）


時間の都合上、6つ以上の企画申請があった場合は、ご提出いただいた資料を審査
させていただき、6つの企画を優先的にプレゼンしていただきます。

一線で活躍する企業や京都府の方に、フィードバックコメントを直接いただける
貴重な機会ですので、皆さんの積極的なチャレンジをお待ちしています!!


また今回は、過日実施しました「国民文化祭・300日前イベント Link Our Life」の
事業報告も合わせて実施いたします。よろしくお願いいたします。

イベント後には、大人と若者が入り交じっての大交流会も企画しておりますので、
皆さまどうぞ、ご自由にご参加くださいませ。


主催：国民文化祭・京都2011 学生・若者事業ネットワーク委員会（学生団体 cotolink ）
