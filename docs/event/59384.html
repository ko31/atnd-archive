description

「coins全体の技術力向上」を目的にgit勉強会を開催します。
大学の授業では扱わない内容ですし、最初はどのように勉強すればいいかわからないと思うので、有志で勉強会をしようと思い立った次第です。

内容としてはgitを3つの世界に分けてわかりやすく説明をした後に、実際にコマンドを叩いてもらおうと思っています。
実際の開発でのgitの使い方などは余力があればと思いますが、基本的に初学者に向けての内容となっています。

この活動はCOINS-Project EDUが主催しています。

対象者

  • gitを使おうと思っている人
  • gitを使い慣れていない人

coinsであれば何年生でも歓迎です！！！

好評であれば他学類向けにも実施したいです。
