description

この度、１周年を迎える「こい☆こい」では、テーマを『詫び・寂び的』『禅的』『ニホン的』文化の再構成プロジェクトと、題しまして、ライブペインティングと映像を主体と致した"華道×書道×水墨画×ジャパニメーション”のトップクリエイターと究極の映像アニメーションによる体感型コラボレーションアートフェスを開催する運びとなりました。
主な出演者・コラボ作品と致しまして【華道】からは、活け花のオートクチュール性を空間デザインに積極的に反映させ、その自由な創造性は古典アートの革新スタイルとして評価の高い、草月流・華道家『横井紅炎』氏。【書道】からは、今や【美人書道家】という揺ぎない金字塔を打ち立て、その確かな足跡においても多くのファンを虜にし「書は机上のみで作られる芸術ではない」というスタイルを確立した『涼 風花』氏。【水墨画】からは、イメージを視聴化する独特の画法で、その筆遣いは墨画に命を吹き込む気鋭のアーティストとして有名ミュージシャン・作曲家からも支持の高い『CHONTO』氏。そこにオマージュされるアニメーション作品は、制作期間7年、作画枚数10万枚。世界トップクラスのクリエイターが集結し、CG全盛の時代にあえて手描きにこだわり【見る】映像ではなく、【体験する】映像表現に挑戦した、日本が世界に誇るアニメーション『REDLINE』が決定しております。
また、イベント用フライヤー（パンフレット）の表紙イラストデザインのコラボアーティストと致しましては、【ハイネ生誕200年公式ポスター】【世界遺産ポスター・ゴールドメダル】を手掛けられた『ペドロ山下』教授のデザイン協力を得て、デザイン構成致しました。なお、イベント後半には、【箏奏者】『森川浩恵』氏のDJとセッションした、ネオトラディショナルライブも決定しております。今回も、来場者もアーティストも主体的に参加して楽しめるイベントを目指しております。
