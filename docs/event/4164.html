description

World Café By OKWave は世の中の様々な悩みや課題に対してみんなで考える場所です。
毎回選りすぐりのゲストに登場いただきテーマにそったトークをしていただきます。
そしてトークタイムの後にゲストの人が考えている抱えている課題を投げてもらいます。
その課題に対して参加者のみなさんと解決策を探ります。
課題と発表いただいた内容はOKWaveコミュニティでも公開しOKWaveユーザーからも解決策を募ります。


第２回テーマ

「グローバルで僕たちができること　
〜なぜGoogleは中国から撤退したのか？〜」



僕たちは海外にチャンスがあると聞くけど実際にビジネスで世界に出て行くことにハードルを感じている。

留まることはリスク？

実際に海外で活躍されている経営者にお話をお聞きします。

世界に衝撃を与えたGoogleの中国撤退の真相も聞けるらしい。

世界に出たい、中国に興味がある、うずうずしている社会人、学生歓迎です。
