description

真夏の川越で、ビールを飲みながらのライトニングトークしましょう。

小江戸らぐ（小江戸 Linux Users’ Group）を主宰しているはとちゃんです。

東急東横線が地下にもぐり、地下鉄副都心線や東武東上線とつながって横浜中華街から川越まで一本で来れるようになりました。これを記念して、小江戸らぐの本拠地"川越"で、Open Source Conferenceのスピンオフ企画で Open Source UN-Conference 川越 を開催します。

小江戸らぐといえば、ゆるく活動を続けて今年で11年目になるので、それにあわせて小江戸らぐが春に開催するBBQパーティで出している生ビールを会場に持ち込み、これを飲みながら ライトニングトーク をやります。みなさんが日頃やっていること、これからやろうとしていることなど、思いの丈を ５分間 にこめて語ってください。

ビール飲み放題と地の食べ物

ライトニングトークをゆるく聞きたいので、 飲み放題 の 生ビール と地の食べ物を用意します。

ライトニングトーク

各自 ５分間 でLinuxやOpen Source Softwareに関して日頃やっていることや、これからやろうとしていることなど思いの丈を語ってください。

発表者は、受付に発表リストを用意しておきますので、そこに記入してください。また、参加表明時のコメントに、ライトニングトークしますとか、話す内容を書いていただけると、進行の目安になりますので、よろしくお願いします。

場所

東上パールビルヂング地下1階 第一バンケットホール
住所: 埼玉県川越市脇田本町15-13

ビールと食べ物など

ビールなど

小江戸らぐの春のBBQパーティで飲んでいる 生ビール を用意します。とりあえず 飲み放題 でやろうと思っておりますが、ライトニングトークが聞けなくならないようにほどほどに。
ソフトドリンクも用意しておきます。

食べ物など

西埼玉の食べ物を用意しますので、ビールのお供としてご堪能ください。

  • 川越やきそば
  • 東松山（ひびき）やきとん（2本/人）
  • ポテトサラダ
  • カット野菜
  • 日高（サイボクハム）ソーセージ

食べ物はそれほど多くは用意できないので、当日の 持ち込み も歓迎します。ただし、このシーズンですので、腐りやすいものは避けてください。
はじまる前に昼食をお取りすることをお勧めします。東武東上線の下りホームの池袋よりには「文殊」という立ち食い蕎麦があります。上りホームにはそば処吉野家があります。
その他の川越駅のグルメ・レストラン
