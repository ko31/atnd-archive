description

8/16に葉山の一色海岸で恒例の海水浴をしたいと思います。
例年通りのスケジュールです。

1.12時半ごろ：JR逗子駅集合
2.いつものカレー屋（マッチポイント）でうまいカレーを食べる
3.逗子駅発葉山行きのバスで一色海岸へ
4.いつもの海の家（一色ロイヤルビーチ）でロッカーを借りる（シャワー代込みで1500円くらい）
5.夕方5時前まで泳ぐ（海の家は5時で閉店）
6.オシャレ海の家でオシャレDJのオシャレ音楽を聴きながら飲む or 鎌倉か江の島か藤沢で飲む
7.解散
