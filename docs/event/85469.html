DENSO A.I. Tech Talk

プレゼンター

桑島 洋

東京支社 / 電子基盤先行開発室　担当係長

2008年大阪大学大学院 工学研究科 修士課程修了。2009年 株式会社デンソー入社。2013年から2015年までStanford University客員研究員。2016年から人工知能の品質基盤の構想と技術開発に従事。

桑島による前回の講演より

講演内容：AI搭載システムの品質
AI搭載システムの品質保証技術に関する取り組みを講演。高度運転支援システム搭載車で生じた事故事例など交えながら、品質保証へのアプローチや今後の動向を解説。

曽田 尚宏

基礎研究所 / 先端研究部 / ＡＩ研究課

2012年 東北大学 情報科学研究科 博士後期課程修了。博士（情報科学）。2013年株式会社デンソー入社。ＡＤＡＳや自動運転システムのためのニューラルネットワークの研究に従事。

曽田による前回の講演より

講演内容：自動運転を実現する深層学習
深層学習を用いた高度運転支援システム、自動運転技術に関する取り組みを講演。画像認識や物体検知など豊富な事例・技術を解説。

F様 フランクに質問もでき、他のカンファレンスでは聞けない内容だった。
