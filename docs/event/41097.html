7/23（火）19:30〜渋谷「Stocker.jp / Space」にて、PHPを記述せずにカスタマイズ可能な国産CMS「a-blog cms」の入門勉強会を行います。

a-blog cmsは、4周年を迎えた6月24日に、商用利用可能な機能制限付き無料版のライセンスが発表されました。

リリース情報: liteライセンス新登場

それに伴い、今回は a-blog cms について、

  • Webサイト制作者・運営者視点でのセミナー
  • 静的な（つまり普通の）HTML で構築したWebサイトをCMS化する実習

を無料で行います。

今回の勉強会の内容

  • Webサイト運営者に嬉しい機能の紹介
  • Webサイト制作者に必要な9つのキーワードを紹介
  • 無償版で出来る事、出来ない事を紹介
  • 静的サイトをCMS化するハンズオン

講師紹介: 山本一道さん

名古屋のホームページ制作会社「有限会社アップルップル」の代表をしています。

CMS「a-blog cms」の開発や、名古屋の「WCAN」という名古屋のWeb制作者のためのセミナーの主催や、コワーキングスペース「ベースキャンプ名古屋」も始めました。
