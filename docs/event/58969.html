description

■講座概要
・PlayStation4、XboX-ONEなどハイエンドの次世代ゲームが次々に発売され、スマートフォンの3DCGもそれに追従して発展しています。この講座では、そういったゲームコンテンツを今お使いの3DCGソフトから制作できるノウハウを学べる実践的な体験実習です。
・Maya、3dsmax、Blender、Metasequoia、Shade、Cinema4Dなど、いずれかの3Dデザインツールで作成したキャラクターモデリング、シェーディング、アニメーションを最新技術で話題沸騰のゲームエンジン「Unreal Engine 4」の連携させるノウハウを学びます。
特にMMDでアーティストをされている方にオススメです！教材素材は、いま話題の「Unityちゃん」です！→http://unity-chan.com/
・ゲームエンジンのセミナーは、とかくエンジニア視点が多い中で、デザイナーをメインとしたものであるのが特徴です。エンジニア的なノウハウもデザイナー視点で理解できるような授業展開です。
・「映像中心なので、ゲームは関係ない」という方もこの講座はオススメ！ゲームエンジンをリアルタイムレンダリングツールとして使えば、レンダリング待ち時間なしです。これを知らないと損？

■カリキュラム内容：　17:30～20:30　３時間（途中休憩・質疑応答あり）
１．Unreal Engine 4の基本操作、システムを学ぶ
２．Maya、3dsmaxのデータからUnreal Engine 4へモデルデータ変換ノウハウ
３．Unreal Engine 4での独自の高画質リアルタイムシェーダー設定でレンダリング
４．キャラクターアニメーションをUnreal Engine 4での再生、キーでリアルタイム操作
５．カメラアニメーションと切り替え→カメラマンになった気持ちで！
６．プループリントによるＧＵＩプログラミング基礎→テクスチャー変更で服を着替える
７．フリーアセットの利用で背景作成
８．リアルタイムレンダリングを映像として書き出す方法

■対象
＊ゲームエンジンに関して初心者のＣＧデザイナー、プランナー、ディレクター
＊何かしらのツールで３ＤＣＧ開発経験がある方。初心者でも歓迎です！
＊学校や独学で２Ｄ・３Ｄデジタルコンテンツ制作を学ばれている方
＊企業での新規事業でゲームエンジンの活用を検討・採用されている企画担当の方

■準備いただくもの
特になし。会場のＰＣを使って受講できます。
出来上がった作品を持ち帰っていただくＵＳＢメモリがあれば便利です。
PC持参いただく場合、フリーのBlenderをインストールしていただくと、より楽しめます。
こちらからダウンロードできます→http://blender.jp/

■講師からひとこと
さまざまな３ＤＣＧツールをお使いのプロの方から学生まで、ゲームエンジンをやってみたいけどチャンスがない、という方にぜひおすすめします。とにかくデザイナーのための講座ですので、難しいエンジニアの話はありません。ゲーム制作から映像制作まで幅広く活用できます。
いろいろな職種の方々が集まって魅力的なコンテンツを企画するコミュニティづくりも考えております。ぜひご参加ください。
