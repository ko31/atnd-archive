ウェブ開発に革命を起こすといわれる「Web Components」を先取り。

事前準備

事前に Git、ChromeのインストールとGruntの準備をお願いします。
http://gruntjs.com/getting-started

フロントエンド開発が初めての方、開発環境構築が面倒な方は、
ブラウザ上で簡単にフロントエンド開発ができる「 Cloud9 」のアカウントを作成して、これを利用してください。

Github アカウントがあれば尚良。
