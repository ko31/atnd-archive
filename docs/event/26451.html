description

暖かくなって参りましたのでゲームを企画いたしました！
4/21（土）に千葉のSPLASHのインドア・フィールドでの開催です。
インドアフィールドですので雨が降っても大丈夫です！
原則的に身内ゲームとなっておりますが 趣旨をご理解頂けるご友人の方が
いらっしゃればお誘い合わせの上 ご参加下さると幸いです。

■参加表明に関しまして
ATNDを使用できないゲスト参加の方を お連れになられる場合、お手数ですがコメント欄に
人数とその方のお名前 （HNで可）をご記入頂くよう願いします。 またお連れの方には大会の
レギュレーションをかならずお伝え下さい。

■必要な物、参加費
・参加費：2,500円～3,000円（参加者が1２人以下なら使用料の30,000円を頭割）
・エアガン：電動ガンレンタルあり。2,500円/一日
・BB弾：６ｍｍＢＢ弾０．２ｇ以下、８ｍｍＢＢ弾０．３４ｇ以下
・装備：ゴーグル（シューティンググラス不可）、顔面を保護するマスク、
　　　　　ニーパッドや手袋があるとベター(装備各種レンタルあり)
　　　　　フィールドは多量のBB弾が散乱しており非常に滑りやすくなっています。
　　　　　急なスロープ、階段等もありますので足に合った滑りにくい靴を推奨。
　　　　　お名前（HN）が判るような名札をご用意いただければ幸いです。
・昼食：関しては各自、事前にご用意下さい。
　 　　　飲み物とカップラーメンの自販機はございます。
　　　　　また、お弁当の注文も可能ですのでご利用下さい（700円～800円）
　　　　　徒歩10分程度の所にコンビニもありますがフィールド外に出られる際は
　 　　　装備、BDU等着用のまま外出されないようお願い致します。

■レギュレーション
・パワー制限は銃刀法に定める規制値以下（０．９８J以下）
（インドアで近接戦闘になりやすのでパワーに関しては優しさを持った仕様推奨）
・フリーズタッチ、ナイフアタック不可（痛くなさそうな場所を撃って下さい）
・基本的にブラインドファイア（遮蔽物より銃のみ出しての発砲)禁止
・外部ソースの使用禁止
・レーザーサイトの使用禁止

弾数制限、フル/セミ制限等は特に定めませんし装備によるハンデも設けません。
ただ、ハンドガンオンリー戦、セミオート限定戦等も開催する予定ですし、
シューティングマッチも計画しておりますのでハンドガンのみの方もお楽しみ
いただけると思います。

■ちょっとしたお願い
諸般の事情により参加を伏せたい方も居らっしゃいますので大会参加者の個人名を
ご自分のブログ等で上げるのはご遠慮いただきますようお願い致します。
個人で「今日サバゲして来た」と言うのは自由ですが誰が参加していたかに関しては
「触れない」方向でお願いします。
TwitterやMixi等で「○○さんお疲れ様でした」等も出来るだけ空気読んで下さると
助かる人も多いかと思います。
基本的に本ページコメ欄使って頂きますようお願い致します。

■その他
・フィールドの各詳細確認はSPLASHのHPをご参照下さい。
・レギュレーション、ゲーム内容等に関しては随時、更新をして参ります。

・主催者は8時過ぎに会場入りしております。
　最終的な装備確認（弾道調整等）、諸注意を簡単に行い９時半を目処に
　ゲームを開始しようかと考えております。
