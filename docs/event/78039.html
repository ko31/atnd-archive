description

【一緒に手紙を書きませんか？】

センスよく、気持ちを伝える手書きの手紙。

コツを知れば、自信をもって手紙が書けるようになります。

≪字がきれいじゃない≫
≪文章を書くのが苦手≫

そうやって「手紙をうまく書けない」
と思い込んでいらっしゃる方は多いもの。

もったいない！！

せっかく相手に伝えたい気持ちがあるのに、
一瞬、手紙を書こうかなと思ったのに、
かわいいレターセットがあるのに、
書かなければ、その気持ちは届きません。

相手への思いを伝える要素は、
文字や文章だけではないのです。

この講座では、
【5ステップで気持ちを伝える手紙の書き方】を
講義＋ワーク形式で学びます。

講師は、自ら年間1000通以上の手紙を書く
（社）手紙文化振興協会認定　手紙の書き方コンサルタントの青木多香子。
一般・企業向けの手紙の書き方講座や通信講座添削指導のほか、
新聞・雑誌の取材協力、記事監修等、幅広く活動しています。
「愛に満ちたことばと笑顔が世界中にあふれることを願い、
人を癒し、励まし、幸せにする」をミッションとし、
手紙を通じて、自分らしく輝きたい人を応援しています。

受講後には、
「誰かのために手紙を書きたい」
「手書きで手紙を書こう！！書きたい！！」
と思ってもらえるはずです。

手紙は、相手に喜んでもらえるだけでなく、
職場や家庭の人間関係向上、取引先やお客様との信頼関係構築にも役立てられます。

さらに、手紙を書くことで、言葉が磨かれ、自分の心を整えたり、
自分自身をランクアップさせることもできてしまう、
まさに魔法のツールなのです。

一緒に手紙を書きませんか？

★受講者の方には、すぐに手紙が書きたくなる
万年筆と一筆箋・カードなどのプレゼントもご用意しています。

【想い】
手紙を書くと、言葉が磨かれ、心も磨かれます。
手紙は人を笑顔にもします。
「手紙を書く→書き手も受け手も幸せになる→手紙を書く・・・」
の連鎖で、手紙の書き手が増えたら、
世界中が素敵な言葉と笑顔であふれると思うのです。
そんな日を夢見て、手紙の魅力を多くの人に伝えていきたいと思っています。

【こんな方を対象としています】
・さらりと手紙を書けるようになりたい方
・自信をもって手紙を書けるようになりたい方
・文字・文章を書くことに苦手意識がある方
・気の利いたひとことが書きたい方
・接客業、営業担当の方
・手紙好き、文具好きな仲間と知り合いたい方
・職場や家庭の人間関係を改善したいと思っている方

【過去の受講生の声】
・基本の5ステップの説明がとてもわかりやすかったです。
・等身大のことばでかけばいいのがわかったので、気が楽になりました。
・文章だけでなく、用紙の選び方、ペンなどでも
　思いを伝えることができることがわかってよかった。
・相手に自分らしさを伝えられる言葉を考えるようになりそう。
・手書きで手紙を書こう！！書きたい！！と思うようになりました。

【プロフィール】
青木多香子
贈る手紙の会　代表
（社）手紙文化振興協会認定　手紙の書き方コンサルタント

1976年生まれ。ミシガン州立大学コミュニケーション学部卒業。
カリフォルニアで編集・ライターの仕事をした後、帰国。
広報担当として、日本語と英語でお礼状を書くようになり、
企業間の潤滑油となる手紙の重要性を体感。
フリーライター・通訳業を経て、教育の世界へ。
発達障害児の学習指導・教材づくり、外国人留学生の就職支援、
職業訓練校校長兼キャリアコンサルタント、
英会話スクールマネージャーを経験し、
2014年手紙の書き方コンサルタントとなる。
