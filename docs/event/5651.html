ビジネスラボ６月は通常タイプのラボです。

■ビジネスラボとは？

ビジネスラボは「一見さんお断り」形式の、信頼のおけるビジネパーソン同志の会です。売る・買う・コラボするといったビジネスマッチングを行っています。スタッフもしくは過去参加者の紹介でのみ参加可能です。

■ビジネスに対するスタンス

ここで出会った方とのスムーズなビジネスを推奨していますが、片方にのみ利益のあるようなビジネスを行う方にはお断りしています。また、焼畑営業を行う方もお断りします。

■会のタイプ

ビジネスラボは通常タイプおよびイベントタイプの２種類があります。目的の通り、２０名程度のビジネスマッチングを行う会は通常タイプと呼ばれており、イベントタイプは通常タイプで知り合った人脈との懇親会的な位置づけになっております。
