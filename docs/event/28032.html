description

今や多くの個人、企業が使っているソーシャルメディア！あなたはfacebook、Twitter、YouTubeを活用していますか？ソーシャルメディアの利用者が増加する中で良い事もたくさんありますが、逆にリスクも高まっています。

実際にソーシャルメディアでの情報発信が元となって多くの炎上と言われる事件が起こっています。
また、リスクを考えるあまり手をつけれていないという悩みも多く聞かれます。

そこで今回はソーシャルメディアに取り組む前にしておかなければいけない基本事項とリスク対策についてお話しします。
下記の悩みをお持ちの方は特にオススメです。

・自社でもソーシャルメディアを活用したいけど何から始めていいのかわからない
・自社でfacebookを始めたけどどんな内容は良くてどんな内容がダメなのかわからない
・従業員が個人で使っている事に対してどのように対策したらいいのかわからない
・従業員にも営業ツールのひとつとして活用をすすめたい
・社内の人にも考え方に違いがあって方向が定まらない
