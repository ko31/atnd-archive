※好評につき増席決定です！！※

ソーシャルゲームの開発をおこなうのであればネイティブで開発をするのは当たり前。
そんな中、最近需要がメキメキと増えてきている「cocos2d-x」について、
cocos2d-xの開発元であるChukong Technologies Japanの清水様に講師をしていただきながら
「cocos2d-x」の魅力がよくわかるハンズオンセミナーを開催いたします。

今までUnityでの開発はしたことがあったけど、cocos2d-xは触ったことなかった方や
ネイティブアプリの開発にこれから取り組んでいこうと言う方
過去にcocos2d-xを使ってみたけど、色々訳あって断念をしてしまった方。

実際に「cocos2d-x」に触れながらその魅力を体験していきましょう！

セミナー内容

そもそも「cocos2d-x」とは？
cocos2d-xってオープンソースのゲームフレームワークだよね。
でもそれだけではないんです。
「cocos2d-x」で何が出来るのか。シェア率や開発事例などを交えながらお話をさせていただきます。
新ツール「Cocos」の魅力を体験しよう
新ツール「Cocos」って知ってますか？
「Cocos」を利用することにより、Cocos2d-xの開発環境構築・プロジェクト作成が容易になり「cocos2d-x」での開発がしやすくなっています。
話題の「Cocos」について解説をおこないます。
まだまだ進化中の「cocos2d-x」!!
6月25日にCocos2d-x v3.7がリリースされ、3D Navigation mesや3D Navigation meshなどの機能追加によりますます注目を集める3D機能ですが、実はベースシステムもしっかり強化されているんです。
今回は強化されたベースシステムについて解説しちゃいます！
わからない部分についてはChukong Technologies Japanの先生たちが丁寧に教えてくれますのでわからない所はバンバン質問をしちゃってください。
