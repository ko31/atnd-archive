ストレス社会と言われる現代、
また2015年12月に施行されるメンタルヘルス義務化に向け、
心のサポートをするカウンセラーの存在が必要性を増してきております。

日本メンタルヘルスケアサポート協会では、
心理カウンセラーの中でもメンタルヘルスにもしっかりと対応できるカウンセラー、
もしくはメンタルヘルス担当者を養成します。

メンタルヘルスカウンセラー養成講座は、
メンタルヘルスに対応できる心理カウンセラー、
職場のメンタルヘルス推進者を養成する資格取得講座です。

特徴：
①具体的なメンタルヘルス対応策が身に付けられる
②より良い人間関係の構築の仕方が身に付けられる
③心のケアに欠かせないカウンセリングスキルが身に付けられる
