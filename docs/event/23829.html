■a-blog cms 1.5 リリース記念(予定)！！九州合宿！

2012年に1.5がリリースされます。新機能を中心に、実装を含めた合宿を2011年5月に行った場所でふたたび開催することになりました。

「a-blog cms使っているけど、まだよく分からない。」「他の人がやったカスタマイズの方法が知りたい。」「というか、よさそうな感じだけどまだ使ったことがない。」という様々なユーザーにお答えできる合宿になればと考えています。

山本 一道 ( Kazumichi Yamamoto / @kazumich )

  • 有限会社アップルップル 代表取締役
  • WCAN ( Web Creators Association Nagoya）主宰
  • kazumich.log

2000年秋に最初の「a-nikki」という日記 CGI を公開しました（この時に考えたインターフェイスが a-blog cms まで引き継がれています）。当時はプロバイダに勤めていましたが、2002年に独立し Web 制作を個人事業主として開始、2004年秋にはアップルップルを法人化して「a-blog」を開発しリリースしています。また、名古屋の Web 業界を盛り上げるための活動として、WCAN という Web 制作者のための勉強会の主宰を2000年より行っております。

野崎 晃(Akira Nozaki / @nozaki_ewn )

  • エコーワークス 代表
  • Jimdoでつくる簡単ホームページ！
  • ノザキアキラの108

愛知県名古屋市を中心に活動する、Web制作のフリーランス。 製造業の社内SE、自社サービスのサイト制作・管理などの経験を経て、2008年にフリーのWeb＆印刷物制作として独立。2009年末にKDDIウェブコミュニケーションズのJimdoに触れ、衝撃を受ける。より多くの人にJimdoの便利さ、楽しさを伝えるべく解説ページ「Jimdoでつくる簡単ホームページ！」や制作＆カスタマイズサービスの「SITENEXT（サイトネクスト）」を運営。Jimdoに関するセミナーやイベントなども多数主催。JimdoEvangelist/JimdoExpert

山口 有由希（Yuki Yamaguchi / @yuki930 ）

  • 有限会社フロッグマンオフィス デザイナー
  • YUKI YAMAGUCHI Design Portfolio
  • VIVID COLORS + BLOG
  • Fireworksマニア

長崎県佐世保市で生まれ、現在は福岡県にて活動しているWebデザイナー。
IT関連会社でプログラマーとして勤務ののち、2005年にフロッグマンオフィスに入社。デザインからコーディングまで幅広く制作経験を重ねるかたわら、個人プロダクトとしてさまざまなサイトの制作・運営も行っています。ツールや方法にとらわれず、より効率的に、成果の出るものを作るための方法を日々追求しています。著書に『Fireworks CS5/4/3（速習デザイン）』（技術評論社）。

赤木 崇（Takashi Akagi / @majosystems ）

広島在住。Web系システムの仕事。主にFlex(Flash)を使ったRIA環境で、仕事の効率化を提案!外部に向けてのWebサイトの企画・構築から、社内への連携、システム制作まで幅広く展開中です。

小佐々 康生（Yasuo Kozasa / @samurai_yasu ）

グラフィックデザイン事務所「パワーグラフィック」千々和洋氏に師事。
2003年に「サムライデザイン」として独立。FLASHデザイン構築に多くの実績を残していたが、現在にCMS構築が中心となって活躍中。

■コースは3種類！選ぶのはあなた次第！

開催日時を考慮して3つのコースをご用意いたしました。

■日帰りコース（参加料4,200円）

1日目の夕食前までのプログラムに参加できます。

■日帰り＆夕食付きコース（参加料9,450円）

1日目のプログラムに参加できます。

■合宿コース（参加料14,700円）

全プログラムに参加できます。2日目の朝食も付いています。

参加希望の方の状況にできるだけ対応はいたしますので、ご相談ください。

■参加条件

今回の合宿では、ある程度CMS実装経験のある方に参加をお願いしたいと思っております。a-blog cmsは実装経験がなくても、WPやMTなどの実装経験があれば大丈夫だと考えております。

■PCの持ち込みについて

今回はプロジェクタの画面を見ながらのご紹介もありますが、実際にさわってカスタマイズを行っていただきます。なのでノートPCが必要となります。ネット環境に関してはめかり山荘様のご好意で用意いたしますが、光回線ではないため、ご用意出来る方は用意していただけると助かります。

■開催地：めかり山荘について

今回お世話になる国民宿舎めかり山荘は、2012年の3月末で惜しまれつつ閉館の予定となっております。今回の合宿が一つの思い出として記念の合宿にしてください。
