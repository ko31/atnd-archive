バランススクーター『Kintone』を実際に乗ってみよう！

バランススクーター…気になるから乗ってみたい！

テレビやネットで話題のバランススクーター。
重心を傾けるだけで進める次世代の乗り物なので、気になっている方も多いはず。

いきなり買うとなると値段も結構かかるので、1度乗って乗り心地を体験してみたいですよね。
ただし、探してみるとバランススクーターを試乗できる場所ってあまりないんですよね・・・。

だったら、自分たちで体験会を開催すればいいのではないか。
ということで、バランススクーター『Kintone』を販売している株式会社「EarthShip」さんへお願いし、我々AppBankとともに「体験会」を開催できることになりました！

バランススクーターに興味があり、実際に乗ってみたい方はぜひ参加してください！

イベント詳細

2017年6月24日（土）15:00〜18:00

受付は14:30からで、15:00から体験会を開始します。

15:30からの体験会では、5人1組のグループを作りそれぞれ体験していただきます。
グループの振り分けは受付で行います。

イベント内容

バランススクーター『Kintone』製品を実際に試乗して乗り心地を体験していただきます。
防具や補助員も用意していますので、安全にバランススクーターを楽しめるようになっています。

体験会に使用する『Kintone』製品は以下の通りです。

▼KINTONE セルフバランススクーター▼

慣れるとスイスイ進めるようになるので、体験会内でぜひ乗り方をマスターしてください！

参加費

無料

応募条件

16歳以上の方
下記の理由により、お子様の使用を制限しているため、年齢制限を設けております。
当日は年齢確認をさせていただく場合がございます。

・体重が足りずセンサーが正しく作動しない場合がある
・時速10km近い速度が出るため制御できないと危険である
