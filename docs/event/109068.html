　数学好きもそうでない人も部室みたいに集まれる憩いの場

【目的】

埼玉で数学をきっかけにした交流。

【主催から】

『第７回 数学デーin埼玉』 を開催致します。

東京の「数学デー」公式様に参加させて頂いたところ、参加者の方々が自由に意見交流を楽しめる素晴らしいイベントでした。

そこで『数学デーin埼玉』 と称し、数学をテーマに自由に意見交流出来るイベントを開催致します。

数学が苦手で交流重視の方も安心してご参加頂けます。

本企画を切っ掛けにして埼玉での数学イベントの普及に寄与できれば幸いです。

【これまでにやってきた事】

  • 素数大富豪
  • ナブラ演算子ゲーム
  • 集合トランプで集合スピード
  • 埼玉県立高校入試過去問
  • SNSで話題になった数学問題
  • ライトニングトーク
  • AtCoder
  • 数学同人誌読書会 など
