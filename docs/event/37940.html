主催： Firefox 学生マーケティングチーム

Firefox 学生マーケティングチーム主催の勉強会を行います！
「 Add-On は使っているけど、 Add-On の開発は未経験」、
「興味を持っているけど、どうしたらいいかわからない！」
そんな人の為の“ Add-On 開発はじめの一歩”をテーマにした勉強会です。

Add-On 開発の環境の導入までは、
全員向けにチュートリアルを実施した後、
自由な発想で、 Add-On を各自開発して頂きます。

JavaScript や、プログラミングをしたことがない方でも OK。
講師含め、Firefox 学生マーケティングチームがお手伝い致します。
“Hello world!!” 経験済みの方ならベスト。
プログラミングをしたことがない人でもこの機会に挑戦してみましょう！
ぜひお気軽にご参加ください！

＜形式＞
プログラミング初心者の学生にはプログラミングの勉強から、
開発経験者は Add-On 作者への入り口となる勉強会を行います。
そのため講師：受講者 が 1:1 or 1:2 程度を想定し、
ワークショップ形式をとって勉強をします。
