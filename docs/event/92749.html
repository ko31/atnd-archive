概要

サービス開発する際、サーバー構成がスケーラブルかどうか問われた経験はありませんか？

近年のスマートフォン普及に伴い、ユーザートラフィック量が膨大なってきております。サーバーを落とすこと無く、大量のトラフィックをさばき、且つ管理画面の動作などによるユーザーストレスへ配慮する事はとても重要なミッションです。

今回の勉強会では、アマゾンウェブサービス現役ソリューションアーキテクトとして活躍される篠原英治氏をお招きし、多くのサービスがあるAmazon Web Services (AWS)の中で、特にスケーラブルに構成するという事に注目したサービスをご紹介いただき、AWSのこれからについてもお話しいただきます。

また、実際にAWSを利用しているタイレルシステムズ技術部マネージャーの高津英輔氏より実際の活用法などもご紹介いただきます。当社バレットグループ開発グループリーダー菊山からは、自社広告システムの開発運用において、小規模チームでも安定して広告計測を行えるシステム構築のノウハウをお話しいたします。

AWSの活用法をもっと知りたい方や、スケーラブルなサービス構成を実現するノウハウに興味のある方など、経験の多い少ないに関わらず、今後に向けての気づきの機会になれば幸いです。

勉強会終了後に１時間程度の技術懇親会をご用意しております。軽食やソフトドリンクやお酒をご用意しておりますので、是非ともお気軽にご参加下さい。 AWS篠原氏からはイベント1週間前に行われるAWS re:Inventの振り返りなどホットな情報の発信もあろうかと思います。様々な技術者交流の場としてぜひご参加ください！
