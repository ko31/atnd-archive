「ところで、サーバサイドってなんなんだろう。」

ゼロからはじめるサーバサイド勉強会 やります！

  • HTML / CSSを理解している
  • JSをコピペで動かせる（JSのコードを見たことある・なんとなく読める）
  • フロントエンドエンジニアになりたい
  • バックエンドはまだよくわからない(´･ω･)

という感じの方を対象に、セミナー形式で行う勉強会です。
今回は「サーバサイド」を1時間でさくさくっと網羅する会となります。

▼もくじ

Webの人が関わるサーバーの種類と役割

  • Webサーバー(HTTP/HTTPS)の話
    Perl/PHP/Ruby/Java/ASP.NET などサーバーサイド言語の紹介
  • データベースサーバー(SQL,NoSQL)の話
    データベースの役割(さらっとメールサーバー、DNSサーバー)

ブラウザはどうやってページを取得しているか

  • 静的ページ・動的ページってなに
  • GET/POSTとは
  • 簡単なフォームを動作させてみる
  • 暗号化通信(SSL)

最近のWeb技術から見るサーバーサイド

  • Ajaxによる非同期通信
  • Web API
  • OAuthによる認証
