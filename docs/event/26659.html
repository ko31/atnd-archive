　さ～皆さん！今年もモータスポーツが開幕しました！
　そこで…昨年に引き続き今年も鈴鹿サーキットで行われるイベントでお越し頂ける皆さんに少しでも交流出来る場をご用意させて頂く事にしました。

　今年も老若男女集まって賑やかにやりますので、鈴鹿２＆４の週末は土曜から来なくちゃ勿体無い(!?)ですよ～

　そこで今年の鈴鹿での催しを彩る企画を考えました!!
　題して…

　【皆のセンスを問います！1,050円(税込)以内で買えるプレゼント交換会】

　表題からしてセンスが無いのはご勘弁を…
　昨年通して鈴鹿サーキットの催しに付随しファンイベントを行い反省点もふまえ今年は皆さんで持ち寄るプレゼント交換用の品を持って来て頂く事が参加必須とします！
　たかが1,000円、されど1,000円の皆さんのセンス溢れるモノを当日のメインイベントじゃんけん大会にて全員行き渡る様にします。
　1,000円じゃ何も買えないよ～とお思いの方！普段使わない脳をフル活用して頂いて洒落た物、ビックリする物等ご用意してきて下さいね！(例えば…1個1,000円もする林檎とか一粒1,000円のイチゴ…普段の生活じゃ1,000円も出して買わないよって言う物とかね^^)

　《ルール》
　①必ず1,050円以内(税込み)で買える物を用意して下さい(物でなくてもいいですよ)
　②1,050円以内で買ったと言う証明を添付して下さい(レシート等)
　③そのレシート等に買った本人の名前を書いて、当日参加費と共に係りに渡して下さい。
　④中身がわからない用に黒い袋等に入れて陳列します。
　⑤じゃんけん大会にて買った順番にプレゼントを決めその場で開封して発表します！
