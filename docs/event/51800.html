description

　オープンCAE（OpenFOAM，Salome-mecaなど）に関心を持つ技術者，研究者，学生などが集まり，気軽に情報交換・相談する場所を作ることを目的に，勉強会を開催します。
　すでに使用経験のある方はもちろん，興味はあるけれど使い方が分らないという方も大歓迎です。お気軽にご参加ください。

　今回は，ミニ講習会として，「cfMeshによるメッシュ作成入門」を実施します。講師は秋山様です。OpenFOAMのためのメッシュ生成ユーティリティーとして最近登場したcfMeshは，前回の勉強会でも関心を集めていました。cfMeshの設定方法、簡単な形状による作成、境界層の作成、閉じていない表面形状によるメッシュ作成、時間があればsnappyHexMeshとのメッシュ比較を行いたいと思います。公開されて間がないため設定方法等模索しており参加者の方と議論しながら進めていきたいと考えています。

　OpenFOAMのプリ処理にお悩みの方，簡単な設定で物体形状にフィットしたヘキサメッシュを作成したい方など，興味をお持ちの方は，ぜひ，ご参加下さい。

　デモ操作をご覧いただく，あるいは，ご持参のPCで操作していただくことができます。

　参加ご希望の方は，ATNDのサイトから参加登録をしてください。
