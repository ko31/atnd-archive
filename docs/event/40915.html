コンテナ型仮想化の情報交換会について

コンテナ型の仮想化とか OS レベルの仮想化とか言われる仮想化技術に関する勉強会・情報交換をする会 を 首都圏で 実施します．コンテナ関連の技術概要，最新情報，コンテナ仮想化に関係するカーネルの濃いお話，活用事例などのテーマを扱う予定です．

当初，このようなイベントをしたら参加してみたいという人がどれくらいいるのかを調べるための 仮想イベントとして立てていましたが， ありがたいことに多数の参加希望をいただきましたので実際に開催します!! (当初 「コンテナ型仮想化の勉強会があれば参加したいと思う人を募る会」 というイベント名でお伺いを立てていました)

VMware, KVM, Xen 等のハイパーバイザー型の仮想化技術に比べて，コンテナに関する書籍や記事，情報などは少なく，コンテナの情報交換に使える場も非常に少ないのが現状だと思います．そこでみんなでコンテナの情報を持ち寄ってワイワイやりながら情報交換ができればなあと思っています．
