description

日本の製造業は韓国や中国などの企業との厳しい競争に直面していますが、今後は、製品のまわりにいかに新しいサービスを生み出していくかということが大切になるでしょう。その意味で注目されるのが、インターネットに接続できる家電です。家電がインターネットに接続されることで新しい情報がネット上に流れるようになり、その情報をもとにソーシャルな人間関係をベースにしたコミュニティも生まれます。つまり、家電はコミュニティ形成のためのツールになりえるわけです。


@nbcでは、以前からネットコミュニティに関わるビジネスについて勉強会を開いてきました。次回の勉強会では、ソフトウェアやサービスが中心だった今までとは少し趣きを変えて、ネット家電あるいは「ソーシャル家電」とビジネスについて考えたいと思っています。


講師は、ネット家電のベンチャー企業「Cerevo」の岩佐琢磨社長です。岩佐社長は、2003年に松下電器産業株式会社（現パナソニック）に入社。ネット対応のテレビ、録画機、カメラといった「ネット家電」関連の事業に従事し、DIGA向け宅外予約サービス“DIMORA”の立ち上げなどを担当されました。そして、ネットと家電で生活をもっと便利に・豊かにしたいとの思いから、2007年に独立し、2009年12月には、最初のネット家電「CEREVO CAM」を発表しました。著名ブログ「キャズムを超えろ！」の「和蓮和尚」としても知られています。
