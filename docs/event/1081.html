JPUG合宿2009盛岡解散後、わんこそばを食べに行きませんか？


日時：2009年7月19日 (日)　JPUG合宿2009盛岡　解散後

場所：東家駅前店
　　　http://www.wankosoba-azumaya.co.jp/
　　　JR盛岡駅東口の目の前のお店です。

目的：「わんこそば」に挑戦！
　　　・「わんこそば」とはなんぞ？という方はコチラ
　　　・食べ方のルールはコチラ
　　　・わんこそば全日本選手権＜歴代記録＞　第1位 559杯　第2位 530杯　第3位 451杯
　　　　( http://www.wankosoba-azumaya.co.jp/naze/index.shtmlより抜粋 )

※当日は三連休中日のため混雑が予想されます。ATND登録状況により随時お店に予約人数変更をする予定ですが、満席になる可能性もあるので、早めの参加登録をお願いします。
