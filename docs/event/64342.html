受講するとなにができるの？

カラーセラピストとして、カウンセリングを行うことができます

  • 自宅サロンでの開業を考えている方
  • カラーセラピーに興味のある方
  • カラーコーディネイトやパーソナルカラーに
    カラーセラピーの要素も含めたアドバイスを行いたい方
  • パーワーストーンにカラーセラピーをプラスしてサービスを広げたい方　など

仕事として、事業の拡大として、趣味として
カラーセラピーを取り入れて頂くことができます。

他のカラーセラピーとなにが違うの？

日本人向けに作られている・ボトルが綺麗・諸費がトータルすると安い

  1. 日本初のパワーストーン入りカラーボトルを使います
  2. 日本発のカラーセラピーなので色合いや意味が日本人向けになっています
  3. セラピストになるための受講は10時間のみ【希望しない限り追加は必要ありません】
  4. ロイヤリティーなどは発生しません。協会への登録も任意
  5. ボトルやマットなどは強制ではなく、高額なものではありません
