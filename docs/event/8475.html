eZ PublishによるWebCMS活用事例セミナー

description

エンタープライズWebCMSとして世界で様々な事例を持つeZ Publishのセミナーを2010年10月22日、駐日ノルウェー大使館で開催いたします。
2008年11月の日本オフィス誕生から2年近く経過いたしましたが、おかげさまで日本においても多くの事例が生まれました。それらの事例をベースにしたノウハウを中心に、9月末にリリースされるeZ Publish4.4での新機能について、およびiphoneアプリやipadアプリのためのToolkit提供の状況等、eZ Publishの将来、そして、今後の日本におけるeZ Publishの展開について共有させていただければと思います。

また、途中、eZ PublishによるCMSの実装経験もあり、Ecommerce実装の経験も豊富な神戸デジタル・ラボ様による世界No.1のオープンソースEcommerceパッケージ Magentoのご紹介もいただきます。

お忙しい中かとは存じますが、ぜひご参加いただければと思います。

尚、当日は株式会社ミツエーリンクス様の映像チームによるUstreamも予定しております。
