セクシーチョッコレイツ主催のフリージャンルのもくもく会です。

もくもく会とは

参加者が思い思いの作業や話題を持ち寄って一つの場所で行うイベントの一種です。 勉強会のような堅苦しいものではなく、

  1. 各々の作業に没頭してもよし。
  2. 雑談や情報交換をしてもよし。

という気ままなイベントです。このイベントで何かを達成するのが目的ではなく、 普段はなかなか交流のない異業種の方々と出会える場でもあります。

もちろん交流することも必須ではないので、開始から終わりまでずっと一人で仕事をしたり、本を読んだり、プラモデルを作っていても構いません。

ゆるゆるでいきましょう。
