description

星野リゾート 青森屋（青森県三沢市）は、青森が誇る食や文化を人のぬくもりとともに目一杯体感できる宿です。屋台村「もつけんど」では、2月1日～2月29日の期間限定で青森の地酒の「にごり酒フェア」を開催いたします。
【提供銘柄】
八戸酒類三戸工場　三戸のどんべり
寒い冬にじっくり楽しみたい！そんな方におススメです！

八戸酒造　どぶろっく
蔵でしか味わうことが出来なかった、正に活きたぴっちぴちの見た目と味わいが楽しめます。

豊盃「雪燈籠」　活性にごり酒
厳寒の地・青森県弘前市に蔵を構える人気銘柄「豊盃」の冬の風物詩、酒造期だけの季節限定酒として蔵出しされる新酒の活性にごり生酒です。舌を心地よく刺激するピチピチとした炭酸ガスは、和製スパークリングと呼ぶに相応しいフレッシュな美味しさです。
