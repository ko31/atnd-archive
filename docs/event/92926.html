description

「就活で人生の半分が決まる」とよく言われます。 それは「大手・有名企業に入ることが正しい」というようは話しではなく、どのような考え方で就職活動に取り組み、どのような考え方で入社先を決めたのか？が、後々の人生に大きな影響を与える、ということを意味しています。
この講座では、より良い就職活動とより良い将来選択の意思決定を行い、心物両面で豊かな社会人生活を送るための『就活の極意』を皆さんと学んでいきます。
昨年は1000名を超える先輩たちがこの講座を受講し、今年度も現時点で約700名の就活生が受講した、大人気の講座です。

【こんな方にオススメ】

①就活何から手をつけたらいいかわからない
②今は人に自慢できるような実績や経験、実力があるわけではないが、社会に出たら仕事でしっかりと活躍したいと考えている
③同じような境遇の就活仲間から刺激を受けたい
④少しでも早く準備を進めて他の学生に差をつけたい

【開催日程】

2日目：12/3(日) 14:00～19:00

※両日参加が必須となります

１日目
「就活の本質と自己理解」

【受講することで得られるもの】

①自己理解ワークを通じて、「仕事を通じて自分のなりたい姿」をイメージできる

②企業から求められる仕事価値観を学ぶことができる

③webテスト受験し、その結果のデータをもらうことができる

④修了生の先輩から様々な業界・企業の就活体験談を聞くイベントも随時開催！

⑤エントリーシートやグループディスカッション、面接対策といった実践講座も無料で受講できる特典付き！(※スタートアップ講座受講生のみ)

【講師紹介】

入江健一郎
エン・ジャパン株式会社にて、企業の人材採用および組織活性化に関するコンサルティングに従事した後、学生向け就職情報サイトの新規立ち上げプロジェクトに参加。主にキャリア教育セミナーなどユーザー向け新規サービスの開発を担当し、2011年より編集長。大学内での講演実績多数。キャリア教育セミナーの受講者はのべ約5万人。2015年、特定非営利活動法人キャリアクルーズ理事に就任。

中本 佳伸
エン・ジャパン株式会社入社後、主に新卒採用のコンサルティングに従事。国内、外資系大手企業からベンチャー企業まで、数多くの企業の新卒採用活動を設計。これまで3万人を超える大学生の採用選考に立ち会ってきた。その後、新卒採用に関するサービス開発の責任者を経て、2014年より en Career Creation Lab.の立ち上げ責任者に就任。コンサルティングで培ったノウハウを大学生の学びに活かすべくプログラム作りから取り組んでいる。
