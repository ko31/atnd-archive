description

誰でも手軽にiPhone/Android両方のアプリ開発が可能なTitaniumMobile。

『アプリ制作は難しい！』という概念は、もう昔のこと。
だれもがアイディアをアプリにすることが可能な時代となりました。

今回はデジハリのTitaniumMobile講座の カリキュラム開発者であり講師でもある茂木健一氏を招き、
開発環境の魅力、 そして今後のアプリ開発現場の発展予測などを独自の視点で解説します。

アプリ制作の第一歩を踏み出したい！
そんな皆さまにおススメのセミナーとなっております。

【ゲスト】茂木健一氏
ガシュファラ・インク代表取締役社長　デジタルハリウッド大学院客員教授
（株）東洋情報システム退社後、（株）エイチアイ、（株）ジェイアイビー、グランスフィアなど
数社のベンチャー企業の立ち上げ・創業期に参画し、ガシュファラ・インクをUS で起業。
システム開発ではゲーム、M&A企業評価システム、医療DB、動画配信システム、
電子マネーシステム、キャラクターを使ったPush型配信システムなど、多彩な分野を経験。
IT関連教育ではBrew、Java、セキュリティなどの講師として活躍。
