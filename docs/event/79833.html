【セミナー概要】
人工知能が社会に登場するまではIQ（Intelligence Quotient＝知能指数）が良しとされてきました。これはインプットに対してアウトプットを最短で出すという問題解決能力であり、これまでのビジネスや教育で好まれてきた能力です。しかし、インターネットが普及し、クラウドで大量のビッグデータを保有することができ、コンピュータで超高速計算を行うことができるようになった現代、これらのテクノロジーを使ってついに人工知能が誕生しました。さらに、物理世界を移動することができるロボットが登場し、その頭脳に人工知能が採用され始めました。

みなさんはこういった人工知能社会にどう対応しているでしょうか？ またはどのように人工知能を使ってみたいでしょうか？
本セミナーでは、最新人工知能を使用している久保田達也先生（以下、くぼたつさん）を東京からお招きし、大阪でセミナーを開催いたします。

くぼたつさんは３０年以上にわたる企画業務の経験があり、東京を中心にして全国的に法人や学術分野（大学、高校、専門学校、中学校）でコンサルタントおよび教育家として活躍してきました。日本人個人として最初にインターネット上にホームページを開設しました。また、東急ハンズ、LOFT、無印良品、電通、東京ディズニーランドなどの企画関連業務を歴任しました。さらに、Googleのワークショップにて共同研究を行いました。現在は事業を通してその思考ノウハウを後世に伝え、多くの起業家を輩出し、教育分野から若人たちを未来へ導いています。

人工知能が社会のサービスに組み込まれてきた今、IQは人工知能が担い、人間は人工知能のIQに到底及びません。
では、私たちはどうすればいいのでしょうか？

くぼたつさんは最新人工知能のテスターとして、その利用経験からどうすれば人間が人間らしくいられるかを体験から学び、その秘訣をセミナー形式で演習学習しながら、本来、人が持つ創造的能力の向上を育成することで日本から人工知能社会に対応できる人を一人でも多く輩出することを願っております。

本セミナーでは、３０年間に渡る企画業務から発明されたノウハウを集大成した『くぼたつ式思考カード54』を使って、来るべき人工知能社会に適応できる能力を磨きます。

みんなで一緒に楽しく学び、共通の関心事を持つ仲間たちとも知り合えます。ネット社会が浸透した現在、片手一つで相手にチャットやソーシャルのやりとりが手軽にできるようになったからこそ、今まで以上にリアル交流が大切になりました。ビジョンや価値観を共有でき、考えや意見を言い合える仲間がとても重要です。このセミナーがきっかけで、たくさんの方々と交流できることを楽しみにしております。

下記のようなことに興味ある方々はストライクゾーンに入るかと思います。
　・来るべき人工知能社会に適応したい
　・人工知能に興味がある
　・人工知能を使いこなして何かに挑戦したい
　・起業したい
　・アイデアや発想力を伸ばしたい
　・これらに興味ある有志たちと出会って交流を広めたい

【講師・久保田達也 先生（愛称・くぼたつさん）の紹介】
東急ハンズ、電通などの企画業務を歴任し、 1985年に(株)イッツ設立、代表取締役就任。イタリアンデザート「ティラミス」の日本市場開拓、インターネットを使い世界を探検する企画マンとして研究、執筆、講演活動中。教育分野ではサイバー大学 IT総合学部 教授、モバイル教育研究所 所長、金沢工業大学 教授など歴任した。
