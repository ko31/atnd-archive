FLASH10 – ActionScript3.0 初心者勉強会


難易度は『割と簡単』です。



今回のActionScript3.0勉強会は、PHP初心者勉強会の1周年記念行事で行ったActionScript勉強会が好評で、リクエストが多いために再び開催します。

PHP初心者勉強会1周年記念大勉強会
http://atnd.org/events/1806

PHP初心者勉強会は定期的に開催しています。
http://sites.google.com/site/phpbeginners

普段の勉強会は宿題方式ですが、今回は実際に作成しながら勉強会を進めます。

  1. 例題に沿って、実際にプログラムを作成します。

  2. FlashDevelopというフリーソフトを使用します。

  3. Windowsノートパソコンが必須です。

  4. 飲食物の持ち込みはできません。

  5. 会議室の利用料が1,500円かかります。

  6. インターネットは有線だけになります。
