■詳細は……

　こちらのブログ記事を参照してください。

  • いつも二人で話すときのように小声でぼそぼそと短歌の話をするのを、みんなで盗み聞きしているみたいな小さなイベント。
  • 何を話すかは、まだ相談していません。2004年に出会って「かなしーおもちゃ」という本を一緒につくってからの10年間に二人が見てきた、短歌や出版にまつわる話ができれば。
  • いまだに誤解の多い、枡野さんがなぜ芸人を目指し、そして頓挫したかの話も、きっとすることになるのでしょう。
  • お店はご飯がとても美味しいので、真夜中に気が向いたときになんだか美味しいものをちょっとつまんだりすることができます。
  • 夏の夜のゆるゆるしたイベントです。徹底的に短歌の議論がしたいというような方より、明け方まで続くとりとめのない話をまどろみながら聞いてみるのもいいかな、というぐらいの方におすすめです。
