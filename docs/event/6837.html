description

Vine LinuxユーザやVine Linuxに興味がある方で、会議室を借りてオフラインで集まろうというイベントです。
セミナー形式ではありませんのでご注意ください (*1)。

Project Vineメンバや開発者も参加します (*2) ので、Linuxの使い方の質問なども可能です。

今回は「vlm201008」です。

ツイッターのハッシュタグ #vlm201008 や
ブログ、SBM などのタグに使ってください。

今回は、"みんなでドキュメント更新大会" としまして、
Vine-manual を初め、po や spec などの翻訳、
Web ページの更新や Trac/Wiki の整理など、
ドキュメント作業を黙々と行うイベントとなっております。
もちろん、ワイワイと議論しながら行っていただいても構いません。

また、ドキュメントチームのリーダ赤星さんに「ドキュメントチームの現状と今後について（仮）」をお話していただくかもしれません。
その上でドキュメント作成ハンズオンなどを計画中です。

なお、今回はプロジェクタがありません。

今回の会場は、利用料金（会場費）2,580 円を参加者で割り勘します。10 名参加ですと、300 円でおつりがくる計算です。

飲食の持込は自由ですが、ゴミは各自お持ち帰り下さい。

*1: セミナー形式のイベントも年に4回ぐらいのペースで別途企画される予定です。ご期待ください。

*2: 毎回参加しているとは限りません。参加表明者のリストをご覧ください。

定員がありますし、料金の支払いもありますので、当日これなくなってしまった方はATNDで必ずキャンセルをお願いします。

関連イベントとしまして、Doc/TransFesta Tokyo 2010 が開催されます。ドキュメント・翻訳に興味のある方はぜひとも合わせてご参加下さいませ。
