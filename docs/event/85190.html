Web系適職診断＋キャリアデザイン相談会

Web系の全職業を体験、何が自分に向いているか適職診断をしよう！

こんな方にオススメ！

  • これから就職を考える大学3年生
  • 今年中に就職を決めたい大学4年生
  • 第二新卒
  • 本当に向いているかわからないまま就職してしまい、転職を考えたい方

「Web業界ってどんな仕事があるの？」 「興味はあるけど、仕事がよくわからない」 「収入は？何をどれくらい学べばいい？」 など、様々な疑問を一気に解決できるよう、ディレクション、デザイン、コーディング、プログラミングすべての仕事を少しずつ体験して頂きます。 初心者「だからこそ」歓迎の、初めてのWeb体験会です。

セミナー内容

Program1
Web系のすべての仕事を体験して・適職診断をしよう

実際の仕事と同じように、ひとつの案件をどのような職種の人がどのような仕事で進めるのかを体験していただきます！ 現場と同じ仕事を体験することで、あなたに合った仕事をみつけることができます！

Web業界へのカウンセリング

いろいろな仕事を経験してみて、「ならどんな勉強をすればいい？」「Web業界ならではの面接対策はあるの？」など、ご希望に応じて個別のカウンセリングも受けて頂けます！

Special Program
「文系からエンジニアに内定した成功体験談」

牧野勇太

プロフィール：
「技術力で評価され、何か残るものを作りたい、発信したい」という理由でWeb業界を志望し、ネットで検索。 プロとして内定を獲得するには個別指導で徹底的にプロから学んだ方が良いと判断してWEB塾に入学。 まったくの初めてからの学習だったので大変さを感じながらもHTML,CSS,jQueryをはじめWeb業界に必要なスキルを習得し、株式会社antにフロントエンジニアとして内定。

Program2 キャリアデザイン相談会

「やりたいことが見つからない」「目標を持って今の学部に入学したけど迷い始めた」「やりたいことがあるけど自信が無い」「夢を描いてもどうせ無理だろうと考えてしまう」等、こんな自分でも大丈夫なのだろうか・・・現在の大学生のリアルな悩みとして聞こえて来る声は将来に対する不安です。

職種はこの世の中に約3万種と言われていますが、まず、その中のひとつを選んで働くのですから、何をどうすれば良いのか分からなくて当然と思います。 そして、そのひとつの職種が一生の仕事になるかもしれないと考えると悩みますよね。

売り手と言われる市場でも油断は禁物です。 不安を抱えたままエントリーシートを書こうとしても、面接に臨もうとしても、良い結果を招きません。

今回のキャリアデザインセミナーは30分間ですが、自分の内的な部分に焦点を当て、ピンッと来るキーワードから、漠然としたものが確信に変わるきっかけを掴み、自分に合った仕事を見つける方法を体感していただきます。
