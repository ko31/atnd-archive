■参加方法

  1. 「Reaching for the Stars」を踊っている動画を撮影して下さい。
    　※お一人様1回投稿可能です。
  2. 動画をYouTube 又は ファイル共有サイト (DropBoxなど) へアップロード下さい。
    　※音声入れ替え不要。
    　※スマホやタブレットで撮影頂いたものをそのままアップ頂いてOK。
    　※既にアップロード済みのもので良ければ撮り直し不要。
  3. 下記の「必要事項」をアンケートにご記入下さい。

＜必要事項＞

  1. サムネイル希望
    　※必須
    　「希望しない」にチェックするとサムネイル条件に合っても１番になりません。
  2. 動画に表示するお名前
    　※無記名でもOK
  3. 動画に表示するプレイ場所（ゲームセンター名、撮影場所など）
    　※無記名でもOK
  4. 動画のアドレス(2.でアップロードしたアドレス)
    　※必須

■編集した動画について

  1. 編集した動画はこちらのチャンネルで公開します。
  2. マスターデータが欲しい方は、ご一報ください。
  3. 制作した動画をYoutubeやニコニコ動画に転載されたい方は、ご一報下さい。
  4. 許可なく動画を転載する行為は禁止します。

■再生順／サムネイルについて

  1. サムネイル及び再生順は、アミダクジで決めます。
  2. サムネイルは、過去リレー動画でサムネイルに採用されていない方を優先します。
  3. サムネイル採用者は、無条件で再生順１番となります。
  4. サムネイル「希望しない」を選択された方サムネイルになりませんが同時に１番にもなれません。

■注意事項

  1. 下記の場合、「不参加」として扱います。
    1. 正規の手順以外で投稿された方
    2. 途中で途切れている動画を投稿された方
    3. 無断で締め切り以降に投稿された方
    4. 再生位置を指定された方
  2. お一人様１回限り、投稿者がダンスしている動画のみ投稿頂けます。
  3. 既に動画を投稿された方が別の方の動画に映ってもＯＫです。ただし、意図的に何度も繰り返し登場する場合は、参加をご辞退頂く場合があります。
  4. 編集は、たけぷ～ が実施します。
  5. 投稿頂いた元の動画に対し、大幅な改編は行いません。
  6. 公開後、諸事情により編集した動画を削除する可能性があります。
