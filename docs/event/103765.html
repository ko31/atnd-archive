description

東京から約2時間、山梨県小菅村は東京湾に注ぐ「多摩川」の源流に位置しています。豊かな山里の自然や代々受け継がれてきた伝統や暮らしが残り、散策やアウトドアを楽しむことができるスポットも満載です。

また、地域おこしに興味のある若者や田舎暮らしに興味のある親子の移住も多く受け入れています。

そうは言っても移住は大きな決断。
　「移住には興味があるけどイメージがわかない・・」
　「協力隊ってどんなお仕事しているのか、実際目で見てみたい・・」
そういった方も多いのではないでしょうか？

そんな皆様のために、小菅村の暮らしの「リアル」をお伝えするスタディツアー（日帰り）を実施します！

・第2回　2019年2月16日（土）
