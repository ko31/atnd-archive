夏のCPU殻割り大会

イベント ユーザー

title 夏のCPU殻割り大会
catch
description

Intel CPUの殻割りツール「Rockit 88」を使ってさまざまなCPUを割ります

PC Watch編集部員とインプレスの愉快な仲間達がCPU殻割り専用ツールを使った殻割りを実況します。専用ツールの使い方から、殻割り後の冷却方法、OCにおける効果の検証などを行なう予定です。CPUは各スタッフの私物なり。

応募/参加の際の注意事項

  • Rockit 88で殻割りできるCPUを必ず最低1つ持参してください
  • LGA1366/LGA2011系のCPU、AMD全てのCPUは割れませんのでご了承ください
  • 弊社オフィスまでの交通費はご自身でご負担ください
  • 殻割り後、グリスをLiquid Proなどの液体金属に置き換える場合はご自身でご購入ください
  • 殻割りによってCPUが使用不能になってもPC Watchおよびインプレスでは一切の責任を負いかねます
  • 日時:2016年7月5日　19時～
  • 場所:弊社オフィス　東京都千代田区神田神保町1-105 23階
  • 応募多数の場合は抽選となり、弊社より追ってメールにてご連絡いたします
  • 当日は本人確認ができるものをお持ちください
