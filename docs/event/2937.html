2月6日(土)の18時よりAppleStore銀座店にて、三枝こころアプリ発売記念としてイベントが開催されます。

銀座店のイベント自体は19時終了予定ですが、AppBankメンバーがその時間帯に都内にいることはめったにないため、せっかくなのでその後オフ会を開催しようと思っています。

時間：19時半〜21時半

お店：miso bank(名前ねらってないよ)

場所：AppleStore銀座店のすぐ近くだよ。詳しくはHP見て。

会費：4000円(料理＋2時間飲み放題とのこと)（予定）

幹事：@nashiko_さん(http://twitter.com/nashiko_)

なお、当日はAppBankからリリース直前のポケットベガスを総合プロデューサーである@entrypostmanが首からiPhone2台ぶらさげてお店の中をうろつきますので、ポケットベガス第1弾ゲームである「クロンダイクソリティア」で勝負してください。1対1です。勝者にはその場でiTunesカード1500円分をプレゼントします。

※その場でルール教えて、とかだと進まなくなるので、対戦希望の方は事前にルール勉強しておいてください！

※未成年は参加不可です！

※キャンセルは前日までにお願いします。

では、多くの皆様ご参加をお待ちしております！

※なお、三枝こころは参加しません。
