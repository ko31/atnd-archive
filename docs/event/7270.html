Twitter, Facebookなど、ソーシャルネットワークサービスの全てを知り尽くしたコンサルタントによるセミナー＆ワークショップを開催します。

■あなたが得られるもの
パーソナルブランドの形成、Webでの総合的な自己発信力（マーケティング力）が身に付きます。つまり、先端的ポジショニングの確保をすることができるようになります。様々なソーシャルネットワークサービスが一つのものとして繋がります。

■主な対象者
この講座は主に以下の方を対象としています。

など

　その他、ソーシャルグラフ、社会起業、ソーシャルコマース等の分野にも踏み込みます。
