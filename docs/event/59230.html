description

クリスマス前にスイーツ写真の撮り方を学んでみませんか？

この講座ではスイーツをうまく綺麗に撮影する方法をレッスンします。ブログやフェイスブックなどにスイーツの写真をアップされている方も多いと思いますが、うまく撮影するコツをプロのフォトグラファーが教えます。

iPhoneなどスマホでの参加もOKですので、初心者の方、またこれからカメラを始めてみたい方も大歓迎です。
