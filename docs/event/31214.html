Hackathon for SamurAI Coding 2012 @ 関東
（関東・関西同時開催）

目的

  1. SamurAI Coding 2012の告知・ルール説明
  2. AIプログラムの簡単な開発方法の解説
  3. 国内参加者同士の交流機会の創造

場所

関東・関西の同時開催

  • 関東：早稲田大学　グリーン・コンピューティング・システム研究機構
    入館にはカードキーが必要ですので，玄関でお待ち頂くかスタッフまで連絡下さい（090-7800-6023：坂本）
    場所（ http://www.kikou.waseda.ac.jp/gcs/contact/index.html ）
    ATND（ http://atnd.org/events/31214 ）
  • 関西：京都西陣 IT路地 セミナールーム
    場所（ http://it-rouji.net/access/index.html ）
    ATND（ http://atnd.org/events/31469 ）

タイムテーブル

時間 内容
14:00 – 14:10 オープニング，SamurAI Coding 2012の紹介
14:10 – 15:00 ゲームルールの説明
AIプログラム開発の簡単な解説
15:00 – 17:00 各自でAIプログラムの開発
17:00 – 17:45 お互いに開発したAIプログラムを見せ合う
関東・関西で連絡を取り合う
17:50 – 18:00 クロージング

SamurAI coding 2012 とは

  • 競技内容
    1. 陣取りゲームのAIプログラムを作成して対戦
    2. http://members.samuraicoding.org
    3. ログインするとルールのドキュメントを入手可能
    4. アンオフィシャルなサイト
    5. https://sites.google.com/site/samuraicodingjapan
  • 参加条件：
    1. 25歳以下（1987年1月1日以降に生まれた方）
    2. チーム制：1チーム3〜5名（決勝には最大で3名が参加可能）
  • 公用語：英語
