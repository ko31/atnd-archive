【現在25名】株式会社ディー・エヌ・エーの人事が「キャリアの在りかた」について本質を語るトークセッションのご案内です！

　

■テーマ：企業がいま、人材紹介エージェントに期待すること

　 エンジニアやクリエイターの人材不足が大きな課題となっている現状から、採用成功のためのエッセンスであったり、人材紹介会社に期待していることを株式会社ディー・エヌ・エーの人事担当　大月様より語っていただきます。 本音ベースでぶつかる為に、アールストーンの切り込み隊長とのトークセッションという形式で開催いたしますので、ご興味のある人材紹介関係者の方はぜひご参加ください！ 　

■日時

　2016年3月22日(火) 　20:00～21:30　※開場19:45～ 　

■会場

　ポイント渋谷道玄坂 　東京都渋谷区円山町1-3 猪鼻ビル2F 　

■定員

　20名 ⇒　30名に増枠します（現在25名） 　

■参加資格

　人材紹介業界でご活躍中の方 　人材紹介業界にご興味をお持ちの方 　

■持ち物

　名刺を2枚ご提示いただきます 　※懇親会では名刺交換も予定しております 　

■参加費

　無料

＜パネラー＞

①DeNA 技術企画グループ（中途採用担当）

　大月 英照様 1980年生まれ、工学修士（MOT） 学生時代にエンジニアとしてベンチャーを起業。 その後エンジニアとして大手SIerにシステム開発、PMを経験。 さらに大手人材紹介会社にてIT・Web業界向けの法人営業、テクニカルアドバイザーとして従事。 現在はDeNAにてエンジニアを中心に中途採用、新卒教育、技術広報やブランディング各種イベントの企画、実施を行っている。

②アールストーン　営業本部長

　 ITバブルが崩壊した2000年代初頭にヘッドハンティングに従事。インターネット業界の創成期からネットベンチャーを中心にエグゼクティブの採用を支援。 その後は上場企業で人材紹介事業を立ち上げ、現在もWebクリエイター、ITエンジニアに特化した転職コンサルティングでエグゼクティブ層からミドル層まで幅広い方々をサポートしている。
