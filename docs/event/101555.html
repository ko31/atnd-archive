【チャリティーイベント いやしの祭典 2018 サテライト福山】
　～福山にビューティ＆ウェルネスが集結！！食と暮らしにやさしいマーケット～

主催：AromaRUB
後援：日本アロマコーディネーター協会
協賛企業様　30社
開催日時：11月23日（金・祝）10：30～16：30　
会場：福山市ものづくり交流館
住所：〒720-0067 福山市西町1-1-1 エフピコRiM9F
山陽新幹線・山陽本線・福塩線 福山駅から徒歩5分
山陽自動車道 福山東I.C.から車で20分

「いやしの祭典」とは
いやし」「健康」「自然」といったテーマを通じ、アロマセラピー、その他各種のセラピー、カルチャー、地域物産などを取り上げ、国内でのアロマセラピーや自然療法の普及、会員のスキルUPや地域交流の場を提供することを目的に、2012年から開催しています。
毎年各会場ともアロマセラピーその他の企業、メーカー、団体が多数出展し、展示即売、セラピー体験、ワークショップなどを開いています。
また、著名な講師をお招きしてのセミナーも開催しています。
収益金の一部は、毎年チャリティーとして東日本大震災遺児支援募金（あしなが育英会）、その他団体へ募金させていただいています。
また、全国でボランティア活動をされるJAA会員の方々への支援にも使用されています。

【セミナー】
開催時間：Ⅰ部　11：00～12：30　Ⅱ部　13：00～15：00
～Ⅰ部内容～
講師：国際薬膳学院 学院長　赤堀 真澄 先生
「風邪予防＆免疫アップの薬膳　晩秋から冬の薬膳的過ごし方」
～Ⅱ部内容～
講師：株式会社MANABICIA 代表　池原 真佐子 先生
「コンプレックスを魅力に変える講演会～自信をもつって美しい～」
セミナーでは、ナチュラルで健康的なライフスタイルや、自分の魅力を最大限に引き出して望むキャリアと人生を手に入れる方法を学んだりいたします。

【体験ブース】
福山、尾道、広島、岡山から
選りすぐりのセラピストが集結！！
今まで興味はあるけど受けられなかった、セラピーやセッションが盛りだくさん！
この機会に身近にセラピストさんたちと触れ合い、ご自身にピッタリの癒しのツールをみつけてみてくださいね。

【マルシェ】
セミナーでお勉強したり、癒しの体験を堪能した後は
ナチュラルで美味しいものを、食べたり飲んだり
ご当地のアロマや石鹸、梅香油、高品質なオーストラリアのホホバオイル、布ナプキンなど身体に優しいお買い物を楽しんで下さい!!
