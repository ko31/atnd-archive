応募方法

  1. 右の参加リンクから参加してください
  2. 話す内容が決まっていたら、コメント欄に下記の内容を書いてください
    • 発表タイトル
    • 通常トラックかLTか
    • 発表に要する時間
      • 通常トラック : 10, 15, 20, 25, 30, 35, 40分のいずれか
      • LT : 3分もしくは5分
    • 発表内容のサマリ
    • 発表者のURL
    • 発表者のtwitter id
  3. 話す内容が決まっていなければ、決まり次第コメント欄に前述の内容を書いてください
