description

受付システム(ATND)の使い方が良くわからない、という方はmaimuzo@gmail.comに直接メールいただいても大丈夫です

OSC2009北海道で宣言したとおり、北海道でAndroidのHello worldハンズオンを開催しようと思います。

また、自分である程度Androidアプリを作れる方向けに、ハッカソンも併催します。

開催日時: 2009年7月25日(土曜) 13:30～16:30
(開場は13:00ですが準備の都合により開始は13:30となります)

会場: 札幌市産業振興センター　技能訓練棟3F　セミナールーム1
http://sangyo.center.sec.or.jp/
※昨年11月に第１回Android勉強会を開いた会場です

住所: 札幌市白石区東札幌５条１丁目１－１
http://sangyo.center.sec.or.jp/access/map.html

参加料: 500円(会場費として)

内容:
ハンズオンについては、下記テキストに沿って講義と実習を繰り返すスタイルです。
http://java-sapporo.googlegroups.com/web/android-handson-text-pre.pdf?gda=Bi5LBFAAAADucAiDWyOFvMdW0EWgBaaW6Y2QwF9KhyTrJPjyaWGFyaWAbFI3GGe1VdxbeOZsRGYw4OHyB5jiJFx_qhtvM59qo5-brIMPe3eIzqC7EbFFtA&gsc=g87DBQsAAAA9M431O4ChUtRgoFWpwQTd
※最終版ではないので、後ほど最終版のURLをお送りします。
※時間の都合により全エクササイズを消化できないかも知れません。

ハッカソンについては、参加者各自にて目標を決め、時間内に動くものを作って最後に発表するスタイルです。
個人でも、グループでも参加は自由です。
ハンズオンと同室ですが、離れ島を作ってもくもくしてもらうつもりです。

必要なもの:
・ノートPCを持参ください。会場側ではPCは用意しません。
・ノートPCにはandroid-handson-text-pre.pdfを参考にAndroid SDK 1.5 R2と、Eclipse、ADT(Android Development Tools)をインストールしておいてください。ハンズオンはインストール済みである前提で始めます。
http://developer.android.com/sdk/1.5_r2/index.html
・ハンズオン参加予定者は、必要であれば後ほどお伝えする最終版のテキスト(PDF)を印刷して持参ください。当日はスクリーンに投影しますし、PCでPDFを開いていてもOKです。
・ハッカソン参加予定者は、当日コーディングに集中できるように、下調べや設計などを済ませておいてください。
※うまくインストールできなかった場合は、会場にちょっと早めに来ると、インストールのお手伝いできるかも?
※会場では無線LANを使用可能とする予定ですが、ネットワーク設備の関係上HTTPプロトコルのみ使用可能となります。

講師: 我らが木南さんが担当くださる予定です
http://d.hatena.ne.jp/hkinami/

懇親会: ハンズオン終了後には近くで懇親会を開催予定です。懇親会参加予定人数を見ながら詳細を詰めようと思っております。予算は4000円弱で考えています。
※都合により木南さんは参加されないそうです。ご注意ください

ちょうどDocomoから日本初のAndroid端末であるHT-03Aが発売されたタイミングですので、実機をお持ちの方もいらっしゃると思います。
Androidのプログラミングは、実際にAndroidケータイ上で自分の作ったプログラムが動くとすごくモチベーションが上がります。一度プログラミングの流れがわかると、その後は比較的アイデア勝負な部分もありますので、この機会に波に乗ってください。=)
