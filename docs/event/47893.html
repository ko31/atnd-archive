iWest勉強会開催のお知らせです。次回（第15回）勉強会は、2014年3月8日(土)の午後に開催いたします。テーマはただいま調整中です。

また、勉強会終了後は懇親会も予定しております。都合のよい方はこちらにも引き続きご参加ください。

皆さまのご参加をお待ちしております。

【日　時】2014年3月8日(土) 13:30〜16:30
【会　場】ふくふくプラザ（福岡市市民福祉プラザ）602研修室
【テーマ】iPhone／iPad／Mac活用意見交換会

今回は、いつものように講師がお話をする形式ではなく、参加した皆さまからApple製品の活用法をご紹介いただいたり、日頃の疑問や質問をみんなで解決する『iPhone／iPad／Mac活用意見交換会』を開催します。

例えば…

・お気に入りのアプリを紹介したい
・面白いサービスを発見したので紹介したい
・アプリを作ったので見てほしい
・最近買ったデジタルガジェットを紹介したい

といった情報共有（自慢？）や、

・こういう用途のアプリを探しています
・購入したけどイマイチ使い方がわからない
・トラブルの解決方法を教えてほしい
・購入を考えているけど使いこなせるか不安

といった質問などなど。みんなで自由にテーマを出し合って、楽しく意見交換をしましょう。

会場にはプロジェクターとプロジェクタに接続するアダプタ（Lightning／Dockコネクタ／Mini DisplayPort）、スピーカーをご用意いたしますので、大画面でのアプリ紹介（ゲーム！）も可能です。

もちろん、紹介したり質問したいことはないけど、お話だけ聞きたいという方も大歓迎！

皆さまのご参加をお待ちしております。

【参加費】無料（懇親会参加者は3,000円）
