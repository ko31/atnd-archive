【映画タイタニックの最も重要なシーンはここだ！】

映画・ゲーム・出版業界に興味のある方であれば、どなたでも新発見のある「北加賀屋シネマスタート」第4弾（2015年初）を開催いたします。

2014年1月以降、下松シネマトーク、神戸元町シネマスタートを含め、今回で8回目の開催。講演内容はさらにバージョンアップしております。

【開催内容】

マスコミ向け映画資料（プレスシート）の展示・閲覧をはじめ、プレゼン、座談会を行う予定です。

※途中参加・退出自由です。

■意外と知らない映画ライターの日常（19:30〜20:10）

例えば、映画ライターは年間何本見ているのか、雑誌の原稿ギャラはいくらなのか…といった映画メディア・出版業界などのお話をします。

■映画「タイタニック」の最も重要のシーン（34秒）はここだ！（20:15〜21:00）

シナリオ的分析から、映画「タイタニック」の最も重要なシーンを解説。
この映画の見方を覚えただけで、明日から映画の見方がちょっと変わります。

■座談会（21:00〜）

参加者で好きな映画タイトルの話などを語り合います。

■開催日時：4月11日（土）19:30〜21:30（19:00開場）
■開催場所：コワーキングスペース「往来」
■住所：大阪市中央区谷町6-5-26 複合文化施設「萌」2F
　　　　TEL 06-6796-8834
http://www.ourai.jimdo.com

■参加費：1000円
