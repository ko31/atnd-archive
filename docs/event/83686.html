description

マーケターたちが今抱えている課題を本音で語り合う

こんな課題をもつ人のための勉強会です

  • 先入観のない違う業界の人たちの話を聞いてみたい
  • 自分の抱えている課題にダイレクトなヒントや気づきがほしい
  • 具体的な体験談や事例を交えた話が聞きたい
  • 交流会を通して悩みを相談できるような人脈を広げたい

勉強会・交流会を通して得られるであろうメリット

  • 抱えている課題に対するヒントや気づきを、
    自社と利害がない（可能性が高い）立場の第三者から得られる
  • 質問もその場でできるグループディスカッション形式なので、
    一般的な勉強会では得にくい、自社での応用等具体的な解が見つけやすい
  • マーケティング担当者たちの課題意識は今どこにあるのかの傾向がつかめる
  • リアルな繋がりができることで、視野やビジネスチャンスが広がる可能性がある

対象者

企業のマーケティング担当者

内容詳細

4～5名の小グループを作り、
現在どのような業務や役割を担っていて、どこに課題や
問題意識を持っているのかを5分程度でお話しいただき、
その後10分で各グループメンバーより意見やアイデアをもらいます。

※社内のブレストよりも、先入観や業界の慣習などに縛られていない
外部の方のほうが新たな気づきやアイデアがでやすいと考えています。
また、多くの経験値がある人も参加するので、そのケーススタディも
聞ける貴重な機会になるかと思います。

その後、10分程度で代表者がグループででた課題とアイデアを
皆に共有します。

ワークショップ後には交流会も予定しています。

形式

お酒を飲みながら気楽に参加できる、ビアバッシュ形式です。

会費

1,000円　※会費は当日現地にて徴収します
