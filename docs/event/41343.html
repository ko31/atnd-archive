(2013.7.28追記)
誠に勝手ながら都合により今回の観戦オフは中止とさせていただきます。

今年もファイターズの二軍が北海道にやってきます！ まずは滝川でスワローズ戦！ 昨年は北海道で4戦負けなしだっただけに今年も…！

＊開催要項＊

 試合の開催要項に関してはファイターズ公式サイトをご覧ください。
・交通
今回は私が参加する皆さんの近所まで（可能な限り^^；；）送迎します。
参加者がまとまった時点で時間調整をいたします。
そのため募集人数が少なめとなっております。
（車を出せる方が増えればこの限りではありません）
・飲食
当日は売店も出ますが、混雑が予想されるのである程度は持参しておいた方がいいかもしれません。
・チケット
「内・外野芝生席」を購入願います（前売大人1,000円、当日大人1,500円）。
なお、オフィシャルショップや札幌ドームまで買いに行けないという方は代理購入を承ります。
（締切7月12日）
代理購入された方のチケットは当日引き換えとなります。
・その他
熱中症、日射病、日焼け、虫よけは各自で対策願います。

＊中止の場合＊
雨天、荒天、その他の理由などで試合が中止になった場合、この観戦オフも中止とします。
チケットの払い戻しは各自でお願いします。
ただし、代理購入された分は当方で払い戻しますので特に手続きは必要ありません。
