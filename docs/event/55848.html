description

英語は誰にだって話せるようになります！

年齢だって学歴だって関係ありません。文法や構文、TOEICばかりの勉強では何歳から始めようが、何年続けようが話せるようにはなりません。また、日常に必要な英会話力をつける練習方法に高額な授業料も必要ありません。

１人でも多くの人に英会話力の獲得を通じて、広い世界を見て頂きたいと思います。また、英会話力が自分の目標の障壁となっている方の手助けができれば幸甚です。

【こんな方にオススメ】
・英語が話せるようになりたい方、話すための練習方法が知りたい方
・発音、リスニングが苦手な方
・どんな教材を使えばいいのかわからない方
・語学留学の経歴を就職活動でどうアピールしていいかわからない方
・ニューヨークでのインターンシップ、ボランティア活動に興味がある方
・ビジネスで英語を使う機会がある方
・TOEICの点数を上げたい方

【セミナーの期待効果】
・英語に関する苦手意識を克服できる
・英語を武器に持つことで人生の選択肢を増やすことができる
・高額な授業料を軽減することができる

【セミナー内容】
・英語を話す力すなわち”英会話力の獲得”のための英語練習法をお伝えします
　終盤では実際に練習法を体験してもらうデモプラクティスも実施いたします

※少人数で実施するため、質問しやすい環境となっております

【アドバイザー】
眞壁　暁(マカベ　アキラ)

・中学、高校時代の英語の成績は常に「３」(※５段階評価)
　英語に対し苦手意識こそなかったが、決して得意とは言えない立ち位置が続く

・大学入学後、社会学を専攻する傍ら独学で"会話力"に重点を置いた英会話練習を開始
　一銭も英会話スクールに投じることなく英会話力を獲得

“この英会話力を使い自分には何ができるのか”

・２１歳の時、大学を休学し、１年間単身でニューヨークへ留学
　⇒地域の住民を巻き込み文化交流サークルを立ち上げる(参加者30人以上)
　⇒マンハッタンにある出版会社にてインターンシップを経験
　⇒非営利団体"Herts&Minds"に所属。日本人１人の環境の中、電話応対による基金調達のボランティアを経験

・帰国後、小田急電鉄株式会社にて日英通訳業務を経験

・大学卒業後、大手工作機械商社の海外営業部に所属
　アメリカ、カナダ、タイ市場を担当

・2014年より日本で販路を獲得したい海外の方へのコンサルティングを開始

※３度の飯よりラーメンが好き。ラーメンに対する思いは世界一を自負
