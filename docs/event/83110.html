◆　11/8(火)11/9(水)　12:30—16:00

■スマホ対応の会員限定ページを持ったサイトを、ドメイン・サーバーの用意から始め、素人でも二日で完成させる,経験20年のベテラン講師による驚きのワークショップ。
ゴールド会員、シルバー会員、通常会員などいろんな種類・レベルの会員を作成、有効期限も設定。そして、その会員だけが閲覧できるページを作成。複数のページをカテゴリーAとして、カテゴリーごとに制限をつけることも可能。また、会員登録情報を各自が変更できるページも作成。会員用のログインエリアもサイトのトップページのすきな場所に配置可能。
参加者一人ひとりにドメインとサーバーを用意、落ち着いて完成まで取り組める。

■用意するもの：　使い慣れたノートブックPC（Mac /Windows 問わず）

＜ワークショップ内容＞
1.　ドメイン設定 / サーバー設定
　　*参加者一人ひとりにドメインとサーバースペースを設定
2.　基本システム　アップロード　設定
3.　基本システム 初期説明・操作実習
　　01　 サイトデザイン基本設定
　　02　 ページ作成・メニュー作成
　—質問タイム—
4.　会員制システム　インストール
5.　会員設定・閲覧ページ作成
　　01　 メンバーの種類・レベルを設定
　　02　各メンバーの有効期間を設定
　　03　だれでも閲覧できるページA作成
　　04　メンバーだけが閲覧できるページB作成
　　05　メンバーの中で有料会員だけが閲覧できるページC作成
　—質問タイム—
6.　会員登録のページ作成
7.　会員登録し、自分のシステムが正常に稼動しているか確認する
　—質問タイム—
8.　希望者には　補講タイム　1時間
