名古屋のWindows Phone 7 ハッカソンが帰ってきた！！

ハッカソンとは

みんなで集まり、アプリの開発・発表をする会のこと。
昨年末に開催し、好評でしたので第２回を開催します。

参加するとこんな良いことが！

  • みんなと一緒にアプリ開発ができる
  • WPアプリ開発者との交流ができる
  • 自分の知らない、技術・情報が共有できる

※今回も日本マイクロソフト株式会社のエバンジェリストの 大西 彰さん をお招きする予定です。

イベント内容

みんなでワイワイ／ガヤガヤ、WP7のアプリ作っちゃいます。
自由なアイデアでみなさんとご一緒にアプリ開発を楽しみましょう。

参加条件

特にございません。初めての方、初心者の方、大歓迎です！
デベロッパー、デザイナー何方でもご参加頂けます。
開発用PCを用意してください。

au NAGOYA

用意して頂くもの

  • 開発PC
    • OS Windwos Vista 又は Windows 7 (WindowsXPは使用できません)
    • 仮想環境は非推奨です Macを使う場合はBootcampでの動作を推奨します。
  • Windows Phone（用意出来る方）

※開発用PCには下記のSDKをインストールし、開発環境とエミュレータの実行が問題なく行えることを確認して参加をお願いいたします。

Windows Phone SDK 7.1
Zune Software
