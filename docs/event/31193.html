TeX Cafe Aug. 2012: 暑い夏に TeX で熱く盛り上がりましょう！

Cafe でまったりと、TeX なひと時を過ごしてみませんか？

TeX Cafe のテーマ

利用者、現場、開発者の情報共有 がテーマです。

基本的には、TeX を使っているさまざまな立場の人々が集い、TeX に関することを何かやろうということを想定しています。

TeX について雑談したり、黙々と執筆や製作作業をしたり、開発したり、クラスファイルを設計したり、マクロを書いてスタイルファイルを作ったり、お絵かきしたりなどなど。もちろん、TeX 環境の構築やインストールのやり方、GUI による製作環境なども OK です。

また、組版やフォント、デザイン、印刷に関すること、最近なにかと話題の電子書籍も何でもありです！

対象

あらゆる TeX ユーザが対象です！

  • 今日から TeX 組版環境を整えるという超初心者の方
  • 卒論を書くために、LaTeX を初めて間もない初心者の方
  • LaTeX でなにか執筆、製作なさっている方
  • TeX/LaTeX/ConTeXt や関連ツール、アプリケーションなどのなにか作業をしたい方、開発したい方
  • ほかの人がどんな TeX 環境を使っているのか、ひたすらのぞいてみたい方（笑
  • とにかく、TeX が好きな方（笑
