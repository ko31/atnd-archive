"シューマイ"とは？

本イベントは、“世界をテックリードする日本人エンジニアを多く輩出する”をビジョンに、 日本のエンジニアのレベルの底上げを目指すコミュニティです！

登壇者

【戸澤 拓也氏＊みんなのマーケット株式会社 CTO】
大学在学中より、みんなのマーケット株式会社にてエンジニアとして、出張サービスのプラットフォーム「くらしのマーケット」の立ち上げに携わる。
その後新卒入社し、主にバックエンドとインフラを担当。現在は技術部門を管掌。

LTテーマ：出張サービス業界における機械学習の活用
LT概要：
クロステック(X-Tech)という言葉が示すように、既存産業にテクノロジーを注入することにより、イノベーションを起こし、生活を豊かにすることができます。
ハウスクリーニングやリフォームなど出張サービス業界のプラットフォーマーとして、機械学習というテクノロジーをどう活用しているかについて、お話します。


【今井 太宗氏＊Repro株式会社 Repro AI Labs 所長】
2016年1月にRepro株式会社に入社。WEBエンジニアとして、マーケティング機能や分散処理基盤を利用した機能の開発に従事。 2018年7月、よりデータ分析・機械学習にフォーカスしたチーム「Repro AI Labs」を立ち上げ、さらなる付加価値を提供するべく奮闘中。

LTテーマ：Python が支える機械学習フロー
LT概要：
機械学習の機能開発で用いられる Python はもちろん pandas や scikit-learn だけではありません。
Repro で利用している機械学習のフローについての紹介をもとに、Python が活躍する広さについて話せたらと思います。


【脇坂 琢也氏＊LeapMind株式会社 シニアエンジニア】
横浜国立大学工学部電子情報科在学中からソフトウエア技術者として活動。退職後フリーランスとして、インターネットのサービスやアプリの設計・構築・開発をメインに、インタラクティブなプロジェクションなどの技術支援を行うフルスタックエンジニアとして活動。
15年にリープマインドに入社し、シニアエンジニアとしてBlueoilの開発を担当。

LTテーマ：量子化によるdeeplearningモデルの圧縮
LT概要：
弊社で開発しているローエンドエッジデバイス向けのdeeplearning学習・推論オープンソース blueoil で用いている、量子化によるdeeplearningモデルの圧縮について説明します。


【渡邊 直樹氏＊株式会社RUTILEA R&D責任者】
大学で物理研究したあと、外資ITでの開発を経て、友人と立ち上げた会社でディープラーニング関係の開発をしております。

LTテーマ：最新のディープラーニング動向
LT概要：
日進月歩の激しいディープラーニングの基本の説明と最新の動向を話します。
