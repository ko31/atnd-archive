「バックアップの設計に悩んでいる」「現在のバックアップ方法が適切であるかどうか
わからない」「これまでに一度もリカバリ作業を実施したことがない」
という方に朗報です！

※セミナーの詳細はこちらから※
※http://www.cosol.jp/seminar/detail/000087.shtml※

Oracle DBA＆Developer Days2010、2011にて弊社社員が講演をさせていただき、
大好評を博した「Start! Oracle Database～Oracle Database管理者になるため
のStep by Step～」セミナーをベースにした、定期セミナー第2弾を開催いたします。

弊社では、約7割のエンジニアが、「Oracle DB製品アナリスト」（Oracle製品
テクニカルサポートセンター勤務）として、DB運用管理者の皆様をはじめとする
エンジニアの方々から寄せられる多種多様なお問い合わせ（システム障害の
復旧方法、新機能に関するご相談、パフォーマンス改善のアドバイスなど）に
対応しています。
本セミナーでは、そうしたノウハウを活かし、“明日現場で使える実践的な
エッセンス”を多数お話しいたします。

さらに、講義終了後には、「よろず相談会」もご用意いたします。
皆様が日々業務の中で感じていらっしゃる「どうしてもここが上手くいかない」
「もっとスムーズに運用管理業務を進めたい」といったお悩みを、ぜひお気軽に
ご相談ください！
弊社のエンジニアが、可能な範囲でお答えいたします。

沢山の方のご参加を、心よりお待ち申し上げております。
