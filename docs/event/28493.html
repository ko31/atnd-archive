熱硬化性樹脂の基礎と挙動解析【大阪開催】

イベント ユーザー
event_id 28493
title 熱硬化性樹脂の基礎と挙動解析【大阪開催】
catch ～フェノール樹脂とエポキシ樹脂を中心に～
description

［講座のポイント］

　講演者が３０年に亘って精密合成と高機能用途開発に携わったフェノール樹脂と、近縁のエポキシ樹脂を中心に基礎から合成法や硬化方法と挙動解析などを解説し、用途展開についても実例を踏まえて述べる。さらに、アミノ樹脂、ウレタン樹脂、シリコーン樹脂などその他の熱硬化水樹脂についてもその概論を解説する。

［受講対象者・レベル］

　熱硬化性樹脂の合成・応用に関わる研究開発者とその部材使用者

［プログラム］

１．熱硬化性樹脂とは？

２．フェノール樹脂

　　2-1　レゾール樹脂の合成とその性質、硬化方法と挙動解析改良
　　2-2　ノボラック樹脂の合成とその性質、硬化方法と挙動解析改良
　　2-3　汎用フェノール樹脂の用途（合板、成形材料、鋳物他）
　　2-4　高機能フェノール樹脂の合成とその用途（電気電子材料）

３．エポキシ樹脂

　　3-1　エポキシ樹脂の合成とその性質、硬化方法と挙動解析改良
　　3-2　汎用BIP-A型エポキシ樹脂とその用途（塗料、接着剤、土木・建築、複合材料）
　　3-3　高機能エポキシ樹脂の合成とその用途（電子材料）
　　3-4　エポキシ樹脂の用途

４．その他の熱硬化性樹脂

　　4-1　アミノ樹脂（尿素樹脂とメラミン樹脂）
　　4-2　ウレタン樹脂
　　4-3　不飽和結合型樹脂（アクリル樹脂、メタクリル樹脂、ジアリルフタレート樹脂他）
　　4-4　その他の高機能樹脂（シリコーン樹脂、シアノエーテル樹脂、ポリイミド樹脂他）

５．まとめ
