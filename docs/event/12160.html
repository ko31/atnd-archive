description

ママとして頑張っている自分がいる。
だけど子どもと思うようにうまくコミュニケーションが取れないことがある。
子どもを愛しているけれどうまく表現できない自分がいる。
自分はママとしてちゃんと子どもに想いを届けられているだろうか。。。
そんなママたちにこのセミナーを提案します。
ママであるあなたが本当に望む方向へと変化を遂げていくストーリーの始まりです。

＜こんな方におすすめします＞
◇子どもとのコミュニケーションを上手くとりたい
◇子どもが勉強をしないことが気になる
◇子どもが何を考えているのかわからない
◇なんだかイライラ、もやもやするときがある
◇自分の存在価値を感じたい
◇自分に自信がない
◇やりたいことがあっても一歩踏み出せない

～ママである私とは・・・潜在意識とのコミュニケーション～

２月１６日：私の成分表
２月２３日：私が望むママとは？

３月１６日：私はどのように現実を創り上げているの？　五感で体験　私の知覚
３月２３日：私の言葉をひも解く方法
