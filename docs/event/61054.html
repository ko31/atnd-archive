通勤や通学前の時間を有意義な時間にしませんか？
注目の最新DJ機器に触れて学べる朝DJ講座を開催します！

■目覚ましDJとは？

世界初の完全自立型ポータブルDJシステム「 GODJ 」（ゴーディージェイ）を教材として、早朝に開催する日本初のDJ講座です。

  • DJを始めたいけどハードルが高いんです・・
  • 気軽にDJの練習ができる環境がほしい・・
  • 楽器系ガジェットの最新機を使ってみたい。
  • 朝の時間を有効に活用したい。

そんな方や、GODJをすでに持っている人やGODJを知らなかった人でも、
専門の講師とGODJ開発スタッフが機能や使い方などをじっくりと解説いたします！

■GODJ（MONSTER GODJ）とは？

世界初の完全自立型の持ち運べるDJシステムです。
2つの高機能ターンテーブル、精細な物理スイッチを搭載したミキサー、多彩なエフェクター、音楽ファイルの解析、それら全てを、幅250mm x 奥行 60mm x 高さ16mm、重さ286gのコンパクトボディとして一体化。持ち運びが可能なため、これまでDJ環境の構築が難しいとされていたあらゆるシーンで、本格的なDJプレイを楽しむことができます。
2013年、モンスターケーブルでお馴染みのアメリカMonster社とのライセンス受託により現在のGODJとしてワールドワイドで販売されており、国内外で活躍中のトップDJからガジェットユーザーまで近年注目を集めているのがMONSTER GODJです。
