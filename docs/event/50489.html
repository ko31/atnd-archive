行なってるカフェで行なう英会話教室です。
初期費用や教材費は一切かからず、レッスン代のみ（別途各自お茶代）

講師のAyumiさんは本業は留学コンサルトをされていて、
高校を卒業とともに単身でアメリカの大学に通い、
飛び級をし、わずか2年4ヶ月で卒業をしてきました。
留学の相談はもちろんですが、セミナーの司会業、英語の講演、パブリックスピーキングを交えた英語の特別授業、企業の英語研修などを各方面から頼まれている若き講師です。

大学ではコミュニケーション学を専攻し、副専攻として演劇を行なってきました。
日本人が苦手の発音や表現力、コミュニケーションに特化しており、日本人だからこそ、わかる日本人が苦手なこと、ネイティブじゃわからないところを教えてくれます。

2020年にはオリンピックが控え、外国人観光客は1000万人を越え、政府は2000万人を目指しております。また、外国人労働者も現在の倍にすると政府も発表しております。
さらには今年の6月頃にはカジノ法案も通ると言われ、より日本に外国人が増える流れが来ております。

ただでさえ、英語が苦手な日本人・・・
道を少し聞かれたら答えられるくらいにはなりたい
日常会話くらいはできていたいな
今更英語を習うのは恥ずかしい
話せるけど、表現の仕方を鍛えたい

などなど

英会話に興味ある方なら

誰でもウェルカム☆☆

小中学校の英語の教育指導要領も平成２８年度には改訂され若い世代は英語がより身近になっていきます。
英語に慣れ親しんでなかった世代、英語はやってたけど身になっていない世代は肩身が狭くなってしまいます。

そんなことにならないよう、
気軽にみんなででき、日々のトレーニング方法も教えます！！！

日本で人前で普通に英語が出来るようになりましょう！！
