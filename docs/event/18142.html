description

【講座のポイント】
　リチウムイオン電池において電解液は不可欠の構成要素であり、出力や低温特性などの特性に大きく影響する。また電池の安全性向上に対して、電解液の材料選択を含めて本質的な改善が求められている。本講ではリチウムイオンの基礎および安全性に関する問題を概説するとともに、難燃性の電解液材料として期待を集めているイオン液体の構造制御について、演者の検討例を紹介する。

【プログラム】
１．リチウムイオン電池電解液の基礎
　　1-1 要求特性と基本構成
　　1-2 負極表面保護膜の作用機構
　　1-3 表面保護膜および電極特性の制御
２．電池の安全性と電解液
　　2-1 電池の発熱・発火と電解液の寄与
　　2-2 難燃性電解液の検討例
３．イオン液体のリチウムイオン電池電解液への適用
　　2-1 イオン液体について
　　2-2 リチウム二次電池用イオン液体
　　2-3 構造制御による特性制御例
　　2-4 イオン液体電解液を用いた場合のレート特性
４．イオン液体ゲル電解質
　　4-1 リチウムイオン伝導性に及ぼす組成の影響
　　4-2 リチウムイオン伝導性改善に向けて
