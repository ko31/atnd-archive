講演概要

■ヒットアプリデベロッパーに学ぶ、最新マネタイズ手法

カジュアルゲームアプリでヒットアプリをリリースしている2社に今回ご登壇頂きます。

株式会社GOODROID　代表取締役社長 / プロデューサー　松田　和彬様

大ヒットした「もやしびと」「私のヒモ男～イケメン拾いました～」を始め数々のヒットアプリを創出。
今アプリデベロッパーが取るべき最新のマネタイズ手法を実績と事例を用いてご紹介頂きます。
動画リワードの活用事例や収益実績、その他今取り入れているマネタイズ手法に関してお話頂きます。
※当日公開されるデータは、外部公開できないものを含みます。

グッディア株式会社　マネージャー　吉田和史様

カジュアルゲーム界の大御所、グッディア株式会社様に登壇頂きます。
今までリリースされたアプリ本数の数は500本以上。総DL数は5,000万を超え、
更にAppStore25位以内に君臨した回数、なんと200回以上！
最近リリースされたアプリで活用した最新マネタイズ方法や新しい広告メニューの活用方法を
グッディア吉田様にお話し頂きます。
※当日公開されるデータは、外部公開できないものを含みます。

■成功事例はカジュアルゲームだけじゃない！？
気になる動画リワードの「ここだけの話」（パネルディスカッション）

日本での動画リワード成功事例はまだまだ少ない状況。
既にメジャーとなっている海外アプリの動画リワード利用に関して、
パネルディスカッション形式で色々聞いちゃいます。

今回は、Mr.AppLovin坂本氏と、UnityAdsの森嶋氏をお迎えしてお送りします。
海外ではこうやって利用して成功してるよ？や
もっとこうしたら収益上がるのに・・・といった情報や
カジュアルゲーム事例だけでなく、課金メインのゲームアプリでの活用方法もお話頂く予定です。

■ついにベールを脱ぐ、「AdStir動画リワードSSP」について

昨年7月にリリースしてからしばらくクローズド展開していた
「AdStir動画リワードSSP」がとうとう情報を開示します。
他社とどう違うのか、どんな実績があるのか、公開します。
