description

IDCフロンティアのクラウドサービス「NOAH」をご利用のお客様や利用を検討されているお客様を
対象に、クラウド技術の最新動向やNOAHの開発状況などをお伝えし、お客様同士の交流を深める
ユーザー会を開催いたします。

今回、通信・メディア・ハイテク中心に有名なコンサルタントの八子様をお招きしてクラウド全般に
ついて、また、ソーシャルゲームの運用や課金・決済で多数の実績がある株式会社ISAO様を
お招きしてソーシャルアプリの運用ノウハウなどについてお話していただきます。

また、日本で初めてCloudStackの採用を発表したNOAHについて、今後のサービス開発ロード
マップも含めてIDCフロンティアの大屋からお話させていただきます。

　　7月28日（木）　開場18:00

　　懇親会　　　　20:00-21:00
