catch Unity4.6の新UIを解説！ 触って、ためして、楽しいモックアップアプリをお手軽作成！
description

広告、ゲーム・電子カタログ制作に使える、Unityの教材データを提供致します。初心者でも短時間でモックアップアプリ制作を楽しめる、実践的な体験実習です。講義では、大きく進化した期待の新UI Unity4.6 「uGUI」の制作手法を学びます。プログラミングが未経験のデザイナー、プランナーでも、開発の仕組みを理解できます。

実践的な体験実習ですので、Unity4.6がインストールされたノートPCを持参ください。

◆脇本亮 (わきもとりょう)
ECサイト、プロモーションサイト等のＷｅｂシステム開発、Androidネイティブゲームアプリ制作、専門学校で初心者むけJavaScriptゲーム講座やLAMP講座に携わる。世界中のWebAPIと連動し、生活を楽しく便利にするようなアプリ開発を勉強中です。

【ひとこと】     
みなさんのアイディアを形にするため、Unityを活用して完成度の高いアプリを短時間で作成する体験をしていただきたいとの思いで講座を企画しました。広告アプリやゲームアプリのひな型を編集しながら、実用的な操作・知識を中心に身に着けられます。プログラマーはもとより、Webデザイナーや2D・3DCGデザイナーの方々でUnityに挑戦したいという方にご参加いただきたきたいと思っています。
いろいろな職種の方々が集まって魅力的なゲームや広告アプリを企画するコミュニティづくりに貢献できればうれしいです。
