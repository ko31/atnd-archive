TEAMJET presents RUN Fwd: closing party 開催決定!!!

NIKE RUN Fwd:終了から早いものでもうすぐ３ヶ月。
実際参加してみて、ソーシャルのつながりが
ランニングというフィジカルな行為を通じて顕在化していく。
そしてその過程で人のつながりが深くなるという
とても素晴らしいイベントだと思いました。

そこで、参加ランナー同士の交流が深まるPARTYができないかなと
水面下で企画やNIKEさんへの提案を行っていた結果、
NIKEさんのご協力を頂きこの度晴れて
TEAMJET presents RUN Fwd: closing party と言う形で
6/15（水）の20:00〜23:00 東京都江東区にて
アフターパーティーを開催する事となりました！
（本当は全国からみんなが集まれる様な形が良かったのですが…。）

ちなみにこのパーティー、二つのサプライズがあります。
一つはRUN Fwd:のテーマを演奏していたtoeのライブがある事！！
もう一つはなんと入場料無料という事！！
（別途ドリンク代500円だけ頂きます。）

そのかわり、チャリティイベントとしてNIKEさんの主催する
ALL FOR JAPAN プロジェクトへの募金を募ります。
是非ご協力頂ければと思います。

ちなみに、toeのLIVEが無料ということで、一般に告知してしまうと
すぐに埋まってしまう危険性があり、今回のパーティーはRUN Fwdの参加者のみ
参加が可能というクローズドな形式にさせていただきます。
ですのでこの告知はRUN Fwd:を走っていない方にはしないでください。
走っていないお友達同伴したくても今回はごめんなさい！
その代わり、RUN Fwd:に参加されたお友達にはどんどん広めてください！

また、定員１５０名の先着順での予約制となっています。
参加ご希望の方は以下のフォームからお願いします。
https://ssl.form-mailer.jp/fms/8ed605b3152346
（web、スマートフォン、携帯のいずれからもエントリーできます。）
予約が完了すると場所等の詳細が送られてきます。

申込期限は6/13（月）24:00までですが、
現状で既に５０名程の予約が入っています。
今後告知を拡大していくので、確実に参加されたい方はお早めにご予約ください。

会場でいろんな方がつながったらホント楽しいと思います。
それでは！！

TEAMJET 桑原慶@kkwbr
