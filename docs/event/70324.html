description

アイデア（事業のネタ）は、即実行、行動無ければいつまでも“絵に描いた餅”であり、アイデアを他の誰かに先に実現されてしまいます。

そのスキルが現実化のスピード化につながります。実際に開発してみて

「なんか違う」「進捗状況が悪い」「時間をかけて考えたことなのに、良くなかった」

新しいサービスを考え、開発した時によく起こる事象でしょう。でもそれは『必ず通る道』であることを先に知っておくことが大事です。

どのような考えをもって取り組むか、そして最新事例も踏まえて新しいサービスを作る上での１つの良い例をご紹介します。

今回の説明会では、ビジネス、クリエイティブ、ＩＣＴの融合で実践的手法を身につける本学のカリキュラムにおいて、Webサービス開発の指導を行なう山崎大助教員より、 「アイデア」を「実装」するための基盤となる特別講義を実施します。

世界で10人しかいないMicrosoftMVP(Bing Maps Development )の１人

である山崎講師は 『jQueryレッスンブック』 『レスポンシブWebデザイン「超」実践デザイン集中講義』 などの著書があり、多くのスタートアップ系エンジニアの養成実績を持ち、過去には人気フリーソフトを作成した実績をもっています。


各業界における

新規事業開発に携わり、テクノロジーとクリエイティブの知見を必要としている方

や、

Webデザイン・プログラミングのスキルを持ち、ビジネスプラン策定に関心のある方

にお勧めです。


本大学院での研究内容、またご入学にご関心をお持ちの方は、ぜひこの機会をご活用ください。

山崎大助 氏（やまざき・だいすけ）

デジタルハリウッド大学大学院 非常勤講師

デジタルハリウッド大学大学院、大妻女子大学の非常勤講師、G’s Academy TOKYOの主席講師として人材の育成に務める。世界で10人しかいないMicrosoft MVP (Bing Maps Development )の一人。 雑誌、Webメディアでの連載執筆のほか、企業研修、エンジニア向けイベントにもたびたび登壇している。 また、過去には日経PC21にて「名作フリーソフト」として選出されたソフトを開発提供してた実績もあり。著書に、 「レスポンシブWebデザイン「超」実践デザイン集中講義」 （ソフトバンククリエイティブ） 「jQuery レッスンブック jQuery2.X/1.X対応」 など。
