RaspberryPi使っていろいろやってみよう

みなさま、RaspberryPiをお持ちでしょうか？
お持ちの方、どんなふうにお使いでしょうか？
お持ちでない方、どんなことができるか知りたくないですか？

えれくら！TypeRでは、ふつうのえれくら！とは別に、RaspberryPiを主題とした勉強会といたします。
RaspberryPiをお持ちの方には「今こんなことやってて楽しい！」という話をしていただけたり、お持ちでない方には「こんなことができるんだなー」ということを知っていただけたり、そんなイベントにしたいなーと考えています。

第2回目ですがまだまだ手さぐりしてます。とりあえず、以下のような方々を対象に、と考えています：

  1. RasPi買ったけど買っただけ or 欲しいけどまだ買ってない、な方
  2. RasPi買って使ってみたけど Linux をちょびっと起動してみたくらい、な方
  3. RasPi使ってるけどGPIO使ってなくて電子工作してみたい、な方
  4. RasPiガチで使い込んでるので何でも聞いてちょうだい、な方

ご用意いただくもの

  • RaspberryPi
  • ノートPC
  • SDカード

RaspberryPiにはモデルが2種類ありますが、どちらでもかまいません。お持ちのものを持参くださいませ。

RaspberryPi・ノートPCをお持ちでない方でも、会場からのご配慮により、少数ですが貸し出し用のRaspberryPiを貸していただけるそうですので、ご参加いただけます。

SDカードは、OSをインストールするのに使います。4GB以上の容量があるClass 4以上のものをご用意ください。変換アダプタがあれば、MicroSDでも大丈夫です。

内容

第2回目、なのに、まだジャブを打つ感じで :-)

  1. RaspberryPiについて
    RaspberryPiとは何か、とか、各モデルの違いとかあたりを軽く紹介します。
    あと、RaspberryPiで動作するOSそれぞれについて簡単な紹介と、インストール方法の解説をしたりします。
  2. GPIOを利用するプログラミングについて(前回の続き)
    前回、RaspberryPiのGPIOピンを使ったプログラミング例として挙げたインターネットラジオの完成形を示し、簡単に解説とかします。
  3. シリアル通信の解説
    GPIOポートを使った、外部の機器との通信を行う方法を解説・実験します。
  4. 発表・LTなど(発表希望者ありの場合)
    • RaspberryPi用3G通信モジュール「3GPI」の紹介(発表者：K.M.A.G.)
      RaspberryPi TypeB/B+に対応する拡張モジュール「3GPI」を入手しましたのでサラッと紹介します。何かプログラム作って持って行けそうならデモもするかもしれません。

発表・LTコーナーについては、他にも何か持ちネタを発表したいという方やちょびっとだけど喋りたいという方がいましたら、枠を作りますので、ぜひお申し出くださいませ。
