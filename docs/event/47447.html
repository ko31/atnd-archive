description

ああ、毛ガニや石狩鍋が呼んでる声が、聞こえる～！心に直接語りかけてくる～！
というわけで、2月半ば、雪まつりもひと段落した札幌で、さくらの夕べを開催します。

今回は、札幌から世界へ羽ばたくサービスを展開するクリプトン・フューチャー・メディア株式会社様にご登壇いただき、インフラの乗り換えによりいかにしてコスト削減を実現できたのか、実際の構築運用例を交えつつ語っていただきます。

弊社からは、今まであまり語られることのなかったデータセンターの運用現場について、石狩DC運用スタッフからご紹介させていただきます。後半は「さくらのクラウド開発者に一言物申す！」と題した、クリプトン様と弊社クラウド開発責任者との対談を実施、ユーザ様と弊社とで、ゲレンデが溶けるほどアツいディスカッションを繰り広げる予定です。

参加者には、桜葉愛からのちょっと遅めのバレンタイン的なプレゼントがある･･･かも？

懇親会では、スクリーンを開放していますので、ネタのある方は自由にお使いください。

ご友人をお誘いの上、ぜひ遊びにいらしてください。
