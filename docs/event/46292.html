15:15-15:30Session01「About modern.IE」
Internet Explorer はその性質上、異なるバージョンを単一の Windows 上にインストールすることはできません。
modern.IE では、この問題に対処するため Windows、Mac で動作する検証用の VM イメージの提供、Web サイトの解析と各メジャー Web ブラウザーでの画面ショット、リモートデスクトップサービスなど、Web ページを検証するうえでの様々な機能を提供します。
このセッションでは、modern.IE のこれらの機能について紹介します。		物江 修さん
15:30-15:45Session02「誰かと一緒に仕事をするときに私が気をつけていること」
今年も２週間を切りますね。仕事に対しての向き合い方のおさらいをしようとおもいましてこのプログラムにしました。今回はデザインの部分と基本的な業務での話をしようと思います。		仲野 宏美さん
15:45-16:00休憩（15分間）　
16:00-16:15Session03「新しいCreateJSのマウスイベント」
CreateJSで最近大きく変わったのは、EaselJSのイベントモデルです。とくに、マウスイベントについては、今週リリースされたEaselJS 0.7.1でもまた少し動きが変わっています。そこで、マウスイベントの変遷をご紹介します。		野中 文雄さん
16:15-16:30Session04「ブッくんの描き方」
白山 からんさん
16:30-16:45Session05「クリエイターにすすめるゴルフ入門」
おっさんのスポーツだったり、ゴルコンだったり、何となくメディア的に香ばしい印象がしちゃうゴルフなのですが、クリエイターの趣味にぴったりと思うのです。自転車にはまってる人は次はゴルフはどうでしょう？		平岡 拓夫さん
16:45-17:00休憩（15分間）　
17:00-17:15Session06 「Webサービスの企画書 – 準備を楽しむしおりアプリ」
「プライベートデスマ」を銘打って本年5月に行った開発合宿の成果物、「おれたちのしおり」
http://rechiba3.net/?p=675 をスマートフォンアプリ化しちゃいます！開発にあたり、あらためてアイデアを固め直しました。”誰かを幸せにするものづくり”の過程をまとめた企画書をお披露目します。		千葉 礼美さん
17:15-17:30Session07 「CSS Selectorを書こう」
CSSプリプロセッサーとかBEMとかコンポーネントとか話題になっていますが、そういうツールや概念の前にCSS Selectorの書き方をおさらいしておきましょう。		小林 和弘さん
