title 日本酒呑み比べ
catch しぼりたて生原酒をメインに
description

次のお酒を用意しています。

  • 春鹿 立春朝搾り 純米吟醸生原酒
  • 富久錦 立春朝搾り 純米吟醸生原酒
  • 白鹿 黒松白鹿 大吟醸 垂壺（たれつぼ）無濾過生原酒
  • 小鼓 今朝しぼり
  • 小鼓 濃旨純米大吟醸
  • 一（はじめ）純米吟醸・しぼりたて生原酒
  • 島美人 美人蔵部 本醸造生酒原酒
  • 剣菱 瑞穂
  • 〆張鶴 しぼりたて
  • 八海山 しぼりたて原酒 越後で候
  • 雪小町 純米大吟醸原酒生
  • 末広酒造 壷中春
  • 源平酒造 萬寿恵 純米原酒

どれも四合瓶（720 ml）です。ガッツリ呑むと一瞬でなくなります。呑み比べが趣旨なので、みんなが不自由しないように配慮をお願いします。人数制限は例えば一人当たり 50ml x2 ぐらい呑めるといいかな、というところから来ています。この辺は適当なので、人数調整は応相談で。あまり無茶にならなければ ok です。

日程は 2/6。失敗したら 2/8 ということで。

本数も多くなってきましたし、次回も開こうと思うので、今回はいくつか開けるものを選びましょうか。当日集まったときに選びましょう。
