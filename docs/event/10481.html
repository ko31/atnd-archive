■ 集合場所

  • 16:50に開催地の店頭（がんばって守ってください）
    • 17:00スタートなので。

■ 一次会の予算とスケジュール

  • 一次会の予算：一人5000円。
  • 一次会の時間：17:00〜19:30の2時間半の予定。

■ 内容・開催の経緯

  • 年末なので TokyoNLP と MG勉強会 と IIR輪講 の合同忘年会を開催することにしました。
  • 2010年の自然言語処理・機械学習・検索・データ圧縮について語ったり語らなかったりしましょう。
  • 開催日は @overlast と @nokuno さんと @yto さんの3人で決めました。
    • 最初は12/18に開催しようとしましたが楽天研究開発シンポジウムとかぶったので後ろにズラしました。

■ 参加条件（以下のうち、どれか１つ以上を満たしてください）

  • TokyoNLP と MG勉強会 と IIR輪講 のうち、どれかに参加したことがある
  • TokyoNLP と MG勉強会 と IIR輪講 のどれかの参加者が、この忘年会に誘った
  • @overlast か @nokuno さんか @yto さんと知り合いで、日常的に自然言語処理・機械学習・検索・データ圧縮などの技術を扱っている開発者 or 研究者
  • 自然言語処理・機械学習・検索・データ圧縮などのテーマで研究している現役学生
  • 自然言語処理・機械学習・検索・データ圧縮関連のアプリケーションを開発し、ソースコードやWebサービスを公開している人
  • 生駒日記を1年以上前から読んでいる人
