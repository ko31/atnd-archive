お花見会 2012 Tokyo/Spring

イベント ユーザー
event_id 26896
title お花見会 2012 Tokyo/Spring
catch 〜松尾芭蕉が愛した江戸川橋を訪ねて〜
description

桜の季節です。
みんなで楽しく、まったりと散策＆お花見しましょうヽ(*´∀｀)ノ

写真は3月22日に江戸川公園で撮影した桜です。
4月1日は花見だぜ〜♪　(∩´∀｀)∩

【集合時間】

・4月1日(日)11:00

　※遅れる人は、 @mana_cat に連絡してください

【集合場所】

　　有楽町線江戸川橋駅 「1a出口」

【スケジュール】

11:00〜12:00

　　　江戸川公園〜椿山荘を散策＆カメラ撮影
　　　カメラを持参して桜を撮ってみよう！(桜の名所です)

12:00〜15:00

　　　江戸川公園でお花見
　　　みんなでワイワイ楽しくお花見〜♪

【参加対象】

　以下いずれかに該当する人
　・桜を綺麗に撮影したいカメラ女子・カメラ男子
　・参加者の知り合い

【会費】2,000円

　※仕出しや紙皿などを注文します

　　　(あくまでも目安、安く済んだらお釣りを出します)

【持ってくるもの】

・お気に入りのカメラ
　コンデジ、一眼カメラ問いません
　　(カメラを持っていなくても、散策するだけでも楽しいのでカメラ必須ではありません)
　
　　桜の他に、歴史的建造物と川鵜・白鷺・目白・目黒などの野鳥がいます♪
　　レンズ選びは上記を考慮した選出をおすすめします。

・飲み物持参 (酒など自分が飲む分)
　重たい場合は、江戸川橋駅1b出口直結「コモディイイダ」や、
　公園の近くにローソンで買ってくるなど方法があります。

　※お茶は主催者側で用意します
　※最初、散策に出てからのお花見になるので、各自用意していただいた飲み物は場所取りした場所に置いて頂いてもOKです。
　※クーラーボックスは用意していないので、「冷えたビールが飲みたい！」という人はお花見の直前に買いに行ったほうがいいかも。

【スタッフ募集！】

・当日9時から、花見の場所取りを @htairaが頑張ります。
　一緒に場所取りを手伝ってくれる人が居ましたら、是非お声掛け下さい(^^ゞ

＜雨天時＞

　　・鳩山会館→おいしいカレー屋「カルカ」でカレーパーティ
　　　(鳩山会館は"はとバスツアー"にもなっている洋館の名所です)

　　※3月31日10時の時点で、翌日4/1の降水確率が50%を超えた時は、花見をとりやめ、カレーパーティにします。
　　※当日晴れたら、桜の散策は実施します。
　　※雨天時の場合でも、4月1日(日)11時に有楽町線江戸川橋駅 「1a出口」に集合です。
　　※同じく会費は2,000円になります。

【注意点】
　キャンセルする場合は3月31日10時までにご連絡ください。
　それ以降になる場合はキャンセル料を請求するかもしれません
