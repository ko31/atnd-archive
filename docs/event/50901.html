catch 市民テクノロジーによる行政改革
description

プログラミングとオープンデータによる千葉県野田市の市政改革をテーマにしたハッカソンを行ないます。

今月5月25日に野田市議選が行なわれますが、現在、野田市の経済と行政は近隣の自治体から大幅に遅れをとっております(議会改革ランキング: 流山1位、松戸142位、野田684位)。

そうした現状を踏まえ、今回は、他の先進的な自治体で用いられているITソリューションや今現在公開されている市の情報を元に、アイディア出しと成果物（webアプリ、公共データのまとめとgooglemapへの出力…etc）の作成を行ないます。

プログラミングの経験に関わらず、政治行政へのIT活用にご興味のある方は是非ご参加下さい！
