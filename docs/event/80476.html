いまさら聞けない「Git」講座

こんな方にオススメ！
WEB業界で複数人で作業をする場合、ファイルの管理でお困りの方、また人手を増やすのではなく
業務の効率化を図りたいと思っている方、もはや使うことが常識となった「Git」で、バージョン管理とは何か？、どうすればGitで効率化できるのを学んでみませんか？

※会場にてご用意するwindowsを使用します。

セミナー内容

業界の定番となったGitですが、「実はまだ……」「どんなものか知っておきたいけど、今さら部下に聞くのも……」という方に、基礎の基礎から教えるGitのハンズオンセミナーです。

Git ってそもそも何が良いの?
Git とは何か、どう使うのか、どんなメリットがあるのか、 今さら部下に聞けないしなあ、という方も安心！基礎から Git を説明します。
Git ってどう使う?
どんな環境が必要で、どう操作するのか
実機を用いて実際に Git 環境を構築し、自社での導入ヒントとしてください。
