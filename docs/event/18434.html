【開催概要】

前回好評につきまして、今をときめく沖縄のエンジニアを熱く滾らせる祭典の第二弾 eXtreme HAGO LT 大会 2 を開催します。

日頃のエンジニアライフをお話してみませんか？

  • ほんの些細なスクリプトから感動巨編まで
  • うっかり狂気の領域へ踏み込んでしまったアプリ
  • 組み込みとか web サービスとか
  • スマートフォンなプログラムやデザインについて
  • 成功談、失敗談
  • 迸る妄想、そして暴走などなど

貴方の、エンジニアとして滾るお話をお待ちしております。

HAGO とは

英語には素敵な言葉がございます。 hacker, geek, wizard, guru など。
日本語だと、さしずめオタクという意味になりますか。
ある種の理解がある人でなければ、いずれもあまりいい言葉ではないかもしれませんね(笑)
これらの言葉に近い言葉を沖縄の方言から探してみました。

それが はごー (本来は「汚い」という意味)
・　　・　　・　　・　　・

最近、高揚して何かを作っていますか?東日本で起きた震災、原発事故の収束の見通しも不透明、仕事、学業、今後もますます忙しく…

しかし、それでも研ぎすませてきた技が私たちにはあります。

あなたの高揚を、技を、経験を、考えを、教科書に書いてあることだけじゃなく、あなたの培ったことを、私たちに聞かせてください。そうすることでみえてくる、大切な物がきっとここにあります。

それがはごんちゅぬ宝

そして、いつの日かこの言葉も世界へ出れるといいです。

hacker, geek, wizard, guru, OTAKU, HAGO …
