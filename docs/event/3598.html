description

国民からの意見募集サイト「経済産業省アイディアボックス」 http://open-meti.go.jp/ が3月15日（月）で提案・投票の受付を終了します。この経済産業省の取り組みを題材（ネタ）として、オープンガバメントのもつ現状や課題、またその望ましいあり方等について、興味関心のある方が情報・意見を交換する場になればと思い本イベントを企画しました。（http://open-meti.go.jp/ja/idea/00819/）


初めての試みですので、まずは少人数でもご関心のある方に集まって頂ければと考えております。当日ショートプレゼンをやって頂ける方がいらっしゃいましたら、ぜひ参加登録時にコメントをお願いできれば幸いです。


会場は高円寺プラットフォーム・ワークショップにて、開催時間は3月20日（土）の16時より18時半に決定いたしました。それ以外の時間帯をご希望されていた方には大変申し訳ありません。なお、事前準備のため15時30分には開場しておりますので、お時間に余裕のある方は少々早めにご来場頂いても結構です。

会場へのアクセスについては以下のURLを参照下さい：
http://koenji.platform.ne.jp/guide-map
