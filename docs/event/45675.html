description

英語スキルがないことに焦りを感じている
◇同僚や友人が相次いで英語学習を始め、出遅れ感がある
◆転職を考えているが、英語力が必要な求人はどのくらいあるか知りたい
◇TOEICスコアはそこそこだが、ビジネスシーンで英語を使うイメージが沸かない
◆海外転勤や国際業務を視野に入れているなど、キャリアの可能性を広げたい
◇そうはいっても語学学校に通う時間がとれない…

こんな悩みを抱えている方も多いのではないでしょうか。

ひと昔前までは、一部の外資系企業や国内企業の海外拠点で活躍する人材を中心に、一部のビジネスパーソンのみが
必要とされるにとどまっていた英語力。が、今ではグローバル展開をしている企業も幅広い業界で増え、社員採用時の選考基準や評価に導入する企業も珍しくなくなりました。

ビジネスパーソンの転職やキャリア形成・スキルアップを支援する【typeの人材紹介】では、特に重要性の増してきた「使える英語力」を効率的に学習するヒントを提供差し上げる場をご用意しました。

「業種別/職種別に必要な英語だけを学べる！」をキャッチコピーに新感覚の英語学習法を展開、忙しいビジネスパーソンでもスキマ時間で実践的な英語力を身につけられる点で支持を集める新進気鋭のオンライン英会話「Best Teacher」の吉野白竜氏を講師に迎え、今後のグローバル社会を勝ち抜く「転職／キャリア形成」成功のためのメソッドをお伝え致します。
