名古屋でSOY CMSを使っている方、同じCMSを使っている方と話をする機会ってありますか？

-

SOY CMS ユーザ会 in 名古屋では特定の日時に、"場所のみ"用意します。

時間内であれば、各自で話をしても、持ち込んだ作業をして頂いても
発表、告知をして頂いても、何をしてもokです。

主催者、スピーカー等は特にいませんので、内容は
皆様でそれぞれ考えていただく必要があります。


【参加費用】
￥3,000程度(会場費用)の人数割り、参加費用は当日お支払いください


【参加方法】
右上の「このイベントに参加登録する」から参加登録してください。
Googleグループに参加していない方は、以下からGoogleグループにも参加してください。
はじめて参加された方は、簡単な自己紹介をお願いします。
http://groups.google.co.jp/group/soycms-users-nagoya

【最低参加人数】
5人

【場所】
CHUTOホール栄店 第5会議室（301）


【日時】
7月3日(土) 13:00-17:00

【プログラム（予定/時間は目安です）】
初参加の方の自己紹介、やること発表
自由制作タイム
休憩
LT（発表される方がいれば）
自由制作タイム
本日の成果発表、終了

【LTとは？】
ライトニングトークの略で、5分ほどの短いプレゼンテーションのことです。

【さいごに】
特に参加条件等はありませんが、SOY CMSについて関心がある方で、
事前にインストールや簡単に使用しておいた方が、より一層楽しめます。

【会場について】
予算の都合上、12人まで収容可能な小会議室になりました。
費用としては13:00-17:00までで、￥8,000になりました。
この費用を参加人数で割った金額が、参加費用となります。


【会場の設備について】
・プロジェクター、スクリーン　無し
・インターネット　無線LAN付パソコンなら利用可能
