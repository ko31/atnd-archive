どんなイベントなの？

“xtnight” == "@xtetsuji’s night" と銘打って、 みんなと歓談したいので飲み会をする のが目的です。

…と書くと飲み会をしたいだけと思われてしまうのですが、実際は 2013年12月に胃潰瘍で入院したときに心配してくれた多くの皆さんに感謝を伝えたい というのが目的です。 特にITエンジニアの皆さんには多大な心配をしていただきました 。このイベントの場を借りて感謝を申し上げたいです。

というわけで主催者の私が多少出費するつもりでいます。

体調回復した！

その後も風邪を引いたり中耳炎になったりとお騒がせな @xtetsuji ですが、皆さんにご心配をおかけした胃潰瘍からはだいぶ解放されました。まだ食事制限的なものはありますが、入院当時に比べたら本当に元気です！

退職する！

現職を2014年1月31日に 退職することになりました 。新天地に向けて期待半分不安半分なので、励ましの言葉をかけてください！

新しい勤務先や転職活動の裏話が聞けるのも #xtnight だけ！日本でも珍しい mod_perl ハッカーの30代男性がどのような転職活動をしたのか、裏話が聞けちゃうぞ！

場所

新宿の居酒屋「月の雫」新宿郵便局前店です。渋谷からもアクセス良好。地下からだと京王新線・都営新宿線が近い感じ。新宿西口のビックカメラの近くです。名前通り新宿郵便局がランドマークです。スマホの地図で目立つ！

混雑時は2時間制限があるかもということで、21時以降場所が変わるかもしれません。#xtnight のハッシュタグを要チェック！
