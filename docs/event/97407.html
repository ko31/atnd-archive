36回目のSWWDC(Sendai World-wide Developers Community)は、 毎年恒例の「WWDC キーノート復習会」です。

開催概要

日本時間6月5日(水) AM2:00-から行われる、Appleの開発者向けイベント「WWDC18」のキーノート(基調講演)復習会を行います。発表のムービーを見つつ、内容をあれこれ語り合います。

  • 「眠いので基調講演を見れなかった」という方も
  • 「起きてたけど英語がわからんかった」という方も

みんなで集まれば、バッチリ内容が理解できるのではないかと思います。発表を見て、今後のApple関連の技術をおさらいしましょう。

タイムテーブル

  • 18:30 - 21:00 基調講演の映像を見ながらあれこれ語りあいます
  • 参考：昨年のムービーはこんな感じでした。今年も日本語字幕が付くかも？
    • WWDC 2017 Keynote - WWDC 2017 - Videos - Apple Developer

参加費

  • 参加費は無料です。

懇親会

終了後、 懇親会を行います。だんまや水産とか串鳥とかへ行く予定です。（予算3500くらいが目安）

会場

  • 株式会社ナナイロ セミナールーム
  • 〒980-0014 仙台市青葉区本町2-3-10 仙台本町ビル3F
  • Wifi・電源コンセントあります

会場諸注意

  • 注意1: 18時以降は正面玄関が閉鎖されます。ビルの北側入り口(StreetView)からお入りください。
  • 注意2: 20時以降は北側入り口も施錠されてしまいます。20時以降に来られる方は、@totottiまでどうにかして連絡ください
