企業イメージを変革し、毎年1,500名を採用する方法を語ります！

catch
description

通信を中心とした事業を展開し、150社以上の企業グループから成る光通信。同社の戦略上、人材採用が成長の要になっており、グループで毎年1,500名の採用が行われています。

しかしながら、
・光通信はノルマが多くて大変だというイメージがある
・離職率が高いのでは（ブラックなのでは）？
・なにをしている会社なのが分かりにくい
など、求職者が抱くイメージは必ずしもよいものばかりではありません。
と同社 人事本部部長の竹内氏は語ります。

過去のイメージからの脱却のため、正しい会社の実態を伝える、働いている姿をイメージできるように多くの時間を求職者とのコミュニケーションに費やす、積極的に他業界の人事とイベントを開催し、多くの方に光通信を知ってもらうなど、多くの努力と試行錯誤を繰り返されています。

今年５月には、“ダイバーシティ採用“という新しい採用の手法を取り入れるなど、まさに攻めの採用を実践し、より光通信を飛躍させる人材の獲得に意欲を燃やす竹内氏。「どうすれば欲しい人材が来てくれるのかを常に考え、イメージ変革や必要な制度づくりを行っている」といいます。

今回、同社がどのようにして求職者の『負』のイメージを払拭し、年間1,500名の採用を行っているのか、その秘訣を竹内氏に語っていただきます。

このような方にオススメです：
l 欲しい人材を採用するのに苦戦をしている方
l 人事の仕事から企業のイメージを向上させたいと感じている方
l 他企業の人事の方と交流をしたいと考えている方

■講師プロフィール:

その後倒産を経験し、再度光通信へ入社。
株式会社イーサポートをはじめ、光通信のグループ会社および関係会社13社の役員を歴任し、
現在は光通信 にて人事本部部長にて採用、育成を行う。
