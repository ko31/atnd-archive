description

初対面でご挨拶した方のお名前を覚えることは、大切なビジネスマナーの一つですが、なかなか覚えることができませんね。しかし、それは相手の方にとっても同じです。しっかり名前を覚えてもらうことでビジネスチャンスを広げてみませんか？印象的で記憶に残る「名前紹介」を作ってみましょう。

プログラム
・名前のマナー
・名前が覚えられない理由
・名前紹介の事例と作り方
・名前紹介を作ってみよう＜ワークショップ＞

名前紹介には、有名人や歴史、洒落、名前のルーツなどいろいろなアプローチが考えられます。

ワークショップでは、簡単なワークシートとグループディスカッションから、あなたの名前紹介を作っていきます。

名前紹介の例
「佐藤です。甘い性格なので、シュガー佐藤と読んで下さい」（よくある名前）
「元総理の福田です」（女性）
「“ハジメマシテ！”のハジメです」（外国人への名前紹介）

楽しく、名前紹介をつくってみましょう(^_^)/
