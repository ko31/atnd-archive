セッションの目的

能登で生まれた方、能登に移住された方、能登移住希望の方が能登半島を安心してさらに魅力ある生活の場とするには何を創り、どう具体化すれば良いのかを探る。

対象者

地域活性化に興味のある方。新しい働き方、魅力ある能登半島づくりに興味のある方。

詳細説明

能登半島にはまつりがあり、伝統工芸があり、観光があり、世界農業遺産に指定されるなど里山里海との調和もあり様々な魅力があります。しかしながら、若者の流出に歯止めがかからない実態があります。何をどうすれば若者に魅力ある地域になるのかを探っていきたいと思います。
