OpenSolaris ハッカソン

今回のハッカソンは 2月に行われた静岡キャンプ の補講会になります。

Solaris 11 や OpenIndiana ユーザ向けのパッケージレポジトリサーバの立ち上げとレポジトリに載せるパッケージを作成するため、レポジトリサーバとパッケージ構築のためのハッカソンとなります。

参加者が用意する物

必須

  • SSH できる端末 (Windows, Mac, Linux, Solaris なんでもOK)
    ネットワーク (有線/Ether, 無線/WiFi どちらか必須)
    　　各自にハッカソンの時に利用する「開発用Zone」を割り当てます。
    Solaris/OI環境を用意する必要はありません。

推奨

  • Solaris 11 or OpenIndiana 151a がインストールされた PC/Note

事前準備

  • 予習/復習
    瀧さんの１月の勉強会の資料
    Solaris11用のCBE作成
    ビルド準備
    初心者むけガイド- yamachanさんからの投稿 -
  • Build に参加する方は、
    SourceForge.JP
    でアカウントを作成して、jposug プロジェクトに参加申請をしてください。
  • 公開鍵を登録しておくことで、 commit ができようになります
  • 自分がメンテナーになりたいと思うソフトウェアを tar ball, gz で用意
  • RedHat でパッケージ作成の経験がある方は、RPM の spec ファイルもあるといいそうです。

懇親会

ハッカソンの後には交流会も開催いたします。会場にて参加人数を確認させてください。(一人3000円ぐらいを予定)

注意事項

Oracleのセミナーではありませんので、ご注意ください。
開場は 11 時を予定しております。設定で作業開始は遅れるかもしれません。
