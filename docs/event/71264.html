あなたは、ぐっすり眠れていますか？ 毎日気持ちよく目覚められているでしょうか？

たとえば・・・ 寝たいのに眠れない 寝起きが悪い、起きれない 早過ぎる時間に目が覚めてしまう 寝ても寝ても眠い 疲れが取れた気がしない

なんて事はありませんか？

睡眠の悩みって意外に多く持っておられて、意外に幅広く、しかも改善がなかなかしにくかったりします。 お医者さんに通ってお薬をもらうしかない方も多いかもしれません

たしかに、一時的に日常生活の支障をきたさないためにも 対処療法のお薬は必要な場合はあります。

でも残念ながら、それは根本的な解決にはならないのですね。 急にお薬を止めてしまうのは危険ですが 日常生活などから必要なことを改善していくことは可能なんですよ！！

しかも 世の中には様々なサービスや商品が溢れていますのでなお一層何が良いのやら、迷ってしまいます. じつは、ご自身の問題を自己判断で解決しようとすると逆効果の場合もあったりするのです

睡眠の悩みを軽視していると,鬱や過食、生殖機能不全など病気の引き金になってしまったり・・・・

そこで！！

睡眠の専門家である、睡眠健康指導士手島泉よりお一人ひとりの悩みに向き合う為、お茶会を開催することとなりました。

＜このお茶会に参加すると＞

眠りが楽しみになる ♬

自分の状態が分かるので、無理をしなくなる

疲れが取れるようになる ♬

お医者さまにも聞けなかった悩みに向き合ってもらえる

健康で時間を有効に使えるようになる

などなど。 やはり人生の豊かさってお金だけではなく、健康な心身あってこそですものね。 衣食住の視点で幅広くアドバイス頂けるのは、認定された睡眠健康指導士（上級）だからです。

どこにもない、睡眠のお茶会 睡眠に関する基本的なこと、プラス、毎回テーマが変わっていきます。

初回は「睡眠時間と眠りの悩みについて」 みんなどれぐらい眠っているの？ 一番最適な睡眠時間って？ ゴールデンタイムって本当？ 仕事や勉強の効率が上がるのは？ 自分に合った睡眠時間って？

どんどん質問をご持参ください！

あなたの、そのどうにかしたい悩みの原因を知り、１００問１００答で自分だけに合った改善方法のアドバイスを頂きます。

＜講師ご紹介＞ 睡眠健康指導士（上級）　手島　泉 ( 快眠カウンセラー )

有限会社ティ・エス・ディ・取締役 GuGu-Life事業部代表 一般社団法人日本睡眠教育機構 睡眠健康指導士（上級）1962年生まれ 眠りは一番シンプルな美容法であり健康法。 肌色を見るだけで、その人の睡眠状態が解ります。眠りの悩みは人それぞれ、眠りで社会が元気になることを使命として活動しています。
