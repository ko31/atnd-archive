Mapillary Meetup 2019 Nara, Kansai

位置情報付写真の世界共有サービス Mapillary のユーザー交流会"Mapillary Meetup"を3月30日（土）奈良市で開催します。Mapillary Meetup国内開催3回目、関西では初の開催になります。
世界中をカバーするMapillary写真は4億5千万枚を超え、国内でも共有写真数は増え続けています。位置情報を持った写真共有の楽しみはもちろん、OpenStreetMapのマッピングツール、ビッグデータとして画像認識、AIの分野からも注目の的です。ユーザー目線のMapillaryに関する貴重な撮影how-toや体験発表、利用法、テーマ別ディスカッションなどを予定しています。Mapillaryに興味をお持ちの皆さん、情報共有、交流を深めてみませんか。気軽にご参加ください。
