Moverio BT-200 勉強会

EPSON から発売されたスマートグラス Moverio BT-200 に関する開発者向け勉強会です。使い方の Tips や開発のギモン、期待感などを情報交換しましょう。デモの持ち込みも大歓迎です。BT-200 は持っていないけど開発に興味がある方も是非ご参加ください。参加費は無料です。

LT (1人5分)や発表も募集しています。Twitter: @ksasao か ATND での登録の際にその旨ご記入ください。

当日は ustream での配信を予定しています。Twitter のハッシュタグは #bt200dev です。
