セミナー内容
【貨幣と肖像】
平木啓一先生による【貨幣と肖像】をテーマにしたセミナーです。
コインに描かれた肖像について知識を深め、コイン蒐集の面白さをさらに知っていただきたいと思います。
今回は食事会形式ですので、講演の途中であっても、質問事項などはその都度お受け致します。

平木啓一プロフィール
平木啓一顧問のご紹介
1942年　愛知県蒲郡市生まれ
1965年 世界のコイン研究を開始。
コイン専門誌<ボザンナ>に「世界の大型銀貨」を2年にわたって連載
67年　徳間書店より『入門世界のコイン』
68年　文藝春秋社より『コイン集め方楽しみ方』
73年　ジェミニより『世界貨幣大辞典』
『コインとインフレヘッジ』を著す。この年ジェミニを設立、社長として
6年間、世界のコインを輸入販売に当たる。
2010年　ＰＨＰ研究所より『新・世界貨幣大辞典』を改訂出版する。
12年　PHP研究所より『アンティ—ク・コインで資産を防衛せよ』
13年　『楽しみながら資産を殖やす コイン・コレクションのはじめかた』
を著す。
2013年　　CoinExpert合同会社顧問に就任
