ZendEngine勉強会@東京

PHP 拡張を主眼にした ZendEngine 勉強会を実施します。

周りに相談する人がいなかったり、日本語どころか英語のマニュアルも真っ白だったりと、

発表に感化された人達が仲間になってくれる事を夢見てます。
