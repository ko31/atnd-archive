「クラウド」という言葉が登場してから久しい昨今。
IT環境は常に変革し、効率化・スピードが求められる時代。クラウドサーバによるインフラ管理は欠かせないものとなりました。

今回は、クラウドサーバとしてグローバルで利用されているAmazon Web Service(AWS)を使うことで「何ができるのか」を主に解説します。
