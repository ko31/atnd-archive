======= クラウディ会 (第３回、日本語、12.13＠天神 ) 10:30～14:00 =======
● クラウディ会の目的：クラウドにおける或る具体的な問題を解決すること
● (第３回の）課題：コンテナ技術：Docker vs Heroku vs… Fog Computing
● 概要 :
-
コンテナ技術が導入されたクラウドでは、アプリ・リソースのマイグレーション
が非常に楽になりました。ただし、１つの欠点として、コンテナを使うときに、
普通の仮想マシン(VM)と異なる環境になり、各種ルールに従うことに手間が
かかってしまう。本クラウディ会では、クラウド・コンテナ技術の背景を明確に
してから、具体的にDockerとHerokuの技術を（実際のソフト・環境で）体験する。
最後に、Fog Computingというクラウドの最新技術とそのコンテナ形式を議論する。
-
● 日程：12月13日（土）10:30~13:00
　　・10:30～12:00 講義、Docker・Heroku体験
　　・12:00～　休憩
　　・休憩中後～14:00 体験・演習、議論・結論探し

● 会場：九工大＠天神
