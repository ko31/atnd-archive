■概要

コミュニティメンバが、実際にMarketplace に公開されたアプリのソースコードを持ち寄って、
Windows Phone アプリの開発方法を習得しよう！という勉強会です。

今回のテーマは「○○検索」というタイトルで公開されている Webブラウザを使ったWindows Phone アプリです。

■内容・到達目標

・準備-ソースコードを共有する方法の解説。
（Team Foundation Service Preview、あるいは Subversion を予定）
・サンプルコードの配布と、開発の実際と展開。
・Marketplace公開へ向けて-AppHub への申請手順と審査に際してのポイント-

勉強会参加後には、新しいアプリを実際に作って AppHub に申請・Marketplaceへ公開できるようになることを目標とします。

■前提知識

Visual Basic, C# での開発経験
(※ユーザーインターフェイスの開発経験は必須ではありませんが、
WPF, Silverlight, ASP.net, Windows Form のいずれかの経験があるとわかりやすいと思います）
