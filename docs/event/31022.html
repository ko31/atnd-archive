せきゅぽろ　ナイトセミナー#05

　せきゅぽろ・ナイトセミナーは北海道で開催している北海道情報セキュリティ勉強会(せきゅぽろ）の平日夜に開催する勉強会です。

　今回のスピーカーは参加者の皆さんです。持ち時間15分以内でみなさんの熱い思いを語って下さい。もちろん傍聴のみの参加もOKです。

　お題はセキュリティに係るものから、熱いと思っているもの、苦労話、などお好みでお願いします。勉強会終了後には懇親会を開催します。ミニセッションで語りきれなかった熱い思いをぶつけあいましょう。
