プレミア逆求人フェスティバルとは？？

リーダーシップ、アントレプレナーシップを持つ就職活動生のためのスペシャルイベント。将来のビジョンについて、企業に直接アピールできる“内定直結”の企画です。将来ビジョンを実現するための学習意欲の高さ、地に足をつけて行動できる成熟さ、自分のメジャーメント（モノサシ）、こだわりの強さ、他人に依存をせず自己成長できる強さを持っている就職活動生のみが参加できます。

★詳細はこちら
http://www.studenthunting.com/event/premium/index.asp

【ステップ①】
事前選考会に備え、「我究館」会長の杉村太郎氏による特別セミナーを受講していただきます。テーマは『キャリアデザインを考える』。将来のキャリアビジョンを考える上でのポイントをお話頂きます。

【ステップ②】
セミナーで自分のキャリアビジョンを整理できたら…次は、プレミア逆求人フェスティバルに参加するための、事前選考会（面談）です。選考委員との面談で、あなたのキャリアビジョンをアピールしてください。
プレゼンテーションテーマは『将来ビジョンと現在の取り組みについて（仮）』。

【ステップ③】
プレミア逆求人フェスティバル当日。
企業の経営者や人事担当者に、思う存分、あなた自身と、あなたのキャリアビジョンについてアピールしてください。
お互いが十分に納得できれば、その場で内定が出されることもあります。
