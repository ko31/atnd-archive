初心者・未経験者歓迎のiOSアプリ開発交流会及びハンズオンです。

第13回アプリ開発教室では、「日付計算アプリ（人生の残り日数を表示するアプリ）」を作ります。

経験者の方もお待ちしております。

プロジェクトについて話したり、知見交換しましょう。

集合場所

Co:Lab 難波まで直接お越しください。

スケジュール

10:00 〜 10:30 : 情報交換、プログラミング、各自タスクなど、自由に交流してください。

10:30 〜 13:00 : アプリ開発教室 (※参加、退出自由)

持ち物

アプリ開発教室ご参加の場合、Macbookをお持ちください (見学のみでも大丈夫です)

料金

参加費 : ¥500 ※ Co:Lab難波の利用料になります。

第13回 アプリ開発教室

「日付計算アプリ」を作ろう。

参加条件 : Macbookをお持ちの方

準備 : 「XCode」 をApp Store からダウンロードして下さい。

※ XCodeのダウンロードはかなり時間がかかるので、事前にお願い致します。
