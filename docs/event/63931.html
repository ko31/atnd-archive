description

羊毛はふわふわで手触りがよく、色の種類も豊富なので、
手にとって眺めているだけでワクワクしてきます。
今回はその羊毛を使って、無地のコットンバッグに
ニードル（専用の針）でチクチクし、絵をえがくように、
思うがままに描いていきます。

お子様には難しく思われがちなチクチク作業ですが、
ここではテーブルに置いて安定させながら作業できるので大丈夫です。
親子で一緒に作るもよし、もちろん、おひとりで思う存分描くもよし♪

みなさまのご参加、お待ちしております。

講師：やまだみかこ（イラストレーター・キノコ作家）

参加人数（対象年齢）： 6組（4歳以上）※親子での参加も可

注意事項： 先のとがった針を使いますので、小さなお子様は大人の方と一緒に作業されることをおすすめします
