description

未来を創造する国際協力のありかたを考えることを目的として、この度社会起業大学の在校生が中心となって、映画の自主上映会及びゲスト講師による講演会を開催いたします。
前半は、映画『懐かしい未来　ラダックから学ぶ』（ダイジェスト版）監修：ヘレナ・ノーバーグ＝ホッジをご覧頂き、マインドセットの共有を行います。
ヒマラヤの辺境ラダックに関するドキュメンタリー映像を見て考え想像することで、世界、そして日本の過去・現在・未来について、さまざまな気づきを得て頂き、参加者の皆さまが世界を捉える新たな視点を獲得し、人としての本当の豊かさや幸せ、今の社会のあり方に関する問題意識が育まれ、未来は選択できるということへの理解を促せればと考えております。
閉塞感のある現在の時代状況を大局的な視点で捉えなおすことで、参加者の皆さまの心の中に、未来に対する希望を芽生えさせる事が目的です。
後半は、ゲスト講師(株式会社ピースダイヤモンド代表取締役　村上千恵氏)をお招きし、「ピースダイヤモンドと未来の創造」というテーマでご講演いただきます。
