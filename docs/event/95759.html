イベント概要

KROWLではインターネット/Web業界現場のナマの声や、スキルアップに役立つ情報を毎月お届けしています。Web制作・運営、インターネット広告、SNS、ソーシャルゲーム、オンラインショッピング、ポータルサイトの運営・・・など。インターネット/Web業界へこれからチャレンジしたいとお考えの方、スキルアップがしたい方、どなたでも歓迎ですのでふるってご応募ください。また、キャリアコンサルティング、インターンシップ情報を含めたKROWLの最新情報もお知らせいたします。

※ご参加にはKROWL会員登録が必要です
https://krowl.jp/seminar/detail.php?seminar_id=23

ワークショップ概要

デザインに興味がある、これからWeb業界にチャレンジしたい、スキルアップがしたい、
そんなデザイン未経験の方・初心者の方にぴったりな社会人向けワークショップです。
デザインがスムーズになる思考方法を、課題（バナーデザイン案の発想）を通して学びます。
Web業界での実務経験のない方や、ワークショップ・勉強会に参加したことが無いという方でも
気軽にお楽しみいただけるワークショップです。

※デザイン職未経験の方や、制作経験の浅い方に向けた基礎的な内容です
※デザインの思考方法に重点を置いた内容です。Photoshopの操作方法など、ツールに関する内容はございません。

地図

主催： KROWL

KROWLとは、株式会社アイ・ファクトリーが行っている、これからWeb業界でチャレンジしていきたい方を対象に、セミナーや面談、キャリアコンサルティング、スキルアップ研修等の転職支援サービスです。
Web業界を知っている企業がサポートしているからこそ、転職活動もスムーズに。ぜひKROWLの転職支援を体験してください。

また、アイ・ファクトリーは博報堂アイ・スタジオグループとして、2011年に設立したデジタル制作×人材サービスの会社です。大手・有名企業のコーポレートサイト、キャンペーンサイト、バナーなどのデジタル制作と、自社の教育システムを活かしたIT/デジタル人材の育成を行っており、業界未経験から現場で活躍できるクリエイターをこれまでに多く輩出した実績があります。
