毎月最終金曜日の夜だけ営業する幻のお店『二丁目ごはんBAR』

いつも通り美味しいお酒とごはんを用意して、内田と美由紀もお待ちしております♪

★スペシャルゲスト：佐々木ののかさん★

今回は「体当たり系ライター」の佐々木ののかさんをスペシャルゲストとしてお招きします。
みんなこの記事読んだ事ありますよね？ 佐々木さんがどんな風にして記事を作っているのか「逆取材」できるチャンスですよ〜！

【LINE晒し】金曜夜の新橋サラリーマンに「愛してる」と送らせた
http://wedding.mynavi.jp/kekoon/entry/2016/01/23/130000

2015年5月末で新卒で入った会社を辞め、フリーライター。体当たりライターと世間では呼ばれていますが、意識の上ではインタビューや対談記事に軸足を置いています。築50年の風呂なしアパートに住んでいる、お酒が大好きなポップなメンヘラです。よろしくお願いします。
https://twitter.com/sasakinonoka

トークタイムは20時〜1時間程度ですが、その後、佐々木さんと一緒にお酒を飲みながらお話いただけます。

もちろん、いつも通り美味しいお酒とごはんもご用意して、内田と美由紀もお待ちしております♪
★今回は18時スタートです。お間違えなく★
