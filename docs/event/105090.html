これ is 何？

競技プログラミングを始めたばかりのはにーま（@mtmnrc）が
競プロ勉強合宿みたいなものを開催してみたいと思い企画しました。

「競プロを始めたばかりなので、もっと色んな人と交流してみたい」
「イベントに行くと、みんなつよい人ばかりで怖い…参加しにくい…」
「いつもよりまったりとした雰囲気で他の人と競プロをやってみたい」

この合宿で主に対象としているのは、こんな感じの競プロ初心者の方です。
ぜひお気軽にご参加ください！

もちろん、普段からコンテストをバリバリこなしている上級者の方も大歓迎です！

内容

2つのプラン（チーム戦／個人戦）が用意されています。
どちらのプランにしてもゆるふわに競争し、
ランキング上位の方には何らかのささやかな景品をお贈りする予定です。

問題は全て運営の方による新作です。
（作問を手伝って頂いた運営の方、ありがとうございます！）

なお、どちらのプランでも難易度的には
AtCoder Beginner Contest（ABC）からAtCoder Regular Contest（ARC）程度を目安とします。

ルール

  • AtCoderでのレーティング総和が3500以下となるようにすること。（チーム戦）
  • 同時にコーディングできる端末は1台のみ。ただし、問題と検索に限り、2台以上用いてもよい。（チーム戦）
  • 1人当たり1問以上コーディング（提出）を行うこと。（チーム戦）
  • ライブラリのコピペ可。
  • インターネットでの検索可。

なお、参加者に初心者が多いことから
1日目の状況を踏まえてルールが変更される可能性があります。

詳しくは合宿中にお伝えします。
