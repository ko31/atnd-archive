もし仮設ネットカフェ以外の企画や、ITで復興を加速するための相談などあればそちらもお気軽にお申し出いただければと思います。

アイディア＆ニーズ募集！

当日、遠野に足を運ぶのは無理なんだけど、アイディアやニーズがある方は、開催当日までに、是非こちらに一言お書きいただけると幸いです

被災地シェアタイムスピーカー募集！

10/15に、岩手の被災地の今をシェアしたいがある方！是非５分間スピーチをお願いします！
立候補してくださる方は、こちらに書き込みをお願いします

会場について

  • お弁当の持参を推奨します。飲み物は会場内に自販機があります。
  • 電源、ネットワークが利用可能です。ノートPCと延長コードをお持込ください。

当日、遠野まごころネットに宿泊を希望の方

  • 15日の夜、遠野まごころネットに泊まる方は、10月13日までに申請が必要なため、akiko.iwakiri [at] gmail.com （ [at] を@に置き換えてください）に、「氏名」「生年月日」「住所」「電話番号」をメールでお送り下さい。
  • 宿泊場所は遠野市総合福祉センター体育館になりますので、各自寝袋等の寝具と洗面用具をご持参ください。
  • 遠野の朝晩は冷え込むようになってきました。就寝時の服装は厚手の物をご用意ください。
  • ボランティア登録を当日していただくことになります。
