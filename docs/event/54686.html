title エデュコレ
catch 多様な教育の博覧会
description

エデュコレとは、さまざまな「教育のカタチ」を一望できる「教育の博覧会」です。
学校、塾、フリースクール、オルタナティブスクール、教育NPO、企業などなど、いろいろな立場から子ども・教育にかかわっている団体が、この日一堂に会します！
学びの場をつくっている人たちが、実践に込めた思いを語る講演、
それぞれの現場の雰囲気や取組みを「つまみ食い」できるようなワークショップ、活動紹介のブース出展など盛り沢山の内容です。
教育に関わる一人ひとりが、未だ見ぬ多様な教育のカタチに出会い、
自分の知っている教育を根っこから見つめなおす。
参加者・出展者と共に「教育の未来を考える」機会をつくりたい。
このような想いで出来たイベントが「エデュコレ」です。
