南極入門講座

ども、中谷です。 スーパーコンシューマー の打ち合わせの後に、南サンと飲みに行ったら、南極の話題になった。
南極旅行なんて考えたこともなかったけど、興味が湧いたので調べてみた。
そしたら、気になることがドンドン出たんです。

  • 南極にはコンビニが2つあるらしい。
  • そもそもどうやって行くのだ？
  • 南極ピラミッドって何だ？

気になって仕方ないので、ネットを徘徊して、いつものように突撃メールを送っていたら、南極について教えてくれる方を見つけた。

南極観測隊を40年やってた方だ。

お会いしたことないからどんな人か分からないけど、この人に聞けば、僕の疑問なんてすべてクリアになりそう。

ということで、その方に来て頂いて、僕と南さんで授業を受けるのですが、もし興味ある人がいたら、一緒に受けたらいいんじゃないかと思ってイベントを立ち上げます。先着順です。

講演概要

写真や映像を使って解説してもらう予定。僕から講師にお願いした質問は以下。

  • 南極の歴史を知りたい。
  • 南極の現状を知りたい。
  • 南極探検史を知りたい。
  • そもそも南極ってどうやって行くんでしょうか？
  • 南極観測隊で提供される料理。現地ではどのような食事を食べているのか？南極料理人みたいなのか？
  • 南極に行った人はどんなことをしているのか知りたい。
  • 安く行く方法があるのか知りたい。
  • 南極ピラミッドの謎、人類が住んでいたという噂は本当なのでしょうか。
  • オーロラがよく見える季節があれば知りたい。
  • 南極のコンビニにATMがあるそうですが、お金を引き出す方法があれば知り* い（アメリカの何銀行？）
  • 南極から葉書を送ってみたい。
  • 南極と北極、どっちが寒い？
  • 南極での生活は体力がいるのでしょうか。年を取ってから行くのは厳しいのでしょうか？
  • 南極研究者って何を研究してるのか知りたい。

正直このイベントが面白いのかどうか僕らも分からない（笑）ですが、きっとオモロくなる気がするので、興味を持った人は是非ご検討くださいませ。
