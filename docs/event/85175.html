ポップコーントークとは

ポップコーントークは好きな映画を2本ずつプレゼンし、紹介し合う、シンプルな映画シェアイベントです。
昔から大好きなあの映画、最近観て久しぶりに感動したその映画、、あなたの大好きな映画、教えてください。

今まで出会ったことのない映画、そして仲間に出会える、そんなイベントを目指しています:)

Time table

1. 自己紹介&プレゼン（15:00-16:30）
→受付であなたの紹介したい映画2本を伺います
→こちらでYoutubeでプロモーション動画を検索し、参加者全員の動画リストを作成します
→1人6分持ち時間で前でご紹介（プレゼン）していただきます

2. 座談会（16:30-18:00）
→トピックごとに分かれ、参加者同士で映画の話をします
→内容は当日発表します！
