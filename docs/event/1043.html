description

その目的すら定かではない「ええじゃないか」という社会現象から150年が過ぎ去った今の時代を生きる私達が、この現象を通じて学びと出来うることは何か？
資料にもあるように、「ええじゃないか」を構成する多くの民衆は、町々を熱狂的に踊り狂うという、その行為自体が楽しそうだったから、そこに意味を求めずただただ楽しく騒いでいただけのようである。

物事には本質が付き物だという。言わずもがな、この旅も然りである。

今日、そして明日と諸君にはあらゆる興奮や欲望が降りかかることだろう。
その際諸君には、本来「ええじゃないか」という現象が社会へもたらす影響、その行動が放つメッセージなど気にもかけず、ただ「楽しそう」というだけで、踊りに興じ「楽」をとっていたような男達と同じようなことにはなって欲しくない。

もう一度考えて欲しい。この旅は自分にとってどんな意味があるのか。この旅は誰のための旅なのか。
希薄な欲望に打ち勝ち、是非この旅を各個人が充実し、本質を見出せるような旅となることを切に願うばかりである。
しらねーけど。
