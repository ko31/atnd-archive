1.0〜1.7の1系統でも多くのWeb制作者に利用されたa-blog cmsは、
昨年12月にリリースされた2.0にメジャーバージョンアップし管理画面の改善、多段階ワークフロー、テンプレート機能の向上といった多くの改善を実現しています。

今回はa-blog cmsの開発元アップルップル代表の山本一道さんにa-blog cmsの魅力とその機能をお話いただき、更にはKDDI Web Communicationさまの事例やMovable Type/WordPressとの徹底比較等々のセッションも予定しております。

Movable TypeやWordPressでCMS案件をやっているというWeb制作者の方々や、よりエンタープライズなCMS要件を考えていらっしゃるWebマスターの方々にも参考になる内容を目指しています。

ぜひご参加ください！
