『Web業界への転職を有利にするポートフォリオとは？』

OPEN 18:30 / START 19:00 / CLOSE 20:30

KROWLセミナーについて

KROWLセミナーではWeb業界現場のナマの声や、スキルアップに役立つ情報を毎月お届けしています。
Web制作・運営、インターネット広告、SNS、ソーシャルゲーム、オンラインショッピング、ポータルサイトの運営など・・・
Web業界への転職をお考えの方、キャリアチェンジやスキルアップをお考えの方に向けた内容となっています。
特に実務未経験の方、初心者の方にピッタリの内容ですので、お気軽にご参加下さい！

イベント概要

メイントーク

今回のセミナーではポートフォリオを徹底解説！制作×採用を通して見えてきた《企業が本当に求めているもの》をお伝えいたします。

Webデザイナー・ディレクター・エンジニアなど職種ごとのポートフォリオ制作のコツや、より魅力的に見せるための方法、採用担当者が見たい部分・重視する部分はどこか、など、ここでしか聞けない情報が満載です！ 実例やサンプルも交えながら、転職に有利なポートフォリオを作るための具体的なポイントを解説いたします。

これからポートフォリオを作りたいという方も、今あるポートフォリオをもっとブラッシュアップしたいという方も、必見の内容です！

【スピーカー情報】

堀内　裕文

IT商材の営業を経て、Web制作会社にて大手企業のキャンペーンサイト等の企画から運用までを経験。担当案件数は100件以上。
その後、KROWLを立ち上げ、ポートフォリオディレクターとして100名以上の作品をディレクションし、制作の話ができるキャリアコンサルタントとしても活動中。

質問・相談会

ここをもっと詳しく聞いてみたい、そんな疑問・質問にお答えします！

主催： 株式会社 アイ・ファクトリー

アイ・ファクトリーは博報堂アイ・スタジオグループとして、2011年に設立したデジタル制作×人材サービスの会社です。大手・有名企業のコーポレートサイト、キャンペーンサイト、バナーなどのデジタル制作と、自社の教育システムを活かしたIT/デジタル人材の育成を行っています。また、業界未経験から現場で活躍できるクリエイターをこれまでに多く輩出した実績をもつ会社です。

KROWL とは

これかWeb業界でチャレンジしていきたい方を対象に、セミナーや面談、キャリアコンサルティング、スキルアップ研修等の転職支援を行っています。
Web業界を知っている企業がサポートしているからこそ、転職活動もスムーズに。ぜひKROWLの転職支援を体験してください。
