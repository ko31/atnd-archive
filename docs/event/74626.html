【参加無料】2/19　18:30～　crash.academy主催　採用・育成担当者対象セミナー

昨今、事業でITの重要性が上がる中、優秀なITエンジニアを獲得することが今後の事業を左右するという企業が多いかと思います。

しかし、市場動向としてはITエンジニアの求人数が増える中、転職希望者は減少を始め、今後さらに採用が難しくなると予想されます。

探してはいるのですが、なかなか御社に合う候補者がいなくてと、エージェントから紹介してもらえないという採用担当者様のお声もよくお聞きします。 そんなかでも採用に成功している企業は存在します。

本セミナーではそうした観点から、そもそも今どういったIT人材が求められているのかという根本的な問題から、エージェント視点から見た、採用に成功する企業・失敗する企業という点についてお話します。

【参加特典】　南政樹先生によるオリジナルIT研修プログラム作成！

ご希望の採用・育成担当者様へ社内IT人材の育成課題に合わせた、南先生オリジナルIT研修プログラムを作成いたします。

※お手数ですが、当日は出席確認のため名刺を1枚お持ちください。
