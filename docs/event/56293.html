description


ありがとうロバナイト！
ありがとう区ト間！
ありがとうすべての音楽！
ありがとうすべての変態映像！
ありがとうみんな！

ロバナイトは東京のクラブシーンに確かな爪痕を刻み込んだ。
素敵な音楽とどうしようもない変態映像と無闇にアルコール度数の高い梅酒でオーディエンスの視覚と聴覚と精神と肝臓に消えない傷跡を残した。
だが今ロバナイトは静かに潰えようとしている。
そう、区ト間閉店とともに。
おしゃれな空間と、小粋な店長と、おいしい梅酒のくつろぎスポット、区ト間。
店の雰囲気にそぐわない変態プレイの数々を朝まで見なかったことにしてサポートしてくれた区ト間の閉店により、今ロバナイトは終わろうとしている。
だが、このまま消え去りはしない。
ロバナイトは新たな仲間達、日本最強の童貞集団JCCの力を借りて最後の一夜に
その全てを叩き込む。
オーディエンス諸君、君たちは歴史の終わり、そして伝説の始まりを見る。
良識的な人々が眉を顰めるような変態映像がエンドレスで流れ続けるだろう。
いくつものUstreamアカウントが無慈悲に垢BANされるだろう。
大量のアルコールで沢山の肝細胞と脳細胞が死滅するだろう。
DJは顔に落書きをされて寝コケるだろう。
おしゃれな音楽と甘い雰囲気で新たな恋の１ページが始まっちゃったりするだろう。
狂乱の夜が始まる。

DJ

さとやん
今回のロバナイトがこの日になったのは彼のせい。

DJ中にフロアに投げ込まれるブーケトスに期待だ！
拾った人は40過ぎてやっと結婚できるぞ！
みんな祝ってあげよう！

Yamajet
前回に引き続き参戦。
世界で一番ビールジョッキの似合う三十路。

DJ急○の後継者と噂される歌って踊れてトークもできて腹筋をするDJアイドル。

そしてJCCからは

TEKKA
渋谷clubasiaや代官山AIRで開催される“UP BEAT!”でUKハードコア、ドラムンDJでフロアを沸かせるジェントルマン。
お忍びで正体を隠して謎のDJをやってたりするらしい。
この前四角いお面で正体を隠した謎のDJを見たが、もしやあれは…

Azsa
JCCと双璧を成すハードコアの底辺of底辺パーティー「nitro-x」の主催。
JCC大阪では彼に風俗を誘われたが行きそびれて大変に残念。

DJ庭師　a.k.a. Black Butterfly
庭師。
