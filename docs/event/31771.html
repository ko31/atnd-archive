■登壇内容

●サイバーエージェント社

【テーマ】Javaによるゲーム開発パッケージ化への取り組みと開発ノウハウ
【登壇者】山田　元基（やまだ　げんき）氏

「ゲーム開発におけるスピードとクオリティを担保するためにゲーム開発のパッケージ化を進めています。
パッケージ化を行う時の課題と実際にサービスに組み込むノウハウをお伝えします。」

●gloops社

【テーマ】ソーシャルゲーム業界における、C#の真の魅力
【登壇者】河合 宜文（かわい　よしふみ）氏

「急速な勢いで拡大し続けるソーシャルゲーム業界において、その成長と同じ様に日々進化する「C#」は、
想像を超える安定性と、ソーシャルゲームへの適応力を持っていることは知られていません。
今回はこれまでの実例を通し、「C#」が持つポテンシャルを余すことなくお伝えします。」

～C#の魅力～
2012年の現在もアップデートされている、進化し続ける言語。
C#は、Visual Studio, SQL Server, Windows Serverなど、
オールインワンで取り巻く環境が提供され、どれもが強力なGUIを備えているなど、
非常に安定したリソースを備えている。

■登壇者プロフィール　

　◎山田　元基（やまだ　げんき）氏
　　@ygenki

　　株式会社サイバーエージェント　アメーバ事業本部　ソーシャルゲームディビジョン
　　コアシスグループ　

　　2008年新卒でサイバーエージェントに入社。
　　アメーバブログの開発・運用の経験を経て、これまで4本のソーシャルゲームの開発・運用に携わる。
　　MongoDBを利用したソーシャルゲームなど新しい技術にも取り組んでいる。
　　現在はソーシャルゲームの基盤開発を行っている。

　【山田氏より一言】
　「Javaによるソーシャルゲーム開発の利点がとてもわかる内容にしたいと思っております。
　　ゲームのライブラリ化を進める事での事例をお伝えしたいと思います。」

　◎河合 宜文（かわい　よしふみ）氏
　　@neuecc

　　株式会社gloops　ソーシャルゲーム事業本部　アプリケーション開発部
　　アプリケーショングループ

　　2008年新卒でソフトハウスに入社。C#での開発を行う。自身のブログ「neue cc」での
　　活動などが評価され、2011年4月「Microsoft MVP for Visual C#」受賞。
　　ゲームはGeometry Warsが好きで、GW1のRetroモードでは無限プレイ可能まで極め、
　　24時間以上プレイしてランキング10位ぐらいにランクイン。
　　また、GW2のEvolvedモードでは日本5位以内（世界ランクでは50位）ぐらいにランクイン。
　　2012年1月gloops入社。

　　【河合氏より一言】
　　「今回はC#という進化する言語の魅力を、余すことなくお伝えする場にしたいと思っています。
　　もちろん、非Windowsの方であっても、聞いて損ではない内容をご用意していますので、
　　みなさまからの参加をお待ちしています。」
