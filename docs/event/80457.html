【日本の主食のコメについて考える】

大阪堂島の地で興ったデリバティブ市場の原点、
『堂島米会所』…
その流れを汲む大阪堂島商品取引所で、
2011年8月に72年ぶりに復活したコメ先物。

今年のコメは豊作なの？不作なの？
コメの価格は上がるの？下がるの？
そういった、コメ先物をとりまく状況や、
コメの今後の見通しを弊社コンシェルジュスタッフが詳しく解説致します。

「コメの投資の仕組みってどうなってるの？」
「コメって現物で受け取れるの？」
「コメの種類ってどんなのがあるの？」
「もしかして、飲食店の仕入れとかにも使えるの？」
など、今まで知らなかった、コメの魅力の詰まったセミナーです。

今一度、コメについて勉強してみませんか？

【セミナー特典！】
今回のセミナーの特典として、
お取引にエントリーされた方にお米のプレゼント。

更に値段当てクイズにピタリと正解された方に、
お米１俵（60ｋｇ）のプレゼントが御座います！
