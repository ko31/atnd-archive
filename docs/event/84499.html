マニフレックス大阪ショールームでは、質のよい眠りに関心のある女性のためのアカカツ（セミナー）“ボンジョルノ！マニフレックス”を開催しています。朝の時間を有効活用していただけるよう、朝7時30分にスタートする体験型セミナーで、参加費500円は全額寄付されます。おいしい軽食と飲み物をご用意して、皆様のお越しをお待ちしています。

マニフレックス朝活 2017年1月プログラム

2017年1月27日（金）朝7:30～8:30

メンテナンスヨガ

毎日の生活の中で知らず知らずの間に身につけてしまっている「癖」。ひと言に癖といっても種類は口癖だけでなく、身体の使い方や考え方などさまざまです。こうした癖は放っておくと「こわばり」へと変化し、疲れやトラブルの原因となってしまうことも。
今回のアサカツは、簡単なヨガの考え方と動きを通して日常の癖をリセットするコツを学んでみます。気分が変わったり、寝つきがよくなったり、コリが解消されたり・・・いいことずくめのメンテナンスヨガをぜひ体験してみてください！初心者の方、大歓迎♪ 定員がございますので、お早めに参加のお申込みをください。

講師：数珠 孝 先生
生活ヨガ研究所代表　「働く人をヨガで応援」をテーマに活動を続ける。インド政府公認ヨガインストラクター、樹木医、森林インストラクター

参加費：500円（焼きたてクロワッサンと紅茶付き）

※要予約、女性限定
