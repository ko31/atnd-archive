【概要】

最近、フィジカルコンピューティングと言う言葉を良く聞くようになりました。その言葉の解説と、それを実現する為にArduinoと言う機材が役に立つので、それについて解説します。

本会では、主にArduinoという物が、どんなものなのか、どんなことが出来るのかについてざっと知ることを目標とします。当日、実機も数台持って行きますので、自分のPCにつないで実際に動作をさせてみることも出来ます（Windows,
OS X, Linux であれば開発環境が動作します）。

電子工作の知識や、はんだ付けの知識は不要です。
何か開発言語をやっている方ならすぐに作れます。
