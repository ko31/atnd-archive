description

初心者向けにC++11の機能を紹介する勉強会です

C++11で追加された機能を全部紹介しようと思います

  • C++03からの変更点ってなんだろう？
  • そもそも自分が使っているC++って？規格とかよくわからないなぁ…

そんなあなたにC++11をお伝えします！

この勉強会に参加すれば、C++11に対する用語や文法について
少なくともどんな機能が入っているのかを、知ることが出来ます

文法の解説もですが、出来るだけ利用シーンに基づいた紹介を行う予定です

C++初心者も上級者も、どなたでも参加できます

C++11あまり使えてないなぁという人から、
え、もうC++14も使えるのに、まだC++11のこと知らないの？というな方まで
みなさま参加をお待ちしております

どうぞ、よろしくお願いします！
