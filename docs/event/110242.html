description

スタートアップや中小企業がサービスをグロースさせるにあたって、
重要となるのが優秀なエンジニアの採用です。

ただ、エンジニアは今は売り手市場。
どこも優秀なエンジニアを求め、争奪戦が起きています。


・スタートアップは知名度が無いゆえに、そもそも応募が来ない
・応募が来ても、自社が求めている人材とマッチしていない
・良い人材の応募が来ても、面接後に辞退されてしまう


など、どこからも引く手あまたであるエンジニアの採用は、
企業にとって最難関課題と言えます。


では、この問題をどう解決すればいいのでしょうか？

そこで、本イベントでは、スタートアップや中小企業でありながら、
試行錯誤を繰り返し、独自の方法でエンジニア採用を成功させてきた4社をお招きし、
実際に実践してきた採用の取り組みについて、そのノウハウを語っていただきます。

4社のノウハウを吸収しつつ、セミナー最後の参加者主導で
質問ができるディスカッションで、抱えている採用問題を一緒に解決しましょう！
