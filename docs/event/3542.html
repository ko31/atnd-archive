description

「第2回があるか分からないのに…」、「忙しいのに…」勉強会のための資料作りは億劫だと思います。

そこで、例えばタグの意味を意識したコーディング、 SEO知識などのWEBサイト制作者なら知っていて当たり前の事は、異業種の方には当たり前じゃないかも、ということを着目し、自分たちが持っている知識を持ち寄り、整理し、誰にでも分かるように文章化し、ゆくゆくは異業種も交えた上で勉強することを意識した勉強を企画しています。

ゼロからの勉強会です。勉強がしたいなと思うより、まずは行動と思い、企画しました。

ご賛同いただけましたら幸いです。

■ 会場について
山口県宇部市松島町17-3 ハイウッドビル 3F

■ その他
・ 勉強会中の途中退出可能を予定しております。

■ 懇親会について
勉強会終了後、19時ごろから懇親会を予定しております。
懇親会の予算は1人当たり3,150円を予定しております。


■ 現在の業務、活動内容の発表について
第2回、第3回勉強会の方向性を模索するため、参加者1人、1人に現在の業務、活動内容(※)を発表していただきたいと考えております。

1人につき5分から10分程度を考えておりますが、発表の場というよりは、ディスカッション、ブレインストーミングに近い形式のものを予定しております。

※ 勉強したいこと、またはシステムが開発できない、デザインできないを別にした開発してみたいサービスなどアイディアをお持ちの場合は、現在の業務、活動内容の発表の際に、発表いただいても問題ございません。
