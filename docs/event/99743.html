★資格や経験を生かして製薬業界へ！★
人気の製薬業界への転職。資格や経験を生かしてチャレンジしてみたい方へのスペシャルイベントを開催します。
今回は製薬業界に精通した講師が登壇し、製薬業界の現状や新薬開発の流れについてお教えいたします。また個別相談会では、Spring転職エージェントのコンサルタントが、あなたのライフプランに合わせたご提案をさせていただきます。
製薬業界への理解を深め、転職成功への第一歩を踏み出しましょう。
