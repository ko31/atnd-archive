description

iPhone開発者のお祭りに乗っかってみましょう

6月10日から、AppleのWWDCが開催されています。今年は仙台でも、このイベントに乗っかってみたいと思います。

そんなわけで、「WWDCキーノート鑑賞会 〜AppleのWWDCキーノートを見ながら、新製品や新技術について、なんだりかんだり語り合う会〜 powered by SWWDC」 を開催します。

いつもは技術系の話に特化した勉強会をしているのですが、今回は趣向を変えて、若干ライトに、新製品や新技術の事を学ぼうと思います。

ニュースを見て、iOS7やMacOSX MavericksやMacProやMacBook Airが気になった方は、ぜひお越しくださいませ。

内容

Appleのティム・クックCEOによるWWDC基調講演のビデオをプロジェクターで放映します。動画を見ながら、発表されたことについてあれやこれや雑談する感じです。

MacProやMacBook Airなどの新製品や、iOS7やMacOSX 10.9 Mavericksなどの発表がありました。それらを通じて、これからのハードウェア・ソフトウェアの将来像についてお話出来ればと思います。

タイムテーブル

17:00-19:00 WWDCの基調講演を鑑賞

　基調講演の動画を流しつつ、ツッコミを入れたり、新機能や新技術についてあーだこーだ話したりします。

19:00-20:00 雑談タイム

　WWDCのおさらいをしつつ、「発表内容を元にすると、今後のAppleは、アプリ開発は、どのような未来を描くのか？」など、今後の展望について語るなどします。

その他、いろいろ

  • 基調講演は全て英語なので、若干リスニング力が必要かもしれません。日本語字幕もないので。。。（事前に日本語サイトなどを見て、予習すれば大丈夫）
  • いつもはプログラマ・技術者向け勉強会ですが、今回は題材に新製品の話題も含まれますので、非技術者の方でも大丈夫だと思います
  • Apple好きな方も是非！

会場

  • 仙台市シルバーセンター 会議室 (宮城県仙台市青葉区花京院１丁目３番２号)
  • 会場使用料を参加者で割り勘にします(300円〜400円程度)
  • ネットワーク環境はありませんので、各自ご用意ください
  • 電源延長タップをご持参いただけると助かります

懇親会

終了後、希望者を募って、仙台駅前あたりの居酒屋で懇親会を実施します。目安は3000円程度だと思います。
