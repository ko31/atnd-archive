こんにちはヒギリと申します。

今、ポケモンカードが熱いのをご存知でしょうか。

昨年の4月ごろからゆっくりとポケモンカードを始め、勢いでイベントオーガナイザーの資格を取得し気が付けば1年経っていました。
せっかく資格があるのでイベントを開きたいと思います。今回は突発なので公認ではないです…（ポケカは公式から手厚いサポートグッズもあるので次回からちゃんと申請しようね( ﾟДﾟ)

7月は特に、GXスタートデッキの発売や限定プロモカードをかけたシールド戦など今から始める人が入りやすい環境となっております。なので、初心者から昔少し触ったことがあるなという経験者、もちろん上級者の方も楽しめるイベントをやりたいなと思います。年齢を問わず等しく楽しめるポケモンカードの魅力を少しでも共有できれば……と話が大きくなりましたが、純粋にポケカを楽しく一緒に遊べる仲間が欲しいです。
カードイベントはカードショップでの開催が多いですが、今回はバーでの開催ということで、アルコールやソフトドリンクを飲みながら楽しくカードゲームを嗜むことができるイベントとなっております。と言いつつも、かしこまる必要はないのです。対戦前の挨拶など、当たり前の節度ある振る舞いを心掛ければ、あとは双方楽しければOKです！

シールド戦については⇒（こちら)

「じゃあルールも覚えたから大会に殴りこみたい！」「今からシールド戦に参加したい！」「優勝してプロモカードをゲットしたい！」そんな方に向けた交流会イベントになります。
参加者の皆さんで本番と同様なシールド戦を行いながら拡張パックの内容の確認、デッキの意見交換、感想戦をに行い理解を深めていくことをメインとしています。
本イベントはポケカ初心者の方も大歓迎しております！٩( ‘ω’ )و　（当日は集まってる方々で現行レギュレーション等の自由な対戦も可能です）

イベント参加の確認事項

●シールド戦基本ルール
・デッキの枚数は、40枚ぴったり
・同じ名前のカードは4枚まで（基本エネルギーカードは枚数自由です）
・サイドカードは4枚
・たねポケモンを必ず1枚以上入れる
・対戦と対戦の間にデッキの中身を入れ替えてもOK
・大会中は当日に開封したカードのみ使用ください（基本エネルギーカードは問題ありません）

・試合形式
4人一組のスイスドロー形式（参加人数によって変更の可能性があります）
1試合30分で行います。
各グループのトップのみがミニトーナメントに進むことが出来ます。
ミニトーナメントの決勝戦時は2本先取で行います。

※各試合で30分経過した際に試合が終わっていない場合は
　サイドの枚数が違う⇒サイドが少ない方が勝ち
　サイドの枚数が同じ⇒じゃんけんで買ったほうが勝ち
