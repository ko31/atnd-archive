Perlをもっとカジュアルに！

Hachioji.pm に対するオマージュです。
最初に全員でLT（ライトニングトーク）をした後に、
ビギナーズセッションで初心者の疑問をみんなで解決したいです。
5, 6人しか集まらない場合は、ビギナーズセッションに時間を回します。

月1開催（第3水曜日）で、気軽に立ち寄れる感じを目指してます。

LT（ライトニングトーク）とは？

短い時間（今回は5分）の中で、スライドを用いて何かお話して頂きます。
ですが、負担になるようでしたら自己紹介だけでも構いません！

例えば？

  • あなたのPerl歴について
    • プログラミングしたことがないレベル？
    • Perlは使ったことがないレベル？
    • Perl初心者を脱したいレベル？
  • Hokkaido.pm Casualでやって欲しいこと
    • Perlでやってみたいこと
    • Perlで困ってること
    • Perl以外で困っていること（フロントエンド、サーバサイド等）
  • 他には？
    • Perlでこんなことできるよ！
    • Perl以外のネタもOK！
    • ビギナーズセッションで話したいこととか！

スライドのありなしは問いません。
カンペ代わりに1枚くらい用意すると良いかも知れません。
（LTの準備が負担になるようでしたら、自己紹介だけでもアリです！）

ビギナーズセッション

ビギナーズ向けに、学習方法（オススメのサイトとか）、デバッグ方法など、
つまずき易いところをトークして貰えるとうれしいです。
Perl CPANモジュールガイドブック からモジュールを紹介するとかも大歓迎！

  • 前回出た内容
    • Perlの5.16の紹介
    • Perlの勉強の仕方（サイト, 書籍, コード, … etc）
  • Per入門
    • 開発環境
    • 文法
    • ドキュメントの探し方
    • Perldocの書き方
    • テスト
    • PerlModuleの作り方
    • 正規表現
    • ワンライナー
    • Webアプリケーション入門
    • WAF入門
