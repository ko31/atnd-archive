description

能登･七尾には数々のまちづくりプロジェクトや企業の経営改革の取組があふれていますが、各活動の主体となるのは40代～ 60代であり、どのような取組も地域に若者が少ないということが恒常的な問題となっています。
七尾合宿で地域に若者が入りはじめましたが、練習場所と宿泊場所の往復のみで、地域との交流機会はほとんど無い状態であります。地域の元気づくりのためには、合宿からさらに一歩深めて、まちづくりの現場と意欲的な若者を結びつけ、さらに地域雇用や起業に結びつけるしくみの創出が課題であります。
そのためには、合宿地としての強みを活かし、学生との「縁」を地域のエンパワーメントにつなげ、さらに経済効果（円パワー）を生み出す人材（＝縁パワーメンター）の育成が必要であると考えます。
　今回、七尾合宿の推進と地域との交流機会の創出、および能登･七尾の企業やまちづくり団体への長期実践型インターン「能登留学」の受入によって、能登･七尾を“ 縁パワー”（縁によって力づけ）するという方向性を共有することを目的として、本フォーラムを開催します。
