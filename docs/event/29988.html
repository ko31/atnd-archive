NFCと仕事術。

イベント ユーザー
event_id 29988
title NFCと仕事術。
catch 〜GTDにおけるNFCの活用を考える〜
description

経緯

最近、NFCハッカソンとか勉強会とか活発に行われていて非常に楽しいのですが、
もっと極小規模で、テーマを絞り込んで深く考えてみる系の勉強会をできないものか、
と思ってATNDたててみた次第です。

ちなみに、6/16は八重洲のRFID/NFC Real Touch Shopが月に1度の土曜日営業をやってまして、
それに合わせてプチイベントを試行してみよう、というのもあります。

勉強会前後、希望される方がいらっしゃれば、Shopのデモツアーなんかもやっちゃいます。

概要

裾野の広さが魅力的なNFCのアプリ／サービス開発について、
決済系以外の何かを皆様いろいろと模索されていらっしゃる真っ最中かと思います。

今回は「仕事術」というテーマで、NFCの活用方法を考えてみたいと思います。

（本当は「仕事術」じゃなくて「ライフハック」のイメージなんですが
　広くなりすぎるので今回は「仕事術」にしてみました）

NFC対応のスマートフォンや、PCに接続されたNFCリーダー、
あるいは自作のNFCチェックイン端末などなど。

身の回りの様々なNFCデバイスを活用して、日常生活をもっと快適で便利なものにする方法や、
もっとカンタンに日常業務を片付けられる方法など、様々な活用方法について情報交換しませんか！？

「実践はできてないけど、こんなアイデアがある！」という方も是非！！

当日のSchedule（案）

14:30〜

　趣旨／方向性の確認

14:45〜16:15

　ネタ出し。

　「日常生活の中で、こんな風に便利にNFCを使ってるよ！」とか、
　「こんなアイデアを考えてきたよ！」というネタを各自発表。

　用意してきた人だけ発表でも良いかと。

16:15〜17:00

　ネタ提供を受けてのフリーディスカッション。

　そんなに綺麗にNFCとGTDが組み合わさるとも思えないので、
　NFCまで辿り着かないGTDネタ、GTDまで辿り着かないNFCネタ、
　どちらもOK。
