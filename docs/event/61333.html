来る2015年1月24日(土)、四ツ谷の「一凛」にて、
クスール新年会を開催します！

久々の開催！クスール受講生の皆さんや普段お世話になっている方々をお招きして開催する、クスールの新年会です。

同期や違う期の受講生、講師やクスール界隈の業界の方々とも交流できる絶好の機会です。

ご都合付く方、是非是非ご参加ください。ご友人を誘ってのご参加も大歓迎です！

クスール一同、皆様のお越しをお待ちしています！！

■開催概要

開催日時 : 2015/1/24(土) 19:00～22:00
開催場所 : 一凛 四ツ谷店(東京都千代田区六番町3-11 テシコ六番町ビル2F)
アクセス : JR四ツ谷駅 徒歩5分 / JR市ケ谷駅 徒歩5分 / 地下鉄麹町駅 5番出口 徒歩2分
