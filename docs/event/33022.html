今年も開催！東京Node学園祭2012

今年もNode.jsのユーザーカンファレンス「東京Node学園祭」が11月18日 (日) に開催されることを記念して、10月15日 (月) から11月11日 (日) まで参加者が日替わりでブログを書くイベントです。

みんなでブログに Node.js に関連する話題を書いて、イベントを盛り上げよう！

参加方法

  1. このイベントに参加します（先着28名まで）
  2. 先着順に期間中の各日、午前0時から午後12時までの24時間の間にブログを書きます
  3. 告知のため、ブログを書いたらここのコメント欄に URL をお知らせ下さい

参考: JavaScript Advent Calendar 2011
