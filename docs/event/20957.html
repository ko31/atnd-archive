開催概要

関西のJavaエンジニアのためのコミュニティ「関ジャバ」主催の
JavaOne2011報告会 を開催します。

10月の2～6日にサンフランシスコで開催された、
業界最大規模のカンファレンス JavaOne 2011。

そのJavaOneに参加したエンジニアが、技術・雰囲気・周辺観光などなど、
JavaOneに参加して得たもの、得られなかったもの（？）をフィードバックします。

なお、当日はJavaOneで手に入れたグッズのプレゼント企画もあります！
Javaエンジニアの方も、そうでない方も、ぜひ奮ってご参加ください。


また、セミナー終了後に、会場近くで懇親会を予定しています。

セッション概要

1. JavaOne 2011 サンフランシスコの最新レポートについて

日本オラクル株式会社
シニア Java エバンジェリスト 寺田 佳央 ( @yoshioterada )

2011年10月、JavaOne 2011 San Franciscoが開催されました。
本セッションではオラクルの立場からJavaOneで発表された新技術や
ホットトピック等をわかりやすく紹介します。
また、来年７年ぶりに開催予定のJavaOne 2012 Tokyoについて概要を紹介します。
本セッションではJavaの最新トレンドを掴んで頂くと共に、
本場JavaOneの盛り上がりを是非ご堪能ください。

2. 初めてのJavaOne ～ところで俺のiPhoneどこいった～

岡山Javaユーザ会
吉田 貴文 ( @zephiransas )

不幸なアクシデント（？）に見舞われながらも、
初めてのJavaOneに参加した自分が感じたこと、出会った方々、
また刺激を受けたことなどを、まったり、ゆる～く報告します。

3. 日本一「細かい」JavaOne報告！

関西Javaエンジニアの会 / Acroquest Technology株式会社
谷本 心 ( @cero_t )

大きな話は皆さんに任せて、私は細かい話をしましょうか (^^
コレクション、GC、メモリ使用量、HotRockitで使えるようになる機能、
誰もフィードバックしてくれない、そんな小さな話題をしっかり拾って話します！

4. パネルディスカッション

JavaOneに参加したメンバーがパネルディスカッション形式で
今年のJavaOneを振り返りや、技術への突っ込み、来年への展望を話し合います。
もちろん、観客の皆さんからの意見や質問など、Joinを期待しています。
