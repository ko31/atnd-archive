>What is “noreturn”?
それはタフでキュート。ベイシック・アンダーグラウンド。 ジャズ・ロッキン・ブラジリアン・ジャポネーゼ。 渋谷系の正当な子孫を主張する30歳±5歳の大人が、 レコードを一晩中かけて酒を呑んで踊る音楽のパーティー。
1999年、立教大学「クラブDJ」サークル出身のメンバーを中心に結成。サラリーマンになったので、オフィス用語で「直帰」を意味するNO RETURNと命名。その後、就職先での同僚や友人など、同じ嗜好を持つメンバーが増殖。「ベーシックでアンダーグラウンド、ジャズでロッキンでブラジリアンでジャポネーゼ！」と掲げたテーマの裏には「渋谷系最後の生き残り」という隠れキーワード。
渋谷を中心にイヴェントを開催しています。
