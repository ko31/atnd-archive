SEEDS COMPANYでは今春リリース(予定)の新規サービスを控えております。

　- frontend
　　- react/redux
　　- node.js
　- backend
　　- golang/echo
　- database
　　- aws aurora
　　- couchbase
　- infra
　　- aws
　　- docker
　　- terraform
　- etc
　　- concourse ci
　　- gatling

【詳細】
- なぜその技術を選んだか、技術選定プロセス
- どのように使っているか
- 苦労した点、ハマりどころ

★何を学べるか★
・AWSを活用したサービス運営
