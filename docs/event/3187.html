はじめまして。松本と申します。

総合プロダクション　PURE MUSIC PROJECT CORPORATIONでは、スクール生を募集しております。
当事務所の代表は、松本謙二（社団法人・日本作曲家協会員）になります。
社団法人　日本作曲家協会とは（ＴＢＳ放送・日本レコード大賞制定委員会）の団体です 。
プロを目指されている方、歌の実力を付けたい方、曲作りの指導を受けたい方、などの指導に力を入れております。
頑張る人達を完全バックアップして行く会社方針ですので、将来メディア系を目指されている方にとってチャンスが多いと思います。

将来性のある人材やバンドは、バックアップしていく体制です。
優秀なバンドは、輩出していく方針です。

作曲家の指導についてですが、
プロのシンガーソングライター、または作曲家として、

対応できる音大卒業レベルまでを取得できます。
編曲家としても、対応できる知識を身につける事ができます。

指導を受けて実力が認められた場合、プロの作曲家として活動ができます。

アーティストの楽曲提供や、ＣＭ、音楽学校の講師やさまざまです。

当事務所のレッスン場所は、近鉄布施駅から、徒歩　約５分のスタジオになります。

スタジオの行き方は、私が直接、御案内させて頂きますので、

ご質問等ありましたらお気軽にお問い合わせ下さいませ。

レッスンの見学も受け付けております！！
チラシや、ライブの写真もご用意しております。
ご興味のある方は、是非ふるってご応募ください。
ご応募を心よりお待ち申し上げております。
