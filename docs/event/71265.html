 ■結婚力アップ集中プログラム


理想のバートナーと出会い、幸せな結婚生活を続ける素晴らしい方法があります。

結婚力を上げると
未婚者なら、理想のパートナーと出会い、結婚できます。
既婚者なら、幸せな結婚生活を謳歌できます。

このプログラムを受けて多くの方が、幸せな結婚生活を続けています。
私たち夫婦も、出会ってから一度も喧嘩したことがありません。

今回のセミナーでは、この「結婚力アップ集中プログラム」の中の一部を紹介します。

■あなたも幸せな結婚生活を送れる特別な瞑想法


ソニーで研究してきた瞑想法は、初めての方でも、上手に瞑想状態に入れます。

すると、雑念が少なくなり、ひらめき力や洞察力に富んだ意識状態になります。
これを「内なる叡智」に繋がった状態といいます。

この意識状態になると、幸せな結婚生活を妨げていた、様々な原因が分かります。
それが、心の傷です。

さらに、それらの心の傷を癒していけるので、幸せな結婚生活が送れるようになります。

■セミナー参加特典


心の傷を癒し、幸せな結婚生活を送れるようにする瞑想の誘導音声を聞けるメンバーサイトに一か月間無料ご招待。

 ■天国言葉


銀座まるかんの創始者の斉藤ひとりさんの「天国言葉」があります。
・愛してます　　・ついてる　　・うれしい　　　・楽しい
・感謝してます　・しあわせ　　・ありがとう　　・許します

愛してます　から、　許します　まで、何度もつぶやいていると、不安が解消し、良い気分になります。

■心の傷


Dr.マルツという、敏腕整形外科医がいました。
彼の腕にかかり美貌を手に入れても、何度手術を重ねても、幸せ感を感じられない人がいました。
Dr.マルツが、原因を調べていったところ、それらの人たちは、心の傷という不安を抱えていたのです。

マイケル・ジャクソンの一生を考えると誰もが納得することでしょう。

■頑張るより、心の傷を癒す


誰にでも、心の傷はあります。
そこで、上記の天国言葉を唱えたときに、心がスーッとしたのです。

男性は、出世して、ワンランク上の自分になったら、幸せな結婚ができると頑張ります。
女性は、お化粧やダイエットをし、容姿が美しくなったら幸せな結婚ができると頑張ります。

しかし、頑張り続けたところで、心の傷が解消できるわけではないので、やがて疲弊してしまいます。

○○したらと頑張るより、心の傷を癒した方が着実に幸せになれるのです。

Dr.マルツも、サイコ・サイバネティクスという理論を打ち立て、多くの人の頃の傷を癒して、幸せ感を感じやすくしていき、有名になりました。

■心の傷を癒し幸せな結婚をする最良の方法


それが、本セミナーで紹介する「瞑想で内なる叡智につながる」方法です。

瞑想により心を落ち着けていくと、まるでコップに入れた泥水の泥が沈殿して、水が澄んでいくように、心が澄んでいきます。

すると、もやもやして見えなかった心の傷がはっきりしてきます。

通常の意識状態で、心の傷がわかっても、苦しみが増すだけです。
しかし、瞑想の状態になると、ひらめき力や洞察力も増していきます。

すると、なぜ心の傷ができたか、
どのようにすれば、解消できるかの名案も、浮かんできます。

さらに、瞑想のテクニックを駆使して、ひらめき力と洞察力が高まった意識状態を「内なる叡智」として擬人化してあげると、さらにすごいことが起きます。

様々な相談に乗ってくれ、ベストアンサーを教えてくれるメンターとなってくれるのです。

このメンターに何でも相談していけば、どんどん心の傷を癒し、幸せな結婚ができるようになります。
