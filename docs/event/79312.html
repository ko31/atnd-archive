はじめまして、今回のセミナーを担当させていただきます。松本です。

プログラミングを学びたい!と思っていてもどうしたらよいかわからない・・・と途方にくれてしまうのが現状ではないでしょうか。

私自身現在、WEBプログラマとして仕事をしておりますが、元々は学習塾で仕事をしており、
独学でWEBプログラミングの勉強を始めました。

プログラムの技術を見につけるのは正直簡単ではありません。 独学で進めようと思っても

  • そもそも何をしたらいいかわからない
  • いろんな情報がでてくるが、どれが正解かわからない。
  • プログラムに興味はあるが今一歩が踏み出せない
  • 身の回りにきける人がおらず、1つわからないことが出てくるともうすすめない

といった壁があり、なかなか独学で勉強を進めるのは困難です。

私自身、過去そういった壁にぶつかりましたが、講習会に参加し、講師に質問することで自分の疑問が一気に解決でき、
簡単なショッピングシステムを作ることができました。

システムを作る段階で知識を身につけ、今ではWEBエンジニアとしてお仕事をさせていただいております。

どの受講生の方も過去私がたどってきたような悩みを抱えております。
書籍や動画サイトなども数多くありますが、プログラミングの独学は非常に壁が高く、多くの方が挫折してしまうのが現状です。

プログラミングをやりたい、という方は多くても、専門的な授業か就職以外で身につけることができません。
そういった方の「０からプログラムを覚えたい」
というお手伝いができればと思っており、このたびセミナーを開催させていただきました。

このセミナーでは私が説明するだけではなく、WEB系で使われている「PHP」というプログラミング言語を使い、
初歩的なプログラミングを実際に体験していただきます。

超少人数(最大4名)で行いますので、セミナーに参加したけれどわからずに放置されたしまった・・・
といったこともありません。

「プログラミングを体験してみたい」という方の架け橋になれればと思います。

講座内容

第1章 プログラミングを学ぶには(WEBプログラミングの概要説明)

  • どのプログラミング言語がいいのか？
  • PHPがおすすめな理由は?
  • プログラミングが難しい理由は？
  • どうやって学習したらいいのか？

第2章 実際のコーディングを体験してみよう

  • XAMPPをインストールしてみよう
  • PHPを実際に書いてみよう
  • 変数、配列といったプログラミングの基本に触れよう
  • 基本的なプログラミングを書いてみよう

対象者

  • HTML/CSSは何となく触ったことがあるけれどプログラミングはよくわからない
  • 以前、プログラミングを学ぼうと思ったが挫折してしまった

※基本的にはプログラミング未経験者の方を想定しています。

講座で得られること

  • プログラミングでできることやPHPを選ぶメリットがわかる
  • WEBプログラミングを学習する上での便利な道具の存在をしることができる
  • 簡単なPHPのプログラミングを体験することができる

当日のスケジュール

  • 14:00～15:00 WEBプログラミングの概要説明
  • 15:00～17:00 PHPを使ったプログラミングの実務演習
