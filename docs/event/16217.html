内容　

Android初心者向けのハンズオン勉強会

時間

１１：００～１６：３０

初心者の方を対象にしています。

講義パート（午前）


  1. ソフトウェアの仕組み

  2. プログラミングとは？

  3. Androidとは？

実践パート（午後）


  1. ４人１組でチームを作ります。

  2. Eclipseのセットアップ、ADTのセットアップを行う

  3. エミュレーターを作成する

  4. サンプルプログラムを読み込み実行する
    （あらかじめ準備しておいたプログラムを改造します。
    例) テキスト連携するアプリ、電話をかけるアプリ、電卓アプリ、ホームページを表示するアプリなど）

  5. サンプルプログラムの解説

  6. サンプルプログラムを修正し実行する。（お題）

  7. 成果発表
