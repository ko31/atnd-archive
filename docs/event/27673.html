Kyoto.lisp

Kyoto.lispは京都近郊のLisp方言コミュニティです。Common Lisp、Scheme、Arc、Clojure、Emacs Lispなどの言語の話題を扱います。

今回のイベント「Tech Talk」は、参加者の方々がプレゼンを持ち寄り、交代で前に立って発表するスタイルのイベントです。

発表内容は、Lispに関連することであれば何でも構いません。普段どのようにLispを使っているかとか、こういうソフトウェアを作ったなど、自由に発表してください。

発表枠について

発表時間によって2種類の枠を設けます。

1. Tech Talk (15分)

Tech Talkで30分とか1時間とか言われても話す人いなさそうなので 15分 の枠に限定します。 4〜6人程度 を想定しています。

2. Lightning Talk (5分)

5分枠 で質問なしで適当に話す場です。 5人程度 を想定しています。

発表するには

Kyoto.lisp Tech Talkで発表される方は、以下の手順で登録をお願いします。

1. このイベントに参加登録 してください。その際、参加時アンケートでTT(15分枠)かLT(5分枠)のいずれかを選択していただけると助かります
