description

「マイナンバー、いったい何をするの？」
「マイナンバー制度の案内だけで、実務のセミナーは？」
「規程に盛り込みたいが、手掛かりがない。」

ご安心ください。
参加者の知りたいことをお話しします。

開催時点での入手可能な情報からお話しします。

１時間ですので、具体的な事項（実際の番号収集の方法、就業規則等への規定の変更・条文例、特定個人情報取扱規程の作り方、安全管理措置の具体例、社内研修の方法、従業員の扶養家族のマイナンバー、など）について、お話しします。

「罰則」「今後のマイナンバー」などは含みません。

対象者：
経営者様・社長様、担当の役員様、人事労務総務の管理職様、士業の方（同業者の社会保険労務士もご遠慮なく）

会場：
アワザ・ビジネス・カンファレンス（≫アクセス）
大阪市西区西本町２－４－１０　浪華ビル２階
（大阪社労士事務所のヨコの部屋です）
（最寄り駅：地下鉄阿波座駅１号出口東へ徒歩２分、
　本町駅西へ徒歩７分～、京阪中之島駅南へ徒歩１２分）
※阿波座駅からホテルルートイン大阪本町を目印に、横断歩道あたりを１本裏の道の方へ見ると、右手に１階が飲食店の雑居ビルがあります。その２階が目的地です。

時間：約１時間、その後個別相談が可能
個別相談だけの方も、是非お申し込みください。当日の参加者の状況により、時間の無駄の無いよう臨機応変に対応します。

マイナンバーセミナーへの参加方法
参加費用：
参加は、１名様に付き1000円です。
１度ご参加いただいた場合、次月以降の参加費はいただきません。
なお、顧問先様・関与先様・SR-CLUBメンバーは無料でご参加いただけます。
