description

聖蹟桜ヶ丘にあるカフェ『HUGSY』の8畳の空間で、世界の文化や風習を体験してみるワークショップ型のイベント『8畳ジャーニー』。
今回のテーマは「インド」×「色彩」です。

もうすぐ春ですね。
木々も まちも ファッションも 気分も
色とりどりに染まる春。

世界でも春を祝うお祭りが開催されます。
その一つがインドの『ホーリー』。
誰彼構わず色水を掛け合って春の訪れを祝う
豊作祈願、悪魔払いの意味が込められたお祭りです。

今回は、
・『ホーリー』に実際に参加されたHUGSYまつかわさんの写真付きトーク
・心の赴くままに塗り絵タイム
・インドの彩り豊かなごはんを盛りつけて、祝祭ランチ
の3部構成。

「インド」×「色彩」というテーマを
お祭り・塗り絵・ごはんという切り口から体験し、
春の訪れを楽しみましょう！
