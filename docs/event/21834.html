概要

「enchant.js meetup! OSAKA」とは「enchant.js でゲーム作りをはじめたいけど、どうしたらいいか分からない」「enchant.js でゲーム作りのスキルを磨きたい！」「enchant PRO ってどんなもの?」という皆さまのためのイベントです。

「enchant.js」とは、PC, Mac, iPhone, iPad, Android すべてで動作するアプリやゲームを開発できる、 日本で開発されたHTML5/JavaScriptベースのゲームエンジンです。リリースから半年で400本近くのゲームが製作されています。

「enchant.js」を使えばゲームをはじめとした高度なプログラムに必要な要素をスマート・簡単に記述できるため、初心者のプログラミング入門にも、すでに習熟している方の本格的なゲーム開発にもぴったりです。初心者の方から、すでにゲームをいくつも作ったことがある方まで、 enchant.js に興味のある方ならどなたでもお楽しみいただけます。お気軽にお越しください！

enchant.js について、詳しくはこちらのリンクから。

  • 公式サイト http://enchantjs.com/
  • Developers Blog http://blog.enchantjs.com/
  • github https://github.com/wise9/enchant.js

FAQ

  • 何をするイベントなの?

  • 参加費用は?
    無料です。
  • 電源や回線はあるの?
    申し訳ありませんが、参加者の電源・回線の保証はできません。電源タップはできる限り準備しますが、バッテリーをフル充電してきてください。また、各自Wi-Fiルータなどをお持ちください。
  • enchant.js や JavaScriptの開発経験がなくても大丈夫?
