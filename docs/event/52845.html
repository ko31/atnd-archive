description

イメージしたデザインを「tacスケッチ」で描いてみませんか。
照明計画や壁の造作・配置バランス・大きさなど確認調整することで、
より洗練された室内空間の創造が可能です。
お客様や施工業者の方にもイメージを伝えることができます。

簡単なスケッチを覚えてしまえば
ワイドなスケッチも基本は同じですので徐々に描けるようになります。
講習時間は３時間ですが描き方の基本を凝縮してお伝えいたします。
あとはご自宅などで自由な時間に講習レジュメを参考に練習してください。

この講習会を出発点としてワイドなイメージスケッチを描けるようになり、
お客様に素敵なインテリアデザインをご提案して頂ければ幸いです。

「tacスケッチ」講習内容
1.線の引き方①(25□・NS法・1,2タッチ)
2.線の引き方②(立方体・NSTD法・洗面台)
3.スケッチ法(基本・応用)
4.陰影・照明テクニック・カラーリング
※製図用0.5シャープペンシル(B芯)ポケットテッシュ・消しゴムをご持参ください。

対象者:インテリアデザイナー、新築・リフォームアドバイザーの皆様
