★「日清公式！チキンラーメンを粉から作る体験講座」★

　ワル匠プレゼンツ。スマート○○シリーズたぶん第5弾、スマートフォンランチシリーズ第2弾。
　今回の舞台は横浜・桜木町。
　9/17にオープンしたカップヌードルミュージアムにて、「チキンラーメンを粉から作ってみよう！」という、体験型のイベントです。
　スマートフォンをお持ちの方もそうでない方も、横浜の方もそうでない方も、男性も女性も、大人も子供も、普段お湯を入れて食べられるだけのチキンラーメンを「粉から」作ってみませんか？
　イベントの後は、オリジナルのカップヌードルを作ってみたり(肉肉肉肉だけも可能です！)、横浜市内でおいしいものを食べにいくというイベントもありますよ。

　■日時
　　2011年10月8日(土) 14:00～

　■場所
　　カップヌードルミュージアム(横浜市中区新港2-3-4)
　　http://www.cupnoodles-museum.jp/

　■集合場所・集合時間
　　集合場所：JR桜木町駅改札前キヨスク周辺
　　集合時間：13:00(時間厳守)

　■募集人数
　　11人(人数のみ予約済)

　■参加費用(交通費・諸経費・その後の食事会費は除く)
　　大人500円/小学生300円

　■交通アクセス
　　・京浜東北根岸線・桜木町駅下車。
　　・横浜市営地下鉄ブルーライン・桜木町駅でも可。

　■二次会について ←New!!
　　以下の箇所でちょっとした食事会を行う予定です。Wifi完備です。
　　・ナチュラルレリーフ / パセラリゾーツ横浜店
　　　http://www.pasela.co.jp/shop/yokohama/natural_relief/

　■注意事項(超重要)
　　・開催場所及び開催時間と、集合場所及び集合時間が異なります。事前に受付をする必要がありますので、集合時間厳守でお願い致します。
　　・募集人数は11名ですが、できるだけ当日のキャンセルはご遠慮頂きますようお願い致します。
　　・エプロンやバンダナは会場で用意しておりますので、動きやすい服装で参加頂ければ幸いです。
