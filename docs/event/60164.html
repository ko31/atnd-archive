イベント・講演概要

超汎用2Dアニメーションツール 『OPTPiX SpriteStudio』 勉強会！

毎月実施している初心者向け勉強会の開催と合算して、記念すべき第10回！　となる今回は、
なんと 国民的RPG と ニンテンドー3DS と UnrealEngine4 ！

１）アルテピアッツァ株式会社さんによる、 『ドラゴンクエスト』シリーズ の
　　アニメーション製作事例のご紹介

２）株式会社メビウスさんによる和風ハック＆スラッシュ 『異史戦国伝～宿業』
　　SpriteStudioをベースとしたプロジェクトの運用事例のご紹介
　　キャラクターアニメやエフェクトはもちろん、「それ以外」に幅広く！

３）株式会社ヒストリアさん、UnrealEngine4プラグイン開発者さんたちによる
　　 『SS5Player for UnrealEngine4』 のご紹介

全セッション終了後、軽食をご用意しての懇親会＆名刺交換会を実施いたします。

参加対象

“OPTPiX SpriteStudio”既存のユーザーの皆さんはもちろん、これから SpriteStudio を
触ってみようという方。また、SpriteStudioのデータを利用した内製ツールの話題のほか、
話題の UnrealEngine4 で、2Dゲームを作ってみよう！　と野望に燃えている
皆さんの参加をお待ちしております。若干デザイナーさん向けの
セッションではありますが、組込を担当される方のご参加ももちろん是非です！
