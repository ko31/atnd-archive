クリーン化技術による微小ゴミ不良軽減のポイント

イベント ユーザー
event_id 14281
title クリーン化技術による微小ゴミ不良軽減のポイント
catch ゴミの「見える化」ツールの活用法、安価で効果的な導入方法
description

【受講対象】
　エレクトロニクス、塗装、フィルム加工、食品などのクリーンルーム関係者

【習得できる知識】
クリーン化、クリーンルームの維持管理、クリーン化による良品率の向上、ゴミ・ホコリ対策、クリーンルームの入退室、クリーンルームの作業員教育、毛髪混入防止、エアシャワーの効果、クリーンルームのダストの可視化、クリーンルーム規格

【講座のポイント】
　半導体をはじめとする各種デバイスやエレクトロニクス、機能性フィルムの製造及び加工、薄膜塗装や食品業界に至るまで微小な異物による不良対策としてクリーンルーム技術を導入するケースが近年急激に増加している。その現場で効果を発揮しているのがクリーンルームライトをはじめとするクリーンルームの「見える化」ツールの各種である。
　本講では、これらを利用したクリーン化対策の基本と事例を挙げ、安価で効果的なクリーン化技術の導入方法について解説する。
※デモ機による実演を行います。

【プログラム】
１．クリーン化技術とゴミの「見える化」
　　1-1．クリーン化技術とそのニーズ
　　1-2．クリーンルームの定義と規格
　　1-3．良品率とクリーン化対策の相関　
　　1-4．微粒子だけでなく粗大粒子の対策も重要
　　1-5. ゴミ・ホコリの正体と発生源
　　1-6. クリーンルームの４原則
２．「見える化」ツールの紹介（デモ・実演付）
　　2-1．クリーンルームのゴミの見える化ツールとは？
　　2-2．ポラリオン　クリーンルーム　ライト
　　2-3．落下塵カウンター（表面に付着した粗粒子をカウント）
　　2-4．ミストストリーム（気流可視化装置）
３．クリーン機器の効果と問題点
　　3-1．エアシャワーの効果を検証する
（除去率、エアシャワーが苦手なもの、内部クリーンナップ）
　　3-2．エアシャワーで落としたゴミはどこへ行く
　　3-3．エアシャワーと静電気
　　3-4.　クリーンブースの効果と問題点
４．クリーン化事例の紹介　
　　4-1． プラスチック塗装（静電気対策と局所クリーン化の応用）
　　4-2．半導体製造装置アセンブリ（安価なクリーンルーム設置）
　　4-3．フィルム加工　（各種加工工程におけるクリーン化）
　　4-4．露光装置でのクリーン化対策（装置からの発塵対策、付着塵対策）
５．クリーン化技術の導入と活用　
　　5-1．組織ぐるみのクリーン化対策を進めよう
　　5-2．静電気対策とクリーン化の検証
　　5-3．作業員教育の重要性、継続できることが重要
　　5-4．まとめ
