【学生向け】HTMLとJavaScriptで作るAndroid・iPhoneアプリ開発勉強会

HTML,CSS,JavaScriptを知っていて、これからAndroidアプリや
iPhoneアプリを開発したいと思っている学生の方に向けた
Android・iPhoneアプリ開発のハンズオン勉強会です。

PhoneGapというフレームワークを使用し、1つのソースで
Android・iPhoneの両端末で動くアプリを開発します。

開発環境はAndroid,iPhoneで異なるため、
実際にはどちらの端末に対応したアプリを作成するか選択していただき、
選んだ端末に合わせた開発環境で作成を行います。

対象はHTML,CSS,JavaScriptを知っている方ですので、
JavaやObjective-Cといった言語の経験の無い方でも大歓迎です。

一般的な勉強会は敷居が高くて参加し辛いという方、
普段プログラミングの出来る学生が周りに居ない方、
現役のプログラマと話してみたい方など、色々な方の助けに
なれればと思っていますので、興味のある人はぜひ参加して頂ければと思います。

*学生向けではありますが、社会人の方でも参加可能です。
