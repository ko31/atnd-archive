description

［講座のポイント］
　プラスチック製品開発の進歩は眼を見張るものがあり、今ではシンデレラのガラスの靴も注型法で足に合わせてプラスチックで作ることが可能です。わが国の世界的材料メーカー16社を取材したプラスチックの最新情報を紹介しながら、私の近刊「プラスチックが一番わかる」をテキストに、入門しましょう！

［受講対象者・レベル］
　◇新製品開発に携わる一般技術者

　◇プラスチック材料を扱う企画・開発・応用・営業に携わる人

　◇プラスチック製品のクレーム処理に携わる人人など

［プログラム］
１．どんなプラスチック製品が開発されたか？
　　1.1　プラスチック製品にはどんなものがあるか？
　　1.2　どんな形体のプラスチック製品があるか？
　　1.3　どんなプラスチック製品が開発されたか？
２．プラスチックとは何か？種類別の特徴は？
　　2.1　プラスチックとは何か？
　　2.2　プラスチックの誕生と歴史
　　2.3　プラスチックの製品量はどのくらいか？
　　2.4　プラスチックには、どんな種類と特徴があるか？
３．プラスチックの基本的な性質は？
　　3.1　高分子材料の性質は何で決まるか？
　　3.2　プラスチックの利点と欠点
　　3.3　プラスチックの分類
　　3.4　ポリマーブレンドとポリマーアロイ
４．プラスチックの選び方と活用法
　　4.1　プラスチックを活かす設計法
　　4.2　プラスチックの活用法の例
　　4.3　プラスチックの材種の選び方
　　4.4　プラスチック化はコスト上有利か？
　　4.5　プラスチックの信頼性と品質管理
５．プラスチックの製品をいかに成形するか？
　　5.1　成膜法と成形法
　　5.2　一時成形法
　　5.3　二時成形法
　　5.4　その他の成形法
　　5.5　押出成形法・射出成形法の標準成形条件
　　5.6　成形不良とその解決方法
６．プラスチックのリサイクル
　　6.1　廃プラスチックの処理
　　6.2　成形工場内でのリサイクルと産業廃棄物のリサイクル
　　6.3　リサイクルの最新技術
　　6.4　LCA（ライフリサイクルアセスメント）
　　6.5　プラスチックのリサイクル研究
７．プラスチック製品の夢
　　7.1　夢の新材料
　　7.2　夢の新用途
　　7.3　電気自動車（EV）とハイブリッドカー（HV）の夢
　　7.4　リニアモーターカー（超電導磁気浮上鉄道）の夢
