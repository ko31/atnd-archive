description

webnowはほぼ毎月開催されている非営利の無料web勉強会です。

IT/Web系を中心とした(でも比較的テーマの自由な)、アンカンファレンス形式のカンファレンスを行います

やわいライトニングトーク形式で、参加者が1つずつお題を持ち寄って、
5分から10分ほど前に出て話しをします。
話題は「最近仕事中に見つけたワンポイント」といった感じで
IT/Web系を中心とした(でも比較的テーマの自由な)プレゼン型 勉強会です。。

[ライトニングトークとは？]

ライトニングトーク（Lightning Talk）とはカンファレンスやフォーラムなどで行われる短いプレゼンテーションのことです。もともと「ショートトーク」と呼ばれていたそうですが、10年ほど前から「ライトニング（Lightning：稲妻、電光石火）」という言葉が使えるようになったようです。

webnowでは、5分から10分を持ち時間として
用意されたネタを話ししてもらいます。

発表を終えた後に質疑応答とか議論を長めに取っていただいても全然構いません。

人前に出て話しをする、プレゼンの練習にもなります。

マルチメディアシアターには、MacとWindows機、ものすごく立派なプロジェクターに
各自の机にディスプレイ完備です。
発表される資料はUSBメモリーにPDFなどでご用意されるとスムーズにプレゼンできると思いますが
部屋に設置されている機械はネットにも繋げますので、ブラウザなども使えます。
(持ち込みされたノートパソコンなどはセンターの無線LANには入れません。)

また、「話すネタはないけど、参加して聞くだけ聞いてみたい」という方も大歓迎です。

時間が余った場合はアンカンファレンス形式に、二回目、三回目の発表をするのも有りです。

今まで発表された話題の例をいくつか


  • 「ＨＴＭＬ品質チェック」に活用しているツールの紹介
  • コーディングに便利なfirefoxのアドオン
  • RIAって何？何ができるの？
  • 最近のネットショップ業界
  • Twitterのフォロー数を増やす方法
  • サイトスピードを向上しましょう
  • sshのすすめ
  • ペルソナの使用例

などなど

[参加資格]

IT, Web業界で働いてたり、日曜プログラミングしたり、ブログやtwitterを書いてたり、何かを勉強していたり、その他、人がやってる知らない話を聞くのが好きな人

今まで参加された方の職業をいくつか


  • 個人で開業されているwebデザイナー
  • web製作会社の社員
  • 求職中のかた
  • ネットショップ店長
  • webプログラマー
  • webコンサルタント
  • iPhoneアプリ制作者
  • 社内web担当
  • IT関係の専門学校講師

などなど

[よくある質問]

発表は必須ですか?
絶対発表ということはないです。でも会場で「自分もやれそう」と思ったときのために一つぐらい用意しておくといいかもしれません。
途中参加でもいいですか? 途中で帰ってもいいですか?
一つ話しを聞くだけの参加でも大丈夫です。週末なので他に用事のある方もいるでしょうし、遠方から参加されている方もいるでしょう。そのあたりを厳しくするような催しではないので安心してください。
家族や子供連れでもいいですか?
ご興味がある方は、誰でもウェルカムです。ただ参加人数をある程度把握できるとありがたいので、ご家族の方にもそれぞれ参加登録していただければと思います。
学生・在校生でも参加できますか?
もちろんです！
富山以外から参加してもいいですか?
もちろんです! 帰省のついでや観光のついでなどにぜひ。
発表者が途中で居なくなったら
残念ですが、発表が尽きたら予定よりも早く終了・解散になります。
2人や3人で発表をしても良いですか？
その方が発表しやすいようでしたら、全然問題ありません。
駐車場はありますか? 使えますか?
大丈夫です。建物の奥に広い駐車場があります。
ネットワークは繋がりますか？
無線LANがありますが、ITセンターのパソコンしかネットワークには入れませんので、ご自分のパソコンを持ってきていただいてもネットには繋げません。
二次会はあるの?
終了後ファミレスで雑談することが多いです。毎回特に予定は立てておりません。希望があれば、コメント欄にご記入下さい。
