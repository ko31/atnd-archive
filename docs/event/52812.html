title 社会の無関心を打破するということ
catch リディラバ安部さん　×　GrowAsPeople角間
description

社会には様々な社会課題が存在しています。
わかりやすい課題、課題かどうか見えにくい課題。

カテゴリー分けすると環境、子ども、男女共同参画、、、様々です。
しかし実は最も根深い問題って、

『課題に対する無関心』

ではないのでしょうか？

社会の無関心を打破するをミッションとして
先進的な活動しているリディラバ代表の安倍さんをお招きし、

社会的な関心はすごく高いけれども、
かなり無関心で要られている国内の夜の世界の課題解決に取り組む
GrowAsPeople角間がゆるゆると対談します。
無関心を打破することの必要性やその方法について考えてみませんか？

普段全国で活動している２名ですが
今回は越谷の地で話すってことであえて少人数の講座です。

対談を通して、
課題全体で見た時に、関心度が相対的に低い『男女共同参画』
現状を打破するヒントを探してもらえればと思います。

※安倍さんと角間が共催した企画が記事になりました

「風俗嬢と話す女子会」　越谷の社団法人が企画

【講　　師】
・安部敏樹（あべ・としき）
1987年生まれ。現在、東京大学博士 課程で脳と社会論のインタラクションの 研究に取り組む。「社会問題のプラットフォーム」リディラバを立ち上げ。「社会の無関心の打破」をミッションに掲げ、「世の中の個別の問題を解決する以上に、社会の人々の意識を高めることが根本的解決に近づく」という考えのもと、多岐分野において社会的に十分に認知されていない問題を取り上げたスタディツアーを展開。
Twitter : @toshikiabe

【当日運営】
・角間惇一郎（かくま・じゅんいちろう）
1983年生まれ。GrowAsPeople代表。建築、空間デザイナー。
性風俗等の夜の世界で働く女性たちが抱えがちな“立場を知られる事を恐れ、困ったときに相談や支援を受けられない”という課題の存在を知り、2010年からサポートと課題の顕在化を目指し活動を開始。現在はデザイン的手法による解決に取り組む。越谷市男女共同参画推進委員。
Twitter : @kakumaro
