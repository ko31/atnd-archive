catch ウェブ周りの各分野に関して触り部分の説明をお互いにしあう会
description

■会費：300円(※会場費を人数で割っています、参加人数によって下がります)

■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、

　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。

■現在予定している内容：
　

●セッション1(40分)
　『Windows8.1アップグレードのすゝめ！』神守さん

　＜主な内容＞
　2013年10月17日20時　ついにWindows8.1がリリースされました！
　本日はWindows8.1と8と7を比較した結果、
　8.1アップグレードのおすすめポイントをお話しします。
　おまけ、ノンプログラムでちょっぴりWindows8.1アプリを作る
　チップスご紹介させていただきます。（時間があれば…）

　＜神守さんの紹介＞
　2011年に約30個たった２人でWindows phone7のアプリを
　死に物狂いで作ったことをきっかけにWindowsPhone7、Windows8などの
　アプリを作ってます。
　コミュニティでWindow系のアプリを作ってます。

　Windows女子部広島支部長

　WindowsPhone HandsOn in 広島 ヒーロー島（時々顔だします）
　Windows phone Hands on in Hiroshima

　

●セッション2(40分)
　『自信のなさが武器になる？仕事を継続していくモチベーション』長嶋さん

　＜主な内容＞
　会社の中で少数でしかないWeb制作の立場での
　将来への不安からしてきたいままでのこと。
　現状への不満が発端となって今ある人とのつながりや
　方向性を定めるきっかけができてきたなと、振り返ると思えます。
　しがないWeb屋ではありますが、そんな小さな経験の話を中心に、
　失敗したこと・していてよかったなど、
　今きをつけていることなども含めて話せたらと思います。
　＋先日KDDIWebコミュニケーションズ新サービス発表会に
　出席させていただたときに出会った、ステキなサービスの話。

　＜長嶋さんの紹介＞
　Augment8イエローリベロ。
　昼間は気ままな会社員として、
　イベントページ制作や、SNS運営などに携わる。
　CSS Nite in HIROSHIMA実行委員として立ち上げから関わっています。
　Jimdoエバンジェリスト。
　幾度かさまざまな場所での登壇機会をいただいておりますが、
　伝えることのむつかしさを感じています。お手柔らかに・・・
