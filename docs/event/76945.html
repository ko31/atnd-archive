セミナー概要

2015年11月にリリースした次世代クラウド「IIJ GIOインフラストラクチャーP2」（以下、IIJ GIO P2）のハンズオンセミナーを開催いたします。
今回は、仮想サーバを中心としたパブリッククラウドのサービスについて、実際に仮想サーバの契約から起動、設定変更などハンズオン体験していただきます。
