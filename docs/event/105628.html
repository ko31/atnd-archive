～自分の体重を利用して、しなやかな筋肉を作る～

TRXトレーニングは、吊り輪状のハンドルに手や足を引っかけ、不安定な状態をつくり、自らの体重を利用して様々な姿勢を取ることにより、身体の内部の筋肉（体幹）が強化され、通常のトレーニングより効率的に全身を鍛えることができます。

それにより、普段使っていない筋肉も目覚め、基礎代謝が上がり、痩せやすい身体になります（≧∇≦）

【ＴＲＸトレーニングで期待できる効果】
筋力強化だけではなく、体幹強化、持久力アップ、柔軟性アップ、バランス感覚向上など、あらゆる要素が同時に鍛えられるのが最大の魅力です。

また、お客様の声で多いのが、「お腹周りがすっきりした」「疲れにくくなった」「身体が動くようになった」ということ。ウエスト引き締めにはとても効果的なトレーニングです。

【ＴＲＸトレーニングの誕生】
TRXはアメリカ海軍特殊部隊sealsが、孤島で訓練していた時にカラダ全体の筋肉を鍛えるエクササイズ器具が無いことに気づき、手元にあったパラシュートのコードと柔道の帯を組み合わせ、手縫いしたものを扉に引っ掛けてトレーニングしたのが始まりとされています。

そして、さらに進化し、商品化されたものがTRXです。
現在ではプロ野球選手、Jリーガー、NFL…etc オリンピック選手など様々の競技のアスリートから高齢者まで幅広く使われています。

【ＴＲＸトレーニングの特徴】

1.常に体幹トレーニングになっている

TRXトレーニングと他のトレーニングの大きな違いは、常に体幹への負荷が加わることです。

不安定な状態でバランスを取りながら、トレーニングするため、あらゆる動きの土台である体幹が自然と強化されるので、体が安定し、バランス力もアップします。

2.人間本来が持っている動き、パフォーマンスが高まる

スポーツや日常生活など、実際の動きは1つの関節だけでなく、複数の関節が連動して動きます。

TRXトレーニングでは、自身の体重や重力で上下左右前後など3次元の負荷をかけることで、動き全体の連動性と協調性を高めた動きのトレーニングをすることが可能になります。

3.だれでも効果的で安全にトレーニングすることができる

TRXは使う人の身長に合わせて長さが調整できます。
また、体の位置を調整するだけで、同じエクササイズでも負荷を重くしたり軽くしたりすることができるので、プロのアスリートから高齢者などの低体力の方まで、各々レベルに合わせてだれでも安全かつ効率的なトレーニングが行えます。
