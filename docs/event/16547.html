description

　会場は通常通り地下１階会議室になります。

　三鷹ハードウェアエンジニアカフェとは、エレクトロニクス／
メカトロニクスの 分野におけるエンジニアの人的・技術的な交流を
目的とした集まりです。 以下の分野のエンジニアの方々を対象と
しています：

【対象とするエンジニア】
　プリント基板の設計エンジニア
　マイコンのハードウェア設計エンジニア
　マイコンのファームウェア開発エンジニア
　FPGAの設計エンジニア
　ASICの設計エンジニア
　フルカスタムLSIの設計エンジニア
　電子部品の設計エンジニア
　アナログ回路の設計エンジニア
　高周波回路の設計エンジニア
　パワーエレクトロニクス回路の設計エンジニア
　メカトロニクスの設計エンジニア
　各種ロボットの設計エンジニア
　等々

【活動内容】
　月に２回程度、ウィークデイの午後に集まりライトニングトークや
雑談などの人的・技術的な交流を行います。 飛び入りのライトニング
トーク歓迎します。会場には無線LAN設備がありますので、
ノートパソコンをご持参いただきますとより楽しめるかと思われます。
入退室自由です。

【参加費】無料。ただし、会場にてご注文の飲み物は有料となります。
なお、会場は食事は禁止とさせていただきます。

【お問い合わせ先】
三鷹ハードウェアエンジニアカフェへのお問い合わせは以下の
メールアドレスへお願いいたします：
info@hwcafe.org

【注】三鷹ハードウェアエンジニアカフェは、三鷹プログラマーズ
カフェおよび（株）まちづくり三鷹のご協力のもと、三鷹プログラマーズ
カフェの分科会として設立されました。
