リアル捜査ゲーム
よみうりランド園長誘拐事件2
～よみうりランドの園長は、預かった。～

何者かによって突然誘拐された、よみうりランドの園長。
なぜ、何のために誘拐されたのかも謎のまま。
突如届いた脅迫状には、伝説の秘宝「月の指輪」を用意しないと
園長を殺す、という文字が…。

果たして、犯人グループの狙いは一体何なのか。
月の指輪とは一体なんなのか？
この事件を解決出来るのは、名探偵であるあなただけ。
果たしてあなたは、この謎だらけの事件を解決できるだろうか？

※このイベントは、実際にあなた自身が様々な謎を解き、
事件を解決するゲームイベントです。

【リアル捜査ゲームとは】
トータル5万人以上を動員し、海外でも好評を得ている「リアル脱出ゲーム」を
プロデュースするSCRAPが送る、体験型エンターテイメント第二弾。
自分がまるで物語の中の名探偵になったかのように証拠品を捜査したり、
目撃者に聞き込みをして、事件を解決することができるゲームイベント。
