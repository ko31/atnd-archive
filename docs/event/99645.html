【登壇者紹介】

Bear Create　熊倉洋介

[プロフィール]
クリエイティブくまさんとして映像、デザイン、イベントなどの企画・構成・演出・制作・ディレクションをやってます。

目的に応じたクリエイティブの企画提案から制作までワンストップ対応しております。っていうか面白そうなことなんでもやります。面白そうなやつは大体友達。
　
[経歴]
小学生の頃からお笑いバカでラジオのハガキ職人として青春の大半を費やす。腐った20代前半を経て、構成作家として活動。

知人が監督するインディーズムービーの脚本を担当してから、自らも映画制作願望に目覚める。とりあえずブライダルのカメラマンと編集マンの仕事も始め撮影と編集の知識と技術を身につける。ついでにイベントディレクターの仕事もはじめる。

30代後半になり突如「正社員」に憧れ、映像制作事業を行っているの株式会社vivitoのプロデューサーになる。

しかし、やりたいことが溢れ過ぎて2018年8月に独立。現在はBear Createのクリエイティブくまさんとしてのびのび活動中。

ジョイントメディア　加藤洋平

[プロフィール]
NOMIKAI OSAKA主催。

幼少期よりIT, ゲームに興味があり父親のPCを幾度となくクラッシュさせてきた。ウェブデザイナーになりたい！と奮起して未経験ウェブデザイナーを募集している会社でアダルトサイトの管理人になってしまうところから, 彼の人生はおかしくなっていったのだ…。元ヴィジュアル系バンドベーシスト。

元しょーもない男。最近8キロの減量に成功。ヒュー
　
[経歴]
いじめられっ子 → ヴィジュアル系 → アダルトサイト管理人 → 広告管理 を経て現職へ。

未経験でソーシャルゲームプランナーとしてスタートし, その後ゲーム業界向けのイラスト制作事業を立ち上げ。

オリジナルキャラクター「いやしましゅまろ」のプロデュースを行う中で得た知見を活かしTwitter/facebookでのキャラクタープロモーションを販売開始。個人ではずーっとヴァンパイアのゲームをつくり続けている。

【NOMIKAI OSAKAとは？】
NOMIKAI OSAKAは2016年から毎月1回開催しているゲーム業界中心の交流会です。現在は大阪開催の他に「京都」「東京」でも毎月開催。
そしてNOMIKAI OSAKAはこれまでの飲み会の概念を逸脱した革新的な試み。主催の加藤は語る「オレ、ひとりで飲んでるからみんなもきてもいいよ」
　
　
【システム】
