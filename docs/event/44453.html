"PhotoShop" WEB基礎デザイン・バナー作成講座

description

PhotoShopを使ったWEBデザイン初級者を対象とした基礎講座です。

“ADOBE PhotoShop”は画像編集の定番アプリケーションソフトです。フォトレタッチ機能で既存のデジタル画像を編集したりすることで有名ですが、実は非常に多彩な描画機能も備えております。


今回のセッション(教室)では、この“描画機能”にフォーカスして、いろいろな機能についての解説とともに実践ワークショップ形式でそれぞれの作品（テーマ）を表現して行きます。

また、画像形式などについての基礎知識、色彩・配色とイメージ、文字キャラクターへの理解などについてもポイントをまとめて解説します。

デザインが好きだけど、実際自分でデザインすると思い通りにいかない…

もう少しプロのようなデザインに近づきたい…

そんな方の為のWEBデザイン基礎講座です。

プロの現場で実際に行われるデザイン制作のプロセスについてご興味をお持ちの方にも参考となる内容かと存じます。

内容

基本セクション

  1. 学習環境の準備
  2. PhotoShopの概念、基本操作について
  3. ビットマップ、ベクトルデータの違いについて
  4. jpg,gif,png画像の違いについて
  5. 拡大縮小の概念、スマートオブジェクトについて
  6. パターン、ブラシ、シェイプの使い方
  7. 写真加工について（実習）
  8. レイヤースタイルについて（実習）

色彩について

  1. 色ごとの与えるイメージの違い
  2. コントラストについて
  3. 配色のまとめかた（色相環）
  4. グラデーションの扱いかた（実習）

タイポグラフィ（フォント）について

  1. フォントごとの与えるイメージの違い
  2. 文字詰め・ジャンプ率による訴求ポイント

レイアウトについて

  1. レイアウトのまとめかた
  2. 情報のグルーピング（実習）

バナー作成（実践）

  1. サンプルファイルを基に
  2. サンプル素材を基に
  3. 新規のキャンバスにテーマを反映

学習環境

以下の環境（設備）は、こちらで用意してあります。

  • インターネット接続
  • 電源（延長コード）、LANケーブル（有線LANの場合）

 

必要な準備

当セッションに持ち込めるPCをご用意ください。

  • パソコン（Windows,MacどちらでもOK）
  • Adobe PhotoShop

推奨環境

WindowsOSは、
WindowsXP以上を推奨いたします。
MacOSは、
10.5以上を推奨いたします。
Adobe PhotoShopにつきましては、
「CS2」以降を推奨といたします。

 

対象とする方

  • WEBデザインに興味ある方
  • WEBデザイナーを目指している方
  • PG、SEでスキルの幅を広げるためWEBデザインを勉強したい方、またはWEBデザインに興味のある方

PCの基本操作ができ、PhotoShopを多少なりとも触った事がある方でしたら無理なく進めることが可能です。
