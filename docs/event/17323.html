description

青山ソウルオブソウルが送るパーティー！　７月２３日（土）に、今回１１回目を迎える、エキセントリックパーティー『こい☆こい』は、誰もが、必ずといってもいいほど書店店頭で見かけた事のある人気イラストレーター『さやか』氏の画家人生初の新境地ともよべる大型ライブペイントが実現！

ライブペイントと共にこちらも初となる魅力的なプロダクトとして、フロラワーアーティストの『森友美』氏によるライブによるフラワーオブジェクトが真夏の夜の「酔狂な夏祭り」を彩る！

そして、この完成した壁画とオブジェをバックに大人気セクシーアイドル、『友田彩也香』ちゃんによるこちらも初お披露目となるオリエンタルバーレスクダンスの響宴！

ある意味、イット・ガールともよべる３人の、あらゆる表現をミックスさせることによって、これまでにない感覚を呼び起こす夏の祭典をやっちゃいます！！

もちろん、今回も実力派DJにによる朝までテンションを落とさないプレイに加え、Black Eyes TATTO による、本彫り Live TATTO ！！

サポート陣は、ZIMA、Red Bull、COEDO Beerによる３本柱を筆頭に、オリエンタルバーレスクダンスを最大限に引き出す「たんす屋」による夏祭りには欠かせない『ステキ♡浴衣』の衣装協力！
あのPLAY BOY社も賞賛するセクシーランジェリーブランド「オブセッシブ」社のショーランジェリーサポート！そして、アートペイントには欠かす事の出来ない日本屈指の老舗絵の具メーカー、「ホルベイン工業株式会社」のライブペイントサポートが実現！！フラワーオブジェクトと融合する光演出には「EXPLORE」社提供のレーザーボックスが幻想的な夜を演出！！

今回 【大人の夜の夏祭り！】と題した こい☆こい を、是非とも体験してください。夜中にお腹が空いても大丈夫！「高尾の異国風カフェ＆バー・TOUMAI」出店による本格派カフェ屋台ごはんも食べられますよ！

最後に・・出演アーティストの皆様、協賛企業様、そしてご来場してくださるお客様に心より感謝致します。今回もこい☆こいの収益金の一部は福島県二本松市にございます、善弘寺での"炊き出し""夏用物資"支援に活用させていただきます。
