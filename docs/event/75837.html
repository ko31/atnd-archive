理系小謎を克服する日

title 理系小謎を克服する日
catch
description

謎解きイベントを制作しているクロネコキューブ株式会社（所在地: 兵庫県神戸市東灘区森南町2-8-2）が、自社オフィスsavannaビルにて、「理系小謎を克服する日」を開催いたします。

リアル体験型の脱出ゲームで出てくる様々な謎。これらの問題の中には、みなさんが苦手とする問題もあるのではないでしょうか。今回は特に、魔方陣、覆面算、論理パズル、計算問題といった「理系チック」な謎をディレクターの喜多が徹底解説いたします！

そんな理系チックな小謎が出てくると、すぐにあきらめてしまったり、誰かに丸投げしてしまったりしているあなた！
ここでその苦手を克服してみませんか？コツさえ掴めば、楽しく解けるものばかりですよ！
この春、苦手なものとおさらばしちゃいましょう！

【略歴】
　2011 年に行われた神戸市の「デザインの日記念イベント 2011」をきっかけに発足。その後、活動を重ねて 2013 年に法人化。神戸を本拠地とし、参加型謎解きイベントの企画・制作・運営のほか、研修事業や地域活性化事業なども手がける。主な公演実績として、神戸市営地下鉄御崎車両基地で行った「海岸線大脱出」や、海遊館とコラボした「海遊館と亡霊客船」がある。
