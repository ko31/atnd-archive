HTML5 レベル1 試験対策 Web関連技術 「第1回 JavaScript」講座
≪スクー生放送のご案内です!!≫


Webサイト制作には、基本となるHTML/CSS以外にも、様々な技術に関する知識が必要になります。
HTML5 Professional 認定試験 Level.1 対策として、Webに関連する技術のうち、JavaScript,画像などのデータ形式,Webを拡張する仕組み,セキュリティ,Web広告について、全5回でそれぞれの仕組みを解説していきます。

第1回は・・・「JavaScript」について解説します。

 

最近の動きのあるWebページには欠かせないJavaScriptですが、HTML5ではより重要性が高まっています。
特にHTML5で追加されたCanvasなどの新機能はJavaScriptから操作する前提になっています。
JavaScriptに関してはレベル2試験で詳しく出題され、レベル1での重要度はそれほど高くありませんが、
試験範囲よりほんの少しだけ深いところまで解説をしていきたいと思います。
