description

　第102代早稲田大学広告研究会は「ワカモノ文明開化」と題して、昨年述べ16万人以上が来場した早稲田祭において4つのイベントを開催します。その一環として、俳優の六角精児さんをゲストに迎え、「一人旅」をテーマとしたトークイベントを開催します。

　旅好きで有名な俳優の六角精児さんをゲストに迎え、旅について様々な視点から語ってもらうことで、ワカモノに一人旅の素晴らしさや魅力を再発見してもらうこと、また、六角精児さんの一人旅のエピソードなど、具体的な話を聞いて頂くことで、ワカモノに一人旅を身近に感じてもらい、一人旅をするきっかけになれるイベントを目指しております。
　一人旅は、一人で行くことによる旅の自由さや困難、視点の変化から様々な刺激や発見を得ることができます。また、他の旅とは違い自分を見直すこともでき、私たちの視野を広げさせ、成長させます。2013年にJTBが実施した「『ひとり旅』に関する調査結果」によると、7割を超えるがワカモノ(20代以下)が「ひとり旅をしたい」と回答しています。このように若者が一人旅に興味を持ち、一人旅をする人口も８年連続で増加している一方で、実際にひとり旅をした事がある20代以下の割合は、5割に留まっており、一人旅にあと一歩踏み出せない若者がいることがわかります。私たちのイベントを通じてよりいっそう一人旅の魅力を伝えることを目的としています。
