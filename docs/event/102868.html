description

近年、ビジネスにおけるグローバル化の傾向は一層高まり、もはや業種や部門を問わず海外との英語を使ったやり取りを避けては通れない状況が生まれています。そのため、多くの企業が社員の英会話基礎力の強化のために初級者向けの英会話学校や教材を活用しています。
しかし、初級者向けカリキュラムの「卒業生」や、ある程度英語を使いこなせる中上級者が、実際のビジネス現場で必要になるより高度な英会話スキルを習得できるような教育プログラムまでは、準備できていないというケースも多いのではないでしょうか。
次のステップが明確でない語学研修は、結果的に初心者向け研修受講者のモチベーション低下にもつながります。
Rosetta Stone® Advanced English for Business（略称AEB）は、日ごろから業務に英語を活用している中上級者の方を想定し、その語学力をより実践的にステップアップさせるためのオンライン語学プログラムです。
本Webセミナーでは、ビジネスにおける語学ニーズの現状について考察し、今日の日本企業に足りないものは何なのかを明らかにします。そして、その課題を乗り越えるためのソリューションとして、AEBの特長や活用事例を、デモンストレーションを交えご紹介します。
社員の英語力のアップを真剣にお考えの企業の皆さま、お使いの英会話学校や教材では成果が上がっていないとお悩みの皆様、ぜひふるってご参加ください。
