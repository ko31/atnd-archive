今、最も注目されているビジネス指導！MONEY TREE　~お金のなる木の育て方~

成功・人生を変えてみたいと思うのであれば、私たちから直接学び、習慣から未来を変えて夢を叶える力を身につけるためには、今すぐこちらから参加！

講演概要

ロバート・アレン

  • 概要
    • お金、経済、富に関する名著のブックガイド、「お金と富の哲学世界の名著５０」に『お金と富』をテーマに、世界的古典から話題の新作までの名著５０冊の中ロバート・アレンの著書２冊が紹介されている、世界的マネーコーチ、財産構築のプロ中のプロ。
      多くの億万長者と呼ばれる人々が、「自分の成功は、ロバート・アレンのおかげさまだ」と言っている。
      彼は「グッド・モーニング・アメリカ」や「ラリー・キング・ライブ」といった、アメリカで最も人気の多くのＴＶトークショーにゲストとして度々招かれている著名人である。

h3.董秀珍（Jane Dong）

  • 概要
    • 中国福建省出身。今から２５年前に大きな夢を持って中国から来日。描いた夢は次々叶えた夢実現の達人。アメリカに移住、大手の会社と化粧品・健康食品事業の提携、結果を出す。台湾で言語学校を経営、米国の大手化粧品と健康食品会社と提携し、トップリーダーの実績をマスコミの取材を受けた。１８年前から、潜在能力、心理学、行動科学の研究を始めた。ＳＭＩ成功哲学の伝えと教えに専念、教育事業をスタート。
      現在、企業コンサルタント、執筆、講演活動をしている。
