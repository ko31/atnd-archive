description

世界で最も優れたアイディアは、私たちの日常生活、ハマってるゲーム、好きなキャラクターなどに
ヒントがあるかもしれないです。
「アイディアがあるけど、どう実現するのがわからない」、
そのアイディアを少しでも実現できるようにお手伝いします。
「どこからスタートすればがわからない」
ちょうどいい、ここからです。

１） プログラム知識がなくてもドラッグ＆ドロップ
（Visual Scripting）で簡単なモバイルゲームの作り方を教えます。
２）くるくるパズルの作り方を教えます（プログラミング知識が必要）
３）集まって自分のやりたいことをする、自習します。
※１〜３のいずれか選べます
プログラミングをしている方もいれば、それ以外の方（イラストレーター、サウンドクリエーターの方、ライター）
も大歓迎です。
※ゲームエンジンはUnity
