シュールストレーミングを食べる会

スウェーデン留学中に実際食べたことのあるkororaくんを迎え、 本気で美味しく食べる努力 をします。

細心の注意を払って缶を開ける→お酒や牛乳で洗う→薄いパンとジャガイモや赤タマネギ・ヤギチーズなどと一緒に食べる。の流れです。
kororaくん曰く「白ゴハンとも合うと思う」だそうです。

経験者の方で、これはやったほうがいい！っていう注意点などあればコメント下さいー！

骨があるからナイフを持って行った方が良いとか。

缶詰が届いてからは、なるべく発酵させずに形を保ったまま食べられるようにしようと思ってます。
（すみません、受け取りが遅れて、缶詰がちょっとふくらんでおります…11/7）

もっと良い場所をご存知の方いたら教えて下さい！！人気の無い公園とか。

冬なので寒さ対策はしっかり！首周り、頭、手も防寒しよう！カイロもあるといいね！
