Asakusa Scala DSL デザインレビューの勉強会

Asakusa Scala DSL

@asami224 の浅海さんが、現在サンプルベースで、Asakusa DSLのScala版をプロトタイピングで作成されています。可能であれば、集まって、そのデザイン・レビューをしようという話になりました。AsakusaDSLのサンプルコードも同時に見ながら、あるべきDSL論について、議論をしようという勉強会です。

参加者には事前に、Asakusa Scala DSLと実際のAsakusaDSLのサンプルコードも配布します。AsakusaはOSSとして公開されることが決まっています。公開と同時にecoSystem化を進めようという面白い試みにもなりますので、是非ご参加ください。

https://github.com/asami/AsakusaScalaDslSample
