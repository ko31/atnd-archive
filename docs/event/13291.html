イベント概要

Ubuntu “オフライン” ミーティングは、Ubuntuユーザーが直接顔を合わせいろいろな形で交流を行うためのイベントです。今回は2010年度の忘年会です。会場では恒例の昼食会&お茶会のほか、参加者によるセミナー、ライトニングトークを行ないます。おやつやお茶も楽しみつつ、関西のUbuntuユーザ同士の親睦を深めましょう。

電源と無線LANが使用できますので、ぜひお手持ちのUbuntuマシンをご持参ください。おやつ・お茶と参加者によるセッションを楽しみながら、さまざまな形でUbuntuに関する知見を集めて頂ければ幸いです。Ubuntuをすでに使っている方、まだ使われていない方・単におもしろそうだと思った方などなど、興味がありましたらぜひお越しください。

イベントの詳細はJapanese TeamのWikiページも参照してください。

参加費用

1,000円 (学生無料)

飲食物手配のため、参加する方は一人 1,000 円の参加費を入室時にお支払いください。学生の方は無料ですので、入室時に学生証をご提示ください。

おつりの用意が十分ではないため、なるべく千円札の準備をお願いいたします。

なお領収書の発行はできません。

2011/3/16 追記

Ubuntu Japanese Teamでは、先日の東北地方太平洋沖地震を受けまして、今回のイベントをチャリティイベントとして実施することに決定しました。

イベントは予定通り開催致しますが、皆様から頂いたイベント参加費は全額、日本赤十字社への寄付とさせて頂きます。
