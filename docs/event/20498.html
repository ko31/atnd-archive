株式会社ミクシィとJELLY JELLY CAFEのコラボレーションで、気さくに参加いただけるテクニカル系お茶会, mixi unwired (ミクシィ・アンワイヤード) の第一回です。

今回のあらすじ

今月中旬 (2011.9.13〜16), Windows 8など次期Windowsに関する, Microsoft主催によるデベロッパむけのカンファレンス ‘build windows’ が, USのAnaheimで開かれていました。
弊社からも数名で参加したのですが、そのなかで、iOS版のmixi公式アプリ(などなど)の開発にかかわっている衣川憲治 (@k_kinukawa) から、build windowsに参加してきた面白かったところをご紹介します。
というよりむしろ、カンファレンスでもらってきた Windows 8 tabletの実機(2台)のデモが中心になるかもしれません。
当日は、MacBook Airに、bootcamp経由でWindows 8 tabletを入れたものをもう1台持っていくかも。

Windows 8 tabletは, Metro UIが相当かっこいいです。
まだ日本国内では、実際に動くWindows 8 tabletは、まだ十数台ぐらいしかないと思います。
現物を見るチャンスはかなり貴重です。
ぜひ、みんなで見て、さわって、いじって、軽く飲みながらたのしみましょう!
