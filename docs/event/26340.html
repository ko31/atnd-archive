description

2012年3月31日は「0331」の日でもあり、坂本真綾さんのお誕生日の日でもあります。

2010年に日本武道館での15周年記念ライブ、2011年は中野サンプラザでYou can’t catch meツアー中野公演ライブ開催と続き、今年も3月31日に何かイベントがあるのでは？と思っていたのですが、今のところ何もないようです。

1年に一度の3月31日だし、真綾さんの誕生日でもあるし、せっかくの休日土曜でもあるので、誕生日をお祝いしてパーティーオフを開催します。
今回0331開催は2007年にアルバム「30minutes night flight」でコラボしたTRAVEL CAFEでの誕生日オフを開催して以来、丸5年ぶりになります。

そのTRAVEL CAFEで去年末にin the silenceライブや朗読劇「私の頭の中の消しゴム」を開催した天王洲銀河劇場の真下にあるTRAVEL CAFE天王洲アイル店にて約210分(約3時間半)開催します。

また、武道館ライブから2年、You can’t catch meツアーから1年という節目の日でもあるので今回の開催のコンセプトは

「０３３１を再現します。」

真綾さんファンの方なら、一人でもグループ参加でも可能です。是非３月31日の日にを皆さんでお祝いしましょう。お誘い合わせのうえ、ご参加をお待ちしています。

日時：3月31日 (土)  18:00〜21:30頃 (受付は17:45〜)

会場：TRAVEL CAFE 天王洲アイル店 (銀河劇場の真下になります)

会費：3310円 (飲み放題、料理込)

定員：5名から開催決定〜20名以上で1部屋貸し切り

主な内容
・真綾さんお誕生日と武道館ライブから2年、You can’t catch meライブから1年を祝してファン同士で飲食しながらパーティー
・一定の参加人数により貸切ができた場合、店内の音楽は真綾ソングを流す予定です。
・ここでは言えないことをします。
・参加の方は何でもいいのでグッズを持ってきてください。
(かぜよみ旗、武道館ライブで舞った風船、Tシャツ着用など)
