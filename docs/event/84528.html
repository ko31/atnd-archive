「自由な心と体は音楽と呼吸で手に入れる」
音楽療法とトランスパーソナル心理学を融合させた
呼吸と潜在意識のワークショップを開催します。

音楽に合わせた呼吸法を使いながら　あなたの潜在意識にアクセスしていきます。

日頃　忙しさで　自分の感情に蓋をしたりしていませんか？
感情の貯金はいらない貯金ですね。
わたしたちは　幼いころから　ずっと知らぬ間に　沢山の
「感情貯金」をしてきています。
その感情を　手放してみませんか？

このワークショップでは　生のジャンベという打楽器の演奏から　呼吸法により　ドラムセラピーをはじめ
音楽とヒプノセラピーを融合させた　あなたのインナーチャイルドへのアクセス。
呼吸法を使った　ホロトロピックブレスワークで　潜在意識にアクセスしていきます。
今まで　溜まっていた感情を　吐き出し　本来の自由な自分自身を取り戻して頂きたいワークショップです。

魂に響く　ジャンベのリズムは　五感をフルに使い
あなたの世界も変わっていくでしょう。
ぜひ　この機会に　体験　体感　して頂きたく思います。
