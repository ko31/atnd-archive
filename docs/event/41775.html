■本講習について

昨今のネットワークにおいて、多くのWebサーバがLinux上に構築されています。
しかし、Webサーバを立てるノウハウが欲しいけど、Linuxって敷居が高そう…　とか、Linuxは難しそう…　と躊躇されていませんか？

本講座では、Linuxを利用したWebサーバの構築について、座学とAWS (Amazon Web Service) を利用した実習により、導入から設定までを基礎からわかりやすくレクチャします。

■講習の概要

Linux/Apacheについての基礎を座学で説明し、その後にAWS環境を利用して
Linux(RHEL6.4) Webサーバの構築・設定の一連の流れを実習します。

■想定する受講者の方

・Linuxに触った事がないけど興味がある方、使ってみたい方
・今後業務でLinuxでWebサーバを構築・運用をする予定のある方

■本講習で取得できるスキル (ゴール)

・Linuxの基礎知識を習得できる。
・Apacheの基礎知識を習得できる。
・Apacheを使用したWebサーバの初期構築ができる。
