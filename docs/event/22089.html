description

【こんな方にオススメ！】
・有益な就職活動の情報を手に入れたい
・就職活動関連イベントの情報が欲しい
・能力検査を突破できるか不安だ
・就活仲間が欲しい
・大手企業内定者の話が聴きたい
（キーエンス、三井住友銀行、森永製菓など）＊５０音順

当セミナーは、
同志社大学公認キャリア形成支援サークル
drei＋（ドライプラス）のイベントです。

内定者が就職活動の戦略をプレゼン形式で提供、
簡易ＥＳ作成・模擬面接の実施、
内定者との交流会、
の三本立てでお送りします。

内定者が就活生の目線に戻って、
あったらいいなを形にしたセミナーに参加してみませんか？
就職活動というゲームのルールと攻略法を知りたくはないですか？

【コンセプト】
小規模×コンサル×就活仲間

セミナーに参加してくれた一人ひとりの求めるものに
しっかりと答えるため、
「小規模」で開催。
内定者から就活生にヒアリング、
参考となる今後の方針を
「コンサルティング」する。
頼りになる先輩、励まし合える仲間、
刺激を与えてくれるライバルといった
「就活仲間」との出会い。

このセミナーには就職活動を成功させるための
きっかけが詰まっています。

【具体的なセミナー内容抜粋】
・就職活動のストーリーチャートを確認
・内定を獲るための三要素とは？
・ES・面接対策法の解説と実践
・6種の能力検査の対策法の解説と実践
＊特にテストセンターに的を絞って高得点獲得法を提供

参加決定された方は、dreiplus.plus@gmail.comに

「大学名」「学部名」「名前」「電話番号」を入力しご連絡下さい。
より詳しい詳細を連絡させて頂きます。

皆さんのご参加、
心よりお待ちしております。
