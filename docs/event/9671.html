■内容

テーマ：湘南を描く

今や、風景画はイラストレーションの新ジャンルとして十分に成り立
っています。オリジナリティのある、自分らしい風景を描けることは、
とても重要です。
集合場所と、講評場所のみ指定しますので、各自自分のテーマに合っ
た風景を事前に調査して、雰囲気のある湘南の風景作品を持ち寄るこ
とにしましょう。

このイベントは、SUPER ILLUSTRATION SCHOOLの主催ですが、
誰でも参加できます。
みなさんのご参加をお待ちしています。

■集合場所日時

2010.11.06.sat
13:00　小田急線片瀬江ノ島改札

Sundish は、稲村ジェーンでも度々出てくるお店です。
お店からは江ノ島がとても綺麗に見えます。
特に夕焼け時の江ノ島はここがベストビューで、 歌川広重の作にも
稲村ガ崎から見た江ノ島と富士が描かれています。
系列の venuscafe は、 稲村ジェーンのメインロケ地らしいです。
隣に1500円で入れる温泉もあります。

■参加費

参加費は無料です。
集合場所の Sundish での食事代は実費精算とさせてください。
ここは、ワインもお料理も美味しくてお安いお店です。
皆さんの描いた江ノ島や稲村ヶ崎や湘南をテーマにしたイラストレー
ションを展示させていただけるかもしれませんので、このお店を選
