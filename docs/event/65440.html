Google I/O でサンフランシスコへ行ったついでにおいしいカニを食べようというイベントです。

念のため書いておきますが、日時は現地の時間です。自動的にカレンダーに登録してる場合などに日本時間として解釈されると思うので注意してください。

イベントURL: https://atnd.org/events/65440

場所は毎年恒例の Crustacean Restaurant SF です。

これ とか これ とか これ の 2015年版です。

過去の実績だと、予算は$80-$90 くらいです。おいしいのでついつい食べ過ぎちゃう！

本イベントについて

  • 19:00開始予定です。ただ、時間が多少前後する可能性があります。
  • 当日が金曜日でこれから予約という状況です。最悪お店の予約を取れず中止となる可能性があります。現在エントリされている14名で確定となります。ざきさんが予約してくれました！ドタキャン不可です(正確にはまだお店とのやりとりの途中なので確定ではない)。
  • これ以降の登録は、キャンセル待ちとなります。
  • ハッシュタグは、#kani15 です。Twitterなどで連絡するかもしれませんので、管理者(@zaki50と@korodroid)をフォローしておいていただけると助かります。
  • 当日の連絡は、Twitterで、@zaki50さんまでお願いします。

当日について

  • ドレスコードがあります。エリのついている服(シャツとか、ポロシャツとか)でないと、食べさせてもらえないらしいです。
  • 18:00にMarriott Marquisに各自集合後、みんなで移動します。時間厳守でお願いします。直接お店に合流する方はその旨を事前に連絡してください。ちなみに、Marriottホテルは市内に幾つかあるので、間違えないでくださいね。
  • 当日は現金で支払いをできるよう必ず事前準備をお願いします。過去の実績ではイベント中に銀行に駆け込むことになる人もいたりで大変だったので。特に夜間は危ないのでオススメできません。
