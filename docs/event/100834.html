Startup Arena

Startup Lady x Tokyo Tech Startupsが主催するスタートアップの大規模ピッチイベント、Startup Arena！！

今回のイベントでは、初期のスタートアップ企業、個人の製品、サービス、またはビジネスのアイデアを実績のあるアドバイザー&スポンサーの前で表に出せる舞台を用意しました。

主催概要

Startup Ladyとは？

Startup Ladyは、現在起業しているまたは将来起業予定の女性にトータルサポートを提供する非営利団体です。

リーダーシップのある女性に、質の高い国際ビジネスネットワークを提供。英語で話したい方にバッチリの環境を整えています！

起業している女性は、仲間を見つけると心強いと思います。交流会はコミュニケーションの場です。

どんどん悩みを話して、情報を交換して下さい。そして、これから起業したいと考えている方に大切なのは、質の高い交流会に参加することです。ぜひ、信頼できる質の高い仲間と繋がりを作ってください。

♡スポンサー募集中です！詳しくはメッセージをお願いします！

Tokyo Tech Startupsとは？

Tokyo Tech Startupsは、テクノロジーとスタートアップに興味がある起業家/エンジニアの団体です。

Tokyo Tech Startupsは、東京のテクノロジーとスタートアップの成長を促し、そういった場を提供するという共通の目標のもと、活動を続けています。

彼らの使命は、日本の企業文化の変化を押し進め、東京をアジアのスタートアップの拠点にするために、東京のテクノロジースタートアップエコシステムを育成し、提供することです。
