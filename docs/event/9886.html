description

熱いWeb業界と話題と人気のサービスを盛り上げるべく、業界人、いわゆる「中の人」と、
ユーザー、ファンのみんなが楽しく、心ゆくまで交流できる、ネットワーキングパーティー
を企画いたしました。
SocialWeb2.0Party(そーしゃるうぇぶつーぽいんとおーぱーてぃー)です。
第1回は、2009年7月24日に緊急開催され、60名の参加者が集い盛り上がりました。
「ぜひ次回も」の声に答えて、さっそく「第2回目」を、9月3日に開催し、80名の参加
者を集め、「特別編：Twitter国内関連サービス「中の人」ミーティング」、「第3回」、
「大忘年会」と開催。そして、2010年は、さらにパワーアップして開催。
さらに、3年目へ向けて今回、2010年を総括する大イベントを「SocialWeb2.0Party -2nd
Season-」として新たにスタートします。
後援は、@SOHO／NPO法人 日本ITイノベーション協会様です。
Web業界の最新トレンド、ソーシャルアプリやソーシャルゲームなどをフューチャーした
プレゼンやトークセッションで盛り上がろうというイベントです。
みなさまのご参加をお待ちしております！
