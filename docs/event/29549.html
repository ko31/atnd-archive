description

将来の年金不足に対する不安が毎日のようにマスコミで取り上げられています。国や企業に頼らない自分独自の年金づくりが必要な時代になりました。不動産年金教室は、不動産による自分年金づくりを応援します。セミナーでは特に不動産投資において重要なライフプランと投資モデルについて解説いたします。

・サラリーマン大家さんと不動産年金の違い
・不動産のメリットとリスク
・不動産への投資基準を考える
・ライフプランと投資モデル
・購入前に必要な収支シュミレーション
・具体的な物件で計算
・不動産年金教室の紹介
