description

OSC会津2012の開催に合わせ、その翌日に会津若松市でマッピングイベントを開催します。
今年に入りオープンガバメントやオープンデータの機運が高まっていますが、データは公開しただけでなく、それを活用することが重要です。
どのような使い方ができるのか、その利用事例として、会津若松市様や国土地理院様が公開しているオープンデータを活用し、wheelmap というOpenStreetMap をベースにした車椅子利用者向けの地図作りにトライしてみよう、というイベントです。今回、特別な技術は必要ありませんので、ぜひお気軽にご参加ください。

1.開催日時、集合場所
2012/11/4(日)10:30 (10:15開場)～17:00
会津大学 研究棟2階iLAB2
雨天決行（歩行困難な場合を除く）

2.費用
・参加費無料
・交通費、食費等は各自で負担ください。

3.募集人員
・30名

4.持参するもの
・最低限筆記具があればOKですが、以下のようなものがあるとさらに楽しめます。
・デジカメ
・スマートフォン
・GPSロガー
・モバイルなPC

5.服装など
・徒歩で歩く予定ですので動きやすい服装でお越しください。

6.目標
＜全員＞
・OpenStreetMap の概要を知る。
・オープンデータの概要を知る。
・バリアフリーやユニバーサルデザインについて考える。

＜技術に興味のある方＞
・スマートフォンやPCを操作してwheelmap.org 上に車椅子利用可否情報を入力する。

7.タイムスケジュール（当日の状況により多少変わる可能性があります）

10:15 開場(会津大学に集合)
10:30 OpenStreetMapの概要
11:00 Wheelmapの操作説明、グループ分けなど
12:00 移動と昼食
13:00 街歩き（会津若松駅周辺など）
15:00 wheelmap編集(会津大学に戻ります)
16:00 成果発表
　グループ単位に編集成果や本日の感想などを発表します
17:00 終了
18:00-20:00 懇親会（近場で。希望者のみ。ワリカン）

8.街歩き(13:00～)の内容
・参加人数によって、４～５名ずつのグループに別れ、グループごとに予め決めたコースに沿って歩きます。

・スマートフォン(iPhoneまたはAndroid)を持っている人は歩きながらお店や施設を車椅子利用者が自力で入れるかどうか、下記3種の印を付けます。
　利用可(緑)：入り口の段差が7cm以下で施設内もほぼ自由に通行できる
　一部利用可(黄)：入り口の段差が7cm以下で施設内の一部に通行できない場所がある
　利用不可(赤)：入り口の段差が7cm以上であったり傾斜がきつくて自力では通行できない
(可能であれば事前にwheelmapという無料アプリをインストールしておいてください。iPhone版、Android版)

・スマートフォンをお持ちでない方は、地図を印刷した紙に、上記の印を書き込んで行きます。地図に無いお店や施設はその種類（レストランとかコンビニとか）と名前をメモしておきます。写真を撮っておくと後で便利です。

9.wheelmap 編集(15:00～)の内容
・当日ご説明しますが、大まかには下記のような操作をします。
http://maps.wizu.jp/wp/?p=1701

10.事前作業（主催者側が行います）
・政府、自治体による下記オープンデータを事前にマッピング(OpenStreetMapに投入)し、当日のマッピングをスムーズに行えるようにします。
　会津若松市：公共施設の所在データをインポート
　http://www.city.aizuwakamatsu.fukushima.jp/docs/2009122400048/
　国土地理院：基盤地図情報1/25000を画像としてトレース
　http://www.gsi.go.jp/kiban/etsuran.html
