description

クラウドファーストの考えが定着し、企業がクラウドを活用することを前提にシステム構築やサービスを提供することが主流となったいま、次の流れとして社内サーバのクラウド化や、データセンタへの移行をクラウド導入と共に検討する企業が増えています。かつてはパブリッククラウドと比較して非常にコスト高だったプライベートクラウドも、at+linkが提供するラインナップをはじめ安価なものが増えてきました。本セミナーは 「プライベートクラウド」 をテーマに、OpenStack、KVM、分散ストレージ、Sheepdog といった技術トレンドの基本から活用事例までを解説します。
