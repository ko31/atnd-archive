description

「IT スキルで勝負したい、社会問題解決に貢献したい、世界に挑戦したい、、、、
そんな想いを持った学生の皆さんと一緒にチャレンジしたいのがイマジンカップです。
ビル・ゲイツの「学生達に自分のアイディアや技術を発表する場を提供したい」という思いが形になり、
「テクノロジーを使って今日の世界に変化を起こさせたい」という情熱、想像力をもった学生を応援する
技術コンテストとして、全世界の学生を対象に 2003 年から開催されています。
2012 年の世界大会は、7月にオーストラリアのシドニーで開催されます。
オーストラリアの世界大会に進む日本代表を選出する日本大会は、2012年4月7日に開催が決定しました。

> Imagine Cup 2012 日本大会 URL: http://msdn.microsoft.com/ja-jp/academic/hh673593

今回の勉強会は、4/7の日本大会開催前の最後の勉強会となります。
日本大会に向けたチーム登録や応募方法の説明、応募に必要な Preliminary Summary（ソリューション概要）の記述方法、提出ビデオに必要な項目の解説などを行います。
イマジンカップへのチャレンジ（参加）を決めている学生のみなさんの参加をお待ちしています。

「Imagine Cup 日本大会のスケジュール、応募方法の説明」

イマジンカップの応募要件となっているビデオについて、昨年の応募ビデオの解説を交えて説明します。

「Imagine Cup向けプレゼンテーション講習」
イマジンカップの応募ビデオや本大会で必要となるプレゼンテーションスキルについて説明します。
