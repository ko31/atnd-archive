■ 開催概要

本イベントのテーマは、「エンジニアによる重要な意思決定」についてです。
「マギアレコード魔法少女まどか☆マギカ外伝」他複数のスマホゲームを
開発および運営するf4samuraiならではの、エンジニアが果たしてきた重要な意思決定と
開発の裏側を中心にお話をさせて頂く予定です。


第一部：マギアレコード開発の裏側

「マギアレコード」企画初期からリリースまでの苦労

ゲームが求める要件とそれを実現するためのアーキテクチャの選定、
リリースまでのアーキテクチャの変遷、リリース後に発生した問題への対応など、
プロジェクトの時系列に沿って、そのときにどのような意思決定がなされたのかを、
サーバサイドエンジニアを中心に、プロデューサー、CTO、それぞれの視点でお話しさせて頂きます。


「マギアレコード」の開発環境

サーバサイド： 　　　Java, AWS, HTTP/2, ElasticSearch, Logstash, digdag, …
クライアントサイド： cocos2d-x, WebView, Backbone.js, Live2D, …
どのような技術要素をどのように用いているか等、事例も併せてご紹介いたします！


第二部：懇親会

講演終了後の会場にて、そのままお越しいただいた皆様でお食事と
情報交換をしていただける場を設けております。是非ご参加ください。
