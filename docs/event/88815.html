OPEN 18:30 / START 19:00 / CLOSE 20:30

KROWLセミナーについて

Web制作・運営、インターネット広告、SNS、ソーシャルゲーム、オンラインショッピング、ポータルサイトの運営など・・・IT/デジタル業界現場のナマの声や、スキルアップに役立つ情報を毎月お届けしています。IT/デジタル業界でまだキャリアがなくスキルアップしたい方など、どなたでも歓迎ですのでふるってご応募ください。また、キャリアコンサルティング、インターンシップ情報を含めたKROWLの最新情報もお知らせいたします。
※友人とご一緒に参加もOK！

第22回 イベント概要

メイントーク

今回のセミナーでは、スマートフォン・アプリ開発のスペシャリストを育ててきた経営者をゲストにお迎えします！イマ知っておきたいIT/デジタル業界の最新動向、イマ大注目の「エンジニア」というお仕事について詳しく語って頂きます。実務未経験からエンジニアとしてチャレンジするために必要なこと、すべてのお仕事に共通する重要なポイントなど、ここだけの話をお伝えする講演内容となっておりますので、お聞き逃しなく！

質問・相談会

ここをもっと詳しく聞いてみたい、現場でのココが知りたい、そんな疑問・質問にお答えします！

地図

主催： 株式会社 アイ・ファクトリー

アイ・ファクトリーは博報堂アイ・スタジオグループとして、2011年に設立したデジタル制作×人材サービスの会社です。大手・有名企業のコーポレートサイト、キャンペーンサイト、バナーなどのデジタル制作と、自社の教育システムを活かしたIT/デジタル人材の育成を行っています。また、業界未経験から現場で活躍できるクリエイターをこれまでに多く輩出した実績をもつ会社です。

KROWL とは

これからIT/デジタル業界でチャレンジしていきたい方を対象に、セミナーや面談、キャリアコンサルティング、インターンシップ等の就業支援を行っています。
IT/デジタル業界を知っている企業がサポートしているからこそ、就職活動もスムーズに。是非KROWLの就業支援を体験してください。
