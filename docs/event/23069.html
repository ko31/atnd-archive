【本セミナーの狙い】
ゴムローラによる紙の搬送はプリンタ,コピー機,ドキュメントスキャナなどに共通する基礎的な技術であるが,その理論的な紙搬送特性予測は難しい課題であり,旧来の直感に基づく古い常識が誤りであることが近年になって明らかになってきている.本講ではゴム挟持ローラによる紙搬送の基本特性解析について説明するとともに,受講生が自分で解析が可能となるようプログラムを配布し,その使用方法を解説する.次に,基本特性解析結果を用いた紙のスキュー解析,ニップ部の圧力分布解析の方法を解説し,所望の圧力分布を有するクラウンローラの設計法について教授する.また,ゴムローラを用いた紙の繰り出し機構についても最新の解析結果を示し,解説する.

【キーワード】
1.マイクロスリップ
2.トラクション/フリクション
3.ニップ搬送

【プログラム】

1.短軸ゴム挟持ローラの基本特性解析

　2.1 プログラム概要

3.長軸ゴム挟持ローラによる紙搬送解析
　3.1 解析の仮定と手法
　3.2 解析結果の一例 (紙なし)
　3.3 偏荷重に対する紙のスキュー解析と設計
　3.4 ニップ部の圧力分布解析とクラウンローラの設計
　3.5 長軸ローラによる紙搬送における横ずれの実験解析
　3.6 参考文献
