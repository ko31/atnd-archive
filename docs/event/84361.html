Docker勉強会2017 実践編 @長野県塩尻市

- Dockerの実践方法について学ぼう！ -

昨年2016年に開催して好評だった Docker 再入門 2016 update @長野県塩尻市 の続編を開催します。
前回都合で参加来たかった方や復習の為におさらいも前半に実施します。

今回の勉強会では参加者が実際にハンズオン参加してDockerの実践方法について学びたいと思います。

ハンズオン参加について

ハンズオンも実施します。解説を聞いてご自身で自己学習して頂く事が出来ます。ノートPCをお持ちください。要件は以下にあります。
※PC貸出をご希望される場合は、申込時にお知らせください。

対象OS環境

Dockerは最新の64bit OSが必須です。
LinuxはKernel 3.10以降が対象です。
Windows 10 Proが対象です。
　他のバージョンのWindowsは条件によっては動作可能です。
macOS 10.10 Yosemite以降が対象です。
※入門編ではない為、最新のDocker環境をセットアップ済みでご用意ください。

募集枠：事前登録で5-10名程度

参加費が必要ですので、募集枠以外の見学参加をお断りいたします。
また、会場設備の都合がありますので参加登録をお願い致します。
会場にて簡単なアンケート記入をお願い致します。
追加募集はありません。最大10名までとさせて頂きます。
