概要

AWS（Amazon Web Service）の鹿児島勉強会を開催します！
第２回の今回は、日本のDynamoDB第一人者得上さんをお招きして
NoSQLのAWSサービスであるDynamoDBのワークショップを行います。

NoSQL？DynamoDBって何？どういうときに使えるの？
RDS（AWSのRDBサービス）やSimpleDB（AWSのNoSQLサービス）との使い分けはどうするの？等々
ビッグデータなどで今何かと話題のNoSQLサービスに触って学んでみませんか？
今回はなんと、AWSのサイト内検索サービスCloudSearchの話しもしていただけます。

もちろんAWSに全く触ったことない方や、初心者の方も大歓迎です！！
お気軽にお越しください。
