●今回のテーマは「運命を決める大宇宙の法則」

幸運・不運の運命は、一体、何によって決まるのでしょうか？

仕事・恋愛・勉強やスポーツの成績・事故や病気、
だれでも、幸運を望み、不幸を遠ざけたい。

「苦しい時の神頼み」で、神社に手を合わせれば願いは叶う？

気休めではない、仏が教えた運命の法則。
これを知れば、あなたの仏教観は180度、大転換します。

● この講座に参加することで、得られること

何があってもブレない、人生の軸が得られます。

仕事や子育てや生活に、やりがいをもって臨めるようになります。

打たれ強くなり、 多少の事には、へこたれなくなります。

前向きに生きようとする仲間が得られ、悩みを相談できる人が見つかります。
