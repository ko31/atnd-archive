6月のLeafcup横浜のイベントは、『The Amazing Race!!』

日本ではあまりなじみがないですが、海外ではとても有名なゲームです。
体力面、精神面を試され、あなたのさまざまなスキルが鍵となります！もちろん外国人1人と日本人2人で１チームを結成してのチーム対決なので、楽しく英語も勉強していただけます！
今までとは一味違った横浜リーフカップ主催のスペシャルイベントAmazing Raceで、あなたも普段の退屈な生活から抜け出し、すばらしい達成感を味わいませんか？＾＾

ゲームの後には、横浜みなとみらいでのミニピクニックも開催しますので、盛りだくさんな日になること間違いなし！
ぜひお友達をお誘いの上、ご参加くださいね！もちろんお一人での参加も大歓迎です♪

● 日にち :　６月３０日（土）
● 時間　:　１１時００分～１６時００分まで
● 場所　:　みなとみらいの臨港パーク
● 参加料　:　ペアで参加：1人あたり2,500円、お1人で参加：3,000円
※ご予約はURLよりお申し込みください＾＾

【スケジュール】

11:00 : パシフィコ横浜のけやき通りの近くの入り口に集合：地図はこちら

11:30 : 概要説明

12:00 : ゲーム開始

14:00 : プチピクニック

16:00 : 終了

ルール説明

1) チームは日本人のペアと外国人スタッフが一名となります。

2) 各チームには様々な試練、なぞなぞが出されます。試練を乗り越え次のヒントを得ながらゲーム進めていきます。

3) ヒントは全て12個用意されています。全ての質問に対する答えは最後の問題を解くカギとなるので最後までもっていてください。

4) ヒントはみなとみらいの臨港パーク全体に隠れています。一つクリアするごとにゴール に近づいて いきます。

5) ヒントは各チームそれぞれ異なります。チームの英語力、知識が勝利のカギとなります。

6) チーム対抗となりますので、いち早くゴールに到着したチームが勝ちとなります。

参加者条件

1) 参加者はパートナーが必要となります。
またパートナーがいない方はこちらでパートナーをご紹介いたしますのでお一人でもお気軽にご参加できます。
2) ペアで参加の場合お一人2500円の参加費となります。
3) お一人で参加の場合登録料3000円となります。
4) 参加チームは15チームまでとさせていただきますのでお早めの登録をお勧めいたします
5) 外国人スタッフに関しましてはこちらで決定させていただきます。
