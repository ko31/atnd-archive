ネコ市ネコ座にて「ネコ国大統領からの挑戦状」を開催します。

謎解きイベントを制作しているクロネコキューブ株式会社（所在地: 兵庫県神戸市東灘区森南町2-8-2）が、『ネコ市ネコ座 with ピュリナ～ホゴネコ文化祭～＠ネスカフェ原宿 』において、謎解きイベント「ネコ国大統領からの挑戦状」を企画制作いたします。

ネコ市ネコ座は、猫に特化した手作り雑貨・商品の販売や猫に関わるイベントを行い、「楽しみながら、猫助け」を行うのがテーマのイベントです。猫とニンゲンとが共生し、幸せに暮らせる社会の実現のために、この世のすべての猫が幸せになるように、猫のおかれた悲しい現状や、猫問題の解決策について、楽しみながら知ってもらうキッカケとなるイベントです。
ネコ市ネコ座で行われるイベントの1つとして、会場内を探しまわる謎解きイベント「ネコ国大統領からの挑戦状」をクロネコキューブ株式会社が企画制作いたします。「ネコ国大統領からの挑戦状」は参加者自身が主人公となり、謎を解いて物語を進めていく参加型ゲームです。プレイヤーは暗号やパズルが散りばめられた会場を探しまわって謎を解き、目的を達成することができればクリアです。
“ネコ国”の大統領からの謎の挑戦状。ネコ市ネコ座の会場内に仕掛けられた謎を解き明かし、クリアを目指しましょう！
