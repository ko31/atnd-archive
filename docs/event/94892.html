ご好評につき、CMSの導入をご検討中のご担当者様を対象に『CMS徹底活用セミナー 入門編』を開催します。

コーポレートサイトをはじめ、各種サイトの作成を成功させるための秘訣や、多数あるCMSの中からセキュリティ対策を中心に、失敗しないための選定方法や導入方法、さらにはCMS選定時の社内決裁を通すためのポイントまで徹底的に解説します。

新たにCMSの導入をご検討中のご担当者様だけではなく、これからサイトのリニューアルをご検討中のご担当者様にも特にオススメのセミナーとなります。

セミナー概要

開催日：2018年4月12日（木）15:30 - 17:30（受付15:15〜）

定員：先着10名

会場： 〒108-0071 東京都港区白金台2-26-10 スカイアーク東京 セミナールーム

　　　（都営浅草線 高輪台駅 徒歩1分／JR 品川駅 徒歩10分）

受講料：無料・完全予約制

受講料：無料・完全予約制

セッション内容

1）昨今のCMSに求められる役割の変化

2）セキュリティを考慮したCMS選定のポイント

3）サイトの目的に合わせたツールや施策のご紹介

3）サイトの目的に合わせたツールや施策のご紹介

　　　　- コーポレートサイト

　　　　- メディアサイト

　　　　- 会員制サイト

　　　　- リクルートサイト

　　　　- コミュニティサイト

　　　　- イントラサイト

　　　　- Web社内報

　　　　　etc...

5）CMSプラットフォーム「MTCMS」のご紹介

こんな方にオススメです

・新たにCMSの導入をご検討中の方

・ウェブサイトの新設、またはリニューアルをご検討中の方

・マーケティング部門のご担当者様

・広報部門のご担当者様

・総務、人事部門のご担当様
