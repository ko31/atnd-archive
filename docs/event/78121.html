発表テーマ【Unity の Unit な Test を試してみた】
Unity 5.3 から標準組み込みになった Unit Tests Runner のセットアップ方法や簡単な使い方などを紹介します。
また、時間があれば Integration Test についても触れたいと思います。

・Riftup@VR/MR in JPさま
VR/MRアプリ開発者。日本初？のUnity Certified Developer。
海外で注目が高まっているVRやMRを日本にも広めたくて、コンテンツ出展、記事執筆、Unityでのアプリ開発Tipsをブログ(http://magicbullet.hatenablog.jp/)で紹介、などの活動を行う。
Twitter ID : @WheetTweet

発表テーマ【HoloLens向けゲームアプリを作るために知っておきたいこと】
HoloLensとはMicrosoftが開発したデバイスで、実空間にホログラムを重ねる、MR(Mixed Reality)が体験できます。
現状、日本では発売が未定ですが、エミュレータを用いて動きを見たり、アプリ開発が可能です。
今回の発表では、HoloLensの概要、アプリ開発方法、および、できること、できないことなどを解説し、現在開発中のゲームについて紹介いたします。
（なお、本日の講演ではHoloLensの実機デモ、体験はございません。あらかじめご了承ください）
