description

あなたはこの、お日様すらも恋を応援してくれているかのような
アツい夏の季節、燃えるような恋をしていますか？

こんな身を焦がすような暑い日差しの中でも、
一緒にいるだけで暑さを忘れて笑いあっていられるような
素敵なパートナーはいますか？

いえ、まず
異性に恋をする前に、

【あなた自身に恋をすること】は出来ていますか？

『自分に恋・・・？
そんなの出来るわけない』と思ってしまったあなたに必見です。

★その他、こんなあなたにオススメのセミナーです★

・ひと肌恋しい秋の季節が来る前に、何が何でも最高のパートナーが欲しいというあなた。

・妥協なしで、自分が好きになった相手と恋人同士になりたいというあなた。

・恋をしたいけど自信がないというあなた。

・自分に似合う色が分からないというあなた。

・ファッションに自信がなく、ルックスに自信がないというあなた。

・服に金をかけるけど、友達に褒められたことがない、
褒められたことが少ないというあなた。

・『恋愛なんてテクニックでしょ！外見なんて関係ないぜ！』というあなた。

※こんな方にはオススメ出来ません。。。

・『最高の恋人がいるし、関係も円満。運命の人だと思ってるし、
毎日友達にのろけ話ばっかりしてる！言うことなし！』

という方。

●自分のパーソナルカラーを知り、服装だけで
『顔写り』を１０倍良くする方法。

●自分にも恋にも、心から湧き上がる自信を持つことが出来るようになる、
自分に１００％合った着こなし方。

●第一印象で、異性を魅了し、これは逃せないという恋を成就させる方法。

●骨格に合わせたスタイリングを学ぶ

●ヘアカラー、ヘアスタイルで着るべき服が変わる！？
　９割の人が意識しない服の選び方。

～～～～～～～～～～～～～～～～～～～～～～～～～～～～～～～～～～～

【人は異性のどこに惹かれるのか。】

そう、
その答えは、ズバリ
『ルックス』です。

【第一印象ですべては決まる】ってよく言いますもんね。

ただ、
ここで勘違いしてほしくないんです。

【外見があなたの恋愛を左右する】
なんて言うと、とても心の狭くて極端な意見だと思いますよね。

『そもそも恋人を外見で選ばないし….』
『中身が伴ってこそでしょ。外見なんて二の次』

たしかに、そのような意見を持つ人はとても多くいると思います。

しかしここで言っている『ルックス・第一印象』というのは
生まれ持った素質だとか、どうやったって変えようのないその人の
骨格を意味しているのではありません。

自分のタイプを理解し、認め、自分を引き出すスタイリング
をすることで元々の魅力をさらに最大限に高めることによって
自分自身に対する自信を爆発的に高め、

【内側からキラキラ輝くオーラを放っている様】
を意味しているんです。

つまりは
『自信』があなたのルックスを決めているのです。

さて、では
『自信』はどうやって持つのか？

それは小学生の足し算よりも簡単。

【自分を知る事】だけです。

今日のセミナーのテーマである
『恋するイメージコンサルティング』

あなたのタイプに合わせたカラーや着こなし、スタイリングを
学ぶことで、

街中や電車の中、
納涼船イベントや立ち飲みＢＡＲであなたを
一目見た異性が、

あなたの、他の人とは確実に違う、
スバ抜けた第一印象だけで
恋に落ちる可能性が３０％も４０％も上がってしまうのだとしたら。。。

そんな門外不出の、きれいごと抜きの本気の印象アップ講座。

今日、恋愛はじめました宣言をしませんか？
