description

オープンストリートマップ（OpenStreetMap）とは、「自由な地図」を「みんなの手」で作る世界規模のプロジェクトです。
このイベントは自由参加で開催いたしますので、皆さんが参加して、自分たちのための新しい地図つくりを体験していただきたいと、ご案内をしております。

  • グループに1枚、指令書を配布します。指令書に書かれたものをモリコロパーク内で探して、場所を地図にメモしてください。
  • みんなで集めたメモを元に、巨大地図作りを行います。

主催：OpenStreetMap 東海

時間 内容

10:40 – 13:20 モリコロパークの謎探し（モリコロパーク内を散策）（各自で昼食をおとりください）
グループ毎に、指令書を配布します。

13:20 – 14:20 謎解き地図を作ろう！（会場： 地球市民交流センター 体験学習室１・２にて）
Aチーム、Bチームに分かれて地図作りをします。
　
* Aチーム：巨大地図づくり→だれでもOK。

* Bチーム：オープンストリートマップ編集
　　→OSM編集経験者（無線LAN対応PCは各自持参）
　参加者の集めてきた情報を分担して入力する。
