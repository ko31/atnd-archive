title 人間関係をラクにする！実践コーチング入門
catch
description

　
あなたは人間関係をラクにして
悩みを減らしたいと思いませんか？

良いコミュニケーションを取るための
コツを知りたいと思いませんか？

　
もし、答えがＹＥＳなら
このセミナーは
きっと、あなたの役に立ちます。
　

コーチングの基本スキルをベースに
心理学のエッセンスも取り入れた
最強のコミュニケーション術を
ワークなども交え、
楽しみながら身に付けて頂く、
基礎的な講座です。

　　

【プログラム】

◆そもそも、コミュニケーションとは何か
◆コミュニケーションに一番必要なこととは？
◆コーチングの基本スキルを身に付けよう！
　１）きく
　　　⇒　信頼関係を深める傾聴　etc
　２）承認する、ほめる
　　　⇒　ほめないでほめるテクニック　etc
　３）質問する
　　　⇒　初対面でも話を盛り上げる質問　etc
◆かゆいところに手が届く、心理テクニック

　

【対象】

　・人間関係を良好にし、ストレスを減らしたい方
　・人とのコミュニケーションに自信が無い方
　・友人や同僚と、より良い人間関係を構築したい方
　・パートナーや家族と、より良い関係を築きたい方
　・部下や後輩の育成に活用したい方
　・営業など、ビジネスに活用したい方
