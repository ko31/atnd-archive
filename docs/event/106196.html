「階段ランニングマン」リレームービー参加者募集。

★超久々にリレームービーを企画しました。

  • あの TickTockでお馴染みの「階段ランニングマン」のリレームービーを作る企画です。

  • 参加者の皆さんからご投稿頂いた「階段ランニングマン」のムービーを１本にまとめて凄い作品にしてしまおう！って、壮大な妄想を考えています。

  • 「階段ランニングマン」とは？

    • ランニングマンしながら階段を昇る（または降りる）やつです。
    • 参考 → 【TikTok】シャッフルダンス！階段を友達と楽しく登る♪part 2 【限定公開】

★曲は、「Impress」を使用します。

  • 参考 → #シャッフルダンス 踊ってみた【千日前晒され隊】

★参加方法

  • 1. 「階段ランニングマン」動画を撮影する
  • 2. 撮影した動画を、YouTube、DropBox、GoogleDrive などのファイル共有サイトにアップ
    • ※ YouTubeにアップする際は、「一般公開」か「限定公開」に設定願います。
  • 3. このページのアンケートに必要事項を記入して参加申し込みする
    • 【必要事項】
      • 1.おなまえ（ハンドル名）
      • 2.動画のアドレス

★制作条件

締め切り時点で、参加者が10名未満の場合、リレームービーの制作を中止します。

★参加ルール

  • 投稿は、お1人様１回限りとさせて頂きます。
  • １本の動画に何名映っていてもＯＫです。
  • 投稿頂いた動画について、こちらで不適切と判断したものは、採用を見送らせて頂きます。
    • 【不適切と判断する基準】
      • 不快な動画
      • 公序良俗に反する動画
      • 企画の意図に沿わない動画（ランニングマンしていない、階段を使っていない、など）
      • その他こちらが不適切と判断した動画

★注意事項

  • 撮影するとき、他の方や周囲に迷惑がかからない様、十分ご配慮・ご注意願います。
  • 基本的に Impress のサビの部分のみの映像のみでOK。多少長くても、短くても大丈夫です。
  • 再生順のご指定はできません。こちらで決めます。
  • 編集した動画は、こちらのチャンネルで公開します。
  • 制作した動画は、諸事情で削除する場合があります。
