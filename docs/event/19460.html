「第８回　鬼ゴッター大会」

スポーツ鬼ごっこは、正式には「鬼ゴッター」と言います。
英語表記では「onigotter」です。

この「onigotter」でワールドカップ開催を目指していこうとしているのが、一般社団法人鬼ごっこ協会です。鬼ごっこ協会は、昨年の6月18日に誕生して、現在は約20都道府県で鬼ごっこ大会を開催しています。

鬼ゴッタ―大会では、そんな鬼ゴッタ―をいろいろな方に体験してもらうために開催していて、そして協会としての公式大会でもあります。

【第10回】の大会では、特別な賞品を設けて開催する予定です。
