Eclipse技術勉強会@札幌
- EMF/Xtext 2nd -

Eclipse/Javaによる外部DSL技術であるXtextと、モデリング技術であるEMFに関する勉強会を開催します。

先日開催されたクラウド温泉2.0＠小樽では、F#やScalaによる言語内DSLに関してとても盛り上がりましたが、Eclipse/Javaによる外部DSL技術も学んで、さらにDSL技術カバレッジを上げましょう。

以下のような内容のデモを行います。

  • XtextによるDSL開発とグラマー定義
  • EMFモデリングファーストによるXtext DSL開発
  • (もしかしたらGraphitiの話も..？)

参加される方で、手を動かしたい方は、以下のものを準備お願いします。
(デモ中心なので、どれくらい手を動かせるかは未知数です)

  • Java 6
  • Eclipse Indigo RCP and RAP Developer
    • Xtext SDK
    • Ecore Tools SDK

なお、参加費用は会場費を参加者で折半ですが、500円程度の予定です。

このイベントは、数学勉強会＠札幌 百回記念特別勉強会「プログラマのための圏論の基礎」との姉妹企画です。同じ会場の午前と午後に続けて行われますので、ぜひ併せてご参加ください。
また、このイベントのあとに、姉妹企画と合わせて懇親会が行われます。
数学勉強会 ＆ Eclipse技術勉強会 ＠札幌 合同懇親会
こちらについても、併せてご参加ください。
