MMD愛知作業オフ会開催

お待たせしました！
昨年9月に開催されたMMD作業オフ会（旧MMD意見交流会）を今年もやります！

概要
みんなでパソコンを持ち寄ってワイワイ作業したり意見交流しようという会です。
別に作業しなくても「今まで会った事のない人と交流しよう」という参加でも構いません。

どんな事をするの？

  • 簡単に言えば
    1. パソコンを持参して作業するもよし
    2. 他の人の作業について聞くのもよし
    3. MMD以外の趣味の話をするもよし
    4. アドバイスを貰うもよし
    5. お茶してお菓子食べてまったりするもよし
  • つまり何でもアリです。
    MMD触った事ない人・見る専だって構いません！
    別ジャンルのPさんも大歓迎！
    (ボカロやUTAUのPさん、歌ってみた、踊ってみた、演奏してみた、作ってみたetcetc）

前回の作業オフ会については、 こちら を参考にしてください。

開催日時及び会場について

今回は会場を長時間借りていますので、途中参加・途中退場も可能です。
また、一時的に退場して戻ってくる事も可能です

日時
2017年5月6日(土曜日)午前9:00～午後5:00

午後5時には完全撤収となります。
遠方からの方も参加しやすいゴールデンウィーク終了の1日前です。

場所
"名古屋国際センター3F 第二研修室 "

名古屋国際センターINC
住所: 〒450-0001　名古屋市中村区那古野一丁目47番1号　
名古屋国際センタービル3階 第二研修室

アクセス

  • JR（新幹線）・名鉄・近鉄
    • 名古屋駅から徒歩10分程
  • 地下鉄
    • 名古屋市営地下鉄「国際センター駅」から連絡通路で直通できます。

施設について

  • 今回お借りする会場は、FreeWi-Fiが利用できます。但し、大人数での接続の為、安定しない可能性もありますので、予めご了承ください。
    優先LANも一箇所だけ使用できますので、どうしてもの場合にはそちらを使って頂いても構いませんが、順番にご利用ください。
    （名古屋弁で言うところの「もーやっこ」でお願いします）

会場での注意事項

  • 飲食
    • アルコール以外の飲食物の持ち込みは可能です。また、地下に小さなコンビニもあります。建物周辺にもコンビニや吉野家等の飲食店もあります。
      但し、ゴミは各自での持ち帰りにご協力ください。
  • 喫煙
    • 会場のある建物内は全館禁煙ですが、地下に喫煙コーナーがありますので、そちらをご利用ください。
  • 駐車場
    • 国際センターには地下駐車場がありますが、25分250円と高額です。出来るだけ公共交通機関をご利用ください。万が一お車でお越しの方は、近隣のコインパーキングが便利です。

お願い
作業オフ会という性質上、パソコンを利用する方がほとんどだと思われますが、電源コンセントが5箇所しかございません。電源タップ等を持ち寄って頂くようお願いいたします。
みなさんのご協力で楽しい会にしましょう。

一応定員45人としていますが、変更になる場合もありますので、予めご了承ください。

何かご質問があればペパクラ(@hbkuma)まで
