概要

Spark+AI Summit Europe 2019で発表があったSparkの最新開発状況，ユーザからのユースケース報告，関連OSSであるKoalas/MLflow/Delta Lakeなどに関する最新情報をお伝えします．

  • Spark+AI Summit Europe - イベント内容: https://databricks.com/sparkaisummit/europe
  • Koalas - Spark用のPandas DataFrame APIs: https://pypi.org/project/koalas
  • MLflow - 機械学習によるデータ分析のライフサイクル管理ツール: https://pypi.org/project/mlflow
  • Delta Lake - データレイクを実現するためのSpark用のデータソース: https://delta.io

※ イベント参加はconnpassからお願いします．

会場

NTT Software Innovation Center，〒108-0023 東京都港区芝浦 3-4-1 グランパークタワー34F

  • 当日17時までに，入室用のQRコードをお送りします．connpassからの通知メールを，お手持ちのスマートフォンで受信できるよう，ご準備ください．
  • 万が一届かなかった場合はconnpassの受付票をスタッフにご提示ください.
  • 21時以降は入館できません．
  • 会場内での飲食は自由です（懇親会で軽い飲み物/お菓子は提供されますが、夕食はでません）
