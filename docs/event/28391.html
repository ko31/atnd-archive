#残パン会

イベント ユーザー
event_id 28391
title #残パン会
catch 残念言語ハッカソン＆貧民パンの会
description

趣旨：もそもそと貧民パンを食べながら、残念言語でハッカソンをする会です。

貧民パンとは：画像参照。基本的に安くてボリュームがあってカロリー多いシャレオツ感皆無なパンを指す。

残念言語とは：素晴しい利点はあるのに、流行や採用プロジェクトに運（？）が無くて今ひとつ流行らず、その結果ライブラリやドキュメントが揃わない比較的知名度や利用率が低い言語。例として次の言語を挙げる。
HSP、VBA、なでしこ、ドリトル、Brainf**k, LazyK, TeX, Objective COBOL, Whitespace, Refal, Io, Pietなどなど。
残念さの例：ドリトルで二分探索

※残念言語で何か作るだけではなく、残念言語を作るのも大歓迎です。
※よくある質問：
Q.「C++/Perl/JavaScript/Haskellは残念言語ですか？」
A.「お前が残念言語だと思うなら残念言語なんだろうな、お前の中ではな」

日時：5/3、12時（これ以降ならいつ来てもOK）から19時頃まで。

スケジュール：13時半頃から自己紹介と作る予定のものを簡単に説明。
18時から制作物のお披露目会をします。
一人10分程度のデモをし、もっとも残念なもの（定義は各位の心のままに）を作った方に、
参加者からの残念な拍手と余った残念な貧民パンを賞品として差し上げます。
※余らなかったら残念ながら諦めてください。

持ち物：開発するための何か残念なもの

参加費：貧民パンを購入するため、社会人は300～500円程度（参加数により可変）のカンパをお願いします。
学生は残念ながら無料です。

提供：場所、電源、水、お湯、トイレ
※電波無し、Ust予定無し
※皆様には床に新聞を敷いてコーディングして頂く予定ですが、
腰痛持ちなどの場合は残念ながら椅子と机をご提供します。
