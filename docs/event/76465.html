大好評！だった「ビジネスマンも学ぶ仏教」に続いて、
今回は、「三枚の鏡」の話です！

「鏡」とは自分を映すものです。
皆さんも、鏡を見ない日はないでしょう。
「鏡を見る」とは、鏡に映った自分の姿を見るのです。

先哲は、まず「汝自身を知れ」と教えられます。

なんだか哲学的で難しいなと思いますが、

新年度の４月を向え、新生活、新しい職場、新しい上司、同僚など　新しい環境で、
私は周囲からどのように思われるだろうか、心配になったり、不安になっていませんか。

それだけ、気になる、知りたいのが、自分自身、意外とすごく身近な問題ですね。

その自己を映し出す（知る）鏡に三枚あるのです。
また鏡は正しく自己を映すものでなけばなりません。

では、正しく映し出す鏡は、いったいどんな鏡でしょうか。

新年度をガラッと明るく、楽しくスタートしたい人には、必聴です。

感想

鏡の話面白かったです。

自分を知るためには、自分を正しく知ることとお聞きしてなるほどと感じました。

自分の鏡だけではなく、他人鏡、法鏡などもあることを初めて知り勉強になりました。

どう生きるか考えるヒントになりました。

ありがとうございます。

（20代・女性）

自分が本当の幸せになりたいのならば何をしたら幸せと感じるのか、

自分の姿をよくよく見ていかなければいけないと知らされました。
