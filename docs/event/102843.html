ポケカ新年会

2019年のポケカ初めに、ポケカ新年会を開催します！

ロータスに集まってポケカの新年を迎えましょう(　ﾟдﾟ)

参加者全員に参加賞＋豪華な景品やお年玉オリパも多数用意していますので、ぜひぜひ参加して下さいね！

新年もヤルカ！ポケカ！

◆会場の様子
GAME BAR LOTUS

イベント概要

当日はお年玉オリパが貰えるフリーバトルを開催！

イベントオーガナイザーのロータススタッフによるポケカ相談や、じゃんけん大会、ビンゴ大会など楽しいイベントがいっぱいです！

主催のあしゅん氏がお年玉オリパや、豪華景品を用意しています..！年始の運試しにもぜひご参加下さい！

「今からポケモンカードを始めたい！」「ポケモンカードを遊ぶ仲間が欲しい！」という方もぜひぜひ参加して下さいね。初心者の方は、テキストやルールでわからない事や困ったことがあれば遠慮なく周りスタッフに声をかけて下さい。ゼンリョクでサポートします！

経験者の方も初心者の気持ちに帰って、和気あいあいと楽しく遊びましょう！
