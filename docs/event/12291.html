これからの家を考える食事会

title これからの家を考える食事会
catch 2011.02.04＠稲村ケ崎
description

これからの「家」を考える。

価値観の多様な時代。
働き方、生き方も多様化してきた。
その一方で家族の在り方や地域コミュニティの価値などが見直されている。
これから、新しい「家」をつくることになる世代。
そんな私たちで、これからの「家のかたち」を描いてみませんか？

○開催日： 2011年2月4日（金）
○時間： 12:00-15:00
　　　 （12:00に稲村ケ崎駅（江ノ島電鉄線）集合）
○参加費： 無料
○定員： 15名
　　　 （申し込み者が多い場合は、申し込み順にてご参加頂きます。）

○スケジュール：
　・field trip※1
　・lunch time※2
　・ice break
　・work time
　・share time

※1 稲村ケ崎の古民家の内覧ツアーを行います。
※2 昼食場所は 稲村ケ崎食堂 を予定しております。

★参加にあたってのお願い
当日までに、以下の○○についてあなたのアイデアを考えてきてください。
「もしも私がシェアハウスに住むなら、大切にしたいことは○○です」

【主催】
慶應義塾大学大学院政策・メディア研究科
社会イノベーターコース　鎌倉シェアハウスプロジェクト
（田向、森住、橋本、林、石川、川崎、丸野）

鎌倉シェアハウスプロジェクトは、多様化するライフスタイルに対応した
新しい暮らしのかたちについて考えることを目的としたプロジェクトです。
生き方も働き方も多様化する時代、これからの家や家族、コミュニティ
の在り方について、「シェアハウス」を軸にさまざまな可能性を探ります。
なお、本プロジェクトは、 鎌倉古民家バンクプロジェクト のご協力のもと活動を展開しております。
