みなさん、
こんな会議ありませんでしたか？

・最初から結論が決まっている会議
・偉い人がずーっと話し続ける会議
・同じ議論がひたすら繰り返される会議
・いつ終わるか分からないエンドレスな会議
・何のためにやっているか分からない会議
・だれも発言しない会議
・形式的すぎて非生産的な会議

どうですか？
いくつか経験したことがあるのではないでしょうか？

はっきり言って、
こういった会議は無意味です。

なにも新しいものは生まれません。

クリエイティブさのかけらもない。
時間とエネルギーの浪費です。

しかし一方で、

「もっとクリエイティブな雰囲気がいいな〜」

「もっと斬新なアイデアが生まれないかな〜」

「もっとワクワクするようなミーティングがしたいな〜」

という気持ちもやっぱりありますよね。

こういったものが必要だけど、どうしたらいいかわからない。
というのが現状ですよね。

『ワールドカフェ』

ワールドカフェは、組織変革、まちづくり、教育、コミュニティビルディングなど様々な場面で活用されている対話の手法です。

カフェみたいな場所の方が会議室よりいいアイデアでるよね〜！
ってことでこんな名前がついてます。

具体的にどのような手法なのかと言うと、

1. 4〜5人くらいのグループをいくつかつくる
2. 話すテーマを提示する
3. グループ内で自由に対話させる
4. 20〜30分くらいで区切り、グループを解散させる
5. バラバラに移動し、また4〜5人くらいのグループでまとまる
6. またグループ内で自由に対話させる
7. 3回くらいこれを繰り返す

かなりおおざっぱですがこんなプロセスです。
（なかなかイメージがわかないと思います。笑）

このワールドカフェのメリットは、

1. 自分の意見が尊重される
2. 新しいアイデアが生まれる
3. クリエイティブな雰囲気

まず自分の意見が尊重されます。
ふつうの会議だと「お前の意見はクソだあ！」みたいなことを言われますが、
そういう風にならないような仕組み作りをしています。

そして対話を通して新しいアイデアが生まれます。
他の人の話しを聞くなかでひらめいたり、
自分が話すうちに突然ひらめきます。
そういった対話のプロセス作りをします。

そしてクリエイティブな雰囲気です。
「これって会議？」と思っちゃうようなリラックスした雰囲気で会議をします。
だってみんな飲み会とか、コーヒー飲みながら雑談してるときの方がいい発言しますもんね。

【実際の参加者の声】

「意見を出して話し合うのは良い。大きいグループで話し合うよりも、これくらいの人数で話し合うのがベストだと思う。」

「話し始めて私ってこういう考えをしていたんだなあ〜と知り、自分の考えを出すのっていいことだと思いました。これからもこういう場を作っていきたいと思いました。」

「意外と話さない人と話してみると様々な話がでて、面白い話も聞けたところがよかったです」

「このディスカッションで知れたこと、新しい自分の中に無い意見を聞いて考えたことも多かったので、もっとこんな機会を増やしたら面白いと思いました。」

「正直な感想としては様々な人の意見、考えを聞くことができてとてもよかった。」

「話したこと無い人と話すと様々な意見が聞ける。自分の気づかないことに気づく。」

「自分の意見は普段言えないけれど、今回のような小さなグループでなら言いやすく、集中できて良かったと思います。」

「各々の考えが違うことに楽しさを感じた。」

これは僕が以前に60人くらいの団体で
ワールドカフェした時の参加者の感想です。

今まで形式的な議論の場しかなかったため、
かなりみなさん満足した顔でした。

ワールドカフェはこんな方に向いています。

・かなり大規模な組織を動かしている方
・ベンチャー企業などでとにかく斬新なアイデアが欲しい方
・ボトムアップ型の組織運営がしたい方
・組織のメンバー全員の本音が知りたい方
・会議のクオリティを上げたい方
・大きな学生団体を運営している方
・いまの会議にウンザリな方
