参加対象者

セミナー内容・講演

グローバル原価管理と収益性シミュレーションの実践

円安、アメリカ金融引き締め、国内設備投資動向の変化、日々変化する外部環境の下、社内にある資産を活用しながらも、その時点で最も効率の良い生産方法を決定する事は至難の業であるかのように思えます。しかしながら企業活動の結果である営業キャッシュフローに着目すると、意外にもシンプルな構図が見えてきます。本セミナーではグローバルに事業展開されている製造業様に向け「 理論 」だけではなく最適地生産シミュレーションの具体的な「 実践方法 」まで踏み込んでご紹介します。

講師：EY アドバイザリー株式会社 / シニアアドバイザー / 伊藤 亮様

情報の可視化を実現する高速データ集積基盤と情報の利活用

オープンストリームは、お客様の収益構造の可視化を促進する為のシステム構築サービスを提供しています。本セミナーでは、M-OLAPアーキテクチャを搭載したBIツール「 Jedox 」を活用し、グローバル企業における予算・実績管理システムの構築事例をご紹介します。予実管理における「 計画策定 」「 実績集計 」「 予実差異分析 」サイクルの高速化により問題点の発見を早め、適切な「判断」を導く事が可能となった事例をご紹介します。

講師：株式会社オープンストリーム / ビジネスソリューション部　アプリケーションスペシャリスト / 田口 直志様

情報の可視化を実現する高速データ集積基盤と情報の利活用

ICT の発達に伴い企業における情報戦略手法は進化しています。一方で企業が保有する情報システムや情報はグローバルに多様化が進み、その情報量は爆発的に増加し、サイロ化した情報の管理手法が問われている時代です。ビッグデータを含む膨大な情報を高速に集積し、一気通貫した情報の可視化や利活用を実現する情報基盤の導入手法を BI・ETL ツール「 Pentaho 」を用いてご紹介します。

講師：テクマトリックス株式会社 / カスタマーソリューション営業部　主任 / 佐藤 大悟様
