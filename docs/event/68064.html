description

DMM.comは会員数1500万人を超える急成長を見せており、その中でもDMMオンラインゲームの会員数は昨年比204%と急速な拡大をしています。
今回開催する【DMM.Study Night】では急拡大してきたDMMオンラインゲームの開発体制から失敗談まで弊社エンジニアが経験を、そしてDMMオンラインゲームが導入するログについてお話致します。

■講演内容

DMM.com Labo ゲーム開発の舞台裏

ほぼゼロの状態からプラットフォームを作り上げ、さらには自社のゲームタイトル開発まで行うDMM.com。今回は、現在のDMMオンラインゲームの状況から開発体制や扱う技術、これまで起こった技術的失敗についてなどDMMのゲーム開発の裏側についてわかりやすく解説します。

CDH5によるゲームログ解析

DMMオンラインゲームで開発を進めてるログ収集に関してお話します。
DMMオンラインゲームでは、これまではプロジェクトごとに行っていたログ収集方法を統一し、プロジェクトに依存しない基本KPI機能とゲーム独自のKPI機能を実装するためのフレームワークを開発しました。
今回はそちらの事例をもとにご紹介します。


■講師プロフィール

西川健一/ゲーム開発本部 第一システム部　サブマネージャー

コンシューマゲーム開発、Webサービス開発などを経て、2010年に某ソーシャルゲーム会社に転職し、当時100万人を超えるユーザーを持つソーシャルゲームの開発運営を行いました。2013年11月にDMM.comラボに転職をし、組織が拡大されつつあるタイミングにあったゲームシステム部に配属され、組織運営とチーム運営に関わる仕事全般をやっています。

鈴木誠知/ゲーム開発本部 第二システム部

組み込み系、コンシューマゲーム開発を経て、2013年8月にDMM.comラボに入社。DMM.comラボにおいて、サーバーエンジニアとして1年半のソーシャルゲーム開発、運営を経験した後、内製プロジェクトのKPIシステムを担当しました。現在は、新規ネイティブアプリ開発のクライアントプログラムをメインに担当しています。
