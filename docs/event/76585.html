　オープンCAE（OpenFOAM，Salome-mecaなど）に関心を持つ技術者，研究者，学生などが集まり，気軽に情報交換・相談する場所を作ることを目的に，勉強会を開催します。
　すでに使用経験のある方はもちろん，興味はあるけれど使い方が分らないという方も大歓迎です。お気軽にご参加ください。

今回は，オープンソースのCADソフトウェア FreeCAD の入門講習会を開催します。はじめてFreeCADに触れる人を対象として，実際にソフトウェアを操作しながら，基本的な使い方を学んでいきます。［昨年1月12日(土)に開催した講習会と同等です。］

　今回も，勉強会終了後に，有志による交流会を開催します。交流会では，富山の名産 ますの寿司 食べ比べを開催します。
