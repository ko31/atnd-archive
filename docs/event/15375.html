会場

　長崎市立図書館研修室3、4。

　建物内の研修室の場所が少しわかりにくいです。 長崎市立図書館 Nagasaki City Library 図書館について に図がありますが、正面入り口から入場後、1階図書室へ入らずそのまま右へ折れ多目的ホール脇の階段を登ってください。二階の左奥に会場の研修室があります。

　座席数には余裕がありますので、当日参加も可能です。

備考

　「ひとりぎっと」は初級者向けGitハンズオンです。Gitやバージョン管理システムの経験の有無は問いません。むしろこれをきっかけに始めてみてはいかがでしょうか。そういう方にも参加していただけると嬉しいです。

　ノートPCなどをお持ちの方は是非持参ください。その際、無線LANアダプタ、ssh接続可能なコンソールがあるとより楽しめます。Mac OS Xではターミナル、Windowsではputtyなど。Gitのインストールは必要ありません。
　見てるだけ、他の人との共有での参加ももちろん可能です。

　終了後、この為のというわけではないですが、懇親会を開きたいと思っています。
　参加は自由で、場所や費用なども当日の状況により決めたいと思いますが、できるだけ気楽で安く上がるお店をと考えています。この会と直接関係がないので懇親会だけ参加でもOKです。
　この懇親会の前にカフェでお茶するも多分あります。

　懇親会の会場を決定しました。
　「誠」という居酒屋さんで行いたいと思います。住所は 長崎市興善町７−２ です。
　時間は午後6時くらいからを予定しています。懇親会のみをご予定の方は直接お越しくださっても結構です。

　ご質問などありましたらお気軽にお尋ねください。
