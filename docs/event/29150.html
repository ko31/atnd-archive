description

　 ＮＴＴドコモなど携帯3社のエコ検定対策公式サイトの運用などで環境教育に取り組んでいる環境教育フォーラム21は7月実施の第12回eco検定(エコ検定)合格に向けて、まる一日の学習で必要な知識を習得する「重要ポイント集中講座」を全国各地で開催します。また、お仕事でお忙しく丸一日は難しいという皆様に「eco検定帰宅前の重要ポイント講座」を東京・新宿（6月8日・金、13日・水）、大阪市（6月15日・金）で開催します。

　今回から公式テキストが第3版に変わります。eco検定は問題の8割以上は公式テキストから出題されています。第3版は2011年に起きた東日本大震災、福島原発事故、京都議定書の実質的な離脱など環境問題めぐる大きな変化を反映させており、第2版での学習に比べると、時事的な知識も要求されます。「重要ポイント集中講座」では、改訂された公式テキストの重要ポイントを中心に、時事ニュース解説なども含めて、合格に必要な知識、学習のポイントを習得していただきます。

　eco初心者の方でもわかりやすい指導が特徴で、eco検定を取得する利点・目的から始まり、eco検定の概要と学習方法、eco検定に出題されやすい最新の環境ニュースなどもご用意、最近のエコの話題も交えながらご指導いたします。講義は大学や自治体、企業などで指導で豊富な実績があるベテラン講師があたります。第11回の検定では約200人が参加されましたが、ほとんどどの方が合格しています。

　セミナーや学習方法については受講者のご意見を参考にしてください。

　また、環境教育フォーラム21では5月11日、元国連大学副学長の安井至氏、東京都環境局送料削減課長の荒田有紀氏を講師に、21世紀環境塾公開セミナー「日本の地球温暖化対策の行方」を開催します。温暖化問題はeco検定の重要ポイントのひとつです。無料のセミナーですので、ご都合がつく方はぜひ、ご参加ください。詳細、お申し込みは　http://eco.newsmedia.jp/2012/04/21-1.html
