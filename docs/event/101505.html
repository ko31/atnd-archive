「時代変化を読み解き20代で市場価値を高めるキャリア選択を」

セプテーニグループはキャリア支援セミナーを
10/26(金)に札幌で開催致します！

■イベント内容
「20代で市場価値を高める思考法を身に着ける」

▼「市場価値を意識したキャリア形成の準備」
新しい産業構造に適した雇用システムへの変革が迫られる中　
市場価値を意識してキャリアを形成してゆくことがより重要な時代に突入しています。　
その変化の本質をつかみ、今後どんな準備をしてゆけばよいかを考えます。

▼「時代変化の本質的な理解」
本質的な理解をするために過去から現在にかけての経済史、就職史を振り返りながら　
世界と日本で、どのような変化が起きているかを理解してゆきます。

▼「市場価値を高めるためのキャリア選択」
テクノロジーの発達により、大量のデータが取得できるようになる中で
人材育成や才能を発揮するために必要なことについても
これまでの常識を覆すような結果の最新状況をお伝えし、
今後のキャリアの考え方、20代で市場価値をあげるための思考法についてお話します。

※登壇者は、国家資格であるキャリアコンサルタントの資格を持つ社員です

※参加特典※
「パーソナリティ×ビッグデータ」を活用した
