Network Forensics Puzzle Contest に挑戦

「ネットワーク パケットを読む会(仮)」では Network Forensics Puzzle Contest のPuzzle に挑戦してみたいと思います。これはオンライン上で開催されているパケット解析コンテストで、Puzzle#1 から Puzzle#10 まで出題されています。
今回はまず最初から順に挑戦しよう、という事で第1回の出題「Puzzle #1: Ann’s Bad AIM」の解析を試します。参加される方は、可能な範囲で出されている問題の解析を行ってください。なおこの Puzzle は過去の出題ですので、既に正解が公開されています。勉強会では「どのような手法で解析できるか」を検討していきます。

解析手法とその結果を当日発表できる方は、参加申込時のアンケートでお申し出いただくか、または後からコメントでお知らせください。ぜひふるってご応募ください

以下に、出題の概要を掲載しておきます。

Ann が秘密のレシピを社外に持ち出そうとしているのではないかと考えたセキュリティ部門は Ann のネットワーク トラフィックをキャプチャしていました。すると Ann のコンピュータから無線 LAN 経由で不審なコンピュータ宛てにインスタント メッセージ (IM) が送信されている事がわかりました。その内容を解析し、Ann が送信した情報について以下の設問に答えてください。なお Ann の PC の IP アドレスは 192.168.1.158 です。


  1. Ann の IM の相手の名前は?

  2. IM のやりとりでの最初のコメントは何?

  3. Ann が送信した物の名前は何?

  4. 抽出できるファイルのマジック ナンバー (先頭の 4バイト) は何?

  5. ファイルの MD5sum は何?

  6. 秘密のレシピとは何?

証拠のパケット (evidence.pcap) は以下からダウンロードできます
http://philosecurity.org/558/contest_01/evidence.pcap
MD5 (evidence.pcap) = d187d77e18c84f6d72f5845edca833f5
