ゲヒルンウェブセキュリティ勉強会　第２回　in京都

ゲヒルン株式会社は、2012年12月18日に京都（四条烏丸）のコワーキング・スペース「小脇（こわき）」にて「ゲヒルンウェブセキュリティ勉強会　第２回　in京都」を開催します。

この勉強会では、主にWebサービスの開発を始めたばかりの方や、スタートアップ企業で自社サービスのセキュリティをより強化したいエンジニアの方、または最近VPSに触り始めた方など初級〜中級者を対象としていますが、どなたでもご参加いただけます。

講座内容

  • これまでの脅威とこれからの脅威
  • コンテンツスニッフィング
  • JavaScriptの話

※ 講座内容は変更する場合がございます。

タイムスケジュール

19：00〜19：30　Webアプリケーションによく見られる脆弱性について。
19：30〜19：45　実際に「ダミーサイト」を攻撃してみよう。
19：45〜20：00　質問タイム
20：00〜20：20　「Gehirn Web Services」の紹介
20：20〜20：50　ライトニングトーク枠（発表希望の方はcontact@gehirn.co.jpまでご連絡ください。）

場所・日時

  • 2012年12月18日（火）
  • 19時〜21時（2時間）
  • コワーキング・スペース「小脇（こわき）」http://cowaki.com/
  • 参加費：無料

講演者の紹介

  • ゲヒルン株式会社　代表取締役　石森大貴
  • Twitter：@isidai
  • 平成２年生まれの２２歳。幼稚園時代からワープロに触れ、キー入力をマスター。小学３年からプログラミングを始め、小学６年から独自にサーバーを作り始める。高校時代に経済産業省の主催する「セキュリティキャンプ」に参加。
    　その後、セキュリティキャンプの講師補助を２年間務め、ＴＢＳが放送したドラマ「ブラッディ・マンデイ」のハッキングシーンの技術考察記事で話題を集める。
    筑波大学へ進学していたが、休学し独立、起業した。

その他・注意点等

ノートPCをお持ちの方は是非お持ち寄りください。
なお、会場では無料のWi-Fiが提供されていますが、公衆無線LANとなります。

また、ホスティング会社や情報セキュリティ会社等の同業者の方の参加はご遠慮くださいますようお願い申しあげます。
