【英語で学べるプログラミングの魅力 HTML/CSS講座】

「英語 x IT x 会計」

上記のスキルが、必要不可欠と言われている中、結構日本人は
挫折している印象です。

「英語はできるけど、ITが…」
「プログラミングは得意なんだけど、英語は、どうも…」

英語もプログラミング言語を同時に学びたいあるいは、どちらかの技能を
学びたい方いずれの方もとても魅力的な授業となっております。

では、２つのスキルを同時に、効率的に学びましょう！

英語とITを一気に学べる。
それが私たちが開催している講座 「英語で学ぶWebサイト制作講座」です。

【取得できる技能】

・英会話力（トイック７００点レベル）
・HTML / CSS / Javascript etc..
・開発手法
・CMSを用いたゼロから製作する方法

オンライン通信サービスの「 Progate & Codecademy」を用い、
プログラミングの導入としてHTML/CSS講座を行います。

授業は中学生レベルの英語と日本語による解説を織り交ぜて行います。

英語のヒアリングに自信がない方でも、

具体的な画面つきで「日本語補足あり」操作方法を説明しますので、ご理解いただけると思います。
また、英語は得意だけれども、 プログラミング経験がないという方は、
HTML/CSSに集中して学ぶことができます。

さらに、現役プログラマー（CMS製作部署）が監修としてフォロー。
日本語での質問も受け付けますので、ご安心ください。
