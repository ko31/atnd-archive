Webアプリ開発の最新動向について

  • by アドビ システムズ 株式会社 / 轟 啓介 氏

このセッションではWebを進化させるためのアドビの取り組みについて発表します。
下記のようなWebデザイナーやWeb開発者のための新しいツール、技術、サービスについてもデモを交えながら説明します。
Edge Animate/CreateJS/Brackets/PhoneGap/etc…

Flash Platformの最新動向について

  • by AKABANA 有川 榮一 氏 twitter: @akb7_jp
  • http://akabana.info

Flash/AIRの最新動向について説明します。
公開されたGame Developer Tools についても紹介します。

モバイルアプリ開発を始めよう

  • by AKABANA 有川 榮一 氏 twitter: @akb7_jp
  • http://akabana.info

Adobe AIRやPhoneGapを使ったiOS/Android向けのモバイルアプリ開発の始め方について発表します。
(PhoneGapとは、HTML5/JavaScript/CSS3を使ってスマートフォン向けのハイブリッドアプリケーション制作のためのフレームワークです。)

Amazon Web ServiceでWebサービス開発を始めよう

  • by 株式会社プレイウェル　成田 裕輔　氏
    Amazon Web ServiceのEC2やS3などを使ったクラウド上のWebサービス開発について分かりやすくご説明します。

マルチデバイス対応について

  • by 合同会社デジフィ 吉田 和隆 氏 twitter: @mousepancyo
  • http://www.digifie.jp/
    AIRを使ったモバイルアプリにおいて、マルチデバイス対応する際のポイントについて発表します。

Flashにおける画像処理の最新動向

  • by 株式会社ICS 池田 泰延 氏
    JPEG-XRの採用やBitmapDataの強化やActionScript Workerの搭載によりFlashにおける画像処理の可能性について紹介します。
    (ActionScript Workerとは、FlashPlayer上でマルチスレッド処理の機構です。)

モバイル(iOS/Andorid)向け動画配信について

  • by AKABANA 有川 榮一 氏 twitter: @akb7_jp
  • http://akabana.info
    Flash Media Serverを使ったiOSやAndroidなどのモバイル向け動画配信について発表します。
