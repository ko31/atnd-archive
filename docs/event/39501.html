概要

一般社団法人LOCALが設立されてから、任意団体の期間を含めて５年がたちました。
そこでささやかではありますが、５周年を記念してのパーティーを開催したいと思います。

肩肘の張らない気楽な宴会ですので、

  • LOCALに少しでも縁のある皆さん
  • 今までLOCALが関わってきたイベントや勉強会に参加した皆さん
  • LOCALと一緒にスタッフなどイベント運営をやってきた皆さん
  • LOCALの活動などにほんの少しでも興味のある皆さん

ぜひご参加ください。

いつも昼間のイベント・スタッフ業務で宴会ではすっかりもぬけの殻になっている面々も、この日は全力で懇親しますので、お楽しみに！
