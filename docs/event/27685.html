開催概要

創成の学生で集まってLightning Talkをする会です！
(参加希望があれば、他学類生でもOK！)

・話す内容は創成っぽいことなら何でもおk

・聞くだけでも全然OK !

※LTとは…？
カンファレンスやフォーラムなどで行われる短いプレゼンテーションのこと。
１つのテーマを決めて、だいたい５分くらいで話すこと。

今回のLTのテーマ

『新入生に優しく楽しく』

web系,映像系,デザイン系…いろいろとジャンルはありますが
何か新しいことを始めるきっかけとなるようなLTを今回は募集します。
とは言っても、難しく考えなくて全然大丈夫です、好きなことを楽しく話しましょうw
