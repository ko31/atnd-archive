description

MikuMikuDanceの講座と称した、割とゆるい集まりです。
MMDで動画を作るのに、知っておいたらお得な豆知識的なあれになりそうな、そんな予感。
※まったくもって上級者向けではありません。どちらかと言えば初心者よりちょっとできる～中級者手前くらいの心持ち。会議室借りてるからってちゃんとした勉強会とかじゃないんだからね！！勘違いしないでよね！！

※申し訳ありませんが、参加費がかかります（会場代です）。お一人様￥1000程度、現地にて回収予定。
※人数を入れるために机をとっぱらって椅子だけにします。なお、PC等の持ち込みは自由ですが、電源は各自ご用意ください。
※飲食はペットボトルの飲み物程度まで（会場を汚さないように！）。
※参加キャンセルも可能ですが、なるべく早い段階でお願いします（キャンセル者が出ると自動的に補欠の方が繰り上がるシステムになっていますので、とりあえず申し込むのもOKです）。
※名刺があるといいかもしれません。

ーーーー全体の流れーーーーーー

※１３時より前は部屋が空いていません。１３時より前の会場待機はご遠慮ください。１３時以降に会場に着くよう調整お願いします。
