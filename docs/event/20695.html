 新たな時代を築く、最先端のリーダーシップ理論と実践
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

あなたは、低迷した今のこの状況に、ただ流されているだけで
いいですか？

まったく新しいスキルや考え方を身につけて、
自らが中心となってこの状況を打開し、
明るい未来を築き上げようとは思いませんか？

あなた自身が変革し、あなたの周りに影響を及ぼすことで、
ほんの小さな変化が、地域全体、日本全体、世界全体へと
範囲を広げ、大きなうねりとなるかもしれません。

日本人が世界に誇れる「和」のパワーを最大限にするため、
それぞれが最良のリーダーシップを発揮し、自分の強みを
最高に引き出す事が必要となっています。

ここで学べるのは、世界で実証されている、
人間の行動の心理学や脳科学に基づいた
最先端のリーダーシップ理論と実践スキルです。

世界の著名なリーダー達は知っている重要スキルです。

日本の未来を担う子供達のためにも、
多くの大人がこのスキルを身につけなければなりません。
一緒に成長し続ける仲間を増やし、お互いに切磋琢磨し、
子供達に伝える使命があなたにもあります。

志あるあなたのご参加お待ちしています。

■当塾の特徴

過去13回行われた「思考ヘンカク ワークショップ」
その内容は主に、パーソナルブランディングと
コミュニケーションでした。

今後はその要素を踏まえながらも、
よりリーダーシップを意識した内容としていきます。

社会や組織の中に生きていて、リーダーシップを必要と
しない分野はあり得ないと考えます。

会社や家庭、学校、友人関係・恋愛など、いたるところで
リーダーシップが必要になってきています。

リーダーシップという響きは、何か男っぽいイメージが
あるような気もしますが、今後の社会やビジネスでは、
今まで以上に女性のリーダーシップもかなり重要になって
くると思います。

当ワークショップは、9/13〜9/18に行われた世界№１コーチ
アンソニー・ロビンズが主催するリーダーシップセミナー
「LEADERSHIP ACADEMY BOOT CAMP」の内容と、
アンソニー・ロビンズ日本人初のパートナーと言われる
クリス岡崎のメソッドをベースに組み立てられています。

アンソニー・ロビンズは、ロナルドレーガン元米大統領、
クリントン元米大統領、ネルソン・マンデラ元南ア大統領、
故ダイアナ元妃、ダライラマ、ブリトニー・スピアーズ、
アンドレ・アガシ、セレーナ・ウィリアムズ等々・・・
世界のスーパースター達をコーチングした
まさに、世界№１のピーク・パフォーマンス・コーチです。

そんな世界№１のメソッドを体感できるチャンスは
それほど多くはないと思います。

あなたもこの機会に、最先端の心理学に基づいた
リーダーシップ理論と実践を体感してみてください。

次世代のリーダーを育てるリーダーになりたい。
そんな志を持っている人を私たちは全力でサポートします。

もちろん私たち自身もその様なリーダーを目指しています。

なお、このスキルを学ぶためには、1時間弱という限られた
セミナー枠では到底無理なことです。

出来る限りポイントを絞ってお話ししますが、
それでも最低10時間は必要です。

そのため、当ワークショップは10月以降も継続して実施します。

10月は毎週火曜日（4日、11日、18日）を予定しています。
※10月25日は都合によりお休みです。

11月も毎週火曜日（1日、8日、22日、29日）を予定しています。
※11月15日は都合によりお休みです。

みなさま、毎週火曜日の予定を空けておいてくださいね。
