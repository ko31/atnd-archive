Xojoとは

RAD（Rapid Application Development）ツールです。
かつてのVisualBasicのようにマウスでパーツを配置→コードを定義で開発、更にXojoではWindows/iOS/Web/RaspberryPi/Linuxの各アプリが開発できます。

今回ははじめての文系チャレンジャーを想定し、「寝ながら開発」できるレベルの簡単アプリを作ってみます。

予定

- 11:00 ゆるゆる集まる
- 11:30 ダウンロードしたりインストールしたりお皿に盛りつけたり
- 12:00 おいしく食べながらデスクトップアプリ作成
- 12:30 デザート食べながらWebアプリ作成
- 13:00 DBにアクセス
- 13:30 補習〜自由解散

注意事項

  • 勉強会では寝ません
  • 寝る代わりにおいしいもの食べながら開発します
  • おいしいもの持ち寄りです
  • PC持ってきてください！

株式会社XSHELLの社内勉強会として企画しましたが、おいしいものを持ってきてくれるなら誰でもWelcomeです！！

XSHELLとは

シングルボードコンピューターを使って、プロトタイピングから、プロダクションモデルまでをシームレスにつなぐ、IoT向けのPaaSを開発しています。
WEBアプリケーション技術、ハードウェア技術、3Dモデリング技術を組み合わせて、IoTの事業開発支援も行っています。
オフィスはレーザー加工機、3Dプリンタ、小型CN旋盤、オシロスコープなどがあって、ミニFablabみたいになってるよ！
