インドの高僧・スワミジによる講演会
『愛によって人生は変えられる』

2019 年10月27日（日）参加無料
開場：13 : 30/開演：14 : 00（予定終了時刻17 : 00）

講演会とフルート瞑想　〜南インドからお坊さんがやってくる!〜

どんな人も愛なしには生きられません。
自分の中にある愛をどうやって表し伝えていけばよいのでしょうか。
現代の私たちが忘れかけている「愛」
家族や友人への愛、仕事や環境への愛、そして自分自身への愛。
今を生きる私たちが本物の愛に再び気づくとき、人生は明るく輝き出します。

インドの聖者、抱きしめる聖者と言われる アンマのもとで修業を積まれたスワミ・プールナアムリタナンダジから人生を輝かしく、豊かにする「愛」のメッセージをうけとりましょう。

インドの竹笛の演奏＆めい想・質疑応答もあります！

毎回、素晴らしいお話をして頂いています！！是非ご参加下さい。
