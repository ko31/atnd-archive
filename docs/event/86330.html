マニフレックス大阪ショールームでは、質のよい眠りに関心のある女性のためのアカカツ（セミナー）“ボンジョルノ！マニフレックス”を開催しています。朝の時間を有効活用していただけるよう、朝7時30分にスタートする体験型セミナーで、参加費500円は全額寄付されます。おいしい軽食と飲み物をご用意して、皆様のお越しをお待ちしています。

マニフレックス朝活 2月プログラム

2017年3月24日（金）朝7:30～8:30

～KaQiLa式～美腸エクササイズ

昨年の開催で大好評だった新感覚エクササイズ“KaQiLa(カキラ)”が再登場！
KaQiLaとは、ろっ骨を動かすことで背骨をまっすぐに整え、身体機能を高めるエクササイズです。前回は基礎を行いましたが、今回は美腸を目的とした集中プログラムを行います。
腸が美しくなれば血の巡りがよくなり、冷え症が改善されるだけでなく、むくみや便秘など女性特有の悩みにも効果的！軽快な音楽とともに行う動きは無理のないものばかりなので、体が硬い方や運動が苦手な方でも気軽に参加OKです♪

講師：森 智子 先生
化粧品メーカーの美容部員勤務時に美は内面も大事であることに気付き、ボディメンテナンスの道に。快便セラピーやリンパエステなど幅広く活躍中！

参加費：500円（焼きたてクロワッサンと紅茶付き）

※要予約、女性限定
