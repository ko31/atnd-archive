description

ガーデンスローヨガサークルは毎週土曜日に、朝の偕楽園で公園ヨガを行っているサークルです。
今回は、夜に時間を変えての公園ヨガを企画します。
エアコンや照明を消して、キャンドルの灯火だけでヨガの呼吸をしてみましょう。

どなたでも自由に参加いただけます。※参加の注意事項と詳細地図については上記URLよりサークルブログを参照下さい。

日　時：2010年６月27日（日）18:00～19:00（雨天の場合には７月３日（土）に順延します。）

参加費：500円

場　所：偕楽園公園猩々梅林（地図は上記URLよりサークルブログを参照下さい）

持ち物：ヨガマット、マットの下に敷くレジャーシート、タオルや飲み物など

木々の揺らぎや虫の声、鳥たちのさえずり、季節の移り変わり。。。公園ヨガは私たちの周りにもあるミニマムな自然を実感させてくれます。
そんな世界がいつまでも続くように、私たちに出来る小さな一歩を。
