【パソコン初心者、プログラミング初心者歓迎】

■これから技術を身に付けて、プロのiPhoneアプリエンジニアとして活躍したい方のために、
池袋 iPhoneアプリ開発体験会を東京 東池袋にて開催いたします！

休みの日にゆっくりするのも大切ですが、将来のためにiPhoneアプリを作る技術を身に付けてみませんか？
今を頑張る人を応援します！

【仕事に直結するエンジニアを育てます】

こちらの講義で目指すのは、
・勉強してiPhoneアプリが作れるようになって良かった
・勉強して役にたって良かった
というものを目指しているのではありません
参加者の方を、iPhoneアプリを作る仕事のプロとして養成する事が目的です
iPhoneアプリのプロとして仕事をしていくようになるためには、沢山の本を読んだり
プロとして通用するようになるには、多くの事をおぼえ、長い時間がかかります

全ての事を覚えようと欲張るあまり、挫折してしまう人も多いのですが、
講師はプロのエンジニアとして５年間、企業向けのiPhoneアプリを作ってきました
仕事でプロとしてやっていくには、必要な技術というものがあります
講義では、Macを実際にみなさんに使ってもらいながら、仕事で必要となる技術を身に付くまで繰り返し教えていきます

プロのiPhoneアプリエンジニアになりたい方は、是非来てください
・iPhoneが好き
・iPhoneアプリを作れるプロとしての、あなたの武器が欲しい方
どんな方でもやる気がある方は、一人前のプロになるまでしっかりとサポート致します

【iPhoneアプリという需要の高いスキルを身に付ける】

iPhoneが世の中に浸透し、時代はiPhoneのアプリが作れる人材を求めています
英語を学んだり資格を取ったり、人により努力するものは様々ですが、せっかく多くの時間をかけて勉強するなら、
需要の高いものに時間をかけるべきです
東京都内でのiPhoneアプリを作れる人材を求めている企業の需要はとても高いです

自分で起業を考えているような人でもアプリの作り方の基礎くらいは知っておくのは役に立つ事でしょう

facebook創業者のザッカーバーグさんや、ライブドアの創業者の堀江さんもプログラミングの上級者です

【どんな方にもしっかりとわかりやすく、丁寧に指導致します】

体験会では他のスクールや勉強会のようにテキスト教材を見て大人数で勉強するようなスタイルではありません
企業向けのiPhoneアプリを作っている現役のエンジニアが、
1人1台ずつ、Macのパソコンをご用意して直接指導致します

今まで２００人以上の方に、わかりやすく丁寧にiPhoneアプリの作り方を教えています
仕事として必要になるものを中心に、基礎から実践レベルの内容まで一貫して教える事が出来ます

iPhoneのアプリを作るのは難しいイメージを持たれる方もおられると思いますが、どなたにでも出来るように丁寧にわかりやすく教えていきます

プログラミングが初めての方、iPhoneアプリを作るのが初めての方にも、
アプリが作れるようにしていきますので、どなたでも安心して来てください

【当スクールが選ばれる理由】

・マンツーマン形式に近い形での少人数制の個別指導

講義では、最大4人までの少人数制、マンツーマン形式に近い形での、直接指導を致します
他のスクールのように大人数での講義ではありませんので、1人1人丁寧に指導致します
大人数で教えるスクールでは、途中でついていけなくなったりする事も多いと思いますが
当スクールでは、途中でつまってしまった方にもわかりやすく丁寧にフォロー致します

・現役のプロのiPhoneアプリエンジニアが直接指導しています

現在も数多くの企業向けのiPhoneアプリを作っている現役のエンジニアが直接指導しています
現役のエンジニアとして仕事で得た内容に即して教えているため、仕事で必要となる実践的な内容を学ぶ事が出来ます

・1人1台使えるMacを用意しています

スクール備え付けのMacを、1人1台使えるようにしておりますので、Macをお持ちでない方でも
Macを実際に使いながら覚える事が出来ます

・東池袋駅徒歩５分という好立地

東池袋駅徒歩５分という便利さ、お仕事帰りでも、休日の予定の合間にでも、通うのに便利です

【こんな方に来てもらいたい】

・自分を成長させたい 成長意欲のある方

・プロのエンジニアとして活躍していきたい方

・iPhoneアプリ開発が初めての方、これからiPhoneでアプリを作ってみたい方

・Macを使うのが初めての方、プログラミングが初めての方

・Macを持ってるから、せっかくなら何かに活かしたい

・新しいことに挑戦してみたい人、自分を成長させたいと思っている人

・iPhoneアプリが作れるようになると技術が身につきますので、手に職がつくので女性の方にもオススメ☆
（特に女性のエンジニアはとても少ないので、貴重な存在です)

・プログラムというものが初めてだけど、挑戦してみたいと思っている方
→プログラムは難しく考えなくても大丈夫です。元ライブドアの堀江さんもプログラムは自転車に乗るようなものだとおっしゃっています

・swift、objective-c どちらもやってみたい方

・iPhoneアプリを発注する担当者の方や、起業を考えていてiPhoneアプリの作り方の基礎を学んでみたい方

【体験会の内容】

iPhoneアプリを作るのが初めての方向けに、iPhoneアプリの作り方を基礎からご説明します

・Xcodeの使い方

・iPhoneアプリの作り方

体験会で作成するiPhoneアプリのYouTube動画(横画面用)

体験会では、Macをみなさんに使っていただき、iPhoneアプリを作る基礎を学んでいただきます
プロのエンジニアになるための基礎を学んだのち、少しずつプロのエンジニアに近づくための講義を用意しています
最初は簡単なものから作っていきますので、初心者の方も安心して来てください
