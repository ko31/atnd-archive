夏休みはHaskellで遊ぼう！

Haskell入門者のための勉強会 スタートHaskell の 第1回 ATNDです。

スタートHaskellは、入門者から初心者の方を対象に、実際にコーディングをしながら１からHaskellを学んでいく勉強会です。本勉強会は、講義・演習・宿題のサイクルをベースにHaskellを学習するもので、月1回ペースで全5回を予定しています。講師にはトラビス・カードウェルさんお招きします。

第1回は プログラミングHaskell の1章から4章までを取り上げます。

Haskellに興味ある方、やってみたい方、ぜひ一緒にHaskellを始めましょう！

勉強会の最初に予習者の発表を行います。コメント欄にて発表希望者を受け付けています！

第1回の前に

  • 事前に Haskell Platform をインストールしておいてください。
  • プログラミングHaskell を購入して、１章から４章までを読んでおいてください。
  • プログラミングHaskell：補足 を読んでおいてください。
  • 事前に予習していただいた方に、勉強会冒頭で各章の発表していただきたいと思っています。発表してもいいよ！という方は、コメント欄で教えてください！（※発表は初心者やまだスタートHaskellで発表していない方を優先します）
