イベント概要

Drecom Tech Espresso は、株式会社ドリコムが主催・運営する、濃縮されたエスプレッソのような話が聞けて学べる 技術セミナーです。開催回ごとに異なるテーマで、毎回テーマに沿った登壇者、ゲストをお迎えし、ものづくりにおける取り組みや裏話など様々な角度からお話ししてもらいます。

今回のテーマ

エンジニアリングの現場が抱えている課題は技術的な課題だけではありません。 エンジニアリングの現場の課題として「リーダー不足」はどこの会社、現場も共通なのではないでしょうか。 「優秀なエンジニアはたくさんいるけど、誰にリーダーを任せていいのかわからない」 「いきなりリーダーになったけど、どうしたらいいか不安・・・」 リーダーを任せる側、任される側、どちらも様々な課題を抱えています。 今回は“How to be a Leader -リーダーに抜擢する時される時- “と題しまして、 ニューズピックス様、gumi様、VOYAGE GROUP様と弊社ドリコムの４社から 「リーダーを抜擢した経験がある方」、 「リーダーに抜擢された経験がある方」にお集まりいただき、

  • 初めてリーダーやった時ってこんなにダメだったけどこうやって解決したよ
  • リーダーってそんなに難しくないよ
  • こういう人に任せたいと思ってるよ
  • 抜擢する時の悩み
  • 抜擢の成功例、失敗例 というような事例をみなさんと共有しながら勉強していきたいと思います。 サービス開発/運営の裏側で起こっている普段聞けないような濃ゆい話が聞けるセミナーです！！
