講演内容

櫻庭 祐一

JavaFX 1.3 の新機能について

2010年 4月 23日にリリースされた JavaFX 1.3。JavaFX TV やパフォーマンスの向上、新しい UI Contorlなどさまざまな機能向上が図られています。本セッションでは、JavaFX 1.3 の新しい機能に関して、デモを交えて紹介していきます。

羽生田 恒永

NetBeans6.9 を使ってデータベース接続しよう

NetBeans 6.9 で本格的に搭載された JavaFX Composer を使って データベースに接続するアプリケーションを作成します。アプリケーションの完成までのステップを順に紹介します。
業務アプリケーションのプラットフォームとしてのJavaFXを垣間見れる…かも。

草薙 昭彦

JavaFX Mobile/TV – 3 スクリーンコンテンツ、はじめました

JavaFX のウリの 1 つは PC、携帯端末、テレビといった異なるプラットフォーム向けのコンテンツを単一の開発環境、ソースベースで作成できることです。これは Common Profile API を使用することによって実現できますが、今回は Mobile/TV 向けコンテンツを作成するときのお作法や、SDK に含まれるエミュレータ等、開発環境やランタイムの現状をご紹介したいと思います。

岡崎 隆之

iPhone/iPad 開発者から見た JavaFX

まだまだ未知数の可能性を秘めた JavaFX。iPhone/iPad 開発者の視点からみたビジネス上の価値、技術的な検討を紹介します。
