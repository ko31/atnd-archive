ハースストーン日本語化二周年を迎え、今年も残すところあと僅かとなりました。

今年最後の締めくくりとして、参加自由のハースストーン忘年会を開催致します。興味のある方はぜひぜひお越し下さい。

ハースストーンなど各々語り合いながら、おいしいお寿司で飲み食いしますよっ！

＊予約席数の都合上、先着順の受付とさせて頂きます。お早目の登録をお願い致します。

イベント詳細

日時：12月22日（金）20:00-
場所：すしざんまい道頓堀店
　　　大阪府大阪市中央区道頓堀１丁目７−２１ 中座くいだおれビル1F
会費：4000円（飲み放題付き）

忘年会終了後、希望者でゲームバー二次会を予定しています。一次会、二次会共にオフ会初参加の方も大歓迎です。ハースストーンが好きな方はどなたでも参加出来ますので、ぜひぜひご参加下さい。

二次会会場 ゲームバーpeegee

1時間500円の格安ゲームバーです。時間の許す限り遊びましょ！

注意事項

予約の関係上、万が一のキャンセルはお早めにご連絡をお願いします。年末は会場を抑えるのも大変なためご協力をお願い致します。

・会場wifiがありませんので、各自用意をお願い致します。
・予期せずサーバーエラー等、進行が正常に出来ない可能性もございます。
・貴重品の管理は自己責任です。十分に注意して管理してください。
・勧誘・ナンパなど他の参加者の迷惑となる行為は禁止です。
・未成年者の参加も可能ですが飲酒等はご遠慮下さい。
