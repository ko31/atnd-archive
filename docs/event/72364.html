クラウドでうごくLibreOffice Base

（株）インフォース新製品発表説明会

OSS LibreOfficeのBaseを改良 AccessでできないWEBスマホ開発・サーバー稼働を実現します。

Accessはとても素晴らしいソフトですが、WEB、スマホ開発ができない、サーバーで動かす仕様になっていない、という２つの大きな問題があります。

そこでオープンソースのオフィススイートである、LibreOffice(旧OpenOffice)のBaseを改良し、これを克服するものが作れないかトライしてみました。Accessの開発効率の良さにはかないませんが、WEB、スマホ開発の部分は優れていると思いますし、クエリーをつくる要領でストアド等も作れます。

まだベータ版ですが、ぜひ皆さまにご評価いただきたいので、もしよろしければ下記お越しいただければ幸いです。お時間ございましたら、ぜひよろしくお願いいたします！
