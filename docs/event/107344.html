ココネ社内では、Raspberry Pi が使われたシステムが日常的に動いており、社員の会社生活を便利にしています。（社内のエンジニアがつくりました！）

実際ココネで動いているシステムをエンジニアのサポートとともに1.5ｈで体験してみよう☆//

【必要な知識】
・ソフトウェアのインストールを行ったことがある
・ターミナル(コマンドプロンプト)での操作を行ったことがある

【こんな人にオススメ】
・普段 WEB やサーバーを触っているが、Raspberry Pi を触ったことがない方！
・MongoDB を作ったアプリケーションを作ってみたい方！
・HTML コーディングだけだったがサーバーサイドも経験してみたい！
・Node.js、 Python、 Raspberry Pi に興味がある！

☆ご参加いただける方には、7月23日までに会場の入館書の方を発行させていただくためのご連絡をいたします☆
