【Vol.6】Love Heroku？

Webアプリケーションの開発において、PaaS（Platform as a Service）が注目を集めています。PaaSの活用によって、サーバ/OS/プログラム言語/データベースなどの開発プラットフォームにリソースを割くことなく、エンジニアは今まで以上にアプリ開発に専念できるようになる。それはより魅力的なWebアプリケーションを生む土壌となるかもしれません。

そのPaaSの一つが今回取り上げる『Heroku』。日本初の国際規格プログラミング言語であるRubyを対象としたPaaSとしてスタートし、現在ではJava、Python、node.jsなど複数言語に対応するHerokuは、Rubyの作者・まつもとゆきひろ氏がチーフアーキテクトを務めていることでも知られています。そんなHerokuのポテンシャルと可能性について、Herokuエバンジェリストである相澤歩氏のプレゼンテーションと、実際のサービスにおいて活用した実績を持つ二社の事例紹介を通じて、探っていきます。

プレゼンター

●Heroku, Inc.／相澤 歩

Heroku, Inc.　Developer Marketing / Technical Account Manager
外資系コンサルティングファームを経て2012年にHeroku, Inc.に参画。同社の開発者向けクラウド・プラットフォーム『Heroku』の製品担当として、開発者マーケティングやコンサルティング、エンタープライズ顧客向けのサポートを担当している。また、プログラミング言語Rubyのコミッターとして国内外の開発者コミュニティの活動に参加している。

●ウォンテッドリー株式会社／川崎 禎紀

ウォンテッドリー株式会社　CTO
1981年生まれ。東京大学理学部情報科学科を卒業後、同大学院にて情報理工学系研究科コンピュータ科学専攻の修士課程を修了。2006年に外資系金融機関に入社し、テクノロジー部門VPを経て、2012年4月よりWantedlyの開発・運営にCTOとして参画。

中学生のころから、ゲームを作るために独学でBASICとC言語でプログラミングを覚える。大学在学中、ユーザーインタフェースの研究を行う。Wantedlyを活用し、創業メンバーと出会い参画を決意。人の価値観や人生が変わるようなサービスを提供できるように、日々開発に没頭している。好きなゲームはStarCraft II。好きなアーティストは電気グルーヴ。

Quipper, Ltd.　Web application Engineer
