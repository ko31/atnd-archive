プログラミング教育について

2020年度から始まる新学習指導要領においてプログラミング教育が必須化されます…が！
「何するの…？」
「道徳とか英語の方が大変なんだけど？」
という声も聞かれます。
あと1年とちょっと。ここで簡単な体験もしながら、概要を理解していきましょう！
