正式な申し込みはPeatixからとなります。

2014年、インターネットの閲覧時間がテレビの閲覧時間を超えました。これまで多くの企業にとって、マーケティングにおいては補完的な立ち位置であったインターネットが主役となりつつあります。
主役になれば当然ですが、効果の最大化、また投資対効果がこれまで以上にシビアに求められます。DMPやビッグデータ、ABテストやUX設計などバズワードのようにも聞こえはじめるくらいに連呼されるWebマーケティング施策ですが、これらすべての要になるのは間違いなく「Web解析」です。
これを受けて、我々は、オウンドメディア、ソーシャルメディア、ペイドメディアなどの広い視点から「Web解析」について考察し、ところどころにメンバーズが培ってきた知見を散りばめながら、ツールの活用法にとどまらない解析・分析のあり方やレポートの技術などについてのノウハウを1冊にまとめました。
本セミナーでは、この書籍の著者のうちの3人が書籍内容をベースに「Web解析」についての講演を行います。終了後の懇親会ではネットワーキングも行ういますので、少しでもご興味のある方は、是非、晴海トリトンスクエア、メンバーズのラウンジまでお越しください。

＜スピーカーと講演概要＞

第１部　Webマーケティングの進め方

【スピーカー】小松夕祐
2009年にメンバーズ入社。制作部門にてフロントエンド開発業務およびインフラ保守業務を担当。大手金融機関グループにおいて、ウェブサイト運用・構築・CMS導入支援業務を担当後、現在は自社ウェブサイトの運用・構築部門マネージャーとして国内大手企業のウェブマスター業務を運用・構築の双方から支援。2013年に上級ウェブ解析士の資格を取得。

【講演概要】Webマーケティングの進め方
本書でも記載がある通り、Webマーケティング・デジタルマーケティングの重要性は高くなり、今後もその傾向は続く見込みです。このような状況になった背景と今後Webマーケティングをおこなう上で押さえておくべきポイントを解説いたします。

第2部　ソーシャルメディアの成果って？？　この「？？」解決します！

【スピーカー】小野寺翼
Webサイト運用ディレクションやFacebookを中心としたソーシャルメディアプランナー経験を経て2013年2月より、自社のソーシャルメディア専門の研究機関エンゲージメント・ラボ担当に。趣味は料理とトライアスロン。

【講演概要】ソーシャルメディアの成果って？？　この「？？」解決します！
企業がソーシャルメディアを活用するにあたり、ビジネスにどう貢献するの？　何をもって成果とするの？　そんな「？？」をお持ちの方も多いのでは。本パートではこの「？？」をズバリ解決します。まずは企業のソーシャルメディア活用のあり方を理解しましょう。そのうえで、観るべき数値や効果測定のあり方をご紹介します。この「？？」をなくすことこそが、成果につながるソーシャルメディア活用の第一歩です！

第３部　デジタルマーケティングの数字活用術

【スピーカー】中野 学
株式会社メンバーズ コミュニケーションデザインディビジョン プランニンググループ アナリスト／プランナー
広告プロモーション部門を中心に、マーケティングプランを立案。アクセス解析コンサルタント、先進メディア企業でのウェブ広告の運用等を経てメンバーズに入社。データを活用したマーケティング施策の立案及び、運用型広告の戦略設計に従事。

【講演概要】デジタルマーケティングの数字活用術
デジタルマーケティング、特に広告領域では、ターゲットユーザーの行動インサイト・生活インサイトを読み解き仮説を立て施策を実施し、効果を検証するプロセスが必要になります。特に昨今のマーケティング環境を取り巻く環境変化を鑑みた場合には、データを活用し、数字的側面からネクストステップを決定する事が求められます。本講演では、従来型の広告活用から、DMPのような統合型プラットフォームを活用した数字データを用いたマーケティング・プロモーション手法の概念についてお伝えします。

【ゲストスピーカー】小川卓氏

ウェブアナリスト。マイクロソフト→ウェブマネー→リクルート→サイバーエージェント→アマゾンジャパン。「リアルアクセス解析」をハブに執筆・講演。著書に「ウェブ分析論」「ウェブ分析レポーティング講座」「マンガでわかるウェブ分析」など。趣味はサッカー・ゲーム・お風呂。
