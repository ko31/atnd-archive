description

世界を駆け巡るニュースは全体の量が増え、これまでタブーとされた領域での報道も続出している。W杯開催直前の南アの混乱、『ザ・コーヴ』や「医療大麻」、企業のCSRとソーシャルメディア…ニュースの客観性そのものが薄れ、各陣営からのプロパガンダ合戦が起こる中、どうやって自分なりのバランス・ポイントを見つけるかが、ますます大きな課題となる。

4月17日（土）のAIPカフェと、18日（日）のApple Storeでのイベントもお楽しみに！

###
長年J-WAVEのラジオDJとして活躍し、人気ポッドキャスト／Ustream.tv番組「i-morley」のパーソナリティー・ジャーナリスト・音楽アーティストなど、多岐にわたり活動するモーリー・ロバートソンさんが、福岡にやって来ます。
IT技術やコンテンツと、国内メディアが伝えない世界情勢の真実を、独自の鋭い視点で多角的・総合的に語るセミナー「モーリー大学」は、今年も日本各地で精力的に開催されます。
福岡での開催は、2009年の秋と冬の西日本ツアーに続いて3回目。今回はよりパワーアップして開催します。


モーリー・ロバートソン　Morley Robertson

1980年代に米ハーバード大在学中よりMacintoshを使用。1990年代、インターネットの黎明期に立ち会いながらアート活動を行う。
テレビ・ラジオなどのマスメディアとアートを組み合わせたイベントも数多く実施。2000年以来、個人放送を研究、2005年にポッドキャスト番組「i-morley」を開局。現在は71万人の登録リスナーに向けて配信中。毎週火曜日21時からUstream.tvを使った生放送もレギュラー化している。
