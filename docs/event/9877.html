description

フリーランスでWeb関係な人が集まって飲むよ！
一年間、無事に？生き抜くことが出来たって事で、何はともあれ お疲れ様でした

一応、30人募集ですが、いっぱいな感じだと枠を増やす可能性もあります。
40人近くになるとお店が貸し切れるとかどうとか。

渋谷でジンギスカンなお店で飲んで食べて、フリーランスだからこそのトークを楽しみましょう！
サイトはちとひどいけど、お店はシャレオツな感じで女性も入りやすいと思います。

初めましての方歓迎です。
むしろ、初めましてな方だらけになるんじゃないかと思うので、
アウェイ感とかそう言うの感じずにお気軽に参加して頂ければ幸いです。

参加できる人

基本、Web関係のフリーランスとします。ユニット組んで複数人でやってる人とかも大歓迎！
法人化してても、一人でやってるとか規模が2、3人でフリーとあんまり変わらないとかって人もご参加ください。
様子を見つつですが、近いうちにフリーランスになる！って人も良いんじゃないかな。
その場合、Twitterで @hira あたりに一言もらえると嬉しいです。

ならではのトークが楽しめます。

フリーランスと言えば、営業どうしてる？とか見積りどうしてる？
あー確定申告がぁぁ・・・とか、色々有るよね！！
今後も、どうやってお仕事をしていくか、いつまで続けるのか、ずっと続けるのか。
安定がしない生活で悩みも尽きません。
そんな、フリーランスならではのトークを楽しんで、周りの話しを聞くことで得られることなんかも有るんじゃないかな。

ご予算

4,500円。飲み放題付き
まぁ、こんなモンじゃないでしょうか。

当日は・・・

お店の場所が、渋谷に慣れてないと非常に分かりづらいです。
ボクは分からない気がする・・・

なので、直接お店組と、どっかに集合組で分けるつもりです。

渋谷で怪しくオロオロして迷子になりそうな人は集まりましょう！
