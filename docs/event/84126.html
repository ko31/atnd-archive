何となく知っているつもりの「AI」を、簡単に使ってみませんか？

今回初心者向けに、AI・機械学習についての概要セミナーを行います。
実際に 「IBM Watson API」に触れてみて、機械学習の機能を利用することで、何ができるかを考えてみたいと思います。

セミナー概要

社会では人工知能(AI)やロボットの活用と言われ、世間でも様々な事例を目にするようになってきており、AI・ロボットは特別ではなくより身近なものになっているのを感じます。

初心者にも分かりやすいよう、AIの概念を確認しつつ、機械学習についてお話します。
また実際に 「IBM Watson API」に触れてみて、プログラムを作ったりせずに機械学習の機能を「利用」するハンズオンを行います。

AIって何ができるの？どれほど便利に使えるの？・・・といった疑問が、体験により分かる場所にしたいと思います。

【こんなことを知りたい方】
１、AI(機械学習)の概要を理解したい
２、IBM Watson API 利用方法を理解したい
３、機械学習、AI の可能性についての知識を習得したい
