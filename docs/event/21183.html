description

いま、世界中から新作が待ち望まれる映画監督・園子温が、観る者たちを愛と性の彼岸へと連れて行く
究極のエンタテインメント――映画「恋の罪」。
本作の公開を記念して、中村うさぎさんによるトークショーを開催します！
ショッピング、ホストクラブ通い、美容整形、デリヘルなどのキーワードを越え、
リアルかつ鋭い言葉で読む者聴く者すべてをファンにしてしまう、当代随一の“女”中村うさぎさんに、
<女の地獄>をテーマに存分に語っていただきます！
-——————————————————————
また今回はトークの前に、2005年に公開されたドキュメンタリー映画
「UTAKATA ウタカタ～女王★中村うさぎ「愛のショック療法」～」を上映！
岩井志麻子さんや倉田真由美さんとの対話、豊胸手術の現場への潜入など、中村うさぎさんの“ザ・女”を凝縮した1時間。
率直にしてぶれない女王はそのとき、何を考えて何をまとい、何を決意したのか――
滅多に観られない本作をぜひこの機会にどうぞ。
※映画「恋の罪」本編の上映はございません。
