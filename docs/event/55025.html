catch 12月のeco検定に一回で合格を－－全国主要都市で開催
description

　企業や大学への講師派遣などでeco検定では豊富な実績を持つ環境教育フォーラム21（東京・千代田）は12月実施の第17回eco検定(環境社会検定)合格に向けて、9月27日の東京・文京など全国主要都市（スケジュールは下部参照）で「秋の一日集中講座」を開催します。
　eco検定に出題される問題は地球の46億年の歴史から温暖化をはじめとする地球環境問題、公害などの地域環境問題、循環型社会の構築、環境と経済・社会、企業や個人レベルでの環境への取組みなど驚くほど幅広い分野に及びます。特にこの2月に公式テキストが大改定され、学習すべき範囲はより広く、深くなっています。
　秋の一日集中講座では、検定の2－3ヶ月前の学習開始で、eco初心者の方でも1回の受験で合格できる知識、学習法を習得していただきます。当フォーラムが復習用に開発したアプリ（詳細は下部）も好評です。
