Startup Weekend Tokyo REBOOT! 開催の案内

全世界２５ヶ国、１００ヶ都市以上で数万人の起業家たちを鍛えてやりましたStartup Weekendが皆さんの今回の大震災からの復活を願いながら再び東京で行われます。

日付：５月２０日から２２日までの３日間
場所：Open Network Labs 東京都渋谷区恵比寿南３－５－７代官山DGビル

Startup Weekend Tokyoは金曜の夜から日曜の夜までの週末の３日間行われるイベントで、ずっと想像してきた起業家の夢を現実にすることができるチャンスです。

  • 金曜日 ー エレベータピッチでアイデアを発表して、選ばれたアイデアを実装するチームを組みます。
  • 土曜日 ー ひたすら作ります！チームの力を合わせて、ビジネスプランから実際のサービスまで！
  • 日曜日 ー 最後まで実装に頑張ります。出来上がったプロトタイプやサービスの デモ を披露します。そしてパネラーからの的確な評価で起業の夢に一歩近づくことができます。

食事等を含め、３日間あなたの新しい事業の開発作業に専念できるよう必要な基本は準備します。パソコンだけお持ちください。
もちろんあなたの新事業を立ち上げる為に必要なものはお持ち可能です。 “原子炉等は事前にご相談下さい。^^”

日本を変える技術とアイデアを持ってる方、起業に興味ある方々を待ってます！

さらに詳しい情報は: http://tokyo.startupweekend.org/
