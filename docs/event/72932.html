子供向けプログラミング。BASICを学んでゲームを作ろう！

こどもパソコンIchigoJam（イチゴジャム）でプログラミングを学ぼう！プリント基板にはんだ付けして組み立て、オリジナルゲームを作っちゃおう。

【内容】

26日（土）：電子パーツをはんだごてではんだ付けし、IchigoJamを組み立てます。午後からはプログラミングを体験します。

27日（日）：午前はプログラミング。午後からグループに分かれてテーマを決め、プログラミングを行います。
※昼食は持参となります。

【対象】

小学校3年生くらいから中学生。親子参加大歓迎です!!
