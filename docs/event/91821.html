概要

競プロの大会であるところの CODE FESTIVAL 2017 の予選を突破するための、有志による非公式の練習会です。ワイワイ教え合いながら予選突破を目指しましょう！

去年、今年の以前にも練習会を実施したので こちら と こちら のページもご覧ください。

今回も、会場として貸会議室を確保しましたので、どの大学の方でも参加できます！！！！

教える側に回ってくれるプロも大歓迎です！！

内容

問題を解いてきてもらう→会場で集まってワイワイ教え合う→Codeforcesに出る→解法をワイワイ議論する→（参加希望者のみで飲み会、任意参加）、の予定です！

  1. 前日から始めるオンラインバーチャルコンテストを解いてきてもらって、その解法を教え合います（予定）
    会場内でも、残した問題を解く時間を設けるかもしれません（予定）
  2. その後、 Codeforces Round #440 Div. 1 and Div. 2, based on Technocup 2018 Elimination Round 2 にみんなで参加して、ワイワイ教え合います（予定）
  3. 最後に懇親会として、みんなで飲みにいけるといいなと思っています。任意参加です（予定）
