description

セッションの目的

　すべての子供たちに共通する未来、それは誰もが必ず大人になるということ。自分たちが大人になったとき、どういう自分になっていたい？素敵な自分！かっこいい自分！幸せな自分！子供たちが未来の自分イメージを膨らませ、そのための一歩を踏み出すための、子供向けフューチャーセッション。

対象者

熊本県球磨郡湯前町在住の小学校高学年（５～６年生）

詳細説明

　子供たちに自分が大人になったときに、どういう大人になっていたいのか？セッションを通じて、自分の理想の大人像を見つけていきます。
現在から将来まで自分の親からあこがれの対象まで様々な大人が子供たちに影響を与えていきます。そしてその影響を受ける大人によって、子供たちの未来が大きく変わってしまう可能性があります。
　子供たち自身が、今回のセッションの中で、、今自分は大人に対してどういう印象を持っており、自分が大人になった時にどんな可能性があるのかを考えることで、自分の理想の大人イメージ（ワクワク大人イメージ）を作り上げていきます。
　子どもたちには、フューチャーセッションに参加したことで、自分自身のこれからの行動を変えることができることを体感してもらいます。今後、自分たちの将来についてセッションを行っていくことへの興味関心を引き出していきます。

Program
問いを共有する
14:00 〜 14:15 サークル（Circle）
主催者からドリカムフューチャーセッションについての説明を行い、このセッションを行う目的について説明します。

自分が一番好きな大人の人、あこがれている大人の人を一人ずつ発表。

関係性をつくる
14:15 〜 14:30 ストーリーテリング（Storytelling）
３人一組で最近、大人ってすごい、大人って素敵、大人ってかっこいいと思ったことをお互いに発表。　１人３分
一人が発表した後に、聞き手は今聞いた内容を、相手にフィードバックする。（１人２分）

多様な知識を持ち寄る
14:30 〜 15:00 ワールドカフェ（World Cafe）
４人～５人一組で、大人になったらやってみたいことを話し合う。（ワクワクする大人のイメージ）
３ラウンド（１ラウンド１０分）

アイデアを広げる
15:00 〜 15:30 ブレインストーミング（Brainstorming）
初めのグループに戻りワールドカフェで収集した情報をもとに大人になったらやってみたい夢リストを５０個書き出す。

プロトタイピングする
15:30 〜 15:40 クイックプロトタイピング（quick prototyping）
夢リスト５０を参考にしながら、自分だけの夢リストを１０こ書き出し、自分だけのオリジナル夢リストを完成させる

アクションを生み出す
15:40 〜 15:50 サークル（Circle）
自分の夢リストの発表と、今回のセッションをやってみての感想を一人ずつ発表
主催者より、次回の予告をを行う。
