皆様、いかがお過ごしでしょうか。

このたびAEYAC関西事業部では、梅田にてアニメ制作者及びアニメ制作志望者、
アニメ業界に興味のある方を対象に下記の日程にてアニメ交流会を開催致します。
アニメの事や業界の事、プライベートや趣味の事まで、たくさんの方と共有できる貴重な機会となります。

【日時】
8月25日(日) 15:00〜17:00

【場所】
プラム会議室(梅田レンタルスペース)

会場マップ
　

【こんな方にお勧めです】

● アニメ業界に興味のある方
● アニメ業界を目指す方との交流を持ちたい方
● アニメ業界で就活中の方
● アニメ業界志望者に出会いたい業界人の方
● アニメ業界の方と交流を持ちたい方

【参加費】
200円
(飲み物やお菓子等のご用意はございませんのでご希望の方は各自お持ち込みください。ただしソフトドリンク・お菓子等に限ります)

【お申込み方法】
ATNDよりお申込みください。

【運営団体について】
NPO法人AEYAC　 AEYAC公式HP
アニメ制作業志望者の就職活動、若手アニメ制作者のスキルアップを応援する京都市のNPO法人です。
アニメ制作に関するセミナー、就活イベントを企画・運営、アニメ産業に関する調査研究を行っております。
