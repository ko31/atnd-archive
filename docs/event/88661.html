description

クリーク･アンド･リバー社では若手のゲームクリエイターの転職活動をサポートするイベントを開催いたします。

本イベントはゲーム業界において転職をお考えの皆さんに向けて、キャリアアップにつながる転職ができるようサポートを行うイベントです。

当日はゲーム業界や転職市場の動向、キャリアパスについてご説明し、現在皆さんが業界においてどのようなポジションにいるのかを棚卸ししていただきます。

今の環境のままでスキルや年収を高めていけるのかお悩みの方がキャリアを考え直し、新たな一歩を踏み出すきっかけにしていただければ幸いです。

まだ具体的に転職を考え始めていない方にも有用なイベントになると思いますので、ぜひお気軽にお越しください。

内容
○市場動向、職種別募集状況、労働集約とセカンドスキル
○市場対応スキル、ゲームメーカーの求める人材を知る
○クリエイターのキャリアパスの種類
○クリエイターのキャリアチェンジの事例
○転職ってどんなことかを再度考える
○転職紹介会社ってなに？
○成長の為に何をどうすればいいのか

具体的にこんな方にお勧め
○プランナーとして働いているが未来が見えづらい
○スキルがたまっているのか実感が薄い
○このままこの会社のこのポジションにいていいのか不安
○デバッグだけしかしていないが、開発に興味がある
○作業ばかりでもっと開発がしたい
○新規開発にちっともアサインされない

などなど、現状のゲームクリエイティブ環境でご不安をお持ちの方が対象です。
