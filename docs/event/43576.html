開催概要

10/12（土）、プログラミング生放送勉強会を日本マイクロソフト 関西支店で開催します。IT・開発系ノンジャンルの IT 勉強会です。参加資格は、興味があれば OK ！　初めての方も参加お待ちしてまーす。

当日は、ニコニコ生放送でも配信予定です。オンラインからも参加待ってます！　スピーカーへの質問・感想コメントもお待ちしています。

  • 懇親会があります → http://atnd.org/events/43577

セッション紹介

こちら → http://pronama.azurewebsites.net/2013/09/24/pronama-26-at-osaka/

参加費

  • 一般： 500円
  • 学生： 無料

    • ※ 社会人学生は除きます。学生証を提示してください。
    ※ 未成年の方は、保護者の方の許可を得て参加してください。
    ※ 参加費は運営諸費に使います。

懇親会費（懇親会は別途 参加登録が必要です）

  • 4,000円程度を予定（学生はそれ以下の予定）
  • 懇親会の申し込みは別です → http://atnd.org/events/43577

会場案内

  • 日本マイクロソフト 関西支店 のセミナールームです（建物の入口はファミマの裏です）。
  • マイクロソフトセキュリティポリシーにより、入館には、本名・所属の提出が必要です。受付時に所定の用紙にお書きください。
  • 会場内で、飲食できます（飲酒禁止）。
  • ネットワーク提供予定はありません。
  • 電源は床にあります （通常のプラグ形状以外差し込めません）。
  • 開場は30分前を予定しています。
  • 持ち込んだゴミはお持ち帰りください。会場のゴミの分別は当日の指示に従ってください。
  • セミナルームとお手洗い、喫煙所以外への立ち入りは厳禁です。

その他 諸注意

  • 会場では、スタッフの指示に従ってください。
  • ニコ生などライブストリーミングのコメントやツイートは、会場のスクリーンに反映されます。誹謗中傷・セクハラなど不快なコメント・ツイートはしないでください。会場でされている方は退場していただきます。
  • 参加費と懇親会費は、受付時にお支払いください。返金は原則お受けできません。懇親会の当日キャンセルもできません。
