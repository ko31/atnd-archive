保育業界（保育園・幼稚園・幼児教室など）を志望する
2011年3月卒の学生のための、
就活イベント「ココキャリセミナー」もいよいよ最後の開催になりました。

今までの「ココキャリセミナー」は、
現役の保育士さんや幼稚園教諭さん始め、
各施設の園長先生や社長様にもご参加いただき、
皆さんが就活や実際の保育現場で直面する課題解決につながる情報提供をしてきました。

今回の「ラスト ココキャリセミナー」は、
いよいよ4月から社会に出て働き始める皆さんだからこそ、
社会に出る前に知っておいて欲しいこと
今まで就活のサポートをしてきた私たちが本当にお伝えしたいことが満載のセミナーになっています。

セミナー詳細に関しては、当日のお楽しみですが…

●「社会人になる前に本当に知っておいてほしいこと」

●「ビジネスマナーだけではない、社会人としての“心構え”」

を今だからこそ学んで、同期に差をつけたい方は是非ご参加ください。
