おジャ魔女どれみ16 Naive発売記念オフ会を開催！

前回のオフ会は約20人の方に参加を頂いて大変盛り上がりました！
今回も多くの方の参加を楽しみにしています。
オフ会の参加した事が無いという方も大歓迎です！

場所は秋葉原を予定です。
今回は発売から時間を置いての開催となります。本当はGW中に開催できるかな？っと考えていましたが、主催者多忙のためこの日程にさせて頂きました。

なお、今回は会議室が取れれない関係もあり、読書会は開催しないつもりです。
まぁ前回みんなが読み終わるまで待っている状況もあり、企画としては微妙だったかなと考えています。
発売から少し後の日程なので、皆さん読んでから参加してください！

また、前回作成した記念誌ですが、こちらもちょっと用意出来ないと思います。
なので、今回は交流会（飲み会）のみになると思います。
交流会後は一旦解散しカラオケの流れになるかと思います。
-
