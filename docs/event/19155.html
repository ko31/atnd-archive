description

大学生・大学院生のみなさんにお知らせです!

ETIC.のインターンは、学生が新しい時代を切り開こうとするベンチャー
経営者の右腕となって共に課題を解決し、責任ある立場で事業を作り上げて
成長していくという特徴があります。http://www.etic.jp/about/feature
今年で15年目になるのですが、これまで2400人の学生がインターンを経験し、
現在120人が起業家としてのチャレンジを歩んでいます。みな自身の限界に
チャレンジし、本気の繰り返しで「本物のチカラ」を身につけました。

ETIC.では8月28日に、長期インターンを通じ、経営者の右腕として抜群の成長を
遂げた経験者、今社会の最前線でイノベーションを起こしているOBOG、受入企業
経営者が一堂に集う、ETIC.のインターンの総決算イベント、インターンシップ
成果報告会＆修了式である、『CROSSROAD』を開催します。
このイベントでは、夏休みや後期から何か新しいことを始めたい、内定後の
インターンに関心があるという皆さんを対象に、

★実際、インターンをして、どんな成果が出せるのか？ ★インターンを通した抜群の成長ってどんなもの？ ★インターンをしたことによる将来への変化 ★インターンの経験が、社会に出てからどう活きるのか

というあたりを、現役のインターン生やインターンOBOGが、半年間の
インターンストーリーを、嬉しかったことはもちろん、辛かったことも
含め、赤裸々にお伝えするイベントになっています。
インターンの具体的なイメージを掴みたい！という人や、インターン
経験者、起業家、経営者とガチンコで話したい！という方には、普段は
中々、聞くことができない、生の声を聞ける機会になるかなと思います。

過去2400人が参加し、120人が起業したETIC.インターンの全てが
1日で体感できるイベントとなっています。インターンシップや社会
起業に関心のある学生の方々で、インターンを考えている方、インタ
ーンに興味のある方にとって、実際のインターン後の成果・成長・
その後の挑戦の機会をイメージして頂く場になればと思いますので、
関心のある方は奮ってご参加ください。

お申込みですが、申し込みしただけでは参加完了になりません。
メールで確認をとらせて頂きます。

　～限界を越えて挑戦し「本物のチカラ」を身に着けるインターンとは～

　インターンシップにはそれぞれのドラマがある。
　学生時代に”期間限定の正社員”として企業に参画。
　責任ある仕事を任され、挫折や成功を味わった半年間。
　自身のインターン経験でのドラマ・掴んだ確信・抜群の成果や、インターン
　を終えた今だから語ることのできるビジョンを、経験者がホンネで語ります。
　当日はインターンOBOG、社会の第一線で活躍するイノベーターによるプレ
　ゼンテーションもございます。

　インターンを考えている方、インターンに興味のある方にとって、実際の
　インターン後の成果・成長・その後の挑戦の機会をイメージして頂ける
　場となっております。

　「この人と一緒にシゴトしたい！」と思えるような社長と出会い、
　「こんな先輩のようになりたい！」と思えるような先輩と出会う
　　総勢100名が一同に会する、年に一度の一大イベント！！

　インターンシップに興味をお持ちの方、是非ご参加ください。
