★講演内容

マーケティングやコンサルティングのみならず、一般的なシステム開発の現場においても
日に日にニーズの増していくデータサイエンスのスキルと知識。
その入門編として、エンジニアの皆さんにデータ分析言語にして開発実行環境である「R」を用いて
実際にデータを分析しながら、データサイエンスを分かりやすく習得してもらうことを目指した
実習コースを開講いたします。

本コースでは実際にRによるデータ分析を皆さんご自身のノートPC上で行ってもらうため、
ノートPCの持参を必須させていただきます。
その上で、以下の環境を事前にインストールしておいてきてください。
・Rよりインストール(http://cran.r-project.org/)
・RStudioよりインストール(http://www.rstudio.com/products/RStudio/#Desk)

・サンプルデータセットの取得
以下のGitHubリポジトリのコピーをgit cloneで取得
　　（https://github.com/ozt-ca/tjo.hatenablog.samples.git）
　

また特に必須ではありませんが、講師の近著もご参考にしてみてください。
『手を動かしながら学ぶ ビジネスに活かすデータマイニング』
(http://www.amazon.co.jp/dp/477416674X/)

※内容は変更になる可能性がございます。
