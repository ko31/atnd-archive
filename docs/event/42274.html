CMSインサイドは「OSSコンソーシアム CMSビジネス部会」が運営する、CMSの機能を徹底比較する無料セミナーです。
WEB制作者／担当者の方がCMSの選定をする際の参考資料としていただく事を目的としています。

セミナーは8月と12月を除く毎月一回、最終木曜日16時（15:30開場）より開催します。
各回で取り扱うCMSは一種類で、各CMSを代表する方に2時間半のセミナーをしていただき、質疑応答を含めて機能を洗い出します。

　※新しいお知らせはFacebookページをご覧ください。

第14回「CMSインサイド」は8/23 (金)16:00～18:45の開催

＜今回は8月、しかも金曜日開催です。お間違いないようお願いいたします＞

紹介されるCMSは「XoopsEC」です。

2回目の登場となるXOOPSですが、今回はフォーク版、しかもひと味違います。
有限会社ブルームーン代表取締役であり開発者の酒井能克様にビジネスに役立つXoopsECの機能を150分で解説していただきます。

XoopsEC はオープンソースのCMS＋ECのオールインワンシステムです。
XOOPS Cube を母体とする豊富で使いやすいCMS機能に加えて、ショッピングカート、口コミ機能等々、ヤフオク、アマゾン、価格.comのエッセンスを取り入れたオールインワンなポータルサイトを構築する事ができます。

  1. アマゾンとヤフオクをケーススタディとしてシンプルに構成したEコマースのユーザー・エクスペリエンス
  2. Twitter Bootstrap によるレスポンシブウェブデザインのテーマ＆テンプレートの提供
  3. WordPress によるブログの記述をシステム内に提供
  4. XOOPS Cube による柔軟な開発環境を開発者向けに提供

最新の使ってみたいCMSです。皆様のご参加をお待ちしています。
