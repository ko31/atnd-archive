■会費：300円(※会場費を人数で割っています、参加人数によって下がります)


■趣旨：
　ウェブ周りの技術や知識に関して自分は知ってる当たり前のことが
　他の分野の人にとっては話を聞きたいものかもしれません。
　まずはそういうさわりの部分をお互いにしあいましょうという会です。

　各分野においての高度な話ではなくて、
　触り部分くらいの簡単な話（この技術がなぜ必要か）などを
　各４０分弱くらいで２、３個する予定です。


■現在予定している内容：

　●セッション１(30分)
　『そろそろGitのことも思い出してあげてください。』ひむひむ(＠eielh)さん

　＜内容＞
　バージョン管理システムであるGitのさわり
　(最低限の操作フローとか？)を紹介してみます。

　まだまだ世の中はSubversionばかりですが、
　ソースがgithubで公開されてることも増えてきましたし、(？
　最低限の使い方は覚えておきたいですよね！(？
　サーバがなくても使えるので、難しいことは置いておいて、
　気軽に使うことができると思います。

　githubを使うのは、とても簡単なのでそのあたりも紹介できたら！

　時間があれば基本的な仕組みも追えたらと思います。
　仕組みがわかればトラブルも恐くない！

　＜ひむひむ(＠eielh)さんの紹介＞
　自称無職。
　趣味はプログラミングとか
　Linuxいじりのアマグラマ(アマチュアプログラマ)。
　Hiroshima.rbの運営とか、
　Augment8のサポートとか、
　CSS Nite in HIROSHIMAのおてつだいとか、
　してるような気がしますが、
　基本的には裏でこそこそしています。

　広島のEmacs使いを探しています。
　できる限り全力投球したい今日この頃。


　●セッション２(40分)
　『ちょっとjimdoをさわってみようョ！』aki☆(ナガシマアキ)さん

　＜内容＞
　こんにちは。JimdoEvangelists（ただのjimdoファン・笑）のaki☆です。

　jimdo（ジンドゥー）ってご存知ですか？
　ドイツうまれのCMSサービスで、htmlの知識がなくたって、
　簡単にwebページがつくれます。

　すぐにつくれてすぐにデザインも変更できて、
　youtube埋め込みもフォトギャラリーも簡単設置。twitterもいれこめる。
　もちろん独自テンプレートもつくれるし、ブログも簡単にうめこめます。
　そしてインターネットとブラウザがあれば更新可能！

　・友人がホームページつくってとかいうのでひきうけたけど正直しんどい。
　jimdoですぐつくる。あとは好きに更新してと伝える。
　・子どもがホームページつくりたい！とかゆってたけどソフトかいたくない。
　jimdoですぐつくる。会社の昼休みにいじってあげる。
　・自分のサイトつくりたいけど仕事でそっちまで情熱ついやせない。jimdoですぐつくる。
　・サイトつくりたいというお客様の予算がむつかしかったのでjimdoでつくりこむレクチャーを提案。
　なんてときにはもちろん、

　テンプレートがさわれるので、デザインをつくりこむお仕事などでも選択肢が広がります。

　新着情報など毎月更新料がいくらいくらで、
　明日は日曜日なのにもう定時なのに「●●のページを明日までにつくって新着にもいれておいて！」
　なんて電話がかかってきて・・・

　なんてことはなかったですか？

　CMSにすればいい？外注？ワードプレス？
　でもクライアントにそんな予算はないし、そもそもPHPとか全然わからないよ！

　という、ときにも、

　デザインとcssコーディングができれば導入コストが安価で提案できる、
　jimdoはひとつの選択肢だとおもいます。


　そんなjimdoのことをさらっと、
　こんなかんじでつくれるよ、とか、
　独自テンプレートつくるときにこんなとこにひっかかっちゃった、とか、
　独自テンプレートをさわるときとかに思ったことなどなど

　ざっくり話せたらいいなぁと思います！

　※募集中※==

　正直話の構成がいまいち定まっていなくて、
　どういう話をしたら、jimdoを知りたくなったり使ってみたくなったりするかなぁ。
　とちょっと悩んでいます。

　そこで！

　jimdoを使ったことのない人の、jimdoへの素朴な疑問や、
　jimdoを使った事のある人の、
　・こういうこと、困った。
　・こういうのって、どうしたらできるんだろう？
　・ここってこういうことできんのん？

　・・・みたいなことがあれば、aki☆（ナガシマアキ）宛にメールか、twitterのDMで是非ください！
　私に応えられるかはべつとして、

　知っていることはもちろん、
　わかんなかったことはWTMまでに時間があれば調べてみて、
　みんなでjimdoをもっとさわってみたくなったり好きになれたらなーっておもいます！
