description

人生のクライマックスである高齢期は、自由を満喫し、夢を叶えることが理想です。その舞台となる「終のすみ家」選びに失敗は許されません。超高齢社会になった現代において、多種多様の高齢者住宅や施設が乱立していますが、その中でも特に玉石混淆と言われる『有料老人ホーム』や『サービス付き高齢者向け住宅』の正しい理解と選び方を、有料老人ホーム入居相談のパイオニアである中村寿美子氏にわかりやすくお話いただきます。

■講師:株式会社ニュー・ライフ・フロンティア取締役　
　　　《有料老人ホーム・介護情報館》館長　中村寿美子氏
■日時:2014年8月27日(水)13時30分～15時　※受付13時～
