title マグネシウムハッカソン
catch マグネシウム燃料電池にチャレンジ!
description

マグネシウムを使った燃料電池はいくつか方法がありますが、空気中の酸素を酸化剤として利用できればエネルギー密度が高く取れることが期待できます。

このハッカソンでは、金属のイオン化傾向列
(K)(Ca)(Na)(Mg)(Al)(Zn)(Fe)(Ni)(Sn)(Pb)(H)(Cu)(Hg)(Ag)(Pt)(Au)
を参照しながら、水を溶媒にした上でのMgOHの不動体生成を抑え、なおかつH2の発生を
抑えるための工夫を試します。

空気供給反応にはまだ程遠いですが、予備実験の段階としてのハッカソンです。

当日は、マグネシウム板を切り分けたテストピースを提供します。

*マンガン系塩
*アルミニウム塩
*DHMO
*活性炭
*銅板

は無料で使用できます。
