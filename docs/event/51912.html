description

顧客の「真」の声を見極め、顧客との絆を強めるために提供するサービスの価値を高めていくことは、
業種や規模を問わず、企業にとって急務の最重要課題の１つとなっている。
これからの時代、コンタクトセンターに寄せられるお客様の生の声をどう統合し、質の高いカスタマーケアへと活かしたらいいのか。

今回のセミナーでは、お客様との音声情報を価値ある情報にするためのアプローチや事例を紹介いたします。
また、チャットオペレーションによる運用事例についても、合わせてご紹介いたします。

ご多忙とは存じますが、ぜひご来場頂けますよう、謹んでご案内申し上げます。
