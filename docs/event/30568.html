IT系何でも勝手にもくもく会！

イベント ユーザー
event_id 30568
title IT系何でも勝手にもくもく会！
catch もくもくと自習するだけの集まりです
description

マルチメディアスクール・ウェーヴの船橋校です。

普段はスクールの運営とピアソンVueライセンスセンター試験監督をしています。
さて、来る7月7日、七夕の日ですが弊社WAVE船橋校の教室マシンの調整や移動が若干入る予定でして、教室は休校日となっております。
が、しかし業者の立会などで私は一人で出勤しており教室のスペースと設備が丸ごと空いてしまいました！
約半日教室を一人で過ごすことになり、私自身は一人でプログラミングの自習でもしてようかと思ったのですが、なんだか設備がもったいない。。。(笑)

というわけで、せっかくスペースが空いているので、もくもくしたい人のために場所を開放してしまいます。
週末、家だとダラダラしてしまって勉強ができない方、いっしょにもくもくしませんか。
私自身はインストラクターではないので、
あくまでも

自習がメインのもくもく会

になります。
当然、無料です。

当日、私はRubyの入門書をもくもくしてる予定ですが、もくもくの内容はWebでもプログラミングでもデザインソフトでも、はたまたパソコンの書籍を読んでいるだけ、スマホで何かをいじってるなど何でもOKです。
一応パソコンスクールの教室を使ったもくもくなので、パソコンに関係するもくもくでお願いします。

パソコンスクールの教室だから電源あり、Wi-Fiあり、なんでしたら教室備え付けのパソコンも使っていいですよ。設備は充実！
http://www.mswave.co.jp/school/chiba/funabashi_003.htm

参加者が集まらなかった場合でも上記理由によりいずれにしろ私一人で、もくもくしてますので覗き見だけでもOKです(笑)
希望者がいらっしゃれば今後も定例化できれば。。。と思います。

■こんな人にお勧め！

  • 週末プログラミングやWeb系の勉強しようと思っていたが自宅だと集中できなそう、、、な方。
  • 割と船橋が近くて、誰かがいる所でもくもくしたい方。
  • 学習仲間が欲しい人。

■準備するもの

ノートPC持ち込んでいただいてOKです。
教室なので電源は豊富にとれます。Wi-Fiも飛んでますが有線LANもあります。
なお、教室備え付けのパソコンでよろしければ、それを使っていただくこともできます。


  • PCスペック：Core2Duo 1.8G メモリ 2G
    モニタサイズ：19インチ

  • OS：Windows7、AdobeCS5、Office2010　/　WindowsVISTA、AdobeCS3、Office2007

※3台だけですがMacもあります。Snow Leopard、AdobeCS5です。
AdobeソフトはWeb Premiumです。

あと、教室備え付けのパソコンはアドミン権限がないので、ソフトのインストールなどはできないのでご注意ください。

先生など、教えてあげられる人はいないので参考資料など必要な方はお持ちください。

■当日の予定

13：00　教室オープン。来れる方から来てください。
13：15　せっかくなので自己紹介or名刺交換（希望者のみ）しませんか？
途中1時間に1度くらい、一区切りのために休憩入れましょう。
※遅れて参加の方は下記の休憩時間に自己紹介しましょう
17：30～18：00　私の立会作業が終わり次第区切りの良いタイミングで終了(笑)
18：00～　都合が合う方は、せっかくなので割り勘で軽く飲みに行きませんか？
※もちろん希望される方だけで大丈夫です

■参加費

もくもくは無料です。
その後、飲みに付き合っていただける方だけ飲み代実費（3000円くらい？）

■その他

教室に入る際に校舎受付で名刺を1枚頂戴しますので、名刺をお持ちください。
お持ちでない場合は、別紙に必要事項を記入していただきますのでご了解ください。
（氏名・連絡先などの明示がない方は、校舎内に入ることが出来ないため）
※自己紹介タイムはハンドルネームなどでもOKです
飲み物の持ち込みはアルコール以外何でも可能です。
食べ物はにおいがしなくて、床にぼろぼろ落ちるものでなければOK。
ポテチなどスナック菓子を食べながら教室据え置きマシンを使う方はお箸で食べてください(笑)
