1月31日（土）に第6回ふくしまの市を開催します。
今回もワークショップ形式で行います。

■日時　　　1月31日（土）10:00~12:30
■場所　　　福島ガスショールーム・プロメ
　　　　　　（福島市栄町12-32 3F）
■定員　　　20名
■参加費　　2,000円（材料費込み）

参加をご希望の方は、以下の場所でチケットを販売しておりますので、ご購入ください。

チケット販売：かーちゃんふるさと農園わいわい
　　　　　　　（福島市栄町10-3　キッチンガーデンビル1階　　TEL　024-573-2634）

カカオの香りが際立つ、本格的なチョコレートと福島のおいしい食材がコラボした美味しいスイーツを皆さんと作ります。

チョコレートは市販のものではなく、インドネシア産のカカオを使った本格的なチョコレートです。
この本格的なチョコレートで、福島の果物（ドライフルーツなど）や、県内農家さんが製造する商品をコーティングしたお菓子作りを行います。

とても簡単にできますので、お子様とご一緒のご参加もお待ちしております。
※小学生は保護者の方もご一緒にご参加をお願い致します。原則、小学生以下のお子様はお断り致します。
お子様とご一緒のご参加についてはご相談ください。

■お問い合わせ先
　ふくしまの市実行委員会（携帯　090-6456-0800）
