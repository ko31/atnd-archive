description

フォトウォークをキッカケに地元の良さを知り、さらに同じ趣味を持つ人と繋がれる。
私も過去にそんな経験をしたので、前回の参加者に加えて新たに盛岡近郊の方にも参加して頂けたら嬉しいです。

フォトウォーク概要

  • 開催日時：2014年7月27日(日)　10:30〜15:00（昼食含む）
  • 集合場所：岩手公園下入り口
  • ルート　：岩手公園→中津川→中の橋通り→ほか
  • 持ち物　：iPhoneなどのカメラ（コンデジやデジイチもOK！）
  • 参加費　：無料（飲食代は各自ご用意ください）
  • 参加条件：特になし（途中参加&離脱も可）
  • 募集人数：5名（随時増員）家族参加の方は予め人数をお知らせ下さい
  • 募集締切：2014年7月26日(土)

※雨天決行（屋内探索に変更します）
※昼食のお店の予約があるので、ご家族一緒の場合は事前に人数をお知らせ下さい
※私の子供達も（小学生）も参加予定です。小さいお子さんが一緒でも遊び相手はいるので安心して下さい^^

盛岡市近郊にお住まいでiPhoneで写真を撮り始めたばかりの方、子供達と休日の想い出を作りたい方。

ぜひ、お気楽に申し込んで下さい♪
きっと一人で写真を撮る時とは違った楽しい時間が待ってますよ。
そして、盛岡の人同士が繋がるキッカケになれば嬉しいです！

では、当日会いましょう〜♪

過去3回開催したフォトウォークは、おかげさまでたくさんの人に参加していただき大盛況でした。
その様子はコチラからどうぞ。
