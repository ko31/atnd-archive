練習会からの帰宅に台風19号の影響が見込まれるため、基本的には開催中止といたします。
現時点で各国の予報データを見比べたところ12時時点で最大風速が20mを越えており、(1) 15号のときに計画運休した程度の数値 (2) そもそも京葉線は運行不可能な可能性が出てくる という状況です。

ですが、金曜日時点で発表予定となる計画運休の時間帯によっては、最低限の形での運営も考えております。
参加者の皆さんにメールいたします。
