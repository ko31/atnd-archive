概要

  • 数学の愛好者たちが集まってやさしい入門書をゆっくり読む、初心者向け読書会です。
  • 大田春外 『高校と大学をむすぶ幾何学』
    から第6章「方べきの定理、メネラウスの定理とチェバの定理」を読む予定です。
    ※ 当日までにテキストの用意が難しい方は、ご相談ください。
  • やさしく丁寧に解説された本なので、特に予備知識は必要ありません。
  • 学生の方も社会人の方も、お気軽にご参加ください。
  • 今回で 7回目となりますが、初参加の方も大歓迎です。飛び入りでもどうぞ。
  • これまでの勉強会の板書（写真）とATNDのリンクです。
    • 第1回「ピタゴラスの定理」 : ATND
    • 第2回「三角形の基本定理」 : ATND
    • 第3回「タレスとユークリッド」 : ATND
    • 第4回「円周率」 : ATND
    • 第5回「三角形の五心 §重心と内心」 : ATND
    • 第6回「三角形の五心 §外心,垂心,傍心、オイラーの定理と九点円の定理」 : ATND

場所

東京都新宿区赤城元町1-3 赤城生涯学習館 教養室B

会場費200円と銀行振込手数料210円を参加人数で等分します。
