現役で活躍するプロから、シナリオ制作現場の話やシナリオ制作のコツ、業界の最新事情に関する話が聞けるセミナー「シナリオ塾」。
過去開催のシナリオ塾では、多くのゲームシナリオのライターの方やこれからゲームシナリオライターを目指す方にご参加いただきました。

今回のシナリオ塾はシナリオ制作またはディレクションの実務歴が1～3年前後の方で、よりクリエイティブを高めたい方を対象に、株式会社Craft Eggと共同で、大人気スマートフォンゲーム『バンドリ！ ガールズバンドパーティ！』のシナリオ制作についてお話します。

☑美少女キャラクターやバンド、アイドルキャラクターをもっと可愛く書きたい
☑それぞれのキャラクターを深堀りしてシナリオを書くにはどうすれば良いか知りたい
☑反響の多いシナリオはどうすれば書けるかを学びたい

という方のご参加をお待ちしています。

イベント後は、登壇者と参加者同士の交流の場も設けています。
また、希望者には、採用に直結するカジュアル面談も開催します。

『バンドリ！ ガールズバンドパーティ！』とは

『バンドリ！ ガールズバンドパーティ！』は、キャラクターとリアルライブがリンクする次世代ガールズバンドプロジェクト「BanG Dream!（バンドリ！）」の世界観を軸に展開されるリズム＆アドベンチャーゲームです。
簡単操作のリズムゲームは最大5人で同時に遊ぶことができ、アニメやライブでお馴染みの楽曲や有名アニメソングのカバー楽曲を楽しめます。
また、ストーリーシーンではハイクオリティなLive2Dを搭載しており、キャラクターが生き生きと動きます。
https://bang-dream.com/
