※満員御礼！定員に達しましたのでお申し込みは締切りました。

ユナイテッドピープル忘年会＆サステナブルを探る旅の報告会！

〜オーストラリアで見つけたサステナブルな暮らしとは？〜

こんにちは、ユナイテッドピープルのちはること畠山です。
先日、サステナブルを探るオーストラリア１ヶ月の旅から無事帰国いたしました！
そこで、ユナイテッドピープルでは、旅の報告会を兼ねた忘年会＆交流会をドーンと行いたいと思います。

プログラム

19:00　 開場
19:30　 報告会スタート
20:15　 ゆるりと質疑応答タイム
20:30　 忘年会＆交流会


※会場は、ドリンク（ソフト、アルコール）飲み放題、食事もご用意しております。



オーストラリアでは、WWOOF（お金のやりとりなしで「食事・宿泊場所」と「力」そして「知識・経験」を交換するしくみ。）を利用してパーマカルチャーを学びながら農場で働き、特徴の異なる３つのエコビレッジへ足を運びながら、彼らの暮らしぶりを取材してきました。
雨水・川の水を飲み、雑草を食べ、馬に乗り…とだいぶ野性的な暮らしをしてきましたが、その中にある人と人とのつながり、人と土地とのつながりを体感し「本当に豊かな暮らしってなんだ？」とたくさん考えました。
旅を通じ、多くの人達との対話の中から見えてきた「サステナブルな暮らし」について語ります！（旅のブログはこちら）

そして旅の報告会のあとは、忘年会＆交流会！
社員の半分以上が元バーテンというお酒好き・お祭り好きな我が社のメンバーたちが、新しいお酒を仕入れてお待ちしております♪
ビールーサーバーもありますので、生ビールもお楽しみいただけますよー！

参加費は、手作りご飯（マクロビも有るかも♪）＋お酒飲み放題で2500円！
学生は2000円で参加出来ます♪RT割引は500円OFF！
ちはるのオーストラリア土産、チャイをふんだんに使用したラムチャイもご用意しております♪

旅が好きな人、サステナビリティに興味がある人、お酒が好きな人、もちろんお一人様も大歓迎！
みなさま、お誘い合わせの上ぜひお越しください♪お待ちしております！
