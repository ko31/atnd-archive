【プリコミュ】プチオフ＆カラオケ大会


はじめましての方は、はじめまして。　お久しぶりの方は、お久しぶりです。
プリキュアコミュニティで何回か（と言っても、片手で数えられる程度ですが）放送させていただいた事のある、「たば」と申します。
リスナーとしてもだいぶ前からあまり参加できてませんので、最近の生主さんの放送もあまり行けなかったりします。

ですが、プリキュアは好きですし、コミュニティメンバーの方々との交流を深めたいと思い、オフ会＆カラオケ大会をオールナイトで開催したいと思います。

なお、カラオケ大会は、プリキュアコミュニティにて何枠かラジオ形式で放送する予定です。

開催概要

  • 開催日： ３月２９日（土）　　２３：００～５：００
  • 会費: 1,400円 ＋ 1ドリンク制
  • 場所： カラオケビッグエコー所沢駅前店
  • 集合場所： 西武線 所沢駅　改札付近
  • 集合時間： ２２：４５ 　

  • 締め切り： ３月２６日　２４：００

    • カラオケ前にアギルランさんと近くのファミレスでご飯を食べる予定です。 早く来れる方はご一緒にいかがでしょうか？
      • (ご飯会は１９：３０　改札付近での待ち合わせになります。）

        当日の目印は、参加者に個別でお知らせします

注意事項

  • オールナイトでの開催となりますので、18歳未満の方の参加はお断りします。
  • 1ドリンク制ですので、各自、飲物を持参してくｄゴニョゴニョ・・・・
  • カラオケは、プリコミュでの放送時はプリソン縛りですが、放送後はなんでもＯＫです。
