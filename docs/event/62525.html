【 テ ー マ 】

Google Cloud Platformに触れてみよう！GCPUGに参加して盛り上げよう！

【 概 要 】

Google Cloud Platformのユーザー会、GCPUG(Google Cloud Platform User Group)が立ち上がりました！
GCPUGはGCPを全国に広め、GCPの良さを皆様にお伝えします！
そしてイベントは東京に続き、第2回はなんと…沖縄県那覇市で行いますヽ(=´▽`=)ﾉ☆

初めての方は是非この機会にGCPUGに参加してGCPの素晴らしさを享受しましょう！
そして一緒にGCPを盛り上げていきましょう！

GCPが初めての方のためにGCPの全体概要の解説や、アカウント開設からのハンズオンがあるかもしれません☆
（※無料枠が使えますがクレジットカードが必要です）

こんな方に向いています

・GCPって何それおいしいの？という初心者の方。
・GCPはちょっと知ってるけどもう少し詳しく知りたい方。
・GCPを一緒に盛り上げたい方。
・とにかくGCPが大好物な方。
