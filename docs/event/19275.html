2011年をもちまして、山村龍和が書をはじめて２０年となります。

そこで、来る9月22日(木)～27日(火)、

初の個展となる「龍和展」を大阪・梅田で開催する運びとなりました。

会期中の9月24日(土)には将棋棋士の羽生善治氏を

スペシャルゲストとして、対談イベントを行います。



将棋と書道、この一見して全く異なる２つには、どのような共通点があるのでしょうか？

そのひとつは、伝統文化であり、進化しつづけるものであるということです。

もうひとつは、「直感力」が大切だということです。

羽生善治氏は、将棋を指す際、直感が大切だといいます。

山村龍和は、書には直感が大切だと考えます。

直感とは何でしょうか？　また、どのようにつかい、どのように磨くものでしょうか。

直感力・先見力・実行力・決断力・人間力の秘密にせまり、

日常心がけていること、志、次世代の生き方、

天才の育つ環境など興味深い対談になりそうです。



３・１１以降、私たちを取り巻く世界は急激に変わり出しました。

羽生善治氏が今最も伝えたいこと、山村龍和が今最も伝えたいこと、

被災地に今できる具体的な支援方法についてもお話を伺いたいと思います。

※このイベントの収益の一部は被災地で農漁業を営む方に寄付いたします。

お申込みフォームより質問内容をお書きください。
