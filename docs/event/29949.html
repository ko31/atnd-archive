MetroStyleハッカソン 2Days
～VSハッカソン倶楽部 大阪 第2回～

7/14(土)　MetroStyleDeveloper×VSハッカソン倶楽部 MetroStyleアイデアソン
7/21(土)　VSハッカソン倶楽部×MetroStyleDeveloper MetroStyleハッカソン

「MetroStyleDeveloper×VSハッカソン倶楽部」による初コラボイベントを開催します！！

ハッカソンとは「ハック」と「マラソン」を合わせた造語で、
作ることが好きな人が集まって、その場でアプリを作ってしまおう！というイベントです。

今回は「MetroStyleDeveloper×VSハッカソン倶楽部」による初コラボイベントです。
あなたも一緒に Windows 8 の Metro style アプリ開発にチャレンジしませんか。
Metro style アプリ開発の未経験者大歓迎です!!
Metro style アプリ開発経験者と交流できる絶好の機会ですよ。
さぁ!! 一緒に Metro style アプリ開発を楽しみましょう!!!

アプリが完成するのに越したことはありませんが、完成しなくても大丈夫!!
途中経過を発表し、皆で情報共有していきましょう。

  • MetroStyle アイデアソンで MetroStyle 勉強会などを行いますので、合わせて参加されることをお勧めします。
