ネットワークの勉強会を立ち上げます！

  • 福岡では様々なIT勉強会が開催されていますが、このたびネットワーク（ネットワークに関連した様々な内容）を対象とした勉強会を立ち上げます。
  • 今回はその勉強会の立ち上げにあたって、勉強会の方針や扱う内容などについて、皆さんのご意見やご要望をいただきたいと思いますので、興味がある方が是非ご参加ください。
  • 特に学生の方は大歓迎です！
  • ネットワークビジネス の勉強会ではありませんので、その界隈の方はご遠慮ください。

勉強会で扱うテーマについて

  • 基本的にネットワークに関連するものであれば、レイヤーを問わず様々な内容を扱いたいと思いますが、まずはどのあたりのテーマを取り上げるのか決めたいと思います。
  • 現在考えているのは以下のようなテーマです。
    • ネットワーク設計の勘所
    • 新技術系（IPv6・10GbE・802.11ac・OpenFlowなど）
    • トラブルシューティング（ハマった事例、パケット解析など）
  • もちろんこれら以外のテーマも扱いたいと思いますので、当日は色々なご意見をお願いします。
