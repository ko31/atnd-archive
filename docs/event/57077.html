head-shotでサバゲしよう！

イベント ユーザー

catch 今回は半面。Yゾーン。
description

2014年10月18日(土)に千葉のhead-shotでサバゲ会を催したいと思います。今回は半面のYゾーン側です。

仲間内のサバゲ会です。朝木と一度でもゲームをやった事のある方ならご参加頂けます。

レギュレーションはごくごく普通です。

・セミオート・フルオート問わず全ての銃でパワーは0.98Ｊ以内
・使用BB弾はバイオのみ（プラスチック、セミバイオは禁止。）
・減速アダプタの使用は禁止
・フリーズコール禁止
・基本的にブラインドファイア禁止
・レーザーサイトはすべて使用禁止
・ゾンビ行為禁止
・フラッグへのダッシュアタック禁止（周囲制圧後にプッシュ。）

その他サバゲアイテムの手投げグレネード、グレネード、クレイモアも使用可能。
それ以外に何か使いたい変わったアイテムありましたら、とりあえず聞いて下さい。

■ゲーム時間と第一ゲームまでの流れ

時間は08：00~17：00の予定です。
0740頃にはフィールドゲートが開き、駐車場に入れるようになります。
ゲーム開始は09：30前後を予定しております。
終了は17：00に完全撤収

0800　フィールドオープン
　到着後、速やかに受付を済ませてください。

0900　ブリーフィング
　ルール説明とチーム分け、時間があれば軽く自己紹介するかも。

0930　第一ゲーム開始
そんな感じです。最低でも1030には第一ゲームを開始出来るよう、ご協力下さい。

■参加表明とゲスト参加

参加表明の締め切りは10/16頃とします。
基本的に参加の方はココで表明していただきますが、ATNDを使用できない方なのどの参加の場合、同行する方がコメント欄にゲストの名前表記して参加表明してください。（ただし、お連れの方にも今回のレギュレーションをしっかりお伝えください。）
リストを作る都合上、ハンドルネームでもOKですのでお名前を頂きます。

■必要な物、参加費

・参加費3500円（予定　カレー不要な方は-500円）
・エアガン
・BB弾（バイオのみ）
・ゴーグル
・顔面を保護するマスク、四肢保護のニー、エルボーパッドや手袋があるとベター。
・各自、黄、赤のマーカーをご用意ください。（または、仲間内でガムテーム等をシェアして下さい。）

■お昼ご飯について
カレー出ます。参加費3500円に含まれていますので、カレーが不要な方はコメ欄にご記入ください。

■交通手段と送迎バスについて

基本的に現地集合現地解散ですので、自家用車の乗り合いは個々に打ち合わせをしてください。

■ゲーム内容

基本的にはフラッグ戦メインです。
通常のフラッグ戦のほか、ペットボトルフラッグ戦（ブザーの代わりに3本のPBを倒す事でフラッグゲットとする）や、他にちょっと新ルール考案中です。
ゲームルールによっては、セミオート縛り等もあります。

かなりハイペースで回しますので、「補給が間に合わない」「疲れた動けない」等は遠慮せずに休憩してください。　20分もすれば次のゲームに入れます。

■ちょっとしたお願い

締め切り等やその他諸事情で参加を伏せたい人も多いと思いますので、開催中や終了後は、
・個人単位で『今日サバゲして来た』と言うのは自由で。
・誰が参加していたかに関しては『触れない』方向でお願いします。
・TwitterやMixi等で『○○さんお疲れ様でした～』も出来るだけ空気読んで下さると助かる人も多いかと思います。
（自分が参加していた事を表明する事に問題ない人同士のやり取りについては特に問題ありません）
・ですので、基本的に本ページコメ欄使って頂くのを推奨します

■■雨天の場合■■

前日より確実な雨天（大雨や嵐）と分る場合は10/17のお昼頃にアナウンス致します。
小雨の場合や、判断に困る場合は、フィールドに確認の後になるべく前日には判断できる様に努力いたします。
