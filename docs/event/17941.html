＊変革の鍵は、「異種との交流」
会社組織を越えて、社外の方々と接し、一人ひとりの危機感
を醸成した上での能力・キャリア開発をなくして、更なる進
化はないと考えています。
また、素晴らしい人との出会い、人脈という価値はビジネス
をする上では代えがたいものです。

＊「他流試合X」×「他流試合Y」＝∞（無限大）のシナジー効果
二つの試合「他流試合X」×「他流試合Y」という仕掛けによる、
オープンネットワーク社会に呼応した、全く新しい形の研修プ
ログラムです。

・「他流試合X」・・・能力測定データ上の他流試合です。
事前課題として、"統合型市場価値測定テスト"をご受検いた
だきます。当テストの特性を活かし、他のビジネスパーソン
とのビジネス基礎能力比較を行います。
・「他流試合Y」・・・肌感覚での他流試合です。オープン
参加型の研修形態による幅広い交流、シナジー効果を体感で
きる能力レベル別クラス編成と性格タイプを意識したグルー
プ編成で、生産性や満足度の高いグループディスカッション・
ワークを行います。
コミュニケーション力や多様性への受容力が飛躍的に向上す
る、起爆剤的効果があります。
