イベント概要

Amazon EC2やAmazon S3に代表されるアマゾンクラウド = Amazon Web Services(AWS)は、ユーザの要望に応じて機能がどんどん追加され、まさにかゆいところに手が届くクラウドサービスになってきています。すでに世界中で数十万のエンジニアがアマゾンクラウドを利用しており、今後は開発者/ITプロフェッショナルにとって必須にクラウドサービスになることは間違いありません。今回は、福岡で初回の勉強会ということで、記念すべき0回目と題して勉強会をさせて頂きたいと思います。

AWSに興味があって使ってみたい方、AWSは触ってみたけれどももう少し深く知りたい方、AWSをもう使われていて周りに仲間が欲しい人、AWSをすでに使いこなしていて他の方にその使い方を広めたい方、AWSユーザーグループ福岡のコアメンバーになられたい方、、皆様、すでにAWSユーザー会の一員です。気軽にどしどしご参加ください！！

開催プログラム

  • 19:00 ～ 19:05 アジェンダ紹介・会場案内
  • 19:05 ～ 19:10 JAWS-UGの紹介
    　　　　　　　　　JAWS-UGコアメンバー　CUPA竹下氏
  • 19:05 ～ 19:30『アマゾンクラウド概要紹介』
    マーケティングマネージャー　小島氏、Amazon Data Services Japan株式会社
    • AWS入門者の方に向けて、アマゾンクラウドの概要を紹介いたします。初期費用不要、従量課金、スケールアウト・インが自在、開発スピードを高めることが可能で、システムレベルの作業から解放されるアマゾンクラウドのメリットをお話します。
  • 19:30 ～ 20:00『アマゾンクラウドのデモンストレーション – 既存Webアプリをその場でクラウド化しよう！』
    エバンジェリスト　玉川氏
    • このデモンストレーションでは、典型的なLAMPのWebアプリケーションを例にとって、スケールアウト可能なクラウド・アーキテクチャに変更していく実践例をデモンストレーションを交えご紹介します。フロントエンドにロードバランサ(Amazon ELB)を備え、その後ろに仮想サーバー(Amazon EC2)により並列化されたアプリケーションサーバー、後段にはデータベース・クラスター(Amazon RDS)を備えたクラウドアーキテクチャの魅力をぜひご体験ください。
  • 20:00 ～ 20:20 『耐障害性と柔軟な拡張性を持つ分散処理環境の実現について-Monkey Magic & Volante-』
    矢野恒（株式会社キューデンインフォコム　e-ビジネス部新規事業グループ）
    最首英裕（株式会社イーシー・ワン　代表取締役社長）
    浜崎陽一郎（株式会社Fusic　取締役副社長）
  • 20:20 ～ 20:35 『クラウドコントローラー総特集(タイトル調整中)』
    JAWS-UG コアメンバーのCUPA 荒井氏
    • AWSなどのクラウドベンダーが充実してくるにつれ、そういったクラウドインフラストラクチャをコントロールするツールを提供するベンダーも増えてきています。RightScaleをはじめとしたクラウドコントローラーの数々についてCUPA荒井氏が解説します。
  • 20:35 ～ 21:00 ライトニングトーク & 質疑応答
