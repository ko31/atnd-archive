イベント概要

北海道の小樽と札幌で開催している Drupal CMS（オープンソースのコンテンツ管理システム）の勉強会です。
今回は札幌での開催です。

なお、名前は「勉強会」としていますが、セミナーのような堅苦しいものではなく、Drupal ユーザーが自由に情報交換できる場にしたいと考えています。気楽な集まりですので、お気軽にご参加ください。

北海道の Drupal ユーザで情報交換と交流の輪を広げ、さらなる活用につなげていきましょう。

参加：無料

内容

今回は、メインのセッションとして２つのテーマによる発表を予定しています。また、ショートセッションやライトニングトークも常時募集中。内容は Drupal のほか Web や IT に関するものなら何でも大歓迎です。ネタをお持ちの方は、事前でも（当ページのコメント欄をご利用ください）当日でもお気軽にスタッフまでお知らせください。

(1) Drupal サイトをセキュアに運用するには

speaker: Yoshikazu Aoyama

Web 上のシステムは常にさまざまな攻撃の危険にさらされており、あらゆる管理者にとってシステムのセキュアな運用は最大の関心事のひとつです。本セッションでは、Drupal サイトを前提に、セキュア運用に必要な知識や設定のポイントを整理しながら、Drupal のセキュリティについて一緒に考えてみたいと思います。

(2) Drush を使いこなして Drupal サイトの管理をスマートに

speaker: Kyotaro Kon

Drush とは、Drupal のシェル、つまりスクリプト用インターフェイスのこと。Drush を利用することで、コマンドラインから Drupal の種々の機能にアクセスすることができます。ここでは、Drush の概要、基本的な使い方、便利なコマンド等について一緒に学びましょう。

(3) ショートセッション/LT

1) ローカル環境の Drupal インストールとディレクトリ構造
Kenji Shirane
Drupal をあまり触ったことがないビギナーの方を対象に、ローカルのテスト環境を構築する手順と Drupal サイトを構成するファイルとディレクトリ構造について整理します。
