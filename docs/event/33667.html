テーマはOne ASP.NETということで、ASP.NETがちょっとでも入ってたらなんでもOKのゆるふわスタイルとします。
（自社構築事例、導入事例の紹介などでもOK）

ブログの投稿を終えたら、コメント欄にブログの URL と簡単な紹介を記載お願いいたします。
このページをを見れば、すべてのブログ投稿をチェックできるようにしましょう！

というわけで募集開始です～。
