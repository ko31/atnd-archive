description

2013年12月22日(日)に千葉のhead-shot（全面）で夜戦サバゲ＆BBQ会を催したいと思います。

今まで朝木単独主催でしたが、今回より「チームけろよん」との共催と言う形になります。（ゲーム趣旨は同じなので、今まで朝木会に参加なされた方なら不満のない内容だと思います。ゲーム数多目で、ワイワイやる感じ。）
参加者リストの重複を避ける為、けろよんさんに直に参加の可否を連絡した人は、ここで表明しないでください。
今後リスト関係は、徐々に調整していきます。

レギュレーションはごくごく普通で、弾数制限等はございません。詳細は以下の通り。

・セミオート・フルオート問わず全ての銃でパワーは0.98Ｊ以内
・使用BB弾はバイオのみ（プラスチック、セミバイオは禁止。）
・減速アダプタの使用は禁止
・基本的にブラインドファイア禁止
・レーザーサイトはすべて使用禁止
・ゾンビ行為禁止

その他サバゲアイテムの手投げグレネード、グレネード、クレイモアも使用可能です。

また今回は夜戦と言う事もあり、バイオ発光弾とトレーサーを強く推奨いたします。
他にも、ウェポンライト、ハンディライトもあると有効に使えますので、ご用意してもらえると良いと思います。

■ゲーム時間と第一ゲームまでの流れ

時間は19：00~05：00の予定です。
18:00頃にはフィールドゲートが開き、駐車場に入れるようになります。

メンバーが集まり次第BBQを開始し、食事といたします。
食事終了後にゲーム開始です。

ただ、調理の間、手持ち無沙汰のメンバーを募って小規模なゲームを回したりもしますので、やる気満々の方も大丈夫です（w

終了は05：00に完全撤収

■参加表明とゲスト参加

参加表明の締め切りは12/20頃とします。
基本的に参加の方はココで表明していただきますが、ATNDを使用できない方なのどの参加の場合、同行する方がコメント欄にゲストの名前表記して参加表明してください。（ただし、お連れの方にも今回のレギュレーションをしっかりお伝えください。）
リストを作る都合上、ハンドルネームでもOKですのでお名前を頂きます。

■必要な物、参加費

・参加費フィールド代　3000円（予定）
・BBQ+サイリューム代　1500円
・エアガン
・BB弾（バイオのみ）
・ゴーグル
・顔面を保護するマスク、四肢保護のニー、エルボーパッドや手袋があるとベター。

■夕飯について

今回はBBQを予定しており、食費として1500円徴収いたしますので、食事に関しては、手ぶらで大丈夫です。
個人的に持ち込みたい食材がある場合は自由です。ただし、焼き台には限りがあります。

■交通手段と送迎バスについて

基本的に現地集合現地解散ですので、自家用車の乗り合いは個々に打ち合わせをしてください。

■ゲーム内容

基本的にはフラッグ戦メインです。
ゲームルールによっては、セミオート縛り等もあります。

かなりハイペースで回しますので、「補給が間に合わない」「疲れた動けない」等は遠慮せずに休憩してください。　20分もすれば次のゲームに入れます。

■ちょっとしたお願い

締め切り等やその他諸事情で参加を伏せたい人も多いと思いますので、開催中や終了後は、
・個人単位で『今日サバゲして来た』と言うのは自由で。
・誰が参加していたかに関しては『触れない』方向でお願いします。
・TwitterやMixi等で『○○さんお疲れ様でした～』も出来るだけ空気読んで下さると助かる人も多いかと思います。
（自分が参加していた事を表明する事に問題ない人同士のやり取りについては特に問題ありません）
・ですので、基本的に本ページコメ欄使って頂くのを推奨します

■■雨天の場合■■

前日より確実な雨天（大雨や嵐）と分る場合は12/21のお昼頃にアナウンス致します。
小雨の場合や、判断に困る場合は、フィールドに確認の後になるべく前日には判断できる様に努力いたします。
