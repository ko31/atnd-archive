catch 岩手県内陸初の本格的コワーキングスペースオープン！
description いつもお世話になっております。コワーキングカフェもりおか設立準備協議会の川村でございます。

　この度コワーキングカフェもりおか設立準備協議会は、ビジネスリプロダクション（岩手県盛岡市盛岡駅北通４－４　国労会館３階　Ｃ号室）様の打ち合わせスペースをお借りして、岩手県内陸地域では初の本格的コワーキングスペース「コワーキングスペースもりおか」として５月７日にオープンする運びとなりました。盛岡駅より徒歩５分、鉄路で行けばとてもアクセスの良い場所になります。
　コワーキングとは、様々な職種の人がひとつのスペースに集まって各々の仕事をやりながら、集まった人同士で交流することで新たなビジネスプロジェクトが生まれるなど、ビジネスの成果が出やすい働き方です。全国ではかなり注目をされていますが、県内ではなかなか認知されることが少ない一方、大都市圏など県外からコワーキングスペースがあったらという要望が多い施設となっています。私も実際に行ってみまして、盛岡にコワーキングスペースを開かなければ将来の発展も震災復興にも役に立たないというのを実感し、２年近く関係各所に問い合わせや相談などをしながら活動をしてきました。いろいろ苦労を重ねた中でやっとスタートラインに立てたところですので、今後とも皆様からの変わらぬご支援を重ねてよろしくお願い申し上げます。
　営業日時・時間は平日１２：００～１８：００（土曜日はイベント予約のみ利用可能）に、ビジネスリプロダクション様の後援を受けましてオープンします。ひとつのテーブルで最大１０人程度収容する小さなスペースですが、非常に濃いコミュニティが作れそうなスペースになりそうです。
　今回オープンに際しまして、５月１０日の１４：００から、プレオープンパーティーを同スペースで開催いたします。第１部と第２部に分けまして、第１部は簡単なコワーキングの説明と体験を兼ねたコワーキング体験会を、第２部では場所を移しての飲み会を予定しております。スペースの関係でいずれも最大定員１０名という小さなパーティーになりますが、様々な分野で活躍されている皆様方にこのスペースについてご紹介したいので、お時間よろしければご参加のほどよろしくお願いいたします。

　　　　　　　　　　　　　　　　　　　　　　　ビジネスリプロダクション内　打ち合わせスペース）
