HTML5+α ＠福岡 と Mozilla 勉強会の合同開催イベントです

Mozilla 勉強会は、Firefox や Thunderbird の拡張機能や Jetpack などのアドオン、Web アプリやデスクトップアプリのプラットフォームとしての Mozilla の技術について関心のある方がオフラインで集まり、情報交換するための小規模ワークショップです。

HTML5+α ＠福岡 はプログラマ、Web デザイナ、Web コンテンツ管理者、その他 Web に関わるお仕事や活動をしている人が福岡で集まり、HTML5やその周辺技術について学ぶ会です。

HTML5 や周辺技術について今回はマルチメディア系の技術の解説を中心に、リリース間近(直後？)の Firefox 4 に関する技術や拡張機能開発と併せて勉強したり、普段あまり一緒に話す機会のないメンバーで情報交換などをする機会にしたいと思っています。

その他、時間割や懇親会、Lightning Talk の応募などについて詳細は modest サイトのイベント情報ページをご覧ください:
https://dev.mozilla.jp/events/workshop09

会場には無線 LAN 環境を用意しますが、帯域を占有する用途でのご利用はご遠慮ください。

勉強会にご参加いただける方は、右上の「このイベントに参加登録する」ボタンをクリックしてください。

今回は夕方からの開催となりますので、発表を聞きながら食べられる軽食をご用意します(参加費は不要です)。
また、開催時間は 21 時までと予定していますが、会場は 23 時まで確保しており、自由に残って何か続けたり移動して懇親会をすることも可能です。
