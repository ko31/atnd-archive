アナログゲーム会でもやりましょう！

概要

たまにはTRPG以外でも遊びたい。
ということで、普段あまり見ない種類のアナログゲーム（カードゲームやボードゲームなど）をみんなでわいわいプレイしてみる会です。
好評そうなら、月一ぐらいで開催してみてもいいかも。

日時

9月22日（木）　19:00～
昼から開催は人が集まらなさそうだったので、夕方から開催にしようと思います。
詳しい日程調整の様子は 調整さん を参照してください。

場所

tkhs研実験室（学部棟2号館5階J2526）　いつものところです。

持ち物

お菓子はストックがだいぶあるので、 飲み物を1人1本程度 持ってきてもらえると助かります。

プレイ可能なアナログゲーム
