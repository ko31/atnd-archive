★講演内容

「AWSで実現するエンタープライズ x IoTの勘所」をテーマに、
弊社での取り組みについて紹介いたします。

【スマートファクトリーを支えるIoTインフラをつくった話】

　本発表では製造業におけるセンサーデータを活用した
　スマートファクトリーのためのプラットフォームをAWS上に
　Amazon EMR (Apache SparkやApache HBase)や
　Apache Kafkaを組み合わせて構築を行いました。
　その際のAWSのサービス選定ポイントや実現にあたってのは考慮点、
　苦労話についてお話ししようかと思います。
　システム構成や実装上のポイントなど具体的な話を中心に行います。

【AWSマネージドサービスをフル活用したヘルスケアIoTプラットフォーム】

　現在ヘルスケア領域において、若者からお年寄りまで
　幅広い層をターゲットとしたIoTプラットフォームの構築を行っています。
　600万人ユーザ、数1000万デバイスといった規模になる
　本プラットフォームを実現するにあたって、
　Amazon Kinesis, Amazon EMR, Amazon DynamoDBなどの
　AWSマネージドサービスを積極的に活用しています。
　全体的なアーキテクチャを決めるまでの経緯や、
　実際に構築を進める上での課題やその課題に対するアプローチなど、
　技術的にディープな話をお届けいたします。


※ 内容は変更になる可能性があります。
