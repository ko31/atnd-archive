catch 会社の財政状況を一発で理解する方法
description

「商売は金儲け」と言われます。

では会社が「儲けのお金」を今までいくら稼いだのかご存知ですか？
また、その「儲けのお金」を現在いくら残せているかご存知ですか？

経理・財務が得意な方でも正しく把握できていないと思います。

会社で作成する財務諸表には、この「儲けのお金」はどこにも記載がないからです。利益と儲けのお金が一致する「資金管理会計」についての基礎講座を分かりやすくお伝えいたします。

運営：税活セミナー運営事務局

【セミナー内容】

□　儲けのお金は把握できます
□　資金管理会計でかんたん管理
□　お金の色分けを始めましょう
□　ハッキリ分かる資金の実態
□　資金繰りに困らない財務を目指して

【座談会】
