松戸WordPress部 第2回勉強会

ご関心のある方はFacebookグループ WordPress部 @ 松戸Banana Cluster にご参加いただけると
最新情報を入手することができます。ぜひご参加ください。（参加・退会自由）

隔週水曜日（2週間に一度）のペースでゆるーく集まって、仕事したり勉強したりセミナーを開いたりします。

いちおう時間を設定していますが、途中参加、途中退席完全自由です。
ご自分の作業や質問、情報を持ち寄って自由にお過ごしください。

素人さんも玄人さんも、初心者さんも上級者さんもお気軽にご参加ください。

WordPress.comで自分のサイトを作ってみよう！

第2回勉強会は、無料でアカウントを取得できるWordPress.comを使って実際にサイトを作ってみるハンズオン形式です。（ハンズオンとは、実際にやってみる体験学習です。）

パソコンをお持ちいただければ具体的なアドバイスや作業ができますのでその日のうちにWordPressを使ったブログサイトを開設できます。

日時：

2014年2月19日（水）18:00～20:00
主催者は16時ごろから利用しています。

場所：

松戸のコワーキングスペース BANANA CLUSTER
千葉県松戸市新松戸4丁目37 野沢天祐堂第2ビル5F
（新松戸駅から3分）

料金：

勉強会は無料ですが、 コワーキングスペースの利用料金が別途必要 となります。

  • 2時間利用：430円
  • それ以上の利用：1050円
  • 無線LANあり

詳しくは： http://www.banana-cluster.com/price
