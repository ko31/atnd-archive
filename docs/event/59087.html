description

ドル円6年10ヵ月ぶり安値、日銀緩和で売り続くのか…
アメリカ量的緩和終了発表の中、日銀は追加の金融緩和を発表し、このまま２０１5年も110円台の新たなステージへ向かうのか？？
そんな中、新興国通貨、資源国通貨はいったいどんな動きをするのか？？

もうこの流れの中では乗り遅れたと思われる方もいるはず。。
しかし、その中でも狙える通貨はあるはず、
低金利の時代だからこそ投資をしてみたいと思っている方　必見！！

今トレード中だけどなかなかうまくいかない。何が原因なのかわからない。。という方　必見！！ 外貨預金とFXの違いは？双方のメリット・デメリットとは？？　わからないことだらけという方　必見！！

ここだけでしか聞けない？話にも注目！！

FXトレードは、相場がどのように動いているかを知り、取引することが重要と語る水上氏！！

海外銀行ディーラーの中には、来日時に水上氏を訪問し、相場の見通しを聞きにくる人もいるほど。そこまで信頼を得る水上氏と他のディーラーとの違いとは？？

【手っ取り早く継続的に儲ける方法は無い！

　　　　　　　　　唯一の方法は基本を自分のものにしていくこと】
相場は｢通貨を買ってる人、売ってる人」で動いています。マーケット参加者の動きを先回りして読むことが、今後の相場を読むために必須です。
その投資家たちの動きを読むために３０年相場で培った知識、経験と共に、独自の相場情報を伝授します。

FX最強ポータルサイト‐ザイFX‐で為替のご意見番として活躍の松田哲氏
FX業者ではベストセラーを連発している著者の一人。
さらには羊飼いのＦＸブログ※1も登場する、ZAi FX!※2のＦＸ連載コラム「ＦＸ一刀両断！」も担当し、松田氏の虜になる人が多く、支持を集めています。
※1．ＦＸ情報満載の人気ブログです。
※2．一番売れてるマネー誌『ダイヤモンドZAi』が作ったFXの最強ポータルサイトです。

東京市場で最大規模の資金を動かすトップディーラーとして名を馳せた松田哲氏の長年の相場分析、データ、相場観を高次元で融合させた目から鱗のトレード手法を大公開！

サンワード貿易50周年の2014年最後のセミナーに満を持しての初登場！
サンワード貿易が年の瀬に贈る
2015年チャンスをつかむための緊急セミナー。

水上紀行氏プロフィール
バーニャマーケットフォーカスト代表。
東京外国為替市場で「三和の水上」の名を轟かす。
長年の経験と知識に基づく精度の高い相場予測には定評がある。
公式ブログはFXブロガーランキングのベスト５を常にキープ！（13年12月現在）

１９７８年、上智大学経済学部卒業後、三和銀行(現、三菱東京UFJ銀行）に入行。 １９８３年よりロンドンや東京、ニューヨークで為替ディーラーとして活躍。銀行ディーラー時代、当時の歴代年間収益最高記録を更新！ １９９６年よりＲＢＳ銀行にて、外国為替部長を経て、為替営業部長を歴任。 現在は、ＦＸストラテジストとして日経ラジオや金融情報誌等で活躍中！

松田哲氏プロフィール
早稲田大学法学部卒業。
三菱信託銀行、フランス・パリバ銀行、クレディ・スイス銀行などを経て、オーストラリア・コモンウェルス銀行のチーフ・ディーラーとして活躍。

現在は松田トラスト＆インベストメント代表取締役として、法人・個人向けコンサルティング業務を行っている。著書『外貨崩落』（技術評論社）などで米ドル／円相場、クロス円相場の下落を見事的中。 近著に『１勝９敗でも勝てる松田式FX！』（ダイヤモンド社）がある他、『FXで稼ぐ人はなぜ「1勝9敗」でも勝つのか？』（技術評論社）、『FXの教科書』（扶桑社）、『投資で浮かぶ人、沈む人』（PHP研究所）、『FX「シグナル」を先取りして勝つ!』（技術評論社）などの著書がある。
