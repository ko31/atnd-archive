Advent Calendarとは

かつて地球には、クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーがありました。21世紀にはこれが転じて、ギークコミュニティで、クリスマスと呼ばれる祝日までの25日間に1つずつ箱に閉じ込められた魂を公開する儀式が行われたそうです。これを Advent Calendar と呼びました。

かつての様子はこのようなものでした。

  • Reblog Advent Calend*e*r 2010
  • Reblog Advent Calend*e*r 3011
  • Reblog Advent Calend*e*r 7889
  • Reblog Advent Calend*e*r mach33
  • Reblog Advent Calend*e*r No.5
  • reblog advent calend*e*r episode VI

d*e*r とは

かつて、それはただの typo でした。

RAC ( reblog advent calend*e*r ) とは

RAC/history を、ご覧ください。

ルール

  1. リブログすること

注) ここでいう「リブログ」とは広義の reblog を指します。

参加資格

  • tumblr account を持っていること

参加方法

  1. このページの[このイベントに参加登録する]から登録してください
    • ユーザープロファイルか参加コメントで自分の tumblr の url がわかるようにしてください
  2. コメント欄に自分が担当したい日付を宣言してください
    • 日程は早いもの勝ちです
  3. atnd のメッセージ機能を使用して 会場 にログインする鍵をお渡ししますので、入場してください
    • メッセージが受け取れるようにメールアカウントを設定することをオススメします

競技規則

コンテンツを詰めた麻袋を武器に河川敷で殴り合ってるイメージ（もちろん無言で）

( via mtakeshi )

年に一度、コンテンツを詰めた麻袋を武器に河川敷に集まるイベントです（もちろん無言で）。 会場 はこちらで用意しました。

  1. 会場は RAC 専用の tumblr アカウント( tumblr group ) を使用します
  2. 12/1から順番に、一人一日を担当します
  3. 12/25 0:00 からはバトルロイヤルです。全員で踊ります。12/25 24:00 まで、参加者各位は思い思いにリブログするなりなんなりしてください
  4. 12/25 24:00 の鐘が鳴り終ったら 会場 は消滅します
