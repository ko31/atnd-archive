日本語教師に興味を持ったら、まずはこの講座から始めよう！

外国人に日本語を教える仕事。 これが、「日本語教師」です。

ここ数年、若い方だけでなく、主婦・定年後の方まで、幅広い世代に注目され、マスコミでも
多く取り上げられている職業の一つです。

しかし、その 「中身」となると、あまり知られていない のが現状・・・。

そこで、この講座では、日本語教師にご興味を持っている皆さまに、 現役日本語教師が

  • 世界から見た日本語教育の現状
  • 外国人に日本語を教えるって、具体的にどんなこと？
  • 日本語教師に求められるものとは何？

をテーマに、 日本語教師という仕事について、分かりやすくお話します 。

一般的な講座説明会では紹介されることのない、具体的な日本語教師の世界を知り、
あなたも、日本語教師への あこがれを具体的な第一歩に 変えてみませんか？

語学力、年齢、職業は一切問いません。 「興味がある！」が参加資格です。
