「この間テレビで〇〇が体に良いと言っていた」
「雑誌で海外の有名人が愛用しているらしい」
「値段は高ければ高いほど良いものに違いない」

現代の社会では、こうした様々なメディア情報が氾濫しています。

インパクトの強い情報につい振り回されてしまうことはないでしょうか。何を買えばよいのか、自分で判断できるようになるための正しい基礎知識を身につけ、普段の買い物のストレスをなくしませんか？

「原材料は気にしているけれど見方が分からない」
「野菜はどれも同じに見えるからとりあえず新鮮そうなものを選んでいる」
「良質な油って何？」

買い物をする上でのポイントや知識は学校では教えてくれません。自分・大切な人・家族が食べるものだからこそ、自信を持って買い物をしたい。だからといって高いものばかり購入は出来ない・・・

経済状況に合わせて自分が欲しいものを選ぶことも大切なスキルです。買い物をする上で知っておきたいポイントを実際にスーパーマーケットで見ながら学ぶ講座です。

▼おすすめポイント
・スポンサーや企業とのしがらみがない個人運営なので、正しい公平な情報を学べる。
・一対一なのでわからないことはその場で聞ける。
・栄養士、農家でもある食のプロから学べる。
