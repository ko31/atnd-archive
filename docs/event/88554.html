catch
description

ゼロからITを学ぶための講義形式の勉強会です。
社会人で身につかなければいけないITの内容を網羅的に学びます。

学ぶ範囲は情報処理入門編の国家資格である「ITパスポート」で扱う項目を対象とします。
具体的なプログラミング言語の手法や技術などを取得する勉強会ではないので、ご注意ください。

やる内容

今回はコンピューターの基礎について講義形式で勉強します。
全6回の予定ですが、途中からの参加もできる内容になっています。

第1回「コンピュータの基礎」
　・ハードウェア
　・ソフトウェア
・ビットとバイト
・2進数
・文字の表現
　・コンピュータにおける文字
　・文字コード
・マルチメディア表現
・数値の表現方法
　・符号
　・補数
　・小数

次回予定

第2回「ハードウェアとソフトウェア」
第3回「表計算・データベース」
第4回「ネットワークとセキュリティ」
第5回「業務システムとマネジメント」
第6回「ITと戦略」

日時

2017年6月12日（月）19時30分〜21時30分

スケジュール

時間 内容
18：00- 開場
19：30- 講義形式「コンピュータの基本」
21：00-21：30 ディスカッション& Q&A

※23時までオープンしており、勉強会後、作業等でもご利用いただけます。

対象

・これからITを勉強したい方
・ITエンジニアの新人
・ITの知識を広く浅く知りたい方
※文系、非技術系の方でも安心してご参加いただけます。
※初心者向けの内容のため、上級者の方は参加しても学べることは少ないかと思います。
