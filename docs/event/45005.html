description

老後生活費の柱である『公的年金』。
自助努力としての『生命保険』。

どちらも必要性を感じて加入しているものの、実は多くの方が内容を理解していないのではないでしょうか。

セミナーでは公的年金の基礎知識と、将来を見据えた賢い保険の入り方をわかりやすくお伝えします。

この機会に年金や保険のモヤモヤをすっきり解消しませんか?
基本を押さえておくときっと今後の人生のお役に立ちますよ!

【セミナーの内容】

セミナーでは公的年金の基礎知識や、長期視点で見た生命保険の入り方をわかり易くお伝えします。

・公的年金の仕組み
・老後の生活費はいくら必要?
・年金はいくらもらえるの?
・必要な自助努力について
・高齢時に役立つ保険とは?
・合理的でムダの無い保険の入り方
・貯蓄型保険と掛捨型保険の特徴は?
など

講師は銀行・保険会社・保険代理店で働いてきたファイナンシャルプランナー。
長年金融業界で働く中で『金融知識があるかないかで大きな違いが出る』ことを体感してきました。

公的年金・生命保険も重要な金融知識です。
将来の自分のために、今のうちから学びませんか?

セミナーは少人数制。質問も随時OKです。
アットホームな雰囲気で行っています。
金融商品の勧誘・販売は一切行いません。
お友達を誘ってお気軽にご参加ください。

【無料個別相談】
セミナー後、ご希望の方に個別相談を実施しています。(後日予約制)
・年金定期便の見方がよくわからない
・いま入っている保険の内容を詳しく知りたい
・ライフプランシュミレーションをしてほしい
などのお悩みにお答えします。

【参加者の声】
