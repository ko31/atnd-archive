description

勉強会に参加したのはいいですが、内容を忘れていませんか？

たとえば、 勉強会に参加した。いっぱい学んだ。明日から使えそうだと思った。 とかで終わっていませんか？

本当に使ってますか？ 一生懸命取ったメモを読み返してますか？ 今週学んだ事を人に説明できますか？

このブログdeもくもく会は、そういった疑問を自分に持つ人が率先して参加してアウトプットをすることで、自分に自信をつける場です。

こんな人が向いてます

  • 翌週には勉強会で学んだ内容を忘れてしまうのできちんと身につけたい
  • 自分で書いたメモを振り返った事がない、振り返る機会がない
  • ハンズオンなどで出来なかった範囲を進めたい

やること

  • 年頭所感じゃないけど、今日やろうと思っていることをさらっと
    • 自己紹介
    • 何の勉強会に参加したか
    • 今日の作業をどうやって進めていくか
  • 同じ勉強会に参加した人がいれば共有会やると学びの質が上がります
  • 興味があったけど行けなかった、みたいな勉強会のアウトプットがあれば、その人に聞いてみましょう
  • 書けたら発表とついでに宣伝しちゃいましょう

場所

コワーキングスペース秋葉原Weeyble （東京都千代田区神田須田町２丁目１９−２３ 野村第３ビル ４階）

最寄り駅・道案内
JR秋葉原駅
  • 最短ルート：電気街口からエディオン方面へ。突き当たったら左の橋超えてすぐ左にいきなりステーキ。
  • 中央口から東西自由通路を抜けて電気街口へ。もっと早いルートもありますが慣れないうちは避けましょう。
  • 昭和通り口ならメトロB5出口方面へ向かってカメラのキタムラの交差点を右へまっすぐ、高架を抜けた先です。
メトロ日比谷線：秋葉原駅

つくばエクスプレス：秋葉原駅

困ったときは

秋葉原のいきなりステーキを目指してれっつごー。

参加枠について

なんとなくこういう人でこれぐらの人数を想定しているだけです。人数制限は会場のキャパシティに依存しますので、枠は増やせますし、動かせます。 枠を超えていても気にせず申し込んじゃってください。

時間について

途中参加・途中退出OKです。 普通にドロップインでもご利用できます。

会場について

  • 入場証などは必要ありません
  • イベント時間中はビルの入場口が開放されていますので事前準備はありません。
  • 電源やWifi環境がありますので、お持ちの端末と充電器のみで大丈夫です。
  • 飲み物は会場から提供できます。（別途費用が発生します）
  • 近くにコンビニがありますので、他に必要なものはそちらでどうぞ。

費用について

会場のドロップインの費用です。当日参加・当日キャンセルでもペナルティは発生しません

募集

  • リモート参加も受け付けたいのですが、恥ずかしながら主催はSlackの使い方を理解していないので、Slack参加枠が作れません。そのため当日なりSNSで指導してもいいよ、という人がいれば、大変ありがたいです。
  • ブログのモチベーションや勉強会でやってることとか、LTじゃないけどシェア会とかやりたいと思ってます。やるかどうかはその場のノリで決めます。
