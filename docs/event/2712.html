InfoTalkは，ICT関連の熱い技術，面白い活用等を取り上げ，いろいろと議論したりする場（勉強会&交流会）です。COP15（国連気候変動枠組み条約 第15回締約国会議）がコペンハーゲンで開催され，各国の温室効果ガスの排出削減計画が議論されました。この削減計画は
排出権取引という仕組みまで準備され，数十兆から数百兆円規模の市場が生まれるというレポートもあります。しかし，温室効果ガスは，どこからどのようにどの程度発生しているかがわかりにくく，排出権取引も掴みどころが無いというのが実感でしょう。しかし，排出権取引の実際は，ITが極めて活躍する領域です。今回は宮沢さん（東京都）と近藤さん（株式会社NTTデータ）から，このあたりの興味深い話題をいろいろと提供していただきます。産業技術大学院大学の関係者以外の御参加も歓迎いたします。是非，御参加ください。


  • 「大規模事業所への温室効果ガス排出総量削減義務と排出量取引制度」

    宮沢 浩司 氏＝東京都 環境局 都市地球環境部 排出量取引担当

    東京都は，日本で初めての温室効果ガス総量削減義務と排出量取引制度を導入し，来年4月から総量削減義務を開始する。今回は,
    改めて制度導入の経緯や特徴を説明するとともに，この制度のスタートに向けた準備状況や今後の動き等も報告する。

  • 「排出量取引とIT」

    近藤 亘 氏＝株式会社NTTデータ 第二公共システム事業本部

    低炭素社会の実現に向けた取り組みのなかでも特に注目を集める排出量取引。京都議定書の京都メカニズム，国内排出量取引制度，自治体の排出量取引制度等の様々な制度の運用が始まっている。これらの制度を円滑に運用するためにITは必要不可欠である。今回はこれらの制度とITについて事例を交えて紹介する。
