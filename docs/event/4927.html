いまさら他人に聞けないFlashの基礎的な何かを勉強するひよこの会in大阪です。
超初歩的すぎて引っ込み思案になっちゃうようなあんな問題やこんな問題を積極的にぶつけてみんなで勉強していきましょうという趣旨です。
いつかは羽ばたく鳥になろう！

概要

出来たばっかりでまだ方向性とかも決まってないので、先ずは顔見せ＆方向性とかの意見交換会をやりませんか。
最初なので、今後どんなことをするのかを決めるための話をしようかなという段階です。
株式会社オノフ様より会議室をお借りすることが出来ました！！
ちなみに勉強会とは直接関係ないですが場所を提供頂いたオノフ様は現在Flashクリエイター募集中との事です。

今回の第一回目は勉強会の方向性を決めるための話し合いですので、勉強会そのものを行うわけではありません。念のためご注意を

対象者

Flashを勉強したい全ての人。ひよこの会は初心者の会です。
今のところActionScript3の話題がメインになりそうな雰囲気ですが、タイムラインとかFlashそのものでもOKですよ！
スキル問わずですが、できれば積極的に質問をぶつけられる人が大歓迎です。

お勧めの装備

素手（ステゴロ）でも十分ですが、ノートPC、電源タップなどがあるといいかもしれません（最初なので使わないかもしれません）。
