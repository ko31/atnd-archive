第12回R勉強会＠東京（#TokyoR）を開催します！

ATNDは、連絡する手段がないため、グループへの参加をお願いします！

  • http://groups.google.com/group/r-study-tokyo

当日、ISMSと怪我や病気による緊急対応のため、ニフティさんに渡す名刺と名札用の名刺2枚を用意していただきたいとのことです！
よろしくお願いします！

内容

  1. 自己紹介タイム（30分）
    • 1人1分くらいで簡単に
  2. @yokkuns : Rによる回帰分析入門（30分）
    • Rによるデータサイエンス第Ⅱ部 第7,8章 回帰分析
    • データ解析の基本である回帰分析について、Rを使って学んでいきます。
  3. @sleipnir002: Ｒで学ぶデータサイエンス 5パターン認識 1章 判別能力の評価（15分）
  4. @teramonagi : Rで学ぶ回帰分析と単位根検定（30分）
    • 単回帰分析を復習した後、単純に回帰分析を適用してはいけない『やってはいけないケース』を紹介。
    • そしてそれがなぜ起こるのかを実例を通して紹介した後、この問題を避けるための方法の１つという観点から単位根検定の紹介をします
  5. @aad34210 : RをSQLで操る（20分）
    • データベース（RDB)を利用している人でもRがとっつきやすくなる！
    • dataframeをSQLで操作ができる「sqldf」パッケージを紹介。
  6. @sfchaos : Rあんな時こんな時　～いつか役立つ(かもしれない)Tips～（20分）
    • Rを使っていてこんなことをやりたいとき，こんなことに遭遇したとき，あなたはどうしますか？
    • クイズ形式で進行しながら，Rを使用する際のちょっとしたコツについて議論させていただければと思います
  7. @dichika : 世界のR(仮)（10分）
  8. Rなんでも相談コーナー（30分）
    • 初心者が実際に使っているメンバーとかに相談とか出来る場にしたいと考えています。
