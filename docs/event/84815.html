description

　この度ヒューマンリソシアでは、建築設計業界での就業を目指している方向けに、
　新たな設計手法と言えるBIM（Building　Information　Modeling）のスキル習得を
　目的としたセミナーを開催いたします。

　詳細はこちら
　http://resocia.jp/skillup/view/30111020711/14919

　今回のセミナーでは、建築設計業界のリーディングカンパニーの株式会社日建設計より
　BIMマネージャーの濱田　祐也　氏を講師としてお招きし、
　BIMの活用方法と学ぶべきポイントを5つに絞って解説して頂きます。

　普段ではなかなか聞けないような設計現場でのやりとりの話も交えながら、
　これからBIMを学ぼうとしている方に、
　
　「こんなことを意識して学ぶべき」
　
　という観点で、BIMの実践とコツを教えてくれます。

　建築設計業界において、キャリアアップやジョブチェンジをお考えの方、
　業界最大手の日建設計で働きたいと思っている方、必見です。

　まだBIMを触った事が無い方も、これから使用する予定の方も幅広く参加できる内容になっています。
　お気軽にご参加ください。

　お申込みはこちらから
　http://resocia.jp/skillup/view/30111020711/14919

◆セミナー概要

　・建築業界と時代の流れ
　・なぜ今BIMなのか
　・BIMを活かすには
　・クライアント／社会に貢献するBIMのあり方

　・質疑応答
