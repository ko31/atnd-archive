このページについて

NGK忘年会2010（昼の部） のLTスピーカー募集用のページです。スピーカーとして参加する方も別途 NGK忘年会2010（昼の部） にお申し込みください。（参加人数管理がめんどくさいので。。。）

申し込みにあたって

  • ライトニングトークは厳密に5分で区切ります。もし5分より早く終わってしまっても、5分に達するまでは放置されます。
  • 発表テーマについては特に指定はありません。
  • 発表テーマが決まりましたら、コメント等でご連絡ください。
  • どうしても5分以上話したい方（名古屋式ライトニングトーク希望の方）はご相談ください。
  • 発表順序はランダムです。1週間前をめどに発表順序を発表する予定です。ただし、都合に合わせて調整は可能ですので必要な方はご相談ください。

スピーカーの方へのお願い

ライトニングトーク後、休憩時間をかねて30分程度のスピーカーへの個別質問タイム（オフィスアワー的なもの）を確保しようと思っていますので、ご協力をお願いします。
