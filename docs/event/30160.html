description

アナログもディジタルもCPUも，全部一緒になってるデバイス「PSoC」に心をくすぐられている方，いらっしゃいませんか．
2012年にサイプレスから最新版「PSoC5」が量産される今，またCQ出版からPSoC1，PSoC3の本が発売されてそろい踏みとなった今，PSoCにもっと近寄れるチャンスです．
そこで，PSoCを使った作品を各自持ち寄り，PSoCについて情報交換会をしませんか？

本イベントでは「アナログ・マイコン！？PSoCに目覚める本」の著者である高野慶一さんと，サイプレスの栗川さんをメイン・ゲストに迎え特別講演を開きます．
参加ご希望の方は，PSoCで製作したものや，ちょっとした相談など，
どんなことでも構いませんので，自分だけのミニ発表をご用意くださいませ．
（目安として一人max7分くらい）

日時　2012年7月21日(土) 14:00-17:00（13:30開場）
会場　ガジェットカフェ (高円寺駅 徒歩5分)
対象　トラ技読者またはトラ技，PSoCに興味のある人
定員　30名
出演　ゲストスピーカ　高野慶一（アナログ・マイコン！？PSoCに目覚める本の筆者）、栗川洋平（日本サイプレス）
主催　ガジェットカフェ事務局，協賛：CQ出版トランジスタ技術編集部
会費　1000円（ドリンク代込み）

プログラム：
13:30 開場
14:00 トラ技オフ会のあいさつ＆かんぱい（CQ出版エレクトロニクス編集部員）
14:10 みなさんの自己紹介＆ミニ・プレゼン
15:30 ゲスト・プレゼン
16:10 懇親会
17:00 終了
