◆概要

・日時：5月17日（土）15:00〜16:30

・内容：主催者や経験者を交えてのフリー交流会（70分）
　　　主催者から留学術、英語学習方法のご紹介（20分）

・定員：10名、年齢層20代～30代

・参加費：無料

・会場：株式会社ソラド　会議室
　　　東京都中央区日本橋茅場町3-13-6　第二日高ビル4階

◆内容

語学留学を目指す社会人の方、気軽に楽しく交流しませんか？

会社にはまだ言えないけれど留学してみようと密かに思っている方、
留学してみたいけれど一歩踏み出せない方。

30歳過ぎて会社を辞めて米国留学した私の経験を交えながら、
気軽に話せる会を開催したいと思います。

エージェントを使わない留学の仕方、留学中の就職活動など
社会人だからこその経験や、楽しく賢い英語学習法なども
ご紹介したいと思っています。

同時に、社会人留学をすでに経験した方で
自分の体験からアドバイスをしたい方＆交流されたい方も
ぜひご参加ください！

http://kokucheese.com/event/index/173082/
