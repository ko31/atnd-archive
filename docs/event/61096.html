学費、宿泊、食費込で、1カ月約13万円！
しかも完全マンツーマン授業！

夢のような英会話環境が格安で手に入ります。

そんなフィリピン語学留学に隠された秘密を1月17日土曜日に皆さんにお話しします。

更に、お話するだけではなく実際にフィリピン留学を体験してもらう企画も盛り込んでるので、楽しい時間を過ごせると思います。

私自身、フィリピンとアメリカでの留学経験があるので、体験談も交えながらしっかり比較させていただきます。

留学を経験して思ったことは、もっと行く前に準備をしておくべきだったという事です。
海外留学は、あくまでも英語習得の一つの手段にしかすぎませんし、長く滞在すればいいというモノでもありません。

また多くの方がとりあえず、留学をすれば英語がペラペラになると考えておりますが、決してそんな事はありません。
留学する前に、しっかりと学習プランを計画し勉強を始めるべきです。

出来る限り日本でやれることは、しっかりと勉強しておき海外でしかできない勉強に最大限時間を割くべきだと思います。

この辺りも含め、これから海外留学を検討されている方へ、英語を学ぶ効率的な方法、上手な留学の使い方などを伝えたいです。
英会話を伸ばしたい方、留学を検討されている方々にぜひ来ていただきたいです。

【概要】

・留学で英語が上達する人としない人の違い
・英会話学習のステップ
・欧米留学との違い（体験談込）
・フィリピン留学とは何か？
・フィリピン留学のメリット、デメリット
・具体的なフィリピン留学プログラム

説明会自体は約1時間で終わる内容となっております。
その後、個別相談も随時行う予定ですのでドンドン質問してください。
