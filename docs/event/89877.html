今年の夏は【MUSIC BEYOND】へ！

小さな頃から音楽は好きだけど、最近はめっきり新しいミュージシャンの曲を聴かなくなった。
ライブは好きでよく行くけれど、同じジャンルや音楽スタイルばかりで、少しだけど飽きてきた。
そもそもライブの良さが分からない。

そんなあなたに、この夏ぴったりの音楽イベント【MUSIC BEYOND】をお届けします！

パンクロッカーから弾き語りの道へ、日常のどうしようもない悔しさや厳しさを歌の力にかえて、それでも逃げずに生きていく強さを、ときに激しく、ときに儚く響かせる、魂のシンガーソングライター 『赤塚テレビ』

その音楽性はＣＤジャーナルで「アルバムで聴く21世紀の日本のアイリッシュ / ケルト音楽」（http://www.cdjournal.com/main/cdjpush/-/2000000789）に名を連ねるほどの実力、アイルランドの伝統音楽＆オリジナル曲を演奏するガールズユニット 『きゃめる』
今回は7/23にニューアルバム『Wonder Garden』をリリースした直後のライブということで、どんなステージを見せてくれるのか、さらに期待が高まる

そして15才の頃から作曲を始め、ライブ活動やゲームミュージッククリエイターを経て、ボブマーリーやジャミロクワイのような自然と身体が動き出すグルーヴィーでソウルフルなサウンドを聴かせてくれるのが、アメリカ出身のシンガーソングライター 『BINN SKINNI』

いずれ劣らぬ実力を持つミュージシャンたち
ご来場されるお客さまには、素晴らしい音楽は国境や言語を越え、もっと自由でもっとパワフルで、そしてもっとキラキラ・ワクワクしたものだと実感していただけると思います

この夏の素敵なスタートに、ぜひ皆さまのご来場をお待ちしております

※なお、予定枚数を終了次第、チケットはSOLD OUTとなりますので、ご予約がまだの方はお早めに
