アート滞在プログラムコーディネーター育成研修

catch 平成21年度地方の元気再生事業「多様なNPOの連係による新しいスタイルの滞在プログラムの創出事業」
description

「アート滞在プログラムコーディネーター育成事業」とは別府を訪れる方に別府でアートを楽しむ滞在プログラムを考案し、提供するコーディネーターを育成する事業です。

この事業は、別府市元気再生協議会が内閣府の助成を受けて実施している、平成21年度地方の元気再生事業「多様なNPOの連係による新しいスタイルの滞在プログラムの創出事業」の一環としてNPO法人BEPPU PROJECTが運営主体となり実施されるものです。

今春開催された別府現代芸術フェスティバル2009「混浴温泉世界」など、別府とアートが出会うことで新しい別府の楽しみ方が生まれます。別府のまちでアートを楽しむことが定着していくことは、これまで別府を訪れる機会の少なかった人が訪れる機会を生み出し、またアーティストなど制作する人をまちに呼び込むことにつながります。

「アート滞在プログラムコーディネーター育成事業」は、別府とアートを結びつけていく担い手を育成する研修会です。育成された人材は、 NPO法人BEPPU PROJECTや別府市元気再生協議会と連携しながら事業活動を行っていきます。

研修の概要
人材育成研修は「事業ガイダンス」「関連法規について」「企画ワークショップ」「登録の仕方」の計4回開催され、平成21年度においては10月中旬からのサービスを予定しております。
※各回の研修内容はリンク先を確認ください。（現在準備中）
※現時点（平成21年7月）において決まっているプログラムです。変更になる場合もありますので、あらかじめご了承ください。
