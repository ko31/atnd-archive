概要

Visual Studio を使っている方、興味を持っている方で集まりましょう。

  • 趣味や実務の開発作業をするも良し。
  • 人に聴いてみたい疑問や悩みをぶつけあうのも良し。
  • 普段、開発作業をひとりでしているとか、会社の中でしかしたことがないとかで、他の人はどうしてるん？を聞いてみたいってのも良し。
  • Visual Studio 初挑戦！も良し。

雑談ベースで、ゆるくゆるく…。

対象

  • Visual Studio を頻繁にでも、たまにでも、実務にでも、趣味にでも使っている方
  • Visual Studio を使ったことは無いけど、興味をお持ちの方、挑戦してみたい方
  • 業種・経歴、年齢・性別に制限はありません

ゲスト

Windowsアプリ開発・販売や、SharePointやOffice365を活用するためのクラウドサービスを展開する、
Exceed1 の野呂さんがお越しくださいます。

開発ノウハウが聞けちゃうかも知れませんよ！
