description

このゲーム会を通じて、
お金に関する知識、自分が本当に望んでいる未来、日本の現状、
など何かひとつ、発見・気づきを得てもらえればと
定期的に開催しています。2014年4月から消費税が8％に増えましたね。
消費税が上がったことにより、お財布からお金がより多く飛んでいきます。
それにより、5％だったときに比べ外食の回数が減った方、いらっしゃると思います。

出ていくお金は増えるのに、給料は上がらない。
つまり、これから先、あなたは何か行動をしなければ、
今楽しめている何かを、節約せざるを得ない状況になるでしょう。

世の中は、
このままじゃいけないんだけど、何をしたらいいかわからない人や、
自分にはできないとあきらめてしまっている人たちで
溢れてしまっているように思えます。

そういう人たちのために、キャッシュフローゲーム会はあるのです！

もし、迷っているのであれば、
よちよち歩きの一歩、出してみましょう♪

『キャッシュフローゲーム』とは
『金持ち父さん』シリーズの著者ロバート･キヨサキが作成した、
楽しくゲームをしながらお金に関する知識や考え方
（ファイナンシャル・インテリジェンス）が身につくという画期的なゲームです。

このゲームでは
お金持ちになるまでの疑似体験と
お金持ちになってからの疑似体験ができます。

キャッシュフローゲームを実際にプレイすることにより、
財務諸表の読み書き、不動産投資、株式投資、ROI等の数字の見方、良い借金と悪い借金 etc.
について楽しみながら学んでいきます。
