夢がひろがりんぐなことに定評のあるBoost C++ Librariesの勉強会です。




内容としては、


・Boostに興味があるけど触ったことがない

・バージョンアップについていけなくなった

・全容を知りたい




という方のためにまず、最新バージョンであるBoost 1.40時点での(なるべく)すべてのライブラリを紹介しようと思います。


(担当：私=id:faith_and_brave)





それに加え、もっと深くBoostを知りたい、という方向けに
ディープなセッションも用意する予定です。




ぜひご参加ください。




スピーカーも随時募集しておりますので

Boostについて熱く語りたい方はぜひご連絡ください。

(PStade, Hamigaki, CLXライブラリなどもOK)
