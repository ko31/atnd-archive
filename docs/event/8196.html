Git勉強会 #2

長野ソフトウェア技術者グループ 勉強会の平日夜の部です。

前回 に引き続き、分散バージョン管理システムの Git について勉強しましょう。1回目に参加してない人もお気軽にどうぞ。

前回はまったく資料なかったのですが、今回は簡単な資料を作ったので、20〜30分ほどそれを話すつもりです。その後自由に勉強という感じで。
他に Git について話してもいいという人がいたらお願いします。

今回も書籍『入門git』『入門Git』『実用Git』を持っていくので、それを見てみたいという人もOKです。
