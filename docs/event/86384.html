【スペシャルゲスト２名の登壇決定！！】
　
４月２３日（日）品川駅からスカイウェイで直結3分
シャンデリアがある結婚式の2次会で使われている明るくて素敵な会場で、素敵なゲストの方＆電子書籍のベストセラー著者８名（予祝）と一緒に愛と感謝に溢れた空間を楽しみませんか？
　
　
十人十色の人生を味わい、自分自身と向き合い、新たな自分で人生を生きるスタートの日を一緒に過ごしませんか？
　
　
１．【スペシャルゲスト】小玉泰子さん

（まなゆいの創始者、Precious One 代表）

まなゆいは1人でやることもできますが、応援し合ってやることもできるメソッドです。当日は参加者同士、応援し合って、お互いの夢をサポートする時間をつくります。応援されること＆応援することの喜びとパワーを感じ、出会った人同士が仲間として、これからもつながっていく場になります！皆で、自分を生きて未来を創るメソッドです。

当日は、まなゆいの体験、１０名の方の愛と感謝にまつわるエピソード、そして、数ヶ月前まで、まさか自分が出版なんて思ってもみなかった著者８名の方が、３ヶ月間自分自身と向き合って、出版したことで、どのように人生が変わったかについてお話いただく予定です。

【こんな悩みはありませんか？】

・最近、感動していない

・自分が心からやりたいことが何だかわからない

・毎日仕事が終わったら、疲れて何もやる気がしない

・このまま今の仕事や生活を一生続けていって後悔しないだろうか？

・自分には特段とりえがないと思う

・つい自分と他人を比べてしまって落ち込む

・何かやろうと思っても、いつも三日坊主で終わってしまう

・やりたいことはあるけど、行動に移す自信や勇気がない

・自分の役割/担当がわからない

これまでがんばってきたのにも関わらずこのような悩みがあるとしたら、１日しっかり自分と向き合う時間とり、同じような悩みを解決してきた先輩の話を聞き、まなゆいを実践し、お互いに応援し合うことで、翌日から、本当の自分で、新たな人生を踏み出すことができます。

　
4/23（日）「本の日」に、あなたもベストセラー著者８名＆スペシャルゲストと同じ空間で同じエネルギーを感じませんか？
