「サムライマガジン」「アプリStyle」などでの特集への協力多数。
また、AppleStore銀座店やスマートフォン&モバイルEXPO、
RainbowAppsスクールなどで、iPhoneアプリ開発者向けのセミナーや
勉強会での講演も多数。


株式会社オクトバ代表取締役　丸岡和人

2005年 慶応義塾大学環境情報学部卒業

2010年~ Android関連の講師・講演・執筆・コンサルティング・TV出演など多数


株式会社まさか（準備中）代表取締役　孫 良

1978 年生まれ。在日韓国人３世。iPhone アプリの企画者・プロデューサー。
AppStore 無料総合ランキング日本一４回獲得。この他に５位以上も８回。

サッカー専門誌にユーラシア大陸横断旅行記を連載するなど08年までは

「限定公開！　スマホ業界のイス取りゲームに勝つための秘訣」
～成功への最終電車に乗り遅れないために必要なこと～

19時35分～20時15分（１コマ20分×２）

▼講演形式


※講演者／株式会社YourGolfOnline代表取締役　才神健司（ゴルフというジャンルについてAndroidを中心に）

→黎明期から成長期に入りつつあるスマートフォン＆アプリ業界は、「今なら間に合う」もののうかうかしていると成功に向かう最終電車に乗り遅れてしまうという状況です。

そんな中「いまどのような戦略をとればいいのか？」という点について、既に自らの位置取り（イス取り）に成功しつつある起業家、担当者の方々が講演を行います


■登壇者プロフィール■———————————————————-


株式会社YourGolfOnline代表取締役　才神健児

2008年からAndroidアプリケーションの企画、開発に取り組み、2009年にゴルフスコア管理アプリ「YourGolf」をリリース。
現在は世界188ヶ国で70万人以上のゴルファーに愛用される、世界No.1のゴルフスコア管理アプリ。
2011年には、YourGolf専業会社「YourGolf Online Inc.」を設立。
YourGolfをベースとした各種サービスの企画立案から開発、プロモーション、カスタマーサポートまで全般を統括。


株式会社ポッケ代表取締役社長　廣瀬 周一氏

1976年生まれ。学生時代からベンチャー企業でアルバイトを始める。
卒業後はベンチャー企業に新卒として入社。企画営業として各種サービスを立ち上げる。
その後、所沢市の市議会議員選挙に出馬するために退社。
残念ながら落選となり、仕事での付き合いがあったベルシステム24に入社。
ベルシステム24では、初の占いではないコンテンツ「妊娠・出産・育児サイト・ママニティ」を立ち上げ、ライフスタイル事業という柱をつくる。コンテンツ事業全体の責任者を経て、事業部の新設分割に伴い、株式会社ポッケの代表取締役に就任。

■登壇者プロフィール■———————————————————-


株式会社スパイア取締役 兼 執行役員スマートフォン事業本部長　手嶋浩己氏

1999年から2005年まで博報堂にて、マーケティングプランナー/ブランドコンサルタント。その後インタースパイア取締役副社長を経て現職。これまで、モバイル広告、アドネットワーク、モバイルメディア、コマース、リサーチなどの事業責任者を歴任。2011年よりスマートフォン事業に集中、立ち上げ中。共著書にスマートフォンチャンス！、Mobile2.0など。


エキサイト株式会社　スマートフォン推進室　室長　岡田英之氏

1979年フランス生まれ、東京都三鷹市育ち。
エキサイトにてiOS / Android / Windows Phoneのプロデュースを担当。

2009年2月にアプリの提供を開始。
2012年12月現在、110超のアプリを提供中。累計400万ダウンロード超。
App Store無料総合ランキング1位を3回獲得。Appleの国内TVCMで2回採用。
料理レシピ、絵本読聞かせ、翻訳、ニュース、音楽など、生活に密着したアプリが中心。
代表作は「エキサイトニュース」、「エキサイト英語翻訳」、「みそ汁レシピ一年分」、「おとえほん」、「AKB48」、「機長席」など（＊順不同）。
世界中の毎日の生活を楽しくするアプリの提供を目指して日々奮闘中。米国1位を非ゲームアプリで獲るのが野望。二児の父でもあり、週末に動物園を攻めるのがライフワークでもある。

株式会社ジェネシックス取締役　冨田憲二氏

神泉ではたらくベビースターラーメン。スマートフォンアプリを開発する株式会社ジェネシックス(genesix, Inc.)取締役。東京農工大学大学院修士課程卒。海外スマートフォン/スタートアップ系のブログを書いてます>>http://tommygx90.posterous.com/
