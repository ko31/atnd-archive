第3回ビジネスプラン実践道場2012 　満員御礼・受付終了

イベント ユーザー
event_id 24711
title 第3回ビジネスプラン実践道場2012 　満員御礼・受付終了
catch 「もしあなたがベンチャー経営を任されたら？」
description

起業と経営を学ぶビジネススクール、SBI大学院大学（学長：北尾吉孝）では「もしあなたがベンチャー経営を任されたら？」というテーマで公開ワークショップを開催します。
事業企画の立案・実行の指導を専門とする講師陣による、楽しくかつ実践的な構成になっており、アイデアからプランへ、プランから実行へ、のステップを体感できるセミナー内容となっています。
参加にあたり、必要な資格や条件はありません。

プログラム：
第一部　　
・課題事業案件のビジネスプラン発表
・「成功するビジネスプランとは？」 講師陣によるポイント講義

第二部
・グループワーク形式による課題プランの改善案作成実習（ランチミーティング形式）
第三部
・改善案発表 講師講評
・参加者による投票と優秀案の発表
第四部
・懇親会

講師：（五十音順）
佐藤太一郎（SBI大学院大学講師　株式会社フロイデ　パートナー）
早川智也（SBI大学院大学講師　プロジェクトオーシャン株式会社 代表取締役）
吉田宣也（SBI大学院大学講師　日本MITエンタープライズフォーラム 特別顧問）

　　　参加費：3,000円（事前申込制・昼食代懇親会費含む)　　
定員：45名(残席希少！）
特典：参加者全員にDVD『ＳＢＩユニバーシティで「何を伝え」「何を学ぶべきか」～人間学と経営学』または北尾学長著作本をプレゼントいたします。

受付終了しました！
