NOMIKAI OSAKAとは？

株式会社ジョイントメディアの加藤主催の交流会で, 毎回10名前後の方が集まります。ゲーム業界やゲームに興味のある方, またはクリエイターの方などが集まって色々と情報交換をする場として開放しております。新規の企画が生まれたり, 就職活動中のクリエイターさんと求人中の企業さんが意気投合してそのまま就職が決まったケースもありました！交流会の目的は業界内外での情報交換ですので, 美味しいお酒やフードを頂きながら, 関係値を築いて頂ければ幸いです。

お店は大阪の本町にある「クロスビート」さんを使わせて頂いていて, 加藤主催の交流会といえば「いつもの場所」になりつつあります。そういえば結婚式の二次会としても使わせて頂きました！

お店についたら

お店の入り口にバーカウンターが御座いますので, そこで「加藤さんの交流会への参加です」という旨を伝えて頂ければOKです。受付なども存在しませんので, 飲み放題かキャッシュオンでドリンクをゲットして乾杯しましょう。おいしいウィンナーやポテト, バケット, パスタなど … フードもご用意しておりますのでマスターに聞いてみてくださいね♪

当日の流れ

遅刻もOKですので, 当日のタイムスケジュールは存在しません。来場頂いた方に, その都度システムをご説明させて頂きます。

  1. ドリンクを購入して下さい
  2. ご挨拶に乾杯をしましょう
  3. 当日, 来場者名簿を作成します。
    よろしければエクセルに自己PRを記入して下さい！
    後日, 記入頂いた方にメールで名簿をお渡し致します。
  4. ある程度の人数が集まったらスピーディに紹介をします
  5. しかし基本は当日のノリで！　楽しみましょう。

参加資格

基本的には職業や性別, 国籍問わずどなたでもご参加頂けます！ただし, お酒を提供する場所ですので未成年者のご参加は控えて頂いた方が良いでしょう。どうしてもご参加したいという場合には, 保護者同伴で遊びに来てください。

帰宅時も安心

女性の参加者さんがいる場合, 駅まで送迎もしますので安心です。最寄り駅は「堺筋本町」もしくは「本町」です！
