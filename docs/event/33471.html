description

Red Bull 5G ぷよぷよ大会の開催を記念して、大阪でぷよぷよオフ会を開催します。

※11月10日（土）と11月11日（日）の両日に開催する予定でしたが、10日（土）だけの実施に変更しました。

大会に合わせてWii/DSの『ぷよぷよ！！(20th)』通ルールでの練習をメインに考えていますが、会場では基本的に自由に遊んでもらって構いません。他のゲームもOKです。
大会に出場しない人も参加していただけます。途中参加、途中退室可能です。カラオケはしません。

参加費：1200円
※会場のランチフリーコースの料金です。

会場：カラオケZERO4 南方店
※ドリンク＆アイスクリームフリー。12～15時はランチバイキングがあります。
※最寄り駅は「JR新大阪駅」「阪急南方駅」「大阪地下鉄 御堂筋線 西中島南方駅」です。

持参品：持っていればDS等の本体と『ぷよぷよ』シリーズのソフト、充電ケーブルなど （なくても大丈夫です）
※Wiiを会場のモニターに接続して遊びます。Wiiの本体とソフトはイノセンスが用意します。

会場では1部屋に最大10人程度しか入れないので、2部屋使用します。
参加人数が10人以下の場合は1部屋にします。

当イベントに関するお問い合わせはTwitterでイノセンス @inosendo までお願いします。
