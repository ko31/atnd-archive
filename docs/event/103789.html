description

視聴者とのコミュニケーションが図れる施策として、人気のライブ配信。
企業利用もずいぶんと進み、今では多くの企業でマーケティングや社内教育の方法として取り入れられています。

スマホやアプリでも気軽にできるライブ配信ですが、会社として利用するとクオリティに不安があるのも事実です。

そこで今回、クロスコが初心者・ライトユーザーを対象とした、企業ユース向けライブ配信の体験会を行います。

弊社在籍のライブ配信ディレクターがYouTube Liveを使った配信の基本設定、必要な機材の使い方をレクチャー。
PCを使ったライブ配信と、分かりやすく伝えるための設計・演出の方法が学べます。
さらにセットした機材を使って、企業ユース向けのライブ配信を無料で体験可能です。

▼このような課題、興味をお持ちの方はぜひご参加ください。
