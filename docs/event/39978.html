ベンチャー企業に監査は必要？

～監査法人の現役会計士が語る監査の意味～

イベント概要

「会計監査って何の意味があるんですか？」そんな素朴な質問からこの企画は始まりました。
株式上場のためにだけ監査が存在していると思っていらっしゃる方も多いと思います。
そもそも何で監査が存在するのか？監査を受けるとどんな効果があるのか？等、
監査現場で起きる実例を交えながら監査法人の現役会計士が語ります。

推奨参加者

  • シード/アーリーステージのスタートアップ企業のCEO,CFO
  • これから起業を目指す方

プログラム(随時更新します)

  • １．はじめに（5分）
    株式会社nomad 代表取締役　小笠原　治　
  • ２．「監査は何をするの？」（30分）
    新日本有限責任監査法人　ＩＰＯグループ
    公認会計士　加藤　慶
    監査は何をしているのか、会計士と税理士の違い、監査と税務調査の違い、監査は何故生まれたのか等についてお話します。
  • ３．「あなたの会社に監査は必要か？」（30分）
    新日本有限責任監査法人　ＩＰＯグループ
    公認会計士　阿曾　友淳
    監査が必要な企業とは、監査はどの企業も受けることができるのか、監査によって会社はどう変わるか等について、最近のIPO企業の実例も交えてお話します。
  • ４．会場からの質疑応答（25分）
    参加された方からのご質問を受けて、登壇者が回答します。
