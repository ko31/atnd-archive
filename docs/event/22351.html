Windows Phone Advent Calendar って？

クリスマスまでの日数を数えるために、1日に1つ、穴が空けられるようになっているカレンダーがあり、そこから技術系コミュニティで、クリスマスまでの25日間にTipsを1つずつ公開するイベントをAdvent Calendarというそうです。今年もいろいろなAdvent Calendarが企画されています。

というわけで、本イベントはWindows Phone版です。12月1日から25日までの間、参加者が日替わりでblogを更新していきます。お題はWindows Phoneに絡めばなんでもありです。

ほかのAdvent Calendarはこちらから：http://d.hatena.ne.jp/kanonji/20111118/1321584724

ルール

  • Windows Phoneに関するTipsを順番に1つずつブログに書いていきます。
  • 書く順番はこのイベントに参加した順番です(最初に参加した人が12/1、最後に参加した人が12/25)
  • 記事を書いたらこのイベントのコメントにURLをはってください。
