概要説明

3 月よりスタートした extends Progression 拡張機能コンテストの結果が、いよいよ決定しました。

数多くのエントリーから選ばれたのはどの作品なのか！？

当日は授賞式と併せて、受賞者本人から受賞作品についての紹介から開発秘話までプレゼンテーションしていただきます。

「もっと便利に使いたい！」という Progression ユーザーの方から、まだ使ってないけど Progression の可能性を感じたいという方まで、ぜひご来場ください。
