11月4日（火）～12月16日（火）まで名古屋の錦にありますコミニカルチャースクールにて認知行動療法ワークショップを開催します
　
認知行動療法とはうしろ向きな考えを見直し、幅広い考え方を身につけ、つらい気持ちを軽減させる心理療法です
1人ではなかなかうしろ向きな考え方を変えることは難しいですが認知行動療法をを行っていくと次第に柔軟な考え方ができるようになります　

認知行動療法ってどんなのか知りたい、体験してみたいという方、是非この機会にお申し込みください

料金は今回は、コミニカルチャースクール開講記念短期講座とういうことで全4回の6000円！

今回は短期講座とういうことで４回の短い時間になるため認知行動療法の基本的なベースとなる考え方、そして実際に体験をして頂く講座になっております
　
今まで認知行動療法って聞いたことあるけど実際どんなのだろうか？ちょっと見てみたいなという方には大変お得な講座となっております

心が風邪を引いてしまわないために、予防として認知行動療法を活用してみませんか？
　
この機会に一度お試しください

皆様のご参加心よりお待ちしております
