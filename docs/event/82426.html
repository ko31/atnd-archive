「自社のサービスを英語でプレゼンできますか？」目指すは海外でのピッチイベント登壇！

✓海外のカンファレンスでネットワークを作りたいけど、英会話が全くできない…

✓海外のピッチイベントで登壇したいけど、英語でプレゼンできない…

✓海外出張が決まったけど、英語で自社のサービスが説明できない…

最近、IT・スタートアップ業界も英語を使う機会が増えてきていると感じます。テック系のカンファレンスが日本を含む世界各国で盛り上がる中、英語での情報収集やネットワーキングは必須のスキルとなるかと思います。しかしながら、「英語さえできれば…」と悩まれている方は多いのではないでしょうか？

イベント概要

Tech CrunchやBridge、Tech in Asiaなど、国内外のテックメディアから話題を抽出し、頑張って英語で意見交換しようという趣旨のイベントです。まずは日常的に海外のテックメディアから英語で情報収集することを目指し、ゆくゆくは英語で外国人に対して自社のサービスをプレゼンできるようになれればと思います。
