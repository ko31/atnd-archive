8/1(土)よこすか開国祭にあわせ「ヨコスカフレンドシップデー」として横須賀米軍基地が一般開放されます。
一緒に参加して楽しみましょう！
毎年この一日が待ち遠しくて病まない今日この頃です。

本場のアメリカを肌で感じたい・・・
ドルを使って買い物したい・・・
米軍基地へ行ってみたい・・・
マクドナルド半端ない大きさ・・・
スペアリブまじ美味い・・・
アメ車ってやっぱかっこいい～・・・
クールなロックの野外ライブ・・・
全てがアメリカンサイズ・・・
ここはアメリカなんです！まだまだ書き切れないほどの光景が目白押しなんです。
ネイビーバーガー片手に花火を見て、どぶ板で飲んで帰る。こんな一日を送りましょう。
 
【スケジュール】　雨天決行 荒天中止
■日時：8/1(土)　12時～22時
■集合場所：京浜急行横須賀中央駅
■費用：無料
■申込：http://spr-out.com/friendship-day2015/

①13時：米軍基地内での食事やアトラクションなど何でもあり（ある意味アメリカ遠足）
②19時：基地内から間近で開国花火大会を鑑賞（まじ花火爆音で近い )
③20時：どぶ板通りのBarでおつかれ飲み会（米軍基地の方々が御用達）
国際交流パーティーなんてもう飽き飽き・・・って方も
たまにはパーティーを忘れて日本にあるアメリカで思いっきり遊ぼう！

☆必ず読んでください
※日米交流が目的なので日本人とアメリカ人に限りの入場です。（去年入場できない人がいました。）
※荷物検査があり、身分証明書の提示を求められる場合もあります。
※ビジネスの勧誘、セクハラ、ナンパ目的、迷惑行為ご遠慮願います。
※途中の合流とか出来るようにしますが敷地が広大なので難しい場合もあります。
