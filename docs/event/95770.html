description

スマートフォンアプリのプロモーション/マーケティング活動においても、予算をかける前には適切なKPIを設定し、その効果が本当に見込まれるのかを見積もることが重要です。 また、そのインパクトを損なわせないためには、アプリそのものの品質を一定以上に保たせなければ、効果的なプロモーションは見込めません。

本セミナーでは、プロモーション活動を行う前に必要な市場動向/競合の調査・分析手法と、スマートフォンアプリにおいて忘れられがちな品質管理ノウハウをご紹介いたします！

セミナー終了後には個別相談会の時間を設けておりますので、課題をお持ちのご担当者様がおりましたら講師陣より個別に情報提供をさせて頂きます。

※　こんな方にオススメです！　※
* スマートフォンアプリのプロデューサー/ディレクターの方

* スマートフォンアプリの担当者の方

* スマートフォンアプリの企画/マーケター/グロースハックに取り組む方

* スマートフォンアプリの開発エンジニアの方

* スマートフォンアプリの品質管理に携わっている/課題感がおありの方

* スマートフォンアプリの最新データを知りたい方

セミナー内容

■<講演1>2017年からのゲームアプリ市場動向と、企画,開発を含むマーケティングにおける市場/競合調査事例について

登壇者：按田 洋平 -フラー株式会社　エヴァンジェリスト
マーケティング(含むプロモーション)の基本において、自社アプリの数値に対して、競合および市場動向の影響を考慮することは、適切なKPI設定や施策の結果を分析する為には必要不可欠です。また当たり前のことですが、サービス全体や各フェーズにおけるキャンペーンごとに明確なターゲット像を持ち、その特徴や規模を理解することは、大きな予算や労力を必要とするアプリマーケティングの効果を最大限に高める事に繋がります。 今回はゲームアプリ市場における2017年からの市場動向と合わせて、大型のキャンペーン時の調査事例を共有させていただき、成功させるための調査分析についてお話いたします。

■<講演2>ユーザー獲得効率を上げる為にも意識すべきアプリ品質について

登壇者：吉井 文学 - FROSK株式会社 CMO
マーケティング/プロモーション活動を行う前にやるべきアプリの品質改善手法をご紹介いたします。クラッシュやエラーなどが起きる品質が悪いアプリでは、マーケティング活動を効果的に行うことはできません。導入アプリMAUが1.7億人にのぼるSmartBeatのデータから見る「高品質」なアプリの運用/品質改善の特徴をご紹介するとともに、SmartBeatを活用した高品質アプリを作成するためのポイントをお話致します。
