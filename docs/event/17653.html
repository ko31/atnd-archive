ちょっと涼みに来ませんか？


今年も夏がやってきましたね。
今年も色んな人たちとの出会いを楽しみにしています。

“はこぶ”も2周年です。
この一年、色々な方々がつながり、そこで色んなモノが生まれていきました。
またここで新しいつながりが出来たらいいと考えています。


というわけでただの飲み会を開催いたします！

みなさん、自分の知り合い、友達、恋人、家族、通りすがりの人などを連れて参加してください。
誰かとダラダラと酒を飲みたい人、なにか作ってるものを見てもらいたい人、よく分からないけど面白い人と話したい人、京都好きな人、笑いたい人、交流したい人、お待ちしております。


場所は京都西陣の町家スタジオです。
電源、無線LANがつかえます。
※宿がない方は朝まで使用可能です！(寝る場合は雑魚寝になります)


  • 参加費：2000円程度
    → 飲食費として当日集めます
    → ご飯、菓子類、お酒等の持ち込み，差し入れ大歓迎です
  • 内容：いろいろ
    基本的に飲み食いしながらワイワイします
    → LT (ライトニングトーク)出来ます。プロジェクタ等各設備有り！
    → 時間があれば、なにかしらの作品を観ます！
    → その他 なにかしたいこと募集中です。
