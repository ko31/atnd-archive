MWW

twitterからはじまったこのイベント。
九州の面白いベンチャーから九州に全くゆかりのないベンチャーまで！
自らのサービスの発信とコラボレーションの誕生を目的に開催します！

また、ベンチャー企業とVCとのマッチングや、
イベントを通したコミュニケーションなど、
宮崎ベンチャー文化の活性化を図ります。

プレゼンター

・インタラクティブ領域における時代の潮流とデザインとエンジニアリングの融合 / 阿部(ワンパク)

これからの時代、Webを生業にしてきたプロダクションやシステムハウスはどのように生き残っていけばよいかというヒントをクリエイティブという側面から考えてみたいと思います。

・株式会社ヌーラボなう / 橋本(ヌーラボ)

株式会社ヌーラボの紹介と、今年の７月にシリコンバレーへ行ってプレゼンしてきた際のレポートなどをご紹介します。宮崎や、福岡にいても世界展開などを狙えるのでは？という気持ちを持っていただければ幸いです。狙うだけなら自由ですからね！

・動画サービスGemediarができるまで / 市江(PocketBox LLP)

動画配信インフラGemediarの技術的な成り立ち(インフラ) か、どうやってこういうサービスをすることになったか(モデル) をお話させていただきます。

・オープンソースとコミュニティと私 / 江頭(キャッチアップ)

BaserCMSというソフトウェアが、オープンソースとして提供されるまでの経緯、そして現状など。

・天神のど真ん中にあるAIP Cafeとは？ / 村上(高度IT人材アカデミー)

「AIP Cafeって何？」を分り易く説明。また、どんな人がどんな使い方をしているのか、実例を挙げて福岡の今を感じて頂ければ、と思います。

・セキュリティ技術を使った社内恋愛ライフハック / 石森(ゲヒルン)

日頃叫ばれるセキュリティ問題だが、その実態や実際の攻撃手法、被害はなかなか公開されず、実感することはない。今回は、WindowsXPの管理者権限をリモートから奪取し、そこから社内の気になる女の子と社内恋愛に発展するライフハックを例にしてセキュリティ問題を分かりやすく紹介する。

・デザイナー的勉強術 / 山田(ブランコ)

Webデザイナーな私が、コミュニティーや勉強会を通して様々な人に出会い、刺激をうけてきた経験とそこから学んだ、デザイナーが自分を磨く方法などをご紹介します。
