内容

ハンズオン組とハッカソン組を一応分けて朝9時から夜9時まで行います。随時参加、離脱OK。緩い感じで。

ハンズオン

Scala Hack-a-thon でのyuroyoroさんの 資料 のようなハンズオン用資料を、みんなで読み、書き進めてもらうような内容を考えています。

ある程度Scalaの文法などを把握したら、随時Hackathonに移行してきゃっきゃうふふしましょう！

Hackathon

ある程度Scalaが出来る人は、自由にコード書いたり、周りの人と情報交換しましょう。最後にHackathonでの成果発表を予定しています。

上記はあくまでも予定です。参加人数などによって変更の可能性があります。

日時

8月20日(金) 9:00〜21:00(途中食事休憩を含む)

参加費

無料を予定していますが、学生の参加者がいらっしゃる場合は、社会人のみ参加費を集めます。

集めた参加費は学生の皆さんの交通費、ピザ代等にまわします。(一人あたり千円から2千円程度でしょうか)

用意する物

  • Scala2.8入りのノートパソコン
  • 電源タップがあるといいかもしれません
