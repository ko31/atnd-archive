■開催説明資料を公開しました。

■開催内容の説明をストリーミング配信しました。

第２回ダンエボパフォーマンス大会 on WEBを開催します！

開催に先立ち、審査員を募集します。

審査員10名集まれば、開催決定となります。

#ダンパWEB で検索。

■応募資格

  • 審査期間中に、下記が実施可能である事。
  1. 投稿された最大50本（予定）のダンエボプレイ動画すべて目を通す事が出来る。
  2. 目を通した動画に対し、１位～5位までの順位付けを行う事が出来る。
  3. 結果を、 @danpa_web へメールまたは DMにて連絡する事が出来る。
  4. もし、複数の方が１位となった場合、決戦投票を行います。決戦投票も可能である事。
  • 大会参加／不参加を問いません。
  • 大会参加者は、自分の動画に投票してもOKです。

■Q&A

Q : 審査員は、具体的に何をやるのですか？

A : 審査員のやる事は次の通りです。

(1) 2016/12/25 ～ 2017/01/10迄の審査期間中に投稿頂いた動画をすべて（最大50本の予定）ご覧頂く

(2)ご覧頂いた動画の中から、１位～５位の順位を付ける

(3)１位～５位の順位を、 @danpa_web にＤＭでご報告

Q : 審査ってどういうとこを見ればいいのでしょうか？
上手いとかで選ぶ？楽しそうで選ぶ？両方？

A : 審査条件は審査員の皆さんが自由に決めて頂けます。
例）　①「綺麗」、②「楽しそう」、③「ダイナミック」　
などなど。
