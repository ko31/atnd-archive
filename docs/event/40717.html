アジェンダ

セッション１：　「Packaged Web Appについて - AndroidとChromeの統合」

  • 講演者
    • 日本Androidの会　丸山不二夫
  • 概要
    • 現在、アプリ開発のスタイルに大きな変化が起きようとしています。
      HTML, CSS, JavaScriptといったWebの標準技術をクライアント側のアプリに応用しようと言う流れが大きくなっています。そうしたクライアント・サイドのWebアプリは、現在、Packaged Web Appというスタイルでまとめられています。
      講演では、Packaged Web Appの代表的な取り組みとして、Google Chromeのアプローチを紹介し、Googleのこうした動きが、AndroidへのPackaged Web Appの導入を大きな目標としているという予想を述べたいと思います。

セッション２：　「Webベースのアプリ開発最前線」

  • 講演者
    • 日本Androidの会　中国支部長　重村浩二
  • 概要
    • Webの技術であるHTMLとJavaScriptを用いたアプリ開発では、様々なクロスプラットフォーム開発ツールが提供されています。
      現在の開発ツールを横断形式で紹介したいと思います。

セッション３：　「androidエンタメアプリクッキング」

  • 講演者
    • 日本Androidの会　中国支部　佐々木順昭
  • 概要
    • 割とお気軽に自作アプリの制作に取り組めると言われるandroid。
      そして、今回の会場は広島でもゲームからビジネスまで学べる広島コンピュータ専門学校。
      学生が学んだことをandroidアプリの制作に使用しよう。
      今回の勉強会の主役であるHTML5+JavaScriptと対をなす、ネイティブアプリの制作例を紹介。
      ・ネイティブアプリとは？
      ・サーフェイスビューとフレームアニメーション
      ・シンプルゲームアプリの制作例

セッション４：　「AndroidアプリにおけるHTMLのつかいどころ」

  • 講演者
    • シュガースポット　佐藤慎也（http://sugarspot.net/）
  • 概要
    • Androidアプリを作つとき全てをJavaで作らなきゃいけないってこと無いですよね
      どんな場面でHTMLを使うと良かったか、これまでの仕事の中での話しを含めた使いどころについて少しだけ話をさせて頂きます

ショートセッション１： 「PhoneGapはどこまで使える？徹底解剖！！」

  • 講演者
    • シークレット
  • 概要
    • html5やCSS3、JSフレームワークがどこまで使えるか徹底検証してみました。

ショートセッション２：「mruby(軽量コンパクトなRuby)とAndroid」

  • 講演者
    • 日本Androidの会 中国支部 熊野靖
  • 概要
    • ・mrubyとは何か？
      ・mrubyをAndroidで動かすと何ができるのか？
      ・簡単な実演デモ
