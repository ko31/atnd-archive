MMD愛知カラオケBOXオフ会開催

平成最後に集まってみる？

概要
カラオケボックスを予約しました。
ただそれだけです。

どんな事をするの？

  • 簡単に言えば
    1. とりあえず平成最後に集まってみようず。
    2. 一応電源も使えるからパソコン持ってきてもいいよ？
    3. あんまりたくさん繋げないかもしれないけど。
    4. Wi-Fiも使えるらしいって聞いてるよ。
  • つまり適当です。
    MMD触った事ない人・見る専だって構いません。
    別ジャンルのPさんも大歓迎！
    まぁ突発で勢いだけでやるの決めちゃったオフ会なので。

二次会は未定ですが、カラオケボックスなので料理も頼めるかも？
場所が名駅だからそのままどっかのお店で飲み食いしてもいいし。

開催日時及び会場について
会場のカラオケボックスは「MMD愛知」の名前で予約を取っています。

参加費用
1500円ぐらい(カラオケボックス代)
集まって入る前なら集金するけど、途中から入る場合は各自受付で払ってね。

GW料金がまだ確定してないらしいけど、とりあえず1500円でお釣りが来るらしい。

日時
2019年4月27日(土曜日)午後4:00～午後8:00
(GWの繁忙期につき、お店が混雑した場合午後7時で終了になる事があります。)

途中入室も可能です。

場所
カラオケバンバン 名駅太閤通口店

アクセス

  • JR（新幹線）・名鉄・近鉄
    • 名古屋駅新幹線口(西口・太閤通口)から徒歩約5分。
      ビックカメラ前を右へ進み、１Fにローソンのあるビルの前の交差点を左折。
      最初の信号角にある１Fがパチンコ店「WING LET」のビルです。
      向かい側にダイワロイネットホテルがあります。
  • お車の場合
    • ビルの立体駐車場、またはその先の立体駐車場をご利用ください。

施設について

  • カラオケボックスという性質上、コンセントは使えますがどれだけの台数接続できるかは不明です。また、Wi-Fiもありますが回線速度は期待出来ませんのでご了承ください。

会場での注意事項

  • 飲食
    • 飲食物の持ち込みは可能です。周辺にコンビニもございます。
      但し、ゴミは各自での持ち帰りにご協力ください。
お願い
作業オフ会という性質上、パソコン等を利用する方がほとんどだと思われますが、電源コンセントの数に限りがございます。電源タップ等を持ち寄って頂くようお願いいたします。
みなさんのご協力で楽しい会にしましょう。

一応定員15人としていますが、変更になる場合もありますので、予めご了承ください。

何かご質問があればペパクラ(@hbkuma)まで
