description

世界的に有名なデザインコンサルタント会社「IDEO」の元ディレクターElle LunaとMichelle BranchやRod Stewart等、数多くの有名音楽アーティストが活用している「One Sheet」の創業者Brenden Mulliganをお招きし、Open Network Labでトークセッションを開催します。今回のテーマはスタートアップ戦略「Startup Strategy」。Elleとはデザインやブランディング戦略についてのディスカッションを行い、Brendenとは計るべき数値、収集するべき情報など、サービスローンチ後の戦略についてディスカッションを行います。（＊ディスカッションの内容が変更になる可能性もございます、ご了承ください。）

第２部として、現在募集中Open Network Lab第５期の説明会を行います。

スピーカープロフィール

Elle Luna • IDEOディレクター


サンフランシスコ在住で主にスタートアップをアドバイスするデザイナー。Fast Company社の「Silicon Valley’s Leading Women Designers」に選出される。前職はデザインコンサルタント会社「IDEO」でディレクターを勤め数々のFortune 500企業を担当していた。
http://about.me/elleluna

Brenden Mulligan • 「OneSheet」創業者 / 500 Startupsメンター


コンサート配信プラットフォーム「ArtistData」を創業2006年に創業、2010年に[Sonicbids]に売却。現在は主にUX/UIを中心にシリコンバレーのインキュベーションプログラム「500 Startups」の会社をアドバイスしている。
http://about.me/mulligan
