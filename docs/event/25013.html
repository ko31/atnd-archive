NFCハッカソン に触発されて、福岡でもNFC勉強会を開催してみることにしました。東京な方は 第1回 NFC勉強会 なんていかがでしょう。

NFCとは

Near Field Communicationの略称で、近距離の無線通信規格のことです。
身近なところではnimoca、SUGOCA、はやかけんが有名で、
Galaxy NexusやGalaxy S2 LTEにも搭載されている注目の機能です。
最近では、任天堂 Wii UのコントローラにNFC機能が搭載されることでも話題となりました。

ちなみに、仕様策定をしているNFC ForumによるとNFCには3つのモードがあります。

  • NFC Card Emulation Mode
    一般的なNFCカードに擬態（？）するモード。
    俗にいうオサイフケータイのメイン部分。モバイルSuica便利です。
  • Pear-to-Pear Mode
    端末同士でデータをやり取りするモード。
    Android Beam はこのモードを使っています。
  • Reader/Writer Mode
    端末自身がリーダ・ライタになるモード。スマートポスターの仕組みにはこのモードを使っています。
    個人的には、 FeliCaソリューションセミナー の受付でAndroid端末に参加証（FeliCa Lite）を
    かざすことで参加者チェックされていた衝撃が忘れられません。（ 先生ケータイ を使っていたとか何とか）

これらモードを使うと、何か素敵なことができる気がしませんか？

勉強会の目的

本勉強会では有志によるLTや実際に手を動かす時間を設けることでNFCを理解し、体験することを目的とします。
手を動かすという性質上、 参加される上でご自身のラップトップ等あった方が良いと思います。

主催者の周囲だけかもしれませんが、NFCといっても割とみんな 興味を持ってくれない ので
気軽に参加していただけると嬉しいです。
