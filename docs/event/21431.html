description

【受講対象】
　めっき製品のユーザー、発注者の企画・設計等担当者

【講座のポイント】
　めっきなどの表面技術関連業種は、発注者から素材を預かりそれに表面処理を施してお返しする自社製品を持たない業種である。
　めっきに関わる不良には、めっき不良、素材不良、仕様不良などがあるが、不良の大半は素材にあり、最大の不良要因は発注者の表面処理に対する認識不足と意思疎通の欠如である。
　あらためてめっきの目的、仕様の明確化とめっき素材への再認識を問いながら、事例を通してその要因を探る。

【プログラム】
１．電気めっきの基礎知識
　　1.1めっきの基礎
　　　1.1.1 ファラデーの法則
　　　1.1.2 めっきの析出量と厚さ
　　　1.1.3 電流密度と適正電流密度
　　　1.1.4 電流分布
　　1.2 めっきの防食機構
　　1.3 めっきの機能
２． めっきの不良要因
　　2.1 めっきの不良とは
　　　2.1.1 電気めっきＪＩＳに記された不良
　　　2.1.2 めっき不良と仕様不良
　　　2.1.3 常態と異なるから不良
　　2.2 めっき不良の発生要因
　　　2.2.1 めっき不良主要要因は素材
　　　2.2.2 加工負荷要因
　　　2.2.3 めっきに適した素材形状　-デザインも不良を誘発する－
　　2.3　仕様不良の発生要因
　　　2.3.1 発注書または加工仕様書への記載事項は守られているか
　　　2.3.2 めっきの目的と仕様
　　　2.3.3 めっきの目的は明確か
　　2.4　めっき不良・仕様不良を減少させるために
　　　2.4.1 不良の分類と相互認識
　　　2.4.2 両者相互の意思疎通と現場認識
　　　2.4.3 良品はとっておこう
　　2.5 めっき工場の選択と発注について
