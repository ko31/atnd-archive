title iDevKobe 勉強会 May.2014

description

神戸のiPhoneアプリ開発の勉強会です。

当日、会場代実費(500円)を集金します。
参加にあたって、必ずMacにXcodeをインストールしてきてください。
MacAppStore : Xcodeのサイトへ

会場情報

会場最寄りのコンビニは地下鉄県庁前駅近くのセブン-イレブンです。会場にも自販機ありますが、必要な物は事前に買ってきてください。
会場にはFreeWi-Fiがありますが、使い方がイマイチわかりません。通信環境が必要な方は通信機器(iPhoneのパーソナルホットスポットなど)を準備ください。
プロジェクターは会場の備品を借ります。変換アダプタとして、VGA ←→ディスプレイポート、VGA←→ライトニングコネクタ、は準備します。

場所

生田文化会館　第5会議室
住所:兵庫県神戸市中央区中山手通６−１−４０

前半：説明、自己紹介

・13:00〜 会場設営(プロジェクターやら机並べたりしてます。会場自体は開けているので会場に来てください。)
準備できた後、集まり次第説明etcします。
その際、軽く自己紹介お願います。

発表
・テーブルビューの使い方についてbyようにゃん
　(ライブコーディングー簡易Twitterビューアアプリ)
・その他発表(未定、ネタある人は別途連絡または当日声かけてください。)

後半：フリートーク・フリー質問・フリー開発

発表が終わったら、参加者同士で情報交換する時間にします。アプリ開発で、困っていること・悩んでいることなどを共有して解決して行きましょう。

懇親会

みんなで飲みながら、iOS談義しましょう。会場は元町駅周辺ですが、当日決めます。予算は3000円程度です。
