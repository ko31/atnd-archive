パソコンよりカンタン、ケータイより大きい画面で

話題のタブレットiPadを使ったワークショップを、福岡商工会議所「あ・るっく福岡」の1日体験プログラムとして開催します。
http://www.fukunet.or.jp/alook/

友だちやペットを撮影した動画や、旅行やパーティーのビデオを自由に編集して、自分だけのオリジナル作品を作ってみましょう。
できあがった作品は、会場のみんなで見てワイワイ楽しんだり、遠くの知り合いに見てもらうこともできます。
iPadに興味はあるけどパソコンやケータイしか触ったことがない方でも、無料貸出があるのでお気軽にご参加いただけます（各回限定8台）。また、すでにiPadをお持ちの方は、持ち込みでのご参加もOKなので、ビギナーでも大丈夫（参加費用割引あり）。
さらに、休憩時間には、テレビ番組制作現場を見学できる、ビデオ制作会社ならではのお楽しみもあります。皆さん、お誘い合わせの上、ぜひご参加ください！

内容

ビデオ編集アプリ「 Avid Studio 」を使って、自分で撮影したり用意された動画を使って、ビデオ作品を仕上げてみましょう。作った映像は、YouTubeで見たりDVDにすることも可能です（DVDは有料で後日発送となります）。
iPad持ち込みの方は「 Avid Studio 」をインストールしてご参加ください。
