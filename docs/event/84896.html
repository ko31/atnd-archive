==== はじめに ====
当スクール『ちよまる式English Quest』は、
「英語は難しい」
という常識を変えるために生まれました。
　
＜4つのお約束＞

（１）当スクールでは、『半年で英語が話せるようになる』ことをお約束しています。
1日30～60分の学習時間で英語を習得できますので、普段お忙しい方でもご受講いただけます。
※対面のレッスンが1カ月に1回、毎日の学習サポートはLINEアプリで行います。

（２）『体験会中の営業』や『体験会後の営業』は一切行っておりません。
皆さまのご意思で英語学習を始めていただきたいからです。

（３）ビジネス力があがる
当スクールでは、英語が話せるようになることはもちろん、ビジネス力が向上します。
英語が話せるようになる過程で、『要点を掴む』『結論から述べる』『情報の整理』『論理的思考』『グローバル感覚』などビジネスで役立つ技術も合せて身に着きます。

（４）卒業後も継続的に英語学習をサポート
英語学習は継続が大切です。
卒業したら終わりというわけではなく、卒業後もずっと無料で英語学習を続けられるようなサポートを用意しています。
※通常無料ですが、より高いレベルを目指したい方には有料のサービスもあります。

1､現役で世界のエリート層と英語でビジネスをしている。

3､教員免許を持ち、物事を簡単に説明するのが得意。

※詳細なプロフィールは下部をご覧ください。　

====== そもそも どうして英語は難しい と感じるのでしょうか？ ======

一言でいうと、
『英語が話せるようになるまでの全体像と学習手順を知らない』からです。
　
この状態で、
「よし！英語やろう」と意気込んで、
色々なサイトを見たり、外国人と話してみたり、単語を覚えたり、文法を学んだりしても、
　
「何から学習していいか分からない」
「単語を覚えられない」
「文法を考えながら話せない」
「発音ができないと恥ずかしい」
と沢山の壁が現れて、諦めてしまう方が多いのが現状です。
　
では具体的な学習手順はどのようなものなのでしょうか？
　
====== 英語習得までの 具体的な学習手順 ======
概要は下記の通りです。
各詳細は体験会でお伝えします。
　
（１）発音
→正しい発音ができなければ相手に通じません。
　先ずは、単語を正しく発音できるようにします。
　
　正しい発音が身につくと相手が同じ単語を言ったときに聞き取れるようになります。
　つまり、リスニングができるようになります。
　正しい発音習得までの目安期間は約1ヵ月です。

（２）リズム
→どの言語にも聞きたいリズムがあります。
　リズムに乗せればよりはっきりと相手に伝わります。

（３）英語の構造の理解
→世界基準の英語の構造はとてもシンプルです。
　数パターンを押さえれば話すことができます。

（４）使用頻度が高い単語の習得
→基本動詞・前置詞・副詞は英会話の中でも特に使用頻度が高いです。
　すべての会話に必要ですので、これを効率よく習得します。

（５）日本語の変換技術
→日本語は、英語に限らず他の言語に変換しづらい言語です。
　これを他の言語に変換するためにはテクニックが必要になります。

以上のような順序で学習することにより、
英語が話せるようになります。
　
ここまでが、
英語習得までの具体的な学習手順の概要です。
　
======= 英語習得を後押しする 最後の『スパイス』 =======
　
具体的なステップに沿って
学習をすれば英語を身につけることはできます。
それはさながら、マラソンランナーが道のりとゴールを知っているため
最後まで走り切れるのと同様です。
　
とはいえ、その道のりが辛ければ走るのをやめてしまうこともあると思います。
では、どうすれば皆がゴールまで走りきることができるのでしょうか？
　
それは、楽しみながら走ることです。
　
皆さんは学生時代に、
「好きなことなら覚えられたのに、
テストの暗記となると覚えられない」
という経験はありませんか？
　
英語学習も同じで、
自分の興味がないことを覚えようとするのは…
　
辛いです！
　
ちよまる式English Questでは、
学習方法として「自分の好きなことを話す」ことを重要視しています。
(仕事、スポーツ、ゲーム、マンガ、映画、食事、旅行…自分が好きなことなら何でもOK！)
　
理由は、「自分が興味がないことは覚えられない」からです。

自分の好きなことを話すことで
英語学習が楽しいだけでなく、記憶の定着にも繋がります。
　
まとめると
　
・英語習得までの具体的なステップで学習する
・自分の好きなことを話す
　
上記2つで、
英語は話せるようになります。
　
体験会では
ここまでお伝えした内容をより詳しくお話しさせていただきます。

1982年生まれ。
立教大学卒業後、株式会社永谷園に入社し東京支店で営業に従事する。
退社後、英語を独学で勉強し、英語学習の本を1,000冊以上読破し1年間でTOEICスコアを500点から900点に上げる。
だが、海外に行くも「全く話せない」という状況に陥り、TOIEIC900点のプライドが崩れる。
英語を話すことと覚えることは別と分かり、現地でどんな会話をしているのか？を分析していくうちに「ある共通点」を発見。その共通点を元に、英語を話せるようになる独自メソッドを確立。
その後、海外ファンドの情報提供アドバイザーとして多国籍のビジネスパーソンとの取引を経験する。
2014年、企業のコスト削減をメインとする株式会社フィスを設立。同時に、日本の中小企業と外国大使館を繋ぐことを使命とした一般社団法人おもてなしズムJapan理事に就任。各国大使と日本企業の交渉を目の前で見る中で、あらためて日本人の英語の必要性を痛感。現在、[実践の場で使える英語]を普及すべく活動しながら、世界各国のトップエリートと英語で仕事をしている。英語完全初心者でも半年で世界の人と通じるくらい英語が話せるようになることをコミットしている。
