title 福島・会津応援！
catch 会津蕎麦と地酒を楽しむ会
description

この度の大震災により被災された皆様に対しまして心より御見舞い申し上げます。
被災地の一日も早い復興を社員一同、心よりお祈り申し上げます。
復興に少しでもお役に立てればと「会津のそばを食べる会（そば会）」を6月22日に再開する事に致しました。

御料理：各市場直送の新鮮な御刺身と旬の食材を使った御料理五品。
お蕎麦：福島県会津産の石臼挽き蕎麦粉を手打で御蕎麦に致します。
食べ放題ですので、心ゆくまで御堪能下さい。
お 酒：南会津町（旧南郷村）の地酒、「花泉」を御用意しております。
花泉は本数限定ですがビール焼酎等は飲み放題です。
