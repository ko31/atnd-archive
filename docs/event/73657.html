☆☆☆多店舗展開の３つの秘訣セミナー☆☆☆

複数店展開・多店舗展開をお考えの皆様

今回は、複数店展開や多店舗展開の経営者様から、非常に相談が多い内容のため、このセミナーを企画致しました。既に全国で複数回開催しご好評を頂いている講座です。

皆様、次の悩みをお持ちではありませんか？

〇多店舗展開を考えている
〇フランチャイズ本部展開を考えている
〇コストをかけて多店舗展開するのが不安
〇現在の店舗で売上に限界を感じている

ここに書いてある悩みは私が日頃から、良くご相談される内容です。

複数店展開や多店舗展開を考える経営者様は、こんなことを不安に感じておられます。

〇多店舗展開のタイミングはいつがいいのか？
〇多店舗展開できる基準はどの程度は？
〇多店舗展開を行う場合の資金調達はどうすればいいのか？
〇既存店への影響が大きく出るのではないか？
〇多店舗展開した店が不採算になるのではないか？
〇人の教育はどうしたらいいのか？
〇人手不足に悩むのではないか？
〇従業員不正が発生するのではないか？
〇結果として、失敗し、撤退しなくてはならなくなるのではないか？

多店舗展開をする時は不安と悩みがつきものです。
多店舗展開を実施したからと言って、売上が必ず上がるわけではありません。
むしろ次のようなことも発生しているのが事実です。

〇商圏の喰い合いにより、売上が約1.2倍しかなっていない
〇経費がかさみ逆に赤字になっている
〇3号店、5号店、10号店の壁が破れない
〇業務の標準化が図られずに、クレームが発生している
〇店長の力量が足りない。従業員の管理が出来ない
〇人手不足に拍車がかかっている

こういった現状が現在でも発生しております。
そうすると経営者様は『多店舗展開なんかしなければよかった』と思われる方も多いというのが実情となっております。この問題を解決するには実は３つの秘訣があるのです。

この成功の３つの秘訣を今回のセミナーでは公開させて頂くと共に、これから多店舗展開を検討されている方に本当に多店舗展開をすべきかの判断基準となるメリット・デメリットも併せてお話をしたいと思っております。

☆☆☆販路企画について☆☆☆

【経営コンサルタント会社に入社】
・大学卒業後、経営コンサルタント会社に入社　
・マーケティング戦略立案支援、販路拡大支援、ISO認証取得支援
人事評価制度導入支援コンサルティングを実施。
・マーケティング最先端でフランチャイズ本部の発祥の業界最大手コンビニエンスストアチェーン本部へ転職。　

【業界最大手コンビニエンスストアチェーン本部入社】
・直営店で実際に店舗経営を実践。
・商圏調査・エリアマーケティングでたったの3か月で前年比：124%を実現。　
・店舗ＳＶとして商圏調査・エリアマーケティングによる売上改善コンサルティングを実施。
・成功するフランチャイズ本部の仕組みとポイントを徹底的に習得する。
・エリアの売上改善実績が社内全国表彰。マネージャーへ就任。
・顧客の心理に基づいた『商圏調査・エリアマーケティング』『フランチャイズ本部展開支援』『従業員戦力化研修』を中小店舗に拡大をしたいと独立を決意。

【販路企画を設立】
・17年間のコンサルティング経験をもとに、『多店舗展開支援』『商圏調査・エリアマーケティング』『フランチャイズ本部展開支援』『従業員戦力化研修』『起業家支援』を軸に中小店舗が実践できるようにノウハウ化。全国へ情報を発信している。
・クライアントは飲食業、小売業、サービス業と多岐にわたる
・支援実績200社以上。支援期間も3年間を超える店舗・企業多数
