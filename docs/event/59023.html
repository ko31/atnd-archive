description

ストーリーを使って新しい戦略を創る！

伝えたいことを"物語"にして語る。

昨今では、ストーリーをビジネスに活用することが注目されています。

ストーリーがあると、相手に具体的なイメージを思い浮かべてもらうことができ、
伝わりやすいだけでなく、記憶にも残るからです。

物事の仕組みや構造、伝えたい内容を決めるのに、ロジックは必要です。

ですが、伝えたいことをただ論理的に話をしたとしても、相手がそれを受け入れて
動いてくれるとは限りません。

相手の琴線に触れ、共感を創り出すことが大切です。
そのために、ストーリーはとても重要な役割を担います。

このセミナーでは、ストーリーを構成する要素、戦略について
理解していただくとともに、あなたが創りたいストーリーを
１本創りながら、ストーリーの基本を学んでいただきます。

そして、あなたの企業や商品、さらにはあなた自身が
価格競争に巻き込まれることなく、なくてはならない大切な存在として、
相手から選んでもらえることを目指します。

これにピンと来たら
　・商品の価値を伝えたい
　・我が社の理念を浸透させたい
　・営業でストーリーを使いたい
　・伝わる話し方を知りたい
　・夢を実現する仲間と資金を集めたい
　・感動でV字回復をしたい
　・ストーリー戦略を取り入れたい
　・ストーリーを創れるようになりたい

参加するとこれが手に入ります
　・ストーリーの６つの要素が分かる
　・ストーリーの基本形が身に付く
　・伝わる話し方が分かる
　・ストーリー戦略を理解できる
　・価値の表し方が理解できる
　・売れるために必要な価値が分かる
　・お願い営業さようならの秘訣が分かる
　・効果の即効性を体感できる
