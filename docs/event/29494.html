　成長したい！新しいことにチャレンジしたい！
　そんな学生だった南が、同じ想いをもった学生のために合宿型の起業講座を開設することにしました。

どんな合宿？

開始と終了の数時間以外は、どこで何をするかはほとんど決まっていません。
どんな仕事をどうデザインするのか。
限りある時間をどう過ごし、そこからいったい何を学ぶのか、それは参加者のあなた次第です。
あなたのチャレンジを、京都の若手企業家、大学講師など様々な強者社会人が見守り、応援します！

あなたは週末何をしていますか？
アルバイト？飲み会？

どうせお金を稼ぐなら、初めて会う仲間と新しいやり方で自分を試してみませんか？
どうせお酒を飲むなら、一緒に頑張った仲間や、社会人と終わった後に語らいませんか？

参加すれば、いくら稼げる、何を学べる、と断言はできません。
それは参加する人の取り組み方次第でまったく違うものになるからです。
自力でお金を稼ぐ経験、そこから気づいた学び、新しい仲間、社会人との繋がりや、
仕事のノウハウ、新しく発見する自分自身・・・一つでもたくさんのことを手に入れて
帰ってください。

めいちゃんちのマナビ×アソビ

授業を受け、放課後部活に行く。めいちゃんちはそんな学校のような場でありたいと考えています。
単に座学で学ぶだけでなく、時には身体を動かし、仲間と共に成長し、仲間と共に遊ぶ。
そんな場所がめいちゃんち。
今回は特別、６月から１８歳がリーダーを務める、醍醐のカフェ付きシェアハウスめいマンマで開催です！

【ファシリテーター・強者社会人紹介】（随時更新予定！！）


南　和希
2年の休学期間中に、インド1周、富士山8合目で70日間の山小屋生活、
京都老舗旅館での住み込みアルバイトなど、全国(世界？)津々浦々を、
留学ならぬ「流学」することで、多くの人間と出会い語り合う。
この企画と同じような経験をした際、とある方法で2時間以内に1人で7500円稼いだ。


扇沢　友樹
株式会社めい　代表取締役
中学時代から起業を夢見て、大学では法学を学ぶ。
一日100円生活を強いられていた大学4年間は、古本屋でバイトし、
ひたすら読書に励む日々を送っていた。
卒業直前に内定先を辞退し、起業。
現在ではシェアハウス４件を運営中。


日下部　淑世
学生時代、自身のアイデアが商品化され、2010年秋東京ギフトショーで300万点中グランプリに選ばれる。
2010年度同志社大学奨学生代表。
卒業と同時に会社を立ち上げ、京都をデザイン思考で元気にしようと画策中。


谷　亮治
大学在学中より、まちづくり活動に参加。
大学卒業後は、NPO法人でまちづくりの実践に携わりながら、
大学院で地域コミュニティについて研究する。
現在は市役所でまちづくりのアドバイザー業や、大学で講師業を営みながら、
各地に素敵なコミュニティが増えるように活動中。


小林大祐
京都大学で聴覚障碍者に対するコミュニケーション支援の研究をしながら、
NPO法人Design with all 理事長 として、インクルーシブデザインという手法を用いて、
ものづくり/デザインワークショップを開催し共創の場を提供している。
