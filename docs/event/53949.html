概要

慶大（CBRC協力研究員）の佐藤先生の呼びかけにより「ISMB2014読み会」が行われます。

ISMBは、バイオインフォマティクスの国際学会であるISCBの年会です。
今年は37報のProceedings論文が出ています。
<a href = “http://bioinformatics.oxfordjournals.org/content/30/12.toc” >ISMB 2014 PROCEEDINGS

14/09/01 発表者が確定しました。

CBRC外部の方へ

入館証が必要となります。

事前に氏名と所属を連絡くださった方：1 階の受付にて入館証を受け取ってください。
連絡されていない方：1 階の受付にて書類に記入後、入館証を受け取ってください。

発表者の方へ

論文1本当たり20分程度で紹介していただく予定です。

聴講希望の方へ

聞くだけの方も歓迎です。どうぞご参加ください。
