description

スパイラルテープは、二十年前の箱根駅伝で記録が生まれ、一躍ブームとなったことから「スポーツ」というイメージが強くありますが、実は病院、整骨院、鍼灸院などで多く使用されている「テーピング」です。
一般講演をはじめたのは、様々な症状に対応できるスパイラルテープを知っていただくと共に、家族・友人・知人など「家庭で簡単にできるスパイラルテープ」を知っていただくためです。
「腱鞘炎・冷え性・花粉症・腰痛」講演をするたびに、これらの症状でお困りの方が多くおられることに驚かされます。講演終了後に皆様の笑顔や症状の改善を伝えていただけることで喜びを感じております。今回は「頸・肩の痛み」に対してテーピング法・運動療法を誰にでもわかるように講演致します。
是非お困りの方はご参加ください。
