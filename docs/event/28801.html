description

企業IR担当者様向けのセミナーを開催致します。

「IRページを作りたいが制作会社に知識が足りない…。」
「作ったは良いが費用対効果が測れない…。」そんな悩みにお答えします！
数多くの企業のIR を改善に導き「インベスターリレーションズ」（CFO 協会）、「リスクマネジメントとIR」( 日本リスクマネジメント協会)などの著書で知られる福田 徹がWEB社会に特化したIRの手法についてご紹介致します。

セミナー内容

5つの秘策で御社を変える！

１．魅力的なIRホーム ページとは
２．米国のネット IR事情について
３．アナリストレポートとIR ホームページ
４．皆様のIRホー ムページの内容は？
５．IR 担当者が知って おくべき最近の動向

多様な情報が交錯する現代において、機関・個人投資家にとって企業のIRページは非常に重要な情報収集源となりました。従来通りのアニュアルレポートのみを用いたIR活動では、企業の投資価値を充分に伝える事が出来ず、投資家を集める機会損失に繋がりかねません。
本セミナーでは、実績あるIRのプロが投資家にとって本当に価値のあるIRサイトを構築する為に5つのポイントをお伝え致します。

株式会社MAXIS アソシア IR アドバイザー

豪州ボンド大学MBA ／関東学院大学講師／國學院大學講師
