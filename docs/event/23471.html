＜一般観覧者を限定３０名ご招待＞

東京写真学園主催のフォトサークルWIKIPHOTOがお送りする第2弾、
冬のシークレットワークショップが開催されます。

なんとあのNO.1国際英文ニュース誌「TIME」とApple Store,GINZAから
特別協賛していただき“セルフポートレート撮影レッスン”と“フォトコン
ペティション”を行いあなたの「伝える写真力」を格段にアップさせます。

もし「あのTIME誌の表紙をあなたの写真が飾るとしたら」このお題を通して
あなたは「何」を伝えたいのか、自分を投影して技術やセンス・想いを再確認できる場を提供します。
フォトコンペティションはアップルストア銀座店のシアタールームで行い、
講師のほか以下の審査員をお招きして審査講評をしてまいります。
一般観覧者にも投票権が与えられ審査を後押ししていただく方式です。
前代未聞の“コラボレーション企画型ワークショップ”にぜひご参加ください。

特別審査員：
タイム・インク日本支社 日本代表 金杉一朗さん、プリンティングディレクター松平光弘さん、写真家 青山裕企さん（予定）
