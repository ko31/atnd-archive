9月10日に開催してから、非常にスパンとしては、短いですが、もう第4土曜日が近づいてまいりました。

福岡のコンテンツ業界を盛り上げるべき、業種の｢垣根｣を取り払いメンバーそれぞれの見識を広げ、それぞれがスキルアップすること、ネットワークを築き、ビジネスの幅を持たせること等を目
的として、毎月第4土曜日16時から2時間勉強会、その後飲み会を実施しています。

… メンバーの中心は、ゲームやアニメ、CM制作等のクリエイターの方々の他、大学院生、会計士、ベンチャーキャピタル等の方々がいらっしゃいます。

ご興味があれば、是非ご参加ください。詳細(案)は下記の通りです。
