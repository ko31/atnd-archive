description

実は、あまり知られていない補助金ですが、現在お持ちのサーバを低電力なクラウドデータセンターに移行する費用を助成する、「中小企業等省エネルギー型クラウド利用実証支援事業費補助金」通称【クラウド移行補助金】のご案内および申請に関して必要な作業をお伝えするセミナーです。

サーバを移設するだけで最大1500万円助成される大変大きな補助金です。
助成枠は20億円もあり、初年度という事もあって、まだまだ枠に余裕があります。
需要とマッチすれば非常に通りやすい補助金ですので、リースアップ目前の企業様や、サーバ移転をご検討の企業様、クラウド化にご興味のある企業様等、活用して下さい。

【受講対象】10名限定
　オンプレミスのシステムを導入して3年以上経過していて以下の項目にチェックがつく企業のご担当者様
　　□システムの障害が最近多くなってきている
　　　[想定できる原因]
　　　　・サーバーの老朽化
　　　　・データやトランザクションの成長にサーバーの性能が追いついていない
　　　　・潜在バグ
　　□上記問題へのシステム会社の対応が悪いと感じている
　　□事業展開に応じてクラウド化を検討
　　□システム会社からリニューアルの提案があり検討中
　　□本補助金のことは知っているけど申請が面倒に思っている
