description

ゲームプログラミングについての情報は、最近徐々には増えてはきましたが何かと少ないですよね。
勉強会自体も軽くググってみた感じでは少なく感じます。
でも、知りたいやりたいって人は多いと思いますし、僕個人としてもいろんな人とゲームプログラミングについて話したりしたいと思いましたので開催したいと思います！
初めて１から企画する勉強会ですのでご迷惑をおかけすることがあるかと思いますがよろしくお願いします。

■概要

ゲームプログラミングについての勉強会です。

  • コンシューマーゲーム
  • オンラインゲーム
  • ソーシャルゲーム
  • ゲームエンジン
  • etc…

大体上記のような内容についてをやっていきたいなと思っていますが、少しでも関係していれば何でもありだと思っていますｗ

また、質問がありましたが参加にあたってのゲームプログラミングの経験・知識は一切不要です。
参加したいという気持ちがあればそれで十分ですので、ゲームプログラミングに興味をお持ちの方は気軽のご参加ください。

■場所・日時など

企画はじめたばかりなので未確定の部分が多いです。
決まり次第お知らせしますのでしばらくお待ちください。

日時

５月７日（土曜日）

場所

エビスビル（AAホール本館） ６階 貸会議室
大阪府大阪市中央区淡路町3丁目2番9号 エビスビル(AAホール本館) ６階
http://www.aa-hall.com/conference/ebisu.html

アクセス

地下鉄御堂筋線の本町駅の３番出口を出て右方向に向かって進んで下さい。
しばらく進みますと角にコンビニのサンクスがある交差点がありますので、
そこを右に曲がってしばらく進むと右手に会場であるAAホール本館が見つかると思います。
ビル自体に大きくAAホールと書かれた看板などがありますので比較的わかりやすいと思います。
会場はビルにあるエレベータまたは階段で６階に上がって直ぐです。

遠方から来られる方へ

JR大阪駅へ来られる方
　地下鉄御堂筋線、梅田駅より会場の最寄り駅である本町駅へ
JR天王寺駅へ来られる方
　地下鉄御堂筋線、天王寺駅より会場の最寄り駅である本町駅へ

参加費

無料です。
