description

行きつけのお店に入って「いつもの」。
家族や親しい友人どうしで「あれどうする？」
ツーカーの関係にある上司や同僚から「あれやっといて」。

つき合いが長いと、こんなやりとりでも通じるのですが、
新人採用や人事異動などで「新しい人、よく知らない人」がやってきた時、
こんな経験をしたことはありませんか？

・こちらの意図を理解してくれない
・トンチンカンな答えが返ってくる
・質問に答えず、質問で返す「質問合戦」

そもそも言葉の定義や解釈がひとりひとり微妙に違いますし、
好き嫌いの感情が邪魔をする、といったケースもあるので、
コミュニケーションエラーが起こるのは自然なこと。
相手が悪いわけでも、あなたが悪いわけでもないのです。

本セミナーでは４ピースのパズルを使い、敢えて

・もどかしいコミュニケーション
・言葉があっても100%通じない世界

を体験していただきます。

・思い込みによって、どんなコミュニケーションエラーが起こるのか？
・言葉の解釈に違いがあると、どんなコミュニケーションになるのか？
・ひどいコミュニケーション、最悪のコミュニケーションの先にあるものは？

など、楽しみながら体験することができます。

「相手を不愉快にさせてしまったら？」

大丈夫です。
安心・安全の空間を作りつつ、しっかりフォローいたします！

このパズルワークで、仕事（職場）の人間関係だけでなく、
パートナーや友人、家族と、もっとわかり合えるきっかけに
なるかも知れませんよ。

★講師プロフィール
石津恵子（いしつけいこ）

チーム活性化ナビゲーター。

東証一部上場製造業会社人事・教育部門勤務28年
新人教育の企画・運営（対象者12,000人）
およびOJTトレーナー（年間150名）に対する教育業務に携わる。

OJTトレーナーのサポートを通じて、現場の教育に関する悩み、
チームづくり・再構築への悩み解決に関与。

この経験から、やる気のない社員のモチベーションをあげ、
チーム全体の目的遂行力を向上させる独自の理論
「５ワード３セット～最高チームのつくり方～」を確立。
チームの立て直しはもちろん、活性化にも大きく貢献する。

これを世の中に広めるために2018年独立。
発表と同時に数々の企業から、チームづくりに欠かせない
新たな手法として話題を呼んでいる。

自らの、プロジェクトにおけるチームビルディングの失敗経験を活かし、
メンバーに寄り添いながら「動いてもらう」ためのノウハウを提供している。
