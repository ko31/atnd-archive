今、アナログ・ボードゲームが人気急上昇‼　
なかでもドイツを中心とする欧州発のユーロゲームは、策略をめぐらし交渉したりして勝敗を競う知的な大人のコミュニケーション。
今回は世界で大人気の名作ゲームを中心に、ボドゲ初心者でも楽しめるイベントを開催します。
ボードゲームに興味があるけど、ちょっとハードル高そう…とか思っていた方！
1000種類以上のボドゲを制覇した“ゆーや”さんが、はじめての方にも丁寧に教えてくれますよ☆
この機会にぜひボードゲームの世界に挑戦してみませんか？
