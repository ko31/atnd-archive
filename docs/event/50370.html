description

受講対象・レベル
クリーン化技術の担当者・責任者で実務経験を問わずに現場改善に取り組みたい人。初級・中級レベル。
習得できる知識
・クリーンルームの基礎知識と塵埃の本質
・クリーン化の具体的方法とその根拠
・静電気が塵埃を付着させるメカニズムとその防止対策
・正しい清掃方法、現場での塵埃の観察法
・無塵服からの発塵、現場での色々な発塵とその理由
趣旨
塵埃は製品の不良に大きく影響する。塵埃を製品に付着・吸着させないためにはクリーン化技術理論に基づいた基礎知識と現場での実践にある。本セミナーでは、塵埃の付着・吸着現象を気流・静電気・発塵防止の基礎から解説し、現場での発塵防止対策や静電気対策、また清掃方法や使用器具など知って得する対策の本当のノウハウを限定して解説する。
プログラム
１．塵埃とクリーン化技術の基礎知識（本当は知らない塵埃の本質）
　　1-1　塵埃とは何か、定義と本質
　　1-2　塵埃の発生源とその特徴と挙動
　　1-3　塵埃対策の考え方とクリーン化技術の基礎知識
　　　　 ※実際の測定データーに基づく具体的説明
　　1-4　大きな粒子の挙動
　　1-5　無塵服からの発塵についての新しい解釈とAmenity
　　1-6　拭き取りワイパーの特徴
　　1-7　バクテリアとクリーン化
　　1-8　動画で見る意外な発塵
２．静電気の本質と静電気対策（理屈が分かる静電気）
　　2-1　静電気の本質の分かり易い説明、なぜ塵埃を付着させるのか
　　2-2　静電気の基礎知識、知ってて得する静電気
　　2-3　静電気対策の基本とその根拠
　　2-4　静電気対策の基礎、具体的データーを用いた解説
３．製造現場でのクリーン化技術（教科書では書かない書けない現場の知恵）
　　3-1　クリーンルーム内での作業者の注意事項
　　3-2　クリーンルーム管理のポイント
　　3-3　こんなに発塵する手袋・ノート・素手
　　3-4　一番危険床からの塵埃の巻き上がり
　　3-5　クリーンルームの清掃方法
　　3-6　製造装置や関連設備の注意点
　　3-7　清掃の原則と周期、その理由
　　3-8　各清掃方法の効果、水拭き一方向が一番良い理由
　　3-9　清掃後の塵埃チェック方法
