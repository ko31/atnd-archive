「世の中の『幸せ』みたいなものにあわせてみたら、 幸せなのかなぁ」
と、真似てみた時期が私にはありました。
世間の軸に合わせてみて・・・ それなりの幸せでしたが、少し違ったんです・・・。

自分の理想は、自分にしか描けない事を知りました。

望みを叶えるためには、まずは望みを定めることから。
女性の人生は、結婚、出産、子育て・・・。

選択する機会が多いです。
別に結婚してもイイし、しなくてもイイ。

子どもを産んでもイイし、産まなくてもイイ。

何が幸せなのかは、個人によって違うと思うのです。

ただ、その選択を積極的にしてきたかどうかで、
人生の満足度は変わっていくと思うのです。

ぼんやりとしたものが少しでもはっきりできるための
お手伝いができればと思い、今回企画しました。

人の話を聞いて、自分自身を振り返って・・・。

どんな事を大切にする自分なのか。

そこがわかれば、すーっと、自分の人生の道筋が見えてくると思います。

前半は結婚について。優先したいこと。築いていきたいパートナーシップの形。

後半は出産&子育てについて考えるキッカケを提供します☆
