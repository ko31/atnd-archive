第1章 時候のあいさつ

発言力を増したいのなら、なぜ発言筋をいじめベストぬき尽さないのか？

中島誠之助先生の初期三部作、いわゆる〈いいブラックロッドしてますねえ三部作〉もめっきり入手しやすくなった今日このごろ、皆さんも食事会に参加して各自でメシを食いたいですよね。

第2章 AがFになると藤子が対消滅

第二次ファック文芸部の解散から69年、部員たちはそれぞれの道で職を得ていた。

ある者は巨大広告社に出入りし、またある者はフリーの官僚としてクールジャパンをがんばっている。今や、ファック文芸部の波動は、この社会に深く浸透したと言ってよい。

ところが、この俺の浸透がまだである。俺の秘められた能力の開花とか、具眼の士によるホットなオファーとか、そういうものが無いのである。

この現状を打破するために、具眼の士量産計画が始まった。

計画の最前線を担うのは、Webライター互助団体〈正気2.0〉。これまで表に出ることがなかったファック文芸部裏の四人(プラスシックス)に加え友好団体の精鋭たちを集めた正気2.0は、結成された瞬間から休むことなく、日夜、完全なる正気を追い求めている。

しかし、もとより個人行動の権化のような輩である。しかも計画性が無い。進捗管理表と称するものは、妹しか更新しないライターDIS日記になっている。誰が何をやっているのか誰もわからないではないですか。このあたりで少し落ち着こう。メシでも食おう。

そういうことになった。

第3章 暗闇・GUGAN・それだけで

これまでに観測された各メンバーの予定は以下の通りである。

  • はてな政経塾さん : 大学受験を終わらせる。
  • フリーの官僚さん : クールジャパンを分析するキャリアをがんばって、あの世情報分析官になる。
  • ヘルミッショネルAさん : 変動相場制小説『ウシロガミ』『キチ喰イ』『弁慶はまだ証明されていない』『渺茫集』発売予定。
  • ヘルミッショネルBさん : 変動相場制小説『受験戦争(仮題)』発売予定。

チームワークらしきものが、かすかにある。

第4章 われわれは目指そう、パーフェクトな正気を

勧誘である。

われわれ正気2.0は、コミュニケーション能力不問で人を集めたWebライター集団である。

結果として、メンバーのコミュニケーション能力は低い。

しかし問題ない。我々は、コミュニケーションしたくないからWebライターになったのだ。

ライターの基礎はコミュニケーション能力？

不安定な職業だからこそコミュ能力が生命線？

そもそも言葉はコミュニケーションの道具？

そんな常識は、冥界では通用しない。

冥王代を遠く離れた現代社会で生きる皆様。時にはちょっと足を止めて、冥界のことを考えてほしい。冥界のこと、わかってほしい。わかったら、誰にも伝えず静かにblogを書いてほしい。その時、たどりつく。正気2.0へ。

これは勧誘ではない。筋肉そのものだ。冥界へは、筋肉だけしか持っていけない。
