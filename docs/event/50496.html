本イベントは無事終了いたしました。

twitter（@EzukoAppPJ）で随時お知らせします。

開催概要

  • 日時：2014年5月17日（土）・18日（日）10:00〜17:00
  • 受付場所：くまもと環境フェア2014会場内の主催団体ブース（熊本市動植物園 植物ゾーン／熊本市東区健軍）
    • くまもと環境フェア2014の詳細は こちら
  • 主催：ほっとけない熊本プロジェクト（ 江津湖アプリプロジェクト ）
  • 参加費：無料
  • 定員：各日50名（※要事前申込→詳しくは「 参加申込方法 」節を参照ください）

内容

  • 江津湖アプリプロジェクト（仮称）の企画紹介
  • 江津湖アプリ（仮称）の体験
    • 現在開発を進めている、スマートフォンやタブレット端末を使った、
      江津湖一帯の動植物などに関する情報を知ることができるアプリケーションの体験

参加申込方法

  • 参加無料・当日参加可能ですが、参加者数把握のために下記のいずれかの方法での事前申込をお願いいたします（ 事前申込は、5月16日21時で締め切らせていただきました ）。
  1. 「 参加申込フォーム 」からの申込
  2. 右上の「 このイベントに参加登録する 」ボタンによる申込
    いずれの場合も申込いただいた後2〜3日以内に主催団体の事務局よりご連絡をさせていただきます。
