2015年11月26日に【笑顔で働きたいママのフェスタｉｎ神奈川】がアートフォーラムあざみ野で開催されます。
育児期という制限がある中で、働きたい！起業したい！活動したい！というママたちに有効な情報提供と一歩踏み出すきっかけの場として自治体や企業と協力してサポートするイベントです。

当日は起業や会社経営している沢山のママが集まりますよ。

～催事内容～
◆ワークショップ
◆ママ起業家によるパネルディスカッション
◆講演
◆ママを応援する各企業ブース
◆ライブショー
◆ベリーダンス
◆あざみ野マルシェ
◆カフェスペース
など。

子供同伴可能なイベントなので、ぜひ気軽にいらして下さい。

【開催日】：2015年11月26日（木）　10：30～16：30

【開催場所】：アートフォーラムあざみ野　（ブルーライン・田園都市線あざみ野駅から徒歩5分）

【参加費】：入場無料・各ワークショップやプチ講座は500円～1，000円

【その他】：託児所事前予約・授乳スペースあり
