イベント概要

越前市の某事務所にて、「ドミニオン」の体験会を開催したいと思います。

未経験者歓迎、もちろん、経験者の方もぜひどうぞ（インストしてあげて下さい）。

ドミニオンは２セット、そのうち１セットは @monobyte 氏より全拡張をお借りできる予定です。（私の持っているのは基本＋陰謀のみ）

何度かプレイして慣れたら、最後にちょっとしたトーナメント戦を実施したいと思います。

第一回の勝者は誰だ！？

尚、会場にはプロジェクタ＆スクリーンがございますので、ボードゲームやドミニオンなどの話題でLT（5-10分程度）をしたいというツワモノがいらっしゃいましたら、ぜひおながいしますｗ Ustしましょうｗ（準備があるので、希望者はコメントでお知らせ下さい）

トーナメント戦について

11/26の越前市ドミニオン体験会では、初心者向けのインストを2回行った後、トーナメント戦を行います。

まず、初心者の方がプレイになれる為、最低でも2回「最初のゲーム」で練習プレイを行います。時間があれば3回目にランダムか別の組み合わせでプレイを行います。練習を早く終わらせる為、初心者の方の卓はなるべく3人卓になるよう調整します。また、1回目の練習は例え3人以上でも勝利点カードを8枚にしてプレイします（早く終わらせる為）。2回目以降は人数に適した勝利点カードの枚数で遊びます。

練習プレイはインスト込みで1時間30分程度で終わるようにします。

この間、他の参加者は別途ドミニオンを楽しんでいてください。初心者さん達の練習プレイが終了次第、可能な限り速やかにプレイを終了してください。

変則トーナメント概要

☆1卓の場合

１位だった人は＋３点、２位だった人は＋２点、３位だった人は＋１点、４位だった人は＋０点。
となります。（３人プレイの時は、各－１点する）

これを繰り返し、最終合計が多い人が優勝となります。

☆2卓の場合

１つめの卓を「１軍」、もうひとつを「２軍」とします。１軍が強い人の卓、２軍がまだ弱い人の卓です。
練習プレイなどを通じて、明らかに強かった人は１軍卓へ、明らかにまだ経験不十分な人は２軍卓へ座ります。
どちらか微妙な人は、じゃんけんで勝った順に選んでもらいます。

同時にスタートし、どちらかが終了したら、もう片方が終わるまで待ちます。
点数は1卓の場合と同じです。

その後、２軍の１位と２位は１軍へ移動し、１軍の３位と４位は２軍へ移動し、次のプレイを行います。これにより、卓の強さのばらつきを調整します。

片方の卓があまりにも早く終わってしまった場合、もう片方の卓の進捗具合を見て、全員の同意を得た上で再戦が可能です。２戦目以降については、もう片方の卓のゲームが終了した時点であと１周だけ行い、ゲーム終了とします。２戦目以降の勝敗は、戦績には含まれません。

☆3卓の場合

2卓の場合のルールで、「３軍」を作ります。

後は2卓のルールと同じです。

ドミニオンとは

ドミニオンとは、昔「マジック・ザ・ギャザリング」とか「遊戯王」とかのトレーディングカードゲームと言われるタイプのゲームで流行した、「自分でカードを買ってきて自分だけのカードセットを作って友達と対戦する」という仕組みをボードゲーム風にアレンジしたもので、トレーディングカードゲームのように「お金を使ってレアカードをたくさん手に入れた人が強い」ということがなく、誰かが１セットだけ買ってきてみんなで遊ぶことができます。

このゲーム、これまでいろんな人と遊んできましたが、ボードゲームに免疫のない人であってもすぐにルールを把握し、「もう一度やろう」と言わせる力を持っています。もちろん、トレーディングカードゲームなんか触ったことがない、というような人でも同様です。

これは私も驚くほどで、「この人はあんまりこういうのは喜ばないかもしれないな・・・」という人でも、なぜか必ずみんな気に入ってくれるのです。

ルールはシンプル。カードを５枚引いて、手札からアクションカードを１つ実行、次に手札の残りからお金カードを出して新しいカードを１枚入手（購入）、後はこれを繰り返す、というだけ。これに加えて、カードの種類は基本セットだけでも25種類、拡張セットも入れると既に100種類近いのに、１回のゲームで使うカードの種類がランダムに選ばれる10種類のみだから覚えることが少ないというのも非常にとっつきやすい理由になっていると思います。

カードの種類が毎回ランダムに変わる事でやるたびに違った展開になり、飽きることがないというのも中毒性が高い理由で、「ボードゲーム会」と言いながら延々とこれだけやっていてもたぶん誰も文句を言わないだろうとすら思わせる実力を持っています。
