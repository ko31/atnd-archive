クラウド電話API Twilioのハンズオンイベント
Twilio API 勉強会 Vol.9を開催致します。

■ Twilio API 勉強会？
クラウド電話 API Twilioが好きなデベロッパーのみなさんとTwilioを触りながら情報交換を目的にしています。

■ 参加対象
Twilioに興味があるデベロッパー（言語、分野は問いません。）
・Twilioはウェブはもちろん、ハードウェアなど。あらゆるシステムと電話を簡単に繋ぎます。

■ 持ち物
・Twilio アカウント（トライアルアカウントでも構いません。）
※ ハンズオンが始まる前まではご用意ください。（新規登録時、お試しポイント付き）
・パソコン
※ コードを書きながらTwilioを試して行きます。

★ WEBサーバー（環境）
– PHP環境（遊び場）やサンプルコードなどを用意します！

■ ハンズオンイメージ
・インバウンド：Twilio向けに電話をかける。
・アウットバウンド：Twilioから電話をかける。
・ブラウザを電話機に変身〜！
