HPC勉強会について

HPC：High Performance Computing、高性能計算についての勉強会です。
学会のように難しいことばかりを求めず、ゆる～く、気軽に続けていきたいと考えております。
HPCに関心、興味ある方、何も知らない初心者、昔ちょこっとHPCをかじってた方、自作のアプリケーションを高速化する方法を知りたい方等、いろいろな方面の皆様の参加をお待ちしております。

※HPCというと漠然と広範囲になるので、
　当面はMPI：Message Passing Interface の入門を軸に進める予定です。
