このセッションは、jQuery(JavaScript)の入門者を対象とした教室です。

概要

jQueryとは、javaScriptの軽量ライブラリで、主要ブラウザのほとんどに対応し、手軽に使えて本格的な機能も備えた非常に優れたライブラリです。

W3Techsによると、WEBサイトの過半数はjQueryを利用しているそうで、もはや必須のライブラリといえるでしょうか。

利用されるケースの一つとして、jQueryベースのプラグインを活用したスライドや開閉パネル、ダイアログなどのエフェクト効果を実装しているケースも非常に多いかと思われます。

属性値やスタイルなどをプラグイン指定の方法に合わせる事で、JavaScriptを記述する事無く得たい効果を実現できて大変便利ですが、言い換えるとプラグインのコーディング規則に従う必要がある為、既にマークアップやスタイルが完成したサイトでは、後からプラグインに合わせる為に変更を余儀なくされます。

今回のセッションでは、プラグインに頼らず、基本文法（構文）を学びながら、HTML／CSSに対する効果やエフェクト処理を実践形式で学びます。
一般的によく使われる手法を取り上げて、そのサンプルをいくつか用意しました。

内容

  • 学習環境の準備（共有フォルダ、サンプルテキスト）
  • 基本文法
  • フィルター（要素の絞り込み）
  • スタイルの取得と変更
  • jQueryの各種メソッド
  • CSSメソッド、HTMLメソッド
  • ツリー構造をたどって各要素へアクセス
  • イベント処理
  • アニメーション処理
  • Ajax操作（サンプルの紹介のみ）

セッション終了後には個別に質問を受け付けます。気軽にお尋ねください。

学習環境

以下の環境（設備）は、こちらで用意してあります。参加者が事前に構築する必要はありません。

  • インターネット接続
  • 電源（延長コード）、LANケーブル（有線LANの場合）
  • WEBサーバー
  • ファイルサーバー

必要な準備

当セッションに持ち込めるPCをご用意ください。

  • パソコン（Windows,MacどちらでもOK）
  • LAN接続（無線LAN、有線LANどちらでもOK）
  • ブラウザ（IE,FireFox,Safari,Chrome）
  • テキストエディター（文字コード：UTF-8に対応の）

テキストエディターにつきましては、
文字コード「UTF-8」が編集可能なものをご用意ください。
WindowsXPに標準搭載の「メモ帳」などは、UTF-8に対応していない場合がございます。
「秀丸エディタ」「サクラエディタ」「EMエディター」などがお勧めです。
また、行番号が表示される物が便利です。

対象とする方

  • WEBデザイナー・コーダーの方
  • 趣味でWEBサイトを構築している方

予め用意された、HTMLテキストを使用して講座を進めます。

/* CSS */
p { font-size:14px; }
#alpha { background-color:blue; }
.common { text-align:left; }

上記の記述が理解出来る方であれば、jQueryの理解は難しくない！という前提でこのセッションを組み立てております。
