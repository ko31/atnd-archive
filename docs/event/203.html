Mashup Awards 4 (MA4) にご応募いただき、ありがとうございました。
今回、MA4 にご応募いただいた方の交流会（忘年会）を開催いたします!
師走お忙しい中とは思いますが、皆さま奮ってご参加ください。
　
MA4 の開催や、リクルート Web サービスの運営を担当している
メディアテクノロジーラボのオフィスに、ぜひ遊びに来てください。
受賞者・応募者・事務局メンバーとの交流に！
　
【会場】
東京都中央区銀座8-4-17 リクルートGINZA8ビル 2F
http://map.doko.jp/m/sc=70498705/zm=10/
　
【日程】
2008年12月10日（水）
19:00～　開場
19:30～　第１部：応募作品振り返り、PRタイム
20:30～　第２部：交流会（会場はそのまま）
　
【第２部の参加費】
・一般　無料（差し入れ歓迎♪）
・学生　無料

【事前アンケート】
・もしよろしければ、こちらでアンケートの回答 をお願いいたします。
　http://spreadsheets.google.com/viewform?hl=ja&key=pl-8krv0RI9M-8L55IfTeUw
・アンケート結果は、今後のアワード運営やリクルート Web サービスの
　参考とさせていただきます。（所要時間：10分程度）
・アンケートは忘年会の参加・不参加によらず、ご回答ご協力ください！
・回答いただけた方は、第１部で１分間の PR タイムをご利用いただけます。
　
【備考】
・基本的に、リクルート Web サービスを利用した作品を Mashup Awards 4 にご応募いただいた方が対象です。
　⇒ その他の API を利用された方も歓迎しています♪
・会場が限られるため、グループで応募された場合は、1作品について、最大4名様までの参加でお願いいたします。
・参加登録は、人数確認のため、できるだけ参加される方1名ずつ登録をお願いいたします。
