WebRTC Meetupについて

  • WebRTC MeetupはWebRTCに興味がある全ての人向けの勉強会です
    • 2014年4月東京で第1回勉強会を開催
    • 2017年8月大阪で初開催
    • 2018年4月福岡で初開催
  • 触ってみた的なネタから、WebRTC技術の濃いネタ、本格的な活用事例までコミュニティ・法人・営利・非営利問わず扱います。WebRTC技術の普及、活用サービスの普及につながるものであれば原則なんでもOKです。
    • 注意事項
      • 会の趣旨にそぐわない内容については講演をお断りする可能性もあります
  • 全体的な技術レベルは高めです
  • 発表者が不在の場合は開催を取りやめることがあります
