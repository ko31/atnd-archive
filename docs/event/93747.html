XSHELLが開発しているisaax。Raspberry Pi や Edison , NanoPi などのLinuxが動くSBC(Single Board Computer)をカンタンに使うことのできるシステムです。

今回は Raspberry Pi ＋　カメラを使って、インストール・プログラム登録・画像認識とプログラム変更をお手軽にやってみましょう。(2017/12 に行った勉強会の拡大版です。前回の勉強会の様子はこちら→ https://xshell.io/2017/12/iot-raspberry-pi-camera/ )

画像認識はOpenCV使って動体検出やります。メンドクサイOpenCVのインストールは既に終えているイメージを使うので♪

IoTっぽくRaspberryPiはヘッドレス（ディスプレイ、キーボード、マウス、GUI画面を使わない）で扱い、 -クラウドに送信、isaaxから設定を行います。- → 今回はクラウドからデプロイ、プログラム変更して動作に反映をやっていきます。

isaaxを使うと、 - PCでプログラム作ってGitHubにpushするとRaspberryPiの内容がスグ更新 - 電源とネットワークさえRaspberryPiにつなげばいいのでカフェでファッショナブルにだってできる

使い方を説明しながら、カメラで画像認識までやってみます。 誰でもWelcomeです！！

参加無料です。RaspberryPi持ってない人は、貸出OKです。

XSHELLとは

シングルボードコンピューターを使って、プロトタイピングから、プロダクションモデルまでをシームレスにつなぐ、IoT向けのPaaSを開発しています。 WEBアプリケーション技術、ハードウェア技術、3Dモデリング技術を組み合わせて、IoTの事業開発支援も行っています。 オフィスはレーザー加工機、3Dプリンタ、小型CN旋盤、オシロスコープなどがあって、ミニFablabみたいになってるよ！
