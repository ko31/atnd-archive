description

モバイルワールドコングレスは大盛況でした。
ヨーロッパではモバイルコンテンツ市場が急成長し、コンテンツ各社は鼻息が荒かったです。
次のステップとしてNFCが注目されております。
MWC2012では多くのNFC関係者と会ってきました。
大手銀行、交通機関やリテーラーはNFCに積極的で日本企業にもビジネスのチャンスが有ります。

１６日（月）に弊社CEOが来日し、「モバイルヨーロッパ２０１２」と題しまして、
モバイルワールドコングレスのリポートを行います。
さらに、今回はGMSAの許可を得て、ディレクターのDr.Bains氏の最新NFC戦略についてのセミナーを
代行して発表します。参加者にはGSMAのNFCドキュメント（最新スペックと業界向け資料一式）を
無料で配布します。

是非、ご参加ください。
