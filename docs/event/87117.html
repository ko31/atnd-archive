description

　株式会社クラウドワークスと株式会社シプード共同で、広報に興味がある方、広報としてスキルアップしたい方向けに広報・PR入門セミナーを開催します！

広報・PRとは何かを一から学ぶことができます。基本的な広報・PRの役割、重要性、またどのように広報・PRスキルを高めていけば良いのかを、“広報・PRの家庭教師”としてPR活動計画の立案と実行のアドバイスや、プロモート活動の支援を行っているシプードが伝授します！また、クラウドワークスの広報の成功事例や、過去に失敗例を広報担当者からお話とともに、必要とされる広報人材についても触れていきます！

皆様のご参加お待ちしております。

■「広報・PRの基本について」講師

株式会社シプード　代表取締役　舩木真由美
大学卒業後、人気テレビ番組（とくダネ）の制作に携わる。その後、大手PR会社を経て、楽天株式会社の広報に従事。楽天市場のPR担当として広報組織の立ち上げに参画。楽天賞3回受賞。2014年3月独立。PR代行業が主流のなか、企業の中の広報人材を育てる業界初の「広報・PRの家庭教師サービス」を立ち上げる。約3年でスタートアップ企業中心に60社以上の広報・PR担当者を育成。現在に至る。

「企業内に自走する広報・PR組織の確立」を目指す広報・PRのプロフェッショナル集団、株式会社シプードが、これから広報・PR活動を始めたい、または強化したいという企業や広報・PR担当者に役立つ、広報の基礎的知識やノウハウを解説します。またクラウドワークス広報担当からは、広報活動での成功事例・失敗事例をご紹介させていただきます。
ーーーーーーーーーーーー
■「株式会社クラウドワークス広報・PR事例紹介」登壇者

株式会社クラウドワークス　渉外・広報　石山アンジュ
クラウドワークス渉外担当の傍ら、一般社団法人シェアリングエコノミー協会事務局担当、「シェアガール」の肩書で海外・日本でメディア連載、内閣官房シェアリングエコノミー伝道師、クラウドワークス人事制度の「ハタカク！」の副業制度を活かし、シェアサービスを通した複業や週末に社会活動を行うパラレルキャリア社員。
ICU卒、在学中に日本テレビイベントコンパニオン・NHK 番組補佐を経験後、新卒で株式会社リクルート入社。人材領域で大手法人営業を3年半経験後、現職。
ーーーーーーーーーーーー
株式会社クラウドワークス　広報　稲増祐希
大学卒業後、2年のフリーター期間を経て、人工光で野菜づくりをする植物工場の装置メーカーに就職。野菜栽培や広報を担当。農業の六次産業化支援などを通じて、様々な業種や分野でのクリエイティブの必要性を実感し、様々なスキルやリソースをオンラインを通じて得ることができるクラウドソーシングに興味を持つ。その後、クラウドワークスの広報アシスタントとして入社。ほぼ広報未経験の状態から、広報専任に。現在は、週末にカフェ運営を行う副業社員。
ーーーーーーーーーーーー
