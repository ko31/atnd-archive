description

３日間でPCBの作成までやってしまう強行体験会！

  • 回路CADのインストールからはじめて
  • 工場に基板を発注
  • 部品を集めて
  • 組立て依頼
  • ３日間で作ってもらう！
  • 基板の設計の仕方、BOMの作り方、工場への製造指示の出し方・・・・などなど、一気に体験しよう！

個人のホビイストが驚速で作るためのガイダンスとして！

企業のハードウェア担当が新しいやり方を必要最低限身に着けるためのサバイバルとして！！

ご注意

  • 場所代金￥5000　に、製造費用がかかります。
  • 製造費用は1200元（￥20000）ぐらいです。製造業者に直接支払います。人民元をあらかじめご用意ください。
  • PCをお持ちください。
  • マウスをお持ちください。(KiCADの操作はマウス必須です。)

イベント概要

  • 現地集合、現地解散
  • 1日目 KiCadインストール、基板設計、製造依頼
  • 2日目 パーツを電気街で集めて組立て指示、（ or オプションの深センツアー ）
  • 3日目 製造しあがったものをチェック、仕上げて遊ぶ
