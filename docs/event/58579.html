event_id 58579
title 【成長業界】環境業界転職セミナー[福岡・無料]
catch 他業界での経験や資格を活かして、 成長業界『環境分野』への転職に興味のある方向けに、テレビ会議にて無料セミナーを行います。
description

環境業界への転職に関心のある方に向けて、転職活動をはじめるため、
成功させるためにどうすれば良いかを解説するテレビ会議セミナー。

太陽光やLEDを中心に、成長を続けている環境ビジネス業界。
環境省によると、2020年には120兆円市場に成長すると算出されています。
成長産業であり、社会的な貢献度も高い、この分野への転職に関心を持つ方も増えています。
今回は、『環境業界に関心はあるけど、まだ、具体的には動いていない…。』という方を対象に、
『環境業界』と『転職準備のやり方』の基本を解説致します。


[このような方を対象にしたセミナーです]

『転職に関心はあるけど、時期や希望企業等何も決まっていない…。』
『転職したいけど、どのように進めれば良いかわからない…。』
『転職先の企業で自分のスキルや経験が通用するか不安だ…。』
『環境業界に関心はあるが、安定的に成長する産業なのか気になる…。』
『具体的にどんな求人企業があるの？』

といった声に、環境業界の専任キャリアコンサルタントが徹底的に解説させて頂きます。

[セミナー内容]

1. 環境ビジネス業界の展望
2. 転職市場の動向
3. 成功する転職のポイント
4. 具体的な求人情報と求められるスキル
5. 転職に必要な準備やスケジュール
