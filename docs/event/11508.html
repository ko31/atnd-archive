ソンタナゲーム会、連休中日の日曜日に開催です！

つい先日開催し、大好評をいただいたタイ料理コース懇親会付ゲーム会。様子を見聞きした方より、「自分もゲーム初心者だけど行ってみたい！また是非お休みの日開催をやって欲しい！」とのご要望を多数いただきました。調子にのって、えらい短いサイクルですが良い日が押さえられましたので、やります！

ソンタナゲーム会って何？

世田谷区経堂にありますタイレストラン「タイ屋ソンタナ」で毎月定例開催されている、カードゲーム・ボードゲーム等を一日遊び倒す会です。

今回はいつもと何が違うの？

今回は、開始時間を16時～とします。ランチタイムは「カレーカフェ・ソンタナ」として営業していますが、もっと早くから遊びたい！という方は、もちろん早くからご来店いただき、カレーをお召し上がりの上、昼過ぎからプレイしていただいても問題ありません。どうぞ @satokoichi までお問い合わせください
