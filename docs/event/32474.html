★講演者プロフィール

■杉本 哲（すぎもと あきら）氏

システムスタジオ哲 （個人事業主）
主にAndroidとiOSのアプリ開発やコンサルタントを行う

小さな会社でAndroidアプリの開発を行っていたが
結婚を機に独立、第一子も生まれ自宅で子供の面倒を見つつ仕事をするようになる。
その後、実家に引越し、両親に子育ての協力をしてもらえるようになったため、
今は常駐と在宅の仕事を並行してこなしている。

【杉本様から一言】
　「独立して仕事をするようになりまだ一年ですが、その間に多くのことを学ぶことができ、
　　また充実した仕事・生活を過ごせるようになりました。
　　もっと多くのエンジニアが独立するといいのにな、という想いからこのようなイベントを企画しました。
　　平凡なエンジニアだった私がなぜ独立したのか、
　　そしてフリーランスとなったことでなにが変わったのかを伝えたいと思います。」

■進藤龍之介（しんどう　りゅうのすけ）氏

Computing Initiative（個人事業主）
コンピューターエンジニア

インターネット草創期のネットワークエンジニア、WEBエンジニア（サーバー／アプリ）を経て2001年に独立。
日本Androidの会 コミュニティ運営委員／埼玉支部長

【進藤様から一言】
　「フリーランスは孤独？いえいえ、孤高の戦士なんかじゃありません。ましてやぼっちで仕事したいから
　　フリーになったわけでもありません。
　　時間や業務に縛られない「自由」を活用して、コミュニティ活動に参加してみましょう。」

■染谷 昌利（そめや まさとし）氏

1975年生まれ、37歳。（個人事業主）
主にブログの運営や執筆、講演、企業やブロガー、アフィリエイター向けのコンサルティングを行う

4年制大学卒業後、就職情報誌の営業、自動車関連企業の人事担当者、不動産関連企業の人事担当および
営業担当を12年間経験した後に、2009年に独立という非IT系出身ブロガー。

運営サイトはスマートフォン関連、就職関連、レシピ、ランチ、旅行記など幅広く運営。
メインで更新している「Xperia非公式マニュアル」は月間100万PV、通算2500万PVを超える人気サイト。

また、同サイトはGoogleAdSense成功事例にも取り上げられている。
