もくもく勉強会スペシャル

来る12月12日(土曜日)から、12月13日(日曜日)の2日間、岐阜県は大垣市に御座います、ソフトピアジャパンはドリームコア会議室にて「もくもく勉強会スペシャル」を開催致します。

== 宣伝用 ==
宣伝用画像

エンジニア･クリエイターのための移住応援プロジェクト
ＧＩＦＵ×ＨＵＢと共同開催決定！！

ＧＩＦＵ×ＨＵＢは、岐阜県と株式会社パソナテックがタッグを組み、首都圏から地方に移住を考えているエンジニアやクリエイターの方に、岐阜への理解を深めてもらう移住応援プロジェクトです。 人生のＨＵＢとなり、岐阜県に集まり、人とつながり、その魅力を発信するＨＵＢになりたいをコンセプトに、さまざまなイベントや岐阜ツアーを企画･開催しています。 今回は、【GIFU 行こ。～岐阜で暮らす人に出会うShort Trip～】の大垣ソフトピアエリアの紹介として、もくもく勉強会スペシャルと共同開催させていただくことになりました。是非、多くの方々と交流していただき、出会いの場として繋がっていただければ幸いです。

イベント内容

モグラ叩きゲーム開発合宿(日帰り可能)

会場の前方半分を使い、「モグラ叩きゲーム開発セミナー」を行います。
プログラミング初学者向けの専用セミナーを行いますので、開発経験の無い方でもお気軽に参加頂けます。
プログラミング経験者の方は、独自にオリジナルのゲームを作って頂く事が出来ます。
(デザイン素材については、こちらでご用意致します)
翌日の午前11時より、完成した物の発表発表会を行います。

もくもく勉強会(日帰り可能)

会場の後方半分は、主に「自主学習」をメインにし、参加者様同士の交流等をする場になります。
アプリ開発するも良し、調べものをするも良し、基本的に何をテーマにして大丈夫です。

ドット絵を描こう(参加自由)

「モグラ叩きゲーム開発セミナー」が終わり次第、簡単なドット絵ハンズオンを行います。
こちらも同じ場所で行いますので、ご興味ある方は是非ご参加下さい。
モグラ叩きゲームで利用するモグラを作成します。

ソフトピア1周見学ツアー(参加自由)

ソフトピアジャパンを特急でご案内する約1時間のツアーです。
インキュベーション施設の見学から足湯まで、周辺をご案内します。

FabCoreものづくり体験(参加自由)

NextCore会場の隣にあるFabCoreでは、4時間ほどを使い「ものづくり体験」をする事が出来ます。
定員が御座いますので、お早めにご予約下さい。
募集人員:3~5名

スト2王決定戦(参加自由)

夜は、親睦会を兼ねたゲーム大会です。
昔懐かしいスーパーファミコンで「スト2王決定戦」を行います。
メッセー会場のプロジェクターを使います。
基本的に「持ち寄りスタイル」でやりたいと思っております。
飲み物やお菓子等をご用意しますが、何かしら持ってきて頂けると有難いです。
大会への参加/不参加に関わらずご参加頂けますので、お気軽にお越し下さい。
