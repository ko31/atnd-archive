RubyKaigi2011でこっそり開催した、ゆRubyのtoRubyパートを札幌Ruby会議2012にて行います。

すごいゆるくぬるい勉強会です。

12:00 開場, ランチタイム
12:30 dRuby体験セッション
13:30 終了

島根大の講義のテキストを使った「dRuby体験」です。一台のマシンで複数のプログラムが協調しあう様子を体験します。PC必要。ない人はとなりの席の人とペアプロしよう！

当日使用する資料です。印刷してきてくださいね。

  • http://www.druby.org/shimane2010.pdf (PDF, 無料）
  • http://gum.co/vWGq (Pages書類、$3）
    [注意] 無料版と有料版の資料の内容はどちらも同じです。

セッションの前に短いお昼休みがあります。食べ終わらない人は、dRuby体験をしながら食事をしてくださいね。事前にお弁当などを買っておくとスムーズに進行すると思います。
