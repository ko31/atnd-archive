鹿児島でもLinuxの勉強会がやりたい!

ということで前回は 鹿児島らぐ(Kagoshima Linux User Group) の立ち上げ，RaspberryPi の発表の他 WindowsXP が動作していた IBM ThinkPad X40 に Debian を導入したり，RasberryPi と Arduino の接続などもしました.

- 鹿児島らぐ :: 鹿児島Linux勉強会-第0回-開催(2014/04/26)

今回はOSC Kansai 出展の相談なんかもしつつLinuxやPC UNIX，OSS関連の話題で盛り上がりたいと思っています．

発表

LTのような感じで発表したいネタを持った人が発表できるようにしたいと思います．
発表ネタはLinux やPC-UNIX などにかすってればOK
相談事とかもOK
(出たばっかりのUbuntu 14.04LTS 入れてみたよとかRaspberryPi 買ってこんなことやりたいけどどうやるのとか XPマシンに Linux 入れてみたけどこれまでXPでやっていたこういうことはLinuxではどうするのとか…)

ネタ(こんなことやってみたよ，これ教えて，ここがわからない等々)がある人は参加コメントや下のコメントなどに発表したいネタを書いてください．
ネタがない人も是非参加してください．

会場

ソフトプラザかごしまの204号室「鹿児島大学情報技術事業化支援室」です．
正面玄関は閉まっているので玄関から右の駐輪場を回って裏手のインターホンで呼び出してください.
呼び出し方は「204」+「呼び出しボタン」です.

電源，WiFiあり :)
