description

［講座のポイント］
　泡は誰もが体験する最も身近な界面現象の一つです。泡を立てること、泡を消すことは相反する仕事ですが、お互いを総合的に理解することが重要です。本講座では、界面活性剤による起泡・消泡に関し、最低限知っておいたほうが良い理論、各界面活性剤による特徴、および具体的な応用例について解説します。
［プログラム］
１．泡の生成
　　1-1　泡の生成方法
　　1-2　泡を生成するために要する仕事
２．泡の安定性
　　2-1　泡の評価方法
　　2-2　泡の安定性に関する諸因子　
　　2-3　マランゴニ効果
３．界面活性剤の種類による起泡
　　3-1　陰イオン性界面活性剤
　　3-2　非イオン性界面活性剤
４．消泡
　　4-1　泡による弊害
　　4-2　破泡と抑泡　
　　4-3　化学的消泡
　　4-4　Rossの消泡理論
　　4-5　消泡剤の種類と特徴
　　4-6　消泡剤の使用上の注意
［質疑応答・名刺交換］
