イベント ユーザー
event_id 8644
title OpenSolaris 静岡BootCamp！
catch Solarisのこれからと、OpenSolaris系OSを全部試してみよう！
description

9月のOracle Openworld以降、Solaris 11の姿が徐々にみえはじめてきています。
OpenSolarisの開発を経て深化したSolarisは、11で更に新たな機能拡張され、様々な仮想化技術や、最新のサーバ管理技術、WEB技術を備えた優れたOSになってきています。

OracleによるSun買収以後、Solarisの未来を不安視する人もいますが、Solarisは「かなり」元気です。

本セミナーでは、Solarisの今後と、Solarisを語る上で欠かせないチューニング技術、OpenSolarisプロジェクトの成果物として生まれた、様々なOpenSolaris系ディストリビューションについてお話しします。

期日

  • BootCampの部
    • 11月13日（土）22:00〜

  • BootCampの部
    • 駿河健康ランド
