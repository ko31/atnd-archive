第1回 MongoDB JP ＆ CouchDB JP 合同勉強会 in Tokyo

■MongoDB と CouchDB、一粒で二度おいしい勉強会

NoSQL・ドキュメント指向DBであるMongoDBがこの度、 MongoDB JPとして日本のコミュニティを立ち上げることになりました。その記念すべき第1回の勉強会はMongoDBと双璧をなすCouchDBの日本コミュニティ、CouchDB JPの皆様と合同で行う事になりました。

NoSQL・ドキュメント指向に興味のある方、MongoDB/CouchDBをこれから試されてみたい方、あるいはもうどっぷりつかっている方々、皆様のご参加と積極的な議論が展開されることを期待しております！

勉強会の後は会場での懇親会を予定しております。是非ともそちらもご参加下さい。

お手数ですがアンケートのご記入、宜しくお願いします。

■Ustream中継とハッシュタグについて

当日は gihyo.jp の方にご協力していただいてUstream中継を行います。チャンネルアドレスは
http://www.ustream.tv/channel/mongocouchjp
になります。後日勉強会の模様もgihyo.jpに掲載して頂けることになりました。

併せてハッシュタグですが #docorient に決定しました。これをツイートに付けて頂いて発言していただければ会場に声が届きます。これに併せて話題に合わせて #mongodb か #couchdb を添えて頂ければ最高です。宜しくお願いします。
