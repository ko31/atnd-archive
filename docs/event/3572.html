【テーマ】
「初めて学ぶmixiアプリ」

【概要】
現在注目を集めている「ソーシャルアプリ」。この流れに出遅れるのはエンジニアとして非常にまずい。でも仕事で関わる機会もないしなぁ…という焦りを打破するため、知識ゼロからmixiアプリについて学びます。

講師は@tsuyoshikawa。商用mixiアプリの開発 / リリースに携わった「経験者」から、実際のところどうなの？という実情を引き出します。

「万葉スタイル」を踏襲し、1時間の講義 + 1時間のハンズオンという構成です。

【事前準備】
mixiのアカウントが必要です。お持ちでない方はこちらから登録を済ませておいてください。

後半のハンズオンで実装を行いたい方は、ノートPCをお持ちください。

【参加費】
本編は参加無料です。終了後の懇親会は別途実費が必要です。

【懇親会】
プログラム終了後、そのまま会場をお借りしてビアバッシュ形式の懇親会を行います。会費は￥2,000程度を予定しています。参加ご希望の方は登録時のアンケートにご回答ください。

【ustream中継 / 録画について】
当日の模様はustream.tvにて中継・録画する予定です。顔出しNGの方には最大限配慮しますが、どうしても顔を出したくない場合はマスク / サングラスなどをご用意ください。
