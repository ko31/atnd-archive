Microsoft Azure 西日本リージョンにもDeployされたAzure上で動くHadoop ディストリビューション HDInsight。コア部分は Apache Hadoop をHortonwoks社と共同で Windows に移植した Hortonworks Data Platform で構成されています。そして遂にそのHortonworks社が日本に上陸。

本イベントは米国本社からHortonworks社のVPとAPAC Technical Directorをお招きしておこないます（通訳をご用意する予定です）。

Enterprise の世界でHadoopをいかに利用していくのか、他のHadoop ディストリビューションとの差別化は何か。Azure HDInsight を最大限に活用する方法とは。Power BIと掛け合わせることでどんな相乗効果がうまれるのか。など豪華なスピーカーと共にお送りいたします。

下記アジェンダです。

15:00 – 16:00 “Hadoop and Hortonworks. The next Generation of Hadoop.” By Jamie Engesser (Vice President Global Technical Services at Hortonworks)
16:00 – 17:00 "YARN Application Development – Present and Future”. By Jeff Markham (Technical Director, APAC, Hortonworks)
17:00 – 17:15 small break
17:15 – 18:30 Microsoft Azure / HDInsight のご紹介
18:30 – 19:00 質疑応答
