description

OSC 2010 関西@京都から毎年恒例となっております「一夜／二夜限りのギークハウス」が今年もやってきます。今回は8月1日(金)と2日(土)の2夜開催です。
オープンソース好きな人で夜を語り合いましょう。

  • トイレ・風呂・布団が用意された場所で、一泊 or 二泊できます。個室ではなく、2段ベット2台(計4床)もしくは雑魚寝になります。
  • 会場へのカンパとして1000〜2000円くらい頂きます (学生は500円程度に減免)
  • 食事代は参加者でワリ勘です (学生は半額程度に割引を予定)
  • 食べ物、飲み物持ち込み歓迎です。朝食をみんなで作りましょう。
  • OSC 関西@京都の会場(KRP)からは離れています。夜はタクシー分乗、朝は市バス202系統で移動する予定です (1人220円〜800円くらい)
  • 希望者が多いときはOSC 関西@京都の出展者の方を優先します。出展グループ名の記載をお願いします。
  • デフォルトで2泊としますが、1泊だけ希望の方は希望日を記載してください。
  • 場所が非常に分かりにくいです。カーナビを使う場合は「九条須原の1本北」で探してみてください。迷ったら主催者の Twitter アカウントへ
