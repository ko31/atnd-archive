電子工作と手芸でてれねこブローチをつくろう

description

電子工作フェスティバル会場にて、電子工作ワークショップを開催いたします。
当日参加も受付予定ですが、お好きなワークショップをお選びになった上で12月3日（月）までに参加登録をお願いします。

■ワークショップについて
「てれねこ」は照れたようにほっぺがじんわり染まる、ねこのブローチです。
実際の電子機器の作り方と同じはんだづけ方法「リフロー」を体験できます。
自分で縫ったぬいぐるみに電子回路を入れてかわいいブローチを作りましょう。

■ワークショップ概要
開催時間：13時までに会場にて受付して下さい。（所要時間：2～3時間）
会場：3331 Arts Chiyoda 2F 体育館（メイン会場）
対象：中学生～大人
定員：6名
参加費：1500円

■電子工作フェスティバルについて
生活、エンタメ、ビジネス、アートを支える技術である電子工作の文化を広めることと、才能あるクリエイターを発掘するために開催される、年に一度の作品発表会「電子工作コンテスト」のフィナーレを飾るイベント「電子工作フェスティバル」を今年も開催いたします！

電子工作フェスティバル2012では、電子工作コンテスト受賞作品のアワードの他、コンテストにご応募いただいた作品の中から優れた作品を紹介する展示や、応募いただいたクリエイターによるプレゼンテーションステージ、初心者の方にもお楽しみいただけるワークショップなど、電子工作の楽しさを満喫いただければと思います。

是非、お友達、ご家族をお誘い合わせの上ご来場ください！
