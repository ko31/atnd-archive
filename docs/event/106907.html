★講演内容

貴重なブロックチェーンの裏側、ブロックチェーンエンジニアリングの勉強会を
「ヒカ☆ラボ」と「TECH x GAME COLLEGE」の共同開催でお届けします！

【概要】
ゲームエンジニアに特化した勉強会「TECH x GAME COLLEGE」に、
話題のブロックチェーンゲーム「マイクリプトヒーローズ（MyCryptoHeroes、MCH）」を開発・運営している
double jump tokyo CEO兼CTOの上野広伸氏が登壇します！

世界一のブロックチェーンゲーム「マイクリプトヒーローズ」

ブロックチェーンゲーム「マイクリプトヒーローズ」は、Ethereumにおいて
DAU/Ether取引量/トランザクション数すべてで世界一を記録しました。

Ethereum上にブロックチェーンゲームを構築するためのアーキテクチャや、
web上でサービス展開する上で高速開発と高速稼働を両立するためのアーキテクチャを、
マイクリの事例を元に紹介いたします。

高速開発/高速稼働アーキテクチャについて

・マイクリで使用している gRPC-web のメリット
・protoファイルから生成される構造体ソースのメリット
・HTTP2によるAPI並列呼出について
・独自の工夫として、ProtoBufをそのままDB格納のメリットとテクニック

オンチェーン/オフチェーンのアーキテクチャ

・Ethereum と今まで通りのサーバを並行で稼働
・ブロックチェーンで所有情報だけ管理、
　ゲーム内動作はオフチェーンで稼働させるテクニック

★TECH x GAME COLLEGE とは

TECH x GAME COLLEGE（テック ゲーム カレッジ）は
「毎週渋谷でちょっといい人呼んでゲーム勉強会」をテーマに、
エンジニアのための活動しています。

※主催者は 株式会社テクロス です。

少人数制のmeet up

勉強会のこだわりは【少人数制】で、
登壇者と参加者がじっくりコミュニケーションをとり、
「ここだけの話」や「もっとマニアックに」情報交換ができるように工夫しています。
勉強会後のmeet upもぜひ楽しみにご参加ください。
