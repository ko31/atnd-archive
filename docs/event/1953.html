description

当日はGoogleの方が2.0(エクレア）端末を持って参加していただけるかも
しれません。自分のアプリのチェックや、例のアプリを見ることが
できるかもしれません。お楽しみに！


有限会社アドリブのkabayanと申します。（アカウントは違いますが、今回は人数が多そうなので別アカウントにしています）

いつも一人でコードを書いているのはちょっと寂しいので、androidのアプリ開発の小さな勉強会＆ワークショップをしています。

のべで100人以上の方々と勉強会をしてきたので、そろそろちょっと大きな場所でやってみたいと思ったので、クリークアンドリバーさん＆いつも勉強会に参加していただいている常連さんの協力の元、セミナールームをお借りして「秋の大勉強会」を開催することにしました！

勉強会で得たノウハウを元に、来た方それぞれのレベルで満足していただけるような内容と仕掛けを用意しています。

内容の一部をちょっと紹介します。

・SDK、Eclipsのインストール、セットアップ
・android Marketに出してみる
・admobとかadsenseで収益化する
・ウォームアップコーディング
・Intentをつかいこなす
・スレッド
・ネットワークとマッシュアップ
etc

（※全部やる訳じゃないですよ。念のため）
他に「こんなことやりたい」とか「ここで詰まってるから一緒にデバッグしてくれない？」とかのリクエストも募集中です。アンケートにお書きください。


参加に必要なもの：
①ノートPC:ここ３，４年以内に発売されたもので1024x600以上のディスプレイ、１Ｇ以上のメモリを持つもの。
②何かのプログラムを書いたことがあること。
③（重要）わからない人に教えてあげようという気持ち。


場所：クリークアンドリバーさんのセミナールーム(２Ｆです）
（地下鉄半蔵門駅から２分）

時間：18:00くらいから22:00ごろまで

参加費：３０００円(ビール、チューハイ１本つき）
飲み物、お菓子など持ち込み自由ですが、こちらでもある程度用意します。


＊連絡はgoogle group「Shibuyandroid」にも投稿します。
