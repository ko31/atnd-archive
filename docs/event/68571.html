Unity Japanの伊藤周さんを講師に迎えて、Unityの勉強会を行います。

Unityに興味のある方、是非ご参加下さい。

■参加に必要なもの
UnityがインストールされたPCをご持参ください。
Mac / Windowsのどちらかになります。

Unityのダウンロードはこちらから。
http://japan.unity3d.com/unity/download/

インストールの手順はこちらをご参考下さい。
http://techacademy.jp/magazine/1436

※インストールが終わったらアクティベーションも行って下さい。Unityが起動するところまでご確認下さい！

■スケジュール
13:00〜14:00 Unityの概要紹介
14:00〜17:00 初心者向けハンズオンセミナー（クマがハードルを超えるTemple Run的なゲームを作ります）
17:00〜18:00 質疑応答など

■懇親会
終了後に各自負担で懇親会を近くのお店で行う予定です。
こちらは勉強会を始める時に参加・不参加を確認します。
