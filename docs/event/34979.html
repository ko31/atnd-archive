description

社内システムをクラウド化する重要性は理解しているが、なかなか
踏み出せない。
そんな悩みを実例も交えて『IDCフロンティアの情報システム部』が
自らの経験をお話いたします。

クラウドやデータセンターを提案する立場から、今回は「利用する立場」の観点より
クラウド化のメリットや課題解決の道を探ります。

課題は現場担当者だけでは解決しにくいものですので、是非責任者の方もご参加ください。
また、SIerやMSPなどソリューションプロバイダーの方にとっても情報システム部の課題などを
共有することも出来ますので是非ご参加いただければと思います。
