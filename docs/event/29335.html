description

昨今、テクノロジー企業の海外進出が、話題になっております。中国、東南アジアなど成長市場に注目が集まりがちですが、欧州は成熟した5億人以上の消費者を抱える一大マーケットです。オランダ政府は、テクノロジー企業の欧州進出に様々な優遇政策を取っております。その中でも特にメリットが大きいのは、「イノベーション・ボックス」と呼ばれる優遇税制です。オランダで、独自に開発された知的財産（IP）などの無形資産から生じた純利益に対する法人税は、実効税率で５％になるというものです。IPを開発し、活用するテクノロジー系企業にとってオランダに拠点を構えるメリットは非常に大きいものといえます。また、オランダは欧州の中心地、英語を話す優秀な労働力など日本企業にとってメリットが多く、現在、約400社の日本企業がオランダを拠点に欧州ビジネスを展開しています。

イノベーションボックスを始めとするオランダの優遇税制・政策、テクノロジー企業が欧州進出する際にオランダを拠点とするメリットとIP戦略をご提案するセミナーを、5月31日(木)14時から、オランダ大使館／東京・芝公園にて、開催することになりました。メインスピーカーは、多数の著書や講演を通し日欧で活躍されているアーンスト・アンド・ヤング、パートナーの富永英樹氏です。
