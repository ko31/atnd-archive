イベント概要

mixi Hackathon in Kyotoは、
mixi Platformで提供される機能を利用したサービスやアプリケーションを作って頂くことを目的とし、アイデア出し～実装までを一気に集中して取り組むイベントとなります。

皆さんからどんな素敵なアイデアや成果が出るか、私たちも予想もつきません。
ぜひ斬新なアイデアを実装して形にしてください！

技術に自信がない、ハッカソンは初めてなので緊張する、という方もご安心ください。日頃mixi Platformの開発を行っているミクシィのエンジニアが、会場にて皆さまの開発を全力でサポートいたします。

mixi Hackathonにご参加いただき、共にmixi Platformの機能を存分に生かしたサービスやアプリケーションを開発してみませんか？　

また、Ideathonを開始する前には、mixi API SDK for Android™を使ったアプリ開発のライブコーディングや、mixi Graph APIの簡単な利用方法の解説なども行う予定です。
「○○みたいなことをしたいときは、どの機能を使えばいいですか？」などのご質問にも弊社エンジニアがお答えいたします。

mixi Platformの機能を使ったことがない方も、ぜひmixi Hackathon in Kyotoにご参加いただき、ソーシャルグラフを使ったアプリ開発を体感していただければと思います！
