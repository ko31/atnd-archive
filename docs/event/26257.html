catch どきどきキャンプ・佐藤満春の素顔が丸裸に！？
description

今やお笑い芸人としての顔だけでなく、作家の仕事や
大喜利イベントの実施、さらには趣味のトイレ関係での執筆の仕事など、
活躍の幅を広げている佐藤満春。

これまで、佐藤満春ひとり会などの個人でのイベントを行い、
トークやオリジナルの企画で「佐藤満春」の素顔を見せてきたが、
今回は、100にも及ぶ質問を佐藤満春にぶつけ、
これまでのイベント、更にはテレビやラジオ、ライブなどでは分からない部分、
佐藤満春の発想の仕方、お笑い芸人になる前の情報など、
ファンならずとも気になる「佐藤満春」の素顔や原点、考え方が分かる！？
イベントに来た人だけが知る事のできる必見のイベントです。

このイベントで、幅広い数多の質問とその回答から、
佐藤満春のプロフィールを作成し、佐藤満春とは何者か！？
を明らかにしていきます。
