78米2011　田植えの刻

title 78米2011　田植えの刻
catch 主催：NPO法人78会　協力：農業再生法人hototo
description

先日の5月4日、5日の土づくりは晴天にめぐまれ、最終的には30名の皆様に参加いただきました。
土づくりの次は、メインイベント「田植え」の開催です。6月18日、19日に開催！！
毎年、多くの方に参加頂いております！！
【お申込の方は、本フォームATNDのメール受信設定を必ずお願いします！！】
（掲載写真は昨年のものです。）

■タイムテーブル
6月18日（土）
10：10：電車でお越しの方：塩山駅集合（お迎えに上がります。）
10：50：お車の方：集合（現地）
11：00：昼食準備と田植え準備
12：00：昼食
13：00：田植え作業開始
夕　方：ご希望の方は、近隣の温泉へ　（ほったらかしの湯）、
　　　　夕食（BBQ）
　　　　※18日のみ参加の方は夕食後、解散（20:30終了予定）
夜　　：ご希望者には近隣温泉宿・、人数によってはロッジ等で、宿泊の手配をいたしたします。
　　　　4,000円～／1人（大人）
※別途、各自で宿泊先を探される方は、それでも問題はございません。　

6月19日（日）
10：00：田植え
12：00：昼食：ご参加頂く人数によってですが、お弁当または焼きそば（BBQ型）など
13：00：作業（田圃の作業が終了後、後片付け、）
15：00：ご希望者はまた温泉に。一休みの後、解散。

■費用（実費ご負担いただきます）※農業体験は無料です。
・交通費
【電車でお越しになる方】
　新宿駅発 8：30 → 塩山駅着 10:00 「あずさ7号」　指定3700円　自由3190円
　10：10分に中央線の塩山駅集合ください。（78米運営事務局がお迎えにあがります、）
【自家用車でお越しになる方】
車でお越しになれる方は、別途集合場所地図をお送りいたします。

【78会手配の車「あいのり」にてお越し頂く方】
3000円～5000円/1人（大人）※人数により変動いたします。
※78会あいのりカー希望の方は6月6日20:00までにお申込ください。

・食事代
【昼食】両日とも500円～1,000円程度
【夕食：18日】3,000円予定/1人（大人）
※小学生以下は無料です。

・宿泊費（78会での手配ご希望の方）
希望者には近隣温泉宿・、人数によってはロッジ等で、宿泊の手配をいたしたします。
4000円～／1人（大人）
※78会での宿泊先手配希望の方は6月6日20:00までにお申込ください。

■申込方法
①本申込フォームから登録ください。
【※ATNDのメール登録と受信設定を必ず行ってください。】
②運営事務局より、詳細の集合場所をメールにてご連絡いたします。
ご友人、ご家族の皆様をお誘いの上、ぜひご参加ください！！
皆様のご参加を心よりお待ちしております。
※定員になり次第、受付を終了します。予めご了承ください。

■ご準備いただくもの
★長靴　（田植え用を推奨します。参考：http://item.rakuten.co.jp/fight/10000818/）
・汚れても大丈夫な、動きやすい服装　　（両日参加の方は2日分）
・朝夕は冷え込む可能性がございますので、温かい服装やブランケットなど
・昼間は天気が良ければ気温が上がります。汗をふくタオルや日光対応のものを。
・雨が降る可能性もございますので、その準備を（雨具）
・日焼け止めや虫よけなど、気になられる方はご準備ください
・宿泊の方は、宿泊のご準備を
・農業と自然を楽しむ気持ちで、御参加ください

　保証するものではございません。
