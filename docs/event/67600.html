概要

Retty Tech Cafe とは、Retty株式会社が主催するエンジニア向け勉強会です。

第3回は、「スタートアップ×iOS開発」をテーマに開催します！
改築した社内イベントフロアでやります！

Gunosy、メルカリといった著名サービスのiOSアプリ開発に携わった方、または現在も開発をリードするiOSエンジニアの方々をお招きし、トークセッションをやります。また、RettyのiOSアプリ開発についても、可能な限りご紹介させていただきます。

スタートアップに興味あるiOSエンジニアの方、スタートアップで実際に働いているiOSエンジニアの方など、皆様のご参加をおまちしております！

登壇者紹介

トークセッション①

  • 出演
    • BEST10株式会社 村田佑介氏
  • 概要
    • SwiftとGPUImageを使った動画アプリMagicMovieの裏側
  • プロフィール
    • 京都大学経済学部卒業。大学時代にWEB制作会社のアルバイトでHTML/CSSマークアップやPHPプログラミングを学ぶ。 2010年に楽天株式会社へ入社し、1年間人事でエンジニアの新卒採用を経験した後、希望してエンジニアに転身。 約3年間楽天ツールバーの開発に携わり、2014年6月にCTOとしてBEST10に参画。アプリからインフラまでもりもり開発中。(http://muratayusuke.com)

トークセッション②

  • 出演
    • 株式会社Gunosy 川邉雄介氏
  • 概要
    • 今だからはじめるadaptive user interface
  • プロフィール
    • アメリカに憧れ渡米し、カリフォルニア大学デービス校材料工学科大学院卒業。その後ソフトウェアエンジニアに転身し、リクルートテクノロジーズにてiOSアプリ新機能開発に携わる。2015年Gunosyに参画。現在は主にiOSアプリ開発及び企画を担当。

トークセッション③

  • 出演
    • 株式会社メルカリ 大庭 慎一郎氏
  • 概要
    • ReactiveCocoaのゆるい紹介とメルカリでの活用事例
  • プロフィール
    • 小学生のときに与えられたMSXでプログラミングを始める。iOSアプリの開発は非公式SDKのリリースに衝撃を受けて勝手アプリからスタート。以降、公私でさまざまなアプリを手がける。2013年4月、株式会社メルカリ(旧コウゾウ)に入社。 著書に『入門 LEGO MINDSTORMS NXT 第2版 レゴブロックで作る動くロボット』など。

トークセッション④

  • 出演
    • Retty株式会社 石田憲幸
  • 概要
    • RettyアプリのWeb/Native連携
  • プロフィール
    • 京都大学大学院情報学研究科にて、異文化コラボレーションの研究に従事。卒業後は、楽天株式会社の楽天競馬（馬券販売サービス）のエンジニアとしてフロントエンドの開発・運用を担当。2014年11月にRettyにエンジニアとして入社し、12月からはiPhoneリニューアルプロジェクトにも参加。現在はiPhoneエンジニアとしてアプリの改修を行う。
