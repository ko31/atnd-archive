トップレベルの戦いを観に行こう。
予選参加者限定！ 決勝トーナメントの観戦、受付中

学生プログラマ日本一は誰の手に。
「第1回 CODE VS 決勝トーナメント」いよいよ開催！

★決勝は、予選終了時の学生ランキング上位8名で争います。
★対戦形式は、1対1で競い合うトーナメント形式。
★協賛各社のエンジニアも、観戦・応援に駆けつけます。
★学生同士、協賛企業エンジニアで交流できる懇親会も開催。
★予選参加者限定で、観戦予約を受付中。
★優勝賞金10万円。
★司会進行が、 Julie Wataiさん に決定！
★ ユーストでの中継も決定！http://www.ustream.tv/channel/mtlcafe/

日本最強の学生プログラマは誰なのか。
ハイレベルな戦いが予想されるCODE VS決勝戦を
あなたも観に来ませんか。

決勝戦の観戦だけでなく、
学生プログラマ同士や協賛企業のエンジニアたちと
交流できる懇親会も予定。

懇親会では、観戦者も自作CODEで他の参加者と戦えるよう準備中。
「ただ観るだけじゃつまらない」と思っているあなたも、ぜひぜひ参加してください。

【観戦資格】： CODE VS に予選参加した学生の方

※参加登録する際は、「参加コメント」の欄にCODE VSで使用している“screen name”を記入してください。
（［例］*****です。）“screen name”の記入がない場合、参加登録をキャンセルさせていただく可能性がございます。

※決勝へは、普段着でお越しください。
