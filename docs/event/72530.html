六本木ヒルズに行けないけど、一緒に祝いたい人達が勝手に集うイベントになります。
場所は名古屋の伏見駅から徒歩１０分のところです。

イベント概要

元のイベント Ubuntu 15.10リリース記念オフラインミーティング15.12 を名古屋でもやろうっ！というものです。

  • 元イベントのUstreamを流します。セッションを観ながらみんなでわいわいしましょう。
  • WiFi,電源があるのでハッカソンできます。
  • Ubuntuに関わらずOSS,FLOSSが好きな人等が集まれたらいいなと思っています。

テーマは「半年後のLTSに向けて」。2016年4月リリースのUbuntu 16.04 LTSは2年ぶりのLTSとなる予定です。同時に16.04は「普通のUbuntu」としては「最後のリリース」になるかもしれません。そこで今回のオフラインミーティングでは16.04をより良いものにするための場にしたいと考えています。15.10を使ってみて気づいたこと、16.04までに実現したいことなどを紹介できるといいな（願望）。

参加費・参加資格等

参加費は無料です。
Ubuntuを使ったことがある、あるいは興味がある場合など、どんな方でも来て頂いて問題ありません。

ただし、非公式なので、 無料での唐揚げ・飲料の提供はありません.
近くのスーパーやピザ屋さんで自力で調達する必要があります。
なので、５００～１０００円位を考えています。
非公式なので、そこは柔軟にやらせてください。
参加費設定が１円になってるのはその旨をご理解いただきたくっ・・・！
