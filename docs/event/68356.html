第７回のクラウドファンディング会議は
かつて日本一の支援者数をほこった童話モチーフコスメプロジェクトの仕掛け人を呼んでの
起案アイデアブレスト大会編！

普通の人でもできる資金集めの方法として注目を浴びつつあるクラウドファンディング。
とはいえ、目標金額まで集められるプロジェクトは半分くらい。

日本のクラウドファンディングでは、プロジェクトリリース後にプロジェクトの本文を修正することはできません。
もしもリリース前に、他の人に意見を聞いて、より伝わりやすい方法に変えることができたら。

参加者の実際に立ち上げたいプロジェクトを、参加者全員でブラッシュアップするブレスト大会です！

クラウドファンディングを実際に使って資金集めしたい人を超絶後押しします！(´▽｀)

進行はクラウドファンディング研究家およびライターとして活動中Ouma（オーマ）。
自身でもCAMPFIRE、Makuake、Motion Gallery、Indiegogoでプロジェクトリリース経験があり、現在きびだんごとKickstarterで起案準備中のOumaがクラウドファンディングについて、分かりやすく解説します。

さらにゲスト講師にはKickstarterで６件のプロジェクトリリース経験のある高嶋直之氏をお招きし、具体的なクラウドファンディングの設計についてお伝えいたします。
また、高嶋氏は一時期、日本最大の支援者数を集めた「童話モチーフコスメ」シリーズの仕掛け人でもあります。
クラウドファンディング成功の秘訣に迫ります！

ゲスト講師：高嶋直之氏

【経歴】
3DCGデザイナーとして、某コ○ミや
某SE○Aなど家庭用ゲームソフトを
開発している企業に勤める。
それから数年後、デジタルに飽き飽き
して医療器具製造の町工場職人へと転身する。

0.1mm単位の精度で物を作るように
なった頃、リーマンショックにより製造業が
大ピンチに！

転職に融通が利かない他の職人さん方の負担を
減らすべく、自ら転職を決める。

そして戻ってきたのがデジタルの世界、
時代は家庭用ゲーム機から
スマホゲームの時代へ。

某人気アプリを手がける企業へ転職した
ものの、やはり物足りないデジタルな日々。

気付けば求人サイトを開き、
面白い企業を探していた…その時！

「ロボット系の会社だよ」という変な求人を
発見、早速応募！そして入社！

担当した業務は電子工作ツールの開発。
アナログな設計に充実感を覚えつつ、
試行錯誤して作ったツールを
Kickstarterへ投稿。

すると3日目にして目標額を達成、
最終的には支援者1,000人以上と
大きな反響を得る。

その後1年間の間に6件をKickstarterへ投稿、
うち1件は惨敗、1件はキャンセル、
4件は成功を収める。

それらと平行して国内の
クラウドファンディングへも投稿し、
これらも成功を収める。

今ではすっかりクラウドファンディングの
常連となり、結構なノウハウも蓄積。
これからも個人でデザインした様々な
プロダクトをクラウドファンディングで
展開していく予定。
