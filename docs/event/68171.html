title TM（トランセンデンタル・メディテーション）無料説明会
catch
description
世界で６００万人がその効果を実感！！
【眠った潜在力を１００％引き出す】瞑想法のメカニクスを大公開！

「自分の本当にやりたいことで楽しく豊かに生きたい。
それができれば理想だけど、なかなか現実はね。。。」

というあなた！！

その現実を創りだしているのは、まさにあなた自身です。

わたしたちは、本来自分にしか実現できない、
発揮できない天才性・可能性を一人ひとりが必ず持っています。

にもかかわらず、

ワクワクしない楽しくない仕事に身を投じ、
たった一度しかない人生の一瞬一瞬を不満とあきらめで染めていく。。

もうそんな毎日はやめて、

本当の自分を思いっきり『楽しく』生きてみませんか？

いま、あなたが
仕事やお金、健康、人間関係やコミュニケーション、パートナーシップ・・・
などさまざまな悩みを抱えているとしたら、

その理由はたった一つ。

「あなた自身の本来の可能性が１００％発揮されていないから」

ほとんどすべての人がわずか３％の潜在力しか活用できていないと言われてます。
本来もっている可能性のごくわずかしか活かせていなければ、
当然すべてが不自然に展開し始めます。

わたしたちが１００％の可能性を生きられているとき、
それは本当の自分をまるごと生きている状態であり、
それこそ最も自分らしくパワフルで自然な状態。

このとき、努力せずにひとりでに成功と豊かさはやってきます。

逆にわずかの潜在力しか発揮できていないとき、
人は不安や恐れ、心配や妬み、焦燥感にかられ、
必要以上の気負いと緊張で肩に力が入り、不自然な選択をしてしまう。

そう、すべては

「あなた自身の本来の可能性が眠ったままになっているから」

なのです！

わたしたちが

本来の可能性を１００％発揮して生きるとき、

すべての恐れや不安、心配から解放され、
一瞬一瞬を自分らしく、
喜びとワクワクで思いっきり手を広げて
楽しむことができます。

その思いっきり広げた手で、

あらゆる豊かさをつかむのです。

そのときには、人を妬んだり、うらやんだり、人と比較して
自分をちっぽけだと卑下することもありません。

なぜなら、誰にも生きられない宇宙にたった一人の自分を、
思いっきり無邪気に生きている状態だからです。
あなたの人生はあなたにしか生きられません。
そして、

自分はいったい何のために生まれ、何をして生きたいのか、
本当のあなたはそれをちゃんと知っています。

宇宙にたったの一人の自分、
その自分を生きるためにこの世界に生まれてきたのですから。

ハリウッド女優や男優、世界のトップアーティスト、ビジネスリーダーが実践し、
WHO（世界保健機関）やハーバード大学をはじめ世界中の研究機関で、

「最も効果が高く、簡単で、全員に確実に、その効果が現れる」

と証明された瞑想法。

それが、

トランセンデンタル・メディテーション。

あの有名な「ザ・シークレット～引き寄せの法則」のなかでも紹介された、

「すべての引き寄せを生み出す統一場（無限の潜在力・可能性の源）」
に直接アクセスする瞑想テクニック

６００万人が体感したその驚異の瞑想法＝トランセンデンタル・メディテーションのメカニクスをたっぷり２時間半かけて
お伝えします！！
