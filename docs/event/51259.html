正確に伝えるための英文テクニカル・ライティング

イベント ユーザー
event_id 51259
title 正確に伝えるための英文テクニカル・ライティング
catch 科学・技術英文（論文・報告書等）を書くためのポイントを伝授！
description

趣旨
　外国人が違和感なく自然に読める科学・技術系の英文を書けるようになるためには、文法・語彙以外にも、英作文のルール、パラグラフィング作法など、知っておくべきことが多くあります。それらを1日でマスターすることは不可能ですが、本講座では、情報や意図を、折り目正しい英語で正確に相手に伝えることに焦点をあてて、より良い科学・技術（以下「技術」）英文を書くためのポイントを伝授致します。
　技術系の英語論文や報告書を書く研究者・技術者、日英翻訳者を目指す人、企業内英文文書作成担当の人、さらには外国人と英語でコミュニケーションをとる人にとって、有益な講座ですので、ふるって受講いただけますよう下記のとおりご案内申し上げます。

プログラム
1.導入
　1.1 技術英語の特徴
　1.2 テクニカル・ライティングとその要件
　1.3 日本語文と英文の構造的違い（コミュニケーション・スタイルの違い）
2. 文法のおさらい
　2.1 句読法
　2.2 名詞の可算/不可算
　2.3 冠詞
　2.4 前置詞
3. 単語/フレーズ・レベルの諸ルール
　3.1 明快で正確な単語/用語の使用（1 meaning/word 等）
　3.2 数量の扱い
4. 文レベルの諸ルール
　4.1 1 idea/sentence
　4.2 文頭の忌避事項
　4.3 その他のルール
　4.4 目的に合った文体の選択
　4.5 活用したい一歩進んだ構文
5. パラグラフィングの基本
　5.1 パラグラフとその要件（1 topic/sentence 等）
　5.2 首尾一貫性と結束性
　5.3 修辞と構成
6. 英文ライティングの学習法

●習得できる知識
・技術英語と一般英語の違い（技術英語の特徴）
・日本語文と英文の構造的違い（コミュニケーション・スタイルの違い）
・日本人によく見られる英語の初歩的ミスおよび和製英語
・品詞ごとの文法要点
・単語／フレーズ・レベルの諸ルール
・文レベルの諸ルール
・パラグラフィングの基本
