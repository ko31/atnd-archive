description

自分が入っている生命保険の内容を理解していますか？
保険の仕事をしていると、多くの方が自身の加入している保険の内容を理解していない現状に出くわします。

生命保険の効力が発揮されるのは病気やケガ・死亡など人生におけるマイナスの時期。
きちんと理解していないと“万一の時役に立たない”事態にもなりかねません。

『保険はよくわからない』『専門用語が多く難しい』という声をよく聞きますが、基本を押さえ、シンプルに考えればけっして難しくありませんよ！

セミナーでは生命保険を基本からわかり易く説明します。

・生命保険の仕組みを知りたい
・掛捨型と貯蓄型どちらがいいの？
・主契約と特約ってなに？
・どうして病歴を申告する必要があるの？
・保険の更新ってなに？
・どんな保障があるの？
・ムダの無い入り方は？
など

講師は銀行・保険会社・保険代理店で働いてきたファイナンシャルプランナー。
長年金融業界で働く中で『金融知識はあるかないかで大きな差が出る』ことを体感してきました。

生命保険も重要な金融知識の一つ。
基本を押さえておくと、きっと今後の人生のお役にたちますよ！

セミナーは少人数制。参加者のみなさんと雑談をしながらすすめます。
既に保険に入っている方、これから保険を検討している方、この機会に保険の理解を深めませんか？

セミナーでは金融商品の勧誘・販売は一切行いません。お気軽にご参加ください。
