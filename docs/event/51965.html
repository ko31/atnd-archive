description

日　時：　２０１４年６月２７日（火）　１５：００～１８：００

話題の「大資産家になるためのアジア副業マニュアル 100万円から実現できる人生改革」（PHP研究所）の著者澤木恒則登壇する注目のセミナー開催のお知らせです。

主催は、「チームＩＮＡ」と「JASEAN Co.,Ltd」。

チームＩＮＡ：事業コンサルタント、弁護士、弁理士が結集し中小企業のインド進出支援を行う任意団体

JASEAN Co.,Ltd：ASEAN各国での起業、投資を考える個人の方への啓蒙と進出後の成功までのサポート支援を活動の主旨とする法人

アジア進出に於いて重要なファクターとなる”国選び”。本セミナーでインドとASEANを比較検討することにより、「自社にとって最適な進出先はどの国か」を知ることができます。ぜひこの機会をアジア進出の基盤づくりにご活用ください。

『「インドの平野」と「ASEANの澤木」が失敗しないアジア進出のポイントを大公開！』

◆内　容：

○進出の失敗事例から学ぶこととは？
○日本企業にとって進出の弊害になっている現地の課題とは？
○商習慣や宗教の違いに関して気をつけるべきこととは？　等

第３部　　17：00～18：00
