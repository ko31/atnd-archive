(1) 正規の手順以外で投稿された方

  • YouTub や ファイル共有サービスを使いたく無い方、参加をご遠慮下さい。
  • ツイッターやLINEからの動画投稿は「不参加」として扱います。

(2) 途中で途切れている動画を投稿された方

  • 途切れていると、リレー動画に映像が入らない可能性があります。
  • その場合、「不参加」とさせて頂きます。

(3) 無断で締め切り以降に投稿された方

  • 期限が守れない方、参加をご遠慮下さい。

  • 再生位置を指定されたい方、参加をご遠慮ください。

■制作条件など

  • 現在プレイ可能な７５曲すべて募集致します。
  • 投稿頂いた動画も１票カウントしますので、必ず使用します。
  • 締め切りまでに投稿頂いた動画で「ダンエボ カウントダウンMovie」を制作します。

■参加方法

  1. 現在プレイ可能な７５曲（削除曲除く）いずれかを最大５曲ご投稿頂けます。
    　※お一人様１曲～５曲まで投稿可能です。
  2. 動画をYouTube 又は ファイル共有サイト (DropBoxなど) へアップロード下さい。
  3. 出来るだけ撮影した動画そのままご投稿願います。
    ※「KINEMASTER」で編集するとロゴが入るため
  4. 投稿動画は、2017年1月1日以降に撮影したものに限ります。
  5. 撮影場所は、特に問いません。
    ※ダンエボの前、室内、屋外、いずれの場所でもOK
  6. 下記の「必要事項」をアンケートにご記入下さい。

＜必要事項＞

  1. 動画に表示するお名前
    　※無記名でもOK
  2. 動画に表示するプレイ場所（ゲームセンター名、撮影場所など）
    　※無記名でもOK
  3. 動画のアドレス(2.でアップロードしたアドレス)
    　※必須

■編集した動画について

  1. 編集した動画はこちらのチャンネルで公開します。
  2. マスターデータが欲しい方は、ご一報ください。
  3. 制作した動画をYoutubeやニコニコ動画に転載されたい方は、ご一報下さい。
  4. 許可なく動画を転載する行為は禁止します。

■再生順／サムネイルについて

  1. 再生順は、私「たけぷ～」が決めます。

■注意事項

  1. 下記の場合、「不参加」として扱います。
    1. 正規の手順以外で投稿された方
    2. 途中で途切れている動画を投稿された方
    3. 無断で締め切り以降に投稿された方
    4. 再生位置を指定された方
  2. お一人様１回限り最大５本、投稿者がダンスしている動画のみ投稿頂けます。
  3. 既に動画を投稿された方が別の方の動画に映ってもＯＫです。ただし、意図的に何度も繰り返し登場する場合は、参加をご辞退頂く場合があります。
  4. 編集は、たけぷ～ が実施します。
  5. 投稿頂いた元の動画に対し、大幅な改編は行いません。
  6. 公開後、諸事情により編集した動画を削除する可能性があります。
