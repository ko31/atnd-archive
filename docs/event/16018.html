「PHPを使ったアジャイル開発」をテーマにした勉強会を6/16(木)に開催します。実際の現場でアジャイル開発をどう運用していけるのかを、手を動かしたり人から話を聞きながら考えられる場がほしいと思い、企画しました。

プログラム

以下のプログラムの中から、興味のある内容を選んでご参加下さい。

・PHPを使ったアジャイルの事例のLT
・オススメのアジャイル開発本の紹介や感想の共有をする読書会
・ 永和システムマネジメントの角谷さんにお越しいただいて アジャイル開発に関するＱ＆Ａ会
・PHPを使ったテストファーストの体験ができるミニワークショップ（資料を見ながら各自で手を動かして体験するため、ノートPCが必要となります）

※LTしても良いよ！という方がいらっしゃいましたら、@remoreかコメント欄までご連絡をお願いいたします！

【読書会の進め方】
アジャイル開発本を読んだ気づきをアウトプットし合う場を作ります。この場を通して、

  • 自分が読んだ感想をアウトプットできるので、より理解が深まったり
  • 読んだことのないスゴいアジャイル開発本に出会えたり
    できます。

【ミニワークショップの概要】
PHPのテスティングフレームワーク（多分SimpleTest）を使って、サンプルのMVCフレームワークのソースコードの上でどのようにテストコードを書くミニワークショップを行います。この場を通して、

  • ケント・ベックのテストコードの書き方が少しわかったり
  • PHPのテスティングフレームワークやJenkins(Hudson)とちょっと良い仲になれたり
    できます。

参考となる資料を読みながら適宜質問があれば受け付けるスタイルを想定しています。
なお、テストコードの書き方のチュートリアルは、ケント・ベックの テスト駆動開発入門 を参考に独自で作成したものを利用する予定です。
※TDDのエントリーレベルの方を対象としたチュートリアルとなります
