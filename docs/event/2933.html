description

・当勉強会は、文化庁の「文化力プロジェクト」「関西元気文化圏プロジェクト」の認定事業です。

　 “生活できる収入”の確保なしに、充実したライター生活は望めません。フリーランスにとって、仕事を確保するための「営業活動」、仕事量をこなすための「日程調整」、継続的な仕事を発注してもらうための「先方との関係づくり」を思案し工夫することは、「取材」「文筆」の技術を磨くこと同様に大切です。今回は、様々なジャンルの文筆、様々な発注者とのやり取りの経験を持つフリーライターの高橋順子氏が、効率的な仕事のやり方、発注者との関わり合い方など、充実したライター生活を送るためのコツについて講義します。
