台風がきたために、中止しました。

　
お酒を傾けながら、冒頭は、『CodeIgniter徹底入門アプリ』のコードについていい悪いこっちはこうしろという会です。お酒が進んだら、わいわい楽しく雑談する感じです。トモダチとか情報交換できる仲間ができるとすてきですね（あ、LTしたい方はLTもどうぞ）。ちょっと人数次第で調整入れようと思っておりますー。
　

趣旨１

コードレビューってしたことありますか？
フリーランスや中小企業だと、「動けばOK」「動いたら次だ次！」となりがちで、コードがきれいか、スパゲッティかなんて構ってられません。けど、それは負債となり、積もり積もって苦しめられるのは将来の自分です。まるで夏休みの宿題を後回しにしたかのように・・・。
そこで、フリーランスや中小企業メンバーで集まって、コードレビューをしてみませんか？？「おー、ここってこうやって書いたら行数が半分に減るのか」とかいろいろ気付きがあるかと思います！
　

趣旨２

先日、PHPフレームワークであるCodeigniterがver.3となってネックだったライセンス問題がなくなり
PHPカンファレンスのセッション をはじめとして各所で盛り上がりを見せています。盛り上がってるっていったら盛り上がっているのです。
そこで、 Codeigniter3 の入門アプリを教材にして、コードレビューをしつつCodeigniterについて学ぶ場ですー。
　

対象

PHPを利用したことがないと、おそらく会話についていくのは難しいかと思いますー。echo とか foreach とか function とか意味が分かれば大丈夫かと。初心者歓迎いたしますー！
