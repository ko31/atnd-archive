2012年も始まりました。
System Center User Group Japan は今年もがんばってまいりますので皆様よろしくお願いします。

さて、早速第3回勉強会を2/25(土)　日本マイクロソフト 品川オフィス セミナールームにて行います。
今回のテーマは「セキュアな（クライアント）運用！」です。

昨今またいろいろと騒がしいこともあり、何をどうしたものか悩まれている方も多いかと思います。今回はクライアントをセキュアに保つ、使ってもらうという視点で３セッションを行います。

【開催日】
2012年2月25日(土)

【場所】
日本マイクロソフト 品川オフィス　セミナールームB

【スケジュール】
13:00　受付開始
13:30～13:40　ご挨拶　
13:40～14:40 Session 1 – AD CS入門 ～暗号化の基礎～
14:40～14:55 休憩
14:55～15:55 Session 2 – 「Windows Thin PC」で本格的なシンクライアントを作る！
15:55～16:10 休憩
16:10～17:10 Session 3 – SCCMでセキュアなクライアント構成管理をしよう
17:10～17:20 質疑応答/休憩
17:20～17:35 ライトニングトーク
17：45 終了（予定）

懇親会は別途、会場近くで行います。

【参加費】
勉強会参加は無料です。
懇親会は別途、会費￥5000を予定しています。
懇親会は当日参加希望をお伺いしますので、ふるってご参加ください。
