イベント概要

2015 年 7 月 25 日に 京都リサーチパークにて Web Music ハッカソン#4 @京都 を開催することになりました。

Web Music ハッカソンとは、ブラウザで音声処理・信号処理を行う Web Audio API、外部 MIDI デバイスと連携をする Web MIDI API など、ブラウザの持つ音楽に関わる API を中心に使って、一日でアプリケーションを制作しよう、というイベントです。

１回目 、 2回目 、 3回目 は東京で開催されていたのですが、4回目はWeb Music Developers JP、GDG京都、GDG神戸で京都で開催することになりました。

なお、今までのハッカソンの 当日の様子 はこちらにも掲載されています。どうぞご覧下さい。

会費 ： 無料（ランチは各自でご用意ください）
主催 ： Web Music Developers JP、GDG京都、GDG神戸
協賛 ： AMEI （一般社団法人 音楽電子事業協会）
後援 ： 京都リサーチパーク株式会社
チューター： @aike1000 氏 、 g200kg 氏
*参加資格： Web Audio API、Web MIDI API に関するアプリケーション開発に関心をお持ちの方 （JavaScript を書いたことのある方）

よくあるご質問

Q: ハッカソンとは何ですか？
A: ハッカソンとは、新しい技術を試す 1 つのおもしろい方法です。通常チームで行われ、新しいフレームワーク、API、言語を試すことに焦点を置いたイベントです。各チームでは実験を行ったり、アプリケーションを作ったりします。イベントの最後にはお互いの成果を評価し、表彰されることもあります。

Q: 誰が参加できますか？
A: 経験豊富なソフトウェア開発者、ハードウェア開発者から、Web Music のアプリケーション開発に興味のある方を歓迎いたします。Web Audio API、Web MIDI API をハックする絶好の機会です。当日は楽器、ガジェットの提供を予定しておりますので、思う存分効率よくハックをしていただけます。

Q: チームメンバー全員の参加登録が必要ですか？
A: はい。参加者の人数把握と入館の手続き上、チームで参加される場合もチームメンバー全員の登録をお願いします。

Q: チームに入らないといけませんか？
A: いいえ、必ずしもチームに入る必要はありません。ただし、これまでのハッカソンをでは仲間が 1 人ないし 2 人はいた方がより多くのことを得られるようです。

Q: チームに入りたいのですが、知っている人がいません。どうすればよいですか？
A: 心配はご無用です。お申込時にアイデア、またはその欠片をご記入いただきます。その情報を元にしてハッカソンのキックオフ時にテーマの発表を行いますので、その場でご興味のあるテーマのチームに参加していただけます。
