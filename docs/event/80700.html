ウェブの相談　サイトの贅肉を落として お客が増えるしくみを作ろう

［個別相談］専門家と一緒にその場でサイト診断（1社30分）

専門家と一緒に、自社のWebサイトを点検してみませんか。
実際に今のサイトをみながら、より成果をあげるための施策をマンツーマンで一緒に見つけましょう。
全国でも100数人しかいない『ウェブ解析士マスター※』の資格所有者も参加します。

［スキルアップ講座］自分でホームページがつくれるサービスの使い方

Jimdoなどの、無料でホームページが作れるサービスの使い方を一緒にみてみましょう。パソコン画面を見ながら使い方をじっくり聞けるチャンスです。

［テクノロジー体験］会議中、触ると発想のヒントをくれる、「触るとしゃべる草！？」

会議がピンチになったら触ってください。
人間が触るとブレストのヒントを出してくれる、ものめずらしい植物です。
