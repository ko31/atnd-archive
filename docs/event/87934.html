ぺちぱな。

　ぺちぱな。（PHPer-na）は長崎のPHPを使っている人の意です、一応。別に長崎に限りませんがPHPを使っている人、興味がある人、使っていない人、興味がない人、とりあえず一緒にお話をしてお話を聞いて学んでいきませんか。とか言ってましたけど、今じゃもうめっきりすっきり関係なくなってきた気がします。うん、正直なところ関係ない。でも、好きですよ、We love PHP!

　と言う事を主旨にやっておりました。昨年再開した時には一回だけやってあとは知らん顔していたと思われても仕方のない有様だったのですけど、今年はちゃんと早いうちに二回目です。なんとかこのペースであと一、二回開催したいものでありますね。開催しますよ！

　さて今回は、以前「ぺちぱな。」でも取り上げたことがあったり、サブのお題目としてもちょいちょい登場していた Docker がメインです。Docker の概要、仕組みから始まり実際に使ってみます。講師的立ち位置にはキハラさんにお願いしました。

　PHPerの方も、そうでない方も、是非お気軽に遊びに来てください。待ってます！

dockerを使ってみんなでmastdonインスタンスを立てよう！

　当日ハンズオンに参加されたい方はノートパソコン等をご準備くださるとより楽しめます。また、ハンズオン環境として Docker を使いますので、Docker Toolbox や Docker for Mac 等それぞれの環境に合わせてインストールしてきていただけると助かります。一応、こちらでインストーラー等の準備も行いますので早めにお越しいただければ対応できます（Linux云々の方面は自前でなにとぞ……）。よろしくご協力ください。

会場

　長崎市立図書館新興善メモリアル会議室2（洋）。

　会場の場所が少しわかりにくいです。 長崎市立図書館 Nagasaki City Library 図書館について に図がありますが、正面入り口から入場後、1階図書室へ入らずそのまま右へ折れその奥にもある出口から一旦外に出て左へ少し進むと入り口があります。そこで靴をスリッパに履き替えて右奥の方に会議室の入り口があります。

　館内飲食禁止との事で食べ物の持ち込みはできませんが、水分補給のためのペットボトル入り飲料の持ち込みは可能です。
