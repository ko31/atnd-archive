インプレス＆ThinkITから出版されているMySQL関連書籍の著者が一堂に会したイベントを開催予定です。

現在インプレスからは4冊のMySQL関連書籍が出版されています。今回はこの書籍の著者を集め、それぞれの書籍に関するセミナーを開催いたします。

今回のイベントでは、インプレス様からのご厚意で 書籍のプレゼント をご用意いただける予定です。

インプレスから出版されている書籍について

  • できるPRO MySQL 赤井誠＆できるシリーズ編集部(著) 発売日: 2014/10/24
  • MySQL Cluster による高可用システム運用ガイド 山崎 由章(著) 発売日: 2016/3/25
  • MySQL即効クエリチューニング（Think IT Books） yoku0825(著) 発売日: 2016/11/29
  • やさしく学べるMySQL運用・管理入門【5.7対応】 山崎 由章(著)/梶山 隆輔(著) 発売日: 2016/12/15
