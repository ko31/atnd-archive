★講演内容

Appleから発表された新言語Swiftでさっそくアプリを作ってみましょう！
ハンズオン形式で一緒に手を動かして学んで行きたいと思います。

Swiftの基礎はもちろん、バージョンアップしたXcode6の使い方も
ポチポチ確認していきましょう。

これからiPhoneアプリ開発を始めたい方！
Objective-Cでは作っていたけどSwiftではまだ・・・な方！

ぜひお気軽にご参加ください!!

※当日開発を体験するにあたり、事前の環境準備をお願いします。
①Apple Developer Program登録（無料の方で大丈夫です。）
②Xcode6 betaのインストール
2014/8/30現在、beta6が下記よりダウンロードできます。

https://developer.apple.com/swift/resources/

※内容は変更になる可能性がございます。

七島　偉之氏（ななしま　ひでゆき）

茅ヶ崎在住エンジニア

2013年「野良エンジニア」として独立。

以降、①～③のサイクルを回しながら生きております。
①力をためます。
自宅に引きこもって開発！
（エンジニアはやっぱりこもって開発するのも吉ですよね！)

②ためた力を実践します。
外に出て地域の役に立つ！

立ち上げたりしてます。

「税金はどこへ行った？茅ヶ崎版」
http://chigasaki.spending.jp/

③役に立ったら、皆に伝えます。
いろんな人にITの素晴らしさを教える！
例えばスマホアプリ開発スクールであるRainbowAppsの横浜校と湘南校にて
iPhone, Androidアプリ開発を教えてたりします。
