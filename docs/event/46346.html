description

こんにちはＤＩＰＰの風間です！
あなたは海外でどんな衝撃・どんな気づきを感じましたか？
いつもの当たり前の生活に戻っていくうちに、帰国直後に感じていたフレッシュで明るい意識や、せっかく外で得た感覚を自然と忘れていませんか？

私は２カ国への留学や旅を通して改めて日本という国の中だけでの自分ではなく、世界の中での自分という認識をすることができました。
・海外で言葉が話せなくてうまくコミュニケーションを取れなかったときの悔しさ
・自分とは全く違う文化や習慣に触れたときのワクワク感
・良くも悪くもカルチャーショック！

今回はそんなみんなで集まっていくつかのワークを通じ、Into the GLOBALと題し、”世界の中での自分”という視点で、原点に帰り、自らを見つめなおしてみませんか？

【Shop information】
CROSSBORDERは、「ちょっと世界にでてみない？」メインコンセプトにしたお店です名前の由来は、国境を越えるという意味と自分の殻をやぶる。 という意味を掛け合わせて生まれました。今までやったことなかったことに挑戦したり 新しいことに１歩踏み出せるキッカケをCROSSBORDERから提供できたらと考えています。
世界のビールを始め、地名や世界観が感じられるカクテルがあります。また、南米パラグアイ直送のマテ茶や茶葉や作り方にこだわったチャイ、イランの高級ビールイスタークをはじめカフェドリンクも充実しています。お近くにいらした際はぜひお立ち寄りください。当店のスタッフは、世界一周経験者や留学、ボランティアなど海外経験者が多数います。お気軽に旅の相談してくださいね。

【大阪方面からのオススメの行き方】
淀屋橋、京橋駅から京阪線で終着の出町柳下車。
叡山電鉄にお乗換え後一乗寺駅下車。
【一乗寺駅からの行き方】
線路を渡らず、100均一ローソンさん方面に進み、スーパーを越え3本目を右へ。
（東大路通に出るよりも近道です。）
左側に公園、デイリーヤマザキ（コンビニ)さんがみえると真向かいのびし屋さん（ラーメン屋さん）側に
渡ってください。 くるっと左を向いて見上げると世界地図がみえるので
2FにあがるとCROSSBORDERがあります。

【滋賀方面からのオススメの行き方】
数種類あります。
１、地下鉄松ヶ崎駅で下車。徒歩２０分コース
２、山科駅地下鉄乗換え三条京阪下車　市バスコース
３、三条京阪よりタクシーコース

２、東大路通り沿い：「一乗寺高槻町」下車徒歩1分
　　　　　　　　　　「一乗寺北大丸町」下車
川端通り沿い：「高野玉岡町」下車徒歩５分
白川通沿い：「一乗寺下り松町」下車徒歩１０分
川端通、白川通り沿い下車のバスの方が本数多めです。
東大路通りは３０分〜1時間に1本しかないので注意してください。
どのバスもおよそ20~30分程度かかりますがバスの場合は川端通りを通るものをおススメします。

【バス停からの歩き方】
川端通り沿いでは、高野川沿いを北方向に歩いて頂きましたら、
右側にレンタルビデオのお店があります。そこを右に曲がり、
進むと右手に天天有(ラーメンやさん)があり正面にデイリーヤマザキがあります。その交差点を左に曲がると
オシャレな美容室がありそのお隣の2Fです。

３タクシーの場合
三条京阪駅周辺で東大路通りをまっすぐ進み、
「高野交差点を超え、デイリーヤマザキ（コンビニ）さんか 天天有さんの前で降ろしてください。」
と伝えてください。1台1200円弱。所要10分程度。
お友達数人で来られる際はタクシーが便利です。

【神戸、阪急沿線河原町駅から来られる方へ。】
バスで「一乗寺高槻町」「一乗寺北大丸町」「高野玉岡町」「一乗寺下り松町」目指してご来店ください。
滋賀方面からの２をご覧ください。

タクシーでは15分弱。1300円弱〜1400程度
お友達数人で来られる際はタクシーが便利です。

【立命館大学生のみ】
BKCからシャトルバスをご利用できます。
北大路で下車後(150円)、北大路バスターミナル(220円)からのお乗り換えが大変便利で経済的です。
