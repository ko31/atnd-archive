catch 通称パプオフ
description

＜概要＞
音ゲーマーが集まってゆるゆると麻雀を打つオフです。
ノーレートで一切の賭けなしです。

＜集合場所＞
小田急線町田駅（詳細は個別に送ります。）

＜持ち物＞
・学生の方は学生証

＜麻雀のルール＞
◆基本
・ありありの半荘戦
　喰い断、後付け有り
・25,000点持ち／30,000点返し
　オカはTOP総取り
・順位点（ウマ）なし。

・ダブロン、トリプルロンあり。
供託のリー棒のみ上家取り。他は通常通り全員に支払う。

・５本場から２ハン縛り。（偶然役でも可、ドラを除く。）

◆連荘について
・親のテンパイ・和了連荘。ノーテン・カラテン・形式聴牌可。
・途中流局は全て親の連荘扱い
（九種九牌、四風連打、四槓流れ）
・四家立直は続行。
・オーラス親の和了、テンパイ止めは１位のみ可。

◆役について
・役満の複合あり。
・純正九蓮、国士十三面、四暗刻単騎、大四喜はダブル役満。

・大四喜／大三元／大明カンのリンシャンツモ　にパオ（責任払い）を適用。
・国士無双のみ暗カンをチャンカンできる。
・人和は役満。
・流し満貫は満貫。
流局時のノーテン罰符の支払い処理を行わず、その代わりに流し満貫の点棒処理。
この時親がテンパイor流し満貫なら親の連荘。
・八連荘、大車輪（ピンズのみ）あり。

・赤ドラ計三枚入り（単体で役にはならない。）

他は雀龍門、および咲-saki-原作準拠
http://janryumon.plaync.jp/guide/rule_taikyoku.aspx

ノーレートですので細かい所はその場の裁量で。
楽しく打ちましょう。
