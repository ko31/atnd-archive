【パソコン初心者、プログラミング初心者歓迎】

■これから技術を身に付けて、プロのiPhoneアプリエンジニアとして活躍したい方のために、
池袋 iPhoneアプリ開発体験会を東京 東池袋にて開催いたします！

仕事終わりに飲みに行くのも大切ですが、将来のためにiPhoneアプリを作る技術を身に付けてみませんか？
仕事が終わってからも頑張る人を応援します！

【iPhoneアプリを作れるようになろう】

iPhoneが世の中に浸透し、時代はiPhoneのアプリが作れる人材を求めています
英語を学んだり資格を取ったり、人により努力するものは様々ですが、時間をかけて勉強するなら、
需要の高いものに時間をかけるべきです

iPhoneのアプリを作るスキル、特に仕事としてアプリを作れるスキルを求めている所は非常に多いです
自分で起業を考えているような人でもアプリの作り方の基礎くらいは知っておくのは役に立つ事でしょう

facebook創業者のザッカーバーグさんや、ライブドアの創業者の堀江さんもプログラミングの上級者です

【どんな方にもしっかりとわかりやすく、丁寧に指導致します】

体験会では他のスクールや勉強会のようにテキスト教材を見て大人数で勉強するようなスタイルではありません
企業向けのiPhoneアプリを作っている現役のエンジニアが、
1人1台ずつ、Macのパソコンをご用意して直接指導致します

今まで２００人以上の方に、わかりやすく丁寧にiPhoneアプリの作り方を教えています
仕事として必要になるものを中心に、基礎から実践レベルの内容まで一貫して教える事が出来ます

iPhoneのアプリを作るのは難しいイメージを持たれる方もおられると思いますが、どなたにでも出来るように丁寧にわかりやすく教えていきます

プログラミングが初めての方、iPhoneアプリを作るのが初めての方にも、
アプリが作れるようにしていきますので、どなたでも安心して来てください

【こんな方にオススメ】

・iPhoneアプリ開発が初めての方、これからiPhoneでアプリを作ってみたい方

・Macを使うのが初めての方、プログラミングが初めての方

・Macを持ってるから、せっかくなら何かに活かしたい

・新しいことに挑戦してみたい人、自分を成長させたいと思っている人

・iPhoneアプリが作れるようになると技術が身につきますので、手に職がつくので女性の方にもオススメ☆
（特に女性のエンジニアはとても少ないので、貴重な存在です)

・プログラムというものが初めてだけど、挑戦してみたいと思っている方
→プログラムは難しく考えなくても大丈夫です。元ライブドアの堀江さんもプログラムは自転車に乗るようなものだとおっしゃっています

・swift、objective-c どちらもやってみたい方

・iPhoneアプリを発注する担当者の方や、起業を考えていてiPhoneアプリの作り方の基礎を学んでみたい方
