REZ/MORIGIRL CAMPS'11

title REZ/MORIGIRL CAMPS'11
catch ～MORE CAMP! MORE SMILE!～
description

REZ/MORIGIRLで初めてのキャンプイベントを開催します。

芋煮会をして、遊んで、焚き火をしてキャンプを楽しむだけ。

キャンプをしたことがあるひともないひとも笑って過ごそう。

【はじめに】

　このイベントは、キャンプしてもよし、日帰りで芋煮会の参加でもよし、の楽しい催しです。

【参加登録について（注！）】

　このATNDで参加受付はしませんので、参加する場合は管理者あてにメール・ブログ・twitterにて9/5までご連絡ください。

【開催日】

　2011/9/17（土）〜 2011/9/18（日）

　• 小雨決行ですが、中止の場合は9/17 9:00までに、メール・ブログ・twitterでお知らせします。
　• 途中参加、途中帰宅も可能です。
　　ただし、キャンプ場のチェックインは、17:00までとなっていますので、17:00以降の参加はできません。

【場所】

　吹上高原キャンプ場　：　宮城県大崎市鳴子温泉鬼首字本宮原２３－８９

【設備】

　•　炊事場、水洗トイレがあります。
　•　お風呂については、10:00-18:00の間「すぱ鬼首の湯」が利用できます。
　•　キャンプ場はオートキャンプ場で、すべてフリーサイト（電源なし）になっています。

【参加費】

　ひとり：2,000円
　かぞく：3,000円（お子様が何人いても。ご夫婦でも。）
　• キャンプスタート時にお支払いいただきます。
　• 途中参加、帰宅の場合も金額は変わりません。
　• この参加費は、昼食の芋煮会の会費です。
　• キャンプサイト料、お風呂代等は、申し訳ありませんが各自お支払いください。

【日程】

　9/17（土）
　12:00 集合、設営
　12:30 はじまりの会、芋煮会準備、昼食
　14:00 遊び（フリスビー、シャボン玉、だるまさんがころんだ他）
　17:00 夕食準備、夕食、焚き火（マイムマイム）

　〜各自就寝〜

　9/18（日）
　7:00 起床、朝食準備、朝食
　10:00 片付け、終わりの会
　11:00 解散（家に帰るまでがキャンプ！）

【集合、アクセスについて】

　• 直接現地に向かわれる場合は、集合時間前後にお越しください。
　• 場所がわからない、到着できるか不安だという場合は、管理者へご連絡ください。
　• 公共交通機関（陸羽東線「鳴子温泉駅」から市営バス）の利用を考えている方へ→
　　利用本数が限られていますので、各自確認の上ご参加ください。
　　アクセスがあまり良くないかと思いますので、車での乗り合いをお勧めします。
　• 途中参加、帰宅される方は、交通手段を確保の上お越しください。

【持ち物について】

　• 宿泊される方はテントを必ず用意してください。
　　その他、毛布、シュラフ（寝袋）、マット、パジャマ、洗面用具などは各自お持ちください。
　• 使い捨て食器、箸、コップ等は、用意しません！マイ箸・マイ食器・マイカップを必ず用意してください！
　• 雨具、着替え、タオル、常備薬、虫除けなど各自お持ちください。
　• 会場は山間部にあります。必ず防寒具もお持ちください。
　• だいたいの調理器具、テーブル、ありったけのチェア、ランタンなどは管理者が準備します。

【その他】

　• ゴミは所定の場所へ分別して捨ててください。
　• 他の参加者や利用客に迷惑をかけないよう、節度も持って楽しみましょう。
　• 音響機械の使用は禁止されていませんが、優しいボリュームで楽しみましょう。

【禁止事項】

　• 発電機の使用、直火、花火はこのキャンプ場では禁止されています。
　• 飲酒運転は厳禁です。
　

【お問い合わせ】

　管理者なみおまで！
