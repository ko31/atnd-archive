12/13（日）、Progression開発者の阿部貴弘さんをお招きして、社内勉強会を開催します。
せっかくの機会なので、席の数はわずかですし内容も少し高度な内容となりますが、一般の方にもご参加頂ければと思います。うちの会社からは8人ほど参加させて頂きます。
地図はこちらのPDFをご覧ください。
地図PDF：http://www.1-10.com/maps/kyoto.pdf
[追記 12/12] 懇親会の場所を追加しました。

予定している勉強会の形式は、

・自身の使ったProgression作品の発表（仕事・個人、いづれも可）し、その作品で挑戦したこと・疑問点を発表
・有識者が質問に応答
・阿部さんによるProgression4の簡単なご紹介

です。

また、勉強会を終えましたら、近場で懇親会も行う予定です。
予算は3500～4000円ほどです。
勉強会不参加の懇親会のみの参加も歓迎しています。

なお、会場の設備としては、
　○ノートマシンをぎりぎり設置するスペース
　○その電源
　×有線および無線Lan環境
になります。
オフラインでの勉強会となりますので、ご留意ください。
