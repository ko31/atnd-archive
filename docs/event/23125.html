2012年1月13日開催！cloudpack Night #1

Amazonが提供するパブリッククラウドサービス、Amazon Web Services（AWS）を実運用している各社自慢のTipsやノウハウをお互いに自慢し合い、さらにより良いアイデアを求めてディスカッションする場になればと企画しました。
勉強会形式で2時間程度AWSについて語り合った後は、より密な交流を兼ねて田町駅周辺で親睦会を予定しています。

豪華なゲスト陣が来場！人数限定の小規模なAWSの勉強会＆懇親会

ゲストとしてAWSからエバンジェリストの玉川様をはじめ、ソリューションアーキテクトの皆様（荒木様、大谷様、片山様、松尾様と4名が勢揃いします！）、さらにAWSの忍者、松井様にもご参加いただく予定です。

懇親会は話題の男が盛り上げます

東京のJAWS-UG勉強会よりもなぜか地方の勉強会への出席率が高い話題の男 @shouhei が、21時半から始まる真のcloudpack Night（懇親会）を仕切ります。
