技術ひろば.net 2011年12月 勉強会

以下の通り「技術ひろば.net 2011年12月 勉強会」を開催します。
「技術を楽しむ、技術でつながる」の理念の元、人と技術を大切にする活動を行っています。
どなたでもお気軽にご参加ください。

開催概要

■日時 : 2011年12月3日(土) 13:00～17:45 (受付 12:30～12:55)
■会場 : 日本マイクロソフト品川本社（SGT）31F セミナールームA
　http://www.microsoft.com/ja-jp/mscorp/branch/sgt.aspx
■会費 : 無料
■定員 : 30名

内容

2011年最後の開催となる 12月の技術ひろば.net は、2011年を振り返り、2012年を考える回です。
マイクロソフトの技術、製品にとっても、社会的にも大きな出来事のあった2011年、スピーカーを招いて、開発系、インフラ系それぞれについて2011年を一緒に振り返りたいと思います。
また 2011年1月の勉強会 で参加者が書いた寄せ書きを見ながら、1月の勉強会に参加した人もしなかった人も自分自身の2011年を振り返ってみましょう。

お茶会（フリートーク）も開催します。「懇親会（＝飲み会）まで付き合うのは時間的、金銭的にちょっと」という方も、おしゃべりや情報交換をしましょう。
12月ということでちょっとしたプレゼントももらえるかも？

  • 「ASP.NET Form認証 自由自在」 （小野修司さん/どっとねっとふぁん）
    ASP.NETは標準でForm認証のための仕組みを持っています。
    ただそれをカスタマイズしようと思うとなかなか難しいと感じるかと。
    この機会に、基本の仕組み、IIS 7.xでの設定、WebForm/ASP.NET MVC/WebPagesそれぞれの動作、簡単な独自認証の組み込み方といった内容をまとめてお話したいと思います。
  • 「2011年の振り返り～開発編」 （MS 新村さん）
    2011年、ソフトウェア開発の世界は大きな変革への入り口に立ちました。
    この大きな変革を読み解いていくには、提供されているテクノロジーのコンセプトを理解するのが非常に重要です。
    本セッションではこの1年間のマイクロソフトのテクノロジーの動きをベースに、2011年ソフトウェア開発の世界のトレンドを振り返ってみたいと思います。
  • 「2011年の振り返り～インフラ編」 （MS 高添さん）
    サーバー仮想化もVDIもクラウドも、特別なものではなくなりました。
    ただ、単なるビジネスモデルとしてではなく、テクノロジーや製品として今もすごく進化しています。
    このセッションでは、自分の1年間の活動を振り返りながら、来年への備えのヒントをお伝えできればと思ってます。
  • お茶会
    お茶でも飲みながら、自由におしゃべりや情報交換をしましょう。
    各自お好みのノンアルコール飲料を持ってきてください。
    （アルコールはお茶会の後で）
