４ＳＥＥＮ 人気企画のピザ作り合コンです。ペアまたはグループでピザを作って頂き本格窯で焼き上げます。お家では味わえない窯焼きピザをお楽しみいただきます。共同 作業でお話のきっかけが作れると人気のお料理合コ ン。お料理が出来ない方でも気軽にご参加頂けるお料理内容になっております。
ピザのメニューも毎回違いますので講師でもあるBarcafe店長のオススメ・オリジナル、メニューにはないピザを楽しめる場合もあります☆お店はアクセスに便利な 飯田橋駅から徒歩１分！(恋愛の神様・東京大神宮もすぐ近く＆神楽坂も目の前ですのでお帰りの際に散歩されてみては？）オシャレな店内でゆっく りとお楽しみ頂けるように人数も少なめに設定しております。ピザの他にもおすすめサラダやデザートまで充実しております。キッ チンスタジオで行うお料理合コンとは違ってお店の本格的な美味しいピザが楽しめます。
