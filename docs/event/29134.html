description

デッキ成長型カードゲーム「ハートオブクラウン」のプレイを主とした交流会を開催します。

ハートオブクラウンに興味があるけどプレイ環境がない方や、デッキ成長型カードゲームのベテランの方も、
様々なプレイヤーとの交流を通じて、良い時間を過ごせればと思い企画しました。
関西でハートオブクラウンをプレイしている皆様の参加を是非お待ちしています！
当日は、ちょっとした催しも企画していきたいと思います。
