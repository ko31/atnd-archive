description 【株式会社Eight Good　第2回ワークショップ開催のお知らせ】

初めまして株式会社EightGoodの本間です。

8月17日に開催した第1回ワークショップでは直前の告知にも関わらず15人の方にご参加頂きました。
世間でのフードロスという社会課題に対する関心も徐々に高まっており、11月15日にはファミリーマートで食品廃棄率改善に向けた加盟店支援制度を開始すると発表されております。

私たちはより一層フードロスという社会課題の周知とその削減への意識を持ってもらうために第2回ワークショップを開催することにしました。
フードロスという難しい問題を一緒に解決する方法を考えてみませんか？

各々の些細な行動が社会問題の解決に繋がります！
是非、皆さんのご参加をお待ちしております！

学生の方：1000円
大人の方：1500円
※飲み物と軽食をご用意しております。

運が良ければ特典も付いてきますので、奮ってご参加ください！
