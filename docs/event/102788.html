【街の魅力をみんなで磨こう！地域にもっとフルスイング！】

2018年12月20日(木)に名古屋にて、第6回 地域密着のアイデアソンイベント「さぶみっと！ヨクスル in 名古屋」を開催します。

地域でこんなことをやりたい！盛り上げたい！というプレゼンに対して、みんなでワイワイとアイデアを出し合います。磨けば光る地域の魅力。地域をもっとおもしろくしましょう！地域にもっとフルスイングしようぜ！

プレゼン内容

テクノロジーで社会課題を解決したい！
・Code for AICHI　嬉野 剛士さん

　Code for AICHI 代表の嬉野 剛士です。
　日本は「課題先進国」で世界中が将来直面するいろんな課題を真っ先に体験する国です。日本が抱える課題を解決できれば世界を救うことができると思います。それを実現するにはテクノロジーの力が不可欠だと考えます。
　自分たちの課題を、自分たちで簡単に解決できる明るい未来のために、実際の社会課題に対してどのような解決策があるかを、Code for AICHIと共に一緒に考えてほしい！

東北支援を愛知で盛り上げよう！！
・名古屋工業大学大学院　大原 一哲さん

　東北支援をしようと思ったきっかけは震災の翌年に福島県会津若松市を訪れた際に見た、風評被害により閑散としたまちの様子や仮設住居が建ち並ぶ光景にショックを受け、なにか東北のために力になりたいと思ったからです。

　東北について語り合う、というような場をこの地域の中で私のような若者が中心となって作っていくことによって、幅広い世代が集まるコミュニティが形成されると思います。また、東海地方にいながらも東北をもっと身近に感じてもらう機会を創出することにより、東北地方の関係人口を増加させてお互いに行き来する関係を築きあげていくことでどちらの地域も盛り上がっていくと思います。

　愛知で東北のファンを増やすための企画、より多くの人に知ってもらうための方法。そして、幅広い世代の人がそこに参加できるようにするための方法を一緒に考えませんか？

次の100年も東海地域が日本を牽引できる地域にしたい！
・自動車メーカー勤務／ONE JAPAN Tokai　土井 雄介さん

　金、人、モノ、技術など、どれも国内で高いレベルのものが東海地域にはあるにも関わらず、「学ぶ」「出会う」「創造する」機会が非常に少なく、ポテンシャルを活かしきれていない。

　そんな東海地域に、やりたい人がやりたい時にやりたいことができる環境が作りたい！一人一人が毎日を楽しく生きれるようにしたい！そして今までの100年間自動車産業で地域を潤し、産業を作っていったのと同様に、新しい「産業」「魅力」「経済圏」を東海地域で作れるようになり、「金」「人」「モノ」「技術」がさらに集まるようにしたい！

　その実現のために僕と一緒に考えましょう！

夫婦間の「共感」を育むライフサポートアプリを創りたい！
・ネクトラ / musubi　中川 裕太さん

　昨今、DUALライフな夫婦は増えてきています。また愛知県は製造の街であり、日勤勤務者だけでなく2交替や３交替従事者も数多くいます。

　一緒に過ごせる時間が少ない。その事でコミニケーションの悩みを抱えている人もいます。自分達の時間を減らす方法ではなく、共通の体験を産み、理想の「ファミリー・ビルディング」が実現するアプリを開発したい。そして労働時間形態が多様性な愛知県全体の活性化に繋がっていきたいと考えています。

　共感とはどのように育まれるか？一緒に考えていきましょう！
