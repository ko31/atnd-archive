「楽器弾きながら花見をやりましょうin盛岡」が好評だったので早速第2回をやろうとおもいます。

このイベントについて

  • 現地（前回やった場所=岩手公園の一番高台にあたる場所、つまり天守閣の跡地？って言えばいいんだろうか）に直接集合して下さい。時間になったらぼちぼち始めると思います。
  • 【重要】このイベントは音楽フェスやライブイベントではなく、あくまでもピクニックがてらみんなで音楽を演奏して楽しもう、というものです。
  • また、「いしがきミュージックフェスティバル」とは一切関係がありません。お間違えのないようにご注意下さい。

お願い事項など

  • 屋外なので雨天中止にします。中止判断は開催日当日行い、このページでご案内します。
  • ピクニックなので食料は各自調達されたし。飲みものも同様です。ブルーシートは6帖モノを1枚持っていきますが、必要に応じて椅子やシートなど各自用意してください。追記：【2010.6.17】タープ持っている人がいましたらご持参お願いいたします。
  • 使用楽器については前回同様基本的にはアコースティック楽器または電池駆動式のエレクトリック楽器でお願いします。

    • 要望がありましたので今回は100Vの発電機の使用を検討しています。どなたか持ってきていただければ有り難いです。燃料代や経費は使った人でワリカンしようと思います。【追記：2010.6.17】発電機は今回はなさそうです。すみませんが100Vを使う機器は使えません、ご了承ください。


  • 上記したように今回はAC電源を用意しようとおもいますがあくまでもピクニックなのでライブ用のPAは使いません。基本生音でお考え下さい。

  • 楽器もってない人も参加OKですが、なにかしら持ってきた方が楽しいかもしれません！

  • 音量は近隣のメイワクにならない程度で各自コントロール願います

  • 前回、やる曲がある程度決まっていると参加しやすいかな？と思いましたので、ぜひ「やりたい曲」をコメント欄などに書いて下さい。譜面や音源の入手先などもあるといいかもしれません　※ただし著作権法に抵触するようなデータへのリンクはやめてね！

  • 集まっている場所はTwitterでpostします。ついったー→ @gonoverdrive

  • Twitterのハッシュタグは[#ishiusu2]を使っていただくと便利です。
