ドキッ！開発者だらけのミニ四駆大会！！（その２）

〜ゴメンナサイ、人数が集まらなかったので今回は中止です〜

みんな、ミニ四駆は大好きかい！？
子供の頃爆走していた君も大人になってしまっただろう！
子供の頃を懐かしみながらミニ四駆で盛り上がろうぜ！！

あ、もちろん開いてる時間はみんな観戦するかもくもくしようね。

参加費
3000円を徴収します。余った分はその後に近くの居酒屋で開かれる反省会で使われる予定です
参加者が少なくて費用を賄えそうに無い場合は中止になります

  • イベント会場を借りる都合上、参加費が発生します
  • タミヤのミニ四駆コースレンタルサービスを利用します。
  • 当日は２コース分借りて連結して使う予定です。
    • http://www.tamiya.com/japan/download/pdf/mini4wd/adv/circuits_layout.pdf
    • スピードコース(1)か、テクニカルコース(3)を設営予定
  • 14:00〜14:30は設営、17:00〜17:30は撤収作業の時間です

参加資格

  • 必須
    • 金銭のやり取りが発生するため、念のため当日名刺（or 身分証明証）必須
  • いずれかを満たす
    • 主催者の誰かとtwitter上にて相互フォローしている
    • もしくは技術畑の人（AndroidやiOSやサーバーと日夜戦ってる人とか）

大会内容

コースはレンタル品です！コースを破損するような改造は絶対に行わないでください

  • 事前に組み立ててきても、当日会場で組み立ててもかまいません

対戦方法

今のところこんな感じで考えてる。
対戦方法は随時募集。現地で提案もあり。

  • 3人ごとのトーナメント戦
    1. 出走ごとのセッティング変更は自由（純情の部は予算を守ろうね☆）
    2. 電池の買い足し／交換も自由
    3. コースを6周させる（３レーン×２周ずつ）
    4. 全員がコースアウトした場合、もしくは決勝戦に限り再戦

あの頃の純情をもう一度！の部

  1. 予算制限 3000円
    • 値段は実売価格をベースにする。ただし、値段がわからない場合はAmazon 価格、AmazonのDBにないパーツは定価で換算
    • 限定品（無料配布等）は予算に含めない
    • 電池をまとめ買いした場合は単価で計算
    • 搭載しているパーツ・本体・電池の合計価格が予算内以下ならOK
    • 予備パーツは予算に含まない
    • 出走時の装備が予算内であれば、予備パーツとのセッティング変更自由
    • 自主規制なので、あんまり細かいことは気にしない
    • つまりは「子供の頃お小遣い握りしめておもちゃ屋に向かった気持ちに戻ろうぜ！」
    • ドレスアップパーツは予算に含めない（ステッカー系、塗装系、コックピットをぶちぬいてねんどろいど搭載系）
  2. モーターは現在のタミヤ公式戦ルールで使用できるモーターに限る
    http://www.tamiya.com/japan/cms/serviceregulation/410-regulation.html
    • ノーマルモーター
    • レブチューンモーター
    • トルクチューンモーター
    • アトミックチューンモーター
    • スプリントダッシュモーター
    • パワーダッシュモーター
    • ハイパーダッシュ２モーター
    • キット付属のダブルシャフトモーター
    • ミニ四駆PROシリーズ用のモーター
  3. 利用可能なマシン・モーター・パーツはタミヤ公式に限る
  4. 対戦相手のミニ四駆を壊しちゃったらちゃんとゴメンナサイすること
  5. もちろん俺達は大人だぜ！の部に出てもOK

俺達は大人だぜ！の部

  1. 予算無制限。大人だもん。
  2. モーターはタミヤ公式で無改造であれば制限なし。プラズマダッシュでもウルトラダッシュでもノーマルモーターでもいけるよ。大人だもん。
  3. 相手のミニ四駆に空気砲を当てたり、空気の刃で切り裂いたり、踏みつけたり、針で刺したりしたら失格。大人だもん。
  4. 対戦相手を買収して不戦勝可。大人だもん。

ギーク！の部

  1. コースを壊さなければなんでもあり
  2. 会場を壊さなければなんでもあり
  3. ミニ四駆にAndroidでもiOSでもChrome OSでも搭載するが良い
  4. 参加者が１名以上いたら開催します
