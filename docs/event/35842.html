1, 学生ヨル会議って？

私たち学生ヨル会議とは、学生が夜に寄り集まって話し合う発信型のプラットフォームです。
2012年の1月1日にNHKで放送された「ニッポンのジレンマ」を観た大学生たちがソーシャルメディア上で集まり、京都でこの学生ヨル会議が生まれました。
地域のコミュニティスペース等をお借りして、様々なテーマについて議論し、その様子をUstreamで全国に放送。地域間、世代間を超えた情報共有し、何か新たな気づきを生み出すこと目指しています。

1-2, 医療chについて

医療chでは、若者もオトナ、どちらの視点からも様々な角度から「医療問題」を考えます。
リアルな会場参加者による議論と特別ゲストによるクロストークを組み合わせた"新しいカタチのUstream番組放送"です。

ゲスト（今回のUstream出演論者）
田中伶奈さん（学生）
【身体障害者と社会との差を埋めたい】
立命館大学産業社会学部人間福祉専攻1回生。身体障がい者1級を持つ。自らの闘病生活を活かし、新しい支援システムを構築したい。入院によって周囲差を感じ、学校への不安が生じる。それは最悪不登校をも招く。これを回避する為、入院中はメンタルフレンドや家庭教師の派遣、退院後は補講を実施し、不安を取り除くサポートを行う。学校へ無理なく復帰できる様に子供と社会のクッションになれるような団体を目指す。
野口愛さん（社会人）
【全ての子どもたちに生きる力と希望を与える教育を】
自主自立、世代間交流をテーマに、学習教室を運営。子供を中心に、保護者や学生、社会人、そこに集う全ての人が共に育つ「共育」の場作りを目指す。 日本の将来を担う若者が社会人と共に、ビジネス研究・実践PJを行う場を創出する「アンビジョン」設立メンバー。PJの一つとして、長期入院（病院）から社会（学校）への橋渡しを支援するフリースクールの設立を企画。
