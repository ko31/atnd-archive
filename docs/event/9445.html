『 キャリアバイト 』がプロデュースする、ベンチャー社長と学生との交流会！

『将来に役立つ』『成長できる』アルバイトがしたい！
短期や座学ではなく『長期』『実践型』のインターンシップがしたい！

そんな人のための『キャリアバイト』が、
成長意欲の高い大学生を求める起業家3人と学生20人の交流会を実施します！
　
『キャリアバイトサミット』には、
社長と密にコミュニケーションがとれるコンテンツが盛りだくさんです！
　・社長を囲ってのテーブルトーク
　・社長と自由に話せる交流会
　・社長とお酒が飲める２次会
　
こんな方におススメです！
　・「ベンチャー」に興味がある
　・将来は起業したい
　・長期インターンシップをやってみたい

その場で、インターンシップ先を決めることも可能です！

　
本イベントに関するご質問は　tada@i-tank.jp　まで

内容

◆テーブルトーク
社長１人・学生６人でのテーブルトークを
グループとお題を変えて計３回やります。

（お題予定）
・起業の経緯
・学生時代の過ごし方
・今の事業内容や将来のビジョン
　
◆交流会
社長・学生・キャリアバイトスタッフ
全員が自由に交流できる場です。
　
◆居酒屋での二次会
場所を居酒屋に移動し、
お酒を飲みながらゆっくりと語り合いましょう。
