たった数時間でウェブデザイナーでもWebGLを扱えるようになると話題のJavaScriptライブラリ『jThree』を学ぶハンズオン勉強会です。
この5時間だけで信じられないほど自由自在にWebGLを使いこなせるようになります。

タグで描く。jQueryで動く。世界初インラインWebGLライブラリ『jThree v2』
http://jthree.jp/

この勉強会では主に「WebGL Editor」を使って進行します。
jThreeの機能をフルサポートしていてPCと対応ブラウザさえあればすぐにWebGLを動かすことが出来ます。
事前の準備や環境構築が一切不要なので安心してお越しください。

ブラウザで作って即公開！MMDモデルも動く登録不要の『WebGL Editor』
http://editor.jthree.jp/

以下は過去の勉強会でWebGL初体験の参加者が実際に5時間で作成し公開した作品です。
「world-moshisora」
http://editor.jthree.jp/?id=KcYnng
作者：もしさん https://twitter.com/moshisora990

参加条件はただ１つ、「jQueryが好きである」ということです。
jQueryが好きで最低限、
$( “div” ).animate( { width, “500px” }, 1000 );
の意味さえわかれば技量に関係なくjThreeでWebGLを動かせるようになります。

ただし、「jQueryは使えるけど好きじゃない人」や「実行速度を重視する人」には向かない内容ですのでご注意ください。
ある程度実行速度を犠牲にする代わりにWebGLの操作を全てjQueryでできるようにしたのが『jThree』なのです。

当日のタイムスケジュールは後日公開します。
iOS8が対応して話題沸騰中のWebGLを誰より早くマスターして世界中を楽しませるコンテンツを一緒に作りましょう！

WebGL対応ブラウザインストール済みのノートPC必須。希望者のみ別会場で懇親会あり（当日集計、費用は会場払い）
