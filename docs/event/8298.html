今年もハロパの季節がやってまいりました！

昨年開催した『 TokyoKiteup#2 ハロウィーンパーリィ 』に続き、ジャスト1年後のこの日にbrightkiteユーザーオンリーのハロウィンパーティーを開催します！公式タグは#kiteupです。
フライヤー画像は@7_nanaさんに作成していただきました！！＞多謝！

概要

小洒落た会場を貸し切って、食べ物・飲み物完全持ち込みで、もちろん仮装して、ハロウィンパーティーを開催します。
参加資格はbrightkiteユーザーであることただ一点です！

会場

MINUTE CAFE 30fun 渋谷
住所: 東京都渋谷区宇田川町34-6 M＆Iビル5F-B

前回同様、今回も会場を貸し切ります。
それほど大きい箱ではないので立食がメインになります。
お店に迷惑のかからないレベルで自由にやっちゃってください。

店舗HPには「携帯の電波が入る(ソフトバンク、NTT ドコモ、au)」と謳ってありますが、
会場の圏外率に定評のある@ikkou主催のkiteupです。iPhoneの電波については確認中です。
10/01追記。お店の方からiPhoneが圏内であるとの回答をいただきました！

電波対策として当日は会場内に無線LAN環境を用意します。
SSIDは“TokyoKiteup”です。貸し切りなのでWEPはかけません。
モバイルルーター等をお持ちの方は開放にご協力ください。

店内での電源使用についても確認中です。
10/01追記。電源の使用許可が出ました！充電用ステーションを用意します！

参加費用

会場貸し切り料金＋各自の持ち込み＋当日オーダーとなります。

会場貸し切り料金

3,600円/30分×3時間＝21,600円/当日の参加人数
20人参加の場合は1,080円になります。
30人参加の場合は720円になります。

各自の持ち込み

今回は食べ物も飲み物も完全持ち込みとなります。
ご来場の際には1,000円以上の持ち込みをお願いします。

参加登録時に持ち込むものについてアンケートを行なっています。
是非ご回答ください。（未定の場合は後ほどコメント欄でどうぞ！）

当日オーダー

食べ物も飲み物も完全持ち込みですが、実は安価なドリンクオーダーも出来ます。
【貸切／ホームパーティープラン限定メニュー（全品300円）】 [PDF]
10/03追記。当日オーダーはキャッシュオン形式になります。

仮装について

ハロウィンパーティーということで仮装しましょう！仮装しましょう！大事なことなので2回言いました。
幸い近くには東急ハンズやドン・キホーテがあるので容易に仮装グッズが手に入ります。
-東急ハンズ 渋谷店
-ドン・キホーテ 渋谷店
なにも全身仮装でなくても良いのです。マントを羽織るだけでも仮装です。
もちろん全身仮装は大歓迎であります。
