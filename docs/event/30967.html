description

4月に、ビジネスアプリケーションの雄であるSAP社と、Appcelerator社のTitanium Mobileで新しいモバイルビジネスアプリの世界を切り開くため、パートナーシップが発表されました。

SAP社は世界3位のソフトウエア企業であり、ビジネスアプリケーションの市場リーダとして、圧倒的な実績を持っています。

しかし、モバイル系の開発者から見ると、大規模でビジネス向けのシステムで自分には関係ないなと思われることも多いと思います。
ビジネスアプリケーションの開発者からは、モバイルの重要性が高まる昨今、高い生産性を持ったツールが求められています。

しかし、モバイルとビジネス向けの開発者には、まだ少し距離があるように思えます。
普段、接点のない開発者が多い二社ですが、この機会に新しい分野に触れ、距離を縮めてみませんか？

懇親会では、同会場にてピザとドリンクを用意しております。（ビリヤードやダーツもあるそうです）
時間の合わない方は懇親会のみの参加も歓迎しています！
