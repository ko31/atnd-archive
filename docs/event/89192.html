「Security Tech Lounge」とは

セキュリティ運用や設計構築開発、解析などサイバーセキュリティの業務に携わる方々の親睦と相互交流を目的としたイベントです。 セキュリティにまつわるTIPSやトリビアなどのちょっといい話から、 いろんな業種、立場の方を交えてのパネルディスカッションやライトニングトーク、 みなさんのセキュリティ知識を活かせる？ ミニCTFを予定しています。

おかげさまで満員大盛況だった「Security Tech Lounge vol.1」。第2回のテーマは「クラウド」です。
今回はアマゾン ウェブ サービス ジャパン株式会社様よりクラウドのスペシャリストをお招きし、クラウドとセキュリティについて考えます。近年クラウドサービスの充実によって、様々なモノを使いたいときに使いたいだけ安価に使える時代となりました。その反面、クラウドって大丈夫なの？という漠然とした不安がまだ拭いきれていません。今回のSecurity Tech Loungeではそんな切っても切り離せないクラウドとセキュリティについて情報交換を行います。

講演概要

1. AWSクラウドにおけるセキュリティの考え方
クラウドの利用において、セキュリティ対策は最重要項目のひとつです。AWSでは「責任共有モデル」という考え方に従い高いセキュリティレベルを維持することが可能です。本講演では事例を交えてそれらの概要をご説明いたします。

アマゾン ウェブ サービス ジャパン株式会社
エコシステムソリューション部
パートナーソリューションアーキテクト
市崎 洋平 様

2. クラウドとセキュリティ、担当者を悩ます魔物
「クラウド環境に基幹システムを構築する」という話は近年多くの企業にて採用されています。
一方で「クラウド環境に全て移行してもいいのか」など主にセキュリティ面でクラウドに
対して漠然とした不安（魔物）を持つ情報システム担当者やCIO、CISOも多くいらっしゃいます。
本セッションでは魔物に悩む背景とクラウド環境をフル活用するために必要なセキュリティの考え方に
ついてお話しします。

ファイア・アイ株式会社
技術本部 エンタープライズSE部
杉浦 一洋
