【セミナー概要】
もっと成果を出したいのになかなかうまくいかない、とお悩みのあなた。それはコミュニケーションに問題があります！

30代前半で会社の取締役にまで上りつめ、人との関わりあいの重要性に気づき、独立後、「５分会議」を活用しコミュニケーション力を向上させる人財育成家となり、著書「リーダーは話を１分以内にまとめなさい～結局、何が言いたいの？～」
「相手が“期待以上"に動いてくれる! リーダーのコミュニケーションの教科書 」などが大好評の人財育成家、沖本るり子氏。

部下とのコミュニケーションに悩み、降格人事や減給を経験しながらも、コミュニケーションについて学び続け、独自の方法を確立し、３年連続営業成績MVPという好成績、独立後は企業研修に引っ張りだこ、著書「承認力」「部下のやる気を引き出す 上司のちょっとした言い回し」をお持ちの吉田幸弘氏。

コミュニケーションには「伝える力」と「承認する力」が必要です。
組織で成果を出し、独立してからも大活躍の講師から、コミュニケーションにより相手を期待以上に動かし、人を活かしてより一層成果を出し、あなたの望む未来を手に入れる方法を学びませんか？
