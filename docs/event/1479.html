description

１０月３日に開催を予定していた高専カンファレンスin北海道02ですが、誠に勝手ながら延期させていただくこととなりました.下記にその理由について記載させていただきたいと思います.

　今回高専カンファレンスを延期した理由はインフルエンザによる苫小牧高専の学校閉鎖です.高専カンファレンスは高専生、高専OBが行う発表型技術勉強会です.苫小牧高専が学校閉鎖となり学校閉鎖期間に高専カンファレンスが開催されるため、閉鎖期間に高専カンファレンスに参加し、インフルエンザの感染拡大などに陥った場合高専カンファレンス側として責任をとることが出来ません.
　また、開催地が札幌ということもあり苫小牧高専からの参加者が非常に多かったことも挙げられます.
　参加者の半数が来れない中で開催するよりは延期という手段をとるべきではないかと考えこのような結果となりました.
　楽しみにしていた方や遠征されてくる方、プレゼン準備をされていた方には非常に申し訳ないのですが、高専カンファレンスin北海道02は延期となりましたのでよろしくお願いいたします.
