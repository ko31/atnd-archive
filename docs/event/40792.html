PHPのWebプログラミングについて勉強します！

実際にプログラムしている画面を見ながら行いますので、書籍よりも分かりやすく学べます。

対象者

  1. HTMLについて学習済みの方
  2. PHPが全く分からない方
  3. PHPがなんとなく分かるけど、もっときちんと学習したい方

学習内容

  1. まず、インターネットでは、どのようにしてブラウザがサーバーにアクセスしているのか、その仕組みを解説します。
  2. その後、サーバー上ではどのようにしてPHPのプログラムが実行されるのか解説します。
  3. 後半は実際にフォームを使ったチャットプログラムの作成を行います。

勉強会について

  1. 勉強会はプロジェクターに映した講師の画面を見ながら、自分のノートパソコンにプログラムをします。
  2. ハンズオン（実際にプログラムを行う授業）形式ですが、見学だけでも構いません。
  3. １９：０５～２１：００の約２時間です。
  4. 会議室は１８時から入れます。主催者は１８時４５分頃に到着します。
  5. 参加費として２，０００円かかります。

XAMPPのインストールをお勧めしています

プログラムしたPHPを実行するには、XAMPPというソフトウェアを使うと簡単です。

これはパソコン上（ Windows, MacOS ）で、Webサーバーを実行するプログラムです。
（基本的にインターネットで使用されるWebサーバーと同じプログラムです）

エディターについて

PHPはHTMLなどと同様に、単なるテキストファイルなので、Windows付属のメモ帳でもプログラムすることができます。
しかし、プログラム専用のエディターを使った方が効率的です。
PHP初心者おすすめのエディター集をお読み下さい。

勉強会のスケジュール

  1. チャットの作成（フォームを使ったプログラムの学習）
  2. 掲示板の作成（ページ遷移のあるプログラムの学習）
  3. メールの送信
  4. クッキーの使い方
  5. データベース（MySQL）の使い方
  6. データベースを使ったチャットの作成
  7. データベースを使った掲示板の作成
  8. ショッピングカートの作成
  9. Git『超』入門
