OpenStreetMap( http://www.openstreetmap.org/ ) という、
Wikipediaのようにみんなで自由に使える地図データを作ろう！というプロジェクトがあります。

地図といえば真っ先に思い浮かぶものはGoogleMapsですが、GoogleMapsには色々と制限があります。
GoogleMapsとOpenStreetMapの違い、活用事例などについてご紹介させていただきます。

OpenStreetMap北陸 に加入していない方でも参加可能です。

持ち物

・ノートPC(Windows・Macどちらでも可)を持ってきていただければ、実際に操作しながらOpenStreetMapの編集方法を学べます。
(マウスがあったほうが良いかも知れません。)
・Wi-Fi環境はこちらで用意致します。
・OSMアカウント取得の際には、メールアドレスが必要になります。

お願い

参加される方で、既にマッパーである方には大変恐縮なのですがお願いがあります。
・可能であれば普段行っているマッピングの方法や使用しているツール、体験談などについてご紹介していただきたいです。
(LTで発表していただくと大変嬉しいですが、口頭のみのご紹介でも大歓迎です。)
(ヘビーなマッパーではないからちょっと・・・という方も、本当に簡単にで結構ですので御紹介していただきたいです)
