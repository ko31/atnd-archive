セミナー内容

日本ナンバーワンシェアを誇るWordPress、今ではWeb制作者であれば構築できて当たり前の技術になってきています。我流で作っていて、もっと効率的な方法があるのではないか？この情報は本当に新しいものなのだろうか？と思った事はありませんか？
今回のセミナーでは、WordPressを使っているけど自信がない、もうワンステップ先へ進みたい方を対象に、具体的なWordPress構築について学びます。
★人気講師　瀬口 理恵さんが丁寧に解説します。

講師紹介

瀬口 理恵（せぐち りえ）

約7年間制作会社などでウェブデザイナー、ディレクターとして過ごし、2010年に独立。
ウェブサイト制作、アプリUIデザインなどを手がける。女性向けデザインが得意。
隔週火曜のUstream番組「rie'sCafebar」パーソナリティーとして放送中。
著書に「美しいウェブサイトの作り方」(SBクリエイティブ)。

アジェンダ

・WordPressインストールから初期設定あれこれ
・コーディング済みデータからのテーマ化
・カスタム投稿タイプとカスタムフィールドを使いこなす
・厳選プラグインとその活用法
・サイト完成、公開後の運用を考える

備考

セミナーは聴講形式＋ハンズオン形式を予定しています。
PCをお持ちの方は持参いただき、一緒に手を動かしてみましょう。

【推奨環境】
WordPressをインストールしたサーバーの準備、もしくはローカルサーバー環境
※インストール手順から作業したい方はサーバーの準備のみ
※PCをお持ちでない方も聴講いただけます。
