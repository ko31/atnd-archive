description

2014年は日本のビジネスにクラウドファーストの波が本格的に到来し、さまざまな領域のサービスがクラウドサービスとして出揃った感のあるような 1年であったかと思います。
サーバ運用の現場にいるエンジニアにとってこの 1年もっとも気になった技術トレンドは何でしたか。「OpenStack」「KVM」「Docker」そして「ベアメタル」。2014年も気がつけば残りわずか。今年の技術トレンドは今年のうちに振り返り、次にくる注目の技術をいち早く押さえ、締めは懇親会で一緒に乾杯！そんなセミナーを企画しました。
ささやかですが軽食をご用意して皆様のご参加をお待ちしております。
