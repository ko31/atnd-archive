Xamarinを使用したモバイルアプリの開発とアプリケーション開発サイクルの考え方【参加無料】

～解説概要～

日本マイクロソフト株式会社 テクニカルエバンジェリスト 千代田まどか

コンシューマーが一人で複数台のモバイルデバイスを持ち、それだけでなく企業でさえもモバイルデバイスを活用した働き方革命を推進しようとしている現在、モバイルデバイス向けのアプリケーションの重要性と、その開発が行えるエンジニアのニーズが高まっています。

マイクロソフトは2016年にXamarin社を買収し、マイクロソフトの開発ツールの一部として無償提供を開始しました。
これに伴い、Visual Studio を使用して、Windowsだけでなく Android、iOS に対応したアプリケーションを、ひとつの .sln (ソリューションファイル)内で、大きくコード共有しながら C# だけで記述することができるようになりました。

従来では、上記3種類のデバイス向けにそれぞれ Java,Objective-C/Swift,C# の 3言語で開発しなければならなかったコストを大幅に圧縮し、使いなれた C# で開発がしやすくなったことで、多くのエンジニアがモバイルデバイスアプリの開発に参入できるようになりました。

本セミナーではXamarinを使用したモバイルアプリケーション開発に焦点を当て、開発のライフサイクルを含め

  • Xamarin がなぜ有用なのか
  • Xamarin の使用方法 － 基礎編
  • Xamarin を使用することによるコスト効果
  • よりコスト効果を高めるための Mobile DevOps

    • 　－Hockey App
    　－Xamarin Test Cloud

等について、ご紹介させていただきます。
