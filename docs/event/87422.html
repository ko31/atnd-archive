description

＜＊初心者の方を対象に１日で集中的にUnityの基礎を学べる内容となっています。＞

Unityでゲームを作ってみませんか？

Unityは本格的な3Dゲームを手軽に開発できる素晴らしいツールです。

この講座は、


・Unityが前からちょっと気になっていた方。
・Unityを独学で始めようと思ったけど挫折してしまった方。
・仕事でUnityに触れる機会が出てきたけどプログラミングは全く初心者という方。

など、「入門者」「初心者」の方を対象とした講座となっています。

そのため、講座の内容も「Unityをはじめたばかりの方向けに構成」しています。

（講座内容）
初心者の方限定ですので、ゆっくり丁寧に講義を進めます。また少人数クラスですので、相互にディスカションしながら楽しい講座にしたいと思います。

ちなみに、この講座は「プログラミングは初心者」という方を念頭にカリキュラムを構成していますので、「プログラミングの経験は０」という方でも安心してご参加いただけます。

「応用編」では「基礎編」で学んだ知識をベースに、１日でUnityの基礎をさらに深く学びます。

講座は大きく３つのパートに分かれています。

第１部：Unityの基本的な操作方法を学ぶ②
★Unityの基本的な操作方法の復習
★オブジェクトの親子関係の理解と操作方法
★各種コンポーネントをプログラミングでコントロールする方法

第２部：Unityで必要なプログラミングの基礎を学ぶ②
★プログラミングの基本的な考え方の復習
★「Prefab」＆「Instantiate」機能の基本をマスターする
★実践トレーニング（ゲームを作成）

第３部：UnityのUIの基礎を学ぶ②
★UnityのUIの基礎と基本的な実装手法の復習
★UIをプログラミングでコントロールする方法を学ぶ
★実践トレーニング（ゲームを完成）

（こんなことができるようになります）
講座内で体験用の3Dゲームを１つ完成させます。その過程でUnityの基本的な操作方法と開発手法を学びますので、ご自宅でも簡単なプロジェクトを自分で作れるようになります。短期集中的にUnityの基礎を学びたい方、Unityの可能性を体験しながら確認したい方にはオススメです。

（タイムスケジュール）
★１日８時間のカリキュラムとなります。
　９：００−１１：３０　第１部：Unityの基本的な操作方法を学ぶ②
１１：３０−１２：３０　昼食休憩
１２：３０−１５：００　第２部：Unityで必要なプログラミングの基礎を学ぶ②
１５：００−１７：００　第３部：UnityのUIの基礎を学ぶ②

＊随時、途中休憩を行います。

（対象）
大学生以上の方ならどなたでもご参加いただけます。


・Unityに興味のある方
・Unityを使って自分でゲームを作ってみたい方
・お仕事でUnityに触れる機会が出てきたので基礎知識を得たい方　など

いずれも、Unity及びプログラミングは初心者の方を対象としています。
