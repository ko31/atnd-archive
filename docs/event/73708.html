LIGのディレクターが「ディレクションってなんだろう？」を掘り起こして考える。【Dig Up Direction】開催します

Dig Up Directionとは?

Webディレクターになってある程度経験を積んだつもりでも、ふとした時に考えてしまう「ディレクションってなんだろう？」という悩み。LIG社内でもひとくちにディレクターと言っても得意分野の振れ幅が大きく、本当に同じ職種なんだろうか？と考えてしまうこともしばしば。

答えは出ないかもしれないけれど、悩んでいるなりに「わたしなりのディレクション」を見つけるためのきっかけづくりを目指しています。Dig Up＝掘り起こす。私たちと一緒に、ディレクションとはなんなのかを掘り起こして考えてみませんか？

株式会社LIGとは?

台東区上野のWeb制作会社です。 LIGブログが有名で、砂浜に社長が埋められたり、即戦力な男がいたり、秒速結婚した人が有名だったります。

イベントの内容

第1回目となる今回は、前半にLT×4本！「＋αのわたしの強み」を持っているディレクターって？をテーマに各々の得意分野と、それをどう活かしてディレクションにつなげているのかの実体験トークをお届けします。後半はLIGの制作メンバーによるパネルディスカッション。ディレクター、デザイナー、エンジニアそれぞれの職種の目線で、「チームでつくる」ために「チームで改善すべきこと」を考えます。

01. LT×4本！「＋αのわたしの強み」を持っているディレクターって？

ディレクターと言ってもその仕事の幅は様々。（ディレクターのみなさんは、地元の友達や親に仕事を説明できないシーン、絶対経験ありますよね？）じゃあディレクターって一体なんなの？その答えはすぐには出ないにしても、他のディレクターが自分のどこに強みを置いて仕事をしているのか？は気になるもの。今回は"自分の特色ある"個性豊かなディレクターが、それぞれ考えるディレクションに関してお話しします！

今回トークをお届けするのは以下の4名です。持ち時間は一人10分！

02.「チームでつくる」ために「チームで改善すべきこと」を各職種が考えます会議

後半はLIGの各職種のメンバーが集って、「チームでつくる」ために、「チームで改善すべきこと」をあれこれ考えます。

ディレクターの仕事って、チームがゴールに向かって進むためにぐいぐいまとめていく、という舵取りの役割が超重要になりますよね。チームでやるからこそ自分一人では出来ないものが出来上がるし、やりがいもひとしお。

ただ、チームだからこそ、違う職種だからこそ、悩むこともたくさんあるわけで…。ということで、それぞれの職種の視点から、「よりよいチームであるためにこう改善できるよね？」といった要望をぶつけあい、あれやこれやと会議します。チームづくりはディレクターが案件ごとにぶつかる壁。ちょっとでも、その悩みを解決する鍵になれば嬉しいです。

※この時間からお酒や軽食を解禁しますので、みなさんほろ酔い気分で耳を傾けてもらえれば…。本来ならこういった話は居酒屋で語り合う内容だと思うので、LIGメンバーもお酒を飲みながら話そうかな〜と。職種間で感じるぶっちゃけトークが出るかもしれません（笑）ここで出た内緒トークはのちの懇親会で飲んで酔っ払って忘れてください！

03.せっかくなので懇親会もします

参加者同士の交流を深める為の懇親会を設けております。お酒とちょっとしたご飯をつまみながら、みなさんとあれこれお話できれば嬉しいです。LIGのディレクター陣もがっつり参加しますので、もっと深く知りたいこと、メンバーに直接こっそり聞いてみたいことなど、どんどんお声かけください＆悩めるディレクター達が助け合うために、出会いのきっかけになればいいなと思っています。
