技術ひろば.net 2012年3月 勉強会

以下の通り「技術ひろば.net 2012年3月 勉強会」を開催します。
どなたでもお気軽にご参加ください。

なお、3月の勉強会は運営の都合により土曜日夜の開催です。通常の開催とは時間帯が異なります。ご注意ください。
また一部の内容や時間帯の変更などの可能性があります。atnd.org のユーザー設定でメールアドレスを必ず設定するとともに、開催前に改めてこのページをご確認ください。お手数をおかけしますが、よろしくお願いします。

開催概要

■日時 : 2012年3月17日(土) 18:30～20:30 (受付 18:00～18:30)
■会場 : 首都圏コンピュータ技術者株式会社 東京本社 2階 大会議室
　http://mcea.jp/tokyo#maptitle
■会費 : 無料
■定員 : 15名

3月の勉強会は急遽開催を決定しました。当日の会場整理など行き届かない点もあるかもしれません。参加の皆さんのご協力をお願いします。

内容

3月の勉強会は Visual Studio LightSwitch についてのセッションです。さらに、Windows 8 Consumer Preview をスペックの異なる2台のスレートPC でご覧いただく予定です。
※当日までに内容を一部変更する可能性があります。ご了承ください。

  • 「LightSwitch ～結局何ができるの？～」 （瀬尾）
    わんくま同盟 東京勉強会 #65 のセッションをベースに「そもそも LightSwitch って何？」という方のために基本的な操作も紹介します。
    Visual Studio 11 Beta では Visual Studio 自体に内包する形で提供されています。Visual Studio 11 Beta の評価とあわせて LightSwitch にも触れてみるきっかけにしてください
  • 「Windows 8 Consumer Preview を見てみよう」
    スペックの異なるスレート PC を会場に持ち込みます。実際にご覧いただきながら、低スペック PC でどのくらい “使い物になるか” を自分の目で確かめてみてください。
  • 懇親会 (有料)
    勉強会後に希望者で会場を変えて自由に情報交換やおしゃべりをしましょう。
