ご登壇②　株式会社Vungle　ジョージ・マローニー様　松本様　　　　
　　　　　講演「国内事例から学ぶ動画広告導入の秘訣！」

アプリ内動画広告プラットフォームVungle様の広告は、世界で多くの人気のあるアプリに掲載され、
全世界で毎月2億人以上のユーザが閲覧しています。Vungle様の高精細フルスクリーンHD動画広告は、
クリエーティブとUX向上を意識した広告配置で配信されるため、パブリッシャー様のコンテンツに
シームレスに対応可能です。ユーザーが観たいと思う動画広告を配信することにより、ＵＸを阻害
することなくアプリの収益化 が実現されます。ユーザーにプラスになる広告体験を提供するVungle様は
見事 iOS および Android全体で「No.1ユーザーリテンション モバイル広告パートナー」に選ばれました。

Vungle様は現在、米国、サンフランシスコに世界本社を置き、
世界6カ国でビジネスを展開し、ゲーム業界の企業を中心にサービスを提供しています。

➔ 株式会社Vungleについて・SDKのダウンロード
　※Vungleに動画導入についてはjp-supply@vungle.comまでお問合せ下さい。

19:50～
ご登壇③　カイト株式会社　後藤様
　　　　　講演「全てのアプリにアイテム課金が必要な理由(仮)」

IT、モバイル、特にスマートフォンという市場において“開発者をハッピーにする”をモットーに、事業を
展開されているカイト株式会社様。開発者のさまざまな悩みの種を成功の種に変える為「appC cloud」
という、アプリのマネタイズに必要な機能をもった環境を「利用料完全無料・無制限」で提供しています。

appC cloudSDKを実装するだけで、CPI広告・リワード広告表示、プッシュ通知の実装など様々なサービスが
利用できますが、特筆すべきは最短10分で誰でもアプリ内課金機能を実装できる「アイテムSTORE」。
敷居が高く、敬遠されがちなアプリ内課金ですが、個人アプリ開発者にこそ、夢があるマネタイズ手法です。

【アプリ内広告による収益モデル】

　多くのユーザーにアプリを利用してもらうことで収益が伸びる
　 →ユーザー数増加にはプロモーションコストが必須。予算が潤沢な大手アプリ提供会社が有利。

【アプリ内課金による収益モデル】

　収益の8～9割を支えているような、1-2割のコアユーザーを掴むのがポイント
　→コアユーザーを獲得できれば、収益可が可能。コストを掛けられない個人アプリ開発者にも十分なチャンス！

今回は、カイト株式会社の代表取締役、後藤様より「アイテムSTORE」を中心に
「全てのアプリにアイテム課金が必要な理由」について、お話し頂きます。

20:10～
ご登壇④　UUUM株式会社　高田様
　　　　　講演「YouTuber×カジュアルゲームの可能性」

ヒカキン・はじめしゃちょーなどのトップYouTuberを数多く抱え、マネージメントや動画作成の
サポートなどを行うUUUM株式会社様。

これまでUUUM所属の人気YouTuberは、10タイトル以上のカジュアルゲームにおいてコラボ実績を積んでおり、
2015年9月にリリースされた「俺の校長3D」ではHIKAKIN、はじめしゃちょーなど100名以上のYouTuberが
ゲーム内キャラクターとして登場し、Hikakin Gamesにおいて公開した実況プレイ動画は、再生回数が70万回を
越えるなど、ユーザー様の大きな反響を得てApp Storeゲーム総合ランキングで1位を獲得しました。

今回は、UUUM株式会社 高田様より、YouTuberの影響力、実況動画とゲームアプリとの相性、
コラボ事例などを通じて、YouTuberを活用したカジュアルゲームプロモーションの可能性についてお話頂きます。
