名古屋では久しぶりの Windows Phone 7 ハッカソン

ハッカソンとは

みんなで集まり、アプリの開発・発表をする会のこと。
「プログラムは初めて」という方も大歓迎です。

参加するとこんな良いことが！

  • みんなと一緒にアプリ開発ができる
  • WPアプリ開発者との交流ができる
  • 自分の知らない、技術・情報が共有できる
  • 初心者の方なら、今後もプログラミングを継続していく自信がつく

※ゲストは未定です。

イベント内容

みんなでワイワイ／ガヤガヤとWP7のアプリ作ります。
自由なアイデアで、みなさんとご一緒にアプリ開発を楽しみましょう。

（下記のような内容を予定しています）
*初心者向けには、アプリの作成方法からMarketplaceで公開する方法までをワークショップ形式で解説します。
*経験者向けには、参加者どうしで情報交換しながら、自由にアプリケーションを作成していただきます。
*初級～中級の希望者向けに、Windows Azure や Sky Drive という２つのクラウド環境と連携したアプリケーション作成のワークショップも開催したいと考えています。

参加条件

特にございません。初めての方、初心者の方、大歓迎です！
デベロッパーの方、デザイナーの方、興味のある方ならどなたでもご参加頂けます。
開発環境をインストールしたPCを 持参してください。 持参してもしなくても、どちらでよくなりました。（このページの下方参照）
Windows Phone 7 デバイスは、所持していなくてもかまいません。（エミュレーターで動作確認します）

用意していただくもの

  • 開発用PC（用意できる方）
    • OS Windwos Vista 又は Windows 7 (WindowsXPは使用できません)
    • 仮想環境は非推奨です Macを使う場合はBootcampでの動作を推奨します。
    • 持参できない場合は、会場のPCを使用できます。
  • Windows Phone（用意できる方）
  • Windows Azure のライセンス（クラウド連携参加希望の方のみ、できるだけご用意ください）

※開発用PCには下記のSDKをインストールし、開発環境とエミュレータの実行が問題なく行えることを確認して参加をお願いいたします。
