イベント概要

北海道の小樽と札幌で開催している Drupal CMS（オープンソースのコンテンツ管理システム）の勉強会です。今回は小樽での開催です。なお、名前は「勉強会」としていますが、セミナーのような堅苦しいものではなく、Drupal ユーザーが自由に情報交換できる場にしたいと考えています。気楽な集まりですので、お気軽にご参加ください。北海道の Drupal ユーザで情報交換と交流の輪を広げ、さらなる活用につなげていきましょう。

参加費：無料

内容

今回は、ハンズオン（体験学習）形式で、実際に手を動かしながら Drupal 活用のヒントやテクニックを共有できる場にしたいと思います。もちろん、いつものようにショートセッションやライトニングトーク（LT）も大歓迎。内容は Drupal のほか Web や IT に関するものなら何でもOKです。

ハンズオンでやってみたいこと、内容への要望、またセッション/LTの題材をお持ちの方は、お気軽に管理者までお知らせください。Twitter、メール、または当ページのコメント欄をご利用ください。

フリーセッション

・最近 JavaScript で困っていること（maichさん）

テーマ (1)：便利なモジュールの使い方
これまで利用したことがある便利なモジュールをそれぞれピックアップして紹介。インストールから設定、利用までの手順を実際に操作しながら確認していきます。ちょっと変わった使い方や自慢のテクニックなど共有しましょう。当日紹介予定のネタがあれば以下に記載しますのでお知らせください。

・Feeds と Embedded Media Field で Picasa Web の写真を取り込む方法（bkenroさん）
・Diff モジュールと Revisoning モジュールによるワークフロー（blauerbergさん）

テーマ (2)：Drupalの開発環境
モジュールやテーマ等の開発で使用している環境を紹介しあいましょう。プロジェクト管理、編集、デバッグ、テスト、ソースのバージョン管理など種類は何でもOK。Drupal開発の定番環境をシェアできるといいですね。

・私の Git 利用環境：Bitbucket、SourceTree（luxlaさん）

懇親会
