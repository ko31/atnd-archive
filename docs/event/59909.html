前回11月に告知をした
新サービスのご提案にあたり

お問い合わせ殺到中！

ご注文が殺到しているサービスの
12月限定バージョンとして

新たな付加価値をつけてご案内です。

まずはこのサービスのリリースに
至った想いについて
少しお話させていただきたいと思います。

弊社は２０１４年４月から
本格的に著者支援サービス

ベストセラープロデュースパッケージ
（通称BPP）を開始し、

ご依頼いただいた著者様に

・発売から１か月半で
　５００００部を達成！

・新人著者を発売から２か月で
　２３０００部を達成！

・新人著者を発売から１か月で
　１５０００部達成！

・新人著者を発売から２週間で
　１００００部アマゾン総合１位！

・Amazonキャンペーン４日間で
　１３００冊販売達成！

・１０日間でファンリストを
　２０００人収集！

・著者様の新商品を
　１０００個販売達成の支援

など、一例ではありますが、
上記のような実績を挙げることができました。

著者様と書籍の発売、または
知ってほしいサービスを同じ気持ちで
２人三脚で書籍の拡販に取組み、

多くの著者様と書籍販売の実績や
ファンづくり、そして何よりも
「絆」をつくってこれたと考えています。

しかし、これまでのサービスには
改善不可能な大きな問題点があったのです。

―　『毎月２名の引き受け枠』という限界　―

著者様が伝えたい内容を書籍から読み取り
そして、メルマガの執筆に、SNSでの投稿
初めての書籍を刊行し

不安がいっぱいの
著者様へのサポート・・・

これらの作業を毎日のようにしながら
著者様に結果を出していただくには
引き受け枠に限界があったのです。
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

著者様の出版刊行の日程は、
私たちの都合に合わせて変更することは
できませんから、

『弊社のサービスを申し込みたい！』

というありがたいお申し出すら、
お断りせざるを得ない状況がありました。

そして、多くの著者様から
このようなご意見をいただいたのです。

『文章は自分で書くので
メルマガを配信だけでもお願いできませんか？』

『販促のサポートがなくてもいいので
メルマガで広告してもらえませんか？』

と・・・

弊社が配信しているメルマガの読者様は
書籍の購入者や書籍の関連キャンペーンに
参加していただいた読者様が多く、

他にも語学や自己啓発や
コミュニケーションなど、
知識を求める方がたくさんいらっしゃいます。

とても意識が高く、学びの気持ちが強い
読者様のため、多くの著者様の告知にも
とても良く反応を示してくれます。

あまり知られていませんが、
弊社のメルマガの広告配信サービスは

■メルマガ号外広告
１通　６９,８００円

を頂戴して配信しているのですが
書籍の販売を目的としたとき

書籍の販売冊数だけで費用対効果を
測ろうとすると、非常に難しい計測になります。

メルマガ広告費用と、売れた冊数
の金額的な費用対効果はどうか？

という考え方をするならば

１日２日の短期的な効果の測定では
たしかに回収できないこともありますが、

長期的に行う繰り返しの告知や、
書籍だけでなく著者様のお人柄や
メッセージを読者のみなさまに
お伝えいただくことで効果が
どんどん出てくるものです。

そして、

書籍だけではなく

セミナーへの参加や
サービスの申し込み

にも

つながっていることは検証済みです。

ですから、単発の号外メルマガ広告では
限界のあるサービスも

繰り返しご利用いただくことで
大きな成果が期待できるのです。

しかし、これをお読みなられている方は
こうも感じたかもしれません。

繰り返し６９,８００円のサービスを
使えだなんて、

『いったい、いくら払えというのですか？』

たしかに仰るとおりですね＾＾：

仮に６９,８００円のメルマガ広告を
１０回使えば６９８,０００円ですから、

弊社の著者プロデュースプランの
ベストセラープロデュースパッケージ

に近い金額になってしまうのです。

そこで、私たちは頭を悩ませ考えました。

今よりも、もっと多くの著者様に
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
金額的なご負担をできるだけ軽減しながら
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
繰り返しメルマガを配信できる画期的な
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
サービスをつくれないものか？
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

そこで、思いついたのが

『２か月無制限メルマガ配信サービス』
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

通称UMDSというサービスです。
※Unlimited Mail Delivery Service

２か月間の間、弊社のメルマガ読者様に向けて
「無制限」で配信をできるサービス

を、今回、ご提案させていただきたいのです。

『なぜ、２か月なのか』

と思う方もいるかもしれません。

そこで、

この「２か月間」というポイントについて
少しご説明させてください。

私たちは、著者様の販促サービスを
お引き受けする際に、必ず申し込み前から
お伝えしていることがあります。

書籍販促の効果を最大限に出すための
『２か月前』マーケティング

『書籍の販促は遅くても
２か月前から取り組みましょう』

と、お伝えしています。

そして、これは書籍の販促だけではなく
その他サービス全般にも
言えることでもありますが

特に書籍については、
ご存じの方も多いと思います。

が発売から２週間が勝負と言われています。

そして１か月が経過するころには、
ほぼ勝負は決まってしまうものです。

これは元々、インターネットマーケティングと
コンサルティングの会社であった
弊社のプロダクトローンチの経験からも
間違いのないお話です。

昨今、読者の本の買い方は、
書店に行ってから購入する書籍を決める

「衝動購入」よりも、

先に得た情報を元に書籍を確認しにいく

「目的購入」が増えています。

ですから、発売と同時に、
多くの方にあなたの書籍を購入いただくには
発売前に、より多くの人に、あなたや
あなたの書籍を目に触れさせたり、
知っていただくことが必要不可欠なのです。　

その周知期間を、
私たちは『２か月間』と考えています。

いえ、もっと言えば、世の中のサービス全般に
該当することかもしれませんね。

発売前に時間をかけて、あなたの
お知らせしたいことを、じっくり
伝えていくことはとても重要なのです。

もちろん、
これは発売した後でも有効な手段です。

例えば、発売から１年経過した書籍で、
すでに売れ行きが止まってしまった書籍ですら
２か月間、あるコツに沿って配信し続けたことで
息を吹き返し増刷になったケースもございます。

ここでお伝えしたいのは、

メルマガの読者のみなさんも、
どんなに良い書籍だからと言って

１回告知をしただけでは
すぐに購入してくれるかというと
すぐには反応してくれません。

読者はまず

「信頼関係」を求めているのです。

私たちは繰り返しになりますが
インターネットマーケティング
コンサルティング会社として、

インターネットだけでお客様に
商品やサービスを購入していただく
サービスを提供してきました。

そこで、確実に言えることがあります。

■ザイオン効果による親近感は
　メルマガでも確実に存在する。

■人間性やメッセージを
　伝えることで信頼関係ができる

インターネットといえども
リアルのセールスと同じで
繰り返し接することや、売り手が
どんな人間性で想いをもっているか？

など伝えることがとても重要で
商品、サービスの売れ行きが
大きく変わってくるのです。

ですから、私たちは

『２か月無制限』でご自身のメッセージを
人間性を、そして、商品やサービスを

メルマガの読者様に伝えていただくことで
大きな成果を出していただくことが
可能だと確信したのです。

いかがでしょうか？

株式会社Clover出版の多くの
メルマガ読者様に、あなたの書籍やサービスを
知っていただくことができます。
