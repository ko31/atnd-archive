毎年5月の第三木曜日は Global Accessibility Awareness Day ということでアクセシビリティの事をはなすお茶会をします。
http://www.globalaccessibilityawarenessday.org/

会場はちいさなオフィスなので少人数を予定しています。
会場変更が無いかぎり費用ナシ。ご自分の飲み物等持ち寄りでお願いいたします。

ゆるく雑談する会の予定なので時間の都合がつかない方や遅れそうな方は途中入退場可能です。随時いらしてください。

人が多くなってきた場合や、お酒が必要になってきた場合会場移動する可能性があります。
その場合は当イベントページか、Twitter の #gaad_tokyo でご案内します。
予めご了承ください。

入場方法

該当の時間は1Fのオートロックがかかっているため 501 号室の Gaji-Labo を呼び出してください。
アクセシビリティのイベントとお伝えいただければ解錠いたします。
