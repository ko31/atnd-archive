description

私たちが生きていくためには収入が必要です。その為に働いているという方が大半だと思います。しかし「自分は何のために働いているのか？」「仕事の価値って何なのだろうか？」と悩みながら仕事をこなしている人も多いでしょう。それはあなたが“働く”ことの価値を他に求めているからではないでしょうか。

社会起業大学の名誉学長である田坂広志先生は著書の中で、本来仕事には二つの報酬があると言っています。一つは“目に見える報酬”、もう一つは“目に見えない報酬”です。金銭や、名誉、地位、役職、などに捕らわれる目に見える報酬ではなく、「人として成長していきたい」「人の役に立ちたい」「自分の存在意義を感じたい」など、価値観や考え方、自己の成長、師を求めるなどといった “目に見えない報酬”を私たちは求めています。自分の好きなこと、得意なことで培った能力やスキルを活かすことで、自分らしく生き、新しい働き方が出来るのではないかと私たちは考えます。

今回の無料体験セミナーでは、自分らしさと新しい働き方を両立させる方法の一つとして、「社会起業家」の説明や実際に行われている取り組みについてお話を行います。
どなたでも無料で受講頂けますので、お気軽にご参加ください。
