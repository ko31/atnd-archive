description

TBCは、精密医療を視野に入れたバイオインフォマティクスの臨床応用、それに関連 する研究システムや方法・テクノロジーについての国際会議で、米国・欧州・アジア 諸国から研究者が集います。

TBC2015の開催にあたり、TBCの概要と見どころを日本語で解説するプレカンファレンスを、日本医療情報学会関東支部会主催で実施することにいたしました。皆様のご参加をお待ちしております。

日時

2015年11月6日(金)18:00-19:00

場所

日本大学理工学部駿河台校舎1号館141教室

住所：東京都千代田区神田駿河台1-8-14

  • 東京メトロ丸ノ内線「御茶ノ水」駅　下車徒歩5分
