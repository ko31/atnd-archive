catch 10月26日(デニムの日）から応募投票一斉スタート。
description

2014年10月26日（デニムの日）から12月26日まで、第4回デニムファッションフォトコンテストがインターネット上で開催。
日本国内の一般の方を対象に、年齢、性別、地域、デニムブランドなど一切問わず、インターネット上でデニム（ジャケット、ジーンズ、小物など）をいかしたコーディネートファッションの写真作品を幅広く募集。作品は応募期間中、携帯電話やパソコンを利用し、専用WEBサイトにて「いつでも」「何回でも」気軽に応募が可能。
審査の結果、上位入賞者には、商品券、デニム関連商品、特別賞などが贈呈される。またＷ＆Ｓグループとのコラボ「WORLD AWARD」、HUGPROエンターテーメントとのコラボ「デニムスター賞」も初展開。
