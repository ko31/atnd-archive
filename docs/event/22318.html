イイネ! Project Award 2011とは

2011年、未曾有の災害に見舞われた日本。
その中で、多くの企業、NPO、個人が、 非営利のプロジェクトを立ち上げ、
「世の中のためになにかしたい」と活動を始めました。

それらのプロジェクトを見渡し、
「みんなが イイネ! と感じたプロジェクトを讃える 」と同時に
「いいプロジェクトとはなにか？」
「自分たちのプロジェクトに優れた方法を取り入れる方法」
についての知見を共有していくためのアワードが、
【イイネ！ Project Award 2011】です。

パーティ参加費

参加費：学生¥2,000／社会人¥3,000（drink＆food）

内容

イイネ！Project Awardの発表をはじめとして、
クリスマスパーティに欠かせないプレゼント交換や
ダンス・音楽など企画が盛りだくさんです。

「イイネ！Project Awardが気になる！」
という方はもちろん
「クリスマスっぽいことがしたい！」という方も
ぜひお気軽にご参加ください。

クリスマスプレゼント持参のお願い

クリスマスパーティーのお楽しみの一つといえば、プレゼント交換ですよね！

今回のプレゼント交換の品物は「あなたに勧めたい私の１冊」ということで、、

　【本】です！

プレゼント交換の際はいくつかのグループに分かれて、その中で
なぜその本を選んだのかという手短な紹介をしていただこうと考えております。

つきましては、参加者の皆さまにはプレゼントとなる本を一冊持ってきていただくとともに、、、
・それを選んだ理由もあわせて考えてきていただきたいと思います
・値段の指定は特にございません。新書やマンガでも全然OK です
・古本や自分が読み終わった本でも構いません。ラッピングしてお持ちください
・また、もし本を買い忘れてしまった方は会場付近に山下書店東京ドーム店がございますので
　そちらで買っていただければと思います。
