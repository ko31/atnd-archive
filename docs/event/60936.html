2015年成長する企業をつくる経営者の実践セミナー

title 2015年成長する企業をつくる経営者の実践セミナー
catch 『1000億企業をめざすリーダーの処世術』
description

経営環境や市場の変化、厳しい市場競争や経営課題の下で変わらず成長し続ける企業は何をしているのか。わかりやすく目を引く経営手法やビジネス書籍で学んだことを実践しても成果が出ないのはなぜか。１０年に渡り新規設立法人から創業数十年・売上数百億の企業を支援してきた税理士が、企業の本質的な強みと最適な機会を捉えて経営に活かすために、身近な視点と行動を変えていく体験・診断講座を開催します。

本セミナーでは、「成長する会社の条件」「机上の空論を現実化する」ために必要な極意をお伝えした後、自社の現状や課題から３年度のあるべき姿に向かって成長路線を歩むための自社診断ワークを行います。バランス・スコア・カードや伝統的な経営・財務分析手法を用いて自社の経営を捉えなおし、新年のスタートから成長路線を歩み続けるための行動原則を学びます。

セミナーの対象者
・今かかえている経営課題を捉え直し成長路線へ転換させたい経営者様　
・新規設立法人経営者様、第二創業期を迎え新規事業や経営改善に取り組まれている経営者様
・資金調達、キャシュバランス改善、リスケジュール、組織改善、営業強化など具体的な課題に取り組みたいが、成長路線につなぐ実践方法を知りたい経営者様
・上記の課題に取り組まれている企業ご担当者様

セミナーの流れ
・成長する会社の条件、机上の空論を現実化する：　　問題提起から３年後の自社の姿を考えます。
・３年、１年、毎月のショートスパンで実践する経営・管理術：ワークショップ形式で学びます。
・成長はなるべくしてなる：　企業の成長要因を自社に取り入れる極意をお伝えします。
