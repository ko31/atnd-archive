４SEEN人気企画です☆猫好きの方、写真が好きな方、癒されたい方大歓迎！
恵比寿のおしゃれな猫カフェで男女ペア（またはグループ）で猫photoを撮り ます。
カメラが苦手な方も大丈夫！プロのカメラマンがコツを伝授します♪
～講師紹介～石原さくらさん「幼少の頃から私の傍らにはいつも猫がいました。
絵を描くことが好きで猫や犬を写真に撮っては絵に描いていました。
私たち人間ほど長生きはできない彼らなのでお別れをする悲しみがありました。
寂しくなったときに自分で撮っておいた猫の写真を見ると、その泣き声や感触などの思い出あふれて温かい気持ちになりました。写真の力の素晴らしさと撮ることの楽しさを最初に教えてくれた猫に敬意を込めて、猫たちの写真を撮っています。
