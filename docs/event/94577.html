Mapillary Meetup 2018 Tokyo

ジオタグ付写真の世界規模共有サービス Mapillary の国内初User交流会を3月4日（日）東京：渋谷で開催します。
世界中をカバーする写真は2億4千万枚をこえました。日本国内でも共有写真の枚数は増え続けています。位置情報を持った写真共有の楽しみはもちろん、OpenStreetMapのマッピングツール、ビッグデータとして画像認識の分野からも注目を浴びています。今回の交流会では、ユーザー目線でのMapillaryに関するhow-to講演やゲスト講演、パネルディスカッション、ライトニングトークなどを予定しています。
ぜひこの機会にMapillaryユーザーとして情報共有、交流を深めてみませんか。皆様の参加をお待ちしています。
交流会を楽しみましょう！
