Rfm Code Reading #13

2011年8月21日（日）午前10時よりRfm（FileMaker API for Ruby）のソースコード読書会「Rfm Code Reading #13」を開催します。

Rfm Code Readingの概要

FM-Tokyo（FileMaker Pro 東京ユーザーズミーティング） の分科会活動の一環として、Rfm Code Readingでは、Rfm 1.4（https://github.com/lardawge/rfm/tree/v1.4）のソースコードを読み解くことにより、FileMaker ServerのカスタムWeb公開やRuby、およびRuby on Railsを使った開発のスキルアップを目指します。

RubyおよびFileMaker ServerのカスタムWeb公開に興味のある方のご参加をお待ちしています。
