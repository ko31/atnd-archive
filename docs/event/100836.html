イベント概要

組込、インフラ、保守・運用の仕事をしているけど、Web開発をしてみたい・・・
上流フェーズを担当しているけど、プログラムの仕事をしてみたい・・・
今は勉強中であまり知識はないけど、Webプログラマーにあこがれている・・・

このように感じている人、いませんか？

本イベントは、Webエンジニアへのスキルチェンジ、スキルアップ、キャリアチェンジを考えている方向けに
Webの基本的な仕組みを、書籍を利用した座学と実技を通して理解いただく内容となっています。
弊社社員がサポートいたしますので、Webシステムの開発をしたことがない人はもちろん、
あまり業務経験が少ない方でも気軽に参加いただけます。
また、開発経験豊富な講師が、普段聞くことのできない、業務でおきた体験談、失敗談などを交え、
楽しく勉強会をおこないます。

対象者

  • Webシステムの仕組みに興味がある人
  • Webシステムを基礎から勉強してみたい人
  • 何かしらのプログラム言語を使ったことのある人

※若手エンジニアの方向けを想定したカリキュラムとなっております。
※ご希望の方には転職相談なども可能ですので、希望される方は弊社スタッフにお声掛けください。

勉強会の主な内容

  • WebブラウザでWebページが表示される仕組み
  • WebサーバーからWebブラウザにコンテンツが届くまで
  • Webプログラムがコンテンツを作る仕組み
  • Webプログラムを動かしてみよう

利用する書籍

ちゃんと使える力を身につける　Webとプログラミングのきほんのきほん（大澤文孝 著）
https://goo.gl/uQdhqn
