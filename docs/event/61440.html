新CreateJSをコンテンツ制作に活かす

CreateJS が昨年末の2014年12月12日、ほぼ1年ぶりにライブラリを改めました。正規バージョン1.0もいよいよ近づいてきたようです。WebGLにも対応して、魅力的なWebコンテンツをつくるためのライブラリとして期待が高まります。CreateJSの新機能や、その活用の仕方、さらには表現の可能性についてご紹介します。

内容

新しいCreateJSで書くコードはどう変わるのか

野中 文雄
新たなCreateJSは、内部的な設計が整理され、最適化も進められました。そうした仕組みの中から公開された新しい機能もあります。知っておくとお得なコードや、書き方に注意しなければならない点など、つぎのようなお品書きでご説明します。

  • Graphicsクラスでコマンドを使う
  • CreateJSのクラスの新たな継承の仕方
  • PreloadJSの新しいローダーを使う
  • Matrix2Dの行列演算の仕様が変わった

LT: CreateJSとNode.jsを使ってサーバーでcanvas要素を使おう

鹿野 壮
CreateJSとNode.jsを組み合わせると、サーバーサイドでcanvasを扱うことができます。今回はサーバー側で画像を加工する簡単なデモを通して、サーバーサイドでCreateJSを使うメリットをご紹介します。

CreateJSで制作するスペシャルコンテンツ

池田 泰延
最新のCreateJSではWebGLに対応した一方で従来方式のHTML5 Canvas方式も残しています。それぞれの描画方式に制約があるためですが、適材適所につかえば表現力豊かに高速な描画を得ることができます。今回はそれらのテクニックを使ったスペシャルコンテンツの作り方とワークフローを紹介します。

LT: PreloadJSで「悩ませないローディング」の作り方

蔵下 雅之
外部ファイルを読み込む際によく見かける「くるくる」と回るアニメーションGIFのローディング。読み込んだ進捗がわからず「あとどれだけ待ったらいいの？」と悩むことも多いと思います。そんな悩みをユーザに与えないローディングの作り方をPreloadJSを使用したサンプルでご紹介します。
