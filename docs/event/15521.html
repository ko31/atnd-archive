異種混合麻雀大会

概要
金曜日の夜、女装子・男装子交えた麻雀異種混合戦をやります。
試合は１荘すなわち２半荘。

ルールについては下記のルール参照

  • ルール
    1. 役なし上がり不可。喰いタンあり。喰いピンは当然なし。
    2. 細かい役についてのルールはより一般的なルールで。
    3. ドラは表＋カン＋裏ドラまで。（カン裏なし）
    4. ２５００点持ちの30000返し。ウマ5、１０

なお賭博は禁止です。
