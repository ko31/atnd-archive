～ツリークルージングとは?～

　ツリークルージングとは
　ハーネス・ロープを使用した本格的な木登りの事です。
　ハーネスをつけて登るので、力はそんなにいりません。
　健康な方ならどなたでもご参加して頂ける新感覚のアウトドアスポーツです。

　健康な方ならどなたでも大歓迎。
　専任インストラクターが安全に楽しくご案内いたします。
　木の上から優雅に眺める新緑の奥多摩は最高ですよ。

　今回は6月中旬からのオープンに向けてのプレイベントとして、特別プランでのご案　内となります。
　約2時間のツリークルージング体験の後は多摩川を見下ろすウッドデッキでピザパー　ティ。
　炭火釜焼の本格的なピザをみなさんでトッピング・窯入れまで体験して頂けます。
　ワイワイ焼きながら、楽しんでください。
