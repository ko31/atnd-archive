CAT 15周年企画 プレ公演「ジェニーとガラスのくつ」

ただいま活動15年目を迎えている、ミュージカル＆演劇ワークショップCATの15周年企画第一弾！ 父に連れられて叔母マーゴのもとへとやって来た、心を閉ざした少女ジェニーが遭遇する不思議な世界。CATが2018年夏にお送りするミュージカル・エンターテインメント！
