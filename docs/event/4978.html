description

会社でホームページの担当を任されたけれど、知識に自信もなくどう進めていいのかわからない方って多いのではないでしょうか。
今回、そのようなホームページ担当1年生の方々を対象とした「ホームページ実践セミナー」を開催致します。

内容としては、ホームページを作成やその後の運用管理をおこなって行く上で、どのような作業が必要になるのか、そしてどうのように進めればよいのかをわかりやすく説明致します。ホームページ作成については、ＣＭＳツールのご紹介も行います。

お仕事が終わってご帰宅されるまでのちょっとしたひと時、コーヒーを飲みながら軽食を取りながら、気軽な感じで参加してみて下さい。

もちろん参加費は無料です。

宜しくお願い申し上げます。


AGENDA

+ 18:00～18:30 受付開始

+ 18:30～18:40 本セミナーの主旨と今後について

+ 18:40～19:40 ホームページの制作から運営に必要なスタッフとは？

+ 19:40～19:50 休憩

+ 19:50～20:20 ショートセミナー（実演）

+ 20:20～20:30 質疑応答
