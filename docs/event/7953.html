概要

@ushio_s を筆頭とする世界で一番好きな食べ物が餃子の人の集団、うっしー餃子部の第一回部会を行います。
といっても、みんなで餃子を食べて吞んでワイワイ楽しくお話をしましょうというだけです。

開催日時

2010/09/18 (Sat) 18:00 〜 予定
現地集合・現地解散です。
お店の混み具合によっては、店先で並んで待つことになります。
が、部長がやる気ちょろっとをだしたようなので、予約しておいてくれるらしいです。

開催場所

蒲田 歓迎本店
(このお店は、 @ushio_s と @kzmhr の間ではランキング 1 位のお店です)

主催者

  • @ushio_s (餃子部部長)
    • 元ひもカメラマンです。
  • @kzmhr (餃子部部長代理)
    • 元ゆるふわパーマの人見知りです。

餃子部規則 (@kzmhr が勝手に作りました)

  1. お店に入っても 焼き餃子 しか食べない
    • 飲み物は可。ただし、飲み過ぎは餃子が食べられなくなるので注意すること。
  2. ライスを頼む者は問答無用で即刻退部
  3. 餃子を食べる前に写真撮って up する
  4. ハッシュタグは #club_gyouza (単語登録しておくと便利)
  5. 集まっての部活動がない場合は、各自各地域の餃子屋を巡り自主トレをすること

その他

  • 参加に当たっては、社会人として節度ある行動・ルールは必ず守っていただくようにお願いします。
  • 未成年者の飲酒は法律で禁止されています。
  • 飲酒運転は法律で禁止されています。
  • 何か連絡がありましたら、上記主催者のどちらかまでお願いします。
  • 部会に出たいけど夜は出られないよー＞＜という方は、お昼にもちょろっとやるので、主催者まで連絡をお願いします。
    • お昼に大連餃子基地で腹ごしらえ→芝公園→浜離宮庭園→日比谷公園→皇居と回って写真撮ります。
    • また、間にちょくちょく寺社に寄って御朱印を集めます。
    • 時間が足らなくなったら、途中でもそこで写真＆御朱印コースは終了します。
    • 写真＆御朱印コースは、内容的にどう考えても餃子部ではないので、任意参加です。
