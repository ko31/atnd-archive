description

［講座のポイント］
　高級感・高品質感とは何か、商品開発での高級感・高品質感の重要性、海外の高級感・高品質感から高級感・高品質感のある商品コンセプトの発見方法、高級感・高品質感の評価方法、以上の結果を新商品開発への繋げる方法をこれまでの具体的な新商品開発事例を入れ解説する。

［受講対象者・レベル］
　企業内の新商品開発者、新商品実験、評価担当者、新商品開発の中で高級感・高品質感の重要性を研究対象とする企業内研究者、公設研究所研究者、大学等研究機関研究者に受講していただきたいと考えています。

［プログラム］
１．高級感・高品質感とは何か？
　　1.1　高級感・高品質感の重要性
　　1.2　高級感・高品質感を含む感性価値とは？
　　1.3　マーケテイングの視点からの感性価値の考え方
　　1.4　高級感・高品質感を含む新しい品質の視点(日産自動車Perceived Qualityの事例)
　　1.5　高級感と高品質感
　　1.6　高級感・高品質感と快適性
　　1.7　高級感・高品質感を感性工学から考える
　　1.8　日本人の高級感・高品質感と欧米人の高級感・高品質感
２．高級感・高品質感の計測・評価から商品開発（事例を入れて）
　　2.1　商品ごとにユーザーの高級感・高品質感評価内容を知り商品コンセプト開発に繋げる
　　　2.1.1　評価グリット法による方法
　　　2.1.2　テキストマイニング分析による方法
　　2.2　高級感・高品質感感性評価方法を知り新商品の設計、デザイン、実験に繋げる
　　　2.2.1　言語による印象評価法、
　　　2.2.2　官能評価法による方法
　　　2.2.3　高級感・高品質感と他の感性尺度の関係を分析しユーザーの評価構造を捉える方法
　　　2.2.4　商品の感性評価構造の中での高級感・高品質感の重み分析方法
　　　2.2.5　高級感・高品質感の感性尺度を物理的計測値に置き換える方法
