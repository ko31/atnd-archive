Japan.R（午前の部）: はじめての「R」

Japan.R について

  • Japan.R は、全国各地の R コミュニティが集まって行われる全国規模の勉強会です。
  • 午前は本ワークショップを行い、午後は、各地域コミュニティの活動紹介、R の活用事例の紹介・情報交換を目的とした企画が催されます。
    • 午後の部の申込は次のサイトからお願いします。→ http://atnd.org/events/9806

ワークショップ概要

R を初めて使う方、R を使い始めてこれから本格的に利用しようとされる方を対象とした 入門者・初級者向けのワークショップ です。当日は、R のインストール方法から説明しますので、どうぞお気軽にご参加ください。

本ワークショップの目標は、次の4点です。

  1. 自力で R のインストール・アップグレード・アンインストール ができる
  2. 基本的な 関数と変数の扱い方 が分かる
  3. Excel データなどを読み込み、R で処理 ができる
  4. 簡単な グラフ作成 ができる

また、ワークショップ終盤（11:40 以降）では、阪上が研究対象としている 英語や日本語といった自然言語のデータを用いた研究事例の紹介 と、データ分析の様子をデモとしてお見せする予定です。 人文科学系における研究内容にご関心のある方は、11:40 以降にお越しください。
