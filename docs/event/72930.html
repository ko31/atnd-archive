第13回vvvvオフ会！

毎月継続してきたオフ会もついに13回目！

内容

初心者向けのワークショップです！
ほとんど触ったことない人向けへの基礎講座です。
今回は長めに時間を確保し、ゆっくりやってみようと思います。
内容は、簡単なプロジェクションマッピングをやってみようと思います。

TECH LAB PAAKにツリーに投影という企画をやっており、
そこに乗っかり、ツリー型にプロジェクションしてみましょう！

初心者だけでなく、ベテランの方まで、幅広く参加をお待ちしています！
