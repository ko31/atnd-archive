Arduinoをはじめよう

イベント ユーザー
event_id 69874
title Arduinoをはじめよう
catch
description

Arduinoをこれから始めたい方、買ったものの引き出しの奥に眠らせている方向けの入門ワークショップです。電子工作初心者の人たちのために開発されたマイコンボードArduinoを紹介します。いくつかの基本的なサンプル(プログラムと電子回路）を作成して動作させ、Arduinoの入り口として、とにかく慣れてもらうことを目的とします。

日時

2015年9月13日（日）14:00～17:00
※午前の部も御座います。
午前の部はこちら→ Arduino_午前の部

場所

ファブラボ北加賀屋（コーポ北加賀屋1F）
大阪市営地下鉄四ツ橋線「北加賀屋駅」4番出口から徒歩10分

料金

お一人2,000円
※料金は当日会場でお支払いください。

必要機材

1.　ノートPC (Windows or Mac)
2.　Arduino UNO (互換機も可)
3.　USBケーブル(B)
4.　貸出機あり（Arduino UNOもしくはunconpatino互換機）
※貸出機は数に限りが御座います。

ワークショップの概要

1　Arduino紹介　概略、特徴、多様性等
2　IDEインストール　arduino HP の紹介。
3　ブレッドボードおよびパーツの説明。ブレッドボードの内部、パーツの極性等。
4　スケッチ実習。コメントを日本語化したスケッチと、解説を加えたテキスト(別記）に従い、ブレッドボードで回路を組み、動作させる。
5　Blink, Buton (プルダウンの学習), Digital Read Sirial (デジタルピンの読み方、シリアルモニタの使い方), Analog Read Sirial (アナログピンの読み方),
6　Fade (for()の使い方、インクリメントの使い方), If Statement Conditional (if()の使い方), Analog Read Voltage。
7　応用スケッチの学習(f.e. map()で入力値を適切な出力値に変換、等)
※適宜、休憩をはさみます。
※実施内容は都合により変更になる場合があります、ご了承ください。

講師

児島 正実、萩原 徹
