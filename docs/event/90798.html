吉祥寺朝活とは

朝早く起きて、有意義な一日を過ごしたい。でも、起きれないという方に向けて朝活を企画しました。朝活を習慣づける事により、好きな事を一つでも多くできるようにしましょう。朝活で進める内容に関しましては、自由です。私はWEBエンジニアの端くれなので、アプリケーション開発をしたり、ちょっかいをかけたりします。読書・プログラミング・音楽創作・手芸・ドラクエ何でも歓迎いたします。

日時・場所

2017年9/4（月）7:15 ~9:00 Pico Pico Cafe 吉祥寺

Pico Pico Cafe Web site

吉祥寺駅南口（公園口）から出たら、まっすぐ進んで大通りへと出る。突き当たりの「丸井」を右に曲がる。 「SUIT COMPANY」、「ドンキホーテ」を左手に見ながら歩く。不動産屋を過ぎて、学習塾「駿台」のすぐ先、ラーメン屋さん（麺僧）が一階にあるビルの8階が「ピコピコカフェ」です。（「メガネドラッグ」までいくと、行き過ぎです。戻って～。）

7:00から開場しております。8:30で一回締めさせて頂きます。時間に余裕のある方は9:00までご自由にして下さい。
