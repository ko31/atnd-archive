description

春からの新生活にむけ、これから年度末まで物件の引き渡しが最も多くなるシーズン。

特に注文住宅・建売住宅などの新築一戸建ての工事は、年度末にむけた非常にタイトなスケジュールで進みます。

工事を急ぐあまり、施工の質にもバラつきが出てくるなど、折からの人手不足もあいまって施工ミスや不具合も一年で一番増える時期です。

現在工事中の場合、こちらの４つの中でひとつでも思い当たる点はありませんか？

・いつ現場を訪れても現場監督さんが不在
・数百万円程度の大幅な値引きをしてもらって契約した
・工事のスケジュールが遅れている
・引き渡しの日が具体的に決まっていない

上記のようなことがある場合は、突貫工事になってしまっている恐れがあります。

内覧会に行ってみたら、まだ工事が終わっていなかった・・・というケースも残念ながらこの時期多いご相談です。

たとえば、こんなトラブル・・・（一戸建て内覧会同行サービスでインスペクターが指摘した、実際のケースです）

◆外構の工事がまだ終わっていない！

◆コンセントカバーが付いていない！

◆給湯器が設置されていない！

内覧会・施主検査は、建物の仕上がりを確認する最後の機会ですが、迫る引渡しに向けてやることは山のよう。巷には内覧会の情報や対策も溢れていますが、とてもじゃないけど全部読んでる暇はない！準備できない！というのが本音かもしれません。

そこで、さくら事務所では「今まさに新築一戸建て建築中」「完成物件の引き渡し前」という方に向けて、無事お引渡しを受けるためのチェックポイントをお伝えするセミナーを開催いたします。ご自身でできる内覧会チェックポイントや、竣工検査で確認すべきこと、施工会社さんとの上手なやりとりについてまで、ホームインスペクター（住宅診断士）が徹底解説！

忙しいこの時期だからこそ、引渡しにむけて建物の不安はこの2時間で解消しておきましょう！

一戸建て住宅（注文住宅、建売住宅、売建住宅）工事中の方、近く内覧会・竣工検査を予定されている方には必見のセミナーです！

【こんな方にオススメ！】

２月、３月の引き渡し予定で新築一戸建て（注文住宅・売建住宅）を建築中の方
新築一戸建て（注文住宅・建売住宅・売建住宅）の内覧会・竣工検査を控えた方
内覧会当日、どこをどのようにチェックすればよいか、いまいち自信が持てない方
忙しく、建物の仕上がりについて自分で調べる時間がないという方

【本セミナーで分かること】

引渡しまでに自身でチェックできるポイント集
内覧会当日、専門機材がなくてもできるチェック項目とテクニック
売主・施工会社さんとの上手な交渉の仕方
直し忘れをうやむやにしない！再内覧会・修繕確認の注意点
修繕が引き渡しまでに間に合わない！ときの対処法
