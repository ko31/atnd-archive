description

野球チームは、9人のプレーヤーが各々の役割を果たすと同時に、全体としてのチーム連携が求められます。ビジネスモデルも同じことがいえます。どのようなビジネスにも必須の9つのビジネス要素が相互に良い関係を生み出すことによって、長期にわたって持続可能なビジネスモデルとなります。

本セミナーでは、日本で話題を呼んでいる「ビジネスモデル・ジェネレーション」のルーツであるビジネスモデル・オントロジー（概念体系論）をベースに、ビジネスデザインに必要とされる9つのビジネス要素（およびサブ要素）、その属性やタイプ、要素間の関係性を視覚的に訴えるスライドを使ってご説明するイントロダクションコースです。さらに、ビジネスモデルのイノベーションに関するシステマチックなアプローチを実践するための今後の予定もご紹介します。

会場は、少人数でアットホームな雰囲気を体験していただける場所を選択しました。また、ご参加いただいた方には、ビジネスモデル編集のためのExcelワークシートをご提供します。
