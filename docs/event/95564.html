ＫＩＰは、スタートアップ企業やベンチャー企業を応援しています。

この度、エンジェル投資・ＶＣ投資・ファンド投資などを希望するスタートアップ企業等を対象としたセミナーを開催します。

このセミナーでは、経験豊富な官民ファンドのキャピタリストが、スタートアップ企業の見方、視点について解説します。

皆さま、奮ってご参加ください。

創業・起業セミナー『ベンチャーキャピタルが語る、スタートアップ企業の見方』概要

【日時】平成３０年３月２９日（木）　　１５：００～１７：００　【受付１４：４５～】

【会場】神奈川中小企業センタービル　１３階　第３会議室　(横浜市中区尾上町５－８０)

【対象者】エンジェル投資・ＶＣ投資・ファンド投資などを希望するスタートアップ企業等

【講演】

・時間：１５：００～１６：３０

・講師：株式会社産業革新機構　ベンチャー・グロース投資グループ　マネージングディレクター　鑓水英樹　氏

・タイトル：ベンチャーキャピタルが語る、スタートアップ企業の見方

・内容：ベンチャーキャピタリストとして多くのスタートアップ企業・ベンチャー企業と会ってきた鑓水氏。そんな鑓水氏がスタートアップ支援を通じて培った経験や知識などからスタートアップ企業の見方、視点について解説します。

【座談会】

・時間：１６：３０～１６：４５

・内容：鑓水氏が起業家の悩みにお応えします。

【ＫＩPインキュベート施設見学会】

・時間：１６：４５～１７：００

・内容：ＫＩＰインキュベート施設をご紹介します。

【参加費】無料

【定員】２０名（先着順）　

【申込先】http://www.kipc.or.jp/seminar_event/30512/

■株式会社産業革新機構■

株式会社産業革新機構は、2009年7月、産業や組織の壁を越えて、オープンイノベーションにより次世代の国富を担う産業を育成・創出することを目的に設立された官民ファンドです。

ＵＲＬ：https://www.incj.co.jp/

■講師略歴■

株式会社産業革新機構　ベンチャー・グロース投資グループ　マネージングディレクター　鑓水英樹　氏 国内保険会社、外資系コンサル会社、銀行を経て産業革新機構に参画。アーリーからレイター、北海道から四国・九州、農業から宇宙まで幅広いステージ・分野のベンチャー支援を実践中。
