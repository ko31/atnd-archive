description

自分の魅力を最大限に引き出し、輝かせるには、
勉強や運動などのいろいろな努力が必要。

でも、その努力を根本で支えてくれているのが、
「食生活」です。

イギリスに
「健康は人間が自分に贈ることのできる最高のプレゼントである」
という諺があります。

人間が本来持っている力を思う存分発揮するためにも、
カロリーや栄養素のほかに、
心にエネルギーを与えるような食事が必要です。

■ヒーリングフードとは、

「体に良いだけでなく心も豊かになり、
自分らしく輝くことができ、
社会を明るくする力にもなる食べもの、食べかた」

この講座ではヒーリングフード
「幸せになる食べもの・食べかた」
について、基礎的なことをしっかり学びます。

2日の集中講義を受講し、試験に合格すると、
ヒーリングフード・ジュニア・インストラクター
の認定を取得できます。

食と健康に関する知識が増え、
ご自身の食生活がよりよくなるのはもちろんのこと、
自宅サロンやセミナーを開いたり、
食の楽しみを分かち合いながら活動したりする
インストラクターを育成します。
