Excel マクロ (VBA) ハンズオンセミナー

学生から企業にお勤めの方まで、Excelを使う人ならどなたでも大幅な効率化ができるようになります。 特に非正規（派遣社員）の方は、VBAができるようになると派遣の時給が300円以上も上がることがあります。そういうスキルをたった1日で付けることを目標としたセミナーです。

サンプルとして行いますので通常数万円になるところ、会場費・資料費分となっております。

是非、ご参加ください。

2018年 2月10日（土） 9:30 ～ 16:45（9:15開場）@お台場

2018年 2月25日（日） 9:30 ～ 16:45（9:15開場）＠赤坂

  • Excelの数式・関数を難なく扱える方
