イベント概要

激辛スープカレーで有名なマジスパ下北沢店で、美味しくて辛いスープカレーを食べにいく会です。
激辛好きな人たち向けですが、辛さは調節できるので、辛いものが得意でない人もスープカレーを楽しめます。

マジスパには辛さが7段階あり、以下の順に辛くなるようです。瞑想が中辛程度。
覚醒 → 瞑想 → 悶絶 →→ 涅槃 → 極楽 →→ 天空 → 虚空

集合時間と場所

2011年07月30日 土曜日
18:10 下北沢駅 南口 集合
そろい次第みんなでマジスパへ歩いて行きます。徒歩5分程度です。

諸注意

人気店の為、予約無しで行くと1時間以上行列に並びます。
そのため人数分で予約をとってあります。
全員揃わないと入れないため、 当日は遅刻厳禁 でお願いします。

また、前もって予約の最終人数を店に伝える必要があるので、
外せない用事などにより 参加キャンセルをする場合は遅くとも7/27(水)まで に @takabow に連絡をお願いします。
(当日急病等の場合はこの限りではありません。連絡をください。)
