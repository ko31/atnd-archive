description

Two→Fiveというプロデュースユニットの主催をやっております、しもとりゆうと申します。

純粋に俳優の魅力を武器におもしろいことを追求したいという思いから演劇ユニットを旗揚げする運びとなりました。
魅力をひきだし、俳優の個性から生まれる笑いを軸にした独自のメソッドを用いた演出で公演を行います。
そのためのキャストオーディションも兼ねたワークショップを開催します。

魅力的な人間という対象に興味がある方を募集致します。

本ワークショップは、２０１１年３月２５日ー２７日に行われるツーファイヴ旗揚げ公演『コレナンダパンダ』に参加出来る方を始め、今後参加を希望される方への対象ともなっております。
既に予定が決まっている方、公演に出演するかはわからないけどワークショップだけでも参加してみたいという方等、今後の作品づくりの為にも沢山の人との出会いを広く募集します。
ご応募お待ちしております。
