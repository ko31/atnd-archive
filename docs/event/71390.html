 「結婚力」とは、結婚するだけでなく、幸せな結婚生活を持続する力です。

結婚力を上げれば、
未婚の方は、理想のパートナーを惹き付け、幸せな結婚をし、
既婚の方は、夫婦仲を理想な状態に改善し、幸せな結婚生活が送れます。

 

■とても厳しい現代の結婚事情



男性の5人に一人が、女性の10人に一人が結婚できません。
結婚しても3組に一組が離婚、3組に一組がセックスレス。
本当に幸せな結婚をしている方は極わずかです。

■幸せな結婚が難しい理由


それは、

結婚したいからではなくて、親と一緒にいるのが嫌だから結婚してしまう、
本当に結婚したい人ではなくて、目の前の結婚できる人と結婚してしまうからです。


■セミナーで得られること



自己催眠状態になります。

すると、自分と自分の可能性を、生まれたところから、死んで魂になるまで、自分の一生を通じてしっかり見つめる事ができます。

自分の心を愛で満たし、癒していくと、こんなにも幸せな人生が送れるということが実感できます。

そこで、自分も幸せな結婚ができると分かり、幸せな結婚への意欲が強まります。

■可能性の枠



非力な子象を杭に鎖で縛り付け、逃げ出せないようにしておくと、成長して力が強くなっても、鎖に繋いでおきさえすれば逃げ出さない。

鎖に繋がれていると逃げられない、と学習すると、逃げなくなるのです。

私たち人間も、自分がイメージできる範囲内での行動しかできません。

これを「可能性の枠」と言います


■鮮明にイメージする



私たちの脳には、

鮮明にイメージすればするほど、それがイメージなのか現実なのか区別がつかなくなるという特性があります。

しかし、通常の意識状態でイメージしても、イメージ力が低いので「可能性の枠」を広げるまでに至りません。


■幸せな結婚が困難な理由



私たちの姿形が両親に似ているように、性格も親譲りです。
自分では認めたくなくても、気づきにくくても、他人から見るとよく分かります。

祖父母や両親の時代は、女性の働き口のなさや教育から、今より離婚率は低かったのですが、男尊女卑や、嫁姑の諍い、個人間の結婚でなく家に嫁ぐ、見合い結婚が多い等、苦難に満ちた結婚を我慢していました。

その子孫である私たちは、世の中が自由になったとしても、幸せな結婚の「可能性の枠」は相変わらず小さいのです。

幸せな結婚のイメージが上手にできないので、上手くいかないのです。

■自己催眠で、幸せな結婚の「可能性の枠」を広げる


今回のセミナーでは、自己催眠状態になって

自分の幸せな結婚を妨げている本当の原因と、その問題解決法を探ります。
ある特別な方法を使うと、すんなり分かります。

そして、その問題が解決している様子をイメージして、とても幸せな気分になります。
その状態で、幸せな結婚をして、幸せな人生を送っている自分自身をイメージします。

自己催眠状態で、幸せな自分をイメージすると、とても上手にイメージできるので、この上のない幸福感を感じます。

■自宅でも自己催眠を続ける


セミナーに参加すると、自己催眠の音声誘導が期間限定で自宅でも聞けるようになります。

セミナー中にこの上ない幸福感を感じているので、自宅での自己催眠も難なく続けられるようになります。

気がついて見ると、幸せな結婚の「可能性の枠」が大きく広がっているのです 。

 

■申し込み


さあ、あなたもセミナーに申し込みましょう。
