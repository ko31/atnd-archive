【このイベントは、募集を締め切りました】
　皆さまありがとうございます！

Webクリエイターの皆さま！

今月はいつもの座談会ではなく、
【目指せMVP！街中デザイン激写会】を開催します☆

… 考えてみれば…
街中のもの、自然物以外ほとんどのものが
デザインされている！

ビル、看板、標識、会社のロゴ、マンホール…

街中で見つけた素敵なデザインを、カメラで激写！

その後、皆が撮った写真を持ち寄り、
いかに素晴らしいデザインかを語っていただきます（笑）

ただし、対象となるものが多すぎるので、
街に出掛ける前にくじを引いてもらい、
担当カラーを決定します！

「赤」のくじを引いた人は、赤色のデザインを探すのがミッション☆
（ex.某銀行のロゴ、ポストetc.）

いつもは気が付かないようなものが、きっと目につくはず！

そしてそして！
今回は主催者である小原が、独断と偏見により
MVPを決めさせていただきます☆

MVPには豪華商品付き（^０^）ノ
ぜひMVP目指しましょー！！！

== 主催者紹介 ==

◆小原若奈 （おはらわかな）
「ここから – coco*color-」代表。
Webデザイン、イラスト、キャクター・漫画制作をしつつ、多くの人の力を最大限発揮するため、コーチングやカウンセリングを行っている。

1982年京都府生まれ。
高校1年生で集英社にて漫画家デビュー、大学卒業後、不動産営業を経験し、新人賞・全社達成率1位を獲得。
Webマーケティング会社にてWebデザイナー、ディレクター、クリエイティブマネージャー、社内業務効率化メンバーと多岐に渡る業務に携わる。
心理学を学び社内でのコミュニケーションの向上を図る。
2010年独立。現在に至る。

http://wakana-create.info/

◆野中萌子（のなかもえこ）
WEBデザインを中心に、企画、イラスト、グラフィック、またセミナー等の分野で活動中。

1988年京都府生まれ。
京都芸術デザイン専門学校を卒業後、株式会社AZ(大阪) 勤務。
クリエイティブマネージャーとしてチームのデザインクオリティの向上・進行管理などを主に行う。2010年株式会社ヌーサイト(東京)勤務。
現在に至る。

http://moco2.pupu.jp/blog/

== ShareCafeについて ==

Web制作に関わる全ての人が、互いの情報をシェアするコミュニティです。
参加してくださった方はShareCafe会員となっていただき、
参加回数に合わせて特典をプレゼント！
・自己紹介スペース
・facebookで作品を掲載
・インタビュー
・1講義をお任せ　などなど…

Web制作に関わる人同士の交流を深め、
楽しく互いに高めあっちゃう場を目指しています。

そしてゆくゆくは、参加者同士で大きなプロジェクトを立ち上げられたらステキ！！

あなたに必要な、
Webクリエイティブに関するヒントが見つかることを願っています♪
