コーレル株式会社(本社︓東京都港区、会⻑︓堺 和夫)は、世界中の⼤企業で導⼊実績のある「働き⽅改⾰」を「⾒える化」で⽀援するMindManagerの最新版、MindManager 2019 for Windowsの新製品発表セミナーを⾏います。

これにあたり、急遽、⽶国よりMindManager製品担当副社⻑であるマイケル ドイッチ(Michael Deutch)が来⽇し、なぜ、MindManagerが「働き⽅改⾰」になるのか、MindManager 2019 for Windowsの新機能が「⾒える化」にどのように貢献するのか、各国の事例にも精通する、製品の責任者としてお話をさせていただきます。
※この発表会は、英語で⾏われ逐次通訳いたします。

■開催の背景

「働き⽅改⾰」が叫ばれる中、在宅ワークなどの制度的な⾯しか論じられず効率的な働き⽅について具体的な解決策が⽰されていません。こうしたなか、海外ではMindManagerを使い労働時間の短縮、業務の効率化に成果を上げている企業が多くあります。
アイデアのデジタル化だけにとどまらず、情報の統合や処理までこなす情報の「⾒える化」ソフトの最新版 MindManager について、知っていただく機会になればと考えています。
