catch FM Talkers Oct. 2010
description

福岡市のふくふくプラザにおきましてファイルメーカー Pro 九州ユーザーズミーティング (以下、FM-Kyushu) の定例イベント FM Talkers を開催します。

FM-Kyushu のオーカワによる「FileMakerことはじめ」では、FileMaker の基礎にフォーカスし、フィールド定義、関数やリレーション、一般的なスクリプトについて再考してみようと思います。

続いて大分の本田氏による「初心者が FileMaker を学ぶには」では、全くの初心者が一人で開発や運用を出来るよう指導した経験の中から、効果的だった習得方法や考え方も紹介します。

さらに FM-Kyushu では皆様のご自慢ソリューションを大募集します。

「このテクニックを誰かに見てほしい」
「こういう実装はどうなのか経験者に聞いてみたい」

とお考えの方は、ぜひ FM-Kyushu 事務局にご連絡ください。

その他、お悩み相談や、自分で作ったソリューション自慢など FileMaker 話で盛り上がりましょう。

・お仕事がある方でも参加しやすいように 18 ～ 21 時の開催となります。
・FileMaker ユーザ及び FileMaker にご興味をお持ちの方は是非ご参加ください。
・ソリューションを持ち込まれる方は当日の持ち込みでも構いません。
　ノート PC 等をお持ちいただけると、より盛り上がるかと思います。

詳しくは、ファイルメーカー Pro 九州ユーザーズミーティングの Web サイトをご覧ください。

・勉強会への参加費：500 円 (ドリンク代を含む)

※ 勉強会後に懇親会を予定しております。場所は会場近辺で、予算は 3,500 〜 4,000 円程度で考えております。
