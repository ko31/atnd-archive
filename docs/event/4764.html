ネタ的にはフリーランスの集まりですが６月のTKPのホームパーティーです。お誘い会わせの上。

今回はなんか女性割引男高めというそれだけでアラサーの気配がし、TKPも歳をとったな的なコンフィグです。


== ネタ ========
今探している物、提供できるもの、こころの中に用意する。紙を用意するのでそれに書いてもらう。席替え的に時間決めて話ししたりする、的なノリが１時間に１０分位、東窪の掛け声でやられたりします。なんかこう出会いとかうぜーという人たちが本来的にちゃんと出会うところに集中したいです。パーティーとは本来そういうものだと。




あとは、フリーランスで、１０年やってきた人、１年目の人、なってみたい人、就職しようとしてる人などで情報共有したりサバイブすることについて話すなど。仕事はどこからくるのか。儲けはどこからくるのか。資金繰り、営業、作業、集中コントロール、自由の意味、場所、人脈などについて。起業、スケール（サイズ）、投資、などについても。

会費：
　飯食う男：3000円
　食わない男：2000円。（費用から算出）
　食わない女：1000円。
　食う女：1500円

今度こそTKP初のケータリング的なものを考えています。
