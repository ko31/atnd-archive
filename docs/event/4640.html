title TechWaveハッカソン May 2010
catch TechWaveのiPadアプリを作ろう！
description

我らがTechWave主催の1dayハッカソン。今回はTechWaveのiPadアプリを、参加者全員で役割分担して開発します。

＜募集メンバー＞
プログラマー・・・5-6名
デザイナー・・・1-2名
その他技術・・・1-2名

全員で10名以内を予定しています。応募者多数の場合は、経験が豊富な方を一部優先し、全体の技術バランスが取りつつ、ソフトウェアを完成できるメンバー構成になるよう、主催者側で選抜します。とはいえ初心者も歓迎です。参加意志の強い方は、アンケートフォームで応募の動機をアピールしてください。
なお、会場は禁煙です。

＜運営事務局＞
株式会社ゴーガ 小山(TechWaveゲストブロガー)

＜今回のゲストエンジニア＞
シリウスラボ所長、ジオメディアサミット運営事務局代表も務める株式会社シリウステクノロジーズの関治之さんです。

＜お知らせ＞
TechWaveハッカソンでは、運営スポンサーを募集しています。スポットスポンサー、シーズンスポンサーのいずれも可能ですので、興味がある方はお問い合わせください。 → koyama@goga.co.jp

開催中、湯川さんと本田さんが応援に来ます。
