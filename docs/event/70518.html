独立行政法人国際協力機構（JICA）が実施する青年海外協力隊事業は、今年50周年を迎えました。 あーすぷらざでは、『カナガワから世界へ　世界からカナガワへ』と題したリレーセミナー（全3回）を開催致します。 青年海外協力隊をご存知の方も、そうでない方も、この機会に青年海外協力隊事業の次の50年について考えてみませんか？

Vol.1 ～青年海外協力隊と世界を語ろう～

青年海外協力隊の2 年間は、その後の人生にどんな影響を与えるのでしょうか。 神奈川県在住の協力隊OB・OG をゲストに迎え、世界で得た経験を今どう生かしているのか、座談会形式でお伝えします。 その後、教育、食、命、こどもなどのテーマ別にゲスト、参加者と一緒にディスカッションを行います。 学生、社会人、国際協力に興味のある方の参加をお待ちしています。
