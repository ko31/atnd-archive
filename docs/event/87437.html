〜自動運転ドローンをプログラミングしよう〜

プログラミング学習者の方、ぜひ一緒にゲーム感覚でスキルアップしませんか？当イベントでは、初心者から上級者まで、プログラミング学習者がチームに分かれて自動運転ドローンをプログラミングし、障害物レースを行ないます。必要な持ち物はパソコンだけ。今お申込みいただくと完全無料（4月23日以降は下記料金をいただきますので、予めご了承ください）、ドリンク・2食分のお食事つきです。優勝者には１万円相当の景品もご用意しております。年齢不問、国籍不問（日本語・英語対応）ですので、ご興味のある方は申込フォームにご記入ください。みなさまと当日お会いできることを楽しみにしております！

なお、当イベントはプログラミング言語（C / JavaScript / Python）を学習中または習得している方を対象としています。プログラミングを学習中でない方も、イベント参加をご希望の場合は、お気軽にfacebookにてメッセージください！

Mobility IoT Hackathon in Tokyo “Program an autonomous drone yourself”

Would you like to add some entertainment to your programming learning? In our hackathon, you’ll be working in a team with other programmers to program a drone so that you can have your drone/car join the autonomous obstacle racing at the end of the event! All you need to bring is your computer. The event is free of charge for early applicants (we will start charging fees after April 23rd as shown below), and we’ll serve you drinks and 2 meals. We’ll also have the prize ready for the winner. We welcome all ages and nationalities (Japanese-only / English-only speakers are both welcomed), so if you’re interested in joining us, please fill out the signup form. We are very looking forward to seeing you!

This event is for those who have already acquired OR been learning programming (C / JavaScript / Python). If you’re not a programmer but interested in joining this event, please feel free to message us on facebook!
