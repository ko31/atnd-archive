catch 固体材料の熱拡散率および熱伝導率の代表的な測定方法について、その原理と特徴を解説！
description

【講座のポイント】
　固体材料の熱拡散率および熱伝導率の測定方法には様々な手法があります。信頼性の高い測定結果を得るためには、測定方法の原理と特徴を理解し、目的にあった測定方法を選択することが重要です。

　本講演では、固体材料の熱拡散率および熱伝導率の代表的な測定方法について、その原理と特徴を紹介します。また、測定結果の信頼性をどのように考えたら良いか、測定方法を選択する際に参考になると思われるヒントを説明します。特に、レーザフラッシュ法を例として、熱拡散率測定の不確かさ評価も紹介します。

【プログラム】
１．はじめに
２．熱伝導率・熱拡散率・熱浸透率
　　2-1 熱物性値の定義（熱伝導率・熱拡散率・熱浸透率）
　　2-2 熱伝導率・熱拡散率の評価対象の材料
３．熱伝導率・熱拡散率測定方法の原理と特徴
　　3-1　測定方法の分類
　　3-2　定常法による熱伝導率測定（保護熱板法、熱流計法など）
　　3-3　非定常法による熱拡散率測定
　　　3-3-1　パルス加熱法の原理と応用
　　　　　　（レーザフラッシュ法、高速パルス加熱サーモリフレクタンス法など）
　　　3-3-2　周期加熱法の原理と応用
　　　　　　（光交流加熱法、スポット周期加熱法など）
　　　3-3-3　ステップ加熱法の原理と応用
　　　　　　（非定常細線加熱法など）
４．測定方法の選択のヒント
　　4-1　測定方法の適用範囲の検討
　　4-2　測定方法の選択時における検討事項
　　　4-2-1　測定方法の特徴の視点から
　　　4-2-2　測定対象の材料の視点から
５．測定結果の信頼性評価について
　　5-1　標準物質や測定規格の利用
　　5-2　不確かさ評価（レーザフラッシュ法の場合を例として）
６．まとめ
