■a-blog cms 1.4 リリース記念！！九州合宿！

2011年4月5日に1.4がリリースされました。新機能を中心に、a-blog cms の基本的な紹介を行うセミナーを日本各地で開催する内の一つとして、九州では合宿として開催することになりました。

「a-blog cms使っているけど、まだよく分からない。」「他の人がやったカスタマイズの方法が知りたい。」「というか、よさそうな感じだけどまだ使ったことがない。」という様々なユーザーにお答えできる合宿になればと考えています。

■合宿スケジュール

まだ確定はしていませんが、おおまかなスケジュールは以下のようになっております。詳細につきましては後日お伝えいたします。

■a-blog cms 1.4のご紹介＆1.4をさわってみよう！（13:00-18:00）

■みんなで夕食（18:30-19:30）

■私はこんなカスタマイズをした！ライトニングトーク&1.4をさわってみよう！
（20:00-22:00）

■自由時間（22:00-）

■自由に朝食（7:30-8:30）

■1.4をさわってみよう！＆そして発表タイム！（9:00-11:30）

■コースは3種類！選ぶのはあなた次第！

開催日時を考慮して3つのコースをご用意いたしました。

■日帰りコース（参加料3,150円）

1日目の夕食前までのプログラムに参加できます。

■日帰り＆夕食付きコース（参加料8,400円）

1日目のプログラムに参加できます。

■合宿コース（参加料13,650円）

全プログラムに参加できます。2日目の朝食も付いています。

■参加特典

今回の合宿では、アップルップルさんのご好意で募集時点で以下のような特典を設定させて頂きます。特典の対象は1社(複数のスタッフさんが参加されても1つ)、1会場(紹介、ハンズオンのどちらか1つ)となります。ご了承ください。

特典A. 初めて利用する方にはベースライセンスを割引

  • 1ライセンス 31,500円
  • 2ライセンス 52,500円

特典B. 既にライセンスを購入された事がある方にはオプションを1点プレゼント

  • +10ユーザーライセンス (52,500円)
  • ショップ拡張ライセンス (52,500円)
  • HTMLメルマガ拡張ライセンス (52,500円)

■PCの持ち込みについて

今回はプロジェクタの画面を見ながらのご紹介もありますが、実際にさわってカスタマイズを行っていただきます。なのでノートPCが必要となります。ネット環境に関してはめかり山荘様のご好意で用意いたしますが、光回線ではないため、ご用意出来る方は用意していただけると助かります。

■開催地：めかり山荘について

今回お世話になる国民宿舎めかり山荘は、来年の春で惜しまれつつ閉館の予定となっております。今回の合宿が一つの思い出として記念の合宿にしてください。
