開催概要

東京で開催される jus共催 第42回BLACK HOLEシェル芸勉強会 に、松江からリモートで参加します。

大阪、長崎 と、各地でサテライト会場が開催されており、松江では今回が初開催になります。

YouTube でライブ配信される様子を一緒に見ながら、出題される問題をシェルとコマンドを駆使して解いていきます。 問題の答えがわかったら、是非 Twitter にハッシュタグ #シェル芸 を付けて投稿してみましょう。

同じ問題でも、いろんな人のいろんな解き方があるため、他の人と回答を共有してみると、普段の作業を効率化する意外なヒントが得られるかもしれません。

会場について

フェンリル株式会社 島根支社の会議室を借りて開催します。 12:15 頃を目処にタイムプラザビル 3F (ここ の (7))にお越しください。

遅れて来られる場合は、フェンリルの受付にある電話でスタッフを呼び出してください。

電車・バスでお越しの場合

  • JR 松江駅より
    • 一畑バス「総合体育館・メッセ行き」、総合体育館前下車 徒歩約 5 分
    • または市バス「北循環線内回り」、学園南二丁目下車 徒歩約 10 分

車でお越しの場合

  • タイムプラザビル正面玄関手前のお客様用駐車場 ( 4台まで )
    • 台数に限りがありますので、駐車できない可能性があります。できるだけ電車・バスにてお越しくださいますよう、ご協力をお願いします。

持ち物

UNIX 環境のあるノート PC を持参ください。

  • Mac / Windows (Cygwin, WSL) / Linux / *BSD など
  • VirtualBox や Docker などの仮想環境上でも大丈夫です
  • 会場に WiFi 環境があります
  • 勉強会で使用するデータは、GitHub で公開される予定のため、git で取得できるようにしておくと良いでしょう

シェル芸って何??

  • このあたり を見てみると、なんとなくわかるかも！
  • 前回シェル芸勉強会の様子はこちら
  • Twitter でシェル芸bot (@minyoruminyon) というアカウントが運用されています
    • アカウントをフォローした後、フォローバックされれば、#シェル芸 ハッシュタグを付けて投稿したコマンドを実行して、結果を返してくれます

連絡先

何かわからないことがあれば、Twitter で @3socha まで連絡ください。 ただし、@3socha は普段大阪サテライトの会場担当をしている人なので、道に迷っても案内できない自信があります。会場の場所をよく調べてからお越しください。
